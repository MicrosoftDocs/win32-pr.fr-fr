# <a name="dictionaries-and-factoids"></a>Dictionnaires et Factoids

Un dictionnaire est un lexique interne qu’un module de reconnaissance utilise pour déterminer les correspondances de mots et les alternatives. Un Factoid augmente la précision de la reconnaissance en sélectionnant un contexte spécifique pour un champ particulier.

Pour plus d’informations sur les dictionnaires, consultez [utilisation de dictionnaires d’applications](using-application-dictionaries.md).

Pour plus d’informations sur Factoids, consultez [utilisation du contexte pour améliorer la précision](using-context-to-improve-accuracy.md).
