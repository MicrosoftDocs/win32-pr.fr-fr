# <a name="flicks-enumerations"></a>Énumérations de raccourcis

Cette section contient les énumérations utilisées dans le [**\_ message de \_ raccourci de la tablette WM**](wm-tablet-flick-message.md).



| Énumération                                                 | Description                                                                                                                                                   |
|-------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**FLICKDIRECTION**](/windows/desktop/api/tabflicks/ne-tabflicks-flickdirection)                    | Représente la direction du raccourci.<br/>                                                                                                             |
| [**FLICKMODE**](/windows/desktop/api/tabflicks/ne-tabflicks-flickmode)                              | Spécifie l’endroit où les mouvements de raccourcis sont visibles.<br/>                                                                                                        |
| [**FLICKACTION \_ COMMANDCODE**](/windows/desktop/api/tabflicks/ne-tabflicks-flickaction_commandcode) | Spécifie le code de commande classé dans la [**\_ structure de données de raccourci**](/windows/desktop/api/tabflicks/ns-tabflicks-flick_data) et représente le type d’action de raccourci à effectuer.<br/> |
| [**SCROLLDIRECTION**](/windows/desktop/api/tabflicks/ne-tabflicks-scrolldirection)                  | Spécifie le sens du déplacement de la case de défilement.<br/>                                                                                               |
| [**Keymodifier**](/windows/desktop/api/tabflicks/ne-tabflicks-keymodifier)                          | Spécifie les touches de modification, le cas échéant, qui ont été enfoncées lorsque le mouvement de mouvement a eu lieu.<br/>                                                               |
