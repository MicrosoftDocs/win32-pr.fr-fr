# <a name="recognizer-dll-sample-template"></a>Exemple de modèle de DLL de reconnaissance

L’exemple de DLL de module de reconnaissance contient un modèle pour la création d’une DLL de module de reconnaissance. Le modèle contient des fonctions pour l’inscription et l’annulation de l’inscription du serveur, ainsi que des squelettes pour les fonctions de reconnaissance principales.

Ce modèle montre comment inscrire un CLSID dans le registre. Vous pouvez également effectuer cette opération lorsque vous installez votre module de reconnaissance à l’aide de votre programme d’installation. Veillez à créer votre propre CLSID unique pour vos langues et ne réutilisez pas le CLSID indiqué dans l’exemple.

Par défaut, les exemples sont installés dans le répertoire d’exemples sous le répertoire dans lequel vous avez installé le kit de développement logiciel (SDK).

Pour obtenir un exemple d’utilisation des module de reconnaissance à partir de l’API de la plateforme Tablet PC, consultez l' [exemple de reconnaissance avancée](advanced-recognition-sample.md).
