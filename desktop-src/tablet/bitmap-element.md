# <a name="bitmap-element"></a>Élément Bitmap

Contient les données encodées en base64 pour une image bitmap contenue dans le fichier journal sous la forme d’une image d’arrière-plan.

## <a name="definition"></a>Définition

``` syntax
<xs:element name="Bitmap" type="xs:base64Binary" minOccurs="0" />
```

## <a name="parent-elements"></a>Éléments parents

[**Image**](image-element.md)

## <a name="child-elements"></a>Éléments enfants

Aucun.

## <a name="attributes"></a>Attributs

Aucun.

## <a name="element-information"></a>Informations sur les éléments



|                  | Valeur                                      |
|------------------|--------------------------------------------|
| **Type d’élément** | xs:base64Binary                            |
| **Espace de noms**    | urn : schemas-microsoft-com : TabletPC : RichInk |
| **Nom du schéma**  | Lecteur de journal                             |
