# <a name="docimage-element"></a>Élément DocImage

Si la note de journal en cours de lecture a été créée à l’aide du pilote d’imprimante du rédacteur de note de journal, l’image de document en arrière-plan est conservée dans cet élément.

## <a name="definition"></a>Définition

``` syntax
<xs:element name="DocImage" type="xs:base64Binary" minOccurs="0" />
```

## <a name="parent-elements"></a>Éléments parents

[**JournalPage**](journalpage-element.md)

## <a name="child-elements"></a>Éléments enfants

Aucun.

## <a name="attributes"></a>Attributs

Aucun.

## <a name="element-information"></a>Informations sur les éléments



|  Élément     | Valeur                                                     |
|--------------|--------------------------------------------|
| Type d'élément | **xs:base64Binary**                        |
| Espace de noms    | urn : schemas-microsoft-com : TabletPC : RichInk |
| Nom du schéma  | Lecteur de journal                             |
