# <a name="inkedit-enumerations"></a>Énumérations InkEdit

Cette section contient les énumérations appartenant au contrôle InkEdit.



| Énumération                                            | Description                                                                                                                                              |
|--------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**AppearanceConstants**](/windows/desktop/api/inked/ne-inked-appearanceconstants)     | Spécifie si le contrôle apparaît plat ou 3D.<br/>                                                                                            |
| [**BorderStyleConstants**](/windows/desktop/api/inked/ne-inked-borderstyleconstants)   | Spécifie si le contrôle a une bordure.<br/>                                                                                                   |
| [**InkApplicationGesture**](/windows/desktop/api/msinkaut/ne-msinkaut-inkapplicationgesture) | Spécifie l’intérêt d’un ensemble de mouvements spécifiques à l’application.<br/>                                                                             |
| [**InkDisplayMode**](/windows/desktop/api/inked/ne-inked-inkdisplaymode)               | Spécifie si une sélection apparaît en tant qu’entrée manuscrite ou texte.<br/>                                                                                         |
| [**InkEditStatus**](/windows/desktop/api/inked/ne-inked-inkeditstatus)                 | Spécifie si le contrôle [InkEdit](inkedit-control-reference.md) est inactif, collecte de l’encre ou reconnaissant l’encre.<br/>                           |
| [**InkInsertMode**](/windows/desktop/api/inked/ne-inked-inkinsertmode)                 | Spécifie la façon dont l’entrée manuscrite est insérée dans le contrôle [InkEdit](inkedit-control-reference.md) .<br/>                                                      |
| [**InkMode**](/windows/desktop/api/inked/ne-inked-inkmode)                             | Spécifie les paramètres du mode de collecte pour l’encre dessinée, si la collection d’encres est désactivée, si l’encre est collectée ou si les entrées et les mouvements sont collectés.<br/> |
| [**InkMouseButton**](/windows/desktop/api/msinkaut/ne-msinkaut-inkmousebutton)               | Spécifie le bouton de la souris qui a été enfoncé.<br/>                                                                                                     |
| [**InkMousePointer**](/windows/desktop/api/msinkaut/ne-msinkaut-inkmousepointer)             | Spécifie le type de pointeur de la souris qui s’affiche.<br/>                                                                                             |
| [**MouseButton**](/windows/desktop/api/inked/ne-inked-mousebutton)                     | Spécifie le bouton de la souris qui a été enfoncé.<br/>                                                                                                     |
| [**ScrollBarsConstants**](/windows/desktop/api/inked/ne-inked-scrollbarsconstants)     | Spécifie la façon dont les barres de défilement d’un contrôle [InkEdit](inkedit-control-reference.md) s’affichent à l’écran.<br/>                                    |
| [**SelAlignmentConstants**](/windows/desktop/api/inked/ne-inked-selalignmentconstants) | Spécifie l’alignement du paragraphe par rapport aux marges du contrôle [InkEdit](inkedit-control-reference.md) .<br/>                     |
