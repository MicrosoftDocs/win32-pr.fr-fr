# <a name="conflicts-between-application-gestures"></a>Conflits entre les gestes de l’application

Certains mouvements d’application sont suffisamment similaires pour provoquer des conflits potentiels pendant la reconnaissance. N’utilisez pas simultanément des mouvements d’application potentiellement conflictuels dans votre application.
