# <a name="date-element"></a>Élément Date

Contient la date à laquelle la note du journal a été créée.

## <a name="definition"></a>Définition

``` syntax
<xs:element name="Date" type="xs:dateTime" />
```

[**TitleInfo**](titleinfo-element.md)

Aucun.
