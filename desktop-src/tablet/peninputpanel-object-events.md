# <a name="peninputpanel-object-events"></a>Événements de l’objet PenInputPanel

Le tableau suivant décrit les threads sur lesquels les événements d’objet [**PenInputPanel**](peninputpanel-class.md) peuvent se déclencher.



| Événement                                                  | Threads                                                                                              |
|--------------------------------------------------------|------------------------------------------------------------------------------------------------------|
| [**InputFailed**](peninputpanel-inputfailed.md)       | Se déclenche sur le thread dans lequel le [**PenInputPanel**](peninputpanel-class.md) est créé.<br/> |
| [**PanelChanged**](peninputpanel-panelchanged.md)     | Se déclenche sur le thread dans lequel le [**PenInputPanel**](peninputpanel-class.md) est créé.<br/> |
| [**PanelMoving**](peninputpanel-panelmoving.md)       | Se déclenche sur le thread dans lequel le [**PenInputPanel**](peninputpanel-class.md) est créé.<br/> |
