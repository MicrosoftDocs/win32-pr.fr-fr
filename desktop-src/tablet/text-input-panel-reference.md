# <a name="text-input-panel-reference"></a>Référence du panneau de saisie de texte

Permet d’accéder aux fonctionnalités programmables du panneau de saisie Tablet PC.

## <a name="in-this-section"></a>Dans cette section

-   [Messages du panneau de saisie de texte](tip-messages.md)
-   [Énumérations du panneau de saisie de texte](text-input-panel-enumerations.md)
-   [Interfaces du panneau de saisie de texte](text-input-panel-interfaces.md)

## <a name="com-implementation"></a>Implémentation COM

La rubrique contient des références pour les interfaces et les énumérations COM du panneau de saisie de texte.

Ces interfaces peuvent être instanciées en appelant la méthode [**CoCreateInstance**](/windows/win32/api/combaseapi/nf-combaseapi-cocreateinstance) en C++.

## <a name="remarks"></a>Notes

Les interfaces et les énumérations du panneau de saisie de texte permettent de contrôler l’apparence et le comportement du panneau de saisie.

Ces éléments sont déclarés dans PenInputPanel. h.
