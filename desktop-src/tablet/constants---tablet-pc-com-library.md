# <a name="core-tablet-pc-constants"></a>Constantes Tablet PC principales

Cette section contient les constantes principales pour Tablet PC.

## <a name="in-this-section"></a>Dans cette section



| Constante                                                 | Description                                                                                                                                                                                                                                               |
|----------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Factoid](factoid-constants.md)                         | Définit des valeurs de chaîne constantes qui sont utilisées pour augmenter la précision de la reconnaissance en fournissant des informations contextuelles au module de reconnaissance.<br/>                                                                                                           |
| [PacketPropertyGuids](packetpropertyguids-constants.md) | Définit des valeurs qui spécifient les propriétés du paquet. L’interface de programmation d’applications (API) Tablet PC utilise des identificateurs globaux uniques (GUID) pour identifier les propriétés des paquets, qui sont des chaînes constantes dans Automation.<br/>                     |
| [RecognitionProperty](recognitionproperty-constants.md) | Définit des valeurs qui spécifient les propriétés d’un substitut de reconnaissance. L’interface de programmation d’applications (API) Tablet PC utilise des identificateurs globaux uniques (GUID) pour identifier les propriétés des paquets, qui sont des chaînes constantes dans Automation.<br/> |
