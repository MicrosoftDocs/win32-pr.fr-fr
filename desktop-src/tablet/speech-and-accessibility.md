# <a name="speech-and-accessibility"></a>Reconnaissance vocale et accessibilité

Pour que les applications soient accessibles, les informations sur le contenu de l’écran doivent être exposées à l’aide des utilitaires de reconnaissance vocale. Ces utilitaires doivent identifier et nommer des contrôles spécifiques afin qu’ils puissent reconnaître un nom de contrôle parlé et manipuler ce contrôle en réponse aux commandes vocales.

Par exemple, lors de l’utilisation d’un utilitaire de saisie vocale, un utilisateur peut indiquer « fichier », puis « mise en page » pour choisir l’élément de menu commande correspondant, puis « paysage » pour activer la case à cocher correspondante. Vous devez exposer les informations sur ces éléments pour atteindre l’objectif souhaité.
