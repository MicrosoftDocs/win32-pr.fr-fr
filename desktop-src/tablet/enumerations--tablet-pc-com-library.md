# <a name="core-tablet-pc-enumerations"></a>Énumérations du Tablet PC de base

Cette section contient les énumérations principales pour Tablet PC.

| Énumération                                                                                  | Description                                                                                                                                                                                                                                                                               |
|----------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**InkApplicationGesture**](/windows/desktop/api/msinkaut/ne-msinkaut-inkapplicationgesture)                                       | Spécifie l’intérêt d’un ensemble de mouvements spécifiques à l’application.<br/>                                                                                                                                                                                                               |
| [**Énumération InkBoundingBoxMode**](/windows/desktop/api/msinkaut/ne-msinkaut-inkboundingboxmode)                                 | Spécifie les caractéristiques d’un trait, comme les attributs de dessin, qui sont utilisées pour calculer le cadre englobant de l’encre.<br/>                                                                                                                                                    |
| [**InkClipboardFormats**](/windows/desktop/api/msinkaut/ne-msinkaut-inkclipboardformats)                                           | Spécifie le format d’encre stocké dans le presse-papiers.<br/>                                                                                                                                                                                                                   |
| [**InkClipboardModes**](/windows/desktop/api/msinkaut/ne-msinkaut-inkclipboardmodes)                                               | Spécifie les options de copie du presse-papiers.<br/>                                                                                                                                                                                                                                   |
| [**InkCollectionMode**](/windows/desktop/api/msinkaut/ne-msinkaut-inkcollectionmode)                                               | Détermine si l’encre, les gestes ou l’encre et les gestes sont reconnus au fur et à mesure que l’utilisateur écrit.<br/>                                                                                                                                                                                       |
| [**InkCollectorEventInterest**](/windows/desktop/api/msinkaut/ne-msinkaut-inkcollectoreventinterest)                               | Spécifie si un événement s’est produit sur un collecteur d’encres et, le cas échéant, l’événement déclenché.<br/>                                                                                                                                                                                         |
| [**InkCursorButtonState**](/windows/desktop/api/msinkaut/ne-msinkaut-inkcursorbuttonstate)                                         | Spécifie l’état d’un bouton de curseur.<br/>                                                                                                                                                                                                                                        |
| [**InkDivisionType**](/windows/win32/api/msinkaut15/ne-msinkaut15-inkdivisiontype)                                                   | Définit des valeurs pour les types structurels au sein de l’objet [**IInkDivisionResult**](/windows/desktop/api/msinkaut15/nn-msinkaut15-iinkdivisionresult) .<br/>                                                                                                                                                                    |
| [**InkExtractFlags**](/windows/desktop/api/msinkaut/ne-msinkaut-inkextractflags)                                                   | Spécifie la partie d’un trait à supprimer d’un objet [**InkDisp**](inkdisp-class.md) .<br/>                                                                                                                                                                                     |
| [**InkMouseButton**](/windows/desktop/api/msinkaut/ne-msinkaut-inkmousebutton)                                                     | Spécifie le bouton de la souris qui a été enfoncé.<br/>                                                                                                                                                                                                                                      |
| [**InkMousePointer**](/windows/desktop/api/msinkaut/ne-msinkaut-inkmousepointer)                                                   | Spécifie le type de pointeur de la souris à afficher.<br/>                                                                                                                                                                                                                                 |
| [**InkOverlayAttachMode**](/windows/desktop/api/msinkaut/ne-msinkaut-inkoverlayattachmode)                                         | Spécifie où attacher le nouvel objet [**InkOverlay**](inkoverlay-class.md) , derrière ou devant la couche active.<br/>                                                                                                                                                       |
| [**InkOverlayEditingMode**](/windows/desktop/api/msinkaut/ne-msinkaut-inkoverlayeditingmode)                                       | Spécifie le mode de comportement de l’objet [**InkOverlay**](inkoverlay-class.md) et le contrôle [InkPicture](inkpicture-control-reference.md) .<br/>                                                                                                                               |
| [**InkOverlayEraserMode**](/windows/desktop/api/msinkaut/ne-msinkaut-inkoverlayerasermode)                                         | Spécifie la façon dont l’entrée manuscrite est effacée de l’objet [**InkOverlay**](inkoverlay-class.md) et du contrôle [InkPicture](inkpicture-control-reference.md) .<br/> Ce mode est utilisé lorsque [**InkOverlayEditingMode**](/windows/desktop/api/msinkaut/ne-msinkaut-inkoverlayeditingmode) est défini sur Delete.<br/> |
| [**InkPenTip**](/windows/desktop/api/msinkaut/ne-msinkaut-inkpentip)                                                               | Spécifie si l’info-bulle du stylet est arrondie ou rectangulaire.<br/>                                                                                                                                                                                                                         |
| [**InkPersistenceCompressionMode**](/windows/desktop/api/msinkaut/ne-msinkaut-inkpersistencecompressionmode)                       | Spécifie les modes de compression utilisés pour enregistrer l’objet [**InkDisp**](inkdisp-class.md) dans un format sérialisé.<br/>                                                                                                                                                      |
| [**InkPersistenceFormat**](/windows/desktop/api/msinkaut/ne-msinkaut-inkpersistenceformat)                                         | Spécifie la manière dont l’encre est rendue persistante.<br/>                                                                                                                                                                                                                                                |
| [**InkRasterOperation**](/windows/desktop/api/msinkaut/ne-msinkaut-inkrasteroperation)                                             | Définit des valeurs pour effectuer des opérations raster sur de l’encre dessinée.<br/>                                                                                                                                                                                                                  |

| [**InkRecognitionModes**](/windows/desktop/api/msinkaut/ne-msinkaut-inkrecognitionmodes)                                           | Spécifie la manière dont le recognzier interprète l’encre et détermine la chaîne de résultat.<br/>                                                                                                                                                                                              |
| [**InkRecognitionStatus**](/windows/desktop/api/msinkaut/ne-msinkaut-inkrecognitionstatus)                                         | Spécifie si une erreur s’est produite lors de la reconnaissance et, le cas échéant, quelle erreur s’est produite.<br/>                                                                                                                                                                                       |
| [**InkRecognizerCapabilities**](/windows/desktop/api/msinkaut/ne-msinkaut-inkrecognizercapabilities)                               | Spécifie les attributs d’un module de reconnaissance. Vous utilisez également cette énumération pour déterminer les attributs à utiliser lors de la recherche d’un module de reconnaissance installé.<br/>                                                                                                                      |
| [**InkRecognizerCharacterAutoCompletionMode**](/windows/desktop/api/msinkaut/ne-msinkaut-inkrecognizercharacterautocompletionmode) | Spécifie les types de modes de saisie de caractères.<br/>                                                                                                                                                                                                                                      |
| [**InkShiftKeyModifierFlags**](/windows/desktop/api/msinkaut/ne-msinkaut-inkshiftkeymodifierflags)                                 | Spécifie la touche de modification qui a été enfoncée.<br/>                                                                                                                                                                                                                                      |
| [**InkSystemGesture**](/windows/desktop/api/msinkaut/ne-msinkaut-inksystemgesture)                                                 | Spécifie l’intérêt d’un ensemble de mouvements spécifiques au système d’exploitation.<br/>                                                                                                                                                                                                         |
| [**ItemSelectionConstants**](/windows/win32/api/msinkaut/ne-msinkaut-inkselectionconstants)                                     | Spécifie si le premier élément ou tous les éléments d’un groupe de points ou de valeurs de paquets sont utilisés.<br/>                                                                                                                                                                        |
| [**PanelType**](/windows/win32/api/peninputpanel/ne-peninputpanel-paneltype)                                                               | Définit le type d’entrée actuellement disponible dans l’objet [**PenInputPanel**](peninputpanel-class.md) .<br/>                                                                                                                                                                      |
| [**SelectionHitResult**](/windows/desktop/api/msinkaut/ne-msinkaut-selectionhitresult)                                             | Spécifie la partie d’une sélection, le cas échéant, qui a été atteinte pendant un test de positionnement.<br/>                                                                                                                                                                                                        |
| [**TabletHardwareCapabilities**](/windows/desktop/api/msinkaut/ne-msinkaut-tablethardwarecapabilities)                             | Spécifie les fonctionnalités matérielles du Tablet PC.<br/>                                                                                                                                                                                                                          |
| [**TabletPropertyMetricUnit**](/windows/desktop/api/msinkaut/ne-msinkaut-tabletpropertymetricunit)                                 | Indique l’unité de mesure d’une propriété.<br/>                                                                                                                                                                                                                               |
