# <a name="ink-analysis-events"></a>Événements d’analyse de l’encre

Cette section contient des informations sur les événements utilisés dans l’analyse de l’encre.

## <a name="event-sources"></a>Sources de l’événement



| Source de l’événement                                            | Description                                                                                                         |
|---------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------|
| [\_IAnalysisEvents](-ianalysisevents.md)               | Spécifie les événements associés aux étapes d’analyse d’un objet [**IInkAnalyzer**](iinkanalyzer.md) .<br/>   |
| [**\_IAnalysisProxyEvents**](-ianalysisproxyevents.md) | Spécifie les événements associés aux étapes du proxy de données d’un objet [**IInkAnalyzer**](iinkanalyzer.md) .<br/> |
