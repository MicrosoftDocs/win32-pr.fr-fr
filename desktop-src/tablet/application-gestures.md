# <a name="application-gestures"></a>Mouvements d’application

Les gestes d’application sont des gestes que vous choisissez de prendre en charge par votre application. Ils sont les plus courants dans les applications conçues spécifiquement pour fonctionner avec un stylet de tablette. Une application héritée peut utiliser des gestes, mais les modèles de sélection existants au sein de l’application peuvent compliquer la prise en charge des gestes. Seuls les mouvements pris en charge dans le panneau de saisie Tablet PC sont disponibles pour les applications héritées.

Les sections suivantes décrivent les gestes d’application et leur utilisation.

-   [Utilisation du module de reconnaissance de mouvement Microsoft uniquement](using-the-microsoft-gesture-recognizer-only.md)
-   [Utilisation d’un module de reconnaissance de mouvement personnalisé](using-a-custom-gesture-recognizer.md)
-   [Conflits potentiels avec les gestes d’application](potential-conflicts-with-application-gestures.md)
-   [Documentation des gestes utilisés par vos applications](documenting-the-gestures-used-by-your-applications.md)
-   [Glyphes non implémentés](unimplemented-glyphs.md)
