# <a name="recognizer-enumerations"></a>Énumérations des modules de reconnaissance

Cette section décrit les énumérations de module de reconnaissance.

## <a name="in-this-section"></a>Dans cette section



| Énumération                                   | Description                                                                                |
|-----------------------------------------------|--------------------------------------------------------------------------------------------|
| [**ALT- \_ arrêts**](/windows/win32/api/rectypes/ne-rectypes-alt_breaks)             | Spécifie comment créer des remplacements à partir d’une chaîne de résultat optimale.<br/>                   |
| [**niveau de confiance \_**](/windows/win32/api/rectypes/ne-rectypes-confidence_level) | Indique le niveau de confiance du module de reconnaissance dans le résultat de la reconnaissance.<br/> |
| [**\_métriques de ligne**](/windows/win32/api/rectypes/ne-rectypes-line_metrics)         | Représente les lignes trouvées dans un segment.<br/>                                        |
| [**unités de propriété \_**](/windows/desktop/api/tpcshrd/ne-tpcshrd-property_units)     | Spécifie l’unité de mesure d’une propriété.<br/>                                |
