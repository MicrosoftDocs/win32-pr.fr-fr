# <a name="setting-context-for-windowless-controls"></a>Définition du contexte pour les contrôles sans fenêtre

Les contrôles sans fenêtre peuvent appeler la fonction [**SetInputScope**](/windows/win32/api/inputscope/nf-inputscope-setinputscope) chaque fois que le focus est déplacé parmi les éléments du contrôle sans fenêtre ; Toutefois, cela n’est pas recommandé pour des raisons de performances.
