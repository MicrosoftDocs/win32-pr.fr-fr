# <a name="exposing-the-contents-of-a-window"></a>Exposer le contenu d’une fenêtre

Une application doit exposer le contenu de classes de fenêtres personnalisées qui utilisent Microsoft Active Accessibility. La fenêtre entière doit être traitée comme un contrôle personnalisé, qui est un conteneur pour d’autres contrôles standard, personnalisés ou personnalisés.

Pour plus d’informations sur l’exposition du contenu des fenêtres personnalisées à l’aide de Active Accessibility, consultez la section [accessibilité](/windows/desktop/accessibility) .
