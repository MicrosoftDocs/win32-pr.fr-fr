# <a name="using-microsoft-active-accessibility"></a>Utilisation de Microsoft Active Accessibility

Le moteur de reconnaissance vocale sur le Tablet PC utilise Active Accessibility pour récupérer des informations directement à partir de l’objet et recevoir une notification lorsque des objets sont modifiés. Cette interface est prise en charge par les contrôles Microsoft Windows standard et peut également être utilisée par les contrôles personnalisés et les classes de fenêtres personnalisées.
