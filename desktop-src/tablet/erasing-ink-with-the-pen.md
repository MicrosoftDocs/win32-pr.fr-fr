# <a name="erasing-ink-with-the-pen"></a>Effacement de l’encre à l’aide du stylet

Outre l’activation de l’écriture manuscrite et du dessin, vous pouvez créer votre application pour prendre en charge l’utilisation du stylet pour effacer l’encre. Pour ce faire, utilisez l’objet [**InkOverlay**](inkoverlay-class.md) .

Les sections suivantes décrivent l’effacement de l’encre avec le stylet du Tablet PC.

-   [Effacement à l’aide de l’objet InkOverlay](erasing-by-using-the-inkoverlay-object.md)
-   [Effacement à l’aide du stylet](erasing-by-using-the-pen.md)
