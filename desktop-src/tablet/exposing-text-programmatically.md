# <a name="exposing-text-programmatically"></a>Exposer du texte par programmation

L’exposition d’un texte par programme offre un avantage considérable pour les aides à l’accessibilité, en fournissant une plus grande fiabilité et, dans certains cas, une interprétation plus riche des informations.

Il existe plusieurs méthodes de base pour exposer des objets afin d’exposer du texte par programmation. Toutefois, aucune méthode n’est efficace avec toutes les aides à l’accessibilité. il est donc également nécessaire de dessiner du texte à l’écran pour qu’il puisse être reconnu. Pour plus d’informations sur l’exposition de grandes quantités de texte, consultez [utilisation de Active Accessibility pour exposer du texte](using-active-accessibility-to-expose-text.md).
