# <a name="about-language-models"></a>À propos des modèles de langage

Les reconnaissance de l’écriture manuscrite utilisent un ensemble défini de règles pour une langue donnée comme contexte principal pour la reconnaissance des mots. Ces règles sont appelées modèles de langage. Le modèle de langage est implémenté par le module de reconnaissance et ne peut pas être modifié directement par un développeur à l’aide du kit de développement logiciel (SDK) de Microsoft Windows XP Tablet PC Edition. Toutefois, il existe des moyens d’affiner le contexte pour des champs spécifiques dans une application. Ces techniques sont les suivantes :

-   Étendues d’entrée
-   Factoids
-   Dictionnaires
-   Listes d’expressions
