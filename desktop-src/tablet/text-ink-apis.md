# <a name="text-ink-reference"></a>Référence de l’encre du texte

En plus de prendre en charge les interfaces OLE standard, l’objet Text Ink (tInk) prend également en charge [**IInkLineInfo**](/windows/desktop/api/msinkaut/nn-msinkaut-iinklineinfo).

Pour plus d’informations sur la spécification des propriétés d’affichage d’un objet Ink Text, consultez structure [**INKMETRIC**](/windows/win32/api/msinkaut/ns-msinkaut-inkmetric) .
