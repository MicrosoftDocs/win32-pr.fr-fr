# <a name="text-input-panel-enumerations"></a>Énumérations du panneau de saisie de texte

Cette section contient les énumérations utilisées par l' [**interface ITextInputPanel**](/windows/desktop/api/peninputpanel/nn-peninputpanel-itextinputpanel).

## <a name="in-this-section"></a>Dans cette section



| Énumération                                                  | Description                                                                                                                               |
|--------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------|
| [**Énumération CorrectionMode**](/windows/win32/api/peninputpanel/ne-peninputpanel-correctionmode)         | Définit des valeurs qui représentent les modes de correction du panneau de saisie Tablet PC.<br/>                                               |
| [**Énumération CorrectionPosition**](/windows/win32/api/peninputpanel/ne-peninputpanel-correctionposition) | Définit des valeurs qui représentent les positions de correction de la publication du panneau de saisie.<br/>                                      |
| [**EventMask**](/windows/win32/api/peninputpanel/ne-peninputpanel-eventmask)                               | Spécifie les événements pour lesquels vous pouvez définir une attention.                                                                                      |
| [**Énumération InPlaceDirection**](/windows/win32/api/peninputpanel/ne-peninputpanel-inplacedirection)     | Définit des valeurs qui représentent les positions où le panneau de saisie peut être placé, par rapport au point d’insertion de texte actuel.<br/> |
| [**Énumération InPlaceState**](/windows/win32/api/peninputpanel/ne-peninputpanel-inplacestate)             | Définit des valeurs qui représentent la manière dont le panneau de saisie s’affiche à l’écran.<br/>                                                       |
| [**Énumération InteractionMode**](/windows/win32/api/peninputpanel/ne-peninputpanel-interactionmode)       | Définit des valeurs qui déterminent la position du panneau de saisie.<br/>                                                                |
| [**Énumération PanelInputArea**](/windows/win32/api/peninputpanel/ne-peninputpanel-panelinputarea)         | Définit des valeurs qui représentent les zones d’entrée du panneau de saisie.<br/>                                                              |
