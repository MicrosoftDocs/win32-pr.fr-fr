# <a name="adding-ink-controls-to-a-project"></a>Ajout de contrôles Ink à un Project

pour ajouter un contrôle [InkEdit](inkedit-control-reference.md) ou [InkPicture](inkpicture-control-reference.md) à un projet microsoft Visual Basic .net ou microsoft Visual C \# :

1.  Cliquez avec le bouton droit dans la boîte à outils, puis cliquez sur **personnaliser la boîte à outils**.
2.  dans la boîte de dialogue personnaliser la boîte **à outils** , cliquez sur l’onglet **composants de .NET Framework** .
3.  Dans la zone de liste, sélectionnez les contrôles InkEdit et InkPicture, puis cliquez sur **OK**. Les contrôles sont ajoutés à la boîte à outils sous l’onglet **composants** .
4.  Sélectionnez le contrôle souhaité dans la boîte à outils, puis faites glisser le contrôle à l’emplacement du formulaire où vous souhaitez qu’il apparaisse.

> [!Note]  
> La bibliothèque gérée Tablet PC peut prendre en charge des contrôles Web dans des environnements de confiance partielle, à savoir l’exécution d’assemblys managés à partir de pages Web. toutefois, les contrôles de ActiveX avec écriture manuscrite sur les pages Web ne sont pas pris en charge.
