# <a name="inkobject-element"></a>Élément InkObject

Contient les données binaires encodées en base64 de l’objet Ink d’un [**InkWord**](inkword-element.md) ou d’un [**dessin**](drawing-element.md).

## <a name="definition"></a>Définition

``` syntax
<xs:element name="InkObject" type="xs:base64Binary" />
```

## <a name="parent-elements"></a>Éléments parents

[**InkWord**](inkword-element.md)

[**Dessin**](drawing-element.md)

## <a name="child-elements"></a>Éléments enfants

Aucun.

## <a name="attributes"></a>Attributs

Aucun.

## <a name="element-information"></a>Informations sur les éléments



|  Élément     | Valeur                                                     |
|--------------|--------------------------------------------|
| Type d'élément | **xs:base64Binary**                        |
| Espace de noms    | urn : schemas-microsoft-com : TabletPC : RichInk |
| Nom du schéma  | Lecteur de journal                             |
