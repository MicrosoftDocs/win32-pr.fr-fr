# <a name="getting-started-tablet-pc"></a>Prise en main (Tablet PC)

le Tablet pc est un ordinateur personnel entièrement fonctionnel alimenté par Microsoft Windows et conçu pour les applications prenant en charge l’écriture manuscrite, le stylet et la reconnaissance vocale. La combinaison des logiciels et du matériel d’un Tablet PC permet aux utilisateurs de créer une expérience informatique riche, interactive et productive pour les utilisateurs.

La plateforme Tablet PC permet l’entrée et la sortie de données d’écriture manuscrite et de synthèse vocale sur un Tablet PC, ainsi que l’échange de ces données avec d’autres ordinateurs. les composants de plateforme Tablet PC installés par le kit de développement logiciel (SDK) Windows 7 vous permettent de créer des applications compatibles avec l’encre et le stylet pour Tablet PC.
