# <a name="math-input-classes-and-interfaces"></a>Classes et interfaces d’entrée Math

Cette section contient les classes et les interfaces utilisées pour le contrôle d’entrée mathématique.

## <a name="in-this-section"></a>Dans cette section



| Interface                                                     | Description                                                       |
|---------------------------------------------------------------|-------------------------------------------------------------------|
| [**\_IMathInputControlEvents**](/windows/win32/api/micaut/nn-micaut-_imathinputcontrolevents) | Expose les gestionnaires d’événements de contrôle d’entrée mathématique.                    |
| [**IMathInputControl**](/windows/desktop/api/micaut/nn-micaut-imathinputcontrol)                | Expose des méthodes qui transforment l’entrée manuscrite en sortie mathématique interprétée. |
