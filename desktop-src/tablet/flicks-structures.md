# <a name="flicks-structures"></a>Structures de raccourcis

Cette section contient les structures utilisées dans le [**\_ message du \_ raccourci WM**](wm-tablet-flick-message.md)de la tablette.



| Structure                           | Description                                                                                 |
|-------------------------------------|---------------------------------------------------------------------------------------------|
| [**POINT de raccourci \_**](/windows/desktop/api/tabflicks/ns-tabflicks-flick_point) | Contient les données membres X et Y décrivant le point où le raccourci s’est produit.<br/> |
| [**données de raccourci \_**](/windows/desktop/api/tabflicks/ns-tabflicks-flick_data)   | Contient des informations détaillées sur le mouvement de raccourci et son environnement.<br/>       |
