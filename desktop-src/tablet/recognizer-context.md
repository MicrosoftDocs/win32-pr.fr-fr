# <a name="recognizer-context"></a>Contexte du module de reconnaissance

L’objet [**RecognizerContext**](inkrecognizercontext-class.md) définit les paramètres d’un contexte de reconnaissance unique. Elle reçoit l’encre à reconnaître, définit les contraintes sur l’entrée d’encre et la sortie de reconnaissance, et peut être interrogée pour obtenir les résultats de la reconnaissance.

Pour plus d’informations sur les données d’entrée manuscrite, consultez [données manuscrites](ink-data.md). Pour plus d’informations sur la collecte d’encres, consultez [collection d’encres](ink-collection.md).

Les contraintes que vous pouvez définir sur l’entrée d’encre et la sortie de reconnaissance de l’objet [**RecognizerContext**](inkrecognizercontext-class.md) sont les suivantes :

-   Langue qui est reconnue ou retournée
-   Dictionnaire utilisé
-   Factoids

Pour plus d’informations sur les contraintes que vous pouvez définir sur l’objet [**RecognizerContext**](inkrecognizercontext-class.md) , consultez [Dictionaries et Factoids](dictionaries-and-factoids.md).
