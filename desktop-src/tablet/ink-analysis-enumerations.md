# <a name="ink-analysis-enumerations"></a>Énumérations d’analyse de l’encre

Cette section contient des informations sur les énumérations utilisées dans l’analyse de l’encre.

## <a name="enumerations"></a>Énumérations



| Énumération                                              | Description                                                                                                                               |
|----------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------|
| [**AnalysisModes**](analysismodes.md)                   | Spécifie la manière dont le [**IInkAnalyzer**](iinkanalyzer.md) effectue l’analyse de l’encre.<br/>                                                  |
| [**AnalysisWarningCode**](analysiswarningcode.md)       | Spécifie l’ensemble des avertissements disponibles qui peuvent se produire lors de l’analyse de l’encre.<br/>                                                    |
| [**ConfirmationType**](confirmationtype.md)             | Spécifie le type de confirmation qui peut se produire sur un objet [**IContextNode**](icontextnode.md) .<br/>                           |
| [**ContextLinkDirection**](contextlinkdirection.md)     | Spécifie les types d’objets [**IContextLink**](icontextlink.md) .<br/>                                                           |
| [**RecognitionConfidence**](recognitionconfidence.md)   | Spécifie le niveau de confiance que le [**IInkAnalyzer**](iinkanalyzer.md) a dans la précision du résultat de la reconnaissance.<br/> |
| [**RecognizerCapabilities**](recognizercapabilities.md) | Spécifie les attributs d’un [**IInkAnalysisRecognizer**](iinkanalysisrecognizer.md).<br/>                                       |
| [**StrokeType**](stroketype.md)                         | Spécifie la manière dont le [**IInkAnalyzer**](iinkanalyzer.md) doit classer un trait pendant l’analyse.<br/>                               |
