# <a name="recognizer-reference"></a>Référence du module de reconnaissance

Le module de reconnaissance gère les éléments suivants :

-   Déterminez les identifiants disponibles sur l’ordinateur.
-   Identifiez le module de reconnaissance à utiliser.
-   Créez un contexte de module de reconnaissance.
-   Récupérez les résultats du module de reconnaissance et les alternatives.

## <a name="in-this-section"></a>Dans cette section



| Rubrique                                                      | Description                                                                                                                                                           |
|------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Handle HRECOALT](hrecoalt-handle.md)                     | Récupère les valeurs de propriété et de chaîne alternatives, telles que le niveau de confiance et les métriques.<br/>                                                                  |
| [Handle HRECOCONTEXT](hrecocontext-handle.md)             | Ajoute de l’encre au contexte, effectue la reconnaissance de l’encre et récupère le résultat de la reconnaissance et les autres.<br/>                                                    |
| [Handle HRECOGNIZER](hrecognizer-handle.md)               | Crée un contexte de module de reconnaissance, récupère ses attributs et ses propriétés et détermine les propriétés de paquet dont le module de reconnaissance a besoin pour effectuer la reconnaissance.<br/> |
| [Handle HRECOWORDLIST](hrecowordlist-handle.md)           | Gère une liste de mots que vous attachez à un contexte de module de reconnaissance. Cette liste améliore les résultats de la reconnaissance.<br/>                                                   |
| [Énumérations des modules de reconnaissance](recognizer-api-enumerations.md) | Décrit les types d’énumération de module de reconnaissance.<br/>                                                                                                                |
| [Structures de reconnaissance](recognizer-api-structures.md)     | Décrit les structures de module de reconnaissance.<br/>                                                                                                                       |
| [GUID de reconnaissance](recognizer-guids.md)                   | Spécifie le type de propriété pour les propriétés de paquet et les propriétés de module de reconnaissance.<br/>                                                                            |
