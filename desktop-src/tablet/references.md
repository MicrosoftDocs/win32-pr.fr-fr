# <a name="references-tablet-pc"></a>Références (Tablet PC)

Pour plus d’informations sur l’activation de la reconnaissance vocale dans votre application, consultez le kit de développement logiciel (SDK) Microsoft Speech, qui se trouve dans [Microsoft Speech SDK](https://www.microsoft.com/speech/download/sdk51/). Ce site contient également une vue d’ensemble des instructions de conception d’applications et d’API Microsoft Speech.

pour plus d’informations sur les instructions de Active Accessibility Microsoft pour les applications basées sur Windows, consultez la section [accessibilité](../accessibility/accessibility.md) .
