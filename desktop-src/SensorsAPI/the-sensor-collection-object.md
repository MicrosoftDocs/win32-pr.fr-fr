# <a name="the-sensor-collection-object"></a>Objet de collection Sensor

L’objet collection de capteurs contient un groupe d’objets Sensor.

Il peut arriver que vous souhaitiez travailler avec des capteurs en tant que groupe. Par exemple, vous souhaiterez peut-être utiliser le gestionnaire de capteurs pour découvrir tous les capteurs de mouvement disponibles. L’objet de collection de capteurs fournit des méthodes pour travailler avec un groupe d’objets de capteur. Vous pouvez utiliser l’interface [**ISensorCollection**](/windows/desktop/api/sensorsapi/nn-sensorsapi-isensorcollection) pour appeler des méthodes qui dénombrent, ajoutent, suppriment et récupèrent des capteurs dans la collection.
