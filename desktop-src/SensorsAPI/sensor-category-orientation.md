# <a name="sensor_category_orientation"></a>\_orientation de catégorie de capteur \_

La catégorie d’orientation de catégorie de capteur \_ contient des \_ capteurs qui fournissent des informations sur l’orientation physique. Les compositions fournissent une orientation de navigation, comme celles basées sur le nord magnétique. Inclinometers mesure la pente ou l’élévation. Les capteurs de distance mesurent la proximité d’un objet au capteur.
