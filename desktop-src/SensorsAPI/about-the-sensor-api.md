# <a name="about-the-sensor-api"></a>À propos de l’API de capteur

L’API de capteur fournit ses fonctionnalités via un ensemble d’interfaces COM. L’API de capteur expose certaines de ces interfaces, ce qui signifie que vous pouvez appeler des méthodes sur celles-ci. D’autres interfaces sont définies par l’API afin que vous puissiez implémenter les méthodes dont l’API de capteur a besoin pour fournir des notifications d’événements à votre programme.
