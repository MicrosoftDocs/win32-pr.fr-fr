# <a name="enumeration-types"></a>Types d’énumération

L’API de capteur définit les types d’énumération suivants.



| Énumération                                                                | Description                                      |
|----------------------------------------------------------------------------|--------------------------------------------------|
| [**SensorConnectionType**](/windows/win32/api/sensorsapi/ne-sensorsapi-sensorconnectiontype) | Définit les types de connexions de périphérique de capteur.      |
| [**SensorState**](/windows/win32/api/sensorsapi/ne-sensorsapi-sensorstate)                   | Définit les États opérationnels possibles pour les capteurs. |
