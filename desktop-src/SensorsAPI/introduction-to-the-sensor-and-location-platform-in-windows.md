## <a name="about-sensor-devices"></a><span data-ttu-id="3140f-108">À propos des appareils capteur</span><span class="sxs-lookup"><span data-stu-id="3140f-108">About Sensor Devices</span></span>

<span data-ttu-id="3140f-109">Les capteurs présentent de nombreuses configurations et, d’un point de vue spécifique, presque tout ce qui fournit des données sur les phénomènes physiques peut être appelé capteur.</span><span class="sxs-lookup"><span data-stu-id="3140f-109">Sensors come in many configurations and, from a certain perspective, almost anything that provides data about physical phenomena can be called a sensor.</span></span> <span data-ttu-id="3140f-110">Même si nous considérons généralement les capteurs comme des périphériques matériels, les capteurs logiques peuvent également fournir des informations par le biais de l’émulation de la fonctionnalité de capteur dans le logiciel ou le microprogramme.</span><span class="sxs-lookup"><span data-stu-id="3140f-110">Although we typically think of sensors as hardware devices, logical sensors can also provide information through emulation of sensor functionality in software or firmware.</span></span> <span data-ttu-id="3140f-111">En outre, un seul périphérique matériel peut contenir plusieurs capteurs.</span><span class="sxs-lookup"><span data-stu-id="3140f-111">Also, a single hardware device can contain multiple sensors.</span></span>

<span data-ttu-id="3140f-112">La plateforme de localisation et de capteur Windows organise les capteurs en *catégories*, qui représentent des classes étendues de périphériques de capteur et des *types* qui représentent des genres spécifiques de capteurs.</span><span class="sxs-lookup"><span data-stu-id="3140f-112">The Windows Sensor and Location platform organizes sensors into *categories*, which represent broad classes of sensor devices, and *types*, which represent specific kinds of sensors.</span></span> <span data-ttu-id="3140f-113">Par exemple, un capteur dans un contrôleur de jeu vidéo qui détecte la position et le mouvement de la main d’un joueur (peut-être pour un jeu de bowling vidéo) est catégorisé comme capteur d’orientation, mais son type est un accéléromètre 3D.</span><span class="sxs-lookup"><span data-stu-id="3140f-113">For example, a sensor in a video game controller that detects the position and movement of a player's hand (perhaps for a video bowling game) would be categorized as an Orientation sensor, but its type would be 3-D Accelerometer.</span></span> <span data-ttu-id="3140f-114">Dans le code, Windows représente des catégories et des types à l’aide d’identificateurs globaux uniques (GUID), dont la plupart sont prédéfinis.</span><span class="sxs-lookup"><span data-stu-id="3140f-114">In code, Windows represents categories and types by using globally unique identifiers (GUIDs), many of which are predefined.</span></span> <span data-ttu-id="3140f-115">Les fabricants de périphériques peuvent créer des catégories et des types en définissant et en publiant de nouveaux GUID, le cas échéant.</span><span class="sxs-lookup"><span data-stu-id="3140f-115">Device manufacturers can create new categories and types by defining and publishing new GUIDs, when it is required.</span></span>

<span data-ttu-id="3140f-116">Les périphériques d’emplacement constituent une catégorie particulièrement intéressante.</span><span class="sxs-lookup"><span data-stu-id="3140f-116">Location devices make up one especially interesting category.</span></span> <span data-ttu-id="3140f-117">À l’heure actuelle, la plupart des gens sont familiarisés avec les systèmes de positionnement global (GPS).</span><span class="sxs-lookup"><span data-stu-id="3140f-117">By now, most people are familiar with global positioning systems (GPS).</span></span> <span data-ttu-id="3140f-118">Dans Windows, un capteur GPS fait partie de la catégorie d’emplacement.</span><span class="sxs-lookup"><span data-stu-id="3140f-118">In Windows, a GPS sensor is part of the Location category.</span></span> <span data-ttu-id="3140f-119">La catégorie d’emplacement peut inclure d’autres types de capteur.</span><span class="sxs-lookup"><span data-stu-id="3140f-119">The Location category could include other sensor types.</span></span> <span data-ttu-id="3140f-120">Certains de ces types de capteurs sont basés sur des logiciels, tels qu’un programme de résolution d’adresses IP qui fournit des informations d’emplacement basées sur une adresse Internet, un triangulateur de tour mobile qui détermine l’emplacement en fonction des tours proches ou un fournisseur d’emplacement réseau Wi-Fi qui lit les informations d’emplacement à partir du concentrateur de réseau sans fil connecté.</span><span class="sxs-lookup"><span data-stu-id="3140f-120">Some of these sensor types are software based, such as an IP resolver that provides location information based on an Internet address, a mobile phone tower triangulator that determines location based on nearby towers, or a Wi-Fi network location provider that reads location information from the connected wireless network hub.</span></span>
