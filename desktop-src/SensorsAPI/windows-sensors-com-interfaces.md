# <a name="com-interfaces-sensor-api"></a>Interfaces COM (API de capteur)

L’API de capteur expose les interfaces COM suivantes.



| Interface                                              | Description                                                                                                      |
|--------------------------------------------------------|------------------------------------------------------------------------------------------------------------------|
| [**ILocationPermissions**](/windows/desktop/api/sensorsapi/nn-sensorsapi-ilocationpermissions)   | Fournit l’état du paramètre système qui permet aux utilisateurs de modifier les paramètres d’emplacement.                         |
| [**ILogicalSensorManager**](/previous-versions/windows/desktop/legacy/dd318934(v=vs.85)) | Fournit des méthodes utilisées par les auteurs de capteurs logiques pour gérer les connexions de capteur logique.                            |
| [**ISensor**](/windows/desktop/api/sensorsapi/nn-sensorsapi-isensor)                             | Représente un capteur.                                                                                             |
| [**ISensorCollection**](/windows/desktop/api/sensorsapi/nn-sensorsapi-isensorcollection)         | Représente une collection de capteurs.                                                                              |
| [**ISensorDataReport**](/windows/desktop/api/sensorsapi/nn-sensorsapi-isensordatareport)         | Représente un rapport de données de capteur.                                                                                 |
| [**ISensorEvents**](/windows/desktop/api/sensorsapi/nn-sensorsapi-isensorevents)                 | Interface de rappel pour la réception des événements de capteur.                                                              |
| [**ISensorManager**](/windows/desktop/api/sensorsapi/nn-sensorsapi-isensormanager)               | Fournit des méthodes pour découvrir et récupérer les capteurs disponibles et une méthode pour demander des événements du gestionnaire de capteurs. |
| [**ISensorManagerEvents**](/windows/desktop/api/sensorsapi/nn-sensorsapi-isensormanagerevents)   | Interface de rappel pour la réception des événements du gestionnaire de capteurs.                                                      |
