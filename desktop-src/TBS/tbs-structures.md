# <a name="tbs-structures"></a>Structures TBS

TBS prend en charge les structures suivantes.

## <a name="in-this-section"></a>Contenu de cette section



| Rubrique                                                           | Description                                                                                                                                          |
|-----------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**\_paramètres de contexte tbs \_**](/windows/win32/api/tbs/ns-tbs-tbs_context_params)<br/>   | Spécifie la version de l’implémentation du contexte TBS.<br/>                                                                                  |
| [**\_PARAMS2 de contexte tbs \_**](/windows/win32/api/tbs/ns-tbs-tbs_context_params2)<br/> | Spécifie la version de l’implémentation du contexte TBS. Vous devez utiliser cette structure si votre application fonctionne avec les deux versions de TPM.<br/> |
| [**\_informations sur l’appareil TPM \_**](/windows/desktop/api/Tbs/ns-tbs-tpm_device_info)<br/>         | Fournit des informations sur la version du module de plateforme sécurisée.<br/>                                                                                        |
