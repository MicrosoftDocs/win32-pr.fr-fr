# <a name="the-best-of-windows-and-the-web"></a>Le meilleur de Windows et du Web

Avec Windows 7, il est facile de se connecter et de rester connecté. La mise en réseau Windows offre aux développeurs des options pour une meilleure mise en cache et le partage des données afin d’améliorer les performances du réseau. Les diagnostics réseau dans Windows 7 fournissent aux développeurs des informations pertinentes pour la surveillance des problèmes réseau. Que l’objectif soit de connecter des appareils, de se connecter à un réseau ou simplement de comprendre l’état de votre connexion, Windows 7 offre une plateforme améliorée.
