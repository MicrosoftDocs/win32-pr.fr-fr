# <a name="richer-application-experiences"></a>Expériences d’applications plus riches

Windows 7 permet aux développeurs de créer des applications distinctes et intuitives qui améliorent considérablement la détectabilité, la convivialité et la jouissance. les nouvelles méthodes de desktop integration placent les fonctionnalités d’application directement à l’utilisateur, et Windows Explorer et les *bibliothèques* facilitent l’accès aux informations de valeur élevée. le contrôle de ruban Windows et l’infrastructure d’Animation Windows facilitent la création d’interfaces utilisateur interactives et attrayantes. Les nouvelles API tactiles permettent d’effectuer des interactions naturelles par le biais de l’interaction tactile multipoint et de la panoramisation, et les API de manipulation et d’inertie permettent des effets visuels impressionnants.

Les avancées rapides en matière de technologies matérielles et logicielles peuvent également conduire à des expériences utilisateur plus fidèles. Windows 7 apporte ces avancées sous le contrôle du développeur avec des api nouvelles et flexibles qui tirent pleinement parti de la technologie, tout en facilitant le développement d’applications attrayantes.
