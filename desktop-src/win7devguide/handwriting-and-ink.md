# <a name="handwriting-and-ink"></a>Écriture manuscrite et encre

Avec la prolifération des *Tablet PC* sur le marché, les fonctionnalités des *tablettes* font partie de l’informatique standard. Dans Windows 7, les fonctions tactiles et d’écriture sont des expériences utilisateur de première classe. Windows 7 améliore l’expérience du stylet en fournissant une précision et une vitesse supérieures. La saisie de l’écriture manuscrite est améliorée et d’autres langues sont prises en charge. Le panneau de saisie de texte fournit du texte prédictif pour une plus grande vitesse d’entrée et de correction. La précision de l’écriture manuscrite est améliorée grâce à la personnalisation dans tous les langages, les dictionnaires personnalisés et les innovations en matière de reconnaissance des langues asiatiques. Le modèle d’interaction amélioré offre une meilleure expérience de lecture sur les petits écrans haute résolution courants des ordinateurs portables. (Consultez [programmation du panneau de saisie de texte](../tablet/programming-the-text-input-panel.md).)

![panneau de saisie Tablet PC](images/windows7-4.jpg)

Le panneau de saisie de texte propose une correction de texte simple

## <a name="math-recognition"></a>Reconnaissance mathématique

La nouvelle fonctionnalité de reconnaissance mathématique permet aux utilisateurs d’entrer des mathématiques dans des applications par le biais de l’écriture manuscrite, le moyen le plus naturel et le plus efficace d’entrer des expressions mathématiques. La fonctionnalité est fournie par deux composants de l’interface utilisateur. Le panneau de saisie mathématique est un accessoire Windows autonome qui fonctionne avec toutes les applications prenant en charge les mathématiques. Le contrôle d’entrée mathématique est intégré aux applications via son API.

Les composants de l’interface utilisateur sous-jacents sont le module de reconnaissance mathématique. Ce moteur reconnaît l’expression mathématique manuscrite et convertit le résultat au format *MathML* pour les applications à utiliser. L’expérience de correction a été améliorée pour aider les utilisateurs à effectuer des corrections plus rapidement. (Consultez [programmation du contrôle d’entrée mathématique](../tablet/programming-the-math-input-control.md).)

![panneau de saisie mathématique](images/windows7-5.jpg)

La reconnaissance mathématique permet aux utilisateurs d’entrer des mathématiques dans des applications par le biais de l’écriture manuscrite

## <a name="handwriting-with-personalized-custom-dictionary"></a>Écriture manuscrite avec dictionnaire personnalisé personnalisé

Pour de nombreux scénarios, une bonne précision de l’écriture manuscrite nécessite un dictionnaire adapté au domaine d’utilisation. Windows 7 introduit des dictionnaires personnalisés, qui permettent une meilleure reconnaissance de l’écriture manuscrite pour les vocabulaires spécialisés. Les développeurs qui écrivent des applications verticales (par exemple, un bloc-notes de prescription médicale) peuvent désormais ajouter des termes spécifiques à leur application, tels que les noms de drogues. (Consultez [améliorations apportées aux développeurs sur Windows Touch et Tablet PC](https://www.microsoft.com/whdc/device/input/Touch_tab_enhance.mspx).)
