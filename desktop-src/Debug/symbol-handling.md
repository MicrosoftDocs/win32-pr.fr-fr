# <a name="symbol-handling"></a>Gestion des symboles

Les fonctions de gestionnaire de symboles offrent aux applications un accès facile et portable aux informations de débogage symboliques d’une image exécutable. Ces fonctions sont utilisées exclusivement pour garantir l’accès aux informations symboliques. Cela est nécessaire, car ces fonctions isolent l’application du format des symboles.

Pour plus d'informations, voir les rubriques suivantes :

-   [Initialisation du gestionnaire de symboles](symbol-handler-initialization.md)
-   [Chemins d'accès aux symboles](symbol-paths.md)
-   [Chargement de symboles](symbol-loading.md)
-   [Chargement de symboles différés](deferred-symbol-loading.md)
-   [Noms de symboles décorés](decorated-symbol-names.md)
-   [Recherche de symboles](finding-symbols.md)
-   [Symboles publics, globaux et locaux](public--global--and-local-symbols.md)
-   [Nettoyage du gestionnaire de symboles](symbol-handler-cleanup.md)
