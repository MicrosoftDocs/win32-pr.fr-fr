# <a name="error-handling-error-handling"></a>Gestion des erreurs (gestion des erreurs)

Les applications bien écrites incluent du code de gestion des erreurs qui leur permet de récupérer correctement des erreurs inattendues. Lorsqu’une erreur se produit, l’application peut avoir besoin de demander l’intervention de l’utilisateur, ou elle peut être en mesure de récupérer seule. Dans les cas extrêmes, l’application peut déconnecter l’utilisateur ou arrêter le système.

-   [À propos de la gestion des erreurs](about-error-handling.md)
-   [Utilisation de la gestion des erreurs](using-error-handling.md)
-   [Référence de gestion des erreurs](error-handling-reference.md)

Pour plus d’informations sur la gestion des exceptions, consultez [gestion structurée des exceptions](structured-exception-handling.md).
