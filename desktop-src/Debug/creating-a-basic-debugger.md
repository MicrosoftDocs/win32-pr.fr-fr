# <a name="creating-a-basic-debugger"></a>Création d’un débogueur de base

Le [débogage d’un processus en cours d’exécution](debugging-a-running-process.md) décrit l’attachement du débogueur à un processus en cours d’exécution. [L’écriture de la boucle principale du débogueur](writing-the-debugger-s-main-loop.md) décrit les fonctions utilisées dans la boucle d’exécution principale du débogueur.

Bien que la plupart des fonctions de débogage soient utilisées pour créer un débogueur, plusieurs fonctions sont conçues pour être utilisées dans le processus en cours de débogage. Pour plus d’informations, consultez [communication avec le débogueur](communicating-with-the-debugger.md).
