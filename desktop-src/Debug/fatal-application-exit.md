# <a name="fatal-application-exit"></a>Sortie d’application irrécupérable

La fonction **FatalAppExit** affiche une boîte de message et met fin à l’application lorsque l’utilisateur ferme la boîte de message. Cette fonction ne doit être utilisée qu’en dernier recours, car elle peut ne pas libérer la mémoire ou les fichiers appartenant à l’application.
