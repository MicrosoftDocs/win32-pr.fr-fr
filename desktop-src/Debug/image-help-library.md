# <a name="image-help-library"></a>Bibliothèque d’aide d’images

Cette vue d’ensemble décrit l’ensemble de fonctions fourni par la DLL ImageHlp. Ces fonctions vous permettent d’utiliser une image PE (Portable Executable).

-   [À propos de ImageHlp](about-imagehlp.md)
-   [Référence ImageHlp](imagehlp-reference.md)

Vous ne pouvez pas redistribuer la DLL ImageHlp incluse dans le système d’exploitation. Un sous-ensemble des fonctions est inclus dans la [bibliothèque d’aide de débogage](debug-help-library.md), qui est restributable.
