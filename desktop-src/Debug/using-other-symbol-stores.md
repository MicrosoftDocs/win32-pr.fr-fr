# <a name="using-other-symbol-stores"></a>Utilisation d’autres magasins de symboles

Il est possible d’écrire votre propre programme de création de magasin de symboles, plutôt que d’utiliser SymStore.

Étant donné que les transactions SymStore sont toutes journalisées dans des fichiers texte au format CSV, vous pouvez tirer parti de tous les fichiers journaux SymStore existants afin de les utiliser dans votre propre programme de base de données.

Si vous envisagez d’utiliser le programme SymSrv fourni avec les outils de débogage pour Windows, il est recommandé d’utiliser également SymStore. Les mises à jour de ces deux programmes sont toujours publiées ensemble, et par conséquent, leurs versions correspondent toujours.
