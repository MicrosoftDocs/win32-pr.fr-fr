# <a name="notifying-the-user"></a>Notification de l’utilisateur

Pour avertir l’utilisateur qu’une erreur s’est produite, de nombreuses applications produisent simplement un son à l’aide de la fonction [**MessageBeep**](/windows/desktop/api/WinUser/nf-winuser-messagebeep) ou la fenêtre clignotent à l’aide de la fonction [**flashwindow**](/windows/desktop/api/Winuser/nf-winuser-flashwindow) ou [**FlashWindowEx**](/windows/desktop/api/Winuser/nf-winuser-flashwindowex) . Une application peut également utiliser ces fonctions pour attirer l’attention sur une erreur, puis afficher une boîte de message ou un message d’erreur contenant des détails sur l’erreur.
