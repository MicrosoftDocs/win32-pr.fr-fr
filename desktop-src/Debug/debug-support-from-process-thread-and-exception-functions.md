# <a name="debug-support-from-process-thread-and-exception-functions"></a>Prise en charge du débogage à partir des fonctions de processus, de thread et d’exception

Certaines fonctions essentielles au débogage sont en fait classées en tant que fonctions de traitement, de thread ou de gestion des exceptions. Pour obtenir une description de la façon de démarrer un processus qui va être débogué, consultez [fonctions de traitement](process-functions-for-debugging.md). Pour plus d’informations sur la façon d’examiner et de manipuler le contexte et l’exécution d’un thread, consultez [thread Functions](thread-functions-for-debugging.md). Pour obtenir une description de la façon dont un débogueur doit gérer les exceptions, consultez [fonctions de gestion des exceptions](exception-handling-functions-for-debugging.md).
