# <a name="user-address-book-attributes"></a>Attributs du carnet d’adresses utilisateur

Les attributs de carnet d’adresses sont utilisés pour fournir une identification et des informations supplémentaires à un utilisateur. Leur contenu est défini par l’utilisateur et l’organisation de l’utilisateur. Ces attributs peuvent être affichés et gérés par le composant logiciel enfichable utilisateurs et ordinateurs Active Directory ou le carnet d’adresses Windows (WAB).



| Attribut                         | Description                                                                                                                                                                                                                                                                                                                                                                                                                                   |
|-----------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **c**                             | Pays/région de l’adresse de l’utilisateur. Le pays ou la région est représenté sous la forme d’un code de pays/région à deux caractères basé sur ISO-3166. Pour les codes valides, consultez [valeurs pour CountryCode](values-for-the-countrycode-and-c-properties.md).<br/>                                                                                                                                                                                        |
| **propriétaire**                            | Pays/région où l’utilisateur se trouve.                                                                                                                                                                                                                                                                                                                                                                                              |
| **Notes**                         | Commentaire. Cette chaîne peut être une chaîne NULL.                                                                                                                                                                                                                                                                                                                                                                                                  |
| **department**                    | Nom du service où travaille l’utilisateur.                                                                                                                                                                                                                                                                                                                                                                                          |
| **description**                   | Description à afficher pour l’utilisateur.                                                                                                                                                                                                                                                                                                                                                                                                      |
| **displayName**                   | Nom affiché dans le carnet d’adresses pour un utilisateur particulier. Il s’agit généralement de la combinaison du prénom, de l’initiale du deuxième prénom et du nom de l’utilisateur.                                                                                                                                                                                                                                                                                        |
| **directReports**                 | Liste des utilisateurs qui sont directement signalés à l’utilisateur. Les utilisateurs répertoriés comme rapports sont ceux dont l’attribut **Manager** est défini sur cet utilisateur. Chaque élément de la liste est une référence liée à l’objet qui représente l’utilisateur. par conséquent, le serveur Active Directory met automatiquement à jour cet attribut lorsque l’attribut **Manager** d’un utilisateur ajoute ou supprime cet utilisateur en tant que responsable. Les éléments sont représentés sous forme de noms uniques. |
| **facsimileTelephoneNumber**      | Numéro de téléphone de l’appareil fax professionnel de l’utilisateur.                                                                                                                                                                                                                                                                                                                                                                                      |
| **givenName**                     | Prénom de l’utilisateur.                                                                                                                                                                                                                                                                                                                                                                                                      |
| **homePhone**                     | Numéro de téléphone principal de l’utilisateur.                                                                                                                                                                                                                                                                                                                                                                                               |
| **initiales**                      | Initiales des parties du nom complet de l’utilisateur. Cela peut être utilisé comme initiale du deuxième prénom dans le carnet d’adresses Windows.                                                                                                                                                                                                                                                                                                                           |
| **ipPhone**                       | Utilisé par la téléphonie.                                                                                                                                                                                                                                                                                                                                                                                                                            |
| **l**                             | La localité, telle que la ville ou la ville, dans l’adresse de l’utilisateur.                                                                                                                                                                                                                                                                                                                                                                                |
| **managedObjects**                | Liste des objets gérés par l’utilisateur. Les objets répertoriés sont ceux dont l’attribut **ManagedBy** est défini sur cet utilisateur. Chaque élément de la liste est une référence liée à l’objet managé ; par conséquent, le serveur Active Directory met automatiquement à jour l’attribut **managedObjects** quand l’attribut **ManagedBy** d’un objet ajoute ou supprime cet utilisateur en tant que responsable. Les éléments sont représentés sous forme de noms uniques.  |
| **gestion**                       | Utilisateur qui est le responsable de l’utilisateur. L’objet utilisateur du gestionnaire contient un attribut **DirectReports** qui contient des références à tous les objets utilisateur dont l’attribut **Manager** est défini sur l’objet utilisateur du gestionnaire.                                                                                                                                                                                                                     |
| **mobile**                        | Numéro de portable principal de l’utilisateur.                                                                                                                                                                                                                                                                                                                                                                                           |
| **otherFacsimileTelephoneNumber** | Liste des numéros de téléphone des autres télécopieurs pour l’utilisateur.                                                                                                                                                                                                                                                                                                                                                                         |
| **otherIpPhone**                  | Utilisé par la téléphonie.                                                                                                                                                                                                                                                                                                                                                                                                                            |
| **otherMobile**                   | Liste de numéros de téléphone cellulaire alternatifs pour l’utilisateur.                                                                                                                                                                                                                                                                                                                                                                                |
| **otherPager**                    | Liste des numéros de téléphone de radiomessagerie supplémentaires pour l’utilisateur.                                                                                                                                                                                                                                                                                                                                                                                   |
| **otherTelephone**                | Liste de numéros de téléphone de bureau de substitution pour l’utilisateur.                                                                                                                                                                                                                                                                                                                                                                                |
| **destinés**                         | Numéro de téléphone principal du récepteur de radiomessagerie pour l’utilisateur.                                                                                                                                                                                                                                                                                                                                                                                              |
| **physicalDeliveryOfficeName**    | Emplacement du bureau de l’utilisateur.                                                                                                                                                                                                                                                                                                                                                                                          |
| **Property**                 | Adresse postale de l’utilisateur.                                                                                                                                                                                                                                                                                                                                                                                                                    |
| **postalCode**                    | Code postal correspondant à l’adresse postale de l’utilisateur. Le code postal est spécifique au pays ou à la région de l’utilisateur. Aux États-Unis, cet attribut contient le code ZIP.                                                                                                                                                                                                                                                               |
| **postOfficeBox**                 | Numéro ou identificateur de la boîte postale de l’utilisateur.                                                                                                                                                                                                                                                                                                                                                                                       |
| **sn**                            | Patronyme de l’utilisateur (nom).                                                                                                                                                                                                                                                                                                                                                                                                |
| **St**                            | État ou province dans l’adresse de l’utilisateur.                                                                                                                                                                                                                                                                                                                                                                                                  |
| **streetAddress**                 | Adresse postale du lieu de travail de l’utilisateur.                                                                                                                                                                                                                                                                                                                                                                                           |
| **telephoneNumber**               | Numéro de téléphone principal du lieu de travail de l’utilisateur.                                                                                                                                                                                                                                                                                                                                                                                 |
| **title**                         | Intitulé du poste de l’utilisateur. Cet attribut est couramment utilisé pour indiquer la fonction formelle, telle que le programmeur senior, plutôt que la classe professionnelle, comme le programmeur. Il n’est généralement pas utilisé pour les titres « suffixe » tels que Esq. ou DDS. Exemples : gestion Director, programmeur II, professeur Associate et responsable du développement.<br/>                                                                                                    |
| **url**                           | Liste des URL des pages Web secondaires de l’utilisateur.                                                                                                                                                                                                                                                                                                                                                                                           |
| **wWWHomePage**                   | URL de la page Web principale de l’utilisateur.                                                                                                                                                                                                                                                                                                                                                                                                       |
