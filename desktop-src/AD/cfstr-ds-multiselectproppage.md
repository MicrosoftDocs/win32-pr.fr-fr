# <a name="cfstr_ds_multiselectproppage"></a>CFSTR \_ DS \_ MULTISELECTPROPPAGE

<dl> <dt>

<span id="CFSTR_DS_MULTISELECTPROPPAGE"></span><span id="cfstr_ds_multiselectproppage"></span>**CFSTR \_ DS \_ MULTISELECTPROPPAGE**
</dt> <dd> <dl> <dt>

"DsAdminMultiSelectClipFormat"
</dt> <dt>



Le format de presse-papiers **CFSTR \_ DS \_ MULTISELECTPROPPAGE** fournit un handle de mémoire globale (**HGLOBAL**) qui contient une chaîne Unicode terminée par le caractère null qui contient un nom unique utilisé pour créer l’objet de notification pour une extension de feuille de propriétés à sélection multiple.

> [!Note]  
> Cette valeur de format de presse-papiers n’est pas définie dans un fichier d’en-tête publié. Pour utiliser cette valeur de format de presse-papiers, vous devez la définir vous-même dans le format exact indiqué.

 


</dt> </dl> </dd> </dl>
