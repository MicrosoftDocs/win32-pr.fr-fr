# <a name="inheritance-ad-ds"></a>Héritage (AD DS)

L' [*héritage*](/previous-versions/windows/desktop/legacy/ms681910(v=vs.85)) permet de propager une entrée du contrôle d’accès donnée à partir du conteneur dans lequel elle a été appliquée à tous les enfants du conteneur. L’héritage peut être combiné avec la délégation pour accorder des droits d’administration à une sous-arborescence entière du répertoire en une seule opération.
