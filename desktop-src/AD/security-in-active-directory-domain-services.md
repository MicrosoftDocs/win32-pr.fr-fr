# <a name="security-in-active-directory-domain-services"></a>Sécurité dans Active Directory Domain Services

Active Directory Domain Services font partie de la base Windows 2000 Trusted Computing et sont un participant complet à l’infrastructure de sécurité Windows 2000. Chaque objet d’un serveur de Active Directory est protégé par son propre descripteur de sécurité. Le système valide toute tentative d’accès à un objet ou un attribut dans un serveur de Active Directory en vérifiant les droits d’accès accordés par le descripteur de sécurité de l’objet.

Les rubriques suivantes abordent les éléments importants de la sécurité dans Active Directory Domain Services :

-   [Protection des objets et des attributs](object-and-attribute-protection.md)
-   [La délégation](delegation.md)
-   [Héritage](inheritance.md)

Pour plus d’informations et d’exemples de code, consultez [contrôle de l’accès aux objets dans Active Directory Domain Services](controlling-access-to-objects-in-active-directory-domain-services.md).
