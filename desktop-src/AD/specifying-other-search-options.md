# <a name="specifying-other-search-options"></a>Spécification d’autres options de recherche

Active Directory Domain Services prendre en charge la plupart des options de recherche fournies par la technologie de recherche. Pour plus d’informations sur les autres options de recherche, consultez [spécification d’autres options de recherche avec IDirectorySearch](/windows/desktop/ADSI/specifying-other-search-options-with-idirectorysearch).
