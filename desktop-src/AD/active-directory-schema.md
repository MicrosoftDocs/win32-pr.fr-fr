# <a name="active-directory-schema-ad-ds"></a>Schéma de Active Directory (AD DS)

Le schéma Active Directory contient des définitions formelles de chaque classe d’objet pouvant être créée dans une forêt Active Directory. Le schéma contient également des définitions formelles de chaque attribut qui peut exister dans un objet Active Directory.

Les rubriques qui fournissent une vue d’ensemble du schéma Active Directory sont les suivantes :

-   [Implémentation du schéma](schema-implementation.md)
-   [Caractéristiques des classes d’objets](characteristics-of-object-classes.md)
-   [Caractéristiques des attributs](characteristics-of-attributes.md)
-   [Schéma abstrait](the-abstract-schema.md)

Pour plus d’informations sur la programmation de schéma, notamment la lecture du schéma et la définition de nouvelles classes et attributs dans le schéma, consultez [extension du schéma](extending-the-schema.md).

Pour plus d’informations, consultez la [référence Active Directory Domain Services](active-directory-domain-services-reference.md). Pour obtenir les pages de référence pour les classes de schéma, les attributs et les syntaxes d’attribut prédéfinis, consultez la [référence de schéma Active Directory](/windows/desktop/ADSchema/active-directory-schema).
