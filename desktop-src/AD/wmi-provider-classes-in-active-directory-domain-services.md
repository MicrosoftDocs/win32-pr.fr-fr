# <a name="wmi-provider-classes-in-active-directory-domain-services"></a>Classes du fournisseur WMI dans Active Directory Domain Services

Active Directory Domain Services disposez des classes suivantes pour une utilisation avec les scripts WMI :

-   [**MSAD \_ DomainController**](msad-domaincontroller.md)
-   [**MSAD \_ du DN**](msad-namingcontext.md)
-   [**MSAD \_ ReplCursor**](msad-replcursor.md)
-   [**MSAD \_ ReplNeighbor**](msad-replneighbor.md)
-   [**MSAD \_ ReplPendingOp**](msad-replpendingop.md)

Le fournisseur lui-même est représenté par la classe [**ReplicationProvider1**](replicationprovider1.md) , qui est instanciée avec le nom « ReplProv1 ».
