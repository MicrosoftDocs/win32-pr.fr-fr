# <a name="controlling-object-access-in-active-directory-domain-services"></a>Contrôle de l’accès aux objets dans Active Directory Domain Services

Chaque objet de service d’annuaire Active Directory est protégé par la sécurité Windows 2000. Cette protection de sécurité contrôle les opérations que chaque principal de sécurité peut effectuer dans l’annuaire. Les sections suivantes décrivent comment une application activée pour les annuaires peut utiliser les fonctionnalités de contrôle d’accès dans Active Directory.

-   [Utilisation de l’héritage des autorisations d’accès pour permettre l’accès administratif à l’intégralité d’une sous-arborescence du répertoire](inheritance-and-delegation-of-administration.md)
-   [Création, modification et lecture du descripteur de sécurité par défaut pour une classe d’objets](default-security-descriptor.md)
