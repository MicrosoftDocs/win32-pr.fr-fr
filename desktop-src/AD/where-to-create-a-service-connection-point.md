# <a name="where-to-create-a-service-connection-point"></a><span data-ttu-id="c6f5f-108">Où créer un point de connexion de service</span><span class="sxs-lookup"><span data-stu-id="c6f5f-108">Where to Create a Service Connection Point</span></span>
