# <a name="cfstr_ds_parenthwnd"></a>CFSTR \_ DS \_ PARENTHWND

<dl> <dt>

<span id="CFSTR_DS_PARENTHWND"></span><span id="cfstr_ds_parenthwnd"></span>**CFSTR \_ DS \_ PARENTHWND**
</dt> <dd> <dl> <dt>

"DsAdminParentHwndClipFormat"
</dt> <dt>



Le format de presse-papiers **CFSTR \_ DS \_ PARENTHWND** fournit un handle de mémoire globale (**HGLOBAL**) qui contient le handle de fenêtre (**HWND**) de la fenêtre MMC principale.

> [!Note]  
> Cette valeur de format de presse-papiers n’est pas définie dans un fichier d’en-tête publié. Pour utiliser cette valeur de format de presse-papiers, définissez-la dans le format exact indiqué.
