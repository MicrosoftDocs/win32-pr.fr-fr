L’interface utilisateur de Active Directory Domain Services comprend des feuilles de propriétés et d’autres interfaces utilisateur pour définir et récupérer les attributs des classes couramment utilisées. Les rubriques de cette section contiennent des tables qui mappent les étiquettes qui s’affichent dans les interfaces utilisateur avec les attributs correspondants dans le schéma de Active Directory.
