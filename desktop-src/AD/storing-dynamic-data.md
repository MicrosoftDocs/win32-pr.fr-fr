# <a name="storing-dynamic-data"></a>Stockage des Dynamic Data

La plupart des données stockées à l’aide de Active Directory Domain Services sont statiques. Par exemple, les objets d’annuaire qui représentent des utilisateurs, des groupes, des imprimantes, des services réseau, etc., sont généralement de longue durée et rarement modifiés.

Dans Windows Server 2003 et les versions ultérieures de Windows, Active Directory Domain Services assurer la prise en charge des fonctionnalités suivantes, qui permettent aux applications d’utiliser l’annuaire pour le stockage des données dynamiques :

-   [Objets dynamiques](dynamic-objects.md)
-   [Classes auxiliaires dynamiques](dynamic-auxiliary-classes.md)
