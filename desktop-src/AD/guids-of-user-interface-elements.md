# <a name="guids-of-user-interface-elements"></a>GUID des éléments d’interface utilisateur

Le tableau suivant répertorie l’identificateur de classe, exprimé sous la forme d’un GUID, pour les objets d’interface utilisateur implémentés par un service d’annuaire.



| Identificateur de classe              | Description                                                                                                                                                                                                                                                                 |
|-------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **CLSID \_ DsFolderProperties** | Réservé. Déclaré dans dsclient. h.                                                                                                                                                                                                                                           |
| **CLSID \_ DsPropertyPages**    | Cet objet fournit les interfaces [**IShellExtInit**](/windows/win32/api/shobjidl_core/nn-shobjidl_core-ishellextinit), [**IShellPropSheetExt**](/windows/win32/api/shobjidl_core/nn-shobjidl_core-ishellpropsheetext)et [**IContextMenu**](/windows/win32/api/shobjidl_core/nn-shobjidl_core-icontextmenu) à utiliser avec les objets de service d’annuaire fournis par le système. Déclaré dans dsclient. h. |
