# <a name="distributing-user-interface-components"></a>Distribution des composants de l’interface utilisateur

Les composants COM, les applications et/ou les fichiers utilisés par une extension d’interface utilisateur peuvent être distribués en créant un package Windows Installer et en utilisant une stratégie de groupe pour distribuer le package. Pour plus d’informations sur la création du package d’installation, consultez [Windows Installer](/windows/desktop/Msi/windows-installer-portal). Pour plus d’informations sur les stratégies de groupe, consultez [stratégie de groupe](/previous-versions/windows/desktop/Policy/group-policy-start-page).
