# <a name="creating-dynamic-objects"></a>Création d’objets dynamiques

Spécifiez la classe auxiliaire **DynamicObject** comme valeur de l’attribut **objectClass** au moment de l’instanciation d’un objet. Dans ce cas, la classe auxiliaire spéciale s’affiche dans la liste des classes qui est la valeur de l’attribut **objectClass** pour l’objet. Si une valeur pour l’attribut **entryTTL** n’est pas fournie au moment de l’ajout de la classe auxiliaire spéciale, une valeur par défaut, comme indiqué plus tard, est fournie par Active Directory Domain Services.

Pour plus d’informations sur l’ajout d’une classe auxiliaire à un objet, consultez [classes auxiliaires dynamiques](dynamic-auxiliary-classes.md).
