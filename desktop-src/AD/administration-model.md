# <a name="administration-model"></a>Modèle d’administration

Les utilisateurs autorisés effectuent l’administration dans Active Directory Domain Services. Un utilisateur est autorisé par une autorité supérieure à exécuter un ensemble spécifique d’actions sur un ensemble d’objets et de classes d’objets spécifié dans une sous-arborescence identifiée du répertoire. C’est ce que l’on appelle l' *administration déléguée*. L’administration déléguée permet un contrôle très précis sur les personnes qui peuvent faire ce qui est autorisé à déléguer l’autorité sans accorder de privilèges élevés.
