<span data-ttu-id="e20e3-106">Lors du développement d’une application ou d’un service, vous pouvez utiliser des groupes pour déléguer l’administration ou contrôler l’accès à l’application ou au service en totalité ou en partie.</span><span class="sxs-lookup"><span data-stu-id="e20e3-106">When developing an application or service, you can use groups to delegate administration or control access to the application or service as a whole or part.</span></span> <span data-ttu-id="e20e3-107">Par exemple, une application peut contrôler qui peut effectuer diverses opérations d’administration.</span><span class="sxs-lookup"><span data-stu-id="e20e3-107">For example, an application can control who can perform various administrative operations.</span></span> <span data-ttu-id="e20e3-108">Pour ce faire, créez des ACE qui accordent des droits d’accès spécifiés aux groupes appropriés.</span><span class="sxs-lookup"><span data-stu-id="e20e3-108">To do this, create ACEs that grant specified access rights to the appropriate groups.</span></span> <span data-ttu-id="e20e3-109">Il est recommandé de disposer d’un ACE qui accorde l’accès à un groupe plutôt que d’avoir plusieurs ACE qui accordent l’accès à des utilisateurs individuels ou à des ordinateurs.</span><span class="sxs-lookup"><span data-stu-id="e20e3-109">It is good programming practice to have an ACE that grants access to a group than to have several ACEs that grant access to individual users or computers.</span></span>

    -   <span data-ttu-id="e20e3-119">Utilisez des groupes universels uniquement si nécessaire (les données de membre sont requises globalement à l’aide d’un catalogue global ; le groupe peut contenir n’importe quel utilisateur/groupe).</span><span class="sxs-lookup"><span data-stu-id="e20e3-119">Use universal groups only if necessary (member data is required globally using global catalog; group can contain any user/group).</span></span> <span data-ttu-id="e20e3-120">Si vous utilisez des groupes universels, placez les groupes globaux dans le groupe universel et ajoutez/supprimez des utilisateurs dans le groupe global.</span><span class="sxs-lookup"><span data-stu-id="e20e3-120">If you do use universal groups, place global groups in the universal group and add/remove users from the global group.</span></span> <span data-ttu-id="e20e3-121">Évitez toute modification excessive des groupes universels pour améliorer l’efficacité de la réplication.</span><span class="sxs-lookup"><span data-stu-id="e20e3-121">Avoid excess change to universal groups for replication efficiency.</span></span>
