# <a name="obtaining-a-root-oid-from-an-iso-name-registration-authority"></a>Obtention d’un OID racine auprès d’une autorité d’inscription de noms ISO

La meilleure façon d’obtenir un identificateur d’objet racine (OID) consiste à en demander un auprès d’une autorité d’inscription de noms ISO (International Standards Organization). Il s’agit d’une action ponctuelle ; Lorsque vous avez obtenu un OID racine, l’espace OID qu’il définit vous appartient et vous pouvez l’administrer vous-même.

Il y a généralement un coût associé à l’inscription d’un nom d’organisation et à la réception d’un OID racine. Les frais et les stratégies d’inscription varient en fonction du pays ou de la région. Dans le États-Unis, la norme ISO ARN est la American National Standards Institute (ANSI). Pour plus d’informations sur la procédure d’inscription ANSI et le calendrier des frais, consultez https://web.ansi.org . L’ISO gère une liste d’organisations membres à l’adresse https://www.iso.ch . Si vous n’êtes pas au États-Unis, vous devez contacter l’organisation membre ISO pour votre pays/région pour obtenir des informations sur l’inscription des noms.

Quelle que soit la source utilisée pour obtenir l’OID, si vous avez l’intention d’étendre le schéma Active Directory et que vous souhaitez demander le logo certifié pour Windows, vous devez inscrire votre OID auprès de Microsoft. Pour plus d’informations sur l’inscription de votre OID auprès de Microsoft, consultez [obtention d’un identificateur d’objet auprès de Microsoft](obtaining-an-object-identifier-from-microsoft.md).
