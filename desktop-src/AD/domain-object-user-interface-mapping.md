# <a name="domain-object-user-interface-mapping"></a>Mappage de l’interface utilisateur de l’objet de domaine

Cette rubrique décrit les feuilles de propriétés d’objet de domaine dans le composant logiciel enfichable utilisateurs et ordinateurs Active Directory.

## <a name="general-property-sheet"></a>Feuille de propriétés général

Le tableau suivant présente les étiquettes de l’interface utilisateur dans la feuille de propriétés **général** .



| Étiquette d’interface utilisateur                       | Attribut Active Directory |
|--------------------------------|----------------------------|
| Nom de domaine (antérieur à Windows 2000) | DC\*                       |
| Description                    | description                |



 

\*L’attribut DC fournit l’accès au premier des composants de domaine qui n’est pas nécessairement le même que le DN NetBIOS.
