# <a name="supported-installation-mechanisms"></a>Mécanismes d’installation pris en charge

Active Directory Domain Services prennent en charge quatre mécanismes d’extension du schéma Active Directory. Vous pouvez étendre le schéma Active Directory à l’aide de

-   [Scripts LDIF](ldif-scripts.md) (par défaut)
-   [Scripts de valeurs séparées par des virgules (CSV)](comma-separated-value-csv-scripts.md)
-   [Extension de programmation](programmatic-extension.md)
-   et en [étendant l’interface utilisateur pour les objets d’annuaire](extending-the-user-interface-for-directory-objects.md).
