# <a name="managing-users-on-member-servers-and-windows-2000-professional"></a>gestion des utilisateurs sur les serveurs membres et les Windows 2000 Professional

sur les serveurs membres et Windows Professional 2000, il existe une base de données de sécurité locale. Cette base de données de sécurité locale peut contenir ses propres utilisateurs locaux dont l’étendue est uniquement l’ordinateur sur lequel ils sont créés. Lors de la gestion de ces types d’utilisateurs sur les serveurs et stations de travail membres, vous utilisez le fournisseur Winnt.

lorsque vous gérez des utilisateurs sur un domaine Windows 2000 à l’aide d’ADSI, vous utilisez le fournisseur LDAP. Lorsque vous gérez des utilisateurs sur des serveurs et des stations de travail membres, vous utilisez le fournisseur Winnt.
