# <a name="managing-users-on-member-servers-and-windows-2000-professional"></a>Gestion des utilisateurs sur les serveurs membres et Windows 2000 professionnel

Sur les serveurs membres et Windows 2000 professionnel, il existe une base de données de sécurité locale. Cette base de données de sécurité locale peut contenir ses propres utilisateurs locaux dont l’étendue est uniquement l’ordinateur sur lequel ils sont créés. Lors de la gestion de ces types d’utilisateurs sur les serveurs et stations de travail membres, vous utilisez le fournisseur Winnt.

Lorsque vous gérez des utilisateurs sur un domaine Windows 2000 à l’aide d’ADSI, vous utilisez le fournisseur LDAP. Lorsque vous gérez des utilisateurs sur des serveurs et des stations de travail membres, vous utilisez le fournisseur Winnt.
