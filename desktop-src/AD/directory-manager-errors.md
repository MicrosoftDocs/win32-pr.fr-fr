# <a name="directory-manager-errors"></a>Erreurs du gestionnaire de répertoires

Cette rubrique contient les valeurs renvoyées pour les erreurs du gestionnaire de répertoires dans les fonctions de Active Directory Domain Services.



| Valeur                    | Description                                              |
|--------------------------|----------------------------------------------------------|
| **hrPMRecDeleted**       | L’enregistrement a été supprimé.                                  |
| **hrRemainingVersions**  | Il reste du travail inactif.                            |
| **hrFLDKeyTooBig**       | La clé a été tronquée, car elle est supérieure à 255 octets. |
| **hrFLDTooManySegments** | Le nombre de segments de clé est trop important.                         |
| **hrFLDNullKey**         | La clé est **null**.                                     |
