# <a name="documenting-schema-extensions"></a>Documentation des extensions de schéma

En plus de fournir une procédure d’installation distincte pour les extensions de schéma, la nature des extensions de schéma doit être clairement documentée. La documentation doit contenir :

-   Liste de toutes les nouvelles classes et de nouveaux attributs avec une explication de chacune d’elles.
-   Liste de toutes les classes standard qui sont étendues avec de nouveaux attributs et une liste des attributs
-   Liste de toutes les nouvelles pages de propriétés avec une explication de l’utilisation de chacune d’elles.
-   Explication de la façon dont le service utilise les extensions de schéma
