# <a name="access-rights-for-active-directory-domain-services-objects"></a>Droits d’accès pour les objets Active Directory Domain Services

Tous les objets d’annuaire utilisent le même ensemble de droits d’accès prédéfinis qui correspondent aux opérations d’annuaire courantes. Ces droits sont décrits dans le type d’énumération d’énumération des [**\_ \_ droits ADS**](/windows/win32/api/iads/ne-iads-ads_rights_enum) .

En outre, vous pouvez définir des droits d’accès privés (contrôler les droits d’accès) pour les opérations qui vont au-delà de celles couvertes par les droits prédéfinis. Consultez [contrôler les droits d’accès](control-access-rights.md).
