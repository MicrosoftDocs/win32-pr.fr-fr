# <a name="searching-in-active-directory-domain-services"></a>Recherche dans Active Directory Domain Services

La recherche est une fonctionnalité clé de Active Directory Domain Services. La recherche vous permet de rechercher des objets dans le répertoire en fonction de critères de sélection (requête) et de récupérer les propriétés spécifiées pour les objets trouvés.

La recherche dans Active Directory Domain Services consiste à trouver un contrôleur de domaine (DC), à se lier à l’objet où la recherche doit commencer dans le répertoire, à envoyer une requête et à traiter les résultats.
