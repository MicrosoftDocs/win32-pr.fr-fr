# <a name="using-standard-dialog-boxes-to-handle-objects-in-active-directory-domain-services"></a>Utilisation de boîtes de dialogue standard pour gérer des objets dans Active Directory Domain Services

Windows fournit une bibliothèque de boîtes de dialogue commune pour les opérations courantes, telles que l’ouverture de fichiers, la recherche de fichiers, etc. À partir de Windows 2000, le système fournit des boîtes de dialogue qui peuvent être utilisées pour les opérations d’interface utilisateur courantes dans Active Directory Domain Services.

Les boîtes de dialogue suivantes peuvent être utilisées avec Active Directory Domain Services :

-   [Sélecteur d’objets d’annuaire](directory-object-picker.md)
-   [Explorateur de domaines](domain-browser.md)
-   [Navigateur de conteneurs](container-browser.md)
-   [Active Directory les utilisateurs et les feuilles de propriétés des ordinateurs](active-directory-users-and-computers-property-sheets.md)

En outre, les assistants de création de Active Directory Domain Services peuvent également être appelés à partir d’une application. Pour plus d’informations, consultez [appel de l’Assistant Création à partir de votre application](invoking-creation-wizards-from-your-application.md).
