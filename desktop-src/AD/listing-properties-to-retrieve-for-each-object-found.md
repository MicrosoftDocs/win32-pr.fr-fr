# <a name="deciding-which-attributes-to-retrieve-for-each-object-found"></a>Choix des attributs à récupérer pour chaque objet trouvé

Si vous recherchez des objets d’une classe particulière, il semble logique que vous récupériez des attributs qui existent réellement sur les objets de cette classe. Suivez les mêmes règles que celles décrites dans la [recherche d’une liste d’attributs à interroger](finding-a-list-of-attributes-to-query.md). Sachez que les attributs construits peuvent être récupérés dans la requête, mais les attributs construits ne peuvent pas être utilisés dans les filtres.
