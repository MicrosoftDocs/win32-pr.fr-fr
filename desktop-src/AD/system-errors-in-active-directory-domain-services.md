# <a name="system-errors-in-active-directory-domain-services"></a>Erreurs système dans Active Directory Domain Services

Cette rubrique contient des valeurs de retour d’erreur système dans les fonctions de Active Directory Domain Services.



| Valeur                             | Description                                                           |
|-----------------------------------|-----------------------------------------------------------------------|
| **hrFileClose**<br/>        | Impossible de fermer le fichier DOS.<br/>                              |
| **hrOutOfThreads**<br/>     | Impossible de démarrer un thread, car aucun n’est disponible.<br/> |
| **hrTooManyIO**<br/>        | Le système est occupé en raison d’un trop grand nombre d’e/s.<br/>        |
| **hrBFNotSynchronous**<br/> | La page de la mémoire tampon a été supprimée.<br/>                          |
| **hrBFPageNotFound**<br/>   | Impossible de trouver la page.<br/>                                   |
| **hrBFInUse**<br/>          | Impossible d’abandonner la mémoire tampon.<br/>                              |
