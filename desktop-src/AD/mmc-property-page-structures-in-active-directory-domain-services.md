# <a name="mmc-property-page-structures-in-active-directory-domain-services"></a>Structures de la page de propriétés MMC dans Active Directory Domain Services

Le composant logiciel enfichable MMC Active Directory Manager utilise les feuilles de propriétés pour afficher les attributs des objets dans Active Directory Domain Services. Les structures suivantes aident à fournir l’infrastructure permettant de contrôler les opérations effectuées sur les pages de ces feuilles de propriétés :

-   [**ADSPROPERROR**](/windows/desktop/api/Adsprop/ns-adsprop-adsproperror)
-   [**ADSPROPINITPARAMS**](/windows/desktop/api/Adsprop/ns-adsprop-adspropinitparams)
