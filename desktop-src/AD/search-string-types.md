# <a name="initial-medial-and-final-search-strings"></a>Chaînes de recherche initiale, médiane et finale

Il existe trois types de recherches de caractères génériques référencés dans toute la documentation sur les requêtes de chaîne de recherche de construction. Ces recherches de caractères génériques sont : les chaînes de recherche initiale, médiane et finale. Cette rubrique décrit ces recherches.

## <a name="initial-search-strings"></a>Chaînes de recherche initiales

Les chaînes de recherche initiale correspondent à un ensemble donné de caractères au début d’une chaîne, suivis d’un caractère générique. Par exemple, la chaîne de recherche initiale `Act*` correspondrait à « Active Directory ».

## <a name="medial-search-strings"></a>Chaînes Medial-Search

Les chaînes de recherche médiane correspondent à un ensemble donné de caractères au milieu d’une chaîne, précédés de et/ou suivis d’un caractère générique. Les chaînes de recherche médiane `*Dir*` , `*ive*Dir*` et `*ve*Dir*tor*` correspondent tous à « Active Directory ».

## <a name="final-search-strings"></a>Chaînes de recherche finales

Les chaînes de recherche finale correspondent à un ensemble donné de caractères à la fin d’une chaîne, précédés d’un caractère générique. Par exemple, la chaîne de recherche finale `*ory` correspondrait à « Active Directory ».
