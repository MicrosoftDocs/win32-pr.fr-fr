# <a name="enumerations-windows-update-agent"></a>Énumérations (agent de Windows Update)

Dans le stockage structuré, les énumérations sont des types de données qui se composent d’une séquence de valeurs nommées dans un ordre donné. Un type de données énuméré permet de décrire un membre d’une classe d’objets en langage naturel. Les énumérations sont particulièrement utiles dans le contexte où toutes les valeurs possibles d’une variable sont connues et l’expression de cette variable dans le langage de programmation peut donc être exprimée en mots de langage naturel.

Les énumérations de stockage structuré sont décrites dans les sections suivantes.

-   [**VERROU**](/windows/win32/api/objidl/ne-objidl-locktype)
-   [**STATFLAG**](/windows/win32/api/wtypes/ne-wtypes-statflag)
-   [**STGC**](/windows/win32/api/wtypes/ne-wtypes-stgc)
-   [**STGFMT**](/previous-versions/windows/desktop/legacy/aa380330(v=vs.85))
-   [**STGMOVE**](/windows/win32/api/wtypes/ne-wtypes-stgmove)
-   [**STGTY**](/windows/win32/api/objidl/ne-objidl-stgty)
-   [**recherche de flux \_**](/windows/win32/api/objidl/ne-objidl-stream_seek)
