# <a name="allocating-format-identifiers"></a>Allouer des identificateurs de format

Les FMTIDs sont créés et représentés de la même façon que les CLSID COM et les identificateurs d’interface. Pour créer un FMTID unique, utilisez le programme UUIDGEN.EXE inclus dans l’API® Win32.
