# <a name="count-of-properties"></a>Nombre de propriétés

Ce type de données **DWORD** donne le nombre de valeurs de propriété dans la section. Un jeu de propriétés peut contenir n’importe quel nombre de valeurs de propriété. Les lecteurs doivent être en mesure de gérer le cas où il n’y a aucune propriété.
