# <a name="shared-access-flags"></a>Indicateurs d’accès partagé

Les indicateurs d’accès partagé spécifient le degré d’accès que d’autres applications peuvent avoir à un objet que votre application ouvre. Vous pouvez refuser l’accès en lecture, refuser l’accès en écriture ou refuser tous les accès. Vous pouvez également spécifier explicitement qu’aucun type d’accès n’est refusé.
