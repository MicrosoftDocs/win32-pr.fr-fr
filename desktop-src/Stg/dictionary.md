# <a name="dictionary"></a>Dictionnaire

[**IPropertyStorage :: WritePropertyNames**](/windows/desktop/api/Propidl/nf-propidl-ipropertystorage-writepropertynames) est implémenté à l’aide du dictionnaire de l' [ID de propriété 0](/windows/desktop/Stg/reserved-property-identifiers). L’ID de propriété 0 n’est pas accessible via [**IPropertyStorage :: ReadMultiple**](/windows/desktop/api/Propidl/nf-propidl-ipropertystorage-readmultiple) ou [**IPropertyStorage :: WriteMultiple**](/windows/desktop/api/Propidl/nf-propidl-ipropertystorage-writemultiple).
