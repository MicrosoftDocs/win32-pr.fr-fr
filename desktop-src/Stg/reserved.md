# <a name="reserved"></a>Réservé

La valeur du type de données **DWORD** est réservée à une utilisation ultérieure. Les writers de jeux de propriétés doivent définir cette valeur sur 1 ; les lecteurs de jeux de propriétés doivent s’assurer que cette valeur est au moins égale à 1. Une exception à cette règle est que, pour [les jeux de propriétés DocumentSummaryInformation et UserDefined](the-documentsummaryinformation-and-userdefined-property-sets.md), cette valeur peut être 2.
