# <a name="temporary-creation-flags"></a>Indicateurs de création temporaires

L’indicateur de création temporaire indique que le fichier sous-jacent doit être détruit automatiquement lorsque l’objet de stockage racine est libéré. Cette fonctionnalité est particulièrement utile pour créer des fichiers temporaires.
