# <a name="access-permission-flags"></a>Indicateurs d’autorisation d’accès

Les indicateurs d’autorisation d’accès spécifient le type d’accès d’une application cliente à un objet ouvert : lecture, écriture ou lecture/écriture. L’autorisation lecture permet à une application de lire le contenu d’un flux, mais pas d’écrire des modifications dans le flux. L’autorisation d’écriture permet à une application d’appeler une fonction qui valide les modifications apportées au stockage d’un objet, mais pas de lire le contenu du flux. L’autorisation de lecture/écriture permet à une application de lire les flux de l’objet ou d’écrire dans le stockage de l’objet.
