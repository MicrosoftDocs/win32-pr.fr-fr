# <a name="storage-creation-flags"></a>Indicateurs de création de stockage

Les indicateurs de création de stockage spécifient ce que COM doit faire si un objet de stockage, de flux ou de LockBytes existant a le même nom qu’un nouvel objet de stockage ou de flux que vous créez. La valeur par défaut consiste à retourner un message d’erreur et à ne pas créer le nouvel objet. Vous ne pouvez utiliser qu’un seul de ces indicateurs dans un appel de création donné.
