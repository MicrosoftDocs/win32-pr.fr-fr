# <a name="byte-order-indicator"></a>Indicateur de Byte-Order

L’indicateur d’ordre d’octet est un type de données **Word** et doit toujours contenir la valeur 0xFFFE. Il est identique à l’indicateur d’ordre d’octet Unicode. Cette valeur est toujours écrite dans l’ordre d’octet Intel et, par conséquent, elle apparaît dans le fichier ou le flux sous la forme 0xFE, 0xFF.
