# <a name="class-identifier"></a>Identificateur de classe

L’identificateur de classe (CLSID) est celui d’une classe qui peut afficher et/ou fournir un accès par programme aux valeurs de propriété. S’il n’existe pas de classe de ce type, il est recommandé de définir cette valeur de la même façon que FMTID (voir ci-dessous). Vous pouvez également définir cette valeur sur tous les zéros. Toutefois, l’utilisation de FMTID permet une plus grande flexibilité à l’avenir.
