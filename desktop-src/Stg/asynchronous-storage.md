class="sxs-lookup"><span data-stu-id="f6cd6-106">Asynchronous storage enables both new and legacy applications that use compound files to efficiently render their content when accessed by means of existing Internet protocols.</span></span> <span data-ttu-id="f6cd6-107">Une demande unique à un serveur de World Wide Web déclenche le téléchargement d’objets imbriqués contenus dans une page Web, ce qui évite d’avoir à demander séparément chaque objet.</span><span class="sxs-lookup"><span data-stu-id="f6cd6-107">A single request to a World Wide Web server triggers the download of nested objects class="sxs-lookup"><span data-stu-id="f6cd6-109">The exact order in which elements of a page become available can be specified by the Web publisher and is not dependent on random factors of network
