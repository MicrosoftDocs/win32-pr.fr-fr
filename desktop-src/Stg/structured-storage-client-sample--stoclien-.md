# <a name="structured-storage-client-sample-stoclien"></a>Exemple de client de stockage structuré (StoClien)

L’exemple **StoClien** introduit une application de dessin simple. Cette application permet à l’utilisateur d’utiliser une souris ou un périphérique tablette pour effectuer un dessin libre dans la fenêtre du client. La couleur et la largeur de l’encre électronique peuvent être choisies et les dessins peuvent être enregistrés dans des fichiers.

La création et l’exécution de l’exemple **StoClien** fournissent les informations nécessaires pour comprendre la structure du client de stockage structuré, ses utilisations et ses fonctionnalités.

Les sections suivantes fournissent des exemples d’instructions **StoClien** , des informations de référence et des liens vers l’exemple de code **StoClien** .
