# <a name="ivmvirtualmachineserialports-property"></a>IVMVirtualMachine :: SerialPorts, propriété

\[Windows Virtual PC ne peut plus être utilisé à partir de Windows 8. Au lieu de cela, utilisez le [fournisseur WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Récupère une collection énumérable de ports série.

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```C++
HRESULT get_SerialPorts(
  [out, retval] IVMSerialPortCollection **serialPortCollection
);
```



## <a name="property-value"></a>Valeur de la propriété

Objet [**IVMSerialPortCollection**](ivmserialportcollection.md) .
