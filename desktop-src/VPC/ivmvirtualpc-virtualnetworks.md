# <a name="ivmvirtualpcvirtualnetworks-property"></a>IVMVirtualPC :: VirtualNetworks, propriété

\[Windows Virtual PC n’est plus disponible pour une utilisation à partir de Windows 8. Au lieu de cela, utilisez le [fournisseur WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Récupère une collection énumérable de réseaux virtuels.

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```C++
HRESULT get_VirtualNetworks(
  [out, retval] IVMVirtualNetworkCollection **virtualNetworkCollection
);
```



## <a name="property-value"></a>Valeur de la propriété

Collection d’objets [**IVMVirtualNetwork**](ivmvirtualnetwork.md) . Consultez [**IVMVirtualNetworkCollection**](ivmvirtualnetworkcollection.md).
