# <a name="do-enumerations"></a>Énumérations DO

Les [interfaces d’optimisation de la remise (do)](do-interfaces.md) utilisent les énumérations suivantes.

## <a name="in-this-section"></a>Contenu de cette section

| Rubrique | Description |
|-|-|
| [**BG_ERROR_CONTEXT**](bg-error-context.md) | L’énumération **BG_ERROR_CONTEXT** définit les valeurs constantes qui spécifient le contexte dans lequel l’erreur s’est produite. |
| [**BG_JOB_PRIORITY**](bg-job-priority-.md) | L’énumération **BG_JOB_PRIORITY** définit les valeurs constantes qui spécifient le niveau de priorité d’un travail.  |
| [**BG_JOB_STATE**](bg-job-state-.md) | L’énumération **BG_JOB_STATE** définit des valeurs constantes pour les différents États d’un travail. |
| [**BG_JOB_TYPE**](bg-job-type.md) | L’énumération **BG_JOB_TYPE** définit des valeurs constantes qui spécifient le type de tâche de transfert, par exemple télécharger. |
| [**BITS_FILE_PROPERTY_ID**](bits-file-property-id-.md) | L’énumération **BITS_FILE_PROPERTY_ID** spécifie des valeurs qui définissent des valeurs d’ID correspondant aux propriétés de **BackgroundCopyFile** . |
| [**BITS_JOB_PROPERTY_ID**](bits-job-property-id.md) | L’énumération **BITS_JOB_PROPERTY_ID** spécifie l’ID de la propriété pour le travail do. Cette énumération est utilisée dans l’Union [**BITS_JOB_PROPERTY_VALUE**](bits-job-property-value-.md) pour déterminer le type de valeur contenu dans l’Union. |
| [**BITS_JOB_TRANSFER_POLICY**](bits-job-transfer-policy-.md) | L’énumération **BITS_JOB_TRANSFER_POLICY** définit les valeurs d’ID correspondant aux propriétés do. |
| [**DeliveryOptimizationFileProperty**](deliveryoptimizationfileproperty.md) | Spécifie l’ID d’une propriété facultative pour le fichier DO. |
| [**DODownloadCostPolicy**](./deliveryoptimizationdownloadtypes/ne-deliveryoptimizationdownloadtypes-dodownloadcostpolicy.md) | Spécifie l’ID des options de stratégie de coût associées à la propriété **DODownloadProperty_CostPolicy** . |
| [**DODownloadProperty**](./deliveryoptimizationdownloadtypes/ne-deliveryoptimizationdownloadtypes-dodownloadproperty.md) | Spécifie l’ID des propriétés de l’opération de téléchargement. |
| [**DODownloadPropertyEx**](./dodownloadinternal/ne-dodownloadinternal-dodownloadpropertyex.md) | Spécifie l’ID des propriétés étendues pour l’opération de téléchargement. |
| [**DODownloadState**](./deliveryoptimizationdownloadtypes/ne-deliveryoptimizationdownloadtypes-dodownloadstate.md) | Spécifie l’ID de l’état de téléchargement actuel, qui fait partie de la structure **DO_DOWNLOAD_STATUS** . |
| [**DownloadMode**](downloadmode.md) | Définit les différents modes de téléchargement utilisés par l’optimisation de la distribution.  |
| [**SwarmStatus**](swarmstatus.md) | Définit l’état d’un fichier dans le client d’optimisation de la remise. |
