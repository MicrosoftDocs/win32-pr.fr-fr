# <a name="macros"></a>Macros

Les rubriques de cette section fournissent les spécifications de référence pour les [messages d’entrée de pointeur et](messages-and-notifications-portal.md) les macros de notifications permettant de récupérer des informations à partir de paramètres de message d’entrée de pointeur.

## <a name="in-this-section"></a>Contenu de cette section



| Rubrique                                                                                  | Description                                                                                                                         |
|----------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------|
| [**GET_POINTERID_WPARAM**](/previous-versions/windows/desktop/api)<br/>                      | Récupère l’ID de pointeur à l’aide de la valeur spécifiée. <br/>                                                                     |
| [**HAS_POINTER_CONFIDENCE_WPARAM**](/previous-versions/windows/desktop/api)<br/>   | Vérifie si le message de pointeur spécifié est considéré comme intentionnel plutôt que accidentel.<br/>                           |
| [**IS_POINTER_CANCELED_WPARAM**](/previous-versions/windows/desktop/api)<br/>         | Vérifie si l’entrée de pointeur spécifiée s’est terminée brusquement ou n’est pas valide, ce qui indique que l’interaction n’a pas été effectuée.<br/> |
| [**IS_POINTER_FIFTHBUTTON_WPARAM**](/previous-versions/windows/desktop/api)<br/>   | Vérifie si le pointeur spécifié a pris la cinquième mesure. <br/>                                                                 |
| [**IS_POINTER_FIRSTBUTTON_WPARAM**](/previous-versions/windows/desktop/api)<br/>   | Vérifie si le pointeur spécifié a pris la première action.<br/>                                                                  |
| [**IS_POINTER_FLAG_SET_WPARAM**](/previous-versions/windows/desktop/api)<br/>        | Vérifie si une macro pointeur définit l’indicateur spécifié. <br/>                                                                 |
| [**IS_POINTER_FOURTHBUTTON_WPARAM**](/previous-versions/windows/desktop/api)<br/> | Vérifie si le pointeur spécifié a pris la quatrième action. <br/>                                                                |
| [**IS_POINTER_INCONTACT_WPARAM**](/previous-versions/windows/desktop/api)<br/>       | Vérifie si le pointeur spécifié est en contact. <br/>                                                                     |
| [**IS_POINTER_INRANGE_WPARAM**](/previous-versions/windows/desktop/api)<br/>           | Vérifie si le pointeur spécifié se trouve dans la plage. <br/>                                                                       |
| [**IS_POINTER_NEW_WPARAM**](/previous-versions/windows/desktop/api)<br/>                   | Vérifie si le pointeur spécifié est un nouveau pointeur. <br/>                                                                  |
| [**IS_POINTER_PRIMARY_WPARAM**](/previous-versions/windows/desktop/api)<br/>           | Vérifie si le pointeur spécifié est le pointeur principal. <br/>                                                            |
| [**IS_POINTER_SECONDBUTTON_WPARAM**](/previous-versions/windows/desktop/api)<br/> | Vérifie si le pointeur spécifié a pris la seconde action. <br/>                                                                |
| [**IS_POINTER_THIRDBUTTON_WPARAM**](/previous-versions/windows/desktop/api)<br/>   | Vérifie si le pointeur spécifié a pris la troisième action. <br/>                                                                 |
