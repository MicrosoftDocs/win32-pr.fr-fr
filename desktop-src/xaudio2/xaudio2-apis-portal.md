# <a name="xaudio2-apis"></a>API XAudio2

## <a name="purpose"></a>Objectif

XAudio2 est une API audio de bas niveau qui fournit un traitement de signal et une base de mixage pour le développement de moteurs audio hautes performances pour les jeux.

## <a name="developer-audience"></a>Développeurs concernés

Les API XAudio2 sont conçues pour être utilisées par les développeurs de jeux qui souhaitent améliorer les performances dans leurs jeux.
