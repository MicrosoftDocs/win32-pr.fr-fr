# <a name="windows-sockets-support-for-bluetooth"></a>Prise en charge de Windows Sockets pour Bluetooth

Cette section décrit comment utiliser l’interface Windows Sockets avec les fonctionnalités Bluetooth introduites dans Windows Vista. Pour plus d’informations sur les API qui rendent cela possible, consultez la page [programmation Bluetooth avec Windows Sockets](bluetooth-programming-with-windows-sockets.md) et l' [exemple de connexion Bluetooth](https://github.com/microsoftarchive/msdn-code-gallery-microsoft/tree/master/Official%20Windows%20Platform%20Sample/Bluetooth%20connection%20sample).

Pour plus d’informations sur la façon de gérer directement des appareils à l’aide d’interfaces Bluetooth sans Sockets, consultez [gestion des appareils et des services Bluetooth](managing-bluetooth-devices-and-services.md).
