# <a name="bluetooth-enumerations"></a>Bluetooth Énumérations

les énumérations suivantes prennent en charge l’implémentation Microsoft Bluetooth.



| Section                                                                                  | Content                                                                |
|------------------------------------------------------------------------------------------|------------------------------------------------------------------------|
| [**\_conditions d’authentification Bluetooth \_**](/windows/desktop/api/BluetoothAPIs/ne-bluetoothapis-bluetooth_authentication_requirements) | Définit les méthodes de protection « Man in the Middle » pour l’authentification. |
| [**\_méthode d’authentification Bluetooth \_**](/windows/desktop/api/BluetoothAPIs/ne-bluetoothapis-bluetooth_authentication_method)             | Définit les types d’authentification pris en charge lors de l’appariement des appareils.      |
| [**\_fonctionnalité d’e/s Bluetooth \_**](/windows/desktop/api/BluetoothAPIs/ne-bluetoothapis-bluetooth_io_capability)                             | définit les fonctionnalités d’entrée/sortie d’un appareil Bluetooth.           |
