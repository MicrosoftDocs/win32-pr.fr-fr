# <a name="locale_sortname"></a>paramètres régionaux \_ SORTNAME

**Windows 7 et versions ultérieures :** Nom des paramètres régionaux à utiliser pour le tri, s’il est différent des paramètres régionaux spécifiés par les [paramètres régionaux \_ sName](locale-sname.md). Cette constante est utilisée par les paramètres régionaux personnalisés. Voir aussi [paramètres régionaux \_ SSORTNAME](locale-ssort-constants.md).
