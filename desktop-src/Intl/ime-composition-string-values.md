# <a name="ime-composition-string-values"></a>Valeurs de chaîne de composition IME

Ces valeurs sont utilisées avec [**ImmGetCompositionString**](/windows/desktop/api/Imm/nf-imm-immgetcompositionstringa) et [**la \_ \_ composition de l’IME WM**](wm-ime-composition.md).



| Valeur                 | Description                                                                                |
|-----------------------|--------------------------------------------------------------------------------------------|
| comfromateurs \_ gc         | Récupérez ou mettez à jour l’attribut de la chaîne de composition.                                |
| catalogues globaux \_ COMPCLAUSE       | Récupère ou met à jour les informations de clause de la chaîne de composition.                           |
| catalogues globaux \_ COMPREADATTR     | Récupérez ou mettez à jour les attributs de la chaîne de lecture de la composition actuelle.        |
| catalogues globaux \_ COMPREADCLAUSE   | Récupère ou met à jour les informations de clause de la chaîne de lecture de la chaîne de composition. |
| catalogues globaux \_ COMPREADSTR      | Récupère ou met à jour la chaîne de lecture de la composition actuelle.                          |
| catalogues globaux \_ COMPSTR          | Récupérez ou mettez à jour la chaîne de composition actuelle.                                         |
| catalogues globaux \_ CURSORPOS        | Récupère ou met à jour la position du curseur dans la chaîne de composition.                              |
| catalogues globaux \_ DELTASTART       | Récupère ou met à jour la position de départ de toutes les modifications dans la chaîne de composition.             |
| catalogues globaux \_ RESULTCLAUSE     | Récupère ou met à jour les informations de clause de la chaîne de résultat.                                |
| catalogues globaux \_ RESULTREADCLAUSE | Récupère ou met à jour les informations de clause de la chaîne de lecture.                               |
| catalogues globaux \_ RESULTREADSTR    | Récupérez ou mettez à jour la chaîne de lecture.                                                     |
| GC \_ RESULTSTR        | Récupérez ou mettez à jour la chaîne du résultat de la composition.                                   |
