# <a name="locale_sshortestpm"></a>paramètres régionaux \_ SSHORTESTPM

Chaîne pour l’indicateur PM le plus bref possible pour les paramètres régionaux. Pour l’anglais qui peut être aussi petit que « p »; Toutefois, d’autres langages peuvent avoir plus de points de dépassements.
