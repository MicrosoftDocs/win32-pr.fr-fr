# <a name="supporting-system-language-settings"></a>Prise en charge des paramètres de langue système

Le meilleur moyen pour votre application MUI de définir les préférences de langue de l’application consiste à prendre en charge le même ensemble de langues d’interface utilisateur que le système d’exploitation cible. L’application dépend simplement du système d’exploitation pour sélectionner la langue d’interface utilisateur appropriée pour l’affichage. Un exemple associé est fourni dans [MUI : exemple d’application de paramètres système](mui-system-settings-application-sample.md).
