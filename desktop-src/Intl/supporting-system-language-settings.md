# <a name="supporting-system-language-settings"></a>prise en charge de la langue système Paramètres

Le meilleur moyen pour votre application MUI de définir les préférences de langue de l’application consiste à prendre en charge le même ensemble de langues d’interface utilisateur que le système d’exploitation cible. L’application dépend simplement du système d’exploitation pour sélectionner la langue d’interface utilisateur appropriée pour l’affichage. un exemple associé est fourni dans l' [exemple d’Application MUI : System Paramètres](mui-system-settings-application-sample.md).
