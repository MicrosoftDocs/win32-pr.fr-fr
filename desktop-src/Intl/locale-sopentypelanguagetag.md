# <a name="locale_sopentypelanguagetag"></a>paramètres régionaux \_ SOPENTYPELANGUAGETAG

**Windows 7 et versions ultérieures :** Balise de langue OpenType utilisée pour récupérer des fonctionnalités typographiques appropriées à la culture à partir d’une police. Pour plus d’informations, [**consultez \_ balise OpenType**](opentype-tag.md).
