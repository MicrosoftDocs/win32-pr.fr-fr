# <a name="locale_seng-constants"></a>Paramètres régionaux \_ Seng, \* constantes

Cette rubrique définit les \_ \* constantes Seng de paramètres régionaux utilisées par nls.



| Valeur                | Signification                                                                                                                                                                   |
|----------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| paramètres régionaux \_ SENGCOUNTRY  | déconseillé pour Windows 7 et versions ultérieures. Nom anglais complet du pays ou de la région. Consultez [paramètres régionaux \_ SENGLISHCOUNTRYNAME](locale-senglish-constants.md).                            |
| paramètres régionaux \_ SENGCURRNAME | **Windows Me/98, Windows 2000 :** Nom anglais complet de la devise associée aux paramètres régionaux. Il n’existe aucune limite quant au nombre de caractères autorisés pour cette chaîne. |
| paramètres régionaux \_ SENGLANGUAGE | déconseillé pour Windows 7 et versions ultérieures. Nom anglais complet de la langue à partir de la norme ISO 639. Consultez [paramètres régionaux \_ SENGLISHLANGUAGENAME](locale-senglish-constants.md).           |
