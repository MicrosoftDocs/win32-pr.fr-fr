# <a name="command-line-arguments"></a>Arguments de la ligne de commande

Une application peut utiliser la fonction [GetCommandLine](/windows/win32/api/processenv/nf-processenv-getcommandlinea) Win32 pour récupérer les arguments de ligne de commande Unicode. L’application doit appeler la fonction Win32 en tant que fonction Unicode.
