# <a name="spell-checking-api"></a>API vérification orthographique

L’API vérification orthographique permet aux développeurs d’utiliser la fonctionnalité de vérificateur d’orthographe pour vérifier le texte, obtenir des suggestions et gérer les paramètres et les dictionnaires utilisateur. Les développeurs peuvent également créer des vérificateurs orthographiques intégrés dans le Framework de vérification orthographique en tant que fournisseurs.

-   [À propos de l’API vérification orthographique](about-the-spell-checker-api.md)
-   [Référence de l’API vérification orthographique](spell-checker-api-reference.md)
