# <a name="locale_icurrency"></a>paramètres régionaux \_ ICURRENCY

Position du symbole monétaire en mode monétaire positif.



| Valeur | Signification                                            |
|-------|----------------------------------------------------|
| 0     | Préfixe, pas de séparation, par exemple, $1,1           |
| 1     | Suffixe, pas de séparation, par exemple $1,1           |
| 2     | Préfixe, séparation à 1 caractère, par exemple $1,1 |
| 3     | Suffixe, séparation à 1 caractère, par exemple $1,1 |
