# <a name="locale_spositivesign"></a>paramètres régionaux \_ SPOSITIVESIGN

Valeur de chaîne localisée pour le signe positif des paramètres régionaux. Par exemple, les **paramètres régionaux \_ SPOSITIVESIGN** sont considérés comme « + » pour les paramètres régionaux anglais (États-Unis). Notez que la présence de données pour cette chaîne ne signifie pas qu’un signe positif est nécessairement utilisé lors de la mise en forme d’un nombre. Si les données sont vides ou vides, un « + » est supposé être utilisé. Le nombre maximal de caractères autorisés pour cette chaîne est de cinq, y compris un caractère null de fin.
