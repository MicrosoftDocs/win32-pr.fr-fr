# <a name="national-language-support"></a>Prise en charge des langues nationales

Les fonctions NLS (National Language Support) aident les applications à prendre en charge les différents besoins propres aux langues et aux paramètres régionaux des utilisateurs dans le monde entier. Cette vue d’ensemble décrit les fonctions NLS et explique comment les utiliser dans vos applications.
