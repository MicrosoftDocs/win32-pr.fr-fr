# <a name="locale_ilzero"></a>paramètres régionaux \_ ILZERO

Spécificateur pour les zéros non significatifs dans les champs décimaux.



| Valeur | Signification          |
|-------|------------------|
| 0     | Aucun zéro non significatif |
| 1     | Zéros non significatifs    |
