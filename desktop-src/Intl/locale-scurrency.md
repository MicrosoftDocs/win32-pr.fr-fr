# <a name="locale_scurrency"></a>paramètres régionaux \_ SCURRENCY

Chaîne utilisée comme symbole monétaire local. Le nombre maximal de caractères autorisés pour cette chaîne est 13, y compris un caractère null de fin. Par exemple, dans la États-Unis, ce symbole est « $ ».
