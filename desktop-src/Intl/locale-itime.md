# <a name="locale_itime"></a>paramètres régionaux \_ ITIME

Spécification de format d’heure. La spécification est l’une des valeurs suivantes. Il est préférable que votre application utilise la constante [ \_ sTimeFormat par de paramètres régionaux](locale-stime-constants.md) au lieu des paramètres régionaux \_ ITIME.



| Valeur | Description          |
|-------|----------------------|
| 0     | Format 12 heures AM/PM |
| 1     | format 24 heures       |
