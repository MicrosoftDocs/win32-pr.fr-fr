# <a name="locale_sthousand"></a>paramètres régionaux \_ STHOUSAND

Caractères utilisés pour séparer des groupes de chiffres à gauche du séparateur décimal. Le nombre maximal de caractères autorisés pour cette chaîne est de quatre, y compris un caractère null de fin. En règle générale, ces groupes représentent des milliers. Toutefois, en fonction de la valeur spécifiée pour les [paramètres régionaux \_ SGROUPING](locale-sgrouping.md), ils peuvent représenter autre chose.
