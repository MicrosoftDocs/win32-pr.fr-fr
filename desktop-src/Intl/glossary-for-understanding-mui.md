# <a name="glossary-for-understanding-mui"></a>Glossaire pour comprendre MUI

-   [Globalisation](#globalization)
-   [Facilité de localisation](#localizability)
-   [Localisation](#localization)
-   [Internationalisation](#internationalization)

## <a name="globalization"></a>Globalisation

La *globalisation* des termes indique la nécessité pour la plateforme de fonctionner partout. dans la pratique, cette vision est réalisée dans Windows en ayant un seul fichier binaire mondial pour le système d’exploitation Windows à partir de la version Windows Vista, en prenant en charge le déploiement d’image unique dans un contexte multilingue et en fournissant des api NLS pour récupérer des données spécifiques aux paramètres régionaux.

## <a name="localizability"></a>Facilité de localisation

Le terme d' *adaptabilité* signifie rendre le produit plus facile à localiser, ce qui a pour conséquence de réduire les coûts de localisation. Cela peut être accompli en utilisant efficacement les API de ressources, en contrôlant l’étendue et le nombre de ressources ajoutées au système, et en améliorant les outils utilisés pour travailler avec ces ressources.

## <a name="localization"></a>Localisation

La *localisation* est le processus d’adaptation d’un programme pour un marché local spécifique, qui comprend la traduction de l’interface utilisateur, le redimensionnement des boîtes de dialogue, la personnalisation des fonctionnalités (si nécessaire) et le test des résultats pour s’assurer que le programme fonctionne toujours. Dans le contexte le plus couramment utilisé, la localisation est axée sur la traduction de l’interface utilisateur.

## <a name="internationalization"></a>Internationalisation

Utilisé comme synonyme de globalisation dans certains endroits, ou en tant que globalisation et localisation dans d’autres emplacements, l' *internationalisation* est mieux utilisée pour décrire toutes les activités qui améliorent un produit dans le contexte mondial.
