# <a name="using-ascii-control-codes-0x000d-and-0x000a"></a>Utilisation de codes de contrôle ASCII 0x000D et 0x000A

La norme Unicode ne prescrit pas de significations spécifiques pour les codes de contrôle 0x000D (retour chariot) et 0x000A (saut de code). Il n’est pas nécessaire d’utiliser ces codes en combinaison. S’ils sont utilisés individuellement, l’un ou l’autre ne peut se représenter lui-même ou les deux codes ensemble. L’application doit toujours déterminer ce que ces codes représentent.
