# <a name="locale_sname"></a>paramètres régionaux \_ SNAME

**Windows Vista et versions ultérieures :** Nom des paramètres régionaux, une balise en plusieurs parties pour identifier de manière unique les paramètres régionaux. Le nombre maximal de caractères autorisés pour cette chaîne est 85, y compris un caractère null de fin. La balise est basée sur les conventions de balisage de langue de RFC 4646. Le modèle à utiliser est décrit dans [noms de paramètres régionaux](locale-names.md).
