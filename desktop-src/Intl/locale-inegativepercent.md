# <a name="locale_inegativepercent"></a>paramètres régionaux \_ INEGATIVEPERCENT

**Windows 7 et versions ultérieures :** Modèle de mise en forme de pourcentage négatif pour les paramètres régionaux. Un seul modèle peut être indiqué. Si plusieurs formats sont utilisés pour les paramètres régionaux, choisissez l’option par défaut. Par exemple, si un pourcentage négatif est affiché « -9% » pour « moins de neuf% », le choix approprié pour cette constante est 0.



| Valeur | Format                                                    |
|-------|-----------------------------------------------------------|
| 0     | Signe négatif, nombre, espace, pourcentage ; par exemple,-\# % |
| 1     | Signe négatif, nombre, pourcentage ; par exemple,-\#%         |
| 2     | Signe négatif, pourcentage, nombre ; par exemple,-%\#         |
| 3     | Pourcentage, signe négatif, nombre ; par exemple,%-\#         |
| 4     | Pourcentage, nombre, signe négatif ; par exemple,%\#-         |
| 5     | Nombre, signe négatif, pourcentage ; par exemple, \#-%         |
| 6     | Nombre, pourcentage, signe négatif ; par exemple, \#%-         |
| 7     | Signe négatif, pourcentage, espace, nombre ; par exemple,-% \# |
| 8     | Nombre, espace, pourcentage, signe négatif ; par exemple, \# %- |
| 9     | Pourcentage, espace, nombre, signe négatif ; par exemple,% \#- |
| 10    | Pourcentage, espace, signe négatif, nombre ; par exemple,%-\# |
| 11    | Nombre, signe négatif, espace, pourcentage ; par exemple, \# -% |
