# <a name="locale_snative-constants"></a>Paramètres régionaux \_ SNATIVE \* constantes

Cette rubrique définit les constantes de SNATIVE de paramètres régionaux \_ \* utilisées par nls pour représenter les noms de langages natifs.



| Valeur                       | Signification                                                                                                                                                                                                                                                            |
|-----------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| paramètres régionaux \_ SNATIVECOUNTRYNAME  | **Windows 7 et versions ultérieures :** Nom natif du pays ou de la région, par exemple, España pour l’Espagne. Le nombre maximal de caractères autorisés pour cette chaîne est 80, y compris un caractère null de fin.                                                                 |
| paramètres régionaux \_ SNATIVECTRYNAME     | déconseillé pour Windows 7 et versions ultérieures. Nom natif du pays ou de la région. Consultez Paramètres régionaux \_ SNATIVECOUNTRYNAME.                                                                                                                                                             |
| paramètres régionaux \_ SNATIVECURRNAME     | **Windows Me/98, Windows 2000 :** Nom natif de la devise associée aux paramètres régionaux, dans la langue native des paramètres régionaux. Il n’existe aucune limite quant au nombre de caractères autorisés pour cette chaîne.                                                          |
| paramètres régionaux \_ SNATIVEDIGITS       | Équivalents natifs de ASCII 0 à 9. Le nombre maximal de caractères autorisés pour cette chaîne est onze, y compris un caractère null de fin. Par exemple, l’arabe utilise « 012345 6789 ». Voir aussi [paramètres régionaux \_ IDIGITSUBSTITUTION](locale-idigitsubstitution.md). |
| paramètres régionaux \_ SNATIVEDISPLAYNAME  | **Windows 7 et versions ultérieures :** Nom d’affichage des paramètres régionaux dans sa langue native, par exemple, Deutsch (Deutschland) pour les paramètres régionaux allemand (Allemagne). <br/>                                                                                                        |
| paramètres régionaux \_ SNATIVELANGNAME     | déconseillé pour Windows 7 et versions ultérieures. Nom natif de la langue. Consultez Paramètres régionaux \_ SNATIVELANGUAGENAME.                                                                                                                                                                  |
| paramètres régionaux \_ SNATIVELANGUAGENAME | **Windows 7 et versions ultérieures :** Nom natif de la langue, par exemple, Հայերեն pour l’arménien (Arménie). Le nombre maximal de caractères autorisés pour cette chaîne est 80, y compris un caractère null de fin.                                                         |
