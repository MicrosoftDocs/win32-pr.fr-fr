# <a name="multilingual-user-interface"></a>Interface utilisateur multilingue

interface utilisateur multilingue (MUI) active la localisation des interfaces utilisateur pour les applications globalisées. MUI prend également en charge la création de ressources pour un nombre quelconque de langues d’interface utilisateur.

Cette documentation décrit MUI et vous indique comment ajouter la fonctionnalité MUI à votre application entièrement globalisée. Toutefois, il n’essaie pas de décrire tous les détails du développement d’applications globalisées sur Windows. Pour plus d’informations sur ce type, consultez le [Centre de développement Microsoft Go Global](https://msdn.microsoft.com/goglobal).

La documentation MUI contient les sections suivantes :

-   [à propos de interface utilisateur multilingue](about-multilingual-user-interface.md)
-   [utilisation de interface utilisateur multilingue](using-multilingual-user-interface.md)
-   [interface utilisateur multilingue Faire](multilingual-user-interface-reference.md)
