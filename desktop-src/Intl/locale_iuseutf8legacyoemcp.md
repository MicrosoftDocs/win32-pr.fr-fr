# <a name="locale_iuseutf8legacyoemcp"></a>Paramètres régionaux \_ IUSEUTF8LEGACYOEMCP

**À partir de la prochaine version de Windows :** Il est recommandé que les applications utilisent Unicode à la place. Sur les ordinateurs que l’utilisateur a configurés pour utiliser une page de codes UTF-8, LOCALE_IUSEUTF8LEGACYOEMCP est la page de codes OEM héritée qui aurait été utilisée sur un ordinateur Windows hérité non UTF-8.
