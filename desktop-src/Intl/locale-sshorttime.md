# <a name="locale_sshorttime"></a>paramètres régionaux \_ SSHORTTIME

**Windows 7 et versions ultérieures :** Chaîne de mise en forme de l’heure abrégée pour les paramètres régionaux. Les modèles sont généralement dérivés en supprimant la valeur « SS » (secondes) du modèle de format d’heure longue. Par exemple, si le format d’heure longue est « h :mm : SS TT », le format d’heure abrégée est probablement « h :mm TT ». Cette constante peut spécifier plusieurs formats dans une liste délimitée par des points-virgules. Toutefois, le format d’heure abrégée par défaut doit être la première valeur indiquée.
