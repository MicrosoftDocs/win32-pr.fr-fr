# <a name="locale_ireadinglayout"></a>paramètres régionaux \_ IREADINGLAYOUT

**Windows 7 et versions ultérieures :** Disposition de lecture du texte. Les valeurs possibles sont définies dans le tableau suivant.



| Valeur | Signification                                                                                                                                                             |
|-------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 0     | Lit de gauche à droite, comme pour les paramètres régionaux anglais (États-Unis).                                                                                                 |
| 1     | Lire de droite à gauche, comme pour les paramètres régionaux arabes.                                                                                                                     |
| 2     | Vous pouvez soit lire verticalement de haut en bas avec des colonnes allant de droite à gauche, soit lire des lignes horizontales de gauche à droite, comme pour les paramètres régionaux japonais (Japon). |
| 3     | Lire verticalement de haut en bas avec des colonnes allant de gauche à droite, comme pour les paramètres régionaux mongol (mongol).                                                  |
