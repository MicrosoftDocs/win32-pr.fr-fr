# <a name="locale_stime-constants"></a>Paramètres régionaux \_ stime \* constantes

Cette rubrique définit les \_ \* constantes stime des paramètres régionaux utilisés par nls.



| Valeur               | Signification                                                                                                                                                                                                                                                                                                                                                                                   |
|---------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| paramètres régionaux \_ stime       | Caractère (s) pour le séparateur d’heure. Le nombre maximal de caractères autorisés pour cette chaîne est de quatre, y compris un caractère null de fin. <br/> **Windows Vista et versions ultérieures :** Cette constante est déconseillée. Utilisez les paramètres régionaux sTimeFormat par à la \_ place. Un paramètre régional personnalisé peut ne pas avoir un seul caractère de séparation uniforme. Par exemple, un format tel que « 03:56 ' 23 » est valide.<br/> |
| paramètres régionaux \_ sTimeFormat par | Chaînes de formatage de l’heure pour les paramètres régionaux. Le nombre maximal de caractères autorisés pour cette chaîne est 80, y compris un caractère null de fin. La chaîne peut être constituée d’une combinaison d' [images de format heure, minute et seconde](hour--minute--and-second-format-pictures.md).                                                                                                      |
