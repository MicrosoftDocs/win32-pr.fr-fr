# <a name="locale_ildate"></a>paramètres régionaux \_ ILDATE

Spécificateur de classement de format de date longue. Il est préférable que votre application utilise la constante [ \_ SLONGDATE de paramètres régionaux](locale-slongdate.md) au lieu des paramètres régionaux \_ ILDATE.



| Valeur | Signification        |
|-------|----------------|
| 0     | Mois-jour-année |
| 1     | Jour-mois-année |
| 2     | Année-mois-jour |
