# <a name="about-unicode-and-character-sets"></a>À propos d’Unicode et des jeux de caractères

Les données basées sur les caractères du monde ont été développées à l’aide de jeux de caractères [Unicode](unicode.md) et traditionnels. pour cette raison, Windows fournit des fonctions de jeu de caractères qui permettent aux applications de convertir les données basées sur des caractères de leur jeu de caractères d’origine en Unicode ou un autre jeu de caractères traditionnel. Ces fonctions de jeu de caractères aident également les applications à créer des données basées sur des caractères qui peuvent être transférées et utilisées sur n’importe quel système d’exploitation, y compris celles qui ne prennent pas en charge Unicode.

Pour plus d’informations sur Unicode au-delà de la portée de cette vue d’ensemble, consultez [la norme Unicode](https://www.unicode.org/standard/standard.html).
