# <a name="locale_skeyboardstoinstall"></a>paramètres régionaux \_ SKEYBOARDSTOINSTALL

**Windows Vista et versions ultérieures :** Liste de claviers délimités par des points-virgules à installer potentiellement pour les paramètres régionaux et à utiliser en interne par Windows. Il n’existe aucune limite quant au nombre de caractères autorisés pour cette chaîne. Pour récupérer le nom de l’identificateur de paramètres régionaux d’entrée actif (anciennement appelé disposition de clavier), votre application peut appeler la fonction [**GetKeyboardLayoutName**](/windows/win32/api/winuser/nf-winuser-getkeyboardlayoutnamea) .
