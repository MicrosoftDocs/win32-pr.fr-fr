# <a name="locale_smonthday"></a>paramètres régionaux \_ SMONTHDAY

**Windows 7 et versions ultérieures :** Chaîne de format pour afficher uniquement le mois et le jour. La mise en forme est similaire à celle définie pour les [paramètres régionaux \_ SLONGDATE](locale-slongdate.md). Par exemple, si le modèle de mois/jour est le nom complet du mois suivi du nombre de jours avec des zéros non significatifs, comme dans « septembre 03 », la chaîne de format est « MMMM JJ ». La chaîne peut être constituée d’une combinaison d’images de format de jour et de mois, et de toute chaîne de caractères placée entre guillemets simples. Les caractères des guillemets simples restent tels qu’ils sont spécifiés, par exemple, « de » pour l’espagnol (Espagne). Les paramètres régionaux ne peuvent spécifier qu’un seul format de mois/jour.
