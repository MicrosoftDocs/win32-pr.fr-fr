# <a name="understanding-internationalization"></a><span data-ttu-id="c9dc0-103">Compréhension de l’internationalisation</span><span class="sxs-lookup"><span data-stu-id="c9dc0-103">Understanding Internationalization</span></span>

<span data-ttu-id="c9dc0-104">Le développement de produits logiciels internationaux requiert non seulement la production de code correct, mais également une conception permettant au produit de prendre en charge de nombreux paramètres régionaux.</span><span class="sxs-lookup"><span data-stu-id="c9dc0-104">Developing internationalized software products requires not only the production of correct code, but also a design that permits the product to support many locales.</span></span> <span data-ttu-id="c9dc0-105">Les développeurs et leurs responsables doivent connaître le niveau d’effort et l’attention aux détails nécessaires à la création d’applications mondialisables, qu’elles les diffusent en tant que fichiers binaires uniques prêts à l’emploi sur de nombreux marchés différents ou en tant que fichiers binaires multiples spécifiques à un paramètre régional.</span><span class="sxs-lookup"><span data-stu-id="c9dc0-105">Developers and their managers should be aware of the level of effort and attention to detail required to create world-ready applications, whether they deliver them as single binaries ready for use in many different markets, or as multiple binaries that are each specific to one locale.</span></span> <span data-ttu-id="c9dc0-106">En tant que développeur, assurez-vous que votre gestion comprend que l’internationalisation implique plus que le simple développement de code.</span><span class="sxs-lookup"><span data-stu-id="c9dc0-106">As a developer, make sure your management understands that internationalization involves more than just code development.</span></span> <span data-ttu-id="c9dc0-107">La conception de l’internationalisation au début de votre cycle de produit vous fait gagner du temps, de l’argent et des efforts.</span><span class="sxs-lookup"><span data-stu-id="c9dc0-107">Designing for internationalization at the beginning of your product cycle saves you time, money, and effort.</span></span>

<span data-ttu-id="c9dc0-108">L' **internationalisation** est la création de logiciels qui s’adaptent aux paramètres régionaux dans le monde entier.</span><span class="sxs-lookup"><span data-stu-id="c9dc0-108">**Internationalization** is the creation of software that adapts to locales around the world.</span></span> <span data-ttu-id="c9dc0-109">Les **paramètres régionaux** sont un ensemble de préférences utilisateur liées à la langue.</span><span class="sxs-lookup"><span data-stu-id="c9dc0-109">A **locale** is a collection of language-related user preferences.</span></span> <span data-ttu-id="c9dc0-110">Les paramètres régionaux sont spécifiés par le jumelage d’une langue et d’une région géographique.</span><span class="sxs-lookup"><span data-stu-id="c9dc0-110">A locale is specified by the pairing of a language and a geographical region.</span></span> <span data-ttu-id="c9dc0-111">Par exemple, l’anglais (États-Unis) et l’anglais (Royaume-Uni) sont deux paramètres régionaux différents, tels que l’anglais (Canada) et le français (Canada).</span><span class="sxs-lookup"><span data-stu-id="c9dc0-111">For example, English (United States) and English (United Kingdom) are two different locales, as are English (Canada) and French (Canada).</span></span>

<span data-ttu-id="c9dc0-112">Les logiciels internationaux possèdent les attributs suivants :</span><span class="sxs-lookup"><span data-stu-id="c9dc0-112">Internationalized software has these attributes:</span></span>

-   <span data-ttu-id="c9dc0-113">**Le monde** de la préparation couvre la conception et l’implémentation qui séparent le code indépendant des paramètres régionaux des ressources dépendantes des paramètres régionaux et comprend deux domaines majeurs :</span><span class="sxs-lookup"><span data-stu-id="c9dc0-113">**World-readiness** covers design and implementation that separate locale-independent code from locale-dependent resources, and comprises two major areas:</span></span>
    -   <span data-ttu-id="c9dc0-114">La **globalisation** est le processus de développement d’un noyau de programme avec des fonctionnalités et une conception de code indépendantes de la langue ou des paramètres régionaux.</span><span class="sxs-lookup"><span data-stu-id="c9dc0-114">**Globalization** is the process of developing a program core with features and code design that are independent of language or locale.</span></span> <span data-ttu-id="c9dc0-115">Au lieu de cela, la conception prend en charge l’entrée, l’affichage et la sortie des scripts de langage pris en charge par Unicode, ainsi que les données relatives à des paramètres régionaux spécifiques.</span><span class="sxs-lookup"><span data-stu-id="c9dc0-115">Instead, the design supports the input, display, and output of Unicode-supported language scripts, as well as data related to specific locales.</span></span>
    -   <span data-ttu-id="c9dc0-116">L' **adaptabilité** est la conception de la base de code et des ressources logicielles, de telle sorte qu’un programme peut être localisé, comme indiqué ci-dessous, dans différentes éditions de langue sans aucune modification du code source.</span><span class="sxs-lookup"><span data-stu-id="c9dc0-116">**Localizability** is the design of the software code base and resources such that a program can be localized, as defined below, into different language editions without any changes to the source code.</span></span> <span data-ttu-id="c9dc0-117">Par exemple, les mémoires tampons de chaîne dans le code et les zones de texte de l’interface utilisateur doivent être suffisamment grandes pour prendre en charge des chaînes de texte plus longues dans des langues telles que l’allemand ou le néerlandais.</span><span class="sxs-lookup"><span data-stu-id="c9dc0-117">For example, string buffers in the code, and text boxes in the UI, must be large enough to accommodate longer text strings in languages such as German or Dutch.</span></span>
-   <span data-ttu-id="c9dc0-118">**Localisation** Cela implique la traduction et la personnalisation d’un produit pour un marché spécifique, et il s’agit essentiellement de créer des fichiers de ressources plutôt que d’écrire du code source.</span><span class="sxs-lookup"><span data-stu-id="c9dc0-118">**Localization** This involves translating and customizing a product for a specific market, and it is mainly about creating resource files rather than writing source code.</span></span>

<span data-ttu-id="c9dc0-119">Par exemple, l’utilisation du service NLS (National Language Support) fourni par l’API Microsoft Win32 est un processus de développement de logiciels qui prend en charge le monde, tandis que la modification des éléments de l’interface utilisateur, la traduction de texte et la terminologie de normalisation sont des étapes de localisation généralement effectuées par une personne qui est un spécialiste linguistique, tel qu’un traducteur.</span><span class="sxs-lookup"><span data-stu-id="c9dc0-119">For example, using the National Language Support (NLS) supplied by the Microsoft Win32 API is a software development process that supports world-readiness, whereas modifying the UI elements, translating text, and standardizing terminology are localization steps usually performed by someone who is a language specialist, such as a translator.</span></span> <span data-ttu-id="c9dc0-120">Même lorsque le développement de votre code met l’accent sur le monde, vous devez comprendre la localisation, car vos décisions de conception affectent le travail du traducteur.</span><span class="sxs-lookup"><span data-stu-id="c9dc0-120">Even when your code development emphasizes world-readiness, you need to understand localization because your design decisions affect the work of the translator.</span></span>
