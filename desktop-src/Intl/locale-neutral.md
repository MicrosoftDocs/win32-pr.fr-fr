# <a name="locale_neutral"></a>paramètres régionaux \_ neutres

Paramètres régionaux neutres. Cette constante n’est généralement pas utilisée lors de l’appel d’API NLS. Au lieu de cela, utilisez [paramètres régionaux \_ \_ par](locale-system-default.md) défaut du système ou [paramètres régionaux de l' \_ utilisateur \_ ](locale-user-default.md). La valeur des paramètres régionaux \_ Neutral est 0x0000.
