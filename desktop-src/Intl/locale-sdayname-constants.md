# <a name="locale_sdayname-constants"></a>Paramètres régionaux \_ SDAYNAME \* constantes

Cette rubrique définit les \_ \* constantes SDAYNAME des paramètres régionaux utilisés par nls.



| Valeur             | Signification                                                                                                                                 |
|-------------------|-----------------------------------------------------------------------------------------------------------------------------------------|
| Paramètres régionaux \_ SDAYNAME1 | Nom long natif pour lundi. Le nombre maximal de caractères autorisés pour cette chaîne est 80, y compris un caractère null de fin.    |
| Paramètres régionaux \_ SDAYNAME2 | Nom long natif pour mardi. Le nombre maximal de caractères autorisés pour cette chaîne est 80, y compris un caractère null de fin.   |
| Paramètres régionaux \_ SDAYNAME3 | Nom long natif pour mercredi. Le nombre maximal de caractères autorisés pour cette chaîne est 80, y compris un caractère null de fin. |
| Paramètres régionaux \_ SDAYNAME4 | Nom long natif pour jeudi. Le nombre maximal de caractères autorisés pour cette chaîne est 80, y compris un caractère null de fin.  |
| Paramètres régionaux \_ SDAYNAME5 | Nom long natif pour le vendredi. Le nombre maximal de caractères autorisés pour cette chaîne est 80, y compris un caractère null de fin.    |
| Paramètres régionaux \_ SDAYNAME6 | Nom long natif pour le samedi. Le nombre maximal de caractères autorisés pour cette chaîne est 80, y compris un caractère null de fin.  |
| Paramètres régionaux \_ SDAYNAME7 | Nom long natif pour le dimanche. Le nombre maximal de caractères autorisés pour cette chaîne est 80, y compris un caractère null de fin.    |
