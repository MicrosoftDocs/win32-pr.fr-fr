# <a name="locale_ssort_constants"></a>paramètres régionaux \_ SSORT \_ constantes

Cette rubrique définit les \_ \* constantes SSORT des paramètres régionaux utilisés par nls.



| Valeur               | Signification                                                                                                                                                                                                           |
|---------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| paramètres régionaux \_ SSORTLOCALE | **Windows 7 et versions ultérieures :** Nom des paramètres régionaux à utiliser pour le tri ou le comportement de la casse.                                                                                                                                |
| paramètres régionaux \_ SSORTNAME   | **Windows Me/98, Windows 2000 :** Nom localisé complet du tri pour l’identificateur de paramètres régionaux spécifié, en fonction de la langue de l’interpréteur de commandes. Cette constante est utilisée pour déterminer le comportement de casse et de tri. |
