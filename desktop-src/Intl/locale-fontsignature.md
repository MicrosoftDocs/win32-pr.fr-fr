# <a name="locale_fontsignature"></a>paramètres régionaux \_ fontSignature

**Windows Me/98/95, Windows NT 4,0 :** Modèle binaire spécifique qui détermine la relation entre la couverture de caractères nécessaire pour prendre en charge les paramètres régionaux et le contenu de la police. Notez que \_ les données de paramètres régionaux fontSignature données prennent une forme différente de toutes les autres informations de paramètres régionaux. Toutes les autres informations de paramètres régionaux peuvent être exprimées sous la forme d’une chaîne ou sous la forme d’un nombre. Paramètres régionaux \_ fontSignature les données sont récupérées dans une structure [LOCALESIGNATURE](/windows/win32/api/wingdi/ns-wingdi-localesignature) .
