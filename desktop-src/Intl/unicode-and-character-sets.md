# <a name="unicode-and-character-sets"></a>Unicode et jeux de caractères

Microsoft Windows prend en charge les nombreuses langues écrites de la place de marché internationale grâce aux jeux de caractères [Unicode](unicode.md) et traditionnels.

Unicode est une norme internationale de codage de caractères qui fournit un nombre unique pour représenter chaque caractère utilisé dans l’informatique moderne, y compris les symboles techniques et les caractères spéciaux utilisés dans la publication. Unicode est requis par les normes modernes, telles que XML et ECMAScript (JavaScript), et est le mécanisme officiel d’implémentation de la norme ISO/IEC 10646. Il est pris en charge par de nombreux systèmes d’exploitation, tous les navigateurs modernes et de nombreux autres produits. les nouvelles Windows applications doivent utiliser Unicode pour éviter les incohérences de pages de codes variées et pour faciliter la localisation.

les jeux de caractères traditionnels sont les normes de codage de caractères précédentes, telles que les [pages de codes](code-pages.md) Windows qui utilisent des valeurs de code 8 bits ou des combinaisons de valeurs 8 bits pour représenter les caractères utilisés dans une langue ou une région géographique spécifique.

Cette vue d’ensemble décrit les fonctions de jeu de caractères et explique comment les utiliser dans vos applications.
