# <a name="locale_sdecimal"></a>paramètres régionaux \_ SDECIMAL

Caractère (s) utilisé (s) pour le séparateur décimal, par exemple « . » dans « 3,14 » ou « , » dans « 3, 14 ». Le nombre maximal de caractères autorisés pour cette chaîne est de quatre, y compris un caractère null de fin.
