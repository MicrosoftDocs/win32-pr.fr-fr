# <a name="input-method-manager"></a>Gestionnaire de méthode d’entrée

Le gestionnaire de méthode d’entrée est une technologie utilisée par une application pour communiquer avec un éditeur de méthode d’entrée (IME), qui s’exécute en tant que service. L’IME permet aux utilisateurs de l’ordinateur d’entrer des caractères et des symboles complexes, tels que des caractères Kanji japonais, à l’aide d’un clavier standard.

Cette section décrit l’API IMM et explique comment utiliser la fonctionnalité pour créer et gérer des fenêtres IME. Il comprend les sections suivantes :

-   [À propos du gestionnaire de méthode d’entrée](about-input-method-manager.md)
-   [Utilisation du gestionnaire de méthode d’entrée](using-input-method-manager.md)
-   [Référence du gestionnaire de méthode d’entrée](input-method-manager-reference.md)
