# <a name="locale_iuseutf8legacyacp"></a>Paramètres régionaux \_ IUSEUTF8LEGACYACP

**À partir de la prochaine version de Windows :** Il est recommandé que les applications utilisent Unicode à la place. sur les ordinateurs que l’utilisateur a configurés pour utiliser une page de codes UTF-8, LOCALE_IUSEUTF8LEGACYACP est la page de codes héritée Windows qui aurait été utilisée sur un ordinateur Windows hérité non UTF-8.
