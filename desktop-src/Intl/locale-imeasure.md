# <a name="locale_imeasure"></a>paramètres régionaux \_ IMEASURE

Système de mesure. Le nombre maximal de caractères autorisés pour cette chaîne est deux, y compris un caractère null de fin. Cette valeur est 0 si le système métrique (systéme international d’Units ou S.I.) est utilisé, et 1 si le système États-Unis est utilisé.
