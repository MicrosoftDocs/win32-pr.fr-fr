# <a name="locale_imonlzero"></a>paramètres régionaux \_ IMONLZERO

Spécificateur pour les zéros non significatifs dans les champs de mois pour une date brève uniquement. Il est préférable que votre application utilise la constante [ \_ SSHORTDATE de paramètres régionaux](locale-sshortdate.md) au lieu des paramètres régionaux \_ IMONLZERO.



| Valeur | Signification                     |
|-------|-----------------------------|
| 0     | Aucun zéro non significatif pour les mois |
| 1     | Zéros non significatifs pour les mois    |
