# <a name="candidate-lists"></a>Listes de candidats

Une liste de candidats est représentée par une structure [**CANDIDATELIST**](/windows/win32/api/imm/ns-imm-candidatelist) contenant un tableau de chaînes spécifiant les caractères ou les chaînes de caractères que l’utilisateur peut choisir. Votre application prenant en charge l’IME peut récupérer des informations de liste de candidats à l’aide des fonctions [**ImmGetCandidateListCount**](/windows/desktop/api/Imm/nf-imm-immgetcandidatelistcounta) et [**ImmGetCandidateList**](/windows/desktop/api/Imm/nf-imm-immgetcandidatelista) .
