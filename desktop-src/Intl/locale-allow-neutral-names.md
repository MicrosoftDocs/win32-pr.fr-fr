# <a name="locale_allow_neutral_names"></a>paramètres régionaux- \_ autoriser les \_ noms neutres \_

**Windows 7 et versions ultérieures :** Autorisez le retour de noms neutres ou de LCID lors de la conversion entre les noms de paramètres régionaux et les identificateurs de paramètres régionaux.
