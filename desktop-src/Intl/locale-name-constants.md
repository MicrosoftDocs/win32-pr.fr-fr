# <a name="locale_name-constants"></a>\_Constantes de nom de paramètres régionaux \*

Cette rubrique définit les constantes de nom de paramètres régionaux \_ \* utilisées par nls.




| Valeur | Signification | 
|-------|---------|
| LOCALE_NAME_INVARIANT | Nom d’un paramètre régional indifférent qui fournit des données de paramètres régionaux et de calendrier stables. | 
| LOCALE_NAME_MAX_LENGTH | Longueur maximale d’un nom de paramètres régionaux. Le nombre maximal de caractères autorisés pour cette chaîne est 85, y compris un caractère null de fin.<blockquote>[!Note]<br />Votre application doit utiliser la constante pour la longueur maximale du nom de paramètres régionaux, au lieu de coder en dur la valeur « 85 ».</blockquote><br /> | 
| LOCALE_NAME_SYSTEM_DEFAULT | Nom des paramètres régionaux du système d’exploitation actuel. | 
| LOCALE_NAME_USER_DEFAULT | Nom des paramètres régionaux de l’utilisateur actuel, correspondant à la préférence définie dans la partie Options régionales et linguistiques du panneau de configuration. Ces paramètres régionaux peuvent être différents des paramètres régionaux de la langue actuelle de l’interface utilisateur. |
