# <a name="locale_use_cp_acp"></a>paramètres régionaux \_ use \_ CP \_ ACP

**Windows Me/98, windows 2000 :** Page de codes ANSI Windows par défaut du système (ACP) au lieu de la page de codes des paramètres régionaux utilisée pour la conversion de chaînes. Consultez [identificateurs de page de codes](code-page-identifiers.md) pour obtenir une liste d’ANSI et d’autres pages de codes.
