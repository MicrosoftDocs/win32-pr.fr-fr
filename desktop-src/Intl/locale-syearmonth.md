# <a name="locale_syearmonth"></a>paramètres régionaux \_ SYEARMONTH

**Windows Me/98, Windows 2000 :** Chaîne de mise en forme de mois de l’année pour les paramètres régionaux. Le nombre maximal de caractères autorisés pour cette chaîne est 80, y compris un caractère null de fin. Cette chaîne indique le format approprié pour une chaîne de date qui contient uniquement l’année et le mois, à l’aide d’images de format telles que définies dans les [images de format jour, mois, année et ère](day--month--year--and-era-format-pictures.md).
