# <a name="locale_ipos-constants"></a>Paramètres régionaux \_ IPOS \* constantes

Cette rubrique définit les \_ \* constantes IPOS des paramètres régionaux utilisés par nls.



| Valeur                   | Signification                                                                                                                                                                   |
|-------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| paramètres régionaux \_ IPOSSEPBYSPACE  | Séparation du symbole monétaire dans une valeur monétaire positive. Cette valeur est 1 si le symbole monétaire est séparé par un espace d’un nombre positif, 0 dans le cas contraire.        |
| paramètres régionaux \_ IPOSSIGNPOSN    | Mise en forme de l’index pour les valeurs positives. L’index utilise les mêmes valeurs que les [paramètres régionaux \_ INEGSIGNPOSN](locale-ineg-constants.md), à ceci près qu’il n’utilise pas l’index zéro.    |
| paramètres régionaux \_ IPOSSYMPRECEDES | Position du symbole monétaire dans une valeur monétaire positive. Cette valeur est 1 si le symbole monétaire précède le montant positif, ou 0 si le symbole suit la valeur. |
