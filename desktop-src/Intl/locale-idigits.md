# <a name="locale_idigits"></a>paramètres régionaux \_ IDIGITS

Nombre de chiffres fractionnaires placés après le séparateur décimal. Le nombre maximal de caractères autorisés pour cette chaîne est deux, y compris un caractère null de fin. Par exemple, 2 pour 5,00, 1 pour 5,0.
