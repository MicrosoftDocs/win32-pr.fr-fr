# <a name="locale_windows"></a>paramètres régionaux \_ Windows

**Windows Vista et versions ultérieures :** Énumérer tous les paramètres régionaux fournis avec le système d’exploitation, y compris les paramètres régionaux de remplacement, mais à l’exclusion des tris alternatifs. Pour plus d’informations, consultez [paramètres régionaux personnalisés](custom-locales.md).
