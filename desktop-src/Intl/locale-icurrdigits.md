# <a name="locale_icurrdigits"></a>paramètres régionaux \_ ICURRDIGITS

Nombre de chiffres fractionnaires pour le format monétaire local. Le nombre maximal de caractères autorisés pour cette chaîne est deux, y compris les valeurs 0-9 et un caractère null de fin. Par exemple, 200,00 a une valeur de 2, car il y a deux chiffres fractionnaires, tandis que 200 a la valeur 0, car il n’y a pas de chiffres fractionnaires affichés.
