# <a name="locale_idaylzero"></a>paramètres régionaux \_ IDAYLZERO

Spécificateur pour les zéros non significatifs dans les champs de jour pour une date brève uniquement. Le nombre maximal de caractères autorisés pour cette chaîne est deux, y compris un caractère null de fin. Il est préférable que votre application utilise la constante [ \_ SSHORTDATE de paramètres régionaux](locale-sshortdate.md) au lieu des paramètres régionaux \_ IDAYLZERO.



| Valeur | Signification                   |
|-------|---------------------------|
| 0     | Aucun zéro non significatif pour les jours |
| 1     | Zéros non significatifs pour les jours    |
