# <a name="locale_iintlcurrdigits"></a>paramètres régionaux \_ IINTLCURRDIGITS

\[Les paramètres régionaux \_ IINTLCURRDIGITS ne sont pas pris en charge et peuvent être modifiés ou non disponibles à l’avenir. Utilisez à la place les [paramètres régionaux \_ ICURRDIGITS](locale-icurrdigits.md).\]

Nombre de chiffres fractionnaires pour le format monétaire international. Le nombre maximal de caractères autorisés pour cette chaîne est trois, y compris un caractère null de fin. Par exemple, le nombre de chiffres fractionnaires pour le format monétaire international du dollar des États-Unis est de 2, comme dans $345,25.
