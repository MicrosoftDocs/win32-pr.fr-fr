# <a name="locale_snegativesign"></a>paramètres régionaux \_ SNEGATIVESIGN

Valeur de chaîne pour le signe négatif, par exemple, « - » pour les paramètres régionaux anglais (États-Unis). Le nombre maximal de caractères autorisés pour cette chaîne est de cinq, y compris un caractère null de fin.
