# <a name="locale_itlzero"></a>paramètres régionaux \_ ITLZERO

Spécificateur pour les zéros non significatifs dans les champs d’heure. Il est préférable que votre application utilise la constante [ \_ sTimeFormat par de paramètres régionaux](locale-stime-constants.md) au lieu des paramètres régionaux \_ ITLZERO.



| Valeur | Signification                    |
|-------|----------------------------|
| 0     | Aucun zéro non significatif pour les heures |
| 1     | Zéros non significatifs pour les heures    |
