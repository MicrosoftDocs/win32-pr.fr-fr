# <a name="locale_sdate"></a>paramètres régionaux \_ SDATE

Caractère (s) pour le séparateur de date. Le nombre maximal de caractères autorisés pour cette chaîne est de quatre, y compris un caractère null de fin.

**Windows Vista et versions ultérieures :** Cette constante est déconseillée. Utilisez les [paramètres régionaux \_ SSHORTDATE](locale-sshortdate.md) à la place. Un paramètre régional personnalisé peut ne pas avoir un seul caractère de séparation uniforme. Par exemple, un format tel que « 12/31, 2006 » est valide.
