# <a name="locale_snan"></a>paramètres régionaux \_ SNAN

**Windows Vista et versions ultérieures :** Valeur de chaîne pour « non numérique », par exemple, « Nan » pour les paramètres régionaux anglais (États-Unis). Il n’existe aucune limite quant au nombre de caractères autorisés pour cette chaîne.
