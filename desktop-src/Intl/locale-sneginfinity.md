# <a name="locale_sneginfinity"></a>paramètres régionaux \_ SNEGINFINITY

**Windows Vista et versions ultérieures :** Valeur de chaîne pour « infini négatif », par exemple, « -Infinity » pour les paramètres régionaux anglais (États-Unis). Il n’existe aucune limite quant au nombre de caractères autorisés pour cette chaîne.
