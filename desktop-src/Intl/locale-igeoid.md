# <a name="locale_igeoid"></a>paramètres régionaux \_ IGEOID

Nombre signé 32 bits qui identifie de façon unique un [emplacement géographique](table-of-geographical-locations.md). L’application utilise cette constante pour fournir des services spécifiques aux paramètres régionaux aux clients. Par exemple, il peut être utilisé comme clé pour accéder à un enregistrement de base de données qui contient des informations spécifiques sur un pays ou une région.
