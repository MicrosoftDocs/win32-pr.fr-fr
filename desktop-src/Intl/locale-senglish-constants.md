# <a name="locale_senglish-constants"></a>Paramètres régionaux \_ SENGLISH \* constantes

Cette rubrique définit les \_ \* constantes SENGLISH des paramètres régionaux utilisés par nls.



| Valeur                        | Signification                                                                                                                                                                                                                                                  |
|------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| paramètres régionaux \_ SENGLISHCOUNTRYNAME  | **Windows 7 et versions ultérieures :** Nom anglais du pays ou de la région, par exemple Allemagne pour Deutschland. Ce nom est toujours limité aux caractères qui peuvent être mappés dans le sous-ensemble ASCII 127 caractères. <br/>                                      |
| paramètres régionaux \_ SENGLISHDISPLAYNAME  | **Windows 7 et versions ultérieures :** Nom d’affichage des paramètres régionaux en anglais. En général, le nom complet est constitué de la langue et du pays ou de la région, par exemple allemand (Allemagne) pour Deutsch (Deutschland). <br/>                                           |
| paramètres régionaux \_ SENGLISHLANGUAGENAME | **Windows 7 et versions ultérieures :** Nom anglais de la langue en anglais, par exemple allemand pour Deutsch, à partir de la norme ISO internationale 639. Ce nom est toujours limité aux caractères qui peuvent être mappés dans le sous-ensemble ASCII 127 caractères. <br/> |
