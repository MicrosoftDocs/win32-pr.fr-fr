# <a name="availability-of-mui"></a>Disponibilité de MUI

Les versions suivantes de Windows incluent des API et des constructions pour fournir une prise en charge native du développement d’applications multilingues à l’aide de la technologie MUI :

-   Windows 7 et versions ultérieures.
-   Windows Vista.

Une bibliothèque de niveau inférieur, [**LoadMUILibrary**](/windows/desktop/api/Muiload/nf-muiload-loadmuilibrarya), permet d’écrire des applications multilingues pour les versions de Windows antérieures à Windows Vista tout en fournissant un comportement compatible avec celle des applications MUI écrites pour Windows Vista et les versions ultérieures de Windows.
