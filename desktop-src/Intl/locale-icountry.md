# <a name="locale_idialingcode-and-locale_icountry"></a>Paramètres régionaux \_ IDIALINGCODE et paramètres régionaux \_ ICOUNTRY

Code pays/région, basé sur les codes de téléphone internationaux, également appelés codes pays/région IBM. Le nombre maximal de caractères autorisés pour cette chaîne est de six, y compris un caractère null de fin.

Windows 10 ajout de paramètres régionaux \_ IDIALINGCODE comme synonymes plus précis pour les paramètres régionaux \_ ICOUNTRY. Nous vous encourageons à utiliser le nouveau nom afin d’améliorer la lisibilité du code.

> [!Note]  
> Les codes de numérotation internationaux ne sont pas uniques à un pays particulier. Par exemple, plusieurs pays, y compris le États-Unis et le Canada, utilisent « 1 » comme code de numérotation internationale.
