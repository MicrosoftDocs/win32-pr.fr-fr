# <a name="locale_spercent"></a>paramètres régionaux \_ SPERCENT

**Windows 7 et versions ultérieures :** Symbole utilisé pour indiquer le pourcentage, par exemple, "%". La valeur est une chaîne unique de 0 à 3 caractères.
