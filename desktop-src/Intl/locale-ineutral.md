# <a name="locale_ineutral"></a>paramètres régionaux \_ INEUTRAL

**Windows 7 et versions ultérieures :** Type de paramètres régionaux. Affectez à cette constante la valeur 0 pour des paramètres régionaux spécifiques, ou la valeur 1 pour les paramètres régionaux neutres.
