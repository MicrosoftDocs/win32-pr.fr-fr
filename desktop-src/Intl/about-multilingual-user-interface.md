<span data-ttu-id="392db-104">L’interface utilisateur multilingue (MUI) est une technologie qui fournit aux utilisateurs une interface utilisateur localisée pour les applications globalisées et la gestion des ressources de langue de l’interface utilisateur dans le système d’exploitation Windows.</span><span class="sxs-lookup"><span data-stu-id="392db-104">Multilingual User Interface (MUI) is a technology that provides users a localized user interface for globalized applications and user interface language resource management in the Windows operating system.</span></span> <span data-ttu-id="392db-105">La prise en charge est fournie pour l’ajout de fonctionnalités MUI aux applications globalisées pour s’exécuter sur Windows Vista et versions ultérieures, ainsi que sur de nombreux systèmes d’exploitation antérieurs à Windows Vista.</span><span class="sxs-lookup"><span data-stu-id="392db-105">Support is provided for adding MUI functionality to globalized applications to run on Windows Vista and later, as well as many pre-Windows Vista operating systems.</span></span> <span data-ttu-id="392db-106">Les modèles de gestion des ressources et de localisation MUI améliorent le développement, les tests et la prise en charge des logiciels mondialisables.</span><span class="sxs-lookup"><span data-stu-id="392db-106">The MUI localization and resource management models enhance development, testing, and support for world-ready software.</span></span>

<span data-ttu-id="392db-107">Les fonctionnalités de la technologie MUI sont les suivantes :</span><span class="sxs-lookup"><span data-stu-id="392db-107">Features of MUI technology include:</span></span>

<span data-ttu-id="392db-120">L’avantage le plus visible de MUI est que plusieurs utilisateurs peuvent partager la même station de travail et afficher l’interface utilisateur dans différentes
