# <a name="locale_neutraldata"></a>paramètres régionaux \_ NEUTRALDATA

**Windows 7 et versions ultérieures :** Les données de paramètres régionaux neutres, autrement dit, les données définies par la langue uniquement. Les données de pays/région utilisent la valeur par défaut.
