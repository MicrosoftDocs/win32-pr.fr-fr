# <a name="locale_sintlsymbol"></a>paramètres régionaux \_ SINTLSYMBOL

Trois caractères du symbole monétaire international spécifié dans ISO 4217, suivis du caractère qui sépare cette chaîne de la valeur amount. Le nombre maximal de caractères autorisés pour cette chaîne est neuf, y compris un caractère null de fin.
