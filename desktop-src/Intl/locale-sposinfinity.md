# <a name="locale_sposinfinity"></a>paramètres régionaux \_ SPOSINFINITY

**Windows Vista et versions ultérieures :** Valeur de chaîne pour « infini positif », par exemple, « Infinity » pour les paramètres régionaux anglais (États-Unis). Il n’existe aucune limite quant au nombre de caractères autorisés pour la chaîne.
