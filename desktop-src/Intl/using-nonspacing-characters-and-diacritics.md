# <a name="using-nonspacing-characters-and-diacritics"></a>Utilisation de caractères sans espace et de signes diacritiques

Dans le cas d’une coupure de ligne ou de la séparation du texte, votre application Unicode doit conserver les caractères sans espace avec leurs caractères de base. De nombreux scripts contiennent des caractères qui sont combinés visuellement avec d’autres caractères en sortie, par exemple des marques diacritiques (« accents flottants »).
