# <a name="locale_smonthname-constants"></a>Paramètres régionaux \_ SMONTHNAME \* constantes

Cette rubrique définit les \_ \* constantes SMONTHNAME des paramètres régionaux utilisés par nls.




| Valeur | Signification | 
|-------|---------|
| LOCALE_SMONTHNAME1 | Nom long natif de janvier. Le nombre maximal de caractères autorisés pour cette chaîne est 80, y compris un caractère null de fin.<blockquote>[!Note]<br />L’appel de la fonction <a href="/windows/desktop/api/Winnls/nf-winnls-getlocaleinfoa"><strong>GetLocaleInfo</strong></a> ou <a href="/windows/desktop/api/Winnls/nf-winnls-getlocaleinfoex"><strong>GetLocaleInfoEx</strong></a> avec une constante LOCALE_SMONTHNAME * retourne la forme autonome ou de nom du mois. Pour obtenir la forme génitif du nom du mois, l’application appelle <a href="/windows/desktop/api/datetimeapi/nf-datetimeapi-getdateformata">GetDateFormat</a> ou <a href="/windows/desktop/api/datetimeapi/nf-datetimeapi-getdateformatex">GetDateFormatEx</a> avec une image de date de ddMMMM et supprime les deux chiffres à partir du début de la chaîne récupérée.</blockquote><br /> |
