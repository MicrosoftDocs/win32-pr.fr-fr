# <a name="locale_icalendartype"></a>paramètres régionaux \_ ICALENDARTYPE

Type de calendrier actuel. Les valeurs possibles sont définies dans [informations sur le type de calendrier](calendar-type-information.md). Le type de calendrier peut uniquement représenter un calendrier disponible pour les paramètres régionaux correspondants. Pour récupérer tous les calendriers disponibles pour des paramètres régionaux, l’application peut utiliser les fonctions NLS suivantes :

-   [EnumCalendarInfo](/windows/desktop/api/Winnls/nf-winnls-enumcalendarinfoa)
-   [EnumCalendarInfoEx](/windows/desktop/api/Winnls/nf-winnls-enumcalendarinfoexa)
-   [EnumCalendarInfoExEx](/windows/desktop/api/Winnls/nf-winnls-enumcalendarinfoexex)
