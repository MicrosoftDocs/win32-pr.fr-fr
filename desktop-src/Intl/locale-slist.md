# <a name="locale_slist"></a>paramètres régionaux \_ slist

Caractère (s) utilisé (s) pour séparer les éléments de la liste, par exemple, une virgule est utilisée dans de nombreux paramètres régionaux. Le nombre maximal de caractères autorisés pour cette chaîne est de quatre, y compris un caractère null de fin.
