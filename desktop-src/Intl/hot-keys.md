# <a name="hot-keys"></a>Touches d’accès rapide

Les touches d’accès rapide offrent à l’utilisateur un moyen de modifier rapidement le mode d’entrée d’un IME ou de basculer vers un autre IME. Bien que les applications ne puissent pas ajouter de touches d’accès rapide au système d’exploitation, elles peuvent lancer la même action comme une touche d’accès rapide à l’aide de la fonction [**ImmSimulateHotKey**](/windows/desktop/api/Imm/nf-imm-immsimulatehotkey) .

L’IME HexToUnicode autorise également la conversion entre les caractères hexadécimaux et Unicode. Pour plus d’informations, consultez [HEXTOUNICODE IME](hextounicode-ime.md).
