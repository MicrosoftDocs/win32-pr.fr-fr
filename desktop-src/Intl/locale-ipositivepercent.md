# <a name="locale_ipositivepercent"></a>paramètres régionaux \_ IPOSITIVEPERCENT

**Windows 7 et versions ultérieures :** Modèle de mise en forme du pourcentage positif pour les paramètres régionaux. Un seul modèle peut être indiqué.



| Valeur | Format                                    |
|-------|-------------------------------------------|
| 0     | Nombre, espace, pourcentage ; par exemple, \# % |
| 1     | Nombre, pourcentage ; par exemple, \#%         |
| 2     | Pourcentage, nombre ; par exemple,%\#         |
| 3     | Percent, Space, Number ; par exemple,% \# |
