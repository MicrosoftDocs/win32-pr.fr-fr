# <a name="windows-error-reporting"></a>Rapport d’erreurs Windows

La fonctionnalité de rapport d’erreurs permet aux utilisateurs de signaler à Microsoft les erreurs d’application, les erreurs de noyau, les applications qui ne répondent pas et d’autres problèmes spécifiques à l’application. Microsoft peut utiliser la fonctionnalité de rapport d’erreurs pour fournir aux clients des informations de dépannage, des solutions ou des mises à jour pour leurs problèmes spécifiques. Les développeurs peuvent utiliser cette infrastructure pour recevoir des informations qui peuvent être utilisées pour améliorer leurs applications.

Les utilisateurs peuvent activer le rapport d’erreurs par le biais de l’interface utilisateur Windows. Ils peuvent choisir de signaler les erreurs pour des applications spécifiques. Les administrateurs peuvent remplacer ces paramètres à l’aide de stratégie de groupe.

Les développeurs peuvent s’inscrire auprès du [programme d’application de bureau Windows](../appxpkg/windows-desktop-application-program.md) pour obtenir des informations sur les problèmes rencontrés par les clients avec leurs applications et aider les clients à résoudre ces problèmes. Les développeurs peuvent également utiliser la [récupération et le redémarrage](/windows/desktop/Recovery/application-recovery-and-restart-portal) de l’application pour s’assurer que les clients ne perdent pas de données lorsque leur application se bloque et permettre aux utilisateurs de revenir rapidement à leurs tâches.
