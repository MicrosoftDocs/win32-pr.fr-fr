# <a name="wer-functions"></a>Fonctions WER

Les fonctions suivantes sont utilisées dans Rapport d’erreurs Windows. Pour obtenir des exemples qui utilisent ces fonctions, consultez les exemples dans le SDK Windows sous WinBase \\ WindowsErrorReporting et WinBase \\ Debug \\ FailFast.

| Rubrique                                                                                                               | Description                                                                                                                                                                                                                                                                                                                        |
|---------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**AddERExcludedApplication**](/windows/desktop/api/ErrorRep/nf-errorrep-adderexcludedapplicationa)<br/>                                             | Exclut l’application spécifiée du rapport d’erreurs.<br/>                                                                                                                                                                                                                                                                |
| [*\_événement d' \_ exception d’exécution PFN wer \_ \_*](/windows/desktop/api/Werapi/nc-werapi-pfn_wer_runtime_exception_event)<br/>                          | WER appelle cette fonction pour déterminer si le gestionnaire d’exceptions réclame l’incident.<br/>                                                                                                                                                                                                                               |
| [*\_lancement du \_ \_ \_ débogueur d’exception du runtime \_ PFN wer*](/windows/desktop/api/Werapi/nc-werapi-pfn_wer_runtime_exception_debugger_launch)<br/> | WER appelle cette fonction pour vous permettre de personnaliser les options de lancement du débogueur et de lancer la chaîne.<br/>                                                                                                                                                                                                                             |
| [*\_signature d' \_ événement d’exception du runtime PFN wer \_ \_ \_*](/windows/desktop/api/Werapi/nc-werapi-pfn_wer_runtime_exception_event_signature)<br/>      | WER peut appeler cette fonction plusieurs fois pour récupérer les paramètres de rapport qui décrivent le problème de façon unique.<br/>                                                                                                                                                                                                              |

| [**WerFreeString**](/windows/desktop/api/Werapi/nf-werapi-werfreestring)<br/>                                                                   | Libère la mémoire utilisée pour stocker une chaîne de clé de rapport. Elle doit être appelée après chaque appel successif à [**WerStoreGetFirstReportKey**](/windows/desktop/api/Werapi/nf-werapi-werstoregetfirstreportkey) ou [**WerStoreGetNextReportKey**](/windows/desktop/api/Werapi/nf-werapi-werstoregetnextreportkey), une fois que la chaîne de clé de rapport particulière a été utilisée et n’est plus nécessaire.<br/> |
| [**WerGetFlags**](/windows/desktop/api/Werapi/nf-werapi-wergetflags)<br/>                                                                       | Récupère les paramètres de rapport d’erreurs pour le processus spécifié.<br/>                                                                                                                                                                                                                                                       |
| [**WerRegisterAdditionalProcess**](/windows/desktop/api/Werapi/nf-werapi-werregisteradditionalprocess)<br/>                                     | Inscrit un processus à inclure dans le rapport d’erreurs avec le processus d’application principal. Spécifie éventuellement un thread au sein de ce processus inscrit pour obtenir des données supplémentaires.<br/>                                                                                                                               |
| [**WerRegisterAppLocalDump**](/windows/desktop/api/Werapi/nf-werapi-werregisterapplocaldump)<br/>                                               | Inscrit un chemin d’accès relatif au magasin d’applications local pour l’application appelante dans laquelle Rapport d’erreurs Windows (WER) doit enregistrer une copie de l’image mémoire de diagnostic que WER collecte lorsque l’un des processus de l’application cesse de répondre.<br/>                                                                     |
| [**WerRegisterCustomMetadata**](/windows/desktop/api/Werapi/nf-werapi-werregistercustommetadata)<br/>                                           | Inscrit les métadonnées spécifiques à l’application à collecter (sous forme de chaînes clé/valeur) quand WER crée un rapport d’erreurs.<br/>                                                                                                                                                                                                    |
| [**WerRegisterExcludedMemoryBlock**](/windows/desktop/api/Werapi/nf-werapi-werregisterexcludedmemoryblock)<br/>                                 | Marque un bloc de mémoire (normalement inclus par défaut dans les rapports d’erreurs) à exclure du rapport d’erreurs.<br/>                                                                                                                                                                                                      |
| [**WerRegisterFile**](/windows/desktop/api/Werapi/nf-werapi-werregisterfile)<br/>                                                               | Inscrit un fichier à collecter quand WER crée un rapport d’erreurs.<br/>                                                                                                                                                                                                                                                      |
| [**WerRegisterMemoryBlock**](/windows/desktop/api/Werapi/nf-werapi-werregistermemoryblock)<br/>                                                 | Inscrit un bloc de mémoire à collecter quand WER crée un rapport d’erreurs.<br/>                                                                                                                                                                                                                                              |
| [**WerRegisterRuntimeExceptionModule**](/windows/desktop/api/Werapi/nf-werapi-werregisterruntimeexceptionmodule)<br/>                           | Inscrit un gestionnaire d’exceptions de Runtime personnalisé qui est utilisé pour fournir un rapport d’erreurs personnalisé pour les incidents.<br/>                                                                                                                                                                                                                |
| [**WerRemoveExcludedApplication**](/windows/desktop/api/Werapi/nf-werapi-werremoveexcludedapplication)<br/>                                     | Supprime l’application spécifiée de la liste des applications qui doivent être exclues du rapport d’erreurs.<br/>                                                                                                                                                                                                           |
| [**WerReportAddDump**](/windows/desktop/api/Werapi/nf-werapi-werreportadddump)<br/>                                                             | Ajoute un dump du type spécifié au rapport spécifié.<br/>                                                                                                                                                                                                                                                              |
| [**WerReportAddFile**](/windows/desktop/api/Werapi/nf-werapi-werreportaddfile)<br/>                                                             | Ajoute un fichier au rapport spécifié.<br/>                                                                                                                                                                                                                                                                                    |
| [**WerReportCloseHandle**](/windows/desktop/api/Werapi/nf-werapi-werreportclosehandle)<br/>                                                     | Ferme le rapport spécifié.<br/>                                                                                                                                                                                                                                                                                            |
| [**WerReportCreate**](/windows/desktop/api/Werapi/nf-werapi-werreportcreate)<br/>                                                               | Crée un rapport de problème qui décrit un événement d’application.<br/>                                                                                                                                                                                                                                                           |
| [**WerReportHang**](/windows/desktop/api/ErrorRep/nf-errorrep-werreporthang)<br/>                                                                   | Lance la création de rapports « aucune réponse » dans la fenêtre spécifiée.<br/>                                                                                                                                                                                                                                                              |
| [**WerReportSetParameter**](/windows/desktop/api/Werapi/nf-werapi-werreportsetparameter)<br/>                                                   | Définit les paramètres qui identifient de façon unique un événement pour le rapport spécifié.<br/>                                                                                                                                                                                                                                           |
| [**WerReportSetUIOption**](/windows/desktop/api/Werapi/nf-werapi-werreportsetuioption)<br/>                                                     | Définit les options d’interface utilisateur pour le rapport spécifié.<br/>                                                                                                                                                                                                                                                               |
| [**WerReportSubmit**](/windows/desktop/api/Werapi/nf-werapi-werreportsubmit)<br/>                                                               | Envoie le rapport spécifié.<br/>                                                                                                                                                                                                                                                                                           |

| [**WerStoreClose**](/windows/desktop/api/Werapi/nf-werapi-werstoreclose)<br/>                                                                   | Ferme la collection de rapports stockés.<br/>                                                                                                                                                                                                                                                                                |
| [**WerStoreGetFirstReportKey**](/windows/desktop/api/Werapi/nf-werapi-werstoregetfirstreportkey)<br/>                                           | Obtient une référence au premier rapport dans le magasin de rapports.<br/>                                                                                                                                                                                                                                                               |
| [**WerStoreGetNextReportKey**](/windows/desktop/api/Werapi/nf-werapi-werstoregetnextreportkey)<br/>                                             | Obtient une référence au rapport suivant dans le magasin de rapports d’erreurs.<br/>                                                                                                                                                                                                                                                          |
| [**WerStoreOpen**](/windows/desktop/api/Werapi/nf-werapi-werstoreopen)<br/>                                                                     | Ouvre la collection de rapports d’erreurs stockés.<br/>                                                                                                                                                                                                                                                                           |
| [**WerStoreQueryReportMetadataV2**](/windows/desktop/api/Werapi/nf-werapi-werstorequeryreportmetadatav2)<br/>                                   | Récupère les métadonnées relatives à un rapport dans le magasin.<br/>                                                                                                                                                                                                                                                                         |
| [**WerUnregisterAdditionalProcess**](/windows/desktop/api/Werapi/nf-werapi-werunregisteradditionalprocess)<br/>                                 | Supprime un processus de la liste des processus supplémentaires à inclure dans le rapport d’erreurs.<br/>                                                                                                                                                                                                                             |
| [**WerUnregisterAppLocalDump**](/windows/desktop/api/Werapi/nf-werapi-werunregisterapplocaldump)<br/>                                           | Annule l’inscription effectuée en appelant la fonction [**WerRegisterAppLocalDump**](/windows/desktop/api/Werapi/nf-werapi-werregisterapplocaldump) pour spécifier que rapport D’ERREURS Windows (WER) doit enregistrer une copie de l’image mémoire de diagnostic que wer collecte lorsque l’un des processus de l’application cesse de répondre.<br/>              |
| [**WerUnregisterCustomMetadata**](/windows/desktop/api/Werapi/nf-werapi-werunregistercustommetadata)<br/>                                       | Supprime un élément de métadonnées spécifiques à l’application qui sont collectées pendant le signalement d’erreurs pour l’application.<br/>                                                                                                                                                                                                                    |
| [**WerUnregisterExcludedMemoryBlock**](/windows/desktop/api/Werapi/nf-werapi-werunregisterexcludedmemoryblock)<br/>                             | Supprime un bloc de mémoire qui était précédemment marqué comme exclu (il sera à nouveau inclus dans les rapports d’erreurs).<br/>                                                                                                                                                                                                             |
| [**WerUnregisterFile**](/windows/desktop/api/Werapi/nf-werapi-werunregisterfile)<br/>                                                           | Supprime un fichier de la liste des fichiers à ajouter aux rapports générés pour le processus en cours.<br/>                                                                                                                                                                                                                         |
| [**WerUnregisterMemoryBlock**](/windows/desktop/api/Werapi/nf-werapi-werunregistermemoryblock)<br/>                                             | Supprime un bloc de mémoire de la liste de données à collecter au cours du rapport d’erreurs pour l’application.<br/>                                                                                                                                                                                                                |
| [**WerUnregisterRuntimeExceptionModule**](/windows/desktop/api/Werapi/nf-werapi-werunregisterruntimeexceptionmodule)<br/>                       | Supprime l’inscription de votre gestionnaire d’exceptions WER.<br/>                                                                                                                                                                                                                                                                 |
