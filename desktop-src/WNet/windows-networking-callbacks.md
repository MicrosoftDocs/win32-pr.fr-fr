# <a name="windows-networking-callbacks"></a>Rappels de réseau Windows

Les prototypes de fonction suivants sont utilisés dans la mise en réseau Windows.



| Prototype de fonction                                 | Description                                                                                                           |
|----------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| [**PFNGETPROFILEPATH**](/previous-versions/windows/desktop/legacy/aa385358(v=vs.85))     | Prototype pour une fonction qui obtient un chemin d’accès de profil utilisateur.                                                            |
| [**PFNPROCESSPOLICIES**](/previous-versions/windows/desktop/legacy/aa385360(v=vs.85))   | Prototype pour une fonction qui retourne les informations de stratégie de compte d’utilisateur associées à un processus Windows spécifique. |
| [**PFNRECONCILEPROFILE**](/previous-versions/windows/desktop/legacy/aa385364(v=vs.85)) | Prototype d’une fonction qui rapproche un profil local de son profil global correspondant.                        |
