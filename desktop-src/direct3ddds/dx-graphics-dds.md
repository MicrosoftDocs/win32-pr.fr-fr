# <a name="dds"></a>DDS

Le format de fichier de surface DirectDraw (. DDS) a été introduit avec DirectX 7 pour stocker les textures non compressées et compressées (DXTn). Le format de fichier prend en charge les des mipmaps, les mappages de cube et les mappages de volume. Le format de fichier DDS est pris en charge par [DirectXTex](https://github.com/Microsoft/DirectXTex), [DirectXTK](https://github.com/Microsoft/DirectXTK), D3DX hérité et d’autres outils DirectX. À compter de Direct3D 10, les fichiers DDS prennent en charge les tableaux de texture.

> [!Note]  
> La bibliothèque d’utilitaires D3DX (D3DX 9, D3DX 10 et [D3DX 11](/windows/desktop/direct3d11/d3d11-graphics-reference-d3dx11)) est déconseillée pour Windows 8 et n’est pas prise en charge pour les applications du Windows Store. Nous vous recommandons d’utiliser [DirectXTex](https://github.com/Microsoft/DirectXTex), [DirectXTK](https://github.com/Microsoft/DirectXTK), ou les deux.
