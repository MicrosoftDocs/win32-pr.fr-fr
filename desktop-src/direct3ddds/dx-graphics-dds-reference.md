# <a name="reference-for-dds"></a>Référence pour DDS

La documentation de référence DDS contient les éléments d’API qui définissent la disposition d’un fichier DDS. Ces éléments d’API sont définis dans les fichiers sources **DDSWithoutD3DX** . Cet exemple se trouve dans le `\Samples\C++\Direct3D10\DDSWithoutD3DX\` dossier du kit de développement logiciel (SDK) DirectX (juin 2010).

-   [**\_Facilement en-tête DDS \_**](dds-header-dxt10.md)
