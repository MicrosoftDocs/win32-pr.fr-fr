# <a name="wmdm-samples"></a>Exemples WMDM

le kit de développement logiciel (SDK) Windows Media Gestionnaire de périphériques comprend deux exemples compilables : un exemple d’application de bureau graphique qui vous permet de parcourir des appareils connectés, de créer des sélections et de faire glisser des fichiers vers et à partir d’un appareil. et un exemple de fournisseur de services qui communique avec les « périphériques » virtuels sur l’ordinateur (qui sont en fait les disques durs de votre ordinateur). La documentation explique comment compiler et exécuter ces projets, et fournit également des informations sur leur fonctionnement.

Ces deux projets sont documentés dans les sections suivantes :

-   [Exemple d’application de bureau](sample-desktop-application.md)
-   [Exemple de fournisseur de services](sample-service-provider.md)
