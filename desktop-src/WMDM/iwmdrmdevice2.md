# <a name="iwmdrmdevice2-interface"></a>Interface IWMDRMDevice2

Cette interface n’est pas destinée à être implémentée par un fournisseur de services, mais elle est fournie à des fins de documentation complète.

L’interface **IWMDRMDevice2** étend **IWMDRMDevice** en fournissant des méthodes pour obtenir l’état de licence et une liste de synchronisation partielle.

## <a name="members"></a>Membres

L’interface **IWMDRMDevice2** hérite de [**IWMDRMDevice**](iwmdrmdevice.md). **IWMDRMDevice2** a également les types de membres suivants :

-   [Méthodes](#methods)

### <a name="methods"></a>Méthodes

L’interface **IWMDRMDevice2** possède ces méthodes.



| Méthode                                                         | Description                                     |
|:---------------------------------------------------------------|:------------------------------------------------|
| [**GetLicenseState**](iwmdrmdevice2-getlicensestate.md)       | Obtient l’état de licence.<br/>              |
| [**GetPartialSyncList**](iwmdrmdevice2-getpartialsynclist.md) | Obtient une liste de synchronisation partielle.<br/> |
