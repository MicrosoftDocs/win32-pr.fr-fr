# <a name="interfaces-for-service-providers-and-applications"></a>Interfaces pour les fournisseurs de services et les applications

Les interfaces suivantes peuvent être utilisées par les fournisseurs de services et les applications qui utilisent le kit de développement logiciel (SDK) Windows Media Gestionnaire de périphériques.



| Interface                                                | Description                                                                               |
|----------------------------------------------------------|-------------------------------------------------------------------------------------------|
| [**IWMDMLogger**](/windows/desktop/api/wmdmlog/nn-wmdmlog-iwmdmlogger)                       | Journalise les entrées dans un fichier journal commun.                                                        |
| [**IComponentAuthenticate**](/windows/desktop/api/mswmdm/nn-mswmdm-icomponentauthenticate) | Fournit une communication sécurisée et chiffrée entre les modules de Windows Media Gestionnaire de périphériques. |
