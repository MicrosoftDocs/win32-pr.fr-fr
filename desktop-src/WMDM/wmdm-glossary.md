# <a name="wmdm-glossary"></a>Glossaire WMDM

Pour rechercher un terme dans le glossaire, cliquez sur la lettre de l’alphabet qui correspond à la première lettre du terme que vous souhaitez rechercher.

Vous pouvez également lire les termes du Glossaire dans le texte de l’aide en cliquant sur les liens du Glossaire souligné. Une fois que vous avez cliqué sur un lien de Glossaire, le terme et la définition du Glossaire s’affichent dans une fenêtre indépendante. Pour fermer la fenêtre, cliquez n’importe où sur l’écran.

<dl> <dt>

<span id="projectid114.active_template_library__atl_"></span><span id="PROJECTID114.ACTIVE_TEMPLATE_LIBRARY__ATL_"></span>**Active Template Library (ATL)**
</dt> <dd>

Collection de classes de modèle pour Microsoft Visual C++. Il est utilisé pour créer des serveurs COM lors de l’écriture d’un fournisseur de services pour Windows Media Gestionnaire de périphériques.

</dd> <dt>

<span id="projectid114.atl"></span><span id="PROJECTID114.ATL"></span>**ATL**
</dt> <dd>

Voir la définition de : [Active Template Library (ATL)](/windows)

</dd> <dt>

<span id="projectid114.gls_authentication"></span><span id="PROJECTID114.GLS_AUTHENTICATION"></span>**authentification**
</dt> <dd>

Vérification de l'identité d'un utilisateur, d'un ordinateur, d'un processus ou de toute autre entité via la validation des informations d'identification fournies par l'entité. Les formes courantes d'informations d'identification sont constituées par les signatures numériques, les cartes à puce, les données biométriques et une combinaison de noms et de mots de passe d'utilisateurs.

</dd> <dt>

<span id="projectid114.authorization"></span><span id="PROJECTID114.AUTHORIZATION"></span>**autorisation**
</dt> <dd>

Pour accorder l’accès à une ressource sur un système informatique ou un réseau, généralement lors de l’authentification d’une entité. Une entité authentifiée peut être autorisée pour différents types d’accès ou d’activité.

</dd> <dt>

<span id="projectid114.bit_rate"></span><span id="PROJECTID114.BIT_RATE"></span>**vitesse de transmission**
</dt> <dd>

Nombre de bits transférés par unité de temps, généralement exprimé en bits par seconde.

</dd> <dt>

<span id="projectid114.buffer"></span><span id="PROJECTID114.BUFFER"></span>**mémoire tampon**
</dt> <dd>

Zone de mémoire réservée à la conservation temporaire des données avant que celles-ci ne soient utilisées par un périphérique ou une application de réception. La mise en mémoire tampon protège contre l’interruption du Workflow.

</dd> <dt>

<span id="projectid114.challenge"></span><span id="PROJECTID114.CHALLENGE"></span>**défi**
</dt> <dd>

Demande de licence. Une stimulation contient des informations sur l’ordinateur du consommateur, une liste des droits demandés et d’autres informations sur le contenu, y compris l’en-tête de contenu et l’ID de clé. Ce défi peut également contenir un défi de la version 1 au cas où une licence de version 1 est nécessaire.

</dd> <dt>

<span id="projectid114.clear_content"></span><span id="PROJECTID114.CLEAR_CONTENT"></span>**effacer le contenu**
</dt> <dd>

Fichier Windows Media non chiffré.

</dd> <dt>

<span id="projectid114.client"></span><span id="PROJECTID114.CLIENT"></span>**client**
</dt> <dd>

Tout ordinateur ou programme qui se connecte à ou demande les services d’un autre ordinateur ou programme. Le client peut également faire référence au logiciel qui permet à l’ordinateur ou au programme d’établir la connexion.

</dd> <dt>

<span id="projectid114.content"></span><span id="PROJECTID114.CONTENT"></span>**contenu**
</dt> <dd>

Audio, vidéo, images, texte ou toute autre information contenue dans un fichier multimédia ou un flux numérique.

</dd> <dt>

<span id="projectid114.decrypt"></span><span id="PROJECTID114.DECRYPT"></span>**déchiffrer**
</dt> <dd>

Pour reconvertir le contenu chiffré dans sa forme d’origine.

</dd> <dt>

<span id="projectid114.digital_rights_management__drm_"></span><span id="PROJECTID114.DIGITAL_RIGHTS_MANAGEMENT__DRM_"></span>**gestion des droits numériques (DRM)**
</dt> <dd>

Toute technologie utilisée pour protéger les intérêts des propriétaires de contenu numérique et de services (tels que les propriétaires de droits d’auteur). En règle générale, les destinataires ou les utilisateurs autorisés doivent acquérir une licence pour pouvoir utiliser le contenu.

</dd> <dt>

<span id="projectid114.drm"></span><span id="PROJECTID114.DRM"></span>**DRM**
</dt> <dd>

Voir la définition de : [gestion des droits numériques (DRM)](/windows)

</dd> <dt>

<span id="projectid114.encrypt"></span><span id="PROJECTID114.ENCRYPT"></span>**chiffrer**
</dt> <dd>

Pour déguiser le contenu par programmation pour masquer sa substance.

</dd> <dt>

<span id="projectid114.firewall"></span><span id="PROJECTID114.FIREWALL"></span>**pare-feu**
</dt> <dd>

Solution de sécurité qui sépare une partie d’un réseau d’une autre partie, en autorisant uniquement le trafic réseau autorisé à traverser conformément aux règles de filtrage du trafic.

</dd> <dt>

<span id="projectid114.key"></span><span id="PROJECTID114.KEY"></span>**clé**
</dt> <dd>

Élément de données requis pour déverrouiller un fichier Windows Media empaqueté. Cette clé est incluse dans une licence distincte.

</dd> <dt>

<span id="projectid114.license"></span><span id="PROJECTID114.LICENSE"></span>**licence**
</dt> <dd>

Données jointes au contenu protégé qui décrit comment le contenu peut être utilisé.

</dd> <dt>

<span id="projectid114.microsoft_secure_content_provider__scp_"></span><span id="PROJECTID114.MICROSOFT_SECURE_CONTENT_PROVIDER__SCP_"></span>**Fournisseur de contenu sécurisé Microsoft (SCP)**
</dt> <dd>

Un plug-in pour Windows Media Gestionnaire de périphériques qui fournit des services de gestion des droits numériques (DRM).

</dd> <dt>

<span id="projectid114.microsoft_windows_media_player"></span><span id="PROJECTID114.MICROSOFT_WINDOWS_MEDIA_PLAYER"></span>**Lecteur Microsoft Windows Media**
</dt> <dd>

Client/contrôle qui reçoit un flux à partir d’un serveur Windows Media ou d’un contenu local pour la lecture. Il peut être exécuté en tant que programme exécutable client autonome ou être incorporé dans une page Web, un programme C++ ou un programme Microsoft Visual Basic qui utilise le contrôle ActiveX client.

</dd> <dt>

<span id="projectid114.object"></span><span id="PROJECTID114.OBJECT"></span>**objet**
</dt> <dd>

Collection de propriétés et de méthodes qui remplissent un but particulier et qui sont traitées comme une unité.

</dd> <dt>

<span id="projectid114.player"></span><span id="PROJECTID114.PLAYER"></span>**lecteur**
</dt> <dd>

Programme ou contrôle client qui reçoit du contenu multimédia numérique diffusé en continu à partir d’un serveur ou lu à partir de fichiers locaux. Le lecteur Windows Media est un exemple de lecteur.

</dd> <dt>

<span id="projectid114.plug_in"></span><span id="PROJECTID114.PLUG_IN"></span>**Plug-** in
</dt> <dd>

Composant logiciel auxiliaire qui étend ou améliore les fonctionnalités d’autres logiciels.

</dd> <dt>

<span id="projectid114.portable_device"></span><span id="PROJECTID114.PORTABLE_DEVICE"></span>**appareil mobile**
</dt> <dd>

Appareil électronique mobile qui peut échanger des fichiers ou d’autres données avec un ordinateur ou un périphérique. Les Pocket PC, les lecteurs de musique numériques portables et les smartphones sont des exemples d’appareils mobiles.

</dd> <dt>

<span id="projectid114.portable_media_identifier"></span><span id="PROJECTID114.PORTABLE_MEDIA_IDENTIFIER"></span>**identificateur de média portable**
</dt> <dd>

Identificateur unique pour les supports de stockage amovibles, tels que les cartes CompactFlash. L’identificateur est codé en dur dans le support de stockage et ne peut pas être modifié.

</dd> <dt>

<span id="projectid114.protect"></span><span id="PROJECTID114.PROTECT"></span>**protéger**
</dt> <dd>

Pour chiffrer les fichiers avec une clé et ajouter des informations telles que l’URL d’acquisition de licence.

</dd> <dt>

<span id="projectid114.sdmi"></span><span id="PROJECTID114.SDMI"></span>**SDMI**
</dt> <dd>

Consultez la définition de : l' [initiative de Secure Digital Music (SDMI)](/windows)

</dd> <dt>

<span id="projectid114.secure_digital_music_initiative__sdmi_"></span><span id="PROJECTID114.SECURE_DIGITAL_MUSIC_INITIATIVE__SDMI_"></span>**Initiative de sécuriser la musique numérique (SDMI)**
</dt> <dd>

Organisation qui définit des normes pour la musique numérique sécurisée. L’un des principaux objectifs de SDMI est de créer une infrastructure pour sécuriser la diffusion, le stockage et la distribution de la musique numérique.

</dd> <dt>

<span id="projectid114.windows_media_encoder"></span><span id="PROJECTID114.WINDOWS_MEDIA_ENCODER"></span>**Encodeur Windows Media**
</dt> <dd>

Technologie qui compresse l’audio et la vidéo en direct ou préenregistrés dans un flux Windows Media, qui peut être distribué immédiatement ou enregistré en tant que fichier Windows Media pour une distribution ultérieure.

</dd> <dt>

<span id="projectid114.windows_media_file"></span><span id="PROJECTID114.WINDOWS_MEDIA_FILE"></span>**Fichier Windows Media**
</dt> <dd>

Fichier qui contient des données audio, vidéo ou de script. Le contenu du fichier est encodé à l’aide de l’un des codecs Windows Media.

</dd> <dt>

<span id="projectid114.windows_media_format"></span><span id="PROJECTID114.WINDOWS_MEDIA_FORMAT"></span>**Format Windows Media**
</dt> <dd>

Format d’un fichier multimédia numérique ou d’un flux encodé à l’aide de codecs Windows Media.

</dd> <dt>

<span id="projectid114.windows_media_services"></span><span id="PROJECTID114.WINDOWS_MEDIA_SERVICES"></span>**Media Services Windows**
</dt> <dd>

Service utilisé pour diffuser des données audio et vidéo numériques sur Internet ou sur un intranet.

</dd> </dl>
