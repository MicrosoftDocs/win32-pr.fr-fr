# <a name="privacy-statement-windows-media-device-manager-11-sdk"></a>Déclaration de confidentialité (kit de développement logiciel (SDK) Windows Media Gestionnaire de périphériques 11)

Les développeurs qui créent des applications Windows Media Gestionnaire de périphériques doivent communiquer clairement à leurs utilisateurs comment leur confidentialité pourrait être affectée. Pour vous aider à créer une déclaration de confidentialité de l’utilisateur final, Microsoft a publié des instructions sur la confidentialité sur la [page Web de confidentialité de Windows Media](https://www.microsoft.com/windows/windowsmedia/privacy/10ssdk.aspx).
