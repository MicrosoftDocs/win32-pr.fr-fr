# <a name="component-authentication"></a>Authentification du composant

Toutes les applications et tous les fournisseurs de services doivent être authentifiés avant de pouvoir commencer à appeler ou à implémenter des méthodes Gestionnaire de périphériques Windows Media.

L’authentification est gérée différemment pour les fournisseurs de services et les applications. Consultez la rubrique qui s’applique à votre projet :

-   [Authentification de l’application](authenticating-the-application.md)
-   [Authentification du fournisseur de services](authenticating-the-service-provider.md)
