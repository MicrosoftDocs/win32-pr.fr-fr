# <a name="windows-media-drm-implemented-interfaces"></a>Interfaces Windows Media DRM-Implemented

Les interfaces suivantes ne sont pas destinées à être implémentées par des fournisseurs de services tiers, mais elles sont implémentées et appelées en interne par le Windows Media DRM et Windows Media Gestionnaire de périphériques. L’implémentation de ces interfaces empêchera le traitement du contenu protégé par DRM. Elles sont documentées simplement à des fins d’exhaustivité.

-   [**Interface IWMDRMDevice**](iwmdrmdevice.md)
-   [**Interface IWMDRMDevice2**](iwmdrmdevice2.md)
