# <a name="windows-media-drm-implemented-interfaces"></a>Windows Interfaces de DRM-Implemented de média

les interfaces suivantes ne sont pas destinées à être implémentées par des fournisseurs de services tiers, mais elles sont implémentées et appelées en interne par le Windows media DRM et Windows Gestionnaire de périphériques de média. L’implémentation de ces interfaces empêchera le traitement du contenu protégé par DRM. Elles sont documentées simplement à des fins d’exhaustivité.

-   [**Interface IWMDRMDevice**](iwmdrmdevice.md)
-   [**Interface IWMDRMDevice2**](iwmdrmdevice2.md)
