# <a name="wmdm-programming-guide"></a>Guide de programmation WMDM

cette section décrit les actions qu’une application ou un fournisseur de services reposant sur Windows Media Gestionnaire de périphériques doit effectuer.

-   [Tâches communes aux applications et aux fournisseurs de services](tasks-common-to-applications-and-service-providers.md) décrit les tâches que les applications et les fournisseurs de services effectuent, telles que la journalisation, le chiffrement et le déchiffrement, ainsi que l’authentification.
-   la [création d’une application Windows Media Gestionnaire de périphériques](creating-a-windows-media-device-manager-application.md) décrit toutes les actions qu’une application doit généralement effectuer.
-   La [création d’un fournisseur de services](creating-a-service-provider.md) décrit toutes les actions qu’un fournisseur de services doit généralement effectuer.
