# <a name="compiling-the-sample-service-provider-using-nmake"></a>Compilation de l’exemple de fournisseur de services à l’aide de NMAKE

pour compiler l’exemple de fournisseur de services à l’aide de NMAKE, ouvrez une fenêtre de commande Visual Studio, accédez à <*chemin d’installation du kit de développement logiciel* > \\ WMFSDK95 \\ WMDM \\ mdsp, puis entrez `nmake` à l’invite.
