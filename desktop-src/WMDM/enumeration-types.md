# <a name="wmdm-enumerations"></a>Énumérations WMDM

Cette rubrique documente une fonctionnalité du kit de développement logiciel (SDK) Gestionnaire de périphériques Windows Media. Nous vous recommandons de migrer votre application pour utiliser l’API des appareils mobiles Windows. Pour plus d’informations, consultez le kit de développement logiciel Windows Mobile Devices.

Windows Media Gestionnaire de périphériques définit les énumérations suivantes.



| Type d'énumération                                                                  | Description                                                                                     |
|-----------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------|
| [**\_ \_ \_ formulaire valeurs valides \_ de l’énumération WMDM \_**](wmdm-enum-prop-valid-values-form.md) | Décrit les formes possibles des valeurs valides pour une propriété.                                        |
| [**\_étendue de recherche de WMDM \_**](wmdm-find-scope.md)                                      | Définit la portée de l’objet de stockage.                                                        |
| [**WMDM \_ FORMATCODE**](wmdm-formatcode.md)                                       | Définit une liste de codes de format qui décrivent les types de contenu transférés vers et à partir d’un appareil. |
| [**\_type de session WMDM \_**](wmdm-session-type.md)                                  | Définit le type de session.                                                                       |
| [**\_mode d' \_ énumération de stockage WMDM \_**](wmdm-storage-enum-mode.md)                       | Définit la manière dont le contenu sur le stockage doit être énuméré.                                     |
| [**\_type de données de balise WMDM \_**](wmdm-tag-datatype.md)                                  | Définit un type de données.                                                                            |
| [**WMDMMessage**](wmdmmessage.md)                                                | Définit les types de messages et les États.                                                               |
