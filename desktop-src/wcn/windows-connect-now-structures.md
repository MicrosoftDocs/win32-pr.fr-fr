# <a name="windows-connect-now-structures"></a>Structures Windows Connect Now

Les structures suivantes prennent en charge les fonctions de l’API Windows Connect Now.



| Structure                                                                             | Description                                                                                                                                            |
|---------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**type de valeur WCN type d' \_ \_ \_ \_ appareil principal \_**](/windows/win32/api/wcntypes/ns-wcntypes-wcn_value_type_primary_device_type) | Contient des données énumérées qui identifient le type d’un appareil par catégorie, sous-catégorie et OUI spécifique au fabricant (ID d’organisation).<br/> |
| [**spécification de l' \_ extension du fournisseur WCN \_ \_**](/windows/win32/api/wcndevice/ns-wcndevice-wcn_vendor_extension_spec)                     | Contient des données qui définissent une extension de fournisseur. <br/>                                                                                             |
