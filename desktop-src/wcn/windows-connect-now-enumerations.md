# <a name="windows-connect-now-enumerations"></a>Windows Connect Now Énumérations

Windows Connect Now définit les types d’énumération suivants.



| Énumération                                                                                             | Description                                                                                                                   |
|---------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------|
| [**\_type d’attribut WCN \_**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_attribute_type)                                                      | Définit les différents types de mémoires tampons d’attributs définis pour Wi-Fi installation protégée.<br/>                                      |
| [**\_type de mot de passe WCN \_**](/windows/win32/api/wcndevice/ne-wcndevice-wcn_password_type)                                                        | Définit l’authentification qui sera utilisée dans une session WPS. <br/>                                                    |
| [**\_État de session WCN \_**](/windows/win32/api/wcndevice/ne-wcndevice-wcn_session_status)                                                      | Définit l’état d’une session WPS.<br/>                                                                               |
| [**État de l’Association de type de \_ valeur WCN \_ \_ \_**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_association_state)                        | Définit les États d’association possibles d’une station sans fil pendant une demande de découverte.<br/>                          |
| [**\_type de valeur WCN \_ \_ booléen**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_boolean)                                             | Définit les valeurs utilisées pour représenter les conditions true/false rencontrées pendant la configuration de l’appareil et l’Association.<br/>            |
| [**type de valeur WCN type \_ \_ \_ d’authentification \_**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_authentication_type)                    | Définit les méthodes d’authentification prises en charge par l’inscription (point d’accès ou station).<br/>                            |
| [**\_méthodes de \_ configuration de type de valeur WCN \_ \_**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_config_methods)                              | Définit les méthodes de configuration prises en charge par l’inscription ou le Bureau d’enregistrement.<br/>                                          |
| [**\_erreur de \_ configuration du type de valeur WCN \_ \_**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_configuration_error)                    | Définit les valeurs d’erreur possibles retournées à un appareil lors de la tentative de configuration de sur le réseau local sans fil et de l’associer à celui-ci.<br/> |
| [**\_ \_ type de valeur \_ WCN \_ ID de mot de passe de l’appareil \_**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_device_password_id)                     | Définit les valeurs de mot de passe possibles.<br/>                                                                              |
| [**\_type de \_ \_ chiffrement de type de valeur WCN \_**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_encryption_type)                            | Définit les types de chiffrement WLAN pris en charge.<br/>                                                                       |
| [**\_ \_ bandes RF de type de valeur WCN \_ \_**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_rf_bands)                                          | Définit la bande de fréquence radio sur laquelle un inscrit envoie des demandes de découverte.<br/>                                    |
| [**\_version du \_ type de valeur WCN \_**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_version)                                             | Définit la version prise en charge de Wi-Fi installation protégée (WPS).<br/>                                                      |
| [**TYPE de valeur WCN-état de l' \_ \_ \_ \_ \_ \_ installation protégée Wi-Fi \_**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_wi_fi_protected_setup_state) | Définit des valeurs qui indiquent si un appareil est configuré.<br/>                                                            |
