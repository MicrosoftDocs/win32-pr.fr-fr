# <a name="windows-connect-now-interfaces"></a>Windows Connect Now Interface

l' [API Windows Connect Now](portal.md) (WCNAPI) définit et utilise les interfaces suivantes.



| Interface                                      | Description                                                                                                |
|------------------------------------------------|------------------------------------------------------------------------------------------------------------|
| [**IWCNConnectNotify**](/windows/desktop/api/WcnDevice/nn-wcndevice-iwcnconnectnotify) | recevez une notification de réussite ou d’échec lorsqu’une session de connexion Windows Connect Now se termine.<br/> |
| [**IWCNDevice**](/windows/desktop/api/WcnDevice/nn-wcndevice-iwcndevice)               | Configure l’appareil et lance la session.<br/>                                                |
