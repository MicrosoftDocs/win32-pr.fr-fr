# <a name="windows-connect-now-interfaces"></a>Interfaces Windows Connect Now

L' [API Windows Connect Now](portal.md) (WCNAPI) définit et utilise les interfaces suivantes.



| Interface                                      | Description                                                                                                |
|------------------------------------------------|------------------------------------------------------------------------------------------------------------|
| [**IWCNConnectNotify**](/windows/desktop/api/WcnDevice/nn-wcndevice-iwcnconnectnotify) | Recevez une notification de réussite ou d’échec lorsqu’une session Windows Connect Now se termine.<br/> |
| [**IWCNDevice**](/windows/desktop/api/WcnDevice/nn-wcndevice-iwcndevice)               | Configure l’appareil et lance la session.<br/>                                                |
