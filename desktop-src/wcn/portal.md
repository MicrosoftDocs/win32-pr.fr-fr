# <a name="windows-connect-now"></a>Windows Connect Now

## <a name="purpose"></a>Objectif

Windows Connect Now (WCN) permet aux appareils mobiles et intégrés, aux points d’accès 802,11 (APs) et aux ordinateurs de se connecter en toute sécurité et d’échanger des paramètres entre eux. WCN est conçu pour les particuliers ou les petites entreprises, ce qui constitue un compromis raisonnable entre facilité d’utilisation et sécurité renforcée.

## <a name="developer-audience"></a>Développeurs concernés

l’API Windows Connect Now est conçue pour être utilisée avec C/C++.

## <a name="run-time-requirements"></a>Conditions d’exécution

Windows Connect Now est pris en charge à partir de Windows 7.
