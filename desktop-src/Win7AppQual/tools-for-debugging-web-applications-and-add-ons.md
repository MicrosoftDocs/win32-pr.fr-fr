# <a name="tools-for-debugging-web-applications-and-add-ons"></a>Outils de débogage d’applications et de Add-Ons Web

Au lieu de [résoudre les problèmes de compatibilité des applications Web et des modules](remediating-web-applications-and-add-ons.md)complémentaires, vous pouvez faire en sorte que l’application Web ou le site Web fonctionne en mode natif dans Windows Internet Explorer 8 en mode standard en écrivant votre code dans les normes Web les plus récentes.

Pour évaluer entièrement les problèmes de votre application ou de votre site Web, vous devez rechercher la cause racine du problème. Par exemple, l’équipe Internet Explorer a déterminé les causes racines en utilisant le Outils de développement et en réduisant logiquement les problèmes éventuels. Les sections suivantes décrivent plusieurs outils pour vous aider à simplifier le processus de débogage.



| Section                                                                                                                    | Description                                                                                                                                                                         |
|----------------------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Microsoft Expression Web SuperPreview](microsoft-expression-web-superpreview.md)                                         | Outil de débogage visuel autonome qui accélère et simplifie la migration de vos sites de Microsoft Internet Explorer 6 vers Windows Internet Explorer 7 ou Internet Explorer 8. |
| [Outil de test de compatibilité d'Internet Explorer (IECTT)](inernet-explorer-compatibility-test-tool--iectt-.md)                  | Composant de [Microsoft Application Compatibility Toolkit](/windows-hardware/get-started/adk-install) qui peut vous aider à diagnostiquer les problèmes dans vos applications Web.       |
| [Outils de développement Internet Explorer 8](internet-explorer-8-developer-tools.md)                                             | Les outils de développement inclus dans Internet Explorer 8 pour le débogage.                                                                                                         |
| [Assistant Compatibilité Internet Explorer 8 (Aggiorno)](inernet-explorer-8-compatibility-wizard--aggiorno-.md)              | Outil tiers qui met automatiquement à niveau les sites Web pour qu’ils s’affichent correctement dans Internet Explorer 8 sans aucune compatibilité avec les navigateurs plus anciens.                                |
| [Outil de débogage Web Fiddler](fiddler-web-debugger-tool.md)                                                                 | Un outil tiers pour capturer le trafic réseau entre Internet et les ordinateurs de test.                                                                                           |
| [Test de compatibilité des applications à l'aide d'images Virtual PC](application-compatibility-testing-using-virtual-pc-images.md) | Images Virtual PC qui simplifient les tests de compatibilité.                                                                                                                              |
