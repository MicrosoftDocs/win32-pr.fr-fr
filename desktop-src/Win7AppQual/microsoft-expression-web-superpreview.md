# <a name="microsoft-expression-web-superpreview"></a>Microsoft Expression Web SuperPreview

Microsoft Expression Web SuperPreview pour Internet Explorer est un outil de débogage visuel autonome qui vous aide à déboguer vos sites Web pour la compatibilité entre les navigateurs. L’aperçu Web garantit que vos sites fonctionnent correctement dans Windows Internet Explorer 8 et qu’il assure la compatibilité avec les versions antérieures de Windows Internet Explorer.

L’aperçu Web affiche le rendu de vos pages Web dans Microsoft Internet Explorer 6 et Windows Internet Explorer 7 ou Internet Explorer 8, en fonction de la version que vous installez sur votre ordinateur. Vous pouvez afficher les pages côte à côte ou sous forme de superposition. Vous pouvez utiliser des règles, des repères et des outils de zoom et de panoramique pour identifier précisément les différences de disposition. Vous pouvez même comparer votre page à la façon dont les navigateurs ciblés affichent la page.

L’aperçu Web identifie également la balise, la taille, la position, les styles appliqués et l’emplacement d’un élément dans l’arborescence du Document Object Model (DOM), ce qui vous permet de résoudre rapidement l’erreur.

> [!Note]  
> L’aperçu Web est une application gratuite qui n’a pas d’expiration ni de support technique de Microsoft.

 

Avec l’outil Web SuperPreview, vous pouvez :

-   Déboguez les pages sur votre ordinateur de développement. L’aperçu Web fonctionne directement avec les navigateurs Microsoft Windows installés et comprend le moteur de rendu d’Internet Explorer 6. Vous pouvez utiliser ce moteur côte à côte avec d’autres versions d’Internet Explorer ou de Mozilla Firefox.
-   Déboguez plus rapidement à l’aide des informations d’arborescence d’éléments et DOM. L’aperçu Web combine également le rendu d’image avec les données DOM, ce qui vous permet de mettre en surbrillance des éléments DOM spécifiques dans l’image et d’afficher les différences entre les navigateurs. L’outil fournit des informations sur l’élément, telles que la position, les dimensions et l’ID, pour vous aider à trouver l’élément à l’origine du problème de compatibilité.
-   Prise en charge des navigateurs Windows et Macintosh, y compris Internet Explorer 6 et versions ultérieures, Firefox 3,6 et Macintosh Safari.

Pour plus d’informations sur cet outil et d’autres outils de développement, consultez la [vue d’ensemble d’expression Studio 4 Web](https://www.microsoft.com/expression/products/StudioWebPro_Overview.aspx), [Expression Web Demo](https://msdn.microsoft.com/expression/ff723803.aspx)et [expression forums](https://social.msdn.microsoft.com/forums/announce/threads).
