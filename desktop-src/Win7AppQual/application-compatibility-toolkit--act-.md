# <a name="application-compatibility-toolkit-act"></a>Outils d'analyse de compatibilité des applications (ACT)

> [!IMPORTANT]
> les versions de Shared Computer Toolkit de compatibilité des applications traitées dans cet article ne sont plus prises en charge. la dernière version prise en charge est incluse dans le [Kit de déploiement et d’évaluation Windows 10](/archive/blogs/yongrhee/download-windows-10-assessment-and-deployment-kit-adk)

## <a name="affected-platforms"></a>Plateformes affectées

**Clients :** Windows XP \| Windows Vista \| Windows 7  


**serveurs :** Windows server 2003 \| Windows server 2008 \| Windows server 2008 R2  


## <a name="description"></a>Description

Microsoft application compatibility Shared Computer Toolkit (ACT) est un outil de gestion du cycle de vie qui vous aide à identifier et à gérer votre portefeuille d’applications global, à réduire le coût et le temps nécessaires à la résolution des problèmes de compatibilité des applications et à déployer rapidement Windows et les mises à jour Windows.

**Utilisation**

Avec ACT, vous pouvez :

-   Analyser votre portefeuille d’applications, de sites Web et d’ordinateurs
-   Évaluer les déploiements de système d’exploitation, l’impact des mises à jour du système d’exploitation et la compatibilité avec les sites Web
-   Gérer de manière centralisée les évaluateurs de compatibilité et les paramètres de configuration
-   Rationaliser et organiser les applications, les sites Web et les ordinateurs
-   Hiérarchiser les efforts de compatibilité des applications avec les rapports filtrés
-   Ajoutez et gérez des problèmes et des solutions pour votre environnement informatique d’entreprise
-   Déployer des atténuations automatisées pour les problèmes de compatibilité connus
-   Envoyer et recevoir des informations de compatibilité à partir de la Exchange de compatibilité Microsoft
