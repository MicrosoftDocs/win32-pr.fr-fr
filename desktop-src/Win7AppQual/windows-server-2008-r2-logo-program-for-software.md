# <a name="windows-server-2008-r2-logo-program-for-software"></a>Programme de logo Windows Server 2008 R2 pour les logiciels

## <a name="affected-platforms"></a>Plateformes affectées

 **Serveurs-** Windows Server 2008 R2  

### <a name="description"></a>Description

Le programme de logo Windows Server 2008 R2 a été mis hors service.

Les envois de test Windows Server 2008 R2 2012 ne sont plus acceptés pour les compétences de Microsoft Partner Network (MPN) ou le logo certifié pour Windows Server.

Windows Server 2012 R2 est actuellement la plateforme préférée pour le logo certifié pour Windows Server. Pour plus d’informations, consultez le programme de certification du logo de l' [application Cloud d’entreprise](https://azure.microsoft.com/overview/commercial-marketplace/) .
