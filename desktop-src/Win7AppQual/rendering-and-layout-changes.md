# <a name="rendering-and-layout-changes"></a>Modifications de rendu et de disposition

Windows Internet Explorer 8 interprète de manière plus stricte le balisage HTML par rapport aux versions précédentes, et cette modification peut provoquer l’affichage incorrect des pages Web. par exemple, lorsqu’un site affiche correctement une liste ou une table dans Windows internet explorer 7, il peut apparaître endommagé dans internet explorer 8.

Vous pouvez résoudre ces types de problèmes en réduisant la page Web pour rechercher le code HTML incorrect. si vous corrigez la plupart des problèmes d’affichage, les modifications ne créent pas de problèmes dans les versions précédentes de Windows Internet Explorer, car les versions précédentes ignorent simplement le code HTML superflu. Pour plus d’informations sur ces modifications, consultez le billet de blog [compatibilité de site et IE8](/archive/blogs/ie/site-compatibility-and-ie8) .
