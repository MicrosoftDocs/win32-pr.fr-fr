# <a name="what-is-compatibility-view"></a>Qu’est-ce que l’affichage de compatibilité ?

l' *affichage de compatibilité* est une fonctionnalité de Windows Internet Explorer 8 qui permet au navigateur de restituer une page web presque de la même façon que Windows Internet explorer 7 le rendrait.

Dans Internet Explorer 8, l’affichage de compatibilité modifie la façon dont le navigateur interprète le code écrit en CSS, HTML et le Document Object Model (DOM) pour essayer de correspondre à Internet Explorer 7. Un site qu’un utilisateur consulte dans l’affichage de compatibilité Internet Explorer 8 est presque identique à un site que l’utilisateur affiche dans Internet Explorer 7. Toutefois, l’affichage de compatibilité ne change pas la manière dont le navigateur interprète tout le code. par exemple, les modifications apportées à Internet Explorer 8 quant à la façon dont le navigateur gère ActiveX, l’analyseur, AJAX, JavaScript, la mise en réseau et la sécurité peuvent toujours provoquer des problèmes de compatibilité. L’affichage de compatibilité ne modifie pas ces comportements.

Dans un environnement d'entreprise, certains domaines présentent un risque de problèmes de compatibilité plus faible. Par exemple, les sites Web de la zone intranet utilisent par défaut l'affichage de compatibilité. Les applications Web clientes qui s’affichent à l’aide du contrôle de navigateur Web, ou le moteur de rendu WebOC (Internet Explorer), présentent également un risque faible pour les problèmes de compatibilité, car Internet Explorer 8 utilise par défaut un mode de compatibilité pour le WebOC. Toutefois, les paramètres de configuration par défaut de l’affichage de compatibilité peuvent ne pas garantir une compatibilité complète. Pour déterminer si un site Web ou une application Web est compatible avec Internet Explorer 8, vous devez tester le site Web ou l’application Web.

Pour plus d’informations sur les différences entre l’affichage de compatibilité d’Internet Explorer 8 et Internet Explorer 7, consultez le [blog compatibilité avec le site et Internet Explorer 8](/archive/blogs/ie/site-compatibility-and-ie8). pour obtenir la liste des éléments à vérifier lorsque vous effectuez une mise à niveau vers internet explorer 8, consultez [internet explorer 8 readiness Shared Computer Toolkit](https://www.microsoft.com/windows/internet-explorer/readiness/developers.aspx).

Pour plus d’informations sur l’affichage de compatibilité, consultez le blog de l' [équipe Internet Explorer](/archive/blogs/ie/).
