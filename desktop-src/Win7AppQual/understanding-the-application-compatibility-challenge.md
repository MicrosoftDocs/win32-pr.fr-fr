# <a name="understanding-the-application-compatibility-challenge"></a>Présentation du défi de la compatibilité des applications

Windows Internet Explorer 8 est le navigateur par défaut dans le système d’exploitation Windows 7. Internet Explorer 8 améliore de nombreuses fonctionnalités, notamment une architecture de plateforme améliorée conçue pour rendre le Web plus sécurisé et plus rapide pour les utilisateurs et plus faciles à configurer et à gérer pour les professionnels de l’informatique, et pour créer du contenu novateur pour les développeurs Web.

certaines de ces fonctionnalités utilisent de nouveaux moteurs de rendu et de script, qui sont les éléments principaux que Windows Internet Explorer utilise pour afficher du contenu HTML et CSS et exécuter des scripts. Les modifications architecturales et les améliorations apportées aux plateformes présentent des problèmes de compatibilité des applications que vous pouvez rencontrer lorsque vous migrez des applications vers Internet Explorer 8 à partir d’un navigateur précédent.

Internet Explorer 8 inclut une compatibilité descendante, mais il existe certaines modifications (notamment la prise en charge des fonctionnalités de sécurité et de prise en charge des normes Web) qui peuvent vous obliger à mettre à niveau des applications Web créées pour Microsoft Internet Explorer 6 ou versions antérieures.

Pour plus d’informations sur la façon de tester et résoudre les problèmes de compatibilité pour Internet Explorer 8, consultez adressage de la [compatibilité des applications lors de la migration vers Internet Explorer 8 – informations destinées aux professionnels de](/previous-versions/windows/it-pro/windows-7/ff955640(v=ws.10))l’informatique.
