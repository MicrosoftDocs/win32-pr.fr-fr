# <a name="application-compatibility-testing-using-virtual-pc-images"></a>Test de compatibilité des applications à l'aide d'images Virtual PC

Pour commencer à tester la compatibilité des applications Web, vous devez créer un environnement de test. Le processus de création et de configuration des ordinateurs pour cette tâche prend beaucoup de temps et est souvent coûteux.

Pour faciliter les tests de compatibilité, vous pouvez utiliser les images Virtual PC (VPC) gratuites et prédéfinies de compatibilité des applications. Les images VPC sont des images de disque dur préconfigurées que vous pouvez charger sur des ordinateurs existants qui exécutent le [logiciel Virtual PC](https://support.microsoft.com/help/14223/windows-xp-end-of-support). Ces images vous permettent de tester minutieusement les différentes versions de Microsoft Windows et de Windows Internet Explorer avec les applications existantes dans un laps de temps réduit.

Les images VPC sont disponibles pour toutes les versions de Windows actuellement prises en charge et contiennent de nombreuses applications et ressources de test de compatibilité. Pour rechercher les images de test, consultez [Internet Explorer Application Compatibility VPC image](https://www.microsoft.com/Downloads/details.aspx?FamilyID=21eabb90-958f-4b64-b5f1-73d0a413c8ef&amp;displaylang=en).
