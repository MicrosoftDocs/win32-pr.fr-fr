# <a name="internet-explorer-8-compatibility-wizard-aggiorno"></a>Assistant Compatibilité Internet Explorer 8 (Aggiorno)

L’Assistant Compatibilité de Windows Internet Explorer 8 met automatiquement à niveau les sites Web pour qu’ils s’affichent correctement dans Internet Explorer 8 sans compromettre leur compatibilité dans les navigateurs plus anciens. L’Assistant automatise la tâche de ciblage de la version du navigateur, en conservant l’apparence et les fonctionnalités des pages. Pour plus d’informations sur cet Assistant, consultez l' [Assistant Compatibilité d’Internet Explorer 8 démarrage rapide](https://www.mobilize.net/).
