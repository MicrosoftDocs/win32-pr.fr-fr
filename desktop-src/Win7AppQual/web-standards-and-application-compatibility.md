# <a name="web-standards-and-application-compatibility"></a>Normes Web et compatibilité des applications

Les normes Web fournissent une base de référence commune des technologies qui permettent aux utilisateurs de sélectionner un navigateur Web sans se soucier de l’utilisation des sites Web qu’ils souhaitent et qui permettent aux concepteurs Web et aux développeurs de créer un site Web unique que tout le monde peut utiliser.

Les normes pour les navigateurs Web évoluent constamment et les navigateurs Web sont continuellement mis à jour pour adopter ces normes. Par conséquent, les professionnels de l’informatique et les développeurs doivent s’assurer que les applications Web sont compatibles avec les navigateurs et les normes Web actuels et émergents. Cette maintenance garantit que les applications et les sites Web sont compatibles avec les nouvelles versions des navigateurs et permet d’éviter de nombreux problèmes de compatibilité susceptibles de se produire.
