# <a name="animation-timer-interfaces"></a>Interfaces du minuteur d’animation

Cette section contient les spécifications de référence pour les interfaces du gestionnaire d’animations Windows qui prennent en charge les minuteurs d’animation.

## <a name="in-this-section"></a>Contenu de cette section



| Rubrique                                                                                         | Description                                                                                   |
|-----------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------|
| [**IUIAnimationTimer**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationtimer)<br/>                                     | Définit une minuterie d’animation, qui fournit des services pour la gestion du minutage de l’animation.<br/> |
| [**IUIAnimationTimerClientEventHandler**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationtimerclienteventhandler)<br/> | Définit une méthode pour gérer les événements liés aux modifications de l’état du client de la minuterie.<br/>    |
| [**IUIAnimationTimerEventHandler**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationtimereventhandler)<br/>             | Définit des méthodes pour gérer les événements de minutage.<br/>                                        |
| [**IUIAnimationTimerUpdateHandler**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationtimerupdatehandler)<br/>           | Définit des méthodes pour gérer les événements de mise à jour de minutage.<br/>                                 |
