# <a name="storyboard-interfaces"></a>Interfaces de Storyboard

cette section contient les spécifications de référence pour les interfaces du gestionnaire d’animations Windows qui prennent en charge les storyboards.

## <a name="in-this-section"></a>Contenu de cette section



| Rubrique                                                                                                 | Description                                                                                                           |
|-------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| [**IUIAnimationStoryboard**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationstoryboard)<br/>                                   | Définit une table de montage séquentiel, qui contient un groupe de transitions synchronisées les unes par rapport aux autres.<br/> |
| [**IUIAnimationStoryboard2**](/windows/win32/api/uianimation/nn-uianimation-iuianimationstoryboard2)<br/>                                 | Définit une table de montage séquentiel, qui contient un groupe de transitions synchronisées les unes par rapport aux autres.<br/> |
| [**IUIAnimationStoryboardEventHandler**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationstoryboardeventhandler)<br/>           | Définit des méthodes pour gérer l’État et les événements de mise à jour d’une table de montage séquentiel.<br/>                                    |
| [**IUIAnimationStoryboardEventHandler2**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationstoryboardeventhandler2)<br/>         | Définit des méthodes pour gérer les événements de Storyboard. <br/>                                                           |
| [**IUIAnimationLoopIterationChangeHandler2**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationloopiterationchangehandler2)<br/> | Définit une méthode pour gérer les événements d’itération de la boucle de Storyboard.<br/>                                            |
| [**IUIAnimationPriorityComparison**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationprioritycomparison)<br/>                   | Définit une méthode pour la comparaison de priorité que le gestionnaire d’animations utilise pour résoudre les conflits de planification.<br/>  |
| [**IUIAnimationPriorityComparison2**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationprioritycomparison2)<br/>                 | Définit une méthode qui résout les conflits de planification par comparaison de priorité.<br/>                           |
