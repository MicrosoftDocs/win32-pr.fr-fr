# <a name="animation-management-interfaces"></a>Interfaces de gestion d’animation

cette section contient les spécifications de référence pour les interfaces du gestionnaire d’animations Windows qui prennent en charge la gestion des animations.

## <a name="in-this-section"></a>Contenu de cette section



| Rubrique                                                                                   | Description                                                                                                                                                              |
|-----------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IUIAnimationManager**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationmanager)<br/>                           | Définit le gestionnaire d’animations, qui fournit une interface centrale pour créer et gérer des animations.<br/>                                                       |
| [**IUIAnimationManager2**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationmanager2)<br/>                         | Définit un [**Gestionnaire d’animations**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationmanager2), qui fournit une interface centrale pour créer et gérer des animations dans plusieurs dimensions.<br/> |
| [**IUIAnimationManagerEventHandler**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationmanagereventhandler)<br/>   | Définit une méthode pour gérer les mises à jour d’état d’un gestionnaire d’animations.<br/>                                                                                         |
| [**IUIAnimationManagerEventHandler2**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationmanagereventhandler2)<br/> | Définit une méthode pour gérer les mises à jour d’un [**Gestionnaire d’animations**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationmanager2).<br/>                                                                |
