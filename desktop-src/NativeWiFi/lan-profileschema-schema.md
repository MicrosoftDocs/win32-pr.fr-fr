# <a name="lan_profile-schema"></a>Schéma de profil de réseau local \_

Le \_ schéma de profil de réseau local définit un profil réseau câblé (LAN). Un profil de réseau local contient des paramètres réseau pour une interface Ethernet.

L’élément racine d’un profil de réseau local est l’élément [**LANProfile**](lan-profileschema-lanprofile-element.md) . Chaque profil aura un seul élément racine. Tous les \_ éléments de profil de réseau local se trouvent dans l’espace de noms `https://www.microsoft.com/networking/LAN/profile/v1` .

Pour afficher des exemples de profils de réseau local, consultez la page exemples de profils [câblés](wired-profile-samples.md).
