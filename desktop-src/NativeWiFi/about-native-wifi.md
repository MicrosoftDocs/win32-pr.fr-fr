# <a name="about-native-wifi"></a>À propos de la WiFi Native

Les fonctions d’interface de programmation d’applications (API) WiFi natives ont deux objectifs : pour gérer les profils de réseau sans fil et pour gérer les connexions de réseau sans fil. Les éléments d’API sont exposés par le module de configuration automatique (ACM). Les éléments d’API de connexion et de déconnexion exposés peuvent être utilisés pour remplacer la logique de configuration automatique.
