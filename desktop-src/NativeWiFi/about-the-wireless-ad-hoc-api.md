# <a name="about-the-wireless-ad-hoc-api"></a>À propos de l’API ad hoc sans fil

L’API ad hoc sans fil expose des interfaces pour l’énumération et la connexion aux réseaux ad hoc 802,11. Ces interfaces fournissent une approche orientée objet simplifiée pour la gestion de réseau ad hoc. Les énumérateurs standard sont fournis pour les réseaux et les interfaces. Ces énumérateurs peuvent être filtrés pour énumérer uniquement les réseaux ad hoc créés par votre application.

> [!Note]  
> Le mode ad hoc n’est peut-être pas disponible dans les versions ultérieures de Windows. à partir de Windows 8.1 et Windows Server 2012 R2, utilisez [Wi-Fi Direct](about-the-wi-fi-direct-api.md) à la place.

 

Les développeurs qui utilisent l’API ad hoc doivent savoir que les réseaux et les profils créés à l’aide de l’API ad hoc peuvent être manipulés à l’aide de l’API WiFi native sans notifier les écouteurs de notification d’API ad hoc appropriés.

L’API ad hoc sans fil est décrite en détail dans la [référence ad hoc sans fil](wireless-ad-hoc-reference.md).

> [!Note]  
> l’API ad Hoc sans fil est uniquement prise en charge sur Windows Vista et les versions ultérieures du système d’exploitation.
