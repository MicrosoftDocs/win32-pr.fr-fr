# <a name="onex-schema"></a>Schéma OneX

Le schéma OneX définit des éléments de configuration 802.1 X. Tous les éléments de schéma OneX s’appliquent aux profils filaires et sans fil. Pour obtenir la liste des éléments définis, consultez [éléments de schéma Onex](onexschema-elements.md).

L’élément racine d’un profil 802.1 X est l’élément [**Onex**](onexschema-onex-element.md) . Chaque profil aura un seul élément racine. L’élément **Onex** se trouve dans l' `https://www.microsoft.com/networking/OneX/v1` espace de noms.
