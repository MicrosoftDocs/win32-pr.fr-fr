# <a name="network-allowlist-element"></a>Élément Network (allowList)

L’élément allowList (Network) définit un réseau autorisé. Un ordinateur doit être autorisé à se connecter à un réseau autorisé.

``` syntax
<xs:element name="network"
    type="networkItemType"
 />
```

L’élément **réseau** est défini par l’élément [**allowList**](wlan-policyschema-allowlist-networkfilter-element.md) .
