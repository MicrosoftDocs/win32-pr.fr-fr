# <a name="lan_profile-schema-elements"></a>Éléments de schéma de profil de réseau local \_

Un profil de réseau câblé contient les éléments de schéma suivants. Tous les éléments nommés se trouvent dans l’espace de noms `https://www.microsoft.com/networking/LAN/profile/v1` .

La liste suivante présente les éléments définis dans l’ordre dans lequel les éléments apparaissent dans un profil. L’ordre des éléments est appliqué. Cette liste n’affiche pas tous les éléments possibles qui peuvent apparaître dans un profil, car des éléments peuvent être ajoutés dans **XS : n’importe quel** point d’insertion.

-   [**LANProfile**](lan-profileschema-lanprofile-element.md)
    -   [**MSM (LANProfile)**](lan-profileschema-msm-lanprofile-element.md)
        -   [**sécurité (MSM)**](lan-profileschema-security-msm-element.md)
            -   [**OneXEnforced (sécurité)**](lan-profileschema-onexenforced-security-element.md)
            -   [**OneXEnabled (sécurité)**](lan-profileschema-onexenabled-security-element.md)
