# <a name="description-lanpolicy-element"></a>Description (LANPolicy) (élément)

L’élément Description (LANPolicy) contient la description d’une stratégie de réseau local câblé.

``` syntax
<xs:element name="description"
    type="nameType"
 />
```

L’élément **Description** est défini par l’élément [**LANPolicy**](lan-policyschema-lanpolicy-element.md) .

## <a name="requirements"></a>Spécifications



| Condition requise | Valeur |
|-------------------------------------|------------------------------------------------------|
| Client minimal pris en charge<br/> | Windows \[Applications de bureau Vista uniquement\]<br/>       |
| Serveur minimal pris en charge<br/> | Windows Serveur 2008 \[ applications de bureau uniquement\]<br/> |
