# <a name="wireless-ad-hoc-enumerations"></a>Énumérations ad hoc sans fil

L’interface ad hoc sans fil définit et utilise les énumérations suivantes.

-   [**\_ \_ Algorithme d’authentification ad hoc DOT11 \_**](/windows/win32/api/adhoc/ne-adhoc-dot11_adhoc_auth_algorithm)
-   [**\_ \_ Algorithme de chiffrement adhoc DOT11 \_**](/windows/win32/api/adhoc/ne-adhoc-dot11_adhoc_cipher_algorithm)
-   [**Motif de l’échec de la \_ connexion adhoc DOT11 \_ \_ \_**](/windows/win32/api/adhoc/ne-adhoc-dot11_adhoc_connect_fail_reason)
-   [**\_État de \_ la \_ connexion réseau ad hoc DOT11 \_**](/windows/win32/api/adhoc/ne-adhoc-dot11_adhoc_network_connection_status)

> [!Note]  
> Le mode ad hoc n’est peut-être pas disponible dans les futures versions de Windows. À partir de Windows 8.1 et de Windows Server 2012 R2, utilisez plutôt [le Wi-Fi direct](about-the-wi-fi-direct-api.md) .
