# <a name="name-wlanpolicy-element"></a>Élément Name (WLANPolicy)

L’élément Name (WLANPolicy) contient le nom d’une stratégie de réseau local sans fil.

``` syntax
<xs:element name="name"
    type="nameType"
 />
```

L’élément **Name** est défini par l’élément [**WLANPolicy**](wlan-policyschema-wlanpolicy-element.md) .

## <a name="remarks"></a>Notes

Les noms ne respectent pas la casse.
