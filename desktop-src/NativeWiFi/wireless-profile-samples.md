# <a name="wireless-profile-samples"></a>Exemples de profils sans fil

Les rubriques suivantes présentent des exemples de profils XML pour différentes configurations sans fil. Ces profils XML utilisent [des \_ éléments de schéma de profil WLAN](wlan-profileschema-elements.md) et des [éléments de schéma Onex](onexschema-elements.md).

Vous pouvez ajouter un profil sans fil à la Banque de profils par programmation en appelant [**WLanSetProfile**](/windows/desktop/api/wlanapi/nf-wlanapi-wlansetprofile). Vous pouvez également ajouter un profil au magasin de profils à l’aide de l’utilitaire de ligne de commande **netsh** . Pour plus d’informations, consultez [commandes netsh pour réseau local sans fil (WLAN)](/previous-versions/windows/it-pro/windows-server-2008-R2-and-2008/cc755301(v=ws.10)).
