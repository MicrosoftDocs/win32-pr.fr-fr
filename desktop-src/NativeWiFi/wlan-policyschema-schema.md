# <a name="wlan_policy-schema"></a>\_Schéma de stratégie WLAN

Le \_ schéma de stratégie WLAN définit un profil de stratégie de réseau local sans fil utilisé par le service de configuration automatique WiFi natif.

L’élément racine d’un profil de stratégie de réseau local sans fil est l’élément [**WLANPolicy**](wlan-policyschema-wlanpolicy-element.md) . Chaque profil aura un seul élément racine. Tous les \_ éléments de stratégie WLAN se trouvent dans l’espace de noms `https://www.microsoft.com/networking/WLAN/policy/v1` .

-   [\_Éléments WLAN policySchema](wlan-policyschema-elements.md)
-   [\_Types simples POLICYSCHEMA WLAN](wlan-policyschema-simple-types.md)
-   [\_Types complexes POLICYSCHEMA WLAN](wlan-policyschema-complex-types.md)
