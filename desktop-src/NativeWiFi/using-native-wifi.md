# <a name="using-native-wifi"></a>Utilisation de la WiFi Native

Cette section décrit les procédures et les techniques de programmation utilisées avec le Wi-Fi natif. Il comprend des techniques de programmation WiFi natives de base, telles que l' [exemple d’API WiFi Native](native-wifi-api-sample.md), ainsi que des techniques avancées utiles aux développeurs NativeWifi expérimentés.

Les rubriques suivantes sont disponibles dans cette section :

-   [API d’interface utilisateur sans fil](wireless-user-interface-apis.md)
-   [Utilisation des fonctions Wi-Fi direct](using-the-wi-fi-direct-api.md)
-   [Utilisation du partage de connexion Internet et du réseau hébergé](using-hosted-network-and-internet-connection-sharing.md)
-   [Exemple d’API WiFi Native](native-wifi-api-sample.md)
-   [Exemple de réseau hébergé sans fil](wireless-hosted-network-sample.md)
-   [Exemples de profil câblé](wired-profile-samples.md)
-   [Exemples de profils sans fil](wireless-profile-samples.md)

La rubrique [API d’interface utilisateur sans fil](wireless-user-interface-apis.md) présente du code permettant de lancer des interfaces utilisateur de gestion de réseau sans fil.

L' [utilisation de la rubrique Wi-Fi directe Functions](using-the-wi-fi-direct-api.md) répertorie les Wi-Fi fonctions directes et explique comment les fonctions sont utilisées.

La rubrique [utilisation du réseau hébergé et du partage de connexion Internet](using-hosted-network-and-internet-connection-sharing.md) explique comment utiliser le réseau hébergé sans fil et le partage de connexion Internet (ICS).

Le kit de développement logiciel (SDK) Microsoft Windows comprend deux exemples de code qui illustrent l’utilisation de l’API WiFi native. En outre, des exemples de profils sont disponibles qui illustrent l’utilisation de différents schémas de profil Wi-Fi natifs.

Les rubriques suivantes contiennent des informations supplémentaires sur les exemples :

-   [Exemple d’API WiFi Native](native-wifi-api-sample.md)
-   [Exemple de réseau hébergé sans fil](wireless-hosted-network-sample.md)
-   [Exemples de profil câblé](wired-profile-samples.md)
-   [Exemples de profils sans fil](wireless-profile-samples.md)
