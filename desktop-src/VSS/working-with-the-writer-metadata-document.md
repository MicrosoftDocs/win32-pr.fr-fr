# <a name="working-with-the-writer-metadata-document"></a>Utilisation du document de métadonnées de l’enregistreur

Chaque Writer produit un document de métadonnées d’écriture. L’application de sauvegarde utilise ce document pour obtenir des informations sur ce writer, les données qu’il détient et la manière de restaurer ces données. Une fois que l’enregistreur le produit, le document de métadonnées de l’enregistreur est un document en lecture seule pour l’application de sauvegarde.

Pour utiliser le document de métadonnées d’écriture, vous devez comprendre les éléments suivants :

-   [Cycle de vie des documents de métadonnées de l’enregistreur](writer-metadata-document-life-cycle.md)
-   [Contenu du document de métadonnées de l’enregistreur](writer-metadata-document-contents.md)
