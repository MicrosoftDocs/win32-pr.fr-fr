# <a name="volume-shadow-copy-service-overview"></a>Présentation de Service VSS

Le Service VSS (VSS) est un ensemble d’API COM qui implémente une infrastructure pour permettre l’exécution de sauvegardes de volume alors que les applications sur un système continuent d’écrire sur les volumes.

VSS fournit une interface cohérente qui permet la coordination entre les applications de l’utilisateur qui mettent à jour les données sur le disque (les [*Writers*](vssgloss-w.md)) et celles qui sauvegardent les applications ([*demandeurs*](vssgloss-r.md)).
