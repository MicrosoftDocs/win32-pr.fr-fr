# <a name="power-schemes"></a>Modes de gestion de l’alimentation

Un mode de gestion de l’alimentation est un ensemble de paramètres qui contrôle l’utilisation de l’alimentation de votre ordinateur. Vous pouvez utiliser des modes de gestion de l’alimentation pour réduire la consommation d’énergie des appareils individuels ou de l’ensemble du système. Vous pouvez configurer des fonctionnalités facultatives, telles que la prise en charge de la mise en veille prolongée, la durée d’inactivité du système avant la désactivation de l’affichage ou l’état de veille utilisé lors de l’expiration du délai d’inactivité du système.

Il existe plusieurs façons de créer des modes de gestion de l’alimentation : l’application options d’alimentation dans le panneau de configuration, l’outil en ligne de commande PowerCfg.exe et l’API de mode de gestion de l’alimentation. Pour plus d’informations sur les options d’alimentation ou les PowerCfg.exe, consultez l’aide incluse dans le système d’exploitation.

Les rubriques suivantes décrivent la prise en charge du mode de gestion de l’alimentation fournie par l’API mode de gestion de l’alimentation :

-   [Paramètres de la stratégie d’alimentation](power-policy-settings.md)
-   [Gestion du mode de gestion de l’alimentation](managing-power-schemes.md)
-   [Remplacements de l’administrateur](administrator-overrides.md)
