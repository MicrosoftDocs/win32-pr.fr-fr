# <a name="windows-power-management"></a><span data-ttu-id="d6567-103">Gestion de l’alimentation Windows</span><span class="sxs-lookup"><span data-stu-id="d6567-103">Windows Power Management</span></span>

<span data-ttu-id="d6567-104">La gestion de l’alimentation Windows rend les ordinateurs accessibles instantanément aux utilisateurs en appuyant sur un bouton ou une touche.</span><span class="sxs-lookup"><span data-stu-id="d6567-104">Windows power management makes computers instantly accessible to users at the touch of a button or key.</span></span> <span data-ttu-id="d6567-105">Il garantit également que tous les éléments du système (applications, périphériques et interface utilisateur) peuvent tirer parti des améliorations apportées à la technologie et aux fonctionnalités de gestion de l’alimentation.</span><span class="sxs-lookup"><span data-stu-id="d6567-105">It also ensures that all elements of the system—applications, devices, and user interface—can take advantage of the vast improvements in power management technology and capabilities.</span></span>

<span data-ttu-id="d6567-106">Le système d’exploitation Windows utilise le matériel de gestion de l’alimentation pour mettre l’ordinateur dans un *État de veille* faible au lieu de s’arrêter complètement, afin que le système puisse reprendre rapidement son travail.</span><span class="sxs-lookup"><span data-stu-id="d6567-106">The Windows operating system uses power-management hardware to put the computer into a low-power *sleep state* instead of shutting down completely, so that the system can quickly resume working.</span></span> <span data-ttu-id="d6567-107">Le système d’exploitation passe automatiquement en état de veille lorsque l’ordinateur est inactif ou lorsque l’utilisateur appuie sur un bouton pour indiquer que la session de travail active est terminée.</span><span class="sxs-lookup"><span data-stu-id="d6567-107">The operating system will automatically enter the sleep state when the computer is idle or when the user presses a button to indicate that the current work session is over.</span></span> <span data-ttu-id="d6567-108">Pour l’utilisateur, le système semble être désactivé.</span><span class="sxs-lookup"><span data-stu-id="d6567-108">To the user, the system appears to be off.</span></span> <span data-ttu-id="d6567-109">En état de veille, le processeur de l’ordinateur n’exécute pas de code et aucun travail n’est effectué pour l’utilisateur.</span><span class="sxs-lookup"><span data-stu-id="d6567-109">While in the sleep state, the computer's processor is not executing code and no work is being accomplished for the user.</span></span> <span data-ttu-id="d6567-110">Toutefois, les événements du système à partir des périphériques matériels et de l’horloge en temps réel peuvent être activés pour forcer le système à quitter l’état de veille (c’est-à-dire « réveiller ») et à revenir rapidement à l’état de travail.</span><span class="sxs-lookup"><span data-stu-id="d6567-110">However, events in the system from both hardware devices and the real-time clock can be enabled to cause the system to exit the sleep state (that is, "wake up") and quickly return to the working state.</span></span>

<span data-ttu-id="d6567-111">Lorsque l’ordinateur est en état de veille, le matériel de l’ordinateur, le système et les applications qui s’exécutent sur l’ordinateur doivent être en capacité de répondre immédiatement au bouton d’alimentation, aux événements de communication et à d’autres actions.</span><span class="sxs-lookup"><span data-stu-id="d6567-111">When the computer is in the sleep state, the computer hardware, the system, and applications running on the computer must be capable of responding immediately to the power switch, communications events, and other actions.</span></span> <span data-ttu-id="d6567-112">Si toutes les applications gèrent les transitions d’état d’alimentation normalement, l’utilisateur perçoit un système plus élégant et plus intégré.</span><span class="sxs-lookup"><span data-stu-id="d6567-112">If all applications handle power state transitions gracefully, the user will perceive a more elegant and integrated system.</span></span> <span data-ttu-id="d6567-113">Les applications qui ne gèrent pas ces transitions peuvent échouer lorsque l’alimentation est désactivée puis activée, en raison d’une perte de données ou d’une dépendance sur un appareil qui a peut-être été supprimé.</span><span class="sxs-lookup"><span data-stu-id="d6567-113">Applications that do not handle these transitions can fail when the power is turned off and then on, because of data loss or a dependency on a device that may have been removed.</span></span>

<span data-ttu-id="d6567-114">Voici les avantages de la gestion de l’alimentation Windows :</span><span class="sxs-lookup"><span data-stu-id="d6567-114">The following are benefits of Windows power management:</span></span>

-   <span data-ttu-id="d6567-115">Élimine les délais de démarrage et d’arrêt.</span><span class="sxs-lookup"><span data-stu-id="d6567-115">Eliminates startup and shutdown delays.</span></span> <span data-ttu-id="d6567-116">L’ordinateur n’a pas besoin d’effectuer un démarrage système complet en quittant l’état de veille ou un arrêt complet du système lorsque l’utilisateur démarre l’état de veille.</span><span class="sxs-lookup"><span data-stu-id="d6567-116">The computer need not perform a full system boot when exiting the sleep state or a full system shutdown when the user initiates the sleep state.</span></span>
-   <span data-ttu-id="d6567-117">Permet aux tâches automatisées de s’exécuter pendant que l’ordinateur est en état de veille.</span><span class="sxs-lookup"><span data-stu-id="d6567-117">Enables automated tasks to run while the computer is in the sleep state.</span></span> <span data-ttu-id="d6567-118">Le Planificateur de tâches permet à l’utilisateur de planifier l’exécution des applications. les événements planifiés peuvent s’exécuter même lorsque le système est en état de veille.</span><span class="sxs-lookup"><span data-stu-id="d6567-118">The Task Scheduler enables the user to schedule applications to run; scheduled events can run even when the system is in the sleep state.</span></span> <span data-ttu-id="d6567-119">Le Planificateur de tâches utilise des [minuteries waitables](/windows/desktop/Sync/waitable-timer-objects) pour s’assurer que le système est prêt lorsque l’application est planifiée pour s’exécuter.</span><span class="sxs-lookup"><span data-stu-id="d6567-119">The Task Scheduler uses [waitable timers](/windows/desktop/Sync/waitable-timer-objects) to ensure that the system is ready when the application is scheduled to run.</span></span> <span data-ttu-id="d6567-120">Pour plus d’informations, consultez le fichier d’aide inclus avec l’Planificateur de tâches.</span><span class="sxs-lookup"><span data-stu-id="d6567-120">For more information, see the help file included with the Task Scheduler.</span></span>
-   <span data-ttu-id="d6567-121">Active la gestion de l’alimentation par appareil.</span><span class="sxs-lookup"><span data-stu-id="d6567-121">Enables per-device power management.</span></span> <span data-ttu-id="d6567-122">Les appareils qui ne sont pas en cours d’utilisation peuvent économiser de l’énergie en entrant l’état de veille.</span><span class="sxs-lookup"><span data-stu-id="d6567-122">Devices that are not in use can save power by entering the sleep state.</span></span>
-   <span data-ttu-id="d6567-123">Améliore l’efficacité de l’alimentation.</span><span class="sxs-lookup"><span data-stu-id="d6567-123">Improves power efficiency.</span></span> <span data-ttu-id="d6567-124">L’efficacité de l’alimentation est particulièrement importante sur les ordinateurs portables.</span><span class="sxs-lookup"><span data-stu-id="d6567-124">Power efficiency is particularly important on portable computers.</span></span> <span data-ttu-id="d6567-125">La réduction de la consommation d’énergie du système se traduit directement par des coûts d’énergie réduits et une durée de vie de batterie plus longue.</span><span class="sxs-lookup"><span data-stu-id="d6567-125">Reducing system power consumption translates directly to lower energy costs and longer battery life.</span></span>
-   <span data-ttu-id="d6567-126">Permet aux utilisateurs de créer des [modes d’alimentation](power-schemes.md), de définir des alarmes et de spécifier des options de batterie par le biais de l’application options d’alimentation dans le panneau de configuration.</span><span class="sxs-lookup"><span data-stu-id="d6567-126">Enables users to create [power schemes](power-schemes.md), set alarms, and specify battery options through the Power Options application in Control Panel.</span></span> <span data-ttu-id="d6567-127">Le système d’exploitation coordonne toutes les activités de gestion de l’alimentation, en fonction des paramètres de stratégie d’alimentation.</span><span class="sxs-lookup"><span data-stu-id="d6567-127">The operating system coordinates all power management activities, based on power policy settings.</span></span> <span data-ttu-id="d6567-128">Pour plus d’informations, consultez le fichier d’aide inclus avec l’application options d’alimentation.</span><span class="sxs-lookup"><span data-stu-id="d6567-128">For more information, see the help file included with the Power Options application.</span></span>
