# <a name="power-management-control-codes"></a>Codes de contrôle de gestion de l’alimentation

Les codes de contrôle suivants sont utilisés avec les [batteries](battery-information.md).



| Code de contrôle                                                                  | Description                                        |
|-------------------------------------------------------------------------------|----------------------------------------------------|
| [**\_ \_ informations sur les requêtes de batterie IOCTL \_**](ioctl-battery-query-information.md) | Récupère diverses informations pour la batterie |
| [**État de la \_ requête de batterie IOCTL \_ \_**](ioctl-battery-query-status.md)           | Récupère l’état actuel de la batterie.      |
| [**\_ \_ balise de requête de batterie IOCTL \_**](ioctl-battery-query-tag.md)                 | Récupère la balise actuelle de la batterie.               |
| [**\_ \_ informations sur le jeu de batteries IOCTL \_**](ioctl-battery-set-information.md)     | Définit différentes informations sur la batterie.                  |



 

Les codes de contrôle suivants font partie de l' [interface de contrôle de rétroéclairage](backlight-control-interface.md).



| Code de contrôle                                                                                 | Description                                                                   |
|----------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------|
| [**luminosité de l' \_ \_ affichage des requêtes vidéo \_ IOCTL \_**](ioctl-video-query-display-brightness.md)     | Récupère les niveaux de rétroéclairage AC et DC actuels et l’état d’alimentation actuel. |
| [**\_ \_ \_ luminosité prise en charge des requêtes de vidéo IOCTL \_**](ioctl-video-query-supported-brightness.md) | Récupère les niveaux de rétroéclairage pris en charge.                                     |
| [**luminosité de l’affichage du jeu de \_ vidéos IOCTL \_ \_ \_**](ioctl-video-set-display-brightness.md)         | Définit les niveaux de rétroéclairage AC et DC actuels                                   |
