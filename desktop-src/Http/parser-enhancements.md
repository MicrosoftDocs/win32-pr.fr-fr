# <a name="parser-enhancements"></a>Améliorations de l’analyseur

Dans Windows Server 2003 avec Service Pack 1 (SP1), l’API du serveur HTTP autorise les requêtes suivantes des clients HTTP.

-   Demandes qui utilisent un saut de ligne (LF) comme marque de fin de ligne.
-   Les demandes qui contiennent des espaces linéaires (LWS) entre la ligne de requête HTTP et le début des en-têtes.

Ces comportements sont activés par défaut et ne sont pas contrôlés par les paramètres du Registre.
