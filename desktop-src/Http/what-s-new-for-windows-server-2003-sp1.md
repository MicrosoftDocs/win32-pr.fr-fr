# <a name="whats-new-for-windows-server-2003-sp1"></a>nouveautés de Windows Server 2003 SP1

les fonctionnalités suivantes sont disponibles pour Windows Server 2003 avec Service Pack 1 (SP1) :

-   [Prise en charge SSL en mode noyau.](kernel-mode-ssl.md)
    -   Windows Le serveur 2003 avec SP1 ajoute l’option permettant d’effectuer des opérations SSL en mode noyau pour améliorer les performances.
-   Prise en charge de l’exécution d’applications 32 bits sur le système d’exploitation 64 bits (WOW64).
-   [Améliorations apportées à la journalisation des erreurs HTTP.](error-logging-in-windows-server-2003-sp1.md)
    -   Prise en charge des en-têtes de style W3C.
    -   Champs de journalisation supplémentaires disponibles.
    -   Substitution de fichier journal basée sur la date et l’heure.
-   [Exceptions d’analyse HTTP.](parsing-exceptions.md)
    -   Prend en charge les exceptions à HTTP/1.1 pour des raisons de compatibilité descendante.
-   [Améliorations de l’analyseur HTTP.](parser-enhancements.md)
