# <a name="httpcfgexe"></a>HttpCfg.exe

Cet outil vous permet de configurer l’API du serveur HTTP en encapsulant les [API de configuration](configuration-apis.md) http pour les paramètres suivants :

-   Liste d’écoutes IP
-   Certificats SSL
-   Réservations d’espaces de noms

HttpCfg.exe est fourni avec le système d’exploitation à partir de Windows Server 2003 en tant qu’exécutable binaire, ainsi qu’un fichier d’aide qui décrit comment l’utiliser. Il peut être installé à partir du disque de distribution du système d’exploitation en tant qu’outil de prise en charge.

HttpCfg.exe est également disponible en téléchargement. consultez [outils de support de Windows XP SP2](https://www.microsoft.com/downloads/details.aspx?familyid=49AE8576-9BB9-4126-9761-BA8011FABF38).

**Windows Vista et Windows Server 2008 :** HttpCfg.exe est obsolète et remplacé par une fonctionnalité similaire dans netsh.exe.
