# <a name="configuration-apis"></a>API de configuration

Les API de serveur HTTP fournissent des fonctionnalités permettant aux administrateurs système de définir, d’interroger et de supprimer des paramètres de configuration. Les API de configuration contrôlent les paramètres de toutes les applications à l’aide de l’API du serveur HTTP sur le système.

Les sections listées ci-dessous contiennent des informations sur les paramètres de configuration.

> [!Note]  
> Tous ces éléments sont des étapes de configuration facultatives à appliquer en fonction des exigences du système et/ou de l’application.

 

Liste d’écoutes [IP](ip-listen-list.md)   : décrit comment configurer HTTP pour fonctionner sur des interfaces IP spécifiques.

[Certificats SSL](ssl-certificates.md)   : détaille les procédures de configuration des certificats avec l’API du serveur HTTP.

[Réservations d’espaces de noms, inscriptions et routage](namespace-reservations-registrations-and-routing.md)   : décrit le processus de réservation de portions de l’espace de noms pour toutes les applications sur l’ordinateur à l’aide de l’API du serveur HTTP. Le processus d’inscription et de routage des demandes vers des applications spécifiques est également décrit.

[HttpCfg.exe](httpcfg-exe.md)   : exemple de programme qui utilise les API de configuration. Une version compilée de cet outil est également fournie avec le système d’exploitation parmi les outils de support, ainsi qu’un fichier d’aide qui décrit comment l’utiliser.
