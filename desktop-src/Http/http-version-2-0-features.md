# <a name="http-version-20-features"></a>Fonctionnalités de la version HTTP 2,0

Les fonctionnalités suivantes sont prises en charge pour l’API du serveur HTTP version 2,0.

-   Objets de configuration de session et de groupe d’URL du serveur
-   Propriétés de configuration exposées sur les objets de configuration
-   Gestion et configuration avancées de la file d’attente des requêtes
-   SSL en mode noyau complet
-   Cache de réponse en mode noyau
-   Authentification côté serveur
-   Contrôle de version avec étendue des objets
-   Journalisation côté serveur
-   Arrêt progressif de la file d’attente des demandes
-   Début de la demande dans la file d’attente des demandes
-   Partage de port
-   support Suivi d’v nements pour Windows
-   Compteurs de performance
-   Prise en charge des IDN (Internationalized Domain Name)
-   Utilitaire de configuration de l’interface réseau
