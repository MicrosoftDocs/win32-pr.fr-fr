# <a name="show-iplisten"></a>show iplisten

Répertorie toutes les adresses IP de la liste d’écoutes IP. La liste d’écoutes IP est utilisée pour définir l’étendue de la liste des adresses auxquelles le service HTTP est lié. « 0.0.0.0 » signifie toute adresse IPv4, tandis que « :: » signifie toute adresse IPv6.

``` syntax
show iplisten
 
```
