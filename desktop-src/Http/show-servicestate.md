# <a name="show-servicestate"></a>show servicestate

Affiche un instantané du service HTTP.

``` syntax
show servicestate [[view=]{session|requestq}] [[verbose=]{yes|no}]
 
```

## <a name="parameters"></a>Paramètres

<dl> <dt>

<span id="__view___session_requestq__"></span><span id="__VIEW___SESSION_REQUESTQ__"></span>**\[\[View = \] {session \| requestq}\]**
</dt> <dd>

Affichez l’instantané de l’état du service HTTP en fonction de la session du serveur ou des files d’attente des demandes.

</dd> <dt>

<span id="__verbose___yes_no__"></span><span id="__VERBOSE___YES_NO__"></span>**\[\[verbose = \] {Oui \| non}\]**
</dt> <dd>

Affichez les informations détaillées qui affichent également les informations de propriété.

</dd> </dl>

## <a name="examples"></a>Exemples

**afficher la vue ServiceState = session**

**afficher ServiceState affichage = requestq détaillé = Oui**
