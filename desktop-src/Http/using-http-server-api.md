# <a name="using-http-server-api"></a>Utilisation de l’API du serveur HTTP

Les fichiers sources du programme d’application incluent le fichier d’en-tête http. h pour accéder aux prototypes de fonction et aux définitions de structure pour l’API du serveur HTTP. Les développeurs peuvent utiliser le fichier de bibliothèque httpapi. lib pour créer des applications qui utilisent l’API du serveur HTTP. Au moment de l’exécution, les applications sont liées au Httpapi.dll.
