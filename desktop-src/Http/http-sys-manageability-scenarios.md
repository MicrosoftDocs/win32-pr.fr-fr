# <a name="httpsys-manageability-scenarios"></a>Scénarios de facilité de gestion HTTP.sys

Les scénarios suivants fournissent des exemples de la façon dont ces outils peuvent aider les professionnels de l’informatique à diagnostiquer et à résoudre les problèmes liés au composant de l’API serveur HTTP (HTTP.sys). les outils utilisés dans ces exemples sont disponibles dans Windows Vista et Windows Server 2008.

-   [Scénario 1 : exemple de délai d’expiration HTTP utilisant le suivi ETW et les commandes netsh](scenario-1--http-timeout-example-using-etw-tracing-and-netsh-commands.md)
-   [Scénario 2 : exemple d’analyseur utilisant le suivi ETW](scenario-2--parser-example-using-etw-tracing.md)
-   [Scénario 3 : compteurs de performances](scenario-3--performance-counters.md)

Les scénarios de ce document sont des exemples d’utilisation des outils de gestion du composant de l’API du serveur HTTP. Ces exemples ne sont pas destinés à suggérer des modifications de configuration spécifiques.
