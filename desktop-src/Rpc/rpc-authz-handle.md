# <a name="rpc_authz_handle"></a>\_handle RPC auth \_

Le type de données du **\_ \_ handle RPC Authur** déclare un handle d’autorisation. Ce handle pointe vers les informations de privilèges de l’application cliente qui a effectué l’appel de procédure distante.


```C++
typedef void* RPC_AUTHZ_HANDLE;
```
