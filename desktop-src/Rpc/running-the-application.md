# <a name="running-the-application"></a>Exécution de l'application

Pour exécuter l’application sur un seul ordinateur, ouvrez deux fenêtres de console. Dans la première fenêtre, tapez

**Hello**

et dans la deuxième fenêtre, tapez

**helloc**

Pour expérimenter des séquences de protocoles, des points de terminaison et d’autres options différents, générez l’exemple d’application Hello à partir des fichiers sources dans le \\ répertoire RPC Hello du kit de développement logiciel (SDK) de la plateforme.

> [!Note]  
> la Restriction de l' **Interface RPC** peut entraîner l’échec de cette application en raison d’une erreur d' **accès refusé** sur Windows XP avec Service Pack 2 (SP2) et les versions ultérieures de Windows. Pour plus d’informations, consultez restriction de l' [interface RPC](/previous-versions/windows/it-pro/windows-xp/bb457156(v=technet.10)#EGAA).
