# <a name="configuring-the-security-server"></a>Configuration du serveur de sécurité

La procédure suivante détaille la configuration du serveur de sécurité pour Microsoft RPC.

**Pour configurer le serveur de sécurité pour RPC**

1.  Démarrez le panneau de configuration.
2.  Dans le panneau de configuration, cliquez sur l’icône **réseaux** . la boîte de dialogue **Paramètres réseau** s’affiche.
3.  Dans la liste des **logiciels réseau installés** , sélectionnez **configuration RPC**. La boîte de dialogue **configuration RPC** s’affiche.
4.  Dans la liste **fournisseur de services de sécurité** , sélectionnez un ou plusieurs fournisseurs de sécurité.
5.  Cliquez sur **OK**.
