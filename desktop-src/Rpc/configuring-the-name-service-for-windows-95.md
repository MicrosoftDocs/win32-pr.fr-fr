# <a name="configuring-the-name-service-for-windows-95"></a>Configuration du service de noms pour Windows 95

Windows 95 n’utilise pas Microsoft Locator. Pour pouvoir utiliser un service de noms dans une application Windows 95, l’ordinateur équipé de Windows 95 doit :

-   Faire partie d’un groupe de travail ou d’un domaine qui comprend un ordinateur exécutant Windows 2000 ou Windows NT pour servir de fournisseur de services de noms proxy.
-   Être connecté à un ordinateur hôte exécutant le démon NSI (NSID), qui sert de passerelle à l’service d’annuaire de cellules/service d’annuaires de cellules DCE de l’Digital Equipment Corporation.
