# <a name="the-application-configuration-file-acf"></a>Fichier de configuration de l’application (ACF)

Le fichier de configuration d’application (ACF) se compose de deux parties : un [en-tête d’interface](the-acf-header.md), semblable à l’en-tête d’interface dans le fichier IDL, et un [corps](the-acf-body.md), qui contient des attributs de configuration qui s’appliquent aux types et aux fonctions définis dans le corps de l’interface du fichier IDL. Celles-ci sont présentées dans les rubriques suivantes :

-   [En-tête ACF](the-acf-header.md)
-   [Corps ACF](the-acf-body.md)
