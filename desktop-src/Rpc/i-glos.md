# <a name="i-rpc"></a>I (RPC)

[A](a-glos.md) [B](b-glos.md) [C](c-glos.md) [D](d-glos.md) [e](e-glos.md) [F](f-glos.md) G H I J K [L](l-glos.md) [M](m-glos.md) [N](n-glos.md) [O](o-glos.md) [P](p-glos.md) [Q](q.md) [R](r-glos.md) [S](s-glos.md) [T](t-glos.md) [U](u-glos.md) [V](v-glos.md) [W](w-glos.md) X Y Z

<dl> <dt>

<span id="_rpc_idempotent_glos"></span><span id="_RPC_IDEMPOTENT_GLOS"></span>**idempotent**
</dt> <dd>

Appel de procédure distante qui ne modifie pas un État et retourne les mêmes informations chaque fois qu’il est appelé avec les mêmes paramètres d’entrée.

</dd> <dt>

<span id="_rpc_input_parameter_glos"></span><span id="_RPC_INPUT_PARAMETER_GLOS"></span>**paramètre d’entrée**
</dt> <dd>

Paramètre contenant les données que l’application cliente transmet à l’application serveur. L’attribut **in** désigne un paramètre d’entrée.

</dd> <dt>

<span id="_rpc_idl_glos"></span><span id="_RPC_IDL_GLOS"></span>**Langage IDL (Interface Definition Language)**
</dt> <dd>

Le langage standard OSF-DCE permettant de spécifier l’interface pour les appels de procédure distante. Voir aussi [*MIDL*](m-glos.md).

</dd> <dt>

<span id="_rpc_ip_glos"></span><span id="_RPC_IP_GLOS"></span>**Protocole Internet (IP)**
</dt> <dd>

Protocole de communication de couche réseau sans connexion. Voir aussi [*datagramme*](d-glos.md).

</dd> <dt>

<span id="_rpc_ipx_glos"></span><span id="_RPC_IPX_GLOS"></span>**Exchange de paquets interréseau (IPX)**
</dt> <dd>

Protocole de communication Novell NetWare qui utilise des sockets de [*datagramme*](d-glos.md) pour acheminer des paquets d’informations sur des réseaux locaux et des réseaux étendus.

</dd> <dt>

<span id="_rpc_intranet_glos"></span><span id="_RPC_INTRANET_GLOS"></span>**réseau**
</dt> <dd>

Réseau privé qui utilise des produits et des technologies Internet (par exemple, des serveurs Web), mais qui n’est pas disponible pour les utilisateurs Internet externes.

</dd> </dl>
