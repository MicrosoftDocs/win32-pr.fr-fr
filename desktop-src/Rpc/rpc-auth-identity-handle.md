# <a name="rpc_auth_identity_handle"></a>\_handle d' \_ identité d’authentification RPC \_

Le type de données du handle de l' **\_ identité d’authentification \_ \_ RPC** déclare un handle qui pointe vers une structure contenant les informations d’identification d’authentification et d’autorisation du client spécifiées pour les appels de procédure distante.


```C++
typedef void* RPC_AUTH_IDENTITY_HANDLE;
```
