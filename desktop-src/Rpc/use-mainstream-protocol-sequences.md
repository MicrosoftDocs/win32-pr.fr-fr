# <a name="use-mainstream-protocol-sequences"></a>Utiliser des séquences de protocole courantes

Utilisez [**ncacn \_ IP \_ TCP**](/windows/desktop/Midl/ncacn-ip-tcp), [**ncacn \_ http**](/windows/desktop/Midl/ncacn-http) ou [**ncacn \_ NP**](/windows/desktop/Midl/ncacn-np) pour les appels RPC distants. Utilisez [**Ncalrpc**](/windows/desktop/Midl/ncalrpc) pour les appels RPC locaux. Ces protocoles sont utilisés le plus souvent et bénéficient de la surveillance et des tests les plus courants.
