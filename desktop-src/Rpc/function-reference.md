# <a name="function-reference"></a>Référence de fonction

Cette section décrit les fonctions d’exécution de l’appel de procédure distante (RPC) prises en charge dans Microsoft RPC. Cette section décrit le rôle, la syntaxe, les paramètres d’entrée et les valeurs de retour de chaque fonction. Il fournit également des informations supplémentaires pour vous aider à utiliser les fonctions RPC dans une application.

Tous les pointeurs passés aux fonctions RPC doivent inclure l’attribut **\_ \_ RPC \_ Far** . Par exemple, le **\_ \_ handle \* de liaison RPC** du pointeur devient le **handle de liaison RPC \_ \_ \_ \_ \_ \*** **\* \***   **\_ \_ \_ \* \_ \_ \_ FAR et char PTR devient char RPC Far Far PTR. \***
