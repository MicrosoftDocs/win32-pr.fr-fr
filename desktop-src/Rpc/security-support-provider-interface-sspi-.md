# <a name="security-support-provider-interface-sspi"></a>interface du fournisseur de la prise en charge de la sécurité (Security Support Provider Interface ou SSPI)

Conjointement avec ses systèmes d’exploitation, Microsoft offre l’interface SSPI (Security Support Provider Interface). L’interface SSPI fournit une interface universelle et standard pour les applications distribuées sécurisées.
