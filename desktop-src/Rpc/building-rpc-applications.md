# <a name="building-rpc-applications"></a>Génération d’applications RPC

La procédure de création d’une application d’appel de procédure distante (RPC) distribuée varie légèrement selon les éléments suivants :

-   Plateforme du système d’exploitation sur laquelle vous développez.
-   Plateforme cible.
-   Version du compilateur MIDL et C ou C++ que vous utilisez.
-   Bibliothèques d’API que vous utilisez.

Cette section décrit le processus de création d’applications client/serveur avec un appel de procédure distante Microsoft. Il présente les informations dans les rubriques suivantes :

-   [Procédure de génération générale](general-build-procedure.md)
-   [Gestion des exceptions](exception-handling.md)
