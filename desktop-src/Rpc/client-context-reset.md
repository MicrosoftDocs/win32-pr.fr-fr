# <a name="client-context-reset"></a>Réinitialisation du contexte client

Lorsque le serveur devient indisponible, l’application cliente peut libérer ses données de contexte en appelant la fonction RPC [**RpcSsDestroyClientContext**](/windows/desktop/api/Rpcndr/nf-rpcndr-rpcssdestroyclientcontext).
