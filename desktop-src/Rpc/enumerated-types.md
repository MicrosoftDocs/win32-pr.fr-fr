# <a name="enumerated-types-rpc"></a>Types énumérés (RPC)

La Déclaration [**enum**](/windows/desktop/Midl/enum) n’est pas convertie en instructions **\# define** , comme c’est le cas par certains compilateurs DCE, mais elle est reproduite sous la forme d’une déclaration **enum** de langage C dans le fichier d’en-tête généré.
