# <a name="the-interface-definition-language-idl-file"></a>Fichier IDL (Interface Definition Language)

Un fichier IDL contient une ou plusieurs définitions d’interface. Chaque définition d’interface est composée d’un en-tête d’interface et d’un corps d’interface. L’en-tête d’interface est délimité par des crochets. Le corps de l’interface est contenu entre accolades. Cela est illustré dans l’exemple d’interface suivant :

``` syntax
[
  /*Interface attributes go here. */
]
interface INTERFACENAME
{
  /*The interface body goes here. */
}
```

Cette section donne une vue d’ensemble des composants d’une interface. Elle est organisée dans les rubriques suivantes :

-   [En-tête d’interface IDL](the-idl-interface-header.md)
-   [Corps de l’interface IDL](the-idl-interface-body.md)
