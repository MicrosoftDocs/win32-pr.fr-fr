# <a name="data-and-language-features"></a>Fonctionnalités de langage et de données

Le Microsoft Interface Definition Language (MIDL) fournit l’ensemble des fonctionnalités qui étendent le langage de programmation C pour prendre en charge les appels de procédure distante. MIDL n’est pas une variante de C ; Il s’agit d’un langage formel fortement typé par le biais duquel vous pouvez contrôler les données transmises sur un réseau. MIDL est conçu de sorte que les développeurs familiarisés avec C puissent l’apprendre rapidement.

Cette rubrique présente trois fonctionnalités de langage :

-   [Typage fort](strong-typing.md)
-   [Attributs directionnels (paramètre)](directional-parameter-attributes.md)
-   [Représentation des données](data-representation.md)
