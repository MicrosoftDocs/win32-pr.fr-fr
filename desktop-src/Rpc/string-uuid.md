# <a name="string-uuid"></a>UUID de chaîne

Un UUID de chaîne contient la représentation sous forme de tableau de caractères d’un UUID. Un UUID de chaîne se compose de plusieurs champs de caractères hexadécimaux. Chaque membre a une longueur fixe et les champs sont séparés par le trait d’Union. Par exemple :

`989C6E5C-2CC1-11CA-A044-08002B1BB4F5`

Lorsque vous fournissez un UUID de chaîne en tant que paramètre d’entrée à une fonction runtime RPC, entrez les caractères hexadécimaux alphabétiques comme caractères majuscules ou minuscules. Toutefois, lorsqu’une fonction runtime retourne un UUID de chaîne, les caractères hexadécimaux sont toujours en minuscules.
