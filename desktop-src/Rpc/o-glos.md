<span id="_rpc_odl_glos"></span><span id="_RPC_ODL_GLOS"></span>**ODL (Object Description Language)**
</dt> <dd>

Sous-ensemble d’attributs, de mots clés, d’instructions et de directives [*MIDL*](m-glos.md) utilisés pour définir des bibliothèques de types pour les applications OLE Automation.

</dd> <dt>

<span id="_rpc_open_array_glos"></span><span id="_RPC_OPEN_ARRAY_GLOS"></span>**ouvrir un tableau**

</dd> <dt>

<span id="_rpc_osf_glos"></span><span id="_RPC_OSF_GLOS"></span>**OSF (Open Software Foundation)**

</dd> <dt>

<span id="_rpc_orphaned_memory_glos"></span><span id="_RPC_ORPHANED_MEMORY_GLOS"></span>**mémoire orpheline**

</dd> <dt>

<span id="_rpc_output_parameter_glos"></span><span id="_RPC_OUTPUT_PARAMETER_GLOS"></span>**paramètre de sortie**

</dd> </dl>
