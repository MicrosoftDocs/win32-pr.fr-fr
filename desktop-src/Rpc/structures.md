# <a name="structs-rpc"></a>Structs (RPC)

La sémantique C normale s’applique aux champs de types de base. Les champs de types plus complexes, tels que des pointeurs, des tableaux et d’autres types construits, peuvent être modifiés par [**des \_ attributs**](field-attributes.md)de type ou de champ. Pour plus d’informations, consultez [**struct**](/windows/desktop/Midl/struct).
