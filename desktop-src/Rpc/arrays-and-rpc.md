# <a name="arrays-and-rpc"></a>Tableaux et RPC

Les langages de programmation C et C++ fournissent essentiellement deux types de tableaux : unidimensionnel et multidimensionnel. RPC permet aux développeurs de spécifier des types de tableau supplémentaires à l’aide des attributs MIDL pour décrire les caractéristiques des tableaux dans les applications distribuées.

Cette section décrit les types de tableaux disponibles sous RPC. Il aborde également les attributs MIDL que les développeurs peuvent utiliser pour décrire des tableaux unidimensionnels et multidimensionnels. Ce document est présenté dans les rubriques suivantes :

-   [Genres de tableaux](kinds-of-arrays.md)
-   [Attributs de tableau](array-attributes.md)
-   [Tableaux multidimensionnels](multidimensional-arrays.md)
