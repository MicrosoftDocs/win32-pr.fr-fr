# <a name="authentication-services"></a>Services d'authentification

L’interface SSPI (Security Support Provider Interface) fournit les services d’authentification sous-jacents pour RPC. Par conséquent, lorsque votre application spécifie un service d’authentification, elle sélectionne un SSP. Pour obtenir la liste des fournisseurs de services partagés actuellement pris en charge par SPPI, consultez la page relative aux [constantes de service d’authentification](authentication-service-constants.md). Pour plus d’informations, consultez [SSPI (Security Support Provider Interface)](security-support-provider-interface-sspi-.md).

> [!Note]  
> Les termes fournisseur de sécurité et service d’authentification sont interchangeables.
