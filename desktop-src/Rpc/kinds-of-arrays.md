# <a name="kinds-of-arrays"></a>Genres de tableaux

MIDL offre la possibilité de spécifier les types de tableaux suivants dans votre application RPC :

-   [Tableaux fixes](fixed-arrays.md)
-   [Tableaux variables](varying-arrays.md)
-   [Tableaux conformes](conformant-arrays.md)

Les trois types de tableau peuvent être utilisés en tant que \[ paramètres [in](/windows/desktop/Midl/in) \] , \[ [out](/windows/desktop/Midl/out-idl) \] ou **\[ in, out \]** .
