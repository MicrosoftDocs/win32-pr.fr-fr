# <a name="n-rpc"></a>N (RPC)

[A](a-glos.md) [B](b-glos.md) [C](c-glos.md) [D](d-glos.md) [e](e-glos.md) [F](f-glos.md) G H [I](i-glos.md) J K [L](l-glos.md) [M](m-glos.md) N [O](o-glos.md) [P](p-glos.md) [Q](q.md) [R](r-glos.md) [S](s-glos.md) [T](t-glos.md) [U](u-glos.md) [V](v-glos.md) [W](w-glos.md) X Y Z

<dl> <dt>

<span id="_rpc_name_service_glos"></span><span id="_RPC_NAME_SERVICE_GLOS"></span>**service de noms**
</dt> <dd>

Service qui mappe les noms aux objets et stocke les paires nom/objet dans une base de données. Par exemple, le service de noms RPC mappe un nom logique à un [*handle de liaison*](b-glos.md) afin que les applications clientes puissent faire référence à ce nom logique, plutôt qu’à une séquence de protocole et à une adresse réseau. Voir aussi nom service – interface daemon ([NSID)](/windows), service d’annuaire client ([*CDS)*](c-glos.md), [*localisateur*](l-glos.md).

</dd> <dt>

<span id="_rpc_nca_glos"></span><span id="_RPC_NCA_GLOS"></span>**Architecture NCA (Network Computing Architecture)**
</dt> <dd>

Collection d’instructions pour l’informatique distribuée. Les protocoles de communication RPC suivent ces instructions.

</dd> <dt>

<span id="_rpc_nsi_glos"></span><span id="_RPC_NSI_GLOS"></span>**Service de noms indépendant (NSI)**
</dt> <dd>

Standard pour les fonctions d’API qui permettent à une application distribuée d’accéder à des éléments de base de données de service de noms RPC par le biais de différents fournisseurs de services de noms, tels que OSF-DCE service d’annuaire de cellules/service d’annuaires de cellules ou Microsoft Locator. Voir aussi [-démon d’interface](/windows) (NSID).

</dd> <dt>

<span id="_rpc_nsid_glos"></span><span id="_RPC_NSID_GLOS"></span>**nom du service d’interface (NSID)**
</dt> <dd>

Service qui fournit une interface entre Microsoft [*Locator*](l-glos.md) et les bases de données de service de nom service d’annuaire de cellules/service d’annuaires de cellules OSF-DCE pour les fonctions de service de nom RPC.

</dd> <dt>

<span id="_rpc_named_pipe_glos"></span><span id="_RPC_NAMED_PIPE_GLOS"></span>**canal nommé**
</dt> <dd>

Protocole orienté connexion, basé sur les blocs SMB (Server Message Blocks) et [NetBIOS](/windows), utilisé pour la communication entre un processus serveur et un ou plusieurs processus client.

</dd> <dt>

<span id="_rpc_netbeui_glos"></span><span id="_RPC_NETBEUI_GLOS"></span>**Interface utilisateur étendue NetBIOS (NetBEUI)**
</dt> <dd>

Protocole de transport natif LAN Manager et pilote de périphérique réseau. Voir aussi [NetBIOS](/windows).

</dd> <dt>

<span id="_rpc_netbios_glos"></span><span id="_RPC_NETBIOS_GLOS"></span>**NetBIOS (Network Basic Input/Output System)**
</dt> <dd>

Interface logicielle entre le système d’exploitation Microsoft MS-DOS, le bus des e/s et un réseau local.

</dd> <dt>

<span id="_rpc_ndr_glos"></span><span id="_RPC_NDR_GLOS"></span>**Représentation de données réseau (NDR)**
</dt> <dd>

Format standard utilisé pendant la transmission réseau qui est indépendant du format de type de données sur n’importe quelle architecture d’ordinateur particulière. Les données transmises incluent des informations qui spécifient son format de rapport de non-remise.

</dd> <dt>

<span id="_rpc_network_address_glos"></span><span id="_RPC_NETWORK_ADDRESS_GLOS"></span>**adresse réseau**
</dt> <dd>

Adresse qui identifie un serveur sur un réseau.

</dd> <dt>

<span id="_rpc_nonencapsulated_union_glos"></span><span id="_RPC_NONENCAPSULATED_UNION_GLOS"></span>**Union qui n’est pas encapsulée**
</dt> <dd>

[*Union discriminée*](d-glos.md) qui est moins restrictive qu’une Union encapsulée en ce que discriminante et l’Union ne sont pas étroitement liés. Si l’Union est un paramètre, discriminante est un autre paramètre ; Si l’Union est un champ de structure, discriminante est un autre champ de structure. Le commutateur des mots clés IDL \[ [**\_ est**](/windows/desktop/Midl/switch-is) \] et le \[ [**\_ type de commutateur**](/windows/desktop/Midl/switch-type) \] identifient le discriminante et son type. Voir aussi [*Union encapsulée*](e-glos.md).

</dd> <dt>

<span id="_rpc_nonidempotent_glos"></span><span id="_RPC_NONIDEMPOTENT_GLOS"></span>**nonidempotent**
</dt> <dd>

Indicateur signalant qu’un appel de procédure distante ne peut pas être exécuté plus d’une fois, car il renvoie une valeur différente ou modifie un État.

</dd> </dl>
