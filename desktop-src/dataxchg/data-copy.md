# <a name="data-copy"></a>Copie de données

La fonctionnalité de copie de données vous permet d’envoyer des données d’une application à une autre.

### <a name="in-this-section"></a>Dans cette section



| Nom                                                      | Description                                                                                        |
|-----------------------------------------------------------|----------------------------------------------------------------------------------------------------|
| [Utilisation de la copie de données](using-data-copy.md)<br/>         | Fournit un exemple qui montre comment envoyer des informations entre deux applications.<br/> |
| [Référence de copie de données](data-copy-reference.md)<br/> | Contient la référence de l’API.<br/>                                                             |



 

### <a name="data-copy-messages"></a>Messages de copie de données



| Nom                                           | Description                                           |
|------------------------------------------------|-------------------------------------------------------|
| [**\_COPYDATA WM**](wm-copydata.md)<br/> | Envoyé pour passer des données à une autre application. <br/> |



 

### <a name="data-copy-structures"></a>Structures de copie des données



| Nom                                                | Description                                                                                                       |
|-----------------------------------------------------|-------------------------------------------------------------------------------------------------------------------|
| [**COPYDATASTRUCT**](/windows/win32/api/winuser/ns-winuser-copydatastruct)<br/> | Contient les données à passer à une autre application par le message [**WM \_ COPYDATA**](wm-copydata.md) . <br/> |
