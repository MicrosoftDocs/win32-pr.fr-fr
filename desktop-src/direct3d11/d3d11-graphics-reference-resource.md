# <a name="resource-reference-direct3d-11-graphics"></a>Référence de ressource (graphiques Direct3D 11)

L’API Direct3D définit plusieurs éléments d’API pour vous aider à créer et à gérer des ressources.


## <a name="in-this-section"></a>Contenu de cette section



| Rubrique                                                                              | Description                                                                                                                                                                                                                                                                                                                                               |
|------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Interfaces de ressource](d3d11-graphics-reference-resource-interfaces.md)<br/> | Il existe plusieurs interfaces pour les deux types de ressources de base : les tampons et les textures. Il existe également des interfaces pour les vues des ressources. <br/> Une application utilise une vue pour lier une ressource à une étape de pipeline. La vue définit comment la ressource est accessible pendant le rendu. Cette section décrit les interfaces de vue. <br/> |
| [Fonctions de ressource](d3d11-graphics-reference-resource-functions.md)<br/>   | Cette section contient des informations sur les fonctions de ressources.<br/>                                                                                                                                                                                                                                                                                |
| [Structures de ressources](d3d11-graphics-reference-resource-structures.md)<br/> | Les structures sont utilisées pour créer et utiliser des ressources.<br/>                                                                                                                                                                                                                                                                                               |
| [Énumérations de ressources](d3d11-graphics-reference-resource-enums.md)<br/>    | Les énumérations sont utilisées pour spécifier des informations sur la façon dont les ressources sont créées et accessibles pendant le rendu.<br/>                                                                                                                                                                                                                                    |
