# <a name="d3d11_rect"></a>D3D11 \_ Rect

D3D11 \_ Rect est déclaré comme suit :

``` syntax
typedef RECT D3D11_RECT;
```

Pour plus d’informations sur cette structure de rectangles GDI, consultez [**Rect**](/previous-versions//dd162897(v=vs.85)).

## <a name="remarks"></a>Notes

Cette structure est utilisée pour les rectangles de ciseaux par [**ID3D11DeviceContext :: RSGetScissorRects**](/windows/desktop/api/D3D11/nf-d3d11-id3d11devicecontext-rsgetscissorrects) et [**ID3D11DeviceContext :: RSSetScissorRects**](/windows/desktop/api/D3D11/nf-d3d11-id3d11devicecontext-rssetscissorrects).
