<span data-ttu-id="dc714-107">Utilisez une liste de commandes dans les scénarios suivants :</span><span class="sxs-lookup"><span data-stu-id="dc714-107">Use a command list in the following scenarios:</span></span>

-   <span data-ttu-id="dc714-111">Pré-enregistre une liste de commandes avant de devoir la restituer (par exemple, pendant le chargement d’un niveau) et la relire efficacement plus tard dans votre scène.</span><span class="sxs-lookup"><span data-stu-id="dc714-111">Pre-record a command list before you need to render it (for example, while a level is loading) and efficiently play it back later in your scene.</span></span> <span data-ttu-id="dc714-112">Cette optimisation fonctionne bien lorsque vous devez effectuer un rendu fréquent.</span><span class="sxs-lookup"><span data-stu-id="dc714-112">This optimization works well when you need to render something often.</span></span>

<span data-ttu-id="dc714-113">Une liste de commandes est immuable et conçue pour être enregistrée et lue lors d’une seule exécution d’une application.</span><span class="sxs-lookup"><span data-stu-id="dc714-113">A command list is immutable and is designed to be recorded and played back during a single execution of an application.</span></span> <span data-ttu-id="dc714-114">Une liste de commandes n’est pas conçue pour être préenregistrée avant l’exécution du jeu et chargée à partir de votre média, car il n’existe aucun moyen de conserver la liste.</span><span class="sxs-lookup"><span data-stu-id="dc714-114">A command list is not designed to be pre-recorded ahead of game execution and loaded from your media as there is no way to persist the list.</span></span>
