# <a name="d3dx-11-reference"></a>Référence D3DX 11

L’API D3DX 11 est décrite dans cette section.

> [!NOTE]  
> La bibliothèque d’utilitaires D3DX (D3DX 9, D3DX 10 et D3DX 11) est déconseillée pour Windows 8 et n’est pas prise en charge pour les applications Microsoft Store. Pour plus d’informations sur les remplacements recommandés pour D3DX11, consultez [vivre sans D3DX](https://walbourn.github.io/living-without-d3dx/). Pour les projets plus anciens, vous pouvez utiliser le package NuGet [Microsoft. DXSDK. D3DX](https://www.nuget.org/packages/Microsoft.DXSDK.D3DX) au lieu de vous appuyer sur le SDK DirectX hérité ou DirectSetup.
