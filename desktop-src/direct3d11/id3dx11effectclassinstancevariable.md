# <a name="id3dx11effectclassinstancevariable-interface"></a>Interface ID3DX11EffectClassInstanceVariable

Accède à une instance de classe.

## <a name="members"></a>Membres

L’interface **ID3DX11EffectClassInstanceVariable** hérite de [**ID3DX11EffectVariable**](id3dx11effectvariable.md). **ID3DX11EffectClassInstanceVariable** a également les types de membres suivants :

-   [Méthodes](#methods)

### <a name="methods"></a>Méthodes

L’interface **ID3DX11EffectClassInstanceVariable** possède ces méthodes.



| Méthode                                                                          | Description                       |
|:--------------------------------------------------------------------------------|:----------------------------------|
| [**GetClassInstance**](id3dx11effectclassinstancevariable-getclassinstance.md) | Obtient une instance de classe.<br/> |



 

## <a name="remarks"></a>Notes

> [!Note]  
> Le kit de développement logiciel (SDK) DirectX ne fournit aucun binaire compilé pour les effets. Vous devez utiliser la source Effects 11 pour créer votre application Effects-type. Pour plus d’informations sur l’utilisation de la source Effects 11, consultez [différences entre les effets 10 et 11](d3d11-graphics-programming-guide-effects-differences.md).
