# <a name="id3dx11effectvariableasinterface-method"></a>ID3DX11EffectVariable :: AsInterface, méthode

Obtient une variable d’interface.

## <a name="syntax"></a>Syntaxe


```C++
ID3DX11EffectInterfaceVariable* AsInterface();
```



## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.

## <a name="return-value"></a>Valeur de retour

Type : **[ **ID3DX11EffectInterfaceVariable**](id3dx11effectinterfacevariable.md)\***

Pointeur vers une variable d’interface. Consultez [**ID3DX11EffectInterfaceVariable**](id3dx11effectinterfacevariable.md).

## <a name="remarks"></a>Remarques

> [!Note]  
> Le kit de développement logiciel (SDK) DirectX ne fournit aucun binaire compilé pour les effets. Vous devez utiliser la source Effects 11 pour créer votre application Effects-type. Pour plus d’informations sur l’utilisation de la source Effects 11, consultez [différences entre les effets 10 et 11](d3d11-graphics-programming-guide-effects-differences.md).
