# <a name="id3dx11effectinterfacevariable-interface"></a>Interface ID3DX11EffectInterfaceVariable

Accède à une variable d’interface.

## <a name="members"></a>Membres

L’interface **ID3DX11EffectInterfaceVariable** hérite de [**ID3DX11EffectVariable**](id3dx11effectvariable.md). **ID3DX11EffectInterfaceVariable** a également les types de membres suivants :

-   [Méthodes](#methods)

### <a name="methods"></a>Méthodes

L’interface **ID3DX11EffectInterfaceVariable** possède ces méthodes.



| Méthode                                                                      | Description                       |
|:----------------------------------------------------------------------------|:----------------------------------|
| [**GetClassInstance**](id3dx11effectinterfacevariable-getclassinstance.md) | Obtient une instance de classe.<br/>  |
| [**SetClassInstance**](id3dx11effectinterfacevariable-setclassinstance.md) | Définit une instance de classe.<br/> |



 

## <a name="remarks"></a>Remarques

> [!Note]  
> Le kit de développement logiciel (SDK) DirectX ne fournit aucun binaire compilé pour les effets. Vous devez utiliser la source Effects 11 pour créer votre application Effects-type. Pour plus d’informations sur l’utilisation de la source Effects 11, consultez [différences entre les effets 10 et 11](d3d11-graphics-programming-guide-effects-differences.md).
