# <a name="id3dx11effectstringvariable-interface"></a>Interface ID3DX11EffectStringVariable

Une interface de variable de chaîne accède à une variable de chaîne.

## <a name="members"></a>Membres

L’interface **ID3DX11EffectStringVariable** hérite de [**ID3DX11EffectVariable**](id3dx11effectvariable.md). **ID3DX11EffectStringVariable** a également les types de membres suivants :

-   [Méthodes](#methods)

### <a name="methods"></a>Méthodes

L’interface **ID3DX11EffectStringVariable** possède ces méthodes.



| Méthode                                                               | Description                         |
|:---------------------------------------------------------------------|:------------------------------------|
| [**GetString**](id3dx11effectstringvariable-getstring.md)           | Obtient la chaîne.<br/>          |
| [**GetStringArray**](id3dx11effectstringvariable-getstringarray.md) | Obtient un tableau de chaînes.<br/> |



 

## <a name="remarks"></a>Remarques

> [!Note]  
> Le kit de développement logiciel (SDK) DirectX ne fournit aucun binaire compilé pour les effets. Vous devez utiliser la source Effects 11 pour créer votre application Effects-type. Pour plus d’informations sur l’utilisation de la source Effects 11, consultez [différences entre les effets 10 et 11](d3d11-graphics-programming-guide-effects-differences.md).
