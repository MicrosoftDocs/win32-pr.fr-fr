# <a name="common-version-reference"></a>Référence de version commune

L’API Direct3D définit plusieurs éléments d’API communs aux Direct3D 12, Direct3D 11, Direct3D 10 et Direct3D 10,1. Vous pouvez utiliser ces éléments d’API dans votre code pour l’une de ces versions Direct3D. Ces éléments d’API sont connus comme étant indépendants de la version.
