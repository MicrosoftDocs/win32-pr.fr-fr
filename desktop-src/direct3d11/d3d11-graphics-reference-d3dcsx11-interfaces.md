# <a name="d3dcsx-11-interfaces"></a>Interfaces D3DCSX 11

Cette section contient des informations de référence sur les interfaces COM fournies par la bibliothèque de l’utilitaire D3DCSX.


## <a name="in-this-section"></a>Contenu de cette section



| Rubrique                                                           | Description                                       |
|-----------------------------------------------------------------|---------------------------------------------------|
| [**ID3DX11FFT**](/windows/desktop/api/d3dcsx/nn-d3dcsx-id3dx11fft)<br/>                     | Encapsule les FFTss vers l’avant et vers l’inverse.<br/> |
| [**ID3DX11Scan**](/windows/desktop/api/d3dcsx/nn-d3dcsx-id3dx11scan)<br/>                   | Contexte d’analyse.<br/>                          |
| [**ID3DX11SegmentedScan**](/windows/desktop/api/d3dcsx/nn-d3dcsx-id3dx11segmentedscan)<br/> | Contexte d’analyse segmentée.<br/>                |
