# <a name="types-of-resources"></a>Types de ressources

Les ressources sont des zones de mémoire accessibles par le pipeline Direct3D. il s’agit des blocs de construction de votre scène. Les ressources contiennent de nombreux types de données, telles que des données géométriques, des textures ou des nuanceurs, que Direct3D utilise pour remplir et afficher votre scène. Cette rubrique décrit les types de ressources de Direct3D 10, ainsi que les nouveaux types dans Direct3D 11, y compris les mémoires tampons structurées et les textures et mémoires tampons accessibles en écriture.
