# <a name="opcodes-metadatatype-element"></a>OpCodes (élément de type de données)

Définit une liste d’OpCodes que vous pouvez utiliser pour regrouper des événements au sein d’une tâche.

``` syntax
<xs:element name="opcodes"
    type="OpcodeListType"
 />
```

L’élément **OpCodes** est défini par le type complexe de l’élément de [**données**](eventmanifestschema-metadatatype-complextype.md) .
