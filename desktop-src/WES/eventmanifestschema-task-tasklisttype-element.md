# <a name="task-tasklisttype-element"></a>Élément Task (TaskListType)

Définit un composant ou un sous-composant d’une application.

``` syntax
<xs:element name="task"
    type="TaskType"
 />
```

L’élément **Task** est défini par le type complexe [**TaskListType**](eventmanifestschema-tasklisttype-complextype.md) .
