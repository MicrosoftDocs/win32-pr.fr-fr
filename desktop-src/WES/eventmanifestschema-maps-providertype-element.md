# <a name="maps-providertype-element"></a>Élément Maps (ProviderType)

Définit une liste de paires nom/valeur que vous pouvez référencer dans la section de modèle du manifeste.

``` syntax
<xs:element name="maps"
    type="MapType"
 />
```

L’élément **Maps** est défini par le type complexe [**ProviderType**](eventmanifestschema-providertype-complextype.md) .
