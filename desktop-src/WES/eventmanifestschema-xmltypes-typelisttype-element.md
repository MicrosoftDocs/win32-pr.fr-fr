# <a name="xmltypes-typelisttype-element"></a>Élément xmlTypes (TypeListType)

Définit une liste de types de sortie que le service utilise pour déterminer comment restituer un type de données d’entrée.

``` syntax
<xs:element name="xmlTypes"
    type="XmlTypeListType"
 />
```

L’élément **xmlTypes** est défini par le type complexe [**TypeListType**](eventmanifestschema-typelisttype-complextype.md) .
