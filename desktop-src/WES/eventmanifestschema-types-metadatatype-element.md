# <a name="types-metadatatype-element"></a>types (type de données) (élément)

Définit une liste de types XML.

``` syntax
<xs:element name="types"
    type="TypeListType"
 />
```

L’élément **types** est défini par le type complexe de l’élément de [**données**](eventmanifestschema-metadatatype-complextype.md) .
