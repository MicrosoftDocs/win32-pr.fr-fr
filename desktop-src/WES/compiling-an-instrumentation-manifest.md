# <a name="compiling-an-instrumentation-manifest"></a>Compilation d’un manifeste d’instrumentation

Après avoir [écrit votre manifeste](writing-an-instrumentation-manifest.md), utilisez le compilateur de message pour valider le manifeste et générer les fichiers de ressources et d’en-tête que vous incluez dans votre fournisseur. Pour plus d’informations sur l’utilisation du compilateur, consultez [**message compiler**](message-compiler--mc-exe-.md). Incluez l’en-tête et les fichiers de ressources que le compilateur génère dans le projet qui contient le reste du code source de votre fournisseur. Pour plus d’informations sur la façon d’écrire les événements définis dans votre manifeste, consultez [développement d’un fournisseur](developing-a-provider.md).
