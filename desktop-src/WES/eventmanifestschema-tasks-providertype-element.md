# <a name="tasks-providertype-element"></a>Élément Tasks (ProviderType)

Définit une liste de tâches qu’un fournisseur peut utiliser pour regrouper des événements.

``` syntax
<xs:element name="tasks"
    type="TaskListType"
 />
```

L’élément **Tasks** est défini par le type complexe [**ProviderType**](eventmanifestschema-providertype-complextype.md) .
