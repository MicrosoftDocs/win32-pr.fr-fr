# <a name="map-valuemaptype-element"></a>Élément MAP (ValueMapType)

Définit le mappage entre une valeur entière et une valeur de chaîne.

``` syntax
<xs:element name="map"
    type="ValueMapValueType"
 />
```

L’élément **cartographique** est défini par le type complexe [**ValueMapType**](eventmanifestschema-valuemaptype-complextype.md) .
