# <a name="query-schema-elements"></a>Éléments de schéma de requête

Voici les éléments définis par le schéma de requête. Cette section contient les noms des éléments que vous pouvez utiliser dans votre requête. Toutefois, pour obtenir les détails de chaque élément, consultez le type complexe qui contient l’élément.



| Élément                                                                  | Description                                                                                                                     |
|--------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------|
| [**QueryList**](queryschema-querylist-element.md)                       | Nœud racine de la requête.<br/>                                                                                          |
| [**Requête (QueryListType)**](queryschema-query-querylisttype-element.md) | Définit un ensemble de sélecteurs et de suppressions utilisés pour inclure des événements dans ou exclure des événements du jeu de résultats.<br/> |
| [**Select (QueryType)**](queryschema-select-querytype-element.md)       | Requête XPath qui identifie les événements à inclure dans le jeu de résultats de la requête.<br/>                                        |
| [**Supprimer (QueryType)**](queryschema-suppress-querytype-element.md)   | Requête XPath qui identifie les événements à exclure du jeu de résultats de la requête.<br/>                                      |
