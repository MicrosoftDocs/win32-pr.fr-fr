# <a name="instrumentation-instrumentationmanifest-element"></a>Élément Instrumentation (instrumentationManifest)

Définit les types d’instrumentation que le manifeste contient (par exemple, l’instrumentation d’événement ou de compteur).

``` syntax
<xs:element name="instrumentation"
    type="InstrumentationType"
 />
```

L’élément d' **instrumentation** est défini par l’élément [**instrumentationManifest**](eventmanifestschema-instrumentationmanifest-element.md) .
