# <a name="maxbuffers-channelpublishingtype-element"></a>Élément maxBuffers (ChannelPublishingType)

Nombre maximal de mémoires tampons à allouer pour la session.

``` syntax
<xs:element name="maxBuffers"
    type="unsignedInt"
 />
```

L’élément **maxBuffers** est défini par le type complexe [**ChannelPublishingType**](eventmanifestschema-channelpublishingtype-complextype.md) .
