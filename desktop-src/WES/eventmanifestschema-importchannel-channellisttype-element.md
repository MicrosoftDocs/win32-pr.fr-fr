# <a name="importchannel-channellisttype-element"></a>Élément importChannel (ChannelListType)

Identifie un canal qui a été défini par un autre fournisseur.

``` syntax
<xs:element name="importChannel"
    type="ImportChannelType"
 />
```

L’élément **importChannel** est défini par le type complexe [**ChannelListType**](eventmanifestschema-channellisttype-complextype.md) .
