# <a name="stringtable-resources-element"></a>Élément stringTable (Resources)

Définit une liste de chaînes localisées que vous pouvez référencer dans votre manifeste.

``` syntax
<xs:element name="stringTable"
    type="StringTableType"
 />
```

L’élément **STRINGTABLE** est défini par l’élément [**Resources**](eventmanifestschema-resources-localizationtype-element.md) .
