# <a name="patternmaps-namedquerytype-element"></a>Élément patternMaps (NamedQueryType)

Définit une liste de paires d’expressions régulières utilisées pour modifier la chaîne de message.

``` syntax
<xs:element name="patternMaps"
    type="PatternMapListType"
 />
```

L’élément **patternMaps** est défini par le type complexe [**NamedQueryType**](eventmanifestschema-namedquerytype-complextype.md) .
