# <a name="binaryeventdata-eventtype-element"></a>Élément BinaryEventData (EventType)

Contient les données d’événement sous la forme d’un objet blob binaire.

``` syntax
<xs:element name="BinaryEventData"
    type="hexBinary"
 />
```

L’élément **BinaryEventData** est défini par le type complexe [**eventType**](eventschema-eventtype-complextype.md) .
