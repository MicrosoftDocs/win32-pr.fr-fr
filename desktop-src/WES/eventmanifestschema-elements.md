# <a name="eventmanifest-schema-elements"></a>Éléments de schéma EventManifest

Voici les éléments définis par le schéma EventManifest. Cette section contient les noms des éléments que vous utilisez dans votre manifeste. Toutefois, pour obtenir les détails de chaque élément, consultez le type complexe qui contient l’élément.



| Élément                                                                                                                  | Description                                                                                                                                                                                                                                                                          |
|--------------------------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Sauvegarde autoChannelLoggingType**](eventmanifestschema-autobackup-channelloggingtype-element.md)                     | Détermine s’il est impossible de créer un nouveau fichier journal lorsque le fichier journal actuel atteint sa taille maximale.<br/>                                                                                                                                                                           |
| [**binaire (TemplateItemType)**](eventmanifestschema-binary-templateitemtype-element.md)                                 | Contient les données binaires fournies par l’API du [Journal des événements](/windows/desktop/EventLog/event-logging) .<br/>                                                                                                                                                                                     |
| [**bitMap (MapType)**](eventmanifestschema-bitmap-maptype-element.md)                                                   | Définit une liste de paires nom/valeur qui mappent les valeurs de bit et les valeurs de chaîne.<br/>                                                                                                                                                                                                 |
| [**bufferSize (ChannelPublishingType)**](eventmanifestschema-buffersize-channelpublishingtype-element.md)               | Contient la quantité de mémoire, en kilo-octets, allouée pour les mémoires tampons de session.<br/>                                                                                                                                                                                           |
| [**canal (ChannelListType)**](eventmanifestschema-channel-channellisttype-element.md)                                 | Définit un canal dans lequel les événements peuvent être enregistrés.<br/>                                                                                                                                                                                                                          |
| [**canaux (type de données)**](eventmanifestschema-channels-metadatatype-element.md)                                     | Définit une liste de canaux auxquels les fournisseurs peuvent enregistrer des événements.<br/>                                                                                                                                                                                                             |
| [**canaux (ProviderType)**](eventmanifestschema-channels-providertype-element.md)                                     | Définit une liste de canaux auxquels les fournisseurs peuvent enregistrer des événements.<br/>                                                                                                                                                                                                             |
| [**clockType (ChannelPublishingType)**](eventmanifestschema-clocktype-channelpublishingtype-element.md)                 | Résolution d’horloge à utiliser lors de la journalisation de l’horodatage pour chaque événement.<br/>                                                                                                                                                                                                   |
| [**controlGuid (ChannelPublishingType)**](eventmanifestschema-controlguid-channelpublishingtype-element.md)             | Identifie le GUID de session pour une session ETW qui contient des événements WPP.<br/>                                                                                                                                                                                                  |
| [**données (StructDefinitionType)**](eventmanifestschema-data-structdefinitiontype-element.md)                             | Définit un élément de données que vous souhaitez inclure dans la structure.<br/>                                                                                                                                                                                                            |
| [**données (TemplateItemType)**](eventmanifestschema-data-templateitemtype-element.md)                                     | Définit un élément de données que vous souhaitez inclure avec l’événement.<br/>                                                                                                                                                                                                              |
| [**Description (InputType)**](eventmanifestschema-description-inputtype-element.md)                                     | Description du type d’entrée.<br/>                                                                                                                                                                                                                                          |
| [**événement (DefinitionType)**](eventmanifestschema-event-definitiontype-element.md)                                       | Définit un événement que votre fournisseur peut journaliser.<br/>                                                                                                                                                                                                                              |
| [**événement (OpCode)**](eventmanifestschema-event-opcode-element.md)                                                       | Non disponible. **Windows Server 2008 et Windows Vista :** Définit un événement pour un opcode spécifique à une tâche. non disponible à partir de Windows 7.<br/> <br/>                                                                                                                  |
| [**événement (TaskEventDefinitionType)**](eventmanifestschema-event-taskeventdefinitiontype-element.md)                     | Non disponible. **Windows Server 2008 et Windows Vista :** Définit un événement pour un événement spécifique à une tâche. non disponible à partir de Windows 7.<br/> <br/>                                                                                                                   |
| [**événements (InstrumentationType)**](eventmanifestschema-events-instrumentationtype-element.md)                           | Contient une liste de fournisseurs définis dans le manifeste.<br/>                                                                                                                                                                                                            |
| [**événements (ProviderType)**](eventmanifestschema-events-providertype-element.md)                                         | Contient une liste des événements définis par le fournisseur.<br/>                                                                                                                                                                                                                  |
| [**importChannel (ChannelListType)**](eventmanifestschema-importchannel-channellisttype-element.md)                     | Identifie un canal qui a été défini par un autre fournisseur.<br/>                                                                                                                                                                                                           |
| [**instrumentationManifest**](eventmanifestschema-instrumentationmanifest-element.md)                                   | Nœud racine du manifeste.<br/>                                                                                                                                                                                                                                            |
| [**Instrumentation (instrumentationManifest)**](eventmanifestschema-instrumentation-instrumentationmanifest-element.md) | Définit les types d’instrumentation que le manifeste contient (par exemple, l’instrumentation d’événement ou de compteur).<br/>                                                                                                                                                          |
| [**InType (InputTypeListType)**](eventmanifestschema-intype-inputtypelisttype-element.md)                               | Définit un type de données d’entrée.<br/>                                                                                                                                                                                                                                               |
| [**intypés (TypeListType)**](eventmanifestschema-intypes-typelisttype-element.md)                                       | Contient une liste de définitions de type d’entrée.<br/>                                                                                                                                                                                                                                |
| [**mot clé (KeywordListType)**](eventmanifestschema-keyword-keywordlisttype-element.md)                                 | Définit un mot clé qui identifie une catégorie d’événements.<br/>                                                                                                                                                                                                                   |
| [**Mots clés (ChannelPublishingType)**](eventmanifestschema-keywords-channelpublishingtype-element.md)                   | Masque de réécriture qui détermine la catégorie des événements écrits dans le canal.<br/>                                                                                                                                                                                         |
| [**Mots clés (type de données)**](eventmanifestschema-keywords-metadatatype-element.md)                                     | Définit une liste de mots clés qui classent les événements.<br/>                                                                                                                                                                                                                        |
| [**Mots clés (ProviderType)**](eventmanifestschema-keywords-providertype-element.md)                                     | Définit une liste de mots clés qui classent les événements.<br/>                                                                                                                                                                                                                        |
| [**latence (ChannelPublishingType)**](eventmanifestschema-latency-channelpublishingtype-element.md)                     | Délai d’attente avant le vidage des mémoires tampons, en millisecondes.<br/>                                                                                                                                                                                                            |
| [**niveau (ChannelPublishingType)**](eventmanifestschema-level-channelpublishingtype-element.md)                         | Niveau de gravité des événements à écrire dans le canal.<br/>                                                                                                                                                                                                                 |
| [**niveau (LevelListType)**](eventmanifestschema-level-levellisttype-element.md)                                         | Définit une valeur de gravité qui détermine le niveau de détail des événements lors de la journalisation.<br/>                                                                                                                                                                                          |
| [**niveaux (type de données)**](eventmanifestschema-levels-metadatatype-element.md)                                         | Définit une liste de niveaux qui spécifient la gravité d’un événement.<br/>                                                                                                                                                                                                           |
| [**niveaux (ProviderType)**](eventmanifestschema-levels-providertype-element.md)                                         | Définit une liste de niveaux qui spécifient la gravité d’un événement.<br/>                                                                                                                                                                                                           |
| [**localisation (instrumentationManifest)**](eventmanifestschema-localization-instrumentationmanifest-element.md)       | Définit un groupe de ressources localisées que vous référencez dans votre manifeste.<br/>                                                                                                                                                                                               |
| [**journalisation (ChannelType)**](eventmanifestschema-logging-channeltype-element.md)                                         | Définit les propriétés du fichier journal qui stocke le canal, par exemple sa capacité et s’il est séquentiel ou circulaire.<br/>                                                                                                                                             |
| [**carte (BitMapType)**](eventmanifestschema-map-bitmaptype-element.md)                                                   | Définit le mappage entre une valeur de bit et une valeur de chaîne.<br/>                                                                                                                                                                                                               |
| [**carte (PatternMapType)**](eventmanifestschema-map-patternmaptype-element.md)                                           | Définit les expressions régulières utilisées pour rechercher une chaîne correspondante dans la chaîne de message et la modifier.<br/>                                                                                                                                                                        |
| [**carte (ValueMapType)**](eventmanifestschema-map-valuemaptype-element.md)                                               | Définit le mappage entre une valeur entière et une valeur de chaîne.<br/>                                                                                                                                                                                                          |
| [**mappages (ProviderType)**](eventmanifestschema-maps-providertype-element.md)                                             | Définit une liste de paires nom/valeur que vous pouvez référencer dans la section de modèle du manifeste.<br/>                                                                                                                                                                        |
| [**maxBuffers (ChannelPublishingType)**](eventmanifestschema-maxbuffers-channelpublishingtype-element.md)               | Nombre maximal de mémoires tampons à allouer pour la session.<br/>                                                                                                                                                                                                                |
| [**maxSize (ChannelLoggingType)**](eventmanifestschema-maxsize-channelloggingtype-element.md)                           | Taille maximale, en octets, du fichier journal.<br/>                                                                                                                                                                                                                              |
| [**message (table des messages)**](eventmanifestschema-message-messagetable-element.md)                                      | Contient une référence à une chaîne dans la section localisation du manifeste.<br/>                                                                                                                                                                                             |
| [**Élément messageTable (EventsType)**](eventmanifestschema-messagetable-instrumentationtype-element.md)                | Contient une référence à une chaîne dans la section localisation du manifeste.<br/>                                                                                                                                                                                             |
| [**métadonnées (instrumentationManifest)**](eventmanifestschema-metadata-instrumentationmanifest-element.md)               | Contient des valeurs globales que vous pouvez référencer dans d’autres manifestes.<br/>                                                                                                                                                                                                         |
| [**minBuffers (ChannelPublishingType)**](eventmanifestschema-minbuffers-channelpublishingtype-element.md)               | Nombre minimal de mémoires tampons à allouer pour la session.<br/>                                                                                                                                                                                                                |
| [**namedQueries (type de données)**](eventmanifestschema-namedqueries-metadatatype-element.md)                             | Non utilisé. Définit une liste de requêtes nommées qui interrogent la chaîne de message d’événement pour obtenir une valeur et effectuent une action spécifiée si elle est trouvée.<br/>                                                                                                                                        |
| [**namedQueries (ProviderType)**](eventmanifestschema-namedqueries-providertype-element.md)                             | Non utilisé. Définit une liste de requêtes nommées qui interrogent la chaîne de message d’événement pour obtenir une valeur et effectuent une action spécifiée si elle est trouvée.<br/>                                                                                                                                        |
| [**opcode (OpcodeListType)**](eventmanifestschema-opcode-opcodelisttype-element.md)                                     | Contient une valeur numérique qui identifie l’activité ou un point dans une activité que l’application a exécutée lorsqu’elle a déclenché l’événement (par exemple, initialisation ou fermeture).<br/>                                                                                |
| [**opcode (TaskEventDefinitionType)**](eventmanifestschema-opcode-taskeventdefinitiontype-element.md)                   | Non disponible. **Windows Server 2008 et Windows Vista :** Contient un opcode spécifique à une tâche pour un événement. Toutes les définitions opcode sont supposées être spécifiques à la tâche pour la tâche qui contient les définitions d’événement. non disponible à partir de Windows 7.<br/> <br/> |
| [**OpCodes (type de données)**](eventmanifestschema-opcodes-metadatatype-element.md)                                       | Définit une liste d’OpCodes que vous pouvez utiliser pour regrouper des événements au sein d’une tâche.<br/>                                                                                                                                                                                                 |
| [**OpCodes (ProviderType)**](eventmanifestschema-opcodes-providertype-element.md)                                       | Définit une liste d’OpCodes que vous pouvez utiliser pour regrouper des événements au sein d’une tâche.<br/>                                                                                                                                                                                                 |
| [**OpCodes (TaskType)**](eventmanifestschema-opcodes-tasktype-element.md)                                               | Définit une liste d’OpCodes spécifiques à une tâche.<br/>                                                                                                                                                                                                                                  |
| [**Type de InputType**](eventmanifestschema-outtype-inputtype-element.md)                                             | Liste des types de données de sortie qui déterminent la façon dont les données sont restituées.<br/>                                                                                                                                                                                                      |
| [**patternMap (PatternMapListType)**](eventmanifestschema-patternmap-patternmaplisttype-element.md)                     | Définit un mappage entre deux expressions régulières. Une expression est utilisée pour rechercher une chaîne correspondante dans la chaîne de message, tandis que l’autre est utilisée pour modifier la chaîne avant que le service la replace dans la chaîne de message.<br/>                                             |
| [**patternMaps (NamedQueryType)**](eventmanifestschema-patternmaps-namedquerytype-element.md)                           | Définit une liste de paires d’expressions régulières utilisées pour modifier la chaîne de message.<br/>                                                                                                                                                                                     |
| [**fournisseur (EventsType)**](eventmanifestschema-provider-eventstype-element.md)                                         | Définit un fournisseur et les métadonnées qu’il utilise pour définir ses événements.<br/>                                                                                                                                                                                                    |
| [**publication (ChannelType)**](eventmanifestschema-publishing-channeltype-element.md)                                   | Définit les propriétés de journalisation pour la session utilisée par le canal.<br/>                                                                                                                                                                                                     |
| [**ressources (LocalizationType)**](eventmanifestschema-resources-localizationtype-element.md)                           | Définit un groupe de tables de chaînes qui contiennent les chaînes localisées que vous référencez dans votre manifeste.<br/>                                                                                                                                                                  |
| [**rétention (ChannelLoggingType)**](eventmanifestschema-retention-channelloggingtype-element.md)                       | Détermine si le fichier journal est un fichier journal séquentiel ou circulaire. <br/>                                                                                                                                                                                                    |
| [**sidType (ChannelPublishingType)**](eventmanifestschema-sidtype-channelpublishingtype-element.md)                     | Détermine s’il faut inclure un identificateur de sécurité (SID) du principal avec chaque événement écrit dans le canal.<br/>                                                                                                                                                        |
| [**chaîne (StringTableType)**](eventmanifestschema-string-stringtabletype-element.md)                                   | Définit une chaîne localisée.<br/>                                                                                                                                                                                                                                               |
| [**stringTable (LocalizationType)**](eventmanifestschema-stringtable-localizationtype-element.md)                       | Définit une liste de chaînes localisées que vous pouvez référencer dans votre manifeste.<br/>                                                                                                                                                                                              |
| [**stringTable (ressources)**](eventmanifestschema-stringtable-localizationtype-element.md)                              | Définit une liste de chaînes localisées que vous pouvez référencer dans votre manifeste.<br/>                                                                                                                                                                                              |
| [**struct (TemplateItemType)**](eventmanifestschema-struct-templateitemtype-element.md)                                 | Définit une structure qui inclut un ou plusieurs éléments de données que vous souhaitez inclure avec l’événement.<br/>                                                                                                                                                                         |
| [**tâche (DefinitionType)**](eventmanifestschema-task-definitiontype-element.md)                                         | Non disponible. **Windows Server 2008 et Windows Vista :** Définit un événement spécifique à la tâche que votre fournisseur peut journaliser. non disponible à partir de Windows 7.<br/> <br/>                                                                                                     |
| [**tâche (TaskListType)**](eventmanifestschema-task-tasklisttype-element.md)                                             | Définit un composant ou un sous-composant d’une application.<br/>                                                                                                                                                                                                                    |
| [**tâches (ProviderType)**](eventmanifestschema-tasks-providertype-element.md)                                           | Définit une liste de tâches qu’un fournisseur peut utiliser pour regrouper des événements.<br/>                                                                                                                                                                                                          |
| [**modèle (TemplateListType)**](eventmanifestschema-template-templatelisttype-element.md)                             | Modèle qui définit les données à inclure avec un événement.<br/>                                                                                                                                                                                                                |
| [**modèles (ProviderType)**](eventmanifestschema-templates-providertype-element.md)                                   | Définit une liste de modèles qui spécifient les données à inclure avec les événements.<br/>                                                                                                                                                                                             |
| [**types (type de données)**](eventmanifestschema-types-metadatatype-element.md)                                           | Définit une liste de types XML.<br/>                                                                                                                                                                                                                                              |
| [**UserData (TemplateItemType)**](eventmanifestschema-userdata-templateitemtype-element.md)                             | Fragment XML utilisé pour restituer les données d’événement.<br/>                                                                                                                                                                                                                    |
| [**valueMap (MapType)**](eventmanifestschema-valuemap-maptype-element.md)                                               | Définit une liste de paires nom/valeur qui mappent des valeurs entières et des valeurs de chaîne.<br/>                                                                                                                                                                                             |
| [**xmlType (XmlTypeListType)**](eventmanifestschema-xmltype-xmltypelisttype-element.md)                                 | Définit un type XML.<br/>                                                                                                                                                                                                                                                      |
| [**xmlTypes (TypeListType)**](eventmanifestschema-xmltypes-typelisttype-element.md)                                     | Définit une liste de types de sortie que le service utilise pour déterminer comment restituer un type de données d’entrée.<br/>                                                                                                                                                                       |
