# <a name="levels-metadatatype-element"></a>Élément Levels (type de données)

Définit une liste de niveaux qui spécifient la gravité d’un événement.

``` syntax
<xs:element name="levels"
    type="LevelListType"
 />
```

L’élément **levels** est défini par le type complexe de l’élément de [**données**](eventmanifestschema-metadatatype-complextype.md) .
