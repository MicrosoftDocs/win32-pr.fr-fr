# <a name="levels-providertype-element"></a>Élément Levels (ProviderType)

Définit une liste de niveaux qui spécifient la gravité d’un événement.

``` syntax
<xs:element name="levels"
    type="LevelListType"
 />
```

L’élément **levels** est défini par le type complexe [**ProviderType**](eventmanifestschema-providertype-complextype.md) .
