# <a name="retention-channelloggingtype-element"></a>Élément retention (ChannelLoggingType)

Détermine si le fichier journal est un fichier journal séquentiel ou circulaire.

``` syntax
<xs:element name="retention"
    type="boolean"
 />
```

L’élément de **rétention** est défini par le type complexe [**ChannelLoggingType**](eventmanifestschema-channelloggingtype-complextype.md) .
