# <a name="fileline-debugdatatype-element"></a>Élément FileLine (DebugDataType)

Nom du fichier source et ligne dans le fichier source qui a consigné le message de trace.

``` syntax
<xs:element name="FileLine"
    type="string"
 />
```

L’élément **FileLine** est défini par le type complexe [**DebugDataType**](eventschema-debugdatatype-complextype.md) .
