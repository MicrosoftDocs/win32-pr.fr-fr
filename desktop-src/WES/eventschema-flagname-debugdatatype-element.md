# <a name="flagname-debugdatatype-element"></a>Élément FlagName (DebugDataType)

Valeur d’indicateur passée au fournisseur lorsqu’il a été activé.

``` syntax
<xs:element name="FlagName"
    type="string"
 />
```

L’élément **flagname** est défini par le type complexe [**DebugDataType**](eventschema-debugdatatype-complextype.md) .
