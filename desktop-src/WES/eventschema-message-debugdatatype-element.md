# <a name="message-debugdatatype-element"></a>Élément message (DebugDataType)

Chaîne du message. Le code XML contient cet élément si l’événement WPP a spécifié le champ FormattedString.

``` syntax
<xs:element name="Message"
    type="string"
 />
```

L’élément **message** est défini par le type complexe [**DebugDataType**](eventschema-debugdatatype-complextype.md) .
