# <a name="userdata-templateitemtype-element"></a>Élément UserData (TemplateItemType)

Fragment XML utilisé pour restituer les données d’événement.

``` syntax
<xs:element name="UserData"
    type="XmlType"
 />
```

L’élément **UserData** est défini par le type complexe [**TemplateItemType**](eventmanifestschema-templateitemtype-complextype.md) .
