# <a name="data-structdefinitiontype-element"></a>Élément Data (StructDefinitionType)

Définit un élément de données que vous souhaitez inclure dans la structure.

``` syntax
<xs:element name="data"
    type="DataDefinitionType"
 />
```

L’élément de **données** est défini par le type complexe [**StructDefinitionType**](eventmanifestschema-structdefinitiontype-complextype.md) .
