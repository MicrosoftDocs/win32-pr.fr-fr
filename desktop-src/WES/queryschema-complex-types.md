# <a name="query-schema-complex-types"></a>Types complexes de schéma de requête

Voici les types complexes définis par le schéma de requête.



| Type complexe                                                   | Description                                                                                                                                                         |
|----------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**QueryListType**](queryschema-querylisttype-complextype.md) | Définit une liste de requêtes qui peuvent contenir un ensemble de sélecteurs et de suppressions utilisés pour inclure des événements dans ou exclure des événements du jeu de résultats. <br/> |
| [**Affectée**](queryschema-querytype-complextype.md)         | Définit un ensemble de requêtes de sélecteur et de suppresseur utilisées pour inclure des événements dans ou exclure des événements du jeu de résultats.<br/>                               |
