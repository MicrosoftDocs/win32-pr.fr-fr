# <a name="templates-providertype-element"></a>Élément templates (ProviderType)

Définit une liste de modèles qui spécifient les données à inclure avec les événements.

``` syntax
<xs:element name="templates"
    type="TemplateListType"
 />
```

L’élément **Templates** est défini par le type complexe [**ProviderType**](eventmanifestschema-providertype-complextype.md) .
