# <a name="keywords-providertype-element"></a>Élément Keywords (ProviderType)

Définit une liste de mots clés qui classent les événements.

``` syntax
<xs:element name="keywords"
    type="KeywordListType"
 />
```

L’élément **Keywords** est défini par le type complexe [**ProviderType**](eventmanifestschema-providertype-complextype.md) .
