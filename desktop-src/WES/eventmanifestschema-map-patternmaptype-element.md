# <a name="map-patternmaptype-element"></a>Élément MAP (PatternMapType)

Définit les expressions régulières utilisées pour rechercher une chaîne correspondante dans la chaîne de message et la modifier.

``` syntax
<xs:element name="map"
    type="PatternMapValueType"
 />
```

L’élément **cartographique** est défini par le type complexe [**PatternMapType**](eventmanifestschema-patternmaptype-complextype.md) .
