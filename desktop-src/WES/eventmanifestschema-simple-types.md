# <a name="eventmanifest-schema-simple-types"></a>Types simples de schéma EventManifest

Voici les types simples définis par le schéma EventManifest.



| Type simple                                                       | Description                                                                                                                                |
|-------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------|
| [**CountType**](eventmanifestschema-counttype-simpletype.md)     | Définit un type de nombre qui est utilisé pour spécifier le nombre d’éléments dans un tableau.<br/>                                                |
| [**CSymbolType**](eventmanifestschema-csymboltype-simpletype.md) | Définit un nom de symbole C/C++ valide.<br/>                                                                                              |
| [**cheminfichier**](eventmanifestschema-filepath-simpletype.md)       | Définit une chaîne qui contient un chemin d’accès qualifié complet à un fichier.<br/>                                                                |
| [**GUIDType**](eventmanifestschema-guidtype-simpletype.md)       | Définit un type d’identificateur global unique, au format de registre.<br/>                                                                  |
| [**HexInt8Type**](eventmanifestschema-hex8type-simpletype.md)    | Définit un type hexadécimal sur 1 octet.<br/>                                                                                              |
| [**HexInt16Type**](eventmanifestschema-hex16type-simpletype.md)  | Définit un type hexadécimal sur 2 octets.<br/>                                                                                              |
| [**HexInt32Type**](eventmanifestschema-hex32type-simpletype.md)  | Définit un type hexadécimal sur 4 octets.<br/>                                                                                              |
| [**HexInt64Type**](eventmanifestschema-hex64type-simpletype.md)  | Définit un type hexadécimal sur 8 octets.<br/>                                                                                             |
| [**UInt8Type**](eventmanifestschema-hexint8type-simpletype.md)   | Définit un type d’octet non signé.<br/>                                                                                                  |
| [**UInt16Type**](eventmanifestschema-hexint16type-simpletype.md) | Définit un type short non signé.<br/>                                                                                                 |
| [**UInt32Type**](eventmanifestschema-hexint32type-simpletype.md) | Définit un type d’entier non signé.<br/>                                                                                               |
| [**UInt64Type**](eventmanifestschema-hexint64type-simpletype.md) | Définit un type long non signé.<br/>                                                                                                  |
| [**LengthType**](eventmanifestschema-lengthtype-simpletype.md)   | Définit un type de longueur utilisé pour spécifier le nombre d’octets ou de caractères dans un élément de données de longueur variable.<br/>                 |
| [**QNameList**](eventmanifestschema-qnamelist-simpletype.md)     | Définit une liste de noms qualifiés.<br/>                                                                                              |
| [**strTableRef**](eventmanifestschema-strtableref-simpletype.md) | Définit une chaîne qui fait référence à une chaîne de message définie dans une table de chaînes du manifeste ou dans un fichier de message (. MC).<br/> |
