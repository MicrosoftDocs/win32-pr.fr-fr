# <a name="channels-metadatatype-element"></a>Élément Channels (indatatype)

Définit une liste de canaux auxquels les fournisseurs peuvent enregistrer des événements.

``` syntax
<xs:element name="channels"
    type="ChannelListType"
 />
```

L’élément **Channels** est défini par le type complexe de l’élément de [**données**](eventmanifestschema-metadatatype-complextype.md) .
