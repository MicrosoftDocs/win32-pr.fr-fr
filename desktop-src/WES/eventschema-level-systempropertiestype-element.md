# <a name="level-systempropertiestype-element"></a>Élément Level (SystemPropertiesType)

Contient le niveau de gravité de l’événement.

``` syntax
<xs:element name="Level"
    type="unsignedByte"
 />
```

L’élément **Level** est défini par le type complexe [**SystemPropertiesType**](eventschema-systempropertiestype-complextype.md) .
