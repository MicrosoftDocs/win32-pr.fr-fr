# <a name="struct-templateitemtype-element"></a>Élément struct (TemplateItemType)

Définit une structure qui inclut un ou plusieurs éléments de données que vous souhaitez inclure avec l’événement.

``` syntax
<xs:element name="struct"
    type="StructDefinitionType"
 />
```

L’élément **struct** est défini par le type complexe [**TemplateItemType**](eventmanifestschema-templateitemtype-complextype.md) .
