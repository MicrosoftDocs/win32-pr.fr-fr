# <a name="debugdata-eventtype-element"></a>Élément DebugData (EventType)

Contient les données qui peuvent être journalisées pour les événements du préprocesseur de trace logiciel Windows (WPP).

``` syntax
<xs:element name="DebugData"
    type="DebugDataType"
 />
```

L’élément **DebugData** est défini par le type complexe [**eventType**](eventschema-eventtype-complextype.md) .
