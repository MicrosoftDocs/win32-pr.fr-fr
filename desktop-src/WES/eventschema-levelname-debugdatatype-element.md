# <a name="levelname-debugdatatype-element"></a>Élément LevelName (DebugDataType)

Valeur de niveau passée au fournisseur lorsqu’il a été activé.

``` syntax
<xs:element name="LevelName"
    type="string"
 />
```

L’élément **LevelName** est défini par le type complexe [**DebugDataType**](eventschema-debugdatatype-complextype.md) .

|-------------------------------------|------------------------------------------------------|
| Client minimal pris en charge<br/> | Windows \[Applications de bureau Vista uniquement\]<br/>       |
| Serveur minimal pris en charge<br/> | Windows Serveur 2008 \[ applications de bureau uniquement\]<br/> |
