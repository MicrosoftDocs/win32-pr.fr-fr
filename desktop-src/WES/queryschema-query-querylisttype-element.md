# <a name="query-querylisttype-element"></a>Élément Query (QueryListType)

Définit un ensemble de sélecteurs et de suppressions utilisés pour inclure des événements dans ou exclure des événements du jeu de résultats.

``` syntax
<xs:element name="Query"
    type="QueryType"
 />
```

L’élément **query** est défini par le type complexe [**QueryListType**](queryschema-querylisttype-complextype.md) .
