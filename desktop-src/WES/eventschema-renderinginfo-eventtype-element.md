# <a name="renderinginfo-eventtype-element"></a>Élément RenderingInfo (EventType)

Contient les chaînes de message rendues pour l’événement (inclut la chaîne de message de l’événement et les chaînes de message pour l’une des propriétés de l’événement, telles que le niveau, la tâche et l’opcode).

``` syntax
<xs:element name="RenderingInfo"
    type="RenderingType"
 />
```

L’élément **RenderingInfo** est défini par le type complexe [**eventType**](eventschema-eventtype-complextype.md) .
