# <a name="logging-channeltype-element"></a>Élément Logging (ChannelType)

Définit les propriétés du fichier journal qui stocke le canal, par exemple sa capacité et s’il est séquentiel ou circulaire.

``` syntax
<xs:element name="logging"
    type="ChannelLoggingType"
 />
```

L’élément **Logging** est défini par le type complexe [**ChannelType**](eventmanifestschema-channeltype-complextype.md) .
