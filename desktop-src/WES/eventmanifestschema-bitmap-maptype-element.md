# <a name="bitmap-maptype-element"></a>Élément bitMap (MapType)

Définit une liste de paires nom/valeur qui mappent les valeurs de bit et les valeurs de chaîne.

``` syntax
<xs:element name="bitMap"
    type="BitMapType"
 />
```

L’élément **bitMap** est défini par le type complexe [**maptype**](eventmanifestschema-maptype-complextype.md) .
