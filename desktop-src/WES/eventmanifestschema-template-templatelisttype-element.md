# <a name="template-templatelisttype-element"></a>Élément Template (TemplateListType)

Modèle qui définit les données à inclure avec un événement.

``` syntax
<xs:element name="template"
    type="TemplateItemType"
 />
```

L’élément de **modèle** est défini par le type complexe [**TemplateListType**](eventmanifestschema-templatelisttype-complextype.md) .
