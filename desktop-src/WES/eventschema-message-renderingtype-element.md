# <a name="message-renderinginfotype-element"></a>Élément message (RenderingInfoType)

Contient le message d’événement qui est rendu pour l’événement.

``` syntax
<xs:element name="Message"
    type="string"
 />
```

L’élément **message** est défini par le type complexe [**RenderingInfoType**](eventschema-renderingtype-complextype.md) .
