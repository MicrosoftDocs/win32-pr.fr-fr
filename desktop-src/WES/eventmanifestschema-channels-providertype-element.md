# <a name="channels-providertype-element"></a>Élément Channels (ProviderType)

Définit une liste de canaux auxquels les fournisseurs peuvent enregistrer des événements.

``` syntax
<xs:element name="channels"
    type="ChannelListType"
 />
```

L’élément **Channels** est défini par le type complexe [**ProviderType**](eventmanifestschema-providertype-complextype.md) .
