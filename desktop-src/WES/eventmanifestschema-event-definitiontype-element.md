# <a name="event-definitiontype-element"></a>Élément Event (DefinitionType)

Définit un événement que votre fournisseur peut journaliser.

``` syntax
<xs:element name="event"
    type="EventDefinitionType"
 />
```

L’élément **Event** est défini par le type complexe [**DefinitionType**](eventmanifestschema-definitiontype-complextype.md) .
