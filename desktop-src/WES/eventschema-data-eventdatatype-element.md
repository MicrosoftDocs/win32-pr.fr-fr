# <a name="data-eventdatatype-element"></a>Élément Data (EventDataType)

Élément de données de niveau supérieur défini dans le modèle pour l’événement.

``` syntax
<xs:element name="Data"
    type="DataFieldType"
 />
```

L’élément de **données** est défini par le type complexe [**EventDataType**](eventschema-eventdatatype-complextype.md) .
