# <a name="map-bitmaptype-element"></a>Élément MAP (BitMapType)

Définit le mappage entre une valeur de bit et une valeur de chaîne.

``` syntax
<xs:element name="map"
    type="BitMapValueType"
 />
```

L’élément **cartographique** est défini par le type complexe [**BitMapType**](eventmanifestschema-bitmaptype-complextype.md) .
