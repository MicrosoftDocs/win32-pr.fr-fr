# <a name="keywords-channelpublishingtype-element"></a>Élément Keywords (ChannelPublishingType)

Masque de réécriture qui détermine la catégorie des événements écrits dans le canal.

``` syntax
<xs:element name="keywords"
    type="UInt64Type"
 />
```

L’élément **Keywords** est défini par le type complexe [**ChannelPublishingType**](eventmanifestschema-channelpublishingtype-complextype.md) .
