# <a name="subcomponent-debugdatatype-element"></a>Élément SubComponent (DebugDataType)

Nom du sous-composant qui a consigné le message de trace.

``` syntax
<xs:element name="SubComponent"
    type="string"
 />
```

L’élément de sous- **composant** est défini par le type complexe [**DebugDataType**](eventschema-debugdatatype-complextype.md) .
