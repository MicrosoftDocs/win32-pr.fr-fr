# <a name="namedqueries-providertype-element"></a>Élément namedQueries (ProviderType)

Non utilisé. Définit une liste de requêtes nommées qui interrogent la chaîne de message d’événement pour obtenir une valeur et effectuent une action spécifiée si elle est trouvée.

``` syntax
<xs:element name="namedQueries"
    type="NamedQueryType"
 />
```

L’élément **namedQueries** est défini par le type complexe [**ProviderType**](eventmanifestschema-providertype-complextype.md) .
