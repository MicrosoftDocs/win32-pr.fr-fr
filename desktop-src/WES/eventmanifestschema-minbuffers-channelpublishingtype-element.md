# <a name="minbuffers-channelpublishingtype-element"></a>Élément minBuffers (ChannelPublishingType)

Nombre minimal de mémoires tampons à allouer pour la session.

``` syntax
<xs:element name="minBuffers"
    type="unsignedInt"
 />
```

L’élément **minBuffers** est défini par le type complexe [**ChannelPublishingType**](eventmanifestschema-channelpublishingtype-complextype.md) .
