# <a name="channel-channellisttype-element"></a>Élément Channel (ChannelListType)

Définit un canal dans lequel les événements peuvent être enregistrés.

``` syntax
<xs:element name="channel"
    type="ChannelType"
 />
```

L’élément de **canal** est défini par le type complexe [**ChannelListType**](eventmanifestschema-channellisttype-complextype.md) .
