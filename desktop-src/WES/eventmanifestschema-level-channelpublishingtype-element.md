# <a name="level-channelpublishingtype-element"></a>Élément Level (ChannelPublishingType)

Niveau de gravité des événements à écrire dans le canal.

``` syntax
<xs:element name="level"
    type="UInt8Type"
 />
```

L’élément **Level** est défini par le type complexe [**ChannelPublishingType**](eventmanifestschema-channelpublishingtype-complextype.md) .
