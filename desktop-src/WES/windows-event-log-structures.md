# <a name="windows-event-log-structures"></a>Windows Structures du journal des événements

Windows Le journal des événements définit les structures suivantes.



| Structure                                | Description                                                    |
|------------------------------------------|----------------------------------------------------------------|
| [**\_connexion RPC \_ evt**](/windows/desktop/api/WinEvt/ns-winevt-evt_rpc_login) | Contient les informations utilisées pour se connecter à un ordinateur distant. |
| [**la \_ variante evt**](/windows/desktop/api/WinEvt/ns-winevt-evt_variant)      | Contient des données d’événement ou des valeurs de propriété.                        |
