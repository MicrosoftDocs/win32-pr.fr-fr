# <a name="events-providertype-element"></a>Élément Events (ProviderType)

Contient une liste des événements définis par le fournisseur.

``` syntax
<xs:element name="events"
    type="DefinitionType"
 />
```

L’élément **Events** est défini par le type complexe [**ProviderType**](eventmanifestschema-providertype-complextype.md) .
