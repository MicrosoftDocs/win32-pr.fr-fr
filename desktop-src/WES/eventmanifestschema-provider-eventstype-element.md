# <a name="provider-eventstype-element"></a>Élément Provider (EventsType)

Définit un fournisseur et les métadonnées qu’il utilise pour définir ses événements.

``` syntax
<xs:element name="provider"
    type="ProviderType"
 />
```

L’élément **Provider** est défini par le type complexe [**EventsType**](eventmanifestschema-eventstype-complextype.md) .
