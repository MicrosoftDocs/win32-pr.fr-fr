# <a name="intypes-typelisttype-element"></a>Élément intypés (TypeListType)

Contient une liste de définitions de type d’entrée.

``` syntax
<xs:element name="inTypes"
    type="InputTypeListType"
 />
```

L’élément **Intypes** est défini par le type complexe [**TypeListType**](eventmanifestschema-typelisttype-complextype.md) .
