# <a name="version-systempropertiestype-element"></a>Élément version (SystemPropertiesType)

Contient le numéro de version de la définition de l’événement.

``` syntax
<xs:element name="Version"
    type="unsignedByte"
 />
```

L’élément **version** est défini par le type complexe [**SystemPropertiesType**](eventschema-systempropertiestype-complextype.md) .
