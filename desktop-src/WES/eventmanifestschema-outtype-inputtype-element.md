# <a name="outtype-inputtype-element"></a>Élément de type InputType

Liste des types de données de sortie qui déterminent la façon dont les données sont restituées.

``` syntax
<xs:element name="outType"
    type="OutputType"
 />
```

L’élément de **type** out est défini par le type complexe [**InputType**](eventmanifestschema-inputtype-complextype.md) .
