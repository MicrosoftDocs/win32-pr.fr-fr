# <a name="opcodes-tasktype-element"></a>OpCodes (TaskType) (élément)

Définit une liste d’OpCodes spécifiques à une tâche.

``` syntax
<xs:element name="opcodes"
    type="OpcodeListType"
 />
```

L’élément **OpCodes** est défini par le type complexe [**TaskType**](eventmanifestschema-tasktype-complextype.md) .
