# <a name="processingerrordata-eventtype-element"></a>Élément ProcessingErrorData (EventType)

Contient les détails de l’erreur qui s’est produite lors de la tentative de rendu de l’événement.

``` syntax
<xs:element name="ProcessingErrorData"
    type="ProcessingErrorDataType"
 />
```

L’élément **ProcessingErrorData** est défini par le type complexe [**eventType**](eventschema-eventtype-complextype.md) .
