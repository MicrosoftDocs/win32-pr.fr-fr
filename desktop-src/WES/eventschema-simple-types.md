# <a name="event-schema-simple-types"></a>Types simples de schéma d’événement

Voici les types simples définis par le schéma d’événement.



| Type simple                                                 | Description                                                               |
|-------------------------------------------------------------|---------------------------------------------------------------------------|
| [**GUIDType**](eventschema-guidtype-simpletype.md)         | Définit un type d’identificateur global unique, au format de registre.<br/> |
| [**HexInt32Type**](eventschema-hexint32type-simpletype.md) | Définit un type hexadécimal sur 1 octet.<br/>                             |
| [**HexInt64Type**](eventschema-hexint64type-simpletype.md) | Définit un type hexadécimal sur 8 octets.<br/>                            |
