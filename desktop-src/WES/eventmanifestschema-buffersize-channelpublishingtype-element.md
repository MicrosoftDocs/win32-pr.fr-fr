# <a name="buffersize-channelpublishingtype-element"></a>bufferSize (ChannelPublishingType), élément

Contient la quantité de mémoire, en kilo-octets, allouée pour les mémoires tampons de session.

``` syntax
<xs:element name="bufferSize"
    type="unsignedInt"
 />
```

L’élément **bufferSize** est défini par le type complexe [**ChannelPublishingType**](eventmanifestschema-channelpublishingtype-complextype.md) .
