# <a name="events-instrumentationtype-element"></a>Élément Events (InstrumentationType)

Contient une liste de fournisseurs définis dans le manifeste.

``` syntax
<xs:element name="events"
    type="EventsType"
 />
```

L’élément **Events** est défini par le type complexe [**InstrumentationType**](eventmanifestschema-instrumentationtype-complextype.md) .
