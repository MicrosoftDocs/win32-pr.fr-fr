Les constantes suivantes peuvent être utilisées dans la fonction [**WinBioAsyncMonitorFrameworkChanges**](/windows/desktop/api/Winbio/nf-winbio-winbioasyncmonitorframeworkchanges) pour identifier le type de modification qui s’est produit dans le Framework.
