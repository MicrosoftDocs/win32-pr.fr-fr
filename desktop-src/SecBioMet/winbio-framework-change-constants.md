# <a name="winbio_framework_change-constants"></a>\_Constantes de modification de l’infrastructure WINBIO \_

Les constantes suivantes peuvent être utilisées dans la fonction [**WinBioAsyncMonitorFrameworkChanges**](/windows/desktop/api/Winbio/nf-winbio-winbioasyncmonitorframeworkchanges) pour identifier le type de modification qui s’est produit dans le Framework.

<dl> <dt>

<span id="WINBIO_FRAMEWORK_CHANGE_UNIT"></span><span id="winbio_framework_change_unit"></span>**unité de modification de l' \_ infrastructure WINBIO \_ \_**
</dt> <dd> <dl> <dt>

0x00000001
</dt> <dt>



Une unité biométrique a été attachée ou détachée de l’ordinateur.


</dt> </dl> </dd> </dl>
