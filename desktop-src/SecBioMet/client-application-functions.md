# <a name="client-application-functions"></a>Fonctions de l’application cliente

Les fonctions suivantes sont prises en charge pour le développement d’applications clientes par l’API Windows Biometric Framework.

## <a name="in-this-section"></a>Contenu de cette section



| Rubrique                                                                                       | Description                                                                                                                                                                                                                              |
|---------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**WinBioAcquireFocus**](/windows/desktop/api/Winbio/nf-winbio-winbioacquirefocus)<br/>                                 | Acquiert le focus de la fenêtre.<br/>                                                                                                                                                                                                        |
| [**WinBioAsyncEnumBiometricUnits**](/windows/desktop/api/Winbio/nf-winbio-winbioasyncenumbiometricunits)<br/>           | Énumère de manière asynchrone toutes les unités biométriques attachées qui correspondent au type de facteur d’entrée.<br/>                                                                                                                                      |
| [**WinBioAsyncEnumDatabases**](/windows/desktop/api/Winbio/nf-winbio-winbioasyncenumdatabases)<br/>                     | Énumère de manière asynchrone toutes les bases de données inscrites qui correspondent à un type spécifié.<br/>                                                                                                                                               |
| [**WinBioAsyncEnumServiceProviders**](/windows/desktop/api/Winbio/nf-winbio-winbioasyncenumserviceproviders)<br/>       | Retourne de façon asynchrone des informations sur les fournisseurs de services biométriques installés. À compter de Windows 10, Build 1607, cette fonction peut être utilisée avec une image mobile.<br/>                                                  |
| [**WinBioAsyncMonitorFrameworkChanges**](/windows/desktop/api/Winbio/nf-winbio-winbioasyncmonitorframeworkchanges)<br/> | Démarre un moniteur asynchrone des modifications apportées à l’infrastructure biométrique.<br/>                                                                                                                                                         |
| [**WinBioAsyncOpenFramework**](/windows/desktop/api/Winbio/nf-winbio-winbioasyncopenframework)<br/>                     | Ouvre un handle vers l’infrastructure biométrique. À compter de Windows 10, Build 1607, cette fonction peut être utilisée avec une image mobile.<br/>                                                                                       |
| [**WinBioAsyncOpenSession**](/windows/desktop/api/Winbio/nf-winbio-winbioasyncopensession)<br/>                         | Se connecte de manière asynchrone à un fournisseur de services biométriques et à une ou plusieurs unités biométriques. À compter de Windows 10, Build 1607, cette fonction peut être utilisée avec une image mobile.<br/>                                         |
| [**WinBioCancel**](/windows/desktop/api/Winbio/nf-winbio-winbiocancel)<br/>                                             | Annule toutes les opérations biométriques en attente pour une session spécifiée. À compter de Windows 10, Build 1607, cette fonction peut être utilisée avec une image mobile.<br/>                                                                |
| [**WinBioCaptureSample**](/windows/desktop/api/Winbio/nf-winbio-winbiocapturesample)<br/>                               | Capture un échantillon biométrique et remplit un enregistrement d’informations biométriques (BIR) avec les données brutes ou traitées.<br/>                                                                                                                    |
| [**WinBioCaptureSampleWithCallback**](/windows/desktop/api/Winbio/nf-winbio-winbiocapturesamplewithcallback)<br/>       | Capture un échantillon biométrique de manière asynchrone et retourne les données brutes ou traitées dans un enregistrement d’informations biométriques (BIR).<br/>                                                                                                     |
| [**WinBioCloseFramework**](/windows/desktop/api/Winbio/nf-winbio-winbiocloseframework)<br/>                             | Ferme un handle de Framework ouvert précédemment avec [**WinBioAsyncOpenFramework**](/windows/desktop/api/Winbio/nf-winbio-winbioasyncopenframework). À compter de Windows 10, Build 1607, cette fonction peut être utilisée avec une image mobile.<br/>                   |
| [**WinBioCloseSession**](/windows/desktop/api/Winbio/nf-winbio-winbioclosesession)<br/>                                 | Ferme une session biométrique et libère les ressources associées. À compter de Windows 10, Build 1607, cette fonction peut être utilisée avec une image mobile.<br/>                                                                    |
| [**WinBioControlUnit**](/windows/desktop/api/Winbio/nf-winbio-winbiocontrolunit)<br/>                                   | Permet à l’appelant d’effectuer des opérations de contrôle définies par le fournisseur sur une unité biométrique. À compter de Windows 10, Build 1607, cette fonction peut être utilisée avec une image mobile.<br/>                                              |
| [**WinBioControlUnitPrivileged**](/windows/desktop/api/Winbio/nf-winbio-winbiocontrolunitprivileged)<br/>               | Permet à l’appelant d’effectuer des opérations de contrôle privilégiées définies par le fournisseur sur une unité biométrique. À compter de Windows 10, Build 1607, cette fonction peut être utilisée avec une image mobile.<br/>                                   |
| [**WinBioDeleteTemplate**](/windows/desktop/api/Winbio/nf-winbio-winbiodeletetemplate)<br/>                             | Supprime un modèle biométrique du magasin de modèles. À compter de Windows 10, Build 1607, cette fonction peut être utilisée avec une image mobile.<br/>                                                                            |
| [**WinBioEnrollBegin**](/windows/desktop/api/Winbio/nf-winbio-winbioenrollbegin)<br/>                                   | Lance une séquence d’inscription biométrique et crée un modèle biométrique vide. À compter de Windows 10, Build 1607, cette fonction peut être utilisée avec une image mobile.<br/>                                               |
| [**WinBioEnrollCapture**](/windows/desktop/api/Winbio/nf-winbio-winbioenrollcapture)<br/>                               | Capture un échantillon biométrique et l’ajoute à un modèle. À compter de Windows 10, Build 1607, cette fonction peut être utilisée avec une image mobile.<br/>                                                                           |
| [**WinBioEnrollCaptureWithCallback**](/windows/desktop/api/Winbio/nf-winbio-winbioenrollcapturewithcallback)<br/>       | Capture de manière asynchrone un échantillon biométrique et l’ajoute à un modèle.<br/>                                                                                                                                                         |
| [**WinBioEnrollCommit**](/windows/desktop/api/Winbio/nf-winbio-winbioenrollcommit)<br/>                                 | Finalise un modèle biométrique en attente et l’enregistre dans la base de données associée à l’unité biométrique utilisée pour l’inscription. À compter de Windows 10, Build 1607, cette fonction peut être utilisée avec une image mobile.<br/>      |
| [**WinBioEnrollDiscard**](/windows/desktop/api/Winbio/nf-winbio-winbioenrolldiscard)<br/>                               | Met fin à la séquence d’inscription et ignore un modèle biométrique en attente. À compter de Windows 10, Build 1607, cette fonction peut être utilisée avec une image mobile.<br/>                                                          |
| [**WinBioEnrollSelect**](/windows/desktop/api/winbio/nf-winbio-winbioenrollselect)<br/>                                 | Spécifie l’individu que vous souhaitez inscrire lorsque des données représentant plusieurs individus sont présentes dans l’exemple de mémoire tampon. À compter de Windows 10, Build 1607, cette fonction peut être utilisée avec une image mobile.<br/> |
| [**WinBioEnumBiometricUnits**](/windows/desktop/api/Winbio/nf-winbio-winbioenumbiometricunits)<br/>                     | Énumère toutes les unités biométriques attachées qui correspondent au type d’entrée.<br/>                                                                                                                                                            |
| [**WinBioEnumDatabases**](/windows/desktop/api/Winbio/nf-winbio-winbioenumdatabases)<br/>                               | Énumère toutes les bases de données inscrites qui correspondent à un type spécifié.<br/>                                                                                                                                                              |
| [**WinBioEnumEnrollments**](/windows/desktop/api/Winbio/nf-winbio-winbioenumenrollments)<br/>                           | Récupère les sous-facteurs biométriques inscrits pour une identité et une unité biométriques spécifiées. À compter de Windows 10, Build 1607, cette fonction peut être utilisée avec une image mobile.<br/>                                        |
| [**WinBioEnumServiceProviders**](/windows/desktop/api/Winbio/nf-winbio-winbioenumserviceproviders)<br/>                 | Récupère des informations sur les fournisseurs de services biométriques installés. À compter de Windows 10, Build 1607, cette fonction peut être utilisée avec une image mobile.<br/>                                                               |
| [**WinBioFree**](/windows/desktop/api/Winbio/nf-winbio-winbiofree)<br/>                                                 | Libère la mémoire allouée pour l’application cliente par un appel antérieur à une fonction API Windows Biometric Framework. À compter de Windows 10, Build 1607, cette fonction peut être utilisée avec une image mobile.<br/>           |
| [**WinBioGetCredentialState**](/windows/desktop/api/Winbio/nf-winbio-winbiogetcredentialstate)<br/>                     | Récupère une valeur qui spécifie si les informations d’identification ont été définies pour l’utilisateur spécifié. À compter de Windows 10, Build 1607, cette fonction peut être utilisée avec une image mobile.<br/>                                       |
| [**WinBioGetDomainLogonSetting**](/windows/desktop/api/Winbio/nf-winbio-winbiogetdomainlogonsetting)<br/>               | Récupère une valeur qui spécifie si les utilisateurs peuvent se connecter à un domaine à l’aide des informations biométriques.<br/>                                                                                                                         |
| [**WinBioGetEnabledSetting**](/windows/desktop/api/Winbio/nf-winbio-winbiogetenabledsetting)<br/>                       | Récupère une valeur qui spécifie si la Windows Biometric Framework est actuellement activée.<br/>                                                                                                                                |
| [**WinBioGetEnrolledFactors**](/windows/desktop/api/winbio/nf-winbio-winbiogetenrolledfactors)<br/>                     | Obtient des informations sur les inscriptions biométriques que l’utilisateur spécifié a sur l’ordinateur.<br/>                                                                                                                                 |
| [**WinBioGetLogonSetting**](/windows/desktop/api/Winbio/nf-winbio-winbiogetlogonsetting)<br/>                           | Récupère une valeur qui indique si les utilisateurs peuvent se connecter à l’aide des informations biométriques.<br/>                                                                                                                                     |
| [**WinBioGetProperty**](/windows/desktop/api/Winbio/nf-winbio-winbiogetproperty)<br/>                                   | Récupère une propriété de session, d’unité ou de modèle. À compter de Windows 10, Build 1607, cette fonction peut être utilisée avec une image mobile.<br/>                                                                                 |
| [**WinBioIdentify**](/windows/desktop/api/Winbio/nf-winbio-winbioidentify)<br/>                                         | Capture un échantillon biométrique et détermine s’il correspond à un modèle biométrique existant. À compter de Windows 10, Build 1607, cette fonction peut être utilisée avec une image mobile.<br/>                                    |
| [**WinBioIdentifyWithCallback**](/windows/desktop/api/Winbio/nf-winbio-winbioidentifywithcallback)<br/>                 | Capture de manière asynchrone un échantillon biométrique et détermine s’il correspond à un modèle biométrique existant.<br/>                                                                                                                  |
| [**WinBioLocateSensor**](/windows/desktop/api/Winbio/nf-winbio-winbiolocatesensor)<br/>                                 | Récupère le numéro d’identification d’une unité biométrique sélectionnée de manière interactive par un utilisateur.<br/>                                                                                                                                                 |
| [**WinBioLocateSensorWithCallback**](/windows/desktop/api/Winbio/nf-winbio-winbiolocatesensorwithcallback)<br/>         | Récupère de manière asynchrone le numéro d’identification de l’unité biométrique sélectionnée de manière interactive par un utilisateur.<br/>                                                                                                                                |

À compter de Windows 10, Build 1607, cette fonction peut être utilisée avec une image mobile.<br/>                                                                    |
| [**WinBioLogonIdentifiedUser**](/windows/desktop/api/Winbio/nf-winbio-winbiologonidentifieduser)<br/>                   | provoque un changement rapide d’utilisateur pour le compte associé à la dernière opération d’identification réussie effectuée par la session biométrique.<br/>                                                                                     |
| [**WinBioMonitorPresence**](/windows/desktop/api/winbio/nf-winbio-winbiomonitorpresence)<br/>                           | Active le mécanisme de reconnaissance faciale ou d’analyse d’iris pour l’unité biométrique spécifiée. | [**WinBioOpenSession**](/windows/desktop/api/Winbio/nf-winbio-winbioopensession)<br/>                                   | Établit une connexion à un fournisseur de services biométriques et une ou plusieurs unités biométriques.<br/>                                                                                                                                                     |

| [**WinBioReleaseFocus**](/windows/desktop/api/Winbio/nf-winbio-winbioreleasefocus)<br/>                                 | Libère le focus de la fenêtre.<br/>                                                                                                                                                                                                        |
| [**WinBioRemoveAllCredentials**](/windows/desktop/api/Winbio/nf-winbio-winbioremoveallcredentials)<br/>                 | Supprime toutes les informations d’identification du magasin. À compter de Windows 10, Build 1607, cette fonction peut être utilisée avec une image mobile.<br/>                                                                                          |
| [**WinBioRemoveAllDomainCredentials**](/windows/desktop/api/Winbio/nf-winbio-winbioremovealldomaincredentials)<br/>     | Supprime toutes les informations d’identification de l’utilisateur du domaine actuel du magasin. À compter de Windows 10, Build 1607, cette fonction peut être utilisée avec une image mobile.<br/>                                                              |
| [**WinBioRemoveCredential**](/windows/desktop/api/Winbio/nf-winbio-winbioremovecredential)<br/>                         | Supprime des informations d’identification de connexion biométriques pour un utilisateur spécifié. À compter de Windows 10, Build 1607, cette fonction peut être utilisée avec une image mobile.<br/>                                                                       |
