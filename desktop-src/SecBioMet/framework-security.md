# <a name="framework-security"></a>Sécurité de l’infrastructure

le Service biométrique Windows permet aux applications clientes sans privilèges d’effectuer des opérations biométriques sans leur fournir un accès direct à des exemples ou à du matériel biométriques. Le service crée une limite de sécurité entre les clients et les composants biométriques, comme indiqué dans l’illustration suivante.
