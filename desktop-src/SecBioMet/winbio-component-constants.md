# <a name="winbio_component-constants"></a>\_Constantes de composant WINBIO

Les constantes suivantes peuvent être utilisées lors de l’appel de [**WinBioControlUnit**](/windows/desktop/api/Winbio/nf-winbio-winbiocontrolunit) ou [**WinBioControlUnitPrivileged**](/windows/desktop/api/Winbio/nf-winbio-winbiocontrolunitprivileged) pour spécifier le type d’adaptateur utilisé.



| Constante                                                                                                                                                                                        | Description                             |
|:------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:----------------------------------------|
| <span id="WINBIO_COMPONENT_SENSOR"></span><span id="winbio_component_sensor"></span><dl> <dt>**\_capteur de composant WINBIO \_**</dt> </dl>    | Spécifie un adaptateur de capteur.<br/>  |
| <span id="WINBIO_COMPONENT_ENGINE"></span><span id="winbio_component_engine"></span><dl> <dt>**\_moteur de composant WINBIO \_**</dt> </dl>    | Spécifie un adaptateur de moteur.<br/> |
| <span id="WINBIO_COMPONENT_STORAGE"></span><span id="winbio_component_storage"></span><dl> <dt>**\_stockage du composant WINBIO \_**</dt> </dl> | Spécifie un adaptateur de stockage.<br/> |
