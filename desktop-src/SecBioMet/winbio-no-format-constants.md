# <a name="winbio_no_format-constants"></a>WINBIO \_ pas de \_ constantes de format

Si les membres d’une structure de [**\_ \_ format enregistrée par WINBIO**](winbio-registered-format.md) ne peuvent pas être nuls, vous pouvez utiliser les constantes suivantes pour rechercher les erreurs.



| Constante/valeur                                                                                                                                                                                                                                                                   | Description                                                                                          |
|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------|
| <span id="WINBIO_NO_FORMAT_OWNER_AVAILABLE"></span><span id="winbio_no_format_owner_available"></span><dl> <dt>**WINBIO \_ AUCUN \_ propriétaire de format \_ \_ disponible**</dt> <dt>0</dt> </dl> | Aucune valeur de propriétaire assignée à IBIA (International Biometric Industry Association) n’est disponible.<br/> |
| <span id="WINBIO_NO_FORMAT_TYPE_AVAILABLE"></span><span id="winbio_no_format_type_available"></span><dl> <dt>**WINBIO \_ AUCUN \_ type de format \_ \_ disponible**</dt> <dt>0</dt> </dl>    | Aucun type de format affecté par le propriétaire n’est disponible.<br/>                                               |
