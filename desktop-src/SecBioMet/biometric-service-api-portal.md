# <a name="windows-biometric-framework"></a>Windows Biometric Framework

## <a name="purpose"></a>Objectif

Vous pouvez utiliser l’API Windows Biometric Framework pour créer des applications clientes qui capturent, enregistrent et comparent en toute sécurité les informations biométriques de l’utilisateur final.

## <a name="developer-audience"></a>Développeurs concernés

Les développeurs qui utilisent cette API doivent être familiarisés avec les langages de programmation C et C++ et dans l’environnement de programmation Windows.

## <a name="run-time-requirements"></a>Conditions d’exécution

L’API Windows Biometric Framework est prise en charge à partir de Windows Server 2008 R2 et Windows 7. Pour plus d’informations sur les exigences d’exécution d’un élément de programmation particulier, consultez la section Configuration requise de la page de référence de cet élément.
