# <a name="plug-in-functions"></a>Fonctions de plug-in

Les rubriques suivantes décrivent les fonctions que vous pouvez utiliser pour créer les plug-ins d’adaptateur qui composent une unité biométrique :

## <a name="in-this-section"></a>Contenu de cette section



| Rubrique                                                                 | Description                                                                                                                                                                                                                                  |
|-----------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Fonctions d’adaptateur de moteur](engine-adapter-functions.md)<br/>   | Un adaptateur de moteur génère des modèles biométriques à partir d’exemples capturés, fait correspondre des exemples à des modèles existants et des modèles d’index.<br/>                                                                                              |
| [Fonctions d’adaptateur de capteur](sensor-adapter-functions.md)<br/>   | Un adaptateur de capteur encapsule un périphérique biométrique et fournit une interface standard qui permet au service de biométrie Windows de configurer le capteur, de capturer des échantillons et de contrôler le workflow de données biométriques pour le moteur de traitement.<br/> |
| [Fonctions des adaptateurs de stockage](storage-adapter-functions.md)<br/> | Un adaptateur de stockage gère les bases de données de modèles.<br/>                                                                                                                                                                                     |
