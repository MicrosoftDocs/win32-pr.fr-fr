## <a name="remarks"></a>Notes

Les \_ \_ constantes POS non spécifiées du doigt WINBIO ne \_ \_ \* font pas partie de la spécification ANSI 381. Vous pouvez utiliser ces constantes pour inscrire jusqu’à dix doigts (par SID de compte) qui n’ont pas d’informations de position associées.

> [!IMPORTANT]
>
> N’essayez pas de valider les valeurs de constante d’empreinte digitale. le Service de biométrie Windows validera les valeurs de constantes d’empreinte digitale fournies avant de les transmettre à votre implémentation.
