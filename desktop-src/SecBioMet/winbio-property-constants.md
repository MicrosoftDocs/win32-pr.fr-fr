# <a name="winbio_property-constants"></a>\_Constantes de propriété WINBIO

Les constantes suivantes sont des valeurs d' **\_ \_ ID de propriété WINBIO** qui peuvent être utilisées pour spécifier la propriété à interroger dans le paramètre *PropertyId* de la fonction [**WinBioGetProperty**](/windows/desktop/api/Winbio/nf-winbio-winbiogetproperty) . Certaines propriétés peuvent également être définies à l’aide du même paramètre dans la fonction [**WinBioSetProperty**](/windows/desktop/api/winbio/nf-winbio-winbiosetproperty) . Les résultats de la requête sont retournés ou spécifiés dans le paramètre *PropertyBuffer* de ces fonctions, et le type et la taille du résultat varient en fonction de la propriété interrogée.

Le résultat de la requête de propriété est retourné sous la forme d’une valeur **ULong** qui contient l’indicateur.<br/> Quand cette propriété est interrogée, les paramètres *SessionHandle* et *UnitId* transmis doivent être valides, le paramètre *Identity* doit avoir la **valeur null** *et le paramètre* subversion doit être **WINBIO \_ SubType \_ no \_ information**.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| <span id="WINBIO_PROPERTY_EXTENDED_SENSOR_INFO"></span><span id="winbio_property_extended_sensor_info"></span><dl> <dt>**WINBIO \_ PROPRIÉTÉ \_ étendue \_ - \_ informations sur le capteur**</dt> <dt>2</dt> </dl>                   | Cette propriété biométrique en lecture seule contient des informations étendues sur les capacités et les attributs du composant de capteur qui est connecté à une unité biométrique spécifique. Le résultat de la requête de propriété est retourné sous la forme d’une structure d' [**\_ \_ \_ informations de capteur étendue WINBIO**](winbio-extended-sensor-info.md) .<br/> Quand cette propriété est interrogée, les paramètres *SessionHandle* et *UnitId* transmis doivent être valides, le paramètre *Identity* doit avoir la **valeur null** *et le paramètre* subversion doit être **WINBIO \_ SubType \_ no \_ information**.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |

Le résultat de la requête de propriété est retourné en tant que structure d' [**\_ \_ \_ informations de stockage étendu WINBIO**](winbio-extended-storage-info.md) .<br/> Quand cette propriété est interrogée, les paramètres *SessionHandle* et *UnitId* transmis doivent être valides, le paramètre *Identity* doit avoir la **valeur null** *et le paramètre* subversion doit être **WINBIO \_ SubType \_ no \_ information**.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |

Les utilisateurs sans privilèges peuvent uniquement modifier leur propre paramètre de stratégie. Si un utilisateur sans privilège tente d’appeler la fonction [**WinBioSetProperty**](/windows/desktop/api/winbio/nf-winbio-winbiosetproperty) avec un paramètre d' *identité* qui représente un autre compte d’utilisateur ou contient une valeur d’identificateur de caractère générique, la tentative d’écriture de la propriété échoue.<br/> |
