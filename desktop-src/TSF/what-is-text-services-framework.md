# <a name="what-is-text-services-framework"></a>Qu’est-ce que Text Services Framework ?

Text Services Framework (TSF) est un système indépendant du périphérique, indépendant du langage et extensible pour les technologies d’entrée et de langage de texte.

Les applications compatibles TSF peuvent recevoir des entrées de texte de n’importe quel service de texte qui prend en charge TSF sans avoir à connaître les détails de la source de texte.

Les services de texte fournissent du texte aux applications d’une manière qui ne nécessite aucune connaissance détaillée des applications qui produisent ou reçoivent le texte. Par exemple, un service de texte peut fournir une entrée de texte à partir de périphériques de saisie vocale ou d’entrée manuscrite.
