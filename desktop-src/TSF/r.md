# <a name="r-text-services-framework"></a>R (Text Services Framework)

<dl> <dt>

<span id="_tsf_range_gly"></span><span id="_TSF_RANGE_GLY"></span>**vont**
</dt> <dd>

Objet qui décrit une étendue de texte qui existe dans un flux de texte. Une plage est utilisée pour lire, écrire et manipuler du texte. Également appelée *plage de texte*.

Voir aussi, *région*.

</dd> <dt>

<span id="_tsf_reconversion_gly"></span><span id="_TSF_RECONVERSION_GLY"></span>**reconversion**
</dt> <dd>

Processus permettant à l’utilisateur de sélectionner du texte de remplacement pour le texte inséré.

</dd> <dt>

<span id="_tsf_region_gly"></span><span id="_TSF_REGION_GLY"></span>**sous**
</dt> <dd>

Étendue de texte gérée comme une plage d’édition unique, en fonction de la mise en forme.

</dd> </dl>
