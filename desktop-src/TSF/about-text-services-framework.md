# <a name="about-text-services-framework"></a>À propos de Text Services Framework

Microsoft Windows Text Services Framework (TSF) est une interface de programmation d’applications (API) qui permet l’entrée de texte avancée et indépendante de la source. TSF expose son infrastructure aux applications et aux services de texte via un ensemble d’interfaces COM.

-   [Comment configurer Text Services Framework](how-to-set-up-tsf.md)
