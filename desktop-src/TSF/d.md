# <a name="d-text-services-framework"></a>D (Text Services Framework)

<dl> <dt>

<span id="_tsf_document_lock_gly"></span><span id="_TSF_DOCUMENT_LOCK_GLY"></span>**verrou de document**
</dt> <dd>

État du document dans lequel le gestionnaire TSF peut lire ou modifier le texte sans que le texte soit modifié par d’autres moyens. Un verrou de document peut être en lecture seule ou en lecture/écriture.

</dd> </dl>
