# <a name="tf_-constants"></a>TF, \_ \* constantes

Les appels de fonction TSF Atom utilisent ces constantes pour gérer les processus d’entrée de thread.



| Constante/valeur                                                                                                                                                                                                                                                                       | Description                                                                                          |
|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------|
| <span id="TF_PROCESS_ATOM"></span><span id="tf_process_atom"></span><dl> <dt>**Tf \_ TRAITER \_**</dt> le <dt>texte Atom ( \_ CTF \_ process \_ Atom \_ )</dt> </dl>                               | Utilisé pour notifier les fonctions Atom lancées par un processus d’entrée de thread.<br/>                  |
| <span id="TF_ENABLE_PROCESS_ATOM"></span><span id="tf_enable_process_atom"></span><dl> <dt>**Tf \_ ACTIVER le \_ processus \_ Atom**</dt> <dt>Text ( \_ CTF \_ Enable \_ process \_ Atom \_ )</dt> </dl> | Utilisé pour notifier TSF, par le biais de fonctions Atom, qu’un processus d’entrée de thread a été activé.<br/> |
