# <a name="m-text-services-framework"></a>M (Text Services Framework)

<dl> <dt>

<span id="_tsf_modal_gly"></span><span id="_TSF_MODAL_GLY"></span>**exigeant**
</dt> <dd>

Terme décrivant une fenêtre secondaire qui restreint l’interaction de l’utilisateur avec d’autres fenêtres. Une fenêtre modale, telle qu’une boîte de dialogue, doit être fermée par l’utilisateur pour que l’application puisse continuer.
