# <a name="ienumtfrenderingmarkupreset-method"></a>IEnumTfRenderingMarkup :: Reset, méthode

La méthode **IEnumTfRenderingMarkup :: Reset** réinitialise l’objet énumérateur en déplaçant la position actuelle jusqu’au début de la séquence d’énumération.

## <a name="syntax"></a>Syntaxe


```C++
HRESULT Reset();
```



## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.
