# <a name="mode-bias-values"></a>Valeurs de décalage de mode

Les valeurs prédéfinies suivantes sont des valeurs possibles pour la propriété [GUID \_ prop \_ MODEBIAS](predefined-properties.md) .

<dl> <dt>

<span id="GUID_MODEBIAS_NONE_____________"></span><span id="guid_modebias_none_____________"></span>GUID \_ MODEBIAS \_ aucun 
</dt> <dd>

Il n’y a pas de décalage de mode.

</dd> <dt>

<span id="GUID_MODEBIAS_FILENAME_____________"></span><span id="guid_modebias_filename_____________"></span>\_nom de \_ fichier MODEBIAS GUID 
</dt> <dd>

Le décalage est vers un nom de fichier.

</dd> <dt>

<span id="GUID_MODEBIAS_NAME_____________"></span><span id="guid_modebias_name_____________"></span>\_nom MODEBIAS \_ GUID 
</dt> <dd>

Le biais correspond à un nom.

</dd> <dt>

<span id="GUID_MODEBIAS_READING_____________"></span><span id="guid_modebias_reading_____________"></span>lecture du GUID \_ MODEBIAS \_ 
</dt> <dd>

Le décalage concerne la lecture.

</dd> <dt>

<span id="GUID_MODEBIAS_DATETIME_____________"></span><span id="guid_modebias_datetime_____________"></span>GUID \_ MODEBIAS \_ DateTime 
</dt> <dd>

L’écart concerne une date ou une heure.

</dd> <dt>

<span id="GUID_MODEBIAS_CONVERSATION_____________"></span><span id="guid_modebias_conversation_____________"></span>GUID \_ MODEBIAS \_ conversation 
</dt> <dd>

Le décalage concerne une conversation.

</dd> <dt>

<span id="GUID_MODEBIAS_NUMERIC_____________"></span><span id="guid_modebias_numeric_____________"></span>GUID \_ MODEBIAS \_ numérique 
</dt> <dd>

Le biais est un nombre.

</dd> <dt>

<span id="GUID_MODEBIAS_HIRAGANA_____________"></span><span id="guid_modebias_hiragana_____________"></span>GUID \_ MODEBIAS \_ hiragana 
</dt> <dd>

Le décalage est vers des chaînes Hiragana.

</dd> <dt>

<span id="GUID_MODEBIAS_KATAKANA_____________"></span><span id="guid_modebias_katakana_____________"></span>GUID \_ MODEBIAS \_ Katakana 
</dt> <dd>

Le décalage est le biais des chaînes Katakana.

</dd> <dt>

<span id="GUID_MODEBIAS_CHINESE_____________"></span><span id="guid_modebias_chinese_____________"></span>GUID \_ MODEBIAS \_ chinois 
</dt> <dd>

Le décalage concerne les caractères chinois.

</dd> <dt>

<span id="GUID_MODEBIAS_HANGUL_____________"></span><span id="guid_modebias_hangul_____________"></span>GUID \_ MODEBIAS \_ HANGUL 
</dt> <dd>

Le décalage concerne les caractères Hangul.

</dd> <dt>

<span id="GUID_MODEBIAS_HALFWIDTHKATAKANA_____________"></span><span id="guid_modebias_halfwidthkatakana_____________"></span>GUID \_ MODEBIAS \_ HALFWIDTHKATAKANA 
</dt> <dd>

Le décalage est vers les chaînes katakana à demi-chasse.

</dd> <dt>

<span id="GUID_MODEBIAS_FULLWIDTHALPHANUMERIC_____________"></span><span id="guid_modebias_fullwidthalphanumeric_____________"></span>GUID \_ MODEBIAS \_ FULLWIDTHALPHANUMERIC 
</dt> <dd>

L’écart est de caractères alphanumériques à pleine chasse.

</dd> <dt>

<span id="GUID_MODEBIAS_HALFWIDTHALPHANUMERIC_____________"></span><span id="guid_modebias_halfwidthalphanumeric_____________"></span>GUID \_ MODEBIAS \_ HALFWIDTHALPHANUMERIC 
</dt> <dd>

Le décalage se fait en caractères alphanumériques à demi-chasse.

</dd> <dt>

<span id="GUID_MODEBIAS_FULLWIDTHHANGUL_____________"></span><span id="guid_modebias_fullwidthhangul_____________"></span>GUID \_ MODEBIAS \_ FULLWIDTHHANGUL

</dd> </dl>
