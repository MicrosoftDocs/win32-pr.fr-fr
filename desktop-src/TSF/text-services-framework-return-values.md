# <a name="text-services-framework-return-values"></a>Valeurs de retour de l’infrastructure Text services

Les méthodes d’interface de Text Services Framework produisent des valeurs de retour dans la plage de 0x *hhhh* 0200 à 0x *hhhh* 0600, où *hhhh* est une valeur hexadécimale qui dépend du code de gravité et du code de fonction d’une valeur de retour spécifique.

Les valeurs de retour regroupent dans les catégories suivantes.

-   [Valeurs de retour de service de texte](text-service-return-values.md)
-   [Valeurs de retour du gestionnaire](manager-return-values.md)
-   [Valeurs de retour du magasin de texte](text-store-return-values.md)
