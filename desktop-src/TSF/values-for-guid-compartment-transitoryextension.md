# <a name="values-for-guid_compartment_transitoryextension"></a>Valeurs pour le \_ compartiment GUID \_ TRANSITORYEXTENSION

Les valeurs suivantes du \_ compartiment TRANSITORYEXTENSION du compartiment GUID \_ sont utilisées pour contrôler le comportement de l’extension transitoire.

<dl> <dt>

<span id="TF_TRANSITORYEXTENSION_NONE"></span><span id="tf_transitoryextension_none"></span>**TF \_ TRANSITORYEXTENSION \_ aucun**
</dt> <dd>

Cette valeur arrête l’extension transitoire.

</dd> <dt>

<span id="TF_TRANSITORYEXTENSION_FLOATING"></span><span id="tf_transitoryextension_floating"></span>**\_TRANSITORYEXTENSION TF \_ flottant**
</dt> <dd>

Cette valeur démarre l’extension transitoire avec l’interface utilisateur flottante.

</dd> <dt>

<span id="TF_TRANSITORYEXTENSION_ATSELECTION"></span><span id="tf_transitoryextension_atselection"></span>**TF \_ TRANSITORYEXTENSION \_ ATSELECTION**
</dt> <dd>

Cette valeur démarre l’extension transitoire avec l’interface utilisateur contextuelle à l’adresse IP ou la sélection du gestionnaire de documents parent.

</dd> </dl>
