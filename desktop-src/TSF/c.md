# <a name="c-text-services-framework"></a>C (Text Services Framework)

<dl> <dt>

<span id="_tsf_candidates_gly"></span><span id="_TSF_CANDIDATES_GLY"></span>**finales**
</dt> <dd>

Ensemble de caractères ou de idéogrammes alternatifs affichés dans une fenêtre candidate par une méthode d’entrée. Les candidats sont obtenus à partir d’un dictionnaire de conversion et représentent une chaîne phonétique précédemment entrée par l’utilisateur.

</dd> <dt>

<span id="_tsf_client_gly"></span><span id="_TSF_CLIENT_GLY"></span>**client**
</dt> <dd>

Dans Text Services Framework, une application ou un service de texte compatible TSF.

</dd> <dt>

<span id="_tsf_context_gly"></span><span id="_TSF_CONTEXT_GLY"></span>**contexte**
