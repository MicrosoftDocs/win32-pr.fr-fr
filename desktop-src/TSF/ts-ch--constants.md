# <a name="ts_ch_-constants"></a>\_ \_ \* Constantes TS ch

Les \_ \_ \* constantes TS ch sont utilisées en interne par TSF pour indiquer que des suppressions de texte ont eu lieu.



| Constante/valeur                                                                                                                                                                                                                               | Description                                                        |
|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------------------|
| <span id="TS_CH_PRECEDING_DEL"></span><span id="ts_ch_preceding_del"></span><dl> <dt>**TS \_ CH \_ précédent \_ del**</dt> <dt>(1)</dt> </dl> | Le texte qui précède la position du caractère a été supprimé.<br/> |
| <span id="TS_CH_FOLLOWING_DEL"></span><span id="ts_ch_following_del"></span><dl> <dt>**TS \_ CH \_ suivant \_ del**</dt> <dt>(2)</dt> </dl> | Le texte qui suit la position du caractère a été supprimé.<br/> |
