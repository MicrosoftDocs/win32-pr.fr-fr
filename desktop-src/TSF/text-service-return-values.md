# <a name="text-service-return-values"></a>Valeurs de retour de service de texte

Un service de texte génère une valeur de retour de 0x *HHHH* 0600. Le tableau suivant répertorie la valeur de retour du service de texte.



| Code/valeur de retour                                                                                                                                                                                                                          | Description                                                           |
|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------|
| <span id="TF_E_NOCONVERSION"></span><span id="tf_e_noconversion"></span><dl> <dt>**Tf \_ E \_ noconversion**</dt> <dt>0x80040600</dt> </dl> | Aucune conversion n’est disponible pour la plage de texte sélectionnée.<br/> |
