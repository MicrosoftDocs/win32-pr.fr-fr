# <a name="predefined-language-bar-items"></a>Éléments de la barre de langue prédéfinis

Les valeurs [**GUID**](/windows/desktop/TSF/predefined-lang-bar-items) suivantes identifient les éléments de la barre de langue prédéfinis. Ces valeurs sont utilisées avec [**GetItem**](/windows/desktop/api/Ctfutb/nf-ctfutb-itflangbaritemmgr-getitem) pour obtenir un élément spécifique de la barre de langue.

<dl> <dt>

<span id="_______________GUID_LBI_SAPILAYR_CFGMENUBUTTON_____________"></span><span id="_______________guid_lbi_sapilayr_cfgmenubutton_____________"></span> GUID \_ IFA \_ SAPILAYR \_ CFGMENUBUTTON 
</dt> <dd>

Élément de menu de la barre de langage des outils vocaux.

</dd> <dt>

<span id="_______________GUID_TFCAT_TIP_HANDWRITING_____________"></span><span id="_______________guid_tfcat_tip_handwriting_____________"></span>\_ \_ \_ écriture manuscrite GUID TFCAT Tip 
</dt> <dd>

Élément de type écriture manuscrite.

</dd> <dt>

<span id="_______________GUID_TFCAT_TIP_KEYBOARD_____________"></span><span id="_______________guid_tfcat_tip_keyboard_____________"></span>\_clavier GUID TFCAT \_ Tip \_ 
</dt> <dd>

Élément de type de clavier.

</dd> </dl>
