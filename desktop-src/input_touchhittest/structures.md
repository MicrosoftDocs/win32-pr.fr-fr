# <a name="touch-hit-testing-structures"></a>Structures de test de positionnement tactile

Les rubriques de cette section fournissent les spécifications de référence pour les structures de [test de positionnement tactile](touch-hit-testing-portal.md) .

## <a name="in-this-section"></a>Contenu de cette section

| Rubrique | Description |
| --- | --- |
| [**TOUCH_HIT_TESTING_INPUT**](/windows/win32/api/winuser/ns-winuser-touch_hit_testing_input)<br/> | Contient des informations sur la zone de contact tactile signalée par le digitaliseur tactile.<br/> |
| [**TOUCH_HIT_TESTING_PROXIMITY_EVALUATION**](/windows/win32/api/winuser/ns-winuser-touch_hit_testing_proximity_evaluation)<br/> | Contient le score du test d’atteinte qui indique si l’objet est la cible probable de la zone tactile du contact, par rapport aux autres objets qui croisent la zone tactile du contact.<br/> |
