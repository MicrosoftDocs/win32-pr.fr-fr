# <a name="single-instance-store-and-sis-backup"></a>Sauvegarde Single-Instance Store et SIS

Un *magasin d’instances uniques*, ou SIS, est une architecture conçue pour gérer les fichiers dupliqués avec un minimum de ressources disque, cache et support de sauvegarde. La sauvegarde du *stockage d’instance unique*, ou sauvegarde SIS, est une API que les applications de sauvegarde appellent pour accéder à l’architecture SIS.
