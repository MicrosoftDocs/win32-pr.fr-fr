# <a name="r-upnp-apis"></a>R (API UPnP)

Contient les termes relatifs à UPnP qui commencent par la lettre R.

[A](a-gly.md) B [C](c-gly.md) [D](d-gly.md) [e](e-gly.md) F G [H](h-gly.md) I J K L M [N](n-gly.md) O [P](p-gly.md) Q R [S](s-gly.md) T [U](u-gly.md) V W X Y Z

<dl> <dt>

<span id="upnp.r_1_gly"></span><span id="UPNP.R_1_GLY"></span>**Répertoire de ressources**
</dt> <dd>

Le répertoire de ressources est un répertoire local de l’appareil dans lequel se trouvent des fichiers tels que des fichiers d’icône.

</dd> <dt>

<span id="upnp.r_2_gly"></span><span id="UPNP.R_2_GLY"></span>**appareil racine**
</dt> <dd>

Un appareil racine est l’appareil le plus élevé dans une arborescence d’appareils. Par exemple, si vous accédez à un appareil de combinaison VCR/TV appelé station de visualisation, il y a trois appareils : la station de visualisation, le magnétoscope et la télévision. La station d’affichage est considérée comme l’appareil racine.

</dd> <dt>

<span id="upnp.r_3_gly"></span><span id="UPNP.R_3_GLY"></span>**appareil en cours d’exécution**
</dt> <dd>

Un appareil en cours d’exécution est un appareil sur lequel son code a déjà été chargé avant l’inscription.

Il s’agit des appareils pour lesquels une description de service n’est pas connue avant l’inscription.

Les ponts sont des exemples de ces périphériques, tels qu’un pont UPnP SCP. Ce pont inscrit les appareils en cours d’exécution auprès de l’hôte de l’appareil.

</dd> </dl>
