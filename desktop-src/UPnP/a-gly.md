# <a name="a-upnp-apis"></a>A (API UPnP)

Contient les termes relatifs à UPnP qui commencent par la lettre A.

A B [C](c-gly.md) [D](d-gly.md) [e](e-gly.md) F G [H](h-gly.md) I J K L M [N](n-gly.md) O [P](p-gly.md) Q [R](r-gly.md) [S](s-gly.md) T [U](u-gly.md) V W X Y Z

<dl> <dt>

<span id="upnp.a_1_gly"></span><span id="UPNP.A_1_GLY"></span>**transactionnel**
</dt> <dd>

Une action représente une commande qu’un client émet pour exécuter une fonction spécifique. Il accepte zéro ou plusieurs arguments.

</dd> <dt>

<span id="upnp.a_2_gly"></span><span id="UPNP.A_2_GLY"></span>**annoncé**
</dt> <dd>

L’annonce d’un appareil sur un réseau signifie que les messages sont multidiffusés sur tous les nœuds du réseau à propos de la disponibilité de l’appareil.

</dd> <dt>

<span id="upnp.a_3_gly"></span><span id="UPNP.A_3_GLY"></span>**argument**
</dt> <dd>

Un argument est un paramètre d’une action.

</dd> </dl>
