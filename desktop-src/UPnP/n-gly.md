<dl> <dt>

<span id="upnp.n_1_gly"></span><span id="UPNP.N_1_GLY"></span>**appareil non en cours d’exécution**
</dt> <dd>

Un périphérique qui n’est pas en cours d’exécution est un appareil pour lequel le code est chargé uniquement lors de la réception d’une demande pour cet appareil. Il s’agit souvent de périphériques pour lesquels des descriptions de service sont connues avant l’inscription.

Un appareil logiciel, tel qu’un lecteur multimédia, est un exemple classique d’appareils non exécutés.

</dd> </dl>
