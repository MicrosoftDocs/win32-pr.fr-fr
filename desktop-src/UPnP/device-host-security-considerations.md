# <a name="device-host-security-considerations"></a><span data-ttu-id="2b806-103">Considérations sur la sécurité des hôtes d’appareils</span><span class="sxs-lookup"><span data-stu-id="2b806-103">Device Host Security Considerations</span></span>

<span data-ttu-id="2b806-104">L’utilisation de l’hôte d’appareil crée des problèmes de sécurité en raison des éléments suivants :</span><span class="sxs-lookup"><span data-stu-id="2b806-104">Using the device host creates security issues because of the following:</span></span>

-   <span data-ttu-id="2b806-105">Les appareils hébergés sur un ordinateur exécutant Windows XP envoient des annonces sur tous les réseaux.</span><span class="sxs-lookup"><span data-stu-id="2b806-105">Devices hosted on a computer running Windows XP sends announcements on all networks.</span></span>
-   <span data-ttu-id="2b806-106">Les appareils hébergés sur un ordinateur exécutant Windows XP autorisent le contrôle des appareils à partir de tous les réseaux.</span><span class="sxs-lookup"><span data-stu-id="2b806-106">Devices hosted on a computer running Windows XP allow control of devices from all networks.</span></span>

<span data-ttu-id="2b806-107">Cela augmente le risque pour les particuliers, car les appareils tels qu’un lecteur multimédia ou un système de climatisation ou un appareil HVAC hébergé sur un ordinateur exécutant Windows XP sont visibles et peuvent être contrôlés à partir des points de contrôle en dehors de la famille.</span><span class="sxs-lookup"><span data-stu-id="2b806-107">This increases the risk to home consumers, because devices such as a media player or a bridged lighting or HVAC system hosted on a computer running Windows XP are visible and can be controlled from control points outside the home.</span></span>

<span data-ttu-id="2b806-108">Lorsque vous créez un appareil hébergé, vous devez prendre en considération certains problèmes de sécurité.</span><span class="sxs-lookup"><span data-stu-id="2b806-108">When you are creating a hosted device, you need to take into consideration some security issues.</span></span>

-   <span data-ttu-id="2b806-109">Pour réduire l’étendue de la découverte et de l’attaque des périphériques UPnP, la durée de vie de tous les messages SSDP est de 1.</span><span class="sxs-lookup"><span data-stu-id="2b806-109">To reduce the scope of discovery and attack of UPnP-based devices, the TTL of all SSDP messages is 1.</span></span> <span data-ttu-id="2b806-110">Cela signifie qu’un appareil inscrit est découvert uniquement par des points de contrôle sur le même réseau.</span><span class="sxs-lookup"><span data-stu-id="2b806-110">This means that a registered device is only discovered by control points on the same network.</span></span> <span data-ttu-id="2b806-111">Vous pouvez configurer une durée de vie plus élevée dans le registre.</span><span class="sxs-lookup"><span data-stu-id="2b806-111">You can configure a higher TTL in the registry.</span></span>
-   <span data-ttu-id="2b806-112">L’inscription d’un appareil non en cours d’exécution nécessite la pré-inscription de l’appareil. dll auprès de COM, ce qui nécessite des privilèges d’administrateur.</span><span class="sxs-lookup"><span data-stu-id="2b806-112">Registering a non-running device requires pre-registering the device .dll with COM, which requires administrator privilege.</span></span>
-   <span data-ttu-id="2b806-113">L’inscription d’un appareil en cours d’exécution nécessite un privilège administrateur, service local ou système local.</span><span class="sxs-lookup"><span data-stu-id="2b806-113">Registering a running device requires Administrator, Local Service, or Local System privilege.</span></span>
-   <span data-ttu-id="2b806-114">Lorsque l’hôte d’appareil est démarré, il est exécuté en tant que [LocalService](/windows/desktop/Services/localservice-account).</span><span class="sxs-lookup"><span data-stu-id="2b806-114">When the device host is started, it is run as [LocalService](/windows/desktop/Services/localservice-account).</span></span> <span data-ttu-id="2b806-115">Cela donne à l’appareil la possibilité de générer des audits et de lire la clé de Registre **HKEY \_ local \_ machine** .</span><span class="sxs-lookup"><span data-stu-id="2b806-115">This gives the device the ability to generate audits and read the **HKEY\_LOCAL\_MACHINE** registry key.</span></span> <span data-ttu-id="2b806-116">L’appareil a accès à **HKEY \_ Current \_ User**.</span><span class="sxs-lookup"><span data-stu-id="2b806-116">The device does have access to **HKEY\_CURRENT\_USER**.</span></span> <span data-ttu-id="2b806-117">Le compte LocalService peut utiliser des ressources auxquelles l’accès LocalService a été accordé, ainsi que celles qui accordent l’accès à AuthenticatedUser.</span><span class="sxs-lookup"><span data-stu-id="2b806-117">The LocalService account can use resources to which LocalService has been granted access, as well as those that grant access to AuthenticatedUser.</span></span> <span data-ttu-id="2b806-118">L’appareil a accès limité au système de fichiers.</span><span class="sxs-lookup"><span data-stu-id="2b806-118">The device has restricted file system access.</span></span>
-   <span data-ttu-id="2b806-119">Les listes de contrôle d’accès du système de fichiers doivent être mises à jour pour autoriser l’accès [LocalService](/windows/desktop/Services/localservice-account) au répertoire de ressources.</span><span class="sxs-lookup"><span data-stu-id="2b806-119">The file system ACLs must be updated to allow [LocalService](/windows/desktop/Services/localservice-account) access to the resource directory.</span></span>
