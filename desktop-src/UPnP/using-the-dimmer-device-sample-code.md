# <a name="using-the-dimmer-device-sample-code"></a>Utilisation de l’exemple de code de l’appareil DIMM

Le kit de développement logiciel (SDK) de la plateforme comprend un exemple de code de périphérique DIMM. Cet exemple de code illustre un appareil UPnP simple implémenté à l’aide de l’API d’hôte d’appareil avec la technologie UPnP.

Le \_ répertoire DCO DimmerService contient l’implémentation de Device. dll. Le répertoire RegisterDevice contient l’implémentation pour l’inscription de Device. dll auprès du service d’inscription UPnP sur l’ordinateur. Le répertoire StepByStep contient des projets VC 6.0 qui correspondent au « Guide pas à pas » contenu dans le fichier Readme.txt. Le « guide pas à pas » est destiné aux développeurs qui n’ont pas d’expérience ATL/COM antérieure.
