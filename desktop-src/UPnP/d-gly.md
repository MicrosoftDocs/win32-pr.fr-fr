# <a name="d-upnp-apis"></a>D (API UPnP)

Contient les termes relatifs à UPnP qui commencent par la lettre D.

[A](a-gly.md) B [C](c-gly.md) D [e](e-gly.md) F G [H](h-gly.md) I J K L M [N](n-gly.md) O [P](p-gly.md) Q [R](r-gly.md) [S](s-gly.md) T [U](u-gly.md) V W X Y Z

<dl> <dt>

<span id="upnp.d_1_gly"></span><span id="UPNP.D_1_GLY"></span>**passerelle**
</dt> <dd>

Un appareil est un conteneur d’autres services ou périphériques. Un appareil peut contenir d’autres périphériques logiques.

</dd> <dt>

<span id="upnp.d_2_gly"></span><span id="UPNP.D_2_GLY"></span>**objet de contrôle de périphérique**
</dt> <dd>

Un objet de contrôle d’appareil est le point central de gestion des objets de service d’un appareil. Lorsqu’un appareil est inscrit, son objet de contrôle de l’appareil est transmis à l’hôte de l’appareil.

</dd> <dt>

<span id="upnp.d_3_gly"></span><span id="UPNP.D_3_GLY"></span>**Description de l’appareil**
</dt> <dd>

Une description de l’appareil est une liste des propriétés génériques de tous les appareils d’un appareil physique, y compris les services, la hiérarchie de l’appareil et ses propriétés.

</dd> <dt>

<span id="upnp.d_4_gly"></span><span id="UPNP.D_4_GLY"></span>**hôte de l’appareil**
</dt> <dd>

L’hôte d’appareil gère la découverte, la description, le contrôle et les éléments d’événement des fonctionnalités d’un appareil. Les appareils contiennent le code permettant de gérer les fonctionnalités spécifiques au périphérique, telles que le rembobinage d’un magnétoscope.

</dd> <dt>

<span id="upnp.d_5_gly"></span><span id="UPNP.D_5_GLY"></span>**fournisseur d’appareils**
</dt> <dd>

Un fournisseur d’appareils est une entité inscrite auprès de l’hôte de l’appareil. Ce fournisseur d’appareils n’est pas publié sur le réseau. Toutefois, le fournisseur de périphériques inscrit d’autres périphériques et les publie sur le réseau.

</dd> <dt>

<span id="upnp.d_6_gly"></span><span id="UPNP.D_6_GLY"></span>**type d’appareil**
</dt> <dd>

Un type d’appareil standard est indiqué par urn : schemas-UPnP-org : Device : suivi d’un nom unique attribué par le Forum UPnP.

Il existe une relation un-à-un entre les modèles d’appareil UPnP et les types d’appareils. Les fournisseurs UPnP peuvent spécifier des types d’appareils supplémentaires. celles-ci sont désignées par urn : Domain-Name : Device : suivi d’un nom unique attribué par le fournisseur, où Domain-Name est un nom de domaine inscrit au fournisseur.

</dd> </dl>
