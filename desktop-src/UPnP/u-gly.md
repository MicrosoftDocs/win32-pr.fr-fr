# <a name="u-upnp-apis"></a>U (API UPnP)

Contient les termes relatifs à UPnP qui commencent par la lettre U.

[A](a-gly.md) B [C](c-gly.md) [D](d-gly.md) [e](e-gly.md) F G [H](h-gly.md) I J K L M [N](n-gly.md) O [P](p-gly.md) Q [R](r-gly.md) [S](s-gly.md) T U V W X Y Z

<dl> <dt>

<span id="upnp.u_1_gly"></span><span id="UPNP.U_1_GLY"></span>**UDN**
</dt> <dd>

Représente un nom de périphérique unique. Un UDN identifie de façon unique un appareil ; Il est basé sur l’UUID (Universal unique ID).

</dd> <dt>

<span id="upnp.u_2_gly"></span><span id="UPNP.U_2_GLY"></span>**UTL**
</dt> <dd>

Représente la langue du modèle UPnP. La langue de modèle UPnP est utilisée pour spécifier les descriptions de périphérique ou de service.

</dd> </dl>
