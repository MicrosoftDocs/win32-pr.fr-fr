<span data-ttu-id="8a83c-106">**Pour découvrir des appareils**</span><span class="sxs-lookup"><span data-stu-id="8a83c-106">**To discover devices**</span></span>

1.  <span data-ttu-id="8a83c-107">Sélectionnez le type de recherche que vous souhaitez utiliser : **Rechercher par type**, **Rechercher par UDN** ou **recherche asynchrone**.</span><span class="sxs-lookup"><span data-stu-id="8a83c-107">Select the type of search you want to use: **Find by Type**, **Find by UDN**, or **Async Find**.</span></span>

3.  <span data-ttu-id="8a83c-110">Cliquez sur **Démarrer la découverte**.</span><span class="sxs-lookup"><span data-stu-id="8a83c-110">Click **Start Discovery**.</span></span> <span data-ttu-id="8a83c-111">La recherche est démarrée.</span><span class="sxs-lookup"><span data-stu-id="8a83c-111">The search is started.</span></span> <span data-ttu-id="8a83c-112">Les résultats s’affichent dans la liste **appareils trouvés** .</span><span class="sxs-lookup"><span data-stu-id="8a83c-112">The results are displayed in the **Devices Found** list.</span></span> <span data-ttu-id="8a83c-113">Si aucun appareil n’est trouvé, un message s’affiche pour indiquer cette condition.</span><span class="sxs-lookup"><span data-stu-id="8a83c-113">If no devices are found, a message is displayed indicating this condition.</span></span> <span data-ttu-id="8a83c-114">L’état de la progression de la recherche s’affiche dans le champ **État** .</span><span class="sxs-lookup"><span data-stu-id="8a83c-114">The status of the search progress is displayed in the **Status** field.</span></span>
