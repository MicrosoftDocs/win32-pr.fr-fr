# <a name="e-upnp-apis"></a>E (API UPnP)

Contient les termes relatifs à UPnP qui commencent par la lettre E.

[A](a-gly.md) B [C](c-gly.md) [D](d-gly.md) e F G [H](h-gly.md) I J K L M [N](n-gly.md) O [P](p-gly.md) Q [R](r-gly.md) [S](s-gly.md) T [U](u-gly.md) V W X Y Z

<dl> <dt>

<span id="upnp.e_1_gly"></span><span id="UPNP.E_1_GLY"></span>**événement**
</dt> <dd>

Un événement est une notification d’une ou plusieurs modifications apportées aux variables d’État exposées par un service.

</dd> <dt>

<span id="upnp.e_2_gly"></span><span id="UPNP.E_2_GLY"></span>**variable avec événements**
</dt> <dd>

Une variable d’événement est une variable d’État pour un service qui, lorsqu’elle est modifiée, déclenche un événement. Tous les points de contrôle qui sont abonnés à la source d’événements d’une variable reçoivent alors la notification de la modification. Une variable d’État sans événement n’est jamais incluse dans une notification d’événement.

</dd> </dl>
