# <a name="about-the-device-host-api"></a>À propos de l’API d’hôte d’appareil

l’API d’hôte d’appareil avec la technologie upnp est une infrastructure permettant d’implémenter les fonctionnalités d’appareil upnp sur la plateforme Windows. Les développeurs qui créent des appareils à l’aide de l’API d’hôte d’appareil avec la technologie UPnP (appelés [*appareils hébergés*](h-gly.md)) ont besoin d’implémenter uniquement les fonctionnalités principales de l’appareil. Les développeurs peuvent s’appuyer sur l’hôte de l’appareil pour gérer les détails propres à UPnP en matière de découverte, de description, de contrôle, d’événement et de présentation. L’hôte d’appareil valide les données entrantes à partir de clients UPnP et met en forme toutes les données sortantes à partir d’appareils hébergés en fonction de l’architecture de périphérique UPnP.
