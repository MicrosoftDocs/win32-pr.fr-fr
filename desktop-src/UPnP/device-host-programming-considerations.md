# <a name="device-host-programming-considerations"></a>Considérations sur la programmation des hôtes d’appareils

Une URL de présentation vide dans la description de l’appareil n’est pas autorisée. Si votre appareil n’a pas d’URL de présentation, supprimez les balises d’URL de présentation de la description de l’appareil.
