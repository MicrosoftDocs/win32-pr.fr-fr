# <a name="windows-machine-learning-interfaces"></a>Interfaces Windows Machine Learning

Les rubriques suivantes décrivent les interfaces COM pour le kit de développement logiciel (SDK) Windows Machine Learning.

## <a name="in-this-section"></a>Contenu de cette section



| Rubrique                                                                             | Description                                                                                                                                                                                |
|-----------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IWinMLEvaluationContext**](/windows/win32/api/winml/nn-winml-iwinmlevaluationcontext)<br/> | Représente le contexte pour lier les entrées et les sorties à un modèle WinML.<br/>                                                                                                             |
| [**IWinMLModel**](/windows/win32/api/winml/nn-winml-iwinmlmodel)<br/>                         | Représente un modèle de Machine Learning Windows avec les métadonnées correspondantes ; comprend des descriptions de modèle (nom, auteur, contrôle de version, etc.), ainsi que les entrées et sorties attendues. <br/> |
| [**IWinMLRuntime**](/windows/win32/api/winml/nn-winml-iwinmlruntime)<br/>                     | Représente le runtime pour charger et évaluer un modèle WinML.<br/>                                                                                                                      |
| [**IWinMLRuntimeFactory**](/windows/win32/api/winml/nn-winml-iwinmlruntimefactory)<br/>       | Représente la fabrique qui crée le runtime WinML pour le chargement et l’évaluation du modèle.<br/>                                                                                         |
