# <a name="application-scenarios-for-sens"></a>Scénarios d’application pour SENS

Plusieurs types d’applications peuvent utiliser les fonctions de connectivité et les services de notification proposés par le service de notification d’événements système :

-   Application qui requiert l’état de la connectivité réseau, telle qu’une application qui utilise les services d’annuaire.
-   Application qui adapte ses opérations en fonction du niveau de connectivité et de la qualité des services réseau, tels qu’une application de navigateur Internet qui fonctionne à un niveau réduit sur une connexion à faible bande passante.
-   Application qui peut effectuer des opérations différées, telles qu’un programme de messagerie électronique qui peut mettre en file d’attente des messages en mode hors connexion et les envoyer quand une connexion est établie.
