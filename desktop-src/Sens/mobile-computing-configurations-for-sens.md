# <a name="mobile-computing-configurations-for-sens"></a>Configurations de l’informatique mobile pour SENS

Les fonctions de connectivité et les notifications sont utiles pour les ordinateurs configurés comme suit :

-   Ordinateur mobile utilisé dans une station d’accueil sur un réseau à bande passante élevée qui peut parfois utiliser une connexion d’accès à distance.
-   Un ordinateur mobile utilisant exclusivement une connexion d’accès à distance.
-   Un ordinateur de bureau utilisant exclusivement une connexion d’accès à distance.
-   Un ordinateur de bureau connecté à un réseau à bande passante élevée avec des problèmes de latence.

Dans chacune de ces configurations, les informations sur la bande passante et la latence de la connexion peuvent être utilisées par une application pour optimiser dynamiquement ses opérations en vue de la disponibilité du réseau.
