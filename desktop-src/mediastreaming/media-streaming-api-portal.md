# <a name="windows-media-streaming-api"></a>Windows API de diffusion multimédia en continu

## <a name="purpose"></a>Objectif

L’API de diffusion multimédia en continu modélise les appareils de rendu de média numérique (DMR) et de Digital Media Server (DMS) tels que définis dans les instructions de DLNA pour faciliter l’utilisation par programme, ce qui permet aux développeurs de créer des applications avec des fonctionnalités de contrôleur multimédia numérique (DMC), Digital Media Player (DMP) et des contrôleurs push (+ PU), comme défini

## <a name="developer-audience"></a>Développeurs concernés

L’API de diffusion multimédia en continu est conçue pour être utilisée par les développeurs C++ qui créent des applications multimédias numériques qui interagissent avec des appareils DLNA.
