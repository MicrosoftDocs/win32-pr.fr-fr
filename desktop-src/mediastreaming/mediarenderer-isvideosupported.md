# <a name="mediarendererisvideosupported-property"></a>MediaRenderer. IsVideoSupported, propriété

Obtient une valeur qui indique si DMR est capable de regarder du contenu vidéo.

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```C++
HRESULT get_IsVideoSupported(
  [out] boolean *value
);
```



## <a name="property-value"></a>Valeur de la propriété

Valeur booléenne qui est **true** si DMR est capable de visionner du contenu vidéo et **false** si ce n’est pas le cas.
