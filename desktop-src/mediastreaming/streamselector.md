# <a name="streamselector-class"></a>StreamSelector, classe

Implémente l’interface [**IStreamSelectorStatics**](/previous-versions/windows/desktop/api/windows.media.streaming/nn-windows-media-streaming-istreamselectorstatics) et permet de sélectionner un flux.

**StreamSelector** possède les types de membres suivants :

-   [Méthodes](#methods)

### <a name="methods"></a>Méthodes

La classe **StreamSelector** possède ces méthodes.



| Méthode                                                                      | Description                                                  |
|:----------------------------------------------------------------------------|:-------------------------------------------------------------|
| [**GetStreamPropertiesAsync**](/previous-versions/windows/desktop/legacy/hh829001(v=vs.85)) | Obtient les propriétés du flux de manière asynchrone.<br/> |
| [**SelectBestStreamAsync**](/previous-versions/windows/desktop/legacy/hh829002(v=vs.85))       | Requêtes de manière asynchrone pour le meilleur flux.<br/>       |
