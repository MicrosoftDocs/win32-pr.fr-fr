# <a name="basicdevicecanwakedevices-property"></a>BasicDevice. CanWakeDevices, propriété

Obtient une valeur qui indique si l’appareil peut sortir de veille.

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```C++
HRESULT get_CanWakeDevices(
  [out] boolean *value
);
```



## <a name="property-value"></a>Valeur de la propriété

Pointeur vers une valeur booléenne qui est **true** si l’appareil peut sortir de veille.
