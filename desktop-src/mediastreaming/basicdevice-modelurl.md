# <a name="basicdevicemodelurl-property"></a>BasicDevice. ModelUrl, propriété

Obtient l’URL du modèle de l’appareil.

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```C++
HRESULT get_ModelUrl(
  [out] HSTRING *value
);
```



## <a name="property-value"></a>Valeur de la propriété

Pointeur vers l’URL du modèle de l’appareil.
