# <a name="media-streaming-structures"></a>Structures de diffusion multimédia en continu

L' [API de diffusion multimédia en continu](media-streaming-api-portal.md) fournit les structures suivantes.

## <a name="in-this-section"></a>Contenu de cette section



| Rubrique                                                           | Description                                                                                                                                        |
|-----------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------|
| [**PlaySpeed**](/previous-versions/windows/desktop/api/windows.media.streaming/ns-windows-media-streaming-playspeed)<br/>                       | Représente une vitesse de lecture sous la forme d’un nombre rationnel. <br/>                                                                                      |
| [**PositionInformation**](/previous-versions/windows/desktop/api/windows.media.streaming/ns-windows-media-streaming-positioninformation)<br/>   | Contient les valeurs actuelles des informations de position de lecture du média obtenues à partir de la DMR.<br/>                                               |
| [**RenderingParameters**](/previous-versions/windows/desktop/api/windows.media.streaming/ns-windows-media-streaming-renderingparameters)<br/>   | Contient les valeurs actuelles des paramètres de rendu sur la DMR.<br/>                                                                         |
| [**TrackInformation**](/previous-versions/windows/desktop/api/windows.media.streaming/ns-windows-media-streaming-trackinformation)<br/>         | Contient le numéro de suivi actuel et la durée dans le cadre du [**PositionInformation**](/previous-versions/windows/desktop/api/windows.media.streaming/ns-windows-media-streaming-positioninformation) obtenu à partir de la DMR.<br/> |
| [**TransportInformation**](/previous-versions/windows/desktop/api/windows.media.streaming/ns-windows-media-streaming-transportinformation)<br/> | Contient les valeurs actuelles des informations relatives au transport de lecture du média obtenues à partir du DMR.<br/>                                      |
