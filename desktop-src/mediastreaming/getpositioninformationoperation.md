# <a name="getpositioninformationoperation-class"></a>GetPositionInformationOperation, classe

Inscrit un gestionnaire d’événements qui est appelé lorsque l’opération asynchrone démarrée par [**GetPositionInformationAsync**](/previous-versions/windows/desktop/api/windows.media.streaming/nf-windows-media-streaming-imediarenderer-getpositioninformationasync) se termine, et fournit une méthode qui retourne les résultats de l’opération.

**GetPositionInformationOperation** possède les types de membres suivants :

-   [Méthodes](#methods)
-   [Propriétés](#properties)

### <a name="methods"></a>Méthodes

La classe **GetPositionInformationOperation** possède ces méthodes.



| Méthode                                                           | Description                                                                                                                                                 |
|:-----------------------------------------------------------------|:------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**GetResults**](getpositioninformationoperation-getresults.md) | Retourne les résultats de l’opération asynchrone démarrée par [**GetPositionInformationAsync**](/previous-versions/windows/desktop/api/windows.media.streaming/nf-windows-media-streaming-imediarenderer-getpositioninformationasync). <br/> |



 

### <a name="properties"></a>Propriétés

La classe **GetPositionInformationOperation** possède les propriétés suivantes.



| Propriété                                                                  | Type d’accès           | Description                                                                                                                                                                                          |
|:--------------------------------------------------------------------------|:----------------------|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Completed**](getpositioninformationoperation-completed.md)<br/> | Lecture/écriture<br/> | Obtient ou définit un gestionnaire d’événements qui est appelé lorsque l’opération asynchrone démarrée par [**GetPositionInformationAsync**](/previous-versions/windows/desktop/api/windows.media.streaming/nf-windows-media-streaming-imediarenderer-getpositioninformationasync) est terminée. <br/> |
