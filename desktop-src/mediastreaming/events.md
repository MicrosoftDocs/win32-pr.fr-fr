# <a name="events-media-streaming-api"></a>Événements (API de diffusion multimédia en continu)

L' [API de diffusion multimédia en continu](media-streaming-api-portal.md) génère les événements suivants.

## <a name="in-this-section"></a>Contenu de cette section



| Rubrique                                                                           | Description                                                                                                                                              |
|---------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Événement ConnectionStatusChanged**](connectionstatuschanged.md)<br/>     | Se produit lors de la modification de l’état de la connexion réseau de l’appareil.<br/>                                                                                   |
| [**Événement DeviceArrival**](devicearrival.md)<br/>                         | Se produit lorsqu’un nouvel appareil est ajouté à la liste des appareils retournés par la méthode [**CachedDevices**](idevicecontroller-cacheddevices.md) .<br/>     |
| [**Événement DeviceDeparture**](devicedeparture.md)<br/>                     | Se produit lorsqu’un nouvel appareil est supprimé de la liste des appareils retournés par la méthode [**CachedDevices**](idevicecontroller-cacheddevices.md) .<br/> |
| [**Événement RenderingParametersUpdate**](renderingparametersupdate.md)<br/> | Se produit chaque fois qu’un jeu de paramètres de contrôle de rendu est mis à jour sur le DMR.<br/>                                                          |
| [**Événement TransportParametersUpdate**](transportparametersupdate.md)<br/> | Se produit chaque fois qu’un ensemble de paramètres de transport est mis à jour sur le DMR.<br/>                                                                  |
