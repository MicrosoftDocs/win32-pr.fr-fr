# <a name="createmediarendereroperationgetresults-method"></a>CreateMediaRendererOperation :: GetResults, méthode

Obtient les résultats de l’opération asynchrone.

## <a name="syntax"></a>Syntaxe


```C++
MediaRenderer* GetResults();
```



## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.

## <a name="return-value"></a>Valeur retournée

Résultats de l’opération. Pointeur vers l’objet [**MediaRenderer**](mediarenderer.md) nouvellement créé.
