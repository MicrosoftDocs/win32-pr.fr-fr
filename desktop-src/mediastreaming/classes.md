# <a name="classes-media-streaming-api"></a>Classes (API de diffusion multimédia en continu)

L' [API de diffusion multimédia en continu](media-streaming-api-portal.md) fournit les classes suivantes.

## <a name="in-this-section"></a>Contenu de cette section



| Rubrique                                                                                   | Description                                                                                                                                                                                                                                                                                                                                                                         |
|-----------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**ActiveBasicDevice**](/previous-versions/windows/desktop/legacy/dn385755(v=vs.85))<br/>                               | Implémente l’interface [**IActiveBasicDevice**](/previous-versions/windows/desktop/api/windows.media.streaming/nn-windows-media-streaming-iactivebasicdevice) qui représente un périphérique DLNA (Digital vivant Network Alliance) actif.<br/>                                                                                                                                                                                                                       |
| [**BasicDevice**](/previous-versions/windows/desktop/legacy/hh828813(v=vs.85))<br/>                                           | Implémente l’interface [**IBasicDevice**](ibasicdevice.md) qui représente un appareil DLNA.<br/>                                                                                                                                                                                                                                                                             |
| [**CreateMediaRendererOperation**](createmediarendereroperation.md)<br/>         | Inscrit un gestionnaire d’événements qui est appelé lorsque l’opération asynchrone démarrée par [**CreateMediaRendererAsync**](imediarendererfactory-createmediarendererasync.md) ou [**CreateMediaRendererFromBasicDeviceAsync**](imediarendererfactory-createmediarendererfrombasicdeviceasync.md) se termine, et fournit une méthode qui retourne les résultats de l’opération.<br/> |
| [**DeviceController**](/previous-versions/windows/desktop/legacy/hh828842(v=vs.85))<br/>                                 | Implémente l’interface [**IDeviceController**](/previous-versions/windows/desktop/api/windows.media.streaming/nn-windows-media-streaming-idevicecontroller) qui récupère une liste de convertisseurs multimédias numériques mis en cache (DMRs) et/ou de serveurs multimédias numériques (DMSs), ou pour rechercher de manière asynchrone les DMRs et/ou DMSs qui sont actuellement sur le réseau.<br/>                                                                                                            |
| [**DevicePair**](/previous-versions/windows/desktop/legacy/dn385771(v=vs.85))<br/>                                             | Implémente l’interface [**IDevicePair**](/previous-versions/windows/desktop/api/windows.media.streaming/nn-windows-media-streaming-idevicepair) qui représente une paire d’objets [**ActiveBasicDevice**](/previous-versions/windows/desktop/legacy/dn385755(v=vs.85)) qui est constituée d’un convertisseur et d’un serveur.<br/>                                                                                                                                                                              |
| [**GetMuteOperation**](getmuteoperation.md)<br/>                                 | Inscrit un gestionnaire d’événements qui est appelé lorsque l’opération asynchrone démarrée par [**GetMuteAsync**](/previous-versions/windows/desktop/api/windows.media.streaming/nf-windows-media-streaming-imediarenderer-getmuteasync) se termine, et fournit une méthode qui retourne les résultats de l’opération.<br/>                                                                                                                                                    |
| [**GetPositionInformationOperation**](getpositioninformationoperation.md)<br/>   | Inscrit un gestionnaire d’événements qui est appelé lorsque l’opération asynchrone démarrée par [**GetPositionInformationAsync**](/previous-versions/windows/desktop/api/windows.media.streaming/nf-windows-media-streaming-imediarenderer-getpositioninformationasync) se termine, et fournit une méthode qui retourne les résultats de l’opération.<br/>                                                                                                                      |
| [**GetStreamPropertiesOperation**](getstreampropertiesoperation.md)<br/>         | Inscrit un gestionnaire d’événements qui est appelé lorsque l’opération asynchrone démarrée par [**GetStreamPropertiesAsync**](/previous-versions/windows/desktop/legacy/hh829001(v=vs.85)) se termine, et fournit une méthode qui retourne les résultats de l’opération.<br/>                                                                                                                            |
| [**GetTransportInformationOperation**](gettransportinformationoperation.md)<br/> | Inscrit un gestionnaire d’événements qui est appelé lorsque l’opération asynchrone démarrée par [**GetTransportInformationAsync**](/previous-versions/windows/desktop/api/windows.media.streaming/nf-windows-media-streaming-imediarenderer-gettransportinformationasync) se termine, et fournit une méthode qui retourne les résultats de l’opération.<br/>                                                                                                                    |
| [**GetVolumeOperation**](getvolumeoperation.md)<br/>                             | Inscrit un gestionnaire d’événements qui est appelé lorsque l’opération asynchrone démarrée par [**GetVolumeAsync**](/previous-versions/windows/desktop/api/windows.media.streaming/nf-windows-media-streaming-imediarenderer-getvolumeasync) se termine, et fournit une méthode qui retourne les résultats de l’opération.<br/>                                                                                                                                                |
| [**MediaRenderer**](mediarenderer.md)<br/>                                       | Implémente l’interface [**IMediaRenderer**](imediarenderer.md) qui représente un appareil de convertisseur de médias numériques DLNA (DMR).<br/>                                                                                                                                                                                                                                            |
| [**PlaybackOperation**](playbackoperation.md)<br/>                               | Inscrit un gestionnaire d’événements qui est appelé lorsqu’une opération asynchrone démarrée par l’une des méthodes de lecture [**MediaRenderer**](mediarenderer.md) se termine, et fournit une méthode qui retourne les résultats de l’opération.<br/>                                                                                                                                      |
| [**StreamSelectOperation**](streamselectoperation.md)<br/>                       | Inscrit un gestionnaire d’événements qui est appelé lorsque l’opération asynchrone démarrée par [**GetMuteAsync**](/previous-versions/windows/desktop/api/windows.media.streaming/nf-windows-media-streaming-imediarenderer-getmuteasync) se termine, et fournit une méthode qui retourne les résultats de l’opération.<br/>                                                                                                                                                    |
| [**StreamSelector**](streamselector.md)<br/>                                     | Implémente l’interface [**IStreamSelectorStatics**](/previous-versions/windows/desktop/api/windows.media.streaming/nn-windows-media-streaming-istreamselectorstatics) et permet de sélectionner un flux.<br/>                                                                                                                                                                                                                                                        |
