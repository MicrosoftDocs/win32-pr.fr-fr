# <a name="basicdevicediscoveredoncurrentnetwork-property"></a>BasicDevice. DiscoveredOnCurrentNetwork, propriété

Obtient une valeur qui indique si l’appareil se trouve sur le réseau actuel.

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```C++
HRESULT get_DiscoveredOnCurrentNetwork(
  [out] boolean *value
);
```



## <a name="property-value"></a>Valeur de la propriété

Pointeur vers une valeur booléenne qui est **true** si l’appareil se trouve sur le réseau actuel.
