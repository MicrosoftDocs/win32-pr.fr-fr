# <a name="getmuteoperation-class"></a>GetMuteOperation, classe

Inscrit un gestionnaire d’événements qui est appelé lorsque l’opération asynchrone démarrée par [**GetMuteAsync**](/previous-versions/windows/desktop/api/windows.media.streaming/nf-windows-media-streaming-imediarenderer-getmuteasync) se termine, et fournit une méthode qui retourne les résultats de l’opération.

**GetMuteOperation** possède les types de membres suivants :

-   [Méthodes](#methods)
-   [Propriétés](#properties)

### <a name="methods"></a>Méthodes

La classe **GetMuteOperation** possède ces méthodes.



| Méthode                                            | Description                                                                                                                   |
|:--------------------------------------------------|:------------------------------------------------------------------------------------------------------------------------------|
| [**GetResults**](getmuteoperation-getresults.md) | Retourne les résultats de l’opération asynchrone démarrée par [**GetMuteAsync**](/previous-versions/windows/desktop/api/windows.media.streaming/nf-windows-media-streaming-imediarenderer-getmuteasync). <br/> |



 

### <a name="properties"></a>Propriétés

La classe **GetMuteOperation** possède les propriétés suivantes.



| Propriété                                                   | Type d’accès           | Description                                                                                                                                                            |
|:-----------------------------------------------------------|:----------------------|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Completed**](getmuteoperation-completed.md)<br/> | Lecture/écriture<br/> | Obtient ou définit un gestionnaire d’événements qui est appelé lorsque l’opération asynchrone démarrée par [**GetMuteAsync**](/previous-versions/windows/desktop/api/windows.media.streaming/nf-windows-media-streaming-imediarenderer-getmuteasync) est terminée. <br/> |
