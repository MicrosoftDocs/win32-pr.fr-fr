# <a name="basicdevicedescription-property"></a>Propriété BasicDevice. Description

Obtient une description de l’appareil.

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```C++
HRESULT get_Description(
  [out] HSTRING *value
);
```



## <a name="property-value"></a>Valeur de la propriété

Pointeur vers la description de l’appareil.
