# <a name="basicdevicemanufacturerurl-property"></a>BasicDevice. ManufacturerUrl, propriété

Obtient l’URL du fabricant de l’appareil.

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```C++
HRESULT get_ManufacturerUrl(
  [out] HSTRING *value
);
```



## <a name="property-value"></a>Valeur de la propriété

Pointeur vers l’URL du fabricant de l’appareil.
