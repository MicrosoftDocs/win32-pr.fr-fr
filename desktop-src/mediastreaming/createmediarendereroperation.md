# <a name="createmediarendereroperation-class"></a>CreateMediaRendererOperation, classe

Inscrit un gestionnaire d’événements qui est appelé lorsque l’opération asynchrone démarrée par [**CreateMediaRendererAsync**](imediarendererfactory-createmediarendererasync.md) ou [**CreateMediaRendererFromBasicDeviceAsync**](imediarendererfactory-createmediarendererfrombasicdeviceasync.md) se termine, et fournit une méthode qui retourne les résultats de l’opération.

**CreateMediaRendererOperation** possède les types de membres suivants :

-   [Méthodes](#methods)
-   [Propriétés](#properties)

### <a name="methods"></a>Méthodes

La classe **CreateMediaRendererOperation** possède ces méthodes.



| Méthode                                                        | Description                                                 |
|:--------------------------------------------------------------|:------------------------------------------------------------|
| [**GetResults**](createmediarendereroperation-getresults.md) | Obtient les résultats de l’opération asynchrone. <br/> |



 

### <a name="properties"></a>Propriétés

La classe **CreateMediaRendererOperation** possède les propriétés suivantes.



| Propriété                                                               | Type d’accès           | Description                                                                                                                                                                                                                                                                                                               |
|:-----------------------------------------------------------------------|:----------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Completed**](createmediarendereroperation-completed.md)<br/> | Lecture/écriture<br/> | Obtient ou définit un gestionnaire d’événements qui est appelé lorsque l’opération asynchrone démarrée par [**CreateMediaRendererAsync**](imediarendererfactory-createmediarendererasync.md) ou [**CreateMediaRendererFromBasicDeviceAsync**](imediarendererfactory-createmediarendererfrombasicdeviceasync.md) est terminée. <br/> |
