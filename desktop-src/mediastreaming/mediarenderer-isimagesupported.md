# <a name="mediarendererisimagesupported-property"></a>MediaRenderer. IsImageSupported, propriété

Obtient une valeur qui indique si la DMR est capable d’afficher des images.

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```C++
HRESULT get_IsImageSupported(
  [out] boolean *value
);
```



## <a name="property-value"></a>Valeur de la propriété

Valeur booléenne qui est **true** si DMR est capable d’afficher des images et **false** si ce n’est pas le cas.
