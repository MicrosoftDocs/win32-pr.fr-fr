# <a name="mediarendererisaudiosupported-property"></a>MediaRenderer. IsAudioSupported, propriété

Obtient une valeur qui indique si DMR est capable de diffuser du contenu audio.

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```C++
HRESULT get_IsAudioSupported(
  [out] boolean *value
);
```



## <a name="property-value"></a>Valeur de la propriété

Valeur booléenne qui est **true** si DMR est capable de diffuser du contenu audio et **false** si ce n’est pas le cas.
