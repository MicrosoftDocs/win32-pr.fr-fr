# <a name="basicdevicetype-property"></a>BasicDevice. type, propriété

Obtient une valeur d’énumération indiquant le type d’appareil de l’appareil DLNA.

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```C++
HRESULT get_Type(
  [out] DeviceTypes *value
);
```



## <a name="property-value"></a>Valeur de la propriété

Pointeur vers une valeur [**DeviceTypes**](devicetypes.md) .
