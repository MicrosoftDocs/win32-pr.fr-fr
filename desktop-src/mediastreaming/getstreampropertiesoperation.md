# <a name="getstreampropertiesoperation-class"></a>GetStreamPropertiesOperation, classe

Inscrit un gestionnaire d’événements qui est appelé lorsque l’opération asynchrone démarrée par [**GetStreamPropertiesAsync**](/previous-versions/windows/desktop/legacy/hh829001(v=vs.85)) se termine, et fournit une méthode qui retourne les résultats de l’opération.

**GetStreamPropertiesOperation** possède les types de membres suivants :

-   [Méthodes](#methods)
-   [Propriétés](#getstreampropertiesoperation-class)

### <a name="methods"></a>Méthodes

La classe **GetStreamPropertiesOperation** possède ces méthodes.



| Méthode                                                        | Description                                                                                                                                          |
|:--------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------------------------|
| [**GetResults**](getstreampropertiesoperation-getresults.md) | Retourne les résultats de l’opération asynchrone démarrée par [**GetStreamPropertiesAsync**](/previous-versions/windows/desktop/legacy/hh829001(v=vs.85)).<br/> |



 

### <a name="properties"></a>Propriétés

La classe **GetStreamPropertiesOperation** possède les propriétés suivantes.



| Propriété                                                               | Type d’accès           | Description                                                                                                                                                                                   |
|:-----------------------------------------------------------------------|:----------------------|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Completed**](getstreampropertiesoperation-completed.md)<br/> | Lecture/écriture<br/> | Obtient ou définit un gestionnaire d’événements qui est appelé lorsque l’opération asynchrone démarrée par [**GetStreamPropertiesAsync**](/previous-versions/windows/desktop/legacy/hh829001(v=vs.85)) est terminée.<br/> |
