# <a name="playbackoperation-class"></a>PlaybackOperation, classe

Inscrit un gestionnaire d’événements qui est appelé lorsqu’une opération asynchrone démarrée par l’une des méthodes de lecture [**MediaRenderer**](mediarenderer.md) se termine, et fournit une méthode qui retourne les résultats de l’opération.

**PlaybackOperation** possède les types de membres suivants :

-   [Méthodes](#methods)
-   [Propriétés](#properties)

### <a name="methods"></a>Méthodes

La classe **PlaybackOperation** possède ces méthodes.



| Méthode                                             | Description                                                                                                                                |
|:---------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------------------------|
| [**GetResults**](playbackoperation-getresults.md) | Retourne les résultats d’une opération asynchrone démarrée par l’une des méthodes de lecture [**MediaRenderer**](mediarenderer.md) .<br/> |



 

### <a name="properties"></a>Propriétés

La classe **PlaybackOperation** possède les propriétés suivantes.



| Propriété                                                    | Type d’accès           | Description                                                                                                                                                                           |
|:------------------------------------------------------------|:----------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Completed**](playbackoperation-completed.md)<br/> | Lecture/écriture<br/> | Obtient ou définit un gestionnaire d’événements qui est appelé lorsque l’opération asynchrone démarrée par l’une des méthodes de lecture [**MediaRenderer**](mediarenderer.md) est terminée. <br/> |
