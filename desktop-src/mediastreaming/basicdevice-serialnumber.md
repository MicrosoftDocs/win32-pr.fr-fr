# <a name="basicdeviceserialnumber-property"></a>BasicDevice. SerialNumber, propriété

Obtient le numéro de série de l’appareil.

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```C++
HRESULT get_SerialNumber(
  [out] HSTRING *value
);
```



## <a name="property-value"></a>Valeur de la propriété

Pointeur vers le numéro de série de l’appareil.
