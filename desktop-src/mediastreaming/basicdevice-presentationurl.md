# <a name="basicdevicepresentationurl-property"></a>BasicDevice. PresentationUrl, propriété

Obtient l’URL de présentation de l’appareil.

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```C++
HRESULT get_PresentationUrl(
  [out] HSTRING *value
);
```



## <a name="property-value"></a>Valeur de la propriété

Pointeur vers l’URL de présentation de l’appareil.
