# <a name="network-dynamic-data-exchange"></a>échange dynamique de données réseau

\[Le DDE réseau n’est plus pris en charge. Nddeapi.dll est présent sur Windows Vista, mais tous les appels de fonction renvoient NDDE \_ non \_ implémenté.\]

Les fonctions DDE (Network Dynamic Data Exchange) du réseau permettent à un processus d’établir des conversations avec des processus s’exécutant sur différents ordinateurs d’un réseau.
