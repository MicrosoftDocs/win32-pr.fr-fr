# <a name="mailslot-functions"></a>Fonctions mailslot

Les fonctions suivantes peuvent être utilisées avec les mailslots.



| Fonction                                   | Description                                                                  |
|--------------------------------------------|------------------------------------------------------------------------------|
| [**CreateMailslot**](/windows/desktop/api/Winbase/nf-winbase-createmailslota)   | Crée un mailslot avec le nom spécifié.                                  |
| [**GetMailslotInfo**](/windows/desktop/api/Winbase/nf-winbase-getmailslotinfo) | Récupère des informations sur la mailslot spécifiée.                          |
| [**SetMailslotInfo**](/windows/desktop/api/Winbase/nf-winbase-setmailslotinfo) | Définit la valeur du délai d’attente utilisée par la mailslot spécifiée pour une opération de lecture. |
