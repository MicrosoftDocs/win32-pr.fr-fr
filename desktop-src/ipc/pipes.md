# <a name="pipes-interprocess-communications"></a>Canaux (communications interprocessus)

Un *canal* est une section de la mémoire partagée que les processus utilisent pour la communication. Le processus qui crée un canal est le *serveur de canal*. Un processus qui se connecte à un canal est un *client de canal*. Un processus écrit des informations dans le canal, puis l’autre processus lit les informations à partir du canal. Cette vue d’ensemble décrit comment créer, gérer et utiliser des canaux.
