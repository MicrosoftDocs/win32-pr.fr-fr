# <a name="anonymous-pipes"></a>Canaux anonymes

Un *canal anonyme* est un canal unidirectionnel sans nom qui transfère généralement les données entre un processus parent et un processus enfant. Les canaux anonymes sont toujours locaux. ils ne peuvent pas être utilisés pour la communication sur un réseau.

Pour plus d'informations, voir les rubriques suivantes :

-   [Opérations de canal anonyme](anonymous-pipe-operations.md)
-   [Héritage de handle de canal](pipe-handle-inheritance.md)
-   [Sécurité des canaux anonymes et droits d’accès](anonymous-pipe-security-and-access-rights.md)
