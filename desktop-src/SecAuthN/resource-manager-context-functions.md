# <a name="resource-manager-context-functions"></a>Gestionnaire des ressources les fonctions de contexte

Les fonctions suivantes établissent et libèrent le [*contexte Resource Manager*](../secgloss/r-gly.md) qui est utilisé par les fonctions de requête et de gestion de base de données de la base de données.



| Rubrique                                                  | Description                                                                                                                                          |
|--------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SCardEstablishContext**](/windows/desktop/api/Winscard/nf-winscard-scardestablishcontext) | Établit un contexte pour accéder à la [*base de données de la carte à puce*](../secgloss/s-gly.md). |
| [**SCardReleaseContext**](/windows/desktop/api/Winscard/nf-winscard-scardreleasecontext)     | Ferme un contexte établi.                                                                                                                       |
