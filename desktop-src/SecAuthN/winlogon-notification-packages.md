# <a name="winlogon-notification-packages"></a>Packages de notification Winlogon

Les packages de notification [*Winlogon*](../secgloss/w-gly.md) sont des dll qui reçoivent et gèrent les événements générés par Winlogon. Vous pouvez implémenter ce type de package de notification pour surveiller les événements Winlogon et y répondre. Cela est utile pour les applications qui doivent effectuer un traitement supplémentaire lors de l’ouverture ou de la fermeture de session, ou conserver les informations d' [*État*](../secgloss/s-gly.md) qui doivent être mises à jour lorsque des événements Winlogon se produisent.

Pour plus d’informations sur Winlogon et les GINA, consultez [Winlogon et Gina](winlogon-and-gina.md).
