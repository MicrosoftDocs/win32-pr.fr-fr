# <a name="primary-service-provider"></a>Fournisseur de services principal

Chaque [*carte à puce*](../secgloss/s-gly.md) connue du système peut avoir une interface com inscrite répertoriée en tant que fournisseur de services principal. Lorsque vous accédez à une *carte à puce* via com, ce fournisseur de [*services principal*](../secgloss/p-gly.md) fournit les interfaces de contrôle à la carte. cela permet d’exposer les services de carte à un large éventail d’environnements de programmation, notamment Java et le système de développement Microsoft Visual Basic.
