# <a name="getting-information-about-schannel-connections"></a>Obtention d’informations sur les connexions Schannel

Les rubriques suivantes décrivent les fonctions qui retournent des informations sur les informations d’identification Schannel spécifiées et le [*contexte de sécurité*](/windows/desktop/SecGloss/s-gly) sécurisant une connexion client/serveur.



| Rubrique                                                                                                      | Description                                                                                                                                                                                                                       |
|------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Interrogation des attributs d’un contexte Schannel](querying-the-attributes-of-an-schannel-context.md)       | Décrit comment obtenir les paramètres Schannel spécifiés lors de la création du contexte de sécurité qui sécurise une connexion Schannel.<br/>                                                                                               |
| [Interrogation des attributs d’une information d’identification Schannel](querying-the-attributes-of-an-schannel-credential.md) | Décrit comment obtenir les paramètres Schannel spécifiés lors de la sélection des informations d’identification utilisées pour créer une connexion. Les paramètres sont valides pour toutes les connexions (contextes de sécurité) créés à l’aide des informations d’identification spécifiées.<br/> |
