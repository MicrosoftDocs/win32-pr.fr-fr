# <a name="secure-channel"></a>Canal sécurisé

Un canal sécurisé, également appelé [*Schannel*](../secgloss/s-gly.md), est un [*fournisseur de support de sécurité*](../secgloss/s-gly.md) (SSP) qui contient un ensemble de protocoles de sécurité qui assurent l’authentification des identités et une communication privée sécurisée via le chiffrement.

Schannel est principalement utilisé pour les applications Internet qui nécessitent des communications HTTP (Secure Hypertext Transfer Protocol).
