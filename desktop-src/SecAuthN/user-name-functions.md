# <a name="user-name-functions"></a>Fonctions de nom d’utilisateur

Les fonctions de nom d’utilisateur incluent une seule fonction, [**NPGetUser**](/windows/desktop/api/Npapi/nf-npapi-npgetuser). Cette fonction récupère soit le nom d’utilisateur par défaut actuel, soit le nom d’utilisateur utilisé pour établir la connexion réseau.
