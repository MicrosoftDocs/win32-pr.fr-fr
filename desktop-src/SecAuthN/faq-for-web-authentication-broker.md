# <a name="faq-for-web-authentication-broker"></a>Forum aux questions sur le Service Broker d’authentification Web

Forum aux questions sur le Service Broker d’authentification Web.



| Question                                                                                                    | Réponse                                                                                                                                                                                                            |
|-------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Comment puis-je m’assurer que ma page https://fonctionne avec le Service Broker d’authentification Web ?                         | Essayez de charger la page dans Windows Internet Explorer avant de l’essayer avec le répartiteur d’authentification Web. Si votre page Web se charge sans erreur, elle est également rendue correctement par le Service Broker d’authentification Web. |
| En cas d’erreur, les codes d’erreur s’affichent-ils à l’utilisateur ?                                         | Lorsqu’une page d’erreur s’affiche pour l’utilisateur, le code d’erreur sous-jacent n’est pas affiché. Elle est retournée uniquement à l’application. Vous pouvez également utiliser les journaux des opérations.                                    |
| Comment puis-je trouver plus de détails sur l’erreur rencontrée ?                                                       | Pour plus d’informations, utilisez les journaux des opérations.                                                                                                                                                                             |
| Le conteneur de l’application d’authentification unique (SSO) partage-t-il ses cookies avec Internet Explorer ou d’autres navigateurs ? | Non.                                                                                                                                                                                                               |
