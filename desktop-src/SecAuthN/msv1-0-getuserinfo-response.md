# <a name="msv1_0_getuserinfo_response"></a>\_ \_ Réponse GETUSERINFO MSV1 \_ 0

La \_ structure de \_ réponse GETUSERINFO MSV1 0 \_ est obsolète.

Pour récupérer des informations sur l’utilisateur de la station de travail, consultez la fonction [**NetWkstaUserGetInfo**](/windows/win32/api/lmwksta/nf-lmwksta-netwkstausergetinfo) .
