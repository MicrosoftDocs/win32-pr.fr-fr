# <a name="ietf-specifications-for-microsoft-digest-ssp"></a>Spécifications IETF pour Microsoft Digest SSP

Authentification HTTP : l’authentification d’accès de base et Digest (RFC 2617) est disponible à l’adresse [https://www.ietf.org/rfc/rfc2617.txt](https://www.ietf.org/rfc/rfc2617.txt) .

L’utilisation de l’authentification Digest en tant que mécanisme Internet SASL est disponible à l’adresse [https://www.ietf.org/rfc/rfc2831.txt](https://www.ietf.org/rfc/rfc2831.txt) .

Pour plus d’informations sur SASL, consultez également la spécification Simple Authentication and Security Layer (RFC 2222) disponible à l’adresse [https://www.ietf.org/rfc/rfc2222.txt](https://www.ietf.org/rfc/rfc2222.txt) .
