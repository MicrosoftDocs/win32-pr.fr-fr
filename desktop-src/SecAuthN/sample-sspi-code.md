# <a name="sample-sspi-code"></a>Exemple de code SSPI

Cette section comprend deux exemples complets :

-   [utilisation de SSPI avec un Client Windows sockets](using-sspi-with-a-windows-sockets-client.md)
-   [utilisation de SSPI avec un serveur de sockets Windows](using-sspi-with-a-windows-sockets-server.md)

Les exemples de client et de serveur sont conçus pour fonctionner ensemble. Les deux exemples utilisent le fichier d’en-tête Security. h qui se trouve dans le [fichier d’en-tête pour les exemples de client et de serveur SSPI](header-file-for-sspi-client-and-server.md).

Pour obtenir un exemple d’initialisation d’un tableau de mémoires tampons de sécurité, consultez [exemple de code SecBuffer et SecBufferDesc](secbuffer-and-secbufferdesc-example-code.md).

Pour plus d’informations sur la signature ou le chiffrement des données après l’établissement d’une connexion sécurisée, consultez Utilisation de la [prise en charge des messages](using-message-support.md).
