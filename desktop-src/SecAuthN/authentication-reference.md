# <a name="authentication-reference"></a>Référence d’authentification

Les pages de référence d’authentification contiennent des descriptions détaillées des fonctions, interfaces, objets, structures et autres éléments de programmation de l’authentification Microsoft. Ces rubriques fournissent des informations sur les éléments de programmation utilisés pour établir l’identité des utilisateurs et des services. Une fois qu’une identité est établie, elle peut être utilisée pour tenter d’accéder à un seul ordinateur ou aux ressources d’un réseau informatique.

Les pages de référence sont réparties dans les groupes suivants.



| Section                                                          | Description                                                                          |
|------------------------------------------------------------------|--------------------------------------------------------------------------------------|
| [Constantes d’authentification](authentication-constants.md)         | Constantes utilisées par les éléments de programmation de l’authentification.<br/>                    |
| [Types de données d’authentification](authentication-data-types.md)       | Types de données utilisés par les éléments de programmation de l’authentification.<br/>                   |
| [Énumérations d’authentification](authentication-enumerations.md)   | Énumérations utilisées par les éléments de programmation d’authentification.<br/>                 |
| [Fonctions d’authentification](authentication-functions.md)         | Fonctions utilisées avec l’établissement et la vérification de l’identité de l’utilisateur.<br/>  |
| [Interfaces d’authentification](authentication-interfaces.md)       | Interfaces utilisées avec l’établissement et la vérification de l’identité de l’utilisateur.<br/> |
| [Clés de registre d’authentification](authentication-registry-keys.md) | Clés de Registre utilisées par les éléments de programmation de l’authentification.<br/>                |
| [Valeurs de retour de l’authentification](authentication-return-values.md) | Valeurs de retour utilisées par les éléments de programmation d’authentification.<br/>                |
| [Structures d’authentification](authentication-structures.md)       | Structures utilisées avec les fonctions d’authentification, les interfaces et les objets.<br/>   |
