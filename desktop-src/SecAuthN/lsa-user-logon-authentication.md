# <a name="lsa-user-logon-authentication"></a>Authentification de l’ouverture de session utilisateur LSA

L’autorité LSA gère l’ouverture de session et l’authentification de l’utilisateur sur l’ordinateur local, et si le package d’authentification qui traite la demande d’ouverture de session prend en charge l’authentification directe, l’autorité LSA peut également connecter des utilisateurs à d’autres ordinateurs sur le réseau. Le LSA fournit l’accès aux packages d’authentification pour les fournisseurs de support de sécurité (SSP). L’un des deux types d’authentification suivants est effectué :

-   [Authentification interactive](interactive-authentication.md)
-   [Authentification non interactive](noninteractive-authentication.md)
