# <a name="sspi"></a>SSPI

Cette section présente des informations sur SSPI ( [*Security Support Provider Interface*](../secgloss/s-gly.md) ). L'interface SSPI permet à une application d'utiliser différents modèles de sécurité disponibles sur un ordinateur ou réseau sans modifier l'interface du système de sécurité. Les rubriques sont les suivantes :

-   [Modèle SSPI](sspi-model.md)
-   [Options SSPI pour les applications distribuées](sspi-options-for-distributed-applications.md)
-   [Sémantique du contexte SSPI](sspi-context-semantics.md)
-   [Packages SSP fournis par Microsoft](ssp-packages-provided-by-microsoft.md)
-   [Packages de sécurité personnalisés](custom-security-packages.md)
-   [Informations d’erreur étendues](extended-error-information.md)
