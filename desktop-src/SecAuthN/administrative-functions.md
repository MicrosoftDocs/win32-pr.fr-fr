# <a name="administrative-functions"></a>Fonctions d’administration

les fonctions administratives suivantes permettent à un fournisseur de réseau de prendre des mesures spécifiques au réseau pour afficher et manipuler des répertoires spécifiques au réseau, en d’autres termes, des répertoires mappés à des protocoles non Windows.



| Fonction                                         | Description                                                                                                                                                   |
|--------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**NPGetDirectoryType**](/windows/desktop/api/Npapi/nf-npapi-npgetdirectorytype) | Appelé par le gestionnaire de fichiers pour déterminer le type d’un répertoire réseau.                                                                                          |
| [**NPDirectoryNotify**](/windows/desktop/api/Npapi/nf-npapi-npdirectorynotify)   | Appelé par le gestionnaire de fichiers pour informer le fournisseur réseau des opérations d’annuaire. Cette fonction peut être utilisée pour effectuer un comportement spécial pour certains répertoires. |
