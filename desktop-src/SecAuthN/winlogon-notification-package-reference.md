# <a name="winlogon-notification-package-reference"></a>Référence du package de notification Winlogon

Les rubriques suivantes fournissent des informations de référence pour les packages de notification [*Winlogon*](../secgloss/w-gly.md) :

-   [Entrées de Registre](registry-entries.md) répertorie les valeurs de clés et les paramètres qui doivent être entrés dans le registre système afin d’inscrire un package de notification et ses fonctions de gestionnaire d’événements.
-   Le [prototype de fonction du gestionnaire d’événements](event-handler-function-prototype.md) détaille le prototype pour les fonctions de gestionnaire d’événements. Chaque fonction de gestionnaire d’événements dans votre package de notification doit correspondre à ce prototype.
