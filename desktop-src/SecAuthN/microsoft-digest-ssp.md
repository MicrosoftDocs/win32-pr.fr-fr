# <a name="microsoft-digest-ssp"></a>Microsoft Digest SSP

Microsoft Digest est un [*fournisseur SSP (Security Support Provider*](../secgloss/s-gly.md) ) qui implémente le protocole d’accès Digest, un protocole d’authentification léger pour les parties impliquées dans les communications http (Hypertext Transfer Protocol) ou SASL (Simple Authentication Security Layer).

Microsoft Digest fournit un mécanisme de réponse de stimulation simple pour l’authentification des clients. Ce fournisseur de services partagés est destiné à être utilisé par les applications client/serveur utilisant des communications HTTP ou SASL.

> [!IMPORTANT]
> L’authentification Digest n’est pas prise en charge dans les forêts.
