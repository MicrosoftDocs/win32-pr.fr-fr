# <a name="getting-started-guidance-for-online-providers-integrating-with-the-web-authentication-broker-apis"></a>Guide de prise en main pour les fournisseurs en ligne intégrant les API du Service Broker d’authentification Web

cette section guide les développeurs web et les fournisseurs en ligne pour la création de pages web adaptées aux api du service Broker d’authentification web Windows 8. Il fournit des recommandations visuelles et interactives pour une expérience utilisateur de bout en bout de la page Web, ainsi que l’activation des fonctionnalités d’authentification unique (SSO).
