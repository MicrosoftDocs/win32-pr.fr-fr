# <a name="getting-started-guidance-for-online-providers-integrating-with-the-web-authentication-broker-apis"></a>Guide de prise en main pour les fournisseurs en ligne intégrant les API du Service Broker d’authentification Web

Cette section guide les développeurs Web et les fournisseurs en ligne pour la création de pages Web adaptées aux API du Service Broker d’authentification Web de Windows 8. Il fournit des recommandations visuelles et interactives pour une expérience utilisateur de bout en bout de la page Web, ainsi que l’activation des fonctionnalités d’authentification unique (SSO).
