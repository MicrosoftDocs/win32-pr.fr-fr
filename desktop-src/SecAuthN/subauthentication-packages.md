# <a name="subauthentication-packages"></a>Packages de sous-authentification

les [*packages d’authentification*](../secgloss/a-gly.md) fournis avec Windows prennent en charge la personnalisation à l’aide de packages de sous- [*authentification*](../secgloss/s-gly.md). Un package de sous-authentification est une DLL qui complète ou remplace une partie des critères d’authentification et de validation utilisés par le package d’authentification principal. Par exemple, un serveur particulier peut fournir un package de sous-authentification qui valide le mot de passe d’un utilisateur avec un algorithme différent ou qui spécifie des restrictions de station de travail dans un format différent.

Pour plus d’informations sur les packages de sous-authentification, consultez l’exemple de sous-authentification installé avec le kit de développement logiciel (SDK) de la plateforme.
