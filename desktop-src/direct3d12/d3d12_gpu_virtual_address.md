# <a name="d3d12_gpu_virtual_address"></a>\_ \_ Adresse virtuelle du GPU D3D12 \_

Un D3D12_GPU_VIRTUAL_ADDRESS représente une adresse virtuelle dans la mémoire du GPU. Le type de données est défini dans d3d12. h comme suit :


```
typedef UINT64 D3D12_GPU_VIRTUAL_ADDRESS;
```
