# <a name="interface-hierarchy"></a>Hiérarchie des interfaces

Le diagramme illustre la hiérarchie d’héritage de l’interface.

La plupart des applications utilisent uniquement les interfaces situées en haut de la hiérarchie, et elles n’utilisent pas directement les interfaces héritées ; [**ID3D12Pageable**](/windows/win32/api/d3d12/nn-d3d12-id3d12pageable), [**ID3D12DeviceChild**](/windows/desktop/api/d3d12/nn-d3d12-id3d12devicechild), [**ID3D12Object**](/windows/desktop/api/d3d12/nn-d3d12-id3d12object) et [**ID3D12CommandList**](/windows/desktop/api/d3d12/nn-d3d12-id3d12commandlist).

Les nouvelles interfaces sont affichées dans une zone verte.
