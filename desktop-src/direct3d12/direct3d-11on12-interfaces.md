# <a name="11on12-interfaces"></a>Interfaces 11on12
Les interfaces suivantes sont déclarées dans d3d11on12. h.

## <a name="in-this-section"></a>Contenu de cette section
| Rubrique | Description |
|-|-|
| [**ID3D11On12Device**](/windows/win32/api/d3d11on12/nn-d3d11on12-id3d11on12device) | Gère la création, l’encapsulation et la libération de ressources D3D11 pour Direct3D 11on12. |
| [**ID3D11On12Device1**](/windows/win32/api/d3d11on12/nn-d3d11on12-id3d11on12device1) | Permet une meilleure interopérabilité avec un composant qui peut être transmis à un appareil Direct3D 11, mais qui souhaite utiliser Direct3D 12 à la place. |
| [**ID3D11On12Device2**](/windows/win32/api/d3d11on12/nn-d3d11on12-id3d11on12device2) | Vous permet de prendre des ressources créées par le biais des API Direct3D 11 et de les utiliser dans Direct3D 12. Cette interface étend [ID3D11On12Device1](/windows/win32/api/d3d11on12/nn-d3d11on12-id3d11on12device1). |
