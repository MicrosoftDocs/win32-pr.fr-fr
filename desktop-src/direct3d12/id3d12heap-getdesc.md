# <a name="id3d12heapgetdesc-method"></a>ID3D12Heap :: GetDesc, méthode

Obtient la description du tas.

## <a name="syntax"></a>Syntaxe


```C++
D3D12_HEAP_DESC GetDesc();
```



## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.

## <a name="return-value"></a>Valeur de retour

Type : **[ **\_ \_ Description du tas D3D12**](/windows/desktop/api/d3d12/ns-d3d12-d3d12_heap_desc)**

Retourne la [**structure \_ \_ desc du tas D3D12**](/windows/desktop/api/d3d12/ns-d3d12-d3d12_heap_desc) qui décrit le tas.
