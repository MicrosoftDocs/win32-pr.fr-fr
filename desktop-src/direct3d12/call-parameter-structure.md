# <a name="call-parameter-structure"></a>Structure des paramètres d’appel

Une structure définie par l’utilisateur fournie en tant qu’argument INOUT à un appel [**CallShader**](callshader-function.md) , et en tant que paramètre INOUT pour le [nuanceur pouvant être appelé](callable-shader.md). Le type de structure utilisé dans le nuanceur pouvant être appelé doit correspondre à la structure fournie à l’appel **CallShader** correspondant.
