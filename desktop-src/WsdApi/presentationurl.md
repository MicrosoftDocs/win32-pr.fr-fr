# <a name="presentationurl-element"></a>élément presentationURL

Spécifie l’adresse URL des données de présentation du modèle d’appareil.

## <a name="usage"></a>Usage

``` syntax
<presentationURL/>
```

## <a name="attributes"></a>Attributs

Il n’y a pas d’attributs.

## <a name="child-elements"></a>Éléments enfants

Il n’y a pas d’éléments enfants.

## <a name="parent-elements"></a>Éléments parents



| Élément                                                   | Description                                                                                          |
|-----------------------------------------------------------|------------------------------------------------------------------------------------------------------|
| [**thisModelMetadata**](thismodelmetadata.md)<br/> | Définit les métadonnées du fabricant et du modèle pour l’appareil à implémenter.<br/> <br/> |
