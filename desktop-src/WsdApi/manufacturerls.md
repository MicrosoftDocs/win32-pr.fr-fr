# <a name="manufacturerls-element"></a>élément manufacturerLS

Spécifie une version localisée du nom du fabricant.

## <a name="usage"></a>Utilisation

``` syntax
<manufacturerLS
  Language = "language identifier string"
  Data = "localized manufacturer name string"/>
```

## <a name="attributes"></a>Attributs



| Attribut               | Type                                          | Obligatoire       | Description                                                             |
|-------------------------|-----------------------------------------------|----------------|-------------------------------------------------------------------------|
| **Données**<br/>     | chaîne de nom de fabricant localisée<br/> | Oui<br/> | Nom localisé du fabricant.<br/> <br/>                 |
| **Langage**<br/> | chaîne d’identificateur de langue<br/>         | Oui<br/> | Langue du nom du fabricant localisé.<br/> <br/> |



## <a name="child-elements"></a>Éléments enfants

Il n’y a pas d’éléments enfants.

## <a name="parent-elements"></a>Éléments parents



| Élément                                                   | Description                                                                                          |
|-----------------------------------------------------------|------------------------------------------------------------------------------------------------------|
| [**thisModelMetadata**](thismodelmetadata.md)<br/> | Définit les métadonnées du fabricant et du modèle pour l’appareil à implémenter.<br/> <br/> |
