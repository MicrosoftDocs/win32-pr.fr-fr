# <a name="hostbuilderimplementation-element"></a>élément hostBuilderImplementation

Génère une fonction qui crée un hôte typé.

## <a name="usage"></a>Usage

``` syntax
<hostBuilderImplementation>
  child elements
</hostBuilderImplementation>
```

## <a name="attributes"></a>Attributs

Il n’y a pas d’attributs.

## <a name="child-elements"></a>Éléments enfants



| Élément                                           | Description                                                      |
|---------------------------------------------------|------------------------------------------------------------------|
| [**Service hébergé**](hostedservice.md)<br/> | Service à inclure pour l’hôte. <br/> <br/> |



### <a name="child-element-sequence"></a>Séquence d’éléments enfants

``` syntax
hostedService+
```

## <a name="parent-elements"></a>Éléments parents



| Élément                         | Description                                                    |
|---------------------------------|----------------------------------------------------------------|
| [**txt**](file.md)<br/> | Génère un fichier à partir du générateur de code.<br/> <br/> |
