# <a name="proxybuilderdeclarations-element"></a>élément proxyBuilderDeclarations

Génère des déclarations pour les fonctions qui créent des proxies typés.

## <a name="usage"></a>Utilisation

``` syntax
<proxyBuilderDeclarations>
  child elements
</proxyBuilderDeclarations>
```

## <a name="attributes"></a>Attributs

Il n’y a pas d’attributs.

## <a name="child-elements"></a>Éléments enfants



| Élément                                     | Description                                                                        |
|---------------------------------------------|------------------------------------------------------------------------------------|
| [**proxyClass**](proxyclass.md)<br/> | Nom de la classe à générer à partir de la fonction de générateur de proxy.<br/> <br/> |



### <a name="child-element-sequence"></a>Séquence d’éléments enfants

``` syntax
proxyClass
```

## <a name="parent-elements"></a>Éléments parents



| Élément                         | Description                                                    |
|---------------------------------|----------------------------------------------------------------|
| [**fichier**](file.md)<br/> | Génère un fichier à partir du générateur de code.<br/> <br/> |
