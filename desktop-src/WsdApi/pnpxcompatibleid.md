# <a name="pnpxcompatibleid-element"></a>Élément PnPXCompatibleId

Spécifie l’identificateur compatible PnP-X du service. Les appareils peuvent avoir plus d’un ID compatible.

## <a name="usage"></a>Utilisation

``` syntax
<PnPXCompatibleId/>
```

## <a name="attributes"></a>Attributs

Il n’y a pas d’attributs.

## <a name="child-elements"></a>Éléments enfants

Il n’y a pas d’éléments enfants.

## <a name="parent-elements"></a>Éléments parents



| Élément                             | Description                                                                            |
|-------------------------------------|----------------------------------------------------------------------------------------|
| [**Hébergement**](hosted.md)<br/> | Définit des éléments pour les services définis par l’hôte de service. <br/> <br/> |



## <a name="remarks"></a>Notes

Pour spécifier plusieurs CompatibleID, séparez les identificateurs par un espace, par exemple « PnPX \_ SampleService \_ HWID \_ 1 PnPX \_ SampleService HWID \_ \_ 2 PnPX SampleService1 \_ \_ HWID \_ 3 ».
