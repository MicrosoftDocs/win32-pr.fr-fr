# <a name="web-services-on-devices-constants"></a>Constantes de services Web sur les appareils

Les constantes suivantes sont utilisées par les services Web sur les appareils.



| Constante                                                                                                                                                                                                                        | Description                                                                                                                                                                                      |
|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="WSD_DEFAULT_HOSTING_ADDRESS"></span><span id="wsd_default_hosting_address"></span><dl> <dt>**\_adresse d' \_ hébergement par défaut WSD \_**</dt> </dl>                       | Adresse par défaut (au format [URLPrefix](/windows/desktop/Http/urlprefix-strings) ) qu’un hôte wsdapi utilisera pour écouter les demandes sur le port 5357. <br/>                                                 |
| <span id="WSD_DEFAULT_SECURE_HOSTING_ADDRESS"></span><span id="wsd_default_secure_hosting_address"></span><dl> <dt>**\_adresse d' \_ \_ hébergement sécurisé par défaut WSD \_**</dt> </dl> | Adresse sécurisée par défaut (au format [URLPrefix](/windows/desktop/Http/urlprefix-strings) ) qu’un hôte wsdapi utilisera pour écouter les demandes sur le port 5358. <br/>                                          |
| <span id="WSD_DEFAULT_EVENTING_ADDRESS__"></span><span id="wsd_default_eventing_address__"></span><dl> <dt>**WSD \_ \_ \_ adresse d’événement par défaut**</dt> </dl>               | Adresse par défaut (au format [URLPrefix](/windows/desktop/Http/urlprefix-strings) ) qu’un hôte wsdapi utilise pour écouter les événements sur le port 5358 pour les hôtes sécurisés et sur le port 5357 pour les autres ordinateurs hôtes. <br/> |
