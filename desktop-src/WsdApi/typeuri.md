# <a name="typeuri-element"></a>élément typeUri

Spécifie un type à inclure à partir d’un fichier XSD.

## <a name="usage"></a>Usage

``` syntax
<typeUri
  type = "character_string"
  uri = "character_string"/>
```

## <a name="attributes"></a>Attributs



| Attribut           | Type                         | Obligatoire       | Description                                                            |
|---------------------|------------------------------|----------------|------------------------------------------------------------------------|
| **type**<br/> | chaîne de caractères \_<br/> | Oui<br/> | Nom du type.<br/> <br/>                           |
| **uri**<br/>  | chaîne de caractères \_<br/> | Oui<br/> | Espace de noms du type. Doit être un URI valide.<br/> <br/> |



## <a name="child-elements"></a>Éléments enfants

Il n’y a pas d’éléments enfants.

## <a name="parent-elements"></a>Éléments parents



| Élément                       | Description                                                                       |
|-------------------------------|-----------------------------------------------------------------------------------|
| [**XSD**](xsd.md)<br/> | Spécifie un fichier XSD à traiter pour les informations de contrat.<br/> <br/> |
