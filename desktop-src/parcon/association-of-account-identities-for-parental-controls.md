# <a name="association-of-account-identities-for-parental-controls"></a>Association des identités de compte pour les contrôles parentaux

Comme indiqué dans l’introduction, l’utilisation effective des fonctionnalités de configuration, de journalisation et de restrictions fournies par l’infrastructure de contrôle parental Windows Vista dépend des comptes Windows configurés comme utilisateurs standard. Il est recommandé que les fournisseurs qui fournissent un accès en ligne (tels que les suites de fournisseurs de services Internet) créent et maintiennent une association de noms d’utilisateurs en ligne autorisés par le parent ou Guardian pour chaque compte Windows contrôlé. De cette façon, le contournement des contrôles parentaux basés sur le Cloud, simplement en utilisant des identités en ligne alternatives, est réduit.
