# <a name="parental-controls-activity-reporting"></a>Rapports d’activité du contrôle parental

Les rapports d’activité sont considérés comme essentiels en raison de problèmes d’ingénierie sociale. Des applications ou des filtres compatibles sont demandés pour consigner les activités comme indiqué dans la section utilisation des API de contrôle parental.

Le fait de cliquer sur le lien rapport d’activité permet d’afficher une page de la visionneuse du journal qui affiche toutes les entrées de journal pour un intervalle d’une semaine maximum (limité par les limites de taille du journal), avec les volets des données d’arborescence des catégories et de format de colonne.

La zone Résumé en haut à droite du panneau de contrôle parental indique l’état actuel des rapports d’activité.

Les événements d’activité sont également visibles dans le observateur d’événements avec le nom Microsoft.com/Windows/ParentalControls dans les journaux des applications.

Cette fonctionnalité dépend de la nouvelle fonctionnalité de journalisation des événements dans Windows Vista, à l’aide des API de publication fournies.
