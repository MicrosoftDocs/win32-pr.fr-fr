# <a name="about-parental-controls"></a>À propos des contrôles parentaux

La fonctionnalité de contrôle parental comprend les éléments suivants :

-   Fonctionnalités de l’interface utilisateur centrale intégrées au panneau de configuration.
-   Implémentations de restrictions sélectionnées.
-   Une infrastructure dans le système d’exploitation qui fournit des paramètres pour les stratégies globales et les restrictions implémentées, et qui expose des fonctionnalités d’extensibilité ciblées.
