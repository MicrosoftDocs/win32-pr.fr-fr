# <a name="parental-controls-robustness"></a>Robustesse des contrôles parentaux

Notez que les problèmes d’ingénierie sociale entourant l’application d’une connexion basée sur l’identité, couplés à des limites pratiques en matière de conception de restrictions, empêchent généralement les solutions de sécurité d’être robustes pour les enfants déterminés à outwitr les contrôles. Le principe de Microsoft est que nous ne pouvons pas protéger les enfants qui font des efforts intentionnels sophistiqués pour contourner les restrictions. Nous pouvons vous aider à vous protéger contre un accès intentionnel non intentionnel ou non sophistiqué. Il est important de ne pas définir des attentes pour les parents et les gardiens trop hauts dans les solutions de contrôle parental.
