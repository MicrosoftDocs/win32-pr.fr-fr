# <a name="parental-controls-minimum-compliance-api"></a>API de conformité minimale des contrôles parentaux

L’API de conformité minimale implémente des méthodes d’interface simples pour les applications afin d’obtenir des informations d’état de base sur un compte d’utilisateur contrôlé de manière spécifique, ses paramètres de restriction Web et les paramètres de restrictions de jeux.

-   [**IWindowsParentalControls**](/windows/desktop/api/Wpcapi/nn-wpcapi-iwindowsparentalcontrols)
-   [**IWPCGamesSettings**](/windows/desktop/api/Wpcapi/nn-wpcapi-iwpcgamessettings)
-   [**IWPCSettings**](/windows/desktop/api/Wpcapi/nn-wpcapi-iwpcsettings)
-   [**IWPCWebSettings**](/windows/desktop/api/Wpcapi/nn-wpcapi-iwpcwebsettings)
