# <a name="scheduling"></a>Planification

Le planificateur système contrôle le multitâche en déterminant lequel des threads concurrents reçoit la tranche de temps processeur suivante. Le planificateur détermine le thread qui s’exécute ensuite à l’aide des priorités de planification.
