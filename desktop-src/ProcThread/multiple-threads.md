# <a name="multiple-threads"></a>Threads multiples

Un *thread* est l’entité dans un processus qui peut être planifiée pour l’exécution. Tous les threads d’un processus partagent l’espace d’adressage virtuel et les ressources système. Chaque processus est démarré avec un thread unique, mais peut créer des threads supplémentaires à partir de l’un de ses threads.
