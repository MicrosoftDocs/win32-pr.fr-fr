# <a name="image-lists"></a>Listes d’images

Cette section contient des informations sur les éléments de programmation utilisés avec les listes d’images.

### <a name="overviews"></a>Vues d'ensemble



| Rubrique                          | Contenu                                                                                                            |
|--------------------------------|---------------------------------------------------------------------------------------------------------------------|
| [Listes d’images](image-lists.md) | Une liste d’images est une collection d’images de même taille, dont chacune peut être référencée par son index.<br/> |

| [**\_SetImageCount ImageList**](/windows/desktop/api/Commctrl/nf-commctrl-imagelist_setimagecount)           | Redimensionne une liste d’images existante. <br/>                                                                                                                                                                                                                                          |
| [**\_SetOverlayImage ImageList**](/windows/desktop/api/Commctrl/nf-commctrl-imagelist_setoverlayimage)       | Ajoute une image spécifiée à la liste d’images à utiliser comme masques de superposition. Une liste d’images peut comporter jusqu’à quatre masques de superposition dans la version 4,70 et les versions antérieures, et jusqu’à 15 dans la version 4,71. | [**\_Écriture ImageList**](/windows/desktop/api/Commctrl/nf-commctrl-imagelist_write)                           | Écrit une liste d’images dans un flux. <br/>                                                                                                                                                                                                                                        |
| [**\_WriteEx ImageList**](/windows/desktop/api/Commctrl/nf-commctrl-imagelist_writeex)                       | Écrit une liste d’images dans un flux. <br/>                                                                                                                                                                                                                                        |
