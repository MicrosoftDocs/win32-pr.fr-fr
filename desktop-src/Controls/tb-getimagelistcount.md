# <a name="tb_getimagelistcount-message"></a>TO \_ GETIMAGELISTCOUNT message

Obtient le nombre de listes d’images associées à la barre d’outils.

<dl> <dt>

*wParam* 
</dt> <dd>Doit être zéro.</dd> <dt>

*lParam* 
</dt> <dd>Doit être zéro.</dd> </dl>

## <a name="return-value"></a>Valeur retournée

Retourne le nombre de listes d’images.
