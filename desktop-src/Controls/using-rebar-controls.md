# <a name="using-rebar-controls"></a>Utilisation des contrôles Rebar

Cette section contient des rubriques qui montrent comment créer et utiliser des contrôles Rebar.

## <a name="in-this-section"></a>Contenu de cette section



| Rubrique                                                                | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
|----------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Comment créer des contrôles Rebar](create-rebar-controls.md)<br/> | Une application crée un contrôle rebar en appelant la fonction [**CreateWindowEx**](/windows/desktop/api/winuser/nf-winuser-createwindowexa) , en spécifiant [**REBARCLASSNAME**](common-control-window-classes.md) comme classe de fenêtre. L’application doit tout d’abord inscrire la classe de fenêtre en appelant la fonction [**InitCommonControlsEx**](/windows/desktop/api/Commctrl/nf-commctrl-initcommoncontrolsex) , en spécifiant le \_ bit des classes cool ICC \_ dans la structure [**InitCommonControlsEx**](/windows/win32/api/commctrl/ns-commctrl-initcommoncontrolsex) qui l’accompagne. <br/> |
