# <a name="cbem_getimagelist-message"></a>\_Message CBEM GETIMAGELIST

Obtient le handle d’une liste d’images assignée à un contrôle ComboBoxEx.

## <a name="parameters"></a>Paramètres

<dl> <dt>

*wParam* 
</dt> <dd>Doit être zéro.</dd> <dt>

*lParam* 
</dt> <dd>Doit être zéro.</dd> </dl>

## <a name="return-value"></a>Valeur retournée

Retourne le handle de la liste d’images assignée au contrôle en cas de réussite, ou **null** dans le cas contraire.
