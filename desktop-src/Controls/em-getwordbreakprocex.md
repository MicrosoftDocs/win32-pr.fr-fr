# <a name="em_getwordbreakprocex-message"></a>\_Message GETWORDBREAKPROCEX em

Récupère l’adresse de la procédure d’arrêt de mot étendu actuellement inscrite pour un contrôle RichEdit.

## <a name="parameters"></a>Paramètres

<dl> <dt>

*wParam* 
</dt> <dd>

Non utilisé ; doit être égal à zéro.

</dd> <dt>

*lParam* 
</dt> <dd>

Non utilisé ; doit être égal à zéro.

</dd> </dl>

## <a name="return-value"></a>Valeur de retour

Le message retourne l’adresse de la procédure en cours.
