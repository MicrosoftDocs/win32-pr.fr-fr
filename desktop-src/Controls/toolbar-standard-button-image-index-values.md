## <a name="remarks"></a>Remarques

Vous utilisez ces valeurs pour spécifier un index d’image dans une liste d’images standard qui a été chargée avec le message [**to \_ LOADIMAGES**](tb-loadimages.md) . Les valeurs d’index correspondent aux images dans les bitmaps standard utilisées par les contrôles communs.

Chacun des en-têtes suivants contient une liste d’images des grandes icônes, une liste d’images des petites icônes et une table qui correspond à la constante de chaque icône à sa valeur d’index.

![Capture d’écran montrant les petites icônes historiques.](images/tb-stdsmallhistimages.png)
