# <a name="ttm_activate-message"></a>ATTÉNUATION \_ activer le message

Active ou désactive un contrôle ToolTip.

## <a name="parameters"></a>Paramètres

<dl> <dt>

*wParam* 
</dt> <dd>

Indicateur d’activation. Si ce paramètre a la **valeur true**, le contrôle ToolTip est activé. Si la **valeur est false**, le contrôle ToolTip est désactivé.

</dd> <dt>

*lParam* 
</dt> <dd>Doit être zéro.</dd> </dl>

## <a name="return-value"></a>Valeur de retour

Pas de valeur de retour.
