# <a name="rb_setbandwidth-message"></a>\_Message SETBANDWIDTH RB

Définit la largeur d’une bande ancrée.

## <a name="parameters"></a>Paramètres

<dl> <dt>

*wParam* 
</dt> <dd>

Index de la bande.

</dd> <dt>

*lParam* 
</dt> <dd>

Nouvelle largeur en pixels.

</dd> </dl>

## <a name="return-value"></a>Valeur de retour

Retourne la valeur **true** si la valeur a été définie et **false** dans le cas contraire.
