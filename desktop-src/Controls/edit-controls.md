# <a name="edit-control-windows-controls"></a>Contrôle Edit (contrôles Windows)

Cette section contient des informations sur les éléments de programmation utilisés avec les contrôles d’édition. Un *contrôle d’édition* est une fenêtre de contrôle rectangulaire généralement utilisée dans une boîte de dialogue pour permettre à l’utilisateur d’entrer et de modifier du texte en le tapant sur le clavier.

### <a name="overviews"></a>Vues d'ensemble



| Rubrique                                                             | Contenu                                                                                                                                                                                                                                                                                                                                      |
|-------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [À propos des contrôles d’édition](about-edit-controls.md)                    | Un *contrôle d’édition* est une fenêtre de contrôle rectangulaire généralement utilisée dans une boîte de dialogue pour permettre à l’utilisateur d’entrer et de modifier du texte.<br/>                                                                                                                                                                                                        |
| [Modifier les opérations de texte de contrôle](edit-controls-text-operations.md) | Le système traite automatiquement toutes les opérations de texte initiées par l’utilisateur et notifie l’application lorsque les opérations sont terminées.<br/>                                                                                                                                                                                              |
| [Utilisation des contrôles d’édition](using-edit-controls.md)                    | Les contrôles d’édition sont généralement utilisés dans les boîtes de dialogue, mais vous pouvez également les utiliser dans la zone cliente d’une fenêtre standard. Les contrôles d’édition sur une seule ligne sont utiles pour récupérer une chaîne unique à partir de l’utilisateur. Les contrôles d’édition multiligne permettent à votre application d’implémenter facilement la plupart des fonctionnalités d’un traitement de texte simple.<br/> |

|----------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [*EditWordBreakProc*](/windows/win32/api/winuser/nc-winuser-editwordbreakproca) | Fonction de rappel définie par l’application utilisée avec le message [**\_ SETWORDBREAKPROC em**](em-setwordbreakproc.md) . Un contrôle d’édition multiligne ou un contrôle RichEdit appelle une fonction [*EditWordBreakProc*](/windows/win32/api/winuser/nc-winuser-editwordbreakproca) pour scinder une ligne de texte.<br/> Le type **EDITWORDBREAKPROC** définit un pointeur vers cette fonction de rappel. [*EditWordBreakProc*](/windows/win32/api/winuser/nc-winuser-editwordbreakproca) est un espace réservé pour le nom de la fonction définie par l’application. <br/> |
