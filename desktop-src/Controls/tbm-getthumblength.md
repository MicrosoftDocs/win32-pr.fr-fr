# <a name="tbm_getthumblength-message"></a>\_Message TBM GETTHUMBLENGTH

Récupère la longueur du curseur dans un TrackBar.

<dl> <dt>

*wParam* 
</dt> <dd>Doit être zéro.</dd> <dt>

*lParam* 
</dt> <dd>Doit être zéro.</dd> </dl>
