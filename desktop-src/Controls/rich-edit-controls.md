# <a name="rich-edit"></a>Édition enrichie

Cette section contient des informations sur les éléments de programmation utilisés avec les contrôles RichEdit. Un contrôle RichEdit permet à l’utilisateur d’entrer, de modifier, d’imprimer et d’enregistrer du texte. Le texte peut être affecté à la mise en forme des caractères et des paragraphes, et peut inclure des objets COM (Component Object Model) incorporés.

Étant donné que les contrôles RichEdit prennent en charge presque tous les messages et codes de notification utilisés avec les [contrôles d’édition](edit-controls.md)multilignes, les applications qui utilisent déjà les contrôles d’édition peuvent facilement être modifiées pour utiliser des contrôles RichEdit.
