# <a name="tb_deletebutton-message"></a>TO \_ DELETEBUTTON message

Supprime un bouton de la barre d’outils.

<dl> <dt>

*wParam* 
</dt> <dd>

Index de base zéro du bouton à supprimer.

</dd> <dt>

*lParam* 
</dt> <dd>Doit être zéro.</dd> </dl>
