# <a name="syslink"></a>SysLink

Cette section contient des informations sur les éléments de programmation utilisés avec les contrôles SysLink.

### <a name="overviews"></a>Vues d'ensemble



| Rubrique                                    | Contenu                                                                                       |
|------------------------------------------|------------------------------------------------------------------------------------------------|
| [Contrôle SysLink](syslink-overview.md) | Le contrôle SysLink offre un moyen pratique d’incorporer des liens hypertexte dans une fenêtre.<br/> |



 

### <a name="messages"></a>Messages



| Rubrique                                           | Contenu                                                                             |
|-------------------------------------------------|--------------------------------------------------------------------------------------|
| [**\_GETIDEALHEIGHT LM**](lm-getidealheight.md) | Récupère la hauteur par défaut d’un lien pour la largeur actuelle du contrôle.<br/> |
| [**\_GETIDEALSIZE LM**](lm-getidealsize.md)     | Récupère la hauteur par défaut d’un lien pour la largeur actuelle du contrôle.<br/> |
| [**LM \_ GETITEM**](lm-getitem.md)               | Récupère les États et les attributs d’un élément.<br/>                           |
| [**LANMANAGER \_ HITTEST**](lm-hittest.md)               | Détermine si l’utilisateur a cliqué sur le lien spécifié.<br/>                   |
| [**\_SETITEM LM**](lm-setitem.md)               | Définit les États et les attributs d’un élément.<br/>                                |



 

### <a name="structures"></a>Structures



| Rubrique                                | Contenu                                                                                                                                                                           |
|--------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**LHITTESTINFO**](/windows/win32/api/commctrl/ns-commctrl-lhittestinfo) | Utilisé pour obtenir des informations sur le lien correspondant à un emplacement donné. <br/>                                                                                              |
| [**LITEM**](/windows/win32/api/commctrl/ns-commctrl-litem)               | Utilisé pour définir et récupérer des informations sur un élément de lien.<br/>                                                                                                                 |
| [**NMLINK**](/windows/win32/api/commctrl/ns-commctrl-nmlink)             | [**NMLINK**](/windows/win32/api/commctrl/ns-commctrl-nmlink) contient des informations de notification. Envoie cette structure avec les messages de [ \_ retour](nm-return.md) par [ \_ un clic ou un NM NM](nm-click-syslink.md) .<br/> |
