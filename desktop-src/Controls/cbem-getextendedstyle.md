# <a name="cbem_getextendedstyle-message"></a>\_Message CBEM GETEXTENDEDSTYLE

Obtient les styles étendus en cours d’utilisation pour un contrôle ComboBoxEx.

## <a name="parameters"></a>Paramètres

<dl> <dt>

*wParam* 
</dt> <dd>Doit être zéro.</dd> <dt>

*lParam* 
</dt> <dd>Doit être zéro.</dd> </dl>

## <a name="return-value"></a>Valeur de retour

Retourne une valeur DWORD qui contient les [styles étendus de contrôle ComboBoxEx](comboboxex-control-extended-styles.md) en cours d’utilisation pour le contrôle.
