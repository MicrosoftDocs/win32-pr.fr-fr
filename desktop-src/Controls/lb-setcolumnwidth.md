# <a name="lb_setcolumnwidth-message"></a>\_Message SETCOLUMNWIDTH lb

Définit la largeur, en pixels, de toutes les colonnes dans une zone de liste à plusieurs colonnes.

## <a name="parameters"></a>Paramètres

<dl> <dt>

*wParam* 
</dt> <dd>

Spécifie la largeur, en pixels, de toutes les colonnes.

</dd> <dt>

*lParam* 
</dt> <dd>

Ce paramètre n'est pas utilisé.

</dd> </dl>

## <a name="return-value"></a>Valeur retournée

Ce message ne retourne pas de valeur.
