# <a name="em_isime-message"></a>\_Message ISIME em

Déterminer avec les paramètres régionaux d’entrée actuels d’un contrôle RichEdit est un paramètre régional d’Extrême-Orient.

## <a name="parameters"></a>Paramètres

<dl> <dt>

*wParam* 
</dt> <dd>

Non utilisé ; doit être égal à zéro.

</dd> <dt>

*lParam* 
</dt> <dd>

Non utilisé ; doit être égal à zéro.

</dd> </dl>

## <a name="return-value"></a>Valeur retournée

Retourne **true** s’il s’agit d’un paramètre régional d’Extrême-Orient. Sinon, elle retourne **false**.
