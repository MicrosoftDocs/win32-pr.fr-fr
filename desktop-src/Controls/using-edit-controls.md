# <a name="using-edit-controls"></a>Utilisation des contrôles d’édition

Cette section fournit des informations et un exemple de code pour implémenter les contrôles d’édition.

Les contrôles d’édition sont généralement utilisés dans les boîtes de dialogue, mais vous pouvez également les utiliser dans la zone cliente d’une fenêtre standard. Les contrôles d’édition sur une seule ligne sont utiles pour récupérer une chaîne unique à partir de l’utilisateur. Les contrôles d’édition multiligne vous permettent d’implémenter la plupart des fonctionnalités d’un traitement de texte simple dans votre application.

Les rubriques suivantes sont présentées.
