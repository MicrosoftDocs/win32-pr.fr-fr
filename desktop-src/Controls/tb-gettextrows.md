# <a name="tb_gettextrows-message"></a>TO \_ GETTEXTROWS message

Récupère le nombre maximal de lignes de texte qui peuvent être affichées sur un bouton de barre d’outils.

## <a name="parameters"></a>Paramètres

<dl> <dt>

*wParam* 
</dt> <dd>Doit être zéro.</dd> <dt>

*lParam* 
</dt> <dd>Doit être zéro.</dd> </dl>

## <a name="return-value"></a>Valeur de retour

Retourne une valeur INT représentant le nombre maximal de lignes de texte que le contrôle affichera pour un bouton.
