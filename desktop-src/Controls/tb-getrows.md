# <a name="tb_getrows-message"></a>TO, \_ message GETROWS

Récupère le nombre de lignes de boutons dans une barre d’outils avec le style [**\_ encapsulé TBSTYLE**](toolbar-control-and-button-styles.md) .

<dl> <dt>

</dt> <dd>Doit être zéro.</dd> <dt>
