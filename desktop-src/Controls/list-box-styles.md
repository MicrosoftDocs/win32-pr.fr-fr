# <a name="list-box-styles"></a>Styles de zone de liste

Pour créer une zone de liste à l’aide de la fonction [**CreateWindow**](/windows/desktop/api/winuser/nf-winuser-createwindowa) ou [**CreateWindowEx**](/windows/desktop/api/winuser/nf-winuser-createwindowexa) , utilisez la classe ListBox, les constantes de style de fenêtre appropriées et les constantes de style suivantes pour définir la zone de liste. Une fois le contrôle créé, ces styles ne peuvent pas être modifiés, sauf indication contraire.



| Constante                                                                                                                                                                              | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="LBS_COMBOBOX"></span><span id="lbs_combobox"></span><dl> <dt>**la \_ zone de liste déroulante**</dt> </dl>                            | Notifie une zone de liste qu’elle fait partie d’une zone de liste déroulante. Cela permet de coordonner les deux contrôles afin qu’ils présentent une interface utilisateur unifiée. La zone de liste déroulante elle-même doit définir ce style. Si le style est défini par tout sauf la zone de liste déroulante, la zone de liste se considère comme étant incorrecte comme un enfant d’une zone de liste modifiable et un échec se produit.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| <span id="LBS_DISABLENOSCROLL"></span><span id="lbs_disablenoscroll"></span><dl> <dt>**LBS \_ DISABLENOSCROLL**</dt> </dl>       | Affiche une barre de défilement horizontale ou verticale désactivée lorsque la zone de liste ne contient pas assez d’éléments pour faire défiler. Si vous ne spécifiez pas ce style, la barre de défilement est masquée lorsque la zone de liste ne contient pas assez d’éléments. Ce style doit être utilisé avec le style [**WS \_ VSCROLL**](/windows/desktop/winmsg/window-styles) ou [**WS \_ HSCROLL**](/windows/desktop/winmsg/window-styles) .<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| <span id="LBS_EXTENDEDSEL"></span><span id="lbs_extendedsel"></span><dl> <dt>**LBS \_ EXTENDEDSEL**</dt> </dl>                   | Autorise la sélection de plusieurs éléments à l’aide de la touche Maj et de la souris ou des combinaisons de touches spéciales.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| <span id="LBS_HASSTRINGS"></span><span id="lbs_hasstrings"></span><dl> <dt>**LBS \_ HASSTRINGS**</dt> </dl>                      | Spécifie qu’une zone de liste contient des éléments composés de chaînes. La zone de liste conserve la mémoire et les adresses des chaînes afin que l’application puisse utiliser le message de plan [**\_ GETTEXT**](lb-gettext.md) pour récupérer le texte d’un élément particulier. Par défaut, toutes les zones de liste, à l’exception des zones de liste owner-drawn, ont ce style. Vous pouvez créer une zone de liste owner-drawn avec ou sans ce style.<br/> Pour les zones de liste owner-drawn sans ce style, le message d’attente [**\_ GETTEXT**](lb-gettext.md) récupère la valeur associée à un élément (les données de l’élément). <br/>                                                                                                                                                                                                                                                                                                                                    |
| <span id="LBS_MULTICOLUMN"></span><span id="lbs_multicolumn"></span><dl> <dt>**LB \_ MULTIcolonne**</dt> </dl>                   | Spécifie une zone de liste à plusieurs colonnes qui défile horizontalement. La zone de liste calcule automatiquement la largeur des colonnes, ou une application peut définir la largeur à l’aide du message [**\_ SETCOLUMNWIDTH lb**](lb-setcolumnwidth.md) . Si une zone de liste présente le style **\_ OWNERDRAWFIXED** , une application peut définir la largeur lorsque la zone de liste envoie le message [**WM \_ MEASUREITEM**](wm-measureitem.md) . <br/> Une zone de liste avec le style **\_ multicolonne lbs** ne peut pas défiler verticalement, elle ignore tous les messages [**WM \_ VSCROLL**](wm-vscroll.md) qu’elle reçoit. <br/> Les **styles \_ multicolonnes** et **lbs \_ OWNERDRAWVARIABLE** ne peuvent pas être combinés. Si les deux sont spécifiés, **lbs \_ OWNERDRAWVARIABLE** est ignoré.<br/>                                                                                                                                                                                |
| <span id="LBS_MULTIPLESEL"></span><span id="lbs_multiplesel"></span><dl> <dt>**LBS \_ MULTIPLESEL**</dt> </dl>                   | Active ou désactive la sélection de chaîne chaque fois que l’utilisateur clique ou double-clique sur une chaîne dans la zone de liste. L’utilisateur peut sélectionner un nombre quelconque de chaînes.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| <span id="LBS_NODATA"></span><span id="lbs_nodata"></span><dl> <dt>**LB \_ NOdata**</dt> </dl>                                  | Spécifie une zone de liste sans données. Spécifiez ce style lorsque le nombre d'éléments dans la zone de liste dépasse 1000. Une zone de liste sans données doit également avoir le **style \_ OWNERDRAWFIXED lbs** , mais elle ne doit pas avoir le style **de \_ HASSTRINGS** de **\_ Tri** ou de lbs. <br/> Une zone de liste sans données s'apparente à une zone de liste personnalisée par son propriétaire sauf qu'elle ne contient aucune chaîne ou donnée bitmap d'un élément. Les commandes permettant d’ajouter, d’insérer ou de supprimer un élément ignorent toujours les données d’élément spécifiées ; les requêtes de recherche d’une chaîne dans la zone de liste échouent toujours. Le système envoie le message [**WM \_ DRAWITEM**](wm-drawitem.md) à la fenêtre propriétaire lorsqu’un élément doit être dessiné. Le membre **ItemId** de la structure [**drawitemstruct,**](/windows/win32/api/winuser/ns-winuser-drawitemstruct) transmise avec le message **WM \_ DRAWITEM** spécifie le numéro de ligne de l’élément à dessiner. Une zone de liste sans données n’envoie pas de message [**WM \_ DELETEITEM**](wm-deleteitem.md) .<br/> |
| <span id="LBS_NOINTEGRALHEIGHT"></span><span id="lbs_nointegralheight"></span><dl> <dt>**LBS \_ NOINTEGRALHEIGHT**</dt> </dl>    | Spécifie que la taille de la zone de liste correspond exactement à la taille spécifiée par l’application lors de la création de la zone de liste. Normalement, le système redimensionne une zone de liste afin que la zone de liste n’affiche pas d’éléments partiels.<br/> Pour les zones de liste avec le style **\_ OWNERDRAWVARIABLE lbs** , le style **\_ NOINTEGRALHEIGHT** est toujours appliqué.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| <span id="LBS_NOREDRAW"></span><span id="lbs_noredraw"></span><dl> <dt>**LBS \_ NOREDRAW**</dt> </dl>                            | Spécifie que l’apparence de la zone de liste n’est pas mise à jour lorsque des modifications sont apportées.<br/> Pour modifier l’état de redessin du contrôle, utilisez le message [**WM \_ SETREDRAW**](/windows/desktop/gdi/wm-setredraw) .<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| <span id="LBS_NOSEL"></span><span id="lbs_nosel"></span><dl> <dt>**LBS- \_ nez**</dt> </dl>                                     | Spécifie que la zone de liste contient des éléments qui peuvent être affichés mais non sélectionnés. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| <span id="LBS_NOTIFY"></span><span id="lbs_notify"></span><dl> <dt>**LBS \_ notifier**</dt> </dl>                                  | Fait en sorte que la zone de liste envoie un code de notification à la fenêtre parente chaque fois que l’utilisateur clique sur un élément de zone de liste ([LBN \_ selChange](lbn-selchange.md)), qu’il double-clique sur un élément ([LBN \_ DBLCLK](lbn-dblclk.md)) ou qu’il annule la sélection ([LBN \_ SELCANCEL](lbn-selcancel.md)). <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| <span id="LBS_OWNERDRAWFIXED"></span><span id="lbs_ownerdrawfixed"></span><dl> <dt>**LBS \_ OWNERDRAWFIXED**</dt> </dl>          | Spécifie que le propriétaire de la zone de liste est chargé de dessiner son contenu et que les éléments de la zone de liste sont de la même hauteur. La fenêtre propriétaire reçoit un message [**WM \_ MEASUREITEM**](wm-measureitem.md) lorsque la zone de liste est créée et un message [**WM \_ DRAWITEM**](wm-drawitem.md) lorsqu’un aspect visuel de la zone de liste a changé.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| <span id="LBS_OWNERDRAWVARIABLE"></span><span id="lbs_ownerdrawvariable"></span><dl> <dt>**LBS \_ OWNERDRAWVARIABLE**</dt> </dl> | Spécifie que le propriétaire de la zone de liste est chargé de dessiner son contenu et que les éléments de la zone de liste sont des variables de la hauteur. La fenêtre propriétaire reçoit un message [**WM \_ MEASUREITEM**](wm-measureitem.md) pour chaque élément de la zone lorsque la zone de liste est créée et un message [**WM \_ DRAWITEM**](wm-drawitem.md) lorsqu’un aspect visuel de la zone de liste a changé.<br/> Ce style provoque l’activation du style **\_ NOINTEGRALHEIGHT lbs** .<br/> Ce style est ignoré si le **style \_ multicolonne** est spécifié. <br/>                                                                                                                                                                                                                                                                                                                                                                  |
| <span id="LBS_SORT"></span><span id="lbs_sort"></span><dl> <dt>**LIVRES de \_ Tri**</dt> </dl>                                        | Trie les chaînes dans la zone de liste par ordre alphabétique.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| <span id="LBS_STANDARD"></span><span id="lbs_standard"></span><dl> <dt>**LBS \_ standard**</dt> </dl>                            | Trie les chaînes dans la zone de liste par ordre alphabétique. La fenêtre parente reçoit un code de notification chaque fois que l’utilisateur clique sur un élément de zone de liste, qu’il double-clique sur un élément ou qu’il annule la sélection. La zone de liste comporte une barre de défilement verticale et contient des bordures sur tous les côtés. Ce style combine les styles de [**\_ bordure**](/windows/desktop/winmsg/window-styles) **lb \_ Notify**, **lb \_ sort**, [**WS \_ VSCROLL**](/windows/desktop/winmsg/window-styles)et WS.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| <span id="LBS_USETABSTOPS"></span><span id="lbs_usetabstops"></span><dl> <dt>**LBS \_ USETABSTOPS**</dt> </dl>                   | Permet à une zone de liste de reconnaître et de développer des caractères de tabulation lors du dessin de ses chaînes. Vous pouvez utiliser le [**message \_ SETTABSTOPS**](lb-settabstops.md) pour spécifier les positions des taquets de tabulation. Les positions par défaut des tabulations sont les unités de modèle de boîte de dialogue 32. Les unités de modèle de boîte de dialogue sont les unités indépendantes du périphérique utilisées dans les modèles de boîte de dialogue. Pour convertir des mesures à partir d’unités de modèle de boîte de dialogue en unités d’écran (pixels), utilisez la fonction [**MapDialogRect**](/windows/desktop/api/winuser/nf-winuser-mapdialogrect) .<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| <span id="LBS_WANTKEYBOARDINPUT"></span><span id="lbs_wantkeyboardinput"></span><dl> <dt>**LBS \_ WANTKEYBOARDINPUT**</dt> </dl> | Spécifie que le propriétaire de la zone de liste reçoit les messages [**WM \_ VKEYTOITEM**](wm-vkeytoitem.md) chaque fois que l’utilisateur appuie sur une touche et que la zone de liste a le focus d’entrée. Cela permet à une application d’effectuer un traitement spécial sur l’entrée au clavier.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
