# <a name="tree-view"></a>Arborescence

Cette section contient des informations sur les éléments de programmation utilisés avec les contrôles d’arborescence.

### <a name="overviews"></a>Vues d'ensemble



| Rubrique                                              | Contenu                                                                                                                                                                                    |
|----------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [À propos des contrôles Tree-View](tree-view-controls.md) | Un contrôle Tree-View est une fenêtre qui affiche une liste hiérarchique d’éléments, tels que les en-têtes d’un document, les entrées d’un index ou les fichiers et répertoires d’un disque.<br/> |
| [Utilisation de contrôles Tree-View](using-treeview.md)     | Cette section contient des détails d’implémentation et un exemple de code pour l’utilisation des contrôles d’arborescence.<br/>                                                                               |
