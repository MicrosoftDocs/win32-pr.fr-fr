# <a name="udm_setbuddy-message"></a>\_Message SETBUDDY UDM

Définit la fenêtre associée pour un contrôle up-up.

## <a name="parameters"></a>Paramètres

<dl> <dt>

*wParam* 
</dt> <dd>

Handle vers la nouvelle fenêtre associée.

</dd> <dt>

*lParam* 
</dt> <dd>Doit être zéro.</dd> </dl>

## <a name="return-value"></a>Valeur retournée

La valeur de retour est le handle de la fenêtre associée précédente.
