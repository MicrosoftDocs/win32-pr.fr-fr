# <a name="using-windowless-rich-edit-controls"></a>Utilisation de contrôles RichEdit sans fenêtre

Cette section fournit un exemple de code et des informations sur l’activation de la fonctionnalité de fenêtre dans un contrôle RichEdit sans fenêtre.

## <a name="in-this-section"></a>Contenu de cette section



| Rubrique                                                                                              | Description                                                                                                                                                                                                                                                                                             |
|----------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Comment créer des contrôles RichEdit sans fenêtre](create-windowless-rich-edit-controls.md)<br/> | La fonctionnalité de fenêtre offre la possibilité de recevoir des messages et la propriété d’un contexte de périphérique dans lequel dessiner.
