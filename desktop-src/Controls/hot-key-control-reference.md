# <a name="hot-key"></a>Touche d’accès rapide

Cette section contient des informations sur les éléments de programmation utilisés avec les contrôles de touche d’accès rapide.

### <a name="overviews"></a>Vues d'ensemble



| Rubrique                                    | Contenu                                                                                                                     |
|------------------------------------------|------------------------------------------------------------------------------------------------------------------------------|
| [Contrôles de touche d’accès rapide](hot-key-controls.md) | Un contrôle de touche d’accès rapide est une fenêtre qui permet à l’utilisateur d’entrer une combinaison de touches à utiliser comme touche d’accès rapide.<br/> |



 

### <a name="messages"></a>Messages



| Rubrique                                   | Contenu                                                                                                 |
|-----------------------------------------|----------------------------------------------------------------------------------------------------------|
| [**HKM \_ GETHOTKEY**](hkm-gethotkey.md) | Obtient le code de touche virtuel et les indicateurs de modificateur d’une touche d’accès rapide à partir d’un contrôle de touche d’accès rapide.<br/>             |
| [**HKM \_ SETHOTKEY**](hkm-sethotkey.md) | Définit la combinaison de touches d’accès rapide pour un contrôle de touche d’accès rapide.<br/>                                           |
| [**HKM \_ SETRULES**](hkm-setrules.md)   | Définit les combinaisons non valides et la combinaison de modificateur par défaut pour un contrôle de touche d’accès rapide. <br/> |
