# <a name="pbm_getpos-message"></a>\_Message PBM GETPOS

Récupère la position actuelle de la barre de progression.

<dl> <dt>

*wParam* 
</dt> <dd>Doit être zéro.</dd> <dt>

*lParam* 
</dt> <dd>Doit être zéro.</dd> </dl>
