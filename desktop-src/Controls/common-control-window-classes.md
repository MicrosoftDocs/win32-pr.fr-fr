# <a name="window-classes-commctrlh"></a>Classes de fenêtre (CommCtrl. h)

Cette section répertorie les noms des classes de fenêtre fournies par la bibliothèque de contrôles communs.



| Constante                                                                                                                                                                     | Description                                                                                                                                                                                                                                                    |
|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="ANIMATE_CLASS"></span><span id="animate_class"></span><dl> <dt>**ANIMATe, \_ classe**</dt> </dl>                | Crée des contrôles d’animation. Ces contrôles affichent en mode silencieux un clip AVI (Audio Video entrelacé).<br/>                                                                                                                                                  |
| <span id="DATETIMEPICK_CLASS"></span><span id="datetimepick_class"></span><dl> <dt>**DATETIMEPICK, \_ classe**</dt> </dl> | Crée des contrôles de sélecteur de date et d’heure. Ces contrôles fournissent une interface simple pour échanger des informations de date et d’heure avec un utilisateur. <br/>                                                                                                                |
| <span id="HOTKEY_CLASS"></span><span id="hotkey_class"></span><dl> <dt>**HOTKEY ( \_ classe)**</dt> </dl>                   | Crée des contrôles de touche d’accès rapide. Ces contrôles permettent à l’utilisateur de définir facilement des touches d’accès rapide. <br/>                                                                                                                                                             |
| <span id="LINK_CLASS"></span><span id="link_class"></span><dl> <dt>**LINK ( \_ classe)**</dt> </dl>                         | Crée des contrôles Syslink. Ces contrôles sont utilisés avec les liens. <br/>                                                                                                                                                                                      |
| <span id="MONTHCAL_CLASS"></span><span id="monthcal_class"></span><dl> <dt>**CALENDRIER MONTHCAL, \_ classe**</dt> </dl>             | Crée des contrôles de calendrier mensuel. Ces contrôles permettent à un utilisateur de sélectionner une date à partir d’une interface familière. <br/>                                                                                                                        |
| <span id="NATIVEFNTCTL_CLASS"></span><span id="nativefntctl_class"></span><dl> <dt>**NATIVEFNTCTL, \_ classe**</dt> </dl> | Crée des contrôles de police natifs. Ces contrôles sont utilisés avec les polices natives. <br/>                                                                                                                                                                           |
| <span id="PROGRESS_CLASS"></span><span id="progress_class"></span><dl> <dt>**PROGRESS ( \_ classe)**</dt> </dl>             | Crée des barres de progression. Ces contrôles indiquent la progression d’une longue opération.<br/>                                                                                                                                                                 |
| <span id="REBARCLASSNAME"></span><span id="rebarclassname"></span><dl> <dt>**REBARCLASSNAME**</dt> </dl>              | Crée des contrôles Rebar. Ces contrôles jouent le rôle de conteneur pour les fenêtres enfants. <br/>                                                                                                                                                                       |
| <span id="STANDARD_CLASSES"></span><span id="standard_classes"></span><dl> <dt>**\_classes standard**</dt> </dl>       | Crée des contrôles User32 intrinsèques, tels qu’un bouton, une modification et une barre de défilement.<br/>                                                                                                                                                                            |
| <span id="STATUSCLASSNAME"></span><span id="statusclassname"></span><dl> <dt>**STATUSCLASSNAME**</dt> </dl>           | Crée des fenêtres d’État. Ces contrôles affichent les informations d’État dans une fenêtre horizontale.<br/>                                                                                                                                                           |
| <span id="TOOLBARCLASSNAME"></span><span id="toolbarclassname"></span><dl> <dt>**TOOLBARCLASSNAME**</dt> </dl>        | Crée des barres d’outils. Ces contrôles contiennent des boutons qui exécutent des commandes de menu. <br/>                                                                                                                                                                     |
| <span id="TOOLTIPS_CLASS"></span><span id="tooltips_class"></span><dl> <dt>**INFO-bulles ( \_ classe)**</dt> </dl>             | Crée des contrôles ToolTip. Ces contrôles affichent une petite fenêtre contextuelle contenant une ligne de texte qui décrit l’objectif d’un outil dans une application.<br/>                                                                                            |
| <span id="TRACKBAR_CLASS"></span><span id="trackbar_class"></span><dl> <dt>**TRACKBAR ( \_ classe)**</dt> </dl>             | Crée trackbars. Ces contrôles permettent à l’utilisateur de sélectionner une plage de valeurs en déplaçant un curseur. <br/>                                                                                                                                                   |
| <span id="UPDOWN_CLASS"></span><span id="updown_class"></span><dl> <dt>**classe UpDown \_**</dt> </dl>                   | Crée des contrôles Up-Up. Ces contrôles combinent une paire de flèches avec un contrôle d’édition. Un clic sur les flèches incrémente ou décrémente la valeur dans le contrôle d’édition. <br/>                                                                                |
| <span id="WC_BUTTON"></span><span id="wc_button"></span><dl> <dt>**bouton de la WC \_**</dt> </dl>                            | Crée des contrôles de bouton. Ces contrôles permettent à un utilisateur d’initier une action.<br/>                                                                                                                                                                        |
| <span id="WC_COMBOBOX"></span><span id="wc_combobox"></span><dl> <dt>**\_zone de liste déroulante WC**</dt> </dl>                      | Crée des contrôles de zone de liste déroulante. Ces contrôles contiennent une liste déroulante à partir de laquelle un seul élément peut être sélectionné. <br/>                                                                                                                                       |
| <span id="WC_COMBOBOXEX"></span><span id="wc_comboboxex"></span><dl> <dt>**\_COMBOBOXEX WC**</dt> </dl>                | Crée des contrôles ComboBoxEx. Ces contrôles fournissent une extension du contrôle de zone de liste déroulante qui fournit la prise en charge native pour les images d’élément. <br/>                                                                                                            |
| <span id="WC_EDIT"></span><span id="wc_edit"></span><dl> <dt>**modification de WC \_**</dt> </dl>                                  | Crée des contrôles d’édition. Ces contrôles contiennent du texte modifiable. <br/>                                                                                                                                                                                       |
| <span id="WC_HEADER"></span><span id="wc_header"></span><dl> <dt>**\_en-tête WC**</dt> </dl>                            | Crée des contrôles d’en-tête. Ces contrôles affichent les en-têtes en haut des colonnes d’informations et permettent à l’utilisateur de trier les informations en cliquant sur les en-têtes. <br/>                                                                                      |
| <span id="WC_LISTBOX"></span><span id="wc_listbox"></span><dl> <dt>**zone de liste de WC \_**</dt> </dl>                         | Crée des contrôles ListBox. Ces contrôles contiennent des listes d’éléments. <br/>                                                                                                                                                                                   |
| <span id="WC_IPADDRESS"></span><span id="wc_ipaddress"></span><dl> <dt>**\_adresse IP de WC**</dt> </dl>                   | Crée des contrôles d’adresse IP. Ces contrôles sont similaires à un contrôle d’édition, mais ils vous permettent d’entrer une adresse numérique au format IP (Internet Protocol). <br/>                                                                                       |
| <span id="WC_LINK"></span><span id="wc_link"></span><dl> <dt>**\_lien WC**</dt> </dl>                                  | Crée des contrôles SysLink. Ces contrôles contiennent des liens hypertexte.<br/>                                                                                                                                                                                   |
| <span id="WC_LISTVIEW"></span><span id="wc_listview"></span><dl> <dt>**ListView de WC \_**</dt> </dl>                      | Crée des contrôles d’affichage de liste. Ces contrôles affichent une collection d’éléments, chacun composé d’une icône et d’une étiquette, et offrent plusieurs moyens de réorganiser les éléments.<br/>                                                                                    |
| <span id="WC_NATIVEFONTCTL"></span><span id="wc_nativefontctl"></span><dl> <dt>**\_NATIVEFONTCTL WC**</dt> </dl>       | Crée des contrôles de police natifs. Il s’agit de contrôles invisibles qui fonctionnent en arrière-plan pour permettre aux contrôles prédéfinis d’une boîte de dialogue d’afficher la langue actuelle du système.<br/>                                                                         |
| <span id="WC_PAGESCROLLER"></span><span id="wc_pagescroller"></span><dl> <dt>**\_PAGESCROLLER WC**</dt> </dl>          | Crée des contrôles de pagineur. Ces contrôles sont utilisés pour contenir et faire défiler une autre fenêtre. <br/>                                                                                                                                                              |
| <span id="WC_SCROLLBAR"></span><span id="wc_scrollbar"></span><dl> <dt>**\_barre de défilement WC**</dt> </dl>                   | Crée des contrôles de barre de défilement. Ces contrôles permettent à l’utilisateur de faire défiler le contenu d’une fenêtre.<br/>                                                                                                                                                     |
| <span id="WC_STATIC"></span><span id="wc_static"></span><dl> <dt>**WC \_ statique**</dt> </dl>                            | Crée des contrôles statiques. Ces contrôles contiennent du texte non modifiable. <br/>                                                                                                                                                                                  |
| <span id="WC_TABCONTROL"></span><span id="wc_tabcontrol"></span><dl> <dt>**le \_ TABCONTROL de WC**</dt> </dl>                | Crée des contrôles onglet. Ces contrôles définissent plusieurs pages pour la même zone d’une fenêtre ou d’une boîte de dialogue. Chaque page se compose d’un ensemble d’informations ou d’un groupe de contrôles qu’une application affiche lorsque l’utilisateur sélectionne l’onglet correspondant.<br/> |
| <span id="WC_TREEVIEW"></span><span id="wc_treeview"></span><dl> <dt>**\_arborescence WC**</dt> </dl>                      | Crée des contrôles d’arborescence. Ces contrôles affichent une liste hiérarchique d’éléments. Chaque élément se compose d’une étiquette et d’une image bitmap facultative.<br/>                                                                                                              |
