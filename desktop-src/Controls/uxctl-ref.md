# <a name="visual-styles-reference"></a>Référence des styles visuels

Cette section décrit les éléments d’API suivants utilisés avec les [styles visuels](themes-overview.md).

### <a name="functions"></a>Fonctions



| Rubrique                                                                                  | Contenu                                                                                                                                                                                                                           |
|----------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**BeginBufferedAnimation**](/windows/desktop/api/Uxtheme/nf-uxtheme-beginbufferedanimation)                               | Commence une opération d’animation mise en mémoire tampon. L’animation se compose d’un fondu croisé entre le contenu de deux mémoires tampons sur une période de temps spécifiée. <br/>                                                                     |
| [**BeginBufferedPaint**](/windows/desktop/api/Uxtheme/nf-uxtheme-beginbufferedpaint)                                       | Commence une opération de peinture mise en mémoire tampon.<br/>                                                                                                                                                                                      |
| [**BeginPanningFeedback**](/windows/desktop/api/Uxtheme/nf-uxtheme-beginpanningfeedback)                                   | Indique au système d’envoyer des commentaires sur une fenêtre cible affectée par les gestes panoramiques. <br/>                                                                                                                               |
| [**BufferedPaintClear**](/windows/desktop/api/Uxtheme/nf-uxtheme-bufferedpaintclear)                                       | Efface de la mémoire tampon un rectangle spécifié en ARVB = {0,0,0,0} .<br/>                                                                                                                                                         |
| [**BufferedPaintInit**](/windows/desktop/api/Uxtheme/nf-uxtheme-bufferedpaintinit)                                         | Initialise la peinture mise en mémoire tampon pour le thread actuel.<br/>                                                                                                                                                                    |
| [**BufferedPaintRenderAnimation**](/windows/desktop/api/Uxtheme/nf-uxtheme-bufferedpaintrenderanimation)                   | Peint l’image suivante d’une animation de peinture mise en mémoire tampon.<br/>                                                                                                                                                                    |
| [**BufferedPaintSetAlpha**](/windows/desktop/api/Uxtheme/nf-uxtheme-bufferedpaintsetalpha)                                 | Affecte à l’alpha une valeur spécifiée dans un rectangle donné. Le alpha contrôle la quantité de transparence appliquée lors de la fusion avec la mémoire tampon sur le contexte de périphérique (DC) cible de destination. <br/>                         |
| [**BufferedPaintStopAllAnimations**](/windows/desktop/api/Uxtheme/nf-uxtheme-bufferedpaintstopallanimations)               | Arrête toutes les animations mises en mémoire tampon pour la fenêtre spécifiée.<br/>                                                                                                                                                                     |
| [**BufferedPaintUnInit**](/windows/desktop/api/Uxtheme/nf-uxtheme-bufferedpaintuninit)                                     | Ferme le remplissage mis en mémoire tampon pour le thread actuel. Appelée une fois pour chaque appel à [**BufferedPaintInit**](/windows/desktop/api/Uxtheme/nf-uxtheme-bufferedpaintinit) après que les appels à [**BeginBufferedPaint**](/windows/desktop/api/Uxtheme/nf-uxtheme-beginbufferedpaint) ne sont plus nécessaires.<br/> |
| [**CloseThemeData**](/windows/desktop/api/Uxtheme/nf-uxtheme-closethemedata)                                               | Ferme le handle de données de thème.<br/>                                                                                                                                                                                           |
| [**DrawThemeBackground**](/windows/desktop/api/Uxtheme/nf-uxtheme-drawthemebackground)                                     | Dessine la bordure et le remplissage définis par le style visuel pour la partie de contrôle spécifiée.<br/>                                                                                                                                   |
| [**DrawThemeBackgroundEx**](/windows/desktop/api/Uxtheme/nf-uxtheme-drawthemebackgroundex)                                 | Dessine l’image d’arrière-plan définie par le style visuel pour la partie de contrôle spécifiée.<br/>                                                                                                                                  |
| [**DrawThemeEdge**](/windows/desktop/api/Uxtheme/nf-uxtheme-drawthemeedge)                                                 | Dessine un ou plusieurs bords définis par le style visuel d’un rectangle.<br/>                                                                                                                                                     |
| [**DrawThemeIcon**](/windows/desktop/api/Uxtheme/nf-uxtheme-drawthemeicon)                                                 | Dessine une image à partir d’une liste d’images avec l’effet d’icône défini par le style visuel.<br/>                                                                                                                                     |
| [**DrawThemeParentBackground**](/windows/desktop/api/Uxtheme/nf-uxtheme-drawthemeparentbackground)                         | Dessine la partie d’un contrôle parent qui est couvert par un contrôle enfant partiellement transparent ou à fusion alpha.<br/>                                                                                                           |
| [**DrawThemeParentBackgroundEx**](/windows/desktop/api/Uxtheme/nf-uxtheme-drawthemeparentbackgroundex)                     | Utilisé par les contrôles enfants partiellement transparents ou à contrôle alpha pour dessiner la partie de leur parent devant laquelle ils apparaissent. Envoie un \_ message WM ERASEBKGND suivi d’un \_ PRINTCLIENT WM.<br/>                             |
| [**DrawThemeText**](/windows/desktop/api/Uxtheme/nf-uxtheme-drawthemetext)                                                 | Dessine du texte à l’aide de la couleur et de la police définies par le style visuel.<br/>                                                                                                                                                        |
| [**DrawThemeTextEx**](/windows/desktop/api/Uxtheme/nf-uxtheme-drawthemetextex)                                             | Dessine du texte à l’aide de la couleur et de la police définies par le style visuel. Étend [**DrawThemeText**](/windows/desktop/api/Uxtheme/nf-uxtheme-drawthemetext) en autorisant des options de format de texte supplémentaires.<br/>                                                             |
| [**EnableThemeDialogTexture**](/windows/desktop/api/Uxtheme/nf-uxtheme-enablethemedialogtexture)                           | Active ou désactive le style visuel de l’arrière-plan d’une fenêtre de boîte de dialogue.<br/>                                                                                                                                                   |
| [**EnableTheming**](/windows/desktop/api/Uxtheme/nf-uxtheme-enabletheming)                                                 | Active ou désactive les styles visuels pour l’utilisateur actuel dans les sessions actuelles et ultérieures.<br/>                                                                                                                               |
| [**EndBufferedAnimation**](/windows/desktop/api/Uxtheme/nf-uxtheme-endbufferedanimation)                                   | Génère le rendu de la première image d’une opération d’animation mise en mémoire tampon et démarre le minuteur d’animation.<br/>                                                                                                                               |
| [**EndBufferedPaint**](/windows/desktop/api/Uxtheme/nf-uxtheme-endbufferedpaint)                                           | Termine une opération de peinture en mémoire tampon et libère le handle de peinture mis en mémoire tampon associé.<br/>                                                                                                                                    |
| [**EndPanningFeedback**](/windows/desktop/api/Uxtheme/nf-uxtheme-endpanningfeedback)                                       | Met fin à toute animation existante qui était en cours ou configurée par [**BeginPanningFeedback**](/windows/desktop/api/Uxtheme/nf-uxtheme-beginpanningfeedback) et [**UpdatePanningFeedback**](/windows/desktop/api/Uxtheme/nf-uxtheme-updatepanningfeedback). <br/>                                    |
| [**GetBufferedPaintBits**](/windows/desktop/api/Uxtheme/nf-uxtheme-getbufferedpaintbits)                                   | Récupère un pointeur vers la bitmap de mémoire tampon si la mémoire tampon est une bitmap indépendante du périphérique (DIB).<br/>                                                                                                                            |
| [**GetBufferedPaintDC**](/windows/desktop/api/Uxtheme/nf-uxtheme-getbufferedpaintdc)                                       | Obtient le DC de peinture. Il s’agit de la même valeur récupérée par [**BeginBufferedPaint**](/windows/desktop/api/Uxtheme/nf-uxtheme-beginbufferedpaint).<br/>                                                                                                                |
| [**GetBufferedPaintTargetDC**](/windows/desktop/api/Uxtheme/nf-uxtheme-getbufferedpainttargetdc)                           | Récupère le contrôleur de périphérique cible. <br/>                                                                                                                                                                                               |
| [**GetBufferedPaintTargetRect**](/windows/desktop/api/Uxtheme/nf-uxtheme-getbufferedpainttargetrect)                       | Récupère le rectangle cible spécifié par BeginBufferedPaint.<br/>                                                                                                                                                         |
| [**GetCurrentThemeName**](/windows/desktop/api/Uxtheme/nf-uxtheme-getcurrentthemename)                                     | Récupère le nom du style visuel en cours et, éventuellement, récupère le nom du modèle de couleurs et le nom de la taille.<br/>                                                                                                           |
| [**GetThemeAppProperties**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemeappproperties)                                 | Récupère les indicateurs de propriété qui contrôlent la manière dont les styles visuels sont appliqués dans l’application actuelle.<br/>                                                                                                                     |
| [**GetThemeBackgroundContentRect**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemebackgroundcontentrect)                 | Récupère la taille de la zone de contenu pour l’arrière-plan défini par le style visuel.<br/>                                                                                                                                  |
| [**GetThemeBackgroundExtent**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemebackgroundextent)                           | Calcule la taille et l’emplacement de l’arrière-plan, défini par le style visuel, en fonction de la zone de contenu.<br/>                                                                                                                |
| [**GetThemeBackgroundRegion**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemebackgroundregion)                           | Calcule la région d’un arrière-plan standard ou partiellement transparent délimité par un rectangle spécifié.<br/>                                                                                                         |
| [**GetThemeBitmap**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemebitmap)                                               | Récupère l’image bitmap associée à un thème, une partie, un État et une propriété particuliers.<br/>                                                                                                                                     |
| [**GetThemeBool**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemebool)                                                   | Récupère la valeur d’une propriété **bool** à partir de la section SysMetrics des données de thème.<br/>                                                                                                                                   |
| [**GetThemeColor**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemecolor)                                                 | Récupère la valeur d’une propriété de couleur.<br/>                                                                                                                                                                                |

| [**GetThemeEnumValue**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemeenumvalue)                                         | Récupère la valeur d’une propriété de type énuméré.<br/>                                                                                                                                                                     |
