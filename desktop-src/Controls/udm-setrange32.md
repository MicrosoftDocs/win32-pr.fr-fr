# <a name="udm_setrange32-message"></a>\_Message SETRANGE32 UDM

Définit la plage de bits 32 d’un contrôle up-up.

## <a name="parameters"></a>Paramètres

<dl> <dt>

*wParam* 
</dt> <dd>

Valeur entière signée qui représente la nouvelle limite inférieure de la plage de contrôles Up-Up.

</dd> <dt>

*lParam* 
</dt> <dd>

Valeur entière signée qui représente la nouvelle limite supérieure de la plage de contrôles Up-Up.

</dd> </dl>

## <a name="return-value"></a>Valeur de retour

La valeur de retour de ce message n’est pas utilisée.
