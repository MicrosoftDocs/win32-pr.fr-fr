# <a name="tb_getbuttonsize-message"></a>TO \_ GETBUTTONSIZE message

Récupère la largeur et la hauteur actuelles des boutons de la barre d’outils, en pixels.

## <a name="parameters"></a>Paramètres

<dl> <dt>

*wParam* 
</dt> <dd>Doit être zéro.</dd> <dt>

*lParam* 
</dt> <dd>Doit être zéro.</dd> </dl>

## <a name="return-value"></a>Valeur retournée

Retourne une valeur **DWORD** qui contient les valeurs de largeur et de hauteur dans le mot de poids faible et le mot de poids fort, respectivement.
