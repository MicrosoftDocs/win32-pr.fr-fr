# <a name="using-tooltip-controls"></a>Utilisation des contrôles ToolTip

Cette section contient des exemples qui montrent comment créer différents types d’info-bulles.

## <a name="in-this-section"></a>Contenu de cette section



| Rubrique                                                                                                    | Description                                                                                                                                                                                                                                                                      |
|----------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Comment créer une info-bulle pour un contrôle](create-a-tooltip-for-a-control.md)<br/>                   | L’exemple de fonction suivant crée une info-bulle et l’associe au contrôle dont l’ID de ressource est passé. <br/>                                                                                                                                                  |
| [Comment créer une info-bulle pour une zone rectangulaire](create-a-tooltip-for-a-rectangular-area.md)<br/> | L’exemple suivant montre comment créer un contrôle ToolTip standard pour l’ensemble de la zone client d’une fenêtre. <br/>                                                                                                                                                       |
| [Comment implémenter des info-bulles de suivi](implement-tracking-tooltips.md)<br/>                         | Les info-bulles de suivi restent visibles jusqu’à ce qu’elles soient explicitement fermées par l’application et peuvent changer la position sur l’écran dynamiquement. Ils sont pris en charge par la [version 4,70](common-control-versions.md) et les versions ultérieures des contrôles communs. <br/>                         |
| [Comment implémenter des info-bulles multilignes](implement-multiline-tooltips.md)<br/>                       | Les info-bulles multilignes autorisent l’affichage du texte sur plusieurs lignes. <br/>                                                                                                                                                                                                 |
| [Comment implémenter des info-bulles](implement-balloon-tooltips.md)<br/>                           | Les info-bulles sont similaires aux info-bulles standard, mais elles sont affichées dans un « ballon » de style dessin animé avec une tige pointant sur l’outil. Les info-bulles peuvent être à une seule ligne ou multiligne. Elles sont créées et gérées à peu près de la même façon que les info-bulles standard. <br/> |
| [Comment implémenter des info-bulles pour les icônes de barre d’État](implement-tooltips-for-status-bar-icons.md)<br/> | Une méthode non intrusive pour afficher un message explicatif pour une icône de barre d’état consiste à implémenter une info-bulle. L’info-bulle disparaît lorsque vous cliquez dessus, mais vous pouvez également spécifier une valeur de délai d’attente. <br/>                                                                                |
| [Comment implémenter des info-bulles In-Place](implement-in-place-tooltips.md)<br/>                         | Les info-bulles sur place sont utilisées pour afficher des chaînes de texte pour les objets qui ont été découpés. Pour obtenir une illustration, consultez [à propos des contrôles ToolTip](tooltip-controls.md). <br/>                                                                                                      |
