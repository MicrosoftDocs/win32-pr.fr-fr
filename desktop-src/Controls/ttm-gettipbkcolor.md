# <a name="ttm_gettipbkcolor-message"></a>\_Message atténuation GETTIPBKCOLOR

Récupère la couleur d’arrière-plan dans une fenêtre d’info-bulle.

## <a name="parameters"></a>Paramètres

<dl> <dt>

*wParam* 
</dt> <dd>Doit être zéro.</dd> <dt>

*lParam* 
</dt> <dd>Doit être zéro.</dd> </dl>

## <a name="return-value"></a>Valeur retournée

Retourne une valeur **COLORREF** qui représente la couleur d’arrière-plan.
