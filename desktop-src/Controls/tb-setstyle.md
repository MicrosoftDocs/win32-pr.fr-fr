# <a name="tb_setstyle-message"></a>TO, \_ message SETSTYLE

Définit le style d’un contrôle ToolBar.

## <a name="parameters"></a>Paramètres

<dl> <dt>

*wParam* 
</dt> <dd>Doit être zéro.</dd> <dt>

*lParam* 
</dt> <dd>

Valeur spécifiant les styles à définir pour le contrôle. Cette valeur peut être une combinaison de [styles de contrôle de barre d’outils](toolbar-control-and-button-styles.md).

</dd> </dl>

## <a name="return-value"></a>Valeur retournée

Pas de valeur de retour.
