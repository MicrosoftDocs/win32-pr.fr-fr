# <a name="windows-mixed-reality-com-interfaces"></a>Interfaces COM Windows Mixed Reality

Les rubriques contenues dans cette section fournissent les spécifications de référence pour les interfaces COM Windows Mixed Reality.

## <a name="in-this-section"></a>Contenu de cette section

| Rubrique | Description |
|-|-|
| [**ISpatialInteractionManagerInterop**](/windows/desktop/api/spatialinteractionmanagerinterop/nn-spatialinteractionmanagerinterop-ispatialinteractionmanagerinterop) | Permet l’interopérabilité avec plateforme Windows universelle un objet [SpatialInteractionManager](/uwp/api/Windows.UI.Input.Spatial.SpatialInteractionManager) (UWP), et donne accès aux membres **SpatialInteractionManager** pour accéder aux entrées d’utilisateur à partir de mains, de contrôleurs de mouvement et de commandes vocales du système. |
| [**IHolographicSpaceInterop**](/windows/desktop/api/holographicspaceinterop/nn-holographicspaceinterop-iholographicspaceinterop) | Permet l’interopérabilité avec plateforme Windows universelle un objet [HolographicSpace](/uwp/api/windows.graphics.holographic.holographicspace) (UWP) et donne accès aux membres **HolographicSpace** pour représenter une scène holographique. |
