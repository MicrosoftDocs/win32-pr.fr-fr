# <a name="strict-type-checking"></a>Vérification de type STRICTe

le fichier d’en-tête Windows. h contient des définitions, des macros et des structures pour vous aider à écrire du code source portable entre les versions de Windows. Certaines de ces fonctionnalités sont activées lorsque vous définissez le symbole STRICT lors de la génération de votre application. Les rubriques suivantes expliquent les avantages des fonctionnalités STRICTes et comment les utiliser affectent l’écriture de code.

-   [Activation du STRICT](enabling-strict.md)
-   [Désactivation de STRICT](disabling-strict.md)
-   [Conformité STRICTe](strict-compliance.md)
