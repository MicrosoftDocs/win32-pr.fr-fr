# <a name="dxgi-constants"></a>Constantes DXGI

Cette section contient des informations sur les constantes fournies par DXGI.

## <a name="in-this-section"></a>Contenu de cette section



| Rubrique                                                                    | Description                                                                                                         |
|--------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------|
| [**\_ID de débogage dxgi \_**](dxgi-debug-id.md)<br/>                      | Valeurs d’identificateur global unique (GUID) qui identifient les producteurs de messages de débogage.<br/>                      |
| [**\_modes d’énumération dxgi \_**](dxgi-enum-modes.md)<br/>                  | Options pour énumérer les modes d’affichage.<br/>                                                                   |
| [**\_erreur dxgi**](dxgi-error.md)<br/>                             | Codes d’erreur qui peuvent être retournés par une fonction DXGI.<br/>                                                     |
| [**DXGI \_ présente**](dxgi-present.md)<br/>                         | Les constantes [**dxgi \_ présentes**](dxgi-present.md) spécifient des options pour la présentation des frames à la sortie.<br/> |
| [**\_RW dxgi Shared \_ Resource \_**](dxgi-shared-resource-rw.md)<br/> | Constantes de ressources partagées.<br/>                                                                               |
| [**État de DXGI \_**](dxgi-status.md)<br/>                           | Codes d’État qui peuvent être retournés par les fonctions DXGI.<br/>                                                     |
| [**utilisation de DXGI \_**](dxgi-usage.md)<br/>                             | Indicateurs pour les options de création de surface et de ressource.<br/>                                                         |
