# <a name="dxgi"></a><span data-ttu-id="696a2-103">DXGI</span><span class="sxs-lookup"><span data-stu-id="696a2-103">DXGI</span></span>

<span data-ttu-id="696a2-104">Microsoft DirectX Graphics infrastructure (DXGI) gère l’énumération des cartes graphiques, l’énumération des modes d’affichage, la sélection des formats de mémoire tampon, le partage des ressources entre les processus (par exemple, entre les applications et le Gestionnaire de fenêtrage (DWM)) et la présentation des frames rendus à une fenêtre ou à un moniteur à des fins d’affichage.</span><span class="sxs-lookup"><span data-stu-id="696a2-104">Microsoft DirectX Graphics Infrastructure (DXGI) handles enumerating graphics adapters, enumerating display modes, selecting buffer formats, sharing resources between processes (such as, between applications and the Desktop Window Manager (DWM)), and presenting rendered frames to a window or monitor for display.</span></span>

<span data-ttu-id="696a2-105">DXGI est utilisé par Direct3D 10, Direct3D 11 et Direct3D 12.</span><span class="sxs-lookup"><span data-stu-id="696a2-105">DXGI is used by Direct3D 10, Direct3D 11 and Direct3D 12.</span></span>

<span data-ttu-id="696a2-106">Bien que la plupart des programmations graphiques s’effectuent à l’aide de Direct3D, vous pouvez utiliser DXGI pour présenter des frames à une fenêtre, une analyse ou tout autre composant graphique pour la composition et l’affichage éventuels.</span><span class="sxs-lookup"><span data-stu-id="696a2-106">Though most graphics programming is done using Direct3D, you can use DXGI to present frames to a window, monitor, or other graphics component for eventual composition and display.</span></span> <span data-ttu-id="696a2-107">Vous pouvez également utiliser DXGI pour lire le contenu d’une analyse.</span><span class="sxs-lookup"><span data-stu-id="696a2-107">You can also use DXGI to read the contents on a monitor.</span></span>
