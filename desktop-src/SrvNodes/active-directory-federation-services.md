# <a name="active-directory-federation-services"></a>Services de fédération Active Directory (AD FS)

Active Directory service FS (Federation Service) (AD FS) est un service d’authentification et d’autorisation qui s’exécute sur Windows Server. AD FS fournit aux utilisateurs un accès par authentification unique entre les environnements d’entreprise et les applications Web.
