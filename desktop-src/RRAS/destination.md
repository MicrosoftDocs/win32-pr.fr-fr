# <a name="destination"></a>Destination

Une destination est un hôte qui a rejoint un groupe de multidiffusion. Un tel hôte a informé le routeur local, à l’aide d’IGMP, qu’il souhaite recevoir des données envoyées à un ou plusieurs groupes de multidiffusion.
