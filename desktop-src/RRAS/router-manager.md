# <a name="router-manager"></a>Gestionnaire de routeur

le gestionnaire de routeur est le composant d’un routeur Windows NT/Windows 2000 qui démarre et gère les différents composants de routage. Les composants incluent les protocoles de routage, le gestionnaire de tables de routage et le gestionnaire de groupe de multidiffusion.
