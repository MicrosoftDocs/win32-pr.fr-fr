# <a name="enumerating-routing-table-entries"></a>Énumération des entrées de table de routage

Les fonctions d’énumération permettent à un client de récupérer des informations sur un type spécifique d’objet de table de routage (itinéraires, destinations et sauts suivants). Les protocoles de routage et les programmes d’administration peuvent utiliser ces fonctions pour rechercher des données spécifiques.

Pour obtenir un exemple de code qui montre comment utiliser ces fonctions, consultez [énumérer toutes les destinations](enumerate-all-destinations.md) et [énumérer tous les itinéraires](enumerate-all-routes.md).
