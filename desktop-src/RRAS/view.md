# <a name="view"></a>Affichage

Une vue est un sous-ensemble de la table de routage et contient un groupe d’itinéraires associés (par exemple, des itinéraires de multidiffusion). Actuellement, seuls les affichages monodiffusion et multidiffusion sont pris en charge. Les vues sont parfois appelées des bases d’informations de routage (côtes).
