# <a name="routing-and-remote-access-service"></a>Service de routage et d’accès à distance

## <a name="purpose"></a>Objectif

Le service d’accès à distance (RAS) peut être utilisé pour créer des applications clientes. Ces applications affichent les boîtes de dialogue courantes RAS, gèrent les connexions et les appareils d’accès à distance et manipulent les entrées d’annuaire téléphonique.

Les API de routage permettent de créer des applications pour administrer les fonctionnalités de routage du système d’exploitation.

Les développeurs peuvent utiliser les API de protocole de routage pour implémenter des protocoles de routage.

## <a name="developer-audience"></a>Développeurs concernés

Les API de service de routage et d’accès à distance sont conçues pour être utilisées par les programmeurs C/C++. Les programmeurs doivent également être familiarisés avec les concepts de mise en réseau.
