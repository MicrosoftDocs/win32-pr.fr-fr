-   \_Informations de \_ filtre de numérotation à la demande IP \_ \_ \_ V6
-   \_Informations d’activation du filtre IP \_ \_ \_ V6
-   IP \_ dans \_ les \_ informations de filtre \_ V6
-   IP \_ IFFILTER \_ info \_ V6
-   \_Informations sur le filtre de sortie IP \_ \_ \_ V6
-   \_Informations globales IPv6 \_
-   \_Informations sur l’itinéraire IPv6 \_
