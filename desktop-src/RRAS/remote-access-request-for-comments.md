# <a name="remote-access-request-for-comments"></a>Demande d’accès à distance pour les commentaires

Les RFC (Request for Comments) sont une série de documents en cours qui traitent des technologies Internet et Internet. Les RFC spécifiques sont identifiés par nombre. Les RFC suivantes sont référencées dans la section service d’accès à distance (RAS) du kit de développement logiciel (SDK) de la plateforme. Pour plus d’informations sur les RFC et sur la localisation de copies d’une RFC particulière, consultez [https://www.rfc-editor.org/rfc.html](https://www.rfc-editor.org/rfc.html) .

-   [RFC 791](https://www.ietf.org/rfc/rfc791.txt) – Internet Protocol, DARPA Internet Program Protocol Specification, septembre 1981
-   [RFC 1171](https://www.ietf.org/rfc/rfc1171.txt) – le protocole PPP, le 1990 juillet
-   [RFC 1172](https://www.ietf.org/rfc/rfc1172.txt) – options de configuration initiale PPP, juillet, 1990
-   [RFC 1570](https://www.ietf.org/rfc/rfc1570.txt) – extensions LCP PPP, janvier 1994
-   [RFC 1962](https://www.ietf.org/rfc/rfc1962.txt) – le protocole de contrôle de compression PPP (CCP), juin 1996
-   [RFC 1974](https://www.ietf.org/rfc/rfc1974.txt) – protocole de compression PPP STAC LZS, août 1996
-   [RFC 1990](https://www.ietf.org/rfc/rfc1990.txt) – le protocole PPP MP, août 1996
-   [RFC 2118](https://www.ietf.org/rfc/rfc2118.txt) – protocole Microsoft Point-to-point compression (MPPC), mars 1997
-   [RFC 2284](https://www.ietf.org/rfc/rfc2284.txt) – protocole EAP (Extensible Authentication Protocol), mars 1998
-   [RFC 2433](https://www.ietf.org/rfc/rfc2433.txt) – extensions Microsoft PPP chap, octobre 1998
-   [RFC 2484](https://www.ietf.org/rfc/rfc2484.txt) – option de configuration de l’internationalisation PPP LCP, janvier 1999
-   [RFC 2548](https://www.ietf.org/rfc/rfc2848.txt) – attributs RADIUS spécifiques au fournisseur Microsoft
-   [RFC 2865](https://www.ietf.org/rfc/rfc2865.txt) – Remote Authentication Dial in User Service (RADIUS), juin 2000
-   [RFC 2866](https://www.ietf.org/rfc/rfc2866.txt) – gestion des comptes RADIUS, juin 2000
-   [RFC 2868](https://www.ietf.org/rfc/rfc2868.txt) -attributs Radius pour la prise en charge du protocole Tunnel, juin 2000
-   [RFC 2869](https://www.ietf.org/rfc/rfc2869.txt) – extensions RADIUS, 2000 juin
-   [Draft RADIUS MSCHAP attr 01](https://www.freeradius.org/rfc/draft-ietf-radius-mschap-attr-01.txt) : attributs RADIUS pour la prise en charge de MS-CHAP, novembre 1997
