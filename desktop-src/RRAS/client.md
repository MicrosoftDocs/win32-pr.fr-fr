# <a name="client"></a>Client

Un client de RTMv2 est soit une instance d’un protocole de routage, soit un composant de gestion qui inscrit et interagit avec le gestionnaire de tables de routage à l’aide de l’API RTMv2. Sauf indication contraire, toute référence à un client du gestionnaire de tables de routage fait référence à un client de RTMv2.

Un client doit s’exécuter dans le même processus que le gestionnaire de routage.
