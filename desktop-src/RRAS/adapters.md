# <a name="adapters"></a>Adaptateurs

Un adaptateur représente le matériel qui se connecte à un segment de réseau particulier. Une carte LAN liée est un exemple d’adaptateur. De même, un ordinateur équipé de deux modems, chacun étant en capacité de se connecter à un réseau distant, dispose de deux cartes, une pour représenter chaque modem.
