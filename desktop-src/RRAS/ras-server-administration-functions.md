# <a name="ras-server-administration-functions"></a>Fonctions d’administration du serveur RAS

Pour Windows NT 4,0, utilisez les fonctions suivantes pour implémenter la fonctionnalité d’administration de serveur RAS. Windows 95 ne fournit pas de prise en charge du serveur RAS. Pour les systèmes d’exploitation Windows 2000, Windows Server 2003 et versions ultérieures, utilisez la [référence d’administration du service d’accès à distance](remote-access-service-administration-reference.md).

-   [**RasAdminFreeBuffer**](rasadminfreebuffer.md)
-   [**RasAdminGetErrorString**](rasadmingeterrorstring.md)
-   [**RasAdminGetUserAccountServer**](rasadmingetuseraccountserver.md)
-   [**RasAdminPortClearStatistics**](rasadminportclearstatistics.md)
-   [**RasAdminPortDisconnect**](rasadminportdisconnect.md)
-   [**RasAdminPortEnum**](rasadminportenum.md)
-   [**RasAdminPortGetInfo**](rasadminportgetinfo.md)
-   [**RasAdminServerGetInfo**](rasadminservergetinfo.md)
-   [**RasAdminUserGetInfo**](rasadminusergetinfo.md)
-   [**RasAdminUserSetInfo**](rasadminusersetinfo.md)
-   [Fonctions DLL d’administration RAS](ras-administration-dll-functions.md)
-   [Fonctions de la DLL de sécurité RAS](ras-security-dll-functions.md)
