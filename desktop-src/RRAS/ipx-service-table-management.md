# <a name="ipx-service-table-management"></a>Gestion des tables de services IPX

Un protocole de routage IPX qui s’inscrit pour la gestion des services gère une table de liens de service telle que définie par les spécifications SAP (Service Advertising Protocol) de Novell. Le protocole de routage fournit également l’accès du gestionnaire de routeur à ce tableau via les fonctions et structures suivantes.

-   [Fonctions de gestion des tables de services](service-table-management-functions.md)
-   [Structures de gestion des tables de services](service-table-management-structures.md)

Pour plus d’informations sur la façon dont un protocole de routage s’inscrit pour la gestion des services, consultez la fonction [**RegisterProtocol**](/windows/desktop/api/Routprot/nc-routprot-pregister_protocol) .
