# <a name="transport-identifiers"></a>Identificateurs de transport

Les identificateurs de transport (ou de famille de protocoles) suivants sont également répertoriés dans mprapi. h.



| Valeur      | Transport (famille de protocoles)                               |
|------------|-----------------------------------------------------------|
| PID \_ ATALK | AppleTalk                                                 |
| \_adresse IP PID    | Protocole Internet version 4                               |
| \_IPX PID   | Exchange de paquets Internet                                  |
| \_NBF PID   | Protocole de trames NetBIOS                                   |
| \_IPv6 PID  | Windows Serveur 2008 ou version ultérieure : protocole Internet version 6 |
