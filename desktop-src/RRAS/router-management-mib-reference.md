# <a name="router-management-mib-reference"></a>Référence MIB de la gestion du routeur

Cette documentation décrit les éléments de référence utilisés dans l’API MIB (Management Information base) spécifique à la gestion du routeur. Utilisez ces éléments de référence pour interroger et définir les valeurs des variables MIB. Les gestionnaires de routeur et les protocoles de routage que les services du gestionnaire de routage exportent les variables MIB :

-   [Fonctions MIB de gestion du routeur](mib-functions.md)
-   [Transport et constantes de protocole](transport-and-protocol-constants.md)

Les structures qui définissent les variables MIB sont documentées dans la rubrique mise en réseau de l' [API de base d’informations de gestion](/previous-versions/windows/desktop/mib/management-information-base-reference) agnostique de la technologie.
