# <a name="retrieving-route-information"></a>Récupération des informations d’itinéraire

Trois méthodes sont utilisées pour obtenir des informations d’itinéraire à partir du gestionnaire de tables de routage.

-   Énumération des itinéraires (décrits dans [énumération des entrées de table de routage](enumerating-routing-table-entries.md)).
-   Recherche d’itinéraires spécifiques (décrits dans [recherche d’informations spécifiques dans la table de routage](finding-specific-information-in-the-routing-table.md)).
-   Récupération des destinations modifiées (décrite dans [réception des notifications de modifications](receiving-notification-of-changes.md)).
