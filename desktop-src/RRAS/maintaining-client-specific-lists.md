# <a name="maintaining-client-specific-lists"></a>Maintenance des listes de Client-Specific

RTMv2 fournit des fonctions qui permettent aux clients de créer une liste privée d’itinéraires stockés dans la table de routage. L’utilisation de cette liste peut s’avérer plus efficace que l’énumération d’itinéraires à partir de la table de routage.

Pour obtenir un exemple de code qui montre comment utiliser ces fonctions, consultez [utiliser une liste d’itinéraires Client-Specific](use-a-client-specific-route-list.md).
