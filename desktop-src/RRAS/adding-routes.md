# <a name="adding-routes"></a>Ajout d’itinéraires

Une fois qu’un client a découvert un itinéraire, le client peut ajouter cet itinéraire à la table de routage en appelant la fonction [**RtmAddNextHop**](/windows/desktop/api/Rtmv2/nf-rtmv2-rtmaddnexthop) .

Un client peut supprimer des itinéraires de la table de routage en appelant la fonction [**RtmDeleteRouteToDest**](/windows/desktop/api/Rtmv2/nf-rtmv2-rtmdeleteroutetodest) .

Pour obtenir un exemple de code qui montre comment utiliser ces fonctions, consultez [Ajouter et mettre à jour des itinéraires à l’aide de RtmAddRouteToDest](add-and-update-routes-using-rtmaddroutetodest.md).
