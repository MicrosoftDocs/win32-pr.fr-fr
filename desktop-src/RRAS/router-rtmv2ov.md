# <a name="router-rras"></a>Routeur (RRAS)

Dans le cadre de cette documentation, un routeur est un serveur Windows NT/Windows 2000 qui exécute le service de routage et d’accès à distance (RRAS) et le gestionnaire de tables de routage.
