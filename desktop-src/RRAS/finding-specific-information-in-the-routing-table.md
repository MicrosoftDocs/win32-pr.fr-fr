# <a name="finding-specific-information-in-the-routing-table"></a>Recherche d’informations spécifiques dans la table de routage

Les clients doivent être en mesure de localiser des informations spécifiques dans la table de routage, plutôt que de devoir énumérer toutes les données. L’API RTMv2 offre la possibilité de rechercher des destinations, des itinéraires et des tronçons suivants en fonction de certains critères.
