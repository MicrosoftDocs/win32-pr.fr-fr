# <a name="ras-server-administration-union"></a>Union d’administration du serveur RAS

Pour Windows NT 4,0, utilisez l’Union suivante pour implémenter la fonctionnalité d’administration de serveur RAS. Windows 95 ne fournit pas de prise en charge du serveur RAS. pour les systèmes d’exploitation Windows 2000, Windows Server 2003 et versions ultérieures, utilisez la [référence d’Administration du Service d’accès à distance](remote-access-service-administration-reference.md).
