# <a name="routing-table"></a>Table de routage

Le gestionnaire de tables de routage gère une table de routage distincte pour chaque famille d’adresses (par exemple, IPX et IPv4). La famille d’adresses IPX est prise en charge par l’API RTMv1. IPv4 est pris en charge par l’API RTMv2.

La table de routage est constituée de destinations, d’itinéraires et de tronçons suivants. Ces entrées définissent un itinéraire vers un réseau de destination.
