# <a name="address-family"></a>Famille d’adresses

Les familles d’adresses incluent les protocoles IPX (Internet Packet Exchange), IPv4 (Internet Protocol version 4) et IPv6 (Internet Protocol version 6). Les familles d’adresses peuvent également être appelées familles de protocoles. Ces familles d’adresses sont définies dans le [document RFC 1700](routing-protocols-request-for-comments.md).
