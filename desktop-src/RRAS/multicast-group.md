# <a name="multicast-group"></a>Groupe de multidiffusion

Un groupe de multidiffusion est une adresse IP de classe D (autrement dit, dans la plage 224.0.0.0-239.255.255.255). Les données de multidiffusion envoyées à une adresse comprise dans cette plage ne sont pas destinées à une seule cible. Au lieu de cela, ces données peuvent être reçues par n’importe quel hôte qui effectue une demande de jointure d’un groupe spécifique. Un groupe de multidiffusion est désigné par le symbole G ou g.
