<dl> <dt>

<span id="_rras_point_to_multipoint_interface_gly"></span><span id="_RRAS_POINT_TO_MULTIPOINT_INTERFACE_GLY"></span>**interface point à multipoint**
</dt> <dd>

La technologie de gestion de groupe de multidiffusion utilise ce terme pour signifier une interface qui assure la communication entre une source unique et une destination unique, telle qu’une liaison PPP.

</dd> <dt>

<span id="_rras_point_to_point_interface_gly"></span><span id="_RRAS_POINT_TO_POINT_INTERFACE_GLY"></span>**interface point à point**
</dt> <dd>

La technologie de gestion de groupe de multidiffusion utilise ce terme pour signifier une interface qui assure la communication entre un seul hôte et plusieurs destinations. Les interfaces point à multipoint peuvent être considérées comme une collection de liens point à point avec un seul arrêt, tel qu’un lien ATM.

</dd> </dl>
