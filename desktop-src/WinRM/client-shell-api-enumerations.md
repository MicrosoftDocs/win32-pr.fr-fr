# <a name="client-shell-api-enumerations"></a>Énumérations de l’API de l’interpréteur de commandes client

Le tableau suivant fournit une vue d’ensemble des énumérations de l’interface de programmation d’applications (API) du shell client Windows Remote Management (WinRM).



| Énumérations                                                 | Description                                                                                      |
|--------------------------------------------------------------|--------------------------------------------------------------------------------------------------|
| [**WSManAuthenticationFlags**](/windows/desktop/api/Wsman/ne-wsman-wsmanauthenticationflags) | Définit le schéma d’authentification à utiliser pour l’authentification du serveur et du proxy.               |
| [**WSManCallbackFlags**](/windows/desktop/api/Wsman/ne-wsman-wsmancallbackflags)             | Définit un jeu d’indicateurs utilisé par toutes les fonctions de rappel.                                           |
| [**WSManDataType**](/windows/desktop/api/Wsman/ne-wsman-wsmandatatype)                       | Spécifie le type de données actuel de l’Union dans la structure de [**\_ données WSMan**](/windows/desktop/api/Wsman/ns-wsman-wsman_data) . |
| [**WSManProxyAccessType**](/windows/desktop/api/Wsman/ne-wsman-wsmanproxyaccesstype)         | Définit le type d’accès du proxy.                                                                   |
| [**WSManSessionOption**](/windows/desktop/api/Wsman/ne-wsman-wsmansessionoption)             | Définit un ensemble d’options étendues pour la session.                                               |
