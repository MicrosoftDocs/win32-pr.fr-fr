# <a name="resourcelocatorerror-property"></a>ResourceLocator. Error, propriété

Obtient une représentation XML des informations d’erreur supplémentaires.

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```VB
ResourceLocator.Error As BSTR
```



## <a name="property-value"></a>Valeur de la propriété

Représentation XML des informations d’erreur supplémentaires.
