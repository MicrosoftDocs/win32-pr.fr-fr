# <a name="client-shell-api-additional-functions"></a>API de l’interpréteur de commandes client fonctions supplémentaires

le tableau suivant fournit une vue d’ensemble des fonctions non-cœurs de l’interface de programmation d’applications (API) du Shell Client Windows Remote Management (WinRM).



| Fonctions supplémentaires                                                   | Description                       |
|------------------------------------------------------------------------|-----------------------------------|
| [**WSManGetErrorMessage**](/windows/desktop/api/Wsman/nf-wsman-wsmangeterrormessage)                   | Obtient les messages d’erreur.              |
| [**WSManGetSessionOptionAsDword**](/windows/desktop/api/Wsman/nf-wsman-wsmangetsessionoptionasdword)   | Obtient les options de session en tant que DWORD.  |
| [**WSManGetSessionOptionAsString**](/windows/desktop/api/Wsman/nf-wsman-wsmangetsessionoptionasstring) | Obtient les options de session sous la forme d’une chaîne. |
| [**WSManSetSessionOption**](/windows/desktop/api/Wsman/nf-wsman-wsmansetsessionoption)                 | Définit les options de session.             |
