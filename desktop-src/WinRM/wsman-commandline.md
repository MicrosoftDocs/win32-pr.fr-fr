# <a name="wsmancommandline-property"></a>WSMan. CommandLine, propriété

Obtient la ligne de commande du processus qui charge le composant Automation.

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```VB
WSMan.CommandLine As BSTR
```



## <a name="property-value"></a>Valeur de la propriété

Chaîne qui contient la ligne de commande.
