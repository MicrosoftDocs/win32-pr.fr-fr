# <a name="getting-started-developing-user-interfaces-for-windows-applications"></a>Prise en main développement d’interfaces utilisateur pour les applications Windows

## <a name="purpose"></a>Objectif

Les sections suivantes proposent des conseils généraux pour les développeurs qui conçoivent, implémentent et testent l’interface utilisateur d’une application Windows.

En plus des principes de conception de l’interface utilisateur de base, de nombreuses recommandations et suggestions sont fournies pour aider les développeurs à fournir une expérience utilisateur aussi simple, efficace et agréable que possible.

> [!Note]  
> Ces recommandations ne sont pas destinées à être exhaustives et sont soumises à la portée et aux fonctionnalités spécifiques d’une application. Pour obtenir des instructions plus complètes, consultez les [instructions relatives à l’interaction avec l’expérience utilisateur Windows](../uxguide/guidelines.md).
