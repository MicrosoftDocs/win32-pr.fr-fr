# <a name="dwm-structures"></a>Structures DWM

Cette section contient des informations sur les structures de Gestionnaire de fenêtrage (DWM).

## <a name="in-this-section"></a>Contenu de cette section



| Rubrique                                                                     | Description                                                                                                                                               |
|---------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**\_BLURBEHIND DWM**](/windows/desktop/api/Dwmapi/ns-dwmapi-dwm_blurbehind)<br/>                      | Spécifie les propriétés de l’effet de flou DWM. Utilisé par la fonction [**DwmEnableBlurBehindWindow**](/windows/desktop/api/Dwmapi/nf-dwmapi-dwmenableblurbehindwindow) .<br/>                     |
| [**paramètres de DWM \_ présents \_**](/windows/desktop/api/Dwmapi/ns-dwmapi-dwm_present_parameters)<br/>     | Spécifie les paramètres de trame vidéo DWM pour la composition de frame. Utilisé par la fonction [**DwmSetPresentParameters**](/windows/desktop/api/Dwmapi/nf-dwmapi-dwmsetpresentparameters) .<br/>   |
| [**Propriétés de la \_ miniature DWM \_**](/windows/desktop/api/Dwmapi/ns-dwmapi-dwm_thumbnail_properties)<br/> | Spécifie les propriétés de la miniature DWM. Utilisé par la fonction [**DwmUpdateThumbnailProperties**](/windows/desktop/api/Dwmapi/nf-dwmapi-dwmupdatethumbnailproperties) .<br/>                 |
| [**\_informations de minutage DWM \_**](/windows/desktop/api/Dwmapi/ns-dwmapi-dwm_timing_info)<br/>                   | Spécifie les informations de minutage de la composition DWM. Utilisé par la fonction [**DwmGetCompositionTimingInfo**](/windows/desktop/api/Dwmapi/nf-dwmapi-dwmgetcompositiontiminginfo) .<br/>         |
| [**MilMatrix3x2D**](/windows/desktop/api/Dwmapi/ns-dwmapi-milmatrix3x2d)<br/>                         | Spécifie une matrice matrice qui décrit une transformation. <br/>                                                                                            |
| [**RATIO non signé \_**](/windows/desktop/api/Dwmapi/ns-dwmapi-unsigned_ratio)<br/>                      | Définit un type de données utilisé par les API DWM. Il représente un ratio générique et est utilisé à des fins et des unités différentes, même dans une seule API.<br/> |
