# <a name="reset-method-of-the-msvm_diskettecontroller-class"></a>Méthode Reset de la \_ classe MSVM DisketteController

Demande une réinitialisation.

## <a name="syntax"></a>Syntaxe


```mof
uint32 Reset();
```



## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.
