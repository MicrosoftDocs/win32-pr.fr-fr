# <a name="hyper-v-vm-placement-api"></a>API de placement de machine virtuelle Hyper-V

L’API de placement de machine virtuelle Hyper-V contient les informations de compatibilité pour les machines virtuelles ou le système informatique d’hébergement.

## <a name="in-this-section"></a>Contenu de cette section



| Rubrique                                                                    | Description                                                                                                                                                |
|--------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**MSVM \_ CompatibilityVector**](msvm-compatibilityvector.md)<br/> | Fait référence aux informations de compatibilité d’un ordinateur virtuel (en cas d’exécution sur un système informatique d’ordinateur virtuel) ou à un ordinateur hôte (lorsqu’il est exécuté sur un système d’ordinateur hôte).<br/> |
