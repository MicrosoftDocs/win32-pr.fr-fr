# <a name="cim_serialcontroller-class-hyper-v-management"></a>Classe CIM_SerialController (gestion Hyper-V)

Décrit les fonctionnalités et la gestion d’un contrôleur série.

## <a name="syntax"></a>Syntaxe

``` syntax
[Abstract, Version("2.10.0"), UMLPackagePath("CIM::Device::Controller"), AMENDMENT]
class CIM_SerialController : CIM_Controller
{
  uint16 Capabilities[];
  string CapabilityDescriptions[];
  uint32 MaxBaudRate;
  uint16 Security;
};
```
