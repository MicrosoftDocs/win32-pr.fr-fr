# <a name="msvm_persistentmemorycontroller-class"></a>MSVM \_ PersistentMemoryController, classe

Représente un contrôleur de mémoire persistante virtuelle.

La syntaxe suivante est simplifiée par rapport au code MOF (Managed Object Format) et inclut toutes les propriétés héritées.

## <a name="syntax"></a>Syntaxe

``` syntax
[Dynamic, Provider("VmmsWmiInstanceAndMethodProvider"), AMENDMENT]
class Msvm_PersistentMemoryController : CIM_Controller
{
};
```

## <a name="members"></a>Membres

La classe **MSVM \_ PersistentMemoryController** ne définit aucun membre.
