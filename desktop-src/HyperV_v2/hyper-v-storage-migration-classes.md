# <a name="hyper-v-migration-api-reference"></a>Informations de référence sur l’API de migration Hyper-V

L’API de migration Hyper-V définit les éléments de programmation suivants.

## <a name="in-this-section"></a>Contenu de cette section



| Rubrique                                                                                                                                | Description                                                                                                                                                                                         |
|--------------------------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**MSVM \_ MigrationJob**](msvm-migrationjob.md)<br/>                                                                           | Cette classe représente un travail d’opération de migration créé pour le stockage ou la migration du système virtuel par le service de migration de système virtuel.<br/>                                                 |
| [**MSVM \_ VirtualSystemMigrationCapabilities**](msvm-virtualsystemmigrationcapabilities.md)<br/>                               | Définit les moyens permettant à un client de découvrir les méthodes fournies par le service de migration, ainsi que la plage valide des données de paramètres de migration de système virtuel.<br/>                                |
| [**MSVM \_ VirtualSystemMigrationNetworkSettingData**](msvm-virtualsystemmigrationnetworksettingdata.md)<br/>                   | Représente le réseau sur lequel le service de migration de système virtuel écoute la migration du système virtuel entrant.<br/>                                                                 |
| [**MSVM \_ VirtualSystemMigrationService**](msvm-virtualsystemmigrationservice.md)<br/>                                         | Représente le service de migration de système virtuel. Il est utilisé pour la migration d’un système virtuel ou pour la migration du stockage d’un système virtuel d’une plate-forme de virtualisation vers une autre.<br/> |
| [**MSVM \_ VirtualSystemMigrationServiceSettingData**](msvm-virtualsystemmigrationservicesettingdata.md)<br/>                   | Représente les paramètres pour le service de migration de système virtuel sur un ordinateur hôte.<br/>                                                                                                              |
| [**MSVM \_ VirtualSystemMigrationServiceSettingDataComponent**](msvm-virtualsystemmigrationservicesettingdatacomponent.md)<br/> | Association utilisée pour représenter les paramètres réseau de migration de système virtuel du service de migration de système virtuel.<br/>                                                                      |
| [**MSVM \_ VirtualSystemMigrationSettingData**](msvm-virtualsystemmigrationsettingdata.md)<br/>                                 | Représente les paramètres de migration pour la migration d’un système virtuel et du stockage attaché à un système virtuel.<br/>                                                                           |
