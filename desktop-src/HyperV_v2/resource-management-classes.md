# <a name="resource-management-classes"></a>Classes de gestion des ressources

Voici les classes WMI de virtualisation associées à la gestion des ressources.

## <a name="in-this-section"></a>Contenu de cette section



| Rubrique                                                                                                        | Description                                                                                                                                                      |
|--------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**MSVM \_ AbstractResourcePoolSettingData**](msvm-abstractresourcepoolsettingdata.md)<br/>             | Représente les paramètres d’une instance [**MSVM \_ ResourcePool**](msvm-resourcepool.md) qui ne sont pas liés à l’allocation.<br/>                                |
| [**MSVM \_ AllocationCapabilities**](msvm-allocationcapabilities.md)<br/>                               | Définit le moyen par lequel un client peut découvrir la plage valide de paramètres par défaut pour une ressource virtuelle.<br/>                                          |
| [**MSVM \_ ElementAllocatedFromPool**](msvm-elementallocatedfrompool.md)<br/>                           | Associe une instance d’une ressource allouée au pool de ressources à partir duquel elle a été allouée.<br/>                                                   |
| [**MSVM \_ ElementCapabilities**](msvm-elementcapabilities.md)<br/>                                     | Représente l’association entre les éléments managés et leurs fonctionnalités.<br/>                                                                           |
| [**MSVM \_ FeatureSettingData**](msvm-featuresettingdata.md)<br/>                                       | Classe abstraite qui représente les paramètres d’une fonctionnalité spécifique d’un système ou d’un composant.<br/>                                                           |
| [**MSVM \_ FeatureSettingsDefineCapabilities**](msvm-featuresettingsdefinecapabilities.md)<br/>         | Fournit un lien entre l’instance des fonctionnalités du commutateur Ethernet et les paramètres minimal, maximal, incrémentiel et par défaut d’une ressource.<br/> |
| [**MSVM \_ ResourceAllocationFromPool**](msvm-resourceallocationfrompool.md)<br/>                       | Associe une instance d’une allocation de ressources à la liste de ressources partagées à partir de laquelle elle est allouée.<br/>                                                    |
| [**MSVM \_ ResourceAllocationSettingData**](msvm-resourceallocationsettingdata.md)<br/>                 | Représente les États d’allocation actuels et enregistrés d’une ressource virtuelle.<br/>                                                                          |
| [**MSVM \_ ResourcePool**](msvm-resourcepool.md)<br/>                                                   | Décrit un type de ressource virtuelle disponible pour une utilisation dans des ordinateurs virtuels.<br/>                                                                           |
| [**MSVM \_ ResourcePoolConfigurationCapabilities**](msvm-resourcepoolconfigurationcapabilities.md)<br/> | Décrit les fonctionnalités de la classe [**MSVM \_ ResourcePoolConfigurationService**](msvm-resourcepoolconfigurationservice.md) associée.<br/>           |
| [**MSVM \_ ResourcePoolConfigurationService**](msvm-resourcepoolconfigurationservice.md)<br/>           | Fournit la gestion active des pools de ressources.<br/>                                                                                                     |
| [**MSVM \_ ResourcePoolSettingData**](msvm-resourcepoolsettingdata.md)<br/>                             | Représente les paramètres d’une instance [**MSVM \_ ResourcePool**](msvm-resourcepool.md) qui ne sont pas liés à l’allocation.<br/>                                |
| [**MSVM \_ SettingsDefineCapabilities**](msvm-settingsdefinecapabilities.md)<br/>                       | Fournit un lien entre l’instance des fonctions et les paramètres minimal, maximal, incrémentiel et par défaut d’une ressource.<br/>                         |
