# <a name="virtual-system-management-service"></a>Service de gestion du système virtuel

Le profil de système virtuel décrit les objets qui composent un ordinateur virtuel. Ces objets incluent le système de base, les périphériques qui composent le système, les paramètres du système et de ses appareils, ainsi que le service de gestion qui effectue des opérations sur le système.
