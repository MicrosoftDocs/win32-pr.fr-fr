# <a name="virtual-system-classes"></a>Classes du système virtuel

Les classes WMI de virtualisation associées à la machine virtuelle sont les suivantes.

## <a name="in-this-section"></a>Contenu de cette section



| Rubrique                                                                                                | Description                                                                                                                                                                                                                                                                                                                                                                                                                            |
|------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**MSVM \_ BootSourceComponent**](msvm-bootsourcecomponent.md)<br/>                             | Associe [**le \_ BootSourceSettingData MSVM**](msvm-bootsourcesettingdata.md) au [**\_ VirtualSystemSettingData MSVM**](msvm-virtualsystemsettingdata.md)global. <br/>                                                                                                                                                                                                                                                   |
| [**MSVM \_ BootSourceSettingData**](msvm-bootsourcesettingdata.md)<br/>                         | Représente les paramètres permettant de définir la source de démarrage d’un ordinateur virtuel. <br/>                                                                                                                                                                                                                                                                                                                                                     |
| [**MSVM \_ ComputerSystem**](msvm-computersystem.md)<br/>                                       | Représente un système d’ordinateur physique ou un ordinateur virtuel.<br/>                                                                                                                                                                                                                                                                                                                                                                   |
| [**MSVM \_ ConcreteComponent**](msvm-concretecomponent.md)<br/>                                 | Association générique utilisée pour établir des relations « partie de » entre des éléments managés.<br/>                                                                                                                                                                                                                                                                                                                                   |
| [**MSVM \_ LogicalIdentity**](msvm-logicalidentity.md)<br/>                                     | Associe deux éléments managés qui représentent différents aspects de la même entité sous-jacente. <br/>                                                                                                                                                                                                                                                                                                                            |
| [**MSVM \_ MostCurrentSnapshotInBranch**](msvm-mostcurrentsnapshotinbranch.md)<br/>             | Associe une instance de la classe [**MSVM \_ ComputerSystem**](msvm-computersystem.md) ou [**MSVM \_ PlannedComputerSystem**](msvm-plannedcomputersystem.md) représentant un système virtuel, avec une instance de la classe [**\_ VirtualSystemSettingData MSVM**](msvm-virtualsystemsettingdata.md) qui représente l’instantané du système virtuel qui est l’instantané le plus récent dans une branche de captures instantanées dépendantes.<br/>            |
| [**MSVM \_ ParentChildSettingData**](msvm-parentchildsettingdata.md)<br/>                       | Association entre une instance de [**MSVM \_ VirtualSystemSettingData**](msvm-virtualsystemsettingdata.md) et l’instance **\_ VirtualSystemSettingData MSVM** qui représente l’instantané le plus récent sur lequel cet objet est basé.<br/>                                                                                                                                                                                 |
| [**MSVM \_ PlannedComputerSystem**](msvm-plannedcomputersystem.md)<br/>                         | Représente un ordinateur virtuel planifié.<br/>                                                                                                                                                                                                                                                                                                                                                                                       |
| [**MSVM \_ SnapshotOfVirtualSystem**](msvm-snapshotofvirtualsystem.md)<br/>                     | Associe un système virtuel à un instantané qui a été capturé à partir du système virtuel.<br/>                                                                                                                                                                                                                                                                                                                                      |
| [**MSVM \_ SummaryInformation**](msvm-summaryinformation.md)<br/>                               | Utilisé dans les méthodes [**GetSummaryInformation**](getsummaryinformation-msvm-virtualsystemmanagementservice.md) et [**GetDefinitionFileSummaryInformation**](getdefinitionfilesummaryinformation-msvm-virtualsystemmanagementservice.md) de la classe [**MSVM \_ VirtualSystemManagementService**](msvm-virtualsystemmanagementservice.md) pour récupérer rapidement les informations communes relatives à un ordinateur virtuel ou à un instantané.<br/> |
| [**MSVM \_ SystemComponent**](msvm-systemcomponent.md)<br/>                                     | Établit une relation « partie de » entre un système et tout élément système géré dont il est composé.<br/>                                                                                                                                                                                                                                                                                                               |
| [**MSVM \_ SystemDevice**](msvm-systemdevice.md)<br/>                                           | Associe un appareil logique au système parent.<br/>                                                                                                                                                                                                                                                                                                                                                                         |
| [**MSVM \_ VirtualSystemCapabilities**](msvm-virtualsystemcapabilities.md)<br/>                 | Décrit les fonctionnalités de l' [**MSVM \_ ComputerSystem**](msvm-computersystem.md)associé.<br/>                                                                                                                                                                                                                                                                                                                           |
| [**MSVM \_ VirtualSystemSettingData**](msvm-virtualsystemsettingdata.md)<br/>                   | Représente les paramètres spécifiques à la virtualisation pour un ordinateur virtuel.<br/>                                                                                                                                                                                                                                                                                                                                                      |
| [**MSVM \_ VirtualSystemSettingDataComponent**](msvm-virtualsystemsettingdatacomponent.md)<br/> | Association générique utilisée pour établir des relations de « partie de » entre une instance [**de \_ VirtualSystemSettingData CIM**](/previous-versions//cc136954(v=vs.85)) et une ou plusieurs instances [**de \_ ResourceAllocationSettingData CIM**](/previous-versions/windows/desktop/clushyperv/cim-resourceallocationsettingdata).<br/>                                                                                                                                                 |
