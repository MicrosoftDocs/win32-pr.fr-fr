# <a name="remote-desktop-classes"></a>Classes Bureau à distance

les classes WMI de Bureau à distance virtualization permettent de gérer les Bureau à distance et les RemoteFX les ressources et les machines virtuelles.

## <a name="in-this-section"></a>Contenu de cette section



| Rubrique                                                                                                            | Description                                                                                                                                                  |
|------------------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**MSVM \_ InteractiveSessionACE**](msvm-interactivesessionace.md)<br/>                                     | Représente une *entrée de contrôle d’accès* (ACE) qui détermine l’accès à la session interactive d’un ordinateur virtuel.<br/>                                |
| [**MSVM \_ Physical3dGraphicsProcessor**](msvm-physical3dgraphicsprocessor.md)<br/>                         | Décrit l’unité de traitement graphique (GPU) 3D physique.<br/>                                                                                        |
| [**MSVM \_ PhysicalGPUInfo**](msvm-physicalgpuinfo.md)<br/>                                                 | contient des informations sur une unité de traitement graphique (GPU) physique RemoteFX.<br/>                                                                    |
| [**MSVM \_ RdvComponent**](msvm-rdvcomponent.md)<br/>                                                       | Représente l’état du composant RDV, qui est chargé de fournir un transport pour le parent à l’invité à des fins de configuration.<br/> |
| [**MSVM \_ RdvComponentSettingData**](msvm-rdvcomponentsettingdata.md)<br/>                                 | Représente l’état configuré du composant de virtualisation des Bureau à distance (RDV). L’État par défaut est activé.<br/>                               |
| [**MSVM \_ Synth3dVideoPool**](msvm-synth3dvideopool.md)<br/>                                               | Contient des informations sur les unités de traitement graphique (GPU) vidéo 3D synthétiques disponibles sur le système hôte.<br/>                                 |
| [**MSVM \_ Synthetic3DDisplayController**](msvm-synthetic3ddisplaycontroller.md)<br/>                       | Représente le contrôleur d’affichage 3D synthétique affecté à un ordinateur virtuel.<br/>                                                            |
| [**MSVM \_ Synthetic3DDisplayControllerSettingData**](msvm-synthetic3ddisplaycontrollersettingdata.md)<br/> | Représente les paramètres d’un contrôleur d’affichage 3D synthétique pour un ordinateur virtuel.<br/>                                                                 |
| [**MSVM \_ Synthetic3DService**](msvm-synthetic3dservice.md)<br/>                                           | Décrit le service de GPU 3D synthétique.<br/>                                                                                                          |
| [**MSVM \_ Synthetic3DServiceSettingData**](msvm-synthetic3dservicesettingdata.md)<br/>                     | Représente les paramètres pour le service synthétique 3D présent sur un système hôte unique.<br/>                                                            |
| [**MSVM \_ TerminalService**](msvm-terminalservice.md)<br/>                                                 | Gère toutes les connexions de terminal distant à un hôte particulier.<br/>                                                                                     |
| [**MSVM \_ TerminalServiceSettingData**](msvm-terminalservicesettingdata.md)<br/>                           | Représente les paramètres pour les services Terminal Server des ordinateurs virtuels sur un ordinateur hôte.<br/>                                                                     |
