# <a name="msvm_resourcepoolconfigurationservice-class"></a>MSVM \_ ResourcePoolConfigurationService, classe

Fournit la gestion active des pools de ressources.

La syntaxe suivante est simplifiée format MOF (MOF) et comprend toutes les propriétés héritées.

## <a name="syntax"></a>Syntaxe

``` syntax
[Dynamic, Provider("VmmsWmiInstanceAndMethodProvider")]
class Msvm_ResourcePoolConfigurationService : CIM_Service
{
  string   InstanceID;
  string   Caption;
  string   Description;
  string   ElementName;
  datetime InstallDate;
  string   Name;
  uint16   OperationalStatus[];
  string   StatusDescriptions[];
  string   Status;
  uint16   HealthState;
  uint16   CommunicationStatus;
  uint16   DetailedStatus;
  uint16   OperatingStatus;
  uint16   PrimaryStatus;
  uint16   EnabledState = 2;
  string   OtherEnabledState;
  uint16   RequestedState = 12;
  uint16   EnabledDefault = 2;
  datetime TimeOfLastStateChange;
  uint16   AvailableRequestedStates[];
  uint16   TransitioningToState;
  string   SystemCreationClassName;
  string   SystemName;
  string   CreationClassName;
  string   PrimaryOwnerName;
  string   PrimaryOwnerContact;
  string   StartMode;
  boolean  Started;
};
```

### <a name="methods"></a>Méthodes

La classe **MSVM \_ ResourcePoolConfigurationService** possède ces méthodes.



| Méthode                                                                                   | Description                                                                                  |
|:-----------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------|
| [**CreatePool**](createpool-msvm-resourcepoolconfigurationservice.md)                   | Crée un pool de ressources enfant.<br/>                                                    |
| [**DeletePool**](deletepool-msvm-resourcepoolconfigurationservice.md)                   | Supprime un pool de ressources.<br/>                                                          |
| [**ModifyPoolResources**](modifypoolresources-msvm-resourcepoolconfigurationservice.md) | Modifie les paramètres de ressource du pool parent pour les ressources affectées à un pool enfant.<br/> |
| [**ModifyPoolSettings**](modifypoolsettings-msvm-resourcepoolconfigurationservice.md)   | Modifie les paramètres d’un pool enfant qui ne sont pas liés à l’allocation.<br/>             |
| **RequestStateChange**                                                                   | Cette méthode n'est pas prise en charge.<br/>                                                     |
| **StartService**                                                                         | Cette méthode n'est pas prise en charge.<br/>                                                     |
| **StopService**                                                                          | Cette méthode n'est pas prise en charge.<br/>                                                     |
