# <a name="msvm_batterysettingdata-class"></a>MSVM \_ BatterySettingData, classe

Représente les paramètres d’un périphérique de batterie virtuelle.

La syntaxe suivante est simplifiée par rapport au code MOF (Managed Object Format) et inclut toutes les propriétés héritées.

## <a name="syntax"></a>Syntaxe

``` syntax
[Dynamic, Provider("VmmsWmiInstanceAndMethodProvider"), AMENDMENT]
class Msvm_BatterySettingData : CIM_ResourceAllocationSettingData
{
};
```

## <a name="members"></a>Membres

La classe **MSVM \_ BatterySettingData** ne définit aucun membre.
