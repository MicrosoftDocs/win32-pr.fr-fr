# <a name="hyper-v-virtual-fibre-channel-api-reference"></a>Informations de référence sur l’API de Fibre Channel virtuel Hyper-V

L’API de Fibre Channel virtuel Hyper-V définit les éléments de programmation suivants.

## <a name="in-this-section"></a>Contenu de cette section



| Rubrique                                                                                    | Description                                                                                                    |
|------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------|
| [**MSVM \_ ExternalFcPort**](msvm-externalfcport.md)<br/>                           | Représente un port de Fibre Channel externe.<br/>                                                          |
| [**MSVM \_ FcActiveConnection**](msvm-fcactiveconnection.md)<br/>                   | Connecte un port de commutateur au point de terminaison Fibre Channel auquel le port est connecté.<br/>                |
| [**MSVM \_ FcDeviceSAPImplementation**](msvm-fcdevicesapimplementation.md)<br/>     | Représente une association entre un point d’accès de service et l’appareil logique qui l’implémente.<br/> |
| [**MSVM \_ FcEndpoint**](msvm-fcendpoint.md)<br/>                                   | Représente le point de connexion logique d’un port de Fibre Channel.<br/>                                   |
| [**MSVM \_ FcPortAllocationSettingData**](msvm-fcportallocationsettingdata.md)<br/> | Représente l’état configuré d’un port de Fibre Channel synthétique ou d’un port de commutateur Fibre Channel.<br/>   |
| [**MSVM \_ FcSwitchPort**](msvm-fcswitchport.md)<br/>                               | Représente un port sur le commutateur de Fibre Channel virtuel.<br/>                                              |
| [**MSVM \_ SyntheticFcPort**](msvm-syntheticfcport.md)<br/>                         | Représente un port de Fibre Channel synthétique.<br/>                                                          |
| [**MSVM \_ SyntheticFcPortSettingData**](msvm-syntheticfcportsettingdata.md)<br/>   | Représente l’état configuré d’un port de Fibre Channel synthétique.<br/>                                  |
| [**MSVM \_ VirtualFcSwitch**](msvm-virtualfcswitch.md)<br/>                         | Représente un commutateur de Fibre Channel virtuel.<br/>                                                          |
| [**MSVM \_ VirtualFcSwitchSettingData**](msvm-virtualfcswitchsettingdata.md)<br/>   | Représente la configuration d’un commutateur de Fibre Channel virtuel.<br/>                                     |
