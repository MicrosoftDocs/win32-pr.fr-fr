# <a name="hyper-v-application-health-monitoring-wmi-classes"></a>Classes WMI de surveillance de l’intégrité des applications Hyper-V

L’API de surveillance de l’intégrité des applications Hyper-V définit les classes WMI suivantes.

## <a name="in-this-section"></a>Contenu de cette section



| Rubrique                                                                                        | Description                                                                                                                                                                 |
|----------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**MSVM \_ HeartbeatComponent**](msvm-heartbeatcomponent.md)<br/>                       | Représente l’état du service de pulsation, qui est responsable de la surveillance de l’état d’un ordinateur virtuel en signalant une pulsation à intervalles réguliers.<br/> |
| [**MSVM \_ HeartbeatComponentSettingData**](msvm-heartbeatcomponentsettingdata.md)<br/> | Représente l’état configuré du service de pulsation.<br/>                                                                                                        |
