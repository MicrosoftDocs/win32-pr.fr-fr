# <a name="hyper-v-replication-api-reference"></a>Informations de référence sur l’API de réplication Hyper-V

L’API de réplication Hyper-V définit les éléments de programmation suivants.

## <a name="in-this-section"></a>Contenu de cette section



| Rubrique                                                                                                    | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
|----------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**MSVM \_ FailoverNetworkAdapterSettingData**](msvm-failovernetworkadaptersettingdata.md)<br/>     | Représente les paramètres d’une carte réseau au sein du système d’exploitation invité, qui seront appliqués au moment du basculement.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
| [**MSVM \_ ReplicationProvider**](msvm-replicationprovider.md)<br/>                                 | Représente les fournisseurs disponibles pour la réplication. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| [**MSVM \_ ReplicationRelationship**](msvm-replicationrelationship.md)<br/>                         | Représente l’état de la réplication pour une relation de réplication. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| [**MSVM \_ ReplicaSystemDependency**](msvm-replicasystemdependency.md)<br/>                         | Représente une association entre une instance de la [**classe \_ CIM CIM**](/windows/desktop/CIMWin32Prov/cim-computersystem) qui représente le réplica d’ordinateur virtuel et une instance de la classe **\_ ComputerSystem CIM** qui représente le réplica de machine virtuelle de test.<br/>                                                                                                                                                                                                                                                                                                                                 |
| [**MSVM \_ ReplicationAuthorizationSettingData**](msvm-replicationauthorizationsettingdata.md)<br/> | Représente une entrée d’autorisation pour un serveur de récupération.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| [**MSVM \_ ReplicationService**](msvm-replicationservice.md)<br/>                                   | Gère la réplication d’un ordinateur virtuel.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| [**MSVM \_ ReplicationServiceSettingData**](msvm-replicationservicesettingdata.md)<br/>             | Représente les paramètres pour le service de réplication sur un hôte de récupération. Les propriétés de cette classe ne peuvent pas être modifiées directement. Le client doit appeler la méthode [**MSVM \_ ReplicationService. ModifyServiceSettings**](modifyservicesettings-msvm-replicationservice.md) pour modifier l’une de ces propriétés.<br/>                                                                                                                                                                                                                                                                                        |
| [**MSVM \_ ReplicationSettingData**](msvm-replicationsettingdata.md)<br/>                           | Représente les paramètres spécifiques à la réplication pour un ordinateur virtuel. Le client passe une instance de cette classe à [**MSVM \_ ReplicationService. CreateReplicationRelationship**](createreplicationrelationship-msvm-replicationservice.md) pour créer une relation de réplication. Le client ne peut pas modifier directement les valeurs des propriétés de cette classe ; il doit appeler la méthode [**MSVM \_ ReplicationService. ModifyReplicationSettings**](modifyreplicationsettings-msvm-replicationservice.md) pour modifier les valeurs. Chaque relation de réplication possède une seule instance de paramètres.<br/> |
| [**MSVM \_ ReplicationStatistics**](msvm-replicationstatistics.md)<br/>                             | Fournit des statistiques de réplication pour un ordinateur virtuel.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| [**MSVM \_ SystemReplicationRelationship**](msvm-systemreplicationrelationship.md)<br/>             | Représente une association entre une instance de [**MSVM \_ ComputerSystem**](msvm-computersystem.md) qui représente l’ordinateur virtuel et une instance de [**MSVM \_ ReplicationRelationship**](msvm-replicationrelationship.md) qui représente une relation de réplication de l’ordinateur virtuel. <br/>                                                                                                                                                                                                                                                                                                |
