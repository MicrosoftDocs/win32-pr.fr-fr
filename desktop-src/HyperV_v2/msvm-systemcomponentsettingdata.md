# <a name="msvm_systemcomponentsettingdata-class"></a>MSVM \_ SystemComponentSettingData, classe

Classe de base générique pour définir des classes de données qui représentent les composants d’un système virtuel.

La syntaxe suivante est simplifiée par rapport au code MOF (Managed Object Format) et inclut toutes les propriétés héritées.

## <a name="syntax"></a>Syntaxe

``` syntax
[Abstract, AMENDMENT]
class Msvm_SystemComponentSettingData : CIM_SettingData
{
};
```

## <a name="members"></a>Membres

La classe **MSVM \_ SystemComponentSettingData** ne définit aucun membre.
