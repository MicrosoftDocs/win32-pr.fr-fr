# <a name="profile-registration-classes"></a>Classes d’inscription de profil

Voici les classes WMI de virtualisation associées à l’inscription de profil.

## <a name="in-this-section"></a>Contenu de cette section



| Rubrique                                                                                                    | Description                                                                                                                                                                      |
|----------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**MSVM \_ ElementConformsToProfile**](msvm-elementconformstoprofile.md)<br/>                       | Définit les profils enregistrés conformes au système référencé.<br/>                                                                                              |
| [**MSVM \_ HostedResourcePool**](msvm-hostedresourcepool.md)<br/>                                   | Représente une spécialisation de l’Association de composants système qui établit que le pool de ressources est défini dans le contexte du système.<br/>                      |
| [**MSVM \_ ReferencedProfile**](msvm-referencedprofile.md)<br/>                                     | Décrit un profil qui est référencé par un autre profil enregistré.<br/>                                                                                                 |
| [**MSVM \_ RegisteredProfile**](msvm-registeredprofile.md)<br/>                                     | Décrit un ensemble de classes avec les propriétés et les méthodes requises, nécessaires pour gérer une entité réelle ou pour prendre en charge un scénario d’utilisation, de manière interopérable.<br/> |
| [**MSVM \_ ResourcePoolComponent**](msvm-resourcepoolcomponent.md)<br/>                             | Représente un élément de pool de ressources de la plateforme Microsoft Windows Hyper-V.<br/>                                                                                         |
| [**MSVM \_ ResourcePoolRegistration**](msvm-resourcepoolregistration.md)<br/>                       | Inscrit un service qui fournit des objets liés aux pools de ressources globaux.<br/>                                                                                               |
| [**MSVM \_ ResourceTypeDefinition**](msvm-resourcetypedefinition.md)<br/>                           | Définit un mappage d’un type de ressource à ses classes d’implémentation.<br/>                                                                                                   |
| [**MSVM \_ VirtualizationComponent**](msvm-virtualizationcomponent.md)<br/>                         | Représente un service de la plateforme Microsoft Windows Hyper-V.<br/>                                                                                                       |
| [**MSVM \_ VirtualizationComponentRegistration**](msvm-virtualizationcomponentregistration.md)<br/> | Représente l’inscription d’un service dans la plateforme Microsoft Hyper-V.<br/>                                                                                           |
| [**MSVM \_ VirtualSystemResourceComponent**](msvm-virtualsystemresourcecomponent.md)<br/>           | Représente un service d’appareil virtuel de la plateforme Microsoft Windows Hyper-V.<br/>                                                                                        |
| [**MSVM \_ VirtualSystemResourceRegistration**](msvm-virtualsystemresourceregistration.md)<br/>     | Inscrit un service qui fournit des objets liés aux ressources propres à l’ordinateur virtuel.<br/>                                                                                  |
