# <a name="memory-classes"></a>Classes de mémoire

Les classes de mémoire modélisent la mémoire pour un ordinateur virtuel.

Les classes WMI de virtualisation associées à la mémoire sont les suivantes.

## <a name="in-this-section"></a>Contenu de cette section



| Rubrique                                                                                      | Description                                                                                                         |
|--------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------|
| [**MSVM \_ ElementAllocatedFromNumaNode**](msvm-elementallocatedfromnumanode.md)<br/> | Associe une instance d’une ressource allouée au nœud NUMA physique à partir duquel elle a été allouée.<br/> |
| [**\_Mémoire MSVM**](msvm-memory.md)<br/>                                             | Représente la mémoire actuellement allouée à un ordinateur virtuel.<br/>                                          |
| [**MSVM \_ MemorySettingData**](msvm-memorysettingdata.md)<br/>                       | Représente l’état configuré de la mémoire pour un ordinateur virtuel.<br/>                                     |
| [**MSVM \_ NumaNode**](msvm-numanode.md)<br/>                                         | Représente un nœud NUMA (non-Uniform Memory Access) d’un système virtuel.<br/>                                  |
