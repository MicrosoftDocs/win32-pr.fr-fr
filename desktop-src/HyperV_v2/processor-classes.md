# <a name="processor-classes"></a>Classes de processeur

Les classes de processeur modélisent un processeur pour un ordinateur virtuel.

Les classes WMI de virtualisation associées au processeur sont les suivantes.

## <a name="in-this-section"></a>Contenu de cette section



| Rubrique                                                                      | Description                                                                               |
|----------------------------------------------------------------------------|-------------------------------------------------------------------------------------------|
| [**\_Processeur MSVM**](msvm-processor.md)<br/>                       | Représente le processeur virtuel sur un ordinateur virtuel.<br/>                         |
| [**MSVM \_ ProcessorPool**](msvm-processorpool.md)<br/>               | Agrège les ressources du processeur qui peuvent être allouées à un ordinateur virtuel.<br/> |
| [**MSVM \_ ProcessorSettingData**](msvm-processorsettingdata.md)<br/> | Représente les paramètres de processeur virtuel pour un ordinateur virtuel.<br/>               |
