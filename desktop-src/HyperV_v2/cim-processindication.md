# <a name="cim_processindication-class"></a>\_Classe CIM ProcessIndication

**CIM \_ ProcessIndication** est une superclasse abstraite pour les classes d’indication spécialisées, qui adresse des modifications spécifiques et des alertes publiées par les fournisseurs et l’instrumentation.

## <a name="syntax"></a>Syntaxe

``` syntax
[Indication, Version("2.6.0"), UMLPackagePath("CIM::Event"), AMENDMENT]
class CIM_ProcessIndication : CIM_Indication
{
};
```

## <a name="members"></a>Membres

La classe **CIM \_ ProcessIndication** ne définit aucun membre.
