# <a name="benefits-of-windows-certification"></a>avantages de la certification Windows

lorsque vous certifiez votre Windows 8.1 ou Windows 8 des applications de bureau, vous êtes éligible à :

-   participez à plusieurs milliards de dollars Windows le marché des logiciels et la plus grande version de Windows
-   être proposé comme compatible dans le rapport de compatibilité de l’installation Windows
-   figurer dans le magasin de Windows, avec des millions de clients potentiels à travers le monde (Windows 8.1 uniquement)
-   Avoir accès aux ressources marketing et à la formation à partir du Microsoft Partner Network
-   obtenez des logos de certification pour identifier facilement votre application comme haute qualité et Windows compatible
-   Donnez à vos clients la certitude de la qualité de votre produit
