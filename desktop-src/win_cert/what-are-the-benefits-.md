# <a name="benefits-of-windows-certification"></a>Avantages de la certification Windows

Lorsque vous certifiez vos applications de bureau Windows 8.1 ou Windows 8, vous êtes éligible à :

-   Participez au marché des logiciels Windows à plusieurs milliards de dollars et à la plus grande version de Windows
-   Être proposé comme compatible dans le rapport de compatibilité de l’installation de Windows
-   Figurer dans le Windows Store, avec des millions de clients potentiels à travers le monde (Windows 8.1 uniquement)
-   Avoir accès aux ressources marketing et à la formation à partir du Microsoft Partner Network
-   Obtenez des logos de certification pour identifier facilement votre application en tant que haute qualité et compatible avec Windows
-   Donnez à vos clients la certitude de la qualité de votre produit
