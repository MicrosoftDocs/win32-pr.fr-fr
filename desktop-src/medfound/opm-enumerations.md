# <a name="opm-enumerations"></a>Énumérations OPM

Le [Gestionnaire de protection de sortie](output-protection-manager.md) (OPM) utilise les constantes et types d’énumération suivants.



| Énumération                                                                    | Description                                                                                                                                                                             |
|--------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Indicateurs de type de bus OPM**](opm-bus-type-flags.md)                               | Spécifie le type de bus d’e/s utilisé par la carte graphique.                                                                                                                             |
| [**Indicateurs de type de connecteur OPM**](opm-connector-type-flags.md)                   | Spécifie le type de connecteur physique pour une sortie vidéo.                                                                                                                            |
| [**Indicateurs de type de protection OPM**](opm-protection-type-flags.md)                 | Spécifie des mécanismes de protection de sortie pour la sortie de protection Manager (OPM).                                                                                                             |
| [**Indicateurs d’État OPM**](opm-status-flags.md)                                   | Spécifie l’état d’une session OPM.                                                                                                                                                 |
| [**niveau de protection de l' \_ ACP OPM \_ \_**](/windows/desktop/api/opmapi/ne-opmapi-opm_acp_protection_level)                | Spécifie le niveau de protection pour la protection contre la copie analogique (ACP).                                                                                                                        |
| [**\_niveau de \_ protection \_ DPCP OPM**](/windows/desktop/api/opmapi/ne-opmapi-opm_dpcp_protection_level)              | Spécifie le niveau de protection pour le protection du contenu DisplayPort (DPCP).                                                                                                               |
| [**niveau de protection pour OPM \_ HDCP \_ \_**](/windows/desktop/api/opmapi/ne-opmapi-opm_hdcp_protection_level)              | Spécifie le niveau de protection pour High-Bandwidth protection du contenu numérique (HDCP).                                                                                                    |
| [**\_Rapport hauteur/largeur de l’image OPM \_ \_ \_ EN300294**](/windows/desktop/api/opmapi/ne-opmapi-opm_image_aspect_ratio_en300294) | Spécifie les proportions pour ETSI en 300 294.                                                                                                                                         |
| [**\_sémantique de \_ sortie \_ vidéo OPM**](/windows/desktop/api/opmapi/ne-opmapi-opm_video_output_semantics)            | Spécifie si l’interface [**IOPMVideoOutput**](/windows/desktop/api/opmapi/nn-opmapi-iopmvideooutput) utilise la sémantique de sortie Protection Manager (OPM) ou la sémantique de la protection de sortie certifiée (COPP). |
