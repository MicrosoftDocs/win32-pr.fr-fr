# <a name="i-have-an-avi-file-containing-a-windows-media-video-stream-how-can-i-convert-it-to-a-wmv-file"></a>je dispose d’un fichier AVI contenant un flux de Windows Media Video. Comment puis-je le convertir en fichier WMV ?

les interfaces de codec vidéo et Windows Media Audio ne fournissent pas de méthodes pour créer des fichiers à l’aide du format ASF (Advanced Systems format), qui est le format utilisé pour les fichiers WMV. si vous souhaitez convertir un fichier (à l’aide d’un conteneur) en fichier ASF, vous devez utiliser les objets du kit de développement logiciel (SDK) du Format de média Windows.

récupérez les exemples de supports compressés Windows à partir du fichier source et transmettez-les à l’objet writer en tant qu’exemples de flux. pour plus d’informations, consultez la documentation du kit de développement logiciel (SDK) Windows Media Format 9 Series.
