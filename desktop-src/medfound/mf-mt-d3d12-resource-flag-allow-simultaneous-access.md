# <a name="mf_mt_d3d12_resource_flag_allow_simultaneous_access-attribute"></a>\_Indicateur de \_ ressource D3D12 MF MT \_ \_ \_ allow \_ simultanément \_

Indique si les ressources Direct3D dans le flux peuvent être accessibles simultanément par plusieurs files d’attente de commandes différentes.

## <a name="data-type"></a>Type de données

**UINT32**

## <a name="remarks"></a>Notes

La valeur de cet attribut correspond au membre D3D12_RESOURCE_FLAG_ALLOW_SIMULTANEOUS_ACCESS de l’énumération [D3D12_RESOURCE_FLAGS](/windows/win32/api/d3d12/ne-d3d12-d3d12_resource_flags) .
