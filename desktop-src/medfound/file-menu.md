# <a name="file-menu"></a>Menu Fichier

Le menu **fichier** fournit les commandes suivantes :




| Commande | Description | 
|---------|-------------|
| <strong>Nouveau...</strong> | Efface la topologie existante et supprime les nœuds de topologie du <strong>volet topologie</strong>.<br /><ul><li>Si une topologie est actuellement ouverte, invite l’utilisateur à enregistrer la topologie. Pour plus d’informations sur l’enregistrement de la topologie, consultez <a href="saving-and-loading-topologies.md">enregistrement et chargement de topologies</a>.<br /></li><li>Crée une topologie vide. L’utilisateur peut ajouter des nœuds de topologie à la nouvelle topologie. Pour plus d’informations, consultez <a href="building-topologies-by-using-topoedit.md">création de topologies à l’aide de TopoEdit</a>.<br /></li></ul> | 
| <strong>Fichier de rendu...</strong> | Crée une topologie de lecture pour un fichier multimédia. TopoEdit crée et connecte automatiquement les nœuds dans la topologie. Pour plus d’informations, consultez <a href="building-a-playback-topology-with-topoedit.md">génération d’une topologie de lecture avec TopoEdit</a>. | 
| <strong>Restituer le transcodage...</strong> | Crée une topologie de transcodage. Pour plus d’informations, consultez <a href="building-a-transcode-topology-with-topoedit.md">génération d’une topologie de transcodage avec TopoEdit</a>.<br /><blockquote>[!Note]<br />cette fonctionnalité requiert Windows 7.</blockquote><br /> | 
| <strong>Ouvrir...</strong> | Charge une topologie existante enregistrée sous la forme d’un fichier XML. Consultez <a href="saving-and-loading-topologies.md">enregistrement et chargement de topologies</a>. | 
| <strong>Enregistrer...</strong> | Enregistre la topologie actuellement ouverte sous la forme d’un fichier XML. Consultez <a href="saving-and-loading-topologies.md">enregistrement et chargement de topologies</a>. | 
| <strong>Quitter</strong> | Ferme l’application TopoEdit. |
