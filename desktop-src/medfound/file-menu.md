# <a name="file-menu"></a>Menu Fichier

Le menu **fichier** fournit les commandes suivantes :



<table>
<colgroup>
<col style="width: 50%" />
<col style="width: 50%" />
</colgroup>
<thead>
<tr class="header">
<th>Commande</th>
<th>Description</th>
</tr>
</thead>
<tbody>
<tr class="odd">
<td><strong>Nouveau...</strong></td>
<td>Efface la topologie existante et supprime les nœuds de topologie du <strong>volet topologie</strong>.<br/>
<ul>
<li>Si une topologie est actuellement ouverte, invite l’utilisateur à enregistrer la topologie. Pour plus d’informations sur l’enregistrement de la topologie, consultez <a href="saving-and-loading-topologies.md">enregistrement et chargement de topologies</a>.<br/></li>
<li>Crée une topologie vide. L’utilisateur peut ajouter des nœuds de topologie à la nouvelle topologie. Pour plus d’informations, consultez <a href="building-topologies-by-using-topoedit.md">création de topologies à l’aide de TopoEdit</a>.<br/></li>
</ul></td>
</tr>
<tr class="even">
<td><strong>Fichier de rendu...</strong></td>
<td>Crée une topologie de lecture pour un fichier multimédia. TopoEdit crée et connecte automatiquement les nœuds dans la topologie. Pour plus d’informations, consultez <a href="building-a-playback-topology-with-topoedit.md">génération d’une topologie de lecture avec TopoEdit</a>.</td>
</tr>
<tr class="odd">
<td><strong>Restituer le transcodage...</strong></td>
<td>Crée une topologie de transcodage. Pour plus d’informations, consultez <a href="building-a-transcode-topology-with-topoedit.md">génération d’une topologie de transcodage avec TopoEdit</a>.<br/>
<blockquote>
[!Note]<br />
Cette fonctionnalité nécessite Windows 7.
</blockquote>
<br/></td>
</tr>
<tr class="even">
<td><strong>Ouvrir...</strong></td>
<td>Charge une topologie existante enregistrée sous la forme d’un fichier XML. Consultez <a href="saving-and-loading-topologies.md">enregistrement et chargement de topologies</a>.</td>
</tr>
<tr class="odd">
<td><strong>Enregistrer...</strong></td>
<td>Enregistre la topologie actuellement ouverte sous la forme d’un fichier XML. Consultez <a href="saving-and-loading-topologies.md">enregistrement et chargement de topologies</a>.</td>
</tr>
<tr class="even">
<td><strong>Quitter</strong></td>
<td>Ferme l’application TopoEdit.</td>
</tr>
</tbody>
</table>
