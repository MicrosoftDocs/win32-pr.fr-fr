# <a name="mf_mt_d3d_resource_version-attribute"></a>Attribut de version de la \_ \_ ressource D3D MF MT \_ \_

Spécifie la version Direct3D des ressources stockées dans le flux de données associé au type de média. 

## <a name="data-type"></a>Type de données

**UINT32**

## <a name="remarks"></a>Notes

La valeur de cet attribut correspond à un membre de l’énumération [MF_MT_D3D_RESOURCE_VERSION_ENUM](/windows/win32/api/mfd3d12/ne-mfd3d12-mf_mt_d3d_resource_version_enum) .
