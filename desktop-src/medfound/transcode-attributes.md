# <a name="transcode-attributes"></a>Attributs de transcodage

Les attributs suivants sont utilisés avec l’API de transcodage.



| Attribut                                                                          | Description                                                                                  |
|------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------|
| [\_profil d’ajustement de transcodage MF \_ \_](mf-transcode-adjust-profile.md)                  | Indicateurs de profil qui définissent les paramètres de flux pour la topologie de transcodage.                    |
| [\_CONTAINERTYPE de transcodage MF \_](mf-transcode-containertype.md)                     | Spécifie le type de conteneur d’un fichier encodé.                                             |
| [\_ \_ \_ encodeur d’insertion ne de transcodage MF \_](mf-transcode-donot-insert-encoder.md)     | Spécifie si un encodeur doit être inclus dans la topologie de transcodage.                     |
| [\_ENCODINGPROFILE de transcodage MF \_](mf-transcode-encodingprofile.md)                 | Spécifie le profil de conformité de périphérique pour l’encodage des fichiers ASF. |
| [\_QUALITYVSSPEED de transcodage MF \_](mf-transcode-qualityvsspeed.md)                   | Spécifie le compromis entre la qualité d’encodage et la vitesse d’encodage.                          |
| [\_transfert de \_ \_ métadonnées d’omission de transcodage MF \_](mf-transcode-skip-metadata-transfer.md) | Spécifie si les métadonnées sont écrites dans le fichier transcodé.                                |
| [**\_TOPOLOGYMODE de transcodage MF \_**](/windows/desktop/api/mfidl/ne-mfidl-mf_transcode_topologymode_flags)             | Spécifie si le chargeur de topologie chargera les transformations matérielles.                   |
