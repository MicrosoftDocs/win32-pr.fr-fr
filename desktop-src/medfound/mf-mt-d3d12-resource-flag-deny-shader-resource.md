# <a name="mf_mt_d3d12_resource_flag_deny_shader_resource-attribute"></a>\_Indicateur de \_ ressource D3D12 MF MT Deny-attribut de \_ ressource de \_ \_ \_ nuanceur \_

Indique si la création d’une vue de ressource de nuanceur n’est pas autorisée pour les ressources Direct3D dans le flux associé au type de média. 

## <a name="data-type"></a>Type de données

**UINT32**

## <a name="remarks"></a>Notes

La valeur de cet attribut correspond au membre D3D12_RESOURCE_FLAG_DENY_SHADER_RESOURCE de l’énumération [D3D12_RESOURCE_FLAGS](/windows/win32/api/d3d12/ne-d3d12-d3d12_resource_flags) .
