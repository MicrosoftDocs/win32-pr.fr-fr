# <a name="mfpkey_dyn_bandtrunc_qfloor-property"></a>MFPKEY \_ dyn \_ BANDTRUNC \_ QFLOOR, propriété

Spécifie la qualité à laquelle la bande passante codée maximale est autorisée. À une qualité supérieure ou égale à la valeur de cette propriété, la bande passante codée maximale est autorisée.

## <a name="constant-for-ipropertybag"></a>Constante pour IPropertyBag

Disponible uniquement à l’aide de [**IPropertyStore**](/windows/win32/api/propsys/nn-propsys-ipropertystore).

## <a name="data-type"></a>Type de données

**VT \_ R4**
