# <a name="mf_devicestream_stream_category-attribute"></a>\_Attribut de \_ catégorie de flux DEVICESTREAM MF \_

Représente la catégorie de flux.

## <a name="data-type"></a>Type de données

**GUID**

## <a name="remarks"></a>Notes

Cet attribut représente le GUID qui identifie la catégorie de flux définie dans ksmedia. h.
