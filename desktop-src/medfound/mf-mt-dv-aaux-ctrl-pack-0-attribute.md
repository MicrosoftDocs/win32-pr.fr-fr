# <a name="mf_mt_dv_aaux_ctrl_pack_0-attribute"></a>\_Attribut de \_ l' \_ \_ attribut Ctrl \_ Pack \_ 0 de l’AAUX MF-DV

Pack de contrôle source AAUX (audio auxiliaire) pour le premier bloc audio dans un type de média Digital Video (DV).

## <a name="data-type"></a>Type de données

**UINT32**

## <a name="remarks"></a>Remarques

cet attribut correspond au membre **dwDVAAuxCtl** de la structure [**DVINFO**](/windows/win32/api/strmif/ns-strmif-dvinfo) DirectShow.

La constante GUID de cet attribut est exportée à partir de mfuuid. lib.
