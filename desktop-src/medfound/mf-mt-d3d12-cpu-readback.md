# <a name="mf_mt_d3d12_cpu_readback-attribute"></a>Attribut READBACK de l’UC MF \_ MT \_ D3D12 \_ \_

Indique si l’accès au processeur est requis pour les ressources Direct3D associées. 

## <a name="data-type"></a>Type de données

**UINT32**

## <a name="remarks"></a>Notes

La valeur par défaut de cet attribut est 0 (FALSe).
