## <a name="media-sharing-properties"></a>Propriétés du partage de médias

Pour qu’un fichier multimédia soit compatible avec la fonctionnalité de partage de médias, le gestionnaire de propriétés doit exposer les propriétés de métadonnées suivantes. Ces propriétés permettent au service de partage de médias d’offrir les options appropriées pour Transcoder le contenu dans différents formats ou vitesses de transmission.

-   [\_ID de \_ \_ Profil DLNA de contenu \_ MFPKEY](mfpkey-content-dlna-profile-id.md)
-   **\_ChannelCount de contenu \_**
-   **\_EncodingBitrate de contenu \_**
-   **\_Format audio du modèle de contenu \_**
-   A-la **\_ \_Samplerate audio** (facultatif)
-   A-la **\_ \_Échantillonnage audio** (facultatif)
-   A-la **\_ DRM \_ IsProtected** (requis pour le contenu DRM)
-   **Durée du média de la \_ bandex \_**
-   **Compression vidéo de la carte de contenu \_ \_**
-   **\_EncodingBitrate, vidéo \_**
-   **Vidéo de la- \_ carte vidéo \_ FourCC**
-   **\_FrameHeight, vidéo \_**
-   A-la **\_ \_Fréquence vidéo** (facultative)
-   **\_FrameWidth, vidéo \_**
-   **\_TotalBitrate, vidéo \_**

La **propriété \_ \_ IsProtected** de la propriété hyperproperty est requise si le contenu est protégé à l’aide de DRM. Dans le cas contraire, cette propriété est facultative.

Les **propriétés \_ \_ samplerate audio** de la carte son, de l' **\_ \_ échantillonneur audio** et de la vidéo de la **\_ carte \_** de son, sont facultatives. Le service de partage de médias les expose s’ils sont disponibles.

Les propriétés dans **le \_ \_ \* *groupe _ hyperson _ _ son s’appliquent uniquement aux fichiers avec un flux audio, et les propriétés dans le groupe de* \_ vidéos \_ \* _** hypergroupements s’appliquent uniquement aux fichiers avec un flux vidéo.
