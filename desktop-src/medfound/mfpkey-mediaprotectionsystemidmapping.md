# <a name="mfpkey_mediaprotectionsystemidmapping-property"></a>MFPKEY \_ propriété MediaProtectionSystemIdMapping

Spécifie un [**IPropertySet**](/uwp/api/Windows.Foundation.Collections.IPropertySet?view=winrt-19041) qui définit le mappage d’un ID de système de propriétés à un ID d’activation du système de propriétés.



Type de données

Type de PROPVARIANT (VT)

Membre PROPVARIANT

**IUNKNOWN**

VT \_ inconnu

**punkVal**



## <a name="remarks"></a>Notes

Cette clé de propriété est utilisée pour passer un [**IPropertySet**](/uwp/api/Windows.Foundation.Collections.IPropertySet?view=winrt-19041) qui contient des mappages entre les GUID du système de protection et les chaînes de la classe Runtime à une source multimédia.
