# <a name="mf_sa_d3d12_heap_flags-attribute"></a>\_ \_ \_ Attribut flags de segment de mémoire D3D12 MF sa \_

Contient une valeur qui spécifie les options de tas utilisées pour les ressources Direct3D dans le flux.

## <a name="data-type"></a>Type de données

**UINT32**

## <a name="remarks"></a>Notes

La valeur de cet attribut est une combinaison or au niveau du bit des valeurs de l’énumération [D3D12_HEAP_FLAGS](/windows/win32/api/d3d12/ne-d3d12-d3d12_heap_flags) .
