# <a name="codecapi_avdecsoftwaredynamicformatchange-property"></a>CODECAPI \_ propriété AVDecSoftwareDynamicFormatChange

Définit s’il faut utiliser le changement de format logiciel dynamique pour le redimensionnement interne.

## <a name="data-type"></a>Type de données

**Boolean**

## <a name="property-guid"></a>Guid de propriété

**CODECAPI \_ AVDecSoftwareDynamicFormatChange**

## <a name="property-value"></a>Valeur de la propriété

Si la valeur est **true**, le changement de format logiciel dynamique est utilisé pour le redimensionnement interne.
