# <a name="media-session-attributes"></a>Attributs de session multimédia

Les attributs suivants sont utilisés pour configurer la session multimédia.



| Attribut                                                                                            | Description                                                                                                 |
|------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------|
| [**\_Gestionnaire de \_ protection de contenu de session MF \_ \_**](mf-session-content-protection-manager-attribute.md) | Fournit une interface de rappel pour que l’application reçoive un objet activateur de contenu à partir de la session PMP. |
| [**\_ \_ heure globale de session MF \_**](mf-session-global-time-attribute.md)                                | Spécifie si les topologies auront une heure de début et de fin globale.                                        |
| [**\_Gestionnaire de \_ qualité de session MF \_**](mf-session-quality-manager-attribute.md)                        | Contient le CLSID d’un gestionnaire de qualité pour la session multimédia.                                              |
| [**\_ \_ mode source à distance de session MF \_ \_**](mf-session-remote-source-mode-attribute.md)                 | Spécifie que la source du média s’exécute dans un processus distant.                                             |
| [**\_contexte du \_ serveur de session MF \_**](mf-session-server-context-attribute.md)                          | Permet à deux instances de la session multimédia de partager le même processus PMP.                                   |
| [**\_TOPOLOADER de session MF \_**](mf-session-topoloader-attribute.md)                                   | Contient le CLSID d’un chargeur de topologie pour la session multimédia.                                              |
