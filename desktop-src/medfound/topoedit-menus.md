# <a name="topoedit-menus"></a>Menus TopoEdit

Le tableau suivant décrit les menus TopoEdit :



| Menu                               | Description                                                                    |
|------------------------------------|--------------------------------------------------------------------------------|
| [Menu fichier](file-menu.md)         | Ouvre une topologie nouvelle ou existante.                                              |
| [Menu Topology](topology-menu.md) | Ajoute des nœuds de source, de transformation et de sortie. Résout et supprime les nœuds de topologie. |
| [Menu contrôles](controls-menu.md) | Lit, suspend et arrête une topologie résolue.                                  |
| [Menu outils](tools-menu.md)       | Journalise les activités d’un nœud de topologie sélectionné.                                   |
| [Menu aide](help-menu.md)         | Fournit un lien vers les rubriques d’aide de TopoEdit.                                   |
