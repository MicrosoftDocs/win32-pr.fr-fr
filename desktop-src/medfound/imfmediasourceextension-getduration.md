# <a name="imfmediasourceextensiongetduration-method"></a>IMFMediaSourceExtension :: GetDuration, méthode

Obtient la durée de la source du média en unités de 100 nanosecondes.

## <a name="syntax"></a>Syntaxe


```C++
double GetDuration();
```



## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.

## <a name="return-value"></a>Valeur retournée

Durée de la source du média en unités de 100 nanosecondes.
