# <a name="mf_devicestream_frameserver_shared-attribute"></a>\_ \_ Attribut partagé DEVICESTREAM FRAMESERVER MF \_

Lorsqu’il est défini sur un flux, cet attribut marque explicitement le flux comme étant partagé par le serveur de frames.

## <a name="data-type"></a>Type de données
