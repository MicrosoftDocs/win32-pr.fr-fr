# <a name="topoedit-main-window"></a>Fenêtre principale TopoEdit

Après le lancement de TopoEdit, la fenêtre illustrée dans la capture d’écran suivante s’affiche.

![capture d’écran montrant la fenêtre principale topoedit](images/809bce6e-43b7-4399-a41b-deff894ba151.jpg)

Les éléments de la fenêtre principale TopoEdit sont les suivants :

-   **Volet de topologie**. S’affiche à gauche de la fenêtre. Cette zone est utilisée pour ajouter des nœuds de topologie, afficher une topologie et connecter les nœuds de topologie. Pour plus d’informations sur l’utilisation de ce volet, consultez [création de topologies à l’aide de TopoEdit](building-topologies-by-using-topoedit.md).
-   **Volet attributs**. S’affiche à droite de la fenêtre. Ce volet affiche les attributs du nœud de topologie actuellement sélectionné. Pour plus d’informations sur les attributs qui sont affichés dans ce volet, consultez [affichage des informations de topologie](viewing-topology-information.md).
-   Menus. Consultez les [menus TopoEdit](topoedit-menus.md).
-   Barre d'outils. Consultez la [barre d’outils TopoEdit](topoedit-toolbar.md).
