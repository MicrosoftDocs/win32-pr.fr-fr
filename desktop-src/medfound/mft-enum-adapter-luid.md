# <a name="mft_enum_adapter_luid-attribute"></a>\_Attribut LUID de l’adaptateur d’énumération MFT \_ \_

Spécifie l’identificateur unique d’une carte vidéo. Utilisez cet attribut lors de l’appel de [**MFTEnum2**](/windows/desktop/api/mfapi/nf-mfapi-mftenum2) pour énumérer les MFTS associés à un adaptateur spécifique.

## <a name="data-type"></a>Type de données

**[**LUID**](/windows/win32/api/devicetopology/ns-devicetopology-luid)**
