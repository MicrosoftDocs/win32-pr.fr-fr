# <a name="content-protection-commands"></a>Commandes protection du contenu

Répertorie les commandes pour la méthode [**IDirect3DAuthenticatedChannel9 :: configure**](/windows/desktop/api/d3d9/nf-d3d9-idirect3dauthenticatedchannel9-configure) .



| Commande                                                                                                           | Description                                                                                                           |
|-------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| [**Initialisation de D3DAUTHENTICATEDCONFIGURE \_**](d3dauthenticatedconfigure-initialize.md)                             | Initialise le canal authentifié.                                                                                |
| [**D3DAUTHENTICATEDCONFIGURE \_ CRYPTOSESSION**](d3dauthenticatedconfigure-cryptosession.md)                       | Associe une session de chiffrement à un appareil de décodage de DirectX Video Acceleration 2 (DXVA-2) et à un appareil Direct3D. |
| [**D3DAUTHENTICATEDCONFIGURE \_ ENCRYPTIONWHENACCESSIBLE**](d3dauthenticatedconfigure-encryptionwhenaccessible.md) | Définit le niveau de chiffrement effectué avant que le contenu protégé soit accessible au processeur ou au bus.         |
| [**\_Protection D3DAUTHENTICATEDCONFIGURE**](d3dauthenticatedconfigure-protection.md)                             | Active ou désactive la protection de l’appareil.                                                                        |
| [**D3DAUTHENTICATEDCONFIGURE \_ SHAREDRESOURCE**](d3dauthenticatedconfigure-sharedresource.md)                     | Permet à un processus d’ouvrir une ressource partagée ou désactive l’ouverture de ressources partagées par un processus.                     |
