# <a name="d3dauthenticatedquery_channeltype"></a>D3DAUTHENTICATEDQUERY \_ CHANNELTYPE

| Condition requise | Valeur |
|-------------|--------------------------------------------------------------------------------------------------------------|
| GUID de la requête  | **D3DAUTHENTICATEDQUERY \_ CHANNELTYPE**                                                                       |
| Données d’entrée  | [**\_Entrée de requête D3DAUTHENTICATEDCHANNEL \_**](d3dauthenticatedchannel-query-input.md)                         |
