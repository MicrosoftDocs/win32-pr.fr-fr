# <a name="codecapi_avench264cabacenable-property"></a>CODECAPI \_ propriété AVEncH264CABACEnable

Active ou désactive le codage arithmétique binaire CABAC (Context-adaptative) pour le codage d’entropie H. 264.

## <a name="data-type"></a>Type de données

**Variante \_ BOOL** (VT \_ bool)

## <a name="property-guid"></a>Guid de propriété

**CODECAPI \_ AVEncH264CABACEnable**

## <a name="property-value"></a>Valeur de la propriété

Si la valeur est **\_ true**, CABAC est activé.
