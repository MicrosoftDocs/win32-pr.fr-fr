# <a name="direct3d-12-media-foundation-guids"></a>GUID de Media Foundation Direct3D 12

Cette page spécifie les GUID pour les GUID Direct3D 12 pour Media Foundation.

<dl> <dt>

**\_Objet de \_ synchronisation \_ D3D12 MF**
</dt> <dd> <dl> <dt>



GUID identifiant un objet de synchronisation Direct3D Media Foundation. Transmettez cette valeur à [IMFDXGIBuffer :: setunknown,](/windows/win32/api/mfobjects/nf-mfobjects-imfdxgibuffer-setunknown) et [IMFDXGIBuffer :: GetUnknown](/windows/win32/api/mfobjects/nf-mfobjects-imfdxgibuffer-getunknown) pour accéder à un objet de synchronisation Direct3D 12 Media Foundation.
