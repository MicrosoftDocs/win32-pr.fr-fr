# <a name="mf_mt_d3d12_resource_flag_allow_unordered_access-attribute"></a>\_ \_ L’indicateur de ressource D3D12 MF MT- \_ \_ \_ autoriser \_ \_ l’attribut d’accès non ordonné

Indique si une vue d’accès non ordonnée peut être créée pour les ressources Direct3D dans le flux associé au type de média. 

## <a name="data-type"></a>Type de données

**UINT32**

## <a name="remarks"></a>Notes

La valeur de cet attribut correspond au membre D3D12_RESOURCE_FLAG_ALLOW_UNORDERED_ACCESS de l’énumération [D3D12_RESOURCE_FLAGS](/windows/win32/api/d3d12/ne-d3d12-d3d12_resource_flags) .
