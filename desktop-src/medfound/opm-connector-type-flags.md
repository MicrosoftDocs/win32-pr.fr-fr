## <a name="remarks"></a>Notes

Si un connecteur est décrit comme étant *incorporé* ou *intégré*, cela implique que le connecteur est interne. Ces connecteurs ont « EMBEDDed » dans le nom de la constante d’énumération.

Les applications doivent ignorer l’indicateur **\_ interne du \_ \_ \_ type \_ de connecteur compatible avec OPM Copp** et rechercher à la place les types de connecteur avec « incorporé » dans le nom de la constante.

À l’exception du **\_ type de \_ Connecteur compatible OPM Copp \_ \_ \_ Internal**, les valeurs répertoriées ici ne sont pas des indicateurs binaires et ne peuvent pas être combinées.

Certaines de ces valeurs sont équivalentes à celles de l’énumération **Copp \_ ConnectorType** utilisée dans le protocole Copp (Certified Output Protection Protocol).
