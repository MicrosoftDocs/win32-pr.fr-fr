# <a name="saving-and-loading-topologies"></a>Enregistrement et chargement de topologies

Avec TopoEdit, vous pouvez enregistrer une topologie sous la forme d’un fichier XML et la charger ultérieurement. Lorsqu’une topologie complète est chargée, elle doit être résolue à nouveau. Pour plus d’informations sur cette procédure, consultez [résolution d’une topologie avec TopoEdit](resolving-a-topology-with-topoedit.md).

## <a name="to-save-a-topology"></a>Pour enregistrer une topologie

1.  Dans le menu **File** (Fichier), cliquez sur **Save** (Enregistrer).

    La boîte de dialogue **enregistrer le fichier** s’ouvre.

2.  Accédez au dossier dans lequel le fichier de topologie doit être enregistré.

3.  Dans le champ **nom du fichier :** , entrez le nom du fichier.

4.  Cliquez sur **Enregistrer**.

Le fichier de topologie est enregistré sous la forme d’un fichier XML.

## <a name="to-load-a-saved-topology"></a>Pour charger une topologie enregistrée

1.  Dans le menu **Fichier**, cliquez sur **Ouvrir**.

    La boîte de dialogue **Ouvrir un fichier** s’ouvre.

2.  Accédez au dossier où se trouve le fichier XML de topologie.

3.  Spécifiez le nom du fichier dans le champ **nom du fichier** ou sélectionnez le fichier dans la liste.

4.  Cliquez sur **Ouvrir**.

Le fichier de topologie est chargé dans le **volet topologie**.
