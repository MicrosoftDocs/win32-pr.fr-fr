# <a name="controls-menu"></a>Menu contrôles

Le menu **contrôles** fournit des commandes de contrôle de transport pour une topologie qui est définie sur la session de média. Ce menu est activé uniquement quand une topologie est correctement résolue. Pour accéder rapidement à ces commandes, vous pouvez utiliser la [barre d’outils TopoEdit](topoedit-toolbar.md).

Le menu **contrôles** fournit les commandes suivantes :



| Commande   | Description                                                              |
|-----------|--------------------------------------------------------------------------|
| **Lire**  | Démarre la lecture de la dernière topologie résolue.                           |
| **Stop**  | Arrête la lecture de la dernière topologie résolue actuellement en cours.  |
| **Pause** | Suspend la lecture de la dernière topologie résolue actuellement en cours. |
