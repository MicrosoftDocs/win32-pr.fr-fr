# <a name="mfpkey_wmavoice_enc_bufferwindow-property"></a>MFPKEY \_ WMAVOICE \_ enc \_ BufferWindow, propriété

Spécifie la fenêtre de mémoire tampon, en millisecondes, à utiliser pour le codec vocal.

## <a name="constant-for-ipropertybag"></a>Constante pour IPropertyBag

\_wszWMACVoiceBuffer g

## <a name="data-type"></a>Type de données

VT \_

## <a name="default-value"></a>Valeur par défaut

1500 pour 4, 5, 8, 12 et 16 Kbits/s

2177 pour 10 et 20 Kbits/s
