# <a name="topology-menu"></a>Menu Topology

Le menu **Topology (topologie** ) fournit les commandes suivantes :



<table>
<colgroup>
<col style="width: 50%" />
<col style="width: 50%" />
</colgroup>
<thead>
<tr class="header">
<th>Commande</th>
<th>Description</th>
</tr>
</thead>
<tbody>
<tr class="odd">
<td><strong>Ajouter une source...</strong></td>
<td>Ajoute un nœud source pour le fichier multimédia spécifié. Consultez <a href="adding-source-nodes-with-topoedit.md">Ajout de nœuds sources avec TopoEdit</a>.</td>
</tr>
<tr class="even">
<td><strong>Ajouter un SAR</strong></td>
<td>Ajoute un nœud de sortie pour le convertisseur audio en continu (SAR). Consultez <a href="adding-output-nodes-with-topoedit.md">Ajout de nœuds de sortie avec TopoEdit</a>.</td>
</tr>
<tr class="odd">
<td><strong>Ajouter EVR</strong></td>
<td>Ajoute un nœud de sortie pour le convertisseur vidéo amélioré (EVR). Consultez <a href="adding-output-nodes-with-topoedit.md">Ajout de nœuds de sortie avec TopoEdit</a>.</td>
</tr>
<tr class="even">
<td><strong>Ajouter une transformation...</strong></td>
<td>Ajoute le nœud de transformation spécifié. Consultez <a href="adding-transform-nodes-with-topoedit.md">Ajout de nœuds de transformation avec TopoEdit</a>.</td>
</tr>
<tr class="odd">
<td><strong>Ajouter un tee</strong></td>
<td>Ajoute un nœud tee pour l’envoi de la sortie vers plusieurs nœuds de sortie.</td>
</tr>
<tr class="even">
<td><strong>Ajouter une table MFT personnalisée...</strong></td>
<td>Ajoute une table de Media Foundation personnalisée (MFT). Consultez <a href="adding-transform-nodes-with-topoedit.md">Ajout de nœuds de transformation avec TopoEdit</a>.</td>
</tr>
<tr class="odd">
<td><strong>Ajouter un récepteur personnalisé...</strong></td>
<td>Ajoute un nœud de sortie pour un récepteur de flux personnalisé. Consultez <a href="adding-output-nodes-with-topoedit.md">Ajout de nœuds de sortie avec TopoEdit</a>.</td>
</tr>
<tr class="even">
<td><strong>Ajouter une source de capture vidéo...</strong></td>
<td>Ajoute un nœud source pour un appareil de capture vidéo.<br/>
<blockquote>
[!Note]<br />
Cette fonctionnalité nécessite Windows 7.
</blockquote>
<br/></td>
</tr>
<tr class="odd">
<td><strong>Ajouter une source de capture audio...</strong></td>
<td>Ajoute un nœud source pour un périphérique de capture audio.<br/>
<blockquote>
[!Note]<br />
Cette fonctionnalité nécessite Windows 7.
</blockquote>
<br/></td>
</tr>
<tr class="even">
<td><strong>Résoudre la topologie</strong></td>
<td>Crée une topologie complète pour la lecture. Consultez <a href="resolving-a-topology-with-topoedit.md">résolution d’une topologie à l’aide de TopoEdit</a>.</td>
</tr>
<tr class="odd">
<td><strong>Supprimer</strong></td>
<td>Supprime un élément, des nœuds de topologie ou des connexions de nœud dans le <strong>volet de topologie</strong>.</td>
</tr>
</tbody>
</table>
