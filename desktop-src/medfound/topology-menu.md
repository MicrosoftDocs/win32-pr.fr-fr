# <a name="topology-menu"></a>Menu Topology

Le menu **Topology (topologie** ) fournit les commandes suivantes :




| Commande | Description | 
|---------|-------------|
| <strong>Ajouter une source...</strong> | Ajoute un nœud source pour le fichier multimédia spécifié. Consultez <a href="adding-source-nodes-with-topoedit.md">Ajout de nœuds sources avec TopoEdit</a>. | 
| <strong>Ajouter un SAR</strong> | Ajoute un nœud de sortie pour le convertisseur audio en continu (SAR). Consultez <a href="adding-output-nodes-with-topoedit.md">Ajout de nœuds de sortie avec TopoEdit</a>. | 
| <strong>Ajouter EVR</strong> | Ajoute un nœud de sortie pour le convertisseur vidéo amélioré (EVR). Consultez <a href="adding-output-nodes-with-topoedit.md">Ajout de nœuds de sortie avec TopoEdit</a>. | 
| <strong>Ajouter une transformation...</strong> | Ajoute le nœud de transformation spécifié. Consultez <a href="adding-transform-nodes-with-topoedit.md">Ajout de nœuds de transformation avec TopoEdit</a>. | 
| <strong>Ajouter un tee</strong> | Ajoute un nœud tee pour l’envoi de la sortie vers plusieurs nœuds de sortie. | 
| <strong>Ajouter une table MFT personnalisée...</strong> | Ajoute une table de Media Foundation personnalisée (MFT). Consultez <a href="adding-transform-nodes-with-topoedit.md">Ajout de nœuds de transformation avec TopoEdit</a>. | 
| <strong>Ajouter un récepteur personnalisé...</strong> | Ajoute un nœud de sortie pour un récepteur de flux personnalisé. Consultez <a href="adding-output-nodes-with-topoedit.md">Ajout de nœuds de sortie avec TopoEdit</a>. | 
| <strong>Ajouter une source de capture vidéo...</strong> | Ajoute un nœud source pour un appareil de capture vidéo.<br /><blockquote>[!Note]<br />cette fonctionnalité requiert Windows 7.</blockquote><br /> | 
| <strong>Ajouter une source de capture audio...</strong> | Ajoute un nœud source pour un périphérique de capture audio.<br /><blockquote>[!Note]<br />cette fonctionnalité requiert Windows 7.</blockquote><br /> | 
| <strong>Résoudre la topologie</strong> | Crée une topologie complète pour la lecture. Consultez <a href="resolving-a-topology-with-topoedit.md">résolution d’une topologie à l’aide de TopoEdit</a>. | 
| <strong>Supprimer</strong> | Supprime un élément, des nœuds de topologie ou des connexions de nœud dans le <strong>volet de topologie</strong>. |
