# <a name="mfpkey_decoder_deinterlacing-property"></a>\_Propriété désentrelacer du DÉcodeur MFPKEY \_

Spécifie si le décodeur décode des images vidéo entrelacées à partir du flux compressé en tant qu’images progressives.

## <a name="constant-for-ipropertybag"></a>Constante pour IPropertyBag

\_wszWMVCDecoderDeinterlacing g

## <a name="data-type"></a>Type de données

**VT \_ bool**

## <a name="default-value"></a>Valeur par défaut

**VARIANTE \_ false**
