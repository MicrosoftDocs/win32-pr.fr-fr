# <a name="direct3d-11-video-interfaces"></a>Interfaces vidéo Direct3D 11

Cette section contient des informations de référence sur les interfaces vidéo de Microsoft Direct3D 11.

## <a name="in-this-section"></a>Contenu de cette section



| Rubrique                                                                                 | Description                                                                                              |
|---------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------|
| [**ID3D11AuthenticatedChannel**](/windows/desktop/api/d3d11/nn-d3d11-id3d11authenticatedchannel)<br/>           | Fournit un canal de communication avec le pilote Graphics ou le runtime Microsoft Direct3D. <br/> |
| [**ID3D11CryptoSession**](/windows/desktop/api/d3d11/nn-d3d11-id3d11cryptosession)<br/>                         | Représente une session de chiffrement. <br/>                                                          |
| [**ID3D11VideoContext**](/windows/desktop/api/d3d11/nn-d3d11-id3d11videocontext)<br/>                           | Fournit la fonctionnalité vidéo d’un appareil Direct3D 11. <br/>                                    |
| [**ID3D11VideoContext1**](/windows/desktop/api/d3d11_1/nn-d3d11_1-id3d11videocontext1)<br/>                         | Fournit la fonctionnalité vidéo d’un appareil Direct3D 11.<br/>                                     |
| [**ID3D11VideoContext2**](/windows/win32/api/d3d11_4/nn-d3d11_4-id3d11videocontext2)<br/>                         | Fournit la fonctionnalité vidéo d’un appareil Direct3D 11. <br/>                                    |
| [**ID3D11VideoDecoder**](/windows/desktop/api/d3d11/nn-d3d11-id3d11videodecoder)<br/>                           | Représente un décodeur vidéo accéléré par le matériel pour Direct3D 11.<br/>                              |
| [**ID3D11VideoDecoderOutputView**](/windows/desktop/api/d3d11/nn-d3d11-id3d11videodecoderoutputview)<br/>       | Identifie les surfaces de sortie accessibles pendant le décodage vidéo.<br/>                    |
| [**ID3D11VideoDevice**](/windows/desktop/api/d3d11/nn-d3d11-id3d11videodevice)<br/>                             | Fournit le décodage vidéo et les fonctionnalités de traitement vidéo d’un appareil Direct3D 11. <br/>       |
| [**ID3D11VideoDevice1**](/windows/desktop/api/d3d11_1/nn-d3d11_1-id3d11videodevice1)<br/>                           | Fournit le décodage vidéo et les fonctionnalités de traitement vidéo d’un appareil Direct3D 11. <br/>       |
| [**ID3D11VideoProcessor**](/windows/desktop/api/d3d11/nn-d3d11-id3d11videoprocessor)<br/>                       | Représente un processeur vidéo pour Direct3D 11.<br/>                                                 |
| [**ID3D11VideoProcessorEnumerator**](/windows/desktop/api/d3d11/nn-d3d11-id3d11videoprocessorenumerator)<br/>   | Énumère les fonctionnalités du processeur vidéo d’un appareil Direct3D 11. <br/>                         |
| [**ID3D11VideoProcessorEnumerator1**](/windows/desktop/api/d3d11_1/nn-d3d11_1-id3d11videoprocessorenumerator1)<br/> | Énumère les fonctionnalités du processeur vidéo d’un appareil Direct3D 11.<br/>                          |
| [**ID3D11VideoProcessorInputView**](/windows/desktop/api/d3d11/nn-d3d11-id3d11videoprocessorinputview)<br/>     | Identifie les surfaces d’entrée accessibles pendant le traitement vidéo.<br/>                   |
| [**ID3D11VideoProcessorOutputView**](/windows/desktop/api/d3d11/nn-d3d11-id3d11videoprocessoroutputview)<br/>   | Identifie les surfaces de sortie accessibles pendant le traitement vidéo.<br/>                  |
