# <a name="mf_mt_d3d12_resource_flag_allow_render_target-attribute"></a>\_Indicateur de \_ ressource D3D12 MF MT allow pour l' \_ \_ \_ \_ \_ attribut cible de rendu

Indique si une vue de cible de rendu peut être créée pour les ressources Direct3D dans le flux associé au type de média. 

## <a name="data-type"></a>Type de données

**UINT32**

## <a name="remarks"></a>Notes

La valeur de cet attribut correspond au membre D3D12_RESOURCE_FLAG_ALLOW_RENDER_TARGET de l’énumération [D3D12_RESOURCE_FLAGS](/windows/win32/api/d3d12/ne-d3d12-d3d12_resource_flags) .
