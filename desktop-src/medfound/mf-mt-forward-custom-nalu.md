# <a name="mf_mt_forward_custom_nalu-attribute"></a>\_ \_ \_ Attribut Nalu personnalisé MF MT Forward \_

Spécifie que les types d’unités NAL (Network Abstraction Layer) doivent être transférés sur les exemples de sortie par le décodeur.

## <a name="data-type"></a>Type de données

**UINT32**

## <a name="remarks"></a>Remarques

Si le décodeur analyse un NALU, il ne sera pas transféré.
