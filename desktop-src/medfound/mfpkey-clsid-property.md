# <a name="mfpkey_clsid-property"></a>MFPKEY \_ CLSID, propriété

Contient le CLSID d’une Media Foundation transformation (MFT).



Type de données

Type de PROPVARIANT (VT)

Membre PROPVARIANT

**GUID** (**CLSID** \* )

\_CLSID VT

**puuid**



## <a name="remarks"></a>Notes

Cette propriété est facultative et est à titre d’information uniquement. Pour obtenir cette propriété, interrogez la table MFT pour l’interface **IPropertyStore** .
