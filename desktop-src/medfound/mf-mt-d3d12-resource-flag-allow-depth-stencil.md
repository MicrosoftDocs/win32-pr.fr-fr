# <a name="mf_mt_d3d12_resource_flag_allow_depth_stencil-attribute"></a>\_Attribut de \_ \_ stencil de \_ \_ profondeur allow \_ \_ MT D3D12

Indique si la vue du stencil de profondeur peut être créée pour les ressources Direct3D dans le flux associé au type de média. 

## <a name="data-type"></a>Type de données

**UINT32**

## <a name="remarks"></a>Notes

La valeur de cet attribut correspond au membre D3D12_RESOURCE_FLAG_ALLOW_DEPTH_STENCIL de l’énumération [D3D12_RESOURCE_FLAGS](/windows/win32/api/d3d12/ne-d3d12-d3d12_resource_flags) .
