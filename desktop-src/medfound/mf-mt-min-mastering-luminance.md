# <a name="mf_mt_min_mastering_luminance-attribute"></a>\_Attribut de \_ luminance de mastérisation MT min \_ \_ . MF

Spécifie la luminance maximale de l’affichage sur lequel le contenu a été créé, en nits. Cet attribut a la même sémantique que la **\_ \_ \_ luminance de Master Display min** , comme défini dans la norme CEA-861,3.

## <a name="data-type"></a>Type de données

**UINT32**
