# <a name="media-foundation-pipeline"></a>Pipeline Media Foundation

La couche de *pipeline* dans Microsoft Media Foundation est la couche de l’architecture qui génère ou traite directement les données multimédias.

La plupart des applications n’appellent pas directement des méthodes sur les composants de pipeline, bien qu’il soit possible de le faire. Lisez ces rubriques si vous écrivez un composant de pipeline personnalisé.

## <a name="in-this-section"></a>Contenu de cette section



| Rubrique                                                                     | Description                                                                                                                           |
|---------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------|
| [Sources multimédias](media-sources.md)<br/>                             | Les sources multimédias génèrent des données multimédias, généralement à partir d’un fichier, d’un périphérique de capture ou d’un flux réseau. <br/>                              |
| [Transformations de Media Foundation](media-foundation-transforms.md)<br/> | Media Foundation transforme les données du support de traitement (MFTs). Par exemple, les codecs dans Media Foundation sont implémentés en tant que MFTs. <br/>   |
| [Récepteurs de médias](media-sinks.md)<br/>                                 | Les récepteurs multimédias consomment des données multimédias. Par exemple, un récepteur multimédia peut écrire les données dans un fichier ou envoyer les données sur un réseau.<br/> |
