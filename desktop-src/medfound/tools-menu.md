# <a name="tools-menu"></a>Menu Outils

Le menu **Outils** fournit les options suivantes :




| Commande | Description | 
|---------|-------------|
| <strong>Nœud sélectionné Spy</strong> | Journalise tous les appels de méthode sur un nœud de transformation ou un nœud de sortie sélectionné, puis enregistre les résultats dans un fichier texte. Pour plus d’informations, consultez <a href="logging-topology-node-activities.md">journalisation des activités de nœuds de topologie</a> | 
| <strong>Topoloader personnalisé...</strong> | Permet à l’utilisateur de fournir un chargeur de topologie personnalisé.<br /><blockquote>[!Note]<br />cette fonctionnalité a été ajoutée pour Windows 7.</blockquote><br /> |
