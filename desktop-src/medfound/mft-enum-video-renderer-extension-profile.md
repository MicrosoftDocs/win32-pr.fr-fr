# <a name="mft_enum_video_renderer_extension_profile-attribute"></a>Attribut de profil de l' \_ \_ extension de convertisseur vidéo de l’ÉNUMÉRation MFT \_ \_ \_

Contient une liste de toutes les entrées **VideoRendererExtensionProfile** dans la balise **VideoRendererExtensionProfiles** d’un fichier manifeste d’application UWP. [**MFTEnumEx**](/windows/desktop/api/mfapi/nf-mfapi-mftenumex) stocke cela dans le magasin d’attributs de l’objet [**IMFActivate**](/windows/desktop/api/mfobjects/nn-mfobjects-imfactivate) créé par **MFTEnumEx** pour MFTS qui a un manifeste UWP associé contenant la balise **VideoRendererExtensionProfiles** .
