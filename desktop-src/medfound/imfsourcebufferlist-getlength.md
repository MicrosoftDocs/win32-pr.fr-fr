# <a name="imfsourcebufferlistgetlength-method"></a>IMFSourceBufferList :: GetLength, méthode

Obtient le nombre d’objets [**IMFSourceBuffer**](/windows/desktop/api/mfmediaengine/nn-mfmediaengine-imfsourcebuffer) dans la liste.

## <a name="syntax"></a>Syntaxe


```C++
DWORD GetLength();
```



## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.

## <a name="return-value"></a>Valeur de retour

Nombre de mémoires tampons sources dans la liste.
