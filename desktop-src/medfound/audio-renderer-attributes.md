# <a name="audio-renderer-attributes"></a>Attributs du convertisseur audio

Les attributs suivants peuvent être utilisés pour configurer le convertisseur audio.



| Attribut                                                                                                     | Description                                                                                                 |
|---------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------|
| [**\_ID de \_ point de \_ terminaison d’attribut de convertisseur audio \_ MF \_**](mf-audio-renderer-attribute-endpoint-id-attribute.md)     | Spécifie l’identificateur pour l’appareil de point de terminaison audio.                                                     |
| [**\_rôle de \_ point de \_ terminaison d’attribut de convertisseur audio \_ MF \_**](mf-audio-renderer-attribute-endpoint-role-attribute.md) | Spécifie le rôle de point de terminaison audio pour le convertisseur audio.                                                   |
| [**\_indicateurs d' \_ attribut de convertisseur audio \_ MF \_**](mf-audio-renderer-attribute-flags-attribute.md)                  | Contient des indicateurs pour configurer le convertisseur audio.                                                             |
| [**ID de session de l' \_ \_ attribut de convertisseur audio \_ \_ MF \_**](mf-audio-renderer-attribute-session-id-attribute.md)       | Spécifie la classe de stratégie audio pour le convertisseur audio.                                                    |
| [\_catégorie de \_ flux d' \_ attributs \_ de convertisseur audio \_ MF](mf-audio-renderer-attribute-stream-category.md)           | Spécifie la catégorie de flux audio pour le [convertisseur audio de streaming](streaming-audio-renderer.md) (SAR). |
