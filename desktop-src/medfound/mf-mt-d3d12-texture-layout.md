# <a name="mf_mt_d3d12_texture_layout-attribute"></a>\_Attribut de \_ \_ \_ mise en page de texture MF MT D3D12

Indique les options de mise en page de texture qui ont été utilisées pour créer les ressources Direct3D associées.

## <a name="data-type"></a>Type de données

**UINT32**

## <a name="remarks"></a>Notes

La valeur de cet attribut correspond à un membre de l’énumération [D3D12_TEXTURE_LAYOUT](/windows/win32/api/d3d12/ne-d3d12-d3d12_texture_layout) .
