# <a name="mf_pd_total_file_size-attribute"></a>\_Attribut de \_ taille totale des \_ fichiers MF PD \_

Spécifie la taille totale, en octets, du fichier source. Cet attribut s’applique aux descripteurs de présentation. Une source de média peut éventuellement définir cet attribut.

## <a name="data-type"></a>Type de données

**UINT64**

## <a name="remarks"></a>Notes

La constante GUID de cet attribut est exportée à partir de mfuuid. lib.
