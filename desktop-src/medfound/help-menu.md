# <a name="help-menu"></a>Menu ? (Aide)

Le menu **aide** fournit les commandes suivantes :



| Commande      | Description                                                                      |
|--------------|----------------------------------------------------------------------------------|
| **Aide...**  | Ouvre l’aide de TopoEdit sur le site Web MSDN.                                         |
| **À propos de...** | Ouvre la boîte de dialogue **à propos** de qui affiche des informations de version sur TopoEdit. |
