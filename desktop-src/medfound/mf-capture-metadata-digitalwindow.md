# <a name="mf_capture_metadata_digitalwindow-attribute"></a>\_ \_ Attribut DIGITALWINDOW de métadonnées de capture MF \_

Valeur représentant les paramètres de limites de la fenêtre numérique pour la capture vidéo.

## <a name="data-type"></a>Type de données

**OBJET BLOB**

## <a name="remarks"></a>Remarques

La valeur **BLOB** de cette propriété est une structure [DigitalWindowSetting](/windows/win32/api/mfapi/ns-mfapi-digitalwindowsetting) .
