# <a name="sequencer-source"></a>Source de Sequencer

La source de Sequencer permet à une application de lire une collection de sources multimédias séquentiellement, avec des transitions transparents entre les sources. Vous pouvez l’utiliser pour créer des sélections ou pour lire simultanément des flux de plusieurs sources.
