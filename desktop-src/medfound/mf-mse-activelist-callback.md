# <a name="mf_mse_activelist_callback-attribute"></a>\_Attribut de rappel MF MSE \_ ACTIVELIST \_

Contient un pointeur vers l’interface de rappel de l’application pour l’interface [**IMFBufferListNotify**](/windows/desktop/api/mfmediaengine/nn-mfmediaengine-imfbufferlistnotify) pour la liste de mémoires tampons active.

## <a name="data-type"></a>Type de données

**IMFBufferListNotify \* *_ stocké en tant que _* IUnknown\***
