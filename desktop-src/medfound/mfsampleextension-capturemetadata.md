# <a name="mfsampleextension_capturemetadata-attribute"></a>\_Attribut MFSampleExtension CaptureMetadata

Magasin [**IMFAttributes**](/windows/desktop/api/mfobjects/nn-mfobjects-imfattributes) pour toutes les métadonnées associées au pipeline de capture.

## <a name="data-type"></a>Type de données

**[**IMFAttributes**](/windows/desktop/api/mfobjects/nn-mfobjects-imfattributes)** stocké en tant que **IUnknown** 

## <a name="remarks"></a>Remarques

Cet attribut peut être présent sur n’importe quel [**IMFSample**](/windows/desktop/api/mfobjects/nn-mfobjects-imfsample).

Cet attribut est en lecture seule.
