# <a name="adding-source-nodes-with-topoedit"></a>Ajout de nœuds sources avec TopoEdit

Un nœud source représente un flux dans un fichier multimédia. Pour créer un nœud source dans TopoEdit, vous spécifiez un fichier multimédia. TopoEdit énumère les flux dans le fichier et crée les nœuds sources appropriés.

Pour plus d’informations sur l’ajout par programme de nœuds sources à l’aide d’API Media Foundation, consultez [création de nœuds sources](creating-source-nodes.md).

## <a name="to-add-a-source-node-to-a-topology"></a>Pour ajouter un nœud source à une topologie

1.  Dans le menu **Topology (topologie** ), cliquez sur Ajouter une **source**.

    La boîte de dialogue **Sélectionner la source du média** s’ouvre.

2.  Accédez au dossier où se trouve le fichier multimédia.

3.  Dans le champ **nom du fichier :** , entrez le nom du fichier.

4.  Cliquez sur **Ouvrir**.

TopoEdit crée des nœuds sources pour les flux. Le **volet topologie** affiche le nœud source contenu dans une zone grise qui indique le nom du fichier multimédia. Le nœud source affiche le type de flux du nœud.
