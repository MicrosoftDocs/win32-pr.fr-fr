# <a name="mf_mt_max_luminance_level-attribute"></a>\_Attribut de \_ \_ niveau de luminance maximale MF MT \_

Spécifie le niveau de luminance maximal du contenu dans nits. Cet attribut a la même sémantique que **MaxCLL** , comme défini dans la norme CEA-861,3.

## <a name="data-type"></a>Type de données

**UINT32**
