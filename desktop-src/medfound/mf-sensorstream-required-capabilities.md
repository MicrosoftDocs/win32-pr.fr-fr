# <a name="mf_sensorstream_required_capabilities-attribute"></a>\_Attribut des \_ fonctionnalités requises SENSORSTREAM \_ MF

Cet attribut contient une liste de chaînes de capacité séparées par des points-virgules, qui spécifie les fonctionnalités requises pour un flux spécifique. Pour obtenir la liste des chaînes de capacité qui peuvent être incluses dans cet attribut, consultez [DeviceCapability](/uwp/schemas/appxpackage/appxmanifestschema/element-devicecapability).

## <a name="data-type"></a>Type de données

**WCHAR\***
