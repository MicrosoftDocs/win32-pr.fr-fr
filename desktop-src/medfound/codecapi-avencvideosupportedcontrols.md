# <a name="codecapi_avencvideosupportedcontrols-property"></a>CODECAPI \_ propriété AVEncVideoSupportedControls

Définit les contrôles pris en charge pour un encodeur vidéo.

## <a name="data-type"></a>Type de données

**UINT64 (VT \_ UI8)**

## <a name="property-guid"></a>Guid de propriété

**CODECAPI \_ AVEncVideoSupportedControls**

## <a name="remarks"></a>Notes

Cette propriété est utilisée uniquement avec les [encodeurs de caméra H. 264 UVC 1,5](camera-encoder-h264-uvc-1-5.md).
