# <a name="mf_mt_dv_vaux_ctrl_pack-attribute"></a>\_ \_ \_ \_ Attribut Pack de Ctrl Vaux \_

Le Pack de contrôle source VAUX (Video auxiliaire) dans un type de média vidéo numérique (DV).

## <a name="data-type"></a>Type de données

**UINT32**

## <a name="remarks"></a>Remarques

cet attribut correspond au membre **dwDVVAuxCtl** de la structure [**DVINFO**](/windows/win32/api/strmif/ns-strmif-dvinfo) DirectShow.

La constante GUID de cet attribut est exportée à partir de mfuuid. lib.
