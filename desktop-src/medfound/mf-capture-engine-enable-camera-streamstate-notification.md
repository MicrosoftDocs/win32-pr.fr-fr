# <a name="mf_capture_engine_enable_camera_streamstate_notification-attribute"></a>Le \_ moteur de capture MF \_ active l' \_ attribut de \_ \_ notification STREAMSTATE de l’appareil photo \_

Indique si les notifications d’état de flux doivent être activées.

## <a name="data-type"></a>Type de données

## <a name="remarks"></a>Notes

Une valeur différente de zéro spécifie que les notifications doivent être activées. Si l’attribut n’est pas présent ou si la valeur de l’attribut est 0, les notifications ne sont pas activées.
