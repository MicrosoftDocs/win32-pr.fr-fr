# <a name="mfpkey_dyn_bandtrunc_qceil-property"></a>MFPKEY \_ dyn \_ BANDTRUNC \_ QCEIL, propriété

Spécifie la qualité à laquelle la bande passante minimale codée est autorisée. À une qualité inférieure ou égale à la valeur de cette propriété, la bande passante codée minimale est autorisée.

## <a name="constant-for-ipropertybag"></a>Constante pour IPropertyBag

Disponible uniquement à l’aide de [**IPropertyStore**](/windows/win32/api/propsys/nn-propsys-ipropertystore).

## <a name="data-type"></a>Type de données

**VT \_ R4**
