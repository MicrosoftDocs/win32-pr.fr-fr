# <a name="mf_disable_locally_registered_plugins-attribute"></a>MF- \_ désactiver l' \_ \_ \_ attribut plug-ins inscrits localement

Spécifie si les plug-ins inscrits localement sont désactivés.

## <a name="data-type"></a>Type de données

**UINT32**

## <a name="remarks"></a>Remarques

Affectez la valeur **true** à cet attribut pour empêcher le chargeur de topologie MF d’insérer des transformations inscrites localement.
