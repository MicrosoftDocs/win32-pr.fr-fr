# <a name="table-of-contents-parser-interfaces"></a>Interfaces de l’analyseur de table des matières

Les interfaces suivantes ont prouvé l’accès aux services de l’analyseur de table des matières.



| Interface                                | Description                                                                                                           |
|------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| [**IToc**](/windows/desktop/api/wmcodecdsp/nn-wmcodecdsp-itoc)                     | Représente une table des matières individuelle.                                                                           |
| [**ITocCollection**](/windows/desktop/api/wmcodecdsp/nn-wmcodecdsp-itoccollection) | Représente une collection de tables de matières.                                                                        |
| [**ITocEntry**](/windows/desktop/api/wmcodecdsp/nn-wmcodecdsp-itocentry)           | Représente une entrée individuelle dans une table des matières.                                                                |
| [**ITocEntryList**](/windows/desktop/api/wmcodecdsp/nn-wmcodecdsp-itocentrylist)   | Représente une liste d’entrées dans une table des matières.                                                                  |
| [**ITocParser**](/windows/desktop/api/wmcodecdsp/nn-wmcodecdsp-itocparser)         | Fournit des méthodes pour incorporer une table des matières dans un fichier vidéo et lire des tables de contenu à partir d’un fichier vidéo. |
