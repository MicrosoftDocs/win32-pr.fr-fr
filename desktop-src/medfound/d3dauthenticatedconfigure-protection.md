# <a name="d3dauthenticatedconfigure_protection"></a>\_Protection D3DAUTHENTICATEDCONFIGURE

Active ou désactive la protection de l’appareil.



| Condition requise | Valeur |
|--------------|-----------------------------------------------------------------------------------------------------|
| GUID de la commande | **\_Protection D3DAUTHENTICATEDCONFIGURE**                                                           |
| Données d’entrée   | [**D3DAUTHENTICATEDCHANNEL \_ CONFIGUREPROTECTION**](d3dauthenticatedchannel-configureprotection.md) |



 

## <a name="remarks"></a>Notes

Cette commande est valide pour tous les types de canaux.
