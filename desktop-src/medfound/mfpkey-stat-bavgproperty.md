# <a name="mfpkey_stat_bavg-property"></a>MFPKEY \_ Stat \_ BAVG, propriété

Spécifie la fenêtre de mémoire tampon moyenne, en millisecondes, d’un flux encodé. Lecture seule.

## <a name="constant-for-ipropertybag"></a>Constante pour IPropertyBag

**\_wszWMVCBAvg g**

## <a name="data-type"></a>Type de données

**VT \_**

## <a name="remarks"></a>Remarques

Pour déterminer la fenêtre de mémoire tampon moyenne d’un flux encodé, lisez cette propriété à la fin de l’encodage.
