# <a name="mf_mt_video_3d-attribute"></a>\_Attribut 3D de la vidéo MF MT \_ \_

Spécifie si un flux vidéo contient du contenu 3D.

## <a name="data-type"></a>Type de données

**Bool** stocké comme **UInt32**

## <a name="remarks"></a>Remarques

Cet attribut s’applique aux types de média vidéo. Si cet attribut a la **valeur true**, le flux vidéo contient du contenu 3D. La valeur par défaut est **FALSE**.
