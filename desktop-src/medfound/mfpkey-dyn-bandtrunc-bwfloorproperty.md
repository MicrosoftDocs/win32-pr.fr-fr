# <a name="mfpkey_dyn_bandtrunc_bwfloor-property"></a>MFPKEY \_ dyn \_ BANDTRUNC \_ BWFLOOR, propriété

Spécifie la bande passante codée minimale autorisée par la troncation de la bande dans l’encodeur. Lecture-écriture.

## <a name="constant-for-ipropertybag"></a>Constante pour IPropertyBag

Disponible uniquement à l’aide de [**IPropertyStore**](/windows/win32/api/propsys/nn-propsys-ipropertystore).

## <a name="data-type"></a>Type de données
