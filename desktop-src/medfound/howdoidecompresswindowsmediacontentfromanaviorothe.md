# <a name="how-do-i-decompress-streams-from-an-avi-file-or-other-file-about-which-i-have-no-format-information"></a>Comment faire décompresser des flux à partir d’un fichier AVI ou d’un autre fichier sur lequel je n’ai pas d’informations de format ?

sans les informations de mise en forme, y compris les données de format étendu, vous ne pouvez pas décompresser les données qui ont été encodées avec un codec de média Windows. Les codecs ont été conçus pour créer des flux de données à utiliser avec le conteneur de fichiers ASF (Advanced Systems Format). La section d’en-tête d’un fichier ASF stocke les informations de mise en forme pour tous les flux du fichier. lorsque vous utilisez le contenu encodé à l’aide de l’un des codecs Windows Media Audio et vidéo en dehors d’un fichier ASF, vous devez vous assurer que les informations de format sont disponibles. La façon de procéder varie d’un conteneur à un autre.
