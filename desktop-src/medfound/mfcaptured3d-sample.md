# <a name="mfcaptured3d-sample"></a>Exemple MFCaptureD3D

Montre comment afficher un aperçu de la vidéo d’une caméra vidéo, en utilisant Direct3D pour afficher la vidéo.

## <a name="apis-demonstrated"></a>API illustrées

Cet exemple illustre les API suivantes.

-   [**IMFMediaSource**](/windows/desktop/api/mfidl/nn-mfidl-imfmediasource)
-   [**IMFSourceReader**](/windows/desktop/api/mfreadwrite/nn-mfreadwrite-imfsourcereader)
-   [**MFEnumDeviceSources**](/windows/desktop/api/mfidl/nf-mfidl-mfenumdevicesources)

## <a name="requirements"></a>Configuration requise



| Produit                                                        | Version   |
|----------------------------------------------------------------|-----------|
| [SDK Windows](https://msdn.microsoft.com/windowsvista/bb980924.aspx) | Windows 7 |
