# <a name="mfpkey_volheaderforreencode-property"></a>MFPKEY \_ propriété VOLHEADERFORREENCODE

Pour le ré-encodage de segment, spécifie les données privées du codec du fichier qui est à nouveau codé.

## <a name="constant-for-ipropertybag"></a>Constante pour IPropertyBag

Disponible uniquement à l’aide de [**IPropertyStore**](/windows/win32/api/propsys/nn-propsys-ipropertystore).

## <a name="data-type"></a>Type de données

**\_objet BLOB VT**
