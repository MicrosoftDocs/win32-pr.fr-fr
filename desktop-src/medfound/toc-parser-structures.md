# <a name="table-of-contents-parser-structures"></a>Structures de l’analyseur de table des matières

Les structures suivantes sont utilisées par l’analyseur de table des matières.



| Structure                                              | Description                                                        |
|--------------------------------------------------------|--------------------------------------------------------------------|
| [**\_descripteur TOC**](/windows/desktop/api/wmcodecdsp/ns-wmcodecdsp-toc_descriptor)              | Contient des informations descriptives pour une table des matières.             |
| [**\_descripteur d’entrée TOC \_**](/windows/desktop/api/wmcodecdsp/ns-wmcodecdsp-toc_entry_descriptor) | Contient des informations descriptives pour une entrée dans une table des matières. |
