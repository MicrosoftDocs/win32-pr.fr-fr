# <a name="decoding-to-spdif-microsoft-media-foundation"></a>Décodage vers S/PDIF (Microsoft Media Foundation)

l’identificateur de classe pour l’objet décodeur qui transforme Windows Media Audio 9,1 Professional contenu pour la remise par le biais du Format d’interconnexion numérique Sony/Philips (S/PDIF) est CLSID \_ CWMAudioSpdTxDMO. Vous utilisez cet objet comme vous le feriez pour tout autre décodeur. il prend en charge les mêmes propriétés que le décodeur Windows Media Audio 9. Les types de sortie énumérés par ce décodeur sont tous des formats S/PDIF.

Pour prendre entièrement en charge la sortie S/PDIF, un matériel spécial doit être installé sur l’ordinateur client.
