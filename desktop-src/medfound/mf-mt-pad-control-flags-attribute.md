# <a name="mf_mt_pad_control_flags-attribute"></a>\_ \_ \_ Attribut indicateurs de contrôle du PAD MT MF \_

Spécifie les proportions du rectangle de sortie pour un type de média vidéo.

## <a name="data-type"></a>Type de données

**UINT32**

## <a name="remarks"></a>Notes

La valeur de cet attribut est un membre de l’énumération [**MFVideoPadFlags**](/windows/desktop/api/mfapi/ne-mfapi-mfvideopadflags) .

La constante GUID de cet attribut est exportée à partir de mfuuid. lib.
