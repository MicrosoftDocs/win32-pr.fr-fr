# <a name="overview-of-table-of-contents-parser"></a>Vue d’ensemble de l’analyseur de table des matières

l’analyseur de table des matières (analyseur de table des matières) est un composant Windows qui vous permet de stocker des informations d’index avancées dans des fichiers multimédias. Les tables de matières, les index sémantiques et les tableaux d’histoires sont des exemples d’informations d’index avancées.

Avec l’analyseur TOC, vous pouvez écrire un logiciel qui permet aux utilisateurs de trouver des scènes d’intérêt dans un fichier vidéo. Par exemple, vous pouvez fournir à l’utilisateur une table des matières afin qu’il puisse identifier les principales sections de la vidéo, puis afficher une section particulière. Vous pouvez également fournir à l’utilisateur un index afin qu’il puisse trouver une liste de scènes associées à un mot ou une expression particulière.

le générateur de table des matières (tm generator) est un composant Windows qui fonctionne avec l’analyseur toc. Le générateur de table des matières peut détecter des scènes et des clusters de scènes dans un fichier vidéo, puis générer une table des matières que vous pouvez utiliser comme point de départ de votre logiciel.

La liste suivante présente certains scénarios dans lesquels vous pouvez utiliser les services fournis par l’analyseur de table des matières ou le générateur de tables des matières.

-   Créez une table des matières à partir de zéro et incorporez-la dans un fichier vidéo.
-   Lire une table des matières qui est déjà incorporée dans un fichier vidéo.
-   Modifiez une table des matières qui est déjà incorporée dans un fichier vidéo.
-   Utilisez le générateur TOC pour créer une table des matières basée sur des scènes et des clusters détectés dans un fichier vidéo. Modifiez la table des matières générée et incorporez-la dans le fichier vidéo.
