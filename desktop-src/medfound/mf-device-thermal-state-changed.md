# <a name="mf_device_thermal_state_changed-attribute"></a>Attribut de modification de l' \_ \_ \_ état thermique \_ de l’appareil MF

Représente un événement qui signale une modification de l’état thermique de l’appareil.

## <a name="data-type"></a>Type de données

**UINT32**

## <a name="remarks"></a>Notes

La valeur de cet attribut spécifie la valeur thermique de l’appareil en pourcentage.
