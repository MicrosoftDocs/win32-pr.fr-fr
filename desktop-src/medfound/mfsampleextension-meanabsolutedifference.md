# <a name="mfsampleextension_meanabsolutedifference-attribute"></a>\_Attribut MFSampleExtension MeanAbsoluteDifference

Cet attribut retourne la différence absolue moyenne (MAD) sur tous les blocs de macro dans le plan Y.

## <a name="data-type"></a>Type de données

**UINT32**

## <a name="remarks"></a>Notes

L’application peut demander à encodeur de définir cet attribut sur les exemples de sortie à l’aide de [CODECAPI \_ AVEncVideoMeanAbsoluteDifference](codecapi-avencvideomeanabsolutedifference.md).
