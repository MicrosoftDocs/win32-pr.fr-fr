# <a name="mf_stf_version_info-attribute"></a>MF \_ STF \_ version \_ info (attribut)

Cet attribut représente la fabrique de transformation du capteur d’informations sur la version. Il s’agit du seul attribut requis pour les transformations de capteur. La valeur est définie par le développeur de transformation de capteur et est traitée comme opaque par le pipeline de média.

## <a name="data-type"></a>Type de données

**WCHAR\***
