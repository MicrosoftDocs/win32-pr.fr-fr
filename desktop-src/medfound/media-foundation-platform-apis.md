# <a name="media-foundation-platform-apis"></a>API de plateforme Media Foundation

La couche de plateforme de Media Foundation contient des primitives et des objets d’assistance utilisés par les autres couches.

Cette section contient les rubriques suivantes :



| Rubrique                                                                           | Description                                                                                                                                                       |
|---------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Initialisation de la plateforme Media Foundation](initializing-media-foundation.md) | Comment initialiser la plateforme Media Foundation.                                                                                                                  |
| [Media Foundation et COM](media-foundation-and-com.md)                        | Décrit l’interaction entre COM et Microsoft Media Foundation, et définit certaines meilleures pratiques à utiliser lors du développement de composants de plug-in Media Foundation. |
| [Méthodes de rappel asynchrones](asynchronous-callback-methods.md)              | Comment appeler des méthodes asynchrones et comment implémenter des opérations asynchrones dans Media Foundation.                                                                |
| [Files d’attente de travail](work-queues.md)                                                  | Une file d’attente de travail est un moyen efficace d’effectuer des opérations asynchrones sur un autre thread.                                                                            |
| [Générateurs d’événements de média](media-event-generators.md)                            | Comment recevoir des événements asynchrones et comment déclencher des événements dans Media Foundation.                                                                                   |
| [Interfaces de service](service-interfaces.md)                                    | Une interface de service est une interface COM fournie par un objet, mais exposée à l’application par le biais d’un autre objet.                                             |
| [Objets d’activation](activation-objects.md)                                    | Un objet d’activation est un objet qui crée un autre objet.                                                                                                    |
| [Horloge de présentation](presentation-clock.md)                                    | L’horloge de présentation génère l’heure de l’horloge qui est utilisée pour contrôler la lecture, ainsi que les flux audio et vidéo synchrones.                                |
