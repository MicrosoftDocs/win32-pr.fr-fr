# <a name="mf_mt_d3d12_resource_flag_allow_cross_adapter-attribute"></a>\_Indicateur de \_ ressource D3D12 MF MT \_ \_ \_ allow \_ travers l' \_ attribut Cross adapter

Indique si les ressources du flux peuvent être utilisées pour les données entre les adaptateurs.

## <a name="data-type"></a>Type de données

**UINT32**

## <a name="remarks"></a>Notes

La valeur de cet attribut correspond au membre D3D12_RESOURCE_FLAG_ALLOW_CROSS_ADAPTER de l’énumération [D3D12_RESOURCE_FLAGS](/windows/win32/api/d3d12/ne-d3d12-d3d12_resource_flags) .
