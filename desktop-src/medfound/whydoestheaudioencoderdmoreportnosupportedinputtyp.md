# <a name="why-does-the-audio-encoder-report-no-supported-input-types"></a>Pourquoi l’encodeur audio signale-t-il aucun type d’entrée pris en charge ?

L’encodeur audio ne peut pas déterminer les types d’entrée pris en charge tant que vous n’avez pas défini un type de sortie. Énumérez les types de sortie et sélectionnez celui qui correspond à vos spécifications. Une fois que vous avez défini le type de sortie, l’objet retourne les types d’entrée pris en charge.
