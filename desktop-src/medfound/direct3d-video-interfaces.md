# <a name="direct3d-9-video-interfaces"></a>Interfaces vidéo Direct3D 9

Décrit les interfaces vidéo de Microsoft Direct3D 9.



| Interface                                                                | Description                                                                                                |
|--------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------|
| [**IDirect3D9ExOverlayExtension**](/windows/desktop/api/d3d9/nn-d3d9-idirect3d9exoverlayextension)     | Interroge les fonctionnalités matérielles de superposition d’un appareil Direct3D.                                            |
| [**IDirect3DAuthenticatedChannel9**](/windows/desktop/api/d3d9/nn-d3d9-idirect3dauthenticatedchannel9) | Fournit un canal de communication au pilote Graphics ou au runtime Direct3D.                        |
| [**IDirect3DCryptoSession9**](/windows/desktop/api/d3d9/nn-d3d9-idirect3dcryptosession9)               | Représente une session de chiffrement.                                                                        |
| [**IDirect3DDevice9Video**](/windows/desktop/api/d3d9/nn-d3d9-idirect3ddevice9video)                   | Permet à une application d’utiliser les services de protection et de chiffrement de contenu implémentés par un pilote Graphics. |
| [**IDirect3DDXVADevice9**](idirect3ddxvadevice9.md)                     | Représente un accélérateur matériel pour DirectX Video Acceleration (DXVA) 1,0.                               |
| [**IDirect3DVideoDevice9**](idirect3dvideodevice9.md)                   | Active le décodage accéléré par le matériel à partir d’un périphérique Direct3D, à l’aide de la version DXVA 1,0.                      |
