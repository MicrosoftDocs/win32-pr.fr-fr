# <a name="media-foundation-primitives"></a>Primitives de Media Foundation

Media Foundation définit plusieurs types d’objets de base utilisés dans les API Media Foundation.



| Rubrique                                       | Description                                                                         |
|---------------------------------------------|-------------------------------------------------------------------------------------|
| [Attributs](attributes-and-properties.md) | Les attributs et les propriétés sont des paires clé/valeur stockées sur un objet.                  |
| [Types de médias](media-types.md)              | Un type de média décrit le format d’un flux multimédia numérique.                        |
| [Mémoires tampons de média](media-buffers.md)          | Une mémoire tampon de média gère un bloc de mémoire, afin qu’elle puisse être partagée entre des objets. |
| [Exemples de supports](media-samples.md)          | Un exemple de média est un objet qui contient une liste de mémoires tampons de média.                  |
