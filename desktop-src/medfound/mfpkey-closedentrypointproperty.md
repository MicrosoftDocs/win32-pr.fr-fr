# <a name="mfpkey_closedentrypoint-property"></a>MFPKEY \_ propriété CLOSEDENTRYPOINT

Spécifie le modèle d’encodage que l’encodeur doit utiliser au début d’un groupe d’images.

## <a name="constant-for-ipropertybag"></a>Constante pour IPropertyBag

**\_wszWMVCClosedEntryPoint g**

## <a name="data-type"></a>Type de données

**VT \_ bool**

## <a name="default-value"></a>Valeur par défaut

**VARIANTE \_ false**

## <a name="remarks"></a>Notes

Si vous affectez à cette propriété la **\_ valeur variant true**, le modèle d’encodage est BBIBBP. Si vous affectez à cette propriété la valeur **Variant \_ false**, le modèle d’encodage est IBBPBBP.
