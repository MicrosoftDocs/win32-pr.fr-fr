# <a name="mf_mt_video_lighting-attribute"></a>\_Attribut d' \_ éclairage vidéo MF MT \_

Spécifie les conditions d’éclairage optimales pour un type de média vidéo.

## <a name="data-type"></a>Type de données

**UINT32**

## <a name="remarks"></a>Notes

La valeur de cet attribut est un membre de l’énumération [**MFVideoLighting**](/windows/desktop/api/mfobjects/ne-mfobjects-mfvideolighting) .

La constante GUID de cet attribut est exportée à partir de mfuuid. lib.
