# <a name="mfpkey_fullframerate-property"></a>MFPKEY \_ propriété FULLFRAMERATE

Spécifie si le endoder est autorisé à déposer des frames.

## <a name="constant-for-ipropertybag"></a>Constante pour IPropertyBag

**\_wszWMVCFullFrameRate g**

## <a name="data-type"></a>Type de données

**VT \_**

## <a name="remarks"></a>Notes

Si vous affectez la valeur 1 à cette propriété, l’encodeur ne supprimera pas les frames. Si vous affectez à cette propriété la valeur 0, l’encodeur peut supprimer des frames.
