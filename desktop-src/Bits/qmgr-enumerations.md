# <a name="qmgr-enumerations"></a>Énumérations QMGR

\[Les énumérations du gestionnaire de files d’attente (QMGR) peuvent être utilisées dans les systèmes d’exploitation spécifiés dans la section Configuration requise de la rubrique. Il est possible qu’ils soient modifiés ou indisponibles dans les versions ultérieures. Utilisez plutôt les [énumérations bits](bits-enumerations.md).\]

Les [interfaces](qmgr-interfaces.md) du gestionnaire de files d’attente (QMGR) utilisent l’énumération suivante.



| Énumération                               | Description                                                                              |
|-------------------------------------------|------------------------------------------------------------------------------------------|
| [**GROUPPROP**](/windows/desktop/api/Qmgr/ne-qmgr-groupprop)<br/> | Définit les valeurs de constante pour la récupération et la définition des valeurs de propriété de groupe.<br/> |
