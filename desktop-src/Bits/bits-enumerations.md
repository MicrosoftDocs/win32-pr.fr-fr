# <a name="bits-enumerations"></a>Énumérations BITS

Les [interfaces service de transfert intelligent en arrière-plan (bits)](bits-interfaces.md) utilisent les énumérations suivantes.

## <a name="in-this-section"></a>Contenu de cette section

| Rubrique | Description |
|-|-|
| [**BG_AUTH_SCHEME**](/windows/win32/api/bits1_5/ne-bits1_5-bg_auth_scheme) | Définit des constantes qui spécifient le schéma d’authentification à utiliser lorsqu’un proxy ou un serveur demande l’authentification de l’utilisateur. |
| [**BG_AUTH_TARGET**](/windows/win32/api/bits1_5/ne-bits1_5-bg_auth_target) | Définit des constantes qui spécifient si les informations d’identification sont utilisées pour les demandes d’authentification utilisateur ou de serveur proxy. |
| [**BG_CERT_STORE_LOCATION**](/windows/win32/api/bits2_5/ne-bits2_5-bg_cert_store_location) | Définit des constantes qui spécifient l’emplacement du magasin de certificats. |
| [**BG_ERROR_CONTEXT**](/windows/win32/api/bits/ne-bits-bg_error_context) | Définit des constantes qui spécifient le contexte dans lequel l’erreur s’est produite. |
| [**BG_JOB_PRIORITY**](/windows/win32/api/bits/ne-bits-bg_job_priority) | Définit des constantes qui spécifient le niveau de priorité d’un travail.  |
| [**BG_JOB_PROXY_USAGE**](/windows/win32/api/bits/ne-bits-bg_job_proxy_usage) | Définit les constantes qui spécifient le proxy à utiliser pour les transferts de fichiers. Vous pouvez définir différents paramètres de proxy pour chaque travail. |
| [**BG_JOB_STATE**](/windows/win32/api/bits/ne-bits-bg_job_state) | Définit des constantes qui spécifient les différents États d’un travail. |
| [**BG_JOB_TYPE**](/windows/win32/api/bits/ne-bits-bg_job_type) | Définit des constantes qui spécifient le type de tâche de transfert, par exemple télécharger. |
| [**BITS_COST_STATE**](./bits-cost-state.md) | Définit des constantes qui spécifient l’état du coût des BITS. |
| [**BITS_FILE_PROPERTY_ID**](/windows/win32/api/bits5_0/ne-bits5_0-bits_file_property_id) | Définit des constantes qui spécifient des valeurs d’ID correspondant aux propriétés du fichier de copie en arrière-plan. Consultez [IBackgroundCopyFile5 :: GetProperty](../delivery_optimization/ibackgroundcopyfile5-getproperty.md) et [SetProperty](../delivery_optimization/ibackgroundcopyfile5-setproperty.md). |
| [**BITS_JOB_PROPERTY_ID**](/windows/win32/api/bits5_0/ne-bits5_0-bits_job_property_id) | Définit des constantes qui spécifient l’ID de la propriété pour le travail BITS. |
| [**BITS_JOB_TRANSFER_POLICY**](/windows/win32/api/bits5_0/ne-bits5_0-bits_job_transfer_policy) | Définit des constantes qui spécifient des valeurs d’ID correspondant aux propriétés BITS. |
