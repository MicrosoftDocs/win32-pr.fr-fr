# <a name="using-the-iis-ui-to-configure-the-virtual-directory"></a>Utilisation de l’interface utilisateur IIS pour configurer le répertoire virtuel

Pour utiliser l’interface utilisateur IIS afin de configurer le répertoire virtuel pour les téléchargements BITS, sélectionnez **Propriétés** dans le menu contextuel du répertoire virtuel. Appuyez sur l’onglet **extension bits** pour accéder à la page de propriétés bits (l’extension ISAPI du serveur bits doit être installée). Modifiez les valeurs de propriété par défaut comme il convient et activez le répertoire pour les téléchargements. vous ne pouvez pas télécharger des fichiers tant que le répertoire n’est pas activé. Pour plus d’informations sur la définition des valeurs de propriété, consultez l’aide en ligne.
