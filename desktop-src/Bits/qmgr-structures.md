# <a name="qmgr-structures"></a>Structures QMGR

\[Les structures du gestionnaire de files d’attente (QMGR) peuvent être utilisées dans les systèmes d’exploitation spécifiés dans la section Configuration requise de la rubrique. Il est possible qu’ils soient modifiés ou indisponibles dans les versions ultérieures. Utilisez plutôt les [structures et les unions bits](bits-c---structures-and-unions.md).\]

Les [interfaces](qmgr-interfaces.md) du gestionnaire de files d’attente (QMGR) utilisent les structures suivantes.



| Structure                                     | Objectif                                                                   |
|-----------------------------------------------|---------------------------------------------------------------------------|
| [**FILESETINFO**](/windows/desktop/api/Qmgr/ns-qmgr-filesetinfo)<br/> | Identifie les noms locaux et distants du fichier à télécharger.<br/> |
| **\_heures de copie en arrière-plan \_**<br/>        | Non pris en charge.<br/>                                                 |
