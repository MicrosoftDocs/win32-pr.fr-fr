# <a name="ibackgroundcopycallback3-methods"></a>Méthodes IBackgroundCopyCallback3

L’interface [**IBackgroundCopyCallback3**](/windows/desktop/api/Bits10_1/nn-bits10_1-ibackgroundcopycallback3) expose les méthodes suivantes.

## <a name="in-this-section"></a>Contenu de cette section



| Rubrique                                                                                             | Description                                                                                                                                                                                                                                                                                                                             |
|---------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Méthode FileRangesTransferred**](/windows/desktop/api/Bits10_1/nf-bits10_1-ibackgroundcopycallback3-filerangestransferred)<br/> | BITS appelle votre implémentation de la méthode [**FileRangesTransferred**](/windows/desktop/api/Bits10_1/nf-bits10_1-ibackgroundcopycallback3-filerangestransferred) quand une ou plusieurs plages de fichiers ont été téléchargées. Des plages de fichiers sont ajoutées à la tâche à l’aide de la méthode [**IBackgroundCopyFile6 :: RequestFileRanges**](/windows/desktop/api/Bits10_1/nf-bits10_1-ibackgroundcopyfile6-requestfileranges) .<br/> |
