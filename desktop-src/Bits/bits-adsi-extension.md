# <a name="bits-adsi-extension"></a>Extension ADSI BITS

Le serveur BITS fournit les interfaces d’extension ADSI suivantes pour activer les téléchargements BITS.



| Interface                                                        | Description                                                                                                                                    |
|------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IBITSExtensionSetup**](/windows/desktop/api/Bitscfg/nn-bitscfg-ibitsextensionsetup)               | Utilisez pour activer et désactiver les téléchargements BITS vers un répertoire virtuel.                                                                                 |
| [**IBITSExtensionSetupFactory**](/windows/desktop/api/Bitscfg/nn-bitscfg-ibitsextensionsetupfactory) | Utilisez pour récupérer un pointeur vers une interface [**IBITSExtensionSetup**](/windows/desktop/api/Bitscfg/nn-bitscfg-ibitsextensionsetup) dans un programme d’installation qui installe le serveur bits. |
