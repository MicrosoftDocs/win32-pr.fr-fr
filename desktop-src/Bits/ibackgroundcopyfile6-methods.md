# <a name="ibackgroundcopyfile6-methods"></a>Méthodes IBackgroundCopyFile6

L’interface [**IBackgroundCopyFile6**](/windows/desktop/api/bits10_1/nn-bits10_1-ibackgroundcopyfile6) expose les méthodes suivantes.

## <a name="in-this-section"></a>Contenu de cette section



| Rubrique                                                                                           | Description                                                                   |
|-------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------|
| [**Méthode UpdateDownloadPosition**](/windows/desktop/api/Bits10_1/nf-bits10_1-ibackgroundcopyfile6-updatedownloadposition)<br/> | Spécifie la position à partir de laquelle télécharger les données manquantes. <br/> |
| [**Méthode RequestFileRanges**](/windows/desktop/api/Bits10_1/nf-bits10_1-ibackgroundcopyfile6-requestfileranges)<br/>           | Ajoute un nouvel ensemble de plages de fichiers à hiérarchiser pour le téléchargement. <br/>     |
| [**Méthode GetFilledFileRanges**](/windows/desktop/api/Bits10_1/nf-bits10_1-ibackgroundcopyfile6-getfilledfileranges)<br/>       | Retourne l’ensemble des plages de fichiers qui ont été téléchargées.<br/>          |
