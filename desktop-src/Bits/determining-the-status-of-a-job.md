# <a name="determining-the-status-of-a-job"></a>Détermination de l’état d’un travail

L’état d’un travail fait référence aux modifications de l’état d’un travail (par exemple, le travail passe à l’état de \_ travail BG \_ \_ transférer vers \_ l’état de travail BG \_ \_ transféré) et à la progression du transfert des fichiers. Pour capturer les modifications de l’état d’un travail, utilisez les options suivantes :

-   [Interrogez l’état du travail](polling-for-the-status-of-the-job.md).
-   [Inscrivez-vous pour recevoir la notification d’événement](registering-a-com-callback.md).
-   [Inscrivez-vous pour exécuter un programme](registering-to-execute-a-program.md) basé sur un événement.

Pour obtenir des informations sur la progression d’un travail, consultez [détermination de la progression d’un travail](determining-the-progress-of-a-job.md).
