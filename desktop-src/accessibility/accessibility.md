# <a name="windows-accessibility"></a>Accessibilité de Windows

Cette section contient la documentation pour les développeurs Windows qui conçoivent des applications accessibles, les développeurs de technologie d’assistance qui créent des outils tels que les lecteurs d’écran et les loupes, ainsi que les ingénieurs de test de logiciels qui créent des scripts automatisés pour tester des applications Windows

Utilisez les ressources fournies ici pour créer des applications Windows accessibles qui peuvent être utilisées par autant de personnes que possible, y compris celles handicapées, les préférences personnelles et les styles de travail spécifiques.
