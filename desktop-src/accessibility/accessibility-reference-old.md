# <a name="windows-accessibility-and-automation-reference"></a>Informations de référence sur l’accessibilité et l’accessibilité Windows

L’accessibilité Windows prend en charge deux catégories de fonctionnalités pour aider les développeurs Windows à concevoir des applications accessibles, les développeurs de technologie d’assistance peuvent créer des outils tels que les lecteurs d’écran et les loupes, et les ingénieurs de test de logiciel créent des scripts automatisés pour tester les applications Windows : les paramètres utilisateur et un ensemble complet d’API Win32.

Pour plus d’informations, consultez [fonctionnalités d’accessibilité de Windows](../WinAuto/about-windows-accessibility-features.md).
