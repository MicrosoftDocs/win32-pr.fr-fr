# <a name="developing-assistive-technology-for-windows"></a>Développement des technologies d’assistance pour Windows

Les produits de technologie d’assistance tiers, tels que les lecteurs d’écran, les loupes et le matériel spécialisé, sont des outils essentiels pour de nombreux utilisateurs de Windows (et d’autres produits Microsoft).

Cette vue d’ensemble décrit comment créer des outils de technologie d’assistance qui sont compatibles avec les produits Microsoft pour les personnes souffrant de troubles de la vision, de l’apprentissage, de la dextérité et de la mobilité, ainsi que les limitations de langue et de communication.
