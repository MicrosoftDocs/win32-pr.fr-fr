# <a name="developing-accessible-applications-for-windows"></a>Développement d’applications accessibles pour Windows

Chaque infrastructure d’application Windows prend en charge de nombreuses fonctionnalités d’accessibilité qui permettent aux utilisateurs handicapés, aux préférences personnelles ou aux styles de travail spécifiques d’utiliser correctement n’importe quel appareil Windows qu’ils choisissent.

Cette vue d’ensemble décrit les fonctionnalités d’accessibilité prises en charge dans chaque infrastructure, ainsi que la façon dont vous pouvez les incorporer au fur et à mesure que vous générez votre application.
