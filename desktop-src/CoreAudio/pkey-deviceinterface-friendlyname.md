# <a name="pkey_deviceinterface_friendlyname"></a>\_Deviceinterface \_ FriendlyName FriendlyName

La **propriété \_ hyperDeviceInterface \_ FriendlyName FriendlyName** contient le nom convivial de la carte audio à laquelle l’appareil de point de terminaison est attaché (par exemple, « carte audio XYZ »).

Le membre **VT** de la structure **PROPVARIANT** est défini sur VT \_ LPWStr.

Le membre **pwszVal** de la structure **PROPVARIANT** pointe vers une chaîne de caractères larges se terminant par un caractère null qui contient le nom convivial.
