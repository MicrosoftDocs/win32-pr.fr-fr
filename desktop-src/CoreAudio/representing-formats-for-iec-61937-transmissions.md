<span data-ttu-id="53af3-104">Avec l’augmentation des appareils de stockage multimédia qui requièrent des formats audio compressés, les applications doivent identifier, décrire et utiliser un grand nombre de nouveaux contenus audio encodés pour transmettre du contenu à partir de PC à des appareils tels que HDMI ou DisplayPort récepteur.</span><span class="sxs-lookup"><span data-stu-id="53af3-104">With the increase in media storage devices that require compressed audio formats, applications must identify, describe, and use a variety of new encoded audio content for transmitting content from PCs to devices such as HDMI or DisplayPort receiver.</span></span>

<span data-ttu-id="53af3-105">Pour représenter un flux audio encodé à transmettre sur une interface compatible IEC 61937, une application doit fournir les éléments suivants :</span><span class="sxs-lookup"><span data-stu-id="53af3-105">To represent an encoded audio stream to be transmitted over an IEC 61937-compatible interface, an application must provide:</span></span>

-   <span data-ttu-id="53af3-106">Caractéristiques d’un flux audio encodé à transmettre.</span><span class="sxs-lookup"><span data-stu-id="53af3-106">The characteristics of an encoded audio stream to be transmitted.</span></span>

<span data-ttu-id="53af3-108">Dans Windows Vista et les systèmes d’exploitation Windows antérieurs, une application peut déduire le niveau de qualité d’un format audio à partir du nombre de canaux, de la taille de l’échantillon et du débit de données d’un flux audio qui utilise le format.</span><span class="sxs-lookup"><span data-stu-id="53af3-108">In Windows Vista and earlier Windows operating data-ttu-id="53af3-109">Pour un format PCM, ces informations sont disponibles à partir des membres **nChannels**, **nSamplesPerSec** et **nAvgBytesPerSec** de la structure **WAVEFORMATEX** qui spécifie le format.</span><span class="sxs-lookup"><span data-stu-id="53af3-109">For a PCM format, this information is available from the **nChannels**, **nSamplesPerSec**, and **nAvgBytesPerSec** members of the **WAVEFORMATEX** structure that specifies the format.</span></span> <span data-ttu-id="53af3-110">Pour un format non-PCM, ces trois membres ont été commandeered pour stocker des informations sur les données compressées dans le flux audio.</span><span class="sxs-lookup"><span data-stu-id="53af3-110">For a non-PCM format, these three members have been commandeered to store information about the compressed data in the audio stream.</span></span> <span data-ttu-id="53af3-111">Par conséquent, la structure **WAVEFORMATEX** ne dispose pas d’informations sur le nombre le niveau de qualité du flux non-PCM.</span><span class="sxs-lookup"><span data-stu-id="53af3-112">Based on the information in this structure, a user or an application might have difficulty inferring <span data-ttu-id="53af3-113">**WAVEFORMATEX** a été étendu à la structure **WAVEFORMATEXTENSIBLE** pour fournir les caractéristiques de flux supplémentaires.</span><span class="sxs-lookup"><span data-stu-id="53af3-113">The **WAVEFORMATEX** was extended to the **WAVEFORMATEXTENSIBLE** structure to provide the extra stream characteristics.</span></span> <span data-ttu-id="53af3-114">Toutefois, cette structure ne convient pas non plus à la description du flux des transmissions IEC 61937, car elle a été conçue pour représenter un ensemble unique de caractéristiques et utilisée pour les données PCM multicanaux non compressées.</span><span class="sxs-lookup"><span data-stu-id="53af3-114">However, this structure is also not adequate in describing the stream for IEC 61937 transmissions because it was intended to represent a single set of characteristics and used for uncompressed, multi-channel PCM data.</span></span>

<span data-ttu-id="53af3-118">La structure **WAVEFORMATEXTENSIBLE \_ IEC61937** est déclarée dans l’en-tête KsMedia. h inclus dans le kit de développement logiciel (SDK) Windows 7.</span><span class="sxs-lookup"><span data-stu-id="53af3-118">The **WAVEFORMATEXTENSIBLE\_IEC61937** structure is declared in the KsMedia.h header included with the Windows 7 SDK.</span></span> <span data-ttu-id="53af3-119">Le membre **FormatExt** est la structure **WAVEFORMATEXTENSIBLE** qui stocke les caractéristiques du flux à transmettre.</span><span class="sxs-lookup"><span data-stu-id="53af3-119">The **FormatExt** member is the **WAVEFORMATEXTENSIBLE** structure that stores the characteristics of the stream to be transmitted.</span></span> <span data-ttu-id="53af3-120">Le membre de **format** de la structure **WAVEFORMATEXTENSIBLE** est la structure **WAVEFORMATEX** .</span><span class="sxs-lookup"><span data-stu-id="53af3-120">The **Format** member of the **WAVEFORMATEXTENSIBLE** structure is the **WAVEFORMATEX** structure.</span></span> <span data-ttu-id="53af3-121">Le contenu de ce **WAVEFORMATEX** et **WAVEFORMATEXTENSIBLE** indique à une application si la structure peut être interprétée comme une structure **\_ IEC61937 WAVEFORMATEXTENSIBLE** .</span><span class="sxs-lookup"><span data-stu-id="53af3-121">The contents of this **WAVEFORMATEX** and **WAVEFORMATEXTENSIBLE** indicate to an application whether the structure can be interpreted as a **WAVEFORMATEXTENSIBLE\_IEC61937** structure.</span></span> <span data-ttu-id="53af3-122">Pour une structure **WAVEFORMATEXTENSIBLE \_ IEC61937** :</span><span class="sxs-lookup"><span data-stu-id="53af3-122">For a **WAVEFORMATEXTENSIBLE\_IEC61937** structure:</span></span>
