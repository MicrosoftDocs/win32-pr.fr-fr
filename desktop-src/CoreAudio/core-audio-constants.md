# <a name="core-audio-constants"></a>Constantes audio principales

Cette section décrit les constantes utilisées par les API audio de base dans Windows Vista et versions ultérieures.

| Constante                                                                            | Description                                                                         |
|-------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------|
| [**AUDCLNT \_ SESSIONFLAGS \_ xxx**](audclnt-sessionflags-xxx-constants.md)<br/> | Indicateurs de la session audio ouverte pendant l’initialisation du flux.<br/> |
| [**AUDCLNT \_ STREAMFLAGS \_ xxx**](audclnt-streamflags-xxx-constants.md)              | Indicateurs d’initialisation pour un flux audio.                                           |
| [**État de l’appareil \_ \_ xxx**](device-state-xxx-constants.md)                            | Indicateurs d’État pour un périphérique de point de terminaison audio.                                           |
| [**SUPPORT matériel de point de terminaison \_ \_ \_ xxx**](endpoint-hardware-support-xxx-constants.md) | Indicateurs de support matériel pour un périphérique de point de terminaison audio.                                |
