# <a name="core-audio-functions"></a>Fonctions audio principales

Cette section décrit les fonctions utilisées par les API audio principales à partir de Windows 8.



| Fonction                                                           | Description                                                                                                                                                                         |
|--------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**ActivateAudioInterfaceAsync**](/windows/desktop/api/mmdeviceapi/nf-mmdeviceapi-activateaudiointerfaceasync) | permet aux applications de Windows Store d’accéder aux interfaces COM (component Object Model) préexistantes de la famille [WASAPI](wasapi.md) .<br/> S’applique à partir de Windows 8.<br/> |
