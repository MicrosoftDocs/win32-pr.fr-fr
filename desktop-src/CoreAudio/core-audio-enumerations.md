# <a name="core-audio-enumerations"></a>Énumérations audio principales

cette section décrit les énumérations utilisées par les api Audio principales dans Windows Vista et versions ultérieures.



| Énumération                                                                   | Description                                                                                        |
|-------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------|
| [**\_AUDCLNT \_ BUFFERFLAGS**](/windows/win32/api/audioclient/ne-audioclient-_audclnt_bufferflags)                        | Indicateurs d’État pour une mémoire tampon de point de terminaison audio.                                                         |
| [**AUDCLNT \_ SHAREMODE**](/windows/desktop/api/Audiosessiontypes/ne-audiosessiontypes-audclnt_sharemode)                               | Mode de partage d’un flux.                                                                     |
| [**AUDCLNT \_ STREAMOPTIONS**](/windows/desktop/api/audioclient/ne-audioclient-audclnt_streamoptions)                       | Définit des valeurs qui décrivent les caractéristiques d’un flux audio.                               |
| [**\_catégorie de flux audio \_**](/windows/desktop/api/audiosessiontypes/ne-audiosessiontypes-audio_stream_category)                      | Spécifie la catégorie d’un flux audio.                                                         |
| [**AudioSessionState**](/windows/desktop/api/Audiosessiontypes/ne-audiosessiontypes-audiosessionstate)                                | État d’une session audio.                                                                     |
| [**ConnectorType**](/windows/win32/api/devicetopology/ne-devicetopology-connectortype)                                        | Type de connecteur dans une topologie d’appareil.                                                        |
| [**DataFlow**](/windows/win32/api/devicetopology/ne-devicetopology-dataflow)                                                  | Direction du flux de données d’un flux audio.                                                        |
| [**EDataFlow**](/windows/win32/api/mmdeviceapi/ne-mmdeviceapi-edataflow)                                                | Sens dans lequel les données audio circulent entre un périphérique de point de terminaison audio et une application cliente. |
| [**EndpointFormFactor**](/windows/win32/api/mmdeviceapi/ne-mmdeviceapi-endpointformfactor)                              | Attributs physiques généraux d’un périphérique de point de terminaison audio.                                       |
| [**ERole**](/windows/win32/api/mmdeviceapi/ne-mmdeviceapi-erole)                                                        | Rôle de système d’un périphérique de point de terminaison audio.                                                       |
| [**\_récepteur KSJACK \_ ConnectionType**](/windows/win32/api/devicetopology/ne-devicetopology-ksjack_sink_connectiontype)<br/> | Type de connexion.<br/>                                                                 |
| [**PartType**](/windows/win32/api/devicetopology/ne-devicetopology-parttype)                                                  | Type de partie d’une partie (connecteur ou sous-unité) dans une topologie d’appareil.                               |
