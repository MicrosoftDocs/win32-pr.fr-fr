# <a name="pkey_sfx_processingmodes_supported_for_streaming-property"></a>\_SFX \_ ProcessingModes \_ pris en charge pour la \_ \_ propriété streaming

Répertorie les modes de traitement de signal pris en charge par l’APO d’effet de flux.

Type de PROPVARIANT (VT) : VT **\_ Vector** \| **\_** VT VT

## <a name="remarks"></a>Remarques

Cette liste comprend uniquement les modes de traitement des signaux dans lesquels l’APO traite réellement le signal audio pendant la diffusion en continu.

Cette liste n’inclut aucun mode de traitement de signal pris en charge par le APO à des fins de découverte uniquement.
