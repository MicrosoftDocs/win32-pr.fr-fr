# <a name="iremotedesktopclientsettings-methods"></a>Méthodes IRemoteDesktopClientSettings

L’interface [**IRemoteDesktopClientSettings**](/windows/win32/api/rdpappcontainerclient/nn-rdpappcontainerclient-iremotedesktopclientsettings) expose les méthodes suivantes.

## <a name="in-this-section"></a>Dans cette section

<dl> <dt>

[**Méthode ApplySettings**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclientsettings-applysettings)
</dt> <dd>

Stocke le contenu spécifié dans le fichier RDP.

</dd> <dt>

[**Méthode GetRdpProperty**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclientsettings-getrdpproperty)
</dt> <dd>

Récupère une valeur de propriété RDP nommée unique.

</dd> <dt>

[**Méthode RetrieveSettings**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclientsettings-retrievesettings)
</dt> <dd>

Récupère l’ensemble du fichier RDP sous forme de chaîne.

</dd> <dt>

[**Méthode SetRdpProperty**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclientsettings-setrdpproperty)
</dt> <dd>

Définit la valeur d’une propriété RDP nommée unique.

</dd> </dl>
