# <a name="iremotedesktopclienttouchpointer-properties"></a>Propriétés IRemoteDesktopClientTouchPointer

L’interface [**IRemoteDesktopClientTouchPointer**](/windows/win32/api/rdpappcontainerclient/nn-rdpappcontainerclient-iremotedesktopclienttouchpointer) expose les propriétés suivantes.

## <a name="in-this-section"></a>Dans cette section

<dl> <dt>

[**Propriété Enabled**](iremotedesktopclienttouchpointer-enabled.md)
</dt> <dd>

Indique si la fonctionnalité de pointeur tactile est activée sur le contrôle client du conteneur d’applications RDP.

</dd> <dt>

[**EventsEnabled, propriété**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclienttouchpointer-get_eventsenabled)
</dt> <dd>

Indique si les notifications d’événements du pointeur tactile sont activées pour le contrôle client du conteneur d’applications RDP.

</dd> <dt>

[**Propriété PointerSpeed**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclienttouchpointer-get_pointerspeed)
</dt> <dd>

La vitesse à laquelle le curseur tactile se déplacera sur le bureau virtuel par rapport à la vitesse du mouvement sur le client.

</dd> </dl>
