# <a name="iremotedesktopclient-methods"></a>Méthodes IRemoteDesktopClient

L’interface [**IRemoteDesktopClient**](/windows/win32/api/rdpappcontainerclient/nn-rdpappcontainerclient-iremotedesktopclient) expose les méthodes suivantes.

## <a name="in-this-section"></a>Dans cette section

<dl> <dt>

[**méthode attachEvent**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclient-attachevent)
</dt> <dd>

Attache un gestionnaire d’événements à un événement.

</dd> <dt>

[**méthode Connecter**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclient-connect)
</dt> <dd>

Établit une connexion à l’aide des propriétés actuellement définies sur le contrôle client du conteneur d’applications protocole RDP (Remote Desktop Protocol) (RDP).

</dd> <dt>

[**Méthode DeleteSavedCredentials**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclient-deletesavedcredentials)
</dt> <dd>

Supprime les informations d’identification enregistrées pour l’ordinateur distant spécifié.

</dd> <dt>

[**méthode detachEvent**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclient-detachevent)
</dt> <dd>

Détache un gestionnaire d’événements d’un événement.

</dd> <dt>

[**Méthode Disconnect**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclient-disconnect)
</dt> <dd>

Déconnecte la connexion active.

</dd> <dt>

[**Reconnect, méthode**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclient-reconnect)
</dt> <dd>

Lance une reconnexion automatique du contrôle client du conteneur d’applications protocole RDP (Remote Desktop Protocol) (RDP) pour ajuster la taille de la session à la nouvelle largeur et à la hauteur.

</dd> <dt>

[**Méthode UpdateSessionDisplaySettings**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclient-updatesessiondisplaysettings)
</dt> <dd>

Met à jour les paramètres de largeur et de hauteur pour le contrôle client du conteneur d’applications protocole RDP (Remote Desktop Protocol) (RDP).

</dd> </dl>
