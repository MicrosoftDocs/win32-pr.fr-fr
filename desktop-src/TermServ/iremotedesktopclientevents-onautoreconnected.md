# <a name="iremotedesktopclienteventsonautoreconnected-method"></a>IRemoteDesktopClientEvents :: OnAutoReconnected, méthode

Appelé lorsque le contrôle client se reconnecte automatiquement à une session à distance.

## <a name="syntax"></a>Syntaxe


```C++
void OnAutoReconnected();
```



## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.

## <a name="return-value"></a>Valeur retournée

Cette méthode ne retourne pas de valeur.
