# <a name="itssbservicenotification-methods"></a>Méthodes ITsSbServiceNotification

Les interfaces [**ITsSbServiceNotification**](/windows/desktop/api/sbtsv/nn-sbtsv-itssbservicenotification) exposent les méthodes suivantes.

## <a name="in-this-section"></a>Dans cette section

<dl> <dt>

[**Méthode NotifyServiceFailure**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbservicenotification-notifyservicefailure)
</dt> <dd>

Avertit Connexion Bureau à distance les plug-ins enregistrés que le Service Broker pour les connexions Bureau à distance a été arrêté.

</dd> <dt>

[**Méthode NotifyServiceSuccess**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbservicenotification-notifyservicesuccess)
</dt> <dd>

Notifie les plug-ins enregistrés que le service du Service Broker pour les connexions Bureau à distance a démarré.

</dd> </dl>
