# <a name="imsrdpclientnonscriptable3drivecollection-property"></a>IMsRdpClientNonScriptable3 ::D propriété riveCollection

Récupère la collection d’objets Drive à rediriger.

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```C++
HRESULT get_DriveCollection(
  [out] IMsRdpDriveCollection **ppDriveCollection
);
```



## <a name="property-value"></a>Valeur de la propriété

Récupère la collection d’objets Drive de type [**IMSRdpDrive**](imsrdpdrive.md).
