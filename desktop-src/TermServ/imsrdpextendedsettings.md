# <a name="imsrdpextendedsettings-interface"></a>Interface IMsRdpExtendedSettings

Utilisé pour définir et récupérer des propriétés nommées pour le contrôle client.

L’interface **IMsRdpExtendedSettings** hérite de l’interface [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) . **IMsRdpExtendedSettings** a également les types de membres suivants :

-   [Propriétés](#properties)

L’interface **IMsRdpExtendedSettings** possède les propriétés suivantes.
