# <a name="remoteapp-and-desktop-connection-management-service-classes"></a>Classes de service Administration des connexions aux programmes RemoteApp et aux services Bureau à distance

Le fournisseur WMI du service Administration des connexions aux programmes RemoteApp et aux services Bureau à distance fournit les classes suivantes.

## <a name="in-this-section"></a>Dans cette section

<dl> <dt>

[**\_CentralPublishingChangeEvent Win32**](win32-centralpublishingchangeevent.md)
</dt> <dd>

Événement qui représente une modification des paramètres RDV centraux.

</dd> <dt>

[**\_RDCentralPublishedDeploymentSettings Win32**](win32-rdcentralpublisheddeploymentsettings.md)
</dt> <dd>

Contient les paramètres de déploiement utilisés pour générer des fichiers RDP pour les ressources publiées à partir d’une batterie de serveurs.

</dd> <dt>

[**\_RDCentralPublishedFarm Win32**](win32-rdcentralpublishedfarm.md)
</dt> <dd>

Liste des batteries de serveurs à partir desquelles les ordinateurs de bureau ou les applications ont été publiés.

</dd> <dt>

[**\_RDCentralPublishedFileAssociation Win32**](win32-rdcentralpublishedfileassociation.md)
</dt> <dd>

Informations pour une extension de fichier associée à une application

</dd> <dt>

[**\_RDCentralPublishedRemoteApplication Win32**](win32-rdcentralpublishedremoteapplication.md)
</dt> <dd>

Décrit une application publiée sur un autre ordinateur, pour une utilisation à distance par le biais des services Terminal Server.

</dd> <dt>

[**\_RDCentralPublishedRemoteDesktop Win32**](win32-rdcentralpublishedremotedesktop.md)
</dt> <dd>

Desktop publié sur un autre ordinateur, pour une utilisation à distance via les services Terminal Server

</dd> <dt>

[**\_RDPersonalDesktopAssignment Win32**](win32-rdpersonaldesktopassignment.md)
</dt> <dd>

Liste des affectations de bureau personnelles.

</dd> <dt>

[**\_TSCPubPlugin Win32**](win32-tscpubplugin.md)
</dt> <dd>

Représente un plug-in configuré pour être utilisé par le biais du service Administration des connexions aux programmes RemoteApp et aux services Bureau à distance.

</dd> <dt>

[**\_TSLegacyPlugin Win32**](win32-tslegacyplugin.md)
</dt> <dd>

Représente un serveur de Bureau à distance que les plug-ins intégrés de service Administration des connexions aux programmes RemoteApp et aux services Bureau à distance interrogent pour les programmes RemoteApp.

</dd> <dt>

[**\_Espace de travail Win32**](win32-workspace.md)
</dt> <dd>

Spécifie Services Bureau à distance informations de configuration de l’espace de travail.

</dd> </dl>
