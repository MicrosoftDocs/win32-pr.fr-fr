# <a name="activateserverautomatic-method-of-the-win32_tslicenseserver-class"></a>Méthode ActivateServerAutomatic de la \_ classe Win32 TSLicenseServer

Active automatiquement le serveur de licences Bureau à distance via Internet.

## <a name="syntax"></a>Syntaxe


```mof
uint32 ActivateServerAutomatic(
  [out] uint32 ActivationStatus
);
```



## <a name="parameters"></a>Paramètres

<dl> <dt>

*ActivationStatus* \[ à\]
</dt> <dd>

L’état d’activation renvoyé peut être l’un des suivants.

<dt>

0
</dt> <dd>

Le serveur de licences Bureau à distance est activé.

</dd> <dt>

1
</dt> <dd>

Le serveur de licences Bureau à distance n’est pas activé.

</dd> <dt>

2
</dt> <dd>

Une erreur inconnue s'est produite. Vous ne savez pas si le serveur de licences Bureau à distance est activé.

</dd> </dl> </dd> </dl>

## <a name="return-value"></a>Valeur de retour

Si la méthode est réussie, elle retourne zéro. Si la méthode échoue, elle retourne une valeur différente de zéro. Pour obtenir la liste des codes d’erreur, consultez [services Bureau à distance codes d’erreur du fournisseur WMI](terminal-services-wmi-provider-error-codes.md).

## <a name="remarks"></a>Notes

Cette méthode échoue sauf si les propriétés **FirstName**, **LastName**, **Company** et **PaysRégion** sont définies.

Vous devez être membre du groupe administrateurs pour appeler cette méthode.
