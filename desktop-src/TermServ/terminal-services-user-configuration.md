# <a name="remote-desktop-services-user-configuration"></a>Configuration de l’utilisateur Services Bureau à distance

Les fonctions [**WTSQueryUserConfig**](/windows/desktop/api/Wtsapi32/nf-wtsapi32-wtsqueryuserconfiga) et [**WTSSetUserConfig**](/windows/desktop/api/Wtsapi32/nf-wtsapi32-wtssetuserconfiga) vous permettent de définir et de récupérer des informations de configuration pour un compte d’utilisateur spécifié. Le type d’énumération de la [**\_ \_ classe**](/windows/desktop/api/Wtsapi32/ne-wtsapi32-wts_config_class) de configuration WTS définit les paramètres de configuration utilisateur.
