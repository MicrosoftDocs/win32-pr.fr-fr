# <a name="imsrdpclient7advancedsettings8-property"></a>IMsRdpClient7 :: AdvancedSettings8, propriété

Récupère un objet qui prend en charge l’interface [**IMsRdpClientAdvancedSettings7**](imsrdpclientadvancedsettings7.md) .

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```C++
HRESULT get_AdvancedSettings8(
  [out, retval] IMsRdpClientAdvancedSettings7 **ppAdvSettings
);
```



## <a name="property-value"></a>Valeur de la propriété

Adresse d’un pointeur d’interface [**IMsRdpClientAdvancedSettings7**](imsrdpclientadvancedsettings7.md) qui reçoit l’objet.
