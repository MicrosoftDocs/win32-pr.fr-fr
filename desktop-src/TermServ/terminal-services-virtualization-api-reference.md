# <a name="remote-desktop-virtualization-api-reference"></a>Informations de référence sur l’API de virtualisation Bureau à distance

L’API de virtualisation Bureau à distance prend en charge les interfaces et les structures que vous pouvez utiliser pour créer des plug-ins personnalisés qui remplacent la logique de redirection standard de Connexion Bureau à distance Broker (Service Broker pour les connexions Bureau à distance). Les plug-ins personnalisés peuvent rediriger les connexions vers les serveurs Bureau à distance hôte de session (hôte de session Bureau à distance), ainsi que des machines virtuelles, des bureaux virtuels, des serveurs lames et des bureaux physiques.
