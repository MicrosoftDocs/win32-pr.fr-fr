# <a name="terminal-services-is-now-remote-desktop-services"></a>Les services Terminal Server sont désormais Services Bureau à distance

Les services Terminal Server ont été renommés Services Bureau à distance.

dans Windows Server 2008 R2, tous les Services de rôle Services Bureau à distance ont été renommés. Le tableau suivant répertorie l’ancien nom et le nouveau nom de chaque Services Bureau à distance service de rôle.



| Nom précédent                                                   | nom dans Windows Server 2008 R2                                         |
|-----------------------------------------------------------------|------------------------------------------------------------------------|
| Terminal Services <br/>                                   | Services Bureau à distance<br/>                                     |
| Terminal Server<br/>                                      | Hôte de session Bureau à distance<br/>               |
| Gestionnaire de licences des services Terminal Server (gestionnaire de licences TS)<br/>          | Gestionnaire de licences des services Bureau à distance<br/>                     |
| Passerelle des services Terminal Server (Passerelle TS)<br/>               | Passerelle des services Bureau à distance<br/>                         |
| Session Broker Terminal Services (Session Broker TS)<br/> | Connexion Bureau à distance Broker (Service Broker pour les connexions Bureau à distance)<br/>     |
| Accès Web Terminal Services (Accès Web TS)<br/>         | Accès web Bureau à distance (RD Accès web)<br/>                  |
| Virtualisation des services Terminal Server<br/>                     | Serveur hôte de virtualisation des services Bureau à distance<br/> |



 

Notez également qu’un serveur sur lequel est installé le service de rôle hôte de session Bureau à distance est maintenant appelé serveur hôte de session Bureau à distance, au lieu d’un serveur Terminal Server.
