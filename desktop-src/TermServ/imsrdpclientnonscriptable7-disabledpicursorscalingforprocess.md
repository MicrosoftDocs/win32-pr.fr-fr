# <a name="imsrdpclientnonscriptable7disabledpicursorscalingforprocess-method"></a>IMsRdpClientNonScriptable7 ::D méthode isableDpiCursorScalingForProcess

Désactive la mise à l’échelle locale du curseur de la souris reçue du serveur, garantissant ainsi un rendu correct de la forme du curseur sans modification.

## <a name="syntax"></a>Syntaxe

```C++
HRESULT DisableDpiCursorScalingForProcess();
```

## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.

## <a name="return-value"></a>Valeur retournée

Retourne **S \_ OK** en cas de réussite.
