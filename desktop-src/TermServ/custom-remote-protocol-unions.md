# <a name="remote-desktop-protocol-provider-unions"></a>Unions de fournisseurs protocole RDP (Remote Desktop Protocol)

L’API de protocole distant personnalisé prend en charge les unions suivantes.

## <a name="in-this-section"></a>Dans cette section

<dl> <dt>

[**\_paramètre de connexion Wrds \_**](/windows/desktop/api/Wtsdefs/ns-wtsdefs-wrds_connection_setting)
</dt> <dd>

Contient différents niveaux de paramètres pour une connexion Bureau à distance.

</dd> <dt>

[**paramètre de l' \_ écouteur Wrds \_**](/windows/desktop/api/Wtsdefs/ns-wtsdefs-wrds_listener_setting)
</dt> <dd>

Contient différents niveaux de paramètres d’écouteur pour une connexion Bureau à distance.

</dd> <dt>

[**\_paramètre Wrds**](/windows/desktop/api/Wtsdefs/ns-wtsdefs-wrds_setting)
</dt> <dd>

Contient différents niveaux de paramètres relatifs à la stratégie pour une connexion Bureau à distance.

</dd> <dt>

[**WTS \_ statistiques de CACHE non \_ \_**](/windows/desktop/api/Wtsdefs/ns-wtsdefs-wts_cache_stats_un)
</dt> <dd>

Contient des statistiques de cache.

</dd> </dl>
