# <a name="iremotedesktopclient-properties"></a>Propriétés IRemoteDesktopClient

L’interface [**IRemoteDesktopClient**](/windows/win32/api/rdpappcontainerclient/nn-rdpappcontainerclient-iremotedesktopclient) expose les propriétés suivantes.

## <a name="in-this-section"></a>Dans cette section

<dl> <dt>

[**Action, propriété**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclient-get_actions)
</dt> <dd>

Récupère l’objet actions pour le client de conteneur d’application protocole RDP (Remote Desktop Protocol) (RDP).

</dd> <dt>

[**Paramètres, propriété**](iremotedesktopclient-settings.md)
</dt> <dd>

Récupère l’objet de paramètres pour le client de conteneur d’applications protocole RDP (Remote Desktop Protocol) (RDP).

</dd> <dt>

[**Propriété TouchPointer**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclient-get_touchpointer)
</dt> <dd>

Contient l’objet [**RemoteDesktopClientTouchPointer**](/windows/win32/api/rdpappcontainerclient/nn-rdpappcontainerclient-iremotedesktopclienttouchpointer) pour le client de conteneur d’application protocole RDP (Remote Desktop Protocol) (RDP).

</dd> </dl>
