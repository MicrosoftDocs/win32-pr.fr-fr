# <a name="imsrdpclienttransportsettings4gatewaybrokeringtype-property"></a>IMsRdpClientTransportSettings4 :: GatewayBrokeringType, propriété

Cette propriété est en écriture seule.

## <a name="syntax"></a>Syntaxe


```C++
HRESULT put_GatewayBrokeringType(

);
```
