# <a name="imsrdpdrivev2-interface"></a>Interface IMsRdpDriveV2

Contient des informations sur un objet lecteur. Il s’agit d’une amélioration de l’interface [**IMsRdpDrive**](imsrdpdrive.md) .

## <a name="members"></a>Membres

L’interface **IMsRdpDriveV2** hérite de [**IMsRdpDrive**](imsrdpdrive.md). **IMsRdpDriveV2** a également les types de membres suivants :

-   [Propriétés](#properties)

### <a name="properties"></a>Propriétés

L’interface **IMsRdpDriveV2** possède les propriétés suivantes.



| Propriété                                                              | Type d’accès          | Description                                                |
|:----------------------------------------------------------------------|:---------------------|:-----------------------------------------------------------|
| [**DriveLetterIndex**](imsrdpdrivev2-driveletterindex.md)<br/> | Lecture seule<br/> | Contient l’index de la lettre du lecteur.<br/> |
