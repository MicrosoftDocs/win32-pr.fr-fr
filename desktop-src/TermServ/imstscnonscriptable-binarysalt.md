# <a name="imstscnonscriptablebinarysalt-property"></a>IMsTscNonScriptable :: BinarySalt, propriété

Cette propriété ne peut plus être utilisée.

Cette propriété est en lecture/écriture.

## <a name="syntax"></a>Syntaxe


```C++
HRESULT put_BinarySalt(
  [in]  BSTR newSalt
);

HRESULT get_BinarySalt(
  [out] BSTR *pSalt
);
```



## <a name="property-value"></a>Valeur de la propriété

Nouvelle partie Salt binaire pour un mot de passe encodé binaire.
