# <a name="imsrdpclient5remoteprogram-property"></a>IMsRdpClient5 :: RemoteProgram, propriété

Récupère un objet qui prend en charge l’interface [**ITSRemoteProgram**](itsremoteprogram.md) .

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```C++
HRESULT get_RemoteProgram(
  [out] ITSRemoteProgram **ppRemoteProgram
);
```



## <a name="property-value"></a>Valeur de la propriété

Pointeur d’interface [**ITSRemoteProgram**](itsremoteprogram.md) .
