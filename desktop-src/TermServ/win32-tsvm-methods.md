# <a name="win32_tsvm-methods"></a>\_Méthodes TSVm Win32

La classe [**Win32 \_ TSVm Methods**](win32-tsvm.md) expose les méthodes suivantes.

## <a name="in-this-section"></a>Dans cette section

<dl> <dt>

[**Méthode QueryOfflineInformation**](queryofflineinformation-win32-tsvm.md)
</dt> <dd>

Récupère les propriétés du serveur de bureau virtuel (VDS) actuel, mais uniquement lorsque le serveur est hors connexion.

</dd> <dt>

[**Méthode DumpVmInfo**](dumpvminfo-win32-tsvm.md)
</dt> <dd>

Ce membre est destiné à des fins de test interne et ne doit pas être utilisé dans votre code.

</dd> <dt>

[**Méthode d’exportation**](export-win32-tsvm.md)
</dt> <dd>

Récupère les informations de surveillance de session de l’ordinateur virtuel.

</dd> </dl>
