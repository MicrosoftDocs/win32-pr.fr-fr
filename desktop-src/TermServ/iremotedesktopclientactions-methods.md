# <a name="iremotedesktopclientactions-methods"></a>Méthodes IRemoteDesktopClientActions

L’interface [**IRemoteDesktopClientActions**](/windows/win32/api/rdpappcontainerclient/nn-rdpappcontainerclient-iremotedesktopclientactions) expose les méthodes suivantes.

## <a name="in-this-section"></a>Dans cette section

<dl> <dt>

[**Méthode ExecuteRemoteAction**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclientactions-executeremoteaction)
</dt> <dd>

Entraîne l’exécution d’une action dans la session à distance.

</dd> <dt>

[**Méthode GetSnapshot**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclientactions-getsnapshot)
</dt> <dd>

Provoque l’arrêt d’un instantané du Bureau du client du conteneur d’applications protocole RDP (Remote Desktop Protocol) (RDP).

</dd> <dt>

[**Méthode ResumeScreenUpdates**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclientactions-resumescreenupdates)
</dt> <dd>

Reprend l’envoi des mises à jour à l’écran au client.

</dd> <dt>

[**Méthode SuspendScreenUpdates**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclientactions-suspendscreenupdates)
</dt> <dd>

Interrompt l’envoi des mises à jour de l’écran au client.

</dd> </dl>
