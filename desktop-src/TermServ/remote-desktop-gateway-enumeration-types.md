# <a name="remote-desktop-gateway-enumeration-types"></a>Types d’énumération de passerelle Bureau à distance

L’API Bureau à distance Gateway (passerelle des services Bureau à distance) prend en charge les types d’énumération suivants.

## <a name="in-this-section"></a>Dans cette section

<dl> <dt>

[**AAAccountingDataType**](/windows/win32/api/tsgpolicyengine/ne-tsgpolicyengine-aaaccountingdatatype)
</dt> <dd>

Spécifie le type d’événement dont la méthode [**ITSGAccountingEngine ::D oaccounting**](/windows/desktop/api/TSGPolicyEngine/nf-tsgpolicyengine-itsgaccountingengine-doaccounting) est notifiée.

</dd> <dt>

[**AAAuthSchemes**](/windows/win32/api/tsgpolicyengine/ne-tsgpolicyengine-aaauthschemes)
</dt> <dd>

Spécifie le type d’authentification utilisé pour se connecter à la passerelle des services Bureau à distance.

</dd> <dt>

[**PolicyAttributeType**](/windows/win32/api/tsgpolicyengine/ne-tsgpolicyengine-policyattributetype)
</dt> <dd>

Spécifie les paramètres de redirection associés à une connexion.

</dd> </dl>
