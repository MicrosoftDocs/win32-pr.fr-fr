# <a name="remote-desktop-connection-broker-client-interfaces"></a>Interfaces du client Connexion Bureau à distance Broker

L’API du client Connexion Bureau à distance Broker définit les interfaces suivantes.

## <a name="in-this-section"></a>Dans cette section

<dl> <dt>

[**IConnectionBrokerClient**](iconnectionbrokerclient.md)
</dt> <dd>

Fournit les méthodes nécessaires pour utiliser le client Connexion Bureau à distance Broker.

</dd> <dt>

[**IConnectionBrokerRequest**](iconnectionbrokerrequest.md)
</dt> <dd>

Fournit les méthodes nécessaires pour obtenir les résultats de la méthode [**IConnectionBrokerClient :: GetTargetInfo**](iconnectionbrokerclient-gettargetinfo.md) asynchrone.

</dd> </dl>
