# <a name="reference-for-the-adsi-extension-for-remote-desktop-services-user-configuration"></a>Référence pour l’extension ADSI pour Services Bureau à distance Configuration utilisateur

L’extension ADSI pour Services Bureau à distance Configuration utilisateur permet aux administrateurs de domaine d’écrire des scripts pour automatiser la maintenance des propriétés utilisateur spécifiques à Services Bureau à distance. Il comprend l’interface double suivante.

## <a name="in-this-section"></a>Dans cette section

<dl> <dt>

[**IADsTSUserEx**](/windows/desktop/api/Tsuserex/nn-tsuserex-iadstsuserex)
</dt> <dd>

Utilisé pour examiner et configurer Services Bureau à distance propriétés de l’utilisateur.

</dd> </dl>
