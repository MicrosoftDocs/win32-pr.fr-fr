# <a name="imsrdpdevicecollection2-interface"></a>Interface IMsRdpDeviceCollection2

Représente une collection d’objets périphériques. Il s’agit d’une amélioration de l’interface [**IMsRdpDeviceCollection**](imsrdpdevicecollection.md) .

## <a name="members"></a>Membres

L’interface **IMsRdpDeviceCollection2** hérite de [**IMsRdpDeviceCollection**](imsrdpdevicecollection.md). **IMsRdpDeviceCollection2** a également les types de membres suivants :

-   [Méthodes](#methods)

### <a name="methods"></a>Méthodes

L’interface **IMsRdpDeviceCollection2** possède ces méthodes.



| Méthode                                                                         | Description                                                                                                 |
|:-------------------------------------------------------------------------------|:------------------------------------------------------------------------------------------------------------|
| [**AddDeviceByInstanceId**](imsrdpdevicecollection2-adddevicebyinstanceid.md) | Ajoute un appareil non répertorié au regroupement de périphériques.<br/>                                                |
| [**RedirectNow**](imsrdpdevicecollection2-redirectnow.md)                     | Force la redirection ou la suppression des appareils qui ont été sélectionnés ou désélectionnés dans la collection.<br/> |
