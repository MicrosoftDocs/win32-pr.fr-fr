# <a name="remote-desktop-services-audioendpoint-structures"></a>Structures AudioEndpoint Services Bureau à distance

Les structures suivantes sont utilisées avec l’API AudioEndpoint.

## <a name="in-this-section"></a>Dans cette section

<dl> <dt>

[**\_position actuelle \_ AE**](/windows/desktop/api/Audioengineendpoint/ns-audioengineendpoint-ae_current_position)
</dt> <dd>

Signale la position actuelle du frame de l’appareil aux clients.

</dd> <dt>

[**\_propriété de connexion APO \_**](/windows/desktop/api/Audioapotypes/ns-audioapotypes-apo_connection_property)
</dt> <dd>

Contient les propriétés de connexion de modification dynamique.

</dd> </dl>

## <a name="remarks"></a>Notes

L’API Services Bureau à distance AudioEndpoint est destinée à être utilisée dans Bureau à distance scénarios ; ce n’est pas le cas pour les applications clientes.
