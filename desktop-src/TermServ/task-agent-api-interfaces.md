# <a name="personal-desktop-task-agent-api-interfaces"></a>Interfaces d’API de l’agent des tâches du bureau personnel

L’API agent de tâche de bureau personnel prend en charge les interfaces suivantes.

## <a name="in-this-section"></a>Dans cette section

<dl> <dt>

[**IRDVTaskPluginNotifySink**](irdvtaskpluginnotifysink.md)
</dt> <dd>

L’interface [**IRDVTaskPluginNotifySink**](irdvtaskpluginnotifysink.md) est utilisée par l’agent de tâche pour communiquer avec l’agent déclencheur.

</dd> <dt>

[**IRDVTaskPlugin**](irdvtaskplugin.md)
</dt> <dd>

L’interface [**IRDVTaskPlugin**](irdvtaskplugin.md) est implémentée par un agent de *tâche* de mise à jour de machine virtuelle pour permettre à l’agent de tâche de gérer les mises à jour système pour un ordinateur virtuel.

</dd> </dl>
