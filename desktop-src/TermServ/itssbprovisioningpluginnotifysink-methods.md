# <a name="itssbprovisioningpluginnotifysink-methods"></a>Méthodes ITsSbProvisioningPluginNotifySink

Les interfaces [**ITsSbProvisioningPluginNotifySink**](/windows/desktop/api/sbtsv/nn-sbtsv-itssbprovisioningpluginnotifysink) exposent les méthodes suivantes.

## <a name="in-this-section"></a>Dans cette section

<dl> <dt>

[**Méthode LockVirtualMachine**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbprovisioningpluginnotifysink-lockvirtualmachine)
</dt> <dd>

Avertit Connexion Bureau à distance Broker (Service Broker pour les connexions Bureau à distance) que l’ordinateur virtuel est verrouillé.

</dd> <dt>

[**Méthode OnJobCancelled**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbprovisioningpluginnotifysink-onjobcancelled)
</dt> <dd>

Indique au service Broker pour les connexions Bureau à distance que le travail est annulé.

</dd> <dt>

[**Méthode OnJobCompleted**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbprovisioningpluginnotifysink-onjobcompleted)
</dt> <dd>

Indique au service Broker pour les connexions Bureau à distance que le travail est terminé.

</dd> <dt>

[**Méthode OnJobCreated**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbprovisioningpluginnotifysink-onjobcreated)
</dt> <dd>

Avertit le Service Broker pour les connexions Bureau à la création d’un travail d’approvisionnement.

</dd> <dt>

[**Méthode OnVirtualMachineHostStatusChanged**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbprovisioningpluginnotifysink-onvirtualmachinehoststatuschanged)
</dt> <dd>

Notifie le Service Broker pour les connexions Bureau à distance que l’état de l’hôte d’un ordinateur virtuel est modifié.

</dd> <dt>

[**Méthode OnVirtualMachineStatusChanged**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbprovisioningpluginnotifysink-onvirtualmachinestatuschanged)
</dt> <dd>

Notifie le Service Broker pour les connexions Bureau à distance que l’état d’un ordinateur virtuel a été modifié.

</dd> </dl>
