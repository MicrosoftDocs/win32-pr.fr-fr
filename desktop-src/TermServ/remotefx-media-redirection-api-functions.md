# <a name="remotefx-media-redirection-api-functions"></a>Fonctions de l’API de redirection multimédia RemoteFX

L’API de redirection multimédia RemoteFX prend en charge les fonctions suivantes.

## <a name="in-this-section"></a>Dans cette section

<dl> <dt>

[**WTSSetRenderHint**](/windows/desktop/api/WtsHintApi/nf-wtshintapi-wtssetrenderhint)
</dt> <dd>

Utilisé par une application affichant du contenu qui peut être optimisé pour l’affichage dans une session à distance pour identifier la région d’une fenêtre qui est le contenu réel.

</dd> </dl>
