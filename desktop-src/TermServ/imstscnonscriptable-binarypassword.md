# <a name="imstscnonscriptablebinarypassword-property"></a>IMsTscNonScriptable :: BinaryPassword, propriété

Cette propriété ne peut plus être utilisée.

Cette propriété est en lecture/écriture.

## <a name="syntax"></a>Syntaxe


```C++
HRESULT put_BinaryPassword(
  [in]  BSTR newPassword
);

HRESULT get_BinaryPassword(
  [out] BSTR *pBinaryPassword
);
```



## <a name="property-value"></a>Valeur de la propriété

Nouvelle partie du mot de passe, au format binaire encodé.

## <a name="error-codes"></a>Codes d’erreur

Retourne **E \_ NOTIMPL**.
