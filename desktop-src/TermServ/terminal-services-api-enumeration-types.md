# <a name="remote-desktop-services-api-enumeration-types"></a>Types d’énumération d’API Services Bureau à distance

Les types d’énumération suivants sont utilisés avec l’API Services Bureau à distance.

## <a name="in-this-section"></a>Dans cette section

<dl> <dt>

[**\_classe de configuration WTS \_**](/windows/desktop/api/Wtsapi32/ne-wtsapi32-wts_config_class)
</dt> <dd>

Contient des valeurs qui indiquent le type d’informations de configuration utilisateur à définir ou à récupérer dans un appel aux fonctions [**WTSQueryUserConfig**](/windows/desktop/api/Wtsapi32/nf-wtsapi32-wtsqueryuserconfiga) et [**WTSSetUserConfig**](/windows/desktop/api/Wtsapi32/nf-wtsapi32-wtssetuserconfiga) .

</dd> <dt>

[**\_source de configuration WTS \_**](/windows/desktop/api/Wtsapi32/ne-wtsapi32-wts_config_source)
</dt> <dd>

Spécifie la source des informations de configuration retournées par la fonction [**WTSQueryUserConfig**](/windows/desktop/api/Wtsapi32/nf-wtsapi32-wtsqueryuserconfiga) .

</dd> <dt>

[**WTS \_ CONNECTSTATE, \_ classe**](/windows/desktop/api/Wtsapi32/ne-wtsapi32-wts_connectstate_class)
</dt> <dd>

Spécifie l’état de connexion d’une session de Services Bureau à distance.

</dd> <dt>

[**\_classe WTS info \_**](/windows/desktop/api/Wtsapi32/ne-wtsapi32-wts_info_class)
</dt> <dd>

Contient des valeurs qui indiquent le type d’informations de session à récupérer dans un appel à la fonction [**WTSQuerySessionInformation**](/windows/desktop/api/Wtsapi32/nf-wtsapi32-wtsquerysessioninformationa) .

</dd> <dt>

[**\_classe de type WTS \_**](/windows/desktop/api/Wtsapi32/ne-wtsapi32-wts_type_class)
</dt> <dd>

Spécifie le type de structure qu’une fonction de Services Bureau à distance a retourné dans une mémoire tampon.

</dd> <dt>

[**\_classe virtuelle \_ WTS**](/windows/desktop/api/Wtsapi32/ne-wtsapi32-wts_virtual_class)
</dt> <dd>

Contient des valeurs qui indiquent le type d’informations de canal virtuel à récupérer.

</dd> <dt>

[**\_famille d’adresses WTSSBX \_**](/windows/win32/api/tssbx/ne-tssbx-wtssbx_address_family)
</dt> <dd>

Contient des valeurs qui indiquent la famille d’adresses d’une adresse réseau utilisée pour la redirection.

</dd> <dt>

[**\_drain machine \_ WTSSBX**](/windows/win32/api/tssbx/ne-tssbx-wtssbx_machine_drain)
</dt> <dd>

Contient des valeurs qui indiquent l’état de drainage d’un serveur hôte de session Bureau à distance (hôte de session Bureau à distance).

</dd> <dt>

[**MODE de session de l' \_ ordinateur WTSSBX \_ \_**](/windows/win32/api/tssbx/ne-tssbx-wtssbx_machine_session_mode)
</dt> <dd>

Contient des valeurs qui indiquent le mode de session d’un serveur hôte de session Bureau à distance.

</dd> <dt>

[**État de l' \_ ordinateur WTSSBX \_**](/windows/win32/api/tssbx/ne-tssbx-wtssbx_machine_state)
</dt> <dd>

Contient des valeurs qui indiquent l’état actuel d’un serveur.

</dd> <dt>

[**\_type de notification WTSSBX \_**](/windows/win32/api/tssbx/ne-tssbx-wtssbx_notification_type)
</dt> <dd>

Contient des valeurs qui indiquent le type de modification d’État qui s’est produit sur un serveur hôte de session Bureau à distance ou une session utilisateur.

</dd> <dt>

[**\_État de session WTSSBX \_**](/windows/win32/api/tssbx/ne-tssbx-wtssbx_session_state)
</dt> <dd>

Contient des valeurs qui indiquent l’état de connexion d’une session utilisateur.

</dd> </dl>
