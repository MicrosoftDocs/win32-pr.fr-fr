# <a name="iremotedesktopclientevents-methods"></a>Méthodes IRemoteDesktopClientEvents

L’interface [**IRemoteDesktopClientEvents**](iremotedesktopclientevents.md) expose les méthodes suivantes.

## <a name="in-this-section"></a>Dans cette section

<dl> <dt>

[**Méthode OnConnecting**](iremotedesktopclientevents-onconnecting.md)
</dt> <dd>

Appelé lorsque le contrôle client tente d’établir une connexion à une session à distance.

</dd> <dt>

[**Méthode OnConnected**](iremotedesktopclientevents-onconnected.md)
</dt> <dd>

Appelé lorsque le contrôle client s’est connecté à une session à distance.

</dd> <dt>

[**Méthode OnLoginCompleted**](iremotedesktopclientevents-onlogincompleted.md)
</dt> <dd>

Appelé lorsque le contrôle client a réussi à se connecter à une session à distance.

</dd> <dt>

[**Méthode OnDisconnected**](iremotedesktopclientevents-ondisconnected.md)
</dt> <dd>

Appelé lorsque le contrôle client a été déconnecté d’une session à distance.

</dd> <dt>

[**Méthode OnStatusChanged**](iremotedesktopclientevents-onstatuschanged.md)
</dt> <dd>

Appelé lorsque le contrôle client a mis à jour son état.

</dd> <dt>

[**Méthode OnAutoReconnecting**](iremotedesktopclientevents-onautoreconnecting.md)
</dt> <dd>

Appelé lorsque le contrôle client tente de rétablir automatiquement une connexion à une session à distance.

</dd> <dt>

[**Méthode OnAutoReconnected**](iremotedesktopclientevents-onautoreconnected.md)
</dt> <dd>

Appelé lorsque le contrôle client se reconnecte automatiquement à une session à distance.

</dd> <dt>

[**Méthode OnDialogDisplaying**](iremotedesktopclientevents-ondialogdisplaying.md)
</dt> <dd>

Appelé avant que le contrôle affiche une boîte de dialogue.

</dd> <dt>

[**Méthode OnDialogDismissed**](iremotedesktopclientevents-ondialogdismissed.md)
</dt> <dd>

Appelé après la fermeture d’une boîte de dialogue affichée par le contrôle client.

</dd> <dt>

[**Méthode OnNetworkStatusChanged**](iremotedesktopclientevents-onnetworkstatuschanged.md)
</dt> <dd>

Appelé lorsque l’état du réseau a changé.

</dd> <dt>

[**Méthode OnAdminMessageReceived**](iremotedesktopclientevents-onadminmessagereceived.md)
</dt> <dd>

Appelé lorsque le contrôle client reçoit un message d’administration.

</dd> <dt>

[**Méthode OnKeyCombinationPressed**](iremotedesktopclientevents-onkeycombinationpressed.md)
</dt> <dd>

Appelé lorsque des combinaisons de touches spéciales sont activées dans la session à distance.

</dd> <dt>

[**Méthode OnRemoteDesktopSizeChanged**](iremotedesktopclientevents-onremotedesktopsizechanged.md)
</dt> <dd>

Appelé lorsque la taille du Bureau à distance a changé.

</dd> <dt>

[**Méthode OnTouchPointerCursorMoved**](iremotedesktopclientevents-ontouchpointercursormoved.md)
</dt> <dd>

Appelé lorsque le curseur tactile a été déplacé et que la propriété [**EventsEnabled**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclienttouchpointer-get_eventsenabled) a la valeur true.

</dd> </dl>
