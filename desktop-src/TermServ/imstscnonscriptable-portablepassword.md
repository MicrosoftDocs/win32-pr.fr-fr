# <a name="imstscnonscriptableportablepassword-property"></a>IMsTscNonScriptable ::P propriété ortablePassword

Cette propriété ne peut plus être utilisée.

Cette propriété est en lecture/écriture.

## <a name="syntax"></a>Syntaxe


```C++
HRESULT put_PortablePassword(
  [in]  BSTR newPortablePassVal
);

HRESULT get_PortablePassword(
  [out] BSTR *pPortablePass
);
```



## <a name="property-value"></a>Valeur de la propriété

Nouvelle partie du mot de passe, dans un format encodé portable.

## <a name="error-codes"></a>Codes d’erreur

Retourne **E \_ NOTIMPL**.
