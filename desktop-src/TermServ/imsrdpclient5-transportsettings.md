# <a name="imsrdpclient5transportsettings-property"></a>IMsRdpClient5 :: TransportSettings, propriété

Récupère ce qui a été passé via un script à l’interface [**IMsRdpClientTransportSettings**](imsrdpclienttransportsettings.md) .

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```C++
HRESULT get_TransportSettings(
  [out] IMsRdpClientTransportSettings **ppXportSet
);
```



## <a name="property-value"></a>Valeur de la propriété

Pointeur d’interface [**IMsRdpClientTransportSettings**](imsrdpclienttransportsettings.md) .
