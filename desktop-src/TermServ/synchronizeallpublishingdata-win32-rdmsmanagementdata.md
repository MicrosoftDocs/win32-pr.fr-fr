# <a name="synchronizeallpublishingdata-method-of-the-win32_rdmsmanagementdata-class"></a>Méthode SynchronizeAllPublishingData de la \_ classe Win32 RDMSManagementData

Synchronise toutes les données de publication pour les services de gestion de Bureau à distance (RDMS).

## <a name="syntax"></a>Syntaxe


```mof
uint32 SynchronizeAllPublishingData();
```



## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.

## <a name="return-value"></a>Valeur retournée

Retourne 0 en cas de réussite, sinon retourne un code d’erreur WMI.
