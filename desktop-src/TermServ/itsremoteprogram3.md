# <a name="itsremoteprogram3-interface"></a>Interface ITSRemoteProgram3

Définit des propriétés et des méthodes à utiliser avec un RemoteApp.

## <a name="members"></a>Membres

L’interface **ITSRemoteProgram3** hérite de **ITSRemoteProgram2**. **ITSRemoteProgram3** a également les types de membres suivants :

-   [Méthodes](#methods)

### <a name="methods"></a>Méthodes

L’interface **ITSRemoteProgram3** possède ces méthodes.



| Méthode                                                     | Description                                                  |
|:-----------------------------------------------------------|:-------------------------------------------------------------|
| [**ServerStartApp**](itsremoteprogram3-serverstartapp.md) | Spécifie une application à démarrer dans la session à distance. <br/> |
