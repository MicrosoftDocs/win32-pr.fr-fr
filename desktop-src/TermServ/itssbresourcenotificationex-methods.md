# <a name="itssbresourcenotificationex-methods"></a>Méthodes ITsSbResourceNotificationEx

Les interfaces [**ITsSbResourceNotificationEx**](/windows/desktop/api/sbtsv/nn-sbtsv-itssbresourcenotificationex) exposent les méthodes suivantes.

## <a name="in-this-section"></a>Dans cette section

<dl> <dt>

[**Méthode NotifyClientConnectionStateChangeEx**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbresourcenotificationex-notifyclientconnectionstatechangeex)
</dt> <dd>

Notifie les plug-ins inscrits sur les changements d’État dans une connexion cliente.

</dd> <dt>

[**Méthode NotifySessionChangeEx**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbresourcenotificationex-notifysessionchangeex)
</dt> <dd>

Avertit les plug-ins enregistrés des modifications d’État dans un objet de session.

</dd> <dt>

[**Méthode NotifyTargetChangeEx**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbresourcenotificationex-notifytargetchangeex)
</dt> <dd>

Avertit les plug-ins enregistrés des modifications d’État dans un objet cible.

</dd> </dl>
