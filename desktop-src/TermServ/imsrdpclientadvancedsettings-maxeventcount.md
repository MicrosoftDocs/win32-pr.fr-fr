# <a name="imsrdpclientadvancedsettingsmaxeventcount-property"></a>IMsRdpClientAdvancedSettings :: maxEventCount, propriété

Cette propriété n'est pas prise en charge.

Cette propriété est en lecture/écriture.

## <a name="syntax"></a>Syntaxe


```C++
HRESULT put_maxEventCount(
  [in]  LONG maxEventCount
);

HRESULT get_maxEventCount(
  [out] LONG pmaxEventCount
);
```



## <a name="property-value"></a>Valeur de la propriété

Nouveau nombre d’événements. La valeur par défaut est 100.

## <a name="error-codes"></a>Codes d’erreur

Retourne **S \_ false**.
