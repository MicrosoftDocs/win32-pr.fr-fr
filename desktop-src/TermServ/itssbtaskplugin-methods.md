# <a name="itssbtaskplugin-methods"></a>Méthodes ITsSbTaskPlugin

Les interfaces [**ITsSbTaskPlugin**](/windows/desktop/api/sbtsv/nn-sbtsv-itssbtaskplugin) exposent les méthodes suivantes.

## <a name="in-this-section"></a>Dans cette section

<dl> <dt>

[**Méthode InitializeTaskPlugin**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbtaskplugin-initializetaskplugin)
</dt> <dd>

Initialise une tâche qui se trouve dans la file d’attente d’un plug-in Broker Connexion Bureau à distance.

</dd> <dt>

[**Méthode SetTaskQueue**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbtaskplugin-settaskqueue)
</dt> <dd>

Met à jour une tâche dans la file d’attente d’un plug-in Broker Connexion Bureau à distance.

</dd> </dl>
