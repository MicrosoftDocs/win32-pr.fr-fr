# <a name="imsrdpclient5msrdpclientshell-property"></a>IMsRdpClient5 :: MsRdpClientShell, propriété

Récupère l’interface de paramètre client scriptable [**IMsRdpClientShell**](imsrdpclientshell.md).

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```C++
HRESULT get_MsRdpClientShell(
  [out] IMsRdpClientShell **ppLauncher
);
```



## <a name="property-value"></a>Valeur de la propriété

Pointeur d’interface [**IMsRdpClientShell**](imsrdpclientshell.md) .
