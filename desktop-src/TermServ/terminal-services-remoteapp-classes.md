# <a name="remoteapp-classes"></a>Classes RemoteApp

Le fournisseur WMI RemoteApp fournit les classes suivantes.

## <a name="in-this-section"></a>Dans cette section

<dl> <dt>

[**\_RDAllowListFileAssociation Win32**](win32-rdallowlistfileassociation.md)
</dt> <dd>

Décrit une association de type de fichier publiée avec un RemoteApp.

</dd> <dt>

[**\_RDFileAssociation Win32**](win32-rdfileassociation.md)
</dt> <dd>

Représente une association de types de fichiers pour un RemoteApp.

</dd> <dt>

[**\_RemoteAppChangeEvent Win32**](win32-remoteappchangeevent.md)
</dt> <dd>

représente une modification apportée aux paramètres RemoteApp de Windows server 2008 R2 sur le serveur hôte de session Bureau à distance (hôte de session bureau à distance).

</dd> <dt>

[**\_TSApplicationFileExtensions Win32**](win32-tsapplicationfileextensions.md)
</dt> <dd>

les extensions de nom de fichier qui sont gérées par une application sur un serveur hôte de session Bureau à distance.

</dd> <dt>

[**\_TSDeploymentLicensing Win32**](win32-tsdeploymentlicensing.md)
</dt> <dd>

Paramètres de licence pour le déploiement de la virtualisation des Bureau à distance (RDV).

</dd> <dt>

[**\_TSDeploymentSettings Win32**](win32-tsdeploymentsettings.md)
</dt> <dd>

Définit les paramètres par défaut utilisés par la Gestionnaire RemoteApp lors de la création de fichiers protocole RDP (Remote Desktop Protocol) (RDP).

</dd> <dt>

[**\_TSExpandEnvironmentVariables Win32**](win32-tsexpandenvironmentvariables.md)
</dt> <dd>

Développe les variables d’environnement définies par le système.

</dd> <dt>

[**\_TSGetIcon Win32**](win32-tsgeticon.md)
</dt> <dd>

Retourne le contenu de l’icône spécifiée par le chemin d’accès et l’index de fichier

</dd> <dt>

[**\_TSPublishedApplication Win32**](win32-tspublishedapplication.md)
</dt> <dd>

définit les applications qui sont disponibles pour une utilisation à distance par le biais de Windows Server 2008 R2 RemoteApp.

</dd> <dt>

[**\_TSPublishedApplicationList Win32**](win32-tspublishedapplicationlist.md)
</dt> <dd>

Représente la liste des programmes qui se trouvent dans la liste programmes RemoteApp sur un serveur hôte de session Bureau à distance.

</dd> <dt>

[**\_TSRemoteDesktop Win32**](win32-tsremotedesktop.md)
</dt> <dd>

Décrit une connexion Bureau à distance qui est disponible via Bureau à distance Accès web (RD Accès web).

</dd> <dt>

[**\_TSStartMenuApplication Win32**](win32-tsstartmenuapplication.md)
</dt> <dd>

décrit les applications qui se trouvent sur le menu Démarrer d’un serveur hôte de Session bureau à distance.

</dd> <dt>

[**\_TSSystemInfo Win32**](win32-tssysteminfo.md)
</dt> <dd>

Représente le service d’informations du serveur de publication centralisé

</dd> </dl>
