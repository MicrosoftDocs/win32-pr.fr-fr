# <a name="imsrdpclientshellshowtrustedsitesmanagementdialog-method"></a>IMsRdpClientShell :: ShowTrustedSitesManagementDialog, méthode

Affiche la liste de la boîte de dialogue sites de confiance.

## <a name="syntax"></a>Syntaxe


```C++
HRESULT ShowTrustedSitesManagementDialog();
```



## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.

## <a name="return-value"></a>Valeur retournée

Si cette méthode est réussie, elle retourne la valeur **\_ OK**. Sinon, elle retourne un code d’erreur **HRESULT** .
