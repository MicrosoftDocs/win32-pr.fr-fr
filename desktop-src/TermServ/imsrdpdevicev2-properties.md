# <a name="imsrdpdevicev2-properties"></a>Propriétés IMsRdpDeviceV2

L’interface [**IMsRdpDeviceV2**](imsrdpdevicev2.md) expose les propriétés suivantes.

## <a name="in-this-section"></a>Dans cette section

<dl> <dt>

[**Propriété CmClassGuid**](imsrdpdevicev2-cmclassguid.md)
</dt> <dd>

Contient le GUID de la classe d’installation de Configuration Manager pour l’appareil.

</dd> <dt>

[**Propriété CmDeviceInstance**](imsrdpdevicev2-cmdeviceinstance.md)
</dt> <dd>

Contient l’instance d’appareil Configuration Manager de l’appareil.

</dd> <dt>

[**Propriété DeviceText**](imsrdpdevicev2-devicetext.md)
</dt> <dd>

Contient le texte de l’appareil.

</dd> <dt>

[**Propriété DriveLetterBitmap**](imsrdpdevicev2-driveletterbitmap.md)
</dt> <dd>

Contient un champ de binaire qui représente une carte des lettres de lecteur contenues dans l’appareil.

</dd> <dt>

[**Propriété IsCompositeDevice**](imsrdpdevicev2-iscompositedevice.md)
</dt> <dd>

Spécifie si l’appareil est un appareil composite.

</dd> <dt>

[**Propriété IsOptionalDevice**](imsrdpdevicev2-isoptionaldevice.md)
</dt> <dd>

Spécifie si l’appareil est facultatif pour la redirection USB.

</dd> <dt>

[**Propriété IsUSBDevice**](imsrdpdevicev2-isusbdevice.md)
</dt> <dd>

Spécifie si l’appareil est destiné à la redirection USB.

</dd> </dl>
