# <a name="iworkspacescriptable2-methods"></a>Méthodes IWorkspaceScriptable2

L’interface [**IWorkspaceScriptable2**](/windows/desktop/api/workspaceruntime/nn-workspaceruntime-iworkspacescriptable2) prend en charge les méthodes suivantes.

## <a name="in-this-section"></a>Dans cette section

<dl> <dt>

[**Méthode StartWorkspaceEx**](/windows/desktop/api/workspaceruntime/nf-workspaceruntime-iworkspacescriptable2-startworkspaceex)
</dt> <dd>

Associe les informations d’identification de l’utilisateur et les certificats à un ID de connexion ; contient également des éléments de sécurité et d’interface utilisateur supplémentaires.

</dd> <dt>

[**Méthode ResourceDismissed**](/windows/desktop/api/workspaceruntime/nf-workspaceruntime-iworkspacescriptable2-resourcedismissed)
</dt> <dd>

Avertit l’utilisateur qu’une ressource a été désactivée ou ignorée.

</dd> </dl>
