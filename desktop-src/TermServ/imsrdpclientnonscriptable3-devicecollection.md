# <a name="imsrdpclientnonscriptable3devicecollection-property"></a>IMsRdpClientNonScriptable3 ::D propriété eviceCollection

Récupère la collection d’objets d’appareil Plug-and-Play (PnP) à rediriger.

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```C++
HRESULT get_DeviceCollection(
  [out] IMSRdpDeviceCollection **ppDeviceCollection
);
```



## <a name="property-value"></a>Valeur de la propriété

Récupère la collection d’objets périphériques PnP de type [**IMSRdpDevice**](imsrdpdevice.md).
