# <a name="whats-new-in-windows-7"></a>nouveautés de Windows 7

Windows 7 introduit les nouveaux éléments de programmation suivants pour Services Bureau à distance.




| Condition requise | Valeur |
|--------|-------|
| Interfaces de contrôle de ActiveX Bureau à distance<br /> | les interfaces suivantes (et leurs propriétés associées) ont été ajoutées au <a href="remote-desktop-web-connection-reference.md">Bureau à distance ActiveX Interface de contrôle</a>.<br /><ul><li><a href="imsrdpclientadvancedsettings7.md"><strong>Interface IMsRdpClientAdvancedSettings7</strong></a></li></ul> | 
| API de contrôle de Accès web Services Bureau à distance<br /> | Services Bureau à distance contrôle de Accès web expose des interfaces qui prennent en charge la publication de ressources spécifiques à l’utilisateur dans Connexions aux programmes RemoteApp et aux services Bureau à distance.<br /><ul><li><a href="/windows/desktop/TermServ/remote-desktop-services-web-access-control-interfaces">API de contrôle de Accès web Services Bureau à distance</a></li></ul> | 
| API Connexions aux programmes RemoteApp et aux services Bureau à distance Runtime<br /> | Connexions aux programmes RemoteApp et aux services Bureau à distance API Runtime prend en charge le développement de clients personnalisés dans Connexions aux programmes RemoteApp et aux services Bureau à distance.<br /><ul><li><a href="/windows/desktop/TermServ/remoteapp-and-desktop-connection-runtime-interfaces">API Connexions aux programmes RemoteApp et aux services Bureau à distance Runtime</a></li></ul> |
