# <a name="win32_rdvhmanagement-methods"></a>\_Méthodes RdvhManagement Win32

La classe [**Win32 \_ RdvhManagement Methods**](win32-rdvhmanagement.md) expose les méthodes suivantes.

## <a name="in-this-section"></a>Dans cette section

<dl> <dt>

[**Méthode RdvMigrateVmToDifferentHost**](rdvmigratevmtodifferenthost-win32-rdvhmanagement.md)
</dt> <dd>

Lance une migration dynamique d’un ordinateur virtuel vers un ordinateur hôte spécifié.

</dd> <dt>

[**Méthode RdvCopyBaseVmLocally**](rdvcopybasevmlocally-win32-rdvhmanagement.md)
</dt> <dd>

Copie un ordinateur virtuel de base local sur un serveur hôte de virtualisation des services Bureau à distance (RDV) spécifié.

</dd> <dt>

[**Méthode RdvSetupVMPermissions**](rdvsetupvmpermissions-win32-rdvhmanagement.md)
</dt> <dd>

Définit les autorisations sur un ordinateur virtuel pour l’utilisateur actuel.

</dd> <dt>

[**Méthode RdvUndoVMPermissions**](rdvundovmpermissions-win32-rdvhmanagement.md)
</dt> <dd>

Restaure les autorisations définies par [**RdvSetupVMPermissions**](rdvsetupvmpermissions-win32-rdvhmanagement.md) sur la machine virtuelle spécifiée.

</dd> <dt>

[**Méthode RdvCreateUserDiskTemplate**](rdvcreateuserdisktemplate-win32-rdvhmanagement.md)
</dt> <dd>

Crée un modèle de disque utilisateur, avec la taille maximale spécifiée, à l’emplacement spécifié.

</dd> </dl>
