# <a name="imsrdpclientadvancedsettings7redirectdirectx-property"></a>IMsRdpClientAdvancedSettings7 :: RedirectDirectX, propriété

Cette propriété n'est pas utilisée.

Cette propriété est en lecture/écriture.

## <a name="syntax"></a>Syntaxe


```C++
HRESULT put_RedirectDirectX(
  [in]          VARIANT_BOOL fRedirectDirectX
);

HRESULT get_RedirectDirectX(
  [out, retval] VARIANT_BOOL *pfRedirectDirectX
);
```



## <a name="property-value"></a>Valeur de la propriété

Cette propriété n'est pas utilisée.

## <a name="error-codes"></a>Codes d’erreur

Cette propriété n'est pas utilisée. La méthode setter retourne toujours **S \_ false** et la méthode Getter retourne toujours **E \_ NOTIMPL**.
