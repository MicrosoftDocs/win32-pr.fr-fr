L’API du client Connexion Bureau à distance Broker définit les structures suivantes.

## <a name="in-this-section"></a>Dans cette section

<dl> <dt>

[**\_informations de connexion CB \_**](cb-connection-info.md)
</dt> <dd>

Contient des informations sur une demande de connexion entrante.

</dd> <dt>

[**\_infos sur la cible CB \_**](cb-target-info.md)
</dt> <dd>

</dd> </dl>
