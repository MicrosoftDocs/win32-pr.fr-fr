# <a name="imstscaxeventsondevicesbuttonpressed-method"></a>IMsTscAxEvents :: OnDevicesButtonPressed, méthode

Appelé lorsque le bouton appareils de la barre de connexion a été enfoncé.

## <a name="syntax"></a>Syntaxe


```C++
void OnDevicesButtonPressed();
```



## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.

## <a name="return-value"></a>Valeur retournée

Cette méthode ne retourne pas de valeur.

## <a name="remarks"></a>Notes

Le bouton appareils est affiché dans la barre de connexion lorsque la redirection USB est activée à la fois sur le client et sur le serveur.
