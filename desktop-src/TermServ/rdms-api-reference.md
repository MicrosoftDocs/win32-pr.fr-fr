# <a name="remote-desktop-management-services-provider"></a>Fournisseur de services de gestion Bureau à distance

Le fournisseur Bureau à distance Management Services (RDMS) gère les environnements VDI (Virtual Desktop Infrastructure).

## <a name="in-this-section"></a>Dans cette section

<dl> <dt>

[**\_RDMSCollectionProperties Win32**](win32-rdmscollectionproperties.md)
</dt> <dd>

Gère les propriétés d’une collection de bureaux virtuels.

</dd> <dt>

[**\_RDMSDeploymentSettings Win32**](win32-rdmsdeploymentsettings.md)
</dt> <dd>

Gère les paramètres de déploiement pour une collection de bureaux virtuels.

</dd> <dt>

[**\_RDMSDesktopAssignment Win32**](win32-rdmsdesktopassignment.md)
</dt> <dd>

Décrit l’attribution de bureau de l’utilisateur de la collection des services Bureau à distance.

</dd> <dt>

[**\_RDMSEnvironment Win32**](win32-rdmsenvironment.md)
</dt> <dd>

Gère un environnement Bureau à distance Management Services (RDMS).

</dd> <dt>

[**\_RDMSFileTypeAssociation Win32**](win32-rdmsfiletypeassociation.md)
</dt> <dd>

Gère une association de types de fichiers pour une application publiée.

</dd> <dt>

[**\_RDMSJoinedNode Win32**](win32-rdmsjoinednode.md)
</dt> <dd>

Représente un nœud de serveur géré par Bureau à distance Management Services (RDMS).

</dd> <dt>

[**\_RDMSManagementData Win32**](win32-rdmsmanagementdata.md)
</dt> <dd>

Synchronise les données de publication pour les services de gestion de Bureau à distance (RDMS).

</dd> <dt>

[**\_RDMSPublishedApplication Win32**](win32-rdmspublishedapplication.md)
</dt> <dd>

Gère une application publiée.

</dd> <dt>

[**\_RDMSVirtualDesktop Win32**](win32-rdmsvirtualdesktop.md)
</dt> <dd>

Représente un bureau virtuel.

</dd> <dt>

[**\_RDMSVirtualDesktopCollection Win32**](win32-rdmsvirtualdesktopcollection.md)
</dt> <dd>

Crée et gère une collection de bureaux virtuels.

</dd> <dt>

[**\_RDSHCollection Win32**](win32-rdshcollection.md)
</dt> <dd>

Gère une collection d’hôtes de session Bureau à distance.

</dd> <dt>

[**\_RDSHServer Win32**](win32-rdshserver.md)
</dt> <dd>

Gère un serveur hôte de session Bureau à distance.

</dd> <dt>

[**\_VirtualDesktopSession Win32**](win32-virtualdesktopsession.md)
</dt> <dd>

Gère une session de bureau virtuel.

</dd> </dl>
