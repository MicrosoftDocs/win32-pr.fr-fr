# <a name="itssbtaskpluginnotifysink-methods"></a>Méthodes ITsSbTaskPluginNotifySink

Les interfaces [**ITsSbTaskPluginNotifySink**](/windows/desktop/api/sbtsv/nn-sbtsv-itssbtaskpluginnotifysink) exposent les méthodes suivantes.

## <a name="in-this-section"></a>Dans cette section

<dl> <dt>

[**Méthode OnDeleteTaskTime**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbtaskpluginnotifysink-ondeletetasktime)
</dt> <dd>

Avertit Connexion Bureau à distance Broker (Service Broker pour les connexions Bureau à distance) qu’une tâche a été supprimée de la file d’attente.

</dd> <dt>

[**Méthode OnReportTasks**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbtaskpluginnotifysink-onreporttasks)
</dt> <dd>

Notifie le Service Broker pour les connexions Bureau à distance d’un nouveau rapport de tâche.

</dd> <dt>

[**Méthode OnSetTaskTime**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbtaskpluginnotifysink-onsettasktime)
</dt> <dd>

Notifie le Service Broker pour les connexions Bureau à distance qu’une tâche a été planifiée.

</dd> <dt>

[**Méthode OnUpdateTaskStatus**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbtaskpluginnotifysink-onupdatetaskstatus)
</dt> <dd>

Notifie le Service Broker pour les connexions Bureau à distance que l’état d’une tâche a changé.

</dd> </dl>
