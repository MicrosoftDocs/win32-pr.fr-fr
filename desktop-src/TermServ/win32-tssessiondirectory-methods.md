# <a name="win32_tssessiondirectory-methods"></a>\_Méthodes TSSessionDirectory Win32

La classe [**Win32 \_ TSSessionDirectory**](win32-tssessiondirectory.md) expose les méthodes suivantes.

## <a name="in-this-section"></a>Dans cette section

<dl> <dt>

[**Méthode CreateUserDiskTemplate**](createuserdisktemplate-win32-tssessiondirectory.md)
</dt> <dd>

Crée un modèle de disque utilisateur.

</dd> <dt>

[**Méthode DisableUserVhd**](disableuservhd-win32-tssessiondirectory.md)
</dt> <dd>

Désactive un disque dur virtuel de profil utilisateur.

</dd> <dt>

[**Méthode EnableUserVhd**](enableuservhd-win32-tssessiondirectory.md)
</dt> <dd>

Active un disque dur virtuel de profil utilisateur sur un serveur hôte hôte.

</dd> <dt>

[**Méthode GetCurrentRedirectableAddresses**](getcurrentredirectableaddresses-win32-tssessiondirectory.md)
</dt> <dd>

Obtient la liste configurée actuellement des adresses DNS éligibles qui peuvent être utilisées pour la redirection.

</dd> <dt>

[**Méthode GetRedirectableAddresses**](getredirectableaddresses-win32-tssessiondirectory.md)
</dt> <dd>

Obtient la liste complète des adresses DNS éligibles.

</dd> <dt>

[**Méthode PingSessionDirectory**](pingsessiondirectory-win32-tssessiondirectory.md)
</dt> <dd>

Vérifie si le serveur Connexion Bureau à distance Broker (Service Broker pour les connexions Bureau à distance) est disponible.

</dd> <dt>

[**Méthode SetCurrentRedirectableAddresses**](setcurrentredirectableaddresses-win32-tssessiondirectory.md)
</dt> <dd>

Définit la liste configurée des adresses DNS éligibles qui peuvent être utilisées pour la redirection.

</dd> <dt>

[**Méthode SetLoadBalancingState**](setloadbalancingstate-win32-tssessiondirectory.md)
</dt> <dd>

Définit la valeur pour indiquer si le serveur doit participer à l’équilibrage de charge du Service Broker pour les connexions Bureau à distance.

</dd> <dt>

[**Méthode SetServerWeight**](setserverweight-win32-tssessiondirectory.md)
</dt> <dd>

Définit la valeur du poids du serveur pour l’équilibrage de charge du Service Broker pour les connexions Bureau à distance.

</dd> <dt>

[**Méthode SetSessionDirectoryActive**](win32-tssessiondirectory-setsessiondirectoryactive.md)
</dt> <dd>

La méthode **SetSessionDirectoryActive** définit la propriété **SessionDirectoryActive** .

</dd> <dt>

[**Méthode SetSessionDirectoryExposeServerIP**](win32-tssessiondirectory-setsessiondirectoryexposeserverip.md)
</dt> <dd>

Définit la propriété **SessionDirectoryExposeServerIP** .

</dd> <dt>

[**Méthode SetSessionDirectoryProperty**](win32-tssessiondirectory-setsessiondirectoryproperty.md)
</dt> <dd>

Définit la propriété **SessionDirectoryLocation** ou la propriété **SessionDirectoryClusterName** de la classe.

</dd> <dt>

[**Méthode SetTSRedirectorMode**](settsredirectormode-win32-tssessiondirectory.md)
</dt> <dd>

Définit la valeur pour indiquer si le serveur agira en tant que redirecteur de Services Bureau à distance.

</dd> </dl>
