# <a name="scriptable-virtual-channels"></a>Canaux virtuels scriptables

Pour plus d’informations sur l’utilisation de canaux virtuels scriptables, consultez [implémentation de canaux virtuels scriptables à l’aide de connexion Bureau à distance par le Web](implementing-scriptable-virtual-channels-using-remote-desktop-web-connection.md).

Pour plus d’informations sur les canaux virtuels, consultez [services Bureau à distance des canaux virtuels](terminal-services-virtual-channels.md).

**Windows xp :** Windows xp comprend Connexion Bureau à distance par le Web, qui est une version mise à jour du client avancé des Services Terminal server (TSAC). Connexion Bureau à distance par le Web prend en charge [protocole RDP (Remote Desktop Protocol)](remote-desktop-protocol.md) (RDP) 5,1 et offre une compatibilité descendante avec RDP 5,0.
