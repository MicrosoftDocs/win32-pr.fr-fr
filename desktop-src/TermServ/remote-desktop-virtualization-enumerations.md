# <a name="remote-desktop-virtualization-enumerations"></a>Énumérations de la virtualisation Bureau à distance

L’API de virtualisation Bureau à distance prend en charge les types d’énumération suivants.

## <a name="in-this-section"></a>Dans cette section

<dl> <dt>

[**\_type de modification cible \_**](/windows/desktop/api/SessDirPublicTypes/ne-sessdirpublictypes-target_change_type)
</dt> <dd>

Spécifie le type de modification qui s’est produite dans une cible.

</dd> <dt>

[**\_État cible**](/windows/desktop/api/SessDirPublicTypes/ne-sessdirpublictypes-target_state)
</dt> <dd>

Indique l’état d’une cible.

</dd> <dt>

[**TYPE de cible \_**](/windows/desktop/api/SessDirPublicTypes/ne-sessdirpublictypes-target_type)
</dt> <dd>

Indique si une cible appartient à un pool ou une batterie de serveurs.

</dd> <dt>

[**État de TSSESSION \_**](/windows/desktop/api/SessDirPublicTypes/ne-sessdirpublictypes-tssession_state)
</dt> <dd>

Indique l’état d’une session.

</dd> <dt>

[**TSSD \_ AddrV46Type**](/windows/desktop/api/SessDirPublicTypes/ne-sessdirpublictypes-tssd_addrv46type)
</dt> <dd>

Indique le type d’adresse IP.

</dd> <dt>

[**\_Tri TS \_ TS \_ par**](/windows/desktop/api/sbtsv/ne-sbtsv-ts_sb_sort_by)
</dt> <dd>

Spécifie l’ordre de tri. Elle est utilisée comme paramètre dans la méthode [**EnumerateTargets**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbresourcepluginstore-enumeratetargets) .

</dd> </dl>
