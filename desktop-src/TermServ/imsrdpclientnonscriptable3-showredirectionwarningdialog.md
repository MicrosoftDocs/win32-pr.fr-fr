# <a name="imsrdpclientnonscriptable3showredirectionwarningdialog-property"></a>IMsRdpClientNonScriptable3 :: ShowRedirectionWarningDialog, propriété

Spécifie ou récupère une valeur indiquant si la boîte de dialogue d’avertissement de redirection doit s’afficher.

Cette propriété est en lecture/écriture.

## <a name="syntax"></a>Syntaxe


```C++
HRESULT put_ShowRedirectionWarningDialog(
  [in]  VARIANT_BOOL fShowRdrDlg
);

HRESULT get_ShowRedirectionWarningDialog(
  [out] VARIANT_BOOL *pfShowRdrDlg
);
```



## <a name="property-value"></a>Valeur de la propriété

Spécifie si la boîte de dialogue d’avertissement de redirection doit s’afficher.
