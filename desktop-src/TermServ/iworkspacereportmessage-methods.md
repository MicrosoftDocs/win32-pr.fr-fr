# <a name="iworkspacereportmessage-methods"></a>Méthodes IWorkspaceReportMessage

L’interface [**IWorkspaceReportMessage**](/windows/desktop/api/workspaceruntime/nn-workspaceruntime-iworkspacereportmessage) prend en charge les méthodes suivantes.

## <a name="in-this-section"></a>Dans cette section

<dl> <dt>

[**Méthode RegisterErrorLogMessage**](/windows/desktop/api/workspaceruntime/nf-workspaceruntime-iworkspacereportmessage-registererrorlogmessage)
</dt> <dd>

Inscrit le message d’erreur spécifié à utiliser dans l’interface utilisateur.

</dd> <dt>

[**Méthode IsErrorMessageRegistered**](/windows/desktop/api/workspaceruntime/nf-workspaceruntime-iworkspacereportmessage-iserrormessageregistered)
</dt> <dd>

Détermine si un message d’erreur spécifié est inscrit dans un espace de travail spécifié.

</dd> <dt>

[**Méthode RegisterErrorEvent**](/windows/desktop/api/workspaceruntime/nf-workspaceruntime-iworkspacereportmessage-registererrorevent)
</dt> <dd>

Inscrit le message d’événement d’erreur spécifié à utiliser dans l’interface utilisateur.

</dd> </dl>
