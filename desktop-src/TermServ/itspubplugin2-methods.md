# <a name="itspubplugin2-methods"></a>Méthodes ItsPubPlugin2

L’interface [**ItsPubPlugin2**](/windows/desktop/api/tspubplugin2com/nn-tspubplugin2com-itspubplugin2) expose les méthodes suivantes.

## <a name="in-this-section"></a>Dans cette section

<dl> <dt>

[**Méthode DeletePersonalDesktopAssignment**](/windows/desktop/api/tspubplugin2com/nf-tspubplugin2com-itspubplugin2-deletepersonaldesktopassignment)
</dt> <dd>

Appelé pour supprimer un mappage entre l’utilisateur spécifié et un ordinateur virtuel dans une collection de bureaux virtuels personnels.

</dd> <dt>

[**Méthode GetResource2**](/windows/desktop/api/tspubplugin2com/nf-tspubplugin2com-itspubplugin2-getresource2)
</dt> <dd>

Cette méthode est réservée et doit toujours retourner **E \_ NOTIMPL**.

</dd> <dt>

[**Méthode GetResource2List**](/windows/desktop/api/tspubplugin2com/nf-tspubplugin2com-itspubplugin2-getresource2list)
</dt> <dd>

Récupère une liste des ressources affectées à l’utilisateur spécifié.

</dd> <dt>

[**Méthode ResolvePersonalDesktop**](/windows/desktop/api/tspubplugin2com/nf-tspubplugin2com-itspubplugin2-resolvepersonaldesktop)
</dt> <dd>

Appelé pour résoudre un mappage entre l’utilisateur spécifié et un ordinateur virtuel dans une collection de bureaux virtuels personnels.

</dd> </dl>
