# <a name="iremotedesktopclienteventsonconnecting-method"></a>IRemoteDesktopClientEvents :: OnConnecting, méthode

Appelé lorsque le contrôle client tente d’établir une connexion à une session à distance.

## <a name="syntax"></a>Syntaxe


```C++
void OnConnecting();
```



## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.

## <a name="return-value"></a>Valeur retournée

Cette méthode ne retourne pas de valeur.
