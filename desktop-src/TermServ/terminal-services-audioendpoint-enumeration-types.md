# <a name="remote-desktop-services-audioendpoint-enumeration-types"></a>Types d’énumération AudioEndpoint Services Bureau à distance

Les types d’énumération suivants sont utilisés avec l’API AudioEndpoint.

## <a name="in-this-section"></a>Dans cette section

<dl> <dt>

[**\_indicateurs de position AE \_**](/windows/desktop/api/Audioengineendpoint/ne-audioengineendpoint-ae_position_flags)
</dt> <dd>

Définit des constantes pour la structure de la [**\_ \_ position actuelle AE**](/windows/desktop/api/Audioengineendpoint/ns-audioengineendpoint-ae_current_position) . Ces constantes décrivent le degré de validité de la position actuelle.

</dd> <dt>

[**\_indicateurs de mémoire tampon APO \_**](/windows/desktop/api/Audioapotypes/ne-audioapotypes-apo_buffer_flags)
</dt> <dd>

Définit les indicateurs de validation de la mémoire tampon pour la structure de [**\_ \_ propriété de connexion APO**](/windows/desktop/api/Audioapotypes/ns-audioapotypes-apo_connection_property) associée à chaque connexion APO.

</dd> </dl>

## <a name="remarks"></a>Notes

L’API Services Bureau à distance AudioEndpoint est destinée à être utilisée dans Bureau à distance scénarios ; ce n’est pas le cas pour les applications clientes.
