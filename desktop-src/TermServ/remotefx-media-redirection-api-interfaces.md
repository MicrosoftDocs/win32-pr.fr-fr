# <a name="remotefx-media-redirection-api-interfaces"></a>RemoteFX Interfaces d’API de redirection de média

l’API de redirection de média RemoteFX prend en charge les interfaces suivantes.

## <a name="in-this-section"></a>Dans cette section

<dl> <dt>

[**IWTSBitmapRenderer**](/windows/desktop/api/tsvirtualchannels/nn-tsvirtualchannels-iwtsbitmaprenderer)
</dt> <dd>

Utilisé par un plug-in de canal virtuel dynamique pour restituer des bitmaps.

</dd> <dt>

[**IWTSBitmapRendererCallback**](/windows/desktop/api/tsvirtualchannels/nn-tsvirtualchannels-iwtsbitmaprenderercallback)
</dt> <dd>

Un plug-in de canal virtuel dynamique implémente cette interface pour être averti lorsque la taille de la zone de rendu change.

</dd> <dt>

[**IWTSBitmapRenderService**](/windows/desktop/api/tsvirtualchannels/nn-tsvirtualchannels-iwtsbitmaprenderservice)
</dt> <dd>

Ce service est utilisé pour créer un mappage visuel sur le client correspondant à une fenêtre mappée sur le serveur.

</dd> <dt>

[**IWTSPluginServiceProvider**](/windows/desktop/api/tsvirtualchannels/nn-tsvirtualchannels-iwtspluginserviceprovider)
</dt> <dd>

Offre un moyen pour les plug-ins de canal virtuel dynamiques d’interroger plusieurs services de Bureau à distance client.

</dd> </dl>
