# <a name="remote-desktop-connection-broker-client-api"></a>API du client Connexion Bureau à distance Broker

Le Connexion Bureau à distance Broker est un composant du déploiement d’infrastructure VDI (Virtual Desktop Infrastructure), qui, pour une connexion à distance, trouvera une cible appropriée à partir d’un pool d’ordinateurs dans lequel la connexion peut être redirigée et renvoie l’adresse IP de la cible. L’API cliente du Service Broker pour les connexions permet aux fournisseurs de protocoles tiers de tirer parti du répartiteur de connexions pour accélérer la gestion des connexions à l’aide de leur protocole sur des ordinateurs virtuels ou des serveurs Bureau à distance dans une batterie.
