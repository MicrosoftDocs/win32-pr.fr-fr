# <a name="requirements-for-remote-desktop-web-connection"></a>Configuration requise pour Connexion Bureau à distance par le Web

Connexion Bureau à distance par le Web nécessite deux systèmes configurés comme suit :

-   Un serveur hôte exécutant Windows et IIS 4,0 ou une version ultérieure.
-   Client qui peut télécharger et exécuter le contrôle ActiveX Bureau à distance. Tout client exécutant Microsoft Internet Explorer 4,01 (de préférence Service Pack 2) ou une version ultérieure peut exécuter le contrôle ActiveX Bureau à distance.

L’utilisateur qui installe le logiciel doit disposer des privilèges suffisants pour installer le logiciel. Par exemple, sur Windows XP, l’utilisateur doit être membre du groupe utilisateurs avec pouvoir ou administrateurs.
