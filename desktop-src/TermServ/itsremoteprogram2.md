# <a name="itsremoteprogram2-interface"></a>Interface ITSRemoteProgram2

Définit des propriétés à utiliser avec un RemoteApp.

## <a name="members"></a>Membres

L’interface **ITSRemoteProgram2** hérite de [**ITSRemoteProgram**](itsremoteprogram.md). **ITSRemoteProgram2** a également les types de membres suivants :

-   [Propriétés](#properties)

### <a name="properties"></a>Propriétés

L’interface **ITSRemoteProgram2** possède les propriétés suivantes.



| Propriété                                                                                  | Type d’accès           | Description                                                 |
|:------------------------------------------------------------------------------------------|:----------------------|:------------------------------------------------------------|
| [**RemoteApplicationArgs**](itsremoteprogram2-remoteapplicationargs.md)<br/>       | Écriture seule<br/> | Arguments de ligne de commande pour le RemoteApp.<br/>    |
| [**RemoteApplicationName**](itsremoteprogram2-remoteapplicationname.md)<br/>       | Écriture seule<br/> | Nom du RemoteApp.<br/>                       |
| [**RemoteApplicationProgram**](itsremoteprogram2-remoteapplicationprogram.md)<br/> | Écriture seule<br/> | Chemin d’accès et nom de fichier du programme RemoteApp.<br/> |
