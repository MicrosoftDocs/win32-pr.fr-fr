# <a name="remote-desktop-protocol-provider-enumerations"></a>Énumérations du fournisseur protocole RDP (Remote Desktop Protocol)

Les types d’énumération suivants sont pris en charge par l’API du fournisseur protocole RDP (Remote Desktop Protocol).

## <a name="in-this-section"></a>Dans cette section

<dl> <dt>

[**\_niveau de \_ paramètre de connexion Wrds \_**](/windows/desktop/api/Wtsdefs/ne-wtsdefs-wrds_connection_setting_level)
</dt> <dd>

Spécifie le type de structure contenu dans le membre **WRdsConnectionSetting** de la structure des [**\_ \_ paramètres de connexion Wrds**](/windows/desktop/api/Wtsdefs/ns-wtsdefs-wrds_connection_settings) .

</dd> <dt>

[**niveau de paramètre de l' \_ écouteur Wrds \_ \_**](/windows/desktop/api/Wtsdefs/ne-wtsdefs-wrds_listener_setting_level)
</dt> <dd>

Utilisé pour spécifier le type de structure contenu dans le membre **WRdsListenerSetting** de la structure des [**paramètres de l' \_ écouteur \_ Wrds**](/windows/desktop/api/Wtsdefs/ns-wtsdefs-wrds_listener_settings) .

</dd> <dt>

[**\_niveau de paramètre Wrds \_**](/windows/desktop/api/Wtsdefs/ne-wtsdefs-wrds_setting_level)
</dt> <dd>

Spécifie le type de structure contenu dans le membre **WRdsSetting** de la structure de [**\_ paramètres Wrds**](/windows/desktop/api/Wtsdefs/ns-wtsdefs-wrds_settings) .

</dd> <dt>

[**\_État du paramètre Wrds \_**](/windows/desktop/api/Wtsdefs/ne-wtsdefs-wrds_setting_status)
</dt> <dd>

Spécifie l’état d’un paramètre de stratégie pour les différents membres de la structure [**Wrds \_ Settings \_ 1**](/windows/desktop/api/Wtsdefs/ns-wtsdefs-wrds_settings_1) .

</dd> <dt>

[**\_type de paramètre Wrds \_**](/windows/desktop/api/Wtsdefs/ne-wtsdefs-wrds_setting_type)
</dt> <dd>

Spécifie la catégorie des paramètres qui sont stockés dans une structure de [**\_ paramètres Wrds**](/windows/desktop/api/Wtsdefs/ns-wtsdefs-wrds_settings) .

</dd> <dt>

[**WTS \_ TYPE de certificat \_**](/windows/desktop/api/Wtsdefs/ne-wtsdefs-wts_cert_type)
</dt> <dd>

Contient des valeurs qui spécifient le type de certificat utilisé pour obtenir une licence.

</dd> <dt>

[**WTS \_ \_réponse du \_ redirecteur d’erreur de connexion \_**](/windows/desktop/api/Wtsdefs/ne-wtsdefs-wts_logon_error_redirector_response)
</dt> <dd>

Contient des valeurs qui spécifient la réponse par défaut du protocole à une erreur d’ouverture de session.

</dd> <dt>

[**WTS \_ \_État de drainage du RCM \_**](/windows/desktop/api/Wtsdefs/ne-wtsdefs-wts_rcm_drain_state)
</dt> <dd>

Contient des informations sur l’état de drainage du serveur hôte de session Bureau à distance (hôte de session Bureau à distance).

</dd> <dt>

[**WTS \_ \_État du service RCM \_**](/windows/desktop/api/Wtsdefs/ne-wtsdefs-wts_rcm_service_state)
</dt> <dd>

Contient des informations sur l’état du service Services Bureau à distance.

</dd> </dl>
