# <a name="retrieving-device-information"></a>Récupération des informations de l’appareil

L’API WPD fournit un ensemble complet d’interfaces et de méthodes que vous pouvez utiliser pour récupérer des informations pour un appareil donné. En général, les données de l’appareil peuvent être divisées en trois catégories, comme décrit dans le tableau suivant.



| Données des appareils                                                                             | Description                                                                               |
|-----------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------|
| [Catégories fonctionnelles](retrieving-the-functional-categories-supported-by-a-device.md) | Les catégories fonctionnelles décrivent les opérations qu’un appareil est capable d’effectuer. |
| [Types de contenu](retrieving-the-content-types-supported-by-a-device.md)                 | Les types de contenu indiquent les types de contenu qu’un appareil peut afficher.         |
| [Événements](retrieving-the-events-supported-by-a-device.md)                               | Certains appareils sont capables de déclencher des événements que votre application peut suivre et auxquels elle doit répondre. |
