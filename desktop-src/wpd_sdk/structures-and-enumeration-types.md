# <a name="structures-and-enumeration-types-wpd-api"></a>Structures et types énumération (API WPD)

Les appareils mobiles Windows définissent les structures et les types d’énumération suivants.



| Énumération                                                                                                 | Description                                                                                          |
|-------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------|
| [**OPTIONS de suppression d' \_ objet \_**](delete-object-options.md)                                                    | Décrit les options prises en charge par un appareil lors de la suppression d’un objet.                            |
| [**\_types de messages SMS \_**](sms-message-types.md)                                                            | Décrit le type de contenu d’un message SMS (Short Message Service).                                 |
| [**\_types de débit wpd \_**](wpd-bitrate-types.md)                                                            | Décrit le type de vitesse de transmission d’un fichier audio.                                                             |
| [**\_modes de capture wpd \_**](wpd-capture-modes.md)                                                            | Décrit le mode de minutage de capture d’une capture d’image continue.                                          |
| [**valeurs d’état de la \_ couleur wpd \_ corrigé \_ \_**](wpd-color-corrected-status-values.md)                          | Décrit l’état de correction de couleur d’un fichier image ou vidéo.                                     |
| [**\_valeurs d' \_ État du rognage wpd \_**](wpd-cropped-status-values.md)                                           | Décrit l’état de rognage d’une image.                                                           |
| [**\_transports d’appareils wpd \_**](wpd-device-transports.md)                                                    | Décrit les transports pris en charge par un appareil (USB, IP ou Bluetooth)                               |
| [**\_modes d’effet wpd \_**](wpd-effect-modes.md)                                                              | Décrit différents effets visuels qui peuvent être appliqués à une image.                                    |
| [**\_modes de \_ contrôle d’exposition wpd \_**](wpd-exposure-metering-modes.md)                                       | Décrit le mode de contrôle à utiliser lors de l’estimation de l’exposition pour la capture d’image continue par un appareil.     |
| [**\_modes de \_ programme d’exposition wpd \_**](wpd-exposure-program-modes.md)                                         | Décrit un mode d’exposition à utiliser lors de la capture d’images avec un appareil.                               |
| [**\_modes flash \_ wpd**](wpd-flash-modes.md)                                                                | Décrit un mode flash à utiliser lors de la capture d’images avec un appareil.                                   |
| [**\_modes de \_ contrôle de focus wpd \_**](wpd-focus-metering-modes.md)                                             | Décrit comment un appareil doit déterminer quelle partie d’un cadre utiliser pour définir le focus.                       |
| [**\_modes de focus wpd \_**](wpd-focus-modes.md)                                                                | Décrit le mode Focus utilisé par un appareil de capture d’image continue.                                       |
| [**\_méta- \_ genres wpd**](wpd-meta-genres.md)                                                                | Décrit le type de genre large d’un fichier multimédia.                                                      |
| [**\_types d' \_ utilisation des paramètres wpd \_**](wpd-parameter-usage-types.md)                                           | Décrit l’utilisation d’un paramètre de méthode.                                                            |
| [**\_sources d’alimentation wpd \_**](wpd-power-sources.md)                                                            | Décrit la source d’alimentation utilisée par un appareil.                                                       |
| [**\_types d' \_ entrée de profil d’informations de rendu wpd \_ \_ \_**](wpd-rendering-information-profile-entry-types.md) | Indique si une entrée de type de profil d’informations de rendu correspond à un objet ou à une ressource. |
| [**\_valeurs d' \_ unités de données \_ \_ de la section wpd**](wpd-section-data-units-values.md)                                  | Décrit les unités d’une section de référence de données (octets ou millisecondes)                          |
| [**\_types d' \_ encodage SMS wpd \_**](wpd-sms-encoding-types.md)                                                 | Décrit le type d’encodage d’un message SMS.                                                       |
| [**\_types d' \_ analyses \_ vidéo wpd**](wpd-video-scan-types.md)                                                     | Décrit comment les champs d’un fichier vidéo sont encodés.                                                |
| [**\_paramètres d' \_ équilibre des blancs wpd \_**](wpd-white-balance-settings.md)                                         | Décrit comment un appareil vidéo ou d’images pondération les canaux de couleurs pour obtenir une balance des blancs appropriée.      |
| [**WpdAttributeForm**](wpdattributeform.md)                                                                | Décrit comment une propriété stocke ses valeurs.                                                          |
| [**WpdParameterAttributeForm**](wpd-parameter-attribute-form.md)                                           | Décrit comment un paramètre stocke ses valeurs.                                                         |
| [**\_valeurs de \_ type de stockage wpd \_**](wpd-storage-type-values.md)                                               | Décrit les différents types de stockage.                                                               |
| [**\_types d’appareils wpd \_**](wpd-device-types.md)                                                              | Décrit le type de périphérique portable.                                                                  |
| [**États de l' \_ opération wpd \_**](wpd-operation-states.md)                                                      | Décrit l’état actuel d’une opération en cours.                                             |
