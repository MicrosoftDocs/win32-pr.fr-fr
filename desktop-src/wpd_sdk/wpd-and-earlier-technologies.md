# <a name="wpd-and-earlier-technologies"></a>WPD et technologies antérieures

WPD contient de nombreuses améliorations pour les appareils photo numériques fixes et les lecteurs multimédias numériques portables. il s’agit de la méthode recommandée pour accéder à ces appareils dans les nouvelles versions de Windows. les api d’acquisition d’images Windows (WIA) et Windows Media Gestionnaire de périphériques (WMDM) existantes dans Windows continueront d’être prises en charge et la compatibilité avec WPD sera assurée par les couches de compatibilité. Ces couches de compatibilité garantissent que les appareils WPD peuvent être exposés de manière transparente aux applications WIA et WMDM existantes en tant que périphériques WIA ou fournisseurs de services WMDM, respectivement. Toutefois, ces applications peuvent ne pas avoir les mêmes fonctionnalités qu’avec la nouvelle API WPD et les nouveaux appareils basés sur la MTP.

Le pilote MTP WPD prend également en charge les appareils de protocole de transfert d’images (PTP). Les appareils PTP pris en charge dans WIA continueront de fonctionner dans des applications WIA via la couche de compatibilité. Microsoft s’engage à garantir la compatibilité des applications et des appareils numériques toujours dans la prochaine version de Windows et ces nouvelles technologies sont conçues pour compléter et améliorer l’expérience de l’appareil photo numérique dans Windows.
