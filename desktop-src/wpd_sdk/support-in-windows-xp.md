pour que votre application s’exécute dans Windows XP, l’utilisateur a besoin d’une copie du runtime Windows Media Format sur son ordinateur. Vous pouvez redistribuer ce Runtime avec votre application WPD. (le runtime se trouve dans un dossier de redistribuables dans le kit de développement logiciel (SDK) de Format multimédia Windows.)
