# <a name="support-in-windows-xp"></a>Prise en charge dans Windows XP

Pour que votre application s’exécute sous Windows XP, l’utilisateur doit disposer d’une copie du runtime Windows Media format sur son ordinateur. Vous pouvez redistribuer ce Runtime avec votre application WPD. (Le runtime se trouve dans un dossier de redistribuables dans le kit de développement logiciel (SDK) du format Windows Media.)
