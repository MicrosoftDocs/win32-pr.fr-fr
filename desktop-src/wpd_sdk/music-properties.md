# <a name="music-properties"></a>Musique Sous

Windows Appareils mobiles prend en charge les propriétés de musique suivantes.



| Propriété               | VarType        | Description                                |
|------------------------|----------------|--------------------------------------------|
| **\_album de musique wpd \_**  | **\_LPWStr VT** | L’album auquel le suivi appartient.            |
| **\_paroles de musique wpd \_** | **\_LPWStr VT** | Paroles pour la musique.                  |
| **\_ambiance musicale \_ wpd**   | **\_LPWStr VT** | Chaîne décrivant l’ambiance de la musique. |
| **\_piste de musique wpd \_**  | **VT \_ UI4**    | Numéro de piste de l’enregistrement sur l’album. |
