# <a name="wpd-constants"></a>Constantes WPD

Windows Les appareils mobiles définissent les types de constantes suivants.



| Type de constante                                              | Description                                                                                                                                                                                                                               |
|------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Constantes générales](general-constants.md)                 | Constantes d’utilisation générale pour les appareils et les applications.                                                                                                                                                                                       |
| [Constantes d’erreur](error-constants.md)                     | Valeurs d’erreur retournées par les méthodes.                                                                                                                                                                                                         |
| [Constantes d’événement](event-constants.md)                     | types d’événements définis par Windows appareils mobiles. Les types d’événements incluent l’objet ajouté, objet supprimé, objet mis à jour, objet déplacé, objet copié, transfert d’objets demandé, réinitialisation de l’appareil, appareil supprimé et fonctionnalités de l’appareil mises à jour. |
| [GUID de type d’objet](object-type-guids.md)                 | types d’objets définis par Windows appareils mobiles. Les types d’objets incluent les objets audio, de courrier électronique et d’image.                                                                                                                                  |
| [GUID de format d’objet](object-format-guids.md)             | types de format de fichier définis par Windows appareils mobiles. Les types de format sont WMA, WMV et JPEG.                                                                                                                                           |
| [GUID de type de ressource](resource-type-guids.md)             | types de ressources définis par Windows appareils mobiles. Les types de ressources incluent des données audio et des images miniatures.                                                                                                                                |
| [Propriétés et attributs](properties-and-attributes.md) | propriétés et attributs définis par Windows appareils mobiles.                                                                                                                                                                            |
