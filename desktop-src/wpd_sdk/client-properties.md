# <a name="client-properties"></a>Propriétés du client

Windows Les appareils mobiles prennent en charge les propriétés de client suivantes.



| Propriété                                          | VarType                   | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
|---------------------------------------------------|---------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **\_ \_ accès souhaité au client wpd \_**                  | **VT \_ UI4**               | Valeur qui spécifie que le client requiert des autorisations d’accès uniques, telles que la \_ lecture seule générique. Si cette propriété n’est pas définie, WPD suppose \_ \| l’écriture générique Read générique \_ , qui est la valeur par défaut recommandée.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| **\_ \_ version principale du client wpd \_**                   | **VT \_ UI4**               | Nombre qui spécifie la version principale du client.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| **\_ \_ version mineure du client wpd \_**                   | **VT \_ UI4**               | Nombre qui spécifie la version mineure du client.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| **\_nom du client wpd \_**                             | **\_LPWStr VT**            | Chaîne de nom pour l’application cliente.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| **\_révision du client wpd \_**                         | **VT \_ UI4**               | Nombre qui spécifie la révision (ou le numéro de Build) du client.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| **\_ \_ \_ qualité \_ de service de la sécurité du client wpd \_**   | **VT \_ UI4**               | Valeur qui spécifie la qualité de service de sécurité pour la connexion au pilote. Cela concerne la qualité de sécurité des indicateurs de service pour **CreateFile**.<br/> Par exemple, ces indicateurs autorisent ou non un pilote à emprunter l’identité du client. \_L’identification de sécurité est supposée si cette valeur n’est pas fournie. En règle générale, les applications doivent définir cette valeur sur l’emprunt d’identité de sécurité \_ , car certains pilotes requièrent l’emprunt d’identité pour fonctionner. Toutefois, les applications qui s’exécutent avec des exigences de sécurité strictes avec un sous-ensemble connu de périphériques peuvent souhaiter limiter la capacité du pilote à identifier ou emprunter l’identité de l’appelant. Notez que le fait de réduire le niveau d’emprunt d’identité à une valeur inférieure à l’emprunt d’identité de sécurité \_ peut entraîner la réduction des fonctionnalités des pilotes.<br/>                                         |
| **\_mode de \_ partage du client wpd \_**                      | **VT \_ UI4**               | Valeur qui spécifie le mode de partage demandé par le client à partir du pilote. Les valeurs possibles sont les mêmes que pour **CreateFile**, par exemple la lecture de partage de fichiers \_ , l’écriture de partage de \_ fichiers, etc \_ \_ .<br/> \_ \_ \| \_ \_ L’écriture du partage de fichiers en lecture du partage de fichiers est supposée si le client ne spécifie pas ce paramètre. Il s’agit de la valeur par défaut recommandée. Toutefois, lorsqu’un mode de partage différent est requis, par exemple, \_ \_ le partage de fichiers lu pour demander un partage de lecture uniquement, ce paramètre est obligatoire.<br/>                                                                                                                                                                                                                                                                                                                                                          |
| **\_ \_ \_ clé privée d’application \_ WMDRM \_ du client wpd** | **VT \_ UI1 VT Vector \_ \| \_** | valeur qui spécifie la clé privée de l’application DRM Windows Media du client.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| **\_certificat d' \_ \_ application WMDRM du client \_ wpd**  | **VT \_ UI1 VT Vector \_ \| \_** | valeur qui spécifie le certificat DRM Windows du client.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| **\_cookie d' \_ événement de client wpd \_**                    | **\_LPWStr VT**            | Cookie fourni par le client retourné par le pilote dans les événements publiés en tant que résultat direct des opérations émises par ce client. Il s’agit d’une propriété facultative que le pilote peut fournir en tant que paramètre d’événement. Les clients utilisent généralement cette valeur pour optimiser leur gestion des événements. Par exemple, si le client supprime un objet, son gestionnaire d’événements peut choisir d’ignorer l’événement résultant, sachant qu’il a déjà pris l’action appropriée (par exemple, mise à jour de la vue dans le thread qui a émis la suppression).                                                                                                                                                                                                                                                                                                                                                                |
| **\_ \_ taille minimale du \_ tampon des résultats du \_ client wpd \_**   | **VT \_ UI4**               | Spécifie la taille minimale de la mémoire tampon (en octets) pour l’envoi de commandes au pilote. Les clients peuvent utiliser cette option pour recommander la taille minimale de la mémoire tampon interne utilisée pour envoyer des commandes au pilote. Cela peut entraîner un plus petit encombrement de mémoire pour les clients qui envoient uniquement des commandes avec des tailles de données de petite taille, telles que l’obtention ou la définition de propriétés. Le pilote peut remplacer cette taille de mémoire tampon en définissant **PortableDeviceResultsBufferMin** dans la section paramètres de l’appareil. Cette taille de mémoire tampon doit être 1024 ou une version ultérieure. Si ni le client ni le pilote ne définit la taille de la mémoire tampon, la valeur par défaut est 1 Mo.                                                                                                                                                                                                                                                  |
| **\_ \_ fermeture manuelle du client wpd \_ \_ lors de la \_ déconnexion**    | **VT \_ bool**              | Option avancée pour les clients qui appellent manuellement **IPortableDevice :: Close** ou **IPortableDeviceService :: Close** pour chaque objet déconnecté, au lieu de s’appuyer sur l’API pour effectuer cette opération. La valeur FALSe est supposée si le client ne spécifie pas ce paramètre. Il s’agit de la valeur par défaut recommandée si le client n’écoute pas les messages **WM \_ DEVICECHANGE** . Les clients qui spécifient cette option écoutent généralement les messages **WM \_ DEVICECHANGE** pour l’appareil, puis ferment manuellement les instances **IPortableDevice** et **IPortableDeviceService** lors de la déconnexion de l’appareil. Les clients qui choisissent cette option doivent gérer avec soin la durée de vie de chaque instance **IPortableDevice** et **IPortableDeviceService** en libérant ces objets lorsque l’appareil est déconnecté. Si vous ne le faites pas, vous risquez de provoquer des fuites dans les processus de WUDFHost.exe pour le pilote. |
