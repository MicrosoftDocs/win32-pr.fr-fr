# <a name="wpd_section_data_units_values-enumeration"></a>\_ \_ \_ Énumération des valeurs d’unités de données de la section wpd \_

L’énumération des valeurs d’unités de données de la **\_ section \_ \_ \_ wpd** indique les unités pour une section de données référencée.

## <a name="syntax"></a>Syntaxe


```C++
typedef enum WPD_SECTION_DATA_UNITS_VALUES { 
  WPD_SECTION_DATA_UNITS_BYTES         = 0,
  WPD_SECTION_DATA_UNITS_MILLISECONDS  = 1
} ;
```
