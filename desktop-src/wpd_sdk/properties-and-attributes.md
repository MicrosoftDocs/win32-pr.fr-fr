# <a name="properties-and-attributes"></a>Propriétés et attributs

Les propriétés décrivent les objets, les propriétés, les ressources et les périphériques. Les attributs sont un type spécial de propriété qui décrit d’autres propriétés ou ressources.

Les propriétés et les attributs sont définis comme des structures **PROPERTYKEY** avec deux parties : un GUID de catégorie et un ID unique pour cette catégorie. Les propriétés qui décrivent les éléments connexes partagent un GUID commun ; ce GUID est sa catégorie de propriété. Une catégorie peut décrire une ou plusieurs propriétés, chacune avec un ID unique. Microsoft définit un ensemble de catégories de propriétés. vous pouvez les utiliser pour vos propres propriétés ou créer vous-même des catégories.
