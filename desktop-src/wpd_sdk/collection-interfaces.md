# <a name="collection-interfaces"></a>Interfaces de collection

Les interfaces suivantes sont utilisées pour contenir des groupes de valeurs de différents types, y compris des valeurs **PROPERTYKEY** , des valeurs **GUID** ou d’autres interfaces de collection. Ces interfaces sont utilisées par les applications clientes et les pilotes pour conserver les données des paramètres de la méthode.



| Interface                                                                            | Description                                                            |
|--------------------------------------------------------------------------------------|------------------------------------------------------------------------|
| [**IPortableDeviceKeyCollection**](iportabledevicekeycollection.md)                 | Contient une collection de valeurs **PROPERTYKEY** .                          |
| [**IPortableDevicePropVariantCollection**](iportabledevicepropvariantcollection.md) | Contient une collection de valeurs **PROPVARIANT** du même VarType.      |
| [**IPortableDeviceValues**](iportabledevicevalues.md)                               | Contient une collection de paires **PROPERTYKEY** / **PROPVARIANT** .           |
| [**IPortableDeviceValuesCollection**](iportabledevicevaluescollection.md)           | Contient une collection d’interfaces **IPortableDeviceValues** de base zéro. |
