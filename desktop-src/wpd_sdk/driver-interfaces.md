# <a name="driver-interfaces"></a>Interfaces du pilote

Les interfaces suivantes sont utilisées uniquement par un pilote Windows Mobile Devices.



| Interface                                | Description                                                                                                                                            |
|------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IWpdSerializer**](iwpdserializer.md) | Utilisé par le pilote de périphérique pour sérialiser des interfaces **IPortableDeviceValues** vers et à partir des mémoires tampons de données brutes utilisées pour communiquer avec l’application. |
