# <a name="retrieving-object-information"></a>Récupération des informations sur les objets

En plus de récupérer les informations de l’appareil, l’API WPD fournit un ensemble d’interfaces et de méthodes que vous pouvez utiliser pour récupérer des informations pour un objet donné. En général, la récupération des informations sur les objets peut être divisée en deux catégories, comme décrit dans le tableau suivant.



| Données des appareils                                                                                    | Description                                                                               |
|------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------|
| [Identificateur d’objet](retrieving-an-object-identifier-from-a-persistent-unique-identifier.md)   | Vous pouvez récupérer l’identificateur d’objet qui est mappé à un identificateur unique persistant donné. |
| [Identificateurs d’objets fonctionnels](retrieving-the-functional-object-identifiers-for-a-device.md) | Chaque catégorie fonctionnelle prend en charge un ou plusieurs objets fonctionnels.                         |
