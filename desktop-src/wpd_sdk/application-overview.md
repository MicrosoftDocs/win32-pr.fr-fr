# <a name="application-overview"></a>Vue d’ensemble de l’application

Les appareils mobiles Windows (WPD) fournissent une infrastructure qui permet d’accéder aux appareils multifonctions qui stockent différents contenus multimédias et non multimédias. Cette infrastructure Active les appareils convergents et se focalise initialement sur des périphériques de stockage intelligent tels que des lecteurs multimédias portables, des appareils photo numériques fixes et des téléphones mobiles.

WPD est constitué d’une interface de pilote de périphérique (DDI) et d’un ensemble d’interfaces de programmation d’applications (API). Cette vue d’ensemble décrit comment vous pouvez tirer parti de ces API dans votre application Windows.

L’infrastructure WPD offre des mécanismes de contrôle et de représentation d’appareil hautement extensibles. Cela permet aux fournisseurs de solutions verticales d’utiliser l’API WPD pour créer des expériences utilisateur améliorées qui se trouvent en dehors de l’ensemble standardisé WPD. Voici des exemples de logiciels fournis par un fournisseur qui effectuent des mises à jour du microprogramme ou des diagnostics de l’appareil (pour le support à distance).
