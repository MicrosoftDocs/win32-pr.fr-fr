# <a name="supported-tasks-in-wpd-applications"></a>Tâches prises en charge dans les applications WPD

Les interfaces d’application WPD prennent en charge les tâches suivantes :

-   Connecter à un appareil
-   Énumérer les appareils attachés
-   Examiner les capacités d’un appareil
-   Explorer le contenu d’un appareil
-   Transférer du contenu vers ou créer du contenu sur un appareil
-   Modifier le contenu sur l’appareil
-   Modifier des objets sur un appareil
-   Contrôler l’appareil
-   Surveiller les événements des appareils

pour plus d’informations sur chaque tâche, ainsi que des détails sur la façon de programmer une tâche donnée, reportez-vous au [Guide de programmation des appareils mobiles Windows](programming-guide.md).
