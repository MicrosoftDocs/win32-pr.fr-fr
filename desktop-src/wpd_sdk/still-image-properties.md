# <a name="still-image-properties"></a>Propriétés de l’image continue

Appareils mobiles Windows prend en charge les propriétés d’image continue suivantes.



| Propriété                                                                                                                                                       | VarType        | Description                                                                                                                                                                                                                                                                                                                  |
|----------------------------------------------------------------------------------------------------------------------------------------------------------------|----------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **l' \_ \_ artiste d’images fixes wpd \_**                                                                                                                                  | **\_LPWStr VT** | Nom du propriétaire/de l’utilisateur de l’appareil.                                                                                                                                                                                                                                                                                    |
| <span id="wpd_still_image_burst_interval"></span><span id="WPD_STILL_IMAGE_BURST_INTERVAL"></span>**\_intervalle de \_ \_ rafales d’images fixes pour wpd \_**                       | **VT \_ UI4**    | Dans une capture d’image en mode rafale, délai entre les captures d’images, en millisecondes.                                                                                                                                                                                                                                       |
| <span id="wpd_still_image_burst_number"></span><span id="WPD_STILL_IMAGE_BURST_NUMBER"></span>**\_nombre de \_ \_ rafales d’images fixes wpd \_**                             | **VT \_ UI4**    | Dans une capture d’image en mode rafale, nombre d’images à capturer.                                                                                                                                                                                                                                                              |
| **\_ \_ retardement de capture d’image wpd \_ \_**                                                                                                                          | **VT \_ UI4**    | Spécifie l’intervalle de temps entre la commande pour capturer une image et la capture réelle, en millisecondes.                                                                                                                                                                                                                    |
| **\_format de \_ capture d’images wpd toujours \_ \_**                                                                                                                         | **\_CLSID VT**  | Format à utiliser pour capturer l’image. Il s’agit de l’une des [valeurs de GUID de format](object-format-guids.md) définies par les appareils mobiles Windows.                                                                                                                                                                        |
| <span id="wpd_still_image_capture_mode"></span><span id="WPD_STILL_IMAGE_CAPTURE_MODE"></span>**\_mode de \_ capture d’image toujours \_ wpd \_**                             | **VT \_ UI4**    | Énumérateur [**de \_ \_ modes de capture wpd**](wpd-capture-modes.md) qui spécifie le mode de capture à utiliser pour capturer une image.                                                                                                                                                                                                    |
| **\_résolution de \_ capture d’images wpd toujours \_ \_**                                                                                                                     | **\_LPWStr VT** | Chaîne qui spécifie la taille de l’image à capturer. La taille de l’image est spécifiée en pixels, au format «*largeur* x *hauteur*». Par exemple : « 800x600 ».                                                                                                                                                                          |
| **\_paramètre de \_ compression d’image wpd Still \_ \_**                                                                                                                    | **\_UI8 VT**    | Décrit les paramètres de compression d’image qui sont utilisés lors de la capture d’une image. Ceci est spécifique à l’appareil.                                                                                                                                                                                                                     |
| **le \_ \_ contraste des images wpd continue \_**                                                                                                                                | **VT \_ UI4**    | Nombre qui spécifie le paramètre de contraste à appliquer lors de la capture d’une image, avec zéro étant le plus petit contraste et un grand nombre qui indiquent un contraste plus élevé.                                                                                                                                                            |
| **réinitialisation du \_ \_ \_ Zoom numérique d’images wpd \_**                                                                                                                           | **VT \_ UI4**    | Nombre qui spécifie la quantité de zoom numérique à utiliser lors de la capture d’une image. Le nombre est égal ou supérieur à 1, mis à l’échelle par un facteur de 10. La valeur 1 (représentant 10) n’indique aucun zoom numérique. la valeur 2 (représentant 20) indique un zoom 2x (1/4 de l’image affichée par un zoom 1x est capturé).                  |
| **WPD \_ - \_ \_ mode d’effet d’image continue \_**                                                                                                                            | **VT \_ UI4**    | Énumérateur [**des \_ \_ modes d’effet wpd**](wpd-effect-modes.md) qui spécifie un effet d’image à utiliser lors de la capture d’une image.                                                                                                                                                                                                   |
| **correction du décalage de l’exposition de l' \_ image wpd continue \_ \_ \_ \_**                                                                                                            | **VT \_ UI4**    | Nombre qui spécifie les unités APEX pour ajuster l’exposition. Les unités sont des arrêts x 1 000. La spécification de zéro indique la valeur par défaut de l’usine.                                                                                                                                                                                       |
| **INDEX d’exposition de l' \_ \_ image de l' \_ API WPD \_**                                                                                                                         | **VT \_ UI4**    | Nombre qui spécifie la vitesse de film ISO à émuler lors de la capture d’une image. La valeur 0xFFFF indique que cette valeur doit être définie automatiquement.                                                                                                                                                                      |
| **le \_ mode de contrôle de l’exposition des images wpd continue \_ \_ \_ \_**                                                                                                                | **VT \_ UI4**    | Énumérateur [**de \_ \_ \_ modes de contrôle d’exposition wpd**](wpd-exposure-metering-modes.md) qui spécifie le mode de contrôle à utiliser lors de la capture d’une image.                                                                                                                                                                          |
| <span id="wpd_still_image_exposure_program_mode"></span><span id="WPD_STILL_IMAGE_EXPOSURE_PROGRAM_MODE"></span>**le \_ \_ mode de \_ programme d’exposition des images wpd \_ continue \_** | **VT \_ UI4**    | Énumérateur [**de \_ \_ \_ modes de programme d’exposition wpd**](wpd-exposure-program-modes.md) qui spécifie le programme d’exposition à utiliser lors de la capture d’une image.                                                                                                                                                                         |
| **heure d’exposition de l' \_ image wpd toujours \_ \_ \_**                                                                                                                          | **VT \_ UI4**    | Vitesse d’obturation à utiliser lors de la capture d’une image, en secondes x 10 000.                                                                                                                                                                                                                                                       |
| **WPD \_ - \_ image \_ toujours \_ en mode flash**                                                                                                                             | **VT \_ UI4**    | Énumérateur [**de \_ \_ modes flash wpd**](wpd-flash-modes.md) qui spécifie le mode flash à utiliser lors de la capture d’une image.                                                                                                                                                                                                      |
| **\_FNUMBER d' \_ image de l’API WPD \_**                                                                                                                                 | **VT \_ UI4**    | Nombre qui spécifie l’ouverture de la lentille lors de la capture d’une image. Le nombre est le numéro d’arrêt f 100 (donc 14 indiquera f/1.4). Cette propriété est généralement valide uniquement lorsque [le \_ \_ mode de \_ \_ programme d' \_ exposition des images wpd](/windows) est défini sur une priorité manuelle ou d’ouverture.<br/> |
| **\_ \_ longueur focale d’image toujours \_ wpd \_**                                                                                                                           | **VT \_ UI4**    | Longueur focale, en millimètres x 100, à utiliser lors de la capture d’une image. Par exemple, 35mm serait 3500.                                                                                                                                                                                                                 |
| **la \_ \_ distance de \_ focus \_ de l’image de wpd est toujours**                                                                                                                         | **VT \_ UI4**    | Distance de focus qu’un appareil doit utiliser, en millimètres. La valeur 0xFFFF indique un focus infini (plus de 655 mètres).                                                                                                                                                                                              |
| **\_ \_ \_ le \_ mode de contrôle du focus \_ de l’image wpd continue**                                                                                                                   | **VT \_ UI4**    | Énumérateur [**de \_ \_ \_ modes de contrôle de focus wpd**](wpd-focus-metering-modes.md) qui spécifie le mode Focus qu’un appareil de capture d’image doit utiliser pour déterminer comment se concentrer sur un objet.                                                                                                                                     |
| **WPD \_ toujours \_ \_ le \_ mode focus de l’image**                                                                                                                             | **VT \_ UI4**    | Énumérateur [**de \_ \_ modes de focus wpd**](wpd-focus-modes.md) qui spécifie le mode focus à utiliser lors de la capture d’une image.                                                                                                                                                                                                      |
| **\_ \_ gain RVB d’image toujours \_ wpd \_**                                                                                                                               | **\_LPWStr VT** | Chaîne terminée par le caractère null qui spécifie le gain RVB à utiliser lors de la capture d’une image. La chaîne est de trois nombres délimités par des points-virgules sans espaces intermédiaires, au format «*R*:*G*:*B*».                                                                                                                                    |
| **la \_ \_ netteté de l’image wpd est toujours \_**                                                                                                                               | **VT \_ UI4**    | Nombre qui spécifie la netteté à utiliser lors de la capture d’une image, où zéro correspond au moins à la netteté et les nombres plus grands indiquent une netteté accrue.                                                                                                                                                                        |
| <span id="wpd_still_image_timelapse_interval"></span><span id="WPD_STILL_IMAGE_TIMELAPSE_INTERVAL"></span>**\_intervalle de \_ TIMELAPSE d’image toujours \_ wpd \_**           | **VT \_ UI4**    | Délai, en millisecondes, entre les captures d’images dans une capture d’image temporelle.                                                                                                                                                                                                                                        |
| <span id="wpd_still_image_timelapse_number"></span><span id="WPD_STILL_IMAGE_TIMELAPSE_NUMBER"></span>**\_TIMELAPSE images wpd toujours- \_ \_ \_ nombre**                 | **VT \_ UI4**    | Nombre d’images à capturer dans une capture d’image en mode temporel.                                                                                                                                                                                                                                                          |
| **\_URL de \_ chargement d’image wpd toujours \_ \_**                                                                                                                             | **\_LPWStr VT** | Spécifie une URL à laquelle l’appareil doit envoyer l’image après sa capture.                                                                                                                                                                                                                                                 |
| **\_ \_ équilibrer l’image d’images wpd toujours \_ \_**                                                                                                                          | **VT \_ UI4**    | Valeur énumérée des [**\_ \_ \_ paramètres d’équilibre blanc wpd**](wpd-white-balance-settings.md) qui spécifie les paramètres d’équilibre des blancs à utiliser lors de la capture d’une image.                                                                                                                                                             |
