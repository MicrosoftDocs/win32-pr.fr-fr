# <a name="support-in-windows-7-and-windows-vista"></a>prise en charge dans Windows 7 et Windows Vista

pour toutes les références (sku) de Windows 7 et Windows Vista, le runtime WPD est regroupé dans le cadre du système d’exploitation.

### <a name="suported-api-elements"></a>Éléments d’API en surprises

le tableau suivant identifie les fonctionnalités prises en charge pour Windows 7, Windows vista et Windows vista avec la mise à jour de la plateforme.



| Caractéristique                        | Windows 7 | Windows Vista | Windows Vista avec la mise à jour de la plateforme |
|--------------------------------|-----------|---------------|----------------------------------------|
| MTP sur USB                   | Oui       | Oui           | Oui                                    |
| MTP sur IP                    | Oui       | Oui           | Oui                                    |
| MTP sur Bluetooth             | Oui       | Non            | Oui                                    |
| Services d’appareil WPD et MTP    | Oui       | Non            | Oui                                    |
| Automation WPD                 | Oui       | Non            | Non                                     |
| Multi-fonctions/transport multiple | Oui       | Non            | Non                                     |
| Device Stage                   | Oui       | Non            | Non                                     |
| Plateforme de synchronisation des appareils           | Oui       | Non            | Non                                     |



 

### <a name="the-platform-update-for-windows-vista"></a>mise à jour de la plateforme pour Windows Vista

la mise à jour de la plateforme pour Windows Vista prend en charge les extensions Windows 7 sur la plateforme des appareils mobiles Windowss (WPD).

### <a name="windows-vista-n-and-kn-versions-require-installation-of-windows-media-format-11-sdk"></a>Windows les Versions Vista N et KN requièrent l’Installation du kit de développement logiciel (SDK) Windows Media Format 11

-   pour les versions Windows 7 N et KN, vous devez installer le kit de développement logiciel (SDK) Windows Media Format 11 pour activer la fonctionnalité WPD.
-   pour les versions Windows Vista N et KN, vous devez installer le kit de développement logiciel (SDK) Windows Media Format 11 pour activer la fonctionnalité WPD.
