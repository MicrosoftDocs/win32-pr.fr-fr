# <a name="supported-operating-systems"></a>Systèmes d’exploitation pris en charge

Les appareils mobiles Windows sont pris en charge dans les systèmes d’exploitation Windows 7, Windows Vista et Windows XP. Votre application est compilée et liée à ces systèmes d’exploitation à l’aide des en-têtes et des bibliothèques fournis.
