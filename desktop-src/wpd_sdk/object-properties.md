# <a name="object-properties"></a>Propriétés des objets

Windows Les appareils mobiles prennent en charge les propriétés d’objet suivantes.



| Propriété                                                                                                                                    | VarType         | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
|---------------------------------------------------------------------------------------------------------------------------------------------|-----------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **\_ \_ références arrière des objets wpd \_**                                                                                                           | **VT \_ inconnu** | [**IPortableDevicePropVariantCollection**](iportabledevicepropvariantcollection.md) de type VT \_ LPWStr indiquant une liste de ObjectIDs. Lorsqu’un objet est référencé par un objet conteneur (par exemple, un objet audio référencé par une sélection), l’objet référencé utilise cette propriété pour faire référence à son objet conteneur. L’objet audio peut faire référence à l’objet playlist dans cet exemple.                                                                                                                   |
| **l' \_ objet wpd \_ peut \_ Supprimer**                                                                                                                | **VT \_ bool**    | Valeur booléenne qui spécifie si l’objet donné peut être supprimé.                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| **\_ID d' \_ \_ objet fonctionnel \_ du conteneur d’objets wpd \_**                                                                                          | **\_LPWStr VT**  | ID d’objet de l’objet fonctionnel le plus proche qui contient cet objet. Par exemple, pour un fichier situé dans un objet fonctionnel de stockage, cette propriété est définie sur l’ID de l’objet fonctionnel de stockage.                                                                                                                                                                                                                                                                                                                        |
| <span id="wpd_object_content_type"></span><span id="WPD_OBJECT_CONTENT_TYPE"></span>**TYPE de contenu de l' \_ objet wpd \_ \_**                          | **\_CLSID VT**   | **GUID** qui identifie le type générique de cet objet, par exemple, un document ou un message électronique. il peut s’agir d’un [type d’objet défini par Windows périphériques portables](requirements-for-objects.md)ou d’un type de contenu de pilote personnalisé. L’objet Device est le seul objet qui ne signale pas cette propriété.                                                                                                                                                                                                                        |
| <span id="wpd_object_date_authored"></span><span id="WPD_OBJECT_DATE_AUTHORED"></span>**Date de création de l' \_ objet wpd \_ \_**                       | **\_Date VT**    | Valeur qui spécifie la date et l’heure de création du contenu. Cela peut ne pas être identique à la date de création du fichier. Par exemple, un fichier de musique a une date de création du moment où la musique a été enregistrée, mais une date de création du fichier WMA créé sur l’appareil.                                                                                                                                                                                                                                |
| <span id="wpd_object_date_created"></span><span id="WPD_OBJECT_DATE_CREATED"></span>**Date de création de l' \_ objet wpd \_ \_**                          | **\_Date VT**    | Valeur qui spécifie la date et l’heure de création de l’objet sur l’appareil.                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| <span id="wpd_object_date_modified"></span><span id="WPD_OBJECT_DATE_MODIFIED"></span>**Date de modification de l' \_ objet wpd \_ \_**                       | **\_Date VT**    | Valeur qui spécifie la date et l’heure auxquelles l’objet a été modifié sur l’appareil.                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| <span id="wpd_object_format"></span><span id="WPD_OBJECT_FORMAT"></span>**\_format d’objet wpd \_**                                             | **\_CLSID VT**   | **GUID** identifiant le format des données de l’objet. il peut s’agir d’un [format défini par Windows appareils portables](object-format-guids.md) ou un format de pilote personnalisé.                                                                                                                                                                                                                                                                                                                                                            |
| **\_objet wpd \_ générer une \_ miniature \_ à partir de la \_ ressource**                                                                                        | **VT \_ bool**    | Valeur booléenne qui spécifie si l’image miniature de cet objet doit être créée à partir des données de ressource par défaut. Cela permet aux objets sans ressource miniature de fournir une expérience de navigation plus conviviale. L’utilisation de cet indicateur peut affecter la première réponse d’affichage, puisque l’application doit récupérer et calculer une image miniature à partir de l’appareil ; Il est plus efficace de fournir une ressource d’image miniature distincte, si possible.<br/>                                                  |
| **\_ \_ \_ \_ \_ nom complet de l’emplacement de l’indicateur d’objet wpd**                                                                                              | **\_LPWStr VT**  | Si l’objet donné apparaît comme un emplacement d’indication, cette propriété indique le nom spécifique à l’indicateur à afficher à la place du nom de l’objet. Les pilotes peuvent spécifier des indications d’emplacement pour différents types de contenu. Ils peuvent être considérés comme des raccourcis vers des objets de dossier de niveau supérieur qui contiennent des objets des types spécifiés. Les clients qui utilisent ces indicateurs d’emplacement peuvent afficher un nom pour ces raccourcis qui diffère du nom d’objet du dossier. Si cette propriété n’existe pas, le **\_ \_ nom de l’objet wpd** est généralement utilisé à la place. |
| <span id="wpd_object_id"></span><span id="WPD_OBJECT_ID"></span>**\_ID d’objet wpd \_**                                                         | **\_LPWStr VT**  | ID de chaîne qui identifie de façon unique l’objet sur l’appareil. Cet ID ne doit pas être stocké entre les sessions. Si cette propriété est à la fois unique et persistante, le pilote peut définir à la fois l' **\_ \_ \_ \_ ID unique persistant de l’objet wpd** et l' **\_ \_ ID d’objet wpd** sur la même valeur.<br/>                                                                                                                                                                                                                                                 |
| <span id="wpd_object_is_drm_protected"></span><span id="WPD_OBJECT_IS_DRM_PROTECTED"></span>**l' \_ objet \_ wpd \_ est \_ protégé par DRM**             | **VT \_ bool**    | Valeur booléenne qui spécifie si les données multimédia sont protégées par DRM. S’il n’est pas présent, il est supposé que la valeur est false.                                                                                                                                                                                                                                                                                                                                                                                                            |
| <span id="wpd_object_ishidden"></span><span id="WPD_OBJECT_ISHIDDEN"></span>**WPD, \_ objet \_ ISHIDDEN**                                       | **VT \_ bool**    | Valeur booléenne qui spécifie si l’objet doit être masqué. S’il n’est pas présent, l’objet est supposé ne pas être masqué.                                                                                                                                                                                                                                                                                                                                                                                                     |
| **l' \_ objet wpd \_ peut \_ Supprimer**                                                                                                                | **VT \_ bool**    | Valeur booléenne qui spécifie si l’objet peut être supprimé.                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| <span id="wpd_object_issystem"></span><span id="WPD_OBJECT_ISSYSTEM"></span>**WPD, \_ objet \_ ISSYSTEM**                                       | **VT \_ bool**    | Valeur booléenne qui spécifie si l’objet représente des données système (par exemple, un fichier système). S’il n’est pas présent, l’objet est supposé ne pas être un objet système.                                                                                                                                                                                                                                                                                                                                                              |
| <span id="wpd_object_keywords"></span><span id="WPD_OBJECT_KEYWORDS"></span>**\_Mots clés d’objet wpd \_**                                       | **\_LPWStr VT**  | Chaîne contenant une liste de mots clés délimités par des espaces associés à cet objet.                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| **\_paramètres régionaux de la langue de l’objet wpd \_ \_**                                                                                                           | **\_LPWStr VT**  | Chaîne qui dentifies le langage utilisé par un objet donné. Si plusieurs langues sont contenues dans cet objet, elle doit identifier la langue principale. Cette propriété peut contenir un code de langue, tel que défini dans ISO-639, tel que : « fr ». Il peut également contenir un code de pays langue, qui se compose d’un code de langue de deux ou trois caractères, comme défini dans la norme ISO-639, suivi d’un trait d’Union, puis d’un code de pays tel que défini dans ISO-3166, par exemple : « en-US ».                                |
| <span id="wpd_object_name"></span><span id="WPD_OBJECT_NAME"></span>**nom de l' \_ objet wpd \_**                                                   | **\_LPWStr VT**  | Nom complet de l’objet.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| <span id="wpd_object_non_consumable"></span><span id="WPD_OBJECT_NON_CONSUMABLE"></span>**\_objet wpd \_ non \_ utilisable**                    | **VT \_ bool**    | Valeur booléenne qui spécifie si cet objet est destiné à être compris ou simplement stocké par l’appareil. Si cette propriété n’est pas présente, toutes les données sont supposées être destinées à être consommées.                                                                                                                                                                                                                                                                                                                            |
| <span id="wpd_object_original_file_name"></span><span id="WPD_OBJECT_ORIGINAL_FILE_NAME"></span>**\_nom du \_ fichier d’origine de l’objet wpd \_ \_**       | **\_LPWStr VT**  | Nom de chaîne pour le fichier.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| <span id="wpd_object_parent_id"></span><span id="WPD_OBJECT_PARENT_ID"></span>**\_ \_ ID parent de l’objet wpd \_**                                   | **\_LPWStr VT**  | ID d’objet de l’objet parent. Le seul objet qui peut retourner une chaîne vide pour cette valeur est l’objet d’appareil racine. Pour modifier cette propriété, appelez [**IPortableDevice :: SendCommand**](/windows/desktop/api/PortableDeviceApi/nf-portabledeviceapi-iportabledevice-sendcommand) (**\_ commande wpd \_ Storage Storage \_ Move**).<br/>                                                                                                                                                                                                                                                    |
| <span id="wpd_object_persistent_unique_id"></span><span id="WPD_OBJECT_PERSISTENT_UNIQUE_ID"></span>**\_ \_ \_ ID unique persistant de l’objet wpd \_** | **\_LPWStr VT**  | ID de chaîne qui identifie de façon unique l’objet sur l’appareil, similaire à l' **\_ \_ ID d’objet wpd**, mais qui doit être stocké entre les sessions. Si l' **\_ \_ ID** d’objet wpd de l’identificateur d’objet est à la fois unique et persistant, le pilote peut définir à la fois l' **\_ \_ \_ \_ ID unique persistant de l’objet wpd** et l' **\_ \_ ID d’objet wpd** sur la même valeur.<br/>                                                                                                                                                                          |
| <span id="wpd_object_references"></span><span id="WPD_OBJECT_REFERENCES"></span>**\_références d’objets wpd \_**                                 | **VT \_ inconnu** | Un [**IPortableDevicePropVariantCollection**](iportabledevicepropvariantcollection.md) contenant une collection d' \_ ID d’objet de LPWStr VT identifiant les objets référencés. Cela est requis uniquement si l’objet est un objet de référence tel qu’un dossier ou une sélection.                                                                                                                                                                                                                                                            |
| <span id="wpd_object_size"></span><span id="WPD_OBJECT_SIZE"></span>**taille de l' \_ objet wpd \_**                                                   | **\_UI8 VT**     | Taille des données de ressource de l’objet.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| <span id="wpd_object_sync_id"></span><span id="WPD_OBJECT_SYNC_ID"></span>**ID de synchronisation de l' \_ objet wpd \_ \_**                                         | **\_LPWStr VT**  | Chaîne opaque créée par un client pour conserver l’état entre les sessions sans conserver le catalogue du contenu de l’appareil connecté.                                                                                                                                                                                                                                                                                                                                                                                                |
