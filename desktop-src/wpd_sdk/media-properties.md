# <a name="media-properties"></a>Propriétés du média

Appareils mobiles Windows prend en charge les propriétés de support suivantes.



| Propriété                                                                                                                 | VarType        | Description                                                                                                                                                                                                                                                                                                    |
|--------------------------------------------------------------------------------------------------------------------------|----------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **artiste de l' \_ album multimédia wpd \_ \_**                                                                                            | **\_LPWStr VT** | Chaîne qui identifie l’artiste qui effectue le média.                                                                                                                                                                                                                                                      |
| **\_profil d' \_ \_ encodage de contenu multimédia wpd \_**                                                                                 | \_LPWStr VT     | Chaîne qui spécifie un profil d’encodage. Les codecs multimédias peuvent être encodés conformément à un profil, qui définit un algorithme de codage ou un processus d’optimisation particulier.                                                                                                                                  |
| **\_type de \_ débit binaire \_ wpd**                                                                                            | **VT \_ UI4**    | Énumération de [**\_ \_ types de débits wpd**](wpd-bitrate-types.md) qui qualifie davantage la vitesse de transmission des données audio ou vidéo.                                                                                                                                                                                     |
| **\_achat de médias wpd \_ \_ maintenant**                                                                                                 | **VT \_ bool**   | Valeur booléenne qui spécifie si l’utilisateur souhaite acheter le contenu la prochaine fois qu’une connexion Internet est établie.                                                                                                                                                                                         |
| **\_signet d' \_ octet \_ multimédia wpd**                                                                                           | **\_UI8 VT**    | Spécifie un signet (offset d’octet de base zéro) qui correspond à la dernière position jouée ou affichée sur le média donné.                                                                                                                                                                                     |
| **\_compositeur de médias wpd \_**                                                                                                 | **\_LPWStr VT** | Compositeur, si le compositeur et l’intervenant sont différents.                                                                                                                                                                                                                                                     |
| **\_Copyright du support wpd \_**                                                                                                | **\_LPWStr VT** | Chaîne qui fournit des informations de Copyright à propos de l’objet.                                                                                                                                                                                                                                                 |
| <span id="wpd_media_description"></span><span id="WPD_MEDIA_DESCRIPTION"></span>**\_Description du support wpd \_**              | **\_LPWStr VT** | Description de l’objet donné. Lorsque la propriété du [ \_ sous- \_ \_ Description du média wpd](/windows) peut décrire un contenu unique, cette propriété décrit une collection de médias associés.<br/>                                                                                 |
| **\_URL de \_ destination du support wpd \_**                                                                                         | **\_LPWStr VT** | Chaîne explicite qui se termine par un caractère null et qui indique l’URL d’un objet. Cette URL identifie le contenu qui s’affichera si l’utilisateur clique sur l’objet. Les objets **Mediacast** utilisent souvent cette propriété pour spécifier l’emplacement des informations connexes.<br/>                                          |
| **\_durée du média wpd \_**                                                                                                 | **\_UI8 VT**    | Durée du contenu, en millisecondes.                                                                                                                                                                                                                                                                  |
| **\_ \_ évaluation efficace des médias wpd \_**                                                                                        | **VT \_ UI4**    | Nombre compris entre 1 et 100 inclus, qui évalue le contenu, où 1 est la valeur la plus basse. Cette valeur est basée sur des statistiques d’utilisation qui ne sont pas définies explicitement par l’utilisateur.                                                                                                                                   |
| **\_profil d' \_ encodage de média wpd \_**                                                                                        | **\_LPWStr VT** | Nom de profil, tel que défini par le codec particulier utilisé. Si le codec ne requiert pas de nom de profil spécifique, vous devez spécifier à la place des informations audio dans les propriétés appropriées.                                                                                                            |
| **\_genre de média wpd \_**                                                                                                    | **\_LPWStr VT** | Chaîne explicite qui se termine par un caractère null et qui indique le genre auquel appartient ce média.                                                                                                                                                                                                                     |
| **\_GUID du média wpd \_**                                                                                                     | **\_LPWStr VT** | Spécifie un identificateur global unique (GUID) 128 bits.                                                                                                                                                                                                                                                         |
| <span id="wpd_media_height"></span><span id="WPD_MEDIA_HEIGHT"></span>**\_hauteur du média wpd \_**                             | **VT \_ UI4**    | Valeur qui spécifie la hauteur de la ressource principale (en général, une vidéo ou une image), en pixels. Consultez [ \_ \_ largeur du média wpd](/windows).                                                                                                                                                                    |
| **\_Date de \_ dernière \_ génération du média \_ wpd**                                                                                        | **\_Date VT**   | Date qui spécifie la dernière heure de modification ou de modification d’une série d’un Mediacast.                                                                                                                                                                                                                                 |
| **\_heure du \_ dernier \_ accès au support \_ wpd**                                                                                     | **\_Date VT**   | Valeur qui spécifie la date et l’heure auxquelles le média a été accédé pour la dernière fois sur l’appareil.                                                                                                                                                                                                                       |
| **\_éditeur de \_ gestion du média wpd \_**                                                                                         | **\_LPWStr VT** | Chaîne explicite qui se termine par un caractère null et qui indique l’adresse de messagerie de l’éditeur de gestion du média. Les objets **Mediacast** utilisent souvent cette propriété pour spécifier l’éditeur de gestion d’un canal ou d’une série RSS.<br/>                                                                                 |
| **\_méta- \_ \_ genre de média wpd**                                                                                              | **VT \_ UI4**    | Énumération de [**\_ méta- \_ genres wpd**](wpd-meta-genres.md) qui décrit le type de média.                                                                                                                                                                                                                   |
| **signet de l' \_ objet multimédia wpd \_ \_**                                                                                         | **\_LPWStr VT** | Spécifie l' \_ \_ ID d’objet wpd pour le dernier objet affiché ou lu. Cette propriété s’applique aux sélections et mediacasts.<br/>                                                                                                                                                                               |
| **\_propriétaire du média wpd \_**                                                                                                    | **\_LPWStr VT** | Chaîne explicite qui se termine par un caractère null et qui indique l’adresse de messagerie du propriétaire du média. Les objets **Mediacast** utilisent souvent cette propriété pour spécifier le propriétaire d’un canal ou d’une série RSS.<br/>                                                                                                     |
| **\_ \_ classification parentale des médias wpd \_**                                                                                         | **\_LPWStr VT** | Chaîne explicite qui se termine par un caractère null et qui indique la classification parentale du fichier multimédia. Il s’agit généralement d’une évaluation d’MPAA (motion image Association of America).                                                                                                                                        |
| **\_Date de \_ publication du support wpd \_**                                                                                            | **\_Date VT**   | Date à laquelle le contenu a été publié à l’origine.                                                                                                                                                                                                                                                             |
| **\_taux d' \_ échantillonnage du média wpd \_**                                                                                             | **VT \_ UI4**    | Taux d’échantillonnage de l’encodage, en échantillons par seconde, pour le contenu.                                                                                                                                                                                                                                              |
| **\_nombre d' \_ omissions du support wpd \_**                                                                                              | **VT \_ UI4**    | Nombre total de fois où ce média a été mis en file d’attente pour la lecture, mais explicitement ignoré par l’utilisateur. Cela peut être utilisé pour calculer automatiquement une évaluation de l’utilisateur ou modifier les règles de mise à jour du contenu.                                                                                                                  |
| **URL de la \_ source du média wpd \_ \_**                                                                                              | **\_LPWStr VT** | Chaîne explicite qui se termine par un caractère null et qui indique l’URL source du média donné. Les objets **Mediacast** utilisent souvent cette propriété pour spécifier l’origine du contenu.<br/>                                                                                                                   |
| **évaluation de l' \_ étoile du média wpd \_ \_**                                                                                             | **VT \_ UI4**    | Nombre compris entre 0 et 5 qui évalue le contenu, où 0 est évalué, 1 est l’évaluation la plus basse et 5 l’évaluation la plus élevée. Cette valeur est entrée explicitement par l’utilisateur.                                                                                                                                        |
| <span id="wpd_media_sub_description"></span><span id="WPD_MEDIA_SUB_DESCRIPTION"></span>**\_sous- \_ \_ Description du média wpd** | **\_LPWStr VT** | Description subordonnée pour l’objet donné. Cette propriété est souvent utilisée pour fournir des informations supplémentaires sur le contenu. Lorsque la propriété de [ \_ \_ Description du média wpd](/windows) peut décrire une collection de supports associés, cette propriété décrira un seul élément de contenu.<br/> |
| <span id="wpd_media_sub_title"></span><span id="WPD_MEDIA_SUB_TITLE"></span>**\_sous- \_ \_ titre du média wpd**                   | **\_LPWStr VT** | Sous-titre de l’objet. Si l’objet est un titre d’épisode dans une série, il s’agit du titre de l’épisode et le [ \_ \_ titre du support wpd](/windows) est le nom de la série.                                                                                                                                 |
| **ID de contenu de l' \_ abonnement au support wpd \_ \_ \_**                                                                                | **\_LPWStr VT** | ID utilisé pour identifier une partie du contenu d’un service d’abonnement en ligne. En règle générale, il est défini par le client plutôt que par le pilote ou l’appareil.                                                                                                                                                    |
| **\_signet d' \_ heure du média wpd \_**                                                                                           | **\_UI8 VT**    | Spécifie un signet (en millisecondes) qui correspond à la dernière position jouée ou affichée sur le média donné.                                                                                                                                                                                              |
| **\_ \_ durée \_ de vie des \_ médias wpd**                                                                                           | **\_UI8 VT**    | Spécifie la durée (en minutes) avant la prochaine mise à jour du contenu.                                                                                                                                                                                                                                                 |
| <span id="wpd_media_title"></span><span id="WPD_MEDIA_TITLE"></span>**\_titre du support wpd \_**                                | **\_LPWStr VT** | Titre du contenu. Si l’objet est un épisode dans une série, il s’agit du titre de la série, et le [ \_ sous- \_ \_ titre du média wpd](/windows) est le nom de l’épisode.                                                                                                                               |
| **\_ \_ débit binaire total des médias wpd \_**                                                                                           | **VT \_ UI4**    | Vitesse de transmission de l’objet.                                                                                                                                                                                                                                                                                    |
| **\_nombre d' \_ utilisations du média wpd \_**                                                                                               | **VT \_ UI4**    | Nombre total de fois que ce média a été lu ou affiché sur l’appareil.                                                                                                                                                                                                                                  |
| **\_ \_ \_ évaluation efficace de l’utilisateur multimédia wpd \_**                                                                                  | **VT \_ UI4**    | Nombre compris entre 1 et 100 qui évalue ce contenu, 1 étant le plus bas. Cette valeur est définie explicitement par l’utilisateur. La valeur par défaut est 0, ce qui indique que cette valeur n’a pas été définie.                                                                                                                    |
| **\_webmaster multimédia \_ wpd**                                                                                                | **\_LPWStr VT** | Chaîne explicite qui se termine par un caractère null et qui indique l’adresse de messagerie de l’administrateur de média. Les objets **Mediacast** utilisent souvent cette propriété pour spécifier l’administrateur Web d’un canal ou d’une série RSS.<br/>                                                                                             |
| <span id="wpd_media_width"></span><span id="WPD_MEDIA_WIDTH"></span>**\_largeur du média wpd \_**                                | **VT \_ UI4**    | Valeur qui spécifie la largeur de la ressource principale (en général, une vidéo ou une image), en pixels. Consultez [ \_ \_ hauteur du média wpd](/windows).                                                                                                                                                                   |
| **\_artiste multimédia \_ wpd**                                                                                                   | **\_LPWStr VT** | Valeur qui identifie l’artiste multimédia.                                                                                                                                                                                                                                                                      |
