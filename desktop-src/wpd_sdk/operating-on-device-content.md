# <a name="operating-on-device-content"></a>Fonctionnement sur le contenu de l’appareil

L’API WPD prend en charge un certain nombre d’opérations sur les appareils. Ces opérations peuvent être notamment les suivantes :

-   Transfert de contenu d’un PC vers un appareil connecté.
-   Transfert de contenu d’un appareil connecté à un PC.
-   Déplacement du contenu sur un appareil connecté.
-   Suppression du contenu d’un appareil connecté.
-   Ajout d’une ressource à un objet sur un appareil connecté.

Ces opérations sont décrites en détail dans les sections suivantes.
