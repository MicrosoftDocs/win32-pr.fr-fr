# <a name="selecting-a-device-wpd-api"></a>Sélection d’un appareil (API WPD)

Une fois que votre application a énuméré les appareils connectés, vous pouvez donner à l’utilisateur la possibilité de sélectionner un appareil particulier et d’établir une connexion pour d’autres opérations. Dans l’exemple d’application, la `ChooseDevice` fonction dans le module DeviceEnumeration. cpp effectue ces tâches.

La `ChooseDevice` fonction utilise le code d’énumération d’appareil que nous avons déjà vu pour énumérer les appareils disponibles. (Pour plus d’informations, consultez la section [énumération des appareils](enumerating-devices.md) ).
