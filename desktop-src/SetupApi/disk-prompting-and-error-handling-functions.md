# <a name="disk-prompting-and-error-handling-functions"></a>Invite de disque et fonctions de gestion des erreurs

Les fonctions suivantes fournissent une invite de disque et une interface utilisateur de gestion des erreurs.



| Fonction                                         | Description                                                                            |
|--------------------------------------------------|----------------------------------------------------------------------------------------|
| [**SetupCopyError**](/windows/desktop/api/Setupapi/nf-setupapi-setupcopyerrora)         | Génère une boîte de dialogue qui informe l’utilisateur d’une erreur de copie.                          |
| [**SetupDeleteError**](/windows/desktop/api/Setupapi/nf-setupapi-setupdeleteerrora)     | Génère une boîte de dialogue qui informe l’utilisateur d’une erreur de suppression.                      |
| [**SetupPromptForDisk**](/windows/desktop/api/Setupapi/nf-setupapi-setuppromptfordiska) | Génère une boîte de dialogue qui invite l’utilisateur à entrer un disque multimédia ou un emplacement de fichier source. |
| [**SetupRenameError**](/windows/desktop/api/Setupapi/nf-setupapi-setuprenameerrora)     | Génère une boîte de dialogue qui informe l’utilisateur d’une erreur de changement de nom.                        |
