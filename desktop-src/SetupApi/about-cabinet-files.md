# <a name="about-cabinet-files"></a>À propos des fichiers CAB

Un fichier cab est un fichier unique, généralement avec le suffixe. CAB, qui stocke les fichiers compressés dans une bibliothèque de fichiers. Un fichier compressé peut être réparti sur plusieurs fichiers CAB. Pendant l’installation, l’application d’installation décompresse les fichiers stockés dans un fichier CAB et les copie sur le système de l’utilisateur.

Si des fichiers CAB sont utilisés, les fonctions d’installation utilisent le nom de l’armoire comme fichier de balises pour le disque source sur lequel il réside.
