# <a name="releasing-other-system-resources"></a>Libération d’autres ressources système

Si un fichier journal ou une liste source a été utilisé, les ressources doivent être libérées en appelant [**SetupFreeSourceList**](/windows/desktop/api/Setupapi/nf-setupapi-setupfreesourcelista) ou [**SetupTerminateFileLog**](/windows/desktop/api/Setupapi/nf-setupapi-setupterminatefilelog).
