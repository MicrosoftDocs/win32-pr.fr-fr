# <a name="setup-applications"></a>Configurer des applications

En règle générale, vous créez un *fichier INF* avant de créer une application d’installation. Un fichier INF est un fichier texte qui contient des informations utilisées par votre application d’installation au cours d’une installation. Les informations qui peuvent être stockées dans un fichier INF sont notamment INI et les modifications du Registre, les noms de fichiers et les emplacements des fichiers sources sur le média source.

Après avoir créé un fichier INF, vous écrivez le code source de votre application d’installation. Vous pouvez utiliser les fonctions d’installation de votre application d’installation pour accéder aux informations contenues dans le fichier INF et effectuer des opérations d’installation.

Les sections suivantes décrivent l’utilisation des fichiers INF et des fonctions de traitement de l’installation :

-   [À propos des fichiers INF](about-inf-files.md)
-   [Utilisation des fichiers INF](using-inf-files.md)
-   [Fonctions d’installation des fichiers INF](inf-file-setup-functions.md)

Pour plus d’informations sur l’utilisation des fonctions d’installation dans une application, consultez [création d’applications d’installation](creating-setup-applications.md).
