# <a name="mru-source-list"></a>Liste de sources MRU

L’API d’installation fournit des fonctions qui stockent les répertoires sources utilisés le plus récemment (MRU). Ces informations sont stockées sur le système de l’utilisateur et sont accessibles par les installations suivantes.

Les rubriques suivantes décrivent la liste de sources MRU et expliquent comment utiliser les fonctions de configuration pour les ouvrir, les modifier, les analyser et les fermer :

-   [À propos de la liste de sources MRU](about-the-mru-source-list.md)
-   [Utilisation de la liste de sources MRU](using-the-mru-source-list.md)
-   [Référence de liste de sources MRU](mru-source-list-reference.md)
