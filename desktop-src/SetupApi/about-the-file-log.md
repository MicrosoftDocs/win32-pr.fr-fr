# <a name="about-the-file-log"></a>À propos du journal des fichiers

Un journal de fichier enregistre des informations sur les fichiers copiés dans un système pendant une installation. Il peut être utilisé pour la réparation d’urgence ou comme un outil de diagnostic.

Le fichier utilisé pour l’installation des fichiers journaux peut être le journal système ou tout autre journal des fichiers que vous spécifiez.
