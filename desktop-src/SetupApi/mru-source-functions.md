# <a name="mru-source-functions"></a>Fonctions sources MRU

Les fonctions suivantes sont utilisées avec les listes de fichiers sources MRU.



| Fonction                                                                 | Description                                                                                       |
|--------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------|
| [**SetupAddToSourceList**](/windows/desktop/api/Setupapi/nf-setupapi-setupaddtosourcelista)                     | Ajoute une entrée à une liste source.                                                                   |
| [**SetupCancelTemporarySourceList**](/windows/desktop/api/Setupapi/nf-setupapi-setupcanceltemporarysourcelist) | Annule tout comportement de liste ou de non-navigation temporaire et rétablit le comportement de liste standard.        |
| [**SetupFreeSourceList**](/windows/desktop/api/Setupapi/nf-setupapi-setupfreesourcelista)                       | Libère les ressources allouées à une liste source.                                                       |
| [**SetupQuerySourceList**](/windows/desktop/api/Setupapi/nf-setupapi-setupquerysourcelista)                     | Interroge la liste actuelle des sources d’installation.                                                 |
| [**SetupRemoveFromSourceList**](/windows/desktop/api/Setupapi/nf-setupapi-setupremovefromsourcelista)           | Supprime une entrée d’une liste de sources d’installation.                                                |
| [**SetupSetSourceList**](/windows/desktop/api/Setupapi/nf-setupapi-setupsetsourcelista)                         | Définit la liste source d’installation sur la liste des fichiers utilisés par le système, la liste des utilisateurs récents ou une liste temporaire. |
