# <a name="disk-space-list"></a>Liste de Disk-Space

L’API d’installation fournit des fonctionnalités de liste d’espace disque. Une liste d’espace disque est une liste d’opérations de fichiers que vous pouvez interroger pour déterminer la quantité d’espace disque nécessaire pour ces opérations.

Les rubriques suivantes décrivent en détail les listes d’espace disque, comment elles calculent l’espace requis et comment les utiliser dans une application d’installation :

-   [À propos de la liste Disk-Space](about-the-disk-space-list.md)
-   [Utilisation de la liste Disk-Space](using-the-disk-space-list.md)
-   [Référence de liste d’espace disque](disk-space-list-reference.md)
