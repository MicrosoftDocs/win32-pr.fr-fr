# <a name="gathering-setup-information-from-the-user"></a>Collecte d’informations de configuration auprès de l’utilisateur

Il est fortement recommandé que votre application d’installation inclue une interface utilisateur qui acquiert toutes les informations nécessaires de la part de l’utilisateur. Cette fonctionnalité n’est pas fournie par les fonctions d’installation et doit être implémentée par votre application d’installation, généralement sous la forme d’un assistant d’installation.
