# <a name="file-log-functions"></a>Fonctions du journal des fichiers

Les fonctions suivantes sont utilisées avec les journaux de fichiers.



| Fonction                                                   | Description                                 |
|------------------------------------------------------------|---------------------------------------------|
| [**SetupInitializeFileLog**](/windows/desktop/api/Setupapi/nf-setupapi-setupinitializefileloga)   | Initialise un fichier journal à utiliser.             |
| [**SetupLogFile**](/windows/desktop/api/Setupapi/nf-setupapi-setuplogfilea)                       | Ajoute une entrée au fichier journal.              |
| [**SetupQueryFileLog**](/windows/desktop/api/Setupapi/nf-setupapi-setupqueryfileloga)             | Récupère des informations à partir d’un fichier journal.      |
| [**SetupRemoveFileLogEntry**](/windows/desktop/api/Setupapi/nf-setupapi-setupremovefilelogentrya) | Supprime une entrée d’un fichier journal.           |
| [**SetupTerminateFileLog**](/windows/desktop/api/Setupapi/nf-setupapi-setupterminatefilelog)     | Libère les ressources allouées à un fichier journal. |
