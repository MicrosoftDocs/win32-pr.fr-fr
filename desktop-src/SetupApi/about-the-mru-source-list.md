# <a name="about-the-mru-source-list"></a>À propos de la liste de sources MRU

La liste source des derniers fichiers utilisés (MRU) reste résidente sur le système de l’utilisateur. Les fonctions d’installation gèrent en interne la création de nouvelles listes sources et leur emplacement sur l’ordinateur de l’utilisateur. Les fonctions d’installation utilisent cette liste pour stocker des informations sur les chemins source utilisés dans les installations précédentes.

Vous pouvez utiliser ces informations pour inviter l’utilisateur à entrer un chemin source. Par exemple, vous pouvez créer une liste déroulante des connexions réseau utilisées comme chemins source dans les installations précédentes.

Selon vos autorisations, vous pouvez créer une liste d’utilisateurs, qui est spécifique à un utilisateur particulier ou une liste système, qui est la même pour tous les utilisateurs. Outre les listes des sources système et utilisateur, vous pouvez créer une liste source temporaire qui est ignorée lors de la fermeture de l’application d’installation.
