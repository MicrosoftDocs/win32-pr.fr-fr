# <a name="default-queue-callback-routine"></a>Routine de rappel de file d’attente par défaut

La routine de rappel par défaut, [**SetupDefaultQueueCallback**](/windows/desktop/api/Setupapi/nf-setupapi-setupdefaultqueuecallbacka), que vous pouvez utiliser pour traiter les notifications retournées par [**SetupCommitFileQueue**](/windows/desktop/api/Setupapi/nf-setupapi-setupcommitfilequeuea)est fournie avec les fonctions d’installation.

Les sections suivantes décrivent le format de la routine de rappel de file d’attente par défaut, l’utilisation de la routine de rappel de file d’attente par défaut avec [**SetupCommitFileQueue**](/windows/desktop/api/Setupapi/nf-setupapi-setupcommitfilequeuea)et la création d’une routine de rappel de filtre qui s’appuie sur les fonctionnalités fournies par la routine de rappel de file d’attente par défaut.

-   [À propos de la routine de rappel de file d’attente par défaut](about-the-default-queue-callback-routine.md)
-   [Utilisation de la routine de rappel de file d’attente par défaut](using-the-default-queue-callback-routine.md)
-   [Référence de routine de rappel de file d’attente par défaut](default-queue-callback-routine-reference.md)
