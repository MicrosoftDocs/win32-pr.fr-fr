# <a name="inf-file-structures"></a>Structures de fichier INF

Les structures suivantes sont utilisées avec les fichiers INF.



| Structure                                              | Description                        |
|--------------------------------------------------------|------------------------------------|
| [**INFCONTEXT**](/windows/desktop/api/Setupapi/ns-setupapi-infcontext)                   | Contexte positionnel dans un fichier INF. |
| [**\_informations sur le fichier INF SP \_**](/windows/desktop/api/Setupapi/ns-setupapi-sp_inf_information) | Informations sur le fichier INF.    |



 

> [!Note]  
> La structure INFCONTEXT est utilisée en interne par les fonctions d’installation et ne doit pas être modifiée ou référencée par les applications d’installation. Elle est incluse ici à titre d’information uniquement.
