# <a name="webdav-api-structures"></a>Structures de l’API WebDAV

Les structures suivantes sont utilisées dans l’API WebDAV.



| Structure                                                   | Description                                                                                                                                                                                         |
|-------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**\_ \_ objet blob d’authentification de rappel DAV \_**](/windows/desktop/api/Davclnt/ns-davclnt-dav_callback_auth_blob) | Cette structure est utilisée pour stocker un [*objet BLOB*](/windows/desktop/SecGloss/b-gly) d’authentification qui a été récupéré par la fonction de rappel [*DavAuthCallback*](/windows/desktop/api/Davclnt/nc-davclnt-pfndavauthcallback) . |
| [**\_authentification de rappel DAV \_ \_ UNP**](/windows/desktop/api/Davclnt/ns-davclnt-dav_callback_auth_unp)   | Cette structure est utilisée pour stocker les informations de nom d’utilisateur et de mot de passe qui ont été récupérées par la fonction de rappel [*DavAuthCallback*](/windows/desktop/api/Davclnt/nc-davclnt-pfndavauthcallback) .                                               |
| [**Réf. de \_ rappel DAV \_**](/windows/desktop/api/Davclnt/ns-davclnt-dav_callback_cred)            | La fonction de rappel [*DavAuthCallback*](/windows/desktop/api/Davclnt/nc-davclnt-pfndavauthcallback) utilise cette structure pour stocker les informations d’identification de l’utilisateur.                                                                               |
