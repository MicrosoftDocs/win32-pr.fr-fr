# <a name="d1231-unknown-dxgi-format-support"></a>D1231 : prise en charge du format DXGI inconnu

Un appel à IsDxgiFormatSupported a été effectué avec le \_ format dxgi \_ inconnu. Cette API retourne la valeur false, bien que le \_ format dxgi \_ inconnu puisse être valide en tant qu’entrée pour certaines API.






 

## <a name="possible-causes"></a>Causes possibles

Un appel à IsDxgiFormatSupported a été effectué avec le \_ format dxgi \_ inconnu.

## <a name="possible-fixes"></a>Corrections possibles

Déterminez si \_ le format dxgi \_ inconnu est une précision de mémoire tampon valide pour une API donnée en lisant la documentation de cette API au lieu d’essayer de la déterminer par programmation.
