# <a name="d1249-inaccessible-target"></a>D1249 : cible inaccessible

Une tentative de dessin vers une cible inaccessible a été détectée.






 

## <a name="possible-causes"></a>Causes possibles

Une tentative de dessin vers une cible inaccessible a été détectée. L’une des raisons pour lesquelles une cible peut être inaccessible est si elle a été créée à partir d’une texture partagée et que la texture n’a pas été acquise.

## <a name="possible-fixes"></a>Corrections possibles

Assurez-vous que toutes les cibles sont accessibles avant leur rendu avec Direct2D.
