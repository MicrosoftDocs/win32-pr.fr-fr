# <a name="d1165-read-and-discard-map-options-are-incompatible"></a>D1165 : les options de mappage lire et ignorer ne sont pas compatibles

\_ \_ Les options de mappage de d2d1 ne \_ sont pas compatibles avec les options de mappage de d2d1 \_ \_ \_ ignorent



|             |         |
|-------------|---------|
| Niveau d’erreur | Avertissement |



 

## <a name="possible-causes"></a>Causes possibles

Direct2D peut générer ce message si une application appelle ID2D1Bitmap1 :: Map avec les options de mappage D2D1 \_ \_ \_ Read et d2d1 options de \_ carte \_ \_ ignorées.

## <a name="possible-fixes"></a>Corrections possibles

Les options lecture et ignorer ne s’accordent pas les unes aux autres, car l’autorisation de l’accès en lecture aux données non définies a une utilité limitée pour une application. Pour obtenir \_ \_ une description des options de mappage, consultez Options de mappage de d2d1. Lors de l’appel de ID2D1Bitmap1 :: Map, déterminez le type d’accès dont l’application a besoin.
