# <a name="d1229-d1229-immersive-call-to-getdesktopdpi"></a>D1229 : D1229 : appel immersif à GetDesktopDpi

ID2D1Factory :: GetDesktopDpi a été appelé dans une application du Windows Store. Ce n’est pas recommandé. Au lieu de cela, DisplayProperties :: LogicalDpi doit être utilisé.






 

## <a name="possible-causes"></a>Causes possibles

ID2D1Factory :: GetDesktopDpi a été appelé dans une application du Windows Store.

## <a name="possible-fixes"></a>Corrections possibles

DisplayProperties :: LogicalDpi doit être utilisé pour obtenir la valeur PPP dans une application du Windows Store.
