# <a name="d1126-alpha-mode-unknown-not-allowed"></a>D1126 : mode Alpha inconnu non autorisé

\_ \_ Le mode Alpha d2d1 \_ inconnu n’est pas autorisé en tant que paramètre pour cette API.



|             |         |
|-------------|---------|
| Niveau d’erreur | Avertissement |



 

## <a name="possible-causes"></a>Causes possibles

Le \_ spécificateur de mode Alpha \_ inconnu est autorisé dans certains contextes, par exemple une description de format pour ID2D1RenderTarget :: CreateCompatibleRenderTarget. Toutefois, de la même façon que le \_ format dxgi \_ est inconnu, Direct2D exige généralement que les formats soient entièrement spécifiés.

## <a name="possible-fixes"></a>Corrections possibles

Lorsque vous choisissez un spécificateur de format, considérez le traitement de l’aire d’alpha pour utiliser un \_ mode Alpha autre que le \_ mode Alpha \_ inconnu. Utilisez D2D1 \_ alpha \_ mode \_ ignore si la surface n’utilise pas alpha. Consultez la section Notes en [**\_ \_ mode Alpha d2d1**](/windows/desktop/api/dcommon/ne-dcommon-d2d1_alpha_mode).
