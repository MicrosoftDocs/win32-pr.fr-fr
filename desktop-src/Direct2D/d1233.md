# <a name="d1233-begindraw-invalid-from-previous-error"></a>D1233 : BeginDraw non valide à partir de l’erreur précédente

BeginDraw a été appelé mais sera ignoré, car l’état du contexte de périphérique n’est pas valide.






 

## <a name="possible-causes"></a>Causes possibles

L’application appelée BeginDraw, mais le contexte de périphérique était dans un État non valide d’une opération précédente.

## <a name="possible-fixes"></a>Corrections possibles

Assurez-vous que l’application répond correctement aux erreurs retournées par Flush ou EndDraw, en recréant son appareil si nécessaire. Si une commande a causé un contexte de périphérique dans un État non valide, le Flush ou EndDraw correspondant retourne une erreur.
