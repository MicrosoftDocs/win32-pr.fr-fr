# <a name="id2d1devicecontextfillopacitymask-methods"></a>ID2D1DeviceContext :: FillOpacityMask, méthodes

Remplissage à l’aide du canal alpha de la bitmap du masque d’opacité fourni. L’opacité du pinceau sera modulée par le masque. Le mode d’anticrénelage de la cible de rendu doit être défini sur un alias.
