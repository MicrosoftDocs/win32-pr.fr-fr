# <a name="d1140-stroke-transform-type-uses-default-dpi"></a>D1140 : le type de transformation Stroke utilise la valeur par défaut PPP

Lors de l’élargissement d’un ID2D1Geometry avec un trait fixe ou de fin, Direct2D suppose une résolution de 96. Lors du rendu de la géométrie étendue, la largeur du trait peut changer.






 

## <a name="possible-causes"></a>Causes possibles

Lorsque Direct2D dessine une géométrie sur une cible, il élargit la géométrie. Toutefois, une application peut également construire des objets ID2D1Geometry sans cible et les élargir, à l’aide de méthodes de ID2D1Geometry telles que Widening, StrokeContainsPoint et GetWidenedBounds. Au cours de ces opérations, une application peut utiliser D2D1 \_ Stroke \_ transformation \_ type \_ maigre ou Fixed. Ces attributs considèrent généralement une résolution PPP, mais en l’absence de cible, aucune PPP n’est spécifiée. Ce message d’information avertit que Direct2D prendra en compte une valeur PPP par défaut de 96,0.

## <a name="possible-fixes"></a>Corrections possibles

Ce message est à titre d’information uniquement et n’indique pas nécessairement un problème avec l’application.
