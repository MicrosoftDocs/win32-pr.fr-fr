# <a name="d1203-copyfrombitmap-with-multisampled-source"></a>D1203 : CopyFromBitmap avec source multiéchantillonnée

CopyFromBitmap a été utilisé avec une image bitmap source à échantillonnage. L’appel sera toujours émis, mais le résultat copié ne sera peut-être pas représenté par le modèle d’échantillonnage multicritère attendu.






 

## <a name="possible-causes"></a>Causes possibles

L’application a géré une texture Direct3D multiéchantillonnée comme une bitmap Direct2D, et a appelé CopyFromBitmap sur la bitmap.

## <a name="possible-fixes"></a>Corrections possibles

Évitez l’interopérabilité des textures à échantillonnages en tant que bitmaps si la fonctionnalité CopyFromBitmap est importante. Le résultat copié ne peut pas représenter le modèle d’échantillonnage multicritère attendu.
