# <a name="d1198-copy-to-read-only-bitmap"></a>D1198 : copier dans Read-Only bitmap

Une tentative a été effectuée pour copier des données vers une image bitmap en lecture seule ou immuable.






 

## <a name="possible-causes"></a>Causes possibles

Une tentative a été effectuée pour copier des données vers une image bitmap en lecture seule ou immuable.

## <a name="possible-fixes"></a>Corrections possibles

Vérifiez que les bitmaps qui sont écrites dans ne sont pas créées comme immuables.
