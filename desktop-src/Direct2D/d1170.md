# <a name="d1170-invalid-flags-with-dynamic-texture-bitmap"></a>D1170 : indicateurs non valides avec une image bitmap de texture dynamique

Options de mappage de D2D1 \_ \_ \_ Write \| d2d1 \_ options de mappage \_ \_ ignore est requis lors du mappage de la bitmap créée à partir d’une texture dynamique.






 

## <a name="possible-causes"></a>Causes possibles

Direct2D peut générer ce message si une application appelle ID2D1Bitmap1 :: Map avec une texture dont la ressource Direct3D sous-jacente est attribuée avec \_ l’utilisation dynamique de d3d11 \_ , mais n’a pas spécifié à la fois les options de mappage d2d1 \_ \_ \_ Write et d2d1 \_ options de mappage \_ \_ ignorées.

## <a name="possible-fixes"></a>Corrections possibles

Si une application requiert l’accès à une image bitmap à l’aide de ID2D1Bitmap1 :: Map et que sa ressource Direct3D sous-jacente est attribuée avec \_ l’utilisation dynamique de d3d11 \_ , spécifiez les options de mappage d’d2d1 \_ \_ \_ Write et d2d1 \_ options de mappage \_ \_ ignorées. Pour obtenir \_ \_ une description des options de mappage, consultez Options de mappage de d2d1.
