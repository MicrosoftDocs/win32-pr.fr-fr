# <a name="d1132-dxgi-format-not-hwnd-or-dc-compatible"></a>D1132 : format DXGI non compatible HWND ou DC

Le format DXGI donné n’est pas valide pour cette méthode de création de cible de rendu.



|             |         |
|-------------|---------|
| Niveau d’erreur | Avertissement |



 

## <a name="possible-causes"></a>Causes possibles

La création de HWND ou DC rendertargets à l’aide de ID2D1Factory :: CreateHwndRenderTarget ou ID2D1Factory :: CreateDCRenderTarget introduit des restrictions supplémentaires sur le format de surface de la cible. En particulier, ces cibles doivent être de type DXGI \_ \_ R8G8B8A8 \_ UNORM ou dxgi \_ format \_ B8G8R8A8 \_ UNORM.

## <a name="possible-fixes"></a>Corrections possibles

Pour résoudre ce problème, vérifiez que le format spécifié pour les renderTarget HWND ou DC d’une application est l’un des formats mentionnés ci-dessus. Si l’application n’a pas nécessairement besoin de ces types de cibles, envisagez le rendu d’une texture à la place, ses options de format de surface pour la compatibilité comme cible peuvent être moins restrictives.
