# <a name="matrix5x4f-class"></a>Matrix5x4F, classe

La classe **Matrix5x4F** représente une matrice 5 par 4 et fournit des méthodes pratiques pour la création de matrices.

## <a name="members"></a>Membres

La classe **Matrix5x4F** hérite de [**d2d1 \_ Matrix \_ 5x4 \_ F**](/windows/desktop/api/dcommon/ns-dcommon-d2d_matrix_5x4_f). **Matrix5x4F** a également les types de membres suivants :

-   [Constructeurs](#constructors)

### <a name="constructors"></a>Constructeurs

La classe **Matrix5x4F** contient ces constructeurs.



| Constructeur                                                                                                                                                                                                                                                                                           | Description                                                                                                                        |
|:------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------|
| [**Matrix5x4F()**](matrix5x4f-matrix5x4f--.md)                                                                                                                                                                                                                                                       | Instancie une nouvelle instance d’une classe **Matrix5x4F** non initialisée.<br/>                                                   |
| [**Matrix5x4F (FLOAT, FLOAT, FLOAT, FLOAT, float, FLOAT, FLOAT, FLOAT, FLOAT, FLOAT, float, FLOAT, FLOAT, FLOAT, FLOAT, FLOAT, FLOAT, FLOAT, FLOAT, FLOAT) (FLOAT, float, FLOAT, FLOAT, FLOAT, FLOAT, FLOAT, FLOAT, FLOAT, FLOAT, float, float, FLOAT, FLOAT, FLOAT, FLOAT, FLOAT, FLOAT, FLOAT, FLOAT)**](matrix5x4f-matrix5x4f-floats-.md) | Instancie une nouvelle instance d’une classe **Matrix5x4F** qui est initialisée avec toutes les valeurs de la matrice à virgule flottante.<br/> |
