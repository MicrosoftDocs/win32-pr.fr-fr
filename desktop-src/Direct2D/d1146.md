# <a name="d1146-invalid-render-controls"></a>D1146 : contrôles de rendu non valides

Les contrôles de rendu spécifiés comportaient des paramètres non valides et seront ignorés.






 

## <a name="possible-causes"></a>Causes possibles

Ce message peut se produire pendant ID2D1DeviceContext :: SetRenderingControls si l’application spécifie une valeur de précision de la mémoire tampon non valide, ou une taille de vignette égale à zéro.

## <a name="possible-fixes"></a>Corrections possibles

Assurez-vous que l’application utilise une valeur de précision de la mémoire tampon valide. Consultez [**\_ \_ précision de la mémoire tampon d2d1**](/windows/desktop/api/D2d1_1/ne-d2d1_1-d2d1_buffer_precision). Assurez-vous que l’application spécifie une taille de vignette différente de zéro.
