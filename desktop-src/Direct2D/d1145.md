# <a name="d1145-print-control-dpi-incorrect"></a>D1145 : PPP du contrôle d’impression incorrect

Impression Direct2D : propriétés du contrôle d’impression \_ \_ . la valeur rasterDPI n’est pas valide.






 

## <a name="possible-causes"></a>Causes possibles

Lors de la création d’un contrôle d’impression, ce message indique que l’application a spécifié une résolution non valide ; en particulier, une résolution inférieure à 1,0. Si Direct2D ne considère pas qu’il s’agit d’une erreur, l’impression Direct2D se comportera comme si l’application avait spécifié une résolution de 150,0.

## <a name="possible-fixes"></a>Corrections possibles

Pour traiter ce message, utilisez une valeur PPP de 1,0 ou plus lors de la création d’un contrôle d’impression.
