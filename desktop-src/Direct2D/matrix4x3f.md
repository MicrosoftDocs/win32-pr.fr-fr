# <a name="matrix4x3f-class"></a>Matrix4x3F, classe

La classe **Matrix4x3F** représente une matrice 4 par 3 et fournit des méthodes pratiques pour la création de matrices.

## <a name="members"></a>Membres

La classe **Matrix4x3F** hérite de [**d2d1 \_ Matrix \_ 4X3 \_ F**](/windows/desktop/api/dcommon/ns-dcommon-d2d_matrix_4x3_f). **Matrix4x3F** a également les types de membres suivants :

-   [Constructeurs](#constructors)

### <a name="constructors"></a>Constructeurs

La classe **Matrix4x3F** contient ces constructeurs.



| Constructeur                                                                                                                                                                                           | Description                                                                                                                        |
|:------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------|
| [**Matrix4x3F()**](matrix4x3f-matrix4x3f--.md)                                                                                                                                                       | Instancie une nouvelle instance d’une classe **Matrix4x3F** non initialisée.<br/>                                                   |
| [**Matrix4x3F (FLOAT, FLOAT, FLOAT, FLOAT, FLOAT, FLOAT, FLOAT, FLOAT, FLOAT, FLOAT, FLOAT, FLOAT) (FLOAT, FLOAT, FLOAT, FLOAT, FLOAT, FLOAT, FLOAT, FLOAT, FLOAT, FLOAT, FLOAT, FLOAT)**](matrix4x3f-matrix4x3f-floats-.md) | Instancie une nouvelle instance d’une classe **Matrix4x3F** qui est initialisée avec toutes les valeurs de la matrice à virgule flottante.<br/> |
