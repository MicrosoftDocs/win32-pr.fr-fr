# <a name="d1239-source-transform-bitmap-invalid-call"></a>D1239 : appel de la bitmap de transformation source non valide

La bitmap transmise à ID2D1SourceTransform ::D RAW ne prend pas en charge cette méthode.






 

## <a name="possible-causes"></a>Causes possibles

L’application a appelé une méthode non prise en charge sur une image bitmap transmise à ID2D1SourceTransform ::D RAW.

## <a name="possible-fixes"></a>Corrections possibles

Pour mettre à jour ce type de bitmap, utilisez ID2D1Bitmap1 :: Map.
