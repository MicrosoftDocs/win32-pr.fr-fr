# <a name="d1144-effect-is-not-a-target"></a>D1144 : l’effet n’est pas une cible

SetTarget a échoué, car un effet a été passé en tant que cible.






 

## <a name="possible-causes"></a>Causes possibles

ID2D1DeviceContext :: SetTarget prend un \* paramètre ID2D1Image. Les images Direct2D peuvent être des bitmaps, des listes de commandes ou des effets. Direct2D considère uniquement les bitmaps et les listes de commandes comme des cibles valides.

## <a name="possible-fixes"></a>Corrections possibles

Modifiez l’application pour utiliser une image bitmap ou une liste de commandes comme cible du contexte de périphérique. Direct2D n’effectue pas l’initialisation des effets à l’aide de SetTarget. Au lieu de cela, pour choisir le contenu d’un effet, une application crée l’effet à l’aide de ID2D1DeviceContext :: CreateEffect et utilise un GUID d’effet correspondant.
