# <a name="d2d1_matrix_4x4_f"></a>\_Matrice d2d1 \_ 4x4 \_ F

Représente une matrice 4 par 4.


```C++
typedef D2D_MATRIX_4X4_F D2D1_MATRIX_4X4_F;
```



## <a name="remarks"></a>Notes

**D2d1 \_ MATRIX \_ 4x4** est un nouveau nom pour la [**structure \_ \_ x 4x4 \_ F de la matrice D2D**](/windows/desktop/api/dcommon/ns-dcommon-d2d_matrix_4x4_f) . Pour obtenir la liste des champs fournis par la matrice, consultez la **\_ matrice D2D \_ 4x4 \_ F**.
