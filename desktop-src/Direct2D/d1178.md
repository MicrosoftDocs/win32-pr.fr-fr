# <a name="d1178-custom-dash-style-requires-dash-array"></a>D1178 : un style de tiret personnalisé requiert un tableau de tirets

La création d’un style de trait avec un style de tiret \_ \_ personnalisé requiert un tableau de tirets.






 

## <a name="possible-causes"></a>Causes possibles

Lorsque l’application a tenté de créer un ID2D1StrokeStyle, elle a utilisé \_ l' \_ attribut personnalisé de style de tiret d2d1 \_ mais a fourni un tableau de tirets null.

## <a name="possible-fixes"></a>Corrections possibles

Si l’application requiert des tirets personnalisés, assurez-vous qu’un tableau de tirets valide est spécifié. Pour obtenir un exemple, consultez CreateStrokeStyle. Si un style de tiret intégré est acceptable, laissez le tableau de tirets comme NULL et utilisez un spécificateur tel que le style tiret D2D1 Dash \_ \_ \_ .
