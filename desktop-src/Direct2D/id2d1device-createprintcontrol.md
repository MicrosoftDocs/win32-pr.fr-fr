# <a name="id2d1devicecreateprintcontrol-methods"></a>ID2D1Device :: CreatePrintControl, méthodes

Crée un objet [**ID2D1PrintControl**](/windows/win32/api/d2d1_1/nn-d2d1_1-id2d1printcontrol) qui convertit les primitives [Direct2D](/windows/desktop/direct2d/direct2d-portal) stockées dans [**ID2D1CommandList**](/windows/win32/api/d2d1_1/nn-d2d1_1-id2d1commandlist) en une représentation de page fixe. Le sous-système d’impression consomme alors les primitives.
