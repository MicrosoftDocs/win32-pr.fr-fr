# <a name="d1183-createwicbitmaprendertarget-with-incompatible-feature-level"></a>D1183 : CreateWicBitmapRenderTarget avec un niveau de fonctionnalité incompatible

Il n’est pas valide de passer un niveau de fonctionnalité Direct2D autre que le \_ \_ niveau de fonctionnalité d2d1 \_ par défaut à CreateWicBitmapRenderTarget.






 

## <a name="possible-causes"></a>Causes possibles

Quand une application appelée ID2D1Factory :: CreateWicBitmapRenderTarget, le champ minLevel de l’argument des propriétés de la \_ cible de rendu d2d1 \_ \_ n’est pas d2d1 \_ \_ valeur par défaut du niveau de fonctionnalité \_ .

## <a name="possible-fixes"></a>Corrections possibles

Vérifiez que le niveau de fonctionnalité utilisé lors de la création d’une cible de rendu WIC-interopped est défini par défaut. L’interopérabilité avec WIC implique des fonctionnalités logicielles uniquement ; la sémantique au niveau des fonctionnalités est pertinente pour le comportement du matériel.
