## <a name="examples"></a><span data-ttu-id="e6a55-117">Exemples</span><span class="sxs-lookup"><span data-stu-id="e6a55-117">Examples</span></span>

<span data-ttu-id="e6a55-118">L’exemple de code suivant montre comment utiliser paver pour créer un ensemble de triangles enroulés dans le sens des aiguilles d’une montre qui couvre la géométrie.</span><span class="sxs-lookup"><span data-stu-id="e6a55-118">The following code example shows how to use Tessellate to create a set of clockwise-wound triangles that cover the geometry.</span></span>


```C++
 ID2D1GeometrySink *pGeometrySink = NULL;
 hr = pPathGeometry->Open(&pGeometrySink);
 if (SUCCEEDED(hr))
 {
     hr = pGeometry->Widen(
             strokeWidth,
             pIStrokeStyle,
             pWorldTransform,
             pGeometrySink
             );

     if (SUCCEEDED(hr))
     {
         hr = pGeometrySink->Close();
         if (SUCCEEDED(hr))
         {
             ID2D1Mesh *pMesh = NULL;
             hr = m_pRT->CreateMesh(&pMesh);
             if (SUCCEEDED(hr))
             {
                 ID2D1TessellationSink *pSink = NULL;
                 hr = pMesh->Open(&pSink);
                 if (SUCCEEDED(hr))
                 {
                     hr = pPathGeometry->Tessellate(
                             NULL, // world transform (already handled in Widen)
                             pSink
                             );
                     if (SUCCEEDED(hr))
                     {
                         hr = pSink->Close();
                         if (SUCCEEDED(hr))
                         {
                             SafeReplace(&m_pStrokeMesh, pMesh);
                         }
                     }
                     pSink->Release();
                 }
                 pMesh->Release();
             }
         }
     }
     pGeometrySink->Release();
 }
 pPathGeometry->Release();
```
