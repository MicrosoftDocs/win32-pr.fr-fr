# <a name="d1163-print-bitmap-too-big"></a>D1163 : imprimer l’image bitmap trop grande

La bitmap est trop grande pour l’impression, envisagez de la scinder en vignettes plus petites.






 

## <a name="possible-causes"></a>Causes possibles

L’impression Direct2D impose une limite sur les tailles des bitmaps imprimables. Cette limite est de 65535 unités dépendantes de l’appareil. Si l’application comprend une bitmap qui dépasse cette taille dans le contenu imprimable, Direct2D signale une erreur et la couche de débogage génère ce message.

## <a name="possible-fixes"></a>Corrections possibles

Vérifiez que les bitmaps utilisées dans les opérations d’impression ne dépassent pas la limite de taille de l’impression Direct2D. Envisagez de partitionner la bitmap en plusieurs mosaïques.
