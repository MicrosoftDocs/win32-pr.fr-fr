# <a name="d1199-changed-command-list-resource"></a>D1199 : ressource de liste de commandes modifiée

Une image dessinée dans une liste de commandes vient d’être modifiée. Cela peut entraîner des conséquences inattendues. La liste de commandes contient désormais l’image modifiée, plutôt que le contenu de l’image au moment où elle a été dessinée.






 

## <a name="possible-causes"></a>Causes possibles

L’application a ajouté une image à une liste de commandes et a ensuite modifié l’image.

## <a name="possible-fixes"></a>Corrections possibles

Vérifiez que le comportement résultant est le même que prévu. Dans ce cas, le comportement, où une liste de commandes est la cible, produit des résultats différents de ceux d’une image bitmap définie comme cible. Si le comportement n’est pas le même que prévu, l’application peut souhaiter cloner l’image avant de la mettre à jour.
