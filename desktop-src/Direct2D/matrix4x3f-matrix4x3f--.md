# <a name="matrix4x3fmatrix4x3f-constructor"></a>Matrix4x3F :: Matrix4x3F (), constructeur

Instancie une nouvelle instance d’une classe [**Matrix4x3F**](matrix4x3f.md) non initialisée.

## <a name="syntax"></a>Syntaxe


```C++
inline Matrix4x3F();
```



## <a name="parameters"></a>Paramètres

Ce constructeur n’a aucun paramètre.
