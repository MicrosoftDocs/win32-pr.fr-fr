# <a name="d1168-write-map-option-requires-cpu-writable-bitmap"></a>D1168 : l’option d’écriture de mappage requiert une image bitmap accessible en écriture sur le processeur

Les \_ \_ options de mappage d2d1 \_ sont uniquement valides sur une image bitmap inscriptible de l’UC






 

## <a name="possible-causes"></a>Causes possibles

Direct2D peut générer ce message si une application appelle ID2D1Bitmap1 :: Map avec l' \_ attribut Write des options de mappage d2d1 \_ \_ , mais que l’image bitmap n’est pas accessible en écriture sur l’UC.

## <a name="possible-fixes"></a>Corrections possibles

Si une application requiert un accès en écriture à l’aide de ID2D1Bitmap1 :: Map, vérifiez que la ressource Direct3D sous-jacente a l' \_ attribut d’écriture d’accès au processeur d3d11 \_ \_ .
