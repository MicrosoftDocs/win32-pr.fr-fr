# <a name="d1177-unsupported-wic-pixel-format"></a>D1177 : format de pixel WIC non pris en charge

Le format de pixel WIC fourni n’est pas pris en charge par Direct2D. En particulier, Direct2D ne prend pas en charge les bitmaps WIC qui utilisent l’alpha direct, sauf si la profondeur est 8bpp.






 

## <a name="possible-causes"></a>Causes possibles

L’application a tenté d’interopérer une ressource WIC dont le format n’est pas pris en charge par Direct2D. En règle générale, ces formats sont ceux avec l’alpha rectiligne, et dont la profondeur n’est pas de 8 bits par plan.

## <a name="possible-fixes"></a>Corrections possibles

Envisagez de créer la ressource WIC à l’aide d’un autre format pour l’interopérabilité avec Direct2D, ou de la convertir.
