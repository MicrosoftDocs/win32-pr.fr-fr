# <a name="d1202-copyfromrendertarget-with-multisampled-source"></a>D1202 : CopyfromRenderTarget avec source multiéchantillonnée

CopyFromRenderTarget a été utilisé avec un renderTarget à échantillonnage. L’appel sera toujours émis, mais le résultat copié ne sera peut-être pas représenté par le modèle d’échantillonnage multicritère attendu.






 

## <a name="possible-causes"></a>Causes possibles

L’application a géré une texture Direct3D multiéchantillonnée comme cible Direct2D, et a appelé CopyFromRenderTarget sur la cible

## <a name="possible-fixes"></a>Corrections possibles

Évitez l’interopérabilité des textures multiéchantillonnées en tant que cibles si la fonctionnalité CopyFromRenderTarget est importante. Le résultat copié ne peut pas représenter le modèle d’échantillonnage multicritère attendu.
