# <a name="d1171-redundant-call-to-map-bitmap"></a>D1171 : appel redondant à la bitmap de mappage

Une tentative a été effectuée pour mapper une bitmap déjà mappée.






 

## <a name="possible-causes"></a>Causes possibles

Direct2D peut générer ce message et signaler une erreur si une application tente de mapper une bitmap déjà mappée à l’aide de ID2D1Bitmap1 :: Map.

## <a name="possible-fixes"></a>Corrections possibles

De la même façon que ID2D1RenderTarget :: EndDraw doit suivre les appels à ID2D1RenderTarget :: BeginDraw, assurez-vous que ID2D1Bitmap1 :: DEMAPPAGE suit des appels à ID2D1Bitmap1 :: Map. Si une application utilise des rappels ou des chemins de code récursifs pour mapper des bitmaps, assurez-vous que son comportement est correct.
