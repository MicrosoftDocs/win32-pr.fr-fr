# <a name="d1254-operation-on-single-threaded-domain"></a>D1254 : opération sur un domaine à thread unique

Un appel à [**ID2D1Multithread :: Enter**](/windows/win32/api/d2d1_1/nf-d2d1_1-id2d1multithread-enter) ou [**ID2D1Multithread :: Leave**](/windows/win32/api/d2d1_1/nf-d2d1_1-id2d1multithread-leave) a été effectué sur un domaine de la fabrique à thread unique.

Cela peut se produire si l’application a créé par inadvertance la fabrique D2D avec le [**\_ type de fabrique \_ \_ \_ d2d1**](/windows/desktop/api/d2d1/ne-d2d1-d2d1_factory_type)multithread. Dans ce cas, l’appel sera ignoré.
