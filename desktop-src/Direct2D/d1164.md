# <a name="d1164-invalid-map-option"></a>D1164 : option de mappage non valide

Les \_ \_ options de mappage d2d1 \_ None ne sont pas un paramètre valide à mapper






 

## <a name="possible-causes"></a>Causes possibles

Direct2D peut générer ce message si une application appelle ID2D1Bitmap1 :: Map avec l' \_ option None des options de mappage de d2d1 \_ \_ .

## <a name="possible-fixes"></a>Corrections possibles

Lors de l’appel de ID2D1Bitmap1 :: Map, utilisez une option Map valide. Pour obtenir \_ \_ une description des options de mappage, consultez Options de mappage de d2d1. Déterminez si l’application doit avoir accès en lecture ou en écriture.
