# <a name="d1240-source-tranform-still-mapped"></a>D1240 : la source transformation est toujours mappée

La bitmap transmise à ID2D1SourceTransform ::D RAW n’a jamais été démappée.






 

## <a name="possible-causes"></a>Causes possibles

L’application appelée ID2D1Bitmap1 :: Map mais n’a pas appelé ID2D1Bitmap1 :: DEMAPPAGE sur une bitmap transmise à ID2D1SourceTransform ::D RAW.

## <a name="possible-fixes"></a>Corrections possibles

Assurez-vous que le mappage est appelé chaque fois que map est appelé.
