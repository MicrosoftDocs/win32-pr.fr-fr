# <a name="id2d1devicecontextcreatebitmapfromdxgisurface-methods"></a>ID2D1DeviceContext :: CreateBitmapFromDxgiSurface, méthodes

Crée une image bitmap à partir d’une surface DXGI qui peut être définie en tant que surface cible ou dont les informations de contexte de couleur supplémentaires sont spécifiées.
