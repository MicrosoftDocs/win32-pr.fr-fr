# <a name="d1232-32bpp-rgba-causes-conversion"></a>D1232:32bpp RVBA provoque la conversion

Envisagez d’utiliser BGR plutôt que RGB pour les formats 32bpp afin que l’effet BitmapSource puisse éviter les conversions de format supplémentaires.
