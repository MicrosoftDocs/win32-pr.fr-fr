# <a name="d1166-discard-map-option-requires-write-map-option"></a>D1166 : l’option ignorer le mappage nécessite l’option écrire un mappage

\_ \_ Les options de mappage d2d1 \_ ignorées doivent être combinées avec les \_ options de mappage d2d1 \_ \_ Write






 

## <a name="possible-causes"></a>Causes possibles

Direct2D peut générer ce message si une application appelle ID2D1Bitmap1 :: Map avec les \_ options de mappage de d2d1 \_ \_ ignorées, mais pas les \_ options de mappage d2d1 \_ \_ Write.

## <a name="possible-fixes"></a>Corrections possibles

L’option Discard requiert WRITE pour être une opération utile pour une application. Pour obtenir \_ \_ une description des options de mappage, consultez Options de mappage de d2d1. Lors de l’appel de ID2D1Bitmap1 :: Map, déterminez le type d’accès dont l’application a besoin.
