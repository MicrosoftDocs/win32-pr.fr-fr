# <a name="d1129-dxgi-format-incompatible-with-alpha-mode"></a>D1129 : format DXGI incompatible avec le mode Alpha

La combinaison du mode Alpha et du format DXGI fourni n’est pas compatible les unes avec les autres.





 

## <a name="possible-causes"></a>Causes possibles

Pour les opérations Direct2D qui utilisent un format de surface et le mode alpha, il existe des cas dans lesquels le format de surface et le mode Alpha ne sont pas d’accord. Par exemple, un format de surface tel que \_ le format dxgi \_ a8 \_ UNORM n’a pas de sens d’être utilisé avec l’option Ignorer alpha, car il effectue un rendu uniquement sur un canal alpha. En outre, l’alpha direct n’est pas pris en charge avec certains \_ formats dxgi, mais l’alpha prémultiplié est.

## <a name="possible-fixes"></a>Corrections possibles

Consultez [formats de pixel pris en charge et modes alpha](supported-pixel-formats-and-alpha-modes.md) pour obtenir des conseils sur les combinaisons de format prises en charge et celles qui ne le sont pas. Modifiez l’application pour qu’elle utilise une combinaison de formats prise en charge.
