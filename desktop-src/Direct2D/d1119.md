# <a name="d1119-bitmap-bound-as-target"></a>D1119 : bitmap lié en tant que cible

Une opération de dessin a échoué, car la bitmap cible et une bitmap source étaient identiques.






 

## <a name="possible-causes"></a>Causes possibles

Ce message est probablement dû au fait que l’application a tenté une opération de dessin dans laquelle la même bitmap a été utilisée pour la source et la cible. Sémantiquement, cela est possible pour les textures Direct3D interopérables avec la \_ ressource de nuanceur de liaison et la \_ cible de rendu de liaison \_ \_ .

## <a name="possible-fixes"></a>Corrections possibles

Assurez-vous que l’application n’essaie pas d’utiliser sa bitmap cible comme source d’une opération de dessin, y compris les effets. Dans certains cas, ce message peut indiquer que l’application a défini la cible du contexte de périphérique sur une image bitmap différente de celle prévue.
