# <a name="direct2d-helper-structures"></a>Structures d’assistance Direct2D

L’espace de noms D2D1 définit les structures suivantes.



| Nom                                            | Description                                                                                                   |
|-------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [**TypeTraits<Type>**](/windows/win32/api/d2d1_1helper/ns-d2d1_1helper-typetraits) | Contient des implémentations de **point**, **Size** et **Rect** qui stockent leurs données à l’aide du type spécifié. |
