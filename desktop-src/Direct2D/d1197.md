# <a name="d1197-wrong-d3d-device"></a>D1197 : appareil D3D incorrect

La surface DXGI transmise à cette API provient d’un appareil incorrect.





 

## <a name="possible-causes"></a>Causes possibles

La surface DXGI transmise à cette API provient d’un appareil incorrect.

## <a name="possible-fixes"></a>Corrections possibles

Vérifiez que toutes les surfaces DXGI ont le même appareil D3D que le contexte de périphérique D2D.
