# <a name="d1189-cpu-read-bitmap-option-was-used-incorrectly"></a>D1189 : l’option de bitmap Read de l’UC a été utilisée de manière incorrecte

\_Options de bitmap d2d1 la \_ \_ \_ lecture de l’UC doit être combinée avec les \_ options de bitmap d2d1 \_ \_ ne peuvent pas être \_ dessinées






 

## <a name="possible-causes"></a>Causes possibles

L’application a spécifié une combinaison incompatible des \_ indicateurs d’options de bitmap d2d1 \_ .

## <a name="possible-fixes"></a>Corrections possibles

Supprimez l’un des indicateurs afin de créer un jeu d’options de \_ bitmap d2d1 valide \_ .
