# <a name="d1153-closed-command-lists-cannot-be-set-as-targets"></a>D1153 : les listes de commandes fermées ne peuvent pas être définies en tant que cibles

Une tentative de définition d’une liste de commandes fermée sur une cible a été effectuée.






 

## <a name="possible-causes"></a>Causes possibles

Une fois qu’une liste de commandes a été fermée, une application ne doit pas tenter de la définir en tant que cible. Direct2D considère que cet appel à ID2D1DeviceContext :: SetTarget est une erreur, car le contenu d’une liste de commandes ne peut pas changer une fois qu’il a été fermé. La définition d’une liste de commandes en tant que cible indique l’intention de l’afficher.

## <a name="possible-fixes"></a>Corrections possibles

Pour traiter ce message, n’appelez pas ID2D1DeviceContext :: SetTarget avec une liste de commandes qui a été fermée. S’il est nécessaire d’utiliser une liste de commandes avec de nouveaux contenus, créez une liste de commandes.
