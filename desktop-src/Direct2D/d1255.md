# <a name="d1255-tile-opacity-metadata-conflict"></a>D1255 : conflit de métadonnées d’opacité de la vignette

Le rectangle opaque calculé pour un graphique d’effet chevauche une zone marquée comme transparente. Cela peut se produire en raison d’une utilisation incorrecte d’un effet Atlas ou d’une transformation d’ajustement de limites.
