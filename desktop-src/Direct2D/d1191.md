# <a name="d1191-gdi-compatible-bitmap-option-was-used-incorrectly"></a>D1191 : l’option bitmap compatible GDI a été utilisée de manière incorrecte

\_ \_ Les options de bitmap d2d1 \_ \_ compatibles GDI doivent être associées à la \_ cible des options de bitmap d2d1 \_ \_






 

## <a name="possible-causes"></a>Causes possibles

L’application a spécifié une combinaison incompatible des \_ indicateurs d’options de bitmap d2d1 \_ .

## <a name="possible-fixes"></a>Corrections possibles

Ajoutez \_ \_ les options de bitmap d2d1 \_ cible aux \_ indicateurs d’options de bitmap d2d1 \_ .
