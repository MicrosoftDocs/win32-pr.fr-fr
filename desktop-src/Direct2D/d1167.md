# <a name="d1167-read-map-option-requires-cpu-readable-bitmap"></a>D1167 : l’option lire le mappage requiert une image bitmap lisible par l’UC

Les \_ \_ options \_ de mappage de d2d1 sont uniquement valides sur une image bitmap lisible par le processeur






 

## <a name="possible-causes"></a>Causes possibles

Direct2D peut générer ce message si une application appelle ID2D1Bitmap1 :: Map avec l' \_ attribut de lecture des options de mappage d2d1 \_ \_ , mais que l’image bitmap n’est pas lisible par le processeur.

## <a name="possible-fixes"></a>Corrections possibles

Si une application requiert un accès en lecture à l’aide de ID2D1Bitmap1 :: Map, créez l’image bitmap à l’aide des options de bitmap D2D1, option de lecture de l' \_ \_ \_ UC \_ .
