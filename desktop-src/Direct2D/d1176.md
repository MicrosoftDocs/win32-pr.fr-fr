# <a name="d1176-negative-dpi"></a>D1176 : PPP négatif

La spécification d’une valeur PPP négative lors de la création d’un renderTarget est une erreur.






 

## <a name="possible-causes"></a>Causes possibles

Une application a tenté de spécifier une valeur PPP négative, via l’utilisation de ID2D1RenderTarget :: SetDpi ou en créant une cible avec une propriété PPP négative.

## <a name="possible-fixes"></a>Corrections possibles

Choisissez des valeurs non négatives valides pour les PPP lors de la création ou de la modification des cibles de rendu. Si une résolution par défaut est acceptable, évitez d’appeler SetDpi ou de spécifier une option DPI lors de la création d’une cible. La spécification d’une valeur PPP égale à 0 équivaut à utiliser une valeur PPP par défaut.
