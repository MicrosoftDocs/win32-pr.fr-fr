# <a name="d1230-unknown-buffer-precision-support"></a>D1230 : prise en charge de la précision de la mémoire tampon inconnue

Un appel à IsBufferPrecisionSupported a été effectué avec la précision de la \_ mémoire tampon d2d1 \_ \_ inconnue. Cette API retourne la valeur false, bien que la \_ précision de la mémoire tampon d2d1 \_ \_ Unknown puisse être valide en tant qu’entrée pour certaines API.






 

## <a name="possible-causes"></a>Causes possibles

Un appel à IsBufferPrecisionSupported a été effectué avec la précision de la \_ mémoire tampon d2d1 \_ \_ inconnue.

## <a name="possible-fixes"></a>Corrections possibles

Déterminez si \_ la précision de la mémoire tampon d2d1 \_ \_ inconnue est une précision de mémoire tampon valide pour une API donnée en lisant la documentation de cette API au lieu de tenter de la déterminer par programmation.
