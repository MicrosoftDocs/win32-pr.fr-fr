# <a name="direct2d-glossary"></a>Glossaire Direct2D

<dl> <dt>

<span id="direct2d.direct2d_glossary_a8_target"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_A8_TARGET"></span>**Cible a8**
</dt> <dd>

Une cible de rendu qui utilise un format de pixel représentant un canal alpha avec huit bits. Les cibles a8 sont utiles pour dessiner des géométries comme masques.

</dd> <dt>

<span id="direct2d.direct2d_glossary_aliasing"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_ALIASING"></span>**alias**
</dt> <dd>

Dans les graphiques informatiques, apparence en escalier des courbes ou des diagonales. L’alias est plus visible à des résolutions inférieures.

</dd> <dt>

<span id="direct2d.direct2d_glossary_antialiasing"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_ANTIALIASING"></span>**anticrénelage**
</dt> <dd>

Technique permettant de lisser l’apparence en escalier des lignes courbées ou diagonales. Les méthodes d’anticrénelage incluent les pixels environnants avec des nuances intermédiaires et la manipulation de la taille et de l’alignement horizontal des pixels.

</dd> <dt>

<span id="direct2d.direct2d_glossary_batch"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_BATCH"></span>**feuilles**
</dt> <dd>

Jeu de demandes ou de transactions qui ont été regroupées.

</dd> <dt>

<span id="direct2d.direct2d_glossary_bitmap"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_BITMAP"></span>**pixels**
</dt> <dd>

Représentation de caractères ou de graphiques par pixels individuels. Les pixels peuvent être disposés horizontalement, en lignes ou verticalement, dans des colonnes. Chaque pixel peut être représenté par un ou plusieurs bits.

</dd> <dt>

<span id="direct2d.direct2d_glossary_bits_per_pixel__bpp_"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_BITS_PER_PIXEL__BPP_"></span>**bits par pixel (BPP)**
</dt> <dd>

Nombre de bits utilisés pour stocker et afficher les données de couleur pour un seul pixel. Il s’agit de l’unité de mesure standard pour la profondeur de bit (également appelée profondeur de couleur). Les profondeurs courants sont les suivants : 8, 16 et 32.

</dd> <dt>

<span id="direct2d.direct2d_glossary_brush"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_BRUSH"></span>**pinceau**
</dt> <dd>

Ressource Direct2D qui peint un plan infini avec une couleur unie, un dégradé ou une bitmap. Un pinceau est généralement utilisé pour rayer et remplir une géométrie.

</dd> <dt>

<span id="direct2d.direct2d_glossary_cleartype"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_CLEARTYPE"></span>**ClearType**
</dt> <dd>

Technologie d’affichage des polices qui améliore considérablement la résolution de l’affichage des polices, de telle sorte que les lettres de l’écran de l’ordinateur s’affichent en douceur et non en escalier. ClearType améliore l’apparence du texte sur les moniteurs LCD qui ont une interface numérique, tels que les moniteurs d’ordinateurs portables et les écrans plats de haute qualité.

</dd> <dt>

<span id="direct2d.direct2d_glossary_dc"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_DC"></span>**MÉTAFICHIER**
</dt> <dd>

Consultez la définition de : contexte de périphérique.

</dd> <dt>

<span id="direct2d.direct2d_glossary_device_context__dc_"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_DEVICE_CONTEXT__DC_"></span>**contexte de périphérique (DC)**
</dt> <dd>

Contexte de périphérique GDI. Voir aussi : Graphics Device Interface.

</dd> <dt>

<span id="direct2d.direct2d_glossary_direct2d"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_DIRECT2D"></span>**Direct2D**
</dt> <dd>

Une API de rendu à accélération matérielle et en mode immédiat en mode 2 D.

</dd> <dt>

<span id="direct2d.direct2d_glossary_direct3d"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_DIRECT3D"></span>**Direct3D**
</dt> <dd>

La plateforme d’accélération matérielle et le runtime pour les graphiques 3D dans Windows.

</dd> <dt>

<span id="direct2d.direct2d_glossary_directwrite"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_DIRECTWRITE"></span>**DirectWrite**
</dt> <dd>

API de texte qui fournit une prise en charge du rendu du texte, de la gestion des polices et de la disposition du texte indépendante de tout système de rendu spécifique.

</dd> <dt>

<span id="direct2d.direct2d_glossary_directx_graphics_infrastructure__dxgi_"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_DIRECTX_GRAPHICS_INFRASTRUCTURE__DXGI_"></span>**Infrastructure DirectX Graphics (DXGI)**
</dt> <dd>

Infrastructure qui gère les tâches de bas niveau liées aux graphiques qui sont indépendantes du runtime Graphics DirectX. DXGI fournit une infrastructure commune pour les composants graphiques.

</dd> <dt>

<span id="direct2d.direct2d_glossary_dxgi"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_DXGI"></span>**DXGI**
</dt> <dd>

Consultez la définition pour : DirectX Graphics infrastructure (DXGI) (en anglais).

</dd> <dt>

<span id="direct2d.direct2d_glossary_figure"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_FIGURE"></span>**Configuration**
</dt> <dd>

Une forme ouverte ou fermée définie par un point de départ et une collection de segments.

</dd> <dt>

<span id="direct2d.direct2d_glossary_flatten"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_FLATTEN"></span>**écrasé**
</dt> <dd>

Pour générer une approximation polygonale d’une géométrie (potentiellement courbée).

</dd> <dt>

<span id="direct2d.direct2d_glossary_gdi"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_GDI"></span>**GDI**
</dt> <dd>

Consultez la définition de : Graphics Device Interface (GDI).

</dd> <dt>

<span id="direct2d.direct2d_glossary_geometric_mask"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_GEOMETRIC_MASK"></span>**masque géométrique**
</dt> <dd>

Élément ou découpage, défini par un objet ID2D1Geometry, qui masque une couche lorsqu’elle est dessinée par une cible de rendu. Un masque géométrique peut être un alias ou avoir des bords de sous-pixel.

</dd> <dt>

<span id="direct2d.direct2d_glossary_geometry_transform"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_GEOMETRY_TRANSFORM"></span>**transformation Geometry**
</dt> <dd>

Transformation appliquée à une géométrie avant qu’elle ne soit rayée ou remplie.

</dd> <dt>

<span id="direct2d.direct2d_glossary_glyph"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_GLYPH"></span>**variante**
</dt> <dd>

Représentation graphique d’un caractère, d’une partie d’un caractère ou d’une séquence de caractères.

</dd> <dt>

<span id="direct2d.direct2d_glossary_glyph_run"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_GLYPH_RUN"></span>**exécution de glyphes**
</dt> <dd>

Exécution continue de glyphes qui partagent un format commun.

</dd> <dt>

<span id="direct2d.direct2d_glossary_gradient_brush"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_GRADIENT_BRUSH"></span>**pinceau de dégradé**
</dt> <dd>

Pinceau qui peint une zone dans une progression progressive d’une couleur à une autre ou d’une nuance à une autre nuance de la même couleur.

</dd> <dt>

<span id="direct2d.direct2d_glossary_gradient_stop"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_GRADIENT_STOP"></span>**point de dégradé**
</dt> <dd>

Couleur spécifique définie pour un emplacement dans la zone de dégradé. Les points de dégradé sont utilisés pour définir la couleur à des emplacements spécifiques le long du tracé de dégradé.

</dd> <dt>

<span id="direct2d.direct2d_glossary_grahics_primitive"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_GRAHICS_PRIMITIVE"></span>**primitive Grahics**
</dt> <dd>

Dans les graphiques informatiques, forme telle qu’une ligne, un cercle, une courbe ou un polygone qu’une carte graphique peut dessiner, stocker et manipuler en tant qu’entité discrète.

</dd> <dt>

<span id="direct2d.direct2d_glossary_graphics_device_interface__gdi_"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_GRAPHICS_DEVICE_INTERFACE__GDI_"></span>**Graphics Device Interface (GDI)**
</dt> <dd>

API Graphics qui permet aux applications d’envoyer des commandes graphiques à un périphérique d’affichage ou d’impression, généralement sans tenir compte de ses spécifications ou de ses capacités. Un GDI Microsoft Win32 reçoit les demandes de rendu des applications et transmet ces demandes au GDI en mode noyau.

</dd> <dt>

<span id="direct2d.direct2d_glossary_handle_to_a_device_context__hdc_"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_HANDLE_TO_A_DEVICE_CONTEXT__HDC_"></span>**handle vers un contexte de périphérique (HDC)**
</dt> <dd>

Référence à un contexte de périphérique GDI.

</dd> <dt>

<span id="direct2d.direct2d_glossary_hdc"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_HDC"></span>**HDC**
</dt> <dd>

Consultez la définition de : handle vers un contexte de périphérique (Device Context).

</dd> <dt>

<span id="direct2d.direct2d_glossary_mesh"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_MESH"></span>**déjà**
</dt> <dd>

Collection de vertex qui définissent un ensemble de triangles qui constituent une forme.

</dd> <dt>

<span id="direct2d.direct2d_glossary_multi-sample_anti-aliasing__msaa_"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_MULTI-SAMPLE_ANTI-ALIASING__MSAA_"></span>**Anti-aliasing à plusieurs échantillons (MSAA)**
</dt> <dd>

Technique d’anticrénelage qui traite une scène entière.

</dd> <dt>

<span id="direct2d.direct2d_glossary_opacity_mask"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_OPACITY_MASK"></span>**masque d’opacité**
</dt> <dd>

Masque, décrit par un pinceau ou une bitmap, qui est appliqué à un autre objet pour rendre cet objet partiellement ou complètement transparent. Un masque d’opacité utilise des informations de canal alpha pour spécifier la façon dont les pixels source de l’objet sont fusionnés dans la cible de destination finale. Les parties transparentes du masque indiquent les zones où l’image sous-jacente est masquée, tandis que les parties opaques du masque indiquent où l’objet masqué est autorisé.

</dd> <dt>

<span id="direct2d.direct2d_glossary_per_primitive_anti-aliasing__ppaa_"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_PER_PRIMITIVE_ANTI-ALIASING__PPAA_"></span>**Par anticrénelage primitif (PPAA)**
</dt> <dd>

Technique d’anticrénelage appliquée à chaque primitive graphique plutôt qu’à une scène entière.

</dd> <dt>

<span id="direct2d.direct2d_glossary_radial_gradient_brush"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_RADIAL_GRADIENT_BRUSH"></span>**pinceau dégradé radial**
</dt> <dd>

Un dégradé où le point de départ est défini par un point focal et le point de terminaison est défini par un dégradé.

</dd> <dt>

<span id="direct2d.direct2d_glossary_render_target"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_RENDER_TARGET"></span>**cible de rendu**
</dt> <dd>

Ressource Direct2D qui émet des commandes de dessin. Il existe différents types de cibles de rendu, chaque rendu dans une destination différente. Par exemple, un ID2D1HwndRenderTarget est restitué dans une fenêtre et un ID2D1BitmapRenderTarget est rendu sur une image bitmap.

</dd> <dt>

<span id="direct2d.direct2d_glossary_segment"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_SEGMENT"></span>**partie**
</dt> <dd>

Partie d’un tracé géométrique, définie par les points de début et de fin.

</dd> <dt>

<span id="direct2d.direct2d_glossary_tessellate"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_TESSELLATE"></span>**paver**
</dt> <dd>

Pour scinder une forme en une collection de triangles.

</dd> <dt>

<span id="direct2d.direct2d_glossary_transform"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_TRANSFORM"></span>**Modifiez**
</dt> <dd>

Matrice qui représente un mappage linéaire dans plusieurs dimensions. Direct2D utilise une matrice 2 par 3, qui limite l’API aux déformations affines.

</dd> <dt>

<span id="direct2d.direct2d_glossary_translation"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_TRANSLATION"></span>**interprétation**
</dt> <dd>

Processus d’utilisation d’une matrice de transformation pour déplacer un objet.

</dd> <dt>

<span id="direct2d.direct2d_glossary_vertex"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_VERTEX"></span>**vertex**
</dt> <dd>

Le point le plus élevé d’une courbe, le point où se termine une courbe ou le point où deux segments se rencontrent dans un polygone ou un tracé de forme libre.

</dd> <dt>

<span id="direct2d.direct2d_glossary_vertex_shader"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_VERTEX_SHADER"></span>**nuanceur de sommets**
</dt> <dd>

Type de nuanceur programmable qui contient un ensemble d’instructions d’assembly qui sont exécutées sur le GPU par vertex et par pixel pour l’appel de dessin actuel. Un nuanceur de sommets décharge les calculs intensifs de l’UC au GPU.

</dd> <dt>

<span id="direct2d.direct2d_glossary_windows_imaging_component__wic_"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_WINDOWS_IMAGING_COMPONENT__WIC_"></span>**Composant WIC (Windows Imaging Component)**
</dt> <dd>

API qui permet aux applications d’afficher et de modifier n’importe quel format d’image pour lequel un CODEC compatible avec WIC est installé, et de (2) lire et écrire des métadonnées ou des fichiers image.

</dd> <dt>

<span id="direct2d.direct2d_glossary_xml_paper_specification__xps_"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_XML_PAPER_SPECIFICATION__XPS_"></span>**XPS (XML Paper Specification)**
</dt> <dd>

Format de document, décrit par la spécification Paper XML, qui peut être utilisé pour stocker des documents, pour les traiter pour l’impression et pour les imprimer.

</dd> <dt>

<span id="direct2d.direct2d_glossary_xps"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_XPS"></span>**MODÈLE**
</dt> <dd>

Consultez la définition de : XML Paper Specification (XPS).

</dd> </dl>
