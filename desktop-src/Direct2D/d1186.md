# <a name="d1186-dc-render-target-requires-binddc"></a>D1186 : la cible de rendu DC requiert BindDC

L’utilisation d’une cible de rendu DC n’est pas valide avant l’appel de sa méthode BindDC.






 

## <a name="possible-causes"></a>Causes possibles

L’application a créé un ID2D1DCRenderTarget et l’a utilisé pour appeler BeginDraw, mais n’a pas encore appelé sa méthode BindDC. Ou BindDC a entraîné une erreur.

## <a name="possible-fixes"></a>Corrections possibles

Avant d’utiliser une cible de rendu DC-interopped, les applications doivent appeler BindDC pour associer la cible à un contexte de périphérique GDI. La cible n’est pas considérée comme valide tant qu’un contexte de périphérique GDI n’a pas été attribué. Pour obtenir un exemple, consultez ID2D1DCRenderTarget :: BindDC. Si l’application appelle déjà BindDC, vérifiez que BindDC n’a pas renvoyé d’erreur.
