# <a name="d1117-bitmap-is-not-a-target"></a>D1117 : la bitmap n’est pas une cible

SetTarget a échoué, car la bitmap n’a pas l’option TARGET.






 

## <a name="possible-causes"></a>Causes possibles

Pour définir un ID2D1Bitmap en tant que cible, vous devez le créer à l’aide de \_ l' \_ indicateur de cible options de bitmap d2d1 \_ . Cet indicateur est disponible sous la forme d’un champ de D2D1 \_ bitmap \_ PROPERTIES1 utilisé avec ID2D1DeviceContext :: CreateBitmap.

## <a name="possible-fixes"></a>Corrections possibles

Assurez-vous que l’application crée sa bitmap cible à l’aide des \_ options de bitmap d2d1 \_ \_ cible. Si la bitmap cible est créée à l’aide de ID2D1RenderTarget :: CreateBitmap, modifiez-la pour utiliser ID2D1DeviceContext :: CreateBitmap. Lorsque l’indicateur cible est spécifié, Direct2D vérifie également que les attributs Direct3D de la texture sont compatibles avec son utilisation en tant que cible.
