# <a name="d1118-target-not-a-bitmap"></a>D1118 : la cible n’est pas une bitmap

Une opération de dessin a échoué, car la cible est NULL ou une liste de commandes.





 

## <a name="possible-causes"></a>Causes possibles

La cause la plus probable de ce message est que l’application appelée GetDC ou ReleaseDC alors que la cible de son contexte de périphérique était définie sur NULL ou sur une liste de commandes.

## <a name="possible-fixes"></a>Corrections possibles

Si une application utilise GetDC ou ReleaseDC exposée par IGdiInteropRenderTarget, assurez-vous que le ID2D1DeviceContext correspondant a un ensemble de bitmaps valide défini comme cible.
