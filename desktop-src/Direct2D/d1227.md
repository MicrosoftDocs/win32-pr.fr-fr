# <a name="d1227-print-rasterization-event"></a>D1227 : imprimer l’événement de rastérisation

Ce message est émis à partir de Direct2D Print. Elle indique que l’une des opérations suivantes a entraîné un besoin de pixellisation :

-   Utilisation d’une \_ couche d2d1 \_ options1 \_ initialiser \_ à partir de la \_ couche d’arrière-plan.
-   Utilisation de la copie SOURCE du mode de fusion primitif ou de tout autre mode de fusion que la SOURCE sur.
-   Utilisation de certains modes composites, tels que \_ la \_ copie de source délimitée, la sortie de destination, la source par- \_ \_ dessus, Xor, plus ou inversée.
-   Utilisation de [**DrawGdiMetafile**](id2d1devicecontext-drawgdimetafile-overload.md).
-   Utilisation d’une transformation 4x4 (ou perspective) dans un appel [**DrawBitmap**](id2d1devicecontext-drawbitmap-overload.md) .

Pour des performances optimales, ces opérations doivent être évitées si possible.
