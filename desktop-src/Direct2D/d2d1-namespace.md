# <a name="d2d1-namespace"></a>Espace de noms D2D1

L’espace de noms D2D1 fournit des classes, des fonctions et des structures d’assistance qui simplifient les tâches courantes, telles que la création d’une matrice d’identité ou la création d’une couleur.
