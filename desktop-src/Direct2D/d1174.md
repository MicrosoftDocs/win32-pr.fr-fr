# <a name="d1174-pushlayer-to-command-list-will-ignore-layer-object"></a>D1174 : PushLayer à la liste de commandes va ignorer l’objet de couche

Lors du push d’une couche (non NULL) à une liste de commandes, l’objet de couche sera ignoré.





 

## <a name="possible-causes"></a>Causes possibles

Quand une commande PushLayer utilise un objet ID2D1Layer et que la cible est une liste de commandes, Direct2D se comporte comme si aucune couche n’avait été spécifiée, et n’impose pas de les limites fixe sur la couche si la liste de commandes est rendue ultérieurement.

## <a name="possible-fixes"></a>Corrections possibles

Les applications qui anticipent les commandes de diffusion en continu vers une liste de commandes doivent envisager d’utiliser des objets de couche NULL. sinon, sachez que les paramètres de restriction de taille spécifiés dans l’objet ID2D1Layer ne seront pas honorés.
