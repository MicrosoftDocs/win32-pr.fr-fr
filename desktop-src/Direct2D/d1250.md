# <a name="d1250-inaccessible-source"></a>D1250 : source inaccessible

Une tentative de dessin à partir d’une image bitmap inaccessible a été détectée.






 

## <a name="possible-causes"></a>Causes possibles

Une tentative de dessin à partir d’une source inaccessible a été détectée. Une source peut être inacessible si elle a été créée à partir d’une texture partagée et que la texture n’a pas été acquise.

## <a name="possible-fixes"></a>Corrections possibles

Assurez-vous que toutes les bitmaps sont accessibles avant de les dessiner avec Direct2D.
