# <a name="d1152-command-list-closed-twice"></a>D1152 : liste de commandes fermée deux fois

Une tentative a été effectuée pour fermer une liste de commandes pour la deuxième fois.






 

## <a name="possible-causes"></a>Causes possibles

Après avoir rempli une liste de commandes, elle ne doit être fermée qu’une seule fois. Direct2D considère les appels suivants à ID2D1CommandList :: Close comme une erreur.

## <a name="possible-fixes"></a>Corrections possibles

Pour éviter ce message, assurez-vous qu’une liste de commandes n’est fermée qu’une seule fois. Si une application utilise des rappels ou des chemins de code récursifs pour interagir avec les listes de commandes, assurez-vous que son comportement est correct.
