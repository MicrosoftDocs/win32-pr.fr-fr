# <a name="d1175-dxgi-target-must-not-use-software-flag"></a>D1175 : la cible DXGI ne doit pas utiliser l’indicateur logiciel

Ce type de renderTarget doit être marqué comme valeur par défaut ou matériel.






 

## <a name="possible-causes"></a>Causes possibles

Une application appelée ID2D1Factory :: CreateDxgiSurfaceRenderTarget avec un \_ objet de propriétés de cible de rendu d2d1 dont le \_ \_ champ de type a la valeur d2d1 le \_ type de cible de rendu est \_ \_ \_ logiciel. Direct2D considère cette combinaison de spécificateurs comme étant incorrecte, car l’indicateur de logiciel est réservé pour les cibles créées à l’aide de CreateWicBitmapRenderTarget et CreateHwndRenderTarget.

## <a name="possible-fixes"></a>Corrections possibles

Lors de l’utilisation de CreateDxgiSurfaceRenderTarget, les applications doivent spécifier le \_ matériel du type de cible de rendu d2d1 \_ \_ \_ .
