# <a name="infiniterectu-function"></a>InfiniteRectU fonction)

Crée un rectangle dont l’angle supérieur gauche est défini sur (0U, 0U) et l’angle inférieur droit défini sur (infini, infini).

## <a name="syntax"></a>Syntaxe


```C++
D2D1_RECT_U InfiniteRectU(void);
```



## <a name="parameters"></a>Paramètres

Cette fonction n’a pas de paramètres.

## <a name="return-value"></a>Valeur retournée

Type : **[ **d2d1 \_ rect \_ U**](d2d1-rect-u.md)**

Nouveau rectangle.
