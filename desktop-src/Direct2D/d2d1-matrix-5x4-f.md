# <a name="d2d1_matrix_5x4_f"></a>D2D1 \_ matrice \_ 5x4 \_ F

Représente une matrice 5 par 4.


```C++
typedef D2D_MATRIX_5X4_F D2D1_MATRIX_5X4_F;
```



## <a name="remarks"></a>Notes

**D2d1 \_ MATRIX \_ 5x4** est un nouveau nom pour la [**structure \_ \_ 5x4 \_ F de la matrice D2D**](/windows/desktop/api/dcommon/ns-dcommon-d2d_matrix_5x4_f) . Pour obtenir la liste des champs fournis par la matrice, consultez la [**\_ matrice D2D \_ 5x4 \_ F**](/windows/desktop/api/dcommon/ns-dcommon-d2d_matrix_5x4_f).
