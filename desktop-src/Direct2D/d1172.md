# <a name="d1172-invalid-to-unmap-nonmapped-bitmap"></a>D1172 : non valide pour annuler le mappage d’une image bitmap non mappée

Une tentative a été effectuée pour annuler le mappage d’une bitmap qui n’est pas mappée.






 

## <a name="possible-causes"></a>Causes possibles

Direct2D peut générer ce message et signaler une erreur si une application tente de annuler le mappage d’une bitmap qui n’a pas encore été mappée à l’aide de ID2D1Bitmap1 :: Map.

## <a name="possible-fixes"></a>Corrections possibles

De la même façon que ID2D1RenderTarget :: EndDraw doit suivre les appels à ID2D1RenderTarget :: BeginDraw, assurez-vous que ID2D1Bitmap1 :: DEMAPPAGE suit des appels à ID2D1Bitmap1 :: Map. N’appelez pas ID2D1Bitmap1 :: DEMAPPAGE plusieurs fois pour annuler le mappage d’une image bitmap.
