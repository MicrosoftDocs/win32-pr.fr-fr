# <a name="d1173-texture-cube-interop-unsupported"></a>D1173 : interopérabilité de cube de texture non prise en charge

Les textures marquées comme D3D11RESOURCE \_ misc \_ TEXTURECUBE ne peuvent pas être utilisées par Direct2D.






 

## <a name="possible-causes"></a>Causes possibles

L’application a tenté d’interopérabilité une texture Direct3D avec \_ l' \_ option D3D11RESOURCE misc TEXTURECUBE spécifiée.

## <a name="possible-fixes"></a>Corrections possibles

S’abstenir des textures d’interopérabilité qui ont le spécificateur TEXTURECUBE. Au lieu de cela, réorganisez la conception de rendu de l’application ou envisagez l’interopérabilité d’un intermédiaire.
