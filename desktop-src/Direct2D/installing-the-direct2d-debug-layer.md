# <a name="installing-the-direct2d-debug-layer"></a>Installation de la couche de débogage Direct2D

À compter de Windows 8, la couche de débogage Direct2D est incluse avec [Microsoft Visual Studio Express 2012 pour Windows 8](https://www.microsoft.com/visualstudio/11/downloads).

La couche de débogage Direct2D pour Windows 7 n’est plus disponible au téléchargement.
