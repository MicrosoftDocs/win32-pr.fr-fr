# <a name="d1184-createwicbitmaprendertarget-incompatible-with-hw-target-type"></a>D1184 : CreateWicBitmapRenderTarget incompatible avec le type de cible HW

Il n’est pas valide de passer le \_ type de cible de rendu de l’indicateur d2d1 \_ \_ \_ matériel à CreateWicBitmapRenderTarget.






 

## <a name="possible-causes"></a>Causes possibles

Quand une application appelée ID2D1Factory :: CreateWicBitmapRenderTarget, le champ de type de l' \_ argument de propriétés de la cible de rendu d2d1 \_ \_ était d2d1 \_ d2d1 type de cible de \_ rendu \_ \_ \_ matériel.

## <a name="possible-fixes"></a>Corrections possibles

Vérifiez que le type de cible utilisé lors de la création d’une cible de rendu WIC-interopped est logiciel ou par défaut. Direct2D réserve généralement une sémantique matérielle pour décrire l’interopérabilité de Direct3D.
