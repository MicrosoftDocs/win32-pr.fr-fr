# <a name="d1196-a8-shared-bitmap"></a>D1196 : bitmap partagée a8

Une surface DXGI de format DXGI \_ \_ a8 \_ UNORM n’est pas prise en charge dans cette API.





 

## <a name="possible-causes"></a>Causes possibles

\_Le format dxgi \_ a8 \_ UNORM n’est pas pris en charge en tant que format bitmap Interop dxgi, même s’il est pris en charge dans d’autres API.

## <a name="possible-fixes"></a>Corrections possibles

Créer une image bitmap DXGI Interop d’un format différent.
