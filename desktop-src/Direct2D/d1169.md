# <a name="d1169-discard-map-option-requires-dynamic-texture"></a>D1169 : l’option Ignorer la carte nécessite une texture dynamique

\_Les options de mappage d2d1 \_ \_ ignore sont valides uniquement avec une bitmap créée à partir d’une texture dynamique.






 

## <a name="possible-causes"></a>Causes possibles

Direct2D peut générer ce message si une application appelle ID2D1Bitmap1 :: Map avec l' \_ attribut d2d1 map \_ options \_ ignore, mais que la ressource Direct3D sous-jacente de la bitmap n’est pas attribuée avec l' \_ utilisation dynamique de d3d11 \_ .

## <a name="possible-fixes"></a>Corrections possibles

Si une application requiert l’accès à une image bitmap à l’aide de ID2D1Bitmap1 :: Map, vérifiez que la ressource Direct3D sous-jacente a l' \_ attribut dynamique d’utilisation d3d11 \_ .
