# <a name="d1247-dynamic-copy-from-memory"></a>D1247 : copie dynamique à partir de la mémoire

CopyFromMemory ne peut pas copier la mémoire dans une image bitmap créée à partir d’une texture dynamique.





 

## <a name="possible-causes"></a>Causes possibles

CopyFromMemory a été appelé sur une bitmap qui a été créée en tant qu' \_ utilisation dynamique de d3d11 \_ .

## <a name="possible-fixes"></a>Corrections possibles

Utilisez Map et DEMAPPAGE à la place pour ce type de bitmap.
