# <a name="direct2d-helper-classes"></a>Classes d’assistance Direct2D

L’espace de noms D2D1 définit les classes suivantes.

## <a name="in-this-section"></a>Contenu de cette section



| Rubrique                                       | Description                                                                                                                              |
|---------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------|
| [**ColorF**](/windows/win32/api/d2d1helper/nl-d2d1helper-colorf)<br/>         | Définit un ensemble de constantes qui représentent des couleurs connues et fournit des méthodes pratiques pour l’instanciation de différentes couleurs.<br/>  |
| [**Matrix3x2F**](/windows/win32/api/d2d1helper/nl-d2d1helper-matrix3x2f)<br/> | La classe [**Matrix3x2F**](/windows/win32/api/d2d1helper/nl-d2d1helper-matrix3x2f) représente une matrice 3 par 2 et fournit des méthodes pratiques pour la création de matrices.<br/> |
| [**Matrix4x3F**](matrix4x3f.md)<br/> | La classe [**Matrix4x3F**](matrix4x3f.md) représente une matrice 4 par 3 et fournit des méthodes pratiques pour la création de matrices.<br/> |
| [**Matrix4x4F**](/windows/desktop/api/d2d1_1helper/nl-d2d1_1helper-matrix4x4f)<br/> | La classe [**Matrix4x4F**](/windows/desktop/api/d2d1_1helper/nl-d2d1_1helper-matrix4x4f) représente une matrice 4 par 4 et fournit des méthodes pratiques pour la création de matrices.<br/> |
| [**Matrix5x4F**](matrix5x4f.md)<br/> | La classe [**Matrix5x4F**](matrix5x4f.md) représente une matrice 5 par 4 et fournit des méthodes pratiques pour la création de matrices.<br/> |
