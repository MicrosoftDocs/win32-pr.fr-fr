# <a name="d1139-unused-stroke-width"></a>D1139 : largeur du trait inutilisé

Un type de transformation de trait maigre a été spécifié, mais la largeur de trait fournie n’est pas 1. La largeur du trait sera ignorée.






 

## <a name="possible-causes"></a>Causes possibles

L’application a dessiné une primitive à l’aide d’un ID2D1StrokeStyle avec \_ \_ le type de transformation d2d1 Stroke \_ \_ fine, mais la largeur du trait utilisée dans cet appel de dessin n’était pas 1. Lorsque l’attribut maigre est spécifié, il remplace en fait la largeur du trait, en le fixant à 1,0 unités dépendantes de l’appareil, quelle que soit la transformation d’échelle ou le choix en DPI.

## <a name="possible-fixes"></a>Corrections possibles

Lorsque l’attribut maigre est défini sur un ID2D1StrokeStyle, l’application doit utiliser une largeur de trait de 1,0 unités.
