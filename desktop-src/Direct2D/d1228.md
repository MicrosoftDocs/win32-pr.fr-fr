# <a name="d1228-not-original-target"></a>D1228 : cible non originale

Un appel à ID2D1HwndRenderTarget :: Resize ou ID2D1DCRenderTarget :: BindDC a été effectué alors qu’une cible autre que l’original était définie sur le contexte de périphérique.
