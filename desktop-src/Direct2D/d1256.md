# <a name="d1256-invalid-analysis-effect-usage"></a>D1256 : utilisation d’un effet d’analyse non valide

Un effet d’analyse a été utilisé de façon incorrecte. Cela peut se produire lors de l’utilisation d’un effet d’analyse comme entrée d’un autre effet ou d’un pinceau d’image, de son dessin dans une liste de commandes ou de la mise en cache de sa sortie.
