# <a name="d1160-cycle-in-graph"></a>D1160 : cycle dans le graphique

Une tentative a été effectuée pour dessiner un graphique d’image avec un cycle dans celui-ci.






 

## <a name="possible-causes"></a>Causes possibles

Direct2D permet aux applications de construire des graphiques d’effet pour une plus grande flexibilité en matière de comportement. Toutefois, les applications ne doivent pas créer un graphique d’effet qui contient un cycle. Prenons l’exemple d’un graphique des effets contenant les nœuds A, B et C : il n’est pas possible de lier un à B, puis B à C, puis C à un. Direct2D identifie les graphiques d’effets cycliques et signale une erreur. la couche de débogage génère ce message.

## <a name="possible-fixes"></a>Corrections possibles

Pour éviter ce message, évitez de créer des graphiques d’effet qui ont un cycle. Créer des graphiques d’effet de manière efficace, y compris au moins un nœud qui agit en sortie uniquement.
