# <a name="d1162-fillmesh-requires-aliased-rendering"></a>D1162 : FillMesh requiert un rendu avec alias

FillMesh requiert que le mode anticrénelage soit défini sur le \_ mode anticrénelage d2d1 \_ \_ .






 

## <a name="possible-causes"></a>Causes possibles

Quand une application appelle ID2D1RenderTarget :: FillMesh, le mode antialias de la renderTarget impliquée doit être D2D1 \_ \_ en mode anticrénelage \_ . Le comportement par défaut d’une cible de rendu consiste à utiliser l’anticrénelage. il est donc nécessaire d’appeler ID2D1RenderTarget :: SetAntialiasMode avant de remplir les mailles.

## <a name="possible-fixes"></a>Corrections possibles

Utilisez ID2D1RenderTarget :: SetAntialiasMode pour vous assurer qu’un renderTarget utilise un rendu avec alias lors du remplissage des maillages. Si les opérations suivantes nécessitent un anticrénelage, redéfinissez le mode d’anticrénelage de la cible sur D2D1 mode d’anticrénelage \_ \_ \_ par \_ primitive. Assurez-vous que les appels à ID2D1RenderTarget :: SetAntialiasMode et ID2D1RenderTarget :: FillMesh sont émis sur la même cible.
