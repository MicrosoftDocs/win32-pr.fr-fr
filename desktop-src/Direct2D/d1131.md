# <a name="d1131-dxgi-format-not-gdi-compatible"></a>D1131 : format DXGI non compatible GDI

Une option compatible GDI a été passée à cette API, mais le format DXGI n’est pas compatible GDI.






 

## <a name="possible-causes"></a>Causes possibles

L’utilisation de l' \_ \_ indicateur compatible GDI des options de bitmap d2d1 \_ \_ pendant la création de bitmaps permet la compatibilité de la bitmap avec le rendu GDI, mais introduit des restrictions supplémentaires sur le format de surface de la bitmap. En général, le rendu GDI est compatible avec \_ les \_ formats dxgi B8G8R8A8 \_ UNORM et dxgi \_ format \_ B8G8R8A8 \_ \_ sRGB.

## <a name="possible-fixes"></a>Corrections possibles

Pour résoudre ce problème, vérifiez que le format spécifié pour une bitmap compatible GDI est un format BGRA à quatre canaux comme indiqué ci-dessus. Si la bitmap ne requiert pas l’interopérabilité avec GDI, omettez le \_ spécificateur compatible GDI.
