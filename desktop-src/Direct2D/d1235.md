# <a name="d1235-target-rendered-to-from-multiple-dcs"></a>D1235 : cible rendue à partir de plusieurs contrôleurs de contrôleur

Une tentative a été effectuée pour effectuer un rendu sur une image alors qu’elle était toujours rendue par un contexte de périphérique précédent. EndDraw doit être appelé sur ce contexte de périphérique avant d’être rendu avec le contexte de périphérique actuel. Lorsque la couche de débogage est utilisée, l’adresse de ce contexte de périphérique est également générée.
