# <a name="using-event-logging"></a>Utilisation de la journalisation des événements

Les applications peuvent utiliser l’API de journalisation des événements pour signaler et afficher les événements. Pour plus d’informations sur les événements de création de rapports, consultez [rapports d’événements](reporting-an-event.md).

Pour afficher les événements qui ont été écrits dans des fichiers journaux, consultez [interrogation des messages de la source d’événements](querying-for-event-source-messages.md) et réception d’une [notification d’événement](receiving-event-notification.md). Vous pouvez également utiliser le observateur d’événements pour afficher les événements.
