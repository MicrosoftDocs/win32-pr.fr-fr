# <a name="removing-a-component-from-a-com-application"></a>Suppression d’un composant d’une application COM+

Vous pouvez utiliser l’outil d’administration Services de composants pour supprimer des composants d’une application COM+.

> [!Note]  
> Soyez attentif lors de la suppression et de la réinstallation de composants individuels à partir d’une application COM+. Ce processus nécessite une connaissance détaillée des classes du composant, notamment ses attributs, les configurations de sécurité sur les interfaces et les méthodes, ainsi que d’autres paramètres.

 

**Pour supprimer un composant d’une application COM+**

1.  Dans l’arborescence de la console de l’outil d’administration Services de composants, sélectionnez l’application qui contient le composant que vous souhaitez supprimer.

2.  Ouvrez le dossier des **composants** de l’application.

3.  Sélectionnez le composant que vous souhaitez supprimer.

4.  Dans le menu **Action**, cliquez sur **Supprimer**. Vous pouvez également sélectionner le composant, cliquer avec le bouton droit, puis cliquer sur **supprimer**.

5.  Cliquez sur **Oui** lorsque vous êtes invité à confirmer votre action.
