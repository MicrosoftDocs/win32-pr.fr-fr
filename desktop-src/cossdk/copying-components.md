# <a name="copying-components"></a>Copier des composants

Vous pouvez copier un composant d’une application COM+ vers une autre. Une fois que vous avez copié un composant dans une autre application COM+, vous pouvez configurer ce composant différemment du composant d’origine.

**Pour copier un composant d’une application COM+ vers une autre**

1.  Dans le volet d’informations de l’outil d’administration Services de composants, cliquez avec le bouton droit sur le composant que vous souhaitez copier, puis cliquez sur **copier**.

2.  Dans la boîte de dialogue **copier le composant** , dans le volet **Veuillez sélectionner une destination** , sélectionnez l’application com+ sur laquelle le composant sera copié.

3.  Entrez le nouvel identificateur de programme (ProgID) pour le composant copié.

    > [!Note]  
    > Le ProgID du composant d’origine est affiché pour référence et ne peut pas être modifié.

     

4.  Entrez le nouveau CLSID du composant copié. COM+ affiche un CLSID généré automatiquement. Dans la plupart des cas, cela suffit pour identifier de manière unique le composant copié. Entrez un nouveau CLSID uniquement si vous souhaitez modifier celui fourni par COM+.

5.  Cliquez sur **OK**.
