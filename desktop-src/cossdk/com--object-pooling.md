# <a name="com-object-pooling"></a>Mise en pool d’objets COM+

Le service de mise en pool d’objets COM+ vous permet de configurer un composant pour que des instances de lui-même restent actives dans un pool, prêtes à être utilisées par n’importe quel client qui demande le composant.

Les rubriques décrites dans le tableau suivant fournissent des informations générales et sur les tâches relatives à l’utilisation de la mise en pool d’objets dans COM+.



| Rubrique                                                                       | Description                                                                                                |
|-----------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------|
| [Concepts de mise en pool d’objets COM+](com--object-pooling-concepts.md)<br/> | Fournit des informations sur l’utilisation du service de mise en pool d’objets COM+ pour améliorer les performances du système.<br/> |
| [Tâches de mise en pool d’objets COM+](com--object-pooling-tasks.md)<br/>       | Fournit des instructions détaillées pour l’implémentation et la surveillance du regroupement d’objets COM+.<br/>             |
