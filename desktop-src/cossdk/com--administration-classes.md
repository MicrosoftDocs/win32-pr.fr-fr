# <a name="com-administration-classes"></a>Classes d’administration COM+

Les classes d’administration COM+ sont répertoriées ci-dessous.



| Classe                                                          | Description                                                                                                                                            |
|----------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**COMAdminCatalog**](comadmincatalog.md)                     | Accède aux données stockées dans le catalogue COM+.                                                                                                  |
| [**COMAdminCatalogCollection**](comadmincatalogcollection.md) | Représente n’importe quelle collection dans le catalogue COM+. Utilisez-le pour énumérer, ajouter, supprimer et récupérer des éléments dans une collection et pour accéder aux regroupements associés. |
| [**COMAdminCatalogObject**](comadmincatalogobject.md)         | Représente des éléments dans les collections du catalogue COM+. Utilisez-le pour récupérer et modifier les propriétés exposées par un élément dans une collection.                      |
