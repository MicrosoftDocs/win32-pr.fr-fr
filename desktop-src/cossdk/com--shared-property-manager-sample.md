# <a name="com-shared-property-manager-sample"></a>Exemple de Gestionnaire de propriétés COM+ Shared

Pour obtenir un exemple de code sur l’utilisation de l’Gestionnaire de propriétés partagé COM+, consultez [**ISharedProperty ::p \_ valeur ut**](/windows/desktop/api/ComSvcs/nf-comsvcs-isharedproperty-put_value). Cet exemple C++ montre comment créer un groupe de propriétés partagées, créer une propriété partagée dans ce groupe, puis définir la valeur de cette propriété.
