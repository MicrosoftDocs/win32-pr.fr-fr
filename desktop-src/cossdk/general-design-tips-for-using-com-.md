# <a name="general-design-tips-for-using-com"></a>Conseils de conception générale pour l’utilisation de COM+

La clé de bonne conception est la planification. Si vous n’êtes pas familiarisé avec la conception d’une application à trois niveaux, consultez la section intitulée [conception de l’application com+ à l’aide de UML](designing-the-com--application-using-uml.md).

La façon dont vous construisez les composants COM qui composent la couche intermédiaire de votre application COM+ peut avoir un impact considérable sur les performances, l’évolutivité et la facilité de déploiement. Les rubriques suivantes fournissent des considérations relatives à la conception et des conseils d’implémentation que vous devez connaître lors de la conception des composants COM.



| Rubrique                                                                   | Description                                                                                                                |
|-------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------|
| [Conception pour l’évolutivité](designing-for-scalability.md)<br/>   | Suggère des méthodes pour augmenter l’évolutivité de votre application COM+.<br/>                                             |
| [Conception pour la disponibilité](designing-for-availability.md)<br/> | Décrit les services que vous pouvez utiliser pour améliorer la disponibilité de vos fonctionnalités de niveau intermédiaire dans les applications COM+.<br/> |
| [Conception pour la sécurité](designing-for-security.md)<br/>         | Décrit les méthodes permettant de configurer de manière plus efficace la sécurité pour une application COM+.<br/>                                      |
| [Conception pour le déploiement](designing-for-deployment.md)<br/>     | Explique comment réduire les problèmes de déploiement lors de la conception d’une application distribuée.<br/>                       |



 

En outre, assurez-vous d' [améliorer les performances avec](improving-performance-with-object-pooling.md) la mise en pool d’objets pour utiliser le plus efficacement possible la mise en pool d’objets.
