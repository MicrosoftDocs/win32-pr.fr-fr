# <a name="specifying-an-object-constructor-string-for-a-component"></a>Spécification d’une chaîne de constructeur d’objet pour un composant

Pour spécifier une chaîne de constructeur d’objet pour un composant, procédez comme suit :

1.  Dans le volet d’informations de l’outil d’administration Services de composants, cliquez avec le bouton droit sur le composant que vous souhaitez configurer, puis cliquez sur **Propriétés**.
2.  Dans la boîte de dialogue Propriétés du composant, cliquez sur l’onglet **activation** .
3.  Pour activer l’utilisation de la chaîne du constructeur d’objet, activez la case à cocher **activer la construction** de l’objet.
4.  Dans la zone **chaîne du constructeur** , entrez la chaîne de construction (par exemple, DSN = AccountingServer). Vous avez la possibilité d’entrer une chaîne vide.

> [!Note]  
> Les chaînes de constructeur d’objet ne doivent pas être utilisées pour stocker des informations sensibles à la sécurité.
