# <a name="com-low-memory-activation-gates-tasks"></a>Tâches du portail d’activation COM+ Low-Memory

La fonctionnalité des portes d’activation en mémoire basse COM+ est fournie automatiquement par COM+. Cette fonctionnalité n’est pas configurable, donc aucune tâche ne lui est associée.
