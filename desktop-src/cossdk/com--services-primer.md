# <a name="com-services-primer"></a>Initiation aux services COM+

COM+ fournit un modèle de programmation simple pour l’utilisation de ses services automatiques. Vous pouvez simplement définir ces services en tant qu’attributs déclaratifs sur des composants et leurs méthodes. Lorsque vous définissez ces attributs, COM+ remet automatiquement les services demandés à l’objet en fonction des besoins.

Ce guide d’initiation montre les services COM+ en action en parcourant l’exemple de code de composant en trois étapes définies. La complexité des informations est générée au fur et à mesure de la progression des étapes. Le code se concentre sur le modèle de programmation simplifié, en utilisant le traitement des transactions pour illustrer les principes de base de COM+.

Chacune des étapes suivantes illustre un aspect fondamental des services COM+ :

-   [Étape 1 : création d’un composant transactionnel](step-1--creating-a-transactional-component.md)
-   [Étape 2 : extension d’une transaction à travers plusieurs composants](step-2--extending-a-transaction-across-multiple-components.md)
-   [Étape 3 : réutilisation des composants](step-3--reusing-components.md)
