# <a name="com-applications-running-as-service-applications"></a>Applications COM+ s’exécutant en tant qu’applications de service

Les développeurs COM+ peuvent utiliser l’outil d’administration Services de composants pour configurer et implémenter une application serveur COM+ en tant qu’application de service. Cela signifie que le serveur peut être démarré ou redémarré automatiquement si votre application doit toujours être en cours d’exécution ; que votre application COM+ peut s’exécuter en tant que compte système local s’il doit effectuer des opérations privilégiées ; et que les services dépendants de votre application peuvent maintenant être démarrés automatiquement.

Les rubriques suivantes fournissent des informations générales et sur les tâches relatives à la configuration de vos applications pour qu’elles s’exécutent en tant qu’applications de service.



| Rubrique                                                                      | Description                                                                                         |
|----------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------|
| [Concepts des applications de service COM+](com--service-application-concepts.md) | Fournit une vue d’ensemble des avantages de l’exécution d’une application serveur COM+ en tant qu’application de service. |
| [Tâches de l’application de service COM+](com--service-application-tasks.md)       | Fournit des instructions pour la configuration d’une application serveur COM+ en tant que service.                       |
