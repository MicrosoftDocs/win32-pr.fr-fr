# <a name="deploying-com-applications"></a>Déploiement d’applications COM+

Cette section présente les fonctionnalités de COM+ qui facilitent le déploiement d’applications COM+. La section décrit également la façon dont les applications COM+ sont inscrites dans le catalogue COM+. Pour plus d’informations, consultez les rubriques décrites dans le tableau suivant.

> [!Note]  
> Pour obtenir des procédures détaillées sur le déploiement d’applications COM+, voir la rubrique relative à l’installation des applications COM+ dans l’aide de l’administration des services de composants.

|-----------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|

| [Déploiement de proxys d’application](deploying-application-proxies.md)<br/>                                               | Décrit comment exporter une application serveur COM+ en tant que proxy d’application à l’aide de l’outil d’administration Services de composants.<br/>                                                       |
| [Le catalogue COM+](the-com--catalog.md)<br/>                                                                         | Décrit comment le catalogue COM+ stocke les attributs d’application COM+, les attributs de classe et les attributs au niveau de l’ordinateur.<br/>                                                                 |
| [L’utilitaire de réplication COMREPL](the-comrepl-replication-utility.md)<br/>                                           | Décrit l’utilitaire COMREPL, qui permet de répliquer le catalogue COM+ d’un ordinateur source donné sur un ou plusieurs ordinateurs cibles.<br/>                                            |
