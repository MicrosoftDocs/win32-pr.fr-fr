# <a name="com-object-pooling-tasks"></a>Tâches de mise en pool d’objets COM+

Vous pouvez configurer un composant à regrouper à l’aide de l’outil d’administration Services de composants ou des fonctions d’administration COM+. Avant de le configurer, vous devez être certain que le composant est correctement écrit pour prendre en charge le regroupement.

Les rubriques suivantes de cette section fournissent des instructions pas à pas pour l’utilisation du regroupement d’objets COM+.



| Rubrique                                                                                       | Description                                                                                                                                 |
|---------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------|
| [Configuration d’un composant à regrouper](configuring-a-component-to-be-pooled.md)<br/> | Décrit comment créer un composant regroupé.<br/>                                                                                      |
| [Analyse des statistiques des objets](monitoring-object-statistics.md)<br/>                 | Décrit comment surveiller les objets afin que vous puissiez ajuster les caractéristiques du pool d’objets pour obtenir les performances souhaitées. <br/> |
