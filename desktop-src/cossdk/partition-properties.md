# <a name="partition-properties"></a>propriétés de la partition

Les partitions ont cinq propriétés disponibles. Ces propriétés et leurs descriptions sont les suivantes :

<dl> <dt>

<span id="Partition_Name"></span><span id="partition_name"></span><span id="PARTITION_NAME"></span>Nom de la partition
</dt> <dd>

Chaîne de texte associée à la partition.

</dd> <dt>

<span id="Partition_ID"></span><span id="partition_id"></span><span id="PARTITION_ID"></span>ID de partition
</dt> <dd>

GUID utilisé par le système pour mettre en corrélation un utilisateur et une partition.

</dd> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>Descriptive
</dt> <dd>

Description textuelle de la partition.

</dd> <dt>

<span id="Disable_Deletion"></span><span id="disable_deletion"></span><span id="DISABLE_DELETION"></span>Désactiver la suppression
</dt> <dd>

Mécanisme permettant d’interdire à un utilisateur de supprimer une partition.

</dd> <dt>

<span id="Disable_Changes"></span><span id="disable_changes"></span><span id="DISABLE_CHANGES"></span>Désactiver les modifications
</dt> <dd>

Mécanisme permettant d’interdire à un utilisateur d’apporter des modifications à une partition.

</dd> </dl>

Pour afficher ces propriétés, cliquez avec le bouton droit sur le nom d’une partition dans l’outil d’administration Services de composants et choisissez **Propriétés**.

Pour plus d’informations sur la propriété ID de partition, consultez la page [inscription et activation de composants dans des partitions](registering-and-activating-components-in-partitions.md).
