# <a name="com-object-constructor-strings"></a>Chaînes du constructeur d’objet COM+

Les chaînes du constructeur d’objet COM+ prennent en charge la programmation générique. Elles vous permettent d’écrire un composant unique avec un degré de généralisation afin qu’il puisse être personnalisé pour une tâche particulière à l’aide de chaînes de constructeur spécifiées administrativement.

Les rubriques décrites dans le tableau suivant fournissent des informations générales et sur les tâches relatives à l’utilisation des chaînes de constructeur d’objet COM+.



| Rubrique                                                                                               | Description                                                                                   |
|-----------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------|
| [Concepts des chaînes du constructeur d’objet COM+](com--object-constructor-strings-concepts.md)<br/> | Fournit une vue d’ensemble de l’utilisation des chaînes de constructeur d’objet COM+.<br/>                     |
| [Tâches des chaînes du constructeur d’objet COM+](com--object-constructor-strings-tasks.md)<br/>       | Fournit des instructions pour l’utilisation de COM+ afin de créer et d’utiliser des chaînes de constructeur d’objet.<br/> |
