# <a name="moving-components"></a>Déplacement des composants

Vous pouvez déplacer un composant d’une application COM+ à une autre. En effet, le déplacement d’un composant supprime le composant de l’application COM+ d’origine et l’installe dans l’application COM+ cible.

**Pour déplacer un composant d’une application COM+ à une autre**

1.  Dans le volet d’informations de l’outil d’administration Services de composants, cliquez avec le bouton droit sur le composant que vous souhaitez déplacer, puis cliquez sur **déplacer**.

2.  Dans la boîte de dialogue **déplacer le ou les composants** , sélectionnez l’application com+ de destination vers laquelle vous souhaitez déplacer le composant.

3.  Cliquez sur **OK**.
