# <a name="com-soap-service"></a>Service SOAP COM+

En sélectionnant simplement la case à cocher **utilise SOAP** dans l’outil d’administration Services de composants, vous pouvez exposer une application com+ en tant que service Web XML. Vous pouvez également utiliser de manière transparente un service Web XML, qu’il soit déployé à l’aide de COM+ ou non, en tant que composant COM. Cela signifie que vous pouvez facilement créer des services Web XML à partir d’applications COM+ existantes et incorporer facilement des services Web XML dans de nouvelles applications COM+.

Les rubriques décrites dans le tableau suivant fournissent des informations générales et sur les tâches relatives à l’utilisation de COM+ pour exposer vos composants COM configurés en tant que services Web XML.



| Rubrique                                                                   | Description                                                                            |
|-------------------------------------------------------------------------|----------------------------------------------------------------------------------------|
| [Concepts du service SOAP COM+](com--soap-service-concepts.md)<br/> | Fournit une vue d’ensemble des services Web XML et de SOAP.<br/>                          |
| [Tâches de service SOAP COM+](com--soap-service-tasks.md)<br/>       | Fournit des instructions sur l’utilisation de COM+ pour créer et déployer des services Web XML.<br/> |
