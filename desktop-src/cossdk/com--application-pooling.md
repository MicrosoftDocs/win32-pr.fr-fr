# <a name="com-application-pooling"></a>Regroupement d’applications COM+

Le regroupement d’applications COM+ permet aux processus monothreads de se mettre à l’échelle et peut également vous aider à récupérer après des défaillances dans les processus uniques en fournissant d’autres processus en cours d’exécution capables de gérer les activations.

Les rubriques suivantes fournissent des informations générales et sur les tâches relatives à l’utilisation du regroupement d’applications COM+.

| [Tâches de regroupement d’applications COM+](com--application-pooling-tasks.md)       | Fournit des instructions pour la configuration du regroupement d’applications COM+. |
