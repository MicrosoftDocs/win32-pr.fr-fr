# <a name="designing-com-applications"></a>Conception d’applications COM+

Que vous conceviez une application pour l’exécution de commerce en ligne, le suivi des informations, l’accès aux données dans les systèmes hérités, la messagerie, la planification des ressources, la publication et la gestion des connaissances, ou une application LOB stratégique (métier), vous pouvez améliorer et rationaliser votre conception de niveau intermédiaire en intégrant des services COM+. Le moyen le plus courant consiste à regrouper des composants dans des applications COM+.

Bien qu’une analyse complète de la méthodologie pour l’utilisation de COM+ dans toutes les situations dépasse le cadre de ce document, les rubriques suivantes présentent les informations que vous pouvez utiliser lors de la création d’une application COM+.
