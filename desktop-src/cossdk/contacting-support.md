# <a name="contacting-support"></a>Contacter le support technique

Si vous rencontrez un problème que vous ne pouvez pas résoudre, vous pouvez contacter le support technique Microsoft avec les informations suivantes :

-   Topographie de l’application dans laquelle l’erreur s’est produite, telle qu’une description des applications, des composants et des interfaces.
-   Journal des événements de l’application sur tous les ordinateurs.
-   Description de la manière de reproduire l’erreur, si possible.
