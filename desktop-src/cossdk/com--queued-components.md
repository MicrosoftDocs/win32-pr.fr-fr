# <a name="com-queued-components"></a>Composants en file d’attente COM+

La fonctionnalité COM+ Queued Components permet à un client de demander un service à partir d’une application serveur COM+, même lorsque l’application serveur est temporairement indisponible. La demande du client est stockée dans une file d’attente et traitée lorsque le serveur devient disponible.

> [!Note]  
> Pour utiliser le service composants en file d’attente COM+, [Message Queuing](/previous-versions/windows/desktop/legacy/ms711472(v=vs.85)) doit être installé sur votre ordinateur.
