# <a name="local-partitions"></a>Partitions locales

L’une des façons dont les partitions peuvent être utilisées est locale sur un serveur d’applications. Dans ce scénario, un administrateur peut créer une partition à l’aide de l’outil d’administration Services de composants. Les partitions implémentées localement et qui ne se trouvent pas dans Active Directory sont accessibles uniquement aux utilisateurs locaux, et non aux utilisateurs d’autres ordinateurs du domaine. Une fois qu’une partition a été créée et qu’une application COM+ y est installée, l’administrateur peut attribuer cette partition en tant que partition par défaut pour un ou plusieurs utilisateurs sur ce serveur. Pour affecter une partition par défaut à un utilisateur, l’administrateur peut utiliser l’outil d’administration Services de composants.
