# <a name="com-just-in-time-activation"></a>Activation juste-à-temps de COM+

L’activation juste-à-temps (JIT) est un service COM+ automatique qui peut vous aider à utiliser plus efficacement les ressources du serveur, en particulier lors de la mise à l’échelle de votre application pour effectuer des transactions à volume élevé.

Les rubriques suivantes fournissent des informations générales et sur les tâches relatives à l’activation juste-à-temps de COM+.



| Rubrique                                                                              | Description                                                                                                        |
|------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------|
| [Concepts d’activation juste-à-temps de COM+](com--just-in-time-activation-concepts.md) | Fournit une vue d’ensemble des fonctionnalités et des avantages de l’activation du service d’activation JIT pour vos composants COM+. |
| [Tâches d’activation juste-à-temps COM+](com--just-in-time-activation-tasks.md)       | Fournit des instructions sur la configuration de l’activation JIT pour vos applications COM+.                                       |
