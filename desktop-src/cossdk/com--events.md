# <a name="com-events"></a>Événements COM+

Le service d’événements COM+ est un système d’événements automatisé et faiblement couplé qui stocke les informations d’événements provenant de différents serveurs de publication dans le catalogue COM+. Les abonnés peuvent interroger ce magasin et sélectionner les événements qu’ils souhaitent connaître.

| [Tâches des événements COM+](com--events-tasks.md)       | Fournit des instructions sur l’utilisation du service d’événements COM+ dans vos applications. |
