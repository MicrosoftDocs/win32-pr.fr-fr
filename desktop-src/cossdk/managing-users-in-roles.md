# <a name="managing-users-in-roles"></a>Gestion des utilisateurs dans les rôles

La plupart des tâches d’administration pour la configuration de la sécurité des applications COM+ impliquent le remplissage des rôles prédéfinis avec les utilisateurs, à la fois lors de l’installation d’applications et de manière continue. Pour obtenir des informations de présentation sur le fonctionnement de ces fonctionnalités de sécurité, consultez Configuration de la [sécurité Role-Based](configuring-role-based-security.md).

Dans cette section, vous trouverez des instructions pas à pas pour les tâches suivantes :

-   [Activation de la vérification d’autorisation Role-Based](enabling-role-based-authorization-checking.md)
-   [Attribution d’un compte d’utilisateur ou d’un groupe à un rôle](assigning-a-user-account-or-group-to-a-role.md)
-   [Suppression d’un compte d’utilisateur ou d’un groupe d’un rôle](removing-a-user-account-or-group-from-a-role.md)
-   [Déplacement d’un compte d’utilisateur ou d’un groupe vers un autre rôle](moving-a-user-account-or-group-to-a-different-role.md)
