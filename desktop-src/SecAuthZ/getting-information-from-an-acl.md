# <a name="getting-information-from-an-acl"></a><span data-ttu-id="944ef-103">Obtention d’informations à partir d’une liste de contrôle d’accès</span><span class="sxs-lookup"><span data-stu-id="944ef-103">Getting Information from an ACL</span></span>

<span data-ttu-id="944ef-104">Plusieurs fonctions sont fournies pour récupérer les informations de contrôle d’accès à partir d’une [*liste de contrôle d’accès*](/windows/desktop/SecGloss/a-gly) (ACL).</span><span class="sxs-lookup"><span data-stu-id="944ef-104">Several functions are provided that retrieve access control information from an [*access control list*](/windows/desktop/SecGloss/a-gly) (ACL).</span></span> <span data-ttu-id="944ef-105">Celles-ci incluent des fonctions permettant de déterminer les droits d’accès qu’une liste de contrôle d’accès accorde ou audite à un [tiers de confiance](trustees.md)spécifié.</span><span class="sxs-lookup"><span data-stu-id="944ef-105">These include functions for determining the access rights that an ACL grants or audits for a specified [trustee](trustees.md).</span></span> <span data-ttu-id="944ef-106">D’autres fonctions vous permettent d’extraire des informations sur les [*entrées de contrôle d’accès*](/windows/desktop/SecGloss/a-gly) (ACE) d’une liste de contrôle d’accès.</span><span class="sxs-lookup"><span data-stu-id="944ef-106">Other functions enable you to extract information about the [*access control entries*](/windows/desktop/SecGloss/a-gly) (ACEs) in an ACL.</span></span>

<span data-ttu-id="944ef-107">La fonction [**GetExplicitEntriesFromAcl**](/windows/desktop/api/Aclapi/nf-aclapi-getexplicitentriesfromacla) récupère un tableau de structures [**d' \_ accès explicites**](/windows/desktop/api/AccCtrl/ns-accctrl-explicit_access_a) qui décrivent les ACE dans une liste de contrôle d’accès.</span><span class="sxs-lookup"><span data-stu-id="944ef-107">The [**GetExplicitEntriesFromAcl**](/windows/desktop/api/Aclapi/nf-aclapi-getexplicitentriesfromacla) function retrieves an array of [**EXPLICIT\_ACCESS**](/windows/desktop/api/AccCtrl/ns-accctrl-explicit_access_a) structures that describe the ACEs in an ACL.</span></span> <span data-ttu-id="944ef-108">Cela peut être utile lors de la copie d’informations ACE d’une liste de contrôle d’accès à une autre.</span><span class="sxs-lookup"><span data-stu-id="944ef-108">This can be useful when copying ACE information from one ACL to another.</span></span> <span data-ttu-id="944ef-109">Par exemple, un appel à **GetExplicitEntriesFromAcl** pour obtenir des informations sur les ACE dans une ACL peut être suivi en passant les structures d' **\_ accès explicites** retournées dans un appel à la fonction [**SetEntriesInAcl**](/windows/desktop/api/Aclapi/nf-aclapi-setentriesinacla) pour créer des ACE équivalentes dans une nouvelle liste de contrôle d’accès.</span><span class="sxs-lookup"><span data-stu-id="944ef-109">For example, a call to **GetExplicitEntriesFromAcl** to get information about the ACEs in one ACL can be followed by passing the returned **EXPLICIT\_ACCESS** structures in a call to the [**SetEntriesInAcl**](/windows/desktop/api/Aclapi/nf-aclapi-setentriesinacla) function to create equivalent ACEs in a new ACL.</span></span>

<span data-ttu-id="944ef-110">La fonction [**GetEffectiveRightsFromAcl**](/windows/desktop/api/Aclapi/nf-aclapi-geteffectiverightsfromacla) vous permet de déterminer les droits d’accès effectifs qu’une liste DACL accorde à un tiers de confiance spécifié.</span><span class="sxs-lookup"><span data-stu-id="944ef-110">The [**GetEffectiveRightsFromAcl**](/windows/desktop/api/Aclapi/nf-aclapi-geteffectiverightsfromacla) function enables you to determine the effective access rights that a DACL grants to a specified trustee.</span></span> <span data-ttu-id="944ef-111">Les droits d’accès effectifs du tiers de confiance sont les droits d’accès qu’un DACL accorde au tiers de confiance ou à tous les groupes dont le tiers de confiance est membre.</span><span class="sxs-lookup"><span data-stu-id="944ef-111">The trustee's effective access rights are the access rights that a DACL grants to the trustee or to any groups of which the trustee is a member.</span></span> <span data-ttu-id="944ef-112">**GetEffectiveRightsFromAcl** vérifie tous les ACE access-allowed et Access-denied dans la liste DACL spécifiée.</span><span class="sxs-lookup"><span data-stu-id="944ef-112">**GetEffectiveRightsFromAcl** checks all access-allowed and access-denied ACEs in the specified DACL.</span></span>

<span data-ttu-id="944ef-113">**Utilisez les étapes suivantes pour déterminer les droits d’accès d’un tiers de confiance à un objet.**</span><span class="sxs-lookup"><span data-stu-id="944ef-113">**Use the following steps to determine a trustee's access rights to an object**</span></span>

1.  <span data-ttu-id="944ef-114">Appelez la fonction [**GetSecurityInfo**](/windows/desktop/api/Aclapi/nf-aclapi-getsecurityinfo) ou [**GetNamedSecurityInfo**](/windows/desktop/api/Aclapi/nf-aclapi-getnamedsecurityinfoa) pour obtenir un pointeur vers la liste DACL d’un objet.</span><span class="sxs-lookup"><span data-stu-id="944ef-114">Call the [**GetSecurityInfo**](/windows/desktop/api/Aclapi/nf-aclapi-getsecurityinfo) or [**GetNamedSecurityInfo**](/windows/desktop/api/Aclapi/nf-aclapi-getnamedsecurityinfoa) function to get a pointer to an object's DACL.</span></span>
2.  <span data-ttu-id="944ef-115">Appelez la fonction [**GetEffectiveRightsFromAcl**](/windows/desktop/api/Aclapi/nf-aclapi-geteffectiverightsfromacla) pour récupérer les droits d’accès que la liste DACL accorde à un tiers de confiance spécifié.</span><span class="sxs-lookup"><span data-stu-id="944ef-115">Call the [**GetEffectiveRightsFromAcl**](/windows/desktop/api/Aclapi/nf-aclapi-geteffectiverightsfromacla) function to retrieve the access rights that the DACL grants to a specified trustee.</span></span>

<span data-ttu-id="944ef-116">La fonction [**GetAuditedPermissionsFromAcl**](/windows/desktop/api/Aclapi/nf-aclapi-getauditedpermissionsfromacla) vous permet de vérifier une liste SACL pour déterminer les droits d’accès audités pour un tiers de confiance spécifié ou pour tous les groupes dont le tiers de confiance est membre.</span><span class="sxs-lookup"><span data-stu-id="944ef-116">The [**GetAuditedPermissionsFromAcl**](/windows/desktop/api/Aclapi/nf-aclapi-getauditedpermissionsfromacla) function enables you to check a SACL to determine the audited access rights for a specified trustee or for any groups of which the trustee is a member.</span></span> <span data-ttu-id="944ef-117">Les droits d’audit indiquent les types de tentatives d’accès qui obligent le système à générer un enregistrement d’audit dans le journal des événements de sécurité.</span><span class="sxs-lookup"><span data-stu-id="944ef-117">The audited rights indicate the types of access attempts that cause the system to generate an audit record in the security event log.</span></span> <span data-ttu-id="944ef-118">La fonction retourne deux [*masques d’accès*](/windows/desktop/SecGloss/a-gly): l’un contenant les droits d’accès surveillés pour les échecs de tentatives d’accès et l’autre contenant les droits d’accès surveillés pour un accès réussi.</span><span class="sxs-lookup"><span data-stu-id="944ef-118">The function returns two [*access masks*](/windows/desktop/SecGloss/a-gly): one containing the access rights monitored for failed access attempts, and another containing the access rights monitored for successful access.</span></span> <span data-ttu-id="944ef-119">**GetAuditedPermissionsFromAcl** vérifie toutes les entrées de contrôle d’intégrité système dans une liste SACL.</span><span class="sxs-lookup"><span data-stu-id="944ef-119">**GetAuditedPermissionsFromAcl** checks all system-audit ACEs in a SACL.</span></span>
