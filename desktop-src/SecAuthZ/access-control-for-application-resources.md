# <a name="access-control-for-application-resources"></a>Access Control pour les ressources d’application

l’API du gestionnaire d’autorisations et le composant logiciel enfichable MMC disponibles dans Windows Server 2003 fournissent aux applications un framework de contrôle d’accès en fonction du rôle flexible. Cette infrastructure comprend un ensemble d’objets COM supprimables pour les applications Web et métier.

Les sections suivantes fournissent des informations sur le contrôle de l’accès aux ressources de l’application :

-   [Contrôle d’accès en fonction du rôle](role-based-access-control.md)
-   [Access Control basées sur les listes de contrôle d’accès](acl-based-access-control.md)
