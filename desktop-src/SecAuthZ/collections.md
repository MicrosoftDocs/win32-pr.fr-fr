# <a name="collections-authorization"></a>Regroupements (autorisation)

L’API du gestionnaire d’autorisations fournit des interfaces qui représentent des collections d’autres objets. Par exemple, l’interface [**IAzRoles**](/windows/desktop/api/Azroles/nn-azroles-iazroles) représente une collection d’objets [**IAzRole**](/windows/desktop/api/Azroles/nn-azroles-iazrole) .

Les autres interfaces qui représentent des collections sont [**IAzApplications**](/windows/desktop/api/Azroles/nn-azroles-iazapplications), [**IAzApplicationGroups**](/windows/desktop/api/Azroles/nn-azroles-iazapplicationgroups), [**IAzOperations**](/windows/desktop/api/Azroles/nn-azroles-iazoperations), [**IAzScopes**](/windows/desktop/api/Azroles/nn-azroles-iazscopes)et [**IAzTasks**](/windows/desktop/api/Azroles/nn-azroles-iaztasks). Chacune de ces interfaces fournit des propriétés qui facilitent l’énumération des objets dans une collection.
