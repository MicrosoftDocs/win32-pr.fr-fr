# <a name="authorization-objects"></a>Objets d’autorisation

Les objets suivants sont utilisés avec les applications d’autorisation.

ActiveX Le programme d’installation fournit l’objet suivant.

## <a name="in-this-section"></a>Contenu de cette section



| Rubrique                                                               | Description                                                                                                                        |
|---------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------|
| [**CIeAxiInstallerService**](cieaxiinstallerservice.md)<br/> | Implémente les interfaces [**IAxiService**](ieaxiservice.md) et [**IeAxiServiceCallback**](ieaxiservicecallback.md) .<br/> |



 


Le gestionnaire d’autorisations fournit les objets suivants.



| Object                                               | Description                                                                          |
|------------------------------------------------------|--------------------------------------------------------------------------------------|
| [**AzAuthorizationStore**](/windows/desktop/api/Azroles/nn-azroles-iazauthorizationstore) | Définit le conteneur qui est la racine du magasin de stratégies d’autorisation.<br/> |
| [**AzBizRuleContext**](/windows/desktop/api/Azroles/nn-azroles-iazbizrulecontext)         | Contient des informations sur une opération de règle d’entreprise (BizRule).<br/>           |
