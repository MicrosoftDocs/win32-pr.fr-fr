# <a name="authorization-data-types"></a>Types de données d’autorisation

Les types de données suivants sont utilisés avec les applications d’autorisation.

## <a name="in-this-section"></a>Contenu de cette section



| Rubrique                                                                           | Description                                                                                                                                                                                                                                                                                     |
|---------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**masque d’accès \_**](access-mask.md)<br/>                                  | Définit des droits standard, spécifiques et génériques. Ces droits sont utilisés dans les [entrées de contrôle d’accès](/windows/desktop/SecGloss/a-gly) (ACE) et constituent le moyen principal de spécifier l’accès demandé ou accordé à un objet.<br/> |
| [**\_contrôle de descripteur de sécurité \_**](security-descriptor-control.md)<br/> | Jeu d’indicateurs binaires qui qualifient la signification d’un [descripteur de sécurité](/windows/desktop/SecGloss/s-gly) ou de ses composants.<br/>                                                                                                          |
| [**informations de sécurité \_**](security-information.md)<br/>                | Identifie les informations de sécurité relatives aux objets qui sont définies ou interrogées.<br/>                                                                                                                                                                                                             |
