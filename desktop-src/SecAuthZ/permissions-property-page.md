# <a name="permissions-property-page"></a>Autorisations, page de propriétés

La page de propriétés autorisations contient des contrôles pour la modification avancée des [*entrées*](/windows/desktop/SecGloss/a-gly) du contrôle d' **accès** dans la [*liste DACL*](/windows/desktop/SecGloss/d-gly)d’un objet. Cette page permet à l’utilisateur d’afficher et de modifier toutes les informations de la liste DACL, y compris les informations qui ne sont pas disponibles sur la [page de propriétés sécurité de base](basic-security-property-page.md). Les informations avancées incluent des entrées de contrôle d’accès [spécifiques aux objets](object-specific-aces.md) et des informations sur [l’héritage d’ACE](ace-inheritance.md).

La page de propriétés **autorisations** est toujours incluse dans la [feuille de propriétés de sécurité avancée](advanced-security-property-sheet.md) qui s’affiche lorsque l’utilisateur clique sur le bouton **avancé** de la page de propriétés sécurité de base.
