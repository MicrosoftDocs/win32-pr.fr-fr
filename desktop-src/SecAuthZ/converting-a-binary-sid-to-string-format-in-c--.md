# <a name="converting-a-binary-sid-to-string-format-in-c"></a>Conversion d’un SID binaire en format de chaîne en C++

Les fonctions [**ConvertSidToStringSid a**](/windows/desktop/api/Sddl/nf-sddl-convertsidtostringsida) et [**ConvertStringSidToSid**](/windows/desktop/api/Sddl/nf-sddl-convertstringsidtosida) convertissent un [*identificateur de sécurité*](/windows/desktop/SecGloss/s-gly) (SID) vers et à partir du format de chaîne. Pour obtenir une description du format de chaîne SID, consultez [sid Components](sid-components.md).
