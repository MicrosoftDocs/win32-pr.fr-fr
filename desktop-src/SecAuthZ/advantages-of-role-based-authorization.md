# <a name="advantages-of-role-based-authorization"></a>Avantages de l’autorisation basée sur les rôles

L’API du gestionnaire d’autorisations fournit un modèle d’autorisation qui présente plusieurs avantages par rapport à l’autorisation de bas niveau (basée sur les listes de contrôle d’accès) :

-   Simplifie l’administration du contrôle d’accès.
-   rend l’autorisation disponible pour les scripts et les applications développés à l’aide du système de développement Microsoft Visual Basic.
-   Fournit un mécanisme permettant d’appliquer une logique métier d’exécution lors de la vérification de l’accès.
