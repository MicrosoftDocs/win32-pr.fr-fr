# <a name="primary-group-of-a-new-object"></a>Groupe principal d’un nouvel objet

Le groupe principal d’un nouvel objet est le groupe principal du [*descripteur de sécurité*](/windows/desktop/SecGloss/s-gly) spécifié par le créateur de l’objet. Si le créateur d’un objet ne spécifie pas de groupe principal, le groupe principal de l’objet est le groupe principal par défaut à partir du jeton principal ou d' [*emprunt d’identité*](/windows/desktop/SecGloss/i-gly)du créateur.
