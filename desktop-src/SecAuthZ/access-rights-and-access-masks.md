<span data-ttu-id="7c367-104">Un *droit d’accès* est un indicateur binaire qui correspond à un ensemble particulier d’opérations qu’un thread peut effectuer sur un objet sécurisable.</span><span class="sxs-lookup"><span data-stu-id="7c367-104">An *access right* is a bit flag that corresponds to a particular set of operations that a thread can perform on a securable object.</span></span> <span data-ttu-id="7c367-105">Par exemple, une clé de registre a le \_ \_ droit d’accès à la valeur de jeu de clés, qui correspond à la capacité d’un thread à définir une valeur sous la clé.</span><span class="sxs-lookup"><span data-stu-id="7c367-105">For example, a registry key has the KEY\_SET\_VALUE access right, which corresponds to the ability of a thread to set a value under the key.</span></span> <span data-ttu-id="7c367-106">Si un thread essaie d’exécuter une opération sur un objet, mais n’a pas le droit d’accès nécessaire à l’objet, le système n’effectue pas l’opération.</span><span class="sxs-lookup"><span data-stu-id="7c367-106">If a thread tries to perform an operation on an object, but does not have the necessary access right to the object, the system does not carry out the operation.</span></span>

<span data-ttu-id="7c367-107">Un [*masque d’accès*](/windows/desktop/SecGloss/a-gly) est une valeur 32 bits dont les bits correspondent aux droits d’accès pris en charge par un objet.</span><span class="sxs-lookup"><span data-stu-id="7c367-107">An [*access mask*](/windows/desktop/SecGloss/a-gly) is a 32-bit value whose bits correspond to the access rights supported by an object.</span></span> <span data-ttu-id="7c367-108">Tous les objets sécurisables Windows utilisent un [format de masque d’accès](access-mask-format.md) qui comprend des bits pour les types suivants de droits d’accès :</span><span class="sxs-lookup"><span data-stu-id="7c367-108">All Windows securable objects use an [access mask format](access-mask-format.md) that includes bits for the following types of access rights:</span></span>

-   [<span data-ttu-id="7c367-110">Droits d’accès standard</span><span class="sxs-lookup"><span data-stu-id="7c367-110">Standard access rights</span></span>](standard-access-rights.md)
-   [<span data-ttu-id="7c367-111">Droit d’accès SACL</span><span class="sxs-lookup"><span data-stu-id="7c367-111">SACL access right</span></span>](sacl-access-right.md)

<span data-ttu-id="7c367-113">Quand un thread tente d’ouvrir un handle vers un objet, le thread spécifie généralement un masque d’accès pour [demander un jeu de droits d’accès](requesting-access-rights-to-an-object.md).</span><span class="sxs-lookup"><span data-stu-id="7c367-113">When a thread tries to open a handle to an object, the thread typically specifies an access mask to [request a set of access rights](requesting-access-rights-to-an-object.md).</span></span> <span data-ttu-id="7c367-114">Par exemple, une application qui doit définir et interroger les valeurs d’une clé de Registre peut ouvrir la clé à l’aide d’un masque d’accès pour demander la valeur du jeu de clés et les droits d’accès à la \_ valeur de requête de \_ clé \_ \_ .</span><span class="sxs-lookup"><span data-stu-id="7c367-114">For example, an application that needs to set and query the values of a registry key can open the key by using an access mask to request the KEY\_SET\_VALUE and KEY\_QUERY\_VALUE access rights.</span></span>
