# <a name="common-structures"></a>Structures communes

L’infrastructure homologue utilise les structures communes suivantes.



| Structure                                                                          | Description                                                                                                                         |
|------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------|
| [**adresse HOMOLOGUe \_**](/windows/desktop/api/P2P/ns-p2p-peer_address)                                              | Spécifie les informations sur l’adresse IP.                                                                                     |
| [**\_informations sur la connexion homologue \_**](/windows/desktop/api/P2P/ns-p2p-peer_connection_info)                             | Contient des informations sur une connexion. Cette structure est retournée lorsque vous énumérez des connexions de représentation graphique ou de regroupement d’homologues. |
| [**données HOMOLOGUes \_**](/windows/desktop/api/P2P/ns-p2p-peer_data)                                                    | Contient des données binaires.                                                                                                               |
| [**\_données de \_ modification des connexions d’événements homologues \_ \_**](/windows/desktop/api/P2P/ns-p2p-peer_event_connection_change_data) | Contient des informations mises à jour qui incluent des modifications apportées à un voisin ou une connexion directe.                                              |
| [**\_ \_ données entrantes des événements homologues \_**](/windows/desktop/api/P2P/ns-p2p-peer_event_incoming_data)                    | Contient des informations mises à jour qui incluent des modifications de données qu’un nœud reçoit d’une connexion voisine ou directe.                       |
| [**\_données de \_ modification des enregistrements d’événements homologues \_ \_**](/windows/desktop/api/P2P/ns-p2p-peer_event_record_change_data)         | Contient des informations mises à jour qu’une application demande des modifications de données à un enregistrement ou à un type d’enregistrement.                              |
| [**enregistrement d’HOMOLOGUe \_**](/windows/desktop/api/P2P/ns-p2p-peer_record)                                                | Contient l’objet record utilisé par une application.                                                                                |
| [**\_données de version de l’homologue \_**](/windows/desktop/api/P2P/ns-p2p-peer_version_data)                                   | Contient les informations de version sur les API de regroupement et de représentation graphique des homologues.                                                         |
