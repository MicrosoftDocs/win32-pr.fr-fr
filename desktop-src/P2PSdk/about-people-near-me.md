# <a name="about-people-near-me"></a>À propos des gens près de moi

Voisinage immédiat (PNM) est un moyen pour les personnes et les applications de se publier eux-mêmes et leurs intérêts pour faciliter la connexion et la collaboration. Le « Near » dans voisinage immédiat est l’ensemble des utilisateurs qui se trouvent sur le même sous-réseau. PNM est une nouvelle fonctionnalité de [Windows réseau pair à pair](what-is-peer-networking-.md) inclus dans Windows Vista.

Cette section de la documentation contient des informations sur PNM et inclut les rubriques suivantes :

-   [Pourquoi utiliser voisinage immédiat ?](why-use-people-near-me-.md)
-   [Conditions requises pour les personnes près de moi](people-near-me-requirements.md)

Pour plus d’informations sur l’architecture de base, ainsi que sur les API, les magasins et les composants spécifiques utilisés par PNM, consultez la [documentation relative](people-near-me-architecture.md)à l’architecture voisinage immédiat.
