# <a name="pnrp-constants"></a>Constantes PNRP

L’API du fournisseur d’espace de noms PNRP utilise les constantes suivantes.



| Constante                                                 | Description                                                |
|----------------------------------------------------------|------------------------------------------------------------|
| [**\_indicateurs de Cloud PNRP \_**](/windows/desktop/api/Pnrpdef/ne-pnrpdef-pnrp_cloud_flags)           | Spécifie la validité d’un nom de Cloud.                    |
| [**\_État du Cloud PNRP \_**](/windows/desktop/api/Pnrpdef/ne-pnrpdef-pnrp_cloud_state)           | Spécifie les différents États dans lesquels peut se trouver un Cloud PNRP.     |
| [**\_critères de résolution PNRP \_**](/windows/desktop/api/Pnrpdef/ne-pnrpdef-pnrp_resolve_criteria) | Spécifie les critères que le protocole PNRP utilise pour résoudre les recherches. |
