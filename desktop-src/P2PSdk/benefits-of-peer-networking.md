# <a name="benefits-of-peer-networking"></a>Avantages de la mise en réseau pair à pair

L’infrastructure d’égal à égal offre une solution complète qui vous permet de créer des applications de réseau pair à pair. Vous pouvez utiliser les API de création de graphiques d’égal à égal, de regroupement, de gestionnaire d’identité et de fournisseur d’espace de noms PNRP pour créer des applications pair à pair robustes et attrayantes. L’infrastructure d’égal à égal fournit une infrastructure de mise en réseau solide qui vous permet de vous concentrer sur le **développement** d’applications, car l’infrastructure est développée pour vous.

## <a name="design-features"></a>Fonctionnalités de conception

L’infrastructure d’égal à égal offre les fonctionnalités suivantes :

| Fonctionnalité            | Description                                          |
|--------------------|------------------------------------------------------|
| **Sécurisé**         | Robustesse en cas d’échecs ou d’attaques dirigées |
| **Évolutif**       | Évolutivité de réseaux locaux simples vers l’ensemble d’Internet  |
| **Sans serveur**     | Aucun point de défaillance central                          |
| **Réglage automatique**    | S’adapte aux modifications au sein de l’infrastructure          |
| **Réparation automatique** | Répare et corrige automatiquement elle-même            |
| **Partage**        | Active le partage à partir de points de terminaison de réseau Edge-of-Network       |



 

À l’aide de l’infrastructure d’égal à égal, vous pouvez créer les types d’applications suivants :

-   Mise en réseau et communication en temps réel
-   Collaboration
-   Distribution de contenu
-   Coordination du traitement distribué
-   Technologies et protocoles Internet

## <a name="alternative-to-dns"></a>Alternative à DNS

Étant donné que les entités d’égal à égal participent et communiquent sur une base ad hoc, l’infrastructure d’égal à égal est une alternative efficace à DNS pour la résolution de noms. L’infrastructure d’égal à égal utilise l’API du fournisseur d’espace de noms PNRP pour fournir un mécanisme sans serveur pour la résolution des noms d’égal à égal.
