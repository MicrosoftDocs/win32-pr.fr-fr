# <a name="installing-the-peer-infrastructure"></a>Installation de l’infrastructure homologue

l’Infrastructure de réseau homologue Windows est une plateforme de développement permettant de créer des applications homologues pour les ordinateurs exécutant Windows xp avec service pack 1 (SP1) avec Advanced networking Pack, Windows xp avec service pack 2 (SP2), Windows Vista et Windows Server 2008.

Windows Infrastructure réseau homologue est déjà installée sur les ordinateurs exécutant Windows Vista ou Windows Server 2008. pour les ordinateurs exécutant Windows XP avec SP2, les étapes suivantes sont requises pour installer Windows réseau homologue :

-   Cliquez sur **Démarrer**, sur **Panneau de configuration**, puis sur **Ajout/Suppression de programmes**.
-   cliquez sur **ajouter/supprimer des composants de Windows**.
-   Dans **composants**, cliquez sur **services de mise en réseau** (mais n’activez pas sa case à cocher), puis cliquez sur **Détails**.
-   Activez la case à cocher **homologue** , puis cliquez sur **OK**.
-   Cliquez sur **suivant**, puis suivez les instructions de l’Assistant.

les ordinateurs exécutant Windows XP avec SP1 requièrent l’installation du [Pack de mise en réseau avancée](https://www.microsoft.com/downloads/details.aspx?FamilyID=e88cc382-8ce6-4739-97c0-1a52a6f005e4), un téléchargement gratuit, pour activer la mise en réseau pair à pair.
