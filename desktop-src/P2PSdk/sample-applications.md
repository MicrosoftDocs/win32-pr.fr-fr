# <a name="sample-applications-peer-infrastructure"></a>Exemples d’applications (infrastructure d’homologue)

Les exemples d’applications suivants sont inclus dans le kit de développement logiciel (SDK) homologue Windows XP. Les exemples peuvent vous aider quand vous développez vos propres applications homologues à l’aide de l’infrastructure homologue.

-   [Conversation Graph](#graph-chat)
-   [Conversation de groupe](#group-chat)
-   [Explorateur de groupes](#group-browser)

## <a name="graph-chat"></a>Conversation Graph

L’exemple d’application Graph chat est une simple application de conversation qui montre comment utiliser les API Graph Peer et le fournisseur d’espace de noms PNRP (Peer Name Resolution Protocol) avec l’API Winsock 2. L’application illustre les tâches suivantes :

-   Création d’un graphique
-   Connexion à un graphique existant
-   Déconnexion d’un graphique existant
-   Énumération des entités homologues
-   Ajout d’enregistrements au graphique
-   Utilisation de connexions directes avec un graphique
-   Utilisation de l’infrastructure de notifications et d’événements avec des graphiques
-   Inscription de noms avec PNRP
-   Résolution de noms avec PNRP
-   Annulation de l’inscription des noms avec PNRP

## <a name="group-chat"></a>Conversation de groupe

L’exemple d’application de conversation de groupe est une simple application de conversation qui montre comment utiliser les API de regroupement pair et Identity Manager. L’application illustre les tâches suivantes :

-   Création d’une identité
-   Création et obtention des informations d’identité
-   Énumération des identités
-   Énumération des groupes associés à une identité
-   Création d’un groupe
-   Création d’invitations pour un groupe
-   Connexion à un groupe existant
-   Déconnexion d’un groupe existant
-   Extraction d’informations à partir des propriétés de groupe
-   Utilisation de connexions directes avec un groupe
-   Utilisation des fonctions d’énumération dans un groupe
-   Énumération des membres d’un groupe
-   Ajout d’enregistrements à un groupe
-   Utilisation de l’infrastructure de notifications et d’événements avec des groupes

## <a name="group-browser"></a>Explorateur de groupes

L’exemple d’application de l’Explorateur de groupes est un outil de gestion de groupe homologue simple qui montre comment utiliser les API de regroupement pair et Identity Manager. L’application illustre les tâches suivantes :

-   Énumération des clouds PNRP
-   Énumération des identités
-   Énumération des groupes associés à une identité
-   Création et suppression d’identités
-   Création d’un groupe et Association de ce groupe à une identité
-   Création d’une invitation et enregistrement
-   Ouverture d’une invitation et utilisation de celle-ci pour rejoindre un groupe
-   Suppression d’une identité et appartenance à un groupe
-   Connexion à un groupe existant
-   Déconnexion d’un groupe existant
-   Extraction d’informations à partir des propriétés de groupe
-   Utilisation des fonctions d’énumération dans un groupe
-   Énumération des membres d’un groupe
-   Utilisation de l’infrastructure de notifications et d’événements avec des groupes
