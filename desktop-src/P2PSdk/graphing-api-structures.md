# <a name="graphing-api-structures"></a>Structures d’API graphiques

L’API graphique utilise les structures suivantes :



| Structure                                                                 | Description                                                                                                   |
|---------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [**\_données de \_ modification du nœud d’événement d’homologue \_ \_**](/windows/desktop/api/P2P/ns-p2p-peer_event_node_change_data)    | Contient un pointeur vers les données si un événement de **modification de nœud d' \_ événement de graphique \_ \_ \_ homologue** est déclenché.                  |
| [**\_ \_ données synchronisées des événements homologues \_**](/windows/desktop/api/P2P/ns-p2p-peer_event_synchronized_data)   | Indique le type d’enregistrement qui a été synchronisé.                                                      |
| [**\_données d' \_ événement du graphique de pairs \_**](/windows/desktop/api/P2P/ns-p2p-peer_graph_event_data)                 | Contient des données associées à un événement homologue.                                                                   |
| [**\_inscription des \_ événements du graphique homologue \_**](/windows/desktop/api/P2P/ns-p2p-peer_graph_event_registration) | Spécifie les événements homologues pour lesquels une application requiert des notifications.                                        |
| [**\_Propriétés du graphique homologue \_**](/windows/desktop/api/P2P/ns-p2p-peer_graph_properties)                  | Contient des données relatives à la stratégie d’un graphique homologue, d’un ID, d’une portée et d’autres informations.                             |
| [**\_informations sur le nœud homologue \_**](/windows/desktop/api/P2P/ns-p2p-peer_node_info)                                | Contient des informations qui sont spécifiques à un nœud particulier dans un graphique homologue.                                   |
| [**\_interface de sécurité de l’homologue \_**](/windows/desktop/api/P2P/ns-p2p-peer_security_interface)              | Spécifie les interfaces de sécurité que les appels aux API de représentation graphique d’homologue utilisent pour valider, sécuriser et libérer des enregistrements. |
