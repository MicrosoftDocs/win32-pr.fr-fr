# <a name="grouping-api-enumerations"></a>Regroupement des énumérations d’API

L’API de regroupement utilise les énumérations suivantes :



| Énumération                                                                        | Description                                                                                                                                                                                                                                               |
|------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**\_ \_ schéma d’authentification du groupe homologue \_**](/windows/desktop/api/P2P/ne-p2p-peer_group_authentication_scheme)    | Définit l’ensemble des schémas d’authentification possibles qui peuvent être utilisés pour authentifier des pairs qui se joignent à un groupe homologue.                                                                                                                                           |
| [**\_type d' \_ événement de groupe pair \_**](/windows/desktop/api/P2P/ne-p2p-peer_group_event_type)                          | Contient les types d’événements homologues spécifiques qui peuvent se produire dans un groupe homologue.                                                                                                                                                                                |
| [**\_ \_ \_ indicateurs d’informations d’identification du problème du groupe homologue \_**](/windows/win32/api/p2p/ne-p2p-peer_group_issue_credential_flags) | Spécifie si les informations d’identification de l’utilisateur sont stockées dans un groupe.                                                                                                                                                                                                  |
| [**\_indicateurs de \_ propriété de groupe pair \_**](/windows/desktop/api/P2P/ne-p2p-peer_group_property_flags)                  | Spécifie les paramètres d’appartenance au groupe homologue.                                                                                                                                                                                                                 |
| [**\_État du groupe homologue \_**](/windows/desktop/api/P2P/ne-p2p-peer_group_status)                                   | Indique si le groupe homologue a des connexions présentes.                                                                                                                                                                                          |
| [**\_type de \_ modification du membre pair \_**](/windows/desktop/api/P2P/ne-p2p-peer_member_change_type)                      | Définit l’ensemble des États de présence et d’appartenance aux groupes d’homologues possibles pour un homologue.                                                                                                                                                                         |
| [**\_indicateurs de membre pair \_**](/windows/desktop/api/P2P/ne-p2p-peer_member_flags)                                   | Permet à une application de spécifier si tous les membres ou uniquement ceux présents doivent être énumérés lorsque la fonction [**PeerGroupEnumMembers**](/windows/desktop/api/P2P/nf-p2p-peergroupenummembers) est appelée, ou pour indiquer si un membre est présent ou non dans le groupe homologue. |
