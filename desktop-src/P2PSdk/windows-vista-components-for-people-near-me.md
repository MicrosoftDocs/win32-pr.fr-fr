# <a name="windows-vista-components-for-people-near-me"></a>Composants Windows Vista pour voisinage immédiat

Voisinage [immédiat](about-people-near-me.md) (PNM) utilise les composants suivants de Windows Vista :

### <a name="web-service-discovery"></a>Découverte de service Web

Le composant de détection de service Web permet la publication et la découverte de services sur le sous-réseau local. Dans PNM, un utilisateur est un service qui est publié sur le sous-réseau local. Pour plus d’informations sur la découverte de service Web, consultez [XML Web Service Discovery](/previous-versions/dotnet/netframework-4.0/fxx6cfx2(v=vs.100)).

### <a name="schannel"></a>Schannel

Le composant Schannel permet une communication chiffrée entre deux points de terminaison.

### <a name="windows-sockets"></a>Windows Sockets

Le composant Windows Sockets est utilisé par les applications pour les communications sur TCP/IP.

### <a name="next-generation-tcpip-stack"></a>Pile TCP/IP nouvelle génération

Nouvelle pile TCP/IP dans Windows Vista qui intègre IPv6 et qui est activée par défaut. Pour plus d’informations, consultez [pile TCP/IP nouvelle génération dans Windows Vista](https://www.microsoft.com/technet/itsolutions/network/tcpip/default.mspx).
