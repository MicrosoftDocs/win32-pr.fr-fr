# <a name="collaboration-api-enumerations"></a>Énumérations de l’API de collaboration

Les énumérations suivantes prennent en charge les fonctions d’infrastructure de collaboration pair à pair.



| Énumération                                                                         | Description                                                                                                                                                                                                                                                                   |
|-------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**\_ \_ type d’inscription de l’application homologue \_**](/windows/desktop/api/P2P/ne-p2p-peer_application_registration_type) | Définit le jeu d’indicateurs d’inscription d’application homologue.                                                                                                                                                                                                                       |
| [**\_type de modification d’homologue \_**](/windows/desktop/api/P2P/ne-p2p-peer_change_type)                                      | Définit l’ensemble des modifications qui peuvent être effectuées sur un objet homologue, un point de terminaison ou une application.                                                                                                                                                                                  |
| [**indicateurs de connexion d’HOMOLOGUe \_ \_**](/windows/desktop/api/P2P/ne-p2p-peer_signin_flags)                                    | Définit l’ensemble des comportements de publication de présence d’homologue disponibles lorsque l’homologue se connecte à un réseau de collaboration pair à pair.                                                                                                                                                      |
| [**\_autorisation Watch \_ pair**](/windows/desktop/api/P2P/ne-p2p-peer_watch_permission)                            | Définit le jeu d’autorisations qui représentent la présence d’un contact auprès d’un homologue à l’observation.                                                                                                                                                                                        |
| [**\_type d' \_ événement \_ collab pair**](/windows/desktop/api/P2P/ne-p2p-peer_collab_event_type)                         | Définit l’ensemble des événements qui peuvent être déclenchés sur un homologue par l’infrastructure d’événements réseau de collaboration pair à pair.                                                                                                                                                                |
| [**\_État de présence de l’homologue \_**](/windows/desktop/api/P2P/ne-p2p-peer_presence_status)                              | Définit l’ensemble des paramètres d’état de présence possibles disponibles pour un homologue qui participe à un réseau de collaboration pair à pair. Ces paramètres peuvent être définis par un point de terminaison réseau de collaboration pair à pair pour indiquer le niveau actuel de participation du contact pair à ses observateurs. |
| [**\_type de \_ réponse d’invitation d’homologue \_**](/windows/desktop/api/P2P/ne-p2p-peer_invitation_response_type)           | Définit l’ensemble des réponses reçues à une invitation pour joindre une activité de collaboration pair à pair.                                                                                                                                                                                 |
| [**\_étendue de publication d’homologue \_**](/windows/desktop/api/P2P/ne-p2p-peer_publication_scope)                          | Définit le jeu d’étendues pour la publication d’objets homologues ou de données.                                                                                                                                                                                                        |
