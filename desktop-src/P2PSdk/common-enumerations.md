# <a name="common-constants"></a>Constantes communes

L’infrastructure homologue utilise les constantes courantes suivantes.



| Constante                                                      | Description                                                   |
|---------------------------------------------------------------|---------------------------------------------------------------|
| [**\_indicateurs de connexion homologue \_**](/windows/desktop/api/P2P/ne-p2p-peer_connection_flags)      | Spécifie les types de connexions qu’un homologue peut avoir.      |
| [**État de la connexion HOMOLOGUe \_ \_**](/windows/desktop/api/P2P/ne-p2p-peer_connection_status)    | Spécifie l’état d’une connexion d’homologue directe ou voisin. |
| [**\_type de \_ modification d’enregistrement d’homologue \_**](/windows/desktop/api/P2P/ne-p2p-peer_record_change_type) | Spécifie les modifications qui peuvent se produire dans un enregistrement.             |
| [**\_indicateurs d’enregistrement d’homologue \_**](/windows/desktop/api/P2P/ne-p2p-peer_record_flags)              | Spécifie un jeu d’indicateurs pour les comportements d’enregistrement homologue.           |
