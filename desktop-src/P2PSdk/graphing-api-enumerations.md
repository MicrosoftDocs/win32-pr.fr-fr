# <a name="graphing-api-constants"></a>Graphiques, constantes d’API

L’API graphique utilise les constantes suivantes :



| Constante                                                          | Description                                                       |
|-------------------------------------------------------------------|-------------------------------------------------------------------|
| [**\_type d' \_ événement de graphique homologue \_**](/windows/desktop/api/P2P/ne-p2p-peer_graph_event_type)         | Spécifie les types d’événements homologues pour lesquels l’application doit être notifiée. |
| [**\_indicateurs de \_ propriété de graphique pair \_**](/windows/desktop/api/P2P/ne-p2p-peer_graph_property_flags) | Spécifie les propriétés d’un graphique homologue.                             |
| [**\_étendue graphique d’homologue \_**](/windows/desktop/api/P2P/ne-p2p-peer_graph_scope)                    | Spécifie l’étendue réseau d’un graphique homologue.                      |
| [**\_indicateurs d' \_ État du graphique des homologues \_**](/windows/desktop/api/P2P/ne-p2p-peer_graph_status_flags)     | Spécifie l’état actuel d’un nœud dans le graphique homologue.     |
| [**\_type de \_ modification du nœud homologue \_**](/windows/desktop/api/P2P/ne-p2p-peer_node_change_type)         | Spécifie les types de modifications d’État du graphique du nœud homologue.            |
