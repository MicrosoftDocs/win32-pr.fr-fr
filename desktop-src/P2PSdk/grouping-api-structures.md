# <a name="grouping-api-structures"></a>Regroupement des structures d’API

L’API de regroupement utilise les structures suivantes.



| Structure                                                                  | Description                                                                                                   |
|----------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [**\_informations d’identification de l’homologue \_**](/windows/desktop/api/P2P/ns-p2p-peer_credential_info)                     | Définit les informations utilisées pour obtenir et émettre les informations d’identification de sécurité d’un homologue.                                   |
| [**\_données de \_ modification des membres d’événements homologues \_ \_**](/windows/desktop/api/P2P/ns-p2p-peer_event_member_change_data) | Contient des données qui décrivent une modification de l’état d’un membre du groupe homologue.                                   |
| [**\_données d' \_ événement du groupe homologue \_**](/windows/win32/api/p2p/ns-p2p-peer_group_event_data-r1)                  | Contient des informations sur un événement de groupe homologue spécifique qui s’est produit.                                     |
| [**\_inscription des \_ événements du groupe homologue \_**](/windows/desktop/api/P2P/ns-p2p-peer_group_event_registration)  | Définit l’événement de groupe homologue particulier pour lequel un membre peut s’inscrire.                                            |
| [**\_Propriétés du groupe homologue \_**](/windows/desktop/api/P2P/ns-p2p-peer_group_properties)                   | Contient des données sur la stratégie d’appartenance d’un groupe homologue.                                                    |
| [**\_informations sur l’invitation de l’homologue \_**](/windows/desktop/api/P2P/ns-p2p-peer_invitation_info)                     | Définit des informations sur une invitation pour joindre un groupe homologue. Les invitations sont représentées sous forme de chaînes Unicode. |
| [**membre HOMOLOGUe \_**](/windows/desktop/api/P2P/ns-p2p-peer_member)                                        | Contient des informations qui décrivent un membre d’un groupe homologue.                                                 |
