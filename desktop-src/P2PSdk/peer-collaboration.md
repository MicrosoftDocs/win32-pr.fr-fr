# <a name="peer-collaboration"></a>Collaboration pair à pair

L’infrastructure de collaboration pair à pair fournit un ensemble d’interfaces de programmation d’applications qui peuvent être utilisées conjointement avec l’infrastructure homologue pour permettre l’utilisation d’applications réseau sans serveur collaboratives, telles que la messagerie instantanée et le jeu multijoueur.
