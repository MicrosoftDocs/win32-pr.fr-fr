# <a name="why-use-people-near-me"></a>Pourquoi utiliser voisinage immédiat ?

Voici quelques exemples de scénarios que les [gens près de moi](about-people-near-me.md) (PNM) activent :

-   Lancez une conversation avec une personne d’une salle de conférence pendant une réunion. Utilisez une application de conversation pour découvrir dynamiquement d’autres utilisateurs de la salle, sélectionner un utilisateur spécifique et démarrer une session de conversation pour collaborer ou échanger des informations.
-   L’API PNM autorise le partage de documents avec tous les membres de la salle de conférence. Bien que les lecteurs flash USB (Universal Serial Bus) (UFD) facilitent le transfert de documents entre les ordinateurs dans un paramètre fermé, par exemple une réunion, il faut toujours le temps et les efforts nécessaires pour connecter le disque mémoire flash USB à chaque ordinateur et copier le document dans un dossier local. Avec les applications capables de PNM, vous pouvez sélectionner les utilisateurs de la salle avec lesquels vous souhaitez partager ou envoyer un document.
-   Diffusez les informations sur le Bureau d’un ordinateur sur l’ordinateur d’un collègue pour le partage des ressources.
-   Démarrez un jeu avec quelqu’un pendant un escale sur un terminal aéroportuaire qui prend en charge la connectivité sans fil. L’API PNM permet à une application de promouvoir votre intérêt pour jouer à un jeu, découvrir les autres personnes qui souhaitent également jouer le jeu et démarrer une session de jeu.
-   Publiez les centres d’intérêt d’une conférence du secteur et découvrez ceux des autres utilisateurs à l’aide d’une connectivité sans fil. Après la découverte, les utilisateurs sont en mesure de converser ou de configurer des réunions impromptues pour échanger des idées.
