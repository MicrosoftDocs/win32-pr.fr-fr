# <a name="eaphost-registry-settings"></a>Paramètres du registre EAPHost

Les valeurs de Registre dans les clés de Registre suivantes contrôlent les aspects de la fonctionnalité d’EAPHost.



| Sous-clé                                                                 | Description                                                                                                                                                              |
|------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [BypassNegotiation](bypassnegotiation.md)                             | Détermine si la négociation des fonctionnalités entre le serveur et le client RAS se produit.<br/>                                                                              |
| [AssumePhase2Fragmentation](assumephase2fragmentation.md)             | Détermine si le serveur et le client RAS supposent la fragmentation de la phase 2.<br/>                                                                                         |
| [OverrideEAPCertificateSelection](overrideeapcertificateselection.md) | Détermine si le client remplacera le comportement de sélection par défaut du certificat de carte à puce et fournira à l’utilisateur plus de certificats à choisir.<br/> |
| [PeapServerUseAllPurposeCert](peapserveruseallpurposecert.md)         | Détermine si les certificats polyvalents sont utilisés pour l’authentification PEAP.<br/>                                                                                      |
| [PeapServerAcceptAllPurposeCert](peapserveracceptallpurposecert.md)   | Détermine si les certificats polyvalents sont acceptés pour l’authentification PEAP.<br/>                                                                                  |
| [TlsServerUseAllPurposeCert](tlsserveruseallpurposecert.md)           | Détermine si les certificats polyvalents sont utilisés pour l’authentification EAP-TLS.<br/>                                                                                   |
| [TlsServerAcceptAllPurposeCert](tlsserveracceptallpurposecert.md)     | Détermine si les certificats polyvalents sont acceptés pour l’authentification EAP-TLS.<br/>                                                                               |
