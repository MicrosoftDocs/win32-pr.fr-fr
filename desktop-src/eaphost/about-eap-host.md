# <a name="about-eaphost"></a>À propos d’EAPHost

EAPHost implémente l’ordinateur d’état EAP (Extensible Authentication Protocol) et l’infrastructure de protocole EAP, conformément à la [norme RFC 3748](https://go.microsoft.com/fwlink/p/?linkid=84063).

EAPHost prend en charge la [protection d’accès réseau](/windows/desktop/NAP/network-access-protection-start-page) (NAP) basée sur EAP.
