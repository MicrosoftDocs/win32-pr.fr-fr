# <a name="eaphost-peer-method-api-reference"></a>Informations de référence sur l’API de méthode homologue EAPHost

La documentation sur les API de méthode homologue EAPHost se compose des sections suivantes.

-   [Fonctions de configuration des méthodes homologues EAPHost](eaphost-peer-method-configuration-functions.md)
-   [Fonctions de Run-Time de la méthode homologue EAPHost](eaphost-peer-method-run-time-functions.md)
-   [Énumérations de méthodes homologues EAPHost](eap-host-peer-method-enumerations.md)
-   [Structures de méthode homologue EAPHost](eap-host-peer-method-structures.md)

Notez que ces éléments doivent être implémentés dans une DLL conforme par l’auteur de la méthode d’homologue EAP. La DLL avec l’implémentation de ces éléments d’API est chargée et appelée par EAPHost côté client.
