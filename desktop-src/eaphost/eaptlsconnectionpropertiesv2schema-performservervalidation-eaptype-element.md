# <a name="tlsextensions-v2-schema"></a>TLSExtensions (schéma v2)

L’élément **TLSExtensions (TLSExtensionsType)** permet de futures améliorations du schéma.

``` syntax
<xs:element name="TLSExtensions"
    type="TLSExtensionsType"
 />
```

## <a name="remarks"></a>Remarques

L’élément **TLSExtensions** est facultatif.
