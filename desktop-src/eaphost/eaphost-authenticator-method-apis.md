# <a name="eaphost-authenticator-method-apis"></a>API de méthode d’authentificateur EAPHost

La documentation sur les API de méthode d’authentificateur EAPHost se compose des sections suivantes.

-   [Fonctions de méthode de l’authentificateur EAPHost](eap-host-authenticator-method-functions.md)
-   [Énumérations de méthodes d’authentificateur EAPHost](eap-host-authenticator-method-enumerations.md)
-   [Structures de méthode d’authentificateur EAPHost](eap-host-authenticator-method-structures.md)

Notez que ces éléments doivent être implémentés dans une DLL conforme par l’auteur de la méthode de l’authentificateur EAP. La DLL avec l’implémentation de ces éléments d’API est chargée et appelée par EAPHost côté serveur.
