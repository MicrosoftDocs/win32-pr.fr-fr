# <a name="eaphost-peer-method-enumerations"></a>Énumérations de méthodes homologues EAPHost

Les énumérations de l’API de méthode homologue EAPHost sont les suivantes.



| Énumération                                                        | Description                                                                                                                |
|--------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------|
| [**EapCredentialType**](/windows/desktop/api/eaptypes/ns-eaptypes-eapcredential)                         | Définit les types d’informations d’identification EAP et les informations d’identification appropriées.                                                          |
| [**EapPeerMethodResponseAction**](/windows/win32/api/eapauthenticatoractiondefine/ne-eapauthenticatoractiondefine-eappeermethodresponseaction) | Définit l’ensemble des actions qu’un authentificateur EAP peut indiquer à un demandeur ou une méthode d’homologue EAP pendant l’authentification.     |
| [**EapPeerMethodResultReason**](/windows/win32/api/eapauthenticatoractiondefine/ne-eapauthenticatoractiondefine-eappeermethodresultreason)     | Définit l’ensemble de résultats d’une session d’authentification EAP retournée par une méthode d’authentificateur EAP à une méthode d’homologue EAP. |
