# <a name="eaphost-supplicant-enumerations"></a>Énumérations du demandeur EAPHost

Les énumérations de l’API du demandeur EAPHost sont les suivantes.



| Énumération                                                                  | Description                                                                                                        |
|------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------|
| [**\_type de \_ données de l’interface utilisateur interactive EAP \_ \_**](/windows/desktop/api/eaptypes/ne-eaptypes-eap_interactive_ui_data_type)     | Spécifie le jeu de types de données de contexte d’interface utilisateur interactif fourni à certains appels d’API demandeur.    |
| [**TYPE de propriété de la \_ méthode EAP \_ \_**](/windows/desktop/api/EapTypes/ne-eaptypes-eap_method_property_type)              | Windows 7 ou version ultérieure : spécifie le type de propriété de la méthode.                                                            |
| [**\_type de \_ valeur de propriété \_ \_ de la méthode EAP**](/windows/desktop/api/EapTypes/ne-eaptypes-eap_method_property_value_type) | Windows 7 ou version ultérieure : spécifie le type de données d’une valeur de propriété de méthode.                                            |
| [**\_État d’authentification EAPHOST \_**](/windows/desktop/api/eaphostpeertypes/ne-eaphostpeertypes-eaphost_auth_status)                         | Définit l’ensemble des valeurs possibles d’état de session d’authentification EAP.                                              |
| [**EapHostPeerAuthParams**](/windows/win32/api/eaphostpeertypes/ne-eaphostpeertypes-eaphostpeerauthparams)                       | Définit l’ensemble des valeurs de paramètre d’authentification possibles.                                                       |
| [**EapHostPeerMethodResultReason**](/windows/win32/api/eaphostpeertypes/ne-eaphostpeertypes-eaphostpeermethodresultreason)       | Définit l’ensemble des raisons possibles qui décrivent les résultats retournés par une méthode EAP à un demandeur.           |
| [**EapHostPeerResponseAction**](/windows/win32/api/eaphostpeertypes/ne-eaphostpeertypes-eaphostpeerresponseaction)               | Définit l’ensemble d’actions qu’un authentificateur ou une méthode homologue EAP peut indiquer à un demandeur pendant l’authentification. |
| [**État d’isolement \_**](/windows/desktop/api/eaphostpeertypes/ne-eaphostpeertypes-isolation_state)                                  | Définit l’ensemble des valeurs d’état d’isolement possibles d’un ordinateur.                                                  |
