# <a name="peapextensions-eaptype-element-user-properties"></a>Élément PeapExtensions (EapType) (propriétés utilisateur)

L’élément **PeapExtensions (EapType)** permet de futures améliorations du schéma.

``` syntax
<xs:element name="PeapExtensions"
    type="PeapExtensionsType"
 />
```

L’élément **PeapExtensions** est défini par l’élément [**EapType**](mspeapuserpropertiesv1schema-eaptype-element.md) .

## <a name="remarks"></a>Notes

L’élément **PeapExtensions (EapType)** est facultatif.
