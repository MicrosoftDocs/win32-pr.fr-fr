# <a name="eaphost-supplicant-callbacks"></a>Rappels du demandeur EAPHost

Les fonctions de rappel de l’API du demandeur EAPHost sont les suivantes.



| Rappel                                           | Description                                                                                                                                                                                                                                     |
|----------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**NotificationHandler**](/previous-versions/windows/desktop/api) | Un prototype de rappel qui informe le demandeur qu’il y a une modification dans la déclaration d’intégrité (SoH) et que la réauthentification d’une connexion système de [protection d’accès réseau](/windows/desktop/NAP/network-access-protection-start-page) (NAP) est requise. |
