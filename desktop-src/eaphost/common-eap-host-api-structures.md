# <a name="common-eaphost-api-structures"></a>Structures d’API EAPHost courantes

Le tableau suivant répertorie les structures communes à toutes les technologies d’API EAPHost.



| Structure                                                                | Description                                                                                                                                                                                                             |
|--------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**\_attribut EAP**](/windows/desktop/api/eaptypes/ns-eaptypes-eap_attribute)                                  | Contient un attribut EAP.                                                                                                                                                                                              |
| [**\_attributs EAP**](/windows/desktop/api/eaptypes/ns-eaptypes-eap_attributes)                                | Contient un tableau d’attributs EAP.                                                                                                                                                                                    |
| [**\_tableau de \_ champs d’entrée de configuration \_ EAP \_**](/windows/desktop/api/eaptypes/ns-eaptypes-eap_config_input_field_array) | Contient un ensemble de structures de [**\_ \_ \_ \_ données de champ d’entrée de configuration EAP**](/windows/desktop/api/eaptypes/ns-eaptypes-eap_config_input_field_data) qui contiennent collectivement les données de champ d’entrée utilisateur obtenues à partir d’une boîte de dialogue de configuration de l’authentification unique (SSO) EAP. |
| [**\_données du \_ champ d’entrée de la configuration EAP \_ \_**](/windows/desktop/api/eaptypes/ns-eaptypes-eap_config_input_field_data)   | Contient les données associées à un champ d’entrée unique dans une boîte de dialogue de configuration de l’authentification unique (SSO) EAP.                                                                                                              |
| [**\_erreur EAP**](/windows/desktop/api/eaptypes/ns-eaptypes-eap_error)                                          | Contient des données relatives à une erreur qui s’est produite pendant une opération EAPHost.                                                                                                                                                 |
| [**\_informations sur la méthode EAP \_**](/windows/desktop/api/eaptypes/ns-eaptypes-eap_method_info)                             | Contient des données spécifiques pour une méthode EAP.                                                                                                                                                                               |
| [**\_informations de méthode EAP \_ \_ ex**](/windows/desktop/api/eaptypes/ns-eaptypes-eap_method_info_ex)                      | Windows Vista avec Service Pack 1 (SP1) ou version ultérieure : contient des données spécifiques pour une méthode EAP.                                                                                                                             |
| [**\_tableau d' \_ informations de méthode EAP \_**](/windows/desktop/api/eaptypes/ns-eaptypes-eap_method_info_array)                | Contient des informations sur les méthodes EAP installées sur l’ordinateur client.                                                                                                                                                   |
| [**\_tableau d’informations de méthode EAP \_ \_ \_ ex**](/windows/desktop/api/eaptypes/ns-eaptypes-eap_method_info_array_ex)         | Windows Vista avec SP1 ou version ultérieure : contient des informations sur toutes les méthodes EAP installées sur l’ordinateur client.                                                                                                    |
| [**\_type de méthode EAP \_**](/windows/desktop/api/eaptypes/ns-eaptypes-eap_method_type)                             | Contient les données de type, d’identification et d’auteur pour une méthode EAP.                                                                                                                                                       |
| [**\_type EAP**](/windows/desktop/api/eaptypes/ns-eaptypes-eap_type)                                            | Contient les données d’identification de type et de fournisseur pour une méthode EAP.                                                                                                                                                         |
| [**EapPacket**](/windows/win32/api/eapmethodtypes/ns-eapmethodtypes-eappacket)                                           | Contient un paquet de données opaques envoyées au cours d’une session d’authentification EAP.                                                                                                                                             |
