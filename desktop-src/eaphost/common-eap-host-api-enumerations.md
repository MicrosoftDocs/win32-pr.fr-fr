# <a name="common-eaphost-api-enumerations"></a>Énumérations d’API EAPHost courantes

Les énumérations suivantes sont communes à toutes les technologies d’API EAPHost.



| Énumération                                                                  | Description                                                                                   |
|------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------|
| [**\_type d’attribut EAP \_**](/windows/desktop/api/eaptypes/ne-eaptypes-eap_attribute_type)                           | Définit l’ensemble des types d’attributs EAP possibles disponibles sur une entité d’authentification.        |
| [**\_type de \_ champ d’entrée de configuration \_ EAP \_**](/windows/desktop/api/eaptypes/ne-eaptypes-eap_config_input_field_type)       | Définit un ensemble de types de champs d’entrée disponibles lors de l’interrogation des informations d’identification de l’utilisateur. |
| [**TYPE de propriété de la \_ méthode EAP \_ \_**](/windows/desktop/api/EapTypes/ne-eaptypes-eap_method_property_type)              | Définit le jeu de propriétés de méthode EAP possibles.                                            |
| [**\_type de \_ valeur de propriété \_ \_ de la méthode EAP**](/windows/desktop/api/EapTypes/ne-eaptypes-eap_method_property_value_type) | Définit l’ensemble des types de données possibles pour les propriétés de la méthode.                             |
| [**EapCode**](/windows/win32/api/eapmethodtypes/ne-eapmethodtypes-eapcode)                                                   | Définit l’ensemble des types de paquets EAP.                                                          |
