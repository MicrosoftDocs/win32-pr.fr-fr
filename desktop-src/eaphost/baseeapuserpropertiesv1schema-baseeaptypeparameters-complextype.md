# <a name="baseeaptypeparameters-complex-type---user-properties"></a>Type complexe BaseEapTypeParameters-propriétés de l’utilisateur

Le type complexe **BaseEapTypeParameters** est un élément de base de l’élément spécifique à la méthode qui apparaît dans l’élément dérivé du [schéma baseeapuserpropertiesv1](baseeapuserpropertiesv1schema-schema.md).

``` syntax
<xs:complexType name="BaseEapTypeParameters"
    abstract="true"
 />
```

## <a name="remarks"></a>Notes

**BaseEapTypeParameters** est un élément abstrait. Un élément dérivé de **BaseEapTypeParameters** doit être utilisé dans les documents d’instance.
