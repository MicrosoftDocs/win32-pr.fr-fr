# <a name="direct-annotation"></a>Annotation directe

Cette section fournit des informations sur l’utilisation de l’annotation directe.

## <a name="summary"></a>Résumé

L’annotation directe est la forme la plus simple d’annotation dynamique. Elle est utilisée pour remplacer la valeur d’un ou d’un nombre limité de propriétés d’un élément accessible. Elle est particulièrement utile pour les éléments accessibles dont les propriétés annotées ne dépendent pas de l’état du contrôle et ne nécessitent pas la prise en charge supplémentaire fournie par l’annotation de la [carte de valeur](value-map-annotation.md) et l' [annotation du serveur](server-annotation.md).

## <a name="when-to-use-this-technique"></a>Quand utiliser cette technique

Utilisez cette technique lorsque vous souhaitez remplacer la valeur d’un ou d’un nombre limité de propriétés pour un élément accessible.

Pour plus d’informations, consultez [utilisation de l’annotation directe](using-direct-annotation.md).
