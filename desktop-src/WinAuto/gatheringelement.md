# <a name="gatheringelement"></a>GatheringElement

## <a name="text"></a>Texte

Cet élément a été collecté

## <a name="type"></a>Type

Avertissement

## <a name="description"></a>Description

Ce message n’est pas un avertissement ou une erreur et ne nécessite aucune action. Elle indique que les informations de l’élément ont été collectées et stockées dans le fichier journal.
