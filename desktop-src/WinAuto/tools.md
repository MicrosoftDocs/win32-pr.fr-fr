# <a name="tools"></a>Outils

Vous pouvez utiliser les outils suivants pour tester les implémentations de Microsoft Active Accessibility. Pour plus d’informations sur les outils de test, consultez [outils de test](testing-tools.md).



| Outil                                                     | Description                                                                                                                                                                           |
|----------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Accessible Event Watcher](accessible-event-watcher.md) | Journalise les [winEvents](winevents-infrastructure.md) à mesure qu’ils se produisent dans le système.                                                                                                         |
| [Inspecter](inspect-objects.md)                           | Identifie les problèmes liés à l’implémentation des propriétés et des méthodes dans les objets accessibles de Microsoft Active Accessibility et dans les éléments d’automatisation de Microsoft UI Automation. |
