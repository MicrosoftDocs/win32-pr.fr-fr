# <a name="custom-properties-events-and-control-patterns"></a>Propriétés personnalisées, événements et modèles de contrôle

L’API Microsoft UI Automation spécifie un ensemble de propriétés, des modèles de contrôle et des événements de base prédéfinis. Toutefois, les applications ne se limitent pas à l’utilisation de ces spécifications prédéfinies. Les fonctionnalités d’extensibilité d’UI Automation permettent aux tiers d’introduire des propriétés, des événements et des modèles de contrôle personnalisés, mutuellement convenus pour prendre en charge les nouveaux éléments d’interface utilisateur et les scénarios d’application. Les fournisseurs UI Automation et les clients peuvent commencer à utiliser les propriétés personnalisées, les événements et les modèles de contrôle immédiatement sans qu’il soit nécessaire de mettre à jour l’infrastructure d’automatisation d’interface utilisateur de base.
