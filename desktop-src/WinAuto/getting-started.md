# <a name="getting-started-microsoft-active-accessibility"></a>Prise en main (Microsoft Active Accessibility)

Microsoft Active Accessibility est un ensemble d’interfaces COM (Component Object Model) et d’éléments d’API qui offrent un moyen fiable pour exposer et collecter des informations sur les éléments de l’interface utilisateur Microsoft Windows et le contenu Web. À l’aide de ces informations, les fournisseurs de technologies d’assistance peuvent représenter l’interface utilisateur dans d’autres formats, tels que Speech ou braille, et les applications de commande vocale et de contrôle peuvent manipuler à distance l’interface. Microsoft Active Accessibility s’appuie sur la technologie Windows et peut être utilisé conjointement avec les contrôles Windows et d’autres applications Windows.
