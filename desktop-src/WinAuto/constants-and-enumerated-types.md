# <a name="constants-active-accessibility-user-interface-services"></a>Constantes (Active Accessibility les services d’interface utilisateur)

Cette section contient des informations sur les constantes utilisées avec Microsoft Active Accessibility.

## <a name="in-this-section"></a>Contenu de cette section



| Rubrique                                                               | Description                                                                                                                                                                                                                                                                                                                                                                           |
|---------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Constantes d’événement**](event-constants.md)<br/>               | Cette rubrique décrit les événements qui sont générés par le système d’exploitation et par les applications serveur.<br/>                                                                                                                                                                                                                                                                     |
| [**Constantes de navigation**](navigation-constants.md)<br/>     | Cette rubrique décrit les valeurs constantes, définies dans oleacc. h, qui indiquent le sens *spatial* (haut, bas, gauche et droit) ou *logique* (premier enfant, dernier, suivant et précédent) observé lorsque les clients utilisent [**IAccessible :: accNavigate**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-accnavigate) pour naviguer d’un élément d’interface utilisateur à un autre dans le même conteneur.<br/> |
| [**Identificateurs d'objet**](object-identifiers.md)<br/>         | Cette rubrique décrit les identificateurs d’objets Microsoft Active Accessibility, les valeurs 32 bits qui identifient les *catégories* d’objets accessibles dans une fenêtre.<br/>                                                                                                                                                                                                                |
| [**Rôles d’objet**](object-roles.md)<br/>                     | Cette rubrique décrit les valeurs constantes utilisées pour décrire les rôles de différents objets d’interface utilisateur dans une application.<br/>                                                                                                                                                                                                                                                               |
| [**Constantes d’état d’objet**](object-state-constants.md)<br/> | Cette rubrique décrit les valeurs constantes utilisées pour décrire l’état des objets dans une interface utilisateur d’application.<br/>                                                                                                                                                                                                                                                                       |
| [**Constantes SELFLAG**](selflag.md)<br/>                     | Cette rubrique décrit les valeurs constantes utilisées pour spécifier la façon dont un objet accessible est sélectionné ou qui prend le focus.<br/>                                                                                                                                                                                                                                                     |
