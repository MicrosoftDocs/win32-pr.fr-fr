# <a name="event-constants-winuserh"></a>Constantes d’événement (winuser. h)

Cette rubrique décrit les événements qui sont générés par le système d’exploitation et par les applications serveur. Les constantes sont répertoriées par ordre alphabétique.

Avant d’utiliser ces événements, les applications clientes doivent utiliser l' [Observateur d’événements accessible](accessible-event-watcher.md) pour vérifier que ces événements sont utilisés par les éléments d’interface utilisateur.

Pour plus d’informations sur les événements en général, voir [qu’est-ce que WinEvents ?](what-are-winevents.md) et les [événements au niveau système et au niveau](system-level-and-object-level-events.md)de l’objet. Pour plus d’informations sur les événements envoyés par le système, consultez [annexe A : informations de référence sur les éléments d’interface utilisateur pris en charge](appendix-a--supported-user-interface-elements-reference.md).
