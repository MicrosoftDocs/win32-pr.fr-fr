# <a name="taborderinquestion"></a>TabOrderInQuestion

## <a name="text"></a>Texte

L’ordre de tabulation peut ne pas suivre les instructions Windows standard

## <a name="type"></a>Type

Informationnel

## <a name="description"></a>Description

L’utilisation de la navigation au clavier standard (Tab ou Maj + Tab) expose un ordre de tabulation non standard. Par exemple, selon les paramètres du système local, l’ordre de tabulation imite généralement l’ordre de lecture.

Ce problème provoque des problèmes pour les personnes qui reposent sur un lecteur d’écran et un clavier pour la navigation, car le parcours des éléments peut être erratique et imprévisible.
