# <a name="labeling-owner-drawn-controls"></a><span data-ttu-id="51212-103">Étiquetage des contrôles de Owner-Drawn</span><span class="sxs-lookup"><span data-stu-id="51212-103">Labeling Owner-Drawn Controls</span></span>

<span data-ttu-id="51212-104">Les développeurs d’applications doivent définir une étiquette, également appelée texte de légende, pour tous les contrôles owner-drawn, même si l’étiquette n’est pas visible à l’écran.</span><span class="sxs-lookup"><span data-stu-id="51212-104">Application developers should define a label, also known as caption text, for all owner-drawn controls, even if the label is not visible on the screen.</span></span> <span data-ttu-id="51212-105">L’étiquette doit précéder le contrôle qu’elle étiquette et doit se trouver dans l’élément suivant dans l’ordre de tabulation.</span><span class="sxs-lookup"><span data-stu-id="51212-105">The label must come immediately before the control that it is labeling, and must be in the next item in the tab order.</span></span>

<span data-ttu-id="51212-106">Par exemple, si vous définissez une étiquette lors de la création d’un bouton owner-drawn dans lequel un graphique s’affiche à la place du texte de légende normal, les applications clientes obtiennent l’étiquette et l’utilisent pour identifier le contrôle.</span><span class="sxs-lookup"><span data-stu-id="51212-106">For example, if you define a label when creating an owner-drawn button in which a graphic is displayed instead of the normal caption text, client applications get the label and use it to identify the control.</span></span>
