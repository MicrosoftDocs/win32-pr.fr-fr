# <a name="labeling-owner-drawn-controls"></a>Étiquetage des contrôles de Owner-Drawn

Les développeurs d’applications doivent définir une étiquette, également appelée texte de légende, pour tous les contrôles owner-drawn, même si l’étiquette n’est pas visible à l’écran. L’étiquette doit précéder le contrôle qu’elle étiquette et doit se trouver dans l’élément suivant dans l’ordre de tabulation.

Par exemple, si vous définissez une étiquette lors de la création d’un bouton owner-drawn dans lequel un graphique s’affiche à la place du texte de légende normal, les applications clientes obtiennent l’étiquette et l’utilisent pour identifier le contrôle.
