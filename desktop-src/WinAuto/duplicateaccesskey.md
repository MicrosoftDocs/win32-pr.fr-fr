# <a name="duplicateaccesskey"></a>DuplicateAccessKey

## <a name="text"></a>Texte

Il existe un élément d’automatisation frère qui a la même clé d’accès/touche accélérateur

## <a name="type"></a>Type

Error

## <a name="description"></a>Description

La touche d’accès rapide assignée à l’élément est également assignée à un autre élément dans la cible de vérification.

Ce problème provoque des problèmes pour les personnes qui reposent sur le clavier pour la navigation, car plusieurs éléments peuvent être appelés ou le focus du cycle à l’aide du même raccourci ou touche accélérateur.
