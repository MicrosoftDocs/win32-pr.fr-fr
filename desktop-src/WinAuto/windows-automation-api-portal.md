# <a name="windows-accessibility-api-overview"></a>Windows Vue d’ensemble de l’API d’accessibilité

cette section fournit des vues d’ensemble de haut niveau et une référence d’api détaillée pour l’api d’automatisation de microsoft Windows 3,0, qui inclut l’implémentation Windows de la [spécification microsoft UI automation](./uiauto-specandcommunitypromise.md)et microsoft Active Accessibility (fonctionnalité héritée introduite dans Windows 95 comme complément de plateforme).

Nous décrivons les similitudes et les différences entre Microsoft Active Accessibility et UI Automation, les composants et les fonctionnalités qui permettent aux deux technologies de travailler ensemble, et fournissent des instructions pour le choix de la technologie à implémenter pour votre scénario spécifique.
