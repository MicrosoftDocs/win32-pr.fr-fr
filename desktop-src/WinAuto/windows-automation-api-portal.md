# <a name="windows-accessibility-api-overview"></a>Vue d’ensemble de l’API d’accessibilité Windows

Cette section fournit des vues d’ensemble de haut niveau et une référence d’API détaillée pour l’API Microsoft Windows Automation 3,0, qui inclut l’implémentation Windows de la [spécification Microsoft UI Automation](./uiauto-specandcommunitypromise.md)et Microsoft Active Accessibility (fonctionnalité héritée introduite dans Windows 95 en tant que complément de plateforme).

Nous décrivons les similitudes et les différences entre Microsoft Active Accessibility et UI Automation, les composants et les fonctionnalités qui permettent aux deux technologies de travailler ensemble, et fournissent des instructions pour le choix de la technologie à implémenter pour votre scénario spécifique.
