# <a name="multiple-annotations-on-a-single-accessible-element"></a>Plusieurs annotations sur un seul élément accessible

Actuellement, un élément accessible ne prend en charge qu’une seule annotation par propriété. Les annotations consécutives sur une propriété remplacent et remplacent les annotations antérieures de cette propriété.
