# <a name="methodreturnedunexpectedhresult"></a>MethodReturnedUnexpectedHResult

## <a name="text"></a>Texte

{0} retourné {1} , mais la valeur de retour attendue doit avoir été {2}

## <a name="type"></a>Type

Avertissement

## <a name="description"></a>Description

Un HRESULT inattendu a été retourné lors de la tentative de récupération de tous les enfants de l’élément.

## <a name="possible-causes"></a>Causes possibles

Implémentation MSAA incorrecte ou non valide.
