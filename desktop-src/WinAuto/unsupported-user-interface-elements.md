# <a name="unsupported-user-interface-elements"></a>Éléments d’interface utilisateur non pris en charge

Microsoft Active Accessibility ne prend pas en charge les contrôles communs listés dans le tableau suivant.



| Nom du contrôle                  | Nom de la classe de fenêtre   |
|-------------------------------|---------------------|
| Contrôles de sélecteur de date et heure | DATETIMEPICK, \_ classe |
| Contrôles de pagineur                | \_PAGESCROLLER WC    |
| Contrôles Rebar                | REBARCLASSNAME      |



 

Microsoft Active Accessibility ne prend pas en charge les contrôles prédéfinis et les éléments de fenêtre indiqués dans le tableau suivant.



| Nom de la classe de fenêtre | Description                                                 |
|-------------------|-------------------------------------------------------------|
| DDEMLEvent        | Windows NT : classe pour les événements DDEML                      |
| Message           | Windows 2000 et versions ultérieures : la classe pour une fenêtre de message uniquement |
| \#32772           | Windows NT : classe pour les titres des icônes                       |
