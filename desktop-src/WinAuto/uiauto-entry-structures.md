# <a name="ui-automation-structures"></a>Structures UI Automation

Cette section décrit les structures utilisées par les applications clientes UI Automation et le fournisseur UI Automation pour Microsoft Win32.

## <a name="in-this-section"></a>Dans cette section



| Structure                                                                            | Description                                                                                   |
|--------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------|
| [**ExtendedProperty**](/windows/desktop/api/UIAutomationClient/ns-uiautomationclient-extendedproperty)<br/>                       | Contient des informations sur une propriété étendue. <br/>                                  |
| [**UIAutomationEventInfo**](/windows/desktop/api/UIAutomationCore/ns-uiautomationcore-uiautomationeventinfo)<br/>       | Contient des informations sur un événement personnalisé.<br/>                                         |
| [**UIAutomationMethodInfo**](/windows/desktop/api/UIAutomationCore/ns-uiautomationcore-uiautomationmethodinfo)<br/>     | Contient des informations sur une méthode prise en charge par un modèle de contrôle personnalisé.<br/> |
| [**UIAutomationParameter**](/windows/desktop/api/UIAutomationCore/ns-uiautomationcore-uiautomationparameter)<br/>       | Contient des informations sur un paramètre d’un modèle de contrôle personnalisé.<br/>                |
| [**UIAutomationPatternInfo**](/windows/desktop/api/UIAutomationCore/ns-uiautomationcore-uiautomationpatterninfo)<br/>   | Contient des informations sur un modèle de contrôle personnalisé.<br/>                               |
| [**UIAutomationPropertyInfo**](/windows/desktop/api/UIAutomationCore/ns-uiautomationcore-uiautomationpropertyinfo)<br/> | Contient des informations sur une propriété personnalisée.<br/>                                      |
| [**UiaChangeInfo**](/windows/desktop/api/UIAutomationCore/ns-uiautomationcore-uiachangeinfo)<br/>                                    | Contient des données relatives à une modification d’UI Automation qui s’est produite.<br/>                          |
| [**UiaPoint**](/windows/desktop/api/UIAutomationCore/ns-uiautomationcore-uiapoint)<br/>                                 | Contient les coordonnées d’un point. <br/>                                              |
| [**UiaRect**](/windows/desktop/api/UIAutomationCore/ns-uiautomationcore-uiarect)<br/>                                   | Contient la position et la taille d’un rectangle, en coordonnées d’écran.<br/>              |
