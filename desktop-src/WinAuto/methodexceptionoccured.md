# <a name="methodexceptionoccured"></a>MethodExceptionOccured

## <a name="text"></a>Texte

Une exception s’est produite \\ « {0} \\ »

## <a name="type"></a>Type

Error

## <a name="description"></a>Description

Une exception inattendue ou HRESULT a été rencontré. Certaines exceptions/valeurs HRESULT sont valides, mais dans ce cas, les résultats ne sont pas attendus. Cette erreur doit être corrigée.
