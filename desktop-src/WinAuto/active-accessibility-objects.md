# <a name="active-accessibility-objects"></a>Objets Active Accessibility

Dans Microsoft Active Accessibility terminologie, il existe des *objets accessibles* et des *éléments simples*. Bien que la plupart des applications contiennent des objets accessibles, ils sont plus courants que les éléments simples. Cette section définit et décrit les objets accessibles et les éléments simples et fournit des exemples d’interface utilisateur appropriés.
