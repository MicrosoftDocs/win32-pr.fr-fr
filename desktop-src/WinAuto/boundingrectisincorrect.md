# <a name="boundingrectisincorrect"></a>BoundingRectIsIncorrect

## <a name="text"></a>Texte

Le {0} bord du rectangle signalé est incohérent avec le test d’atteinte {1} du Rect au point {2} .

## <a name="type"></a>Type

Erreur

## <a name="description"></a>Description

Le rectangle englobant de l’élément n’est pas cohérent avec les résultats de test de positionnement attendus. Par exemple, le test de positionnement d’un point situé en dehors du rectangle englobant de l’élément ne retourne pas un élément différent.
