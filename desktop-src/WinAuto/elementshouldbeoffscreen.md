# <a name="elementshouldbeoffscreen"></a>ElementShouldBeOffScreen

## <a name="text"></a>Texte

L’élément doit être hors écran

## <a name="type"></a>Type

Avertissement

## <a name="description"></a>Description

Le rectangle englobant de l’élément est coupé par le rectangle englobant de son élément parent. L’élément doit être ignoré et pris en compte en dehors des limites du bureau.
