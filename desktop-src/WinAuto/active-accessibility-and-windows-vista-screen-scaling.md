## <a name="scaling-in-windows-vista"></a><span data-ttu-id="f6d45-108">Mise à l’échelle dans Windows Vista</span><span class="sxs-lookup"><span data-stu-id="f6d45-108">Scaling in Windows Vista</span></span>

<span data-ttu-id="f6d45-122">La mise à l’échelle de l’écran pose de nouveaux défis pour les applications qui sont concernées d’une manière ou d’une autre par les coordonnées d’écran.</span><span class="sxs-lookup"><span data-stu-id="f6d45-122">Screen scaling creates new challenges for applications that are concerned in any way with screen coordinates.</span></span> <span data-ttu-id="f6d45-123">L’écran contient maintenant deux systèmes de coordonnées : l’un physique et l’autre logique.</span><span class="sxs-lookup"><span data-stu-id="f6d45-123">The screen now contains two coordinate systems: physical and logical.</span></span> <span data-ttu-id="f6d45-124">Les coordonnées physiques d’un point correspondent au décalage réel en pixels par rapport au point d’origine en haut à gauche.</span><span class="sxs-lookup"><span data-stu-id="f6d45-124">The physical coordinates of a point are the actual offset in pixels from the top left of the origin.</span></span> <span data-ttu-id="f6d45-125">Les coordonnées logiques correspondent aux décalages tels qu’ils seraient si les pixels eux-mêmes étaient mis à l’échelle.</span><span class="sxs-lookup"><span data-stu-id="f6d45-125">The logical coordinates are the offsets as they would be if the pixels themselves were scaled.</span></span>
