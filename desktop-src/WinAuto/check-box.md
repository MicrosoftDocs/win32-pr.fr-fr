# <a name="check-box-msaa-ui-element-reference"></a>Case à cocher (référence des éléments d’interface utilisateur MSAA)

> [!Note]  
> Cette rubrique décrit les objets case **à cocher** à des fins de référence des éléments d’interface utilisateur MSAA. La procédure de création d’objets de case à **cocher** dans différentes infrastructures d’interface utilisateur n’est pas décrite ici. Consultez la documentation de référence sur les API pour l’infrastructure d’interface utilisateur que vous utilisez.

 

Les cases à cocher sont utilisées pour activer ou désactiver une ou plusieurs fonctionnalités ou options d’un jeu, généralement dans une boîte de dialogue. En règle générale, une case à cocher contient une petite zone avec du texte adjacent. Lorsqu’une option est sélectionnée, une coche apparaît dans la zone.

Le nom de la classe de fenêtre d’une case à cocher est « BUTTON ».
