# <a name="list-box-msaa-ui-element-reference"></a>Zone de liste (référence des éléments d’interface utilisateur MSAA)

> [!Note]  
> Cette rubrique décrit les objets de **zone de liste** à des fins de référence d’élément d’interface utilisateur MSAA. La procédure de création d’objets de **zone de liste** dans différentes infrastructures d’interface utilisateur n’est pas décrite ici. Consultez la documentation de référence sur les API pour l’infrastructure d’interface utilisateur que vous utilisez.

 

Les zones de liste affichent une liste à partir de laquelle un utilisateur peut sélectionner un ou plusieurs éléments. Une zone de liste est créée avec une barre de défilement verticale, une barre de contrôle horizontale, ou les deux.

Le nom de la classe de fenêtre d’une zone de liste est « LISTBOX ».

## <a name="iaccessible-methods"></a>Méthodes IAccessible

Les zones de liste prennent en charge les méthodes [**IAccessible**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible) suivantes :



| Méthode                                                                    | Commentaires                                                                                                                                                                                                                                                                   |
|---------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**accDoDefaultAction**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-accdodefaultaction) | L’objet de zone de liste lui-même n’a pas de propriété **DefaultAction** ; Toutefois, [**accDoDefaultAction**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-accdodefaultaction) double-clique sur les éléments de la zone de liste.                                                                                    |
| [**accHitTest**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-acchittest)                 | La méthode [**accHitTest**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-acchittest) retourne zéro si le point testé est dans la zone de liste, mais pas sur un élément de la zone de liste. Si le point testé se trouve sur un élément de la zone de liste, **accHitTest** retourne l’index de l’élément de zone de liste. |
| [**accLocation**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-acclocation)               |                                                                                                                                                                                                                                                                            |
| [**accNavigate**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-accnavigate)               |                                                                                                                                                                                                                                                                            |
| [**accSelect**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-accselect)                   |                                                                                                                                                                                                                                                                            |
