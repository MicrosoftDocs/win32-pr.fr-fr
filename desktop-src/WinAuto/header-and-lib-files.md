# <a name="header-and-lib-files"></a>Fichiers d’en-tête et lib

Les fichiers d’en-tête et de bibliothèque suivants prennent en charge le développement d’applications Microsoft Active Accessibility.



| Fichier         | Description                                                                                                                                                                  |
|--------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Winuser. h    | fichier d’en-tête C et C++ pour l’utilisation de contrôles Windows                                                                                                                             |
| Oleacc. h     | Fichier d’en-tête C et C++ contenant les définitions requises par les clients et serveurs Microsoft Active Accessibility                                                                    |
| Winver. h     | Fichier d’en-tête C et C++ pour les informations de version                                                                                                                                |
| Oleacc. lib   | Importer la bibliothèque pour Oleacc.dll                                                                                                                                                |
| Oleacc.dll   | Bibliothèque de liens dynamiques qui fournit le temps d’exécution de Microsoft Active Accessibility, gère les demandes des clients Microsoft Active Accessibility et les éléments d’interface utilisateur communs des proxies |
| Oleaccrc.dll | Bibliothèque de liens dynamiques des ressources multilingues utilisée par Oleacc.dll                                                                                                                |
