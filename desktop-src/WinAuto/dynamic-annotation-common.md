# <a name="dynamic-annotation"></a>Annotation dynamique

Les annotations dynamiques étaient auparavant disponibles uniquement pour les serveurs et les clients Microsoft Active Accessibility. La technique est désormais disponible pour tous les clients UI Automation, ainsi que pour les implémentations de serveur MSAA et UI Automation Provider. En outre, les annotations apportées aux contrôles personnalisés à l’aide de l’annotation dynamique (les trois techniques) sont accessibles aux clients UI Automation.

Pour plus d’informations, consultez [API d’annotation dynamique](dynamic-annotation-api.md).
