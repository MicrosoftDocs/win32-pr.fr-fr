# <a name="helptopic-property"></a>Propriété HelpTopic

La propriété **HelpTopic** obtient les détails utilisés avec la fonction [WinHelp](/windows/win32/api/winuser/nf-winuser-winhelpa) de Microsoft Win32 pour afficher une rubrique d’aide associée à l’objet. Tous les objets ne prennent pas en charge cette propriété.

La propriété **HelpTopic** est récupérée en appelant [**IAccessible :: obten \_ accHelpTopic**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-get_acchelptopic).
