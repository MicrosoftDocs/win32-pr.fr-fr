# <a name="shortcuts-for-exposing-custom-user-interface-elements"></a>Raccourcis pour exposer des éléments d’interface utilisateur personnalisés

Cette section décrit les raccourcis que les développeurs d’applications peuvent utiliser pour exposer certains genres d’éléments d’interface utilisateur personnalisés. Les techniques décrites dans cette section simplifient l’implémentation de l’interface [**IAccessible**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible) ou fournissent une alternative à l’exposition d’informations sans implémenter **IAccessible**.

> [!Note]  
> Vous devez envisager d’utiliser une forme d’annotation dynamique (directe, carte de valeur ou serveur) avant de tenter les techniques décrites dans cette section. Pour plus d’informations, consultez [API d’annotation dynamique](dynamic-annotation-api.md).
