# <a name="scope-and-longevity-of-identity-string"></a>Portée et longévité de la chaîne d’identité

Une chaîne d’identité identifie une instance particulière d’un élément d’interface utilisateur. Plusieurs instances d’un élément accessible auront des chaînes d’identité différentes ; par conséquent, il n’est généralement pas utile de conserver une chaîne d’identité.
