# <a name="cc-developers-guide"></a>Guide du développeur C/C++

Cette section fournit des informations sur le fonctionnement de Microsoft Active Accessibility du point de vue d’un développeur d’applications C ou C++. Il aborde les problèmes spécifiques aux développeurs client et serveur.

Notez qu’un *client* est une aide à l’accessibilité ou une autre application qui utilise Microsoft Active Accessibility pour obtenir des informations sur les éléments de l’interface utilisateur. Un *serveur* est une application qui utilise Microsoft Active Accessibility pour fournir des informations sur les éléments de l’interface utilisateur.
