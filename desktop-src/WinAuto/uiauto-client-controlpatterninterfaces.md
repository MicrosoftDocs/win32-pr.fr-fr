# <a name="control-pattern-interfaces-for-clients"></a>Interfaces de modèle de contrôle pour les clients

Cette section décrit les interfaces de modèle de contrôle pour les applications clientes UI Automation non managées.

## <a name="in-this-section"></a>Dans cette section



| Interface                                                                                                | Description                                                                                                                                                                                                                                                                                                                                             |
|----------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IUIAutomationAnnotationPattern**](/windows/desktop/api/uiautomationclient/nn-uiautomationclient-iuiautomationannotationpattern)<br/>           | Fournit l’accès aux propriétés d’une annotation dans un document.<br/>                                                                                                                                                                                                                                                                            |
| [**IUIAutomationCustomNavigationPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationcustomnavigationpattern)<br/>   | Expose une méthode pour prendre en charge l’accès par un client UI Automation Microsoft aux contrôles qui prennent en charge un ordre de navigation personnalisé.<br/>                                                                                                                                                                                                                   |
| [**IUIAutomationDockPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationdockpattern)<br/>                           | Fournit l’accès à un contrôle qui permet aux éléments enfants d’être disposés horizontalement et verticalement, l’un par rapport à l’autre.<br/>                                                                                                                                                                                                                 |
| [**IUIAutomationDragPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationdragpattern)<br/>                           | Fournit l’accès aux informations exposées par un fournisseur UI Automation pour un élément qui peut être glissé dans le cadre d’une opération de glisser-déplacer.<br/>                                                                                                                                                                                                  |
| [**IUIAutomationDropTargetPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationdroptargetpattern)<br/>               | Fournit l’accès aux informations de glisser-déplacer exposées par un fournisseur UI Automation pour un élément qui peut être la cible de déplacement d’une opération de glisser-déplacer.<br/>                                                                                                                                                                                    |
| [**IUIAutomationExpandCollapsePattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationexpandcollapsepattern)<br/>       | Fournit l’accès à un contrôle qui peut être visuellement développé pour afficher du contenu, et réduire pour masquer le contenu.<br/>                                                                                                                                                                                                                                      |
| [**IUIAutomationGridItemPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationgriditempattern)<br/>                   | Fournit l’accès à un contrôle enfant dans un conteneur de style grille qui prend en charge l’interface [**IUIAutomationGridPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationgridpattern) .<br/>                                                                                                                                                                                |
| [**IUIAutomationGridPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationgridpattern)<br/>                           | Fournit l’accès à un contrôle qui agit comme un conteneur pour une collection de contrôles enfants organisés dans un système de coordonnées logiques à deux dimensions qui peut être parcouru par ligne et par colonne.<br/>                                                                                                                                          |
| [**IUIAutomationInvokePattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationinvokepattern)<br/>                       | Expose une méthode qui permet à une application cliente d’appeler l’action d’un contrôle (généralement un bouton).<br/>                                                                                                                                                                                                                                   |
| [**IUIAutomationItemContainerPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationitemcontainerpattern)<br/>         | Expose une méthode qui récupère un élément d’un conteneur, tel qu’une liste virtuelle.<br/>                                                                                                                                                                                                                                                            |
| [**IUIAutomationLegacyIAccessiblePattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationlegacyiaccessiblepattern)<br/> | Expose des méthodes et des propriétés qui permettent aux clients UI Automation de récupérer des informations sur l’interface utilisateur à partir de serveurs Microsoft Active Accessibility (MSAA).<br/>                                                                                                                                                                                              |
| [**IUIAutomationMultipleViewPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationmultipleviewpattern)<br/>           | Fournit l’accès à un contrôle qui peut basculer entre plusieurs représentations des mêmes informations ou jeux de contrôles enfants.<br/>                                                                                                                                                                                                              |
| [**IUIAutomationObjectModelPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationobjectmodelpattern)<br/>             | Fournit l’accès au modèle objet sous-jacent implémenté par un contrôle ou une application. <br/>                                                                                                                                                                                                                                                     |
| [**IUIAutomationPatternHandler**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iuiautomationpatternhandler)<br/>                     | Retourne un objet wrapper d’API cliente et des demandes de propriétés et de méthodes non marshalées à une instance de fournisseur réelle.<br/>                                                                                                                                                                                                                            |
| [**IUIAutomationPatternInstance**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iuiautomationpatterninstance)<br/>                   | Représente un objet de modèle de contrôle. Le wrapper d’API client utilise cette interface pour implémenter tous les appels de méthode et de propriété en termes de méthodes [**GetProperty**](/windows/desktop/api/UIAutomationCore/nf-uiautomationcore-iuiautomationpatterninstance-getproperty) et [**CallMethod**](/windows/desktop/api/UIAutomationCore/nf-uiautomationcore-iuiautomationpatterninstance-callmethod) .<br/>                                          |
| [**IUIAutomationRangeValuePattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationrangevaluepattern)<br/>               | Fournit l’accès à un contrôle qui présente une plage de valeurs.<br/>                                                                                                                                                                                                                                                                                |
| [**IUIAutomationScrollItemPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationscrollitempattern)<br/>               | Expose une méthode qui permet de placer un élément dans une vue à défilement dans une partie visible de la vue.<br/>                                                                                                                                                                                                                                    |
| [**IUIAutomationScrollPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationscrollpattern)<br/>                       | Fournit l’accès à un contrôle qui agit comme un conteneur à défilement pour une collection d’éléments enfants.<br/>                                                                                                                                                                                                                                         |
| [**IUIAutomationSelectionItemPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationselectionitempattern)<br/>         | Fournit l’accès aux éléments enfants sélectionnables d’un contrôle conteneur qui prend en charge [**IUIAutomationSelectionPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationselectionpattern).<br/>                                                                                                                                                                            |
| [**IUIAutomationSelectionPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationselectionpattern)<br/>                 | Fournit l’accès à un contrôle qui contient des éléments enfants sélectionnables. Les enfants de cet élément prennent en charge [**IUIAutomationSelectionItemPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationselectionitempattern).<br/>                                                                                                                                               |
| [**IUIAutomationSelectionPattern2**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationselectionpattern2)<br/>               | Étend l’interface [**IUIAutomationSelectionPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationselectionpattern) pour fournir des informations sur les éléments sélectionnés.<br/>                                                                                                                                                                                             |
| [**IUIAutomationSpreadsheetItemPattern**](/windows/desktop/api/uiautomationclient/nn-uiautomationclient-iuiautomationspreadsheetitempattern)<br/> | Permet à une application cliente de récupérer des informations sur un élément (cellule) dans une feuille de calcul.<br/>                                                                                                                                                                                                                                                  |
| [**IUIAutomationSpreadsheetPattern**](/windows/desktop/api/uiautomationclient/nn-uiautomationclient-iuiautomationspreadsheetpattern)<br/>         | Permet à une application cliente d’accéder aux éléments (cellules) d’une feuille de calcul.<br/>                                                                                                                                                                                                                                                                   |
| [**IUIAutomationStylesPattern**](/windows/desktop/api/uiautomationclient/nn-uiautomationclient-iuiautomationstylespattern)<br/>                   | Permet aux clients UI Automation de récupérer les styles visuels associés à un élément dans un document.<br/>                                                                                                                                                                                                                                        |
| [**IUIAutomationSynchronizedInputPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationsynchronizedinputpattern)<br/> | Fournit l’accès au clavier ou à la souris d’un contrôle.<br/>                                                                                                                                                                                                                                                                                 |
| [**IUIAutomationTableItemPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationtableitempattern)<br/>                 | Fournit l’accès à un élément enfant dans un conteneur qui prend en charge [**IUIAutomationTablePattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationtablepattern). <br/>                                                                                                                                                                                                      |
| [**IUIAutomationTablePattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationtablepattern)<br/>                         | Fournit l’accès à un contrôle qui agit comme un conteneur pour une collection d’éléments enfants. Les enfants de cet élément prennent en charge les [**IUIAutomationTableItemPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationtableitempattern) et sont organisés en un système de coordonnées logiques à deux dimensions qui peut être parcouru par ligne et par colonne.<br/>                       |
| [**IUIAutomationTextChildPattern**](/windows/desktop/api/uiautomationclient/nn-uiautomationclient-iuiautomationtextchildpattern)<br/>             | Fournit l’accès à un contrôle textuel (ou à un objet incorporé dans du texte) qui est un enfant ou un descendant d’un autre contrôle textuel. <br/>                                                                                                                                                                                                           |
| [**IUIAutomationTextEditPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationtexteditpattern)<br/>                   | Fournit l’accès à un contrôle qui modifie du texte, par exemple un contrôle qui effectue une correction automatique ou permet une composition d’entrée via un éditeur de méthode d’entrée (IME).<br/>                                                                                                                                                                      |
| [**IUIAutomationTextPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationtextpattern)<br/>                           | Fournit l’accès à un contrôle qui contient du texte.<br/>                                                                                                                                                                                                                                                                                             |
| [**IUIAutomationTextPattern2**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationtextpattern2)<br/>                         | Étend l’interface [**IUIAutomationTextPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationtextpattern) . <br/>                                                                                                                                                                                                                                                  |
| [**IUIAutomationTextRange**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationtextrange)<br/>                               | Fournit l’accès à une étendue de texte continu dans un conteneur qui prend en charge l’interface [**IUIAutomationTextPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationtextpattern) . Les applications clientes peuvent utiliser l’interface [**IUIAutomationTextRange**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationtextrange) pour sélectionner, comparer et récupérer des objets incorporés à partir de l’étendue de texte.<br/> |
| [**IUIAutomationTextRange2**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationtextrange2)<br/>                             | Étend l’interface [**IUIAutomationTextRange**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationtextrange) pour permettre aux clients UI Automation d’appeler par programme des menus contextuels.<br/>                                                                                                                                                                              |
| [**IUIAutomationTextRange3**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationtextrange3)<br/>                             | Étend l’interface [**IUIAutomationTextRange2**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationtextrange2) pour prendre en charge un accès plus rapide aux données de texte enrichi sous-jacentes sur une plage de texte.<br/>                                                                                                                                                                           |
| [**IUIAutomationTextRangeArray**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationtextrangearray)<br/>                     | Représente une collection d’objets [**IUIAutomationTextRange**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationtextrange) .<br/>                                                                                                                                                                                                                                          |
| [**IUIAutomationTogglePattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationtogglepattern)<br/>                       | Fournit l’accès à un contrôle qui peut parcourir un ensemble d’États et maintenir un état après qu’il a été défini.<br/>                                                                                                                                                                                                                                   |
| [**IUIAutomationTransformPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationtransformpattern)<br/>                 | Permet d’accéder à un contrôle qui peut être déplacé, redimensionné ou pivoté.<br/>                                                                                                                                                                                                                                                                         |
| [**IUIAutomationTransformPattern2**](/windows/desktop/api/uiautomationclient/nn-uiautomationclient-iuiautomationtransformpattern2)<br/>           | Étend l’interface [**IUIAutomationTransformPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationtransformpattern) pour permettre aux clients UI Automation d’accéder par programme à la fonctionnalité de zoom de fenêtre d’affichage d’un contrôle.<br/>                                                                                                                              |
| [**IUIAutomationValuePattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationvaluepattern)<br/>                         | Fournit l’accès à un contrôle qui contient une valeur qui ne couvre pas une plage et qui peut être représentée sous la forme d’une chaîne.<br/>                                                                                                                                                                                                                       |
| [**IUIAutomationVirtualizedItemPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationvirtualizeditempattern)<br/>     | Représente un élément virtualisé, qui est un élément représenté par un élément Automation d’espace réservé dans l’arborescence UI Automation.<br/>                                                                                                                                                                                                          |
| [**IUIAutomationWindowPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationwindowpattern)<br/>                       | Permet d’accéder aux fonctionnalités fondamentales d’une fenêtre.<br/>                                                                                                                                                                                                                                                                                |
