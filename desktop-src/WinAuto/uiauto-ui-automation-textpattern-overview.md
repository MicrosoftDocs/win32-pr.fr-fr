# <a name="ui-automation-support-for-textual-content"></a>Prise en charge d’UI Automation pour le contenu textuel

Cette section décrit comment utiliser l’automatisation de l’interface utilisateur de Microsoft pour exposer le contenu textuel, y compris les attributs de format et de style, des contrôles de texte dans les plateformes prises en charge par UI Automation. Les contrôles de texte incluent, sans s’y limiter, les documents, les info-bulles, les contrôles d’édition et les contrôles RichEdit.
