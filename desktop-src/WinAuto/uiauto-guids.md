# <a name="deprecated-guids"></a>GUID déconseillés

Cette rubrique décrit les GUID utilisés avec la fonction [**UiaLookupId**](/windows/desktop/api/UIAutomationCoreApi/nf-uiautomationcoreapi-uialookupid) .

> [!Note]  
> La fonction [**UiaLookupId**](/windows/desktop/api/UIAutomationCoreApi/nf-uiautomationcoreapi-uialookupid) et les GUID décrits dans cette rubrique sont déconseillés. Au lieu de cela, les applications doivent utiliser les identificateurs décrits dans les sections suivantes :

 

-   [Identificateurs de modèle de contrôle](uiauto-controlpattern-ids.md)
-   [Identificateurs de type de contrôle](uiauto-controltype-ids.md)
-   [Identificateurs d’événements](uiauto-event-ids.md)
-   [Identificateurs de propriété](uiauto-entry-propids.md)
-   [Identificateurs d’attribut de texte](uiauto-textattribute-ids.md)

Cette section répertorie les GUID suivants :

-   [GUID PATTERNID](#patternid-guids)
-   [GUID CONTROLTYPEID](#controltypeid-guids)
-   [GUID d’ID d’événement](#eventid-guids)
-   [GUID de PROPERTYID](#propertyid-guids)
-   [GUID TEXTATTRIBUTEID](#textattributeid-guids)
