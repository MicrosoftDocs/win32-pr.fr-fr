Cette section contient des informations de référence sur les structures d’API d’agrandissement.

## <a name="in-this-section"></a>Contenu de cette section

| Rubrique | Description |
|:---|:---|
| [**MAGCOLOREFFECT**](/windows/win32/api/magnification/ns-magnification-magcoloreffect)<br/>  | Décrit une matrice de transformation des couleurs utilisée par un contrôle loupe pour appliquer un effet de couleur au contenu d’écran agrandi.<br/> |
| [***MAGIMAGEHEADER** _ _](/windows/win32/api/magnification/ns-magnification-magimageheader)<br/> | _ * déconseillé dans Windows 7 et versions ultérieures * *<br/>Décrit un format d’image. |
| [**MAGTRANSFORM**](/windows/win32/api/magnification/ns-magnification-magtransform)<br/> | Décrit une matrice de transformation qu’un contrôle loupe utilise pour agrandir le contenu de l’écran.<br/> |
