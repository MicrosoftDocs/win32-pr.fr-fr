# <a name="boundingrectnothittestable"></a>BoundingRectNotHitTestable

## <a name="text"></a>Texte

Le rectangle signalé ne semble pas contenir un point où cet élément peut être récupéré avec le test de positionnement.

## <a name="type"></a>Type

Avertissement

## <a name="description"></a>Description

Le rectangle englobant de l’élément n’est pas accessible par le test de positionnement. Par exemple, il se trouve en dehors des coordonnées d’écran physiques (ou de la région visible) du bureau.
