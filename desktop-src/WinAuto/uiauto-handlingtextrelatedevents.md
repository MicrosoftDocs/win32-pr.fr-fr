# <a name="handling-text-related-events"></a>Gestion des événements liés au texte

Cette rubrique décrit les événements déclenchés par les contrôles textuels lorsque le contenu est modifié, sélectionné ou désélectionné.


Les contrôles textuels déclenchent un événement de [**\_ \_ TextChangedEventId de texte UIA**](uiauto-event-ids.md) chaque fois que le contenu textuel change. Ils déclenchent l’événement [**\_ \_ TextSelectionChangedEventId de texte UIA**](uiauto-event-ids.md) chaque fois que du contenu textuel est sélectionné ou désélectionné. Pour plus d’informations, consultez [vue d’ensemble des événements UI Automation](uiauto-eventsoverview.md) et [abonnement à des événements UI Automation](uiauto-eventsforclients.md).
