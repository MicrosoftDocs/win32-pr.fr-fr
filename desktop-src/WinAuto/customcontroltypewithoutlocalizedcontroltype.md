# <a name="customcontroltypewithoutlocalizedcontroltype"></a>CustomControlTypeWithoutLocalizedControlType

## <a name="text"></a>Texte

L’élément n’a pas la propriété LocalizedControlType définie

## <a name="type"></a>Type

Erreur

## <a name="description"></a>Description

La propriété [**LocalizedControlType**](/windows/desktop/api/UIAutomationClient/nf-uiautomationclient-iuiautomationelement-get_currentlocalizedcontroltype) de l’élément n’est pas définie.

La plupart des lecteurs d’écran lisent le type de contrôle localisé pour décrire la nature de l’élément. Si le type de contrôle localisé n’est pas défini, un lecteur d’écran revient à lire le type de contrôle principal, qui est toujours en anglais et peut ne pas avoir de sens pour tous les utilisateurs.
