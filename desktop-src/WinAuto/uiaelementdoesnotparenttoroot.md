# <a name="uiaelementdoesnotparenttoroot"></a>UiaElementDoesNotParentToRoot

## <a name="text"></a>Texte

La chaîne parente de l’élément ne passe pas à l’élément racine

## <a name="type"></a>Type

Avertissement

## <a name="description"></a>Description

La navigation à partir de l’élément donné vers la racine de l’arborescence d’interface utilisateur est rompue.

Ce problème peut provoquer des résultats imprévisibles pour une application qui tente de naviguer dans l’arborescence de l’interface utilisateur.

## <a name="possible-causes"></a>Causes possibles

Implémentation incorrecte de la hiérarchie enfant parente de l’arborescence de l’interface utilisateur.
