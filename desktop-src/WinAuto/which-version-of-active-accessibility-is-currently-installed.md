# <a name="determine-the-installed-version-of-active-accessibility"></a>Déterminer la version installée de Active Accessibility

Pour savoir quelle version de Microsoft Active Accessibility (MSAA) est installée sur votre système, vous devez vérifier la version du fichier Oleacc.dll dans le dossier system32 sur votre disque dur. Le tableau suivant mappe la version de Microsoft Active Accessibility à la version Oleacc.dll.

| Version de Active Accessibility     | Version de Oleacc.dll                                                                       |
|----------------------------------|------------------------------------------------------------------------------------------|
| 1.0                              | 4.01.0970.0                                                                              |
| 1.2                              | 4.02.2900.0                                                                              |
| 1.3                              | 4.02.3100.0                                                                              |
| 2.0                              | 5.0.0.0                                                                                  |
| 3,0 (API Windows Automation 3,0) | 7.0. x. y, où *x* est le numéro de build et *y* est le numéro QFE (Quick Fix Engineering). |
