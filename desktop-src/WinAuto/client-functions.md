# <a name="client-functions"></a>Fonctions clientes

Cette section contient des informations sur les fonctions clientes utilisées avec Microsoft Active Accessibility.

## <a name="in-this-section"></a>Contenu de cette section



| Rubrique                                                                             | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
|-----------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**AccessibleChildren**](/windows/desktop/api/Oleacc/nf-oleacc-accessiblechildren)<br/>                       | Récupère l’ID enfant ou [**IDispatch**](idispatch-interface.md) de chaque enfant dans un objet conteneur accessible.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| [**AccessibleObjectFromEvent**](/windows/desktop/api/Oleacc/nf-oleacc-accessibleobjectfromevent)<br/>         | Récupère l’adresse de l’interface [**IAccessible**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible) pour l’objet qui a généré l’événement qui est actuellement traité par la fonction de raccordement d’événement du client.<br/>                                                                                                                                                                                                                                                                                                                                                                             |
| [**AccessibleObjectFromPoint**](/windows/desktop/api/Oleacc/nf-oleacc-accessibleobjectfrompoint)<br/>         | Récupère l’adresse du pointeur d’interface [**IAccessible**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible) pour l’objet affiché à un point spécifié sur l’écran.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                       |
| [**AccessibleObjectFromWindow**](/windows/desktop/api/Oleacc/nf-oleacc-accessibleobjectfromwindow)<br/>       | Récupère l’adresse de l’interface spécifiée pour l’objet associé à la fenêtre spécifiée.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| [**GetOleaccVersionInfo**](/windows/desktop/api/Oleacc/nf-oleacc-getoleaccversioninfo)<br/>                   | Récupère le numéro de version et le numéro de build du fichier Microsoft Active Accessibility Oleacc.dll.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| [**GetProcessHandleFromHwnd**](getprocesshandlefromhwnd.md)<br/>           | Récupère un handle de processus à partir d’un handle de fenêtre.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| [**GetRoleText**](/windows/desktop/api/Oleacc/nf-oleacc-getroletexta)<br/>                                     | Récupère la chaîne localisée qui décrit le rôle de l’objet pour la valeur de rôle spécifiée.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| [**GetStateText**](/windows/desktop/api/Oleacc/nf-oleacc-getstatetexta)<br/>                                   | Récupère une chaîne localisée qui décrit l’état d’un objet pour un seul indicateur de bit d’État prédéfini. Étant donné que les valeurs d’État sont une combinaison d’un ou plusieurs indicateurs de bits, les clients appellent cette fonction plusieurs fois pour récupérer toutes les chaînes d’État.<br/>                                                                                                                                                                                                                                                                                                                      |
| [**SetWinEventHook**](/windows/desktop/api/Winuser/nf-winuser-setwineventhook)<br/>                             | Définit une fonction de raccordement d’événement pour une plage d’événements.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| [**UnhookWinEvent**](/windows/desktop/api/Winuser/nf-winuser-unhookwinevent)<br/>                               | Supprime une fonction de raccordement d’événement créée par un appel précédent à [**SetWinEventHook**](/windows/desktop/api/Winuser/nf-winuser-setwineventhook).<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| [**WindowFromAccessibleObject**](/windows/desktop/api/Oleacc/nf-oleacc-windowfromaccessibleobject)<br/>       | Récupère le handle de fenêtre qui correspond à une instance particulière d’une interface [**IAccessible**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible) .<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| [*WinEventProc fonction de rappel*](/windows/desktop/api/Winuser/nc-winuser-wineventproc)<br/> | Fonction de rappel (ou de raccordement) définie par l’application que le système appelle en réponse aux événements générés par un objet accessible. La fonction de raccordement traite les notifications d’événements comme requis. Les clients installent la fonction de raccordement et demandent des types spécifiques de notifications d’événements en appelant [**SetWinEventHook**](/windows/desktop/api/Winuser/nf-winuser-setwineventhook).<br/> Le type [**WINEVENTPROC**](/previous-versions/windows/desktop/legacy/dd373882(v=vs.85)) définit un pointeur vers cette fonction de rappel. [*WinEventProc*](/windows/desktop/api/winuser/nc-winuser-wineventproc) est un espace réservé pour le nom de la fonction définie par l’application.<br/> |
