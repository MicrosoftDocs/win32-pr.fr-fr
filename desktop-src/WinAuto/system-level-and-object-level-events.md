<span data-ttu-id="1ebff-112">Les classes d’événements système et de niveau objet sont toutes les deux générées par le système d’exploitation et les applications serveur.</span><span class="sxs-lookup"><span data-stu-id="1ebff-112">Both the system- and object-level classes of events are generated by the operating system and server applications.</span></span> <span data-ttu-id="1ebff-113">Le système d’exploitation génère des événements au niveau du système et de l’objet pour les scénarios suivants :</span><span class="sxs-lookup"><span data-stu-id="1ebff-113">The operating system generates system-level and object-level events for the following scenarios:</span></span>

<span data-ttu-id="1ebff-117">Les applications serveur génèrent des événements de niveau système pour les objets personnalisés qui répliquent des objets système, tels que des menus personnalisés et des barres de défilement.</span><span class="sxs-lookup"><span data-stu-id="1ebff-117">Server applications generate system-level events for custom objects that replicate system objects, such as custom menus and scroll bars.</span></span>
