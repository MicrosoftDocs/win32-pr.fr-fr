# <a name="interfaces-active-accessibility-user-interface-services"></a>Interfaces (Active Accessibility les services d’interface utilisateur)

Cette section contient des informations sur les interfaces utilisées avec Microsoft Active Accessibility.

## <a name="in-this-section"></a>Contenu de cette section



| Rubrique                                                                                | Description                                                                                                                                                                                          |
|--------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IAccessible**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible)<br/>                                        | Expose des méthodes et des propriétés qui rendent un élément d’interface utilisateur et ses enfants accessibles aux applications clientes.<br/>                                                                     |
| [**IAccessibleWindowlessSite**](/windows/desktop/api/oleacc/nn-oleacc-iaccessiblewindowlesssite)<br/> | Un site de contrôle ActiveX Microsoft implémente cette interface pour activer un contrôle ActiveX sans fenêtre qui a une implémentation Microsoft Active Accessibility pour exprimer son accessibilité. <br/> |
| [**IAccIdentity**](/windows/desktop/api/oleacc/nn-oleacc-iaccidentity)<br/>                                      | Expose une méthode qui fournit un identificateur unique pour un élément accessible.<br/>                                                                                                             |
| [**IAccPropServer**](/windows/desktop/api/oleacc/nn-oleacc-iaccpropserver)<br/>                                  | Expose une méthode qui récupère une valeur de propriété pour un élément accessible.<br/>                                                                                                               |
| [**IAccPropServices**](/windows/desktop/api/oleacc/nn-oleacc-iaccpropservices)<br/>                              | Expose des méthodes pour annoter des éléments accessibles et pour manipuler des chaînes d’identité. <br/>                                                                                                |
| [**IAccessibleHandler**](/windows/desktop/api/oleacc/nn-oleacc-iaccessiblehandler)<br/>                          | Expose une méthode qui récupère un élément accessible à partir d’un ID d’objet.<br/>                                                                                                                  |
