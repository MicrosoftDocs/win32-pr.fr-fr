# <a name="landmark-type-identifiers"></a>Identificateurs de type d’éléments de repère

Cette rubrique décrit les constantes nommées utilisées pour identifier les types repère Microsoft UI Automation.

<dl> <dt>

<span id="UIA_CustomLandmarkTypeId"></span><span id="uia_customlandmarktypeid"></span><span id="UIA_CUSTOMLANDMARKTYPEID"></span>**UIA \_ CustomLandmarkTypeId**
</dt> <dd> <dl> <dt>

80000
</dt> <dt>



Indique qu’une chaîne personnalisée doit être utilisée pour décrire le repère à l’aide de la valeur de [**UIA \_ LocalizedLandmarkTypePropertyId**](uiauto-automation-element-propids.md).


</dt> </dl> </dd> <dt>

<span id="UIA_FormLandmarkTypeId"></span><span id="uia_formlandmarktypeid"></span><span id="UIA_FORMLANDMARKTYPEID"></span>**UIA \_ FormLandmarkTypeId**
</dt> <dd> <dl> <dt>

80001
</dt> <dt>



Indique que le repère est lié aux éléments de type de formulaire généralement utilisés pour l’entrée de données.


</dt> </dl> </dd> <dt>

<span id="UIA_MainLandmarkTypeId"></span><span id="uia_mainlandmarktypeid"></span><span id="UIA_MAINLANDMARKTYPEID"></span>**UIA \_ MainLandmarkTypeId**
</dt> <dd> <dl> <dt>

80002
</dt> <dt>



Indique que le repère est pour le contenu principal et non secondaire ou une priorité moindre.


</dt> </dl> </dd> <dt>

<span id="UIA_NavigationLandmarkTypeId"></span><span id="uia_navigationlandmarktypeid"></span><span id="UIA_NAVIGATIONLANDMARKTYPEID"></span>**UIA \_ NavigationLandmarkTypeId**
</dt> <dd> <dl> <dt>

80003
</dt> <dt>



Indique que le repère est lié aux éléments de type de navigation.


</dt> </dl> </dd> <dt>

<span id="UIA_SearchLandmarkTypeId"></span><span id="uia_searchlandmarktypeid"></span><span id="UIA_SEARCHLANDMARKTYPEID"></span>**UIA \_ SearchLandmarkTypeId**
</dt> <dd> <dl> <dt>

80004
</dt> <dt>



Indique que le repère est lié aux éléments de type de recherche.


</dt> </dl> </dd> </dl>
