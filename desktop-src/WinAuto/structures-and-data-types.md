# <a name="structures-and-data-types"></a>Structures et types de données

Cette section contient des informations sur les structures et les types de données utilisés avec Microsoft Active Accessibility.

## <a name="in-this-section"></a>Contenu de cette section



| Rubrique                                             | Description                                                                                                                                                              |
|---------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**MSAAMENUINFO**](/windows/win32/api/oleacc/ns-oleacc-msaamenuinfo)<br/>   | Utilisé par les développeurs de serveur pour exposer les noms des éléments de menu owner-drawn.<br/>                                                                                      |
| [**HWINEVENTHOOK**](hwineventhook.md)<br/> | Utilisé pour déclarer une fonction de raccordement d’événement de fenêtre.<br/>                                                                                                                 |
| [**WINEVENTPROC**](/previous-versions/windows/desktop/legacy/dd373882(v=vs.85))<br/>   | Utilisé pour déclarer un pointeur vers une fonction de rappel définie par l’application que le système appelle en réponse aux événements qui sont générés par un objet accessible.<br/> |
