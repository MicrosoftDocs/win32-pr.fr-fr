# <a name="newdevh"></a>Newdev. h

Cette section contient des rubriques de référence sur l’en-tête newdev. h.

## <a name="in-this-section"></a>Contenu de cette section



| Rubrique                                                                                     | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
|-------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**DiInstallDevice**](/windows/desktop/api/Newdev/nf-newdev-diinstalldevice)<br/>                                     | La fonction **DiInstallDevice** installe un pilote spécifié préinstallé dans le [magasin de pilotes](/windows-hardware/drivers/install/driver-store) sur un périphérique spécifié qui est présent dans le système.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| [**DiInstallDriver**](/windows/desktop/api/Newdev/nf-newdev-diinstalldrivera)<br/>                                     | La fonction **DiInstallDriver** préinstalle un pilote dans le [magasin de pilotes](/windows-hardware/drivers/install/driver-store) , puis installe le pilote sur les appareils présents dans le système que le pilote prend en charge.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| [**DiRollbackDriver**](/windows/desktop/api/Newdev/nf-newdev-dirollbackdriver)<br/>                                   | La fonction **DiRollbackDriver** restaure le pilote qui est installé sur un périphérique spécifié.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| [**DiShowUpdateDevice**](/windows/desktop/api/Newdev/nf-newdev-dishowupdatedevice)<br/>                               | La fonction **DiShowUpdateDevice** affiche l’Assistant Mise à jour du matériel pour un appareil spécifié.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| [**DiUninstallDevice**](/windows/desktop/api/Newdev/nf-newdev-diuninstalldevice)<br/>                                 | La fonction [**DiUninstallDevice**](/windows/desktop/api/Newdev/nf-newdev-diuninstalldevice) désinstalle un appareil et supprime son nœud d’appareil ([devnode](/windows-hardware/drivers/)) du système. Cela diffère de l’utilisation de [**SetupDiCallClassInstaller**](/windows/desktop/api/Setupapi/nf-setupapi-setupdicallclassinstaller) avec le code [**DIF \_ Remove**](/windows-hardware/drivers/install/dif-remove) , car il tente de désinstaller le nœud d’appareil en plus des devnodes enfants présents au moment de l’appel.<br/> Avant Windows 8, les appareils enfants qui ne sont pas présents au moment de l’appel ne seront pas désinstallés. Toutefois, à compter de Windows 8, tous les appareils enfants qui ne sont pas présents au moment de l’appel seront désinstallés.<br/> |
| [**UpdateDriverForPlugAndPlayDevices**](/windows/desktop/api/Newdev/nf-newdev-updatedriverforplugandplaydevicesa)<br/> | À partir d’un fichier INF et d’un [ID matériel](/windows-hardware/drivers/install/hardware-ids), la fonction **UpdateDriverForPlugAndPlayDevices** installe les pilotes mis à jour pour les périphériques qui correspondent à l’ID de matériel. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
