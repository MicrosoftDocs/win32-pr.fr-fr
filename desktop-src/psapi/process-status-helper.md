# <a name="process-status-api"></a>API État du processus

Le process Status Application Programming Interface (PSAPI) est une bibliothèque d’assistance qui vous permet d’obtenir plus facilement des informations sur les processus et les pilotes de périphérique. Pour plus d'informations, voir les rubriques suivantes :

-   [À propos de PSAPIi](about-psapi.md)
-   [À l’aide de PSAPI](using-psapi.md)
-   [Référence PSAPI](psapi-reference.md)

Ces fonctions sont disponibles dans Psapi.dll.

Les mêmes informations sont généralement disponibles via les données de performances du Registre. Pour plus d’informations, consultez [Compteurs de performances](/windows/desktop/PerfCtrs/performance-counters-portal).
