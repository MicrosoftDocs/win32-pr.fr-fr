# <a name="allowincell-attribute"></a>Attribut AllowInCell

Détermine si une forme peut être placée dans une table. En lecture/écriture. **VgTriState**.

**S’applique à**

[Forme](shape-element--vml.md)

**Syntaxe de balise**

v :*Element* o :allowincell = "*expression*" >

**Remarques**

Utilisé par Microsoft Word. La valeur par défaut est **False**.
