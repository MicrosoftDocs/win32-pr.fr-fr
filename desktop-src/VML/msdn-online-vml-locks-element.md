# <a name="vml-locks-element"></a>Élément Locks VML

Cette rubrique décrit VML, une fonctionnalité déconseillée à partir de Windows Internet Explorer 9. Les pages Web et les applications qui reposent sur VML doivent être migrées vers SVG ou d’autres normes largement prises en charge.

> [!Note]  
> Depuis le 2011 décembre, cette rubrique a été archivée. Par conséquent, il n’est plus activement conservé. Pour plus d’informations, consultez [contenu archivé](/previous-versions/windows/internet-explorer/ie-developer/). Pour obtenir des informations, des recommandations et des conseils relatifs à la version actuelle de Windows Internet Explorer, consultez le [Centre de développement Internet Explorer](https://msdn.microsoft.com/ie/).

 

Définit un verrou pour une forme.

Les attributs suivants modifient un verrou.



| Attribut                                                    | Description                                                                 |
|--------------------------------------------------------------|-----------------------------------------------------------------------------|
| [AdjustHandles](msdn-online-vml-adjusthandles-attribute.md) | Détermine si les handles d’une forme peuvent être modifiés.                    |
| [AspectRatio](msdn-online-vml-aspectratio-attribute.md)     | Détermine si les proportions d’une forme peuvent être modifiées par un éditeur. |
| [Rognage](msdn-online-vml-cropping-attribute.md)           | Détermine si le rognage est autorisé dans un éditeur.                   |
| [Ext](ext-attribute--lock--vml.md)                          | Définit le comportement des actions de verrouillage pour un éditeur graphique.             |
| [Regroupement](msdn-online-vml-grouping-attribute.md)           | Détermine si les formes peuvent être regroupées dans un éditeur.                      |
| [Position](position-attribute--lock--vml.md)                | Détermine si la position d’une forme est verrouillée dans un éditeur.          |
| [Rotation](rotation-attribute--lock--vml.md)                | Détermine si la rotation des formes est autorisée dans un éditeur.         |
| [Sélection](msdn-online-vml-selection-attribute.md)         | Détermine si la forme peut être sélectionnée dans un éditeur.                    |
| [Typedeforme](msdn-online-vml-shapetype-attribute.md)         | Détermine si le type de formes automatiques peut être modifié par un éditeur.         |
| [Text](msdn-online-vml-text-attribute.md)                   | Détermine si le texte attaché à une forme peut être modifié.              |
| [Derniers](msdn-online-vml-vertices-attribute.md)           | Détermine si les vertex d’un chemin d’accès peuvent être modifiés par un éditeur.      |



 

**Remarques**

Cet élément doit être défini dans un élément [Shape](shape-element--vml.md) .

L’élément **Locks** est une Extension de Microsoft Office à Vml.
