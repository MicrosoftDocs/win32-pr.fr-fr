# <a name="vml-borderrightcolor-attribute"></a>BorderRightColor VML (attribut)

cette rubrique décrit VML, une fonctionnalité déconseillée à partir de Windows Internet Explorer 9. Les pages Web et les applications qui reposent sur VML doivent être migrées vers SVG ou d’autres normes largement prises en charge.

> [!Note]  
> Depuis le 2011 décembre, cette rubrique a été archivée. Par conséquent, il n’est plus activement conservé. Pour plus d’informations, consultez [contenu archivé](/previous-versions/windows/internet-explorer/ie-developer/). pour obtenir des informations, des recommandations et des conseils relatifs à la version actuelle de Windows internet explorer, consultez le [centre de développement internet explorer](https://msdn.microsoft.com/ie/).

 

Couleur de bordure droite d’une forme Inline. En lecture/écriture. **VgColor**.

**S’applique à**

[Forme](shape-element--vml.md)

**Syntaxe de balise**

<v : *Element* borderRightColor = " *expression* " >

**Remarques**

Utilisé par Microsoft Word.

Ces valeurs comprennent :



| Valeur | Description                                                                         |
|-------|-------------------------------------------------------------------------------------|
| Null  | Aucune couleur de bordure. Par défaut.                                                           |
| color | Couleur utilisée pour la bordure.                                                              |
| Cette  | Indique que l’option **automatique** a été sélectionnée dans l’interface utilisateur de l’application. |
