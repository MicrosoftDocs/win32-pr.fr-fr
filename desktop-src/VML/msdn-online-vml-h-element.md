# <a name="vml-h-element"></a>Élément VML H

Cette rubrique décrit VML, une fonctionnalité déconseillée à partir de Windows Internet Explorer 9. Les pages Web et les applications qui reposent sur VML doivent être migrées vers SVG ou d’autres normes largement prises en charge.

> [!Note]  
> Depuis le 2011 décembre, cette rubrique a été archivée. Par conséquent, il n’est plus activement conservé. Pour plus d’informations, consultez [contenu archivé](/previous-versions/windows/internet-explorer/ie-developer/). Pour obtenir des informations, des recommandations et des conseils relatifs à la version actuelle de Windows Internet Explorer, consultez le [Centre de développement Internet Explorer](https://msdn.microsoft.com/ie/).

 

Définit un handle pour une forme.

Les attributs suivants modifient un handle.



| Attribut                                                | Description                                                    |
|----------------------------------------------------------|----------------------------------------------------------------|
| [InvX](msdn-online-vml-invx-attribute.md)               | Détermine si la position *x* du handle est inversée. |
| [InvY](msdn-online-vml-invy-attribute.md)               | Détermine si la position *y* du handle est inversée. |
| [Map](msdn-online-vml-map-attribute.md)                 | Définit la plage de mappage d’un handle.                         |
| [Polaire](msdn-online-vml-polar-attribute.md)             | Définit la position centrale des handles polaires.                 |
| [Position](position-attribute--h--vml.md)               | Spécifie les valeurs *x* et *y* du handle.                |
| [RadiusRange](msdn-online-vml-radiusrange-attribute.md) | Définit la plage d’une poignée polaire.                           |
| [Switch](msdn-online-vml-switch-attribute.md)           | Détermine si les directions du handle sont échangées.          |
| [XRange](msdn-online-vml-xrange-attribute.md)           | Définit la plage *x* d’une poignée de manipulation.                             |
| [YRange](msdn-online-vml-yrange-attribute.md)           | Définit la plage *y* d’une poignée de manipulation.                             |



 

**Remarques**

Cet élément doit être défini dans un élément [Shape](shape-element--vml.md) .
