# <a name="vml-extrusion-element"></a>VML Extrusion, élément

Cette rubrique décrit VML, une fonctionnalité déconseillée à partir de Windows Internet Explorer 9. Les pages Web et les applications qui reposent sur VML doivent être migrées vers SVG ou d’autres normes largement prises en charge.

> [!Note]  
> Depuis le 2011 décembre, cette rubrique a été archivée. Par conséquent, il n’est plus activement conservé. Pour plus d’informations, consultez [contenu archivé](/previous-versions/windows/internet-explorer/ie-developer/). Pour obtenir des informations, des recommandations et des conseils relatifs à la version actuelle de Windows Internet Explorer, consultez le [Centre de développement Internet Explorer](https://msdn.microsoft.com/ie/).

 

Définit une extrusion pour une forme.

Les attributs suivants modifient une extrusion.



| Attribut                                                              | Description                                                                                             |
|------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------|
| [AutoRotationCenter](msdn-online-vml-autorotationcenter-attribute.md) | Détermine si le centre de rotation sera le centre géométrique de l’extrusion.                |
| [Profondeur](msdn-online-vml-backdepth-attribute.md)                   | Définit la quantité d’extrusion vers l’arrière.                                                               |
| [Luminosité](msdn-online-vml-brightness-attribute.md)                 | Spécifie la quantité de luminosité d’une scène.                                                          |
| [Color](color-attribute--extrusion--vml.md)                           | Définit la couleur des visages d’extrusion.                                                               |
| [ColorMode](msdn-online-vml-colormode-attribute.md)                   | Détermine le mode de couleur de l’extrusion.                                                                 |
| [Diffusion](msdn-online-vml-diffusity-attribute.md)                   | Définit la quantité de diffusion de la lumière réfléchie à partir d’une forme extrudée.                              |
| [Edge](msdn-online-vml-edge-attribute.md)                             | Définit le biseau apparent des bords de l’extrusion.                                                      |
| [Ext](ext-attribute--extrusion--vml.md)                               | Définit le comportement d’extrusion par défaut pour les éditeurs graphiques.                                           |
| [Facette](msdn-online-vml-facet-attribute.md)                           | Définit le nombre de facettes utilisées pour décrire les surfaces courbes d’une extrusion.                          |
| [ForeDepth](msdn-online-vml-foredepth-attribute.md)                   | Définit la quantité d’extrusion vers l’avant.                                                                |
| [LightFace](msdn-online-vml-lightface-attribute.md)                   | Détermine si la face avant de l’extrusion répondra aux modifications apportées à l’éclairage.             |
| [LightHarsh](msdn-online-vml-lightharsh-attribute.md)                 | Détermine si la source de lumière principale est sévère.                                              |
| [LightHarsh2](msdn-online-vml-lightharsh2-attribute.md)               | Détermine si la source de lumière secondaire est sévère.                                            |
| [LightLevel](msdn-online-vml-lightlevel-attribute.md)                 | Définit l’intensité de la source de lumière principale de la scène.                                        |
| [LightLevel2](msdn-online-vml-lightlevel2-attribute.md)               | Définit l’intensité de la source de lumière secondaire pour la scène.                                      |
| [LightPosition](msdn-online-vml-lightposition-attribute.md)           | Spécifie la position de la lumière principale dans une scène.                                                 |
| [LightPosition2](msdn-online-vml-lightposition2-attribute.md)         | Spécifie la position de la lumière secondaire dans une scène.                                               |
| [LockRotationCenter](msdn-online-vml-lockrotationcenter-attribute.md) | Détermine si la rotation de l’objet extrudé est spécifiée par l’attribut **RotationAngle** . |
| [Métallique](msdn-online-vml-metal-attribute.md)                           | Détermine si la surface de la forme extrudée est semblable à Metal.                               |
| [Activé](on-attribute--extrusion--vml.md)                                 | Détermine si une extrusion sera affichée.                                                      |
| [Orientation](msdn-online-vml-orientation-attribute.md)               | Spécifie le vecteur autour duquel une forme doit pivoter.                                              |
| [OrientationAngle](msdn-online-vml-orientationangle-attribute.md)     | Définit l’angle selon lequel une extrusion pivote autour de l’orientation.                                     |
| [Verticale](msdn-online-vml-plane-attribute.md)                           | Spécifie le plan perpendiculaire à l’extrusion.                                           |
| [Render](msdn-online-vml-render-attribute.md)                         | Définit le mode de rendu de l’extrusion.                                                            |
| [RotationAngle](msdn-online-vml-rotationangle-attribute.md)           | Spécifie la rotation de l’objet sur les axes x et y.                                           |
| [RotationCenter](msdn-online-vml-rotationcenter-attribute.md)         | Spécifie le centre de rotation d’une forme.                                                           |
| [Éclat](msdn-online-vml-shininess-attribute.md)                   | Définit la concentration de la lumière réfléchie d’une surface d’extrusion.                                   |
| [SkewAmt](msdn-online-vml-skewamt-attribute.md)                       | Définit le degré d’inclinaison d’une extrusion.                                                             |
| [SkewAngle](msdn-online-vml-skewangle-attribute.md)                   | Définit l’angle d’inclinaison d’une extrusion.                                                              |
| [Specularity](msdn-online-vml-specularity-attribute.md)               | Définit le specularity d’une forme extrudée.                                                           |
| [Type](type-attribute--extrusion--vml.md)                             | Définit la façon dont la forme est extrudée.                                                             |
| [Point](msdn-online-vml-viewpoint-attribute.md)                   | Définit le point de vue de l’observateur.                                                                  |
| [ViewpointOrigin](msdn-online-vml-viewpointorigin-attribute.md)       | Définit l’origine du point de vue dans le cadre englobant de la forme.                               |



 

**Remarques**

Cet élément doit être défini dans un élément [Shape](shape-element--vml.md) .

En outre, l’attribut [on](on-attribute--extrusion--vml.md) doit avoir la valeur **true**.

Voici le code minimal nécessaire pour générer une extrusion.


```HTML
   <v:rect style="width:50px;height:50px">
   <v:extrusion on="True"/>
   </v:rect>
```
