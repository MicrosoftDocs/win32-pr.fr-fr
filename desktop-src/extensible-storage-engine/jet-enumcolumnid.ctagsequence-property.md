# <a name="jet_enumcolumnidctagsequence-property"></a>JET_ENUMCOLUMNID. ctagSequence, propriété

Obtient ou définit le nombre de valeurs de colonne (par index de base un) à énumérer pour l’ID de colonne spécifié. Si ctagSequence a la valeur 0 (zéro), rgtagSequence est ignoré et toutes les valeurs de colonne pour l’ID de colonne spécifié seront énumérées.
