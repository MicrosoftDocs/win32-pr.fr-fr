# <a name="apijetstopbackupinstance-method"></a>API. JetStopBackupInstance, méthode

Empêche l’activité relative à la sauvegarde en continu de se poursuivre sur une instance en cours d’exécution spécifique, ce qui met fin à la sauvegarde en continu de manière prévisible.
