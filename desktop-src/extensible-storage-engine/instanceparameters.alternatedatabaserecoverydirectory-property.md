# <a name="instanceparametersalternatedatabaserecoverydirectory-property"></a>InstanceParameters. AlternateDatabaseRecoveryDirectory, propriété

Obtient ou définit le chemin d’accès relatif ou absolu du système de fichiers d’un dossier dans lequel la récupération après incident ou une opération de restauration peut trouver les bases de données référencées dans le journal des transactions dans le dossier spécifié.

**Espace de noms :**  [Microsoft. ISAM. esent. Interop](./microsoft.isam.esent.interop-namespace.md)  
**Assembly :**  Microsoft. ISAM. esent. Interop (en Microsoft.Isam.Esent.Interop.dll)

    Set

public string AlternateDatabaseRecoveryDirectory { get; set; }

Type : [System. String](/dotnet/api/system.string)  

## <a name="remarks"></a>Notes

Ce paramètre est ignoré sur Windows XP.
