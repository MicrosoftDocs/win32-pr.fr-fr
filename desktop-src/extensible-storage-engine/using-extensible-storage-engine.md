# <a name="using-extensible-storage-engine"></a>Utilisation du moteur de stockage extensible


_**S’applique à :** Windows | Serveur Windows_

## <a name="using-extensible-storage-engine"></a>Utilisation du moteur de stockage extensible

Cette section contient des informations générales sur l’utilisation des éléments suivants de l’API ESE.

  - [Colonnes](./columns.md)

  - [Indexation dans la table](./indexing-in-the-table.md)

  - [Création de bases de données](./creating-databases.md)

  - [Transactions](./transactions.md)

  - [Erreurs ESE](./extensible-storage-engine-errors.md)

  - [Fichiers ESE](./extensible-storage-engine-files.md)

Les fichiers sources de votre programme doivent inclure le fichier d’en-tête esent. h pour accéder aux prototypes de fonction et aux définitions de structure pour l’API ESE (Extensible Storage Engine). Les développeurs peuvent utiliser le fichier de bibliothèque esent. lib pour créer des applications qui utilisent l’API ESE (Extensible Storage Engine). Au moment de l’exécution, les applications sont liées au esent.dll.
