# <a name="jet_retinfoitagsequence-property"></a>JET_RETINFO. itagSequence, propriété

Obtient ou définit le numéro de séquence de la valeur dans une colonne à valeurs multiples. Le tableau de valeurs est de base un. La première valeur est séquence 1, et non 0. Si la colonne d’enregistrement n’a qu’une seule valeur, 1 doit être passé en tant que itagSequence.

**Espace de noms :**  [Microsoft. ISAM. esent. Interop](./microsoft.isam.esent.interop-namespace.md)
