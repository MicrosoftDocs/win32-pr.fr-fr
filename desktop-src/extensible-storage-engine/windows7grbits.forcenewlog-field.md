# <a name="windows7grbitsforcenewlog-field"></a>Champ Windows7Grbits. ForceNewLog

Force la création d’un nouveau fichier journal. Cette option peut être utilisée même si la session n’est pas actuellement dans une transaction. Cette option ne peut pas être utilisée conjointement avec une autre option.
