# <a name="jet_indexlistcolumnidcentry-property"></a>JET_INDEXLIST. columnidcEntry, propriété

Obtient le ColumnID de la colonne dans la table temporaire qui stocke le nombre d’entrées dans l’index. Cette valeur n’est pas actuelle et est uniquement mise à jour par « API. JetComputeStats ». La colonne est de type [long](./jet-coltyp-enumeration.md).

**Espace de noms :**  [Microsoft. ISAM. esent. Interop](./microsoft.isam.esent.interop-namespace.md)
