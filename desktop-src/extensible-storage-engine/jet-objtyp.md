# <a name="jet_objtyp"></a>JET_OBJTYP


_**S’applique à :** Windows | Windows Serveurs_

## <a name="jet_objtyp"></a>JET_OBJTYP

Le **JET_OBJTYP** groupe de constantes représente le type d’un objet de base de données. Actuellement, seules les tables sont prises en charge.
