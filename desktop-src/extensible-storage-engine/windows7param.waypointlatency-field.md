# <a name="windows7paramwaypointlatency-field"></a>Champ Windows7Param. WaypointLatency

Ce paramètre définit le nombre de journaux pour lesquels esent ajournera les vidages de base de données. Cela peut être utilisé pour augmenter la capacité de récupération de base de données en cas de perte de fichiers journaux.
