# <a name="instanceparametersmaxtransactionsize-property"></a>InstanceParameters. MaxTransactionSize, propriété

Obtient ou définit le pourcentage de la Banque des versions qui peut être utilisée par la plus ancienne transaction avant [VersionStoreOutOfMemory](./jet-err-enumeration.md) (valeur par défaut = 100).
