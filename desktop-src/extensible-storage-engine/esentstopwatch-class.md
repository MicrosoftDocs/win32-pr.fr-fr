# <a name="esentstopwatch-class"></a>EsentStopwatch, classe

Fournit un ensemble de méthodes et de propriétés que vous pouvez utiliser pour mesurer les statistiques de travail ESENT d’un thread. Si la version actuelle de ESENT ne prend pas en charge [JetGetThreadStats (JET_THREADSTATS)](./vistaapi.jetgetthreadstats-method.md) , toutes les statistiques esent seront 0.
