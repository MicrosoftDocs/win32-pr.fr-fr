# <a name="instanceparametersmaxcursors-property"></a>InstanceParameters. MaxCursors, propriété

Obtient ou définit le nombre de ressources de curseur réservées pour cette instance. Une ressource de curseur correspond directement à un JET_TABLEID.

**Espace de noms :**  [Microsoft. ISAM. esent. Interop](./microsoft.isam.esent.interop-namespace.md)  
**Assembly :**  Microsoft. ISAM. esent. Interop (en Microsoft.Isam.Esent.Interop.dll)
