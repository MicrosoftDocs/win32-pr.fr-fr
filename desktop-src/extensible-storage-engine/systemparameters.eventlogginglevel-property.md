# <a name="systemparameterseventlogginglevel-property"></a>Propriété SystemParameters. EventLoggingLevel

Obtient ou définit le niveau de détail des messages EventLog émis dans le journal des événements par le moteur de base de données. Des numéros plus élevés entraînent des messages EventLog plus détaillés.

**Espace de noms :**  [Microsoft. ISAM. esent. Interop](./microsoft.isam.esent.interop-namespace.md)
