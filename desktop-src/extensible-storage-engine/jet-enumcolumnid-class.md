# <a name="jet_enumcolumnid-class"></a>Classe JET_ENUMCOLUMNID

Énumère un ensemble spécifique de colonnes et, éventuellement, un ensemble spécifique de plusieurs valeurs pour ces colonnes lorsque la fonction JetEnumerateColumns est utilisée. JetEnumerateColumns accepte éventuellement un tableau de structures de JET_ENUMCOLUMNID.
