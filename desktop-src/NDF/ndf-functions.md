# <a name="ndf-functions"></a>Fonctions NDF

Les fonctions suivantes permettent aux développeurs de logiciels d’utiliser les fonctionnalités fournies par l’infrastructure de diagnostic réseau (NDF). En cas de problème réseau, NDF peut diagnostiquer la cause racine et gérer la résolution nécessaire de manière appropriée, parfois même effectuer les réparations nécessaires.

> [!Note]  
> Ces fonctions sont destinées aux développeurs qui implémentent la fonctionnalité NDF de base dans leurs applications.
