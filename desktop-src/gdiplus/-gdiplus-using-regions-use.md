# <a name="using-regions-gdi"></a>Utilisation des régions (GDI+)

la classe Windows GDI+ [**Region**](/windows/desktop/api/gdiplusheaders/nl-gdiplusheaders-region) vous permet de définir une forme personnalisée. La forme peut être constituée de lignes, de polygones et de courbes.

Deux utilisations courantes des régions sont le test de positionnement et le découpage. Le test de positionnement détermine si l’utilisateur a cliqué avec la souris dans une certaine région de l’écran. Le découpage limite le dessin à une certaine région.

Les rubriques suivantes traitent plus en détail les régions :

-   [Test de positionnement avec une région](-gdiplus-hit-testing-with-a-region-use.md)
-   [Découpage avec une région](-gdiplus-clipping-with-a-region-use.md)
