# <a name="rectfrectf-constructors"></a>Constructeurs RectF. RectF

Cette rubrique répertorie les constructeurs de la classe [**RectF**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-rectf) . Pour obtenir une liste complète des classes, consultez **RectF, classe**.

### <a name="overload-list"></a>Liste de surcharge



| Constructeur                                                                        | Description                                                                                                                                                                                                                                                                                                                     |
|:-----------------------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**RectF (PointF&, SizeF&)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-rectf-rectf(inconstpointf__inconstsizef_))         | Crée un objet [**RectF**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-rectf) à l’aide d’un objet [**PointF**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-pointf) pour initialiser les données membres **X** et **Y** et utilise un objet [**SizeF**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-sizef) pour initialiser la **largeur** et la **hauteur** des membres de données de ce rectangle.<br/> |
| [**RectF (réel, réel, réel, réel)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-rectf-rectf(inreal_inreal_inreal_inreal)) | Crée un objet [**RectF**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-rectf) à l’aide de quatre entiers pour initialiser les membres de données **X**, **Y**, **Width** et **Height** .<br/>                                                                                                                                                          |
| [**RectF()**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-rectf-rectf(inconstpointf__inconstsizef_))                                     | Crée un objet [**RectF**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-rectf) et initialise les données des membres **X**, **Y**, **Width** et **Height** à zéro. Il s'agit du constructeur par défaut.<br/>                                                                                                                                      |
