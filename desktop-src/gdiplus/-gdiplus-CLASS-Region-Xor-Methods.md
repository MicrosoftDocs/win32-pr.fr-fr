# <a name="regionxor-methods"></a>Region. Xor, méthodes

Cette rubrique répertorie les méthodes XOR de la classe [**Region**](/windows/win32/api/gdiplusheaders/nl-gdiplusheaders-region) . Pour obtenir la liste complète des méthodes pour la classe **Region** , consultez [méthodes Region](-gdiplus-class-region-methods.md).

### <a name="overload-list"></a>Liste de surcharge



| Méthode                                                         | Description                                                                                                                                                                              |
|:---------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XOR (Rect&)**](/previous-versions//ms534787(v=vs.85))     | La méthode [**Region :: XOR**](/previous-versions//ms534787(v=vs.85)) met à jour cette région avec les parties qui ne se croisent pas et l’intérieur du rectangle spécifié.<br/>  |
| [**XOR (RectF&)**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-xor(inconstrectf_))   | La méthode [**Region :: XOR**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-xor(inconstrectf_)) met à jour cette région avec les parties qui ne se croisent pas et l’intérieur du rectangle spécifié.<br/> |
| [**XOR (région \* )**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-xor(inconstregion))     | La méthode [**Region :: XOR**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-xor(inconstregion)) met à jour cette région avec les parties qui ne se croisent pas et une autre région.<br/>                         |
| [**XOR (GraphicsPath \* )**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-xor(inconstgraphicspath)) | La méthode [**Region :: XOR**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-xor(inconstgraphicspath)) met à jour cette région avec les parties qui ne se croisent pas et l’intérieur du chemin d’accès spécifié.<br/>            |
