# <a name="using-a-brush-to-fill-shapes"></a>Utilisation d'un pinceau pour remplir des formes

un Windows GDI+ objet [**Brush**](/windows/desktop/api/gdiplusbrush/nl-gdiplusbrush-brush) est utilisé pour remplir l’intérieur d’une forme fermée. GDI+ définit plusieurs styles de remplissage : couleur unie, motif hachuré, texture d’image et dégradé de couleur.

Les rubriques suivantes traitent de l’utilisation des pinceaux plus en détail :

-   [Remplissage d’une forme avec une couleur unie](-gdiplus-filling-a-shape-with-a-solid-color-use.md)
-   [Remplissage d’une forme avec un motif hachuré](-gdiplus-filling-a-shape-with-a-hatch-pattern-use.md)
-   [Remplissage d’une forme avec une texture d’image](-gdiplus-filling-a-shape-with-an-image-texture-use.md)
-   [Mosaïque d’une forme avec une image](-gdiplus-tiling-a-shape-with-an-image-use.md)
-   [Remplissage d’une forme avec un dégradé de couleur](-gdiplus-filling-a-shape-with-a-color-gradient-use.md)
