# <a name="image-effect-constants"></a>Constantes d’effet d’image

La classe [**Effect**](/windows/desktop/api/gdipluseffects/nl-gdipluseffects-effect) a plusieurs descendants qui peuvent être utilisés pour appliquer un effet de couleur ou un ajustement à une image bitmap. Chacun des descendants est identifié par un GUID. Les constantes suivantes, définies dans Gdipluseffects. h, représentent des GUID qui identifient les différents effets.

-   BlurEffectGuid
-   BrightnessContrastEffectGuid
-   ColorBalanceEffectGuid
-   ColorCurveEffectGuid
-   ColorLUTEffectGuid
-   ColorMatrixEffectGuid
-   HueSaturationLightnessEffectGuid
-   LevelsEffectGuid
-   RedEyeCorrectionEffectGuid
-   SharpenEffectGuid
-   TintEffectGuid
