# <a name="characterrangecharacterrange-constructors"></a>Constructeurs CharacterRange. CharacterRange

### <a name="overload-list"></a>Liste de surcharge



| Constructeur                                                                                   | Description                                                                                                                                                                                |
|:----------------------------------------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**CharacterRange()**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-characterrange-characterrange-r2)                     | Crée un objet [**CharacterRange**](/windows/win32/api/gdiplustypes/nl-gdiplustypes-characterrange) avec les membres de données dont la valeur est zéro.<br/>                                     |
| [**CharacterRange (INT, INT)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-characterrange-characterrange(int_int)) | Crée un objet [**CharacterRange**](/windows/win32/api/gdiplustypes/nl-gdiplustypes-characterrange) et initialise les membres de données avec les valeurs spécifiées.<br/> |
