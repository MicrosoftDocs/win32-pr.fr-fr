# <a name="pointpoint-constructors"></a>Constructeurs point. point

Cette rubrique répertorie les constructeurs de la classe [**point**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-point) . Pour obtenir une liste complète des classes, consultez **point, classe**.

### <a name="overload-list"></a>Liste de surcharge



| Constructeur                                                | Description                                                                                                                                                                      |
|:-----------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Point (taille&)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-point-point(inconstsize_))   | Crée un objet [**point**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-point) à l’aide d’un objet de [**taille**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-size) pour initialiser les données membres **X** et **Y** .<br/> |
| [**Point (point&)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-point-point(inconstpoint_)) | Crée un nouvel objet [**point**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-point) et copie les données membres à partir d’un autre objet **point** .<br/>                                           |
| [**Point (INT, INT)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-point-point(inint_inint))  | Crée un objet [**point**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-point) à l’aide de deux entiers pour initialiser les données membres **X** et **Y** .<br/>                                       |
| [**Point ()**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-point-point)             | Crée un objet [**point**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-point) et initialise les données membres **X** et **Y** à zéro. Il s'agit du constructeur par défaut.<br/>               |
