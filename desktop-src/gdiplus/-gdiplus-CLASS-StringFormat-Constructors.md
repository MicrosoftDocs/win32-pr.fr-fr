# <a name="stringformatstringformat-constructors"></a>StringFormat. StringFormat, constructeurs

Cette rubrique répertorie les constructeurs de la classe [**StringFormat**](/windows/win32/api/gdiplusstringformat/nl-gdiplusstringformat-stringformat) . Pour obtenir une liste complète des classes, consultez **StringFormat, classe**.

### <a name="overload-list"></a>Liste de surcharge



| Constructeur                                                                                        | Description                                                                                                                        |
|:---------------------------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------|
| [**StringFormat (INT, LANGID)**](/windows/win32/api/gdiplusstringformat/nf-gdiplusstringformat-stringformat-stringformat(inint_inlangid)) | Crée un objet [**StringFormat**](/windows/win32/api/gdiplusstringformat/nl-gdiplusstringformat-stringformat) basé sur des indicateurs de format de chaîne et un langage.<br/> |
| [**StringFormat (StringFormat \* )**](/windows/win32/api/gdiplusstringformat/nf-gdiplusstringformat-stringformat-stringformat(conststringformat_))           | Crée un objet [**StringFormat**](/windows/win32/api/gdiplusstringformat/nl-gdiplusstringformat-stringformat) à partir d’un autre objet **StringFormat** .<br/>        |
