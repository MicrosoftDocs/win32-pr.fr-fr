# <a name="sizesize-constructors"></a>Constructeurs Size. Size

Cette rubrique répertorie les constructeurs de la classe [**Size**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-size) . Pour obtenir une liste complète des classes, consultez **classe Size**.

### <a name="overload-list"></a>Liste de surcharge



| Constructeur                                                     | Description                                                                                                                                          |
|:----------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Taille (taille&)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-size-size(inconstsize_))           | Crée un objet de [**taille**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-size) et initialise ses membres en copiant les membres d’un autre objet de **taille** .<br/> |
| [**Taille (INT, INT)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-size-size(inint_inint)) | Crée un objet de [**taille**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-size) et initialise ses membres de données de **largeur** et de **hauteur** .<br/>                     |
| [**Taille ()**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-size-size)                     | Crée un objet de [**taille**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-size) et initialise les membres à zéro. Il s'agit du constructeur par défaut.<br/>      |
