# <a name="regioncomplement-methods"></a>Region. Complement, méthodes

Cette rubrique répertorie les méthodes de complément de la classe [**Region**](/windows/win32/api/gdiplusheaders/nl-gdiplusheaders-region) . Pour obtenir la liste complète des méthodes pour la classe **Region** , consultez [méthodes Region](-gdiplus-class-region-methods.md).

### <a name="overload-list"></a>Liste de surcharge



| Méthode                                                                       | Description                                                                                                                                                                                                    |
|:-----------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Complément (Rect&)**](/previous-versions//ms534918(v=vs.85))     | La méthode [**Region :: complement**](/previous-versions//ms534918(v=vs.85)) met à jour cette région avec la partie de l’intérieur du rectangle spécifiée qui ne croise pas cette région.<br/>  |
| [**Complément (RectF&)**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-complement(inconstrectf_))   | La méthode [**Region :: complement**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-complement(inconstrectf_)) met à jour cette région avec la partie de l’intérieur du rectangle spécifiée qui ne croise pas cette région.<br/> |
| [**Complément (région \* )**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-complement(inconstregion))     | La méthode [**Region :: complement**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-complement(inconstregion)) met à jour cette région vers la partie d’une autre région qui n’interfère pas avec cette région.<br/>                         |
| [**Complément (GraphicsPath \* )**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-complement(inconstgraphicspath)) | La méthode [**Region :: complement**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-complement(inconstgraphicspath)) met à jour cette région vers la partie de l’intérieur du chemin d’accès spécifié qui ne croise pas cette région.<br/>            |
