# <a name="coordinate-systems-and-transformations"></a>Systèmes de coordonnées et transformations

Windows GDI+ fournit une transformation universelle et une transformation de page qui vous permet de transformer (faire pivoter, mettre à l’échelle, traduire, etc.) les éléments que vous dessinez. Les deux transformations vous permettent également de travailler dans différents systèmes de coordonnées.

-   [Représentation matricielle des transformations](-gdiplus-matrix-representation-of-transformations-about.md)
-   [Transformations globales et locales](-gdiplus-global-and-local-transformations-about.md)
