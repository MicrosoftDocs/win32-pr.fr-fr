# <a name="rectrect-constructors"></a>Rect. Rect, constructeurs

Cette rubrique répertorie les constructeurs de la classe [**Rect**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-rect) . Pour obtenir une liste complète des classes, consultez **Rect, classe**.

### <a name="overload-list"></a>Liste de surcharge



| Constructeur                                                                 | Description                                                                                                                                                                                                                                                                                        |
|:----------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Rect (point&, taille&)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-rect-rect(inconstpoint__inconstsize_))       | Crée un objet [**Rect**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-rect) à l’aide d’un objet [**point**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-point) pour initialiser les membres de données **X** et **Y** et un objet de [**taille**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-size) pour initialiser les membres de données de **largeur** et de **hauteur** .<br/> |
| [**Rect (INT, INT, INT, INT)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-rect-rect(inint_inint_inint_inint)) | Crée un objet [**Rect**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-rect) en utilisant quatre entiers pour initialiser les membres de données **X**, **Y**, **Width** et **Height** .<br/>                                                                                                                               |
| [**Rect ()**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-rect-rect)                                 | Crée un objet [**Rect**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-rect) dont la coordonnée x, la coordonnée y, la largeur et la hauteur ne sont pas égales à zéro. Il s'agit du constructeur par défaut. <br/>                                                                                                                          |
