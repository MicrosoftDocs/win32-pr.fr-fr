# <a name="regionexclude-methods"></a>Region. Exclude, méthodes

Cette rubrique répertorie les méthodes Exclude de la classe [**Region**](/windows/win32/api/gdiplusheaders/nl-gdiplusheaders-region) . Pour obtenir la liste complète des méthodes pour la classe **Region** , consultez [méthodes Region](-gdiplus-class-region-methods.md).

### <a name="overload-list"></a>Liste de surcharge



| Méthode                                                                 | Description                                                                                                                                                                                         |
|:-----------------------------------------------------------------------|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Exclude (Rect&)**](/previous-versions//ms534825(v=vs.85))     | La méthode [**Region :: Exclude**](/previous-versions//ms534825(v=vs.85)) met à jour cette région avec la partie de elle-même qui ne croise pas l’intérieur du rectangle spécifié.<br/>  |
| [**Exclude (RectF&)**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-exclude(inconstrectf_))   | La méthode [**Region :: Exclude**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-exclude(inconstrectf_)) met à jour cette région avec la partie de elle-même qui ne croise pas l’intérieur du rectangle spécifié.<br/> |
| [**Exclure (région \* )**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-exclude(inconstregion))     | La méthode [**Region :: Exclude**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-exclude(inconstregion)) met à jour cette région avec la partie de elle-même qui n’interfère pas avec une autre région.<br/>                         |
| [**Exclure (GraphicsPath \* )**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-exclude(inconstgraphicspath)) | La méthode [**Region :: Exclude**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-exclude(inconstgraphicspath)) met à jour cette région avec la partie de elle-même qui ne croise pas l’intérieur du chemin d’accès spécifié.<br/>            |
