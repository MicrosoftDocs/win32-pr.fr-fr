# <a name="regionintersect-methods"></a>Region. Intersect, méthodes

Cette rubrique répertorie les méthodes Intersect de la classe [**Region**](/windows/win32/api/gdiplusheaders/nl-gdiplusheaders-region) . Pour obtenir la liste complète des méthodes pour la classe **Region** , consultez [méthodes Region](-gdiplus-class-region-methods.md).

### <a name="overload-list"></a>Liste de surcharge



| Méthode                                                                     | Description                                                                                                                                                                                     |
|:---------------------------------------------------------------------------|:------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Intersect (Rect&)**](/previous-versions//ms534804(v=vs.85))     | La méthode [**Region :: Intersect**](/previous-versions//ms534804(v=vs.85)) met à jour cette région avec la partie de elle-même qui croise l’intérieur du rectangle spécifié.<br/>  |
| [**Intersect (RectF&)**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-intersect(inconstrectf_))   | La méthode [**Region :: Intersect**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-intersect(inconstrectf_)) met à jour cette région avec la partie de elle-même qui croise l’intérieur du rectangle spécifié.<br/> |
| [**Intersect (région \* )**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-intersect(inconstregion))     | La méthode [**Region :: Intersect**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-intersect(inconstregion)) met à jour cette région avec la partie de elle-même qui croise une autre région.<br/>                         |
| [**Intersect (GraphicsPath \* )**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-intersect(inconstgraphicspath)) | La méthode [**Region :: Intersect**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-intersect(inconstgraphicspath)) met à jour cette région avec la partie de elle-même qui croise l’intérieur du chemin d’accès spécifié.<br/>            |
