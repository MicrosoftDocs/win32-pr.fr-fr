# <a name="pointfpointf-constructors"></a>PointF. PointF, constructeurs

Cette rubrique répertorie les constructeurs de la classe [**PointF**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-pointf) . Pour obtenir une liste complète des classes, consultez **PointF, classe**.

### <a name="overload-list"></a>Liste de surcharge



| Constructeur                                                    | Description                                                                                                                                                                       |
|:---------------------------------------------------------------|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**PointF (SizeF&)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-pointf-pointf(inconstsizef_))   | Crée un objet [**PointF**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-pointf) à l’aide d’un objet [**SizeF**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-sizef) pour spécifier les données membres **X** et **Y** .<br/> |
| [**PointF (PointF&)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-pointf-pointf(inconstpointf_)) | Crée un objet [**PointF**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-pointf) et copie les données à partir d’un autre objet **PointF** .<br/>                                                 |
| [**PointF (réel, réel)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-pointf-pointf(inreal_inreal)) | Crée un objet [**PointF**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-pointf) à l’aide de deux nombres réels pour spécifier les données membres **X** et **Y** .<br/>                                     |
| [**PointF ()**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-pointf-pointf)              | Crée un objet [**PointF**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-pointf) et initialise les données membres **X** et **Y** à zéro. Il s'agit du constructeur par défaut.<br/>              |
