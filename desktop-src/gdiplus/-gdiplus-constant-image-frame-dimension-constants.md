# <a name="image-frame-dimension-constants"></a>Constantes de dimension Frame image

Les formats de fichier image GIF (Graphics Interchange Format) et TIFF (Tagged Image File Format) vous permettent de stocker plusieurs frames dans un seul fichier image. Dans un fichier GIF, plusieurs frames sont utilisés pour l’animation, de sorte que les frames sont considérés comme étant dans la dimension de temps. Dans un fichier TIFF, plusieurs frames sont généralement utilisés en tant que pages distinctes. les frames sont donc considérés comme étant dans la dimension page.

Les constantes suivantes, définies dans Gdiplusimaging. h, représentent des GUID qui identifient les dimensions de page et de temps.

-   FrameDimensionPage
-   FrameDimensionTime
