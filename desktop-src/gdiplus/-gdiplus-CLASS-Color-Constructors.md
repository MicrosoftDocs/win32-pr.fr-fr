# <a name="colorcolor-constructors"></a>Constructeurs Color. Color

Cette rubrique répertorie les constructeurs de la classe [**Color**](/windows/win32/api/gdipluscolor/nl-gdipluscolor-color) . Pour obtenir une liste complète des classes, consultez **Color, classe**.

### <a name="overload-list"></a>Liste de surcharge



| Constructeur                                                               | Description                                                                                                                                                                                                         |
|:--------------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Couleur (ARGB)**](/windows/win32/api/gdipluscolor/nf-gdipluscolor-color-color(inargb))                   | Crée un objet [**Color :: Color**](/windows/win32/api/gdipluscolor/nf-gdipluscolor-color-color(inargb)) à l’aide d’une valeur **ARVB** .<br/>                                                                                                    |
| [**Couleur (BYTE, Byte, BYTE)**](/windows/win32/api/gdipluscolor/nf-gdipluscolor-color-color(inbyte_inbyte_inbyte))        | Crée un objet [**Color :: Color**](/windows/win32/api/gdipluscolor/nf-gdipluscolor-color-color(inbyte_inbyte_inbyte)) en utilisant les valeurs spécifiées pour les composants rouge, vert et Blue. Ce constructeur affecte au composant alpha la valeur 255 (opaque).<br/> |
| [**Couleur (BYTE, BYTE, Byte, BYTE)**](/windows/win32/api/gdipluscolor/nf-gdipluscolor-color-color(inbyte_inbyte_inbyte_inbyte)) | Crée un objet [**Color :: Color**](/windows/win32/api/gdipluscolor/nf-gdipluscolor-color-color(inbyte_inbyte_inbyte_inbyte)) en utilisant les valeurs spécifiées pour les composants alpha, rouge, vert et Blue.<br/>                                                   |
| [**Couleur ()**](/windows/win32/api/gdipluscolor/nf-gdipluscolor-color-color)                            | Crée un objet [**Color :: Color**](/windows/win32/api/gdipluscolor/nf-gdipluscolor-color-color) et l’initialise sur le noir opaque. Il s'agit du constructeur par défaut.<br/>                                                                |
