# <a name="graphicspathgetpathpoints-methods"></a>GraphicsPath. GetPathPoints, méthodes

Cette rubrique répertorie les méthodes GetPathPoints de la classe [**GraphicsPath**](/windows/win32/api/gdipluspath/nl-gdipluspath-graphicspath) . Pour obtenir la liste complète des méthodes pour la classe **GraphicsPath** , consultez [**GraphicsPath**](/windows/win32/api/gdipluspath/nl-gdipluspath-graphicspath).

### <a name="overload-list"></a>Liste de surcharge



| Méthode                                                                                                    | Description                                                                                                                                                                                                                                                                           |
|:----------------------------------------------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**GetPathPoints (point \* , int)**](/windows/win32/api/gdipluspath/nf-gdipluspath-graphicspath-getpathpoints(outpoint_inint))   | La méthode [**GraphicsPath :: GetPathPoints**](/windows/win32/api/gdipluspath/nf-gdipluspath-graphicspath-getpathpoints(outpoint_inint)) obtient le tableau de points de ce tracé. Le tableau contient les points de terminaison et les points de contrôle des lignes et des splines de Bézier utilisés pour dessiner le tracé.<br/>  |
| [**GetPathPoints (PointF \* , int)**](/previous-versions//ms535582(v=vs.85)) | La méthode [**GraphicsPath :: GetPathPoints**](/previous-versions//ms535582(v=vs.85)) obtient le tableau de points de ce tracé. Le tableau contient les points de terminaison et les points de contrôle des lignes et des splines de Bézier utilisés pour dessiner le tracé.<br/> |
