# <a name="graphicsgraphics-constructors"></a>Graphics. Graphics, constructeurs

Cette rubrique répertorie les constructeurs de la classe [**Graphics**](/windows/win32/api/gdiplusgraphics/nl-gdiplusgraphics-graphics) . Pour obtenir une liste complète des classes, consultez **Graphics, classe**.

### <a name="overload-list"></a>Liste de surcharge



| Constructeur                                                                   | Description                                                                                                                                                                        |
|:------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Graphiques (HDC)**](/previous-versions//ms536160(v=vs.85))                | Crée un objet [**Graphics :: Graphics**](/previous-versions//ms536160(v=vs.85)) associé à un contexte de périphérique spécifié.<br/>                                |
| [**Graphiques (image \* )**](/windows/win32/api/gdiplusgraphics/nf-gdiplusgraphics-graphics-graphics(inimage))          | Crée un objet [**Graphics :: Graphics**](/windows/win32/api/gdiplusgraphics/nf-gdiplusgraphics-graphics-graphics(inimage)) associé à un objet [**image**](/windows/win32/api/gdiplusheaders/nl-gdiplusheaders-image) .<br/>   |
| [**Graphics (HWND, BOOL)**](/windows/win32/api/gdiplusgraphics/nf-gdiplusgraphics-graphics-graphics(inhwnd_inbool))     | Crée un objet [**Graphics :: Graphics**](/windows/win32/api/gdiplusgraphics/nf-gdiplusgraphics-graphics-graphics(inhwnd_inbool)) associé à une fenêtre spécifiée.<br/>                                   |
| [**Graphics (HDC, HANDLE)**](/windows/win32/api/gdiplusgraphics/nf-gdiplusgraphics-graphics-graphics(inhdc_inhandle)) | Crée un objet [**Graphics :: Graphics**](/windows/win32/api/gdiplusgraphics/nf-gdiplusgraphics-graphics-graphics(inhdc_inhandle)) associé à un contexte de périphérique et un périphérique spécifiés.<br/> |
