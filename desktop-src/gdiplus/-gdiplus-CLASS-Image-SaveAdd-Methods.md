# <a name="imagesaveadd-methods"></a>Méthodes image. SaveAdd

Cette rubrique répertorie les méthodes SaveAdd de la classe [**image**](/windows/win32/api/gdiplusheaders/nl-gdiplusheaders-image) . Pour obtenir la liste complète des méthodes de la classe **image** , consultez [méthodes d’image](-gdiplus-class-image-methods.md).

### <a name="overload-list"></a>Liste de surcharge



| Méthode                                                                                               | Description                                                                                                                                                                                                                                                                            |
|:-----------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SaveAdd (EncoderParameters \* )**](/previous-versions//ms535408(v=vs.85))                  | La méthode [**image :: SaveAdd**](/previous-versions//ms535408(v=vs.85)) ajoute un frame à un fichier ou à un flux spécifié dans un appel précédent à la méthode **Save** . Utilisez cette méthode pour enregistrer les frames sélectionnés d'une image à plusieurs frames dans une autre image à plusieurs frames.<br/> |
| [**SaveAdd (image \* , EncoderParameters \* )**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-image-saveadd(inimage_inconstencoderparameters)) | La méthode [**image :: SaveAdd**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-image-saveadd(inimage_inconstencoderparameters)) ajoute un frame à un fichier ou à un flux spécifié dans un appel précédent à la méthode **Save** .<br/>                                                                                             |
