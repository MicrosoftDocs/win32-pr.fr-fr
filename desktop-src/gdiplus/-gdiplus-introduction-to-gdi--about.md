# <a name="introduction-to-gdi"></a>Présentation de GDI+

Windows GDI+ est une interface de périphérique graphique qui permet aux programmeurs d’écrire des applications indépendantes du périphérique. les services de GDI+ sont exposés via un ensemble de classes C++.
