# <a name="matrixtransformpoints-methods"></a>Méthodes Matrix. TransformPoints

Cette rubrique répertorie les méthodes TransformPoints de la classe [**Matrix**](/windows/win32/api/gdiplusmatrix/nl-gdiplusmatrix-matrix) . Pour obtenir la liste complète des méthodes de la classe **Matrix** , consultez [méthodes de matrice](-gdiplus-class-matrix-methods.md).

### <a name="overload-list"></a>Liste de surcharge



| Méthode                                                                                               | Description                                                                                                                                                                                                                                                                                                 |
|:-----------------------------------------------------------------------------------------------------|:------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**TransformPoints (point \* , int)**](/windows/win32/api/gdiplusmatrix/nf-gdiplusmatrix-matrix-transformpoints(inoutpoint_inint))   | La méthode [**Matrix :: TransformPoints**](/windows/win32/api/gdiplusmatrix/nf-gdiplusmatrix-matrix-transformpoints(inoutpoint_inint)) multiplie chaque point dans un tableau par cette matrice. Chaque point est traité comme une matrice de lignes. La multiplication est effectuée avec la matrice de lignes à gauche et cette matrice à droite.<br/>  |
| [**TransformPoints (PointF \* , int)**](/previous-versions//ms535321(v=vs.85)) | La méthode [**Matrix :: TransformPoints**](/previous-versions//ms535321(v=vs.85)) multiplie chaque point dans un tableau par cette matrice. Chaque point est traité comme une matrice de lignes. La multiplication est effectuée avec la matrice de lignes à gauche et cette matrice à droite.<br/> |
