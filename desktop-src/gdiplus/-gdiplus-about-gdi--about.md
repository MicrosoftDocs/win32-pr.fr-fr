# <a name="about-gdi"></a>À propos de GDI+

Windows GDI+ est la partie du système d’exploitation Windows XP ou du système d’exploitation Windows Server 2003 qui fournit les graphiques vectoriels à deux dimensions, la création d’images et la typographie. GDI+ améliore Windows Graphics Device Interface (GDI) (interface de périphérique graphique incluse avec les versions antérieures de Windows) en ajoutant de nouvelles fonctionnalités et en optimisant les fonctionnalités existantes.
