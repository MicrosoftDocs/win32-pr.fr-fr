# <a name="rectfcontains-methods"></a>RectF. Contains, méthodes

Cette rubrique répertorie les méthodes Contains de la classe [**RectF**](/windows/win32/api/gdiplustypes/nl-gdiplustypes-rectf) . Pour obtenir la liste complète des méthodes de la classe **RectF** , consultez [méthodes RectF](-gdiplus-class-rectf-methods.md).

### <a name="overload-list"></a>Liste de surcharge



| Méthode                                                            | Description                                                                                                                                           |
|:------------------------------------------------------------------|:------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Contains (RectF&)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-rectf-contains(inconstrectf_))   | La méthode [**RectF :: Contains**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-rectf-contains(inconstrectf_)) détermine si un autre rectangle est à l’intérieur de ce rectangle.<br/>   |
| [**Contains (PointF&)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-rectf-contains(inconstpointf_))    | La méthode [**RectF :: Contains**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-rectf-contains(inconstpointf_)) détermine si un point se trouve à l’intérieur de ce rectangle.<br/>               |
| [**Contains (réel, réel)**](/previous-versions//ms534956(v=vs.85)) | La méthode [**RectF :: Contains**](/previous-versions//ms534956(v=vs.85)) détermine si le point (*x*, *y*) est à l’intérieur de ce rectangle.<br/> |
