# <a name="fontgetheight-methods"></a>Méthodes font. GetHeight

Cette rubrique répertorie les méthodes GetHeight de la classe [**font**](/windows/win32/api/gdiplusheaders/nl-gdiplusheaders-font) . Pour obtenir la liste complète des méthodes de la classe **font** , consultez [méthodes font](-gdiplus-class-font-methods.md).

### <a name="overload-list"></a>Liste de surcharge



| Méthode                                                                   | Description                                                                                                                                                                                                                                                                                                                                                                                                                              |
|:-------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**GetHeight (réel)**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-font-getheight(inreal))            | La méthode [**font :: GetHeight**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-font-getheight(inreal)) obtient l’interligne, en pixels, de cette police. L'interligne est la distance verticale séparant les lignes de base de deux lignes de texte consécutives. Ainsi, l’interligne comprend l’espace blanc entre les lignes et la hauteur du caractère lui-même.<br/>                                                                                      |
| [**GetHeight (graphiques \* )**](/previous-versions//ms536201(v=vs.85)) | La méthode [**font :: GetHeight**](/previous-versions//ms536201(v=vs.85)) obtient l’interligne de cette police dans l’unité actuelle d’un objet [**Graphics**](/windows/win32/api/gdiplusgraphics/nl-gdiplusgraphics-graphics) spécifié. L'interligne est la distance verticale séparant les lignes de base de deux lignes de texte consécutives. Ainsi, l’interligne comprend l’espace blanc entre les lignes et la hauteur du caractère lui-même.<br/> |
