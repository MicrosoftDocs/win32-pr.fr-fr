# <a name="whats-new-in-gdi"></a>Nouveautés de GDI+

Windows GDI+ diffère de Windows Graphics Device Interface (GDI) de deux manières. Tout d’abord, GDI+ étend les fonctionnalités de GDI en fournissant de nouvelles fonctionnalités, telles que les pinceaux de dégradé et la fusion alpha. Deuxièmement, le modèle de programmation a été révisé pour faciliter la programmation graphique et la flexibilité.

-   [Nouvelles fonctionnalités](-gdiplus-new-features-about.md)
-   [Changements dans le modèle de programmation](-gdiplus-changes-in-the-programming-model-about.md)
