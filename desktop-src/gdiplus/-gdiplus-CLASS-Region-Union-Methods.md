# <a name="regionunion-methods"></a>Region. Union, méthodes

Cette rubrique répertorie les méthodes d’Union de la classe [**Region**](/windows/win32/api/gdiplusheaders/nl-gdiplusheaders-region) . Pour obtenir la liste complète des méthodes pour la classe **Region** , consultez [méthodes Region](-gdiplus-class-region-methods.md).

### <a name="overload-list"></a>Liste de surcharge



| Méthode                                                             | Description                                                                                                                                                                                                                     |
|:-------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Union (Rect&)**](/previous-versions//ms534789(v=vs.85))     | La méthode [**Region :: Union**](/previous-versions//ms534789(v=vs.85)) met à jour cette région pour toutes les parties (intersection et sans intersection) de elle-même et de toutes les parties de l’intérieur du rectangle spécifié.<br/>  |
| [**Union (RectF&)**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-union(inconstrectf_))   | La méthode [**Region :: Union**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-union(inconstrectf_)) met à jour cette région pour toutes les parties (intersection et sans intersection) de elle-même et de toutes les parties de l’intérieur du rectangle spécifié.<br/> |
| [**Union (région \* )**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-union(inconstregion))     | La méthode [**Region :: Union**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-union(inconstregion)) met à jour cette région pour toutes les parties (intersection et sans intersection) de elle-même et de toutes les parties d’une autre région.<br/>                         |
| [**Union (GraphicsPath \* )**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-union(inconstgraphicspath)) | La méthode [**Region :: Union**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-union(inconstgraphicspath)) met à jour cette région pour toutes les parties (intersection et sans intersection) de elle-même et de toutes les parties de l’intérieur du tracé spécifié.<br/>            |
