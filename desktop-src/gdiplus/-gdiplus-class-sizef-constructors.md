# <a name="sizefsizef-constructors"></a>SizeF. SizeF, constructeurs

Cette rubrique répertorie les constructeurs de la classe [**SizeF**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-sizef) . Pour obtenir une liste complète des classes, consultez la **classe SizeF**.

### <a name="overload-list"></a>Liste de surcharge



| Constructeur                                                          | Description                                                                                                                                             |
|:---------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SizeF (SizeF&)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-sizef-sizef(inconstsizef_))            | Crée un objet [**SizeF**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-sizef) et initialise ses membres en copiant les membres d’un autre objet **SizeF** .<br/> |
| [**SizeF (réel, réel)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-sizef-sizef(inreal_inreal)) | Crée un objet [**SizeF**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-sizef) et initialise ses membres de données de **largeur** et de **hauteur** .<br/>                      |
| [**SizeF ()**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-sizef-sizef)                       | Crée un objet [**SizeF**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-sizef) et initialise les membres à zéro. Il s'agit du constructeur par défaut.<br/>           |
