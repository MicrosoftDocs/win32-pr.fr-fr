# <a name="t-network-monitor"></a>T (Moniteur réseau)

<dl> <dt>

<span id="_netmon_total_statistics_gly"></span><span id="_NETMON_TOTAL_STATISTICS_GLY"></span>**statistiques totales**
</dt> <dd>

Statistiques que la carte d’interface réseau (NIC) signale et Moniteur réseau génère. Par exemple, les statistiques pour les trames réseau incluent le nombre de trames que la carte réseau gère, le nombre de trames gérées par la carte réseau, le nombre de trames transmises via le filtre de capture et le nombre de frames stockés dans la mémoire tampon ou le fichier de capture. Pour obtenir la liste complète de ces statistiques, consultez la structure des **statistiques** .

</dd> </dl>
