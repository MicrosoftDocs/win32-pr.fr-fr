# <a name="monitor-interfaces"></a>Analyser les interfaces

Les interfaces COM suivantes sont utilisées pour développer des analyses.



| Interface                              | Description                                                                                                                                             |
|----------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------|
| [IMonitor](imonitor.md)               | Fournit des méthodes qui contrôlent l’opération de surveillance. Cette interface doit être implémentée par le moniteur.                                                     |
| [IMonitorEventer](imonitoreventer.md) | Fournit des méthodes utilisées pour envoyer des événements. Cette interface est fournie par le [*service de contrôle d’analyse*](m.md) (MCSVC). |
