# <a name="writing-an-event-reference-page"></a>Écriture d’une page de référence d’événement

Utilisez les pages de référence des événements (vos solutions ERP) pour décrire comment résoudre un problème détecté par l’expert. Étant donné que vos solutions ERP est constitué de texte HTML, vous pouvez utiliser n’importe quel éditeur HTML ou éditeur de texte pour en créer un.

Le volet Explorateur d’événements Moniteur réseau qui affiche l’ERP prend en charge toutes les fonctionnalités de Microsoft Internet Explorer 4,01 et versions ultérieures. Incorporez les contrôles pris en charge ou le langage de script dans votre ERP pour afficher des informations ou lancer d’autres fonctionnalités à partir de votre ERP. Par exemple, vous pouvez créer un contrôle qui envoie du courrier électronique aux opérations réseau.

Pour plus d’informations sur la création d’une ERP, consultez les rubriques suivantes :

-   [Création d’un expert et surveillance des pages de référence des événements](creating-expert-and-monitor-event-reference-pages.md)
-   [Attribution d’un nom à une page de référence d’événement](naming-an-event-reference-page.md)
-   [Test d’une page de référence d’événement](testing-an-event-reference-page.md)
