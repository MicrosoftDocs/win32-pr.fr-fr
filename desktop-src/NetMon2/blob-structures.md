# <a name="blob-structures"></a>Structures d’objets BLOB

L’API d’assistance d’objet BLOB comprend les structures suivantes.



| Structure                                                        | Description                                                            |
|------------------------------------------------------------------|------------------------------------------------------------------------|
| [**-**](address.md)                                       | Contient une adresse unique et prise en charge.                                  |
| [**ADDRESSPAIR**](addresspair.md)                               | Construit un filtre de capture.                                           |
| [**ADDRESSTABLE**](addresstable.md)                             | Spécifie une table de paires d’adresses.                                    |
| [**ANDEXP**](andexp.md)                                         | Spécifie un tableau de correspondances de modèle.                                 |
| [**TABLE d’objets BLOB \_**](blob-table.md)                                | Spécifie un tableau d’objets BLOB.                                           |
| [**\_valeurs d’en-tête CAPTUREFILE \_**](capturefile-header-values.md) | Définit la structure du fichier d’en-tête Moniteur réseau.              |
| [**CAPTUREFILTER**](capturefilter.md)                           | Spécifie les données de filtre de capture.                                         |
| [**FORMULE**](expression.md)                                 | Spécifie une expression d’homologues structurés dans des instructions et logiques. |
| [**FRAME**](frame.md)                                           | Spécifie un frame.                                                     |
| [**descripteur de FRAME \_**](frame-descriptor.md)                    | Spécifie les détails des frames bruts.                                       |
| [**FRAMETABLE**](frametable.md)                                 | Spécifie un ensemble de frames.                                             |
| [**\_port générique**](generic-port.md)                            | Spécifie un numéro de port utilisé dans les protocoles IP ou IPX.                   |
| [**\_adresse IPX**](ipx-address.md)                              | Spécifie une adresse IPX.                                              |
| [**NETWORKINFO**](networkinfo.md)                               | Spécifie une carte réseau.                                                       |
| [**PATTERNMATCH**](patternmatch.md)                             | Spécifie les éléments de modèle utilisés pour évaluer les frames.                    |
| [**Stead**](trigger.md)                                       | Spécifie qu’un événement se produit à un moment donné.                      |
| [**METTRE à jour un \_ événement**](update-event.md)                            | Met à jour les événements.                                                        |
| [**\_adresse IP Vines \_**](vines-ip-address.md)                   | Spécifie une adresse sur un réseau Vines.                               |
