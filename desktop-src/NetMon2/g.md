# <a name="g-network-monitor"></a>G (Moniteur réseau)

<dl> <dt>

<span id="_netmon_generic_formatter_gly"></span><span id="_NETMON_GENERIC_FORMATTER_GLY"></span>**formateur générique**
</dt> <dd>

Formateur de données de propriété fourni par Moniteur réseau. Le formateur générique peut mettre en forme les données affichées dans le volet d’informations de l’interface utilisateur Moniteur réseau. Quand vous souhaitez Moniteur réseau utiliser le formateur générique, vous devez spécifier la fonction **FormatPropertyInstance** dans le membre **InstanceData** de la structure **PROPERTYINFO** d’une propriété. Voir aussi [sortie de formateur générique](generic-formatter-output.md)

</dd> </dl>
