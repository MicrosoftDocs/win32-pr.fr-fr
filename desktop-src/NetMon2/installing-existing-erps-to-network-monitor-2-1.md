# <a name="installing-existing-erps-to-network-monitor-21"></a>Installation de vos solutions ERP existante sur Moniteur réseau 2,1

Après avoir écrit et nommé une page de référence d’événement (ERP), vous devez l’installer.

**Pour installer une page de référence d’événement**

1.  Créez un répertoire ERP. Ces exemples illustrent les annuaires pour les experts :

    ``` syntax
    C:\Program Files\NetMon2\Bok\Expert\
    ```

2.  Créez un sous-répertoire pour chaque nouvel expert qui affichera le vos solutions ERP. Les listes de sous-répertoires suivantes sont destinées à un expert :

    ``` syntax
    C:\Program Files\NetMon2\Bok\Experts\MyExpert
    ```

3.  Copiez les fichiers ERP dans le sous-répertoire associé. Les exemples suivants fournissent des chemins d’accès complets à l’expert vos solutions ERP installé :

    ``` syntax
    C:\Program Files\NetMon2\Bok\Experts\MyExpert\MyExpert1.htm
    ```
