# <a name="network-monitor-architecture"></a>Architecture Moniteur réseau

L’architecture Moniteur réseau peut être divisée en deux ensembles de concepts de programmation, ceux liés au développement d' [*experts*](e.md) et d' [*analyseurs*](p.md) et ceux [*liés au développement de l’analyse.*](m.md)

Bien que les deux utilisent les mêmes composants pour capturer le trafic réseau, ils utilisent chacun des interfaces différentes pour capturer les données et différents composants afin d’analyser ce qui a été capturé.
