# <a name="d-network-monitor"></a>D (Moniteur réseau)

<dl> <dt>

<span id="_netmon_delayed_capture_gly"></span><span id="_NETMON_DELAYED_CAPTURE_GLY"></span>**capture différée**
</dt> <dd>

Le trafic réseau collecté qui est stocké dans un fichier de capture. Un fournisseur de paquets réseau (NPP) effectue une capture différée. Les analyseurs et les experts utilisent des captures retardées. Voir aussi [*capture*](c.md)et [*capture en temps réel*](/windows)

</dd> <dt>

<span id="_netmon_destination_address_gly"></span><span id="_NETMON_DESTINATION_ADDRESS_GLY"></span>**adresse de destination**
</dt> <dd>

Adresse de l’ordinateur auquel les données sont envoyées.

</dd> <dt>

<span id="_netmon_detail_pane_gly"></span><span id="_NETMON_DETAIL_PANE_GLY"></span>**volet d’informations**
</dt> <dd>

Volet dans la fenêtre frame de Moniteur réseau qui affiche des informations détaillées sur les propriétés du protocole. Par défaut, le volet apparaît au centre de la fenêtre.

</dd> <dt>

<span id="_netmon_display_filter_gly"></span><span id="_NETMON_DISPLAY_FILTER_GLY"></span>**filtre d’affichage**
</dt> <dd>

Filtre utilisé pour limiter la quantité de données capturées qui est affichée dans la fenêtre de l’observateur de trames. Un filtre d’affichage est une pile suffixée d’objets qui peut être un opérateur, une valeur ou une propriété.

</dd> </dl>
