# <a name="monitor-event-reference-pages"></a>Pages de référence des événements Monitor

Les pages de référence des événements (vos solutions ERP) peuvent être utilisées pour connecter votre moniteur à l’interface utilisateur Moniteur réseau. Ils peuvent être utilisés pour alerter les utilisateurs finaux sur les conditions et problèmes détectés, ou ils peuvent être utilisés pour fournir des corrections suggérées ou des données explicatives.
