# <a name="a-network-monitor"></a>A (Moniteur réseau)

<dl> <dt>

<span id="_netmon_address_gly"></span><span id="_NETMON_ADDRESS_GLY"></span>**-**
</dt> <dd>

Identificateur d’un nœud réseau qui différencie un nœud d’un autre sur un réseau. Également appelée adresse *réseau* ou *adresse Mac*. Dans Microsoft Systems Management Server (SMS), une adresse est utilisée pour connecter un site et un système de site. Un expéditeur utilise une adresse spécifique pour envoyer des instructions et des données à d’autres sites.

</dd> <dt>

<span id="_netmon_address_pair_gly"></span><span id="_NETMON_ADDRESS_PAIR_GLY"></span>**paire d’adresses**
</dt> <dd>

Adresses source et de destination d’une transmission de données. La destination peut être une adresse unique, une multidiffusion ou une diffusion. Dans les données capturées, l’adresse source est une adresse spécifique. Dans les spécifications de filtre, l’adresse source peut être une adresse spécifique, ou n’importe laquelle, mais elle ne peut pas être diffusée ou multidiffusion.

</dd> </dl>
