# <a name="obtaining-network-monitor-21"></a>Obtention de Moniteur réseau 2,1

Moniteur réseau 2,1 est disponible dans le cadre d’une installation distincte sur le support Microsoft Windows Systems Management Server (SMS) 2003 ou Windows Server 2003.

> [!Note]  
> actuellement, Moniteur réseau 2,1 n’est pas disponible dans le cadre du SDK Windows ; il ne l’est pas non plus dans Windows Vista.

 

## <a name="microsoft-windows-systems-management-server-2003"></a>Microsoft Windows Systems Management Server 2003

Installez Moniteur réseau 2,1 en exécutant « NetmonSetup.exe » à partir du \\ dossier « netmon \\ i386 » sur le CD-ROM du produit SMS 2003.

Si vous effectuez une mise à niveau à partir de SMS 2,0, vous devez d’abord désinstaller Moniteur réseau 2,0 en utilisant **Ajout/suppression de programmes** dans le panneau de configuration.

## <a name="microsoft-windows-server-2003"></a>Microsoft Windows Server 2003

suivez les instructions ci-dessous pour installer Moniteur réseau 2,1 à partir du support Windows Server 2003.

-   cliquez sur **démarrer**, pointez sur **Paramètres** , puis sélectionnez l’élément de menu **panneau de configuration** .
-   Double-cliquez sur **Ajout/suppression de programmes**.
-   cliquez sur **ajouter/supprimer des composants de Windows**.
-   Cliquez sur **outils de gestion et de surveillance**, puis sur **Détails**.
-   Cliquez pour activer la case à cocher **Moniteur réseau Tools** , puis cliquez sur **OK**.
-   Cliquez sur **Suivant**.
