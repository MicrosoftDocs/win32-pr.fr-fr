# <a name="installing-network-monitor-files"></a>Installation des fichiers Moniteur réseau

Pour installer et exécuter la partie Moniteur réseau du kit de développement logiciel (SDK) de la plateforme, procédez comme suit.

**Pour installer et exécuter Moniteur réseau fichiers du kit de développement logiciel (SDK) de plateforme**

1.  Pour installer les fichiers, pendant l’installation du kit de développement Platform SDK, sélectionnez les documents Moniteur réseau et l’exemple de code.

    Une fois l’installation terminée, le programme d’installation du kit de développement Platform SDK crée un fichier de commandes, Setenv.bat.

2.  Pour initialiser les variables d’environnement requises, exécutez Setenv.bat.
3.  Pour compiler les exemples, exécutez **NMAKE**, qui est inclus dans Microsoft Visual Studio.
