# <a name="b-network-monitor"></a>B (Moniteur réseau)

<dl> <dt>

<span id="_netmon_binary_large_object_gly"></span><span id="_NETMON_BINARY_LARGE_OBJECT_GLY"></span>**objet BLOB (Binary Large Object)**
</dt> <dd>

Structure de données générique de Moniteur réseau. Vous pouvez utiliser un objet BLOB pour communiquer et stocker des informations de configuration et d’emplacement, ainsi que pour stocker des informations spécifiques à l’application, sans affecter les autres données d’un objet BLOB.

</dd> <dt>

<span id="_netmon_blob_gly"></span><span id="_NETMON_BLOB_GLY"></span>**OBJET BLOB**
</dt> <dd>

Consultez objet binaire volumineux.

</dd> <dt>

<span id="_netmon_buffer_gly"></span><span id="_NETMON_BUFFER_GLY"></span>**mémoire tampon**
</dt> <dd>

Une partie réservée de la mémoire dans laquelle les données sont conservées temporairement en attente pour effectuer un transfert vers ou à partir d’un périphérique de stockage ou d’un autre emplacement en mémoire.

</dd> </dl>
