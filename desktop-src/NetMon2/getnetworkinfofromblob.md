# <a name="getnetworkinfofromblob-function"></a>GetNetworkInfoFromBlob fonction)

La fonction **GetNetworkInfoFromBlob** récupère les informations réseau à partir d’un objet BLOB donné.

## <a name="syntax"></a>Syntaxe


```C++
DWORD GetNetworkInfoFromBlob(
  _In_    HBLOB         hBlob,
  _Inout_ LPNETWORKINFO lpNetworkInfo
);
```



## <a name="parameters"></a>Paramètres

<dl> <dt>

*hBlob* \[ dans\]
</dt> <dd>

Handle vers un objet BLOB.

</dd> <dt>

*lpNetworkInfo* \[ in, out\]
</dt> <dd>

Pointeur vers la structure [NETWORKINFO](networkinfo.md) allouée par l’utilisateur qui est remplie.

</dd> </dl>

## <a name="return-value"></a>Valeur de retour

Si la fonction **GetNetworkInfoFromBlob** réussit, la valeur de retour est NMERR \_ Success.

Si la fonction échoue, la valeur de retour est une valeur NMERR qui décrit l’erreur.

## <a name="remarks"></a>Notes

Les informations réseau sont stockées dans la section BLOB **NetworkInfo** de la catégorie **propriétaire** NPP d’objet BLOB.
