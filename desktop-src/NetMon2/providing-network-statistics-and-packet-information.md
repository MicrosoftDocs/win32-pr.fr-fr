# <a name="providing-network-statistics-and-packet-information"></a>Fournir des statistiques réseau et des informations sur les paquets

Le fournisseur de paquets réseau (NPP) est une DLL qui communique avec le pilote Moniteur réseau et fournit des statistiques réseau et des données de paquets.

Les méthodes les plus courantes pour utiliser le NPP sont les suivantes :

-   Statistiques réseau.
-   Frames passés à l’application en temps réel.
-   Les paquets sont accessibles uniquement après l’arrêt de la capture.

Cette section comprend les rubriques suivantes :

-   [Sélection d’une carte réseau](selecting-a-nic-using-getnppblobfromui.md)
-   [Utilisation de GetNPPBlobTable](using-getnppblobtable.md)
