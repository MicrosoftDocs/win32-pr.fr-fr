# <a name="event-reference-page"></a>Page de référence des événements

Cette section décrit la page de référence des événements (ERP), un document HTML que vous pouvez écrire pour afficher les événements qui se produisent pendant le fonctionnement d’un expert en Moniteur réseau. Une fois que vous avez défini les conditions requises pour la création d’experts, vous pouvez utiliser vos solutions ERP pour aider Moniteur réseau résoudre les problèmes réseau.
