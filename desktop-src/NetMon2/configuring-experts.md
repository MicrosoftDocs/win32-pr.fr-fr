# <a name="configuring-experts"></a>Configuration d’experts

Pour configurer des experts, utilisez les boîtes de dialogue personnalisées et la fonction [**configurer**](configure.md) . Lorsque vous écrivez des experts du moniteur réseau, vous avez le choix. Vous pouvez créer un ensemble d’éléments de configuration par défaut qui peuvent être utilisés lors de la première exécution de l’expert, ou vous pouvez autoriser les utilisateurs Moniteur réseau à configurer l’expert au moment de l’exécution. Les configurations de chaque expert sont stockées par Moniteur réseau. En option, les utilisateurs peuvent rétablir les paramètres par défaut d’origine de la configuration des experts. N’oubliez pas que vous pouvez contourner l' **utilisation de la configuration en** développant un expert qui passe une configuration fixe à Moniteur réseau.
