# <a name="network-monitor"></a>Moniteur réseau

## <a name="purpose"></a>Objectif

Moniteur réseau capture le trafic réseau à des fins d’affichage et d’analyse. Elle vous permet d’effectuer des tâches telles que l’analyse de données capturées précédemment dans des méthodes définies par l’utilisateur et l’extraction de données à partir d’analyseurs de protocole définis.

## <a name="developer-audience"></a>Développeurs concernés

Tous les jeux d’API fournis par Moniteur réseau sont accessibles à l’aide de C/C++. Les développeurs qui travaillent également avec des [*fournisseurs de paquets réseau*](n.md) doivent également être familiarisés avec com.

## <a name="run-time-requirements"></a>Conditions d’exécution

Pour appeler à partir de l’API Moniteur réseau, vous devez exécuter Windows NT Server 4,0, Windows 2000 Server ou Windows Server 2003, ou disposer de Microsoft Systems Management Server installé.

Le pilote NPP et les fonctionnalités prises en charge incluent toutes les versions de Windows NT 4,0 et Windows 2000 Server.

> [!Note]  
> Moniteur réseau 2,1 et versions antérieures ne sont pas pris en charge sur Windows Vista et versions ultérieures.
