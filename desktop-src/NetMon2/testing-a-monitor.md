# <a name="testing-a-monitor"></a>Test d’une analyse

Une fois qu’une analyse est installée, vous pouvez la tester avec des données actives à partir de vos opérations réseau, ou vous pouvez tester un ensemble d’événements connus en capturant les données avec Moniteur réseau, en enregistrant le fichier de capture et en transmettant les données localement. (Pour plus d’informations sur la transmission d’un fichier de capture sur un réseau, consultez Moniteur réseau l’aide en ligne.)
