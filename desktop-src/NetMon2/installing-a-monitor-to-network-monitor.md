# <a name="installing-a-monitor-to-network-monitor"></a>Installation d’une analyse pour Moniteur réseau

L’installation d’une DLL d’analyse est un processus simple. Tout d’abord, copiez la DLL et le formulaire de configuration HTML associé dans le sous-répertoire des analyses NPP (par exemple, C : \\ Windows \\ system32 \\ NPP \\ Monitors). Une fois copié dans le répertoire, le moniteur est reconnu et disponible la prochaine fois que l’outil de contrôle de l’analyse ou le service de contrôle d’analyse (MCSVC) est activé.

Par exemple, si la variable d’environnement% SystemRoot% est C : \\ Windows, le sous-répertoire Monitor est c : \\ Windows \\ system32 \\ NPP \\ Monitors.
