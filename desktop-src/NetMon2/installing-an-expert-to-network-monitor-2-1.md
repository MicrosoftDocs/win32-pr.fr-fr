# <a name="installing-an-expert-to-network-monitor-21"></a>Installation d’un expert sur Moniteur réseau 2,1

Pour installer une DLL d’expert, copiez-la dans le \\ sous-répertoire experts Moniteur réseau (par exemple, C : \\ Program Files \\ NetMon2 \\ experts). Après l’installation, l’expert sera reconnu et disponible lors du prochain démarrage de Moniteur réseau.

L’illustration suivante montre la DLL MyExpert, MyExpert créée à partir de l’exemple de code et copiée dans le \\ sous-répertoire experts.
