# <a name="specifying-protocol-information"></a>Spécification des informations de protocole

Le pilote de Moniteur réseau peut ajouter des informations à la table de frames en spécifiant le protocole de bas niveau (IP ou IPX) qui a été trouvé et le décalage au début du protocole. Pour ajouter ces informations à la table de frames, définissez le filtre de capture ANY.
