# <a name="capture-filters"></a>Filtres de capture

Le kit de développement logiciel (SDK) Moniteur réseau comprend des filtres de capture, avec lesquels vous pouvez limiter le nombre de trames reçues par une application de fournisseur de paquets réseau (NPP) pendant son fonctionnement. En spécifiant les trames de trafic réseau que vous voulez qu’une application de surveillance personnalisée ou NPP examine, vous pouvez améliorer son efficacité de traitement.

Cette section comprend les rubriques suivantes :

-   [À propos des filtres de capture](about-capture-filters.md)
-   [**Structure CAPTUREFILTER**](the-capturefilter-structure.md)
