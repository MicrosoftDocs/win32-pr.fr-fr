# <a name="e-network-monitor"></a>E (Moniteur réseau)

<dl> <dt>

<span id="_netmon_erp_gly"></span><span id="_NETMON_ERP_GLY"></span>**ERP**
</dt> <dd>

Consultez la page de référence des événements.

</dd> <dt>

<span id="_netmon_event_gly"></span><span id="_NETMON_EVENT_GLY"></span>**événement**
</dt> <dd>

Action importante qui se produit dans le système d’exploitation ou dans une application telle que Moniteur réseau. Un événement peut notifier les utilisateurs, écrire une entrée dans un journal ou les deux. Dans Moniteur réseau un expert peut signaler des événements qui s’affichent dans la fenêtre de l’observateur d’événements.

</dd> <dt>

<span id="_netmon_event_reference_page_gly"></span><span id="_NETMON_EVENT_REFERENCE_PAGE_GLY"></span>**page de référence des événements (ERP)**
</dt> <dd>

Document HTML affiché dans le Moniteur réseau fenêtre Observateur d’événements et associé à un événement défini dans un expert.

</dd> <dt>

<span id="_netmon_expert_gly"></span><span id="_NETMON_EXPERT_GLY"></span>**conseils**
</dt> <dd>

DLL qui manipule les frames collectés dans un fichier de capture. Un expert peut fournir un résumé ou un rapport basé sur les données capturées. Par exemple, un rapport utilisateur supérieur.

</dd> </dl>
