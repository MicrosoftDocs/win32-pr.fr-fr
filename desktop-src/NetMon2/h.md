# <a name="h-network-monitor"></a>H (Moniteur réseau)

<dl> <dt>

<span id="_netmon_handoff_set_gly"></span><span id="_NETMON_HANDOFF_SET_GLY"></span>**jeu de remise**
</dt> <dd>

Ensemble de protocoles qui suit un protocole. Un analyseur utilise le jeu de remise d’un protocole si l’analyseur peut détecter le protocole suivant à partir des données de la capture. Voir aussi [ *jeu de suivi*](f.md)

</dd> <dt>

<span id="_netmon_handoff_table_gly"></span><span id="_NETMON_HANDOFF_TABLE_GLY"></span>**table de remise**
</dt> <dd>

Table qui contient le jeu de remise de l’analyseur. Vous pouvez utiliser une table de remise pour accéder aux informations du jeu de remise, sans accéder au fichier INI défini par l’analyseur. Les informations du jeu de remise sont stockées dans le fichier INI de chaque analyseur. Voir aussi *jeu de remise*

</dd> <dt>

<span id="_netmon_hex_pane_gly"></span><span id="_NETMON_HEX_PANE_GLY"></span>**volet hex**
</dt> <dd>

Volet de la fenêtre de l’observateur de frames Moniteur réseau qui affiche les octets hexadécimaux transmis dans les frames inclus dans une capture. Par défaut, le volet hex s’affiche au bas de la fenêtre.

</dd> </dl>
