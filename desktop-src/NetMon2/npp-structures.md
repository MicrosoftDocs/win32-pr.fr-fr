# <a name="npp-structures"></a>Structures NPP

Cette section décrit les structures NPP utilisées par les méthodes NPP. Ces structures sont utilisées pour récupérer des statistiques, fournir l’état du système et des informations statistiques, et indiquer les ordinateurs qui exécutent Moniteur réseau. Les tableaux suivants décrivent ces structures.



| Structures NPP                     | Description                                                                                                                      |
|------------------------------------|----------------------------------------------------------------------------------------------------------------------------------|
| [SESSIONSTATS](sessionstats.md)   | Fournit des informations de session lorsque les statistiques de conversation sont récupérées.                                                         |
| [STATIONSTATS](stationstats.md)   | Fournit des statistiques sur une [*station*](s.md)spécifique.                                                     |
| [PORTENT](statistics.md)       | Fournit des statistiques réseau lorsque les statistiques totales sont récupérées et lorsque la capture en cours est suspendue ou arrêtée.              |
| [QUERYTABLE](querytable.md)       | Indique les ordinateurs qui utilisent Moniteur réseau.                                                                             |
| [STATIONQUERY](stationquery.md)   | Fournit des informations sur un ordinateur qui utilise Moniteur réseau. Cette structure est utilisée par la structure de **table de table** NPP. |
| [NETWORKSTATUS](networkstatus.md) | Indique l’état du réseau actuel en termes d’État du système.                                                                        |
