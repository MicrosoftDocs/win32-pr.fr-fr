# <a name="system-requirements"></a>Configuration requise

Les configurations d’ordinateur minimales diffèrent ; à des fins de développement Moniteur réseau, il est recommandé qu’un développeur ait un ordinateur qui comprend les spécifications minimales suivantes :

-   Processeur Pentium
-   32 MO DE RAM
-   30 Mo d’espace disponible sur le disque dur
-   Moniteur vidéo et prise en charge de cartes vidéo pour 256 couleurs à une résolution de 800 X 600
-   Connexion réseau
-   Microsoft Internet Explorer 4,01
-   Microsoft Visual C++ version 5,0

Le test d’une application développée avec Moniteur réseau nécessite un ordinateur avec les spécifications minimales suivantes :

-   486 processeur exécutant au moins 50 MHz
-   16 MO DE RAM
-   20 Mo d’espace disque disponible
-   Connexion réseau
