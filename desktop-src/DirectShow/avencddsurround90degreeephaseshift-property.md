# <a name="avencddsurround90degreeephaseshift-property"></a>Propriété AVEncDDSurround90DegreeePhaseShift

Spécifie si un décalage de phase de 90 degré est appliqué aux canaux surround dans un flux audio Dolby Digital. Cette propriété s’applique aux encodeurs audio Dolby Digital.

Cette propriété est en lecture/écriture.

## <a name="data-type"></a>Type de données

**Variante \_ BOOL** (**VT \_ bool**)

## <a name="property-guid"></a>Guid de propriété

**CODECAPI \_ AVEncDDSurround90DegreeePhaseShift**
