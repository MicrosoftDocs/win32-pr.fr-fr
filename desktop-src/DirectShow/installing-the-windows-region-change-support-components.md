# <a name="installing-the-windows-region-change-support-components"></a>installation de la région Windows modification des composants de Support

les développeurs d’applications peuvent utiliser la prise en charge de modification de la région des dvd intégrée à Windows pour modifier la région dans les applications de lecture de dvd.

sur Windows 2000 et versions ultérieures, le composant qui fournit la prise en charge du changement de région de DVD se trouve dans le storprop.dll DLL (disponible dans% windir% \\ system32). Cette DLL est installée par défaut et aucune étape supplémentaire n’est requise pour s’assurer que cette DLL est disponible.
