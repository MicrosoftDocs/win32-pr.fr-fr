# <a name="unicable-frequency-table"></a>Table de fréquence Unicable

En raison de sa longueur, la table Unicable n’est pas incluse ici. L’objectif de la table Unicable est d’inclure un ensemble complet de fréquences (fondamentalement toutes les autres tables combinées, ainsi que des observations concrètes) dans une table pour faciliter une analyse automatique des canaux sans avoir recours à une recherche linéaire sur l’ensemble du spectre de fréquences. Pour les fréquences qui ne sont pas incluses dans ce tableau, le consommateur est tenu d’entrer une fréquence via une interface utilisateur personnalisée, ce qui entraîne une entrée de remplacement de fréquence dans le registre.
