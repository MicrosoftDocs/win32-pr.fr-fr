# <a name="avencdddchighpassfilter-property"></a>Propriété AVEncDDDCHighPassFilter

Spécifie si un filtre de passe élevé bloquant le contrôleur de l’objet est appliqué à un flux audio Dolby Digital. Cette propriété s’applique aux encodeurs audio Dolby Digital.

Cette propriété est en lecture/écriture.

## <a name="data-type"></a>Type de données

**Variante \_ BOOL** (**VT \_ bool**)

## <a name="property-guid"></a>Guid de propriété

**CODECAPI \_ AVEncDDDCHighPassFilter**
