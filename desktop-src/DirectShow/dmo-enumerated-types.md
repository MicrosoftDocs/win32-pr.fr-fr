# <a name="dmo-enumerated-types"></a>DMO Types énumérés

cette section décrit les types énumérés Microsoft DirectX Media Object (DMO).



| Type énuméré                                                            | Description                                                                  |
|----------------------------------------------------------------------------|------------------------------------------------------------------------------|
| [**DMO \_ \_indicateurs enum**](/previous-versions/windows/desktop/api/Dmoreg/ne-dmoreg-dmo_enum_flags)                                 | Définit des indicateurs qui spécifient des critères de recherche lors de l’énumération de DMOs.            |
| [**DMO \_ \_indicateurs de \_ tampon de données d’entrée \_**](/previous-versions/windows/desktop/api/Mediaobj/ne-mediaobj-_dmo_input_data_buffer_flags)     | Définit des indicateurs qui décrivent une mémoire tampon d’entrée.                                 |
| [**DMO \_ \_indicateurs d' \_ informations de flux d’entrée \_**](/previous-versions/windows/desktop/api/Mediaobj/ne-mediaobj-_dmo_input_stream_info_flags)     | Définit des indicateurs qui décrivent un flux d’entrée.                                 |
| [**DMO \_ \_indicateurs de \_ tampon de données de sortie \_**](/previous-versions/windows/desktop/api/Mediaobj/ne-mediaobj-_dmo_output_data_buffer_flags)   | Définit des indicateurs qui décrivent une mémoire tampon de sortie.                                |
| [**DMO \_ \_indicateurs d' \_ informations de flux de sortie \_**](/previous-versions/windows/desktop/api/Mediaobj/ne-mediaobj-_dmo_output_stream_info_flags)   | Définit des indicateurs qui décrivent un flux de sortie.                                |
| [**DMO \_ TRAITER \_ les \_ indicateurs de sortie**](/previous-versions/windows/desktop/api/Mediaobj/ne-mediaobj-_dmo_process_output_flags)            | Définit les indicateurs qui spécifient les demandes de traitement de sortie.                       |
| [**DMO \_ indicateurs d’inscription \_**](/previous-versions/windows/desktop/api/Dmoreg/ne-dmoreg-dmo_register_flags)                         | Définit des indicateurs qui spécifient des informations de Registre pour un DMO.                   |
| [**DMO \_ DÉFINIR \_ des \_ indicateurs de type**](/previous-versions/windows/desktop/api/Mediaobj/ne-mediaobj-_dmo_set_type_flags)                        | Définit des indicateurs pour définir le type de média sur un flux.                        |
| [**DMO \_ \_indicateurs de \_ flux de sortie vidéo \_**](/previous-versions/windows/desktop/api/Mediaobj/ne-mediaobj-_dmo_video_output_stream_flags) | Définit des indicateurs qui décrivent les fonctionnalités demandées pour les optimisations vidéo.      |
| [**\_type de courbe MP \_**](/previous-versions/windows/desktop/api/Medparam/ne-medparam-mp_curve_type)                                   | Définit la courbe qu’un paramètre multimédia suit dans un segment d’enveloppe. |
| [**TYPE de Pack d’e \_**](/previous-versions/windows/desktop/api/Medparam/ne-medparam-mp_type)                                                | Spécifie le type de données d’un paramètre.                                     |
