Cette section décrit les types énumérés Microsoft DirectX Media Object (DMO).



| Type énuméré                                                            | Description                                                                  |
|----------------------------------------------------------------------------|------------------------------------------------------------------------------|
| [**\_indicateurs d’énumération DMO \_**](/previous-versions/windows/desktop/api/Dmoreg/ne-dmoreg-dmo_enum_flags)                                 | Définit des indicateurs qui spécifient des critères de recherche lors de l’énumération de DMOs.            |
| [**\_indicateurs de \_ tampon de données d’entrée DMO \_ \_**](/previous-versions/windows/desktop/api/Mediaobj/ne-mediaobj-_dmo_input_data_buffer_flags)     | Définit des indicateurs qui décrivent une mémoire tampon d’entrée.                                 |
| [**\_indicateurs d' \_ informations de flux d’entrée \_ DMO \_**](/previous-versions/windows/desktop/api/Mediaobj/ne-mediaobj-_dmo_input_stream_info_flags)     | Définit des indicateurs qui décrivent un flux d’entrée.                                 |
| [**\_indicateurs de \_ tampon de données de sortie DMO \_ \_**](/previous-versions/windows/desktop/api/Mediaobj/ne-mediaobj-_dmo_output_data_buffer_flags)   | Définit des indicateurs qui décrivent une mémoire tampon de sortie.                                |
| [**\_indicateurs d' \_ informations de flux de sortie DMO \_ \_**](/previous-versions/windows/desktop/api/Mediaobj/ne-mediaobj-_dmo_output_stream_info_flags)   | Définit des indicateurs qui décrivent un flux de sortie.                                |
