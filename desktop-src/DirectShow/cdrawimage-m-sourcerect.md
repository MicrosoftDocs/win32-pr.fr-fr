# <a name="cdrawimagem_sourcerect-member"></a>CDrawImage :: m \_ SourceRect, membre

La variable de membre **m \_ SourceRect** spécifie le rectangle source pour le dessin. La valeur initiale est un rectangle vide, indiquant que la totalité de l’image doit être dessinée.

## <a name="syntax"></a>Syntaxe


```C++
RECT m_SourceRect;
```
