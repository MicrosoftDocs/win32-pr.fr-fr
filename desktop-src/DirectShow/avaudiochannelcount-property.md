# <a name="avaudiochannelcount-property"></a>Propriété AVAudioChannelCount

Obtient le nombre de canaux dans le flux de bits audio.

Cette propriété est en lecture seule.

## <a name="data-type"></a>Type de données

**UInt32** (**VT \_ UI4**)

## <a name="property-guid"></a>Guid de propriété

**CODECAPI \_ AVAudioChannelCount**

## <a name="remarks"></a>Notes

Le nombre de canaux comprend le canal de l’effet de fréquence faible (LFE), le cas échéant.
