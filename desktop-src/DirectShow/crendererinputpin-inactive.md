# <a name="crendererinputpininactive-method"></a>CRendererInputPin. inactive, méthode

La `Inactive` méthode notifie le code confidentiel que le filtre n’est plus actif. Cette méthode remplace la méthode [**CBaseInputPin :: inactive**](cbaseinputpin-inactive.md) .

## <a name="syntax"></a>Syntaxe


```C++
HRESULT Inactive();
```



## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.

## <a name="return-value"></a>Valeur retournée

Retourne une valeur **HRESULT** .
