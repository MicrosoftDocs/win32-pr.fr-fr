# <a name="cbasepropertypagem_dialogid-member"></a>CBasePropertyPage :: m \_ DialogId, membre

La `m_DialogId` variable membre spécifie l’identificateur de ressource pour la boîte de dialogue. Cette variable est initialisée dans la méthode du constructeur.

## <a name="syntax"></a>Syntaxe


```C++
int m_DialogId;
```
