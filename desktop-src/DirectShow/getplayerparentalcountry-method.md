# <a name="getplayerparentalcountry-method"></a>Méthode GetPlayerParentalCountry

> [!Note]  
> ce composant peut être utilisé dans les systèmes d’exploitation Microsoft Windows 2000, Windows XP et Windows Server 2003. Il sera peut-être modifié ou indisponible dans les versions ultérieures.

 

La `GetPlayerParentalCountry` méthode récupère le pays ou la région actuel tels qu’ils sont définis dans l’objet **mswebdvd** .

``` syntax
[ iCountryCode = ] MSWebDVD.GetPlayerParentalCountry()
```

## <a name="return-value"></a>Valeur renvoyée

Retourne une valeur entière avec le code pays/région.
