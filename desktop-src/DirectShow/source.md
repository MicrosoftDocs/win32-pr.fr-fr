# <a name="source"></a>Source

> [!Note]  
> \[Déconseillé. Cette API peut être supprimée des futures versions de Windows.\]

 

L’objet source représente un élément source dans la chronologie. Pour créer cet objet, appelez la méthode [**IAMTimeline :: CreateEmptyNode**](iamtimeline-createemptynode.md) .

L’objet source expose les interfaces suivantes :

-   [**IAMTimelineEffectable**](iamtimelineeffectable.md)
-   [**IAMTimelineObj**](iamtimelineobj.md)
-   [**IAMTimelineSrc**](iamtimelinesrc.md)
-   [**IAMTimelineSplittable**](iamtimelinesplittable.md)
