# <a name="time-format-guids"></a>GUID de format d’heure

Les identificateurs globaux uniques (GUID) suivants définissent des formats d’heure différents.



| GUID                                                                                                                                                                                       | Description                                       |
|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:--------------------------------------------------|
| <span id="TIME_FORMAT_NONE"></span><span id="time_format_none"></span><dl> <dt>**FORMAT d’heure \_ \_ aucun**</dt> </dl>                    | Aucun format.<br/>                             |
| <span id="TIME_FORMAT_FRAME"></span><span id="time_format_frame"></span><dl> <dt>**\_cadre de format d’heure \_**</dt> </dl>                 | Images vidéo.<br/>                          |
| <span id="TIME_FORMAT_SAMPLE"></span><span id="time_format_sample"></span><dl> <dt>**\_exemple de format d’heure \_**</dt> </dl>              | Exemples dans le flux.<br/>                 |
| <span id="TIME_FORMAT_FIELD"></span><span id="time_format_field"></span><dl> <dt>**\_champ de format d’heure \_**</dt> </dl>                 | Champs vidéo entrelacés.<br/>               |
| <span id="TIME_FORMAT_BYTE"></span><span id="time_format_byte"></span><dl> <dt>**\_octet de format d’heure \_**</dt> </dl>                    | Offset d’octet dans le flux.<br/>         |
| <span id="TIME_FORMAT_MEDIA_TIME"></span><span id="time_format_media_time"></span><dl> <dt>**heure du \_ format du \_ support \_**</dt> </dl> | Temps de référence (unités de 100 nanosecondes).<br/> |
