# <a name="cbaseinputpinm_breadonly-member"></a>CBaseInputPin :: m \_ bReadOnly, membre

Indicateur qui spécifie si l’allocateur produit des exemples de supports en lecture seule. Si la **valeur est true**, l’allocateur produit des échantillons en lecture seule.

## <a name="syntax"></a>Syntaxe


```C++
BYTE m_bReadOnly;
```
