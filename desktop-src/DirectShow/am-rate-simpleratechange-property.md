# <a name="am_rate_simpleratechange-property"></a>\_ \_ Propriété SIMPLERATECHANGE rate AM

Cette propriété est utilisée pour envoyer des modifications de taux au décodeur. Le type de données de cette propriété est une structure **am \_ SimpleRateChange** , qui donne la nouvelle vitesse de lecture et l’heure de présentation à laquelle le nouveau taux prend effet.



| Étiquette | Valeur |
|-------------------|-----------------------------------------------------|
| GUID de jeu de propriétés | AM \_ KSPROPSETID \_ TSRateChange                       |
| ID de propriété       | \_Taux am \_ SimpleRateChange                          |
| Type de données         | [**AM \_ SimpleRateChange**](/previous-versions/windows/desktop/api/Dvdmedia/ns-dvdmedia-am_simpleratechange) |
