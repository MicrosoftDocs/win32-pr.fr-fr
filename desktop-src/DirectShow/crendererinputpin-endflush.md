# <a name="crendererinputpinendflush-method"></a>Méthode CRendererInputPin. EndFlush

La `EndFlush` méthode termine une opération de vidage. Cette méthode remplace la méthode [**CBaseInputPin :: EndFlush**](cbaseinputpin-endflush.md) .

## <a name="syntax"></a>Syntaxe


```C++
HRESULT EndFlush();
```



## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.

## <a name="return-value"></a>Valeur de retour

Retourne une valeur **HRESULT** .
