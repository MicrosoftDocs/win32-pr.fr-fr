# <a name="avencddlfelowpassfilter-property"></a>Propriété AVEncDDLFELowPassFilter

Spécifie si un filtre passe-bas est appliqué au canal de l’effet faible fréquence (LFE) dans un flux audio Dolby Digital. Cette propriété s’applique aux encodeurs audio Dolby Digital.

Cette propriété est en lecture/écriture.

## <a name="data-type"></a>Type de données

**Variante \_ BOOL** (**VT \_ bool**)

## <a name="property-guid"></a>Guid de propriété

**CODECAPI \_ AVEncDDLFELowPassFilter**
