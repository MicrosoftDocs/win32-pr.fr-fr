# <a name="dmo-functions"></a>Fonctions DMO

Cette section décrit les fonctions de Microsoft DirectX Media Object (DMO).



| Fonction                                             | Description                                                   |
|------------------------------------------------------|---------------------------------------------------------------|
| [**DMOEnum**](/previous-versions/windows/desktop/api/Dmoreg/nf-dmoreg-dmoenum)                           | Énumère les DMOs répertoriés dans le registre.                       |
| [**DMOGetName**](/previous-versions/windows/desktop/api/Dmoreg/nf-dmoreg-dmogetname)                     | Récupère le nom d’un DMO à partir du Registre.                |
| [**DMOGetTypes**](/previous-versions/windows/desktop/api/Dmoreg/nf-dmoreg-dmogettypes)                   | Récupère les types de supports enregistrés pour une DMO.               |
| [**DMORegister**](/previous-versions/windows/desktop/api/Dmoreg/nf-dmoreg-dmoregister)                   | Inscrit un DMO.                                              |
| [**DMOUnregister**](/previous-versions/windows/desktop/api/Dmoreg/nf-dmoreg-dmounregister)               | Annule l’inscription d’un DMO.                                            |
| [**MoCopyMediaType**](/previous-versions/windows/desktop/api/Dmort/nf-dmort-mocopymediatype)           | Copie une structure de type de média vers une autre.                   |
| [**MoCreateMediaType**](/previous-versions/windows/desktop/api/Dmort/nf-dmort-mocreatemediatype)       | Alloue une nouvelle structure de type de média.                         |
| [**MoDeleteMediaType**](/previous-versions/windows/desktop/api/Dmort/nf-dmort-modeletemediatype)       | Supprime une structure de type de média précédemment allouée. |
| [**MoDuplicateMediaType**](/previous-versions/windows/desktop/api/Dmort/nf-dmort-moduplicatemediatype) | Duplique une structure de type de média.                            |
| [**MoFreeMediaType**](/previous-versions/windows/desktop/api/Dmort/nf-dmort-mofreemediatype)           | Libère les membres alloués d’une structure de type de média.        |
| [**MoInitMediaType**](/previous-versions/windows/desktop/api/Dmort/nf-dmort-moinitmediatype)           | Initialise une structure de type de média.                           |
