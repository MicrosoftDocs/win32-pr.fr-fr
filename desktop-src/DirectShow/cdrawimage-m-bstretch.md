# <a name="cdrawimagem_bstretch-member"></a>CDrawImage :: m \_ bStretch, membre

La `m_bStretch` variable membre indique si l’image vidéo doit être étirée pour s’ajuster à la fenêtre de destination. La valeur est **false** si le rectangle source est de la même taille que le rectangle de destination, ou **true** dans le cas contraire.

## <a name="syntax"></a>Syntaxe


```C++
BOOL m_bStretch;
```
