# <a name="cdrawimagem_targetrect-member"></a>CDrawImage :: m \_ TargetRect, membre

La `m_TargetRect` variable membre spécifie le rectangle cible pour le dessin. La valeur initiale est un rectangle vide, indiquant que l’image doit être dessinée dans l’intégralité de la fenêtre de destination.

## <a name="syntax"></a>Syntaxe


```C++
RECT m_TargetRect;
```
