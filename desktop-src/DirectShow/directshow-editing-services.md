# <a name="directshow-editing-services"></a>Services d’édition DirectShow

\[Cette API n’est pas prise en charge et peut être modifiée ou non disponible à l’avenir.\]

Les services d’édition DirectShow (DES) sont une interface de programmation d’applications (API) qui simplifie grandement les tâches impliquées dans l’édition vidéo. DES est basé sur l’architecture de base de DirectShow. Elle résume la plus grande partie de la complexité de DirectShow et fournit un ensemble d’interfaces conçues spécifiquement pour la manipulation des projets d’édition vidéo. En tant que développeur d’applications, vous bénéficiez des avantages de DirectShow dans un Framework bien mieux adapté à la création d’applications de modification vidéo.
