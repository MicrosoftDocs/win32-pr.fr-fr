# <a name="using-the-dmotest-application"></a>Utilisation de l’application DMOTest

> [!Note]  
> L’application DMOTest n’est plus disponible dans le kit de développement logiciel (SDK).

 

L’utilitaire DMOTest peut vous aider à vérifier qu’un DMO répond aux spécifications d’DMO. Si vous développez un module DMO, vous devez inclure l’application de test DMO dans le cadre de vos tests.

DMOTest est installé dans le \\ répertoire bin DXUtils, sous le répertoire racine du kit de développement logiciel (SDK). Pour utiliser DMOTest, vous devez d’abord générer des données de test pour DMO. Le kit de développement logiciel (SDK) comprend un filtre DirectShow à cet effet. Vous trouverez des détails sur le filtre et l’application DMOTest dans l’aide de l’application DMOTest. Dans l’application, choisissez **contenu** dans le menu **aide** . L’aide contient des instructions pas à pas pour la génération de données de test et l’exécution de tests.
