# <a name="using-the-dmotest-application"></a>Utilisation de l’application DMOTest

> [!Note]  
> L’application DMOTest n’est plus disponible dans le kit de développement logiciel (SDK).

 

l’utilitaire DMOTest peut vous aider à vérifier qu’un DMO répond aux spécifications de DMO. si vous développez un DMO, vous devez inclure l’application de test DMO dans le cadre de vos tests.

DMOTest est installé dans le \\ répertoire bin DXUtils, sous le répertoire racine du kit de développement logiciel (SDK). Pour utiliser DMOTest, vous devez d’abord générer des données de test pour le DMO. le kit de développement logiciel (SDK) comprend un filtre de DirectShow à cet effet. Vous trouverez des détails sur le filtre et l’application DMOTest dans l’aide de l’application DMOTest. Dans l’application, choisissez **contenu** dans le menu **aide** . L’aide contient des instructions pas à pas pour la génération de données de test et l’exécution de tests.
