# <a name="coutputqueuem_bflushing-member"></a>COutputQueue :: m \_ bFlushing, membre

Indicateur qui spécifie si l’objet effectue une opération de vidage. Si la **valeur est true**, l’objet a démarré le vidage.

## <a name="syntax"></a>Syntaxe


```C++
BOOL m_bFlushing;
```
