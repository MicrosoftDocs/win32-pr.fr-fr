# <a name="cimagesamplem_binit-member"></a>CImageSample :: m \_ bInit, membre

La `m_bInit` variable membre indique si l’objet **CImageSample** a été initialisé. Si c’est le cas, valeur si **true**. Sinon, la **valeur est false**.

## <a name="syntax"></a>Syntaxe


```C++
BOOL m_bInit;
```
