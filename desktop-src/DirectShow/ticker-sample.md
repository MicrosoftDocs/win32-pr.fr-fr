# <a name="ticker-sample"></a>Exemple de ticker

## <a name="description"></a>Description

Cet exemple utilise le convertisseur de mixage vidéo pour fusionner la vidéo et le texte. Elle utilise l’interface [**IVMRMixerBitmap9**](/previous-versions/windows/desktop/api/Vmr9/nn-vmr9-ivmrmixerbitmap9) pour fusionner du texte sur la partie inférieure de la fenêtre vidéo.

L’application crée une image bitmap avec une chaîne de texte par défaut et la fait défiler dans la partie inférieure de l’écran. Pour modifier le texte, choisissez **définir la chaîne de texte...** dans le menu du **téléscripteur** . Pour définir la police, choisissez **définir la police...** dans le menu du **téléscripteur** . Si vous sélectionnez **utiliser une image statique** dans le menu du **ticker** , l’application passe d’une chaîne de texte dynamique à une image bitmap statique.

## <a name="downloading-the-sample"></a>Téléchargement de l’exemple

Pour télécharger les exemples du kit de développement logiciel (SDK) DirectShow, installez la dernière version du [SDK Windows](https://msdn.microsoft.com/windowsvista/bb980924.aspx).
