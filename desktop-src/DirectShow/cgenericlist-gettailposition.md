# <a name="cgenericlistgettailposition-method"></a>Méthode CGenericList. GetTailPosition

La `GetTailPosition` méthode récupère la position du dernier élément de la liste.

## <a name="syntax"></a>Syntaxe


```C++
POSITION GetTailPosition();
```



## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.

## <a name="return-value"></a>Valeur retournée

Retourne une valeur de POSITION, ou **null** si la liste est vide.
