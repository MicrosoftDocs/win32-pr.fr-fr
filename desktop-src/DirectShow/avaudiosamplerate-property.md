# <a name="avaudiosamplerate-property"></a>Propriété AVAudioSampleRate

Obtient le taux d’échantillonnage du flux de bits audio, en échantillons par seconde.

Cette propriété est en lecture seule.

## <a name="data-type"></a>Type de données

**UInt32** (**VT \_ UI4**)

## <a name="property-guid"></a>Guid de propriété

**CODECAPI \_ AVAudioSampleRate**

## <a name="property-value"></a>Valeur de la propriété

L’ensemble des valeurs possibles dépend de l’implémentation du décodeur et du flux de bits audio.
