# <a name="directshow-objects"></a>Objets DirectShow

Cette rubrique contient des entrées de référence pour les objets COM de base inclus avec DirectShow, à l’exception des filtres. Pour obtenir la liste des filtres inclus avec DirectShow, consultez la page [filtres DirectShow](directshow-filters.md).



| Object                                                   | Description                                                      |
|----------------------------------------------------------|------------------------------------------------------------------|
| [Générateur de graphiques de capture](capture-graph-builder.md)       | Crée des graphiques de capture vidéo.                                     |
| [Générateur de graphiques DVD](dvd-graph-builder.md)               | Crée des graphiques de lecture DVD.                                      |
| [Gestionnaire de graphique de filtre](filter-graph-manager.md)         | Génère et contrôle des graphiques de filtre.                               |
| [Mappeur de filtre](filter-mapper.md)                       | Recherche les filtres inscrits dans le registre.                    |
| [Conteneur des propriétés du média](media-property-bag.md)             | Définit ou récupère des blocs INFO et DISP dans des fichiers AVI.             |
| [Allocateur de mémoire](memory-allocator.md)                 | Alloue des mémoires tampons pour les exemples de supports.                             |
| [Horloge de référence système](system-reference-clock.md)     | Implémente une horloge de référence.                                    |
| [Énumérateur de périphérique système](system-device-enumerator.md) | Énumère les filtres et les périphériques matériels installés sur le système. |
