# <a name="username-attribute"></a>Attribut de nom d’utilisateur

> [!Note]  
> \[Déconseillé. Cette API peut être supprimée des futures versions de Windows.\]

 

L' `userid` attribut spécifie un nom défini par l’application pour un objet.

## <a name="possible-values"></a>Valeurs possibles

Peut être n’importe quelle valeur.

## <a name="applies-to"></a>S'applique à

[**clip**](clip-element.md), [**composite**](composite-element.md), [**Effect**](effect-element.md), [**Group**](group-element.md), [**Timeline**](timeline-element.md), [**transition**](transition-element.md)
