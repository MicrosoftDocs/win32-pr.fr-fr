# <a name="tv-audio-filter"></a>Filtre audio TV

Le filtre audio TV fournit le contrôle du décodage audio de télévision, de la sélection stéréo ou monoaural et de la sélection du programme audio secondaire (SAP).



| Étiquette | Valeur |
|------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------|
| Interfaces de filtre                        | [**IAMTVAudio**](/windows/desktop/api/Strmif/nn-strmif-iamtvaudio), **IPersistPropertyBag**, **ISpecifyPropertyPages**, [**IKsPropertySet**](ikspropertyset.md) |
| Types de média de broche d’entrée                    | AnalogAudio de MEDIATYPE \_                                                                                                         |
| Interfaces pin d’entrée                     | **IKsPropertySet**                                                                                                             |
| Types de média de broche de sortie                   | AnalogAudio de MEDIATYPE \_                                                                                                         |
| Interfaces de broche de sortie                    | **IKsPropertySet**                                                                                                             |
| CLSID du filtre                             | CLSID \_ TVAudioFilter                                                                                                           |
| CLSID de page de propriétés                      | AM \_ KSCATEGORY \_ TVAUDIO                                                                                                        |
| [Mérite](merit.md)                       | MÉRITE \_ n' \_ \_ utilise pas                                                                                                            |
