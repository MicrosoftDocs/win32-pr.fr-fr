# <a name="avencddpreferredstereodownmixmode-property"></a>Propriété AVEncDDPreferredStereoDownMixMode

Spécifie le mode de downmix stéréo par défaut pour un flux audio Dolby Digital. Cette propriété s’applique aux encodeurs audio Dolby Digital.

Cette propriété est en lecture/écriture.

## <a name="data-type"></a>Type de données

**UInt32** (**VT \_ UI4**)

## <a name="property-guid"></a>Guid de propriété

**CODECAPI \_ AVEncDDPreferredStereoDownMixMode**

## <a name="property-value"></a>Valeur de la propriété

La valeur de cette propriété est un membre de l’énumération [**eAVEncDDPreferredStereoDownMixMode**](/windows/desktop/api/codecapi/ne-codecapi-eavencddpreferredstereodownmixmode) .

## <a name="remarks"></a>Remarques

Cette propriété est en lecture/écriture.
