# <a name="cbasewindowm_brealizing-member"></a>CBaseWindow :: m \_ bRealizing, membre

Indicateur qui spécifie si une nouvelle palette est réalisée. Dans les versions Debug, la valeur est **true** pendant la modification de la palette et **false** dans le cas contraire. Cette variable n’est pas utilisée dans les versions commerciales.

## <a name="syntax"></a>Syntaxe


```C++
BYTE m_bRealizing;
```
