# <a name="miscellaneous-helper-functions"></a>Fonctions d’assistance diverses

Les éléments suivants sont des fonctions d’assistance diverses.



| Fonction                                                   | Description                                                                                                   |
|------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [**EqualPins**](equalpins.md)                             | Vérifie si deux broches se trouvent sur le même objet.                                                                    |
| [**Int64x32Div32**](int64x32div32.md)                     | Implémente la formule `((a*b)+rnd)/c` où `a` est une valeur 64 bits et `b` , `c` et `rnd` sont des valeurs 32 bits. |
| [**IsEqualObject**](isequalobject.md)                     | Vérifie si deux interfaces se trouvent sur le même objet.                                                              |
| [**llMulDiv**](llmuldiv.md)                               | Implémente la formule `((a*b)+rnd)/c` dans laquelle chaque terme est une valeur 64 bits.                                     |
| [**WaitDispatchingMessages**](waitdispatchingmessages.md) | Attend qu’un objet soit signalé tout en distribuant les messages de fenêtre.                                        |
