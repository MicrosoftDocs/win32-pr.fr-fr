# <a name="avencvideoinputcolortransferfunction-property"></a>Propriété AVEncVideoInputColorTransferFunction

Spécifie la fonction de conversion de RGB en R’G’B’pour la vidéo d’entrée.

Cette propriété est en lecture/écriture.

## <a name="data-type"></a>Type de données

**UInt32** (**VT \_ UI4**)

## <a name="property-guid"></a>Guid de propriété

**CODECAPI \_ AVEncVideoInputColorTransferFunction**

## <a name="property-value"></a>Valeur de la propriété

La valeur de cette propriété est un membre de l’énumération [**eAVEncVideoColorTransferFunction**](/windows/desktop/api/codecapi/ne-codecapi-eavencvideocolortransferfunction) .
