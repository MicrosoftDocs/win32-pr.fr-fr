# <a name="oafilterstate"></a>OAFilterState

Le `OAFilterState` type de données définit l’état du graphique de filtre. La valeur est un membre de l’énumération de l' [**\_ État du filtre**](/windows/win32/api/strmif/ne-strmif-filter_state) . ce type de données est défini pour les clients OLE Automation, tels que Microsoft Visual Basic 6,0.


```C++
typedef long OAFilterState;
```
