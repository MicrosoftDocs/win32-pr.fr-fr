# <a name="getdvdtextstring-method"></a>Méthode GetDVDTextString

> [!Note]  
> Ce composant peut être utilisé dans les systèmes d’exploitation Microsoft Windows 2000, Windows XP et Windows Server 2003. Il sera peut-être modifié ou indisponible dans les versions ultérieures.

 

La `GetDVDTextString` méthode récupère la chaîne de texte spécifiée.

``` syntax
[ sText = ] MSWebDVD.GetDVDTextString(iLangIndex, iStringIndex)
```

## <a name="parameters"></a>Paramètres

<dl> <dt>

<span id="iLangIndex"></span><span id="ilangindex"></span><span id="ILANGINDEX"></span>*iLangIndex*
</dt> <dd>

Spécifie l’index de la langue sous la forme d’un entier.

</dd> <dt>

<span id="iStringIndex"></span><span id="istringindex"></span><span id="ISTRINGINDEX"></span>*iStringIndex*
</dt> <dd>

Spécifie le numéro d’index de la chaîne de texte sous la forme d’un entier.

</dd> </dl>

## <a name="return-value"></a>Valeur renvoyée

Retourne la chaîne spécifiée par *iStringIndex*.
