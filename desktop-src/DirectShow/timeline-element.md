# <a name="timeline-element"></a>Élément timeline

> [!Note]  
> \[Action déconseillée. Cette API peut être supprimée dans les versions futures de Windows.\]

 

L' `timeline` élément définit la chronologie. Cet élément est le nœud racine dans le fichier XML.

## <a name="attributes"></a>Attributs

[**defaultfx**](defaultfx-attribute.md), [**defaulttrans**](defaulttrans-attribute.md), [**enablefx**](enablefx-attribute.md), [**enabletrans**](enabletrans-attribute.md), [**cadence**](framerate-attribute.md)

## <a name="parentchild-information"></a>Informations parent/enfant



| Étiquette | Valeur |
|----------|-------------------------------------------------------|
| Parent   | Aucun. Cet élément doit être le nœud racine dans le fichier. |
| Children | [**Communauté**](group-element.md)                        |



 

## <a name="examples"></a>Exemples


```XML
<timeline> </timeline>
```
