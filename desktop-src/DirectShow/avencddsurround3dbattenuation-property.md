# <a name="avencddsurround3dbattenuation-property"></a>Propriété AVEncDDSurround3dBAttenuation

Spécifie si les canaux surround d’un flux audio Dolby Digital sont atténués 3 décibels (dB) avant l’encodage. Cette propriété s’applique aux encodeurs audio Dolby Digital.

Cette propriété est en lecture/écriture.

## <a name="data-type"></a>Type de données

**Variante \_ BOOL** (**VT \_ bool**)

## <a name="property-guid"></a>Guid de propriété

**CODECAPI \_ AVEncDDSurround3dBAttenuation**
