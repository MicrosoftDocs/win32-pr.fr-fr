# <a name="ec_skip_frames"></a>\_frames d’omission EC \_

Envoyé par le navigateur DVD lorsqu’il doit ignorer des frames.

## <a name="parameters"></a>Paramètres

<dl> <dt>

<span id="lParam1"></span><span id="lparam1"></span><span id="LPARAM1"></span>*lParam1*
</dt> <dd>

Nombre de frames à ignorer.

</dd> <dt>

<span id="lParam2"></span><span id="lparam2"></span><span id="LPARAM2"></span>*lParam2*
</dt> <dd>

Réservé.

</dd> </dl>

## <a name="default-action"></a>Action par défaut

Cet événement n’est pas envoyé à l’application. Les applications ne peuvent pas remplacer l’action par défaut de cet événement.
