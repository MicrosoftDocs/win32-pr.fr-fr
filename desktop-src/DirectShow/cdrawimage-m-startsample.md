# <a name="cdrawimagem_startsample-member"></a>CDrawImage :: m \_ StartSample, membre

La variable de membre **m \_ StartSample** spécifie l’heure de début de l’exemple le plus récent.

## <a name="syntax"></a>Syntaxe


```C++
CRefTime m_StartSample;
```



## <a name="remarks"></a>Notes

La valeur est valide uniquement à l’intérieur de la méthode [**CDrawImage ::D isplaysampletimes**](cdrawimage-displaysampletimes.md) .
