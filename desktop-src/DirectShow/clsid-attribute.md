# <a name="clsid-attribute"></a>CLSID (attribut)

> [!Note]  
> \[Déconseillé. Cette API peut être supprimée des futures versions de Windows.\]

 

L' `clsid` attribut spécifie un identificateur de classe (CLSID) pour un objet.

## <a name="possible-values"></a>Valeurs possibles

Doit être une chaîne au format {00000000-0000-0000-0000-000000000000} .

## <a name="applies-to"></a>S'applique à

[**à**](at-element.md), [**effet**](effect-element.md), [**transition**](transition-element.md)
