# <a name="about-video-capture-in-directshow"></a>À propos de la capture vidéo dans DirectShow

Le terme *capture vidéo*, tel qu’il est utilisé dans cette documentation, décrit toute application sur laquelle la vidéo est reçue d’un périphérique matériel. Les périphériques de capture vidéo incluent non seulement les caméras, mais également les cartes tuner TV, les magnétoscopes vidéo, etc. La vidéo capturée peut être enregistrée sur le disque ou prévisualisée en direct.

Cette section décrit certains des concepts que vous devez comprendre pour pouvoir utiliser DirectShow pour la capture vidéo :

-   [À propos du générateur de graphiques de capture](about-the-capture-graph-builder.md)
-   [À propos des périphériques de capture vidéo](about-video-capture-devices.md)
-   [Filtres de capture vidéo DirectShow](directshow-video-capture-filters.md)
