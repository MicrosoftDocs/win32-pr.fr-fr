# <a name="avencvideoforcesourcescantype-property"></a>Propriété AVEncVideoForceSourceScanType

Spécifie si les frames d’entrée sont progressifs ou entrelacés.

Cette propriété est en lecture/écriture.

## <a name="data-type"></a>Type de données

**UInt32** (**VT \_ UI4**)

## <a name="property-guid"></a>Guid de propriété

**CODECAPI \_ AVEncVideoForceSourceScanType**

## <a name="property-value"></a>Valeur de la propriété

La valeur de cette propriété est un membre de l’énumération [**eAVEncVideoSourceScanType**](/windows/desktop/api/codecapi/ne-codecapi-eavencvideosourcescantype) .
