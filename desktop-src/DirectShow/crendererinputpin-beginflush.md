# <a name="crendererinputpinbeginflush-method"></a>Méthode CRendererInputPin. BeginFlush

La `BeginFlush` méthode commence une opération de vidage. Cette méthode remplace la méthode [**CBaseInputPin :: BeginFlush**](cbaseinputpin-beginflush.md) .

## <a name="syntax"></a>Syntaxe


```C++
HRESULT BeginFlush();
```



## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.

## <a name="return-value"></a>Valeur retournée

Retourne une valeur **HRESULT** .
