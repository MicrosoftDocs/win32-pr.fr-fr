| Énumération                                                         | Description                                                                  |
|---------------------------------------------------------------------|------------------------------------------------------------------------------|
| [**\_indicateurs de \_ recherche DEXTERF Track \_**](dexterf-track-search-flags.md) | Spécifie les conditions limites sur une recherche d’un objet dans la chronologie. |
