# <a name="getclipvideorect-method"></a>Méthode GetClipVideoRect

> [!Note]  
> ce composant peut être utilisé dans les systèmes d’exploitation Microsoft Windows 2000, Windows XP et Windows Server 2003. Il sera peut-être modifié ou indisponible dans les versions ultérieures.

 

La `GetClipVideoRect` méthode récupère le rectangle de découpage actuellement défini pour l’affichage vidéo.

``` syntax
[ oClipRect = ] MSWebDVD.GetClipVideoRect()
```

## <a name="return-value"></a>Valeur renvoyée

Retourne un objet [DVDRect](dvdrect-object.md) qui définit le rectangle.
