# <a name="samplingrate-attribute"></a>Attribut samplingrate

> [!Note]  
> \[Action déconseillée. Cette API peut être supprimée dans les versions futures de Windows.\]

 

L' `samplingrate` attribut spécifie la fréquence d’échantillonnage du son de sortie, en Hz.

## <a name="possible-values"></a>Valeurs possibles

La valeur doit être 8000, 11025, 22050, 32000, 44100 ou 48000. La valeur par défaut est 44100.

## <a name="applies-to"></a>S'applique à

[**Communauté**](group-element.md)

## <a name="remarks"></a>Notes

Définissez cet attribut uniquement si l’attribut de **type** est `audio` .
