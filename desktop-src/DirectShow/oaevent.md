# <a name="oaevent"></a>OAEVENT

Le type de données **OAEVENT** définit un descripteur d’événement qui peut être utilisé par les clients OLE Automation, tels que Microsoft Visual Basic 6,0.


```C++
typedef LONG_PTR OAEVENT;
```
