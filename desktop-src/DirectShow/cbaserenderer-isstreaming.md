# <a name="cbaserendererisstreaming-method"></a>Méthode CBaseRenderer. IsStreaming

La `IsStreaming` méthode interroge si le filtre diffuse des données.

## <a name="syntax"></a>Syntaxe


```C++
BOOL IsStreaming();
```



## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.

## <a name="return-value"></a>Valeur retournée

Retourne l’indicateur [**CBaseRenderer :: m \_ bStreaming**](cbaserenderer-m-bstreaming.md) .
