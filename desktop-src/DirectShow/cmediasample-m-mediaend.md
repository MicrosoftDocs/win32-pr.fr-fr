# <a name="cmediasamplem_mediaend-member"></a>CMediaSample :: m \_ MediaEnd, membre

Heure d’arrêt du support. Cette valeur est calculée sous la forme d’un décalage par rapport à l’heure de début du support, spécifié par la variable membre [**CMediaSample :: m \_ MediaStart**](cmediasample-m-mediastart.md) .

## <a name="syntax"></a>Syntaxe


```C++
LONG m_MediaEnd;
```
