# <a name="avencaudiomapdestchanneln-property"></a>Propriété AVEncAudioMapDestChannelN

Spécifie le canal audio mappé au canal *N* dans le flux audio encodé.

Cette propriété est en lecture/écriture.

## <a name="data-type"></a>Type de données

**UInt32** (**VT \_ UI4**)

## <a name="property-guid"></a>Guid de propriété

**Consultez la section Notes.**

## <a name="remarks"></a>Remarques

Cette rubrique de référence décrit un ensemble de propriétés associées. Chaque propriété mappe un canal d’entrée à un canal de sortie spécifié. Les propriétés suivantes sont définies.



| Guid de propriété                            | Description             |
|------------------------------------------|-------------------------|
| **CODECAPI \_ AVEncAudioMapDestChannel0**  | Canal de destination 0.  |
| **CODECAPI \_ AVEncAudioMapDestChannel1**  | Canal de destination 1.  |
| **CODECAPI \_ AVEncAudioMapDestChannel2**  | Canal de destination 2.  |
| **CODECAPI \_ AVEncAudioMapDestChannel3**  | Canal de destination 3.  |
| **CODECAPI \_ AVEncAudioMapDestChannel4**  | Canal de destination 4.  |
| **CODECAPI \_ AVEncAudioMapDestChannel5**  | Canal de destination 5.  |
