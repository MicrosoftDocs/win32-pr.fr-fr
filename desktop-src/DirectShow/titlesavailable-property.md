# <a name="titlesavailable-property"></a>Propriété TitlesAvailable

> [!Note]  
> ce composant peut être utilisé dans les systèmes d’exploitation Microsoft Windows 2000, Windows XP et Windows Server 2003. Il sera peut-être modifié ou indisponible dans les versions ultérieures.

 

La `TitlesAvailable` propriété récupère le nombre de titres disponibles sur le disque.

``` syntax
[ iTitles = ] MSWebDVD.TitlesAvailable
```

## <a name="return-value"></a>Valeur renvoyée

Retourne le nombre de titres disponibles sous la forme d’une valeur entière comprise entre 1 et 99.

## <a name="remarks"></a>Remarques

Cette propriété est en lecture seule sans valeur par défaut.
