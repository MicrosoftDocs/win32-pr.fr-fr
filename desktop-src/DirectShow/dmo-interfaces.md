# <a name="dmo-interfaces"></a>Interfaces DMO

Cette section contient des entrées de référence pour toutes les interfaces COM Microsoft DirectX Media Object (DMO) et leurs méthodes.



| Interface                                                            | Description                                                              |
|----------------------------------------------------------------------|--------------------------------------------------------------------------|
| [**IDMOQualityControl**](/previous-versions/windows/desktop/api/Mediaobj/nn-mediaobj-idmoqualitycontrol)                     | Prend en charge le contrôle de qualité sur un DMO.                                       |
| [**IDMOVideoOutputOptimizations**](/previous-versions/windows/desktop/api/Mediaobj/nn-mediaobj-idmovideooutputoptimizations) | Prend en charge les optimisations vidéo sur un DMO.                                   |
| [**IEnumDMO**](/previous-versions/windows/desktop/api/Mediaobj/nn-mediaobj-ienumdmo)                                         | Fournit des méthodes pour énumérer les DMOs.                                   |
| [**IMediaBuffer**](/previous-versions/windows/desktop/api/Mediaobj/nn-mediaobj-imediabuffer)                                 | Fournit des méthodes pour manipuler une mémoire tampon de données.                         |
| [**IMediaObject**](/previous-versions/windows/desktop/api/Mediaobj/nn-mediaobj-imediaobject)                                 | Fournit des méthodes pour manipuler un DMO.                                 |
| [**IMediaObjectInPlace**](/previous-versions/windows/desktop/api/mediaobj/nn-mediaobj-imediaobjectinplace)                   | Fournit des méthodes pour le traitement des données sur place.                           |
| [**IMediaParamInfo**](/previous-versions/windows/desktop/api/Medparam/nn-medparam-imediaparaminfo)                           | Récupère des informations sur les paramètres de média pris en charge par un objet |
| [**IMediaParams**](/previous-versions/windows/desktop/api/Medparam/nn-medparam-imediaparams)                                 | Définit et récupère des paramètres de média sur un objet.                        |
