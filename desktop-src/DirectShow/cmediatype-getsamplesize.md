# <a name="cmediatypegetsamplesize-method"></a>Méthode CMediaType. GetSampleSize

La `GetSampleSize` méthode récupère la taille de l’échantillon.

## <a name="syntax"></a>Syntaxe


```C++
ULONG GetSampleSize() const;
```



## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.

## <a name="return-value"></a>Valeur de retour

Si la taille de l’échantillon est fixe, retourne la taille de l’échantillon en octets. Sinon, retourne zéro.
