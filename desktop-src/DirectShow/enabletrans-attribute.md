# <a name="enabletrans-attribute"></a>Attribut enabletrans

> [!Note]  
> \[Action déconseillée. Cette API peut être supprimée des futures versions de Windows.\]

 

L' `enabletrans` attribut spécifie si les transitions sont activées. Si la valeur est **true**, le moteur de rendu restitue entièrement les transitions. Dans le cas contraire, il les affiche sous forme de coupes. La valeur par défaut est **true**.

## <a name="possible-values"></a>Valeurs possibles

Les valeurs suivantes sont définies sur TRUE : y, Y, t, T, 1. Les valeurs suivantes sont définies avec la valeur FALSe : n, N, f, F, 0 (zéro).
