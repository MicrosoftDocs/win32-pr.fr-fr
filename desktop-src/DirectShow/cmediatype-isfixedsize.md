# <a name="cmediatypeisfixedsize-method"></a>CMediaType. IsFixedSize, méthode

La `IsFixedSize` méthode détermine si les échantillons ont une taille fixe ou une taille variable.

## <a name="syntax"></a>Syntaxe


```C++
BOOL IsFixedSize() const;
```



## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.

## <a name="return-value"></a>Valeur retournée

Retourne la valeur du membre **bFixedSizeSamples** .
