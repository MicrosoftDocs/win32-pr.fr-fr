# <a name="filter-mapper"></a>Mappeur de filtre

Le mappeur de filtre recherche les filtres inscrits dans le registre. le gestionnaire de Graph de filtre utilise ce composant pour générer des graphiques de filtres. Les applications peuvent l’utiliser pour rechercher des filtres qui correspondent aux critères de recherche spécifiés. Créez cet objet en appelant **CoCreateInstance**.
