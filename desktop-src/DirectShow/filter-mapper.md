# <a name="filter-mapper"></a>Mappeur de filtre

Le mappeur de filtre recherche les filtres inscrits dans le registre. Le gestionnaire de graphique de filtre utilise ce composant pour créer des graphiques de filtres. Les applications peuvent l’utiliser pour rechercher des filtres qui correspondent aux critères de recherche spécifiés. Créez cet objet en appelant **CoCreateInstance**.
