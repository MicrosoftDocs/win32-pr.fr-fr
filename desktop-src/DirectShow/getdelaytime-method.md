# <a name="getdelaytime-method"></a>Méthode GetDelayTime

> [!Note]  
> Ce composant peut être utilisé dans les systèmes d’exploitation Microsoft Windows 2000, Windows XP et Windows Server 2003. Il sera peut-être modifié ou indisponible dans les versions ultérieures.

 

La `GetDelayTime` méthode récupère le délai d’attente pour l’info-bulle associée à l’objet **mswebdvd** .

``` syntax
[ iDelay = ] MSWebDVD.GetDelayTime()
```

## <a name="return-value"></a>Valeur renvoyée

Retourne une valeur entière indiquant le délai d’info-bulle, en millisecondes.
