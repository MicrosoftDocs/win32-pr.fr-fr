# <a name="shortcuts-to-c-dvd-reference-pages"></a>Raccourcis vers les pages de référence des DVD C++

Les interfaces suivantes sont généralement utilisées lors de l’écriture d’une application DVD en C++.



| Interface                                    | Objectif                                                                                          |
|----------------------------------------------|--------------------------------------------------------------------------------------------------|
| [**IAMLine21Decoder**](/previous-versions/windows/desktop/api/il21dec/nn-il21dec-iamline21decoder) | Contrôle l’affichage des sous-titres.                                                              |
| [**IDvdCmd**](/windows/desktop/api/Strmif/nn-strmif-idvdcmd)                   | Utilisé dans les techniques de synchronisation de commande avancées pour contrôler un objet Command.                 |
| [**IDvdControl2**](/windows/desktop/api/Strmif/nn-strmif-idvdcontrol2)         | Émet des commandes pour le filtre de [navigateur DVD](dvd-navigator-filter.md) (méthodes « Set »).     |
| [**IDvdGraphBuilder**](/windows/desktop/api/Strmif/nn-strmif-idvdgraphbuilder) | Crée le graphique de filtre de DVD.                                                                    |
| [**IDvdInfo2**](/windows/desktop/api/Strmif/nn-strmif-idvdinfo2)               | Interroge le navigateur DVD pour obtenir des informations d’État et des informations sur le disque (méthodes « obtenir »). |
| [**IDvdState**](/windows/desktop/api/Strmif/nn-strmif-idvdstate)               | Enregistre les informations, telles que les signets, sur le disque.                                                   |
| [**IVideoFrameStep**](/windows/desktop/api/Strmif/nn-strmif-ivideoframestep)   | Contrôle le « pas à pas de frame » si le décodeur le prend en charge.                                            |
