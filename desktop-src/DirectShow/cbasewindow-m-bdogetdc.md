# <a name="cbasewindowm_bdogetdc-member"></a>CBaseWindow :: m \_ bDoGetDC, membre

Indicateur qui spécifie s’il faut récupérer le contexte de périphérique (DC). Si la **valeur est true**, l’objet récupère un handle vers le contrôleur de périphérique de la fenêtre et crée un contrôleur de périphérique de mémoire compatible.

## <a name="syntax"></a>Syntaxe


```C++
BOOL m_bDoGetDC;
```
