# <a name="directshow-structures"></a>structures de DirectShow

> [!Note]  
> \[Déconseillé. Cette API peut être supprimée des futures versions de Windows.\]

 

DirectShow Les services d’édition définissent les structures suivantes.



| Structure                             | Description                                          |
|---------------------------------------|------------------------------------------------------|
| [**\_paramètre Dexter**](dexter-param.md) | Spécifie une propriété sur une transition ou un effet.      |
| [**valeur de DEXTERity \_**](dexter-value.md) | Spécifie la valeur d’une propriété.                   |
| [**SCompFmt0**](scompfmt0.md)        | Contient des informations de format pour la recompression intelligente. |
