# <a name="ixml2dexdelete-method"></a>IXml2Dex ::D méthode supprimable

> [!Note]  
> \[Action déconseillée. Cette API peut être supprimée dans les versions futures de Windows.\]

 

Non implémenté.

## <a name="syntax"></a>Syntaxe


```C++
HRESULT Delete(
   IUnknown *pTimeline,
   double   dStart,
   double   dEnd
);
```
