# <a name="cmediatypeformatlength-method"></a>Méthode CMediaType. FormatLength

La `FormatLength` méthode récupère la longueur du bloc de format.

## <a name="syntax"></a>Syntaxe


```C++
ULONG FormatLength() const;
```



## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.

## <a name="return-value"></a>Valeur retournée

Retourne le membre **cbFormat** .
