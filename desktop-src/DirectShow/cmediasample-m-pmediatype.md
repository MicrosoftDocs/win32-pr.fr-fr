# <a name="cmediasamplem_pmediatype-member"></a>CMediaSample :: m \_ pMediaType, membre

Pointeur vers le type de média, si le type a été modifié par rapport à l’exemple précédent dans le flux de données. Dans le cas contraire, cette variable membre a la **valeur null**.

## <a name="syntax"></a>Syntaxe


```C++
AM_MEDIA_TYPE *m_pMediaType;
```
