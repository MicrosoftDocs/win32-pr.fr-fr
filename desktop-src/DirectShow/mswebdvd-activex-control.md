# <a name="mswebdvd-activex-control"></a>Contrôle ActiveX MSWebDVD

> [!Note]  
> Ce composant peut être utilisé dans les systèmes d’exploitation Microsoft Windows 2000, Windows XP et Windows Server 2003. Il sera peut-être modifié ou indisponible dans les versions ultérieures.

 

> [!Note]  
> Ce composant est déconseillé. Pour plus d’informations sur la lecture et la navigation sur DVD dans DirectShow, consultez [applications DVD](dvd-applications.md).

 

Le contrôle de® ActiveX Microsoft® MSWebDVD peut être utilisé pour créer une application DVD basée sur un script. Il expose les objets et les événements suivants à une application.

-   [MSWebDVD](mswebdvd-object.md)
-   [MSDVDAdm](msdvdadm-object.md)
-   [DVDRect](dvdrect-object.md)
-   [Événements MSWebDVD](mswebdvd-events.md)
