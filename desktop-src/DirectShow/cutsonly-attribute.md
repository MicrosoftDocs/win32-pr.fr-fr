# <a name="cutsonly-attribute"></a>Attribut cutsonly

> [!Note]  
> \[Action déconseillée. Cette API peut être supprimée des futures versions de Windows.\]

 

L' `cutsonly` attribut spécifie si une transition est rendue sous la forme d’une coupe. Si la valeur est **true**, la transition est une coupe instantanée. Dans le cas contraire, la transition est rendue normalement. La valeur par défaut est **FALSE**.

## <a name="possible-values"></a>Valeurs possibles

Les valeurs suivantes sont définies sur TRUE : y, Y, t, T, 1. Les valeurs suivantes sont définies avec la valeur FALSe : n, N, f, F, 0 (zéro).
