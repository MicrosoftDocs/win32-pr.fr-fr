# <a name="enablefx-attribute"></a>Attribut enablefx

> [!Note]  
> \[Action déconseillée. Cette API peut être supprimée dans les versions futures de Windows.\]

 

L' `enablefx` attribut spécifie si les effets sont activés. Si la valeur est **true**, le moteur de rendu restitue les effets. Sinon, elle ne l'a pas. La valeur par défaut est **true**.

## <a name="possible-values"></a>Valeurs possibles

Les valeurs suivantes sont définies sur TRUE : y, Y, t, T, 1. Les valeurs suivantes sont définies avec la valeur FALSe : n, N, f, F, 0 (zéro).
