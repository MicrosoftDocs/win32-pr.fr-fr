# <a name="cdynamicoutputpinstopusingoutputpin-method"></a>Méthode CDynamicOutputPin. StopUsingOutputPin

La `StopUsingOutputPin` méthode libère l’accès au code PIN après une opération de diffusion en continu.

## <a name="syntax"></a>Syntaxe


```C++
virtual void StopUsingOutputPin();
```



## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.

## <a name="return-value"></a>Valeur retournée

Cette méthode ne retourne pas de valeur.
