# <a name="stream_time"></a>temps de flux \_

> [!Note]  
> Cette API est déconseillée. Les nouvelles applications ne doivent pas l’utiliser.

 

Le \_ type de données temps de flux est utilisé pour exprimer les temps de référence dans le streaming multimédia Microsoft DirectShow. Les unités sont des nanosecondes de 100.


```C++
typedef LONGLONG STREAM_TIME;
```



## <a name="remarks"></a>Notes

Ce type de données est équivalent au type de données de [**\_ temps de référence**](reference-time.md) DirectShow.
