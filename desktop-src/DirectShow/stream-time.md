# <a name="stream_time"></a>temps de flux \_

> [!Note]  
> Cette API est déconseillée. Les nouvelles applications ne doivent pas l’utiliser.

 

le \_ type de données temps de flux est utilisé pour exprimer les temps de référence dans Microsoft DirectShow la diffusion multimédia en continu. Les unités sont des nanosecondes de 100.


```C++
typedef LONGLONG STREAM_TIME;
```



## <a name="remarks"></a>Remarques

ce type de données est équivalent à l’DirectShow type de données de [**\_ temps de référence**](reference-time.md) .
