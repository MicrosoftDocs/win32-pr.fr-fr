# <a name="avdeccommonmeanbitrate-property"></a>Propriété AVDecCommonMeanBitRate

Obtient la vitesse de transmission moyenne actuelle du décodeur.

Cette propriété est en lecture seule.

## <a name="data-type"></a>Type de données

**UInt32** (**VT \_ UI4**)

## <a name="property-guid"></a>Guid de propriété

**CODECAPI \_ AVDecCommonMeanBitRate**
