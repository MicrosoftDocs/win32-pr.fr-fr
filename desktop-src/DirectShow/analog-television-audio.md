# <a name="analog-television-audio"></a>Télévision analogique audio

Avec la télévision analogique, le son provient du tuner TV. Différentes cartes tuner doivent être configurées différemment. Certaines cartes utilisent un connecteur externe pour corriger l’audio sur la ligne de la carte son. D’autres utilisent un connecteur interne pour corriger l’audio sur l’entrée de CD de la carte son.

Les rubriques suivantes décrivent comment afficher un aperçu et capturer des données audio TV dans DirectShow :

-   [Aperçu de l’audio TV](previewing-tv-audio.md)
-   [Capture d’audio TV](capturing-tv-audio.md)
