# <a name="cbasepropertypagem_dlg-member"></a>CBasePropertyPage :: m \_ DLG, membre

La `m_Dlg` variable membre contient un handle vers la fenêtre de boîte de dialogue. Cette variable membre est initialisée lorsque la fenêtre de dialogue traite le \_ message WM INITDIALOG.

## <a name="syntax"></a>Syntaxe


```C++
HWND m_Dlg;
```
