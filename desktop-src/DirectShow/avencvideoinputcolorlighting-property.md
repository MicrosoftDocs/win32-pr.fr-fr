# <a name="avencvideoinputcolorlighting-property"></a>Propriété AVEncVideoInputColorLighting

Spécifie les conditions d’éclairage prévues pour l’affichage de la vidéo d’entrée.

Cette propriété est en lecture/écriture.

## <a name="data-type"></a>Type de données

**UInt32** (**VT \_ UI4**)

## <a name="property-guid"></a>Guid de propriété

**CODECAPI \_ AVEncVideoInputColorLighting**

## <a name="property-value"></a>Valeur de la propriété

La valeur de cette propriété est un membre de l’énumération [**eAVEncVideoColorLighting**](/windows/desktop/api/codecapi/ne-codecapi-eavencvideocolorlighting) .
