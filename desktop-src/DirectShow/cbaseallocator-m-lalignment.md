# <a name="cbaseallocatorm_lalignment-member"></a>CBaseAllocator :: m \_ lAlignment, membre

Alignement de chaque mémoire tampon. L’adresse de chaque mémoire tampon doit être un multiple pair de cette valeur. Le préfixe doit être calculé dans l’alignement ; consultez [**CBaseAllocator :: m \_ lPrefix**](cbaseallocator-m-lprefix.md).

## <a name="syntax"></a>Syntaxe


```C++
long m_lAlignment;
```
