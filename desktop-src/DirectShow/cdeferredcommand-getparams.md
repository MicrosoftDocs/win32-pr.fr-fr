# <a name="cdeferredcommandgetparams-method"></a>Méthode CDeferredCommand. GetParams

La `GetParams` méthode récupère la liste d’arguments **DISPPARAMS** à la méthode.

## <a name="syntax"></a>Syntaxe


```C++
DISPPARAMS* GetParams();
```



## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.

## <a name="return-value"></a>Valeur retournée

Retourne une liste d’arguments **DISPPARAMS** .
