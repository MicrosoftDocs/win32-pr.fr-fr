# <a name="property-setter"></a>Accesseur Set de propriété

> [!Note]  
> \[Déconseillé. Cette API peut être supprimée des futures versions de Windows.\]

 

L’objet Setter de propriété définit les propriétés **IDispatch** sur un objet Timeline. Pour créer cet objet, appelez **CoCreateInstance**. L’identificateur de classe est CLSID \_ PropertySetter.

L’objet accesseur Set de propriété expose les interfaces suivantes :

-   [**IAMSetErrorLog**](iamseterrorlog.md)
-   [**IPropertySetter**](ipropertysetter.md)
