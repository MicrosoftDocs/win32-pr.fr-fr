# <a name="cbasevideorenderer-class"></a>CBaseVideoRenderer, classe

![hiérarchie de la classe cbasevideorenderer](images/rbase03.png)

La `CBaseVideoRenderer` classe de base est utilisée pour générer des filtres de convertisseur vidéo.



| Membres de données protégés                                                                 | Description                                                                                                                                                                                                                          |
|----------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| m \_ bDrawLateFrames                                                                     | Indicateur signalant qu’il n’est pas possible de supprimer des frames. Déboguer uniquement. Cela détruit la synchronisation.                                                                                                                                          |
| m \_ bSupplierHandlingQuality                                                            | **True** indique que les messages de contrôle de qualité sont gérés. Cela permet au convertisseur de savoir qu’il doit attendre le plus tard possible pour supprimer les frames lui-même et afficher le frame suivant très tôt après que le fournisseur a supprimé une image. |
| m \_ cFramesDrawn                                                                        | Nombre total de trames qui ont été dessinées depuis le début de la diffusion en continu.                                                                                                                                                                 |
| m \_ cFramesDropped                                                                      | Trames cumulées qui ont été supprimées dans le convertisseur depuis le début de la diffusion en continu. Les frames peuvent également être supprimés en amont sans que le convertisseur ne les reconnaisse.                                                                         |
| m \_ idDecision                                                                          | \_ID de la MSR pour le code de décision de [**ShouldDrawSampleNow**](cbasevideorenderer-shoulddrawsamplenow.md).                                                                                                                              |
| m \_ idDuration                                                                          | \_ID MSR pour la durée d’une image.                                                                                                                                                                                                 |
| m \_ idFrameAccuracy                                                                     | Identificateur du journal des performances pour la durée en millisecondes pendant laquelle le frame était en retard.                                                                                                                                                     |
| m \_ idFrameAvg                                                                          | Identificateur du journal des performances pour la durée d’image moyenne utilisée pour la synchronisation et le contrôle de qualité.                                                                                                                          |
| m \_ idQualityRate                                                                       | \_ID MSR pour le taux de qualité demandé.                                                                                                                                                                                              |
| m \_ idQualityTime                                                                       | \_ID MSR pour le temps de qualité demandé.                                                                                                                                                                                              |
| m \_ idRenderAvg                                                                         | Identificateur du journal des performances pour le temps de rendu moyen enregistré.                                                                                                                                                                   |
| m \_ idSchLateTime                                                                       | \_L’ID de la MSR correspond à la date à laquelle le frame a été planifié.                                                                                                                                                                                   |
| m \_ idSendQuality                                                                       | \_ID MSR pour chronométrer les notifications (inutilisées).                                                                                                                                                                                       |
| m \_ idTimeStamp                                                                         | \_ID MSR pour un horodatage de frame.                                                                                                                                                                                                      |
| m \_ idWait                                                                              | Identificateur du journal des performances pour le temps d’attente enregistré (inutilisé).                                                                                                                                                                      |
| m \_ idWaitReal                                                                          | Identificateur du journal des performances pour le temps d’attente réel.                                                                                                                                                                                   |
| m \_ iSumFrameTime                                                                       | Somme des temps d’intertrame ; nécessaire pour la page de propriétés.                                                                                                                                                                           |
| m \_ iSumSqAcc                                                                           | Somme des carrés des précisions (en millisecondes) nécessaires pour la page de propriétés.                                                                                                                                                 |
| m \_ iSumSqFrameTime                                                                     | Somme des carrés des temps d’intertrame ; nécessaire pour la page de propriétés.                                                                                                                                                                |
| m \_ iTotAcc                                                                             | Somme des précisions (en millisecondes) nécessaires pour la page de propriétés.                                                                                                                                                                |
| m \_ nNormal                                                                             | Nombre de frames consécutifs dessinés à l’heure planifiée. Un nombre négatif indique qu’une image vient d’être supprimée par le convertisseur.                                                                                          |
| m \_ trDuration                                                                          | Durée de la dernière trame (différence entre les heures de début et de fin).                                                                                                                                                             |
| m \_ trEarliness                                                                         | La durée pendant laquelle un frame est autorisé à être lu lorsqu’un frame vient d’être supprimé.                                                                                                                                                        |
| m \_ trFrame                                                                             | Temps enregistré le plus récemment entre les frames. Utilisé dans les mesures statistiques.                                                                                                                                                        |
| m \_ trFrameAvg                                                                          | Durée d’intertrame moyenne en unités de temps de référence.                                                                                                                                                                                     |
| m \_ trLastDraw                                                                          | Heure du frame précédent. Utilisé pour les références de temps d’intertrame.                                                                                                                                                                         |
| m \_ trLate                                                                              | Durée pendant laquelle le frame actuel était en retard. Utilisé dans les mesures statistiques.                                                                                                                                                    |
| m \_ trRenderAvg                                                                         | Temps pris par les trames pour effectuer le transfert de bloc de bits.                                                                                                                                                                       |
| m \_ trRenderLast                                                                        | Heure du dernier transfert de bloc de bits de trame.                                                                                                                                                                                          |
| m \_ trRenderStart                                                                       | Heure à laquelle le transfert de bloc de bits a commencé. Utilisé pour accéder à **m \_ trRenderLast**.                                                                                                                                                                |
| m \_ trThrottle                                                                          | Période à insérer après le rendu de chaque image, généralement utilisée lorsque la qualité audio a été augmentée et que les performances vidéo doivent être réduites pour permettre cela.                                                                             |
| m \_ trWaitAvg                                                                           | Temps d’attente moyen en unités de temps de référence.                                                                                                                                                                                           |
| m \_ tStreamingStart                                                                     | Utilisé pour les statistiques de page de propriétés. Représente l’heure de début du processus de diffusion en continu ou du processus de streaming précédent s’il n’est pas en cours de diffusion en continu.                                                                          |
