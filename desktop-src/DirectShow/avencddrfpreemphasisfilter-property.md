# <a name="avencddrfpreemphasisfilter-property"></a>Propriété AVEncDDRFPreEmphasisFilter

Spécifie le paramètre de protection RF de surmodulation dans un flux audio Dolby Digital. Cette propriété s’applique aux encodeurs audio Dolby Digital.

Cette propriété est en lecture/écriture.

## <a name="data-type"></a>Type de données

**Variante \_ BOOL** (**VT \_ bool**)

## <a name="property-guid"></a>Guid de propriété

**CODECAPI \_ AVEncDDRFPreEmphasisFilter**
