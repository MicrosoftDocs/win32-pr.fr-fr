# <a name="dvd-region-change-support-in-windows"></a>Prise en charge du changement de région des DVD dans Windows

cette section explique le fonctionnement du processus de sélection des régions de DVD dans Windows 2000 et versions ultérieures. La première section décrit les trois sources d’informations sur les régions DVD. La deuxième section décrit comment les fabricants de systèmes PC peuvent définir la région du DVD initiale d’un système. La troisième section explique comment le navigateur DVD détermine la correspondance des régions entre le lecteur, le décodeur et le disque. La dernière section montre comment les développeurs d’applications et de décodeurs peuvent implémenter la prise en charge correcte de la sélection des régions DVD dans leur code.
