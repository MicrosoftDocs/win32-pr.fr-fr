# <a name="cbasecontrolvideoonupdaterectangles-method"></a>Méthode CBaseControlVideo. OnUpdateRectangles

La `OnUpdateRectangles` méthode est appelée lorsque le rectangle source ou de destination change.

## <a name="syntax"></a>Syntaxe


```C++
virtual HRESULT OnUpdateRectangles();
```



## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.

## <a name="return-value"></a>Valeur retournée

Retourne une erreur.
