# <a name="effect"></a>Effet

> [!Note]  
> \[Déconseillé. Cette API peut être supprimée des futures versions de Windows.\]

 

L’objet Effect produit un effet vidéo ou audio. Pour créer cet objet, appelez la méthode [**IAMTimeline :: CreateEmptyNode**](iamtimeline-createemptynode.md) .

L’objet Effect expose les interfaces suivantes :

-   [**IAMTimelineEffect**](iamtimelineeffect.md)
-   [**IAMTimelineObj**](iamtimelineobj.md)
-   [**IAMTimelineSplittable**](iamtimelinesplittable.md)
-   **IPropertyPage**
