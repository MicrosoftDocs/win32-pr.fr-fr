# <a name="directshow-data-types"></a>DirectShow Types de données

DirectShow fournit les définitions de type suivantes.



| Type de données                                 | Description                                                                                       |
|-------------------------------------------|---------------------------------------------------------------------------------------------------|
| [**Registre de DVD \_**](dvd-register.md)     | Contient une valeur pour un registre de paramètres généraux de DVD (GPRM) ou un registre de paramètres système (SPRM). |
| [**GPRMARRAY**](gprmarray.md)            | Contient un tableau de valeurs GPRM.                                                                 |
| [**OAEVENT**](oaevent.md)                | Définit un handle d’événement compatible Automation.                                                    |
| [**OAFilterState**](oafilterstate.md)    | Définit l’état du graphique de filtre.                                                            |
| [**OAHWND**](oahwnd.md)                  | Définit un handle de fenêtre compatible Automation.                                                   |
| [**temps de référence \_**](reference-time.md) | Définit le type de données pour le temps de référence.                                                         |
| [**REFTIME**](reftime.md)                | Définit le type de données utilisé pour exprimer le temps en secondes.                                            |
| [**SPRMARRAY**](sprmarray.md)            | Contient un tableau de valeurs SPRM.                                                                 |
