# <a name="avencddltrtcentermixlvl_x10-property"></a>AVEncDDLtRtCenterMixLvl \_ x10, propriété

Spécifie le niveau de décalage appliqué au canal central pour lt/RT downmixing. Cette propriété s’applique aux encodeurs audio Dolby Digital.

Cette propriété est en lecture/écriture.

## <a name="data-type"></a>Type de données

**UInt32** (**VT \_ UI4**)

## <a name="property-guid"></a>Guid de propriété

**CODECAPI \_ AVEncDDLtRtCenterMixLvl \_ x10**

## <a name="property-value"></a>Valeur de la propriété

La valeur est spécifiée comme décibels (dB) x 10.

## <a name="remarks"></a>Remarques

Cette propriété est en lecture/écriture.
