# <a name="avencmpaprivateuserbit-property"></a>Propriété AVEncMPAPrivateUserBit

Définit la valeur du bit d’utilisateur privé dans le flux MPEG-1. Cette propriété s’applique aux encodeurs audio MPEG.

Cette propriété est en lecture/écriture.

## <a name="data-type"></a>Type de données

**UInt32** (**VT \_ UI4**)

## <a name="property-guid"></a>Guid de propriété

**CODECAPI \_ AVEncMPAPrivateUserBit**

## <a name="property-value"></a>Valeur de la propriété

Cette propriété peut avoir la valeur 0 ou 1.
