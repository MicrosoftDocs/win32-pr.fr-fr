# <a name="cimageallocatorfree-method"></a>CImageAllocator. free, méthode

La `Free` méthode libère toute la mémoire tampon. Cette méthode implémente la méthode [**CBaseAllocator :: Free**](cbaseallocator-free.md) virtuelle pure.

## <a name="syntax"></a>Syntaxe


```C++
void Free();
```



## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.

## <a name="return-value"></a>Valeur de retour

Cette méthode ne retourne pas de valeur.
