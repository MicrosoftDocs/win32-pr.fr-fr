# <a name="ccactive-property"></a>Propriété CCActive

> [!Note]  
> ce composant peut être utilisé dans les systèmes d’exploitation Microsoft Windows 2000, Windows XP et Windows Server 2003. Il sera peut-être modifié ou indisponible dans les versions ultérieures.

 

La `CCActive` propriété définit ou récupère l’état actuel du sous-titrage.

``` syntax
[ bCCActive = ] MSWebDVD.CCActive
```

## <a name="return-value"></a>Valeur renvoyée

Retourne une valeur booléenne indiquant si le sous-titrage est activé.

## <a name="remarks"></a>Remarques

Cette propriété est en lecture/écriture et sa valeur par défaut est false.
