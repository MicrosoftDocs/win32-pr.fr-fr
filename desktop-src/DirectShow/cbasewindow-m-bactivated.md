# <a name="cbasewindowm_bactivated-member"></a>CBaseWindow :: m \_ bActivated, membre

Indicateur qui spécifie si la fenêtre a été activée. Si la **valeur est true**, la fenêtre a été activée.

## <a name="syntax"></a>Syntaxe


```C++
BOOL m_bActivated;
```
