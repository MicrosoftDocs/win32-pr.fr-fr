# <a name="coutputqueuem_bsendanyway-member"></a>COutputQueue :: m \_ bSendAnyway, membre

Indicateur de remplacement du traitement par lots. L’affectation de la **valeur true** à cet indicateur remplace l’indicateur [**COutputQueue :: m \_ bBatchExact**](coutputqueue-m-bbatchexact.md) et remet tous les exemples en attente.

## <a name="syntax"></a>Syntaxe


```C++
BOOL m_bSendAnyway;
```
