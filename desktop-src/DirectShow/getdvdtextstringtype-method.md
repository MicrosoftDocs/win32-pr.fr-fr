# <a name="getdvdtextstringtype-method"></a>Méthode GetDVDTextStringType

> [!Note]  
> ce composant peut être utilisé dans les systèmes d’exploitation Microsoft Windows 2000, Windows XP et Windows Server 2003. Il sera peut-être modifié ou indisponible dans les versions ultérieures.

 

La `GetDVDTextStringType` méthode récupère une valeur qui indique le type d’informations contenues dans la chaîne de texte de DVD spécifiée.

``` syntax
[ iStringType = ] MSWebDVD.GetDVDTextStringType(iLangIndex, iStringIndex)
```

## <a name="parameters"></a>Paramètres

<dl> <dt>

<span id="iLangIndex"></span><span id="ilangindex"></span><span id="ILANGINDEX"></span>*iLangIndex*
</dt> <dd>

Spécifie la langue de la chaîne de texte sous la forme d’un entier.

</dd> <dt>

<span id="iStringIndex"></span><span id="istringindex"></span><span id="ISTRINGINDEX"></span>*iStringIndex*
</dt> <dd>

Spécifie le numéro d’index de la chaîne de texte sous la forme d’un entier.

</dd> </dl>

## <a name="return-value"></a>Valeur renvoyée

Retourne une valeur entière qui indique le type d’informations contenues dans la chaîne.
