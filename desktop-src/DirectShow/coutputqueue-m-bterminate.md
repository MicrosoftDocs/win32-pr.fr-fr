# <a name="coutputqueuem_bterminate-member"></a>COutputQueue :: m \_ bTerminate, membre

Indicateur qui spécifie si le thread doit se terminer. Si la **valeur est true**, le thread se termine.

## <a name="syntax"></a>Syntaxe


```C++
BOOL m_bTerminate;
```
