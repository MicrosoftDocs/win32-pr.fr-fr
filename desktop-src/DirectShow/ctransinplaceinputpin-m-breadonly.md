# <a name="ctransinplaceinputpinm_breadonly-member"></a>CTransInPlaceInputPin :: m \_ bReadOnly, membre

Indicateur qui spécifie si le flux d’entrée est en lecture seule. Le filtre amont spécifie ces informations lorsqu’il appelle la méthode [**NotifyAllocator**](ctransinplaceinputpin-notifyallocator.md) . Par défaut, la valeur est **false**.

## <a name="syntax"></a>Syntaxe


```C++
BOOL m_bReadOnly;
```
