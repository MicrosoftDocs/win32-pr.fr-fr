# <a name="french-overseas-territories"></a>Territoires français d’outre-mer

Pour obtenir une explication des tableaux suivants, consultez [tables et affectations](tables-and-assignments.md).

Les index de fréquence dans la table de diffusion des territoires français d’outre-mer peuvent être utilisés comme numéros de canaux ; les index de fréquence de câble ne conviennent pas pour une utilisation en tant que numéros de canaux.

### <a name="french-overseas-territories-broadcast-assignments"></a>Attributions de diffusion de territoires français d’outre-mer


```C++
         4L,            // Lowest channel
         9L,            // Highest channel

       175250000L,      // 4
       183250000L,      // 5
       191250000L,      // 6
       199250000L,      // 7
       207250000L,      // 8
       215250000L,      // 9
```



### <a name="french-overseas-territories-cable-assignments"></a>Attributions de câbles aux territoires français d’outre-mer


```C++
         4L,            // Lowest channel
         9L,            // Highest channel

       175250000L,      // 4
       183250000L,      // 5
       191250000L,      // 6
       199250000L,      // 7
       207250000L,      // 8
       215250000L,      // 9
```
