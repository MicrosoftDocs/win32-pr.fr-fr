# <a name="system-device-enumerator"></a>Énumérateur de périphérique système

L’énumérateur de périphérique système énumère les filtres et les périphériques matériels installés sur le système. Les applications peuvent utiliser ce composant pour localiser des filtres et des périphériques dans une catégorie donnée. Créez cet objet en appelant **CoCreateInstance**.



| Étiquette | Valeur |
|------------------|------------------------------------------|
| Identificateur de classe | CLSID \_ SystemDeviceEnum                  |
| Interfaces       | [**ICreateDevEnum**](/windows/desktop/api/Strmif/nn-strmif-icreatedevenum) |
