# <a name="dvd-interfaces"></a>Interfaces DVD

Ces interfaces prennent en charge la lecture et la navigation sur DVD.



| Interface                                    | Description                                                                                            |
|----------------------------------------------|--------------------------------------------------------------------------------------------------------|
| [**IAMDecoderCaps**](/windows/desktop/api/Strmif/nn-strmif-iamdecodercaps)     | Implémenté par les décodeurs pour indiquer diverses fonctionnalités.                                              |
| [**IDvdCmd**](/windows/desktop/api/Strmif/nn-strmif-idvdcmd)                   | Bloquer le [filtre du navigateur DVD](dvd-navigator-filter.md) jusqu’à ce qu’une commande démarre ou se termine.        |
| [**IDvdControl2**](/windows/desktop/api/Strmif/nn-strmif-idvdcontrol2)         | Parcourez et lisez DVD-Video titres, y compris les titres de karaoké.                                          |
| [**IDvdGraphBuilder**](/windows/desktop/api/Strmif/nn-strmif-idvdgraphbuilder) | Créez un graphique de filtre pour DVD-Video la lecture à partir des logiciels et du matériel disponibles sur le système.    |
| [**IDvdInfo2**](/windows/desktop/api/Strmif/nn-strmif-idvdinfo2)               | Interrogez les différents attributs d’un disque DVD ou de l’état du navigateur DVD.                               |
| [**IDvdState**](/windows/desktop/api/Strmif/nn-strmif-idvdstate)               | Enregistrez l’état de la session de l’utilisateur sur le disque, y compris l’emplacement de lecture, le niveau parental et la région. |
