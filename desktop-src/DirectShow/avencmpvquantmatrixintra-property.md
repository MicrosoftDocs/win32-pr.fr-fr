# <a name="avencmpvquantmatrixintra-property"></a>Propriété AVEncMPVQuantMatrixIntra

Spécifie la matrice de quantification de luminance pour blocs macros. Cette propriété s’applique aux encodeurs vidéo MPEG.

Cette propriété est en lecture/écriture.

## <a name="data-type"></a>Type de données

**BSTR** (**VT \_ BSTR**)

## <a name="property-guid"></a>Guid de propriété

**CODECAPI \_ AVEncMPVQuantMatrixIntra**

## <a name="property-value"></a>Valeur de la propriété

La valeur de cette propriété est une chaîne qui contient les coefficients 64 8 bits encodés sous la forme d’une chaîne de valeurs hexadécimales (128 caractères).
