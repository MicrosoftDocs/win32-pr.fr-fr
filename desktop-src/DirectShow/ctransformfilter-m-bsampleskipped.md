# <a name="ctransformfilterm_bsampleskipped-member"></a>CTransformFilter :: m \_ bSampleSkipped, membre

Indicateur qui spécifie si l’échantillon le plus récent a été supprimé. Si la méthode [**Receive**](ctransformfilter-receive.md) supprime un exemple, elle définit la valeur sur **true**.

## <a name="syntax"></a>Syntaxe


```C++
BOOL m_bSampleSkipped;
```
