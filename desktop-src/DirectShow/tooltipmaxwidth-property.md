# <a name="tooltipmaxwidth-property"></a>Propriété ToolTipMaxWidth

> [!Note]  
> ce composant peut être utilisé dans les systèmes d’exploitation Microsoft Windows 2000, Windows XP et Windows Server 2003. Il sera peut-être modifié ou indisponible dans les versions ultérieures.

 

La `ToolTipMaxWidth` propriété définit ou récupère la largeur maximale de l’info-bulle associée à l’objet **mswebdvd** .

``` syntax
[ iMaxWidth = ] MSWebDVD.ToolTipMaxWidth
```

## <a name="return-value"></a>Valeur renvoyée

Retourne la largeur maximale de l’info-bulle de l’application sous la forme d’un entier.

## <a name="remarks"></a>Remarques

Cette propriété est en lecture/écriture avec une valeur par défaut de 200 pixels.
