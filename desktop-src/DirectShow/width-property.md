# <a name="width-property-directshow"></a>Width, propriété (DirectShow)

> [!Note]  
> ce composant peut être utilisé dans les systèmes d’exploitation Microsoft Windows 2000, Windows XP et Windows Server 2003. Il sera peut-être modifié ou indisponible dans les versions ultérieures.

 

La `Width` propriété définit ou récupère la largeur du rectangle.

``` syntax
[ iWidth = ] DVDRect.Width
```

## <a name="return-value"></a>Valeur renvoyée

Retourne la largeur, en pixels, du rectangle sous la forme d’un entier.

## <a name="remarks"></a>Remarques

Cette propriété est en lecture/écriture sans valeur par défaut.
