# <a name="cimagedisplayispalettised-method"></a>Méthode CImageDisplay. IsPalettised

La `IsPalettised` méthode détermine si le format d’affichage actuel est en palette.

## <a name="syntax"></a>Syntaxe


```C++
BOOL IsPalettised();
```



## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.

## <a name="return-value"></a>Valeur de retour

Retourne la **valeur true** si le format est en palette, ou **false** dans le cas contraire.
