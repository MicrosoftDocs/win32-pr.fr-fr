# <a name="dvd_register"></a>Registre de DVD \_

Le type de données **\_ Registre de DVD** contient une valeur pour un registre de paramètres généraux de DVD (GPRM) ou un registre de paramètres système (SPRM).


```C++
typedef WORD DVD_REGISTER;
```
