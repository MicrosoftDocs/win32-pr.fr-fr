# <a name="working-with-sources"></a>Utilisation des sources

\[Cette API n’est pas prise en charge et peut être modifiée ou non disponible à l’avenir.\]

Un objet source représente un flux de média unique à partir d’un fichier source. Les sources fournissent la matière brute d’un projet. Tout le reste de la chronologie existe pour fournir une structure (pistes, compositions et groupes) ou pour modifier les sources (effets et transitions).

Cet article décrit les différents détails relatifs aux sources dans Microsoft® DirectShow® édition services (DES). Pour plus d’informations sur les étapes de base nécessaires pour ajouter une source à la chronologie, consultez [construction d’une chronologie](constructing-a-timeline.md).

Cet article contient les sections suivantes.

-   [Définition des propriétés sur les sources](setting-properties-on-sources.md)
-   [Utilisation du détecteur de média](using-the-media-detector.md)
-   [Utilisation du localisateur de média](using-the-media-locator.md)
-   [Autres objets sources](other-source-objects.md)
