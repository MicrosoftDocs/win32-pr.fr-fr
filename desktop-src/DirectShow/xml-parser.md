# <a name="xml-parser"></a>Analyseur XML

> [!Note]  
> \[Action déconseillée. Cette API peut être supprimée dans les versions futures de Windows.\]

 

L’objet analyseur XML convertit entre les fichiers de projet XML et les chronologies. Pour créer cet objet, appelez **CoCreateInstance**.

L’objet analyseur XML expose l’interface suivante :

-   [**IXml2Dex**](ixml2dex.md)

**Identificateur de classe :** CLSID \_ Xml2Dex
