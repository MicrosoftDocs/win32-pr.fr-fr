# <a name="cbasepropertypagem_bdirty-member"></a>CBasePropertyPage :: m \_ bDirty, membre

L’indicateur **m \_ bDirty** indique si l’une des propriétés a changé. Affectez à cet indicateur la **valeur true** si l’utilisateur modifie des propriétés par le biais des contrôles de boîte de dialogue.

## <a name="syntax"></a>Syntaxe


```C++
BOOL m_bDirty;
```
