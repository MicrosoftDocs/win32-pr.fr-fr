# <a name="xtl-elements"></a>Éléments XTL

> [!Note]  
> \[Déconseillé. Cette API peut être supprimée des futures versions de Windows.\]

 

Le format XTL définit les éléments suivants.



| Élément                                  | Description                                                                    |
|------------------------------------------|--------------------------------------------------------------------------------|
| [**à**](at-element.md)                 | Spécifie un paramètre basé sur l’heure pour un élément **param** .                        |
| [**capture**](clip-element.md)             | Spécifie un fichier source.                                                       |
| [**composite**](composite-element.md)   | Définit une composition.                                                         |
| [**résultat**](effect-element.md)         | Définit un effet.                                                             |
| [**Communauté**](group-element.md)           | Définit un groupe (audio ou vidéo).                                              |
| [**proportionnelle**](linear-element.md)         | Spécifie un paramètre basé sur l’heure pour un élément **param** .                        |
| [**param**](param-element.md)           | Spécifie la valeur d’une propriété sur une transition, un effet ou un autre sous-objet. |
| [**actualité**](timeline-element.md)     | Définit la chronologie.                                                          |
| [**track**](track-element.md)           | Définit une piste.                                                               |
| [**passe**](transition-element.md) | Définit une transition.                                                          |
