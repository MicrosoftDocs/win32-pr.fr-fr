# <a name="ctransinplacefiltertypesmatch-method"></a>Méthode CTransInPlaceFilter. TypesMatch

La `TypesMatch` méthode détermine si le type de média d’entrée correspond au type de média de sortie.

## <a name="syntax"></a>Syntaxe


```C++
BOOL TypesMatch();
```



## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.

## <a name="return-value"></a>Valeur retournée

Retourne la **valeur true** si les types de média d’entrée et de sortie sont identiques, ou **false** dans le cas contraire.
