# <a name="avencddoriginalbitstream-property"></a>Propriété AVEncDDOriginalBitstream

Spécifie l’indicateur de flux de bits d’origine dans un flux audio Dolby Digital. Cette propriété s’applique aux encodeurs audio Dolby Digital.

\\

Cette propriété est en lecture/écriture.

## <a name="data-type"></a>Type de données

**Variante \_ BOOL** (**VT \_ bool**)

## <a name="property-guid"></a>Guid de propriété
