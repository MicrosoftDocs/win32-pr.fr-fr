# <a name="cbasewindowm_bnorealize-member"></a>CBaseWindow :: m \_ bNoRealize, membre

Indicateur qui spécifie si la fenêtre doit réaliser sa palette. Si la **valeur est true**, la fenêtre ne réalise pas sa palette. La valeur par défaut est **FALSE**.

## <a name="syntax"></a>Syntaxe


```C++
BYTE m_bNoRealize;
```
