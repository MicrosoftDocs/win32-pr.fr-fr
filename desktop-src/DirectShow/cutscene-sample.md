# <a name="cutscene-sample"></a>Exemple CutScene

## <a name="description"></a>Description

L’exemple CutScene lit un fichier vidéo en mode plein écran jusqu’à ce que l’utilisateur tape ÉCHAP, ENTER ou la barre d’espace.

Exécutez cet exemple à partir de la ligne de commande, avec le nom d’un fichier vidéo en tant qu’argument de ligne de commande.

## <a name="downloading-the-sample"></a>Téléchargement de l’exemple

pour télécharger les exemples du kit de développement logiciel (SDK) DirectShow, installez la dernière version du [SDK Windows](https://msdn.microsoft.com/windowsvista/bb980924.aspx).

cet exemple est installé sous le chemin d’accès suivant : exemples *\[ racine \] du kit de développement logiciel (SDK)* \\ DirectShow les \\ \\ \\ lecteurs \\ CutScene.
