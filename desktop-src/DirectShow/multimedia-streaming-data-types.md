# <a name="multimedia-streaming-data-types"></a>Types de données de diffusion multimédia

> [!Note]  
> Cette API est déconseillée. Les nouvelles applications ne doivent pas l’utiliser.

 

Les interfaces de diffusion multimédia en continu Microsoft DirectShow utilisent les types de données suivants.



| Type de données                           | Description                                            |
|-------------------------------------|--------------------------------------------------------|
| [**MSPID**](mspid.md)              | Identifie l’objectif d’un flux multimédia.              |
| [**temps de flux \_**](stream-time.md) | Définit le type de données utilisé pour exprimer les temps de référence. |
