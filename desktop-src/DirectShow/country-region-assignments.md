# <a name="countryregion-assignments"></a>Attributions de pays/région

Les codes d’appel du pays standard de l’Union internationale des télécommunications (ITU) sont passés à la méthode [**IAMTuner ::p ut \_ CountryCode**](/windows/desktop/api/Strmif/nf-strmif-iamtuner-put_countrycode) pour sélectionner les mappages de fréquences TV par défaut.

Par exemple, si vous passez 20, le code appelant ITU-T pour l’Égypte, sélectionnera les fréquences de télévision par défaut pour l’Égypte.

Pour obtenir la liste des codes d’appel des pays, reportez-vous à la recommandation ITU-T actuelle des codes de pays attribués.
