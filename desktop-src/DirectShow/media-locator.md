# <a name="media-locator"></a>Localisateur de média

> [!Note]  
> \[Action déconseillée. Cette API peut être supprimée dans les versions futures de Windows.\]

 

L’objet localisateur de média localise les fichiers associés aux objets sources. Utilisez cet objet pour valider les sources dans la chronologie. Pour créer cet objet, appelez **CoCreateInstance**. L’identificateur de classe est CLSID \_ MediaLocator.

L’objet Media Locator expose l’interface suivante :

-   [**IMediaLocator**](imedialocator.md)
