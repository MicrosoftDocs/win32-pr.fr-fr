# <a name="aspectratio-property"></a>Propriété AspectRatio

> [!Note]  
> ce composant peut être utilisé dans les systèmes d’exploitation Microsoft Windows 2000, Windows XP et Windows Server 2003. Il sera peut-être modifié ou indisponible dans les versions ultérieures.

 

La `AspectRatio` propriété récupère les proportions du flux vidéo actuel tel qu’il est créé sur le disque.

``` syntax
[  nAspectRatio = ] MSWebDVD.AspectRatio 
```

## <a name="return-value"></a>Valeur renvoyée

Retourne un nombre représentant les proportions du flux vidéo actuel tel qu’il a été créé sur le disque (1,33 pour un proportions de 4:3 ou 1,78 pour un proportions de 16:9).

## <a name="remarks"></a>Remarques

Cette propriété est en lecture seule sans valeur par défaut.
