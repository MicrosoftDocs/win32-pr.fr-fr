# <a name="oahwnd"></a>OAHWND

le `OAHWND` type de données définit un handle de fenêtre qui peut être utilisé par les clients OLE Automation, tels que Microsoft Visual Basic 6,0.


```C++
typedef LONG_PTR OAHWND;
```
