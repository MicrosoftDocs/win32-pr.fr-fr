# <a name="cbaseinputpinm_bflushing-member"></a>CBaseInputPin :: m \_ bFlushing, membre

Indicateur qui spécifie si le pin est actuellement en cours de vidage. Si la **valeur est true**, le code PIN est vidé.

## <a name="syntax"></a>Syntaxe


```C++
BYTE m_bFlushing;
```
