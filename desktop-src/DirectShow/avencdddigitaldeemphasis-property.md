# <a name="avencdddigitaldeemphasis-property"></a>Propriété AVEncDDDigitalDeemphasis

Spécifie si la mise en évidence numérique est activée dans un flux de données audio Dolby Digital. Cette propriété s’applique aux encodeurs audio Dolby Digital.

Cette propriété est en lecture/écriture.

## <a name="data-type"></a>Type de données

**Variante \_ BOOL** (**VT \_ bool**)

## <a name="property-guid"></a>Guid de propriété

**CODECAPI \_ AVEncDDDigitalDeemphasis**
