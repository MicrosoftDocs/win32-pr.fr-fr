# <a name="ec_dvd_beginnavigationcommands"></a>\_BeginNavigationCommands de DVD EC \_

Envoyé lors du démarrage d’un jeu de commandes de navigation sur DVD.

## <a name="parameters"></a>Paramètres

<dl> <dt>

<span id="lParam1"></span><span id="lparam1"></span><span id="LPARAM1"></span>*lParam1*
</dt> <dd>

Valeur de l’énumération [**\_ NavCmdType DVD**](/windows/win32/api/strmif/ne-strmif-dvd_navcmdtype) .

</dd> <dt>

<span id="lParam2"></span><span id="lparam2"></span><span id="LPARAM2"></span>*lParam2*
</dt> <dd>

Zéro.

</dd> </dl>

## <a name="remarks"></a>Notes

Cet événement est désactivé par défaut. Pour activer cet événement, appelez [**IDvdControl2 :: SetOption**](/windows/desktop/api/Strmif/nf-strmif-idvdcontrol2-setoption) et affectez à l’option **DVD \_ EnableLoggingEvents** la **valeur true**.
