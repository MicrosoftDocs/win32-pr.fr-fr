# <a name="cmediasamplem_lactual-member"></a>CMediaSample :: m \_ lActual, membre

Longueur des données valides dans la mémoire tampon, en octets. La valeur doit être inférieure ou égale à la taille de la mémoire tampon, spécifiée par la variable membre [**CMediaSample :: m \_ cbBuffer**](cmediasample-m-cbbuffer.md) .

## <a name="syntax"></a>Syntaxe


```C++
LONG m_lActual;
```
