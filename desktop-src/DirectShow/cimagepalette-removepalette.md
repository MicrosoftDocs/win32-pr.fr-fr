# <a name="cimagepaletteremovepalette-method"></a>Méthode CImagePalette. RemovePalette

La `RemovePalette` méthode supprime la palette logique existante et appelle [**CBaseWindow :: UnsetPalette**](cbasewindow-unsetpalette.md) sur l’objet **CBaseWindow** .

## <a name="syntax"></a>Syntaxe


```C++
HRESULT RemovePalette();
```



## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.

## <a name="return-value"></a>Valeur retournée

Retourne S \_ OK.
