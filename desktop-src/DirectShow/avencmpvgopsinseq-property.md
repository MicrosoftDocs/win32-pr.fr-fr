# <a name="avencmpvgopsinseq-property"></a>Propriété AVEncMPVGOPSInSeq

Spécifie le nombre de groupes d’images (GOP secondes) entre les en-têtes de séquence. Cette propriété s’applique aux encodeurs vidéo MPEG.

Cette propriété est en lecture/écriture.

## <a name="data-type"></a>Type de données

**UInt32** (**VT \_ UI4**)

## <a name="property-guid"></a>Guid de propriété

**CODECAPI \_ AVEncMPVGOPSInSeq**

## <a name="remarks"></a>Remarques

Cette propriété est en lecture/écriture.
