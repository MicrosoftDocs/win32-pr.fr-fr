# <a name="ec_stream_error_stopped"></a>\_erreur de flux EC \_ \_ arrêtée

Un flux s’est arrêté en raison d’une erreur.

## <a name="parameters"></a>Paramètres

<dl> <dt>

<span id="lParam1"></span><span id="lparam1"></span><span id="LPARAM1"></span>*lParam1*
</dt> <dd>

(**HRESULT**) Code d’échec de l’opération qui a échoué.

</dd> <dt>

<span id="lParam2"></span><span id="lparam2"></span><span id="LPARAM2"></span>*lParam2*
</dt> <dd>

(**DWORD**) Code d’erreur mineur. Cette valeur est généralement égale à zéro.

</dd> </dl>

## <a name="default-action"></a>Action par défaut

Aucun.
