# <a name="cimagesamplem_dibdata-member"></a>CImageSample :: m \_ DibData, membre

La `m_DibData` variable membre est une structure [**DIBDATA**](dibdata.md) qui contient des informations sur une image bitmap indépendante du périphérique (DIB) GDI.

## <a name="syntax"></a>Syntaxe


```C++
DIBDATA m_DibData;
```
