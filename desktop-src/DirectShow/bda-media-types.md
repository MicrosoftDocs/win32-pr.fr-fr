Ce type de média est utilisé pour la connexion de code confidentiel entre le filtre du fournisseur réseau et le filtre tuner.



| Étiquette | Valeur |
|-------------|----------------------------------|
| Type principal  | \_ \_ antenne BDA de type KSDATAFORMAT \_ |
| Subtype     | MEDIASUBTYPE \_ aucun               |
| Type de format | FORMAT \_ aucun                     |
