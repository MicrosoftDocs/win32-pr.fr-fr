# <a name="media-property-bag"></a>Conteneur des propriétés du média

L’objet Media Property Bag définit ou récupère des blocs d’informations et d’extraction dans des fichiers AVI. Créez cet objet en appelant **CoCreateInstance**.

|------------------|------------------------------------------------|
| Identificateur de classe | CLSID \_ MediaPropertyBag                        |
| Interfaces       | [**IMediaPropertyBag**](/windows/desktop/api/Strmif/nn-strmif-imediapropertybag) |
