# <a name="avencvideoheaderminutes-property"></a>Propriété AVEncVideoHeaderMinutes

Spécifie le numéro de la minute de début dans l’en-tête groupe d’images (GOP).

Cette propriété est en lecture/écriture.

## <a name="data-type"></a>Type de données

**UInt32** (**VT \_ UI4**)

## <a name="property-guid"></a>Guid de propriété

**CODECAPI \_ AVEncVideoHeaderMinutes**

## <a name="property-value"></a>Valeur de la propriété

La plage valide de cette propriété est comprise entre 0 et 59 inclus.
