# <a name="src-attribute"></a>Attribut SRC

> [!Note]  
> \[Déconseillé. Cette API peut être supprimée des futures versions de Windows.\]

 

L' `src` attribut spécifie le nom du chemin d’accès d’un fichier multimédia. le fichier peut être n’importe quel type pris en charge par DirectShow ou une image continue. Les formats pris en charge pour les images fixes sont bitmap, bitmap indépendante du périphérique, GIF et JPEG.

## <a name="possible-values"></a>Valeurs possibles

Doit être une chaîne qui spécifie un nom de chemin d’accès.
