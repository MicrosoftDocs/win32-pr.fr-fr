# <a name="pointer-validation-macros"></a>Macros de validation du pointeur

Microsoft DirectShow fournit plusieurs macros pour valider les pointeurs.



| Macro                                                | Description                                                                   |
|------------------------------------------------------|-------------------------------------------------------------------------------|
| [**Point de contrôle**](checkpointer.md)                 | Vérifie si un pointeur est **null**.                                         |
| [**ValidateReadPtr**](validatereadptr.md)           | Vérifie que le processus appelant dispose d’un accès en lecture à un bloc de mémoire.          |
| [**ValidateReadWritePtr**](validatereadwriteptr.md) | Vérifie que le processus appelant dispose d’un accès en lecture/écriture à un bloc de mémoire.    |
| [**ValidateStringPtr**](validatestringptr.md)       | Vérifie que le processus appelant dispose d’un accès en lecture à une chaîne.                |
| [**ValidateStringPtrA**](validatestringptra.md)     | Vérifie que le processus appelant dispose d’un accès en lecture à une chaîne ANSI.          |
| [**ValidateStringPtrW**](validatestringptrw.md)     | Vérifie que le processus appelant dispose d’un accès en lecture à une chaîne de caractères larges. |
| [**ValidateWritePtr**](validatewriteptr.md)         | Vérifie que le processus appelant dispose d’un accès en écriture à un bloc de mémoire.         |
