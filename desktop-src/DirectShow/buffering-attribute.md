# <a name="buffering-attribute"></a>Attribut de mise en mémoire tampon

> [!Note]  
> \[Déconseillé. Cette API peut être supprimée des futures versions de Windows.\]

 

L' `buffering` attribut spécifie le nombre d’images à restituer à l’avance pendant la version préliminaire.

## <a name="possible-values"></a>Valeurs possibles

La valeur doit être un nombre supérieur à 1. La valeur par défaut est 30.
