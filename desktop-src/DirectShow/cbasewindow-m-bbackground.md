# <a name="cbasewindowm_bbackground-member"></a>CBaseWindow :: m \_ bBackground, membre

Indicateur qui spécifie si la palette doit être une palette d’arrière-plan. Si la **valeur est true**, la palette est sélectionnée en tant que palette d’arrière-plan.

## <a name="syntax"></a>Syntaxe


```C++
BYTE m_bBackground;
```
