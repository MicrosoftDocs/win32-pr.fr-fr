# <a name="amcap-sample"></a>Exemple AmCap

## <a name="description"></a>Description

Application de capture vidéo.

Cet exemple d’application montre les tâches suivantes liées à la capture audio et vidéo :

-   Capturer dans un fichier
-   Aperçu instantané
-   Allocation du fichier de capture
-   Affichage des pages de propriétés de l’appareil
-   Énumération des appareils
-   Contrôle de flux

AMCap prend en charge l’entrée de flux de programme MPEG-2, par exemple à partir de tuner TV analogiques qui diffusent du contenu MPEG-2. un décodeur MPEG-2 compatible DirectShow est requis pour décoder les flux.
