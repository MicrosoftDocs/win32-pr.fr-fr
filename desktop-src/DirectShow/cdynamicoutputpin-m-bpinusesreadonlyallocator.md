# <a name="cdynamicoutputpinm_bpinusesreadonlyallocator-member"></a>CDynamicOutputPin :: m \_ bPinUsesReadOnlyAllocator, membre

Indicateur qui spécifie si les échantillons de l’allocateur du pin sont en lecture seule. Si la valeur est **true**, l’exemple est en lecture seule. La valeur par défaut est **FALSE**.

## <a name="syntax"></a>Syntaxe


```C++
BOOL m_bPinUsesReadOnlyAllocator;
```
