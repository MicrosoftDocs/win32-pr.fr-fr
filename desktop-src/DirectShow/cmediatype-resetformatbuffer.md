# <a name="cmediatyperesetformatbuffer-method"></a>Méthode CMediaType. ResetFormatBuffer

La `ResetFormatBuffer` méthode supprime le bloc de format. La méthode définit les membres **cbFormat** et **pbFormat** de la structure de **\_ \_ type de média am** sur la **valeur null**.

## <a name="syntax"></a>Syntaxe


```C++
void ResetFormatBuffer();
```



## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.

## <a name="return-value"></a>Valeur retournée

Cette méthode ne retourne pas de valeur.
