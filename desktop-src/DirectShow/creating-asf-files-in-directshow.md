# <a name="creating-asf-files-in-directshow-directshow"></a>Création de fichiers ASF en DirectShow (DirectShow)

cette section décrit comment utiliser le filtre DirectShow [WM asf Writer](wm-asf-writer-filter.md) pour créer des fichiers au Format ASF (Advanced Systems Format). ASF est un format de conteneur qui peut contenir n’importe quel type de données compressées ou non compressées. Windows les fichiers au format de média sont des fichiers ASF qui utilisent certains codecs, comme spécifié dans le kit de développement logiciel (SDK) Windows Media format. ces fichiers utilisent les extensions « . wma » pour les fichiers Windows Media Audio et « . wmv » pour les fichiers Windows Media Video.
