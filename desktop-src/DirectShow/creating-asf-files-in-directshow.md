# <a name="creating-asf-files-in-directshow-directshow"></a>Création de fichiers ASF dans DirectShow (DirectShow)

Cette section décrit comment utiliser le filtre de l' [enregistreur ASF](wm-asf-writer-filter.md) DirectShow DirectShow pour créer des fichiers au format ASF (Advanced Systems Format). ASF est un format de conteneur qui peut contenir n’importe quel type de données compressées ou non compressées. Les fichiers de format Windows Media sont des fichiers ASF qui utilisent certains codecs, comme spécifié dans le kit de développement logiciel (SDK) Windows Media format. Ces fichiers utilisent les extensions « . WMA » pour les fichiers Windows Media Audio et « . wmv » pour les fichiers Windows Media Video.
