# <a name="cenummediatypesreset-method"></a>CEnumMediaTypes. Reset, méthode

La `Reset` méthode réinitialise la séquence d’énumération au début. Cette méthode implémente la méthode [**IEnumMediaTypes :: Reset**](/windows/desktop/api/Strmif/nf-strmif-ienummediatypes-reset) .

## <a name="syntax"></a>Syntaxe


```C++
HRESULT Reset();
```



## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.

## <a name="return-value"></a>Valeur retournée

Retourne S \_ OK.
