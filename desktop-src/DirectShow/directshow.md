# <a name="directshow"></a>DirectShow

l’interface de programmation d’applications (API) de microsoft DirectShow est une architecture de diffusion multimédia en continu pour Microsoft Windows. à l’aide de DirectShow, vos applications peuvent effectuer des lectures ou des captures vidéo et audio de haute qualité.

les en-têtes DirectShow, les bibliothèques, les outils du kit de développement logiciel (SDK) et les exemples sont disponibles dans le [SDK Windows](https://msdn.microsoft.com/windows/aa904949.aspx).

> [!Note]  
> les versions précédentes du kit de développement logiciel (sdk) DirectShow étaient incluses dans le sdk DirectX. la dernière version du kit de développement logiciel (sdk) directx à inclure DirectShow était la mise à jour directx 9,0 SDK-(février 2005). après cette version, DirectShow a été déplacé vers le SDK Windows. pour accéder à la dernière version des en-têtes, bibliothèques et exemples DirectShow, téléchargez le SDK Windows.
