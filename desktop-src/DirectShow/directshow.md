# <a name="directshow"></a>DirectShow

L’interface de programmation d’applications (API) Microsoft DirectShow est une architecture de diffusion multimédia en continu pour Microsoft Windows. À l’aide de DirectShow, vos applications peuvent effectuer des lectures ou des vidéos de haute qualité.

Les en-têtes DirectShow, les bibliothèques, les outils du kit de développement logiciel (SDK) et les exemples sont disponibles dans le [SDK Windows](https://msdn.microsoft.com/windows/aa904949.aspx).

> [!Note]  
> Les versions précédentes du kit de développement logiciel (SDK) DirectShow étaient incluses dans le SDK DirectX. La dernière version du kit de développement logiciel (SDK) DirectX pour inclure DirectShow était DirectX 9,0 SDK Update-(février 2005) Extras. Après cette version, DirectShow a été déplacé vers le SDK Windows. Pour accéder à la dernière version des en-têtes, bibliothèques et exemples DirectShow, téléchargez le SDK Windows.
