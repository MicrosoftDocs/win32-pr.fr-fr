# <a name="width-attribute"></a>Attribut width

> [!Note]  
> \[Déconseillé. Cette API peut être supprimée des futures versions de Windows.\]

 

L' `width` attribut spécifie la largeur de la vidéo de sortie, en pixels.

## <a name="possible-values"></a>Valeurs possibles

Valeur à virgule flottante. La valeur doit inclure le zéro non significatif avant la décimale. Par exemple, 0,3, pas. 3. N’utilisez pas plus de sept chiffres décimaux.

## <a name="applies-to"></a>S'applique à

[**Communauté**](group-element.md)

## <a name="remarks"></a>Remarques

Définissez cet attribut uniquement si l’attribut de **type** est `video` .
