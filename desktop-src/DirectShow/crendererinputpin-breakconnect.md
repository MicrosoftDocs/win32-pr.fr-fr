# <a name="crendererinputpinbreakconnect-method"></a>Méthode CRendererInputPin. BreakConnect

La méthode **BreakConnect** libère le code confidentiel d’une connexion. Cette méthode remplace la méthode [**CBaseInputPin :: BreakConnect**](cbaseinputpin-breakconnect.md) .

## <a name="syntax"></a>Syntaxe


```C++
HRESULT BreakConnect();
```



## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.

## <a name="return-value"></a>Valeur de retour

Retourne une valeur **HRESULT** .
