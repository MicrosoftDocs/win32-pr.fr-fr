# <a name="eject-method"></a>EJECT, méthode

> [!Note]  
> Ce composant peut être utilisé dans les systèmes d’exploitation Microsoft Windows 2000, Windows XP et Windows Server 2003. Il sera peut-être modifié ou indisponible dans les versions ultérieures.

 

La `Eject` méthode éjecte ou insère un disque à partir de ou dans le lecteur de DVD.

``` syntax
        MSWebDVD.Eject()
```

## <a name="return-value"></a>Valeur renvoyée

Pas de valeur de retour.

## <a name="remarks"></a>Notes

Sur certains lecteurs de DVD, cette méthode agit comme un basculement, l’éjection et l’insertion sur d’autres appels. Cela dépend du matériel.
