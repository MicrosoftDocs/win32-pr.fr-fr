# <a name="property-page-helper-functions"></a>Fonctions d’assistance de page de propriétés

Le fichier d’en-tête videoctl. h dans les classes de base Microsoft® DirectShow® fournit des fonctions pour faciliter l’implémentation des pages de propriétés.



| Fonction                                                 | Description                                                                            |
|----------------------------------------------------------|----------------------------------------------------------------------------------------|
| [**GetDialogSize**](getdialogsize.md)                   | Récupère la taille d’une boîte de dialogue de ressource en pixels d’écran.                          |
| [**StringFromResource**](stringfromresource.md)         | Charge une chaîne à partir d’un fichier de ressources avec l’identificateur de ressource donné.                |
| [**WideStringFromResource**](widestringfromresource.md) | Charge une chaîne de caractères larges à partir d’un fichier de ressources avec l’identificateur de ressource donné. |
