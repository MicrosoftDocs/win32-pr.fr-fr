# <a name="cbaselistgetheadpositioni-method"></a>Méthode CBaseList. GetHeadPositionI

La `GetHeadPositionI` méthode récupère la position du premier élément de la liste.

## <a name="syntax"></a>Syntaxe


```C++
POSITION GetHeadPositionI();
```



## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.

## <a name="return-value"></a>Valeur retournée

Retourne une valeur de POSITION, ou **null** si la liste est vide.
