# <a name="loading-and-previewing-a-project"></a>Chargement et affichage de l’aperçu d’un projet

\[Cette API n’est pas prise en charge et peut être modifiée ou non disponible à l’avenir.\]

Cette rubrique explique comment charger et afficher un aperçu d’un fichier projet avec les [services d’édition DirectShow](directshow-editing-services.md) (des). Il présente un exemple d’application qui charge un fichier de projet XML dans une chronologie vide et restitue la chronologie pour l’aperçu. Vous pouvez modifier l’exemple d’application pour qu’il fonctionne avec n’importe quel exemple de fichier de projet XML fourni avec ce kit de développement logiciel (SDK).

Cet article contient les rubriques suivantes.

-   [Chargement d’un fichier projet](loading-a-project-file.md)
-   [Aperçu du projet](previewing-the-project.md)
-   [Aperçu d’un projet : exemple de code](previewing-a-project--example-code.md)
