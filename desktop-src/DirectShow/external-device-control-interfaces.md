# <a name="external-device-control-interfaces"></a>Interfaces de contrôle de périphérique externe

Ces interfaces prennent en charge le contrôle des applications sur les périphériques externes, tels que les caméscopes DV.



| Interface                                            | Description                                                                             |
|------------------------------------------------------|-----------------------------------------------------------------------------------------|
| [**IAMCameraControl**](/windows/desktop/api/Strmif/nn-strmif-iamcameracontrol)         | Contrôler une caméra.                                                                       |
| [**IAMExtDevice**](/windows/desktop/api/Strmif/nn-strmif-iamextdevice)                 | Contrôler un appareil externe tel qu’un magnétoscope ou un caméscope.                                  |
| [**IAMExtTransport**](/windows/desktop/api/Strmif/nn-strmif-iamexttransport)           | Contrôler les fonctions de transport des MAGNÉTOSCOPEs, telles que la lecture, l’interruption, l’enregistrement, l’avance rapide et le rembobinage. |
| [**IAMTimecodeDisplay**](/windows/desktop/api/Strmif/nn-strmif-iamtimecodedisplay)     | Définissez les propriétés sur un appareil qui affiche le code temporel SMPTE.                                |
| [**IAMTimecodeGenerator**](/windows/desktop/api/Strmif/nn-strmif-iamtimecodegenerator) | Définissez les propriétés sur un appareil qui génère le code temporel SMPTE.                               |
| [**IAMTimecodeReader**](/windows/desktop/api/Strmif/nn-strmif-iamtimecodereader)       | Lire le code temporel SMPTE à partir d’un périphérique externe.                                            |
