# <a name="avencddlorosurroundmixlvl_x10-property"></a>AVEncDDLoRoSurroundMixLvl \_ x10, propriété

Spécifie le décalage de niveau appliqué aux canaux Surround pour les downmixing Lo/ro. Cette propriété s’applique aux encodeurs audio Dolby Digital.

Cette propriété est en lecture/écriture.

## <a name="data-type"></a>Type de données

**UInt32** (**VT \_ UI4**)

## <a name="property-guid"></a>Guid de propriété

**CODECAPI \_ AVEncDDLoRoSurroundMixLvl \_ x10**

## <a name="property-value"></a>Valeur de la propriété

La valeur est spécifiée comme décibels (dB) x 10.
