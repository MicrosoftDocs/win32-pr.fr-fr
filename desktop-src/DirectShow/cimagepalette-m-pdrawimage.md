# <a name="cimagepalettem_pdrawimage-member"></a>CImagePalette :: m \_ pDrawImage, membre

La `m_pDrawImage` variable membre est un pointeur vers l’objet [**CDrawImage**](cdrawimage.md) qui dessine l’image vidéo.

## <a name="syntax"></a>Syntaxe


```C++
CDrawImage *m_pDrawImage;
```
