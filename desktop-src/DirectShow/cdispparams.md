# <a name="cdispparams-class"></a>CDispParams, classe

![hiérarchie de la classe cdispparams](images/cutil11.png)

La `CDispParams` classe implémente la structure **DISPPARAMS** utilisée dans Automation en tant que classe de base C++. La méthode **IDispatch :: Invoke** utilise la structure OLE **DISPPARAMS** pour contenir les arguments passés à toute méthode ou propriété.
