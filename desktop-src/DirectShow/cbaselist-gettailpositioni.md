# <a name="cbaselistgettailpositioni-method"></a>Méthode CBaseList. GetTailPositionI

La `GetTailPositionI` méthode récupère la position du dernier élément de la liste.

## <a name="syntax"></a>Syntaxe


```C++
POSITION GetTailPositionI();
```



## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.

## <a name="return-value"></a>Valeur retournée

Retourne une valeur de POSITION, ou **null** si la liste est vide.
