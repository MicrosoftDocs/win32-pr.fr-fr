# <a name="getdvdtextnumberoflanguages-method"></a>Méthode GetDVDTextNumberOfLanguages

> [!Note]  
> ce composant peut être utilisé dans les systèmes d’exploitation Microsoft Windows 2000, Windows XP et Windows Server 2003. Il sera peut-être modifié ou indisponible dans les versions ultérieures.

 

La `GetDVDTextNumberOfLanguages` méthode récupère le nombre de langues de texte disponibles sur le disque.

``` syntax
[ iLanguages = ] MSWebDVD.GetDVDTextNumberOfLanguages()
```

## <a name="return-value"></a>Valeur renvoyée

Retourne une valeur entière spécifiant le nombre de langues de texte disponibles.
