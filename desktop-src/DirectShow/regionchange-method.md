# <a name="regionchange-method"></a>Méthode RegionChange

> [!Note]  
> ce composant peut être utilisé dans les systèmes d’exploitation Microsoft Windows 2000, Windows XP et Windows Server 2003. Il sera peut-être modifié ou indisponible dans les versions ultérieures.

 

La `RegionChange` méthode affiche une boîte de dialogue système qui permet à l’utilisateur de modifier la région associée au lecteur de DVD.

``` syntax
MSWebDVD.RegionChange()
```

## <a name="return-value"></a>Valeur renvoyée

Pas de valeur de retour.

## <a name="remarks"></a>Remarques

La région d’un lecteur de DVD ne peut être modifiée que cinq fois.
