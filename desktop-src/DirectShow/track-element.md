# <a name="track-element"></a>Élément Track

> [!Note]  
> \[Action déconseillée. Cette API peut être supprimée dans les versions futures de Windows.\]

 

L' `track` élément définit un objet Track.

## <a name="attributes"></a>Attributs

[**Lock**](lock-attribute.md), [**muet**](mute-attribute.md), [**UserData**](userdata-attribute.md), [**userid**](userid-attribute.md), [**username**](username-attribute.md)

| Children | [**clip**](clip-element.md), [**effet**](effect-element.md), [**transition**](transition-element.md) |
