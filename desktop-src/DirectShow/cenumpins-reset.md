# <a name="cenumpinsreset-method"></a>CEnumPins. Reset, méthode

La `Reset` méthode réinitialise la séquence d’énumération au début. Cette méthode implémente la méthode [**IEnumPins :: Reset**](/windows/desktop/api/Strmif/nf-strmif-ienumpins-reset) .

## <a name="syntax"></a>Syntaxe


```C++
HRESULT Reset();
```



## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.

## <a name="return-value"></a>Valeur de retour

Retourne S \_ OK.
