# <a name="value-attribute"></a>Attribut de valeur

> [!Note]  
> \[Déconseillé. Cette API peut être supprimée des futures versions de Windows.\]

 

L' `value` attribut spécifie la valeur d’un paramètre.

## <a name="possible-values"></a>Valeurs possibles

Les valeurs possibles dépendent du paramètre.

## <a name="applies-to"></a>S'applique à

[**at**](at-element.md), [**linéaire**](linear-element.md), [**param**](param-element.md)
