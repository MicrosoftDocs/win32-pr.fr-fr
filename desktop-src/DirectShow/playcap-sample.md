# <a name="playcap-sample"></a>Exemple PlayCap

## <a name="description"></a>Description

Application de capture simple.

Cette application crée une fenêtre d’aperçu pour le premier périphérique de capture vidéo qu’il localise sur le système de l’utilisateur (le cas échéant). Il montre un exemple simple d’utilisation des interfaces [**ICaptureGraphBuilder2**](/windows/desktop/api/Strmif/nn-strmif-icapturegraphbuilder2) et [**ICreateDevEnum**](/windows/desktop/api/Strmif/nn-strmif-icreatedevenum) pour créer un graphique de capture.

Pour obtenir un exemple d’application de capture qui prend en charge davantage de fonctionnalités, consultez [exemple AMCAP](amcap-sample.md).
