# <a name="currenttime-property"></a>Propriété CurrentTime

> [!Note]  
> Ce composant peut être utilisé dans les systèmes d’exploitation Microsoft Windows 2000, Windows XP et Windows Server 2003. Il sera peut-être modifié ou indisponible dans les versions ultérieures.

 

La `CurrentTime` propriété récupère l’heure de lecture actuelle.

``` syntax
[ sCurTime = ] MSWebDVD.CurrentTime
```

## <a name="return-value"></a>Valeur renvoyée

Retourne une **chaîne** de 11 caractères représentant l’heure de lecture actuelle au format «*hh : mm : SS : FF*» (heures, minutes, secondes, frames).

## <a name="remarks"></a>Notes

Cette propriété est en lecture seule sans valeur par défaut.
