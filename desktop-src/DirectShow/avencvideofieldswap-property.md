# <a name="avencvideofieldswap-property"></a>Propriété AVEncVideoFieldSwap

Inverse l’ordre des champs entrelacés dans la vidéo source.

Cette propriété est en lecture/écriture.

## <a name="data-type"></a>Type de données

**Variante \_ BOOL** (**VT \_ bool**)

## <a name="property-guid"></a>Guid de propriété

**CODECAPI \_ AVEncVideoFieldSwap**

## <a name="remarks"></a>Notes

Si la valeur est **\_ true**, l’encodeur permute l’ordre des champs.
