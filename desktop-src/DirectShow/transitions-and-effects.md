# <a name="transitions-and-effects"></a>Transitions et effets

> [!Note]  
> \[Action déconseillée. Cette API peut être supprimée dans les versions futures de Windows.\]

 

Cette section décrit les effets et les transitions fournis avec les services d’édition DirectShow.



| Transition                              | Description                                                                            |
|-----------------------------------------|----------------------------------------------------------------------------------------|
| [Compositor](compositor-transition.md) | Étire le premier plan dans un rectangle désigné, sans modifier l’arrière-plan. |
| [Clé](key-transition.md)               | Effectue une incrustation basée sur la chrominance, l’alpha, la teinte ou la luminance.                             |
| [Réinitialisation SMPTE](smpte-wipe-transition.md) | Effectue les réinitialisations SMPTE standard.                                                     |



 



| Effet                                        | Description                           |
|-----------------------------------------------|---------------------------------------|
| [Setter alpha](alpha-setter-effect.md)       | Définit les bits alpha sur une image vidéo. |
| [Enveloppe du volume](volume-envelope-effect.md) | Définit le volume sur un flux audio.   |



 



| Objet source                                | Description                                        |
|----------------------------------------------|----------------------------------------------------|
| [Source de couleur vidéo](video-color-source.md) | Crée une image vidéo continue d’une couleur unie. |
