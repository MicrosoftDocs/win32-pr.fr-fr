# <a name="cbasepropertypagem_titleid-member"></a>CBasePropertyPage :: m \_ TitleID, membre

La variable membre **m \_ pPageSite** spécifie un identificateur de ressource pour une chaîne qui contient le titre de la boîte de dialogue. Cette variable est initialisée dans la méthode du constructeur.

## <a name="syntax"></a>Syntaxe


```C++
int m_TitleId;
```
