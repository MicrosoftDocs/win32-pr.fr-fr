# <a name="idxtcompositorput_srcheight-method"></a>IDxtCompositor ::p ut \_ SrcHeight, méthode

> [!Note]  
> \[Action déconseillée. Cette API peut être supprimée dans les versions futures de Windows.\]

 

La `put_SrcHeight` méthode spécifie la hauteur du rectangle source.

## <a name="syntax"></a>Syntaxe


```C++
HRESULT put_SrcHeight(
  [in] long newVal
);
```



## <a name="parameters"></a>Paramètres

<dl> <dt>

*newVal* \[ dans\]
</dt> <dd>

Hauteur, en pixels, du rectangle source.

</dd> </dl>

## <a name="return-value"></a>Valeur retournée

Si cette méthode est réussie, elle retourne la valeur **\_ OK**. Sinon, elle retourne un code d’erreur **HRESULT** .
