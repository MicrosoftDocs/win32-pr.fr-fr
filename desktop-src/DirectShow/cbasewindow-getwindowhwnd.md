# <a name="cbasewindowgetwindowhwnd-method"></a>Méthode CBaseWindow. GetWindowHWND

La `GetWindowHWND` méthode récupère un handle vers la fenêtre.

## <a name="syntax"></a>Syntaxe


```C++
HWND GetWindowHWND();
```



## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.

## <a name="return-value"></a>Valeur retournée

Retourne un handle vers la fenêtre.
