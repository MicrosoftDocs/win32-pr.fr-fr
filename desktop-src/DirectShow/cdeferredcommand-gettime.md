# <a name="cdeferredcommandgettime-method"></a>CDeferredCommand. GetTime, méthode

La `GetTime` méthode récupère l’heure à laquelle la méthode sera exécutée.

## <a name="syntax"></a>Syntaxe


```C++
CRefTime GetTime();
```



## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.

## <a name="return-value"></a>Valeur retournée

Retourne un objet [**CRefTime**](creftime.md) contenant une heure de référence.
