# <a name="dmo-demo-sample"></a>DMO Exemple de démonstration

## <a name="description"></a>Description

cet exemple d’application montre comment utiliser [DirectX Media objects](directx-media-objects.md) (DMO) dans une application. il diffuse en continu des données audio à partir d’un fichier WAV à l’aide d’un effet audio DMO vers une mémoire tampon DirectSound.

## <a name="usage"></a>Usage

Cliquez sur le bouton **fichier son** pour charger un fichier audio. Utilisez la zone de liste déroulante déroulante pour sélectionner un effet audio DMO. Cliquez sur le bouton **lecture** pour écouter le fichier audio lu avec l’effet audio.

## <a name="downloading-the-sample"></a>Téléchargement de l’exemple

pour télécharger les exemples du kit de développement logiciel (SDK) DirectShow, installez la dernière version du [SDK Windows](https://msdn.microsoft.com/windowsvista/bb980924.aspx).

cet exemple est installé sous le chemin d’accès suivant : exemples de fichiers multimédias *\[ racine \] SDK* \\ \\ \\ DirectShow \\ DMO \\ DMODemo.
