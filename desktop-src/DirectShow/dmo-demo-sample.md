# <a name="dmo-demo-sample"></a>Exemple de démonstration DMO

## <a name="description"></a>Description

Cet exemple d’application montre comment utiliser [DirectX Media Objects](directx-media-objects.md) (DMO) dans une application. Il diffuse en continu des données audio à partir d’un fichier WAV via un effet audio DMO dans une mémoire tampon DirectSound.

## <a name="usage"></a>Utilisation

Cliquez sur le bouton **fichier son** pour charger un fichier audio. Utilisez la zone de liste déroulante déroulante pour sélectionner un effet audio DMO. Cliquez sur le bouton **lecture** pour écouter le fichier audio lu avec l’effet audio.

## <a name="downloading-the-sample"></a>Téléchargement de l’exemple

Pour télécharger les exemples du kit de développement logiciel (SDK) DirectShow, installez la dernière version du [SDK Windows](https://msdn.microsoft.com/windowsvista/bb980924.aspx).

Cet exemple est installé sous le chemin d’accès suivant : exemples *\[ racine \] du kit de développement logiciel (SDK)* \\ \\ \\ DirectShow \\ DMO \\ DMODemo.
