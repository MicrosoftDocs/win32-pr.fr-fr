# <a name="parameter-capabilities-flags"></a>Indicateurs des fonctionnalités des paramètres

Ces indicateurs spécifient les courbes d’enveloppe prises en charge par un paramètre Media.



| Constante                                                                                                                                                                                      | Description                                              |
|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:---------------------------------------------------------|
| <span id="MP_CAPS_CURVE_INVSQUARE"></span><span id="mp_caps_curve_invsquare"></span><dl> <dt>**\_INVSQUARE de \_ courbe d’extrémité MP \_**</dt> </dl> | Courbe carrée inverse.<br/>                         |
| <span id="MP_CAPS_CURVE_JUMP"></span><span id="mp_caps_curve_jump"></span><dl> <dt>**\_saut de courbe en majuscules MP \_ \_**</dt> </dl>                | Accéder au point suivant sans interpolation.<br/> |
| <span id="MP_CAPS_CURVE_LINEAR"></span><span id="mp_caps_curve_linear"></span><dl> <dt>**\_courbe d’extrémité MP \_ \_ linéaire**</dt> </dl>          | Interpolation linéaire.<br/>                         |
| <span id="MP_CAPS_CURVE_SINE"></span><span id="mp_caps_curve_sine"></span><dl> <dt>**point de terminaison de la \_ \_ courbe MP \_**</dt> </dl>                | Courbe sinusoïdale.<br/>                                   |
| <span id="MP_CAPS_CURVE_SQUARE"></span><span id="mp_caps_curve_square"></span><dl> <dt>**\_carré de \_ courbe d’extrémité MP \_**</dt> </dl>          | Courbe parabolique.<br/>                              |
