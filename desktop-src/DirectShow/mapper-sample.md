# <a name="mapper-sample"></a>Exemple de Mappeur

## <a name="description"></a>Description

Cet exemple illustre l’utilisation de la méthode [**IFilterMapper2 :: EnumMatchingFilters**](/windows/desktop/api/Strmif/nf-strmif-ifiltermapper2-enummatchingfilters) pour localiser des filtres dans le registre.

L’interface utilisateur de l’exemple d’application contient des contrôles qui correspondent aux paramètres de la méthode [**EnumMatchingFilters**](/windows/desktop/api/Strmif/nf-strmif-ifiltermapper2-enummatchingfilters) , tels que la valeur mérite minimale, les catégories pin et les types de média. Sélectionnez les options de recherche souhaitées, puis cliquez sur le bouton **Rechercher** pour énumérer les filtres qui correspondent à ces options.

## <a name="downloading-the-sample"></a>Téléchargement de l’exemple

Pour télécharger les exemples du kit de développement logiciel (SDK) DirectShow, installez la dernière version du [SDK Windows](https://msdn.microsoft.com/windowsvista/bb980924.aspx).

Cet exemple est installé sous le chemin d’accès suivant : \[ exemples racine SDK \] \\ exemple de \\ \\ \\ \\ Mappeur divers DirectShow.
