# <a name="windowless-sample"></a>Exemple sans fenêtre

## <a name="description"></a>Description

Cet exemple illustre le mode sans fenêtre dans le filtre de mixage vidéo 9 (VMR-9).

Vous pouvez capturer l’image à partir de la fenêtre vidéo en cliquant avec le bouton droit sur l’image, ou en sélectionnant **capturer le frame actuel** dans le menu **image** .

## <a name="downloading-the-sample"></a>Téléchargement de l’exemple

pour télécharger les exemples du kit de développement logiciel (SDK) DirectShow, installez la dernière version du [SDK Windows](https://msdn.microsoft.com/windowsvista/bb980924.aspx).
