# <a name="video-mixing-renderer-video-subtypes"></a>Sous-types vidéo du convertisseur de mixage vidéo

Les sous-types suivants sont définis pour être utilisés avec le filtre de convertisseur de mixage vidéo (VMR).

## <a name="vmr-7-subtypes"></a>Sous-types VMR-7



| Subtype                              | Description                                      |
|--------------------------------------|--------------------------------------------------|
| MEDIASUBTYPE \_ RGB32 \_ D3D \_ DX7 \_ RT    | cible de rendu RGB 32 bits.                        |
| MEDIASUBTYPE \_ RGB16 \_ D3D \_ DX7 \_ RT    | cible de rendu RGB 16 bits.                        |
| MEDIASUBTYPE \_ ARGB32 \_ D3D \_ DX7 \_ RT   | cible de rendu ARVB 32 bits.                       |
| MEDIASUBTYPE \_ ARGB4444 \_ D3D \_ DX7 \_ RT | Cible de rendu ARGB4444. Pour les graphiques de sous-image. |
| MEDIASUBTYPE \_ ARGB1555 \_ D3D \_ DX7 \_ RT | Cible de rendu ARGB1555. Pour les graphiques de sous-image. |



 

## <a name="vmr-9-subtypes"></a>Sous-types VMR-9



| Subtype                              | Description                                      |

| MEDIASUBTYPE \_ RGB16 \_ D3D \_ virtuel DX9 \_ RT    | cible de rendu RGB 16 bits.                        |
| MEDIASUBTYPE \_ ARGB32 \_ D3D \_ virtuel DX9 \_ RT   | cible de rendu ARVB 32 bits.                       |
| MEDIASUBTYPE \_ ARGB4444 \_ D3D \_ virtuel DX9 \_ RT | Cible de rendu ARGB4444. Pour les graphiques de sous-image. |
