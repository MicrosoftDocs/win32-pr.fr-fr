# <a name="functions-programming-reference"></a>Fonctions (référence de programmation)

Fonctions XInput disponibles.

## <a name="in-this-section"></a>Contenu de cette section

| Rubrique                                                                                 | Description                                                                                                                                             |
|---------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XInputEnable**](/windows/desktop/api/XInput/nf-xinput-xinputenable)<br/>                                       | Définit l’état de création de rapports de XInput.<br/>                                                                                                          |
| [**XInputGetAudioDeviceIds**](/windows/desktop/api/XInput/nf-xinput-xinputgetaudiodeviceids)<br/>                 | Récupère les ID de périphérique audio de rendu sonore et de capture audio associés au casque connecté au contrôleur spécifié.<br/> |
| [**XInputGetBatteryInformation**](/windows/desktop/api/XInput/nf-xinput-xinputgetbatteryinformation)<br/>         | Récupère le type de batterie et le statut de charge d’un contrôleur sans fil.<br/>                                                                       |
| [**XInputGetCapabilities**](/windows/desktop/api/XInput/nf-xinput-xinputgetcapabilities)<br/>                     | Récupère les fonctionnalités d’un contrôleur connecté.<br/>                                                                           |
| [**XInputGetDSoundAudioDeviceGuids**](/windows/desktop/api/XInput/nf-xinput-xinputgetdsoundaudiodeviceguids)<br/> | Obtient les GUID de rendu du son et de capture audio associés au casque connecté au contrôleur spécifié.<br/>          |
| [**XInputGetKeystroke**](/windows/desktop/api/XInput/nf-xinput-xinputgetkeystroke)<br/>                           | Récupère un événement d’entrée de manette.<br/>                                                                                                             |
| [**XInputGetState**](/windows/desktop/api/XInput/nf-xinput-xinputgetstate)<br/>                                   | Récupère l’état actuel du contrôleur spécifié.<br/>                                                                                     |
| [**XInputSetState**](/windows/desktop/api/XInput/nf-xinput-xinputsetstate)<br/>                                   | Envoie des données à un contrôleur connecté. Cette fonction est utilisée pour activer la fonction vibratoire d’un contrôleur.<br/>                              |
