# <a name="xinput-game-controller-apis"></a>API du contrôleur de jeu XInput

## <a name="purpose"></a>Objectif

L’API du contrôleur de jeu XInput permet aux applications de recevoir l’entrée du contrôleur Xbox pour Windows.

| Rubrique                                                         | Description                                                                                                                                             |

| [Guide de programmation](programming-guide.md)<br/>         | Ce guide contient des informations sur l’utilisation de l’API XInput pour interagir avec le contrôleur Xbox lorsqu’il est connecté à un PC Windows. <br/> |
| [Guide de référence de programmation](programming-reference.md)<br/> | Structures et fonctions XInput.<br/>                                                                                                             |

## <a name="developer-audience"></a>Développeurs concernés

Les API du contrôleur de jeu XInput sont conçues pour être utilisées par les développeurs qui souhaitent utiliser le contrôleur Xbox pour leurs applications Windows.
