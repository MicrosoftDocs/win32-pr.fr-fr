# <a name="flush-method-of-the-control-class"></a>Méthode Flush de la classe Control

Efface les mémoires tampons de transfert pour le flux et écrit toutes les données mises en mémoire tampon dans le fichier de configuration.

## <a name="syntax"></a>Syntaxe


```mof
void Flush();
```



## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.

## <a name="return-value"></a>Valeur retournée

Cette méthode ne retourne pas de valeur.
