# <a name="iwebproxy-methods"></a>Méthodes IWebProxy

L’interface [**IWebProxy**](/windows/desktop/api/Wuapi/nn-wuapi-iwebproxy) définit les méthodes suivantes.



| Méthode                                                                         | Description                                                         |
|--------------------------------------------------------------------------------|---------------------------------------------------------------------|
| [**PromptForCredentials**](/windows/desktop/api/Wuapi/nf-wuapi-iwebproxy-promptforcredentials)                 | Invite l’utilisateur à entrer le mot de passe à utiliser pour l’authentification du proxy.  |
| [**PromptForCredentialsFromHwnd**](/windows/desktop/api/Wuapi/nf-wuapi-iwebproxy-promptforcredentialsfromhwnd) | Invite l’utilisateur à entrer le mot de passe à utiliser pour l’authentification du proxy.  |
| [**Attribution**](/windows/desktop/api/Wuapi/nf-wuapi-iwebproxy-setpassword)                                   | Définit le mot de passe à envoyer au serveur proxy pour l’authentification. |
