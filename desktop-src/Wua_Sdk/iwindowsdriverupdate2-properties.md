# <a name="iwindowsdriverupdate2-properties"></a>Propriétés IWindowsDriverUpdate2

L’interface [**IWindowsDriverUpdate2**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate) définit les propriétés suivantes.



| Propriété                                                       | Description                                                                                                             |
|----------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------|
| [**CveIDs**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate2-get_cveids)                 | Contient une collection des identificateurs de vulnérabilité et d’exposition courants (CVE) associés à une mise à jour. |
| [**IsPresent**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate2-get_ispresent)           | Obtient une valeur booléenne qui indique si une mise à jour est installée sur l’ordinateur.                                     |
| [**RebootRequired**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate2-get_rebootrequired) | Obtient une valeur booléenne qui indique si l’ordinateur doit être redémarré après l’installation ou la désinstallation d’une mise à jour.    |



 

Pour plus d’informations sur les membres hérités par cette interface, consultez l’interface suivante.

-   [**IWindowsDriverUpdate**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate)
