# <a name="iupdatedownloadresult-properties"></a>Propriétés IUpdateDownloadResult

L’interface [**IUpdateDownloadResult**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatedownloadresult) définit les propriétés suivantes.



| Propriété                                               | Description                                                                                                                          |
|--------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------|
| [**Signé**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatedownloadresult-get_hresult)       | Obtient la valeur **HRESULT** d’exception, le cas échéant, levée pendant une opération sur une mise à jour.                                       |
| [**ResultCode**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatedownloadresult-get_resultcode) | Obtient une valeur d’énumération [**OperationResultCode**](/windows/win32/api/wuapi/ne-wuapi-operationresultcode) qui spécifie le résultat d’une opération sur une mise à jour. |
