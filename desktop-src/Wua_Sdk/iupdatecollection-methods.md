# <a name="iupdatecollection-methods"></a>Méthodes IUpdateCollection

L’interface [**IUpdateCollection**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatecollection) définit les méthodes suivantes.



| Méthode                                         | Description                                                    |
|------------------------------------------------|----------------------------------------------------------------|
| [**Complémentaires**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatecollection-add)           | Ajoute un élément à la collection.                                |
| [**Effacé**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatecollection-clear)       | Supprime tous les éléments de la collection.                  |
| [**Copier**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatecollection-copy)         | Crée une copie en lecture/écriture superficielle de la collection.           |
| [**Insérer**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatecollection-insert)     | Insère un élément dans la collection à la position spécifiée. |
| [**RemoveAt**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatecollection-removeat) | Supprime de la collection l'élément à l'index spécifié.   |
