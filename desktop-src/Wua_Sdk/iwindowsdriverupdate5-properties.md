# <a name="iwindowsdriverupdate5-properties"></a>Propriétés IWindowsDriverUpdate5

L’interface [**IWindowsDriverUpdate5**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate5) définit les propriétés suivantes.



| Propriété                                                     | Description                                                                                                                                               |
|--------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Téléchargement AutoDownload**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate5-get_autodownload)   | Obtient une valeur [**AutoDownloadMode**](/windows/win32/api/wuapi/ne-wuapi-autodownloadmode) qui indique le mode de téléchargement automatique de la mise à jour.                                          |
| [**Sélection sélectionnée**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate5-get_autoselection) | Obtient une valeur [**AutoSelectionMode**](/windows/win32/api/wuapi/ne-wuapi-autoselectionmode) indiquant le mode de sélection automatique d’une mise à jour dans le panneau de configuration de Windows Update. |



 

Pour plus d’informations sur les membres hérités par cette interface, consultez les interfaces suivantes.

-   [**IWindowsDriverUpdate**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate)
-   [**IWindowsDriverUpdate2**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate2)
-   [**IWindowsDriverUpdate3**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate3)
-   [**IWindowsDriverUpdate4**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate4)
