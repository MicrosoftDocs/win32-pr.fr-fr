# <a name="iwindowsdriverupdateentry-properties"></a>Propriétés IWindowsDriverUpdateEntry

L’interface [**IWindowsDriverUpdateEntry**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdateentry) définit les propriétés suivantes.



| Propriété                                                                     | Description                                                                                                          |
|------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| [**DeviceProblemNumber**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdateentry-get_deviceproblemnumber) | Obtient le numéro de problème de l’appareil correspondant à la mise à jour du pilote Windows.                                    |
| [**DeviceStatus**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdateentry-get_devicestatus)               | Obtient l’état de l’appareil correspondant à la mise à jour du pilote Windows.                                            |
| [**DriverClass**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdateentry-get_driverclass)                 | Obtient la classe de la mise à jour du pilote Windows.                                                                         |
| [**DriverHardwareID**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdateentry-get_driverhardwareid)       | Obtient le matériel ou l’identificateur compatible que la mise à jour du pilote Windows doit mettre en correspondance pour pouvoir être installée. |
| [**DriverManufacturer**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdateentry-get_drivermanufacturer)   | Obtient le nom invariant du langage du fabricant de la mise à jour du pilote Windows.                                   |
| [**DriverModel**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdateentry-get_drivermodel)                 | Obtient le nom de modèle de l’appareil pour lequel la mise à jour du pilote Windows est prévue.                |
| [**DriverProvider**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdateentry-get_driverprovider)           | Obtient le nom invariant du langage du fournisseur de la mise à jour du pilote Windows.                                       |
| [**DriverVerDate**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdateentry-get_driververdate)             | Obtient la date de version du pilote de la mise à jour du pilote Windows.                                                           |
