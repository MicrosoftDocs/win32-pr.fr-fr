# <a name="iupdatedownloadcontent-properties"></a>Propriétés IUpdateDownloadContent

L’interface [**IUpdateDownloadContent**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatecollection) définit la propriété suivante.



| Propriété                                                  | Description                                                                    |
|-----------------------------------------------------------|--------------------------------------------------------------------------------|
| [**DownloadUrl**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatedownloadcontent-get_downloadurl) | Obtient l’emplacement du contenu de téléchargement sur le serveur qui héberge la mise à jour. |
