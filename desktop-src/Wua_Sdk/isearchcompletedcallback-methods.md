# <a name="isearchcompletedcallback-methods"></a>Méthodes ISearchCompletedCallback

L’interface [**ISearchCompletedCallback**](/windows/desktop/api/Wuapi/nn-wuapi-isearchcompletedcallback) définit la méthode suivante.



| Méthode                                            | Description                                                                                                                                                                     |
|---------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Appeler**](/windows/desktop/api/Wuapi/nf-wuapi-isearchcompletedcallback-invoke) | Gère la notification de la fin d’une recherche asynchrone lancée en appelant la méthode [**IUpdateSearcher. BeginSearch**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatesearcher-beginsearch) . |
