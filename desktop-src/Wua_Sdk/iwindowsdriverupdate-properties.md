# <a name="iwindowsdriverupdate-properties"></a>Propriétés IWindowsDriverUpdate

L’interface [**IWindowsDriverUpdate**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate) définit les propriétés suivantes.



| Propriété                                                                                                    | Description                                                                                           |
|-------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------|
| [](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_deviceproblemnumber)[**Adresse** DeviceProblemNumber](/windows/desktop/api/Wuapi/nf-wuapi-iwebproxy-get_address) | Obtient le numéro de problème de l’appareil correspondant à la mise à jour du pilote Windows.                     |
| [**DeviceStatus**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_devicestatus)                                                   | Obtient l’état de l’appareil correspondant à la mise à jour du pilote Windows.                             |
| [**DriverClass**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_driverclass)                                                     | Obtient la classe de la mise à jour du pilote Windows.                                                          |
| [**DriverHardwareID**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_driverhardwareid)                                           | Obtient l’ID matériel ou l’ID compatible que la mise à jour du pilote Windows doit mettre en correspondance pour pouvoir être installée.    |
| [**DriverManufacturer**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_drivermanufacturer)                                       | Obtient le nom invariant du langage du fabricant de la mise à jour du pilote Windows.                    |
| [**DriverModel**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_drivermodel)                                                     | Obtient le nom de modèle de l’appareil pour lequel la mise à jour du pilote Windows est prévue. |
| [**DriverProvider**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_driverprovider)                                               | Obtient le nom invariant du langage du fournisseur de la mise à jour du pilote Windows.                        |
| [**DriverVerDate**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_driververdate)                                                 | Obtient la date de version du pilote de la mise à jour du pilote Windows.                                            |
