# <a name="iautomaticupdates-methods"></a>Méthodes IAutomaticUpdates

L’interface [**IAutomaticUpdates**](/windows/desktop/api/Wuapi/nn-wuapi-iautomaticupdates) définit les méthodes suivantes.



| Méthode                                                             | Description                                                         |
|--------------------------------------------------------------------|---------------------------------------------------------------------|
| [**DetectNow**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdates-detectnow)                   | Démarre la mise à jour automatique si elle n’a pas encore démarré.            |
| [**EnableService**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdates-enableservice)           | Active tous les composants requis par Mises à jour automatiques.         |
| [**Suspendre**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdates-pause)                           | Suspend la mise à jour automatique.                                          |
| [**Sort**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdates-resume)                         | Redémarre la mise à jour automatique si elle est suspendue.                        |
| [**ShowSettingsDialog**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdates-showsettingsdialog) | Affiche une boîte de dialogue qui contient les paramètres de Mises à jour automatiques. |
