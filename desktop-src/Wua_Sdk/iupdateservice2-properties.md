# <a name="iupdateservice2-properties"></a>Propriétés IUpdateService2

L’interface **IUpdateService2** définit la propriété suivante.



| Propriété                                                         | Description                                                                                                                                                                         |
|------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IsDefaultAUService**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateservice2-get_isdefaultauservice) | Obtient une valeur booléenne qui indique si le service est inscrit avec Mises à jour automatiques et si le service est actuellement utilisé par Mises à jour automatiques comme service par défaut. |



 

Pour plus d’informations sur les membres hérités par cette interface, consultez l’interface suivante.

-   [**IUpdateService**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateservice)
