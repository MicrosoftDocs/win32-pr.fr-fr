# <a name="iinstallationcompletedcallback-methods"></a>Méthodes IInstallationCompletedCallback

L’interface [**IInstallationCompletedCallback**](/windows/desktop/api/Wuapi/nn-wuapi-iinstallationcompletedcallback) définit la méthode suivante.



| Méthode                                                  | Description                                                                                                                                                                                                                                                                       |
|---------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Appeler**](/windows/desktop/api/Wuapi/nf-wuapi-iinstallationcompletedcallback-invoke) | Gère la notification de la fin d’une installation ou d’une désinstallation asynchrone qui est lancée par un appel à [**IUpdateInstaller. BeginInstall**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateinstaller-begininstall) ou [**IUpdateInstaller. BeginUninstall**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateinstaller-beginuninstall). |
