# <a name="istringcollection-methods"></a>Méthodes IStringCollection

L’interface [**IStringCollection**](/windows/desktop/api/Wuapi/nn-wuapi-istringcollection) définit les méthodes suivantes.



| Méthode                                         | Description                                                    |
|------------------------------------------------|----------------------------------------------------------------|
| [**Complémentaires**](/windows/desktop/api/Wuapi/nf-wuapi-istringcollection-add)           | Ajoute un élément à la collection.                                |
| [**Effacé**](/windows/desktop/api/Wuapi/nf-wuapi-istringcollection-clear)       | Supprime tous les éléments de la collection.                  |
| [**Copier**](/windows/desktop/api/Wuapi/nf-wuapi-istringcollection-copy)         | Crée une copie en lecture/écriture profonde de la collection.              |
| [**Insérer**](/windows/desktop/api/Wuapi/nf-wuapi-istringcollection-insert)     | Insère un élément dans la collection à la position spécifiée. |
| [**RemoveAt**](/windows/desktop/api/Wuapi/nf-wuapi-istringcollection-removeat) | Supprime de la collection l'élément à l'index spécifié.   |
