# <a name="isearchresult-properties"></a>Propriétés ISearchResult

L’interface [**ISearchResult**](/windows/desktop/api/Wuapi/nn-wuapi-isearchresult) définit les propriétés suivantes.



| Propriété                                               | Description                                                                                                   |
|--------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [**ResultCode**](/windows/desktop/api/Wuapi/nf-wuapi-isearchresult-get_resultcode)         | Obtient une énumération [**OperationResultCode**](/windows/win32/api/wuapi/ne-wuapi-operationresultcode) qui spécifie le résultat d’une recherche. |
| [**RootCategories**](/windows/desktop/api/Wuapi/nf-wuapi-isearchresult-get_rootcategories) | Obtient une collection d’interfaces des catégories racines actuellement disponibles sur l’ordinateur.             |
| [**Mises à jour**](/windows/desktop/api/Wuapi/nf-wuapi-isearchresult-get_updates)               | Obtient une collection d’interfaces des mises à jour qui résultent d’une recherche.                                        |
| [**Avertissements**](/windows/desktop/api/Wuapi/nf-wuapi-isearchresult-get_warnings)             | Obtient une collection des avertissements qui résultent d’une recherche.                                                  |
