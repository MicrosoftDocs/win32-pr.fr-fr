# <a name="iinstallationprogress-methods"></a>Méthodes IInstallationProgress

L’interface [**IInstallationProgress**](/windows/desktop/api/Wuapi/nn-wuapi-iinstallationprogress) définit la méthode suivante.



| Méthode                                                           | Description                                                                     |
|------------------------------------------------------------------|---------------------------------------------------------------------------------|
| [**GetUpdateResult**](/windows/desktop/api/Wuapi/nf-wuapi-iinstallationprogress-getupdateresult) | Retourne le résultat de l’installation ou de la désinstallation d’une mise à jour spécifiée. |
