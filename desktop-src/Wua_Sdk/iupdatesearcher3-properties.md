# <a name="iupdatesearcher3-methods"></a>Méthodes IUpdateSearcher3

L’interface [**IUpdateSearcher3**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatesearcher3) définit la méthode suivante :



| Méthode                                              | Description                                                                                                                                    |
|-----------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SearchScope**](/previous-versions/windows/desktop/ee694850(v=vs.85)) | Obtient et définit une valeur d’énumération [**SearchScope**](/windows/win32/api/wuapi/ne-wuapi-searchscope) qui indique la portée des mises à jour retournées par la recherche. |



 

Pour plus d’informations sur les membres hérités par cette interface, consultez les interfaces suivantes.

-   [**IUpdateSearcher**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatesearcher)
-   [**IUpdateSearcher2**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatesearcher2)
