# <a name="iupdatesession2-properties"></a>Propriétés IUpdateSession2

L’interface [**IUpdateSession2**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatesession2) définit la propriété suivante.



| Propriété                                         | Description                                                                   |
|--------------------------------------------------|-------------------------------------------------------------------------------|
| [**UserLocale**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatesession2-get_userlocale) | Obtient et définit les paramètres régionaux par défaut pour lesquels les informations de mise à jour sont récupérées. |



 

Pour plus d’informations sur les membres hérités par cette interface, consultez l’interface suivante.

-   [**IUpdateSession**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatesession)
