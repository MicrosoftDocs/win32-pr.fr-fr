# <a name="iautomaticupdates-properties"></a>Propriétés IAutomaticUpdates

L’interface [**IAutomaticUpdates**](/windows/desktop/api/Wuapi/nn-wuapi-iautomaticupdates) définit les propriétés suivantes.



| Propriété                                                   | Description                                                                                                   |
|------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [**ServiceEnabled**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdates-get_serviceenabled) | Obtient une valeur booléenne qui indique si tous les composants requis par Mises à jour automatiques sont disponibles. |
| [**Paramètres**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdates-get_settings)             | Obtient les paramètres de configuration pour Mises à jour automatiques.                                                        |
