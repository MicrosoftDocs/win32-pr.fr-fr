# <a name="iinstallationresult-methods"></a>Méthodes IInstallationResult

L’interface [**IInstallationResult**](/windows/desktop/api/Wuapi/nn-wuapi-iinstallationresult) définit la méthode suivante.



| Méthode                                                         | Description                                                                                                                                        |
|----------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------|
| [**GetUpdateResult**](/windows/desktop/api/Wuapi/nf-wuapi-iinstallationresult-getupdateresult) | Retourne une interface [**IUpdateInstallationResult**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateinstallationresult) qui contient les résultats de l’installation d’une mise à jour spécifiée. |
