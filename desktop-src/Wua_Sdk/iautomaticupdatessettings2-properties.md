# <a name="iautomaticupdatessettings2-properties"></a>Propriétés IAutomaticUpdatesSettings2

L’interface [**IAutomaticUpdatesSettings2**](/windows/desktop/api/Wuapi/nn-wuapi-iautomaticupdatessettings2) définit la propriété suivante.



| Propriété                                                                                  | Description                                                                                                                                                         |
|-------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IncludeRecommendedUpdates**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdatessettings2-get_includerecommendedupdates) | Obtient et définit une valeur booléenne qui indique s’il faut inclure les mises à jour facultatives ou recommandées lors de la recherche des mises à jour et de l’installation des mises à jour. |



 

Pour plus d’informations sur les membres hérités par cette interface, consultez l’interface suivante.

-   [**IAutomaticUpdatesSettings**](/windows/desktop/api/Wuapi/nn-wuapi-iautomaticupdatessettings)
