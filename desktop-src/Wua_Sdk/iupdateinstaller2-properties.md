# <a name="iupdateinstaller2-properties"></a>Propriétés IUpdateInstaller2

L’interface [**IUpdateInstaller2**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateinstaller2) définit la propriété suivante.



| Propriété                                           | Description                                                                                                                       |
|----------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------|
| [**ForceQuiet**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateinstaller2-get_forcequiet) | Obtient et définit une valeur booléenne qui indique si Windows Installer est forcé d’installer les mises à jour sans intervention de l’utilisateur. |



 

Pour plus d’informations sur les membres hérités par cette interface, consultez l’interface suivante.

-   [**IUpdateInstaller**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateinstaller)
