# <a name="iupdate2-methods"></a>Méthodes IUpdate2

L’interface [**IUpdate2**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate2) définit la méthode suivante.



| Méthode                                      | Description                                                                                                            |
|---------------------------------------------|------------------------------------------------------------------------------------------------------------------------|
| [**CopyToCache**](/windows/desktop/api/Wuapi/nf-wuapi-iupdate2-copytocache) | Copie les fichiers d’une mise à jour à partir d’un emplacement source spécifié vers le cache de téléchargement de l’agent de Windows Update interne (WUA). |



 

Pour plus d’informations sur les membres hérités par cette interface, consultez l’interface suivante :

-   [**IUpdate**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate)
