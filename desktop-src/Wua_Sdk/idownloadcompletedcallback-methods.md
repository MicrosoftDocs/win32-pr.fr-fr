# <a name="idownloadcompletedcallback-methods"></a>Méthodes IDownloadCompletedCallback

L’interface [**IDownloadCompletedCallback**](/windows/desktop/api/Wuapi/nn-wuapi-idownloadcompletedcallback) définit la méthode suivante.



| Méthode                                              | Description                                        |
|-----------------------------------------------------|----------------------------------------------------|
| [**Appeler**](/windows/desktop/api/Wuapi/nf-wuapi-idownloadcompletedcallback-invoke) | Informe l’appelant que le téléchargement est terminé. |
