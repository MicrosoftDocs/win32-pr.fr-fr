# <a name="iupdatelockdown-methods"></a>Méthodes IUpdateLockdown

L’interface [**IUpdateLockdown**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatelockdown) définit la méthode suivante.



| Méthode                                       | Description                                                                               |
|----------------------------------------------|-------------------------------------------------------------------------------------------|
| [**Lockdown**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatelockdown-lockdown) | Restreint l’accès aux méthodes et aux propriétés de l’objet qui implémente cette méthode. |
