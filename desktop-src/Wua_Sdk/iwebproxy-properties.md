# <a name="iwebproxy-properties"></a>Propriétés IWebProxy

L’interface [**IWebProxy**](/windows/desktop/api/Wuapi/nn-wuapi-iwebproxy) définit les propriétés suivantes.



| Propriété                                                   | Description                                                                                                               |
|------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------|
| [**Adresse**](/windows/desktop/api/Wuapi/nf-wuapi-iwebproxy-get_address)                       | Obtient et définit l’adresse et le numéro de port décimal du serveur proxy.                                                |
| [**Détection automatique**](/windows/desktop/api/Wuapi/nf-wuapi-iwebproxy-get_autodetect)                 | Obtient et définit une valeur booléenne qui indique si l' [**IWebProxy**](/windows/desktop/api/Wuapi/nn-wuapi-iwebproxy) détecte automatiquement les paramètres du proxy. |
| [**BypassList**](/windows/desktop/api/Wuapi/nf-wuapi-iwebproxy-get_bypasslist)                 | Obtient et définit une collection d’adresses qui n’utilisent pas le serveur proxy.                                                 |
| [**BypassProxyOnLocal**](/windows/desktop/api/Wuapi/nf-wuapi-iwebproxy-get_bypassproxyonlocal) | Obtient et définit une valeur booléenne qui indique si les adresses locales ignorent le serveur proxy.                             |
| [**Seulement**](/windows/desktop/api/Wuapi/nf-wuapi-iwebproxy-get_readonly)                     | Obtient une valeur booléenne qui indique si l’objet [**WebProxy**](/windows/desktop/api/Wuapi/nn-wuapi-iwebproxy) est en lecture seule.                        |
| [**Nom d’utilisateur**](/windows/desktop/api/Wuapi/nf-wuapi-iwebproxy-get_username)                     | Obtient et définit le nom d’utilisateur à envoyer au serveur proxy pour l’authentification.                                             |
