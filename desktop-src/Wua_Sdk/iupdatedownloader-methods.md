# <a name="iupdatedownloader-methods"></a>Méthodes IUpdateDownloader

L’interface [**IUpdateDownloader**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatedownloader) définit les méthodes suivantes.



| Méthode                                                   | Description                                                                                |
|----------------------------------------------------------|--------------------------------------------------------------------------------------------|
| [**BeginDownload**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatedownloader-begindownload) | Démarre un téléchargement asynchrone des fichiers de contenu qui sont associés aux mises à jour. |
| [**Télécharger**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatedownloader-download)           | Démarre un téléchargement synchrone des fichiers de contenu qui sont associés aux mises à jour.   |
| [**Car EndDownload**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatedownloader-enddownload)     | Termine un téléchargement asynchrone.                                                        |
