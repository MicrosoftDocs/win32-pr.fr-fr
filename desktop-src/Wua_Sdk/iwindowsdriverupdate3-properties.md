# <a name="iwindowsdriverupdate3-properties"></a>Propriétés IWindowsDriverUpdate3

L’interface **IWindowsDriverUpdate3** définit la propriété suivante.



| Propriété                                               | Description                                                                                                             |
|--------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------|
| [**BrowseOnly**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate3-get_browseonly) | Obtient une valeur booléenne qui indique si une mise à jour peut être détectée uniquement en parcourant les mises à jour disponibles. |



 

Pour plus d’informations sur les membres hérités par cette interface, consultez l’interface suivante.

-   [**IWindowsDriverUpdate**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate)
-   [**IWindowsDriverUpdate2**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate2)
