# <a name="idownloadresult-properties"></a>Propriétés IDownloadResult

L’interface [**IDownloadResult**](/windows/desktop/api/Wuapi/nn-wuapi-idownloadresult) définit les propriétés suivantes.



| Propriété                                         | Description                                                                                                     |
|--------------------------------------------------|-----------------------------------------------------------------------------------------------------------------|
| [**HResult**](/windows/desktop/api/Wuapi/nf-wuapi-idownloadresult-get_hresult)       | Obtient le numéro de code d’exception si un numéro de code d’exception est déclenché pendant un téléchargement.                         |
| [**ResultCode**](/windows/desktop/api/Wuapi/nf-wuapi-idownloadresult-get_resultcode) | Obtient une énumération [**OperationResultCode**](/windows/win32/api/wuapi/ne-wuapi-operationresultcode) qui spécifie le résultat d’un téléchargement. |
