# <a name="iupdatesearcher2-properties"></a>Propriétés IUpdateSearcher2

L’interface [**IUpdateSearcher2**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatesearcher2) définit la propriété suivante.



| Propriété                                                                  | Description                                                                           |
|---------------------------------------------------------------------------|---------------------------------------------------------------------------------------|
| [**IgnoreDownloadPriority**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatesearcher2-get_ignoredownloadpriority) | Obtient et définit une valeur booléenne qui indique s’il faut ignorer la priorité de téléchargement. |



 

Pour plus d’informations sur les membres hérités par cette interface, consultez l’interface suivante.

-   [**IUpdateSearcher**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatesearcher)
