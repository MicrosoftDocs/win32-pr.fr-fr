# <a name="iupdate5-properties"></a>Propriétés IUpdate5

L’interface [**IUpdate5**](/windows/win32/api/wuapi/nn-wuapi-iupdate5) définit les propriétés suivantes.



| Propriété                                        | Description                                                                                            |
|-------------------------------------------------|--------------------------------------------------------------------------------------------------------|
| [**Téléchargement AutoDownload**](/windows/desktop/api/Wuapi/nf-wuapi-iupdate5-get_autodownload)   | Obtient une valeur indiquant le mode de téléchargement automatique de la mise à jour.                                         |
| [**Sélection sélectionnée**](/windows/desktop/api/Wuapi/nf-wuapi-iupdate5-get_autoselection) | Obtient une valeur qui indique le mode de sélection automatique de la mise à jour dans le panneau de configuration de Windows Update. |



 

Pour plus d’informations sur les membres hérités par cette interface, consultez les interfaces suivantes :

-   [**IUpdate4**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate4)
-   [**IUpdate3**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate3)
-   [**IUpdate2**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate2)
-   [**IUpdate**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate)
