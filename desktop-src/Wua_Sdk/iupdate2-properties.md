# <a name="iupdate2-properties"></a>Propriétés IUpdate2

L’interface [**IUpdate2**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate2) définit les propriétés suivantes.



| Propriété                                          | Description                                                                                                                                             |
|---------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**CveIDs**](/windows/desktop/api/Wuapi/nf-wuapi-iupdate2-get_cveids)                 | Obtient une collection d’ID de vulnérabilités et d’expositions courants (CVE) associés à la mise à jour.                                                |
| [**IsPresent**](/windows/desktop/api/Wuapi/nf-wuapi-iupdate2-get_ispresent)           | Obtient une valeur booléenne qui indique si une mise à jour est présente sur un ordinateur.                                                                         |
| [**RebootRequired**](/windows/desktop/api/Wuapi/nf-wuapi-iupdate2-get_rebootrequired) | Obtient une valeur booléenne qui indique si un redémarrage système est requis sur un ordinateur pour terminer l’installation ou la désinstallation d’une mise à jour. |



 

Pour plus d’informations sur les membres hérités par cette interface, consultez l’interface suivante :

-   [**IUpdate**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate)
