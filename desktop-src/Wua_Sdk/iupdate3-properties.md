# <a name="iupdate3-properties"></a>Propriétés IUpdate3

L’interface [**IUpdate3**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate3) définit la propriété suivante.



| Propriété                                  | Description                                                                                                             |
|-------------------------------------------|-------------------------------------------------------------------------------------------------------------------------|
| [**BrowseOnly**](/windows/desktop/api/Wuapi/nf-wuapi-iupdate3-get_browseonly) | Obtient une valeur booléenne qui indique si une mise à jour peut être détectée uniquement en parcourant les mises à jour disponibles. |



 

Pour plus d’informations sur les membres hérités par cette interface, consultez les interfaces suivantes :

-   [**IUpdate2**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate2)
-   [**IUpdate**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate)
