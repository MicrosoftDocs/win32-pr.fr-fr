# <a name="istringcollection-properties"></a>Propriétés IStringCollection

L’interface [**IStringCollection**](/windows/desktop/api/Wuapi/nn-wuapi-istringcollection) définit les propriétés suivantes.



| Propriété                                        | Description                                                                                                                     |
|-------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------|
| [**\_NewEnum**](/windows/desktop/api/Wuapi/nf-wuapi-istringcollection-get__newenum) | Obtient une interface [IEnumVARIANT](/windows/win32/api/oaidl/nn-oaidl-ienumvariant) qui peut être utilisée pour énumérer la collection. |
| [**Saut**](/windows/desktop/api/Wuapi/nf-wuapi-istringcollection-get_count)        | Obtient le nombre d’éléments de la collection.                                                                                  |
| [**Élément**](/windows/desktop/api/Wuapi/nf-wuapi-istringcollection-get_item)          | Obtient ou définit une chaîne dans la collection.                                                                                        |
| [**Seulement**](/windows/desktop/api/Wuapi/nf-wuapi-istringcollection-get_readonly)  | Obtient une valeur booléenne qui indique si la collection est en lecture seule.                                                        |
