# <a name="iupdateexception-properties"></a>Propriétés IUpdateException

L’interface [**IUpdateException**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateexception) définit les propriétés suivantes.



| Propriété                                    | Description                                                |
|---------------------------------------------|------------------------------------------------------------|
| [**Contexte**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateexception-get_context) | Obtient le contexte de l’exception.                         |
| [**Signé**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateexception-get_hresult) | obtient le code **HRESULT** de base Windows pour l’exception. |
| [**Message**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateexception-get_message) | Obtient un message qui décrit l'exception active.       |
