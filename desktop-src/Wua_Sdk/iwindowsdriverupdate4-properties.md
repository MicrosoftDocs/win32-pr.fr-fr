# <a name="iwindowsdriverupdate4-properties"></a>Propriétés IWindowsDriverUpdate4

L’interface [**IWindowsDriverUpdate4**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate4) définit les propriétés suivantes.



| Propriété                                                                               | Description                                                                 |
|----------------------------------------------------------------------------------------|-----------------------------------------------------------------------------|
| [**PerUser**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate4-get_peruser)                                       | Obtient une valeur booléenne qui indique si une mise à jour est une mise à jour par utilisateur. |
| [**WindowsDriverUpdateEntries**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate4-get_windowsdriverupdateentries) | Obtient les entrées de mise à jour de pilote applicables à la mise à jour.          |



 

Pour plus d’informations sur les membres hérités par cette interface, consultez l’interface suivante.

-   [**IWindowsDriverUpdate**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate)
-   [**IWindowsDriverUpdate2**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate2)
-   [**IWindowsDriverUpdate3**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate3)
