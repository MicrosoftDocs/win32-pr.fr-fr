# <a name="iupdatehistoryentry2-properties"></a>Propriétés IUpdateHistoryEntry2

L’interface [**IUpdateHistoryEntry2**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatehistoryentry2) définit la propriété suivante.



| Propriété                                              | Description                                                            |
|-------------------------------------------------------|------------------------------------------------------------------------|
| [**Catégories**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatehistoryentry2-get_categories) | Obtient une collection des catégories de mise à jour auxquelles une mise à jour appartient. |



 

Pour plus d’informations sur les membres hérités par cette interface, consultez l’interface suivante :

-   [**IUpdateHistoryEntry**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatehistoryentry)
