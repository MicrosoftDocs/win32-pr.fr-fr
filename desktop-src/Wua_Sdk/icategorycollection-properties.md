# <a name="icategorycollection-properties"></a>Propriétés ICategoryCollection

L’interface [**ICategoryCollection**](/windows/desktop/api/Wuapi/nn-wuapi-icategorycollection) définit les propriétés suivantes.



| Propriété                                          | Description                                                                                                                     |
|---------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------|
| [**\_NewEnum**](/windows/desktop/api/Wuapi/nf-wuapi-icategorycollection-get__newenum) | Obtient une interface [IEnumVARIANT](/windows/win32/api/oaidl/nn-oaidl-ienumvariant) qui peut être utilisée pour énumérer la collection. |
| [**Saut**](/windows/desktop/api/Wuapi/nf-wuapi-icategorycollection-get_count)        | Obtient le nombre d’éléments de la collection.                                                                                  |
| [**Élément**](/windows/desktop/api/Wuapi/nf-wuapi-icategorycollection-get_item)          | Obtient une interface [**ICategory**](/windows/desktop/api/Wuapi/nn-wuapi-icategory) à partir de la collection.                                                           |
