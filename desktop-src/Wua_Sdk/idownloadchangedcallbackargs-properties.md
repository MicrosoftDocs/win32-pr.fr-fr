| Propriété                                                          | Description                                                                                                       |
|-------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------|
| [**Progress**](/windows/desktop/api/Wuapi/nf-wuapi-idownloadprogresschangedcallbackargs-get_progress) | Obtient une interface qui contient la progression du téléchargement asynchrone au moment où le rappel a été effectué. |
