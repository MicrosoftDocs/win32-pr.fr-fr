# <a name="other-sources-of-windows-update-agent-information"></a>autres Sources d’informations sur l’Agent de Windows Update

le kit de développement logiciel (SDK) de l' [api d’Agent Windows Update](portal-client.md)contient les descriptions les plus complètes et actuelles de l’api. Il s’agit de la principale source de documents de référence pour les développeurs pour WUA.

## <a name="software-development-kit-sdk-for-windows-update-agent-wua"></a>kit de développement logiciel (SDK) pour l’Agent de Windows Update (WUA)

les ressources suivantes décrivent comment déployer et administrer Windows Server Update Services et WUA :

-   [Windows Server Update Services](/documentation/)

    vous pouvez télécharger le kit de développement logiciel (SDK) Windows Server Update Services et consulter d’autres documents qui décrivent comment installer, déployer et gérer des Services Windows Update.

-   [Téléchargement MSDN et centre de code](/powerapps/developer/common-data-service/org-service/subscribe-sdk-assembly-updates-using-nuget)

    Vous pouvez télécharger les mises à jour vers les exemples, les en-têtes et les bibliothèques du kit de développement logiciel (SDK) WUA. Les téléchargements du téléchargement MSDN et du centre de code sont organisés par rubrique. Vous pouvez également trouver des téléchargements sur MSDN, Microsoft Developer Network, à l’aide de l’outil de recherche du [Centre de téléchargement Microsoft](https://go.microsoft.com/fwlink/p/?linkid=83468) .
