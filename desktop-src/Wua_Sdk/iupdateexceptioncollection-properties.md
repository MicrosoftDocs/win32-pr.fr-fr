# <a name="iupdateexceptioncollection-properties"></a>Propriétés IUpdateExceptionCollection

L’interface [**IUpdateExceptionCollection**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateexceptioncollection) définit les propriétés suivantes.



| Propriété                                                 | Description                                                                                                          |
|----------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| [**\_NewEnum**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateexceptioncollection-get__newenum) | Obtient une interface [**IEnumVARIANT**](/windows/win32/api/oaidl/nn-oaidl-ienumvariant) utilisée pour énumérer la collection. |
| [**Count**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateexceptioncollection-get_count)        | Obtient le nombre d’éléments de la collection.                                                                       |
| [**Élément**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateexceptioncollection-get_item)          | Obtient une interface [**IUpdateException**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateexception) dans la collection.                                    |
