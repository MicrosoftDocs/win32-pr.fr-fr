# <a name="iupdateidentity-properties"></a>Propriétés IUpdateIdentity

L’interface [**IUpdateIdentity**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateidentity) définit les propriétés suivantes.



| Propriété                                                 | Description                                            |
|----------------------------------------------------------|--------------------------------------------------------|
| [**RevisionNumber**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateidentity-get_revisionnumber) | Obtient le numéro de révision d’une mise à jour.                 |
| [**UpdateID**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateidentity-get_updateid)             | Obtient l’identificateur indépendant de la révision d’une mise à jour. |
