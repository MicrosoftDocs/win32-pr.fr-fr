# <a name="wua-controls-and-property-pages-enumerations"></a>Contrôles WUA et énumérations des pages de propriétés

Windows Update Agent (WUA) utilise les énumérations répertoriées dans le tableau suivant pour représenter l’état des opérations.



| Énumération                                                                                  | Description                                                                                                                                                                                                                                                                           |
|----------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**AddServiceFlag**](/windows/win32/api/wuapi/ne-wuapi-addserviceflag)                                                     | Définit les méthodes possibles de traitement de l’inscription du service.                                                                                                                                                                                                         |
| [**AutoDownloadMode**](/windows/win32/api/wuapi/ne-wuapi-autodownloadmode)                                                 | Définit la logique utilisée pour déterminer si Mises à jour automatiques télécharge automatiquement une mise à jour une fois qu’elle est déterminée pour s’appliquer à l’ordinateur.                                                                                                                  |
| [**AutomaticUpdatesNotificationLevel**](/windows/win32/api/wuapi/ne-wuapi-automaticupdatesnotificationlevel)               | Définit les méthodes possibles dans lesquelles les utilisateurs élevés sont avertis des événements de Mises à jour automatiques.                                                                                                                                                                                        |
| [**AutomaticUpdatesPermissionType**](/windows/win32/api/wuapi/ne-wuapi-automaticupdatespermissiontype)                     | Définit les méthodes possibles pour définir la propriété [**IAutomaticUpdatesSettings :: NotificationLevel**](/windows/win32/api/wuapi/ne-wuapi-automaticupdatesnotificationlevel) ou la propriété ou [**IAutomaticUpdatesSettings2 :: IncludeRecommendedUpdates**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdatessettings2-get_includerecommendedupdates) . |
| [**AutomaticUpdatesScheduledInstallationDay**](/windows/win32/api/wuapi/ne-wuapi-automaticupdatesscheduledinstallationday) | Définit les jours de la semaine où Mises à jour automatiques installe ou désinstalle des mises à jour.                                                                                                                                                                                                   |
| [**AutomaticUpdatesUserType**](/windows/win32/api/wuapi/ne-wuapi-automaticupdatesusertype)                                 | Décrit le type d’utilisateur.                                                                                                                                                                                                                                                           |
| [**AutoSelectionMode**](/windows/win32/api/wuapi/ne-wuapi-autoselectionmode)                                               | Définit la logique utilisée pour déterminer si une mise à jour particulière sera sélectionnée automatiquement lorsque l’utilisateur affichera les mises à jour disponibles dans l’interface utilisateur Windows Update.                                                                                                        |
| [**DeploymentAction**](/windows/win32/api/wuapi/ne-wuapi-deploymentaction)                                                 | Définit l’action pour laquelle une mise à jour est déployée explicitement.                                                                                                                                                                                                                        |
| [**DownloadPhase**](/windows/win32/api/wuapi/ne-wuapi-downloadphase)                                                       | Définit la progression du téléchargement d’une mise à jour en cours qui est retournée par la propriété [**IDownloadProgress :: CurrentUpdateDownloadPhase**](/windows/desktop/api/Wuapi/nf-wuapi-idownloadprogress-get_currentupdatedownloadphase) .                                                                                      |
| [**DownloadPriority**](/windows/win32/api/wuapi/ne-wuapi-downloadpriority)                                                 | Définit les priorités possibles pour une opération de téléchargement.                                                                                                                                                                                                                             |
| [**InstallationImpact**](/windows/win32/api/wuapi/ne-wuapi-installationimpact)                                             | Définit les niveaux d’impact possibles qui peuvent être causés par l’installation ou la désinstallation d’une mise à jour.                                                                                                                                                                                     |
| [**InstallationRebootBehavior**](/windows/win32/api/wuapi/ne-wuapi-installationrebootbehavior)                             | Définit les comportements de redémarrage possibles pour une mise à jour.                                                                                                                                                                                                                                 |
| [**OperationResultCode**](/windows/win32/api/wuapi/ne-wuapi-operationresultcode)                                           | Définit les résultats possibles d’une opération de téléchargement, d’installation, de désinstallation ou de vérification sur une mise à jour.                                                                                                                                                                               |
| [**SearchScope**](/windows/win32/api/wuapi/ne-wuapi-searchscope)                                                           | Spécifie la variété des mises à jour qui doivent être retournées par la recherche : les mises à jour par ordinateur, les mises à jour par utilisateur ou les deux. Les mises à jour par utilisateur sont des mises à jour conçues pour n’affecter que l’environnement d’un seul utilisateur. Pour plus d’informations, consultez [**IUpdate4 ::P eruser**](/windows/desktop/api/Wuapi/nf-wuapi-iupdate4-get_peruser).    |
| [**ServerSelection**](/openspecs/windows_protocols/ms-uamg/07e2bfa4-6795-4189-b007-cc50b476181a)                                                   | Définit les services de mise à jour sur lesquels Windows Update peut opérer. .                                                                                                                                                                                                                |
| [**UpdateEndpointAuthTokenType**](updateendpointauthtokentype.md)                           | Définit le type des jetons qui peuvent être utilisés pour l’authentification auprès d’un point de terminaison.                                                                                                                                                                                                      |
| [**UpdateEndpointType**](updateendpointtype.md)                                             | Définit le type de points de terminaison qui peuvent être utilisés pour se connecter à un service.                                                                                                                                                                                                               |
| [**UpdateExceptionContext**](/windows/win32/api/wuapi/ne-wuapi-updateexceptioncontext)                                     | Définit le contexte dans lequel un objet [**IUpdateException**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateexception) peut être fourni.                                                                                                                                                                                  |
| [**UpdateLockdownOption**](/windows/win32/api/wuapi/ne-wuapi-updatelockdownoption)                                         | Définit les fonctionnalités accessibles à l’objet agent de Windows Update (WUA) à partir de Windows Update.                                                                                                                                                                                  |
| [**Update**](/windows/win32/api/wuapi/ne-wuapi-updateoperation)                                                   | Définit les opérations qui peuvent être tentées sur une mise à jour.                                                                                                                                                                                                                            |
| [**UpdateServiceOption**](/windows/win32/api/wuapi/ne-wuapi-updateserviceoption)                                           | Définit les options pour supprimer l’inscription de service pour un service de package d’analyse.                                                                                                                                                                                                    |
| [**UpdateServiceRegistrationState**](/windows/win32/api/wuapi/ne-wuapi-updateserviceregistrationstate)                     | Définit les États possibles pour un service de mise à jour.                                                                                                                                                                                                                                    |
| [**UpdateType**](/windows/win32/api/wuapi/ne-wuapi-updatetype)                                                             | Indique si une mise à jour est une mise à jour logicielle ou une mise à jour de pilote.                                                                                                                                                                                                                  |
