# <a name="iupdate-methods"></a>Méthodes IUpdate

L’interface [**IUpdate**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate) définit les méthodes suivantes.



| Méthode                                         | Description                                                                                      |
|------------------------------------------------|--------------------------------------------------------------------------------------------------|
| [**AcceptEula**](/windows/desktop/api/Wuapi/nf-wuapi-iupdate-accepteula)       | accepte les termes du contrat de licence logiciel Microsoft associés à Windows Update.<br/> |
| [**CopyFromCache**](/windows/desktop/api/Wuapi/nf-wuapi-iupdate-copyfromcache) | Copie le contenu d’une mise à jour dans un chemin d’accès spécifié.<br/>                                 |
