# <a name="iinstallationprogresschangedcallbackargs-properties"></a>Propriétés IInstallationProgressChangedCallbackArgs

L’interface [**IInstallationProgressChangedCallbackArgs**](/windows/desktop/api/Wuapi/nn-wuapi-iinstallationprogresschangedcallbackargs) définit la propriété suivante.



| Propriété                                                              | Description                                                                                                                        |
|-----------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------|
| [**Progress**](/windows/desktop/api/Wuapi/nf-wuapi-iinstallationprogresschangedcallbackargs-get_progress) | Obtient une interface qui contient la progression de l’installation ou de la désinstallation asynchrone au moment où le rappel a été effectué. |
