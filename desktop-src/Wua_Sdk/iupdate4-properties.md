# <a name="iupdate4-properties"></a>Propriétés IUpdate4

L’interface [**IUpdate4**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate4) définit la propriété suivante.



| Propriété                            | Description                                                            |
|-------------------------------------|------------------------------------------------------------------------|
| [**PerUser**](/windows/desktop/api/Wuapi/nf-wuapi-iupdate4-get_peruser) | Obtient une valeur booléenne qui indique s’il s’agit d’une mise à jour par utilisateur. |



 

Pour plus d’informations sur les membres hérités par cette interface, consultez les interfaces suivantes :

-   [**IUpdate3**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate3)
-   [**IUpdate2**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate2)
-   [**IUpdate**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate)
