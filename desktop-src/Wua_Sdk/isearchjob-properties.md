# <a name="isearchjob-properties"></a>Propriétés ISearchJob

L’interface [**ISearchJob**](/windows/desktop/api/Wuapi/nn-wuapi-isearchjob) définit les propriétés suivantes.



| Propriété                                      | Description                                                                                                                                                 |
|-----------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**AsyncState**](/windows/desktop/api/Wuapi/nf-wuapi-isearchjob-get_asyncstate)   | Obtient l’objet d’état spécifique à l’appelant qui est passé à la méthode [**IUpdateSearch. BeginSearch**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatesearcher-beginsearch) .                         |
| [**IsCompleted**](/windows/desktop/api/Wuapi/nf-wuapi-isearchjob-get_iscompleted) | Obtient une valeur booléenne qui indique si l’appel à la méthode [**IUpdateSearch. BeginSearch**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatesearcher-beginsearch) est complètement traité. |
