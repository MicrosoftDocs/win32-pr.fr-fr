# <a name="iupdateservicemanager2-methods"></a>Méthodes IUpdateServiceManager2

L’interface [**IUpdateServiceManager2**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateservicemanager2) définit les méthodes suivantes.



| Méthode                                                                                      | Description                                                                                                                                                                                |
|---------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**AddService2**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateservicemanager2-addservice2)                           | Inscrit un service auprès de WUA sans nécessiter de fichier cab d’autorisation (. cab) et retourne un pointeur vers une interface [**IUpdateServiceRegistration**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateservicecollection) . |
| [**QueryServiceRegistration**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateservicemanager2-queryserviceregistration) | Retourne un pointeur vers une interface [**IUpdateServiceRegistration**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateserviceregistration) .                                                                                        |



 

Pour plus d’informations sur les membres hérités par cette interface, consultez l’interface suivante.

-   [**IUpdateServiceManager**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateservicemanager)
