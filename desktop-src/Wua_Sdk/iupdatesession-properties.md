# <a name="iupdatesession-properties"></a>Propriétés IUpdateSession

L’interface [**IUpdateSession**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatesession) définit les propriétés suivantes.



| Propriété                                                          | Description                                                                  |
|-------------------------------------------------------------------|------------------------------------------------------------------------------|
| [**ClientApplicationID**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatesession-get_clientapplicationid) | Obtient et définit l’application cliente actuelle.                                |
| [**Seulement**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatesession-get_readonly)                       | Obtient une valeur booléenne qui indique si l’objet de session est en lecture seule. |
| [**WebProxy**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatesession-get_webproxy)                       | Obtient et définit les paramètres de proxy utilisés pour accéder au serveur.         |
