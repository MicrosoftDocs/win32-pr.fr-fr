# <a name="isysteminformation-properties"></a>Propriétés ISystemInformation

L’interface [**ISystemInformation**](/windows/desktop/api/Wuapi/nn-wuapi-isysteminformation) définit les propriétés suivantes.



| Propriété                                                                    | Description                                                                                                                                     |
|-----------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------|
| [**OemHardwareSupportLink**](/windows/desktop/api/Wuapi/nf-wuapi-isysteminformation-get_oemhardwaresupportlink) | Obtient un lien hypertexte vers des informations de support technique pour le matériel OEM.                                                                             |
| [**RebootRequired**](/windows/desktop/api/Wuapi/nf-wuapi-isysteminformation-get_rebootrequired)                 | Obtient une valeur booléenne qui indique si un redémarrage du système est nécessaire pour terminer l’installation ou la désinstallation d’une ou de plusieurs mises à jour. |
