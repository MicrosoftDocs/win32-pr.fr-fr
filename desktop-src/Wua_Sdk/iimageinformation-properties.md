# <a name="iimageinformation-properties"></a>Propriétés IImageInformation

L’interface [**IImageInformation**](/windows/desktop/api/Wuapi/nn-wuapi-iimageinformation) définit les propriétés suivantes.



| Propriété                                     | Description                              |
|----------------------------------------------|------------------------------------------|
| [**AltText**](/windows/desktop/api/Wuapi/nf-wuapi-iimageinformation-get_alttext) | Obtient le texte de remplacement de l’image.   |
| [**Hauteur**](/windows/desktop/api/Wuapi/nf-wuapi-iimageinformation-get_height)   | Obtient la hauteur de l’image, en pixels. |
| [**Source**](/windows/desktop/api/Wuapi/nf-wuapi-iimageinformation-get_source)   | Obtient l’emplacement source de l’image.   |
| [**Largeur**](/windows/desktop/api/Wuapi/nf-wuapi-iimageinformation-get_width)     | Obtient la largeur de l’image, en pixels.  |
