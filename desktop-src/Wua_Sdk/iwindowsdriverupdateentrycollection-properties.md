# <a name="iwindowsdriverupdateentrycollection-properties"></a>Propriétés IWindowsDriverUpdateEntryCollection

L’interface [**IWindowsDriverUpdateEntryCollection**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdateentrycollection) définit les propriétés suivantes.



| Propriété                                                          | Description                                                                                                          |
|-------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| [**\_NewEnum**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdateentrycollection-get__newenum) | Obtient une interface [**IEnumVARIANT**](/windows/win32/api/oaidl/nn-oaidl-ienumvariant) utilisée pour énumérer la collection. |
| [**Count**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdateentrycollection-get_count)        | Obtient le nombre d’éléments de la collection.                                                                       |
| [**Élément**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdateentrycollection-get_item)          | Obtient un intrface [**IWindowsDriverUpdateEntry**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdateentry) dans la collection.                   |
