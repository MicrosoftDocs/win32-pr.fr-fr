# <a name="iautomaticupdatessettings3-properties"></a>Propriétés IAutomaticUpdatesSettings3

L’interface [**IAutomaticUpdatesSettings3**](/windows/desktop/api/Wuapi/nn-wuapi-iautomaticupdatessettings3) définit les propriétés suivantes.



| Propriété                                                                                  | Description                                                                                                                                     |
|-------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------|
| [**FeaturedUpdatesEnabled**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdatessettings3-get_featuredupdatesenabled)       | Obtient et définit une valeur booléenne qui indique s’il faut afficher des notifications pour les mises à jour proposées.                                             |
| [**NonAdministratorsElevated**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdatessettings3-get_nonadministratorselevated) | Obtient et définit une valeur booléenne qui indique si des non-administrateurs peuvent effectuer des actions liées à la mise à jour sans l’approbation de l’administrateur. |



 

Pour plus d’informations sur les membres hérités par cette interface, consultez les interfaces suivantes.

-   [**IAutomaticUpdatesSettings**](/windows/desktop/api/Wuapi/nn-wuapi-iautomaticupdatessettings)
-   [**IAutomaticUpdatesSettings2**](/windows/desktop/api/Wuapi/nn-wuapi-iautomaticupdatessettings2)
