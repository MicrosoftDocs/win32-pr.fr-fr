# <a name="iupdateendpointauthprovider-methods"></a>Méthodes IUpdateEndpointAuthProvider

L’interface [**IUpdateEndpointAuthProvider**](iupdateendpointauthprovider.md) définit les méthodes suivantes.



| Méthode                                                                                             | Description                                                                         |
|----------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------|
| [**GetEndpointToken**](iupdateendpointauthprovider-getendpointtoken.md)                           | Demandez un jeton pour le point de terminaison du service à l’aide des informations d’identification spécifiées.    |
| [**GetPreferredEndpointTokenType**](iupdateendpointauthprovider-getpreferredendpointtokentype.md) | Retourne le type de jeton d’authentification par défaut pour le point de terminaison du service. |
