# <a name="iupdatedownloadcontentcollection-properties"></a>Propriétés IUpdateDownloadContentCollection

L’interface **IUpdateDownloadContentCollection** définit les propriétés suivantes.



| Propriété                                                       | Description                                                                                                                             |
|----------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------|
| [**\_NewEnum**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatedownloadcontentcollection-get__newenum) | Obtient une interface [**IEnumVARIANT**](/windows/win32/api/oaidl/nn-oaidl-ienumvariant) utilisée pour énumérer la collection.                    |
| [**Count**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatedownloadcontentcollection-get_count)        | Obtient le nombre d’éléments de la collection.                                                                                          |
| [**Élément**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatedownloadcontentcollection-get_item)          | Obtient le contenu téléchargé pour une mise à jour à partir d’une interface [**IUpdateDownloadContentCollection**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatedownloadcontentcollection) . |
