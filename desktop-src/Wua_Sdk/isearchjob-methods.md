# <a name="isearchjob-methods"></a>Méthodes ISearchJob

L’interface [**ISearchJob**](/windows/desktop/api/Wuapi/nn-wuapi-isearchjob) définit les méthodes suivantes.



| Méthode                                          | Description                                                                          |
|-------------------------------------------------|--------------------------------------------------------------------------------------|
| [**Nettoyage**](/windows/desktop/api/Wuapi/nf-wuapi-isearchjob-cleanup)           | Attend qu’une opération asynchrone se termine, puis libère tous les rappels. |
| [**RequestAbort**](/windows/desktop/api/Wuapi/nf-wuapi-isearchjob-requestabort) | Effectue une demande d’annulation d’une recherche asynchrone.                                    |
