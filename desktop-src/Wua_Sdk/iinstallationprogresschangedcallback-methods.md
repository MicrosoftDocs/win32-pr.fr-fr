# <a name="iinstallationprogresschangedcallback-methods"></a>Méthodes IInstallationProgressChangedCallback

L’interface [**IInstallationProgressChangedCallback**](/windows/desktop/api/Wuapi/nn-wuapi-iinstallationprogresschangedcallback) définit la méthode suivante.



| Méthode                                                        | Description                                                                                                                                                                                                                                                                                           |
|---------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Appeler**](/windows/desktop/api/Wuapi/nf-wuapi-iinstallationprogresschangedcallback-invoke) | Gère la notification de la progression d’une installation ou d’une désinstallation asynchrone qui a été lancée par un appel à la méthode [**IUpdateInstaller. BeginInstall**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateinstaller-begininstall) ou [**IUpdateInstaller. BeginUninstall**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateinstaller-beginuninstall) . |
