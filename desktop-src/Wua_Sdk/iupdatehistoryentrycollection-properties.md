# <a name="iupdatehistoryentrycollection-properties"></a>Propriétés IUpdateHistoryEntryCollection

L’interface [**IUpdateHistoryEntryCollection**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatehistoryentrycollection) définit les propriétés suivantes.



| Propriété                                                    | Description                                                                                                          |
|-------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| [**\_NewEnum**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatehistoryentrycollection-get__newenum) | Obtient une interface [**IEnumVARIANT**](/windows/win32/api/oaidl/nn-oaidl-ienumvariant) utilisée pour énumérer la collection. |
| [**Saut**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatehistoryentrycollection-get_count)        | Obtient le nombre d’éléments de la collection.                                                                       |
| [**Élément**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatehistoryentrycollection-get_item)          | Obtient une interface [**IUpdateHistoryEntry**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatehistoryentry) dans la collection.                              |
