# <a name="about-cng"></a>À propos de CNG

API de chiffrement : Next Generation (CNG) est le remplacement à long terme de l’interface [*CryptoAPI*](/windows/desktop/SecGloss/c-gly). CNG est conçu pour être extensible à de nombreux niveaux et un chiffrement indépendant dans le comportement.

## <a name="features"></a>Fonctionnalités

Pour plus d’informations sur les fonctionnalités de CNG, consultez [fonctionnalités CNG](cng-features.md).

## <a name="cryptographic-primitives"></a>Primitives de chiffrement

Pour plus d’informations sur les primitives de chiffrement dans CNG, consultez [primitives de chiffrement](cryptographic-primitives.md).

## <a name="key-storage-and-retrieval"></a>Stockage et récupération de clés

Pour plus d’informations sur le stockage et la récupération de clés dans CNG, consultez [stockage et récupération de clés](key-storage-and-retrieval.md).

## <a name="key-import-and-export"></a>Importation et exportation de clé

Pour plus d’informations sur l’importation et l’exportation des clés, consultez [importation et exportation de clés](key-import-and-export.md).

## <a name="data-protection-api-next-generation"></a>API de protection des données : nouvelle génération

CNG fournit un ensemble d’API qui peuvent être utilisées pour chiffrer et déchiffrer facilement le contenu des principaux d’autorisation sur plusieurs ordinateurs. Pour plus d’informations, consultez [CNG DPAPI](cng-dpapi.md).
