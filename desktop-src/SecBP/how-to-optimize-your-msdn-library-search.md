# <a name="how-to-optimize-your-msdn-library-search"></a>Comment optimiser votre recherche dans la bibliothèque MSDN

pour effectuer une recherche dans msdn, entrez un mot clé ou une chaîne de caractères dans la zone **rechercher dans msdn avec Bing** . Une fois que vous avez effectué une recherche sur le site Web [Microsoft Developer Network](https://msdn.microsoft.com/default.aspx) (MSDN), vous pouvez affiner vos résultats de recherche en cliquant sur les options disponibles dans la colonne **affiner la recherche** sous **par source** ou **par rubrique**. Ces options deviennent disponibles après votre première tentative de recherche.

**pour retourner uniquement les rubriques de la section développement Windows de MSDN Library**

1.  Dans la liste **par rubrique** , cliquez sur **Win32 & com**. Une liste supplémentaire appelée **affinements actuel** s’affiche. La ligne d' **affinements actuelle** contient **Win32 & com** .
2.  Dans la liste **par source** , cliquez sur **documentation & Articles**. La liste des **affinements actuels** comprend désormais la **documentation & Articles**.

à mesure que vous recherchez, des filtres supplémentaires peuvent devenir disponibles, par exemple, **Sécurité Windows**, **Windows graphics and Multimedia**, ou **Windows messaging and Collaboration**.

**Pour rechercher uniquement les rubriques du kit de développement logiciel (SDK) de sécurité**

-   dans la liste **par rubrique** , cliquez sur **Sécurité Windows**. la liste des **affinements actuels** comprend désormais **Sécurité Windows**.

Lorsque vous tapez une chaîne de recherche, les suggestions s’affichent automatiquement sous la zone de recherche. Pour rechercher du contenu dans le kit de développement logiciel (SDK), recherchez **API** ou **Windows** dans les suggestions.

Lorsque vous cliquez sur une rubrique dans les résultats de la recherche, vous pouvez déterminer rapidement si la rubrique fournit le contenu du kit de développement logiciel (SDK) en affichant le chemin de navigation qui apparaît en haut de la rubrique. Par exemple, une recherche de **chiffrement** génère des suggestions qui incluent l' **API de chiffrement**. En cliquant sur l' **API de chiffrement** par suggestion, génère un volet des résultats de recherche qui comprend des **fonctions de chiffrement (Windows)**. en cliquant sur **fonctions de chiffrement (Windows)**, vous pouvez accéder à une rubrique qui contient le chemin de navigation suivant en haut de la rubrique :

**msdn > msdn Library > Windows Development > Security > cryptography > cryptography > cryptography functions >**

Tout le contenu du SDK de sécurité contient cette partie de ce chemin de navigation :

**msdn > msdn Library > Windows Development > Security >**

**Pour supprimer les filtres de résultats de recherche de votre recherche MSDN**

-   Dans la liste des **affinements actuels** , cliquez sur le **(x)** qui suit le filtre de recherche que vous souhaitez supprimer. Par exemple, pour supprimer la **documentation & Articles** des **affinements actuels**, cliquez sur la **(x)** qui suit **documentation & Articles**. Vous pouvez également cliquer sur **Supprimer tout** pour supprimer tous les filtres de votre recherche MSDN.
