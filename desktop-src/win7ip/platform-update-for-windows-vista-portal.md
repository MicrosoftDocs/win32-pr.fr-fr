# <a name="platform-update-for-windows-vista"></a>Mise à jour de plateforme pour Windows Vista

## <a name="purpose"></a>Objectif

La mise à jour de plateforme pour Windows Vista et la mise à jour de plateforme pour Windows Server 2008 sont des mises à jour du système d’exploitation de l’utilisateur final qui prennent en charge l’utilisation des technologies Windows 7 sélectionnées sur les versions précédentes du système d’exploitation Windows. Les mises à jour incluent un ensemble de bibliothèques Runtime qui permettent aux développeurs d’applications de cibler les versions actuelles, Windows 7 et Windows Server 2008 R2, ainsi que les versions précédentes, Windows Vista et Windows Server 2008.

## <a name="developer-audience"></a>Développeurs concernés

L’API prise en charge par la mise à jour de plateforme pour Windows Vista et la mise à jour de plateforme pour Windows Server 2008 sont conçues pour être utilisées par les développeurs C/C++ expérimentés.

Compétences recommandées :

-   Programmation COM
-   Programmation d’API Windows

## <a name="run-time-requirements"></a>Conditions d’exécution

Pour Windows Vista, les utilisateurs finaux obtiennent la mise à jour de la plate-forme pour Windows Vista à partir de Windows Update. Pour Windows Server 2008, les utilisateurs finaux obtiennent de Windows Update la mise à jour de la plateforme pour Windows Server 2008.
