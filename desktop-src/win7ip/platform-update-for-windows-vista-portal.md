# <a name="platform-update-for-windows-vista"></a>mise à jour de plateforme pour Windows Vista

## <a name="purpose"></a>Objectif

la mise à jour de la plateforme pour Windows Vista et la mise à jour de la plateforme pour Windows Server 2008 sont des mises à jour du système d’exploitation de l’utilisateur final qui prennent en charge l’utilisation des technologies Windows 7 sélectionnées sur les versions précédentes du système d’exploitation Windows. les mises à jour incluent un ensemble de bibliothèques runtime qui permettent aux développeurs d’applications de cibler les versions actuelles, Windows 7 et Windows server 2008 R2, ainsi que les versions précédentes, Windows Vista et Windows server 2008.

## <a name="developer-audience"></a>Développeurs concernés

l’API prise en charge par la mise à jour de la plateforme pour Windows Vista et la mise à jour de la plateforme pour Windows Server 2008 sont conçues pour être utilisées par les développeurs C/C++ expérimentés.

Compétences recommandées :

-   Programmation COM
-   Windows Programmation d’API

## <a name="run-time-requirements"></a>Conditions d’exécution

pour Windows vista, les utilisateurs finaux obtiennent la mise à jour de la plate-forme pour Windows Vista à partir d’Windows Update. pour Windows server 2008, les utilisateurs finaux obtiennent la mise à jour de la plate-forme pour Windows Server 2008 à partir de Windows Update.
