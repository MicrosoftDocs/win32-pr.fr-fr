# <a name="winsat-mof-classes"></a>Classes MOF WinSAT

\[Les classes MOF WinSAT peuvent être modifiées ou non disponibles pour les mises en production après Windows 8.1.\]

L’outil d’évaluation système Windows (WinSAT) définit la classe de Windows Management Instrumentation (WMI) format MOF (MOF) suivante, qui se trouve dans l' \\ \\ espace de noms CIMV2 racine :

-   [**Win32 \_ WinSAT**](win32-winsat.md)

Vous pouvez utiliser cette classe pour récupérer des informations récapitulatives sur l’évaluation formelle la plus récente.
