# <a name="winsat-interfaces"></a>Interfaces WinSAT

\[Les interfaces WinSAT peuvent être modifiées ou non disponibles pour les mises en production après Windows 8.1.\]

L’outil d’évaluation système Windows (WinSAT) définit les interfaces suivantes que vous pouvez utiliser pour initier des évaluations sur l’ordinateur et pour interroger des évaluations formelles dans le magasin de données WinSAT :

-   [**IInitiateWinSATAssessment**](/windows/desktop/api/Winsatcominterfacei/nn-winsatcominterfacei-iinitiatewinsatassessment)
-   [**IProvideWinSATAssessmentInfo**](/windows/desktop/api/Winsatcominterfacei/nn-winsatcominterfacei-iprovidewinsatassessmentinfo)
-   [**IProvideWinSATResultsInfo**](/windows/desktop/api/Winsatcominterfacei/nn-winsatcominterfacei-iprovidewinsatresultsinfo)
-   [**IProvideWinSATVisuals**](/windows/desktop/api/Winsatcominterfacei/nn-winsatcominterfacei-iprovidewinsatvisuals)
-   [**IQueryAllWinSATAssessments**](/windows/desktop/api/Winsatcominterfacei/nn-winsatcominterfacei-iqueryallwinsatassessments)
-   [**IQueryRecentWinSATAssessment**](/windows/desktop/api/Winsatcominterfacei/nn-winsatcominterfacei-iqueryrecentwinsatassessment)
-   [**IWinSATInitiateEvents**](/windows/desktop/api/Winsatcominterfacei/nn-winsatcominterfacei-iwinsatinitiateevents)

WinSAT définit également une [classe](winsat-mof-classes.md) Windows Management Instrumentation (WMI) format MOF (MOF) qui offre des fonctionnalités similaires.
