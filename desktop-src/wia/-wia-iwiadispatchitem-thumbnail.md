# <a name="itemthumbnail-property"></a>Item. Thumbnail, propriété

Chemin d’accès et nom de fichier de l’image miniature. Cette propriété a la **valeur null** si l’élément ne prend pas en charge les miniatures, ou si un chemin d’accès ne peut pas être généré. Lecture seule.

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```JScript
propVal = Item.Thumbnail
```



## <a name="property-value"></a>Valeur de la propriété

Chaîne qui reçoit le chemin d’accès et le nom de fichier.
