# <a name="item-properties"></a>Propriétés d'élément

les propriétés d’élément sont des propriétés qui contrôlent la fonction d’un élément d’Acquisition d’Image Windows (WIA). Ces propriétés sont accessibles par le biais de n’importe quel élément d’une arborescence qui représente un périphérique. Il existe trois types de propriétés d’élément : courant, scanneur et appareil photo.

Pour obtenir des informations de référence sur ces propriétés, consultez :

-   [**Constantes de propriété d’élément WIA courantes**](-wia-wiaitempropcommonitem.md)
-   [**Constantes de propriété d’élément WIA de l’appareil photo**](-wia-wiaitempropcameraitem.md)
-   [**Constantes de propriété d’élément WIA du scanneur**](-wia-wiaitempropscanneritem.md)
