# <a name="itemthumbheight-property"></a>Item. ThumbHeight, propriété

Hauteur, en pixels, de l’image miniature. Cette propriété retourne-1 si cet élément ne prend pas en charge les miniatures. Lecture seule.

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```JScript
propVal = Item.ThumbHeight
```



## <a name="property-value"></a>Valeur de la propriété

Variable qui reçoit la hauteur de la miniature.
