# <a name="syntax-conventions"></a>Conventions de syntaxe

Le tableau suivant répertorie les termes utilisés dans les instructions de syntaxe pour la référence du modèle de script WIA (Windows Image Acquisition). 

| Terme                      | Description                                                    |
|---------------------------|----------------------------------------------------------------|
| *Nom du fichier*                | Chemin d’accès et nom d’un fichier.                                   |
| *objDeviceInfo*           | Objet [**DeviceInfo**](-wia-deviceinfo.md) .                |
| *objDeviceInfoCollection* | Collection d’objets [**DeviceInfo**](-wia-deviceinfo.md) . |
| *objItem*                 | Objet d' [**élément**](-wia-item.md) .                           |
| *objItemCollection*       | Collection d’objets [**Item**](-wia-item.md) .             |
| *objWia*                  | Objet [**WIA**](-wia-wia.md) .                              |
| *PropID*                  | Constante de propriété WIA.                                       |
| *var*                     | Valeur de **type Variant** .                                           |
