# <a name="wia-tutorial"></a>Didacticiel WIA

Ce didacticiel fournit des instructions pas à pas pour utiliser l’acquisition d’images Windows (WIA) dans des applications réelles.

Ce didacticiel présente le code des exemples d’applications inclus dans le kit de développement logiciel (SDK). Il est destiné à illustrer la façon dont les développeurs peuvent implémenter les fonctionnalités majeures d’WIA et n’est pas un plan de création d’une application complète.

Ce didacticiel se concentre sur les tâches suivantes :

-   [Création d’un Gestionnaire de périphériques WIA](-wia-creating-a-wia-device-manager.md)
-   [Énumération des périphériques système](-wia-enumerating-system-devices.md)
-   [Lecture des propriétés de l’appareil](-wia-reading-device-properties.md)
-   [Création d’un appareil](-wia-creating-a-device.md)
-   [Énumération d’éléments](-wia-enumerating-items.md)
-   [Transfert de données image](-wia-transferring-image-data.md)
-   [Inscription pour les événements](-wia-registering-for-events.md)
-   [Capture d’une image continue à partir de streaming video](-wia-capturing-a-still-image-from-streaming-video.md)
