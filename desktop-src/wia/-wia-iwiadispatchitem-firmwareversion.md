# <a name="itemfirmwareversion-property"></a>Item. FirmwareVersion, propriété

Récupère la version du microprogramme de l’appareil. Cette propriété s’applique uniquement aux éléments de type périphérique (éléments racines). Lecture seule.

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```JScript
propVal = Item.FirmwareVersion
```



## <a name="property-value"></a>Valeur de la propriété

Chaîne qui reçoit la version du microprogramme.
