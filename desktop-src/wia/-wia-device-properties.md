# <a name="device-properties-wia"></a>Propriétés de l’appareil (WIA)

Les propriétés de l’appareil sont des propriétés qui contrôlent la fonction de l’appareil entier. Ces propriétés sont accessibles uniquement par le biais de l’élément racine de l’arborescence qui représente un périphérique. Il existe trois types de propriétés d’appareil : courant, scanneur et appareil photo.

Pour obtenir des informations de référence, consultez :

-   [**Constantes de propriété d’appareil courantes**](-wia-wiaitempropcommondevice.md)
-   [**Constantes de propriété de périphérique d’appareil photo**](-wia-wiaitempropcameradevice.md)
-   [**Constantes de propriété de périphérique de scanneur**](-wia-wiaitempropscannerdevice.md)
