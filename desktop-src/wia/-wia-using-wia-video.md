# <a name="using-wia-video"></a>Utilisation de la vidéo WIA

WIA fournit l’interface [**IWiaVideo**](/windows/desktop/api/Wiavideo/nn-wiavideo-iwiavideo) pour permettre aux applications d’afficher la vidéo en continu dans une fenêtre, de capturer des images fixes à partir du flux vidéo et d’enregistrer ces images en tant que fichiers JPEG.

les sections suivantes expliquent comment utiliser les fonctionnalités vidéo d’Acquisition d’images Windows (WIA) :

-   [Aperçu de la vidéo](-wia-previewing-video.md)
-   [Capture d’une image continue à partir d’un flux vidéo](-wia-capturing-a-still-image-from-a-video-stream.md)

> [!Note]  
> WIA ne prend pas en charge les périphériques vidéo dans Windows Server 2003, Windows Vista ou version ultérieure. pour ces versions du Windows, utilisez [DirectShow](/previous-versions//ms783323(v=vs.85)) pour acquérir des images à partir d’une vidéo.
