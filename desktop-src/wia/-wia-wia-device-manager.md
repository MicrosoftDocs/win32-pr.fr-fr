# <a name="wia-device-manager"></a>Gestionnaire de périphériques WIA

L’Gestionnaire de périphériques WIA (Windows Image Acquisition) est une extension de l’analyse d’événements image continue (STI). L’Gestionnaire de périphériques WIA fournit des objets, des méthodes et des interfaces pour les éléments suivants :

-   Installation d’appareils
-   Énumération des appareils
-   Interrogation des propriétés des appareils installés
-   Création d’objets appareil
-   Surveillance des événements des appareils
-   Acquisition d’images
-   Inscription des applications de destination.
