# <a name="wia-event-identifiers"></a>Identificateurs d’événements WIA

Les identificateurs d’événements WIA sont des identificateurs globaux uniques (GUID) définis par le système WIA (Windows Image Acquisition).

Une application WIA peut s’abonner à ces événements non persistants en utilisant [**IwiaDevMgr2 :: RegisterEventCallbackInterface**](-wia-iwiadevmgr2-registereventcallbackinterface.md) pour recevoir des notifications de changement d’état des appareils à partir de l’appareil WIA auquel la session d’application WIA en cours est connectée. Le mini-pilote WIA doit s’assurer que \_ l' \_ \_ État de gestion des documents DPS WIA \_ est correctement mis à jour avant que le pilote ne signale l’un de ces événements.

Voici les GUID d’événements WIA valides : 

| GUID                             | Signification                                                                                                                                                                                                                                                                                                                               |
|----------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| \_ \_ e/s d’annulation d’événement WIA \_           | L’application tente d’annuler un transfert de données.                                                                                                                                                                                                                                                                                   |
| \_couverture d’événement WIA \_ \_ fermée        | Toutes les couvertures du scanneur sont fermées. Si plusieurs couvertures de scanneur étaient ouvertes, le minipilote WIA doit signaler cet événement uniquement lorsque la dernière couverture a été fermée. Cette fonctionnalité est disponible à partir de Windows 8.                                                                                                                      |
| \_couverture d’événement WIA \_ \_ ouverte          | Un couvercle de scanneur a été ouvert par l’opérateur. L’appareil ne peut pas effectuer d’analyse pour l’instant. Le minipilote WIA doit signaler cet événement une seule fois si plusieurs couvertures de scanneur sont ouvertes successivement. Cette fonctionnalité est disponible à partir de Windows 8.                                                                                        |
| \_appareil d’événement WIA \_ \_ connecté    | Un périphérique matériel WIA a été connecté à l’ordinateur de l’utilisateur ou a démarré.                                                                                                                                                                                                                                                                |
| \_appareil d’événement WIA \_ \_ déconnecté | Un périphérique matériel WIA a été déconnecté de l’ordinateur de l’utilisateur ou s’est arrêté.                                                                                                                                                                                                                                                           |
| \_appareil d’événement WIA \_ \_ non \_ prêt   | L’appareil n’est pas à l’état prêt. \_ \_ \_ \_ Pour plus d’informations, l’application doit lire l’état de gestion des documents du DPS WIA. Cette fonctionnalité est disponible à partir de Windows 8.                                                                                                                                                         |
| \_appareil d’événements WIA \_ \_ prêt        | L’appareil est entré en état prêt. L’état non prêt précédent signalé avec l’appareil d' \_ événement \_ WIA \_ non \_ prêt a été effacé. Cette fonctionnalité est disponible à partir de Windows 8.                                                                                                                                                             |
| le \_ flux d’événements WIA est \_ \_ vidé      | L’opérateur a supprimé tous les documents du bac d’entrée du chargeur et le chargeur est vide. L’appareil ne peut pas effectuer d’analyse à partir du chargeur pour l’instant. Important : le minipilote ne doit pas signaler cet événement lorsque le chargeur devient vide à la suite d’un travail d’analyse exécuté par une application. Cette fonctionnalité est disponible à partir de Windows 8. |
| \_chargeur d’événements WIA \_ \_ chargé       | L’opérateur place les documents dans le bac d’entrée du chargeur. Le chargeur est chargé avec du papier. L’application peut lancer un travail d’analyse à partir du chargeur pour l’instant. Cette fonctionnalité est disponible à partir de Windows 8.                                                                                                                     |
| \_couvercle à plateau d’événement WIA \_ \_ \_ fermé | Le couvercle du plateau a été fermé par l’opérateur. L’application peut lancer un travail d’analyse à partir du plateau pour l’instant. Cette fonctionnalité est disponible à partir de Windows 8.                                                                                                                                                                 |
| \_couvercle à plateau d’événement WIA \_ \_ \_ ouvert   | Le capot du plateau a été ouvert par l’opérateur. L’appareil ne peut pas effectuer d’analyse à partir du plateau pour l’instant. Cette fonctionnalité est disponible à partir de Windows 8.                                                                                                                                                                                  |
| \_élément d’événement WIA \_ \_ créé        | Un élément a été ajouté à l’arborescence d’objets [**IWiaItem**](/windows/desktop/api/wia_xp/nn-wia_xp-iwiaitem) ou [**IWiaItem2**](-wia-iwiaitem2.md) pour un appareil WIA.                                                                                                                                                                                                   |
| \_élément d’événement WIA \_ \_ supprimé        | Un élément a été supprimé de l’arborescence d’objets [**IWiaItem**](/windows/desktop/api/wia_xp/nn-wia_xp-iwiaitem) ou [**IWiaItem2**](-wia-iwiaitem2.md) pour un appareil WIA.                                                                                                                                                                                               |
| reprise de l' \_ alimentation des événements WIA \_ \_        | L’alimentation a été restaurée sur l’appareil ou l’ordinateur.                                                                                                                                                                                                                                                                                    |
| interruption de l' \_ alimentation des événements WIA \_ \_       | L’alimentation a été suspendue sur l’appareil ou l’ordinateur.                                                                                                                                                                                                                                                                                   |
| IMAGE de l’E-mail d’analyse des \_ événements WIA \_ \_ \_   | Une analyse a été lancée avec l’intention d’envoyer par courrier électronique l’image résultante.                                                                                                                                                                                                                                                                   |
| \_image de \_ télécopie d’analyse des événements WIA \_ \_     | Une analyse a été lancée avec l’intention de télécopier l’image résultante.                                                                                                                                                                                                                                                                      |
| \_image de \_ film d’analyse des événements WIA \_ \_    | Une image a été scannée à partir d’un adaptateur de film.                                                                                                                                                                                                                                                                                             |
| IMAGE de l' \_ analyse des événements WIA \_ \_          | Une image a été analysée.                                                                                                                                                                                                                                                                                                                 |
| \_IMAGE2 d' \_ analyse des événements WIA \_         | Une image a été analysée. Cet événement permet aux scanneurs de plusieurs boutons d’analyse de diriger les boutons vers différentes applications.                                                                                                                                                                                                      |
| \_IMAGE3 d' \_ analyse des événements WIA \_         | Une image a été analysée. Cet événement permet aux scanneurs de plusieurs boutons d’analyse de diriger les boutons vers différentes applications.                                                                                                                                                                                                      |
| \_IMAGE4 d' \_ analyse des événements WIA \_         | Une image a été analysée. Cet événement permet aux scanneurs de plusieurs boutons d’analyse de diriger les boutons vers différentes applications.                                                                                                                                                                                                      |
| \_ \_ image OCR d’analyse des événements WIA \_ \_     | Une analyse a été lancée avec l’intention de convertir l’image résultante avec la reconnaissance optique de caractères (OCR).                                                                                                                                                                                                                         |
| IMAGE d’impression de l' \_ analyse des événements WIA \_ \_ \_   | Une analyse a été lancée avec l’intention d’imprimer l’image résultante.                                                                                                                                                                                                                                                                    |
| \_événement WIA \_ \_ proxy ICT           | Destiné exclusivement à un usage interne. Ne pas utiliser.                                                                                                                                                                                                                                                                                           |
| \_stockage des événements WIA \_ \_ créé     | Une unité de stockage, telle qu’une carte mémoire flash, a été ajoutée à l’appareil. L’application doit recréer l’arborescence d’éléments qui représente cet appareil.                                                                                                                                                                                            |
| \_stockage des événements WIA \_ \_ supprimé     | Une unité de stockage, telle qu’une carte mémoire flash, a été supprimée de l’appareil. L’application doit recréer l’arborescence d’éléments qui représente cet appareil.                                                                                                                                                                                        |
| \_arborescence d’événements WIA \_ \_ mise à jour        | L’arborescence d’éléments WIA a été mise à jour.                                                                                                                                                                                                                                                                                                        |
| \_volume d’événements WIA \_ \_ inséré     | Un dispositif de stockage, tel qu’une carte Flash, a été inséré dans l’appareil.                                                                                                                                                                                                                                                                 |
