# <a name="deviceinfoport-property"></a>DeviceInfo. port, propriété

Récupère le port auquel le périphérique matériel d’acquisition d’images Windows (WIA) est connecté.

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```JScript
propVal = DeviceInfo.Port
```



## <a name="property-value"></a>Valeur de la propriété

Chaîne qui reçoit le port.
