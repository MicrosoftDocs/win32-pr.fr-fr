# <a name="itempicturewidth-property"></a>Item. PictureWidth, propriété

Largeur, en pixels, des images produites par cet appareil photo numérique. S’applique uniquement aux appareils photo numériques. Lecture seule.

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```JScript
propVal = Item.PictureWidth
```



## <a name="property-value"></a>Valeur de la propriété

Variable qui reçoit la largeur.
