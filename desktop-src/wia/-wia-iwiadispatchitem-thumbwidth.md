# <a name="itemthumbwidth-property"></a>Item. ThumbWidth, propriété

Largeur, en pixels, de l’image miniature. Cette propriété retourne-1 si cet élément ne prend pas en charge les miniatures. Lecture seule.

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```JScript
propVal = Item.ThumbWidth
```



## <a name="property-value"></a>Valeur de la propriété

Variable qui reçoit la largeur.
