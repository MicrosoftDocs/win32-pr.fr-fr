# <a name="wia-property-constants"></a>Constantes de propriété WIA

Windows les constantes de propriété d’Acquisition d’images (WIA) incluent les groupes de constantes suivants pris en charge par Windows Millennium edition (Windows Me), Windows XP et versions ultérieures.

-   [**Constantes de propriété d’informations sur l’appareil**](-wia-wiadeviceinfoprop.md)
-   [**Constantes de propriété d’appareil courantes**](-wia-wiaitempropcommondevice.md)
-   [**Constantes de propriété de périphérique de scanneur**](-wia-wiaitempropscannerdevice.md)
-   [**Constantes de propriété d’élément WIA courantes**](-wia-wiaitempropcommonitem.md)
-   [**Constantes de propriété d’élément WIA du scanneur**](-wia-wiaitempropscanneritem.md)

les groupes de constantes suivants sont pris en charge par Windows Me, Windows XP et Windows Server 2003 ; mais ne sont pas pris en charge par Windows Vista et versions ultérieures.

-   [**Constantes de propriété de périphérique d’appareil photo**](-wia-wiaitempropcameradevice.md)
-   [**Constantes de propriété d’élément WIA de l’appareil photo**](-wia-wiaitempropcameraitem.md)

le groupe de constantes suivant est pris en charge par Windows Me et Windows XP ; toutefois, n’est pas pris en charge par Windows Server 2003 et versions ultérieures.

-   [**Constantes de propriété d’appareil vidéo WIA**](-wia-wiaitempropvideodevice.md)
