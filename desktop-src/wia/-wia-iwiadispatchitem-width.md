# <a name="itemwidth-property"></a>Item. Width, propriété

Largeur, en pixels, de l’élément. Lecture seule.

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```JScript
propVal = Item.Width
```



## <a name="property-value"></a>Valeur de la propriété

Variable qui reçoit la largeur.

## <a name="remarks"></a>Notes

Si l’élément est un appareil photo numérique, la propriété **Width** représente la largeur des images générées par l’appareil photo. Si l’élément est un scanneur, cette propriété représente la largeur de la vitre de numérisation. Si l’élément est une image, cette propriété représente la largeur réelle de l’image.
