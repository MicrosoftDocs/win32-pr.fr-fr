# <a name="sti-compatibility"></a>Compatibilité STI

L’architecture WIA (Windows Image Acquisition) est basée sur la fondation établie par l’architecture d’image continue (STI). Les pilotes de périphérique WIA sont conformes au modèle de pilote de mode utilisateur (USD) de STI. Alors que l’objectif de l’USD d’origine est de prendre en charge les sources de données TWAIN et d’autres API, les pilotes WIA prennent en charge la nouvelle interface d’application WIA.
