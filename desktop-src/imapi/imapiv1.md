# <a name="imapiv1-support"></a>Support IMAPIv1

L’API Image Mastering version 1 (IMAPIv1) permet à une étape de l’application et de graver une image audio ou de données simple sur des périphériques CD-R et CD-RW. L’API prend en charge les formats Joliet et ISO 9660 pour les disques de données et audio Redbook.

> [!Note]  
> IMAPi a été entièrement repensé pour la version 2,0. La plupart des fonctionnalités de la version 1,0 sont disponibles dans la version 2,0. Ceux qui écrivent des applications de mastérisation d’image ou qui effectuent le développement de nouveaux appareils et formats sont encouragés à utiliser la [version 2,0](using-imapi.md) au lieu de la version 1,0. Si vous exécutez une application version 1,0 sur Windows Vista, les interfaces de la version 1,0 convertissent ces appels en appels de version 2,0 appropriés pour obtenir des fonctionnalités identiques.

 

Pour plus d’informations sur l’utilisation de la version 1,0 sur Windows XP, consultez les rubriques suivantes :

-   [À propos de l’API de mastérisation d’image](about-the-image-mastering-api.md)
-   [Utilisation de l’API de mastérisation d’image](using-the-image-mastering-api.md)
-   [Référence IMAPIv1](imapiv1-reference.md)
