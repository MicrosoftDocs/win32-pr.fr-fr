# <a name="msftstreamzero-object"></a>Objet MsftStreamZero

Utilisez cet objet pour créer un flux de données en lecture seule dont les données sont initialisées à des zéros. Vous devez appeler la **méthode Set pour définir la taille** demandée pour le flux.

## <a name="members"></a>Membres

L’objet **MsftStreamZero** ne définit aucun membre.
