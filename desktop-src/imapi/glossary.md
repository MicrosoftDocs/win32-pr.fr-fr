# <a name="glossary-imapi"></a>Glossaire (IMAPi)

<dl> <dt>

<span id="imapi.imapi2_glossary_ANSI"></span><span id="imapi.imapi2_glossary_ansi"></span><span id="IMAPI.IMAPI2_GLOSSARY_ANSI"></span>**CARACTÈRES**
</dt> <dd>

American National Standards Institute.

</dd> <dt>

<span id="imapi.imapi2_glossary_BurnEngine"></span><span id="imapi.imapi2_glossary_burnengine"></span><span id="IMAPI.IMAPI2_GLOSSARY_BURNENGINE"></span>**Moteur de gravure**
</dt> <dd>

Composant qui lit des données à partir d’un flux de données fourni (par exemple, IStream ou IO. Stream) et écrit ces données sur un périphérique CD/DVD. Le composant doit prendre en compte la mémoire tampon disponible sur le lecteur, mais ne contient aucune connaissance du format des données, préparer le lecteur à l’écriture ou informer le lecteur que le processus d’écriture est terminé.

</dd> <dt>

<span id="imapi.imapi2_glossary_CDDVDDevice"></span><span id="imapi.imapi2_glossary_cddvddevice"></span><span id="IMAPI.IMAPI2_GLOSSARY_CDDVDDEVICE"></span>**Périphérique CD/DVD**
</dt> <dd>

Appareil qui se présente comme un appareil « type 5 ». Cela comprend les lecteurs CD-ROM, DVD-ROM et la plupart des lecteurs CD et DVD Writer. En outre, certains lecteurs dotés de supports magnétiques (tels que IOMEGA REV) s’affichent eux-mêmes en tant que périphériques « type 5 » et s’affichent en tant que périphériques CD-ROM dans Windows.

</dd> <dt>

<span id="imapi.imapi2_glossary_CDG"></span><span id="imapi.imapi2_glossary_cdg"></span><span id="IMAPI.IMAPI2_GLOSSARY_CDG"></span>**CD-V** 
</dt> <dd>

Acronyme de CD Graphics, technologie exclusive aux CD audio, qui permet d’ajouter des graphiques à faible vitesse de transmission à une piste audio. Cette technologie est le plus souvent utilisée sur les CD utilisés pour le karaoké.

</dd> <dt>

<span id="imapi.imapi2_glossary_CD_R"></span><span id="imapi.imapi2_glossary_cd_r"></span><span id="IMAPI.IMAPI2_GLOSSARY_CD_R"></span>**CD-R**
</dt> <dd>

Un CD-ROM en écriture unique.

</dd> <dt>

<span id="imapi.imapi2_glossary_CD_RandRW"></span><span id="imapi.imapi2_glossary_cd_randrw"></span><span id="IMAPI.IMAPI2_GLOSSARY_CD_RANDRW"></span>**CD-R/RW**
</dt> <dd>

Disque compact CD-R ou CD-RW.

</dd> <dt>

<span id="imapi.imapi2_glossary_CD_ROM"></span><span id="imapi.imapi2_glossary_cd_rom"></span><span id="IMAPI.IMAPI2_GLOSSARY_CD_ROM"></span>**CD-ROM**
</dt> <dd>

Un CD-ROM en lecture seule.

</dd> <dt>

<span id="imapi.imapi2_glossary_CD_RW"></span><span id="imapi.imapi2_glossary_cd_rw"></span><span id="IMAPI.IMAPI2_GLOSSARY_CD_RW"></span>**CD-RW**
</dt> <dd>

Un CD réinscriptible.

</dd> <dt>

<span id="imapi.imapi2_glossaryCDText"></span><span id="imapi.imapi2_glossarycdtext"></span><span id="IMAPI.IMAPI2_GLOSSARYCDTEXT"></span>**CD-texte** 
</dt> <dd>

Technologie exclusive aux CD audio, qui permet de stocker des informations de faible taux de bits, telles que le nom de l’artiste et le nom de la chanson, sur le support physique. Les applications ou périphériques qui lisent le CD résultant peuvent ensuite récupérer ces informations, si elles existent, pour les afficher à l’utilisateur.

</dd> <dt>

<span id="imapi.imapi2_glossary_Disc"></span><span id="imapi.imapi2_glossary_disc"></span><span id="IMAPI.IMAPI2_GLOSSARY_DISC"></span>**ROM**
</dt> <dd>

Terme générique pour le média optique rond, tel qu’un CD-ROM ou un DVD.

</dd> <dt>

<span id="imapi.imapi2_glossary_DiscAtOnce"></span><span id="imapi.imapi2_glossary_discatonce"></span><span id="IMAPI.IMAPI2_GLOSSARY_DISCATONCE"></span>**Disque à la fois** 
</dt> <dd>

Méthode d’écriture sur un support CD où tout le contenu de l’image de gravure est écrit sur le disque dans un flux continu. En outre, les données non accessibles, telles que le complément et les LEADOUT, sont incluses dans l’image de gravure.

</dd> <dt>

<span id="imapi.imapi2_glossary_Disk"></span><span id="imapi.imapi2_glossary_disk"></span><span id="IMAPI.IMAPI2_GLOSSARY_DISK"></span>**Libérer**
</dt> <dd>

Terme générique pour les supports magnétiques ronds, tels qu’une disquette ou un disque dur.

</dd> <dt>

<span id="imapi.imapi2_glossary_Dispatch"></span><span id="imapi.imapi2_glossary_dispatch"></span><span id="IMAPI.IMAPI2_GLOSSARY_DISPATCH"></span>**Distribution** 
</dt> <dd>

Désignation d’interface COM dans laquelle une interface donnée prend en charge la méthode **IDispatch** d’appel d’une fonction. Pour plus d’informations sur cette rubrique COM avancée, consultez MSDN.

</dd> <dt>

<span id="imapi.imapi2_glossary_Dual"></span><span id="imapi.imapi2_glossary_dual"></span><span id="IMAPI.IMAPI2_GLOSSARY_DUAL"></span>**Double** 
</dt> <dd>

Désignation d’une interface COM où une interface donnée prend en charge les méthodes VTable et **IDispatch** pour appeler une fonction. Pour plus d’informations sur ces rubriques avancées relatives à COM, consultez MSDN.

</dd> <dt>

<span id="imapi.imapi2_glossary_DVD"></span><span id="imapi.imapi2_glossary_dvd"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVD"></span>**DVD**
</dt> <dd>

Un disque numérique polyvalent. à l’origine, cette abréviation signifiait « Digital Video Disc ».

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDPlusR"></span><span id="imapi.imapi2_glossary_dvdplusr"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDPLUSR"></span>**DVD + R**
</dt> <dd>

Un DVD en écriture unique. L’acronyme est prononcé « DVD plus R ».

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDPlusRDL"></span><span id="imapi.imapi2_glossary_dvdplusrdl"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDPLUSRDL"></span>**DVD + R DL**
</dt> <dd>

Un disque DVD + R double couche. Ce type de disque est également appelé « DVD + R double couche ».

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDPlusRRW"></span><span id="imapi.imapi2_glossary_dvdplusrrw"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDPLUSRRW"></span>**DVD + R/RW**
</dt> <dd>

Un disque DVD + R et un disque DVD + RW.

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDPlusRW"></span><span id="imapi.imapi2_glossary_dvdplusrw"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDPLUSRW"></span>**DVD + RW**
</dt> <dd>

Un DVD réinscriptible. L’acronyme est prononcé « DVD plus RW ».

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDPlusDashR"></span><span id="imapi.imapi2_glossary_dvdplusdashr"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDPLUSDASHR"></span>**DVD ± R**
</dt> <dd>

Un disque DVD + R et un disque DVD-R.

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDPlusDashRRW"></span><span id="imapi.imapi2_glossary_dvdplusdashrrw"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDPLUSDASHRRW"></span>**DVD ± R/RW**
</dt> <dd>

Un disque DVD + R, un disque DVD-R, un disque DVD + RW et un disque DVD-RW.

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDPlusDashRW"></span><span id="imapi.imapi2_glossary_dvdplusdashrw"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDPLUSDASHRW"></span>**DVD ± RW**
</dt> <dd>

Un disque DVD + RW et un support DVD + R

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDDashR"></span><span id="imapi.imapi2_glossary_dvddashr"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDDASHR"></span>**DVD-R**
</dt> <dd>

Un DVD en écriture unique. L’acronyme est prononcé « DVD Dash R ».

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDDashRDL"></span><span id="imapi.imapi2_glossary_dvddashrdl"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDDASHRDL"></span>**DVD-R DL**
</dt> <dd>

Un support DVD-R double couche.

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDRAM"></span><span id="imapi.imapi2_glossary_dvdram"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDRAM"></span>**DVD-RAM**
</dt> <dd>

Un DVD inscriptible de manière aléatoire. Ce type de média prend en charge la gestion des défauts matériels qui identifie et évite d’utiliser des pistes endommagées. Les lecteurs prenant en charge ce format sont plus répandus au Japon qu’aux États-Unis.

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDROM"></span><span id="imapi.imapi2_glossary_dvdrom"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDROM"></span>**DVD-ROM**
</dt> <dd>

Un DVD en lecture seule.

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDDashRRW"></span><span id="imapi.imapi2_glossary_dvddashrrw"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDDASHRRW"></span>**DVD-R/RW**
</dt> <dd>

Un disque DVD-R et un disque DVD-RW.

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDDashRW"></span><span id="imapi.imapi2_glossary_dvddashrw"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDDASHRW"></span>**DVD-RW**
</dt> <dd>

Un DVD réinscriptible. L’acronyme est prononcé « DVD Dash RW ».

</dd> <dt>

<span id="imapi.imapi2_glossary_ECMA"></span><span id="imapi.imapi2_glossary_ecma"></span><span id="IMAPI.IMAPI2_GLOSSARY_ECMA"></span>**ECMA** 
</dt> <dd>

L’Association des fabricants d’ordinateurs européens est une association de l’industrie fondée dans 1961 qui est consacrée à la normalisation des systèmes d’information et de communication. L’ECMA modifie les normes et les rapports techniques. Toutes les publications ECMA sont disponibles gratuitement.

</dd> <dt>

<span id="imapi.imapi2_glossary_IMAPI"></span><span id="imapi.imapi2_glossary_imapi"></span><span id="IMAPI.IMAPI2_GLOSSARY_IMAPI"></span>**IMAPI** 
</dt> <dd>

API de mastérisation d’images qui permet de créer et d’écrire des images masterées (par opposition à un FS Live) sur des périphériques CD/DVD. Ce terme s’applique de façon générique à toutes les versions de l’API de contrôle d’image.

</dd> <dt>

<span id="imapi.imapi2_glossary_IMAPIv1"></span><span id="imapi.imapi2_glossary_imapiv1"></span><span id="IMAPI.IMAPI2_GLOSSARY_IMAPIV1"></span>**IMAPIv1** 
</dt> <dd>

la version d’imapi exposée pour la première fois dans Windows XP. IMAPi 1,0 prend en charge les supports CD-R et CD-RW.

</dd> <dt>

<span id="imapi.imapi2_glossary_IMAPIv2"></span><span id="imapi.imapi2_glossary_imapiv2"></span><span id="IMAPI.IMAPI2_GLOSSARY_IMAPIV2"></span>**IMAPIv2** 
</dt> <dd>

la deuxième version d’imapi, publiée avec Windows Vista.

</dd> <dt>

<span id="imapi.imapi2_glossary_INCITS"></span><span id="imapi.imapi2_glossary_incits"></span><span id="IMAPI.IMAPI2_GLOSSARY_INCITS"></span>**INCITS**
</dt> <dd>

Le Comité InterNational des normes des technologies de l’information, une organisation agréée et approuvée par ANSI.

</dd> <dt>

<span id="imapi.imapi2_glossary_ISO9660"></span><span id="imapi.imapi2_glossary_iso9660"></span><span id="IMAPI.IMAPI2_GLOSSARY_ISO9660"></span>**ISO9660**
</dt> <dd>

Système de fichiers conçu pour la première génération de CD. Ce système de fichiers présente des limitations de tri et de nom de fichier importantes.

</dd> <dt>

<span id="imapi.imapi2_glossary_ISOImage"></span><span id="imapi.imapi2_glossary_isoimage"></span><span id="IMAPI.IMAPI2_GLOSSARY_ISOIMAGE"></span>**Image ISO**
</dt> <dd>

Nom commun d’une image secteur par secteur de la zone de données entière d’un CD ou d’un DVD de données. Le nom a augmenté de l’utilisation courante de ISO9660 en tant que système de fichiers pour CD.

</dd> <dt>

<span id="imapi.imapi2_glossary_Joliet"></span><span id="imapi.imapi2_glossary_joliet"></span><span id="IMAPI.IMAPI2_GLOSSARY_JOLIET"></span>**Joliet**
</dt> <dd>

Ensemble d’extensions standardisées de ISO9660 qui prend en charge les noms de fichiers UNICODE jusqu’à 64 caractères. Pour plus d’informations, consultez la spécification ECMA numéro 119.

</dd> <dt>

<span id="imapi.imapi2_glossary_MMC"></span><span id="imapi.imapi2_glossary_mmc"></span><span id="IMAPI.IMAPI2_GLOSSARY_MMC"></span>**INTÉGR**
</dt> <dd>

Jeu de commandes multimédia qui est défini par le groupe de travail T10 de l’INCITS. La console MMC définit des commandes de paquets pour les périphériques CD et DVD. Les brouillons de la console MMC sont disponibles sur le site Web de la version T10.

</dd> <dt>

<span id="imapi.imapi2_glossary_MtFuji"></span><span id="imapi.imapi2_glossary_mtfuji"></span><span id="IMAPI.IMAPI2_GLOSSARY_MTFUJI"></span>**Mt. Fuji**
</dt> <dd>

Ensemble de commandes permettant de contrôler les appareils multimédias, comme le jeu de commandes MMC. Certains éléments (tels que les fonctionnalités ou les types de média) sont décrits dans le Mt. Les documents Fuji, tandis que d’autres sont décrits en premier dans les documents MMC. Les brouillons de commande actuels du jeu de commandes sont disponibles dans [ftp://ftp.AVC-Pioneer.com](ftp://ftp.avc-pioneer.com/).

</dd> <dt>

<span id="imapi.imapi2_glossary_SCSI"></span><span id="imapi.imapi2_glossary_scsi"></span><span id="IMAPI.IMAPI2_GLOSSARY_SCSI"></span>**SCSI**
</dt> <dd>

Interface du petit système informatique.

</dd> <dt>

<span id="imapi.imapi2_glossary_SCSICommand"></span><span id="imapi.imapi2_glossary_scsicommand"></span><span id="IMAPI.IMAPI2_GLOSSARY_SCSICOMMAND"></span>**Commande SCSI**
</dt> <dd>

Une série de commandes de paquets comme celles définies dans le document MMC. Ces commandes peuvent être envoyées à l’aide d’un mécanisme de transport approprié, tel que SCSI, ATAPI, USB ou IEEE1394.

</dd> <dt>

<span id="imapi.imapi2_glossary_SessonAtOnce"></span><span id="imapi.imapi2_glossary_sessonatonce"></span><span id="IMAPI.IMAPI2_GLOSSARY_SESSONATONCE"></span>**Session à la fois**
</dt> <dd>

Méthode de création d’un disque CD plus. Une session d’écriture enregistre les pistes audio sur le disque et une deuxième session écrit une piste CD-XA unique sur le disque et ferme le disque.

</dd> <dt>

<span id="imapi.imapi2_glossary_TrackAtOnce"></span><span id="imapi.imapi2_glossary_trackatonce"></span><span id="IMAPI.IMAPI2_GLOSSARY_TRACKATONCE"></span>**Suivi à la fois**
</dt> <dd>

Méthode d’écriture sur un support CD où les données d’une seule piste sont écrites en même temps. Lorsque toutes les pistes sont écrites, la session est fermée comme dernière étape consistant à écrire les zones d’interconnexion et de sortie du disque.

</dd> <dt>

<span id="imapi.imapi2_glossary_UDF"></span><span id="imapi.imapi2_glossary_udf"></span><span id="IMAPI.IMAPI2_GLOSSARY_UDF"></span>**UDF (Universal Disk Format)**
</dt> <dd>

ce système de fichiers est défini par OSTA (Association de technologie de Stockage optique) qui prend en charge les disques à session unique et à plusieurs sessions.

</dd> <dt>

<span id="imapi.imapi2_glossary_VTable"></span><span id="imapi.imapi2_glossary_vtable"></span><span id="IMAPI.IMAPI2_GLOSSARY_VTABLE"></span>**VTable** 
</dt> <dd>

Désignation d’une interface COM où une interface donnée prend en charge la méthode vtable d’appel d’une fonction. Pour plus d’informations sur cette rubrique COM avancée, consultez MSDN.

</dd> </dl>
