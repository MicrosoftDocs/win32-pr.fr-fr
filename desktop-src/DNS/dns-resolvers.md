# <a name="dns-resolvers"></a>Programmes de résolution DNS

le programme de [*résolution*](r-gly.md) est un composant logiciel fourni dans la pile Windows 2000 TCP/IP (Transmission control protocol/Internet protocol) qui communique avec les serveurs DNS pour créer et résoudre des requêtes de résolution de noms. Le bundle est également disponible dans n’importe quel client non-Microsoft qui ne prend pas en charge le DNS. Les programmes de résolution existent sur les serveurs DNS et DNS.
