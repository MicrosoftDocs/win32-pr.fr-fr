# <a name="dns-resolvers"></a>Programmes de résolution DNS

Le programme de [*résolution*](r-gly.md) est un composant logiciel intégré à la pile TCP/IP (Transmission Control Protocol/Internet Protocol) Windows 2000 qui communique avec les serveurs DNS pour créer et résoudre des requêtes de résolution de noms. Le bundle est également disponible dans n’importe quel client non-Microsoft qui ne prend pas en charge le DNS. Les programmes de résolution existent sur les serveurs DNS et DNS.
