# <a name="getdistinguishedname-method-of-the-microsoftdns_roothints-class"></a>Méthode GetDistinguishedName de la \_ classe MicrosoftDNS RootHints

La méthode **GetDistinguishedName** récupère le nom unique de la zone.

## <a name="syntax"></a>Syntaxe


```mof
string GetDistinguishedName();
```



## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.

## <a name="return-value"></a>Valeur retournée

Retourne le nom de la zone.
