# <a name="using-the-dns-wmi-provider"></a>Utilisation du fournisseur WMI DNS

Le fournisseur WMI DNS définit des [*classes WMI CIM*](c-gly.md) qui permettent à votre application de contrôler un serveur DNS. Les sections suivantes fournissent des informations générales qui peuvent vous aider à prendre en main le fournisseur WMI DNS. Des liens sont fournis vers des exemples de scripts qui montrent comment utiliser le fournisseur WMI DNS.

pour plus d’informations sur les opérations wmi de base, reportez-vous à Windows Management Instrumentation (wmi) dans le kit de développement logiciel (SDK) de la plateforme.
