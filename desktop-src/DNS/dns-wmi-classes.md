# <a name="dns-wmi-classes"></a>Classes WMI DNS

Le fournisseur WMI DNS définit les classes WMI DNS suivantes.



| Classe WMI de Microsoft DNS                                                                               | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
|-------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**\_Serveur MicrosoftDNS**](microsoftdns-server.md)                                                   | Décrit un serveur DNS. Chaque instance de cette classe peut être associée à une instance de la classe de [**\_ cache MicrosoftDNS**](microsoftdns-cache.md), à une instance de la classe [**MicrosoftDNS \_ RootHints**](microsoftdns-roothints.md)et à plusieurs instances de la [**\_ zone**](microsoftdns-zone.md)de la classe MicrosoftDNS.                                                                                                                                                                                                                                                                                                                                                                                       |
| [**\_Domaine MicrosoftDNS**](microsoftdns-domain.md)                                                   | Représente un domaine dans une arborescence de hiérarchie DNS.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| [**\_Zone MicrosoftDNS**](microsoftdns-zone.md)                                                       | Décrit une zone DNS. Chaque instance de la [**\_ zone**](microsoftdns-zone.md) de la classe MicrosoftDNS doit être assignée à un seul serveur DNS. Les zones peuvent être associées à plusieurs instances des classes [**MicrosoftDNS \_ Domain**](microsoftdns-domain.md) et [**MicrosoftDNS \_ ResourceRecord**](microsoftdns-resourcerecord.md).                                                                                                                                                                                                                                                                                                                                                                    |
| [**\_Cache MicrosoftDNS**](microsoftdns-cache.md)                                                     | Décrit un cache existant sur un serveur DNS (ne pas confondre avec un fichier cache qui contient des indications de [*racine*](r-gly.md) ). Cette classe simplifie la visualisation de la relation contenant-contenu des objets DNS, plutôt que la représentation d’un objet réel. La classe, [**MicrosoftDNS \_ cache**](microsoftdns-cache.md), est un conteneur pour les [*enregistrements de ressource*](r-gly.md) mis en cache par le serveur DNS. Chaque instance du **\_ cache** de la classe MicrosoftDNS doit être affectée à un seul serveur DNS. Il peut être associé à plusieurs instances du [**\_ domaine MicrosoftDNS**](microsoftdns-domain.md) et [**à \_ MicrosoftDNS ResourceRecord**](microsoftdns-resourcerecord.md). |
| [**MicrosoftDNS \_ RootHints**](microsoftdns-roothints.md)                                             | Décrit le RootHints stocké dans un fichier cache sur un serveur DNS. Cette classe simplifie la visualisation de la relation contenant-contenu des objets DNS, plutôt que la représentation d’un objet réel. La classe [**MicrosoftDNS \_ RootHints**](microsoftdns-roothints.md) est un conteneur pour les enregistrements de ressource stockés par le serveur DNS dans un fichier cache. Chaque instance de la classe **MicrosoftDNS \_ RootHints** doit être affectée à un seul serveur DNS. Il peut être associé à plusieurs instances de la [**classe \_ ResourceRecord MicrosoftDNS**](microsoftdns-resourcerecord.md) .                                                                                                                                               |

| [**MicrosoftDNS \_ DomainDomainContainment**](microsoftdns-domaindomaincontainment.md)                 | Chaque instance de la [**classe \_ DomainDomainContainment MicrosoftDNS**](microsoftdns-domaindomaincontainment.md) peut contenir plusieurs autres instances du [**\_ domaine MicrosoftDNS**](microsoftdns-domain.md).                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
