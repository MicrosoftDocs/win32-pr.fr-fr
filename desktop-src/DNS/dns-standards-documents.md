# <a name="dns-standards-documents"></a>Documents de normes DNS

Le DNS (Domain Name System) est un protocole ouvert. Ainsi, les efforts de collaboration du secteur dans son ensemble garantissent que son implémentation sur différents systèmes ne limite pas l’interopérabilité. Le corps de normes qui supervise ces recommandations est l’IETF (Internet Engineering Task Force). Voici les demandes RFC (Request for Comments) de l’IETF. Pour plus d’informations sur l’un de ces documents, visitez [www.ietf.org](https://www.ietf.org/).

## <a name="dns-rfcs"></a>RFC DNS

-   [RFC 1034](https://www.ietf.org/rfc/rfc1034.txt): noms de domaine-concepts et installations
-   [RFC 1035](https://www.ietf.org/rfc/rfc1035.txt): Domain Names-Implementation and Specification
-   [RFC 1122](https://www.ietf.org/rfc/rfc1122.txt): configuration requise pour les hôtes Internet-couches de communication
-   [RFC 1123](https://www.ietf.org/rfc/rfc1123.txt): configuration requise pour les hôtes Internet-application et support
-   [RFC 1876](https://www.ietf.org/rfc/rfc1876.txt): un moyen d’exprimer des informations d’emplacement dans le système de noms de domaine
-   [RFC 1995](https://www.ietf.org/rfc/rfc1995.txt): transfert de zone INCRÉMENTIEL dans DNS
-   [RFC 1996](https://www.ietf.org/rfc/rfc1996.txt): mécanisme de notification d’invite des modifications de zone (DNS Notify)
-   [RFC 2136](https://www.ietf.org/rfc/rfc2136.txt): mises à jour dynamiques dans le système de noms de domaine (mise à jour DNS)
-   [RFC 2181](https://www.ietf.org/rfc/rfc2181.txt): clarifications de la spécification DNS
-   [RFC 2308](https://www.ietf.org/rfc/rfc2308.txt): mise en cache négative des requêtes DNS (DNS NCACHE)
-   [RFC 2535](https://www.ietf.org/rfc/rfc2535.txt): extensions de sécurité DNS (Domain Name System)
-   [RFC 2782](https://www.ietf.org/rfc/rfc2782.txt): un enregistrement de ressource DNS pour spécifier l’emplacement des services (DNS SRV)
-   [RFC 2845](https://www.ietf.org/rfc/rfc2845.txt): authentification par clé secrète pour DNS (TSIG)
-   [RFC 2915](https://www.ietf.org/rfc/rfc2915.txt): enregistrement de ressource DNS du pointeur d’autorité de nommage (NAPTR)
-   [RFC 2930](https://www.ietf.org/rfc/rfc2930.txt): établissement de la clé secrète pour DNS (TKey RR)
-   [RFC 2931](https://www.ietf.org/rfc/rfc2931.txt): signatures DNS des demandes et des transactions (SIG (0) s)
-   [RFC 3110](https://www.ietf.org/rfc/rfc3110.txt): RSA/SHA-1 sigs et clés RSA dans le système DNS (Domain Name System)
-   [RFC 3445](https://www.ietf.org/rfc/rfc3445.txt): limitation de l’étendue de l’enregistrement de ressource de clé (RR)
-   [RFC 3596](https://www.ietf.org/rfc/rfc3596.txt): extensions DNS pour prendre en charge IP version 6
-   [RFC 3645](https://www.ietf.org/rfc/rfc3645.txt): algorithme Generic Security service pour l’authentification de la transaction de clé secrète pour DNS (GSS-TSIG)
