# <a name="getdistinguishedname-method-of-the-microsoftdns_server-class"></a>Méthode GetDistinguishedName de la \_ classe de serveur MicrosoftDNS

La méthode **GetDistinguishedName** récupère le nom unique du service d’annuaire pour la zone.

## <a name="syntax"></a>Syntaxe


```mof
string GetDistinguishedName();
```



## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.

## <a name="return-value"></a>Valeur retournée

Retourne le nom unique des services de domaine pour la zone.
