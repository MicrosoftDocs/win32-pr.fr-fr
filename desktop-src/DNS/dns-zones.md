# <a name="dns-zones"></a>Zones DNS

Une zone DNS est un ensemble de fichiers ou d’enregistrements (plus précisément, une base de données d’entrées d’enregistrement de ressource) qui correspond à une partie de l’espace de noms hiérarchique DNS. Les zones DNS servent à découper les serveurs DNS qui sont responsables (faisant autorité) pour la résolution des requêtes de résolution de noms pour une section donnée de la hiérarchie DNS. Les zones DNS diffèrent de la structure de domaine de la manière suivante : les zones peuvent être composées d’un ou plusieurs domaines DNS. Une zone de l’arborescence de domaine gadgets.widgets.microsoft.com peut faire autorité pour les domaines gadgets et widgets. En d’autres termes, il n’est pas nécessaire que les zones DNS aient une relation un-à-un avec les domaines DNS.
