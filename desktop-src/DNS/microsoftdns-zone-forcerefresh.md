# <a name="forcerefresh-method-of-the-microsoftdns_zone-class"></a>Méthode ForceRefresh de la \_ classe de zone MicrosoftDNS

> [!NOTE]
> Cet article contient des références au terme serveur maître, un terme que Microsoft n’utilise plus. Lorsque le terme sera supprimé du logiciel, nous le supprimerons de cet article.

La méthode **ForceRefresh** force une mise à jour du serveur DNS secondaire à partir du serveur maître.

## <a name="syntax"></a>Syntaxe


```mof
void ForceRefresh();
```



## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.

## <a name="return-value"></a>Valeur retournée

Cette méthode ne retourne pas de valeur.
