# <a name="dns-concepts"></a>Concepts DNS

Il existe quelques concepts de base sur lesquels fonctionne la structure fonctionnelle du système DNS. Ces concepts de base vont des définitions d’espace de noms à la délimitation des limites de responsabilité de résolution de noms. La compréhension de ces concepts vous permet d’obtenir une bonne connaissance du DNS. Les concepts DNS suivants sont définis et expliqués dans cette section :

-   [Espace de noms](name-space.md)
-   [Domaines DNS](dns-domains.md)
-   [Zones DNS](dns-zones.md)
-   [Causes](root.md)
-   [Résolution de noms](name-resolution.md)
-   [Serveurs DNS](dns-servers.md)
-   [Programmes de résolution DNS](dns-resolvers.md)
-   [Enregistrements de ressources](resource-records.md)
