# <a name="clearcache-method-of-the-microsoftdns_cache-class"></a>Méthode ClearCache de la \_ classe de cache MicrosoftDNS

La méthode **ClearCache** efface le cache du serveur DNS des enregistrements de ressources.

## <a name="syntax"></a>Syntaxe


```mof
void ClearCache();
```



## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.

## <a name="return-value"></a>Valeur retournée

Cette méthode ne retourne pas de valeur.
