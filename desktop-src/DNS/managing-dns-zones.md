# <a name="managing-dns-zones"></a>Gestion des Zones DNS

Une zone DNS est une base de données d’entrées RR qui correspond à une partie de l’espace de noms hiérarchique DNS. Les zones DNS servent à découper les serveurs DNS faisant autorité pour la résolution des requêtes de résolution de noms pour une section donnée de la hiérarchie DNS.

## <a name="administration-steps"></a>Étapes d’administration

Le fournisseur WMI DNS permet d’administrer des zones DNS à partir du serveur DNS faisant autorité, ou à partir d’hôtes distants. Les étapes générales nécessaires à l’administration d’une zone à l’aide du fournisseur WMI DNS sont les suivantes :

-   Connexion au fournisseur WMI DNS
-   Obtention d’une instance de zone
-   Énumération ou modification d’une propriété de zone, ou utilisation d’une méthode
