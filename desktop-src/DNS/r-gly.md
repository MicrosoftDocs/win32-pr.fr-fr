<dl> <dt>

<span id="_dns_relative_distinguished_name_gly"></span><span id="_DNS_RELATIVE_DISTINGUISHED_NAME_GLY"></span>**nom unique relatif**
</dt> <dd>

Nom qui identifie de façon unique un hôte dans son propre domaine, mais pas dans l’ensemble de la hiérarchie DNS. Par exemple, server1 est un nom unique relatif, tandis que server1.products.Microsoft.com est un nom de domaine complet (FQDN).

</dd> <dt>

<span id="_dns_resolver_gly"></span><span id="_DNS_RESOLVER_GLY"></span>**outil**
</dt> <dd>

Composant logiciel intégré à la pile Windows Transmission Control Protocol/Internet Protocol (TCP/IP) qui communique avec les serveurs DNS pour créer et résoudre des requêtes de résolution de noms.

</dd> <dt>

<span id="_dns_resource_record_rr__gly"></span><span id="_DNS_RESOURCE_RECORD_RR__GLY"></span>**enregistrement de ressource (RR)**
</dt> <dd>

Unité d’information dans le DNS. Il existe de nombreux types d’enregistrements de ressources.

</dd> <dt>

<span id="_dns_root_gly"></span><span id="_DNS_ROOT_GLY"></span>**causes**
</dt> <dd>

Niveau le plus élevé dans une hiérarchie donnée.

</dd> </dl>
