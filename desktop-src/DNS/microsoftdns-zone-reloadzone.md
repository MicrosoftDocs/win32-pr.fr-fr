# <a name="reloadzone-method-of-the-microsoftdns_zone-class"></a>Méthode ReloadZone de la \_ classe de zone MicrosoftDNS

La méthode **ReloadZone** recharge la zone DNS à partir de sa base de données.

## <a name="syntax"></a>Syntaxe


```mof
void ReloadZone();
```



## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.

## <a name="return-value"></a>Valeur retournée

Cette méthode ne retourne pas de valeur.
