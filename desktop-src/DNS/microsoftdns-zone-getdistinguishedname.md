# <a name="getdistinguishedname-method-of-the-microsoftdns_zone-class"></a>Méthode GetDistinguishedName de la \_ classe de zone MicrosoftDNS

La méthode **GetDistinguishedName** récupère le nom unique de la zone.

## <a name="syntax"></a>Syntaxe


```mof
string GetDistinguishedName();
```



## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.

## <a name="return-value"></a>Valeur retournée

Retourne le nom de la zone.
