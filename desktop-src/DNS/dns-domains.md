# <a name="dns-domains"></a>Domaines DNS

Les domaines DNS sont familiers à la plupart des professionnels de l’informatique qui utilisent Internet ; les domaines sont des nœuds de l’espace de noms de la hiérarchie DNS, et les domaines peuvent être divisés en sous-domaines. Par exemple, les domaines de la hiérarchie gadgets.widgets.microsoft.com fictive incluent les gadgets, les widgets, les domaines Microsoft et com.
