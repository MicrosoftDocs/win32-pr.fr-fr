# <a name="pausezone-method-of-the-microsoftdns_zone-class"></a>Méthode PauseZone de la \_ classe de zone MicrosoftDNS

La méthode **PauseZone** suspend la zone DNS.

## <a name="syntax"></a>Syntaxe


```mof
void PauseZone();
```



## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.

## <a name="return-value"></a>Valeur retournée

Cette méthode ne retourne pas de valeur.
