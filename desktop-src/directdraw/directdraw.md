# <a name="directdraw"></a>DirectDraw

Cette section contient des informations sur la programmation avec le composant DirectDraw de l’interface de programmation d’applications (API) DirectX.

DirectDraw n’est plus recommandé pour une utilisation. Avec la mise en route de Direct3D 9,0, toutes les fonctionnalités à deux dimensions sont contenues dans Direct3D, les fonctions d’assistance associées dans D3DX et le Direct2D de la technologie DirectX 11. Toutefois, la documentation de référence DirectDraw est toujours disponible dans cette section.

## <a name="in-this-section"></a>Dans cette section

Cette section contient la documentation de [référence de DirectDraw](directdraw-reference.md) .
