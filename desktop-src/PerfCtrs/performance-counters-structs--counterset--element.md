# <a name="structs-counterset-element"></a>Élément structs (counterSet)

Liste des structures qui contiennent les valeurs des compteurs de l’ensemble de compteurs parent.

``` syntax
<xs:element name="structs"
    type="man:structs"
 />
```

L’élément **structs** est défini par le type complexe [**counterSet**](performance-counters-counterset-complex-type.md) .

## <a name="remarks"></a>Notes

Utilisé uniquement par les fournisseurs en mode noyau.
