# <a name="counterset-provider-element"></a>Élément counterSet (Provider)

Contient une liste de compteurs logiquement liés.

``` syntax
<xs:element name="counterSet"
    type="man:counterSet"
 />
```

L’élément **counterSet** est défini par le type complexe [**Provider**](performance-counters-provider-complex-type.md) .
