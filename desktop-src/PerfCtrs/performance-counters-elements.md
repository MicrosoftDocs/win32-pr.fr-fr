# <a name="performance-counters-elements"></a>Éléments des compteurs de performance

Le schéma des compteurs de performance définit les éléments suivants :

-   [**compteur (counterSet)**](performance-counters-counter--counterset--element.md)
-   [**counterAttribute (counterAttributes)**](performance-counters-counterattribute--counterattributes--element.md)
-   [**counterAttributes (compteur)**](performance-counters-counterattributes--counter--element.md)
-   [**compteurs**](performance-counters-counters-element.md)
-   [**counterSet (fournisseur)**](performance-counters-counterset--provider--element.md)
-   [**fournisseur (compteurs)**](performance-counters-provider--counters--element.md)
-   [**struct (structs)**](performance-counters-struct--structs--element.md)
-   [**structs (counterSet)**](performance-counters-structs--counterset--element.md)

Cette section contient les noms des éléments que vous utilisez dans votre manifeste. Toutefois, pour obtenir les détails de chaque élément, consultez le type complexe qui contient l’élément. L’élément racine de la section compteurs de performance du manifeste est l’élément [**Counters**](performance-counters-counters-element.md) .
