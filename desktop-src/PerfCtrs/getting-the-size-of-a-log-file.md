# <a name="getting-the-size-of-a-log-file"></a>Obtention de la taille d’un fichier journal

[**PdhGetLogFileSize**](/windows/desktop/api/Pdh/nf-pdh-pdhgetlogfilesize) retourne la taille, en octets, du fichier journal spécifié. Utilisez cette fonction lorsque vous devez allouer de la mémoire pour une mémoire tampon qui recevra le contenu du fichier journal.
