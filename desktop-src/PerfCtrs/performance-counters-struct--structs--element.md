# <a name="struct-structs-element"></a>struct (structs), élément

Identifie une structure qui contient les valeurs des compteurs de l’ensemble de compteurs parent.

``` syntax
<xs:element name="struct"
    type="man:struct"
 />
```

L’élément **struct** est défini par le type complexe [**structs**](performance-counters-structs-complex-type.md) .
