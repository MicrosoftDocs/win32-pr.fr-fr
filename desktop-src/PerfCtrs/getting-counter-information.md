# <a name="getting-counter-information"></a>Obtention d’informations sur les compteurs

Pour récupérer des informations sur un compteur, telles que le type de compteur, le texte d’aide et le chemin d’accès, appelez la fonction [**PdhGetCounterInfo**](/windows/desktop/api/Pdh/nf-pdh-pdhgetcounterinfoa) . La fonction retourne les informations de compteur sous la forme d’une structure d' [**\_ \_ informations de compteur PDH**](/windows/desktop/api/Pdh/ns-pdh-pdh_counter_info_a) .
