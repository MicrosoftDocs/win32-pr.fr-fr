# <a name="performance-counters"></a>Compteurs de performance

Les compteurs de performances Windows fournissent une couche d’abstraction de haut niveau qui fournit une interface cohérente pour la collecte de différents types de données système telles que l’utilisation du processeur, de la mémoire et du disque. Les administrateurs système utilisent souvent des compteurs de performances pour surveiller les problèmes de performances ou de comportement des systèmes. Les développeurs de logiciels utilisent souvent des compteurs de performances pour examiner l’utilisation des ressources de leurs programmes.
