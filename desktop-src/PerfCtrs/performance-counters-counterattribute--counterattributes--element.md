# <a name="counterattribute-counterattributes-element"></a>Élément counterAttribute (counterAttributes)

Identifie un attribut d’un compteur qui décrit comment le compteur doit être affiché.

``` syntax
<xs:element name="counterAttribute"
    type="man:counterAttribute"
 />
```

L’élément **counterAttribute** est défini par le type complexe [**counterAttributes**](performance-counters-counterattributes-complex-type.md) .
