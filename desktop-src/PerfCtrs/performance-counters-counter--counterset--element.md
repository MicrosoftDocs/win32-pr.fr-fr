# <a name="counter-counterset-element"></a>Élément Counter (counterSet)

Identifie un compteur dans un ensemble de compteurs fourni par le fournisseur.

``` syntax
<xs:element name="counter"
    type="man:counter"
 />
```

L’élément **Counter** est défini par le type complexe [**counterSet**](performance-counters-counterset-complex-type.md) .
