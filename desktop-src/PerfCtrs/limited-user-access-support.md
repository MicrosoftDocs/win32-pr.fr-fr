# <a name="limited-user-access-support"></a>Prise en charge limitée de l’accès utilisateur

Seul l’administrateur de l’ordinateur ou les utilisateurs du groupe d’utilisateurs journaux de performances peuvent enregistrer des données de compteur. Les utilisateurs du groupe administrateurs peuvent enregistrer des données de compteur uniquement si l’outil qu’ils utilisent pour enregistrer les données de compteur est démarré à partir d’une fenêtre d’invite de commandes ouverte avec **exécuter en tant qu’administrateur...**. Tous les utilisateurs dans les sessions d’ouverture de session interactive peuvent afficher des données de compteur. Toutefois, les utilisateurs dans les sessions non interactives doivent se trouver dans le groupe utilisateurs d’analyse des performances pour afficher les données des compteurs.

**Windows XP :** L’administrateur ou les utilisateurs du groupe administrateurs peuvent enregistrer et afficher les données des compteurs sans restriction.
