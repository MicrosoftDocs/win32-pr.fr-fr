# <a name="ivectorchangedeventargs-interface"></a>Interface IVectorChangedEventArgs

Fournit des données pour un événement [**VectorChanged**](/uwp/api/windows.foundation.collections.iobservablevector-1?view=winrt-19041) .

## <a name="members"></a>Membres

L’interface **IVectorChangedEventArgs** hérite de **IInspectable**. **IVectorChangedEventArgs** a également les types de membres suivants :

-   [Méthodes](#methods)

### <a name="methods"></a>Méthodes

L’interface **IVectorChangedEventArgs** possède ces méthodes.



| Méthode                                                                        | Description                                                           |
|:------------------------------------------------------------------------------|:----------------------------------------------------------------------|
| [**Télécharger \_ CollectionChange**](ivectorchangedeventargs-get-collectionchange.md) | Obtient le type de modification qui s’est produite dans le vecteur.<br/>       |
| [**récupérer l' \_ index**](ivectorchangedeventargs-get-index.md)                       | Obtient la position dans le vecteur où la modification s’est produite.<br/> |
