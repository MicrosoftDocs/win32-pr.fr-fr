## <a name="design-concepts"></a><span data-ttu-id="76a72-136">Principes de conception</span><span class="sxs-lookup"><span data-stu-id="76a72-136">Design concepts</span></span>

<span data-ttu-id="76a72-144">L' **utilisation correcte des ellipses est importante pour indiquer que les utilisateurs peuvent effectuer des choix supplémentaires avant d’effectuer l’action, ou même annuler complètement l’action**.</span><span class="sxs-lookup"><span data-stu-id="76a72-144">**Proper use of ellipses is important to indicate that users can make further choices before performing the action, or even cancel the action entirely**.</span></span> <span data-ttu-id="76a72-145">Le signal visuel fourni par des points de suspension permet aux utilisateurs d’explorer vos logiciels sans crainte.</span><span class="sxs-lookup"><span data-stu-id="76a72-145">The visual cue offered by an ellipsis allows users to explore your software without fear.</span></span>

<span data-ttu-id="76a72-146">**Cela ne signifie pas que vous devez utiliser des points de suspension chaque fois qu’une action affiche une autre fenêtre** uniquement lorsque des informations supplémentaires sont requises pour effectuer l’action.</span><span class="sxs-lookup"><span data-stu-id="76a72-146">**This doesn't mean you should use an ellipsis whenever an action displays another window** only when additional information is required to perform the action.</span></span> <span data-ttu-id="76a72-147">Par conséquent, **n’importe quel bouton de commande dont le verbe implicite est « afficher une autre fenêtre » ne prend pas de points de suspension**, comme avec les commandes à propos de, avancé, aide (ou tout autre lien de commande vers une rubrique d’aide), des options, des propriétés ou des paramètres.</span><span class="sxs-lookup"><span data-stu-id="76a72-147">Consequently, **any command button whose implicit verb is to "show another window" doesn't take an ellipsis**, such as with the commands About, Advanced, Help (or any other command linking to a Help topic), Options, Properties, or Settings.</span></span>

<span data-ttu-id="76a72-148">En règle générale, les ellipses sont utilisées dans les interfaces utilisateur pour indiquer l’inexhaustivité.</span><span class="sxs-lookup"><span data-stu-id="76a72-148">Generally, ellipses are used in user interfaces to indicate incompleteness.</span></span> <span data-ttu-id="76a72-149">Les commandes qui affichent d’autres fenêtres ne sont pas incomplètes. elles doivent afficher une autre fenêtre et des informations supplémentaires ne sont pas nécessaires pour effectuer leur action.</span><span class="sxs-lookup"><span data-stu-id="76a72-149">Commands that show other windows aren't incomplete they must display another window and additional information isn't needed to perform their action.</span></span> <span data-ttu-id="76a72-150">Cette approche élimine l’encombrement à l’écran dans les situations où les ellipses ont peu de valeur.</span><span class="sxs-lookup"><span data-stu-id="76a72-150">This approach eliminates screen clutter in situations where ellipses have little value.</span></span>

<span data-ttu-id="76a72-151">**Remarque :** Lorsque vous déterminez si un bouton de commande a besoin de points de suspension, n’utilisez pas le besoin d' [élever les privilèges](winenv-uac.md) en tant que facteur.</span><span class="sxs-lookup"><span data-stu-id="76a72-151">**Note:** When determining if a command button needs an ellipsis, don't use the need to [elevate privileges](winenv-uac.md) as a factor.</span></span> <span data-ttu-id="76a72-152">L’élévation n’est pas nécessaire pour exécuter une commande (au lieu de cela, il s’agit d’une autorisation) et le besoin d’élévation est indiqué par la protection de la sécurité.</span><span class="sxs-lookup"><span data-stu-id="76a72-152">Elevation isn't information needed to perform a command (rather, it's for permission) and the need to elevate is indicated with the security shield.</span></span>

<span data-ttu-id="76a72-153">**Si vous n’avez qu’une seule chose...** Utilisez une étiquette concise, spécifique et concise qui décrit clairement l’action effectuée par le bouton de commande et utilisez des points de suspension quand cela est approprié.</span><span class="sxs-lookup"><span data-stu-id="76a72-153">**If you do only one thing...** Use a concise, specific, self-explanatory label that clearly describes the action that the command button performs, and use an ellipsis when appropriate.</span></span>
