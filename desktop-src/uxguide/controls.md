# <a name="controls-design-basics"></a>Contrôles (concepts de base de la conception)

> [!NOTE]
> ce guide de conception a été créé pour Windows 7 et n’a pas été mis à jour pour les versions plus récentes de Windows. La plupart des conseils s’appliquent toujours en principe, mais la présentation et les exemples ne reflètent pas nos [recommandations en](/windows/uwp/design/)matière de conception.

Les contrôles sont des éléments d’interface utilisateur avec lesquels vos utilisateurs interagissent sur la zone de fenêtre principale de votre application. consultez des exemples visuels de contrôles dans les applications de bureau basées sur Windows et obtenez des liens vers des instructions pour chaque contrôle.



|          Exemples                                                                                                                                                                                                                                                                                                                                                                           |
|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| ![ballon](images/controls-image1.png)<br/> Les [bulles](ctrl-balloons.md) informent les utilisateurs d’un problème non critique ou d’une condition spéciale dans un contrôle.<br/>                                                                                                                                                                                                                 |
| ![activé](images/controls-image2.png)<br/> Les [cases à cocher](ctrl-check-boxes.md) permettent aux utilisateurs de prendre une décision entre deux ou plusieurs choix manifestement différents.<br/>                                                                                                                                                                                                     |
| ![bouton de commande](images/controls-image3.png)<br/> Les [boutons de commande](ctrl-command-buttons.md) permettent aux utilisateurs d’effectuer une action immédiate.<br/>                                                                                                                                                                                                                         |
| ![lien de commande](images/controls-image4.png)<br/> Les [liens de commande](ctrl-command-links.md) permettent aux utilisateurs de faire un choix parmi un ensemble de choix s’excluant mutuellement.<br/>                                                                                                                                                                                          |
| ![liste déroulante et zone de liste modifiable](images/controls-image5.png)<br/> Les [listes déroulantes et les zones de liste](/windows/desktop/uxguide/ctrl-drop) déroulante permettent aux utilisateurs de faire un choix parmi une liste de valeurs s’excluant mutuellement.<br/>                                                                                                                                                                           |
| ![zone de groupe](images/controls-image6.png)<br/> Les [zones de groupe](ctrl-group-boxes.md) permettent aux utilisateurs de voir les relations entre un ensemble de contrôles connexes.<br/>                                                                                                                                                                                                                |
| ![lien](images/controls-image7.png)<br/> Les [liens](ctrl-links.md) permettent aux utilisateurs d’accéder à une autre page, fenêtre ou rubrique d’aide. afficher une définition ; lancer une commande ; ou choisissez une option.<br/>                                                                                                                                                                    |
| ![zone de liste](images/controls-image8.png)<br/> Les [zones de liste](ctrl-list-boxes.md) permettent aux utilisateurs de sélectionner à partir d’un ensemble de valeurs présentées dans une liste qui est toujours visible. Avec une zone de liste à sélection unique, les utilisateurs sélectionnent un élément dans une liste de valeurs s’excluant mutuellement. Avec une zone de liste à sélection multiple, les utilisateurs sélectionnent zéro, un ou plusieurs éléments dans une liste de valeurs.<br/> |
| ![mode liste](images/controls-image9.png)<br/> Les [affichages de liste](ctrl-list-views.md) permettent aux utilisateurs d’afficher et d’interagir avec une collection d’objets de données, à l’aide d’une sélection unique ou d’une sélection multiple.<br/>                                                                                                                                                           |
| ![notification](images/controls-image10.png)<br/> Les [notifications](mess-notif.md) informent les utilisateurs d’événements qui ne sont pas liés à l’activité de l’utilisateur actuel.<br/>                                                                                                                                                                                                          |
| ![barre de progression](images/controls-image11.png)<br/> Les [barres de progression](progress-bars.md) permettent aux utilisateurs de suivre la progression d’une opération longue.<br/>                                                                                                                                                                                                                    |
| ![Divulgation progressive](images/controls-image12.png)<br/> Les [contrôles de divulgation progressive](ctrl-progressive-disclosure-controls.md) permettent aux utilisateurs d’afficher ou de masquer des informations supplémentaires, notamment des données, des options ou des commandes.<br/>                                                                                                                                   |
| ![case d’option](images/controls-image13.png)<br/> Les [cases d’option](ctrl-radio-buttons.md) permettent aux utilisateurs de faire un choix parmi un ensemble de choix s’excluant mutuellement.<br/>                                                                                                                                                                                         |
| ![zone de recherche](images/controls-image14.png)<br/> Les [zones de recherche](ctrl-search-boxes.md) fournissent aux utilisateurs un moyen de rechercher rapidement des objets ou du texte spécifiques.<br/>                                                                                                                                                                                                              |
| ![curseur](images/controls-image15.png)<br/> Les [curseurs](ctrl-sliders.md) permettent aux utilisateurs de choisir parmi une plage continue de valeurs.<br/>                                                                                                                                                                                                                                   |
| ![toupie](images/controls-image16.png)<br/> Les [contrôles spin](ctrl-spin-controls.md) permettent aux utilisateurs de modifier de façon incrémentielle la valeur dans la zone de texte numérique qui lui est associée.<br/>                                                                                                                                                                                            |
| ![barre d'état](images/controls-image17.png)<br/> Les [barres d’État](ctrl-status-bars.md) affichent des informations sur l’état de la fenêtre active, les tâches en arrière-plan ou d’autres informations contextuelles.<br/>                                                                                                                                                                  |
| ![onglet](images/controls-image18.png)<br/> Les [onglets](ctrl-tabs.md) présentent aux utilisateurs des informations connexes sur des pages étiquetées distinctes.<br/>                                                                                                                                                                                                                                   |
| ![zone de texte](images/controls-image19.png)<br/> Les [zones de texte](ctrl-text-boxes.md) permettent aux utilisateurs d’afficher, d’entrer ou de modifier un texte ou une valeur numérique.<br/>                                                                                                                                                                                                                    |
| ![bulle](images/controls-image20.png)<br/> Les [info-bulles](ctrl-tooltips-and-infotips.md) étiquettent un contrôle sans étiquette.<br/>                                                                                                                                                                                                                                                |
| ![info-bulle](images/controls-image21.png)<br/> [Info-bulles](ctrl-tooltips-and-infotips.md) décrit un objet vers lequel l’utilisateur pointe.<br/>                                                                                                                                                                                                                          |
| ![TreeView](images/controls-image22.png)<br/> Les [arborescences](ctrl-tree-views.md) permettent aux utilisateurs d’afficher et d’interagir avec une collection hiérarchique d’objets, à l’aide d’une sélection unique ou d’une sélection multiple.<br/>                                                                                                                                        |
