# <a name="customizing-icons"></a>Personnalisation des icônes

Windows fournit des icônes par défaut pour chaque élément affiché sur le bureau et dans Windows Explorer. Les périphériques de stockage de masse, tels que les lecteurs de disque, reçoivent également des icônes par défaut. Toutefois, ces icônes fournissent souvent peu d’informations à l’utilisateur en ce qui concerne le contenu du fichier ou le programme qui lui est associé. Vous pouvez affecter une icône personnalisée à un dossier de système de fichiers en créant un fichier [Desktop.ini](how-to-customize-folders-with-desktop-ini.md) . Consultez les documents suivants pour savoir comment utiliser le registre pour associer des icônes personnalisées à des types de fichiers et des lettres de lecteur.

-   [Affecter une icône personnalisée à un type de fichier](how-to-assign-a-custom-icon-to-a-file-type.md)
-   [Affecter une icône et une étiquette personnalisées à une lettre de lecteur](how-to-assign-a-custom-icon-and-label-to-a-drive-letter.md)
