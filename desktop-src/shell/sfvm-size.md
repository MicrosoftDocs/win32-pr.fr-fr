# <a name="sfvm_size-message"></a>\_Message de taille SFVM

Avertit l’objet de rappel que l’affichage des dossiers a été redimensionné. Utilisé par [**IShellFolderViewCB :: MessageSFVCB**](/windows/win32/api/shlobj_core/nf-shlobj_core-ishellfolderviewcb-messagesfvcb).


```C++
SFVM_SIZE
```



## <a name="parameters"></a>Paramètres

Ce message n’a pas de paramètres.
