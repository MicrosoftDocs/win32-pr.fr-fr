# <a name="newwdevents-object"></a>Objet NewWDEvents

Étend l’objet [**WebWizardHost**](webwizardhost.md) en activant les pages côté serveur hébergées dans un Assistant pour vérifier que l’utilisateur a été authentifié par le biais d’un compte Microsoft.

## <a name="members"></a>Membres

L’objet **NewWDEvents** possède les types de membres suivants :

-   [Méthodes](#methods)

### <a name="methods"></a>Méthodes

L’objet **NewWDEvents** a ces méthodes.



| Méthode                                                            | Description                                                                                                                         |
|:------------------------------------------------------------------|:------------------------------------------------------------------------------------------------------------------------------------|
| [**PassportAuthenticate**](inewwdevents-passportauthenticate.md) | Permet aux pages côté serveur hébergées dans un assistant de vérifier que l’utilisateur a été authentifié par le biais d’un compte Microsoft.<br/> |
