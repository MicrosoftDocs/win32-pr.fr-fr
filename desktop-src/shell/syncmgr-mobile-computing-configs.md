# <a name="mobile-computing-configurations"></a>Configurations de l’informatique mobile

Le gestionnaire de synchronisation est utile pour les ordinateurs configurés comme suit :

-   Ordinateur mobile utilisé dans une station d’accueil sur un réseau à bande passante élevée, mais utilisé occasionnellement via une connexion d’accès à distance.
-   Ordinateur mobile utilisé principalement par le biais d’une connexion d’accès à distance.
-   Ordinateur de bureau utilisé strictement via une connexion d’accès à distance, tel qu’un ordinateur utilisé par un télétravailleur.
-   Ordinateur de bureau utilisé strictement via un réseau à bande passante élevée.

Même si le dernier cas n’est pas un scénario mobile classique, les problèmes de latence liés à l’accès au réseau peuvent faciliter la mise en cache des ressources localement. Dans toutes ces configurations, l’application peut utiliser le SyncMgr pour conserver les fichiers et autres ressources mises en cache localement et synchronisés entre l’utilisation en ligne et hors connexion.
