# <a name="didiskquotauserquotalimittext-property"></a>DIDiskQuotaUser. QuotaLimitText, propriété

Obtient la [**limite de quota**](diskquotacontrol-object.md) actuelle de l’utilisateur sous la forme d’une chaîne de texte.

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```JScript
QuotaLimitText = DIDiskQuotaUser.QuotaLimitText
```



## <a name="property-value"></a>Valeur de la propriété

Valeur de chaîne qui contient la limite de quota actuelle de l’utilisateur.
