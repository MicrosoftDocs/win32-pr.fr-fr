# <a name="virtualdesktopmanager-class"></a>VirtualDesktopManager, classe

**VirtualDesktopManager** implémente l’interface [**IVirtualDesktopManager**](/windows/win32/api/shobjidl_core/nn-shobjidl_core-ivirtualdesktopmanager) .

## <a name="remarks"></a>Notes

Utilisez cette interface pour interagir avec les bureaux virtuels et contrôler où les fenêtres de programme sont lancées. Pour plus d’informations, consultez [**IVirtualDesktopManager**](/windows/win32/api/shobjidl_core/nn-shobjidl_core-ivirtualdesktopmanager).
