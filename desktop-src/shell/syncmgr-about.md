# <a name="about-synchronization-manager"></a>À propos du gestionnaire de synchronisation

Le gestionnaire de synchronisation fournit une technologie centralisée et standard pour la synchronisation des fichiers en vue d’une utilisation hors connexion sur un ordinateur local. L’utilisateur peut préparer un ordinateur pour une utilisation hors connexion en mettant à jour les fichiers nécessaires à partir du stockage réseau. Ensuite, lorsque l’ordinateur est remis en ligne, toute modification hors connexion peut être recopiée sur le réseau. SyncMgr fournit une interface utilisateur commune que toutes les applications peuvent partager pour synchroniser leurs fichiers.

Les fichiers sont transférés indépendamment du protocole. Par exemple, un programme de messagerie peut transférer ses messages à l’aide de SMTP, NMTP ou POP3, tandis qu’un navigateur peut utiliser le protocole HTTP et une base de données peut utiliser RPC. Le protocole est transparent pour le SyncMgr.

SyncMgr est également indépendant du stockage et peut transférer des fichiers, des messages électroniques, des pages HTML ou des informations de base de données en toute transparence.
