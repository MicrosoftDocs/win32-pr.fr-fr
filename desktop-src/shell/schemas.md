|-------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Schéma de description de la bibliothèque](library-schema-entry.md)<br/> | Les fichiers de description de bibliothèque sont des fichiers XML qui définissent des bibliothèques.
