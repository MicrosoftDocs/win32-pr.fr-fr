# <a name="sfvm_backgroundenumdone-message"></a>\_Message SFVM BACKGROUNDENUMDONE

Avertit l’objet de rappel que l’énumération d’arrière-plan est terminée. Utilisé par [**IShellFolderViewCB :: MessageSFVCB**](/windows/win32/api/shlobj_core/nf-shlobj_core-ishellfolderviewcb-messagesfvcb).


```C++

                SFVM_BACKGROUNDENUMDONE

            
```



## <a name="parameters"></a>Paramètres

Ce message n’a pas de paramètres.
