# <a name="application-scenarios"></a>Scénarios d’application

Les applications et services qui peuvent utiliser le gestionnaire de synchronisation sont les suivants :

-   Microsoft Office les applications qui doivent préparer des fichiers pour une utilisation hors connexion.
-   Mise en cache côté client qui vous permet de mettre en cache des fichiers localement.
-   Navigateurs qui peuvent mettre en cache des pages HTML localement.
-   Programmes de messagerie qui peuvent mettre en cache des messages électroniques localement.
-   Bases de données qui peuvent stocker des informations localement.
