# <a name="user-profiles"></a>Profils utilisateur

Les informations de configuration qui peuvent être conservées pour chaque utilisateur sont enregistrées dans des [profils utilisateur](about-user-profiles.md). Ces informations incluent la disposition du bureau, les groupes de programmes personnels, les éléments de programme dans les groupes de programmes personnels, les couleurs d’écran, les économiseurs d’écran, les connexions réseau, les connexions d’imprimante, les paramètres de souris, la taille et la position de la fenêtre, etc. Lorsqu’un utilisateur ouvre une session, le système charge le profil de l’utilisateur et configure l’environnement en fonction des informations contenues dans le profil.

Cette section comprend les rubriques suivantes.

-   [À propos des profils utilisateur](about-user-profiles.md)
-   [Référence des profils utilisateur](user-profiles-reference.md)
