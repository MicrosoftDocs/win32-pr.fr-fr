# <a name="diskquotacontroldefaultquotalimit-property"></a>DiskQuotaControl. DefaultQuotaLimit, propriété

Définit ou obtient la limite de quota par défaut.

Cette propriété est en lecture/écriture.

## <a name="syntax"></a>Syntaxe


```JScript
iDefaultQuotaLimit = DiskQuotaControl.DefaultQuotaLimit
DiskQuotaControl.DefaultQuotaLimit = iDefaultQuotaLimit
```



## <a name="property-value"></a>Valeur de la propriété

Valeur **entière** qui spécifie ou reçoit la limite de quota par défaut pour les nouveaux utilisateurs, en octets.
