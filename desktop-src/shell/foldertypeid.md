## <a name="remarks"></a>Notes

Les valeurs **FOLDERTYPEID** sont des GUID et sont déclarées dans shlguid. h.

Dans la pratique, toute valeur **FOLDERTYPEID** peut être appliquée à n’importe quel dossier, quel que soit le contenu de ce dossier. **FOLDERTYPEID** est un modèle de vue uniquement, fournissant la disposition et les colonnes appropriées pour le contenu de dossier attendu. Il n’y a aucune vérification du contenu du dossier par rapport à la valeur **FOLDERTYPEID** pour garantir l’adéquation.

Ces types de dossiers et d’autres sont définis dans le Registre sous la sous-clé **FolderTypes** , qui se trouve comme indiqué ici :

```
HKEY_LOCAL_MACHINE
   SOFTWARE
      Microsoft
         Windows
            CurrentVersion
               Explorer
                  FolderTypes
```
