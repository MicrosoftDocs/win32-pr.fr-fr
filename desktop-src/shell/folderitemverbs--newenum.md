# <a name="folderitemverbs_newenum-method"></a>FolderItemVerbs. \_ NewEnum, méthode

Crée et retourne un nouvel objet [**FolderItemVerbs**](folderitemverbs.md) qui est une copie de cet objet FolderItemVerbs.

## <a name="syntax"></a>Syntaxe


```JScript
FolderItemVerbs._NewEnum()
```



## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.

## <a name="return-value"></a>Valeur de retour

Cette méthode ne retourne pas de valeur.
