# <a name="folderitemverb-object"></a>Objet FolderItemVerb

Représente un verbe unique disponible pour un élément. Cet objet contient des propriétés et des méthodes qui vous permettent de récupérer des informations sur le verbe.

## <a name="members"></a>Membres

L’objet **FolderItemVerb** possède les types de membres suivants :

-   [Méthodes](#methods)
-   [Propriétés](#properties)

### <a name="methods"></a>Méthodes

L’objet **FolderItemVerb** a ces méthodes.



| Méthode                              | Description                                                                                  |
|:------------------------------------|:---------------------------------------------------------------------------------------------|
| [**DoIt**](folderitemverb-doit.md) | Exécute un verbe sur le [**FolderItem**](folderitem.md) associé au verbe.<br/> |



 

### <a name="properties"></a>Propriétés

L’objet **FolderItemVerb** a ces propriétés.



| Propriété                                                     | Type d’accès          | Description                          |
|:-------------------------------------------------------------|:---------------------|:-------------------------------------|
| [**Application**](folderitemverb-application.md)<br/> | Lecture seule<br/> | Non implémenté.<br/>          |
| [**Nom**](folderitemverb-name.md)<br/>               | Lecture seule<br/> | Contient le nom du verbe.<br/> |
| [**Parent**](folderitemverb-parent.md)<br/>           | Lecture seule<br/> | Non implémenté.<br/>          |
