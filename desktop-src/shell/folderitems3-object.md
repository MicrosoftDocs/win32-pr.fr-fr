# <a name="folderitems3-object"></a>Objet FolderItems3

Étend l’objet [**FolderItems2**](folderitems2-object.md) . Cet objet prend en charge une méthode et une propriété supplémentaires.

## <a name="members"></a>Membres

L’objet **FolderItems3** possède les types de membres suivants :

-   [Méthodes](#methods)
-   [Propriétés](#properties)

### <a name="methods"></a>Méthodes

L’objet **FolderItems3** a ces méthodes.



| Méthode                                | Description                                                       |
|:--------------------------------------|:------------------------------------------------------------------|
| [**Filtrer**](folderitems3-filter.md) | Définit un filtre de caractères génériques à appliquer aux éléments retournés.<br/> |



 

### <a name="properties"></a>Propriétés

L’objet **FolderItems3** a ces propriétés.



| Propriété                                       | Type d’accès          | Description                                                       |
|:-----------------------------------------------|:---------------------|:------------------------------------------------------------------|
| [**Verbes**](folderitems3-verbs.md)<br/> | Lecture seule<br/> | Obtient la liste des verbes communs à tous les éléments de dossier.<br/> |
