# <a name="folderitems2-object"></a>Objet FolderItems2

Étend l’objet [**FolderItems**](folderitems.md) . Il prend en charge une méthode supplémentaire.

## <a name="members"></a>Membres

L’objet **FolderItems2** possède les types de membres suivants :

-   [Méthodes](#methods)

### <a name="methods"></a>Méthodes

L’objet **FolderItems2** a ces méthodes.



| Méthode                                            | Description                                                                                                                                                                                                                                         |
|:--------------------------------------------------|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**InvokeVerbEx**](folderitems2-invokeverbex.md) | Exécute un verbe sur une collection d’objets [**FolderItem**](folderitem.md) . Cette méthode est une extension de la méthode [**InvokeVerb**](folderitem-invokeverb.md) , qui permet un contrôle supplémentaire de l’opération via un ensemble d’indicateurs.<br/> |
