# <a name="didiskquotauserlogonname-property"></a>DIDiskQuotaUser. LogonName, propriété

Obtient le nom du compte d’ouverture de session de l’utilisateur.

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```JScript
LogonName = DIDiskQuotaUser.LogonName
```



## <a name="property-value"></a>Valeur de la propriété

Valeur de chaîne qui est définie sur le nom d’ouverture de session du compte de l’utilisateur.
