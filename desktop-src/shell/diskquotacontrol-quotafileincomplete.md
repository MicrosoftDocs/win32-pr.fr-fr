# <a name="diskquotacontrolquotafileincomplete-property"></a>DiskQuotaControl. QuotaFileIncomplete, propriété

Obtient une valeur booléenne qui indique si le fichier de quota du volume est terminé.

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```JScript
bQuotaFileIncomplete = DiskQuotaControl.QuotaFileIncomplete
```



## <a name="property-value"></a>Valeur de la propriété

Cette propriété a la valeur **true** si le fichier de quota est incomplet, ou **false** dans le cas contraire.
