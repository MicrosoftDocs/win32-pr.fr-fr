# <a name="how-to-choose-a-file-type-extension"></a>Comment choisir une extension de type de fichier

Les types de fichiers sont identifiés par une extension de nom de fichier. Avant de définir un nouveau type de fichier, vous devez d’abord choisir une extension de nom de fichier.

## <a name="instructions"></a>Instructions

### <a name="step-1"></a>Étape 1 :

Choisissez une extension de nom de fichier long que vous souhaitez utiliser pour identifier le type de fichier. Pour plus d’informations, consultez [gestionnaires de types de fichiers](fa-file-extensions.md).

### <a name="step-2"></a>Étape 2 :

Vérifiez la base de données MIME [IANA (Internet Assigned Numbers Authority)](https://www.iana.org/assignments/media-types) pour déterminer si l’extension de nom de fichier est déjà associée à une autre application.

### <a name="step-3"></a>Étape 3 :

Définissez un type MIME lors de la définition d’un nouveau type de fichier public et inscrivez-le auprès de l’IANA.
