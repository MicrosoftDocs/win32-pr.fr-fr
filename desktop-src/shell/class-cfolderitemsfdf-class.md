# <a name="cfolderitemsfdf-class"></a>CFolderItemsFDF, classe

**CFolderItemsFDF** est une collection de [**FolderItems**](folderitems.md). Il implémente l’interface [**IInsertItem**](/windows/desktop/api/Shobjidl/nn-shobjidl-iinsertitem) .

## <a name="members"></a>Membres

La classe **CFolderItemsFDF** hérite de [**CFolderItems**](class-cfolderitems-class.md) mais n’a pas de membres supplémentaires.

## <a name="remarks"></a>Notes

FDF est l’abréviation de *de différents dossiers*.

La classe [**CFolderItems**](class-cfolderitems-class.md) énumère les éléments de plusieurs dossiers et utilise l’interface [**IInsertItem**](/windows/desktop/api/Shobjidl/nn-shobjidl-iinsertitem) pour insérer les éléments dans la collection.
