# <a name="menu-statement"></a>MENU (instruction)

Définit le menu d’une boîte de dialogue. Si aucune instruction n’est fournie, la boîte de dialogue n’a pas de menu.

``` syntax
MENU menuname
```

<dl> <dt>

<span id="menuname"></span><span id="MENUNAME"></span>*MenuName*
</dt> <dd>

Nom ou identificateur du menu à utiliser.

</dd> </dl>

## <a name="examples"></a>Exemples

L’exemple suivant illustre l’utilisation de l’instruction de boîte de dialogue de **menu** :

``` syntax
MENU errmenu
```
