<span id="tools.f_1_gly"></span><span id="TOOLS.F_1_GLY"></span>**Fichier introuvable : nom de fichier**
</dt> <dd>

Le fichier spécifié dans la commande RC est introuvable. Assurez-vous que le fichier n’a pas été déplacé vers un autre répertoire et que le nom de fichier ou le chemin d’accès est correctement typé.

</dd> <dt>

<span id="tools.f_2_gly"></span><span id="TOOLS.F_2_GLY"></span>**Les noms de police doivent être des ordinaux**
</dt> <dd>

Le paramètre de type de *pointage* de l’instruction [**font**](font-statement.md) doit être un entier, et non une chaîne.
