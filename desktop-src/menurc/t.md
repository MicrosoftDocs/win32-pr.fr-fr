<span id="tools.t_1_gly"></span><span id="TOOLS.T_1_GLY"></span>**Chaîne de texte ou ordinal attendu dans le contrôle**
</dt> <dd>

Si vous utilisez un ordinal, assurez-vous que vous disposez d’une instruction de [**\# définition**](-define.md) pour le contrôle.
