# <a name="endif"></a>\#endif

La directive **\# endif** marque la fin d’un bloc de compilation conditionnelle défini par une directive **\# ifdef** . Un **\# endif** est requis pour chaque directive **\# If**, **\# ifdef** ou **\# ifndef** .

``` syntax
#endif
```

Cette directive n’a aucun paramètre.
