# <a name="rc-diagnostic-messages"></a>Messages de diagnostic RC

Cette section décrit les messages de diagnostic produits par RC. Un grand nombre de ces messages s’affichent quand RC ne peut pas compiler correctement les ressources. Les descriptions clarifient la cause de chaque message.

Un V majuscule entre parenthèses (V) au début d’une description de message indique que le message s’affiche uniquement si RC est exécuté avec l’option **/v** (verbose). Ces messages sont généralement informatifs et n’indiquent pas nécessairement des erreurs.

Les messages sont répertoriés par ordre alphabétique dans les rubriques suivantes :

[A](a.md) [B](b.md) [C](c.md) [e](e.md) [F](f.md) [I](i.md) [N](n.md) [O](o.md) [R](r.md) [T](t.md) [U](u.md) [V](v.md) [W](w.md)
