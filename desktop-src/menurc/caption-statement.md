# <a name="caption-statement"></a>Instruction de légende

Définit le titre d’une boîte de dialogue. Le titre s’affiche dans la barre de légende de la zone (le cas échéant).

La légende par défaut est vide.

``` syntax
CAPTION "captiontext"
```

<dl> <dt>

<span id="captiontext"></span><span id="CAPTIONTEXT"></span>*CaptionText*
</dt> <dd>

Chaîne de caractères placée entre guillemets doubles (").

</dd> </dl>

## <a name="examples"></a>Exemples

L’exemple suivant illustre l’utilisation de l’instruction de **légende** :

``` syntax
CAPTION "Error!" 
```
