<dl> <dt>

<span id="tools.w_1_gly"></span><span id="TOOLS.W_1_GLY"></span>**AVERTISSEMENT : caractère ASCII non équivalent au code de touche virtuelle**
</dt> <dd>

Il existe un code de clé virtuelle non valide dans l’instruction [**Accelerators**](accelerators-resource.md) . Les valeurs ASCII de certains caractères, tels que \* , ^ ou &, ne sont pas équivalentes aux codes de touche virtuelle pour les clés correspondantes. Dans le cas de l’astérisque ( \* ), le code de la touche virtuelle est équivalent à la valeur ASCII de 8, le caractère numérique de la même clé.

</dd> <dt>

<span id="tools.w_2_gly"></span><span id="TOOLS.W_2_GLY"></span>**AVERTISSEMENT : SHIFT ou CONTROL utilisé sans VIRTKEY**
</dt> <dd>

Les options **ALT**, **MAJ** et [**CTRL**](control-control.md) s’appliquent uniquement aux clés virtuelles de l’instruction [**Accelerators**](accelerators-resource.md) . Assurez-vous que l’option **VIRTKEY** est utilisée avec l’une de ces options.

</dd> <dt>

<span id="tools.w_3_gly"></span><span id="TOOLS.W_3_GLY"></span>**Écriture de resourcetypeimagelang : Language : taille**
</dt> <dd>

(V) le compilateur de ressources Microsoft Windows (RC) écrit la ressource spécifiée. Le *type* est le type de ressource. Il peut s’agir d’un nombre ou d’une chaîne. L' *image* peut être un nombre ou une chaîne. La *langue* est la langue nationale de la ressource. La *taille* est la taille décimale de la ressource, en octets.

</dd> </dl>
