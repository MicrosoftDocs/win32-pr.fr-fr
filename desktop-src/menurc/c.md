<dl> <dt>

<span id="tools.c_1_gly"></span><span id="TOOLS.C_1_GLY"></span>**Impossible de réutiliser les constantes de chaîne**
</dt> <dd>

Vous utilisez la même valeur deux fois dans une instruction [**STRINGTABLE**](stringtable-resource.md) . Assurez-vous que vous n’avez pas mélangé des valeurs décimales et hexadécimales qui se chevauchent.

</dd> <dt>

<span id="tools.c_2_gly"></span><span id="TOOLS.C_2_GLY"></span>**Caractère de contrôle hors de \[ la plage A ? Lettre\]**
</dt> <dd>

Un caractère de contrôle dans l’instruction [**Accelerators**](accelerators-resource.md) n’est pas valide. Le caractère qui suit le signe insertion (^) doit être compris entre A et Z.

</dd> <dt>

<span id="tools.c_3_gly"></span><span id="TOOLS.C_3_GLY"></span>**Impossible d’ouvrir-fichier-nom**
</dt> <dd>

Le compilateur de ressources Microsoft Windows (RC) n’a pas pu ouvrir le fichier spécifié. Assurez-vous que le fichier existe et que vous avez correctement tapé le nom de fichier.

</dd> <dt>

<span id="tools.c_4_gly"></span><span id="TOOLS.C_4_GLY"></span>**Openresource-nom introuvable**
</dt> <dd>

Le compilateur de ressources Microsoft Windows (RC) n’a pas pu ouvrir le fichier spécifié. Assurez-vous que le fichier existe et que vous avez correctement tapé le nom de fichier.

</dd> <dt>

<span id="tools.c_5_gly"></span><span id="TOOLS.C_5_GLY"></span>**Creatingresource-nom**
</dt> <dd>

(V) le compilateur de ressources Microsoft Windows (RC) crée un fichier de ressources binaires (. res).

</dd> </dl>
