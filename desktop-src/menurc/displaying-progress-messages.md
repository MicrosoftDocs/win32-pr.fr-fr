# <a name="displaying-progress-messages"></a>Affichage des messages de progression

Par défaut, RC compile sans afficher les messages de progression. Toutefois, vous pouvez spécifier que la RC doit afficher les messages de progression. Pour ce faire, utilisez l’option **/v** .

Dans l’exemple suivant, RC signale sa progression lors de la compilation du fichier de définition de ressources Sample. RC et crée le fichier de ressources compilé Sample. res :

**RC/v Sample. RC**
