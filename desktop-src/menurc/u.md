# <a name="u-menus-and-other-resources"></a>U (menus et autres ressources)

[A](a.md) [B](b.md) [C](c.md) D [e](e.md) [F](f.md) G H [I](i.md) J K L M [N](n.md) [O](o.md) P Q [R](r.md) S [T](t.md) U [V](v.md) [W](w.md) X Y Z

<dl> <dt>

<span id="tools.u_1_gly"></span><span id="TOOLS.U_1_GLY"></span>**Impossible de createdestination**
</dt> <dd>

Le compilateur de ressources Microsoft Windows (RC) n’a pas pu créer le fichier de destination. Assurez-vous que l’espace disque est suffisant.

</dd> <dt>

<span id="tools.u_2_gly"></span><span id="TOOLS.U_2_GLY"></span>**Parenthèses non équilibrées**
</dt> <dd>

Assurez-vous que vous avez fermé chaque parenthèse ouvrante dans l’instruction [**Dialog**](dialog-resource.md) .

</dd> <dt>

<span id="tools.u_3_gly"></span><span id="TOOLS.U_3_GLY"></span>**Valeur inattendue dans RCData**
</dt> <dd>

Les valeurs du paramètre *Raw-Data* de l’instruction [**RCDATA**](rcdata-resource.md) doivent être des entiers ou des chaînes, séparées par des virgules. Assurez-vous que vous n’avez pas laissé une virgule ou un guillemet autour d’une chaîne.

</dd> <dt>

<span id="tools.u_4_gly"></span><span id="TOOLS.U_4_GLY"></span>**Valeur inattendue dans les données de valeur**
</dt> <dd>

Une instruction contient des informations avec un format ou une taille différente de la valeur attendue pour ce paramètre.

</dd> <dt>

<span id="tools.u_5_gly"></span><span id="TOOLS.U_5_GLY"></span>**Format d’en-tête DIB inconnu**
</dt> <dd>

L’en-tête DIB (Device-Independent Bitmap) n’est pas une structure [**BITMAPCOREHEADER**](/windows/win32/api/wingdi/ns-wingdi-bitmapcoreheader) ou [**BITMAPINFOHEADER**](/previous-versions//dd183376(v=vs.85)) .

</dd> <dt>

<span id="tools.u_6_gly"></span><span id="TOOLS.U_6_GLY"></span>**Sous-type de menu inconnu**
</dt> <dd>

Le paramètre *Item-Definitions* de l’instruction [**menu**](menu-resource.md) peut contenir uniquement des instructions [**MenuItem**](menuitem-statement.md) et [**Popup**](popup-resource.md) .

</dd> <dt>

<span id="tools.u_7_gly"></span><span id="TOOLS.U_7_GLY"></span>**Champ VERSIONINFO non reconnu ; BEGIN ou virgule ATTENDU**
</dt> <dd>

Le format des informations qui suivent une instruction [**VERSIONINFO**](versioninfo-resource.md) est incorrect.

</dd> </dl>
