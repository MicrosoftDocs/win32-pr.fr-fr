# <a name="resource-compiler"></a>compilateur de ressources

Le compilateur de ressources Microsoft Windows (RC) est un outil utilisé dans la création d’applications Windows. Cette vue d’ensemble décrit comment créer un fichier de définition de ressource (script), comment compiler les ressources de votre application et comment ajouter des ressources compilées à votre application.

Cet outil est disponible dans Visual Studio et le kit de développement logiciel (SDK) Microsoft Windows.

Les rubriques suivantes décrivent la syntaxe du fichier de ressources et la ligne de commande RC :

-   [À propos des fichiers de ressources](about-resource-files.md)
-   [Utilisation de RC (ligne de commande RC)](using-rc-the-rc-command-line-.md)
