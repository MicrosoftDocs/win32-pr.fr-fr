# <a name="package-resource-indexing-pri-reference"></a>Référence PRI (package Resource indexer)

Ensemble d’API pour l’utilisation d’un indexeur de ressource. Un indexeur de ressource est utilisé pour générer des fichiers PRI (package Resource index) pour une application UWP. Pour plus d’informations et pour obtenir des procédures pas à pas sur l’utilisation de ces API, consultez [API PRI (package Resource Indexing) et systèmes de génération personnalisés](/windows/uwp/app-resources/pri-apis-custom-build-systems).
