# <a name="preprocessor-operators"></a>Opérateurs de préprocesseur

RC prend en charge l’utilisation des opérateurs de prétraitement standard C dans les définitions de macros. Ces opérateurs sont décrits dans le tableau suivant.



| Opérateur | Description                                                 |
|----------|-------------------------------------------------------------|
| \#       | Encadre l’argument entre guillemets.                            |
| \#@      | Encadre l’argument entre guillemets simples.                     |
| \#\#     | Concatène les jetons utilisés comme arguments pour former d’autres jetons. |
