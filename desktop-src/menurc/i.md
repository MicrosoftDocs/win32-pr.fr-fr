# <a name="i-menus-and-other-resources"></a>I (menus et autres ressources)

[A](a.md) [B](b.md) [C](c.md) D [e](e.md) [F](f.md) G H I J K L M [N](n.md) [O](o.md) P Q [R](r.md) S [T](t.md) [U](u.md) [V](v.md) [W](w.md) X Y Z

<dl> <dt>

<span id="tools.i_1_gly"></span><span id="TOOLS.I_1_GLY"></span>**Accélérateur non valide**
</dt> <dd>

Un paramètre d' *événement* de l’instruction [**Accelerators**](accelerators-resource.md) n’a pas été reconnu ou sa longueur dépasse deux caractères.

</dd> <dt>

<span id="tools.i_2_gly"></span><span id="TOOLS.I_2_GLY"></span>**Type d’accélérateur non valide (ASCII ou VIRTKEY)**
</dt> <dd>

Le paramètre de *type* dans l’instruction [**Accelerators**](accelerators-resource.md) doit contenir la valeur **ASCII** ou **VIRTKEY** .

</dd> <dt>

<span id="tools.i_3_gly"></span><span id="TOOLS.I_3_GLY"></span>**Caractère de contrôle non valide**
</dt> <dd>

Un caractère de contrôle dans l’instruction [**Accelerators**](accelerators-resource.md) n’est pas valide. Un caractère de contrôle valide est constitué d’un signe insertion (^) suivi d’une seule lettre.

</dd> <dt>

<span id="tools.i_4_gly"></span><span id="TOOLS.I_4_GLY"></span>**Type de contrôle non valide**
</dt> <dd>

L’instruction de contrôle dans une instruction [**Dialog**](dialog-resource.md) doit être l’une des suivantes [**: AUTO3STATE**](auto3state-control.md), [**autocase**](autocheckbox-control.md), RadioButton, [**CheckBox**](checkbox-control.md), [**ComboBox**](combobox-control.md), [**Control**](control-control.md), [**CTEXT**](ctext-control.md), [**DEFPUSHBUTTON**](defpushbutton-control.md), [**EDITTEXT**](edittext-control.md), [**GroupBox**](groupbox-control.md), [](autoradiobutton-control.md) [**icône**](icon-control.md), [**ListBox**](listbox-control.md), [**LTEXT**](ltext-control.md), [**PUSHBOX**](pushbox-control.md), [**PUSHBUTTON**](pushbutton-control.md), [**RadioButton**](radiobutton-control.md), [**rtext**](rtext-control.md), [**ScrollBar**](scrollbar-control.md), [**STATE3**](state3-control.md).

</dd> <dt>

<span id="tools.i_5_gly"></span><span id="TOOLS.I_5_GLY"></span>**Directive non valide dans le fichier RC prétraité**
</dt> <dd>

Le nom de fichier spécifié contient un caractère de saut de ligne incorporé.

</dd> <dt>

<span id="tools.i_6_gly"></span><span id="TOOLS.I_6_GLY"></span>**Fichier de .EXE non valide**
</dt> <dd>

Le fichier exécutable (.exe) n’est pas valide. Assurez-vous que l’éditeur de liens l’a créé correctement et que le fichier existe.

</dd> <dt>

<span id="tools.i_7_gly"></span><span id="TOOLS.I_7_GLY"></span>**Commutateur non valide, option**
</dt> <dd>

Une option utilisée n’est pas valide. Pour obtenir la liste des options de ligne de commande, utilisez la commande **RC/ ?** .

</dd> <dt>

<span id="tools.i_8_gly"></span><span id="TOOLS.I_8_GLY"></span>**Commutateur non valide, commutateurs trop nombreux**
</dt> <dd>

**Un** trop grand nombre d’options a été spécifié. Pour définir un grand nombre de symboles, utilisez la directive [**\# include**](-include.md) pour inclure un fichier de définitions.

</dd> <dt>

<span id="tools.i_9_gly"></span><span id="TOOLS.I_9_GLY"></span>**Type non valide**
</dt> <dd>

Le type de ressource ne faisait pas partie des types définis dans le fichier include.

</dd> <dt>

<span id="tools.i_10_gly"></span><span id="TOOLS.I_10_GLY"></span>**Utilisation non valide. Utilisez RC- ? pour obtenir de l’aide**
</dt> <dd>

Assurez-vous que vous avez au moins un nom de fichier à utiliser. Pour obtenir la liste des options de ligne de commande, utilisez la commande **RC- ?** .

</dd> <dt>

<span id="tools.i_11_gly"></span><span id="TOOLS.I_11_GLY"></span>**Erreur d’e/s lors de la lecture du fichier**
</dt> <dd>

Échec de la lecture. Étant donné qu’il s’agit d’une routine générique, aucun nom de fichier spécifique n’est fourni.

</dd> <dt>

<span id="tools.i_12_gly"></span><span id="TOOLS.I_12_GLY"></span>**Erreur d’e/s lors de la recherche dans le fichier**
</dt> <dd>

Échec de la recherche dans le fichier. Étant donné qu’il s’agit d’une routine générique, aucun nom de fichier spécifique n’est fourni.

</dd> <dt>

<span id="tools.i_13_gly"></span><span id="TOOLS.I_13_GLY"></span>**Erreur d’e/s lors de l’écriture du fichier**
</dt> <dd>

Échec de l’écriture. Étant donné qu’il s’agit d’une routine générique, aucun nom de fichier spécifique n’est fourni.

</dd> </dl>
