# <a name="user-interface-windows-installer"></a>Interface utilisateur (Windows Installer)

Windows Le programme d’installation fournit une interface utilisateur complète pour l’installation d’une application ou d’un produit. L’interface utilisateur présente à l’utilisateur les options disponibles pour configurer l’installation et obtient des informations de l’utilisateur sur le processus d’installation en attente.

[À propos de l’interface utilisateur](about-the-user-interface.md) décrit les fonctionnalités de l’interface utilisateur du programme d’installation. L’utilisation de l' [interface utilisateur](using-the-user-interface.md) décrit l’utilisation de l’interface utilisateur interne du programme d’installation.

Les informations de référence sur la boîte de dialogue interne du programme d’installation et les options et les styles du contrôle sont présentés dans référence de l' [interface utilisateur](user-interface-reference.md).
