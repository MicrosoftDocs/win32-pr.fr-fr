# <a name="ice87"></a>ICE87

ICE87 valide que les propriétés suivantes n’ont pas été créées dans la [table de propriétés](property-table.md). Ces propriétés doivent être définies à la place sur une ligne de commande.

-   [**Propriété ADDLOCAL**](addlocal.md)
-   [**REMOVE (propriété)**](remove.md)
-   [**Propriété ADDSOURCE**](addsource.md)
-   [**Propriété ADDDEFAULT**](adddefault.md)
-   [**RÉINSTALLER la propriété**](reinstall.md)
-   [**Propriété ADVERTISE**](advertise.md)
-   [**Propriété COMPADDLOCAL**](compaddlocal.md)
-   [**Propriété COMPADDSOURCE**](compaddsource.md)
-   [**Propriété FILEADDLOCAL**](fileaddlocal.md)
-   [**Propriété FILEADDSOURCE**](fileaddsource.md)
-   [**Propriété FILEADDDEFAULT**](fileadddefault.md)

## <a name="result"></a>Résultats

ICE87 publie l’avertissement suivant.



| AVERTISSEMENT ICE87                                                                                                                        | Description                                                                                                                       |
|--------------------------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------|
| La propriété « \[ 1 \] » ne doit pas être créée dans la table de propriétés. Cela peut entraîner la désinstallation correcte du produit | La propriété spécifiée ne doit pas être définie dans la [table de propriétés](property-table.md). Définissez plutôt la propriété sur une ligne de commande. |
