# <a name="ice89"></a>ICE89

ICE89 vérifie que la valeur de la \_ colonne parent ProgID dans la [table ProgID](progid-table.md) est une clé étrangère valide dans la colonne ProgID de la table ProgID. Chaque parent ProgId doit avoir un enregistrement dans la table ProgId.

## <a name="result"></a>Résultats

ICE89 publie les erreurs suivantes.



| Erreur ICE89                                                           | Description                                                                |
|-----------------------------------------------------------------------|----------------------------------------------------------------------------|
| Le \_ parent ProgID « \[ 1 \] » dans la table ProgID n’est pas un ProgID valide. | Un parent ProgId spécifié n’est pas listé dans la table ProgId. |
