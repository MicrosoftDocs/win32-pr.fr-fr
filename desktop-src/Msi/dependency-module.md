# <a name="dependencymodule-property"></a>Dependency. module, propriété

La propriété **module** en lecture seule de l' [**objet de dépendance**](dependency-object.md) retourne le ModuleID du module requis par la chaîne actuelle sous la forme d’un **BSTR**. Le ModuleID est le même formulaire que celui utilisé dans la [table ModuleSignature](modulesignature-table.md).

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```JScript
propVal = Dependency.Module
```



## <a name="property-value"></a>Valeur de la propriété

## <a name="c"></a>C++

Consultez la méthode [**IMsmDependency \_ obtenir le \_ module**](/windows/win32/api/mergemod/nf-mergemod-imsmdependency-get_module) .
