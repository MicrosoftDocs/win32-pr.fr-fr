# <a name="nousername-property"></a>NOUSERNAME (propriété)

Définissez la propriété **NOUSERNAME** sur 1 pour supprimer le programme d’installation de la valeur de la propriété [**username**](username.md) . Cette propriété est utilisée par les applications qui doivent collecter le nom d’utilisateur lors de la première exécution.

## <a name="default-value"></a>Valeur par défaut

Non défini. Le programme d’installation définit la propriété [**username**](username.md) automatiquement à l’aide des valeurs du Registre.

Windows Installer 4,0 ou Windows Installer 4,5 sur Windows Server 2008 ou Windows Vista. Windows Installer sur Windows Server 2003 ou Windows XP. pour plus d’informations sur la Service Pack de Windows minimale requise par une version de Windows Installer, consultez la [configuration requise pour la Windows Installer Run-Time](windows-installer-portal.md) .<br/> |
