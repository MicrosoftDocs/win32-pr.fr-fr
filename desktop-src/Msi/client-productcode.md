# <a name="clientproductcode-property"></a>Propriété client. ProductCode

Produit client du composant en question. Cette propriété appelle la fonction [**MsiEnumClientsEx**](/windows/desktop/api/Msi/nf-msi-msienumclientsexa) .

**[Windows Installer 4,5 ou version antérieure](not-supported-in-windows-installer-4-5.md):** Non pris en charge. cette propriété est disponible à partir de Windows Installer 5,0.
