# <a name="shortcut"></a>Raccourci

Le type de données de raccourci est généralement utilisé dans la colonne cible de la [table de raccourcis](shortcut-table.md). S’il contient des crochets ( \[ \] ), la cible de raccourci est évaluée comme une chaîne mise en forme. Dans le cas contraire, le raccourci est évalué comme un [identificateur](identifier.md) et doit être une clé étrangère valide dans la [table des fonctionnalités](feature-table.md).
