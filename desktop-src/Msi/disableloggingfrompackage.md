# <a name="disableloggingfrompackage"></a>DisableLoggingFromPackage

La valeur de cette [stratégie système](system-policy.md) par ordinateur est définie sur « 1 » pour désactiver la journalisation qui est spécifiée pour le package par la propriété [**MsiLogging**](msilogging.md) pour tous les utilisateurs de l’ordinateur.

la propriété [**MsiLogging**](msilogging.md) et la stratégie DisableLoggingFromPackage requièrent Windows Installer 4,0.

## <a name="registry-key"></a>Clé de Registre

**HKEY \_ \_** \\ **stratégies logicielles** de l’ordinateur LOCAL \\  \\ **Microsoft** \\ **Windows** \\ **Installer**

## <a name="data-type"></a>Type de données

**\_valeur DWORD reg**
