# <a name="disableflyweightpatching"></a>DisableFlyWeightPatching

Si cette stratégie système par ordinateur est définie sur 1 (un), toutes les options d' [optimisation des correctifs](patch-optimization.md) sont désactivées pendant l’installation.

Les options d' [optimisation des correctifs](patch-optimization.md) et la stratégie DisableFlyWeightPatching sont disponibles à partir de Windows Installer version 3,0.

## <a name="registry-key"></a>Clé de Registre

**HKEY \_ \_** \\ **Stratégies logicielles** de l’ordinateur local \\  \\ **Microsoft** \\ **Windows** \\ **installer**

## <a name="data-type"></a>Type de données

**\_valeur DWORD reg**
