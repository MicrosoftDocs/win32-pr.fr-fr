# <a name="transparent-control-attribute"></a>Attribut de contrôle transparent

Si le bit de contrôle transparent est défini sur un contrôle de texte, le contrôle est affiché de manière transparente avec l’arrière-plan qui s’affiche dans le contrôle où il n’y a aucun caractère. Si ce bit n’est pas défini, le contrôle de texte est opaque.

## <a name="valid-controls"></a>Contrôles valides

[Text](text-control.md)

## <a name="value"></a>Valeur



| Decimal | Valeur hexadécimale | Constante                              |
|---------|-------------|---------------------------------------|
| 65536   | 0x00010000  | **msidbControlAttributesTransparent** |
