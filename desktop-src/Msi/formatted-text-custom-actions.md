# <a name="formatted-text-custom-actions"></a>Actions personnalisées du texte mis en forme

Ce type d’action personnalisée définit une propriété ou un répertoire d’installation à partir d’une chaîne de texte mise en forme. Les paramètres de cette chaîne peuvent être des propriétés, des variables d’environnement, des chemins d’accès aux fichiers ou les chemins d’accès aux répertoires des composants. Cela est utile pour transmettre des informations aux actions personnalisées suivantes.

Les types d’actions personnalisées de base suivants définissent une propriété ou un répertoire. Pour plus d’informations, consultez le type d’action personnalisé particulier.



| Type d’action personnalisé                                 | Description                                    |
|----------------------------------------------------|------------------------------------------------|
| [Type d’action personnalisé 51](custom-action-type-51.md) | Définit une propriété avec une chaîne de texte mise en forme.  |
| [Type d’action personnalisé 35](custom-action-type-35.md) | Définit un répertoire avec une chaîne de texte mise en forme. |



 

Consultez la [liste récapitulative de tous les types d’actions personnalisées](summary-list-of-all-custom-action-types.md) pour obtenir un résumé de tous les types d’actions personnalisées et la façon dont elles sont encodées dans la [table CustomAction](customaction-table.md).
