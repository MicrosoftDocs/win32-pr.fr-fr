# <a name="codepage-summary-property"></a>Propriété de résumé de la page de codes

La propriété Résumé de la page de **codes** est la valeur numérique de la page de codes ANSI utilisée pour toutes les chaînes stockées dans les informations de résumé. Notez qu’il ne s’agit pas de la même page de codes pour les chaînes de la base de données d’installation. La propriété Résumé de la **page de codes** est utilisée pour convertir les chaînes dans les informations de synthèse en Unicode lors de l’appel des fonctions d’API Unicode. La propriété de résumé de la **page de codes** doit être définie avant que toutes les propriétés de chaîne soient définies dans les informations de résumé.
