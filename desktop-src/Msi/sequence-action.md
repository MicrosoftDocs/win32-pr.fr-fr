# <a name="sequence-action"></a>Action de séquence

L’action de séquence effectue le tri et l’exécution dans l’ordre des actions d’une table.

La propriété [**Sequence**](sequence.md) définit la table utilisée par cette action. La table doit avoir le même schéma que la [table InstallExecuteSequence](installexecutesequence-table.md). La table doit avoir une colonne action, condition et séquence.

## <a name="sequence-restrictions"></a>Restrictions de séquence

Il n’existe aucune restriction de séquence.

## <a name="actiondata-messages"></a>Messages ActionData

Il n’y a aucun message ActionData.
