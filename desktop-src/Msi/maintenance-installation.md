# <a name="maintenance-installation"></a>Installation de maintenance

Une fois qu’une application a été correctement installée, l’utilisateur peut souhaiter revenir à l’installation pour ajouter ou supprimer des fonctionnalités. C’est ce qu’on appelle une installation de maintenance.

Les [transformations](transforms.md) qui ont été appliquées à une installation doivent être disponibles lors des installations de maintenance suivantes.
