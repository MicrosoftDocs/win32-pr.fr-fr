# <a name="automation-interface"></a>Interface d’automatisation

Cette section s’adresse aux développeurs qui écrivent leurs propres programmes d’installation et qui souhaitent en savoir plus sur les tables de base de données Windows Installer.

Pour plus d’informations sur l’automatisation et l’accès aux bibliothèques C++, consultez [à propos de l’interface d’automatisation](about-the-automation-interface.md).

Pour plus d’informations sur l’utilisation d’Automation et la création de l' [**objet installer**](installer-object.md), consultez [utilisation de l’interface Automation](using-the-automation-interface.md).

Pour obtenir des documents de référence pour les objets du programme d’installation, consultez Référence de l' [interface Automation](automation-interface-reference.md).

Pour plus d’informations sur WMI, consultez [utilisation de Windows Installer avec WMI](using-windows-installer-with-wmi.md).
