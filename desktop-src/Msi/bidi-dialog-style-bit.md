# <a name="bidi-dialog-style-bit"></a>Bit de style de boîte de dialogue BiDi

Il s’agit d’une combinaison de l’ordre de lecture de droite à gauche [RTLRO](rtlro-dialog-style-bit.md), [RightAligned](rightaligned-dialog-style-bit.md)et du [style de boîte de dialogue LeftScroll](leftscroll-dialog-style-bit.md).

## <a name="value"></a>Valeur



| Decimal | Valeur hexadécimale | Constante                                                                                                                                       |
|---------|-------------|------------------------------------------------------------------------------------------------------------------------------------------------|
| 896     | 0x00000380  | **msidbDialogAttributesBiDi**  =  **msidbDialogAttributesRTLRO** \| **msidbDialogAttributesRightAligned** \| **msidbDialogAttributesLeftScroll** |
