# <a name="msicheckcrcs-property"></a>Propriété MSICHECKCRCS

le contrôle de redondance cyclique (CRC) est disponible avec Windows Installer version 2,0 et versions ultérieures. Le programme d’installation effectue un CRC sur les fichiers uniquement si la propriété **MSICHECKCRCS** est définie. Pour plus d’informations, consultez [vérification CRC durant une installation](crc-checking-during-an-installation.md).
