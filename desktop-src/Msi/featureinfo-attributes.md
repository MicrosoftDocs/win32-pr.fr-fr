# <a name="featureinfoattributes-property"></a>FeatureInfo. Attributes, propriété

La propriété **attributes** est une propriété en lecture-écriture qui retourne la valeur de la fonctionnalité dans la colonne attributs de la [table Feature](feature-table.md).

Cette propriété est en lecture/écriture.

## <a name="syntax"></a>Syntaxe


```JScript
propVal = FeatureInfo.Attributes
FeatureInfo.Attributes = propVal 
```



## <a name="property-value"></a>Valeur de la propriété
