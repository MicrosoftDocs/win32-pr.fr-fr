# <a name="componentusersid-property"></a>Component. UserSID, propriété

SID de l’utilisateur pour le composant énuméré. Cette propriété appelle la fonction [**MsiEnumComponentsEx**](/windows/desktop/api/Msi/nf-msi-msienumcomponentsexa) .

**[Windows Installer 4,5 ou version antérieure](not-supported-in-windows-installer-4-5.md):** Non pris en charge. Cette propriété est disponible à partir de Windows Installer 5,0.
