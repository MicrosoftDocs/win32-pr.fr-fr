# <a name="configurableitemcontext-property"></a>ConfigurableItem. Context, propriété

La propriété **Context** retourne la valeur de la colonne Context de la [table ModuleConfiguration](moduleconfiguration-table.md).

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```JScript
propVal = ConfigurableItem.Context
```
