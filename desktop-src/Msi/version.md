# <a name="version-windows-installer"></a>Version (Windows Installer)

Le type de données version est une chaîne de texte contenant une chaîne de version valide. Le format d’une chaîne de version est

``` syntax
xxxxx.xxxxx.xxxxx.xxxxx
```

où *x* est un chiffre.

La chaîne de version maximale acceptable est 65535.65535.65535.65535.

Voici des exemples de chaînes de version valides :

-   1
-   1,0
-   1.00
-   10.00
-   1.00.1
-   1.0.1
-   1.00.10
-   1.00.100
-   1.0.1000.0
