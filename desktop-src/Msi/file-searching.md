# <a name="file-searching"></a>Recherche de fichiers

Pendant l’installation, il est souvent nécessaire de rechercher un fichier spécifique. Le programme d’installation a quatre actions de recherche standard à cet effet. L' [action AppSearch](appsearch-action.md) utilise la [table AppSearch](appsearch-table.md) pour déterminer si un fichier existe sur le système d’un utilisateur. Pendant une installation de mise à niveau, l' [action CCPSearch](ccpsearch-action.md) et l' [action RMCCPSearch](rmccpsearch-action.md) peuvent utiliser leurs tables correspondantes pour vérifier le système de l’utilisateur pour une version précédente d’un produit.
