# <a name="ttcsupport-property"></a>Propriété TTCSupport

Le programme d’installation définit la propriété **TTCSupport** pour indiquer si le système d’exploitation prend en charge les fichiers. TTC (collections de polices de type true).

## <a name="remarks"></a>Remarques

cette propriété est définie si le programme d’installation détecte Windows 2000 ou l’un des systèmes d’exploitation suivants :

-   Japon-932
-   Chine-936
-   Taïwan-950
-   Corée-949
-   Hong Kong (R.A.S.)-950
