# <a name="ice85"></a>ICE85

ICE85 vérifie que la colonne SourceName de la [table MoveFile](movefile-table.md) est un nom de fichier long valide. Ce champ peut contenir des caractères génériques ( \* et ?).

## <a name="result"></a>Résultat

ICE85 publie les erreurs suivantes.



| Erreur ICE85                                                      | Description                                                              |
|------------------------------------------------------------------|--------------------------------------------------------------------------|
| \[ \] Le format de SourceName « 2 » trouvé dans la table MoveFile est incorrect. | Le champ SourceName dans la table MoveFile n’est pas un nom de fichier long valide |
