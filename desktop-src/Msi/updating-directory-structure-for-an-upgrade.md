# <a name="updating-directory-structure-for-an-upgrade"></a>Mise à jour de la structure de répertoires pour une mise à niveau

Utilisez l’éditeur de base de données Orca fourni avec le kit de développement logiciel (SDK), ou un autre éditeur, pour ouvrir la [table de répertoires](directory-table.md) dans MNP2001.msi. La structure de répertoires de l’exemple de mise à niveau est identique à la structure de répertoire du produit d’origine et aucune modification n’est requise dans la table de répertoire d’origine décrite dans spécification de la [structure de répertoires](specifying-directory-structure.md).
