# <a name="ice01"></a>ICE01

ICE01 valide le fonctionnement du mécanisme ICE. Cette glace utilise la propriété [**Time**](time.md) pour connaître l’heure et retourne l’heure système ou aucune.

## <a name="result"></a>Résultats

ICE01 publie un message indiquant l’heure à laquelle le programme d’installation a appelé ICE01. Cette glace ne doit jamais retourner une erreur.
