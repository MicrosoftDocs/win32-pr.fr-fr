# <a name="path"></a>Chemin d’accès

Le type de données Path est une chaîne de texte contenant un chemin d’accès complet valide.

La chaîne peut également contenir un nom de propriété entre crochets \[ \] . Dans ce cas, le nom de la propriété, y compris les crochets, est remplacé dans la chaîne par la valeur de la propriété.

Exemples :

-   Chemin UNC : \\ \\ serveur \\ partage
-   Lecteur local : c : \\ temp
-   Avec un nom de propriété : \[ lecteur \] \\ temp
