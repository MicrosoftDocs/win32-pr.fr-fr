# <a name="updatestarted-property"></a>Propriété UpdateStarted

Le programme d’installation définit la propriété **UpdateStarted** lorsque les modifications apportées au système ont commencé pour cette installation, y compris la reprise d’une installation interrompue.

Les instructions de condition de l’interface utilisateur utilisent cette valeur pour sélectionner une boîte de dialogue. Si cette propriété est définie, l’utilisateur est invité après une erreur et annule s’il faut restaurer ou continuer ultérieurement.
