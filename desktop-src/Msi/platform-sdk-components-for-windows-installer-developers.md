# <a name="windows-sdk-components-for-windows-installer-developers"></a>Composants SDK Windows pour les développeurs Windows Installer

Les composants du kit de développement logiciel (SDK) Windows Installer sont inclus dans le kit de développement logiciel (SDK) Microsoft Windows. Le SDK Windows comprend des composants redistribuables, de la documentation, l’outil de validation de la base de données du programme d’installation, l’éditeur de table de base de données, le schéma de base de données, les outils de développement, les outils Visual Basic Scripting Edition (VBScript), les exemples de produits et

Pour la page de téléchargement de la SDK Windows, consultez [Kit de développement logiciel (SDK) Microsoft Windows](https://developer.microsoft.com/windows/downloads/windows-10-sdk).
