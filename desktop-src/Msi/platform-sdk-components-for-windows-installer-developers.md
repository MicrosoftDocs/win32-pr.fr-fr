# <a name="windows-sdk-components-for-windows-installer-developers"></a>Windows composants du kit de développement logiciel pour les développeurs Windows Installer

les composants du kit de développement logiciel (SDK) Windows Installer sont inclus dans le kit de développement logiciel (SDK) de Microsoft Windows. le SDK Windows comprend les composants redistribuables, la documentation, l’outil de validation de la base de données du programme d’installation, l’éditeur de table de base de données, le schéma de base de données, les outils de développement, les outils de Visual Basic scripting Edition (VBScript), les exemples de produits et
