# <a name="authoring-merge-module-summary-information-streams"></a>Création de flux d’informations de résumé du module de fusion

Lors de la création d’un module de fusion, le flux d’informations de résumé est requis pour inclure ces propriétés d’informations de résumé requises.

Chaque module de fusion doit avoir un flux d’informations de résumé de module de fusion avec les propriétés d’informations de résumé suivantes :

-   [**Résumé du modèle**](template-summary.md)
-   [**Résumé du numéro de révision**](revision-number-summary.md)
-   [**Résumé du nombre de pages**](page-count-summary.md)
-   [**Résumé du nombre de mots**](word-count-summary.md)

Pour obtenir la liste complète des propriétés des informations de résumé dans les modules de fusion, consultez [Référence du flux de données résumées du module de fusion](merge-module-summary-information-stream-reference.md).
