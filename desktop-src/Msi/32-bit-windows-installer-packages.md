# <a name="32-bit-windows-installer-packages"></a>Packages de Windows Installer 32 bits

Un package 32 bits se compose uniquement de [composants Windows Installers](windows-installer-components.md) 32 bits et doit avoir la valeur « Intel » entrée dans le champ plateforme de la propriété [**Résumé du modèle**](template-summary.md) .

Pour plus d’informations, consultez [Windows Installer sur les systèmes d’exploitation 64 bits](windows-installer-on-64-bit-operating-systems.md) et les [packages Windows Installer 64 bits](64-bit-windows-installer-packages.md).
