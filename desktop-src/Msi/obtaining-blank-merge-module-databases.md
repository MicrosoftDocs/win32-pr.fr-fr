# <a name="obtaining-blank-merge-module-databases"></a>Obtention de bases de données de module de fusion vides

Obtenez une base de données de module de fusion vide. Vous pouvez utiliser le fichier Schema. msm fourni avec le kit de développement logiciel (SDK) Windows Installer en tant que base de données de départ pour votre module de fusion. Pour plus d’informations, consultez [SDK Windows Components for Windows Installer Developers](platform-sdk-components-for-windows-installer-developers.md).

Les développeurs doivent créer des modules de fusion à l’aide du schéma de base de données le plus simple qui installe leurs composants. L’utilisation d’un schéma simple garantit la plus grande compatibilité pour le module de fusion. La fusion d’un module de fusion dans un package d’installation avec un schéma de base de données différent entraîne généralement des conflits de fusion.

Pour obtenir la liste complète de toutes les tables obligatoires et facultatives dans les modules de fusion, consultez [tables de base de données de module de fusion](merge-module-database-tables.md).
