# <a name="configurableitemhelplocation-property"></a>ConfigurableItem. HelpLocation, propriété

La propriété **HelpLocation** retourne la valeur de la colonne HelpLocation de la [table ModuleConfiguration](moduleconfiguration-table.md).

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```JScript
propVal = ConfigurableItem.HelpLocation
```



## <a name="property-value"></a>Valeur de la propriété

## <a name="c"></a>C++

Consultez la fonction [**obtenir \_ HelpLocation**](/windows/desktop/api/Mergemod/nf-mergemod-imsmconfigurableitem-get_helplocation) .
