# <a name="about-installer-functions"></a>À propos des fonctions de programme d’installation

Windows Installer permet aux administrateurs système d’installer, de mettre à jour et de supprimer des composants. Pour plus d’informations sur l’ajout de fonctionnalités de programme d’installation, consultez [utilisation des fonctions de programme d’installation](using-installer-functions.md).
