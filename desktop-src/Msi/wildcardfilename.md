# <a name="wildcardfilename"></a>WildCardFilename

Le type de données WildCardFilename est un [nom de fichier](filename.md) qui peut également contenir les caractères génériques «  ? » pour tout caractère unique ou « \* » pour zéro occurrence, ou plus, d’un caractère quelconque.
