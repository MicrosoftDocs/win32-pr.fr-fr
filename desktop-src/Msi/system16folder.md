# <a name="system16folder-property"></a>Propriété System16Folder

Windows Le programme d’installation définit la propriété **System16Folder** sur le chemin d’accès complet au dossier des DLL système 16 bits. notez que la propriété **System16Folder** n’est pas définie pour le Windows 64 bits, car ce système ne prend pas en charge les applications ou les composants 16 bits.
