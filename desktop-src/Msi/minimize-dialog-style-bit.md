# <a name="minimize-dialog-style-bit"></a>Réduire le bit de style de boîte de dialogue

Si ce bit est défini, la boîte de dialogue peut être réduite. Ce bit est ignoré pour les boîtes de dialogue modales, qui ne peuvent pas être réduites.

## <a name="value"></a>Valeur



| Decimal | Valeur hexadécimale | Constante                          |
|---------|-------------|-----------------------------------|
| 4       | 0x00000004  | **msidbDialogAttributesMinimize** |
