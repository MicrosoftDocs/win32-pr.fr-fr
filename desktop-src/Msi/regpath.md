# <a name="regpath"></a>RegPath

Le type de données RegPath est une chaîne de texte contenant un chemin d’accès au registre. Les chemins d’accès au registre peuvent inclure des propriétés, comme avec le type de données [mis en forme](formatted.md) . Un RegPath ne peut pas commencer ou se terminer par une barre oblique inverse ( \\ ). La clé de \[ \# fichier \] et la \[ clé \] de $Component peuvent être incorporées dans le chemin d’accès ou précédées d’autres caractères.
