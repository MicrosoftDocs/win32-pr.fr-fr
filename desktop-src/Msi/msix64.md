# <a name="msix64-property"></a>Propriété Msix64

La propriété **Msix64** est définie uniquement en cas d’exécution sur un processeur x64. La valeur est définie par le programme d’installation au niveau du processeur numérique. Les valeurs sont les mêmes que celles du champ **wProcessorLevel** de la structure des [**\_ informations système**](/windows/win32/api/sysinfoapi/ns-sysinfoapi-system_info) .

## <a name="remarks"></a>Remarques

**Windows Installer 2,0 et Windows Installer 3,0 :** Non pris en charge. la propriété **Msix64** est disponible à partir de Windows Installer 3,1.
