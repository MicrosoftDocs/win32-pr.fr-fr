# <a name="mergemodulecabinet"></a>MergeModule.CABinet

Chaque fichier fourni par le module de fusion doit être stocké dans un fichier CAB qui est incorporé en tant que flux dans le stockage structuré du module de fusion. Dans un module de fusion standard, le nom de ce fichier cab est toujours : MergeModule.CABinet. Pour plus d’informations, consultez [génération de fichiers CAB inet MergeModule.CAB](generating-mergemodule-cabinet-cabinet-files.md).
