# <a name="mergemodulecabinet"></a>MergeModule. cab

Chaque fichier fourni par le module de fusion doit être stocké dans un fichier CAB qui est incorporé en tant que flux dans le stockage structuré du module de fusion. Dans un module de fusion standard, le nom de ce fichier cab est toujours : MergeModule. cab. Pour plus d’informations, consultez [génération de fichiers CAB MergeModule. cab](generating-mergemodule-cabinet-cabinet-files.md).
