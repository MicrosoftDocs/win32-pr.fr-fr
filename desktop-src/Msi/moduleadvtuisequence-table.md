# <a name="moduleadvtuisequence-table"></a>Table ModuleAdvtUISequence

Ne pas créer cette table. La Windows Installer n’utilise pas la [table AdvtUISequence](advtuisequence-table.md). La table AdvtUISequence ne doit pas exister ou doit être laissée vide dans la base de données d’installation. Vous ne devez pas non plus créer la table ModuleAdvtUISequence.
