# <a name="featureinfotitle-property"></a>FeatureInfo. title, propriété

La propriété **title** est une propriété en lecture seule qui retourne le titre de la fonctionnalité dans la colonne title de la [table Feature](feature-table.md).

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```JScript
propVal = FeatureInfo.Title
```



## <a name="property-value"></a>Valeur de la propriété
