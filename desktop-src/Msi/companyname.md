# <a name="companyname-property"></a>COMPANYNAME, propriété

La propriété **CompanyName** est l’organisation ou la société de l’utilisateur qui effectue l’installation.

## <a name="default-value"></a>Valeur par défaut

Une valeur par défaut peut être placée dans la [table de propriétés](property-table.md). Si la propriété **CompanyName** n’est pas définie, le programme d’installation la définit automatiquement à l’aide des valeurs du Registre.

## <a name="remarks"></a>Notes

Définissez la propriété [**NOCOMPANYNAME**](nocompanyname.md) pour supprimer le paramètre automatique **CompanyName**.
