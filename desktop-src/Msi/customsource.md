# <a name="customsource"></a>CustomSource

Le type de données CustomSource est un [identificateur](identifier.md) valide ou une clé externe, en fonction de la colonne type de la [table CustomAction](customaction-table.md). Il peut s’agir d’une clé externe dans la table [binaire](binary-table.md), la [table de fichiers](file-table.md), la table de [répertoires](directory-table.md)et la [table de propriétés](property-table.md). Notez que les clés externes dans la table de propriétés ne sont pas validées, car les propriétés peuvent être ajoutées au moment de l’exécution.
