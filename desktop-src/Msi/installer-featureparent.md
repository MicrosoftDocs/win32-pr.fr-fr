# <a name="installerfeatureparent-property"></a>Installer. FeatureParent, propriété

La propriété **FeatureParent** spécifie la fonctionnalité parente d’une fonctionnalité. Une chaîne vide pour cette propriété indique que la fonctionnalité n’a pas de parent et qu’il s’agit d’un élément racine.

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```JScript
propVal = Installer.FeatureParent
```



## <a name="property-value"></a>Valeur de la propriété
