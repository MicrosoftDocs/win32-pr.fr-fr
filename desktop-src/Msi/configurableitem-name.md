# <a name="configurableitemname-property"></a>Propriété ConfigurableItem.Name

La propriété **Name** de l’objet [**ConfigurableItem**](configurableitem-object.md) retourne la valeur de la colonne Name de la table ModuleConfiguration.

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```JScript
propVal = ConfigurableItem.Name
```



## <a name="property-value"></a>Valeur de la propriété

## <a name="c"></a>C++

Consultez la fonction [**obtenir un \_ nom**](/windows/desktop/api/Mergemod/nf-mergemod-imsmconfigurableitem-get_name) .
