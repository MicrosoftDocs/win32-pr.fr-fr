# <a name="authoring-modulecomponents-tables"></a>Création de tables ModuleComponents

Répertorie tous les composants fournis par le module de fusion dans la [table ModuleComponents](modulecomponents-table.md). Notez que les composants du module de fusion sont également spécifiés dans la table des composants. Pour plus d’informations, consultez [création de tables de composants de module de fusion](authoring-merge-module-component-tables.md).
