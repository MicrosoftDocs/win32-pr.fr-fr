# <a name="componentinfocomponentcode-property"></a>ComponentInfo. ComponentCode, propriété

Code du composant en question. Cette propriété appelle la fonction [**MsiGetComponentPathEx**](/windows/desktop/api/Msi/nf-msi-msigetcomponentpathexa) .

**[Windows Installer 4,5 ou version antérieure](not-supported-in-windows-installer-4-5.md):** Non pris en charge. cette propriété est disponible à partir de Windows Installer 5,0.
