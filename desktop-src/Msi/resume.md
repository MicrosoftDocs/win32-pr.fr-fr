# <a name="resume-property"></a>RESUME, propriété

La propriété **Resume** est définie lors de la reprise d’une installation interrompue. Cette propriété peut déterminer le texte à afficher dans l’interface utilisateur. Par exemple, lorsqu’il n’est pas en mode de reprise, l’utilisateur peut être invité à « êtes-vous prêt à installer ? » en mode de reprise, l’utilisateur peut être invité « êtes-vous prêt à terminer votre installation ? ».

## <a name="remarks"></a>Remarques

Pour la compatibilité descendante, le programme d’installation prend également en charge la **reprise** de nom pour cette propriété.
