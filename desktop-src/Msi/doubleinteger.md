# <a name="doubleinteger"></a>DoubleInteger

Le type de données DoubleInteger est une valeur entière de quatre octets. Sauf limitation contraire, la plage de valeurs autorisées est comprise entre – 2 147 483 647 et + 2 147 483 647.
