# <a name="logaction-property"></a>Propriété LOGACTION

La propriété **LOGACTION** est une liste de noms d’action séparés uniquement par des points-virgules et sans espaces. Les noms d’action dans cette propriété ne respectent pas la casse et la liste peut se terminer par un point-virgule.

## <a name="remarks"></a>Notes

Le programme d’installation enregistre les messages de données d’action pour les actions qui figurent dans la liste. La propriété n’a aucun effet sur les autres types de journalisation.
