# <a name="creating-application-summary-property"></a>Création de la propriété Résumé de l’application

La propriété de résumé de la création de l' **application** indique l’application qui a créé la base de données du programme d’installation. En général, la valeur de cette propriété Summary est le nom du logiciel utilisé pour créer cette base de données.

## <a name="requirements"></a>Configuration requise



| Condition requise | Valeur |
|--------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Version<br/> | Windows Installer 5,0 sur Windows Server 2012, Windows 8, Windows Server 2008 R2 ou Windows 7. Windows Installer 4,0 ou Windows Installer 4,5 sur Windows Server 2008 ou Windows Vista. Windows Installer sur Windows Server 2003 ou Windows XP<br/> |
