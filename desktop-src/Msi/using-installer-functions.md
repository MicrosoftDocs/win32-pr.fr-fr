# <a name="using-installer-functions"></a>Utilisation des fonctions de programme d’installation

pour ajouter Windows Installer fonctionnalité à votre application, utilisez les fonctions du service d’installation. Étant donné que le programme d’installation est un utilitaire d’installation et un système de gestion des composants, une application qui utilise le programme d’installation doit incorporer les fonctions du service d’installation dans le cadre du fonctionnement normal.

Les rubriques et procédures suivantes expliquent comment implémenter les fonctionnalités du programme d’installation dans votre application :

-   [Initialisation d’une application](initializing-an-application.md)
-   [Obtention d’informations sur l’application](getting-application-information.md)
-   [Demande d’une fonctionnalité](requesting-a-feature.md)
-   [Installation d’une application](installing-an-application.md)
-   [Réinstallation d’une fonctionnalité ou d’une application](reinstalling-a-feature-or-application.md)
-   [Suppression d’une application publiée](removing-an-advertised-application.md)
-   [Utilisation des fonctionnalités et des composants](working-with-features-and-components.md)
-   [Installation d’un composant manquant](installing-a-missing-component.md)
-   [Inventaire des produits et des correctifs](inventory-products-and-patches-.md)
-   [Gestion des sources d’installation](managing-installation-sources.md)
