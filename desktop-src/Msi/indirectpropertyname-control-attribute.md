# <a name="indirectpropertyname-control-attribute"></a>Attribut de contrôle IndirectPropertyName

Cet attribut est le nom d’une propriété indirecte associée au contrôle. Si le bit [indirect](indirect-control-attribute.md) est défini, le contrôle affiche ou modifie la valeur de la propriété portant ce nom. ce nom est la valeur de la propriété figurant dans la colonne propriété de la [table de contrôle](control-table.md).

Pour les contrôles inertes, cet attribut retourne la valeur null.

## <a name="valid-controls"></a>Contrôles valides

Tous les contrôles actifs.

## <a name="associated-bit-flags"></a>Indicateurs binaires associés

Cet attribut n’utilise pas d’indicateurs binaires.

## <a name="remarks"></a>Notes

Consultez les [attributs de contrôle](control-attributes.md) et le contrôle que vous devez créer sous [contrôles](controls.md).
