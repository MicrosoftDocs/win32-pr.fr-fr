# <a name="template"></a>Modèle

Le type de données du modèle est une chaîne de texte qui peut contenir des propriétés placées entre crochets \[ \] . Le type de modèle autorise tous les formats de type [mis en forme](formatted.md) , plus \[ *1* , \] où *1* est un nombre.
