# <a name="ice16"></a>ICE16

ICE16 vérifie que la valeur de la propriété [**ProductName**](productname.md) dans la [table Property](property-table.md) ne dépasse pas 63 caractères. Il n’existe aucune limite sur la longueur de la clé de Registre pour DisplayName.

## <a name="result"></a>Résultat

ICE16 publie un message d’erreur si la valeur ProductName définie dans la table de propriétés dépasse 63 caractères.
