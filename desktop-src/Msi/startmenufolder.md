# <a name="startmenufolder-property"></a>Propriété StartMenuFolder

Le programme d’installation définit la propriété **StartMenuFolder** sur le chemin d’accès complet du dossier du menu Démarrer. Par défaut, cette propriété est définie sur le dossier de l’utilisateur actuel. Si un profil « tous les utilisateurs » existe et que la propriété [**ALLUSERS**](allusers.md) est définie, cette propriété est définie sur le dossier dans le profil « tous les utilisateurs ».
