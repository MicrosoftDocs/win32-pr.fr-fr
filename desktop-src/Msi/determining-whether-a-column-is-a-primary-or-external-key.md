# <a name="determining-whether-a-column-is-a-primary-or-external-key"></a>Déterminer si une colonne est une clé primaire ou externe

Vous pouvez déterminer si une colonne spécifique est une clé primaire ou une clé externe de la table en consultant la page de référence de la table. Une liste de liens vers les pages de référence de toutes les tables de la base de données d’installation se trouve sous [tables de base de données](database-tables.md). Chacune des pages de référence identifie les colonnes qui sont une clé primaire ou une clé externe.

Les colonnes de base de données qui sont des clés externes prennent le nom de la colonne de clé primaire avec un trait de soulignement ajouté. Par exemple, une clé externe de la colonne file de la table file est toujours nommée file \_ .

Pour afficher les diagrammes des relations d’entité qui illustrent les relations entre les tables, consultez [groupe tables principales](core-tables-group.md), [groupe tables de fichiers](file-tables-group.md), [groupe tables de Registre](registry-tables-group.md), [groupe de tables système](system-tables-group.md)ou [schéma d’interface utilisateur](user-interface-schema.md).
