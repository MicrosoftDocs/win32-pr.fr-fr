# <a name="mergelog-method"></a>Méthode Merge. log

La méthode **log** de l’objet [**Merge**](merge-object.md) écrit une chaîne de texte dans le fichier journal actuellement ouvert.

## <a name="syntax"></a>Syntaxe


```JScript
Merge.Log(
  Message
)
```



## <a name="parameters"></a>Paramètres

<dl> <dt>

*Message* 
</dt> <dd>

Chaîne de texte à écrire dans le fichier journal.

</dd> </dl>

## <a name="return-value"></a>Valeur retournée

Cette méthode ne retourne pas de valeur.

## <a name="c"></a>C++

Consultez [**log**](/windows/win32/api/mergemod/nf-mergemod-imsmmerge-log) , fonction.
