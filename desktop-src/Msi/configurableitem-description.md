# <a name="configurableitemdescription-property"></a>Propriété ConfigurableItem. Description

La propriété **Description** retourne la valeur de la colonne Description de la [table ModuleConfiguration](moduleconfiguration-table.md).

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```JScript
propVal = ConfigurableItem.Description
```



## <a name="property-value"></a>Valeur de la propriété

## <a name="c"></a>C++
