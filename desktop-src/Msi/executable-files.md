# <a name="executable-files"></a>Fichiers exécutables

Une action personnalisée peut lancer un fichier exécutable stocké sur l’ordinateur de l’utilisateur ou contenu dans le package d’installation.

Les types d’actions personnalisées suivants lancent un fichier exécutable.



| Type d’action personnalisé                                 | Description                                         |
|----------------------------------------------------|-----------------------------------------------------|
| [Type d’action personnalisé 2](custom-action-type-2.md)   | Fichier EXE stocké dans un flux de table binaire.           |
| [Type d’action personnalisée 18](custom-action-type-18.md) | Fichier EXE installé avec un produit.                  |
| [Type d’action personnalisé 50](custom-action-type-50.md) | Fichier exécutable avec un chemin d’accès spécifié par une valeur de propriété. |
| [Type d’action personnalisé 34](custom-action-type-34.md) | Fichier EXE avec un chemin d’accès faisant référence à un répertoire.       |



 

Consultez la [liste récapitulative de tous les types d’actions personnalisées](summary-list-of-all-custom-action-types.md) pour obtenir un résumé de tous les types d’actions personnalisées et la façon dont elles sont encodées dans la [table CustomAction](customaction-table.md).
