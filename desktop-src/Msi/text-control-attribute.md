# <a name="text-control-attribute"></a>Attribut de contrôle de texte

Cet attribut est la chaîne de texte affichée dans le contrôle. Sur le paramètre, si le champ 0 de l’enregistrement n’a pas la valeur null, l’enregistrement est mis en forme à l’aide de FormatText. Si le champ 0 est null, le premier champ de l’enregistrement définit le texte. L’obtention de la valeur est toujours retournée dans le premier champ. Pour certains contrôles, ce texte peut ne pas être visible. dans Windows la touche d’accès rapide d’un contrôle est définie en plaçant un « & » devant le caractère souhaité dans cette chaîne.

## <a name="valid-controls"></a>Contrôles valides

Tous les contrôles à l’exception du [contrôle Line](line-control.md).

## <a name="associated-bit-flags"></a>Indicateurs binaires associés

Aucun.
