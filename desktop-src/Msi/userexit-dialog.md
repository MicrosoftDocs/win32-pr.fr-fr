# <a name="userexit-dialog"></a>Boîte de dialogue UserExit

Cette boîte de dialogue modale s’affiche à la fin d’une installation qui a été interrompue par l’utilisateur. Cette boîte de dialogue contient généralement uniquement du texte et un bouton **OK** . Le nom de cette boîte de dialogue doit apparaître dans la [table AdminUISequence](adminuisequence-table.md) et la [table InstallUISequence](installuisequence-table.md) avec-2 comme numéro de séquence.
