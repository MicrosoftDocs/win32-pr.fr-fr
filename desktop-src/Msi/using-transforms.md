# <a name="using-transforms"></a>Utilisation des transformations

Les transformations augmentent la flexibilité de l’installation d’applications en fournissant un moyen d’appliquer des modifications à une base de données sans modifier la base de données d’origine. Pour plus d’informations, consultez [transformations de base de données](database-transforms.md).

Les sections suivantes traitent de l’utilisation des transformations :

[Application de transformations](applying-transforms.md)

 

[Génération d’une transformation](generating-a-transform.md)

 

[Utilisation des transformations pour ajouter des ressources](using-transforms-to-add-resources.md)
