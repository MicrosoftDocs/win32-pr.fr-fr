# <a name="componentinfopath-property"></a>ComponentInfo. Path, propriété

Chemin d’accès du composant. Cette propriété appelle la fonction [**MsiGetComponentPathEx**](/windows/desktop/api/Msi/nf-msi-msigetcomponentpathexa) .

**[Windows Installer 4,5 ou version antérieure](not-supported-in-windows-installer-4-5.md):** Non pris en charge. cette propriété est disponible à partir de Windows Installer 5,0.
