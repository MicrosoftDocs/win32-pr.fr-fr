# <a name="table-names"></a>Noms de tables

Les noms de table sont limités à 31 caractères. Si un nom de table contient un point (pratique qui est autorisé mais non recommandé), le nom doit être placé entre des points d’accent graves \` (valeur ASCII 96), par exemple, \` nom_fichier. ext \` . Cette syntaxe d’échappement empêche les conflits entre le nom de la table et les mots réservés, tels que les mots clés SQL. Pour plus d’informations, consultez [syntaxe SQL](sql-syntax.md).
