# <a name="table-names"></a>Noms de tables

Les noms de table sont limités à 31 caractères. Si un nom de table contient un point (pratique qui est autorisé mais non recommandé), le nom doit être placé entre des points d’accent graves \` (valeur ASCII 96), par exemple, \` nom_fichier. ext \` . cette syntaxe d’échappement empêche les conflits entre le nom de la table et les mots réservés, par exemple les mots clés SQL. pour plus d’informations, consultez [SQL syntaxe](sql-syntax.md).
