# <a name="windows-installer-features"></a>Windows Fonctionnalités du programme d’installation

Une fonctionnalité fait partie de la fonctionnalité totale de l’application qu’un utilisateur reconnaît et peut décider d’installer de manière indépendante. Par exemple, une fonctionnalité peut être un vérificateur d’orthographe, un dictionnaire des synonymes ou un ensemble d’images clipart. Les relations hiérarchiques des fonctionnalités parent et enfant existent généralement de telle sorte que si une fonctionnalité enfant est installée, la fonctionnalité parente est également automatiquement installée.
