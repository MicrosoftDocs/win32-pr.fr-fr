# <a name="ice09"></a>ICE09

ICE09 vérifie que le bit permanent est défini pour chaque composant marqué pour l’installation dans le SystemFolder. ICE09 publie un avertissement, car vous devez éviter d’installer des composants système non permanents dans le SystemFolder. Pour éviter cet avertissement, définissez le bit permanent dans la colonne attributs de la [table des composants](component-table.md) pour chaque composant marqué pour l’installation dans le SystemFolder. Aucune validation n’est effectuée si la base de données n’a pas de table de composants.
