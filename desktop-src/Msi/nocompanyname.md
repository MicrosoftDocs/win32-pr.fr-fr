# <a name="nocompanyname-property"></a>Propriété NOCOMPANYNAME

Affectez la valeur 1 à la propriété **NOCOMPANYNAME** pour supprimer le paramètre automatique de la propriété [**CompanyName**](companyname.md) par le programme d’installation. Cette propriété est utilisée par les applications qui doivent collecter le nom de la société lors de la première exécution.

## <a name="default-value"></a>Valeur par défaut

Non défini. Le programme d’installation définit la propriété [**CompanyName**](companyname.md) automatiquement à l’aide des valeurs du Registre.
