# <a name="comments-summary-property"></a>Propriété Résumé des commentaires

La propriété **Résumé des commentaires** indique l’usage général du package d’installation, de la transformation ou du package de correctifs.

L’auteur d’un package d’installation, d’une transformation ou d’un package correctif doit définir la valeur de la propriété **Résumé des commentaires** sur l’une des valeurs suivantes :

<dl> « Cette base de données du programme d’installation contient la logique et les données requises pour installer <*produit*> ».  
« Cette transformation contient la logique et les données requises pour installer <*produit*> ».  
« Ce correctif contient la logique et les données requises pour installer <*produit*> ».  
</dl>

où <*produit*> correspond au nom du produit.
