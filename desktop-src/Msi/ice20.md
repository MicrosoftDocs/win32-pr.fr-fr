# <a name="ice20"></a>ICE20

ICE20 valide les boîtes de dialogue et les exigences de dialogue standard d’un package d’installation avec une interface utilisateur créée. Les packages d’installation avec une interface utilisateur créée ont une [table de boîtes de dialogue](dialog-table.md) et la propriété [**LIMITUI**](limitui.md) n’est pas définie.

ICE20 valide les tables et les spécifications suivantes.



| Boîte de dialogue standard                            | Conditions requises pour les boîtes de dialogue                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
|--------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Boîte de dialogue FilesInUse](filesinuse-dialog.md) | [Table de zone de liste](listbox-table.md). [Contrôle ListBox](listbox-control.md) avec la colonne de propriété de la [table de contrôle](control-table.md) définie sur FileInUseProcess.<br/> Un [contrôle PUSHBUTTON](pushbutton-control.md) avec EndDialog entré dans la colonne d’événement de la [table ControlEvent,](controlevent-table.md) et ignoré dans la colonne argument.<br/> Un [contrôle PUSHBUTTON](pushbutton-control.md) avec EndDialog entré dans la colonne d’événement de la [table ControlEvent,](controlevent-table.md) et s’arrête dans la colonne argument.<br/> Un [contrôle PUSHBUTTON](pushbutton-control.md) avec EndDialog entré dans la colonne d’événement de la [table ControlEvent,](controlevent-table.md) , puis réessayez dans la colonne argument.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| [Boîte de dialogue d’erreur](error-dialog.md)           | Le [bit de style](error-dialog-style-bit.md) de la boîte de dialogue d’erreur doit être défini pour spécifier la boîte de dialogue, avec n’importe quel nom, est une boîte de dialogue d’erreur. Contrôle de texte nommé ErrorText.<br/> ErrorText dans la \_ première colonne Control de la [table Dialog](dialog-table.md).<br/> Un contrôle PUSHBUTTON, nommé A, avec EndDialog dans la colonne Event de la [table ControlEvent,](controlevent-table.md) et ErrorAbort dans la colonne argument.<br/> Un contrôle PUSHBUTTON, nommé C, avec EndDialog dans la colonne Event de la [table ControlEvent,](controlevent-table.md) et ErrorCancel dans la colonne argument.<br/> Un contrôle PUSHBUTTON, nommé I, avec EndDialog dans la colonne Event de la [table ControlEvent,](controlevent-table.md) et ErrorIgnore dans la colonne argument.<br/> Un contrôle PUSHBUTTON, nommé N, avec EndDialog dans la colonne d’événement de la [table ControlEvent,](controlevent-table.md) et ErrorNo dans la colonne argument.<br/> Un contrôle PUSHBUTTON, nommé O, avec EndDialog dans la colonne Event de la [table ControlEvent,](controlevent-table.md) et ErrorOk dans la colonne argument.<br/> Un contrôle PUSHBUTTON, nommé R, avec EndDialog dans la colonne d’événement de la [table ControlEvent,](controlevent-table.md) et ErrorRetry dans la colonne argument.<br/> Un contrôle PUSHBUTTON, nommé Y, avec EndDialog dans la colonne d’événement de la [table ControlEvent,](controlevent-table.md) et ErrorYes dans la colonne argument.<br/> Les coordonnées de largeur, de hauteur et Y doivent être identiques pour tous les boutons. Les coordonnées X sont déterminées par le programme d’installation.<br/> Si un contrôle nommé ErrorIcon existe, il doit avoir une icône dans la colonne type de la [table de contrôle](control-table.md).<br/> |
| [Quitter la boîte de dialogue](exit-dialog.md)             | Il doit y avoir une action de dialogue ou personnalisée dans la [table AdminUISequence](adminuisequence-table.md) et la [table InstallUISequence](installuisequence-table.md) avec la valeur-1 dans la colonne séquence.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| [Boîte de dialogue UserExit](userexit-dialog.md)     | Il doit y avoir une action de dialogue ou personnalisée dans la [table AdminUISequence](adminuisequence-table.md) et la [table InstallUISequence](installuisequence-table.md) avec la valeur-2 dans la colonne séquence.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| [Boîte de dialogue FatalError](fatalerror-dialog.md) | Il doit y avoir une action de boîte de dialogue ou personnalisée dans la [table AdminUISequence](adminuisequence-table.md) et la [table InstallUISequence](installuisequence-table.md) avec une valeur de-3 dans la colonne séquence.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |



 

## <a name="result"></a>Résultats

ICE20 publie un message d’erreur s’il ne peut pas vérifier que les boîtes de dialogue et exigences standard sont présentes dans le package d’installation.
