# <a name="validation-automation"></a>Automation de validation

Vous pouvez utiliser la Evalcom2.dll pour implémenter des opérations de validation pour les packages d’installation et les modules de fusion.

Les sections suivantes de cette documentation contiennent des informations sur l’utilisation de la validation de Windows Installer à partir des outils de création :

-   [Utilisation de Evalcom2](using-evalcom2.md)
-   [Interfaces EvalCom2](evalcom2-interfaces.md)

Il est recommandé d’installer Evalcom2.dll à l’aide du Windows Installer. Le package Orca.msi fourni avec le kit de développement logiciel (SDK) Microsoft Windows pour Windows XP avec Service Pack 2 (SP2) ou une installation plus Evalcom2.dll. L’ID du composant qui contient l’interface COM est {53315473-B8D6-470A-9951-D9AE5C11FC91}.

Pour plus d’informations sur l’obtention de la SDK Windows, consultez [SDK Windows Components for Windows Installer Developers](platform-sdk-components-for-windows-installer-developers.md).
