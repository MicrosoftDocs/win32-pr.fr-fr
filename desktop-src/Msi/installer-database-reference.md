# <a name="installer-database-reference"></a>Référence de base de données du programme d’installation

pour ajouter des fonctionnalités d’installation à votre application, vous pouvez utiliser les tables de base de données Windows Installer et les fonctions de base de données du programme d’installation. Les rubriques de cette section décrivent chacune des fonctions de base de données et des tables de base de données.

Pour afficher la liste de toutes les tables de la base de données d’installation, consultez [tables de base de données](database-tables.md).

Pour afficher la liste des fonctions utilisées pour travailler avec les informations contenues dans les bases de données d’installation, consultez [fonctions de base de données](database-functions.md).
