# <a name="recordisnull-property"></a>Record. IsNull, propriété

La propriété **IsNull** de l’objet [**Record**](record-object.md) est une propriété en lecture seule qui retourne true si le champ indiqué est null et false si le champ contient des données.

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```JScript
propVal = Record.IsNull
```



## <a name="property-value"></a>Valeur de la propriété

Numéro de champ obligatoire de la valeur dans l’enregistrement, de base 1.
