# <a name="properties-windows-installer"></a>propriétés (Windows Installer)

les propriétés sont des variables globales que Windows Installer utilise pendant une installation. Les sections suivantes décrivent les propriétés utilisées par le programme d’installation :

-   [À propos des propriétés](about-properties.md)
-   [Utilisation de propriétés](using-properties.md)
-   [Référence de propriété](property-reference.md)

Le terme « propriété » fait également référence à un attribut d’un objet Automation. Voir [interface d’automatisation](automation-interface.md).
