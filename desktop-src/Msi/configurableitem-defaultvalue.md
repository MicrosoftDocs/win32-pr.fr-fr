# <a name="configurableitemdefaultvalue-property"></a>ConfigurableItem. DefaultValue, propriété

La propriété **DefaultValue** de l’objet [**ConfigurableItem**](configurableitem-object.md) retourne la valeur de la colonne DefaultValue de la [table ModuleConfiguration](moduleconfiguration-table.md).

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```JScript
propVal = ConfigurableItem.DefaultValue
```



## <a name="property-value"></a>Valeur de la propriété

## <a name="c"></a>C++

Consultez [**obtenir \_**](/windows/desktop/api/Mergemod/nf-mergemod-imsmconfigurableitem-get_defaultvalue) la fonction DefaultValue
