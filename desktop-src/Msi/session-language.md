# <a name="sessionlanguage-property"></a>Session. Language, propriété

La propriété **Language** de l’objet [**session**](session-object.md) est une propriété en lecture seule qui représente l’ID de langue numérique utilisé par la session d’installation en cours.

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```JScript
propVal = Session.Language
```



## <a name="property-value"></a>Valeur de la propriété
