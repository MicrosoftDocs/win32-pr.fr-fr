# <a name="featureinfodescription-property"></a>Propriété FeatureInfo. Description

La propriété **Description** est une propriété en lecture seule qui retourne la description de la fonctionnalité dans la colonne Description de la [table de fonctionnalités](feature-table.md).

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```JScript
propVal = FeatureInfo.Description
```



## <a name="property-value"></a>Valeur de la propriété
