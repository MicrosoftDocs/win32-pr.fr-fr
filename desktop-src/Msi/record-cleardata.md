# <a name="recordcleardata-method"></a>Record. ClearData, méthode

La méthode **ClearData** de l’objet [**Record**](record-object.md) efface les données de tous les champs, en leur affectant la valeur null. Tous les objets stockés dans les champs sont libérés.

## <a name="syntax"></a>Syntaxe


```JScript
Record.ClearData()
```



## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.

## <a name="return-value"></a>Valeur retournée

Cette méthode ne retourne pas de valeur.
