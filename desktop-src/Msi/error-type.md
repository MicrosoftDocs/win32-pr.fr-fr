# <a name="errortype-property"></a>Error. type, propriété

La propriété de **type** en lecture seule retourne l’une des valeurs de msmErrorType, indiquant le type d’erreur représenté par cet objet.

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```JScript
propVal = Error.Type
```



## <a name="property-value"></a>Valeur de la propriété

## <a name="c"></a>C++

Consultez fonction d' [**obtenir un \_ type de fonction**](/windows/win32/api/mergemod/nf-mergemod-imsmerror-get_type) (objet d’erreur).
