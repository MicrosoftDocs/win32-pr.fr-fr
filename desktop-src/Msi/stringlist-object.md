# <a name="stringlist-object"></a>Objet StringList

L’objet **StringList** est une collection de chaînes. Le client doit vérifier que l’objet **StringList** existe et n’est pas vide avant de référencer ses propriétés.

## <a name="members"></a>Membres

L’objet **StringList** possède les types de membres suivants :

-   [Propriétés](#properties)

### <a name="properties"></a>Propriétés

L’objet **StringList** a ces propriétés.



| Propriété                                     | Description                                                          |
|:---------------------------------------------|:---------------------------------------------------------------------|
| [**Count**](stringlist-count.md)<br/> | Retourne le nombre d’éléments dans l’objet **StringList** .<br/> |
| [**Élément**](stringlist-item.md)<br/>   | Retourne une chaîne dans la collection d’objets **StringList** .<br/> |
