# <a name="last-printed-summary-property"></a>Dernière propriété de résumé imprimée

Dans un package d’installation, la dernière propriété de **Résumé imprimée** peut être définie sur la date et l’heure d’une [installation d’administration](administrative-installation.md) à enregistrer lors de la création de l’image administrative. Pour les installations non administratives, cette propriété est identique à la propriété de résumé de la [**création de date/heure**](create-time-date-summary.md) .

Dans une transformation, cette propriété doit avoir la valeur null.

Dans un package correctif, cette propriété doit avoir la valeur null.

| Condition requise | Valeur |
