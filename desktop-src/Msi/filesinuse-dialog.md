# <a name="filesinuse-dialog"></a>Boîte de dialogue FilesInUse

Une boîte de dialogue FilesInUse avertit l’utilisateur des processus en cours d’exécution des fichiers qui doivent être remplacés ou supprimés par l’installation. Cela donne à l’utilisateur la possibilité d’arrêter ces processus et d’éviter d’avoir à redémarrer l’ordinateur pour terminer le remplacement ou la suppression de ces fichiers. Consultez [redémarrages du système](system-reboots.md).

Cette boîte de dialogue est créée comme requis par l' [action InstallValidate](installvalidate-action.md).
