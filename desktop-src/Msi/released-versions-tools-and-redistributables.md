# <a name="released-versions-tools-and-redistributables"></a>Versions, outils et redistribuables publiés

Les rubriques suivantes fournissent des informations supplémentaires sur les versions, les outils et les fichiers redistribuables de Windows Installer.

-   [SDK Windows Components for Windows Installer Developers](platform-sdk-components-for-windows-installer-developers.md) explique où obtenir les composants redistribuables, la documentation, l’outil de validation de base de données, l’éditeur de table de base de données, le schéma de base de données, les outils de développement, les outils VBScript, les exemples de produits et les exemples de code.
-   Les [Redistribuables Windows Installer](windows-installer-redistributables.md) fournissent des informations sur l’emplacement des composants redistribuables du programme d’installation.
-   [Versions publiées de Windows Installer](released-versions-of-windows-installer.md) répertorie les versions publiées du programme d’installation qui ont été livrées avec des systèmes d’exploitation différents.
-   [Windows Installer outils de développement](windows-installer-development-tools.md) contient des pages de référence pour l’utilisation des outils de développement du programme d’installation.
-   [Instmsi.exe](instmsi-exe.md) décrit le Instmsi qui est compatible avec le système d’exploitation. InstMsi est le package redistribuable pour l’installation du programme d’installation.
