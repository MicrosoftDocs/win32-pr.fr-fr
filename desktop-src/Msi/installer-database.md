# <a name="installer-database"></a>Base de données du programme d’installation

Ce document est principalement destiné aux développeurs qui souhaitent écrire des outils qui créent de nouveaux packages d’installation et qui ont besoin d’informations détaillées sur la base de données relationnelle du programme d’installation. Des packages d’installation complets sont adoptés par les développeurs d’applications comme un moyen efficace de gérer l’installation de leurs applications. La connaissance de la base de données d’installation est également utile pour les développeurs d’applications et les administrateurs qui souhaitent modifier des packages d’installation et des processus d’installation existants à l’aide d’actions personnalisées. pour obtenir des informations générales sur les Windows Installer et la technologie de gestion des composants, consultez [à propos de Windows Installer](about-windows-installer.md).

La base de données du programme d’installation contient toutes les informations nécessaires à l’installation d’une application. Par conséquent, les développeurs d’outils pour la création de packages d’installation et les développeurs qui créent des packages d’installation sans utiliser ces outils doivent comprendre la base de données du programme d’installation. Les principales zones des fonctionnalités de la base de données du programme d’installation sont présentées dans les rubriques suivantes :

[À propos de la base de données du programme d’installation](about-the-installer-database.md)

[Utilisation de la base de données du programme d’installation](using-the-installer-database.md)

[Référence de base de données du programme d’installation](installer-database-reference.md)
