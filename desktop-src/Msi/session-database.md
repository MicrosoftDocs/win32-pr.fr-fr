# <a name="sessiondatabase-property"></a>Session. Database, propriété

La propriété **de base de données** de l’objet [**session**](session-object.md) est une propriété en lecture seule qui retourne la base de données pour la session d’installation actuelle en tant qu’objet [**de base de données**](database-object.md) .

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```JScript
propVal = Session.Database
```



## <a name="property-value"></a>Valeur de la propriété
