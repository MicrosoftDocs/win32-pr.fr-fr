# <a name="ice-103"></a>ICE 103

ICE 103 vérifie les événements de contrôle [MsiPrint](msiprint-controlevent.md) et [MsiLaunchApp](msilaunchapp-controlevent.md) .

ICE 103 vérifie que l’événement de contrôle [MsiPrint](msiprint-controlevent.md) est utilisé uniquement sur une boîte de dialogue avec un contrôle [ScrollableText](scrollabletext-control.md) .

**[Windows Installer 4,5 ou version antérieure](not-supported-in-windows-installer-4-5.md):** Non pris en charge. cette glace est disponible à partir de Windows Installer 5,0.
