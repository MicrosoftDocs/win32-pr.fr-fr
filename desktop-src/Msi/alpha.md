# <a name="alpha-property"></a>Propriété alpha

La propriété **alpha** est définie uniquement en cas d’exécution sur un processeur Alpha. Le programme d’installation lui affecte la valeur du niveau de processeur numérique.

> [!Note]  
> La plateforme alpha n’est pas prise en charge par Windows Installer.

Windows Installer 4,0 ou Windows Installer 4,5 sur Windows Server 2008 ou Windows Vista. Windows Installer sur Windows Server 2003 ou Windows XP.
