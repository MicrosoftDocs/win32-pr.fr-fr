# <a name="installerpatchtransforms-property"></a>Installer. PatchTransforms, propriété

La propriété **PatchTransforms** en lecture seule retourne la liste des transformations délimitées par des points-virgules qui se trouvent dans le package de correctifs spécifié et appliquées au produit spécifié.

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```JScript
propVal = Installer.PatchTransforms
```



## <a name="property-value"></a>Valeur de la propriété
