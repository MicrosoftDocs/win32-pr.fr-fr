# <a name="mediasourcedir-property"></a>Propriété MediaSourceDir

Le programme d’installation définit la propriété **MediaSourceDir** sur 1 lorsque l’installation utilise une source située sur un support, tel qu’un CD-ROM. Cette propriété n’est pas définie lorsque l’installation utilise une source située à un emplacement réseau. Par exemple, le [contrôle SelectionTree](selectiontree-control.md) utilise **MediaSourceDir** pour déterminer si l’installation est exécutée à partir de la source ou s’il est exécuté à partir d’un emplacement réseau.
