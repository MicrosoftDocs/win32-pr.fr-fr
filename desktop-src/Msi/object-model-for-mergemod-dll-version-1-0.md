# <a name="object-model-for-mergemoddll-version-10"></a>Modèle objet pour Mergemod.dll version 1,0

Le modèle objet pour Mergemod.dll est organisé comme suit :

![modèle objet pour mergemod.dll](images/mergobj.png)

L' [objet de fusion](merge-object.md) est l’objet principal du modèle. L' [**objet GetFiles**](getfiles-object.md) est un objet secondaire. Les dépendances sont des collections d' [**objets de dépendance**](dependency-object.md). Les erreurs sont des collections d' [**objets Error**](error-object.md).
