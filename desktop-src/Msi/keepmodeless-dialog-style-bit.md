# <a name="keepmodeless-dialog-style-bit"></a>KeepModeless bit de style de boîte de dialogue

Normalement, lorsque ce bit n’est pas défini et qu’une boîte de dialogue est créée par action, toutes les autres boîtes de dialogue (généralement non modales) sont détruites. Si ce bit est défini, les autres boîtes de dialogue restent actives lorsque cette boîte de dialogue est créée.

## <a name="value"></a>Valeur



| Decimal | Valeur hexadécimale | Constante                              |
|---------|-------------|---------------------------------------|
| 16      | 0x00000010  | **msidbDialogAttributesKeepModeless** |
