# <a name="internal-consistency-evaluators---ices"></a><span data-ttu-id="ca196-103">Évaluateurs de cohérence internes-CIEM</span><span class="sxs-lookup"><span data-stu-id="ca196-103">Internal Consistency Evaluators - ICEs</span></span>

<span data-ttu-id="ca196-104">Les évaluateurs de cohérence internes, également appelés « ICEs », sont des actions personnalisées écrites en VBScript, JScript, ou en tant que DLL ou EXE.</span><span class="sxs-lookup"><span data-stu-id="ca196-104">Internal consistency evaluators, also called ICEs, are custom actions written in VBScript, JScript, or as a DLL or EXE.</span></span> <span data-ttu-id="ca196-105">Lorsque ces actions personnalisées sont exécutées, elles analysent la base de données à la recherche d’entrées dans les enregistrements de base de données qui sont valides lorsqu’elles sont examinées individuellement, mais qui peuvent provoquer un comportement incorrect dans le contexte de la base de données entière</span><span class="sxs-lookup"><span data-stu-id="ca196-105">When these custom actions are executed, they scan the database for entries in database records that are valid when examined individually but that may cause incorrect behavior in the context of the whole database.</span></span> <span data-ttu-id="ca196-106">Notez que cela est différent de la validation effectuée sur des enregistrements individuels à l’aide de [**MsiViewModify**](/windows/desktop/api/Msiquery/nf-msiquery-msiviewmodify).</span><span class="sxs-lookup"><span data-stu-id="ca196-106">Note that this is different than the validation done on individual records using [**MsiViewModify**](/windows/desktop/api/Msiquery/nf-msiquery-msiviewmodify).</span></span>

<span data-ttu-id="ca196-107">Par exemple, la [table des composants](component-table.md) peut répertorier plusieurs composants qui sont tous valides lorsqu’ils sont testés individuellement avec [**MsiViewModify**](/windows/desktop/api/Msiquery/nf-msiquery-msiviewmodify).</span><span class="sxs-lookup"><span data-stu-id="ca196-107">For example, the [Component table](component-table.md) may list several components that are all valid when tested individually with [**MsiViewModify**](/windows/desktop/api/Msiquery/nf-msiquery-msiviewmodify).</span></span> <span data-ttu-id="ca196-108">Toutefois, **MsiViewModify** n’intercepte pas l’erreur lorsque deux composants utilisent le même [GUID](guid.md) que leur code de composant.</span><span class="sxs-lookup"><span data-stu-id="ca196-108">However, **MsiViewModify** would not catch the error when two components use the same [GUID](guid.md) as their component code.</span></span> <span data-ttu-id="ca196-109">L’action personnalisée [ICE08](ice08.md) est conçue pour valider que la table des composants ne contient pas de GUID de code de composant en double.</span><span class="sxs-lookup"><span data-stu-id="ca196-109">The custom action [ICE08](ice08.md) is designed to validate that the Component table does not contain duplicate component code GUIDs.</span></span>

<span data-ttu-id="ca196-110">Les actions personnalisées ICE retournent quatre types de messages :</span><span class="sxs-lookup"><span data-stu-id="ca196-110">ICE custom actions return four kinds of messages:</span></span>

-   <span data-ttu-id="ca196-111">[**Erreurs**](merge-errors.md) Les messages d’erreur signalent la création de base de données qui provoque un comportement incorrect.</span><span class="sxs-lookup"><span data-stu-id="ca196-111">[**Errors**](merge-errors.md) Error messages report database authoring that cause incorrect behavior.</span></span> <span data-ttu-id="ca196-112">Par exemple, les GUID de composant en double provoquent l’inscription incorrecte des composants par le programme d’installation.</span><span class="sxs-lookup"><span data-stu-id="ca196-112">For example, duplicate component GUIDs cause the installer to incorrectly register components.</span></span>
-   <span data-ttu-id="ca196-113">**Avertissements** Les messages d’avertissement signalent la création de base de données qui provoque un comportement incorrect dans certains cas.</span><span class="sxs-lookup"><span data-stu-id="ca196-113">**Warnings** Warning messages report database authoring that causes incorrect behavior in certain cases.</span></span> <span data-ttu-id="ca196-114">Les avertissements peuvent également signaler des effets secondaires inattendus de la création de bases de données.</span><span class="sxs-lookup"><span data-stu-id="ca196-114">Warnings can also report unexpected side-effects of database authoring.</span></span> <span data-ttu-id="ca196-115">Par exemple, si vous entrez le même nom de propriété dans deux conditions qui diffèrent uniquement par la casse des lettres du nom.</span><span class="sxs-lookup"><span data-stu-id="ca196-115">For example, entering the same property name in two conditions that differ only by the case of letters in the name.</span></span> <span data-ttu-id="ca196-116">Étant donné que le programme d’installation respecte la casse, le programme d’installation traite ces derniers comme des propriétés différentes.</span><span class="sxs-lookup"><span data-stu-id="ca196-116">Because the installer is case-sensitive, the installer treats these as different properties.</span></span>
-   <span data-ttu-id="ca196-117">**Échecs** Les messages d’échec signalent l’échec de l’action personnalisée ICE.</span><span class="sxs-lookup"><span data-stu-id="ca196-117">**Failures** Failure messages report the failure of the ICE custom action.</span></span> <span data-ttu-id="ca196-118">L’échec est généralement dû à une base de données avec des problèmes graves que la glace ne peut même pas exécuter.</span><span class="sxs-lookup"><span data-stu-id="ca196-118">Failure is commonly caused by a database with such severe problems that the ICE cannot even run.</span></span>
-   <span data-ttu-id="ca196-119">**Informations** Les messages d’information fournissent des informations à partir de la glace et n’indiquent pas de problème avec la base de données.</span><span class="sxs-lookup"><span data-stu-id="ca196-119">**Informational** Informational messages provide information from the ICE and do not indicate a problem with the database.</span></span> <span data-ttu-id="ca196-120">Il s’agit souvent d’informations sur la glace elle-même, par exemple une brève description.</span><span class="sxs-lookup"><span data-stu-id="ca196-120">Often they are information about the ICE itself, such as a brief description.</span></span> <span data-ttu-id="ca196-121">Ils peuvent également fournir des informations de progression au fur et à mesure de l’exécution de la glace.</span><span class="sxs-lookup"><span data-stu-id="ca196-121">They can also provide progress information as the ICE runs.</span></span>
