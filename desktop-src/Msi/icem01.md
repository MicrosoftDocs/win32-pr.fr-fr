# <a name="icem01"></a>ICEM01

ICEM01 valide le fonctionnement du mécanisme ICE. Cette glace utilise la propriété [**Time**](time.md) pour connaître l’heure et retourne l’heure système ou aucune.

Le module de fusion CIEM est stocké dans un fichier de module de fusion. CUB appelé Mergemod. CUB et non dans le fichier. CUB contenant le CIEM utilisé pour la validation du package.

## <a name="result"></a>Résultats

ICEM01 publie un message indiquant l’heure à laquelle le programme d’installation a appelé ICEM01. Cette glace ne doit jamais retourner une erreur.
