# <a name="sessionproductproperty-property"></a>Session. ProductProperty, propriété

La propriété **ProductProperty** de l’objet [**session**](session-object.md) est une propriété en lecture seule qui récupère les propriétés du produit de la base de données du produit.

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```JScript
propVal = Session.ProductProperty
```



## <a name="property-value"></a>Valeur de la propriété

Nom de propriété obligatoire qui respecte la casse.
