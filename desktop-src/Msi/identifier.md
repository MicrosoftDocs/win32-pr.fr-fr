# <a name="identifier"></a>Identificateur

Le type de données de l’identificateur est une chaîne de texte. Les identificateurs peuvent contenir des caractères ASCII A-Z (a-z), des chiffres, des traits de soulignement ( \_ ) ou des points (.). Toutefois, chaque identificateur doit commencer par une lettre ou un trait de soulignement.
