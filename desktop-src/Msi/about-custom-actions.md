# <a name="about-custom-actions"></a>À propos des actions personnalisées

Bien que les actions standard soient suffisantes pour exécuter une installation dans la plupart des cas, les actions personnalisées permettent à l’auteur d’un package d’installation d’étendre les fonctionnalités des actions standard en incluant des fichiers exécutables, des bibliothèques de liens dynamiques et des scripts. Les actions personnalisées ne peuvent pas être utilisées dans les tables de séquence utilisées pour la publication, la [table AdvtUISequence](advtuisequence-table.md) et la [table AdvtExecuteSequence](advtexecutesequence-table.md).
