# <a name="admintoolsfolder-property"></a>Propriété AdminToolsFolder

La propriété **AdminToolsFolder** contient le chemin d’accès complet au répertoire du système de fichiers qui stocke les outils d’administration. La console MMC (Microsoft Management Console) enregistre les consoles personnalisées dans ce répertoire et se déplace avec l’utilisateur. Si la propriété [**ALLUSERS**](allusers.md) est définie, cette propriété pointe vers le répertoire du système de fichiers qui contient les outils d’administration pour tous les utilisateurs de l’ordinateur.
