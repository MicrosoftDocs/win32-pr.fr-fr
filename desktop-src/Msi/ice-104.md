# <a name="ice-104"></a>ICE 104

ICE 104 vérifie qu’une seule des deux tables : [MsiLockPermissionsEx](msilockpermissionsex-table.md) et [LockPermissions](lockpermissions-table.md) est présente dans le package.

ICE 104 vérifie la syntaxe utilisée dans les champs LockObject, table et SDDLText de la table [MsiLockPermissionsEx](msilockpermissionsex-table.md) . ICE 104 ne vérifie pas si la valeur du champ SDDLText de la table MsiLockPermissionsEx est un descripteur de sécurité valide.

**[Windows Installer 4,5 ou version antérieure](not-supported-in-windows-installer-4-5.md):** Non pris en charge. cette glace est disponible à partir de Windows Installer 5,0.

## <a name="result"></a>Résultat

ICE 104 publie les erreurs suivantes.



| Erreur ICE104                                                                                                                            | Description                                                                                                                                                                             |
|-----------------------------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Les tables LockPermissions et MsiLockPermissionsEx existent dans la base de données. Une seule table de ce type doit être présente.                      | Un package ne peut pas contenir à la fois les tables [MsiLockPermissionsEx](msilockpermissionsex-table.md) et [LockPermissions](lockpermissions-table.md) .                                       |
| Impossible de trouver l’élément' \[ 2 \] 'dans la table' \[ 3 ', \] qui est référencée dans l’entrée de la table MsiLockPermissionsEx (MsiLockPermissionsEx =) \[ 1\] | Windows Le programme d’installation ne parvient pas à trouver et à sécuriser un élément spécifié dans la table [MsiLockPermissionsEx](msilockpermissionsex-table.md) .                                                  |
| Dans l’entrée de la table MsiLockPermissionsEx (MsiLockPermissionsEx =) \[ 1 \] , SDDLText =% s semble être une valeur FormattedSDDLText non valide  | La valeur du champ SDDLText de la table [MsiLockPermissionsEx](msilockpermissionsex-table.md) doit être un type de données [**FormattedSDDLText**](formattedsddltext.md) valide. |
