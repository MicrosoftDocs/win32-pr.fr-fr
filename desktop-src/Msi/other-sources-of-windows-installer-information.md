## <a name="orca-documentation"></a>Documentation Orca

[Orca](orca-exe.md) est un éditeur de fichier gui. msi fourni avec le kit de développement logiciel (SDK) Windows Installer. Il fournit un accès complet aux tables de base de données Windows Installer.

Un ensemble de fichiers d’aide est fourni avec Orca. Pour obtenir la documentation orca.exe et Orca, vous devez télécharger et installer le kit de développement logiciel (SDK) Windows Installer, fourni sous la forme d’un fichier de Orca.msi. Après avoir installé le [Kit de développement logiciel (SDK) Microsoft Windows](https://developer.microsoft.com/windows/downloads/windows-10-sdk/), double-cliquez sur le fichier Orca.msi pour installer Orca.

> [!Note]  
> Il n’est pas possible de télécharger Orca séparément.

 

Bien qu’Orca donne accès à toutes les fonctionnalités de la Windows Installer, il n’est pas destiné à remplacer un environnement de création de package complet. Dans de nombreux cas, il est plus facile de créer une installation Windows Installer pour une application à l’aide de l’un des outils de création de package commerciaux disponibles auprès des éditeurs de logiciels indépendants.

## <a name="windows-installer-software-vendors"></a>Fournisseurs de logiciels Windows Installer

Vous pouvez contacter un éditeur de logiciels indépendant à propos des outils pour créer un package Windows Installer pour une application. Ces outils peuvent fournir un environnement de création de packages qui peut être plus facile à utiliser que les outils fournis dans le kit de développement logiciel (SDK) Windows Installer.
