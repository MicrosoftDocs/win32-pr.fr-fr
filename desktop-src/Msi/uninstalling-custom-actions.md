# <a name="uninstalling-custom-actions"></a>Désinstallation des actions personnalisées

Les développeurs de packages d’installation qui utilisent des actions personnalisées doivent déterminer, dans chaque cas, comment faire en sorte que l’action personnalisée fonctionne correctement lorsque le package est désinstallé. Il revient à l’auteur du package de décider s’il faut conditionner l’action personnalisée à exécuter lors de la désinstallation, de ne pas l’exécuter lors de la désinstallation ou d’exécuter une autre action personnalisée lors de la désinstallation.
