# <a name="dependencylanguage-property"></a>Dependency. Language, propriété

La propriété **Language** en lecture seule retourne l' **ID de langue** du module requis.

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```JScript
propVal = Dependency.Language
```



## <a name="property-value"></a>Valeur de la propriété

## <a name="c"></a>C++

Consultez [**obtenir une \_ fonction de langage (objet de dépendance)**](/windows/win32/api/mergemod/nf-mergemod-imsmdependency-get_language).
