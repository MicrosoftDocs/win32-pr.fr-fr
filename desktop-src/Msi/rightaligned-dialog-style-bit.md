# <a name="rightaligned-dialog-style-bit"></a>RightAligned bit de style de boîte de dialogue

Si ce bit de style est défini, le texte est aligné sur le côté droit de la boîte de dialogue.

## <a name="value"></a>Valeur



| Decimal | Valeur hexadécimale | Constante                              |
|---------|-------------|---------------------------------------|
| 256     | 0x00000100  | **msidbDialogAttributesRightAligned** |
