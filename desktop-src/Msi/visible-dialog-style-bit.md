# <a name="visible-dialog-style-bit"></a>Bit de style de boîte de dialogue visible

Si ce bit est défini, la boîte de dialogue est créée à l’origine comme visible ; sinon, elle est masquée.

## <a name="value"></a>Valeur



| Decimal | Valeur hexadécimale | Constante                         |
|---------|-------------|----------------------------------|
| 1       | 0x00000001  | **msidbDialogAttributesVisible** |
