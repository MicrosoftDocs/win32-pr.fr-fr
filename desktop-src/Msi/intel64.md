# <a name="intel64-property"></a>Propriété Intel64

La propriété **Intel64** est définie uniquement lorsqu’elle s’exécute sur un processeur Itanium. La valeur est définie par le Windows Installer au niveau du processeur numérique. Cette propriété n’est pas définie lors de l’exécution sur un processeur étendu 64 bits. Pour plus d’informations, consultez [**Intel**](intel.md) , propriété.
