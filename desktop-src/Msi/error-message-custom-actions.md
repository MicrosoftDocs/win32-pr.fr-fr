# <a name="error-message-custom-actions"></a>Actions personnalisées de message d’erreur

Ce type d’action personnalisée affiche un message d’erreur spécifié et retourne un échec, en terminant l’installation. Le message d’erreur à afficher peut être fourni sous la forme d’une chaîne ou d’un index dans la [table d’erreurs](error-table.md).



| Type d’action personnalisé                                 | Description                                                |
|----------------------------------------------------|------------------------------------------------------------|
| [Type d’action personnalisée 19](custom-action-type-19.md) | Met fin à l’installation et affiche un message d’erreur. |



 

Consultez la [liste récapitulative de tous les types d’actions personnalisées](summary-list-of-all-custom-action-types.md) pour obtenir un résumé de l’encodage de chacun des types d’actions personnalisées dans la [table CustomAction](customaction-table.md).
