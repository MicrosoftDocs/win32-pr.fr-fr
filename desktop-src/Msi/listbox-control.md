# <a name="listbox-control"></a>ListBox, contrôle

Le contrôle ListBox est une zone de liste standard qui permet à l’utilisateur d’effectuer une sélection unique à partir d’une liste de valeurs prédéterminées. Les valeurs possibles sont lues à partir de la [table ListBox](listbox-table.md). Vous pouvez associer une propriété de type chaîne ou entier en entrant le nom de la propriété dans la colonne propriété de la [table de contrôle](control-table.md).

## <a name="control-attributes"></a>Attributs du contrôle

Entrez l’identificateur du ControlEvent, dans la colonne d’événement.

## <a name="remarks"></a>Notes

Ce contrôle peut être créé à partir de la classe LISTBOX à l’aide de la fonction [**CreateWindowEx**](/windows/win32/api/winuser/nf-winuser-createwindowexa) . Il possède les styles **WS \_ TABSTOP**, **WS \_ Group** et **WS \_ Child** . Si le bit de style de contrôle trié est activé, le contrôle est créé avec les styles **\_ Notify**, **WS \_ VSCROLL** et **WS \_ Border** . dans le cas contraire, le contrôle est créé avec le style **\_ standard lbs** .
