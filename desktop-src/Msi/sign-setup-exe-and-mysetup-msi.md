# <a name="sign-setupexe-and-mysetupmsi"></a>Signer Setup.exe et MySetup.msi

Avant de placer Setup.exe et MySetup.msi sur le serveur Web, vous devez signer les fichiers avec votre certificat numérique et votre clé privée, MyCert. cer et MyCert. pvk, à l’aide de l’utilitaire [SignTool](../seccrypto/signtool.md) . pour plus d’informations sur l’utilisation de l’utilitaire SignTool, consultez les informations de référence sur les [outils CryptoAPI](../seccrypto/cryptoapi-tools-reference.md) dans le kit de développement logiciel (SDK) de Microsoft Windows.
