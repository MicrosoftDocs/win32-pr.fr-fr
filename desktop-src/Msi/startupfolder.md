# <a name="startupfolder-property"></a>Propriété StartupFolder

Le programme d’installation définit la propriété **startupfolder** sur le chemin d’accès complet du dossier Startup. Par défaut, cette propriété est définie sur le dossier de l’utilisateur actuel. Si un profil « tous les utilisateurs » existe et que la propriété [**ALLUSERS**](allusers.md) est définie, cette propriété est définie sur le dossier dans le profil « tous les utilisateurs ».
