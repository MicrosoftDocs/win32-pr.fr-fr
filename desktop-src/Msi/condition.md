# <a name="condition"></a>Condition

Le type de données condition est une chaîne de texte contenant une instruction conditionnelle valide qui peut être évaluée comme true ou false. Pour plus d’informations sur la syntaxe des instructions conditionnelles, consultez [syntaxe d’instruction conditionnelle](conditional-statement-syntax.md).
