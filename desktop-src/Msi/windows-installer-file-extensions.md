# <a name="windows-installer-file-name-extensions"></a>Windows Installer les extensions de nom de fichier

Les extensions de nom de fichier suivantes sont utilisées avec le Windows Installer.



| Extension | Description                                |
|-----------|--------------------------------------------|
| .msi      | Windows Installer base de données.                |
| . msm      | Windows Installer module de fusion.            |
| .msp      | Windows Installer correctif logiciel.                   |
| .mst      | Transformation de Windows Installer.               |
| . IDT      | Windows Installer la table de base de données exportée. |
| . CUB      | Module de validation.                         |
| . PCP      | Fichier de création de correctif logiciel Windows Installer      |
