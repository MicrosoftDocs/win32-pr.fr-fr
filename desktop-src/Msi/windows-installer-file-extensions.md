# <a name="windows-installer-file-name-extensions"></a>Windows Extensions de nom de fichier du programme d’installation

les extensions de nom de fichier suivantes sont utilisées avec le Windows Installer.



| Extension | Description                                |
|-----------|--------------------------------------------|
| .msi      | Windows Base de données du programme d’installation.                |
| . msm      | Windows Module de fusion du programme d’installation.            |
| .msp      | Windows Correctif du programme d’installation.                   |
| .mst      | Windows Transformation du programme d’installation.               |
| . IDT      | Windows Installer la Table de base de données exportée. |
| . CUB      | Module de validation.                         |
| . PCP      | Windows Fichier de création des correctifs du programme d’installation      |
