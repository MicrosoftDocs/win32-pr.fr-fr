# <a name="exit-dialog"></a>Quitter la boîte de dialogue

La boîte de dialogue quitter s’affiche à la fin d’une installation réussie. Cette boîte de dialogue contient généralement uniquement du texte et un bouton **OK** . Le nom de cette boîte de dialogue doit apparaître dans la [table AdminUISequence](adminuisequence-table.md) et la [table InstallUISequence](installuisequence-table.md) avec-1 comme numéro de séquence. Il s’agit d’une boîte de dialogue modale.
