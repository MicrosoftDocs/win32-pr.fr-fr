# <a name="windows-installer-and-logo-requirements"></a>Windows Configuration requise pour le programme d’installation et le logo

le Logo certifié pour Microsoft Windows identifie les produits qui ont été vérifiés par le biais de tests indépendants pour se conformer aux spécifications de l’Application pour Windows. Cette spécification a été développée par Microsoft en collaboration avec les clients et d’autres développeurs pour fournir une carte routière permettant de créer des applications fiables et gérables. les éditeurs de logiciels qui se conforment à la spécification sont éligibles pour le logo de la Windows certifiée pour Microsoft, puis ils disposent d’une licence pour leur utilisation sur le packaging, la publicité, la documentation et d’autres documents marketing du produit.

pour plus d’informations sur la certification Windows Vista, Windows XP et Windows Server 2008 pour vos applications, consultez le [Microsoft Partner Network]( https://partner.microsoft.com).

pour plus d’informations sur la certification Windows Server 2003 et Windows 2000 pour vos applications, consultez [https://msdn.microsoft.com/certification](https://www.microsoft.com/windowsserver2003/partners/isvs/cfw.mspx) .

si vous créez un package d’installation pour votre application, vous pouvez tirer parti des fonctionnalités de la Microsoft Windows Installer pour répondre à plusieurs exigences de certification. le tableau suivant fournit des liens vers des informations dans Windows Installer documentation en rapport avec certaines des conditions de certification. notez que les exigences non toutes pour la certification abordées dans la spécification de l’Application dépendent de Windows Installer.
