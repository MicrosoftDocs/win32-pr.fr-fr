# <a name="hasborder-control-attribute"></a>Attribut de contrôle HasBorder

Si ce bit est défini, le [RadioButtonGroup](radiobuttongroup-control.md) contient du texte et une bordure qui l’entoure.

Si le bit de style n’est pas défini, la bordure n’est pas affichée et aucun texte n’est affiché sur le groupe.

## <a name="valid-controls"></a>Contrôles valides

[RadioButtonGroup](radiobuttongroup-control.md)

## <a name="valid"></a>Valide



| Decimal  | Valeur hexadécimale | Constante                            |
|----------|-------------|-------------------------------------|
| 16 777 216 | 0x01000000  | **msidbControlAttributesHasBorder** |



 

## <a name="remarks"></a>Notes

Pour définir cet attribut sur un contrôle, incluez les bits HasBorder dans la colonne attributs de l’enregistrement du contrôle dans la [table de contrôle](control-table.md).

Consultez les [attributs de contrôle](control-attributes.md) et le contrôle que vous devez créer sous [contrôles](controls.md).
