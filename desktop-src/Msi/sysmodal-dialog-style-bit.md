# <a name="sysmodal-dialog-style-bit"></a>SysModal bit de style de boîte de dialogue

Si ce bit de style est défini, la boîte de dialogue arrête toutes les autres applications et aucune autre application ne peut prendre le focus. Cet État est conservé jusqu’à la fermeture de la boîte de dialogue SysModal.

## <a name="value"></a>Valeur



| Decimal | Valeur hexadécimale | Constante                          |
|---------|-------------|-----------------------------------|
| 8       | 0x00000008  | **msidbDialogAttributesSysModal** |
