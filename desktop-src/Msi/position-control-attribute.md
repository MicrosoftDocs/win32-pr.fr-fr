# <a name="position-control-attribute"></a>Position (attribut de contrôle)

Cet attribut détermine la position du contrôle dans la boîte de dialogue. Les quatre champs de l’attribut sont : gauche, haut, largeur et hauteur. Entrez la largeur, la hauteur et les coordonnées du contrôle dans la largeur, la hauteur, le X et les colonnes Y de la [table de contrôle](control-table.md). Utilisez les [unités d’installation](installer-units.md) pour les unités de longueur et de distance.

## <a name="valid-controls"></a>Contrôles valides

Tous les contrôles actifs.

## <a name="associated-bit-flags"></a>Indicateurs binaires associés

Cet attribut n’utilise pas d’indicateurs binaires.

## <a name="remarks"></a>Remarques

Consultez les [attributs de contrôle](control-attributes.md) et le contrôle que vous devez créer sous [contrôles](controls.md).

Une unité d’installation est égale à 1 douzième la hauteur de la taille de police MS sans serif à 10 points.
