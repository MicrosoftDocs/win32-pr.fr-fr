# <a name="programmenufolder-property"></a>Propriété ProgramMenuFolder

Le programme d’installation définit la propriété **ProgramMenuFolder** sur le chemin d’accès complet du dossier de menu de programme de l’utilisateur actuel. Si un profil « tous les utilisateurs » existe et que la propriété [**ALLUSERS**](allusers.md) est définie, cette propriété est définie sur le dossier dans le profil « tous les utilisateurs ».
