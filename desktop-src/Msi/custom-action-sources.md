# <a name="custom-action-sources"></a>Sources d’action personnalisées

Une source d’action personnalisée peut exister sous différentes formes. Le tableau suivant répertorie les types d’actions personnalisées qui peuvent avoir chaque forme de source. Pour plus d’informations, consultez la rubrique relative au type d’action personnalisé de base.



| Source de l’action personnalisée                                           | Types d’actions personnalisées associés                                                                                                                                                                                 |
|----------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Action personnalisée stockée dans la table binaire.                      | [Type](custom-action-type-1.md)d’action personnalisée 1, [type d’action personnalisée 2](custom-action-type-2.md), [type d’action personnalisée 5](custom-action-type-5.md), [type d’action personnalisée 6](custom-action-type-6.md)         |
| Action personnalisée copiée lors de l’installation.                      | [Type d’action personnalisée 17](custom-action-type-17.md), [type d’action personnalisée 18](custom-action-type-18.md), [type d’action personnalisée 21](custom-action-type-21.md), type d’action [personnalisée 22](custom-action-type-22.md) |
| Action personnalisée faisant référence à un répertoire.                         | [Type d’action personnalisé 34](custom-action-type-34.md)                                                                                                                                                             |
| Action personnalisée faisant référence à une propriété.                          | [Type](custom-action-type-50.md)d’action personnalisée 50, [type d’action personnalisée 53](custom-action-type-53.md), [type d’action personnalisée 54](custom-action-type-54.md)                                                     |
| Action personnalisée stockée en tant que code de script littéral dans la table de base de données. | [Type d’action personnalisé 37](custom-action-type-37.md), [type d’action personnalisée 38](custom-action-type-38.md)                                                                                                         |
| Action personnalisée affichant un message d’erreur.                     | [Type d’action personnalisée 19](custom-action-type-19.md)                                                                                                                                                             |



 

Consultez la [liste récapitulative de tous les types d’actions personnalisées](summary-list-of-all-custom-action-types.md) pour obtenir un résumé de tous les types d’actions personnalisées et de la façon dont les sources sont spécifiées dans la [table CustomAction](customaction-table.md).
