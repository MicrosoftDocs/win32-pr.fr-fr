# <a name="clientusersid-property"></a>Propriété client. UserSID

SID de l’utilisateur du composant. Cette propriété appelle la fonction [**MsiEnumClientsEx**](/windows/desktop/api/Msi/nf-msi-msienumclientsexa) .

**[Windows Installer 4,5 ou version antérieure](not-supported-in-windows-installer-4-5.md):** Non pris en charge. cette propriété est disponible à partir de Windows Installer 5,0.
