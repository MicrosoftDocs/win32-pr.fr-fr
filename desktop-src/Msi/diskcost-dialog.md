# <a name="diskcost-dialog"></a>Boîte de dialogue DiskCost

Une boîte de dialogue DiskCost s’affiche généralement lorsque la propriété [**OutOfDiskSpace**](outofdiskspace.md) est définie. Cela indique qu’il n’y a pas assez d’espace disque pour l’installation sélectionnée.

Cette boîte de dialogue peut également être activée à partir de la [boîte de dialogue de sélection](selection-dialog.md) pour fournir à l’utilisateur une vue d’ensemble de l’espace disque nécessaire à l’installation.

Une boîte de dialogue DiskCost est une boîte de dialogue modale. Il contient généralement un [bouton](pushbutton-control.md) de commande qui renvoie l’utilisateur à la boîte de dialogue précédente et un [contrôle VolumeCostList](volumecostlist-control.md).
