# <a name="rtlro-dialog-style-bit"></a>RTLRO bit de style de boîte de dialogue

Si ce bit de style est défini, le texte de la boîte de dialogue s’affiche dans l’ordre de lecture de droite à gauche.

## <a name="value"></a>Valeur



| Decimal | Valeur hexadécimale | Constante                       |
|---------|-------------|--------------------------------|
| 128     | 0x00000080  | **msidbDialogAttributesRTLRO** |
