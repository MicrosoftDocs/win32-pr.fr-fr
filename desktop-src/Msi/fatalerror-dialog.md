# <a name="fatalerror-dialog"></a>Boîte de dialogue FatalError

Cette boîte de dialogue modale s’affiche à la fin d’une installation si l’installation est interrompue en raison d’une erreur irrécupérable. Cette boîte de dialogue contient généralement uniquement du texte et un bouton **OK** . Le nom de cette boîte de dialogue doit apparaître dans la [table AdminUISequence](adminuisequence-table.md) et la [table InstallUISequence](installuisequence-table.md) avec-3 comme numéro de séquence.
