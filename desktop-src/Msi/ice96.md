# <a name="ice96"></a>ICE96

ICE96 vérifie que l' [action PublishFeatures](publishfeatures-action.md) et l' [action PublishProduct](publishproduct-action.md) sont entrées dans la [table AdvtExecuteSequence](advtexecutesequence-table.md). Un package ne peut pas être publié sans ces actions.

## <a name="result"></a>Résultat

ICE96 publie les avertissements suivants.



| AVERTISSEMENT ICE96                                                            | Description                                                                                                  |
|--------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------|
| L’action PublishFeatures est requise dans la table AdvtExecuteSequence. | Un package ne peut pas être publié sans l’action PublishFeatures incluse dans la table AdvtExecuteSequence. |
| L’action PublishProduct est requise dans la table AdvtExecuteSequence.  | Un package ne peut pas être publié sans l’action PublishProduct incluse dans la table AdvtExecuteSequence.  |
