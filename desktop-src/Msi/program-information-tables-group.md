# <a name="program-information-tables-group"></a>Groupe de tables d’informations sur les programmes

Le groupe tables d’informations sur les programmes contient des informations importantes qui ont été utilisées tout au long de l’installation.

-   La [table de propriétés](property-table.md) fournit un moyen de spécifier toutes les [Propriétés](properties.md) d’une installation.
-   La [table binaire](binary-table.md) contient les données binaires pour les éléments tels que les bitmaps, les animations et les icônes. La table binaire est également utilisée pour stocker des données pour les actions personnalisées. Ce tableau peut contenir une image bitmap pour un panneau, l’icône de votre programme ou la forme exécutable d’une action personnalisée.
-   La [table d’erreurs](error-table.md) permet de rechercher les modèles de mise en forme des messages d’erreur lors du traitement des erreurs avec un code d’erreur défini. C’est le cas habituellement où aucun modèle de mise en forme n’est défini. Le programme d’installation a son propre mécanisme de traitement des erreurs. Les erreurs sont transmises en tant qu’enregistrements.
-   [Tableau de raccourcis](shortcut-table.md) la table de raccourcis contient les informations dont l’application a besoin pour créer des raccourcis sur l’ordinateur de l’utilisateur.
-   La [table ReserveCost](reservecost-table.md) contient l’espace disque nécessaire à chaque composant pour fonctionner correctement.
