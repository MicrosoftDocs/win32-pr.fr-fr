# <a name="using-windows-installer-with-wmi"></a>Utilisation de Windows Installer avec WMI

Le fournisseur Windows Installer permet à Windows Management Instrumentation (WMI) d’accéder aux informations collectées à partir d’applications Windows Installer.

Le [fournisseur Windows Installer](/previous-versions/windows/desktop/msiprov/windows-installer-provider) est un composant Windows facultatif. L’installation facultative du fournisseur Windows Installer garantit la compatibilité descendante, mais n’indique pas la disponibilité future du fournisseur. Pour plus d’informations sur la disponibilité du fournisseur de Windows Installer, consultez [disponibilité du système d’exploitation des composants WMI](../wmisdk/operating-system-availability-of-wmi-components.md) dans la documentation de [Windows Management Instrumentation (WMI)](../wmisdk/wmi-start-page.md) .

Pour obtenir la description actuelle de ce fournisseur WMI, consultez [Windows Installer fournisseur](/previous-versions/windows/desktop/msiprov/windows-installer-provider) dans la documentation principale de [Windows Management Instrumentation (WMI)](../wmisdk/wmi-start-page.md) . Pour obtenir la liste des classes WMI prises en charge par Windows Installer, consultez [classes d’applications installées](/previous-versions//aa390887(v=vs.85)).
