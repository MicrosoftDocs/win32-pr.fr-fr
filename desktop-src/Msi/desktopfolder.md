# <a name="desktopfolder-property"></a>Propriété DesktopFolder

Le programme d’installation définit la propriété **DesktopFolder** sur le chemin d’accès complet du dossier Desktop de l’utilisateur actuel. Si un profil « tous les utilisateurs » existe et que la propriété [**ALLUSERS**](allusers.md) est définie, cette propriété est définie sur le dossier dans le profil « tous les utilisateurs ».
