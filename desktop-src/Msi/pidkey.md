# <a name="pidkey-property"></a>PIDKEY, propriété

La propriété **PIDKEY** contient la partie de l’ID de produit entrée par l’utilisateur. L' [Action ValidateProductID](validateproductid-action.md) valide l’ID de produit entré en comparant la propriété **PIDKEY** à la propriété [**PIDTemplate**](pidtemplate.md) .

La propriété **PIDKEY** peut être définie sur la ligne de commande ou via l’interface utilisateur. Cette propriété contient uniquement la partie de l’ID de produit qui est entrée par l’utilisateur.
