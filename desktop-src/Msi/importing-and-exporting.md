# <a name="importing-and-exporting"></a>Importation et exportation

Vous pouvez utiliser le programme d’installation pour importer une archive de texte dans une base de données active, ainsi que pour exporter un fichier de base de données vers une archive de texte. Cela peut être utile pour les systèmes de contrôle de code source basés sur du texte.

Pour importer une archive de texte dans une base de données active, appelez la fonction [**MsiDatabaseImport**](/windows/desktop/api/Msiquery/nf-msiquery-msidatabaseimporta) .

Pour exporter un fichier de base de données vers une archive de texte, appelez la fonction [**MsiDatabaseExport**](/windows/desktop/api/Msiquery/nf-msiquery-msidatabaseexporta) .
