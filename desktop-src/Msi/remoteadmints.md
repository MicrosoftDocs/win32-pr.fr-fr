# <a name="remoteadmints-property"></a>Propriété RemoteAdminTS

le programme d’installation définit la propriété **RemoteAdminTS** lorsque le système est configuré pour l’Administration à distance à l’aide des Services Terminal Windows. Notez que dans ce cas, le programme d’installation ne définit pas la propriété [**TerminalServer**](terminalserver.md) .

## <a name="default-value"></a>Valeur par défaut

Aucun. La propriété n’est pas définie lorsqu’elle n’est pas exécutée sur un serveur d’administration à distance avec les services Terminal Server.
