# <a name="programfiles64folder-property"></a>Propriété ProgramFiles64Folder

Le programme d’installation définit la propriété **ProgramFiles64Folder** sur le chemin d’accès complet du dossier de fichiers programmes 64 bits prédéfini. La propriété [**ProgramFilesFolder**](programfilesfolder.md) existante est définie sur le dossier 32 bits correspondant.

## <a name="remarks"></a>Remarques

Le programme d’installation définit cette propriété. par exemple, sur l’Windows 64 bits, la valeur peut être C : \\ Program Files. Cette propriété n’est pas utilisée sur les Windowss 32 bits.
