# <a name="summaryinfopropertycount-property"></a>SummaryInfo. PropertyCount, propriété

La propriété **PropertyCount** de l’objet [**SummaryInfo**](summaryinfo-object.md) est une propriété en lecture seule qui indique le nombre actuel de valeurs de propriété dans l’objet d’informations de résumé. Il prend en compte les propriétés qui ont été ajoutées, supprimées ou remplacées.

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```JScript
propVal = SummaryInfo.PropertyCount
```
