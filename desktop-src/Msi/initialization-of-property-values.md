# <a name="initialization-of-property-values"></a>Initialisation des valeurs de propriété

Il est recommandé que toutes les propriétés utilisées par l’installation soient entrées dans la [table de propriétés](property-table.md) avec une valeur initiale. Le programme d’installation définit les propriétés sur ces valeurs au lancement de l’installation. Les propriétés pour lesquelles un vide est une valeur acceptable et les propriétés intégrées dans le programme d’installation n’ont pas besoin d’être initialisées.
