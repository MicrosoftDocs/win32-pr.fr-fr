# <a name="defaultdir"></a>DefaultDir

Le type de données DefaultDir est une chaîne de texte contenant un [nom de fichier](filename.md) valide ou un [identificateur](identifier.md)valide. Utilisé uniquement dans la table de [répertoires](directory-table.md). Il doit s’agir d’un identificateur si le répertoire est un répertoire racine. Si le répertoire est une non-racine, cette valeur doit être un nom de fichier ou une paire nom de fichier : nom de fichier. Notez que « . » est autorisé comme nom de fichier et a une signification particulière dans la table de répertoires.
