# <a name="installation-on-demand"></a><span data-ttu-id="0cb64-103">Installation à la demande</span><span class="sxs-lookup"><span data-stu-id="0cb64-103">Installation-On-Demand</span></span>

<span data-ttu-id="0cb64-104">Avec la technologie d’installation traditionnelle, il est nécessaire de quitter une application et de réexécuter le programme d’installation pour effectuer une tâche d’installation.</span><span class="sxs-lookup"><span data-stu-id="0cb64-104">With traditional installation technology, it is necessary to exit an application and rerun setup to perform an installation task.</span></span> <span data-ttu-id="0cb64-105">Cela se produit généralement lorsqu’un utilisateur souhaitait une fonctionnalité ou un produit qui n’a pas été choisi lors de la première exécution du programme d’installation.</span><span class="sxs-lookup"><span data-stu-id="0cb64-105">This commonly occurred when a user wanted a feature or product not chosen during the first run of setup.</span></span> <span data-ttu-id="0cb64-106">Cela rend souvent le processus de configuration du produit inefficace, car il a demandé à l’utilisateur d’anticiper les fonctionnalités requises avant qu’il n’ait jamais utilisé le produit.</span><span class="sxs-lookup"><span data-stu-id="0cb64-106">This often made the process of product configuration inefficient because it required the user to anticipate the functionality required before they ever used the product.</span></span>

<span data-ttu-id="0cb64-107">L’installation à la demande permet d’offrir des fonctionnalités aux utilisateurs et aux applications en l’absence des fichiers eux-mêmes.</span><span class="sxs-lookup"><span data-stu-id="0cb64-107">Installation-on-demand makes it possible to offer functionality to users and applications in the absence of the files themselves.</span></span> <span data-ttu-id="0cb64-108">Cette notion porte le nom de publication.</span><span class="sxs-lookup"><span data-stu-id="0cb64-108">This notion is known as advertisement.</span></span> <span data-ttu-id="0cb64-109">Le Windows Installer a la possibilité de publier des fonctionnalités et de rendre possible l’installation à la demande de fonctionnalités d’application ou de produits entiers.</span><span class="sxs-lookup"><span data-stu-id="0cb64-109">The Windows Installer has the capability of advertising functionality and to make installation-on-demand of application features or entire products possible.</span></span> <span data-ttu-id="0cb64-110">Lorsqu’un utilisateur ou une application active une fonctionnalité ou un produit publié, le programme d’installation se poursuit avec l’installation des composants nécessaires.</span><span class="sxs-lookup"><span data-stu-id="0cb64-110">When a user or application activates an advertised feature or product, the installer proceeds with installation of the needed components.</span></span> <span data-ttu-id="0cb64-111">Cela réduit le processus de configuration car des fonctionnalités supplémentaires sont accessibles sans avoir à quitter et à réexécuter une autre procédure d’installation.</span><span class="sxs-lookup"><span data-stu-id="0cb64-111">This shortens the configuration process because additional functionality can be accessed without having to exit and rerun another setup procedure.</span></span>

<span data-ttu-id="0cb64-112">Lorsqu’un produit utilise le programme d’installation, un utilisateur peut choisir, au moment de l’installation, les fonctionnalités ou les applications à installer et à publier.</span><span class="sxs-lookup"><span data-stu-id="0cb64-112">When a product uses the installer, a user can choose at setup time which features or applications to install and which to advertise.</span></span> <span data-ttu-id="0cb64-113">Puis, si lors de l’exécution d’une application, l’utilisateur demande une fonctionnalité publiée qui n’a pas encore été installée, l’application appelle le programme d’installation pour mettre en œuvre une installation de niveau de fonctionnalité juste-à-temps des fichiers nécessaires.</span><span class="sxs-lookup"><span data-stu-id="0cb64-113">Then if while running an application the user requests an advertised feature that has not yet been installed, the application calls the installer to enact a just-in-time feature level installation of the necessary files.</span></span> <span data-ttu-id="0cb64-114">Si l’utilisateur active un produit publié qui n’a pas encore été installé, le système d’exploitation appelle le programme d’installation pour mettre en œuvre une installation juste-à-temps du produit.</span><span class="sxs-lookup"><span data-stu-id="0cb64-114">If the user activates an advertised product that has not yet been installed, the operating system calls the installer to enact a just-in-time product level installation.</span></span>

<span data-ttu-id="0cb64-115">La [publication](advertisement.md) et l’installation-à la demande peuvent également faciliter la gestion du système en permettant aux administrateurs de désigner les applications comme requises ou facultatives pour différents groupes d’utilisateurs.</span><span class="sxs-lookup"><span data-stu-id="0cb64-115">[Advertisement](advertisement.md) and installation-on-demand can also facilitate system management by enabling administrators to designate applications as required or optional for different groups of users.</span></span> <span data-ttu-id="0cb64-116">Il existe deux types de publicité appelés « affectation » et « publication ».</span><span class="sxs-lookup"><span data-stu-id="0cb64-116">There are two types of advertising known as "assigning" and "publishing."</span></span> <span data-ttu-id="0cb64-117">Si un administrateur attribue une application à un groupe, ces utilisateurs peuvent installer l’application à la demande.</span><span class="sxs-lookup"><span data-stu-id="0cb64-117">If an administrator assigns an application to a group, then these users can install the application on-demand.</span></span> <span data-ttu-id="0cb64-118">Toutefois, si l’administrateur publie l’application dans le groupe, aucun point d’entrée n’apparaît à ces utilisateurs et l’installation à la demande n’est activée que si une autre application active l’application publiée.</span><span class="sxs-lookup"><span data-stu-id="0cb64-118">If, however, the administrator publishes the application to the group, no entry points appear to these users and installation-on-demand is only activated if another application activates the published application.</span></span>
