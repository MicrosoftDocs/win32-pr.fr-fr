# <a name="suggested-advtuisequence"></a>AdvtUISequence suggéré

Ne créez pas la table AdvtUISequence. Le programme d’installation n’utilise pas cette table. La table AdvtUISequence ne doit pas exister dans la base de données d’installation ou elle doit être laissée vide. Le nom de l’application à publier doit être transmis au programme d’installation lors de l’initialisation.
