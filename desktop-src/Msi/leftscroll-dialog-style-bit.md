# <a name="leftscroll-dialog-style-bit"></a>LeftScroll bit de style de boîte de dialogue

Si ce bit de style est défini, la barre de défilement se trouve sur le côté gauche de la boîte de dialogue.

## <a name="value"></a>Valeur



| Decimal | Valeur hexadécimale | Constante                            |
|---------|-------------|-------------------------------------|
| 512     | 0x00000200  | **msidbDialogAttributesLeftScroll** |
