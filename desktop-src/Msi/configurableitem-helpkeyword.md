# <a name="configurableitemhelpkeyword-property"></a>ConfigurableItem. HelpKeyword, propriété

La propriété **HelpKeyword** retourne la valeur de la colonne HelpKeyword de la [table ModuleConfiguration](moduleconfiguration-table.md).

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```JScript
propVal = ConfigurableItem.HelpKeyword
```



## <a name="property-value"></a>Valeur de la propriété

## <a name="c"></a>C++

Consultez [**obtenir \_**](/windows/desktop/api/Mergemod/nf-mergemod-imsmconfigurableitem-get_helpkeyword) la fonction HelpKeyword.
