# <a name="pointer-device-input-stack"></a>Pile d’entrée du périphérique pointeur

## <a name="purpose"></a>Objectif

Les rubriques de cette section fournissent une vue d’ensemble de la pile d’entrée des appareils du pointeur Windows. Les API de [pile d’entrée de périphérique du pointeur](/windows/win32/api/_input_pointerdevice/) fournissent des informations sur les périphériques d’entrée connectés au système.

## <a name="in-this-section"></a>Contenu de cette section

| Rubrique | Description |
|---|---|
| [Référence de pile d’entrée de périphérique pointeur](unified-input-stack-reference.md)<br/> | Les rubriques de cette section fournissent les spécifications de référence pour la pile d’entrée du périphérique pointeur dans Windows 8.<br/> |

## <a name="developer-audience"></a>Développeurs concernés

Les StackAPIs d’entrée d’appareil de pointeurs sont conçus pour les développeurs qui requièrent des informations sur l’appareil de pointeur utilisé pour interagir avec leur application.
