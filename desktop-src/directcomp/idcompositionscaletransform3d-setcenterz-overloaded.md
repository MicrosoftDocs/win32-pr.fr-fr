# <a name="idcompositionscaletransform3dsetcenterz-methods"></a>IDCompositionScaleTransform3D :: SetCenterZ, méthodes

Modifie ou anime la valeur de la propriété CenterZ d’une transformation d’échelle 3D. La propriété CenterZ spécifie la coordonnée z du point sur lequel la mise à l’échelle est effectuée.

### <a name="overload-list"></a>Liste de surcharge



| Méthode                                                                                                          | Description                                            |
|:----------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------|
| [**SetCenterZ (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionscaletransform3d-setcenterz(float))                                     | Modifie la valeur de la propriété CenterZ.<br/>  |
| [**SetCenterZ (IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionscaletransform3d-setcenterz(idcompositionanimation)) | Anime la valeur de la propriété CenterZ.<br/> |
