# <a name="idcompositionskewtransformsetangley-methods"></a>IDCompositionSkewTransform :: semêley, méthodes

Modifie ou anime la valeur de la propriété AngleY d’une transformation d’inclinaison 2D. La propriété AngleY spécifie l’angle de rotation, en degrés. La valeur par défaut est zéro.

| Méthode                                                                                                     | Description                                           |
|:-----------------------------------------------------------------------------------------------------------|:------------------------------------------------------|
| [**Semêler (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionskewtransform-setangley(float))                                     | Modifie la valeur de la propriété AngleY.<br/>  |
| [**Semêler (IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionskewtransform-setangley(idcompositionanimation)) | Anime la valeur de la propriété AngleY.<br/> |
