# <a name="idcompositiontranslatetransform3dsetoffsetx-methods"></a>IDCompositionTranslateTransform3D :: SetOffsetX, méthodes

Modifie ou anime la valeur de la propriété OffsetX d’un effet de transformation de translation 3D. La propriété OffsetX spécifie la translation le long de l’axe x.

### <a name="overload-list"></a>Liste de surcharge



| Méthode                                                                                                              | Description                                            |
|:--------------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------|
| [**SetOffsetX (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositiontranslatetransform3d-setoffsetx(float))                                     | Modifie la valeur de la propriété OffsetX.<br/>  |
| [**SetOffsetX (IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositiontranslatetransform3d-setoffsetx(idcompositionanimation)) | Anime la valeur de la propriété OffsetX.<br/> |
