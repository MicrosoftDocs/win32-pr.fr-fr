# <a name="idcompositionrotatetransform3dsetaxisy-methods"></a>IDCompositionRotateTransform3D :: SetAxisY, méthodes

Modifie ou anime la valeur de la propriété d’axe d’une transformation de rotation. La propriété AxisY spécifie la coordonnée y du vecteur d’axe de rotation. La valeur par défaut est zéro.

### <a name="overload-list"></a>Liste de surcharge



| Méthode                                                                                                       | Description                                          |
|:-------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------|
| [**SetAxisY (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform3d-setaxisy(float))                                     | Modifie la valeur de la propriété de l’axe.<br/>  |
| [**SetAxisY (IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform3d-setaxisy(idcompositionanimation)) | Anime la valeur de la propriété de l’axe.<br/> |
