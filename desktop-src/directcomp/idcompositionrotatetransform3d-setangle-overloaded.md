# <a name="idcompositionrotatetransform3dsetangle-methods"></a>IDCompositionRotateTransform3D :: seenchevêtrement, méthodes

Modifie ou anime la valeur de la propriété angle d’une transformation de rotation 3D. La propriété angle spécifie l’angle de rotation, en degrés. La valeur par défaut est zéro.
