# <a name="idcompositionrotatetransform3dsetcenterz-methods"></a>IDCompositionRotateTransform3D :: SetCenterZ, méthodes

Modifie ou anime la valeur de la propriété CenterZ d’une transformation de rotation 3D. La propriété CenterZ spécifie la coordonnée z du point sur lequel la rotation est effectuée. La valeur par défaut est zéro.

### <a name="overload-list"></a>Liste de surcharge



| Méthode                                                                                                           | Description                                            |
|:-----------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------|
| [**SetCenterZ (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform3d-setcenterz(float))                                     | Modifie la valeur de la propriété CenterZ.<br/>  |
| [**SetCenterZ (IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform3d-setcenterz(idcompositionanimation)) | Anime la valeur de la propriété CenterZ.<br/> |
