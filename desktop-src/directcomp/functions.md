# <a name="directcomposition-functions"></a>DirectComposition, fonctions

Cette section décrit les fonctions fournies par l’API Microsoft DirectComposition.

## <a name="in-this-section"></a>Contenu de cette section



| Rubrique                                                                                       | Description                                                                                                                                          |
|---------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**DCompositionAttachMouseDragToHwnd**](/windows/desktop/api/dcomp/nf-dcomp-dcompositionattachmousedragtohwnd)<br/>   | Crée une interaction/InputSink pour router le bouton de la souris vers le haut et tous les événements de déplacement et de déplacement ultérieurs vers le HWND donné.<br/>                      |
| [**DCompositionAttachMouseWheelToHwnd**](/windows/desktop/api/dcomp/nf-dcomp-dcompositionattachmousewheeltohwnd)<br/> | Crée une interaction/InputSink pour acheminer les messages de roulette de la souris vers le HWND donné. <br/>                                                        |
| [**DCompositionCreateDevice**](/windows/desktop/api/Dcomp/nf-dcomp-dcompositioncreatedevice)<br/>                     | Crée un nouvel objet d’appareil qui peut être utilisé pour créer d’autres objets DirectComposition.<br/>                                                   |
| [**DCompositionCreateDevice2**](/windows/desktop/api/Dcomp/nf-dcomp-dcompositioncreatedevice2)<br/>                   | Crée un nouvel objet d’appareil qui peut être utilisé pour créer d’autres objets DirectComposition.<br/>                                                   |
| [**DCompositionCreateDevice3**](/windows/desktop/api/dcomp/nf-dcomp-dcompositioncreatedevice3)<br/>                   | Crée un nouvel objet d’appareil DirectComposition, qui peut être utilisé pour créer d’autres objets DirectComposition.<br/>                               |
| [**DCompositionCreateSurfaceHandle**](/windows/desktop/api/Dcomp/nf-dcomp-dcompositioncreatesurfacehandle)<br/>       | Crée un objet de surface de composition qui peut être lié à une chaîne de permutation Microsoft DirectX ou à un tampon d’échange et associé à un visuel.<br/> |
| [**DCompositionGetFrameStatistics**](/previous-versions/windows/desktop/legacy/mt589902(v=vs.85))<br/>         | Récupère des informations sur les statistiques de la composition.<br/>                                                                                             |
