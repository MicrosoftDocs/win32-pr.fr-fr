# <a name="idcompositionrotatetransformsetcentery-methods"></a>IDCompositionRotateTransform :: SetCenterY, méthodes

Modifie ou anime la valeur de la propriété CenterY d’une transformation de rotation 2D. La propriété CenterY spécifie la coordonnée y du point sur lequel la rotation est effectuée.

### <a name="overload-list"></a>Liste de surcharge



| Méthode                                                                                                         | Description                                            |
|:---------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------|
| [**SetCenterY (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform-setcentery(float))                                     | Modifie la valeur de la propriété CenterY.<br/>  |
| [**SetCenterY (IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform-setcentery(idcompositionanimation)) | Anime la valeur de la propriété CenterY.<br/> |
