# <a name="directcomposition-interfaces"></a>Interfaces DirectComposition

Cette section décrit les interfaces fournies par l’API Microsoft DirectComposition.

## <a name="in-this-section"></a>Contenu de cette section



| Rubrique                                                                                               | Description                                                                                                                                                                                                                                                                                                                                                                                               |
|-----------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IDCompositionAffineTransform2DEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositionaffinetransform2deffect)<br/>     | L’effet composite arithmétique est utilisé pour combiner 2 images à l’aide d’une somme pondérée de pixels à partir des images d’entrée. <br/>                                                                                                                                                                                                                                                                             |
| [**IDCompositionAnimation**](/windows/desktop/api/DcompAnimation/nn-dcompanimation-idcompositionanimation)<br/>                                 | Représente une fonction permettant d’animer une ou plusieurs propriétés d’un ou plusieurs objets DirectComposition.<br/>                                                                                                                                                                                                                                                                                           |
| [**IDCompositionArithmeticCompositeEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositionarithmeticcompositeeffect)<br/> | L’effet composite arithmétique est utilisé pour combiner 2 images à l’aide d’une somme pondérée de pixels à partir des images d’entrée.<br/>                                                                                                                                                                                                                                                                              |
| [**IDCompositionBlendEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositionblendeffect)<br/>                             | L’effet de fusion est utilisé pour combiner 2 images. <br/>                                                                                                                                                                                                                                                                                                                                                 |
| [**IDCompositionBrightnessEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositionbrightnesseffect)<br/>                   | L’effet luminosité contrôle la luminosité de l’image.<br/>                                                                                                                                                                                                                                                                                                                                    |
| [**IDCompositionColorMatrixEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositioncolormatrixeffect)<br/>                 | L’effet matrice de couleurs modifie les valeurs RVBA d’une image bitmap.<br/>                                                                                                                                                                                                                                                                                                                                    |
| [**IDCompositionCompositeEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositioncompositeeffect)<br/>                     | L’effet composite est utilisé pour combiner deux images ou plus. Cet effet a 13 modes composites différents. L’effet composite accepte au moins 2 entrées. Lorsque vous spécifiez 2 images, la destination est la première entrée (index 0) et la source est la deuxième entrée (index 1). Si vous spécifiez plus de 2 entrées, les images sont composées à partir de la première entrée, puis de la seconde, et ainsi de suite. <br/> |
| [**IDCompositionClip**](/windows/win32/api/dcomp/nn-dcomp-idcompositionclip)<br/>                                           | Représente un objet clip qui est utilisé pour restreindre le rendu d’un sous-arbre visuel à une zone rectangulaire. <br/>                                                                                                                                                                                                                                                                                    |
| [**IDCompositionDesktopDevice**](/windows/win32/api/dcomp/nn-dcomp-idcompositiondesktopdevice)<br/>                         | Une application doit utiliser l’interface IDCompositionDesktopDevice pour pouvoir utiliser DirectComposition dans une application de bureau Win32. Cette interface permet à l’application de connecter une arborescence d’éléments visuels à une fenêtre et d’héberger des fenêtres enfants superposées pour la composition<br/>                                                                                                                                |
| [**IDCompositionDevice**](/windows/win32/api/dcomp/nn-dcomp-idcompositiondevice)<br/>                                       | Sert de fabrique pour tous les autres objets DirectComposition et fournit des méthodes pour contrôler la composition transactionnelle.<br/>                                                                                                                                                                                                                                                                         |
| [**IDCompositionDevice2**](/windows/win32/api/dcomp/nn-dcomp-idcompositiondevice2)<br/>                                     | Sert de fabrique pour tous les autres objets DirectComposition et fournit des méthodes pour contrôler la composition transactionnelle.<br/>                                                                                                                                                                                                                                                                         |
| [**IDCompositionDevice3**](/windows/win32/api/dcomp/nn-dcomp-idcompositiondevice3)<br/>                                     | Sert de fabrique pour tous les autres objets DirectComposition et fournit des méthodes pour contrôler la composition transactionnelle. <br/>                                                                                                                                                                                                                                                                        |
| [**IDCompositionDeviceDebug**](/windows/win32/api/dcomp/nn-dcomp-idcompositiondevicedebug)<br/>                             | Fournit l’accès aux fonctionnalités de rendu qui facilitent le débogage des applications et le réglage des performances. Cette interface peut être interrogée à partir de l’interface de l’appareil DirectComposition.<br/>                                                                                                                                                                                                                  |
| [**IDCompositionEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositioneffect)<br/>                                       | Représente un effet bitmap qui modifie la pixellisation d’une sous-arborescence d’un visuel. <br/>                                                                                                                                                                                                                                                                                                             |
| [**IDCompositionEffectGroup**](/windows/win32/api/dcomp/nn-dcomp-idcompositioneffectgroup)<br/>                             | Représente un groupe d’effets bitmap appliqués ensemble pour modifier la pixellisation du sous-arbre d’un visuel. <br/>                                                                                                                                                                                                                                                                             |
| [**IDCompositionFilterEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositionfiltereffect)<br/>                           | Représente un effet de filtre.<br/>                                                                                                                                                                                                                                                                                                                                                                    |
| [**IDCompositionFloodEffect**](/previous-versions/windows/desktop/legacy/dn919732(v=vs.85))<br/>                             | L’effet de flux est utilisé pour générer une image bitmap basée sur la couleur spécifiée et la valeur alpha. Vous pouvez utiliser cet effet lorsque vous souhaitez qu’une couleur spécifique soit une entrée pour un effet, comme une couleur d’arrière-plan. <br/>                                                                                                                                                                                        |
| [**IDCompositionGaussianBlurEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositiongaussianblureffect)<br/>               |                                                                                                                                                                                                                                                                                                                                                                                                           |
| [**IDCompositionHueRotationEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositionhuerotationeffect)<br/>                 | L’effet de rotation de la teinte modifie la teinte d’une image en appliquant une matrice de couleurs en fonction de l’angle de rotation. <br/>                                                                                                                                                                                                                                                                                      |
| [**IDCompositionLinearTransferEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositionlineartransfereffect)<br/>           | L’effet de transfert linéaire est utilisé pour mapper les intensités de couleur d’une image à l’aide d’une fonction linéaire créée à partir d’une liste de valeurs que vous fournissez pour chaque canal. <br/>                                                                                                                                                                                                                               |
| [**IDCompositionMatrixTransform**](/windows/win32/api/dcomp/nn-dcomp-idcompositionmatrixtransform)<br/>                     | Représente une transformation 2D affine arbitraire définie par une matrice 3 par 2.<br/>                                                                                                                                                                                                                                                                                                                   |
| [**IDCompositionMatrixTransform3D**](/windows/win32/api/dcomp/nn-dcomp-idcompositionmatrixtransform3d)<br/>                 | Représente une transformation 3D arbitraire définie par une matrice 4 par 4.<br/>                                                                                                                                                                                                                                                                                                                          |
| [**IDCompositionRectangleClip**](/windows/win32/api/dcomp/nn-dcomp-idcompositionrectangleclip)<br/>                         | Représente un objet clip qui restreint le rendu d’un sous-arbre visuel à la zone rectangulaire spécifiée. Si vous le souhaitez, l’objet clip peut avoir des angles arrondis spécifiés.<br/>                                                                                                                                                                                                                 |
| [**IDCompositionRotateTransform**](/windows/win32/api/dcomp/nn-dcomp-idcompositionrotatetransform)<br/>                     | Représente une transformation 2D qui affecte la rotation d’un visuel autour de l’axe z. Le système de coordonnées pivote autour du point central spécifié. <br/>                                                                                                                                                                                                                                   |
| [**IDCompositionSaturationEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositionsaturationeffect)<br/>                   | Cet effet est utilisé pour modifier la saturation d’une image. L’effet de saturation est une spécialisation de l’effet de matrice de couleurs. <br/>                                                                                                                                                                                                                                                                |
| [**IDCompositionRotateTransform3D**](/windows/win32/api/dcomp/nn-dcomp-idcompositionrotatetransform3d)<br/>                 | Représente une transformation 3D qui affecte la rotation d’un visuel le long d’un axe arbitraire dans l’espace 3D. Le système de coordonnées pivote autour du point central spécifié. <br/>                                                                                                                                                                                                                 |
| [**IDCompositionScaleTransform**](/windows/win32/api/dcomp/nn-dcomp-idcompositionscaletransform)<br/>                       | Représente une transformation 2D qui affecte l’échelle d’un visuel le long de l’axe x et de l’axe y. Le système de coordonnées est mis à l’échelle à partir du point central spécifié. <br/>                                                                                                                                                                                                                               |
| [**IDCompositionScaleTransform3D**](/windows/win32/api/dcomp/nn-dcomp-idcompositionscaletransform3d)<br/>                   | Représente un effet de transformation 3D qui affecte l’échelle d’un visuel le long de l’axe x, de l’axe y et de l’axe z. Le système de coordonnées est mis à l’échelle à partir du point central spécifié. <br/>                                                                                                                                                                                                               |
| [**IDCompositionSkewTransform**](/windows/win32/api/dcomp/nn-dcomp-idcompositionskewtransform)<br/>                         | Représente une transformation 2D qui affecte l’inclinaison d’un visuel le long de l’axe x et de l’axe y. Le système de coordonnées est incliné autour du point central spécifié. <br/>                                                                                                                                                                                                                              |
| [**IDCompositionShadowEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositionshadoweffect)<br/>                           | L’effet d’ombre est utilisé pour générer une ombre à partir du canal alpha d’une image. L’ombre est plus opaque pour les valeurs alpha supérieures et plus transparente pour les valeurs alpha inférieures. Vous pouvez définir la quantité de flou et la couleur de l’ombre. <br/>                                                                                                                                                 |
| [**IDCompositionSurface**](/windows/win32/api/dcomp/nn-dcomp-idcompositionsurface)<br/>                                     | Représente une bitmap physique qui peut être associée à un visuel pour la composition dans une arborescence d’éléments visuels. Cette interface peut également être utilisée pour mettre à jour le contenu de l’image bitmap. <br/>                                                                                                                                                                                                                            |
| [**IDCompositionTableTransferEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositiontabletransfereffect)<br/>             | L’effet de transfert de table est utilisé pour mapper les intensités de couleur d’une image à l’aide d’une fonction de transfert créée à partir de l’interpolation d’une liste de valeurs que vous fournissez.<br/>                                                                                                                                                                                                                                  |
| [**IDCompositionSurfaceFactory**](/windows/win32/api/dcomp/nn-dcomp-idcompositionsurfacefactory)<br/>                       | Crée des objets surface et des surfaces virtuelles associés à un périphérique de rendu fourni par l’application.<br/>                                                                                                                                                                                                                                                                                          |
| [**IDCompositionTarget**](/windows/win32/api/dcomp/nn-dcomp-idcompositiontarget)<br/>                                       | Représente une liaison entre une arborescence d’éléments visuels DirectComposition et une destination au-dessus de laquelle l’arborescence d’éléments visuels doit être composée. <br/>                                                                                                                                                                                                                                                            |
| [**IDCompositionTransform**](/windows/win32/api/dcomp/nn-dcomp-idcompositiontransform)<br/>                                 | Représente une transformation 2D qui peut être utilisée pour modifier l’espace de coordonnées d’une sous-arborescence visuelle. <br/>                                                                                                                                                                                                                                                                                           |
| [**IDCompositionTransform3D**](/windows/win32/api/dcomp/nn-dcomp-idcompositiontransform3d)<br/>                             | Représente un effet de transformation 3D qui peut être utilisé pour modifier la pixellisation d’une sous-arborescence visuelle. <br/>                                                                                                                                                                                                                                                                                       |
| [**IDCompositionTranslateTransform**](/windows/win32/api/dcomp/nn-dcomp-idcompositiontranslatetransform)<br/>               | Représente une transformation 2D qui affecte uniquement le décalage d’un visuel le long de l’axe x et de l’axe y.<br/>                                                                                                                                                                                                                                                                                           |
| [**IDCompositionTurbulenceEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositionturbulenceeffect)<br/>                   | L’effet turbulence est utilisé pour générer une image bitmap basée sur la fonction de bruit perl. L’effet turbulence n’a aucune image d’entrée. <br/>                                                                                                                                                                                                                                                              |
| [**IDCompositionTranslateTransform3D**](/windows/win32/api/dcomp/nn-dcomp-idcompositiontranslatetransform3d)<br/>           | Représente une transformation 3D qui affecte le décalage d’un visuel le long de l’axe x, de l’axe y et de l’axe z. <br/>                                                                                                                                                                                                                                                                                      |
| [**IDCompositionVirtualSurface**](/windows/win32/api/dcomp/nn-dcomp-idcompositionvirtualsurface)<br/>                       | Représente une bitmap peu allouée qui peut être associée à un visuel pour la composition dans une arborescence d’éléments visuels.<br/>                                                                                                                                                                                                                                                                                  |
| [**IDCompositionVisual**](/windows/win32/api/dcomp/nn-dcomp-idcompositionvisual)<br/>                                       | Représente un visuel DirectComposition. <br/>                                                                                                                                                                                                                                                                                                                                                        |
| [**IDCompositionVisual2**](/windows/win32/api/dcomp/nn-dcomp-idcompositionvisual2)<br/>                                     | Représente un visuel DirectComposition dans une arborescence d’éléments visuels.<br/>                                                                                                                                                                                                                                                                                                                                      |
| [**IDCompositionVisual3**](/windows/win32/api/dcomp/nn-dcomp-idcompositionvisual3)<br/>                                     | Représente un visuel DirectComposition dans une arborescence d’éléments visuels.<br/>                                                                                                                                                                                                                                                                                                                                      |
| [**IDCompositionVisualDebug**](/windows/win32/api/dcomp/nn-dcomp-idcompositionvisualdebug)<br/>                             | Représente un visuel de débogage.<br/>                                                                                                                                                                                                                                                                                                                                                                     |
