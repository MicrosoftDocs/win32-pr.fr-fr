# <a name="idcompositionskewtransformsetanglex-methods"></a>IDCompositionSkewTransform :: SetAngleX, méthodes

Modifie ou anime la valeur de la propriété AngleX d’une transformation d’inclinaison 2D. La propriété AngleX spécifie l’angle de rotation, en degrés. La valeur par défaut est zéro.

### <a name="overload-list"></a>Liste de surcharge



| Méthode                                                                                                     | Description                                           |
|:-----------------------------------------------------------------------------------------------------------|:------------------------------------------------------|
| [**SetAngleX (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionskewtransform-setanglex(float))                                     | Modifie la valeur de la propriété AngleX.<br/>  |
| [**SetAngleX (IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionskewtransform-setanglex(idcompositionanimation)) | Anime la valeur de la propriété AngleX.<br/> |
