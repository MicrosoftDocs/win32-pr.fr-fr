# <a name="idcompositionrotatetransform3dsetaxisx-methods"></a>IDCompositionRotateTransform3D :: SetAxisX, méthodes

Modifie ou anime la valeur de la propriété AxisX d’une transformation de rotation 3D. La propriété AxisX spécifie la coordonnée x du vecteur d’axe de rotation. La valeur par défaut est zéro.

### <a name="overload-list"></a>Liste de surcharge



| Méthode                                                                                                       | Description                                          |
|:-------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------|
| [**SetAxisX (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform3d-setaxisx(float))                                     | Modifie la valeur de la propriété AxisX.<br/>  |
| [**SetAxisX (IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform3d-setaxisx(idcompositionanimation)) | Anime la valeur de la propriété AxisX.<br/> |
