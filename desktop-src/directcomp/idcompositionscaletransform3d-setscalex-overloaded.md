# <a name="idcompositionscaletransform3dsetscalex-methods"></a>IDCompositionScaleTransform3D :: SetScaleX, méthodes

Modifie ou anime la valeur de la propriété ScaleX d’une transformation d’échelle 3D. La propriété ScaleX spécifie le facteur d’échelle le long de l’axe x.

### <a name="overload-list"></a>Liste de surcharge



| Méthode                                                                                                        | Description                                           |
|:--------------------------------------------------------------------------------------------------------------|:------------------------------------------------------|
| [**SetScaleX (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionscaletransform3d-setscalex(float))                                     | Modifie la valeur de la propriété ScaleX.<br/>  |
| [**SetScaleX (IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionscaletransform3d-setscalex(idcompositionanimation)) | Anime la valeur de la propriété ScaleX.<br/> |
