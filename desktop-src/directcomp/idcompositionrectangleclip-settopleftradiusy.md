# <a name="idcompositionrectangleclipsettopleftradiusy-methods"></a>IDCompositionRectangleClip :: SetTopLeftRadiusY, méthodes

Modifie ou anime la valeur de la propriété TopLeftRadiusY de ce clip. La propriété TopLeftRadiusY spécifie le rayon y de l’ellipse qui arrondit l’angle supérieur gauche du clip.

### <a name="overload-list"></a>Liste de surcharge



| Méthode                                                                                                                     | Description                                                                |
|:---------------------------------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------|
| [**SetTopLeftRadiusY (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-settopleftradiusy(idcompositionanimation))                                     | Modifie la valeur de la propriété TopLeftRadiusY de ce clip.<br/>  |
| [**SetTopLeftRadiusY (IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-settopleftradiusy(idcompositionanimation)) | Anime la valeur de la propriété TopLeftRadiusY de ce clip.<br/> |
