# <a name="idcompositionskewtransformsetcentery-methods"></a>IDCompositionSkewTransform :: SetCenterY, méthodes

Modifie ou anime la valeur de la propriété CenterY d’une transformation d’inclinaison 2D. La propriété CenterY spécifie la coordonnée y du point sur lequel l’inclinaison est effectuée.

### <a name="overload-list"></a>Liste de surcharge



| Méthode                                                                                                       | Description                                            |
|:-------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------|
| [**SetCenterY (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionskewtransform-setcentery(float))                                     | Modifie la valeur de la propriété CenterY.<br/>  |
| [**SetCenterY (IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionskewtransform-setcentery(idcompositionanimation)) | Anime la valeur de la propriété CenterY.<br/> |
