# <a name="idcompositionrectangleclipsettopleftradiusx-methods"></a>IDCompositionRectangleClip :: SetTopLeftRadiusX, méthodes

Modifie ou anime la valeur de la propriété TopLeftRadiusX de ce clip. La propriété TopLeftRadiusX spécifie le rayon x de l’ellipse qui arrondit l’angle supérieur gauche du clip.

### <a name="overload-list"></a>Liste de surcharge



| Méthode                                                                                                                     | Description                                                                |
|:---------------------------------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------|
| [**SetTopLeftRadiusX (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-settopleftradiusx(idcompositionanimation))                                     | Modifie la valeur de la propriété TopLeftRadiusX de ce clip.<br/>  |
| [**SetTopLeftRadiusX (IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-settopleftradiusx(idcompositionanimation)) | Anime la valeur de la propriété TopLeftRadiusX de ce clip.<br/> |
