# <a name="idcompositionscaletransformsetcenterx-methods"></a>IDCompositionScaleTransform :: SetCenterX, méthodes

Modifie ou anime la valeur de la propriété CenterX d’une transformation d’échelle 2D. La propriété CenterX spécifie la coordonnée x du point sur lequel la mise à l’échelle est effectuée.

### <a name="overload-list"></a>Liste de surcharge



| Méthode                                                                                                        | Description                                            |
|:--------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------|
| [**SetCenterX (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionscaletransform-setcenterx(float))                                     | Modifie la valeur de la propriété CenterX.<br/>  |
| [**SetCenterX (IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionscaletransform-setcenterx(idcompositionanimation)) | Anime la valeur de la propriété CenterX.<br/> |
