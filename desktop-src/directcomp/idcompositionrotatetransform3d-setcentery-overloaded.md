# <a name="idcompositionrotatetransform3dsetcentery-methods"></a>IDCompositionRotateTransform3D :: SetCenterY, méthodes

Modifie ou anime la valeur de la propriété CenterY d’une transformation de rotation 3D. La propriété CenterY spécifie la coordonnée y du point sur lequel la rotation est effectuée. La valeur par défaut est zéro.

### <a name="overload-list"></a>Liste de surcharge



| Méthode                                                                                                           | Description                                            |
|:-----------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------|
| [**SetCenterY (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform3d-setcentery(float))                                     | Modifie la valeur de la propriété CenterY.<br/>  |
| [**SetCenterX (IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform3d-setcentery(idcompositionanimation)) | Anime la valeur de la propriété CenterY.<br/> |
