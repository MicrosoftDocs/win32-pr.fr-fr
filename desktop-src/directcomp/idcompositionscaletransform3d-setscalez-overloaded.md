# <a name="idcompositionscaletransform3dsetscalez-methods"></a>IDCompositionScaleTransform3D :: SetScaleZ, méthodes

Modifie ou anime la valeur de la propriété ScaleZ d’une transformation d’échelle. La propriété ScaleZ spécifie le facteur d’échelle le long de l’axe z.

### <a name="overload-list"></a>Liste de surcharge



| Méthode                                                                                                        | Description                                           |
|:--------------------------------------------------------------------------------------------------------------|:------------------------------------------------------|
| [**SetScaleZ (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionscaletransform3d-setscalez(float))                                     | Modifie la valeur de la propriété ScaleZ.<br/>  |
| [**SetScaleZ (IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionscaletransform3d-setscalez(idcompositionanimation)) | Anime la valeur de la propriété ScaleZ.<br/> |
