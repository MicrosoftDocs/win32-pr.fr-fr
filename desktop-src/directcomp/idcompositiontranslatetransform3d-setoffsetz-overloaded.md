# <a name="idcompositiontranslatetransform3dsetoffsetz-methods"></a>IDCompositionTranslateTransform3D :: SetOffsetZ, méthodes

Modifie ou anime la valeur de la propriété OffsetZ d’un effet de transformation de translation 3D. La propriété OffsetZ spécifie la translation le long de l’axe z.

### <a name="overload-list"></a>Liste de surcharge



| Méthode                                                                                                              | Description                                            |
|:--------------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------|
| [**SetOffsetZ (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositiontranslatetransform3d-setoffsetz(float))                                     | Modifie la valeur de la propriété OffsetZ.<br/>  |
| [**SetOffsetZ (IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositiontranslatetransform3d-setoffsetz(idcompositionanimation)) | Anime la valeur de la propriété OffsetZ.<br/> |
