# <a name="idcompositionscaletransform3dsetcenterx-methods"></a>IDCompositionScaleTransform3D :: SetCenterX, méthodes

Modifie ou anime la valeur de la propriété CenterX d’une transformation d’échelle 3D. La propriété CenterX spécifie la coordonnée x du point sur lequel la mise à l’échelle est effectuée.

### <a name="overload-list"></a>Liste de surcharge



| Méthode                                                                                                          | Description                                            |
|:----------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------|
| [**SetCenterX (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionscaletransform3d-setcenterx(float))                                     | Modifie la valeur de la propriété CenterX.<br/>  |
| [**SetCenterX (IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionscaletransform3d-setcenterx(idcompositionanimation)) | Anime la valeur de la propriété CenterX.<br/> |
