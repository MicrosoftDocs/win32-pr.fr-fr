# <a name="idcompositionrectangleclipsetbottomleftradiusy-methods"></a>IDCompositionRectangleClip :: SetBottomLeftRadiusY, méthodes

Modifie ou anime la valeur de la propriété BottomLeftRadiusY de ce clip. La propriété BottomLeftRadiusY spécifie le rayon y de l’ellipse qui arrondit l’angle inférieur gauche du clip.

### <a name="overload-list"></a>Liste de surcharge



| Méthode                                                                                                                           | Description                                                                   |
|:---------------------------------------------------------------------------------------------------------------------------------|:------------------------------------------------------------------------------|
| [**SetBottomLeftRadiusY (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-setbottomleftradiusy(idcompositionanimation))                                     | Modifie la valeur de la propriété BottomLeftRadiusY de ce clip.<br/>  |
| [**SetBottomLeftRadiusY (IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-setbottomleftradiusy(idcompositionanimation)) | Anime la valeur de la propriété BottomLeftRadiusY de ce clip.<br/> |
