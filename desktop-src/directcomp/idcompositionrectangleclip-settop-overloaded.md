# <a name="idcompositionrectangleclipsettop-methods"></a>IDCompositionRectangleClip :: SetTop, méthodes

Anime ou modifie la valeur de la propriété Top d’un rectangle de découpage. La propriété Top spécifie la coordonnée y de l’angle supérieur gauche du rectangle de découpage.

### <a name="overload-list"></a>Liste de surcharge



| Méthode                                                                                               | Description                                        |
|:-----------------------------------------------------------------------------------------------------|:---------------------------------------------------|
| [**SetTop (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-settop(float))                                     | Modifie la valeur de la propriété Top.<br/>  |
| [**SetTop (IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-settop(idcompositionanimation)) | Anime la valeur de la propriété Top.<br/> |
