# <a name="idcompositionskewtransformsetcenterx-methods"></a>IDCompositionSkewTransform :: SetCenterX, méthodes

Modifie ou anime la valeur de la propriété CenterX d’une transformation d’inclinaison 2D. La propriété CenterX spécifie la coordonnée x du point sur lequel l’inclinaison est effectuée.

### <a name="overload-list"></a>Liste de surcharge



| Méthode                                                                                                       | Description                                            |
|:-------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------|
| [**SetCenterX (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionskewtransform-setcenterx(float))                                     | Modifie la valeur de la propriété CenterX.<br/>  |
| [**SetCenterX (IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionskewtransform-setcenterx(idcompositionanimation)) | Anime la valeur de la propriété CenterX.<br/> |
