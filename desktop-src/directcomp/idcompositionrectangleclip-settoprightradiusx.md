# <a name="idcompositionrectangleclipsettoprightradiusx-methods"></a>IDCompositionRectangleClip :: SetTopRightRadiusX, méthodes

Modifie ou anime la valeur de la propriété TopRightRadiusX de ce clip. La propriété TopRightRadiusX spécifie le rayon x de l’ellipse qui arrondit l’angle supérieur droit du clip.

### <a name="overload-list"></a>Liste de surcharge



| Méthode                                                                                                                       | Description                                                                 |
|:-----------------------------------------------------------------------------------------------------------------------------|:----------------------------------------------------------------------------|
| [**SetTopRightRadiusX (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-settoprightradiusx(idcompositionanimation))                                     | Modifie la valeur de la propriété TopRightRadiusX de ce clip.<br/>  |
| [**SetTopRightRadiusX (IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-settoprightradiusx(idcompositionanimation)) | Anime la valeur de la propriété TopRightRadiusX de ce clip.<br/> |
