# <a name="idcompositionrectangleclipsetbottom-methods"></a>IDCompositionRectangleClip :: SetBottom, méthodes

Anime ou modifie la valeur de la propriété Bottom d’un rectangle de découpage. La propriété Bottom spécifie la coordonnée y du coin inférieur droit du rectangle de découpage.

### <a name="overload-list"></a>Liste de surcharge



| Méthode                                                                                                     | Description                                           |
|:-----------------------------------------------------------------------------------------------------------|:------------------------------------------------------|
| [**SetBottom (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-setbottom(float))                                     | Modifie la valeur de la propriété Bottom.<br/>  |
| [**SetBottom (IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-setbottom(idcompositionanimation)) | Anime la valeur de la propriété Bottom.<br/> |
