# <a name="idcompositiontranslatetransformsetoffsetx-methods"></a>IDCompositionTranslateTransform :: SetOffsetX, méthodes

Modifie ou anime la valeur de la propriété OffsetX d’une transformation de translation 2D. La propriété OffsetX spécifie la translation le long de l’axe x.

### <a name="overload-list"></a>Liste de surcharge



| Méthode                                                                                                            | Description                                            |
|:------------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------|
| [**SetOffsetX (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositiontranslatetransform-setoffsetx(float))                                     | Modifie la valeur de la propriété OffsetX.<br/>  |
| [**SetOffsetX (IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositiontranslatetransform-setoffsetx(idcompositionanimation)) | Anime la valeur de la propriété OffsetX.<br/> |
