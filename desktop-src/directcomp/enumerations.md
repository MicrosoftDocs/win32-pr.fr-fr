# <a name="directcomposition-enumerations"></a>Énumérations DirectComposition

Cette section décrit les énumérations fournies par l’API Microsoft DirectComposition.

## <a name="in-this-section"></a>Contenu de cette section



| Rubrique                                                                                                  | Description                                                                                                                                                                                         |
|--------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**visibilité de la \_ FACE arrière DCOMPOSITION \_**](/windows/desktop/api/DcompTypes/ne-dcomptypes-dcomposition_backface_visibility)<br/>              | Spécifie la visibilité de la face arrière à appliquer à un visuel. <br/>                                                                                                                            |
| [**\_mode d' \_ interpolation bitmap \_ DCOMPOSITION**](/windows/desktop/api/DcompTypes/ne-dcomptypes-dcomposition_bitmap_interpolation_mode)<br/> | Spécifie le mode d’interpolation à utiliser lorsqu’une image bitmap est composée d’une transformation où les pixels de la bitmap ne sont pas alignés exactement un à un avec les pixels à l’écran. <br/>         |
| [**\_mode de bordure DCOMPOSITION \_**](/windows/desktop/api/DcompTypes/ne-dcomptypes-dcomposition_border_mode)<br/>                              | Spécifie le mode de bordure à utiliser lors de la composition d’une image bitmap ou de l’application d’un clip avec une transformation telle que les bords de l’image bitmap ou du clip ne sont pas alignés sur l’axe avec les coordonnées entières. <br/> |
| [**DCOMPOSITION \_ mode composite \_**](/windows/desktop/api/DcompTypes/ne-dcomptypes-dcomposition_composite_mode)<br/>                        | Mode à utiliser pour fusionner le contenu de la bitmap d’un visuel avec la cible de rendu.<br/>                                                                                                          |
| [**\_mode d’opacité DCOMPOSITION \_**](/windows/desktop/api/dcomptypes/ne-dcomptypes-dcomposition_opacity_mode)<br/>                            | Spécifie comment la valeur d’opacité effective d’un visuel est appliquée à ce contenu et à ses enfants.<br/>                                                                                  |
| [**\_mode de transformation DCOMPOSITION \_**](/previous-versions/windows/desktop/legacy/dn904487(v=vs.85))<br/>                        | Spécifie le style de transformation à appliquer à un visuel. <br/>                                                                                                                                |
