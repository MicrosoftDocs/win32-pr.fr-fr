# <a name="idcompositionrectangleclipsetleft-methods"></a>IDCompositionRectangleClip :: SetLeft, méthodes

Anime ou modifie la valeur de la propriété Left d’un rectangle de découpage. La propriété Left spécifie la coordonnée x de l’angle supérieur gauche du rectangle de découpage.

### <a name="overload-list"></a>Liste de surcharge



| Méthode                                                                                                 | Description                                         |
|:-------------------------------------------------------------------------------------------------------|:----------------------------------------------------|
| [**SetLeft (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-setleft(float))                                     | Modifie la valeur de la propriété Left.<br/>  |
| [**SetLeft (IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-setleft(idcompositionanimation)) | Anime la valeur de la propriété de gauche.<br/> |
