# <a name="directcomposition-samples"></a>Exemples DirectComposition

> [!NOTE]
> Pour les applications sur Windows 10, nous vous recommandons d’utiliser des API Windows. UI. composition au lieu de DirectComposition. Pour plus d’informations, consultez [moderniser votre application de bureau à l’aide de la couche visuelle](/windows/uwp/composition/visual-layer-in-desktop-apps).

Les exemples d’applications suivants montrent comment utiliser l’API Microsoft DirectComposition et illustrer ses fonctionnalités.

| [Exemple DirectComposition Effects with Direct2D bitmap content](https://github.com/microsoft/Windows-classic-samples/tree/master/Samples/DirectCompositionEffects)<br/>                                                     | Montre comment utiliser DirectComposition pour appliquer des animations et des effets aux visuels qui ont du contenu bitmap Direct2D. <br/>                                                                                                                                           |
