# <a name="idcompositionscaletransform3dsetcentery-methods"></a>IDCompositionScaleTransform3D :: SetCenterY, méthodes

Modifie ou anime la valeur de la propriété CenterY d’une transformation d’échelle 3D. La propriété CenterY spécifie la coordonnée y du point sur lequel la mise à l’échelle est effectuée.

### <a name="overload-list"></a>Liste de surcharge



| Méthode                                                                                                          | Description                                            |
|:----------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------|
| [**SetCenterY (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionscaletransform3d-setcentery(float))                                     | Modifie la valeur de la propriété CenterY.<br/>  |
| [**SetCenterY (IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionscaletransform3d-setcentery(idcompositionanimation)) | Anime la valeur de la propriété CenterY.<br/> |
