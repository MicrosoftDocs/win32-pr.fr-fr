# <a name="idcompositiontranslatetransform3dsetoffsety-methods"></a>IDCompositionTranslateTransform3D :: SetOffsetY, méthodes

Modifie ou anime la valeur de la propriété OffsetY d’un effet de transformation de translation 3D. La propriété OffsetY spécifie la translation le long de l’axe y.

### <a name="overload-list"></a>Liste de surcharge



| Méthode                                                                                                              | Description                                            |
|:--------------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------|
| [**SetOffsetY (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositiontranslatetransform3d-setoffsety(float))                                     | Modifie la valeur de la propriété OffsetY.<br/>  |
| [**SetOffsetY (IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositiontranslatetransform3d-setoffsety(idcompositionanimation)) | Anime la valeur de la propriété OffsetY.<br/> |
