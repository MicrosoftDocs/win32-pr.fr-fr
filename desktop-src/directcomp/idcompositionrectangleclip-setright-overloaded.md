# <a name="idcompositionrectangleclipsetright-methods"></a>IDCompositionRectangleClip :: SetRight, méthodes

Anime ou modifie la valeur de la propriété de droite d’un rectangle de découpage. La propriété Right spécifie la coordonnée x du coin inférieur droit du rectangle de découpage.

### <a name="overload-list"></a>Liste de surcharge



| Méthode                                                                                                   | Description                                          |
|:---------------------------------------------------------------------------------------------------------|:-----------------------------------------------------|
| [**SetRight (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-setright(float))                                     | Modifie la valeur de la propriété de droite.<br/>  |
| [**SetRight (IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-setright(idcompositionanimation)) | Anime la valeur de la propriété de droite.<br/> |
