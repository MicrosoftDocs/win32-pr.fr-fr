# <a name="idcompositionscaletransform3dsetscaley-methods"></a>IDCompositionScaleTransform3D :: SetScaleY, méthodes

Modifie ou anime la valeur de la propriété ScaleY d’une transformation d’échelle. La propriété ScaleY spécifie le facteur d’échelle le long de l’axe y.

### <a name="overload-list"></a>Liste de surcharge



| Méthode                                                                                                        | Description                                           |
|:--------------------------------------------------------------------------------------------------------------|:------------------------------------------------------|
| [**SetScaleY (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionscaletransform3d-setscaley(float))                                     | Modifie la valeur de la propriété scaleY.<br/>  |
| [**SetScaleY (IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionscaletransform3d-setscaley(idcompositionanimation)) | Anime la valeur de la propriété scaleY.<br/> |
