# <a name="ui_pkey_fontproperties_backgroundcolor"></a>IU \_ \_ FontProperties \_ backgroundColor

Identifie la \_ \_ propriété FontProperties BackgroundColor de l’interface utilisateur \_ .

```
propertyDescription
   name = UI_PKEY_FontProperties_BackgroundColor
   shellPKey = UI_PKEY_FontProperties_BackgroundColor
   formatID = 00000309-7363-696e-8441798acf5aebb7
   propID = 309
   typeInfo
      type = COLORREF
```

## <a name="remarks"></a>Notes

L’interface utilisateur \_ \_ FontProperties \_ backgroundColor est utilisée par une application, conjointement avec [l' \_ UI \_ FontProperties \_ BackgroundColorType](windowsribbon-reference-properties-uipkey-fontproperties-backgroundcolortype.md), pour interroger les paramètres de la Galerie de **couleurs de surbrillance du texte** .

La valeur par défaut est `0x00000000`.
