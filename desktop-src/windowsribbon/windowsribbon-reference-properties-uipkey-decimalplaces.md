# <a name="ui_pkey_decimalplaces"></a>IU \_ , \_ décimal

Identifie la propriété de la propriété de l’interface utilisateur \_ \_ .

```
propertyDescription
   name = UI_PKEY_DecimalPlaces
   shellPKey = UI_PKEY_DecimalPlaces
   formatID = 00000206-7363-696e-8441798acf5aebb7
   propID = 206
   typeInfo
      type = VT_UI4
```

## <a name="remarks"></a>Remarques

L’interface utilisateur de l’IU \_ \_ décimale est utilisée par une application pour interroger le nombre de chiffres affichés après la virgule décimale dans le champ d’édition du contrôle [Spinner](windowsribbon-controls-spinner.md) .
