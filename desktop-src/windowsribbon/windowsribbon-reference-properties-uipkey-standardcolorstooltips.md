# <a name="ui_pkey_standardcolorstooltips"></a>IU \_ \_ StandardColorsTooltips

Identifie la propriété StandardColorsTooltips de l’interface utilisateur \_ \_ .

```
propertyDescription
   name = UI_PKEY_StandardColorsTooltips
   shellPKey = UI_PKEY_StandardColorsTooltips
   formatID = 00000412-7363-696e-8441798acf5aebb7
   propID = 412
   typeInfo
      type = VT_VECTOR | VT_LPWSTR
```

## <a name="remarks"></a>Notes

L’interface utilisateur \_ \_ StandardColorsTooltips est utilisée par une application pour interroger les info-bulles d’échantillon de couleur d’un [**DropDownColorPicker**](windowsribbon-element-dropdowncolorpicker.md).

La valeur de la propriété est un tableau de valeurs de chaîne.

Chaque valeur de chaîne correspond à l’info-bulle d’un échantillon de couleur dans un [**DropDownColorPicker**](windowsribbon-element-dropdowncolorpicker.md) , quelle que soit la valeur spécifiée pour l’attribut **ColorTemplate** .
