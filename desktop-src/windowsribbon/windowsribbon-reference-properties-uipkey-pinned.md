# <a name="ui_pkey_pinned"></a>IU de l’IU \_ \_ épinglé

Identifie la \_ propriété épinglée de l’interface utilisateur \_ .

```
propertyDescription
   name = UI_PKEY_Pinned
   shellPKey = UI_PKEY_Pinned
   formatID = 00000351-7363-696e-8441798acf5aebb7
   propID = 351
   typeInfo
      type = VT_BOOL
   booleanFormat
      formatAs = VARIANT_TRUE=-1, VARIANT_FALSE=0
```

## <a name="remarks"></a>Notes

L’interface utilisateur de \_ \_ l’IU attachée est utilisée par une application pour demander si un élément du menu de l' [application](windowsribbon-controls-applicationmenu.md) est « épinglé », ou persistant, entre les instances d’application. Par exemple, un élément de la liste des éléments utilisés le plus récemment (MRU) est accessible et ne supprime pas la liste des éléments MRU tant qu’il n’est pas « non épinglé ».
