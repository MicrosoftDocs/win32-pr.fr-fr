# <a name="ui_pkey_maxvalue"></a>IU-# \_ \_ MaxValue

Identifie la propriété de la \_ \_ propriété hypermaxvalue de l’interface utilisateur.

```
propertyDescription
   name = UI_PKEY_MaxValue
   shellPKey = UI_PKEY_MaxValue
   formatID = 00000203-7363-696e-8441798acf5aebb7
   propID = 203
   typeInfo
      type = VT_DECIMAL
```

## <a name="remarks"></a>Remarques

L' \_ UI \_ hypermaxvalue est utilisé par une application pour interroger la valeur maximale du contrôle [Spinner](windowsribbon-controls-spinner.md) .
