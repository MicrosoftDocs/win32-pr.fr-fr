# <a name="ui_pkey_colormode"></a>IU de l’interface utilisateur \_ \_

Identifie la propriété de la propriété de l’interface utilisateur \_ \_ .

```
propertyDescription
   name = UI_PKEY_ColorMode
   shellPKey = UI_PKEY_ColorMode
   formatID = 00000402-7363-696e-8441798acf5aebb7
   propID = 402
   typeInfo
      type = UI_SWATCHCOLORMODE
```

## <a name="remarks"></a>Remarques

La valeur de la propriété provient de l’énumération [**\_ SWATCHCOLORMODE de l’interface utilisateur**](/windows/desktop/api/Uiribbon/ne-uiribbon-ui_swatchcolormode) .
