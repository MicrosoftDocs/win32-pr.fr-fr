# <a name="ui_pkey_morecolorslabel"></a>IU \_ \_ MoreColorsLabel

Identifie la propriété MoreColorsLabel de l’interface utilisateur \_ \_ .

```
propertyDescription
   name = UI_PKEY_MoreColorsLabel
   shellPKey = UI_PKEY_MoreColorsLabel
   formatID = 00000408-7363-696e-8441798acf5aebb7
   propID = 408
   typeInfo
      type = VT_LPWSTR
```

## <a name="remarks"></a>Notes

L’interface utilisateur \_ \_ MoreColorsLabel est utilisée par une application pour interroger la valeur de l’étiquette associée au bouton **couleurs supplémentaires** d’un [**DropDownColorPicker**](windowsribbon-element-dropdowncolorpicker.md).

L’interface utilisateur \_ \_ de l’MoreColorsLabel est valide uniquement pour un [**DropDownColorPicker**](windowsribbon-element-dropdowncolorpicker.md) où `ThemeColors` ou `StandardColors` est spécifié comme valeur de l’attribut **ColorTemplate** .
