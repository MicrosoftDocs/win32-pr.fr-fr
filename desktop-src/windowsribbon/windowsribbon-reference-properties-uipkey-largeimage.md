# <a name="ui_pkey_largeimage"></a>IU \_ \_ LARGEIMAGE

Identifie la propriété LARGEIMAGE de l’interface utilisateur \_ \_ .

```
propertyDescription
   name = UI_PKEY_LargeImage
   shellPKey = UI_PKEY_LargeImage
   formatID = 00000007-7363-696e-8441798acf5aebb7
   propID = 7
   typeInfo
      type = IUIImage
```

## <a name="remarks"></a>Notes

L’interface utilisateur \_ \_ LARGEIMAGE est utilisée par une application pour interroger la grande image associée à un contrôle de ruban.

La valeur de la propriété est un objet [**IUIImage**](/windows/desktop/api/uiribbon/nn-uiribbon-iuiimage) .
