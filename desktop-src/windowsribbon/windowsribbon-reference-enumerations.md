# <a name="constants-and-enumerations"></a>Constantes et énumérations

documentation de référence pour les Windows les constantes et les énumérations de l’infrastructure du ruban.

### <a name="constants-and-enumerations"></a>Constantes et énumérations




| Constante | Description | 
|----------|-------------|
| <a href="/windows/desktop/windowsribbon/windowsribbon-ui-all-commands"><strong>UI_ALL_COMMANDS</strong></a><br /> | Spécifie une constante qui identifie la collection de commandes déclarée dans le fichier de ressources de balisage.<br /> | 
| <a href="/windows/desktop/windowsribbon/windowsribbon-ui-collection-invalidindex"><strong>UI_COLLECTION_INVALIDINDEX</strong></a><br /> | Spécifie une constante qui identifie un index non valide dans une collection.<br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_collectionchange"><strong>UI_COLLECTIONCHANGE</strong></a><br /> | Spécifie des valeurs qui identifient les types de modifications qui peuvent être apportées à une collection.<br /> | 
| <a href="/previous-versions/windows/desktop/legacy/hh448736(v=vs.85)"><strong>UI_COMMANDCATEGORY</strong></a><br /> | Spécifie des valeurs qui indiquent si une commande a été définie au moment de la compilation ou au moment de l’exécution.<br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_commandtype"><strong>UI_COMMANDTYPE</strong></a><br /> | Spécifie des valeurs qui identifient le type de commande associé à un contrôle de ruban.<br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_contextavailability"><strong>UI_CONTEXTAVAILABILITY</strong></a><br /> | Spécifie des valeurs qui identifient la disponibilité d’un <a href="windowsribbon-element-ribbon-contextualtabs.md"><strong>onglet contextuel</strong></a>.<br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_controldock"><strong>UI_CONTROLDOCK</strong></a><br /> | Spécifie des valeurs qui identifient l’état d’ancrage de la barre d’outils accès rapide (QAT). <br /> | 
| <a href="/windows/desktop/api/Uiribbon/ne-uiribbon-ui_eventlocation"><strong>UI_EVENTLOCATION</strong></a><br /> | Identifie les emplacements où les événements associés à un contrôle de ruban peuvent provenir.<br /> | 
| <a href="/windows/desktop/api/Uiribbon/ne-uiribbon-ui_eventtype"><strong>UI_EVENTTYPE</strong></a><br /> | Identifie les types d’événements associés à un <a href="windowsribbon-element-ribbon.md"><strong>ruban</strong></a>.<br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_executionverb"><strong>UI_EXECUTIONVERB</strong></a><br /> | Spécifie des valeurs qui identifient les ID d’exécution qui correspondent aux actions qu’un utilisateur peut lancer sur une <a href="windowsribbon-element-command.md"><strong>commande</strong></a>. <br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_fontdeltasize"><strong>UI_FONTDELTASIZE</strong></a><br /> | Spécifie des valeurs qui identifient si la taille de police d’une séquence de texte en surbrillance doit être incrémentée ou décrémentée.<br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_fontproperties"><strong>UI_FONTPROPERTIES</strong></a><br /> | Spécifie des valeurs qui identifient l’état de propriété de police d’un <a href="windowsribbon-element-fontcontrol.md"><strong>FontControl</strong></a>, tel que <strong>barré</strong>.<br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_fontunderline"><strong>UI_FONTUNDERLINE</strong></a><br /> | Spécifie des valeurs qui identifient l’état de soulignement d’un <a href="windowsribbon-element-fontcontrol.md"><strong>FontControl</strong></a>.<br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_fontverticalposition"><strong>UI_FONTVERTICALPOSITION</strong></a><br /> | Spécifie des valeurs qui identifient l’état d’alignement vertical d’un <a href="windowsribbon-element-fontcontrol.md"><strong>FontControl</strong></a>.<br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_invalidations"><strong>UI_INVALIDATIONS</strong></a><br /> | Spécifie des valeurs qui identifient l’aspect d’une commande à invalider.<br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_ownership"><strong>UI_OWNERSHIP</strong></a><br /> | Spécifie des valeurs qui identifient les conditions de propriété sous lesquelles une image est créée par l’infrastructure du ruban.<br /> | 
| <a href="/windows/desktop/api/Uiribbon/ne-uiribbon-ui_swatchcolormode"><strong>UI_SWATCHCOLORMODE</strong></a><br /> | Spécifie si une nuance est en mode normal ou monochrome.<br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_swatchcolortype"><strong>UI_SWATCHCOLORTYPE</strong></a><br /> | Spécifie les valeurs qui identifient la façon dont un échantillon de couleur d’un <a href="windowsribbon-element-dropdowncolorpicker.md"><strong>DropDownColorPicker</strong></a> ou d’un sélecteur de couleurs <a href="windowsribbon-element-fontcontrol.md"><strong>FontControl</strong></a> (<strong>couleur du texte</strong> ou <strong>mise en surbrillance du texte</strong>) est rempli.<br /><blockquote>[!Note]<br />Il s’agit uniquement de recommandations. L’application peut associer n’importe quel type de remplissage à ces valeurs.</blockquote><br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_viewtype"><strong>UI_VIEWTYPE</strong></a><br /> | Spécifie des valeurs qui identifient la vue du Framework du ruban.<br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_viewverb"><strong>UI_VIEWVERB</strong></a><br /> | Spécifie des valeurs qui identifient le type d’action à effectuer sur une vue de l’infrastructure du ruban.<br /> |
