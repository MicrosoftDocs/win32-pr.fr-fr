# <a name="dynamicui-element"></a>Élément DynamicUI

Représente un conteneur pour le balisage du ruban. Ce conteneur et son contenu sont déclarés au moment de l’exécution.

## <a name="usage"></a>Utilisation

``` syntax
<DynamicUI>
  child elements
</DynamicUI>
```

## <a name="attributes"></a>Attributs

Il n’y a pas d’attributs.

## <a name="child-elements"></a>Éléments enfants



| Élément                                                               | Description                                     |
|-----------------------------------------------------------------------|-------------------------------------------------|
| [**ContextPopup**](windowsribbon-element-contextpopup.md)<br/> | Doit se produire exactement une fois <br/> <br/> |
| [**Onglet**](windowsribbon-element-tab.md)<br/>                   | Doit se produire exactement une fois <br/> <br/> |
| [**TabGroup**](windowsribbon-element-tabgroup.md)<br/>         | Doit se produire exactement une fois <br/> <br/> |



## <a name="parent-elements"></a>Éléments parents

Il n’y a pas d’éléments parents.

## <a name="element-information"></a>Informations sur les éléments



|                                     |                                                                         |
|-------------------------------------|-------------------------------------------------------------------------|
| Système minimal pris en charge<br/> | Windows 7, Windows Vista avec SP2 et la mise à jour de la plateforme pour Windows Vista |
| Peut être vide                        | Non                                                                      |
