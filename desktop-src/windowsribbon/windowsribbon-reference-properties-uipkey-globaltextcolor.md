# <a name="ui_pkey_globaltextcolor"></a>IU \_ \_ GlobalTextColor

Identifie la propriété GlobalTextColor de l’interface utilisateur \_ \_ .

```
propertyDescription
   name = UI_PKEY_GlobalTextColor
   shellPKey = UI_PKEY_GlobalTextColor
   formatID = 00002002-7363-696e-8441798acf5aebb7
   propID = 2002
   typeInfo
      type = VT_UINT
```

## <a name="remarks"></a>Notes

L’interface utilisateur \_ \_ GlobalTextColor est utilisée par une application pour interroger la couleur de texte globale d’un ruban personnalisé.

introduite dans Windows 8 **:** les modifications apportées à la valeur par défaut de l' [interface utilisateur \_ \_ GlobalBackgroundColor](windowsribbon-reference-properties-uipkey-globalbackgroundcolor.md) dans Windows 8 peuvent nécessiter un ajustement de l’interface utilisateur \_ \_ GlobalTextColor dans les applications de ruban conçues pour Windows 7.
