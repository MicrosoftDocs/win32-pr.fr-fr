# <a name="ui_pkey_formatstring"></a>IU de l’interface utilisateur \_ \_ FormatString

Identifie la propriété de la propriété de l’interface utilisateur \_ \_ FormatString.

```
propertyDescription
   name = UI_PKEY_FormatString
   shellPKey = UI_PKEY_FormatString
   formatID = 00000207-7363-696e-8441798acf5aebb7
   propID = 207
   typeInfo
      type = VT_LPWSTR
```

## <a name="remarks"></a>Remarques

L’interface utilisateur de l’IU \_ \_ FormatString est utilisée par une application pour interroger le format (ou l’unité de mesure) pour la valeur du contrôle [Spinner](windowsribbon-controls-spinner.md) .
