# <a name="ui_pkey_itemimage"></a>IU \_ \_ ItemImage

Identifie la propriété ItemImage de l’interface utilisateur \_ \_ .

```
propertyDescription
   name = UI_PKEY_ItemImage
   shellPKey = UI_PKEY_ItemImage
   formatID = 00000106-7363-696e-8441798acf5aebb7
   propID = 106
   typeInfo
      type = IUIImage
```

## <a name="remarks"></a>Remarques

L’interface utilisateur \_ \_ ItemImage est utilisée par une application pour interroger l’image associée à un élément ou une commande dans un contrôle Gallery.

La valeur de la propriété est un objet [**IUIImage**](/windows/desktop/api/uiribbon/nn-uiribbon-iuiimage) .
