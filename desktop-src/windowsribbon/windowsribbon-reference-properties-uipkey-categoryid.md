# <a name="ui_pkey_categoryid"></a>Code de catégorie d’interface utilisateur \_ \_

Identifie la propriété de la propriété de code de catégorie de l’interface utilisateur \_ \_ .

```
propertyDescription
   name = UI_PKEY_CategoryId
   shellPKey = UI_PKEY_CategoryId
   formatID = 00000103-7363-696e-8441798acf5aebb7
   propID = 103
   typeInfo
      type = VT_UI4
```

## <a name="remarks"></a>Remarques

L’interface de \_ \_ groupe de lignes de l’interface utilisateur est utilisée par une application pour interroger les ID des catégories utilisées pour regrouper des éléments connexes dans un contrôle de Galerie.

La valeur de propriété est l’index de base zéro d’un élément dans une collection de catégories.

La capture d’écran suivante illustre l’utilisation des catégories dans un menu [**SplitButton**](windowsribbon-element-splitbutton.md) dans lequel l’ID des **formes de sélection** a pour valeur 0 et l’ID des options de **sélection** a la valeur 1.
