# <a name="others"></a>Autres

documentation de référence pour les éléments de balisage supplémentaires dans l’infrastructure de ruban Windows.



| Rubrique                                                                              | Contenu                                                                                                                                                                                                       |
|------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Application**](windowsribbon-element-application.md)                           | Représente l’élément de niveau supérieur dans la spécification du balisage de l’infrastructure du ruban.<br/>                                                                                                                      |
| [**Application. commandes**](windowsribbon-element-application-commands.md)         | Représente un conteneur pour toutes les commandes qui sont définies par l’application.<br/>                                                                                                                    |
| [**Application. views**](windowsribbon-element-application-views.md)               | Représente un conteneur pour chaque vue de l’infrastructure du ruban, en particulier le [**ruban**](windowsribbon-element-ribbon.md) et le [**ContextPopup**](windowsribbon-element-contextpopup.md).<br/>          |
| [**ColumnBreak**](windowsribbon-element-columnbreak.md)                           | Représente un séparateur vertical (visible ou masqué) dans les modèles de disposition [**SizeDefinition**](windowsribbon-element-sizedefinition.md) personnalisés.<br/>                                                      |
| [**ControlGroup**](windowsribbon-element-controlgroup.md)                         | Représente un groupe de contrôles dans un modèle de disposition [**SizeDefinition**](windowsribbon-element-sizedefinition.md) .<br/>                                                                                 |
| [**ControlNameDefinition**](windowsribbon-element-controlnamedefinition.md)       | Représente le nom d’un contrôle dans un modèle de disposition [**SizeDefinition**](windowsribbon-element-sizedefinition.md) personnalisé.<br/>                                                                          |
| [**ControlNameMap**](windowsribbon-element-controlnamemap.md)                     | Représente un conteneur pour les noms de contrôles dans un modèle de disposition [**SizeDefinition**](windowsribbon-element-sizedefinition.md) personnalisé.<br/>                                                                |
| [**ControlSizeDefinition**](windowsribbon-element-controlsizedefinition.md)       | Représente le style de disposition d’un groupe de contrôles dans un modèle personnalisé.<br/>                                                                                                                            |
| [**FlowMenuLayout**](windowsribbon-element-flowmenulayout.md)                     | Représente une disposition horizontale avec des sauts de ligne pour les éléments d’une galerie.<br/>                                                                                                                             |
| [**GroupSizeDefinition**](windowsribbon-element-groupsizedefinition.md)           | Représente la taille de disposition d’un groupe de contrôles dans un modèle personnalisé.<br/>                                                                                                                              |
| [**Ligne**](windowsribbon-element-row.md)                                           | Représente une ligne de contrôles dans un modèle de disposition SizeDefinition personnalisé.<br/>                                                                                                                            |
| [**Scale**](windowsribbon-element-scale.md)                                       | Représente la préférence de taille et de disposition d’un [**groupe**](windowsribbon-element-group.md) de contrôles via une paire {**Group**, [**SizeDefinition**](windowsribbon-element-sizedefinition.md)}.<br/> |
| [**ScalingPolicy**](windowsribbon-element-scalingpolicy.md)                       | Représente un conteneur pour la mise à l’échelle des spécifications.<br/>                                                                                                                                                  |
| [**ScalingPolicy.IdealSizes**](windowsribbon-element-scalingpolicy-idealsizes.md) | Représente un conteneur de spécifications de mise à l’échelle pour le modèle [**SizeDefinition**](windowsribbon-element-sizedefinition.md) préféré, en fonction de la taille du ruban.<br/>                                   |
| [**SizeDefinition**](windowsribbon-element-sizedefinition.md)                     | Représente un modèle de disposition personnalisé de contrôles de ruban. <br/>                                                                                                                                            |
| [**VerticalMenuLayout**](windowsribbon-element-verticalmenulayout.md)             | Représente une disposition verticale pour les éléments d’une galerie.<br/>                                                                                                                                                |
