# <a name="ui_pkey_representativestring"></a>IU \_ \_ RepresentativeString

Identifie la propriété RepresentativeString de l’interface utilisateur \_ \_ .

```
propertyDescription
   name = UI_PKEY_RepresentativeString
   shellPKey = UI_PKEY_RepresentativeString
   formatID = 00000208-7363-696e-8441798acf5aebb7
   propID = 208
   typeInfo
      type = VT_LPWSTR
```

## <a name="remarks"></a>Notes

L’interface utilisateur \_ \_ RepresentativeString est utilisée par une application pour interroger la longueur du champ d’édition d’un contrôle [Spinner](windowsribbon-controls-spinner.md) .

> [!Note]  
> La valeur de l’interface utilisateur \_ \_ RepresentativeString est utilisée par l’infrastructure pour calculer la largeur du contrôle [Spinner](windowsribbon-controls-spinner.md) et n’est jamais affichée.
