# <a name="commands-and-resources"></a>Commandes et ressources

documentation de référence pour les éléments de balisage de commande et de ressource dans l’infrastructure de ruban Windows.



| Rubrique                                                                                            | Contenu                                                                                                                |
|--------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------|
| [**Commande**](windowsribbon-element-command.md)                                                 | Représente une définition de commande.<br/>                                                                             |
| [**Commande. Comment.**](windowsribbon-element-command-comment.md)                                 | Représente un commentaire, ou annotation, pour une commande.<br/>                                                          |
| [**Command.Id**](windowsribbon-element-command-id.md)                                           | Représente un ID unique pour une commande.<br/>                                                                        |
| [**Commande. KeyTip**](windowsribbon-element-command-keytip.md)                                   | Représente la touche d’interfaut pour un contrôle.<br/>                                                                         |
| [**Commande. LabelDescription**](windowsribbon-element-command-labeldescription.md)               | Représente une description de l’étiquette.<br/>                                                                              |
| [**Commande. LabelTitle**](windowsribbon-element-command-labeltitle.md)                           | Représente le titre d’une étiquette.<br/>                                                                                    |
| [**Commande. LargeHighContrastImages**](windowsribbon-element-command-largehighcontrastimages.md) | Représente un conteneur d’images ; dans ce cas, les grandes images à utiliser avec les paramètres système à contraste élevé.<br/>     |
| [**Commande. LargeImages**](windowsribbon-element-command-largeimages.md)                         | Représente un conteneur d’images ; dans ce cas, les grandes images.<br/>                                                |
| [**Command.Name**](windowsribbon-element-command-name.md)                                       | Représente le nom d’une commande.<br/>                                                                            |
| [**Commande. SmallHighContrastImages**](windowsribbon-element-command-smallhighcontrastimages.md) | Représente un conteneur d’images ; dans ce cas, il s’agit de petites images à utiliser avec les paramètres système à contraste élevé.<br/>     |
| [**Commande. SmallImages**](windowsribbon-element-command-smallimages.md)                         | Représente un conteneur d’images ; dans ce cas, les petites images.<br/>                                                |
| [**Command. Symbol**](windowsribbon-element-command-symbol.md)                                   | Représente le nom d’une [**commande**](windowsribbon-element-command.md) qui peut être référencée en externe.<br/> |
| [**Commande. TooltipDescription**](windowsribbon-element-command-tooltipdescription.md)           | Représente une description de l’info-bulle.<br/>                                                                            |
| [**Commande. TooltipTitle**](windowsribbon-element-command-tooltiptitle.md)                       | Représente un titre d’info-bulle.<br/>                                                                                  |
| [**Image**](windowsribbon-element-image.md)                                                     | Représente une image.<br/>                                                                                         |
| [**Image. source**](windowsribbon-element-image-source.md)                                       | Représente le chemin d’accès au répertoire d’une image.<br/>                                                                   |
| [**String**](windowsribbon-element-string.md)                                                   | Représente une ressource de type chaîne.<br/>                                                                                |
| [**String. Content**](windowsribbon-element-string-content.md)                                   | Représente le contenu d’une ressource de type chaîne.<br/>                                                                 |
| [**String.Id**](windowsribbon-element-string-id.md)                                             | Représente l’ID unique d’une ressource de type chaîne.<br/>                                                               |
| [**String. Symbol**](windowsribbon-element-string-symbol.md)                                     | Représente le nom d’une ressource de type chaîne.<br/>                                                                    |
