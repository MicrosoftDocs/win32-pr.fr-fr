# <a name="ui_pkey_contextavailable"></a>IU \_ \_ ContextAvailable

Identifie la propriété ContextAvailable de l’interface utilisateur \_ \_ .

```
propertyDescription
   name = UI_PKEY_ContextAvailable
   shellPKey = UI_PKEY_ContextAvailable
   formatID = 00001100-7363-696e-8441798acf5aebb7
   propID = 1100
   typeInfo
      type = UI_CONTEXTAVAILABILITY
```

## <a name="remarks"></a>Notes

L’interface utilisateur \_ \_ ContextAvailable est utilisée par une application pour interroger la disponibilité d’un contrôle de [groupe d’onglets](windowsribbon-controls-tabgroup.md) en fonction du contexte d’application actuel.
