# <a name="ui_pkey_increment"></a>Incrément de l’interface utilisateur \_ \_

Identifie la propriété d’incrément de l’interface utilisateur \_ \_ .

```
propertyDescription
   name = UI_PKEY_Increment
   shellPKey = UI_PKEY_Increment
   formatID = 00000205-7363-696e-8441798acf5aebb7
   propID = 205
   typeInfo
      type = VT_DECIMAL
```

## <a name="remarks"></a>Remarques

L’incrément de l’interface utilisateur \_ \_ est utilisé par une application pour interroger le degré d’augmentation ou de diminution de la valeur du champ d’édition du contrôle [Spinner](windowsribbon-controls-spinner.md) lorsque l’utilisateur clique sur le bouton d’incrémentation ou de décrémentation.
