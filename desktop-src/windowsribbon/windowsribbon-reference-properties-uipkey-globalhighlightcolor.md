# <a name="ui_pkey_globalhighlightcolor"></a>IU \_ \_ GlobalHighlightColor

Identifie la propriété GlobalHighlightColor de l’interface utilisateur \_ \_ .

```
propertyDescription
   name = UI_PKEY_GlobalHighlightColor
   shellPKey = UI_PKEY_GlobalHighlightColor
   formatID = 00002001-7363-696e-8441798acf5aebb7
   propID = 2001
   typeInfo
      type = VT_UI4
```

## <a name="remarks"></a>Notes

L’interface utilisateur \_ \_ GlobalHighlightColor est utilisée par une application pour interroger la couleur de surbrillance globale d’un ruban personnalisé.

**Introduit dans Windows 8 :**   L’interface utilisateur \_ \_ GlobalHighlightColor ne peut pas être définie indépendamment de [l’interface utilisateur \_ \_ GlobalBackgroundColor](windowsribbon-reference-properties-uipkey-globalbackgroundcolor.md).
