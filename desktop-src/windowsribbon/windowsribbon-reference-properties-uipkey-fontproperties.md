# <a name="ui_pkey_fontproperties"></a>IU \_ \_ FontProperties

Identifie la propriété FontProperties de l’interface utilisateur \_ \_ .

```
propertyDescription
   name = UI_PKEY_FontProperties
   shellPKey = UI_PKEY_FontProperties
   formatID = 00000300-7363-696e-8441798acf5aebb7
   propID = 300
   typeInfo
      type = IPropertyStore
```

## <a name="remarks"></a>Notes

L’interface utilisateur \_ \_ FontProperties est un magasin de propriétés utilisé par une application pour interroger toutes les propriétés de police.
