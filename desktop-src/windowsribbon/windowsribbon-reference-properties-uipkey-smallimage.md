# <a name="ui_pkey_smallimage"></a>IU \_ \_ SmallImage

Identifie la propriété SmallImage de l’interface utilisateur \_ \_ .

```
propertyDescription
   name = UI_PKEY_SmallImage
   shellPKey = UI_PKEY_SmallImage
   formatID = 00000009-7363-696e-8441798acf5aebb7
   propID = 9
   typeInfo
      type = IUIImage
```

## <a name="remarks"></a>Remarques

L’interface utilisateur \_ \_ SmallImage est utilisée par une application pour interroger la petite image associée à un contrôle de ruban.

La valeur de la propriété est un objet [**IUIImage**](/windows/desktop/api/uiribbon/nn-uiribbon-iuiimage) .
