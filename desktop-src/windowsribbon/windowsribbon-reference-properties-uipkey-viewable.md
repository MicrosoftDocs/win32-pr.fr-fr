# <a name="ui_pkey_viewable"></a>IU de l’interface utilisateur \_ \_ visible

Identifie la \_ propriété affichable de l’interface utilisateur \_ .

```
propertyDescription
   name = UI_PKEY_Viewable
   shellPKey = UI_PKEY_Viewable
   formatID = 00001000-7363-696e-8441798acf5aebb7
   propID = 1000
   typeInfo
      type = VT_BOOL
   booleanFormat
      formatAs = VARIANT_TRUE=-1, VARIANT_FALSE=0
```

## <a name="remarks"></a>Notes

L’interface utilisateur de l’IU \_ \_ est affichable par une application pour interroger l’état visible ou masqué de l’interface ruban.



| État d’affichage | Valeur de la clé de propriété |
|---------------|--------------------|
| Visible       | **false**          |
| Hidden        | **true**           |
