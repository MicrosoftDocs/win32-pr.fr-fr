# <a name="ui_pkey_decimalvalue"></a>IU \_ \_ DecimalValue

Identifie la propriété DecimalValue de l’interface utilisateur \_ \_ .

```
propertyDescription
   name = UI_PKEY_DecimalValue
   shellPKey = UI_PKEY_DecimalValue
   formatID = 00000201-7363-696e-8441798acf5aebb7
   propID = 201
   typeInfo
      type = VT_DECIMAL
```

## <a name="remarks"></a>Notes

L’interface utilisateur \_ \_ DecimalValue est utilisée par une application pour interroger la valeur dans le champ d’édition du contrôle [Spinner](windowsribbon-controls-spinner.md) .
