# <a name="ui_pkey_enabled"></a>IU-au-dessus \_ \_ activé

Identifie la propriété de l’interface utilisateur \_ \_ activée.

```
propertyDescription
   name = UI_PKEY_Enabled
   shellPKey = UI_PKEY_Enabled
   formatID = 00000004-7363-696e-8441798acf5aebb7
   propID = 1
   typeInfo
      type = VT_BOOL
   booleanFormat
      formatAs = VARIANT_TRUE=-1, VARIANT_FALSE=0
```

## <a name="remarks"></a>Remarques

L’interface utilisateur de l’IU \_ \_ activée est utilisée par une application pour demander si un contrôle est activé et en mesure de répondre à l’interaction de l’utilisateur.

> [!Note]  
> La valeur n’est pas automatiquement définie sur 0 lorsque le mode d’application est modifié et que la visibilité d’une commande est désactivée.
