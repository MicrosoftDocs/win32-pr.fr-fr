# <a name="ui_pkey_globalbackgroundcolor"></a>IU \_ \_ GlobalBackgroundColor

Identifie la propriété GlobalBackgroundColor de l’interface utilisateur \_ \_ .

```
propertyDescription
   name = UI_PKEY_GlobalBackgroundColor
   shellPKey = UI_PKEY_GlobalBackgroundColor
   formatID = 00002000-7363-696e-8441798acf5aebb7
   propID = 2000
   typeInfo
      type = VT_UI4
```

## <a name="remarks"></a>Notes

L’interface utilisateur \_ \_ GlobalBackgroundColor est utilisée par une application pour interroger la couleur d’arrière-plan globale d’un ruban personnalisé.

**introduite dans Windows 8 :** [l’interface de groupe d' \_ \_ GlobalHighlightColor de l’interface utilisateur](windowsribbon-reference-properties-uipkey-globalhighlightcolor.md) ne peut pas être définie indépendamment de l’interface utilisateur \_ \_ GlobalBackgroundColor.
