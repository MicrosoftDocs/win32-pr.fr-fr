# <a name="create-a-windows-runtime-component-for-device-access"></a>Créer un composant Windows Runtime pour l’accès des appareils

Vous pouvez créer un composant Windows Runtime pour inclure dans un wrapper le composant qui accède à votre pilote. Vous pouvez ensuite écrire une application d’appareil Windows Store pour votre appareil en C# ou JavaScript.

## <a name="instructions"></a>Instructions

Consultez l' [exemple d’accès personnalisé aux pilotes](https://github.com/microsoftarchive/msdn-code-gallery-microsoft/tree/411c271e537727d737a53fa2cbe99eaecac00cc0/Official%20Windows%20Platform%20Sample/Custom%20driver%20access%20sample). Cet exemple implémente un composant Windows Runtime pour les fonctionnalités providingthe d’un appareil spécialisé, et il utilise la fonctionnalité dans une application d’appareil Windows Store.
