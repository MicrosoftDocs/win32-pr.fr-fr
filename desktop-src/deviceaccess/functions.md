# <a name="device-access-api-functions"></a>Fonctions de l’API d’accès aux appareils

Fonctions dans l’API d’accès à l’appareil.

## <a name="in-this-section"></a>Contenu de cette section

| Rubrique | Description |
|---|---|
| [**CreateDeviceAccessInstance**](/windows/win32/api/deviceaccess/nf-deviceaccess-createdeviceaccessinstance)<br/> | Crée l’objet utilisé pour accéder à un appareil. L’objet instancié implémente les interfaces [**IDeviceIoControl**](/windows/win32/api/Deviceaccess/nn-deviceaccess-ideviceiocontrol) et [**ICreateDeviceAccessAsync**](/windows/win32/api/Deviceaccess/nn-deviceaccess-icreatedeviceaccessasync) .<br/> |
