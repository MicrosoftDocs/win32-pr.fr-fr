# <a name="d3dxcolor-structure-d3dx10mathh"></a>D3DXCOLOR, structure (D3DX10Math. h)

## <a name="syntax"></a>Syntaxe


```C++
typedef struct D3DXCOLOR {
  FLOAT r;
  FLOAT g;
  FLOAT b;
  FLOAT a;
} D3DXCOLOR, *LPD3DXCOLOR;
```
