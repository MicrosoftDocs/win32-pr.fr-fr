# <a name="shader-functions-direct3d-10-graphics"></a>Fonctions du nuanceur (Direct3D 10 Graphics)

Cette section contient des informations sur les fonctions de nuanceur suivantes :



| Fonctions                                                                          | Description                                                                 |
|------------------------------------------------------------------------------------|-----------------------------------------------------------------------------|
| [**D3D10CompileShader**](/windows/desktop/api/D3D10Shader/nf-d3d10shader-d3d10compileshader)                                   | Compilez un nuanceur.                                                           |
| [**D3D10DisassembleShader**](/windows/desktop/api/D3D10Shader/nf-d3d10shader-d3d10disassembleshader)                           | Action déconseillée. Consultez [**D3DX10DisassembleShader**](d3dx10disassembleshader.md). |
| [**D3D10GetGeometryShaderProfile**](/windows/desktop/api/D3D10Shader/nf-d3d10shader-d3d10getgeometryshaderprofile)             | Obtenir le profil de nuanceur Geometry le mieux adapté à un appareil donné.              |
| [**D3D10GetInputAndOutputSignatureBlob**](/windows/desktop/api/D3D10Shader/nf-d3d10shader-d3d10getinputandoutputsignatureblob) | Obtient une mémoire tampon qui contient des signatures de nuanceur.                               |
| [**D3D10GetInputSignatureBlob**](/windows/desktop/api/D3D10Shader/nf-d3d10shader-d3d10getinputsignatureblob)                   | Obtient une mémoire tampon qui contient des signatures d’entrée de nuanceur.                         |
| [**D3D10GetOutputSignatureBlob**](/windows/desktop/api/D3D10Shader/nf-d3d10shader-d3d10getoutputsignatureblob)                 | Obtient une mémoire tampon qui contient les signatures de sortie du nuanceur.                        |
| [**D3D10GetPixelShaderProfile**](/windows/desktop/api/D3D10Shader/nf-d3d10shader-d3d10getpixelshaderprofile)                   | Obtenir le profil de nuanceur de pixels le mieux adapté à un appareil donné.                 |
| [**D3D10GetShaderDebugInfo**](/windows/desktop/api/D3D10Shader/nf-d3d10shader-d3d10getshaderdebuginfo)                         | Obtenir les informations de débogage du nuanceur à partir d’un nuanceur compilé.                               |
| [**D3D10GetVertexShaderProfile**](/windows/desktop/api/D3D10Shader/nf-d3d10shader-d3d10getvertexshaderprofile)                 | Obtenir le profil de nuanceur de vertex le mieux adapté à un appareil donné.                |
| [**D3D10PreprocessShader**](/windows/desktop/api/D3D10Shader/nf-d3d10shader-d3d10preprocessshader)                             | Générez une chaîne de texte qui contient le flux de jetons du nuanceur.               |
| [**D3D10ReflectShader**](/windows/desktop/api/D3D10Shader/nf-d3d10shader-d3d10reflectshader)                                   | Action déconseillée. Consultez [**D3DX10ReflectShader**](d3dx10reflectshader.md).         |
