# <a name="d3d10_rect"></a>D3D10 \_ Rect

D3D10 \_ Rect est déclaré comme suit :

``` syntax
typedef RECT D3D10_RECT;
```

Pour plus d’informations sur cette structure de rectangles GDI, consultez [Rect](/previous-versions//ms536136(v=vs.85)).

## <a name="remarks"></a>Remarques

Cette structure est utilisée pour les rectangles de ciseaux par [**ID3D10Device :: RSGetScissorRects**](/windows/desktop/api/D3D10/nf-d3d10-id3d10device-rsgetscissorrects) et [**ID3D10Device :: RSSetScissorRects**](/windows/desktop/api/D3D10/nf-d3d10-id3d10device-rssetscissorrects).
