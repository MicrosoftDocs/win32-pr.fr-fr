# <a name="effect-reference-direct3d-10-graphics"></a>Référence des effets (graphiques Direct3D 10)

L’API Direct3D définit plusieurs éléments d’API pour vous aider à créer et à gérer le système d’effet.

-   [Interfaces](d3d10-graphics-reference-effect-interfaces.md)
-   [Fonctions](d3d10-graphics-reference-effect-functions.md)
-   [Structures](d3d10-graphics-reference-effect-structures.md)
-   [Constantes](d3d10-graphics-reference-effect-constants.md)

Le système d’effet encapsule les assignations d’État dans un effet. Chaque effet est une collection d’États de pipeline qui peut être décomposée en affectations d’État à l’aide de blocs d’État, de ressources et de nuanceurs. Selon la taille d’un effet, vous pouvez choisir d’en implémenter un dans une chaîne de texte ASCII ou un fichier d’effet (. FX). Pour organiser les informations dans un effet, consultez le format de l' [effet](d3d10-effect-format.md).

Après avoir conçu un effet, utilisez l’API Effect pour définir l’état de l’appareil pendant le rendu. En outre, le système d’effet prend en charge une série d’API de réflexion pour l’obtention et la définition des données d’effet. Pour plus d’informations, consultez [effets système interfaces (Direct3D 10)](d3d10-graphics-programming-guide-effects-interfaces.md).
