# <a name="id3dx10meshbuffergetsize-method"></a>ID3DX10MeshBuffer :: deméthode, méthode

Obtient la taille de la mémoire tampon de maillage, en octets.

## <a name="syntax"></a>Syntaxe


```C++
SIZE_T GetSize();
```



## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.

## <a name="return-value"></a>Valeur retournée

Type : **[ **taille \_ T**](../winprog/windows-data-types.md)**

Taille de la mémoire tampon de maillage, en octets.
