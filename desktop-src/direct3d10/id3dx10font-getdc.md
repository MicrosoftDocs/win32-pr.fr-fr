# <a name="id3dx10fontgetdc-method"></a>ID3DX10Font :: GetDC, méthode

Retourne un handle vers un contexte de périphérique (DC) d’affichage sur lequel la police est définie.

## <a name="syntax"></a>Syntaxe


```C++
HDC GetDC();
```



## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.

## <a name="return-value"></a>Valeur retournée

Type : **[ **HDC**](../winprog/windows-data-types.md)**

Handle vers un DC d’affichage.
