# <a name="effect-functions-direct3d-10"></a>Fonctions Effect (Direct3D 10)

Cette section contient des informations sur les fonctions d’effet suivantes :



| Fonctions                                                                      | Description                                                         |
|--------------------------------------------------------------------------------|---------------------------------------------------------------------|
| [**D3D10CompileEffectFromMemory**](/windows/desktop/api/D3D10Effect/nf-d3d10effect-d3d10compileeffectfrommemory)           | Compilez un effet.                                                  |
| [**D3D10CreateEffectFromMemory**](/windows/desktop/api/D3D10Effect/nf-d3d10effect-d3d10createeffectfrommemory)             | Créez un effet.                                                   |
| [**D3D10CreateEffectPoolFromMemory**](/windows/desktop/api/D3D10Effect/nf-d3d10effect-d3d10createeffectpoolfrommemory)     | Créez un pool d’effets.                                              |
| [**D3D10CreateStateBlock**](/windows/desktop/api/D3D10Effect/nf-d3d10effect-d3d10createstateblock)                         | Créez un bloc d’État.                                               |
| [**D3D10DisassembleEffect**](/windows/desktop/api/D3D10Effect/nf-d3d10effect-d3d10disassembleeffect)                       | Désassemblez un effet compilé en instructions d’assembly de nuanceur.    |
| [**D3D10StateBlockMaskDifference**](/windows/desktop/api/D3D10Effect/nf-d3d10effect-d3d10stateblockmaskdifference)         | Combinez deux masques de bloc d’État avec un XOR au niveau du bit.                   |
| [**D3D10StateBlockMaskDisableAll**](/windows/desktop/api/D3D10Effect/nf-d3d10effect-d3d10stateblockmaskdisableall)         | Désactivez la capture d’État.                                            |
| [**D3D10StateBlockMaskDisableCapture**](/windows/desktop/api/D3D10Effect/nf-d3d10effect-d3d10stateblockmaskdisablecapture) | Désactivez la capture d’État avec un masque de bloc d’État.                    |
| [**D3D10StateBlockMaskEnableAll**](/windows/desktop/api/D3D10Effect/nf-d3d10effect-d3d10stateblockmaskenableall)           | Activez un masque de bloc d’État pour capturer et appliquer toutes les variables d’État. |
| [**D3D10StateBlockMaskEnableCapture**](/windows/desktop/api/D3D10Effect/nf-d3d10effect-d3d10stateblockmaskenablecapture)   | Active une plage de valeurs d’État dans un masque de bloc d’État.               |
| [**D3D10StateBlockMaskGetSetting**](/windows/desktop/api/D3D10Effect/nf-d3d10effect-d3d10stateblockmaskgetsetting)         | Obtient un élément dans un masque de bloc d’État.                               |
| [**D3D10StateBlockMaskIntersect**](/windows/desktop/api/D3D10Effect/nf-d3d10effect-d3d10stateblockmaskintersect)           | Combinez deux masques de bloc d’État avec un and au niveau du bit.                   |
| [**D3D10StateBlockMaskUnion**](/windows/desktop/api/D3D10Effect/nf-d3d10effect-d3d10stateblockmaskunion)                   | Combinez deux masques de bloc d’État avec une opération or au niveau du bit.                    |
