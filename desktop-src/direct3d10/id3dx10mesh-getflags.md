# <a name="id3dx10meshgetflags-method"></a>ID3DX10Mesh :: GetFlags, méthode

Accédez aux indicateurs de création du maillage.

## <a name="syntax"></a>Syntaxe


```C++
UINT GetFlags();
```



## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.

## <a name="return-value"></a>Valeur retournée

Type : **[ **uint**](../winprog/windows-data-types.md)**

Indicateurs de création passés dans le paramètre *options* de [**D3DX10CreateMesh**](d3d10-d3dx10createmesh.md) lors de la création du maillage. Consultez [**d3dx10 \_ Mesh**](d3dx10-mesh.md).
