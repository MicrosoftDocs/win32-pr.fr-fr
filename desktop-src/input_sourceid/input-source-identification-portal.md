# <a name="input-source-identification"></a>Identification de la source d’entrée

## <a name="purpose"></a>Objectif

Les rubriques de cette section fournissent une vue d’ensemble de la prise en charge de l’identification de la source d’entrée dans Windows 8. L’identification de la source d’entrée permet aux applications de détecter de manière précise et fiable une source d’entrée, d’identifier le type d’entrée et de fournir une expérience utilisateur adaptée au type d’entrée.

La collection de rubriques fournie ici décrit comment votre application peut obtenir ces informations et les utiliser au mieux pour optimiser l’expérience utilisateur de votre client.

## <a name="in-this-section"></a>Contenu de cette section

| Rubrique | Description |
|---|---|
| [Référence d’identification de la source d’entrée](input-source-identification-reference.md)<br/> | Les rubriques de cette section fournissent les spécifications de référence pour l’identification de la source d’entrée dans Windows 8.<br/> |

## <a name="developer-audience"></a>Développeurs concernés

Les API d’identification de la source d’entrée sont conçues pour les développeurs qui créent des infrastructures d’interface utilisateur qui offrent une expérience utilisateur cohérente et optimisée pour les applications de bureau.
