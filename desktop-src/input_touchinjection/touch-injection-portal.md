## <a name="purpose"></a>Objectif

les rubriques de cette section fournissent une vue d’ensemble de l’Injection tactile pour Windows applications. les api d' [Injection tactile](/windows/win32/api/_input_touchinjection/) permettent Windows aux développeurs de simuler par programmation l’entrée tactile.
