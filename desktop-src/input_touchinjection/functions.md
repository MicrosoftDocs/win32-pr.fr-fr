# <a name="touch-injection-functions"></a>Fonctions d’injection tactile

Les rubriques de cette section fournissent les spécifications de référence pour les fonctions d' [injection tactile](touch-injection-portal.md) .

## <a name="in-this-section"></a>Contenu de cette section

| Rubrique | Description |
|---|---|
| [**InitializeTouchInjection**](/windows/win32/api/winuser/nf-winuser-initializetouchinjection)<br/> | Configure le contexte d’injection tactile pour l’application appelante et initialise le nombre maximal de contacts simultanés que l’application peut injecter.<br/> |
| [**InjectTouchInput**](/windows/win32/api/winuser/nf-winuser-injecttouchinput)<br/>                 | Simule l’entrée tactile.<br/> |
