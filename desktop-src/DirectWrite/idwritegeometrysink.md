# <a name="idwritegeometrysink-interface"></a>Interface IDWriteGeometrySink

**IDWriteGeometrySink** est un **typedef** de l’interface [**ID2D1SimplifiedGeometrySink**](/windows/win32/api/d2d1/nn-d2d1-id2d1simplifiedgeometrysink) . Pour plus d’informations, consultez la page de référence de [**ID2D1SimplifiedGeometrySink**](/windows/win32/api/d2d1/nn-d2d1-id2d1simplifiedgeometrysink) .

## <a name="members"></a>Membres

L’interface **IDWriteGeometrySink** hérite de l’interface [**IUnknown**](/windows/win32/api/unknwn/nn-unknwn-iunknown) , mais n’a pas de membres supplémentaires.
