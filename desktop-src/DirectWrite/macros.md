# <a name="directwrite-macros"></a>Macros DirectWrite

DirectWrite définit les macros suivantes.

## <a name="in-this-section"></a>Contenu de cette section

| Rubrique | Description |
|-|-|
| [**DWRITE_MAKE_FONT_AXIS_TAG**](/windows/win32/api/dwrite_3/nf-dwrite_3-dwrite_make_font_axis_tag) | Crée une balise OpenType pour un axe de police. |
| [**DWRITE \_ créer \_ une \_ étiquette OpenType**](/windows/win32/api/dwrite/nf-dwrite-dwrite_make_opentype_tag) | Crée une balise OpenType sous la forme d’un entier 32 bits, de sorte que le premier caractère de la balise est l’octet le plus bas (le moins significatif sur les architectures avec primauté des octets de poids faible), qui peut être utilisé pour effectuer une comparaison avec des balises dans le fichier de police. Cette macro est compatible avec [**la \_ \_ \_ balise de fonctionnalité de police DWRITE**](/windows/win32/api/dwrite/ne-dwrite-dwrite_font_feature_tag). |
