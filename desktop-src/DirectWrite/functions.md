# <a name="directwrite-functions"></a>Fonctions de DirectWrite

DirectWrite fournit la fonction suivante. Toutes les autres opérations sont effectuées à l’aide d’objets d’interface DirectWrite créés par l’objet [**IDWriteFactory**](/windows/win32/api/dwrite/nn-dwrite-idwritefactory) créé avec la fonction ci-dessous.

## <a name="in-this-section"></a>Contenu de cette section



| Rubrique                                                         | Description                                                                                                             |
|---------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------|
| [**DWriteCreateFactory**](/windows/win32/api/dwrite/nf-dwrite-dwritecreatefactory)<br/> | Crée un objet de fabrique DirectWrite utilisé pour la création suivante d’objets DirectWrite individuels.<br/> |
