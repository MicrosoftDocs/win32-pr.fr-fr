# <a name="mib-name-tree"></a>Arborescence du nom MIB

L’espace de noms pour les identificateurs d’objet MIB est hiérarchique. Elle est structurée de sorte que chaque objet gérable peut recevoir un nom global unique.

L’autorité pour les parties de l’espace de noms est affectée à des organisations individuelles. Cela permet aux organisations d’attribuer des noms sans consulter une autorité Internet pour chaque affectation. Par exemple, l’espace de noms attribué à Microsoft est 1.3.6.1.4.1.311, qui est défini dans MSFT. MIB. Microsoft a l’autorité pour attribuer des noms à des objets situés en dessous de cet espace de noms.

L’identificateur d’objet dans la hiérarchie est écrit sous la forme d’une séquence de sous-identificateurs commençant à la racine et se terminant à l’objet. Les sous-identificateurs sont séparés par un point.
