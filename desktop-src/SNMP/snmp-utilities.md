# <a name="snmp-utilities"></a>Utilitaires SNMP

Le tableau suivant répertorie les utilitaires SNMP disponibles dans le kit de ressources Microsoft Windows.



| Nom de fichier     | Description                                                                             |
|--------------|-----------------------------------------------------------------------------------------|
| EVNTCMD.EXE  | Application de ligne de commande pour la configuration du traducteur d’événements SNMP                    |
| EVNTWIN.EXE  | Application qui fournit une interface utilisateur pour la configuration du traducteur d’événements SNMP |
| MIBCC.EXE    | Compilateur MIB SNMP                                                                   |
| SNMPUTIL.EXE | Exemple d’application de console du gestionnaire SNMP                                               |



 

Le fournisseur SNMP dans le kit de développement logiciel (SDK) Windows Management Instrumentation (WMI) permet aux applications clientes d’accéder à des informations SNMP statiques et dynamiques par le biais de WMI. Pour plus d’informations, consultez [fournisseur SNMP](/windows/desktop/WmiSdk/snmp-provider) dans la documentation du kit de développement logiciel (SDK) WMI.
