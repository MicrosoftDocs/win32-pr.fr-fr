# <a name="configuring-snmp-security"></a>Configuration de la sécurité SNMP

Les fonctionnalités de sécurité de SNMP vous permettent de spécifier les communautés et les hôtes à partir desquels un ordinateur accepte les requêtes, ainsi que le type d’opérations à accepter à partir des ordinateurs appartenant à une communauté. SNMP vous permet également de spécifier s’il faut envoyer une interruption d’authentification lorsqu’une communauté ou un hôte non autorisé demande des informations.

pour plus d’informations, reportez-vous au Kit de ressources Windows et à la documentation en ligne du service SNMP.

**Windows Server 2003 :** Par défaut, les noms de communautés ne sont pas configurés et le service SNMP est configuré pour accepter les demandes SNMP uniquement à partir du nom d’hôte « localhost ».
