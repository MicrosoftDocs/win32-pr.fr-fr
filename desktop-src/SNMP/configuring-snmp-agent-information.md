# <a name="configuring-snmp-agent-information"></a>Configuration des informations de l’agent SNMP

Les informations de l’agent SNMP vous permettent de spécifier des commentaires sur l’utilisateur et l’emplacement physique de l’ordinateur, et d’indiquer les types de service à signaler. Les types de service qui peuvent être signalés sont basés sur la configuration de l’ordinateur.
