# <a name="winsnmp-sessions"></a>Sessions WinSNMP

Une *session* est l’unité de base de la gestion des ressources et des communications entre une application WinSNMP d’appel et l’implémentation de Microsoft winsnmp. Il est recommandé qu’une application utilise des sessions pour organiser ses opérations et réduire la demande sur les ressources de l’implémentation.

Pour plus d’informations, consultez [ouverture et fermeture d’une session WinSNMP](opening-and-closing-a-winsnmp-session.md).
