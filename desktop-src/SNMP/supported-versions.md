# <a name="supported-versions"></a>Versions prises en charge

\[SNMP peut être utilisé dans les systèmes d’exploitation spécifiés dans la section Configuration requise. Il sera peut-être modifié ou indisponible dans les versions ultérieures. utilisez plutôt [Windows Remote Management](/windows/desktop/WinRM/portal), qui est l’implémentation Microsoft de WS-Man.\]

Windows Vista et Windows Server 2008 prennent en charge SNMPv1 et SNMPv2c. Cette rubrique fournit des informations de base sur chacune de ces versions de SNMP.

Le protocole SNMPv1 (simple Network Management Protocol version 1) est la version la plus ancienne du protocole. SNMPv1 est défini par RFC 1157, RFC 1155 et RFC 1212. Pour SNMPv1, la sécurité est basée sur les chaînes de communauté.

Le protocole SNMPv2c (simple Network Management Protocol version 2C) est défini par RFC 1901, RFC 1905 et RFC 1906. SNMPv2c met à jour les opérations de protocole et les types de données qui ont été définis dans SNMPv2p. SNMPv2c, connu sous le nom de « based String-base SNMPv2 », utilise la sécurité basée sur la communauté SNMPv1.

SNMPv2x est pris en charge sur les plates-formes de systèmes d’exploitation client et serveur Windows 2000 et versions ultérieures, y compris Windows XP et Windows server 2003.
