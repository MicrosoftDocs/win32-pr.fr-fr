# <a name="about-snmp-versions"></a>À propos des versions SNMP

L’infrastructure de gestion de réseau Internet standard d’origine, décrite dans les RFC 1155, 1157 et 1213, est appelée Framework SNMP version 1 (SNMPv1). Les parties pertinentes de l’infrastructure proposée pour la version 2C du protocole SNMPv2C (simple Network Management Protocol) sont décrites dans les RFC 1901 à 1908.

L’API WinSNMP prend en charge les fonctionnalités de protocole SNMP décrites dans les RFC Internet pertinentes. WinSNMP n’impose aucune contrainte quant à l’utilisation de SNMPv1 ou SNMPv2C par les applications WinSNMP.

Une entité de gestion peut prendre en charge une version différente de SNMP que celle prise en charge par l’application WinSNMP. L’implémentation de l’WinSNMP Microsoft effectue les traductions appropriées de SNMPv1 vers SNMPv2C conformément à la RFC appropriée.
