# <a name="duplicating-a-pdu"></a>Duplication d’une PDU

La fonction [**SnmpDuplicatePdu**](/windows/desktop/api/Winsnmp/nf-winsnmp-snmpduplicatepdu) duplique un PDU et alloue la mémoire nécessaire. Pour libérer les ressources allouées par **SnmpDuplicatePdu** pour une nouvelle PDU, une application WinSNMP doit appeler la fonction [**SnmpFreePdu**](/windows/desktop/api/Winsnmp/nf-winsnmp-snmpfreepdu) .
