# <a name="extensible-authentication-protocol-reference"></a>Référence du protocole EAP (Extensible Authentication Protocol)

Cette section décrit les éléments de référence utilisés pour implémenter le protocole EAP (Extensible Authentication Protocol). Parmi ces éléments de référence, citons les fonctions qui permettent d’implémenter des protocoles d’authentification, des fournisseurs d’authentification et des fournisseurs de comptes. Cette section comprend également les structures et les types énumérés utilisés par ces fonctions.
