# <a name="learn-about-the-eap-and-eaphost-relationship"></a>En savoir plus sur la relation EAP et EAPHost

Cette rubrique décrit la relation entre le protocole EAP (Extensible Authentication Protocol) et l’hôte du protocole EAP (Extensible Authentication Protocol). Pour le développement de nouvelles méthodes EAP, consultez la page [Extensible Authentication Protocol Host](../eaphost/portal.md).

## <a name="eaphost-framework"></a>Framework EAPHost

EAPHost est considéré comme une infrastructure pour le protocole EAP pour deux raisons.

-   EAPHost est un composant d’infrastructure Windows qui héberge les plug-ins EAP
-   EAPHost implémente le protocole EAP conforme à la [norme RFC 3748](https://go.microsoft.com/fwlink/p/?linkid=84063). Une authentification EAP se compose du protocole EAP conforme à la [norme RFC 3748](https://go.microsoft.com/fwlink/p/?linkid=84063)et des aspects spécifiques à la méthode EAP du protocole.
