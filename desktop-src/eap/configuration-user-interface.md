# <a name="configuration-user-interface"></a>Interface utilisateur de configuration

Les interfaces utilisateur de configuration pour les protocoles d’authentification sont implémentées différemment selon que l’interface utilisateur est destinée à la configuration du protocole d’authentification sur le client ou sur le serveur. Les rubriques suivantes décrivent le processus utilisé pour implémenter une interface utilisateur de configuration pour le client et pour le serveur :

-   [Interface utilisateur de configuration côté serveur](server-side-configuration-user-interface.md)
-   [Interface utilisateur de configuration côté client](client-side-configuration-user-interface.md)
