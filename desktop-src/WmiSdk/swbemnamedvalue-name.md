# <a name="swbemnamedvaluename-property"></a>Propriété SWbemNamedValue.Name

La propriété **Name** de l’objet [**SWbemNamedValue**](swbemnamedvalue.md) est une chaîne unique qui ne respecte pas la casse qui est utilisée lors de l’obtention d’un élément **SWbemNamedValue** . Cette propriété est en lecture seule.

Pour une explication de cette syntaxe, consultez [conventions de document pour l’API de script](document-conventions-for-the-scripting-api.md).

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```VB
SWbemNamedValue.Name As String
```
