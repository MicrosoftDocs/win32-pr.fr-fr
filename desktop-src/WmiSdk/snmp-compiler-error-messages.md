# <a name="snmp-compiler-error-messages"></a>Messages d’erreur du compilateur SNMP

Cette section décrit les différentes conditions d’erreur qui peuvent se produire lorsque le compilateur de module d’informations SNMP (simple Network Management Protocol) analyse un module MIB (Management Information base).

Les messages d’erreur ont trois niveaux de gravité :

-   Erreurs irrécupérables
-   Avertissements
-   Messages d’information

Le niveau de gravité est indiqué par des crochets pointus au début de la description du message d’erreur. La liste suivante répertorie les erreurs du fournisseur SNMP WMI :

-   [Erreurs 1001 à 1010](errors-1001-through-1010.md)
-   [Erreurs 1011 à 1020](errors-1011-through-1020.md)
-   [Erreurs 1021 à 1030](errors-1021-through-1030.md)
-   [Erreurs 1031 à 1040](errors-1031-through-1040.md)
-   [Erreurs 1041 à 1050](errors-1041-through-1050.md)
-   [Erreurs 1051 à 1060](errors-1051-through-1060.md)
-   [Erreurs 1061 à 1070](errors-1061-through-1070.md)
