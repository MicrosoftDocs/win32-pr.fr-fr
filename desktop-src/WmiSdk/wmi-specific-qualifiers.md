# <a name="wmi-specific-qualifiers"></a>Qualificateurs de WMI-Specific

Outre les qualificateurs CIM, WMI fournit un certain nombre de qualificateurs supplémentaires pour affiner davantage la fourniture des instances de classe et des opérations.

La liste suivante répertorie les qualificateurs supplémentaires fournis par WMI :

-   [Qualificateurs WMI standard](standard-wmi-qualifiers.md)
-   [Qualificateurs spécifiques au fournisseur SNMP](qualifiers-specific-to-the-snmp-provider.md)
-   [Qualificateurs spécifiques au fournisseur de vues](qualifiers-specific-to-the-view-provider.md)
-   [Qualificateurs spécifiques du fournisseur WDM](qualifiers-specific-to-the-wdm-provider.md)
-   [Qualificateurs spécifiques aux classes de performance WMI](qualifiers-specific-to-wmi-performance-classes.md)

Outre les qualificateurs, les versions décrivant le comportement des qualificateurs peuvent être appliquées aux différents qualificateurs. Pour plus d’informations, consultez [versions de qualificateur](qualifier-flavors.md).
