# <a name="swbemlasterrorproperties_-property"></a>Propriété SWbemLastError. Properties \_

La **propriété \_ Properties** de l’objet [**SWbemLastError**](swbemlasterror.md) retourne un objet [**SWbemPropertySet**](swbempropertyset.md) qui est une collection des propriétés WMI de la classe ou de l’instance en cours. Cette propriété est en lecture seule.

Pour une explication de cette syntaxe, consultez [conventions de document pour l’API de script](document-conventions-for-the-scripting-api.md).

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```VB
SWbemLastError.Properties_ As Object
```
