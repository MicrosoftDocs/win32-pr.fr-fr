# <a name="operating-system-availability-of-wmi-components"></a>Disponibilité du système d’exploitation des composants WMI

WMI est installé par défaut sur toutes les plateformes de serveur et de bureau Windows. Toutefois, certains fournisseurs WMI peuvent ou non être installés, en fonction de la version et de la configuration du système d’exploitation. Par exemple, le fournisseur SNMP n’est pas activé par défaut, et le fournisseur Windows Installer (fournisseur MSI) n’est pas installé par défaut sur les systèmes d’exploitation 64 bits. Pour plus d’informations sur la disponibilité d’un fournisseur spécifique, consultez [fournisseurs WMI](/windows/desktop/SrvNodes/wmi-mi-omi-providers).
