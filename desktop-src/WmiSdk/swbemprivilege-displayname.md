# <a name="swbemprivilegedisplayname-property"></a>SWbemPrivilege. DisplayName, propriété

La propriété **DisplayName** d’un objet [**SWbemPrivilege**](swbemprivilege.md) est une chaîne qui affiche une description d’un objet **SWbemPrivilege** . Par exemple, le privilège qui contrôle si un utilisateur peut exécuter la méthode Shutdown d’un objet a la chaîne « arrêter le système » dans la propriété **SWbemPrivilege. DisplayName** . Cette propriété est en lecture seule.

Pour une explication de cette syntaxe, consultez [conventions de document pour l’API de script](document-conventions-for-the-scripting-api.md).

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```VB
SWbemPrivilege.DisplayName As 
```
