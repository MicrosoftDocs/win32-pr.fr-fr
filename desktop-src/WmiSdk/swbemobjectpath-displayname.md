# <a name="swbemobjectpathdisplayname-property"></a>SWbemObjectPath. DisplayName, propriété

La propriété **DisplayName** de l’objet [**SWbemObjectPath**](swbemobjectpath.md) est une chaîne qui contient le chemin d’accès dans un formulaire qui peut être utilisé comme nom d’affichage du moniker.

Pour une explication de cette syntaxe, consultez [conventions de document pour l’API de script](document-conventions-for-the-scripting-api.md).

Cette propriété est en lecture/écriture.

## <a name="syntax"></a>Syntaxe


```VB
SWbemObjectPath.DisplayName As String
```
