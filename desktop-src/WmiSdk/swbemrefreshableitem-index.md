# <a name="swbemrefreshableitemindex-property"></a>Propriété SWbemRefreshableItem. index

La propriété **SWbemRefreshableItem. index** représente l’index de l’élément dans son objet [**SWbemRefresher**](swbemrefresher.md) parent.

Pour une explication de cette syntaxe, consultez [conventions de document pour l’API de script](document-conventions-for-the-scripting-api.md).

Cette propriété est en lecture/écriture.

## <a name="syntax"></a>Syntaxe


```VB
SWbemRefreshableItem.Index As Long
```
