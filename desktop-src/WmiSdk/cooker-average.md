# <a name="cooker_average"></a>moyenne de cuiseur \_

La \_ formule de type de compteur moyenne de cuiseur fournit le point médian statistique des données pour un compteur. La formule de ce type de compteur additionne les observations répétées d’une propriété dans une instance de [**\_ PerfRawData Win32**](/windows/desktop/CIMWin32Prov/win32-perfrawdata) et divise la somme par le nombre d’observations. Ce type de compteur est défini uniquement dans WMI et n’est pas disponible pour les technologies d’analyse des performances, telles que les [compteurs de performance Version 6,0](/windows/desktop/PerfCtrs/performance-counters-portal).

Pour plus d’informations sur la formule de type de compteur, consultez [types de compteurs](/previous-versions/windows/it-pro/windows-server-2003/cc785636(v=ws.10)).

Pour plus d’informations sur les fournisseurs et les scripts WMI à hautes performances, consultez [types de compteurs de performance WMI](wmi-performance-counter-types.md).
