# <a name="accessing-other-operating-system-features-with-wmi"></a>Accès à d’autres fonctionnalités du système d’exploitation avec WMI

WMI contient des extensions ou des fournisseurs qui vous permettent d’accéder à d’autres fonctionnalités du système d’exploitation, telles que Active Directory, ODBC et SNMP. Vous pouvez également accéder aux données à partir des pilotes de périphérique.

Les extensions WMI ne sont pas disponibles dans tous les systèmes d’exploitation. Pour plus d’informations, consultez [disponibilité du système d’exploitation des composants WMI](operating-system-availability-of-wmi-components.md).

Les rubriques suivantes fournissent des informations sur l’accès aux autres fonctionnalités du système d’exploitation :

-   [Accès aux données à partir des pilotes de périphérique](accessing-data-from-device-drivers.md)
-   [Accès Active Directory](accessing-active-directory.md)
-   [Accès aux appareils SNMP](accessing-snmp-devices.md)
