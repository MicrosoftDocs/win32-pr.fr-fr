# <a name="m-wmi"></a>M (WMI)

[A](gloss-a.md) B [C](gloss-c.md) [D](gloss-d.md) [e](gloss-e.md) [F](gloss-f.md) G [H](gloss-h.md) [I](gloss-i.md) J [K](gloss-k.md) [L](gloss-l.md) M [N](gloss-n.md) [O](gloss-o.md) [P](gloss-p.md) [Q](gloss-q.md) [R](gloss-r.md) [S](gloss-s.md) [T](gloss-t.md) U V [W](gloss-w.md) X Y Z

<dl> <dt>

<span id="wmi.gloss_managed_object"></span><span id="WMI.GLOSS_MANAGED_OBJECT"></span>**objet géré**
</dt> <dd>

Composant du système matériel ou logiciel qui est représenté sous la forme d’une instance d’une classe WMI.

</dd> <dt>

<span id="wmi.gloss_managed_object_format"></span><span id="WMI.GLOSS_MANAGED_OBJECT_FORMAT"></span>**Format MOF (MOF)**
</dt> <dd>

Langage basé sur le langage IDL (Interface Definition Language) qui décrit les informations de gestion.

</dd> <dt>

<span id="wmi.gloss_management_information_base"></span><span id="WMI.GLOSS_MANAGEMENT_INFORMATION_BASE"></span>**base d’informations de gestion (MIB)**
</dt> <dd>

Base de données qui stocke une collection de représentations d’objets réseau.

</dd> <dt>

<span id="wmi.gloss_metamodel"></span><span id="WMI.GLOSS_METAMODEL"></span>**MetaModel**
</dt> <dd>

Composant CIM qui décrit les entités et les relations qui représentent des *objets managés*. Par exemple, les classes, les instances et les associations sont incluses dans le modèle de modèle.

</dd> <dt>

<span id="wmi.gloss_metaschema"></span><span id="WMI.GLOSS_METASCHEMA"></span>**schéma**
</dt> <dd>

[*Schéma*](gloss-s.md) du *modèle*.

</dd> <dt>

<span id="wmi.gloss_method_provider"></span><span id="WMI.GLOSS_METHOD_PROVIDER"></span>**fournisseur de méthode**
</dt> <dd>

Fournisseur qui active l’accès WMI aux méthodes d’une classe. Consultez également [écriture d’un fournisseur de méthodes](writing-a-method-provider.md).

</dd> <dt>

<span id="wmi.gloss_mib"></span><span id="WMI.GLOSS_MIB"></span>**MIB**
</dt> <dd>

Consultez la *base de données MIB (Management Information base)*.

</dd> <dt>

<span id="wmi.gloss_mof"></span><span id="WMI.GLOSS_MOF"></span>**GÉRÉ**
</dt> <dd>

Voir *format MOF (MOF)*.

</dd> <dt>

<span id="wmi.gloss_mof_file"></span><span id="WMI.GLOSS_MOF_FILE"></span>**Fichier MOF**
</dt> <dd>

Fichier texte qui contient des définitions de classes et d’instances spécifiées à l’aide du langage *format MOF (MOF)* .

</dd> </dl>
