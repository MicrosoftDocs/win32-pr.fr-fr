# <a name="swbemobjectpathisclass-property"></a>SWbemObjectPath. IsClass, propriété

La propriété **IsClass** de l’objet [**SWbemObjectPath**](swbemobjectpath.md) est une valeur booléenne qui indique si ce chemin d’accès représente une classe. Cette propriété est en lecture seule.

Pour une explication de cette syntaxe, consultez [conventions de document pour l’API de script](document-conventions-for-the-scripting-api.md).

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```VB
SWbemObjectPath.IsClass As Boolean
```
