# <a name="swbemqualifiername-property"></a>Propriété SWbemQualifier.Name

La propriété **Name** de l’objet [**SWbemQualifier**](swbemqualifier.md) définit le nom de ce qualificateur. Cette propriété est en lecture seule.

Pour une explication de cette syntaxe, consultez [conventions de document pour l’API de script](document-conventions-for-the-scripting-api.md).

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```VB
SWbemQualifier.Name As String
```
