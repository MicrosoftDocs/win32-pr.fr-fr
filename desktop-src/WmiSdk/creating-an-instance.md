# <a name="creating-an-instance"></a>Création d’une instance

Une instance de est un cas d’une classe qui décrit un objet particulier de votre entreprise. Par exemple, si vous avez deux lecteurs de CD-ROM dans une plateforme de bureau, vous pouvez utiliser deux instances de la classe [**Win32 \_ CDROMDrive**](/windows/desktop/CIMWin32Prov/win32-cdromdrive) pour les décrire. Étant donné qu’une instance représente une véritable propriété de votre entreprise, il est très probable que vous créez plus d’instances que de classes. Chaque classe à partir de laquelle une instance est créée a une ou plusieurs propriétés de clé. Vous ne pouvez pas créer une instance avec plus de 256 clés.

Pour plus d’informations sur la création d’instances, consultez les rubriques suivantes :

-   [Création et déclaration d’une instance à l’aide de C++](creating-and-declaring-an-instance-using-c-.md)
-   [Création d’une instance à l’aide de MOF](creating-an-instance-using-mof.md)
-   [Création d’un objet à l’aide de VBScript](creating-an-object-using-vbscript.md)
