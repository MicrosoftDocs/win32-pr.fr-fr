# <a name="swbemprivilegeidentifier-property"></a>SWbemPrivilege. identifier, propriété

La propriété d' **identificateur** d’un objet [**SWbemPrivilege**](swbemprivilege.md) est un entier [WbemPrivilegeEnum](/windows/desktop/api/Wbemdisp/ne-wbemdisp-wbemprivilegeenum) qui représente le privilège qui est défini ou récupéré. Cette propriété est en lecture seule.

Pour une explication de cette syntaxe, consultez [conventions de document pour l’API de script](document-conventions-for-the-scripting-api.md).

Cette propriété est en lecture/écriture.

## <a name="syntax"></a>Syntaxe


```VB
SWbemPrivilege.Identifier As Integer
```
