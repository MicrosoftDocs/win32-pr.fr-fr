# <a name="retrieving-wmi-class-or-instance-data"></a>Récupération des données de classe ou d’instance WMI

Les principaux objets de données dans WMI sont les classes et les instances. Le tableau suivant répertorie les différentes techniques disponibles pour récupérer des données d’une instance ou d’une classe WMI.



| Rubrique                                                                   | Description                                                                       |
|-------------------------------------------------------------------------|-----------------------------------------------------------------------------------|
| [Récupération d’une classe WMI](retrieving-a-class.md)                        | Récupération d’informations sur les paramètres généraux d’un objet.                    |
| [Récupération d’une instance WMI](retrieving-an-instance.md)                 | Récupération d’informations décrivant un objet unique et individuel sur votre entreprise. |
| [Récupération d’une partie d’une instance WMI](retrieving-part-of-an-instance.md) | Récupération d’une ou plusieurs parties d’une instance, mais pas de la totalité de l’instance.         |
| [Accès à un qualificateur WMI](accessing-a-qualifier.md)                  | Accès à plus d’informations contenues dans un qualificateur.                              |



 

Pour plus d’informations, consultez [manipulation d’informations sur les classes et les instances](manipulating-class-and-instance-information.md).
