# <a name="msft-classes"></a>Classes MSFT

les classes MSFT sont spécifiques à l’implémentation Windows du [*Common Information Model*](gloss-c.md) (CIM).

Les objets peuvent être divisés en plusieurs catégories principales :

-   [Classes du fournisseur de stratégie](policy-provider-classes.md)
-   [Classes de résolution des problèmes WMI](wmi-troubleshooting-classes.md)

Les classes MSFT suivantes sont définies dans System. mof, mais ne sont pas implémentées par WMI.

-   \_WMICOREEVENT msft
-   \_WMICORELOGOFFEVENT msft
-   \_WMICORELOGONEVENT msft
-   \_WMICOREOBJECT msft
-   \_WMICORESTATUS msft
-   \_WMICOREUSER msft
-   \_WMITASKFAILURE msft
