# <a name="swbemqualifierisamended-property"></a>SWbemQualifier. IsAmended, propriété

La propriété **IsAmended** de l’objet [**SWbemQualifier**](swbemqualifier.md) est une valeur booléenne qui indique si ce qualificateur a été localisé à l’aide d’une opération de fusion.

Pour une explication de cette syntaxe, consultez [conventions de document pour l’API de script](document-conventions-for-the-scripting-api.md).

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```VB
SWbemQualifier.IsAmended As Boolean
```
