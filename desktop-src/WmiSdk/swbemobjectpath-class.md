# <a name="swbemobjectpathclass-property"></a>SWbemObjectPath. Class, propriété

La propriété de **classe** de l’objet [**SWbemObjectPath**](swbemobjectpath.md) est le nom de la classe qui fait partie du chemin d’accès de l’objet.

Pour une explication de cette syntaxe, consultez [conventions de document pour l’API de script](document-conventions-for-the-scripting-api.md).

Cette propriété est en lecture/écriture.

## <a name="syntax"></a>Syntaxe


```VB
SWbemObjectPath.Class As String
```
