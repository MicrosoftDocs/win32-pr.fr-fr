# <a name="datetime"></a>DATETIME

Le type de données DATETIME est une chaîne VT BSTR de longueur fixe \_ qui représente une date et une heure ou un intervalle spécifiques. Pour plus d’informations sur DATETIME en tant que date, consultez [format de date et d’heure](date-and-time-format.md). Pour plus d’informations sur DATETIME comme intervalle, consultez [format d’intervalle](interval-format.md).
