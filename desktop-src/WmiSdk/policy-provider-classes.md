# <a name="policy-provider-classes"></a>Classes du fournisseur de stratégie

Le fournisseur de stratégie WMI fournit des extensions à la stratégie de groupe via des classes, ce qui permet d’affiner l’application de la stratégie. La stratégie est définie à l’aide du Langage de requêtes WMI (WQL) (WQL) et réside dans la Active Directory pour un déploiement facile.

Le tableau suivant répertorie les classes du fournisseur de stratégie WMI.



| Classe                                               | Description                                                       |
|-----------------------------------------------------|-------------------------------------------------------------------|
| [**\_Fournisseurs msft**](/previous-versions/windows/desktop/wmisystemprov/msft-providers) | Expose la configuration relative aux instances de fournisseur.             |
| [**\_Règle msft**](/previous-versions/windows/desktop/policmanprov/msft-rule)                | Définit une seule règle au sein d’une étendue de gestion (SOM).         |
| [**\_SOMFILTER msft**](/previous-versions/windows/desktop/policmanprov/msft-somfilter)      | Fournit la liste des règles qui sont évaluées sur un ordinateur cible. |
