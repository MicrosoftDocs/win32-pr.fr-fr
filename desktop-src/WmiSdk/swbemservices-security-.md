# <a name="swbemservicessecurity_-property"></a>SWbemServices. Security, \_ propriété

La **propriété \_ Security** de l’objet [**SWbemServices**](swbemservices.md) est utilisée pour obtenir ou définir les paramètres de sécurité d’un objet **SWbemServices** . Cette propriété est un objet [**SWbemSecurity**](swbemsecurity.md) .

Pour une explication de cette syntaxe, consultez [conventions de document pour l’API de script](document-conventions-for-the-scripting-api.md).

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```VB
SWbemServices.Security_ As Object
```
