# <a name="swbemnamedvaluesetcount-property"></a>SWbemNamedValueSet. Count (propriété)

Utilisez la propriété **Count** de l’objet [**SWbemNamedValueSet**](swbemnamedvalueset.md) pour déterminer le nombre d’éléments présents dans la collection. Cette propriété est en lecture seule.

Pour une explication de cette syntaxe, consultez [conventions de document pour l’API de script](document-conventions-for-the-scripting-api.md).

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```VB
SWbemNamedValueSet.Count As Integer
```
