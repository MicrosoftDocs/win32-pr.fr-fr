# <a name="swbemnamedvaluevalue-property"></a>SWbemNamedValue. Value (propriété)

La propriété **value** de l’objet [**SWbemNamedValue**](swbemnamedvalue.md) retourne la valeur de type variant d’un élément **SWbemNamedValue** . Il s’agit de la propriété par défaut pour les objets **SWbemNamedValue** . Les modifications apportées à la propriété value sont répercutées automatiquement dans la collection [**SWbemNamedValueSet**](swbemnamedvalueset.md) à laquelle appartient l’objet **SWbemNamedValue** .

Pour une explication de cette syntaxe, consultez [conventions de document pour l’API de script](document-conventions-for-the-scripting-api.md).

Cette propriété est en lecture/écriture.

## <a name="syntax"></a>Syntaxe


```VB
SWbemNamedValue.Value As Variant
```
