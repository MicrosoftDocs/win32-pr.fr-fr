# <a name="application-inventory-classes"></a>Classes d’inventaire des applications

## <a name="purpose"></a>Objectif

les classes WMI pour l’inventaire des applications permettent la découverte des applications Win32 installées et des applications de Windows store sur un système Windows.

## <a name="in-this-section"></a>Contenu de cette section



| Rubrique                                                                                  | Description                                                                                                                                                                                                                                                    |
|----------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**\_InstalledWin32Program Win32**](win32-installedwin32program.md)<br/>         | La classe [**Win32 \_ InstalledWin32Program**](win32-installedwin32program.md) représente une application Win32 installée.<br/>                                                                                                                            |
| [**\_InstalledStoreProgram Win32**](win32-installedstoreprogram.md)<br/>         | la [**classe \_ InstalledStoreProgram Win32**](win32-installedstoreprogram.md) représente une application Windows store installée.<br/>                                                                                                                    |
| [**\_InstalledProgramFramework Win32**](win32-installedprogramframework.md)<br/> | La [**classe \_ InstalledProgramFramework Win32**](win32-installedprogramframework.md) représente un Framework technologique sur lequel une instance [**\_ InstalledWin32Program Win32**](win32-installedwin32program.md) est compilée ou utilisée au moment de l’exécution.<br/> |



 

## <a name="developer-audience"></a>Développeurs concernés

les classes WMI pour l’inventaire des applications sont conçues pour être utilisées par les développeurs de solutions de gestion Enterprise et d’outils d’administration locaux ou distants qui doivent générer un inventaire des applications sur un système.
