# <a name="mof-data-types"></a>Types de données MOF

Le langage MOF (format MOF) prend en charge de nombreux types de données qui utilisent pour assigner des valeurs à des propriétés, à des qualificateurs et à des paramètres de méthode.

Le langage MOF prend en charge les types de données et les catégories de types de données suivants :

-   [BitMap et BitValue](bitmap-and-bitvalues.md)
-   [boolean](boolean.md)
-   [Date/heure](date-and-time-format.md)
-   [Tableaux MOF](mof-arrays.md)
-   [Nombreux](numbers.md)
-   [DESSIN](object.md)
-   [Informations de référence](references.md)
-   [Chaînes](strings.md)
-   [Mappage des valeurs](value-map.md)
-   [VOID](void.md)

Pour plus d’informations sur les commandes, consultez [commandes de préprocesseur](preprocessor-commands.md).
