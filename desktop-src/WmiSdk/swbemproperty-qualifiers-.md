# <a name="swbempropertyqualifiers_-property"></a>SWbemProperty. Qualifiers, \_ propriété

La propriété **Qualifiers \_** de l’objet [**SWbemProperty**](swbemproperty.md) retourne un objet [**SWbemQualifierSet**](swbemqualifierset.md) qui est une collection de qualificateurs pour cette propriété WMI. Cette propriété est en lecture seule.

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```VB
SWbemProperty.Qualifiers_ As Object
```
