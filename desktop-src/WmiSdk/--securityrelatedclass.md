# <a name="__securityrelatedclass-class"></a>\_\_SecurityRelatedClass, classe

La classe système abstraite **\_ \_ SecurityRelatedClass** sert de classe parente pour tous les types de classes de sécurité. Aucune instance de cette classe n’est jamais créée, et toutes les classes de sécurité doivent dériver de cette classe de base.

La syntaxe suivante est simplifiée à partir de code au format MOF (Managed Object Format) et inclut toutes les propriétés héritées. Les propriétés sont répertoriées par ordre alphabétique, et non par ordre MOF.

## <a name="syntax"></a>Syntaxe

``` syntax
[abstract]
class __SecurityRelatedClass
{
};
```

## <a name="members"></a>Membres

La classe **\_ \_ SecurityRelatedClass** ne définit aucun membre.
