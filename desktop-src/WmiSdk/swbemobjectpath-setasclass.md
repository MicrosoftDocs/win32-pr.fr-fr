# <a name="swbemobjectpathsetasclass-property"></a>SWbemObjectPath. SetAsClass, propriété

La méthode **SetAsClass** de l’objet [**SWbemObjectPath**](swbemobjectpath.md) force le chemin d’accès à une classe WMI.

Cette propriété est en écriture seule.

## <a name="syntax"></a>Syntaxe


```VB
SWbemObjectPath.SetAsClass
```



## <a name="property-value"></a>Valeur de la propriété

## <a name="error-codes"></a>Codes d’erreur

Une fois que vous avez obtenu ou défini la propriété **SetAsClass** , l’objet **Err** peut contenir le code d’erreur ci-dessous.

<dl> <dt>

**wbemErrFailed** -2147749889 (0x80041001)
</dt> <dd>

Erreur non spécifiée.

</dd> </dl>
