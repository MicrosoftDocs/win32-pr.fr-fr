# <a name="__indicationrelated-class"></a>\_\_IndicationRelated, classe

La classe système abstraite **\_ \_ IndicationRelated** sert de classe parente pour toutes les classes liées aux événements. La classe ne contient aucune propriété.

La syntaxe suivante est simplifiée à partir de code au format MOF (Managed Object Format) et inclut toutes les propriétés héritées. Les propriétés sont répertoriées par ordre alphabétique, et non par ordre MOF.

## <a name="syntax"></a>Syntaxe

``` syntax
[abstract]
class __IndicationRelated : __SystemClass
{
};
```

## <a name="members"></a>Membres

La classe **\_ \_ IndicationRelated** ne définit aucun membre.

## <a name="remarks"></a>Notes

La classe **\_ \_ IndicationRelated** est dérivée de [**\_ \_ SystemClass**](--systemclass.md), qui n’a pas de propriétés.
