# <a name="swbemrefreshableitemrefresher-property"></a>SWbemRefreshableItem. actualiser, propriété

La propriété **SWbemRefreshableItem. actualiser** représente l’objet [**SWbemRefresher**](swbemrefresher.md) parent qui contient l’objet [**SWbemRefreshableItem**](swbemrefreshableitem.md) .

Pour une explication de cette syntaxe, consultez [conventions de document pour l’API de script](document-conventions-for-the-scripting-api.md).

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```VB
SWbemRefreshableItem.Refresher As Object
```
