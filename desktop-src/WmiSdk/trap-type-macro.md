# <a name="trap-type-macro"></a>TYPE d’interruption (macro)

Pour l’uniformité, chaque instance de la macro TRAP-TYPE est d’abord mappée à une instance de la macro de [type notification](notification-type-macro.md) à l’aide du mappage défini dans le [document RFC 1908](https://www.ietf.org/rfc/rfc1908.txt).

> [!Note]  
> Pour plus d’informations sur l’installation du fournisseur, consultez [configuration de l’environnement SNMP WMI](setting-up-the-wmi-snmp-environment.md).
