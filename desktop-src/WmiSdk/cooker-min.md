# <a name="cooker_min"></a>minimum de cuiseur \_

La \_ formule de type de compteur minimum de cuiseur identifie la plus petite valeur d’un ensemble d’observations d’une propriété dans une instance de [**\_ PerfRawData Win32**](/windows/desktop/CIMWin32Prov/win32-perfrawdata) . Cette valeur minimale calculée est placée dans une propriété dans une instance de [**\_ PerfFormattedData Win32**](/windows/desktop/CIMWin32Prov/win32-perfformatteddata) . Ce type de compteur est défini uniquement dans WMI et n’est pas disponible pour les technologies d’analyse des performances, telles que les [compteurs de performance Version 6,0](/windows/desktop/PerfCtrs/performance-counters-portal).

Pour plus d’informations sur la formule de type de compteur, consultez [types de compteurs](/previous-versions/windows/it-pro/windows-server-2003/cc785636(v=ws.10)).

Pour plus d’informations sur les fournisseurs et les scripts WMI à hautes performances, consultez [types de compteurs de performance WMI](wmi-performance-counter-types.md).
