# <a name="swbempropertyisarray-property"></a>SWbemProperty. IsArray, propriété

La propriété **IsArray** de l’objet [**SWbemProperty**](swbemproperty.md) est une valeur booléenne qui peut être utilisée pour déterminer si cette propriété a un type tableau. Cette propriété est en lecture seule.

Pour une explication de cette syntaxe, consultez [conventions de document pour l’API de script](document-conventions-for-the-scripting-api.md).

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```VB
SWbemProperty.IsArray As Boolean
```
