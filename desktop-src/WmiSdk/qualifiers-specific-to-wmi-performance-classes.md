# <a name="qualifiers-specific-to-wmi-performance-classes"></a>Qualificateurs spécifiques aux classes de performance WMI

Les classes de compteur de performances brutes, dérivées de [**Win32 \_ PerfRawData**](/windows/desktop/CIMWin32Prov/win32-perfrawdata), contiennent des valeurs non calculées qui servent d’entrée pour les classes [**Win32 \_ PerfFormattedData**](/windows/desktop/CIMWin32Prov/win32-perfformatteddata) , ainsi que des calculs du moniteur système.

Les qualificateurs de compteur de performance spécifient des informations sur l’objet de performance et les compteurs de cet objet. Les objets de performance sont mappés aux [classes de compteur de performance](/windows/desktop/CIMWin32Prov/performance-counter-classes) WMI et les compteurs de performance sont représentés par les propriétés de la classe WMI.

-   [Qualificateurs de classe pour les classes de compteurs de performances](class-qualifiers-for-performance-counter-classes.md)
-   [Qualificateurs de propriété pour les classes de compteur de performance mises en forme](property-qualifiers-for-performance-counter-classes.md)
