# <a name="swbempropertyvalue-property"></a>SWbemProperty. Value (propriété)

La propriété **value** de l’objet [**SWbemProperty**](swbemproperty.md) définit la valeur de type variant de la propriété WMI. Il s’agit de la propriété Automation par défaut de cet objet.

Pour une explication de cette syntaxe, consultez [conventions de document pour l’API de script](document-conventions-for-the-scripting-api.md).

Cette propriété est en lecture/écriture.

## <a name="syntax"></a>Syntaxe


```VB
SWbemProperty.Value As Variant
```
