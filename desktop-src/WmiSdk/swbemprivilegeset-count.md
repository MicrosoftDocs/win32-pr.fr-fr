# <a name="swbemprivilegesetcount-property"></a>SWbemPrivilegeSet. Count (propriété)

La propriété **Count** de l’objet [**SWbemPrivilege**](swbemprivilege.md) contient le nombre d’éléments d’une collection [**SWbemPrivilegeSet**](swbemprivilegeset.md) . Cette propriété est en lecture seule.

Pour une explication de cette syntaxe, consultez [conventions de document pour l’API de script](document-conventions-for-the-scripting-api.md).

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```VB
SWbemPrivilegeSet.Count As Integer
```
