# <a name="swbemqualifiervalue-property"></a>SWbemQualifier. Value (propriété)

La propriété **value** de l’objet [**SWbemQualifier**](swbemqualifier.md) contient la valeur variant de ce qualificateur. Il s’agit de la propriété par défaut de cet objet.

Pour une explication de cette syntaxe, consultez [conventions de document pour l’API de script](document-conventions-for-the-scripting-api.md).

Cette propriété est en lecture/écriture.

## <a name="syntax"></a>Syntaxe


```VB
SWbemQualifier.Value As Variant
```
