# <a name="swbemnamedvalue-object"></a>Objet SWbemNamedValue

L’objet **SWbemNamedValue** représente une valeur nommée unique qui appartient à l’objet de collection [**SWbemNamedValueSet**](swbemnamedvalueset.md) . Cet objet ne peut pas être créé par l’appel VBScript [**CreateObject**](creating-an-object-using-vbscript.md) .

## <a name="members"></a>Membres

L’objet **SWbemNamedValue** possède les types de membres suivants :

-   [Propriétés](#properties)

### <a name="properties"></a>Propriétés

L’objet **SWbemNamedValue** a ces propriétés.



| Propriété                                          | Type d’accès           | Description                                                                                    |
|:--------------------------------------------------|:----------------------|:-----------------------------------------------------------------------------------------------|
| [**Nom**](swbemnamedvalue-name.md)<br/>   | Lecture seule<br/>  | Nom de l’élément **SWbemNamedValue** .<br/>                                               |
| [**Valeur**](swbemnamedvalue-value.md)<br/> | Lecture/écriture<br/> | Valeur de l’élément **SWbemNamedValue** . Il s’agit de la propriété par défaut de cet objet.<br/> |
