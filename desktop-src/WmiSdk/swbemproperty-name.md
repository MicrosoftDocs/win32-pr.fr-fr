# <a name="swbempropertyname-property"></a>Propriété SWbemProperty.Name

La propriété **Name** de l’objet [**SWbemProperty**](swbemproperty.md) retourne le nom de cette propriété WMI. Cette propriété est en lecture seule.

Pour une explication de cette syntaxe, consultez [conventions de document pour l’API de script](document-conventions-for-the-scripting-api.md).

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```VB
SWbemProperty.Name As String
```
