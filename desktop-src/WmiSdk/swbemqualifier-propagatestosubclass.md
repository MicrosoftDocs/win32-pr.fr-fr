# <a name="swbemqualifierpropagatestosubclass-property"></a>SWbemQualifier. PropagatesToSubClass, propriété

La propriété **PropagatesToSubClass** de l’objet [**SWbemQualifier**](swbemqualifier.md) est une valeur booléenne qui indique si ce qualificateur peut être propagé à une sous-classe.

Pour une explication de cette syntaxe, consultez [conventions de document pour l’API de script](document-conventions-for-the-scripting-api.md).

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```VB
SWbemQualifier.PropagatesToSubClass
```
