# <a name="__systemclass-class"></a>\_\_SystemClass, classe

La classe système abstraite **\_ \_ SystemClass** est la classe de base à partir de laquelle dérivent la plupart des classes système. Les instances de cette classe ne peuvent pas être créées.

La syntaxe suivante est simplifiée à partir de code au format MOF (Managed Object Format) et inclut toutes les propriétés héritées. Les propriétés sont répertoriées par ordre alphabétique, et non par ordre MOF.

## <a name="syntax"></a>Syntaxe

``` syntax
[abstract]
class __SystemClass
{
};
```

## <a name="members"></a>Membres

La classe **\_ \_ SystemClass** ne définit aucun membre.
