# <a name="modifying-an-instance-property"></a>Modification d’une propriété d’instance

Une fois que vous avez créé ou trouvé une instance d’une classe, vous souhaiterez probablement modifier l’instance. Par exemple, vous souhaiterez peut-être modifier le nom du système stocké dans l’instance locale de la classe [**Win32 \_ ComputerSystem**](/windows/desktop/CIMWin32Prov/win32-computersystem) . La méthode la plus courante pour mettre à jour une instance consiste à mettre à jour l’instance entière. Pour plus d’informations, consultez [mise à jour d’une instance entière](updating-an-entire-instance.md). La deuxième option consiste à mettre à jour uniquement une partie de l’instance. Pour plus d’informations, consultez [mise à jour d’une partie d’une instance](updating-part-of-an-instance.md).

Pour plus d’informations sur l’utilisation des classes, consultez [manipulation des informations relatives](manipulating-class-and-instance-information.md)aux classes et aux instances.
