# <a name="swbemqualifierislocal-property"></a>SWbemQualifier. IsLocal, propriété

La propriété **IsLocal** de l’objet [**SWbemQualifier**](swbemqualifier.md) est une valeur booléenne qui indique si ce qualificateur est local. Cette propriété est en lecture seule.

Pour une explication de cette syntaxe, consultez [conventions de document pour l’API de script](document-conventions-for-the-scripting-api.md).

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```VB
SWbemQualifier.IsLocal As Boolean
```
