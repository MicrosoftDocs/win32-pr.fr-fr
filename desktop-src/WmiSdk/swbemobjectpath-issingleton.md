# <a name="swbemobjectpathissingleton-property"></a>SWbemObjectPath. IsSingleton, propriété

La propriété **IsSingleton** de l’objet [**SWbemObjectPath**](swbemobjectpath.md) est une valeur booléenne qui indique si ce chemin représente une instance singleton. Une instance singleton est une instance d’une classe qui ne peut jamais avoir plus d’une instance. Cette propriété est en lecture seule.

Pour une explication de cette syntaxe, consultez [conventions de document pour l’API de script](document-conventions-for-the-scripting-api.md).

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```VB
SWbemObjectPath.IsSingleton As Boolean
```
