# <a name="swbemobjectpathserver-property"></a>SWbemObjectPath. Server, propriété

La propriété **Server** de l’objet [**SWbemObjectPath**](swbemobjectpath.md) contient le nom du serveur qui fait partie du chemin d’accès de l’objet.

Pour une explication de cette syntaxe, consultez [conventions de document pour l’API de script](document-conventions-for-the-scripting-api.md).

Cette propriété est en lecture/écriture.

## <a name="syntax"></a>Syntaxe


```VB
SWbemObjectPath.Server As String
```
