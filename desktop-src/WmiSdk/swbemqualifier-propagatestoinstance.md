# <a name="swbemqualifierpropagatestoinstance-property"></a>SWbemQualifier. PropagatesToInstance, propriété

La propriété **PropagatesToInstance** de l’objet [**SWbemQualifier**](swbemqualifier.md) est une valeur booléenne qui indique si ce qualificateur peut être propagé à une instance.

Pour une explication de cette syntaxe, consultez [conventions de document pour l’API de script](document-conventions-for-the-scripting-api.md).

Cette propriété est en lecture/écriture.

## <a name="syntax"></a>Syntaxe


```VB
SWbemQualifier.PropagatesToInstance As Boolean
```
