# <a name="swbemrefreshercount-property"></a>SWbemRefresher. Count (propriété)

La propriété **Count** de l’objet [**SWbemRefresher**](swbemrefresher.md) contient le nombre d’éléments d’une collection [**SWbemRefresher**](swbemrefresher.md) . Cette propriété est en lecture seule.

Pour une explication de cette syntaxe, consultez [conventions de document pour l’API de script](document-conventions-for-the-scripting-api.md).

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```VB
SWbemRefresher.Count As Integer
```
