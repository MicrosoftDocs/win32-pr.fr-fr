# <a name="preprocessor-commands"></a>Commandes de préprocesseur

Une commande de préprocesseur est une ligne de code placée à l’intérieur d’un fichier MOF qui contrôle la façon dont le compilateur MOF compile le code. Deux commandes de préprocesseur sont utilisées pour la compilation des fichiers MOF :

-   [\#inclusion](-include.md)
-   [\#Bali](-pragma.md)
