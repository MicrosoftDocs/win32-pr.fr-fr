# <a name="wmi-c-application-examples"></a>Exemples d’applications WMI C++

Les exemples d’applications WMI de cette section sont écrits en C++. ils illustrent une série de tâches qui peuvent être effectuées à l’aide de composants WMI et offrent une alternative à l’utilisation de scripts de Visual Basic. Chaque application est divisée en une série d’étapes de manière similaire afin que les sections de code des différents exemples puissent être facilement combinées pour former des applications personnalisées.

Le tableau suivant répertorie les exemples C++ de cette section.



| Exemple                                                                                                                                  | Description                                                                                                                                                |
|------------------------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Exemple : obtention de données WMI à partir de l’ordinateur local](example--getting-wmi-data-from-the-local-computer.md)                               | Cet exemple se connecte à l’espace de noms WMI sur l’ordinateur local et récupère les données d’une requête sur l’ordinateur local. Les données sont collectées semisynchronously. |
| [Exemple : obtention de données WMI à partir de l’ordinateur local de façon asynchrone](example--getting-wmi-data-from-the-local-computer-asynchronously.md) | Cet exemple se connecte à l’espace de noms WMI sur l’ordinateur local et récupère les données d’une requête sur l’ordinateur local. Les données sont collectées de manière asynchrone.    |
| [Exemple : obtention de données WMI à partir d’un ordinateur distant](example--getting-wmi-data-from-a-remote-computer.md)                                 | Cet exemple se connecte à l’espace de noms WMI sur un ordinateur distant et récupère les données à partir d’une requête sur l’ordinateur distant. Les données sont collectées semisynchronously. |
| [Exemple : appel d’une méthode de fournisseur](example--calling-a-provider-method.md)                                                             | Cet exemple se connecte à l’espace de noms WMI sur l’ordinateur local et appelle une méthode dans WMI. La méthode est exécutée de façon synchrone.                          |
| [Exemple : réception de notifications d’événements via WMI](example--receiving-event-notifications-through-wmi-.md)                            | Cet exemple se connecte à l’espace de noms WMI sur l’ordinateur local et reçoit un événement de l’ordinateur local. L’événement est reçu semisynchronously.   |
