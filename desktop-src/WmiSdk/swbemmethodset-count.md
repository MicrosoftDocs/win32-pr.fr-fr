# <a name="swbemmethodsetcount-property"></a>SWbemMethodSet. Count (propriété)

La propriété **Count** de l’objet [**SWbemMethodSet**](swbemmethodset.md) définit le nombre d’éléments dans la collection. Cette propriété est en lecture seule.

Pour une explication de cette syntaxe, consultez [conventions de document pour l’API de script](document-conventions-for-the-scripting-api.md).

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```VB
SWbemMethodSet.Count As Integer
```
