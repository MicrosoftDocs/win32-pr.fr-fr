# <a name="swbemmethodqualifiers_-property"></a>SWbemMethod. Qualifiers, \_ propriété

La propriété **Qualifiers \_** de l’objet [**SWbemMethod**](swbemmethod.md) retourne un objet [**SWbemQualifierSet**](swbemqualifierset.md) qui est la collection de qualificateurs pour cette méthode. Cette propriété est en lecture seule.

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```VB
SWbemMethod.Qualifiers_ As Object
```
