# <a name="the-performance-monitoring-provider"></a>Fournisseur d’analyse des performances

Les données des bibliothèques de performances sont disponibles en tant que [classes de compteurs de performances](/windows/desktop/CIMWin32Prov/performance-counter-classes) WMI via le fournisseur [wmiperfclass](wmiperfclass-provider.md) et le [fournisseur WMIPerfInst](wmiperfinst-provider.md).
