# <a name="swbemqualifierisoverridable-property"></a>SWbemQualifier. IsOverridable, propriété

La propriété **IsOverridable** de l’objet [**SWbemQualifier**](swbemqualifier.md) est une valeur booléenne qui indique si ce qualificateur peut être substitué lorsqu’il est propagé.

Pour une explication de cette syntaxe, consultez [conventions de document pour l’API de script](document-conventions-for-the-scripting-api.md).

Cette propriété est en lecture/écriture.

## <a name="syntax"></a>Syntaxe


```VB
SWbemQualifier.IsOverridable As Boolean
```
