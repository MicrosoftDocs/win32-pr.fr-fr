# <a name="swbemobjectpathpath-property"></a>SWbemObjectPath. Path, propriété

La propriété **path** de l’objet [**SWbemObjectPath**](swbemobjectpath.md) contient le chemin d’accès absolu. Il est identique à la propriété [ \_ \_ path](wmi-system-properties.md) de l’API com. Il s’agit de la propriété par défaut de cet objet.

Pour une explication de cette syntaxe, consultez [conventions de document pour l’API de script](document-conventions-for-the-scripting-api.md).

Cette propriété est en lecture/écriture.

## <a name="syntax"></a>Syntaxe


```VB
SWbemObjectPath.Path As String
```
