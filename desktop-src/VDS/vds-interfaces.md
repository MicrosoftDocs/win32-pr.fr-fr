# <a name="vds-interfaces"></a>Interfaces VDS

\[À compter de Windows 8 et de Windows Server 2012, l’interface com du [service de disque virtuel](virtual-disk-service-portal.md) est remplacée par l' [API de gestion de stockage Windows](/previous-versions/windows/desktop/stormgmt/windows-storage-management-api-portal).\]

Les objets VDS (Virtual Disk Service) fournissent des interfaces qui exposent des méthodes d’interrogation, de configuration et de maintenance des dispositifs de stockage.



| Interface                                                            | Description                                                                                                                                                                                           |
|----------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IEnumVdsObject**](/windows/desktop/api/Vds/nn-vds-ienumvdsobject)                             | Énumère un ensemble d’objets VDS d’un type donné.                                                                                                                                              |
| [**IVdsAdmin**](/windows/desktop/api/VdsHwPrv/nn-vdshwprv-ivdsadmin)                                       | Inscrit un fournisseur auprès de VDS.                                                                                                                                                                        |
| [**IVdsAdvancedDisk**](/windows/desktop/api/Vds/nn-vds-ivdsadvanceddisk)                         | Crée et supprime des partitions, et modifie les attributs de partition.                                                                                                                                    |
| [**IVdsAdvancedDisk2**](/windows/desktop/api/Vds/nn-vds-ivdsadvanceddisk2)                       | Fournit une méthode pour modifier les types de partition.                                                                                                                                                          |
| [**IVdsAdviseSink**](/windows/desktop/api/Vds/nn-vds-ivdsadvisesink)                             | Reçoit des notifications VDS.                                                                                                                                                                           |
| [**IVdsAsync**](/windows/desktop/api/Vds/nn-vds-ivdsasync)                                       | Gère les opérations asynchrones.                                                                                                                                                                      |
| [**IVdsController**](/windows/desktop/api/Vds/nn-vds-ivdscontroller)                             | Expose des méthodes pour effectuer des opérations de requête et de configuration sur un contrôleur.                                                                                                                    |
| [**IVdsControllerControllerPort**](/windows/desktop/api/Vds/nn-vds-ivdscontrollercontrollerport) | Fournit une méthode pour énumérer les ports de contrôleur pour une classe qui implémente l’interface [**IVdsController**](/windows/desktop/api/Vds/nn-vds-ivdscontroller) .                                                                      |
| [**IVdsControllerPort**](/windows/desktop/api/Vds/nn-vds-ivdscontrollerport)                     | Fournit des méthodes pour effectuer des opérations de requête et de configuration sur un port de contrôleur.                                                                                                              |
| [**IVdsCreatePartitionEx**](/windows/desktop/api/Vds/nn-vds-ivdscreatepartitionex)               | Crée une partition sur un disque de base.                                                                                                                                                                  |
| [**IVdsDisk**](/windows/desktop/api/Vds/nn-vds-ivdsdisk)                                         | Interroge et configure les disques de base et dynamiques.                                                                                                                                                       |
| [**IVdsDisk2**](/windows/desktop/api/Vds/nn-vds-ivdsdisk2)                                       | Fournit une méthode pour définir le mode SAN d’un disque en mode hors connexion ou en ligne.                                                                                                                                 |
| [**IVdsDiskPartitionMF**](/windows/desktop/api/Vds/nn-vds-ivdsdiskpartitionmf)                   | Fournit des méthodes pour effectuer des opérations de gestion du système de fichiers sur les partitions.                                                                                                                          |
| [**IVdsDiskPartitionMF2**](/windows/desktop/api/Vds/nn-vds-ivdsdiskpartitionmf2)                 | Fournit une méthode pour mettre en forme une partition avec des options de mise en forme supplémentaires.                                                                                                                           |
| [**IVdsDrive**](/windows/desktop/api/Vds/nn-vds-ivdsdrive)                                       | Fournit des méthodes pour effectuer des opérations de requête et de configuration sur un lecteur.                                                                                                                        |
| [**IVdsDrive2**](/windows/desktop/api/Vds/nn-vds-ivdsdrive2)                                     | Fournit une méthode pour interroger les propriétés d’un lecteur.                                                                                                                                             |
| [**IVdsHbaPort**](/windows/desktop/api/Vds/nn-vds-ivdshbaport)                                   | Fournit des méthodes pour interroger et interagir avec les ports HBA sur le système local.                                                                                                                            |
| [**IVdsHwProvider**](/windows/desktop/api/Vds/nn-vds-ivdshwprovider)                             | Fournit des méthodes pour effectuer des opérations de requête, de réénumération et d’actualisation sur un fournisseur de matériel.                                                                                                  |
| [**IVdsHwProviderPrivate**](/windows/desktop/api/VdsHwPrv/nn-vdshwprv-ivdshwproviderprivate)               | Fournit une méthode qui permet à VDS de déterminer si le fournisseur de matériel possède un numéro d’unité logique spécifié.                                                                                                   |
| [**IVdsHwProviderPrivateMpio**](/windows/desktop/api/VdsHwPrv/nn-vdshwprv-ivdshwproviderprivatempio)       | Fournit une méthode qui définit l’état des chemins d’accès provenant d’un port d’adaptateur HBA particulier au fournisseur.                                                                                               |
| [**IVdsHwProviderStoragePools**](/windows/desktop/api/Vds/nn-vds-ivdshwproviderstoragepools)     | Fournit des méthodes pour créer des numéros d’unités logiques dans un [pool de stockage](storage-pool-object.md) et énumérer les pools de stockage gérés par un fournisseur de matériel.                                                          |
| [**IVdsHwProviderType**](/windows/desktop/api/Vds/nn-vds-ivdshwprovidertype)                     | Fournit une méthode pour récupérer le type de fournisseur de matériel.                                                                                                                                          |
| [**IVdsHwProviderType2**](/windows/desktop/api/Vds/nn-vds-ivdshwprovidertype2)                   | Cette interface n’est pas implémentée. Utilisez [**IVdsHwProviderType**](/windows/desktop/api/Vds/nn-vds-ivdshwprovidertype) à la place.                                                                                                      |
| [**IVdsIscsiInitiatorAdapter**](/windows/desktop/api/Vds/nn-vds-ivdsiscsiinitiatoradapter)       | Fournit des méthodes pour interroger et interagir avec les adaptateurs d’initiateur iSCSI sur le système local.                                                                                                             |
| [**IVdsIscsiInitiatorPortal**](/windows/desktop/api/Vds/nn-vds-ivdsiscsiinitiatorportal)         | Fournit des méthodes pour interroger et interagir avec les portails de l’initiateur iSCSI sur le système local.                                                                                                              |
| [**IVdsIscsiPortal**](/windows/desktop/api/Vds/nn-vds-ivdsiscsiportal)                           | Fournit des méthodes pour effectuer des opérations de requête et de configuration sur un portail iSCSI.                                                                                                                |
| [**IVdsIscsiPortalGroup**](/windows/desktop/api/Vds/nn-vds-ivdsiscsiportalgroup)                 | Fournit des méthodes pour exécuter des services de requête et de configuration sur un groupe de portails iSCSI.                                                                                                            |
| [**IVdsIscsiPortalLocal**](/windows/desktop/api/Vds/nn-vds-ivdsiscsiportallocal)                 | Fournit des méthodes pour définir des clés prépartagées IPSEC spécifiques à l’initiateur local sur un portail iSCSI.                                                                                                       |
| [**IVdsIscsiTarget**](/windows/desktop/api/Vds/nn-vds-ivdsiscsitarget)                           | Fournit des méthodes pour effectuer des opérations de requête et de configuration sur une cible iSCSI.                                                                                                                |
| [**IVdsLun**](/windows/desktop/api/Vds/nn-vds-ivdslun)                                           | Fournit des méthodes pour effectuer des opérations de requête et de configuration sur un numéro d’unité logique (LUN).                                                                                                    |
| [**IVdsLun2**](/windows/desktop/api/Vds/nn-vds-ivdslun2)                                         | Fournit des méthodes pour l’application et l’interrogation des indicateurs de numéro d’unité logique (LUN).                                                                                                                           |
| [**IVdsLunControllerPorts**](/windows/desktop/api/Vds/nn-vds-ivdsluncontrollerports)             | Fournit des méthodes pour effectuer des opérations de configuration de port de contrôleur sur un numéro d’unité logique.                                                                                                                    |
| [**IVdsLunIscsi**](/windows/desktop/api/Vds/nn-vds-ivdsluniscsi)                                 | Fournit des méthodes pour effectuer des opérations de requête et de configuration sur un numéro d’unité logique iSCSI.                                                                                                                   |
| [**IVdsLunMpio**](/windows/desktop/api/Vds/nn-vds-ivdslunmpio)                                   | Fournit des méthodes pour effectuer des opérations de requête et de configuration sur un numéro d’unité logique avec des extensions MPIO.                                                                                                     |
| [**IVdsLunNaming**](/windows/desktop/api/Vds/nn-vds-ivdslunnaming)                               | Fournit une méthode pour nommer des numéros d’unité logique pour une classe qui implémente l’interface [**IVdsLun**](/windows/desktop/api/Vds/nn-vds-ivdslun) .                                                                                                     |
| [**IVdsLunNumber**](/windows/desktop/api/Vds/nn-vds-ivdslunnumber)                               | Fournit une méthode pour interroger le numéro d’unité logique d’un numéro d’unité logique.                                                                                                                                                  |
| [**IVdsLunPlex**](/windows/desktop/api/Vds/nn-vds-ivdslunplex)                                   | Fournit des méthodes pour effectuer des opérations de requête et de configuration sur un plex LUN.                                                                                                                     |
| [**IVdsMaintenance**](/windows/desktop/api/Vds/nn-vds-ivdsmaintenance)                           | Fournit des méthodes pour effectuer des opérations de maintenance sur un sous-système, un contrôleur ou un lecteur.                                                                                                          |
| [**IVdsOpenVDisk**](/windows/desktop/api/Vds/nn-vds-ivdsopenvdisk)                               | Définit des méthodes pour gérer un disque virtuel.                                                                                                                                                          |
| [**IVdsPack**](/windows/desktop/api/Vds/nn-vds-ivdspack)                                         | Interroge et configure des packs contenant des disques, et crée des volumes.                                                                                                                                   |
| [**IVdsPack2**](/windows/desktop/api/Vds/nn-vds-ivdspack2)                                       | Fournit une méthode pour créer des volumes alignés sur un Pack.                                                                                                                                                |
| [**IVdsProvider**](/windows/desktop/api/Vds/nn-vds-ivdsprovider)                                 | Retourne les propriétés d’un fournisseur de matériel ou de logiciel.                                                                                                                                                 |
| [**IVdsProviderPrivate**](/windows/desktop/api/VdsHwPrv/nn-vdshwprv-ivdsproviderprivate)                   | Fournit des méthodes pour permettre à VDS d’effectuer diverses opérations sur les objets du fournisseur.                                                                                                               |
| [**IVdsProviderSupport**](/windows/desktop/api/Vds/nn-vds-ivdsprovidersupport)                   | Fournit une méthode pour indiquer quelles versions des interfaces VDS sont prises en charge par le fournisseur.                                                                                                      |
| [**IVdsRemovable**](/windows/desktop/api/Vds/nn-vds-ivdsremovable)                               | Interroge et éjecte les disques amovibles, tels que les CD-ROM.                                                                                                                                                  |
| [**IVdsService**](/windows/desktop/api/Vds/nn-vds-ivdsservice)                                   | Fournit des méthodes de niveau de service pour travailler avec VDS.                                                                                                                                                  |
| [**IVdsServiceHba**](/windows/desktop/api/Vds/nn-vds-ivdsservicehba)                             | Fournit une méthode pour interroger les ports HBA sur le système local.                                                                                                                                             |
| [**IVdsServiceIscsi**](/windows/desktop/api/Vds/nn-vds-ivdsserviceiscsi)                         | Fournit des méthodes pour interagir avec le service initiateur local.                                                                                                                                       |
| [**IVdsServiceLoader**](/windows/desktop/api/Vds/nn-vds-ivdsserviceloader)                       | Lance VDS.                                                                                                                                                                                         |
| [**IVdsServiceUninstallDisk**](/windows/desktop/api/Vds/nn-vds-ivdsserviceuninstalldisk)         | Fournit des méthodes pour désinstaller des disques de base et dynamiques.                                                                                                                                                |
| [**IVdsStoragePool**](/windows/desktop/api/Vds/nn-vds-ivdsstoragepool)                           | Fournit des méthodes pour interroger des informations et énumérer des objets connexes pour un [pool de stockage](storage-pool-object.md).                                                                                    |
| [**IVdsSubSystem**](/windows/desktop/api/Vds/nn-vds-ivdssubsystem)                               | Fournit des méthodes pour effectuer des opérations de requête et de configuration sur un sous-système.                                                                                                                    |
| [**IVdsSubSystem2**](/windows/desktop/api/Vds/nn-vds-ivdssubsystem2)                             | Fournit des méthodes pour effectuer des opérations de requête et de configuration sur un sous-système à l’aide des structures [**\_ HINTS2**](/windows/desktop/api/Vds/ns-vds-vds_hints2) et [**\_ Sub \_ System \_ PROP2**](/windows/desktop/api/Vds/ns-vds-vds_sub_system_prop2) VDS. |
| [**IVdsSubSystemImportTarget**](/windows/desktop/api/Vds/nn-vds-ivdssubsystemimporttarget)       | Fournit des méthodes pour interroger et configurer la cible d’importation VSS par défaut pour le sous-système.                                                                                                              |
| [**IVdsSubSystemInterconnect**](/windows/desktop/api/Vds/nn-vds-ivdssubsysteminterconnect)       | Fournit une méthode pour interroger les types d’interconnexion pris en charge par un sous-système.                                                                                                                  |
| [**IVdsSubSystemIscsi**](/windows/desktop/api/Vds/nn-vds-ivdssubsystemiscsi)                     | Fournit des méthodes pour interroger et configurer des portails et des cibles iSCSI sur un sous-système.                                                                                                                     |
| [**IVdsSubSystemNaming**](/windows/desktop/api/Vds/nn-vds-ivdssubsystemnaming)                   | Fournit une méthode pour nommer des sous-systèmes pour une classe qui implémente l’interface [**IVdsSubSystem**](/windows/desktop/api/Vds/nn-vds-ivdssubsystem) .                                                                                   |
| [**IVdsSwProvider**](/windows/desktop/api/Vds/nn-vds-ivdsswprovider)                             | Effectue des opérations de fournisseur de logiciels.                                                                                                                                                                |
| [**IVdsVDisk**](/windows/desktop/api/Vds/nn-vds-ivdsvdisk)                                       | Définit des méthodes pour gérer un disque virtuel.                                                                                                                                                          |
| [**IVdsVdProvider**](/windows/desktop/api/Vds/nn-vds-ivdsvdprovider)                             | Définit des méthodes pour créer et gérer des disques virtuels.                                                                                                                                              |
| [**IVdsVolume**](/windows/desktop/api/Vds/nn-vds-ivdsvolume)                                     | Crée et supprime des plex de volume et modifie les attributs de volume.                                                                                                                                    |
| [**IVdsVolume2**](/windows/desktop/api/Vds/nn-vds-ivdsvolume2)                                   | Fournit une méthode pour retourner les informations sur les propriétés de volume, y compris les GUID de volume.                                                                                                              |
| [**IVdsVolumeMF**](/windows/desktop/api/Vds/nn-vds-ivdsvolumemf)                                 | Effectue des opérations sur le chemin d’accès et le système de fichiers sur les objets de volume.                                                                                                                                    |
| [**IVdsVolumeMF2**](/windows/desktop/api/Vds/nn-vds-ivdsvolumemf2)                               | Fournit des méthodes pour effectuer des opérations de gestion de système de fichiers supplémentaires sur l’objet volume.                                                                                                        |
| [**IVdsVolumeMF3**](/windows/desktop/api/Vds/nn-vds-ivdsvolumemf3)                               | Fournit des méthodes pour effectuer des opérations de gestion de système de fichiers supplémentaires sur l’objet volume.                                                                                                        |
| [**IVdsVolumeOnline**](/windows/desktop/api/Vds/nn-vds-ivdsvolumeonline)                         | Fournit une méthode pour mettre en ligne des volumes uniques.                                                                                                                                                     |
