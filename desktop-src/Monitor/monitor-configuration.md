# <a name="monitor-configuration-monitor-configuration"></a>Configuration de l’analyse (configuration de l’analyse)

Vous pouvez utiliser les fonctions de configuration de l’analyse pour obtenir des informations à partir d’une analyse et modifier les paramètres d’un moniteur. Vous pouvez utiliser ces fonctions pour :

-   Modifiez les paramètres de géométrie d’un moniteur, tels que la largeur et la hauteur de l’affichage.
-   Modifiez les paramètres de l’image, tels que la luminosité et le contraste.
-   Réinitialiser les paramètres d’usine des paramètres d’un analyseur.
-   Démagnétisez l’analyse.

En interne, les fonctions de configuration de l’analyse utilisent l’interface de commande de canal de données d’affichage (DDC/CI) pour envoyer des commandes au moniteur.

Les fonctions de configuration de l’analyse sont documentées dans les sections suivantes :

-   [À propos de la configuration de l’analyse](about-monitor-configuration.md)
-   [Utilisation de la configuration de l’analyse](using-monitor-configuration.md)
-   [Référence de configuration de l’analyse](monitor-configuration-reference.md)
