# <a name="monitor-configuration-enumeration-types"></a>Surveiller les types d’énumération de configuration

Les énumérations suivantes sont utilisées par les fonctions de configuration de l’analyse.



| Type d'énumération                                                    | Description                                                                                      |
|---------------------------------------------------------------------|--------------------------------------------------------------------------------------------------|
| [**\_température de couleur MC \_**](/windows/desktop/api/HighLevelMonitorConfigurationAPI/ne-highlevelmonitorconfigurationapi-mc_color_temperature)              | Décrit la température de couleur d’un moniteur.                                                         |
| [**\_type de \_ technologie d’affichage MC \_**](/windows/desktop/api/HighLevelMonitorConfigurationAPI/ne-highlevelmonitorconfigurationapi-mc_display_technology_type) | Identifie les technologies d’affichage du moniteur.                                                         |
| [**\_type de lecteur MC \_**](/windows/desktop/api/HighLevelMonitorConfigurationAPI/ne-highlevelmonitorconfigurationapi-mc_drive_type)                            | Spécifie s’il faut définir ou obtenir le lecteur rouge, vert ou bleu d’un moniteur.                           |
| [**\_type de gain MC \_**](/windows/desktop/api/HighLevelMonitorConfigurationAPI/ne-highlevelmonitorconfigurationapi-mc_gain_type)                              | Spécifie s’il faut obtenir ou définir le gain rouge, vert ou bleu d’un moniteur.                            |
| [**\_type de position MC \_**](/windows/desktop/api/HighLevelMonitorConfigurationAPI/ne-highlevelmonitorconfigurationapi-mc_position_type)                      | Spécifie s’il faut obtenir ou définir la position verticale ou horizontale de la zone d’affichage d’un moniteur. |
| [**\_type de taille MC \_**](/windows/desktop/api/HighLevelMonitorConfigurationAPI/ne-highlevelmonitorconfigurationapi-mc_size_type)                              | Spécifie s’il faut obtenir ou définir la largeur ou la hauteur de la zone d’affichage d’un moniteur.                 |
| [**\_type de \_ code MC VCP \_**](/windows/desktop/api/LowLevelMonitorConfigurationAPI/ne-lowlevelmonitorconfigurationapi-mc_vcp_code_type)                     | Décrit un type de code du panneau de configuration virtuel (VCP).                                               |
