# <a name="ink-input"></a>Entrée d’encre

En savoir plus sur les différentes API d’entrée de bureau pour les applications Windows.

## <a name="in-this-section"></a>Contenu de cette section

| Rubrique | Description |
|---|---|
| [Présentateur d’encre](ink-presenter.md)<br/> | Les API de présentateur d’encre permettent aux applications Microsoft Win32 de gérer l’entrée, le traitement et le rendu des entrées manuscrites (standard et modifiées) via un objet [**InkPresenter**](/uwp/api/Windows.UI.Input.Inking.InkPresenter) inséré dans l’arborescence d’éléments visuels [DirectComposition](../directcomp/directcomposition-portal.md) de l’application.<br/> |
| [Convertisseur d’encre](ink-renderer.md)<br/>   | Les API de [rendu d’encre](ink-renderer.md) activent le rendu des traits d’encre dans le contexte de périphérique Direct2D désigné d’une application Windows universelle, au lieu du contrôle [**InkCanvas**](/uwp/api/Windows.UI.Xaml.Controls.InkCanvas) par défaut.<br/>                                                                        |
