# <a name="proof-of-possession-cookie-api"></a>API de cookie de preuve de possession

Vous permet de créer des preuves de possession de cookies, qui prennent en charge l’authentification unique pour les applications Web authentifiées par le protocole d’authentification Internet OAuth. Vous pouvez utiliser les informations retournées par cette API pour implémenter les comptes Microsoft de l’authentification unique.

## <a name="in-this-section"></a>Contenu de cette section



| Rubrique                                                                                         | Description                                                      |
|-----------------------------------------------------------------------------------------------|------------------------------------------------------------------|
| [**IProofOfPossessionCookieInfoManager**](/windows/desktop/api/ProofOfPossessionCookieInfo/nn-proofofpossessioncookieinfo-iproofofpossessioncookieinfomanager)<br/> | Prend en charge la création de cookies de preuve de possession.<br/> |
