# <a name="win32_programgrouporitem-class"></a>\_Classe ProgramGroupOrItem Win32

La [classe WMI](../wmisdk/retrieving-a-class.md) **\_ ProgramGroupOrItem** abstraite Win32 représente un regroupement logique de programmes dans le menu **Démarrer des \\ programmes** de l’utilisateur. Il contient les groupes de programmes et les éléments de groupe de programmes.

La syntaxe suivante est simplifiée par rapport au code MOF (Managed Object Format) et inclut toutes les propriétés héritées. Les propriétés et les méthodes sont classées par ordre alphabétique, et non par ordre MOF.

## <a name="syntax"></a>Syntaxe

``` syntax
[Abstract, UUID("{86E30E86-7DB2-11d2-90CB-0060081A46FD}"), AMENDMENT]
class Win32_ProgramGroupOrItem : CIM_LogicalElement
{
  string   Caption;
  string   Description;
  datetime InstallDate;
  string   Name;
  string   Status;
};
```
