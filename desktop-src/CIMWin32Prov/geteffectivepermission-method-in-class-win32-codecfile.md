# <a name="geteffectivepermission-method-of-the-win32_codecfile-class"></a>Méthode GetEffectivePermission de la \_ classe Win32 CodecFile

La méthode de [classe WMI](/windows/desktop/WmiSdk/retrieving-a-class) [**GetEffectivePermission**](geteffectivepermission-method-in-class-win32-shortcutfile.md) utilise les valeurs définies dans le paramètre *permissions* pour déterminer si l’utilisateur dispose des autorisations spécifiées dans la propriété **AccessMask** de l’objet [**Win32 \_ CodecFile**](win32-codecfile.md) qui représente le fichier de codec. Les autorisations s’appliquent au fichier, au répertoire dans lequel le fichier réside et au partage, si le répertoire ou le fichier se trouve dans un partage.

Cette rubrique utilise la syntaxe format MOF (MOF). Pour plus d’informations sur l’utilisation de cette méthode, consultez [appel d’une méthode](/windows/desktop/WmiSdk/calling-a-method).

## <a name="syntax"></a>Syntaxe


```mof
boolean GetEffectivePermission(
  [in] uint32 Permissions
);
```
