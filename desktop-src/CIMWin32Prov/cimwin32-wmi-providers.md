# <a name="cimwin32-wmi-providers"></a>Fournisseurs WMI CIMWin32

Les fournisseurs WMI CIMWin32 prennent en charge les classes implémentées dans CimWin32.dll, et se composent des classes WMI CIM principales, de l’implémentation Win32 de ces classes et des événements de gestion de l’alimentation.
