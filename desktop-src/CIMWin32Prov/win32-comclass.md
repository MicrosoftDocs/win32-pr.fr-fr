# <a name="win32_comclass-class"></a>\_Classe Win32 Comclasse

La [classe WMI](/windows/desktop/WmiSdk/retrieving-a-class) de la classe abstraite **Win32 \_** représente les propriétés d’un composant COM (Component Object Model).

La syntaxe suivante est simplifiée par rapport au code MOF (Managed Object Format) et inclut toutes les propriétés héritées. Les propriétés sont répertoriées par ordre alphabétique, et non par ordre MOF.

## <a name="syntax"></a>Syntaxe

``` syntax
[abstract, UUID("{0F73ED50-8ED9-11d2-B340-00105A1F8569}"), AMENDMENT]
class Win32_COMClass : CIM_LogicalElement
{
  string   Caption;
  string   Description;
  datetime InstallDate;
  string   Name;
  string   Status;
};
```
