# <a name="win32_comapplication-class"></a>\_Classe Comapplication Win32

La [classe WMI](/windows/desktop/WmiSdk/retrieving-a-class) de l’application abstraite **Win32 \_ complicative** représente une application COM (Component Object Model). Dans ce contexte, une application COM est un regroupement logique de classes COM.

La syntaxe suivante est simplifiée par rapport au code MOF (Managed Object Format) et inclut toutes les propriétés héritées. Les propriétés sont répertoriées par ordre alphabétique, et non par ordre MOF.

## <a name="syntax"></a>Syntaxe

``` syntax
[abstract, UUID("{0F73ED4F-8ED9-11d2-B340-00105A1F8569}"), AMENDMENT]
class Win32_COMApplication : CIM_LogicalElement
{
  string   Caption;
  string   Description;
  datetime InstallDate;
  string   Name;
  string   Status;
};
```
