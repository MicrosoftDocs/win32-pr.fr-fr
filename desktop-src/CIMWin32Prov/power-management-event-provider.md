# <a name="power-management-event-provider"></a>Fournisseur d’événements de gestion de l’alimentation

le fournisseur de gestion de l’alimentation permet à WMI de modéliser les Windows les protocoles de gestion de l’alimentation. En tant que fournisseur d’événements, le fournisseur de gestion de l’alimentation fournit des informations à la classe [**Win32 \_ PowerManagementEvent**](win32-powermanagementevent.md) pour décrire les événements de gestion de l’alimentation qui résultent des modifications de l’état d’alimentation. Ces modifications d’État sont associées à la gestion avancée de l’alimentation (APM) ou aux protocoles de gestion du système ACPI (Advanced Configuration and Power Interface). Le nom de l’instance [**\_ \_ Win32Provider**](/windows/desktop/WmiSdk/--win32provider) est « MS \_ Power \_ Management \_ Event \_ Provider ».
