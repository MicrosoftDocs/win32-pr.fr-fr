# <a name="win32_potsmodem-class"></a>\_Classe POTSModem Win32

La [classe WMI](../wmisdk/retrieving-a-class.md) **\_ POTSModem** WMI représente les services et les caractéristiques d’un modem pots (Plain Old Telephone Service) sur un système informatique exécutant Windows.

La syntaxe suivante est simplifiée par rapport au code MOF (Managed Object Format) et inclut toutes les propriétés héritées. Les propriétés sont répertoriées par ordre alphabétique, et non par ordre MOF.
