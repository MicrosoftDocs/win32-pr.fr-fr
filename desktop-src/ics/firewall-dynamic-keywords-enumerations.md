# <a name="firewall-dynamic-keywords-enumerations"></a>Énumérations de mots clés dynamiques de pare-feu

Les mots clés dynamiques de pare-feu contiennent les énumérations suivantes.

## <a name="in-this-section"></a>Contenu de cette section

| Rubrique | Description |
|-|-|
| [**FW_DYNAMIC_KEYWORD_ADDRESS_ENUM_FLAGS**](/windows/win32/api/netfw/ne-netfw-fw_dynamic_keyword_address_enum_flags) | Définit des constantes qui spécifient le ou les genres d’objets à inclure dans une opération d’énumération. |
| [**FW_DYNAMIC_KEYWORD_ADDRESS_FLAGS**](/windows/win32/api/netfw/ne-netfw-fw_dynamic_keyword_address_flags) | Définit les constantes qui spécifient la façon dont les adresses IP doivent être résolues. |
| [**FW_DYNAMIC_KEYWORD_ORIGIN_TYPE**](/windows/win32/api/netfw/ne-netfw-fw_dynamic_keyword_origin_type) | Définit des constantes qui spécifient l’origine de l’objet d’adresse de mot clé dynamique dans un [**FW_DYNAMIC_KEYWORD_ADDRESS_DATA0**](/windows/win32/api/netfw/ns-netfw-fw_dynamic_keyword_address_data0). |
