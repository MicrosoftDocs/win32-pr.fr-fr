# <a name="firewall-dynamic-keywords-structures"></a>Structures de mots clés dynamiques de pare-feu

Les mots clés dynamiques de pare-feu contiennent les structures suivantes.

## <a name="in-this-section"></a>Contenu de cette section

| Rubrique | Description |
|-|-|
| [**FW_DYNAMIC_KEYWORD_ADDRESS_DATA0**](/windows/win32/api/netfw/ns-netfw-fw_dynamic_keyword_address_data0) | Contient les données retournées au client lorsque les API d' **énumération** sont appelées. |
| [**FW_DYNAMIC_KEYWORD_ADDRESS0**](/windows/win32/api/netfw/ns-netfw-fw_dynamic_keyword_address0) | Permet au client de créer une adresse de mot clé dynamique, qui contient une liste d’adresses IP. |
