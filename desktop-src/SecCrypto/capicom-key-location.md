# <a name="capicom_key_location-enumeration"></a>Énumération de l’emplacement de la \_ clé CAPICOM \_

L’énumération de l' **\_ \_ emplacement de la clé CAPICOM** définit les types d’emplacement de clé.

## <a name="members"></a>Membres



| Membre                           | Description                          | Valeur |
|----------------------------------|--------------------------------------|-------|
| **clé de l' \_ utilisateur actuel CAPICOM \_ \_**  | La clé est une clé utilisateur.<br/>    | 0     |
| **\_clé d’ordinateur local \_ CAPICOM \_** | La clé est une clé d’ordinateur.<br/> | 1     |



## <a name="remarks"></a>Remarques

Cette énumération est utilisée par la méthode suivante :
