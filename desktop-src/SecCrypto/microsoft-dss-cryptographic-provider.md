# <a name="microsoft-dss-cryptographic-provider"></a>Fournisseur de services de chiffrement DSS Microsoft

Le fournisseur de services de chiffrement DSS de Microsoft prend en charge le hachage, la signature des données et la vérification de signature à l’aide des algorithmes SHA ( [*Secure Hash Algorithm*](../secgloss/s-gly.md) ) et DSS ( [*Digital Signature standard*](../secgloss/d-gly.md) ). Il peut être exporté en dehors du États-Unis et du Canada.
