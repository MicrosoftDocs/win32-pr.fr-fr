# <a name="adherence-to-standards"></a>Respect des normes

Les services de certificats acceptent \# les demandes PKCS 10 standard et les problèmes des certificats X. 509 3,0. Les services de certificats fonctionnent avec les clients et navigateurs non-Microsoft, ainsi que pour les serveurs Web non Microsoft.
