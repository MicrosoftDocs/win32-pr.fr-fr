# <a name="active-directory"></a>Active Directory

Les services de certificats prennent en charge l’utilisation de Active Directory. Dans le cas de l' [*autorité de certification*](../secgloss/c-gly.md) d’entreprise (ca), les services de certificats requièrent l’utilisation de la Active Directory. L’autorité de certification d’entreprise publie des certificats utilisateur et des [*listes de révocation de certificats*](../secgloss/c-gly.md) (CRL) sur le Active Directory. Une autorité de certification autonome peut être administrée pour effectuer les mêmes actions que si la Active Directory est disponible pour l’autorité de certification autonome et qu’un module de stratégie personnalisé est utilisé.
