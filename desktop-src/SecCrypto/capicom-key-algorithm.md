# <a name="capicom_key_algorithm-enumeration"></a>CAPICOM \_ Key Algorithm ( \_ énumération)

L’énumération de l' **\_ \_ algorithme de clé CAPICOM** définit des algorithmes de clé.

## <a name="members"></a>Membres



| Membre                             | Description                                | Valeur |
|------------------------------------|--------------------------------------------|-------|
| **\_algorithme de clé \_ CAPICOM \_ autre** | Utilisez un algorithme de clé secondaire.<br/> | 0     |
| **\_algorithme de clé \_ CAPICOM \_ RSA**   | Utilisez l’algorithme de clé RSA.<br/>      | 1     |
| **\_algorithme de clé \_ CAPICOM \_ DSS**   | Utilisez l’algorithme de clé DSS.<br/>      | 2     |



## <a name="remarks"></a>Notes

Cette énumération est réservée à une utilisation ultérieure.
