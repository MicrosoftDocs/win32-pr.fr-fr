# <a name="utilitiesbinarystringtobytearray-method"></a>Méthode Utilities. BinaryStringToByteArray

\[La méthode **BinaryStringToByteArray** peut être utilisée dans les systèmes d’exploitation spécifiés dans la section relative à la configuration requise.\]

La méthode **BinaryStringToByteArray** convertit une chaîne binaire compressée en un tableau d’octets.

## <a name="syntax"></a>Syntaxe


```VB
Utilities.BinaryStringToByteArray( _
  ByVal BinaryString _
)
```



## <a name="parameters"></a>Paramètres

<dl> <dt>

*BinaryString* \[ dans\]
</dt> <dd>

Chaîne compressée binaire à convertir en tableau d’octets.

</dd> </dl>

## <a name="return-value"></a>Valeur de retour

Tableau qui contient le résultat de la conversion.
