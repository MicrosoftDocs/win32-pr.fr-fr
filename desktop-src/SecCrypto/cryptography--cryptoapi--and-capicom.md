# <a name="cryptography-cryptoapi-and-capicom"></a>Chiffrement, CryptoAPI et CAPICOM

CryptoAPI et CAPICOM fournissent des services qui permettent aux développeurs d’applications d’ajouter le chiffrement/déchiffrement des données, l’authentification à l’aide de [*certificats*](../secgloss/c-gly.md)numériques, ainsi que l’encodage et le décodage de ASN. 1 ( [*Abstract Syntax Notation One*](../secgloss/a-gly.md) ) dans leurs applications Windows. Les développeurs d’applications peuvent utiliser les fonctions dans CryptoAPI sans connaître les détails de l’implémentation sous-jacente, à peu près de la même façon qu’ils peuvent utiliser une bibliothèque de graphiques sans connaître la configuration matérielle graphique particulière. CryptoAPI fonctionne avec un certain nombre de [fournisseurs de services de chiffrement](microsoft-cryptographic-service-providers.md) qui exécutent les fonctions de chiffrement réelles.
