# <a name="digital-messages"></a>Messages numériques

Un message est défini comme toute donnée qui a été encodée. Les messages sont encodés pour être transmis à une autre personne ou entité ou à des données encodées reçues d’une autre personne ou entité. Dans l’environnement de chiffrement, les messages sont généralement signés, chiffrés ou les deux. Le terme « messages » comprend souvent des certificats.

Dans CryptoAPI version 2,0, les fonctions de gestion des messages sont divisées en [*fonctions de message de bas niveau*](../secgloss/l-gly.md) et [*fonctions de message simplifiées*](../secgloss/s-gly.md).
