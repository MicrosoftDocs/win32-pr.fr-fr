# <a name="error-handling-cryptography"></a>Gestion des erreurs (chiffrement)

Dans tous les exemples C qui accompagnent la documentation de chaque fonction, l’exemple de fonction [**MyHandleError**](myhandleerror.md) est utilisé pour imprimer un message d’erreur et quitter une procédure. Dans les situations réelles, des procédures de gestion des erreurs plus étendues peuvent être nécessaires. Pour plus d’informations, consultez [exemples de limitations de programme](example-program-limitations.md).
