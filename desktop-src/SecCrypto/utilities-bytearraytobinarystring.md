# <a name="utilitiesbytearraytobinarystring-method"></a>Méthode Utilities. ByteArrayToBinaryString

\[La méthode **ByteArrayToBinaryString** peut être utilisée dans les systèmes d’exploitation spécifiés dans la section relative à la configuration requise.\]

La méthode **ByteArrayToBinaryString** convertit un tableau d’octets en chaîne binaire.

## <a name="syntax"></a>Syntaxe


```VB
Utilities.ByteArrayToBinaryString( _
  ByVal varByteArray _
)
```



## <a name="parameters"></a>Paramètres

<dl> <dt>

*varByteArray* \[ dans\]
</dt> <dd>

Tableau d’octets à convertir en chaîne binaire.

</dd> </dl>

## <a name="return-value"></a>Valeur retournée

Chaîne binaire contenant le résultat de la conversion.
