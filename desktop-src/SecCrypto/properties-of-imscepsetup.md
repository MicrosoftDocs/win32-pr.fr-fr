# <a name="properties-of-imscepsetup"></a>Propriétés de IMSCEPSetup

Les propriétés suivantes sont définies par l’interface [**IMSCEPSetup**](/windows/desktop/api/Casetup/nn-casetup-imscepsetup) .



| Propriété                                                 | Description                                                                                                                                              |
|----------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**MSCEPErrorId**](/windows/desktop/api/Casetup/nf-casetup-imscepsetup-get_msceperrorid)         | Obtient l’ID pour les informations d’erreur supplémentaires relatives à une spécification NDES ayant échoué. Tout appel de méthode sur l’objet parent réinitialise cette propriété.          |
| [**MSCEPErrorString**](/windows/desktop/api/Casetup/nf-casetup-imscepsetup-get_msceperrorstring) | Obtient les données de chaîne pour les informations d’erreur supplémentaires liées à une spécification NDES ayant échoué. Tout appel de méthode sur l’objet parent réinitialise cette propriété. |
