# <a name="methods-of-iocspcaconfigurationcollection"></a>Méthodes de IOCSPCAConfigurationCollection

Les méthodes suivantes sont définies par l’interface [**IOCSPCAConfigurationCollection**](/windows/desktop/api/Certadm/nn-certadm-iocspcaconfigurationcollection) . Les méthodes d’accès à la propriété ne sont pas indiquées ici. Pour afficher les propriétés de **IOCSPCAConfigurationCollection**, consultez [Propriétés de IOCSPCAConfigurationCollection](properties-of-iocspcaconfigurationcollection.md)



| Méthode                                                                                       | Description                                                                                                                                                                                    |
|----------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**CreateCAConfiguration**](/windows/desktop/api/Certadm/nf-certadm-iocspcaconfigurationcollection-createcaconfiguration) | Crée une nouvelle configuration de l' [*autorité de certification*](../secgloss/c-gly.md) (ca) et l’ajoute au jeu de configuration. |
| [**DeleteCAConfiguration**](/windows/desktop/api/Certadm/nf-certadm-iocspcaconfigurationcollection-deletecaconfiguration) | Supprime une configuration d’autorité de certification nommée du jeu de configuration.                                                                                                                                   |
