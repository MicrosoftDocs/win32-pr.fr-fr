# <a name="cryptography-constants"></a>Constantes de chiffrement

Les constantes suivantes sont utilisées par les fonctions de chiffrement et définies dans wincrypt. h. Les constantes de chiffrement sont classées en fonction de l’utilisation comme suit.



| Constante                                                                                                           | Description                                                                                                                                                                                                 |
|--------------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Constantes pour CryptEncodeObject et CryptDecodeObject](constants-for-cryptencodeobject-and-cryptdecodeobject.md) | Constantes prédéfinies utilisées avec les opérations d’encodage et de décodage ASN. 1 ( [*Abstract Syntax Notation One*](../secgloss/a-gly.md) ). |
| [Constantes pour les extensions Netscape](constants-for-netscape-extensions.md)                                         | Constantes prédéfinies utilisées pour les extensions de certificat Netscape. Cette section fournit des informations sur l’utilisation de ces extensions avec les opérations d’encodage et de décodage ASN. 1.                                      |
| [Noms des fournisseurs de services de chiffrement](cryptographic-provider-names.md)                                                   | Constantes de chaîne prédéfinies pour les noms des [*fournisseurs de services de chiffrement*](../secgloss/c-gly.md) (CSP) courants.   |
| [**MAXUIDLEN**](maxuidlen.md)                                                                                     | Constante numérique qui spécifie le nombre maximal de caractères que certains des fournisseurs de services de chiffrement Microsoft utiliseront lors de l’obtention de l’ID utilisateur.                                                  |
| [**Constantes de mise en cache de clés privées**](private-key-caching-constants.md)                                             | Constantes qui représentent des entrées de Registre qui contrôlent la mise en cache de [*clé privée*](../secgloss/p-gly.md) par un CSP.                                                |
| [Indicateurs de protocole](protocol-flags.md)                                                                               | Constantes prédéfinies pour les protocoles de chiffrement.                                                                                                                                                            |
