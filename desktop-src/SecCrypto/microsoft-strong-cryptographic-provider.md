# <a name="microsoft-strong-cryptographic-provider"></a>Microsoft Strong Cryptographic Provider

Le fournisseur de services de chiffrement renforcé Microsoft est utilisé en tant que [*fournisseur de services de chiffrement*](../secgloss/c-gly.md) complet RSA par défaut. Il prend en charge tous les algorithmes du fournisseur de services de chiffrement amélioré Microsoft et toutes les mêmes [*longueurs de clé*](../secgloss/k-gly.md). Pour plus d’informations sur les algorithmes et les longueurs de clé par défaut, consultez fournisseur de services de chiffrement de base Microsoft. Pour plus d’informations sur les longueurs de clé et sur les algorithmes disponibles, consultez le tableau avec le [fournisseur de services de chiffrement amélioré Microsoft](microsoft-enhanced-cryptographic-provider.md).

|-------------------|-------------------|
