# <a name="indespolicy-methods"></a>Méthodes INDESPolicy

L’interface [**INDESPolicy**](/windows/desktop/api/certpol/nn-certpol-indespolicy) prend en charge les méthodes suivantes.

## <a name="in-this-section"></a>Contenu de cette section



| Rubrique                                                                        | Description                                                                                       |
|------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------|
| [**Méthode GenerateChallenge**](/windows/desktop/api/certpol/nf-certpol-indespolicy-generatechallenge)<br/> | Exécute la décision de stratégie indiquant si un mot de passe de stimulation doit être émis pour le client SCEP.<br/> |
| [**Initialize (méthode)**](/windows/desktop/api/certpol/nf-certpol-indespolicy-initialize)<br/>               | Initialise le module de stratégie NDES.<br/>                                                    |
| [**Notify, méthode**](/windows/desktop/api/certpol/nf-certpol-indespolicy-notify)<br/>                       | Notifie le plug-in de l’état de la transaction de la demande de certificat SCEP.<br/>        |
| [**Uninitialize (méthode)**](/windows/desktop/api/certpol/nf-certpol-indespolicy-uninitialize)<br/>           | Réinitialise le module de stratégie NDES.<br/>                                                  |
| [**Méthode VerifyRequest**](/windows/desktop/api/certpol/nf-certpol-indespolicy-verifyrequest)<br/>         | Vérifie la demande de certificat NDES à envoyer à l’autorité de certification.<br/>                        |
