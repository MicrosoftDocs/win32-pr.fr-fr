# <a name="methods-of-icenroll4"></a>Méthodes de ICEnroll4

Les méthodes suivantes sont définies par l’interface [**ICEnroll4**](/windows/desktop/api/Xenroll/nn-xenroll-icenroll4) ou sont des méthodes que l’interface **ICEnroll4** hérite de [**ICEnroll**](/windows/desktop/api/Xenroll/nn-xenroll-icenroll), [**ICEnroll2**](/windows/desktop/api/Xenroll/nn-xenroll-icenroll2)et [**ICEnroll3**](/windows/desktop/api/Xenroll/nn-xenroll-icenroll3). Les méthodes d’accès à la propriété ne sont pas indiquées ici. Pour afficher les propriétés de **ICEnroll4**, consultez [Propriétés de ICEnroll4](properties-of-icenroll4.md).



| Méthode                                                                         | Description                                                                                                                                                                                                                                                                                                                                                                            |
|--------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**acceptFilePKCS7**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll-acceptfilepkcs7)                           | Accepte et traite un fichier qui contient un \# message PKCS 7 contenant un certificat.<br/>                                                                                                                                                                                                                                                                                     |
| [**acceptFileResponse**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll4-acceptfileresponse)                     | Accepte la remise des informations d’identification émises en réponse à un appel antérieur à [**createFileRequest**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll4-createfilerequest)et place les informations d’identification dans le magasin approprié.<br/>                                                                                                                                                                              |
| [**acceptPKCS7**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll-acceptpkcs7)                                   | Accepte et traite un \# message PKCS 7 contenant un certificat. Le PKCS \# 7 est entré en tant que paramètre.<br/>                                                                                                                                                                                                                                                                    |
| [**acceptResponse**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll4-acceptresponse)                             | Accepte la remise des informations d’identification émises en réponse à un appel antérieur à [**CreateRequest**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll4-createrequest) et place les informations d’identification dans le magasin approprié.<br/>                                                                                                                                                                                          |
| [**addAttributeToRequest**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll4-addattributetorequest)               | Ajoute un [*attribut*](../secgloss/a-gly.md) à la [*demande de certificat*](../secgloss/c-gly.md).<br/>                                                                                                                                                                   |
| [**addBlobPropertyToCertificate**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll4-addblobpropertytocertificate) | Ajoute une propriété d' [*objet BLOB*](../secgloss/b-gly.md) à un certificat.<br/>                                                                                                                                                                                                                                                                            |
| [**addCertTypeToRequest**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll2-addcerttypetorequest)                 | Ajoute un modèle de certificat à une demande (utilisé pour prendre en charge l' [*autorité de certification*](../secgloss/c-gly.md) d’entreprise).<br/>                                                                                                                                                                     |
| [**addCertTypeToRequestEx**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll4-addcerttypetorequestex)             | Ajoute un modèle de certificat (ou « type de certificat ») à une demande.<br/>                                                                                                                                                                                                                                                                                                           |
| [**addExtensionToRequest**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll4-addextensiontorequest)               | Ajoute une extension à la demande.<br/>                                                                                                                                                                                                                                                                                                                                           |
| [**addNameValuePairToRequest**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll4-addnamevaluepairtorequest)       | Ajoute une paire nom-valeur de chaîne à la demande.<br/>                                                                                                                                                                                                                                                                                                                               |
| [**addNameValuePairToSignature**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll2-addnamevaluepairtosignature)   | Ajoute la paire nom/valeur d’un [*attribut*](../secgloss/a-gly.md) à la demande. Il revient à l’autorité de certification d’interpréter la signification de la paire nom-valeur.<br/>                                                                                                                                                                        |
| [**binaryToString**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll4-binarytostring)                             | Convertit un [*objet BLOB*](../secgloss/b-gly.md) de données binaires en chaîne.<br/>                                                                                                                                                                                                                                                                          |
| [**createFilePFX**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll4-createfilepfx)                               | enregistre la chaîne de certificats et la [*clé privée*](../secgloss/p-gly.md) acceptées dans un fichier au format PFX (Personal Information Exchange).<br/>                                                                                                                                                                                       |
| [**createFilePKCS10**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll-createfilepkcs10)                         | Crée une demande de certificat PKCS 10 encodé \# en [](../secgloss/c-gly.md) Base64 et l’enregistre dans un fichier.<br/>                                                                                                                                                                                                      |
| [**createFileRequest**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll4-createfilerequest)                       | Crée une demande de \# certificat PKCS 10, une \) demande PKCS (CMC) et le stocke dans un fichier.<br/>                                                                                                               |
| [**createPFX**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll4-createpfx)                                       | enregistre la chaîne de certificats et la clé privée acceptées dans une chaîne de format PFX (Personal Information Exchange). Le format PFX est également appelé PKCS \# 12.<br/>                                                                                                                                                                                                                     |
| [**createPKCS10**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll-createpkcs10)                                 | Crée une demande de certificat PKCS 10 encodé en base64 \# .<br/>                                                                                                                                                                                                                                                                                                                     |
| [**createRequest**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll4-createrequest)                               | Crée une \# demande de certificat de format PKCS 10, PKCS \) (CMC) et la stocke dans une chaîne. [](../secgloss/c-gly.md)<br/>                                  |
| [**EnumAlgs**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll3-enumalgs)                                         | Récupère les ID des algorithmes de chiffrement dans une classe d’algorithme donnée qui sont pris en charge par le fournisseur de services de chiffrement actuel.<br/>                                                                                                                                                                                                                                                             |
| [**enumContainers**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll-enumcontainers)                             | Récupère les noms des conteneurs pour le fournisseur de [*services de chiffrement*](../secgloss/c-gly.md) (CSP) spécifié par la propriété [**providerName**](/windows/win32/api/xenroll/nf-xenroll-icenroll-get_providername) .<br/>                                                                                                  |
| [**enumPendingRequest**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll4-enumpendingrequest)                     | Énumère les [*demandes de certificat*](../secgloss/c-gly.md) en attente et récupère une propriété spécifiée à partir de chaque.<br/>                                                                                                                                                                                               |
| [**enumProviders**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll-enumproviders)                               | Récupère les noms des fournisseurs de services de chiffrement disponibles spécifiés par la propriété [**ProviderType**](/windows/win32/api/xenroll/nf-xenroll-icenroll-get_providertype) .<br/>                                                                                                                                                                                                                                                         |
| [**freeRequestInfo**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll-freerequestinfo)                           | Nettoie les magasins si une erreur se produit. Non implémenté actuellement.<br/>                                                                                                                                                                                                                                                                                                         |
| [**GetAlgName**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll3-getalgname)                                     | Récupère le nom d’un algorithme de chiffrement en fonction de son ID. Les valeurs récupérées par cette méthode dépendent du CSP en cours.<br/>                                                                                                                                                                                                                                                |
| [**getCertFromFileResponse**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll4-getcertfromfileresponse)           | Récupère le certificat à partir d’un fichier contenant une réponse d’une [*autorité de certification*](../secgloss/c-gly.md).<br/>                                                                                                                                                                                 |
| [**getCertFromPKCS7**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll-getcertfrompkcs7)                         | Récupère le certificat, contenu dans un message PKCS \# 7, qui a été émis en réponse à une \# demande de certificat PKCS 10.<br/>                                                                                                                                                                                                                                                 |
| [**getCertFromResponse**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll4-getcertfromresponse)                   | Récupère le certificat à partir de la réponse [*d’une autorité de certification*](../secgloss/c-gly.md) .<br/>                                                                                                                                                                                                        |
| [**GetKeyLen**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll3-getkeylen)                                       | Récupère les longueurs de clé minimale et maximale pour la [*signature*](../secgloss/s-gly.md) et les [*clés d’échange*](../secgloss/e-gly.md).<br/>                                                                                                                |
| [**GetKeyLenEx**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll4-getkeylenex)                                   | Récupère les informations de taille pour la [*signature*](../secgloss/s-gly.md) et les [*clés d’échange*](../secgloss/e-gly.md).<br/>                                                                                                                                   |
| [**GetSupportedKeySpec**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll3-getsupportedkeyspec)                   | Récupère des informations sur la prise en charge par le CSP des clés de signature ou d’échange.<br/>                                                                                                                                                                                                                                                                                           |
| [**InstallPKCS7**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll3-installpkcs7)                                 | Traite un certificat ou une chaîne de certificats, en les plaçant dans les [*magasins de certificats*](../secgloss/c-gly.md)appropriés. Cette méthode diffère de la méthode [**acceptPKCS7**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll-acceptpkcs7) dans le fait que [**InstallPKCS7**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll3-installpkcs7) ne reçoit pas de certificat de demande.<br/> |
| [**InstallPKCS7Ex**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll4-installpkcs7ex)                             | Identique à [**InstallPKCS7**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll3-installpkcs7) , sauf qu’elle retourne le nombre de certificats réellement installés dans les magasins locaux.<br/>                                                                                                                                                                                                                            |
| [**removePendingRequest**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll4-removependingrequest)                 | Supprime une demande en attente du magasin de demandes du client.<br/>                                                                                                                                                                                                                                                                                                                  |
| [**Initialisation**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll3-reset)                                               | Retourne l’objet de contrôle d’inscription de certificat à son [*État*](../secgloss/s-gly.md)initial.<br/>                                                                                                                                                                                                                                         |
| [**resetAttributes**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll4-resetattributes)                           | Supprime tous les attributs de la requête.<br/>                                                                                                                                                                                                                                                                                                                                    |
| [**resetBlobProperties**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll4-resetblobproperties)                   | Réinitialise les propriétés d’un [*objet BLOB*](../secgloss/b-gly.md).<br/>                                                                                                                                                                                                                                                                                  |
| [**resetExtensions**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll4-resetextensions)                           | Supprime toutes les extensions de la demande.<br/>                                                                                                                                                                                                                                                                                                                                    |
| [**setPendingRequestInfo**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll4-setpendingrequestinfo)               | Définit les propriétés d’une demande en attente.<br/>                                                                                                                                                                                                                                                                                                                                      |
| [**stringToBinary**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll4-stringtobinary)                             | Convertit une chaîne encodée en un [*objet BLOB*](../secgloss/b-gly.md)de données binaires.<br/>                                                                                                                                                                                                                                                                 |
