# <a name="common-hresult-values"></a>Valeurs HRESULT communes

Les valeurs HRESULT suivantes sont les plus courantes. D’autres valeurs sont contenues dans le fichier d’en-tête winerror. h.

Voici les valeurs répertoriées par ordre alphabétique par nom.



| Nom            | Description                         | Valeur      |
|-----------------|-------------------------------------|------------|
| \_OK           | L’opération a réussi                | 0x00000000 |
| E \_ Abort        | Opération abandonnée                   | 0x80004004 |
| \_ACCESSDENIED E | Erreur d’accès général refusé         | 0x80070005 |
| E \_ échec         | Échec non spécifié                 | 0x80004005 |
| \_handle E       | Handle non valide            | 0x80070006 |
| E \_ INVALIDARG   | Un ou plusieurs arguments ne sont pas valides | 0x80070057 |
| E \_ NOinterface  | Interface non prise en charge         | 0x80004002 |
| \_NOTIMPL E      | Non implémenté                     | 0x80004001 |
| \_OUTOFMEMORY E  | Échec de l’allocation de mémoire nécessaire | 0x8007000E |
| \_pointeur E      | Pointeur non valide           | 0x80004003 |
| E \_ inattendu   | Défaillance inattendue                  | 0x8000FFFF |
