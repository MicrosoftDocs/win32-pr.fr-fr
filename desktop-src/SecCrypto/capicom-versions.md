# <a name="capicom-versions"></a>Versions de CAPICOM

\[capicom 2.1.0.2 peut être utilisé dans les systèmes d’exploitation suivants : Windows Server 2008, Windows Vista et Windows XP. utilisez plutôt le .NET Framework pour implémenter des fonctionnalités de sécurité. Pour plus d’informations, consultez [alternatives à l’utilisation de](alternatives-to-using-capicom.md)CAPICOM.\]

CAPICOM 1,0 fournit les fonctionnalités suivantes :

-   Génération et vérification de \# signatures numériques basées sur PKCS 7
-   Chiffrement et déchiffrement des données à l’aide de certificats
-   Chiffrement et déchiffrement des données à l’aide de mots de passe

CAPICOM 2,0 inclut les fonctionnalités de CAPICOM 1,0 et ajoute les fonctionnalités suivantes :

-   Génération et vérification des signatures numériques Authenticode
-   Prise en charge des informations d’identification basées sur des fichiers
-   Recherche robuste dans le magasin de certificats
-   Génération de hachages de données arbitraires
-   Prise en charge de l’algorithme Advanced Encryption Standard (AES)
-   Prise en charge des propriétés de certificat avancées, telles que les stratégies et les modèles

Pour déterminer quelles sont les méthodes et les propriétés prises en charge sur les différentes plateformes, consultez les rubriques de référence pour les objets CAPICOM. Le package redistribuable de CAPICOM 2,0 est disponible dans [Platform SDK Redistributable : CAPICOM](https://www.microsoft.com/download/details.aspx?id=25281).
