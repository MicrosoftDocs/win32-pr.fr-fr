# <a name="properties-of-iocsppropertycollection"></a>Propriétés de IOCSPPropertyCollection

Les propriétés suivantes sont définies par l’interface [**IOCSPPropertyCollection**](/windows/desktop/api/Certadm/nn-certadm-iocsppropertycollection) .



| Propriété                                                        | Description                                              |
|-----------------------------------------------------------------|----------------------------------------------------------|
| [**\_NewEnum**](/windows/desktop/api/Certadm/nf-certadm-iocsppropertycollection-get__newenum)    | Obtient un énumérateur pour un jeu de propriétés.                   |
| [**Saut**](/windows/desktop/api/Certadm/nf-certadm-iocsppropertycollection-get_count)           | Obtient le nombre de propriétés dans un jeu de propriétés.         |
| [**Élément**](/windows/desktop/api/Certadm/nf-certadm-iocsppropertycollection-get_item)             | Obtient la propriété identifiée par l’index dans un jeu de propriétés. |
| [**ItemByName**](/windows/desktop/api/Certadm/nf-certadm-iocsppropertycollection-get_itembyname) | Obtient la propriété identifiée par nom dans un jeu de propriétés.  |
