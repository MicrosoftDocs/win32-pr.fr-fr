# <a name="utilitieshextobinary-method"></a>Méthode Utilities. HexToBinary

\[La méthode **HexToBinary** peut être utilisée dans les systèmes d’exploitation spécifiés dans la section relative à la configuration requise.\]

La méthode **HexToBinary** convertit une chaîne hexadécimale en chaîne binaire.

## <a name="syntax"></a>Syntaxe


```VB
Utilities.HexToBinary( _
  ByVal HexString _
)
```



## <a name="parameters"></a>Paramètres

<dl> <dt>

*HexString* \[ dans\]
</dt> <dd>

Chaîne hexadécimale à convertir en chaîne binaire.

</dd> </dl>

## <a name="return-value"></a>Valeur retournée

Chaîne binaire qui est équivalente à la chaîne hexadécimale spécifiée.
