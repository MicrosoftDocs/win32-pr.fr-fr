# <a name="certificates-and-messages"></a>Certificats et messages

Les fonctions de chiffrement de base utilisent des [*certificats*](../secgloss/c-gly.md), et les fonctions de [*message de bas niveau*](../secgloss/l-gly.md) et les [*fonctions de message simplifiées*](../secgloss/s-gly.md) les utilisent également. Pour envoyer ou recevoir des messages qui incorporent des certificats, utilisez des fonctions de message de bas niveau.

Pour obtenir des exemples d’utilisation de certificats avec les fonctions de message de bas niveau, consultez [exemple de programme c : encodage et décodage d’un message contresigné](example-c-program-encoding-and-decoding-a-countersigned-message.md), [exemple de programme c : encodage et décodage d’un message haché](example-c-program-encoding-and-decoding-a-hashed-message.md).

Pour obtenir des exemples d’utilisation de certificats avec les fonctions de message simplifiées [, consultez exemple de programme C : utilisation de CryptEncryptMessage et CryptDecryptMessage](example-c-program-using-cryptencryptmessage-and-cryptdecryptmessage.md).

Pour plus d’informations sur les fonctions de message de bas niveau, consultez [messages de bas niveau](low-level-messages.md).

Pour plus d’informations sur les fonctions de message simplifiées, consultez [messages simplifiés](simplified-messages.md).
