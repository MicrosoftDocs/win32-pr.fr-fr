# <a name="prov_ssl"></a>SSL PROV. \_

Le \_ type de fournisseur SSL Prov prend en charge le protocole SSL (Secure Sockets Layer) (SSL).

## <a name="algorithms-supported"></a>Algorithmes pris en charge

Pour obtenir une description de chacun de ces algorithmes, consultez le Glossaire.



| Objectif      | Algorithmes pris en charge                                        |
|--------------|-------------------------------------------------------------|
| Exchange de clé | [*RSA*](../secgloss/r-gly.md) |
| Signature    | RSA                                                         |
| Chiffrement   | Variable                                                      |
| Hashing      | Variable                                                      |
