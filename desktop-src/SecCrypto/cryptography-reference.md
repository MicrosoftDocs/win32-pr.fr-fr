# <a name="cryptography-reference"></a>Référence de chiffrement

Les pages de référence sur le chiffrement contiennent des descriptions détaillées des fonctions, interfaces, objets et autres éléments de programmation de Microsoft Cryptography. Ces pages incluent des descriptions de référence de l’API pour l’utilisation des certificats numériques, des services de certificats et du contrôle d’inscription de certificats.

Les pages de référence sont réparties dans les groupes suivants. 

| Section                                                      | Description                                                                                        |
|--------------------------------------------------------------|----------------------------------------------------------------------------------------------------|
| [Constantes de chiffrement](cryptography-constants.md)         | Constantes utilisées par les fonctions de chiffrement.                                                      |
| [Types de données de chiffrement](cryptography-data-types.md)       | Types de données utilisés par les fonctions de chiffrement, les interfaces et les objets.                                |
| [Énumérations de chiffrement](cryptography-enumerations.md)   | Énumérations utilisées par les fonctions et les méthodes de chiffrement.                                          |
| [Fonctions de chiffrement](cryptography-functions.md)         | Fonctions utilisées avec les services de chiffrement et de certificat.                                         |
| [Interfaces de chiffrement](cryptography-interfaces.md)       | Interfaces utilisées avec les moteurs de serveur de certificats, l’encodage de certificat et l’inscription de certificats. |
| [Objets de chiffrement](cryptography-objects.md)             | Objets utilisés avec CAPICOM et l’inscription de certificats.                                              |
| [Valeurs de retour de chiffrement](cryptography-return-values.md) | Valeurs de retour utilisées par les fonctions de chiffrement.                                                  |
| [Structures de chiffrement](cryptography-structures.md)       | Structures utilisées avec les fonctions de chiffrement, les interfaces et les objets.                              |
| [Informations de référence sur les outils CryptoAPI](cryptoapi-tools-reference.md)   | Outils en ligne de commande de chiffrement.                                                                   |
| [Informations de référence sur CAPICOM](capicom-reference.md)                   | Liste des objets CAPICOM et des types énumérés.                                                      |
