# <a name="microsoft-dss-and-diffie-hellmanschannel-cryptographic-provider"></a>Fournisseur de services de chiffrement Microsoft DSS et Diffie-Hellman/Schannel

Le fournisseur de services de chiffrement Microsoft DSS et [*Diffie-Hellman*](../secgloss/d-gly.md) / [*Schannel*](../secgloss/s-gly.md) prend en charge le hachage, la signature des données avec DSS, la génération de clés Diffie-Hellman (d-h), l’échange de clés d-h et l’exportation d’une clé d-h. Ce CSP prend en charge la dérivation de clés pour les protocoles SSL3 et TLS1. Elle peut être exportée vers d’autres pays/régions.



|                   | Valeur                       |
|-------------------|-----------------------------|
| **Type de fournisseur** | PROUVER \_ DH \_ Schannel          |
| **Nom du fournisseur** | MS \_ Def \_ DH \_ Schannel \_ Prov |
