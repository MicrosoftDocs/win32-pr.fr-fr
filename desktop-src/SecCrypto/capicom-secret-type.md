# <a name="capicom_secret_type-enumeration"></a>\_Énumération du type de secret de CAPICOM \_

L’énumération de **\_ \_ type secret CAPICOM** indique le type de secret utilisé pour dériver une clé à utiliser pour le chiffrement/déchiffrement des données.

## <a name="members"></a>Membres



| Membre                        | Description                                                     | Valeur |
|-------------------------------|-----------------------------------------------------------------|-------|
| **\_ \_ mot de passe secret de CAPICOM** | La clé de chiffrement doit être dérivée d’un mot de passe.<br/> | 0     |
