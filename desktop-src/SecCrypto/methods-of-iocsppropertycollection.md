# <a name="methods-of-iocsppropertycollection"></a>Méthodes de IOCSPPropertyCollection

Les méthodes suivantes sont définies par l’interface [**IOCSPPropertyCollection**](/windows/desktop/api/Certadm/nn-certadm-iocsppropertycollection) . Les méthodes d’accès à la propriété ne sont pas indiquées ici. Pour afficher les propriétés de **IOCSPPropertyCollection**, consultez [Propriétés de IOCSPPropertyCollection](properties-of-iocsppropertycollection.md)



| Méthode                                                                                      | Description                                                                               |
|---------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------|
| [**CreateProperty**](/windows/desktop/api/Certadm/nf-certadm-iocsppropertycollection-createproperty)                     | Crée une nouvelle propriété et l’ajoute à un jeu de propriétés.                                    |
| [**DeleteProperty**](/windows/desktop/api/Certadm/nf-certadm-iocsppropertycollection-deleteproperty)                     | Supprime une propriété nommée d’un jeu de propriétés.                                             |
| [**GetAllProperties**](/windows/desktop/api/Certadm/nf-certadm-iocsppropertycollection-getallproperties)                 | Obtient toutes les propriétés dans un jeu de propriétés.                                                    |
| [**InitializeFromProperties**](/windows/desktop/api/Certadm/nf-certadm-iocsppropertycollection-initializefromproperties) | Crée un jeu de propriétés à partir des propriétés contenues dans une configuration de serveur existante. |
