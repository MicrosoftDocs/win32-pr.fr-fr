# <a name="tools-to-sign-files-and-check-signatures"></a>Outils pour signer des fichiers et vérifier les signatures

Les outils CryptoAPI sont les outils suivants que vous pouvez utiliser pour signer des fichiers numériquement, pour confirmer des signatures sur les fichiers et pour créer des fichiers de catalogue.



| Outil                     | Notes                                                                                                                                                                                                                                                                            |
|--------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [MakeCat](makecat.md)   | Crée un fichier de catalogue non signé. Ce fichier catalogue est ensuite utilisé pour comparer les [*hachages*](../secgloss/h-gly.md) dans le fichier catalogue aux hachages de fichiers individuels, ce qui permet de prouver que les fichiers d’origine ne sont pas altérés. |
| [SetReg](setreg.md)     | Définit les clés de Registre qui contrôlent le processus de vérification du certificat.                                                                                                                                                                                                              |
| [SignTool](signtool.md) | Signe numériquement les fichiers et confirme la signature numérique des fichiers.                                                                                                                                                                                                                    |
