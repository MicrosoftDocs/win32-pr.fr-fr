# <a name="hcryptprov_legacy"></a>HCRYPTPROV \_ hérité

Le type de données **\_ hérité HCRYPTPROV** est utilisé pour remplacer le type de données [**HCRYPTPROV**](hcryptprov.md) où le type de données **HCRYPTPROV** n’est plus utilisé. Les paramètres de fonction et les structures de données de type **HCRYPTPROV \_ hérité** doivent avoir la valeur **null**.


```C++
typedef ULONG_PTR HCRYPTPROV_LEGACY;
```
