# <a name="properties-of-icertsrvsetupkeyinformationcollection"></a>Propriétés de ICertSrvSetupKeyInformationCollection

Les propriétés suivantes sont définies par l’interface [**ICertSrvSetupKeyInformationCollection**](/windows/desktop/api/Casetup/nn-casetup-icertsrvsetupkeyinformationcollection) .



| Propriété                                                            | Description                                                                                                              |
|---------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------|
| [**\_NewEnum**](/windows/desktop/api/Casetup/nf-casetup-icertsrvsetupkeyinformationcollection-get__newenum) | Obtient un énumérateur pour la collection.                                                                                   |
| [**Saut**](/windows/desktop/api/Casetup/nf-casetup-icertsrvsetupkeyinformationcollection-get_count)        | Obtient le nombre d’objets [**ICertSrvSetupKeyInformation**](/windows/desktop/api/Casetup/nn-casetup-icertsrvsetupkeyinformation) dans la collection.         |
| [**Élément**](/windows/desktop/api/Casetup/nf-casetup-icertsrvsetupkeyinformationcollection-get_item)          | Obtient un objet [**ICertSrvSetupKeyInformation**](/windows/desktop/api/Casetup/nn-casetup-icertsrvsetupkeyinformation) identifié par index dans la collection. |
