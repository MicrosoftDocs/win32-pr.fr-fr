# <a name="hcryptprov"></a>HCRYPTPROV

Le type de données **HCRYPTPROV** est utilisé pour représenter les descripteurs des fournisseurs de services de chiffrement. Ces descripteurs sont utilisés pour indiquer le module CSP qui effectue des opérations spécifiques.


```C++
typedef ULONG_PTR HCRYPTPROV;
```
