# <a name="noticenumbersitem-property"></a>NoticeNumbers. Item, propriété

\[La propriété **Item** peut être utilisée dans les systèmes d’exploitation spécifiés dans la section Configuration requise. Pour plus d’informations, consultez [**qualifier**](qualifier.md).\]

La propriété **Item** récupère la **valeur de type long** qui représente le numéro d’avis indexé de la collection. Il s’agit de la propriété par défaut.

## <a name="syntax"></a>Syntaxe


```VB
NoticeNumbers.Item( _
  ByVal Index _
) As Variant
```



## <a name="property-value"></a>Valeur de la propriété

Objet d' [**extension**](extension.md) qui représente le numéro d’avis indexé de la collection.
