# <a name="required-libraries"></a>Bibliothèques requises

La liaison de fonctions CryptoAPI requiert la disponibilité de fichier crypt32. lib. Les programmes utilisant certaines fonctions peuvent également nécessiter advapi32. lib. Lorsque vous utilisez le compilateur Visual C++, ces fichiers doivent être listés parmi les modules objet/bibliothèque.
