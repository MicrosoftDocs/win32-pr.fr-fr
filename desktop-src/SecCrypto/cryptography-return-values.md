# <a name="cryptography-return-values"></a>Valeurs de retour de chiffrement

Les valeurs de retour suivantes sont utilisées par les fonctions de chiffrement et définies dans wincrypt. h. Les valeurs de retour de chiffrement sont classées en fonction de l’utilisation comme suit.



| Valeur retournée                                                                       | Description                                                                                                |
|------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------|
| [Valeurs de retour d’encodage/décodage ASN. 1](asn-1-encoding-decoding-return-values.md) | Constantes prédéfinies qui sont retournées par les fonctions qui encodent ou décodent ASN. 1.                    |
| [Valeurs de retour de certificat et d’approbation](certificate-and-trust-return-values.md)     | Constantes prédéfinies qui sont retournées lorsque la validité et l’état de confiance d’un certificat sont déterminés. |



 

Pour obtenir la liste des valeurs de retour de CAPICOM, consultez [**code d' \_ erreur \_ CAPICOM**](capicom-error-code.md).
