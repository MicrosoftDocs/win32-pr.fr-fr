# <a name="general-purpose-functions-cryptography"></a>Fonctions de usage général (chiffrement)

Les exemples de fonctions à usage général suivants effectuent un certain nombre de tâches auxiliaires dans les exemples de programmes :



| Fonction                               | Description                                                              |
|----------------------------------------|--------------------------------------------------------------------------|
| [**ByteToStr**](bytetostr.md)         | Convertit un tableau d’octets en une chaîne de caractères.                  |
| [**GetSignerCert**](getsignercert.md) | Récupère un certificat à partir d’un magasin qui a une extension de signature de clé. |
| [**MyHandleError**](myhandleerror.md) | Écrit un message d’erreur et quitte un programme.                             |
