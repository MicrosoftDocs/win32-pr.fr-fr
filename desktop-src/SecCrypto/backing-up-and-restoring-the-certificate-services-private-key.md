# <a name="backing-up-and-restoring-the-certificate-services-private-key"></a><span data-ttu-id="abb6b-103">Sauvegarde et restauration de la clé privée des services de certificats</span><span class="sxs-lookup"><span data-stu-id="abb6b-103">Backing Up and Restoring the Certificate Services Private Key</span></span>

<span data-ttu-id="abb6b-104">Vous ne pouvez pas utiliser les fonctions de sauvegarde et de restauration de Certadm.dll pour sauvegarder les [*clés privées*](../secgloss/p-gly.md)des services de certificats.</span><span class="sxs-lookup"><span data-stu-id="abb6b-104">You cannot use the Certadm.dll's backup and restore functions to back up the Certificate Services [*private keys*](../secgloss/p-gly.md).</span></span> <span data-ttu-id="abb6b-105">Les clés privées ne peuvent pas être sauvegardées par ces fonctions, car elles sont destinées à sauvegarder et à restaurer la base de données des services de certificats (et les fichiers associés), et cette base de données ne contient pas de clés privées (même pour les certificats auto-émis).</span><span class="sxs-lookup"><span data-stu-id="abb6b-105">Private keys cannot be backed up by these functions because these functions are intended to backup and restore the Certificate Services database (and related files), and this database does not contain any private keys (even for self-issued certificates).</span></span>

<span data-ttu-id="abb6b-106">Pour sauvegarder une clé privée des services de certificats, utilisez le composant logiciel enfichable MMC Autorité de certification ou la commande certutil (avec-Backup ou-backupkey spécifié).</span><span class="sxs-lookup"><span data-stu-id="abb6b-106">To back up a Certificate Services private key, use the Certification Authority MMC snap-in, or the certutil command (with -backup or -backupkey specified).</span></span> <span data-ttu-id="abb6b-107">La sauvegarde de la clé privée avec le composant logiciel enfichable MMC de l’autorité de certification ou de la commande certutil entraîne l’écriture de la clé privée dans un \# fichier PKCS 12.</span><span class="sxs-lookup"><span data-stu-id="abb6b-107">Backing up the private key with the Certification Authority MMC snap-in or certutil results in the private key being written to PKCS \#12 file.</span></span> <span data-ttu-id="abb6b-108">Même si ce \# fichier PKCS 12 est protégé par un mot de passe, il doit être considéré comme extrêmement sensible et doit être stocké de manière sécurisée ; le mot de passe du \# fichier PKCS 12 doit également être protégé contre les personnes non autorisées.</span><span class="sxs-lookup"><span data-stu-id="abb6b-108">Even though this PKCS \#12 file is password-protected, it should be considered extremely sensitive and must be stored securely; the password to the PKCS \#12 file should also be guarded from unauthorized persons.</span></span>

<span data-ttu-id="abb6b-109">De même, les clés privées ne peuvent pas être restaurées par les fonctions de sauvegarde et de restauration des services de certificats.</span><span class="sxs-lookup"><span data-stu-id="abb6b-109">Similarly, private keys cannot be restored by the Certificate Services backup and restore functions.</span></span> <span data-ttu-id="abb6b-110">Une clé de sauvegarde des services de certificats contenue dans un \# fichier PKCS 12 peut être restaurée par le composant logiciel enfichable MMC de l’autorité de certification ou par la commande certutil (en spécifiant les verbes-Restore ou-restorekey); Notez que la personne effectuant l’opération de restauration doit connaître le mot de passe du \# fichier PKCS 12.</span><span class="sxs-lookup"><span data-stu-id="abb6b-110">A Certificate Services backup key contained in a PKCS \#12 file can be restored by the Certification Authority MMC snap-in, or by the certutil command (specifying the -restore or -restorekey verbs); note that the person performing the restore operation will need to know the password for the PKCS \#12 file.</span></span>

<span data-ttu-id="abb6b-111">Il n’existe que deux cas dans lesquels une clé privée de services de certificats doit être sauvegardée.</span><span class="sxs-lookup"><span data-stu-id="abb6b-111">There are only two cases in which a Certificate Services private key must be backed up.</span></span> <span data-ttu-id="abb6b-112">Le premier cas est après l’installation des services de certificats.</span><span class="sxs-lookup"><span data-stu-id="abb6b-112">The first case is after the installation of Certificate Services.</span></span> <span data-ttu-id="abb6b-113">Le deuxième cas est après toute opération de renouvellement du certificat des services de certificats.</span><span class="sxs-lookup"><span data-stu-id="abb6b-113">The second case is after any renewal operation of the Certificate Services certificate.</span></span>
