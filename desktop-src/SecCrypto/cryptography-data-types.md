# <a name="cryptography-data-types"></a>Types de données de chiffrement

Les types de données suivants sont utilisés par les fonctions de chiffrement, les interfaces et les objets.



| Type de données                                                                      | Description                                                                                                                                                                                      |
|--------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**\_ID ALG**](alg-id.md)                                                      | Spécifie les identificateurs d’algorithme.                                                                                                                                                                 |
| [**\_ \_ réponse OCSP du serveur HCERT \_**](hcert-server-ocsp-response.md)            | Représente un handle vers une réponse OCSP associée à une chaîne de certificat de serveur.                                                                                                              |
| [**HCRYPTHASH**](hcrypthash.md)                                               | Spécifie des handles vers un [*objet de hachage*](../secgloss/h-gly.md).                                                                                      |
| [**HCRYPTKEY**](hcryptkey.md)                                                 | Spécifie des handles vers des clés de chiffrement.                                                                                                                                                         |
| [**HCRYPTOIDFUNCADDR**](hcryptoidfuncaddr.md)                                 | Représente un handle vers une fonction qui peut être installée à l’aide d’un [*identificateur d’objet*](../secgloss/o-gly.md) (OID).                 |
| [**HCRYPTOIDFUNCSET**](hcryptoidfuncset.md)                                   | Représente un handle vers un ensemble de fonctions qui peuvent être installées à l’aide d’un OID.                                                                                                                 |
| [**HCRYPTPROV \_ hérité**](hcryptprov-legacy.md)                                | Utilisé pour remplacer le type de données [**HCRYPTPROV**](hcryptprov.md) dans lequel le type de données **HCRYPTPROV** n’est plus utilisé.                                                                             |
| [**\_handle de clé HCRYPTPROV ou \_ NCRYPT \_ \_**](hcryptprov-or-ncrypt-key-handle.md) | Spécifie un descripteur pour un [*fournisseur de services de chiffrement*](../secgloss/c-gly.md) (CSP) CryptoAPI ou un CSP CNG. |
| [**HCRYPTPROV**](hcryptprov.md)                                               | Spécifie un descripteur pour un fournisseur de services de chiffrement CryptoAPI.                                                                                                                                                           |
| [**\_handle KEYSVCC**](keysvcc-handle.md)                                      | Spécifie des handles pour le service de clé.                                                                                                                                                            |
