# <a name="microsoft-base-dss--diffie-hellman-cryptographic-provider"></a>Fournisseur de services de chiffrement DSS Microsoft de base & Diffie-Hellman

Le fournisseur de services de chiffrement DSS et Diffie-Hellman de base Microsoft prend en charge l’échange de clés Diffie-Hellman (D-H) (dérivé de 40 bits), le hachage SHA, la signature de données DSS et la vérification de signature DSS. Elle peut être exportée vers d’autres pays/régions.
