# <a name="using-a-cert_info-data-structure"></a>Utilisation d’une \_ structure de données d’informations de certificat

[**Certificat \_**](/windows/desktop/api/Wincrypt/ns-wincrypt-cert_context) Les informations de contexte et de [**certificat \_**](/windows/desktop/api/Wincrypt/ns-wincrypt-cert_info) sont les principales structures de données contenant les informations dans un certificat. Lorsqu’un certificat est utilisé, il peut être nécessaire de le décoder. Pour plus d’informations, consultez [décodage d’une \_ structure d’informations de certificat](decoding-a-cert-info-structure.md). Des informations peuvent être ajoutées à une structure décodée d' **\_ informations** sur le certificat, puis la structure peut être à nouveau codée. Pour plus d’informations, consultez [encodage d’une \_ structure d’informations de certificat](encoding-a-cert-info-structure.md).
