# <a name="properties-of-iocspcaconfigurationcollection"></a>Propriétés de IOCSPCAConfigurationCollection

Les propriétés suivantes sont définies par l’interface [**IOCSPCAConfigurationCollection**](/windows/desktop/api/Certadm/nn-certadm-iocspcaconfigurationcollection) .



| Propriété                                                               | Description                                                                                                                                                                              |
|------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**\_NewEnum**](/windows/desktop/api/Certadm/nf-certadm-iocspcaconfigurationcollection-get__newenum)    | Obtient un énumérateur pour le jeu de configuration.                                                                                                                                            |
| [**Saut**](/windows/desktop/api/Certadm/nf-certadm-iocspcaconfigurationcollection-get_count)           | Obtient le nombre de configurations de l' [*autorité de certification*](../secgloss/c-gly.md) (ca) dans le jeu de configuration. |
| [**Élément**](/windows/desktop/api/Certadm/nf-certadm-iocspcaconfigurationcollection-get_item)             | Obtient une configuration d’autorité de certification identifiée par l’index dans le jeu de configuration.                                                                                                                    |
| [**ItemByName**](/windows/desktop/api/Certadm/nf-certadm-iocspcaconfigurationcollection-get_itembyname) | Obtient une configuration d’autorité de certification identifiée par son nom dans le jeu de configuration.                                                                                                                     |
