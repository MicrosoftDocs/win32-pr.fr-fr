# <a name="property-names"></a>Noms de propriété

Un nom de propriété de [*demande de certificat*](../secgloss/c-gly.md) peut être le nom « convivial » de l’une des propriétés communes que les services de certificats comprennent ou un [*identificateur d’objet*](../secgloss/o-gly.md) (OID). Pour plus d’informations sur les noms de propriété de requête, consultez [Propriétés de nom](name-properties.md).
