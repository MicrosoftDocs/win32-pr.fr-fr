# <a name="publickeylength-property"></a>Propriété PublicKey. Length

\[La propriété **longueur** peut être utilisée dans les systèmes d’exploitation spécifiés dans la section Configuration requise. Utilisez plutôt la [**propriété X509Certificate2. PublicKey**](/dotnet/api/system.security.cryptography.x509certificates.x509certificate2.publickey?view=netcore-3.1) dans l’espace de noms [**System. Security. Cryptography. X509Certificates**](/dotnet/api/system.security.cryptography.x509certificates.publickey.-ctor?view=netcore-3.1) .\]

La propriété **Length** récupère la longueur de la clé publique en bits.

## <a name="syntax"></a>Syntaxe


```VB
PublicKey.Length As Long
```



## <a name="property-value"></a>Valeur de la propriété

Longueur de la clé publique en bits.
