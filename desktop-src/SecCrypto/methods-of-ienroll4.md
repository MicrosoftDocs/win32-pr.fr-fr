# <a name="methods-of-ienroll4"></a>Méthodes de IEnroll4

Les méthodes suivantes sont définies par l’interface [**IEnroll4**](/windows/desktop/api/Xenroll/nn-xenroll-ienroll4) ou sont des méthodes que l’interface **IEnroll4** hérite de [**IEnroll**](/windows/desktop/api/Xenroll/nn-xenroll-ienroll) et [**IEnroll2**](/windows/desktop/api/Xenroll/nn-xenroll-ienroll2). Les méthodes d’accès à la propriété ne sont pas indiquées ici. Pour afficher les propriétés de **IEnroll4**, consultez [Propriétés de IEnroll4](properties-of-ienroll4.md).



| Méthode                                                                                                | Description                                                                                                                                                                                                                                                                                           |
|-------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**acceptFilePKCS7WStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-acceptfilepkcs7wstr)                                           | Accepte et traite un \# message PKCS 7 contenant un certificat, puis stocke le message dans un fichier.                                                                                                                                                                                                 |
| [**acceptFileResponseWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll4-acceptfileresponsewstr)                                     | Accepte la remise des informations d’identification émises en réponse à un appel antérieur à [**createFileRequestWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll4-createfilerequestwstr) et place les informations d’identification dans le magasin approprié.                                                                                                     |
| [**acceptPKCS7Blob**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-acceptpkcs7blob)                                                   | Accepte et traite un \# message PKCS 7 contenant un certificat. Le PKCS \# 7 est entré en tant que paramètre.                                                                                                                                                                                              |
| [**acceptResponseBlob**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll4-acceptresponseblob)                                             | Accepte la remise des informations d’identification émises en réponse à un appel antérieur à [**IEnroll4 :: createRequestWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll4-createrequestwstr) et place les informations d’identification dans le magasin approprié.                                                                                                   |
| [**addAttributeToRequestWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll4-addattributetorequestwstr)                               | Ajoute un attribut à la demande de certificat.                                                                                                                                                                                                                                                         |
| [**AddAuthenticatedAttributesToPKCS7Request**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-addauthenticatedattributestopkcs7request) | Ajoute des attributs authentifiés à une \# demande de certificat PKCS 7.                                                                                                                                                                                                                                      |
| [**addBlobPropertyToCertificateWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll4-addblobpropertytocertificatewstr)                 | Ajoute une propriété d’objet BLOB à un certificat.                                                                                                                                                                                                                                                                |
| [**AddCertTypeToRequestWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-addcerttypetorequestwstr)                                 | Ajoute un modèle de certificat à une demande (utilisé pour prendre en charge l’autorité de certification d’entreprise).                                                                                                                                                                                               |
| [**AddCertTypeToRequestWStrEx**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll4-addcerttypetorequestwstrex)                             | Ajoute un modèle de certificat (ou « type de certificat ») à une demande.                                                                                                                                                                                                                                     |
| [**AddExtensionsToRequest**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-addextensionstorequest)                                     | Ajoute des extensions à la demande de certificat.                                                                                                                                                                                                                                                           |
| [**addExtensionToRequestWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll4-addextensiontorequestwstr)                               | Ajoute une extension à la demande.                                                                                                                                                                                                                                                                     |
| [**addNameValuePairToRequestWStr**](/windows/win32/api/xenroll/nf-xenroll-ienroll4-addnamevaluepairtorequestwstr)                       | Ajoute une paire nom-valeur de chaîne à la demande.                                                                                                                                                                                                                                                         |
| [**AddNameValuePairToSignatureWStr**](/windows/win32/api/xenroll/nf-xenroll-ienroll-addnamevaluepairtosignaturewstr)                   | Ajoute la paire nom/valeur d’un attribut à la demande. Il revient à l’autorité de certification d’interpréter la signification de la paire nom-valeur.                                                                                                                                                                      |
| [**binaryBlobToString**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll4-binaryblobtostring)                                             | Convertit un objet BLOB de données binaires en chaîne.                                                                                                                                                                                                                                                              |
| [**createFilePFXWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll4-createfilepfxwstr)                                               | Enregistre la chaîne de certificats et la clé privée acceptées dans un fichier au format PFX.                                                                                                                                                                                                                         |
| [**createFilePKCS10WStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-createfilepkcs10wstr)                                         | Crée une demande de certificat PKCS 10 encodé \# en base64 et l’enregistre dans un fichier.                                                                                                                                                                                                                        |
| [**createFileRequestWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll4-createfilerequestwstr)                                       | Crée une demande de \# certificat PKCS 10, une \# demande PKCS 7 ou une demande de certificat CMC complète et le stocke dans un fichier.                                                                                                                                                                               |
| [**createPFXWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll4-createpfxwstr)                                                       | enregistre la chaîne de certificats et la clé privée acceptées dans une chaîne de format PFX (Personal Information Exchange). Le format PFX est également appelé PKCS \# 12.                                                                                                                                               |
| [**createPKCS10WStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-createpkcs10wstr)                                                 | Crée une demande de certificat PKCS 10 encodé en base64 \# .                                                                                                                                                                                                                                               |
| [**CreatePKCS7RequestFromRequest**](/windows/win32/api/xenroll/nf-xenroll-ienroll-createpkcs7requestfromrequest)                       | Crée une \# demande PKCS 7 à partir d’une demande de certificat existante.                                                                                                                                                                                                                                      |
| [**createRequestWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll4-createrequestwstr)                                               | Crée une \# demande de certificat au format PKCS 10, PKCS \# 7 ou Full CMC et le stocke dans une chaîne.                                                                                                                                                                                                      |
| [**EnumAlgs**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll2-enumalgs)                                                                 | Récupère les ID des algorithmes de chiffrement dans une classe d’algorithme donnée qui sont pris en charge par le fournisseur de services de chiffrement actuel.                                                                                                                                                                                       |
| [**enumContainersWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-enumcontainerswstr)                                             | Récupère les noms des conteneurs pour le CSP spécifié par la propriété [**ProviderNameWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_providernamewstr) .                                                                                                                                                                    |
| [**enumPendingRequestWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll4-enumpendingrequestwstr)                                     | Énumère les demandes de certificat en attente et récupère une propriété spécifiée à partir de chaque.                                                                                                                                                                                                                 |
| [**enumProvidersWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-enumproviderswstr)                                               | Récupère les noms des fournisseurs de services de chiffrement (CSP) disponibles spécifiés par la propriété [**ProviderType**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_providertype) .                                                                                                                                                  |
| [**freeRequestInfoBlob**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-freerequestinfoblob)                                           | Supprime un contexte de certificat.                                                                                                                                                                                                                                                                        |
| [**GetAlgNameWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll2-getalgnamewstr)                                                     | Récupère le nom d’un algorithme de chiffrement en fonction de son ID. Les valeurs récupérées par cette méthode dépendent du CSP en cours.                                                                                                                                                                          |
| [**getCAStore**](/windows/win32/api/xenroll/nf-xenroll-ienroll-getcastore)                                                             | Réservé pour un usage futur.                                                                                                                                                                                                                                                                              |
| [**getCertContextFromFileResponseWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll4-getcertcontextfromfileresponsewstr)             | Récupère le certificat à partir d’un fichier contenant une réponse d’une autorité de certification.                                                                                                                                                                                                                                |
| [**getCertContextFromPKCS7**](/windows/win32/api/xenroll/nf-xenroll-ienroll-getcertcontextfrompkcs7)                                   | Récupère le certificat, contenu dans un message PKCS \# 7, qui a été émis en réponse à une \# demande de certificat PKCS 10.                                                                                                                                                                           |
| [**getCertContextFromResponseBlob**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll4-getcertcontextfromresponseblob)                     | Récupère le certificat à partir de la réponse d’une autorité de certification.                                                                                                                                                                                                                                                       |
| [**GetKeyLen**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll2-getkeylen)                                                               | Récupère les longueurs de clé minimale et maximale pour la signature et les clés d’échange.                                                                                                                                                                                                                    |
| [**GetKeyLenEx**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll4-getkeylenex)                                                           | Récupère les informations de taille pour la signature et les clés d’échange.                                                                                                                                                                                                                                       |
| [**getMyStore**](/windows/win32/api/xenroll/nf-xenroll-ienroll-getmystore)                                                             | Réservé pour un usage futur.                                                                                                                                                                                                                                                                              |
| [**GetPrivateKeyArchiveCertificate**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll4-getprivatekeyarchivecertificate)                   | Récupère le certificat utilisé pour archiver la clé privée.                                                                                                                                                                                                                                            |
| [**getProviderTypeWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll4-getprovidertypewstr)                                           | Récupère le type du fournisseur de services de chiffrement spécifié.                                                                                                                                                                                                                                                              |
| [**getROOTHStore**](/windows/win32/api/xenroll/nf-xenroll-ienroll-getroothstore)                                                       | Réservé pour un usage futur.                                                                                                                                                                                                                                                                              |
| [**GetSupportedKeySpec**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll2-getsupportedkeyspec)                                           | Récupère des informations sur la prise en charge par le CSP des clés de signature ou d’échange.                                                                                                                                                                                                                     |
| [**InstallPKCS7Blob**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll2-installpkcs7blob)                                                 | Traite un certificat ou une chaîne de certificats, en les plaçant dans les magasins de certificats appropriés. Cette méthode diffère de la méthode [**acceptPKCS7Blob**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-acceptpkcs7blob) dans le fait que [**InstallPKCS7Blob**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll2-installpkcs7blob) ne reçoit pas de certificat de demande. |
| [**InstallPKCS7BlobEx**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll4-installpkcs7blobex)                                             | Étend [**InstallPKCS7Blob**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll2-installpkcs7blob) en retournant le nombre de certificats réellement installés dans les magasins locaux.                                                                                                                                                             |
| [**removePendingRequestWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll4-removependingrequestwstr)                                 | Supprime une demande en attente du magasin de demandes du client.                                                                                                                                                                                                                                            |
| [**Initialisation**](/windows/win32/api/xenroll/nf-xenroll-ienroll2-reset)                                                                       | Retourne l’objet de contrôle d’inscription de certificat à son état initial.                                                                                                                                                                                                                               |
| [**resetAttributes**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll4-resetattributes)                                                   | Supprime tous les attributs de la requête.                                                                                                                                                                                                                                                              |

| [**SetHStoreCA**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll2-sethstoreca)                                                           | Spécifie le descripteur à utiliser pour le magasin de l’autorité de certification.                                                                                                                                                                                                                                                         |
| [**SetHStoreMy**](/windows/win32/api/xenroll/nf-xenroll-ienroll2-sethstoremy)                                                           | Spécifie le descripteur à utiliser pour le magasin MY.                                                                                                                                                                                                                                                         |
| [**SetHStoreRequest**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll2-sethstorerequest)                                                 | Spécifie le handle à utiliser pour le magasin de demandes.                                                                                                                                                                                                                                                    |
| [**SetHStoreROOT**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll2-sethstoreroot)                                                       | Spécifie le handle à utiliser pour le magasin racine.                                                                                                                                                                                                                                                       |
| [**setPendingRequestInfoWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll4-setpendingrequestinfowstr)                               | Définit les propriétés d’une demande en attente.                                                                                                                                                                                                                                                                |
| [**SetPrivateKeyArchiveCertificate**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll4-setprivatekeyarchivecertificate)                   | Spécifie le certificat utilisé pour l’archivage de la clé privée.                                                                                                                                                                                                                                            |
| [**SetSignerCertificate**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll4-setsignercertificate)                                         | Spécifie le certificat du signataire.                                                                                                                                                                                                                                                                   |
| [**stringToBinaryBlob**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll4-stringtobinaryblob)                                             | Convertit une chaîne en un objet BLOB de données binaires.                                                                                                                                                                                                                                                              |
