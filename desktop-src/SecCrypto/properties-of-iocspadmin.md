# <a name="properties-of-iocspadmin"></a>Propriétés de IOCSPAdmin

Les propriétés suivantes sont définies par l’interface [**IOCSPAdmin**](/windows/desktop/api/certadm/nn-certadm-iocspadmin) .



| Propriété                                                                                 | Description                                                                                                                                                                                                                      |
|------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**OCSPCAConfigurationCollection**](/windows/desktop/api/Certadm/nf-certadm-iocspadmin-get_ocspcaconfigurationcollection) | Obtient une instance d’un objet [**OCSPCAConfigurationCollection**](/windows/desktop/api/Certadm/nf-certadm-iocspadmin-get_ocspcaconfigurationcollection) . Cet objet représente l’ensemble de certificats pour lequel un service de répondeur peut gérer des demandes d’État. |
| [**OCSPServiceProperties**](/windows/desktop/api/Certadm/nf-certadm-iocspadmin-get_ocspserviceproperties)                 | Obtient une instance d’un objet [**OCSPPropertyCollection**](/windows/desktop/api/Certadm/nn-certadm-iocsppropertycollection) . Cet objet représente les attributs d’un service de répondeur.                                                                            |
