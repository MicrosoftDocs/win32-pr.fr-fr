# <a name="cryptxml"></a>CryptXML

CryptXML fournit un ensemble de bas niveau d’API natives qui implémentent la création et la vérification de signature numérique XML basées sur la recommandation du W3C sur la syntaxe et le traitement des signatures XML (deuxième édition). Les développeurs doivent être familiarisés avec cette norme pour utiliser efficacement les API de signature numérique CryptXML.

CryptXML contient les rubriques suivantes :

-   [Vue d’ensemble de la signature numérique XML](xml-digital-signature-overview.md)
-   [Fonctionnalité de l’API de signature numérique XML](xml-digital-signature-api-functionality.md)
-   [Algorithmes de chiffrement de signature numérique XML](xml-digital-signature-cryptographic-algorithms.md)
-   [Extensions de chiffrement de signature numérique XML](xml-digital-signature-cryptographic-extensions.md)
-   [Considérations sur la sécurité des signatures numériques XML](xml-digital-signature-security-considerations.md)
