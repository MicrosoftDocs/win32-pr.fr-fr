# <a name="properties-of-icertsrvsetup"></a>Propriétés de ICertSrvSetup

Les propriétés suivantes sont définies par l’interface [**ICertSrvSetup**](/windows/desktop/api/Casetup/nn-casetup-icertsrvsetup) .



| Propriété                                             | Description                                                                                                                                                                                                  |
|------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**CAErrorId**](/windows/desktop/api/Casetup/nf-casetup-icertsrvsetup-get_caerrorid)         | Obtient l’ID pour les informations d’erreur supplémentaires liées à une spécification de l' [*autorité de certification*](../secgloss/c-gly.md) (ca) qui a échoué. |
| [**CAErrorString**](/windows/desktop/api/Casetup/nf-casetup-icertsrvsetup-get_caerrorstring) | Obtient les données de chaîne pour les informations d’erreur supplémentaires liées à une spécification d’autorité de certification qui a échoué.                                                                                                                  |
