# <a name="application-uses"></a>L’application utilise

Les services de certificats délivrent des certificats qui peuvent être utilisés dans les applications d’infrastructure à clé publique (PKI). Par exemple, les certificats fonctionnent dans les applications SSL, S/MIME et IPSEC. Les scénarios possibles dans lesquels ces certificats peuvent être utilisés sont la signature numérique et le chiffrement des messages. Les services de certificats fournissent des certificats, des [*listes de révocation de certificats*](../secgloss/c-gly.md) (CRL) et des points de distribution de liste de révocation de certificats (CDP) utilisables par les applications Microsoft et non-Microsoft.
