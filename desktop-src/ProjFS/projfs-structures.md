# <a name="projfs-structures"></a>Structures ProjFS

Les structures suivantes sont déclarées dans projectedfslib. h.

## <a name="in-this-section"></a>Contenu de cette section

| Rubrique | Description |
|-|-|
| [**PRJ_CALLBACK_DATA**](/windows/win32/api/projectedfslib/ns-projectedfslib-prj_callback_data) | Définit les informations standard passées à un fournisseur pour chaque rappel d’opération. |
| [**PRJ_CALLBACKS**](/windows/win32/api/projectedfslib/ns-projectedfslib-prj_callbacks) | Ensemble de pointeurs vers l’emplacement où le fournisseur stocke ses implémentations des routines de rappel. |
| [**PRJ_COMPLETE_COMMAND_EXTENDED_PARAMETERS**](/windows/win32/api/projectedfslib/ns-projectedfslib-prj_complete_command_extended_parameters) | Spécifie les paramètres requis pour l’exécution de certains rappels. |
| [**PRJ_FILE_BASIC_INFO**](/windows/win32/api/projectedfslib/ns-projectedfslib-prj_file_basic_info) | Informations de base sur un élément. |
| [**PRJ_NOTIFICATION_MAPPING**](/windows/win32/api/projectedfslib/ns-projectedfslib-prj_notification_mapping) | Décrit un mappage de notification, qui est un jumelage entre un répertoire (appelé « racine de notification ») et un ensemble de notifications, exprimé sous la forme d’un masque de bits. |
| [**PRJ_NOTIFICATION_PARAMETERS**](/windows/win32/api/projectedfslib/ns-projectedfslib-prj_notification_parameters) | Paramètres supplémentaires pour les notifications. |
| [**PRJ_PLACEHOLDER_INFO**](/windows/win32/api/projectedfslib/ns-projectedfslib-prj_placeholder_info) | Mémoire tampon de métadonnées pour le fichier ou le répertoire d’espace réservé. |
| [**PRJ_PLACEHOLDER_VERSION_INFO**](/windows/win32/api/projectedfslib/ns-projectedfslib-prj_placeholder_version_info) | Informations qui identifient de façon unique le contenu d’un fichier d’espace réservé. |
| [**PRJ_STARTVIRTUALIZING_OPTIONS**](/windows/win32/api/projectedfslib/ns-projectedfslib-prj_startvirtualizing_options) | Options à fournir lors du démarrage d’une instance de virtualisation. |
| [**PRJ_VIRTUALIZATION_INSTANCE_INFO**](/windows/win32/api/projectedfslib/ns-projectedfslib-prj_virtualization_instance_info) | Informations sur une instance de virtualisation. |
