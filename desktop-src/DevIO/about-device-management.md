# <a name="about-device-management"></a>À propos de la gestion des appareils

Le système utilise des messages pour avertir les applications des modifications de l’appareil et de l’alimentation. Les messages d’appareil notifient les applications des événements de modification d’appareil ; les messages d’alimentation informent les applications des événements de gestion de l’alimentation. Les applications et les pilotes peuvent également définir et utiliser des messages personnalisés pour activer la notification d’autres types d’événements. Cette vue d’ensemble fournit des informations sur les rubriques suivantes :

-   [Messages des appareils](device-messages.md)
-   [Types d’événements d’appareil](device-event-types.md)
-   [Notifications de l’appareil](device-notifications.md)
-   [Événements de l’appareil](device-events.md)
-   [Contrôle d’entrée et de sortie de l’appareil (IOCTL)](device-input-and-output-control-ioctl-.md)

Pour plus d’informations sur les messages d’alimentation, consultez Gestion de l' [alimentation](/windows/desktop/Power/power-management-portal).
