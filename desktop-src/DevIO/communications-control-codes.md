# <a name="communications-control-codes"></a>Codes de contrôle des communications

Le code de contrôle suivant est utilisé avec l’entrée et la sortie de l’appareil.



| Code de contrôle                                                        | Opération                                                                           |
|---------------------------------------------------------------------|-------------------------------------------------------------------------------------|
| [**\_ \_ insertion LSRMST série \_ IOCTL**](/windows/desktop/api/WinIoCtl/ni-winioctl-ioctl_serial_lsrmst_insert) | Active ou désactive le placement de données d’état de ligne et de modem dans le flux de données. |
