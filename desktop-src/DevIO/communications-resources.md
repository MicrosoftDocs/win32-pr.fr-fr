# <a name="communications-resources"></a>Ressources de communication

Une *ressource de communication* est un appareil physique ou logique qui fournit un seul flux de données asynchrone bidirectionnel. Les ports série, les ports parallèles, les télécopieurs et les modems sont des exemples de ressources de communication. Pour chaque ressource de communication, il existe un fournisseur de services, constitué d’une bibliothèque ou d’un pilote, qui permet aux applications d’accéder à la ressource.
