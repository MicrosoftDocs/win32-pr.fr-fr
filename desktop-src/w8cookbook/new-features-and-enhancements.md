# <a name="new-features-and-enhancements"></a>Nouvelles fonctionnalités et améliorations

Cette section du document décrit les améliorations et les nouvelles fonctionnalités qui peuvent améliorer l’expérience des utilisateurs et des développeurs. Certaines d’entre elles ont un impact sur le fait que les développeurs doivent connaître les produits existants et nouveaux. Vous trouverez ci-dessous la liste des nouvelles fonctionnalités et améliorations, regroupées par domaine de fonctionnalité :

## <a name="security"></a>Sécurité

-   Logiciel anti-programme malveillant à lancement anticipé (ELAM)
-   Conditions requises pour la signature de la fonctionnalité de démarrage sécurisé pour les pilotes en mode noyau
-   Démarrage mesuré

## <a name="performance"></a>Performances

-   Applications de démarrage
-   Systèmes de fichiers résilients
-   Prise en charge des API du serveur de fichiers
-   Maintenance automatique

## <a name="usability"></a>Facilité d'utilisation

-   Éditeurs de méthode d’entrée tiers

## <a name="storage-and-file-system"></a>Stockage et système de fichiers

-   Suivi des plages modifiées d’un fichier dans Windows 8.1
-   La nouvelle API permet aux applications d’envoyer des indicateurs « supprimer et annuler le mappage » sur des supports de stockage
-   MPIO (Multipath I/O) prend désormais en charge les blocs de demande de stockage étendu
-   Systèmes de fichiers résilients
-   Prise en charge des API du serveur de fichiers
-   Nouvelle fonctionnalité historique des fichiers
-   Le système d’exploitation contrôle désormais les lecteurs de disque optique

## <a name="usb"></a>USB

-   Prise en charge de la norme USB 3,0
