# <a name="windows-and-windows-server-compatibility-cookbook-windows-10-windows-8-windows-81-windows-server-2012-and-windows-server-2012-r2"></a>Guide de compatibilité Windows et Windows Server : Windows 10, Windows 8, Windows 8.1, Windows Server 2012 et Windows Server 2012 R2

## <a name="purpose"></a>Objectif

Dans le livre de *recettes*, nous fournissons des informations sur les modifications apportées aux nouvelles fonctionnalités des systèmes d’exploitation Windows 10, Windows 8, Windows 8.1, windows server 2012 et windows server 2012 R2. Nous vous proposons également des instructions pour vérifier la compatibilité de vos programmes existants et planifiés avec les nouveaux systèmes d’exploitation. Nous partons du principe que vous connaissez les versions précédentes de Windows.

Le contenu s’applique à :

-   Windows 10
-   Windows 8.1
-   Windows Server 2012 R2
-   Windows 8
-   Windows Server 2012
-   Windows 7
-   Windows Server 2008 R2
