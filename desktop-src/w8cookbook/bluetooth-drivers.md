# <a name="bluetooth-drivers"></a>Pilotes Bluetooth

Les fabricants de matériel et les entreprises doivent savoir que les pilotes Bluetooth et les packs de profils tiers ne seront pas migrés vers Windows 10 dans le cadre de la mise à niveau à partir de Windows 7, Windows 8 ou Windows 8.1. Reportez-vous à [cet article](/windows-hardware/design/component-guidelines/bluetooth) pour plus d’informations sur les modifications apportées à Bluetooth pour Windows 10.
