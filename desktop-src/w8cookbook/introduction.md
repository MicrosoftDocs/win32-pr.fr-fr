# <a name="introduction-compatibility-cookbook-for-windows"></a>Introduction (Guide de compatibilité pour Windows)

Windows 10, Windows 8, Windows 8.1 et Windows Server 2012 R2 introduisent les dernières plateformes de développement logiciel et de technologie de système d’exploitation pour une utilisation par les développeurs d’applications et les entreprises du monde entier. dans le cadre de l’amélioration de la sécurité, de la fiabilité, des performances et de l’expérience utilisateur des Windows, Microsoft a introduit de nombreuses nouvelles fonctionnalités, amélioré les fonctionnalités existantes et supprimé d’autres.

bien que l’objectif de Windows 10, Windows 8, Windows 8.1 et Windows Server 2012 R2 soit de rester hautement compatible avec la plupart de leurs applications respectives écrites pour les systèmes d’exploitation précédemment publiés, certains arrêts de compatibilité sont inévitables en raison d’innovations, d’une sécurité renforcée et d’une fiabilité accrue. Globalement, la compatibilité des nouveaux systèmes d’exploitation avec les applications existantes est élevée.

ce livre de *recettes* est divisé en trois sections principales, Windows 10, Windows 8.1 et Windows 8 :

-   Windows 10
-   Windows 8.1 et Windows Server 2012 R2
-   Windows 8 et Windows Server 2012

Les rubriques suivantes sont nouvelles dans cette version du livre de *recettes* ou ont été mises à jour :

-   Windows Démarrage du fichier image (WimBoot)
-   Kit de certification des applications Windows

nous vous invitons à consulter ces rubriques pour savoir comment optimiser vos applications et tirer parti des fonctionnalités de cette version de Windows la plus récente.
