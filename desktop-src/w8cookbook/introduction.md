# <a name="introduction-compatibility-cookbook-for-windows"></a>Introduction (Guide de compatibilité pour Windows)

Windows 10, Windows 8, Windows 8.1 et Windows Server 2012 R2 présentent la dernière technologie de système d’exploitation et les plateformes de développement logiciel pour une utilisation par les développeurs d’applications et les entreprises du monde entier. Dans le cadre de l’amélioration de la sécurité, de la fiabilité, des performances et de l’expérience utilisateur de Windows, Microsoft a introduit de nombreuses nouvelles fonctionnalités, amélioré les fonctionnalités existantes et supprimé d’autres.

Bien que l’objectif de Windows 10, Windows 8, Windows 8.1 et Windows Server 2012 R2 soit de rester hautement compatible avec la plupart de leurs applications respectives écrites pour les systèmes d’exploitation précédemment publiés, certains arrêts de compatibilité sont inévitables en raison des innovations, de la sécurité renforcée et de la fiabilité accrue. Globalement, la compatibilité des nouveaux systèmes d’exploitation avec les applications existantes est élevée.

Ce livre de *recettes* est divisé en trois sections principales, Windows 10, Windows 8.1 et Windows 8 :

-   Windows 10
-   Windows 8.1 et Windows Server 2012 R2
-   Windows 8 et Windows Server 2012

Les rubriques suivantes sont nouvelles dans cette version du livre de *recettes* ou ont été mises à jour :

-   Démarrage du fichier image Windows (WimBoot)
-   Kit de certification des applications Windows

Nous vous invitons à consulter ces rubriques pour savoir comment optimiser vos applications et tirer parti de ce que cette version plus récente de Windows a à offrir.

Les éditions de Windows Compatibility Cookbook qui répondent aux versions antérieures des systèmes d’exploitation Windows et Windows Server sont disponibles dans [Windows Vista et Windows server 2008](/previous-versions/bb757005(v=msdn.10)) et Windows [7 et Windows Server 2008 R2 application Quality Cookbook](/windows/desktop/Win7AppQual/windows-7-application-quality-cookbook) .
