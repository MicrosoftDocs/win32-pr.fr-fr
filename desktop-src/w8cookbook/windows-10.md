# <a name="windows-10"></a>Windows 10

ce livre de recettes fournit aux développeurs des informations sur les modifications apportées aux plateformes pour les systèmes d’exploitation Windows 10. Nous vous proposons également des instructions pour vérifier la compatibilité de vos applications existantes et planifiées avec la dernière version du système d’exploitation. Nous partons du principe que vous connaissez les versions précédentes de Windows.

Le contenu s’applique à : Windows 10

le livre de recettes est destiné aux développeurs tiers d’applications et d’appareils conçus pour être utilisés dans l’environnement Microsoft Windows.

## <a name="introduction"></a>Introduction

Windows 10 présente les nouvelles plateformes de développement logiciel et de technologie de système d’exploitation pour une utilisation par les développeurs d’applications et de pilotes dans le monde entier. pour améliorer la sécurité, la fiabilité, les performances et l’expérience utilisateur des Windows, Microsoft a introduit de nombreuses nouvelles fonctionnalités, amélioré les fonctionnalités existantes et supprimé d’autres.

bien que l’objectif de Windows 10 est de maintenir la compatibilité avec les applications écrites pour les versions antérieures du système d’exploitation Windows, certains arrêts de compatibilité sont inévitables en raison des innovations, de la sécurité renforcée et de la fiabilité accrue. globalement, la compatibilité de la dernière version du système d’exploitation Windows avec les applications et les appareils existants est élevée.

Ce document vous montre comment vérifier que vos applications et appareils sont compatibles avec la dernière version du système d’exploitation et fournit une vue d’ensemble des problèmes d’incompatibilité d’applications connus.

nous vous invitons à consulter ces rubriques pour savoir comment optimiser vos applications et tirer parti des fonctionnalités de cette version de Windows la plus récente.
