# <a name="windows-app-certification-kit"></a>Kit de certification des applications Windows

## <a name="platforms"></a>Plateformes

**Clients-** Windows 7 \| Windows 8 \| Windows 8.1  


## <a name="description"></a>Description

Windows le Kit de Certification des applications 3,1 est inclus dans le SDK Windows pour Windows 8.1. le kit peut être utilisé pour précertifier les applications de Windows Store pour Windows 8 et Windows 8.1 avant l’intégration, ainsi que pour les programmes de Certification Windows 7, Windows 8 et Windows 8.1 et les applications Windows 8.1 Téléphone.

le Kit de Certification des applications Windows 3,1 a été mis à jour pour fournir aux développeurs une expérience utilisateur transparente, y compris l’exécution de tests en parallèle pour économiser du temps global, sélectionner des tests sélectifs et améliorer la création de rapports, pour nommer quelques-unes des améliorations.

**Remarque :** quand vous êtes invité à effectuer une mise à niveau, veuillez mettre à niveau votre version de Windows kit de Certification des applications pour tirer parti des dernières fonctionnalités incluses dans le kit. lorsque votre application est testée par rapport à la dernière version de Windows Kit de certification des applications, il est assuré d’améliorer la conformité avec les stratégies de Windows Store et les programmes de certification de bureau.
