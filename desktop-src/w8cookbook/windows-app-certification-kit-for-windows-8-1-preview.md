# <a name="windows-app-certification-kit"></a>Kit de certification des applications Windows

## <a name="platforms"></a>Plateformes

**Clients-** Windows 7 \| Windows 8.1 Windows 8 \|  


## <a name="description"></a>Description

Le kit de certification des applications Windows 3,1 est inclus dans le SDK Windows pour Windows 8.1. Le kit peut être utilisé pour précertifier les applications du Windows Store pour Windows 8 et Windows 8.1 avant l’intégration, ainsi que pour les programmes de certification des applications de bureau Windows 7, Windows 8 et Windows 8.1 et les applications Windows 8.1 Phone.

Le kit de certification des applications Windows 3,1 a été mis à jour pour fournir aux développeurs une expérience utilisateur transparente, y compris l’exécution de tests en parallèle pour économiser du temps global, sélectionner des tests sélectifs et améliorer la création de rapports, pour nommer quelques-unes des améliorations.

**Remarque :** Quand vous êtes invité à effectuer une mise à niveau, veuillez mettre à niveau votre version du kit de certification des applications Windows pour tirer parti des dernières fonctionnalités incluses dans le kit. Lorsque votre application est testée par rapport à la dernière version du kit de certification des applications Windows, il est assuré de mieux respecter les stratégies du Windows Store et les programmes de certification de bureau.
