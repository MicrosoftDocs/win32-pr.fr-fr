# <a name="systemmonitorshowverticalgrid-property"></a>SystemMonitor. ShowVerticalGrid, propriété

Récupère ou définit une valeur qui détermine si les lignes verticales du quadrillage s’affichent dans le graphique.

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```VB
Property ShowVerticalGrid As Boolean
```



## <a name="property-value"></a>Valeur de la propriété

True indique que le quadrillage vertical est affiché dans le graphique ; Sinon, false. La valeur par défaut pour cette propriété est False.
