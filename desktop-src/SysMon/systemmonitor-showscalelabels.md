# <a name="systemmonitorshowscalelabels-property"></a>SystemMonitor. ShowScaleLabels, propriété

Récupère ou définit une valeur qui détermine si les étiquettes d’échelle sont affichées sur l’axe vertical du graphique.

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```VB
Property ShowScaleLabels As Boolean
```



## <a name="property-value"></a>Valeur de la propriété

True indique que les étiquettes d’échelle sont affichées sur l’axe vertical du graphique ; Sinon, false. La valeur par défaut de cette propriété est True.
