# <a name="systemmonitorshowhorizontalgrid-property"></a>SystemMonitor. ShowHorizontalGrid, propriété

Récupère ou définit une valeur qui détermine si les lignes de grille horizontales sont affichées dans le graphique.

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```VB
Property ShowHorizontalGrid As Boolean
```



## <a name="property-value"></a>Valeur de la propriété

True indique que le quadrillage horizontal est affiché dans le graphique ; Sinon, false. La valeur par défaut pour cette propriété est False.
