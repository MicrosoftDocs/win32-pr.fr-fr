# <a name="systemmonitorshowlegend-property"></a>SystemMonitor. ShowLegend, propriété

Récupère ou définit une valeur qui détermine si la légende est affichée. La légende répertorie les compteurs et affiche la couleur utilisée pour représenter chaque compteur.

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```VB
Property ShowLegend As Boolean
```



## <a name="property-value"></a>Valeur de la propriété

True indique que la légende est affichée dans le contrôle ; Sinon, false. La valeur par défaut de cette propriété est True.
