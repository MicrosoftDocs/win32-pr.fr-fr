# <a name="logfileitem-object"></a>Objet LogFileItem

Utilisez cette classe pour récupérer le chemin d’accès d’un fichier journal auquel accède le contrôle du moniteur système.

Pour récupérer cet objet, appelez [**LogFiles. Item**](systemmonitor-logfiles-item.md).

## <a name="members"></a>Membres

L’objet **LogFileItem** possède les types de membres suivants :

-   [Propriétés](#properties)

### <a name="properties"></a>Propriétés

L’objet **LogFileItem** a ces propriétés.



| Propriété                                    | Description                                    |
|:--------------------------------------------|:-----------------------------------------------|
| [**D**](logfileitem-path.md)<br/> | Récupère le chemin d’accès du fichier journal.<br/> |
