# <a name="systemmonitorbackcolorctl-property"></a>SystemMonitor. BackColorCtl, propriété

Récupère ou définit la couleur d’arrière-plan du contrôle.

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```VB
Property BackColorCtl As stdole.OLE_COLOR
```



## <a name="property-value"></a>Valeur de la propriété

Couleur d’arrière-plan du contrôle.
