# <a name="systemmonitorcopy-method"></a>SystemMonitor :: Copy, méthode

Copie les paramètres de propriété, la liste de compteurs et les données de compteur du contrôle dans le presse-papiers sous la forme d’un objet HTML.

## <a name="syntax"></a>Syntaxe


```VB
Sub Copy()
```



## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.

## <a name="return-value"></a>Valeur de retour

Cette méthode ne retourne pas de valeur.

## <a name="remarks"></a>Notes

Une seule copie peut exister dans le presse-papiers.
