# <a name="systemmonitorshowtoolbar-property"></a>SystemMonitor. ShowToolbar, propriété

Récupère ou définit une valeur qui détermine si la barre d’outils est affichée sur le contrôle.

## <a name="syntax"></a>Syntaxe


```VB
Property ShowToolbar As Boolean
```



## <a name="property-value"></a>Valeur de la propriété

True indique que la barre d’outils est affichée sur le contrôle ; Sinon, false. La valeur par défaut de cette propriété est True.

## <a name="remarks"></a>Notes

La barre d’outils permet à l’utilisateur d’ajouter ou de supprimer des compteurs, de choisir le type de graphique utilisé pour afficher les données de compteur et de modifier d’autres propriétés du contrôle.
