# <a name="systemmonitorreset-method"></a>SystemMonitor :: Reset, méthode

Supprime tous les objets [**CounterItem**](counteritem.md) de l’objet collection de **compteurs** et efface les vues graphique et rapport.

## <a name="syntax"></a>Syntaxe


```VB
Sub Reset()
```



## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.

## <a name="return-value"></a>Valeur retournée

Cette méthode ne retourne pas de valeur.
