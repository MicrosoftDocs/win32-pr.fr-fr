# <a name="systemmonitorgridcolor-property"></a>SystemMonitor. GridColor, propriété

Récupère ou définit la couleur des lignes de la grille utilisées dans le graphique.

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```VB
Property GridColor As stdole.OLE_COLOR
```



## <a name="property-value"></a>Valeur de la propriété

Couleur du quadrillage horizontal et vertical utilisé dans un graphique si les propriétés [**ShowHorizontalGrid**](systemmonitor-showhorizontalgrid.md) ou [**ShowVerticalGrid**](systemmonitor-showverticalgrid.md) sont sélectionnées.
