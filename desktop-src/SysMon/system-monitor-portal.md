# <a name="system-monitor"></a>Moniteur système

## <a name="purpose"></a>Objectif

le moniteur système (SYSMON) est l’interface de programmation d’applications (API) que vous utilisez pour configurer le contrôle ActiveX du moniteur système Microsoft. Le contrôle du moniteur système vous permet d’afficher les données du compteur de performances en temps réel et journalisées précédemment, telles que la mémoire, le disque et les données des compteurs de processeur.

## <a name="where-applicable"></a>Le cas échéant

utilisez l’API SYSMON lorsque vous ajoutez le contrôle moniteur système à un conteneur ActiveX, tel qu’un document Microsoft Word ou une page web affichée par Internet Explorer, et que vous souhaitez modifier son comportement par défaut. Par exemple, vous pouvez créer un analyseur de performances qui affiche uniquement un ensemble prédéfini de compteurs système ou des paramètres d’affichage prédéfinis.

## <a name="developer-audience"></a>Développeurs concernés

SYSMON est conçu pour être utilisé par les développeurs C, C++, Visual Basic et de script.

## <a name="run-time-requirements"></a>Conditions d’exécution

SYSMON est disponible à partir de Microsoft Windows 2000.

Pour plus d’informations sur les exigences d’exécution d’un élément de programmation particulier, consultez la section Configuration requise de la page de référence de cet élément.
