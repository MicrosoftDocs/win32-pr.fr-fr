# <a name="systemmonitorforecolor-property"></a>SystemMonitor. ForeColor, propriété

Récupère ou définit la couleur du texte qui apparaît dans le contrôle.

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```VB
Property ForeColor As stdole.OLE_COLOR
```



## <a name="property-value"></a>Valeur de la propriété

Couleur du texte qui apparaît dans le contrôle

## <a name="remarks"></a>Notes

Il s’agit d’une propriété ambiante. La valeur de cette propriété est déterminée par le conteneur. La définition de la valeur de cette propriété peut affecter l’illusion du contrôle et du conteneur en tant qu’application unique.
