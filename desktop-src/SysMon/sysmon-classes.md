# <a name="sysmon-classes"></a>Classes SYSMON

Vous pouvez utiliser les classes suivantes pour configurer le contrôle du moniteur système (SYSMON).



| Nom de classe                             | Description                                                                                                  |
|----------------------------------------|--------------------------------------------------------------------------------------------------------------|
| [**CounterItem**](counteritem.md)     | Fournit des informations sur les propriétés de l’élément de compteur.                                                                  |
| [**Counters**](counters.md)           | Fournit les méthodes de gestion d’une collection d’objets [**CounterItem**](counteritem.md) .                |
| [**LogFileItem**](logfileitem.md)     | Fournit une propriété pour récupérer le chemin d’accès d’un seul fichier journal auquel accède le contrôle du moniteur système.     |
| [**LogFiles**](logfiles.md)           | Fournit les méthodes et les propriétés permettant de gérer une collection d’objets [**LogFileItem**](logfileitem.md) . |
| [**SystemMonitor**](systemmonitor.md) | Fournit les méthodes et les propriétés utilisées pour configurer le contrôle du moniteur système.                            |
