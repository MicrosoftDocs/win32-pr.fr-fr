# <a name="systemmonitorlogsourcestarttime-property"></a>SystemMonitor. LogSourceStartTime, propriété

Récupère l’horodatage de la valeur de compteur la plus ancienne d’un compteur de votre collection de compteurs qui a été enregistré dans les fichiers journaux.

## <a name="syntax"></a>Syntaxe


```VB
Property LogSourceStartTime As Date
```



## <a name="property-value"></a>Valeur de la propriété

Horodatage de la valeur de compteur la plus ancienne d’un compteur de votre collection de compteurs qui a été consigné dans les fichiers journaux.
