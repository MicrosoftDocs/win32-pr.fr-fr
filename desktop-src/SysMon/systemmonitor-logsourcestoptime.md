# <a name="systemmonitorlogsourcestoptime-property"></a>SystemMonitor. LogSourceStopTime, propriété

Récupère l’horodatage de la dernière valeur de compteur d’un compteur de votre collection de compteurs qui a été enregistré dans les fichiers journaux.

## <a name="syntax"></a>Syntaxe


```VB
Property LogSourceStopTime As Date
```



## <a name="property-value"></a>Valeur de la propriété

Horodatage de la dernière valeur de compteur d’un compteur de votre collection de compteurs qui a été enregistré dans les fichiers journaux.
