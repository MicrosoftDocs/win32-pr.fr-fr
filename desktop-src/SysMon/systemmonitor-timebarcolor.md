# <a name="systemmonitortimebarcolor-property"></a>SystemMonitor. TimeBarColor, propriété

Récupère ou définit la couleur de la barre d’heure. La barre de temps est la barre verticale qui se déplace dans la fenêtre du graphique pour indiquer le passage de chaque intervalle d’échantillonnage dans la vue du graphique linéaire.

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```VB
Property TimeBarColor As stdole.OLE_COLOR
```



## <a name="property-value"></a>Valeur de la propriété

Couleur de la barre d’heure.
