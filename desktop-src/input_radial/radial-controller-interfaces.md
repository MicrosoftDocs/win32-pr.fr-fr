# <a name="radial-controller-interfaces"></a>Interfaces de contrôleur radial

Les rubriques contenues dans cette section fournissent les spécifications de référence pour les interfaces [d’entrée du contrôleur radial](radialcontroller-portal.md) .

## <a name="in-this-section"></a>Contenu de cette section

| Rubrique | Description |
|---|---|
| [**IRadialControllerConfigurationInterop**](/windows/win32/api/radialcontrollerinterop/nn-radialcontrollerinterop-iradialcontrollerconfigurationinterop)<br/> | permet l’interopérabilité avec plateforme Windows universelle un objet [**RadialControllerConfiguration**](/uwp/api/Windows.UI.Input.RadialControllerConfiguration) (UWP) et donne accès aux membres **RadialControllerConfiguration** pour la personnalisation d’un menu [**RadialController**](/uwp/api/Windows.UI.Input.RadialController) .<br/> |
| [**IRadialControllerInterop**](/windows/win32/api/radialcontrollerinterop/nn-radialcontrollerinterop-iradialcontrollerinterop)<br/> | Permet l’interopérabilité avec un objet [**RADIALCONTROLLER**](/uwp/api/Windows.UI.Input.RadialController) UWP et donne accès aux membres **RadialController** pour la personnalisation de l’expérience d’interaction.<br/> |
