# <a name="java-type-library-wizard"></a>Assistant bibliothèque de types Java

L’Assistant bibliothèque de types Java est un composant de Visual J++ 5,0. Cet Assistant génère des fichiers de classe et des informations de bibliothèque de types Java pour un objet COM.

**Pour générer des fichiers de classe Java et des informations sur la bibliothèque de types**

1.  Dans le menu **Outils** , cliquez sur **Assistant bibliothèque de types Java**. La liste des composants COM installés s’affiche.
2.  Activez les cases à cocher en regard des éléments que vous souhaitez convertir.
3.  Cliquez sur **OK**. L’Assistant bibliothèque de types génère des fichiers de classe pour les composants sélectionnés. Il imprime l’instruction **Import** requise pour insérer chaque composant dans une application ou une applet Java et un fichier texte résumé qui contient les informations de la bibliothèque de types Java pour l’objet.
