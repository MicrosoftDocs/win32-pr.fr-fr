# <a name="property-pages-and-property-sheets"></a>Pages de propriétés et feuilles de propriétés

Les pages de propriétés OLE permettent à un objet d’afficher ses propriétés dans une boîte de dialogue à onglets appelée feuille de propriétés. Un utilisateur final peut ensuite afficher et modifier les propriétés de l’objet. Un objet peut afficher ses pages de propriétés indépendamment de son client, ou le client peut gérer l’affichage des pages de propriétés à partir d’un certain nombre d’objets contenus dans une même feuille de propriétés. Les pages de propriétés fournissent également un moyen de notifier un client des modifications apportées aux propriétés d’un objet.

Tout objet qui souhaite fournir une interface utilisateur pour ses propriétés variables peut utiliser cette technologie.
