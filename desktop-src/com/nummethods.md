# <a name="nummethods"></a>NumMethods

Contient le nombre de méthodes dans l’interface associée, y compris les méthodes des interfaces dérivées.

## <a name="registry-entry"></a>Entrée de Registre

```
HKEY_LOCAL_MACHINE\SOFTWARE\Classes\Interface
   {IID}
      NumMethods = value
```

## <a name="remarks"></a>Remarques

Il s’agit d’une valeur de **reg \_ SZ** .
