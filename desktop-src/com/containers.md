# <a name="containers"></a>Conteneurs

Un conteneur de contrôles ActiveX est un conteneur OLE qui prend en charge les fonctionnalités supplémentaires suivantes :

-   Objets incorporés à partir de serveurs in-process
-   Activation sur place
-   OLEMISC \_ ACTIVATEWHENVISIBLE
-   Gestion des événements

Les conteneurs de contrôles ActiveX doivent assurer la prise en charge de toutes ces fonctionnalités.
