# <a name="containers"></a>Containers

un conteneur de contrôle ActiveX est un conteneur OLE qui prend en charge les fonctionnalités supplémentaires suivantes :

-   Objets incorporés à partir de serveurs in-process
-   Activation sur place
-   OLEMISC \_ ACTIVATEWHENVISIBLE
-   Gestion des événements

ActiveX Les conteneurs de contrôle doivent assurer la prise en charge de toutes ces fonctionnalités.
