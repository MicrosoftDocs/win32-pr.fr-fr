# <a name="translating-to-vbscript"></a>Conversion en VBScript

VBScript est un sous-ensemble du langage Microsoft Visual Basic pour Applications. Étant donné que VBScript est destiné à être utilisé en toute sécurité dans les applications côté client telles que les pages Web, il ne fournit pas d’entrée et de sortie de fichier, ni d’accès direct au système d’exploitation sous-jacent. Cela empêche un script basé sur le Web de supprimer ou de modifier des fichiers sur votre ordinateur. pour obtenir la liste complète des fonctionnalités de Visual Basic qui ne sont pas prises en charge dans VBScript, consultez la documentation Visual Basic pour Applications.

Outre les fonctionnalités de langage fournies par VBScript, l’hôte de script peut également exposer un modèle objet VBScript auquel votre script peut accéder. Par exemple, Internet Explorer expose un modèle objet qui permet aux scripts d’interagir avec le navigateur et de le contrôler par programmation. Pour plus d’informations sur l’utilisation de ces modèles objet, consultez la documentation de votre hôte de script.

-   [Conversion en VBScript à partir de JavaScript](translating-to-vbscript-from-javascript.md)
