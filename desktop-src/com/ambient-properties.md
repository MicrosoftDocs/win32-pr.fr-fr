# <a name="ambient-properties"></a>Propriétés ambiantes

au minimum, les conteneurs de contrôle ActiveX doivent prendre en charge les propriétés ambiantes suivantes à l’aide des dispid standard.



| Propriété ambiante             | Égal          | Commentaires/conditions                                                      |
|------------------------------|-----------------|--------------------------------------------------------------------------|
| LocaleID<br/>          | -705<br/> |   <br/>                                                          |
| UserMode<br/>          | -709<br/> | Pour les conteneurs qui ont des environnements d’utilisateur et d’exécution différents.<br/> |
| DisplayAsDefault <br/> | -713<br/> | Pour les conteneurs où un bouton par défaut est pertinent.<br/>      |
