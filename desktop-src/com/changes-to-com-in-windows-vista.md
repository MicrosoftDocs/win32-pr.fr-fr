# <a name="changes-to-com-in-windows-vista"></a>modifications apportées à COM dans Windows Vista

les modifications suivantes ont été apportées à COM dans Windows Vista :

-   Le moniker d’élévation COM autorise les applications qui s’exécutent sous un compte d’utilisateur limité (LUA) à activer des classes COM avec des fonctionnalités avec élévation de privilèges. Pour plus d’informations, consultez [le moniker d’élévation com](the-com-elevation-moniker.md).
-   lorsque vous installez une application qui effectue des appels COM sur un ordinateur Windows Vista et que vous envisagez d’exécuter l’application en tant qu’administrateur, vous devez installer l’application par ordinateur plutôt que par utilisateur, sinon elle ne fonctionnera pas correctement.
