# <a name="visual-basic-object-browser"></a>Visual Basic Explorateur d’objets

une fois qu’un objet COM a été ajouté à votre projet en tant que référence ou composant, vous pouvez utiliser l’explorateur d’objets Visual Basic pour afficher les informations de la bibliothèque de types pour cet objet dans Visual Basic.

Pour afficher un objet COM dans l’Explorateur d’objets :

-   Dans le menu **affichage** , cliquez sur **Explorateur d’objets**.

La partie gauche de la fenêtre de l’Explorateur d’objets affiche toutes les classes des références et des composants ajoutés au projet actif. La partie droite contient les méthodes, les propriétés et les événements de ces classes.

Vous pouvez filtrer les classes d’objets COM affichées en sélectionnant une bibliothèque d’objets dans la liste déroulante de sélection de la bibliothèque.

pour plus d’informations, consultez [exploration ActiveX bibliothèques de types de composants](/previous-versions/visualstudio/visual-basic-6/aa240782(v=vs.60)) dans la documentation de Visual Basic.

dans certains cas, l’explorateur d’objets Visual Basic s’affiche pour ne pas afficher de type de retour pour une fonction ou une propriété. Aucun type de retour n’apparaît lorsque le type de retour est le type par défaut, **Variant**. Si le volet Description de l’Explorateur d’objets décrit l’élément comme une fonction ou une propriété et qu’aucun type de retour n’est affiché, l’instruction de type de retour est « As Variant ».
