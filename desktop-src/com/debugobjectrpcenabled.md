# <a name="debugobjectrpcenabled"></a>DebugObjectRPCEnabled

**DebugObjectRPCEnabled** active le débogage com distant sur RPC. Si cette clé n’est pas définie sur le serveur, le débogage à distance est désactivé.

## <a name="registry-key"></a>Clé de Registre

**HKEY \_ LOCAL \_ MACHINE \\ SOFTWARE \\ Microsoft \\ Windows NT \\ CurrentVersion \\ DebugObjectRPCEnabled**



| Clé de Registre               | Description                       |
|----------------------------|-----------------------------------|
| [**AeDebug**](aedebug.md) | Stocke les options spécifiques au débogueur. |
