# <a name="simple-data-binding"></a>liaison de données simple

l’architecture des contrôles ActiveX définit un mécanisme de liaison de données, dans lequel un contrôle ActiveX peut spécifier qu’une ou plusieurs de ses propriétés peuvent être liées. Dans la plupart des cas, un contrôle lié aux données ne doit absolument pas exiger la liaison de données, afin qu’il puisse être inséré dans un conteneur qui ne prend pas en charge la liaison de données. Évidemment, dans une telle situation, les fonctionnalités du contrôle peuvent être réduites.
