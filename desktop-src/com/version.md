# <a name="version-com"></a>Version (COM)

Spécifie le numéro de version du contrôle.

## <a name="registry-entry"></a>Entrée de Registre

```
HKEY_LOCAL_MACHINE\SOFTWARE\Classes\CLSID
   {CLSID}
      Version = value
```

## <a name="remarks"></a>Remarques

Il s’agit d’une valeur de **reg \_ SZ** .

Le numéro de version doit correspondre à la version de la bibliothèque de types associée au contrôle.
