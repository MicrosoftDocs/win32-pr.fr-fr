# <a name="component-categories"></a>Catégories de composant

Les catégories de composants OLE permettent d’identifier les capacités et les exigences d’un composant logiciel par des entrées dans le registre. Dans un scénario où un conteneur peut ne pas être en mesure de prendre en charge un domaine de fonctionnalités, tel que DataBinding, par exemple, le conteneur ne souhaitera pas héberger des contrôles qui nécessitent une liaison de liaison pour exécuter sa fonction de travail. Les catégories de composants permettent d’identifier les zones de fonctionnalités telles que la liaison de type de liaison, afin que le conteneur de contrôle puisse éviter ces contrôles qui l’exigent. Les catégories de composants sont spécifiées séparément dans le cadre d’OLE et ne sont pas spécifiques à l’architecture du contrôle ActiveX, la spécification pour les catégories de composants comprend des API pour la manipulation des clés de registre de la catégorie de composants.
