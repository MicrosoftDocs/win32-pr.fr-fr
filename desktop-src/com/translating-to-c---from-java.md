# <a name="translating-to-c-from-java"></a>Conversion en C++ à partir de Java

Java gère implicitement les pointeurs et la gestion de la mémoire. En C++, votre application est chargée d’allouer et de libérer de la mémoire, et d’effectuer les opérations arithmétiques de pointeur nécessaires.
