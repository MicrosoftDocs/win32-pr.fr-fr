# <a name="service-identity"></a>Identité du service

Le choix de cette identité entraîne l’exécution de l’application en tant que service. Vous pouvez définir les comptes de service en choisissant l’option **services** dans le **panneau de configuration**. Pour plus d’informations, consultez [installation de en tant qu’application de service](installing-as-a-service-application.md).
