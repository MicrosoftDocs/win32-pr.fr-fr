# <a name="hkey_local_machinesoftwaremicrosoftwindows-ntcurrentversion"></a>HKEY \_ LOCAL \_ MACHINE \\ SOFTWARE \\ Microsoft \\ Windows NT \\ CurrentVersion

les sous-clés et les valeurs de registre associées à la clé de l' **\_ application HKEY LOCAL \_ MACHINE \\ SOFTWARE \\ Microsoft \\ Windows NT \\ CurrentVersion** contiennent des informations relatives à la fonctionnalité de débogage RPC COM.



| Sous-clé                                                 | Description                                         |
|--------------------------------------------------------|-----------------------------------------------------|
| [**DebugObjectRPCEnabled**](debugobjectrpcenabled.md) | Options de configuration pour le débogage distant COM RPC. |
