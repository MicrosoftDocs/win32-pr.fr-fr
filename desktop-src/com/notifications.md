# <a name="notifications-com"></a>Notifications (COM)

Les notifications sont des rappels générés par un objet lorsqu’il détecte une modification dans son nom, son état, ses données ou sa présentation. Les conteneurs et les autres clients requièrent des notifications pour répondre de manière appropriée à ces modifications. Un conteneur s’inscrit pour recevoir des notifications en définissant une connexion consultative à un objet d’intérêt. Les autres clients intéressés peuvent faire de même. Le conteneur crée également un récepteur de notifications pour recevoir les notifications. À l’aide de la connexion établie par le conteneur, un objet qui rencontre une modification notifie le récepteur de notifications. Lors de la réception d’une notification, le conteneur prend toute action définie pour le type de modification qui s’est produit.

Pour plus d'informations, voir les rubriques suivantes :

-   [Types de notifications](types-of-notifications.md)
-   [Fonctionnement des notifications](how-notifications-work.md)
