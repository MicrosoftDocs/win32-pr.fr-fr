# <a name="enhanced-metafiles"></a>Les sous-fichiers améliorés

Il n’est pas étonnant de faire en sorte que les sous-fichiers améliorés offrent davantage de fonctionnalités que les fichiers de fichier standard. l’utilisation de reformateurs améliorés simplifie généralement le code de rendu. Un contrôleur de périphérique de métafichier amélioré est utilisé exactement de la même façon qu’un DC de métafichier standard. OLE prend en charge les conversions améliorées et offre une compatibilité descendante avec les fichiers de fichiers standard.

les conteneurs de contrôle ActiveX doivent utiliser des fichiers de sous-fichiers améliorés à la place des fichiers de fichier standard.
