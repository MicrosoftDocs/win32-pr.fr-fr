# <a name="linking-and-embedding"></a>Liaison et incorporation

Les utilisateurs peuvent créer deux types d’objets de document composé : *liés* ou *incorporés*. La différence entre les deux types réside dans le mode de stockage des données sources de l’objet. L’emplacement où l’objet réside affecte, à son tour, la portabilité et les méthodes d’activation de l’objet, la façon dont les mises à jour des données sont effectuées, ainsi que la taille et la structure de son conteneur.

-   [Objets liés](linked-objects.md)
-   [Objets incorporés](embedded-objects.md)
