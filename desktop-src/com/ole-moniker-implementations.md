# <a name="ole-moniker-implementations"></a>Implémentations du moniker OLE

OLE fournit des implémentations de plusieurs monikers pour différentes situations, comme décrit dans les rubriques suivantes :

-   [Monikers de fichiers](file-monikers.md)
-   [Monikers composites](composite-monikers.md)
-   [Monikers d’élément](item-monikers.md)
-   [Anti-monikers](anti-monikers.md)
-   [Monikers de pointeur](pointer-monikers.md)
-   [Monikers de classe](class-monikers.md)
-   [Monikers asynchrones](asynchronous-monikers.md)
-   [Monikers d’URL](url-monikers.md)

Les monikers de fichiers, composites et d’éléments sont les monikers les plus fréquemment utilisés, car ils peuvent être utilisés pour créer quasiment n’importe quel objet dans n’importe quel emplacement. L’anti-moniker et les monikers de pointeur sont principalement utilisés dans OLE, mais ont une application dans l’implémentation des monikers personnalisés.
