# <a name="using-oleview"></a>Utilisation d’OleView

Alors que l’éditeur du Registre affiche une vue non traite du Registre et est utile pour l’apprentissage de l’organisation du Registre, l’utilisation de l’éditeur du Registre peut prendre du temps.

OleView fournit une vue de niveau supérieur des informations contenues dans le registre. Au lieu de consulter une longue liste de CLSID, OleView propose des contrôles d’arborescence avec des noms conviviaux.
