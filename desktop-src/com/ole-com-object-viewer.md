# <a name="ole-com-object-viewer"></a>Explorateur d’objets OLE-COM

L’Explorateur d’objets OLE-COM, Oleview.exe, est une application fournie avec Visual C++ qui affiche les objets COM installés sur votre ordinateur et les interfaces qu’ils prennent en charge. Vous pouvez utiliser cette visionneuse d’objets pour afficher les bibliothèques de types.

Pour afficher la bibliothèque de types d’un objet COM :

1.  Dans le menu **fichier** de l’Explorateur d’objets, cliquez sur **afficher la TypeLib**. Une boîte de dialogue **ouvrir** s’affiche.
2.  Spécifiez le fichier de bibliothèque de types que vous souhaitez ouvrir, puis cliquez sur **OK**.

L’Explorateur d’objets affiche les interfaces de l’objet.

Pour plus d’informations, consultez [utilisation de l’Explorateur d’objets OLE/com](/previous-versions/d0kh9f4c(v=vs.140)) dans la documentation Visual C++.
