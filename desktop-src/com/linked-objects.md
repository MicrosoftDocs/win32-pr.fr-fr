<span data-ttu-id="9b964-104">Lorsqu’un lien vers un objet est inséré dans un document composé, les données sources, ou la source de la *liaison*, continuent de résider partout où il a été initialement créé, généralement dans un autre document.</span><span class="sxs-lookup"><span data-stu-id="9b964-104">When a link to an object is inserted in a compound document, the source data, or *link source*, continues to reside wherever it was initially created, usually in another document.</span></span> <span data-ttu-id="9b964-105">Le document composé contient uniquement une référence ou un *lien* vers les données réelles stockées dans la source de la liaison, ainsi que des informations sur la façon de présenter ces données à l’utilisateur.</span><span class="sxs-lookup"><span data-stu-id="9b964-105">The compound document contains only a reference, or *link*, to the actual data stored at the link source, along with information about how to present that data to the user.</span></span>
