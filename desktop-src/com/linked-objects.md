# <a name="linked-objects"></a>Objets liés

Lorsqu’un lien vers un objet est inséré dans un document composé, les données sources, ou la source de la *liaison*, continuent de résider partout où il a été initialement créé, généralement dans un autre document. Le document composé contient uniquement une référence ou un *lien* vers les données réelles stockées dans la source de la liaison, ainsi que des informations sur la façon de présenter ces données à l’utilisateur.

L’activation d’un lien exécute l’application serveur de la source de liaison, dont l’utilisateur a besoin pour modifier ou manipuler les données de liaison. La liaison permet de conserver la taille d’un document composé en minuscules. Elle est également utile lorsque la source de données est gérée par une autre personne et qu’elle doit être partagée entre plusieurs utilisateurs. Si la personne qui gère la source de liaison modifie les données, la modification est automatiquement mise à jour dans tous les documents contenant un lien vers ces données. En plus de créer des liens simples, les utilisateurs peuvent imbriquer des liens et combiner des objets liés et incorporés pour créer des documents complexes.
