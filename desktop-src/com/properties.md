# <a name="exposing-properties-through-idispatch"></a>Exposition des propriétés via IDispatch

Bien que la plupart des contrôles aient des propriétés, les contrôles ne sont pas requis pour exposer des propriétés et, par conséquent, le contrôle ne requiert pas **IDispatch**. Si le contrôle a des propriétés, il n’existe aucune condition requise pour les propriétés qu’un contrôle doit exposer.
