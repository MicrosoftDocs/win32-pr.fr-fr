# <a name="aedebug"></a>AeDebug

**AeDebug** définit les options de débogage à distance pour com.

## <a name="registry-key"></a>Clé de Registre

**HKEY \_ LOCAL \_ MACHINE \\ SOFTWARE \\ Microsoft \\ Windows NT \\ CurrentVersion \\ DebugObjectRPCEnabled \\ AeDebug**



| Valeur de Registre               | Description                                           |
|------------------------------|-------------------------------------------------------|
| [**Automatique**](auto.md)         | Détermine si le débogueur est lancé automatiquement. |
| [**Débogueur**](debugger.md) | Stocke le nom du débogueur.                             |
