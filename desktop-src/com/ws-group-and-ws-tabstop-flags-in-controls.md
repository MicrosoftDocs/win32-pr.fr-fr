# <a name="ws_group-and-ws_tabstop-flags-in-controls"></a>\_Indicateurs WS Group et WS \_ TABSTOP dans les contrôles

Un contrôle ne doit pas utiliser les \_ indicateurs WS Group et WS \_ TABSTOP en interne ; certains conteneurs s’appuient sur ces indicateurs pour gérer la gestion du clavier.
