# <a name="channel-com"></a>Canal (COM)

Le canal est chargé de transmettre tous les messages entre le client et l’objet à travers la limite du processus. Le canal a été conçu pour fonctionner de manière transparente avec différents types de canaux, est compatible avec l’appel de procédure distante standard de l’ETCD OSF et prend en charge les applications mono et multithread.
