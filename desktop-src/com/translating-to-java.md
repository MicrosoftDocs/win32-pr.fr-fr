# <a name="translating-to-java"></a>Conversion en Java

Le système de développement Microsoft Visual J++ pour Java fournit deux outils que vous pouvez utiliser pour générer des informations sur la bibliothèque de types pour un objet COM dans la syntaxe Java, l’Assistant bibliothèque de types Java et l’outil en ligne de commande JActiveX. À l’aide de l’un de ces outils, vous pouvez générer des fichiers de classe Java pour un objet COM.

Dans Visual J++ version 6,0, l’outil JActiveX remplace l’outil en ligne de commande JavaTLB. Pour plus d’informations, consultez la documentation de Visual J++.

Si vous utilisez un environnement de développement Java autre que Visual J++, consultez la documentation de ce produit pour plus d’informations sur les outils de conversion COM.

Une fois que vous avez généré les fichiers de classe d’un composant, vous pouvez ajouter les classes de composant à votre application Java à l’aide de l’instruction **Import** .

Vous pouvez ensuite créer et utiliser les classes de l’objet dans votre application Java. Pour plus d’informations sur l’instruction **Import** Java, consultez une référence sur le langage de programmation Java.

Pour plus d'informations, voir les rubriques suivantes :
