# <a name="access-flags"></a>Indicateurs d’accès

Ces valeurs indiquent si une entrée de la liste d’accès décrit les droits qui sont autorisés ou refusés.



| Constante/valeur                                                                                                                                                                                                                                      | Description                                   |
|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:----------------------------------------------|
| <span id="ACTRL_ACCESS_ALLOWED"></span><span id="actrl_access_allowed"></span><dl> <dt>**ACTRL \_ ACCÈS \_ autorisé**</dt> <dt>0x00000000</dt> </dl> | Indique une entrée d’autorisation d’accès.<br/> |
| <span id="ACTRL_ACCESS_DENIED"></span><span id="actrl_access_denied"></span><dl> <dt>**ACTRL \_ ACCÈS \_ refusé**</dt> <dt>0x10000000</dt> </dl>    | Indique une entrée de refus d’accès.<br/>  |
