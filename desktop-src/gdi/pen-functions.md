# <a name="pen-functions-windows-gdi"></a>Fonctions de stylet (Windows GDI)

Les fonctions suivantes sont utilisées avec les stylets.



| Fonction                                       | Description                                                                                            |
|------------------------------------------------|--------------------------------------------------------------------------------------------------------|
| [**CreatePen**](/windows/desktop/api/Wingdi/nf-wingdi-createpen)                 | Crée un stylet logique qui a le style, la largeur et la couleur spécifiés.                                  |
| [**CreatePenIndirect**](/windows/desktop/api/Wingdi/nf-wingdi-createpenindirect) | Crée un stylet cosmétique logique dont le style, la largeur et la couleur sont spécifiés dans une structure.          |
| [**ExtCreatePen**](/windows/desktop/api/Wingdi/nf-wingdi-extcreatepen)           | Crée un stylet cosmétique ou géométrique logique qui a le style, la largeur et les attributs de pinceau spécifiés. |
| [**SetDCPenColor**](/windows/desktop/api/Wingdi/nf-wingdi-setdcpencolor)         | Définit la couleur actuelle du stylet du contexte de périphérique.                                                             |
