# <a name="icm-enabled-brush-functions"></a>ICM-Enabled les fonctions de pinceau

La gestion des couleurs ICM (Microsoft Image Color Management) garantit qu’une image de couleur, un graphique ou un objet texte est rendu le plus près possible de son intention d’origine sur n’importe quel appareil, malgré les différences entre les technologies d’imagerie et les fonctionnalités de couleur entre les appareils. Si vous Scannez une image ou un autre graphique sur un scanneur de couleurs, que vous la téléchargez sur Internet, que vous l’affichez ou que vous la modifiez à l’écran ou que vous la générez sur papier, film ou autre média, ICM 2,0 vous aide à conserver des couleurs cohérentes et précises. Pour plus d’informations sur ICM, consultez [système de couleurs Windows](/previous-versions//dd372446(v=vs.85)).

Les fonctions pinceau suivantes sont activées pour une utilisation avec ICM :

-   [**CreateBrushIndirect**](/windows/desktop/api/Wingdi/nf-wingdi-createbrushindirect)
-   [**CreateDIBPatternBrush**](/windows/desktop/api/Wingdi/nf-wingdi-createdibpatternbrush)
-   [**CreateDIBPatternBrushPt**](/windows/desktop/api/Wingdi/nf-wingdi-createdibpatternbrushpt)
-   [**CreateHatchBrush**](/windows/desktop/api/Wingdi/nf-wingdi-createhatchbrush)
-   [**CreatePatternBrush**](/windows/desktop/api/Wingdi/nf-wingdi-createpatternbrush)
-   [**CreateSolidBrush**](/windows/desktop/api/Wingdi/nf-wingdi-createsolidbrush)
