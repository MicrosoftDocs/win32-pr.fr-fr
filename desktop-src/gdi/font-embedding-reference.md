# <a name="font-embedding-reference"></a>Référence d’incorporation de police

La bibliothèque de services d’incorporation de polices fournit un mécanisme permettant de regrouper des polices TrueType et Microsoft OpenType dans un document ou un fichier. En règle générale, un document contenant des polices incorporées a besoin des polices pour le rendu du document sur un autre ordinateur. L’incorporation d’une police garantit qu’une police spécifiée dans un fichier sera présente sur l’ordinateur recevant le fichier. Certaines polices, toutefois, ne peuvent pas être déplacées vers d’autres ordinateurs en raison de problèmes de copyright qui limitent la distribution.

Les éléments suivants sont utilisés avec les services d’incorporation de polices.

-   [Fonctions d’incorporation de police](font-embedding-functions.md)
-   [Structures d’incorporation de polices](font-embedding-services-structures.md)
-   [Message d’erreur d’incorporation de police](font-embedding-function-error-messages.md)
