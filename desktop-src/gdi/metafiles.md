# <a name="metafiles-windows-gdi"></a>fichiers (Windows GDI)

Un *métafichier* est une collection de structures qui stockent une image dans un format indépendant du périphérique. L’indépendance des appareils est l’une des fonctionnalités qui désignent les sous-fichiers en dehors des bitmaps. Contrairement à une image bitmap, un métafichier garantit l’indépendance des appareils. Toutefois, il existe un inconvénient pour les sous-fichiers, mais ils sont généralement plus lents que les bitmaps. Par conséquent, si une application requiert un dessin rapide et que l’indépendance de l’appareil n’est pas un problème, elle doit utiliser des bitmaps au lieu de fichiers.

-   [Utilisation des fichiers de sous-fichier](using-metafiles.md)
-   [Référence du métafichier](metafile-reference.md)

Pour plus d’informations sur les bitmaps, consultez [bitmaps](bitmaps.md).
