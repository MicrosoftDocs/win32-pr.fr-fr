# <a name="color-approximations-and-dithering"></a><span data-ttu-id="f381a-103">Approximations de couleurs et tramage</span><span class="sxs-lookup"><span data-stu-id="f381a-103">Color Approximations and Dithering</span></span>

<span data-ttu-id="f381a-104">Bien qu’une application puisse utiliser la couleur sans tenir compte des capacités de couleur de l’appareil, le résultat obtenu peut ne pas être aussi instructif et agréable que la sortie pour laquelle la couleur est choisie soigneusement.</span><span class="sxs-lookup"><span data-stu-id="f381a-104">Although an application can use color without regard to the color capabilities of the device, the resulting output may not be as informative and pleasing as output for which color is carefully chosen.</span></span> <span data-ttu-id="f381a-105">Peu, le cas échéant, les appareils garantissent une correspondance exacte pour chaque valeur de couleur possible ; par conséquent, si une application demande une couleur que l’appareil ne peut pas générer, le système se rapproche de cette couleur à l’aide d’une couleur que l’appareil peut générer.</span><span class="sxs-lookup"><span data-stu-id="f381a-105">Few, if any, devices guarantee an exact match for every possible color value; therefore, if an application requests a color that the device cannot generate, the system approximates that color by using a color that the device can generate.</span></span> <span data-ttu-id="f381a-106">Par exemple, si une application tente de créer un stylet rouge pour une imprimante noir et blanc, elle reçoit un stylet noir. le système utilise alors le noir comme approximation pour le rouge.</span><span class="sxs-lookup"><span data-stu-id="f381a-106">For example, if an application attempts to create a red pen for a black and white printer, it will receive a black pen instead the system uses black as the approximation for red.</span></span>

<span data-ttu-id="f381a-107">Une application peut déterminer si le système doit arrondir une couleur donnée à l’aide de la fonction [**GetNearestColor**](/windows/desktop/api/Wingdi/nf-wingdi-getnearestcolor) .</span><span class="sxs-lookup"><span data-stu-id="f381a-107">An application can discover whether the system will approximate a given color by using the [**GetNearestColor**](/windows/desktop/api/Wingdi/nf-wingdi-getnearestcolor) function.</span></span> <span data-ttu-id="f381a-108">La fonction prend une valeur de couleur et retourne la valeur de couleur de la couleur de correspondance la plus proche que l’appareil peut générer.</span><span class="sxs-lookup"><span data-stu-id="f381a-108">The function takes a color value and returns the color value of the closest matching color the device can generate.</span></span> <span data-ttu-id="f381a-109">La méthode utilisée par le système pour déterminer cette approximation dépend du pilote de périphérique et de ses fonctionnalités de couleur.</span><span class="sxs-lookup"><span data-stu-id="f381a-109">The method the system uses to determine this approximation depends on the device driver and its color capabilities.</span></span> <span data-ttu-id="f381a-110">Dans la plupart des cas, l’intensité globale de la couleur approximative est la plus proche de celle de la couleur demandée.</span><span class="sxs-lookup"><span data-stu-id="f381a-110">In most cases, the approximated color's overall intensity is closest to that of the requested color.</span></span>

<span data-ttu-id="f381a-111">Quand une application crée un stylet ou définit la couleur du texte, le système se rapproche toujours d’une couleur si aucune correspondance exacte n’existe.</span><span class="sxs-lookup"><span data-stu-id="f381a-111">When an application creates a pen or sets the color for text, the system always approximates a color if no exact match exists.</span></span> <span data-ttu-id="f381a-112">Quand une application crée un pinceau plein, le système peut tenter de simuler la couleur demandée par le biais du tramage.</span><span class="sxs-lookup"><span data-stu-id="f381a-112">When an application creates a solid brush, the system may attempt to simulate the requested color by dithering.</span></span> <span data-ttu-id="f381a-113">Le *tramage* simule pattern.</span></span> <span data-ttu-id="f381a-114">Par exemple, différentes nuances de rose peuvent être simulées en alternant différentes combinaisons de rouge et de blanc.</span><span class="sxs-lookup"><span data-stu-id="f381a-114">For example, different shades of pink can be simulated by alternating different combinations of red and white.</span></span> <span data-ttu-id="f381a-115">Selon les couleurs et le modèle, le tramage peut produire des simulations raisonnables.</span><span class="sxs-lookup"><span data-stu-id="f381a-115">Depending on the colors and the pattern, dithering can produce reasonable simulations.</span></span> <span data-ttu-id="f381a-116">Elle est particulièrement utile pour les appareils monochromes, car elle augmente le nombre de « couleurs » disponibles au-delà du noir et du blanc simples.</span><span class="sxs-lookup"><span data-stu-id="f381a-116">It is most useful for monochrome devices, because it expands the number of available "colors" well beyond simple black and white.</span></span>

<span data-ttu-id="f381a-117">La méthode utilisée pour créer des couleurs dépendantes dépend du pilote de périphérique.</span><span class="sxs-lookup"><span data-stu-id="f381a-117">The method used to create dithered colors depends on the device driver.</span></span> <span data-ttu-id="f381a-118">La plupart des pilotes de périphérique utilisent un algorithme de tramage standard, qui génère un peut pas être générée par l’appareil est sujette à la simulation, mais une application n’est pas avertie lorsque le système simule une couleur.</span><span class="sxs-lookup"><span data-stu-id="f381a-119">In general, any requested color that cannot be generated by the device is subject to simulation, but an application is not notified when the system simulates a
