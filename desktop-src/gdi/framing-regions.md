# <a name="framing-regions"></a>Régions de tramage

Une application dessine une bordure autour d’une région en appelant la fonction [**FrameRgn**](/windows/desktop/api/Wingdi/nf-wingdi-framergn) et en spécifiant la largeur de bordure et le motif de pinceau que le système utilise pour dessiner le frame.
