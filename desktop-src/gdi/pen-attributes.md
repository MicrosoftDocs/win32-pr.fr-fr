# <a name="pen-attributes"></a>Attributs PEN

Sept attributs PEN définissent le type de stylet et ses caractéristiques : la largeur, le style, la couleur, le modèle, le hachurage, le style de fin et le style de jointure. Les plumes cosmétiques et géométriques ont des attributs de largeur, de style et de couleur. Seuls les plumes géométriques ont les attributs Pattern, Hatch, end style et Join style. Les attributs pattern et Hatch facultatif sont généralement associés à un pinceau, mais peuvent également être utilisés avec des plumes géométriques.

Pour plus d'informations, voir les rubriques suivantes :

-   [Largeur du stylet](pen-width.md)
-   [Style de stylet](pen-style.md)
-   [Couleur du stylet](pen-color.md)
-   [Modèle Pen](pen-pattern.md)
-   [Hachurage du stylet](pen-hatch.md)
-   [Extrémité de fin du stylet](pen-end-cap.md)
-   [Jointure de stylet](pen-join.md)
