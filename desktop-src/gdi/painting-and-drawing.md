# <a name="painting-and-drawing"></a>Peinture et dessin

Cette vue d’ensemble décrit comment le système gère la sortie à l’écran et explique quelles applications doivent faire pour dessiner dans une fenêtre. En particulier, cette vue d’ensemble décrit les *contextes de périphérique d’affichage* (ou, plus simplement, les *contrôleurs* de l’affichage) et la façon de les préparer et de les utiliser. La vue d’ensemble n’explique pas comment utiliser les fonctions GDI (Graphics Device Interface) pour générer la sortie, ni expliquer comment imprimer.

-   [Utilisation du \_ message WM Paint](using-the-wm-paint-message.md)
-   [Utilisation de la fonction GetDC](using-the-getdc-function.md)
-   [Dessin et référence de dessin](painting-and-drawing-reference.md)
