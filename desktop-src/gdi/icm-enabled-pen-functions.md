# <a name="icm-enabled-pen-functions"></a>ICM-Enabled les fonctions de stylet

la gestion des couleurs de l’image Microsoft (ICM) garantit qu’une image de couleur, un graphique ou un objet texte est rendu le plus près possible de son intention d’origine sur n’importe quel appareil, en dépit des différences entre les technologies d’imagerie et les capacités de couleur entre les appareils. que vous scanniez une image ou d’autres graphiques sur un scanneur de couleurs, que vous la téléchargeiez sur Internet, que vous la visualisiez ou la modifiiez à l’écran, que vous la déformiez sur du papier, sur un film ou sur un autre média, ICM la version 2,0 vous aide à conserver les couleurs cohérentes et précises. pour plus d’informations sur la ICM, consultez [Windows système de couleurs](/previous-versions//dd372446(v=vs.85))

Il existe différentes fonctions dans l’interface GDI (Graphics Device Interface) qui utilisent ou opèrent sur les données de couleur. Les fonctions de stylet suivantes sont activées pour une utilisation avec ICM :

-   [**CreatePen**](/windows/desktop/api/Wingdi/nf-wingdi-createpen)
-   [**ExtCreatePen**](/windows/desktop/api/Wingdi/nf-wingdi-extcreatepen)
