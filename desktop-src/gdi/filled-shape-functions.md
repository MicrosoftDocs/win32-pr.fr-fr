# <a name="filled-shape-functions"></a>Fonctions de forme remplie

Les fonctions suivantes sont utilisées avec des formes remplies.



| Fonction                           | Description                                                     |
|------------------------------------|-----------------------------------------------------------------|
| [**Chord**](/windows/desktop/api/Wingdi/nf-wingdi-chord)             | Dessine une zone délimitée par une ellipse et un segment de ligne.         |
| [**Es**](/windows/desktop/api/Wingdi/nf-wingdi-ellipse)         | Dessine une ellipse.                                               |
| [**FillRect**](/windows/desktop/api/Winuser/nf-winuser-fillrect)       | Remplit un rectangle à l’aide d’un pinceau.                                |
| [**FrameRect**](/windows/desktop/api/Winuser/nf-winuser-framerect)     | Dessine une bordure autour d’un rectangle à l’aide d’un pinceau.                |
| [**InvertRect**](/windows/desktop/api/Winuser/nf-winuser-invertrect)   | Inverse les valeurs de couleur des pixels dans un rectangle.          |
| [**Pie**](/windows/desktop/api/Wingdi/nf-wingdi-pie)                 | Dessine un secteur délimité par une ellipse et deux radiales. |
| [**Polygon**](/windows/desktop/api/Wingdi/nf-wingdi-polygon)         | Dessine un polygone.                                                |
| [**Polypolygone**](/windows/desktop/api/Wingdi/nf-wingdi-polypolygon) | dessine une série de polygones fermés.                              |
| [**Rectangle**](/windows/desktop/api/Wingdi/nf-wingdi-rectangle)     | Dessine un rectangle.                                              |
| [**RoundRect**](/windows/desktop/api/Wingdi/nf-wingdi-roundrect)     | Dessine un rectangle avec des angles arrondis.                         |
