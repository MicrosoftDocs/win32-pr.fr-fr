# <a name="line-and-curve-functions"></a>Fonctions de ligne et de courbe

Les fonctions suivantes sont utilisées avec les lignes et les courbes.



| Fonction                                   | Description                                                                                                   |
|--------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [**AngleArc**](/windows/desktop/api/Wingdi/nf-wingdi-anglearc)               | Dessine un segment de ligne et un arc.                                                                              |
| [**Arc**](/windows/desktop/api/Wingdi/nf-wingdi-arc)                         | Dessine un arc elliptique.                                                                                      |
| [**ArcTo**](/windows/desktop/api/Wingdi/nf-wingdi-arcto)                     | Dessine un arc elliptique.                                                                                      |
| [**GetArcDirection**](/windows/desktop/api/Wingdi/nf-wingdi-getarcdirection) | Récupère la direction d’arc actuelle pour le contexte de périphérique spécifié.                                         |
| [**LineDDA**](/windows/desktop/api/Wingdi/nf-wingdi-linedda)                 | Détermine les pixels à mettre en surbrillance pour une ligne définie par les points de départ et de fin spécifiés. |
| [**LineDDAProc**](/windows/desktop/api/Wingdi/nc-wingdi-lineddaproc)         | Fonction de rappel définie par l’application utilisée avec la fonction LineDDA.                                      |
| [**LineTo**](/windows/desktop/api/Wingdi/nf-wingdi-lineto)                   | Dessine une ligne à partir de la position actuelle jusqu’au point spécifié, mais sans l’inclure.                         |
| [**MoveToEx**](/windows/desktop/api/Wingdi/nf-wingdi-movetoex)               | Met à jour la position actuelle au point spécifié et retourne éventuellement la position précédente.             |
| [**Polybézier**](/windows/desktop/api/Wingdi/nf-wingdi-polybezier)           | Dessine une ou plusieurs courbes de Bézier.                                                                              |
| [**PolyBezierTo**](/windows/desktop/api/Wingdi/nf-wingdi-polybezierto)       | Dessine une ou plusieurs courbes de Bézier.                                                                              |
| [**Polydraw**](/windows/desktop/api/Wingdi/nf-wingdi-polydraw)               | Dessine un ensemble de segments de ligne et de courbes de Bézier.                                                               |
| [**Polyligne**](/windows/desktop/api/Wingdi/nf-wingdi-polyline)               | Dessine une série de segments de ligne en connectant les points du tableau spécifié.                              |
| [**PolylineTo**](/windows/desktop/api/Wingdi/nf-wingdi-polylineto)           | Dessine une ou plusieurs lignes droites.                                                                             |
| [**Polyligne**](/windows/desktop/api/Wingdi/nf-wingdi-polypolyline)       | Dessine plusieurs séries de segments de ligne connectés.                                                             |
| [**SetArcDirection**](/windows/desktop/api/Wingdi/nf-wingdi-setarcdirection) | Définit la direction de dessin à utiliser pour les fonctions d’arc et de rectangle.                                        |
