# <a name="logical-brush-types"></a>Types Brush logiques

Il existe quatre types de pinceaux logiques : [solide](solid-brush.md), [boursier](stock-brush.md), [hachuré](hatch-brush.md)et [motif](pattern-brush.md). Ces pinceaux sont présentés dans l’illustration suivante.

![Illustration montrant quatre cercles, chacun rempli d’un type de pinceau différent : solide, boursier, hachuré et motif](images/csbru-02.png)

Les types de bruts et de hachures ont chacun plusieurs pinceaux prédéfinis.

La fonction [**CreateBrushIndirect**](/windows/desktop/api/Wingdi/nf-wingdi-createbrushindirect) crée un pinceau logique avec un style, une couleur et un motif spécifiés.
