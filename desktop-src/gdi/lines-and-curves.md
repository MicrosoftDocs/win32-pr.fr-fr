# <a name="lines-and-curves"></a>Lignes et courbes

Les lignes et les courbes sont utilisées pour dessiner la sortie des graphiques sur les périphériques raster. Comme indiqué dans cette vue d’ensemble, une *ligne* est un ensemble de pixels en surbrillance sur un affichage raster (ou un ensemble de points sur une page imprimée) identifiée par deux points : un point de départ et un point de fin. Une *courbe régulière* est un ensemble de pixels en surbrillance sur un affichage raster (ou des points sur une page imprimée) qui définit le périmètre (ou une partie du périmètre) d’une section conique. Une *courbe irrégulière* est un ensemble de pixels qui définit une courbe qui ne correspond pas au périmètre d’une section conique.
