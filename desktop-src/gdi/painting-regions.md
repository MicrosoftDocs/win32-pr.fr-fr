# <a name="painting-regions"></a>Zones de dessin

Une application remplit l’intérieur d’une région à l’aide du pinceau actuellement sélectionné dans un contexte de périphérique par la fonction [**PaintRgn**](/windows/desktop/api/Wingdi/nf-wingdi-paintrgn) . Cette fonction utilise les modes de remplissage de polygone actuels (alternative et enroulement).
