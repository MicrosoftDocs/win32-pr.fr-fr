# <a name="moving-regions"></a>Déplacement vers d’autres régions

Une application déplace une région en appelant la fonction [**OffsetRgn**](/windows/desktop/api/Wingdi/nf-wingdi-offsetrgn) . Les décalages donnés le long de l’axe x et de l’axe y déterminent le nombre d’unités logiques à déplacer vers la gauche ou vers la droite, vers le haut ou vers le haut.
