# <a name="filled-shapes"></a>Formes pleines

Les *formes remplies* sont des formes géométriques qui sont délimitées à l’aide du stylet actuel et remplies à l’aide du pinceau actuel. Il existe cinq formes remplies :

-   Ellipse
-   Chord
-   Secteurs
-   Polygone
-   Rectangle

Cette vue d’ensemble décrit les formes remplies.

-   [Référence de forme remplie](filled-shape-reference.md)
