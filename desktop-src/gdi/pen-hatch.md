# <a name="pen-hatch"></a>Hachurage du stylet

L’attribut Hatch spécifie le type de hachurage d’un stylet géométrique avec l’attribut de modèle de hachurage. Six modèles sont disponibles. L’illustration suivante montre des lignes dessinées à l’aide de différents motifs de hachurage.
