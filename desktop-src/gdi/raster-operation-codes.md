# <a name="raster-operation-codes"></a>Codes d’opération Raster

Les codes d’opération Raster définissent la façon dont l’interface GDI (Graphics Device Interface) combine les bits du stylet sélectionné avec les bits dans le bitmap de destination.

Cette vue d’ensemble répertorie et décrit les opérations raster binaires et ternaires utilisées par GDI. Une [opération Raster binaire](binary-raster-operations.md) implique deux opérandes : un stylet et une bitmap de destination. Une [opération Raster ternaire](ternary-raster-operations.md) implique trois opérandes : une bitmap source, un pinceau et une bitmap de destination. Les opérations raster binaires et ternaires utilisent des opérateurs booléens.
