# <a name="regions-windows-gdi"></a>Régions (Windows GDI)

Une *région* est un rectangle, un polygone ou une ellipse (ou une combinaison de deux ou plusieurs de ces formes) qui peut être remplie, peint, inversé, encadré et utilisé pour effectuer un test de positionnement (test de l’emplacement du curseur).
