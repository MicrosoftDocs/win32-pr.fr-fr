# <a name="about-colors"></a>À propos des couleurs

La couleur peut être utilisée pour communiquer des idées, afficher les relations entre les éléments et améliorer l’attrait et la qualité de la sortie. Windows permet aux applications de découvrir les capacités de couleur des appareils fournis et de choisir parmi les couleurs disponibles celles qui répondent le mieux à leurs besoins.

Cette vue d’ensemble fournit des informations sur les rubriques suivantes :

-   [Notions de base des couleurs](color-basics.md)
-   [Palettes de couleurs](color-palettes.md)

Bien que cela ne soit pas décrit dans cette vue d’ensemble, la correspondance des couleurs d’images est une fonctionnalité importante de la gestion des couleurs qui permet de s’assurer que les images de couleur paraissent identiques, qu’elles soient affichées ou imprimées sur papier. pour plus d’informations, consultez [Windows système de couleurs](/previous-versions//dd372446(v=vs.85)).
