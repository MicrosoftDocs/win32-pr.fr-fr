# <a name="pen-width"></a>Largeur du stylet

L’attribut width spécifie une largeur de stylet cosmétique en unités de périphérique. Toutefois, en cas d’utilisation avec un stylet géométrique, il spécifie la largeur du stylet en unités logiques. Pour plus d’informations sur les unités de périphérique, consultez [espaces de coordonnées et transformations](coordinate-spaces-and-transformations.md).

Actuellement, le système limite la largeur des plumes cosmétiques à un seul pixel ; Toutefois, les versions ultérieures peuvent supprimer cette limitation.
