# <a name="multiple-display-monitors"></a>Moniteurs multiples

Plusieurs moniteurs d’affichage sont un ensemble de fonctionnalités associées qui permettent aux applications d’utiliser plusieurs appareils d’affichage en même temps. Il existe deux façons d’utiliser plusieurs moniteurs : un grand bureau ou un nombre d’écrans indépendants. Lorsqu’ils sont utilisés comme un grand bureau, les moniteurs créent davantage d’espace à l’écran pour les applications. Cela est utile lorsque vous avez besoin d’optimiser votre espace de travail à l’écran et qu’il est particulièrement utile pour la publication sur le bureau, le développement Web ou l’édition vidéo. Lorsqu’ils sont utilisés en tant qu’affichages indépendants, les moniteurs peuvent être utilisés pour jouer à des jeux, déboguer des applications plein écran, enseigner ou effectuer des présentations.
