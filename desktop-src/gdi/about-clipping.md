# <a name="about-clipping"></a>À propos du découpage

Le découpage est utilisé par les applications de différentes façons. Les applications de traitement de texte et de feuille de calcul découpent le clavier pour empêcher l’affichage des marges d’une page ou d’une feuille de calcul. La conception assistée par ordinateur (CAO) et les applications de dessin capturent la sortie graphique pour empêcher le remplacement des bords d’un dessin ou d’une image.

Une *zone de découpage* est une région avec des arêtes qui sont des lignes droites ou des courbes. Un *tracé de découpage* est une région avec des arêtes qui sont des lignes droites, des courbes de Bézier ou des combinaisons des deux. Pour plus d’informations sur les régions, consultez [régions](regions.md). Pour plus d’informations sur les chemins d’accès, consultez [chemins d’accès](paths.md).

Cette présentation couvre les rubriques suivantes :

-   [Tracés de clip](clip-paths.md)
