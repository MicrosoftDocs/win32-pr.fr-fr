# <a name="inverting-regions"></a>Inverser les régions

Une application inverse les couleurs qui apparaissent dans une région en appelant la fonction [**InvertRgn**](/windows/desktop/api/Wingdi/nf-wingdi-invertrgn) . Sur les écrans monochrome, **InvertRgn** fait en sorte que les pixels blancs sont noirs et noirs. Sur les écrans de couleur, cette inversion dépend du type de technologie utilisée pour générer les couleurs de l’écran.
