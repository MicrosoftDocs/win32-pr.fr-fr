# <a name="font-embedding-function-error-messages"></a>Font-Embedding des messages d’erreur de fonction

Les valeurs longues suivantes sont retournées par les fonctions d’incorporation de polices lorsque des erreurs sont rencontrées. Lorsque les fonctions réussissent, la valeur E \_ None est retournée.



| Valeur de retour                  | Description                                                                                                                                                                                                                                                                              |
|-------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| E \_ aucun                       | Pas d'erreur.                                                                                                                                                                                                                                                                                |
| \_ADDFONTFAILED E              | Une erreur s’est produite lorsque les fonctions de chargement ont essayé d’ajouter la nouvelle police à l’aide de [**AddFontResource**](/windows/desktop/api/Wingdi/nf-wingdi-addfontresourcea).                                                                                                                                                                    |
| \_CHARCODECOUNTINVALID E       | Le nombre de caractères sous-ensembles spécifiés dans [**TTEmbedFont**](/windows/desktop/api/T2embapi/nf-t2embapi-ttembedfont) n’est pas valide.                                                                                                                                                                                            |
| \_CHARCODESETINVALID E         | Le jeu de caractères spécifié dans [**TTEmbedFont**](/windows/desktop/api/T2embapi/nf-t2embapi-ttembedfont) n’est pas valide.                                                                                                                                                                                                            |
| \_COULDNTCREATETEMPFILE E      | Les fonctions de chargement n’ont pas pu créer un fichier temporaire nécessaire dans pour installer un nouveau fichier de polices ou de ressources.                                                                                                                                                                                   |
| \_DEVICETRUETYPEFONT E         | La police TrueType® spécifiée n’est pas une police système. La police peut exister en tant que police de périphérique dans une imprimante.                                                                                                                                                                                     |
| \_ERRORACCESSINGEXCLUDELIST E  | Une erreur s’est produite lors de la tentative d’accès à la liste d’exclusion de police.                                                                                                                                                                                                                |
| \_ERRORACCESSINGFACENAME E     | Une erreur non liée à un contrôleur de périphérique s’est produite lors de la tentative d’allocation d’une structure [**OUTLINETEXTMETRIC**](/windows/desktop/api/Wingdi/ns-wingdi-outlinetextmetrica) .                                                                                                                                                             |
| \_ERRORACCESSINGFONTDATA E     | Une erreur s’est produite lors de la tentative d’utilisation de [**GetFontData**](/windows/desktop/api/Wingdi/nf-wingdi-getfontdata).                                                                                                                                                                                                     |
| \_ERRORCOMPRESSINGFONTDATA E   | Une erreur s’est produite lors de la tentative de compression des données de police par [**TTEmbedFont**](/windows/desktop/api/T2embapi/nf-t2embapi-ttembedfont) .                                                                                                                                                                                          |
| \_ERRORCONVERTINGCHARS E       | Une erreur a empêché la conversion d’une chaîne de caractères codés sur un octet en caractères Unicode. Cela peut se produire dans [**TTCharToUnicode**](/windows/desktop/api/T2embapi/nf-t2embapi-ttchartounicode) si *pucCharCodes* ou *pusShortCodes* sont des valeurs non null, ou si la conversion échoue lors de l’utilisation de MultiByteToWideChar. |
| \_ERRORCREATINGFONTFILE E      | Une erreur s’est produite lors de la tentative de création du fichier de police.                                                                                                                                                                                                                              |
| \_ERRORDECOMPRESSINGFONTDATA E | Une erreur s’est produite lors de la tentative de décompression des données dans un fichier de police.                                                                                                                                                                                                                        |
| \_ERROREXPANDINGFONTDATA E     | Une erreur s’est produite pendant que les fonctions de chargement ont tenté de développer des données de police incorporées et compressées.                                                                                                                                                                                           |
| \_ERRORGETTINGDC E             | Une erreur s’est produite lors de la tentative d’allocation d’un contrôleur de périphérique, arrêt du traitement.                                                                                                                                                                                                                     |
| \_ERRORREADINGFONTDATA E       | Une erreur s’est produite lors de la tentative de lecture des données de police.                                                                                                                                                                                                                                    |
| \_ERRORUNICODECONVERSION E     | Une erreur s’est produite lors de l’allocation de mémoire pour convertir une chaîne de nom au format Unicode.                                                                                                                                                                                                           |
| \_ERRORUSINGTEMPFILE E         | Une erreur s’est produite pendant que les fonctions de chargement utilisaient un fichier temporaire pour installer un nouveau fichier de polices ou un nouveau fichier de ressources.                                                                                                                                                                      |
| E ( \_ exception)                  | Une exception a été levée par une cause inconnue.                                                                                                                                                                                                                                             |
| \_FACENAMEINVALID E            | Un paramètre *szFaceName* null a été passé à la fonction.                                                                                                                                                                                                                                |
| \_FLAGSINVALID E               | Le paramètre *ulFlags* dans la fonction active n’est pas valide.                                                                                                                                                                                                                              |
| \_FONTALREADYEXISTS E          | La police incorporée a le même nom et la même somme de contrôle qu’une police déjà installée sur le système.                                                                                                                                                                                              |
| \_FONTDATAINVALID E            | Les données de police lues à partir du disque ne sont pas une structure de police incorporée valide.                                                                                                                                                                                                                         |
| \_FONTFILECREATEFAILED E       | Les fonctions Load n’ont pas pu créer le fichier de police (. ttf)                                                                                                                                                                                                                                 |
| \_FONTFILENOTFOUND E           | Le fichier de police du nom de fichier spécifié n’existe pas.                                                                                                                                                                                                                                 |
| \_FONTINSTALLFAILED E          | Une tentative d’installation de la police incorporée dans le système a échoué.                                                                                                                                                                                                                            |
| \_FONTNAMEALREADYEXISTS E      | La police incorporée a le même nom, mais un checksum différent qu’une police déjà installée.                                                                                                                                                                                                |
| \_FONTNOTEMBEDDABLE E          | Impossible d’incorporer la police spécifiée en raison de restrictions du fabricant de la police. L’incorporation de cette police dans un document viole les lois sur les droits d’auteur.                                                                                                                                         |
| \_FONTREFERENCEINVALID E       | Un *phFontReference* null a été passé à la fonction.                                                                                                                                                                                                                                     |
| \_HDCINVALID E                 | Le contexte de périphérique spécifié pour la fonction [**TTEmbedFont**](/windows/desktop/api/T2embapi/nf-t2embapi-ttembedfont) n’est pas valide.                                                                                                                                                                                             |
| \_NAMECHANGEFAILED E           | [**TTLoadEmbeddedFont**](/windows/desktop/api/T2embapi/nf-t2embapi-ttloadembeddedfont) n’a pas pu modifier le nom de la police en cours de chargement.                                                                                                                                                                                 |
| \_NOFREEMEMORY E               | Une opération interne a échoué lors de la tentative d’allocation de mémoire.                                                                                                                                                                                                                        |
| \_NOOS2 E                      | Une table OS/2 est introuvable dans la police.                                                                                                                                                                                                                                                 |
| \_NOTATRUETYPEFONT E           | La police spécifiée n’est pas une police TrueType.                                                                                                                                                                                                                                               |
| \_PBENABLEDINVALID E           | Un paramètre *pbEnabled* null a été passé à la fonction.                                                                                                                                                                                                                                 |
| \_PERMISSIONSINVALID E         | Un paramètre *pulPermissions* null a été passé à la fonction.                                                                                                                                                                                                                            |
| \_PRIVSINVALID E               | Le paramètre *ulPrivs* spécifié dans les fonctions de chargement n’est pas valide.                                                                                                                                                                                                                      |
| \_PRIVSTATUSINVALID E          | Un paramètre *pulPrivStatus* null a été passé à la fonction.                                                                                                                                                                                                                             |
| \_READFROMSTREAMFAILED E       | Une erreur s’est produite lors de la tentative de lecture de la structure de police incorporée dans le flux.                                                                                                                                                                                                  |
| \_RESOURCEFILECREATEFAILED E   | Les fonctions de chargement n’ont pas pu créer le fichier de ressources de police (. pour).                                                                                                                                                                                                                       |
| \_SAVETOSTREAMFAILED E         | Une erreur s’est produite lors de la tentative d’enregistrement de la structure de police incorporée dans un flux.                                                                                                                                                                                                      |
| \_STATUSINVALID E              | Un paramètre *pulStatus* null a été passé à la fonction.                                                                                                                                                                                                                                 |
| \_STREAMINVALID E              | Le flux spécifié dans [**TTEmbedFont**](/windows/desktop/api/T2embapi/nf-t2embapi-ttembedfont) ou les fonctions de chargement n’est pas valide.                                                                                                                                                                                             |
| \_SUBSETTINGFAILED E           | [**TTEmbedFont**](/windows/desktop/api/T2embapi/nf-t2embapi-ttembedfont) a échoué lors de la tentative de création d’un sous-ensemble d’une police.                                                                                                                                                                                                 |
| \_T2NOFREEMEMORY E             | Une erreur s’est produite lors de la tentative de libération de mémoire. La mémoire en question a échoué pendant l’opération libre.                                                                                                                                                                              |
| \_WINDOWSAPI E                 | une erreur interne s’est produite lors de l’une des fonctions appelées API Windows, telles que [**GetTextMetrics**](/windows/desktop/api/Wingdi/nf-wingdi-gettextmetrics) ou [**GetOutlineTextMetrics**](/windows/desktop/api/Wingdi/nf-wingdi-getoutlinetextmetricsa).                                                                                                   |
| \_NOTIMPL API \_               | cette fonction d’API n’est pas implémentée dans la version de Windows sur laquelle elle s’exécute.                                                                                                                                                                                                   |
