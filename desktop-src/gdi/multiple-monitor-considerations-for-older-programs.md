# <a name="multiple-monitor-considerations-for-older-programs"></a>Considérations relatives à la surveillance multiple pour les anciens programmes

En règle générale, les applications plus anciennes n’ont pas besoin de modifications pour fonctionner sur plusieurs systèmes de surveillance. Pour la plupart, le système semble avoir un seul affichage. Les métriques système reflètent l’affichage principal et les applications plein écran apparaissent sur le serveur principal. Le système gère la minimisation, l’agrandissement, les menus et les boîtes de dialogue.

Toutefois, certains programmes rencontrent des problèmes. Certains peuvent avoir des problèmes : les applications de contrôle à distance, celles qui disposent d’un accès direct au matériel et celles qui ont corrigé le GDI ou le pilote d’affichage. Dans ce cas, l’utilisateur peut être amené à désactiver une analyse au-delà du moniteur principal.
