# <a name="font-and-text-macros"></a>Macros de police et de texte

Les macros suivantes sont utilisées avec des polices et du texte.



| Macro                            | Description                                            |
|----------------------------------|--------------------------------------------------------|
| [**DeleteFont**](/windows/desktop/api/Windowsx/nf-windowsx-deletefont) | Supprime une police spécifiée.                              |
| [**SelectFont**](/windows/desktop/api/Windowsx/nf-windowsx-selectfont) | Sélectionne une police dans le contexte de périphérique (DC) spécifié. |
