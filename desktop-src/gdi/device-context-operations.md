# <a name="device-context-operations"></a>Opérations de contexte de périphérique

Une application peut effectuer les opérations suivantes sur un contexte de périphérique :

-   Énumérer les objets graphiques existants.
-   Sélectionnez nouveaux objets graphiques.
-   Supprimer les objets graphiques existants.
-   Enregistrez les objets graphiques actuels, leurs attributs et les modes graphiques.
-   Restaurez les objets graphiques précédemment enregistrés, leurs attributs et les modes graphiques.

En outre, une application peut utiliser un contexte de périphérique pour :

-   Déterminez la façon dont la sortie graphique est traduite.
-   Annuler les opérations de dessin longues (commencées par un thread dans une application multithread).
-   Réinitialisation d’une imprimante à un état particulier.
