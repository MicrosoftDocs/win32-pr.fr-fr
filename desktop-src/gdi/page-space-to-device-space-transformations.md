# <a name="page-space-to-device-space-transformations"></a>Page-Space pour Device-Space des transformations

La transformation espace de page en espace périphérique détermine le mode de mappage de toutes les sorties graphiques associées à un contrôleur de domaine particulier. Un *mode de mappage* est une transformation de mise à l’échelle qui spécifie la taille des unités utilisées pour les opérations de dessin. Le mode de mappage peut également effectuer une traduction. Dans certains cas, le mode de mappage modifie l’orientation de l’axe des abscisses et de l’axe des y dans l’espace de l’appareil. Les rubriques suivantes décrivent les modes de mappage.

-   [Modes de mappage et traductions](mapping-modes-and-translations.md)
-   [Modes de mappage prédéfinis](predefined-mapping-modes.md)
-   [Modes de mappage définis par l’application](application-defined-mapping-modes.md)
