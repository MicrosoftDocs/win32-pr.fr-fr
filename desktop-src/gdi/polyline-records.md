# <a name="polyline-records"></a>Enregistrements Polyline

Les enregistrements [Polyline](/windows/desktop/api/Wingdi/nf-wingdi-polyline) sont une série de points. les lignes dessinées entre les points décrivent le contour du caractère. Un enregistrement de polyligne commence par le dernier point de l’enregistrement précédent (ou, pour le premier enregistrement du contour, le point de départ). Chaque point de l’enregistrement se trouve sur le contour du glyphe et peut être connecté simplement à l’aide de lignes droites.
