# <a name="pen-join"></a>Jointure de stylet

L’attribut Join spécifie la manière dont les extrémités de deux lignes géométriques sont jointes : biseautées, mitres ou arrondies. L’illustration suivante montre des paires de lignes connectées dessinées à l’aide de chaque type de jointure.
