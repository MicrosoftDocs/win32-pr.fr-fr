# <a name="update-assessor-structures"></a>Mettre à jour les structures d’évaluateur

Les fonctions suivantes sont utilisées par le fournisseur d’accesseurs de mise à jour.



| Fonction                                         | Description                                                           |
|--------------------------------------------------|-----------------------------------------------------------------------|
| [**OSUpdateAssessment**](/windows/win32/api/waasapitypes/ns-waasapitypes-osupdateassessment) | Définit le mode de mise à jour du système d’exploitation sur un appareil ciblé.                |
| [**UpdateAssessment**](/windows/win32/api/waasapitypes/ns-waasapitypes-osupdateassessment)   | Contient des informations qui évaluent la manière dont un système d’exploitation installé est à jour. |
