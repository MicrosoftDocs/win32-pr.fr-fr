# <a name="update-assessor-interfaces"></a>Mettre à jour les interfaces d’évaluateur

Les interfaces suivantes sont utilisées par le fournisseur d’accesseurs de mise à jour.



| Fonction                               | Description                                                                                                                 |
|----------------------------------------|-----------------------------------------------------------------------------------------------------------------------------|
| [**IWaaSAssessor**](/windows/desktop/api/waasapi/nn-waasapi-iwaasassessor) | Obtient l’évaluation des mises à jour du système d’exploitation en comparant la build la plus récente de Microsoft à la build en cours d’exécution sur l’appareil actuel. |
