# <a name="handles-and-objects"></a>Handles et objets

Un *objet* est une structure de données qui représente une ressource système, telle qu’un fichier, un thread ou une image graphique. Une application ne peut pas accéder directement aux données d’objet ou aux ressources système qu’un objet représente. Au lieu de cela, une application doit obtenir un *descripteur* d’objet, qu’elle peut utiliser pour examiner ou modifier la ressource système. Chaque descripteur a une entrée dans une table gérée en interne. Ces entrées contiennent les adresses des ressources et les moyens d’identifier le type de ressource.
