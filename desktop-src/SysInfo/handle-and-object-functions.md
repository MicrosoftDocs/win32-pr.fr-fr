# <a name="handle-and-object-functions"></a>Handle et fonctions d’objet

Les fonctions suivantes sont utilisées avec les handles et les objets.



| Fonction                                             | Description                                                                                  |
|------------------------------------------------------|----------------------------------------------------------------------------------------------|
| [**CloseHandle**](/windows/win32/api/handleapi/nf-handleapi-closehandle)                   | Ferme un handle d’objet ouvert.                                                                |
| [**CompareObjectHandles**](/windows/desktop/api/handleapi/nf-handleapi-compareobjecthandles) | Compare deux handles d’objet pour déterminer s’ils font référence au même objet de noyau sous-jacent. |
| [**DuplicateHandle**](/windows/win32/api/handleapi/nf-handleapi-duplicatehandle)           | Duplique un handle d’objet.                                                                 |
| [**GetHandleInformation**](/windows/win32/api/handleapi/nf-handleapi-gethandleinformation) | Récupère certaines propriétés d’un handle d’objet.                                            |
| [**SetHandleInformation**](/windows/win32/api/handleapi/nf-handleapi-sethandleinformation) | Définit certaines propriétés d’un handle d’objet.                                                 |
