# <a name="about-system-information"></a>À propos de System Information

Les fonctions décrites dans cette vue d’ensemble récupèrent une variété de données. Les informations de configuration du système décrivent le matériel de l’ordinateur, le nom de l’ordinateur et le nom d’utilisateur. Les informations de configuration du système d’exploitation décrivent la version du système d’exploitation, les répertoires de clés et les variables d’environnement. Les paramètres système décrivent les attributs système. Les métriques du système fournissent les dimensions des éléments dans la zone d’affichage.

Les sections suivantes expliquent comment obtenir ces types d’informations :

-   [Configuration matérielle](hardware-configuration.md)
-   [Noms des ordinateurs](computer-names.md)
-   [Version du système d’exploitation](operating-system-version.md)
-   [Configuration du système d’exploitation](operating-system-configuration.md)
