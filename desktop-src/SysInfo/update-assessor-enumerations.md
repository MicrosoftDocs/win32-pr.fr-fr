# <a name="update-assessor-enumerations"></a>Mettre à jour les énumérations de l’évaluateur

Les énumérations suivantes sont utilisées par le fournisseur d’accesseurs de mise à jour.



| Fonction                                                 | Description                                                                    |
|----------------------------------------------------------|--------------------------------------------------------------------------------|
| [**UpdateAssessmentStatus**](updateassessmentstatus.md) | Décrit comment mettre à jour le système d’exploitation sur un appareil.                                |
| [**UpdateImpactLevel**](updateimpactlevel.md)           | Indique un impact élevé, moyen ou faible sur un appareil exécutant un système d’exploitation obsolète. |
