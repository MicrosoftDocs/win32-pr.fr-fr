# <a name="desktop-environment"></a>Environnement de bureau

Cette section fournit des conseils pour l’intégration et l’extension des fonctionnalités de Windows pour les utilisateurs du bureau. Vous pouvez apprendre à vous assurer que vos applications et formats de fichiers s’affichent et se comportent correctement dans Démarrer, dans la barre des tâches, sur le bureau et dans l’Explorateur de fichiers. Vous pouvez également vous assurer que vos formats de fichiers et vos magasins de données sont indexables et pouvant faire l’objet de recherches.
