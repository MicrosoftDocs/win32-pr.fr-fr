# <a name="tapi-object"></a>Objet TAPI

L’objet TAPI représente toutes les ressources de téléphonie auxquelles l’ordinateur local a accès, ce qui permet à une application d’énumérer toutes les adresses locales et distantes. Une application TAPI 3 doit créer une instance de l’objet TAPI, puis l’initialiser.

L’exemple de code [Initialize TAPI](initialize-tapi.md) illustre un exemple de création d’objet TAPI.

Pour obtenir la liste de toutes les interfaces associées à l’objet TAPI, consultez [interfaces d’objets TAPI](tapi-object-interfaces.md) .
