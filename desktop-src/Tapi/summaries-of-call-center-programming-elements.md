# <a name="summaries-of-call-center-programming-elements"></a>Résumés des éléments de programmation du centre d’appels

La fonctionnalité du centre d’appels est une fonctionnalité de l’interface TAPI 3. x. Les éléments de programmation qui apparaissent dans le jeu de données TAPI 2,2 prennent en charge un centre d’appels TAPI 3. x. Pour plus d’informations sur les centres d’appels, consultez [à propos des contrôles du centre d’appels](./about-call-center-controls.md).

Les résumés des éléments de programmation du centre d’appels TAPI 2,2 sont les suivants :

-   [Fonctions du centre d’appels](call-center-functions.md)
-   [Structures du centre d’appels](call-center-structures.md)
-   [Messages du centre d’appels](call-center-messages.md)
-   [Constantes de centre d’appels](call-center-constants.md)
