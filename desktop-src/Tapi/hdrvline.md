# <a name="hdrvline"></a>HDRVLINE

Le type de données **HDRVLINE** représente le descripteur opaque d’un fournisseur de services pour une structure de données de ligne. Le fournisseur de services doit résoudre une valeur de ce type en une référence à l’instance de structure de données appropriée.
