# <a name="cmspstream-pure-virtual-methods"></a>CMSPStream des méthodes virtuelles pures

Ces méthodes doivent être remplacées par des classes dérivées.



| CMSPStream des méthodes virtuelles pures        | Description                                                                                                                                                                                               |
|----------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**nom d’accès \_**](/windows/win32/api/tapi3if/nf-tapi3if-itstream-get_name) | Obtient le nom du flux de données. La classe de flux dérivée doit fournir un nom qui permet à un utilisateur d’effectuer des distinctions significatives entre les flux si cette méthode est appelée pour dans l’interface utilisateur de l’application. |
