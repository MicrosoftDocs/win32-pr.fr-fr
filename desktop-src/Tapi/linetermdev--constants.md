# <a name="linetermdev_-constants"></a>\_Constantes LINETERMDEV

Les constantes d’indicateur de bit **LINETERMDEV \_** décrivent les différents types d’appareils terminaux.

<dl> <dt>

<span id="LINETERMDEV_HEADSET"></span><span id="linetermdev_headset"></span>**\_casque LINETERMDEV**
</dt> <dd> <dl> <dt>



Le terminal est un casque.


</dt> </dl> </dd> <dt>

<span id="LINETERMDEV_PHONE"></span><span id="linetermdev_phone"></span>**\_téléphone LINETERMDEV**
</dt> <dd> <dl> <dt>



Le terminal est un ensemble de téléphones.


</dt> </dl> </dd> <dt>

<span id="LINETERMDEV_SPEAKER"></span><span id="linetermdev_speaker"></span>**LINETERMDEV \_ Speaker**
</dt> <dd> <dl> <dt>



Le terminal est un conférencier externe et un microphone.


</dt> </dl> </dd> </dl>

## <a name="remarks"></a>Notes

Aucune extensibilité. Tous les 32 bits sont réservés.

Ces constantes sont utilisées pour caractériser le périphérique terminal d’une ligne et aider une application à déterminer la nature d’un périphérique terminal.
