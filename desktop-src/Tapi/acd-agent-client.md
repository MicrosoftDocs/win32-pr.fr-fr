# <a name="acd-agent-client"></a>Client de l’agent ACD

Un agent du centre d’appels est chargé de recevoir et de traiter les appels entrants et, parfois, d’effectuer des appels sortants aux clients ou aux prospects. Une application cliente de l’agent ACD fournit des communications avec l’application de [proxy ACD](acd-proxy.md) , en lui fournissant des informations indiquant notamment si l’agent est disponible pour les appels et les types d’appels que l’agent peut gérer. Le diagramme suivant illustre la façon dont les objets TAPI s’affichent sur un agent.
