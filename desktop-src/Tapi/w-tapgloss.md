<dl> <dt>

<span id="tapi2.web_telephony_tapgloss"></span><span id="TAPI2.WEB_TELEPHONY_TAPGLOSS"></span>**Téléphonie Web**
</dt> <dd>

Une plate-forme d’application qui prend en charge les sites Web à accès double (accessibles par téléphone ou navigateur Internet) et les applications IVR HTML. La plateforme comprend un moteur d’exécution intégré au serveur, une interface d’administration COM pour l’installation et la configuration d’applications de téléphonie Web, ainsi que l’accès à l’objet d’appel et l’API de script TAPI 3. x, pour l’implémentation d’opérations de téléphonie.

</dd> <dt>

<span id="tapi2.web_telephony_engine_webte__tapgloss"></span><span id="TAPI2.WEB_TELEPHONY_ENGINE_WEBTE__TAPGLOSS"></span>**Moteur de téléphonie Web (WTE)**
</dt> <dd>

Prend en charge les applications IVR HTML et les sites Web à accès double (un site Web accessible par téléphone et via un navigateur graphique). Voir *téléphonie Web*.

</dd> <dt>

<span id="tapi2.windows_open_services_architecture_wosa__tapgloss"></span><span id="TAPI2.WINDOWS_OPEN_SERVICES_ARCHITECTURE_WOSA__TAPGLOSS"></span>**Windows Architecture WOSA (Open Services Architecture)**
</dt> <dd>

Architecture logicielle qui fournit une interface de niveau système unique pour connecter des applications frontales à des services principaux. Windows La téléphonie fait partie de l’architecture WOSA.

</dd> </dl>
