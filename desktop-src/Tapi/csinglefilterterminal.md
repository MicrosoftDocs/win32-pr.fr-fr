# <a name="csinglefilterterminal"></a>CSingleFilterTerminal

**CSingleFilterTerminal** est une classe de base de terminal supplémentaire applicable aux terminaux statiques et dynamiques. Elle est dérivée de [**CBaseTerminal**](cbaseterminal.md) et rend l’implémentation plus spécifique en supposant que le terminal possède un filtre DirectShow unique. Lorsque cette condition est remplie, la dérivation d’une implémentation de terminal à partir de **CSingleFilterTerminal** est beaucoup plus facile que la dérivation à partir de **CBaseTerminal**.

Défini dans MSPterm. h.
