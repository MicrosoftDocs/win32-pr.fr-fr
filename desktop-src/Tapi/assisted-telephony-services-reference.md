# <a name="assisted-telephony-services-reference"></a>Référence des services de téléphonie assistée

Les fonctions de téléphonie assistée sont répertoriées dans le tableau suivant. Les fonctions de téléphonie assistée permettent à n’importe quelle application d’établir des appels vocaux et des appels multimédias. Pour obtenir des informations générales, consultez la vue d’ensemble de la [téléphonie assistée](./assisted-telephony-overview.md) .

## <a name="assisted-telephony-services-functions"></a>Fonctions de services de téléphonie assistée



| Fonction                                             | Description                                                 |
|------------------------------------------------------|-------------------------------------------------------------|
| [**tapiRequestMakeCall**](/windows/desktop/api/Tapi/nf-tapi-tapirequestmakecall)   | Envoie une demande pour passer un appel vocal.                    |
| [**tapiRequestMediaCall**](tapirequestmediacall.md) | Obsolète. Ne pas utiliser.                                       |
| [**tapiRequestDrop**](/windows/desktop/api/Tapi/nf-tapi-tapirequestdrop)           | Obsolète. Ne pas utiliser.                                       |
| [**tapiGetLocationInfo**](/windows/desktop/api/Tapi/nf-tapi-tapigetlocationinfo)   | Retourne les codes identifiant le pays/la région et la ville/zone. |
