# <a name="ikeyframecontrolupdatepicture-method"></a>IKeyFrameControl :: UpdatePicture, méthode

\[**UpdatePicture** n’est pas disponible pour une utilisation dans Windows Vista, windows Server 2008 et les versions ultérieures du système d’exploitation. L’API cliente RTC offre des fonctionnalités similaires.\]

La méthode **UpdatePicture** est appelée par une application pour demander à l’expéditeur de la vidéo de mettre à jour l’image. Chaque fois que cette méthode est appelée, une demande est envoyée à l’expéditeur vidéo. Le flux doit être actif.
