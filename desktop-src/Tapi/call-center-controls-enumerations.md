# <a name="call-center-controls-enumerations"></a>Énumération des contrôles du centre d’appels

Les contrôles de centre d’appels TAPI 3. x utilisent les énumérations suivantes pour définir et créer des rapports sur les événements et les États.

-   [**\_événement ACDGROUP**](/windows/win32/api/tapi3/ne-tapi3-acdgroup_event)
-   [**\_événement ACDQUEUE**](/windows/win32/api/tapi3/ne-tapi3-acdqueue_event)
-   [**événement de l’AGENT \_**](/windows/win32/api/tapi3cc/ne-tapi3cc-agent_event)
-   [**\_événement AGENTHANDLER**](/windows/win32/api/tapi3/ne-tapi3-agenthandler_event)
-   [**\_événement de session de l’agent \_**](/windows/win32/api/tapi3cc/ne-tapi3cc-agent_session_event)
-   [**\_État de session de l’agent \_**](/windows/win32/api/tapi3cc/ne-tapi3cc-agent_session_state)
-   [**État de l’AGENT \_**](/windows/win32/api/tapi3cc/ne-tapi3cc-agent_state)
