# <a name="linetolllistoption_-constants"></a>\_Constantes LINETOLLLISTOPTION

Les constantes d’indicateur de bit **LINETOLLLISTOPTION \_** décrivent les options de manipulation d’une liste de péages.

<dl> <dt>

<span id="LINETOLLLISTOPTION_ADD"></span><span id="linetolllistoption_add"></span>**LINETOLLLISTOPTION \_ Ajouter**
</dt> <dd> <dl> <dt>



Un préfixe doit être ajouté à la liste de numéros de téléphone.


</dt> </dl> </dd> <dt>

<span id="LINETOLLLISTOPTION_REMOVE"></span><span id="linetolllistoption_remove"></span>**LINETOLLLISTOPTION \_ Supprimer**
</dt> <dd> <dl> <dt>



Un préfixe doit être supprimé de la liste de numéros de téléphone.


</dt> </dl> </dd> </dl>

## <a name="remarks"></a>Notes

Aucune extensibilité. Tous les 32 bits sont réservés.
