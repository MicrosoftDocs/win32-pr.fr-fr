# <a name="basic-telephony-services"></a>Services de téléphonie de base

Les services de téléphonie de base sont un sous-ensemble minimal de la spécification de téléphonie. Étant donné que tous les fournisseurs de services doivent prendre en charge les fonctions des services de téléphonie de base, les applications qui utilisent uniquement ces fonctions fonctionnent avec n’importe quel fournisseur de services TAPI. Les fonctionnalités contenues dans la téléphonie de base correspondent approximativement aux fonctionnalités des POTS.

De nombreux programmeurs utilisent uniquement les services fournis par la téléphonie de base. D’autres, telles que celles qui écrivent du code pour les systèmes téléphoniques PBX, auront besoin des fonctions de téléphonie supplémentaire.

Pour obtenir la liste des fonctions de téléphonie de base, consultez [référence des fonctions rapides TAPI](./tapi-quick-function-reference.md).

Étant donné que le contrôle des appareils téléphoniques n’est pas censé être proposé par tous les fournisseurs de services, les services pour appareils téléphoniques sont considérés comme facultatifs. Autrement dit, ils ne font pas partie de la téléphonie de base. Pour obtenir la liste des services pour appareils téléphoniques, consultez [services de téléphonie supplémentaires](supplementary-telephony-services.md). Pour plus d’informations sur les appareils téléphoniques, consultez [classes de périphériques TAPI](./tapi-device-classes.md).
