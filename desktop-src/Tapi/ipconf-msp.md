# <a name="ipconf-msp"></a>MSP IPConf

\[IPConf MSP ne peut pas être utilisé dans Windows Vista, Windows Server 2008 et les versions ultérieures du système d’exploitation. L’API cliente RTC offre des fonctionnalités similaires.\]

Le IPConf MSP (Confmsp.dll), en association avec son TSP (IPconf. tsp), offre un accès aux conférences aux réseaux IP (Internet Protocol), tels qu’Internet et de nombreux intranets d’entreprise. Outre les méthodes MSPI standard, ce MSP expose des interfaces qui permettent de contrôler les participants individuels d’une conférence.

Pour obtenir un résumé des interfaces spécifiques au fournisseur exposées par ce MSP, consultez [IPCONF MSP interfaces](ipconf-msp-interfaces.md) .

le MSP IPConf a été amélioré dans Windows 2000 SP1 pour permettre la communication entre les clients H323 et SDP. Il expose désormais également l’interface [**IMulticastControl**](imulticastcontrol.md) .

le MSP IPConf est installé avec Windows 2000.
