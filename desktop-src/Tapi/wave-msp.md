# <a name="wave-msp"></a>MSP Wave

Le MSP Wave (Wavemsp.dll) correspond aux TSPs hérités écrits avant l’introduction de MSP dans l’architecture de diffusion en continu basée sur le MSP de TAPI 3. Certains des TSPs hérités exposent des identificateurs de périphérique Wave, ce qui a été un moyen pour l’application de contrôler la diffusion en continu sous TAPI 2,1. Tapi3.dll utilise le MSP Wave pour tout PVC qui n’a pas son propre MSP, mais qui a un appareil Wave.

le MSP Wave est installé avec Windows 2000.
