# <a name="simple-record"></a>Enregistrement simple

Pour effectuer un enregistrement simple des flux, procédez comme suit :

-   Sélectionner le terminal d’enregistrement de fichier au niveau de l’appel
-   Créez le terminal d’enregistrement de fichier.
-   Définir les propriétés : le nom et le type de stockage du fichier.
-   Sélectionnez le terminal d’enregistrement de fichier sur l’appel.
-   Appelez la méthode [**Start**](/windows/desktop/api/tapi3if/nf-tapi3if-itmediacontrol-start) sur le terminal d’enregistrement de fichier pour démarrer l’enregistrement des flux.
