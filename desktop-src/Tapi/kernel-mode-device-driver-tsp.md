# <a name="kernel-mode-device-driver-tsp"></a>Kernel-Mode TSP de pilote de périphérique

Le TSP en mode noyau (kmddsp. tsp) permet à une application de gérer les miniportages WAN hérités (par exemple, les pilotes NDIS 4,0) en tant qu’appareils TAPI.

le TSP en mode noyau est installé avec les systèmes d’exploitation Windows Server 2003, Windows XP, Windows 2000, Windows NT, Windows Millennium Edition, Windows 98 et Windows 95.
