# <a name="rendezvous-ip-telephony-conferencing-reference"></a>Informations de référence sur la Conférence de téléphonie IP Rendezvous

\[ Les interfaces et les contrôles de conférence de téléphonie IP Rendezvous ne peuvent pas être utilisés dans Windows Vista, Windows Server 2008 et les versions ultérieures du système d’exploitation. L’API cliente RTC offre des fonctionnalités similaires.\]

L’API Rendezvous de TAPI 3 est un groupe d’interfaces COM (Component Object Model) qui peuvent être utilisées pour publier et découvrir des conférences multimédias multiparties. La documentation suivante fournit des détails sur les objets, les interfaces et les méthodes fournis.

TAPI 3 utilise des filtres DirectShow et des gestionnaires de filtres pour contrôler le flux de média après la configuration de la connexion. COM est le modèle de codage de base pour TAPI 3, et vous savez qu’il est supposé dans ce document.
