# <a name="tapi-13"></a>TAPI 1,3

TAPI 1,3 (parfois appelé TAPI 1,03) était la première version publiée de la spécification TAPI. Les binaires système TAPI 1,3 (Tapi.dll et autres) se trouvent dans le kit de développement logiciel (SDK) TAPI 1,0, qui se trouve uniquement sur l’abonnement Microsoft Developer Network (MSDN) Level II. Les fichiers binaires système (qui se trouvent dans le répertoire Redist) sont redistribuables, mais ils sont conçus pour s’exécuter uniquement sur Microsoft Windows 3,1 et Windows pour les groupes de travail 3,11. Ils ne sont pas pris en charge et ne fonctionnent pas sur les versions de Windows 32 bits.

Les en-têtes et les bibliothèques utilisés pour développer une application TAPI 1,3 sont uniquement disponibles dans le kit de développement logiciel (SDK) TAPI 1,0 et ne prennent en charge que le développement 16 bits. En fait, les bibliothèques TAPI 1,3 sont les seules bibliothèques TAPI 16 bits disponibles et doivent être utilisées pour toutes les applications 16 bits, quelle que soit la plate-forme cible ou la version cible.
