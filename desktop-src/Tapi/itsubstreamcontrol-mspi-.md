# <a name="itsubstreamcontrol-mspi"></a>ITSubStreamControl (MSPI)

L’interface [**ITSubStreamControl**](/windows/win32/api/tapi3if/nn-tapi3if-itsubstreamcontrol) est exposée sur l’objet de flux s’il existe un MSP associé et qu’il implémente des sous-flux. Les méthodes de cette interface permettent à une application de créer et de supprimer des sous-flux.

L’interface [**ITSubStreamControl**](/windows/win32/api/tapi3if/nn-tapi3if-itsubstreamcontrol) est implémentée par un MSP. Pour plus d’informations sur cette interface, consultez **ITSubStreamControl** dans la section de l’interface MSP.
