# <a name="h323-msp"></a>MSP H323

\[le MSP H323 n’est pas disponible pour une utilisation dans Windows Vista, Windows Server 2008 et les versions ultérieures du système d’exploitation. L’API cliente RTC offre des fonctionnalités similaires.\]

Le MSP H323 (H323msp.dll), lorsqu’il est associé à son TSP (h323. tsp), permet à une application de passer et de recevoir des appels vidéo et audio à l’aide du protocole H. 323 pour la téléphonie IP.

le MSP h323 a été amélioré dans Windows 2000 SP1 pour permettre la communication entre les clients h323 et SDP. Il expose désormais les interfaces [**IH323LineEx**](ih323lineex.md) et [**IKeyFrameControl**](ikeyframecontrol.md) .

le MSP H323 est installé avec Windows 2000.
