## <a name="remarks"></a>Notes

-   BSTR TAPIPROTOCOL \_ chaîne \_ RTPC
-   Chaîne BSTR \_ TAPIPROTOCOL \_ H323
-   Multidiffusion de chaîne BSTR TAPIPROTOCOL \_ \_
