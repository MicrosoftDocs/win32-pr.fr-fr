# <a name="terminal-type"></a>Type de terminal

Le **type de terminal** détermine si un terminal est statique ou dynamique. Par exemple, un modem est un terminal statique, tandis que la fenêtre vidéo sur un écran d’ordinateur est créée lorsqu’il est nécessaire et détruite quand son utilité est terminée.

Ce concept est principalement utilisé par les applications TAPI 3.

**TAPI 3. x :** Consultez [**ITTerminal :: obtenir \_ TerminalType**](/windows/win32/api/tapi3if/nf-tapi3if-itterminal-get_terminaltype), [**\_ type de terminal**](/windows/desktop/api/Tapi3if/ne-tapi3if-terminal_type).
