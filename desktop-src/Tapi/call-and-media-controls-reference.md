# <a name="call-and-media-controls-reference"></a>Référence des contrôles d’appel et de média

La documentation de référence sur l’appel et le contrôle multimédia contient des informations sur les interfaces, les méthodes, les énumérateurs et les types utilisés dans la configuration et le contrôle d’appel.

Vous pouvez également rechercher des informations sur DirectShow et le COM (Component Object Model) dans le kit de développement logiciel (SDK) de la plateforme. TAPI 3. x utilise des filtres et des gestionnaires de filtres DirectShow pour contrôler le flux de média après la configuration de la connexion. COM est le modèle de codage de base pour TAPI 3. x et vous savez qu’il est pris en part dans ce document.
