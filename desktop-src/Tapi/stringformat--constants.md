# <a name="stringformat_-constants"></a>\_Constantes STRINGFORMAT

Les **\_ constantes STRINGFORMAT** décrivent des formats de chaîne différents.

<dl> <dt>

<span id="STRINGFORMAT_ASCII"></span><span id="stringformat_ascii"></span>**STRINGFORMAT \_ ASCII**
</dt> <dd> <dl> <dt>



Spécifie le format de caractère ASCII standard en utilisant un octet par caractère.


</dt> </dl> </dd> <dt>

<span id="STRINGFORMAT_BINARY"></span><span id="stringformat_binary"></span>**STRINGFORMAT ( \_ binaire)**
</dt> <dd> <dl> <dt>



Il s’agit d’un tableau de caractères non signés. peut être utilisé pour les valeurs numériques.


</dt> </dl> </dd> <dt>

<span id="STRINGFORMAT_DBCS"></span><span id="stringformat_dbcs"></span>**STRINGFORMAT \_ DBCS**
</dt> <dd> <dl> <dt>



Spécifie le format de caractère DBCS standard à l’aide d’un ou de deux octets par caractère.


</dt> </dl> </dd> <dt>

<span id="STRINGFORMAT_UNICODE"></span><span id="stringformat_unicode"></span>**STRINGFORMAT \_ Unicode**
</dt> <dd> <dl> <dt>



Spécifie le format de caractère Unicode standard à l’aide de deux octets par caractère.


</dt> </dl> </dd> </dl>

## <a name="remarks"></a>Notes

Aucune extensibilité. Tous les 32 bits sont réservés.
