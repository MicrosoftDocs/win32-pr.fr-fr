# <a name="lineproxystatus_-constants"></a>\_Constantes LINEPROXYSTATUS

Les **\_ constantes LINEPROXYSTATUS** indiquent l’état du proxy sur une ligne ouverte par l’application.

Pour obtenir une liste et une description de toutes les valeurs possibles de la requête de proxy, consultez [**\_ constantes LINEPROXYREQUEST**](lineproxyrequest--constants.md) .

<dl> <dt>

<span id="LINEPROXYSTATUS_ALLOPENFORACD"></span><span id="lineproxystatus_allopenforacd"></span>**LINEPROXYSTATUS \_ ALLOPENFORACD**
</dt> <dd> <dl> <dt>



La ligne a maintenant des proxies ouverts pour tous les types de demande proxy requis pour les opérations ACD par TAPI version 3,0 et ultérieure.


</dt> </dl> </dd> <dt>

<span id="LINEPROXYSTATUS_CLOSE"></span><span id="lineproxystatus_close"></span>**LINEPROXYSTATUS \_ Fermer**
</dt> <dd> <dl> <dt>



Un proxy a été fermé.


</dt> </dl> </dd> <dt>

<span id="LINEPROXYSTATUS_OPEN"></span><span id="lineproxystatus_open"></span>**LINEPROXYSTATUS \_ ouvrir**
</dt> <dd> <dl> <dt>



Un nouveau proxy s’est ouvert.


</dt> </dl> </dd> </dl>
