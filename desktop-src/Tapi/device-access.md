# <a name="device-access"></a>Accès des appareils

La référence principale pour les opérations et les informations sur les appareils est la principale vue d’ensemble de ce kit de développement logiciel (SDK), sous [contrôle de périphérique](./device-control.md). Cette vue d’ensemble répertorie et définit la terminologie impliquée, et fournit des liens vers les pages de référence TAPI 2,2 et TAPI 3 pour les éléments de programmation associés.

La documentation suivante décrit certaines informations qui s’appliquent uniquement à l’interface TAPI 2,2 (TAPI/C).

-   [Fonctions de ligne étendues](extended-line-functions.md)
-   [Mode passthrough](passthrough-mode.md)
-   [Routage d’événements](event-routing.md)
