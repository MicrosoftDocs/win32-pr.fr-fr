# <a name="basic-phone-services"></a>Services téléphoniques de base

Les services téléphoniques de base sont l’ensemble de services principal pris en charge par tous les appareils téléphoniques. Étant donné que les services téléphoniques de base sont définis par le fournisseur de services, il existe une grande variété dans le jeu de cœurs.
