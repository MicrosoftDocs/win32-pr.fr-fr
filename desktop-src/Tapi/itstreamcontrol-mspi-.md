# <a name="itstreamcontrol-mspi"></a>ITStreamControl (MSPI)

L’interface [**ITStreamControl**](/windows/win32/api/tapi3if/nn-tapi3if-itstreamcontrol) est exposée sur l’objet d’appel si un MSP est associé. Les méthodes de cette interface permettent à une application de créer et de supprimer des flux.

L’interface [**ITStreamControl**](/windows/win32/api/tapi3if/nn-tapi3if-itstreamcontrol) est implémentée par un MSP. Pour plus d’informations sur cette interface, consultez **ITStreamControl** dans la section de l’interface MSP.
