# <a name="tapi-22-overview"></a>Vue d’ensemble de TAPI 2,2

L’interface de programmation d’applications de téléphonie (TAPI) Microsoft version 2,2 est une interface basée sur le langage C qui permet aux développeurs d’implémenter une gamme d’applications de communication. La documentation suivante présente des informations générales spécifiques à l’utilisation de TAPI 2,2. La documentation de présentation principale est contenue dans le [modèle de programmation de téléphonie de Microsoft](./microsoft-telephony-programming-model.md). Il est supposé que vous êtes familiarisé avec ce document.

-   Le contrôle de [version TAPI](tapi-versioning.md) couvre les versions disponibles de l’interface TAPI, comment elles sont distribuées et comment votre application peut négocier la version de TAPI à utiliser.
-   [L’accès aux sessions](session-access.md) couvre certains éléments de session d’appel spécifiques à TAPI 2,2 (TAPI/C). Consultez la vue d’ensemble générale du [contrôle de session](./session-control.md) pour obtenir une référence principale sur les opérations et les informations de session d’appel.
-   [L’accès aux appareils](device-access.md) couvre les fonctionnalités spécifiques à TAPI 2,2 (TAPI/C). Consultez la vue d’ensemble générale sur le contrôle de l' [appareil](./device-control.md) pour référence principale sur les opérations et les informations sur les appareils.
-   L' [accès aux médias](media-access.md) couvre la documentation sur les informations de média et les contrôles disponibles pour une application TAPI 2,2. Les fournisseurs de services de média vous donnent accès à des contrôles plus détaillés, mais seules les applications TAPI 3 (TAPI/COM) peuvent utiliser leurs fonctionnalités. Pour plus d’informations, consultez [à propos du fournisseur de services multimédia](./about-the-media-service-provider-msp-.md) .
-   [Le contrôle du centre d’appels](call-center-control.md) couvre la documentation sur la création d’applications ACD (Automatic Call distribution) à l’aide de TAPI 2,2. Pour TAPI 3. x, consultez [contrôles du centre d’appels](./about-call-center-controls.md).
-   Les [appareils téléphoniques](phone-devices.md) couvrent les contrôles d’appareil téléphonique supplémentaires fournis par TAPI 2,2 (TAPI/C).
