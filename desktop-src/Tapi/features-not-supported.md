# <a name="features-not-supported"></a>Fonctionnalités non prises en charge

-   La gestion dynamique des graphiques sera implémentée par le MSP dérivé, si elle le souhaite.
-   Les sous-flux seront implémentés par le MSP dérivé, s’il le souhaite.
-   Les interfaces privées, telles que les participants, le contrôle de codec, etc., ne sont pas implémentées par les classes de base.
-   La communication TSP-MSP sera implémentée par le MSP dérivé.
