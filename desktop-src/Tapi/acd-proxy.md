<span data-ttu-id="ae857-104">Un serveur ACD (Automatic Call distribution) est une combinaison de matériel et de logiciels qui classifie, met en file d’attente et distribue les appels entrants aux agents ou les appels sortants vers les lignes.</span><span class="sxs-lookup"><span data-stu-id="ae857-104">An Automatic Call Distribution (ACD) server is a combination of hardware and software that classifies, queues, and distributes incoming calls to agents or outgoing calls to lines.</span></span>

<span data-ttu-id="ae857-109">La classification des appels entrants est conçue pour appeler un agent qui sera en mesure de la gérer efficacement.</span><span class="sxs-lookup"><span data-stu-id="ae857-109">Classification of incoming calls is designed to get a call to an agent who will be able to handle it effectively.</span></span> <span data-ttu-id="ae857-110">La classification peut être basée sur le numéro composé, un système de reconnaissance vocale interactive (IVR) ou d’autres critères.</span><span class="sxs-lookup"><span data-stu-id="ae857-110">Classification may be based on the number dialed, an interactive voice recognition (IVR) system, or other criteria.</span></span> <span data-ttu-id="ae857-111">L’interface TAPI utilise le concept de [groupe ACD](about-call-center-controls.md) pour représenter cette opération.</span><span class="sxs-lookup"><span data-stu-id="ae857-111">TAPI uses the concept of an [ACD Group](about-call-center-controls.md) to represent this operation.</span></span>

<span data-ttu-id="ae857-112">Les files d’attente sont un moyen normal et attendu pour que les centres d’appels gèrent des périodes de charge élevée.</span><span class="sxs-lookup"><span data-stu-id="ae857-112">Queues are a graceful and expected way for call centers to deal with heavy load periods.</span></span> <span data-ttu-id="ae857-113">Une file d’attente est normalement associée à un groupe ACD, mais peut être créée pour gérer les lignes sortantes ou les appels entrants en attente d’une application IVR.</span><span class="sxs-lookup"><span data-stu-id="ae857-113">A queue is normally associated with an ACD Group but may be created to handle outgoing lines or incoming calls waiting for an IVR application.</span></span> <span data-ttu-id="ae857-114">Pour plus d’informations, consultez [files d’attente](about-call-center-controls.md) .</span><span class="sxs-lookup"><span data-stu-id="ae857-114">See [Queues](about-call-center-controls.md) for more information.</span></span>

<span data-ttu-id="ae857-115">La distribution des appels à des agents ou à des groupes d’agents peut être uniforme, mais elle est généralement basée sur des données telles que les informations d’appel, la disponibilité de l’agent et la capacité.</span><span class="sxs-lookup"><span data-stu-id="ae857-115">Call distribution to agents or groups of agents may be uniform but typically is based on data such as call information, agent availability, and ability.</span></span> <span data-ttu-id="ae857-116">Le [Gestionnaire d’agents](about-call-center-controls.md) de l’interface TAPI fournit un accès aux informations telles que la disponibilité et les adresses des agents pour le transfert des appels aux agents.</span><span class="sxs-lookup"><span data-stu-id="ae857-116">TAPI's [Agent Handler](about-call-center-controls.md) provides access to information such as agent availability and addresses for the transfer of calls to agents.</span></span>
