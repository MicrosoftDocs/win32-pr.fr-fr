# <a name="assisted-telephony-functions"></a>Fonctions de téléphonie assistée

L’interface TAPI prend en charge les fonctions de [téléphonie assistée](./assisted-telephony-overview.md) suivantes :

-   [**tapiGetLocationInfo**](/windows/desktop/api/Tapi/nf-tapi-tapigetlocationinfo)
-   [**tapiRequestDrop**](/windows/desktop/api/Tapi/nf-tapi-tapirequestdrop)
-   [**tapiRequestMakeCall**](/windows/desktop/api/Tapi/nf-tapi-tapirequestmakecall)
-   [**tapiRequestMediaCall**](tapirequestmediacall.md)

Pour obtenir une catégorisation des fonctions TAPI par niveau de service et tâche, consultez Référence des fonctions [rapides TAPI](tapi-quick-function-reference.md).
