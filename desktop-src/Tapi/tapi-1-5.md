# <a name="tapi-15"></a>TAPI 1,5

TAPI 1,5 a été conçu spécifiquement pour la version 1,0 de Windows CE. il s’agit d’un sous-ensemble de TAPI 1,4, qui est disponible uniquement sur Windows CE et qui est documenté dans la documentation du développeur Windows CE.
