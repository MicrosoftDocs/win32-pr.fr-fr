# <a name="extending-data-structures-and-types"></a>Extension des structures et des types de données

Une plage de valeurs est réservée pour prendre en charge les extensions futures de l’ensemble de fonctions TAPI de base et supplémentaires. Pour plus d’informations sur la quantité d’extension possible d’une structure de données, consultez [extensibilité](extensibility.md). Pour obtenir la liste des fonctions utilisées pour étendre la téléphonie, consultez [référence des fonctions rapides TAPI](./tapi-quick-function-reference.md).
