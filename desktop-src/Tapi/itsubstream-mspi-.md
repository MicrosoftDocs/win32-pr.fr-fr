# <a name="itsubstream-mspi"></a>ITSubStream (MSPI)

L’interface [**ITSubStream**](/windows/win32/api/tapi3if/nn-tapi3if-itsubstream) est disponible si un MSP est associé à l’appel et qu’il implémente des sous-flux. Les méthodes de cette interface permettent à une application de récupérer des informations sur un sous-flux ; pour démarrer, suspendre ou arrêter le sous-flux ; et pour sélectionner ou désélectionner des terminaux sur un sous-flux.

L’interface [**ITSubStream**](/windows/win32/api/tapi3if/nn-tapi3if-itsubstream) est implémentée par un MSP. Pour plus d’informations sur cette interface, consultez **ITSubStream** dans la section de l’interface MSP.
