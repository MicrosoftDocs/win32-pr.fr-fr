# <a name="extended-telephony-phone-functions"></a>Fonctions téléphoniques étendues de téléphonie

Les services téléphoniques étendus (ou les services Device-Specific Phone) incluent toutes les extensions de l’interface TAPI définies par le fournisseur de services. L’interface TAPI définit un mécanisme qui permet aux fournisseurs de fournisseurs de services d’étendre l’interface TAPI à l’aide d’extensions spécifiques à l’appareil. L’interface TAPI définit uniquement le mécanisme d’extension, ce qui permet d’accéder aux extensions spécifiques à l’appareil. L’interface TAPI ne définit pas leur comportement, qui est complètement défini par le fournisseur de services.

L’interface TAPI est constituée de définitions de constantes de données scalaires et de données d’indicateur de bit, de structures de données, de fonctions et de messages. Des procédures sont définies pour permettre à un fournisseur d’étendre la plupart de ces éléments, comme décrit dans les rubriques suivantes.

-   [Constantes de données scalaires](scalar-data-constants.md)
-   [Constantes de données d’indicateur de bit](bit-flag-data-constants.md)
-   [Fonctions et messages](functions-and-messages.md)
