# <a name="session-access"></a>Accès aux sessions

La référence principale pour les opérations et les informations de session est la principale vue d’ensemble de ce kit de développement logiciel (SDK), sous [contrôle de session](./session-control.md). Cette vue d’ensemble répertorie et définit la terminologie impliquée, et fournit des liens vers les pages de référence TAPI 2,2 et TAPI 3 pour les éléments de programmation associés.

La documentation suivante décrit certaines informations qui s’appliquent uniquement à l’interface TAPI 2,2 (TAPI/C).

-   [Handles d’appel](call-handles.md)
