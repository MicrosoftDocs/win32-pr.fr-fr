# <a name="telephony-application-programming-interface-version-31"></a>Interface de programmation d’applications de téléphonie version 3,1

## <a name="purpose"></a>Objectif

La version 3,1 de l’interface de programmation d’applications de téléphonie (TAPI) Microsoft est une API COM (Component Object Model) qui fusionne les téléphonie classiques et IP.

## <a name="where-applicable"></a>Le cas échéant

Les applications TAPI possibles sont les suivantes :

-   Vidéoconférence multidiffusion d’IP multimédia avec qualité de service (QOS)
-   Appels vocaux via Internet à l’aide du protocole H. 323
-   Applications de centre d’appels pouvant suivre plusieurs agents
-   Appels vocaux de base sur le réseau téléphonique public commuté (RTPC)
-   Contrôle PBX
-   Systèmes de réponse vocale interactive (IVR)
-   Messagerie vocale

## <a name="developer-audience"></a>Développeurs concernés

vous pouvez écrire des applications compatibles TAPI dans de nombreux langages, notamment Java, Visual Basic et C/C++. Vous devez vous familiariser avec COM. L’expérience de développement avec les télécommunications ou d’autres applications de téléphonie est utile, mais pas nécessaire.

## <a name="run-time-requirements"></a>Conditions d’exécution

TAPI version 3,1 permet le développement d’applications de communication pour les systèmes d’exploitation Windows Server 2003, Windows XP et Windows 2000.
