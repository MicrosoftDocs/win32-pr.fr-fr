# <a name="tspi_linedroponclose"></a>TSPI \_ lineDropOnClose

La fonction **TSPI \_ lineDropOnClose** est obsolète. L’interface TAPI ne supprime plus automatiquement un appel lorsque le propriétaire unique ferme son handle sur le périphérique de ligne, afin de prendre en charge le contrôle distribué des appels sur plusieurs ordinateurs clients dans un environnement client/serveur.
