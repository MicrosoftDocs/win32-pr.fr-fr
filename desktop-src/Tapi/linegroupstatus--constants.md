# <a name="linegroupstatus_-constants"></a>\_Constantes LINEGROUPSTATUS

Les **\_ constantes LINEGROUPSTATUS** spécifient l’état du groupe ACD.

<dl> <dt>

<span id="LINEGROUPSTATUS_GROUPREMOVED"></span><span id="linegroupstatus_groupremoved"></span>**LINEGROUPSTATUS \_ GROUPREMOVED**
</dt> <dd> <dl> <dt>



Ce groupe a été supprimé de la liste de ceux qui sont disponibles.


</dt> </dl> </dd> <dt>

<span id="LINEGROUPSTATUS_NEWGROUP"></span><span id="linegroupstatus_newgroup"></span>**LINEGROUPSTATUS \_**
</dt> <dd> <dl> <dt>



Un groupe a été ajouté à la liste de ceux qui sont disponibles.


</dt> </dl> </dd> </dl>
