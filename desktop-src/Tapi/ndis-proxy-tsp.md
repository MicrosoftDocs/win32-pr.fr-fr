# <a name="ndis-proxy-tsp"></a>TSP proxy NDIS

Le proxy NDIS est un composant en mode noyau qui expose les minidumps NDIS en tant que lignes TAPI. Le TSP proxy NDIS (ndptsp. tsp) permet à TAPI de communiquer avec le proxy NDIS.

Le TSP proxy NDIS est installé avec les systèmes d’exploitation Windows Server 2003, Windows XP, Windows 2000, Windows NT, Windows Millennium Edition, Windows 98 et Windows 95.
