# <a name="tspi_linedropnoowner"></a>TSPI \_ lineDropNoOwner

La fonction **TSPI \_ lineDropNoOwner** est obsolète. L’interface TAPI ne supprime plus automatiquement un appel si aucun propriétaire initial ne peut être identifié, afin de prendre en charge le contrôle distribué des appels sur plusieurs ordinateurs clients dans un environnement client/serveur.
