# <a name="raw-channel-access-msp"></a>MSP d’accès au canal brut

Le Rcsmsp.dll (RAW Channel Access MSP) diffuse via ksproxy vers RCA, qui permet la diffusion en continu sur des appareils NDIS (Network Driver Interface Specification) orientés connexion, tels que des cartes RNIS et ATM. Le MSP RCA est créé par le ndptsp, qui est le TSP fourni pour les adaptateurs WAN NDIS. Avec la version 2000 de Windows, seule la diffusion audio est implémentée.

Le MSP d’accès au canal brut est installé avec Windows 2000.
