# <a name="h323-tsp"></a>TSP H323

\[Le TSP H323 n’est pas disponible pour une utilisation dans Windows Vista, Windows Server 2008 et les versions ultérieures du système d’exploitation. L’API cliente RTC offre des fonctionnalités similaires.\]

Le TSP H323 (h323. tsp), en association avec le [MSP H323](./h323-msp.md) (H323msp.dll), permet à une application de prendre et de recevoir des appels vidéo et audio à l’aide du protocole H. 323 pour la téléphonie IP.

Ce TSP prend en charge les valeurs de [**type d’adresse**](./lineaddresstype--constants.md) LINEADDRESSTYPE \_ PHONENUMBER, LINEADDRESSTYPE \_ IPAddress et LINEADDRESSTYPE \_ nomdomaine. Le LINEADDRESSTYPE \_ nomdomaine fait référence au nom de domaine de l’ordinateur.

Le MSP H323 a été amélioré dans Windows 2000 SP1 pour permettre la communication entre les clients H323 et SDP.

Le TSP H323 est installé avec Windows 2000 et versions ultérieures.
