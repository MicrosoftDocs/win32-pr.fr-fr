# <a name="terminal"></a>Terminal

Un terminal est une source ou un convertisseur pour une adresse et un type de support particuliers. Le concept de terminal est principalement utilisé dans TAPI 3 et n’est que légèrement important pour les applications TAPI 2. Pour plus d’informations, consultez la vue d’ensemble de l' [objet terminal](terminal-object.md) TAPI 3.
