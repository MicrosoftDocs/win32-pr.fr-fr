# <a name="telephony-application-programming-interface-version-22"></a>Interface de programmation d’applications de téléphonie version 2,2

## <a name="purpose"></a>Objectif

L’interface de programmation d’applications de téléphonie (TAPI) Microsoft, version 2,2 (TAPI/C) permet l’implémentation d’applications de communication allant du contrôle de modem de base aux centres d’appel avec plusieurs agents et commutateurs.

## <a name="where-applicable"></a>Le cas échéant

Les applications TAPI 2,2 possibles sont les suivantes :

-   Appels vocaux de base sur le réseau téléphonique public commuté (RTPC)
-   Applications de centre d’appels pour le suivi de plusieurs agents
-   Contrôle de modem
-   Contrôle PBX
-   Systèmes de réponse vocale interactive (IVR)
-   Messagerie vocale
-   Contrôle détaillé de l’appareil téléphonique

## <a name="developer-audience"></a>Développeurs concernés

TAPI/C est conçu pour être utilisé par les programmeurs C/C++. L’expérience de développement avec les télécommunications ou d’autres applications de téléphonie est utile, mais pas nécessaire.

## <a name="run-time-requirements"></a>Conditions d’exécution

TAPI version 2,2 permet le développement d’applications de communication pour les systèmes d’exploitation Windows Server 2003, Windows XP, Windows 2000, Windows NT, Windows, Windows 98 et Windows 95. Pour plus d’informations sur les systèmes d’exploitation qui prennent en charge une fonction particulière, consultez la section Configuration requise de la documentation relative à cette fonction.
