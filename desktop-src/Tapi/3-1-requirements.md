# <a name="tapi-31-requirements"></a>Configuration requise pour TAPI 3,1

Voici la configuration requise pour le système d’exploitation pour TAPI 3. x :

-   TAPI version 3,0 requiert Windows 2000 ou version ultérieure.
-   TAPI version 3,1 requiert les systèmes d’exploitation Windows Server 2003 ou Windows XP.

Notez que le [contrôle de conférence IP](about-rendezvous-ip-telephony-conferencing.md) à l’aide de Rendezvous requiert l’accès à au moins un serveur ils ou NTDS. Si un serveur NTDS est en place, il peut être utilisé pour localiser un serveur ILS. S’il n’existe aucun serveur NTDS, l’utilisateur final doit connaître le nom du serveur ILS.
