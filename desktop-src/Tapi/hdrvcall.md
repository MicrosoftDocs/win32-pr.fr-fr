# <a name="hdrvcall"></a>HDRVCALL

Le type de données [**HDRVCALL**](hdrvline.md) représente le handle opaque d’un fournisseur de services pour une structure de données d’appel. Le fournisseur de services doit résoudre une valeur de ce type en une référence à l’instance de structure de données appropriée.
