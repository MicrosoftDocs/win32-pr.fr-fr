# <a name="itstream-mspi"></a>ITStream (MSPI)

L’interface [**ITStream**](/windows/win32/api/tapi3if/nn-tapi3if-itstream) est disponible si un MSP est associé à l’appel. Les méthodes de cette interface permettent à une application de récupérer des informations sur un flux de données ; pour démarrer, suspendre ou arrêter le flux ; et pour sélectionner ou désélectionner des terminaux sur un flux.

L’interface [**ITStream**](/windows/win32/api/tapi3if/nn-tapi3if-itstream) est implémentée par un MSP. Pour plus d’informations sur cette interface, consultez **ITStream** dans la section de l’interface MSP.
