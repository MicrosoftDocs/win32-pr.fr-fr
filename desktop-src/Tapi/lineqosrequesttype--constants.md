# <a name="lineqosrequesttype_-constants"></a>\_Constantes LINEQOSREQUESTTYPE

Ces constantes sont utilisées par un TSP pour identifier le type d’une demande QoS (Quality of service) qui est effectuée. À l’heure actuelle, seule une demande de niveau de service est prise en charge. Consultez [ \_ constantes LINEQOSSERVICELEVEL](lineqosservicelevel--constants.md).

<dl> <dt>

<span id="LINEQOSREQUESTTYPE_SERVICELEVEL"></span><span id="lineqosrequesttype_servicelevel"></span>**LINEQOSREQUESTTYPE \_ SERVICELEVEL**
</dt> <dd> <dl> <dt>

 0x00000001
</dt> <dt>



La demande de qualité de service effectuée concerne un niveau de service particulier.


</dt> </dl> </dd> </dl>
