# <a name="tapi-service-providers"></a>Fournisseurs de services TAPI

Les fournisseurs de services TAPI sont une couche d’abstraction entre les applications TAPI et le matériel et les protocoles de transport sous-jacents. Un [fournisseur de services de téléphonie](telephony-service-providers-start-page.md) (TSP) fournit des contrôles d’appel et un [fournisseur de services de média](./media-service-providers-start-page.md) (MSP) fournit un contrôle détaillé sur le média d’un appel. Une paire TSP/MSP peut implémenter les fonctionnalités nécessaires pour les applications allant de la téléphonie de base à la Conférence multimédia multidiffusion IP.
