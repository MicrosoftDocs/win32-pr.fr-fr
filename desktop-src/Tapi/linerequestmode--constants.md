# <a name="linerequestmode_-constants"></a>\_Constantes LINEREQUESTMODE

Les constantes d’indicateur de bit **LINEREQUESTMODE** décrivent différents types de demandes de téléphonie qui peuvent être effectuées d’une application à une autre.

<dl> <dt>

<span id="LINEREQUESTMODE_DROP"></span><span id="linerequestmode_drop"></span>**LINEREQUESTMODE \_ Drop**
</dt> <dd> <dl> <dt>



Demande [tapiRequestDrop](/windows/desktop/api/Tapi/nf-tapi-tapirequestdrop) .


</dt> </dl> </dd> <dt>

<span id="LINEREQUESTMODE_MAKECALL"></span><span id="linerequestmode_makecall"></span>**LINEREQUESTMODE \_ MAKECALL**
</dt> <dd> <dl> <dt>



Demande [tapiRequestMakeCall](/windows/desktop/api/Tapi/nf-tapi-tapirequestmakecall) .


</dt> </dl> </dd> <dt>

<span id="LINEREQUESTMODE_MEDIACALL"></span><span id="linerequestmode_mediacall"></span>**LINEREQUESTMODE \_ MEDIACALL**
</dt> <dd> <dl> <dt>



Cette constante est obsolète. Ne pas utiliser.


</dt> </dl> </dd> </dl>

## <a name="remarks"></a>Notes

Aucune extensibilité. Tous les 32 bits sont réservés.
