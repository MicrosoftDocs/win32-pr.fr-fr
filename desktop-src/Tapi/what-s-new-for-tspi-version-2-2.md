# <a name="whats-new-for-tspi-version-22"></a>Nouveautés de TSPI version 2,2

Les modifications apportées à la version 2,2 de TSPI se composent principalement d’un certain nombre de structures de données qui ont des éléments supplémentaires. La plupart de ces structures sont définies dans TAPI. h et sont utilisées par les applications TAPI et les fournisseurs de services comme un moyen de passer des informations.

Les fonctions et les messages qui ont été nouveaux ou modifiés pour TSPI version 2,2 sont les suivants :

-   [**LINE_AGENTSESSIONSTATUS**](./line-agentsessionstatus.md)
-   [**LINE_QUEUESTATUS**](./line-queuestatus.md)
-   [**LINE_AGENTSTATUSEX**](./line-agentstatusex.md)
-   [**LINE_GROUPSTATUS**](./line-groupstatus.md)
-   [**LINE_PROXYSTATUS**](./line-proxystatus.md)

**LINEADDRCAPFLAGS_ACDGROUP** membre de [LINEADDRCAPFLAGS_ constantes](./lineaddrcapflags--constants.md)

**LINEERR_INVALAGENTSESSIONSTATE** et **LINEERR_DISCONNECTED** les membres des [constantes LINEERR_](./lineerr--constants.md).

-   [Constantes LINEAGENTSTATEEX_](./lineagentstateex--constants.md)
-   [Constantes LINEAGENTSTATUSEX_](./lineagentstatusex--constants.md)
-   [Constantes LINEAGENTSESSIONSTATE_](./lineagentsessionstate--constants.md)
-   [Constantes LINEAGENTSESSIONSTATUS_](./lineagentsessionstatus--constants.md)
-   [Constantes LINEQUEUESTATUS_](./linequeuestatus--constants.md)
-   [Constantes LINEGROUPSTATUS_](./linegroupstatus--constants.md)
-   [Constantes LINEPROXYSTATUS_](./lineproxystatus--constants.md)
-   [Constantes LINEPROXYREQUEST_](./lineproxyrequest--constants.md)
