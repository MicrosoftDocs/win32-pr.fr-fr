# <a name="tapi-3-terminal-manager-reference"></a>Référence du gestionnaire de terminal TAPI 3



| Interfaces du gestionnaire des services Terminal Server                      | Description                                                                                                                                                                            |
|--------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**ITTerminalControl**](/windows/desktop/api/Termmgr/nn-termmgr-itterminalcontrol)   | Fournit des méthodes pour connecter et déconnecter un terminal à partir d’un graphique de filtre, et pour exécuter ou arrêter un convertisseur.                                                                                 |
| [**ITTerminalManager**](/windows/desktop/api/Termmgr/nn-termmgr-itterminalmanager)   | Fournit des méthodes pour créer des terminaux dynamiques.                                                                                                                                          |
| [**ITTerminalManager2**](/windows/desktop/api/Termmgr/nn-termmgr-itterminalmanager2) | Fournit des méthodes pour récupérer des informations sur les classes terminal enfichables inscrites dans le système actuel. dérive de l’interface [**ITTerminalManager**](/windows/desktop/api/Termmgr/nn-termmgr-itterminalmanager) . |
