# <a name="hdrvphone"></a>HDRVPHONE

Le type de données **HDRVPHONE** représente le descripteur opaque d’un fournisseur de services pour une structure de données de téléphone. Le fournisseur de services doit résoudre une valeur de ce type en une référence à l’instance de structure de données appropriée.
