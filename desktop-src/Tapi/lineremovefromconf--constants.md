# <a name="lineremovefromconf_-constants"></a>\_Constantes LINEREMOVEFROMCONF

Les constantes scalaires **LINEREMOVEFROMCONF \_** décrivent comment les tiers participant à un appel de conférence peuvent être supprimés d’un appel de conférence.

<dl> <dt>

<span id="LINEREMOVEFROMCONF_ANY"></span><span id="lineremovefromconf_any"></span>**LINEREMOVEFROMCONF \_ tout**
</dt> <dd> <dl> <dt>



Tout tiers participant peut être supprimé de l’appel de conférence.


</dt> </dl> </dd> <dt>

<span id="LINEREMOVEFROMCONF_LAST"></span><span id="lineremovefromconf_last"></span>**LINEREMOVEFROMCONF en \_ dernier**
</dt> <dd> <dl> <dt>



Seule la partie la plus récemment ajoutée peut être supprimée de l’appel de conférence


</dt> </dl> </dd> <dt>

<span id="LINEREMOVEFROMCONF_NONE"></span><span id="lineremovefromconf_none"></span>**LINEREMOVEFROMCONF \_ aucun**
</dt> <dd> <dl> <dt>



Les tiers ne peuvent pas être supprimés de l’appel de conférence.


</dt> </dl> </dd> </dl>

## <a name="remarks"></a>Notes

Aucune extensibilité. Tous les 32 bits sont réservés.
