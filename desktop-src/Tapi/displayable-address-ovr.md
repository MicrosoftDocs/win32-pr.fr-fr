# <a name="displayable-address"></a>Adresse affichable

L’adresse affichable est une chaîne qui présente l’adresse dans un format approprié pour l’affichage dans une interface utilisateur.

**TAPI 2. x :** Consultez les membres [**lineGetCallInfo**](/windows/win32/api/tapi/nf-tapi-linegetcallinfo), **dwDisplayableAddressSize** et **dwDisplayableAddressOffset** de [**LINECALLINFO**](/windows/win32/api/tapi/ns-tapi-linecallinfo).

**TAPI 3. x :** Consultez [**ITCallInfo :: obtenir \_ CallInfoString**](/windows/desktop/api/tapi3if/nf-tapi3if-itcallinfo-get_callinfostring) ou [**ITCallInfo ::p ut \_ CallInfoString**](/windows/desktop/api/tapi3if/nf-tapi3if-itcallinfo-put_callinfostring), appelée avec le membre **CIS \_ DISPLAYABLEADDRESS** de [**la \_ chaîne CALLINFO**](/windows/desktop/api/Tapi3if/ne-tapi3if-callinfo_string).
