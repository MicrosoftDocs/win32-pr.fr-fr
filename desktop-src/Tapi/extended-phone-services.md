# <a name="extended-phone-services"></a>Services téléphoniques étendus

Les services téléphoniques étendus (ou les services téléphoniques spécifiques aux appareils) incluent toutes les extensions de TSPI définies par le fournisseur de services. TSPI définit un mécanisme qui permet aux fournisseurs de fournisseurs de services d’étendre TSPI à l’aide d’extensions spécifiques à l’appareil. TSPI définit uniquement le mécanisme d’extension, ce qui permet d’accéder aux extensions spécifiques à l’appareil, dont le comportement est entièrement défini par le fournisseur de services.

TSPI se compose de définitions de constantes de données scalaires et de données d’indicateur de bit, de structures de données, de fonctions et de messages de rappel. Les procédures définies permettent à un fournisseur d’étendre la plupart de ces opérations, comme décrit dans les rubriques de cette section.
