# <a name="linequeuestatus_-constants"></a>\_Constantes LINEQUEUESTATUS

Les **\_ constantes LINEQUEUESTATUS** indiquent la modification de l’état d’une file d’attente ACD sur un gestionnaire d’agent.

<dl> <dt>

<span id="LINEQUEUESTATUS_NEWQUEUE"></span><span id="linequeuestatus_newqueue"></span>**LINEQUEUESTATUS \_ NEWQUEUE**
</dt> <dd> <dl> <dt>



Une file d’attente a été ajoutée à celles qui sont disponibles.


</dt> </dl> </dd> <dt>

<span id="LINEQUEUESTATUS_QUEUEREMOVED"></span><span id="linequeuestatus_queueremoved"></span>**LINEQUEUESTATUS \_ QUEUEREMOVED**
</dt> <dd> <dl> <dt>



Cette file d’attente a été supprimée de celles qui sont disponibles.


</dt> </dl> </dd> <dt>

<span id="LINEQUEUESTATUS_UPDATEINFO"></span><span id="linequeuestatus_updateinfo"></span>**LINEQUEUESTATUS \_ UPDATEINFO**
</dt> <dd> <dl> <dt>



Mise à jour des informations sur la file d’attente ACD.


</dt> </dl> </dd> </dl>
