# <a name="tapi-3-terminal-manager"></a>Gestionnaire de terminal TAPI 3

Le gestionnaire de terminal TAPI 3 est créé par Tapi3.dll à la demande d’un MSP et est utilisé par le MSP pour des tâches telles que la création et la suppression de terminaux dynamiques.

Les détails d’implémentation précis pour les terminaux dynamiques d’un MSP n’entrent pas dans le cadre de cette documentation. Toutefois, de nombreux auteurs peuvent choisir d’utiliser certaines des fonctionnalités du terminal de diffusion multimédia en continu (MST). Ce terminal est basé sur DirectShow et les filtres spécifiques à un protocole donné peuvent être écrits et utilisés conjointement avec le MST.
