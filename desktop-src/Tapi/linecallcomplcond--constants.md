# <a name="linecallcomplcond_-constants"></a>\_Constantes LINECALLCOMPLCOND

Les constantes d’indicateur de bit **LINECALLCOMPLCOND \_** décrivent les conditions dans lesquelles un appel peut être effectué.

<dl> <dt>

<span id="LINECALLCOMPLCOND_BUSY"></span><span id="linecallcomplcond_busy"></span>**LINECALLCOMPLCOND \_ occupé**
</dt> <dd> <dl> <dt>



Fin de l’appel sous conditions d’occupation.


</dt> </dl> </dd> <dt>

<span id="LINECALLCOMPLCOND_NOANSWER"></span><span id="linecallcomplcond_noanswer"></span>**LINECALLCOMPLCOND \_ NOanswer**
</dt> <dd> <dl> <dt>



Fin de l’appel sous le rappel de conditions de réponse.


</dt> </dl> </dd> </dl>

## <a name="remarks"></a>Remarques

Aucune extensibilité. Tous les 32 bits sont réservés.
