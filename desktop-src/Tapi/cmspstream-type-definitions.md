# <a name="cmspstream-type-definitions"></a>Définitions de type CMSPStream

Les définitions suivantes fournissent un ensemble d’États de flux qui peuvent être utiles à la classe dérivée, bien que leur utilisation ne soit pas obligatoire. Les classes dérivées peuvent facilement étendre l’ensemble des États disponibles en définissant des valeurs supplémentaires.

``` syntax
#define STRM_INITIAL            0x00000000
#define STRM_TERMINALSELECTED   0x00000001
#define STRM_CONFIGURED         0x00000002
#define STRM_RUNNING            0x00000004
#define STRM_PAUSED             0x00000008
#define STRM_STOPPED            0x00000010
```
