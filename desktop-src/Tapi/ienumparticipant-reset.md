# <a name="ienumparticipantreset-method"></a>IEnumParticipant :: Reset, méthode

\[La **réinitialisation** n’est pas disponible pour une utilisation dans Windows Vista, windows Server 2008 et les versions ultérieures du système d’exploitation. L’API cliente RTC offre des fonctionnalités similaires.\]

La méthode **Reset** réinitialise le début de la séquence d’énumération. Cette méthode est masquée dans les Visual Basic et les langages de script.
