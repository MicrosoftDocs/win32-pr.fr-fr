# <a name="l-telephony-api"></a>L (API de téléphonie)

<dl> <dt>

<span id="tapi2.line_device_tapgloss"></span><span id="TAPI2.LINE_DEVICE_TAPGLOSS"></span>**périphérique de ligne**
</dt> <dd>

Un canal physique composé d’une ligne téléphonique et du matériel pour le contrôler comme un tableau de télécopie, un modem ou une carte RNIS. Le type de périphérique de ligne le plus simple correspond à une seule ligne téléphonique.

</dd> <dt>

<span id="tapi2.line_device_class_tapgloss"></span><span id="TAPI2.LINE_DEVICE_CLASS_TAPGLOSS"></span>**classe de périphérique de ligne**
</dt> <dd>

Représentation indépendante du périphérique d’un périphérique de ligne physique, tel qu’un modem.

</dd> <dt>

<span id="tapi2.local_loop"></span><span id="TAPI2.LOCAL_LOOP"></span>**boucle locale**
</dt> <dd>

Partie du réseau téléphonique entre le téléphone et le Bureau de commutation centrale de l’entreprise téléphonique. Dans cette boucle, les paroles humaines d’un téléphone domestique sont généralement transmises au format analogique, et les données numériques d’un ordinateur doivent d’abord être converties en analogique par un modem. Les réseaux numériques remplacent progressivement les paramètres analogiques dans la boucle locale.

</dd> </dl>
