# <a name="d-telephony-api"></a>D (API de téléphonie)

[A](a-tapgloss.md) [B](b-tapgloss.md) [C](c-tapgloss.md) D [e](e-tapgloss.md) F G [H](h-tapgloss.md) [I](i-tapgloss.md) J [K](k-tapgloss.md) [L](l-tapgloss.md) [M](m-tapgloss.md) N O [P](p-tapgloss.md) Q [R](r-tapgloss.md) [S](s-tapgloss.md) [T](t-tapgloss.md) [U](u-tapgloss.md) [V](v-tapgloss.md) [W](w-tapgloss.md) X Y Z

<dl> <dt>

<span id="tapi2.d_channel_tapgloss"></span><span id="TAPI2.D_CHANNEL_TAPGLOSS"></span>**Canal D**
</dt> <dd>

Canal de données de l’interface RNIS. Il s’agit de l’un des trois canaux standard sur une ligne RNIS à débit de base. Sur une ligne BRI-RNIS, un canal D peut transporter des informations de signalisation et des données de paquets à vitesse faible à un débit de 16 000 bps. Sur une ligne PRI-RNIS, le canal D s’exécute à 64 000 bps. Voir [*interface de taux de base (BRI-RNIS)*](b-tapgloss.md), [*canal B*](b-tapgloss.md), [*interface de taux primaire (PRI-RNIS)*](p-tapgloss.md).

</dd> <dt>

<span id="tapi2.device_class_tapgloss"></span><span id="TAPI2.DEVICE_CLASS_TAPGLOSS"></span>**classe d’appareil**
</dt> <dd>

Groupe de périphériques physiques ou de pilotes de périphérique reliés par le biais desquels les applications envoient et reçoivent les informations ou les données qui constituent un appel. Pour obtenir une liste, consultez [classes de périphériques TAPI](tapi-device-classes.md).

</dd> <dt>

<span id="tapi2.did_tapgloss"></span><span id="TAPI2.DID_TAPGLOSS"></span>**N’A**
</dt> <dd>

Consultez *la page numérotation directe*.

</dd> <dt>

<span id="tapi2.direct_inward_dialing_did__tapgloss"></span><span id="TAPI2.DIRECT_INWARD_DIALING_DID__TAPGLOSS"></span>**numérotation directe (DID)**
</dt> <dd>

Fonctionnalité fournie par pratiquement tous les PBX modernes qui permet à un appelant externe d’atteindre une extension interne en numérotant un numéro de téléphone à sept chiffres standard sans passer par un opérateur ou une surveillance. Voir [*PBX (Private Branch Exchange)*](p-tapgloss.md).

</dd> </dl>
