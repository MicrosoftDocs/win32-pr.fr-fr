# <a name="itterminal-mspi"></a>ITTerminal (MSPI)

L’interface [**ITTerminal**](/windows/win32/api/tapi3if/nn-tapi3if-itterminal) est l’interface de base d’un [objet terminal](terminal-object.md). Il fournit des méthodes pour obtenir des informations telles que la classe terminal et le support pris en charge.

L’interface [**ITTerminal**](/windows/win32/api/tapi3if/nn-tapi3if-itterminal) est implémentée par un MSP et n’est pas disponible s’il n’existe aucun fournisseur de services multimédias associé à l’adresse. Pour plus d’informations sur cette interface, consultez **ITTerminal** dans la section de l’interface MSP.
