# <a name="using-rendezvous-ip-telephony-conferencing"></a>Utilisation de la Conférence de téléphonie IP Rendezvous

\[ Les interfaces et les contrôles de conférence de téléphonie IP Rendezvous ne peuvent pas être utilisés dans Windows Vista, Windows Server 2008 et les versions ultérieures du système d’exploitation. L’API cliente RTC offre des fonctionnalités similaires.\]

Rendezvous se compose de trois ensembles d’interfaces de base :

-   Les [contrôles d’annuaire](directory-controls.md) offrent un moyen de découvrir, de créer ou de modifier la Conférence et les informations utilisateur sur un serveur ils ou NTDS.
-   Les [contrôles d’objets BLOB de conférence](conference-blob-controls.md) autorisent les modifications spécifiques au fournisseur pour une conférence.
-   Les [interfaces COM de multidiffusion](multicast-com-interfaces.md) offrent un moyen de réserver des adresses en vue d’une utilisation par des conférences de multidiffusion.
