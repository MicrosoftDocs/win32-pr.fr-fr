# <a name="remote-tsp"></a>TSP distant

Le TSP distant (remotesp. tsp) permet d’accéder aux ressources de communication qui ne sont pas locales sur l’ordinateur actuel, par exemple un PBX. Lorsque le TSP distant est configuré pour pointer vers un serveur donné, les ressources apparaissent dans une application locale comme si elles existaient sur l’ordinateur de l’utilisateur.

Tcmsetup est un utilitaire de ligne de commande qui permet la configuration d’un TSP distant. pour plus d’informations sur cet utilitaire, consultez Windows d’aide ou le Kit de ressources Windows.

le TSP distant est installé avec les systèmes d’exploitation Windows Server 2003, Windows XP, Windows 2000, Windows NT, Windows Me, Windows 98 et Windows 95.
