# <a name="cbaseterminal"></a>CBaseTerminal

**CBaseTerminal** est une classe de base de terminal applicable aux terminaux statiques et dynamiques. Comme elle est entièrement générique, toute implémentation de terminal peut dériver de cette classe directement ou indirectement.

Défini dans Mspterm. h.
