# <a name="tapi-3x-vs-tapi-2x"></a>TAPI 3. x et TAPI 2. x

l’API de téléphonie COM, TAPI 3. x, est disponible à partir de Windows XP. TAPI 3. x fournit des outils de développement améliorés pour le monde moderne de la programmation de communication, où un « appel » peut être un flux vidéo sur un réseau basé sur IP, et aucun jeu de téléphone n’est impliqué dans la session. En règle générale, les créateurs d’applications recherchent TAPI 3. x offre une meilleure prise en charge de l’implémentation des contrôles de communication flexibles.

Toutefois, TAPI 2. x peut être le bon choix dans certaines situations. Si des modifications mineures sont nécessaires pour une application héritée, ou s’il existe des plateformes héritées à prendre en charge, la conversion vers l’API COM peut ne pas être rentable.
