# <a name="tapi-21"></a>TAPI 2,1

TAPI 2,1 fournit un ajout à la précédente spécification TAPI 2,0 : fonctionnalités client/serveur. Avec TAPI 2,1 et versions ultérieures, le matériel de téléphonie peut être installé sur un serveur de téléphonie accessible à partir de n’importe quel ordinateur d’un réseau. Un nouvel ensemble de fonctions permet d’écrire des dll d’administration pour contrôler qui a accès aux ressources distantes.

**Windows Me/98/95 :** TAPI 2,1 prend en charge 32 bits TSPs. Les binaires système TAPI 2,1, lors de l’installation de prennent en charge :

-   applications TAPI 1,3, TAPI 1,4 et TAPI 2. x 32 bits
-   TSPs TAPI 2. x 32 bits
-   applications TAPI 1,3 et TAPI 1,4 16 bits
-   TAPI 16 bits 1,3 et TAPI 1,4 TSPs
