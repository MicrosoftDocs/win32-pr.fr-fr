# <a name="b-telephony-api"></a>B (API de téléphonie)

[A](a-tapgloss.md) B [C](c-tapgloss.md) [D](d-tapgloss.md) [e](e-tapgloss.md) F G [H](h-tapgloss.md) [I](i-tapgloss.md) J [K](k-tapgloss.md) [L](l-tapgloss.md) [M](m-tapgloss.md) N O [P](p-tapgloss.md) Q [R](r-tapgloss.md) [S](s-tapgloss.md) [T](t-tapgloss.md) [U](u-tapgloss.md) [V](v-tapgloss.md) [W](w-tapgloss.md) X Y Z

<dl> <dt>

<span id="tapi2.b_channel_tapgloss"></span><span id="TAPI2.B_CHANNEL_TAPGLOSS"></span>**Canal B**
</dt> <dd>

Canal *porteur* qui est un composant fondamental des interfaces RNIS. Il comporte 64 000 bits par seconde (BPS) dans les deux sens, est commuté et peut transporter la voix ou les données. Pour plus d’informations, consultez [mode porteur](./bearer-mode-ovr.md). Consultez [*canal D*](d-tapgloss.md), [*réseau numérique à intégration de services (RNIS)*](i-tapgloss.md).

</dd> <dt>

<span id="tapi2.band_tapgloss"></span><span id="TAPI2.BAND_TAPGLOSS"></span>**élastique**
</dt> <dd>

Plage de fréquences entre deux points définis.

</dd> <dt>

<span id="tapi2.bandwidth_tapgloss"></span><span id="TAPI2.BANDWIDTH_TAPGLOSS"></span>**la**
</dt> <dd>

Dans les communications, différence entre les fréquences les plus élevées et les plus basses dans une plage donnée. Par exemple, un téléphone prend en charge une bande passante de 3000 Hz, la différence entre les fréquences les plus basses (300 Hz) et les plus élevées (3300 Hz) qu’il peut transporter.

</dd> <dt>

<span id="tapi2.bandwidth_allocation_control_protocol_bacp__tapgloss"></span><span id="TAPI2.BANDWIDTH_ALLOCATION_CONTROL_PROTOCOL_BACP__TAPGLOSS"></span>**Protocole BACP (Bandwidth Allocation Control Protocol)**
</dt> <dd>

Ensemble de règles qui gère la bande passante sur les connexions PPP à liaisons multiples dynamiques. Consultez [*Multi-Link PPP*](m-tapgloss.md), [*point-to-Point Protocol (PPP)*](p-tapgloss.md).

</dd> <dt>

<span id="tapi2.basic_rate_interface_bri_isdn__tapgloss"></span><span id="TAPI2.BASIC_RATE_INTERFACE_BRI_ISDN__TAPGLOSS"></span>**BRI-RNIS (Basic Rate Interface)**
</dt> <dd>

Ligne RNIS standard, composée de deux canaux B et d’un canal D. Consultez [*RNIS (réseau numérique à intégration de services)*](i-tapgloss.md), [*PRI-RNIS (Primary Rate Interface)*](p-tapgloss.md).

</dd> <dt>

<span id="tapi2.basic_telephony_tapgloss"></span><span id="TAPI2.BASIC_TELEPHONY_TAPGLOSS"></span>**Téléphonie de base**
</dt> <dd>

Un niveau élémentaire de service de téléphonie, avec un sous-ensemble minimal de fonctionnalités, qui fournit un ensemble minimal de fonctions qui correspondent à des POTS (Plain Old Telephone Service). Les fournisseurs de services TAPI sont requis pour prendre en charge toutes les fonctions de téléphonie de base. Voir [assistance assistée](./assisted-telephony-overview.md), téléphonie [*étendue*](e-tapgloss.md), [*téléphonie supplémentaire*](s-tapgloss.md).

</dd> <dt>

<span id="tapi2.bri_isdn_tapgloss"></span><span id="TAPI2.BRI_ISDN_TAPGLOSS"></span>**BRI-RNIS**
</dt> <dd>

Voir *interface de taux de base (BRI-RNIS)*.

</dd> <dt>

<span id="tapi2.bps_tapgloss"></span><span id="TAPI2.BPS_TAPGLOSS"></span>**Vitesse**
</dt> <dd>

Voir *bits par seconde (BPS)*.

</dd> <dt>

<span id="tapi2.bits_per_second_bps__tapgloss"></span><span id="TAPI2.BITS_PER_SECOND_BPS__TAPGLOSS"></span>**bits par seconde (BPS)**
</dt> <dd>

Mesure du taux de transfert indiquant le nombre de bits de données ordinateur envoyés pendant une période d’une seconde via un modem sur une ligne téléphonique.

</dd> </dl>
