# <a name="ipconf-tsp"></a>TSP IPConf

\[IPConf TSP ne peut pas être utilisé dans Windows Vista, Windows Server 2008 et les versions ultérieures du système d’exploitation. L’API cliente RTC offre des fonctionnalités similaires.\]

Le TSP IPConf (ipconf. tsp), en association avec le [MSP ipconf](./ipconf-msp.md) (Confmsp.dll), fournit un accès aux conférences aux réseaux IP, tels qu’Internet et de nombreux intranets d’entreprise.

Ce TSP prend en charge une valeur de [**type d’adresse**](./lineaddresstype--constants.md) du \_ SDP LINEADDRESSTYPE.

le MSP IPConf a été amélioré dans Windows 2000 SP1 pour permettre la communication entre les clients H323 et SDP.

le TSP IPConf est installé avec Windows 2000 et versions ultérieures.
