# <a name="xps-om-digital-signatures"></a>Signatures numériques XPS OM

Cette rubrique répertorie les interfaces de signature numérique de l’API de document XPS.

Le modèle d’objet XPS prend en charge les signatures numériques. Toutefois, il est également possible d’accéder directement aux signatures numériques dans un document XPS sans avoir à créer un modèle d’objet XPS. Pour plus d’informations sur l’accès aux signatures numériques XPS sans modèle XPS, consultez [API de signature numérique XPS](xps-digital-signatures.md).
