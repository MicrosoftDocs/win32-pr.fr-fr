# <a name="asynchronous-printing-notification-enumerations"></a>Énumérations de notification d’impression asynchrone

Les énumérations suivantes sont utilisées dans la communication asynchrone entre les applications et les composants qui sont hébergés par le spouleur d’impression, tels que les pilotes d’imprimante et les moniteurs de port.

## <a name="in-this-section"></a>Dans cette section



| Énumération                                                                               | Description                                                                                                                                                                                                    |
|-------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**PrintAsyncNotifyConversationStyle**](/windows/desktop/api/prnasnot/ne-prnasnot-printasyncnotifyconversationstyle)<br/> | Spécifie si la communication est bidirectionnelle ou unidirectionnelle entre les applications et les composants hébergés par le spouleur d’impression, tels que les pilotes d’imprimante, les processeurs d’impression et les moniteurs de port.<br/>           |
| [**PrintAsyncNotifyError**](/windows/desktop/api/prnasnot/ne-prnasnot-printasyncnotifyerror)<br/>                         | Spécifie la partie du code d’erreur du **HRESULT** retourné après un échec de notification asynchrone.<br/>                                                                                            |
| [**PrintAsyncNotifyUserFilter**](/windows/desktop/api/prnasnot/ne-prnasnot-printasyncnotifyuserfilter)<br/>               | Spécifie si les notifications vont uniquement aux applications d’écoute associées au même utilisateur que l’expéditeur hébergé par le spouleur d’impression ou à un ensemble plus large d’applications d’écoute.<br/> |
