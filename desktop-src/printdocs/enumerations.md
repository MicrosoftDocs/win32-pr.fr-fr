# <a name="print-spooler-api-enumerations"></a>Énumérations des API du spouleur d’impression

Cette section contient une liste alphabétique des énumérations utilisées par l’API spouleur d’impression.

## <a name="in-this-section"></a>Dans cette section



| Énumération                                                             | Description                                                                                                        |
|-------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------|
| [**EPrintXPSJobOperation**](eprintxpsjoboperation.md)<br/>       | Spécifie si un travail d’impression XPS est dans la phase de mise en file d’attente ou de rendu.<br/>                           |
| [**EPrintXPSJobProgress**](eprintxpsjobprogress.md)<br/>         | Spécifie ce que le spouleur effectue actuellement lors du traitement d’un travail d’impression XPS.<br/>                         |
| [**Imprimer \_ le \_ contexte d’exécution**](print-execution-context.md)<br/> | Représente le contexte d’exécution lorsque [**GetPrintExecutionData**](getprintexecutiondata.md) est appelé.<br/> |
| [**\_indicateurs d’option d’imprimante \_**](printer-option-flags.md)<br/>       | Spécifie la mise en cache d’un handle pour une imprimante ouverte avec [**OpenPrinter2**](openprinter2.md).<br/>       |
