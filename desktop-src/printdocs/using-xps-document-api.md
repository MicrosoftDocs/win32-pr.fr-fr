# <a name="using-xps-document-api"></a>Utilisation de l’API de document XPS

Cette section décrit comment utiliser l' [API de document XPS](documents-xps.md) pour effectuer des tâches de programmation.

Pour obtenir des exemples d’utilisation de l' [API de document XPS](documents-xps.md) dans un programme, consultez la section tâches de programmation de l' [API de document XPS](#xps-document-api-programming-tasks) .

Pour plus d’informations sur l’utilisation du modèle d’objet XPS et sur son implémentation par l' [API de document XPS](documents-xps.md), consultez [à propos de l’API de document XPS](about-xps-document-api.md).

## <a name="getting-started-with-the-xps-document-api"></a>Prise en main avec l’API de document XPS

Avant de commencer à utiliser l’API de document XPS, assurez-vous que vous êtes familiarisé avec les rubriques de programmation suivantes :<dl>

[Programmation COM](/windows/desktop/com/component-object-model--com--portal)  
[Gestion des erreurs dans COM](/windows/desktop/com/error-handling-in-com)  
</dl>

Lorsque vous utilisez l’API de document XPS, vous pouvez également utiliser les technologies suivantes :<dl>

[DirectWrite](/windows/desktop/DirectWrite/direct-write-portal)  
[API d’impression XPS](./printing-with-the-xpsprint-api.md)  
[API de signature numérique XPS](xps-digital-signatures.md)  
</dl>
