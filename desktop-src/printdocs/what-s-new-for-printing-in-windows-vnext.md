# <a name="whats-new-for-printing"></a>Nouveautés pour l’impression

Windows 8 prend en charge [l’impression pour les applications du Windows Store en JavaScript et html](/previous-versions/windows/apps/hh465225(v=win.10)) , et [l’impression pour les applications du Windows Store en C#/VB/C + + et XAML](/previous-versions/windows/apps/hh465196(v=win.10)).

Windows 8 inclut également une nouvelle API COM qui offre une prise en charge complète de l' [impression OpenXPS](app-support-for-openxps-printing.md) et de l’accès aux portions des nouveaux pilotes d’imprimante pris en charge par Windows 8. Pour plus d’informations sur la nouvelle API, consultez [Print document package API](tailored-app-printing-api.md).

Le programme de boîte de réception du pilote d’impression Windows vérifie que Windows 8 prend en charge un pourcentage élevé des imprimantes populaires.

| [Nouveautés pour l’impression dans Windows Vista](./what-s-new-in-windows-vista.md)<br/> | Répertorie les interfaces de programmation d’applications (API) d’impression introduites dans Windows Vista.<br/> |
