# <a name="whats-new-for-printing"></a>Nouveautés pour l’impression

Windows 8 prend en charge [l’impression pour Windows les applications du windows store en JavaScript et HTML](/previous-versions/windows/apps/hh465225(v=win.10)) et [l’impression pour Windows les applications du windows store en C#/VB/c + + et XAML](/previous-versions/windows/apps/hh465196(v=win.10)).

Windows 8 inclut également une nouvelle API basée sur COM qui prend entièrement en charge l' [impression OpenXPS](app-support-for-openxps-printing.md) et l’accès aux portions des nouveaux pilotes d’imprimante pris en charge par Windows 8. Pour plus d’informations sur la nouvelle API, consultez [Print document package API](tailored-app-printing-api.md).

le programme de boîte de réception du pilote d’impression Windows permet de s’assurer que Windows 8 prend en charge un pourcentage élevé des imprimantes populaires.

Les rubriques suivantes décrivent les fonctionnalités d’impression et de document qui ont été introduites dans les versions antérieures de Windows.

| [nouveautés pour l’impression dans Windows Vista](./what-s-new-in-windows-vista.md)<br/> | répertorie les interfaces de programmation d’applications (api) d’impression introduites dans Windows Vista.<br/> |
