# <a name="printticket-public-keywords"></a>Mots clés publics PrintTicket

Cette rubrique n’est pas à jour. Pour obtenir les informations les plus récentes, consultez la [spécification du schéma d’impression](https://download.microsoft.com/download/D/E/C/DECA6E6B-3E81-48E7-B7EF-6D92A547D03C/print-schema-spec-2-0.zip).

La section suivante spécifie les descriptions de propriétés et les initialiseurs de paramètres spécifiques à PrintTicket qui peuvent s’appliquer à un PrintTicket.

## <a name="printticket-specific-property-descriptions-usage"></a>Utilisation des descriptions de propriétés spécifiques à PrintTicket

Les descripteurs de propriété spécifiques à PrintTicket représentent les mots clés publics du schéma d’impression qui peuvent apparaître dans un PrintTicket. Elles contiennent des informations spécifiques qu’une application ou un utilisateur peut vouloir spécifier dans un PrintTicket.

## <a name="parameter-initializers-usage"></a>Utilisation des initialiseurs de paramètres

Les initialiseurs de paramètres prennent la forme de types d’éléments ParameterInit dans un PrintTicket. Les types d’éléments ParameterInit contiennent la valeur du paramètre et sont initialisés dans un PrintTicket. Pour plus d’informations, consultez la section [constructions de paramètres](parameter-constructs.md) .
