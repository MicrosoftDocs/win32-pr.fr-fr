# <a name="xps-print-api-functions"></a>Fonctions de l’API d’impression XPS

\[Les fonctions décrites dans cette section sont déconseillées. Les applications clientes doivent utiliser la référence de l' [API package de documents d’impression](./tailored-app-printing-api.md) à la place.\]

\[StartXpsPrintJob n’est pas pris en charge et peut être modifié ou non disponible à l’avenir. \]

\[StartXpsPrintJob1 n’est pas pris en charge et peut être modifié ou non disponible à l’avenir. \]

## <a name="in-this-section"></a>Dans cette section



| Fonction                                                  | Description                                                                   |
|-----------------------------------------------------------|-------------------------------------------------------------------------------|
| [**StartXpsPrintJob**](/windows/desktop/api/XpsPrint/nf-xpsprint-startxpsprintjob)<br/>   | Démarre l’impression d’un flux de document XPS sur une imprimante.<br/>               |
| [**StartXpsPrintJob1**](/windows/desktop/api/XpsPrint/nf-xpsprint-startxpsprintjob1)<br/> | Crée un travail d’impression pour l’envoi de contenu de document XPS à une imprimante.<br/> |
