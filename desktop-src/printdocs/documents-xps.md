# <a name="xps-document-api"></a>API de document XPS

l’api de Document XPS est une api de Windows native qui permet aux programmes de créer, lire, modifier, enregistrer et imprimer des documents XPS. l’API de Document XPS est prise en charge dans Windows 7 et les versions ultérieures de Windows et peut être utilisée dans les programmes en mode utilisateur et les pilotes d’imprimante XPSDrv.

l’API de Document xps prend à la fois en tant que xps et OpenXPS, et est conçue pour les programmeurs C++ qui écrivent des applications Windows natives. Pour plus d’informations sur la spécification Open XPS, consultez [Open XML Paper Specification (en anglais) (OpenXPS)](https://www.ecma-international.org/publications/standards/Ecma-388.htm).
