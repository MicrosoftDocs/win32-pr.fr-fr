## <a name="disclaimer"></a>Clause d'exclusion de responsabilité

Les exemples de code ne sont pas destinés à être des programmes complets et fonctionnels. Les exemples de code qui sont référencés sur cette page, par exemple, n’effectuent pas la vérification des paramètres, la vérification des erreurs, la gestion complète de la mémoire ou des ressources ou la gestion des erreurs. Utilisez ces exemples comme point de départ, puis ajoutez le code nécessaire pour créer une application fiable. Pour plus d’informations sur les valeurs de retour **HRESULT** et les stratégies de gestion des erreurs, consultez [gestion des erreurs dans com](../com/error-handling-in-com.md).

Par souci de clarté, ces exemples de code utilisent des scénarios de document et de signature XPS très simples, qui peuvent ne pas être suffisamment complexes pour répondre à vos besoins.
