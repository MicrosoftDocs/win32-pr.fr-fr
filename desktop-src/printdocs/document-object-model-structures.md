# <a name="xps-document-structures"></a>Structures de document XPS

## <a name="contents"></a>Contenu

## <a name="in-this-section"></a>Dans cette section



| Structure                                                   | Description                                                                 |
|-------------------------------------------------------------|-----------------------------------------------------------------------------|
| [**\_couleur XPS**](xps-color.md)<br/>                  | Structure qui décrit une seule valeur de couleur.<br/>                 |
| [**\_tiret XPS**](/windows/win32/api/xpsobjectmodel/ns-xpsobjectmodel-xps_dash)<br/>                    | Cette structure décrit un élément Dash d’un chemin d’accès.<br/>               |
| [**\_index de glyphes XPS \_**](/windows/win32/api/xpsobjectmodel/ns-xpsobjectmodel-xps_glyph_index)<br/>     | Décrit le positionnement et l’emplacement d’un glyphe.<br/>                 |
| [**\_mappage de glyphes XPS \_**](/windows/win32/api/xpsobjectmodel/ns-xpsobjectmodel-xps_glyph_mapping)<br/> | Décrit un mappage de glyphe à index.<br/>                              |
| [**\_matrice XPS**](/windows/win32/api/xpsobjectmodel/ns-xpsobjectmodel-xps_matrix)<br/>                | Décrit les deux colonnes gauches d’une matrice 3 par 3.<br/>               |
| [**\_point XPS**](/windows/win32/api/xpsobjectmodel/ns-xpsobjectmodel-xps_point)<br/>                  | Représente une paire de coordonnées x et y dans l'espace à deux dimensions.<br/> |
| [**\_Rect XPS**](/windows/win32/api/xpsobjectmodel/ns-xpsobjectmodel-xps_rect)<br/>                    | Décrit la largeur, la hauteur et l’emplacement d’un rectangle.<br/>        |
| [**\_taille XPS**](/windows/win32/api/xpsobjectmodel/ns-xpsobjectmodel-xps_size)<br/>                    | Décrit la taille d’un objet.<br/>                                 |
