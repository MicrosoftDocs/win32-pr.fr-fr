# <a name="xps-om-trunk-interfaces"></a>Interfaces du Trunk XPS OM

Les interfaces de niveau Trunk de l’API de document XPS décrivent la structure d’un document XPS lorsque le modèle d’objet XPS le représente.

<dl>

[Interfaces du package OM XPS](xps-object-model-package-interfaces.md) Les interfaces de package décrivent les fonctionnalités et les opérations qui agissent sur le niveau supérieur d’un modèle d’objet XPS.  
[Interfaces de document XPS OM](xps-object-model-document-interfaces.md) Les interfaces de document décrivent l’organisation et la séquence du FixedDocuments du package et des documents FixedPages.  
[Interfaces du composant XPS OM](working-with-xps-object-model-part-interfaces.md) Les interfaces de composant décrivent les ressources utilisées par le document et ses composants.  
</dl>
