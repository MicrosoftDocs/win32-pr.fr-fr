# <a name="xps-print-api-interfaces"></a>Interfaces de l’API d’impression XPS

\[Les interfaces décrites dans cette section sont déconseillées. Les applications clientes doivent utiliser l' [API Print document package](./tailored-app-printing-api.md) à la place.\]

\[IXpsPrintJob n’est pas pris en charge et peut être modifié ou non disponible à l’avenir. \]

\[IXpsPrintJobStream n’est pas pris en charge et peut être modifié ou non disponible à l’avenir. \]

## <a name="in-this-section"></a>Dans cette section



| Interface                                                   | Description                                                                               |
|-------------------------------------------------------------|-------------------------------------------------------------------------------------------|
| [**IXpsPrintJob**](/windows/desktop/api/XpsPrint/nn-xpsprint-ixpsprintjob)<br/>             | Permet d’accéder à un travail d’impression qui est actuellement en cours.<br/>                  |
| [**IXpsPrintJobStream**](/windows/desktop/api/XpsPrint/nn-xpsprint-ixpsprintjobstream)<br/> | Interface de flux en écriture seule dans laquelle une application écrit des données de travail d’impression.<br/> |
