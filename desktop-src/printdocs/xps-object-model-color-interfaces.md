# <a name="xps-om-color-management"></a>Gestion des couleurs XPS OM

Cette rubrique répertorie les interfaces de gestion des couleurs de l’API de document XPS dans un modèle d’objet XPS.

-   [**Interface IXpsOMColorProfileResource**](/windows/desktop/api/xpsobjectmodel/nn-xpsobjectmodel-ixpsomcolorprofileresource)
-   [**Interface IXpsOMColorProfileResourceCollection**](/windows/desktop/api/xpsobjectmodel/nn-xpsobjectmodel-ixpsomcolorprofileresourcecollection)

Lors de l’utilisation d’une structure de [**\_ couleur XPS**](xps-color.md) , une ressource de profil de couleurs est requise lorsque la valeur du membre **colorType** est un **contexte de \_ \_ type \_ de couleur XPS**.
