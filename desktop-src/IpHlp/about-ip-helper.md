# <a name="about-ip-helper"></a><span data-ttu-id="364f6-103">À propos de l’assistance IP</span><span class="sxs-lookup"><span data-stu-id="364f6-103">About IP Helper</span></span>

<span data-ttu-id="364f6-104">Internet Protocol Helper (assistance IP) aide l’administration réseau de l’ordinateur local en permettant aux applications de récupérer des informations sur la configuration réseau de l’ordinateur local et de modifier cette configuration.</span><span class="sxs-lookup"><span data-stu-id="364f6-104">Internet Protocol Helper (IP Helper) assists network administration of the local computer by enabling applications to retrieve information about the network configuration of the local computer, and to modify that configuration.</span></span> <span data-ttu-id="364f6-105">L’assistance IP fournit également des mécanismes de notification pour garantir qu’une application est avertie lorsque certains aspects de la modification de la configuration du réseau de l’ordinateur local sont modifiés.</span><span class="sxs-lookup"><span data-stu-id="364f6-105">IP Helper also provides notification mechanisms to ensure that an application is notified when certain aspects of the local computer network configuration change.</span></span>

<span data-ttu-id="364f6-106">La plupart des fonctions d’assistance IP passent des paramètres de structure qui représentent des types de données associés à la technologie de [base d’informations de gestion](/previous-versions/windows/desktop/mib/about-management-information-base) .</span><span class="sxs-lookup"><span data-stu-id="364f6-106">Many of the IP Helper functions pass structure parameters that represent data types associated with the [Management Information Base](/previous-versions/windows/desktop/mib/about-management-information-base) technology.</span></span> <span data-ttu-id="364f6-107">Les fonctions d’assistance IP utilisent ces structures pour représenter diverses informations de mise en réseau, telles que les entrées de cache ARP.</span><span class="sxs-lookup"><span data-stu-id="364f6-107">The IP Helper functions use these structures to represent various networking information, such as ARP cache entries.</span></span> <span data-ttu-id="364f6-108">Étant donné que ces structures sont également utilisées par l’API MIB, elles sont décrites dans la [référence de base d’informations de gestion](/previous-versions/windows/desktop/mib/management-information-base-reference).</span><span class="sxs-lookup"><span data-stu-id="364f6-108">Since these structures are also used by the MIB API, they are described in the [Management Information Base Reference](/previous-versions/windows/desktop/mib/management-information-base-reference).</span></span> <span data-ttu-id="364f6-109">Bien que l’API d’assistance IP utilise ces structures, l’application d’assistance IP est distincte de la base d’informations de gestion (MIB) et du protocole SNMP (simple Network Management Protocol).</span><span class="sxs-lookup"><span data-stu-id="364f6-109">Although the IP Helper API uses these structures, IP Helper is distinct from the Management Information Base (MIB) and Simple Network Management Protocol (SNMP).</span></span>

<span data-ttu-id="364f6-110">La documentation de l’assistance IP utilise les termes « adaptateur » et « interface » de manière intensive.</span><span class="sxs-lookup"><span data-stu-id="364f6-110">The IP Helper documentation uses the terms "adapter" and "interface" extensively.</span></span> <span data-ttu-id="364f6-111">Un « adaptateur » est un terme hérité qui est une forme abrégée de carte réseau, qui a initialement fait l’appel d’une certaine forme de matériel réseau.</span><span class="sxs-lookup"><span data-stu-id="364f6-111">An "adapter" is a legacy term that is an abbreviated form of network adapter which originally referred to some form of network hardware.</span></span> <span data-ttu-id="364f6-112">Un adaptateur est une abstraction au niveau de Datalink.</span><span class="sxs-lookup"><span data-stu-id="364f6-112">An adapter is a datalink-level abstraction.</span></span>

<span data-ttu-id="364f6-113">Une « interface » est un terme plus récent utilisé dans les documents RFC de l’IETF.</span><span class="sxs-lookup"><span data-stu-id="364f6-113">An "interface" is a newer term used in the IETF RFC documents.</span></span> <span data-ttu-id="364f6-114">Les documents RFC définissent une interface en tant que concept abstrait qui représente la pièce jointe d’un nœud à un lien.</span><span class="sxs-lookup"><span data-stu-id="364f6-114">The RFCs define an interface as an abstract concept that represents a node's attachment to a link.</span></span> <span data-ttu-id="364f6-115">Une interface est une abstraction au niveau de l’adresse IP.</span><span class="sxs-lookup"><span data-stu-id="364f6-115">An interface is an IP-level abstraction.</span></span>

<span data-ttu-id="364f6-116">Les termes de l’adaptateur et de l’interface sont utilisés de manière interchangeable dans la documentation de l’assistance IP.</span><span class="sxs-lookup"><span data-stu-id="364f6-116">The adapter and interface terms are used somewhat interchangeably in the IP Helper documentation.</span></span> <span data-ttu-id="364f6-117">Dans la documentation de l’assistance IP, une liste de cartes peut inclure une interface de réseau étendu (WAN) logicielle, ainsi que l’interface de bouclage (aucune d’elles ne fait référence à un périphérique matériel).</span><span class="sxs-lookup"><span data-stu-id="364f6-117">In IP Helper documentation, a list of adapters could include a software WAN interface as well as the loopback interface (neither of these refer to a hardware device).</span></span> <span data-ttu-id="364f6-118">Dans les API d’assistance IP, il existe un mappage un-à-un entre les adaptateurs et les interfaces.</span><span class="sxs-lookup"><span data-stu-id="364f6-118">In the IP Helper APIs, there is a one-to-one mapping of adapters to interfaces.</span></span>
