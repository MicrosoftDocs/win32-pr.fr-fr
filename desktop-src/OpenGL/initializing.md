# <a name="initializing"></a>Initialisation

Avec GLU version 1,1 ou ultérieure, [**gluGetString**](glugetstring.md) retourne le numéro de version de la bibliothèque Glu ou le numéro de version et les appels d’extension Glu spécifiques au fournisseur.

Le prototype de **gluGetString** est le suivant :

const **GLubyte** \* **gluGetString**(**GLenum** *Name* );
