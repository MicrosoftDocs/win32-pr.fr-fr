# <a name="header-files"></a>Fichiers d’en-tête

Applications qui utilisent :

-   Les fonctions OpenGL principales doivent inclure le fichier d’en-tête <GL \\ GL. h>.
-   La bibliothèque d’utilitaires OpenGL doit inclure le fichier d’en-tête <GL \\ Glu. h>.
-   La bibliothèque auxiliaire du Guide de programmation OpenGL doit inclure le fichier d’en-tête <la \\> GL glaux. h.
-   Les fonctions WGL doivent inclure le fichier d’en-tête Windows. h.
-   les fonctions Windows qui prennent en charge l’implémentation de OpenGL par Microsoft dans Windows doivent inclure le fichier d’en-tête Windows. h.
