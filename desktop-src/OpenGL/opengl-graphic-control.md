# <a name="opengl-graphic-control"></a>Contrôle graphique OpenGL

OpenGL vous offre un contrôle relativement direct sur les opérations fondamentales des graphiques en deux et trois dimensions. Cela comprend la spécification des paramètres tels que les matrices de transformation, les coefficients d’équation d’éclairage, les méthodes d’anticrénelage et les opérateurs de mise à jour de pixels. Toutefois, il ne vous fournit aucun moyen de décrire ou de modéliser des objets géométriques complexes. Ainsi, les commandes OpenGL que vous émettez spécifient la façon dont un résultat donné doit être produit (quelle procédure doit être suivie) plutôt que ce qui doit ressembler exactement à ce résultat. Autrement dit, OpenGL est fondamentalement procédural plutôt que descriptif. Pour comprendre parfaitement comment utiliser OpenGL, il est utile de connaître l’ordre dans lequel il effectue ses opérations.
