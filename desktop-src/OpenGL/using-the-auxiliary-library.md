# <a name="using-the-auxiliary-library"></a>Utilisation de la bibliothèque auxiliaire

À l’aide de OpenGL, Silicon Graphics (SGI) a créé la bibliothèque auxiliaire pour écrire des exemples de programmes simples pour le *Guide de programmation OpenGL*. Le code source de la bibliothèque auxiliaire est fourni avec le kit de développement logiciel (SDK) de la plate-forme Microsoft, ainsi que les exemples OpenGL. Pour comprendre comment la bibliothèque auxiliaire a été développée à partir de fonctions et de routines OpenGL, examinez le code source. Vous pouvez utiliser des fonctions de bibliothèque auxiliaire dans vos propres programmes. Pour obtenir une description de la bibliothèque auxiliaire, consultez le *Guide de programmation OpenGL*.
