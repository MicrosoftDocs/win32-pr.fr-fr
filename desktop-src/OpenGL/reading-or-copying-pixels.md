# <a name="reading-or-copying-pixels"></a>Lecture ou copie de pixels

Vous pouvez lire les pixels du trame dans la mémoire, les Encoder de différentes façons et stocker le résultat encodé en mémoire avec [**glReadPixels**](glreadpixels.md). En outre, vous pouvez copier un rectangle de valeurs de pixel d’une région du trame vers une autre à l’aide de [**glCopyPixels**](glcopypixels.md). La fonction [**glReadBuffer**](glreadbuffer.md) contrôle la couleur de la mémoire tampon à partir de laquelle les pixels sont lus ou copiés.
