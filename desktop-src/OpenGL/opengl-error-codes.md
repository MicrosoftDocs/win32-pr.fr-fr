# <a name="opengl-error-codes"></a>Codes d’erreur OpenGL

OpenGL comprend les codes d’erreur suivants.



| Code d'erreur             | Description                                 |
|------------------------|---------------------------------------------|
| \_enum GL non valide \_      | Argument **GLenum** hors limites.           |
| \_valeur non valide du GL \_     | Argument numérique hors limites.              |
| \_opération non valide du GL \_ | Opération non conforme dans l’état actuel.         |
| dépassement de capacité de la \_ pile GL \_    | La fonction provoquerait un dépassement de capacité de la pile.      |
| DÉPASSEMENT de capacité de la \_ pile GL \_   | La fonction provoquerait un dépassement de capacité négatif de la pile.     |
| \_mémoire insuffisante \_ du GL \_    | Mémoire restante insuffisante pour exécuter la fonction. |
