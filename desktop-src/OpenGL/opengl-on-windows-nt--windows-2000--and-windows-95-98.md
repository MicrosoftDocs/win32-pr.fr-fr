# <a name="opengl-on-windows"></a>OpenGL sur Windows

L’implémentation Microsoft d’OpenGL dans le système d’exploitation Microsoft Windows est une implémentation de l’interface de logiciel graphique standard OpenGL à trois dimensions (3d), avec laquelle les programmeurs créent des images de couleur 3D, toujours et animées de haute qualité. Cette vue d’ensemble décrit l’implémentation Windows de OpenGL.
