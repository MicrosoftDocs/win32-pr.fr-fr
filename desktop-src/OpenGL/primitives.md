# <a name="primitives-opengl"></a>Primitives (OpenGL)

Au cours de l’étape suivante du traitement, les primitives sont converties en fragments de pixel dans les étapes suivantes :

-   Les primitives sont découpées correctement.
-   Les ajustements correspondants nécessaires sont apportés aux données de couleur et de texture, et les coordonnées pertinentes sont transformées en coordonnées de fenêtre.
-   La pixellisation convertit les primitives découpées en fragments de pixels.

Pour plus d'informations, voir les rubriques suivantes :

-   [Découpage](clipping.md)
-   [Transformer en coordonnées de fenêtre](transforming-to-window-coordinates.md)
-   [Rastérisation](rasterizing.md)
-   [Référence des primitives](primitives-reference.md)
