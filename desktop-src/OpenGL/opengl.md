# <a name="opengl"></a>Technologie

## <a name="purpose"></a>Objectif

En tant qu’interface logicielle pour le matériel graphique, OpenGL effectue le rendu des objets multidimensionnels dans un trame. L’implémentation Microsoft d’OpenGL pour le système d’exploitation Windows est un logiciel graphique standard avec lequel les programmeurs peuvent créer des images de couleur à trois dimensions toujours et animées de haute qualité. La version d’OpenGL décrite dans cette section est 1,1.

Pour plus d’informations sur OpenGL ES exécuté sur Windows, consultez [angle pour le Windows Store](https://github.com/microsoft/angle/wiki).

## <a name="where-applicable"></a>Le cas échéant

OpenGL est conçu pour la compatibilité sur le matériel et les systèmes d’exploitation. Cette architecture facilite le portage de programmes OpenGL d’un système vers un autre. Même si chaque système d’exploitation a des exigences uniques, le code OpenGL dans de nombreux programmes peut être utilisé tel quel.

## <a name="developer-audience"></a>Développeurs concernés

Conçu pour être utilisé par les programmeurs C/C++, OpenGL requiert une connaissance de l’interface utilisateur graphique de Windows et de l’architecture pilotée par les messages.
