# <a name="appendix-opengl"></a>Annexe (OpenGL)

OpenGL, développé à l’origine par Silicon Graphics Incorporated (SGI) pour leurs stations de travail graphiques, permet aux applications de créer des images couleur de haute qualité, indépendamment des systèmes de fenêtrage, des systèmes d’exploitation et du matériel.

Le Comité de révision de l’architecture OpenGL (ARB), un consortium industriel, est actuellement responsable de la définition de OpenGL. Les membres du ARB incluent Silicon Graphics Incorporated, Microsoft Corporation, Intel, IBM et Digital Equipment Corporation.

Le document de référence officiel pour OpenGL version 1 est le *Manuel de référence OpenGL*, par le tableau de révision de l’architecture OpenGL (ISBN 0-201-63276-4). Le guide officiel d’apprentissage d’OpenGL version 1, est le *Guide de programmation OpenGL*, de Jackie Neider, Tom Davis et Mason Python (ISBN 0-201-63274-8). Les deux livres sont publiés par Addison-Wesley.
