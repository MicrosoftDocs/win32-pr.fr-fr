# <a name="p-opengl"></a>P (OpenGL)

[A](a.md) [B](b.md) [C](c.md) [D](d.md) [e](e.md) [F](f.md) [G](g.md) [H](h.md) [I](i.md) [J K](jk.md) [L](l.md) [M](m.md) [N](n.md) [O](o.md) P [Q](q.md) [R](r.md) [S](s.md) [T](t.md) [U V](u-v.md) [W](w.md) [X Y Z](x-y-z.md)

<dl> <dt>

<span id="opengl_parameter"></span><span id="OPENGL_PARAMETER"></span>**paramètre**
</dt> <dd>

Valeur passée comme argument à une commande OpenGL. Parfois, l’une des valeurs passées par référence à une commande OpenGL.

</dd> <dt>

<span id="opengl_perspective_division"></span><span id="OPENGL_PERSPECTIVE_DIVISION"></span>**Division de perspective**
</dt> <dd>

Division de x, y et z par w, effectuée en coordonnées de clip. Voir aussi [coordonnées](c.md)de l’élément.

</dd> <dt>

<span id="opengl_pixel"></span><span id="OPENGL_PIXEL"></span>**pixellisé**
</dt> <dd>

Élément image. Les bits à l’emplacement (x, y) de tous les bitplanes dans le trame constituent le pixel unique (x, y). Dans une image de la mémoire du client, un pixel est un groupe d’éléments. Dans les coordonnées de fenêtre OpenGL, chaque pixel correspond à une zone d’écran 1.0 x 1.0. Les coordonnées de l’angle inférieur gauche des noms de Pixel x, y sont (x, y) et le coin supérieur droit sont (x + 1, y + 1).

</dd> <dt>

<span id="opengl_point"></span><span id="OPENGL_POINT"></span>**point**
</dt> <dd>

Emplacement exact dans l’espace, qui est rendu sous la forme d’un point de diamètre fini.

</dd> <dt>

<span id="opengl_polygon"></span><span id="OPENGL_POLYGON"></span>**Polygone**
</dt> <dd>

Surface quasi-planaire délimitée par des arêtes spécifiées par des sommets. Chaque triangle d’un maillage de triangle est un polygone, comme chaque quadrilatère d’un filet quadrangulaires. Le rectangle spécifié par glRect est également un polygone.

</dd> <dt>

<span id="opengl_primitive"></span><span id="OPENGL_PRIMITIVE"></span>**primitifs**
</dt> <dd>

Forme (telle qu’un point, une ligne, un polygone, une image bitmap ou une image) qui peut être dessinée, stockée et manipulée en tant qu’entité discrète ; éléments à partir desquels les conceptions graphiques volumineuses sont créées.

</dd> <dt>

<span id="opengl_projection_matrix"></span><span id="OPENGL_PROJECTION_MATRIX"></span>**matrice de projection**
</dt> <dd>

Matrice 4x4 qui transforme des points, des lignes, des polygones et des positions raster des coordonnées oculaires aux coordonnées de l’élément.

</dd> </dl>
