# <a name="porting-to-opengl"></a>Portage vers OpenGL

OpenGL est conçu pour la compatibilité sur le matériel et les systèmes d’exploitation. Cette conception permet aux programmeurs de déplacer plus facilement les programmes OpenGL d’un système à un autre. Même si chaque système d’exploitation a des exigences uniques, une grande partie du code OpenGL dans vos programmes actuels peut être utilisé tel quel. pour porter votre application OpenGL sur Microsoft Windows, vous devez modifier vos programmes pour qu’ils fonctionnent avec les systèmes de fenêtrage Windows.

dans, les applications générales sont reportées à OpenGL pour Windows à partir de l’une des deux plateformes suivantes :

-   Applications OpenGL développées pour le système X Window et la bibliothèque X (Xlib)
-   Applications du GL IRIS

Pour plus d'informations, voir les rubriques suivantes :

-   [Fonctions OpenGL et leurs équivalents de la comptabilité IRIS](opengl-functions-and-their-iris-gl-equivalents.md)
-   [Différences entre l’IRIS dans le GL et OpenGL](iris-gl-and-opengl-differences.md)
