# <a name="porting-other-glx-code"></a>Portage d’autres codes GLX

Outre les fonctions Xlib et GLX décrites dans les sections précédentes, votre programme contient probablement certaines des autres fonctions GLX ou Xlib répertoriées dans conversion de [la bibliothèque glx](translating-the-glx-library.md). Réécrivez votre code système X Window sur Windows en remplaçant les fonctions appropriées.
