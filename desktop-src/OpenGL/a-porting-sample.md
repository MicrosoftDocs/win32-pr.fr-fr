# <a name="a-porting-sample"></a>Exemple de Portage

Il est plus facile de comprendre comment modifier votre programme X Window System OpenGL pour Windows si vous pouvez comparer des exemples avant et après, et vous pouvez mieux voir comment le code traduit est utilisé dans le contexte approprié. Cette section présente un exemple de programme OpenGL de système X Window, puis montre comment le programme se présente après avoir été porté vers Windows. Notez que le code OpenGL est le même dans les deux programmes.

-   [Un programme X Windows système OpenGL](an-x-window-system-opengl-program.md)
-   [Le programme est porté sur Windows](the-program-ported-to-win32.md)
