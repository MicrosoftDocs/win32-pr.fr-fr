# <a name="introduction-to-opengl"></a>Présentation de OpenGL

En tant qu’interface logicielle pour le matériel graphique, l’objectif principal de OpenGL est de restituer des objets en deux et trois dimensions dans un trame. Ces objets sont décrits comme des séquences de vertex (qui définissent des objets géométriques) ou des pixels (qui définissent des images). OpenGL effectue plusieurs processus sur ces données afin de les convertir en pixels pour former l’image finale souhaitée dans le trame.

Les rubriques suivantes présentent une vue globale du fonctionnement d’OpenGL :

-   Les [primitives et les commandes](primitives-and-commands.md) décrivent les points, les segments de ligne et les polygones comme unités de base de dessin. et le traitement des commandes.
-   [Le contrôle graphique OpenGL](opengl-graphic-control.md) décrit les opérations graphiques qui contrôlent OpenGL et ce qu’il ne contrôle pas.
-   Le [modèle d’exécution](execution-model.md) présente le modèle client/serveur pour interpréter les commandes OpenGL.
-   L' [opération OpenGL de base](basic-opengl-operation.md) fournit une description de haut niveau de la façon dont OpenGL traite les données pour produire une image correspondante dans le trame.
-   Les [noms de fonctions OpenGL](opengl-function-names.md) décrivent les conventions d’affectation de noms utilisées dans OpenGL.
