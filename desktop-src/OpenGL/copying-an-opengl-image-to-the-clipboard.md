# <a name="copying-an-opengl-image-to-the-clipboard"></a>Copie d’une image OpenGL dans le presse-papiers

Bien que la version actuelle de l’implémentation Microsoft de OpenGL dans Windows ne prenne pas directement en charge le presse-papiers, vous pouvez copier une image Windows OpenGL dans le presse-papiers.

**Pour copier une image OpenGL dans le presse-papiers**

1.  Dessinez l’image sur une image mémoire ou un métafichier amélioré.
2.  Copiez cette image bitmap dans le presse-papiers.
