# <a name="r-opengl"></a>R (OpenGL)

<dl> <dt>

<span id="opengl_rasterize"></span><span id="OPENGL_RASTERIZE"></span>**rastérisation**
</dt> <dd>

Pour convertir un point, une ligne ou un polygone projeté, ou les pixels d’une bitmap ou d’une image, en fragments, chacun correspondant à un pixel dans le trame. Notez que toutes les primitives sont pixellisées, pas seulement des points, des lignes et des polygones

</dd> <dt>

<span id="opengl_rectangle"></span><span id="OPENGL_RECTANGLE"></span>**Rectangle**
</dt> <dd>

Quadrilatère dont les bords secondaires sont parallèles les uns aux autres dans les coordonnées d’objet. Les polygones spécifiés avec glRect \* () sont toujours des rectangles ; d’autres quadrilatères peuvent être des rectangles.

</dd> <dt>

<span id="opengl_rendering"></span><span id="OPENGL_RENDERING"></span>**rendu**
</dt> <dd>

Conversion de primitives spécifiées en coordonnées d’objets en image dans le trame. Le rendu est l’opération principale de OpenGL.

</dd> <dt>

<span id="opengl_rgba"></span><span id="OPENGL_RGBA"></span>**RGBA**
</dt> <dd>

Composants de couleur rouge, vert, bleu et alpha du mode RVBA.

</dd> <dt>

<span id="opengl_rgba_mode"></span><span id="OPENGL_RGBA_MODE"></span>**Mode RVBA**
</dt> <dd>

Contexte OpenGL dans lequel ses mémoires tampons de couleurs stockent les composants de couleur rouge, vert, bleu et alpha, plutôt que les index de couleurs.

</dd> </dl>
