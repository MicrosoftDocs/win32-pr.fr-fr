# <a name="u-v"></a>U V

<dl> <dt>

<span id="opengl_vertex"></span><span id="OPENGL_VERTEX"></span>**vertex**
</dt> <dd>

Point dans un espace tridimensionnel.

</dd> <dt>

<span id="opengl_vertices"></span><span id="OPENGL_VERTICES"></span>**derniers**
</dt> <dd>

Pluriel de vertex préféré. Voir *vertex*.

</dd> <dt>

<span id="opengl_viewpoint"></span><span id="OPENGL_VIEWPOINT"></span>**point**
</dt> <dd>

Origine du système de coordonnées oculaires ou de la découpage, selon le contexte. Par exemple, lorsque vous discutez de l’éclairage, le point de vue est l’origine du système de coordonnées oculaires. Lorsque vous discutez de la projection, le point de vue est l’origine du système de coordonnées. Avec une matrice de projection classique, les origines de la coordonnée oculaire et de la coordonnée sont au même emplacement.

</dd> <dt>

<span id="opengl_view_volume"></span><span id="OPENGL_VIEW_VOLUME"></span>**afficher le volume**
</dt> <dd>

Le volume en coordonnées de clip dont les coordonnées répondent aux trois conditions

-   w ≤ x ≤ w
-   w ≤ y ≤ w
-   w ≤ z ≤ w

</dd> </dl>
