# <a name="wic-samples"></a>Exemples WIC

cette section contient les exemples du composant WIC (Windows Imaging Component).

WIC fournit deux types d’exemples :

-   Exemples : segments de code qui utilisent plusieurs fonctionnalités WIC. le code Source de l’exemple est fourni dans la SDK Windows.
-   Exemples de code : exemples plus courts qui illustrent une fonctionnalité spécifique. Le code source de l’exemple n’est pas fourni avec le kit de développement logiciel (SDK).
