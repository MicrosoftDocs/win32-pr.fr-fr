# <a name="wic-structures"></a>Structures WIC

cette section contient des informations sur les structures WIC (Windows Imaging Component).

## <a name="in-this-section"></a>Contenu de cette section



| Rubrique                                                                          | Description                                                                                                                   |
|--------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------|
| [**WICBitmapPattern**](/windows/desktop/api/Wincodec/ns-wincodec-wicbitmappattern)<br/>             | Contient des membres qui identifient un modèle dans un fichier image qui peut être utilisé pour identifier un format particulier.<br/>   |
| [**WICBitmapPlane**](/windows/desktop/api/Wincodec/ns-wincodec-wicbitmapplane)<br/>                            | Spécifie le format de pixel, la mémoire tampon, la Stride et la taille d’un plan de composant pour un format de pixel planaire.<br/>                |
| [**WICBitmapPlaneDescription**](/windows/desktop/api/Wincodec/ns-wincodec-wicbitmapplanedescription)<br/>      | Spécifie le format de pixel et la taille d’un plan de composant.<br/>                                                          |
| [**WICDdsFormatInfo**](/windows/desktop/api/Wincodec/ns-wincodec-wicddsformatinfo)<br/>                        | Spécifie [**le \_ format dxgi**](/windows/desktop/api/dxgiformat/ne-dxgiformat-dxgi_format) et les informations de bloc d’un format DDS.<br/>                  |
| [**WICDdsParameters**](/windows/desktop/api/Wincodec/ns-wincodec-wicddsparameters)<br/>                        | Spécifie la dimension d’image DDS, le [**\_ format dxgi**](/windows/desktop/api/dxgiformat/ne-dxgiformat-dxgi_format) et le mode Alpha des données contenues.<br/>  |
| [**WICImageParameters**](/windows/desktop/api/Wincodec/ns-wincodec-wicimageparameters)<br/>                    | Définit les paramètres que vous pouvez utiliser pour substituer les paramètres par défaut utilisés normalement lors de l’encodage d’une image. <br/> |
| [**WICJpegFrameHeader**](/windows/desktop/api/wincodec/ns-wincodec-wicjpegframeheader)<br/>                    | Représente un en-tête de frame JPEG.<br/>                                                                                    |
| [**WICJpegScanHeader**](/windows/desktop/api/wincodec/ns-wincodec-wicjpegscanheader)<br/>                      | Représente un en-tête de frame JPEG.<br/>                                                                                    |
| [**WICMetadataHeader**](/windows/desktop/api/Wincodecsdk/ns-wincodecsdk-wicmetadataheader)<br/>           | Représente l’en-tête de métadonnées.<br/>                                                                                        |
| [**WICMetadataPattern**](/windows/desktop/api/Wincodecsdk/ns-wincodecsdk-wicmetadatapattern)<br/>         | Représente un modèle de métadonnées.<br/>                                                                                     |
| [**WICRawCapabilitiesInfo**](/windows/desktop/api/Wincodec/ns-wincodec-wicrawcapabilitiesinfo)<br/> | Définit les fonctionnalités brutes du codec.<br/>                                                                                     |
| [**WICRawToneCurve**](/windows/desktop/api/Wincodec/ns-wincodec-wicrawtonecurve)<br/>               | Représente une courbe de tonalité d’image brute.<br/>                                                                                 |
| [**WICRawToneCurvePoint**](/windows/desktop/api/Wincodec/ns-wincodec-wicrawtonecurvepoint)<br/>     | Représente un point de courbe de tonalité d’image brute.<br/>                                                                           |
| [**WICRect**](/windows/desktop/api/Wincodec/ns-wincodec-wicrect)<br/>                               | Représente un rectangle pour l’API WIC.<br/>                                                                                |
