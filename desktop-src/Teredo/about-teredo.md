# <a name="about-teredo"></a>À propos de Teredo

En raison du nombre limité d’adresses IPv4 disponibles, les fournisseurs de services Internet devront commencer la transition vers IPv6. Windows Vista offre une prise en charge complète d’IPv6, ce qui permet aux développeurs de s’assurer que leurs applications sont prêtes pour demain en activant la prise en charge D’ipv6 dès aujourd’hui. avec Teredo, les postes de travail Windows Vista sont compatibles avec la connectivité IPv6 même si les réseaux ISP sont uniquement IPv4.

Les technologies telles que l’homologation d’égal à égal requièrent une connectivité de bout en bout pour l’expérience utilisateur complète. De nombreuses maisons possèdent des appareils de traduction d’adresses réseau (NAT) qui limitent ces technologies et les développeurs d’applications doivent se concentrer sur la recherche de moyens efficaces pour contourner ces problèmes. toutefois, la prise en charge de Teredo avec Windows Vista réduit la charge de travail des développeurs d’applications en fournissant une traversée NAT transparente.
