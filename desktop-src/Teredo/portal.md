# <a name="teredo"></a>Teredo

## <a name="purpose"></a>Objectif

Teredo est une technologie de transition IPv6 qui permet l’attribution d’adresses et le tunneling automatique d’hôte à hôte pour le trafic IPv6 monodiffusion lorsque les hôtes IPv6/IPv4 se trouvent derrière un ou plusieurs traducteurs d’adresses réseau (NAT) IPv4. Pour traverser les NAT IPv4, les paquets IPv6 sont envoyés en tant que messages UDP IPv4 (User Datagram Protocol).

## <a name="developer-audience"></a>Développeurs concernés

Teredo est conçu pour être utilisé par les développeurs C/C++ avec l’expérience de programmation réseau IPv6.

## <a name="run-time-requirements"></a>Conditions d’exécution

L’interface Teredo est principalement prise en charge par Windows Vista et Windows Server 2008. La fonctionnalité limitée de l’interface Teredo prise en charge par Windows XP avec Service Pack 2 (SP2) et Windows Server 2003 est détaillée dans [réception du trafic sollicité sur Teredo](receiving-solicited-traffic-over-teredo.md).
