# <a name="the-teredo-security-model"></a>Modèle de sécurité Teredo

la technologie [Teredo](about-teredo.md) , couplée avec Sécurité Windows Center et [Windows pare-feu](/previous-versions/windows/desktop/ics/windows-firewall-start-page), offre une sécurité renforcée à l’utilisateur final. pour Windows pare-feu, le trafic ipv6 envoyé et reçu via l’interface Teredo est traité de la même façon que le trafic ipv6 natif. L’interface Teredo est reconnue comme un autre mini-port dans le système.

Pour plus d’informations sur l’utilisation, consultez [implémentation du modèle de sécurité Teredo](implementing-the-teredo-security-model.md).
