# <a name="the-account-object-type"></a>Type d’objet de compte

Les objets de [**compte**](account-object.md) sont utilisés pour signifier [*des privilèges*](/windows/desktop/SecGloss/p-gly)ou des droits d’accès au système qui doivent être accordés à un utilisateur ou à des membres de groupes ou de groupes locaux.

Les objets de [**compte**](account-object.md) résident dans l’objet de [**stratégie**](policy-object.md) . Il peut y avoir zéro ou plusieurs de ces objets.

L’objet [**Account**](account-object.md) n’a pas de types de données spécifiques à l’objet.
