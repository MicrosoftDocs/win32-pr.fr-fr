# <a name="using-service-security-attachments"></a>Utilisation des pièces jointes de sécurité de service

Cette section décrit comment créer les composants d’attachement requis pour prendre en charge une configuration spécifique au service dans les composants logiciels enfichables de configuration de la sécurité. Vous devrez implémenter et inscrire deux composants : un composant logiciel enfichable d’extension permettant d’afficher une interface utilisateur et un moteur d’attachement pour traiter les demandes de configuration et d’analyse.

Pour plus d'informations, consultez les rubriques suivantes :

-   [Création d’une extension de composant logiciel enfichable pièce jointe](creating-an-attachment-snap-in-extension.md)
-   [Inscription d’une extension de composant logiciel enfichable pièce jointe](registering-an-attachment-snap-in-extension.md)
-   [Création d’un moteur de pièce jointe](creating-an-attachment-engine.md)
-   [Inscription d’un moteur de pièce jointe](registering-an-attachment-engine.md)
