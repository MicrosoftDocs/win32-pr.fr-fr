# <a name="the-policy-object-type"></a>Type d’objet de stratégie

L’objet de [**stratégie**](policy-object.md) est un objet conteneur. Les types d’objets suivants peuvent résider dans ce conteneur :

-   Objets de [**compte**](account-object.md)
-   Objet [**trustedDomain**](trusteddomain-object.md)
-   Objet secret

Il n’existe qu’une seule instance de l’objet de [**stratégie**](policy-object.md) . Il ne peut être ni créé ni détruit.
