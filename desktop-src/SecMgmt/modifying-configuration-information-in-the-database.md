# <a name="modifying-configuration-information-in-the-database"></a>Modification des informations de configuration dans la base de données

L’extension du composant logiciel enfichable pièce jointe doit permettre aux utilisateurs de modifier les informations de configuration via son nœud analyse. Cela permet aux utilisateurs d’apporter des modifications, puis de réappliquer la configuration dans le temps. La logique de cette opération doit être identique à la modification des informations dans les fichiers de configuration. Les modifications apportées prendront effet sur la configuration enregistrée dans la base de données de sécurité.
