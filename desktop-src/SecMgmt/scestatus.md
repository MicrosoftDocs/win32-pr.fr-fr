# <a name="scestatus"></a>SCESTATUS

Le type de données **SCESTATUS** est utilisé par l’API Set de l’outil de configuration de la sécurité pour retourner des informations sur les résultats d’un appel de fonction. Pour plus d’informations sur les valeurs de code d’état définies, consultez [Attachment Return values](management-return-values.md).


```C++
typedef DWORD SCESTATUS;
```
