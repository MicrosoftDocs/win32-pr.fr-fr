# <a name="lsa-policy"></a>Stratégie LSA

L' [*autorité de sécurité locale (LSA, Local Security Authority*](/windows/desktop/SecGloss/l-gly) ) est un sous-système protégé de Windows qui gère les informations relatives à tous les aspects de la sécurité locale sur un système, collectivement appelée [stratégie de sécurité locale](local-security-policy.md) du système. Outre le logement des informations de stratégie, l’autorité LSA fournit des services pour la traduction entre les noms et les [*identificateurs de sécurité*](/windows/desktop/SecGloss/s-gly) (SID).
