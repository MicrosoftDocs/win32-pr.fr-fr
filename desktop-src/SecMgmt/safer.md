# <a name="safer"></a>Plus sûr

Les fonctions de l’API plus sécurisée fournissent à toute application qui démarre des programmes à partir de sources externes la possibilité de demander l’approbation d’une stratégie de sécurité avant le démarrage d’un exécutable. Ces fonctions peuvent être appelées avant le chargement et l’exécution d’un fichier exécutable ou d’un contenu actif. Les applications qui gèrent les pièces jointes (par exemple, les clients de messagerie et les messageries instantanées pouvant transférer des fichiers) et les interpréteurs de script sont des types d’applications où l’API plus sécurisée est utile. la prise en charge des fonctions dans l’API plus sécurisée commence par Windows XP. Pour plus d’informations, consultez [fonctions plus sûres](management-functions.md).
