# <a name="security-configuration-and-analysis"></a>Configuration et analyse de la sécurité

La configuration et l’analyse de la sécurité est un outil de composant logiciel enfichable autonome que les utilisateurs peuvent utiliser pour importer une ou plusieurs configurations enregistrées dans une base de données de sécurité privée. L’importation de configurations génère une base de données de sécurité spécifique à l’ordinateur qui stocke une configuration composite. Vous pouvez appliquer cette configuration composite à l’ordinateur et analyser la configuration système actuelle par rapport à la configuration composite stockée stockée dans la base de données.

Les [modèles de sécurité](security-templates.md) et la configuration et l’analyse de la sécurité composent les composants logiciels enfichables de configuration de la sécurité.
