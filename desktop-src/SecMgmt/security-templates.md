# <a name="security-templates"></a>Modèles de sécurité

Les modèles de sécurité sont un outil de composant logiciel enfichable autonome que les utilisateurs peuvent utiliser pour définir des configurations de sécurité indépendantes des ordinateurs. Ces configurations sont enregistrées sous forme de fichiers. inf textuels.

Les modèles de sécurité et la [configuration et l’analyse](security-configuration-and-analysis.md) de la sécurité composent les composants logiciels enfichables de configuration de la sécurité.
