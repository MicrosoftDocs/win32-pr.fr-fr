# <a name="xmuint3-extensions"></a>Extensions XMUINT3

[**XMUINT3**](/windows/win32/api/directxmath/ns-directxmath-xmuint3) décrit une structure contenant un vecteur 3D constitué de trois valeurs entières non signées.

> [!Note]  
> Les fonctionnalités répertoriées ici sont disponibles uniquement lors du développement avec C++.
