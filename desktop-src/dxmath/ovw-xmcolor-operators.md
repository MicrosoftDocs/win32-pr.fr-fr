# <a name="xmcolor-operators"></a>XMCOLOR, opérateurs

La structure [**XMCOLOR**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmcolor) expose les opérateurs suivants.

> [!Note]  
> Les opérateurs répertoriés ici sont uniquement disponibles en C++.

 

## <a name="in-this-section"></a>Dans cette section



| Méthodes                                                            | Description                                                                                                           |
|--------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| [**UInt32 t (opérateur) \_**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmcolor-operatoruint32_t)<br/> | Retourne une instance de `uint32_t` contenant les composants de l' `XMCOLOR` instance dans un format compressé. <br/> |
| [**opérateur =**](xmcolor-operator-eq.md)<br/>               | Assigne des données de composant vectoriels à l’instance actuelle de `XMCOLOR` .<br/>                                        |
