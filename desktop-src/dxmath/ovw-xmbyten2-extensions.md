# <a name="xmbyten2-extensions"></a>Extensions XMBYTEN2

[**XMBYTEN2**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmbyten2) décrit un vecteur 2D pour le stockage des valeurs signées et normalisées en tant qu’entiers 8 bits signés (1 octet).

> [!Note]  
> Les fonctionnalités répertoriées ici sont disponibles uniquement lors du développement avec C++.
