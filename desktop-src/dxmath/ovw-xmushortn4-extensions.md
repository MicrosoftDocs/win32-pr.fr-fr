# <a name="xmushortn4-extensions"></a>Extensions XMUSHORTN4

[**XMUSHORTN4**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmushortn4) décrit un vecteur 4D pour stocker des valeurs non signées et normalisées en tant qu’entiers 16 bits signés, (type `uint16_t` ).

> [!Note]  
> Les fonctionnalités répertoriées ici sont disponibles uniquement lors du développement avec C++.
