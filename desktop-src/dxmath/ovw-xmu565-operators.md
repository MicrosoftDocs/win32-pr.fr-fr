# <a name="xmu565-operators"></a>XMU565, opérateurs

La structure [**XMU565**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmu565) expose les opérateurs suivants.

> [!Note]  
> Les opérateurs répertoriés ici sont uniquement disponibles en C++.

 

## <a name="in-this-section"></a>Dans cette section



| Méthodes                                                           | Description                                                                                                          |
|-------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| [**UInt16, opérateur \_ t**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmu565-operatoruint16_t)<br/> | Retourne une instance de `uint16_t` contenant les composants de l' `XMU555` instance dans un format compressé. <br/> |
| [**opérateur =**](xmu565-operator-eq.md)<br/>               | Assigne des données de composant vectoriels à l’instance actuelle de `XMU565` . <br/>                                       |
