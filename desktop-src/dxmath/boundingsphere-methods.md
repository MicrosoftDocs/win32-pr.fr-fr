# <a name="boundingsphere-methods"></a>Méthodes BoundingSphere



| Méthode                                                                           | Description                                                                                                  |
|----------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------|
| [**Comprend**](boundingsphere-contains.md)<br/>                           | Teste si le BoundingSphere contient un objet spécifié.<br/>                                     |
| [**Croise**](boundingsphere-intersects.md)<br/>                       | Teste le BoundingSphere pour l’intersection avec un objet.<br/>                                         |
| [**Transform**](boundingsphere-transform.md)<br/>                         | Transforme le BoundingSphere.<br/>                                                                    |
| [**ContainedBy**](/windows/desktop/api/DirectXCollision/nf-directxcollision-boundingbox-containedby)<br/>                     | Teste si le [**BoundingSphere**](/windows/win32/api/directxcollision/ns-directxcollision-boundingsphere) est contenu dans le frustum spécifié.<br/> |
| [**CreateFromBoundingBox**](/windows/win32/api/directxcollision/nf-directxcollision-boundingsphere-createfromboundingbox(boundingsphere__constboundingorientedbox_))<br/> | Crée un BoundingSphere contenant le BoundingBox spécifié.<br/>                                    |
| [**CreateFromPoints**](/windows/win32/api/directxcollision/nf-directxcollision-boundingsphere-createfrompoints)<br/>           | Crée un BoundingSphere à partir d’une liste de points.<br/>                                               |
| [**CreateMerged**](/windows/win32/api/directxcollision/nf-directxcollision-boundingsphere-createmerged)<br/>                   | Crée un BoundingSphere qui contient les deux objets BoundingSphere spécifiés.<br/>                  |
| [**affectation d’opérations \_**](/windows/win32/api/directxcollision/nf-directxcollision-boundingsphere-operator-assign(boundingsphere__))<br/>                | Initialise le BoundingSphere avec les valeurs d’un BoundingSphere spécifié.<br/>                       |
