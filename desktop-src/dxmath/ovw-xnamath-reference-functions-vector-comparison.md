# <a name="vector-comparison-functions"></a>Fonctions de comparaison de vecteurs

Répertorie les fonctions de comparaison de vecteurs.

## <a name="in-this-section"></a>Contenu de cette section



| Rubrique                                                                 | Description                                                                                                                                                                                                                                                               |
|-----------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XMVectorEqual**](/windows/win32/api/directxmath/nf-directxmath-xmvectorequal)<br/>                     | Effectue un test par composant pour l’égalité de deux vecteurs.<br/>                                                                                                                                                                                                     |
| [**XMVectorEqualInt**](/windows/win32/api/directxmath/nf-directxmath-xmvectorequalint)<br/>               | Effectue un test par composant pour l’égalité de deux vecteurs, en traitant chaque composant en tant qu’entier non signé.<br/>                                                                                                                                                 |
| [**XMVectorEqualIntR**](/windows/win32/api/directxmath/nf-directxmath-xmvectorequalintr)<br/>             | Effectue un test par composant pour l’égalité de deux vecteurs, en traitant chaque composant en tant qu’entier non signé. En outre, cette fonction définit une valeur de comparaison qui peut être examinée à l’aide de fonctions telles que [**XMComparisonAllTrue**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue).<br/> |
| [**XMVectorEqualR**](/windows/win32/api/directxmath/nf-directxmath-xmvectorequalr)<br/>                   | Effectue un test par composant pour l’égalité de deux vecteurs et définit une valeur de comparaison qui peut être examinée à l’aide de fonctions telles que [**XMComparisonAllTrue**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue).<br/>                                                                         |
| [**XMVectorGreater**](/windows/win32/api/directxmath/nf-directxmath-xmvectorgreater)<br/>                 | Effectue un test par composant pour plus de deux vecteurs.<br/>                                                                                                                                                                                            |
| [**XMVectorGreaterOrEqual**](/windows/win32/api/directxmath/nf-directxmath-xmvectorgreaterorequal)<br/>   | Effectue un test par composant pour une valeur supérieure ou égale à deux vecteurs.<br/>                                                                                                                                                                                   |
| [**XMVectorGreaterOrEqualR**](/windows/win32/api/directxmath/nf-directxmath-xmvectorgreaterorequalr)<br/> | Effectue un test par composant pour une valeur supérieure ou égale à deux vecteurs et définit une valeur de comparaison qui peut être examinée à l’aide de fonctions telles que [**XMComparisonAllTrue**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue).<br/>                                                       |
| [**XMVectorGreaterR**](/windows/win32/api/directxmath/nf-directxmath-xmvectorgreaterr)<br/>               | Effectue un test par composant pour plus de deux vecteurs et définit une valeur de comparaison qui peut être examinée à l’aide de fonctions telles que [**XMComparisonAllTrue**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue).<br/>                                                                |
| [**XMVectorLess**](/windows/win32/api/directxmath/nf-directxmath-xmvectorless)<br/>                       | Effectue un test par composant pour moins de deux vecteurs.<br/>                                                                                                                                                                                               |
| [**XMVectorLessOrEqual**](/windows/win32/api/directxmath/nf-directxmath-xmvectorlessorequal)<br/>         | Effectue un test par composant pour un nombre inférieur ou égal à deux vecteurs.<br/>                                                                                                                                                                                      |
| [**XMVectorNearEqual**](/windows/win32/api/directxmath/nf-directxmath-xmvectornearequal)<br/>             | Effectue un test par composant pour l’égalité de deux vecteurs dans un seuil donné. <br/>                                                                                                                                                                           |
