# <a name="xmfloat3pk-operators"></a>XMFLOAT3PK, opérateurs

La structure [**XMFLOAT3PK**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmfloat3pk) expose les opérateurs suivants.

> [!Note]  
> Les opérateurs répertoriés ici sont uniquement disponibles en C++.

 

## <a name="in-this-section"></a>Dans cette section



| Méthodes                                                               | Description                                                                                                             |
|-----------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------|
| [**UInt32 t (opérateur) \_**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmfloat3pk-operatoruint32_t)<br/> | Retourne une instance de `uint32_t` contenant les composants de l' `XMFLOAT3PK` instance dans un format compressé.<br/> |
| [**opérateur =**](xmfloat3pk-operator-eq.md)<br/>               | Assigne des données de composant vectoriels à l’instance actuelle de `XMFLOAT3PK` .<br/>                                       |
