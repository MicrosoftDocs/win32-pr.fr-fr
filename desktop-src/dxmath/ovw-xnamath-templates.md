# <a name="directxmath-library-template-functions"></a>Fonctions de modèle de bibliothèque DirectXMath

Répertorie les fonctions de modèle fournies par DirectXMath.

## <a name="in-this-section"></a>Contenu de cette section



| Rubrique                                                                  | Description                                                                                                                                                                                                                               |
|------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XMMax**](xmmax-template.md)<br/>                             | Compare deux instances de type de données numériques, ou deux instances d’un objet qui prend en charge une surcharge de <, et retourne la plus grande des deux instances. Le type de données des arguments et la valeur de retour sont les mêmes.<br/>  |
| [**XMMin**](xmmin-template.md)<br/>                             | Compare deux instances de type de données numériques, ou deux instances d’un objet qui prend en charge une surcharge de <, et retourne la plus petite des deux instances. Le type de données des arguments et la valeur de retour sont les mêmes.<br/> |
| [**XMVectorPermute**](xmvectorpermute-template.md)<br/>         | Permute les composants de deux vecteurs pour créer un nouveau vecteur.<br/>                                                                                                                                                                 |
| [**XMVectorRotateLeft**](xmvectorrotateleft-template.md)<br/>   | Fait pivoter le vecteur vers la gauche d’un nombre donné d’éléments de 32 bits.<br/>                                                                                                                                                                  |
| [**XMVectorRotateRight**](xmvectorrotateright-template.md)<br/> | Fait pivoter le vecteur vers la droite d’un nombre donné d’éléments de 32 bits.<br/>                                                                                                                                                                 |
| [**XMVectorShiftLeft**](xmvectorshiftleft-template.md)<br/>     | Déplace un vecteur vers la gauche d’un nombre donné d’éléments 32 bits, en remplissant les éléments libérés avec les éléments d’un second vecteur.<br/>                                                                                                    |
| [**XMVectorSwizzle**](xmvectorswizzle-template.md)<br/>         | Swizzles un vecteur.<br/>                                                                                                                                                                                                             |
| [**XMVectorInsert**](xmvectorinsert-template.md)<br/>           | Fait pivoter un vecteur vers la gauche d’un nombre donné de composants 32 bits et insérer les éléments sélectionnés de ce résultat dans un autre vecteur.<br/>                                                                                                  |
