# <a name="xmudec4-operators"></a>XMUDEC4, opérateurs

La structure [**XMUDEC4**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmudec4) expose les opérateurs suivants.

> [!Note]  
> Les opérateurs répertoriés ici sont uniquement disponibles en C++.

 

## <a name="in-this-section"></a>Dans cette section



| Méthodes                                                            | Description                                                                                                           |
|--------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| [**UInt32 t (opérateur) \_**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmudec4-operatoruint32_t)<br/> | Retourne une instance de `uint32_t` contenant les composants de l' `XMUDEC4` instance dans un format compressé. <br/> |
| [**opérateur =**](xmudec4-operator-eq.md)<br/>               | Assigne des données de composant vectoriels à l’instance actuelle de `XMUDEC4` . <br/>                                       |
