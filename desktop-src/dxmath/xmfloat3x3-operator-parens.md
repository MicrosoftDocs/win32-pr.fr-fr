# <a name="xmfloat3x3-operator--operators"></a>Opérateurs XMFLOAT3X3 Operator ()

Accède à des éléments de matrice spécifiques référencés par ligne et colonne à partir de l’instance actuelle de [ **XMFLOAT3X3**](/windows/win32/api/directxmath/ns-directxmath-xmfloat3x3)
