# <a name="xmfloat3se-extensions"></a>Extensions XMFLOAT3SE

Vecteur 3D de trois composants à virgule flottante avec 9 bits mantisses, chacun partageant le même exposant 5 bits.

> [!Note]  
> Les fonctionnalités répertoriées ici sont disponibles uniquement lors du développement avec C++.
