# <a name="xmubyte2-operators"></a>XMUBYTE2, opérateurs

> [!Note]  
> Les opérateurs répertoriés ici sont uniquement disponibles avec C++.

 

## <a name="in-this-section"></a>Dans cette section



| Méthodes                                               | Description                                                                                                         |
|-------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------|
| [**opérateur =**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmubyte2-operator-assign)<br/> | Assigne les données du composant vectoriel d’une instance de `XMUBYTE2` à l’instance actuelle de `XMUBYTE2` .<br/> |
