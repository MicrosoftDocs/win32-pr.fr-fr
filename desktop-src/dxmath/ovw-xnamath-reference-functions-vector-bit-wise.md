# <a name="bit-wise-vector-functions"></a>Fonctions de vecteur Bit-Wise

Répertorie les fonctions de vecteurs au niveau du bit.

## <a name="in-this-section"></a>Contenu de cette section



| Rubrique                                                         | Description                                                                                                                             |
|---------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------|
| [**XMVectorAndCInt**](/windows/win32/api/directxmath/nf-directxmath-xmvectorandcint)<br/>         | Calcule le et logique d’un vecteur avec la négation d’un second vecteur, en traitant chaque composant comme un entier non signé.<br/> |
| [**XMVectorAndInt**](/windows/win32/api/directxmath/nf-directxmath-xmvectorandint)<br/>           | Calcule le AND logique de deux vecteurs, en traitant chaque composant en tant qu’entier non signé.<br/>                                     |
| [**XMVectorNorInt**](/windows/win32/api/directxmath/nf-directxmath-xmvectornorint)<br/>           | Calcule le or logique de deux vecteurs, en traitant chaque composant en tant qu’entier non signé.<br/>                                     |
| [**XMVectorNotEqual**](/windows/win32/api/directxmath/nf-directxmath-xmvectornotequal)<br/>       | Effectue un test par composant pour l’inégalité de deux vecteurs.<br/>                                                             |
| [**XMVectorNotEqualInt**](/windows/win32/api/directxmath/nf-directxmath-xmvectornotequalint)<br/> | Effectue un test par composant pour l’inégalité de deux vecteurs, en traitant chaque composant en tant qu’entier non signé.<br/>             |
| [**XMVectorOrInt**](/windows/win32/api/directxmath/nf-directxmath-xmvectororint)<br/>             | Calcule le OR logique de deux vecteurs, en traitant chaque composant en tant qu’entier non signé.<br/>                                      |
| [**XMVectorXorInt**](/windows/win32/api/directxmath/nf-directxmath-xmvectorxorint)<br/>           | Calcule le XOR logique de deux vecteurs, en traitant chaque composant en tant qu’entier non signé.<br/>                                     |
