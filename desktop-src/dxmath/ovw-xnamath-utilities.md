# <a name="directxmath-library-utility-functions"></a>Fonctions de l’utilitaire de bibliothèque DirectXMath

Répertorie les fonctions utilitaires fournies par DirectXMath.

## <a name="in-this-section"></a>Contenu de cette section



| Rubrique                                                                       | Description                                                                                                                |
|-----------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------|
| [**XMComparisonAllFalse**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonallfalse)<br/>             | Teste la valeur de comparaison pour déterminer si tous les composants comparés ont la valeur false.<br/>                            |
| [**XMComparisonAllInBounds**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonallinbounds)<br/>       | Teste la valeur de comparaison pour déterminer si tous les composants comparés se trouvent dans des limites définies.<br/>                |
| [**XMComparisonAllTrue**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue)<br/>               | Teste la valeur de comparaison pour déterminer si tous les composants comparés ont la valeur true.<br/>                             |
| [**XMComparisonAnyFalse**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonanyfalse)<br/>             | Teste la valeur de comparaison pour déterminer si l’un des composants comparés est false.<br/>                            |
| [**XMComparisonAnyOutOfBounds**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonanyoutofbounds)<br/> | Teste la valeur de comparaison pour déterminer si l’un des composants comparés se trouve en dehors des limites définies.<br/>           |
| [**XMComparisonAnyTrue**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonanytrue)<br/>               | Teste la valeur de comparaison pour déterminer si l’un des composants comparés est true.<br/>                             |
| [**XMComparisonMixed**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonmixed)<br/>                   | Teste la valeur de comparaison pour déterminer si les composants comparés comportaient des résultats mixtes--some true et Some false.<br/> |
| [**XMFresnelTerm**](/windows/win32/api/directxmath/nf-directxmath-xmfresnelterm)<br/>                           | Calcule le terme de la Fresnel pour la lumière dépolarisée.<br/>                                                              |
| [**XMVerifyCPUSupport**](/windows/win32/api/directxmath/nf-directxmath-xmverifycpusupport)<br/>                 | Indique si la bibliothèque DirectXMath prend en charge la plateforme actuelle.<br/>                                             |
