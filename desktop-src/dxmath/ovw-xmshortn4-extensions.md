# <a name="xmshortn4-extensions"></a>Extensions XMSHORTN4

[**XMSHORTN4**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmshortn4) décrit le vecteur 4D pour le stockage des valeurs signées et normalisées en tant qu’entiers 16 bits signés, (type `int16_t` ).

> [!Note]  
> Les fonctionnalités répertoriées ici sont disponibles uniquement lors du développement avec C++.
