# <a name="boundingfrustum-methods"></a>Méthodes BoundingFrustum



| Méthode                                                                  | Description                                                                                                    |
|-------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------|
| [**Comprend**](boundingfrustum-contains.md)<br/>                 | Teste si le [**BoundingFrustum**](/windows/win32/api/directxcollision/ns-directxcollision-boundingfrustum) contient un objet spécifié.<br/>           |
| [**Croise**](boundingfrustum-intersects.md)<br/>             | Teste le [**BoundingFrustum**](/windows/win32/api/directxcollision/ns-directxcollision-boundingfrustum) pour l’intersection avec un autre objet.<br/>          |
| [**Transform**](boundingfrustum-transform.md)<br/>               | Transforme le [**BoundingFrustum**](/windows/win32/api/directxcollision/ns-directxcollision-boundingfrustum).<br/>                                          |
| [**ContainedBy**](/windows/desktop/api/DirectXCollision/nf-directxcollision-boundingbox-containedby)<br/>           | Teste si le [**BoundingFrustum**](/windows/win32/api/directxcollision/ns-directxcollision-boundingfrustum) est contenu dans le frustum spécifié.<br/> |
| [**CreateFromMatrix**](/windows/win32/api/directxcollision/nf-directxcollision-boundingfrustum-createfrommatrix)<br/> | Crée un [**BoundingFrustum**](/windows/win32/api/directxcollision/ns-directxcollision-boundingfrustum) à partir de la matrice de projection spécifiée.<br/>          |
| [**GetCorners**](/windows/win32/api/directxcollision/nf-directxcollision-boundingfrustum-getcorners)<br/>             | Obtient les coins qui composent le [**BoundingFrustum**](/windows/win32/api/directxcollision/ns-directxcollision-boundingfrustum).<br/>                          |
| [**GetPlanes**](/windows/win32/api/directxcollision/nf-directxcollision-boundingfrustum-getplanes)<br/>               | Obtient les plans qui composent le [**BoundingFrustum**](/windows/win32/api/directxcollision/ns-directxcollision-boundingfrustum).<br/>                           |
| [**affectation d’opérations \_**](/windows/win32/api/directxcollision/nf-directxcollision-boundingfrustum-operator-assign(boundingfrustum__))<br/>      | Copie les valeurs d’un autre [**BoundingFrustum**](/windows/win32/api/directxcollision/ns-directxcollision-boundingfrustum).<br/>                              |
