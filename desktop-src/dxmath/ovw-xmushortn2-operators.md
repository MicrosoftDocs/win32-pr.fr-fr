# <a name="xmushortn2-operators"></a>XMUSHORTN2, opérateurs

> [!Note]  
> Les opérateurs répertoriés ici sont uniquement disponibles en C++.

 

## <a name="in-this-section"></a>Dans cette section



| Méthodes                                                 | Description                                                                                                             |
|---------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------|
| [**opérateur =**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmushortn2-operator-assign)<br/> | Assigne les données du composant vectoriel d’une instance de `XMUSHORTN2` à l’instance actuelle de `XMUSHORTN2` .<br/> |
