# <a name="xmubyten2-operators"></a>XMUBYTEN2, opérateurs

> [!Note]  
> Les opérateurs répertoriés ici sont uniquement disponibles avec C++.

 

## <a name="in-this-section"></a>Dans cette section



| Méthodes                                                | Description                                                                                                           |
|--------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| [**opérateur =**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmubyten2-operator-assign)<br/> | Assigne les données du composant vectoriel d’une instance de `XMUBYTEN2` à l’instance actuelle de `XMUBYTEN2` .<br/> |
