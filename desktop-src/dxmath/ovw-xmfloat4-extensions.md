# <a name="xmfloat4-extensions"></a>Extensions XMFLOAT4

[**XMFLOAT4**](/windows/win32/api/directxmath/ns-directxmath-xmfloat4) décrit une structure contenant un vecteur 4D constitué de quatre valeurs à virgule flottante simple précision.

> [!Note]  
> Les fonctionnalités répertoriées ici sont disponibles uniquement lors du développement avec C++.
