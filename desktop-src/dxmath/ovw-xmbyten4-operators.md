# <a name="xmbyten4-operators"></a>XMBYTEN4, opérateurs

> [!Note]  
> Les opérateurs répertoriés ici sont uniquement disponibles en C++.

|-------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| [**opérateur =**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmbyten4-operator-assign)<br/> | Assigne les données du composant vectoriel d’une instance de `XMBYTEN4` à l’instance actuelle de `XMBYTEN4` . <br/> |
