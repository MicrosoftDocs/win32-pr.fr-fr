# <a name="xmubyten2-extensions"></a>Extensions XMUBYTEN2

[**XMUBYTEN2**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmubyten2) décrit un vecteur 2D pour stocker des valeurs non signées et normalisées en tant qu’entiers 8 bits non signés (1 octet).

> [!Note]  
> Les fonctionnalités répertoriées ici sont disponibles uniquement lors du développement avec C++.
