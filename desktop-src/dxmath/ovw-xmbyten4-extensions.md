# <a name="xmbyten4-extensions"></a>Extensions XMBYTEN4

[**XMBYTEN4**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmbyten4) décrit un vecteur 3D pour le stockage des valeurs signées et normalisées en tant qu’entiers 8 bits signés (1 octet).

> [!Note]  
> Les fonctionnalités répertoriées ici sont disponibles uniquement lors du développement avec C++.
