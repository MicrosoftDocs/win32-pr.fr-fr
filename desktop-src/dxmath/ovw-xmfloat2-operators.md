# <a name="xmfloat2-operators"></a>XMFLOAT2, opérateurs

La structure [**XMFLOAT2**](/windows/win32/api/directxmath/ns-directxmath-xmfloat2) expose les opérateurs suivants.

> [!Note]  
> Les opérateurs répertoriés ici sont uniquement disponibles en C++.

 

## <a name="in-this-section"></a>Dans cette section



| Méthodes                                               | Description                                                                                                         |
|-------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------|
| [**opérateur =**](/windows/win32/api/directxmath/nf-directxmath-xmfloat2-operator-assign)<br/> | Assigne les données du composant vectoriel d’une instance de `XMFLOAT2` à l’instance actuelle de `XMFLOAT2` .<br/> |
