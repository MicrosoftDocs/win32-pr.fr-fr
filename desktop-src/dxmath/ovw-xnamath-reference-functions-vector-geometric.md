# <a name="geometric-vector-functions"></a>Fonctions vectorielles géométriques

Répertorie les fonctions vectorielles géométriques.

## <a name="in-this-section"></a>Contenu de cette section



| Rubrique                                                           | Description                                                                                                                                                                                                      |
|-----------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XMVectorBaryCentric**](/windows/win32/api/directxmath/nf-directxmath-xmvectorbarycentric)<br/>   | Retourne un point dans les coordonnées Barycentric, à l’aide des vecteurs de position spécifiés.<br/>                                                                                                                     |
| [**XMVectorBaryCentricV**](/windows/win32/api/directxmath/nf-directxmath-xmvectorbarycentricv)<br/> | Retourne un point dans les coordonnées Barycentric, à l’aide des vecteurs de position spécifiés.<br/>                                                                                                                     |
| [**XMVectorCatmullRom**](/windows/win32/api/directxmath/nf-directxmath-xmvectorcatmullrom)<br/>     | Effectue une interpolation Catmull-Rom à l’aide des vecteurs de position spécifiés.<br/>                                                                                                                           |
| [**XMVectorCatmullRomV**](/windows/win32/api/directxmath/nf-directxmath-xmvectorcatmullromv)<br/>   | Effectue une interpolation Catmull-Rom à l’aide des vecteurs de position spécifiés.<br/>                                                                                                                           |
| [**XMVectorHermite**](/windows/win32/api/directxmath/nf-directxmath-xmvectorhermite)<br/>           | Effectue une interpolation spline Hermite à l’aide des vecteurs spécifiés.<br/>                                                                                                                                 |
| [**XMVectorHermiteV**](/windows/win32/api/directxmath/nf-directxmath-xmvectorhermitev)<br/>         | Effectue une interpolation spline Hermite à l’aide des vecteurs spécifiés.<br/>                                                                                                                                 |
| [**XMVectorInBounds**](/windows/win32/api/directxmath/nf-directxmath-xmvectorinbounds)<br/>         | Teste si les composants d’un vecteur donné se trouvent dans des limites définies.<br/>                                                                                                                                 |
| [**XMVectorInBoundsR**](/windows/win32/api/directxmath/nf-directxmath-xmvectorinboundsr)<br/>       | Teste si les composants d’un vecteur donné se trouvent dans certaines limites et définit une valeur de comparaison qui peut être examinée à l’aide de fonctions telles que [**XMComparisonAllTrue**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue).<br/> |
| [**XMVectorLerp**](/windows/win32/api/directxmath/nf-directxmath-xmvectorlerp)<br/>                 | Effectue une interpolation linéaire entre deux vecteurs.<br/>                                                                                                                                                  |
| [**XMVectorLerpV**](/windows/win32/api/directxmath/nf-directxmath-xmvectorlerpv)<br/>               | Effectue une interpolation linéaire entre deux vecteurs.<br/>                                                                                                                                                  |
