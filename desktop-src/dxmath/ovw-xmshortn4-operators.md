# <a name="xmshortn4-operators"></a>XMSHORTN4, opérateurs

> [!Note]  
> Les opérateurs répertoriés ici sont uniquement disponibles en C++.

 

## <a name="in-this-section"></a>Dans cette section



| Méthodes                                                | Description                                                                                                            |
|--------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------|
| [**opérateur =**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmshortn4-operator-assign)<br/> | Assigne les données du composant vectoriel d’une instance de `XMSHORTN4` à l’instance actuelle de `XMSHORTN4` . <br/> |
