# <a name="xmxdec4-operators"></a>XMXDEC4, opérateurs

La structure [**XMXDEC4**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmxdec4) expose les opérateurs suivants.

> [!Note]  
> Les opérateurs répertoriés ici sont uniquement disponibles en C++.

 

## <a name="in-this-section"></a>Dans cette section



| Méthodes                                                            | Description                                                                                                           |
|--------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| [**UInt32 t (opérateur) \_**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmxdec4-operatoruint32_t)<br/> | Retourne une instance de `uint32_t` contenant les composants de l' `XMXDEC4` instance dans un format compressé. <br/> |
| [**opérateur =**](xmxdec4-operator-eq.md)<br/>               | Assigne des données de composant vectoriels à l’instance actuelle de `XMXDEC4` . <br/>                                       |
