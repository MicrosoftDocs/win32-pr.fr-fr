# <a name="directxmath-programming-guide"></a>Guide de programmation DirectXMath

DirectXMath fournit une solution mathématique optimisée pour Windows.

## <a name="in-this-section"></a>Contenu de cette section



| Rubrique                                                             | Description                                                                                                                                                                                                                                                             |
|-------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Prise en main](pg-xnamath-getting-started.md)<br/>      | La bibliothèque DirectXMath implémente une interface optimale et portable pour les opérations algébriques arithmétiques et linéaires sur des vecteurs à virgule flottante simple précision (2D, 3D et 4D) ou des matrices (3 × 3 et 4 × 4). <br/>                                                    |
| [Nouveautés](pg-xnamath-whatsnew.md)<br/>                  | La bibliothèque DirectXMath est basée sur la [bibliothèque XNA Math C++ SIMD version 2,04](https://walbourn.github.io/). Ici, nous décrivons comment DirectXMath diffère de XNA Math et comment les versions de DirectXMath diffèrent. <br/> |
| [Migration de code](pg-xnamath-migration.md)<br/>             | Cette vue d’ensemble décrit les modifications requises pour migrer le code existant à l’aide de la bibliothèque Math XNA vers la bibliothèque DirectXMath.<br/>                                                                                                                                 |
| [Utilisation de D3DXMath](pg-xnamath-migration-d3dx.md)<br/> | D3DXMath est une bibliothèque d’assistance mathématique pour les applications Direct3D. <br/>                                                                                                                                                                                                |
| [Optimisation du code](pg-xnamath-optimizing.md)<br/>         | Cette rubrique décrit les considérations et les stratégies d’optimisation avec la bibliothèque DirectXMath.<br/>                                                                                                                                                                |
| [Éléments internes de bibliothèque](pg-xnamath-internals.md)<br/>          | Cette rubrique décrit la conception interne de la bibliothèque DirectXMath.<br/>                                                                                                                                                                                         |
