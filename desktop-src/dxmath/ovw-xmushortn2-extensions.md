# <a name="xmushortn2-extensions"></a>Extensions XMUSHORTN2

[**XMUSHORTN2**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmushortn2) décrit un vecteur 2D pour stocker des valeurs non signées et normalisées en tant qu’entiers 16 bits non signés, (type `int16_t` ).

> [!Note]  
> Les fonctionnalités répertoriées ici sont disponibles uniquement lors du développement avec C++.
