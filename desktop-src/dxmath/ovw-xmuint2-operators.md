# <a name="xmuint2-operators"></a>XMUINT2, opérateurs

La structure [**XMUINT2**](/windows/win32/api/directxmath/ns-directxmath-xmuint2) expose les opérateurs suivants.

> [!Note]  
> Les opérateurs répertoriés ici sont uniquement disponibles en C++.

 

## <a name="in-this-section"></a>Dans cette section



| Méthodes                                              | Description                                                                                                       |
|------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------|
| [**opérateur =**](/windows/win32/api/directxmath/nf-directxmath-xmuint2-operator-assign)<br/> | Assigne les données du composant vectoriel d’une instance de `XMUINT2` à l’instance actuelle de `XMUINT2` .<br/> |
