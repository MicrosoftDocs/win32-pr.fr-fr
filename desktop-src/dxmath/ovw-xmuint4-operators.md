# <a name="xmuint4-operators"></a>XMUINT4, opérateurs

La structure [**XMUINT4**](/windows/win32/api/directxmath/ns-directxmath-xmuint4) expose les opérateurs suivants.

> [!Note]  
> Les opérateurs répertoriés ici sont uniquement disponibles en C++.

 

## <a name="in-this-section"></a>Dans cette section



| Méthodes                                              | Description                                                                                                       |
|------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------|
| [**opérateur =**](/windows/win32/api/directxmath/nf-directxmath-xmuint4-operator-assign)<br/> | Assigne les données du composant vectoriel d’une instance de `XMUINT4` à l’instance actuelle de `XMUINT4` .<br/> |
