# <a name="directxmath-library-3d-vector-comparison-functions"></a>Fonctions de comparaison de vecteurs 3D de la bibliothèque DirectXMath

Répertorie les fonctions de comparaison de vecteurs 3D.

## <a name="in-this-section"></a>Contenu de cette section



| Rubrique                                                                   | Description                                                                                                                                                                                                                                               |
|-------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XMVector3Equal**](/windows/win32/api/directxmath/nf-directxmath-xmvector3equal)<br/>                     | Teste si deux vecteurs 3D sont égaux.<br/>                                                                                                                                                                                                        |
| [**XMVector3EqualInt**](/windows/win32/api/directxmath/nf-directxmath-xmvector3equalint)<br/>               | Teste si deux vecteurs 3D sont égaux, en traitant chaque composant comme un entier non signé.<br/>                                                                                                                                                        |
| [**XMVector3EqualIntR**](/windows/win32/api/directxmath/nf-directxmath-xmvector3equalintr)<br/>             | Teste si deux vecteurs 3D sont égaux, en traitant chaque composant comme un entier non signé. En outre, cette fonction retourne une valeur de comparaison qui peut être examinée à l’aide de fonctions telles que [**XMComparisonAllTrue**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue).<br/> |
| [**XMVector3EqualR**](/windows/win32/api/directxmath/nf-directxmath-xmvector3equalr)<br/>                   | Teste si deux vecteurs 3D sont égaux. En outre, cette fonction retourne une valeur de comparaison qui peut être examinée à l’aide de fonctions telles que [**XMComparisonAllTrue**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue).<br/>                                                 |
| [**XMVector3Greater**](/windows/win32/api/directxmath/nf-directxmath-xmvector3greater)<br/>                 | Teste si un vecteur 3D est supérieur à un autre vecteur 3D.<br/>                                                                                                                                                                                 |
| [**XMVector3GreaterOrEqual**](/windows/win32/api/directxmath/nf-directxmath-xmvector3greaterorequal)<br/>   | Teste si un vecteur 3D est supérieur ou égal à un autre vecteur 3D.<br/>                                                                                                                                                                     |
| [**XMVector3GreaterOrEqualR**](/windows/win32/api/directxmath/nf-directxmath-xmvector3greaterorequalr)<br/> | Teste si un vecteur 3D est supérieur ou égal à un autre vecteur 3D et retourne une valeur de comparaison qui peut être examinée à l’aide de fonctions telles que [**XMComparisonAllTrue**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue).<br/>                                      |
| [**XMVector3GreaterR**](/windows/win32/api/directxmath/nf-directxmath-xmvector3greaterr)<br/>               | Teste si un vecteur 3D est supérieur à un autre vecteur 3D et retourne une valeur de comparaison qui peut être examinée à l’aide de fonctions telles que [**XMComparisonAllTrue**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue).<br/>                                                  |
| [**XMVector3IsInfinite**](/windows/win32/api/directxmath/nf-directxmath-xmvector3isinfinite)<br/>           | Teste si un composant d’un vecteur 3D est un infini positif ou négatif.<br/>                                                                                                                                                                   |
| [**XMVector3IsNaN**](/windows/win32/api/directxmath/nf-directxmath-xmvector3isnan)<br/>                     | Teste si un composant d’un vecteur 3D est un NaN.<br/>                                                                                                                                                                                           |
| [**XMVector3Less**](/windows/win32/api/directxmath/nf-directxmath-xmvector3less)<br/>                       | Teste si un vecteur 3D est inférieur à un autre vecteur 3D.<br/>                                                                                                                                                                                    |
| [**XMVector3LessOrEqual**](/windows/win32/api/directxmath/nf-directxmath-xmvector3lessorequal)<br/>         | Teste si un vecteur 3D est inférieur ou égal à un autre vecteur 3D.<br/>                                                                                                                                                                        |
| [**XMVector3NearEqual**](/windows/win32/api/directxmath/nf-directxmath-xmvector3nearequal)<br/>             | Teste si un vecteur 3D est proche d’un autre vecteur 3D.<br/>                                                                                                                                                                                         |
| [**XMVector3NotEqual**](/windows/win32/api/directxmath/nf-directxmath-xmvector3notequal)<br/>               | Teste si deux vecteurs 3D ne sont pas égaux.<br/>                                                                                                                                                                                                    |
| [**XMVector3NotEqualInt**](/windows/win32/api/directxmath/nf-directxmath-xmvector3notequalint)<br/>         | Teste si deux vecteurs 3D ne sont pas égaux, en traitant chaque composant en tant qu’entier non signé.<br/>                                                                                                                                                     |
