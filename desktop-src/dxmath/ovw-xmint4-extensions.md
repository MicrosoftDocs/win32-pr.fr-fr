# <a name="xmint4-extensions"></a>Extensions XMINT4

[**XMINT4**](/windows/win32/api/directxmath/ns-directxmath-xmint4) décrit une structure contenant un vecteur 4D constitué de quatre valeurs entières.

> [!Note]  
> Les fonctionnalités répertoriées ici sont disponibles uniquement lors du développement avec C++.
