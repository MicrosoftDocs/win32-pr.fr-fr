# <a name="xmint4-operators"></a>XMINT4, opérateurs

La structure [**XMINT4**](/windows/win32/api/directxmath/ns-directxmath-xmint4) expose les opérateurs suivants.

> [!Note]  
> Les opérateurs répertoriés ici sont uniquement disponibles en C++.

 

## <a name="in-this-section"></a>Dans cette section



| Méthodes                                             | Description                                                                                                     |
|-----------------------------------------------------|-----------------------------------------------------------------------------------------------------------------|
| [**opérateur =**](/windows/win32/api/directxmath/nf-directxmath-xmint4-operator-assign)<br/> | Assigne les données du composant vectoriel d’une instance de `XMINT4` à l’instance actuelle de `XMINT4` .<br/> |
