# <a name="operator--operators"></a>opérateurs opérateur +

Opérateurs d’addition.

### <a name="overload-list"></a>Liste de surcharge




| Opérateur | Description | 
|----------|-------------|
| <a href="/previous-versions/windows/desktop/legacy/ee421395(v=vs.85)"><strong>XMVECTOR :: Operator + (XMVECTOR)</strong></a> | Performances d’une opération d’identité sur une <code>XMVECTOR</code> instance.<br /> <code>operator +</code>Prend une instance du <a href="xmvector-data-type.md"><strong>type de données XMVECTOR</strong></a> et retourne une nouvelle instance de <code>XMVECTOR</code> , avec un opérateur d’identité appliqué à chaque composant.<br /><blockquote>[!Note]<br />Cet opérateur est uniquement disponible en C++.</blockquote><br /> | 
| <a href="/previous-versions/windows/desktop/legacy/ee421396(v=vs.85)"><strong>XMVECTOR :: Operator + (XMVECTOR, XMVECTOR)</strong></a> | Ajoute deux instances de <code>XMVECTOR</code> , en retournant le résultat dans une nouvelle instance.<br /> Le <code>operator +</code> ajoute chaque composant de deux instances du <a href="xmvector-data-type.md"><strong>type de données XMVECTOR</strong></a>, puis retourne le résultat dans une nouvelle <code>XMVECTOR</code> instance. <br /><blockquote>[!Note]<br />Cet opérateur est uniquement disponible en C++.</blockquote><br /> |
