# <a name="xmdec4-operators"></a>XMDEC4, opérateurs

La structure [**XMDEC4**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmdec4) expose les opérateurs suivants.

> [!Note]  
> Les opérateurs répertoriés ici sont uniquement disponibles en C++.

 

## <a name="in-this-section"></a>Dans cette section



| Méthodes                                                           | Description                                                                                                          |
|-------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| [**UInt32 t (opérateur) \_**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmdec4-operatoruint32_t)<br/> | Retourne une instance de `uint32_t` contenant les composants de l' `XMDEC4` instance dans un format compressé. <br/> |
| [**opérateur =**](xmdec4-operator-eq.md)<br/>               | Assigne des données de composant vectoriels à l’instance actuelle de `XMDEC4` . <br/>                                       |
