# <a name="xmvector-operators"></a>XMVECTOR, opérateurs

Les opérateurs suivants sont exposés par la structure [**XMVECTOR**](xmvector-data-type.md) .

> [!Note]  
> Les opérateurs répertoriés ici sont uniquement disponibles en C++.

 

## <a name="in-this-section"></a>Dans cette section



| Méthodes                                                    | Description                                                                                                                                 |
|------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------|
| [**opérateur + =**](/previous-versions/windows/desktop/legacy/ee421394(v=vs.85))<br/>  | Ajoute une valeur à virgule flottante à une `XMVECTOR` instance et retourne une référence à l’instance mise à jour. <br/>                         |
| [**opérateur =**](/previous-versions/windows/desktop/legacy/ee421383(v=vs.85))<br/>    | Soustrait une valeur à virgule flottante de l’instance actuelle de `XMVECTOR` , en retournant le résultat dans l’instance actuelle mise à jour. <br/> |
| [**opérateur \** _](xmvector-operator-mul.md)<br/>    | Opérateurs de multiplication<br/>                                                                                                         |
| [_ *, \* = opérateur**](xmvector-operator-muleq.md)<br/> | Opérateurs d’assignation de multiplication<br/>                                                                                              |
| [**and**](xmvector-operator-div.md)<br/>     | Opérateur de division.<br/>                                                                                                               |
| [**opérateur/=**](xmvector-operator-diveq.md)<br/>  | Opérateur d’assignation de division.<br/>                                                                                                    |
| [**and**](xmvector-operator--.md)<br/>       | Opérateurs de soustraction et de négation<br/>                                                                                               |
| [**opérateur +**](xmvector-operator-pls.md)<br/>     | Opérateurs d’addition.<br/>                                                                                                              |
