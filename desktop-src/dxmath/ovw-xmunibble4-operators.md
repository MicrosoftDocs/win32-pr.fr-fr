# <a name="xmunibble4-operators"></a>XMUNIBBLE4, opérateurs

La structure [**XMUNIBBLE4**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmunibble4) expose les opérateurs suivants.

> [!Note]  
> Les opérateurs répertoriés ici sont uniquement disponibles en C++.

 

## <a name="in-this-section"></a>Dans cette section



| Méthodes                                                               | Description                                                                                                              |
|-----------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------|
| [**UInt16, opérateur \_ t**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmunibble4-operatoruint16_t)<br/> | Retourne une instance de `uint16_t` contenant les composants de l' `XMUNIBBLE4` instance dans un format compressé. <br/> |
| [**opérateur =**](xmunibble4-operator-eq.md)<br/>               | Assigne des données de composant vectoriels à l’instance actuelle de `XMUNIBBLE4` . <br/>                                       |
