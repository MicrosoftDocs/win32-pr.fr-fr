# <a name="xmint3-operators"></a>XMINT3, opérateurs

La structure [**XMINT3**](/windows/win32/api/directxmath/ns-directxmath-xmint3) expose les opérateurs suivants.

> [!Note]  
> Les opérateurs répertoriés ici sont uniquement disponibles en C++.

 

## <a name="in-this-section"></a>Dans cette section



| Méthodes                                             | Description                                                                                                      |
|-----------------------------------------------------|------------------------------------------------------------------------------------------------------------------|
| [**opérateur =**](/windows/win32/api/directxmath/nf-directxmath-xmint3-operator-assign)<br/> | Assigne les données du composant vectoriel d’une instance de `XMINT3` à l’instance actuelle de `XMINT3` . <br/> |
