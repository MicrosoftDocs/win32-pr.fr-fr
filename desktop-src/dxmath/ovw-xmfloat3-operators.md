# <a name="xmfloat3-operators"></a>XMFLOAT3, opérateurs

La structure [**XMFLOAT3**](/windows/win32/api/directxmath/ns-directxmath-xmfloat3) expose les opérateurs suivants.

> [!Note]  
> Les opérateurs répertoriés ici sont uniquement disponibles en C++.

 

## <a name="in-this-section"></a>Dans cette section



| Méthodes                                               | Description                                                                                                          |
|-------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| [**opérateur =**](/windows/win32/api/directxmath/nf-directxmath-xmfloat3-operator-assign)<br/> | Assigne les données du composant vectoriel d’une instance de `XMFLOAT3` à l’instance actuelle de `XMFLOAT3` . <br/> |
