# <a name="xmu555-operators"></a>XMU555, opérateurs

La structure [**XMU555**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmu555) expose les opérateurs suivants.

> [!Note]  
> Les opérateurs répertoriés ici sont uniquement disponibles en C++.

 

## <a name="in-this-section"></a>Dans cette section



| Méthodes                                                           | Description                                                                                                          |
|-------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| [**UInt16, opérateur \_ t**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmu555-operatoruint16_t)<br/> | Retourne une instance de `uint16_t` contenant les composants de l' `XMU555` instance dans un format compressé. <br/> |
| [**opérateur =**](xmu555-operator-eq.md)<br/>               | Assigne des données de composant vectoriels à l’instance actuelle de `XMU555` . <br/>                                       |
