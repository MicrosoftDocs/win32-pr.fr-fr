# <a name="directxmath-3d-vector-transformation-functions"></a>DirectXMath, fonctions de transformation de vecteur 3D

Répertorie les fonctions de transformation vectorielle 3D.

## <a name="in-this-section"></a>Contenu de cette section



| Rubrique                                                                               | Description                                                                                                                                      |
|-------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XMVector3InverseRotate**](/windows/win32/api/directxmath/nf-directxmath-xmvector3inverserotate)<br/>                 | Fait pivoter un vecteur 3D à l’aide de l’inverse d’un Quaternion.<br/>                                                                                |
| [**XMVector3Project**](/windows/win32/api/directxmath/nf-directxmath-xmvector3project)<br/>                             | Projetez un vecteur 3D à partir de l’espace d’objet dans l’espace à l’écran.<br/>                                                                              |
| [**XMVector3ProjectStream**](/windows/win32/api/directxmath/nf-directxmath-xmvector3projectstream)<br/>                 | Projette un flux de vecteurs 3D à partir de l’espace d’objet dans l’espace à l’écran.<br/>                                                                  |
| [**XMVector3Rotate**](/windows/win32/api/directxmath/nf-directxmath-xmvector3rotate)<br/>                               | Fait pivoter un vecteur 3D à l’aide d’un Quaternion.<br/>                                                                                               |
| [**XMVector3Transform**](/windows/win32/api/directxmath/nf-directxmath-xmvector3transform)<br/>                         | Transforme un vecteur 3D par une matrice.<br/>                                                                                                   |
| [**XMVector3TransformCoord**](/windows/win32/api/directxmath/nf-directxmath-xmvector3transformcoord)<br/>               | Transforme un vecteur 3D par une matrice donnée, en reprojetant le résultat dans w = 1.<br/>                                                      |
| [**XMVector3TransformCoordStream**](/windows/win32/api/directxmath/nf-directxmath-xmvector3transformcoordstream)<br/>   | Transforme un flux de vecteurs 3D par une matrice donnée, en projetant les vecteurs résultants de telle sorte que leurs coordonnées w soient égales à 1,0.<br/> |
| [**XMVector3TransformNormal**](/windows/win32/api/directxmath/nf-directxmath-xmvector3transformnormal)<br/>             | Transforme le vecteur 3D normal par la matrice donnée.<br/>                                                                                  |
| [**XMVector3TransformNormalStream**](/windows/win32/api/directxmath/nf-directxmath-xmvector3transformnormalstream)<br/> | Transforme un flux de vecteurs normaux 3D par une matrice donnée.<br/>                                                                           |
| [**XMVector3TransformStream**](/windows/win32/api/directxmath/nf-directxmath-xmvector3transformstream)<br/>             | Transforme un flux de vecteurs 3D par une matrice donnée.<br/>                                                                                  |
| [**XMVector3Unproject**](/windows/win32/api/directxmath/nf-directxmath-xmvector3unproject)<br/>                         | Projette un vecteur 3D à partir de l’espace à l’écran dans l’espace d’objet.<br/>                                                                             |
| [**XMVector3UnprojectStream**](/windows/win32/api/directxmath/nf-directxmath-xmvector3unprojectstream)<br/>             | Transforme un flux de vecteurs 3D de l’espace à l’écran en espace d’objet.<br/>                                                                  |
