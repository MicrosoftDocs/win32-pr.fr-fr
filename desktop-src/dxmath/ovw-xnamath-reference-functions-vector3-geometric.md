# <a name="directxmath-library-3d-vector-geometric-functions"></a>Fonctions géométriques Vector 3D de la bibliothèque DirectXMath

Répertorie les fonctions géométriques du vecteur 3D.

## <a name="in-this-section"></a>Contenu de cette section



| Rubrique                                                                                 | Description                                                                                                                       |
|---------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------|
| [**XMVector3AngleBetweenNormals**](/windows/win32/api/directxmath/nf-directxmath-xmvector3anglebetweennormals)<br/>       | Calcule l’angle en radians entre deux vecteurs 3D normalisés.<br/>                                                           |
| [**XMVector3AngleBetweenNormalsEst**](/windows/win32/api/directxmath/nf-directxmath-xmvector3anglebetweennormalsest)<br/> | Évalue l’angle en radians entre deux vecteurs 3D normalisés.<br/>                                                          |
| [**XMVector3AngleBetweenVectors**](/windows/win32/api/directxmath/nf-directxmath-xmvector3anglebetweenvectors)<br/>       | Calcule l’angle en radians entre deux vecteurs 3D.<br/>                                                                      |
| [**XMVector3ClampLength**](/windows/win32/api/directxmath/nf-directxmath-xmvector3clamplength)<br/>                       | Fixe la longueur d’un vecteur 3D à une plage donnée.<br/>                                                                     |
| [**XMVector3ClampLengthV**](/windows/win32/api/directxmath/nf-directxmath-xmvector3clamplengthv)<br/>                     | Fixe la longueur d’un vecteur 3D à une plage donnée.<br/>                                                                     |
| [**XMVector3ComponentsFromNormal**](/windows/win32/api/directxmath/nf-directxmath-xmvector3componentsfromnormal)<br/>     | À l’aide d’un vecteur de référence normal, divise un vecteur 3D en composants parallèles et perpendiculaires à la normale.<br/> |
| [**XMVector3Cross**](/windows/win32/api/directxmath/nf-directxmath-xmvector3cross)<br/>                                   | Calcule le produit croisé entre deux vecteurs 3D.<br/>                                                                     |
| [**XMVector3Dot**](/windows/win32/api/directxmath/nf-directxmath-xmvector3dot)<br/>                                       | Calcule le produit scalaire entre les vecteurs 3D.<br/>                                                                           |
| [**XMVector3InBounds**](/windows/win32/api/directxmath/nf-directxmath-xmvector3inbounds)<br/>                             | Teste si les composants d’un vecteur 3D se trouvent dans des limites définies.<br/>                                                     |
| [**XMVector3Length**](/windows/win32/api/directxmath/nf-directxmath-xmvector3length)<br/>                                 | Calcule la longueur d’un vecteur 3D.<br/>                                                                                    |
| [**XMVector3LengthEst**](/windows/win32/api/directxmath/nf-directxmath-xmvector3lengthest)<br/>                           | Évalue la longueur d’un vecteur 3D.<br/>                                                                                   |
| [**XMVector3LengthSq**](/windows/win32/api/directxmath/nf-directxmath-xmvector3lengthsq)<br/>                             | Calcule le carré de la longueur d’un vecteur 3D.<br/>                                                                      |
| [**XMVector3LinePointDistance**](/windows/win32/api/directxmath/nf-directxmath-xmvector3linepointdistance)<br/>           | Calcule la distance minimale entre une ligne et un point.<br/>                                                              |
| [**XMVector3Normalize**](/windows/win32/api/directxmath/nf-directxmath-xmvector3normalize)<br/>                           | Retourne la version normalisée d’un vecteur 3D.<br/>                                                                         |
| [**XMVector3NormalizeEst**](/windows/win32/api/directxmath/nf-directxmath-xmvector3normalizeest)<br/>                     | Estime la version normalisée d’un vecteur 3D.<br/>                                                                       |
| [**XMVector3Orthogonal**](/windows/win32/api/directxmath/nf-directxmath-xmvector3orthogonal)<br/>                         | Calcule un vecteur perpendiculaire à un vecteur 3D.<br/>                                                                        |
| [**XMVector3ReciprocalLength**](/windows/win32/api/directxmath/nf-directxmath-xmvector3reciprocallength)<br/>             | Calcule la réciproque de la longueur d’un vecteur 3D.<br/>                                                                  |
| [**XMVector3ReciprocalLengthEst**](/windows/win32/api/directxmath/nf-directxmath-xmvector3reciprocallengthest)<br/>       | Estime la réciproque de la longueur d’un vecteur 3D.<br/>                                                                 |
| [**XMVector3Reflect**](/windows/win32/api/directxmath/nf-directxmath-xmvector3reflect)<br/>                               | Reflète un vecteur 3D d’incident sur un vecteur normal 3D.<br/>                                                              |
| [**XMVector3Refract**](/windows/win32/api/directxmath/nf-directxmath-xmvector3refract)<br/>                               | Réfraction d’un vecteur d’incident 3D à travers un vecteur normal 3D.<br/>                                                              |
| [**XMVector3RefractV**](/windows/win32/api/directxmath/nf-directxmath-xmvector3refractv)<br/>                             | Réfraction d’un vecteur d’incident 3D à travers un vecteur normal 3D.<br/>                                                              |
