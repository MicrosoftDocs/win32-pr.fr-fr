# <a name="directxmath-library-scalar-functions"></a>Fonctions scalaires de la bibliothèque DirectXMath

Répertorie les fonctions scalaires fournies par DirectXMath.

## <a name="in-this-section"></a>Contenu de cette section



| Rubrique                                                     | Description                                                          |
|-----------------------------------------------------------|----------------------------------------------------------------------|
| [**XMScalarACos**](/windows/win32/api/directxmath/nf-directxmath-xmscalaracos)<br/>           | Calcule l’arc cosinus d’un nombre à virgule flottante.<br/>        |
| [**XMScalarACosEst**](/windows/win32/api/directxmath/nf-directxmath-xmscalaracosest)<br/>     | Évalue l’arc cosinus d’un nombre à virgule flottante.<br/>       |
| [**XMScalarASin**](/windows/win32/api/directxmath/nf-directxmath-xmscalarasin)<br/>           | Calcule l’arc sinus d’un nombre à virgule flottante.<br/>          |
| [**XMScalarASinEst**](/windows/win32/api/directxmath/nf-directxmath-xmscalarasinest)<br/>     | Évalue l’arc sinus d’un nombre à virgule flottante.<br/>         |
| [**XMScalarCos**](/windows/win32/api/directxmath/nf-directxmath-xmscalarcos)<br/>             | Calcule le cosinus d’un angle en radians.<br/>                    |
| [**XMScalarCosEst**](/windows/win32/api/directxmath/nf-directxmath-xmscalarcosest)<br/>       | Estime le cosinus d’un angle en radians.<br/>                   |
| [**XMScalarModAngle**](/windows/win32/api/directxmath/nf-directxmath-xmscalarmodangle)<br/>   | Calcule un angle entre-XM \_ pi et XM \_ pi.<br/>             |
| [**XMScalarNearEqual**](/windows/win32/api/directxmath/nf-directxmath-xmscalarnearequal)<br/> | Détermine si deux valeurs à virgule flottante sont presque égales.<br/> |
| [**XMScalarSin**](/windows/win32/api/directxmath/nf-directxmath-xmscalarsin)<br/>             | Calcule le sinus d’un angle en radians.<br/>                      |
| [**XMScalarSinCos**](/windows/win32/api/directxmath/nf-directxmath-xmscalarsincos)<br/>       | Calcule le sinus et le cosinus d’un angle radian.<br/>      |
| [**XMScalarSinCosEst**](/windows/win32/api/directxmath/nf-directxmath-xmscalarsincosest)<br/> | Estime à la fois le sinus et le cosinus d’un angle en radians.<br/>     |
| [**XMScalarSinEst**](/windows/win32/api/directxmath/nf-directxmath-xmscalarsinest)<br/>       | Évalue le sinus d’un angle en radians.<br/>                     |
