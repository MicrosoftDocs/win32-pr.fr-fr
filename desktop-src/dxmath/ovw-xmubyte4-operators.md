# <a name="xmubyte4-operators"></a>XMUBYTE4, opérateurs

La structure [**XMUBYTE4**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmubyte4) expose les opérateurs suivants.

> [!Note]  
> Les opérateurs répertoriés ici sont uniquement disponibles en C++.

 

## <a name="in-this-section"></a>Dans cette section



| Méthodes                                               | Description                                                                                                          |
|-------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| [**opérateur =**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmubyte4-operator-assign)<br/> | Assigne les données du composant vectoriel d’une instance de `XMUBYTE4` à l’instance actuelle de `XMUBYTE4` . <br/> |
