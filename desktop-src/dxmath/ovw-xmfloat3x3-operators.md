# <a name="xmfloat3x3-operators"></a>XMFLOAT3X3, opérateurs

La structure [**XMFLOAT3X3**](/windows/win32/api/directxmath/ns-directxmath-xmfloat3x3) expose les opérateurs suivants.

> [!Note]  
> Les opérateurs répertoriés ici sont uniquement disponibles en C++.

 

## <a name="in-this-section"></a>Dans cette section



| Méthodes                                                      | Description                                                                                                              |
|--------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------|
| [**opérateur =**](/windows/win32/api/directxmath/nf-directxmath-xmfloat3x3-operator-assign)<br/>      | Assigne les données du composant vectoriel d’une instance de `XMFLOAT3X3` à l’instance actuelle de `XMFLOAT3X3` . <br/> |
| [**, opérateur ()**](xmfloat3x3-operator-parens.md)<br/> | Accède à des éléments de matrice spécifiques référencés par ligne et colonne à partir de l’instance actuelle de `XMFLOAT3X3` . <br/>    |
