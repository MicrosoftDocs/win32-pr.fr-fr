# <a name="directxmath-library-2d-vector-transformation-functions"></a>Fonctions de transformation de vecteur 2D de la bibliothèque DirectXMath

Répertorie les fonctions de transformation de vecteur 2D.

## <a name="in-this-section"></a>Contenu de cette section



| Rubrique                                                                               | Description                                                                                                                                      |
|-------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XMVector2Transform**](/windows/win32/api/directxmath/nf-directxmath-xmvector2transform)<br/>                         | Transforme un vecteur 2D par une matrice.<br/>                                                                                                   |
| [**XMVector2TransformCoord**](/windows/win32/api/directxmath/nf-directxmath-xmvector2transformcoord)<br/>               | Transforme un vecteur 2D par une matrice donnée, en reprojetant le résultat dans w = 1.<br/>                                                      |
| [**XMVector2TransformCoordStream**](/windows/win32/api/directxmath/nf-directxmath-xmvector2transformcoordstream)<br/>   | Transforme un flux de vecteurs 2D par une matrice donnée, en projetant les vecteurs résultants de telle sorte que leurs coordonnées w soient égales à 1,0.<br/> |
| [**XMVector2TransformNormal**](/windows/win32/api/directxmath/nf-directxmath-xmvector2transformnormal)<br/>             | Transforme le vecteur 2D normal par la matrice donnée.<br/>                                                                                  |
| [**XMVector2TransformNormalStream**](/windows/win32/api/directxmath/nf-directxmath-xmvector2transformnormalstream)<br/> | Transforme un flux de vecteurs normaux 2D par une matrice donnée.<br/>                                                                           |
| [**XMVector2TransformStream**](/windows/win32/api/directxmath/nf-directxmath-xmvector2transformstream)<br/>             | Transforme un flux de vecteurs 2D par une matrice donnée.<br/>                                                                                  |
