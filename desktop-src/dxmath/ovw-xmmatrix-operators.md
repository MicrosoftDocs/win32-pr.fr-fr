# <a name="xmmatrix-operators"></a>XMMATRIX, opérateurs

La structure [**XMMATRIX**](/windows/win32/api/directxmath/ns-directxmath-xmmatrix) expose les opérateurs suivants.

> [!Note]  
> Les opérateurs répertoriés ici sont uniquement disponibles en C++.

 

## <a name="in-this-section"></a>Dans cette section



| Méthodes                                                    | Description                                                                                                                                                                                   |
|------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**opérateur \** _](/windows/win32/api/directxmath/nf-directxmath-xmmatrix-operator-mult)<br/>    | Effectue une multiplication de matrice de l’instance actuelle de `XMMATRIX` par une autre instance de `XMMATRIX` .<br/>                                                                          |
| [_ *, \* = opérateur**](/windows/win32/api/directxmath/nf-directxmath-xmmatrix-operator-mult-assign)<br/> | Effectue une multiplication de matrice de l’instance actuelle de `XMMATRIX` par une autre instance de `XMMATRIX` et retourne une référence à l’instance actuelle, qui a été mise à jour. <br/> |
| [**opérateur =**](/windows/win32/api/directxmath/nf-directxmath-xmmatrix-operator-assign)<br/>      | Assigne les données de matrice d’une instance de `XMMATRIX` à l’instance actuelle de `XMMATRIX` et retourne un `reference` à l’instance actuelle. <br/>                                    |
| [**, opérateur ()**](xmmatrix-operator-parens.md)<br/> | Accède à des éléments de matrice spécifiques référencés par ligne et colonne à partir de l’instance actuelle de `XMMATRIX` . <br/>                                                                           |
