# <a name="xmshortn2-operators"></a>XMSHORTN2, opérateurs

> [!Note]  
> Les opérateurs répertoriés ici sont uniquement disponibles en C++.

 

## <a name="in-this-section"></a>Dans cette section



| Méthodes                                                | Description                                                                                                           |
|--------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| [**opérateur =**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmshortn2-operator-assign)<br/> | Assigne les données du composant vectoriel d’une instance de `XMSHORTN2` à l’instance actuelle de `XMSHORTN2` .<br/> |
