# <a name="boundingbox-methods"></a>Méthodes BoundingBox



| Méthode                                                              | Description                                                                                            |
|---------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------|
| [**Comprend**](boundingbox-contains.md)<br/>                 | Teste si BoundingBox contient un objet spécifié.<br/>                                  |
| [**CreateFromPoints**](boundingbox-createfrompoints.md)<br/> | Crée un BoundingBox à partir de points.<br/>                                                          |
| [**Croise**](boundingbox-intersects.md)<br/>             | Teste BoundingBox pour l’intersection avec un autre objet.<br/>                                 |
| [**Transform**](boundingbox-transform.md)<br/>               | Transforme le BoundingBox.<br/>                                                                 |
| [**ContainedBy**](/windows/desktop/api/DirectXCollision/nf-directxcollision-boundingbox-containedby)<br/>           | Teste si l' [**BoundingBox**](/windows/desktop/api/DirectXCollision/ns-directxcollision-boundingbox) est contenu dans le frustum spécifié.<br/> |
| [**CreateFromSphere**](/windows/win32/api/directxcollision/nf-directxcollision-boundingbox-createfromsphere)<br/> | Crée un BoundingBox suffisamment grand pour contenir le BoundingSphere spécifié.<br/>               |
| [**CreateMerged**](/windows/win32/api/directxcollision/nf-directxcollision-boundingbox-createmerged)<br/>         | Crée un BoundingBox suffisamment grand pour contenir deux instances BoundBox spécifiés.<br/>             |
| [**GetCorners**](/windows/win32/api/directxcollision/nf-directxcollision-boundingbox-getcorners)<br/>             | Récupère les angles du BoundingBox.<br/>                                                   |
| [**affectation d’opérations \_**](/windows/win32/api/directxcollision/nf-directxcollision-boundingbox-operator-assign)<br/>      | Copie les valeurs d’un autre BoundingBox.<br/>                                                     |
