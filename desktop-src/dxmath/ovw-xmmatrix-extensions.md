# <a name="xmmatrix-extensions"></a>Extensions XMMATRIX

[**XMMATRIX**](/windows/win32/api/directxmath/ns-directxmath-xmmatrix) décrit une \* matrice 4 4 alignée sur une limite de 16 octets qui mappe à quatre registres vectoriels matériels.

> [!Note]  
> Les fonctionnalités répertoriées ici sont uniquement disponibles en C++.
