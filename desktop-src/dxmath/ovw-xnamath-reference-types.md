# <a name="directxmath-library-types"></a>Types de bibliothèque DirectXMath

La bibliothèque DirectXMath fournit un certain nombre de structures et de types définis pour encapsuler les données afin de prendre en charge la facilité d’utilisation, l’optimisation et la portabilité.

La liste ci-dessous comprend des structures qui font actuellement partie de la bibliothèque DirectXMath et sont disponibles via l’en-tête DirectXMath. h.

## <a name="in-this-section"></a>Contenu de cette section



| Rubrique                                                             | Description                                                                                                                                                                       |
|-------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**DEMI-type de données**](half-data-type.md)<br/>               | Alias de **UInt16 \_ t** compressé avec un nombre à virgule flottante de 16 bits constitué d’un bit de signe, d’un exposant biaisé à 5 bits et d’une mantisse 10 bits.<br/>                         |
| [**Type de données XMVECTOR**](xmvector-data-type.md)<br/>       | Type portable utilisé pour représenter un vecteur de composants à virgule flottante ou entier 4 32 bits, chacun étant aligné de manière optimale et mappé à un registre de vecteur matériel.<br/>       |
| [**Type de données XMVECTORF32**](xmvectorf32-data-type.md)<br/> | Type portable opaque pour prendre en charge l’utilisation de la syntaxe d’initialiseur C/C++ pour charger des valeurs à virgule flottante dans une instance de type [**XMVECTOR**](xmvector-data-type.md) .<br/> |
| [**Type de données XMVECTORI32**](xmvectori32-data-type.md)<br/> | Type portable opaque pour prendre en charge l’utilisation de la syntaxe d’initialiseur C/C++ pour charger des valeurs entières dans une instance de type [**XMVECTOR**](xmvector-data-type.md) .<br/>        |
| [**Type de données XMVECTORU32**](xmvectoru32-data-type.md)<br/> | Type portable opaque pour prendre en charge l’utilisation de la syntaxe d’initialiseur C/C++ pour charger les \_ valeurs UInt32 t dans une instance de type XMVECTOR.<br/>                                    |
| [**Type de données XMVECTORU8**](xmvectoru8-data-type.md)<br/>   | Type portable opaque pour prendre en charge l’utilisation de la syntaxe d’initialiseur C/C++ pour charger les valeurs de UInt8 \_ t dans une instance de type XMVECTOR.<br/>                                     |
