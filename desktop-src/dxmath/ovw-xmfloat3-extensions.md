# <a name="xmfloat3-extensions"></a>Extensions XMFLOAT3

[**XMFLOAT3**](/windows/win32/api/directxmath/ns-directxmath-xmfloat3) décrit une structure contenant un vecteur 3D constitué de trois valeurs à virgule flottante simple précision.

> [!Note]  
> Les fonctionnalités répertoriées ici sont disponibles uniquement lors du développement avec C++.
