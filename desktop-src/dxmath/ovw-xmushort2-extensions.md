# <a name="xmushort2-extensions"></a>Extensions XMUSHORT2

[**XMUSHORT2**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmushort2) décrit le vecteur a2D pour le stockage des valeurs non signées et normalisées en tant qu’entiers 16 bits non signés, (type `uint16_t` ).

> [!Note]  
> Les fonctionnalités répertoriées ici sont disponibles uniquement lors du développement avec C++.
