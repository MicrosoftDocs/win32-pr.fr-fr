# <a name="xmfloat4x3-operator--operators"></a>Opérateurs XMFLOAT4X3 Operator ()

Accède à des éléments de matrice spécifiques référencés par ligne et colonne à partir de l’instance actuelle de [ **XMFLOAT4X3**](/windows/win32/api/directxmath/ns-directxmath-xmfloat4x3)
