# <a name="half-data-type"></a>DEMI-type de données

Alias de **UInt16 \_ t** compressé avec un nombre à virgule flottante de 16 bits constitué d’un bit de signe, d’un exposant biaisé à 5 bits et d’une mantisse 10 bits.


```C++
typedef uint16_t HALF;
```



## <a name="remarks"></a>Notes

Le type de données demi est équivalent au format IEEE 754 binary16.

DEMI \_ -min = 6.10352 e-5F

MOITIÉ \_ Max = 65504. f

Pour plus d’informations sur le demi-type de données, consultez [format à virgule flottante demi-précision](https://en.wikipedia.org/wiki/Half_precision_floating-point_format).

**Espace de noms**: utilisez DirectX ::P ackedvector
