# <a name="xmdecn4-operators"></a>XMDECN4, opérateurs

La structure [**XMDECN4**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmdecn4) expose les opérateurs suivants.

> [!Note]  
> Les opérateurs répertoriés ici sont uniquement disponibles en C++.

 

## <a name="in-this-section"></a>Dans cette section



| Méthodes                                                            | Description                                                                                                           |
|--------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| [**UInt32 t (opérateur) \_**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmdecn4-operatoruint32_t)<br/> | Retourne une instance de `uint32_t` contenant les composants de l' `XMDECN4` instance dans un format compressé. <br/> |
| [**opérateur =**](xmdecn4-operator-eq.md)<br/>               | Assigne des données de composant vectoriels à l’instance actuelle de `XMDECN4` . <br/>                                       |
