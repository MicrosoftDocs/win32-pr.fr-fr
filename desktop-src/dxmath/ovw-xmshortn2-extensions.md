# <a name="xmshortn2-extensions"></a>Extensions XMSHORTN2

[**XMSHORTN2**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmshortn2) décrit un vecteur 2D pour le stockage des valeurs signées et normalisées en tant qu’entiers 16 bits signés, (type `int16_t` ).

> [!Note]  
> Les fonctionnalités répertoriées ici sont disponibles uniquement lors du développement avec C++.
