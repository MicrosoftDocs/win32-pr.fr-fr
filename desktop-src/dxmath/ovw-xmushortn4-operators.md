# <a name="xmushortn4-operators"></a>XMUSHORTN4, opérateurs

> [!Note]  
> Les opérateurs répertoriés ici sont uniquement disponibles en C++.

 

## <a name="in-this-section"></a>Dans cette section



| Méthodes                                                 | Description                                                                                                              |
|---------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------|
| [**opérateur =**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmushortn4-operator-assign)<br/> | Assigne les données du composant vectoriel d’une instance de `XMUSHORTN4` à l’instance actuelle de `XMUSHORTN4` . <br/> |
