# <a name="directxmath-library-4d-vector-comparison-functions"></a>Fonctions de comparaison vectorielle 4D de la bibliothèque DirectXMath

Répertorie les fonctions de comparaison vectorielle 4D.

## <a name="in-this-section"></a>Contenu de cette section



| Rubrique                                                                   | Description                                                                                                                                                                                                                                               |
|-------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XMVector4Equal**](/windows/win32/api/directxmath/nf-directxmath-xmvector4equal)<br/>                     | Teste si deux vecteurs 4D sont égaux.<br/>                                                                                                                                                                                                        |
| [**XMVector4EqualInt**](/windows/win32/api/directxmath/nf-directxmath-xmvector4equalint)<br/>               | Teste si deux vecteurs 4D sont égaux, en traitant chaque composant comme un entier non signé.<br/>                                                                                                                                                        |
| [**XMVector4EqualIntR**](/windows/win32/api/directxmath/nf-directxmath-xmvector4equalintr)<br/>             | Teste si deux vecteurs 4D sont égaux, en traitant chaque composant comme un entier non signé. En outre, cette fonction retourne une valeur de comparaison qui peut être examinée à l’aide de fonctions telles que [**XMComparisonAllTrue**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue).<br/> |
| [**XMVector4EqualR**](/windows/win32/api/directxmath/nf-directxmath-xmvector4equalr)<br/>                   | Teste si deux vecteurs 4D sont égaux. En outre, cette fonction retourne une valeur de comparaison qui peut être examinée à l’aide de fonctions telles que [**XMComparisonAllTrue**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue).<br/>                                                 |
| [**XMVector4Greater**](/windows/win32/api/directxmath/nf-directxmath-xmvector4greater)<br/>                 | Teste si un vecteur 4D est supérieur à un autre vecteur 4D.<br/>                                                                                                                                                                                 |
| [**XMVector4GreaterOrEqual**](/windows/win32/api/directxmath/nf-directxmath-xmvector4greaterorequal)<br/>   | Teste si un vecteur 4D est supérieur ou égal à un autre vecteur 4D.<br/>                                                                                                                                                                     |
| [**XMVector4GreaterOrEqualR**](/windows/win32/api/directxmath/nf-directxmath-xmvector4greaterorequalr)<br/> | Teste si un vecteur 4D est supérieur ou égal à un autre vecteur 4D et retourne une valeur de comparaison qui peut être examinée à l’aide de fonctions telles que [**XMComparisonAllTrue**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue).<br/>                                      |
| [**XMVector4GreaterR**](/windows/win32/api/directxmath/nf-directxmath-xmvector4greaterr)<br/>               | Teste si un vecteur 4D est supérieur à un autre vecteur 4D et retourne une valeur de comparaison qui peut être examinée à l’aide de fonctions telles que [**XMComparisonAllTrue**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue).<br/>                                                  |
| [**XMVector4IsInfinite**](/windows/win32/api/directxmath/nf-directxmath-xmvector4isinfinite)<br/>           | Teste si un composant d’un vecteur 4D est un infini positif ou négatif.<br/>                                                                                                                                                                   |
| [**XMVector4IsNaN**](/windows/win32/api/directxmath/nf-directxmath-xmvector4isnan)<br/>                     | Teste si un composant d’un vecteur 4D est un NaN.<br/>                                                                                                                                                                                           |
| [**XMVector4Less**](/windows/win32/api/directxmath/nf-directxmath-xmvector4less)<br/>                       | Teste si un vecteur 4D est inférieur à un autre vecteur 4D.<br/>                                                                                                                                                                                    |
| [**XMVector4LessOrEqual**](/windows/win32/api/directxmath/nf-directxmath-xmvector4lessorequal)<br/>         | Teste si un vecteur 4D est inférieur ou égal à un autre vecteur 4D.<br/>                                                                                                                                                                        |
| [**XMVector4NearEqual**](/windows/win32/api/directxmath/nf-directxmath-xmvector4nearequal)<br/>             | Teste si un vecteur 4D est proche d’un autre vecteur 4D.<br/>                                                                                                                                                                                         |
| [**XMVector4NotEqual**](/windows/win32/api/directxmath/nf-directxmath-xmvector4notequal)<br/>               | Teste si deux vecteurs 4D ne sont pas égaux.<br/>                                                                                                                                                                                                    |
| [**XMVector4NotEqualInt**](/windows/win32/api/directxmath/nf-directxmath-xmvector4notequalint)<br/>         | Teste si deux vecteurs 4D ne sont pas égaux, en traitant chaque composant en tant qu’entier non signé.<br/>                                                                                                                                                     |
