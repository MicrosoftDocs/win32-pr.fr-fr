# <a name="xmuint3-operators"></a>XMUINT3, opérateurs

La structure [**XMUINT3**](/windows/win32/api/directxmath/ns-directxmath-xmuint3) expose les opérateurs suivants.

> [!Note]  
> Les opérateurs répertoriés ici sont uniquement disponibles en C++.

 

## <a name="in-this-section"></a>Dans cette section



| Méthodes                                              | Description                                                                                                        |
|------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------|
| [**opérateur =**](/windows/win32/api/directxmath/nf-directxmath-xmuint3-operator-assign)<br/> | Assigne les données du composant vectoriel d’une instance de `XMUINT3` à l’instance actuelle de `XMUINT3` . <br/> |
