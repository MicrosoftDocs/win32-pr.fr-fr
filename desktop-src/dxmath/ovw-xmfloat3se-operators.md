# <a name="xmfloat3se-operators"></a>XMFLOAT3SE, opérateurs

La structure [**XMFLOAT3SE**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmfloat3se) expose les opérateurs suivants.

> [!Note]  
> Les opérateurs répertoriés ici sont uniquement disponibles en C++.

 

## <a name="in-this-section"></a>Dans cette section



| Méthodes                                                               | Description                                                                                                             |
|-----------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------|
| [**UInt32 t (opérateur) \_**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmfloat3se-operatoruint32_t)<br/> | Retourne une instance de `uint32_t` contenant les composants de l' `XMFLOAT3SE` instance dans un format compressé.<br/> |
| [**opérateur =**](xmfloat3se-operator-eq.md)<br/>               | Assigne des données de composant vectoriels à l’instance actuelle de `XMFLOAT3SE` .<br/>                                       |
