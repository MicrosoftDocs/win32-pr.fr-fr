# <a name="xmcolor-extensions"></a>Extensions XMCOLOR

[**XMCOLOR**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmcolor) décrit un vecteur de couleur rouge vert vert à 32 bits (ARGB), où chaque canal de couleurs est spécifié par 8 bits.

> [!Note]  
> Les fonctionnalités répertoriées ici sont disponibles uniquement lors du développement avec C++.
