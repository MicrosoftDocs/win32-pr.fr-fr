# <a name="xmuint4-extensions"></a>Extensions XMUINT4

[**XMUINT4**](/windows/win32/api/directxmath/ns-directxmath-xmuint4) décrit une structure contenant un vecteur 4D constitué de quatre valeurs entières non signées.

> [!Note]  
> Les fonctionnalités répertoriées ici sont disponibles uniquement lors du développement avec C++.
