# <a name="component-wise-vector-functions"></a>Fonctions vectorielles au niveau du composant

Répertorie les fonctions vectorielles au niveau du composant.

## <a name="in-this-section"></a>Contenu de cette section



| Rubrique                                                             | Description                                                                                                                              |
|-------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------|
| [**XMVectorInsert**](/windows/win32/api/directxmath/nf-directxmath-xmvectorinsert)<br/>               | Fait pivoter un vecteur vers la gauche d’un nombre donné de composants 32 bits et insérer les éléments sélectionnés de ce résultat dans un autre vecteur.<br/> |
| [**XMVectorMergeXY**](/windows/win32/api/directxmath/nf-directxmath-xmvectormergexy)<br/>             | Crée un nouveau vecteur en combinant les composants x et y de deux vecteurs.<br/>                                                      |
| [**XMVectorMergeZW**](/windows/win32/api/directxmath/nf-directxmath-xmvectormergezw)<br/>             | Crée un nouveau vecteur en combinant les composants z et w de deux vecteurs.<br/>                                                      |
| [**XMVectorPermute**](/windows/win32/api/directxmath/nf-directxmath-xmvectorpermute)<br/>             | Permute les composants de deux vecteurs pour créer un nouveau vecteur.<br/>                                                                |
| [**XMVectorRotateLeft**](/windows/win32/api/directxmath/nf-directxmath-xmvectorrotateleft)<br/>       | Fait pivoter le vecteur vers la gauche d’un nombre donné d’éléments de 32 bits.<br/>                                                                 |
| [**XMVectorRotateRight**](/windows/win32/api/directxmath/nf-directxmath-xmvectorrotateright)<br/>     | Fait pivoter le vecteur vers la droite d’un nombre donné d’éléments de 32 bits.<br/>                                                                |
| [**XMVectorSelect**](/windows/win32/api/directxmath/nf-directxmath-xmvectorselect)<br/>               | Effectue une sélection par composant entre deux vecteurs d’entrée et retourne le vecteur résultant.<br/>                                |
| [**XMVectorSelectControl**](/windows/win32/api/directxmath/nf-directxmath-xmvectorselectcontrol)<br/> | Définit un vecteur de contrôle à utiliser dans [**XMVectorSelect**](/windows/win32/api/directxmath/nf-directxmath-xmvectorselect).<br/>                                                 |
| [**XMVectorShiftLeft**](/windows/win32/api/directxmath/nf-directxmath-xmvectorshiftleft)<br/>         | Déplace un vecteur vers la gauche d’un nombre donné d’éléments 32 bits, en remplissant les éléments libérés avec les éléments d’un second vecteur.<br/>   |
| [**XMVectorSplatW**](/windows/win32/api/directxmath/nf-directxmath-xmvectorsplatw)<br/>               | Réplique le composant w d’un vecteur sur tous les composants.<br/>                                                              |
| [**XMVectorSplatX**](/windows/win32/api/directxmath/nf-directxmath-xmvectorsplatx)<br/>               | Réplique le composant x d’un vecteur sur tous les composants.<br/>                                                              |
| [**XMVectorSplatY**](/windows/win32/api/directxmath/nf-directxmath-xmvectorsplaty)<br/>               | Réplique le composant y d’un vecteur sur tous les composants.<br/>                                                              |
| [**XMVectorSplatZ**](/windows/win32/api/directxmath/nf-directxmath-xmvectorsplatz)<br/>               | Réplique le composant z d’un vecteur sur tous les composants.<br/>                                                              |
| [**XMVectorSwizzle**](/windows/win32/api/directxmath/nf-directxmath-xmvectorswizzle)<br/>             | Swizzles un vecteur.<br/>                                                                                                            |
