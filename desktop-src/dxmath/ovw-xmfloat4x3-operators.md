# <a name="xmfloat4x3-operators"></a>XMFLOAT4X3, opérateurs

La structure [**XMFLOAT4X3**](/windows/win32/api/directxmath/ns-directxmath-xmfloat4x3) expose les opérateurs suivants.

> [!Note]  
> Les opérateurs répertoriés ici sont uniquement disponibles en C++.

 

## <a name="in-this-section"></a>Dans cette section



| Méthodes                                                      | Description                                                                                                              |
|--------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------|
| [**opérateur =**](/windows/win32/api/directxmath/nf-directxmath-xmfloat4x3-operator-assign)<br/>      | Assigne les données du composant vectoriel d’une instance de `XMFLOAT4X3` à l’instance actuelle de `XMFLOAT4X3` . <br/> |
| [**, opérateur ()**](xmfloat4x3-operator-parens.md)<br/> | Accède à des éléments de matrice spécifiques référencés par ligne et colonne à partir de l’instance actuelle de `XMFLOAT4X3` . <br/>    |
