# <a name="directxmath-library-2d-vector-geometric-functions"></a>Fonctions géométriques du vecteur 2D de la bibliothèque DirectXMath

Répertorie les fonctions géométriques du vecteur 2D.

## <a name="in-this-section"></a>Contenu de cette section



| Rubrique                                                                                 | Description                                                                   |
|---------------------------------------------------------------------------------------|-------------------------------------------------------------------------------|
| [**XMVector2AngleBetweenNormals**](/windows/win32/api/directxmath/nf-directxmath-xmvector2anglebetweennormals)<br/>       | Calcule l’angle en radians entre deux vecteurs 2D normalisés.<br/>       |
| [**XMVector2AngleBetweenNormalsEst**](/windows/win32/api/directxmath/nf-directxmath-xmvector2anglebetweennormalsest)<br/> | Évalue l’angle en radians entre deux vecteurs 2D normalisés.<br/>      |
| [**XMVector2AngleBetweenVectors**](/windows/win32/api/directxmath/nf-directxmath-xmvector2anglebetweenvectors)<br/>       | Calcule l’angle en radians entre deux vecteurs 2D.<br/>                  |
| [**XMVector2ClampLength**](/windows/win32/api/directxmath/nf-directxmath-xmvector2clamplength)<br/>                       | Fixe la longueur d’un vecteur 2D à une plage donnée.<br/>                 |
| [**XMVector2ClampLengthV**](/windows/win32/api/directxmath/nf-directxmath-xmvector2clamplengthv)<br/>                     | Fixe la longueur d’un vecteur 2D à une plage donnée.<br/>                 |
| [**XMVector2Cross**](/windows/win32/api/directxmath/nf-directxmath-xmvector2cross)<br/>                                   | Calcule le produit croisé 2D.<br/>                                     |
| [**XMVector2Dot**](/windows/win32/api/directxmath/nf-directxmath-xmvector2dot)<br/>                                       | Calcule le produit scalaire entre les vecteurs 2D.<br/>                       |
| [**XMVector2InBounds**](/windows/win32/api/directxmath/nf-directxmath-xmvector2inbounds)<br/>                             | Teste si les composants d’un vecteur 2D se trouvent dans des limites définies.<br/> |
| [**XMVector2IntersectLine**](/windows/win32/api/directxmath/nf-directxmath-xmvector2intersectline)<br/>                   | Recherche l’intersection de deux lignes.<br/>                               |
| [**XMVector2Length**](/windows/win32/api/directxmath/nf-directxmath-xmvector2length)<br/>                                 | Calcule la longueur d’un vecteur 2D.<br/>                                |
| [**XMVector2LengthEst**](/windows/win32/api/directxmath/nf-directxmath-xmvector2lengthest)<br/>                           | Évalue la longueur d’un vecteur 2D.<br/>                               |
| [**XMVector2LengthSq**](/windows/win32/api/directxmath/nf-directxmath-xmvector2lengthsq)<br/>                             | Calcule le carré de la longueur d’un vecteur 2D.<br/>                  |
| [**XMVector2LinePointDistance**](/windows/win32/api/directxmath/nf-directxmath-xmvector2linepointdistance)<br/>           | Calcule la distance minimale entre une ligne et un point.<br/>          |
| [**XMVector2Normalize**](/windows/win32/api/directxmath/nf-directxmath-xmvector2normalize)<br/>                           | Retourne la version normalisée d’un vecteur 2D.<br/>                     |
| [**XMVector2NormalizeEst**](/windows/win32/api/directxmath/nf-directxmath-xmvector2normalizeest)<br/>                     | Estime la version normalisée d’un vecteur 2D.<br/>                   |
| [**XMVector2Orthogonal**](/windows/win32/api/directxmath/nf-directxmath-xmvector2orthogonal)<br/>                         | Calcule un vecteur perpendiculaire à un vecteur 2D.<br/>                    |
| [**XMVector2ReciprocalLength**](/windows/win32/api/directxmath/nf-directxmath-xmvector2reciprocallength)<br/>             | Calcule la réciproque de la longueur d’un vecteur 2D.<br/>              |
| [**XMVector2ReciprocalLengthEst**](/windows/win32/api/directxmath/nf-directxmath-xmvector2reciprocallengthest)<br/>       | Estime la réciproque de la longueur d’un vecteur 2D.<br/>             |
| [**XMVector2Reflect**](/windows/win32/api/directxmath/nf-directxmath-xmvector2reflect)<br/>                               | Reflète un vecteur 2D d’incident à travers un vecteur normal 2D.<br/>          |
| [**XMVector2Refract**](/windows/win32/api/directxmath/nf-directxmath-xmvector2refract)<br/>                               | Réfraction d’un vecteur 2D d’incident à travers un vecteur normal 2D.<br/>          |
| [**XMVector2RefractV**](/windows/win32/api/directxmath/nf-directxmath-xmvector2refractv)<br/>                             | Réfraction d’un vecteur 2D d’incident à travers un vecteur normal 2D.<br/>          |
