# <a name="xmint2-operators"></a>XMINT2, opérateurs

La structure [**XMINT2**](/windows/win32/api/directxmath/ns-directxmath-xmint2) expose les opérateurs suivants.

> [!Note]  
> Les opérateurs répertoriés ici sont uniquement disponibles en C++.

 

## <a name="in-this-section"></a>Dans cette section



| Méthodes                                             | Description                                                                                                     |
|-----------------------------------------------------|-----------------------------------------------------------------------------------------------------------------|
| [**opérateur =**](/windows/win32/api/directxmath/nf-directxmath-xmint2-operator-assign)<br/> | Assigne les données du composant vectoriel d’une instance de `XMINT2` à l’instance actuelle de `XMINT2` .<br/> |
