# <a name="xmfloat4-operators"></a>XMFLOAT4, opérateurs

La structure [**XMFLOAT4**](/windows/win32/api/directxmath/ns-directxmath-xmfloat4) expose les opérateurs suivants.

> [!Note]  
> Les opérateurs répertoriés ici sont uniquement disponibles en C++.

 

## <a name="in-this-section"></a>Dans cette section



| Méthodes                                               | Description                                                                                                         |
|-------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------|
| [**opérateur =**](/windows/win32/api/directxmath/nf-directxmath-xmfloat4-operator-assign(xmfloat4__))<br/> | Assigne les données du composant vectoriel d’une instance de `XMFLOAT4` à l’instance actuelle de `XMFLOAT4` .<br/> |
