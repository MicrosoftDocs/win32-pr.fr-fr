# <a name="interaction-context-structures"></a>Structures de contexte d’interaction

Les rubriques de cette section fournissent les spécifications de référence pour les structures de [contexte d’interaction](interaction-context-portal.md) .

## <a name="in-this-section"></a>Contenu de cette section

| Rubrique | Description |
|---|---|
| [**\_paramètre de diapositives croisées \_**](/windows/win32/api/interactioncontext/ns-interactioncontext-cross_slide_parameter)<br/>                           | Définit le seuil de la diapositive croisée et son seuil de distance.<br/>                                                                                                    |
| [**\_intersection des arguments d’interaction \_ \_**](/windows/win32/api/interactioncontext/ns-interactioncontext-interaction_arguments_cross_slide)<br/>  | Définit l’état de l’interaction entre les diapositives.<br/>                                                                                                                |
| [**\_manipulation des arguments d’interaction \_**](/windows/win32/api/interactioncontext/ns-interactioncontext-interaction_arguments_manipulation)<br/> | Définit l’état d’une manipulation.<br/>                                                                                                                             |
| [**\_robinet des arguments d’interaction \_**](/windows/win32/api/interactioncontext/ns-interactioncontext-interaction_arguments_tap)<br/>                   | Définit l’état de l’interaction TAP.<br/>                                                                                                                        |
| [**\_configuration du contexte d’interaction \_**](/windows/win32/api/interactioncontext/ns-interactioncontext-interaction_context_configuration)<br/>   | Définit la configuration d’un objet de [contexte d’interaction](interaction-context-portal.md) qui active, désactive ou modifie le comportement d’une interaction.<br/> |
| [**\_sortie du contexte d’interaction \_**](/windows/win32/api/interactioncontext/ns-interactioncontext-interaction_context_output)<br/>                 | Définit la sortie de l’objet de [contexte d’interaction](interaction-context-portal.md) .<br/>                                                                          |
| [**transformation de MANIPULATION \_**](/windows/win32/api/interactioncontext/ns-interactioncontext-manipulation_transform)<br/>                          | Définit les données de transformation pour une manipulation.<br/>                                                                                                              |
| [**vélocité de MANIPULATION \_**](/windows/win32/api/interactioncontext/ns-interactioncontext-manipulation_velocity)<br/>                            | Définit les données de vélocité d’une manipulation.<br/>                                                                                                                     |
