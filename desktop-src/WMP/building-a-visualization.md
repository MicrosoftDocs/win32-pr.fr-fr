# <a name="building-a-visualization"></a>Génération d’une visualisation

Vous pouvez créer une visualisation personnalisée à l’aide de Visual Studio et de l’Assistant de plug-in du lecteur Windows Media. Pour plus d’informations sur la configuration de l’Assistant et la création d’un plug-in, consultez [création d’un plug-](building-a-plug-in.md)in.

Lorsque vous exécutez l’Assistant, vous pouvez choisir plusieurs types de plug-ins différents. Choisissez visualisation.

Une fois que vous avez terminé l’Assistant et créé votre plug-in, démarrez une chanson dans le lecteur Windows Media, puis cliquez sur les visualisations. Les présélections générées automatiquement pour vous sont appelées barres *ProjectName* et *NomProjet* Wave, où *nom_projet* est le nom de votre projet.
