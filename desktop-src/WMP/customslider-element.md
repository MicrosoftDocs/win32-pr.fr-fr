# <a name="customslider-element"></a>Élément CUSTOMSLIDER

L’élément **CUSTOMSLIDER** fournit un moyen de créer et de manipuler un contrôle Slider d’une forme quelconque, tel qu’un bouton circulaire. Les tableaux suivants répertorient les attributs et les gestionnaires d’événements pris en charge par cet élément.

L’élément **CUSTOMSLIDER** prend en charge les attributs suivants.



| Attribut                                               | Description                                                                                                                 |
|---------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------|
| [cursor](customslider-cursor.md)                       | Spécifie ou récupère la valeur du curseur de contrôle de curseur qui apparaît lorsque la souris se trouve sur le curseur.               |
| [disabledImage](customslider-disabledimage.md)         | Spécifie ou récupère l’image du curseur utilisé lorsque le curseur est désactivé.                                            |
| [downImage](customslider-downimage.md)                 | Spécifie ou récupère l’image de l’état inactif du curseur personnalisé.                                                           |
| [hoverImage](customslider-hoverimage.md)               | Spécifie ou récupère l’image qui s’affiche lorsque la souris se trouve sur le curseur personnalisé.                                    |
| [image](customslider-image.md)                         | Spécifie ou récupère le nom du fichier contenant les images correspondant aux différents États du curseur personnalisé. |
| [max](customslider-max.md)                             | Spécifie ou récupère la valeur maximale de la plage définie par le curseur personnalisé.                                         |
| [min](customslider-min.md)                             | Spécifie ou récupère la valeur minimale de la plage définie par le curseur personnalisé.                                         |
| [positionImage](customslider-positionimage.md)         | Spécifie ou récupère l’image interactive utilisée pour déterminer l’image de position à partir du fichier **image** à afficher.             |
| [Bulle](customslider-tooltip.md)                     | Spécifie ou récupère le texte d’info-bulle pour le curseur.                                                                     |
| [Transparente](customslider-transparencycolor.md) | Spécifie ou récupère la couleur de transparence des images de curseur personnalisées.                                                  |
| [value](customslider-value.md)                         | Spécifie ou récupère la position actuelle du curseur.                                                                  |



 

L’élément **CUSTOMSLIDER** peut implémenter les gestionnaires d’événements suivants.



| Gestionnaire d'événements                                         | Description                                                                                                          |
|-------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| [onDragBegin](customslider-ondragbegin.md)           | Gère un événement qui se produit lorsque l’utilisateur clique et maintient le bouton gauche de la souris enfoncé et commence à faire glisser la souris. |
| [onDragEnd](customslider-ondragend.md)               | Gère un événement qui se produit lorsque le bouton gauche de la souris est relâché après une opération de glissement.                      |
| [onPositionChange](customslider-onpositionchange.md) | Gère un événement qui se produit lorsque la position du curseur change suite à un clic ou un glissement de l’utilisateur.   |



 

L’élément **CUSTOMSLIDER** prend en charge les attributs ambiants et peut implémenter les gestionnaires d’événements ambiants. Pour plus d’informations, consultez [attributs ambiants](ambient-attributes.md) et [gestionnaires d’événements ambiants](ambient-event-handlers.md).
