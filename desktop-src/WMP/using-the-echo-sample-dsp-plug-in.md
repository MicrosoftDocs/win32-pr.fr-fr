# <a name="using-the-echo-sample-dsp-plug-in"></a>Utilisation du plug-in Echo Sample DSP

lorsque vous créez une build sans erreur de votre plug-in Lecteur Windows Media d’écho DSP, vous pouvez l’utiliser dans Lecteur Windows Media pour écouter l’effet. lorsque vous démarrez Lecteur Windows Media, le plug-in doit être activé par défaut. Si vous lisez du contenu audio ou vidéo avec de l’audio, vous devez entendre un écho d’une seconde dans le son qui est lu sur le même volume que le son d’origine.

Vous pouvez faire des essais avec des temps de retard et des niveaux d’effets différents en modifiant les valeurs dans la page de propriétés du plug-in Echo. Essayez l’effet sur différents types de matériel source, tels que la musique ou la parole. Un délai d’attente de 200 ms avec un niveau d’effets de 30 entraînera un écho fortement espacé qui ressemble à l’effet utilisé sur les premiers enregistrements de rock et de Roll. Les niveaux d’effets supérieurs à 50 entraînent une diminution du volume du signal d’origine par rapport à l’écho, appelé effet pré-Echo.
