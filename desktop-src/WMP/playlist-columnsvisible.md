# <a name="playlistcolumnsvisible"></a>PLAYLIST. columnsVisible

L’attribut **columnsVisible** spécifie ou récupère une valeur indiquant si les en-têtes de colonnes sont affichés.

``` syntax
        elementID.columnsVisible
```

## <a name="possible-values"></a>Valeurs possibles

Cet attribut est une **valeur booléenne** en lecture/écriture.



| Valeur | Description                        |
|-------|------------------------------------|
| true  | Par défaut. Les en-têtes de colonnes sont affichés. |
| false | Les en-têtes de colonne ne sont pas affichés.      |



 

## <a name="requirements"></a>Spécifications



| Condition requise | Valeur |
|--------------------|------------------------------------------------------|
| Version<br/> | Lecteur Windows Media version 7,0 ou ultérieure<br/> |
