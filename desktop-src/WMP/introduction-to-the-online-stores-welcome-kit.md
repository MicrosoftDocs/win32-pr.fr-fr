# <a name="introduction-to-the-online-stores-welcome-kit"></a>Présentation du kit de bienvenue des magasins en ligne

le Kit de bienvenue des magasins en ligne décrit le processus de mise en ligne d’un magasin de supports numériques en ligne pour Lecteur Windows Media. Lecteur Windows Media prend en charge deux types de magasins en ligne : type 1 et type 2. pour plus d’informations sur la différence entre les magasins de type 1 et de type 2, consultez [Lecteur Windows Media magasins en ligne](windows-media-player-online-stores.md).

Le processus d’intégration décrit dans les sections suivantes s’applique aux magasins de type 1 et de type 2, à l’exception de quelques différences, qui sont indiquées.

## <a name="eligibility"></a>Éligibilité

pour pouvoir bénéficier d’un magasin de musique en ligne intégré à Lecteur Windows Media, votre entreprise doit respecter les exigences d’intégration et les exigences de maintenance indiquées dans le contrat de magasins en ligne. Les types de spécifications dans l’accord incluent, sans s’y limiter, les éléments suivants.

-   votre entreprise doit disposer des droits nécessaires pour distribuer du contenu en ligne via Lecteur Windows Media.
-   votre entreprise doit accepter d’effectuer un certain nombre de formations à la disposition des utilisateurs de Lecteur Windows Media.
-   Votre entreprise doit fournir un certain niveau de service ininterrompu.

certaines exigences spécifiques de l’accord sont mises en évidence dans la [configuration requise pour les magasins de Musique en ligne](requirements-for-online-music-stores.md).

## <a name="contacting-microsoft"></a>Contacter Microsoft

si vous avez des questions concernant les magasins en ligne et le processus d’intégration, contactez l’équipe virtuelle des Services Lecteur Windows Media à l’adresse mpsvctm@microsoft.com .
