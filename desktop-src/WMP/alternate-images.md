# <a name="alternate-images"></a>Images de remplacement

Les images de remplacement sont des images qui ne sont pas visibles lors de l’affichage initial de l’apparence, mais qui deviennent visibles en réponse à des événements de souris. Par exemple, vous pouvez configurer l’apparence de sorte que lorsque la souris pointe sur une zone spécifique, l’illustration de cette zone change. Cela peut informer visuellement les utilisateurs qu’ils peuvent cliquer sur une zone pour effectuer une opération. vous pouvez également utiliser une autre image lorsque l’utilisateur clique sur une zone pour lui fournir un retour visuel indiquant que le clic a été reçu par Lecteur Windows Media.
