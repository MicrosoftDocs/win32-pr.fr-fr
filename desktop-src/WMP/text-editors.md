# <a name="text-editors"></a>Éditeurs de texte

Vous devez utiliser un éditeur de texte qui n’inclut pas de caractères de mise en forme. Un éditeur de texte simple, tel que le bloc-notes, est recommandé, bien que les environnements de développement tels que Microsoft Visual Studio offrent des fonctionnalités avancées. Vous ne devez pas utiliser un éditeur tel que Microsoft Word, car le format de fichier Word par défaut enregistre des codes de mise en forme supplémentaires qui ne sont pas utilisables. Si vous utilisez un traitement de texte tel que Word, vous devez penser à enregistrer votre travail sous forme de texte non mis en forme chaque fois que vous enregistrez votre fichier.

Vous devez probablement désactiver le retour automatique à la ligne si votre éditeur de texte le permet, et vous pouvez activer la possibilité d’afficher tous les caractères pour vous assurer que vous n’ajoutez pas d’onglets ou d’autres caractères de contrôle. N’oubliez pas que l’inclusion des caractères de tabulation dans votre fichier de définition d’apparence entraîne l’échec de l’apparence.
