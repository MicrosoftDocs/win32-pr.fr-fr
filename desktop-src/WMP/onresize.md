# <a name="onresize"></a>OnResize

Le gestionnaire d’événements **OnResize** gère un événement qui se produit lorsqu’un contrôle est redimensionné.

``` syntax
onresize
```

## <a name="requirements"></a>Configuration requise



| Condition requise | Valeur |
|--------------------|-----------------------------------------------------|
| Version<br/> | Lecteur Windows Media version 70 ou ultérieure<br/> |
