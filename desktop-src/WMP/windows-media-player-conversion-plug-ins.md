# <a name="windows-media-player-conversion-plug-ins"></a>Lecteur Windows Media Plug-ins de conversion

Microsoft Lecteur Windows Media fournit une architecture pour les programmes de plug-in qui convertissent les formats de fichiers multimédias numériques.

La documentation du plug-in de conversion est composée des sections suivantes.



| Section                                                                                    | Description                                                                                                                                           |
|--------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------|
| [À propos des plug-ins de conversion](about-conversion-plug-ins.md)                                 | Fournit une vue d’ensemble de l’architecture utilisée pour les plug-ins de conversion. Lisez cette section pour découvrir les concepts généraux associés à cette technologie. |
| [Référence de programmation de plug-ins de conversion](conversion-plug-ins-programming-reference.md) | fournit une référence détaillée pour l’interface COM prise en charge par le kit de développement logiciel (SDK) Lecteur Windows Media pour les plug-ins de conversion.                                |
