# <a name="custom-graphics-in-windows-media-player"></a>graphiques personnalisés dans Lecteur Windows Media

avec Windows script de métafichier multimédia, vous pouvez ajouter des éléments de graphique à Lecteur Windows Media pendant la lecture du flux. Vous pouvez ajouter un type de graphique de base.



| Type   | Description                                                                                                                                                                                                                        |
|--------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| ENCART | à l’aide de l’élément **bannière** , vous pouvez placer une image de bannière (32 pixels de haut en 194 pixels de large) sur Lecteur Windows Media en bas de la zone d’affichage vidéo. Vous pouvez ensuite utiliser un élément enfant pour ajouter un lien hypertexte à la bannière. |
