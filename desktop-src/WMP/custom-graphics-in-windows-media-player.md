# <a name="custom-graphics-in-windows-media-player"></a>Graphiques personnalisés dans le lecteur Windows Media

Avec les scripts Windows Media Metafile, vous pouvez ajouter des éléments graphiques au lecteur Windows Media au fur et à mesure que le flux est lu. Vous pouvez ajouter un type de graphique de base.



| Type   | Description                                                                                                                                                                                                                        |
|--------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| ENCART | À l’aide de l’élément **bannière** , vous pouvez placer une image de bannière (32 pixels de haut en 194 pixels de large) sur le lecteur Windows Media, en bas de la zone d’affichage vidéo. Vous pouvez ensuite utiliser un élément enfant pour ajouter un lien hypertexte à la bannière. |
