# <a name="effectseffectcount"></a>EFFECTs. effectCount

L’attribut **effectCount** récupère le nombre de visualisations disponibles.

``` syntax
        elementID.effectCount
```

## <a name="possible-values"></a>Valeurs possibles

Cet attribut est un **nombre** en lecture seule (**long**).

## <a name="requirements"></a>Configuration requise



| Condition requise | Valeur |
|--------------------|---------------------------------------------------------|
| Version<br/> | Lecteur Windows Media pour Windows XP ou version ultérieure<br/> |
