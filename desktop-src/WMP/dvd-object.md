# <a name="dvd-object"></a>Objet DVD

L’objet **DVD** fournit des propriétés et des méthodes pour travailler avec des DVD.

L’objet **DVD** prend en charge les propriétés suivantes.



| Propriété                           | Description                                                                                        |
|------------------------------------|----------------------------------------------------------------------------------------------------|
| [Domain](dvd-domain.md)           | Récupère le domaine actuel du DVD.                                                           |
| [isAvailable](dvd-isavailable.md) | Récupère une valeur indiquant si un type d’informations spécifié est disponible ou si une action donnée peut être exécutée. |



 

L’objet **DVD** prend en charge les méthodes suivantes.



| Méthode                         | Description                                                                                          |
|--------------------------------|------------------------------------------------------------------------------------------------------|
| [Précédent](dvd-back.md)           | Modifie l’affichage d’un sous-menu vers son menu parent.                                               |
| [sort](dvd-resume.md)       | Passe en mode lecture à partir du mode menu, en reprenant à la même position que lorsque le menu a été appelé. |
| [titleMenu](dvd-titlemenu.md) | Arrête la lecture et affiche le menu du titre.                                                          |
| [Menu de la](dvd-topmenu.md)     | Arrête la lecture et affiche le menu racine.                                                           |



 

L’objet **DVD** est accessible par le biais de la propriété suivante.



| Object                      | Propriété              |
|-----------------------------|-----------------------|
| [Lecteur](player-object.md) | [DVD](player-dvd.md) |
