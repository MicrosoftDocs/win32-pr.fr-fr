# <a name="marquee-color"></a>Couleur du texte défilant

La valeur de couleur du type doit être spécifiée. La valeur doit être spécifiée à l’aide de trois entiers positifs séparés par des virgules. Les valeurs représentent la quantité de rouge, de vert et de bleu pour une image bitmap, et les plages comprises entre 0 et 255.
