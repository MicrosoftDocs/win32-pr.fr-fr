# <a name="eventaltkey"></a>Event. altKey

L’attribut **altKey** récupère une valeur indiquant si la touche Alt était enfoncée lorsque l’événement s’est produit.

``` syntax
event.altKey
```

## <a name="possible-values"></a>Valeurs possibles

Cet attribut est une **valeur booléenne** en lecture seule.



| Valeur | Description                                     |
|-------|-------------------------------------------------|
| true  | Indique que la touche ALT était en position inférieure. |
| false | Indique que la touche ALT était en haut.   |



 

## <a name="requirements"></a>Configuration requise



| Condition requise | Valeur |
|--------------------|------------------------------------------------------|
| Version<br/> | Lecteur Windows Media version 7,0 ou ultérieure<br/> |
