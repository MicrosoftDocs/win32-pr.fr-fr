# <a name="iwmperroritem-vb-and-c-interface"></a>Interface IWMPErrorItem (VB et C#)

Fournit l’accès aux informations d’erreur.

L’interface **IWMPErrorItem** expose les propriétés suivantes.



| Propriété                                                                             | Description                                                                                |
|--------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------|
| [customUrl](wmplibiwmperroritem-iwmperroritem-customurl--vb-and-c.md)               | Obtient l’URL d’un site Web qui affiche des informations spécifiques sur l’échec du téléchargement du codec. |
| [errorCode](wmplibiwmperroritem-iwmperroritem-errorcode--vb-and-c.md)               | Obtient le code d’erreur actuel.                                                               |
| [errorContext](wmplibiwmperroritem-iwmperroritem-errorcontext--vb-and-c.md)         | Obtient une valeur indiquant le contexte de l’erreur.                                          |
| [errorDescription](wmplibiwmperroritem-iwmperroritem-errordescription--vb-and-c.md) | Obtient une description de l'erreur.                                                           |
| [corriger](wmplibiwmperroritem-iwmperroritem-remedy--vb-and-c.md)                     | Réservé pour un usage futur.                                                                   |



 

Procurez-vous une interface **IWMPErrorItem** à l’aide de la propriété suivante.



| Interface                            | Propriété                                                                   |
|--------------------------------------|----------------------------------------------------------------------------|
| [IWMPError](iwmperror--vb-and-c.md) | [Item](iwmperror-item--vb-and-c.md) (en C#, utilisez la méthode d' **extraction d' \_ élément** ) |



 

## <a name="members"></a>Membres

L’interface **IWMPErrorItem (VB et C#)** ne définit aucun membre.
