# <a name="playlistarray-object"></a>Objet PlaylistArray

L’objet **PlaylistArray** offre un moyen d’accéder aux sélections par numéro d’index.

L’objet **PlaylistArray** prend en charge la propriété suivante.



| Propriété                         | Description                                              |
|----------------------------------|----------------------------------------------------------|
| [count](playlistarray-count.md) | Récupère le nombre de sélections dans le tableau de sélection. |



 

L’objet **PlaylistArray** prend en charge la méthode suivante.



| Méthode                         | Description                                                              |
|--------------------------------|--------------------------------------------------------------------------|
| [item](playlistarray-item.md) | Récupère l’objet de [sélection](playlist-object.md) au niveau de l’index donné. |



 

L’objet **PlaylistArray** est accessible par le biais des méthodes suivantes.



| Object                                              | Méthode                                                                                 |
|-----------------------------------------------------|----------------------------------------------------------------------------------------|
| [PlaylistCollection](playlistcollection-object.md) | [GetAll](playlistcollection-getall.md), [GetByName](playlistcollection-getbyname.md) |



 

À des fins d’illustration, *joueur*. *playlistCollection*. **GetAll** est utilisé dans les sections de syntaxe de référence.
