# <a name="playerapplicationswitchtocontrol-method"></a>Méthode PlayerApplication. switchToControl

La méthode **switchToControl** bascule un contrôle du lecteur Windows Media distant vers l’état d’ancrage.

## <a name="syntax"></a>Syntaxe


```JScript
PlayerApplication.switchToControl()
```



## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.

## <a name="return-value"></a>Valeur retournée

Cette méthode ne retourne pas de valeur.

## <a name="remarks"></a>Notes

Cette méthode est utilisée uniquement lors de la communication à distance du contrôle du lecteur Windows Media.

**Lecteur Windows Media 10 Mobile :** Cette méthode n’est pas prise en charge.
