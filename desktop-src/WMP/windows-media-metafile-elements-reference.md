# <a name="windows-media-metafile-elements-reference"></a>Windows Informations de référence sur les éléments de métafichier multimédia

cette section contient la documentation de référence pour tous les éléments de métafichier de média Windows pour les applications côté client. Les entrées de référence incluent les définitions des éléments, leurs attributs et leurs valeurs, ainsi que les conditions spéciales associées à chaque élément.

Les éléments de métafichier suivants sont pris en charge pour les applications côté client.



| Élément                                           | Description                                                                                                            |
|---------------------------------------------------|------------------------------------------------------------------------------------------------------------------------|
| [ABSTRACTION](abstract-element.md)                  | Contient le texte qui décrit l’élément **ASX**, **Banner** ou **entry** associé.                                 |
| [ASX](asx-element.md)                            | définit un fichier en tant que métafichier Windows Media.                                                                            |
| [AUTEUR](author-element.md)                      | contient le nom de l’auteur d’un clip multimédia ou d’un métafichier Windows media.                                           |
| [ENCART](banner-element.md)                      | spécifie une URL pour un graphique qui s’affiche dans le panneau d’affichage de Lecteur Windows Media.                               |
| [BASE](base-element.md)                          | spécifie une chaîne ajoutée au début des url envoyées à Lecteur Windows Media.                                 |
| [Commentaires](comments.md)                          | Spécifie la syntaxe XML pour les commentaires ( <!--...--> ).                                                            |
| [INTELLECTUELLE](copyright-element.md)                | Contient les informations de copyright pour un élément **ASX** ou **entry** .                                                |
| [DURATION](duration-element.md)                  | spécifie la durée pendant laquelle Lecteur Windows Media restitue un flux.                                                    |
| [ENDMARKER](endmarker-element.md)                | spécifie un marqueur auquel Lecteur Windows Media cesse de restituer le flux.                                           |
| [MENTION](entry-element.md)                        | Spécifie le chemin d’accès d’un clip multimédia.                                                                                   |
| [ENTRYREF](entryref-element.md)                  | liens vers les éléments d' **entrée** dans un métafichier externe Windows Media qui a une extension. asx,. wax,. wvx ou. wmx. |
| [ÉVÉNEMENT](event-element.md)                        | définit un comportement ou une action entreprise par Lecteur Windows Media lorsqu’il reçoit une commande de script appelée comme un événement.      |
| [LOGURL](logurl-element.md)                      | ordonne à Lecteur Windows Media d’envoyer des données de journal à l’URL spécifiée.                                            |
| [MOREINFO](moreinfo-element.md)                  | Spécifie l’URL d’un site Web, d’une adresse de messagerie ou d’une commande de script associée à un affichage, un clip ou une bannière.               |
| [ENVOYÉS](param-element.md)                        | Spécifie la valeur d’un paramètre personnalisé associé à une sélection ou à un élément d’une sélection.                      |
| [PREVIEWDURATION](previewduration-element.md)    | Spécifie la durée pendant laquelle un clip est lu en mode aperçu.                                                         |
| [Réf](ref-element.md)                            | Spécifie une URL pour un élément de contenu multimédia numérique.                                                                  |
| [Tab](repeat-element.md)                      | spécifie le nombre de fois Lecteur Windows Media répète un ou plusieurs éléments d' **entrée** ou de **ENTRYREF** .             |
| [APPARENCE (déconseillée)](skin-element--deprecated.md) | Spécifie une URL vers une apparence incorporée.                                                                                   |
| [STARTMARKER](startmarker-element.md)            | spécifie un marqueur auquel Lecteur Windows Media démarre le rendu du flux.                                          |
| [HEURE](starttime-element.md)                | spécifie l’heure à laquelle Lecteur Windows Media démarrera le rendu du flux.                                        |
| [BONHOMME](title-element--metafile.md)              | Contient le titre d’un élément **ASX** ou d’un élément d' **entrée** .                                                                 |
