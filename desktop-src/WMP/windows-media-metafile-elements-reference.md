# <a name="windows-media-metafile-elements-reference"></a>Informations de référence sur les éléments de métafichier Windows Media

Cette section contient la documentation de référence pour tous les éléments de métafichier Windows Media pour les applications côté client. Les entrées de référence incluent les définitions des éléments, leurs attributs et leurs valeurs, ainsi que les conditions spéciales associées à chaque élément.

Les éléments de métafichier suivants sont pris en charge pour les applications côté client.



| Élément                                           | Description                                                                                                            |
|---------------------------------------------------|------------------------------------------------------------------------------------------------------------------------|
| [ABSTRACTION](abstract-element.md)                  | Contient le texte qui décrit l’élément **ASX**, **Banner** ou **entry** associé.                                 |
| [ASX](asx-element.md)                            | Définit un fichier en tant que métafichier Windows Media.                                                                            |
| [AUTEUR](author-element.md)                      | Contient le nom de l’auteur d’un clip multimédia ou d’un métafichier Windows Media.                                           |
| [ENCART](banner-element.md)                      | Spécifie l’URL d’un graphique qui s’affiche dans le panneau d’affichage du lecteur Windows Media.                               |
| [BASE](base-element.md)                          | Spécifie une chaîne qui est ajoutée au début des URL envoyées au lecteur Windows Media.                                 |
| [Commentaires](comments.md)                          | Spécifie la syntaxe XML pour les commentaires ( <!--...--> ).                                                            |
| [INTELLECTUELLE](copyright-element.md)                | Contient les informations de copyright pour un élément **ASX** ou **entry** .                                                |
| [DURATION](duration-element.md)                  | Spécifie la durée pendant laquelle le lecteur Windows Media effectue le rendu d’un flux.                                                    |
| [ENDMARKER](endmarker-element.md)                | Spécifie un marqueur auquel le lecteur Windows Media cesse de restituer le flux.                                           |
| [MENTION](entry-element.md)                        | Spécifie le chemin d’accès d’un clip multimédia.                                                                                   |
| [ENTRYREF](entryref-element.md)                  | Liens vers les éléments d' **entrée** dans un métafichier Windows Media externe qui a une extension. asx,. Wax,. wvx ou. WMX. |
| [ÉVÉNEMENT](event-element.md)                        | Définit un comportement ou une action entreprise par le lecteur Windows Media lorsqu’il reçoit une commande de script appelée comme un événement.      |
| [LOGURL](logurl-element.md)                      | Indique au lecteur Windows Media d’envoyer toutes les données de journal à l’URL spécifiée.                                            |
| [MOREINFO](moreinfo-element.md)                  | Spécifie l’URL d’un site Web, d’une adresse de messagerie ou d’une commande de script associée à un affichage, un clip ou une bannière.               |
| [ENVOYÉS](param-element.md)                        | Spécifie la valeur d’un paramètre personnalisé associé à une sélection ou à un élément d’une sélection.                      |
| [PREVIEWDURATION](previewduration-element.md)    | Spécifie la durée pendant laquelle un clip est lu en mode aperçu.                                                         |
| [Réf](ref-element.md)                            | Spécifie une URL pour un élément de contenu multimédia numérique.                                                                  |
| [Tab](repeat-element.md)                      | Spécifie le nombre de fois que le lecteur Windows Media a répété un ou plusieurs éléments d' **entrée** ou de **ENTRYREF** .             |
| [APPARENCE (déconseillée)](skin-element--deprecated.md) | Spécifie une URL vers une apparence incorporée.                                                                                   |
| [STARTMARKER](startmarker-element.md)            | Spécifie un marqueur auquel le lecteur Windows Media démarre le rendu du flux.                                          |
| [HEURE](starttime-element.md)                | Spécifie l’heure à laquelle le lecteur Windows Media va démarrer le rendu du flux.                                        |
| [BONHOMME](title-element--metafile.md)              | Contient le titre d’un élément **ASX** ou d’un élément d' **entrée** .                                                                 |
