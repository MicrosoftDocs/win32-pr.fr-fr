# <a name="playermediacollectionchange-event"></a>Événement Player. MediaCollectionChange

L’événement **MediaCollectionChange** se produit lorsque la collection de supports change.

## <a name="syntax"></a>Syntaxe


```JScript
Player.MediaCollectionChange()
```



## <a name="parameters"></a>Paramètres

Cet événement n’a pas de paramètres.

## <a name="return-value"></a>Valeur retournée

Cet événement ne retourne pas de valeur.

## <a name="remarks"></a>Notes

**Lecteur Windows Media 10 Mobile :** Cet événement n’est pas pris en charge.
