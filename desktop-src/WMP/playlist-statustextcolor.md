# <a name="playliststatustextcolor"></a>PLAYLIST. statusTextColor

L’attribut **statusTextColor** spécifie ou récupère une valeur indiquant la couleur du texte d’État.

``` syntax
        elementID.statusTextColor
```

## <a name="possible-values"></a>Valeurs possibles

Cet attribut est une **chaîne** en lecture/écriture représentant une couleur. La valeur par défaut est « None ».

## <a name="requirements"></a>Configuration requise



| Condition requise | Valeur |
|--------------------|------------------------------------|
| Version<br/> | Lecteur Windows Media 10<br/> |
