# <a name="equalizersettingspresetcount"></a>EQUALIZERSETTINGS.presetCount

L’attribut **presetCount** récupère le nombre de présélections disponibles.

``` syntax
        elementID.presetCount
```

## <a name="possible-values"></a>Valeurs possibles

Cet attribut est un **nombre** en lecture seule (**long**).

## <a name="requirements"></a>Configuration requise



| Condition requise | Valeur |
|--------------------|------------------------------------------------------|
| Version<br/> | Lecteur Windows Media version 7,0 ou ultérieure<br/> |
