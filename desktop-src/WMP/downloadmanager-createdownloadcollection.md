# <a name="downloadmanagercreatedownloadcollection-method"></a>Méthode DownloadManager. createDownloadCollection

> [!Note]  
> Cette section décrit les fonctionnalités conçues pour être utilisées par les magasins en ligne. L’utilisation de cette fonctionnalité en dehors du contexte d’un magasin en ligne n’est pas prise en charge.

 

La méthode **createDownloadCollection** crée une collection de téléchargements.

## <a name="syntax"></a>Syntaxe


```JScript
retVal = DownloadManager.createDownloadCollection()
```



## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.

## <a name="return-value"></a>Valeur retournée

Cette méthode retourne un objet **DownloadCollection** .
