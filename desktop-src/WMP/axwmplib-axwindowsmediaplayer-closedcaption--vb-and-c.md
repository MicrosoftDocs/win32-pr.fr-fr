# <a name="axwindowsmediaplayerclosedcaption-property"></a>AxWindowsMediaPlayer. closedCaption, propriété

La propriété closedCaption obtient une interface **IWMPClosedCaption** pour l’élément multimédia actuel.

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```CSharp
public IWMPClosedCaption closedCaption {get;}
```


```VB

Public ReadOnly Property closedCaption As IWMPClosedCaption
```





## <a name="property-value"></a>Valeur de la propriété

WMPLIB. Interface IWMPClosedCaption.
