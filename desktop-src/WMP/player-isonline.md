# <a name="playerisonline"></a>Player. isOnline

La propriété **isOnline** récupère une valeur indiquant si l’utilisateur est connecté à un réseau.

## <a name="syntax"></a>Syntaxe

*lecteur* . **isOnline**

## <a name="possible-values"></a>Valeurs possibles

Cette propriété est une **valeur booléenne** en lecture seule.



| Valeur | Description                             |
|-------|-----------------------------------------|
| true  | L’utilisateur est connecté à un réseau.     |
| false | L’utilisateur n’est pas connecté à un réseau. |



 

## <a name="remarks"></a>Remarques

**Lecteur Windows Media 10 Mobile :** Cette propriété retourne toujours true.
