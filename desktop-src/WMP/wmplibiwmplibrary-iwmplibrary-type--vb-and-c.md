# <a name="iwmplibrarytype-property"></a>IWMPLibrary :: type, propriété

La propriété de **type** obtient une valeur qui indique le type de bibliothèque.

## <a name="syntax"></a>Syntaxe


```CSharp
public WMPLibraryType type {get; set;}
```


```VB

Public ReadOnly Property type As WMPLibraryType
```





## <a name="property-value"></a>Valeur de la propriété

**Wmplib. WMPLibraryType** qui est une valeur de l’énumération **WMPLibraryType** qui indique le type de bibliothèque.
