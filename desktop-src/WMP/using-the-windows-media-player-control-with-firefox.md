# <a name="using-the-windows-media-player-control-with-firefox"></a>Utilisation du contrôle du lecteur Windows Media avec Firefox

Microsoft fournit un plug-in qui vous permet d’incorporer le contrôle ActiveX du lecteur Windows Media dans une page Web afin qu’il s’affiche correctement par un navigateur Firefox. Le plug-in, qui doit être installé sur l’ordinateur client, peut être téléchargé à partir du [site Web Microsoft Port25](https://www.mozilla.org/).

Pour plus d’informations sur le navigateur Firefox, consultez le [site Web Firefox](https://www.mozilla.org/).

Pour plus d’informations sur l’utilisation du lecteur Windows Media avec Firefox, consultez la section Lecteur Windows Media du [site Web mozillaZine](http://kb.mozillazine.org/Windows_Media_Player).

Les rubriques suivantes décrivent comment incorporer le contrôle du lecteur Windows Media dans une page Web qui peut être affichée correctement à la fois par Internet Explorer et Firefox.
