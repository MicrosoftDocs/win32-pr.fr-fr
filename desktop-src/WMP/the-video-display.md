# <a name="the-video-display"></a>Affichage vidéo

Vous pouvez choisir d’inclure une zone d’affichage vidéo dans votre apparence. La vidéo est affichée dans une zone rectangulaire, qui est spécifiée dans la section [vidéo](video.md) du fichier de définition d’apparence.
