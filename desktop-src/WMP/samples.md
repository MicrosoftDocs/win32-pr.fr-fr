# <a name="wmp-sdk-samples"></a>Exemples du kit de développement logiciel WMP

le kit de développement logiciel (SDK) Lecteur Windows Media comprend un large éventail d’exemples qui illustrent la plupart des techniques de programmation décrites dans ce document.

-   les exemples de script montrent comment incorporer le contrôle de ActiveX Player dans une page web. un exemple de page web complet montre comment détecter la version actuelle du lecteur ActiveX contrôle.
-   un exemple skins montre comment modifier l’apparence visuelle et le comportement de Lecteur Windows Media.
-   un assistant pour Microsoft Visual C++ peut créer un exemple fonctionnel unique de plusieurs types de plug-in : un plug-in de visualisation, un plug-in d’interface utilisateur, un plug-in de rendu et un plug-in de traitement de signal numérique.
-   les exemples de magasins en ligne incluent un assistant pour Microsoft Visual C++ qui peut créer un exemple fonctionnel unique d’un objet COM de magasin en ligne.
-   les exemples de Microsoft Visual C++ illustrent l’incorporation de l’Lecteur Windows Media contrôle ActiveX, la communication à distance du contrôle ActiveX Player et l’utilisation de la bibliothèque.
-   un exemple C# montre comment utiliser le contrôle Lecteur Windows Media ActiveX avec le .NET Framework.
-   un exemple de Visual Basic .net montre l’utilisation de l’Lecteur Windows Media ActiveX contrôle avec le .NET Framework.

pour utiliser les exemples, vous devez installer le SDK Windows, qui comprend le kit de développement logiciel (SDK) Lecteur Windows Media. pour télécharger le SDK Windows, accédez à la page [SDK Windows](https://msdn.microsoft.com/windows/bb980924.aspx) dans le centre de développement Windows.
