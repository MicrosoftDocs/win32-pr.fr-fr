# <a name="wmp-sdk-samples"></a>Exemples du kit de développement logiciel WMP

Le kit de développement logiciel (SDK) Windows Media Player comprend un large éventail d’exemples qui illustrent la plupart des techniques de programmation décrites dans ce document.

-   Les exemples de script montrent comment incorporer le contrôle ActiveX Player dans une page Web. Un exemple de page Web complet montre comment détecter la version actuelle du contrôle ActiveX Player.
-   Un exemple Skins montre comment modifier l’apparence visuelle et le comportement du lecteur Windows Media.
-   Un Assistant pour Microsoft Visual C++ peut créer un exemple fonctionnel unique de plusieurs types de plug-in : un plug-in de visualisation, un plug-in d’interface utilisateur, un plug-in de rendu et un plug-in de traitement de signal numérique.
-   Les exemples de magasins en ligne incluent un Assistant pour Microsoft Visual C++ qui peut créer un exemple fonctionnel unique d’un objet COM de magasin en ligne.
-   Les exemples de Microsoft Visual C++ montrent comment incorporer le contrôle ActiveX du lecteur Windows Media, communiquer à distance le contrôle ActiveX du lecteur et travailler avec la bibliothèque.
-   Un exemple C# montre comment utiliser le contrôle Windows Media Player ActiveX avec l' .NET Framework.
-   Un exemple .NET Visual Basic montre comment utiliser le contrôle Windows Media Player ActiveX avec l' .NET Framework.

Pour utiliser les exemples, vous devez installer le SDK Windows, qui comprend le kit de développement logiciel (SDK) du lecteur Windows Media. Pour télécharger le SDK Windows, accédez à la page [SDK Windows](https://msdn.microsoft.com/windows/bb980924.aspx) dans le centre de développement Windows.
