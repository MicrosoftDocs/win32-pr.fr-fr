# <a name="wmpvideo"></a>WMPVIDEO

Il s’agit d’un élément **Video** prédéfini avec les valeurs par défaut suivantes.

``` syntax
backgroundColor="black"
horizontalAlignment="stretch"
verticalAlignment="stretch"
```

## <a name="remarks"></a>Notes

Cela crée un élément **Video** qui étire la fenêtre vidéo lorsque l’apparence est redimensionnée. La vidéo numérique affichée dans cette fenêtre s’étire pour s’adapter à l’espace disponible ou sera mise à l’échelle en fonction des paramètres du menu **affichage** du lecteur sous taille de la **vidéo**. Le menu **affichage** est affiché en mode complet du lecteur ou lorsque l’attribut **View. TitleBar** a la valeur true dans une apparence.

Toutes les propriétés de cet élément **Video** peuvent être remplacées en les spécifiant explicitement.
