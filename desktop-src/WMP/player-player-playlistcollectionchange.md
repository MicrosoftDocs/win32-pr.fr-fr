# <a name="playerplaylistcollectionchange-event"></a>Événement Player. PlaylistCollectionChange

L’événement **PlaylistCollectionChange** se produit lorsqu’une modification est apportée à la collection de sélections.

## <a name="syntax"></a>Syntaxe


```JScript
Player.PlaylistCollectionChange()
```



## <a name="parameters"></a>Paramètres

Cet événement n’a pas de paramètres.

## <a name="return-value"></a>Valeur retournée

Cet événement ne retourne pas de valeur.

## <a name="remarks"></a>Notes

**Lecteur Windows Media 10 Mobile :** Cet événement n’est pas pris en charge.
