<span data-ttu-id="53fe0-109">Le lecteur Windows Media peut afficher une page de propriétés pour chaque plug-in DSP afin de permettre aux utilisateurs de définir des valeurs qui modifient le comportement du plug-in.</span><span class="sxs-lookup"><span data-stu-id="53fe0-109">Windows Media Player can display a property page for each DSP plug-in to enable users to set values that change the behavior of the plug-in.</span></span> <span data-ttu-id="53fe0-110">Les utilisateurs peuvent accéder à la page de propriétés à partir de l’onglet **plug-ins** de la boîte de dialogue Options en cliquant sur le nom du plug-in DSP pour le sélectionner, puis en cliquant sur **Propriétés**.</span><span class="sxs-lookup"><span data-stu-id="53fe0-110">Users can access the property page from the **Plug-ins** tab of the Options dialog box by clicking the name of the DSP plug-in to select it and then clicking **Properties**.</span></span>

<span data-ttu-id="53fe0-111">L’exemple de code du plug-in du lecteur Windows Media fournit une implémentation par défaut d’une page de propriétés qui inclut une seule zone d’édition qui reçoit de l’utilisateur une valeur représentant un facteur d’échelle pour le volume audio.</span><span class="sxs-lookup"><span data-stu-id="53fe0-111">The Windows Media Player Plug-in Wizard sample code provides a default implementation of a property page that includes a single edit box that receives from the user a value representing a scale factor for the audio volume.</span></span> <span data-ttu-id="53fe0-112">Quand l’utilisateur clique sur **appliquer**, la page de propriétés transmet cette valeur à l’objet de plug-in afin que le plug-in puisse modifier la valeur qu’il utilise pour mettre à l’échelle le volume au cours du traitement.</span><span class="sxs-lookup"><span data-stu-id="53fe0-112">When the user clicks **Apply**, the property page passes this value to the plug-in object so the plug-in can change the value it uses to scale the volume during processing.</span></span>

lorsque l’utilisateur clique sur **appliquer**.</span><span class="sxs-lookup"><span data-stu-id="53fe0-116">Your code at least should provide override implementations for <span data-ttu-id="53fe0-120">En outre, vous devrez ajouter du code pour gérer l’événement qui se produit lorsque l’utilisateur modifie une valeur dans un contrôle que vous fournissez dans la page de propriétés.</span><span class="sxs-lookup"><span data-stu-id="53fe0-120">Additionally, you will need to add code to handle the event that occurs when the user changes a value in a control you provide in the property page.</span></span> <span data-ttu-id="53fe0-121">Par exemple, l’Assistant de plug-in implémente une fonction nommée **OnChangeScale**, qui active simplement le bouton **appliquer** lorsque l’utilisateur modifie le texte de la zone d’édition sur la page de propriétés en appelant la méthode **SetDirty** avec la valeur d’argument **true**.</span><span class="sxs-lookup"><span data-stu-id="53fe0-121">For example, the plug-in wizard implements a function named **OnChangeScale**, which simply enables the **Apply** button when the user changes the text in the edit box on the property page by calling the **SetDirty** method with an argument value of **TRUE**.</span></span>

## <a name="about-ispecifypropertypages"></a><span data-ttu-id="53fe0-127">À propos de ISpecifyPropertyPages</span><span class="sxs-lookup"><span data-stu-id="53fe0-127">About ISpecifyPropertyPages</span></span>
