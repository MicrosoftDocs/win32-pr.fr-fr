# <a name="onmouseout"></a>onmouseout

Le gestionnaire d’événements **onmouseout** gère un événement qui se produit lorsque l’utilisateur déplace le pointeur hors de l’élément.

``` syntax
onmouseout
```

## <a name="remarks"></a>Notes

Ce gestionnaire d’événements n’est pas applicable à l’élément **playlist** .

## <a name="requirements"></a>Spécifications



| Condition requise | Valeur |
|--------------------|-----------------------------------------------------|
| Version<br/> | Lecteur Windows Media version 70 ou ultérieure<br/> |
