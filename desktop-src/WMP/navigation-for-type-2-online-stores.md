# <a name="navigation-for-type-2-online-stores"></a>Navigation pour les magasins de type 2 en ligne

Les pages Web hébergées par le lecteur Windows Media 10 (ou version ultérieure), telles que celles des volets de tâches du service de magasin en ligne, peuvent utiliser la méthode [External. NavigateTaskPaneURL](external-navigatetaskpaneurl.md) pour contrôler la navigation dans le lecteur Windows Media. La navigation peut se produire des manières suivantes :

-   Entre les volets des tâches de service. Consultez [navigation entre les volets Office de service](navigating-between-service-task-panes.md)
-   À partir de fonctionnalités comme Info Center View, HTMLView Web pages et la page d’informations sur les albums pour les volets des tâches de service. Consultez [navigation à partir des fonctionnalités du lecteur Windows Media](navigating-from-windows-media-player-features.md).
-   Entre les magasins en ligne. Consultez [navigation vers d’autres magasins en ligne](navigating-to-other-online-stores.md).
