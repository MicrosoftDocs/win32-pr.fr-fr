# <a name="iwmpsettingsenableerrordialogs-property"></a>IWMPSettings :: enableErrorDialogs, propriété

La propriété **enableErrorDialogs** obtient ou définit une valeur indiquant si les boîtes de dialogue d’erreur s’affichent automatiquement.

## <a name="syntax"></a>Syntaxe


```CSharp
public System.Boolean enableErrorDialogs {get; set;}
```


```VB

Public Property enableErrorDialogs As System.Boolean
```





## <a name="property-value"></a>Valeur de la propriété

Valeur **System. Boolean** indiquant si les boîtes de dialogue d’erreur s’affichent automatiquement. La valeur par défaut est **true**.
