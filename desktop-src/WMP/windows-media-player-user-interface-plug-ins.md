# <a name="windows-media-player-user-interface-plug-ins"></a>Lecteur Windows Media Plug-ins d’interface utilisateur

Microsoft Lecteur Windows Media fournit une architecture qui permet à l’utilisateur d’installer et d’activer des programmes de plug-in qui ajoutent des fonctionnalités d’interface utilisateur (iu) au mode complet du lecteur. Un plug-in d’interface utilisateur classique peut permettre à un utilisateur d’acheter le CD contenant la piste en cours de diffusion ou d’accéder à des informations supplémentaires sur l’artiste. cette section du kit de développement logiciel (SDK) Lecteur Windows Media vous fournit les informations de programmation dont vous avez besoin pour créer votre propre plug-in d’interface utilisateur.

La documentation du plug-in d’interface utilisateur est divisée en trois sections :



| Section                                                                                            | Description                                                                                                                                   |
|----------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------|
| [À propos des plug-ins d’interface utilisateur](about-user-interface-plug-ins.md)                                 | Fournit une vue d’ensemble de l’architecture utilisée pour les plug-ins d’interface utilisateur. Lisez cette section pour découvrir les concepts généraux associés à cette technologie. |
| [Guide de programmation de plug-ins d’interface utilisateur](user-interface-plug-ins-programming-guide.md)         | Explique ce que vous devez faire pour créer un plug-in d’interface utilisateur. Cette section contient des exemples de code et des procédures pas à pas.                          |
| [Informations de référence sur la programmation de plug-ins d’interface utilisateur](user-interface-plug-ins-programming-reference.md) | fournit une référence détaillée pour l’interface COM et les méthodes prises en charge par le kit de développement logiciel (SDK) Lecteur Windows Media pour les plug-ins d’interface utilisateur.                    |



 

> [!Note]  
> Lecteur Windows Media 10 Mobile fournit le même modèle de plug-in que le Lecteur Windows Media de bureau. Ce modèle de plug-in vous permet d’utiliser des plug-ins d’arrière-plan pour surveiller les événements, afficher l’état des propriétés, etc. le guide de programmation de cette section décrit comment créer un plug-in d’interface utilisateur d’arrière-plan pour Lecteur Windows Media 10 Mobile à l’aide de l’assistant de plug-in du Lecteur Windows Media de bureau.
