# <a name="windows-media-player-user-interface-plug-ins"></a>Plug-ins de l’interface utilisateur du lecteur Windows Media

Le lecteur Microsoft Windows Media fournit une architecture qui permet à l’utilisateur d’installer et d’activer des programmes de plug-in qui ajoutent des fonctionnalités d’interface utilisateur (IU) au mode complet du lecteur. Un plug-in d’interface utilisateur classique peut permettre à un utilisateur d’acheter le CD contenant la piste en cours de diffusion ou d’accéder à des informations supplémentaires sur l’artiste. Cette section du kit de développement logiciel (SDK) du lecteur Windows Media vous fournit les informations de programmation dont vous avez besoin pour créer votre propre plug-in d’interface utilisateur.

La documentation du plug-in d’interface utilisateur est divisée en trois sections :



| Section                                                                                            | Description                                                                                                                                   |
|----------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------|
| [À propos des plug-ins d’interface utilisateur](about-user-interface-plug-ins.md)                                 | Fournit une vue d’ensemble de l’architecture utilisée pour les plug-ins d’interface utilisateur. Lisez cette section pour découvrir les concepts généraux associés à cette technologie. |
| [Guide de programmation de plug-ins d’interface utilisateur](user-interface-plug-ins-programming-guide.md)         | Explique ce que vous devez faire pour créer un plug-in d’interface utilisateur. Cette section contient des exemples de code et des procédures pas à pas.                          |
| [Informations de référence sur la programmation de plug-ins d’interface utilisateur](user-interface-plug-ins-programming-reference.md) | Fournit une référence détaillée pour l’interface COM et les méthodes prises en charge par le kit de développement logiciel (SDK) du lecteur Windows Media pour les plug-ins d’interface utilisateur.                    |



 

> [!Note]  
> Le lecteur Windows Media 10 Mobile fournit le même modèle de plug-in que le lecteur Windows Media du bureau. Ce modèle de plug-in vous permet d’utiliser des plug-ins d’arrière-plan pour surveiller les événements, afficher l’état des propriétés, etc. Le Guide de programmation de cette section décrit comment créer un plug-in d’IU d’arrière-plan pour le lecteur Windows Media 10 mobile à l’aide de l’Assistant de plug-in du lecteur Windows Media.
