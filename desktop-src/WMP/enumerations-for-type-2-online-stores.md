# <a name="enumerations-for-type-2-online-stores"></a>Énumérations pour les magasins de type 2 en ligne

> [!Note]  
> Cette section décrit les fonctionnalités conçues pour être utilisées par les magasins en ligne. L’utilisation de cette fonctionnalité en dehors du contexte d’un magasin en ligne n’est pas prise en charge.

 

Le type d’énumération suivant est disponible pour les magasins en ligne à l’aide de code C++.



| Énumération                                                    | Description                                         |
|----------------------------------------------------------------|-----------------------------------------------------|
| [WMPSubscriptionServiceEvent](/previous-versions/windows/desktop/api/subscriptionservices/ne-subscriptionservices-wmpsubscriptionserviceevent) | Définit les types d’événements de service qui peuvent se produire. |
