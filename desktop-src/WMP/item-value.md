# <a name="itemvalue"></a>ITEM. Value

L’attribut **value** spécifie ou récupère le texte qui s’affiche pour une zone de liste ou un élément contextuel.

``` syntax
        elementID.value
```

## <a name="possible-values"></a>Valeurs possibles

Cet attribut est une **chaîne** en lecture/écriture.

## <a name="requirements"></a>Configuration requise



| Condition requise | Valeur |
|--------------------|---------------------------------------------------------|
| Version<br/> | Lecteur Windows Media pour Windows XP ou version ultérieure<br/> |
