# <a name="equalizersettingssplinetension"></a>EQUALIZERSETTINGS.splineTension

L’attribut **splineTension** spécifie ou récupère la tension spline pour le contrôle égaliseur.

``` syntax
        elementID.splineTension
```

## <a name="possible-values"></a>Valeurs possibles

Cet attribut est un **nombre** en lecture/écriture (**float**) compris entre 0,0 (pas de tension) et 10,0 (pleine tension) avec une valeur par défaut de 3,0.

## <a name="requirements"></a>Configuration requise



| Condition requise | Valeur |
|--------------------|---------------------------------------------------|
| Version<br/> | Lecteur Windows Media série 9 ou version ultérieure<br/> |
