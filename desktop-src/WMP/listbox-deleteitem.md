# <a name="listboxdeleteitem"></a>LISTBOX. deleteItem

La méthode **deleteItem** supprime l’élément de contrôle de zone de liste à l’index spécifié.

``` syntax
        elementID.deleteItem(index)
```

## <a name="parameters"></a>Paramètres

<dl> <dt>

<span id="index"></span><span id="INDEX"></span>*évaluer*
</dt> <dd>

**Nombre** (**long**) contenant l’index de l’élément à supprimer.

</dd> </dl>

## <a name="return-value"></a>Valeur renvoyée

Cette méthode ne retourne pas de valeur.

## <a name="remarks"></a>Notes

Les lignes situées sous la ligne supprimée déplacent une valeur d’index vers le haut.
