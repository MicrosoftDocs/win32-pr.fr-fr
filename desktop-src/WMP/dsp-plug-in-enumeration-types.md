# <a name="dsp-plug-in-enumeration-types"></a>Types d’énumération du plug-in DSP

Le kit de développement logiciel (SDK) du lecteur Windows Media implémente le type d’énumération suivant pour créer des plug-ins DSP.



| Type d’énumération                                        | Description                                                                                    |
|---------------------------------------------------------|------------------------------------------------------------------------------------------------|
| [WMPPlugin \_ Cap](/previous-versions/windows/desktop/api/wmpservices/ne-wmpservices-wmpplugin_caps)                   | Utilisé avec **IWMPPlugin :: GetCaps** pour indiquer si le plug-in peut effectuer une conversion entre des formats. |
| [WMPServices \_ StreamState](/previous-versions/windows/desktop/api/wmpservices/ne-wmpservices-wmpservices_streamstate) | Indique si le flux est actuellement arrêté, suspendu ou en cours d’exécution.                     |
