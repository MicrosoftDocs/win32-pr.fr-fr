# <a name="registry-settings"></a>Paramètres du Registre

Le lecteur Windows Media série 9 ou version ultérieure utilise le registre pour stocker certains paramètres. Vous pouvez apporter des modifications à ces paramètres pour modifier le comportement du lecteur Windows Media et le contrôle du lecteur Windows Media.

Les sections suivantes détaillent les paramètres de Registre pris en charge.



| Section                                                                                                        | Description                                                                                                                                                           |
|----------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Paramètres de registre de l’extension de nom de fichier](file-name-extension-registry-settings.md)                             | Décrit comment inscrire une extension de nom de fichier personnalisée.                                                                                                               |
| [Paramètres de Registre du schéma personnalisé](custom-scheme-registry-settings.md)                                         | Décrit comment inscrire un schéma personnalisé.                                                                                                                            |
| [Paramètres du registre de surveillance des dossiers](folder-monitoring-registry-settings.md)                                 | Décrit comment inscrire des dossiers de fichiers à surveiller par le lecteur Windows Media.                                                                                       |
| [Paramètres de Registre du port du pare-feu](firewall-port-registry-settings.md)                                         | Décrit une entrée de registre que le lecteur Windows Media définit pour spécifier si les pare-feu doivent conserver les ports ouverts pour le partage de bibliothèque.                                   |
| [Paramètre de registre d’acquisition en mode silencieux](silent-acquisition-registry-setting.md)                                 | Décrit une entrée de Registre utilisée par le lecteur Windows Media pour déterminer s’il faut télécharger automatiquement les droits d’utilisation lorsque l’utilisateur lit ou synchronise le contenu protégé. |
| [Inscription de la dépendance d’application](registering-application-dependency.md)                                   | Fournit des informations sur l’inscription de votre application pour vous assurer que les fichiers d’exécution corrects existent sur l’ordinateur de l’utilisateur.                                         |
| [Entrées de Registre pour le suivi de la progression de l’installation](registry-entries-for-tracking-installation-progress.md) | Décrit les entrées de registre que les programmes d’installation peuvent utiliser pour suivre la progression de l’installation du lecteur Windows Media.                                                    |
