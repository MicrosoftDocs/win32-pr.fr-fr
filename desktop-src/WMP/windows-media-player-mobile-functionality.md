# <a name="windows-media-player-mobile-functionality"></a>Lecteur Windows Media Fonctionnalités mobiles

Les actions suivantes sont disponibles pour l’utilisateur de votre apparence :

-   Cliquer ou cliquer sur des boutons
-   Trackbars coulissant
-   Lecture du texte défilant
-   Regarder la vidéo
-   Affichage d’une pochette d’album
-   Évaluation des éléments multimédias

Les sections suivantes décrivent les types d’éléments que vous pouvez créer pour activer ces actions utilisateur :

-   [Fonctions de bouton](button-functions.md)
-   [Types TrackBar](trackbar-types.md)
-   [Types d’affichages de texte](text-display-types.md)
-   [Affichage vidéo](the-video-display.md)
-   [Orientation](orientation.md)
