# <a name="echo-sample-properties"></a>Exemples de propriétés Echo

L’exemple Echo expose deux propriétés : le délai et le niveau d’effet (mélange humide). La valeur du niveau de signal pur (mélange sec) est toujours dérivée de la valeur de la combinaison humide. Vous devez modifier le code existant et ajouter du code pour rendre ces propriétés accessibles.

Les sections suivantes expliquent comment modifier le code des propriétés :

-   [Fonctionnement des propriétés](how-properties-work.md)
-   [Variables pour stocker les propriétés](variables-to-store-properties.md)
-   [Modification de la propriété Scale](modifying-the-scale-property.md)
-   [Ajout de la propriété de combinaison humide](adding-the-wet-mix-property.md)
