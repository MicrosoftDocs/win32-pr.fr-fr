# <a name="listboxborder"></a>LISTBOX. Border

L’attribut **Border** spécifie ou récupère une valeur indiquant si le contrôle de zone de liste a une bordure.

``` syntax
        elementID.border
```

## <a name="possible-values"></a>Valeurs possibles

Cet attribut est une valeur **booléenne** .



| Valeur | Description                                  |
|-------|----------------------------------------------|
| true  | Par défaut. Le contrôle de zone de liste a une bordure.  |
| false | Le contrôle de zone de liste n’a pas de bordure. |



 

## <a name="remarks"></a>Notes

La bordure apparaît sous la forme d’une bordure enfoncée à deux pixels de largeur autour du contrôle de zone de liste avec les couleurs système.

## <a name="requirements"></a>Configuration requise



| Condition requise | Valeur |
|--------------------|---------------------------------------------------------|
| Version<br/> | Lecteur Windows Media pour Windows XP ou version ultérieure<br/> |
