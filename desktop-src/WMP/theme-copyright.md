# <a name="themecopyright"></a>THÈME. Copyright

L’attribut **Copyright** spécifie ou récupère la chaîne de copyright pour l’apparence.

``` syntax
theme.copyright
```

## <a name="possible-values"></a>Valeurs possibles

Cet attribut est une **chaîne** en lecture/écriture sans valeur par défaut.

## <a name="requirements"></a>Configuration requise



| Condition requise | Valeur |
|--------------------|------------------------------------------------------|
| Version<br/> | Lecteur Windows Media version 7,0 ou ultérieure<br/> |
