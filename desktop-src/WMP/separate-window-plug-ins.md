# <a name="separate-window-plug-ins"></a>Plug-ins de fenêtre distincts

Les plug-ins d’interface utilisateur de fenêtre distincts sont affichés dans leurs propres fenêtres. Ces fenêtres peuvent avoir une taille fixe ou peuvent être rendues redimensionnables. Plusieurs fenêtres de plug-in d’IU distinctes peuvent être ouvertes en même temps.

Les plug-ins de l’interface utilisateur de fenêtre distincts sont utiles lorsque vous souhaitez que les informations ou les contrôles que vous fournissez restent visibles lorsque l’utilisateur quitte le volet de **diffusion en cours** d’exécution ou charge un autre plug-in d’interface utilisateur. Tous les plug-ins de l’interface utilisateur de fenêtre distincts qui sont en cours d’exécution lorsque le lecteur Windows Media est fermé seront rechargés et affichés dans la même taille et dans la même position lors du redémarrage du lecteur.
