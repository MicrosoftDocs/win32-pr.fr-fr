# <a name="using-windows-media-player-with-netscape-71"></a>Utilisation de Windows Media Player avec Netscape 7.1

le navigateur Web Netscape 7,1 fournit une prise en charge native pour les contrôles de Lecteur Windows Media ActiveX à l’aide du composant ActiveX Gecko. cela signifie que vous pouvez incorporer le contrôle Lecteur Windows Media dans vos pages web pour Netscape 7,1 en utilisant la même syntaxe que celle utilisée pour Microsoft Internet Explorer. Netscape 7,1 prend également en charge les scripts de gestion des contrôles et des événements de la même manière.

si vous souhaitez utiliser le contrôle Lecteur Windows Media dans netscape, nous vous recommandons d’utiliser netscape 7,1.

pour plus d’informations sur l’utilisation du contrôle Lecteur Windows Media avec netscape 7,1, consultez le [site web de netscape](https://developer.mozilla.org/docs/Archive/Windows_Media_in_Netscape).

pour plus d’informations sur le composant Gecko ActiveX, consultez le [site web StreamingMedia.com](https://www.streamingmedia.com/Articles/Editorial/Featured-Articles/The-Windows-Media-ActiveX-Control-8211%3b-Not-Just-for-Internet-Explorer-Anymore-64608.aspx).

Le téléchargement complet de ce kit de développement logiciel (SDK) comprend un exemple qui illustre la détection du navigateur Web. Pour plus d’informations sur les exemples inclus dans le kit de développement logiciel (SDK), consultez [exemples](samples.md).
