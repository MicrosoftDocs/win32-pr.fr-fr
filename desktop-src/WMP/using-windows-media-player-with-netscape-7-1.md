# <a name="using-windows-media-player-with-netscape-71"></a>Utilisation de Windows Media Player avec Netscape 7.1

Le navigateur Web Netscape 7,1 fournit une prise en charge native pour les contrôles Windows Media Player ActiveX à l’aide du composant ActiveX Gecko. Cela signifie que vous pouvez incorporer le contrôle du lecteur Windows Media dans vos pages Web pour Netscape 7,1 en utilisant la même syntaxe que celle utilisée pour Microsoft Internet Explorer. Netscape 7,1 prend également en charge les scripts de gestion des contrôles et des événements de la même manière.

Si vous souhaitez utiliser le contrôle du lecteur Windows Media dans Netscape, nous vous recommandons d’utiliser Netscape 7,1.

Pour plus d’informations sur l’utilisation du contrôle du lecteur Windows Media avec Netscape 7,1, consultez le [site Web de Netscape](https://developer.mozilla.org/docs/Archive/Windows_Media_in_Netscape).

Pour plus d’informations sur le composant ActiveX Gecko, consultez le [site web StreamingMedia.com](https://www.streamingmedia.com/Articles/Editorial/Featured-Articles/The-Windows-Media-ActiveX-Control-8211%3b-Not-Just-for-Internet-Explorer-Anymore-64608.aspx).

Le téléchargement complet de ce kit de développement logiciel (SDK) comprend un exemple qui illustre la détection du navigateur Web. Pour plus d’informations sur les exemples inclus dans le kit de développement logiciel (SDK), consultez [exemples](samples.md).
