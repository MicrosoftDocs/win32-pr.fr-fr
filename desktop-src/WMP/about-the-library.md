# <a name="about-the-library"></a>À propos de la bibliothèque

La bibliothèque est une base de données d’informations, ou *métadonnées*, sur le contenu multimédia numérique disponible pour le lecteur Windows Media. Certaines des informations s’affichent dans le volet **bibliothèque** du lecteur. une plus grande disponibilité est disponible par le biais du code.

(Dans le lecteur Windows Media série 9 et versions antérieures, cette fonctionnalité est appelée **bibliothèque multimédia**.)

La bibliothèque offre aux utilisateurs un moyen simple d’organiser et d’accéder à du contenu multimédia numérique. Par exemple, les utilisateurs peuvent afficher la musique organisée par album, par artiste, par genre ou simplement comme une liste de toutes les musiques. Vous pouvez utiliser le modèle objet du kit de développement logiciel (SDK) du lecteur Windows Media pour accéder à la bibliothèque et lire du contenu, récupérer des sélections, ajouter du contenu, supprimer du contenu et modifier les métadonnées associées. Le lecteur Windows Media protège la confidentialité des utilisateurs en restreignant votre capacité à accéder à la bibliothèque à partir du code sous certaines conditions. Pour plus d’informations sur les droits d’accès, consultez accès à la [bibliothèque](library-access.md).

La bibliothèque stocke les métadonnées sur deux types de contenu multimédia numérique de base. Le premier type, les éléments multimédias numériques, sont des fichiers de contenu individuels, tels qu’une piste musicale ou une photo. Le deuxième type, les sélections, sont des fichiers qui représentent un groupe d’éléments multimédias numériques, généralement destinés à être lus dans un ordre spécifié. Les métadonnées associées au contenu multimédia numérique sont stockées en tant que paires nom-valeur. Par exemple, les métadonnées qui décrivent la personne qui a effectué une chanson sont nommées « Artist » et la valeur qui lui est associée est généralement une chaîne de texte contenant le nom de l’artiste. Chaque nom de métadonnées unique est appelé *attribut*. Pour obtenir la liste des attributs pris en charge, consultez la [référence d’attribut](attribute-reference.md). Pour plus d’informations sur l’utilisation d’attributs dans le code, consultez [attributs d’élément multimédia](media-item-attributes.md).
