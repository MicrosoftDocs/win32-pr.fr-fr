# <a name="file-name"></a>Nom de fichier

Il s’agit du nom du fichier qui contient l’image. Le chemin d’accès ne doit pas être donné et le fichier doit se trouver dans le répertoire Skin. L’extension de nom de fichier doit être. bmp ou. gif.
