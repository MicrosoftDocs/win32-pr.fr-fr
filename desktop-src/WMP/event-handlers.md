# <a name="event-handlers"></a>Gestionnaires d'événements

Microsoft JScript est utilisé pour traiter les événements dans le fichier de définition d’apparence. Pour plus d’informations sur les gestionnaires d’événements, consultez [gestion des événements](handling-events.md) .

Vous pouvez avoir plusieurs lignes de code dans un gestionnaire d’événements, mais vous devez veiller à ne pas dépasser la longueur de ligne autorisée par JScript. Séparez les lignes par des points-virgules.


```C++
onclick = "JScript: player.URL = 'https://proseware.com/cool.wma' ; myText.value = 'Playing'; "

```
