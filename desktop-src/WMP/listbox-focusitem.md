# <a name="listboxfocusitem"></a>LISTBOX. focusItem

L’attribut **focusItem** spécifie ou récupère la ligne qui contient le focus.

``` syntax
        elementID.focusItem
```

## <a name="possible-values"></a>Valeurs possibles

Cet attribut est un **nombre** en lecture/écriture (**long**).

## <a name="remarks"></a>Remarques

L’élément de focus et l’élément sélectionné peuvent être différents. Un élément peut être sélectionné par le biais d’un script, mais un autre élément peut recevoir le focus par le biais d’un script. L’élément ayant le focus aura un rectangle en pointillés autour et sera sélectionné si la touche entrée est enfoncée.
