# <a name="iwmpclosedcaption-vb-and-c-interface"></a>interface IWMPClosedCaption (VB et C#)

Offre un moyen d’inclure des sous-titres avec un fichier multimédia numérique. Le texte de sous-titrage se trouve dans un fichier SAMI (Synchronized Accessible Media Interchange).

## <a name="members"></a>Membres

l’interface **IWMPClosedCaption (VB et C#)** ne définit aucun membre.

L’interface **IWMPClosedCaption** expose les propriétés suivantes.



| Propriété                                                                             | Description                                                                                |
|--------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------|
| [captioningId](wmplibiwmpclosedcaption-iwmpclosedcaption-captioningid--vb-and-c.md) | Obtient ou définit le nom de l’élément HTML affichant le sous-titrage.                       |
| [SAMIFileName](wmplibiwmpclosedcaption-iwmpclosedcaption-samifilename--vb-and-c.md) | Obtient ou définit le nom du fichier contenant les informations nécessaires pour le sous-titrage. |
| [SAMILang](wmplibiwmpclosedcaption-iwmpclosedcaption-samilang--vb-and-c.md)         | Obtient ou définit la langue affichée pour le sous-titrage.                                 |
| [SAMIStyle](wmplibiwmpclosedcaption-iwmpclosedcaption-samistyle--vb-and-c.md)       | Obtient ou définit le style de sous-titrage.                                                  |



 

Procurez-vous une interface **IWMPClosedCaption** à l’aide de la propriété suivante.



| Object                                                            | Propriété                                                                   |
|-------------------------------------------------------------------|----------------------------------------------------------------------------|
| [AxWindowsMediaPlayer](axwindowsmediaplayer-object--vb-and-c.md) | [closedCaption](axwmplib-axwindowsmediaplayer-closedcaption--vb-and-c.md) |
