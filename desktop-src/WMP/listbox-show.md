# <a name="listboxshow"></a>LISTBOX. Show

La méthode **Show** affiche le contrôle.

``` syntax
        elementID.show()
```

## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.

## <a name="return-value"></a>Valeur renvoyée

Cette méthode ne retourne pas de valeur.

## <a name="remarks"></a>Notes

Cette méthode est principalement utilisée avec l’élément **Popup** , qui représente un contrôle de zone de liste qui s’affiche uniquement lorsque cela est nécessaire.

## <a name="requirements"></a>Configuration requise



| Condition requise | Valeur |
|--------------------|---------------------------------------------------------|
| Version<br/> | Lecteur Windows Media pour Windows XP ou version ultérieure<br/> |
