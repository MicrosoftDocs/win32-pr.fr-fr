# <a name="statustext"></a>STATUSTEXT

Il s’agit d’un élément de **texte** prédéfini avec les valeurs par défaut suivantes.

``` syntax
value="wmpprop:player.status"
tabstop="true"
```

## <a name="remarks"></a>Notes

Cela permet de créer un élément de **texte** qui affichera l’état de lecture. Toutes les propriétés de cet élément de **texte** peuvent être substituées en les spécifiant explicitement.

## <a name="requirements"></a>Spécifications



| Condition requise | Valeur |
|--------------------|----------------------------------------------|
| Version<br/> | Lecteur Windows Media 7,0 ou version ultérieure<br/> |
