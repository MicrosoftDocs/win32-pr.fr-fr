# <a name="submitting-your-visualization"></a>Envoi de votre visualisation

L’envoi de votre visualisation via le site Web WMPlugins n’est plus pris en charge.

## <a name="test-your-visualization"></a>Tester votre visualisation

Vous pouvez essayer d’installer votre visualisation sur plusieurs ordinateurs pour vous assurer que tout fonctionne comme vous le souhaitez. Essayez chaque présélection pour vous assurer que votre code fonctionne correctement.

## <a name="sign-your-visualization"></a>Signer votre visualisation

Vous devez soumettre votre visualisation pour la signature de code afin que l’utilisateur puisse décider d’approuver ou non votre visualisation avant de l’installer. Pour plus d’informations sur la signature de code, consultez [Introduction à la signature de code](/previous-versions/windows/internet-explorer/ie-developer/platform-apis/ms537361(v=vs.85)).

## <a name="distributing-your-visualization"></a>Distribution de votre visualisation

Bien que le site Web WMPlugins ne prenne plus en charge la distribution de votre visualisation, vous pouvez toujours distribuer votre visualisation à partir d’un site Web de votre choix.
