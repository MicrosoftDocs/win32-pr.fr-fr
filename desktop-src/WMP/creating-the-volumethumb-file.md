# <a name="creating-the-volumethumb-file"></a>Création du fichier VolumeThumb

Le fichier VolumeThumb contient l’image Thumb qui sera utilisée avec un TrackBar de volume.

L’image suivante est le fichier VolumeThumb.

![fichier volumethumb](images/ceswmvol.png)

Contient les deux images Thumb. Celui qui se trouve à gauche est l’état normal, et celui sur la droite est l’état d’envoi (push). Il n’y a pas d’état désactivé. Si un TrackBar est désactivé, l’image d’arrière-plan désactivée du TrackBar est affichée à partir du fichier super.
