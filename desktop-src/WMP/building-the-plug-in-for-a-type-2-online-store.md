# <a name="building-the-plug-in-for-a-type-2-online-store"></a>Création du plug-in pour un magasin de type 2 en ligne

Vous pouvez créer un plug-in de magasin en ligne de type 2 à l’aide de l’Assistant de plug-in de magasin en ligne. L’Assistant projet est inclus dans le kit de développement logiciel (SDK) du lecteur Windows Media, qui fait partie du SDK Windows. L’Assistant est un complément pour Microsoft Visual Studio qui crée un projet C++ et génère un exemple de code que vous pouvez utiliser comme point de départ pour développer votre propre magasin en ligne.

Les rubriques suivantes expliquent comment utiliser l’Assistant de plug-in de boutique en ligne :

-   [Installation de l’Assistant projet](installing-the-project-wizard.md)
-   [Création d’un exemple de projet](creating-a-sample-project.md)
-   [Compilation de l’exemple de projet](compiling-the-sample-project.md)
-   [Exécution de l’exemple de projet](running-the-sample-project.md)
-   [Personnalisation de l’exemple de projet](customizing-the-sample-project.md)
