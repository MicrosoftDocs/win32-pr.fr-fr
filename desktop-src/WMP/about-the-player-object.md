# <a name="about-the-player-object"></a>À propos de l’objet Player

l’objet **lecteur** est l’objet principal pour le contrôle Lecteur Windows Media. Tous les autres objets connexes sont connectés à cet objet via des propriétés spécifiques qui retournent l’objet. par exemple, l’objet **Paramètres** est accessible via la propriété **settings** . l’objet **Player** fournit des méthodes, des propriétés et des événements liés à la fonctionnalité de base de Lecteur Windows Media.

Étant donné que cette référence est également utilisée pour la programmation des apparences, l’ID de l’objet **Player** sera « Player » pour les exemples de syntaxe.

L’utilisation de l’attribut global **Player** dans un fichier de définition d’apparence donne accès à l’objet **Player** pour une utilisation dans les scripts. par le biais de l’objet **Player** , tous les autres objets du contrôle Lecteur Windows Media deviennent également accessibles aux scripts. Les événements de l’objet **Player** et de la propriété **URL** peuvent également être spécifiés au moment de la conception à l’aide de l’élément Skin [Player](player-element.md) . pour plus d’informations, consultez [Lecteur Windows Media skins](windows-media-player-skins.md).
