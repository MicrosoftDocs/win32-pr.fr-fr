# <a name="about-the-player-object"></a>À propos de l’objet Player

L’objet **lecteur** est l’objet principal pour le contrôle du lecteur Windows Media. Tous les autres objets connexes sont connectés à cet objet via des propriétés spécifiques qui retournent l’objet. Par exemple, l’objet **Settings** est accessible via la propriété **Settings** . L’objet **Player** fournit des méthodes, des propriétés et des événements liés aux fonctionnalités principales du lecteur Windows Media.

Étant donné que cette référence est également utilisée pour la programmation des apparences, l’ID de l’objet **Player** sera « Player » pour les exemples de syntaxe.

L’utilisation de l’attribut global **Player** dans un fichier de définition d’apparence donne accès à l’objet **Player** pour une utilisation dans les scripts. Par le biais de l’objet **Player** , tous les autres objets dans le contrôle du lecteur Windows Media sont également accessibles aux scripts. Les événements de l’objet **Player** et de la propriété **URL** peuvent également être spécifiés au moment de la conception à l’aide de l’élément Skin [Player](player-element.md) . Pour plus d’informations, consultez [apparences du lecteur Windows Media](windows-media-player-skins.md).
