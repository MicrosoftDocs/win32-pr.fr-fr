# <a name="playlistitemselectedcolor"></a>PLAYLIST. itemSelectedColor

L’attribut **itemSelectedColor** spécifie ou récupère une valeur indiquant la couleur du texte d’un élément sélectionné dans la sélection.

``` syntax
        elementID.itemSelectedColor
```

## <a name="possible-values"></a>Valeurs possibles

Cet attribut est une **chaîne** en lecture/écriture représentant une couleur. La valeur par défaut est la couleur de texte en surbrillance Windows.
