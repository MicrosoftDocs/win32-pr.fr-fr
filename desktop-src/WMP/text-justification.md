# <a name="textjustification"></a>TEXT. Justification

L’attribut de **Justification** spécifie ou récupère l’alignement du texte dans le contrôle de texte.

``` syntax
        elementID.justification
```

## <a name="possible-values"></a>Valeurs possibles

Cet attribut est une **chaîne** en lecture/écriture.



| Valeur  | Description                                                   |
|--------|---------------------------------------------------------------|
| Gauche   | Par défaut. Aligne le texte à gauche du contrôle de texte.     |
| Right  | Aligne le texte à droite du contrôle de texte.             |
| Center | Aligne le texte sur le centre horizontal du contrôle de texte. |



 

## <a name="remarks"></a>Remarques

Pour obtenir un exemple illustrant l’utilisation des attributs de l’élément de **texte** , consultez l’attribut [value](text-value.md) .
