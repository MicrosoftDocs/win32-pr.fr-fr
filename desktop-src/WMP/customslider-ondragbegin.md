# <a name="customsliderondragbegin"></a>CUSTOMSLIDER.onDragBegin

Le gestionnaire d’événements **onDragBegin** gère un événement qui se produit lorsque l’utilisateur clique et maintient le bouton gauche de la souris enfoncé et commence à faire glisser la souris.

``` syntax
onDragBegin
```

## <a name="requirements"></a>Spécifications



| Condition requise | Valeur |
|--------------------|------------------------------------------------------|
| Version<br/> | Lecteur Windows Media version 7,0 ou ultérieure<br/> |
