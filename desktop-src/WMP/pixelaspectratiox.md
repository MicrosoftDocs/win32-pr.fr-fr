# <a name="pixelaspectratiox"></a>PixelAspectRatioX

L’attribut **PixelAspectRatioX** est la partie horizontale des proportions d’une vidéo.

## <a name="applies-to"></a>S'applique à

-   [Éléments vidéo](video-item-attributes.md)

## <a name="remarks"></a>Notes

Pour une vidéo avec des proportions de 16:9, **PixelAspectRatioX** est égal à « 16 ».

## <a name="requirements"></a>Configuration requise



| Condition requise | Valeur |
|--------------------|------------------------------------|
| Version<br/> | Lecteur Windows Media 11<br/> |
