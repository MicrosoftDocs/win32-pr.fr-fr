# <a name="embedding-the-windows-media-player-control-in-a-web-page"></a>Incorporation du contrôle du lecteur Windows Media dans une page Web

Lorsque vous incorporez le contrôle du lecteur Windows Media dans une page Web, vous pouvez adopter une approche de base ou une approche plus élaborée.

Dans l’approche de base, vous utilisez le contrôle simplement pour lire de la musique en arrière-plan ou pour ajouter un affichage vidéo à votre page.

Dans l’approche la plus élaborée, vous rendez votre page interactive en ajoutant du code de script pour manipuler le contrôle par le biais du modèle objet du lecteur. Cela vous permet de fournir une interface utilisateur de votre propre conception ou de répondre de manière dynamique aux commandes ou événements de script tels que les transitions de médias.

Pour plus d’informations, consultez [utilisation du contrôle du lecteur Windows Media dans une page Web](using-the-windows-media-player-control-in-a-web-page.md).
