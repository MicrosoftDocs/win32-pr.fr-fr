# <a name="shufflebutton"></a>SHUFFLEBUTTON

Il s’agit d’un **bouton** prédéfini avec les valeurs par défaut suivantes.

``` syntax
onclick="jscript:player.settings.setMode('shuffle',down);"
upToolTip="Turn Shuffle On"
downToolTip="Turn Shuffle Off"
down="wmpprop:player.settings.getMode('shuffle')"
sticky="true"
```

## <a name="remarks"></a>Notes

Cela crée un contrôle **Button** qui active ou désactive la lecture aléatoire. Les info-bulles sont localisées. Toutes les propriétés de ce **bouton** peuvent être remplacées en les spécifiant explicitement.

## <a name="requirements"></a>Configuration requise

|--------------------|----------------------------------------------|
| Version<br/> | Lecteur Windows Media 7,0 ou version ultérieure<br/> |
