# <a name="user-interface-plug-ins-programming-reference"></a>Informations de référence sur la programmation de plug-ins d’interface utilisateur

Le kit de développement logiciel (SDK) Microsoft Windows Media Player prend en charge une interface pour les plug-ins d’interface utilisateur. La section suivante décrit cela en détail.



| Section                                                  | Description                                                                                                   |
|----------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [WMPNotifyPluginAddRemove](/previous-versions/windows/desktop/api/wmpplug/nf-wmpplug-wmpnotifypluginaddremove) | Fonction indépendante utilisée pour informer le lecteur Windows Media qu’un plug-in a été installé ou désinstallé. |
| [Interface IWMPPluginUI](/previous-versions/windows/desktop/api/wmpplug/nn-wmpplug-iwmppluginui)               | Interface aux plug-ins d’interface utilisateur.                                                                                  |
| [Indicateurs d’inscription](registration-flags.md)             | Liste d’indicateurs de paramètres qui affectent le comportement des plug-ins d’interface utilisateur.                                             |
