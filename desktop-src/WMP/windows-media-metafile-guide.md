# <a name="windows-media-metafile-guide"></a>Windows Guide du métafichier multimédia

un métafichier Windows Media peut être aussi simple ou complexe que vous le souhaitez. le métafichier Windows Media le plus basique contient uniquement les Uniform Resource Locator (URL) de certains contenus multimédias sur un serveur. le client, Lecteur Windows Media, analyse ces informations, puis ouvre le fichier multimédia ou le flux défini dans le métafichier Windows media. un métafichier complexe peut contenir plusieurs fichiers ou flux organisés dans une sélection, des instructions sur la façon de lire les fichiers, les flux, le texte et les éléments graphiques tels que le titre, l’auteur et le texte de copyright, l’insertion de publicité personnalisée dans un flux en direct, des liens hypertexte associés à des éléments de l’interface Lecteur Windows Media, et bien plus encore.
