# <a name="windows-media-metafile-guide"></a>Guide des métafichiers Windows Media

Un métafichier Windows Media peut être aussi simple ou complexe que vous le souhaitez. Le métafichier Windows Media le plus basique contient uniquement l’Uniform Resource Locator (URL) de certains contenus multimédias sur un serveur. Le client, le lecteur Windows Media, analyse ces informations, puis ouvre le fichier multimédia ou le flux défini dans le métafichier Windows Media. Un métafichier complexe peut contenir plusieurs fichiers ou flux organisés dans une sélection, des instructions sur la façon de lire les fichiers, les flux, le texte et les éléments graphiques tels que le titre, l’auteur et le texte de copyright, l’insertion de publicité personnalisée dans un flux en direct, des liens hypertexte associés à des éléments de l’interface du lecteur Windows Media, et bien plus encore.
