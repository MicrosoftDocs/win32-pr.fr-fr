# <a name="download-manager"></a>Gestionnaire de téléchargement

Le gestionnaire de téléchargement de Microsoft Windows Media Player permet aux magasins de type 2 en ligne de contrôler le processus de téléchargement de contenu à partir d’un volet de tâches de Windows Media Player Online Store. Un volet de tâches de magasin en ligne héberge une page Web qui s’affiche lorsque l’utilisateur choisit d’utiliser un magasin en ligne.

Cette présentation du gestionnaire de téléchargement contient les sections suivantes.



| Section                                                            | Description                                                     |
|--------------------------------------------------------------------|-----------------------------------------------------------------|
| [Présentation du gestionnaire de téléchargement](download-manager-overview.md)         | Présente le gestionnaire de téléchargement et certains scénarios d’utilisation de base. |
| [Architecture du gestionnaire de téléchargement](download-manager-architecture.md) | Fournit un résumé du modèle objet du gestionnaire de téléchargement.        |
