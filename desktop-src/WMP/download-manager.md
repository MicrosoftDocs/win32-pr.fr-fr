# <a name="download-manager"></a>Gestionnaire de téléchargement

le gestionnaire de téléchargement Microsoft Lecteur Windows Media permet aux magasins de type 2 en ligne de contrôler le processus de téléchargement de contenu à partir d’un volet de tâches Lecteur Windows Media store en ligne. Un volet de tâches de magasin en ligne héberge une page Web qui s’affiche lorsque l’utilisateur choisit d’utiliser un magasin en ligne.

Cette présentation du gestionnaire de téléchargement contient les sections suivantes.

| [Présentation du gestionnaire de téléchargement](download-manager-overview.md)         | Présente le gestionnaire de téléchargement et certains scénarios d’utilisation de base. |
