# <a name="about-dsp-plug-ins"></a>À propos des plug-ins DSP

Microsoft Lecteur Windows Media fournit une architecture pour la connexion de logiciels complémentaires qui exécutent des fonctions de traitement des signaux numériques (DSP) sur les flux audio ou vidéo pendant la lecture. Ces programmes additionnels sont appelés plug-ins DSP.

les sections suivantes fournissent une vue d’ensemble conceptuelle des plug-ins Lecteur Windows Media DSP.



| Section                                                              | Description                                                                                            |
|----------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------|
| [Vue d’ensemble de l’utilisateur du plug-in DSP](dsp-plug-in-user-overview.md)           | Décrit les plug-ins DSP du point de vue de l’utilisateur final.                                                |
| [Vue d’ensemble des développeurs de plug-ins DSP](dsp-plug-in-developer-overview.md) | fournit des informations générales que vous devez connaître avant de générer un plug-in DSP pour Lecteur Windows Media. |
| [Génération d’un plug-in DSP](building-a-dsp-plug-in.md)                 | explique comment utiliser l’assistant de plug-in Lecteur Windows Media pour créer un exemple de plug-in DSP.            |
| [Implémentation de votre code DSP](implementing-your-dsp-code.md)         | décrit les considérations relatives à la création de plug-ins DSP audio et vidéo personnalisés pour Lecteur Windows Media.    |
