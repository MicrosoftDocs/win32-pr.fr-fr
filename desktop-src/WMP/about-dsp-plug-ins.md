# <a name="about-dsp-plug-ins"></a>À propos des plug-ins DSP

Le lecteur Microsoft Windows Media fournit une architecture permettant de connecter des logiciels complémentaires qui exécutent des fonctions DSP (Digital Signal Processing) sur les flux audio ou vidéo pendant la lecture. Ces programmes additionnels sont appelés plug-ins DSP.

Les sections suivantes fournissent une vue d’ensemble conceptuelle des plug-ins du lecteur Windows Media DSP.



| Section                                                              | Description                                                                                            |
|----------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------|
| [Vue d’ensemble de l’utilisateur du plug-in DSP](dsp-plug-in-user-overview.md)           | Décrit les plug-ins DSP du point de vue de l’utilisateur final.                                                |
| [Vue d’ensemble des développeurs de plug-ins DSP](dsp-plug-in-developer-overview.md) | Fournit des informations générales que vous devez connaître avant de générer un plug-in DSP pour Windows Media Player. |
| [Génération d’un plug-in DSP](building-a-dsp-plug-in.md)                 | Explique comment utiliser l’Assistant de plug-in du lecteur Windows Media pour créer un exemple de plug-in DSP.            |
| [Implémentation de votre code DSP](implementing-your-dsp-code.md)         | Décrit les considérations relatives à la création de plug-ins DSP audio et vidéo personnalisés pour le lecteur Windows Media.    |
