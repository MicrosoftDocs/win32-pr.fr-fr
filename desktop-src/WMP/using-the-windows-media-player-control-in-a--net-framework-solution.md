# <a name="using-the-windows-media-player-control-in-a-net-framework-solution"></a>Utilisation du contrôle du lecteur Windows Media dans une solution .NET Framework

Le contrôle ActiveX du lecteur Windows Media série 9 ou version ultérieure prend en charge l’incorporation dans les applications .NET Framework. Les sections suivantes fournissent des informations spécifiques à l’utilisation du contrôle du lecteur Windows Media avec l' .NET Framework.



| Section                                                                                                                                                      | Description                                                                                                                  |
|--------------------------------------------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------|
| [Utilisation du contrôle du lecteur Windows Media avec Microsoft Visual Studio](using-the-windows-media-player-control-with-microsoft-visual-studio.md)               | Fournit des détails sur l’ajout du contrôle Windows Media Player 9 ou ultérieur à une solution Visual Studio.                |
| [Incorporation du contrôle du lecteur Windows Media dans une solution C#](embedding-the-windows-media-player-control-in-a-c--solution.md)                              | Fournit des instructions pour créer une application C# simple qui utilise le contrôle du lecteur Windows Media.                      |
| [Intégration du contrôle du lecteur Windows Media dans une solution Visual Basic .NET](embedding-the-windows-media-player-control-in-a-visual-basic--net-solution.md) | Fournit des instructions pour la création d’une simple application Visual Basic .NET qui utilise le contrôle du lecteur Windows Media.        |
| [Création du contrôle du lecteur Windows Media par programmation](creating-the-windows-media-player-control-programmatically.md)                                 | Décrit comment créer un contrôle de lecteur Windows Media dans une application sans utiliser la boîte à outils Visual Studio.            |
| [À propos des types de données .NET Framework](about--net-framework-data-types.md)                                                                                       | Fournit les informations nécessaires pour convertir la référence de modèle objet orienté script en .NET Framework types de données de base. |
