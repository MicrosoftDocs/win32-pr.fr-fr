# <a name="type-2-online-stores"></a>Tapez 2 magasins en ligne

> [!Note]  
> Cette section décrit les fonctionnalités conçues pour être utilisées par les magasins en ligne. L’utilisation de cette fonctionnalité en dehors du contexte d’un magasin en ligne n’est pas prise en charge.

 

Lecteur Windows Media 11 prend en charge les magasins en ligne de type 1 et de type 2. Les magasins de type 1 sont plus profondément intégrés à l’interface utilisateur du lecteur que les magasins de type 2. Lecteur Windows Media 10 prend en charge les magasins de type 2, mais pas les magasins de type 1. Cette documentation, qui s’applique uniquement aux magasins de type 2, est composée des sections suivantes.



| Section                                                                                      | Description                                                                                                                            |
|----------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------|
| [À propos des magasins en ligne de type 2](about-type-2-online-stores.md)                                 | Fournit une vue d’ensemble des magasins en ligne de type 2 et des informations générales sur le plug-in et les exemples.                                |
| [Guide de programmation pour les magasins en ligne de type 2](programming-guide-for-type-2-online-stores.md) | Explique comment créer un plug-in pour un magasin de type 2 en ligne et fournit des informations sur les pages Web associées et le gestionnaire de téléchargement. |
| [Référence pour les magasins en ligne de type 2](reference-for-type-2-online-stores.md)                 | Fournit une référence détaillée pour les interfaces, les objets, les énumérations et les entrées de Registre utilisées par les magasins en ligne de type 2.              |
