# <a name="iwmperroritem2-vb-and-c-interface"></a>Interface IWMPErrorItem2 (VB et C#)

Fournit une propriété pour obtenir un code de condition d’erreur.

Outre les propriétés héritées de **IWMPErrorItem**, l’interface **IWMPErrorItem2** expose la propriété suivante.



| Propriété                                                                 | Description                                          |
|--------------------------------------------------------------------------|------------------------------------------------------|
| [condition](wmplibiwmperroritem2-iwmperroritem2-condition--vb-and-c.md) | Obtient une valeur indiquant la condition de l’erreur. |



 

Obtient une interface **IWMPErrorItem2** en effectuant un cast de la valeur retournée par la propriété **Item** (en C#, la méthode d' **extraction d' \_ élément**) de l’interface **IWMPErrorItem** .

## <a name="members"></a>Membres

L’interface **IWMPErrorItem2 (VB et C#)** ne définit aucun membre.
