# <a name="skin-files"></a>Fichiers d’apparence

Quand vous voyez une apparence, vous voyez uniquement l’aire. Derrière la surface se trouvent plusieurs fichiers qui composent l’apparence complète. Techniquement parlant, une apparence est un groupe de fichiers, chaque fichier contenant un type spécifique d’informations. Voici les types de fichiers que vous pouvez utiliser pour créer une apparence complète.



| Type de fichier                                        | Description                                                                                                                                                               |
|--------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Fichier de définition d’apparence](skin-definition-file.md) | Fichier maître qui définit la manière dont les autres fichiers seront utilisés. Il s’agit d’un fichier texte dont l’extension de nom de fichier est. WMS.                                                    |
| [Fichiers art](art-files.md)                       | Des fichiers artistiques qui contiennent les éléments graphiques de votre apparence. Il peut s’agir de fichiers bitmap, GIF, JPEG et PNG.                                                                 |
| [JScript Fichiers](jscript-files.md)               | Si vous souhaitez créer des réponses plus complexes aux événements, vous pouvez créer des fichiers de script à l’aide de Microsoft JScript. Il s’agit de fichiers texte dont l’extension de nom de fichier est .js. |
