# <a name="videosettingsreset"></a>VIDEOSETTINGS. Reset

La méthode **Reset** rétablit la valeur par défaut de tous les attributs de zéro.

``` syntax
        elementID.reset()
```

## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.

## <a name="return-value"></a>Valeur renvoyée

Cette méthode ne retourne pas de valeur.

## <a name="requirements"></a>Configuration requise



| Condition requise | Valeur |
|--------------------|-----------------------------------------------------------------|
| Version<br/> | Version du lecteur Windows Media pour Windows XP ou version ultérieure<br/> |
