# <a name="adding-closed-captions-to-digital-media"></a>Ajout de sous-titres à des médias numériques

Le format SAMI (Synchronized Accessible Media Interchange) est un format de fichier conçu pour fournir du texte synchronisé comme des sous-titres, des sous-titres ou des descriptions audio avec du contenu multimédia numérique. intégré dans un navigateur Web à l’aide du modèle d’objet Lecteur Windows Media, le same favorise l’accessibilité. En utilisant le same, vous pouvez créer du contenu multimédia riche qui atteint un public plus large et plus diversifié.

En plus des polices standard, le same peut prendre en charge d’autres styles de texte, tels que des couleurs, des tailles ou des langues différentes pour aider un grand nombre d’utilisateurs. Le same peut être particulièrement utile pour les personnes malvoyantes ou sourdes ou malentendantes. Le format SAMI peut également aider à des fins éducatives, telles que l’enseignement d’un autre langage pour les élèves.

cette rubrique se concentre sur l’incorporation de SAMI avec le Lecteur Windows Media ActiveX modèle objet de contrôle. Les fichiers SAMI existent indépendamment des fichiers multimédias numériques et ne s’appuient pas sur un format vidéo ou audio spécifique pour fonctionner. étant donné que les fichiers sont séparés, le contrôle Lecteur Windows Media localise, analyse, synchronise et restitue chaque fichier sur l’ordinateur du client. Cela offre une flexibilité et une fonctionnalité accrues, car elle permet la modification de fichiers SAMI individuels, l’incorporation du fichier SAMI avec différents formats multimédias numériques et le stockage de fichiers SAMI sur différents emplacements de serveurs.
