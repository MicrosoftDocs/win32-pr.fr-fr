# <a name="music-file-attributes"></a>Attributs du fichier musical

Les attributs suivants sont disponibles pour un fichier musical qui ne se trouve pas dans la bibliothèque. Ces attributs correspondent aux entrées trouvées dans la boîte de dialogue **éditeur de balise avancée** du lecteur Windows Media.

-   [Description (attribut)](description-attribute.md)
-   [Attribut WM/OriginalAlbumTitle](wm-originalalbumtitle-attribute.md)
-   [Attribut WM/OriginalArtist](wm-originalartist-attribute.md)
-   [Attribut WM/OriginalLyricist](wm-originallyricist-attribute.md)
-   [Attribut WM/PartOfSet](wm-partofset-attribute.md)
