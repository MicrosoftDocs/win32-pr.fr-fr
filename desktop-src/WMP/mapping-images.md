# <a name="mapping-images"></a>Mapper des images

Les images de mappage sont utilisées pour des contrôles spécifiques afin de spécifier les régions qui répondront aux clics de souris et pour déterminer quels contrôles reçoivent les événements. L’image de mappage doit avoir la même hauteur et la même largeur que l’image principale et doit être créée afin que les images des deux fichiers soient alignées exactement. Pour cette raison, il est recommandé d’utiliser un programme d’édition de graphiques qui prend en charge les couches, car vous pouvez être certain que vos images sont parfaitement alignées.

Différents contrôles requièrent différents types de mappages. Un élément **ButtonElement** a besoin d’un mappage qui a une couleur différente pour chaque bouton. Les zones colorées dans le fichier de mappage doivent correspondre aux zones des boutons que vous souhaitez mapper.
