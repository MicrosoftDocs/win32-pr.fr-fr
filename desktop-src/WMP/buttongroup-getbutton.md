# <a name="buttongroupgetbutton"></a>BUTTONGROUP. getButton

La méthode **GetButton** récupère le **BUTTONELEMENT** avec l’index spécifié.

``` syntax
        elementID.getButton(button)
```

## <a name="parameters"></a>Paramètres

<dl> <dt>

<span id="button"></span><span id="BUTTON"></span>*bouton*
</dt> <dd>

**Nombre** (**long**) contenant l’index du **BUTTONELEMENT** à récupérer.

</dd> </dl>

## <a name="return-value"></a>Valeur renvoyée

Cette méthode retourne un objet correspondant à un élément **BUTTONELEMENT** .

## <a name="requirements"></a>Configuration requise



| Condition requise | Valeur |
|--------------------|---------------------------------------------------|
| Version<br/> | Lecteur Windows Media série 9 ou version ultérieure<br/> |
