# <a name="about-the-stringcollection-object"></a>À propos de l’objet StringCollection

L’objet **StringCollection** régit les collections de chaînes, qui sont utilisées pour fournir des fonctionnalités de gestion des chaînes pour l’objet **MediaCollection** . L’objet **StringCollection** peut être récupéré par le biais de la méthode **getAttributeStringCollection** de l’objet **MediaCollection** . Vous pouvez l’utiliser pour travailler avec des collections de chaînes qui représentent les valeurs de divers attributs d’éléments multimédias.
