# <a name="compatibility-issues"></a>Problèmes de compatibilité

## <a name="netscape-support"></a>Support Netscape

Les pages Web ouvertes par la fonctionnalité HTMLView sont affichées à l’aide d’une instance incorporée de Microsoft Internet Explorer. Aucune compatibilité Netscape n’est requise pour le contenu HTMLView.

## <a name="no-htmlview-support-in-earlier-versions-of-windows-media-player"></a>aucune prise en charge de HTMLView dans les versions antérieures de Lecteur Windows Media

actuellement, seule la série Lecteur Windows Media 9 ou version ultérieure prend en charge la fonctionnalité HTMLView. Les versions antérieures du lecteur et les versions créées pour d’autres plateformes ignorent le paramètre HTMLView lors de l’analyse d’un fichier. asx. Dans ce cas, le contenu multimédia numérique s’exécute comme prévu, mais le contenu Web n’est pas affiché dans la fonctionnalité de **lecture** en cours.
