# <a name="playlistallowcolumnsorting"></a>PLAYLIST. allowColumnSorting

L’attribut **allowColumnSorting** spécifie ou récupère une valeur indiquant si le contenu des colonnes de tri est autorisé.

``` syntax
        elementID.allowColumnSorting
```

## <a name="possible-values"></a>Valeurs possibles

Cet attribut est une **valeur booléenne** en lecture/écriture.



| Valeur | Description                                  |
|-------|----------------------------------------------|
| true  | Par défaut. Le tri du contenu des colonnes est autorisé. |
| false | Le tri du contenu des colonnes n’est pas autorisé.      |



 

## <a name="requirements"></a>Spécifications



| Condition requise | Valeur |
|--------------------|------------------------------------------------------|
| Version<br/> | Lecteur Windows Media version 7,0 ou ultérieure<br/> |
