# <a name="onmousedown"></a>OnMouseDown

Le gestionnaire d’événements **OnMouseDown** gère un événement qui se produit lorsque l’utilisateur clique sur un bouton de la souris.

``` syntax
onmousedown
```

## <a name="remarks"></a>Remarques

Ce gestionnaire d’événements n’est pas applicable à l’élément **playlist** , élément **Video** when *Video*. **sans fenêtre** a la valeur false, ou l’élément **Effects** en cas d' *effets*. **Windowed** a la valeur true.

## <a name="requirements"></a>Configuration requise



| Condition requise | Valeur |
|--------------------|-----------------------------------------------------|
| Version<br/> | Lecteur Windows Media version 70 ou ultérieure<br/> |
