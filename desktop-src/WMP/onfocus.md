# <a name="onfocus"></a>onfocus

Le gestionnaire d’événements **onfocus** gère un événement qui se produit lorsque l’élément reçoit le focus clavier.

``` syntax
onfocus
```

## <a name="remarks"></a>Remarques

Le gestionnaire d’événements **onfocus** n’est pas applicable à l’élément **Effects** ou à l’élément **Popup** .

## <a name="requirements"></a>Configuration requise



| Condition requise | Valeur |
|--------------------|-----------------------------------------------------|
| Version<br/> | Lecteur Windows Media version 70 ou ultérieure<br/> |
