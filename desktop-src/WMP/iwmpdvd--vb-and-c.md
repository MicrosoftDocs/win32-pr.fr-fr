# <a name="iwmpdvd-vb-and-c-interface"></a>Interface IWMPDVD (VB et C#)

L’interface **IWMPDVD** fournit des propriétés et des méthodes pour travailler avec des DVD.

L’interface **IWMPDVD** expose les propriétés suivantes.

## <a name="members"></a>Membres

L’interface **IWMPDVD (VB et C#)** possède les types de membres suivants :

-   [Méthodes](#methods)
-   [Propriétés](#properties)

### <a name="methods"></a>Méthodes

L’interface **IWMPDVD (VB et C#)** possède ces méthodes.



| Méthode                                                         | Description                                                                                                     |
|:---------------------------------------------------------------|:----------------------------------------------------------------------------------------------------------------|
| [**Précédent**](wmplibiwmpdvd-iwmpdvd-back--vb-and-c.md)           | Modifie l’affichage d’un sous-menu vers son menu parent.<br/>                                               |
| [**sort**](wmplibiwmpdvd-iwmpdvd-resume--vb-and-c.md)       | Passe en mode lecture à partir du mode menu, en reprenant à la même position que lorsque le menu a été appelé.<br/> |
| [**titleMenu**](wmplibiwmpdvd-iwmpdvd-titlemenu--vb-and-c.md) | Arrête la lecture et affiche le menu du titre.<br/>                                                          |
| [**Menu de la**](wmplibiwmpdvd-iwmpdvd-topmenu--vb-and-c.md)     | Arrête la lecture et affiche le menu racine.<br/>                                                           |



 

### <a name="properties"></a>Propriétés

L’interface **IWMPDVD (VB et C#)** a ces propriétés.



| Propriété                                                            | Type d’accès          | Description                                                                                                                                                                          |
|:--------------------------------------------------------------------|:---------------------|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Domain**](wmplibiwmpdvd-iwmpdvd-domain--vb-and-c.md)<br/> | Lecture seule<br/> | Obtient le domaine actuel du DVD.<br/>                                                                                                                                       |
| [**isAvailable**](iwmpdvd-isavailable--vb-and-c.md)<br/>     | Lecture seule<br/> | Obtient une valeur qui indique si un type spécifié d’informations est disponible ou si une action spécifiée peut être exécutée. En C#, il s’agit de la méthode d' **extraction de \_ isAvailable** .<br/> |



 

Procurez-vous une interface **IWMPDVD** à l’aide de la propriété suivante.



| Object                                                                   | Propriété                                                   |
|--------------------------------------------------------------------------|------------------------------------------------------------|
| [Objet AxWindowsMediaPlayer](axwindowsmediaplayer-object--vb-and-c.md) | [**DVD**](axwmplib-axwindowsmediaplayer-dvd--vb-and-c.md) |
