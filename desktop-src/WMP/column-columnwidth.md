# <a name="columncolumnwidth"></a>COLUMN. columnWidth

L’attribut **ColumnWidth** spécifie ou récupère une largeur de colonne dans le contrôle **playlist** .

``` syntax
        elementID.columnWidth
```

## <a name="possible-values"></a>Valeurs possibles

Cet attribut est un **nombre** en lecture/écriture (**long**) représentant la largeur de la colonne en pixels.

## <a name="remarks"></a>Notes

La propriété **columnResizeMode** doit avoir la valeur « fixed » pour que cette propriété fonctionne.

## <a name="requirements"></a>Configuration requise



| Condition requise | Valeur |
|--------------------|---------------------------------------------------|
| Version<br/> | Lecteur Windows Media série 9 ou version ultérieure<br/> |
