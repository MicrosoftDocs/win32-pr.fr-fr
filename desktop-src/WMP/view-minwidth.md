# <a name="viewminwidth"></a>VIEW. minWidth

L’attribut **minWidth** spécifie ou récupère la largeur minimale, en pixels, de la **vue** lors du redimensionnement.

``` syntax
        elementID.minWidth
```

## <a name="possible-values"></a>Valeurs possibles

Cet attribut est un **nombre** en lecture/écriture (**long**) dont la valeur est égale à zéro ou supérieure. Sa valeur par défaut est zéro, ce qui signifie qu’il n’y a aucune restriction sur la largeur minimale de la **vue**.

## <a name="requirements"></a>Configuration requise



| Condition requise | Valeur |
|--------------------|------------------------------------------------------|
| Version<br/> | Lecteur Windows Media version 7,0 ou ultérieure<br/> |
