# <a name="about-the-controls-object"></a>À propos de l’objet contrôles

L’objet **Controls** régit le transport de contenu multimédia numérique via le contrôle à l’aide de méthodes telles que **Play** et **Stop**. Elle est accessible uniquement via la propriété **Controls** de l’objet **Player** . La propriété **Controls** retourne l’objet **Controls** . Vous pouvez uniquement accéder aux propriétés de l’objet **contrôles** après l’avoir créé. Par exemple, pour utiliser la méthode **Play** , vous devez utiliser le code suivant :


```C++
player.controls.play();

```
