# <a name="creating-the-pushed-file"></a>Création du fichier ayant fait l’objet d’un push

Vous pouvez prendre les éléments que vous avez créés et les réutiliser pour créer les autres images dont vous avez besoin. En associant des couches et en coloriant les résultats, vous pouvez créer une image bitmap Push contenant les images que vous souhaitez voir apparaître lorsque l’utilisateur pousse un bouton. Ce fichier contient également l’image secondaire pour l’état suspendu du bouton PlayPause.

L’image suivante est le fichier faisant l’objet d’un push.

![fichier Push](images/ceswmpus.png)

Les deux boutons à droite correspondent à l’état de pause normal et à l’état de pause pushd du bouton PlayPause. La couleur des boutons est jaune, à l’exception de l’état de pause normal du bouton PlayPause, comme un bon contraste avec la couleur verte utilisée pour l’état normal de tous les boutons.
