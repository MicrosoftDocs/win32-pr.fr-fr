# <a name="about-imfgetservice"></a>À propos de IMFGetService

L’interface **IMFGetService** est l’une des interfaces qui doivent être implémentées par un plug-in DSP à double mode. Il n’a qu’une seule méthode, **GetService**. Un plug-in DSP implémente la méthode **GetService** afin que les clients puissent obtenir des pointeurs vers les interfaces prises en charge par le plug-in lorsque le plug-in s’exécute en mode natif dans le pipeline Media Foundation.
