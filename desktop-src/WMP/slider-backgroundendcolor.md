# <a name="sliderbackgroundendcolor"></a>Slider. backgroundEndColor

L’attribut **backgroundEndColor** spécifie ou récupère la couleur de fin d’arrière-plan du contrôle Slider.

``` syntax
        elementID.backgroundEndColor
```

## <a name="possible-values"></a>Valeurs possibles

Cet attribut est une **chaîne** en lecture/écriture contenant toute valeur de couleur Microsoft Internet Explorer. Il n'a aucune valeur par défaut.

## <a name="remarks"></a>Notes

**BackgroundEndColor** est utilisé conjointement avec un **backgroundColor**. L’effet créé est un fondu dégradé entre **backgroundColor** et **backgroundEndColor**.

## <a name="requirements"></a>Configuration requise



| Condition requise | Valeur |
|--------------------|------------------------------------------------------|
| Version<br/> | Lecteur Windows Media version 7,0 ou ultérieure<br/> |
