# <a name="organization-of-the-documentation"></a>Organisation de la documentation

le kit de développement logiciel (SDK) Lecteur Windows Media documente un large éventail de technologies liées à Lecteur Windows Media. Pour faciliter la localisation des informations dont vous avez besoin, la documentation est divisée en sections qui vous aident à trouver rapidement des rubriques sur une fonctionnalité ou une technologie particulière.

Le cas échéant, les sections principales comportent trois sous-sections : à propos de, Guide de programmation et référence. La section à propos de décrit la fonctionnalité ou la technologie particulière pour vous aider à comprendre comment vous pouvez l’utiliser. La section Guide de programmation contient des didacticiels qui vous aideront à approfondir la technologie avec des instructions pas à pas. La section de référence détaille les interfaces, les objets, les méthodes, les propriétés, les événements, les éléments et les attributs qui sont disponibles pour votre utilisation.
