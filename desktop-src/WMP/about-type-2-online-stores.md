# <a name="about-type-2-online-stores"></a>À propos des magasins en ligne de type 2

> [!Note]  
> Cette section décrit les fonctionnalités conçues pour être utilisées par les magasins en ligne. L’utilisation de cette fonctionnalité en dehors du contexte d’un magasin en ligne n’est pas prise en charge.

 

Le lecteur Windows Media prend en charge deux types de magasins en ligne : type1 et type 2. Les magasins de type 1 sont pris en charge par le lecteur Windows Media 11. Les magasins de type 2 sont pris en charge par le lecteur Windows Media 10 et le lecteur Windows Media 11. Les magasins de type 2 fournissent un niveau d’intégration de base avec l’interface utilisateur du joueur, et les magasins de type 1 offrent une intégration plus poussée. Pour plus d’informations sur la différence entre les magasins de type 1 et de type 2, consultez [Windows Media Player Online stores](windows-media-player-online-stores.md).
