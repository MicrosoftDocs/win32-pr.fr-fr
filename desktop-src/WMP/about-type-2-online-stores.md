# <a name="about-type-2-online-stores"></a>À propos des magasins en ligne de type 2

> [!Note]  
> Cette section décrit les fonctionnalités conçues pour être utilisées par les magasins en ligne. L’utilisation de cette fonctionnalité en dehors du contexte d’un magasin en ligne n’est pas prise en charge.

 

Lecteur Windows Media prend en charge deux types de magasins en ligne : type1 et type 2. les magasins de Type 1 sont pris en charge par Lecteur Windows Media 11. les magasins de Type 2 sont pris en charge par les Lecteur Windows Media 10 et Lecteur Windows Media 11. Les magasins de type 2 fournissent un niveau d’intégration de base avec l’interface utilisateur du joueur, et les magasins de type 1 offrent une intégration plus poussée. pour plus d’informations sur la différence entre les magasins de type 1 et de type 2, consultez [Lecteur Windows Media magasins en ligne](windows-media-player-online-stores.md).
