# <a name="the-constructor"></a>Le constructeur

Le constructeur fourni par l’Assistant stocke un pointeur vers la classe de plug-in principale dans une variable membre. L’Assistant utilise cette référence pour accéder aux variables membres déclarées dans la classe CSearch. Aucune modification de l’implémentation n’est nécessaire.
