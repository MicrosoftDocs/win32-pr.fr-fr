# <a name="skin-reference"></a>Référence d’apparence

Cette référence est divisée en sections suivantes.



| Section                        | Description                                                                     |
|--------------------------------|---------------------------------------------------------------------------------|
| [Pochette de l’album](album-art.md)     | Informations sur l’inclusion d’une pochette d’album dans des apparences.                                 |
| [Images bitmap](bitmaps.md)         | Informations sur les fichiers art utilisés dans la création de l’apparence.                          |
| [Boutons](buttons.md)         | Informations sur les contrôles PUSHBUTTON disponibles dans les apparences.                       |
| [Description](description.md) | Informations sur la spécification des dimensions et de l’orientation de la peau.                   |
| [Chapiteau](marquee.md)         | Informations sur l’utilisation des zones de texte déroulantes dans les apparences.                          |
| [Évaluations](ratings.md) \*      | Informations sur la valeur de classement affectée à un élément multimédia.                   |
| [État](status.md)           | Informations sur l’inclusion d’un état d’affichage dans les apparences.                          |
| [Text](text.md)               | Informations sur l’utilisation des zones de texte pour afficher des informations dans les apparences.             |
| [Trackbars](trackbars.md)     | Informations sur l’utilisation des contrôles TrackBar coulissants pour la position du volume et de la recherche. |
| [Vidéo](video.md)             | Informations sur l’insertion d’une zone d’affichage vidéo dans les apparences.                      |



 

\*nouveauté de Lecteur Windows Media Mobile 10,1.

Chaque section correspond à une section du fichier de définition d’apparence que vous devez fournir pour utiliser des éléments spécifiques. Les sections sont répertoriées dans l’ordre alphabétique. Pour connaître l’ordre dans lequel ils doivent apparaître dans un fichier de définition d’apparence, consultez [création d’un fichier de définition d’apparence](creating-a-skin-definition-file.md). Pour plus d’informations sur la composition du fichier de définition d’apparence, reportez-vous à la section [à propos des apparences](about-skins-mobile.md) .
