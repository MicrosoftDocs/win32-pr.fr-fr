# <a name="coordinates"></a>Coordinates

Cela définit les coordonnées x et y de l’image par rapport au fichier d’arrière-plan. Deux entiers positifs sont requis, séparés par une virgule. Il existe deux types de coordonnées que vous devez utiliser, selon le type d’image que vous définissez. Le tableau suivant indique les valeurs de coordonnées autorisées pour les types bitmap.



| Type de bitmap               | Valeurs de coordonnées                                                             |
|---------------------------|-------------------------------------------------------------------------------|
| Arrière-plan ou Super       | 0,0                                                                           |
| Désactivé, push ou région | L’angle supérieur gauche de l’image par rapport à l’image d’arrière-plan, en pixels. |
