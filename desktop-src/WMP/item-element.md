# <a name="item-element"></a>Élément ITEM

L’élément **Item** représente un élément dans une zone de liste ou des contrôles contextuels. Les éléments **Item** sont toujours des enfants des éléments **ListBox** ou **Popup** .

L’élément **Item** prend en charge les attributs suivants.



| Attribut               | Description                                                                  |
|-------------------------|------------------------------------------------------------------------------|
| [value](item-value.md) | Spécifie ou récupère le texte qui s’affiche pour une zone de liste ou un élément contextuel. |



 

> [!Note]  
> Cet élément requiert le lecteur Windows Media pour Windows XP ou version ultérieure.
