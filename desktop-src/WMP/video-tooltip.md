# <a name="videotooltip"></a>VIDÉO. info-bulle

L’attribut **ToolTip** spécifie ou récupère le texte d’info-bulle pour la fenêtre vidéo.

``` syntax
        elementID.toolTip
```

## <a name="possible-values"></a>Valeurs possibles

Cet attribut est une **chaîne** en lecture/écriture, qui ne doit pas dépasser 1024 caractères. Il n'a aucune valeur par défaut.

## <a name="remarks"></a>Notes

Lorsque cet attribut a la valeur "" (chaîne vide), aucune info-bulle n’est affichée.

## <a name="requirements"></a>Configuration requise



| Condition requise | Valeur |
|--------------------|------------------------------------------------------|
| Version<br/> | Lecteur Windows Media version 7,0 ou ultérieure<br/> |
