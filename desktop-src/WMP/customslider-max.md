# <a name="customslidermax"></a>CUSTOMSLIDER. max.

L’attribut **Max** spécifie ou récupère la valeur maximale de la plage définie par le curseur personnalisé.

``` syntax
        elementID.max
```

## <a name="possible-values"></a>Valeurs possibles

Cet attribut est un **nombre** en lecture/écriture (**float**) avec une valeur par défaut de 100.

## <a name="remarks"></a>Notes

La valeur de **Max** doit être supérieure à celle de **min**.

## <a name="examples"></a>Exemples

Pour obtenir un exemple illustrant l’utilisation des attributs de l’élément **CUSTOMSLIDER** , consultez l’attribut [positionImage](customslider-positionimage.md) .
