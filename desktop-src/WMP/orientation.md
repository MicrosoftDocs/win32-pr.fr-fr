# <a name="orientation"></a>Orientation

Certains périphériques Windows Mobile peuvent être utilisés dans différentes orientations physiques. Les orientations disponibles sont les suivantes :

-   **Portrait**. L’appareil est maintenu en mode vertical pour qu’il soit plus haut que grand.
-   **Paysage**. L’appareil est maintenu de manière horizontale afin d’être plus grand que sa hauteur.
-   **Carré**. Certains appareils ont un affichage en forme de carré, ce qui signifie qu’il existe un mode d’orientation unique.

Chaque apparence que vous créez doit spécifier une seule orientation. Vous devez fournir une apparence distincte pour chaque orientation. Le sélecteur d’apparence affiche uniquement les apparences créées pour l’orientation et les dimensions actuelles de l’affichage de l’appareil.
