# <a name="debugging-the-page"></a>Débogage de la page

lorsque Lecteur Windows Media héberge la page web de votre boutique en ligne, il désactive la possibilité d’ouvrir des boîtes de dialogue à partir de la page web. Pour déboguer votre code de page Web, vous devez plutôt ajouter du code qui imprime temporairement la sortie vers un élément HTML sur la page Web, comme un élément DIV ou TEXT.
