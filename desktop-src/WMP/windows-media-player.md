# <a name="windows-media-player"></a>Lecteur Windows Media

le kit de développement logiciel (SDK) Lecteur Windows Media fournit des fonctionnalités qui affectent le comportement des Lecteur Windows Media et des Lecteur Windows Media pour Windows Mobile.

les sections suivantes décrivent les fonctionnalités du kit de développement logiciel (SDK) qui s’appliquent à Lecteur Windows Media en général.



| Section                                                                                                        | Description                                                                                                                                                           |
|----------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Paramètres du Registre](registry-settings.md)                                                                     | les valeurs de détails que vous pouvez modifier dans le registre de l’utilisateur pour permettre à Lecteur Windows Media de reconnaître les extensions de nom de fichier personnalisées.                                    |
| [Paramètres de ligne de commande](command-line-parameters.md)                                                         | détaille le jeu de paramètres de ligne de commande qui spécifient comment Lecteur Windows Media se comporte au démarrage.                                                              |
| [Sorties audio](audio-outputs.md)                                                                             | décrit comment Lecteur Windows Media et le contrôle Lecteur Windows Media ActiveX choisir des appareils de sortie audio par défaut.                                                  |
| [redistribution du logiciel Lecteur Windows Media](redistributing-windows-media-player-software.md)               | fournit des informations sur la redistribution des Lecteur Windows Media.                                                                                                       |
| [publication de codecs pour Lecteur Windows Media](publishing-codecs-for-windows-media-player.md)                   | La publication de votre codec sur le site Web WMPlugins n’est plus prise en charge.                                                                                                |
| [Redimensionnement de la boîte de dialogue acquisition de licence](resizing-the-license-acquisition-dialog-box.md)                 | décrit comment modifier un attribut de fichier de média Windows pour spécifier une taille pour la boîte de dialogue Lecteur Windows Media 10 ou une acquisition de licence ultérieure.                     |
| [Extensions d’appareil pour le transfert de métadonnées accéléré](device-extensions-for-accelerated-metadata-transfer.md) | décrit comment Lecteur Windows Media récupère des informations à partir d’appareils portables sur les modifications apportées à des éléments de contenu spécifiques entre des sessions de synchronisation. |
| [Extensions d’appareil pour la création de rapports sur le contenu acquis](device-extensions-for-reporting-acquired-content.md)       | décrit comment Lecteur Windows Media peut récupérer une liste de nouveaux contenus acquis par un périphérique portable entre des sessions de synchronisation.                                 |
| [Extensions d’appareil pour les préférences de l’objet playlist](device-extensions-for-playlist-object-preferences.md)     | décrit comment un appareil mobile peut spécifier les types d’objets de sélection Lecteur Windows Media doit copier sur l’appareil lors de la synchronisation automatique.                |
| [Codes de contrôle d’e/s de périphérique](device-i-o-control-codes.md)                                                       | répertorie les codes de contrôle d’e/s de périphérique utilisés par Lecteur Windows Media 10 ou version ultérieure pour communiquer avec les périphériques portables via le kit de développement logiciel (SDK) Windows Media Gestionnaire de périphériques.            |
| [Prise en charge d’images personnalisées pour les appareils](custom-image-support-for-devices.md)                                       | décrit deux fichiers image que les fabricants de périphériques portables peuvent créer pour personnaliser la personnalisation dans Lecteur Windows Media 10 ou version ultérieure.                                    |
