# <a name="windows-media-player"></a>Lecteur Windows Media

Le kit de développement logiciel (SDK) du lecteur Windows Media fournit des fonctionnalités qui affectent le comportement du lecteur Windows Media et du lecteur Windows Media pour Windows Mobile.

Les sections suivantes détaillent les fonctionnalités du kit de développement logiciel (SDK) qui s’appliquent au lecteur Windows Media en général.



| Section                                                                                                        | Description                                                                                                                                                           |
|----------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Paramètres du Registre](registry-settings.md)                                                                     | Les valeurs de détails que vous pouvez modifier dans le registre de l’utilisateur pour permettre au lecteur Windows Media de reconnaître les extensions de nom de fichier personnalisées.                                    |
| [Paramètres de ligne de commande](command-line-parameters.md)                                                         | Décrit en détail l’ensemble des paramètres de ligne de commande qui spécifient le comportement du lecteur Windows Media lorsqu’il démarre.                                                              |
| [Sorties audio](audio-outputs.md)                                                                             | Décrit comment le lecteur Windows Media et le contrôle ActiveX du lecteur Windows Media choisissent les périphériques de sortie audio par défaut.                                                  |
| [Redistribution du logiciel Windows Media Player](redistributing-windows-media-player-software.md)               | Fournit des informations sur la redistribution du lecteur Windows Media.                                                                                                       |
| [Publication de codecs pour le lecteur Windows Media](publishing-codecs-for-windows-media-player.md)                   | La publication de votre codec sur le site Web WMPlugins n’est plus prise en charge.                                                                                                |
| [Redimensionnement de la boîte de dialogue acquisition de licence](resizing-the-license-acquisition-dialog-box.md)                 | Décrit comment modifier un attribut de fichier Windows Media pour spécifier une taille pour la boîte de dialogue acquisition de licence du lecteur Windows Media 10 ou version ultérieure.                     |
| [Extensions d’appareil pour le transfert de métadonnées accéléré](device-extensions-for-accelerated-metadata-transfer.md) | Décrit comment le lecteur Windows Media récupère des informations à partir d’appareils portables sur les modifications apportées à des éléments de contenu spécifiques entre des sessions de synchronisation. |
| [Extensions d’appareil pour la création de rapports sur le contenu acquis](device-extensions-for-reporting-acquired-content.md)       | Décrit comment le lecteur Windows Media peut récupérer une liste de nouveaux contenus acquis par un périphérique portable entre les sessions de synchronisation.                                 |
| [Extensions d’appareil pour les préférences de l’objet playlist](device-extensions-for-playlist-object-preferences.md)     | Décrit comment un appareil mobile peut spécifier les types d’objets playlist que le lecteur Windows Media doit copier sur l’appareil lors de la synchronisation automatique.                |
| [Codes de contrôle d’e/s de périphérique](device-i-o-control-codes.md)                                                       | Répertorie les codes de contrôle d’e/s de périphérique utilisés par le lecteur Windows Media 10 ou version ultérieure pour communiquer avec les appareils mobiles via le kit de développement logiciel (SDK) Windows Media Gestionnaire de périphériques.            |
| [Prise en charge d’images personnalisées pour les appareils](custom-image-support-for-devices.md)                                       | Décrit deux fichiers image que les fabricants de périphériques portables peuvent créer pour personnaliser la personnalisation dans le lecteur Windows Media 10 ou version ultérieure.                                    |
