# <a name="equalizersettingspresettitle"></a>EQUALIZERSETTINGS.presetTitle

La méthode **presetTitle** récupère le nom de la présélection de l’égaliseur avec l’index spécifié.

``` syntax
        elementID.presetTitle(index)
```

## <a name="parameters"></a>Paramètres

<dl> <dt>

<span id="index"></span><span id="INDEX"></span>*évaluer*
</dt> <dd>

**Nombre** (**long**) contenant l’index prédéfini.

</dd> </dl>

## <a name="return-value"></a>Valeur renvoyée

Cette méthode retourne une **chaîne**.

## <a name="requirements"></a>Configuration requise



| Condition requise | Valeur |
|--------------------|---------------------------------------------------|
| Version<br/> | Lecteur Windows Media série 9 ou version ultérieure<br/> |
