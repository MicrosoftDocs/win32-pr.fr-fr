# <a name="installing-your-skin"></a>Installation de votre apparence

Utilisez la procédure suivante pour installer votre apparence :

1.  Copiez tous vos fichiers d’apparence dans un nouveau dossier.
2.  Copiez le dossier contenant vos fichiers d’apparence dans le dossier lecteur Windows Media sur votre appareil mobile.

    Démarrez le lecteur Windows Media Mobile. Cela lance le **Sélecteur d’apparence**, qui comporte deux boutons de navigation qui vous permettent de faire défiler les habillages disponibles et d’afficher un aperçu de chacun d’eux. Le nom affiché pour votre apparence sera le nom de votre fichier de définition d’apparence.

3.  Appuyez sur **OK** pour sélectionner votre apparence.
