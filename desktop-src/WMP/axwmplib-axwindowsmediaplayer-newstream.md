# <a name="newstream-event-of-the-axwindowsmediaplayer-object"></a>Événement NewStream de l’objet AxWindowsMediaPlayer

L’événement NewStream est réservé pour une utilisation ultérieure.

``` syntax
[C#]
private void player_NewStream(
  object sender,
  System.EventArgs e
)

[Visual Basic]
Private Sub player_NewStream(  
  sender As Object,
  e As System.EventArgs
) Handles player.NewStream
```

## <a name="event-data"></a>Données d'événements

Cet événement ne contient pas de données.

## <a name="remarks"></a>Remarques

Cet événement est réservé à une utilisation ultérieure.
