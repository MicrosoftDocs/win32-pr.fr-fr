# <a name="iwmpmetadatatexttext-property"></a>IWMPMetadataText :: Text, propriété

La `text` propriété obtient le texte des métadonnées.

## <a name="syntax"></a>Syntaxe


```CSharp
public System.String text {get; set;}
```


```VB

Public ReadOnly Property text As System.String
```





## <a name="property-value"></a>Valeur de la propriété

**System. String** qui est le texte des métadonnées.

## <a name="remarks"></a>Notes

Avant d’utiliser cette propriété, vous devez disposer d’un accès en lecture à la bibliothèque. Pour plus d’informations, consultez [accès à la bibliothèque](library-access.md).
