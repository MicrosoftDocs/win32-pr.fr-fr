# <a name="trackbars-section"></a>Section Trackbars

La section Trackbars contient des informations sur les Trackbars que vous souhaitez inclure :


```C++
[ Trackbars ]

//  <Function>  <Location>     <Dis Image Src>  <Thumb Image Src>  <Thumb Size>
//  ----------  ----------     ---------------  -----------------  ------------
    Volume      40,200,170,40  Disabled @ 15,10    VolumeThumb.bmp    23,22

```



Seul le TrackBar du volume est défini ici. Pour plus d’informations sur la section TrackBar du fichier de définition d’apparence, consultez [Trackbars](trackbars.md) dans la référence de l’apparence.
