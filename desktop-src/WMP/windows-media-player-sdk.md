# <a name="windows-media-player-sdk"></a>SDK du lecteur Windows Media

Cette documentation décrit le kit de développement logiciel (SDK) de Microsoft Windows Media Player. Le kit de développement logiciel (SDK) du lecteur Windows Media est l’un des composants de la Microsoft Windows SDK. Les autres composants multimédias incluent le SDK Microsoft Media Foundation, le kit de développement logiciel (SDK) Microsoft Windows Media format et le kit de développement logiciel (SDK) Microsoft Windows Media Services.

Le kit de développement logiciel (SDK) du lecteur Windows Media documente les technologies de programmation qui peuvent être utilisées pour étendre les fonctionnalités du lecteur Windows Media et de Windows Media Player Mobile. Ces technologies sont documentées dans les sections suivantes :



| Section                                                                      | Description                                                                                                                                                                                                                                                                                                                       |
|------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [À propos du kit de développement logiciel (SDK) Windows Media Player](about-the-windows-media-player-sdk.md) | Cette section fournit des informations détaillées sur la façon de trouver des informations spécifiques dans le kit de développement logiciel (SDK). Il inclut une section sur les nouvelles fonctionnalités et les informations relatives à l’utilisation des exemples inclus dans le kit de développement logiciel (SDK).                                                                                                                                     |
| [Modèle objet du lecteur Windows Media](windows-media-player-object-model.md)   | Le contrôle Microsoft Windows Media Player est un contrôle Microsoft ActiveX utilisé pour ajouter des fonctionnalités de lecture de médias numériques à des pages Web. Il fournit une interface de programmation pour le rendu des fichiers multimédias et des flux numériques.                                                                                                     |
| [Apparences du lecteur Windows Media](windows-media-player-skins.md)                 | Les apparences sont une technologie XML utilisée pour personnaliser l’interface utilisateur du lecteur Windows Media. Vous pouvez également utiliser les apparences mobiles du lecteur Windows Media pour personnaliser l’interface utilisateur du lecteur Windows Media Mobile.                                                                                                                |
| [Plug-ins du lecteur Windows Media](windows-media-player-plug-ins.md)           | Les plug-ins sont des objets qui étendent les fonctionnalités du lecteur Windows Media de différentes manières. Les types de plug-in incluent les visualisations personnalisées, les plug-ins d’interface utilisateur, les plug-ins DSP et les plug-ins de rendu. La prise en charge du plug-in Windows Media Player Mobile est également décrite dans cette section.                                              |
| [Fichiers Windows Media](windows-media-metafiles.md)                       | Les fichiers Windows Media sont des documents XML qui fournissent des informations sur un élément multimédia et sa présentation. | [Sélections Windows Media](windows-media-playlists.md)                       | Les sélections sont des fichiers qui utilisent des éléments XML pour définir une sélection « intelligente » dynamique d’éléments multimédias ou un ensemble statique d’éléments multimédias.                                                                                                                                                                                              |
| [Windows Media Player Online stores](windows-media-player-online-stores.md) | Le lecteur Windows Media fournit des fonctionnalités qui permettent aux fournisseurs de contenus multimédias numériques d’intégrer leurs services au lecteur Windows Media. L’intégration entre le lecteur et un magasin de supports numériques en ligne permet à l’utilisateur de rechercher du contenu, de télécharger et de gérer des fichiers, de lire du contenu et de copier du contenu sur des CD ou des appareils. |
| [Lecteur Windows Media](windows-media-player.md)                             | Certaines fonctionnalités du kit de développement logiciel (SDK) s’appliquent au lecteur Windows Media, au contrôle ActiveX du lecteur Windows Media et au lecteur Windows Media Mobile. Cette section fournit des informations sur ces fonctionnalités.                                                                                                                                        |
| [Glossaire](glossary.md)                                                     | Cette section contient les définitions des termes utilisés dans le kit de développement logiciel (SDK).                                                                                                                                                                                                                                                               |



 

> [!Note]  
> L’installation du SDK Windows n’installe pas le lecteur Windows Media ni Windows Media Player Mobile. Le lecteur Windows Media doit être installé pour pouvoir utiliser les éléments de ce kit de développement logiciel (SDK). Windows Media Player Mobile fait partie de Windows Mobile, qui est disponible uniquement sur un Pocket PC ou un smartphone fourni par un fabricant d’appareils ou un opérateur mobile.
