# <a name="about-windows-media-player-sdk-versions"></a>à propos des Versions Lecteur Windows Media SDK

la version actuelle du kit de développement logiciel (SDK) Lecteur Windows Media est la version 12. le kit de développement logiciel (sdk) Lecteur Windows Media 12 s’appuie sur les fonctionnalités introduites dans les versions précédentes du kit de développement logiciel (sdk) Lecteur Windows Media, depuis la version 7. les informations contenues dans cette documentation incluent les fonctionnalités des versions suivantes du kit de développement logiciel (SDK) Lecteur Windows Media :

-   SDK Lecteur Windows Media 7
-   kit de développement logiciel (SDK) Lecteur Windows Media 7,1
-   kit de développement logiciel (SDK) Lecteur Windows Media série 9
-   SDK Lecteur Windows Media 10
-   SDK Lecteur Windows Media 11
-   kit de développement logiciel (SDK) Lecteur Windows Media 12

chaque version du kit de développement logiciel (SDK) est associée à une version de Lecteur Windows Media. par exemple, le kit de développement logiciel (SDK) Lecteur Windows Media 10 a introduit de nouvelles fonctionnalités prises en charge par Lecteur Windows Media 10. notez qu’aucune mise à jour du kit de développement logiciel (SDK) Lecteur Windows Media n’a été publiée avec Lecteur Windows Media pour Windows XP.

Tout au long de la documentation, des exigences de versioning sont fournies pour chaque fonctionnalité, le cas échéant. si aucune information de version spécifique n’est fournie, la fonctionnalité a été introduite dans le kit de développement logiciel (SDK) Lecteur Windows Media 7 et est prise en charge dans toutes les versions ultérieures.

pour plus d’informations sur la façon de mettre à niveau des projets basés sur le kit de développement logiciel (SDK) Lecteur Windows Media 6,4 pour utiliser le modèle objet pour Lecteur Windows Media 7 ou version ultérieure, consultez [Guide de Migration du modèle d’objet](object-model-migration-guide.md).
