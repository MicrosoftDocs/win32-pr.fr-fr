# <a name="about-windows-media-player-sdk-versions"></a>À propos des versions SDK du lecteur Windows Media

La version actuelle du kit de développement logiciel (SDK) du lecteur Windows Media est la version 12. Le kit de développement logiciel (SDK) du lecteur Windows Media 12 s’appuie sur les fonctionnalités introduites dans les versions précédentes du kit de développement logiciel (SDK) du lecteur Windows Media, depuis la version 7. Les informations contenues dans cette documentation incluent les fonctionnalités des versions suivantes du kit de développement logiciel (SDK) Windows Media Player :

-   Windows Media Player 7 SDK
-   Windows Media Player 7,1 SDK
-   Kit de développement logiciel Windows Media Player 9 Series
-   Windows Media Player 10 SDK
-   Windows Media Player 11 SDK
-   Kit de développement logiciel (SDK) lecteur Windows Media 12

Chaque version du kit de développement logiciel (SDK) est associée à une version du lecteur Windows Media. Par exemple, le kit de développement logiciel (SDK) du lecteur Windows Media 10 a introduit de nouvelles fonctionnalités prises en charge par le lecteur Windows Media 10. Notez qu’aucune mise à jour du kit de développement logiciel (SDK) du lecteur Windows Media n’a été publiée avec le lecteur Windows Media pour Windows XP.

Tout au long de la documentation, des exigences de versioning sont fournies pour chaque fonctionnalité, le cas échéant. Si aucune information de version spécifique n’est fournie, la fonctionnalité a été introduite dans le kit de développement logiciel (SDK) lecteur Windows Media 7 et est prise en charge dans toutes les versions ultérieures.

Pour plus d’informations sur la mise à niveau de projets basés sur le kit de développement logiciel (SDK) Windows Media Player 6,4 afin d’utiliser le modèle objet pour le lecteur Windows Media 7 ou version ultérieure, consultez [Guide de migration du modèle d’objet](object-model-migration-guide.md).
