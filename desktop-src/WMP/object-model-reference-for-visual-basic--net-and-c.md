# <a name="object-model-reference-for-visual-basic-net-and-c"></a>référence du modèle objet pour Visual Basic .net et C #

> [!Note]  
> l’utilisation du contrôle Lecteur Windows Media ActiveX avec Visual Basic .net et C# est prise en charge uniquement pour le kit de développement logiciel (SDK) Lecteur Windows Media série 9 et versions ultérieures.

 

cette section de référence fournit la documentation de l’objet **AxWindowsMediaPlayer** et des interfaces au contrôle Lecteur Windows Media ActiveX. cette fonctionnalité est disponible lorsque vous utilisez le contrôle de Lecteur Windows Media ActiveX par le biais de l’interopérabilité COM dans une solution Visual Studio .net 2002 ou ultérieure.

pour une démonstration de l’utilisation du contrôle de Lecteur Windows Media avec le .NET Framework, consultez les exemples Visual Basic et C# inclus dans ce kit de développement logiciel (SDK).

les sections suivantes documentent les objets et les interfaces Visual Basic .net et C# fournis par l’assembly COM Interop créé par Visual Studio.

> [!Note]  
> les fonctionnalités qui requièrent la communication à distance du contrôle Lecteur Windows Media ne sont pas prises en charge pour la programmation .net.
