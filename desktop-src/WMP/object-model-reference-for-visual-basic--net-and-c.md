# <a name="object-model-reference-for-visual-basic-net-and-c"></a>Référence du modèle objet pour Visual Basic .NET et C #

> [!Note]  
> L’utilisation du contrôle ActiveX du lecteur Windows Media avec Visual Basic .NET et C# est prise en charge uniquement pour le kit de développement logiciel (SDK) du lecteur Windows Media série 9 et versions ultérieures.

 

Cette section de référence fournit la documentation de l’objet **AxWindowsMediaPlayer** et des interfaces au contrôle ActiveX du lecteur Windows Media. Cette fonctionnalité est disponible lorsque vous utilisez le contrôle ActiveX du lecteur Windows Media via l’interopérabilité COM dans une solution Visual Studio .NET 2002 ou ultérieure.

Pour une démonstration de l’utilisation du contrôle du lecteur Windows Media avec le .NET Framework, consultez les exemples Visual Basic et C# inclus dans ce kit de développement logiciel (SDK).

Les sections suivantes documentent les objets et interfaces Visual Basic .NET et C# fournis par l’assembly COM Interop créé par Visual Studio.

> [!Note]  
> Les fonctionnalités qui requièrent la communication à distance du contrôle du lecteur Windows Media ne sont pas prises en charge pour la programmation .NET.

| [Interfaces pour Visual Basic .NET et C #](interfaces-for-visual-basic--net-and-c.md) | Répertorie les interfaces disponibles pour les programmeurs .NET qui utilisent le contrôle ActiveX du lecteur Windows Media.                           |
