# <a name="reference-for-type-1-online-stores"></a>Référence pour les magasins en ligne de type 1

> [!Note]  
> Cette section décrit les fonctionnalités conçues pour être utilisées par les magasins en ligne. L’utilisation de cette fonctionnalité en dehors du contexte d’un magasin en ligne n’est pas prise en charge.

 

Lecteur Windows Media 11 fournit un compilateur de catalogue, un objet et plusieurs interfaces qui prennent en charge les magasins de type 1 en ligne. Les sections suivantes documentent ces informations en détail.



| Section                                                                                    | Description                                                                                                                                                                                                                        |
|--------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Compilateur de catalogue pour les magasins en ligne de type 1](catalog-compiler-for-type-1-online-stores.md) | Fournit des documents de référence pour le compilateur qu’un magasin en ligne utilise pour compiler son catalogue musical.                                                                                                                               |
| [Interface IWMPContentContainer](/previous-versions/windows/desktop/api/contentpartner/nn-contentpartner-iwmpcontentcontainer)                                 | Fournit des pages de référence pour les méthodes que le plug-in du magasin en ligne appelle pour inspecter un conteneur de contenu. Un conteneur de contenu représente un ensemble d’éléments multimédias à télécharger ou à acheter. implémenté par Lecteur Windows Media. |
| [Interface IWMPContentContainerList](/previous-versions/windows/desktop/api/contentpartner/nn-contentpartner-iwmpcontentcontainerlist)                         | Fournit des pages de référence pour les méthodes que le plug-in du magasin en ligne appelle pour inspecter une liste de conteneurs de contenu. implémenté par Lecteur Windows Media.                                                                           |
| [Interface IWMPContentPartner](/previous-versions/windows/desktop/api/contentpartner/nn-contentpartner-iwmpcontentpartner)                                     | fournit des pages de référence pour les méthodes que Lecteur Windows Media appelle pour obtenir des informations à partir du magasin en ligne et pour informer le magasin en ligne des actions de l’utilisateur. Implémenté par le plug-in du magasin en ligne.                  |
| [Interface IWMPContentPartnerCallback](/previous-versions/windows/desktop/api/contentpartner/nn-contentpartner-iwmpcontentpartnercallback)                     | fournit des pages de référence pour les méthodes que le plug-in du magasin en ligne appelle pour notifier Lecteur Windows Media sur l’achèvement des opérations asynchrones. implémenté par Lecteur Windows Media.                                    |
| [Objet externe pour les magasins de type 1 en ligne](external-object-for-type-1-online-stores.md)   | Fournit des pages de référence pour les propriétés, les méthodes et les événements utilisés par le script dans les pages Web fournies par le magasin en ligne.                                                                                                              |
| [Énumérations pour les magasins de type 1 en ligne](enumerations-for-type-1-online-stores.md)         | fournit des pages de référence pour les énumérations utilisées dans la communication entre Lecteur Windows Media et le plug-in du magasin en ligne.                                                                                               |
| [Structures pour les magasins de type 1 en ligne](structures-for-type-1-online-stores.md)             | fournit des pages de référence pour les structures utilisées dans la communication entre Lecteur Windows Media et le plug-in du magasin en ligne.                                                                                                 |
