# <a name="videomaintainaspectratio"></a>VIDÉO. maintainAspectRatio

L’attribut **maintainAspectRatio** spécifie ou récupère une valeur indiquant si la vidéo conserve ses proportions lors de la tentative d’ajustement de la largeur et de la hauteur définies pour le contrôle.

``` syntax
        elementID.maintainAspectRatio
```

## <a name="possible-values"></a>Valeurs possibles

Cet attribut est une **valeur booléenne** en lecture/écriture.



| Valeur | Description                                              |
|-------|----------------------------------------------------------|
| true  | Par défaut. La vidéo conserve ses proportions lors du redimensionnement. |
| false | La vidéo ne conserve pas ses proportions lors du redimensionnement.  |



 

## <a name="requirements"></a>Spécifications



| Condition requise | Valeur |
|--------------------|------------------------------------------------------|
| Version<br/> | Lecteur Windows Media version 7,0 ou ultérieure<br/> |
