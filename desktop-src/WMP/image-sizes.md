# <a name="image-sizes"></a>Tailles d’image

Vous devez prendre en compte la taille de la zone d’affichage lors de la création d’illustrations pour les habillages. Le tableau suivant indique la taille de la zone d’affichage pour différentes configurations d’écran.



| DPI (mode)                    | Largeur en pixels | Hauteur en pixels |
|-------------------------------|-----------------|------------------|
| 96 (portrait)                 | 240             | 320              |
| 96 (paysage)                | 320             | 240              |
| 192 (portrait)                | 480             | 640              |
| 192 (paysage)               | 640             | 220              |
| 96 (carré)                   | 240             | 240              |
| 192 (carré)                  | 480             | 480              |
| 96 (portrait pour smartphone)  | 176             | 220              |
| 131 (portrait pour smartphone) | 240             | 320              |



 

> [!Note]  
> Les versions antérieures à Windows Media Player 9 Series pour Windows Mobile 2003 prennent uniquement en charge les habillages en mode Portrait de 96 ppp. Le lecteur Windows Media 10 mobile ou version ultérieure prend en charge les habillages en mode portrait 131 PPP sur les périphériques Smartphone, et cette prise en charge est prévue dans les prochaines versions de Windows Media Player Mobile.
