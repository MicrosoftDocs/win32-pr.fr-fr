# <a name="externaltask"></a>External. Task

> [!Note]  
> Cette rubrique décrit les fonctionnalités conçues pour être utilisées par les magasins en ligne. L’utilisation de cette fonctionnalité en dehors du contexte d’un magasin en ligne n’est pas prise en charge.

 

La propriété de **tâche** récupère le nom du volet de tâches actuel.

## <a name="syntax"></a>Syntaxe

``` syntax
window.external.task
```

## <a name="possible-values"></a>Valeurs possibles

Cette propriété est une **chaîne** en lecture seule. Les valeurs possibles sont parcourir, graver et synchroniser.
