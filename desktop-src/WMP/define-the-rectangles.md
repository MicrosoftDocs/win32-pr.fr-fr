# <a name="define-the-rectangles"></a>Définir les rectangles

Étant donné qu’aucun élément d’interface utilisateur ne peut se chevaucher, il est utile de définir les rectangles dans lesquels chaque élément sera dessiné. En outre, la création de rectangles s’avère utile lorsque vous devez écrire les coordonnées.

Activez le quadrillage Photoshop avec les lignes de grille tous les dix pixels. Activez l’alignement sur la grille pour faciliter le dessin du rectangle.

Créez une couche et dessinez des rectangles. En regroupant la couche d’esquisse, vous connaissez l’emplacement général où vous souhaitez l’élément d’interface utilisateur, mais vous pouvez maintenant ajuster l’emplacement exact. Vous pouvez toujours modifier les éléments ultérieurement, mais cela vous permettra d’obtenir un accès aussi proche que possible. L’illustration suivante montre plusieurs rectangles dans une couche.

![couche de rectangle](images/ceswmrek.png)

Maintenant que les rectangles sont définis, vous pouvez désactiver la couche d’esquisse.
