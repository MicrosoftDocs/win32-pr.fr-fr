# <a name="playing-digital-media-from-online-store-web-pages"></a>Diffusion de contenus multimédias numériques à partir de pages Web de boutique en ligne

Vous pouvez lire du contenu multimédia numérique à partir de vos pages Web de boutique en ligne. incorporez simplement le contrôle Lecteur Windows Media comme décrit dans [utilisation du contrôle Lecteur Windows Media dans une Page Web](using-the-windows-media-player-control-in-a-web-page.md). lorsque vous effectuez cette opération dans un volet de tâches de service, le contrôle incorporé se connecte au programme Lecteur Windows Media de manière similaire au mode distant (consultez communication à distance [du contrôle Lecteur Windows Media](remoting-the-windows-media-player-control.md)). Cela signifie que les utilisateurs peuvent utiliser les contrôles de transport du lecteur en mode complet pour contrôler la lecture des médias numériques dans la page Web.

Nous vous recommandons de définir **UIMODE** sur « None » ou « invisible » lors de l’incorporation du contrôle Player dans une page Web du volet Office service.
