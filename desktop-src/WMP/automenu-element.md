# <a name="automenu-element"></a>Élément automenu

L’élément **automenu** offre un moyen d’afficher le **panneau d’accès rapide** dans une apparence. Il s’agit d’un menu qui s’affiche en mode complet du lecteur lorsque vous cliquez sur la flèche à droite du bouton **lecture** en cours. Il fournit un accès immédiat aux fichiers multimédias numériques organisés de différentes façons, par exemple dans les sélections ou par album, artiste ou genre. Utilisez les attributs **Left** et **Top** pour spécifier l’emplacement où le menu doit apparaître lorsque la méthode **Show** est appelée.

L’élément **automenu** prend en charge la méthode suivante.



| Méthode                    | Description                          |
|---------------------------|--------------------------------------|
| [show](automenu-show.md) | Affiche le **panneau d’accès rapide**. |



 

L’élément **automenu** prend en charge les attributs ambiants suivants : gauche et haut.

> [!Note]  
> cet élément requiert Lecteur Windows Media série 9 ou une version ultérieure.
