# <a name="windows-media-player-sdk-enumeration-types"></a>Types d’énumération du kit de développement logiciel Windows Media Player

Le kit de développement logiciel (SDK) Windows Media Player prend en charge les types d’énumération suivants.



| Type d'énumération                                                             | Description                                                                                                            |
|------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------|
| [WMPBurnFormat](/previous-versions/windows/desktop/api/wmp/ne-wmp-wmpburnformat)                                           | Définit les types de CD pouvant être gravés.                                                                         |
| [WMPBurnState](/previous-versions/windows/desktop/api/wmp/ne-wmp-wmpburnstate)                                             | Définit les États opérationnels possibles du lecteur Windows Media au fur et à mesure qu’il s’agit d’un CD.                                      |
| [WMPDeviceStatus](/previous-versions/windows/desktop/api/wmp/ne-wmp-wmpdevicestatus)                                       | Définit les valeurs possibles pour l’état actuel d’un appareil.                                                        |
| [WMPFolderScanState](/previous-versions/windows/desktop/api/wmp/ne-wmp-wmpfolderscanstate)                                 | Définit les États opérationnels possibles du lecteur Windows Media lors de l’analyse des dossiers de fichiers pour le contenu multimédia numérique. |
| [WMPLibraryType](/previous-versions/windows/desktop/api/wmp/ne-wmp-wmplibrarytype)                                         | Définit les types de bibliothèque possibles auxquels le lecteur Windows Media peut se connecter.                                          |
| [WMPOpenState](/previous-versions/windows/desktop/api/wmp/ne-wmp-wmpopenstate)                                             | Définit les États opérationnels possibles du lecteur Windows Media au fur et à mesure qu’il ouvre un fichier multimédia numérique.                      |
| [WMPPlaylistChangeEventType](/previous-versions/windows/desktop/api/wmp/ne-wmp-wmpplaylistchangeeventtype)                 | Définit les types de modifications qui peuvent être apportées à une sélection.                                                           |
| [WMPPlayState](/previous-versions/windows/desktop/api/wmp/ne-wmp-wmpplaystate)                                             | Définit les États opérationnels possibles du lecteur Windows Media lors de la lecture d’un fichier multimédia numérique.                      |
| [WMPRipState](/previous-versions/windows/desktop/api/wmp/ne-wmp-wmpripstate)                                               | Définit les États opérationnels possibles du lecteur Windows Media lors de l’utilisation d’un CD.                                       |
| [WMPStringCollectionChangeEventType](/previous-versions/windows/desktop/api/wmp/ne-wmp-wmpstringcollectionchangeeventtype) | Définit les types de modifications qui peuvent se produire dans une collection de chaînes.                                                    |
| [WMPSyncState](/previous-versions/windows/desktop/api/wmp/ne-wmp-wmpsyncstate)                                             | Définit les États opérationnels possibles du lecteur Windows Media lorsqu’il synchronise des médias numériques sur un appareil.          |
