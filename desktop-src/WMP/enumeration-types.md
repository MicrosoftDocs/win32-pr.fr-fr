# <a name="windows-media-player-sdk-enumeration-types"></a>Lecteur Windows Media Types d’énumération SDK

le kit de développement logiciel (SDK) Lecteur Windows Media prend en charge les types d’énumération suivants.



| Type d'énumération                                                             | Description                                                                                                            |
|------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------|
| [WMPBurnFormat](/previous-versions/windows/desktop/api/wmp/ne-wmp-wmpburnformat)                                           | Définit les types de CD pouvant être gravés.                                                                         |
| [WMPBurnState](/previous-versions/windows/desktop/api/wmp/ne-wmp-wmpburnstate)                                             | définit les états opérationnels possibles de Lecteur Windows Media lors de la gravure d’un CD.                                      |
| [WMPDeviceStatus](/previous-versions/windows/desktop/api/wmp/ne-wmp-wmpdevicestatus)                                       | Définit les valeurs possibles pour l’état actuel d’un appareil.                                                        |
| [WMPFolderScanState](/previous-versions/windows/desktop/api/wmp/ne-wmp-wmpfolderscanstate)                                 | définit les états opérationnels possibles de Lecteur Windows Media lors de l’analyse des dossiers de fichiers pour le contenu multimédia numérique. |
| [WMPLibraryType](/previous-versions/windows/desktop/api/wmp/ne-wmp-wmplibrarytype)                                         | définit les types de bibliothèque possibles auxquels Lecteur Windows Media peut se connecter.                                          |
| [WMPOpenState](/previous-versions/windows/desktop/api/wmp/ne-wmp-wmpopenstate)                                             | définit les états opérationnels possibles de Lecteur Windows Media lors de l’ouverture d’un fichier multimédia numérique.                      |
| [WMPPlaylistChangeEventType](/previous-versions/windows/desktop/api/wmp/ne-wmp-wmpplaylistchangeeventtype)                 | Définit les types de modifications qui peuvent être apportées à une sélection.                                                           |
| [WMPPlayState](/previous-versions/windows/desktop/api/wmp/ne-wmp-wmpplaystate)                                             | définit les états opérationnels possibles de Lecteur Windows Media lors de la lecture d’un fichier multimédia numérique.                      |
| [WMPRipState](/previous-versions/windows/desktop/api/wmp/ne-wmp-wmpripstate)                                               | définit les états opérationnels possibles de Lecteur Windows Media lors de l’utilisation d’un CD.                                       |
| [WMPStringCollectionChangeEventType](/previous-versions/windows/desktop/api/wmp/ne-wmp-wmpstringcollectionchangeeventtype) | Définit les types de modifications qui peuvent se produire dans une collection de chaînes.                                                    |
| [WMPSyncState](/previous-versions/windows/desktop/api/wmp/ne-wmp-wmpsyncstate)                                             | définit les états opérationnels possibles de Lecteur Windows Media lors de la synchronisation des médias numériques sur un appareil.          |
