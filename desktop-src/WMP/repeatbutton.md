# <a name="repeatbutton"></a>REPEATBUTTON

Il s’agit d’un **bouton** prédéfini avec les valeurs par défaut suivantes.

``` syntax
onclick="jscript:player.settings.setMode('loop',down);"
upToolTip="Turn Repeat On"
downToolTip="Turn Repeat Off"
down="wmpprop:player.settings.getMode('loop')"
sticky="true"
```

## <a name="remarks"></a>Notes

Cela crée un contrôle **Button** qui active ou désactive la répétition. Les info-bulles sont localisées. Toutes les propriétés de ce **bouton** peuvent être remplacées en les spécifiant explicitement.
