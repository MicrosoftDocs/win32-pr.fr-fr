# <a name="attribute-reference"></a>Référence d’attribut

la référence d’attribut Lecteur Windows Media documente des attributs de métadonnées qui présentent un intérêt pour les développeurs Lecteur Windows Media SDK. Le tableau suivant répertorie ces attributs dans différentes catégories.



| Category                                                                                       | Description                                                                                                              |
|------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------|
| [Référence alphabétique](alphabetical-attribute-reference.md)                                 | Fournit une liste alphabétique des attributs prédéfinis.                                                                  |
| [Éléments audio](audio-item-attributes.md)                                                       | Répertorie les attributs qui sont disponibles pour un élément audio dans la bibliothèque.                                                    |
| [Sélections de CD](cd-playlist-attributes.md)                                                     | Répertorie les attributs qui sont disponibles pour un CD.                                                                            |
| [Pistes de CD](cd-track-attributes.md)                                                           | Répertorie les attributs disponibles pour une piste de CD.                                                                      |
| [attributs de fichier multimédia Windows couramment utilisés](commonly-used-windows-media-file-attributes.md) | Répertorie les attributs qui sont disponibles pour un fichier multimédia numérique.                                                            |
| [DVD](dvd-attributes.md)                                                                     | Répertorie les attributs qui sont disponibles pour un DVD.                                                                           |
| [Musique Fichiers](music-file-attributes.md)                                                       | Répertorie les attributs qui sont disponibles pour un fichier musical qui ne se trouve pas dans la bibliothèque.                                         |
| [Autres éléments](other-item-attributes.md)                                                       | Répertorie les attributs qui sont disponibles pour les éléments de la bibliothèque qui ne sont pas des éléments audio, vidéo, de sélection, de photo ou de radio. |
| [Éléments de photo](photo-item-attributes.md)                                                       | Répertorie les attributs qui sont disponibles pour un élément photo dans la bibliothèque.                                                     |
| [Sélections](playlist-attributes-ref.md)                                                       | Répertorie les attributs qui sont disponibles pour les sélections récupérées en tant qu’éléments multimédias de la bibliothèque.                             |
| [Éléments radio](radio-item-attributes.md)                                                       | Répertorie les attributs qui sont disponibles pour un élément radio dans la bibliothèque.                                                     |
| [Éléments vidéo](video-item-attributes.md)                                                       | Répertorie les attributs qui sont disponibles pour un élément vidéo dans la bibliothèque.                                                     |



 

pour plus d’informations sur les attributs de métadonnées, consultez les [instructions relatives à l’utilisation des métadonnées multimédia Windows](/previous-versions/ms867702(v=msdn.10)).
