# <a name="about-custom-visualizations"></a>À propos des visualisations personnalisées

Le lecteur Windows Media fournit une interface pour créer des visualisations personnalisées. Cette section fournit une vue d’ensemble de l’architecture des visualisations, des informations sur la façon dont elles sont liées au lecteur Windows Media, ainsi que des détails techniques dont vous aurez besoin pour créer des visualisations personnalisées.

Notez que plusieurs emplacements dans le code affichent le mot effet. Les visualisations sont des effets.

Les visualisations personnalisées sont décrites plus en détail dans les rubriques suivantes.



| Rubrique                                                              | Description                                                                                |
|--------------------------------------------------------------------|--------------------------------------------------------------------------------------------|
| [Vue d’ensemble de l’utilisateur](user-overview.md)                                 | Décrit les visualisations du point de vue de l’utilisateur.                                          |
| [Vue d’ensemble du développeur](developer-overview.md)                       | Décrit les visualisations du point de vue du développeur.                                     |
| [Génération d’une visualisation](building-a-visualization.md)           | Décrit comment créer un projet de visualisation.                                           |
| [Implémentation de votre code](implementing-your-code.md)               | Décrit comment modifier l’exemple de code généré par l’Assistant de plug-in de visualisation.     |
| [Installation des visualisations](installing-visualizations.md)         | Décrit comment installer une visualisation sur l’ordinateur d’un utilisateur.                             |
| [Envoi de votre visualisation](submitting-your-visualization.md) | La distribution de votre visualisation terminée sur le site Web WMPlugins n’est plus prise en charge. |
