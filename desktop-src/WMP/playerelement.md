# <a name="playerelement"></a>PLAYERELEMENT

Il s’agit d’un **BUTTONELEMENT** prédéfini avec les valeurs par défaut suivantes.

``` syntax
onclick="jscript:player.controls.play()"
upToolTip="Play"
cursor="system"
enabled="wmpenabled:player.controls.play"
```

## <a name="remarks"></a>Notes

Toutes les propriétés de ce **BUTTONELEMENT** peuvent être remplacées en les spécifiant explicitement.

## <a name="requirements"></a>Configuration requise



| Condition requise | Valeur |
|--------------------|------------------------------------------------------|
| Version<br/> | Lecteur Windows Media version 7,0 ou ultérieure<br/> |
