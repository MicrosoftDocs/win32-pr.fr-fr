# <a name="about-the-metadatatext-object"></a>À propos de l’objet MetadataText

Certains attributs de métadonnées, tels que **WM/paroles \_ synchronisés**, fournissent du texte accompagné d’une description du texte. L’objet **MetadataText** fournit un moyen de récupérer le texte et la description séparément. Quand vous utilisez un *média*. **getItemInfoByType** pour récupérer des informations sur les attributs textuels complexes, vous récupérez un objet **MetadataText** qui a des propriétés, qui contiennent les données individuelles.
