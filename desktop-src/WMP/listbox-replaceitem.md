# <a name="listboxreplaceitem"></a>LISTBOX. replaceItem

La méthode **replaceItem** remplace le texte à l’index spécifié par le texte spécifié.

``` syntax
        elementID.replaceItem(index, text)
```

## <a name="parameters"></a>Paramètres

<dl> <dt>

<span id="index"></span><span id="INDEX"></span>*évaluer*
</dt> <dd>

**Nombre** (**long**) contenant l’index de la ligne à récupérer.

</dd> <dt>

<span id="text"></span><span id="TEXT"></span>*financière*
</dt> <dd>

**Chaîne** contenant le nouveau texte.

</dd> </dl>

## <a name="return-value"></a>Valeur renvoyée

Cette méthode ne retourne pas de valeur.

| Condition requise | Valeur |
|--------------------|---------------------------------------------------------|
| Version<br/> | Lecteur Windows Media pour Windows XP ou version ultérieure<br/> |
