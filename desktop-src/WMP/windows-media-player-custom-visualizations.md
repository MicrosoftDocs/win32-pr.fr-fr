# <a name="windows-media-player-custom-visualizations"></a>Lecteur Windows Media Visualisations personnalisées

Microsoft Lecteur Windows Media fournit des visualisations qui permettent à l’utilisateur de voir l’image visuelle qui est synchronisée avec le son du contenu multimédia au fur et à mesure de son lecture. plusieurs visualisations standard sont incluses avec Lecteur Windows Media, y compris les pics, les barres et les DotPlane. Cette section du kit de développement logiciel (SDK) fournit des informations de programmation qui vous permettront de créer vos propres visualisations.

La documentation de visualisation personnalisée est divisée en trois sections :



| Section                                                                                      | Description                                                                                                                                                          |
|----------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [À propos des visualisations personnalisées](about-custom-visualizations.md)                               | Décrit l’architecture des visualisations personnalisées en termes abstraits. vous devez lire cette section pour comprendre comment les visualisations sont liées à Lecteur Windows Media. |
| [Guide de programmation de visualisation personnalisée](custom-visualization-programming-guide.md)         | Explique ce que vous devez faire pour créer une visualisation personnalisée. Cette section contient des conseils et des conseils utiles qui vous aideront à créer des visualisations intéressantes.     |
| [Référence de programmation de visualisation personnalisée](custom-visualization-programming-reference.md) | Fournit une référence pour l’interface que vous devez implémenter, ainsi que d’autres informations de référence technique.                                                          |
