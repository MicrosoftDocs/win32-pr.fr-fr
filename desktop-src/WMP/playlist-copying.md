# <a name="playlistcopying"></a>PLAYLIST. copie

L’attribut de **copie** récupère une valeur indiquant si l’élément de **sélection** est en cours de copie.

``` syntax
        elementID.copying
```

## <a name="possible-values"></a>Valeurs possibles

Cet attribut est une **valeur booléenne** en lecture seule.



| Valeur | Description                              |
|-------|------------------------------------------|
| true  | L’élément de **sélection** est en copie.     |
| false | L’élément de **sélection** n’est pas en copie. |



 

## <a name="requirements"></a>Configuration requise



| Condition requise | Valeur |
|--------------------|------------------------------------------------------|
| Version<br/> | Lecteur Windows Media version 7,0 ou ultérieure<br/> |
