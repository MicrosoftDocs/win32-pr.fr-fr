# <a name="for-more-information-wmp-sdk"></a>Pour plus d’informations (kit de développement logiciel (SDK) WMP)

-   Pour plus d’informations sur le développement de solutions multimédias numériques, consultez le [Centre de développement Microsoft Windows Media](https://msdn.microsoft.com/windowsmedia/default.aspx).
-   Pour plus d’informations sur Windows Media, consultez le [site Web Windows Media](https://support.microsoft.com/help/18612/windows-media-player).
-   Pour plus d’informations sur le lecteur Windows Media, consultez la [page Web du lecteur Windows Media](https://support.microsoft.com/help/18612/windows-media-player) sur le site Web Windows Media.
-   Pour télécharger le SDK Windows, qui comprend le kit de développement logiciel (SDK) du lecteur Windows Media, accédez à la page [SDK Windows](https://msdn.microsoft.com/windows/bb980924.aspx) dans le centre de développement Windows.
-   Pour plus d’informations sur la programmation avec les technologies Microsoft, y compris les technologies Windows Media et les kits de développement logiciel, consultez [MSDN Library]().
-   La version actuelle du kit de développement logiciel (SDK) du lecteur Windows Media est la version 12. Pour les problèmes liés au kit de développement logiciel (SDK) du lecteur Windows Media 11, consultez le [fichier Lisez-moi du lecteur Windows Media 11 SDK](/previous-versions/aa905044(v=msdn.10)).
