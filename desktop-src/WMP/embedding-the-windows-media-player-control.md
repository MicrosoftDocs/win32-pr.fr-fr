# <a name="embedding-the-windows-media-player-control"></a>Incorporation du contrôle du lecteur Windows Media

En plus d’incorporer le contrôle du lecteur Windows Media dans une page Web comme décrit dans la section précédente, vous pouvez l’incorporer dans des documents Microsoft Office et dans des programmes personnalisés que vous créez à l’aide de l’un des langages de programmation suivants, notamment Microsoft Visual Basic, C++ et C#.

Les sections suivantes fournissent des informations sur l’utilisation du contrôle dans différents environnements de programmation.



| Section                                                                                                                                  | Description                                                                                             |
|------------------------------------------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------|
| [Incorporation du contrôle du lecteur Windows Media dans une page Web](embedding-the-windows-media-player-control-in-a-web-page.md)                 | Cette section décrit l’utilisation du contrôle dans une page Web.                                                  |
| [Incorporation du contrôle du lecteur Windows Media dans un document Office](embedding-the-windows-media-player-control-in-an-office-document.md) | Cette section décrit l’utilisation du contrôle dans un document Microsoft Office.                                |
| [Incorporation du contrôle du lecteur Windows Media dans un programme personnalisé](embedding-the-windows-media-player-control-in-a-custom-program.md)     | Cette section décrit l’utilisation du contrôle dans une application Windows écrite en Visual Basic ou C++. |
