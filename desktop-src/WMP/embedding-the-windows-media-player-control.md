# <a name="embedding-the-windows-media-player-control"></a>incorporation du contrôle Lecteur Windows Media

outre l’incorporation du contrôle Lecteur Windows Media dans une page web comme décrit dans la section précédente, vous pouvez l’incorporer dans des documents Microsoft Office et dans des programmes personnalisés que vous créez à l’aide de l’un des différents langages de programmation, notamment Microsoft Visual Basic, C++ et C#.

Les sections suivantes fournissent des informations sur l’utilisation du contrôle dans différents environnements de programmation.



| Section                                                                                                                                  | Description                                                                                             |
|------------------------------------------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------|
| [incorporation du contrôle Lecteur Windows Media dans une Page Web](embedding-the-windows-media-player-control-in-a-web-page.md)                 | Cette section décrit l’utilisation du contrôle dans une page Web.                                                  |
| [incorporation du contrôle Lecteur Windows Media dans un Document Office](embedding-the-windows-media-player-control-in-an-office-document.md) | cette section décrit l’utilisation du contrôle dans un document Microsoft Office.                                |
| [incorporation du contrôle Lecteur Windows Media dans un programme personnalisé](embedding-the-windows-media-player-control-in-a-custom-program.md)     | cette section décrit l’utilisation du contrôle dans une application basée sur Windows écrite en Visual Basic ou C++. |
