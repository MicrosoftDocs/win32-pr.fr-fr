# <a name="listboxfirstvisibleitem"></a>LISTBOX. firstVisibleItem

L’attribut **firstVisibleItem** spécifie ou récupère l’index de la première ligne visible dans le contrôle de zone de liste.

``` syntax
        elementID.firstVisibleItem
```

## <a name="possible-values"></a>Valeurs possibles

Cet attribut est un **nombre** en lecture/écriture (**long**).

## <a name="remarks"></a>Remarques

Cet attribut est utile pour déterminer la première ligne visible lors du défilement de la zone de liste.

## <a name="requirements"></a>Configuration requise



| Condition requise | Valeur |
|--------------------|---------------------------------------------------------|
| Version<br/> | Lecteur Windows Media pour Windows XP ou version ultérieure<br/> |
