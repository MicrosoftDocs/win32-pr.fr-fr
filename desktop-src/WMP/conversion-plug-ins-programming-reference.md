# <a name="conversion-plug-ins-programming-reference"></a>Référence de programmation de plug-ins de conversion

le kit de développement logiciel (SDK) Microsoft Lecteur Windows Media prend en charge une interface pour les plug-ins de conversion qui convertissent un format de fichier multimédia numérique en un autre. Le kit de développement logiciel (SDK) spécifie également la configuration requise pour l’inscription des formats de fichier pour la conversion par un plug-in particulier. Les sections suivantes le décrivent en détail.



| Section                                                                | Description                                                                                                                                                                                           |
|------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Inscription des plug-ins de conversion](registering-conversion-plug-ins.md) | Décrit comment inscrire des formats de fichiers multimédias numériques pour la conversion.                                                                                                                                  |
| [Interface IWMPConvert](/previous-versions/windows/desktop/api/wmpservices/nn-wmpservices-iwmpconvert)                               | fournit des méthodes pour permettre à Lecteur Windows Media plug-ins de conversion de convertir des fichiers multimédias numériques créés à l’aide de technologies non fournies par Microsoft, au Format ASF (Advanced Systems Format). |
