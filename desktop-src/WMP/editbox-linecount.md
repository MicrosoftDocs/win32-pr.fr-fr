# <a name="editboxlinecount"></a>EDITBOX. lineCount

L’attribut **lineCount** récupère le nombre de lignes dans le contrôle zone d’édition.

``` syntax
        elementID.lineCount
```

## <a name="possible-values"></a>Valeurs possibles

Cet attribut est un **nombre** en lecture seule (**long**).

## <a name="remarks"></a>Notes

Cet attribut est utile uniquement lorsque **editStyle** a la valeur « Multiline ».

## <a name="requirements"></a>Configuration requise



| Condition requise | Valeur |
|--------------------|---------------------------------------------------------|
| Version<br/> | Lecteur Windows Media pour Windows XP ou version ultérieure<br/> |
