# <a name="playlistdropdownvisible"></a>PLAYLIST. dropDownVisible

L’attribut **dropDownVisible** spécifie ou récupère une valeur indiquant si la liste déroulante de sélection est visible.

``` syntax
        elementID.dropDownVisible
```

## <a name="possible-values"></a>Valeurs possibles

Cet attribut est une **valeur booléenne** en lecture/écriture.



| Valeur | Description                             |
|-------|-----------------------------------------|
| true  | Par défaut. La liste déroulante est visible. |
| false | La liste déroulante n’est pas visible.      |



 

## <a name="requirements"></a>Spécifications



| Condition requise | Valeur |
|--------------------|------------------------------------------------------|
| Version<br/> | Lecteur Windows Media version 7,0 ou ultérieure<br/> |
