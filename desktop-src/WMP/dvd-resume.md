# <a name="dvdresume-method"></a>DVD. Resume, méthode

La méthode **Resume** retourne en mode lecture à partir du mode menu à la même position de titre que lorsque le menu a été appelé.

## <a name="syntax"></a>Syntaxe


```JScript
DVD.resume()
```



## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.

## <a name="return-value"></a>Valeur retournée

Cette méthode ne retourne pas de valeur.

## <a name="remarks"></a>Notes

**Lecteur Windows Media 10 Mobile :** Cette méthode fonctionne toujours, mais n’effectue pas l’opération prévue.
