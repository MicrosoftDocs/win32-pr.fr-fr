# <a name="themelogstring"></a>THÈME. logString

La méthode **logString** consigne une chaîne définie par l’utilisateur dans le fichier d’erreur si la journalisation est activée.

``` syntax
        theme.logString(message)
```

## <a name="parameters"></a>Paramètres

<dl> <dt>

<span id="message"></span><span id="MESSAGE"></span>*Message*
</dt> <dd>

**Chaîne** spécifiant le message à enregistrer.

</dd> </dl>

## <a name="return-value"></a>Valeur renvoyée

Cette méthode ne retourne pas de valeur.

## <a name="requirements"></a>Configuration requise



| Condition requise | Valeur |
|--------------------|------------------------------------------------------|
| Version<br/> | Lecteur Windows Media version 7,0 ou ultérieure<br/> |
