# <a name="sample-text-section"></a>Exemple de section de texte

Les lignes suivantes affichent une section de texte standard d’un fichier de définition d’apparence :


```C++
//  <Type>       <Location>     <Align>  <Font>          <Color>

//  ------       ----------     -------  ------          -------

    Time         180,46,50,30   Right    Tahoma,16,N     255,255,255

    Playlist     3,46,175,20    Left     Tahoma,10,N     255,255,255

    Track#       10,68,40,20    Left     Tahoma,14,N     255,255,255

```



L’exemple précédent définit trois zones d’affichage de texte qui indiquent l’heure actuelle dans l’élément multimédia, la sélection actuelle et le numéro de piste actuel.
