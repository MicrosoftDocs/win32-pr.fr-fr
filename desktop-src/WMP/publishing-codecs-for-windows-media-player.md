# <a name="publishing-codecs-for-windows-media-player"></a>Publication de codecs pour le lecteur Windows Media

Le site Web WMPlugins n’est plus pris en charge pour la publication de votre codec et sa découverte par les utilisateurs du lecteur Windows Media. Vous pouvez à la place publier votre codec sur un site Web de votre choix.
