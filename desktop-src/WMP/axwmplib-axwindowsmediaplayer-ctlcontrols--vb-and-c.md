# <a name="axwindowsmediaplayerctlcontrols-property"></a>AxWindowsMediaPlayer. Ctlcontrols, propriété

La propriété Ctlcontrols obtient une interface IWMPControls qui fournit un moyen de manipuler la lecture d’un élément multimédia.

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```CSharp
public IWMPControls Ctlcontrols {get;}
```


```VB

Public ReadOnly Property Ctlcontrols As IWMPControls
```





## <a name="property-value"></a>Valeur de la propriété

WMPLIB. Interface IWMPControls.
