# <a name="axwindowsmediaplayerdvd-property"></a>AxWindowsMediaPlayer. DVD, propriété

La propriété DVD obtient une interface IWMPDVD qui fournit des propriétés et des méthodes pour travailler avec des DVD.

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```CSharp
public IWMPDVD dvd {get;}
```


```VB

Public ReadOnly Property dvd As IWMPDVD
```





## <a name="property-value"></a>Valeur de la propriété

Interface WMPLib. IWMPDVD.

## <a name="requirements"></a>Configuration requise

|----------------------|----------------------------------------------------------------------------------------------------------------------------|
