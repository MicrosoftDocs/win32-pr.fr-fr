# <a name="equalizersettingsenhancedaudio"></a>EQUALIZERSETTINGS.enhancedAudio

L’attribut **enhancedAudio** spécifie ou récupère une valeur indiquant si l’audio étendu est activé.

``` syntax
        elementID.enhancedAudio
```

## <a name="possible-values"></a>Valeurs possibles

Cet attribut est une **valeur booléenne** en lecture/écriture.



| Description | Valeur                                  |
|-------------|----------------------------------------|
| true        | L’audio étendu est activé.           |
| false       | Par défaut. L’audio étendu est désactivé. |



 

## <a name="requirements"></a>Configuration requise



| Condition requise | Valeur |
|--------------------|---------------------------------------------------|
| Version<br/> | Lecteur Windows Media série 9 ou version ultérieure<br/> |
