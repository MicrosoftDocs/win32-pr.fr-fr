# <a name="effectscurrentpresettitle"></a>EFFECTs. currentPresetTitle

L’attribut **currentPresetTitle** récupère le titre de la présélection actuelle de la visualisation actuelle.

``` syntax
        elementID.currentPresetTitle
```

## <a name="possible-values"></a>Valeurs possibles

Cet attribut est une **chaîne** en lecture seule.

## <a name="requirements"></a>Configuration requise



| Condition requise | Valeur |
|--------------------|------------------------------------------------------|
| Version<br/> | Lecteur Windows Media version 7,0 ou ultérieure<br/> |
