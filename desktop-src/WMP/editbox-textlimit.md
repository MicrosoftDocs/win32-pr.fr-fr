# <a name="editboxtextlimit"></a>EDITBOX. textLimit

L’attribut **textLimit** spécifie ou récupère le nombre maximal de caractères que l’utilisateur peut taper dans le contrôle de zone d’édition.

``` syntax
        elementID.textLimit
```

## <a name="possible-values"></a>Valeurs possibles

Cet attribut est un **nombre** en lecture/écriture (**long**) dont la valeur par défaut est 0, ce qui indique l’absence de limite.

## <a name="requirements"></a>Configuration requise



| Condition requise | Valeur |
|--------------------|---------------------------------------------------------|
| Version<br/> | Lecteur Windows Media pour Windows XP ou version ultérieure<br/> |
