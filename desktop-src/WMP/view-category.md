# <a name="viewcategory"></a>AFFICHAGE. catégorie

L’attribut **Category** spécifie ou récupère la catégorie pour laquelle la **vue** est prévue.

``` syntax
        elementID.category
```

## <a name="possible-values"></a>Valeurs possibles

Cet attribut est une **chaîne** en lecture/écriture contenant l’une des valeurs suivantes.



| Valeur | Description                           |
|-------|---------------------------------------|
| Tous   | Par défaut. Thème pour tous les types de médias |
| Case d’option | Interface utilisateur pour la lecture radio             |
| CD    | Interface utilisateur pour la lecture de CD                |
| DVD   | Interface utilisateur pour la lecture de DVD               |
| Musique | Interface utilisateur pour MP3, WAV, MIDI, WMA        |
| Vidéo | Interface utilisateur pour la lecture vidéo             |



 

## <a name="requirements"></a>Spécifications



| Condition requise | Valeur |
|--------------------|------------------------------------------------------|
| Version<br/> | Lecteur Windows Media version 7,0 ou ultérieure<br/> |
