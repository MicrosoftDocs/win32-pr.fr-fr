# <a name="viewminheight"></a>VIEW. minHeight

L’attribut **minHeight** spécifie ou récupère la hauteur minimale, en pixels, de la **vue** lors du redimensionnement.

``` syntax
        elementID.minHeight
```

## <a name="possible-values"></a>Valeurs possibles

Cet attribut est un **nombre** en lecture/écriture (**long**) dont la valeur est égale à zéro ou supérieure. Sa valeur par défaut est zéro, ce qui signifie qu’il n’y a aucune restriction sur la hauteur minimale de la **vue**.

## <a name="requirements"></a>Configuration requise



| Condition requise | Valeur |
|--------------------|------------------------------------------------------|
| Version<br/> | Lecteur Windows Media version 7,0 ou ultérieure<br/> |
