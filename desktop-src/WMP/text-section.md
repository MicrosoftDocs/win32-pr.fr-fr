# <a name="text-section"></a>Section de texte

Aucune section de texte n’est utilisée dans cette apparence, mais voici un exemple du format que vous devez utiliser lors de l’implémentation d’une section de texte :


```C++
[ Text ]

//  <Item>       <Location>     <Align>  <Font>          <Color>
//  ------       ----------     -------  ------          -------

```
