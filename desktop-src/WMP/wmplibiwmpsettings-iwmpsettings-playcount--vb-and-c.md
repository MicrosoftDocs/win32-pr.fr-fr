# <a name="iwmpsettingsplaycount-property"></a>IWMPSettings ::p propriété layCount

La propriété **playCount** obtient ou définit le nombre de fois qu’un élément multimédia doit être lu.

## <a name="syntax"></a>Syntaxe


```CSharp
public System.Int32 playCount {get; set;}
```


```VB

Public Property playCount As System.Int32
```





## <a name="property-value"></a>Valeur de la propriété

**System. Int32** qui est le nombre de jeux, avec une valeur minimale de 1 et une valeur par défaut de 1.
