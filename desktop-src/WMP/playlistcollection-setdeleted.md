# <a name="playlistcollectionsetdeleted-method"></a>Méthode PlaylistCollection. setDeleted

La méthode **setDeleted** déplace une sélection vers le dossier éléments supprimés.

## <a name="syntax"></a>Syntaxe


```JScript
PlaylistCollection.setDeleted(
  playlist,
  true
)
```



## <a name="parameters"></a>Paramètres

<dl> <dt>

*sélection* \[ dans\]
</dt> <dd>

Objet de la **sélection** à déplacer.

</dd> <dt>

*true* \[ dans\]
</dt> <dd>

Spécifiez toujours cette valeur.

</dd> </dl>

## <a name="return-value"></a>Valeur retournée

Cette méthode ne retourne pas de valeur.

## <a name="remarks"></a>Notes

**Windows Media Player 10 Mobile**: cette méthode n’est pas prise en charge.
