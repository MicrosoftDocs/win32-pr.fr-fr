# <a name="ambientattributesright"></a>AmbientAttributes. Right

L’attribut **Right** spécifie ou récupère la coordonnée droite du contrôle.

``` syntax
        elementID.right
```

## <a name="possible-values"></a>Valeurs possibles

Cet attribut est un **nombre** en lecture/écriture (**long**) représentant la distance en pixels entre le contrôle et le bord droit de la **vue** parente ou de la sous- **vue**. Le comportement des valeurs négatives, ou lorsque [AmbientAttributes. Width](ambientattributes-width.md) n’est pas spécifié, n’est pas défini.
