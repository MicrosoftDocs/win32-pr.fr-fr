# <a name="effectspreviouspreset"></a>EFFECTs. previousPreset

La méthode **previousPreset** affiche la présélection précédente de la visualisation actuelle.

``` syntax
        elementID.previousPreset()
```

## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.

## <a name="return-value"></a>Valeur renvoyée

Cette méthode ne retourne pas de valeur.

## <a name="remarks"></a>Notes

Si la présélection actuelle est la première de la liste pour la visualisation actuelle, la dernière présélection est mise à jour.

## <a name="requirements"></a>Configuration requise



| Condition requise | Valeur |
|--------------------|------------------------------------------------------|
| Version<br/> | Lecteur Windows Media version 7,0 ou ultérieure<br/> |
