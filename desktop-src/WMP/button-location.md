# <a name="button-location"></a>Emplacement du bouton

Vous devez définir l’emplacement de chaque image de bouton. Pour ce faire, vous devez fournir quatre entiers positifs séparés par des virgules. Les nombres définissent les coordonnées de l’angle supérieur gauche, la largeur et la hauteur d’un bouton, en pixels.

Si vous utilisez des boutons de région (PushHit, ToggleHit et 2PushHit), vous devez toujours entrer ces quatre nombres. Les valeurs sont utilisées pour déterminer les emplacements des images secondaires telles que Pushed et Disabled.

> [!Note]  
> les types de boutons sont déconseillés dans les habillages pour Lecteur Windows Media 10 Mobile ou version ultérieure.
