# <a name="background-plug-ins"></a>Plug-ins d’arrière-plan

Les plug-ins d’IU d’arrière-plan n’ont pas d’interfaces utilisateur, sauf s’ils implémentent des pages de propriétés. lorsqu’un ou plusieurs plug-ins en arrière-plan sont en cours d’exécution, une icône spéciale s’affiche dans le Lecteur Windows Media afficher en regard du temps de suivi en cours. Vous pouvez cliquer sur cette icône pour afficher les options de plug-ins en arrière-plan dans la boîte de dialogue Options. À partir de là, l’utilisateur peut choisir d’ouvrir la page de propriétés de n’importe quel plug-in qui en prend en charge un.

Les plug-ins d’arrière-plan sont utiles pour fournir des services automatiques qui ne nécessitent pas d’intervention de l’utilisateur.
