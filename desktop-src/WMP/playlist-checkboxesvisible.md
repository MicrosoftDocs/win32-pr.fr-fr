# <a name="playlistcheckboxesvisible"></a>PLAYLIST. checkboxesVisible

L’attribut **checkboxesVisible** spécifie ou récupère une valeur indiquant si les cases à cocher sont visibles.

``` syntax
        elementID.checkboxesVisible
```

## <a name="possible-values"></a>Valeurs possibles

Cet attribut est une **valeur booléenne** en lecture/écriture.



| Valeur | Description                           |
|-------|---------------------------------------|
| true  | Les cases à cocher sont visibles.              |
| false | Par défaut. Les cases à cocher ne sont pas visibles. |



 

## <a name="remarks"></a>Notes

Les cases à cocher s’affichent dans la colonne la plus à gauche.

## <a name="requirements"></a>Spécifications



| Condition requise | Valeur |
|--------------------|------------------------------------------------------|
| Version<br/> | Lecteur Windows Media version 7,0 ou ultérieure<br/> |
