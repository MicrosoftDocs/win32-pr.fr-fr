# <a name="playlistcolumnorder"></a>PLAYLIST. columnOrder

L’attribut **columnOrder** spécifie ou récupère l’ordre des colonnes de la sélection.

``` syntax
        elementID.columnOrder
```

## <a name="possible-values"></a>Valeurs possibles

Cet attribut est une **chaîne** en lecture/écriture spécifiant une liste délimitée par des points-virgules d’index de colonne de sélection. La valeur par défaut est « 0 ; 1 ; 2 ; 3 ». Les points-virgules de début et de fin ne doivent pas être présents.

## <a name="requirements"></a>Configuration requise



| Condition requise | Valeur |
|--------------------|------------------------------------------------------|
| Version<br/> | Lecteur Windows Media version 7,0 ou ultérieure<br/> |
