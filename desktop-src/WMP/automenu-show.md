# <a name="automenushow"></a>MENU automatique. afficher

La méthode **Show** affiche le **panneau d’accès rapide**.

``` syntax
        elementID.show("Play")
        
```

## <a name="parameters"></a>Paramètres

<dl> <dt>

<span id="_Play_"></span><span id="_play_"></span><span id="_PLAY_"></span>*Répétition*
</dt> <dd>

Spécifiez toujours cette valeur.

</dd> </dl>

## <a name="return-value"></a>Valeur renvoyée

Cette méthode ne retourne pas de valeur.

## <a name="remarks"></a>Notes

Le **panneau d’accès rapide** apparaît à l’emplacement spécifié par les attributs **gauche** et **supérieur** de l’élément de **menu** .

## <a name="requirements"></a>Spécifications



| Condition requise | Valeur |
|--------------------|---------------------------------------------------|
| Version<br/> | Lecteur Windows Media série 9 ou version ultérieure<br/> |
