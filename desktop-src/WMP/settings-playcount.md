# <a name="settingsplaycount"></a>Paramètres. playCount

La propriété **playCount** spécifie ou récupère le nombre de fois qu’un élément multimédia doit être lu.

## <a name="syntax"></a>Syntaxe

Player. Settings. playCount

## <a name="possible-values"></a>Valeurs possibles

Cette propriété est un **nombre** en lecture/écriture (**long**) avec une valeur minimale de 1 et une valeur par défaut de un.

**Windows Media Player 10 Mobile**: cette propriété n’est pas prise en charge.
