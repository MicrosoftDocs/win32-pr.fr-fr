# <a name="purpose-of-a-metafile-playlist"></a>Objectif d’une sélection de métafichiers

L’objectif de base d’une sélection de métafichiers est de rediriger le contenu multimédia en continu à partir de navigateurs vers le lecteur Windows Media, car la plupart des navigateurs essaient de télécharger le contenu au lieu de le diffuser en continu. Une sélection de métafichiers fournit les informations utilisées par le lecteur Windows Media pour recevoir des flux, comme le Uniform Resource Locator (URL). La sélection de métafichiers peut fournir le chemin d’accès aux fichiers multimédias sur Internet, un intranet ou simplement des médias stockés sur un réseau ou un lecteur local. Avec le traitement redirigé vers le lecteur Windows Media, vous bénéficiez d’un contrôle accru sur la diffusion multimédia en continu.

Les sélections de métafichiers permettent de contrôler l’expérience de diffusion en continu. En combinant différents fichiers multimédias dans un flux de contenu unique, vous pouvez contrôler et personnaliser leur présentation.
