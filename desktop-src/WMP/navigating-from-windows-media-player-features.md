# <a name="navigating-from-windows-media-player-features"></a>Navigation à partir des fonctionnalités du lecteur Windows Media

Si vous fournissez une boutique en ligne de musique, vous pouvez effectuer la navigation à partir de la vue Info Center, des informations sur l’album et des pages HTMLView vers les volets Office de votre service. Le processus fonctionne exactement comme décrit dans [navigation entre les volets des tâches de service](navigating-between-service-task-panes.md). La différence réside dans le fait que vous pouvez naviguer d’une fonctionnalité à un volet de tâches de service, mais pas d’un volet de tâches de service à une fonctionnalité.
