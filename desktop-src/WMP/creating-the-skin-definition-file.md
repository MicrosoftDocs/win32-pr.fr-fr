# <a name="creating-the-skin-definition-file"></a>Création du fichier de définition d’apparence

Le fichier de définition d’apparence est un fichier texte simple. vous pouvez créer des fichiers de définition d’apparence dans n’importe quel éditeur de texte qui enregistre des fichiers texte ASCII bruts, tels que microsoft Bloc-notes, microsoft Visual InterDev® ou Microsoft Visual Studio. Étant donné que vous allez écrire du code XML et non du code HTML, vous ne souhaiterez peut-être pas utiliser un éditeur pour HTML, car il peut ajouter des codes supplémentaires que vous ne voulez pas ou vous informer des erreurs que vous n’avez pas. Vous souhaiterez sans doute éviter les traitements de texte tels que les Microsoft Word ; même si Word peut enregistrer des fichiers texte bruts, vous ne voudrez peut-être pas vous souvenir de l’enregistrer en tant que texte chaque fois que vous l’enregistrez.
