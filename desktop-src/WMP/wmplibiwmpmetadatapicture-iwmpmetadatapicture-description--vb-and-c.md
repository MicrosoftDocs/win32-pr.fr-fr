# <a name="iwmpmetadatapicturedescription-property"></a>IWMPMetadataPicture ::D propriété escription

La `Description` propriété obtient la description de l’image représentée par l’attribut de métadonnées.

## <a name="syntax"></a>Syntaxe


```CSharp
public System.String Description {get; set;}
```


```VB

Public ReadOnly Property Description As System.String
```





## <a name="property-value"></a>Valeur de la propriété

**System. String** qui est la description de l’image.

## <a name="remarks"></a>Notes

Avant d’utiliser cette propriété, vous devez disposer d’un accès en lecture à la bibliothèque. Pour plus d’informations, consultez [accès à la bibliothèque](library-access.md).
