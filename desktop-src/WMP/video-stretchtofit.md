# <a name="videostretchtofit"></a>VIDÉO. stretchToFit

L’attribut **stretchToFit** spécifie une valeur indiquant si la vidéo est étirée sur la largeur et la hauteur définies pour le contrôle Video.

``` syntax
        elementID.stretchToFit
```

## <a name="possible-values"></a>Valeurs possibles

Cet attribut est une **valeur booléenne** en lecture/écriture.



| Valeur | Description                                             |
|-------|---------------------------------------------------------|
| true  | La vidéo est étirée pour s’ajuster au contrôle.              |
| false | Par défaut. La vidéo n’est pas étirée pour s’ajuster au contrôle. |



 

## <a name="remarks"></a>Notes

Si aucune largeur ou hauteur n’est spécifiée, cet attribut est ignoré.

## <a name="requirements"></a>Spécifications



| Condition requise | Valeur |
|--------------------|------------------------------------------------------|
| Version<br/> | Lecteur Windows Media version 7,0 ou ultérieure<br/> |
