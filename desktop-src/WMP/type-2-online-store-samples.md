# <a name="type-2-online-store-samples"></a>Exemples de magasins en ligne de type 2

Le kit de développement logiciel (SDK) Windows Media Player comprend les exemples de magasins en ligne suivants :

-   **Exemples de magasins en ligne.**
-   **Un plug-in de magasin en ligne de type 2.** Cet exemple est généré par un Assistant projet Visual Studio, qui crée un projet C++ que vous pouvez utiliser comme point de départ pour votre propre composant. L’utilisation de l’Assistant garantit l’unicité de votre composant. Pour plus d’informations, consultez [plug-in de magasin en ligne de type 2](type-2-online-store-plug-in.md).
-   **Une page Web de la boutique en ligne.** Cet exemple de page Web affiche une page Web que le lecteur Windows Media 10 ou version ultérieure peut héberger dans un volet de tâches de service.
-   **Spécification de l’attribut ContentDistributor.** Cet exemple de code C++ vous montre comment utiliser le kit de développement logiciel (SDK) Windows Media format pour ajouter l’attribut **ContentDistributor** à l’en-tête d’un fichier Windows Media. Il s’agit de l’attribut que le lecteur Windows Media 10 ou version ultérieure utilise pour identifier des fournisseurs de magasins en ligne individuels. Pour plus d’informations, consultez [Ajout de l’attribut ContentDistributor](adding-the-contentdistributor-attribute.md).

**Affichage des exemples de magasins en ligne de type 2**

Microsoft a fourni deux exemples de magasins en ligne de type 2 que vous pouvez afficher. L’un est un exemple de magasin de commerce et l’autre est un exemple de magasin de musique. Les exemples de magasins en ligne sont des implémentations simples qui illustrent certaines des fonctionnalités de base disponibles pour une utilisation dans vos propres magasins en ligne. Vous pouvez afficher le code source des pages Web de l’exemple de magasin en ligne.

Comme les exemples sont disponibles sur Internet, ils seront mis à jour avec des fonctionnalités supplémentaires de temps en temps.

Pour afficher les exemples de magasins en ligne de type 2, configurez votre registre comme décrit dans [clés de Registre et entrées pour un magasin de type 2 en ligne](registry-keys-and-entries-for-a-type-2-online-store.md). Le tableau suivant donne les clés pour les exemples de magasins de type 2.



| Type de magasin      | Clé  | Versions de lecteur qui prennent en charge l’exemple de magasin |
|-----------------|------|-----------------------------------------------|
| Musique de type 2    | 8540 | 10 et 11                                     |
| Commerce de type 2 | 8541 | 10                                            |
