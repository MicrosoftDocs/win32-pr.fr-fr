# <a name="itemsplaylist"></a>ITEMSPLAYLIST

Il s’agit d’un élément de **sélection** prédéfini avec les valeurs par défaut suivantes.

``` syntax
backgroundColor="black"
columns="name=Name;Duration=Time"
columnsVisible="false"
dropDownVisible="false"
foregroundColor="white"
```

## <a name="remarks"></a>Notes

Cette opération crée une **sélection** simple qui affiche les éléments d’une sélection sans en-tête de liste déroulante ou de colonne visible. Toutes les propriétés de cette **playlist** peuvent être remplacées en les spécifiant explicitement.

## <a name="requirements"></a>Configuration requise



| Condition requise | Valeur |
|--------------------|----------------------------------------------|
| Version<br/> | Lecteur Windows Media 7,0 ou version ultérieure<br/> |
