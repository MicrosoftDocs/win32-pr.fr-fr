# <a name="effectscurrenteffecttitle"></a>EFFECTs. currentEffectTitle

L’attribut **currentEffectTitle** récupère le titre d’affichage de la visualisation actuelle.

``` syntax
        elementID.currentEffectTitle
```

## <a name="possible-values"></a>Valeurs possibles

Cet attribut est une **chaîne** en lecture seule.

## <a name="requirements"></a>Spécifications



| Condition requise | Valeur |
|--------------------|------------------------------------------------------|
| Version<br/> | Lecteur Windows Media version 7,0 ou ultérieure<br/> |
