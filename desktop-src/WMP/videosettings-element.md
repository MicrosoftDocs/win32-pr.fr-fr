# <a name="videosettings-element"></a>Élément VIDEOSETTINGS

L’élément **VIDEOSETTINGS** fournit un moyen de modifier différents paramètres vidéo à l’aide des attributs et de la méthode répertoriés ici.

L’élément **VIDEOSETTINGS** prend en charge les attributs suivants.



| Attribut                                  | Description                                                 |
|--------------------------------------------|-------------------------------------------------------------|
| [lumineuse](videosettings-brightness.md) | Spécifie ou récupère le paramètre de luminosité de la vidéo. |
| [revanche](videosettings-contrast.md)     | Spécifie ou récupère le paramètre de contraste de la vidéo.   |
| [Hue](videosettings-hue.md)               | Spécifie ou récupère le paramètre de teinte de la vidéo.        |
| [saturation](videosettings-saturation.md) | Spécifie ou récupère le paramètre de saturation de la vidéo. |



 

L’élément **VIDEOSETTINGS** prend en charge la méthode suivante.



| Méthode                           | Description                                    |
|----------------------------------|------------------------------------------------|
| [reset](videosettings-reset.md) | Rétablit les valeurs par défaut de tous les attributs. |



 

L’élément **VIDEOSETTINGS** peut implémenter les gestionnaires d’événements [ \_ OnChange d’attribut](attribute-onchange.md) .

> [!Note]  
> cet élément requiert Lecteur Windows Media pour Windows XP ou version ultérieure.
