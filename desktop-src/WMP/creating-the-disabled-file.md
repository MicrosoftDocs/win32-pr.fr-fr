# <a name="creating-the-disabled-file"></a>Création du fichier désactivé

Le fichier désactivé est semblable au fichier envoyé, sauf qu’une autre couleur (rouge) est utilisée pour indiquer quand un objet est désactivé.

L’image suivante est le fichier désactivé.

![fichier désactivé](images/ceswmdis.png)

Notez que les fichiers comme celui-ci peuvent être tronqués sur les côtés droit et inférieur. La taille réelle de la bitmap n’est pas importante, mais uniquement que les éléments sont alignés avec la bitmap de région pour le test de positionnement, comme mesuré à partir des angles supérieurs gauches (y compris les décalages).
