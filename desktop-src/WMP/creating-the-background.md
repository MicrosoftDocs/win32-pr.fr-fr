# <a name="creating-the-background"></a>Création de l’arrière-plan

Vous êtes maintenant prêt à créer l’image d’arrière-plan. Maintenez la couche rectangle visible, mais conservez-la derrière la couche sur laquelle vous dessinez, et utilisez-la comme guide pour créer les images pour votre apparence. Vous pouvez assembler chaque élément dans une ou plusieurs couches pour faciliter la copie ultérieure sur un autre fichier. Par exemple, le bouton PlayPause doit être copié vers et recoloriés dans le fichier suspendu, le fichier désactivé et le fichier de région. Lorsque vous déplacez une image vers le fichier de région, assurez-vous que l’image que vous copiez n’a pas d’effets de bords complexes tels que les ombres ou les biseau.

Une fois que vous avez fini de créer l’image souhaitée, rendez-vous visible toutes les couches que vous souhaitez utiliser et masquez-les autres (par exemple, esquisse et rectangle). Utilisez ensuite la commande Enregistrer une copie pour enregistrer l’illustration dans un fichier. bmp.

Le diagramme suivant est l’image d’arrière-plan.

![fichier d’arrière-plan](images/ceswmbak.png)

Les biseau, le relief et les ombres ont été utilisés pour créer les effets visuels pour les différents éléments. Notez qu’il n’y a aucun bouton pour l’image de curseur de volume. Il sera créé ultérieurement.

Photoshop offre plusieurs effets visuels que vous pouvez appliquer aux images, ce qui facilite la création d’une illustration plus intéressante. Par exemple, le bouton PlayPause a été créé avec un cercle simple. Un biseau et un relief avec un effet d’ombre ont été créés. La flèche pleine était un triangle simple créé avec trois clics de l’outil plume. La flèche a ensuite été attribuée à un aspect en relief à l’aide d’un effet. Lorsque vous utilisez un programme tel que Photoshop, vous n’avez pas à pouvoir dessiner. Vous pouvez simplement créer des images simples et appliquer des effets. Vous pouvez également importer vos propres photos ou illustrations numérisées.
