# <a name="status-location"></a>Emplacement de l’État

Vous devez définir l’emplacement de l’affichage d’État que vous souhaitez utiliser. Vous devez fournir quatre entiers positifs, séparés par des virgules, qui définissent la gauche, le haut, la largeur et la hauteur de l’affichage de l’État, en pixels, par rapport à l’image d’arrière-plan.


```C++
45,193,120,13 

```
