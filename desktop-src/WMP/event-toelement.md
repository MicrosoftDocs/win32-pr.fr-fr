# <a name="eventtoelement"></a>Event. toElement

L’attribut **ToElement** récupère l’élément vers lequel le focus clavier a été déplacé. Cet attribut s’applique uniquement à l’événement **onblur** ; pour tous les autres événements, sa valeur est **null**.

``` syntax
event.toElement
```

## <a name="possible-values"></a>Valeurs possibles

Cet attribut est un objet en lecture seule.

## <a name="requirements"></a>Configuration requise



| Condition requise | Valeur |
|--------------------|------------------------------------------------------|
| Version<br/> | Lecteur Windows Media version 7,0 ou ultérieure<br/> |
