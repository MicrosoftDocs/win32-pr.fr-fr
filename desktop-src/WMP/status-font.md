# <a name="status-font"></a>Police d’État

Vous devez définir la police utilisée par l’affichage de l’état que vous souhaitez utiliser. La police est définie par trois valeurs, séparées par des virgules, représentant la face, la taille et le poids de la police.

**Valeurs de police**

Vous pouvez utiliser n’importe quel nom de police s’il est susceptible d’être installé sur l’ordinateur de l’utilisateur. Si une police est introuvable sur l’ordinateur, une alternative est sélectionnée par le système d’exploitation. le tableau suivant montre les polices qui se trouvent généralement sur Windows appareils mobiles 2003.



| Police       | Description              |
|----------------|--------------------------|
| Tahoma         | Police sans empattement.   |
| Console Lucida | Police à Serif carré. |



 

**Valeurs de taille**

Il s’agit de la taille de la police en points. Toute valeur entière positive est valide, bien que les nombres compris entre 10 et 18 soient recommandés. Les tailles inférieures à 10 peuvent être difficiles à lire, et les tailles supérieures à 18 peuvent ne pas avoir suffisamment d’espace pour afficher plus de quelques lettres à la fois.

**Valeurs de poids**

Les seules valeurs autorisées sont indiquées dans le tableau suivant.



| Valeur | Description |
|-------|-------------|
| B     | Gras        |
| N     | Normal      |
