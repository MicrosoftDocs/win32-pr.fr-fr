# <a name="managing-playlists"></a>Gestion des sélections

Un objet **playlist** représente un ensemble d’éléments multimédias. Vous utilisez les propriétés et les méthodes de l’objet pour manipuler les éléments multimédias qu’il contient et pour récupérer les métadonnées relatives à la sélection. Pour gérer les sélections elles-mêmes, vous utilisez l’objet **PlaylistCollection** et l’objet **MediaCollection** .

les rubriques suivantes décrivent comment gérer les sélections à l’aide du modèle objet de contrôle Lecteur Windows Media :

-   [Attributs de sélection](playlist-attributes.md)
-   [Sélections et éléments multimédias](playlists-and-media-items.md)
-   [Sélections et objet PlaylistCollection](playlists-and-the-playlistcollection-object.md)
-   [Sélections statiques et automatiques](static-and-auto-playlists.md)
-   [Sélections et objet MediaCollection](playlists-and-the-mediacollection-object.md)
