# <a name="erroritemcustomurl"></a>ErrorItem.customUrl

La propriété **customURL** récupère l’URL d’un site Web qui affiche des informations spécifiques sur l’échec du téléchargement du codec.

``` syntax
player.error.item(
        index
        ).customURL
      
```

## <a name="possible-values"></a>Valeurs possibles

Cette propriété est une **chaîne** en lecture seule.

## <a name="remarks"></a>Notes

**Lecteur Windows Media 10 Mobile :** Cette propriété retourne toujours une chaîne vide.
