# <a name="onblur"></a>onblur

Le gestionnaire d’événements **onblur** gère un événement qui se produit lorsque l’élément perd le focus clavier.

``` syntax
onblur
```

## <a name="remarks"></a>Notes

Le gestionnaire d’événements **onblur** n’est pas applicable à l’élément **Effects** ou à l’élément **Popup** .

## <a name="requirements"></a>Configuration requise



| Condition requise | Valeur |
|--------------------|-----------------------------------------------------|
| Version<br/> | Lecteur Windows Media version 70 ou ultérieure<br/> |
