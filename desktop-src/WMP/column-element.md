# <a name="column-element"></a>Élément COLUMN

L’élément **Column** représente une colonne dans un contrôle playlist. Les éléments de **colonne** sont toujours des enfants des éléments de **sélection** .

L’élément **Column** prend en charge les attributs suivants.



| Attribut                                       | Description                                                        |
|-------------------------------------------------|--------------------------------------------------------------------|
| [columnName](column-columnname.md)             | Spécifie ou récupère un nom de colonne dans le contrôle **playlist** .  |
| [columnID](column-columnid.md)                 | Spécifie ou récupère un ID de colonne dans le contrôle **playlist** .    |
| [columnResizeMode](column-columnresizemode.md) | Spécifie ou récupère le mode de redimensionnement pour cette colonne.            |
| [Largeur](column-columnwidth.md)           | Spécifie ou récupère une largeur de colonne dans le contrôle **playlist** . |
