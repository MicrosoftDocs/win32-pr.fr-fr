# <a name="about-windows-media-metafiles"></a>À propos des fichiers Windows Media

Les fichiers Microsoft Windows Media sont des fichiers texte qui fournissent des informations sur un flux de fichier et sa présentation. Les fichiers Windows Media sont basés sur la syntaxe du Extensible Markup Language (XML) et peuvent être codés au format ANSI ou UNICODE (UTF-8). Elles sont constituées de différents éléments avec leurs balises et attributs associés. Chaque élément d’un métafichier Windows Media définit une action ou un paramètre particulier dans le lecteur Windows Media. Certains éléments doivent se trouver à une position spécifique dans le fichier par rapport à d’autres éléments. Certains éléments ont des balises et des attributs obligatoires qui doivent être définis dans le métafichier Windows Media.

Les packages de téléchargement Windows Media offrent un moyen de combiner les bordures du lecteur Windows Media, les informations de sélection et le contenu multimédia en un seul fichier téléchargeable.
