# <a name="about-windows-media-metafiles"></a>à propos des Windows de fichiers multimédias

les sous-fichiers multimédias Microsoft Windows sont des fichiers texte qui fournissent des informations sur un flux de fichier et sa présentation. Windows Les sous-fichiers multimédias sont basés sur la syntaxe du Extensible Markup Language (XML) et peuvent être encodés au format ANSI ou UNICODE (UTF-8). Elles sont constituées de différents éléments avec leurs balises et attributs associés. chaque élément d’un Windows métafichier multimédia définit une action ou un paramètre particulier dans Lecteur Windows Media. Certains éléments doivent se trouver à une position spécifique dans le fichier par rapport à d’autres éléments. certains éléments ont des balises et des attributs obligatoires qui doivent être définis dans le métafichier Windows Media.

Windows les packages de téléchargement de médias offrent un moyen de combiner des bordures Lecteur Windows Media, des informations de sélection et du contenu multimédia en un seul fichier téléchargeable.
