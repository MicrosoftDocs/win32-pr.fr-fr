# <a name="iwmpmetadatatext-vb-and-c-interface"></a>Interface IWMPMetadataText (VB et C#)

Fournit des propriétés pour obtenir des informations sur les attributs de métadonnées textuelles complexes.

L’interface **IWMPMetadataText** expose les propriétés suivantes.



| Propriété                                                                         | Description                              |
|----------------------------------------------------------------------------------|------------------------------------------|
| [Description](wmplibiwmpmetadatatext-iwmpmetadatatext-description--vb-and-c.md) | Obtient une description du texte des métadonnées. |
| [text](wmplibiwmpmetadatatext-iwmpmetadatatext-text--vb-and-c.md)               | Obtient le texte des métadonnées.                  |



 

Obtient une interface **IWMPMetadataText** en passant un nom d’attribut tel que **WM/UserWebURL** à la méthode suivante et en effectuant un cast de l’objet retourné.



| Interface                              | Propriété                                                                         |
|----------------------------------------|----------------------------------------------------------------------------------|
| [IWMPMedia3](iwmpmedia3--vb-and-c.md) | [getItemInfoByType](wmplibiwmpmedia3-iwmpmedia3-getiteminfobytype--vb-and-c.md) |



 

## <a name="members"></a>Membres

L’interface **IWMPMetadataText (VB et C#)** ne définit aucun membre.
