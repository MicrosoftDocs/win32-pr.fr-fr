# <a name="playlistitemselectedfocuslostcolor"></a>PLAYLIST. itemSelectedFocusLostColor

L’attribut **itemSelectedFocusLostColor** spécifie ou récupère une valeur indiquant la couleur du texte d’un élément sélectionné dans la sélection lorsque la sélection perd le focus.

``` syntax
        elementID.itemSelectedFocusLostColor
```

## <a name="possible-values"></a>Valeurs possibles

Cet attribut est une **chaîne** en lecture/écriture représentant une couleur. La valeur par défaut est la couleur du texte du bouton Windows.
