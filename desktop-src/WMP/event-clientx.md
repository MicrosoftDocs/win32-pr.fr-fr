# <a name="eventclientx"></a>Event. webin

L' **attribut** WebClient récupère la coordonnée x du pointeur de la souris par rapport à la région cliente de la fenêtre d’application.

``` syntax
event.clientX
```

## <a name="possible-values"></a>Valeurs possibles

Cet attribut est un **nombre** en lecture seule (**long**).

## <a name="requirements"></a>Configuration requise



| Condition requise | Valeur |
|--------------------|------------------------------------------------------|
| Version<br/> | Lecteur Windows Media version 7,0 ou ultérieure<br/> |
