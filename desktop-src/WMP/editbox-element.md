# <a name="editbox-element"></a>Élément EDITBOX

L’élément **EDITBOX** offre aux utilisateurs la possibilité d’entrer du texte dans une apparence.

L’élément **EDITBOX** prend en charge les attributs suivants.



| Attribut                                      | Description                                                                                                   |
|------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [backgroundColor](editbox-backgroundcolor.md) | Spécifie ou récupère la couleur d’arrière-plan du contrôle zone d’édition.                                         |
| [2S](editbox-border.md)                   | Spécifie ou récupère une valeur indiquant si le contrôle de zone d’édition a une bordure.                          |
| [editStyle](editbox-editstyle.md)             | Spécifie ou récupère le style du contrôle zone d’édition.                                                     |
| [fontFace](editbox-fontface.md)               | Spécifie ou récupère la police utilisée pour le texte dans le contrôle de zone d’édition.                                        |
| [fontSize](editbox-fontsize.md)               | Spécifie ou récupère la taille de police du contrôle zone d’édition.                                                |
| [fontStyle](editbox-fontstyle.md)             | Spécifie ou récupère le style de police pour le contrôle zone d’édition.                                               |
| [foregroundColor](editbox-foregroundcolor.md) | Spécifie ou récupère la couleur du texte dans le contrôle de zone d’édition.                                                |
| [motivation](editbox-justification.md)     | Spécifie ou récupère l’alignement du texte dans le contrôle zone d’édition.                                 |
| [lineCount](editbox-linecount.md)             | Récupère le nombre de lignes dans le contrôle zone d’édition.                                                        |
| [readOnly](editbox-readonly.md)               | Spécifie ou récupère une valeur indiquant si le texte du contrôle zone d’édition est en lecture seule ou s’il peut être modifié. |
| [textLimit](editbox-textlimit.md)             | Spécifie ou récupère le nombre maximal de caractères que l’utilisateur peut taper dans le contrôle de zone d’édition.       |
| [value](editbox-value.md)                     | Spécifie ou récupère le texte affiché dans le contrôle de zone d’édition.                                    |
| [wordWrap](editbox-wordwrap.md)               | Spécifie ou récupère une valeur indiquant si le retour automatique à la ligne est activé dans le contrôle de zone d’édition.               |



 

L’élément **EDITBOX** prend en charge les méthodes suivantes.



| Méthode                                             | Description                                                                                         |
|----------------------------------------------------|-----------------------------------------------------------------------------------------------------|
| [getLine](editbox-getline.md)                     | Récupère le texte de la ligne avec l’index spécifié.                                           |
| [getLineFromChar](editbox-getlinefromchar.md)     | Récupère l’index de ligne pour l’index de caractère spécifié.                                         |
| [getLineIndex](editbox-getlineindex.md)           | Récupère l’index de caractère pour l’index de ligne spécifié.                                         |
| [getSelectionEnd](editbox-getselectionend.md)     | Récupère la position de fin du texte sélectionné dans le contrôle zone d’édition.                         |
| [getSelectionStart](editbox-getselectionstart.md) | Récupère la position de départ du texte sélectionné dans le contrôle de zone d’édition.                       |
| [Revanche](editbox-replaceselection.md)   | Remplace la sélection actuelle par le texte spécifié.                                             |
| [setSelection](editbox-setselection.md)           | Sélectionne le texte dans le contrôle de zone d’édition de l’index de début spécifié à l’index de fin spécifié. |



 

L’élément **EDITBOX** prend en charge les attributs ambiants et peut implémenter tous les gestionnaires d’événements ambiants à l’exception de **OnClick**. Pour plus d’informations, consultez [attributs ambiants](ambient-attributes.md) et [gestionnaires d’événements ambiants](ambient-event-handlers.md).

> [!Note]  
> cet élément requiert Lecteur Windows Media pour Windows XP ou version ultérieure.
