# <a name="settings-area-plug-ins"></a>Plug-ins de zone Paramètres

Pour le lecteur Microsoft Windows Media 11 et les versions antérieures, les plug-ins de l’interface utilisateur des paramètres sont affichés dans la zone outils du volet **lecture** en cours sous la zone d’affichage principale et au-dessus des contrôles de transport. Pour le lecteur Microsoft Windows Media 12 et versions ultérieures, les plug-ins de l’interface utilisateur des paramètres s’affichent sous forme de fenêtres distinctes. Un seul plug-in de zone de paramètres peut être activé à la fois.

Les plug-ins de zone de paramètres sont utiles pour afficher des panneaux de contrôle personnalisés similaires aux contrôles d’égaliseur graphique ou de paramètres vidéo.
