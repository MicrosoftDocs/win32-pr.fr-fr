# <a name="about-skins-mobile"></a>À propos des apparences (mobiles)

Une apparence est une interface utilisateur personnalisée pour Windows Media Player. Vous pouvez créer vos propres boutons pour démarrer et arrêter la lecture des médias numériques, ajouter des curseurs pour modifier la position du volume et de la lecture dans l’élément multimédia, et fournir à l’utilisateur des informations de texte telles que le nom d’une chanson. Mieux encore, vous pouvez ajouter votre propre illustration ou utiliser une illustration existante pour créer une apparence unique pour votre apparence.

Les étapes de base de la création d’une apparence sont les suivantes :

1.  **Concevez** votre interface utilisateur. Choisissez les boutons, les zones de texte et les trackbars que vous souhaitez fournir à l’utilisateur afin qu’il puisse contrôler les fonctions que vous avez choisies à l’étape 1. Vous pouvez dessiner vos idées pour voir comment elles s’adaptent à la taille de l’image que vous allez utiliser.
2.  **Créez** l’image que vous souhaitez montrer à l’utilisateur. Il s’agit de plusieurs images qui affichent les images d’arrière-plan, d’autres images que vous pouvez afficher et qui mappent des images si vous utilisez des boutons de région.
3.  **Écrivez** le fichier de définition d’apparence qui reliera les fonctions multimédias numériques, l’interface utilisateur et les images. Vous devez suivre certaines règles pour entrer les données de texte dans le fichier, mais vous pouvez examiner l’apparence par défaut comme un repère.

Vous n’avez pas besoin de suivre ces étapes dans l’ordre indiqué, mais une bonne conception ne sera pas consciente de toutes les possibilités et prendre en charge tous les détails.
