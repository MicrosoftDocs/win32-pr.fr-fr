# <a name="eventshiftkey"></a>Event. shiftKey

L’attribut **shiftKey** récupère une valeur indiquant si la touche Maj était enfoncée lorsque l’événement s’est produit.

``` syntax
event.shiftKey
```

## <a name="possible-values"></a>Valeurs possibles

Cet attribut est une **valeur booléenne** en lecture seule.



| Valeur | Description                                       |
|-------|---------------------------------------------------|
| true  | Indique que la touche Maj était en position inférieure. |
| false | Indique que la touche Maj était à la position précédente.   |



 

## <a name="requirements"></a>Configuration requise



| Condition requise | Valeur |
|--------------------|------------------------------------------------------|
| Version<br/> | Lecteur Windows Media version 7,0 ou ultérieure<br/> |
