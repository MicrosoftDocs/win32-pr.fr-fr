# <a name="equalizersettingsnormalizationaverage"></a>EQUALIZERSETTINGS.normalizationAverage

L’attribut **normalizationAverage** récupère la valeur de normalisation moyenne.

``` syntax
        elementID.normalizationAverage
```

## <a name="possible-values"></a>Valeurs possibles

Cet attribut est un **nombre** en lecture seule (**float**).

## <a name="requirements"></a>Configuration requise



| Condition requise | Valeur |
|--------------------|---------------------------------------------------|
| Version<br/> | Lecteur Windows Media série 9 ou version ultérieure<br/> |
