# <a name="creating-the-region-file"></a>Création du fichier de région

Vous pouvez créer un fichier de région qui indique les zones qui répondent aux pressions. Vous pouvez simplement copier les couches de base de chaque élément vers une nouvelle couche et les colorier pour les faire correspondre aux numéros de couleur que vous utiliserez dans le fichier de définition d’apparence. Assurez-vous que les images que vous créez dans cette couche sont solides et qu’aucun effet n’est appliqué. Vous souhaiterez probablement écrire les numéros de couleur pour chaque image. Vous pouvez récupérer les numéros de couleur à partir du sélecteur de couleurs Photoshop. Vous souhaitez les nombres RGB, qui sont trois nombres décimaux. Chaque nombre est compris entre 0 et 255. Par exemple, un rouge plein serait 255, 0,0.

> [!Note]  
> les fichiers de région ne sont pas utilisés dans Lecteur Windows Media 10 habillages mobiles, car les types de boutons ne sont pas pris en charge dans Lecteur Windows Media 10 mobile ou version ultérieure.

 

L’image suivante est le fichier de région.

![fichier de région](images/ceswmreg.png)

Ce fichier ne contient que quatre images, car seuls les boutons PlayPause, stop, Next et PREV sont des boutons de type Hit.
