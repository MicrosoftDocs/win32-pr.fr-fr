# <a name="axwindowsmediaplayernetwork-property"></a>AxWindowsMediaPlayer. Network, propriété

La propriété Network Obtient une interface IWMPNetwork qui fournit l’accès aux informations et paramètres réseau.

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```CSharp
public IWMPNetwork network {get;}
```


```VB

Public ReadOnly Property network As IWMPNetwork
```





## <a name="property-value"></a>Valeur de la propriété

Interface WMPLib. IWMPNetwork.
