# <a name="initial-lines"></a>Lignes initiales

Chaque fichier de définition d’apparence pour le lecteur Windows Media Mobile doit commencer par une ligne qui spécifie le numéro de version du fichier d’apparence. La ligne suivante fonctionne pour le lecteur Windows Media série 9 pour Windows Mobile 2003 ou version ultérieure :


```C++
[Pocket WMP Skin File v9.0.1]

```
