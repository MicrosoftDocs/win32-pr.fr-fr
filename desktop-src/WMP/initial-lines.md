# <a name="initial-lines"></a>Lignes initiales

chaque fichier de définition d’apparence pour Lecteur Windows Media Mobile doit commencer par une ligne qui spécifie le numéro de version du fichier d’apparence. la ligne suivante fonctionne pour Lecteur Windows Media série 9 pour Windows Mobile 2003 ou version ultérieure :


```C++
[Pocket WMP Skin File v9.0.1]

```
