# <a name="iwmpmetadatapicturepicturetype-property"></a>IWMPMetadataPicture ::p propriété ictureType

La `pictureType` propriété obtient le type d’image de l’image représentée par l’attribut de métadonnées.

## <a name="syntax"></a>Syntaxe


```CSharp
public System.String pictureType {get; set;}
```


```VB

Public ReadOnly Property pictureType As System.String
```





## <a name="property-value"></a>Valeur de la propriété

**System. String** qui est le type d’image de l’image.

## <a name="remarks"></a>Remarques

Avant d’utiliser cette propriété, vous devez disposer d’un accès en lecture à la bibliothèque. Pour plus d’informations, consultez [accès à la bibliothèque](library-access.md).
