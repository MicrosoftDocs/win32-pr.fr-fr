# <a name="control-of-playlists"></a>Contrôle des sélections

Le contrôle des sélections est géré par le client. Les sélections sont analysées par le lecteur Windows Media, qui effectue des actions basées sur les éléments de la sélection. Le client contrôle l’expérience de diffusion en continu.
