# <a name="themeauthorversion"></a>THÈME. authorVersion

L’attribut **authorVersion** spécifie ou récupère le numéro de version de l’apparence comme assigné par l’auteur.

``` syntax
theme.authorVersion
```

## <a name="possible-values"></a>Valeurs possibles

Cet attribut est une **chaîne** en lecture seule sans valeur par défaut.

## <a name="requirements"></a>Configuration requise



| Condition requise | Valeur |
|--------------------|------------------------------------------------------|
| Version<br/> | Lecteur Windows Media version 7,0 ou ultérieure<br/> |
