# <a name="ambientattributesresizeimages"></a>AmbientAttributes.resizeImages

L’attribut **resizeImages** spécifie ou récupère une valeur indiquant si les images contenues dans le contrôle sont redimensionnées automatiquement lorsque la taille du contrôle change.

``` syntax
        elementID.resizeImages
```

## <a name="possible-values"></a>Valeurs possibles

Cet attribut est une **valeur booléenne** en lecture/écriture.



| Valeur | Description                                           |
|-------|-------------------------------------------------------|
| true  | Les images contenues dans le contrôle sont redimensionnées automatiquement. |
| false | Les images contenues dans le contrôle ont une taille fixe.    |
