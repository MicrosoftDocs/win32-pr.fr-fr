# <a name="iwmpcdromripripstate-property"></a>IWMPCdromRip :: ripState, propriété

La propriété **ripState** obtient une valeur d’énumération qui indique l’état actuel du processus d’extraction.

## <a name="syntax"></a>Syntaxe


```CSharp
public WMPRipState ripState {get; set;}
```


```VB

Public ReadOnly Property ripState As WMPRipState
```





## <a name="property-value"></a>Valeur de la propriété

**Wmplib. WMPRipState** qui est une valeur de l’énumération **WMPRipState** qui indique l’état actuel.
