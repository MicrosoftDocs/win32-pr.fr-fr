# <a name="equalizersettingscurrentspeakername"></a>EQUALIZERSETTINGS.currentSpeakerName

L’attribut **currentSpeakerName** récupère le nom du paramètre d’orateur actuel.

``` syntax
        elementID.currentSpeakerName
```

## <a name="possible-values"></a>Valeurs possibles

Cet attribut est une **chaîne** en lecture seule qui contient l’une des valeurs suivantes.



| Valeur           | Description                              |
|-----------------|------------------------------------------|
| Un casque      | Les haut-parleurs actuels sont des casques.     |
| Haut-parleurs normaux | Les haut-parleurs actuels sont de taille normale. |
| Grands intervenants  | Les haut-parleurs actuels sont volumineux.          |



 

## <a name="requirements"></a>Configuration requise



| Condition requise | Valeur |
|--------------------|---------------------------------------------------|
| Version<br/> | Lecteur Windows Media série 9 ou version ultérieure<br/> |
