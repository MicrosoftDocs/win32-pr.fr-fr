# <a name="art-files-mobile"></a>Fichiers art (mobile)

Vous devrez créer des fichiers artistiques pour votre apparence. Il existe plusieurs types de fichiers artistiques que vous pouvez créer et qui doivent être dans un format de fichier art spécifique.

Le tableau suivant décrit les sept types de fichiers artistiques que vous pouvez créer pour votre apparence.



| Type de fichier art                            | Description                                                                                                |
|------------------------------------------|------------------------------------------------------------------------------------------------------------|
| [Fichier d’arrière-plan](background-file.md)   | Obligatoire. L’arrière-plan est l’image sur laquelle toutes les images de bouton sont superposées                        |
| [Fichier désactivé](disabled-file.md)       | Obligatoire. Images utilisées pour indiquer que les boutons de Poussée n’ont aucun effet.                                |
| [Fichier Push](pushed-file.md)           | Obligatoire. Définit les images qui sont affichées lorsque l’utilisateur pousse les boutons.                              |
| [Fichiers de région](region-files.md)         | Définit des bitmaps qui utilisent des zones de couleur pour représenter les emplacements de réponse TAP pour les boutons de type d’accès      |
| [Super fichiers](super-files.md)           | Définit les images désactivées pour trackbars.                                                                 |
| [Fichier SeekThumb](seekthumb-file.md)     | Définit une image à utiliser conjointement avec un TrackBar pour indiquer la position actuelle dans l’élément multimédia. |
| [Fichier VolumeThumb](volumethumb-file.md) | Définit une image à utiliser conjointement avec un TrackBar pour indiquer la position du contrôle du volume.            |



 

Les sections suivantes fournissent plus d’informations sur les fichiers art.

-   [Formats de fichier art](art-file-formats-mobile.md)
-   [Palettes de couleurs](color-palettes.md)
-   [Tailles d’image](image-sizes.md)
-   [Positionnement du bouton](button-placement.md)

Pour plus d’informations sur les types de fichiers art et sur la façon de les identifier dans le fichier de définition d’apparence, consultez la section [bitmaps](bitmaps.md) de la référence de l’apparence.
