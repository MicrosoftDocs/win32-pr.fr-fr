# <a name="disabled-file"></a><span data-ttu-id="c991a-110">Fichier désactivé</span><span class="sxs-lookup"><span data-stu-id="c991a-110">Disabled File</span></span>

<span data-ttu-id="c991a-111">Le fichier désactivé contient les images qui seront affichées lorsqu’une fonction de bouton particulière ne peut pas être utilisée ou qu’un état de bouton est désactivé.</span><span class="sxs-lookup"><span data-stu-id="c991a-111">The Disabled file contains the images that will be displayed when a particular button function cannot be used or a button state is off.</span></span> <span data-ttu-id="c991a-112">Par exemple, si une sélection vide est définie, les boutons **suivant** et **préc** s’affichent et ils doivent être affichés à l’aide d’une image désactivée.</span><span class="sxs-lookup"><span data-stu-id="c991a-112">For example, if an empty playlist is defined, the **Next** and **Prev** buttons will be displayed, and they should be displayed using a disabled image.</span></span> <span data-ttu-id="c991a-113">En outre, pour les boutons bascule, l’image désactivée est utilisée pour indiquer que l’État est désactivé.</span><span class="sxs-lookup"><span data-stu-id="c991a-113">Also, for toggle buttons, the Disabled image is used to indicate that the state is off.</span></span>

<span data-ttu-id="c991a-114">L’image suivante est un fichier désactivé standard.</span><span class="sxs-lookup"><span data-stu-id="c991a-114">The following image is a typical Disabled file.</span></span>

<span data-ttu-id="c991a-116">Cela stocke les images des boutons de type d’accès qui sont désactivés.</span><span class="sxs-lookup"><span data-stu-id="c991a-116">This stores the images for hit-type buttons that are disabled.</span></span> <span data-ttu-id="c991a-117">Les images sont similaires au fichier d’arrière-plan, mais les couleurs sont plus claires.</span><span class="sxs-lookup"><span data-stu-id="c991a-117">The images are similar to the Background file, but the colors are lighter.</span></span> <span data-ttu-id="c991a-118">À l’aide de l’offset défini dans la section bitmaps du fichier de définition d’apparence, le bouton images est aligné avec l’image de fichier d’arrière-plan.</span><span class="sxs-lookup"><span data-stu-id="c991a-118">Using the offset defined in the Bitmaps section of the skin definition file, the button images line up with the Background file image.</span></span>
