# <a name="background-file"></a>Fichier d’arrière-plan

Chaque apparence a besoin d’un fichier d’arrière-plan, qui contient l’image que l’utilisateur verra quand l’apparence est affichée pour la première fois. Ce fichier est défini en tant que fichier d’arrière-plan dans le fichier de définition d’apparence.

L’image suivante est un fichier d’arrière-plan classique.

![fichier d’arrière-plan](images/cesdkbak.png)

Il s’agit de l’image d’arrière-plan que les utilisateurs verront au préalable. Toutefois, ils verront également les deux images Thumb aux emplacements appropriés sur le trackbars pour Seek (ci-dessus) et le volume (ci-dessous). En outre, tous les boutons désactivés s’affichent à l’aide de leurs images désactivées à la place.
