# <a name="ambient-attributes"></a>Attributs ambiants

Les attributs et les méthodes ambiants sont pris en charge par tous les éléments d’apparence appropriés, sauf indication contraire.

Les attributs suivants sont ambiants.



| Attribut                                                        | Description                                                                                                                               |
|------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------|
| [accName](ambientattributes-accname.md)                         | Spécifie ou récupère un nom pour tout élément.                                                                                            |
| [accDescription](ambientattributes-accdescription.md)           | Spécifie ou récupère une description pour tout élément.                                                                                     |
| [accKeyboardShortcut](ambientattributes-acckeyboardshortcut.md) | Spécifie ou récupère la description d’un raccourci clavier pour tout élément.                                                                   |
| [alphaBlend](ambientattributes-alphablend.md)                   | Spécifie ou récupère une valeur pour la fusion alpha d’une **vue**, d’une sous- **vue** ou d’un widget d’interface utilisateur.                                                |
| [ballon](ambientattributes-bottom.md)                           | Spécifie ou récupère la coordonnée inférieure du contrôle.                                                                              |
| [clippingColor](ambientattributes-clippingcolor.md)             | Spécifie ou récupère la couleur à découper à partir de la bitmap **clippingImage** .                                                           |
| [clippingImage](ambientattributes-clippingimage.md)             | Spécifie ou récupère la zone dans laquelle détourer le contrôle.                                                                                 |
| [elementType](ambientattributes-elementtype.md)                 | Récupère le type de l’élément (par exemple, **Button**).                                                                             |
| [activé](ambientattributes-enabled.md)                         | Spécifie ou récupère une valeur indiquant si le contrôle est activé ou désactivé.                                                     |
| [height](ambientattributes-height.md)                           | Spécifie ou récupère la hauteur du contrôle.                                                                                         |
| [horizontalAlignment](ambientattributes-horizontalalignment.md) | Spécifie ou récupère une valeur qui indique le positionnement horizontal du contrôle lorsque la **vue** ou la sous- **vue** parente est redimensionnée. |
| [id](ambientattributes-id.md)                                   | Spécifie ou récupère l’identificateur d’un contrôle. Ne peut être défini qu’au moment de la conception.                                                       |
| [gauche](ambientattributes-left.md)                               | Spécifie ou récupère la coordonnée gauche du contrôle.                                                                                |
| [nineGridMargins](ambientattributes-ninegridmargins.md)         | Spécifie des largeurs de marge pour la mise à l’échelle non uniforme de l’élément d’apparence.                                                                      |
| [Passage](ambientattributes-passthrough.md)                 | Spécifie ou récupère une valeur indiquant si le contrôle passera tous les événements de souris au contrôle situé sous celui-ci.                 |
| [resizeImages](ambientattributes-resizeimages.md)               | Spécifie ou récupère une valeur indiquant si les images contenues dans le contrôle sont redimensionnées automatiquement lorsque la taille du contrôle change.     |
| [Oui](ambientattributes-right.md)                             | Spécifie ou récupère la coordonnée droite du contrôle.                                                                               |
| [Tabulation](ambientattributes-tabstop.md)                         | Spécifie ou récupère une valeur indiquant si le contrôle sera dans l’ordre de tabulation.                                               |
| [top](ambientattributes-top.md)                                 | Spécifie ou récupère la coordonnée supérieure du contrôle.                                                                                 |
| [verticalAlignment](ambientattributes-verticalalignment.md)     | Spécifie ou récupère une valeur qui indique le positionnement vertical du contrôle lorsque la **vue** ou la sous- **vue** parente est redimensionnée.   |
| [visible](ambientattributes-visible.md)                         | Spécifie ou récupère la visibilité du contrôle.                                                                                     |
| [width](ambientattributes-width.md)                             | Spécifie ou récupère la largeur du contrôle.                                                                                          |
| [zIndex](ambientattributes-zindex.md)                           | Spécifie ou récupère l’ordre dans lequel le contrôle est rendu.                                                                        |



 

Les méthodes suivantes sont ambiantes.



| Méthode                                             | Description                                                                                                                                                          |
|----------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [alphaBlendTo](ambientattributes-alphablendto.md) | Ajuste la propriété **alphaBlend** sur une période donnée.                                                                                                           |
| [moveSizeTo](ambientattributes-movesizeto.md)     | Déplace le contrôle et spécifie une nouvelle taille pour le contrôle dans le nouvel emplacement. Le contrôle se déplace et se redimensionne de manière animée sur la période spécifiée. |
| [DéplacerVers](ambientattributes-moveto.md)             | Déplace le contrôle vers un nouvel emplacement à une vitesse linéaire.                                                                                                               |
| [slideTo](ambientattributes-slideto.md)           | Déplace le contrôle vers un nouvel emplacement. Le contrôle se déplace de manière non linéaire sur la période spécifiée.                                                       |
