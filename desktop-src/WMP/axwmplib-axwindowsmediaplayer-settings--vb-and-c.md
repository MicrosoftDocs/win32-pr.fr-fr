# <a name="axwindowsmediaplayersettings-property"></a>AxWindowsMediaPlayer. Settings, propriété

la propriété settings obtient une interface IWMPSettings qui fournit un moyen de modifier plusieurs paramètres de Lecteur Windows Media.

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```CSharp
public IWMPSettings settings {get;}
```


```VB

Public ReadOnly Property settings As IWMPSettings
```





## <a name="property-value"></a>Valeur de la propriété

Interface WMPLib. IWMPSettings.
