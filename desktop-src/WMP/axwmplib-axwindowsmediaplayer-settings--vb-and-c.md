# <a name="axwindowsmediaplayersettings-property"></a>AxWindowsMediaPlayer. Settings, propriété

La propriété Settings obtient une interface IWMPSettings qui fournit un moyen de modifier les différents paramètres du lecteur Windows Media.

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```CSharp
public IWMPSettings settings {get;}
```


```VB

Public ReadOnly Property settings As IWMPSettings
```





## <a name="property-value"></a>Valeur de la propriété

Interface WMPLib. IWMPSettings.

## <a name="requirements"></a>Configuration requise



| Condition requise | Valeur |
|----------------------|----------------------------------------------------------------------------------------------------------------------------|
| Version<br/>   | Lecteur Windows Media série 9 ou version ultérieure<br/>                                                                          |
