# <a name="about-compatibility"></a>À propos de la compatibilité

Toutes les versions de Windows Media Player Mobile ne fonctionneront pas sur chaque appareil. En outre, certaines versions sont préinstallées sur l’appareil, tandis que d’autres sont disponibles pour le téléchargement et l’installation à partir du site Web de Microsoft. Les versions ultérieures peuvent offrir des fonctionnalités d’apparence non prises en charge par les versions antérieures.

Le tableau suivant décrit en détail la compatibilité de Windows Media Player Mobile avec divers appareils.

|-----------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------|---------------------------------------------------|
| Lecteur Windows Media 1,0          | **Appareils Pocket PC 3,0 :** Casio Cassiopeia E-115/125, EM-500<br/> Compaq Aero 15xx, Aero 21xx, iPAQ 3150, iPAQ 36xx<br/> HP Jornada 54x<br/>                   | Livré avec l’appareil.                                             | Audio uniquement.                                       |
| Lecteur Windows Media 1,1          | **PC de taille Palm :** Appareils de classe E-10 *x* Casio Cassiopeia<br/> Compaq Aero 21 *XX* et Aero 15 *XX*<br/> HP Jornada 430<br/>                                      | Version Web.                                                     | Audio uniquement.                                       |
| Lecteur Windows Media 1,2          | **PC de poche :** HP Jornada 820, HP Jornada 680/690/720<br/> Vadem Clio C-1050<br/> Telios HC-VJ1c/HC-AJ2<br/> Appareils NEC mobile Pro 780/880<br/> | Version Web (HPC 3,0).
