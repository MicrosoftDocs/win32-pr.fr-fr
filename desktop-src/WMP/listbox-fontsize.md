# <a name="listboxfontsize"></a>LISTBOX. FontSize

L’attribut **FontSize** spécifie ou récupère la taille de police pour le contrôle de zone de liste.

``` syntax
        elementID.fontSize
```

## <a name="possible-values"></a>Valeurs possibles

Cet attribut est un **nombre** en lecture/écriture (**long**) qui spécifie la taille de police en points. La valeur par défaut est 10.

## <a name="requirements"></a>Configuration requise



| Condition requise | Valeur |
|--------------------|---------------------------------------------------------|
| Version<br/> | Lecteur Windows Media pour Windows XP ou version ultérieure<br/> |
