# <a name="online-stores-welcome-kit"></a>Kit de bienvenue des magasins en ligne

Cette section contient le kit de bienvenue des magasins en ligne, qui est un ensemble d’informations sur la mise en ligne d’un magasin de musique en ligne pour le lecteur Windows Media. Le kit contient les rubriques suivantes.



| Rubrique                                                                                                          | Description                                                                                                        |
|----------------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------|
| [Présentation du kit de bienvenue des magasins en ligne](introduction-to-the-online-stores-welcome-kit.md)             | Fournit une présentation du kit de bienvenue, des informations de contact et une liste de documents connexes.                 |
| [Obtention d’un magasin de musique en ligne sur un tableau](getting-an-online-music-store-on-board.md)                           | Décrit le processus de mise en ligne d’un magasin de musique en ligne pour le lecteur Windows Media.                         |
| [Configuration requise pour les magasins musicaux en ligne](requirements-for-online-music-stores.md)                               | Décrit certaines exigences clés concernant cusomization et la lecture.                                               |
| [Contrats et licences pour les magasins musicaux en ligne](contracts-and-licenses-for-online-music-stores.md)           | Décrit les contrats qui doivent être signés avant qu’un magasin de musique en ligne puisse être mis sur le tableau.                 |
| [Formulaire d’informations de contact pour un magasin de musique en ligne](contact-information-form-for-an-online-music-store.md)   | Fournit un formulaire que les concepteurs d’un magasin de musique en ligne peuvent utiliser pour fournir des informations de contact à Microsoft. |
| [Formulaire d’informations de démarrage pour un magasin de musique en ligne](startup-information-form-for-an-online-music-store.md)   | Fournit un formulaire que les concepteurs d’un magasin de musique en ligne peuvent utiliser pour fournir des informations de démarrage à Microsoft. |
| [Tests de validation pour les magasins de musique de type 2 en ligne](validation-tests-for-type-2-online-music-stores.md)         | Décrit les tests effectués par Microsoft pour valider un magasin de musique de type 2 en ligne.                               |
| [Problèmes d’intégration courants pour les magasins de musique en ligne](common-on-boarding-problems-for-online-music-stores.md) | Décrit les problèmes courants que les banques de musique en ligne ont rencontrées au cours du processus de mise à la bord.         |
