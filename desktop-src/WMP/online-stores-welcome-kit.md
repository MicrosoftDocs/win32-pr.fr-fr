| Rubrique                                                                                                          | Description                                                                                                        |

| [obtention d’un magasin Musique en ligne sur le tableau](getting-an-online-music-store-on-board.md)                           | décrit le processus de mise en ligne d’un magasin de musique en ligne pour Lecteur Windows Media.                         |
