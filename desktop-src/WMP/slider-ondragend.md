# <a name="sliderondragend"></a>Slider. onDragEnd

Le gestionnaire d’événements **onDragEnd** gère un événement qui se produit lorsque le bouton gauche de la souris est relâché après une opération de glissement.

``` syntax
onDragEnd
```

## <a name="requirements"></a>Configuration requise



| Condition requise | Valeur |
|--------------------|------------------------------------------------------|
| Version<br/> | Lecteur Windows Media version 7,0 ou ultérieure<br/> |
