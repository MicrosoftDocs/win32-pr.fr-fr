# <a name="displaying-web-pages-in-windows-media-player"></a>Affichage des pages Web dans le lecteur Windows Media

La fonctionnalité HTMLView du lecteur Windows Media permet aux fournisseurs de contenu Internet (ICP) d’utiliser la fonctionnalité de **lecture à présent** du lecteur en mode plein écran pour afficher le contenu Web lors de la lecture du contenu multimédia numérique. (HTMLView n’est pas pris en charge par le contrôle du lecteur Windows Media.) Par le passé, les ICP devaient afficher des publications dans des fenêtres de navigateur Web distinctes (appelées « publicité contextuelle ») ou devaient ajouter des lecteurs multimédias numériques personnalisés à leurs pages Web. Avec HTMLView, les ICP peuvent fournir une expérience personnalisée et intégrée créée spécifiquement pour le lecteur Windows Media série 9 et le lecteur Windows Media 10, tout en veillant à ce que le contenu multimédia numérique soit toujours lu dans les versions antérieures du lecteur Windows Media.

Cette section fournit des informations sur l’utilisation de HTMLView, la création de contenu HTMLView et la résolution des problèmes potentiels que vous pouvez rencontrer lors de l’utilisation de la fonctionnalité HTMLView dans les rubriques suivantes :

-   [Création d’une présentation HTMLView](creating-an-htmlview-presentation.md)
-   [Ajout d’un contrôle de lecteur Windows Media incorporé](adding-an-embedded-windows-media-player-control.md)
-   [S’assurer que le lecteur Windows Media ouvre le contenu HTMLView](ensuring-that-windows-media-player-opens-the-htmlview-content.md)
