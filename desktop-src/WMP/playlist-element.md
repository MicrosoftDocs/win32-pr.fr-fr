# <a name="playlist-element"></a>Élément PLAYLIST

L’élément **playlist** offre un moyen d’organiser les éléments multimédias dans une liste pour faciliter leur manipulation à l’aide des méthodes et attributs suivants. Des éléments de **sélection** prédéfinis sont également fournis pour des raisons pratiques. Des colonnes personnalisées peuvent être spécifiées pour une sélection en incluant des éléments de **colonne** comme enfants de l’élément **playlist** .

L’élément **playlist** prend en charge les attributs suivants.



| Attribut                                                                                 | Description                                                                                                                                |
|-------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------|
| [allowColumnSorting](playlist-allowcolumnsorting.md)                                     | Spécifie ou récupère une valeur indiquant si le tri sur les en-têtes de colonnes est autorisé.                                                    |
| [allowItemEditing](playlist-allowitemediting.md)                                         | Spécifie ou récupère une valeur indiquant si les éléments d’une liste de lecture prennent en charge la modification sur place.                                            |
| [backgroundColor](playlist-backgroundcolor.md)                                           | Spécifie ou récupère la couleur d’arrière-plan.                                                                                               |
| [backgroundImage](playlist-backgroundimage.md)                                           | Spécifie ou récupère l’image d’arrière-plan.                                                                                               |
| [checkboxesVisible](playlist-checkboxesvisible.md)                                       | Spécifie ou récupère une valeur indiquant si les cases à cocher sont visibles.                                                                  |
| [columnCount](playlist-columncount.md)                                                   | Récupère le nombre de colonnes affichées.                                                                                                     |
| [columnOrder](playlist-columnorder.md)                                                   | Spécifie ou récupère l’ordre des colonnes de la liste de sélection.                                                                                  |
| [colonnes](playlist-columns.md)                                                           | Définit les colonnes qui apparaissent dans l’élément **playlist** .                                                                               |
| [columnsVisible](playlist-columnsvisible.md)                                             | Spécifie ou récupère une valeur indiquant si les colonnes sont affichées.                                                                       |
| [destination](playlist-copying.md)                                                           | Récupère une valeur indiquant si l’élément de **sélection** est en cours de copie.                                                    |
| [disabledItemColor](playlist-disableditemcolor.md)                                       | Spécifie ou récupère la couleur d’une piste de CD désactivée ou du contenu en ligne en mode hors connexion.                                                 |
| [dropDownBackgroundImage](playlist-dropdownbackgroundimage.md)                           | Spécifie ou récupère le nom de l’image qui s’affiche dans l’arrière-plan de la liste déroulante.                                        |
| [dropDownImage](playlist-dropdownimage.md)                                               | Spécifie ou récupère le nom de l’image utilisée pour le bouton de liste déroulante qui s’affiche sur le bord droit de la liste déroulante. |
| [dropDownList](playlist-dropdownlist.md)                                                 | Spécifie ou récupère une valeur indiquant les éléments qui apparaissent dans la liste déroulante pour une instance donnée de l’élément de **sélection** .   |
| [dropDownToolTip](playlist-dropdowntooltip.md)                                           | Spécifie ou récupère l’info-bulle affichée lorsque l’utilisateur pointe sur le menu déroulant de l’élément de **sélection** .                                |
| [dropDownVisible](playlist-dropdownvisible.md)                                           | Spécifie ou récupère une valeur indiquant si le sélecteur de liste déroulante d’élément de **sélection** est visible.                                  |
| [editButtonVisible](playlist-editbuttonvisible.md)                                       | Spécifie ou récupère une valeur indiquant si le bouton modifier de l’élément de **sélection** est visible.                                         |
| [foregroundColor](playlist-foregroundcolor.md)                                           | Spécifie ou récupère la couleur de premier plan.                                                                                               |
| [hueShift](playlist-hueshift.md)                                                         | Spécifie ou récupère la quantité de décalage de la teinte des images de la liste déroulante.                                                     |
| [itemCount](playlist-itemcount.md)                                                       | Récupère le nombre d’éléments actuellement affichés dans l’élément de **sélection** .                                                             |
| [itemErrorColor](playlist-itemerrorcolor.md)                                             | Spécifie ou récupère la couleur de surbrillance qui indique un élément de sélection qui a une condition d’erreur.                                     |
| [itemMedia](playlist-itemmedia.md)                                                       | Récupère l’objet **Media** correspondant à l’index donné dans l’élément **playlist** .                                               |
| [itemPlayingBackgroundColor](playlist-itemplayingbackgroundcolor.md)                     | Spécifie ou récupère la couleur d’arrière-plan de l’élément de sélection en cours de lecture.                                                        |
| [itemPlayingColor](playlist-itemplayingcolor.md)                                         | Spécifie ou récupère la couleur de surbrillance qui indique l’élément en cours de lecture dans la sélection.                                      |
| [itemPlaylist](playlist-itemplaylist.md)                                                 | Récupère la sélection pour l’élément multimédia qui est affiché à l’index donné dans l’élément **playlist** .                                |
| [itemSelectedBackgroundColor](playlist-itemselectedbackgroundcolor.md)                   | Spécifie ou récupère une valeur indiquant la couleur d’arrière-plan d’un élément sélectionné dans la sélection.                                         |
| [itemSelectedBackgroundFocusLostColor](playlist-itemselectedbackgroundfocuslostcolor.md) | Spécifie ou récupère une valeur indiquant la couleur de texte d’un élément sélectionné dans la sélection.                                               |
| [itemSelectedColor](playlist-itemselectedcolor.md)                                       | Spécifie ou récupère une valeur indiquant la couleur de texte d’un élément sélectionné dans la sélection.                                               |
| [itemSelectedFocusLostColor](playlist-itemselectedfocuslostcolor.md)                     | Spécifie ou récupère une valeur indiquant la couleur de texte d’un élément sélectionné dans la sélection lorsque la sélection perd le focus.                 |
| [leftStatus](playlist-leftstatus.md)                                                     | Spécifie ou récupère le texte d’état affiché sur le côté gauche et le bas de l’élément de **sélection** .                          |
| [playlist](playlist-playlist.md)                                                         | Spécifie ou récupère l’objet **playlist** dans lequel l’élément **playlist** fournit une interface.                                    |
| [playlistItemsVisible](playlist-playlistitemsvisible.md)                                 | Spécifie ou récupère une valeur indiquant si les éléments de la sélection sont visibles.                                                       |
| [rightStatus](playlist-rightstatus.md)                                                   | Spécifie ou récupère le texte d’état affiché sur le côté droit et le bas de l’élément de **sélection** .                         |
| [saturation](playlist-saturation.md)                                                     | Spécifie ou récupère la valeur de saturation des images de liste déroulante.                                                                       |
| [statusColor](playlist-statuscolor.md)                                                   | Spécifie ou récupère la couleur de la ligne d’État dans l’élément de **sélection** .                                                           |
| [statusTextColor](playlist-statustextcolor.md)                                           | Spécifie ou récupère une valeur indiquant la couleur du texte d’État.                                                                    |
| [toolbarVisible](playlist-toolbarvisible.md)                                             | Spécifie ou récupère une valeur indiquant si la barre d’outils de sélection s’affiche.                                                           |



 

L’élément **playlist** prend en charge les méthodes suivantes.



| Méthode                                                              | Description                                                                                                               |
|---------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------|
| [abortCopy](playlist-abortcopy.md)                                 | Annule une opération de copie.                                                                                                 |
| [addSelectedToPlaylist](playlist-addselectedtoplaylist.md)         | Ajoute l’élément sélectionné à la sélection.                                                                                   |
| [copy](playlist-copy.md)                                           | Commence une opération de copie à partir du CD.                                                                                      |
| [deleteSelected](playlist-deleteselected.md)                       | Supprime l’élément sélectionné de la sélection.                                                                              |
| [deleteSelectedFromLibrary](playlist-deleteselectedfromlibrary.md) | Supprime l’élément sélectionné de la sélection et de la bibliothèque.                                                         |
| [getNextCheckedItem](playlist-getnextcheckeditem.md)               | Récupère l’index de l’élément activé suivant dans la liste de sélection à la suite de l’index spécifié.                               |
| [getNextCheckedItem2](playlist-getnextcheckeditem2.md)             | Récupère l’index de l’élément activé suivant dans la liste de sélection à la suite de l’index spécifié. Fonctionne avec des sélections imbriquées.  |
| [getNextSelectedItem](playlist-getnextselecteditem.md)             | Récupère l’index du prochain élément sélectionné dans la liste de sélection à la suite de l’index spécifié.                              |
| [getNextSelectedItem2](playlist-getnextselecteditem2.md)           | Récupère l’index du prochain élément sélectionné dans la liste de sélection à la suite de l’index spécifié. Fonctionne avec des sélections imbriquées. |
| [moveSelectedDown](playlist-moveselecteddown.md)                   | Déplace l’élément sélectionné d’une position vers le dessous dans la liste.                                                                    |
| [moveSelectedUp](playlist-moveselectedup.md)                       | Déplace l’élément sélectionné d’une position vers le haut dans la liste.                                                                      |
| [setCheckedState](playlist-setcheckedstate.md)                     | Spécifie que l’élément indexé de la sélection est activé.                                                               |
| [setCheckedState2](playlist-setcheckedstate2.md)                   | Définit l’état activé de l’élément avec l’index spécifié dans l’élément de **sélection** . Fonctionne avec des sélections imbriquées.     |
| [setColumnResizeMode](playlist-setcolumnresizemode.md)             | Spécifie le mode de redimensionnement de la colonne indexée.                                                                            |
| [setColumnWidth](playlist-setcolumnwidth.md)                       | Spécifie la largeur de colonne et modifie le mode de redimensionnement de la colonne en « wmpcrmFixed ».                                    |
| [setSelectedState](playlist-setselectedstate.md)                   | Spécifie que l’élément indexé dans la sélection est sélectionné.                                                              |
| [setSelectedState2](playlist-setselectedstate2.md)                 | Définit l’état sélectionné de l’élément avec l’index spécifié dans l’élément de **sélection** . Fonctionne avec des sélections imbriquées.    |
| [sortColumn](playlist-sortcolumn.md)                               | Trie les données dans la colonne spécifiée.                                                                                   |



 

L’élément **playlist** prend en charge les attributs ambiants et peut implémenter les gestionnaires d’événements ambiants, sauf indication contraire. Pour plus d’informations, consultez [attributs ambiants](ambient-attributes.md) et [gestionnaires d’événements ambiants](ambient-event-handlers.md).

Les sélections prédéfinies sont des éléments de **sélection** normaux avec différents paramètres d’attribut communs spécifiés par défaut. Les sélections prédéfinies suivantes sont disponibles.



| SÉLECTION prédéfinie                      | Description                                                       |
|------------------------------------------|-------------------------------------------------------------------|
| [DROPDOWNPLAYLIST](dropdownplaylist.md) | **Sélection** de liste déroulante sans éléments visibles.                   |
| [ITEMSPLAYLIST](itemsplaylist.md)       | **Sélection** de liste déroulante sans en-tête d’élément ou de colonne visible. |
