# <a name="color-palettes-windows-media-player-sdk"></a>palettes de couleurs (kit de développement logiciel Lecteur Windows Media)

Différents appareils ont des palettes de couleurs différentes. Vous souhaiterez probablement créer différentes apparences pour chaque palette de couleurs. Les palettes de couleurs suivantes sont recommandées :

-   16 bits
-   8 bits
-   Nuances de gris (2 bits)
