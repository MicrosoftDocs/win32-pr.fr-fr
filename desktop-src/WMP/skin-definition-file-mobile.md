# <a name="skin-definition-file-mobile"></a>Fichier de définition d’apparence (mobile)

Le fichier de définition d’apparence est l’endroit où vous rassemblez tous les éléments. Vous définissez les images à utiliser, spécifiez les éléments d’interface utilisateur que vous souhaitez que l’utilisateur voit, puis tapez les fonctions, les coordonnées, les noms de fichiers, et ainsi de suite.

Les sections suivantes fournissent des détails sur la création du fichier de définition d’apparence :

-   [Création d’un fichier de définition d’apparence](creating-a-skin-definition-file.md)
-   [Éditeurs de texte](text-editors.md)
-   [Règles pour chaque ligne](rules-for-each-line.md)
-   [Éléments obligatoires](required-elements.md)
