# <a name="progressbar-element"></a>PROGRESSBAR, élément

L’élément **PROGRESSBAR** offre un moyen d’afficher les informations de progression dans un contrôle horizontal ou vertical. Cet élément est identique à l’élément **Slider** . Une barre de progression est normalement un curseur qui n’est pas interactif, mais il n’y a rien qui l’empêche d’être utilisée de manière interactive. Cet élément est fourni uniquement comme une balise de commodité pour que la non-interaction soit claire dans votre code, si vous choisissez de l’utiliser de cette façon. Elle n’a aucune autre valeur.
