# <a name="playlistplaylistitemsvisible"></a>PLAYLIST. playlistItemsVisible

L’attribut **playlistItemsVisible** spécifie ou récupère une valeur indiquant si la zone éléments de la sélection est visible.

``` syntax
        elementID.playlistItemsVisible
```

## <a name="possible-values"></a>Valeurs possibles

Cet attribut est une **valeur booléenne** en lecture/écriture.



| Valeur | Description                              |
|-------|------------------------------------------|
| true  | Par défaut. La zone éléments de la sélection est visible. |
| false | La zone éléments de la sélection n’est pas visible.      |



 

## <a name="remarks"></a>Remarques

La zone éléments de la sélection comprend les en-têtes de colonnes, le contenu des colonnes et les barres de défilement (le cas échéant).

| Condition requise | Valeur |
|--------------------|------------------------------------------------------|
| Version<br/> | Lecteur Windows Media version 7,0 ou ultérieure<br/> |
