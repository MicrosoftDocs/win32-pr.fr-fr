# <a name="ambientattributesid"></a>AmbientAttributes.id

L’attribut **ID** spécifie ou récupère l’identificateur d’un contrôle. cet attribut prend en charge tout nom autorisé par Microsoft Visual Basic pour Applications. Ne peut être défini qu’au moment de la conception.

``` syntax
        elementID.id
```

## <a name="possible-values"></a>Valeurs possibles

Cet attribut est une **chaîne** spécifiée au moment de la conception et en lecture seule par la suite. Sa valeur par défaut est

\_ \_ *Nombre* ElementType sans nom

où *ElementType* est le nom du type de contrôle, et *num* est un nombre correspondant à la position du contrôle dans la séquence de tous les contrôles sans nom dans le thème. Ce nombre est remis à zéro pour chaque nouveau thème.
