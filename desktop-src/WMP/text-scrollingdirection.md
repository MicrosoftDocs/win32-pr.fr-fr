# <a name="textscrollingdirection"></a>TEXT. scrollingDirection

L’attribut **scrollingDirection** spécifie ou récupère la direction du texte à faire défiler.

``` syntax
        elementID.scrollingDirection
```

## <a name="possible-values"></a>Valeurs possibles

Cet attribut est une **chaîne** en lecture/écriture.



| Valeur | Description                         |
|-------|-------------------------------------|
| Gauche  | Par défaut. Faites défiler de droite à gauche. |
| Right | Faites défiler de gauche à droite.          |



 

## <a name="remarks"></a>Remarques

Si le **défilement** est défini sur false, **scrollingDirection** est ignoré.

Pour obtenir un exemple illustrant l’utilisation des attributs de l’élément de **texte** , consultez l’attribut [value](text-value.md) .

| Condition requise | Valeur |
|--------------------|------------------------------------------------------|
| Version<br/> | Lecteur Windows Media version 7,0 ou ultérieure<br/> |
