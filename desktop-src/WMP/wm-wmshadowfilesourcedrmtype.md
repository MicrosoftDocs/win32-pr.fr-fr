# <a name="wmwmshadowfilesourcedrmtype-windows-media-player-sdk"></a>WM/WMShadowFileSourceDRMType (kit de développement logiciel (SDK) Lecteur Windows Media)

**WM/WMShadowFileSourceDRMType** est le système de gestion des droits numériques utilisé pour protéger le fichier contenu dans le fichier caché.

## <a name="remarks"></a>Remarques

Un fichier caché peut être un wrapper pour un fichier source. Cet attribut est une chaîne qui identifie le type DRM pour le fichier source.

Le fichier Shadow est spécifié à l’aide de l’attribut [ShadowFilePath](shadowfilepath-attribute.md) .
