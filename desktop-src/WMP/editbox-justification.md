# <a name="editboxjustification"></a>EDITBOX. Justification

L’attribut **Justification** spécifie ou récupère l’alignement du texte dans le contrôle EditBox.

``` syntax
        elementID.justification
```

## <a name="possible-values"></a>Valeurs possibles

Cet attribut est une **chaîne** en lecture/écriture contenant l’une des valeurs suivantes.



| Valeur  | Description                                                        |
|--------|--------------------------------------------------------------------|
| Gauche   | Par défaut. Aligne le texte sur le bord gauche du contrôle zone d’édition. |
| Right  | Aligne le texte sur le bord droit du contrôle zone d’édition.         |
| Center | Aligne le texte sur le centre horizontal du contrôle zone d’édition.  |



 

## <a name="remarks"></a>Notes

Il y a une marge de deux pixels.

## <a name="requirements"></a>Configuration requise



| Condition requise | Valeur |
|--------------------|---------------------------------------------------------|
| Version<br/> | Lecteur Windows Media pour Windows XP ou version ultérieure<br/> |
