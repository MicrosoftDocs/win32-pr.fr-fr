# <a name="button-function"></a>Button, fonction

Le tableau suivant indique les valeurs qui sont valides pour la fonction Button. Vous n’avez pas besoin de définir des valeurs pour les fonctions que vous n’utiliserez pas dans votre apparence.



| Valeur           | Description                                                                                                                                                  |
|-----------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Lire            | Lit la sélection actuelle dans la sélection.                                                                                                                 |
| Suspendre           | Suspend la lecture de l’élément multimédia en cours.                                                                                                                   |
| Arrêter            | Arrête la lecture de l’élément multimédia actuel.                                                                                                                    |
| Précédent            | Accède à l’élément précédent dans la sélection.                                                                                                                  |
| Suivant            | Passe à l’élément suivant dans la sélection.                                                                                                                      |
| Shuffle         | Mélange la playlist de façon aléatoire.                                                                                                                              |
| RepeatPlaylist  | Répète la playlist indéfiniment.                                                                                                                           |
| Désactiver le son            | Interrompt la sortie audio.                                                                                                                                   |
| PlayPause       | Bascule entre l’état de lecture et l’état de pause.                                                                                                          |
| Informations            | Fournit des informations sur l’élément multimédia qui est en train d’être lu. pour Lecteur Windows Media 10 apparences Mobile ou ultérieures, cette fonction n’est pas utilisée.                     |
| FullScreen      | Redimensionne la vidéo à la taille maximale de l’affichage de l’appareil.                                                                                                 |
| WindowsMedia\*  | Ouvre Pocket Internet Explorer et charge le site Web Windowsmedia.com.                                                                                       |
| PlayPauseStop\* | Outre le basculement entre l’état de lecture et l’état de pause, il permet également aux utilisateurs d’arrêter une diffusion en direct lorsque la fonctionnalité de pause n’est pas disponible. |
| VolumeUp\*      | Augmente la sortie audio.                                                                                                                                  |
| VolumeDown\*    | Diminue la sortie audio.                                                                                                                                  |



 

\*disponible pour Lecteur Windows Media 10 Mobile ou version ultérieure.
