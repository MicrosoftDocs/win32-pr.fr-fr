# <a name="status-alignment"></a>Alignement de l’État

Vous devez définir l’alignement de l’affichage d’État que vous souhaitez utiliser. Le tableau suivant indique les valeurs qui sont valides.



| Valeur  | Description                                                                       |
|--------|-----------------------------------------------------------------------------------|
| Gauche   | Le texte est aligné à gauche (aligné avec la marge de gauche de l’affichage de l’État).   |
| Center | Le texte est centré dans l’affichage de l’État.                                       |
| Right  | Le texte est aligné à droite (aligné avec la marge de droite de l’affichage de l’État). |
