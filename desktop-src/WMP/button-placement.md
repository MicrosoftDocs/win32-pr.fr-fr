# <a name="button-placement"></a>Positionnement du bouton

Même si vous pouvez dessiner des boutons pour n’importe quelle forme, ils sont traités en interne comme ayant une limite externe rectangulaire. Pour cette raison, évitez de chevaucher les coordonnées des boutons, car des résultats inattendus peuvent se produire.
