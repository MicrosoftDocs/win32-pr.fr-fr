# <a name="viewfocusobjectid"></a>VUE. focusObjectID

L’attribut **focusObjectID** spécifie ou récupère une valeur indiquant l’élément qui a le focus clavier. Ne peut être défini qu’au moment de l’exécution.

``` syntax
        elementID.focusObjectID
```

## <a name="possible-values"></a>Valeurs possibles

Cet attribut est une **chaîne** en lecture/écriture spécifiant l' **ID** de l’élément qui a le focus. Il n'a aucune valeur par défaut.

## <a name="requirements"></a>Configuration requise



| Condition requise | Valeur |
|--------------------|------------------------------------------------------|
| Version<br/> | Lecteur Windows Media version 7,0 ou ultérieure<br/> |
