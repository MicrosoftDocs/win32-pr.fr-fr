# <a name="ambientattributestop"></a>AmbientAttributes.top

L’attribut **Top** spécifie ou récupère la coordonnée supérieure du contrôle.

``` syntax
        elementID.top
```

## <a name="possible-values"></a>Valeurs possibles

Cet attribut est un **nombre** en lecture/écriture (**long**) représentant la distance en pixels entre le contrôle et le bord supérieur de la **vue** ou de la sous- **vue** parente. Sa valeur par défaut est zéro. Les nombres négatifs sont autorisés, auquel cas la bordure supérieure de la **vue** ou de la sous- **vue** découpe le contrôle.

## <a name="requirements"></a>Configuration requise

|--------------------|------------------------------------------------------|
| Version<br/> | Lecteur Windows Media version 7,0 ou ultérieure<br/> |
