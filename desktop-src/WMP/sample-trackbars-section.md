# <a name="sample-trackbars-section"></a>Exemple de section Trackbars

Les lignes suivantes affichent une section Trackbars typique d’un fichier de définition d’apparence :


```C++
[ Trackbars ]

//  <Type>      <Location>     <Dis Image Src>  <Thumb Image Src>  <Thumb Size>
//  ----------  ----------     ---------------  -----------------  ------------
    Seek      32,220,172,23  Disabled @ 32,27    SeekThumb.bmp    23,22

```



L’exemple précédent définit un TrackBar qui permet à l’utilisateur de rechercher un élément multimédia en appuyant sur l’image SeekThumb et en la faisant glisser.
