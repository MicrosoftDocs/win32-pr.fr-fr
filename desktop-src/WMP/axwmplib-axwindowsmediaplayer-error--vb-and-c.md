# <a name="axwindowsmediaplayererror-property"></a>AxWindowsMediaPlayer. Error, propriété

La propriété Error obtient l’interface IWMPError qui fournit l’accès à une collection d’interfaces IWMPErrorItem.

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```CSharp
public IWMPError Error {get;}
```


```VB

Public ReadOnly Property Error As IWMPError
```





## <a name="property-value"></a>Valeur de la propriété

Interface WMPLib. IWMPError.
