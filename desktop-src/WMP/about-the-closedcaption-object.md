# <a name="about-the-closedcaption-object"></a>À propos de l’objet ClosedCaption

l’objet **ClosedCaption** régit l’interface de sous-titrage pour Lecteur Windows Media. L’objet **ClosedCaption** est obtenu via la propriété **ClosedCaption** de l’objet **Player** . Par exemple, pour obtenir la valeur de la propriété **SAMIFileName** , tapez la commande suivante :


```C++
myfile = player.closedcaption.SAMIFileName;

```
