# <a name="playlistname"></a>Playlist.name

La `name` propriété spécifie ou récupère le nom de la sélection.

## <a name="syntax"></a>Syntaxe

*lecteur*. *currentPlaylist*. **nom**

## <a name="possible-values"></a>Valeurs possibles

Cette propriété est une **chaîne** en lecture/écriture.

## <a name="remarks"></a>Notes

Pour récupérer la valeur de cette propriété, l’accès en lecture à la bibliothèque est requis. Pour spécifier la valeur, un accès complet est nécessaire. Pour plus d’informations, consultez [accès à la bibliothèque](library-access.md).

**Lecteur Windows Media 10 Mobile :** Cette propriété est en lecture seule.
