# <a name="iwmpcdromburnburnstate-property"></a>IWMPCdromBurn :: burnState, propriété

La propriété **burnState** obtient une valeur d’énumération qui indique l’état d’avancement actuel.

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```CSharp
public WMPBurnState burnState {get;}
```


```VB

Public ReadOnly Property burnState As WMPBurnState
```





## <a name="property-value"></a>Valeur de la propriété

**Wmplib. WMPBurnState** qui est une valeur de l’énumération **WMPBurnState** qui indique l’état actuel.
