# <a name="effectsprevious"></a>EFFECTs. Previous

La méthode **précédente** affiche la présélection de visualisation précédente, en se déplaçant vers la dernière présélection de la visualisation précédente, si nécessaire.

``` syntax
        elementID.previous()
```

## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.

## <a name="return-value"></a>Valeur renvoyée

Cette méthode ne retourne pas de valeur.

## <a name="remarks"></a>Notes

Si la présélection actuelle est la première de la série de toutes les visualisations disponibles, la dernière présélection de la dernière visualisation est rendue active.

## <a name="requirements"></a>Configuration requise



| Condition requise | Valeur |
|--------------------|------------------------------------------------------|
| Version<br/> | Lecteur Windows Media version 7,0 ou ultérieure<br/> |
