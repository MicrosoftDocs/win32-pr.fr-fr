# <a name="ffwdbutton"></a>FFWDBUTTON

Il s’agit d’un **bouton** prédéfini avec les valeurs par défaut suivantes.


```C++
onclick="jscript:player.controls.fastForward()"
upToolTip="Fast Forward"
cursor="system"
enabled="wmpenabled:player.controls.fastForward"

```



## <a name="remarks"></a>Notes

Toutes les propriétés de ce **bouton** peuvent être remplacées en les spécifiant explicitement.
