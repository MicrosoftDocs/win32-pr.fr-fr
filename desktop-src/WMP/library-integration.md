# <a name="library-integration"></a><span data-ttu-id="eddd4-112">Intégration de la bibliothèque</span><span class="sxs-lookup"><span data-stu-id="eddd4-112">Library Integration</span></span>

<span data-ttu-id="eddd4-113">L’interface utilisateur du lecteur Windows Media est organisée en zones de fonctionnalités, appelées *volets de tâches*, qui encapsulent les diverses fonctionnalités de haut niveau du programme.</span><span class="sxs-lookup"><span data-stu-id="eddd4-113">The Windows Media Player user interface is organized into feature areas, called *task panes*, which encapsulate the various high-level features of the program.</span></span> <span data-ttu-id="eddd4-114">Celles-ci incluent les volets des tâches **bibliothèque**, **synchroniser** et **graver** (entre autres).</span><span class="sxs-lookup"><span data-stu-id="eddd4-114">These include the **Library**, **Sync**, and **Burn** task panes (among others).</span></span> <span data-ttu-id="eddd4-115">Le volet des tâches de la **bibliothèque** permet aux utilisateurs de travailler avec la bibliothèque. le volet **synchroniser** les tâches permet aux utilisateurs de synchroniser des fichiers multimédias numériques sur un appareil mobile. le volet de tâches **grave** permet aux utilisateurs de graver des fichiers multimédias numériques sur un CD ou un DVD.</span><span class="sxs-lookup"><span data-stu-id="eddd4-115">The **Library** task pane enables users to work with the library; the **Sync** task pane enables users to synchronize digital media files to a portable device; and the **Burn** task pane enables users to burn digital media files to a CD or DVD.</span></span>

> [!Note]  
> <span data-ttu-id="eddd4-116">Le volet des tâches de la **bibliothèque** est parfois appelé volet des tâches **Parcourir** .</span><span class="sxs-lookup"><span data-stu-id="eddd4-116">The **Library** task pane is sometimes called the **Browse** task pane.</span></span>

 

<span data-ttu-id="eddd4-117">Chacun de ces volets de tâches a un certain niveau d’intégration avec la bibliothèque.</span><span class="sxs-lookup"><span data-stu-id="eddd4-117">Each of these task panes has some level of integration with the library.</span></span> <span data-ttu-id="eddd4-118">Par exemple, si l’utilisateur souhaite graver de la musique sur un CD, il est judicieux de laisser l’utilisateur choisir la musique à graver en parcourant la bibliothèque et en faisant simplement glisser-déplacer des éléments multimédias dans une liste.</span><span class="sxs-lookup"><span data-stu-id="eddd4-118">For example, if the user wants to burn music to a CD, it makes sense to let the user choose the music to burn by browsing the library and simply dragging and dropping media items into a list.</span></span> <span data-ttu-id="eddd4-119">Cela signifie que les utilisateurs peuvent afficher et utiliser un catalogue de magasin en ligne entièrement intégré à la bibliothèque lors de l’utilisation des volets des tâches **bibliothèque**, **synchroniser** et **graver** .</span><span class="sxs-lookup"><span data-stu-id="eddd4-119">This means that users can view and use an online store catalog that is fully integrated into the library when working in the **Library**, **Sync**, and **Burn** task panes.</span></span> <span data-ttu-id="eddd4-120">L’énumération [WMPTaskType](/previous-versions/windows/desktop/api/contentpartner/ne-contentpartner-wmptasktype) contient des valeurs qui représentent ces trois volets de tâches afin qu’ils puissent être identifiés par programmation.</span><span class="sxs-lookup"><span data-stu-id="eddd4-120">The [WMPTaskType](/previous-versions/windows/desktop/api/contentpartner/ne-contentpartner-wmptasktype) enumeration contains values that represent these three task panes so they can be identified programmatically.</span></span>

<span data-ttu-id="eddd4-121">Chacun de ces trois volets de tâches est organisé en trois parties principales.</span><span class="sxs-lookup"><span data-stu-id="eddd4-121">Each of these three task with a hierarchical view of the Windows Media Player library, including categorization features by song, artist, album, and so forth.</span></span> <span data-ttu-id="eddd4-124">La deuxième partie du volet des tâches est le volet d’informations.</span><span class="sxs-lookup"><span data-stu-id="eddd4-124">The second task pane part is the details pane.</span></span> <span data-ttu-id="eddd4-125">Ce data-stu-id="eddd4-125">This pane provides detailed information organized according to the category currently selected in the library tree-view control.</span></span> <span data-ttu-id="eddd4-126">Par *panier*.</span><span class="sxs-lookup"><span data-stu-id="eddd4-127">The third part is the list pane, or *basket*.</span></span> <span data-ttu-id="eddd4-128">Les utilisateurs peuvent <span data-ttu-id="eddd4-129">Lorsqu’un catalogue de magasin en ligne est intégré à la bibliothèque, le magasin en ligne s’affiche sous la forme d’une catégorie de niveau supérieur, ou *nœud*, dans le contrôle d’arborescence de la bibliothèque.</span><span class="sxs-lookup"><span data-stu-id="eddd4-129">When an online store catalog is integrated with the library, the online store appears as a top-level category, or *node*, in the library tree-view control.</span></span> <span data-ttu-id="eddd4-130">(Un seul catalogue de magasin en ligne est visible par l’utilisateur à la fois.) <span data-ttu-id="eddd4-132">Le nœud de magasin en ligne de niveau supérieur possède un ensemble de nœuds enfants fournis par le lecteur Windows Media.</span><span class="sxs-lookup"><span data-stu-id="eddd4-132">The top-level online store node has a set of child nodes provided by Windows Media Player.</span></span> <span data-ttu-id="eddd4-133">Par exemple, le nœud de magasin en ligne de niveau supérieur a des nœuds de radio, d’artiste et d’album enfants, entre autres.</span><span class="sxs-lookup"><span data-stu-id="eddd4-133">For example, the top-level online store node has Radio, Artist, and Album child nodes, among others.</span></span> <span data-ttu-id="eddd4-134">Le nœud de magasin en ligne de niveau supérieur peut également avoir jusqu’à huit nœuds enfants personnalisés fournis par le magasin en ligne.</span><span class="sxs-lookup"><span data-stu-id="eddd4-134">The top-level online store node can also have up to eight custom child nodes provided by the online store.</span></span> <span data-ttu-id="eddd4-135">Le lecteur Windows Media crée un nœud enfant personnalisé pour toutes les listes ayant un identificateur de liste compris entre 0 et 7.</span><span class="sxs-lookup"><span data-stu-id="eddd4-135">Windows Media Player creates a custom child node for any list that has a list identifier in the range 0 through 7.</span></span> <span data-ttu-id="eddd4-136">Le magasin en ligne spécifie l’identificateur d’une liste dans le fichier [list.csv](list-csv.md) qui fait partie du catalogue du magasin.</span><span class="sxs-lookup"><span data-stu-id="eddd4-136">The online store specifies the identifier of a list in the [list.csv](list-csv.md) file that is part of the store's catalog.</span></span>

<span data-ttu-id="eddd4-137">Le lecteur Windows Media récupère une icône pour chacun des nœuds de l’arborescence personnalisée de la Banque en ligne en appelant [IWMPContentPartner :: GetItemInfo](/previous-versions/windows/desktop/api/contentpartner/nf-contentpartner-iwmpcontentpartner-getiteminfo), en passant CPListIDIcon dans le paramètre *bstrInfoName* .</span><span class="sxs-lookup"><span data-stu-id="eddd4-137">Windows Media Player retrieves an icon for each of the online store's custom tree nodes by calling [IWMPContentPartner::GetItemInfo](/previous-versions/windows/desktop/api/contentpartner/nf-contentpartner-iwmpcontentpartner-getiteminfo), passing CPListIDIcon in the *bstrInfoName* parameter.</span></span>

<span data-ttu-id="eddd4-138">Lorsque l’utilisateur navigue dans le catalogue, le lecteur Windows Media effectue des appels à **IWMPContentPartner :: GetItemInfo** pour récupérer les métadonnées du plug-in de partenaire de contenu à propos des éléments de musique sélectionnés par l’utilisateur.</span><span class="sxs-lookup"><span data-stu-id="eddd4-138">As the user navigates through the catalog, Windows Media Player makes calls to **IWMPContentPartner::GetItemInfo** to retrieve metadata from the content partner plug-in about the music items the user selects.</span></span> <span data-ttu-id="eddd4-139">Ces métadonnées fournissent des informations au joueur afin que le lecteur puisse afficher des détails sur les éléments du catalogue.</span><span class="sxs-lookup"><span data-stu-id="eddd4-139">This metadata provides information to the Player so that the Player can display details about the catalog items.</span></span> <span data-ttu-id="eddd4-140">Par exemple, si l’utilisateur sélectionne un album, le lecteur Windows Media récupère l’URL de la pochette de l’album afin que l’utilisateur puisse voir l’image de la couverture de l’album.</span><span class="sxs-lookup"><span data-stu-id="eddd4-140">For example, if the user selects an album, Windows Media Player retrieves the album art URL so that the user can see the album cover art.</span></span>
