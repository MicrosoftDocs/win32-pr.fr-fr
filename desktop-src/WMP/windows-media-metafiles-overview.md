# <a name="windows-media-metafiles-overview"></a>Windows Présentation des fichiers multimédias

la partie la plus importante de l’utilisation réussie de Windows métafichiers multimédias est l’utilisation de la syntaxe correcte pour les éléments de métafichier. les erreurs de syntaxe dans un métafichier Windows Media peuvent entraîner la non-reconnaissance d’un attribut unique dans le métafichier qui n’est pas reconnu comme étant valide et qui ne fonctionne pas.

presque aussi important est l’ordre dans lequel les éléments apparaissent dans un métafichier de média Windows. Les attributs de certains éléments substituent temporairement les attributs des éléments similaires dans différentes sections du métafichier. Il existe un [ordre de priorité](order-of-precedence.md)défini.

Windows les sélections de métafichiers de média sont Windows des métafichiers multimédias qui fournissent des informations que Lecteur Windows Media utilise pour recevoir des flux de monodiffusion, des flux de multidiffusion et d’autres supports pris en charge à partir d’un intranet ou d’Internet. Une sélection de métafichiers est fondamentalement un raccourci vers du contenu multimédia. Une sélection de métafichier peut être envoyée sous forme de courrier électronique, utilisée en tant que référence de lien sur une page Web, créée dynamiquement à l’aide de pages Active Server (ASP) ou en tant que fichier autonome sur un lecteur de disque local. une sélection de métafichiers peut faire référence à une autre sélection de métafichiers, à une page ASP ou à un fichier de station de média Windows (avec une extension de nom de fichier. nsc). un fichier. nsc est utilisé pour définir une station de média Windows à Lecteur Windows Media. Le processus de gestion de base est le même pour chaque cas.
