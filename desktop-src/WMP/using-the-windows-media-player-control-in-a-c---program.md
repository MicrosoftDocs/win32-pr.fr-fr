# <a name="using-the-windows-media-player-control-in-a-c-program"></a>utilisation du contrôle Lecteur Windows Media dans un programme C++

> [!Note]  
> l’utilisation de C++ pour incorporer le contrôle de Lecteur Windows Media est prise en charge pour Lecteur Windows Media série 9 ou ultérieure.

 

il existe différentes façons d’utiliser le contrôle Lecteur Windows Media dans un programme C++. vous pouvez créer une instance du contrôle dans une application console, ou incorporer le contrôle dans une application Windows. En outre, vous pouvez implémenter des interfaces qui vous permettent d’exécuter un contrôle de lecteur incorporé en mode distant. Vous pouvez personnaliser l’interface utilisateur d’un contrôle incorporé en appliquant un fichier de définition d’apparence.

Ces informations sont décrites dans les rubriques suivantes.



| Rubrique                                                                                                                                      | Description                                                                                                                                                                 |
|--------------------------------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [utilisation du contrôle Lecteur Windows Media dans une Application Console](using-the-windows-media-player-control-in-a-console-application.md)     | décrit une application console C++ simple qui instancie le contrôle Lecteur Windows Media pour afficher la version.                                                       |
| [hébergement du contrôle Lecteur Windows Media dans une Application Windows](hosting-the-windows-media-player-control-in-a-windows-application.md) | décrit comment utiliser la fenêtre hôte ActiveX ATL pour incorporer le contrôle Lecteur Windows Media dans un programme Windows.                                                            |
| [Utilisation à distance du contrôle Windows Media Player](remoting-the-windows-media-player-control.md)                                                 | décrit comment incorporer le contrôle Lecteur Windows Media dans un programme C++ en mode distant, ce qui permet à vos utilisateurs de détacher le contrôle pour basculer vers le mode complet du lecteur. |
| [Gestion des événements en C++](handling-events-in-c.md)                                                                                         | décrit comment recevoir des notifications d’événements de Lecteur Windows Media.                                                                                                     |
| [utilisation d’habillages avec le contrôle Lecteur Windows Media](using-skins-with-the-windows-media-player-control.md)                                 | décrit comment appliquer un fichier d’apparence à un contrôle Lecteur Windows Media incorporé dans un programme C++.                                                                             |



 

> [!Note]  
> vous pouvez incorporer le contrôle Mobile Lecteur Windows Media 10 dans une application Windows CE. les techniques que vous utilisez pour ce faire sont similaires à celles utilisées avec le contrôle de Lecteur Windows Media de bureau. toutefois, il existe des différences entre atl pour Windows et atl pour Windows CE. Cette documentation décrit les différences entre ces implémentations, le cas échéant.
