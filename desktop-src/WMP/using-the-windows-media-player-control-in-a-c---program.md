# <a name="using-the-windows-media-player-control-in-a-c-program"></a>Utilisation du contrôle du lecteur Windows Media dans un programme C++

> [!Note]  
> L’utilisation de C++ pour incorporer le contrôle du lecteur Windows Media est prise en charge pour le lecteur Windows Media série 9 ou version ultérieure.

 

Il existe différentes façons d’utiliser le contrôle du lecteur Windows Media dans un programme C++. Vous pouvez créer une instance du contrôle dans une application console, ou incorporer le contrôle dans une application Windows. En outre, vous pouvez implémenter des interfaces qui vous permettent d’exécuter un contrôle de lecteur incorporé en mode distant. Vous pouvez personnaliser l’interface utilisateur d’un contrôle incorporé en appliquant un fichier de définition d’apparence.

Ces informations sont décrites dans les rubriques suivantes.



| Rubrique                                                                                                                                      | Description                                                                                                                                                                 |
|--------------------------------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Utilisation du contrôle du lecteur Windows Media dans une application console](using-the-windows-media-player-control-in-a-console-application.md)     | Décrit une application console C++ simple qui instancie le contrôle du lecteur Windows Media pour afficher la version.                                                       |
| [Hébergement du contrôle du lecteur Windows Media dans une application Windows](hosting-the-windows-media-player-control-in-a-windows-application.md) | Décrit comment utiliser la fenêtre hôte ActiveX ATL pour incorporer le contrôle du lecteur Windows Media dans un programme Windows.                                                            |
| [Utilisation à distance du contrôle Windows Media Player](remoting-the-windows-media-player-control.md)                                                 | Décrit comment incorporer le contrôle du lecteur Windows Media dans un programme C++ en mode distant, ce qui permet à vos utilisateurs de détacher le contrôle pour basculer vers le mode complet du lecteur. |
| [Gestion des événements en C++](handling-events-in-c.md)                                                                                         | Décrit comment recevoir des notifications d’événements à partir du lecteur Windows Media.                                                                                                     |
| [Utilisation d’habillages avec le contrôle du lecteur Windows Media](using-skins-with-the-windows-media-player-control.md)                                 | Décrit comment appliquer un fichier d’apparence à un contrôle de lecteur Windows Media incorporé dans un programme C++.                                                                             |



 

> [!Note]  
> Vous pouvez incorporer le contrôle mobile du lecteur Windows Media 10 dans une application Windows CE. Les techniques que vous utilisez pour ce faire sont similaires à celles utilisées avec le contrôle du lecteur Windows Media du bureau. Toutefois, il existe des différences entre ATL pour Windows et ATL pour Windows CE. Cette documentation décrit les différences entre ces implémentations, le cas échéant.
