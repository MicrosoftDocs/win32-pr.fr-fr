# <a name="axwindowsmediaplayerisonline-property"></a>AxWindowsMediaPlayer. isOnline, propriété

La propriété isOnline obtient une valeur indiquant si l’utilisateur est connecté à un réseau.

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```CSharp
public System.Boolean isOnline {get;}
```


```VB

Public ReadOnly Property isOnline As System.Boolean
```





## <a name="property-value"></a>Valeur de la propriété

Valeur System. Boolean qui indique si l’utilisateur est connecté à un réseau.
