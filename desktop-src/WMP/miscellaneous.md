# <a name="miscellaneous"></a>Divers

Cette section décrit les attributs spécialisés et les autres rubriques à utiliser pour créer des apparences.

Les rubriques suivantes sont traitées.



| Rubrique                                            | Description                                                                                                                        |
|--------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------|
| [Référence de couleur](color-reference.md)           | Un graphique de couleurs, des noms de couleurs et des valeurs hexadécimales pour les couleurs prises en charge par les apparences.                                              |
| [Attributs globaux](global-attributes.md)       | Les attributs globaux sont des attributs qui permettent d’accéder facilement à certains éléments ou objets de joueur depuis n’importe quel emplacement d’une apparence.       |
| [Écoute des attributs](listening-attributes.md) | Attributs utilisés pour connecter un attribut à un autre afin que sa valeur change chaque fois que la valeur de l’autre attribut change. |
