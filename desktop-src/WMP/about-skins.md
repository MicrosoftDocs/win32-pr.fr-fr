# <a name="about-skins"></a>À propos des apparences

avec Lecteur Windows Media, les utilisateurs peuvent choisir parmi un large éventail d’habillages standard, chacun offrant une expérience visuelle supplémentaire qui améliore l’écoute et l’affichage du plaisir. Lecteur Windows Media est fourni avec plusieurs apparences, mais il est relativement facile de créer et de distribuer des apparences personnalisées.

les apparences sont simplement des regroupements d’un ou plusieurs fichiers artistiques d’ordinateur, organisés par un fichier texte, qui indique Lecteur Windows Media comment utiliser ces fichiers pour afficher une apparence. Vous pouvez utiliser un grand nombre de programmes pour modifier l’art existant ou créer votre propre image. par exemple, à l’aide des outils de dessin et de l’image clipart fournis avec Microsoft PowerPoint, vous pouvez créer des illustrations simplement en les faisant glisser et en les déposant. De même, vous n’avez pas besoin d’être un programmeur pour créer le fichier texte utilisé pour définir une apparence individuelle.

L’une des raisons pour lesquelles les apparences sont relativement faciles à créer est que vous pouvez examiner les éléments d’art individuels et le code source pour toute apparence et voir comment chaque apparence est mise en ensemble. Plusieurs exemples d’apparences simples sont fournis avec ce kit de développement logiciel (SDK), ainsi qu’une référence pour les éléments d’apparence et les attributs.
