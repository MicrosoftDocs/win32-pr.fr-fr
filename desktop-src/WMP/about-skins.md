<span data-ttu-id="da39f-107">Avec le lecteur Windows Media, les utilisateurs peuvent choisir parmi un large éventail d’habillages standard, chacun offrant une expérience visuelle supplémentaire qui améliore l’écoute et la consultation.</span><span class="sxs-lookup"><span data-stu-id="da39f-107">With Windows Media Player, users can choose from a variety of standard skins, each one providing an additional visual experience that enhances listening and viewing pleasure.</span></span> <span data-ttu-id="da39f-108">Le lecteur Windows Media est fourni avec plusieurs apparences, mais il est relativement facile de créer et de distribuer des apparences personnalisées.</span><span class="sxs-lookup"><span data-stu-id="da39f-108">Windows Media Player comes with several skins to choose from, but it is relatively easy to create and distribute custom skins.</span></span>
