# <a name="text-color"></a>Couleur du texte

La valeur de couleur du type doit être spécifiée. La valeur doit être spécifiée sous la forme de trois entiers positifs séparés par des virgules. Les valeurs représentent la quantité de rouge, de vert et de bleu pour une image bitmap, et est comprise entre 0 et 255.
