# <a name="other-skin-samples"></a>Autres exemples d’apparence

Pour obtenir des exemples d’apparences de travail complètes, cliquez sur le bouton **Démarrer** , pointez sur **programmes**, sur **Windows Media**, puis sur **Windows Media SDK**, puis cliquez sur **exemples du SDK du lecteur**. Vous devez d’abord installer le kit de développement logiciel (SDK) du lecteur Windows Media pour utiliser les exemples.
