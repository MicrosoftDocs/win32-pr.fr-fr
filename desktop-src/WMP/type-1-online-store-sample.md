# <a name="type-1-online-store-sample"></a>Exemple de magasin en ligne de type 1

Le kit de développement logiciel (SDK) Windows Media Player comprend un exemple de magasin en ligne de type 1 pris en charge par le lecteur Windows Media 11.

Pour afficher l’exemple de magasin en ligne de type 1, configurez votre registre comme décrit dans [clés de Registre et entrées pour un magasin de type 1 en ligne](registry-keys-and-entries-for-a-type-1-online-store.md). La clé de l’exemple de magasin de type 1 est 2573.
