# <a name="iwmperroritemcustomurl-property"></a>IWMPErrorItem :: customUrl, propriété

La propriété **customUrl** obtient l’URL d’un site Web qui affiche des informations spécifiques sur l’échec du téléchargement du codec.

## <a name="syntax"></a>Syntaxe


```CSharp
public System.String customUrl {get; set;}
```


```VB

Public ReadOnly Property customUrl As System.String
```





## <a name="property-value"></a>Valeur de la propriété

**System. String** qui est l’URL personnalisée.
