# <a name="playlistisidentical-method"></a>Méthode playlist. isIdentical

La méthode **isIdentical** récupère une valeur indiquant si l’objet de **sélection** fourni est identique à l’objet actuel.

## <a name="syntax"></a>Syntaxe


```JScript
bRetVal = Playlist.isIdentical(
  playlist
)
```



## <a name="parameters"></a>Paramètres

<dl> <dt>

*sélection* \[ dans\]
</dt> <dd>

Objet de **sélection** à comparer à l’objet actuel.

</dd> </dl>

## <a name="return-value"></a>Valeur retournée

Cette méthode retourne une **valeur booléenne**.
