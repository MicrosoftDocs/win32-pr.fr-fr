# <a name="about-the-metadatapicture-object"></a>À propos de l’objet MetadataPicture

L’attribut de métadonnées **WM/Picture** fournit des informations détaillées sur les images de métadonnées, telles que la pochette de l’album. L’objet **MetadataPicture** fournit un moyen de récupérer les données individuelles relatives à l’image. Vous pouvez considérer l’attribut de métadonnées **WM/Picture** comme un attribut composé. Quand vous utilisez un *média*. **getItemInfoByType** pour récupérer des informations sur une image de métadonnées, vous récupérez un objet **MetadataPicture** qui a des propriétés qui contiennent les données individuelles.
