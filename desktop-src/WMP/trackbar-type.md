# <a name="trackbar-type"></a>TrackBar, type

Vous devez définir le type de TrackBar que vous souhaitez utiliser. Le tableau suivant répertorie les types qui sont valides.



| Type   | Description                                                                                                                                                                                                                                                                                                     |
|--------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Seek   | Contrôle la position de lecture. Si la valeur est égale à zéro, l’élément multimédia est au début. La valeur augmente jusqu’à ce que la fin de l’élément multimédia soit atteinte. L’utilisateur peut faire glisser l’image Thumb pour modifier la valeur, et Pendant que l’élément multimédia est lu, l’image Thumb se déplace pour correspondre à la position de lecture actuelle. |
| Volume | Contrôle le volume audio. La valeur varie de 0 à 100. L’utilisateur peut faire glisser l’image Thumb pour changer le volume. Si le volume change, l’image Thumb sera déplacée pour correspondre au nouveau volume.                                                                                                                   |
