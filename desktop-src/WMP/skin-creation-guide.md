# <a name="skin-creation-guide"></a>Guide de création d’apparence

Ce guide est une série d’explications détaillées sur la façon de créer différents types d’apparences. Pour plus d’informations générales sur les apparences, consultez [à propos des apparences](about-skins.md). Pour plus d’informations sur chaque attribut, méthode et événement utilisé dans les apparences, consultez la référence sur la programmation de l' [apparence](skin-programming-reference.md). À mesure que vous êtes plus impliqué dans la programmation de votre apparence, vous pouvez lire la partie de ce kit de développement logiciel (SDK) couvrant le [modèle objet du lecteur Windows Media](windows-media-player-object-model.md).

Dans ce guide, vous trouverez des instructions pour créer l’art pour Adobe Photoshop 5,5, un programme de manipulation des dessins populaires. Les instructions spécifiques peuvent être différentes si vous disposez d’un programme artistique similaire, tel que Jasc Paint Shop Pro ou la viscosité de Sonic found, mais les concepts sont les mêmes. Photoshop a été choisi pour deux raisons : il s’agit d’un programme art populaire pour les artistes commerciaux et il fonctionne avec des couches. Comme vous le verrez dans les didacticiels, les couches sont très utiles pour la création d’apparences.

Ce guide aborde les tâches suivantes.
