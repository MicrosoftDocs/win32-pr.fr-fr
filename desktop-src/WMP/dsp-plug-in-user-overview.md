# <a name="dsp-plug-in-user-overview"></a>Vue d’ensemble de l’utilisateur du plug-in DSP

Pour l’utilisateur, les plug-ins DSP sont des fonctionnalités complémentaires qui modifient d’une manière ou d’une autre la façon dont l’utilisateur rencontre l’audio ou la vidéo que le lecteur Windows Media lit. Le résultat d’un plug-in DSP peut être évident, à l’instar des couleurs de l’image vidéo qui est inversée, de sorte qu’il ressemble à une photo négative, ou à un aspect très subtil, comme un effet de réverbération de la lumière dans l’audio, afin d’ajouter une sensation de la plus grandeté au son.

Les utilisateurs peuvent installer et désinstaller les plug-ins DSP à l’aide de l’onglet **plug-ins** de la boîte de dialogue Options dans le lecteur Windows Media. Les utilisateurs peuvent ensuite activer et désactiver les plug-ins installés à l’aide des cases à cocher fournies sous l’onglet **plug-ins** . les plug-ins DSP peuvent implémenter une page de propriétés, à laquelle les utilisateurs peuvent également accéder à partir de l’onglet **plug-ins** . Une page de propriétés fournit à l’utilisateur un moyen de modifier les paramètres qui affectent la façon dont un plug-in DSP fonctionne. Par exemple, une page de propriétés peut inclure un contrôle Slider qui permet à l’utilisateur de modifier le temps de décroissance d’un effet de réverbération.
