# <a name="text-alignment"></a>Alignement du texte

Vous devez définir l’alignement de la zone d’affichage de texte que vous souhaitez utiliser. Les valeurs indiquées dans le tableau suivant sont valides.



| Valeur  | Description                |
|--------|----------------------------|
| Gauche   | Le texte est aligné à gauche.  |
| Center | Le texte est centré.      |
| Right  | Le texte est aligné à droite. |
