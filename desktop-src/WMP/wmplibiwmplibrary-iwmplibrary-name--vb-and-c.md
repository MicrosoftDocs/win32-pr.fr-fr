# <a name="iwmplibraryname-property"></a>IWMPLibrary :: Name, propriété

La propriété **Name** obtient le nom complet de la bibliothèque actuelle.

## <a name="syntax"></a>Syntaxe


```CSharp
public System.String name {get; set;}
```


```VB

Public ReadOnly Property name As System.String
```





## <a name="property-value"></a>Valeur de la propriété

**System. String** qui est le nom de la bibliothèque actuelle.
