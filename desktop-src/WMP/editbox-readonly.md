# <a name="editboxreadonly"></a>EDITBOX. readOnly

L’attribut **ReadOnly** spécifie ou récupère une valeur indiquant si le texte du contrôle zone d’édition est en lecture seule ou s’il peut être modifié.

``` syntax
        elementID.readOnly
```

## <a name="possible-values"></a>Valeurs possibles

Cet attribut est une valeur **booléenne** en lecture/écriture dont la valeur par défaut est false.

## <a name="requirements"></a>Configuration requise



| Condition requise | Valeur |
|--------------------|---------------------------------------------------------|
| Version<br/> | Lecteur Windows Media pour Windows XP ou version ultérieure<br/> |
