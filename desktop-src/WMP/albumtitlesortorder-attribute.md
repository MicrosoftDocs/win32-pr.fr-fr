# <a name="albumtitlesortorder-attribute"></a>Attribut AlbumTitleSortOrder

L’attribut **AlbumTitleSortOrder** est la valeur à utiliser pour trier la colonne titre de l' **album** dans la bibliothèque.

## <a name="applies-to"></a>S'applique à

-   [Éléments audio](audio-item-attributes.md)

## <a name="remarks"></a>Notes

Cet attribut est stocké uniquement dans la bibliothèque.

Pour déterminer si vous pouvez modifier la valeur de cet attribut, utilisez la méthode [Media. isReadOnlyItem](media-isreadonlyitem.md) .

## <a name="requirements"></a>Spécifications



| Condition requise | Valeur |
|--------------------|-----------------------------------------------|
| Version<br/> | série Lecteur Windows Media 9 uniquement<br/> |
