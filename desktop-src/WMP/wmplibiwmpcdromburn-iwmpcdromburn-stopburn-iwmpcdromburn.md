# <a name="iwmpcdromburnstopburn-method"></a>IWMPCdromBurn :: stopBurn, méthode

La méthode **stopBurn** arrête le processus de gravure de CD.

```CSharp
public void stopBurn();
```


```VB

Public Sub stopBurn()
Implements IWMPCdromBurn.stopBurn
```
