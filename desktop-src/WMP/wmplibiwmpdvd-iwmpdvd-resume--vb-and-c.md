# <a name="iwmpdvdresume-method"></a>IWMPDVD :: Resume, méthode

La méthode **Resume** retourne en mode lecture à partir du mode menu, en reprenant à la même position que lorsque le menu a été appelé.

## <a name="syntax"></a>Syntaxe


```CSharp
public void resume();
```


```VB

Public Sub resume()
Implements IWMPDVD.resume
```





## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.

## <a name="return-value"></a>Valeur retournée

Cette méthode ne retourne pas de valeur.
