# <a name="windows-media-player-dsp-plug-ins"></a>Plug-ins du lecteur Windows Media DSP

Le lecteur Microsoft Windows Media fournit une architecture qui permet à l’utilisateur d’installer et d’activer des programmes de plug-in qui ajoutent la fonctionnalité de traitement des signaux numériques (DSP). Un plug-in DSP est un objet Microsoft DirectX Media Object (DMO) ou une transformation de Media Foundation (MFT) qui se connecte au lecteur à l’aide d’interfaces COM. Un plug-in DSP type peut être un égaliseur audio ou un contrôle de teinte de la vidéo. Cette section du kit de développement logiciel (SDK) fournit les informations de programmation dont vous avez besoin pour créer votre propre plug-in DSP.

La documentation du plug-in DSP est composée des trois sections suivantes.

|------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------|
| [À propos des plug-ins DSP](about-dsp-plug-ins.md)                                 | Fournit une vue d’ensemble de l’architecture utilisée pour les plug-ins DSP. Lisez cette section pour découvrir les concepts généraux associés à cette technologie.  |
| [Guide de programmation de plug-ins DSP](dsp-plug-ins-programming-guide.md)         | Explique ce que vous devez faire pour créer un plug-in DSP. Cette section contient des exemples de code et des procédures pas à pas.                           |
| [Référence de programmation de plug-ins DSP](dsp-plug-ins-programming-reference.md) | Fournit une référence détaillée pour les interfaces COM, les méthodes et les types énumérés pris en charge par le kit de développement logiciel (SDK) du lecteur Windows Media pour les plug-ins DSP. |
