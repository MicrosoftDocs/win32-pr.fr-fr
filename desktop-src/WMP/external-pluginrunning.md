# <a name="externalpluginrunning"></a>External. pluginRunning

> [!Note]  
> Cette rubrique décrit les fonctionnalités conçues pour être utilisées par les magasins en ligne. L’utilisation de cette fonctionnalité en dehors du contexte d’un magasin en ligne n’est pas prise en charge.

 

La propriété **pluginRunning** récupère une valeur qui indique si le plug-in du magasin en ligne est en cours d’exécution.

## <a name="syntax"></a>Syntaxe

``` syntax
window.external.pluginRunning
```

## <a name="possible-values"></a>Valeurs possibles

Cette propriété est une **valeur booléenne** en lecture seule. La **valeur true** indique que le plug-in est en cours d’exécution.
