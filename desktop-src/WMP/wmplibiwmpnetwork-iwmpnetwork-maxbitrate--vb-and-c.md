# <a name="iwmpnetworkmaxbitrate-property"></a>IWMPNetwork :: maxBitRate, propriété

La propriété **maxBitRate** obtient la vitesse de transmission vidéo maximale possible.

## <a name="syntax"></a>Syntaxe


```CSharp
public System.Int32 maxBitRate {get; set;}
```


```VB

Public ReadOnly Property maxBitRate As System.Int32
```





## <a name="property-value"></a>Valeur de la propriété

**System. Int32** qui correspond à la vitesse de transmission maximale.
