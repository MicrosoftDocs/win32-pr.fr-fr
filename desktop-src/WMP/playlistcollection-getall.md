# <a name="playlistcollectiongetall-method"></a>PlaylistCollection. getAll, méthode

La méthode **GetAll** récupère un objet **PlaylistArray** contenant toutes les sélections de la bibliothèque.

## <a name="syntax"></a>Syntaxe


```JScript
retVal = PlaylistCollection.getAll()
```



## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.

## <a name="return-value"></a>Valeur de retour

Cette méthode retourne un objet **PlaylistArray** .

## <a name="remarks"></a>Notes

Pour utiliser cette méthode, l’accès en lecture à la bibliothèque est requis. Pour plus d’informations, consultez [accès à la bibliothèque](library-access.md).
