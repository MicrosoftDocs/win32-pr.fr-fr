# <a name="windows-media-download-packages-deprecated"></a>Windows Packages de téléchargement de médias (déconseillés)

cette page documente une fonctionnalité qui peut ne pas être disponible dans les futures versions de Lecteur Windows Media et le kit de développement logiciel (SDK) Lecteur Windows Media.

Windows les packages de téléchargement de média combinent les bordures Lecteur Windows Media, les informations de sélection et le contenu multimédia dans un fichier téléchargeable unique avec l’extension de nom de fichier. wmd. un package Windows Media Download peut inclure l’intégralité d’un album de vidéos musicales qui affiche également la publicité sous la forme d’une personnalisation graphique et de liens vers un site web de revendeurs musicaux en ligne.

les utilisateurs peuvent télécharger un package de téléchargement Windows Media à partir d’un site web simplement en cliquant sur un lien. une fois le package téléchargé sur l’ordinateur de l’utilisateur, Lecteur Windows Media extrait automatiquement les fichiers à l’intérieur du package, puis ajoute la sélection empaquetée à la zone de liste déroulante des sélections, ajoute le contenu à la bibliothèque, affiche l’apparence de la bordure dans le volet de **lecture** en cours du Lecteur Windows Media en mode complet, puis lit l’élément initial dans la sélection.

Windows Les packages de téléchargement de médias présentent les avantages suivants :

-   Un simple clic télécharge, extrait et catalogue plusieurs fichiers sur l’ordinateur de l’utilisateur.
-   Le contenu est lu immédiatement après son téléchargement.
-   Les bordures personnalisées peuvent afficher des informations sur la publicité et la personnalisation.
-   Les sélections empaquetées sont cataloguées dans la bibliothèque.
-   la redirection de site Web peut se produire à partir de Lecteur Windows Media vers un site associé.
-   Divers contrôles de bordure interactifs sont disponibles.
-   Plusieurs types de fichiers audio et vidéo sont pris en charge pour l’empaquetage.

Tous ces avantages peuvent être utilisés ensemble. par exemple, un seul Windows package de téléchargement de média peut permettre aux utilisateurs d’afficher les paroles à la suite de leur lecture, d’afficher une vidéo qui accompagne les chansons, d’afficher des informations de publicité sur le serveur de distribution des enregistrements, d’afficher une pochette d’album, de visiter un site web de ventilateur et de cataloguer le contenu de la bibliothèque.
