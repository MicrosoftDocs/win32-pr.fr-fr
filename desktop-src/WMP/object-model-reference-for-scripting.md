# <a name="object-model-reference-for-scripting"></a>Référence du modèle objet pour l’écriture de scripts

la référence du modèle objet pour les scripts contient des informations détaillées sur l’Lecteur Windows Media ActiveX modèle objet de contrôle. Les informations de cette section sont présentées dans un style conçu pour être utilisé avec des langages de script tels que Microsoft JScript.

> [!Note]  
> toutes les méthodes, propriétés et événements sont entièrement pris en charge dans Lecteur Windows Media 10 Mobile ou version ultérieure, sauf indication contraire.

 

La référence du modèle objet pour les scripts contient la documentation pour les objets suivants, ainsi que les méthodes, propriétés et événements associés.



| Object                                              | Description                                                                                                                                                                                    |
