# <a name="user-overview"></a>Vue d’ensemble de l’utilisateur

Pour l’utilisateur, les visualisations sont des images qui s’affichent dans le temps jusqu’à la musique. Ils fournissent des divertissements supplémentaires pour améliorer les expériences musicales. Plus une visualisation est divertissante, mieux c’est !

Une visualisation classique s’affiche comme indiqué dans l’illustration suivante.

![visualisation classique](images/vizanim.gif)

## <a name="standard-visualizations"></a>Visualisations standard

Lecteur Windows Media comprend plusieurs visualisations, telles que les suivantes :

-   Intelligent
-   Plenoptic
-   Supplément

## <a name="standard-presets"></a>Présélections standard

En outre, chaque visualisation est accompagnée de présélections spécifiques qui modifient le mode d’affichage de la visualisation. Par exemple, la visualisation du Pique-notes est fournie en deux présélections :

-   Supplément
-   Simplex

Les deux effets font partie de la même visualisation, mais ont des présélections différentes. Une présélection est un ensemble de valeurs pour les attributs d’une visualisation particulière.

## <a name="skin-attributes"></a>Attributs d’apparence

si vous créez des apparences pour Lecteur Windows Media, vous pouvez afficher les visualisations standard dans vos habillages à l’aide de l’élément **effects** . pour plus d’informations sur l’utilisation des visualisations dans les apparences, consultez [Lecteur Windows Media skins](windows-media-player-skins.md).
