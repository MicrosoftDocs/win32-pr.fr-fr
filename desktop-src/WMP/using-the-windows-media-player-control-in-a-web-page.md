# <a name="using-the-windows-media-player-control-in-a-web-page"></a>utilisation du contrôle Lecteur Windows Media dans une Page Web

l’incorporation du contrôle Lecteur Windows Media dans une page web vous permet de personnaliser complètement la manière dont l’utilisateur interagit avec le contrôle. Vous pouvez utiliser l’interface fournie par le contrôle, ou vous pouvez la masquer et afficher votre propre interface utilisateur. vous pouvez spécifier plusieurs propriétés de contrôle Lecteur Windows Media à l’endroit où vous incorporez le contrôle, ou vous pouvez définir des propriétés de joueur et appeler des méthodes de lecteur dans le code de script.

Les sections suivantes décrivent les principes fondamentaux de l’incorporation du contrôle dans une page Web.



| Section                                                                                                        | Description                                                                                                                       |
|----------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------|
| [masquage du contrôle Lecteur Windows Media](hiding-the-windows-media-player-control.md)                         | décrit les options permettant de masquer le contrôle Lecteur Windows Media lorsque vous souhaitez fournir votre propre interface utilisateur.               |
| [Éléments PARAM dans un élément OBJECT](param-elements-in-an-object-element.md)                                 | décrit comment initialiser le contrôle Lecteur Windows Media lorsque vous l’incorporez.                                                   |
| [Exemple simple de script dans une page Web](simple-example-of-scripting-in-a-web-page.md)                     | Décrit un exemple simple, mais complet, d’un contrôle de lecteur incorporé avec une interface utilisateur personnalisée.                             |
| [utilisation du contrôle Lecteur Windows Media avec Firefox](using-the-windows-media-player-control-with-firefox.md) | décrit comment incorporer le contrôle Lecteur Windows Media dans une page web afin qu’il s’affiche correctement par un navigateur Firefox. |
| [Utilisation de Windows Media Player avec Netscape 7.1](using-windows-media-player-with-netscape-7-1.md)               | décrit comment utiliser le contrôle Lecteur Windows Media avec Netscape 7,1 et d’autres navigateurs Web basés sur Gecko.                       |



 

> [!Note]  
> le contrôle Mobile Lecteur Windows Media 10 contient des fonctionnalités basées sur un sous-ensemble des fonctionnalités fournies dans la version de bureau du contrôle Lecteur Windows Media. Par conséquent, il peut être incorporé dans une page Web Pocket Internet Explorer de la même façon que le contrôle Desktop est incorporé dans une page Web Internet Explorer. pour savoir si une méthode, une propriété ou un événement particulier est pris en charge pour le contrôle Mobile Lecteur Windows Media 10, consultez [référence du modèle objet pour les scripts](object-model-reference-for-scripting.md).
