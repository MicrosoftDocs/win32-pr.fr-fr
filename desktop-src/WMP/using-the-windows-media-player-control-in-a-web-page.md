# <a name="using-the-windows-media-player-control-in-a-web-page"></a>Utilisation du contrôle du lecteur Windows Media dans une page Web

L’incorporation du contrôle du lecteur Windows Media dans une page Web vous permet de personnaliser complètement la manière dont l’utilisateur interagit avec le contrôle. Vous pouvez utiliser l’interface fournie par le contrôle, ou vous pouvez la masquer et afficher votre propre interface utilisateur. Vous pouvez spécifier plusieurs propriétés du contrôle du lecteur Windows Media à l’endroit où vous incorporez le contrôle, ou vous pouvez définir les propriétés du lecteur et appeler les méthodes de lecteur dans le code de script.

Les sections suivantes décrivent les principes fondamentaux de l’incorporation du contrôle dans une page Web.



| Section                                                                                                        | Description                                                                                                                       |
|----------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------|
| [Masquage du contrôle du lecteur Windows Media](hiding-the-windows-media-player-control.md)                         | Décrit les options permettant de masquer le contrôle du lecteur Windows Media lorsque vous souhaitez fournir votre propre interface utilisateur.               |
| [Éléments PARAM dans un élément OBJECT](param-elements-in-an-object-element.md)                                 | Décrit comment initialiser le contrôle du lecteur Windows Media quand vous l’incorporez.                                                   |
| [Exemple simple de script dans une page Web](simple-example-of-scripting-in-a-web-page.md)                     | Décrit un exemple simple, mais complet, d’un contrôle de lecteur incorporé avec une interface utilisateur personnalisée.                             |
| [Utilisation du contrôle du lecteur Windows Media avec Firefox](using-the-windows-media-player-control-with-firefox.md) | Décrit comment incorporer le contrôle du lecteur Windows Media dans une page Web afin qu’il s’affiche correctement par un navigateur Firefox. |
| [Utilisation de Windows Media Player avec Netscape 7.1](using-windows-media-player-with-netscape-7-1.md)               | Décrit comment utiliser le contrôle du lecteur Windows Media avec Netscape 7,1 et d’autres navigateurs Web basés sur Gecko.                       |



 

> [!Note]  
> Le contrôle mobile du lecteur Windows Media 10 contient des fonctionnalités basées sur un sous-ensemble des fonctionnalités fournies dans la version de bureau du contrôle du lecteur Windows Media. Par conséquent, il peut être incorporé dans une page Web Pocket Internet Explorer de la même façon que le contrôle Desktop est incorporé dans une page Web Internet Explorer. Pour savoir si une méthode, une propriété ou un événement particulier est pris en charge pour le contrôle mobile du lecteur Windows Media 10, consultez [Référence du modèle objet pour les scripts](object-model-reference-for-scripting.md).
