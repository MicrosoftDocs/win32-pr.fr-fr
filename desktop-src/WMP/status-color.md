# <a name="status-color"></a>Couleur d’État

La valeur de couleur du texte d’affichage de l’État doit être spécifiée. La valeur doit être de trois entiers positifs séparés par des virgules. Les valeurs représentent la quantité de rouge, de vert et de bleu pour une couleur de texte, et sont comprises entre 0 et 255.
