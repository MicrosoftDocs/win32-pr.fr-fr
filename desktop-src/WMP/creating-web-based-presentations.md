# <a name="creating-web-based-presentations"></a>Création de présentations Web-Based

le contrôle Lecteur Windows Media vous permet de créer facilement des présentations basées sur le Web qui combinent du son et de la vidéo avec HTML. En ajoutant des commandes de script de type URL à vos fichiers multimédias, vous pouvez faire en sorte que les pages Web spécifiées s’affichent dans un cadre de navigateur Web spécifié à des moments spécifiques pendant la lecture des médias numériques.

Lecteur Windows Media intègre également une diffusion multimédia riche pour permettre une distribution efficace des données HTML sur un réseau à l’intérieur d’un fichier ou d’un flux de données multimédia unique Windows. Le lecteur et le serveur gèrent la livraison en temps utile de l’audio, de la vidéo et du code HTML dans le navigateur. Tout comme dans les présentations Web qui n’utilisent pas de diffusion multimédia enrichie, les commandes de script incorporées affichent la présentation dans une trame de navigateur spécifiée à des moments précis.

Une disposition classique des présentations Web utilise deux frames. une image contient une page web qui incorpore le contrôle Lecteur Windows Media et affiche la partie vidéo d’une présentation. L’autre cadre affiche des pages Web qui changent à différents moments de la lecture de la vidéo.

si les supports numériques accompagnant vos pages web contiennent uniquement de l’audio, vous pouvez masquer le frame qui contient le contrôle Lecteur Windows Media en définissant sa largeur sur zéro. De cette façon, l’audio peut s’exécuter sans interruption en arrière-plan pendant que vos pages Web s’affichent dans la fenêtre de navigateur complète.

Les sections suivantes décrivent les techniques courantes d’activation des présentations basées sur le Web :

-   [Diffusion multimédia enrichie](rich-media-streaming.md)
