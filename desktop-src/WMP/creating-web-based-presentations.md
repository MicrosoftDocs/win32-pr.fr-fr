# <a name="creating-web-based-presentations"></a>Création de présentations Web-Based

Le contrôle du lecteur Windows Media vous permet de créer facilement des présentations sur le Web qui combinent du contenu audio et vidéo avec du code HTML. En ajoutant des commandes de script de type URL à vos fichiers multimédias, vous pouvez faire en sorte que les pages Web spécifiées s’affichent dans un cadre de navigateur Web spécifié à des moments spécifiques pendant la lecture des médias numériques.

Le lecteur Windows Media offre également une diffusion multimédia riche pour permettre une livraison efficace des données HTML sur un réseau à l’intérieur d’un seul fichier ou flux Windows Media. Le lecteur et le serveur gèrent la livraison en temps utile de l’audio, de la vidéo et du code HTML dans le navigateur. Tout comme dans les présentations Web qui n’utilisent pas de diffusion multimédia enrichie, les commandes de script incorporées affichent la présentation dans une trame de navigateur spécifiée à des moments précis.

Une disposition classique des présentations Web utilise deux frames. Une image contient une page Web qui incorpore le contrôle du lecteur Windows Media et affiche la partie vidéo d’une présentation. L’autre cadre affiche des pages Web qui changent à différents moments de la lecture de la vidéo.

Si les supports numériques accompagnant vos pages Web contiennent uniquement de l’audio, vous pouvez masquer le frame qui contient le contrôle du lecteur Windows Media en définissant sa largeur sur zéro. De cette façon, l’audio peut s’exécuter sans interruption en arrière-plan pendant que vos pages Web s’affichent dans la fenêtre de navigateur complète.
