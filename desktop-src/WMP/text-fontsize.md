# <a name="textfontsize"></a>TEXT. FontSize

L’attribut **FontSize** spécifie ou récupère la taille de police du contrôle de texte.

``` syntax
        elementID.fontSize
```

## <a name="possible-values"></a>Valeurs possibles

Cet attribut est un **nombre** en lecture/écriture (**long**) qui spécifie la taille de police en points. Sa valeur par défaut est 10.

Pour obtenir un exemple illustrant l’utilisation des attributs de l’élément de **texte** , consultez l’attribut [value](text-value.md) .

## <a name="requirements"></a>Spécifications



| Condition requise | Valeur |
|--------------------|------------------------------------------------------|
| Version<br/> | Lecteur Windows Media version 7,0 ou ultérieure<br/> |
