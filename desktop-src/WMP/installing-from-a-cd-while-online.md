# <a name="installing-from-a-cd-while-online"></a>Installation à partir d’un CD-ROM en ligne

les utilisateurs peuvent installer Lecteur Windows Media à partir d’un CD tout en étant connecté à Internet. dans ce cas, Lecteur Windows Media installation localise le document serviceinfo spécifié par le paramètre de ligne de commande *ServiceInfo* . Si l’attribut de **clé** correspond au paramètre de ligne de commande *DefaultService* , le programme d’installation inspecte l’élément d’installation pour personnaliser le processus d’installation. À l’aide des valeurs d’attribut, le programme d’installation affiche le contrat de licence utilisateur final (CLUF) et votre déclaration de confidentialité, et récupère et installe également votre fichier .cab sur l’ordinateur de l’utilisateur. Par exemple, vous pouvez utiliser cette fonctionnalité pour installer la dernière version d’un objet COM requis par votre magasin en ligne.

une fois l’installation terminée, Lecteur Windows Media définit le magasin en ligne initial à l’aide du nom de clé que vous avez spécifié pour le paramètre de ligne de commande *DefaultService* .
