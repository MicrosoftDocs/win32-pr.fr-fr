# <a name="ambient-event-attributes"></a>Attributs d’événement ambiant

Les attributs suivants sont communs à tous les événements d’apparence. Elles sont accessibles avec le mot clé **Event** dans un gestionnaire d’événements pour l’élément.



| Attribut                              | Description                                                                                                  |
|----------------------------------------|--------------------------------------------------------------------------------------------------------------|
| [altKey](event-altkey.md)             | Récupère une valeur indiquant si la touche ALT était enfoncée lorsque l’événement s’est produit.                           |
| [bouton](event-button.md)             | Récupère une valeur indiquant les boutons de la souris qui étaient inactifs lorsque l’événement s’est produit.                          |
| [Une fois](event-clientx.md)           | Récupère la coordonnée x du pointeur de la souris par rapport à la zone cliente de la fenêtre d’application. |
| [client](event-clienty.md)           | Récupère la coordonnée y du pointeur de la souris par rapport à la zone cliente de la fenêtre d’application. |
| [ctrlKey](event-ctrlkey.md)           | Récupère une valeur indiquant si la touche CTRL était enfoncée lorsque l’événement s’est produit.                          |
| [fromElement](event-fromelement.md)   | Récupère l’élément d’origine de l’événement.                                                                   |
| [keycode](event-keycode.md)           | Récupère le code de touche ASCII si une touche a été enfoncée lorsque l’événement s’est produit.                                   |
| [offsetX](event-offsetx.md)           | Récupère la coordonnée x du pointeur de la souris par rapport à l’élément déclenchant l’événement.                |
| [décalage](event-offsety.md)           | Récupère la coordonnée y du pointeur de la souris par rapport à l’élément déclenchant l’événement.                |
| [screenHeight](event-screenheight.md) | Récupère la hauteur de la taille d’écran disponible, en pixels.                                                 |
| [Largeur](event-screenwidth.md)   | Récupère la largeur de la taille d’écran disponible, en pixels.                                                  |
| [screenX](event-screenx.md)           | Récupère la coordonnée x absolue du pointeur de la souris par rapport à l’écran.                         |
| [écran](event-screeny.md)           | Récupère la coordonnée y absolue du pointeur de la souris par rapport à l’écran.                         |
| [shiftKey](event-shiftkey.md)         | Récupère une valeur indiquant si la touche Maj était enfoncée lorsque l’événement s’est produit.                         |
| [srcElement](event-srcelement.md)     | Récupère l’élément qui a déclenché l’événement.                                                                  |
| [toElement](event-toelement.md)       | Récupère l’élément vers lequel le focus clavier a été déplacé.                                                           |
| [x](event-x.md)                       | Récupère la coordonnée x du pointeur de la souris par rapport à la fenêtre d’application.                      |
| [y](event-y.md)                       | Récupère la coordonnée y du pointeur de la souris par rapport à la fenêtre d’application.                      |
