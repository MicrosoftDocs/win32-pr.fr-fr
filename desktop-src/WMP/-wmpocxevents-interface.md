# <a name="_wmpocxevents-interface"></a>\_Interface WMPOCXEvents

L’interface **\_ WMPOCXEvents** fournit les mêmes événements que les interfaces **IWMPEvents**, **IWMPEvents2**, **IWMPEvents3** et **IWMPEvents4** , mais hérite de **IDispatch** et non de **IUnknown**. Cela vous permet de recevoir des notifications d’événements du lecteur Windows Media par le biais d’un récepteur d’événements **IDispatch** .
