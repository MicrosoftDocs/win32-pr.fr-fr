<span data-ttu-id="f835f-109">Cette page décrit une fonctionnalité qui peut ne pas être disponible dans les versions ultérieures du lecteur Windows Media et du kit de développement logiciel (SDK) du lecteur Windows Media.</span><span class="sxs-lookup"><span data-stu-id="f835f-109">This page documents a feature that may be unavailable in future versions of Windows Media Player and the Windows Media Player SDK.</span></span>

<span data-ttu-id="f835f-110">Les plug-ins de l’interface utilisateur de la zone d’affichage s’affichent dans la zone principale du volet **lecture** en cours, sous les informations de l’artiste et du suivi et à gauche de la zone de sélection.</span><span class="sxs-lookup"><span data-stu-id="f835f-110">Display area UI plug-ins are shown in the main area of the **Now Playing** pane, below the artist and track information and to the left of the playlist area.</span></span> <span data-ttu-id="f835f-111">Seul un plug-in de zone d’affichage peut être activé à la fois.</span><span class="sxs-lookup"><span data-stu-id="f835f-111">Only one display area plug-in can be enabled at a time.</span></span> <span data-ttu-id="f835f-112">Quand un plug-in de zone d’affichage est activé, le lecteur ne peut pas passer en mode plein écran.</span><span class="sxs-lookup"><span data-stu-id="f835f-112">When a display area plug-in is enabled, the Player cannot enter full screen mode.</span></span>

<span data-ttu-id="f835f-113">Les plug-ins de zone d’affichage sont utiles pour afficher de grandes quantités d’informations.</span><span class="sxs-lookup"><span data-stu-id="f835f-113">Display area plug-ins are useful for displaying large amounts of information.</span></span> <span data-ttu-id="f835f-114">Par exemple, un plug-in de zone d’affichage peut récupérer des informations sur l’élément multimédia en cours de visualisation et afficher des articles, des entretiens et des avis associés.</span><span class="sxs-lookup"><span data-stu-id="f835f-114">For example, a display area plug-in can retrieve information about the currently playing media item and display related articles, interviews, and reviews.</span></span> <span data-ttu-id="f835f-115">Un plug-in de zone d’affichage peut héberger une fenêtre de navigateur Microsoft Internet Explorer pour afficher ces informations en HTML.</span><span class="sxs-lookup"><span data-stu-id="f835f-115">A display area plug-in can host a Microsoft Internet Explorer browser window to display this information using HTML.</span></span>

> <span data-ttu-id="f835f-116">Une fenêtre de navigateur hébergée peut incorporer la version ActiveX du lecteur Windows Media, mais cela n’est pas recommandé, car le contrôle incorporé ne peut pas communiquer avec le lecteur autonome qui le contient.</span><span class="sxs-lookup"><span data-stu-id="f835f-116">A hosted browser window can embed the ActiveX version of Windows Media Player, but this is not recommended because the embedded control cannot communicate with the standalone player that contains it.</span></span>

 

> [!Note]
