# <a name="equalizersettingscrossfadewindow"></a>EQUALIZERSETTINGS.crossFadeWindow

L’attribut **crossFadeWindow** spécifie ou récupère la quantité de chevauchement de fondu croisé en millisecondes.

``` syntax
        elementID.crossFadeWindow
```

## <a name="possible-values"></a>Valeurs possibles

Cet attribut est un **nombre** en lecture/écriture (**long**) compris entre 0 et 10 000 avec une valeur par défaut de 250.

## <a name="requirements"></a>Configuration requise



| Condition requise | Valeur |
|--------------------|---------------------------------------------------|
| Version<br/> | Lecteur Windows Media série 9 ou version ultérieure<br/> |
