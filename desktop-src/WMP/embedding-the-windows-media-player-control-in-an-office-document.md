# <a name="embedding-the-windows-media-player-control-in-an-office-document"></a>incorporation du contrôle Lecteur Windows Media dans un Document Office

l’incorporation du contrôle Lecteur Windows Media dans un document Office est un moyen simple d’ajouter du contenu multimédia numérique interactif et dynamique à un document statique autrement. par exemple, vous pouvez créer une feuille de calcul dans Microsoft Excel et insérer une vidéo « discuter de l’en-tête » pour résumer un rapport, ou vous pouvez créer un document Microsoft Word et insérer une brève animation illustrant un point créé dans le texte. si vous n’aimez pas l’interface utilisateur fournie par le contrôle, vous pouvez utiliser Microsoft Visual Basic pour Applications (VBA) pour fournir une interface utilisateur personnalisée.

l’incorporation du contrôle Lecteur Windows Media dans un diaporama Microsoft PowerPoint® vous permet de compléter les effets dynamiques déjà fournis par ce programme. lorsque vous enregistrez la présentation sous forme de page web, le contrôle Lecteur Windows Media est incorporé dans le code HTML, comme décrit dans [utilisation du contrôle Lecteur Windows Media dans une Page Web](using-the-windows-media-player-control-in-a-web-page.md). Cela vous permet d’ajouter du code de script supplémentaire que vous avez peut-être développé pour d’autres pages Web.

une autre façon de générer facilement des pages web qui incorporent le contrôle de Lecteur Windows Media est avec Microsoft FrontPage®. FrontPage vous permet de créer rapidement une page, d’ajouter le contrôle Lecteur Windows Media à partir d’un menu et de le configurer dans une boîte de dialogue de propriétés. là encore, le code HTML qui en résulte incorpore le contrôle Lecteur Windows Media comme décrit précédemment.
