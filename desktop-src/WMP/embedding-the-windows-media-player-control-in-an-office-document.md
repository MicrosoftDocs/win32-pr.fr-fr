# <a name="embedding-the-windows-media-player-control-in-an-office-document"></a>Incorporation du contrôle du lecteur Windows Media dans un document Office

L’incorporation du contrôle du lecteur Windows Media dans un document Office est un moyen simple d’ajouter du contenu multimédia numérique interactif et dynamique à un document statique. Par exemple, vous pouvez créer une feuille de calcul dans Microsoft Excel et insérer une vidéo « en chef » résumant un rapport, ou vous pouvez créer un document Microsoft Word et insérer une brève animation illustrant un point dans le texte. Si vous n’aimez pas l’interface utilisateur fournie par le contrôle, vous pouvez utiliser Microsoft Visual Basic pour Applications (VBA) pour fournir une interface utilisateur personnalisée.

L’incorporation du contrôle du lecteur Windows Media dans un diaporama Microsoft PowerPoint® vous permet de compléter les effets dynamiques déjà fournis par ce programme. Lorsque vous enregistrez la présentation sous forme de page Web, le contrôle du lecteur Windows Media est incorporé dans le code HTML, comme décrit dans [utilisation du contrôle du lecteur Windows Media dans une page Web](using-the-windows-media-player-control-in-a-web-page.md). Cela vous permet d’ajouter du code de script supplémentaire que vous avez peut-être développé pour d’autres pages Web.

Une autre façon de générer facilement des pages Web qui incorporent le contrôle du lecteur Windows Media est avec Microsoft FrontPage®. FrontPage vous permet de créer rapidement une page, d’ajouter le contrôle du lecteur Windows Media à partir d’un menu et de le configurer dans une boîte de dialogue de propriétés. Là encore, le code HTML qui en résulte incorpore le contrôle du lecteur Windows Media comme décrit précédemment.
