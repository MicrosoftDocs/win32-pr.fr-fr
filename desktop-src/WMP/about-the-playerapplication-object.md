# <a name="about-the-playerapplication-object"></a>À propos de l’objet PlayerApplication

l’objet **PlayerApplication** est utilisé pour la communication à distance Lecteur Windows Media. il fournit les fonctionnalités requises pour basculer entre un contrôle de Lecteur Windows Media à distance et le mode complet du lecteur.

la communication à distance permet à un contrôle Lecteur Windows Media incorporé dans une application C++ d’utiliser le même moteur de lecture que le lecteur. Cela signifie que, généralement, vous utilisez l’objet **PlayerApplication** dans le code C++ à l’aide des interfaces com. toutefois, il existe un cas particulier où vous pouvez incorporer le contrôle Lecteur Windows Media qui affiche une apparence Lecteur Windows Media en tant qu’interface utilisateur. dans ce cas, **PlayerApplication** peut être programmé à l’aide de JScript dans le code de l’apparence.
