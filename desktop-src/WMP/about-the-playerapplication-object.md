# <a name="about-the-playerapplication-object"></a>À propos de l’objet PlayerApplication

L’objet **PlayerApplication** est utilisé pour la communication à distance du lecteur Windows Media. Il fournit les fonctionnalités requises pour basculer entre un contrôle du lecteur Windows Media distant et le mode complet du lecteur.

La communication à distance permet à un contrôle du lecteur Windows Media incorporé dans une application C++ d’utiliser le même moteur de lecture que le lecteur. Cela signifie que, généralement, vous utilisez l’objet **PlayerApplication** dans le code C++ à l’aide des interfaces com. Toutefois, il existe un cas particulier où vous pouvez incorporer le contrôle du lecteur Windows Media qui affiche une apparence du lecteur Windows Media comme interface utilisateur. Dans ce cas, **PlayerApplication** peut être programmé à l’aide de JScript dans le code de l’apparence.
