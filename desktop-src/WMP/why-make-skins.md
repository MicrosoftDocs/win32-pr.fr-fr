# <a name="why-make-skins"></a>Pourquoi créer des apparences ?

Quand quelqu’un utilise un programme dans un environnement informatique graphique tel que Microsoft Windows, les composants visuels avec lesquels l’utilisateur interagit composent l’interface utilisateur. L’un des objectifs de Microsoft Windows est de fournir une interface utilisateur standard pour que tous les programmes fonctionnent de manière similaire et familière. Par exemple, Microsoft recommande, dans l’expérience utilisateur de Microsoft Windows, que chaque programme fournisse un bouton **Fermer** dans le coin supérieur droit de la fenêtre principale du programme.

Le lecteur Windows Media fournit les fonctionnalités permettant de créer votre propre interface utilisateur. Si vous souhaitez placer le bouton **Fermer** au milieu de l’écran, vous pouvez le faire. Il se peut que vous ne vous plaît pas à l’apparence du bouton **Fermer** (il ressemble à un X à l’intérieur d’une zone); Si vous souhaitez qu’il ressemble à un crâne et à un Crossbones, vous pouvez créer une interface utilisateur où le bouton **Fermer** est là ! Le lecteur Windows Media vous offre tous les outils dont vous avez besoin pour créer une interface utilisateur personnalisée pour écouter de la musique et des vidéos : boutons, barres de curseur, fenêtres vidéo, fenêtres de visualisation, barres d’égalisation, etc.

Il existe plusieurs raisons pour lesquelles vous souhaiterez peut-être créer votre propre interface utilisateur pour le lecteur Windows Media. L’une des raisons est que vous souhaiterez peut-être ajouter des fonctionnalités qui ne sont pas déjà présentes dans le lecteur Windows Media. Par exemple, vous souhaiterez peut-être créer un joueur qui lit de la musique à partir de sélections basées sur l’heure de la journée, ce qui vous permet d’avoir Upbeat Rock le matin et un jazz lent au soir. Ou peut-être souhaitez-vous créer une apparence avec un grand bouton rouge qui arrêtera rapidement la musique. Le lecteur Windows Media n’est pas accompagné d’un bouton « lire les mêmes chansons à chaque fois jusqu’à ce que mon chambre devienne fou », mais si vous le souhaitez, vous pouvez le créer.

Une autre raison pour créer une apparence est de faire de la marque le lecteur Windows Media. Si vous distribuez de la musique à partir de votre site Web et que vous utilisez un logo particulier, vous souhaiterez peut-être concevoir une apparence qui utilise votre logo pour rappeler aux utilisateurs de votre site. Si vous avez une bande de rock, vous pouvez créer une apparence avec des images de votre bande.

Et une autre raison pour créer des apparences, c’est de rendre un aspect unique pouvant habiller votre bureau. Lorsque vos amis se retrouvent et vous demandent ce que vous voulez pour votre écran, vous pouvez le faire vous-même. Vous pouvez même prendre une photo de votre chien, l’analyser sur votre ordinateur, ajouter des boutons et créer une apparence dans laquelle le fait de cliquer sur le nez de votre chien démarre la musique et clique sur la queue pour l’arrêter. Vous pouvez créer différentes apparences pour différents types de musique ou avoir une apparence différente pour chaque jour de la semaine.
