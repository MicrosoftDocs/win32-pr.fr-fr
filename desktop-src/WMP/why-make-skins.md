# <a name="why-make-skins"></a><span data-ttu-id="e2a32-108">Pourquoi créer des apparences ?</span><span class="sxs-lookup"><span data-stu-id="e2a32-108">Why Make Skins?</span></span>

<span data-ttu-id="e2a32-109">Quand quelqu’un utilise un programme dans un environnement informatique graphique tel que Microsoft Windows, les composants visuels avec lesquels l’utilisateur interagit composent l’interface utilisateur.</span><span class="sxs-lookup"><span data-stu-id="e2a32-109">When someone uses a program in a graphical computing environment such as Microsoft Windows, the visual components with which the user interacts make up the user interface.</span></span> <span data-ttu-id="e2a32-110">L’un des objectifs de Microsoft Windows est de fournir une interface utilisateur standard pour que tous les programmes fonctionnent de manière similaire et familière.</span><span class="sxs-lookup"><span data-stu-id="e2a32-110">One of the purposes of Microsoft Windows is to provide a standard user interface so that all programs will operate in a similar, familiar way.</span></span> <span data-ttu-id="e2a32-111">Par exemple, Microsoft recommande, dans l’expérience utilisateur de Microsoft Windows, que chaque programme fournisse un bouton **Fermer** dans le coin supérieur droit de la fenêtre principale du programme.</span><span class="sxs-lookup"><span data-stu-id="e2a32-111">For example, Microsoft recommends, in the Microsoft Windows User Experience, that every program provide a **Close** button in the upper-right corner of the main window of the program.</span></span>

<span data-ttu-id="e2a32-112">Le lecteur Windows Media fournit les fonctionnalités permettant de créer votre propre interface utilisateur.</span><span class="sxs-lookup"><span data-stu-id="e2a32-112">Windows Media Player provides the capabilities for creating your own user interface.</span></span> <span data-ttu-id="e2a32-113">Si vous souhaitez placer le bouton **Fermer** au milieu de l’écran, vous pouvez le faire.</span><span class="sxs-lookup"><span data-stu-id="e2a32-113">If you want to put the **Close** button in the middle of the screen, you can do that.</span></span> <span data-ttu-id="e2a32-114">Il se peut que vous ne vous plaît pas à l’apparence du bouton **Fermer** (il ressemble à un X à l’intérieur d’une zone); Si vous souhaitez qu’il ressemble à un crâne et à un Crossbones, vous pouvez créer une interface utilisateur où le bouton **Fermer** est là !</span><span class="sxs-lookup"><span data-stu-id="e2a32-114">Perhaps you do not like the way the **Close** button looks (it looks like an X inside a box); if you want it to look like a skull and crossbones, you can make a user interface where the **Close** button is just that!</span></span> <span data-ttu-id="e2a32-115">Le lecteur Windows Media vous offre tous les outils dont vous avez besoin pour créer une interface utilisateur personnalisée pour écouter de la musique et des vidéos : boutons, barres de curseur, fenêtres vidéo, fenêtres de visualisation, barres d’égalisation, etc.</span><span class="sxs-lookup"><span data-stu-id="e2a32-115">Windows Media Player gives you all the tools you need to make a custom user interface for playing music and video: buttons, slider bars, video windows, visualization windows, equalization bars, and so on.</span></span>

<span data-ttu-id="e2a32-116">Il existe plusieurs raisons pour lesquelles vous souhaiterez peut-être créer votre propre interface utilisateur pour le lecteur Windows Media.</span><span class="sxs-lookup"><span data-stu-id="e2a32-116">There are several good reasons why you might want to create your own user interface for Windows Media Player.</span></span> <span data-ttu-id="e2a32-117">L’une des raisons est que vous souhaiterez peut-être ajouter des fonctionnalités qui ne sont pas déjà présentes dans le lecteur Windows Media.</span><span class="sxs-lookup"><span data-stu-id="e2a32-117">One reason is that you might want to add functionality that is not already in Windows Media Player.</span></span> <span data-ttu-id="e2a32-118">Par exemple, vous souhaiterez peut-être créer un joueur qui lit de la musique à partir de sélections basées sur l’heure de la journée, ce qui vous permet d’avoir Upbeat Rock le matin et un jazz lent au soir.</span><span class="sxs-lookup"><span data-stu-id="e2a32-118">For example, you might want to create a player that plays music from playlists that are based on the time of day, so that you have upbeat rock in the morning and slow jazz in the evening.</span></span> <span data-ttu-id="e2a32-119">Ou peut-être souhaitez-vous créer une apparence avec un grand bouton rouge qui arrêtera rapidement la musique.</span><span class="sxs-lookup"><span data-stu-id="e2a32-119">Or perhaps you want to make a skin with a big red button that will stop the music quickly.</span></span> <span data-ttu-id="e2a32-120">Le lecteur Windows Media n’est pas accompagné d’un bouton « lire les mêmes chansons à chaque fois jusqu’à ce que mon chambre devienne fou », mais si vous le souhaitez, vous pouvez le créer.</span><span class="sxs-lookup"><span data-stu-id="e2a32-120">Windows Media Player does not come with a "play the same song over and over again until my roommate goes crazy" button, but if you want one, you can create it.</span></span>

<span data-ttu-id="e2a32-121">Une autre raison pour créer une apparence est de faire de la marque le lecteur Windows Media.</span><span class="sxs-lookup"><span data-stu-id="e2a32-121">Another reason for creating a skin is to make a branded look for Windows Media Player.</span></span> <span data-ttu-id="e2a32-122">Si vous distribuez de la musique à partir de votre site Web et que vous utilisez un logo particulier, vous souhaiterez peut-être concevoir une apparence qui utilise votre logo pour rappeler aux utilisateurs de votre site.</span><span class="sxs-lookup"><span data-stu-id="e2a32-122">If you are distributing music from your website and use a particular logo, you might want to design a skin that uses your logo to remind people about your site.</span></span> <span data-ttu-id="e2a32-123">Si vous avez une bande de rock, vous pouvez créer une apparence avec des images de votre bande.</span><span class="sxs-lookup"><span data-stu-id="e2a32-123">If you have a rock band, you can make a skin with pictures of your band on it.</span></span>

<span data-ttu-id="e2a32-124">Et une autre raison pour créer des apparences, c’est de rendre un aspect unique pouvant habiller votre bureau.</span><span class="sxs-lookup"><span data-stu-id="e2a32-124">And another reason to make skins is to make something unique that can dress up your desktop.</span></span> <span data-ttu-id="e2a32-125">Lorsque vos amis se retrouvent et vous demandent ce que vous voulez pour votre écran, vous pouvez le faire vous-même.</span><span class="sxs-lookup"><span data-stu-id="e2a32-125">When your friends come over and ask you what that cool program on your screen is, you can say you made it yourself.</span></span> <span data-ttu-id="e2a32-126">Vous pouvez même prendre une photo de votre chien, l’analyser sur votre ordinateur, ajouter des boutons et créer une apparence dans laquelle le fait de cliquer sur le nez de votre chien démarre la musique et clique sur la queue pour l’arrêter.</span><span class="sxs-lookup"><span data-stu-id="e2a32-126">You can even take a picture of your dog, scan it into your computer, add some buttons, and create a skin in which clicking on your dog's nose starts the music and clicking on the tail stops it.</span></span> <span data-ttu-id="e2a32-127">Vous pouvez créer différentes apparences pour différents types de musique ou avoir une apparence différente pour chaque jour de la semaine.</span><span class="sxs-lookup"><span data-stu-id="e2a32-127">You can create different skins for different kinds of music or have a different skin for every day of the week.</span></span>
