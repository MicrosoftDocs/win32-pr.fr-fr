# <a name="popup-element"></a>POPUP (élément)

L’élément **Popup** offre aux utilisateurs la possibilité de sélectionner des éléments dans une liste. Ces éléments peuvent être spécifiés à l’aide d’éléments **Item** comme enfants de l’élément **Popup** . L’élément **Popup** est identique à l’élément **ListBox** , sauf que la valeur par défaut de l’attribut **Popup** est true. Lorsque **popUp** a la valeur true, le contrôle s’affiche uniquement lorsque la méthode **Show** est appelée, et sa largeur est définie sur la largeur du contenu (l’attribut **Width** ambiant est ignoré). Le contrôle est masqué automatiquement lorsque l’utilisateur sélectionne un élément dans la liste ou lorsque la méthode d' **Ignorer** est appelée.

> [!Note]  
> cet élément requiert Lecteur Windows Media pour Windows XP ou version ultérieure.
