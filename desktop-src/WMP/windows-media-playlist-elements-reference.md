# <a name="windows-media-playlist-elements-reference"></a>Windows Référence des éléments de sélection de média

cette section contient la documentation de référence pour tous les éléments de sélection de média Windows. Les entrées de référence incluent les définitions des éléments, leurs attributs et leurs valeurs, ainsi que les conditions spéciales associées à chaque élément.

Les éléments de sélection suivants sont pris en charge.



| Élément                                    | Description                                                                                                                                                                                          |
|--------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [argument](argument-element.md)           | Contient une partie d’une chaîne de condition.                                                                                                                                                          |
| [body](body-element.md)                   | Contient les éléments qui définissent le contenu d’une sélection.                                                                                                                                        |
| [filter](filter-element.md)               | Contient des éléments qui limitent la taille d’une sélection, la durée d’une sélection ou le nombre d’éléments multimédias dans une sélection.                                                                                 |
| [échantillon](fragment-element.md)           | Spécifie une condition de la requête qui sélectionne des éléments de la bibliothèque.                                                                                                                            |
| [head](head-element.md)                   | Contient des métadonnées qui s’appliquent à la sélection entière.                                                                                                                                               |
| [media](media-element.md)                 | Spécifie l’un des éléments multimédias dans une sélection.                                                                                                                                                      |
| [METADONNEES](meta-element.md)                   | Spécifie les métadonnées qui s’appliquent à la sélection entière.                                                                                                                                              |
| [querySet](queryset-element.md)           | Contient des éléments qui définissent les éléments multimédias qui seront sélectionnés à partir de la bibliothèque.                                                                                                                   |
| [séquentiel](seq-element.md)                     | Contient des éléments qui définissent une sélection statique ou des éléments qui définissent une sélection intelligente.                                                                                                            |
| [smartPlaylist](smartplaylist-element.md) | Contient la partie définie dynamiquement d’une sélection.                                                                                                                                              |
| [SMIL](smil-element.md)                   | l’élément **smil** est toujours l’élément de niveau supérieur dans un fichier de sélection de média Windows (WPL). Il spécifie que le fichier utilise la syntaxe et la grammaire du langage SMIL (Synchronized Multimedia Integration Language). |
| [sourceFilter](sourcefilter-element.md)   | Contient des éléments qui sélectionnent des éléments dans une bibliothèque.                                                                                                                                                  |
| [title](title-element--wpl.md)            | Spécifie un titre pour la sélection.                                                                                                                                                                  |
