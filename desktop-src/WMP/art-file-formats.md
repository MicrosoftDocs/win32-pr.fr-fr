# <a name="art-file-formats"></a>Formats de fichier art

Les formats de fichier art suivants sont reconnus par le lecteur Windows Media pour les habillages :



| Format                                  | Extension   | Description                                                                                        |
|-----------------------------------------|-------------|----------------------------------------------------------------------------------------------------|
| Bitmap                                  | .bmp        | Les images bitmap sont recommandées, car elles offrent le plus de contrôle sur l’image exacte et les couleurs. |
| format GIF (Graphics Interchange Format)       | .gif        | Format d’image compressé utilisé pour les pages Web. Les gif animés sont pris en charge.                            |
| Joint Photographic Experts Group (JPEG) | .jpeg, .jpg | Format d’image compressé utilisé pour les pages Web.                                                         |
| format PNG (Portable Network Graphics)         | .png        | Format d’image compressé utilisé pour les pages Web.                                                         |



 

Si vous utilisez l’un des formats de fichiers compressés qui définissent une couleur transparente pour un navigateur Web, ne définissez pas une couleur comme transparente dans le fichier image. Utilisez une couleur visible pour représenter des zones transparentes dans votre image, puis définissez cette couleur comme transparente dans le fichier de définition d’apparence. Par exemple, si vous créez un fichier GIF avec des zones transparentes, elles ne seront pas transparentes dans votre image finale et vous ne pourrez pas utiliser la couleur que vous définissez comme transparente dans votre fichier GIF pour la couleur de transparence dans votre apparence.
