# <a name="columncolumnname"></a>COLUMN. columnName

L’attribut **ColumnName** spécifie ou récupère un nom de colonne dans le contrôle **playlist** .

``` syntax
        elementID.columnName
```

## <a name="possible-values"></a>Valeurs possibles

Cet attribut est une **chaîne** en lecture/écriture.

## <a name="remarks"></a>Notes

Le nom de colonne est le nom convivial qui s’affiche dans l’en-tête de la colonne.

## <a name="requirements"></a>Configuration requise



| Condition requise | Valeur |
|--------------------|---------------------------------------------------|
| Version<br/> | Lecteur Windows Media série 9 ou version ultérieure<br/> |
