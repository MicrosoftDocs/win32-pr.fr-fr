# <a name="implementing-your-code"></a>Implémentation de votre code

Une fois que vous avez terminé le processus de génération, vous souhaiterez probablement améliorer le code généré par l’Assistant. Dans le cas contraire, votre visualisation ressemble à des barres. Ce que vous devez faire, c’est modifier l’implémentation de la fonction de **rendu** et éventuellement la fonction **RenderWindowed** .

Les sections suivantes fournissent des détails sur la modification de l’exemple de code généré par l’Assistant de plug-in :

-   [Implémentation du rendu](implementing-render.md)
-   [Implémentation d’autres fonctions](implementing-other-functions.md)
-   [Présélections](presets.md)
