# <a name="effectseffecthaspropertypage"></a>EFFECTs. effectHasPropertyPage

L’attribut **effectHasPropertyPage** récupère une valeur indiquant si la visualisation en cours a une page de propriétés.

``` syntax
        elementID.effectHasPropertyPage
```

## <a name="possible-values"></a>Valeurs possibles

Cet attribut est une **valeur booléenne** en lecture seule.



| Valeur | Description                                  |
|-------|----------------------------------------------|
| true  | La visualisation a une page de propriétés.           |
| false | La visualisation n’a pas de page de propriétés. |



 

## <a name="requirements"></a>Configuration requise



| Condition requise | Valeur |
|--------------------|------------------------------------------------------|
| Version<br/> | Lecteur Windows Media version 7,0 ou ultérieure<br/> |
