# <a name="setting-the-initial-online-store"></a>Définition de la boutique en ligne initiale

Pour plus d’informations sur la redistribution du logiciel Windows Media Player avec votre application, consultez [redistribution du logiciel Windows Media Player](redistributing-windows-media-player-software.md).

Lorsqu’un utilisateur installe pour la première fois le lecteur Windows Media, l’application d’installation définit la Banque en ligne active initiale sur une valeur par défaut choisie par Microsoft. Les fabricants d’ordinateurs personnels peuvent choisir de modifier ce paramètre.

Pour spécifier et configurer le premier magasin en ligne que l’utilisateur voit lorsqu’il installe le lecteur Windows Media, vous devez :

-   Créez un document ServiceInfo que vous installez sur le disque dur de l’utilisateur. Ce document contient des informations sur la configuration du magasin en ligne actif initial.
-   Ajoutez un ensemble de paramètres de ligne de commande lors de l’exécution de l’application d’installation.

Il existe trois scénarios pour installer le lecteur Windows Media et définir le magasin en ligne initial. Les sections suivantes fournissent plus de détails sur chaque scénario :

-   [Installation en mode hors connexion](installing-while-offline.md)
-   [Installation à partir d’un CD-ROM en ligne](installing-from-a-cd-while-online.md)
-   [Installation à partir du Web en ligne](installing-from-the-web-while-online.md)
