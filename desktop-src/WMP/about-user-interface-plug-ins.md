# <a name="about-user-interface-plug-ins"></a>À propos des plug-ins d’interface utilisateur

Le lecteur Windows Media propose un large éventail de panneaux de configuration qui permettent à l’utilisateur de modifier divers aspects du lecteur, tels que les paramètres d’égaliseur vidéo et graphique. Ces panneaux de contrôle peuvent améliorer l’expérience multimédia fournie par le lecteur Windows Media, mais ils ne fournissent pas nécessairement toutes les fonctionnalités que l’utilisateur peut rechercher.

Les apparences sont un moyen de fournir des fonctionnalités supplémentaires, mais elles exigent que le développeur recrée l’interface utilisateur entière. En guise d’alternative, le lecteur Windows Media permet de créer des plug-ins d’interface utilisateur personnalisés qui s’affichent en mode complet du lecteur. Cette fonctionnalité est fournie par le biais d’une interface de programmation qui respecte les instructions du modèle COM (Component Object Model) Microsoft standard. Vous pouvez implémenter cette interface dans Microsoft Visual C++ à l’aide de l’Assistant de plug-in du lecteur Windows Media pour vous aider à commencer.
