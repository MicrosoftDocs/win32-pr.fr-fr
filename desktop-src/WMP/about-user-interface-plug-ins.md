# <a name="about-user-interface-plug-ins"></a>À propos des plug-ins d’interface utilisateur

Lecteur Windows Media fournit un large éventail de panneaux de configuration qui permettent à l’utilisateur de modifier divers aspects du lecteur, tels que les paramètres d’égaliseur vidéo et graphique. ces panneaux de contrôle peuvent améliorer l’expérience multimédia fournie par Lecteur Windows Media, mais ils ne fournissent pas nécessairement toutes les fonctionnalités que l’utilisateur peut rechercher.

Les apparences sont un moyen de fournir des fonctionnalités supplémentaires, mais elles exigent que le développeur recrée l’interface utilisateur entière. en guise d’alternative, Lecteur Windows Media permet de créer des plug-ins d’interface utilisateur personnalisés qui s’affichent en mode complet du lecteur. Cette fonctionnalité est fournie par le biais d’une interface de programmation qui respecte les instructions du modèle COM (Component Object Model) Microsoft standard. vous pouvez implémenter cette interface dans Microsoft Visual C++ à l’aide de l’assistant de Plug-in Lecteur Windows Media pour vous aider à commencer.

Les plug-ins d’interface utilisateur sont décrits plus en détail dans les rubriques suivantes.
