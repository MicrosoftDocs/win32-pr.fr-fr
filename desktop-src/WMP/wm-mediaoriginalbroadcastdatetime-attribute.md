# <a name="wmmediaoriginalbroadcastdatetime-attribute"></a>Attribut WM/MediaOriginalBroadcastDateTime

L’attribut **WM/MediaOriginalBroadcastDateTime** spécifie la date et l’heure de la première diffusion d’un affichage donné.

## <a name="applies-to"></a>S'applique à

-   [Éléments vidéo](video-item-attributes.md)

## <a name="remarks"></a>Notes

la constante du kit de développement logiciel (SDK) du Format multimédia Windows pour cet attribut est g \_ wszWMMediaOriginalBroadcastDateTime.

Pour déterminer si vous pouvez modifier la valeur de cet attribut, utilisez la méthode [Media. isReadOnlyItem](media-isreadonlyitem.md) .

## <a name="requirements"></a>Spécifications



| Condition requise | Valeur |
|--------------------|---------------------------------------------|
| Version<br/> | Lecteur Windows Media 10 ou version ultérieure<br/> |
