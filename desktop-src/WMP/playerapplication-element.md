# <a name="playerapplication-element"></a>Élément PLAYERAPPLICATION

L’élément **PLAYERAPPLICATION** peut être utilisé par certaines apparences pour accéder à l’objet **PLAYERAPPLICATION** . Cet élément fournit des fonctionnalités similaires à l’attribut global **playerApplication** . Cet élément est utilisé uniquement par les apparences affichées par les applications C++ qui incorporent une instance distante du contrôle du lecteur Windows Media.

L’élément **PLAYERAPPLICATION** prend en charge les attributs suivants.



| Attribut                                              | Description                                                                                              |
|--------------------------------------------------------|----------------------------------------------------------------------------------------------------------|
| [hasDisplay](playerapplication-hasdisplay-scr.md)     | Récupère une valeur indiquant si la vidéo peut s’afficher via le contrôle du lecteur Windows Media distant. |
| [playerDocked](playerapplication-playerdocked-scr.md) | Récupère une valeur indiquant si le lecteur Windows Media est dans un état d’ancrage.                          |
