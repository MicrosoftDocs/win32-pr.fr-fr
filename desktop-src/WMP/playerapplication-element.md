# <a name="playerapplication-element"></a>Élément PLAYERAPPLICATION

L’élément **PLAYERAPPLICATION** peut être utilisé par certaines apparences pour accéder à l’objet **PLAYERAPPLICATION** . Cet élément fournit des fonctionnalités similaires à l’attribut global **playerApplication** . cet élément est utilisé uniquement par les apparences affichées par les applications C++ qui incorporent une instance distante de Lecteur Windows Media contrôle.

L’élément **PLAYERAPPLICATION** prend en charge les attributs suivants.



| Attribut                                              | Description                                                                                              |
|--------------------------------------------------------|----------------------------------------------------------------------------------------------------------|
| [hasDisplay](playerapplication-hasdisplay-scr.md)     | récupère une valeur indiquant si la vidéo peut s’afficher par le biais du contrôle de Lecteur Windows Media distant. |
| [playerDocked](playerapplication-playerdocked-scr.md) | récupère une valeur indiquant si Lecteur Windows Media est dans un état ancré.                          |
