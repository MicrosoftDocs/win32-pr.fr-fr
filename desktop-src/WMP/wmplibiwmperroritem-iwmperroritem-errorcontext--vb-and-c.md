# <a name="iwmperroritemerrorcontext-property"></a>IWMPErrorItem :: errorContext, propriété

La propriété **errorContext** obtient une valeur indiquant le contexte de l’erreur.

## <a name="syntax"></a>Syntaxe


```CSharp
public System.Object errorContext {get; set;}
```


```VB

Public ReadOnly Property errorContext As System.Object
```





## <a name="property-value"></a>Valeur de la propriété

**System. Object** qui est le contexte de l’erreur.

## <a name="remarks"></a>Notes

Le contexte d’erreur est une information utilisée par Microsoft pour fournir des informations supplémentaires au personnel du support technique.
