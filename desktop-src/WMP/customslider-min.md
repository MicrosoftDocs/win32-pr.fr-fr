# <a name="customslidermin"></a>CUSTOMSLIDER. min.

L’attribut **min** spécifie ou récupère la valeur minimale de la plage définie par le curseur personnalisé.

``` syntax
        elementID.min
```

## <a name="possible-values"></a>Valeurs possibles

Cet attribut est un **nombre** en lecture/écriture (**float**) dont la valeur par défaut est zéro.

## <a name="remarks"></a>Notes

La valeur **min** doit être inférieure à **Max**.

## <a name="examples"></a>Exemples

Pour obtenir un exemple illustrant l’utilisation des attributs de l’élément **CUSTOMSLIDER** , consultez l’attribut [positionImage](customslider-positionimage.md) .
