# <a name="eventfromelement"></a>Event. fromElement

L’attribut **FromElement** récupère l’élément d’origine de l’événement. Si aucun élément ne passe un événement, la valeur est **null**.

``` syntax
event.fromElement
```

## <a name="possible-values"></a>Valeurs possibles

Cet attribut est un objet en lecture seule.

## <a name="requirements"></a>Configuration requise



| Condition requise | Valeur |
|--------------------|------------------------------------------------------|
| Version<br/> | Lecteur Windows Media version 7,0 ou ultérieure<br/> |
