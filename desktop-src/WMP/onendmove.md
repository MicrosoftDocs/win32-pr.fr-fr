# <a name="onendmove"></a>onendmove

Le gestionnaire d’événements **onendmove** gère un événement qui se produit lorsqu’un élément termine une opération MoveTo.

``` syntax
onendmove
```

## <a name="requirements"></a>Spécifications



| Condition requise | Valeur |
|--------------------|-----------------------------------------------------|
| Version<br/> | Lecteur Windows Media version 70 ou ultérieure<br/> |
