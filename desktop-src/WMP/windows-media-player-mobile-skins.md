# <a name="windows-media-player-mobile-skins"></a>Apparences mobiles du lecteur Windows Media

Le logiciel mobile du lecteur Windows Media inclut une apparence par défaut qui se compose de différents boutons et curseurs que vous pouvez utiliser pour lire, rembobiner ou arrêter des médias numériques. Une apparence est une interface utilisateur personnalisable qui fournit une apparence et des fonctionnalités uniques pour Windows Media Player Mobile.

Les sections suivantes décrivent comment créer des habillages pour Windows Media Player Mobile.



| Section                               | Description                                                                                                   |
|---------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [À propos des apparences](about-skins-mobile.md) | Décrit les concepts nécessaires pour comprendre l’architecture de l’apparence et les règles que vous devez suivre pour créer une apparence. |
| [Repère d’apparence](skin-guide.md)          | Montre comment créer une apparence simple.                                                                            |
| [Référence d’apparence](skin-reference.md)  | Contient des références détaillées pour toutes les parties d’une apparence.                                                     |
