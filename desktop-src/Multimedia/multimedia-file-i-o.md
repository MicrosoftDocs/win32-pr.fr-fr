# <a name="multimedia-file-io"></a>E/s de fichier multimédia

Les services d’e/s de fichiers multimédias offrent plus de fonctionnalités que les services de système d’exploitation standard, notamment la prise en charge des e/s mises en mémoire tampon, des fichiers RIFF (Resource Interchange File Format), des fichiers de mémoire et des systèmes de stockage personnalisés. En outre, les services d’e/s de fichier multimédia sont optimisés pour les applications sensibles aux performances.
