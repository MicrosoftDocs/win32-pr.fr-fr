# <a name="wavestream"></a>WaveStream

Les applications utilisent les fonctions de rappel WaveStream pendant la capture en continu pour traiter les tampons audio. La fenêtre de capture appelle une fonction de rappel WaveStream juste avant d’écrire chaque mémoire tampon audio sur le disque.
