# <a name="user-data-messages"></a>Messages de données utilisateur

Vous pouvez associer des données à une fenêtre de capture à l’aide de la fonction [**WM \_ Cap \_ obtenir les \_ \_ données utilisateur**](wm-cap-get-user-data.md) et [**WM \_ Cap \_ Set \_ User \_ Data**](wm-cap-set-user-data.md) (ou les macros [**capGetUserData**](/windows/desktop/api/Vfw/nf-vfw-capgetuserdata) et [**capSetUserData**](/windows/desktop/api/Vfw/nf-vfw-capsetuserdata) ). Vous pouvez récupérer une valeur de données de **type long** à l’aide du \_ message WM embout \_ obtenir \_ \_ des données utilisateur et définir une valeur de données de **type long** à l’aide du \_ message WM Cap \_ Set \_ User \_ Data.
