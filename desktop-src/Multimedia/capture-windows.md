# <a name="capture-windows"></a>Capturer des fenêtres

Les fenêtres de capture sont conceptuellement similaires aux contrôles standard (tels que les boutons, les zones de liste ou les barres de défilement). En règle générale, les fenêtres de capture utilisent les \_ styles de fenêtre WS Child et WS \_ visible.
