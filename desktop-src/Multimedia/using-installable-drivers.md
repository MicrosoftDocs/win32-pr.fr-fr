# <a name="using-installable-drivers"></a>Utilisation des pilotes installables

Vous utilisez des pilotes installables pour fournir aux applications ou aux dll une méthode standard d’accès à un appareil ou à un ensemble de routines utiles. Les sections suivantes montrent comment créer un pilote installable à l’aide d’une fonction [DriverProc](/windows/win32/api/mmiscapi/nc-mmiscapi-driverproc) et comment ouvrir un pilote installable et le diriger pour effectuer des tâches utiles.
