# <a name="mciwnd-window-class"></a>Classe de fenêtre MCIWnd

MCIWnd est une classe de fenêtre pour le contrôle des appareils multimédias. Une bibliothèque de fonctions, de messages et de macros associés à MCIWnd fournit une méthode simple pour ajouter des fonctionnalités de lecture multimédia ou d’enregistrement à vos applications.
