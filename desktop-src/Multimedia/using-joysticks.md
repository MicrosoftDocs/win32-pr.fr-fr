# <a name="using-joysticks"></a>Utilisation de manettes

Cette section contient des exemples montrant comment effectuer les tâches suivantes :

-   [Obtention des fonctionnalités du pilote](getting-the-driver-capabilities.md)
-   [Capture de l’entrée de manette](capturing-joystick-input.md)
-   [Traitement des messages de manette de jeu](processing-joystick-messages.md)

Les exemples sont tirés d’une application de manette de jeu simple qui récupère les informations de position et d’état de bouton à partir des services de la manette de jeu, lit les ressources audio Waveform et peint des trous de puces à l’écran lorsqu’un utilisateur appuie sur les boutons de la manette de jeu.
