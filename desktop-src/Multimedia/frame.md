# <a name="frame-windows-multimedia"></a>cadre (Windows multimédia)

Une fenêtre de capture utilise des messages de notification de rappel de frame pour notifier votre application quand une nouvelle image vidéo est disponible. La fenêtre de capture active ces notifications de rappel uniquement si le taux d’aperçu est différent de zéro et que la capture en continu n’est pas en cours.
