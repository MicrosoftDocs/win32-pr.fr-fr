# <a name="about-installable-drivers"></a>À propos des pilotes installables

Un pilote installable est une bibliothèque de liens dynamiques (DLL) qui fournit une interface standard par le biais de laquelle les applications Windows et les dll communiquent et gèrent le pilote. Les pilotes installables sont utilisés le plus souvent pour les pilotes de périphériques multimédias, mais les pilotes installables peuvent également être utilisés à d’autres fins et sont particulièrement utiles dans les situations qui nécessitent une interface standard et un contrôle de plusieurs instances.
