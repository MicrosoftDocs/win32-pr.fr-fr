# <a name="multimedia-pc-specifications"></a>Spécifications du PC multimédia

Le Conseil marketing PC Multimedia (MPC) a développé deux spécifications pour encourager l’adoption de fonctionnalités multimédias. La spécification de niveau 1, développée dans 1990, fournit une définition de base de l’informatique multimédia dans les fonctionnalités, les composants matériels et les composants logiciels. La spécification de niveau 2, émise dans 1993, s’appuie sur la première spécification et se concentre sur les fonctionnalités multimédias améliorées depuis l’émission de la première spécification.

Cette annexe résume les spécifications.

-   [Spécification de niveau 1](level-1-specification.md)
-   [Spécification de niveau 2](level-2-specification.md)
