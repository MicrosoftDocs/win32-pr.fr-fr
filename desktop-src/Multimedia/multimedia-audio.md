# <a name="multimedia-audio"></a>Audio multimédia

Windows fournit des fonctions qui permettent à une application d’ajouter des services audio. Les fonctions audio multimédias ont été introduites dans Windows 16 bits. La plupart de leurs fonctionnalités ont été remplacées par les fonctionnalités de DirectX. Pour plus d’informations, consultez la documentation de DirectX.

Les rubriques suivantes sont présentées dans la documentation audio multimédia.

-   [Gestionnaire de compression audio](audio-compression-manager.md)
-   [Mixages audio](audio-mixers.md)
-   [Interface MIDI (Musical Instrument Digital Interface)](musical-instrument-digital-interface--midi.md)
-   [Sons Waveform](waveform-audio.md)

L’audio peut également être lu à l’aide de l’interface de contrôle de média. Pour plus d’informations, consultez [MCI](mci.md).
