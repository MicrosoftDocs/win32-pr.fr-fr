# <a name="midi-services"></a>Services MIDI

La plupart des applications peuvent utiliser les mémoires tampons de séquence ou de flux MCI MIDI (et la fonction [**midiStreamOut**](/windows/win32/api/mmeapi/nf-mmeapi-midistreamout) ) pour implémenter toutes les fonctionnalités MIDI dont elles ont besoin. Les développeurs MIDI sérieux, ceux qui produisent des outils de création ou de séquencement MIDI, peuvent utiliser une combinaison des fonctionnalités de flux et des services MIDI, ou utiliser uniquement les services MIDI. Les rubriques suivantes fournissent des informations générales sur l’utilisation des services MIDI.

-   [Gestion des blocs de données MIDI](managing-midi-data-blocks.md)
-   [Demande de formats d’heure](requesting-time-formats.md)
-   [Gestion des erreurs avec les fonctions MIDI](handling-errors-with-midi-functions.md)
