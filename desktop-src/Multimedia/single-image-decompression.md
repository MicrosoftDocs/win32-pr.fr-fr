# <a name="single-image-decompression"></a>Décompression Single-Image

Vous pouvez utiliser la fonction [**ICImageDecompress**](/windows/desktop/api/Vfw/nf-vfw-icimagedecompress) pour décompresser une seule image. Cette fonction retourne un handle du DIB décompressé. Le DIB décompressé est stocké au format de CF \_ dib.
