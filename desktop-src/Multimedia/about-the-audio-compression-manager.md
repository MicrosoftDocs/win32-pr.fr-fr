# <a name="about-the-audio-compression-manager"></a>À propos du gestionnaire de compression audio

Le gestionnaire de compression audio ajoute la prise en charge au niveau du système pour les services suivants :

-   Compression et décompression audio à l’exécution transparente
-   Waveform-sélection du format de données audio
-   Waveform-sélection du filtre de données audio
-   Conversion du format de données Waveform-Audio
-   Filtrage des données Waveform-Audio
