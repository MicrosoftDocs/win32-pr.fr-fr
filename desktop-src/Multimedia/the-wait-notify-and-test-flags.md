# <a name="the-wait-notify-and-test-flags"></a>Indicateurs d’attente, de notification et de test

La plupart des commandes MCI incluent des indicateurs qui modifient la commande. Les indicateurs « Wait » (MCI \_ Wait) et « Notify » (MCI \_ Notify) sont communs à chaque commande. L’indicateur « test » ( \_ test MCI) est disponible pour les appareils vidéo numériques et VCR. Les rubriques suivantes décrivent l’utilisation de ces indicateurs.

-   [L’indicateur d’attente](the-wait-flag.md)
-   [Indicateur Notify](the-notify-flag.md)
-   [Indicateur de test](the-test-flag.md)
