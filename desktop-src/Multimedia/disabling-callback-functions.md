# <a name="disabling-callback-functions"></a>Désactivation des fonctions de rappel

Vous pouvez désactiver définitivement ou temporairement les fonctions de rappel en spécifiant la **valeur null** à la place de la fonction de rappel lors de l’envoi du message approprié pour définir une fonction de rappel.
