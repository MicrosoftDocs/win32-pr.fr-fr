# <a name="about-mci"></a>À propos de MCI

MCI fournit aux applications des fonctionnalités indépendantes du périphérique pour contrôler les périphériques audio et visuels. Votre application peut utiliser MCI pour contrôler tout périphérique multimédia pris en charge, y compris les périphériques audio Waveform, les séquenceurs MIDI, les périphériques audio CD et les périphériques vidéo numériques (lecture vidéo).
