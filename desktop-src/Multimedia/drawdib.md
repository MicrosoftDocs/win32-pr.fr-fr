# <a name="drawdib"></a>DrawDib

Les fonctions DrawDib offrent des fonctionnalités de dessin d’images hautes performances pour les bitmaps indépendantes du périphérique (DIB). Les fonctions DrawDib prennent en charge les images DIB de profondeur d’image 8 bits, 16 bits, 24 bits et 32 bits.

Les fonctions DrawDib écrivent directement dans la mémoire vidéo. Ils ne reposent pas sur les fonctions de l’interface GDI (Graphics Device Interface).

-   [Utilisation de DrawDib](using-drawdib.md)
-   [Référence DrawDib](drawdib-reference.md)
