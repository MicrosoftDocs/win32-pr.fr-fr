# <a name="video-compression-manager"></a>Gestionnaire de compression vidéo

Le gestionnaire de compression vidéo (VCM) permet d’accéder à l’interface utilisée par les compresseurs installables pour gérer les données en temps réel. Les applications peuvent utiliser des compresseurs installables pour effectuer les tâches suivantes :

-   Compressez et décompressez les données vidéo.
-   Envoyer des données vidéo compressées du convertisseur et les faire glisser à l’écran.
-   Compressez, décompressez ou dessinez des données avec des convertisseurs définis par l’application.
-   Utilisez les convertisseurs pour gérer le texte et les données personnalisées.
