# <a name="video-for-windows"></a>Vidéo pour Windows

Video for Windows (VFW) fournit des fonctions qui permettent à une application de traiter des données vidéo. VFW a été introduit dans Windows 16 bits. La plupart de ses fonctionnalités ont été remplacées par les fonctionnalités de DirectX. Pour plus d’informations, consultez la documentation de DirectX.

Les rubriques suivantes sont présentées dans la documentation vidéo pour Windows.

-   [Fonctions et macros AVIFile](avifile-functions-and-macros.md)
-   [Gestionnaire de compression vidéo](video-compression-manager.md)
-   [Capture vidéo](video-capture.md)
-   [Gestionnaires de fichiers et de flux personnalisés](custom-file-and-stream-handlers.md)
-   [DrawDib](drawdib.md)
