# <a name="video-for-windows"></a>Vidéo pour Windows

Video for Windows (VFW) fournit des fonctions qui permettent à une application de traiter des données vidéo. VFW a été introduit en Windows 16 bits. La plupart de ses fonctionnalités ont été remplacées par les fonctionnalités de DirectX. Pour plus d’informations, consultez la documentation de DirectX.
