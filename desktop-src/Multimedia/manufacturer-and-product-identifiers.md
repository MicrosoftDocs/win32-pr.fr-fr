# <a name="manufacturer-and-product-identifiers"></a>Identificateurs du fabricant et du produit

Cette annexe décrit les identificateurs de fabricant et de produit définis pour les applications multimédias. Ces identificateurs sont utilisés lorsqu’une application émet une requête sur les périphériques installés sur un ordinateur. Pour utiliser ces identificateurs dans votre application, incluez MMREG. Fichier H.
