# <a name="vcm-services"></a>Services VCM

En général, une application utilise VCM pour effectuer les tâches suivantes :

-   Recherchez, ouvrez ou installez un compresseur ou un décompresseur.
-   Configurez ou obtenez des informations de configuration sur le compresseur ou le décompresseur.
-   Utilisez une série de fonctions pour compresser, décompresser ou dessiner les données.

Les fonctions et les macros de la bibliothèque DrawDib effectuent ces tâches de manière implicite et peuvent fournir la méthode la plus pratique pour utiliser VCM. Pour plus d’informations sur la bibliothèque DrawDib, consultez [DrawDib](drawdib.md).
