# <a name="video-format"></a>Format vidéo

Vous pouvez récupérer la structure qui spécifie le format vidéo ou la taille de cette structure en envoyant le message [**WM \_ Cap \_ \_ VIDEOFORMAT**](wm-cap-get-videoformat.md) (ou les macros [**capGetVideoFormat**](/windows/desktop/api/Vfw/nf-vfw-capgetvideoformat) et [**capGetVideoFormatSize**](/windows/desktop/api/Vfw/nf-vfw-capgetvideoformatsize) ) à une fenêtre de capture. Vous pouvez définir le format des données de la vidéo capturée en envoyant le message [**WM \_ Cap \_ Set \_ VIDEOFORMAT**](wm-cap-set-videoformat.md) (ou la macro [**capSetVideoFormat**](/windows/desktop/api/Vfw/nf-vfw-capsetvideoformat) ) à une fenêtre de capture.
