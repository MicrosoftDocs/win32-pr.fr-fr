# <a name="basic-capture-options"></a>Options de capture de base

En modifiant un ou plusieurs des paramètres de capture définis dans la structure [**CAPTUREPARMS**](/windows/win32/api/vfw/ns-vfw-captureparms) , vous pouvez effectuer les tâches suivantes :

-   Modifiez la fréquence de capture du frame.
-   Spécifiez le contrôle du clavier ou de la souris pour mettre fin à une session de capture.
-   Spécifiez une durée pour une session de capture.
