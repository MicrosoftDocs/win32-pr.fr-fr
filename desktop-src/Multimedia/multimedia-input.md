# <a name="multimedia-input"></a>Entrée multimédia

Windows fournit des fonctions qui permettent à une application de recevoir une entrée multimédia. Les fonctions d’entrée multimédias ont été introduites en Windows 16 bits. La plupart de leurs fonctionnalités ont été remplacées par les fonctionnalités de DirectX. Pour plus d’informations, consultez la documentation de DirectX.

Les rubriques suivantes sont présentées dans la documentation sur les entrées multimédias.

-   [E/s de fichier multimédia](multimedia-file-i-o.md)
-   [Manette](joysticks.md)
-   [Minuteurs multimédias](multimedia-timers.md)
