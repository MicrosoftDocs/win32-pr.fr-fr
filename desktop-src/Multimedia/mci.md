# <a name="mci"></a>MCI

L’interface de contrôle de contenu (MCI) fournit des commandes standard pour la diffusion de périphériques multimédias et l’enregistrement de fichiers de ressources multimédias. Ces commandes sont une interface générique pour presque tous les types d’appareils multimédias.

-   [Utilisation des messages de commande MCI](using-mci-command-messages.md)
-   [Référence MCI](mci-reference.md)
