# <a name="multimedia"></a>Multimédia

ce glossaire contient les définitions des termes utilisés dans la documentation multimédia Windows.

<dl> <dt>

<span id="_win32_adaptive_differential_pulse_code_modulation_adpcm_gloss"></span><span id="_WIN32_ADAPTIVE_DIFFERENTIAL_PULSE_CODE_MODULATION_ADPCM_GLOSS"></span>**Modulation de code par impulsion différentielle adaptative (ADPCM)**
</dt> <dd>

Technique de compression audio.

</dd> <dt>

<span id="_win32_adpcm_gloss"></span><span id="_WIN32_ADPCM_GLOSS"></span>**ADPCM**
</dt> <dd>

Consultez [modulation adaptative de code pulsé (ADPCM)](/windows).

</dd> <dt>

<span id="_win32_break_key_gloss"></span><span id="_WIN32_BREAK_KEY_GLOSS"></span>**touche pause**
</dt> <dd>

Dans MCI (Media Control Interface), il s’agit d’une séquence d’exécution qui interrompt une opération d’attente. Par défaut, MCI définit cette clé comme CTRL + ATTN. Une application peut redéfinir cette clé à l’aide du \_ message de commande d’interruption MCI.

</dd> <dt>

<span id="_win32_cd_rom_extended_architecture_cd_xa_gloss"></span><span id="_WIN32_CD_ROM_EXTENDED_ARCHITECTURE_CD_XA_GLOSS"></span>**Architecture étendue de CD-ROM (CD-XA)**
</dt> <dd>

Extension du CD-ROM standard qui fournit le stockage des données audio compressées, ainsi que d’autres données sur un disque compact. Cette norme définit également la façon dont les données sont lues à partir d’un disque. Les signaux audio sont combinés au texte et aux données graphiques sur une seule piste pour qu’ils puissent être lus pratiquement en même temps.

</dd> <dt>

<span id="_win32_channel_gloss"></span><span id="_WIN32_CHANNEL_GLOSS"></span>**chaîne**
</dt> <dd>

Méthode, fournie par l’interface MIDI (Musical Instrument Digital Interface), pour envoyer des messages à un appareil individuel au sein d’une installation MIDI. Il y a 16 numéros de canaux MIDI. Les appareils d’une installation MIDI peuvent être dirigés pour répondre uniquement aux messages marqués d’un numéro de canal spécifique à l’appareil.

</dd> <dt>

<span id="_win32_channel_map_gloss"></span><span id="_WIN32_CHANNEL_MAP_GLOSS"></span>**carte de canal**
</dt> <dd>

Carte de canal, fournie par le Mappeur MIDI, qui peut rediriger les messages MIDI (Musical Instrument Digital Interface) d’un canal vers un autre. Voir aussi [Mappeur midi](/windows), [midi (Musical Instrument Digital Interface)](/windows).

</dd> <dt>

<span id="_win32_chunk_gloss"></span><span id="_WIN32_CHUNK_GLOSS"></span>**bloc**
</dt> <dd>

Bloc de construction de base d’un fichier RIFF (Resource Interchange File Format), qui se compose d’un identificateur (appelé identificateur de segment), d’une variable de taille de segment et d’une zone de données de segment de taille variable.

</dd> <dt>

<span id="_win32_command_message_gloss"></span><span id="_WIN32_COMMAND_MESSAGE_GLOSS"></span>**message de commande**
</dt> <dd>

Dans MCI (Media Control Interface), constante symbolique qui représente une commande unique pour un périphérique MCI. Les messages de commande ont des structures de données associées qui fournissent des informations nécessaires à un appareil pour effectuer une requête.

</dd> <dt>

<span id="_win32_command_string_gloss"></span><span id="_WIN32_COMMAND_STRING_GLOSS"></span>**chaîne de commande**
</dt> <dd>

Dans MCI (Media Control Interface), chaîne de caractères se terminant par un caractère null qui représente une commande pour un périphérique MCI. La chaîne de texte contient toutes les informations dont un périphérique MCI a besoin pour exécuter une requête. MCI analyse la chaîne de texte et la convertit en un message de commande équivalent et une structure de données qu’elle envoie ensuite à un pilote de périphérique MCI.

</dd> <dt>

<span id="_win32_compact_disc_digital_audio_cd_da_gloss"></span><span id="_WIN32_COMPACT_DISC_DIGITAL_AUDIO_CD_DA_GLOSS"></span>**disque compact-audio numérique (CD-DA)**
</dt> <dd>

Un format de stockage de données optique qui permet de stocker jusqu’à 73 minutes de données numériques audio de haute qualité sur un disque compact. Également connu sous le nom d’audio de livre rouge.

</dd> <dt>

<span id="_win32_compact_disc_read_only_memory_cd_rom_gloss"></span><span id="_WIN32_COMPACT_DISC_READ_ONLY_MEMORY_CD_ROM_GLOSS"></span>**disque compact – mémoire en lecture seule (CD-ROM)**
</dt> <dd>

Technologie de stockage de données optique qui permet de stocker de grandes quantités de données sur un disque compact.

</dd> <dt>

<span id="_win32_compound_device_gloss"></span><span id="_WIN32_COMPOUND_DEVICE_GLOSS"></span>**appareil composé**
</dt> <dd>

Périphérique MCI (Media Control Interface) qui requiert un élément d’appareil, généralement un fichier de données. Un exemple d’appareil composé est le pilote audio MCI Waveform. Voir aussi [élément d’appareil](/windows).

</dd> <dt>

<span id="_win32_control_change_gloss"></span><span id="_WIN32_CONTROL_CHANGE_GLOSS"></span>**modification du contrôle**
</dt> <dd>

Voir [midi-modifier le message](/windows).

</dd> <dt>

<span id="_win32_device_element_gloss"></span><span id="_WIN32_DEVICE_ELEMENT_GLOSS"></span>**élément Device**
</dt> <dd>

Données requises pour le fonctionnement des appareils composés MCI (Media Control Interface). L’élément Device est généralement un fichier de données d’entrée ou de sortie.

</dd> <dt>

<span id="_win32_division_type_gloss"></span><span id="_WIN32_DIVISION_TYPE_GLOSS"></span>**type de division**
</dt> <dd>

Technique utilisée pour représenter le temps entre les événements MIDI (musical instruments Digital Interface) dans un séquenceur MIDI.

</dd> <dt>

<span id="_win32_file_element_gloss"></span><span id="_WIN32_FILE_ELEMENT_GLOSS"></span>**élément file**
</dt> <dd>

Fichier complet contenu dans un fichier composé RIFF (Resource Interchange File Format).

</dd> <dt>

<span id="_win32_fm_synthesizer_gloss"></span><span id="_WIN32_FM_SYNTHESIZER_GLOSS"></span>**Synthétiseur FM**
</dt> <dd>

Voir [synthétiseur de fréquence (FM)](/windows).

</dd> <dt>

<span id="_win32_fourcc_four_character_code_gloss"></span><span id="_WIN32_FOURCC_FOUR_CHARACTER_CODE_GLOSS"></span>**FourCC (code à quatre caractères)**
</dt> <dd>

Code utilisé pour identifier les segments RIFF (Resource Interchange File Format). Un FOURCC est une quantité de 32 bits représentée sous la forme d’une séquence de un à quatre caractères alphanumériques ASCII, complétée à droite avec des caractères vides.

</dd> <dt>

<span id="_win32_frequency_modulation_fm_synthesizer_gloss"></span><span id="_WIN32_FREQUENCY_MODULATION_FM_SYNTHESIZER_GLOSS"></span>**synthétiseur de fréquence (FM)**
</dt> <dd>

Synthétiseur qui crée des sons en combinant la sortie d’oscillateurs numériques à l’aide d’une technique de modulation de fréquence.

</dd> <dt>

<span id="_win32_general_midi_gloss"></span><span id="_WIN32_GENERAL_MIDI_GLOSS"></span>**Midi général**
</dt> <dd>

Spécification de synthétiseur créée par l’Association de fabricants MIDI (MMA) définissant une configuration commune et un ensemble de fonctionnalités pour les synthétiseurs MIDI (Musical Instrument Digital Interface) grand public.

</dd> <dt>

<span id="_win32_hms_time_format_gloss"></span><span id="_WIN32_HMS_TIME_FORMAT_GLOSS"></span>**Format d’heure HMS**
</dt> <dd>

Format d’heure utilisé par l’interface MCI pour exprimer l’heure en heures, minutes et secondes. Le format d’heure HMS est principalement utilisé par les appareils videodisc.

</dd> <dt>

<span id="_win32_ima_gloss"></span><span id="_WIN32_IMA_GLOSS"></span>**Certificat IMA**
</dt> <dd>

Voir [Association Interactive Multimedia (IMA)](/windows) et [international midi Association (IMA)](/windows).

</dd> <dt>

<span id="_win32_interactive_multimedia_association_ima_gloss"></span><span id="_WIN32_INTERACTIVE_MULTIMEDIA_ASSOCIATION_IMA_GLOSS"></span>**Association Interactive Multimedia (IMA)**
</dt> <dd>

Association professionnelle de sociétés, d’institutions et de personnes impliquées dans la production et l’utilisation d’une technologie multimédia interactive.

</dd> <dt>

<span id="_win32_international_midi_association_ima_gloss"></span><span id="_WIN32_INTERNATIONAL_MIDI_ASSOCIATION_IMA_GLOSS"></span>**IMA (International midi Association)**
</dt> <dd>

Organisation à but non lucratif qui diffuse des informations sur la spécification MIDI (Musical Instrument Digital Interface).

</dd> <dt>

<span id="_win32_list_chunk_gloss"></span><span id="_WIN32_LIST_CHUNK_GLOSS"></span>**Segment de liste**
</dt> <dd>

Segment RIFF (Resource Interchange File Format) avec un identificateur de segment de liste. Les segments de liste contiennent une série de sous-segments.

</dd> <dt>

<span id="_win32_list_type_gloss"></span><span id="_WIN32_LIST_TYPE_GLOSS"></span>**type de liste**
</dt> <dd>

Code à quatre caractères (FOURCC) identifiant le type de données contenues dans un bloc RIFF (Resource Interchange File Format) avec un identificateur de segment de liste. Par exemple, un bloc LIST avec un type list INFO contient une liste d’informations sur un fichier, telles que la date et l’auteur de la création.

</dd> <dt>

<span id="_win32_mci_gloss"></span><span id="_WIN32_MCI_GLOSS"></span>**MCI**
</dt> <dd>

Consultez [interface de contrôle de média](/windows).

</dd> <dt>

<span id="_win32_media_control_interface_mci_gloss"></span><span id="_WIN32_MEDIA_CONTROL_INTERFACE_MCI_GLOSS"></span>**MCI (Media Control Interface)**
</dt> <dd>

Logiciel de contrôle de haut niveau qui fournit une interface indépendante du périphérique pour les périphériques multimédias et les fichiers multimédias. MCI comprend une interface de message de commande et une interface de chaîne de commande.

</dd> <dt>

<span id="_win32_midi_gloss"></span><span id="_WIN32_MIDI_GLOSS"></span>**Midi**
</dt> <dd>

Consultez l' [interface digitale Musical Instrument Digital](/windows).

</dd> <dt>

<span id="_win32_midi_control_change_message_gloss"></span><span id="_WIN32_MIDI_CONTROL_CHANGE_MESSAGE_GLOSS"></span>**Contrôle midi-modifier le message**
</dt> <dd>

Message MIDI (Musical Instrument Digital Interface) envoyé à un synthétiseur pour modifier différents paramètres de contrôle de synthétiseur. Le message de contrôleur de volume est un exemple de message de modification de contrôle qui modifie le volume d’un canal MIDI spécifique.

</dd> <dt>

<span id="_win32_midi_manufacturers_association_mma_gloss"></span><span id="_WIN32_MIDI_MANUFACTURERS_ASSOCIATION_MMA_GLOSS"></span>L' **Association de fabricants midi (MMA)**
</dt> <dd>

Organisation collective composée de fabricants d’instruments MIDI (Musical Instrument Digital Interface) et de sociétés de logiciels MIDI. Le MMA travaille avec le Comité MIDI standard pour maintenir la spécification MIDI.

</dd> <dt>

<span id="_win32_midi_mapper_gloss"></span><span id="_WIN32_MIDI_MAPPER_GLOSS"></span>**Mappeur midi**
</dt> <dd>

Windows les logiciels système qui modifient les messages de sortie midi (Musical Instrument Digital Interface) et les redirige vers un périphérique de sortie midi à l’aide des valeurs stockées dans une carte d’installation midi. Le Mappeur MIDI peut modifier le canal de destination et le périphérique de sortie pour un message, ainsi que modifier les messages de modification de programme, les valeurs de volume et les valeurs de clé.

</dd> <dt>

<span id="_win32_midi_mapping_gloss"></span><span id="_WIN32_MIDI_MAPPING_GLOSS"></span>**Mappage midi**
</dt> <dd>

Processus de traduction et de redirection des messages MIDI (Musical Instrument Digital Interface) selon les données définies dans la configuration d’une carte MIDI.

</dd> <dt>

<span id="_win32_midi_program_change_message_gloss"></span><span id="_WIN32_MIDI_PROGRAM_CHANGE_MESSAGE_GLOSS"></span>**Programme midi-modifier le message**
</dt> <dd>

Message MIDI (Musical Instrument Digital Interface) envoyé à un synthétiseur pour modifier le correctif sur un canal MIDI spécifique.

</dd> <dt>

<span id="_win32_midi_sequence_gloss"></span><span id="_WIN32_MIDI_SEQUENCE_GLOSS"></span>**Séquence MIDI**
</dt> <dd>

Données MIDI (Musical Instrument Digital Interface) qui peuvent être lues par un séquenceur MIDI.

</dd> <dt>

<span id="_win32_midi_sequencer_gloss"></span><span id="_WIN32_MIDI_SEQUENCER_GLOSS"></span>**Séquenceur MIDI**
</dt> <dd>

Programme qui crée ou lit des chansons stockées sous forme de fichiers MIDI (Musical Instrument Digital Interface). Lorsqu’un Sequencer lit des fichiers MIDI, il envoie les données MIDI du fichier à un synthétiseur MIDI, qui produit les sons. Windows fournit un séquenceur midi, accessible via MCI (media control interface), qui lit les fichiers midi. Voir aussi [MCI (Media Control Interface)](/windows).

</dd> <dt>

<span id="_win32_midi_setup_map_gloss"></span><span id="_WIN32_MIDI_SETUP_MAP_GLOSS"></span>**Carte d’installation midi**
</dt> <dd>

Ensemble complet de données pour le Mappeur MIDI (Musical Instrument Digital Interface) à utiliser lors de la redirection des messages MIDI. Une seule carte d’installation peut être appliquée à un moment donné, mais l’utilisateur peut avoir plusieurs cartes de configuration disponibles et peut choisir entre elles à l’aide de l’option du panneau de configuration du mappeur MIDI.

</dd> <dt>

<span id="_win32_mma_gloss"></span><span id="_WIN32_MMA_GLOSS"></span>**MMA**
</dt> <dd>

Consultez l' [Association midi Manufacturers](/windows).

</dd> <dt>

<span id="_win32_msf_time_format_gloss"></span><span id="_WIN32_MSF_TIME_FORMAT_GLOSS"></span>**Format d’heure MSF**
</dt> <dd>

Format d’heure utilisé par l’interface MCI pour exprimer le temps en minutes, secondes et frames. Le nombre de frames dans une seconde dépend du type d’appareil utilisé ; les périphériques audio à disque compact utilisent 75 images par seconde. Le format d’heure MSF est principalement utilisé par les périphériques audio de disque compact.

</dd> <dt>

<span id="_win32_musical_instrument_digital_interface_midi_gloss"></span><span id="_WIN32_MUSICAL_INSTRUMENT_DIGITAL_INTERFACE_MIDI_GLOSS"></span>**Interface MIDI (Musical Instrument Digital Interface)**
</dt> <dd>

Protocole standard pour la communication entre les instruments musicaux et les ordinateurs.

</dd> <dt>

<span id="_win32_parts_per_quarter_note_ppqn_gloss"></span><span id="_WIN32_PARTS_PER_QUARTER_NOTE_PPQN_GLOSS"></span>**Remarque relative aux parties par trimestre (PPQN)**
</dt> <dd>

Format d’heure utilisé pour les séquences MIDI (Musical Instrument Digital Interface). PPQN est le format d’heure le plus courant utilisé avec les fichiers MIDI standard.

</dd> <dt>

<span id="_win32_patch_gloss"></span><span id="_WIN32_PATCH_GLOSS"></span>**correctif logiciel**
</dt> <dd>

Configuration particulière d’un synthétiseur MIDI (Musical Instrument Digital Interface) qui produit un son particulier, généralement un son simulant un instrument musical spécifique. Les correctifs sont également appelés programmes. Un message MIDI de modification de programme modifie le paramètre de correctif dans un synthétiseur. Patch fait également référence à la connexion ou aux connexions entre les périphériques MIDI. Voir aussi [midi (Musical Instrument Digital Interface)](/windows).

</dd> <dt>

<span id="_win32_patch_caching_gloss"></span><span id="_WIN32_PATCH_CACHING_GLOSS"></span>**mise en cache des correctifs**
</dt> <dd>

Technique permettant à certains pilotes de périphérique de synthétiseur MIDI (Musical Instrument Digital Interface) internes de précharger leurs données de correctifs, ce qui réduit le délai entre le moment où le synthétiseur reçoit un message MIDI de modification de programme et le moment où il lit une note en utilisant le nouveau correctif. La mise en cache des correctifs garantit également que les correctifs requis sont disponibles (le synthétiseur peut charger uniquement un sous-ensemble de ses correctifs). Voir aussi [midi (Musical Instrument Digital Interface)](/windows), [correctif](/windows).

</dd> <dt>

<span id="_win32_pitch_scale_factor_gloss"></span><span id="_WIN32_PITCH_SCALE_FACTOR_GLOSS"></span>**facteur d’échelle** de la hauteur
</dt> <dd>

Dans le son de forme d’onde, quantité par laquelle un pilote audio de forme d’onde met à l’échelle le pas. Un facteur d’échelle de deux entraîne une augmentation de la largeur d’un octave. La mise à l’échelle tonale nécessite un matériel spécialisé. La vitesse de lecture et le taux d’échantillonnage ne sont pas modifiés.

</dd> <dt>

<span id="_win32_playback_rate_scale_factor_gloss"></span><span id="_WIN32_PLAYBACK_RATE_SCALE_FACTOR_GLOSS"></span>**facteur d’échelle de la vitesse de lecture**
</dt> <dd>

Dans la forme d’onde audio, quantité par laquelle le pilote audio de la forme d’onde met à l’échelle la vitesse de lecture. La mise à l’échelle de la lecture est effectuée via un logiciel ; le taux d’échantillonnage n’est pas modifié, mais le pilote interpole en ignorant ou en synthétisant les exemples. Par exemple, si la vitesse de lecture est modifiée par un facteur de deux, le pilote ignore chaque autre échantillon.

</dd> <dt>

<span id="_win32_polyphony_gloss"></span><span id="_WIN32_POLYPHONY_GLOSS"></span>**polyphonie**
</dt> <dd>

Nombre maximal de notes qu’un périphérique de sortie MIDI (Musical Instrument Digital Interface) peut lire simultanément.

</dd> <dt>

<span id="_win32_ppqn_gloss"></span><span id="_WIN32_PPQN_GLOSS"></span>**PPQN**
</dt> <dd>

Voir la [Remarque relative aux parties par trimestre](/windows).

</dd> <dt>

<span id="_win32_preimaging_gloss"></span><span id="_WIN32_PREIMAGING_GLOSS"></span>**préimagerie**
</dt> <dd>

Processus de création d’un cadre de film dans une mémoire tampon avant son affichage.

</dd> <dt>

<span id="_win32_red_book_audio_gloss"></span><span id="_WIN32_RED_BOOK_AUDIO_GLOSS"></span>**Red Book audio**
</dt> <dd>

Voir [disque compact-audio numérique (CD-DA)](/windows).

</dd> <dt>

<span id="_win32_resource_interchange_file_format_riff_gloss"></span><span id="_WIN32_RESOURCE_INTERCHANGE_FILE_FORMAT_RIFF_GLOSS"></span>**RIFF (Resource Interchange File Format)**
</dt> <dd>

Spécification de fichier balisé utilisée pour définir les formats standard des fichiers multimédias. La structure de fichiers avec balises permet d’éviter des problèmes de compatibilité qui se produisent souvent lorsque les définitions de format de fichier changent au fil du temps. Étant donné que chaque élément de données du fichier est identifié par un en-tête standard, une application qui ne reconnaît pas un élément de données donné peut ignorer les informations inconnues. Voir aussi [format de fichier balisé](/windows).

</dd> <dt>

<span id="_win32_riff_gloss"></span><span id="_WIN32_RIFF_GLOSS"></span>**Riff**
</dt> <dd>

Voir [format RIFF (Resource Interchange File Format)](/windows).

</dd> <dt>

<span id="_win32_riff_chunk_gloss"></span><span id="_WIN32_RIFF_CHUNK_GLOSS"></span>**Bloc riff**
</dt> <dd>

Segment avec l’identificateur de segment [format d’échange de fichiers (Riff)](/windows) qui comprend un code d’identification et zéro, un ou plusieurs sous-segments, dont le contenu dépend du type de formulaire.

</dd> <dt>

<span id="_win32_riff_file_gloss"></span><span id="_WIN32_RIFF_FILE_GLOSS"></span>**Fichier riff**
</dt> <dd>

Fichier dont le format est conforme à l’un des formulaires [RIFF (Resource Interchange File Format)](/windows) publiés. Exemples de fichiers RIFF : fichiers WAVE pour les données audio Waveform, fichiers RMID pour les séquences MIDI (Musical Instrument Digital Interface) et fichiers RDIB pour les bitmaps indépendantes du périphérique.

</dd> <dt>

<span id="_win32_riff_form_gloss"></span><span id="_WIN32_RIFF_FORM_GLOSS"></span>**Formulaire riff**
</dt> <dd>

Spécification de format de fichier basée sur la norme [RIFF (Resource Interchange File Format)](/windows) .

</dd> <dt>

<span id="_win32_sample_gloss"></span><span id="_WIN32_SAMPLE_GLOSS"></span>**exemple**
</dt> <dd>

Élément discret de données de forme d’onde représenté par une valeur numérique unique. L’échantillonnage est le processus de conversion de données analogiques en données numériques en effectuant des échantillons de la forme d’onde analogique à des intervalles réguliers.

</dd> <dt>

<span id="_win32_sampling_rate_gloss"></span><span id="_WIN32_SAMPLING_RATE_GLOSS"></span>**taux d’échantillonnage**
</dt> <dd>

Vitesse à laquelle un pilote audio de forme d’onde effectue une conversion audio-numérique ou numérique-à-audio. Pour les disques compacts (CD-DA), le taux d’échantillonnage est de 44,1 kHz. Voir aussi [disque compact-audio numérique](/windows)

</dd> <dt>

<span id="_win32_sequence_gloss"></span><span id="_WIN32_SEQUENCE_GLOSS"></span>**séquence**
</dt> <dd>

Consultez [séquence MIDI](/windows).

</dd> <dt>

<span id="_win32_sequencer_gloss"></span><span id="_WIN32_SEQUENCER_GLOSS"></span>**séquenceur**
</dt> <dd>

Consultez [séquenceur MIDI](/windows).

</dd> <dt>

<span id="_win32_simple_device_gloss"></span><span id="_WIN32_SIMPLE_DEVICE_GLOSS"></span>**appareil simple**
</dt> <dd>

Périphérique MCI (Media Control Interface) qui ne requiert pas d’élément d’appareil (fichier de données) pour la lecture. Le pilote audio de disque compact MCI est un exemple de périphérique simple.

</dd> <dt>

<span id="_win32_smpte_gloss"></span><span id="_WIN32_SMPTE_GLOSS"></span>**SMPTE**
</dt> <dd>

Consultez la [société de l’image motion et les ingénieurs de la télévision](/windows).

</dd> <dt>

<span id="_win32_smpte_division_type_gloss"></span><span id="_WIN32_SMPTE_DIVISION_TYPE_GLOSS"></span>**Type de division SMPTE**
</dt> <dd>

L’un des quatre formats de synchronisation SMPTE. L’heure SMPTE est exprimée en heures, minutes, secondes et frames. Le type de division SMPTE spécifie la valeur d’images par seconde correspondant à une heure SMPTE donnée. Par exemple, une heure SMPTE d’une heure, 30 minutes, 24 secondes et 15 images est utile uniquement si la valeur frames par seconde, ou le type de division SMPTE, est connue.

</dd> <dt>

<span id="_win32_smpte_offset_gloss"></span><span id="_WIN32_SMPTE_OFFSET_GLOSS"></span>**Décalage SMPTE**
</dt> <dd>

Événement MIDI (Musical Instrument Digital Interface) qui désigne l’heure SMPTE à laquelle la lecture d’un fichier MIDI doit démarrer. Les décalages SMPTE sont utilisés uniquement avec les fichiers MIDI à l’aide du type de division SMPTE.

</dd> <dt>

<span id="_win32_smpte_time_gloss"></span><span id="_WIN32_SMPTE_TIME_GLOSS"></span>**Heure SMPTE**
</dt> <dd>

Représentation standard du temps développé pour les industries vidéo et cinématographiques. L’heure SMPTE est utilisée avec l’audio MIDI (Musical Instrument Digital Interface), car de nombreuses personnes utilisent MIDI pour noter les films et les vidéos. L’heure SMPTE est un format d’heure absolu exprimé en heures, minutes, secondes et frames. Les types de divisions SMPTE standard sont 24, 25 et 30 images par seconde.

<span id="_win32_society_of_motion_picture_and_television_engineers_smpte_gloss"></span><span id="_WIN32_SOCIETY_OF_MOTION_PICTURE_AND_TELEVISION_ENGINEERS_SMPTE_GLOSS"></span>**Société des ingénieurs de la télévision et des images animées**

Association d’ingénieurs impliqués dans la vidéo, la télévision et la production vidéo. SMPTE fait également référence à l’heure SMPTE, la norme de minutage adoptée par ce groupe.

</dd> <dt>

<span id="_win32_system_exclusive_data_gloss"></span><span id="_WIN32_SYSTEM_EXCLUSIVE_DATA_GLOSS"></span>**données système exclusives**
