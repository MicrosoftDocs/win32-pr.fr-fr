# <a name="digital-video-errors"></a>Erreurs de Digital-Video

Les valeurs de retour supplémentaires suivantes sont définies pour les périphériques vidéo numériques :



| Valeur                           | Signification                                                         |
|---------------------------------|-----------------------------------------------------------------|
| MCIAVI \_ ProductName             | Vidéo                                                           |
| MCIERR \_ AVI \_ AUDIOERROR         | Erreur inconnue lors de la tentative de lecture de l’audio.                   |
| MCIERR \_ AVI \_ BADPALETTE         | Impossible de basculer vers la nouvelle palette.                                |
| MCIERR \_ AVI \_ CANTPLAYFULLSCREEN | Ce fichier AVI ne peut pas être lu en mode plein écran.             |
| MCIERR \_ AVI \_ DISPLAYERROR       | Erreur inconnue lors de la tentative d’affichage de la vidéo.                |
| MCIERR \_ AVI \_ nocompresseur       | Impossible de localiser le compresseur d’installation nécessaire pour lire ce fichier.   |
| MCIERR \_ AVI \_ NODISPDIB          | mode VGA couleur 256 non disponible.                               |
| MCIERR \_ AVI \_ NOTINTERLEAVED     | Ce fichier AVI n’est pas entrelacé.                               |
| MCIERR \_ AVI \_ OLDAVIFORMAT       | Le format de ce fichier AVI est obsolète.                         |
| MCIERR \_ AVI \_ TOOBIGFORVGA       | Ce fichier AVI est trop volumineux pour être lu en mode VGA sélectionné. |
