# <a name="midi-output-data-types"></a>Types de données de sortie MIDI

Windows définit les types de données suivants pour les fonctions de sortie MIDI.



| Valeur                              | Signification                                                                              |
|------------------------------------|--------------------------------------------------------------------------------------|
| **HMIDIOUT**                       | Handle d’un appareil de sortie MIDI.                                                      |
| [**MIDIHDR**](/windows/win32/api/mmeapi/ns-mmeapi-midihdr)         | En-tête pour un bloc de données de flux exclusives du système MIDI ou.                          |
| [**MIDIOUTCAPS**](/windows/win32/api/mmeapi/ns-mmeapi-midioutcaps) | Structure utilisée pour se renseigner sur les fonctionnalités d’un appareil de sortie MIDI particulier. |
