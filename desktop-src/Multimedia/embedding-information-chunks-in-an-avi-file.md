# <a name="embedding-information-chunks-in-an-avi-file"></a>Incorporation de blocs d’informations dans un fichier AVI

Vous pouvez insérer des blocs d’informations, tels que du texte ou des données personnalisées, dans un fichier AVI à l’aide du message [](/windows/desktop/api/Vfw/nf-vfw-capfilesetinfochunk) INFOCHUNK (ou de la macro capFileSetInfoChunk) du [**\_ \_ fichier \_ \_ WM Cap**](wm-cap-file-set-infochunk.md) . Vous pouvez également utiliser ce message pour effacer des blocs d’informations d’un fichier AVI.
