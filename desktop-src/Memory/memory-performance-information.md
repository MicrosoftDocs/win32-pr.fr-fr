# <a name="memory-performance-information"></a>Informations sur les performances de la mémoire

Les informations sur les performances de la mémoire sont disponibles à partir du gestionnaire de mémoire via les [compteurs de performances](../perfctrs/performance-counters-portal.md) système et via des fonctions telles que [**GetPerformanceInfo**](/windows/win32/api/psapi/nf-psapi-getperformanceinfo), [**GetProcessMemoryInfo**](/windows/win32/api/psapi/nf-psapi-getprocessmemoryinfo)et [**GlobalMemoryStatusEx**](/windows/win32/api/sysinfoapi/nf-sysinfoapi-globalmemorystatusex). les Applications telles que le gestionnaire des tâches Windows, le moniteur de fiabilité et de performances, et l’outil [explorateur de processus](/sysinternals/downloads/process-explorer) utilisent des compteurs de performances pour afficher les informations de mémoire pour le système et pour les processus individuels.

cette rubrique associe les compteurs de performances aux données retournées par les fonctions de performances de la mémoire et le gestionnaire des tâches Windows :

-   [Informations sur les performances de la mémoire système](#system-memory-performance-information)
-   [Informations sur les performances de la mémoire du processus](#process-memory-performance-information)
-   [Rubriques connexes](#related-topics)
