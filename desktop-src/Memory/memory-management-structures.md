# <a name="memory-management-structures"></a>Structures de gestion de la mémoire

Les structures suivantes sont utilisées avec la gestion de la mémoire.

## <a name="in-this-section"></a>Contenu de cette section

| Rubrique | Description |
|-|-|
| [**\_ \_ informations sur la cible d’appel cfg \_**](-cfg-call-target-info.md) | Représente des informations sur les cibles d’appel pour la protection du workflow de contrôle (CFG). |
| [**\_ \_ informations sur les ressources d’optimisation du tas \_**](/windows/desktop/api/winnt/ns-winnt-heap_optimize_resources_information) | Spécifie des indicateurs pour une opération HeapOptimizeResources lancée avec [**HeapSetInformation**](/windows/desktop/api/HeapApi/nf-heapapi-heapsetinformation). |
| [**\_ \_ exigences relatives à l’adresse mémoire**](/windows/desktop/api/winnt/ns-winnt-mem_address_requirements) | Spécifie l’adresse de base et l’alignement les plus bas et les plus hauts dans le cadre d’un paramètre étendu à une fonction qui gère la mémoire virtuelle. |
| [**MEM \_ ( \_ paramètre étendu)**](/windows/desktop/api/winnt/ns-winnt-mem_extended_parameter) | Représente un paramètre étendu pour une fonction qui gère la mémoire virtuelle. |
| [**\_informations de base sur la mémoire \_**](/windows/desktop/api/winnt/ns-winnt-memory_basic_information) | Contient des informations sur une plage de pages dans l’espace d’adressage virtuel d’un processus. |
| [**MEMORYSTATUS**](/windows/desktop/api/WinBase/ns-winbase-memorystatus) | Contient des informations sur l’état actuel de la mémoire physique et de la mémoire virtuelle. |
| [**MEMORYSTATUSEX**](/windows/desktop/api/sysinfoapi/ns-sysinfoapi-memorystatusex) | Contient des informations sur l’état actuel de la mémoire physique et de la mémoire virtuelle, y compris la mémoire étendue. |
| [**\_entrée du tas de processus \_**](/windows/desktop/api/minwinbase/ns-minwinbase-process_heap_entry) | Contient des informations sur un élément de segment de mémoire. |
| [**Entrée de la \_ plage de mémoire Win32 \_ \_**](/windows/desktop/api/memoryapi/ns-memoryapi-win32_memory_range_entry) | Spécifie une plage de mémoire. |
| [**\_ \_ Informations sur la région de mémoire Win32 \_**](/windows/desktop/api/memoryapi/ns-memoryapi-win32_memory_region_information) | Contient des informations sur une région de la mémoire. |
| [**AtlThunkData \_ t**](/previous-versions/windows/desktop/legacy/mt805050(v=vs.85)) | Structure de données opaque qui représente un thunk ATL. |
