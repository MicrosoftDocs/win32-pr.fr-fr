# <a name="trusted-execution"></a>Exécution approuvée

Les enclaves sont utilisées pour créer des environnements d’exécution approuvés. Une enclave est une région isolée de code et des données dans l’espace d’adressage d’une application. Seul le code qui s’exécute dans l’enclave peut accéder aux données dans la même enclave.
