# <a name="packcpp"></a>Presse. COTISATIONS

Dans l’exemple de composant fournisseur, un exemple de code de compression et de décompression des types de données dans des VARIANTs est disponible dans Pack. cpp. Les fonctions suivantes sont utilisées lorsque les propriétés sont chargées dans le cache de propriétés et en sont supprimées :

-   **PackStringinVariant**
-   **UnpackStringinVariant**
-   **PackLONGinVariant**
-   **UnpackLONGfromVariant**
-   **PackDATEinVariant**
-   **UnpackDATEinVariant**
-   **PackVARIANT \_ BOOLinVariant**
-   **UnpackVARIANT \_ BOOLfromVariant**
-   **PackVARIANTinVariant**
-   **UnpackVARIANTfromVariant**
