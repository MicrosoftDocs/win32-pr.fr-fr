# <a name="propertycpp"></a>Propriété. COTISATIONS

Dans l’exemple de composant de fournisseur, des exemples de code pour le stockage de différents types de données dans des types de données **Variant** afin qu’ils puissent être enregistrés en tant que propriétés d’un objet se trouvent dans Property. cpp. Les routines prises en charge sont répertoriées dans le tableau suivant.



| Routine                                                           | Description                  |
|-------------------------------------------------------------------|------------------------------|
| en **savoir \_ plus Propriété \_ BSTR** / **put \_ BSTR, \_ propriété**                   | Pour les propriétés de type chaîne.       |
| en **savoir \_ plus Propriété \_ long** / **put \_ long \_** Property                   | Pour les propriétés de type entier long. |
| en **savoir \_ plus Propriété \_ Date** / **put \_ \_** Property                   | Pour les propriétés de format de date.  |
| en **savoir \_ plus Propriété \_ Variant \_ bool** / **put \_ Variant \_ bool \_ Property** | Pour les propriétés booléennes.      |
| en **savoir \_ plus Propriété \_ Variant** / **put \_ Variant \_ Property**             | Pour les propriétés de **type Variant** .  |
