# <a name="dialect"></a>Dialect

Deux des interfaces ADSI pour la recherche d’un répertoire, ADO et OLE DB, utilisent une chaîne de recherche pour spécifier les paramètres de la recherche. Les paramètres de recherche incluent la base de la recherche, le filtre de recherche, les attributs à récupérer et la profondeur de la recherche.

Vous pouvez spécifier la chaîne recherchée dans l’un des deux dialectes :

-   [Dialecte LDAP](ldap-dialect.md)
-   [Dialecte SQL](sql-dialect.md)
