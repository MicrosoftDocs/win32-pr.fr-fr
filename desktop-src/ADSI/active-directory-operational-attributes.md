# <a name="active-directory-operational-attributes"></a>Active Directory attributs opérationnels

Les attributs opérationnels sont des propriétés stockées dans et utilisées par l’annuaire à des fins d’administration. Par exemple, vous pouvez conserver un attribut opérationnel pour enregistrer la date et l’heure de modification d’un autre attribut. Les attributs opérationnels doivent être récupérés explicitement à l’aide d’un appel à [**GetInfoEx**](/windows/desktop/api/Iads/nf-iads-iads-getinfoex).
