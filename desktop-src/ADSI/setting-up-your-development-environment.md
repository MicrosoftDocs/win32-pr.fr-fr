# <a name="setting-up-your-development-environment"></a>Configuration de votre environnement de développement

Vous pouvez accéder à l’interface ADSI à partir de nombreuses plateformes de développement différentes, telles que les scripts, ASP, les systèmes de développement Visual Basic ou Visual C++ Microsoft, ou tout autre langage compatible avec Automation.

Pour plus d’informations sur la configuration d’un environnement de développement et la création d’applications ADSI de base, consultez :

-   [Prise en main avec l’écriture de scripts pour ADSI](getting-started-with-scripting-for-adsi.md)
-   [Prise en main avec ASP pour ADSI](getting-started-with-asp-for-adsi.md)
-   [Configuration de Visual Basic 6,0 pour le développement ADSI](setting-up-visual-basic-for-adsi-development.md)
-   [Configuration de Visual C++ 6,0 pour le développement ADSI](setting-up-c---for-adsi-development.md)
