# <a name="active-directory-service-interfaces-provider"></a>Fournisseur d’interfaces de service Active Directory

Un fournisseur ADSI contient l’implémentation des objets ADSI et des objets dépendants pour un espace de noms particulier. L’illustration suivante montre que les clients s’intéressent uniquement à l’obtention et à l’utilisation d’interfaces sur un objet, et non aux détails de l’implémentation et de la façon dont le logiciel d’un objet est implémenté.
