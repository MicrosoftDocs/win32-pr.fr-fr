# <a name="generic-com-error-codes"></a>Codes d’erreur COM génériques

Les codes d’erreur COM génériques exprime l’état de l’opération tel qu’il est généré par tous les modules COM. Le tableau suivant contient des exemples de ces codes d’erreur sur n’importe quelle plateforme Win32.



| Code d’erreur     | Valeur hexadécimale  | Description              |
|----------------|------------|--------------------------|
| E \_ Abort       | 0x80004004 | Opération abandonnée.       |
| E \_ échec        | 0x80004005 | Erreur non spécifiée.       |
| E \_ NOinterface | 0x80004002 | Interface non prise en charge. |
| \_NOTIMPL E     | 0x80004001 | Non implémenté.         |
| \_pointeur E     | 0x80004003 | Pointeur non valide.         |
| E \_ inattendu  | 0x8000FFFF | Défaillance catastrophique.    |
