# <a name="accessing-active-directory-using-visual-basic"></a>Accès Active Directory à l’aide de Visual Basic

l’une des fonctionnalités les plus puissantes, qui est devenue disponible avec le système d’exploitation microsoft Windows 2000, est le service d’annuaire microsoft Active Directory. lorsque vous vous connectez à un domaine Windows 2000, Active Directory est mis en action ; pour Rechercher l’imprimante couleur la plus proche dans votre bâtiment, vous pouvez utiliser Active Directory. Il peut être utilisé pour une recherche de carnet d’adresses ou une recherche pour les utilisateurs qui travaillent dans le bâtiment 40. avec Active Directory, vous pouvez utiliser une carte à puce pour ouvrir une session sur un domaine Windows 2000. vous pouvez même joindre des données à partir de Microsoft SQL Server logiciel et Active Directory de base de données.

de nombreuses autres technologies microsoft, telles que microsoft Message Queue Server (MSMQ), COM (Store de classes), IPsec (Internet Protocol security), les objets de stratégie de groupe (gpo) et Exchange sont intégrées à Active Directory.

Cette section explique comment utiliser les interfaces de service Active Directory (ADSI) pour accéder à Active Directory. À l’aide d’un scénario pour une société fictive (Fabrikam Corporation), vous apprendrez à effectuer certaines tâches d’administration de base.

Au fur et à mesure que vous progressez dans ce scénario, les exemples de code de chaque section s’appuient sur eux-mêmes.

par souci de concision, tous les exemples de code sont écrits avec le système de développement Microsoft Visual Basic. pour plus d’informations sur la conversion C++, consultez [mappage d’ADSI Visual Basic du code au code C++](mapping-adsi-visual-basic-code-to-c-code.md).
