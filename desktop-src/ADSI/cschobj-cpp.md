| Méthode                                                   | Description                                    |
|----------------------------------------------------------|------------------------------------------------|
| **CSampleDSSchema :: CreateSchema**                        | Constructeur standard.                          |
| **CSampleDSSchema :: ~ CSampleDSSchema**                    | Destructeur standard.                           |
| Méthodes [**IADs**](/windows/desktop/api/Iads/nn-iads-iads) standard.                   |                                                |
| Méthodes [**IADsContainer**](/windows/desktop/api/Iads/nn-iads-iadscontainer) standard. |                                                |
| **CSampleDSSchema :: QueryInterface**                      | Vérifiez l’ID d’interface donné sur cet objet.  |
| **CSampleDSSchema::AllocateSchema**                      | Créez un objet de schéma et chargez ses données de type. |
