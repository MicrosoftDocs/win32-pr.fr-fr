# <a name="cnamespcpp"></a>CNAMESP. COTISATIONS

Dans l’exemple de composant fournisseur, le code qui gère la durée de vie de l’exemple d’objet espace de noms du composant fournisseur se trouve dans cnamesp. cpp. Les méthodes prises en charge sont répertoriées dans le tableau suivant.



| Méthode                                                   | Description                                            |
|----------------------------------------------------------|--------------------------------------------------------|
| **CSampleDSNamespace::CSampleDSNamespace**               | Constructeur standard.                                  |
| **CSampleDSNamespace :: ~ CSampleDSNamespace**              | Destructeur standard.                                   |
| Méthodes [**IADs**](/windows/desktop/api/Iads/nn-iads-iads) standard.                   |                                                        |
| Méthodes [**IADsContainer**](/windows/desktop/api/Iads/nn-iads-iadscontainer) standard. |                                                        |
| **CSampleDSNamespace::AllocateNamespaceObject**          | Créez un objet Namespace et chargez ses données de type.      |
| **CSampleDSNamespace::CreateNamespace**                  | Allouez, initialisez et validez un objet Namespace. |
| **CSampleDSNamespace :: QueryInterface**                   | Vérifiez l’IID donné sur cet objet.                   |
