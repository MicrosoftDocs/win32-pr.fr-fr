# <a name="winnt-schema"></a>Schéma Winnt

Le schéma WinNT définit le jeu d’objets ADSI et leur hiérarchie de classes pour un réseau Windows NT 4,0. Les [objets ADSI de Winnt](adsi-objects-of-winnt.md) incluent l’utilisateur, l’ordinateur, le domaine, le groupe, la ressource, la session, le service, etc. La section hiérarchie de la [classe d’objets Winnt](winnt-object-class-hierarchy.md) qui suit décrit les relations hiérarchiques entre ces objets.

Chaque objet implémente un ensemble d’interfaces ADSI. Toutefois, dans le cas d’une interface implémentée, toutes les propriétés et méthodes ne sont pas prises en charge. Les [Propriétés obligatoires et facultatives du schéma WinNT](winnt-schemaampaposs-mandatory-and-optional-properties.md) répertorient les propriétés prises en charge par les classes d’objets Winnt.
