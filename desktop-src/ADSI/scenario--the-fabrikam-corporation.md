# <a name="scenario-the-fabrikam-corporation"></a>Scénario : Fabrikam Corporation

l’entreprise Fabrikam a mis à niveau son domaine de Windows NT 4,0 vers Windows 2000. Pendant l’installation, Joe worden, l’administrateur réseau, a été invité à fournir un nom DNS de domaine. Jean a tapé « fabrikam.com ». Les utilisateurs, groupes et ordinateurs existants sont migrés vers Active Directory dans le nouveau domaine : fabrikam.com.
