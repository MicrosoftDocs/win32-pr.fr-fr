# <a name="binding-to-an-adsi-object"></a>Liaison à un objet ADSI

La connexion à un objet dans un service d’annuaire est connue sous le nom de liaison. La liaison à un objet ADSI est la première étape de la communication avec le système d’annuaire sous-jacent. Un objet doit être lié pour pouvoir naviguer dans son espace de noms, Rechercher des données, modifier des données ou emprunter l’identité d’un utilisateur. Il est également possible de fournir des caractéristiques de liaison supplémentaires, telles que le nom d’utilisateur, le mot de passe, le nom du serveur, les méthodes de chiffrement et l’authentification sécurisée. Les caractéristiques de liaison supplémentaires réelles qui peuvent être utilisées dépendent du fournisseur.
