# <a name="obsolete-adsi-functions"></a>Fonctions ADSI obsolètes

Les fonctions suivantes ont été exposées dans les versions antérieures du SDK ADSI, mais ne sont plus utilisées dans le kit de développement logiciel (SDK).

-   **ADsDecodeBinaryData**
-   **AdsFreeAdsValues**
-   **ADsFreeAllErrorRecords**
-   **AdsTypeToPropVariant**
-   **AssertADsMemLeaks**
-   **DumpMemoryTracker**
-   **InitADsMem**
-   **PropVariantToAdsType**
