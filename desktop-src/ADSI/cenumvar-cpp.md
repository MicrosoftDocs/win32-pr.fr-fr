|-------------------------------------------------|---------------------------------------------------------------------------------------|
| **CSampleDSEnumVariant::CSampleDSEnumVariant**  | Constructeur standard.                                                                 |

| **CSampleDSEnumVariant :: AddRef**                | Implémentation de [**IUnknown :: AddRef**](/windows/win32/api/unknwn/nf-unknwn-iunknown-addref) standard.                 |
| **CSampleDSEnumVariant :: Release**               | Implémentation de [**IUnknown :: Release**](/windows/win32/api/unknwn/nf-unknwn-iunknown-release) standard.               |
| **CSampleDSEnumVariant :: Skip**                  | Implémentation standard de **IEnumXXXX :: Skip** .                                          |
| **CSampleDSEnumVariant :: Reset**                 | Implémentation standard de **IEnumXXXX :: Reset** .                                         |
| **CSampleDSEnumVariant :: Clone**                 | Implémentation standard de **IEnumXXXX :: Clone** .                                         |
