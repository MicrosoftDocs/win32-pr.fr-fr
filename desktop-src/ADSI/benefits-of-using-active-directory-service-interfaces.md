# <a name="benefits-of-using-active-directory-service-interfaces"></a>Avantages de l’utilisation des interfaces de service Active Directory

ADSI offre de nombreux avantages aux administrateurs système, comme décrit dans le tableau suivant.



| Fonctionnalité                   | Avantage                                                                                                                                                                                                                                                                                                                             |
|---------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Ouvrir                      | Tout fournisseur d’annuaires peut implémenter un fournisseur ADSI ; les utilisateurs gagnent en liberté de choix dans les services d’annuaire sans sacrifier la facilité de gestion.                                                                                                                                                                                        |
| Indépendant DS            | Les applications administratives ne sont pas étroitement liées au service d’annuaire d’un fournisseur donné. Les applications d’administration et d’autres applications d’annuaire peuvent être développées sans avoir besoin de comprendre les API de répertoire spécifiques aux fournisseurs. La même application peut fonctionner sur plusieurs annuaires. Les coûts liés au temps de développement et au support sont réduits. |
| Prise en charge de plusieurs langues | Les objets ADSI offrent un accès facile aux services d’annuaire via le modèle d’objet de composant. Les applications COM peuvent être écrites dans des langages tels que Microsoft Visual Basic, Microsoft Visual Basic Scripting Edition (VBScript) et C/C++.                                                                                             |
| Modèle de programmation simple  | ADSI est constitué d’un petit ensemble d’interfaces facile à apprendre.                                                                                                                                                                                                                                                                          |
| Scriptable                | Tout langage compatible avec Automation (par exemple, Visual Basic, VBScript, Perl, REXX, etc.) peut être utilisé pour développer des applications de service d’annuaire. Les administrateurs et les développeurs peuvent utiliser les outils qu’ils connaissent déjà. La productivité est améliorée : les coûts liés au temps de développement et au support sont réduits.                               |
| Riche fonctionnellement         | Les éditeurs de logiciels indépendants et les utilisateurs finaux expérimentés peuvent développer des applications sérieuses utilisant les mêmes modèles ADSI que ceux utilisés pour les applications d’administration scriptées simples.                                                                                                                                                                         |
| Extensible                | Les fournisseurs d’annuaires, les éditeurs de logiciels indépendants et les utilisateurs finaux peuvent étendre ADSI avec de nouveaux objets et fonctions pour ajouter de la valeur ou répondre à des besoins uniques.                                                                                                                                                                                                          |
| Prise en charge OLE DB              | ADSI fournit une interface OLE DB pour que les programmeurs familiarisés avec la programmation de bases de données via OLE DB puissent rapidement être productifs.                                                                                                                                                                                                  |
