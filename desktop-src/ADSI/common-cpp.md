# <a name="commoncpp"></a>Classiques. COTISATIONS

Dans l’exemple de composant fournisseur, un exemple de code commun à tous les objets génériques se trouve dans Common. cpp. Les objets héritent de ces méthodes et propriétés par le biais des règles d’héritage C++. Ces méthodes incluent la génération de chaînes ADsPath, de méthodes de données de type et d’autres routines de maintenance.
