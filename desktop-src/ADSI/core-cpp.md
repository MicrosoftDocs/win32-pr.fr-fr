# <a name="corecpp"></a>Ebauche. COTISATIONS

Dans l’exemple de composant fournisseur, l’exemple de code, utilisé pour initialiser les propriétés de base d’un objet générique, est dans core. cpp. Les méthodes prises en charge sont répertoriées dans le tableau suivant.



| Méthode                                   | Description                                                                                              |
|------------------------------------------|----------------------------------------------------------------------------------------------------------|
| **CCoreADsObject::InitializeCoreObject** | Initialisez les propriétés des [**IAD**](/windows/desktop/api/Iads/nn-iads-iads) de base pour cet objet, ce qui facilite la création d’objets. |
| **CCoreADsObject::CCoreADsObject**       | Créateur standard.                                                                                        |
| **CCoreADsObject :: ~ CCoreADsObject**      | Destructeur standard.                                                                                     |
