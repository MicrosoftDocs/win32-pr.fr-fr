# <a name="provider-minimum-requirements"></a>Configuration minimale requise pour le fournisseur

Pour écrire un fournisseur ADSI, implémentez les interfaces du service Active Directory, comme décrit dans les sections suivantes.

-   L' [implémentation de base](core-implementation.md) répertorie brièvement les éléments logiciels qu’un fournisseur ADSI doit prendre en charge.
-   L' [implémentation facultative](optional-implementation.md) répertorie brièvement les éléments logiciels qu’un fournisseur ADSI est fortement encouragé à prendre en charge.
-   Les [propriétés personnalisées](custom-properties.md) sont les propriétés et interfaces exposées par chaque fournisseur pour assurer la programmabilité des fonctionnalités natives du service d’annuaire.
-   Les [interfaces doubles](dual-interfaces.md) décrivent la Convention ADSI pour fournir des interfaces qui sont accessibles à la fois à des contrôleurs compatibles Automation et non-Automation.
