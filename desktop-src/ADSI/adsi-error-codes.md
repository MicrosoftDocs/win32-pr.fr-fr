# <a name="adsi-error-codes"></a>Codes d’erreur ADSI

Tous les codes d’erreur ADSI sont retournés en tant que valeurs **HRESULT** com. Ils peuvent être regroupés dans les quatre types suivants :

-   [Codes d’erreur COM génériques](generic-com-error-codes.md)
-   [Codes d’erreur ADSI génériques](generic-adsi-error-codes.md)
-   [Codes d’erreur Win32 pour ADSI](win32-error-codes-for-adsi.md)
-   [Codes d’erreur LDAP pour ADSI](ldap-error-codes-for-adsi.md)

En outre, certaines interfaces fournissent des informations supplémentaires sur l’erreur, appelées messages d’erreur étendus ADSI, que vous pouvez obtenir à l’aide de la fonction [**ADsGetLastError**](/windows/desktop/api/Adshlp/nf-adshlp-adsgetlasterror) .

Enfin, l' [exemple de code](code-example-for-working-with-adsi-error-messages.md) C++ présenté dans cette section montre comment traiter les messages d’erreur ADSI.
