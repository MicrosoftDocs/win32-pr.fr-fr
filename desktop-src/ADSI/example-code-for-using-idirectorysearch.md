# <a name="example-code-for-using-idirectorysearch"></a>Exemple de code pour l’utilisation de IDirectorySearch

Le kit de développement logiciel (SDK) de plateforme inclut une rubrique d’exemple de code, DsSrch, qui fournit une application d’invite de commandes simple qui utilise l’interface [**IDirectorySearch**](/windows/desktop/api/Iads/nn-iads-idirectorysearch) pour interroger un service d’annuaire. L’exemple de code complet est inclus dans le kit de développement logiciel (SDK) de plate-forme dans le répertoire Samples DsSrch Samples \\ \\ \\ General NetDs \\ \\ .
