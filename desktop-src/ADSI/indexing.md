# <a name="indexing"></a>Indexation

Lorsque cela est possible, recherchez des attributs indexés ou au moins un attribut indexé. Comme en général avec les bases de données, la recherche sur un attribut indexé permet au serveur de calculer plus efficacement le jeu de résultats.
