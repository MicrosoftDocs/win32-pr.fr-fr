# <a name="ldap-user-object"></a>Objet utilisateur LDAP

Les rubriques suivantes sont traitées pour l’objet utilisateur LDAP :

-   [Mappage entre les propriétés IADsUser et les attributs Active Directory](mapping-between-iadsuser-properties-and-active-directory-attributes.md)
-   [Création de l’utilisateur avec le fournisseur LDAP ADSI](user-creation-with-the-adsi-ldap-provider.md)
-   [Définition et modification des mots de passe utilisateur à l’aide du fournisseur LDAP](setting-user-passwords-for-ldap-providers.md)
-   [Exemples de gestion des utilisateurs dans Active Directory](examples-of-user-management-in-active-directory.md)

En outre, l’objet utilisateur dans Active Directory contient beaucoup plus de propriétés que celles répertoriées dans la section [IADsUser](iadsuser-property-methods.md) . Étant donné que Active Directory schéma est extensible, les autres fournisseurs peuvent également ajouter des attributs supplémentaires à une classe.
