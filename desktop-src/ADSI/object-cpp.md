# <a name="objectcpp"></a>Dessin. COTISATIONS

Dans l’exemple de composant fournisseur, un exemple de code de listes d’objets, y compris l’utilisation d’un filtre, se trouve dans Object. cpp. Les méthodes prises en charge sont répertoriées dans le tableau suivant.



| Object                               | Description                                 |
|--------------------------------------|---------------------------------------------|
| **ObjectTypeList::ObjectTypeList**   | Créateur : créez une liste à l’aide du filtre.     |
| **ObjectTypeList :: ~ ObjectTypeList**  | Destructeur standard.                        |
| **ObjectTypeList::GetCurrentObject** | Récupérez l’objet actuel dans la liste.  |
| **ObjectTypeList :: suivant**             | Embossage de l’index.                             |
| **ObjectTypeList :: Reset**            | Réinitialisez l’index.                            |
| **IsValidFilter**                    | Vérifiez l’intégrité du filtre.          |
| **BuildDefaultObjectArray**          | Générez un tableau d’objets à l’aide du filtre. |
| **BuildObjectArray**                 | Générez un tableau d’objets.                      |
