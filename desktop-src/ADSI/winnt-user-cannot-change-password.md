# <a name="user-cannot-change-password-winnt-provider"></a>L’utilisateur ne peut pas changer de mot de passe (fournisseur WinNT)

La capacité d’un utilisateur à modifier son mot de passe est une autorisation qui peut être accordée ou refusée. Pour plus d’informations sur la lecture et la modification par programmation de cette autorisation à l’aide du fournisseur WinNT, consultez :

-   [La lecture de l’utilisateur ne peut pas changer le mot de passe (fournisseur WinNT)](reading-user-cannot-change-password-winnt-provider.md)
-   [La modification de l’utilisateur ne peut pas modifier le mot de passe (fournisseur WinNT)](modifying-user-cannot-change-password-winnt-provider.md)

L’attribut **userFlags** de l’objet User n’est disponible que lors de l’utilisation du fournisseur Winnt.

Il est également possible de lire et de modifier cette autorisation à l’aide du fournisseur LDAP. Pour plus d’informations, consultez l' [utilisateur ne peut pas changer de mot de passe (fournisseur LDAP)](user-cannot-change-password.md).
