# <a name="adsi-extensions"></a>Extensions ADSI

Une extension ADSI est un objet COM spécial qui permet à un développeur d’étendre un objet ADSI. Chaque extension est associée à une classe spécifiée dans le répertoire. Avec ce modèle d’extension, les développeurs peuvent ajouter des méthodes pour fournir une signification plus dynamique à un objet. Dans un modèle de programmation d’annuaire traditionnel, l’accès à un objet s’effectue en obtenant et en définissant les attributs de l’objet. À l’aide des extensions ADSI, vous pouvez ajouter des fonctionnalités à un objet d’annuaire.
