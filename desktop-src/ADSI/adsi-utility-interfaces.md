# <a name="adsi-utility-interfaces"></a>Interfaces de l’utilitaire ADSI

ADSI fournit plusieurs interfaces utilitaires pour les objets ADSI qui peuvent vous aider à accéder aux Active Directory et à les gérer. Ces interfaces sont les suivantes :

-   [**IADsDeleteOps**](iadsdeleteops-interface.md)
-   [**IADsObjectOptions**](iadsobjectoptions-interface.md)
-   [**IADsPathname**](iadspathname-interface.md)
-   [**IADsNameTranslate**](iadsnametranslate-interface.md)
