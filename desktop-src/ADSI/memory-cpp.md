# <a name="memorycpp"></a>Capacité. COTISATIONS

Dans l’exemple de composant fournisseur, un exemple de code qui montre l’allocation et la libération de mémoire se trouve dans Memory. cpp. Les routines prises en charge sont répertoriées dans le tableau suivant.



| Élément                | Description                                                           |
|---------------------|-----------------------------------------------------------------------|
| **AllocProvMem**    | Allouer la mémoire spécifiée.                                            |
| **FreeProvMem**     | Mémoire disponible indiquée.                                                |
| **ReallocProvMem**  | Allouez de la mémoire contiguë.                                           |
| **AllocProvStr**    | Allouez une chaîne LPWSTR.                                            |
| **FreeProvStr**     | Chaîne libre si elle n’est pas déjà libérée.                                     |
| **ReallocProvStr**  | Allouez de la mémoire contiguë.                                           |
| **ProvAllocString** | Vérifie la chaîne et le premier paramètre. Si c’est le cas, effectue l’allocation. |
