# <a name="windows-touch-glossary"></a>Glossaire Windows Touch

<dl> <dt>

<span id="wintouch.windows_touch_glossary_application_gesture"></span><span id="WINTOUCH.WINDOWS_TOUCH_GLOSSARY_APPLICATION_GESTURE"></span>**mouvement de l’application**
</dt> <dd>

Mouvement pris en charge par une application particulière.

</dd> <dt>

<span id="wintouch.windows_touch_glossary_digitizer"></span><span id="WINTOUCH.WINDOWS_TOUCH_GLOSSARY_DIGITIZER"></span>**numériseur**
</dt> <dd>

Périphérique matériel haute résolution qui reconnaît l’entrée à partir d’un stylet ou d’un doigt.

</dd> <dt>

<span id="wintouch.windows_touch_glossary_gesture"></span><span id="WINTOUCH.WINDOWS_TOUCH_GLOSSARY_GESTURE"></span>**mouvement**
</dt> <dd>

Mouvement d’un stylet ou d’un doigt sur un écran, que l’ordinateur interprète comme une commande.

</dd> <dt>

<span id="wintouch.windows_touch_glossary_manipulation"></span><span id="WINTOUCH.WINDOWS_TOUCH_GLOSSARY_MANIPULATION"></span>**manoeuvr**
</dt> <dd>

Action physique, telle que la rotation, appliquée à un objet.

</dd> <dt>

<span id="wintouch.windows_touch_glossary_multitouch"></span><span id="WINTOUCH.WINDOWS_TOUCH_GLOSSARY_MULTITOUCH"></span>**multipoint**
</dt> <dd>

Action physique en temps réel appliquée à un objet sur un écran dans lequel plusieurs points de contact sont utilisés simultanément.

</dd> <dt>

<span id="wintouch.windows_touch_glossary_system_gesture"></span><span id="WINTOUCH.WINDOWS_TOUCH_GLOSSARY_SYSTEM_GESTURE"></span>**mouvement système**
</dt> <dd>

L’équivalent du stylet d’un message de souris traditionnel, tel qu’un clic gauche ou un double-clic.

</dd> </dl>
