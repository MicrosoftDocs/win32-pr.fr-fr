# <a name="functions-windows-touch-gestures"></a>Fonctions (gestes tactiles Windows)

Cette section contient des fonctions pour les gestes tactiles Windows.

Les fonctions suivantes sont utilisées pour les gestes tactiles Windows.



| Fonction                                                 | Description                                                                                             |
|----------------------------------------------------------|---------------------------------------------------------------------------------------------------------|
| [**CloseGestureInfoHandle**](/windows/desktop/api/winuser/nf-winuser-closegestureinfohandle) | Ferme les ressources associées à un handle récupéré manuellement vers une structure d’informations de mouvement.    |
| [**GetGestureExtraArgs**](/windows/desktop/api/winuser/nf-winuser-getgestureextraargs)       | Récupère des informations supplémentaires sur un geste à partir de son handle [**GESTUREINFO**](/windows/win32/api/winuser/ns-winuser-gestureinfo) .    |
| [**GetGestureInfo**](/windows/desktop/api/winuser/nf-winuser-getgestureinfo)                 | Récupère manuellement une structure d’informations de mouvement en fonction d’un handle de la structure d’informations de mouvement. |
| [**GetGestureConfig**](/windows/desktop/api/winuser/nf-winuser-getgestureconfig)             | Récupère les informations de configuration de mouvement pour une fenêtre.                                           |
| [**SetGestureConfig**](/windows/desktop/api/winuser/nf-winuser-setgestureconfig)             | Définit la configuration du mouvement pour une fenêtre.                                                            |
