# <a name="properties-imanipulationprocessor"></a>Propriétés (IManipulationProcessor)

Cette section contient les propriétés de l’interface [**IManipulationProcessor**](/windows/desktop/api/manipulations/nn-manipulations-imanipulationprocessor) .

Les propriétés suivantes sont exposées par l’interface IManipulationProcessor.



| Propriété                                                                            | Description                                                                                                |
|-------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------|
| [**MinimumScaleRotateRadius**](imanipulationprocessor-minimumscalerotateradius.md) | Spécifie la taille des contacts à distance sur un mouvement de mise à l’échelle ou de rotation pour déclencher la manipulation. |
| [**PivotPointX**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-get_pivotpointx)                           | Cette propriété est le centre horizontal de l’objet.                                                      |
| [**PivotPointY**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-get_pivotpointy)                           | Cette propriété est le centre vertical de l’objet.                                                        |
| [**PivotRadius**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-get_pivotradius)                           | Cette propriété est utilisée pour déterminer la quantité de rotation utilisée dans la manipulation d’un seul doigt.                |
| [**SupportedManipulations**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-get_supportedmanipulations)     | Cette propriété est utilisée pour indiquer les manipulations prises en charge par un objet.                          |
