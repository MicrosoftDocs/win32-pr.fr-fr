# <a name="touch-input"></a>Entrée tactile

## <a name="purpose"></a>Objectif

Windows prend en charge les entrées tactiles via le modèle d’interaction exposé par la plateforme de bout en bout et l’infrastructure d’interface utilisateur de Windows Touch.

## <a name="developer-audience"></a>Développeurs concernés

Les fonctionnalités de Windows Touch sont conçues pour être utilisées par les développeurs C/C++ expérimentés qui connaissent bien le modèle COM (Component Object Model), les concepts de programmation de l’interface utilisateur et les concepts d’interaction tactile générale.

## <a name="run-time-requirements"></a>Conditions d’exécution

Windows 7
