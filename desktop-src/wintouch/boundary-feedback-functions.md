# <a name="functions-boundary-feedback"></a>Fonctions (commentaires sur les limites)

Cette section décrit les fonctions de commentaires sur les limites.



| Fonction                                               | Description                                                                                        |
|--------------------------------------------------------|----------------------------------------------------------------------------------------------------|
| [**BeginPanningFeedback**](/windows/win32/api/uxtheme/nf-uxtheme-beginpanningfeedback)     | Cette fonction est appelée par l’utilisateur avant d’utiliser la rétroaction de limite.                               |
| [**EndPanningFeedback**](/windows/win32/api/uxtheme/nf-uxtheme-endpanningfeedback)     | Cette fonction est appelée à la fin des commentaires sur la limite et déclenche éventuellement une animation. |
| [**UpdatePanningFeedback**](/windows/win32/api/uxtheme/nf-uxtheme-updatepanningfeedback) | Cette fonction est appelée pendant les commentaires de limite et peut éventuellement déclencher une animation.             |
