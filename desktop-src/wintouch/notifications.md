# <a name="notifications-touch-input"></a>Notifications (entrée tactile)

Cette section contient des notifications pour les gestes tactiles Windows.

Les notifications suivantes sont spécifiées dans cette section.



| Notification                                  | Description                                                     |
|-----------------------------------------------|-----------------------------------------------------------------|
| [**\_mouvement WM**](wm-gesture.md)             | Envoyé à une procédure de fenêtre pour passer des informations sur un geste. |
| [**\_GESTURENOTIFY WM**](wm-gesturenotify.md) | Vous donne la possibilité de définir la configuration du geste.            |
