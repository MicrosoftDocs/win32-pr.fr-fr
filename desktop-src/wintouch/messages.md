# <a name="messages-touch-input"></a>Messages (entrée tactile)

Cette section contient des messages pour une entrée tactile.

Les messages suivants sont utilisés pour l’entrée tactile multipoint.



| Message                           | Description                                                                                                         |
|-----------------------------------|---------------------------------------------------------------------------------------------------------------------|
| [**WM \_ Touch**](wm-touchdown.md) | Indique qu’un ou plusieurs points tactiles, tels qu’un doigt ou un stylet, ont touché une surface de digitaliseur tactile. |
