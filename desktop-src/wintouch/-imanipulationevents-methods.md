# <a name="methods-_imanipulationevents"></a>Méthodes (_IManipulationEvents)

Cette section contient les méthodes de l’interface [**\_ IManipulationEvents**](/windows/win32/api/manipulations/nn-manipulations-_imanipulationevents) .



| Méthode                                                                      | Description                                                   |
|-----------------------------------------------------------------------------|---------------------------------------------------------------|
| [**ManipulationStarted**](/windows/win32/api/manipulations/nf-manipulations-_imanipulationevents-manipulationstarted)     | Gère l’événement pour le début de la manipulation ou de l’inertie.    |
| [**ManipulationDelta**](/windows/win32/api/manipulations/nf-manipulations-_imanipulationevents-manipulationdelta)         | Gère les événements qui se produisent lorsqu’un objet manipulé change. |
| [**ManipulationCompleted**](/windows/win32/api/manipulations/nf-manipulations-_imanipulationevents-manipulationcompleted) | Gère l’événement lorsque la manipulation ou l’inertie se termine.      |
