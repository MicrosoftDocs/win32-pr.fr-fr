# <a name="structures-windows-touch-gestures"></a>Structures (gestes tactiles Windows)

Cette section contient des structures pour les gestes tactiles Windows.

Les structures suivantes sont spécifiées dans cette section.



| Structure                                          | Description                                                                              |
|----------------------------------------------------|------------------------------------------------------------------------------------------|
| [**GESTURECONFIG**](/windows/win32/api/winuser/ns-winuser-gestureconfig)             | Est utilisé pour définir la configuration dont les messages de mouvement sont activés ou désactivés.      |
| [**GESTUREINFO**](/windows/win32/api/winuser/ns-winuser-gestureinfo)                 | Contient des informations sur un geste.                                                    |
| [**GESTURENOTIFYSTRUCT**](/windows/win32/api/winuser/ns-winuser-gesturenotifystruct) | Contient des informations sur une notification [**WM \_ GESTURENOTIFY**](wm-gesturenotify.md) . |
