# <a name="about-windows-touch"></a><span data-ttu-id="7a989-108">À propos de Windows Touch</span><span class="sxs-lookup"><span data-stu-id="7a989-108">About Windows Touch</span></span>

<span data-ttu-id="7a989-109">Cette rubrique fournit une brève présentation de Windows Touch.</span><span class="sxs-lookup"><span data-stu-id="7a989-109">This topic gives a brief overview of Windows Touch.</span></span>

<span data-ttu-id="7a989-110">Les nouveaux éléments de matériel et d’API du système d’exploitation Windows 7 offrent aux applications la possibilité de recevoir des entrées de plusieurs contacts.</span><span class="sxs-lookup"><span data-stu-id="7a989-110">New hardware and API elements in the Windows 7 operating system provide applications the ability to receive input from multiple contacts.</span></span> <span data-ttu-id="7a989-111">Cela donne à ces applications la possibilité de détecter et de répondre à plusieurs points tactiles simultanés sur la surface visible de l’application.</span><span class="sxs-lookup"><span data-stu-id="7a989-111">This gives such applications the ability to detect and respond to multiple simultaneous touch points on the application's visible surface.</span></span> <span data-ttu-id="7a989-112">Les fonctionnalités de cette fonctionnalité dans Windows 7 sont fournies par un nouveau message qui signale et effectue le suivi des touches.</span><span class="sxs-lookup"><span data-stu-id="7a989-112">Functionality for this feature in Windows 7 is provided by a new message that reports and tracks touches.</span></span> <span data-ttu-id="7a989-113">Le nouveau message, [**WM \_ Touch**](wm-touchdown.md), signale l’action (haut, descendre, déplacer), la position et un identificateur pour les points tactiles.</span><span class="sxs-lookup"><span data-stu-id="7a989-113">The new message, [**WM\_TOUCH**](wm-touchdown.md), reports the action (up, down, move), position, and an identifier for touch points.</span></span> <span data-ttu-id="7a989-114">Les messages tactiles Windows sont générés par Windows et sont remis à Windows qui s’inscrivent à l’entrée tactile Windows.</span><span class="sxs-lookup"><span data-stu-id="7a989-114">Windows Touch messages are generated by Windows and are delivered to windows that register for Windows Touch input.</span></span>

<span data-ttu-id="7a989-115">En plus du nouveau message de saisie tactile, des messages de mouvement ont été ajoutés à la liste existante de messages de fenêtre.</span><span class="sxs-lookup"><span data-stu-id="7a989-115">In addition to the new touch input message, gesture messages have been added to the existing list of window messages.</span></span> <span data-ttu-id="7a989-116">La prise en charge de la messagerie pour les gestes est activée par un nouveau message de fenêtre ([**\_ mouvement WM**](wm-gesture.md)) qui est envoyé ou publié dans les fenêtres d’application appropriées lorsque l’entrée utilisateur est reconnue comme un geste.</span><span class="sxs-lookup"><span data-stu-id="7a989-116">Messaging support for gestures is enabled by a single new window message ([**WM\_GESTURE**](wm-gesture.md)) that is sent or posted to appropriate application windows when user input is recognized as a gesture.</span></span> <span data-ttu-id="7a989-117">Les fonctions d’API dédiées encapsulent les détails de la création et de la consommation de ce message.</span><span class="sxs-lookup"><span data-stu-id="7a989-117">Dedicated API functions encapsulate the details for creation and consumption of this message.</span></span> <span data-ttu-id="7a989-118">Cela est dû au fait que les informations associées au message peuvent changer à l’avenir sans interrompre les applications qui consomment déjà ce message.</span><span class="sxs-lookup"><span data-stu-id="7a989-118">This is done because the information associated with the message can change in the future without breaking applications that already consume this message.</span></span>

<span data-ttu-id="7a989-119">En plus des messages de mouvement, des interfaces spécialisées ont été ajoutées au SDK Windows.</span><span class="sxs-lookup"><span data-stu-id="7a989-119">In addition to gesture messages, specialized interfaces have been added to the Windows SDK.</span></span> <span data-ttu-id="7a989-120">Ces interfaces activent la prise en charge avancée de l’entrée tactile pour permettre aux développeurs d’applications de créer facilement des interfaces utilisateur naturelles.</span><span class="sxs-lookup"><span data-stu-id="7a989-120">These interfaces enable advanced support for touch input so that application developers can easily create natural user interfaces.</span></span> <span data-ttu-id="7a989-121">L’interface [**IManipulationProcessor**](/windows/desktop/api/manipulations/nn-manipulations-imanipulationprocessor) interprète les messages [**WM \_ Touch**](wm-touchdown.md) pour déclencher des événements qui contiennent des informations de translation, de rotation et de mise à l’échelle sur une collection de points tactiles.</span><span class="sxs-lookup"><span data-stu-id="7a989-121">The [**IManipulationProcessor**](/windows/desktop/api/manipulations/nn-manipulations-imanipulationprocessor) interface interprets [**WM\_TOUCH**](wm-touchdown.md) messages to raise events that contain translation, rotation, and scale information about a collection of touch points.</span></span> <span data-ttu-id="7a989-122">L’interface [**IInertiaProcessor**](/windows/desktop/api/manipulations/nn-manipulations-iinertiaprocessor) peut être utilisée conjointement avec l’interface **IManipulationProcessor** pour activer l’animation et garantir que les objets restent sur l’écran de l’utilisateur lorsqu’ils sont déplacés.</span><span class="sxs-lookup"><span data-stu-id="7a989-122">The [**IInertiaProcessor**](/windows/desktop/api/manipulations/nn-manipulations-iinertiaprocessor) interface can be used in conjunction with the **IManipulationProcessor** interface to enable animation and ensure that objects stay on the user's screen when they are moved.</span></span>

<span data-ttu-id="7a989-123">Les éléments d’API pour Windows Touch ont des similitudes avec le kit de développement logiciel (SDK) Microsoft PixelSense (anciennement connu sous le nom de Microsoft surface SDK), mais les applications ciblant Microsoft PixelSense ne s’exécutent pas sur les ordinateurs tactiles Windows.</span><span class="sxs-lookup"><span data-stu-id="7a989-123">API elements for Windows Touch have some similarities to the Microsoft PixelSense SDK (formerly known as the Microsoft Surface SDK), but applications targeting Microsoft PixelSense do not run on Windows Touch computers.</span></span> <span data-ttu-id="7a989-124">En outre, les applications ciblant Windows Touch ne s’exécutent pas sur Microsoft PixelSense.</span><span class="sxs-lookup"><span data-stu-id="7a989-124">Also, applications targeting Windows Touch do not run on Microsoft PixelSense.</span></span>

<span data-ttu-id="7a989-125">Certaines fonctionnalités de Windows Touch sont intégrées au cœur de Windows 7.</span><span class="sxs-lookup"><span data-stu-id="7a989-125">Some of the functionality of Windows Touch is built into the core of Windows 7.</span></span> <span data-ttu-id="7a989-126">Cette fonctionnalité est disponible pour les utilisateurs sans que les développeurs n’aient besoin d’activer explicitement la prise en charge.</span><span class="sxs-lookup"><span data-stu-id="7a989-126">This functionality is available to users without the need for developers to explicitly enable support.</span></span> <span data-ttu-id="7a989-127">Toutefois, pour tirer pleinement parti de Windows Touch, les développeurs doivent utiliser l’API tactile Windows.</span><span class="sxs-lookup"><span data-stu-id="7a989-127">However, to take full advantage of Windows Touch, developers must use the Windows Touch API.</span></span> <span data-ttu-id="7a989-128">Pour commencer à découvrir comment fonctionne Windows Touch, consultez le [Guide de programmation](programming-guide.md) ou commencez par [choisir la bonne approche pour Windows Touch](choosing-the-right-approach-to-windows-touch.md).</span><span class="sxs-lookup"><span data-stu-id="7a989-128">To get started with learning how Windows Touch works, see the [Programming Guide](programming-guide.md) or start with [Choosing the Right Approach to Windows Touch](choosing-the-right-approach-to-windows-touch.md).</span></span>
