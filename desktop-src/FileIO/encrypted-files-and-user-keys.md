# <a name="encrypted-files-and-user-keys"></a>Fichiers chiffrés et clés utilisateur

Pour créer une clé pour un utilisateur, utilisez la fonction [**SetUserFileEncryptionKey**](/windows/desktop/api/Winefs/nf-winefs-setuserfileencryptionkey) . Pour ajouter des clés d’utilisateur à un fichier chiffré, utilisez la fonction [**AddUsersToEncryptedFile**](/windows/desktop/api/Winefs/nf-winefs-adduserstoencryptedfile) . Pour interroger les clés utilisateur d’un fichier chiffré, utilisez la fonction [**QueryUsersOnEncryptedFile**](/windows/desktop/api/Winefs/nf-winefs-queryusersonencryptedfile) . Pour supprimer des clés d’utilisateur d’un fichier chiffré, utilisez la fonction [**RemoveUsersFromEncryptedFile**](/windows/desktop/api/Winefs/nf-winefs-removeusersfromencryptedfile) .
