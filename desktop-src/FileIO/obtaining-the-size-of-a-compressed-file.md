# <a name="obtaining-the-size-of-a-compressed-file"></a>Obtention de la taille d’un fichier compressé

Utilisez la fonction [**GetCompressedFileSize**](/windows/desktop/api/fileapi/nf-fileapi-getcompressedfilesizea) pour obtenir la taille compressée d’un fichier. Si le fichier est compressé, sa taille compressée est inférieure à sa taille non compressée. Utilisez la fonction [**GetFileSize**](/windows/desktop/api/FileAPI/nf-fileapi-getfilesize) pour déterminer la taille non compressée d’un fichier.
