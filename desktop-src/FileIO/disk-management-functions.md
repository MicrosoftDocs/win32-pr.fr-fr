# <a name="disk-management-functions"></a>Fonctions de gestion des disques

Les fonctions suivantes sont utilisées dans gestion des disques.

## <a name="in-this-section"></a>Dans cette section



| Fonction                                                    | Description                                                                                                                                                                                                                                                             |
|-------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**GetDiskFreeSpace**](/windows/desktop/api/FileAPI/nf-fileapi-getdiskfreespacea)<br/>     | Récupère des informations sur le disque spécifié, y compris la quantité d’espace libre sur le disque.<br/>                                                                                                                                                              |
| [**GetDiskFreeSpaceEx**](/windows/desktop/api/FileAPI/nf-fileapi-getdiskfreespaceexa)<br/> | Récupère des informations sur la quantité d’espace disponible sur un volume de disque, qui correspond à la quantité totale d’espace, à la quantité totale d’espace libre et à la quantité totale d’espace libre disponible pour l’utilisateur associé au thread appelant.<br/> |



 

Autres fonctions utilisées dans la gestion des disques.



| Fonction                         | Description                                                                                                                                                                                                                        |
|----------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**CreateFile**](/windows/desktop/api/FileAPI/nf-fileapi-createfilea) | Crée ou ouvre un fichier ou un périphérique d’e/s. Les périphériques d’e/s les plus couramment utilisés sont les suivants : fichier, flux de fichier, répertoire, disque physique, volume, mémoire tampon de la console, lecteur de bande, ressource de communication, mailslot et canal.<br/> |
| [**DeleteFile**](/windows/desktop/api/FileAPI/nf-fileapi-deletefilea) | Supprime un fichier existant.<br/>                                                                                                                                                                                               |
