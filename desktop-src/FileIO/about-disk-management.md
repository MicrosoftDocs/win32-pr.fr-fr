# <a name="about-disk-management"></a>À propos de la gestion des disques

Le système de fichiers fournit une abstraction des caractéristiques physiques des dispositifs de stockage, de sorte que les applications peuvent simplement écrire et lire des fichiers. Toutefois, le stockage dans une entreprise s’appuie fortement sur le concept des disques.
