# <a name="file-management-local-file-systems"></a>Gestion des fichiers (systèmes de fichiers locaux)

Un *objet fichier* fournit une représentation d’une ressource (un appareil physique ou une ressource située sur un appareil physique) qui peut être gérée par le système d’e/s. Comme les autres objets, ils permettent le partage de la ressource, ils ont des noms, ils sont protégés par la sécurité basée sur les objets et prennent en charge la synchronisation. Le système d’e/s permet également la lecture ou l’écriture dans la ressource.
