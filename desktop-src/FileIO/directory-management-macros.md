# <a name="directory-management-macros"></a>Macros de gestion d’annuaire

Les macros suivantes sont utilisées avec des points d’analyse.

## <a name="in-this-section"></a>Dans cette section



| Macro                                                                     | Description                                                                                                                             |
|---------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------|
| [**IsReparseTagMicrosoft**](/windows/desktop/api/Winnt/nf-winnt-isreparsetagmicrosoft)<br/>         | Détermine si une balise de point d’analyse indique un point d’analyse Microsoft.<br/>                                                  |
| [**IsReparseTagNameSurrogate**](/windows/desktop/api/Winnt/nf-winnt-isreparsetagnamesurrogate)<br/> | Détermine si le point d’analyse associé d’une balise est un substitut pour une autre entité nommée (par exemple, un dossier monté).<br/> |
