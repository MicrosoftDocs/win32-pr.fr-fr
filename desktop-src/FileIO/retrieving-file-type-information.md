# <a name="retrieving-file-type-information"></a>Récupération des informations de type de fichier

La fonction [**GetFileType**](/windows/desktop/api/FileAPI/nf-fileapi-getfiletype) récupère le type d’un fichier : disque, caractère (tel qu’une console), canal ou inconnu. La fonction [**GetBinaryType**](/windows/desktop/api/WinBase/nf-winbase-getbinarytypea) détermine si un fichier est exécutable et, le cas échéant, le type de fichier exécutable. Pour obtenir la liste des types exécutables pris en charge, consultez **GetBinaryType** .
