| [**capacité \_ \_ d’alimentation du périphérique de stockage IOCTL \_ \_**](/windows/desktop/api/WinIoctl/ni-winioctl-ioctl_storage_device_power_cap)<br/>                   | Les applications Windows peuvent utiliser ce code de contrôle pour spécifier un niveau de consommation énergétique maximal pour un périphérique de stockage. Le système d’exploitation permet de faire passer l’appareil à un état d’alimentation qui ne dépasse pas la valeur maximale spécifiée. Toutefois, cela dépend de ce que l’appareil prend en charge. La valeur maximale réelle peut être inférieure ou supérieure à la valeur maximale souhaitée.<br/> Pour effectuer cette opération, appelez la fonction [**DeviceIoControl**](/windows/desktop/api/ioapiset/nf-ioapiset-deviceiocontrol) avec les paramètres suivants.<br/>                        |
| [**\_ \_ activation du microprogramme de stockage IOCTL \_**](/windows/desktop/api/WinIoctl/ni-winioctl-ioctl_storage_firmware_activate)<br/>                  | Les applications Windows peuvent utiliser ce code de contrôle pour activer une image de microprogramme sur un appareil spécifié.<br/> Pour effectuer cette opération, appelez la fonction [**DeviceIoControl**](/windows/desktop/api/ioapiset/nf-ioapiset-deviceiocontrol) avec les paramètres suivants.<br/>                                                                                                                                                                                                                                                                                          |
| [**\_ \_ Téléchargement du microprogramme de stockage IOCTL \_**](/windows/desktop/api/WinIoctl/ni-winioctl-ioctl_storage_firmware_download)<br/>                  | Les applications Windows peuvent utiliser ce code de contrôle pour télécharger une image de microprogramme sur l’appareil cible, mais pas l’activer. Si la taille de l’image à télécharger est supérieure à la taille maximale du transfert de données du contrôleur, cette IOCTL devra être appelée plusieurs fois jusqu’à ce que la totalité de l’image soit téléchargée.<br/> Pour effectuer cette opération, appelez la fonction [**DeviceIoControl**](/windows/desktop/api/ioapiset/nf-ioapiset-deviceiocontrol) avec les paramètres suivants.<br/>                                                                                     |
| [**\_informations de \_ récupération du microprogramme de stockage IOCTL \_ \_**](/windows/desktop/api/WinIoctl/ni-winioctl-ioctl_storage_firmware_get_info)<br/>                 | Les applications Windows peuvent utiliser ce code de contrôle pour interroger le périphérique de stockage afin d’obtenir des informations détaillées sur le microprogramme. Un appel réussi renverra des informations sur les révisions du microprogramme, l’état de l’activité, ainsi que des attributs en lecture/écriture pour chaque emplacement. La quantité de données retournée varie selon le protocole de stockage.<br/> Pour effectuer cette opération, appelez la fonction [**DeviceIoControl**](/windows/desktop/api/ioapiset/nf-ioapiset-deviceiocontrol) avec les paramètres suivants.<br/>                                                                         |
| [**\_commande du \_ protocole de stockage IOCTL \_**](/windows/desktop/api/winioctl/ni-winioctl-ioctl_storage_protocol_command)<br/>                    | Les applications Windows peuvent utiliser ce code de contrôle pour retourner les propriétés d’un périphérique de stockage ou d’un adaptateur. La demande indique le type d’informations à récupérer, par exemple les données de recherche pour un appareil ou les capacités et les limites d’un adaptateur. <br/>                                                                                                                                                                                                                                                                               |
| [**\_propriété de \_ requête de stockage IOCTL \_**](/windows/desktop/api/WinIoCtl/ni-winioctl-ioctl_storage_query_property)<br/>                        | Les applications Windows peuvent utiliser ce code de contrôle pour retourner les propriétés d’un périphérique de stockage ou d’un adaptateur. La demande indique le type d’informations à récupérer, par exemple les données de recherche d’un appareil ou les capacités et les limites d’un adaptateur. [**IOCTL \_ La \_ \_ propriété de requête de stockage**](/windows/desktop/api/WinIoCtl/ni-winioctl-ioctl_storage_query_property) peut également être utilisée pour déterminer si le pilote de port prend en charge une propriété particulière ou quels champs du descripteur de propriété peuvent être modifiés avec une demande de modification de propriété ultérieure.<br/> |
| [**\_seuil de \_ température du jeu de stockage IOCTL \_ \_**](/windows/desktop/api/WinIoctl/ni-winioctl-ioctl_storage_set_temperature_threshold)<br/> | Les applications Windows peuvent utiliser ce code de contrôle pour définir le seuil de température d’un appareil (lorsqu’il est pris en charge par l’appareil). <br/>                                                                                                                                                                                                                                                                                                                                                                                              |
