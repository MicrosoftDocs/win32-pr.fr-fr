# <a name="volume-management-control-codes"></a>Codes de contrôle de gestion des volumes

Codes de contrôle utilisés dans la gestion des volumes.

## <a name="in-this-section"></a>Contenu de cette section



| Rubrique                                                                                                                                      | Description                                                                                                                                                                                                                                                               |
|--------------------------------------------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**FSCTL \_ créer \_ un \_ journal USN**](/windows/win32/api/winioctl/ni-winioctl-fsctl_create_usn_journal)<br/>                                                                 | Crée un flux de journal de modification de numéro de séquence de mise à jour (USN) sur un volume cible ou modifie un flux de journal de modifications existant.<br/>                                                                                                                               |
| [**\_caractéristiques du \_ \_ système de \_ \_ fichiers de \_ niveau \_ de requête FSCTL CSV**](/windows/win32/api/winioctl/ni-winioctl-fsctl_csv_query_down_level_file_system_characteristics)<br/> | Récupère des informations sur un système de fichiers pour lequel CSVFS est un proxy.<br/>                                                                                                                                                                                          |
| [**FSCTL \_ supprimer \_ le \_ journal USN**](/windows/win32/api/winioctl/ni-winioctl-fsctl_delete_usn_journal)<br/>                                                                 | Supprime le journal des modifications du numéro de séquence de mise à jour (USN) sur un volume ou attend la notification de la suppression du journal des modifications.<br/>                                                                                                                                     |
| [**\_volume de démontage FSCTL \_**](/windows/win32/api/winioctl/ni-winioctl-fsctl_dismount_volume)<br/>                                                                        | Démonte un volume, que le volume soit en cours d’utilisation ou non. Pour plus d'informations, consultez la section Notes.<br/>                                                                                                                                 |
| [**données USN de l' \_ énumération FSCTL \_ \_**](/windows/win32/api/winioctl/ni-winioctl-fsctl_enum_usn_data)<br/>                                                                           | Énumère les données de numéro de séquence de mise à jour (USN) entre deux limites spécifiées pour obtenir les enregistrements de table de fichiers maîtres (MFT).<br/>                                                                                                                                   |
| [**FSCTL \_ étendre le \_ volume**](/windows/win32/api/winioctl/ni-winioctl-fsctl_extend_volume)<br/>                                                                            | Augmente la taille d’un volume monté.<br/>                                                                                                                                                                                                                        |
| [**FSCTL \_ obtient les informations de la \_ zone de démarrage \_ \_**](/windows/win32/api/winioctl/ni-winioctl-fsctl_get_boot_area_info)<br/>                                                                | Récupère les emplacements des secteurs de démarrage d’un volume. <br/> Pour effectuer cette opération, appelez la fonction [**DeviceIoControl**](/windows/desktop/api/ioapiset/nf-ioapiset-deviceiocontrol) avec les paramètres suivants.<br/>                                                                      |
| [**FSCTL \_ Obtient \_ les \_ informations d’intégrité**](/windows/win32/api/winioctl/ni-winioctl-fsctl_get_integrity_information)<br/>                                                   | Récupère l’état d’intégrité d’un fichier ou d’un répertoire sur un volume ReFS.<br/>                                                                                                                                                                                        |
| [**FSCTL \_ Obtient \_ les \_ données du volume NTFS \_**](/windows/win32/api/winioctl/ni-winioctl-fsctl_get_ntfs_volume_data)<br/>                                                            | Récupère des informations sur le volume du système de fichiers NTFS spécifié.<br/>                                                                                                                                                                                             |
| [**FSCTL \_ récupérer la \_ base du pointeur de récupération \_ \_**](/windows/win32/api/winioctl/ni-winioctl-fsctl_get_retrieval_pointer_base)<br/>                                                | Retourne le décalage de secteur au premier numéro de cluster logique (LCN) du système de fichiers par rapport au début du volume.<br/> Pour effectuer cette opération, appelez la fonction [**DeviceIoControl**](/windows/desktop/api/ioapiset/nf-ioapiset-deviceiocontrol) avec les paramètres suivants.<br/> |
| [**FSCTL \_ obtient les \_ \_ pointeurs de récupération**](/windows/win32/api/winioctl/ni-winioctl-fsctl_get_retrieval_pointers)<br/>                                                         | À partir d’un descripteur de fichier, récupère une structure de données qui décrit l’allocation et l’emplacement sur le disque d’un fichier spécifique, ou, à partir d’un handle de volume, les emplacements de clusters défectueux sur un volume.<br/>                                                                   |
| [**FSCTL \_ Obtient \_ l' \_ image bitmap du volume**](/windows/win32/api/winioctl/ni-winioctl-fsctl_get_volume_bitmap)<br/>                                                                   | Récupère une image bitmap de clusters occupés et disponibles sur un volume.<br/>                                                                                                                                                                                             |
| [**FSCTL \_ est \_ un \_ fichier CSV**](/windows/win32/api/winioctl/ni-winioctl-fsctl_is_csv_file)<br/>                                                                               | Détermine si un fichier est stocké sur un volume CSVFS ou récupère des informations sur l’espace de noms.<br/>                                                                                                                                                                     |
| [**FSCTL \_ est un \_ fichier \_ sur le \_ volume partagé de \_ cluster**](/windows/win32/api/winioctl/ni-winioctl-fsctl_is_file_on_csv_volume)<br/>                                                         | Détermine si un fichier est stocké sur un volume CSVFS ou récupère des informations sur l’espace de noms.<br/>                                                                                                                                                                     |
| [**FSCTL \_ est \_ monté en volume \_**](/windows/win32/api/winioctl/ni-winioctl-fsctl_is_volume_mounted)<br/>                                                                   | Détermine si le volume spécifié est monté, ou si le fichier ou le répertoire spécifié se trouve sur un volume monté.<br/>                                                                                                                                              |
| [**FSCTL \_ est \_ propriétaire du volume \_ \_ BYCSVFS**](/windows/win32/api/winioctl/ni-winioctl-fsctl_is_volume_owned_bycsvfs)<br/>                                                      | Détermine si un volume est verrouillé par CSVFS.<br/>                                                                                                                                                                                                                |
| [**\_volume de verrou FSCTL \_**](/windows/win32/api/winioctl/ni-winioctl-fsctl_lock_volume)<br/>                                                                                | Verrouille un volume s’il n’est pas en cours d’utilisation.<br/>                                                                                                                                                                                                                            |
| [**\_ \_ flux de recherche FSCTL \_ à partir du \_ cluster**](/windows/win32/api/winioctl/ni-winioctl-fsctl_lookup_stream_from_cluster)<br/>                                                | À partir d’un handle vers un volume NTFS ou un fichier sur un volume NTFS, retourne une chaîne de structures de données qui décrit les flux qui occupent les clusters spécifiés.<br/>                                                                                                      |
| [**\_descripteur de marque FSCTL \_**](/windows/win32/api/winioctl/ni-winioctl-fsctl_mark_handle)<br/>                                                                                | Marque un fichier ou un répertoire spécifié et son enregistrement de journal des modifications à l’aide d’informations sur les modifications apportées à ce fichier ou répertoire.<br/>                                                                                                                                    |
| [**FSCTL \_ déplacer le \_ fichier**](/windows/win32/api/winioctl/ni-winioctl-fsctl_move_file)<br/>                                                                                    | Déplace un ou plusieurs clusters virtuels d’un fichier d’un cluster logique vers un autre au sein du même volume. Cette opération est utilisée lors de la [défragmentation](defragmenting-files.md).<br/>                                                                         |
| [**\_reconnaissance du \_ système de fichiers de requête FSCTL \_ \_**](/windows/win32/api/winioctl/ni-winioctl-fsctl_query_file_system_recognition)<br/>                                          | Recherche des informations sur la reconnaissance du système de fichiers sur un volume.<br/> Pour effectuer cette opération, appelez la fonction [**DeviceIoControl**](/windows/desktop/api/ioapiset/nf-ioapiset-deviceiocontrol) avec les paramètres suivants.<br/>                                                                |
| [**\_ \_ informations sur la région de requête FSCTL \_**](/windows/win32/api/winioctl/ni-winioctl-fsctl_query_region_info)<br/>                                                                   | Récupère les régions de niveau de stockage définies pour un volume qui prend en charge la hiérarchisation des données.<br/> Pour effectuer cette opération, appelez la fonction [**DeviceIoControl**](/windows/desktop/api/ioapiset/nf-ioapiset-deviceiocontrol) avec les paramètres suivants.<br/>                                         |
| [**\_classes de \_ stockage des requêtes FSCTL \_**](/windows/win32/api/winioctl/ni-winioctl-fsctl_query_storage_classes)<br/>                                                           | Récupère les niveaux de stockage définis pour un volume qui prend en charge la hiérarchisation des données.<br/> Pour effectuer cette opération, appelez la fonction [**DeviceIoControl**](/windows/desktop/api/ioapiset/nf-ioapiset-deviceiocontrol) avec les paramètres suivants.<br/>                                                |
| [**\_ \_ journal USN des requêtes FSCTL \_**](/windows/win32/api/winioctl/ni-winioctl-fsctl_query_usn_journal)<br/>                                                                   | Recherche des informations sur le journal des modifications du numéro de séquence de mise à jour (USN) actuel, ses enregistrements et sa capacité.<br/>                                                                                                                                             |
| [**FSCTL \_ lire \_ les \_ données USN du fichier \_**](/windows/win32/api/winioctl/ni-winioctl-fsctl_read_file_usn_data)<br/>                                                                | Récupère les informations de journalisation du numéro de séquence de mise à jour (USN) pour le fichier ou le répertoire spécifié.<br/>                                                                                                                                                     |
| [**FSCTL \_ lecture \_ à partir du \_ Plex**](/windows/win32/api/winioctl/ni-winioctl-fsctl_read_from_plex)<br/>                                                                         | Lit à partir du plex spécifié.<br/>                                                                                                                                                                                                                                 |
| [**\_ \_ journal USN de lecture FSCTL \_**](/windows/win32/api/winioctl/ni-winioctl-fsctl_read_usn_journal)<br/>                                                                     | Récupère l’ensemble des enregistrements de journal de modification du numéro de séquence de mise à jour (USN) entre deux valeurs USN spécifiées.<br/>                                                                                                                                                     |
| [**FSCTL \_ les \_ copies de réparation**](/windows/win32/api/winioctl/ni-winioctl-fsctl_repair_copies)<br/>                                                                            | Réparez les données endommagées en sélectionnant la copie appropriée à utiliser.<br/>                                                                                                                                                                                                    |
| [**FSCTL \_ définir \_ les \_ informations d’intégrité**](/windows/win32/api/winioctl/ni-winioctl-fsctl_set_integrity_information)<br/>                                                   | Récupère l’état d’intégrité d’un fichier ou d’un répertoire sur un volume ReFS.<br/>                                                                                                                                                                                        |
| [**FSCTL \_ réduire le \_ volume**](/windows/win32/api/winioctl/ni-winioctl-fsctl_shrink_volume)<br/>                                                                            | Signale que le volume doit être préparé pour effectuer l’opération de réduction, que l’opération de réduction doit être validée ou que l’opération de réduction doit être arrêtée.<br/>                                                                                               |
| [**FSCTL \_ déverrouiller le \_ volume**](/windows/win32/api/winioctl/ni-winioctl-fsctl_unlock_volume)<br/>                                                                            | Déverrouille un volume.<br/>                                                                                                                                                                                                                                              |
| [**FSCTL \_ USN \_ suivi \_ des \_ plages modifiées**](/windows/win32/api/winioctl/ni-winioctl-fsctl_usn_track_modified_ranges)<br/>                                                  | Active la fonctionnalité de suivi de plage pour le flux de journal des modifications de numéro de séquence de mise à jour (USN) sur un volume cible, ou modifie les paramètres de suivi de plage déjà activés.<br/>                                                                                               |
| [**FSCTL \_ écrire \_ l' \_ enregistrement USN fermer \_**](/windows/win32/api/winioctl/ni-winioctl-fsctl_write_usn_close_record)<br/>                                                        | Génère un enregistrement dans le flux de journal de modification du numéro de séquence de mise à jour (USN) pour le fichier d’entrée.<br/>                                                                                                                                                               |
| [**attributs de la \_ récupération du volume IOCTL \_ \_ \_**](/windows/desktop/api/WinIoCtl/ni-winioctl-ioctl_volume_get_gpt_attributes)<br/>                                                  | Récupère les attributs d’un volume.<br/> Pour effectuer cette opération, appelez la fonction [**DeviceIoControl**](/windows/desktop/api/ioapiset/nf-ioapiset-deviceiocontrol) avec les paramètres suivants.<br/>                                                                                      |
| [**\_ \_ \_ \_ étendues de disque d’extraction de volume IOCTL \_**](/windows/desktop/api/WinIoCtl/ni-winioctl-ioctl_volume_get_volume_disk_extents)<br/>                                       | Récupère l’emplacement physique d’un volume spécifié sur un ou plusieurs disques.<br/>                                                                                                                                                                                    |
| [**le \_ volume IOCTL \_ est \_ en cluster**](/windows/desktop/api/WinIoCtl/ni-winioctl-ioctl_volume_is_clustered)<br/>                                                               | Détermine si le volume spécifié est en cluster.<br/>                                                                                                                                                                                                          |
| [**le \_ volume IOCTL \_ est \_ CSV**](ioctl-volume-is-csv.md)<br/>                                                                           | Détermine si un volume est un volume partagé de cluster.<br/>                                                                                                                                                                                                                   |
| [**\_volume IOCTL \_ hors connexion**](/windows/desktop/api/WinIoCtl/ni-winioctl-ioctl_volume_offline)<br/>                                                                          | Met un volume hors connexion.<br/>                                                                                                                                                                                                                                        |
| [**\_volume IOCTL \_ en ligne**](/windows/desktop/api/WinIoCtl/ni-winioctl-ioctl_volume_online)<br/>                                                                            | Met un volume en ligne.<br/>                                                                                                                                                                                                                                        |
