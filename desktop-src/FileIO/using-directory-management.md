# <a name="using-directory-management"></a>Utilisation de la gestion d’annuaire

Les rubriques suivantes illustrent l’utilisation des fonctions de gestion d’annuaire.

## <a name="in-this-section"></a>Contenu de cette section



| Rubrique                                                                                               | Description                                                                                                                                                                                                                                                                     |
|-----------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Modification du répertoire actif](changing-the-current-directory.md)<br/>                     | Une application peut modifier le répertoire actif en appelant la fonction SetCurrentDirectory.<br/>                                                                                                                                                                         |
| [Affichage des fichiers dans un répertoire](listing-the-files-in-a-directory.md)<br/>                 | Pour répertorier les fichiers dans un répertoire spécifié, appelez FindFirstFile, FindNextFile et FindClose.<br/>                                                                                                                                                                             |
| [Déplacement de répertoires](moving-directories.md)<br/>                                             | Pour déplacer un répertoire vers un autre emplacement, ainsi que les fichiers et les sous-répertoires qu’il contient, appelez la fonction [**MoveFileEx**](/windows/desktop/api/WinBase/nf-winbase-movefileexa), [**MoveFileWithProgress**](/windows/desktop/api/WinBase/nf-winbase-movefilewithprogressa)ou [**MoveFileTransacted**](/windows/desktop/api/WinBase/nf-winbase-movefiletransacteda) .<br/> |
| [Obtention de notifications de modification de répertoire](obtaining-directory-change-notifications.md)<br/> | Une application peut surveiller le contenu d’un répertoire et de ses sous-répertoires à l’aide de notifications de modification.<br/>                                                                                                                                                         |
