# <a name="span-idvspixenginepixenginetexturesliceindexspanpixenginetexturesliceindex-structure"></a><span id="vspixengine.pixenginetexturesliceindex"></span>PixEngineTextureSliceIndex, structure

Représente l’index d’une tranche de texture

## <a name="syntax"></a>Syntaxe


```C++
} PixEngineTextureSliceIndex;
```

## <a name="members"></a>Membres

**arrayIndex**  
Lorsque la texture est un tableau de textures, l’index de tableau de la tranche.

**mipmapLevel**  
Niveau de mipmap de la tranche.

**sampleIndex**  
Exemple d’index de la tranche.

**zSlice**  
Profondeur (z) de la tranche.
