# <a name="effect-compiler-tool"></a>Outil Effect-Compiler

FXC (fxc.exe) est un outil hors connexion pour la compilation des nuanceurs [HLSL](/windows/desktop/direct3dhlsl/dx-graphics-hlsl) pour toutes les versions de Direct3D. L’outil se trouve à l’emplacement suivant :

(*Racine SDK*) \\ Emplacement des utilitaires \\ \\ x86\\
