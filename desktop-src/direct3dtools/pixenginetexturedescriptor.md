# <a name="span-idvspixenginepixenginetexturedescriptorspanpixenginetexturedescriptor-structure"></a><span id="vspixengine.pixenginetexturedescriptor"></span>PixEngineTextureDescriptor, structure

Représente une description d’une ressource de texture.

## <a name="syntax"></a>Syntaxe


```C++
} PixEngineTextureDescriptor;
```

## <a name="members"></a>Membres

**formatDesc**  
Décrit le format de la texture.

**mipmapCount**  
Nombre de des mipmaps associés à la texture.

**arrayCount**  
Lorsque la texture est un tableau de textures, le nombre de textures dans le tableau.

**profondeur**  

**sampleCount**  
Nombre d’échantillons associés à la texture.
