# <a name="writing-wpp-events"></a>Écriture des événements WPP

Le processeur de suivi logiciel (WPP) Windows fournit un mécanisme efficace pour consigner les événements qui se produisent pendant l’exécution d’une application ou d’un pilote. WPP utilise l’API ETW.

Pour plus d’informations sur l’utilisation de WPP pour consigner des événements, consultez [Forum aux questions](/windows-hardware/drivers/devtest/software-tracing-faq) sur le suivi logiciel et le suivi des logiciels [WPP](/windows-hardware/drivers/devtest/wpp-software-tracing) dans le kit de développement de pilotes (DDK) Microsoft Windows.
