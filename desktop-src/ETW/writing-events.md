# <a name="writing-events"></a>Écrire des événements

La façon dont vous écrivez des événements dans une session de trace est déterminée par le type de votre fournisseur.

Pour les fournisseurs [MOF](about-event-tracing.md) , consultez [écriture d’événements MOF (classiques)](tracing-events.md).

Pour les fournisseurs [WPP](about-event-tracing.md) , consultez [écriture d’événements WPP](windows-software-trace-preprocessor.md).

Pour les fournisseurs [basés sur un manifeste](about-event-tracing.md) , consultez [écriture d’événements basés sur un manifeste](writing-manifest-based-events.md).

Pour les fournisseurs [TraceLogging](about-event-tracing.md) , consultez [utilisation de TraceLogging](../tracelogging/tracelogging-using-tracelogging.md).
