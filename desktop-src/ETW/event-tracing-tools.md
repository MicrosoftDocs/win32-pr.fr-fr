# <a name="event-tracing-tools"></a>Outils de suivi d’événements

Les outils de suivi d’événements suivants sont disponibles pour votre utilisation.



| Outil     | Description                                                                                                                                                                                                                                                                                                                                |
|----------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Logman   | Gère et planifie les collectes de compteurs de performances et de journaux de suivi des événements sur les systèmes locaux et distants. Vous pouvez utiliser cet outil pour répertorier les fournisseurs qui ont [publié la disposition de leurs données d’événement](publishing-your-event-schema.md) dans l' \\ espace de noms WMI racine. Pour plus d’informations sur l’utilisation de cet outil, consultez le **Centre d’aide et de support**. |
| Tracefmt | Met en forme les informations dans un fichier journal de trace sous une forme lisible par l’utilisateur. pour plus d’informations, consultez outils pour le suivi logiciel dans le kit de développement de pilotes Windows Microsoft (DDK). Utilisé uniquement par WPP Software Tracing.                                                                                                                                      |
| Tracelog | Contrôle une session de suivi d’événements. Pour plus d’informations, consultez outils pour le suivi logiciel dans le DDK.                                                                                                                                                                                                                                                   |
| Tracepdb | Crée un fichier de format de message de trace que l’outil tracefmt utilise pour convertir les messages journalisés sous une forme lisible par l’utilisateur. Pour plus d’informations, consultez outils pour le suivi logiciel dans le DDK. Utilisé uniquement par WPP Software Tracing.                                                                                                                            |
| Tracerpt | Traite les journaux de suivi d’événements ou les événements en temps réel à partir des fournisseurs de suivi d’événements instrumentés et vous permet de générer des rapports d’analyse de trace et des fichiers CSV (délimités par des virgules) pour les événements générés. Pour plus d’informations, consultez le **Centre d’aide et de support**.                                                                                       |
