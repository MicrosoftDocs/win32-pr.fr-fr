# <a name="device-drivers"></a>Pilotes de périphériques

Cette documentation concerne les applications en mode utilisateur qui souhaitent utiliser ETW pour le suivi d’événements. Cette documentation n’est pas destinée aux pilotes de périphérique qui s’exécutent en mode noyau. Pour ceux qui écrivent des pilotes de périphérique et des applications en mode noyau qui souhaitent utiliser ETW comme débogueur à faible impact pour résoudre les problèmes liés à la synchronisation, consultez [suivi d’v nements pour Windows](/windows-hardware/drivers/devtest/event-tracing-for-windows--etw-), [WPP Software Tracing](/windows-hardware/drivers/devtest/wpp-software-tracing)et [WMI Event Tracing](/windows-hardware/drivers/kernel/wmi-event-tracing) dans Windows Driver Kit (WDK) pour plus d’informations sur l’instrumentation de votre pilote.
