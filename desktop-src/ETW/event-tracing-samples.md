# <a name="event-tracing-samples"></a>Exemples de traçage d’événements

Le kit de développement logiciel (SDK) de la plateforme contient des exemples complets de suivi des événements. Ces exemples se trouvent dans le répertoire Samples \\ WinBase \\ EventTrace. La racine de ce chemin d’accès est le répertoire d’installation de base du SDK. Vous pouvez télécharger le SDK à partir de [https://www.microsoft.com/downloads/details.aspx?FamilyId=A55B6B43-E24F-4EA3-A93E-40C0EC4F68E5](https://msdn.microsoft.com/windowsserver/bb980924.aspx) .

Le tableau suivant décrit les exemples contenus dans le répertoire Samples \\ WinBase \\ EventTrace.



| Exemple   | Description                                                                                                                            |
|----------|----------------------------------------------------------------------------------------------------------------------------------------|
| TraceDmp | Consommateur de suivi d’événements. Il décode les données d’événement à l’aide des informations de format obtenues à partir de WMI et génère les données dans un fichier. csv. |
| TraceDp  | Utilise l’API du fournisseur de suivi d’événements pour fournir des données de trace d’événements à l’enregistreur d’événements ou à un consommateur.                                             |
| TraceLog | Utilise l’API du contrôleur de trace d’événements pour gérer les sessions de journalisation.                                                                        |



 

Pour obtenir une description détaillée, consultez le fichier Readme.txt inclus avec ces exemples.
