# <a name="retrieving-event-data-using-tdh"></a>Récupération de données d’événement à l’aide de TDH

Pour utiliser des données spécifiques à un événement, le consommateur doit connaître le format des données d’événement. Si le fournisseur a utilisé des fichiers manifeste, MOF ou TMF pour publier le format des données d’événement, vous pouvez utiliser les fonctions TDH (Trace Data Helper) pour analyser les données d’événement. les fonctions TDH sont disponibles à partir de Windows Vista.

Les rubriques suivantes vous montrent comment utiliser TDH pour récupérer les métadonnées et les données d’événement.

-   [Utilisation de TdhFormatProperty pour consommer des données d’événement](using-tdhformatproperty-to-consume-event-data.md)
-   [Utilisation de TdhGetProperty pour consommer des données d’événement](using-tdhgetproperty-to-consume-event-data.md)
-   [Récupération des métadonnées d’événement](retrieving-event-metadata.md)
-   [Énumération des fournisseurs](enumerating-providers.md)
