# <a name="developer-license-apis"></a>API de licence développeur

Utilisez ces API pour vérifier, acquérir et supprimer une licence de développeur.

## <a name="in-this-section"></a>Contenu de cette section



| Rubrique                                                                 | Description                                             |
|-----------------------------------------------------------------------|---------------------------------------------------------|
| [**CheckDeveloperLicense**](/windows/desktop/api/wsdevlicensing/nf-wsdevlicensing-checkdeveloperlicense)<br/>     | Vérifie s’il existe une licence de développeur.<br/> |
| [**AcquireDeveloperLicense**](/windows/desktop/api/wsdevlicensing/nf-wsdevlicensing-acquiredeveloperlicense)<br/> | Acquiert une licence de développeur.<br/>                |
| [**RemoveDeveloperLicense**](/windows/desktop/api/wsdevlicensing/nf-wsdevlicensing-removedeveloperlicense)<br/>   | Supprime une licence de développeur.<br/>                 |
