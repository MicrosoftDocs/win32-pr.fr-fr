# <a name="direct3d"></a>Direct3D

Direct3D est une API de bas niveau pour le dessin de primitives avec le pipeline de rendu ou pour l’exécution d’opérations parallèles avec le nuanceur de calcul. Pour plus d’informations, consultez le contenu ci-dessous.

Pour plus d’informations sur l’obtention et l’installation de Direct3D, consultez Configuration de l' [environnement de programmation Direct3D 12](./direct3d12/directx-12-programming-environment-set-up.md).
