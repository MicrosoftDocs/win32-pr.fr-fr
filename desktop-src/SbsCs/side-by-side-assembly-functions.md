# <a name="side-by-side-assembly-functions"></a>Fonctions d’assembly côte à côte

L’API d’assembly côte à côte utilise les fonctions suivantes pour accéder aux interfaces d’assembly côte à côte.



| Fonction                                                     | Description                                                                    |
|--------------------------------------------------------------|--------------------------------------------------------------------------------|
| [**CreateAssemblyCache,**](/windows/desktop/api/Winsxs/nf-winsxs-createassemblycache)           | Obtient une instance de l’interface [**IAssemblyCache**](/windows/desktop/api/winsxs/nn-winsxs-iassemblycache) . |
| [**CreateAssemblyNameObject,**](/windows/desktop/api/Winsxs/nf-winsxs-createassemblynameobject) | Obtient une instance de l’interface [**IAssemblyName**](/windows/desktop/api/winsxs/nn-winsxs-iassemblyname) .   |
