# <a name="side-by-side-assembly-interfaces"></a>Interfaces d’assembly côte à côte

L’API d’assembly côte à côte utilise les interfaces suivantes.



| Interface                                        | Description                                                                                                                                            |
|--------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IAssemblyCache**](/windows/desktop/api/winsxs/nn-winsxs-iassemblycache)         | Utilisé pour installer, désinstaller ou interroger un assembly côte à côte.                                                                                          |
| [**IAssemblyCacheItem**](/windows/desktop/api/winsxs/nn-winsxs-iassemblycacheitem) | Utilisé pour installer des assemblys côte à côte dans le magasin côte à côte à l’aide d’une installation basée sur un flux.                                                 |
| [**IAssemblyName**](/windows/desktop/api/winsxs/nn-winsxs-iassemblyname)           | Représente un nom d’assembly côte à côte. Le nom de l’assembly côte à côte se compose d’un ensemble de paires nom-valeur qui décrivent l’assembly côte à côte. |
