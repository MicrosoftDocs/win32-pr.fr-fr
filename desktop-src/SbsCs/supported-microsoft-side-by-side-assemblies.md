# <a name="supported-microsoft-side-by-side-assemblies"></a>Assemblys côte à côte Microsoft pris en charge

Les éléments suivants sont les assemblys côte à côte pris en charge par Microsoft. Windows Les service packs peuvent inclure des versions compatibles de ces assemblys avec des correctifs de sécurité ou de bogue.

Vous pouvez rechercher les fichiers redistribuables disponibles dans le [Centre de téléchargement Microsoft](https://www.microsoft.com/Downloads/).

## <a name="windows-server-2003"></a>Windows Server 2003

ces assemblys côte à côte sont pris en charge par Microsoft à partir de Windows Server 2003.

-   WinHTTP 5,1
-   Assemblys d’automatisation de l’isolation Microsoft qui permettent l’utilisation d’assemblys côte à côte avec des scripts.
-   Contrôles communs de shell version 6,0 (Comctl32.dll)
-   GDI plus version 1,0 (GDIplus.dll)
-   Bibliothèques Runtime Visual C++ version 6,0

## <a name="windows-xp"></a>Windows XP

ces assemblys côte à côte étaient pris en charge par Microsoft à partir de Windows XP.

-   Contrôles communs de shell version 6,0 (Comctl32.dll)
-   GDI plus version 1,0 (GDIplus.dll)
-   Bibliothèques Runtime Visual C++ version 6,0
