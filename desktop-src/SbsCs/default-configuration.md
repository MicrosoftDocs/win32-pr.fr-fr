# <a name="default-configuration"></a>Configuration par défaut

Les versions d’assembly spécifiées dans les manifestes d' [assembly](assembly-manifests.md) et les [manifestes d’application](application-manifests.md) constituent la configuration par défaut que l’application doit utiliser lorsque l’application s’exécute. Les applications et les administrateurs peuvent remplacer la configuration de l’assembly par défaut en installant une configuration de serveur de [publication](publisher-configuration.md) ou une [configuration par application](per-application-configuration.md) sur Windows Server 2003.
