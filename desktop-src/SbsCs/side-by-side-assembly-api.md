# <a name="side-by-side-assembly-api"></a>API d’assembly côte à côte

L’API d’assembly côte à côte installe, supprime ou interroge des assemblys côte à côte à partir du magasin d’assemblys côte à côte du système. c’est également ce que l’on appelle le magasin winSxS sur Windows XP et le magasin de composants d’Infrastructure de maintenance des composants (CSI) à partir de Windows Vista et Windows Server 2008.

Les parties de l’API d’assembly côte à côte sont décrites dans les sections suivantes :

-   [Énumérations d’assemblys côte à côte](enumerations.md)
-   [Fonctions d’assembly côte à côte](side-by-side-assembly-functions.md)
-   [Interfaces d’assembly côte à côte](side-by-side-assembly-interfaces.md)
-   [Structures d’assembly côte à côte](structures.md)
