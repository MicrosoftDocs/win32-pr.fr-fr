# <a name="enabling-updated-common-controls-visual-styles-and-themes"></a>Activation des contrôles communs mis à jour, des styles visuels et des thèmes

Les développeurs d’applications qui s’exécutent sur Windows XP peuvent ajouter des styles et des thèmes visuels à l’interface utilisateur de leurs applications en incluant des manifestes qui décrivent la dépendance de l’application vis-à-vis de COMCTL32.DLL version 6,0. Une application requiert les fonctionnalités des contrôles communs mis à jour pour utiliser les thèmes et les styles visuels.

Pour plus d’informations sur l’utilisation des styles visuels, consultez Utilisation des styles visuels Windows XP dans le [Guide du programmeur de Shell](/previous-versions/windows/desktop/legacy/bb776778(v=vs.85)). L’assembly côte à côte contenant COMCTL32.DLL version 6,0 est l’un des [assemblys côte à côte Microsoft pris en charge](supported-microsoft-side-by-side-assemblies.md).
