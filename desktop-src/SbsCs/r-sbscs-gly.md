<span id="_win32_ref_context_assemblyidentity_gly"></span><span id="_WIN32_REF_CONTEXT_ASSEMBLYIDENTITY_GLY"></span>**AssemblyIdentity de contexte REF**
</dt> <dd>

Sous-élément **assemblyIdentity** qui décrit un assembly côte à côte. La version de l’assembly décrit est la version requise par l’assembly ou l’application défini dans le **assemblyIdentity** du contexte def du manifeste. Un sous-élément **assemblyIdentity** de contexte Ref est le premier sous-élément d’un élément **dependentAssembly** .
