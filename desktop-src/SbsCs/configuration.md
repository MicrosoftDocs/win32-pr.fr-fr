# <a name="configuration-isolated-applications-and-side-by-side-assemblies"></a>Configuration (applications isolées et assemblys côte à côte)

[Publisher fichiers de configuration](publisher-configuration-files.md) et les [fichiers de configuration](application-configuration-files.md) de l’application permettent aux éditeurs d’assemblys, aux éditeurs d’applications et aux administrateurs de modifier les dépendances d’assembly côte à côte après le déploiement.

Pour plus d’informations sur l’utilisation de manifestes et de fichiers de configuration pour définir la dépendance de version d’assembly côte à côte, consultez les rubriques suivantes :

-   [Configuration par défaut](default-configuration.md)
-   [Publisher Configuré](publisher-configuration.md)
-   [Configuration par application](per-application-configuration.md)
-   [Configuration par application sur Windows XP](per-application-configuration-on-windows-xp.md)
