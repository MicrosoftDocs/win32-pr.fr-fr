# <a name="input-feedback-configuration"></a>Configuration des commentaires en entrée

## <a name="purpose"></a>Objectif

Les rubriques de cette section fournissent une vue d’ensemble de la configuration des commentaires en entrée dans Windows 8. La configuration d’entrée de commentaires donne accès aux paramètres de commentaires visuels pour les interactions de l’utilisateur.

La collection de rubriques fournie ici décrit comment utiliser la configuration d’entrée de commentaires pour optimiser l’expérience utilisateur de votre application. Par exemple, les paramètres de commentaires visuels par défaut peuvent ne pas être nécessaires ou souhaitables dans une application de jeu en raison d’interférences ou de conflits avec le jeu lui-même.

## <a name="developer-audience"></a>Développeurs concernés

Le ConfigurationAPIs d’entrée de commentaires est conçu pour les développeurs qui créent des infrastructures d’interface utilisateur qui fournissent une expérience utilisateur cohérente et optimisée pour les applications de bureau.
