# <a name="taskdefinitionsettings-property"></a>TaskDefinition. Settings, propriété

Pour les scripts, obtient ou définit les paramètres qui définissent la façon dont le service de Planificateur de tâches effectue la tâche.

## <a name="syntax"></a>Syntaxe


```VB
TaskDefinition.Settings As TaskSettings
```



## <a name="property-value"></a>Valeur de la propriété

Paramètres qui définissent la façon dont le service de Planificateur de tâches effectue la tâche.
