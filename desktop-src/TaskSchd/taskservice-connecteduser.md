# <a name="taskserviceconnecteduser-property"></a>TaskService. ConnectedUser, propriété

Pour les scripts, obtient le nom de l’utilisateur qui est connecté au service Planificateur de tâches.

## <a name="syntax"></a>Syntaxe


```VB
TaskService.ConnectedUser As String
```



## <a name="property-value"></a>Valeur de la propriété

Nom de l’utilisateur connecté au service Planificateur de tâches.
