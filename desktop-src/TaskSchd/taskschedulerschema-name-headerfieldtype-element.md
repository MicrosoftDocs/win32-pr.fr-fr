# <a name="name-headerfieldtype-element"></a>Élément Name (headerFieldType)

Contient le nom d’un champ d’en-tête dans un message électronique.

``` syntax
<xs:element name="Name"
    type="nonEmptyString"
 />
```

L’élément **Name** est défini par le type complexe [**headerFieldType**](taskschedulerschema-headerfieldtype-complextype.md) .

## <a name="parent-element"></a>Élément parent



| Élément                                                                         | Dérivé de                                                               | Description                                                    |
|---------------------------------------------------------------------------------|----------------------------------------------------------------------------|----------------------------------------------------------------|
| [**HeaderField**](taskschedulerschema-headerfield-headerfieldstype-element.md) | [**headerFieldType**](taskschedulerschema-headerfieldtype-complextype.md) | Contient un champ pour un en-tête dans un message électronique. <br/> |
