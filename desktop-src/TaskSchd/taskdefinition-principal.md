# <a name="taskdefinitionprincipal-property"></a>TaskDefinition. principal (propriété)

Pour les scripts, obtient ou définit le principal de la tâche qui fournit les informations d’identification de sécurité pour la tâche.

Cette propriété est en lecture/écriture.

## <a name="syntax"></a>Syntaxe


```VB
TaskDefinition.Principal As Principal
```



## <a name="property-value"></a>Valeur de la propriété

Principal de la tâche qui fournit les informations d’identification de sécurité pour la tâche.
