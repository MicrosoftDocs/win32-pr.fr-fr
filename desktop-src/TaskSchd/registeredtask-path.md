# <a name="registeredtaskpath-property"></a>RegisteredTask. Path, propriété

Pour les scripts, obtient le chemin d’accès à l’emplacement où la tâche inscrite est stockée.

## <a name="syntax"></a>Syntaxe


```VB
RegisteredTask.Path As String
```



## <a name="property-value"></a>Valeur de la propriété

Chemin d’accès de l’emplacement où la tâche inscrite est stockée.
