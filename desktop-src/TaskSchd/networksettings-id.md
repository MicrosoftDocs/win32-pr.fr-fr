# <a name="networksettingsid-property"></a>Propriété NetworkSettings.Id

Pour les scripts, obtient ou définit une valeur GUID qui identifie un profil réseau.

## <a name="syntax"></a>Syntaxe


```VB
NetworkSettings.Id As String
```



## <a name="property-value"></a>Valeur de la propriété

Valeur GUID qui identifie un profil réseau.
