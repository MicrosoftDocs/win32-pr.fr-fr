# <a name="tasknamedvaluecollectioncount-property"></a>TaskNamedValueCollection. Count (propriété)

Pour les scripts, obtient le nombre de paires nom-valeur dans la collection.

## <a name="syntax"></a>Syntaxe


```VB
TaskNamedValueCollection.Count As long
```



## <a name="property-value"></a>Valeur de la propriété

Nombre de paires nom/valeur dans la collection.
