# <a name="registrationinfoxmltext-property"></a>Propriété de texte RegistrationInfo.Xml

Pour les scripts, obtient ou définit une version au format XML des informations d’inscription pour la tâche.

## <a name="syntax"></a>Syntaxe


```VB
RegistrationInfo.XmlText As String
```



## <a name="property-value"></a>Valeur de la propriété

Version au format XML des informations d’inscription de la tâche.
