# <a name="taskserviceconnected-property"></a>TaskService. Connected, propriété

Pour les scripts, obtient une valeur booléenne qui indique si vous êtes connecté au service Planificateur de tâches.

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```VB
TaskService.Connected
```



## <a name="property-value"></a>Valeur de la propriété

Valeur booléenne qui indique si vous êtes connecté au service Planificateur de tâches.
