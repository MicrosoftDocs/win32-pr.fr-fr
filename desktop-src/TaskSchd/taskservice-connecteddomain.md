# <a name="taskserviceconnecteddomain-property"></a>TaskService. ConnectedDomain, propriété

Pour les scripts, obtient le nom du domaine auquel l’ordinateur [**TargetServer**](taskservice-targetserver.md) est connecté.

## <a name="syntax"></a>Syntaxe


```VB
TaskService.ConnectedDomain As String
```



## <a name="property-value"></a>Valeur de la propriété

Le domaine auquel l’ordinateur [**TargetServer**](taskservice-targetserver.md) est connecté.
