# <a name="taskdefinitionregistrationinfo-property"></a>TaskDefinition. RegistrationInfo, propriété

Pour les scripts, obtient ou définit les informations d’inscription utilisées pour décrire une tâche, telles que la description de la tâche, l’auteur de la tâche et la date à laquelle la tâche est inscrite.

## <a name="syntax"></a>Syntaxe


```VB
TaskDefinition.RegistrationInfo As RegistrationInfo
```



## <a name="property-value"></a>Valeur de la propriété

Informations d’inscription utilisées pour décrire une tâche, telles que la description de la tâche, l’auteur de la tâche et la date à laquelle la tâche est inscrite.
