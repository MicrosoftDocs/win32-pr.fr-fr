# <a name="emailactionfrom-property"></a>EmailAction. from, propriété

\[Cet objet n’est plus pris en charge. Utilisez IExecAction avec l’applet de commande PowerShell [**Send-MailMessage**](/powershell/module/microsoft.powershell.utility/send-mailmessage) comme solution de contournement.\]

Pour les scripts, obtient ou définit l’adresse de messagerie à partir de laquelle vous souhaitez envoyer le message électronique.

Cette propriété est en lecture/écriture.

## <a name="syntax"></a>Syntax


```VB
EmailAction.From As String
```



## <a name="property-value"></a>Valeur de la propriété

Adresse de messagerie à partir de laquelle vous souhaitez envoyer le message électronique.
