# <a name="starting-an-executable-weekly"></a>Démarrage d’un exécutable hebdomadaire

L’écriture d’une tâche qui démarre un exécutable sur une base hebdomadaire s’effectue en définissant un déclencheur hebdomadaire et une action exécutable.

## <a name="weekly-triggers"></a>Déclencheurs hebdomadaires

Les déclencheurs hebdomadaires utilisent leur limite de départ pour activer le déclencheur et spécifier l’heure de la journée à laquelle la tâche est exécutée. Lorsque le déclencheur est activé, le déclencheur spécifie également le moment où la tâche est exécutée pendant la semaine et si la tâche est exécutée chaque semaine, toutes les semaines, toutes les trois semaines, et ainsi de suite.
