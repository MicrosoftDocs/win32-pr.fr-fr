# <a name="emailactionheaderfields-property"></a>EmailAction. HeaderFields, propriété

\[Cet objet n’est plus pris en charge. Utilisez IExecAction avec l’applet de commande PowerShell [**Send-MailMessage**](/powershell/module/microsoft.powershell.utility/send-mailmessage) comme solution de contournement.\]

Pour les scripts, obtient ou définit les informations d’en-tête dans le message électronique que vous souhaitez envoyer.

Cette propriété est en lecture/écriture.

## <a name="syntax"></a>Syntax


```VB
EmailAction.HeaderFields As TaskNamedValueCollection
```



## <a name="property-value"></a>Valeur de la propriété

Informations d’en-tête dans le message électronique que vous souhaitez envoyer.
