# <a name="sessionstatechangetriggeruserid-property"></a>SessionStateChangeTrigger. UserId, propriété

Pour les scripts, obtient ou définit l’utilisateur pour la session Terminal Server. Lorsqu’une modification d’état de session est détectée pour cet utilisateur, une tâche est démarrée.

## <a name="syntax"></a>Syntaxe


```VB
SessionStateChangeTrigger.UserId As String
```



## <a name="property-value"></a>Valeur de la propriété

Utilisateur de la session de Terminal Server.
