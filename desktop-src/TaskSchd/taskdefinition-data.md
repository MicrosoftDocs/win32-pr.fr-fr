# <a name="taskdefinitiondata-property"></a>TaskDefinition. Data, propriété

Pour les scripts, obtient ou définit les données associées à la tâche. Ces données sont ignorées par le service Planificateur de tâches, mais elles sont utilisées par des tiers qui souhaitent étendre le format des tâches.

## <a name="syntax"></a>Syntaxe


```VB
TaskDefinition.Data As String
```



## <a name="property-value"></a>Valeur de la propriété

Données associées à la tâche.
