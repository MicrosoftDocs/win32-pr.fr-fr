# <a name="tasknamedvaluepairvalue-property"></a>TaskNamedValuePair. Value (propriété)

Pour les scripts, obtient ou définit la valeur associée à un nom dans une paire nom-valeur.

## <a name="syntax"></a>Syntaxe


```VB
TaskNamedValuePair.Value As String
```



## <a name="property-value"></a>Valeur de la propriété

Valeur associée à un nom dans une paire nom-valeur.
