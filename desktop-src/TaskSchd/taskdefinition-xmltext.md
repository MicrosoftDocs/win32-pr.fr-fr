# <a name="taskdefinitionxmltext-property"></a>Propriété de texte TaskDefinition.Xml

Pour les scripts, obtient ou définit la définition XML de la tâche.

## <a name="syntax"></a>Syntaxe


```VB
TaskDefinition.XmlText As String
```



## <a name="property-value"></a>Valeur de la propriété

Définition XML de la tâche.

## <a name="remarks"></a>Notes

Le XML d’une tâche est défini par le [schéma planificateur de tâches](task-scheduler-schema.md). Pour obtenir un exemple de code XML de tâche, consultez [exemple de déclencheur quotidien (XML)](daily-trigger-example--xml-.md).
