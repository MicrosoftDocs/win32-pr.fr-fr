# <a name="actioncollectionxmltext-property"></a>ActionCollection :: XmlText, propriété

Pour les scripts, obtient ou définit une version au format XML de la collection.

## <a name="syntax"></a>Syntaxe


```VB
.XmlText As String
```



## <a name="property-value"></a>Valeur de la propriété

Version au format XML de la collection.
