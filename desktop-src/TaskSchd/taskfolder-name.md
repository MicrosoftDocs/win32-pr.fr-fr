# <a name="taskfoldername-property"></a>Propriété TaskFolder.Name

Pour les scripts, obtient le nom utilisé pour identifier le dossier qui contient une tâche.

## <a name="syntax"></a>Syntaxe


```VB
TaskFolder.Name As String
```



## <a name="property-value"></a>Valeur de la propriété

Nom utilisé pour identifier le dossier.
