# <a name="starting-an-executable-daily"></a>Démarrage quotidien d’un exécutable

L’écriture d’une tâche qui démarre un exécutable quotidiennement est effectuée en définissant un déclencheur quotidien et une action exécutable.

## <a name="daily-triggers"></a>Déclencheurs quotidiens

Les déclencheurs quotidiens utilisent leur limite de départ pour activer le déclencheur et spécifier l’heure de la journée à laquelle la tâche est exécutée. Une fois le déclencheur activé, l’intervalle de déclenchement est utilisé pour indiquer si la tâche est exécutée quotidiennement, tous les jours, tous les trois jours, etc.
