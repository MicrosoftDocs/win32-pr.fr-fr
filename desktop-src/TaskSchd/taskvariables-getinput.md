# <a name="taskvariablesgetinput-method"></a>Méthode TaskVariables. GetInput

Pour les scripts, obtient les variables d’entrée pour une tâche. Cette méthode n’est pas implémentée.

## <a name="syntax"></a>Syntaxe


```VB
TaskVariables.GetInput( _
  ByRef input _
)
```



## <a name="parameters"></a>Paramètres

<dl> <dt>

*entrée* \[ à\]
</dt> <dd>

Variables d’entrée pour une tâche.

</dd> </dl>

## <a name="return-value"></a>Valeur retournée

Cette méthode ne retourne pas de valeur.
