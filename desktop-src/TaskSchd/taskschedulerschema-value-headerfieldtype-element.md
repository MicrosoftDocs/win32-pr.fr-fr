# <a name="value-headerfieldtype-element"></a>Élément Value (headerFieldType)

Contient la valeur d’un champ d’en-tête dans un message électronique.

``` syntax
<xs:element name="Value"
    type="string"
 />
```

L’élément **value** est défini par le type complexe [**headerFieldType**](taskschedulerschema-headerfieldtype-complextype.md) .

## <a name="parent-element"></a>Élément parent



| Élément                                                                         | Dérivé de                                                               | Description                                                    |
|---------------------------------------------------------------------------------|----------------------------------------------------------------------------|----------------------------------------------------------------|
| [**HeaderField**](taskschedulerschema-headerfield-headerfieldstype-element.md) | [**headerFieldType**](taskschedulerschema-headerfieldtype-complextype.md) | Contient un champ pour un en-tête dans un message électronique. <br/> |
