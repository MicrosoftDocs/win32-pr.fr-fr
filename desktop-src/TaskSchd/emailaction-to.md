# <a name="emailactionto-property"></a>Propriété EmailAction.To

\[Cet objet n’est plus pris en charge. Utilisez IExecAction avec l’applet de commande PowerShell [**Send-MailMessage**](/powershell/module/microsoft.powershell.utility/send-mailmessage) comme solution de contournement.\]

Pour les scripts, obtient ou définit l’adresse e-mail ou les adresses à laquelle vous souhaitez envoyer le message électronique.

Cette propriété est en lecture/écriture.

## <a name="syntax"></a>Syntaxe


```VB
EmailAction.To As String
```



## <a name="property-value"></a>Valeur de la propriété

Adresse (ou les adresses) à laquelle vous souhaitez envoyer le message électronique.
