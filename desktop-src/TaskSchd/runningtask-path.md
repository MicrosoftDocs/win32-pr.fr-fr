# <a name="runningtaskpath-property"></a>RunningTask. Path, propriété

Pour les scripts, obtient le chemin d’accès à l’emplacement où la tâche est stockée.

## <a name="syntax"></a>Syntaxe


```VB
RunningTask.Path As string
```



## <a name="property-value"></a>Valeur de la propriété

Chemin d’accès à l’emplacement où la tâche est stockée.
