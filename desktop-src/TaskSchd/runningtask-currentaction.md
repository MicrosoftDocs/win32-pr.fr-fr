# <a name="runningtaskcurrentaction-property"></a>RunningTask. CurrentAction, propriété

Pour les scripts, obtient le nom de l’action en cours exécutée par la tâche en cours d’exécution.

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```VB
RunningTask.CurrentAction As String
```



## <a name="property-value"></a>Valeur de la propriété

Nom de l’action en cours exécutée par la tâche en cours d’exécution.
