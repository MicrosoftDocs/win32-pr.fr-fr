# <a name="runningtaskcollection-object"></a>Objet RunningTaskCollection

Objet de script qui fournit une collection utilisée pour contrôler les tâches en cours d’exécution.

## <a name="members"></a>Membres

L’objet **RunningTaskCollection** possède les types de membres suivants :

-   [Propriétés](#properties)

### <a name="properties"></a>Propriétés

L’objet **RunningTaskCollection** a ces propriétés.



| Propriété                                                | Description                                                    |
|:--------------------------------------------------------|:---------------------------------------------------------------|
| [**Saut**](runningtaskcollection-count.md)<br/> | Obtient le nombre de tâches en cours d’exécution dans la collection.<br/> |
| [**Élément**](runningtaskcollection-item.md)<br/>   | Obtient la tâche spécifiée à partir de la collection. <br/>       |
