# <a name="tasknamedvaluecollectionremove-method"></a>TaskNamedValueCollection. Remove, méthode

Pour les scripts, supprime une paire nom-valeur sélectionnée de la collection.

## <a name="syntax"></a>Syntaxe


```VB
TaskNamedValueCollection.Remove( _
  ByVal index _
)
```



## <a name="parameters"></a>Paramètres

<dl> <dt>

*index* \[ dans\]
</dt> <dd>

Index de la paire nom-valeur à supprimer.

</dd> </dl>

## <a name="return-value"></a>Valeur retournée

Cette méthode ne retourne pas de valeur.
