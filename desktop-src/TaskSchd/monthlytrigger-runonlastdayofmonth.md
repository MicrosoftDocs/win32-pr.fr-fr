# <a name="monthlytriggerrunonlastdayofmonth-property"></a>MonthlyTrigger. RunOnLastDayOfMonth, propriété

Pour les scripts, obtient ou définit une valeur booléenne qui indique que la tâche s’exécute le dernier jour du mois.

## <a name="syntax"></a>Syntaxe


```VB
MonthlyTrigger.RunOnLastDayOfMonth As Boolean
```



## <a name="property-value"></a>Valeur de la propriété

La valeur true indique que la tâche est exécutée le dernier jour du mois, quelle que soit la date réelle de ce jour. Sinon, false.
