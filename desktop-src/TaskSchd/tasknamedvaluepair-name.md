# <a name="tasknamedvaluepairname-property"></a>Propriété TaskNamedValuePair.Name

Pour les scripts, obtient ou définit le nom associé à une valeur dans une paire nom-valeur.

## <a name="syntax"></a>Syntaxe


```VB
TaskNamedValuePair.Name As String
```



## <a name="property-value"></a>Valeur de la propriété

Nom associé à une valeur dans une paire nom-valeur.
