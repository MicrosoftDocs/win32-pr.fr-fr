# <a name="runningtaskname-property"></a>Propriété RunningTask.Name

Pour les scripts, obtient le nom de la tâche.

## <a name="syntax"></a>Syntaxe


```VB
RunningTask.Name As string
```



## <a name="property-value"></a>Valeur de la propriété

Nom de la tâche.
