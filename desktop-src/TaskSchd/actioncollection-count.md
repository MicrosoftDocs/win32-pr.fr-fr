# <a name="actioncollectioncount-property"></a>ActionCollection. Count (propriété)

Pour les scripts, obtient le nombre d’actions dans la collection.

## <a name="syntax"></a>Syntaxe


```VB
ActionCollection.Count As long
```



## <a name="property-value"></a>Valeur de la propriété

Nombre d’actions dans la collection. La collection peut contenir jusqu’à 32 actions.
