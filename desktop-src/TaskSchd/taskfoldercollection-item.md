# <a name="taskfoldercollectionitem-property"></a>TaskFolderCollection. Item, propriété

Pour les scripts, obtient le dossier spécifié à partir de la collection.

## <a name="syntax"></a>Syntaxe


```VB
TaskFolderCollection.Item( _
  ByVal Index _
) As TaskFolder
```



## <a name="property-value"></a>Valeur de la propriété

Objet [**TaskFolder**](taskfolder.md) qui représente le dossier demandé.
