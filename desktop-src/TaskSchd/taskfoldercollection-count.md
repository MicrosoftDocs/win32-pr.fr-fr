# <a name="taskfoldercollectioncount-property"></a>TaskFolderCollection. Count (propriété)

Pour les scripts, obtient le nombre de dossiers dans la collection.

## <a name="syntax"></a>Syntaxe


```VB
TaskFolderCollection.Count As long
```



## <a name="property-value"></a>Valeur de la propriété

Nombre de déclencheurs dans la collection.
