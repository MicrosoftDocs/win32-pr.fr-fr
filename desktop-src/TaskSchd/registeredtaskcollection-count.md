# <a name="registeredtaskcollectioncount-property"></a>RegisteredTaskCollection. Count (propriété)

Pour les scripts, obtient le nombre de tâches inscrites dans la collection.

Cette propriété est en lecture seule.

## <a name="syntax"></a>Syntaxe


```VB
RegisteredTaskCollection.Count As long
```



## <a name="property-value"></a>Valeur de la propriété

Nombre de tâches inscrites dans la collection.
