# <a name="registeredtasklasttaskresult-property"></a>RegisteredTask. LastTaskResult, propriété

Pour les scripts, obtient les résultats qui ont été retournés lors de la dernière exécution de la tâche inscrite.

## <a name="syntax"></a>Syntaxe


```VB
RegisteredTask.LastTaskResult As String
```



## <a name="property-value"></a>Valeur de la propriété

Résultats qui ont été retournés lors de la dernière exécution de la tâche inscrite.
