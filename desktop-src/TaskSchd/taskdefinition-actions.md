# <a name="taskdefinitionactions-property"></a>TaskDefinition. actions, propriété

Pour les scripts, obtient ou définit une collection d’actions exécutées par la tâche.

## <a name="syntax"></a>Syntaxe


```VB
TaskDefinition.Actions As ActionCollection
```



## <a name="property-value"></a>Valeur de la propriété

Collection d’actions exécutées par la tâche.
