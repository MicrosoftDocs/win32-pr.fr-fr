# <a name="taskvariablessetoutput-method"></a>Méthode TaskVariables. SetOutput

Pour les scripts, définit les variables de sortie d’une tâche. Cette méthode n’est pas implémentée.

## <a name="syntax"></a>Syntaxe


```VB
TaskVariables.SetOutput( _
  ByVal input _
)
```



## <a name="parameters"></a>Paramètres

<dl> <dt>

*entrée* \[ dans\]
</dt> <dd>

Variables de sortie pour une tâche.

</dd> </dl>

## <a name="return-value"></a>Valeur retournée

Cette méthode ne retourne pas de valeur.
