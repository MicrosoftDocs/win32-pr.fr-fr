# <a name="registeredtaskenabled-property"></a>RegisteredTask. Enabled, propriété

Pour les scripts, obtient ou définit une valeur booléenne qui indique si la tâche inscrite est activée.

## <a name="syntax"></a>Syntaxe


```VB
RegisteredTask.Enabled As Boolean
```



## <a name="property-value"></a>Valeur de la propriété

Valeur booléenne qui indique si la tâche inscrite est activée.
