# <a name="registeredtaskname-property"></a>Propriété RegisteredTask.Name

Pour les scripts, obtient le nom de la tâche inscrite.

## <a name="syntax"></a>Syntaxe


```VB
RegisteredTask.Name As String
```



## <a name="property-value"></a>Valeur de la propriété

Nom de la tâche inscrite.
