# <a name="taskfoldergetfolders-property"></a>TaskFolder. GetFolders, propriété

Pour les scripts, obtient tous les sous-dossiers du dossier.

## <a name="syntax"></a>Syntaxe


```VB
TaskFolder.GetFolders( _
  ByVal flags, _
  ByRef folders _
)
```
