# <a name="taskfoldergettasks-property"></a>TaskFolder. GetTasks, propriété

Pour les scripts, obtient toutes les tâches dans le dossier.

## <a name="syntax"></a>Syntaxe


```VB
TaskFolder.GetTasks( _
  ByVal flags, _
  ByRef ppTasks _
)
```
