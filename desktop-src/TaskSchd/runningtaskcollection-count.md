# <a name="runningtaskcollectioncount-property"></a>RunningTaskCollection. Count (propriété)

Pour les scripts, obtient le nombre de tâches en cours d’exécution dans la collection.

## <a name="syntax"></a>Syntaxe


```VB
RunningTaskCollection.Count As long
```



## <a name="property-value"></a>Valeur de la propriété

Nombre de tâches en cours d’exécution dans la collection.
