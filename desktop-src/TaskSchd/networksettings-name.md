# <a name="networksettingsname-property"></a>Propriété NetworkSettings.Name

Pour les scripts, obtient ou définit le nom d’un profil réseau. Le nom est utilisé à des fins d’affichage.

Cette propriété est en lecture/écriture.

## <a name="syntax"></a>Syntaxe


```VB
NetworkSettings.Name As String
```



## <a name="property-value"></a>Valeur de la propriété

Nom d’un profil réseau.
