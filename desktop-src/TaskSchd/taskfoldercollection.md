# <a name="taskfoldercollection-object"></a>Objet TaskFolderCollection

Objet de script qui fournit des informations et un contrôle pour une collection de dossiers qui contiennent des tâches.

## <a name="members"></a>Membres

L’objet **TaskFolderCollection** possède les types de membres suivants :

-   [Propriétés](#properties)

### <a name="properties"></a>Propriétés

L’objet **TaskFolderCollection** a ces propriétés.



| Propriété                                               | Description                                                |
|:-------------------------------------------------------|:-----------------------------------------------------------|
| [**Count**](taskfoldercollection-count.md)<br/> | Obtient le nombre de dossiers dans la collection.<br/>   |
| [**Élément**](taskfoldercollection-item.md)<br/>   | Obtient le dossier spécifié à partir de la collection. <br/> |
