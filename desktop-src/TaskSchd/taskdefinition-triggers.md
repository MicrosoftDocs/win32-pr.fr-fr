# <a name="taskdefinitiontriggers-property"></a>TaskDefinition. Triggers, propriété

Pour les scripts, obtient ou définit une collection de déclencheurs utilisés pour démarrer une tâche.

## <a name="syntax"></a>Syntaxe


```VB
TaskDefinition.Triggers As TriggerCollection
```



## <a name="property-value"></a>Valeur de la propriété

Collection de déclencheurs utilisés pour démarrer une tâche.
