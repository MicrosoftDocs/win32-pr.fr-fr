# <a name="endpoint-data-loss-prevention-structures"></a>Structures de protection contre la perte de données de point de terminaison

Structures utilisées par la fonctionnalité de protection contre la perte de données du point de terminaison.



| Structures                                                       | Description                                                           |
|-------------------------------------------------------------------|-----------------------------------------------------------------------|
| [DLP_APP_OP_ENLIGHTENED_LEVEL](endpointdlp-dlp_app_op_enlightened_level.md)                       | Associe une action de protection contre la perte de données (DLP) au niveau de l’application.                                  |
| [DLP_DOCUMENT_INFO](endpointdlp-dlp_document_info.md)                       | Spécifie des informations sur un document associé à une opération DLP de point de terminaison.                                  |
| [DLP_POSTOP_STATUS](enpointdlp-dlp_postop_status.md)                         | Spécifie les informations d’État relatives à une opération DLP de point de terminaison.  |
| [DLP_PRINT_INFO](endpointdlp-dlp_print_info.md)                       | Spécifie des informations sur un document associé à une opération d’impression DLP de point de terminaison.                                  |
