# <a name="the-output-page"></a>Page sortie

\[Microsoft Agent est déconseillé à partir de Windows 7 et peut ne pas être disponible dans les versions ultérieures de Windows.\]

Cette page comprend des propriétés qui contrôlent la sortie de caractères. Par exemple, l’utilisateur peut déterminer s’il faut afficher la sortie dans la bulle de texte, déterminer la manière dont la sortie de la bulle doit apparaître, lire la sortie parlée comme audio, émettre des effets sonores sur les caractères, afficher l’invite de redémarrage et ajuster la vitesse d’élocution.
