# <a name="fontunderline-property"></a>FontUnderline, propriété

\[Microsoft Agent est déconseillé à partir de Windows 7 et peut ne pas être disponible dans les versions ultérieures de Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Descriptive**
</dt> <dd>

Retourne le style de police actuellement affiché dans la fenêtre de la bulle du mot pour le caractère spécifié.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Stockéesyntaxe**
</dt> <dd>

*agent ***. Caractères («*** CharacterID * * * »). Balloon. FontUnderline**



| Valeur     | Description                         |
|-----------|-------------------------------------|
| **True**  | La police de la bulle est soulignée.     |
| **False** | La police de la bulle n’est pas soulignée. |



 

</dd> </dl>

## <a name="remarks"></a>Notes

La valeur par défaut des paramètres de police de la bulle de texte d’un caractère est définie dans l’éditeur de caractères Microsoft Agent. En outre, l’utilisateur peut remplacer les paramètres de police pour tous les caractères de la feuille de propriétés de l’agent Microsoft.
