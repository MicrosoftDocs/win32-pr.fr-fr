# <a name="requirements-for-supporting-the-microsoft-linguistic-information-sound-editing-tool"></a>Conditions requises pour la prise en charge de l’outil de modification du son des informations linguistiques Microsoft

\[Microsoft Agent est déconseillé à partir de Windows 7 et peut ne pas être disponible dans les versions ultérieures de Windows.\]

L’outil de modification de l’information linguistique Microsoft utilise un moteur de reconnaissance vocale pour produire des césures lexicales et des informations phonétiques pour les fichiers audio Wave Windows (. wav) standard. La version 2,0 prend désormais en charge l’utilisation de avec d’autres moteurs de reconnaissance vocale. Les fournisseurs qui souhaitent prendre en charge l’éditeur de sons doivent s’assurer que leurs moteurs prennent entièrement en charge la spécification SAPI 4,0 pour les moteurs de grammaire sans contexte et les exigences suivantes.

-   [ISRResGraphEx et IAttributes](isrresgraphex-and-iattributes.md)
