# <a name="the-default-character-properties-window"></a>La fenêtre Propriétés des caractères par défaut

\[Microsoft Agent est déconseillé à partir de Windows 7 et peut ne pas être disponible dans les versions ultérieures de Windows.\]

En plus de permettre aux applications de charger un caractère spécifique, les applications peuvent charger un caractère qui est une ressource partagée pour l’utilisateur, connu sous le nom de *caractère par défaut*. Le caractère par défaut est accessible à partir de n’importe quelle application, mais le caractère est sélectionnable uniquement par l’utilisateur. Pour faciliter la sélection de ce caractère, agent fournit une fenêtre qui permet d’accéder à la sélection de ce caractère, appelée la fenêtre Propriétés des caractères par défaut. L’accès à cette fenêtre est pris en charge à partir de l’API de l’agent.

![boîte de dialogue Propriétés du caractère génie](images/f8dpwin.gif)

La fenêtre Propriétés de caractères par défaut ne peut pas être utilisée pour fournir une sélection de caractères autre que pour le caractère par défaut.
