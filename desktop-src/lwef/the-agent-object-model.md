# <a name="the-agent-object-model"></a>Modèle objet de l’agent

\[Microsoft Agent est déconseillé à partir de Windows 7 et peut ne pas être disponible dans les versions ultérieures de Windows.\]

Le modèle objet Microsoft Agent se compose des objets suivants :

-   Requête
-   Agent (contrôle)
-   Caractères (collection)
-   Caractère
-   Commandes (collection)
-   Commande
-   Forfaitaire
-   AnimationNames (collection)
-   SpeechInput
-   AudioOutput
-   CommandsWindow
-   Feuille

Ces objets sont organisés dans la hiérarchie suivante. (La ligne en pointillés suivant un objet indique que plusieurs objets peuvent exister.)
