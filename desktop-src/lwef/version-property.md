# <a name="version-property"></a>Propriété Version

\[Microsoft Agent est déconseillé à partir de Windows 7 et peut ne pas être disponible dans les versions ultérieures de Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Descriptive**
</dt> <dd>

Retourne la version du caractère.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Stockéesyntaxe**
</dt> <dd>

*agent. ***Caractères («*** CharacterID * * * »). Version**

</dd> </dl>

## <a name="remarks"></a>Notes

La propriété **version** retourne une chaîne qui correspond à la version de la définition de jeu d’animations standard pour laquelle le caractère a été compilé. Le numéro de version du caractère est défini automatiquement lorsque vous le générez avec l’éditeur de caractères Microsoft Agent.
