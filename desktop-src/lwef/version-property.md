## <a name="remarks"></a>Remarques

La propriété **version** retourne une chaîne qui correspond à la version de la définition de jeu d’animations standard pour laquelle le caractère a été compilé. Le numéro de version du caractère est défini automatiquement lorsque vous le générez avec l’éditeur de caractères Microsoft Agent.
