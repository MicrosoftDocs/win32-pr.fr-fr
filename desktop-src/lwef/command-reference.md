# <a name="command-reference-linguistic-information-sound-editing-tool"></a>Référence de commande (outil d’édition des informations linguistiques)

\[Microsoft Agent est déconseillé à partir de Windows 7 et peut ne pas être disponible dans les versions ultérieures de Windows.\]

### <a name="the-file-menu"></a>Menu fichier

<dl> <dt>

<span id="New"></span><span id="new"></span><span id="NEW"></span>**Nouveau**
</dt> <dd>

Réinitialise l’éditeur de sons pour la création d’un nouveau fichier audio amélioré. Si un fichier son existant est chargé et que des modifications n’ont pas été enregistrées, l’éditeur Sound affiche un message pour déterminer s’il faut enregistrer ou ignorer les modifications non enregistrées.

</dd> <dt>

<span id="Open"></span><span id="open"></span><span id="OPEN"></span>**Afficher**
</dt> <dd>

Affiche la boîte de dialogue Ouvrir, qui vous permet d’ouvrir un fichier audio existant. Si un fichier son existant est chargé et que des modifications n’ont pas été enregistrées, l’éditeur Sound affiche un message pour déterminer s’il faut enregistrer ou ignorer les modifications non enregistrées.

</dd> <dt>

<span id="Save"></span><span id="save"></span><span id="SAVE"></span>**Été**
</dt> <dd>

Enregistre un fichier audio. Si le fichier audio n’existe pas (n’a pas été nommé), l’éditeur Sound affiche la boîte de dialogue Enregistrer sous pour l’entrée du nom de fichier.

</dd> <dt>

<span id="Save_As"></span><span id="save_as"></span><span id="SAVE_AS"></span>**Enregistrer sous**
</dt> <dd>

Affiche la boîte de dialogue Enregistrer sous, qui vous permet d’entrer un nouveau nom pour le fichier audio.

</dd> <dt>

<span id="Save_Selection_As"></span><span id="save_selection_as"></span><span id="SAVE_SELECTION_AS"></span>**Enregistrer la sélection sous**
</dt> <dd>

Affiche la boîte de dialogue Enregistrer la sélection sous, qui vous permet d’entrer un nom pour la partie sélectionnée du fichier son.

</dd> <dt>

<span id="Most_Recently_Open_Files"></span><span id="most_recently_open_files"></span><span id="MOST_RECENTLY_OPEN_FILES"></span>**Fichiers ouverts le plus récemment**
</dt> <dd>

Effectue le suivi des fichiers de définition de caractères récents que vous avez ouverts. Le choix d’un fichier ouvre automatiquement ce fichier à des fins de modification. Si un caractère existant est chargé et que des modifications ne sont pas enregistrées dans un fichier, l’éditeur Sound affiche un message pour déterminer s’il faut enregistrer ou ignorer les modifications non enregistrées.

</dd> <dt>

<span id="Exit"></span><span id="exit"></span><span id="EXIT"></span>**Terminer**
</dt> <dd>

Quitte l’éditeur de sons. Si un fichier existant est chargé et que des modifications n’ont pas été enregistrées, l’éditeur Sound affiche un message pour déterminer s’il faut enregistrer ou ignorer les modifications non enregistrées.

</dd> </dl>

### <a name="the-edit-menu"></a>Le menu Edition

<dl> <dt>

<span id="Undo"></span><span id="undo"></span><span id="UNDO"></span>**Uler**
</dt> <dd>

Supprime une modification apportée dans l’éditeur de sons.

</dd> <dt>

<span id="Redo"></span><span id="redo"></span><span id="REDO"></span>**Rétabli**
</dt> <dd>

Inverse une action d’annulation dans l’éditeur de sons.

</dd> <dt>

<span id="Cut"></span><span id="cut"></span><span id="CUT"></span>**Réduis**
</dt> <dd>

Supprime le texte sélectionné et le place dans le presse-papiers.

</dd> <dt>

<span id="Copy"></span><span id="copy"></span><span id="COPY"></span>**Reprographie**
</dt> <dd>

Copie le texte sélectionné dans le Presse-papiers.

</dd> <dt>

<span id="Paste"></span><span id="paste"></span><span id="PASTE"></span>**Insérer**
</dt> <dd>

Copie le texte dans le presse-papiers au point d’insertion ou à la sélection dans la zone de texte représentation textuelle.

</dd> <dt>

<span id="Delete"></span><span id="delete"></span><span id="DELETE"></span>**Supprimer**
</dt> <dd>

Supprime le texte sélectionné.

</dd> <dt>

<span id="Select_All"></span><span id="select_all"></span><span id="SELECT_ALL"></span>**Sélectionner tout**
</dt> <dd>

Sélectionne le texte dans la zone de texte représentation textuelle.

</dd> <dt>

<span id="Generate_Linguistic_Info"></span><span id="generate_linguistic_info"></span><span id="GENERATE_LINGUISTIC_INFO"></span>**Générer des informations linguistiques**
</dt> <dd>

Commence à générer des informations de saut de mot et de phonème pour un fichier audio.

</dd> <dt>

<span id="Insert_Phoneme"></span><span id="insert_phoneme"></span><span id="INSERT_PHONEME"></span>**Insérer un phonème**
</dt> <dd>

Affiche la boîte de dialogue Insérer un phonème qui vous permet d’insérer une étiquette de phonème sélectionnée.

</dd> <dt>

<span id="Replace_Phoneme"></span><span id="replace_phoneme"></span><span id="REPLACE_PHONEME"></span>**Remplacer phonème**
</dt> <dd>

Affiche la boîte de dialogue remplacer phonème qui vous permet de remplacer l’étiquette de phonème sélectionnée.

</dd> <dt>

<span id="Delete_Phoneme"></span><span id="delete_phoneme"></span><span id="DELETE_PHONEME"></span>**Supprimer le phonème**
</dt> <dd>

Supprime l’étiquette de phonème sélectionnée.

</dd> <dt>

<span id="Insert_Word"></span><span id="insert_word"></span><span id="INSERT_WORD"></span>**Insérer un mot**
</dt> <dd>

Affiche la boîte de dialogue Insérer un mot qui vous permet d’insérer une étiquette de mot dans la représentation audio.

</dd> <dt>

<span id="Replace_Word"></span><span id="replace_word"></span><span id="REPLACE_WORD"></span>**Remplacer le mot**
</dt> <dd>

Affiche la boîte de dialogue remplacer le mot qui vous permet de remplacer l’étiquette de mot sélectionnée dans la représentation audio.

</dd> <dt>

<span id="Delete_Word"></span><span id="delete_word"></span><span id="DELETE_WORD"></span>**Supprimer le mot**
</dt> <dd>

Supprime l’étiquette de mot sélectionnée dans la représentation audio.

</dd> <dt>

<span id="Phoneme_Label_Display"></span><span id="phoneme_label_display"></span><span id="PHONEME_LABEL_DISPLAY"></span>**Affichage des étiquettes de phonème**
</dt> <dd>

Modifie l’affichage de l’étiquette de phonème entre les noms descriptifs et les valeurs d’octets de la Loi.

</dd> <dt>

<span id="Speech_Engine"></span><span id="speech_engine"></span><span id="SPEECH_ENGINE"></span>**Moteur de reconnaissance vocale**
</dt> <dd>

Vous permet de modifier le moteur de reconnaissance vocale que vous utilisez pour générer les informations de césure et de phonème.

</dd> </dl>

### <a name="the-audio-menu"></a>Menu audio

<dl> <dt>

<span id="Play"></span><span id="play"></span><span id="PLAY"></span>**Répétition**
</dt> <dd>

Lit le fichier audio ou la partie sélectionnée du fichier son.

</dd> <dt>

<span id="Record"></span><span id="record"></span><span id="RECORD"></span>**Enregistrement**
</dt> <dd>

Enregistre un nouveau fichier audio.

</dd> <dt>

<span id="Pause"></span><span id="pause"></span><span id="PAUSE"></span>**Suspen**
</dt> <dd>

Suspend la lecture du fichier audio ou de la partie sélectionnée du fichier son. Utilisez Play pour reprendre la lecture.

</dd> <dt>

<span id="Stop"></span><span id="stop"></span><span id="STOP"></span>**Erreur**
</dt> <dd>

Arrête l’enregistrement ou la diffusion du fichier audio ou de la partie sélectionnée du fichier son.

</dd> </dl>
