# <a name="the-microsoft-agent-user-interface"></a>Interface utilisateur de l’agent Microsoft

\[Microsoft Agent est déconseillé à partir de Windows 7 et peut ne pas être disponible dans les versions ultérieures de Windows.\]

Microsoft Agent permet aux sites Web et aux applications conventionnelles d’inclure une forme améliorée d’interaction utilisateur. Il fournit plusieurs composants d’interface utilisateur qui permettent aux utilisateurs d’accéder au caractère et d’interagir avec celui-ci, de connaître l’état du caractère et de modifier les paramètres globaux qui affectent tous les caractères.

Les sections suivantes décrivent ces éléments de base de l’interface utilisateur de Microsoft Agent :

-   [Fenêtre de caractères](the-character-window.md)
-   [Icône caractère de la barre des tâches](the-character-taskbar-icon.md)
-   [Fenêtre commandes vocales](the-voice-commands-window.md)
-   [La bulle de mot](the-word-balloon.md)
-   [Fenêtre de l’info-bulle d’écoute](the-listening-tip-window.md)
-   [La fenêtre Options de caractères avancés](the-advanced-character-options-window.md)
-   [La fenêtre Propriétés des caractères par défaut](the-default-character-properties-window.md)
