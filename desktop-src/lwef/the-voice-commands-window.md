<span data-ttu-id="afbd3-104">\[Microsoft Agent est déconseillé à partir de Windows 7 et peut ne pas être disponible dans les versions ultérieures de Windows.\]</span><span class="sxs-lookup"><span data-stu-id="afbd3-104">\[Microsoft Agent is deprecated as of Windows 7, and may be unavailable in subsequent versions of Windows.\]</span></span>

<span data-ttu-id="afbd3-105">Si un moteur de reconnaissance vocale compatible est installé, Microsoft Agent fournit une fenêtre spéciale, appelée **fenêtre commandes vocales** , qui affiche les commandes qui ont été activées pour la reconnaissance vocale.</span><span class="sxs-lookup"><span data-stu-id="afbd3-105">If a compatible speech engine is installed, Microsoft Agent supplies a special window called the **Voice Commands Window** that displays the commands that have been voice-enabled for speech recognition.</span></span> <span data-ttu-id="afbd3-106">La **fenêtre commandes vocales** sert d’invite visuelle sur ce qui peut être parlé comme entrée (les commandes ne peuvent pas être sélectionnées avec la souris).</span><span class="sxs-lookup"><span data-stu-id="afbd3-106">The **Voice Commands Window** serves as a visual prompt for what can be spoken as input (commands cannot be selected with the mouse).</span></span>

![boîte de dialogue commandes vocales](images/f2voice.gif)

<span data-ttu-id="afbd3-108">La fenêtre s’affiche lorsqu’un utilisateur sélectionne la commande **ouvrir la fenêtre commandes vocales** , soit en parlant la commande, soit en cliquant avec le bouton droit sur le caractère et en choisissant la commande dans le menu contextuel du caractère.</span><span class="sxs-lookup"><span data-stu-id="afbd3-108">The window appears when a user selects the **Open Voice Commands Window** command, either by speaking the command or right-clicking the character and choosing the command from the character's pop-up menu.</span></span> <span data-ttu-id="afbd3-109">Toutefois, si l’utilisateur désactive l’entrée vocale, la **fenêtre commandes vocales** n’est pas accessible.</span><span class="sxs-lookup"><span data-stu-id="afbd3-109">However, if the user disables speech input, the **Voice Commands Window** is not accessible.</span></span>

<span data-ttu-id="afbd3-110">La **fenêtre commandes vocales** affiche des commandes vocales sous forme d’arborescence.</span><span class="sxs-lookup"><span data-stu-id="afbd3-110">The **Voice Commands Window** displays voice-enabled commands as a tree.</span></span> <span data-ttu-id="afbd3-111">Si l’application d’hébergement actuelle fournit des commandes vocales, elles apparaissent développées et en haut de la fenêtre.</span><span class="sxs-lookup"><span data-stu-id="afbd3-111">If the current hosting application supplies voice commands, they appear expanded and at the top of the window.</span></span> <span data-ttu-id="afbd3-112">Les entrées apparaissent également pour les autres applications qui utilisent le caractère.</span><span class="sxs-lookup"><span data-stu-id="afbd3-112">Entries also appear for other applications using the character.</span></span> <span data-ttu-id="afbd3-113">La fenêtre comprend également les commandes vocales globales fournies par Microsoft Agent.</span><span class="sxs-lookup"><span data-stu-id="afbd3-113">The window also includes the global voice commands supplied by Microsoft Agent.</span></span> <span data-ttu-id="afbd3-114">Si l’application d’hébergement actuelle n’a pas de commandes vocales, les commandes vocales globales s’affichent en haut et en haut de la fenêtre.</span><span class="sxs-lookup"><span data-stu-id="afbd3-114">If the current hosting application has no voice commands, the global voice commands appear expanded and at the top of the window.</span></span>

<span data-ttu-id="afbd3-115">L’utilisateur peut redimensionner et déplacer la **fenêtre commandes vocales**.</span><span class="sxs-lookup"><span data-stu-id="afbd3-115">The user can size and move the **Voice Commands Window**.</span></span> <span data-ttu-id="afbd3-116">Microsoft Agent se souvient du dernier emplacement de la fenêtre et l’affiche à cet emplacement si l’utilisateur ferme et ouvre à nouveau la fenêtre.</span><span class="sxs-lookup"><span data-stu-id="afbd3-116">Microsoft Agent remembers the last location of the window and redisplays it at that location if the user closes and re-opens the window.</span></span> <span data-ttu-id="afbd3-117">Si les entrées de la fenêtre dépassent la taille d’affichage actuelle de la fenêtre, les barres de défilement apparaissent.</span><span class="sxs-lookup"><span data-stu-id="afbd3-117">If the entries in the window exceed the current display size of the window, scroll bars appear.</span></span>
