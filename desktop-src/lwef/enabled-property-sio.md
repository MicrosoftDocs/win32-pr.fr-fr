# <a name="enabled-property-speech-input-object"></a>Propriété Enabled (objet d’entrée vocale)

\[Microsoft Agent est déconseillé à partir de Windows 7 et peut ne pas être disponible dans les versions ultérieures de Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Descriptive**
</dt> <dd>

Retourne une valeur booléenne indiquant si l’entrée vocale est activée.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Stockéesyntaxe**
</dt> <dd>

*agent * * *. SpeechInput. Enabled**



| Valeur     | Description               |
|-----------|---------------------------|
| **True**  | L’entrée vocale est activée.  |
| **False** | L’entrée vocale est désactivée. |



 

</dd> </dl>

## <a name="remarks"></a>Remarques

La propriété [**activé**](enabled-property.md) reflète l’option écouter les caractères de l’entrée dans la page entrée vocale de la feuille de propriétés de l’agent (options des caractères avancés). Le paramètre de propriété affecte tous les caractères de l’agent et est en lecture seule ; seul l’utilisateur peut modifier cette propriété.
