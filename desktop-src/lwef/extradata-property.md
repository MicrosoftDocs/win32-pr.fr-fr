# <a name="extradata-property"></a>Propriété ExtraData

\[Microsoft Agent est déconseillé à partir de Windows 7 et peut ne pas être disponible dans les versions ultérieures de Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Descriptive**
</dt> <dd>

Retourne une chaîne qui spécifie des données supplémentaires stockées dans le cadre du caractère.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Stockéesyntaxe**
</dt> <dd>

*agent*. **Caractères («**_CharacterID_*_»). ExtraData_*

</dd> </dl>

## <a name="remarks"></a>Remarques

La valeur par défaut de la propriété **ExtraData** d’un caractère est définie lorsque le caractère est compilé à l’aide de l’éditeur de caractères Microsoft Agent. Elle ne peut pas être modifiée ou spécifiée au moment de l’exécution.

> [!Note]  
> Le paramètre de la propriété **ExtraData** est facultatif et ne peut pas être fourni pour tous les caractères.
