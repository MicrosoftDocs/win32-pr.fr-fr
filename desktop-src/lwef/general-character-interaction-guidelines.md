# <a name="general-character-interaction-guidelines"></a>Instructions d’interaction de caractères générales

\[Microsoft Agent est déconseillé à partir de Windows 7 et peut ne pas être disponible dans les versions ultérieures de Windows.\]

Les principes de bonne conception de l’interface utilisateur s’appliquent également à la conception d’une interface à l’aide de Microsoft Agent. Vous devez comprendre vos utilisateurs et leurs objectifs, ainsi que suivre un processus de conception de l’interface utilisateur. Les instructions relatives à l’interface Windows pour la conception de logiciels fournissent une excellente vue d’ensemble des principes de conception généraux et de la méthodologie.

-   [Être non exclusif](be-non-exclusive.md)
-   [Fournir les commentaires appropriés](provide-appropriate-feedback.md)
-   [Utiliser la variation naturelle](use-natural-variation.md)
