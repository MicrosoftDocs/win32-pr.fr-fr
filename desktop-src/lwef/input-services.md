# <a name="input-services"></a>Services d’entrée

\[Microsoft Agent est déconseillé à partir de Windows 7 et peut ne pas être disponible dans les versions ultérieures de Windows.\]

Une application cliente fournit l’interface utilisateur principale pour l’interaction avec un caractère. Vous pouvez programmer un caractère pour répondre à toute forme d’entrée, d’un clic de bouton à un texte tapé. En outre, Microsoft Agent fournit des événements qui vous permettent de programmer ce qui se produit lorsque l’utilisateur clique sur, double-clique ou fait glisser le caractère. Le serveur passe les coordonnées du pointeur et tout état de touche de modification pour ces événements.
