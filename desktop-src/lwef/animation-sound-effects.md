# <a name="animation-sound-effects"></a>Effets sonores d’animation

\[Microsoft Agent est déconseillé à partir de Windows 7 et peut ne pas être disponible dans les versions ultérieures de Windows.\]

Microsoft agent vous permet également d’inclure des effets sonores dans le cadre de l’animation d’un caractère. à l’aide de l’éditeur de caractères Microsoft Agent, vous pouvez spécifier le nom de fichier du Windows standard (. WAV) à déclencher sur un frame donné. Notez que Microsoft Agent ne mélange pas les effets sonores et la sortie parlée, de sorte que la sortie vocale ne commence pas tant que l’effet sonore n’est pas terminé. Par conséquent, évitez tout effet sonore long ou en boucle dans le cadre de l’animation d’un caractère.
