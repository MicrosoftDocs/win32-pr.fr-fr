# <a name="the-office-animation-set"></a>ensemble d’animations Office

\[Microsoft Agent est déconseillé à partir de Windows 7 et peut ne pas être disponible dans les versions ultérieures de Windows.\]

le tableau suivant répertorie les animations définies pour les Microsoft Office 2000 caractères. si vous envisagez d’utiliser votre caractère dans Microsoft Office, vous devez prendre en charge toutes les animations de ce tableau. en outre, vous pouvez ajouter toutes les autres animations que vous vivez, mais gardez à l’esprit que Microsoft Office ne les appelle pas. Les animations avec des astérisques ( \* ) doivent être de 100% en boucle. Les autres animations doivent être brèves.



| Animation               | État de l'agent    | Exemple de cas d’utilisation                                                                                 | Exemples d’animation spécifiques                                                                                                               |
|-------------------------|----------------|------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------|
| **Alert**               | Aucun           | Quand le caractère veut alerter l’utilisateur                                                           | Le caractère se présente à l’utilisateur.                                                                                                              |
| **CheckingSomething\*** | Aucun           | Vérification orthographique, vérification de la grammaire                                                                            | Le caractère ressemble à un livre de référence                                                                                          |
| **Féliciter**        | Aucun           | Terminer un Assistant                                                                                    | Big sourire, Look of relief, fatigué, mais heureux                                                                                                 |
| **EmptyTrash**          | Aucun           | La corbeille est vidée dans Outlook                                                                          | Eclairages de caractères sur le feu                                                                                                        |
| **Expliquer**             | Aucun           | Quand le caractère veut expliquer un événement à l’utilisateur                                            | Semble brièvement mais attentively à l’utilisateur, puis                                                                                     |
| **GestureDown**         | GesturingDown  | Caractère désignant un caractère à l’écran                                                         | Le caractère examine l’utilisateur, puis pointe et regarde l’écran                                                                           |
| **GestureLeft**         | GesturingLeft  | Caractère désignant un élément à l’écran, tel qu’une rubrique d’aide ou une partie de l’interface utilisateur                  | Le caractère examine l’utilisateur, puis pointe et regarde l’écran                                                                           |
| **GestureRight**        | GesturingRight | « Présentation » d’une rubrique d’aide ou d’une boîte de dialogue                                                                  | Le caractère examine l’utilisateur, puis pointe et regarde l’écran                                                                           |
| **GestureUp**           | GesturingUp    | Caractère désignant un caractère à l’écran                                                         | Le caractère examine l’utilisateur, puis pointe et regarde l’écran                                                                           |
| **GetArtsy\***          | Aucun           | AutoFormat                                                                                           | Le caractère place sur Beret, contient la palette et peint                                                                                        |
| **GetAttention**        | Aucun           | Conseil haute priorité                                                                                    | Des gestes fortement pour attirer l’attention de l’utilisateur ; par exemple, les sauts vers le haut et vers le haut                                                 |
| **GetTechy**            | Aucun           | S’exécute dans l’environnement de programmation                                                                | Le caractère extrait une calculatrice ou un fer de soudure                                                                                          |
| **GetWizardy\***        | Aucun           | Assistant graphique en cours d’exécution alors que le caractère est visible (action redéclenchée avec chaque nouveau panneau de l’Assistant)        | Insertion de caractères sur le chapeau et les vagues de l’Assistant                                                                                               |
| **Fini**             | Aucun           | Un autre caractère est choisi                                                                          | Il s’agit d’une disparition complexe qui commence dans **RestPose** et se termine par un cadre vide                                                      |
| **Greeting**            | Aucun           | Le caractère est choisi                                                                                  | Il s’agit d’un aspect complexe qui commence par un cadre vide et se termine par **RestPose**                                                      |
| **Audition \_ 1\***        | Aucun           | Fichier de longue durée ouvert                                                                                    | De l’oreille au sol, en écoutant l’ordinateur                                                                                              |
| **Masquer**                | Masquage         | Les caractères quittent temporairement                                                                         | Laisse rapidement dans le souffle de fumée                                                                                                         |
| **Idle1 \_ 1**            | Aucune entrée utilisateur  | Écoute activement, puis se met en veille. (possibilité d’afficher la personnalité des caractères) | Clignotement, recherche en attente, patienter                                                                                               |
| **Idle2**               | Aucune entrée utilisateur  | Périodes d’inactivité plus longues                                                                                  | Caractère réactions et semble en veille                                                                                                          |
| **Idle3**               | Aucune entrée utilisateur  | Inactif profond (lorsque le caractère est resté inactif pendant une longue période)                                         | Le caractère passe en mode veille                                                                                                                   |
| **IdleHit**             | Aucun           | Il s’agit d’un échantillon représentatif non mappé d’animations de niveau inactif                                | Toutes les animations inactives                                                                                                                |
| **LookDown**            | Aucun           | Examine brièvement                                                                                   | Remarque : une ligne est insérée et vue d’ensemble                                                                                               |
| **LookDownLeft**        | Aucun           | Recherche vers le dessous et vers la gauche                                                                          | Remarque : une ligne est insérée et vue d’ensemble                                                                                               |

| **LookUpRight**         | Aucun           | Semble juste et rapide                                                                           | L’utilisateur clique sur le bouton de la barre d’outils des avis (caractère non surpris)                                   |
| **Imprimer**               | Aucun           | Impression d’une page d’un travail d’impression                                                                       | Récupère un morceau de papier et l’envoie à l’imprimante.                                                                                 |
| **Traitement en cours\***        | Aucun           | Action générale pour laquelle nous n’avons pas d’action de caractère spécifique                                     | Le personnage est l’œil de la concentration et tire un marteau au marteau. L’animation doit avoir une entrée rapide dans une boucle, puis une sortie rapide |
| **RestPose**            | Aucun           | Utilisé lorsque le caractère ne lit pas une animation                                                   | Image de l’Assistant                                                                                                                 |
| **Enregistrer\***              | Aucun           | Utilisé pendant une opération d’enregistrement de fichier                                                                    | Le caractère place un tout dans un coffre                                                                                                     |
| **Dans\***         | Aucun           | Utilisé pour la recherche, le vérificateur d’orthographe et la vérification grammaticale                                                        | La tête s’active et regarde à nouveau le document. L’animation doit avoir une entrée rapide dans une boucle, puis une sortie rapide                                 |
| **SendMail**            | Aucun           | Envoi de courrier électronique                                                                                         | Extrait une lettre et la place dans une boîte aux lettres                                                                                             |
| **Afficher**                | Affichage        | Retours de caractères de Brief-départ                                                                   | Ressorts rapidement à l’étape, rapidement                                                                                                         |
| **Opération en cours\***          | Aucun           | Réalisation d’un calcul complexe, tel que le solveur                                                          | Le caractère semble vers le haut et les éraflures. L’animation doit avoir une entrée rapide dans une boucle, puis une sortie rapide                             |
| **Wave**                | Aucun           | Alertes associées                                                                                  | Wave. Semblable à Alert, mais pas à long ou affolé                                                                                     |
| **Écriture\***           | Aucun           | Le client modifie les options des outils. demande de saisie IntelliSearch par le client                   | Extrait le bloc et démarre le griffonnage. L’animation doit avoir une entrée rapide dans une boucle, puis une sortie rapide                                   |
