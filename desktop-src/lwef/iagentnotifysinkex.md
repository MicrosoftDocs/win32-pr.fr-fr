# <a name="iagentnotifysinkex"></a>IAgentNotifySinkEx

\[Microsoft Agent est déconseillé à partir de Windows 7 et peut ne pas être disponible dans les versions ultérieures de Windows.\]

**IAgentNotifySinkEx** est dérivé de l’interface [**IAgentNotifySink**](https://www.bing.com/search?q=**IAgentNotifySink**) . Il inclut toutes les méthodes **IAgentNotifySink** et permet d’accéder à des fonctions supplémentaires.

**Méthodes dans l'ordre Vtable**



| Méthodes IAgentNotifySinkEx                                                   | Description                                                        |
|------------------------------------------------------------------------------|--------------------------------------------------------------------|
| [**HelpComplete**](iagentnotifysinkex--helpcomplete.md)                     | Se produit lorsque l’utilisateur sélectionne un menu ou le caractère en mode d’aide. |
| [**ListeningState**](iagentnotifysinkex--listeningstate.md)                 | Se produit lorsque l’état d’écoute du caractère change.               |
| [**DefaultCharacterChange**](iagentnotifysinkex--defaultcharacterchange.md) | Se produit lorsque l’utilisateur modifie le caractère par défaut.                |
| [**AgentPropertyChange**](iagentnotifysinkex--agentpropertychange.md)       | Se produit lorsque l’utilisateur modifie un paramètre de propriété de l’agent.            |
| [**ActiveClientChange**](iagentnotifysinkex--activeclientchange.md)         | Se produit lorsque le client actif d’un caractère change.              |
