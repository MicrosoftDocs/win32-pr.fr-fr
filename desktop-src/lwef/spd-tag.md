# <a name="spd-tag"></a>Balise SPD

\[Microsoft Agent est déconseillé à partir de Windows 7 et peut ne pas être disponible dans les versions ultérieures de Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Descriptive**
</dt> <dd>

Définit la vitesse moyenne de la ligne de base de la sortie vocale.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Stockéesyntaxe**
</dt> <dd>

**\\SPD =***nombre***\\**



| Partie     | Description                                          |
|----------|------------------------------------------------------|
| *number* | Vitesse moyenne de la conversation de base, en mots par minute. |



 

</dd> </dl>

### <a name="remarks"></a>Notes

Cette balise est prise en charge uniquement pour la sortie générée par TTS. La plage de valeurs du paramètre peut varier selon le moteur TTS installé.
