# <a name="the-listening-tip-window"></a>Fenêtre de l’info-bulle d’écoute

\[Microsoft Agent est déconseillé à partir de Windows 7 et peut ne pas être disponible dans les versions ultérieures de Windows.\]

Si la reconnaissance vocale est activée, une fenêtre d’info-bulle spéciale s’affiche lorsque l’utilisateur appuie sur la touche Push-to-débats pour commencer l’entrée vocale. Le Conseil d’écoute affiche des informations contextuelles relatives à l’état actuel de l’entrée de Microsoft Agent. Si la reconnaissance vocale compatible n’a pas été installée ou a été désactivée, le Conseil d’écoute n’apparaît pas.
