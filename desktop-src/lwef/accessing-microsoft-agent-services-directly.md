# <a name="accessing-microsoft-agent-services-directly"></a>Accès direct aux services Microsoft Agent

\[Microsoft Agent est déconseillé à partir de Windows 7 et peut ne pas être disponible dans les versions ultérieures de Windows.\]

Si vous utilisez C, C++ ou Java, vous pouvez accéder au serveur Microsoft agent directement à l’aide de ses interfaces ActiveX (OLE). Pour plus d’informations sur ces interfaces, consultez [programmation de l’interface du serveur Microsoft Agent](programming-the-microsoft-agent-server-interface.md).
