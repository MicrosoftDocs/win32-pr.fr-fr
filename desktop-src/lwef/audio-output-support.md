# <a name="audio-output-support"></a>Prise en charge de la sortie audio

\[Microsoft Agent est déconseillé à partir de Windows 7 et peut ne pas être disponible dans les versions ultérieures de Windows.\]

Microsoft agent vous permet d’utiliser des fichiers audio pour la sortie orale d’un caractère. Vous pouvez enregistrer des fichiers audio et utiliser la méthode [**Speak**](speak-method.md) pour lire ces données. Les services d’animation Microsoft Agent prennent automatiquement en charge la synchronisation de l’embouchure des caractères à l’aide des caractéristiques audio du fichier audio. Microsoft agent prend également en charge un format spécial pour les fichiers audio, qui comprend des informations de phonème et de césure supplémentaires pour la prise en charge améliorée de la synchronisation Lip. Vous pouvez générer ce format spécial à l’aide de l’outil de modification du son de l’information linguistique Microsoft.
