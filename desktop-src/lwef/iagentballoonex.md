# <a name="iagentballoonex"></a>IAgentBalloonEx

\[Microsoft Agent est déconseillé à partir de Windows 7 et peut ne pas être disponible dans les versions ultérieures de Windows.\]

[**IAgentBalloonEx**](/windows/desktop/lwef/iagentballoon) est dérivé de l’interface **IAgentBalloon** . Il inclut toutes les méthodes **IAgentBalloon** et permet d’accéder à des fonctions supplémentaires.

**Méthodes dans l'ordre Vtable**



| Méthodes IAgentBalloonEx                                  | Description                                                        |
|----------------------------------------------------------|--------------------------------------------------------------------|
| [**GetStyle**](https://www.bing.com/search?q=**GetStyle**)                     | Retourne le style de sortie de l’info-bulle du mot.                           |
| [**SetStyle**](https://www.bing.com/search?q=**SetStyle**)                     | Définit le style de sortie de la bulle de texte.                              |
| [**SetNumLines**](https://www.bing.com/search?q=**SetNumLines**)               | Définit la sortie des nombres dans la bulle de texte.                  |
| [**SetNumCharsPerLine**](https://www.bing.com/search?q=**SetNumCharsPerLine**) | Définit le nombre de caractères par ligne de sortie dans la bulle de mot. |
