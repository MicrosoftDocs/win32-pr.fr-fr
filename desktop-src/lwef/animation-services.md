# <a name="animation-services"></a>Services d’animation

\[Microsoft Agent est déconseillé à partir de Windows 7 et peut ne pas être disponible dans les versions ultérieures de Windows.\]

Les services d’animation de Microsoft Agent gèrent l’animation et le déplacement de l’image d’un caractère dans sa propre fenêtre à l’écran. Une animation est définie sous la forme d’une séquence de frames chronométrés et éventuellement de branches, composée d’une ou de plusieurs images.

-   [Chargement d’un caractère](loading-a-character.md)
-   [Chargement du caractère par défaut](loading-the-default-character.md)
-   [Animation d’un caractère](animating-a-character.md)
