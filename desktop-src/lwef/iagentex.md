# <a name="iagentex"></a>IAgentEx

\[Microsoft Agent est déconseillé à partir de Windows 7 et peut ne pas être disponible dans les versions ultérieures de Windows.\]

**IAgentEx** dérive de l’interface [**IAgent**](iagent.md) . Il inclut toutes les méthodes **IAgent** et permet d’accéder à des fonctions supplémentaires.

**Méthodes dans l'ordre Vtable**



| Méthodes IAgentEx                                                                   | Description                                              |
|------------------------------------------------------------------------------------|----------------------------------------------------------|
| [**ShowDefaultCharacterProperties**](iagentex--showdefaultcharacterproperties.md) | Affiche les propriétés de caractère par défaut.               |
| [**GetVersion**](iagentex--getversion.md)                                         | Retourne le numéro de version de Microsoft Agent (serveur). |
