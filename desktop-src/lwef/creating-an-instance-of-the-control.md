# <a name="creating-an-instance-of-the-control"></a>Création d’une instance du contrôle

\[Microsoft Agent est déconseillé à partir de Windows 7 et peut ne pas être disponible dans les versions ultérieures de Windows.\]

Pour accéder au contrôle de Microsoft Agent, vous devez d’abord créer une instance du contrôle. Le moyen le plus simple consiste à placer une instance du contrôle sur un formulaire. Vous devrez peut-être ajouter le contrôle à votre boîte à outils avant de l’ajouter à votre formulaire. En fonction de votre langage de programmation, vous pouvez également être en mesure de créer le contrôle au moment de l’exécution.
