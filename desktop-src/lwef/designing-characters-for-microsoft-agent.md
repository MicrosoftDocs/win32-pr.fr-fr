# <a name="designing-characters-for-microsoft-agent"></a>Conception de caractères pour Microsoft Agent

\[Microsoft Agent est déconseillé à partir de Windows 7 et peut ne pas être disponible dans les versions ultérieures de Windows.\]

Ce document fournit des informations qui peuvent vous aider à concevoir et à développer un caractère utilisable avec Microsoft Agent. Il comprend des informations techniques et conceptuelles sur la conception de caractères, d’images et d’animations. la taille, l’utilisation de la couleur et les types d’images que vous devez créer ; animations suggérées ; animations parlantes ; et les États de l’agent. Des principes d’animation efficaces sont également décrits pour votre utilisation lors de la création de caractères animés visuellement convaincants.

Les sections suivantes décrivent comment concevoir des caractères pour Microsoft Agent :

-   [Caractères](characters.md)
-   [Animations](animations.md)
-   [États de l’agent](agent-states.md)
-   [Principes d’animation](animation-principles.md)
