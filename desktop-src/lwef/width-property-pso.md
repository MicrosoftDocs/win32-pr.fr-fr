# <a name="width-property-propertysheet-object"></a>Width, propriété (objet feuille)

\[Microsoft Agent est déconseillé à partir de Windows 7 et peut ne pas être disponible dans les versions ultérieures de Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Descriptive**
</dt> <dd>

Retourne une valeur entière spécifiant la largeur actuelle, en pixels, de la fenêtre de la feuille de propriétés de l’agent Microsoft.

</dd> <dt>

<span id="Syntax_"></span><span id="syntax_"></span><span id="SYNTAX_"></span>**Stockéesyntaxe** 
</dt> <dd>

*agent * * *. Feuille. Width**

</dd> </dl>

## <a name="remarks"></a>Notes

Le serveur affiche la fenêtre en fonction de l’emplacement défini par l’utilisateur.
