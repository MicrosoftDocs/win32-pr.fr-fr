# <a name="starting-the-sound-editor"></a>Démarrage de l’éditeur Sound

\[Microsoft Agent est déconseillé à partir de Windows 7 et peut ne pas être disponible dans les versions ultérieures de Windows.\]

Pour exécuter l’outil de modification du son des informations linguistiques de Microsoft, choisissez-le dans le menu **Démarrer** ou double-cliquez sur l’icône de l’éditeur de sons. La fenêtre de l’éditeur de sons s’ouvre et affiche ses menus, une barre d’outils pour les commandes fréquemment utilisées, une zone de texte permettant d’entrer les mots utilisés par l’éditeur pour traiter le fichier audio et une zone d’affichage pour l’affichage et la modification des données audio et linguistiques.

![Capture d’écran montrant la page de démarrage de l’outil de modification du son des informations linguistiques Microsoft.](images/f1listliq.gif)

Après le démarrage de l’éditeur de sons, vous pouvez commencer à enregistrer un nouveau fichier audio ou charger un fichier audio existant.
