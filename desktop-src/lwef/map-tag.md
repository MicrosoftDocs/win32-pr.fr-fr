# <a name="map-tag"></a>Étiquette de la carte

\[Microsoft Agent est déconseillé à partir de Windows 7 et peut ne pas être disponible dans les versions ultérieures de Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Descriptive**
</dt> <dd>

Cartes texte parlé dans le texte affiché dans la bulle.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Stockéesyntaxe**
</dt> <dd>

\\**Map**= "SpokenText" = "balloontext"\\



| Partie          | Description                                           |
|---------------|-------------------------------------------------------|
| *spokentext*  | Chaîne spécifiant le texte pour la sortie parlée.       |
| *balloontext* | Chaîne spécifiant le texte de la sortie de la bulle de texte. |



 

</dd> </dl>

### <a name="remarks"></a>Remarques

Cette balise vous permet d’utiliser un texte parlé différent de celui affiché dans la bulle de texte.
