# <a name="using-the-microsoft-agent-character-editor"></a>Utilisation de l’éditeur de caractères Microsoft Agent

\[Microsoft Agent est déconseillé à partir de Windows 7 et peut ne pas être disponible dans les versions ultérieures de Windows.\]

L’éditeur de caractères Microsoft agent vous permet de compiler des animations de caractères pour une utilisation avec Microsoft Agent. Vous pouvez définir des animations en important des images bitmap Windows, en définissant leur durée et éventuellement en apportant des branchements, des effets sonores et des superpositions de parole. Pour plus d’informations sur la conception des animations d’un caractère, consultez [conception de caractères pour Microsoft Agent](designing-characters-for-microsoft-agent.md).

Avec l’éditeur de caractères de l’agent, vous pouvez définir le nom et la description du caractère ainsi que les options de sortie, notamment la conversion de texte par synthèse vocale, la sortie vocale synthétisée, la prise en charge des menus contextuels et la conception des bulles de caractères du caractère.

Ce document décrit l’éditeur de caractères Microsoft Agent pour Microsoft Agent que vous pouvez télécharger à partir de <https://www.microsoft.com/downloads/details.aspx?displaylang=en&FamilyID=c21be0ec-1712-427d-a43d-94c122ef8eb6> .
