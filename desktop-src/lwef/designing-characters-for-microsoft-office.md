# <a name="designing-characters-for-microsoft-office"></a>Conception de caractères pour Microsoft Office

\[Microsoft Agent est déconseillé à partir de Windows 7 et peut ne pas être disponible dans les versions ultérieures de Windows.\]

Ce document fournit des informations qui peuvent vous aider à concevoir et à développer un caractère Microsoft Agent à utiliser dans Microsoft Office 2000 (les caractères Microsoft Agent ne fonctionnent pas avec Microsoft Office 97). Ce document suppose que vous avez une connaissance pratique de l’éditeur de caractères Microsoft Agent et que vous savez comment créer des caractères Microsoft Agent. Vous trouverez plus d’informations sur l’éditeur de caractères de l’agent et sur la conception et le développement de caractères dans [l’éditeur de caractères Microsoft Agent](using-the-microsoft-agent-character-editor.md). Des informations supplémentaires sur la création de caractères pour Microsoft Agent sont fournies dans [conception de caractères pour Microsoft Agent](designing-characters-for-microsoft-agent.md).
