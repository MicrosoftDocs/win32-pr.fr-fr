# <a name="consider-gender-effects"></a>Prendre en compte les effets du sexe

\[Microsoft Agent est déconseillé à partir de Windows 7 et peut ne pas être disponible dans les versions ultérieures de Windows.\]

Les réponses sociales peuvent être affectées par sexe. Par exemple, les Reeves NAS et « travail » indiquent que, aux États-Unis, les personnalités masculines sont stereotypicallyes pour comprendre plus en détail les sujets techniques, tandis que les femelles sont attribuées à de meilleurs experts dans des sujets interpersonnels comme l’amour et les relations. Cela ne suggère pas de perpétuer les stéréotypes sexes, mais uniquement que vous savez qu’ils peuvent exister et comprendre comment ils peuvent affecter l’interaction avec votre caractère.

N’oubliez pas que le sexe perçu d’un caractère n’est pas uniquement établi par son nom ou son apparence. Même un caractère neutre sur le sexe peut communiquer le sexe en fonction de ses caractéristiques vocales, de ses tendances et de son mouvement.
