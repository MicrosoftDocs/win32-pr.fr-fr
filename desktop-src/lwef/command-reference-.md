# <a name="command-reference-microsoft-agent-character-editor"></a>Référence de commande (éditeur de caractères Microsoft Agent)

\[Microsoft Agent est déconseillé à partir de Windows 7 et peut ne pas être disponible dans les versions ultérieures de Windows.\]

## <a name="the-file-menu"></a>Menu fichier

<dl> <dt>

<span id="New"></span><span id="new"></span><span id="NEW"></span>**Nouveau**
</dt> <dd>

Réinitialise l’éditeur de caractères de l’agent pour la création d’une définition de caractère. Si un caractère existant est chargé et que des modifications n’ont pas été enregistrées, l’éditeur affiche un message pour déterminer s’il faut enregistrer ou ignorer les modifications non enregistrées.

</dd> <dt>

<span id="Open"></span><span id="open"></span><span id="OPEN"></span>**Afficher**
</dt> <dd>

Affiche la boîte de dialogue Ouvrir un fichier, qui vous permet d’ouvrir un fichier de définition de caractère existant pour le modifier. Si un caractère existant est chargé et que des modifications ne sont pas enregistrées dans un fichier, l’éditeur affiche un message pour déterminer s’il faut enregistrer ou ignorer les modifications non enregistrées.

</dd> <dt>

<span id="Save"></span><span id="save"></span><span id="SAVE"></span>**Été**
</dt> <dd>

Enregistre la définition de caractère. Si la définition de caractère n’existe pas (n’a pas été nommée), l’éditeur affiche la boîte de dialogue Enregistrer sous pour l’entrée du nom de fichier.

</dd> <dt>

<span id="Save_As"></span><span id="save_as"></span><span id="SAVE_AS"></span>**Enregistrer sous**
</dt> <dd>

Affiche la boîte de dialogue Enregistrer sous, qui vous permet d’entrer un nouveau nom pour le fichier de définition de caractère.

</dd> <dt>

<span id="Print"></span><span id="print"></span><span id="PRINT"></span>**Étendue**
</dt> <dd>

Affiche la boîte de dialogue Imprimer, qui vous permet de choisir une option d’impression et d’imprimer le fichier de définition des caractères.

</dd> <dt>

<span id="Build_Character"></span><span id="build_character"></span><span id="BUILD_CHARACTER"></span>**Caractère de build**
</dt> <dd>

Affiche la boîte de dialogue créer un caractère, qui comprend des options permettant de définir la manière de générer les fichiers de données et d’animation d’un caractère pour une utilisation avec Microsoft Agent.

</dd> <dt>

<span id="Page_Setup"></span><span id="page_setup"></span><span id="PAGE_SETUP"></span>**Mise en page**
</dt> <dd>

Affiche la boîte de dialogue mise en page qui vous permet de définir les options d’impression du fichier de définition de caractère.

</dd> <dt>

<span id="Most_Recently_Open_Files"></span><span id="most_recently_open_files"></span><span id="MOST_RECENTLY_OPEN_FILES"></span>**Fichiers ouverts le plus récemment**
</dt> <dd>

Effectue le suivi des fichiers de définition de caractères récents que vous avez ouverts. Le choix d’un fichier ouvre automatiquement ce fichier à des fins de modification. Si un caractère existant est chargé et que des modifications ne sont pas enregistrées dans un fichier, l’éditeur affiche un message pour déterminer s’il faut enregistrer ou ignorer les modifications non enregistrées.

</dd> <dt>

<span id="Exit"></span><span id="exit"></span><span id="EXIT"></span>**Terminer**
</dt> <dd>

Quitte l’éditeur de caractères de l’agent. Si un caractère existant est chargé et que des modifications ne sont pas enregistrées dans un fichier, l’éditeur affiche un message pour déterminer s’il faut enregistrer ou ignorer les modifications non enregistrées.

</dd> </dl>

## <a name="the-edit-menu"></a>Le menu Edition

<dl> <dt>

<span id="Undo"></span><span id="undo"></span><span id="UNDO"></span>**Uler**
</dt> <dd>

Supprime une modification apportée dans l’éditeur.

</dd> <dt>

<span id="Redo"></span><span id="redo"></span><span id="REDO"></span>**Rétabli**
</dt> <dd>

Inverse une action d’annulation dans l’éditeur.

</dd> <dt>

<span id="Cut"></span><span id="cut"></span><span id="CUT"></span>**Réduis**
</dt> <dd>

supprime l’élément sélectionné de l’éditeur et le place dans le presse-papiers Windows.

</dd> <dt>

<span id="Copy"></span><span id="copy"></span><span id="COPY"></span>**Reprographie**
</dt> <dd>

copie l’élément sélectionné dans l’éditeur dans le presse-papiers Windows.

</dd> <dt>

<span id="Paste"></span><span id="paste"></span><span id="PASTE"></span>**Insérer**
</dt> <dd>

copie les données du presse-papiers Windows actuel vers l’emplacement sélectionné.

</dd> <dt>

<span id="Delete"></span><span id="delete"></span><span id="DELETE"></span>**Supprimer**
</dt> <dd>

Supprime l’élément sélectionné de l’éditeur.

</dd> <dt>

<span id="New_Animation"></span><span id="new_animation"></span><span id="NEW_ANIMATION"></span>**Nouvelle animation**
</dt> <dd>

Crée un objet d’animation dans l’éditeur.

</dd> <dt>

<span id="New_Frame"></span><span id="new_frame"></span><span id="NEW_FRAME"></span>**Nouveau Frame**
</dt> <dd>

Crée un nouveau Frame pour une animation.

</dd> <dt>

<span id="New_Frames_from_Files"></span><span id="new_frames_from_files"></span><span id="NEW_FRAMES_FROM_FILES"></span>**Nouveaux frames à partir de fichiers**
</dt> <dd>

Affiche la boîte de dialogue Sélectionner les fichiers image et crée des frames à l’aide des fichiers sélectionnés.

</dd> <dt>

<span id="Open_Frame_Window"></span><span id="open_frame_window"></span><span id="OPEN_FRAME_WINDOW"></span>**Ouvrir la fenêtre frame**
</dt> <dd>

Affiche le frame actuel et ses images dans une fenêtre distincte sans mettre à l’échelle les images chargées dans le frame.

</dd> <dt>

<span id="Preview___Stop_Preview"></span><span id="preview___stop_preview"></span><span id="PREVIEW___STOP_PREVIEW"></span>**Aperçu \| arrêter l’aperçu**
</dt> <dd>

Lit (ou arrête la lecture) une animation, en commençant par son cadre sélectionné.

</dd> <dt>

<span id="Preview___Stop_Preview_Exit_Branching"></span><span id="preview___stop_preview_exit_branching"></span><span id="PREVIEW___STOP_PREVIEW_EXIT_BRANCHING"></span>**Version préliminaire arrêter l’aperçu de la \| branche**
</dt> <dd>

Lit (ou arrête la lecture) le branchement de sortie d’une animation, en commençant par son cadre sélectionné.

</dd> </dl>
