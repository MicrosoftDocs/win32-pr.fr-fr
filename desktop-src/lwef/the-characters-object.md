# <a name="the-characters-object"></a>Objet Characters

\[Microsoft Agent est déconseillé à partir de Windows 7 et peut ne pas être disponible dans les versions ultérieures de Windows.\]

Votre application cliente peut prendre en charge un ou plusieurs caractères. En outre, vous pouvez partager un caractère entre plusieurs applications. Microsoft Agent définit l’objet [**Characters**](/windows/desktop/lwef/the-characters-object) comme une collection de caractères. Pour accéder à un caractère, chargez les données du caractère dans la collection de [**caractères**](/windows/desktop/lwef/the-characters-object) et spécifiez que l’élément de la collection utilise les méthodes et les propriétés prises en charge pour ce caractère.

-   [Characters, méthodes d’objet](characters-object-methods.md)
-   [Characters, méthodes d’objet](character-object-methods.md)
-   [Propriétés de l’objet Characters](character-object-properties.md)
