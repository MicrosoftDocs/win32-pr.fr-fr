# <a name="iagentaudiooutputproperties"></a>IAgentAudioOutputProperties

\[Microsoft Agent est déconseillé à partir de Windows 7 et peut ne pas être disponible dans les versions ultérieures de Windows.\]

IAgentAudioOutputProperties fournit l’accès aux propriétés de sortie audio gérées par le serveur Microsoft Agent. Ces fonctions sont également disponibles à partir de IAgentAudioOutputProperties. Les propriétés sont en lecture seule, mais l’utilisateur peut les modifier dans la feuille de propriétés de l’agent Microsoft.

**Méthodes dans l'ordre Vtable**



| Méthodes IAgentAudioOutputProperties                                               | Description                                     |
|-----------------------------------------------------------------------------------|-------------------------------------------------|
| [**GetEnabled**](iagentaudiooutputproperties--getenabled.md)                     | Retourne une valeur indiquant si la sortie audio est activée.        |
| [**GetUsingSoundEffects**](iagentaudiooutputproperties--getusingsoundeffects.md) | Retourne une valeur indiquant si la sortie des effets sonores est activée. |
