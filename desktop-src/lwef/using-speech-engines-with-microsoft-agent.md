# <a name="using-speech-engines-with-microsoft-agent"></a>Utilisation de moteurs de reconnaissance vocale avec Microsoft Agent

\[Microsoft Agent est déconseillé à partir de Windows 7 et peut ne pas être disponible dans les versions ultérieures de Windows.\]

Microsoft fournit un ensemble de moteurs d’entrée vocale (reconnaissance vocale) et de sorties vocales (par synthèse vocale ou TTS) que vous pouvez utiliser avec vos applications ou pages Web compatibles avec Microsoft Agent. L’utilisation de ces moteurs est soumise au contrat de licence Microsoft Agent et à l’addenda supplémentaire affiché lors de l’installation des moteurs. Cette licence vous donne la distribution des moteurs, mais uniquement lorsque vous les utilisez via l’API Microsoft Agent et que vous utilisez un caractère visible.

Les sections ci-dessous décrivent comment utiliser les moteurs de reconnaissance vocale avec Microsoft Agent.

-   [Accès à un moteur de reconnaissance vocale dans votre code](accessing-a-speech-engine-in-your-code.md)
-   [Balises de sortie vocale Microsoft Agent](microsoft-agent-speech-output-tags.md)
-   [Normalisation du texte des moteurs de synthèse vocale](text-to-speech-engines-text-normalization.md)
-   [Autres moteurs de reconnaissance vocale compatibles avec Microsoft Agent](other-speech-engines-compatible-with-microsoft-agent.md)

Recherchez les modifications apportées aux fonctionnalités Microsoft Agent Speech et reconnaissance vocale pour Windows Vista en suivant le lien ci-dessous.

-   [Modifications de Microsoft Agent dans Windows Vista](microsoft-agent-changes-in-windows-vista.md)
