# <a name="installing-the-sound-editor"></a>Installation de l’éditeur Sound

\[Microsoft Agent est déconseillé à partir de Windows 7 et peut ne pas être disponible dans les versions ultérieures de Windows.\]

la configuration système recommandée pour l’utilisation de l’éditeur sound est un PC doté d’un processeur Pentium 166, d’au moins 48 mo de RAM et d’une carte son compatible Windows. Si vous souhaitez enregistrer une entrée parlée avec l’outil, vous aurez également besoin d’un microphone compatible.

Pour installer l’outil de modification linguistique du son, ouvrez le fichier d’installation à extraction automatique. Cette opération installe automatiquement les fichiers appropriés sur votre système. Si vous téléchargez l’éditeur Sound à partir du site Web Microsoft Agent, vous pouvez choisir d’installer l’éditeur après l’avoir téléchargé ou de l’enregistrer sur votre disque pour qu’il soit ouvert et installé par la suite. L’outil d’installation vous propose de l’installer dans le sous-répertoire Tools de Microsoft Agent. Nous vous recommandons d’utiliser cet emplacement.

Le moteur de reconnaissance vocale Microsoft Command and Control (version. 4,0) doit également être installé pour que vous puissiez utiliser l’éditeur Sound. Cela est normalement installé avec l’éditeur de sons, mais s’il a été désinstallé par la suite, vous pouvez le réinstaller à partir du kit de développement logiciel (SDK) Microsoft Platform ou du site Web de l’agent Microsoft à l’adresse <https://www.microsoft.com/downloads/details.aspx?FamilyID=914fef6b-d764-4eca-88dc-d49d8912625c> . L’éditeur de sons peut uniquement générer des informations linguistiques basées sur le langage pris en charge par le moteur de reconnaissance vocale. Pour générer des informations pour d’autres langues, un moteur de reconnaissance vocale compatible pour cette langue doit être installé. Contactez le fournisseur de votre moteur de reconnaissance vocale pour déterminer s’il prend en charge l’outil de modification linguistique de Microsoft.
