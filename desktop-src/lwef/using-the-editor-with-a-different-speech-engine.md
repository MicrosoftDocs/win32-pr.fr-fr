# <a name="using-the-editor-with-a-different-speech-engine"></a>Utilisation de l’éditeur avec un autre moteur de reconnaissance vocale

\[Microsoft Agent est déconseillé à partir de Windows 7 et peut ne pas être disponible dans les versions ultérieures de Windows.\]

Bien que l’éditeur de sons installe le moteur de reconnaissance vocale Microsoft (4,0), il peut être utilisé avec un autre moteur de reconnaissance vocale, si ce moteur prend en charge les interfaces requises documentées dans le document configuration requise du moteur de reconnaissance vocale. Avant de tenter d’utiliser l’éditeur avec un autre moteur, vérifiez auprès de votre fournisseur qu’il est conforme à ces exigences.

Pour utiliser l’éditeur de sons avec un autre moteur de reconnaissance vocale, choisissez la commande moteur vocal dans le menu **Edition** . Cela affiche une boîte de dialogue indiquant le moteur en cours d’utilisation. Pour choisir un autre moteur, affichez la liste des moteurs, puis cliquez sur **OK**. Si aucun autre moteur n’est répertorié, cela signifie que vous n’avez pas d’autres moteurs compatibles.
