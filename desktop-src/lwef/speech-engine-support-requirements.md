# <a name="speech-engine-support-requirements"></a>Exigences de prise en charge du moteur de reconnaissance vocale

\[Microsoft Agent est déconseillé à partir de Windows 7 et peut ne pas être disponible dans les versions ultérieures de Windows.\]

Microsoft Agent utilise Microsoft Speech Application Programming Interface (SAPI) pour prendre en charge l’entrée vocale (reconnaissance vocale ou SR) et la sortie vocale (conversion de texte par synthèse vocale ou TTS). En prenant en charge cette norme, les services vocaux de Microsoft Agent peuvent être pris en charge par d’autres moteurs de reconnaissance vocale. Ce document décrit les interfaces SAPI nécessaires utilisées par Microsoft Agent. Pour plus d’informations sur SAPI, consultez le site Web du groupe [Microsoft Speech](https://msdn.microsoft.com/library/ee705648.aspx) .

-   [Configuration requise pour les moteurs de conversion de texte par synthèse vocale](requirements-for-text-to-speech-engines.md)
-   [Configuration requise pour les moteurs de reconnaissance vocale](requirements-for-speech-recognition-engines.md)
-   [Conditions requises pour la prise en charge de l’outil de modification du son des informations linguistiques Microsoft](requirements-for-supporting-the-microsoft-linguistic-information-sound-editing-tool.md)
