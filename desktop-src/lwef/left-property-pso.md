# <a name="left-property-propertysheet-object"></a>Propriété Left (objet feuille)

\[Microsoft Agent est déconseillé à partir de Windows 7 et peut ne pas être disponible dans les versions ultérieures de Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Descriptive**
</dt> <dd>

Retourne une valeur entière spécifiant le bord gauche actuel, en pixels, de la fenêtre de la feuille de propriétés de l’agent Microsoft.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Stockéesyntaxe**
</dt> <dd>

*agent * * *. Feuille. Left**

</dd> </dl>

## <a name="remarks"></a>Notes

Le serveur affiche la fenêtre en fonction de l’emplacement défini par l’utilisateur.
