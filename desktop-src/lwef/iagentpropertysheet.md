# <a name="iagentpropertysheet"></a>IAgentPropertySheet

\[Microsoft Agent est déconseillé à partir de Windows 7 et peut ne pas être disponible dans les versions ultérieures de Windows.\]

**IAgentPropertySheet** définit une interface qui permet aux applications de définir et d’interroger les propriétés de la feuille de propriétés de l’agent Microsoft (fenêtre).

**Méthodes dans l'ordre Vtable**



| Méthodes IAgentPropertySheet                             | Description                                                                                  |
|---------------------------------------------------------|----------------------------------------------------------------------------------------------|
| [**GetVisible**](iagentpropertysheet--getvisible.md)   | Retourne une valeur indiquant si la feuille de propriétés Microsoft Agent est visible.                               |
| [**SetVisible**](iagentpropertysheet--setvisible.md)   | Définit la propriété [**visible**](visible-property.md) de la feuille de propriétés de l’agent Microsoft. |
| [**GetPosition**](iagentpropertysheet--getposition.md) | Retourne la position de la feuille de propriétés de l’agent Microsoft.                                  |
| [**GetSize**](iagentpropertysheet--getsize.md)         | Retourne la taille de la feuille de propriétés de l’agent Microsoft.                                      |
| [**GetPage**](iagentpropertysheet--getpage.md)         | Retourne la page active de la feuille de propriétés de l’agent Microsoft.                             |
| [**SetPage**](iagentpropertysheet--setpage.md)         | Définit la page actuelle pour la feuille de propriétés de l’agent Microsoft.                                |
