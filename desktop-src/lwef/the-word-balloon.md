# <a name="the-word-balloon"></a>La bulle de mot

\[Microsoft Agent est déconseillé à partir de Windows 7 et peut ne pas être disponible dans les versions ultérieures de Windows.\]

Outre la sortie audio parlée, l’interface de l’agent Microsoft prend en charge le sous-titrage textuel sous la forme d’une sortie de texte dans les bulles de style de dessin animé. Les mots s’affichent dans l’info-bulle à mesure qu’ils sont prononcés. La bulle se masque quand la sortie orale est terminée. La fenêtre Options de caractères avancés fournit des options pour désactiver l’affichage de l’info-bulle, ainsi que pour contrôler les attributs liés à son apparence.
