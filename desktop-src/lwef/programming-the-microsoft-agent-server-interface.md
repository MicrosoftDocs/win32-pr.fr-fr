# <a name="programming-the-microsoft-agent-server-interface"></a>Programmation de l’interface du serveur Microsoft Agent

\[Microsoft Agent est déconseillé à partir de Windows 7 et peut ne pas être disponible dans les versions ultérieures de Windows.\]

Microsoft Agent fournit des services qui vous permettent de programmer des caractères animés à partir d’une application. Ces services sont implémentés en tant que serveur OLE Automation. OLE Automation permet à une application de contrôler l’objet d’une autre application par programme. Ce document suppose une compréhension du modèle COM (Component Object Model) et d’OLE. Pour une introduction à ces services, consultez la [vue d’ensemble](microsoft-agent-programming-interface-overview.md)de l’interface de programmation.
