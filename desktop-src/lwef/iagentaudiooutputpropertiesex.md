# <a name="iagentaudiooutputpropertiesex"></a>IAgentAudioOutputPropertiesEx

\[Microsoft Agent est déconseillé à partir de Windows 7 et peut ne pas être disponible dans les versions ultérieures de Windows.\]

IAgentAudioOutputPropertiesEx est dérivé de l’interface IAgentAudioOutputPropertiesEx. Il inclut toutes les méthodes IAgentAudioOutputPropertiesEx et permet d’accéder à des fonctions supplémentaires.

**Méthodes dans l'ordre Vtable**



| Méthodes IAgentAudioOutputPropertiesEx                         | Description                                     |
|---------------------------------------------------------------|-------------------------------------------------|
| [**GetStatus**](iagentaudiooutputpropertiesex--getstatus.md) | Retourne l’état du canal de sortie audio. |
