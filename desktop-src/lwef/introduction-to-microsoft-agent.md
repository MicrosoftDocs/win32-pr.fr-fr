# <a name="introduction-to-microsoft-agent"></a>Présentation de Microsoft Agent

\[Microsoft Agent est déconseillé à partir de Windows 7 et peut ne pas être disponible dans les versions ultérieures de Windows.\]

Microsoft Agent est un ensemble de services logiciels programmables qui prend en charge la présentation de caractères animés interactifs. Les développeurs peuvent utiliser des caractères comme assistants interactifs pour introduire, guider, divertir ou améliorer leurs pages Web ou applications en plus de l’utilisation conventionnelle des fenêtres, des menus et des contrôles.

Microsoft Agent permet aux développeurs de logiciels et aux créateurs de sites Web d’intégrer une nouvelle forme d’interaction utilisateur, appelée interfaces de conversation, qui tire parti des aspects naturels de la communication sociale humaine. Outre la souris et l’entrée au clavier, Microsoft agent prend en charge la reconnaissance vocale en option pour que les applications puissent répondre aux commandes vocales. Les caractères peuvent répondre à l’aide d’une synthèse vocale, d’un enregistrement audio ou d’un texte dans une bulle de texte animé.

L’approche de l’interface conversationnel, facilitée par les services Microsoft Agent, ne remplace pas la conception de l’interface graphique utilisateur classique. Au lieu de cela, l’interaction des caractères peut être facilement fusionnée avec les composants d’interface conventionnels tels que les fenêtres, les menus et les contrôles pour étendre et améliorer l’interface de votre application.

Les interfaces de programmation de Microsoft Agent facilitent l’animation d’un caractère pour répondre aux entrées de l’utilisateur. Les caractères animés s’affichent dans leur propre fenêtre, ce qui offre une flexibilité maximale pour les emplacements où ils peuvent être affichés à l’écran. Microsoft Agent inclut un contrôle ActiveX qui rend ses services accessibles aux langages de programmation qui prennent en charge ActiveX, y compris les langages de script Web tels que Visual Basic Scripting Edition (VBScript). Cela signifie que l’interaction des caractères peut être programmée même à partir de pages HTML à l’aide de la <OBJECT> balise.
