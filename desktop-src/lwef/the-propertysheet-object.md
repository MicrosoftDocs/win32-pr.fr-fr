# <a name="the-propertysheet-object"></a>Objet feuille

\[Microsoft Agent est déconseillé à partir de Windows 7 et peut ne pas être disponible dans les versions ultérieures de Windows.\]

L’objet [**feuille**](https://www.bing.com/search?q=**PropertySheet**) fournit plusieurs propriétés que vous pouvez utiliser si vous souhaitez manipuler le caractère par rapport à la feuille de propriétés de l’agent Microsoft (également appelée fenêtre Options de caractères avancés).

-   [**Hauteur**](height-property-pso.md)
-   [**Gauche**](left-property-pso.md)
-   [**Page**](page-property.md)
-   [**Retour au début**](top-property-pso.md)
-   [**Parent**](visible-property.md)
-   [**Largeur**](width-property-pso.md)

Si vous interrogez les propriétés de [**hauteur**](height-property-pso.md), de [**gauche**](left-property-pso.md), de [**haut**](top-property-pso.md)et de [**largeur**](width-property-pso.md) avant que la feuille de propriétés n’apparaisse, leurs valeurs retournent la valeur zéro (0). Une fois affichés, ces propriétés retournent la dernière position et taille de la fenêtre (par rapport à votre résolution d’écran actuelle).
