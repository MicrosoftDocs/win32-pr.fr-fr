# <a name="publishing-wizards"></a>Assistants de publication

cette section décrit les assistants de publication Windows XP.



| Rubrique                                               | Contenu                                                                                                                                                                                                                                                                   |
|-----------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Conception côté client](pubwiz-client.md)             | Le script des pages HTML côté serveur communique avec le client de l’Assistant commande d’impression en ligne dans lequel il est hébergé. Cette communication s’effectue par le biais de méthodes et de propriétés accessibles par l’objet **Window. external** .<br/>                                 |
| [Présentation des assistants de publication](pubwiz-intro.md) | Windows XP présente l’Assistant Publication de sites Web et l’Assistant commande d’impression en ligne. À partir du même Framework, ces assistants fournissent à l’utilisateur un mécanisme simple pour publier du contenu sur un site Web ou pour commander des impressions effectuées à partir de fichiers image numériques.<br/> |
| [Inscription d’un service](pubwiz-reg.md)             | pour ajouter votre service à la liste des fournisseurs dans l’assistant publication de sites Web ou l’assistant commande d’impression en ligne, vous devez ajouter la clé appropriée et ses valeurs au registre Windows.<br/>                                                                  |
| [Conception côté serveur](pubwiz-server.md)             | Les fonctions côté serveur communiquent avec l’Assistant client via l’objet **Windows. external** . Le script côté serveur fournit ces fonctions pour répondre aux événements de l’Assistant et récupérer des informations sur l’Assistant.<br/>                                         |
| [Utilisation du manifeste de transfert](pubwiz-manifest.md)  | L’Assistant Publication de sites Web et l’Assistant commande d’impression en ligne utilisent le manifeste de transfert pour communiquer les détails du transfert de données entre l’ordinateur client et le site serveur.<br/>                                                                                   |
