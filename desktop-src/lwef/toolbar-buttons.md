# <a name="toolbar-buttons-linguistic-information-sound-editing-tool"></a>Boutons de la barre d’outils (outil d’édition audio des informations linguistiques)

\[Microsoft Agent est déconseillé à partir de Windows 7 et peut ne pas être disponible dans les versions ultérieures de Windows.\]

:::image type="icon" source="images/f9charnew.gif":::

<dl> <dt>

<span id="New"></span><span id="new"></span><span id="NEW"></span>**Nouveau**
</dt> <dd>

Réinitialise l’éditeur de sons pour la création d’un nouveau fichier audio. Si un fichier son existant est chargé et que des modifications n’ont pas été enregistrées, l’éditeur Sound affiche un message pour déterminer s’il faut enregistrer ou ignorer les modifications non enregistrées.

</dd> </dl>

:::image type="icon" source="images/f10charopen.gif":::

<dl> <dt>

<span id="Open"></span><span id="open"></span><span id="OPEN"></span>**Afficher**
</dt> <dd>

Affiche la boîte de dialogue Ouvrir un fichier, qui vous permet d’ouvrir un fichier audio existant. Si un fichier son existant est chargé et que des modifications n’ont pas été enregistrées, l’éditeur Sound affiche un message pour déterminer s’il faut enregistrer ou ignorer les modifications non enregistrées.

</dd> </dl>

:::image type="icon" source="images/f11charsave.gif":::

<dl> <dt>

<span id="Save"></span><span id="save"></span><span id="SAVE"></span>**Été**
</dt> <dd>

Enregistre le fichier audio. Si le fichier n’existe pas (n’a pas été nommé), l’éditeur affiche la boîte de dialogue Enregistrer sous pour l’entrée du nom de fichier.

</dd> </dl>

:::image type="icon" source="images/f14charcut.gif":::

<dl> <dt>

<span id="Cut"></span><span id="cut"></span><span id="CUT"></span>**Réduis**
</dt> <dd>

Supprime le texte sélectionné de l’éditeur et le place dans le presse-papiers Windows.

</dd> </dl>

:::image type="icon" source="images/f15charcopy.gif":::

<dl> <dt>

<span id="Copy"></span><span id="copy"></span><span id="COPY"></span>**Reprographie**
</dt> <dd>

Copie le texte sélectionné dans l’éditeur dans le presse-papiers Windows.

</dd> </dl>

:::image type="icon" source="images/f16charpaste.gif":::

<dl> <dt>

<span id="Paste"></span><span id="paste"></span><span id="PASTE"></span>**Insérer**
</dt> <dd>

Copie le texte du presse-papiers Windows actuel vers l’emplacement sélectionné dans la zone de texte représentation textuelle.

</dd> </dl>

:::image type="icon" source="images/f17chardel.gif":::

<dl> <dt>

<span id="Delete"></span><span id="delete"></span><span id="DELETE"></span>**Supprimer**
</dt> <dd>

Supprime le texte sélectionné de l’éditeur de sons.

</dd> </dl>

:::image type="icon" source="images/f18charundo.gif":::

<dl> <dt>

<span id="Undo"></span><span id="undo"></span><span id="UNDO"></span>**Uler**
</dt> <dd>

Supprime une modification apportée dans l’éditeur de sons.

</dd> </dl>

:::image type="icon" source="images/f19charredo.gif":::

<dl> <dt>

<span id="Redo"></span><span id="redo"></span><span id="REDO"></span>**Rétabli**
</dt> <dd>

Inverse une action d’annulation dans l’éditeur de sons.

</dd> </dl>

:::image type="icon" source="images/flistlinguist.gif":::

<dl> <dt>

<span id="Generate_Linguistic_Info"></span><span id="generate_linguistic_info"></span><span id="GENERATE_LINGUISTIC_INFO"></span>**Générer des informations linguistiques**
</dt> <dd>

Génère des étiquettes de phonème et de mot pour le fichier audio.

</dd> </dl>

:::image type="icon" source="images/flistpause.gif":::

<dl> <dt>

<span id="Pause"></span><span id="pause"></span><span id="PAUSE"></span>**Suspen**
</dt> <dd>

Suspend la diffusion du fichier son.

</dd> </dl>

:::image type="icon" source="images/flistplay.gif":::

<dl> <dt>

<span id="Play_"></span><span id="play_"></span><span id="PLAY_"></span>**Répétition** 
</dt> <dd>

Lit le fichier audio ou la partie sélectionnée du fichier son.

</dd> </dl>

:::image type="icon" source="images/fliststop.gif":::

<dl> <dt>

<span id="Stop_"></span><span id="stop_"></span><span id="STOP_"></span>**Erreur** 
</dt> <dd>

Arrête l’enregistrement ou la diffusion du fichier audio ou de la partie sélectionnée du fichier son.

</dd> </dl>

:::image type="icon" source="images/flistrecord.gif":::

<dl> <dt>

<span id="Record"></span><span id="record"></span><span id="RECORD"></span>**Enregistrement**
</dt> <dd>

Démarre l’enregistrement d’un fichier audio.

</dd> </dl>
