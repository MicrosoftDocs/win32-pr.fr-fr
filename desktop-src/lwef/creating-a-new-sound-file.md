# <a name="creating-a-new-sound-file"></a>Création d’un nouveau fichier audio

\[Microsoft Agent est déconseillé à partir de Windows 7 et peut ne pas être disponible dans les versions ultérieures de Windows.\]

Lors du premier démarrage de l’éditeur, vous pouvez créer un nouveau fichier audio en choisissant **enregistrement** dans le menu **audio** ou en cliquant sur le bouton **Enregistrer** dans la barre d’outils de l’éditeur de sons, puis en parlant dans le microphone attaché à votre système. Cliquez sur le bouton **arrêter** dans la barre d’outils pour arrêter l’enregistrement. Vous pouvez sélectionner la commande **lire** dans le menu **audio** ou la barre d’outils pour voir comment Microsoft Agent traite le fichier audio sans l’amélioration linguistique. Pour créer un autre fichier, sélectionnez Nouveau dans le menu **fichier** ou dans la barre d’outils.
