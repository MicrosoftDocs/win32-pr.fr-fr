# <a name="advanced-character-options-window-microsoft-agent-user-interface"></a>Fenêtre Options des caractères avancés (interface utilisateur de l’agent Microsoft)

\[Microsoft Agent est déconseillé à partir de Windows 7 et peut ne pas être disponible dans les versions ultérieures de Windows.\]

Microsoft Agent gère certains paramètres globaux qui permettent à un utilisateur de contrôler l’interaction avec tous les caractères. La fenêtre Options de caractères avancés affiche ces options et leurs paramètres actuels et peut être affichée par n’importe quelle application d’hébergement à l’aide de l’interface de programmation Microsoft Agent. Microsoft Agent se souvient de la dernière page affichée et affiche cette page lorsque la feuille de propriétés s’affiche.

-   [Page sortie](the-output-page.md)
-   [La page d’entrée vocale](the-speech-input-page.md)
-   [Page de Copyright](the-copyright-page.md)
