# <a name="windows-defender-reference"></a>Informations de référence sur Windows Defender

Windows Defender aide à protéger les ordinateurs contre les fenêtres publicitaires, les performances ralenties et les menaces de sécurité causées par les logiciels espions et autres logiciels indésirables. Les applications peuvent demander des informations telles que l’intégrité de l’ordinateur, des informations spécifiques sur les programmes malveillants et l’historique des programmes malveillants pour le client, et appeler des analyses à la demande.
