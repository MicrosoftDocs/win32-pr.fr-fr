# <a name="windows-defender-reference"></a>Windows Defender Faire

Windows Defender permet de protéger les ordinateurs contre les fenêtres publicitaires, les performances dégradées et les menaces de sécurité causées par les logiciels espions et autres logiciels indésirables. Les applications peuvent demander des informations telles que l’intégrité de l’ordinateur, des informations spécifiques sur les programmes malveillants et l’historique des programmes malveillants pour le client, et appeler des analyses à la demande.
