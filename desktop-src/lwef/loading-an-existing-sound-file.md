# <a name="loading-an-existing-sound-file"></a>Chargement d’un fichier audio existant

\[Microsoft Agent est déconseillé à partir de Windows 7 et peut ne pas être disponible dans les versions ultérieures de Windows.\]

vous pouvez également charger un fichier audio Windows existant (. wav) ou un fichier audio linguistiquement amélioré (. lwv) en choisissant la commande **ouvrir** dans le menu **fichier** ou dans la barre d’outils. La boîte de dialogue **ouvrir** s’affiche. Sélectionnez un fichier, puis cliquez sur **ouvrir** pour charger le fichier dans l’éditeur.
