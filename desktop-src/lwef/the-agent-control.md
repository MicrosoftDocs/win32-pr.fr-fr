# <a name="the-agent-control"></a>Contrôle de l’agent

\[Microsoft Agent est déconseillé à partir de Windows 7 et peut ne pas être disponible dans les versions ultérieures de Windows.\]

Le fait de référencer le contrôle de l’agent permet d’accéder aux événements et à la plupart des autres objets pris en charge par Microsoft Agent. Le contrôle de l’agent expose aussi directement son propre ensemble de propriétés.

-   [Propriétés du contrôle de l’agent](agent-control-properties.md)
-   [Événements de contrôle de l’agent](agent-control-events.md)
