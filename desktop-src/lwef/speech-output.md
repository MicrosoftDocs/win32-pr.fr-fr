# <a name="speech-output"></a>Sortie vocale

\[Microsoft Agent est déconseillé à partir de Windows 7 et peut ne pas être disponible dans les versions ultérieures de Windows.\]

Comme les entrées vocales, les sorties vocales sont une forme de communication familière et naturelle. c’est pourquoi il s’agit également d’un complément approprié dans une interface basée sur des caractères. Toutefois, la sortie vocale a également ses responsabilités. Dans certains environnements, la sortie vocale peut ne pas être préférée ou audible. Par ailleurs, la reconnaissance vocale est invisible et n’a pas de sortie persistante, ce qui dépend fortement de la mémoire à bref terme. Ces facteurs limitent la capacité et la vitesse de traitement de grandes quantités d’informations. De même, la sortie vocale peut également perturber l’entrée de l’utilisateur, en particulier lorsque la reconnaissance vocale est la méthode d’entrée. Les moteurs de reconnaissance vocale disposent généralement d’une prise en charge limitée qui permet à l’utilisateur d’interrompre le canal de sortie.

Par conséquent, évitez d’utiliser la parole comme forme exclusive de la sortie. Toutefois, étant donné que Microsoft Agent présente des caractères dans le cadre de l’interface Windows, il offre plusieurs avantages par rapport aux environnements vocaux uniquement. Les caractères peuvent être combinés avec d’autres formes d’entrée et de sortie pour rendre les options et les actions visibles, ce qui permet d’obtenir une interface plus efficace qu’une interface exclusivement visuelle ou vocale.

En outre, pour rendre la sortie vocale plus visible, Microsoft Agent comprend la possibilité de créer un caractère avec une bulle de style de dessin animé. D’autres paramètres vous permettent de déterminer l’apparence du texte dans l’info-bulle et de la suppression de l’info-bulle. Vous pouvez également déterminer la police à utiliser. Bien que vous puissiez définir les attributs des bulles de mots d’un caractère, n’oubliez pas que l’utilisateur peut remplacer ces paramètres.
