# <a name="charsperline-property"></a>Propriété CharsPerLine

\[Microsoft Agent est déconseillé à partir de Windows 7 et peut ne pas être disponible dans les versions ultérieures de Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Descriptive**
</dt> <dd>

Retourne les caractères par ligne pris en charge pour le mot-bulle pour le caractère spécifié.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Stockéesyntaxe**
</dt> <dd>

*agent*[**. Caractères («**](/windows/desktop/lwef/the-characters-object)* CharacterID ***»).** Balloon. CharsPerLine

</dd> </dl>

## <a name="remarks"></a>Notes

Cette propriété retourne le nombre moyen de caractères (lettres) affichés dans la bulle de mot sous la forme d’une valeur entière longue. Vous pouvez définir la valeur à l’aide de la propriété [**style**](style-property.md) .
