# <a name="starting-the-agent-character-editor"></a>Démarrage de l’éditeur de caractères de l’agent

\[Microsoft Agent est déconseillé à partir de Windows 7 et peut ne pas être disponible dans les versions ultérieures de Windows.\]

pour exécuter l’éditeur de caractères de l’agent, choisissez l’option **éditeur de caractères** de l’agent dans le menu **démarrer** de Windows barre des tâches, ou double-cliquez sur l’icône éditeur de **caractères Microsoft agent** sur votre bureau. La fenêtre de l’éditeur s’ouvre et affiche ses menus, une barre d’outils avec des commandes fréquemment utilisées, une arborescence répertoriant les composants qui composent la définition d’un caractère et un ensemble de pages avec onglets qui changent en fonction de votre sélection dans l’arborescence des composants.

![Capture d’écran montrant la page « Propriétés » de l’éditeur de caractères « Character-Microsoft Agent ».](images/f1chared.gif)

Pour accéder aux champs de la fenêtre à partir du clavier, vous pouvez utiliser la touche TAB et les touches MAJ + TAB pour naviguer entre les contrôles ou utiliser la touche d’accès (ALT +*lettre soulignée*) pour vous déplacer vers un contrôle spécifique. Une fois l’éditeur terminé, vous pouvez commencer à créer une définition de caractère ou charger une définition de caractère existante.

La barre d’état affiche des informations sur les commandes ou les boutons de la barre d’outils lorsque vous placez le pointeur dessus. Il affiche également des informations récapitulatives sur les données d’animation et les informations d’état de vos caractères lorsque vous générez un caractère.
