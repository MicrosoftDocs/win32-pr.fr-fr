# <a name="the-speech-input-page"></a>La page d’entrée vocale

\[Microsoft Agent est déconseillé à partir de Windows 7 et peut ne pas être disponible dans les versions ultérieures de Windows.\]

Un utilisateur peut ajuster les options d’entrée vocale sur cette page de propriétés. L’utilisateur peut désactiver l’entrée vocale, définir la clé d’écoute, choisir d’afficher ou non la fenêtre de l’info-bulle d’écoute et choisir de lire une tonalité MIDI pour indiquer quand une entrée vocale est disponible.
