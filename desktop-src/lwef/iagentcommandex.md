# <a name="iagentcommandex"></a>IAgentCommandEx

\[Microsoft Agent est déconseillé à partir de Windows 7 et peut ne pas être disponible dans les versions ultérieures de Windows.\]

**IAgentCommandEx** est dérivé de l’interface [**IAgentCommand**](iagentcommand.md) . Il inclut toutes les méthodes **IAgentCommand** et permet d’accéder à des fonctions supplémentaires.

**Méthodes dans l'ordre Vtable**



| Méthodes IAgentCommandEx                                       | Description                                                                                  |
|---------------------------------------------------------------|----------------------------------------------------------------------------------------------|
| [**SetHelpContextID**](iagentcommandex--sethelpcontextid.md) | Définit l’ID de la rubrique d’aide contextuelle pour un objet [**Command**](/windows/desktop/lwef/the-command-object) .    |
| [**GetHelpContextID**](iagentcommandex--gethelpcontextid.md) | Retourne l’ID de la rubrique d’aide contextuelle pour un objet [**Command**](/windows/desktop/lwef/the-command-object) . |
| [**SetVoiceCaption**](iagentcommandex--setvoicecaption.md)   | Définit la légende vocale pour un objet [**Command**](/windows/desktop/lwef/the-command-object) .                      |
| [**GetVoiceCaption**](iagentcommandex--getvoicecaption.md)   | Retourne la légende de la voix pour un objet [**Command**](/windows/desktop/lwef/the-command-object) .                   |
