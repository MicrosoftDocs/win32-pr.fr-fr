# <a name="programming-the-microsoft-agent-control"></a>Programmation du contrôle Microsoft Agent

\[Microsoft Agent est déconseillé à partir de Windows 7 et peut ne pas être disponible dans les versions ultérieures de Windows.\]

bien que les applications puissent écrire directement sur les services Microsoft agent à l’aide de ses interfaces de serveur automation, Microsoft Agent comprend également un contrôle ActiveX (OLE). le contrôle prend en charge la programmation simple à l’aide d’un langage de script tel que Microsoft Visual Basic scripting Edition (VBScript) ou d’autres langages qui prennent en charge l’interface de contrôle ActiveX.
