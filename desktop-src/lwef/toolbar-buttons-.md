# <a name="toolbar-buttons-microsoft-agent-character-editor"></a>Boutons de la barre d’outils (éditeur de caractères Microsoft Agent)

\[Microsoft Agent est déconseillé à partir de Windows 7 et peut ne pas être disponible dans les versions ultérieures de Windows.\]

:::image type="icon" source="images/f9charnew.gif":::

<dl> <dt>

<span id="New_Custom_Character"></span><span id="new_custom_character"></span><span id="NEW_CUSTOM_CHARACTER"></span>**Nouveau caractère personnalisé**
</dt> <dd>

Réinitialise l’éditeur pour créer une nouvelle définition de caractère personnalisée. Si un caractère existant est chargé et que des modifications ne sont pas enregistrées dans un ou plusieurs fichiers, l’éditeur affiche un message pour déterminer s’il faut enregistrer ou ignorer les modifications non enregistrées.

</dd> </dl>

:::image type="icon" source="images/f10charopen.gif":::

<dl> <dt>

<span id="Open_Character_Definition"></span><span id="open_character_definition"></span><span id="OPEN_CHARACTER_DEFINITION"></span>**Ouvrir une définition de caractère**
</dt> <dd>

Affiche la boîte de dialogue Ouvrir un fichier, qui vous permet d’ouvrir un fichier de définition de caractère existant pour le modifier. Si un caractère existant est chargé et que des modifications ne sont pas enregistrées dans un ou plusieurs fichiers, l’éditeur affiche un message pour déterminer s’il faut enregistrer ou ignorer les modifications non enregistrées.

</dd> </dl>

:::image type="icon" source="images/f11charsave.gif":::

<dl> <dt>

<span id="Save_Character_Definition"></span><span id="save_character_definition"></span><span id="SAVE_CHARACTER_DEFINITION"></span>**Enregistrer la définition de caractère**
</dt> <dd>

Enregistre la définition de caractère. Si la définition de caractère n’existe pas (n’a pas été nommée), l’éditeur affiche la boîte de dialogue Enregistrer sous pour l’entrée du nom de fichier.

</dd> </dl>

:::image type="icon" source="images/f12charbuild.gif":::

<dl> <dt>

<span id="Build_Character"></span><span id="build_character"></span><span id="BUILD_CHARACTER"></span>**Caractère de build**
</dt> <dd>

Crée un caractère Microsoft Agent à partir de la définition de caractère.

</dd> </dl>

:::image type="icon" source="images/f13charprint.gif":::

<dl> <dt>

<span id="Print_Character_Definition"></span><span id="print_character_definition"></span><span id="PRINT_CHARACTER_DEFINITION"></span>**Définition du caractère d’impression**
</dt> <dd>

Imprime le fichier de définition de caractère actif ouvert dans l’éditeur.

</dd> </dl>

:::image type="icon" source="images/f14charcut.gif":::

<dl> <dt>

<span id="Cut"></span><span id="cut"></span><span id="CUT"></span>**Réduis**

supprime l’élément sélectionné dans l’éditeur et le place dans le presse-papiers Windows.

</dd> </dl>

<span id="Copy"></span><span id="copy"></span><span id="COPY"></span>**Reprographie**

copie l’élément sélectionné dans l’éditeur dans le presse-papiers Windows.

</dd> </dl>

:::image type="icon" source="images/f16charpaste.gif":::

<dl> <dt>

<span id="Paste"></span><span id="paste"></span><span id="PASTE"></span>**Insérer**

</dd> </dl>
