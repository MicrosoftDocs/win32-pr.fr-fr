# <a name="ittsattributesw"></a>ITTSAttributesW

\[Microsoft Agent est déconseillé à partir de Windows 7 et peut ne pas être disponible dans les versions ultérieures de Windows.\]

Le moteur doit prendre en charge les attributs de tonalité et de vitesse via les méthodes [**PitchSet**](https://www.bing.com/search?q=**PitchSet**), [**PitchGet**](https://www.bing.com/search?q=**PitchGet**), [**SpeedSet**](https://www.bing.com/search?q=**SpeedSet**)et [**SpeedGet**](https://www.bing.com/search?q=**SpeedGet**) .
