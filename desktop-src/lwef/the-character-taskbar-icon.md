# <a name="the-character-taskbar-icon"></a>Icône caractère de la barre des tâches

\[Microsoft Agent est déconseillé à partir de Windows 7 et peut ne pas être disponible dans les versions ultérieures de Windows.\]

Si un caractère a été créé pour inclure une icône, l’icône s’affiche dans la zone de notification de la barre des tâches lors de l’exécution de Microsoft Agent. Cette icône permet d’accéder au menu contextuel du caractère, qui permet d’accéder aux commandes globales de l’agent, telles que celles qui masquent et affichent le caractère.

![zone de notification avec horloge et icône](images/f1tbicon.gif)

Le déplacement du pointeur sur l’icône de la barre des tâches affiche une fenêtre d’info-bulle qui reflète le nom du caractère (dans la langue actuelle du système). Un simple clic sur l’icône de la barre des tâches du caractère affiche le caractère. L’action associée au double-clic sur l’icône dépend de l’application en cours contrôlant ce caractère.

Cliquez avec le bouton droit sur l’icône pour afficher un menu contextuel. Lorsque le caractère est visible, le menu contextuel affiche les mêmes commandes que celles qui s’affichent lorsque vous cliquez avec le bouton droit sur le caractère. Si le caractère est masqué, seule la fenêtre Ouvrir (ou fermer) les commandes vocales et afficher les commandes s’affiche.
