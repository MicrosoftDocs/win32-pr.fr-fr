# <a name="visible-property-commandswindow-object"></a>Visible, propriété (objet CommandsWindow)

\[Microsoft Agent est déconseillé à partir de Windows 7 et peut ne pas être disponible dans les versions ultérieures de Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Descriptive**
</dt> <dd>

Retourne ou définit une valeur indiquant si la fenêtre commandes vocales est visible (ouverte).

</dd> <dt>

<span id="Syntax_"></span><span id="syntax_"></span><span id="SYNTAX_"></span>**Stockéesyntaxe** 
</dt> <dd>

*agent * * *. CommandsWindow.* *  \[  =  *valeur booléenne* visible\]



| Partie      | Description                                                                                                                                                                             |
|-----------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| *boolean* | Expression booléenne spécifiant si la fenêtre commandes vocales est visible.<br/> **True** La fenêtre est visible.<br/> **Valeur false** La fenêtre est masquée (fermée).<br/> |



 

</dd> </dl>

## <a name="remarks"></a>Notes

Le serveur affiche la fenêtre en fonction de l’emplacement et de la taille définis par l’utilisateur.

> [!Note]  
> L’utilisateur peut remplacer cette propriété.
