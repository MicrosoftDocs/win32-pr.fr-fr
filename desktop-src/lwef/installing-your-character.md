# <a name="installing-your-character"></a>Installation de votre caractère

\[Microsoft Agent est déconseillé à partir de Windows 7 et peut ne pas être disponible dans les versions ultérieures de Windows.\]

pour que votre caractère apparaisse dans la galerie de l’Assistant, installez-le dans le \\ dossier Microsoft Office \\ actors à l’intérieur du dossier Application Data de l’utilisateur. le dossier Application Data se trouve dans le répertoire Windows ou dans le répertoire des profils utilisateur à l’intérieur du répertoire Windows. par exemple, dans Windows 2000, vous pouvez installer votre caractère dans « C : \\ Documents and Paramètres \\ \\ Application votrenomdeconnexion Data Data \\ Microsoft \\ Office \\ actors ».
