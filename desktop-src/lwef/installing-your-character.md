# <a name="installing-your-character"></a>Installation de votre caractère

\[Microsoft Agent est déconseillé à partir de Windows 7 et peut ne pas être disponible dans les versions ultérieures de Windows.\]

Pour que votre caractère apparaisse dans la Galerie de l’Assistant, installez-le dans le \\ dossier Microsoft Office \\ acteurs à l’intérieur du dossier Application Data de l’utilisateur. Le dossier Application Data se trouve dans le répertoire Windows ou dans le répertoire des profils utilisateur à l’intérieur du répertoire Windows. Par exemple, dans Windows 2000, vous pouvez installer votre caractère dans « C : \\ Documents and Settings \\ \\ application données de l’application \\ Microsoft \\ Office \\ acteurs ».
