# <a name="windows-search"></a>Windows Search

Windows La recherche est une plateforme de recherche de bureau qui offre des fonctionnalités de recherche instantanée pour la plupart des types de données et de fichiers courants, tels que les courriers électroniques, les contacts, les rendez-vous, les documents, les photos, les éléments multimédias et d’autres formats pouvant être étendus par des développeurs tiers. Ces fonctionnalités permettent aux utilisateurs de rechercher, de gérer et d’organiser la quantité grandissante de données communes dans les environnements de bureau et d’entreprise.
