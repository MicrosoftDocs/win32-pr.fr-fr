# <a name="protocol-handlers-interfaces"></a>Interfaces des gestionnaires de protocole

cette section de référence de recherche Windows se compose d’interfaces de gestionnaires de protocole.


## <a name="in-this-section"></a>Contenu de cette section



| Rubrique                                                                                   | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
|-----------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IItemPropertyBag**](iitempropertybag.md)<br/>                                 | Définit des méthodes pour obtenir des informations sur les propriétés d’un élément de recherche. cette interface est prise en charge uniquement sur Windows XP et Windows Server 2003 et ne doit plus être utilisée.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| [**IOpLockStatus**](/windows/desktop/api/Searchapi/nn-searchapi-ioplockstatus)<br/>                               | fournit des méthodes pour vérifier le verrou opportuniste utilisé par Microsoft Windows Desktop Search (WDS) sur les éléments pendant l’indexation. Si un autre processus verrouille le fichier de manière incompatible, WDS perd son verrou et autorise l’autre processus à avoir le fichier. Ce mécanisme permet à WDS de s’exécuter en arrière-plan. Par conséquent, WDS doit vérifier ses verrous pour s’assurer qu’un autre processus n’a pas été prioritaire pendant que WDS indexe l’élément. <br/> Un objet [**IUrlAccessor**](/windows/desktop/api/Searchapi/nn-searchapi-iurlaccessor) tiers peut implémenter cette interface si le magasin de données sous-jacent fournit un mécanisme permettant d’effectuer le suivi de l’accès simultané aux éléments. Si cette interface est exposée par **IUrlAccessor**, WDS vérifie le [**IOpLockStatus**](/windows/desktop/api/Searchapi/nn-searchapi-ioplockstatus) lors de l’indexation des éléments de ce magasin. <br/> |
| [**IProtocolHandlerServices**](/previous-versions/windows/desktop/legacy/bb231447(v=vs.85))<br/>         |                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| [**IProtocolHandlerSite**](/windows/desktop/api/Searchapi/nn-searchapi-iprotocolhandlersite)<br/>                 | Fournit des méthodes pour l’objet [**IUrlAccessor**](/windows/desktop/api/Searchapi/nn-searchapi-iurlaccessor) d’un gestionnaire de protocole afin d’interroger le démon de filtre pour le filtre approprié pour l’élément URL. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| [**ISearchProtocol**](/windows/desktop/api/Searchapi/nn-searchapi-isearchprotocol)<br/>                           | Fournit des méthodes pour appeler, initialiser et gérer des objets [**IUrlAccessor**](/windows/desktop/api/Searchapi/nn-searchapi-iurlaccessor) . Les méthodes de cette interface sont appelées par l’hôte de protocole lors du traitement des URL à partir du Rassembleur. <br/> Le gestionnaire de protocole implémente le protocole d’accès à une source de contenu dans son format natif. Utilisez cette interface pour implémenter un gestionnaire de protocole personnalisé afin de développer les sources de données qui peuvent être indexées. <br/>                                                                                                                                                                                                                                                                                                                                                                    |
| [**ISearchProtocol2**](/windows/desktop/api/Searchapi/nn-searchapi-isearchprotocol2)<br/>                         | Fournit des méthodes pour appeler, initialiser et gérer des objets [**IUrlAccessor**](/windows/desktop/api/Searchapi/nn-searchapi-iurlaccessor) . Les méthodes de cette interface sont appelées par l’hôte de protocole lors du traitement des URL à partir du Rassembleur. <br/> Le gestionnaire de protocole implémente le protocole d’accès à une source de contenu dans son format natif. Utilisez cette interface pour implémenter un gestionnaire de protocole personnalisé afin de développer les sources de données qui peuvent être indexées. <br/>                                                                                                                                                                                                                                                                                                                                                                    |
| [**ISearchProtocolThreadContext**](/windows/desktop/api/Searchapi/nn-searchapi-isearchprotocolthreadcontext)<br/> | Cette interface facultative permet au gestionnaire de protocole d’effectuer une action sur le thread utilisé pour le filtrage dans l’hôte de protocole. Lorsque l’hôte de protocole démarre, il initialise d’abord tous les gestionnaires de protocole, puis crée le ou les threads de filtrage. Les méthodes sur cette interface permettent aux gestionnaires de protocole de gérer leurs ressources qui sont utilisées par un thread de filtrage.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                            |
| [**ISearchProtocolUI**](-search-isearchprotocolui.md)<br/>                       | Fournit une méthode pour appeler des objets [**ISearchItem**](-search-isearchitem.md) . Les méthodes de cette interface sont appelées par l’hôte de protocole lors du traitement des URL à partir du Rassembleur. Le gestionnaire de protocole implémente le protocole d’accès à une source de contenu dans son format natif, et cette interface implémente un gestionnaire de protocole personnalisé pour développer les sources de données qui peuvent être indexées.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                |
| [**IUrlAccessor**](/windows/desktop/api/Searchapi/nn-searchapi-iurlaccessor)<br/>                                 | Fournit des méthodes pour le traitement d’un élément individuel dans une source de contenu dont l’URL est fournie par le rassembleur à l’hôte de filtre. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| [**IUrlAccessor2**](/windows/desktop/api/Searchapi/nn-searchapi-iurlaccessor2)<br/>                               | Étend les fonctionnalités de l’interface [**IUrlAccessor**](/windows/desktop/api/Searchapi/nn-searchapi-iurlaccessor) .<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| [**IUrlAccessor3**](/windows/desktop/api/Searchapi/nn-searchapi-iurlaccessor3)<br/>                               | Étend les fonctionnalités de l’interface [**IUrlAccessor2**](/windows/desktop/api/Searchapi/nn-searchapi-iurlaccessor2) avec la méthode [**IUrlAccessor3 :: GetImpersonationSidBlobs**](/windows/desktop/api/Searchapi/nf-searchapi-iurlaccessor3-getimpersonationsidblobs) pour identifier les identificateurs de sécurité (SID) utilisateur pour une URL spécifiée.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| [**IUrlAccessor4**](/windows/desktop/api/Searchapi/nn-searchapi-iurlaccessor4)<br/>                               | Étend les fonctionnalités de l’interface [**IUrlAccessor3**](/windows/desktop/api/Searchapi/nn-searchapi-iurlaccessor3) avec la méthode [**IUrlAccessor4 :: ShouldIndexItemContent**](/windows/desktop/api/Searchapi/nf-searchapi-iurlaccessor4-shouldindexitemcontent) qui identifie si le contenu de l’élément doit être indexé. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
