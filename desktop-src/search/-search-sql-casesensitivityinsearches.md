# <a name="case-sensitivity-in-searches"></a>Respect de la casse dans les recherches

Les requêtes de recherche Microsoft Windows ne respectent pas la casse. Cela comprend les mots clés de requête (SELECT est identique à Select, SELECT et sELect) et les termes de recherche. Si un identificateur ou un littéral utilise des caractères Unicode qui ont une sémantique de mappage de casse, le moteur de recherche considère tous les cas comme équivalents.
