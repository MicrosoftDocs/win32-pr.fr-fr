# <a name="full-text-predicates"></a>Prédicats de Full-Text

Le langage de requête de recherche Microsoft Windows prend en charge deux prédicats de recherche en texte intégral. Le prédicat CONTAINs effectue des comparaisons sur les colonnes qui contiennent du texte. La clause CONTAINs peut effectuer une correspondance sur des mots ou expressions uniques, en fonction de la proximité des termes de recherche. En comparaison, le prédicat FREETEXT est paramétré pour correspondre à la signification des expressions de recherche par rapport aux colonnes de texte.

Cette section contient les rubriques suivantes :

-   [CONTAINs, prédicat](-search-sql-contains.md)
-   [Prédicat FREETEXT](-search-sql-freetext.md)
