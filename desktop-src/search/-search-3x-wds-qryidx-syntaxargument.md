# <a name="syntax-argument-windows-search"></a>Argument de syntaxe (Windows Search)

L' `syntax` argument vous permet d’identifier la syntaxe de votre requête. Les valeurs autorisées sont une syntaxe de requête naturelle (NQS) ou une syntaxe de requête avancée (AQS). AQS est la valeur par défaut et est toujours supposé analysé et pris en charge.

Cette rubrique est organisée comme suit :

-   [Exemple](#example)
-   [Rubriques connexes](#related-topics)

## <a name="example"></a>Exemple


```
 search-ms:query=from:paolo&syntax:AQS
```
