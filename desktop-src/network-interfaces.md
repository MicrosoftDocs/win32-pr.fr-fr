<span data-ttu-id="9f84a-108">Une *interface réseau* est le point où deux éléments de l’équipement réseau ou les couches de protocole se connectent.</span><span class="sxs-lookup"><span data-stu-id="9f84a-108">A *network interface* is the point where two pieces of network equipment or protocol layers connect.</span></span> <span data-ttu-id="9f84a-109">En règle générale, il est représenté par une carte d’interface réseau (NIC) physique pour la connexion entre un ordinateur et un réseau privé ou public.</span><span class="sxs-lookup"><span data-stu-id="9f84a-109">Typically, this is represented by a physical Network Interface Card (NIC) for connection between a computer and a private or public network.</span></span> <span data-ttu-id="9f84a-110">Toutefois, il peut également prendre la forme d’un composant logiciel uniquement, tel que l’interface de bouclage ( `127.0.0.1` pour IPv4 ou `::1` pour IPv6).</span><span class="sxs-lookup"><span data-stu-id="9f84a-110">However, it can also take the form of a software-only component such as the loopback interface (`127.0.0.1` for IPv4 or `::1` for IPv6).</span></span>
