# <a name="system-services"></a>Services système

cette section contient la référence pour les api système et les services offerts par Windows pour les applications de bureau. Celles-ci incluent les services traditionnellement disponibles pour :

-   Modèle COM (Component Object Model).
-   Compression de fichier.
-   Bibliothèques de liens dynamiques.
-   Gestion de la mémoire
-   Gestion de l’alimentation.
-   La création et la coordination de plusieurs threads d’exécution.
-   Le développement d’applications de service.
-   messagerie Windows.
-   obtention d’informations sur le système de Windows.
-   API d’aide.
