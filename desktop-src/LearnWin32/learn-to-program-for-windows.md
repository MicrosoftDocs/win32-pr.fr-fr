# <a name="get-started-with-win32-and-c"></a>Bien démarrer avec Win32 et C++

l’objectif de cette série de Prise en main est de vous apprendre à écrire un programme de bureau en C++ à l’aide des api Win32 et COM.

Dans le premier module, vous apprendrez comment créer et afficher une fenêtre. Les modules ultérieurs présentent le modèle COM (Component Object Model), les graphiques et le texte, ainsi que l’entrée utilisateur.

Pour cette série, il est supposé que vous avez une bonne connaissance pratique de la programmation C++. aucune expérience précédente avec la programmation de Windows n’est supposée. Si vous débutez en C++, le matériel de formation est availavble dans la [documentation du langage C++](https://msdn.microsoft.com/vstudio//default.aspx).
