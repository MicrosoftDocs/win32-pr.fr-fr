# <a name="nlm-structures"></a>Structures NLM

Les structures suivantes prennent en charge l’API Network List Manager :



| Structure                                                           | Description                                                          |
|---------------------------------------------------------------------|----------------------------------------------------------------------|
| [**\_ \_ informations de profil simulé \_ NLM**](/windows/desktop/api/Netlistmgr/ns-netlistmgr-nlm_simulated_profile_info) | Représente les informations de profil à utiliser pendant la simulation de coût.    |
| [**\_sockaddr NLM**](/windows/desktop/api/Netlistmgr/ns-netlistmgr-nlm_sockaddr)                               | Représente l’adresse de destination IPv4/IPv6.                        |
| [**\_données d’utilisation NLM \_**](/windows/desktop/api/Netlistmgr/ns-netlistmgr-nlm_usage_data)                          | Représente des informations qui indiquent l’utilisation des données d’un plan de données. |
| [**\_État du \_ plan de données NLM \_**](https://www.bing.com/search?q=**NLM\_DATA\_PLAN\_STATUS**)             | Représente l’état actuel d’un plan de données.                        |
