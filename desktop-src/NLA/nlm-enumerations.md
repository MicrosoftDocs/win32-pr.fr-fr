# <a name="nlm-enumerations"></a>Énumérations NLM

Voici les énumérations NLM.

| Énumération                                                                 | Description                                                                                |
|-----------------------------------------------------------------------------|--------------------------------------------------------------------------------------------|
| [**\_coût de connexion NLM \_**](/windows/desktop/api/Netlistmgr/ne-netlistmgr-nlm_connection_cost)                        | spécifie un ensemble de niveaux de coût et d’indicateurs de coût pris en charge dans Windows 8 api de coût.            |
| [**modification de la \_ propriété de connexion NLM \_ \_**](/windows/desktop/api/Netlistmgr/ne-netlistmgr-nlm_connection_property_change) | Définit les modifications apportées aux propriétés d’une connexion réseau.                            |
| [**\_connectivité NLM**](/windows/desktop/api/Netlistmgr/ne-netlistmgr-nlm_connectivity)                               | Fournit une notification chaque fois que des paramètres liés à la connectivité ont été modifiés.               |
| [**\_type de domaine NLM \_**](/windows/desktop/api/Netlistmgr/ne-netlistmgr-nlm_domain_type)                                | Spécifie le type de domaine d’un réseau.                                                    |
| [**\_réseau d’énumération NLM \_**](/windows/desktop/api/Netlistmgr/ne-netlistmgr-nlm_enum_network)                              | Spécifie les types de réseaux énumérés.                                           |
| [**\_connectivité Internet \_ NLM**](/windows/desktop/api/Netlistmgr/ne-netlistmgr-nlm_internet_connectivity)            | Définit un jeu d’indicateurs qui fournissent des données supplémentaires pour la connectivité réseau IPv4 ou IPv6. |
| [**\_classe de réseau NLM \_**](/windows/desktop/api/Netlistmgr/ne-netlistmgr-nlm_network_class)                            | Définit un jeu d’indicateurs qui spécifient si un réseau a été identifié.                      |
| [**\_catégorie de réseau NLM \_**](/windows/desktop/api/Netlistmgr/ne-netlistmgr-nlm_network_category)                      | Spécifie le type de catégorie d’un réseau.                                                  |
| [**\_modification des \_ Propriétés du réseau NLM \_**](/windows/desktop/api/Netlistmgr/ne-netlistmgr-nlm_network_property_change)       | Définit les modifications apportées aux propriétés d’un réseau.                                       |
