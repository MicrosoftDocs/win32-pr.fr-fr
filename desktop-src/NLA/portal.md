# <a name="network-list-manager"></a>Gestionnaire de listes de réseaux

## <a name="purpose"></a>Objectif

L’API du gestionnaire de listes de réseaux permet aux applications de récupérer une liste des connexions réseau disponibles. Les applications peuvent filtrer les réseaux, en fonction des attributs et des signatures, et choisir les réseaux les mieux adaptés à leur tâche. L’infrastructure du gestionnaire de listes de réseaux avertit les applications des modifications apportées à l’environnement réseau, ce qui permet aux applications de mettre à jour dynamiquement les connexions réseau.

## <a name="developer-audience"></a>Développeurs concernés

Le gestionnaire de listes de réseaux est conçu pour être utilisé par les programmeurs COM.

## <a name="run-time-requirements"></a>Conditions d’exécution

l’API du gestionnaire de listes de réseaux est prise en charge à partir de Windows Vista.
