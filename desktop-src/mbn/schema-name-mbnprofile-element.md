# <a name="name-mbnprofile-element"></a>Élément Name (MBNProfile)

L’élément **Name (MBNProfile)** identifie le nom du profil.

Il s’agit d’une chaîne d’une longueur maximale de 255 caractères.

L’élément est obligatoire.

``` syntax
<xs:element name="Name"
    type="nameType"
 />
```

L’élément **Name** est défini par l’élément [**MBNProfile**](schema-mbnprofile-element.md) .

|-------------------------------------|---------------------------------------------------|
