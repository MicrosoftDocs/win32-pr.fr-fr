# <a name="mobile-broadband"></a>Haut débit mobile

## <a name="purpose"></a>Objectif

L’API haut débit mobile est utilisée pour implémenter la connectivité aux réseaux cellulaires. Les applications ne doivent pas communiquer directement avec ces appareils haut débit mobile. Au lieu de cela, ils doivent utiliser l’API haut débit mobile.

## <a name="developer-audience"></a>Développeurs concernés

L’API haut débit mobile est conçue pour les développeurs C++.

Il existe également une interface pour les développeurs .NET utilisant C \# . Pour plus d’informations, consultez ce [livre blanc](https://www.microsoft.com/whdc/connect/wireless/MB_ManagedCode.mspx).

L’API haut débit mobile peut être utilisée par des applications tierces qui doivent contrôler et gérer les interfaces haut débit mobile.

## <a name="run-time-requirements"></a>Conditions d’exécution

la prise en charge de la bande passante Mobile commence par Windows 7.
