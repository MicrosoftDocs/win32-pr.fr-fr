# <a name="span-idwwan_profile_v2element_displayprovidernamespandisplayprovidername"></a><span id="WWAN_profile_v2.element_DisplayProviderName"></span>DisplayProviderName

l’élément [**DisplayProviderName**](element-displayprovidername.md) est un [**providerNameType**](./schema-providernametype-simpletype.md) facultatif qui contient le nom de la connexion réseau à afficher dans le gestionnaire de connexions Windows. Ce nom s’affiche uniquement si l’abonné se trouve sur un réseau privé et non en itinérance. Le nom du réseau itinérant s’affiche en fonction des informations de l’appareil haut débit mobile.

## <a name="element-hierarchy"></a>Hiérarchie d’éléments

**<DisplayProviderName>**

## <a name="syntax"></a>Syntax

``` syntax
<DisplayProviderName>

  providerNameType

</DisplayProviderName>
```
