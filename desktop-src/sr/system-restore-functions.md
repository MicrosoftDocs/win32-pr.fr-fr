# <a name="system-restore-functions"></a>Fonctions de restauration du système

Les fonctions suivantes sont utilisées avec la restauration du système.



| Fonction                                             | Description                                                                                                   |
|------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [**SRRemoveRestorePoint**](/windows/desktop/api/SRRestorePtAPI/nf-srrestoreptapi-srremoverestorepoint) | Supprime le point de restauration spécifié.                                                                          |
| [**SRSetRestorePoint**](/windows/desktop/api/SRRestorePtAPI/nf-srrestoreptapi-srsetrestorepointa)       | Spécifie le début et la fin d’un ensemble de modifications afin que la restauration du système puisse créer un point de restauration. |
