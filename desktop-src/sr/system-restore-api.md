# <a name="system-restore-api"></a>API restauration du système

L’API restauration du système permet à une application de gérer des points de restauration. une interface scriptable est également disponible via [Windows Management Instrumentation](/windows/desktop/WmiSdk/wmi-start-page) (WMI).

Pour plus d’informations sur l’API de restauration du système, consultez les rubriques suivantes :

-   [Appel de SRSetRestorePoint](calling-srsetrestorepoint.md)
-   [Texte de description du point de restauration](restore-point-description-text.md)
-   [Appels imbriqués à SRSetRestorePoint](nested-calls-to-srsetrestorepoint.md)

Pour plus d’informations sur les classes WMI de restauration du système, consultez Configuration de la [restauration du système](configuring-system-restore.md).
