# <a name="function-status"></a>État de la fonction

Les fonctions de gestion de réseau retournent zéro en cas de réussite ; un code de retour différent de zéro indique une erreur. Étant donné que les fonctions de gestion de réseau utilisent RPC, les définitions d’erreurs incluent les codes d’erreur RPC. Pour plus d’informations, consultez [codes d’erreur de gestion de réseau](network-management-error-codes.md).
