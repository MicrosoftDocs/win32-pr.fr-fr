# <a name="netserviceinstall"></a>NetServiceInstall

La fonction **NetServiceInstall** est obsolète. Il est inclus pour la compatibilité avec les versions 16 bits de Windows. D’autres applications doivent utiliser les [fonctions de service](/windows/desktop/Services/service-functions).
