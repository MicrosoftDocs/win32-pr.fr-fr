<span data-ttu-id="4c4b8-123">Les *connexions* sont établies par utilisateur.</span><span class="sxs-lookup"><span data-stu-id="4c4b8-123">*Connections* are made on a per-user basis.</span></span> <span data-ttu-id="4c4b8-124">Une connexion établie par un utilisateur est supprimée lorsque l’utilisateur se déconnecte.</span><span class="sxs-lookup"><span data-stu-id="4c4b8-124">A connection made by a user is deleted when that user logs off.</span></span> <span data-ttu-id="4c4b8-125">Pour cette raison, les fonctions d’utilisation de la gestion du réseau sont locales uniquement, car une connexion configurée par un utilisateur distant n’est pas accessible à d’autres utilisateurs, même l’utilisateur qui a ouvert une session de manière interactive sur cet ordinateur.</span><span class="sxs-lookup"><span data-stu-id="4c4b8-125">For this reason the network management use functions are local only, because a connection set up by a remote user would not be accessible to any other users, even the user that was interactively logged on to that computer.</span></span>

<span data-ttu-id="4c4b8-126">La fonction [**NetUseAdd**](/windows/desktop/api/Lmuse/nf-lmuse-netuseadd) établit une connexion explicite entre l’ordinateur local et une ressource partagée sur un serveur en redirigeant un nom de périphérique local vers le nom de partage d’une ressource de serveur distant ( \\ \\ *ServerName* \\ *Nom_Partage*).</span><span class="sxs-lookup"><span data-stu-id="4c4b8-126">The [**NetUseAdd**](/windows/desktop/api/Lmuse/nf-lmuse-netuseadd) function establishes an explicit connection between the local computer and a resource shared on a server by redirecting a local device name to the share name of a remote server resource (\\\\*servername*\\*sharename*).</span></span> <span data-ttu-id="4c4b8-127">Une fois qu’une connexion de nom d’appareil est établie, les utilisateurs ou les applications peuvent utiliser la ressource distante en spécifiant le nom de l’appareil local.</span><span class="sxs-lookup"><span data-stu-id="4c4b8-127">Once a device-name connection is made, users or applications can use the remote resource by specifying the local device name.</span></span>
