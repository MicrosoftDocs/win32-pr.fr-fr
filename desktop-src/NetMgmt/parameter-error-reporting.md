# <a name="parameter-error-reporting"></a>Rapport d’erreurs de paramètre

Les fonctions de gestion de réseau Add et SetInfo renvoient un index pour un paramètre qui est erroné. L’appelant peut passer un pointeur **null** pour le paramètre de paramètre *\_ Err* indiquant que le champ ne doit pas être défini par la fonction.
