# <a name="netservicegetinfo"></a>NetServiceGetInfo

La fonction **NetServiceGetInfo** est obsolète. Il est inclus pour la compatibilité avec les versions 16 bits de Windows. D’autres applications doivent utiliser les [fonctions de service](/windows/desktop/Services/service-functions).
