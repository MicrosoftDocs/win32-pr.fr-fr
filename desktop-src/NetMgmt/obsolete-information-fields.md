# <a name="obsolete-information-fields"></a>Champs d’informations obsolètes

La plupart des champs d’informations dans les structures d’informations de base seront obsolètes. Ces champs restent dans la structure des informations pour assurer la compatibilité avec les versions 16 bits de Windows et retournent une valeur par défaut intelligente sur les autres systèmes Windows.
