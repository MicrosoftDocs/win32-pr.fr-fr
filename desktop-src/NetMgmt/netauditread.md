# <a name="netauditread"></a>NetAuditRead

La fonction **NetAuditRead** est obsolète. Il est inclus pour la compatibilité avec les versions 16 bits de Windows. D’autres applications doivent utiliser la [journalisation des événements](/windows/desktop/EventLog/event-logging).
