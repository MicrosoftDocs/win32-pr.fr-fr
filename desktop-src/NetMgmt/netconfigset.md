# <a name="netconfigset"></a>NetConfigSet

La fonction **NetConfigSet** est obsolète. Il est inclus pour la compatibilité avec les versions 16 bits de Windows. D’autres applications doivent utiliser le [Registre](/windows/desktop/SysInfo/registry).
