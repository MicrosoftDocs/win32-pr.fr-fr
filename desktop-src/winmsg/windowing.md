# <a name="windows-and-messages"></a>Fenêtres et messages

Les sections suivantes décrivent les éléments d’une application avec une interface utilisateur graphique Windows.

### <a name="in-this-section"></a>Dans cette section



| Nom                                                           | Description                                                                                                                                                                                                                  |
|----------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Windows](windows.md)                                         | Décrit les fenêtres en général.<br/>                                                                                                                                                                                     |
| [classes de fenêtre](window-classes.md)                           | Décrit les types de classes de fenêtres, comment le système les localise et les éléments qui définissent le comportement par défaut des fenêtres qui leur appartiennent.<br/>                                                             |
| [Procédures de fenêtre](window-procedures.md)                     | Traite des procédures de fenêtre. Chaque fenêtre est associée à une procédure de fenêtre qui traite tous les messages envoyés ou publiés dans toutes les fenêtres de la classe.<br/>                                                              |
| [Messages et files d’attente de messages](messages-and-message-queues.md) | Décrit les messages et les files d’attente de messages et comment les utiliser dans vos applications.<br/>                                                                                                                                   |
| [Minuteurs](timers.md)                                           | Traite des minuteries. Un minuteur est une routine interne qui mesure à plusieurs reprises un intervalle spécifié, en millisecondes.<br/>                                                                                                  |
| [Propriétés de la fenêtre](window-properties.md)                     | Décrit les propriétés de la fenêtre. Une propriété de fenêtre correspond à toutes les données affectées à une fenêtre.<br/>                                                                                                                                  |
| [Configuration](configuration.md)                             | Décrit les fonctions qui peuvent être utilisées pour contrôler la configuration des métriques système et divers attributs système tels que l’heure du double-clic, le délai d’attente de l’économiseur d’écran, la largeur de la bordure de la fenêtre et le modèle de bureau.<br/> |
| [Hooks](hooks.md)                                             | Décrit les hooks. Un hook est un point dans le mécanisme de gestion des messages système où une application peut installer une sous-routine pour surveiller le trafic des messages.<br/>                                                         |
| [Interface multidocument](multiple-document-interface.md) | Présente l’interface multidocument qui est une spécification qui définit une interface utilisateur pour les applications qui permettent à l’utilisateur de travailler avec plusieurs documents en même temps.<br/>                      |
