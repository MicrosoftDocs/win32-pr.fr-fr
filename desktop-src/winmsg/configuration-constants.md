# <a name="configuration-constants"></a>Constantes de configuration

Cette section fournit les spécifications de référence pour les constantes [**SystemParametersInfo**](/windows/win32/api/winuser/nf-winuser-systemparametersinfoa) liées aux attributs du système de [configuration](configuration.md) .

## <a name="in-this-section"></a>Contenu de cette section



| Rubrique                                                             | Description                                                                                                                                                            |
|-------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Visualisation des contacts**](contact-visualization.md)<br/> | Les constantes suivantes sont utilisées par les applications ou les infrastructures d’interface utilisateur pour identifier comment les commentaires de l’interface utilisateur sont traités lorsqu’un contact d’entrée est détecté.<br/>               |
| [**Visualisation de mouvement**](gesture-visualization.md)<br/> | Les constantes suivantes sont utilisées par les applications ou les infrastructures d’interface utilisateur pour identifier comment les commentaires de l’interface utilisateur sont traités lorsque l’un des mouvements répertoriés est détecté.<br/>     |
| [**Visualisation du stylet**](pen-visualization.md)<br/>         | Les constantes suivantes sont utilisées par les applications ou les infrastructures d’interface utilisateur pour identifier comment les commentaires de l’interface utilisateur sont traités lorsque l’un des mouvements de stylet répertoriés est détecté.<br/> |
