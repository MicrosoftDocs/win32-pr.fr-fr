# <a name="windows-windows-and-messages"></a>Fenêtres (fenêtres et messages)

Dans une application graphique Windows, une fenêtre est une zone rectangulaire de l’écran dans laquelle l’application affiche la sortie et reçoit l’entrée de l’utilisateur. Par conséquent, l’une des premières tâches d’une application graphique basée sur Windows consiste à créer une fenêtre.

Une fenêtre partage l’écran avec d’autres fenêtres, y compris celles d’autres applications. Une seule fenêtre à la fois peut recevoir l’entrée de l’utilisateur. L’utilisateur peut utiliser la souris, le clavier ou un autre périphérique d’entrée pour interagir avec cette fenêtre et l’application qui le possède.
