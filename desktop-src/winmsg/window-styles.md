# <a name="window-styles"></a>Styles de fenêtre

Les styles de fenêtre sont les suivants. Une fois la fenêtre créée, ces styles ne peuvent pas être modifiés, sauf indication contraire.



| Constante/valeur                                                                                                                                                                                                                                                                                                                              | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="WS_BORDER"></span><span id="ws_border"></span><dl> <dt>**WS \_ BORDURE**</dt> <dt>0x00800000L</dt> </dl>                                                                                                                          | La fenêtre a une bordure fine.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| <span id="WS_CAPTION"></span><span id="ws_caption"></span><dl> <dt>**WS \_ LÉGENDE**</dt> <dt>0x00C00000L</dt> </dl>                                                                                                                       | La fenêtre a une barre de titre (inclut le style **WS \_ Border** ).<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| <span id="WS_CHILD"></span><span id="ws_child"></span><dl> <dt>**WS \_**</dt> <dt>0x40000000L</dt> enfants </dl>                                                                                                                             | La fenêtre est une fenêtre enfant. Une fenêtre avec ce style ne peut pas avoir de barre de menus. Ce style ne peut pas être utilisé avec le style de **\_ fenêtre contextuelle WS** .<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| <span id="WS_CHILDWINDOW"></span><span id="ws_childwindow"></span><dl> <dt>**WS \_ CHILDWINDOW**</dt> <dt>0x40000000L</dt> </dl>                                                                                                           | Identique au style **WS \_ Child** .<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |

Ce style est utilisé lors de la création de la fenêtre parente.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| <span id="WS_CLIPSIBLINGS"></span><span id="ws_clipsiblings"></span><dl> <dt>**WS \_ CLIPSIBLINGS**</dt> <dt>0x04000000L</dt> </dl>                                                                                                        | Découpe les fenêtres enfants les unes par rapport aux autres ; autrement dit, lorsqu’une fenêtre enfant particulière reçoit un message de [**\_ peinture WM**](../gdi/wm-paint.md) , le style **WS \_ CLIPSIBLINGS** découpe toutes les autres fenêtres enfants qui se chevauchent en dehors de la région de la fenêtre enfant à mettre à jour. Si **WS \_ CLIPSIBLINGS** n’est pas spécifié et que les fenêtres enfants se chevauchent, il est possible, lors du dessin dans la zone cliente d’une fenêtre enfant, de dessiner dans la zone cliente d’une fenêtre enfant voisine.<br/>                                                                                                                                                                                |
| <span id="WS_DISABLED"></span><span id="ws_disabled"></span><dl> <dt>**WS \_ Désactivation**</dt> de <dt>0x08000000L</dt> </dl>                                                                                                                    | La fenêtre est initialement désactivée. Une fenêtre désactivée ne peut pas recevoir d’entrée de l’utilisateur. Pour modifier cette valeur après la création d’une fenêtre, utilisez la fonction [**EnableWindow**](/windows/win32/api/winuser/nf-winuser-enablewindow) .<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| <span id="WS_DLGFRAME"></span><span id="ws_dlgframe"></span><dl> <dt>**WS \_ DLGFRAME**</dt> <dt>0x00400000L</dt> </dl>                                                                                                                    | La fenêtre a une bordure d’un style généralement utilisé avec les boîtes de dialogue. Une fenêtre avec ce style ne peut pas avoir de barre de titre.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| <span id="WS_GROUP"></span><span id="ws_group"></span><dl> <dt>**WS \_**</dt> <dt>0X00020000L</dt> de groupe </dl>                                                                                                                             | La fenêtre est le premier contrôle d’un groupe de contrôles. Le groupe se compose de ce premier contrôle et de tous les contrôles définis après celui-ci, jusqu’au contrôle suivant avec le style de **\_ groupe WS** . Le premier contrôle dans chaque groupe a généralement le style **WS \_ TABSTOP** afin que l’utilisateur puisse se déplacer d’un groupe à un groupe. L’utilisateur peut ensuite changer le focus clavier d’un contrôle du groupe au contrôle suivant dans le groupe à l’aide des touches de direction.<br/> Vous pouvez activer ou désactiver ce style pour modifier la navigation dans la boîte de dialogue. Pour modifier ce style après la création d’une fenêtre, utilisez la fonction [**SetWindowLong**](/windows/win32/api/winuser/nf-winuser-setwindowlonga) .<br/> |
| <span id="WS_HSCROLL"></span><span id="ws_hscroll"></span><dl> <dt>**WS \_ HSCROLL**</dt> <dt>0x00100000L</dt> </dl>                                                                                                                       | La fenêtre a une barre de défilement horizontale.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| <span id="WS_ICONIC"></span><span id="ws_iconic"></span><dl> <dt>**WS \_ SOUS forme**</dt> <dt>0x20000000L</dt> </dl>                                                                                                                          | La fenêtre est initialement réduite. Identique au style **WS \_ Minimize** .<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| <span id="WS_MAXIMIZE"></span><span id="ws_maximize"></span><dl> <dt>**WS \_ MAXIMISER**</dt> <dt>0x01000000L</dt> </dl>                                                                                                                    | La fenêtre est initialement agrandie.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| <span id="WS_MAXIMIZEBOX"></span><span id="ws_maximizebox"></span><dl> <dt>**WS \_ MAXIMIZEBOX**</dt> <dt>0x00010000L</dt> </dl>                                                                                                           | La fenêtre a un bouton Agrandir. Ne peut pas être combiné avec le style **WS \_ ex \_ CONTEXTHELP** . Le style **WS \_ SYSMENU** doit également être spécifié. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| <span id="WS_MINIMIZE"></span><span id="ws_minimize"></span><dl> <dt>**WS \_ RÉDUIRE**</dt> <dt>0x20000000L</dt> </dl>                                                                                                                    | La fenêtre est initialement réduite. Identique au style **WS \_ sous forme** .<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| <span id="WS_MINIMIZEBOX"></span><span id="ws_minimizebox"></span><dl> <dt>**WS \_ MINIMIZEBOX**</dt> <dt>0x00020000L</dt> </dl>                                                                                                           | La fenêtre a un bouton réduire. Ne peut pas être combiné avec le style **WS \_ ex \_ CONTEXTHELP** . Le style **WS \_ SYSMENU** doit également être spécifié. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| <span id="WS_OVERLAPPED"></span><span id="ws_overlapped"></span><dl> <dt>**WS \_**</dt> <dt>0X00000000L</dt> avec chevauchement </dl>                                                                                                              | La fenêtre est une fenêtre Overlapped. Une fenêtre avec chevauchement a une barre de titre et une bordure. Identique au style **WSD en \_ mosaïque** .<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| <span id="WS_OVERLAPPEDWINDOW"></span><span id="ws_overlappedwindow"></span><dl> <dt>**WS \_ OVERLAPPEDWINDOW**</dt> <dt>(WS \_ OVERLAPPED WS \| \_ Caption \| \_ SYSMENU \| WS \_ THICKFRAME \| WS \_ MINIMIZEBOX \| WS \_ MAXIMIZEBOX)</dt> </dl> | La fenêtre est une fenêtre Overlapped. Ce style ne peut pas être utilisé avec le style **WS \_ Child** .<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| <span id="WS_POPUPWINDOW"></span><span id="ws_popupwindow"></span><dl> <dt>**WS \_ POPUPWINDOW**</dt> <dt>(WS \_ contextuel WS \| \_ Border \| WS \_ SYSMENU)</dt> </dl>                                                                              | La fenêtre est une fenêtre indépendante. Les styles **WS \_ Caption** et **WS \_ POPUPWINDOW** doivent être combinés pour rendre le menu fenêtre visible.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| <span id="WS_SIZEBOX"></span><span id="ws_sizebox"></span><dl> <dt>**WS \_ SIZEBOX**</dt> <dt>0x00040000L</dt> </dl>                                                                                                                       | La fenêtre a une bordure de redimensionnement. | <span id="WS_SYSMENU"></span><span id="ws_sysmenu"></span><dl> <dt>**WS \_ SYSMENU**</dt> <dt>0x00080000L</dt> </dl>                                                                                                                       | La fenêtre a un menu fenêtre sur sa barre de titre. Pour modifier ce style après la création d’une fenêtre, utilisez la fonction [**SetWindowLong**](/windows/win32/api/winuser/nf-winuser-setwindowlonga) . Pour les fenêtres créées par l’utilisateur et les boîtes de dialogue non modales pour travailler avec des taquets de tabulation, modifiez la boucle de message pour appeler la fonction [**IsDialogMessage**](/windows/win32/api/winuser/nf-winuser-isdialogmessagea) .<br/>                                                                                           |
