# <a name="dictionaries"></a>Dictionnaires

Les dictionnaires autorisent la création de jetons pour les données de nœud courantes, telles que les noms d’éléments/attributs, les espaces de noms et le texte commun dans un format XML binaire.


Un [**>de dictionnaire**](/windows/desktop/api/WebServices/ns-webservices-ws_xml_dictionary) peut être spécifié dans le cadre du type de [**\_ \_ chaîne WS XML**](/windows/desktop/api/WebServices/ns-webservices-ws_xml_string) . Toutes les chaînes d’un seul **dictionnaire>** doivent être uniques.

Les ID de dictionnaire sont pris en charge pour tous les encodages (pas seulement le format XML binaire).

La fonction suivante est utilisée avec les dictionnaires :

-   [**WsGetDictionary**](/windows/desktop/api/WebServices/nf-webservices-wsgetdictionary)
