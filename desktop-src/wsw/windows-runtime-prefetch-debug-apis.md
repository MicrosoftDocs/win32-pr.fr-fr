# <a name="prefetch-debug-apis-for-windows-store-apps"></a>prérécupération des api de débogage pour les applications Windows store

les api suivantes sont utilisées pour déboguer et tester le comportement de prérécupération de contenu dans une application de Store Windows qui implémente la classe [**ContentPrefetcher**](/uwp/api/Windows.Networking.BackgroundTransfer.ContentPrefetcher) .



| Interface                                                              | Description                                                                                                                                                                      |
|------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IContentPrefetcherTaskTrigger**](/windows/desktop/api/IContentPrefetcherTaskTrigger/nn-icontentprefetchertasktrigger-icontentprefetchertasktrigger) | utilisez cette interface pour vérifier qu’un package d’application Windows Store installé est inscrit pour cette tâche en arrière-plan et déclencher manuellement ses opérations de prérécupération de contenu.<br/> |
