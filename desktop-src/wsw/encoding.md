# <a name="encoding-windows-web-services"></a>Encodage (services Web Windows)

L’encodage est une transformation entre XML et un format de câble linéaire. Le décodage est la transformation inverse.


Un encodage est identifié à l’aide d’une constante d' [**\_ encodage WS**](/windows/desktop/api/WebServices/ne-webservices-ws_encoding) .
