# <a name="winhttp-interfaces"></a>Interfaces WinHTTP

[Microsoft Windows HTTP Services (WinHTTP)](about-winhttp.md) expose les deux interfaces suivantes :

-   [**IWinHttpRequest**](iwinhttprequest-interface.md) implémente des méthodes WinHTTP qui n’impliquent pas d’événements.
-   [**IWinHttpRequestEvents**](iwinhttprequestevents-interface.md) implémente les événements WinHTTP et les méthodes basées sur les événements.
