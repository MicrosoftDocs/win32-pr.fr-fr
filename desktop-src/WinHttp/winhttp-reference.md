# <a name="winhttp-reference"></a>Informations de référence sur WinHTTP

La documentation de référence sur les services HTTP Microsoft Windows (WinHTTP) est disponible pour les langues suivantes.

## <a name="reference"></a>Informations de référence

-   La référence de l' [API c/C++ WinHTTP](winhttp-c-c---api-reference.md) contient une documentation de référence pour l’utilisation de WinHTTP à partir de C ou c++.
-   La [référence d’objet WinHttpRequest](winhttprequest-object-reference.md) contient une documentation de référence pour l’utilisation de WinHTTP à partir de Visual Basic et de langages de script via l’objet com **WinHttpRequest** , ainsi que des informations supplémentaires pour les développeurs Visual Basic.
