# <a name="packaging-and-deployment-tools-and-powershell-cmdlets"></a>Outils de déploiement et d’empaquetage et applets de commande PowerShell

utilisez ces outils et applets de commande pour créer, déployer et inventorier des packages d’application pour les applications Windows.

## <a name="app-packaging-tools-and-guidance"></a>Outils et conseils relatifs à l’empaquetage d’applications

Découvrez comment empaqueter une application, créer un certificat de test et signer votre application à l’aide de l’invite de commandes et de PowerShell. Pour plus d’informations, voir :

-   [Créer un package d’application avec l’outil MakeAppx.exe](/windows/msix/package/create-app-package-with-makeappx-tool)
-   [Créer un certificat de signature de package](/windows/msix/package/create-certificate-package-signing)
-   [Signer un package d’application à l’aide de SignTool](/windows/msix/package/sign-app-package-using-signtool)

vous pouvez également utiliser Visual Studio pour créer et signer un package d’application. Pour plus d’informations, consultez [empaqueter une application UWP avec Visual Studio](/windows/msix/package/packaging-uwp-apps).

## <a name="powershell-cmdlets"></a>Applets de commande PowerShell

utilisez Windows PowerShell pour installer, inventorier et désinstaller des applications métier Windows qui ne sont pas signées par le Store. Pour plus d’informations, consultez : applets de commande du [module AppX](/powershell/module/appx/index?view=win10-ps).
