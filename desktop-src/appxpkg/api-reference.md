utilisez ces api dans vos applications de bureau pour créer, consommer, déployer ou interroger des packages d’application pour les applications Windows :

-   [API d’empaquetage](interfaces.md)
-   [API de déploiement de packages](package-deployment-api.md)

-   [**Windows. ApplicationModel. PackageId**](/uwp/api/Windows.ApplicationModel.PackageId)
