# <a name="packaging-deployment-and-query-apis"></a>API d’empaquetage, de déploiement et de requête

Utilisez ces API dans vos applications de bureau pour créer, consommer, déployer ou interroger des packages d’application pour les applications Windows :

-   [API d’empaquetage](interfaces.md)
-   [API de déploiement de packages](package-deployment-api.md)
-   [API de requête de package](functions.md)

Utilisez ces API dans vos applications de bureau pour déployer ou interroger des packages d’application :

-   [**Windows. ApplicationModel. Package**](/uwp/api/Windows.ApplicationModel.Package)
-   [**Windows. ApplicationModel. PackageId**](/uwp/api/Windows.ApplicationModel.PackageId)
-   [**Windows.Management.Deployment.PackageManager**](/uwp/api/Windows.Management.Deployment.PackageManager)
-   [**Windows.Management.Deployment.PackageUserInformation**](/uwp/api/Windows.Management.Deployment.PackageUserInformation)

Utilisez ces API dans vos applications UWP pour interroger des packages d’application :

-   [**Windows. ApplicationModel. Package**](/uwp/api/Windows.ApplicationModel.Package)
-   [**Windows. ApplicationModel. PackageId**](/uwp/api/Windows.ApplicationModel.PackageId)
