# <a name="idxcoreadapterfactory-interface"></a>Interface IDXCoreAdapterFactory

L’interface **IDXCoreAdapterFactory**   implémente des méthodes permettant de générer des objets d’énumération d’adaptateur dxcore et de récupérer leurs détails. **IDXCoreAdapterFactory** hérite de l’interface [IUnknown](/windows/win32/api/unknwn/nn-unknwn-iunknown) . Pour obtenir des conseils de programmation et des exemples de code, consultez [utilisation de dxcore pour énumérer des adaptateurs](../dxcore-enum-adapters.md).
