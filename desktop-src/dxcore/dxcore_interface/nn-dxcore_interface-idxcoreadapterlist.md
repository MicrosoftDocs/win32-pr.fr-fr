# <a name="idxcoreadapterlist-interface"></a>Interface IDXCoreAdapterList

L’interface **IDXCoreAdapterList** implémente des méthodes pour récupérer des éléments d’adaptateur à partir d’une liste générée, ainsi que des détails sur la liste. **IDXCoreAdapterList** hérite de l’interface [IUnknown](/windows/win32/api/unknwn/nn-unknwn-iunknown) . Pour obtenir des conseils de programmation et des exemples de code, consultez [utilisation de dxcore pour énumérer des adaptateurs](../dxcore-enum-adapters.md).
