# <a name="dxcore-interfaces"></a>Interfaces DXCore

Les interfaces suivantes, ainsi que leurs méthodes, sont déclarées dans dxcore_interface. h.

## <a name="in-this-section"></a>Contenu de cette section

| Rubrique | Description |
|-|-|
| [**IDXCoreAdapter**](./dxcore_interface/nn-dxcore_interface-idxcoreadapter.md) | L’interface **IDXCoreAdapter** implémente des méthodes pour récupérer des détails sur un élément de l’adaptateur. |
| [**IDXCoreAdapterFactory**](./dxcore_interface/nn-dxcore_interface-idxcoreadapterfactory.md) | L’interface **IDXCoreAdapterFactory** implémente des méthodes permettant de générer des objets d’énumération d’adaptateur dxcore et de récupérer leurs détails. |
| [**IDXCoreAdapterList**](./dxcore_interface/nn-dxcore_interface-idxcoreadapterlist.md) | L’interface **IDXCoreAdapterList** implémente des méthodes pour récupérer des éléments d’adaptateur à partir d’une liste générée, ainsi que des détails sur la liste. |
