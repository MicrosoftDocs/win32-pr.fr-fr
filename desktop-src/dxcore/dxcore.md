# <a name="dxcore"></a>DXCore

DXCore est une API d’énumération d’adaptateur pour les graphiques et les appareils de calcul, de sorte que certaines de ses fonctionnalités se chevauchent avec celles de [dxgi](../direct3ddxgi/dx-graphics-dxgi.md). DXCore est utilisé par Direct3D 12.

Cet ensemble de documentation contient des informations sur la programmation avec DXCore, ainsi que sur la référence DXCore.

## <a name="requirements"></a>Configuration requise

| &nbsp; | &nbsp; |
| ---- |:---- |
| **Environnements d’exécution pris en charge** | Windows/C++ |
| **Langages de programmation recommandés** | C++ |
| **Client minimal pris en charge** | Windows 10, version 2004 (10,0 ; Build 19041) |
| **En-tête** | dxcore. h et dxcore_interface. h |
| **Bibliothèque** | dxcore. lib |
| **DLL** | dxcore.dll |
