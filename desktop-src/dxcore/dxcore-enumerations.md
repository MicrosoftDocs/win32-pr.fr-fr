# <a name="dxcore-enumerations"></a>Énumérations DXCore

Les énumérations suivantes sont déclarées dans dxcore_interface. h.

## <a name="in-this-section"></a>Dans cette section

| Rubrique et description |
|-|
| [**DXCoreAdapterPreference**](./dxcore_interface/ne-dxcore_interface-dxcoreadapterpreference.md). Définit des constantes qui spécifient les préférences d’adaptateur DXCore à utiliser comme critères de tri de liste. |
| [**DXCoreAdapterProperty**](./dxcore_interface/ne-dxcore_interface-dxcoreadapterproperty.md). Définit des constantes qui spécifient les propriétés de l’adaptateur DXCore. |
| [**DXCoreAdapterState**](./dxcore_interface/ne-dxcore_interface-dxcoreadapterstate.md). Définit des constantes qui spécifient les genres d’États de l’adaptateur DXCore. |
| [**DXCoreNotificationType**](./dxcore_interface/ne-dxcore_interface-dxcorenotificationtype.md). Définit des constantes qui spécifient les types de notifications déclenchés par les objets [IDXCoreAdapter](./dxcore_interface/nn-dxcore_interface-idxcoreadapter.md) ou [IDXCoreAdapterList](./dxcore_interface/nn-dxcore_interface-idxcoreadapterlist.md) . |
| [**DXCoreSegmentGroup**](./dxcore_interface/ne-dxcore_interface-dxcoresegmentgroup.md). Définit des constantes qui spécifient le regroupement des segments de mémoire d’un adaptateur. |
