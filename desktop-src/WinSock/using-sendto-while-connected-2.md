# <a name="using-sendto-while-connected"></a>Utilisation de SendTo pendant la connexion

[**WSPSendTo**](/previous-versions/windows/desktop/legacy/ms742291(v=vs.85)) remet toujours les données à l’adresse spécifiée, même si un homologue désigné pour le socket d’envoi a été établi dans [**WSPConnect**](/previous-versions/windows/hardware/network/ff566275(v=vs.85)).
