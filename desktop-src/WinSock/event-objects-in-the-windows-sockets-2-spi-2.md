# <a name="event-objects-in-the-windows-sockets-2-spi"></a>objets d’événement dans le SPI Windows sockets 2

les objets d’événement sont introduits dans Windows sockets 2 comme un mécanisme de synchronisation général entre les fournisseurs de services Winsock et les applications. Ils sont utilisés à plusieurs fins, notamment pour indiquer l’achèvement des opérations avec chevauchement et l’occurrence d’un ou plusieurs événements réseau.
