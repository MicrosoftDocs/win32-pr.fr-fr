# <a name="broadcast-to-local-network"></a>Diffusion sur le réseau local

Une diffusion peut être effectuée sur le réseau attaché localement en définissant **sa \_ netnum** sur zéros binaires (0) et **sa \_ nodeNum** sur des valeurs binaires (1). Cette diffusion peut être envoyée au réseau principal de l’appareil, ou à tous les réseaux connectés localement à l’option du fournisseur de services. Les diffusions sur le réseau local ne sont pas propagées par les routeurs.
