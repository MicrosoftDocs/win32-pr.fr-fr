# <a name="usage-model"></a>Modèle d’utilisation

Les instructions d’utilisation et d’implémentation pour les applications et les services compatibles QoS sont fournies dans une section distincte du kit de développement logiciel (SDK) Microsoft Windows qui est dédié à la qualité de service (QoS). Pour plus d’informations sur le modèle d’utilisation de la [qualité de service (QoS), consultez la section Quality of service (QoS)](/previous-versions/windows/desktop/qos/qos-start-page) de la SDK Windows.
