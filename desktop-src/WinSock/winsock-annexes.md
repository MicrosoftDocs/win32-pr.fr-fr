# <a name="winsock-annexes"></a>Annexes Winsock

Les annexes Winsock fournissent des informations d’implémentation pour les suites de protocoles Winsock courantes et expliquent comment utiliser chaque protocole avec Winsock.

Chaque protocole de cette annexe comporte des conventions, des comportements ou des fonctionnalités spéciales qui ne se prêtent pas à une implémentation générique. Cette section décrit les détails que les développeurs doivent prendre en compte lors de l’implémentation ou de l’utilisation des protocoles décrits.

Les annexes Winsock incluent les sections suivantes :

-   [Annexe Winsock ATM](winsock-atm-annex.md)
-   [Annexe IPX/SPX Winsock](winsock-ipx-spx-annex.md)
-   [Winsock TCP/IP annexe](winsock-tcp-ip-annex.md)
