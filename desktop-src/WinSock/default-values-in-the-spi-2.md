# <a name="default-values-in-the-spi"></a>Valeurs par défaut dans l’index SPI

Les détails de l’implémentation de la qualité de service (QoS) sont fournis dans une section de QoS distincte et dédiée du kit de développement logiciel (SDK) Microsoft Windows. Pour plus d’informations sur les règles d’utilisation de QoS concernant les valeurs par défaut, consultez la section [Quality of service](/previous-versions/windows/desktop/qos/qos-start-page) de la SDK Windows.
