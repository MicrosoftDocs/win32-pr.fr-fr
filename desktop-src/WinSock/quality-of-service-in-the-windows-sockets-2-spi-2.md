# <a name="quality-of-service-in-the-windows-sockets-2-spi"></a>qualité de Service dans le SPI Windows sockets 2

la qualité de Service (QoS) est implémentée dans Windows par le biais de différents composants QoS pris en charge sur Windows Server 2003, Windows XP et Windows 2000. une explication détaillée de QoS et de ses composants se trouve dans une section consacrée à la qualité de Service, qui se trouve dans le kit de développement logiciel (SDK) de Microsoft Windows. Chaque composant QoS et le rôle qu’il joue dans l’implémentation de QoS sont expliqués dans cette section QoS, avec des conseils supplémentaires fournis pour l’implémentation d’applications et de services compatibles QoS. pour plus d’informations, consultez la section [qualité de Service (QoS)](/previous-versions/windows/desktop/qos/qos-start-page) de la SDK Windows.

Cette section décrit les fonctionnalités de qualité de service disponibles pour les développeurs Winsock. La liste suivante décrit les rubriques de cette section :

-   [Modèle d’utilisation](usage-model-2.md)
-   [Mises à jour QoS](qos-updates-2.md)
-   [Valeurs par défaut dans l’index SPI](default-values-in-the-spi-2.md)
-   [Modèles QoS dans le SPI](qos-templates-in-the-spi-2.md)
