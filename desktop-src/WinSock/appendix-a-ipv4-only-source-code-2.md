# <a name="appendix-a-ipv4-only-source-code"></a>Annexe A : code source uniquement IPv4

Cette annexe fournit le code source pour les exemples du kit de développement logiciel (SDK) Platform Simplec. c et simple. c, qui sont utilisés dans ce guide IPv6 comme exemples :

-   [Code client IPv4 uniquement](ipv4-only-client-code-2.md)
-   [Code serveur IPv4 uniquement](ipv4-only-server-code-2.md)

> [!WARNING]
>
> Ce code est inclus dans ce guide pour comparer le code IP-version agnostique fourni dans l’annexe B. Cet exemple de code est fourni à des fins de comparaison et fournit une référence claire et immédiatement disponible lors de la lecture de ce guide IPv6. Une version IP agnostique de ce code source est fournie dans [l’annexe B : code source agnostique de la version IP](appendix-b-ip-version-agnostic-source-code-2.md).

 

> [!Note]  
> En comparant le code source de l’annexe A (IPv4 uniquement) et de l’annexe B (adresse IP agnostique), vous avez une idée des modifications nécessaires à la modification de votre application Windows Sockets afin d’ajouter la prise en charge d’IPv6.
