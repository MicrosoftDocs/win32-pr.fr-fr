# <a name="directed-broadcast"></a>Diffusion dirigée

Généralement considéré comme plus convivial pour le réseau qu’une diffusion sur tous les itinéraires, une diffusion dirigée se limite au réseau local que vous spécifiez. Remplissez **sa \_ netnum** avec le numéro de réseau cible et **sa \_ nodeNum** avec des fichiers binaires. Les routeurs transmettent cette demande au réseau de destination où elle devient une diffusion locale. Les réseaux intermédiaires ne voient pas ce paquet comme une diffusion.
