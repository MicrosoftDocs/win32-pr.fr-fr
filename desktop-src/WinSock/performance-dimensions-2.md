# <a name="performance-dimensions"></a>Dimensions de performances

Les performances de l’application varient en fonction de la perspective. Pour un utilisateur, une application idéale offre des performances et une réactivité élevées. Pour un administrateur, une application idéale consommerait le moins de ressources réseau. Les performances diffèrent également en fonction du type d’application. Les performances d’une application de transfert de fichiers en bloc peuvent avoir des implications différentes par rapport aux performances d’une application transactionnelle.

Les sections suivantes traitent des dimensions de performance et des types d’application différents, et expliquent comment les environnements de mise en réseau affectent les performances.

-   [Applications transactionnelles et applications de streaming](transactional-versus-streaming-applications-2.md)
-   [Différents environnements réseau](different-network-environments-2.md)
