## <a name="updated-for-windows-8-and-windows-server-2012"></a>mise à jour pour Windows 8 et Windows Server 2012

Microsoft Windows 8 et Windows Server 2012 introduire de nouveaux éléments de programmation Windows sockets.

Un ensemble d’extensions de réseau à haut débit est disponible pour améliorer les performances de mise en réseau avec une latence et une instabilité inférieures. Ces extensions ciblées principalement pour les applications serveur utilisent des mémoires tampons de données pré-inscrites et des files d’attente de saisie semi-automatique pour améliorer les performances.

voici les nouvelles fonctions de sockets Windows ajoutées pour prendre en charge les extensions d’e/s de réseau à haut débit inscrites de Winsock :

-   [**RIOCloseCompletionQueue**](/previous-versions/windows/desktop/legacy/hh448837(v=vs.85))
