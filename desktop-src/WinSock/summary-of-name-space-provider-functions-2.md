# <a name="summary-of-namespace-provider-functions"></a>Résumé des fonctions du fournisseur d’espace de noms

Les fonctions du fournisseur de services d’espace de noms peuvent être regroupées en cinq catégories :

-   Configuration du fournisseur d’espace de noms (et installation)
-   Initialisation du fournisseur
-   Installation du service
-   Requêtes client
-   Fonctions d’assistance (et macros)

Les sections suivantes identifient les fonctions de chaque catégorie et décrivent brièvement leur utilisation prévue. Les structures de données clés sont également décrites.

-   [Configuration et installation du fournisseur d’espace de noms](name-space-provider-configuration-and-installation-2.md)
-   [Initialisation et nettoyage du fournisseur d’espace de noms](name-space-provider-initialization-and-cleanup-2.md)
-   [Installation du service dans le SPI Windows Sockets 2](service-installation-in-the-windows-sockets-2-spi-2.md)
-   [Requête de service](service-query-2.md)
-   [Fonctions d’assistance dans le SPI](helper-functions-in-the-spi-2.md)
