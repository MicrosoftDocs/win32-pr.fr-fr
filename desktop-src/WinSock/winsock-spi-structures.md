# <a name="winsock-spi-structures"></a>Structures Winsock SPI

La liste suivante fournit des descriptions concises de chaque structure Winsock SPI. Pour plus d’informations sur les structures Winsock SPI, cliquez sur le nom de la structure.



| Structure de l’IPP Winsock                                         | Description                                                                                                   |
|---------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [**\_routine NSP**](/windows/desktop/api/Ws2spi/ns-ws2spi-nsp_routine)                           | Contient des informations sur les fonctions implémentées par un fournisseur NSPv1 (namespace Service Provider version 1). |
| [**\_Routine NSPV2**](/windows/desktop/api/Ws2spi/ns-ws2spi-nspv2_routine)                       | Contient des informations sur les fonctions implémentées par un fournisseur NSPv2 (namespace Service Provider version 2). |
| [**WSATHREADID**](/windows/desktop/api/Ws2spi/ns-ws2spi-wsathreadid)                          | Permet à un fournisseur d’identifier un thread sur lequel les appels de procédure asynchrone (APC) peuvent être mis en file d’attente.           |
| [**\_ \_ informations d’audit du fournisseur WSC \_**](/windows/desktop/api/Ws2spi/ns-ws2spi-wsc_provider_audit_info) | Contient des informations d’audit pour une entrée de fournisseur de services en couche (LSP).                                        |
| [**WSPDATA**](/windows/desktop/api/Ws2spi/ns-ws2spi-wspdata)                                  | Contient des informations sur le fournisseur de services.                                                                        |
| [**\_table WSPPROC**](/windows/desktop/api/Ws2spi/ns-ws2spi-wspproc_table)                       | Contient un tableau de pointeurs vers les fonctions du fournisseur de services.                                                   |
| [**WSPUPCALLTABLE**](/windows/desktop/api/Ws2spi/ns-ws2spi-wspupcalltable)                      | Contient un tableau de pointeurs vers les fonctions d’appel du fournisseur de services.                                            |
