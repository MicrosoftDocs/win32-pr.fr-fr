# <a name="windows-security-center-enumerations"></a>Énumérations Windows Security Center

Les énumérations suivantes sont utilisées avec les Security Center Windows.

## <a name="in-this-section"></a>Contenu de cette section



| Rubrique                                                                                | Description                                                                                                      |
|--------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------|
| [**\_État du \_ produit de sécurité de WSC \_**](/windows/desktop/api/Iwscapi/ne-iwscapi-wsc_security_product_state)<br/>       | Définit l’état actuel du produit de sécurité mis à la disposition de Windows Security Center. <br/> |
| [**\_fournisseur de sécurité de WSC \_**](/windows/desktop/api/Wscapi/ne-wscapi-wsc_security_provider)<br/>                  | Définit tous les services qui sont surveillés par Windows Security Center (WSC).<br/>                         |
| [**\_intégrité du \_ fournisseur de sécurité de WSC \_**](/windows/desktop/api/Wscapi/ne-wscapi-wsc_security_provider_health)<br/>   | Définit les États possibles pour tout service surveillé par Windows Security Center (WSC).<br/>               |
| [**\_État de \_ signature de sécurité de WSC \_**](/windows/desktop/api/Iwscapi/ne-iwscapi-wsc_security_signature_status)<br/> | Indique l’état actuel de la version du produit de sécurité sur Windows Security Center. <br/>               |
