# <a name="windows-security-center-enumerations"></a>Sécurité Windows Énumérations de centre

les énumérations suivantes sont utilisées avec Sécurité Windows Center.

## <a name="in-this-section"></a>Contenu de cette section



| Rubrique                                                                                | Description                                                                                                      |
|--------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------|
| [**\_État du \_ produit de sécurité de WSC \_**](/windows/desktop/api/Iwscapi/ne-iwscapi-wsc_security_product_state)<br/>       | définit l’état actuel du produit de sécurité mis à la disposition du centre de Sécurité Windows. <br/> |
| [**\_fournisseur de sécurité de WSC \_**](/windows/desktop/api/Wscapi/ne-wscapi-wsc_security_provider)<br/>                  | définit tous les services qui sont surveillés par Sécurité Windows Center (WSC).<br/>                         |
| [**\_intégrité du \_ fournisseur de sécurité de WSC \_**](/windows/desktop/api/Wscapi/ne-wscapi-wsc_security_provider_health)<br/>   | définit les états possibles pour tout service surveillé par Sécurité Windows Center (WSC).<br/>               |
| [**\_État de \_ signature de sécurité de WSC \_**](/windows/desktop/api/Iwscapi/ne-iwscapi-wsc_security_signature_status)<br/> | signale l’état actuel de la version du produit de sécurité dans le centre de Sécurité Windows. <br/>               |
