# <a name="termsrvappinstallmode-function"></a>TermsrvAppInstallMode fonction)

\[Cette fonction n’est pas prise en charge et ne doit pas être utilisée. Elle peut changer ou disparaître complètement sans préavis.\]

Détermine si le Terminal Server est en mode d’installation.

## <a name="syntax"></a>Syntaxe


```C++
BOOL TermsrvAppInstallMode(void);
```



## <a name="parameters"></a>Paramètres

Cette fonction n’a pas de paramètres.

## <a name="return-value"></a>Valeur retournée

Cette fonction renvoie la **valeur true** si le Terminal Server est en mode d’installation et **false** s’il est en mode exécution.
