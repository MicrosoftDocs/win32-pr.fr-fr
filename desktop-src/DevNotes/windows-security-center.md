# <a name="windows-security-center"></a>Sécurité Windows Gestionnaire

Sécurité Windows Center (WSC) est un outil de création de rapports complet qui aide les utilisateurs à établir et maintenir une couche de sécurité de protection autour de leurs systèmes informatiques. une fois qu’une couche de sécurité est établie, Sécurité Windows Center est invisible lors de l’analyse de l’état d’intégrité de l’ordinateur. Toutefois, si des vulnérabilités existent, le WSC fournit des alertes et des conseils pour aider l’utilisateur à atteindre un état sécurisé qui est exposé à l’utilisateur final via le centre de maintenance.

pour que les solutions de sécurité tierces (antivirus, logiciel anti-programme malveillant ou anti-logiciel espion) soient conformes à Windows et signalent correctement l’état au centre de maintenance, elles doivent s’inscrire auprès d’Security Center et signaler les modifications d’état ultérieures à l’aide des api privées pour communiquer avec le groupe de sécurité. Le catalogue WSC, à son tour, communique ces mises à jour au centre de maintenance, où elles sont finalement affichées à l’utilisateur final.
