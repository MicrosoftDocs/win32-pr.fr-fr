# <a name="patchapi"></a>PatchAPI

Cette section documente plusieurs fonctions qui appartiennent à l’interface PatchAPI. PatchAPI fait partie de l' [API de compression Delta](/previous-versions/bb417345(v=msdn.10)), conçue pour créer par programmation des deltas (similaires aux correctifs ou aux différences compressées) entre les fichiers.
