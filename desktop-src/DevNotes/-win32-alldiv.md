# <a name="_alldiv-routine"></a>\_Routine alldiv

Divise deux entiers **LongLong** . Par exemple, pour diviser deux valeurs Int64, le compilateur peut générer un appel à la \_ routine alldiv.

## <a name="remarks"></a>Remarques

\_la routine alldiv est une routine d’assistance pour le compilateur C. Si le compilateur appelle la \_ routine alldiv dépend entièrement du jeu d’optimisation.
