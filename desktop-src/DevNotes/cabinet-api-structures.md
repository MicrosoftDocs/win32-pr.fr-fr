# <a name="cabinet-api-structures"></a>Structures de l’API cab

Cette section détaille les structures suivantes de l’API CAB :



| Structure                                              | Description                                                                                     |
|--------------------------------------------------------|-------------------------------------------------------------------------------------------------|
| [**CABINETDLLVERSIONINFO**](cabinetdllversioninfo.md) | Contient les informations de version pour Cabinet.dll.<br/>                                    |
| [**CCAB**](/windows/desktop/api/Fci/ns-fci-ccab)                                   | Contient des informations sur le fichier CAB.<br/>                                                        |
| [**ERF**](/windows/win32/api/fdi_fci_types/ns-fdi_fci_types-erf)                                     | Retourne les informations d’erreur à partir de FCI/FDI. L’appelant ne doit pas modifier cette structure.<br/> |
| [**FDICABINETINFO**](/windows/desktop/api/Fdi/ns-fdi-fdicabinetinfo)               | Contient des détails sur un fichier CAB particulier.<br/>                                    |
| [**FDINOTIFICATION**](/windows/desktop/api/Fdi/ns-fdi-fdinotification)             | Fournissez des informations à [**FNFDINOTIFY**](/windows/desktop/api/fdi/nf-fdi-fnfdinotify).<br/>                           |
| [**SESSION**](session.md)                             | Contient des informations sur la session active.<br/>                                      |
