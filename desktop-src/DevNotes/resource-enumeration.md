# <a name="resource-enumeration"></a>Énumération des ressources

Les développeurs peuvent utiliser l’énumération des ressources dans le cadre de l’instrumentation qui fournit des types spécifiques d’informations sur leurs applications.

Microsoft crée l’instrumentation dans le système d’exploitation Windows, mais ces fonctionnalités sont liées étroitement au système d’exploitation dont l’utilisation est limitée aux développeurs d’applications. Microsoft rend une interface de programmation d’applications (API) d’énumération des ressources qui peut être utilisée avec les ressources d’instrumentation suivantes : allocations de tas et gérer les informations de trace.

Les rubriques suivantes décrivent l’utilisation de cette API dans vos applications :

-   [À propos de l’énumération des ressources](about-resource-enumeration.md)
-   [Référence de l’énumération des ressources](resource-enumeration-reference.md)
