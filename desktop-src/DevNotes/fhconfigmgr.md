# <a name="fhconfigmgr-class"></a>FhConfigMgr, classe

Représente la configuration de l’historique des fichiers de l’utilisateur sous lequel l’objet **FhConfigMgr** est créé. Toutes les opérations de configuration sont effectuées en appelant les méthodes de l’interface [**IFhConfigMgr**](/windows/desktop/api/Fhcfg/nn-fhcfg-ifhconfigmgr) .

## <a name="when-to-implement"></a>Quand implémenter

L’API d’historique des fichiers implémente cette classe. Pour créer une instance de cette classe, utilisez la fonction [**CoCreateInstance**](/windows/win32/api/combaseapi/nf-combaseapi-cocreateinstance) .
