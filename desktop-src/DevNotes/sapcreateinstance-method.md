# <a name="sapcreateinstance-method"></a>Méthode SapCreateInstance

Crée un canal de communication avec le pilote de chemin audio sécurisé.

## <a name="syntax"></a>Syntaxe


```C++
MSSAPDLLObject* SapCreateInstance();
```



## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.

## <a name="return-value"></a>Valeur retournée

Pointeur vers un **MSSAPDLLObject**.
