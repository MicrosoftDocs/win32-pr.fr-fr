# <a name="offline-registry-library-functions"></a>Fonctions de la bibliothèque de Registre hors connexion

Les fonctions de la bibliothèque de Registre hors connexion sont les suivantes.



| Fonction                                       | Description                                                                                                         |
|------------------------------------------------|---------------------------------------------------------------------------------------------------------------------|
| [**ORCloseHive**](orclosehive.md)             | Ferme la ruche de Registre hors connexion spécifiée et libère la mémoire allouée pour la ruche.                                 |
| [**ORCloseKey**](orclosekey.md)               | Ferme un handle vers la clé de Registre spécifiée dans une ruche de Registre hors connexion.                                          |
| [**ORCreateHive**](orcreatehive.md)           | Crée une ruche de Registre hors connexion qui contient une clé racine vide unique.                                             |
| [**ORCreateKey**](orcreatekey.md)             | Crée la clé de Registre spécifiée dans une ruche de Registre hors connexion. Si la clé existe déjà, la fonction l’ouvre.   |
| [**ORDeleteKey**](ordeletekey.md)             | Supprime une sous-clé et ses valeurs d’une ruche de Registre hors connexion.                                                      |
| [**ORDeleteValue**](ordeletevalue.md)         | Supprime une valeur de la clé de Registre spécifiée dans une ruche de Registre hors connexion.                                        |
| [**OREnumKey**](orenumkey.md)                 | Énumère les sous-clés de la clé de Registre ouverte spécifiée dans une ruche de Registre hors connexion.                              |
| [**OREnumValue**](orenumvalue.md)             | Énumère les valeurs de la clé de Registre ouverte spécifiée dans une ruche de Registre hors connexion.                              |
| [**ORGetKeySecurity**](orgetkeysecurity.md)   | Récupère une copie du descripteur de sécurité protégeant la clé de Registre ouverte spécifiée dans une ruche de Registre hors connexion. |
| [**ORGetValue**](orgetvalue.md)               | Récupère le type et les données pour la valeur de Registre spécifiée dans une ruche de Registre hors connexion.                           |
| [**ORGetVersion**](orgetversion.md)           | Récupère la version de la bibliothèque de Registre hors connexion.                                                              |
| [**ORGetVirtualFlags**](orgetvirtualflags.md) | Récupère les indicateurs virtuels sur la clé de Registre ouverte spécifiée dans une ruche de Registre hors connexion.                         |
| [**OROpenHive**](oropenhive.md)               | Charge le fichier Hive spécifié en mémoire et valide la ruche.                                                   |
| [**OROpenKey**](oropenkey.md)                 | Ouvre la clé de Registre spécifiée dans une ruche de Registre hors connexion.                                                       |
| [**ORQueryInfoKey**](orqueryinfokey.md)       | Récupère des informations sur la clé de Registre spécifiée dans une ruche de Registre hors connexion.                                 |
| [**ORSaveHive**](orsavehive.md)               | Écrit la ruche de Registre hors connexion spécifiée dans un fichier.                                                               |
| [**ORSetKeySecurity**](orsetkeysecurity.md)   | Définit la sécurité d’une clé de Registre ouverte dans une ruche de Registre hors connexion.                                              |
| [**ORSetValue**](orsetvalue.md)               | Définit les données pour la valeur d’une clé de Registre spécifiée dans une ruche de Registre hors connexion.                                |
| [**ORSetVirtualFlags**](orsetvirtualflags.md) | Définit des indicateurs de virtualisation sur la clé de Registre ouverte spécifiée dans une ruche de Registre hors connexion.                           |
