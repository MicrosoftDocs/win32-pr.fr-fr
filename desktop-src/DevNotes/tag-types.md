# <a name="tag-types"></a>Types de BALISes

Décrit le type de données de la valeur associée à une BALIse.



| Constante/valeur                                                                                                                                                                                                                            | Description                                           |
|:------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:------------------------------------------------------|
| <span id="TAG_TYPE_NULL"></span><span id="tag_type_null"></span><dl> <dt>**Balise \_ TAPEZ \_ null**</dt> <dt>0x1000</dt> </dl>                | Aucune valeur n’est associée à la BALIse.<br/> |
| <span id="TAG_TYPE_BYTE"></span><span id="tag_type_byte"></span><dl> <dt>**Balise \_ \_Octet de type**</dt> <dt>0x2000</dt> </dl>                | Valeur d' **octet** .<br/>                          |
| <span id="TAG_TYPE_WORD"></span><span id="tag_type_word"></span><dl> <dt>**Balise \_ TAPEZ \_ Word**</dt> <dt>0x3000</dt> </dl>                | Valeur de **mot** .<br/>                          |
| <span id="TAG_TYPE_DWORD"></span><span id="tag_type_dword"></span><dl> <dt>**Balise \_ TAPEZ \_ DWORD**</dt> <dt>0x4000</dt> </dl>             | Valeur **DWORD** .<br/>                         |
| <span id="TAG_TYPE_QWORD"></span><span id="tag_type_qword"></span><dl> <dt>**Balise \_ TAPEZ \_ QWord**</dt> <dt>0x5000</dt> </dl>             | Valeur **ULONGLONG** .<br/>                     |
| <span id="TAG_TYPE_STRINGREF"></span><span id="tag_type_stringref"></span><dl> <dt>**Balise \_ TAPEZ \_ STRINGREF**</dt> <dt>0x6000</dt> </dl> | Valeur de chaîne avec jetons.<br/>                  |
| <span id="TAG_TYPE_LIST"></span><span id="tag_type_list"></span><dl> <dt>**Balise \_ 0x7000 \_ liste de types**</dt> <dt></dt> </dl>                | Liste de valeurs de BALIse.<br/>                      |
| <span id="TAG_TYPE_STRING"></span><span id="tag_type_string"></span><dl> <dt>**Balise \_ \_Chaîne de type**</dt> <dt>0x8000</dt> </dl>          | Valeur de chaîne.<br/>                            |
| <span id="TAG_TYPE_BINARY"></span><span id="tag_type_binary"></span><dl> <dt>**Balise \_ TYPE \_ binaire**</dt> <dt>0x9000</dt> </dl>          | Valeur binaire.<br/>                            |
