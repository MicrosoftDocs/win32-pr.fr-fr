# <a name="ifeedbackhubappinfo-interface"></a>Interface IFeedbackHubAppInfo

Cette API n’est pas disponible pour toutes les applications. À moins que votre application ne soit spécialement approvisionnée par Microsoft, les appels à ces API échouent au moment de l’exécution.

## <a name="members"></a>Membres

L’interface **IFeedbackHubAppInfo** hérite de l’interface [**IUnknown**](/windows/win32/api/unknwn/nn-unknwn-iunknown) , mais n’a pas de membres supplémentaires.
