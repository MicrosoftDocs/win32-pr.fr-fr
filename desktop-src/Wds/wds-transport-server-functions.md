# <a name="wds-transport-server-functions"></a>Fonctions du serveur de transport WDS

Fonctions fournies par le serveur de multidiffusion qui interagit avec les fournisseurs de contenu.



| Fonction                                                                         | Description                                              |
|----------------------------------------------------------------------------------|----------------------------------------------------------|
| [**WdsTransportServerAllocateBuffer**](/windows/desktop/api/Wdstpdi/nf-wdstpdi-wdstransportserverallocatebuffer)     | Alloue une mémoire tampon en mémoire.                            |
| [**WdsTransportServerCompleteRead**](/windows/desktop/api/Wdstpdi/nf-wdstpdi-wdstransportservercompleteread)         | Fournit l’état de l’opération de lecture.                       |
| [**WdsTransportServerFreeBuffer**](/windows/desktop/api/Wdstpdi/nf-wdstpdi-wdstransportserverfreebuffer)             | Libère la mémoire utilisée par une mémoire tampon.                        |
| [**WdsTransportServerRegisterCallback**](/windows/desktop/api/Wdstpdi/nf-wdstpdi-wdstransportserverregistercallback) | Inscrit un rappel de fournisseur avec le serveur de multidiffusion. |
| [**WdsTransportServerTrace**](/windows/desktop/api/Wdstpdi/nf-wdstpdi-wdstransportservertrace)                       | Envoie un message de débogage.                               |
| [**WdsTransportServerTraceV**](/windows/desktop/api/Wdstpdi/nf-wdstpdi-wdstransportservertracev)                     | Envoie un message de débogage.                               |
