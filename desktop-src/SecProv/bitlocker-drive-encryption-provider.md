# <a name="bitlocker-drive-encryption-provider"></a>Fournisseur du chiffrement de lecteur BitLocker

L’interface du fournisseur WMI est destinée à la gestion et à la configuration de Chiffrement de lecteur BitLocker (BDE) sur Windows Server 2008 R2, Windows Server 2008 et uniquement des versions spécifiques de Windows 7, Windows Vista entreprise et Windows Vista Édition intégrale. Il est conçu pour être lu par les scripts d’écriture, les éléments d’interface utilisateur ou d’autres outils d’administration pour BDE, ainsi que pour les personnes qui souhaitent en savoir plus sur les fonctionnalités BDE.
