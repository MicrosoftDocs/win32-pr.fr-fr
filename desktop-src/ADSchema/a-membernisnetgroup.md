# <a name="membernisnetgroup-attribute"></a>attribut memberNisNetgroup

Contient la liste des groupes de regroupement qui sont membres de ce groupe.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | memberNisNetgroup                    |
| LDAP-Display-Name | memberNisNetgroup                    |
| Taille              | \-                                   |
| Mettre à jour le privilège  | \-                                   |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.13                     |
| System-ID-GUID    | 0f6a17dc-53e5-4be8-9442-8f3ce2f9012a |
| Syntaxe            | [**String(IA5)**](s-string-ia5.md)  |
