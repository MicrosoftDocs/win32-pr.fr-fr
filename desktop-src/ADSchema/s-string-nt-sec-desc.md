Chaîne d’octets qui contient un descripteur de sécurité Windows NT ou Windows 2000.

|--------------|------------------------------------------------------------------------------------------------|
| Nom         | String(NT-Sec-Desc)                                                                            |
| ID de syntaxe    | 2.5.5.15                                                                                       |
| ID OM        | 66                                                                                             |
| Type MAPI    | BINARY                                                                                         |
| Type ADS     | \_descripteur de sécurité NT ADSTYPE \_ \_                                                              |
| Type de variante | \_distribution vt                                                                                   |
