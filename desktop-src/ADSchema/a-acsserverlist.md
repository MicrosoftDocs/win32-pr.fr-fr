# <a name="acs-server-list-attribute"></a>ACS-attribut Server-List

Contient les noms DNS des serveurs Windows NT autorisés à exécuter les services ACS.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | ACS-Server-List                             |
| LDAP-Display-Name | aCSServerList                               |
| Taille              | \-                                          |
| Mettre à jour le privilège  | Administrateur de domaine                        |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1312                     |
| System-ID-GUID    | 7cbd59a5-3b90-11d2-90cc-00c04fd91ab1        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
