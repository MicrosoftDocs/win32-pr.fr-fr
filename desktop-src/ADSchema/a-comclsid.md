# <a name="com-clsid-attribute"></a>Attribut COM-CLSID

Cet attribut contient le GUID associé à cette classe d’objet.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | CLSID COM                                   |
| LDAP-Display-Name | comclsid                                    |
| Taille              | \-                                          |
| Mettre à jour le privilège  | \-                                          |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.249                      |
| System-ID-GUID    | 281416d9-1968-11d0-a28f-00aa003049e2        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
