# <a name="ms-ds-per-user-trust-tombstones-quota-attribute"></a>MS-DS-par-utilisateur-confiance-Tombstone-attribut de quota

Utilisé pour appliquer un quota par utilisateur pour la suppression d’objets Trusted-Domain lorsque l’autorisation est basée sur la mise en correspondance du SID de l’utilisateur.



| Entrée | Valeur |
|-------------------|-------------------------------------------|
| CN                | MS-DS-par-utilisateur-confiance-désactivation tombstone-quota     |
| LDAP-Display-Name | msDS-PerUserTrustTombstonesQuota          |
| Taille              | \-                                        |
| Mettre à jour le privilège  | Administrateur de domaine                      |
| Fréquence des mises à jour  | À la création de la forêt et rarement après cela. |
| Attribute-Id      | 1.2.840.113556.1.4.1790                   |
| System-ID-GUID    | 8b70a6c6-50f9-4fa3-a71e-1ce03040449b      |
| Syntaxe            | [**Énumération**](s-enumeration.md)      |
