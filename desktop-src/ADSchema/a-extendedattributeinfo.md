# <a name="extended-attribute-info-attribute"></a>Attribut étendu-attribute-info

Propriété à valeurs multiples qui contient des chaînes qui représentent des informations supplémentaires pour chaque attribut.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | Extended-attribute-info                     |
| LDAP-Display-Name | extendedAttributeInfo                       |
| Taille              | \-                                          |
| Mettre à jour le privilège  | \-                                          |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.909                      |
| System-ID-GUID    | 9a7ad947-ca53-11d1-bbd0-0080c76670c0        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
