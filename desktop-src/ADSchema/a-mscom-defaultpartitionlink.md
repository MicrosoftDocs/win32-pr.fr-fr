# <a name="ms-com-defaultpartitionlink-attribute"></a>attribut ms-COM-DefaultPartitionLink

Lien utilisé pour identifier la partition COM+ « par défaut » dans un PartitionSet COM+.



| Entrée | Valeur |
|-------------------|-----------------------------------------|
| CN                | MS-COM-DefaultPartitionLink             |
| LDAP-Display-Name | msCOM-DefaultPartitionLink              |
| Taille              | \-                                      |
| Mettre à jour le privilège  | \-                                      |
| Fréquence des mises à jour  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1427                 |
| System-ID-GUID    | 998b10f7-aa1a-4364-B867-753d197fe670    |
| Syntaxe            | [**Object(DS-DN)**](s-object-ds-dn.md) |
