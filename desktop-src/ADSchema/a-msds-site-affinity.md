# <a name="ms-ds-site-affinity-attribute"></a>ms-DS-site-attribut Affinity

L’attribut **ms-DS-site-Affinity** est utilisé par le gestionnaire de comptes de sécurité pour l’extension de groupe lors de l’évaluation du jeton.



| Entrée | Valeur |
|-------------------|-----------------------------------------------------------------------------------------|
| CN                | ms-DS-site-Affinity                                                                     |
| LDAP-Display-Name | msDS-site-Affinity                                                                      |
| Taille              | Objet BLOB binaire à valeurs multiples, où chaque valeur est sizeof (GUID) + sizeof ( \_ entier long). |
| Mettre à jour le privilège  | \-                                                                                      |
| Fréquence des mises à jour  | Par défaut, une fois tous les trois mois.                                                     |
| Attribute-Id      | 1.2.840.113556.1.4.1443                                                                 |
| System-ID-GUID    | c17c5602-bcb7-46f0-9656-6370ca884b72                                                    |
| Syntaxe            | [**Object(Replica-Link)**](s-object-replica-link.md)                                   |
