# <a name="ms-ds-az-script-timeout-attribute"></a>ms-DS-AZ-script-Timeout (attribut)

Durée d’attente maximale, en millisecondes, d’un script pour terminer l’audit d’une stratégie spécifique.



| Entrée | Valeur |
|-------------------|-----------------------------------------|
| CN                | ms-DS-AZ-script-Timeout                 |
| LDAP-Display-Name | msDS-AzScriptTimeout                    |
| Taille              | \-                                      |
| Mettre à jour le privilège  | Administrateur AzRoles                           |
| Fréquence des mises à jour  | Pendant l’initialisation ou la modification de la stratégie. |
| Attribute-Id      | 1.2.840.113556.1.4.1797                 |
| System-ID-GUID    | 87d0fb41-2c8b-41f6-b972-11fdfd50d6b0    |
| Syntaxe            | [**Enumeration**](s-enumeration.md)    |
