# <a name="ms-ts-endpoint-data-attribute"></a>attribut ms-TS-Endpoint-Data

Cet attribut représente le nom de la machine virtuelle d’un ordinateur dans un déploiement TSV.

| CN                | MS-TS-Endpoint-Data                         |
| LDAP-Display-Name | msTSEndpointData                            |
| Taille              | \-                                          |
| Mettre à jour le privilège  | \-                                          |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.2070                     |
| System-ID-GUID    | 40e1c407-4344-40f3-ab43-3625a34a63a2        |
