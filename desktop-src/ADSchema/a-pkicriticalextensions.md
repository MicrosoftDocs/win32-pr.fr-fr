# <a name="pki-critical-extensions-attribute"></a>Attribut PKI-Critical-extensions

Liste des extensions critiques dans le modèle de certificat.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | PKI-Critical-extensions                     |
| LDAP-Display-Name | pKICriticalExtensions                       |
| Taille              | \-                                          |
| Mettre à jour le privilège  | \-                                          |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1330                     |
| System-ID-GUID    | fc5a9106-3b9d-11d2-90cc-00c04fd91ab1        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
