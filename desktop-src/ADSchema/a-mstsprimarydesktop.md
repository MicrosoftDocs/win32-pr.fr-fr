# <a name="ms-ts-primary-desktop-attribute"></a>attribut ms-TS-Primary-Desktop

Cet attribut représente le lien suivant vers le bureau principal d’un utilisateur.



| Entrée | Valeur |
|-------------------|-----------------------------------------|
| CN                | MS-TS-principal-Desktop                   |
| LDAP-Display-Name | msTSPrimaryDesktop                      |
| Taille              | \-                                      |
| Mettre à jour le privilège  | \-                                      |
| Fréquence des mises à jour  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.2073                 |
| System-ID-GUID    | 29259694-09e4-4237-9f72-9306ebe63ab2    |
| Syntaxe            | [**Object(DS-DN)**](s-object-ds-dn.md) |
