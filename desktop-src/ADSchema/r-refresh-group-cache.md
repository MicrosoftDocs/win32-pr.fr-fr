# <a name="refresh-group-cache-extended-right"></a>Actualiser-groupe-à droite étendu

Il ne s’agit pas d’une ouverture de session GC. Aucune ouverture de session de catalogue global ne s’appuie sur les appartenances aux groupes de mise en cache, et ce droit d’accès de contrôle est utilisé pour accorder aux administrateurs et aux opérateurs des droits d’autorisation pour provoquer une actualisation immédiate du cache, en contactant un GC disponible.



| Entrée | Valeur |
|--------------|--------------------------------------|
| CN           | Actualiser-groupe-cache                  |
| Display-Name | Actualiser le cache de groupe pour les ouvertures de session       |
| Rights-GUID  | 9432c620-033c-4db7-8b58-14ef6d0bf477 |
