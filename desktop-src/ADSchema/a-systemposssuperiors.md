# <a name="system-poss-superiors-attribute"></a>Attribut System-Poss-Superior

Liste des classes qui peuvent contenir cette classe. Cette liste ne peut être modifiée que par le système.



| Entrée | Valeur |
|-------------------|-----------------------------------------------------------------------------|
| CN                | Système-Poss-Superiors                                                       |
| LDAP-Display-Name | systemPossSuperiors                                                         |
| Taille              | \-                                                                          |
| Mettre à jour le privilège  | Administrateur de schéma                                                        |
| Fréquence des mises à jour  | Lorsque la classe est créée ou qu’une nouvelle valeur supérieure possible est ajoutée à la classe. |
| Attribute-Id      | 1.2.840.113556.1.4.195                                                      |
| System-ID-GUID    | bf967a47-0de6-11d0-a285-00aa003049e2                                        |
| Syntaxe            | [**String(Object-Identifier)**](s-string-object-identifier.md)             |
