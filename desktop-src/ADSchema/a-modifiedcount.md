# <a name="modified-count-attribute"></a>Attribut Modified-Count

Numéro de série du journal des modifications de l’ouverture de session.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | Modified-Count                       |
| LDAP-Display-Name | modifiedCount                        |
| Taille              | \-                                   |
| Mettre à jour le privilège  | Cette valeur est définie par le système.     |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.168               |
| System-ID-GUID    | bf9679c5-0de6-11d0-a285-00aa003049e2 |
| Syntaxe            | [**Défini**](s-interval.md)       |
