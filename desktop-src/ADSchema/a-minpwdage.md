# <a name="min-pwd-age-attribute"></a>Attribut min-pwd-Age

Durée minimale, en intervalles de 100 nanosecondes, qu’un mot de passe est valide.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | Min-pwd-Age                          |
| LDAP-Display-Name | minPwdAge                            |
| Taille              | \-                                   |
| Mettre à jour le privilège  | Administrateur de domaine                 |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.78                |
| System-ID-GUID    | bf9679c2-0de6-11d0-a285-00aa003049e2 |
| Syntaxe            | [**Intervalle**](s-interval.md)       |
