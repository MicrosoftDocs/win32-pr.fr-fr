# <a name="package-name-attribute"></a>Attribut Package-Name

Cet attribut contient le nom du déploiement pour une application.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | Package-Name                                |
| LDAP-Display-Name | packageName                                 |
| Taille              | \-                                          |
| Mettre à jour le privilège  | \-                                          |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.326                      |
| System-ID-GUID    | 7d6c0e98-7e20-11d0-afd6-00c04fd930c9        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
