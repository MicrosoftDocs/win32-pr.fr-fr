# <a name="ms-ds-allowed-dns-suffixes-attribute"></a>attribut ms-DS-allowed-suffixes DNS

Liste des suffixes autorisés pour l’attribut dNSHostName dans les objets ordinateur.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | ms-DS-autorisé-DNS-suffixes                  |
| LDAP-Display-Name | msDS-AllowedDNSSuffixes                     |
| Taille              | \-                                          |
| Mettre à jour le privilège  | Administrateur de domaine                        |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1710                     |
| System-ID-GUID    | 8469441b-9ac4-4e45-8205-bd219dbf672d        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
