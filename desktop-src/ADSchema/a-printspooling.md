# <a name="print-spooling-attribute"></a>Attribut Print-Spooling

Chaîne qui représente le type de mise en file d’attente de l’imprimante.



| Entrée | Valeur |
|-------------------|----------------------------------------------------------------------|
| CN                | Print-Spooling                                                       |
| LDAP-Display-Name | printSpooling                                                        |
| Taille              | Valeurs possibles : PrintDirect, PrintWhileSpooling, PrintAfterSpooled. |
| Mettre à jour le privilège  | \-                                                                   |
| Fréquence des mises à jour  | \-                                                                   |
| Attribute-Id      | 1.2.840.113556.1.4.274                                               |
| System-ID-GUID    | ba305f6c-47e3-11d0-a1a6-00c04fd930c9                                 |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md)                          |
