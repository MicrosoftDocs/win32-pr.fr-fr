# <a name="ldap-admin-limits-attribute"></a>Attribut LDAP-admin-Limits

Contient un ensemble de paires attribut-valeur définissant les limites administratives du serveur LDAP.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | LDAP-administrateurs-limites                           |
| LDAP-Display-Name | lDAPAdminLimits                             |
| Taille              | \-                                          |
| Mettre à jour le privilège  | \-                                          |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.843                      |
| System-ID-GUID    | 7359a352-90f7-11d1-aebc-0000f80367c1        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
