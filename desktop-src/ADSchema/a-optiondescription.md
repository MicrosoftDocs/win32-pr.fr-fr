# <a name="option-description-attribute"></a>Attribut Option-Description

Cet attribut contient une description d’une option définie sur le serveur DHCP.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | Option-Description                          |
| LDAP-Display-Name | protocoledescription                           |
| Taille              | \-                                          |
| Mettre à jour le privilège  | \-                                          |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.712                      |
| System-ID-GUID    | 963d274d-48be-11d1-a9c3-0000f80367c1        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
