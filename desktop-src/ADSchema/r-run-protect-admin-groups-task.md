# <a name="run-protect-admin-groups-task-extended-right"></a>Exécuter-protéger-admin-Groups-tâche étendue à droite

Contrôler les droits d’accès autorisation des opérations de modification rootDSE sur un contrôleur de domaine principal FSMO pour déclencher l’opération de protection AdminSDHolder.



| Entrée | Valeur |
|--------------|--------------------------------------|
| CN           | Exécuter-protéger-admin-groupes-tâche        |
| Display-Name | Exécuter la tâche protéger les groupes d’administration        |
| Rights-GUID  | 7726b9d5-a4b4-4288-a6b2-dce952e80a7f |
