# <a name="creation-time-attribute"></a>Attribut Creation-Time

Date et heure de création de l’objet.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | Creation-Time                        |
| LDAP-Display-Name | creationTime                         |
| Taille              | 8 octets                              |
| Mettre à jour le privilège  | Cette valeur est définie par le système.     |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.26                |
| System-ID-GUID    | bf967946-0de6-11d0-a285-00aa003049e2 |
| Syntaxe            | [**Défini**](s-interval.md)       |
