# <a name="supported-application-context-attribute"></a>Pris en charge-attribut de contexte d’application

Spécifie les identificateurs d’objets des contextes d’application pris en charge par une application OSI.



| Entrée | Valeur |
|-------------------|-------------------------------------------------------|
| CN                | Pris en charge-application-Context                         |
| LDAP-Display-Name | supportedApplicationContext                           |
| Taille              | \-                                                    |
| Mettre à jour le privilège  | \-                                                    |
| Fréquence des mises à jour  | \-                                                    |
| Attribute-Id      | 2.5.4.30                                              |
| System-ID-GUID    | 1677588f-47f3-11d1-a9c3-0000f80367c1                  |
| Syntaxe            | [**Object(Replica-Link)**](s-object-replica-link.md) |
