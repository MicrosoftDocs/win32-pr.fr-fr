# <a name="acs-cache-timeout-attribute"></a>Attribut ACS-cache-Timeout

Durée avant l’expiration des objets ACS mis en cache par le service ACS.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | ACS-cache-délai d’expiration                    |
| LDAP-Display-Name | aCSCacheTimeout                      |
| Taille              | 4 octets                              |
| Mettre à jour le privilège  | \-                                   |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.779               |
| System-ID-GUID    | 1cb355a1-56d0-11d1-a9c6-0000f80367c1 |
| Syntaxe            | [**Enumeration**](s-enumeration.md) |
