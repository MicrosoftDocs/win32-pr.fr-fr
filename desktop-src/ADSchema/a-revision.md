# <a name="revision-attribute"></a>Attribut de révision

Niveau de révision pour un descripteur de sécurité ou autre modification. Utilisé uniquement dans les objets Sam-Server et DS-UI-Settings.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | Révision                             |
| LDAP-Display-Name | revision                             |
| Taille              | 4 octets                              |
| Mettre à jour le privilège  | Cette valeur est définie par le système.     |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.145               |
| System-ID-GUID    | bf967a21-0de6-11d0-a285-00aa003049e2 |
| Syntaxe            | [**Enumeration**](s-enumeration.md) |
