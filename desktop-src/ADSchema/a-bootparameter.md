# <a name="bootparameter-attribute"></a>attribut bootParameter

Fournit les données nécessaires pour démarrer un client sans disque.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | bootParameter                        |
| LDAP-Display-Name | bootParameter                        |
| Taille              | \-                                   |
| Mettre à jour le privilège  | \-                                   |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.23                     |
| System-ID-GUID    | d72a0750-8c7c-416e-8714-e65f11e908be |
| Syntaxe            | [**String(IA5)**](s-string-ia5.md)  |
