# <a name="ms-ts-endpoint-plugin-attribute"></a>MS-TS-Endpoint-attribut de plug-in

Cet attribut représente le nom du plug-in qui gère l’orchestration.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | MS-TS-Endpoint-plug-in                       |
| LDAP-Display-Name | msTSEndpointPlugin                          |
| Taille              | \-                                          |
| Mettre à jour le privilège  | \-                                          |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.2072                     |
| System-ID-GUID    | 3c08b569-801f-4158-b17b-e363d6ae696a        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
