# <a name="www-home-page-attribute"></a>Attribut WWW-page d’hébergement

Page Web qui est la page d’accueil principale d’un site Web.



| Entrée | Valeur |
|-------------------|-----------------------------------------------------------------------------|
| CN                | WWW-page d’hébergement                                                               |
| LDAP-Display-Name | wWWHomePage                                                                 |
| Taille              | \-                                                                          |
| Mettre à jour le privilège  | Administrateur de domaine ou propriétaire du compte.                                      |
| Fréquence des mises à jour  | Lorsque l’enregistrement de l’utilisateur est créé et chaque fois que la page Web doit être modifiée. |
| Attribute-Id      | 1.2.840.113556.1.2.464                                                      |
| System-ID-GUID    | bf967a7a-0de6-11d0-a285-00aa003049e2                                        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md)                                 |
