# <a name="operating-system-version-attribute"></a>Attribut de version du système d’exploitation

Chaîne de version du système d’exploitation, par exemple, 4,0.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | Version du système d’exploitation                    |
| LDAP-Display-Name | operatingSystemVersion                      |
| Taille              | \-                                          |
| Mettre à jour le privilège  | \-                                          |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.364                      |
| System-ID-GUID    | 3e978926-8c01-11d0-afda-00c04fd930c9        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
