# <a name="netboot-initialization-attribute"></a>Attribut Netboot-Initialization

Chemin de démarrage par défaut pour le démarrage sans disque.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | Netboot-Initialization                      |
| LDAP-Display-Name | netbootInitialization                       |
| Taille              | \-                                          |
| Mettre à jour le privilège  | Cette valeur est définie par le système.            |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.358                      |
| System-ID-GUID    | 3e978920-8c01-11d0-afda-00c04fd930c9        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
