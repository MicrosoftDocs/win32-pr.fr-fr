# <a name="ms-ds-nc-repl-outbound-neighbors-attribute"></a>attribut ms-DS-NC-REPL-sortant-voisins

Partenaires de réplication pour cette partition. Ce serveur envoie les données de réplication à ces autres serveurs, qui jouent le rôle de destinations. Ce serveur notifie ces autres serveurs quand de nouvelles données sont disponibles.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | ms-DS-CN-REPL-sortant-voisins            |
| LDAP-Display-Name | msDS-NCReplOutboundNeighbors                |
| Taille              | \-                                          |
| Mettre à jour le privilège  | Cette valeur est définie par le système.            |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1706                     |
| System-ID-GUID    | 855f2ef5-a1c5-4cc4-ba6d-32522848b61f        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
