|-------------------|--------------------------------------|
| CN                | Masque DHCP                            |
| LDAP-Display-Name | dhcpMask                             |
| Taille              | \-                                   |
| Mettre à jour le privilège  | \-                                   |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.706               |
