# <a name="pwd-history-length-attribute"></a>Pwd-History-attribut-length

Nombre d’anciens mots de passe à enregistrer.



| Entrée | Valeur |
|-------------------|------------------------------------------|
| CN                | Pwd-historique-longueur                       |
| LDAP-Display-Name | pwdHistoryLength                         |
| Taille              | 4 octets                                  |
| Mettre à jour le privilège  | Administrateur de domaine                     |
| Fréquence des mises à jour  | Lorsque la stratégie de compte doit être modifiée. |
| Attribute-Id      | 1.2.840.113556.1.4.95                    |
| System-ID-GUID    | bf967a09-0de6-11d0-a285-00aa003049e2     |
| Syntaxe            | [**Énumération**](s-enumeration.md)     |
