# <a name="rid-allocation-pool-attribute"></a>RID-allocation-attribut de pool

Un pool qui a été préalablement récupéré pour être utilisé par le gestionnaire RID lorsque le pool d’allocation RID précédent a été utilisé.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | RID-allocation-pool                  |
| LDAP-Display-Name | rIDAllocationPool                    |
| Taille              | 8 octets                              |
| Mettre à jour le privilège  | Cette valeur est définie par le système.     |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.371               |
| System-ID-GUID    | 66171889-8f3c-11d0-afda-00c04fd930c9 |
| Syntaxe            | [**Défini**](s-interval.md)       |
