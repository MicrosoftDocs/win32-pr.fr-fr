# <a name="script-path-attribute"></a>Attribut Script-Path

Cet attribut spécifie le chemin d’accès pour le script d’ouverture de session de l’utilisateur. La chaîne peut être null.



| Entrée | Valeur |
|-------------------|------------------------------------------------------------------------|
| CN                | Script-Path                                                            |
| LDAP-Display-Name | scriptPath                                                             |
| Taille              | \-                                                                     |
| Mettre à jour le privilège  | Administrateur de domaine ou propriétaire du compte.                                 |
| Fréquence des mises à jour  | Lorsque l’enregistrement de l’utilisateur est créé et chaque fois que le chemin d’accès doit être modifié. |
| Attribute-Id      | 1.2.840.113556.1.4.62                                                  |
| System-ID-GUID    | bf9679a8-0de6-11d0-a285-00aa003049e2                                   |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md)                            |
