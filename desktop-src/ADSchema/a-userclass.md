# <a name="userclass-attribute"></a>userClass (attribut)

Le type d’attribut userClass spécifie une catégorie d’utilisateur d’ordinateur.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | userClass                                   |
| LDAP-Display-Name | userClass                                   |
| Taille              | \-                                          |
| Mettre à jour le privilège  | \-                                          |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 0.9.2342.19200300.100.1.8                   |
| System-ID-GUID    | 11732a8a-e14d-4cc5-b92f-d93f51c6d8e4        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
