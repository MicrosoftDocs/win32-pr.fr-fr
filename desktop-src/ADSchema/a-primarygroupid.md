# <a name="primary-group-id-attribute"></a>Attribut Primary-Group-ID

Contient l’identificateur relatif (RID) pour le groupe principal de l’utilisateur. Par défaut, il s’agit du RID pour le groupe utilisateurs du domaine.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | ID de groupe principal                     |
| LDAP-Display-Name | primaryGroupID                       |
| Taille              | 4 octets                              |
| Mettre à jour le privilège  | Cette valeur est définie par le système.     |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.98                |
| System-ID-GUID    | bf967a00-0de6-11d0-a285-00aa003049e2 |
| Syntaxe            | [**Enumeration**](s-enumeration.md) |
