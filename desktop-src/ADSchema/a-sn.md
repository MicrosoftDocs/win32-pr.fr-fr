# <a name="surname-attribute"></a>Nom de l’attribut

Cet attribut contient la famille ou le nom d’un utilisateur.



| Entrée | Valeur |
|-------------------|----------------------------------------------------------------------------------|
| CN                | Surname                                                                          |
| LDAP-Display-Name | sn                                                                               |
| Taille              | \-                                                                               |
| Mettre à jour le privilège  | Tout le monde peut mettre à jour cet objet en fonction de la sécurité de l’objet en cours de création. |
| Fréquence des mises à jour  | \-                                                                               |
| Attribute-Id      | 2.5.4.4                                                                          |
| System-ID-GUID    | bf967a41-0de6-11d0-a285-00aa003049e2                                             |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md)                                      |
