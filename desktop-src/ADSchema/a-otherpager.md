# <a name="phone-pager-other-attribute"></a>Téléphone-téléavertisseur-autre attribut

Liste de numéros de radiomessagerie secondaires.



| Entrée | Valeur |
|-------------------|----------------------------------------------------------------------------------|
| CN                | Téléphone-téléavertisseur-autre                                                                |
| LDAP-Display-Name | otherPager                                                                       |
| Taille              | \-                                                                               |
| Mettre à jour le privilège  | Administrateur de domaine ou propriétaire du compte.                                           |
| Fréquence des mises à jour  | Lorsque l’enregistrement de l’utilisateur est créé et chaque fois que le numéro de téléphone doit être modifié. |
| Attribute-Id      | 1.2.840.113556.1.2.118                                                           |
| System-ID-GUID    | f0f8ffa4-1191-11d0-a060-00aa006c33ed                                             |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md)                                      |
