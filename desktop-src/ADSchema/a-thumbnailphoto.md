# <a name="picture-attribute"></a>Attribut image

Image de l’utilisateur. Un format économe en espace comme JPEG ou GIF est recommandé.

|-------------------|-------------------------------------------------------|
| CN                | Image                                               |
| LDAP-Display-Name | thumbnailPhoto                                        |
| Taille              | \-                                                    |
| Mettre à jour le privilège  | \-                                                    |
| Fréquence des mises à jour  | \-                                                    |
| Attribute-Id      | 2.16.840.1.113730.3.1.35                              |
| System-ID-GUID    | 8d3bca50-1d7e-11d0-a081-00aa006c33ed                  |
| Syntaxe            | [**Object(Replica-Link)**](s-object-replica-link.md) |
