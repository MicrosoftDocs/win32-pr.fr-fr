# <a name="trust-direction-attribute"></a>Attribut Trust-Direction

Direction d’une approbation.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | Trust-Direction                      |
| LDAP-Display-Name | trustDirection                       |
| Taille              | 4 octets                              |
| Mettre à jour le privilège  | Cette valeur est définie par le système.     |
| Fréquence des mises à jour  | Lors de la création d’une approbation.         |
| Attribute-Id      | 1.2.840.113556.1.4.132               |
| System-ID-GUID    | bf967a5c-0de6-11d0-a285-00aa003049e2 |
| Syntaxe            | [**Énumération**](s-enumeration.md) |
