# <a name="implemented-categories-attribute"></a>Attribut Implemented-Categories

Cet attribut contient une liste d’ID de catégorie de composant que cet objet implémente.



| Entrée | Valeur |
|-------------------|-------------------------------------------------------|
| CN                | Implemented-Categories                                |
| LDAP-Display-Name | implementedCategories                                 |
| Taille              | \-                                                    |
| Mettre à jour le privilège  | \-                                                    |
| Fréquence des mises à jour  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.320                                |
| System-ID-GUID    | 7d6c0e92-7e20-11d0-afd6-00c04fd930c9                  |
| Syntaxe            | [**Object(Replica-Link)**](s-object-replica-link.md) |
