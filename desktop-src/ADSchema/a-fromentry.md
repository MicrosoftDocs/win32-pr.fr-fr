# <a name="from-entry-attribute"></a>Attribut From-Entry

Il s’agit d’un attribut construit qui a la **valeur true** si l’objet est accessible en écriture et **false** s’il est en lecture seule, par exemple, une instance de réplica gc.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | From-Entry                           |
| LDAP-Display-Name | fromEntry                            |
| Taille              | \-                                   |
| Mettre à jour le privilège  | \-                                   |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.910               |
| System-ID-GUID    | 9a7ad949-ca53-11d1-bbd0-0080c76670c0 |
| Syntaxe            | [**Boolean**](s-boolean.md)         |
