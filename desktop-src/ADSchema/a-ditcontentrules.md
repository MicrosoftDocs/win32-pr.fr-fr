# <a name="dit-content-rules-attribute"></a>DIT-content-Rules (attribut)

Cela spécifie le contenu autorisé des entrées d’une classe d’objets structurels particulière à l’aide de l’identification d’un ensemble facultatif de classes d’objets auxiliaires, ainsi que des attributs obligatoires, facultatifs et interdits. Les attributs collectifs sont inclus dans DIT-content-Rules, comme défini dans la norme RFC 2251.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | DIT-content-Rules                           |
| LDAP-Display-Name | dITContentRules                             |
| Taille              | \-                                          |
| Mettre à jour le privilège  | Cette valeur est définie par le système.            |
| Fréquence des mises à jour  | Chaque fois qu’une nouvelle classe est créée.            |
| Attribute-Id      | 2.5.21.2                                    |
| System-ID-GUID    | 9a7ad946-ca53-11d1-bbd0-0080c76670c0        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
