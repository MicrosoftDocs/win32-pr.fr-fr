# <a name="localization-display-id-attribute"></a>Attribut localization-Display-ID

Permet d’indexer dans le fichier Extrts.mc pour obtenir le displayName localisé pour les objets à des fins d’interface utilisateur.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | Localisation-Display-ID              |
| LDAP-Display-Name | localizationDisplayId                |
| Taille              | 4 octets                              |
| Mettre à jour le privilège  | Administrateur de schéma                 |
| Fréquence des mises à jour  | Un ID ne doit jamais être modifié.       |
| Attribute-Id      | 1.2.840.113556.1.4.1353              |
| System-ID-GUID    | a746f0d1-78d0-11d2-9916-0000f87a57d4 |
| Syntaxe            | [**Énumération**](s-enumeration.md) |
