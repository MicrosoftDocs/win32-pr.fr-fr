# <a name="domain-password-property-set"></a>Jeu de propriétés Domain-Password

Jeu de propriétés qui contient tous les attributs associés à l’âge du verrouillage et du mot de passe sur le compte d’utilisateur.



| Entrée | Valeur |
|--------------|--------------------------------------|
| CN           | Domain-Password                      |
| Display-Name | Stratégies de verrouillage de mot de passe de domaine &   |
| Rights-GUID  | c7407360-20bf-11d0-a768-00aa006e0529 |
