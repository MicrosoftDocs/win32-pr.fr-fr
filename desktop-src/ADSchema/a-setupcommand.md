# <a name="setup-command-attribute"></a>Attribut Setup-Command

Cet attribut indique si une commande d’installation est requise pour configurer cette application.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | Setup-Command                               |
| LDAP-Display-Name | setupCommand                                |
| Taille              | \-                                          |
| Mettre à jour le privilège  | \-                                          |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.325                      |
| System-ID-GUID    | 7d6c0e97-7e20-11d0-afd6-00c04fd930c9        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
