# <a name="curr-machine-id-attribute"></a>Attribut-machine-ID

ID de l’ordinateur où se trouve un objet lien-Track-vol-entrée.



| Entrée | Valeur |
|-------------------|-------------------------------------------------------|
| CN                | ID d’ordinateur                                       |
| LDAP-Display-Name | currMachineId                                         |
| Taille              | \-                                                    |
| Mettre à jour le privilège  | Cette valeur est définie par le système.                      |
| Fréquence des mises à jour  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.337                                |
| System-ID-GUID    | 1f0075fe-7e40-11d0-afd6-00c04fd930c9                  |
| Syntaxe            | [**Object(Replica-Link)**](s-object-replica-link.md) |
