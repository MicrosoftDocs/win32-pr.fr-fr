# <a name="additional-trusted-service-names-attribute"></a>Attribut-Trust-Name-Names supplémentaire

Liste des services du domaine qui peuvent être approuvés. Non utilisé par Active Directory.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | Noms de service de confiance supplémentaires            |
| LDAP-Display-Name | additionalTrustedServiceNames               |
| Taille              | \-                                          |
| Mettre à jour le privilège  | \-                                          |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.889                      |
| System-ID-GUID    | 032160be-9824-11d1-aec0-0000f80367c1        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
