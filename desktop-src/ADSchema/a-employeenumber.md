# <a name="employee-number-attribute"></a>Attribut Employee-Number

Numéro affecté à un employé autre que l’ID.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | Employee-Number                             |
| LDAP-Display-Name | employeeNumber                              |
| Taille              | \-                                          |
| Mettre à jour le privilège  | Administrateur de domaine                        |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.2.610                      |
| System-ID-GUID    | a8df73ef-c5ea-11d1-bbcb-0080c76670c0        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
