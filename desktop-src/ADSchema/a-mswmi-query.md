# <a name="ms-wmi-query-attribute"></a>MS-WMI-attribut de requête

Une seule requête WQL.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | MS-WMI-requête                                |
| LDAP-Display-Name | msWMI-requête                                 |
| Taille              | Moins de 250 caractères.                   |
| Mettre à jour le privilège  | Administrateur stratégie de groupe                  |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1642                     |
| System-ID-GUID    | 65fff93e-35e3-45a3-85ae-876c6718297f        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
