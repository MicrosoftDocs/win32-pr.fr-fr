# <a name="ms-sql-keywords-attribute"></a>Attribut MS-SQL-Keywords

Non utilisé. La valeur par défaut est la clé 1, la clé 2, la clé 3, la clé 4 et la clé 5.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | MS-SQL-Mots clés                             |
| LDAP-Display-Name | mS-SQL-Mots clés                             |
| Taille              | \-                                          |
| Mettre à jour le privilège  | \-                                          |
| Fréquence des mises à jour  | Dans le programme d’installation du système.                            |
| Attribute-Id      | 1.2.840.113556.1.4.1401                     |
| System-ID-GUID    | 01e9a98a-ccef-11d2-9993-0000f87a57d4        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
