# <a name="locale-id-attribute"></a>Locale-ID (attribut)

Cet attribut contient une liste d’ID de paramètres régionaux pris en charge par cette application. Un ID de paramètres régionaux représente un emplacement géographique, tel qu’un pays/une région, une ville, un comté, etc.



| Entrée | Valeur |
|-------------------|----------------------------------------------------------------------------|
| CN                | Paramètres régionaux                                                                  |
| LDAP-Display-Name | localeID                                                                   |
| Taille              | 4 octets                                                                    |
| Mettre à jour le privilège  | Administrateur de domaine ou propriétaire du compte.                                     |
| Fréquence des mises à jour  | Lorsque l’enregistrement de l’utilisateur est créé et chaque fois que la localité doit être modifiée. |
| Attribute-Id      | 1.2.840.113556.1.4.58                                                      |
| System-ID-GUID    | bf9679a1-0de6-11d0-a285-00aa003049e2                                       |
| Syntaxe            | [**Enumeration**](s-enumeration.md)                                       |
