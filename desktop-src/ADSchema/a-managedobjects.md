# <a name="managed-objects-attribute"></a>Attribut Managed-Objects

Contient la liste des objets gérés par l’utilisateur. Les objets répertoriés sont ceux dont la propriété managedBy de la propriété est définie sur cet utilisateur. Chaque élément de la liste est une référence liée à l’objet managé.



| Entrée | Valeur |
|-------------------|------------------------------------------------------------------------------------|
| CN                | Managed-Objects                                                                    |
| LDAP-Display-Name | managedObjects                                                                     |
| Taille              | \-                                                                                 |
| Mettre à jour le privilège  | Cette valeur est définie par le système.                                                   |
| Fréquence des mises à jour  | Lorsque l’enregistrement des utilisateurs est créé et chaque fois que les objets gérés doivent être modifiés. |
| Attribute-Id      | 1.2.840.113556.1.4.654                                                             |
| System-ID-GUID    | 0296c124-40da-11d1-a9c0-0000f80367c1                                               |
| Syntaxe            | [**Object(DS-DN)**](s-object-ds-dn.md)                                            |
