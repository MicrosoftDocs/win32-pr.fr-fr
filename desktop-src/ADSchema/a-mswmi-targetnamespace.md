# <a name="ms-wmi-targetnamespace-attribute"></a>attribut ms-WMI-TargetNameSpace

Espace de noms dans lequel l’objet doit être créé.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | MS-WMI-TargetNameSpace                      |
| LDAP-Display-Name | msWMI-TargetNameSpace                       |
| Taille              | Moins de vingt caractères.                |
| Mettre à jour le privilège  | Administrateur stratégie de groupe                  |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1646                     |
| System-ID-GUID    | 1c4ab61f-3420-44e5-849d-8b5dbf60feb7        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
