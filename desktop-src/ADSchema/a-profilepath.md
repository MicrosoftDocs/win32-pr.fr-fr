# <a name="profile-path-attribute"></a>Attribut Profile-Path

Spécifie un chemin d’accès au profil de l’utilisateur. Cette valeur peut être une chaîne NULL, un chemin d’accès absolu local ou un chemin d’accès UNC.



| Entrée | Valeur |
|-------------------|--------------------------------------------------------------------------|
| CN                | Profile-Path                                                             |
| LDAP-Display-Name | profilePath                                                              |
| Taille              | \-                                                                       |
| Mettre à jour le privilège  | Administrateur de domaine ou propriétaire du compte.                                   |
| Fréquence des mises à jour  | Lorsque l’enregistrement de l’utilisateur est créé et chaque fois que le chemin d’accès doit être modifié. |
| Attribute-Id      | 1.2.840.113556.1.4.139                                                   |
| System-ID-GUID    | bf967a05-0de6-11d0-a285-00aa003049e2                                     |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md)                              |
