# <a name="short-server-name-attribute"></a>Attribut de nom de serveur de type short

Nom du serveur compatible avec les versions antérieures à Windows 2000 pour les serveurs d’impression.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | Short-nom-serveur                           |
| LDAP-Display-Name | shortServerName                             |
| Taille              | \-                                          |
| Mettre à jour le privilège  | Administrateur de domaine                        |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1209                     |
| System-ID-GUID    | 45b01501-c419-11d1-bbc9-0080c76670c0        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
