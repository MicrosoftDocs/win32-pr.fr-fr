liste de membres non sécurisés pour une liste de distribution Exchange.

| LDAP-Display-Name | nonSecurityMemberBL                     |
| Taille              | \-                                      |
| Mettre à jour le privilège  | Cette valeur est définie par le système.        |
| Fréquence des mises à jour  | \-                                      |

| System-ID-GUID    | 52458019-ca6a-11D0-AFFF-0000f80367c1    |
