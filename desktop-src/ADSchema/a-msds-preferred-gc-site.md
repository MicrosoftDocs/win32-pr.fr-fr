# <a name="ms-ds-preferred-gc-site-attribute"></a>ms-DS-Preferred-attribut-GC-site

L’attribut **ms-DS-Preferred-gc-site** est utilisé par le gestionnaire de comptes de sécurité pour l’extension de groupe lors de l’évaluation du jeton.



| Entrée | Valeur |
|-------------------|-----------------------------------------|
| CN                | ms-DS-préféré-GC-site                 |
| LDAP-Display-Name | msDS-Preferred-GC-site                  |
| Taille              | \-                                      |
| Mettre à jour le privilège  | Administrateur de domaine                    |
| Fréquence des mises à jour  | À la discrétion de l’administrateur.      |
| Attribute-Id      | 1.2.840.113556.1.4.1444                 |
| System-ID-GUID    | d921b50a-0ab2-42cd-87f6-09cf83a91854    |
| Syntaxe            | [**Object(DS-DN)**](s-object-ds-dn.md) |
