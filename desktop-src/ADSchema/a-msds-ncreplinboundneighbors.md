# <a name="ms-ds-nc-repl-inbound-neighbors-attribute"></a>attribut ms-DS-NC-REPL-enbound-Neighbors

Partenaires de réplication pour cette partition. Ce serveur obtient les données de réplication à partir de ces autres serveurs, qui jouent le rôle de sources.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | ms-DS-CN-REPL-trafic entrant-voisins             |
| LDAP-Display-Name | msDS-NCReplInboundNeighbors                 |
| Taille              | \-                                          |
| Mettre à jour le privilège  | Cette valeur est définie par le système.            |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1705                     |
| System-ID-GUID    | 9edba85a-3e9e-431b-9b1a-a5b6e9eda796        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
