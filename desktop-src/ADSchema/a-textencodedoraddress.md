# <a name="text-encoded-or-address-attribute"></a>Attribut d’encodage texte ou d’adresse

Cet attribut est utilisé pour prendre en charge les adresses X. 400 dans un format texte.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | Texte-ou-adresse                     |
| LDAP-Display-Name | textEncodedORAddress                        |
| Taille              | \-                                          |
| Mettre à jour le privilège  | \-                                          |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 0.9.2342.19200300.100.1.2                   |
| System-ID-GUID    | a8df7489-c5ea-11d1-bbcb-0080c76670c0        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
