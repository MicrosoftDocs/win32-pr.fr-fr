# <a name="ms-pki-oid-user-notice-attribute"></a>attribut ms-PKI-OID-User-Notice

Avis de l’utilisateur pour l’OID de stratégie de l’émetteur d’entreprise.



| Entrée | Valeur |
|-------------------|--------------------------------------------------------------------------------------------|
| CN                | MS-PKI-OID-utilisateur-avis                                                                     |
| LDAP-Display-Name | Attribut mspki-OID-User-Notice                                                                      |
| Taille              | \-                                                                                         |
| Mettre à jour le privilège  | Administrateur d’entreprise                                                                   |
| Fréquence des mises à jour  | À chaque fois qu’un nouveau modèle est créé ou que les attributs d’un modèle existant sont modifiés. |
| Attribute-Id      | 1.2.840.113556.1.4.1673                                                                    |
| System-ID-GUID    | 04c4da7a-e114-4e69-88de-e293f2d3b395                                                       |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md)                                                |
