# <a name="ms-wmi-targetpath-attribute"></a>MS-WMI-TargetPath (attribut)

Liste de paires clé/valeur permettant d’identifier de façon unique un objet WMI.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | MS-WMI-TargetPath                           |
| LDAP-Display-Name | msWMI-TargetPath                            |
| Taille              | Moins de 100 caractères.                   |
| Mettre à jour le privilège  | Administrateur stratégie de groupe                  |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1648                     |
| System-ID-GUID    | 5006a79a-6bfe-4561-9f52-13cf4dd3e560        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
