# <a name="msmq-peek-extended-right"></a>MSMQ-lire le droit étendu

Autorise la lecture des messages dans la file d’attente.



| Entrée | Valeur |
|--------------|--------------------------------------|
| CN           | MSMQ-Peek                            |
| Display-Name | Lire furtivement un message                         |
| Rights-GUID  | 06bd3201-df3e-11d1-9c86-006008764d0e |
