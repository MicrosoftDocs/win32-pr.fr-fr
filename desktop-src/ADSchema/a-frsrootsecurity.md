# <a name="frs-root-security-attribute"></a>Attribut de sécurité de la racine FRS

Descripteur de sécurité de la racine du jeu de réplicas pour la réplication de fichiers.



| Entrée | Valeur |
|-------------------|-----------------------------------------------------|
| CN                | Sécurité de la racine FRS                                   |
| LDAP-Display-Name | fRSRootSecurity                                     |
| Taille              | \-                                                  |
| Mettre à jour le privilège  | \-                                                  |
| Fréquence des mises à jour  | \-                                                  |
| Attribute-Id      | 1.2.840.113556.1.4.535                              |
| System-ID-GUID    | 5245801f-ca6a-11d0-afff-0000f80367c1                |
| Syntaxe            | [**String(NT-Sec-Desc)**](s-string-nt-sec-desc.md) |
