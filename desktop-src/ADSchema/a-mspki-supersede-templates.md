# <a name="ms-pki-supersede-templates-attribute"></a>MS-PKI-remplace-attribut templates

Spécifie les noms des modèles de certificat qui sont remplacés par le modèle actuel.



| Entrée | Valeur |
|-------------------|---------------------------------------------------------------------------------------------------|
| CN                | MS-PKI-remplace-templates                                                                        |
| LDAP-Display-Name | Attribut mspki-remplace-templates                                                                         |
| Taille              | 64 octets                                                                                          |
| Mettre à jour le privilège  | Administrateur de domaine                                                                              |
| Fréquence des mises à jour  | Lorsque l’objet modèle de certificat (MS-PKI-Certificate-template) est modifié, créé ou cloné. |
| Attribute-Id      | 1.2.840.113556.1.4.1437                                                                           |
| System-ID-GUID    | 9de8ae7d-7a5b-421d-b5e4-061f79dfd5d7                                                              |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md)                                                       |
