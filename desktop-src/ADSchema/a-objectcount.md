# <a name="object-count-attribute"></a>Attribut Object-Count

| CN                | Object-Count                         |
| LDAP-Display-Name | objectCount                          |
| Taille              | 4 octets                              |
| Mettre à jour le privilège  | \-                                   |

| System-ID-GUID    | 34aaa216-b699-11d0-afee-0000f80367c1 |
