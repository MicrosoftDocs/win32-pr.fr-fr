# <a name="ms-ds-approx-immed-subordinates-attribute"></a>attribut ms-DS-approx-immed-subordonnés

La valeur retournée par cet attribut est basée sur les tailles d’index. Cette valeur peut être désactivée par +/10% sur les conteneurs volumineux, et l’erreur est théoriquement illimitée, mais l’utilisation de cet attribut permet à l’interface utilisateur d’afficher le contenu d’un conteneur.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | ms-DS-approx-immed-subordonnés      |
| LDAP-Display-Name | msDS-approx-immed-subordonnés       |
| Taille              | \-                                   |
| Mettre à jour le privilège  | Cette valeur est définie par le système.     |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1669              |
| System-ID-GUID    | e185d243-f6ce-4adb-b496-b0c005d7823c |
| Syntaxe            | [**Énumération**](s-enumeration.md) |
