# <a name="ms-ds-phonetic-display-name-attribute"></a>attribut ms-DS-Phonetic-Display-Name

Nom complet phonétique d’un objet. En l’absence d’un nom d’affichage phonétique, le nom complet existant est utilisé.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | ms-DS-phonétique-Display-Name                 |
| LDAP-Display-Name | msDS-PhoneticDisplayName                    |
| Taille              | \-                                          |
| Mettre à jour le privilège  | \-                                          |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1946                     |
| System-ID-GUID    | e21a94e4-2d66-4ce5-b30d-0ef87a776ff0        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
