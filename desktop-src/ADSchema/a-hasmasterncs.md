# <a name="has-master-ncs-attribute"></a>A-attribut-Master-NC

Nom unique des contextes d’attribution de noms pour le contrôleur de périphérique. Lien suivant pour l’attribut Mastered-By.



| Entrée | Valeur |
|-------------------|-----------------------------------------|
| CN                | A-maître-NC                          |
| LDAP-Display-Name | hasMasterNCs                            |
| Taille              | \-                                      |
| Mettre à jour le privilège  | \-                                      |
| Fréquence des mises à jour  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.2.14                   |
| System-ID-GUID    | bf967982-0de6-11d0-a285-00aa003049e2    |
| Syntaxe            | [**Object(DS-DN)**](s-object-ds-dn.md) |
