# <a name="proxy-generation-enabled-attribute"></a>Attribut compatible avec la génération de proxy

**True** si la génération de proxy est activée.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | Compatible avec la génération de proxy             |
| LDAP-Display-Name | proxyGenerationEnabled               |
| Taille              | 4 octets                              |
| Mettre à jour le privilège  | \-                                   |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.2.523               |
| System-ID-GUID    | 5fd424d6-1262-11d0-a060-00aa006c33ed |
| Syntaxe            | [**Boolean**](s-boolean.md)         |
