# <a name="ms-ds-cached-membership-time-stamp-attribute"></a>ms-DS-Cached-Membership-horodatage

L’attribut **ms-DS-Cached-Membership-Time-Stamp** est utilisé par le gestionnaire des comptes de sécurité pour l’extension du groupe lors de l’évaluation du jeton.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | ms-DS-Cached-Membership-horodatage   |
| LDAP-Display-Name | msDS-Cached-Membership-horodatage    |
| Taille              | 8 octets                              |
| Mettre à jour le privilège  | Cette valeur est définie par le système.     |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1442              |
| System-ID-GUID    | 3566bf1f-beee-4dcb-8abe-ef89fcfec6c1 |
| Syntaxe            | [**Défini**](s-interval.md)       |
