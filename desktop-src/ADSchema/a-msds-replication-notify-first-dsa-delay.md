# <a name="ms-ds-replication-notify-first-dsa-delay-attribute"></a>attribut ms-DS-Replication-Notify-First-DSA-Delay

Cet attribut contrôle le délai entre les modifications du DS et la notification du premier partenaire de réplication pour un contexte de nommage.



| Entrée | Valeur |
|-------------------|------------------------------------------|
| CN                | ms-DS-réplication-notification-premier-DSA-Delay |
| LDAP-Display-Name | msDS-Replication-notification-First-DSA-Delay  |
| Taille              | \-                                       |
| Mettre à jour le privilège  | Cette valeur est définie par le système.         |
| Fréquence des mises à jour  | \-                                       |
| Attribute-Id      | 1.2.840.113556.1.4.1663                  |
| System-ID-GUID    | 85abd4f4-0a89-4e49-bdec-6f35bb2562ba     |
| Syntaxe            | [**Énumération**](s-enumeration.md)     |
