# <a name="add-guid-extended-right"></a>Add-GUID étendu droit

Droit étendu nécessaire à la racine du contexte de nommage pour ajouter un objet avec un GUID spécifique.



| Entrée | Valeur |
|--------------|--------------------------------------|
| CN           | Add-GUID                             |
| Display-Name | Ajoutez le GUID                             |
| Rights-GUID  | 440820ad-65b4-11d1-a3da-0000f875ae0d |
