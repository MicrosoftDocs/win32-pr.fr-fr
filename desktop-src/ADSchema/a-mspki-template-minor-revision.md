# <a name="ms-pki-template-minor-revision-attribute"></a>MS-PKI-template-mineur-Revision (attribut)

Effectue le suivi des attributs dans la modification de la classe. Toutefois, cela ne déclenchera pas l’inscription automatique.



| Entrée | Valeur |
|-------------------|---------------------------------------------------------------------------------------------------|
| CN                | MS-PKI-template-mineur-révision                                                                    |
| LDAP-Display-Name | Attribut mspki-template-mineur-révision                                                                     |
| Taille              | 4 octets                                                                                           |
| Mettre à jour le privilège  | Administrateur de domaine                                                                              |
| Fréquence des mises à jour  | Lorsque l’objet modèle de certificat (MS-PKI-Certificate-template) est modifié, créé ou cloné. |
| Attribute-Id      | 1.2.840.113556.1.4.1435                                                                           |
| System-ID-GUID    | 13f5236c-1884-46b1-b5d0-484e38990d58                                                              |
| Syntaxe            | [**Énumération**](s-enumeration.md)                                                              |
