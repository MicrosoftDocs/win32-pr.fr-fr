# <a name="ldap-display-name-attribute"></a>Attribut LDAP-Display-Name

Nom utilisé par les clients LDAP, tels que le fournisseur LDAP ADSI, pour lire et écrire l’attribut à l’aide du protocole LDAP.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | LDAP-Display-Name                           |
| LDAP-Display-Name | lDAPDisplayName                             |
| Taille              | \-                                          |
| Mettre à jour le privilège  | Administrateur de schéma                        |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.2.460                      |
| System-ID-GUID    | bf96799a-0de6-11d0-a285-00aa003049e2        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
