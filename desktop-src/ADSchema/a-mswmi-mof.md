# <a name="ms-wmi-mof-attribute"></a>MS-WMI-attribut MOF

Contient une définition MOF de certains objets WMI.

|-------------------|---------------------------------------------|
| CN                | MS-WMI-MOF                                  |
| LDAP-Display-Name | msWMI-MOF                                   |
| Taille              | \-                                          |
| Mettre à jour le privilège  | Administrateur stratégie de groupe                  |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1638                     |
| System-ID-GUID    | 6736809f-2064-443e-a145-81262b1f1366        |
