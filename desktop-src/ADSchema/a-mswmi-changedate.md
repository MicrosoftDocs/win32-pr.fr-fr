# <a name="ms-wmi-changedate-attribute"></a>MS-WMI-attribut de modification

| Entrée | Valeur |
|-------------------|---------------------------------------------------|
| CN                | MS-WMI-modifier                                 |
| LDAP-Display-Name | msWMI-modifier                                  |
| Taille              | Représentation du type de données DateTime WMI-26 caractères. |
| Mettre à jour le privilège  | Cette valeur est définie par le système.                  |
| Fréquence des mises à jour  | \-                                                |
| Attribute-Id      | 1.2.840.113556.1.4.1624                           |
| System-ID-GUID    | f9cdf7a0-ec44-4937-a79b-cd91522b3aa8              |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md)       |
