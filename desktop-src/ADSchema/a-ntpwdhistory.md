# <a name="nt-pwd-history-attribute"></a>Attribut NT-pwd-History

Historique du mot de passe de l’utilisateur dans le format à sens unique Windows NT (OWF). Windows 2000 utilise Windows NT OWF.



| Entrée | Valeur |
|-------------------|-------------------------------------------------------|
| CN                | NT-pwd-historique                                        |
| LDAP-Display-Name | ntPwdHistory                                          |
| Taille              | \-                                                    |
| Mettre à jour le privilège  | Cette valeur est définie par le système.                      |
| Fréquence des mises à jour  | Chaque fois que l’utilisateur modifie les mots de passe.                 |
| Attribute-Id      | 1.2.840.113556.1.4.94                                 |
| System-ID-GUID    | bf9679e2-0de6-11d0-a285-00aa003049e2                  |
| Syntaxe            | [**Object(Replica-Link)**](s-object-replica-link.md) |
