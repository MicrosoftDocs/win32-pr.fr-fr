# <a name="ms-ds-az-application-name-attribute"></a>ms-DS-AZ-application-name (attribut)

Chaîne qui identifie de façon unique un objet d’application.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | ms-DS-AZ-application-name                   |
| LDAP-Display-Name | msDS-AzApplicationName                      |
| Taille              | 128 caractères                              |
| Mettre à jour le privilège  | Administrateur AzRoles                               |
| Fréquence des mises à jour  | Pendant l’initialisation ou la modification de la stratégie.     |
| Attribute-Id      | 1.2.840.113556.1.4.1798                     |
| System-ID-GUID    | db5b0728-6208-4876-83b7-95d3e5695275        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
