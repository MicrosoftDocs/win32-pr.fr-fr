# <a name="proxied-object-name-attribute"></a>Proxyd-Object-Name (attribut)

Cet attribut est utilisé en interne par Active Directory pour faciliter le suivi des déplacements interdomaines.



| Entrée | Valeur |
|-------------------|-------------------------------------------------|
| CN                | Proxyd-Object-Name                             |
| LDAP-Display-Name | proxiedObjectName                               |
| Taille              | \-                                              |
| Mettre à jour le privilège  | Utilisé par le système.                     |
| Fréquence des mises à jour  | \-                                              |
| Attribute-Id      | 1.2.840.113556.1.4.1249                         |
| System-ID-GUID    | e1aea402-cd5b-11d0-afff-0000f80367c1            |
| Syntaxe            | [**Object(DN-Binary)**](s-object-dn-binary.md) |
