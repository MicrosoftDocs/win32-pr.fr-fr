# <a name="rights-guid-attribute"></a>Attribut Rights-Guid

GUID utilisé pour représenter un droit étendu dans une entrée de contrôle d’accès.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | Rights-Guid                                 |
| LDAP-Display-Name | rightsGuid                                  |
| Taille              | 16 octets                                    |
| Mettre à jour le privilège  | Cette valeur est définie par le système.            |
| Fréquence des mises à jour  | Lors de la création d’un nouveau droit étendu.       |
| Attribute-Id      | 1.2.840.113556.1.4.340                      |
| System-ID-GUID    | 8297931c-86d3-11d0-afda-00c04fd930c9        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
