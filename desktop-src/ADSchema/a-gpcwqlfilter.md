# <a name="gpc-wql-filter-attribute"></a>GPC-WQL-Filter (attribut)

Utilisé pour stocker une chaîne qui contient un GUID pour le filtre et un chemin d’accès d’espace de noms WMI.



| Entrée | Valeur |
|-------------------|---------------------------------------------------------------------------------|
| CN                | GPC-WQL-Filter                                                                  |
| LDAP-Display-Name | gPCWQLFilter                                                                    |
| Taille              | \-                                                                              |
| Mettre à jour le privilège  | Administrateur de domaine                                                            |
| Fréquence des mises à jour  | Uniquement lorsque l’administrateur modifie la propriété de filtre dans l’interface utilisateur stratégie de groupe. |
| Attribute-Id      | 1.2.840.113556.1.4.1694                                                         |
| System-ID-GUID    | 7bd4c7a6-1add-4436-8c04-3999a880154c                                            |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md)                                     |
