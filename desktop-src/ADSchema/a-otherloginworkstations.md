# <a name="other-login-workstations-attribute"></a>Autre attribut-login-stations de travail

Stations de travail non Windows NT ou LAN Manager à partir desquelles un utilisateur peut se connecter.

> [!Note]  
> Active Directory n’utilise pas ou ne renseigne pas ce champ.

 



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | Autres-connexions-stations de travail                    |
| LDAP-Display-Name | otherLoginWorkstations                      |
| Taille              | \-                                          |
| Mettre à jour le privilège  | \-                                          |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.91                       |
| System-ID-GUID    | bf9679f1-0de6-11d0-a285-00aa003049e2        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
