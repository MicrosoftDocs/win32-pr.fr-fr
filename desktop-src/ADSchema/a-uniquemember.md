# <a name="uniquemember-attribute"></a>attribut uniqueMember

Nom unique du membre d’un groupe. Utilisé par groupOfUniqueNames.



| Entrée | Valeur |
|-------------------|-----------------------------------------|
| CN                | uniqueMember                            |
| LDAP-Display-Name | uniqueMember                            |
| Taille              | \-                                      |
| Mettre à jour le privilège  | \-                                      |
| Fréquence des mises à jour  | \-                                      |
| Attribute-Id      | 2.5.4.50                                |
| System-ID-GUID    | 8f888726-f80a-44d7-b1ee-cb9df21392c8    |
| Syntaxe            | [**Object(DS-DN)**](s-object-ds-dn.md) |
