# <a name="user-logon-property-set"></a>Jeu de propriétés User-Logon

Jeu de propriétés contenant les attributs utilisateur qui décrivent les informations d’ouverture de session de l’utilisateur.



| Entrée | Valeur |
|--------------|--------------------------------------|
| CN           | User-Logon                           |
| Display-Name | Informations de connexion                    |
| Rights-GUID  | 5f202010-79a5-11d0-9020-00c04fc2d4cf |
