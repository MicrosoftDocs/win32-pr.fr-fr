# <a name="e-mail-addresses-attribute"></a>Attribut E-mail-addresses

Liste des adresses de messagerie d’un contact.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | Adresses de messagerie                            |
| LDAP-Display-Name | mail                                        |
| Taille              | \-                                          |
| Mettre à jour le privilège  | \-                                          |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 0.9.2342.19200300.100.1.3                   |
| System-ID-GUID    | bf967961-0de6-11d0-a285-00aa003049e2        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
