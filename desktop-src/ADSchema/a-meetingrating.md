# <a name="meetingrating-attribute"></a>attribut meetingRating

Plateforme pour la classification de la sélection de contenu Internet d’une réunion.

|-------------------|---------------------------------------------|
| CN                | meetingRating                               |
| LDAP-Display-Name | meetingRating                               |
| Taille              | \-                                          |
| Mettre à jour le privilège  | \-                                          |
| Fréquence des mises à jour  | \-                                          |

| System-ID-GUID    | 11b6cc8d-48c4-11d1-a9c3-0000f80367c1        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
