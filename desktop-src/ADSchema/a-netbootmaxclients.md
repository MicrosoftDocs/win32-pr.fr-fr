# <a name="netboot-max-clients-attribute"></a>netboot-attribut Max-clients

L’attribut **netboot-Max-clients** est réservé à un usage interne.

|-------------------|--------------------------------------|
| CN                | netboot-Max-clients                  |
| LDAP-Display-Name | netbootMaxClients                    |
| Taille              | \-                                   |
| Mettre à jour le privilège  | Cette valeur est définie par le système.     |
| Fréquence des mises à jour  | \-                                   |

| System-ID-GUID    | 07383078-91df-11d1-aebc-0000f80367c1 |
