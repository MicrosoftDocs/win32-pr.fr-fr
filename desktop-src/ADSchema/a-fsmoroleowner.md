# <a name="fsmo-role-owner-attribute"></a>Attribut FSMO de rôle-propriétaire

Opération de Single-Master flexible : nom unique du contrôleur de l’emplacement où le schéma peut être modifié.



| Entrée | Valeur |
|-------------------|-----------------------------------------|
| CN                | FSMO-Role-owner                         |
| LDAP-Display-Name | fSMORoleOwner                           |
| Taille              | \-                                      |
| Mettre à jour le privilège  | Administrateur de schéma                    |
| Fréquence des mises à jour  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.369                  |
| System-ID-GUID    | 66171887-8f3c-11d0-afda-00c04fd930c9    |
| Syntaxe            | [**Object(DS-DN)**](s-object-ds-dn.md) |
