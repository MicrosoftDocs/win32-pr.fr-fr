# <a name="phone-pager-primary-attribute"></a>Téléphone-téléavertisseur-attribut principal

Numéro du récepteur de radiomessagerie principal.



| Entrée | Valeur |
|-------------------|----------------------------------------------------------------------------------|
| CN                | Téléphone-récepteur de radiomessagerie principal                                                              |
| LDAP-Display-Name | pager                                                                            |
| Taille              | \-                                                                               |
| Mettre à jour le privilège  | Administrateur de domaine ou propriétaire du compte.                                           |
| Fréquence des mises à jour  | Lorsque l’enregistrement de l’utilisateur est créé et chaque fois que le numéro de téléphone doit être modifié. |
| Attribute-Id      | 0.9.2342.19200300.100.1.42                                                       |
| System-ID-GUID    | f0f8ffa6-1191-11d0-a060-00aa006c33ed                                             |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md)                                      |
