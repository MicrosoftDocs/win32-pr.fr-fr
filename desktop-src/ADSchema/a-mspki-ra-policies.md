# <a name="ms-pki-ra-policies-attribute"></a>attribut ms-PKI-RA-Policies

Contient la liste des OID de stratégie requis des autorités d’inscription qui signent la demande d’inscription.



| Entrée | Valeur |
|-------------------|---------------------------------------------------------------------------------------------------|
| CN                | MS-PKI-RA-stratégies                                                                                |
| LDAP-Display-Name | Attribut mspki-RA-stratégies                                                                                 |
| Taille              | 64 octets multiplié par le nombre d’entrées.                                                             |
| Mettre à jour le privilège  | Administrateur de domaine                                                                              |
| Fréquence des mises à jour  | Lorsque l’objet modèle de certificat (MS-PKI-Certificate-template) est modifié, créé ou cloné. |
| Attribute-Id      | 1.2.840.113556.1.4.1438                                                                           |
| System-ID-GUID    | d546ae22-0951-4d47-817e-1c9f96faad46                                                              |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md)                                                       |
