# <a name="other-name-attribute"></a>Attribut Other-Name

Noms supplémentaires pour un utilisateur. Par exemple, Middle Name, patronymic, matronymic, etc.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | Other-Name                                  |
| LDAP-Display-Name | middleName                                  |
| Taille              | \-                                          |
| Mettre à jour le privilège  | \-                                          |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 2.16.840.1.113730.3.1.34                    |
| System-ID-GUID    | bf9679f2-0de6-11d0-a285-00aa003049e2        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
