# <a name="print-duplex-supported-attribute"></a>Print-attribut pris en charge en mode duplex

Indique le type de prise en charge duplex d’une imprimante.



| Entrée | Valeur |
|-------------------|-------------------------------------------------------|
| CN                | Print-duplex-pris en charge                                |
| LDAP-Display-Name | printDuplexSupported                                  |
| Taille              | 4 octets. Duplex pris en charge = 2. Simplex uniquement = 1 ou 0. |
| Mettre à jour le privilège  | \-                                                    |
| Fréquence des mises à jour  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.1311                               |
| System-ID-GUID    | 281416cc-1968-11d0-a28f-00aa003049e2                  |
| Syntaxe            | [**Boolean**](s-boolean.md)                          |
