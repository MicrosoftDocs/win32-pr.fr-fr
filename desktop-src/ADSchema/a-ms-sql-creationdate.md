# <a name="ms-sql-creationdate-attribute"></a>MS-SQL-CreationDate (attribut)

Date à laquelle la base de données a été créée.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | MS-SQL-CreationDate                         |
| LDAP-Display-Name | mS-SQL-CreationDate                         |
| Taille              | \-                                          |
| Mettre à jour le privilège  | Cette valeur est définie par le système.            |
| Fréquence des mises à jour  | Lors de la création d’une base de données.             |
| Attribute-Id      | 1.2.840.113556.1.4.1397                     |
| System-ID-GUID    | ede14754-ccee-11d2-9993-0000f87a57d4        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
