# <a name="is-critical-system-object-attribute"></a>Attribut is-Critical-System-Object

Si la **valeur est true**, l’objet qui héberge cet attribut doit être répliqué lors de l’installation d’un nouveau réplica.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | Is-Critical-System-Object            |
| LDAP-Display-Name | isCriticalSystemObject               |
| Taille              | \-                                   |
| Mettre à jour le privilège  | \-                                   |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.868               |
| System-ID-GUID    | 00fbf30d-91fe-11d1-aebc-0000f80367c1 |
| Syntaxe            | [**Boolean**](s-boolean.md)         |
