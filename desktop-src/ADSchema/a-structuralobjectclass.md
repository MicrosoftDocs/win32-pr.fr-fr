# <a name="structural-object-class-attribute"></a>Structural-Object-Class (attribut)

Cet attribut construit stocke une liste de classes contenues dans une hiérarchie de classes, y compris les classes abstraites. Cette liste contient des classes auxiliaires liées de manière dynamique.



| Entrée | Valeur |
|-------------------|-----------------------------------------------------------------|
| CN                | Structural-Object-Class                                         |
| LDAP-Display-Name | structuralObjectClass                                           |
| Taille              | \-                                                              |
| Mettre à jour le privilège  | Cette valeur est définie par le système.                                |
| Fréquence des mises à jour  | Lors de la création de la classe.                                              |
| Attribute-Id      | 2.5.21.9                                                        |
| System-ID-GUID    | 3860949f-f6a8-4b38-9950-81ecb6bc2982                            |
| Syntaxe            | [**String(Object-Identifier)**](s-string-object-identifier.md) |
