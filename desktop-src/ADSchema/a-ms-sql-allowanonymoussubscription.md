# <a name="ms-sql-allowanonymoussubscription-attribute"></a>Attribut MS-SQL-AllowAnonymousSubscription

True si la publication autorise les abonnés anonymes à s’y abonner.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | MS-SQL-AllowAnonymousSubscription    |
| LDAP-Display-Name | mS-SQL-AllowAnonymousSubscription    |
| Taille              | 4 octets                              |
| Mettre à jour le privilège  | Cette valeur est définie par le système.     |
| Fréquence des mises à jour  | Lorsque la réplication est configurée.           |
| Attribute-Id      | 1.2.840.113556.1.4.1394              |
| System-ID-GUID    | db77be4a-ccee-11d2-9993-0000f87a57d4 |
| Syntaxe            | [**Boolean**](s-boolean.md)         |
