# <a name="ms-ds-az-generate-audits-attribute"></a>attribut ms-DS-AZ-Generate-audites

Champ booléen qui indique si les audits d’exécution doivent être activés (inclure les audits pour les vérifications d’accès, etc.).



| Entrée | Valeur |
|-------------------|-----------------------------------------|
| CN                | ms-DS-AZ-Generate-audites                |
| LDAP-Display-Name | msDS-AzGenerateAudits                   |
| Taille              | \-                                      |
| Mettre à jour le privilège  | Administrateur AzRoles                           |
| Fréquence des mises à jour  | Pendant l’initialisation ou la modification de la stratégie. |
| Attribute-Id      | 1.2.840.113556.1.4.1805                 |
| System-ID-GUID    | f90abab0-186C-4418-bb85-88447c87222a    |
| Syntaxe            | [**Boolean**](s-boolean.md)            |
