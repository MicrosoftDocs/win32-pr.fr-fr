# <a name="proxy-addresses-attribute"></a>Attribut Proxy-Addresses

une adresse proxy est l’adresse par laquelle un objet destinataire Microsoft Exchange Server est reconnu dans un système de messagerie étranger. Des adresses proxy sont requises pour tous les objets destinataires, tels que les destinataires personnalisés et les listes de distribution.



| Entrée | Valeur |
|-------------------|------------------------------------------------------------------------------------------------------|
| CN                | Proxy-Addresses                                                                                      |
| LDAP-Display-Name | proxyAddresses                                                                                       |
| Taille              | \-                                                                                                   |
| Mettre à jour le privilège  | Cette valeur est définie par le système.                                                                     |
| Fréquence des mises à jour  | Créé par une DLL fournie avec l’objet Address-Type Directory lorsque le type d’adresse est installé. |
| Attribute-Id      | 1.2.840.113556.1.2.210                                                                               |
| System-ID-GUID    | bf967a06-0de6-11d0-a285-00aa003049e2                                                                 |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md)                                                          |
