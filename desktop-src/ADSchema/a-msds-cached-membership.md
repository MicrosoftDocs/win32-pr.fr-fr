# <a name="ms-ds-cached-membership-attribute"></a>attribut d’appartenance en cache ms-DS

L’attribut **ms-DS-Cached-Membership** est utilisé par le gestionnaire des comptes de sécurité pour l’extension du groupe lors de l’évaluation du jeton.



| Entrée | Valeur |
|-------------------|-------------------------------------------------------|
| CN                | ms-DS-mis en cache-appartenance                               |
| LDAP-Display-Name | msDS-Cached-Membership                                |
| Taille              | BLOB binaire pouvant atteindre jusqu’à 3 kilo-octets.                     |
| Mettre à jour le privilège  | Cette valeur est définie par le système.                      |
| Fréquence des mises à jour  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.1441                               |
| System-ID-GUID    | 69cab008-cdd4-4bc9-bab8-0ff37efe1b20                  |
| Syntaxe            | [**Object(Replica-Link)**](s-object-replica-link.md) |
