# <a name="ms-ds-all-users-trust-quota-attribute"></a>MS-DS-tous les utilisateurs-attribut de quota de confiance

Utilisé pour appliquer un quota d’utilisateurs combinés sur le nombre total d’objets Trusted-Domain créés à l’aide du nouveau droit d’accès de contrôle, créez-entrant-forêt-confiance.



| Entrée | Valeur |
|-------------------|-------------------------------------------|
| CN                | MS-DS-tout-utilisateurs-approbation-quota               |
| LDAP-Display-Name | msDS-AllUsersTrustQuota                   |
| Taille              | \-                                        |
| Mettre à jour le privilège  | Administrateur de domaine                      |
| Fréquence des mises à jour  | À la création de la forêt et rarement après cela. |
| Attribute-Id      | 1.2.840.113556.1.4.1789                   |
| System-ID-GUID    | d3aa4a5c-4e03-4810-97aa-2b339e7a434b      |
| Syntaxe            | [**Enumeration**](s-enumeration.md)      |
