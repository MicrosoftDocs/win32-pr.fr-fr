# <a name="documenttitle-attribute"></a>attribut documentTitle

Le type d’attribut documentTitle spécifie le titre d’un document.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | documentTitle                               |
| LDAP-Display-Name | documentTitle                               |
| Taille              | \-                                          |
| Mettre à jour le privilège  | \-                                          |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 0.9.2342.19200300.100.1.12                  |
| System-ID-GUID    | de265a9c-ff2c-47b9-91dc-6e6fe2c43062        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
