# <a name="machine-role-attribute"></a>Attribut Machine-Role

Rôle pour un ordinateur : DC, serveur ou station de travail.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | Machine-Role                         |
| LDAP-Display-Name | Rôle                          |
| Taille              | \-                                   |
| Mettre à jour le privilège  | \-                                   |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.71                |
| System-ID-GUID    | bf9679b2-0de6-11d0-a285-00aa003049e2 |
