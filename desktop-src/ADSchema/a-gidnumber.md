# <a name="gidnumber-attribute"></a>attribut gidNumber

Contient une valeur entière qui identifie de façon unique un groupe dans un domaine administratif.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | gidNumber                            |
| LDAP-Display-Name | gidNumber                            |
| Taille              | \-                                   |
| Mettre à jour le privilège  | \-                                   |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.1                      |
| System-ID-GUID    | c5b95f0c-ec9e-41c4-849c-b46597ed6696 |
| Syntaxe            | [**Enumeration**](s-enumeration.md) |
