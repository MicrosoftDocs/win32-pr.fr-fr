# <a name="is-single-valued-attribute"></a>Is-attribut à valeur unique

Si la **valeur est true**, cet attribut ne peut stocker qu’une seule valeur.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | Est de valeur unique                     |
| LDAP-Display-Name | isSingleValued                       |
| Taille              | \-                                   |
| Mettre à jour le privilège  | Administrateur de schéma                 |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.2.33                |
| System-ID-GUID    | bf967992-0de6-11d0-a285-00aa003049e2 |
| Syntaxe            | [**Boolean**](s-boolean.md)         |
