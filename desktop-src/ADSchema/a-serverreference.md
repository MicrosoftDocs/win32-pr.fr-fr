# <a name="server-reference-attribute"></a>Attribut Server-Reference

Trouvé dans un objet ordinateur de site. Contient le nom unique du contrôleur de domaine dans le contexte d’appellation du domaine.



| Entrée | Valeur |
|-------------------|-----------------------------------------|
| CN                | Server-Reference                        |
| LDAP-Display-Name | serverReference                         |
| Taille              | \-                                      |
| Mettre à jour le privilège  | Cette valeur est définie par le système.        |
| Fréquence des mises à jour  | Chaque fois qu’un site est créé.             |
| Attribute-Id      | 1.2.840.113556.1.4.515                  |
| System-ID-GUID    | 26d9736d-6070-11d1-a9c6-0000f80367c1    |
| Syntaxe            | [**Object(DS-DN)**](s-object-ds-dn.md) |
