# <a name="ms-com-partitionsetlink-attribute"></a>attribut ms-COM-PartitionSetLink

Lien utilisé pour associer une partition COM+ à un objet COM+ PartitionSet.



| Entrée | Valeur |
|-------------------|-----------------------------------------|
| CN                | MS-COM-PartitionSetLink                 |
| LDAP-Display-Name | msCOM-PartitionSetLink                  |
| Taille              | \-                                      |
| Mettre à jour le privilège  | Cette valeur est définie par le système.        |
| Fréquence des mises à jour  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1424                 |
| System-ID-GUID    | 67f121dc-7d02-4c7d-82f5-9ad4c950ac34    |
| Syntaxe            | [**Object(DS-DN)**](s-object-ds-dn.md) |
