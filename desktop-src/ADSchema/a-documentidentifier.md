# <a name="documentidentifier-attribute"></a>attribut documentIdentifier

Le type d’attribut documentIdentifier spécifie un identificateur unique pour un document.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | documentIdentifier                          |
| LDAP-Display-Name | documentIdentifier                          |
| Taille              | \-                                          |
| Mettre à jour le privilège  | \-                                          |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 0.9.2342.19200300.100.1.11                  |
| System-ID-GUID    | 0b21ce82-ff63-46d9-90fb-c8b9f24e97b9        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
