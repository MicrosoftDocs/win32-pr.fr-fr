# <a name="ms-ds-nc-ro-replica-locations-attribute"></a>attribut ms-DS-NC-RO-Replica-Locations

Attribut lié sur un objet de référence croisée pour une partition. Répertorie le contrôleur de périphérique qui doit héberger la partition en lecture seule.



| Entrée | Valeur |
|-------------------|-----------------------------------------|
| CN                | ms-DS-NC-RO-Replica-Locations           |
| LDAP-Display-Name | msDS-NC-RO-réplica-emplacements            |
| Taille              | \-                                      |
| Mettre à jour le privilège  | \-                                      |
| Fréquence des mises à jour  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1967                 |
| System-ID-GUID    | 3df793df-9858-4417-A701-735a1ecebf74    |
| Syntaxe            | [**Object(DS-DN)**](s-object-ds-dn.md) |
