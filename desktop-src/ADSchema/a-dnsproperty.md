# <a name="dns-property-attribute"></a>Attribut DNS-Property

Utilisé pour stocker les paramètres binaires (propriétés) sur les objets de zone DNS.



| Entrée | Valeur |
|-------------------|-------------------------------------------------------|
| CN                | DNS-Property                                          |
| LDAP-Display-Name | dNSProperty                                           |
| Taille              | \-                                                    |
| Mettre à jour le privilège  | Cette valeur est définie par le système.                      |
| Fréquence des mises à jour  | Chaque fois que les enregistrements de zone DNS changent.                     |
| Attribute-Id      | 1.2.840.113556.1.4.1306                               |
| System-ID-GUID    | 675a15fe-3b70-11d2-90cc-00c04fd91ab1                  |
| Syntaxe            | [**Object(Replica-Link)**](s-object-replica-link.md) |
