# <a name="msmq-peek-computer-journal-extended-right"></a>MSMQ-Peek-ordinateur-journal étendu droit

Autorise l’aperçu des messages dans la file d’attente du journal de l’ordinateur.



| Entrée | Valeur |
|--------------|--------------------------------------|
| CN           | MSMQ-Peek-Computer-Journal           |
| Display-Name | Lire le journal de l’ordinateur                |
| Rights-GUID  | 4b6e08c3-df3c-11d1-9c86-006008764d0e |
