# <a name="msmq-multicast-address-attribute"></a>Attribut MSMQ-Multicast-Address

Dans le cadre d’un objet MSMQ, il est défini en appelant l’API MQCreateQueue ou MQSetProperties. Il détermine si MSMQ accepte les messages d’une adresse de multidiffusion.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | MSMQ-Multicast-Address                      |
| LDAP-Display-Name | MSMQ-MulticastAddress                       |
| Taille              | 4 octets                                     |
| Mettre à jour le privilège  | Propriétaire de la file d’attente.                            |
| Fréquence des mises à jour  | Lorsqu’une file d’attente est créée.                    |
| Attribute-Id      | 1.2.840.113556.1.4.1714                     |
| System-ID-GUID    | 1d2f4412-f10d-4337-9b48-6e5b125cd265        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
