# <a name="last-set-time-attribute"></a>Attribut Last-Set-Time

Heure de la dernière modification de la clé secrète. Cette valeur est stockée sous la forme d’un grand entier qui représente le nombre d’intervalles de 100 nanosecondes depuis le 1er janvier 1601 (UTC).



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | Last-Set-Time                        |
| LDAP-Display-Name | lastSetTime                          |
| Taille              | 8 octets                              |
| Mettre à jour le privilège  | \-                                   |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.53                |
| System-ID-GUID    | bf967998-0de6-11d0-a285-00aa003049e2 |
| Syntaxe            | [**Défini**](s-interval.md)       |
