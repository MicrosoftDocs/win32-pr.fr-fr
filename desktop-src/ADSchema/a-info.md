# <a name="comment-attribute-ad-schema"></a>Attribut comment (schéma Active Directory)

Commentaires de l’utilisateur. Cette chaîne peut être une chaîne NULL.



| Entrée | Valeur |
|-------------------|--------------------------------------------------------------------------|
| CN                | Commentaire                                                                  |
| LDAP-Display-Name | info                                                                     |
| Taille              | \-                                                                       |
| Mettre à jour le privilège  | Administrateur de domaine ou propriétaire du compte.                                   |
| Fréquence des mises à jour  | Chaque fois que l’utilisateur ou l’administrateur doit ajouter des commentaires sur le compte. |
| Attribute-Id      | 1.2.840.113556.1.2.81                                                    |
| System-ID-GUID    | bf96793e-0de6-11d0-a285-00aa003049e2                                     |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md)                              |
