# <a name="frs-extensions-attribute"></a>Attribut FRS-Extensions

Données binaires utilisées par la réplication de fichiers.



| Entrée | Valeur |
|-------------------|-------------------------------------------------------|
| CN                | FRS-Extensions                                        |
| LDAP-Display-Name | fRSExtensions                                         |
| Taille              | \-                                                    |
| Mettre à jour le privilège  | Cette valeur est définie par le système.                      |
| Fréquence des mises à jour  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.536                                |
| System-ID-GUID    | 52458020-ca6a-11D0-AFFF-0000f80367c1                  |
| Syntaxe            | [**Object(Replica-Link)**](s-object-replica-link.md) |
