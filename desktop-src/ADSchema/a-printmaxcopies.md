# <a name="print-max-copies-attribute"></a>Attribut Print-Max-copies

Nombre maximal de copies qu’un appareil peut imprimer.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | Impression-nombre maximal de copies                     |
| LDAP-Display-Name | printMaxCopies                       |
| Taille              | 4 octets                              |
| Mettre à jour le privilège  | \-                                   |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.241               |
| System-ID-GUID    | 281416d1-1968-11d0-a28f-00aa003049e2 |
| Syntaxe            | [**Enumeration**](s-enumeration.md) |
