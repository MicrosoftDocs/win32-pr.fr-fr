# <a name="unixhomedirectory-attribute"></a>attribut unixHomeDirectory

Contient le chemin d’accès absolu au répertoire de départ.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | unixHomeDirectory                    |
| LDAP-Display-Name | unixHomeDirectory                    |
| Taille              | \-                                   |
| Mettre à jour le privilège  | \-                                   |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.3                      |
| System-ID-GUID    | bc2dba12-000f-464d-bf1d-0808465d8843 |
| Syntaxe            | [**String(IA5)**](s-string-ia5.md)  |
