# <a name="ms-ds-members-for-az-role-bl-attribute"></a>ms-DS-Members-for-AZ-Role-BL Attribute

Lien vers l’arrière du groupe ou de l’utilisateur de l’application membre vers Az-Role objets qui y sont liés.



| Entrée | Valeur |
|-------------------|-----------------------------------------|
| CN                | ms-DS-Members-for-AZ-Role-BL            |
| LDAP-Display-Name | msDS-MembersForAzRoleBL                 |
| Taille              | \-                                      |
| Mettre à jour le privilège  | \-                                      |
| Fréquence des mises à jour  | Cette valeur est définie par le système.        |
| Attribute-Id      | 1.2.840.113556.1.4.1807                 |
| System-ID-GUID    | ececcd20-a7e0-4688-9ccf-02ece5e287f5    |
| Syntaxe            | [**Object(DS-DN)**](s-object-ds-dn.md) |
