# <a name="msradiusframedipaddress-attribute"></a>attribut msRADIUSFramedIPAddress

|-------------------|--------------------------------------|
| CN                | msRADIUSFramedIPAddress              |
| LDAP-Display-Name | msRADIUSFramedIPAddress              |

| Mettre à jour le privilège  | \-                                   |
