# <a name="dsa-signature-attribute"></a>Attribut DSA-Signature

La DSA-Signature d’un objet est l’ID d’appel du dernier répertoire pour modifier l’objet.



| Entrée | Valeur |
|-------------------|-------------------------------------------------------|
| CN                | DSA-Signature                                         |
| LDAP-Display-Name | dSASignature                                          |
| Taille              | \-                                                    |
| Mettre à jour le privilège  | Cette valeur est définie par le système.                      |
| Fréquence des mises à jour  | Chaque fois qu’un objet est modifié.                       |
| Attribute-Id      | 1.2.840.113556.1.2.74                                 |
| System-ID-GUID    | 167757bc-47f3-11d1-a9c3-0000f80367c1                  |
| Syntaxe            | [**Object(Replica-Link)**](s-object-replica-link.md) |
