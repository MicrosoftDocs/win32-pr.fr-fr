# <a name="rid-used-pool-attribute"></a>RID-used-pool (attribut)

Pools RID qui ont été utilisés par un contrôleur de périphérique.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | RID-used-pool                        |
| LDAP-Display-Name | rIDUsedPool                          |
| Taille              | 8 octets                              |
| Mettre à jour le privilège  | Cette valeur est définie par le système.     |
| Fréquence des mises à jour  | À chaque fois qu’un pool RID est vidé.      |
| Attribute-Id      | 1.2.840.113556.1.4.373               |
| System-ID-GUID    | 6617188b-8f3c-11d0-afda-00c04fd930c9 |
| Syntaxe            | [**Défini**](s-interval.md)       |
