# <a name="ms-ds-has-master-ncs-attribute"></a>attribut ms-DS-a-Master-NC

Contient les contextes d’attribution de noms détenus par un contrôleur de domaine. Cet attribut déconseillé a-Master-NC.



| Entrée | Valeur |
|-------------------|-----------------------------------------|
| CN                | ms-DS-a-Master-NC                    |
| LDAP-Display-Name | msDS-hasMasterNCs                       |
| Taille              | \-                                      |
| Mettre à jour le privilège  | Cette valeur est définie par le système.         |
| Fréquence des mises à jour  | À chaque fois qu’un nouveau contexte de nommage est créé.          |
| Attribute-Id      | 1.2.840.113556.1.4.1836                 |
| System-ID-GUID    | ae2de0e2-59d7-4d47-8d47-ed4dfe4357ad    |
| Syntaxe            | [**Object(DS-DN)**](s-object-ds-dn.md) |
