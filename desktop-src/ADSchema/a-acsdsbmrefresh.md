# <a name="acs-dsbm-refresh-attribute"></a>ACS-DSBM-Refresh (attribut)

Cet attribut contient la valeur du minuteur de l’intervalle qui détermine le moment où le gestionnaire de bande passante de sous-réseau (DSBM) désigné envoie un message d’actualisation (« \_ \_ DSBM ») à tous les gestionnaires de bande passante du sous-réseau dans un domaine.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | ACS-DSBM-actualiser                     |
| LDAP-Display-Name | aCSDSBMRefresh                       |
| Taille              | 4 octets                              |
| Mettre à jour le privilège  | \-                                   |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.777               |
| System-ID-GUID    | 1cb3559f-56d0-11d1-a9c6-0000f80367c1 |
| Syntaxe            | [**Énumération**](s-enumeration.md) |
