# <a name="acs-non-reserved-tx-size-attribute"></a>ACS-attribut non réservé-TX-Size

Taille de compartiment de jeton qu’une application peut utiliser avant qu’une réservation soit en place.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | ACS-non réservé-TX-Size             |
| LDAP-Display-Name | aCSNonReservedTxSize                 |
| Taille              | 8 octets                              |
| Mettre à jour le privilège  | \-                                   |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.898               |
| System-ID-GUID    | f072230d-aef5-11d1-bdcf-0000f80367c1 |
| Syntaxe            | [**Défini**](s-interval.md)       |
