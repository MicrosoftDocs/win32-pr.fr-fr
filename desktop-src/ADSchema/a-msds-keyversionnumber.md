# <a name="ms-ds-keyversionnumber-attribute"></a>attribut ms-DS-KeyVersionNumber

Numéro de version Kerberos de la clé actuelle pour ce compte. Il s’agit d’un attribut construit.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | ms-DS-KeyVersionNumber               |
| LDAP-Display-Name | msDS-KeyVersionNumber                |
| Taille              | \-                                   |
| Mettre à jour le privilège  | \-                                   |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1782              |
| System-ID-GUID    | c523e9c0-33b5-4ac8-8923-b57b927f42f6 |
| Syntaxe            | [**Enumeration**](s-enumeration.md) |
