# <a name="when-created-attribute"></a>Attribut When-Created

Date à laquelle cet objet a été créé. Cette valeur est répliquée et se trouve dans le catalogue global.



| Entrée | Valeur |
|-------------------|---------------------------------------------------------------|
| CN                | When-Created                                                  |
| LDAP-Display-Name | whenCreated                                                   |
| Taille              | 8 octets                                                       |
| Mettre à jour le privilège  | Cette valeur est définie par le système.                              |
| Fréquence des mises à jour  | Lorsque l’objet est créé.                                   |
| Attribute-Id      | 1.2.840.113556.1.2.2                                          |
| System-ID-GUID    | bf967a78-0de6-11d0-a285-00aa003049e2                          |
| Syntaxe            | [**String(Generalized-Time)**](s-string-generalized-time.md) |
