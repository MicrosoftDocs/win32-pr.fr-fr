# <a name="acs-non-reserved-tx-limit-attribute"></a>ACS-attribut non réservé-TX-Limit

Bande passante maximale qu’une application utilisateur peut transmettre avant qu’une réservation soit en place.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | ACS-non réservé-TX-Limit            |
| LDAP-Display-Name | aCSNonReservedTxLimit                |
| Taille              | 8 octets                              |
| Mettre à jour le privilège  | \-                                   |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.780               |
| System-ID-GUID    | 1cb355a2-56d0-11d1-a9c6-0000f80367c1 |
| Syntaxe            | [**Défini**](s-interval.md)       |
