# <a name="ms-sql-contact-attribute"></a>Attribut MS-SQL-contact

Chaîne définie par l’utilisateur. La valeur par défaut est contact.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | Contact MS-SQL                              |
| LDAP-Display-Name | Contact mS-SQL                              |
| Taille              | \-                                          |
| Mettre à jour le privilège  | Administrateur de domaine                        |
| Fréquence des mises à jour  | Dans le programme d’installation du système.                            |
| Attribute-Id      | 1.2.840.113556.1.4.1365                     |
| System-ID-GUID    | 4f6cbdd8-ccee-11d2-9993-0000f87a57d4        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
