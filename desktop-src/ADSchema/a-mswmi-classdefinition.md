# <a name="ms-wmi-classdefinition-attribute"></a>attribut ms-WMI-ClassDefinition

Contient une définition de classe à mettre en état dans un espace de noms WMI.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | MS-WMI-ClassDefinition                      |
| LDAP-Display-Name | msWMI-ClassDefinition                       |
| Taille              | \-                                          |
| Mettre à jour le privilège  | Cette valeur est définie par le système.            |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1625                     |
| System-ID-GUID    | 2b9c0ebc-c272-45cb-99d2-4d0e691632e0        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
