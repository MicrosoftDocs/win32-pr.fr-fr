# <a name="netboot-answer-requests-attribute"></a>attribut netboot-réponse-demandes

Permet au serveur RIS d’accepter toutes les demandes RIS.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | netboot-réponse-demandes              |
| LDAP-Display-Name | netbootAnswerRequests                |
| Taille              | \-                                   |
| Mettre à jour le privilège  | Cette valeur est définie par le système.     |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.853               |
| System-ID-GUID    | 0738307a-91df-11d1-aebc-0000f80367c1 |
| Syntaxe            | [**Boolean**](s-boolean.md)         |
