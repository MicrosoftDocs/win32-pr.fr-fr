# <a name="ms-ds-az-task-is-role-definition-attribute"></a>attribut ms-DS-AZ-Task-is-Role-Definition

Champ booléen qui indique si AzTask est une tâche classique ou une définition de rôle.



| Entrée | Valeur |
|-------------------|-----------------------------------------|
| CN                | ms-DS-AZ-Task-is-Role-Definition        |
| LDAP-Display-Name | msDS-AzTaskIsRoleDefinition             |
| Taille              | \-                                      |
| Mettre à jour le privilège  | Administrateur AzRoles                           |
| Fréquence des mises à jour  | Pendant l’initialisation ou la modification de la stratégie. |
| Attribute-Id      | 1.2.840.113556.1.4.1818                 |
| System-ID-GUID    | 7b078544-6c82-4fe9-872f-ff48ad2b2e26    |
| Syntaxe            | [**Boolean**](s-boolean.md)            |
