# <a name="ms-ds-replication-notify-subsequent-dsa-delay-attribute"></a>ms-DS-réplication-notifier-suivant-attribut DSA-Delay

Cet attribut contrôle le délai entre la notification de chaque partenaire de réplica suivant pour un contexte de nommage.



| Entrée | Valeur |
|-------------------|-----------------------------------------------|
| CN                | ms-DS-réplication-notifier-suivant-DSA-Delay |
| LDAP-Display-Name | msDS-Replication-Notify-subséquent-DSA-Delay  |
| Taille              | \-                                            |
| Mettre à jour le privilège  | Cette valeur est définie par le système.              |
| Fréquence des mises à jour  | \-                                            |
| Attribute-Id      | 1.2.840.113556.1.4.1664                       |
| System-ID-GUID    | d63db385-dd92-4b52-b1d8-0d3ecc0e86b6          |
| Syntaxe            | [**Enumeration**](s-enumeration.md)          |
