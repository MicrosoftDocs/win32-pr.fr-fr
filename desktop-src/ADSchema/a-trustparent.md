# <a name="trust-parent-attribute"></a>Attribut Trust-Parent

Parent dans la hiérarchie d’approbation KERBEROS.



| Entrée | Valeur |
|-------------------|-----------------------------------------|
| CN                | Trust-Parent                            |
| LDAP-Display-Name | trustParent                             |
| Taille              | \-                                      |
| Mettre à jour le privilège  | Cette valeur est définie par le système.        |
| Fréquence des mises à jour  | Lorsqu’un domaine enfant est créé.         |
| Attribute-Id      | 1.2.840.113556.1.4.471                  |
| System-ID-GUID    | b000ea7a-a086-11d0-afdd-00c04fd930c9    |
| Syntaxe            | [**Object(DS-DN)**](s-object-ds-dn.md) |
