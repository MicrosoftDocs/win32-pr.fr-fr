# <a name="ipserviceprotocol-attribute"></a>attribut ipServiceProtocol

Contient la section de la carte des services qui stocke le numéro de protocole d’un service UNIX.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | ipServiceProtocol                    |
| LDAP-Display-Name | ipServiceProtocol                    |
| Taille              | \-                                   |
| Mettre à jour le privilège  | \-                                   |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.16                     |
| System-ID-GUID    | cd96ec0b-1ed6-43b4-b26b-f170b645883f |
| Syntaxe            | [**String(IA5)**](s-string-ia5.md)  |
