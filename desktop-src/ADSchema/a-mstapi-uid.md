# <a name="ms-tapi-unique-identifier-attribute"></a>attribut ms-TAPI-unique-identifier

Fournit le nom d’une conférence de multidiffusion TAPI. Il s’agit de l’attribut de nom de la classe Rt-Conference.



| Entrée | Valeur |
|-------------------|-----------------------------------------------------------------------|
| CN                | MS-TAPI-unique-identifier                                             |
| LDAP-Display-Name | msTAPI-UID                                                            |
| Taille              | Peut être une chaîne arbitraire, généralement inférieure à 100 caractères. |
| Mettre à jour le privilège  | Aucun privilège spécial n’est requis.                                       |
| Fréquence des mises à jour  | Défini une fois au moment de la création de l’objet Rt-Conference.            |
| Attribute-Id      | 1.2.840.113556.1.4.1698                                               |
| System-ID-GUID    | 70a4e7ea-b3b9-4643-8918-e6dd2471bfd4                                  |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md)                           |
