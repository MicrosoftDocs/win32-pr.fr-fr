# <a name="help-data16-attribute"></a>Attribut Help-Data16

Cet attribut a été utilisé pour le format de fichier d’aide Win16 pour Exchange 4,0. Elle n’est pas utilisée pour les autres versions d’Exchange.



| Entrée | Valeur |
|-------------------|-------------------------------------------------------|
| CN                | Help-Data16                                           |
| LDAP-Display-Name | helpData16                                            |
| Taille              | \-                                                    |
| Mettre à jour le privilège  | Utilisé par le système.                           |
| Fréquence des mises à jour  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.2.402                                |
| System-ID-GUID    | 5fd424a7-1262-11d0-a060-00aa006c33ed                  |
| Syntaxe            | [**Object(Replica-Link)**](s-object-replica-link.md) |



## <a name="implementations"></a>Implémentations

-   [**Windows Server 2003**](#windows-server-2003)

-   [**Windows Server 2008**](#windows-server-2008)
