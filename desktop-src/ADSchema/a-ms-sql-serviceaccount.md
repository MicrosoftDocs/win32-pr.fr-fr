# <a name="ms-sql-serviceaccount-attribute"></a>Attribut MS-SQL-, compte

Chaîne définie par l’utilisateur. La valeur par défaut est, compte.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | MS-SQL-, compte                       |
| LDAP-Display-Name | mS-SQL-, compte                       |
| Taille              | \-                                          |
| Mettre à jour le privilège  | Administrateur de domaine                        |
| Fréquence des mises à jour  | Dans le programme d’installation du système.                            |
| Attribute-Id      | 1.2.840.113556.1.4.1369                     |
| System-ID-GUID    | 64933a3e-ccee-11d2-9993-0000f87a57d4        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
