# <a name="international-isdn-number-attribute"></a>Attribut international-RNIS-Number

Spécifie un numéro RNIS international associé à un objet.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | International-RNIS-numéro                   |
| LDAP-Display-Name | internationalISDNNumber                     |
| Taille              | \-                                          |
| Mettre à jour le privilège  | \-                                          |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 2.5.4.25                                    |
| System-ID-GUID    | bf96798d-0de6-11d0-a285-00aa003049e2        |
| Syntaxe            | [**String(Numeric)**](s-string-numeric.md) |
