# <a name="loginshell-attribute"></a>attribut loginShell

Contient le chemin d’accès à l’environnement de connexion.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | loginShell                           |
| LDAP-Display-Name | loginShell                           |
| Taille              | \-                                   |
| Mettre à jour le privilège  | \-                                   |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.4                      |
| System-ID-GUID    | a553d12c-3231-4c5e-8adf-8d189697721e |
| Syntaxe            | [**String(IA5)**](s-string-ia5.md)  |
