# <a name="last-backup-restoration-time-attribute"></a>Attribut Last-Backup-Restoring-Time

Lorsque la dernière restauration du système s’est produite.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | Dernière-sauvegarde-restauration-heure         |
| LDAP-Display-Name | lastBackupRestorationTime            |
| Taille              | 8 octets                              |
| Mettre à jour le privilège  | Cette valeur est définie par le système.     |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.519               |
| System-ID-GUID    | 1fbb0be8-ba63-11d0-afef-0000f80367c1 |
| Syntaxe            | [**Défini**](s-interval.md)       |
