# <a name="ms-wmi-querylanguage-attribute"></a>attribut ms-WMI-QueryLanguage

Identifie un langage de requête WQL.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | MS-WMI-QueryLanguage                        |
| LDAP-Display-Name | msWMI-QueryLanguage                         |
| Taille              | Moins de vingt caractères.                |
| Mettre à jour le privilège  | Administrateur stratégie de groupe                  |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1643                     |
| System-ID-GUID    | 7d3cfa98-c17b-4254-8bd7-4de9b932a345        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
