# <a name="frs-primary-member-attribute"></a>FRS-attribut de membre principal

Référence au membre principal d’un jeu de réplicas.



| Entrée | Valeur |
|-------------------|-----------------------------------------|
| CN                | FRS-principal-membre                      |
| LDAP-Display-Name | fRSPrimaryMember                        |
| Taille              | \-                                      |
| Mettre à jour le privilège  | \-                                      |
| Fréquence des mises à jour  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.878                  |
| System-ID-GUID    | 2a132581-9373-11d1-aebc-0000f80367c1    |
| Syntaxe            | [**Object(DS-DN)**](s-object-ds-dn.md) |
