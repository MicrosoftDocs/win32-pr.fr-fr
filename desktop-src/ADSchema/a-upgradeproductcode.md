# <a name="upgrade-product-code-attribute"></a>Upgrade-Product-Code (attribut)

Cet attribut contient le code de produit d’autres packages, tels que les applications, qui peuvent être mis à niveau par ce package ou qui peut mettre à niveau ce package.



| Entrée | Valeur |
|-------------------|-------------------------------------------------------|
| CN                | Upgrade-Product-Code                                  |
| LDAP-Display-Name | upgradeProductCode                                    |
| Taille              | \-                                                    |
| Mettre à jour le privilège  | \-                                                    |
| Fréquence des mises à jour  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.813                                |
| System-ID-GUID    | d9e18312-8939-11d1-aebc-0000f80367c1                  |
| Syntaxe            | [**Object(Replica-Link)**](s-object-replica-link.md) |
