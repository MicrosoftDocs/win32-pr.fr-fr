# <a name="pki-default-csps-attribute"></a>PKI-attribut-default-CSP

Liste des fournisseurs de services de chiffrement pour le modèle de certificat.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | PKI-default-CSP                            |
| LDAP-Display-Name | pKIDefaultCSPs                              |
| Taille              | \-                                          |
| Mettre à jour le privilège  | \-                                          |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1334                     |
| System-ID-GUID    | 1ef6336e-3b9e-11d2-90cc-00c04fd91ab1        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
