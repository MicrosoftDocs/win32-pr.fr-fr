# <a name="pending-parent-ca-attribute"></a>Attribut Pending-parent-CA

Référence aux autorités de certification qui ont émis les certificats en attente pour cette autorité de certification.



| Entrée | Valeur |
|-------------------|-----------------------------------------|
| CN                | En attente-parent-CA                       |
| LDAP-Display-Name | pendingParentCA                         |
| Taille              | \-                                      |
| Mettre à jour le privilège  | \-                                      |
| Fréquence des mises à jour  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.695                  |
| System-ID-GUID    | 963d273e-48be-11d1-a9c3-0000f80367c1    |
| Syntaxe            | [**Object(DS-DN)**](s-object-ds-dn.md) |
