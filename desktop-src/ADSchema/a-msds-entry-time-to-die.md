# <a name="ms-ds-entry-time-to-die-attribute"></a>attribut ms-DS-Entry-Time-to-Die

Contient le délai d’expiration absolu d’un objet dynamique dans l’annuaire.



| Entrée | Valeur |
|-------------------|---------------------------------------------------------------|
| CN                | ms-DS-Entry-Time-to-Die                                       |
| LDAP-Display-Name | msDS-Entry-Time-to-Die                                        |
| Taille              | \-                                                            |
| Mettre à jour le privilège  | Cette valeur est définie par le système.                              |
| Fréquence des mises à jour  | \-                                                            |
| Attribute-Id      | 1.2.840.113556.1.4.1622                                       |
| System-ID-GUID    | e1e9bad7-c6dd-4101-A843-794cec85b038                          |
| Syntaxe            | [**String(Generalized-Time)**](s-string-generalized-time.md) |
