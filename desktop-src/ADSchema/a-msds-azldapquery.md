# <a name="ms-ds-az-ldap-query-attribute"></a>ms-DS-AZ-LDAP-Query (attribut)

Chaîne qui définit la requête LDAP (longueur maximale 4096) qui détermine l’appartenance d’un objet utilisateur au groupe.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | ms-DS-AZ-LDAP-requête                         |
| LDAP-Display-Name | msDS-AzLDAPQuery                            |
| Taille              | 4096 caractères                             |
| Mettre à jour le privilège  | Administrateur AzRoles                               |
| Fréquence des mises à jour  | Pendant l’initialisation ou la modification de la stratégie.     |
| Attribute-Id      | 1.2.840.113556.1.4.1792                     |
| System-ID-GUID    | 5e53368b-fc94-45c8-9d7d-daf31ee7112d        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
