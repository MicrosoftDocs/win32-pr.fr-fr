# <a name="subschemasubentry-attribute"></a>Attribut SubSchemaSubEntry

Nom unique de l’emplacement de l’objet de sous-schéma dans lequel une classe ou un attribut est défini.



| Entrée | Valeur |
|-------------------|----------------------------------------------------------------------------------|
| CN                | SubSchemaSubEntry                                                                |
| LDAP-Display-Name | subSchemaSubEntry                                                                |
| Taille              | \-                                                                               |
| Mettre à jour le privilège  | Administrateur de schéma                                                             |
| Fréquence des mises à jour  | Lors du chargement de la Active Directory ou lors de la définition d’une nouvelle classe ou d’un nouvel attribut. |
| Attribute-Id      | 2.5.18.10                                                                        |
| System-ID-GUID    | 9a7ad94d-ca53-11d1-bbd0-0080c76670c0                                             |
| Syntaxe            | [**Object(DS-DN)**](s-object-ds-dn.md)                                          |
