# <a name="ms-pki-oid-localizedname-attribute"></a>MS-PKI-OID-attribut LocalizedName

Contient une liste de noms complets utilisés pour décrire un OID par paramètres régionaux.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | MS-PKI-OID-LocalizedName                    |
| LDAP-Display-Name | Attribut mspki-OIDLocalizedName                      |
| Taille              | \-                                          |
| Mettre à jour le privilège  | Administrateur de domaine                        |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1712                     |
| System-ID-GUID    | 7d59a816-bb05-4a72-971f-5c1331f67559        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
