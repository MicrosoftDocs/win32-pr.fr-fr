# <a name="attribute-id-attribute"></a>Attribut-ID-attribut

OID X. 500 unique pour identifier un attribut.



| Entrée | Valeur |
|-------------------|-----------------------------------------------------------------|
| CN                | ID d’attribut                                                    |
| LDAP-Display-Name | attributeID                                                     |
| Taille              | \-                                                              |
| Mettre à jour le privilège  | Administrateur de schéma                                            |
| Fréquence des mises à jour  | \-                                                              |
| Attribute-Id      | 1.2.840.113556.1.2.30                                           |
| System-ID-GUID    | bf967922-0de6-11d0-a285-00aa003049e2                            |
| Syntaxe            | [**String(Object-Identifier)**](s-string-object-identifier.md) |
