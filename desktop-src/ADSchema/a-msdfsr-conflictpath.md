# <a name="ms-dfsr-conflictpath-attribute"></a>attribut ms-DFSR-ConflictPath

Contient le chemin d’accès complet du répertoire de conflit.

|-------------------|---------------------------------------------|
| CN                | MS-DFSR-ConflictPath                        |
| LDAP-Display-Name | msDFSR-ConflictPath                         |
| Taille              | \-                                          |
| Mettre à jour le privilège  | \-                                          |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.6.13.3.7                   |
| System-ID-GUID    | 5cf0bcc8-60f7-4bff-bda6-aea0344eb151        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
