# <a name="frs-time-last-config-change-attribute"></a>Attribut de modification de la dernière configuration de FRS

Heure à laquelle la dernière modification de la configuration a été acceptée.



| Entrée | Valeur |
|-------------------|---------------------------------------------------------------|
| CN                | FRS-heure-dernière-config-modification                                   |
| LDAP-Display-Name | fRSTimeLastConfigChange                                       |
| Taille              | \-                                                            |
| Mettre à jour le privilège  | \-                                                            |
| Fréquence des mises à jour  | \-                                                            |
| Attribute-Id      | 1.2.840.113556.1.4.881                                        |
| System-ID-GUID    | 2a132584-9373-11d1-aebc-0000f80367c1                          |
| Syntaxe            | [**String(Generalized-Time)**](s-string-generalized-time.md) |
