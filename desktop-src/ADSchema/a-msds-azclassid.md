# <a name="ms-ds-az-class-id-attribute"></a>ms-DS-AZ-Class-ID (attribut)

ID de classe requis par l’interface utilisateur AzRoles sur l’objet AzApplication.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | ms-DS-AZ-Class-ID                           |
| LDAP-Display-Name | msDS-AzClassId                              |
| Taille              | 36 caractères                               |
| Mettre à jour le privilège  | Administrateur AzRoles                               |
| Fréquence des mises à jour  | Pendant l’initialisation ou la modification de la stratégie.     |
| Attribute-Id      | 1.2.840.113556.1.4.1816                     |
| System-ID-GUID    | 013a7277-5c2d-49ef-a7de-b765b36a3f6f        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
