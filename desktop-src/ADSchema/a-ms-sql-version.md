# <a name="ms-sql-version-attribute"></a>MS-SQL-attribut Version

Version de l’instance actuelle de SQL Server.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | MS-SQL-Version                              |
| LDAP-Display-Name | mS-SQL-Version                              |
| Taille              | \-                                          |
| Mettre à jour le privilège  | Cette valeur est définie par le système.            |
| Fréquence des mises à jour  | Dans le programme d’installation du système.                            |
| Attribute-Id      | 1.2.840.113556.1.4.1388                     |
| System-ID-GUID    | c07cc1d0-ccee-11d2-9993-0000f87a57d4        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
