# <a name="certificate-authority-object-attribute"></a>Certificate-Authority-Object (attribut)

Référence à l’autorité de certification associée à un point de distribution de liste de révocation de certificats.



| Entrée | Valeur |
|-------------------|-----------------------------------------|
| CN                | Certificate-Authority-Object            |
| LDAP-Display-Name | certificateAuthorityObject              |
| Taille              | \-                                      |
| Mettre à jour le privilège  | \-                                      |
| Fréquence des mises à jour  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.684                  |
| System-ID-GUID    | 963d2732-48be-11d1-a9c3-0000f80367c1    |
| Syntaxe            | [**Object(DS-DN)**](s-object-ds-dn.md) |
