# <a name="print-color-attribute"></a>Attribut Print-Color

**True** si une imprimante peut imprimer en couleur.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | Print-Color                          |
| LDAP-Display-Name | printColor                           |
| Taille              | 4 octets                              |
| Mettre à jour le privilège  | \-                                   |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.243               |
| System-ID-GUID    | 281416d3-1968-11d0-a28f-00aa003049e2 |
| Syntaxe            | [**Boolean**](s-boolean.md)         |
