# <a name="rid-available-pool-attribute"></a>RID-available-attribut de pool

Espace à partir duquel les pools RID sont alloués.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | RID-available-pool                   |
| LDAP-Display-Name | rIDAvailablePool                     |
| Taille              | 8 octets                              |
| Mettre à jour le privilège  | Cette valeur est définie par le système.     |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.370               |
| System-ID-GUID    | 66171888-8f3c-11d0-afda-00c04fd930c9 |
| Syntaxe            | [**Défini**](s-interval.md)       |
