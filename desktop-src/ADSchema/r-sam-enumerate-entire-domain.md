# <a name="sam-enumerate-entire-domain-extended-right"></a>SAM-énumérer-totalité du droit étendu du domaine

Ce droit d’accès au contrôle peut être utilisé pour limiter les personnes autorisées à utiliser une API de niveau inférieur, par exemple NetQueryDisplayInformation et netuser/GroupEnum, et pour énumérer l’ensemble du domaine.



| Entrée | Valeur |
|--------------|--------------------------------------|
| CN           | SAM-énumérer-tout-domaine          |
| Display-Name | Énumérer l’intégralité du domaine SAM          |
| Rights-GUID  | 91d67418-0135-4acc-8d79-c08e857cfbec |
