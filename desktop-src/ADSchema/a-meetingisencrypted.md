# <a name="meetingisencrypted-attribute"></a>attribut meetingIsEncrypted

Cela est **vrai** si la réunion doit être chiffrée.



| Entrée | Valeur |
|-------------------|----------------------------------------------------------------------------------|
| CN                | meetingIsEncrypted                                                               |
| LDAP-Display-Name | meetingIsEncrypted                                                               |
| Taille              | \-                                                                               |
| Mettre à jour le privilège  | Tout le monde peut mettre à jour cet objet en fonction de la sécurité de l’objet en cours de création. |
| Fréquence des mises à jour  | \-                                                                               |
| Attribute-Id      | 1.2.840.113556.1.4.585                                                           |
| System-ID-GUID    | 11b6cc8e-48c4-11d1-a9c3-0000f80367c1                                             |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md)                                      |
