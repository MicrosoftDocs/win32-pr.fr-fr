# <a name="photo-attribute"></a>attribut photo

Un objet encodé dans la télécopie G3, comme expliqué dans la recommandation T. 4, avec un wrapper ASN. 1 pour le rendre compatible avec un corps X. 400 comme défini dans X. 420.



| Entrée | Valeur |
|-------------------|-------------------------------------------------------|
| CN                | photos                                                 |
| LDAP-Display-Name | photos                                                 |
| Taille              | \-                                                    |
| Mettre à jour le privilège  | \-                                                    |
| Fréquence des mises à jour  | \-                                                    |
| Attribute-Id      | 0.9.2342.19200300.100.1.7                             |
| System-ID-GUID    | 9c979768-ba1a-4c08-9632-c6a5c1ed649a                  |
| Syntaxe            | [**Object(Replica-Link)**](s-object-replica-link.md) |
