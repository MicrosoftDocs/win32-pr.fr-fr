# <a name="role-occupant-attribute"></a>Attribut Role-Occupant

Nom unique d’un objet qui respecte un rôle organisationnel.



| Entrée | Valeur |
|-------------------|-----------------------------------------|
| CN                | Role-Occupant                           |
| LDAP-Display-Name | roleOccupant                            |
| Taille              | \-                                      |
| Mettre à jour le privilège  | \-                                      |
| Fréquence des mises à jour  | \-                                      |
| Attribute-Id      | 2.5.4.33                                |
| System-ID-GUID    | a8df7465-c5ea-11d1-bbcb-0080c76670c0    |
| Syntaxe            | [**Object(DS-DN)**](s-object-ds-dn.md) |
