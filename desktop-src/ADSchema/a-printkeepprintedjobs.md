# <a name="print-keep-printed-jobs-attribute"></a>Attribut imprimer-conserver-imprimé-tâches

**True** si les travaux imprimés sont conservés.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | Imprimer-conserver-imprimé-tâches              |
| LDAP-Display-Name | printKeepPrintedJobs                 |
| Taille              | 4 octets                              |
| Mettre à jour le privilège  | \-                                   |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.275               |
| System-ID-GUID    | ba305f6d-47e3-11d0-a1a6-00c04fd930c9 |
| Syntaxe            | [**Boolean**](s-boolean.md)         |
