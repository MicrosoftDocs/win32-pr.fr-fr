# <a name="ms-com-userlink-attribute"></a>attribut ms-COM-UserLink

Lien utilisé pour associer un PartitionSet COM+ à un objet utilisateur.



| Entrée | Valeur |
|-------------------|-----------------------------------------|
| CN                | MS-COM-UserLink                         |
| LDAP-Display-Name | msCOM-UserLink                          |
| Taille              | \-                                      |
| Mettre à jour le privilège  | Cette valeur est définie par le système.        |
| Fréquence des mises à jour  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1425                 |
| System-ID-GUID    | 9e6f3a4d-242c-4f37-b068-36b57f9fc852    |
| Syntaxe            | [**Object(DS-DN)**](s-object-ds-dn.md) |
