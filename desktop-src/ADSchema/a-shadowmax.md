# <a name="shadowmax-attribute"></a>attribut shadowMax

Contient le nombre maximal de jours pendant lesquels un mot de passe peut être valide.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | shadowMax                            |
| LDAP-Display-Name | shadowMax                            |
| Taille              | \-                                   |
| Mettre à jour le privilège  | \-                                   |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.7                      |
| System-ID-GUID    | f285c952-50dd-449e-9160-3b880d99988d |
| Syntaxe            | [**Énumération**](s-enumeration.md) |
