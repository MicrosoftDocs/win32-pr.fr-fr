# <a name="msmq-sites-attribute"></a>Attribut MSMQ-Sites

Liste des sites auxquels l’ordinateur appartient (un tableau des ObjectGUID de sites, généralement pas plus d’un GUID).



| Entrée | Valeur |
|-------------------|-------------------------------------------------------|
| CN                | MSMQ-Sites                                            |
| LDAP-Display-Name | mSMQSites                                             |
| Taille              | \-                                                    |
| Mettre à jour le privilège  | \-                                                    |
| Fréquence des mises à jour  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.927                                |
| System-ID-GUID    | 9a0dc32a-c100-11d1-bbc5-0080c76670c0                  |
| Syntaxe            | [**Object(Replica-Link)**](s-object-replica-link.md) |
