# <a name="ms-ds-revealed-list-attribute"></a>attribut ms-DS-Revealed-List

Identifie les principaux de sécurité dont les mots de passe de compte d’ordinateur actuel ont été répliqués sur le RODC.



| Entrée | Valeur |
|-------------------|-------------------------------------------------------|
| CN                | ms-DS-dévoilé-liste                                   |
| LDAP-Display-Name | msDS-RevealedList                                     |
| Taille              | \-                                                    |
| Mettre à jour le privilège  | \-                                                    |
| Fréquence des mises à jour  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.1940                               |
| System-ID-GUID    | cbdad11c-7fec-387b-6219-3a0627d9af81                  |
| Syntaxe            | [**Object(Access-Point)**](s-object-access-point.md) |
