# <a name="ms-wmi-id-attribute"></a>attribut ms-WMI-ID

|-------------------|---------------------------------------------|
| CN                | MS-WMI-ID                                   |
| LDAP-Display-Name | ID d’msWMI                                    |
| Taille              | \-                                          |
| Mettre à jour le privilège  | Cette valeur est définie par le système.            |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1627                     |
| System-ID-GUID    | 9339a803-94b8-47f7-9123-a853b9ff7e45        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
