# <a name="phone-mobile-primary-attribute"></a>Téléphone-mobile-attribut principal

Numéro de téléphone mobile principal.



| Entrée | Valeur |
|-------------------|----------------------------------------------------------------------------------|
| CN                | Téléphone-mobile-principal                                                             |
| LDAP-Display-Name | mobile                                                                           |
| Taille              | \-                                                                               |
| Mettre à jour le privilège  | Administrateur de domaine ou propriétaire du compte.                                           |
| Fréquence des mises à jour  | Lorsque l’enregistrement de l’utilisateur est créé et chaque fois que le numéro de téléphone doit être modifié. |
| Attribute-Id      | 0.9.2342.19200300.100.1.41                                                       |
| System-ID-GUID    | f0f8ffa3-1191-11d0-a060-00aa006c33ed                                             |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md)                                      |
