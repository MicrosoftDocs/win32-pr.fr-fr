# <a name="shell-property-pages-attribute"></a>Shell-Property-pages (attribut)

Le numéro d’ordre et le GUID des pages de propriétés pour la gestion des objets Active Directory. Ces pages de propriétés sont accessibles à partir du shell Windows. Pour plus d’informations, consultez le document, extension de l’interface utilisateur pour les objets d’annuaire.



| Entrée | Valeur |
|-------------------|------------------------------------------------|
| CN                | Shell-Property-pages                           |
| LDAP-Display-Name | shellPropertyPages                             |
| Taille              | \-                                             |
| Mettre à jour le privilège  | Administrateur de domaine ou développeur d’applications. |
| Fréquence des mises à jour  | Chaque fois qu’une page de propriétés est ajoutée ou supprimée.  |
| Attribute-Id      | 1.2.840.113556.1.4.563                         |
| System-ID-GUID    | 52458039-ca6a-11D0-AFFF-0000f80367c1           |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md)    |
