# <a name="server-state-attribute"></a>Attribut Server-State

Indique si le serveur est activé ou désactivé. La valeur 1 indique que le serveur est activé. La valeur 2 indique que le serveur est désactivé. Toutes les autres valeurs ne sont pas valides.



| Entrée | Value |
|-------------------|--------------------------------------|
| CN                | Server-State                         |
| LDAP-Display-Name | serverState                          |
| Taille              | \-                                   |
| Mettre à jour le privilège  | Cette valeur est définie par le système.     |
| Fréquence des mises à jour  | Lorsque la stratégie d’un utilisateur change.  |
| Attribute-Id      | 1.2.840.113556.1.4.154               |
| System-ID-GUID    | bf967a34-0de6-11d0-a285-00aa003049e2 |
| Syntaxe            | [**Énumération**](s-enumeration.md) |
