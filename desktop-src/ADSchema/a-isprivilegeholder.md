# <a name="is-privilege-holder-attribute"></a>Est un attribut de détenteur de privilèges

Lien vers l’arrière vers les privilèges détenus par un principal donné.



| Entrée | Valeur |
|-------------------|-----------------------------------------|
| CN                | Est-titulaire de privilège                     |
| LDAP-Display-Name | isPrivilegeHolder                       |
| Taille              | \-                                      |
| Mettre à jour le privilège  | \-                                      |
| Fréquence des mises à jour  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.638                  |
| System-ID-GUID    | 19405b9c-3cfa-11d1-a9c0-0000f80367c1    |
| Syntaxe            | [**Object(DS-DN)**](s-object-ds-dn.md) |
