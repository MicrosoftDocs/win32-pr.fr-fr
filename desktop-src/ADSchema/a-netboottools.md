# <a name="netboot-tools-attribute"></a>attribut netboot-Tools

L’attribut **netboot-Tools** est réservé à un usage interne.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | netboot-outils                               |
| LDAP-Display-Name | netbootTools                                |
| Taille              | \-                                          |
| Mettre à jour le privilège  | Cette valeur est définie par le système.            |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.858                      |
| System-ID-GUID    | 0738307f-91df-11d1-aebc-0000f80367c1        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
