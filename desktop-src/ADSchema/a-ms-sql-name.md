# <a name="ms-sql-name-attribute"></a>MS-SQL-attribut Name

Nom d’une instance de SQL Server. La valeur par défaut est MSSQLSERVER.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | MS-SQL-nom                                 |
| LDAP-Display-Name | mS-SQL-nom                                 |
| Taille              | \-                                          |
| Mettre à jour le privilège  | Cette valeur est définie par le système.            |
| Fréquence des mises à jour  | Dans le programme d’installation du système.                            |
| Attribute-Id      | 1.2.840.113556.1.4.1363                     |
| System-ID-GUID    | 3532dfd8-ccee-11d2-9993-0000f87a57d4        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
