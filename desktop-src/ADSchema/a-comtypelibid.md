# <a name="com-typelib-id-attribute"></a>Attribut COM-TypeLib-ID

Cet attribut stocke la liste des ID de bibliothèque de types contenus dans ce package d’application.



| Entrée | Valeur |
|-------------------|----------------------------------------------------------------------------------|
| CN                | COM-TypeLib-ID                                                                   |
| LDAP-Display-Name | cOMTypelibId                                                                     |
| Taille              | \-                                                                               |
| Mettre à jour le privilège  | Tout le monde peut mettre à jour cet objet en fonction de la sécurité de l’objet en cours de création. |
| Fréquence des mises à jour  | \-                                                                               |
| Attribute-Id      | 1.2.840.113556.1.4.254                                                           |
| System-ID-GUID    | 281416de-1968-11d0-a28f-00aa003049e2                                             |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md)                                      |
