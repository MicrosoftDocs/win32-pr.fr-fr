# <a name="state-or-province-name-attribute"></a>Attribut State-or-province-Name

Nom de l’État ou de la province d’un utilisateur.



| Entrée | Valeur |
|-------------------|----------------------------------------------------------------------------------|
| CN                | Nom de l’État ou de la province                                                           |
| LDAP-Display-Name | st                                                                               |
| Taille              | \-                                                                               |
| Mettre à jour le privilège  | Tout le monde peut mettre à jour cet objet en fonction de la sécurité de l’objet en cours de création. |
| Fréquence des mises à jour  | \-                                                                               |
| Attribute-Id      | 2.5.4.8                                                                          |
| System-ID-GUID    | bf967a39-0de6-11d0-a285-00aa003049e2                                             |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md)                                      |
