# <a name="ms-ieee-80211-id-attribute"></a>attribut ms-IEEE-80211-ID

Identificateur utilisé pour un objet stratégie sans fil sur Active Directory.



| Entrée | Valeur |
|-------------------|----------------------------------------------------------------------------------|
| CN                | MS-IEEE-80211-ID                                                                 |
| LDAP-Display-Name | ID d’msieee80211                                                                   |
| Taille              | \-                                                                               |
| Mettre à jour le privilège  | Administrateur de domaine                                                             |
| Fréquence des mises à jour  | Chaque fois qu’un administrateur de domaine modifie la stratégie de réseau sans fil pour un domaine ou une unité d’organisation. |
| Attribute-Id      | 1.2.840.113556.1.4.1823                                                          |
| System-ID-GUID    | 7f73ef75-14c9-4c23-81de-dd07a06f9e8b                                             |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md)                                      |
