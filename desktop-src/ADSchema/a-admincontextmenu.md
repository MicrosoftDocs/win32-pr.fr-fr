# <a name="admin-context-menu-attribute"></a>Attribut de menu de contexte d’administration

Le numéro d’ordre et le GUID du menu contextuel à utiliser sur les écrans d’administration.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | Menu contextuel d’administration                          |
| LDAP-Display-Name | adminContextMenu                            |
| Taille              | \-                                          |
| Mettre à jour le privilège  | Administrateur de domaine                        |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.614                      |
| System-ID-GUID    | 553fd038-f32e-11d0-b0bc-00c04fd8dca6        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
