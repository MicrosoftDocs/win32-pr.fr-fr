# <a name="link-id-attribute"></a>Attribut de l’ID de lien

Entier qui indique que l’attribut est un attribut lié. Un entier pair est un lien suivant et un entier impair est un lien vers l’arrière.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | ID de lien                              |
| LDAP-Display-Name | Égale                               |
| Taille              | \-                                   |
| Mettre à jour le privilège  | Administrateur de schéma                 |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.2.50                |
| System-ID-GUID    | bf96799b-0de6-11d0-a285-00aa003049e2 |
| Syntaxe            | [**Énumération**](s-enumeration.md) |
