# <a name="print-end-time-attribute"></a>Attribut d’heure de fin d’impression

Heure à laquelle une file d’attente à l’impression arrête les travaux de maintenance.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | Heure de fin d’impression                       |
| LDAP-Display-Name | printEndTime                         |
| Taille              | \-                                   |
| Mettre à jour le privilège  | \-                                   |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.234               |
| System-ID-GUID    | 281416ca-1968-11d0-a28f-00aa003049e2 |
| Syntaxe            | [**Enumeration**](s-enumeration.md) |
