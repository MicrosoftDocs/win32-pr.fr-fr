# <a name="application-name-attribute"></a>Attribut Application-Name

Le nom de l’application.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | Nom-de-l’application                            |
| LDAP-Display-Name | applicationName                             |
| Taille              | \-                                          |
| Mettre à jour le privilège  | Administrateur de domaine                        |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.218                      |
| System-ID-GUID    | dd712226-10e4-11d0-a05f-00aa006c33ed        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
