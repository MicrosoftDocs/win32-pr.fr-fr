# <a name="pkt-attribute"></a>Attribut PKT

Table de connaissance de la partition DFS. Décrit la structure d’une hiérarchie de système de fichiers DFS.



| Entrée | Valeur |
|-------------------|-------------------------------------------------------|
| CN                | PKT                                                   |
| LDAP-Display-Name | pKT                                                   |
| Taille              | \-                                                    |
| Mettre à jour le privilège  | \-                                                    |
| Fréquence des mises à jour  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.206                                |
| System-ID-GUID    | 8447f9f1-1027-11d0-a05f-00aa006c33ed                  |
| Syntaxe            | [**Object(Replica-Link)**](s-object-replica-link.md) |
