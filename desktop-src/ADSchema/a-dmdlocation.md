# <a name="dmd-location-attribute"></a>Attribut DMD-Location

Nom unique de la partition de schéma.



| Entrée | Valeur |
|-------------------|-----------------------------------------|
| CN                | DMD-Location                            |
| LDAP-Display-Name | dMDLocation                             |
| Taille              | \-                                      |
| Mettre à jour le privilège  | Cette valeur est définie par le système.        |
| Fréquence des mises à jour  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.2.36                   |
| System-ID-GUID    | f0f8ff8b-1191-11d0-a060-00aa006c33ed    |
| Syntaxe            | [**Object(DS-DN)**](s-object-ds-dn.md) |
