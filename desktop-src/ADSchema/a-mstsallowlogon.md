# <a name="ms-ts-allow-logon-attribute"></a>attribut ms-TS-allow-Logon

Spécifie si l’utilisateur est autorisé à se connecter au Terminal Server. La valeur est 1 si l’ouverture de session est autorisée, et 0 si l’ouverture de session n’est pas autorisée.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | MS-TS-autoriser-ouvrir une session                    |
| LDAP-Display-Name | msTSAllowLogon                       |
| Taille              | \-                                   |
| Mettre à jour le privilège  | \-                                   |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1979              |
| System-ID-GUID    | 3a0cd464-bc54-40e7-93ae-a646a6ecc4b4 |
| Syntaxe            | [**Boolean**](s-boolean.md)         |
