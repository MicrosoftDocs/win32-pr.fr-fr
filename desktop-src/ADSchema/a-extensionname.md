# <a name="extension-name-attribute"></a>Attribut Extension-Name

Nom d’une page de propriétés utilisée pour étendre l’interface utilisateur d’un objet d’annuaire.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | Extension-Name                              |
| LDAP-Display-Name | extensionName                               |
| Taille              | \-                                          |
| Mettre à jour le privilège  | Créateur de l’objet d’extension.        |
| Fréquence des mises à jour  | Chaque fois qu’une nouvelle extension d’interface utilisateur est créée.     |
| Attribute-Id      | 1.2.840.113556.1.2.227                      |
| System-ID-GUID    | bf967972-0de6-11d0-a285-00aa003049e2        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
