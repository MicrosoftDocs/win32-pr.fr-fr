# <a name="ms-ds-machine-account-quota-attribute"></a>Attribut de quota MS-DS-machine-Account

Nombre de comptes d’ordinateur qu’un utilisateur est autorisé à créer dans un domaine.



| Entrée | Valeur |
|-------------------|--------------------------------------------------|
| CN                | MS-DS-machine-Account-quota                      |
| LDAP-Display-Name | ms-DS-MachineAccountQuota                        |
| Taille              | 4 octets                                          |
| Mettre à jour le privilège  | Administrateur de domaine                             |
| Fréquence des mises à jour  | Chaque fois que le quota d’un domaine doit être modifié. |
| Attribute-Id      | 1.2.840.113556.1.4.1411                          |
| System-ID-GUID    | d064fb68-1480-11d3-91c1-0000f87a57d4             |
| Syntaxe            | [**Énumération**](s-enumeration.md)             |
