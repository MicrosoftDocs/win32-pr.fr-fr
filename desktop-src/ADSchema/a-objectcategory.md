# <a name="object-category-attribute"></a>Attribut Object-Category

Nom de classe d’objet utilisé pour regrouper des objets de cette classe ou des classes dérivées.



| Entrée | Valeur |
|-------------------|--------------------------------------------------|
| CN                | Object-Category                                  |
| LDAP-Display-Name | objectCategory                                   |
| Taille              | Environ 20 octets en moyenne.                       |
| Mettre à jour le privilège  | Le concepteur de l’objet définit cette valeur. |
| Fréquence des mises à jour  | Cette valeur ne doit jamais changer.                  |
| Attribute-Id      | 1.2.840.113556.1.4.782                           |
| System-ID-GUID    | 26d97369-6070-11d1-a9c6-0000f80367c1             |
| Syntaxe            | [**Object(DS-DN)**](s-object-ds-dn.md)          |
