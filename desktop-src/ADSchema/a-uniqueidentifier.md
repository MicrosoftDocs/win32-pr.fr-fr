# <a name="uniqueidentifier-attribute"></a>attribut uniqueIdentifier

Le type d’attribut uniqueIdentifier spécifie un « identificateur unique » pour un objet représenté dans le répertoire.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | uniqueIdentifier                            |
| LDAP-Display-Name | uniqueIdentifier                            |
| Taille              | \-                                          |
| Mettre à jour le privilège  | \-                                          |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 0.9.2342.19200300.100.1.44                  |
| System-ID-GUID    | ba0184c7-38c5-4bed-a526-75421470580c        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
