# <a name="phone-office-other-attribute"></a>Téléphone-Office-autre attribut

Liste de numéros de téléphone de bureau de substitution.



| Entrée | Valeur |
|-------------------|---------------------------------------------------------------------------------|
| CN                | Téléphone-Office-autre                                                              |
| LDAP-Display-Name | otherTelephone                                                                  |
| Taille              | \-                                                                              |
| Mettre à jour le privilège  | Administrateur de domaine ou propriétaire du compte.                                          |
| Fréquence des mises à jour  | Lorsque l’enregistrement des utilisateurs est créé et chaque fois que le numéro de téléphone doit être modifié. |
| Attribute-Id      | 1.2.840.113556.1.2.18                                                           |
| System-ID-GUID    | f0f8ffa5-1191-11d0-a060-00aa006c33ed                                            |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md)                                     |
