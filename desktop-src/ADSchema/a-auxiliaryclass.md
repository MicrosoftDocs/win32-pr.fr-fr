# <a name="auxiliary-class-attribute"></a>Attribut Auxiliary-Class

Liste des classes auxiliaires à associer à cette classe.



| Entrée | Valeur |
|-------------------|-----------------------------------------------------------------|
| CN                | Auxiliary-Class                                                 |
| LDAP-Display-Name | auxiliaryClass                                                  |
| Taille              | \-                                                              |
| Mettre à jour le privilège  | Administrateur de schéma                                            |
| Fréquence des mises à jour  | \-                                                              |
| Attribute-Id      | 1.2.840.113556.1.2.351                                          |
| System-ID-GUID    | bf96792c-0de6-11d0-a285-00aa003049e2                            |
| Syntaxe            | [**String(Object-Identifier)**](s-string-object-identifier.md) |
