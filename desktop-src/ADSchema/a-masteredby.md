# <a name="mastered-by-attribute"></a>Attribut Mastered-By

Lien vers l’arrière de l’attribut a-Master-NC. nom unique de ses objets Paramètres NTDS.



| Entrée | Valeur |
|-------------------|-----------------------------------------|
| CN                | Mastered-By                             |
| LDAP-Display-Name | masteredBy                              |
| Taille              | \-                                      |
| Mettre à jour le privilège  | Cette valeur est définie par le système.        |
| Fréquence des mises à jour  | Lorsque le contexte d’appellation est créé.     |
| Attribute-Id      | 1.2.840.113556.1.4.1409                 |
| System-ID-GUID    | e48e64e0-12c9-11d3-9102-00c04fd91ab1    |
| Syntaxe            | [**Object(DS-DN)**](s-object-ds-dn.md) |
