# <a name="documentauthor-attribute"></a>attribut documentAuthor

Le type d’attribut documentAuthor spécifie le nom unique de l’auteur d’un document.



| Entrée | Valeur |
|-------------------|-----------------------------------------|
| CN                | documentAuthor                          |
| LDAP-Display-Name | documentAuthor                          |
| Taille              | \-                                      |
| Mettre à jour le privilège  | \-                                      |
| Fréquence des mises à jour  | \-                                      |
| Attribute-Id      | 0.9.2342.19200300.100.1.14              |
| System-ID-GUID    | f18a8e19-AF5F-4478-b096-6f35c27eb83f    |
| Syntaxe            | [**Object(DS-DN)**](s-object-ds-dn.md) |
