# <a name="site-server-attribute"></a>Attribut Site-Server

Licence du serveur principal pour un site donné.



| Entrée | Valeur |
|-------------------|----------------------------------------------|
| CN                | Site-Server                                  |
| LDAP-Display-Name | siteServer                                   |
| Taille              | \-                                           |
| Mettre à jour le privilège  | Administrateur de domaine                         |
| Fréquence des mises à jour  | Chaque fois que le site de gestion des licences doit être modifié. |
| Attribute-Id      | 1.2.840.113556.1.4.494                       |
| System-ID-GUID    | 1be8f17c-a9ff-11d0-afe2-00c04fd930c9         |
| Syntaxe            | [**Object(DS-DN)**](s-object-ds-dn.md)      |
