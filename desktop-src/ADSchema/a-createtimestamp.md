# <a name="create-time-stamp-attribute"></a>Attribut d’horodatage de création

Date à laquelle cet objet a été créé. Cette valeur est répliquée.



| Entrée | Valeur |
|-------------------|---------------------------------------------------------------|
| CN                | Date et heure de création                                             |
| LDAP-Display-Name | createTimeStamp                                               |
| Taille              | 8 octets                                                       |
| Mettre à jour le privilège  | Cette valeur est définie par le système.                              |
| Fréquence des mises à jour  | Lorsque l’objet est créé.                                   |
| Attribute-Id      | 2.5.18.1                                                      |
| System-ID-GUID    | 2df90d73-009f-11d2-aa4c-00c04fd7d83a                          |
| Syntaxe            | [**String(Generalized-Time)**](s-string-generalized-time.md) |
