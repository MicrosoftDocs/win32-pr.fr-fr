# <a name="service-class-name-attribute"></a>Attribut de nom de classe de service

Nom de chaîne du service représenté par un point d’administration.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | Nom-classe-service                          |
| LDAP-Display-Name | serviceClassName                            |
| Taille              | \-                                          |
| Mettre à jour le privilège  | \-                                          |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.509                      |
| System-ID-GUID    | b7b1311d-b82e-11d0-afee-0000f80367c1        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
