# <a name="driver-version-attribute"></a>Attribut Driver-Version

Numéro de version du pilote de périphérique.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | Driver-Version                       |
| LDAP-Display-Name | driverVersion                        |
| Taille              | 4 octets                              |
| Mettre à jour le privilège  | \-                                   |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.276               |
| System-ID-GUID    | ba305f6e-47e3-11d0-a1a6-00c04fd930c9 |
| Syntaxe            | [**Enumeration**](s-enumeration.md) |
