# <a name="recalculate-security-inheritance-extended-right"></a>Recalculer-sécurité-héritage-droit étendu

Droit étendu nécessaire pour forcer DS à recalculer l’héritage d’une liste de contrôle d’accès dans un contexte d’appellation.



| Entrée | Valeur |
|--------------|--------------------------------------|
| CN           | Recalculer-sécurité-héritage     |
| Display-Name | Recalculer l’héritage de sécurité     |
| Rights-GUID  | 62dd28a8-7f46-11d2-b9ad-00c04f79f805 |
