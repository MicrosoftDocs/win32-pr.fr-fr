# <a name="dhcp-servers-attribute"></a>attribut DHCP-Servers

Contient la liste des serveurs autorisés dans l’entreprise.



| Entrée | Valeur |
|-------------------|--------------------------------------------------------|
| CN                | Serveurs DHCP                                           |
| LDAP-Display-Name | dhcpServers                                            |
| Taille              | \-                                                     |
| Mettre à jour le privilège  | Administrateur de domaine.                                  |
| Fréquence des mises à jour  | Lorsqu’un nouveau serveur est ajouté ou supprimé de la forêt. |
| Attribute-Id      | 1.2.840.113556.1.4.704                                 |
| System-ID-GUID    | 963d2745-48be-11d1-a9c3-0000f80367c1                   |
| Syntaxe            | [**String(IA5)**](s-string-ia5.md)                    |
