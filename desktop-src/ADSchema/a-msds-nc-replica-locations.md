# <a name="ms-ds-nc-replica-locations-attribute"></a>attribut ms-DS-NC-Replica-Locations

Liste des serveurs qui sont le jeu de réplicas pour le contexte d’appellation non-domaine correspondant.



| Entrée | Valeur |
|-------------------|-----------------------------------------|
| CN                | ms-DS-NC-réplica-emplacements              |
| LDAP-Display-Name | msDS-NC-Replica-Locations               |
| Taille              | \-                                      |
| Mettre à jour le privilège  | \-                                      |
| Fréquence des mises à jour  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1661                 |
| System-ID-GUID    | 97de9615-b537-46bc-ac0f-10720f3909f3    |
| Syntaxe            | [**Object(DS-DN)**](s-object-ds-dn.md) |
