# <a name="ms-ds-external-key-attribute"></a>ms-DS-External-Key (attribut)

Chaîne qui identifie un objet dans un magasin externe, tel qu’un enregistrement dans une base de données.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | ms-DS-externe-clé                          |
| LDAP-Display-Name | msDS-ExternalKey                            |
| Taille              | \-                                          |
| Mettre à jour le privilège  | \-                                          |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1833                     |
| System-ID-GUID    | b92fd528-38ac-40d4-818d-0433380837c1        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
