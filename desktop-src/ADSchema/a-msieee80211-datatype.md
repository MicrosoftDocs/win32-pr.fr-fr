# <a name="ms-ieee-80211-data-type-attribute"></a>MS-IEEE-80211-attribut de type de données

Type de données utilisé en interne pour l’objet BLOB msieee80211-Data.



| Entrée | Valeur |
|-------------------|----------------------------------------------------------------------------------|
| CN                | MS-IEEE-80211-type de données                                                          |
| LDAP-Display-Name | msieee80211-DataType                                                             |
| Taille              | \-                                                                               |
| Mettre à jour le privilège  | Administrateur de domaine                                                             |
| Fréquence des mises à jour  | Chaque fois qu’un administrateur de domaine modifie la stratégie de réseau sans fil pour un domaine ou une unité d’organisation. |
| Attribute-Id      | 1.2.840.113556.1.4.1822                                                          |
| System-ID-GUID    | 6558b180-35da-4efe-beed-521f8f48cafb                                             |
| Syntaxe            | [**Enumeration**](s-enumeration.md)                                             |
