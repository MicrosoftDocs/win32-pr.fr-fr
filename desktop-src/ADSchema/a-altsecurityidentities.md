# <a name="alt-security-identities-attribute"></a>Attribut alt-Security-Identities

Contient des mappages pour les certificats X. 509 ou des comptes d’utilisateur Kerberos externes à cet utilisateur à des fins d’authentification.



| Entrée | Valeur |
|-------------------|-----------------------------------------------------|
| CN                | Alt-sécurité-identités                             |
| LDAP-Display-Name | altSecurityIdentities                               |
| Taille              | \-                                                  |
| Mettre à jour le privilège  | Administrateur de domaine                                |
| Fréquence des mises à jour  | Chaque fois qu’un nouveau mécanisme d’authentification est nécessaire. |
| Attribute-Id      | 1.2.840.113556.1.4.867                              |
| System-ID-GUID    | 00fbf30c-91fe-11d1-aebc-0000f80367c1                |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md)         |
