# <a name="non-security-member-attribute"></a>Attribut de membre de non-sécurité

Membres non-sécurité d’un groupe. utilisé pour les listes de distribution Exchange.



| Entrée | Valeur |
|-------------------|--------------------------------------------------|
| CN                | Membre non de sécurité                              |
| LDAP-Display-Name | nonSecurityMember                                |
| Taille              | \-                                               |
| Mettre à jour le privilège  | Administrateur de domaine                             |
| Fréquence des mises à jour  | Chaque fois qu’un utilisateur est ajouté ou supprimé de la LD. |
| Attribute-Id      | 1.2.840.113556.1.4.530                           |
| System-ID-GUID    | 52458018-ca6a-11D0-AFFF-0000f80367c1             |
| Syntaxe            | [**Object(DS-DN)**](s-object-ds-dn.md)          |
