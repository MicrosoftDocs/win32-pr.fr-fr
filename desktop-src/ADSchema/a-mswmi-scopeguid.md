# <a name="ms-wmi-scopeguid-attribute"></a>attribut ms-WMI-ScopeGuid

GUID de l’étendue dans laquelle se trouve l’encodage associé.



| Entrée | Valeur |
|-------------------|-----------------------------------------------------------------|
| CN                | MS-WMI-ScopeGuid                                                |
| LDAP-Display-Name | msWMI-ScopeGuid                                                 |
| Taille              | 16 octets                                                        |
| Mettre à jour le privilège  | Administrateur de domaine                                            |
| Fréquence des mises à jour  | Chaque fois qu’une classe contenant l’attribut est ajoutée ou modifiée. |
| Attribute-Id      | 1.2.840.113556.1.4.1686                                         |
| System-ID-GUID    | 87b78d51-405f-4b7f-80ed-2bd28786f48d                            |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md)                     |
