# <a name="acs-max-size-of-rsvp-log-file-attribute"></a>ACS-Max-size-of-RSVP-attribut de fichier journal

Taille maximale, en octets, d’un fichier journal RSVP.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | ACS-Max-size-of-RSVP-log-file        |
| LDAP-Display-Name | aCSMaxSizeOfRSVPLogFile              |
| Taille              | 4 octets                              |
| Mettre à jour le privilège  | \-                                   |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.775               |
| System-ID-GUID    | 1cb3559d-56d0-11d1-a9c6-0000f80367c1 |
| Syntaxe            | [**Enumeration**](s-enumeration.md) |
