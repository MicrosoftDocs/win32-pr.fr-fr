# <a name="omt-indx-guid-attribute"></a>Attribut OMT-indx-GUID

Identificateur d’index pour une entrée de table de lien-Track-objet-Move.



| Entrée | Valeur |
|-------------------|-------------------------------------------------------|
| CN                | OMT-indx-GUID                                         |
| LDAP-Display-Name | oMTIndxGuid                                           |
| Taille              | 16 octets                                              |
| Mettre à jour le privilège  | Cette valeur est définie par le système.                      |
| Fréquence des mises à jour  | Chaque fois que le lien d’un fichier est modifié.                 |
| Attribute-Id      | 1.2.840.113556.1.4.333                                |
| System-ID-GUID    | 1f0075fa-7e40-11d0-afd6-00c04fd930c9                  |
| Syntaxe            | [**Object(Replica-Link)**](s-object-replica-link.md) |
