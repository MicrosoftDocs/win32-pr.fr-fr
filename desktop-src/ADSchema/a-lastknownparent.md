# <a name="last-known-parent-attribute"></a>Dernier attribut parent connu

Nom unique (DN) du dernier parent connu d’un objet orphelin.



| Entrée | Valeur |
|-------------------|-----------------------------------------|
| CN                | Dernier-parent connu                       |
| LDAP-Display-Name | lastKnownParent                         |
| Taille              | \-                                      |
| Mettre à jour le privilège  | Cette valeur est définie par le système.        |
| Fréquence des mises à jour  | Chaque fois qu’un objet est orphelin.         |
| Attribute-Id      | 1.2.840.113556.1.4.781                  |
| System-ID-GUID    | 52ab8670-5709-11d1-a9c6-0000f80367c1    |
| Syntaxe            | [**Object(DS-DN)**](s-object-ds-dn.md) |
