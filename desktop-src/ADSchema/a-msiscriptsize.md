# <a name="msi-script-size-attribute"></a>Attribut MSI-script-Size

Taille de fichier du fichier de script de publication Microsoft installer pour cette application.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | MSI-script-taille                      |
| LDAP-Display-Name | msiScriptSize                        |
| Taille              | \-                                   |
| Mettre à jour le privilège  | \-                                   |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.846               |
| System-ID-GUID    | 96a7dd63-9118-11d1-aebc-0000f80367c1 |
| Syntaxe            | [**Énumération**](s-enumeration.md) |
