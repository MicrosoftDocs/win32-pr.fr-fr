# <a name="ms-sql-lastupdateddate-attribute"></a>attribut MS-SQL-LastUpdatedDate

cette valeur représente l’heure de la dernière mise à jour de SQL Active Directory.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | MS-SQL-LastUpdatedDate                      |
| LDAP-Display-Name | mS-SQL-LastUpdatedDate                      |
| Taille              | \-                                          |
| Mettre à jour le privilège  | Cette valeur est définie par le système.            |
| Fréquence des mises à jour  | Au démarrage du système ou lors de la mise à jour d’AD.    |
| Attribute-Id      | 1.2.840.113556.1.4.1381                     |
| System-ID-GUID    | 9fcc43d4-ccee-11d2-9993-0000f87a57d4        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
