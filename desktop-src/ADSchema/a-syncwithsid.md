# <a name="sync-with-sid-attribute"></a>Attribut sync-with-SID

Pour la synchronisation de la stratégie locale/objet du groupe builtin SAM, il s’agit du groupe local auquel correspond un objet.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | Synchronisation avec SID                        |
| LDAP-Display-Name | syncWithSID                          |
| Taille              | \-                                   |
| Mettre à jour le privilège  | \-                                   |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.667               |
| System-ID-GUID    | 037651e5-441d-11d1-a9c3-0000f80367c1 |
| Syntaxe            | [**Chaîne (SID)**](s-string-sid.md)  |
