# <a name="employee-id-attribute"></a>Attribut Employee-ID

ID d’un employé.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | ID d’employé                                 |
| LDAP-Display-Name | employeeID                                  |
| Taille              | \-                                          |
| Mettre à jour le privilège  | Administrateur de domaine                        |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.35                       |
| System-ID-GUID    | bf967962-0de6-11d0-a285-00aa003049e2        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
