# <a name="ms-ds-members-for-az-role-attribute"></a>ms-DS-Members-for-AZ-Role, attribut

Liste des groupes d’applications membres ou des utilisateurs liés à AZ-Role.



| Entrée | Valeur |
|-------------------|-----------------------------------------|
| CN                | ms-DS-Members-for-AZ-Role               |
| LDAP-Display-Name | msDS-MembersForAzRole                   |
| Taille              | \-                                      |
| Mettre à jour le privilège  | Administrateur AzRoles                           |
| Fréquence des mises à jour  | Pendant l’initialisation ou la modification de la stratégie. |
| Attribute-Id      | 1.2.840.113556.1.4.1806                 |
| System-ID-GUID    | cbf7e6cd-85a4-4314-8939-8bfe80597835    |
| Syntaxe            | [**Object(DS-DN)**](s-object-ds-dn.md) |
