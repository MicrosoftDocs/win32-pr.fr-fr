# <a name="ms-wmi-class-attribute"></a>MS-WMI-attribut de classe

Nom d’un objet de classe WMI dans un encodage associé (par exemple, Win32 \_ ComputerSystem). L’encodage peut être une instance ou un objet de classe.



| Entrée | Valeur |
|-------------------|-----------------------------------------------------------------|
| CN                | MS-WMI-classe                                                    |
| LDAP-Display-Name | msWMI-classe                                                     |
| Taille              | \-                                                              |
| Mettre à jour le privilège  | Administrateur de domaine                                            |
| Fréquence des mises à jour  | Chaque fois qu’une classe contenant l’attribut est ajoutée ou modifiée. |
| Attribute-Id      | 1.2.840.113556.1.4.1676                                         |
| System-ID-GUID    | 90c1925f-4a24-4b07-b202-be32eb3c8b74                            |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md)                     |
