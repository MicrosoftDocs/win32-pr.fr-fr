# <a name="ms-ds-dnsrootalias-attribute"></a>attribut ms-DS-DnsRootAlias

Utilisé pour stocker l’alias de domaine.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | ms-DS-DnsRootAlias                          |
| LDAP-Display-Name | msDS-DnsRootAlias                           |
| Taille              | Taille maximale de 255 octets.                     |
| Mettre à jour le privilège  | Cette valeur est définie par le système.            |
| Fréquence des mises à jour  | Uniquement pendant la restructuration de domaine.             |
| Attribute-Id      | 1.2.840.113556.1.4.1719                     |
| System-ID-GUID    | 2143acca-eead-4d29-b591-85fa49ce9173        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
