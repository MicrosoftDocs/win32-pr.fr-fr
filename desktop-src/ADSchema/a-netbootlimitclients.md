# <a name="netboot-limit-clients-attribute"></a>attribut netboot-Limit-clients

L’attribut **netboot-Limit-clients** est réservé à un usage interne.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | netboot-Limit-clients                |
| LDAP-Display-Name | netbootLimitClients                  |
| Taille              | \-                                   |
| Mettre à jour le privilège  | Cette valeur est définie par le système.     |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.850               |
| System-ID-GUID    | 07383077-91df-11d1-aebc-0000f80367c1 |
| Syntaxe            | [**Boolean**](s-boolean.md)         |
