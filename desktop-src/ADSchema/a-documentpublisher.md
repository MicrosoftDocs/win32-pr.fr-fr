# <a name="documentpublisher-attribute"></a>attribut documentPublisher

L’attribut documentPublisher est la personne ou l’organisation qui a publié un document.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | documentPublisher                           |
| LDAP-Display-Name | documentPublisher                           |
| Taille              | \-                                          |
| Mettre à jour le privilège  | \-                                          |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 0.9.2342.19200300.100.1.56                  |
| System-ID-GUID    | 170f09d7-eb69-448a-9a30-f1afecfd32d7        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
