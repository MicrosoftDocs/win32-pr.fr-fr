# <a name="msmq-recipient-formatname-attribute"></a>Attribut MSMQ-Recipient-FormatName

Utilisé comme nom de format de destinataire dans la classe MSMQ-Custom-Recipient.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | MSMQ-Recipient-FormatName                   |
| LDAP-Display-Name | msMQ-Recipient-FormatName                   |
| Taille              | 1 à 255 caractères.                        |
| Mettre à jour le privilège  | Propriétaire de la file d’attente.                            |
| Fréquence des mises à jour  | Uniquement après le déplacement de la file d’attente externe.    |
| Attribute-Id      | 1.2.840.113556.1.4.1695                     |
| System-ID-GUID    | 3bfe6748-b544-485a-b067-1b310c4334bf        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
