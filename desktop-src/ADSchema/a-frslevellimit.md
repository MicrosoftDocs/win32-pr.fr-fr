# <a name="frs-level-limit-attribute"></a>Attribut de limite de niveau FRS

Limitez la profondeur de l’arborescence de répertoires à répliquer pour la réplication de fichiers.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | Limite de niveau de service FRS                      |
| LDAP-Display-Name | fRSLevelLimit                        |
| Taille              | 4 octets                              |
| Mettre à jour le privilège  | \-                                   |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.534               |
| System-ID-GUID    | 5245801e-ca6a-11d0-afff-0000f80367c1 |
| Syntaxe            | [**Énumération**](s-enumeration.md) |
