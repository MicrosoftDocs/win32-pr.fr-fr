# <a name="ms-pki-key-recovery-agent-class"></a>classe ms-PKI-Key-Recovery-agent

Une instance d’objet de l’agent de récupération de clé (KRA) est créée pour chaque serveur de certificat installé (avec un nom commun unique) au cours de l’installation du serveur de certificats. Si deux autorités de certification se sont vu attribuer le même nom commun lors de la configuration de l’autorité de certification, elles partageront une seule instance d’objet KRA.



| Entrée | Valeur |
|-------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| CN                | MS-PKI-Key-Recovery-agent                                                                                                                                                    |
| LDAP-Display-Name | Attribut mspki-Key-Recovery-agent                                                                                                                                                     |
| Mettre à jour le privilège  | Un administrateur qui installe une autorité de certification doit être en mesure de créer une instance KRA dans le conteneur KRA. Les serveurs de certificats installés doivent être en mesure de mettre à jour l’attribut userCertificate. |
| Fréquence des mises à jour  | Quelques certificats seront ajoutés au plus tous les quelques mois.                                                                                                                   |
| Schema-ID-GUID    | 26ccf238-a08e-4b86-9a82-a8c9ac7ee5cb                                                                                                                                         |
