# <a name="ms-ds-default-quota-attribute"></a>attribut de quota ms-DS-default

Quota par défaut qui s’applique à un principal de sécurité qui crée un objet dans le contexte de nommage s’il n’existe aucune spécification de quota qui couvre le principal de sécurité.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | ms-DS-default-quota                  |
| LDAP-Display-Name | msDS-DefaultQuota                    |
| Taille              | \-                                   |
| Mettre à jour le privilège  | \-                                   |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1846              |
| System-ID-GUID    | 6818f726-674b-441b-8a3a-f40596374cea |
| Syntaxe            | [**Enumeration**](s-enumeration.md) |
