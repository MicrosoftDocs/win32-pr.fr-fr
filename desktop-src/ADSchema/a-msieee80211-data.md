# <a name="ms-ieee-80211-data-attribute"></a>MS-IEEE-80211-attribut de données

Stocke une liste de configurations réseau préférées dans stratégie de groupe pour les réseaux sans fil.



| Entrée | Valeur |
|-------------------|----------------------------------------------------------------------------------|
| CN                | MS-IEEE-80211-données                                                               |
| LDAP-Display-Name | msieee80211-données                                                                 |
| Taille              | \-                                                                               |
| Mettre à jour le privilège  | Administrateur de domaine                                                             |
| Fréquence des mises à jour  | Chaque fois qu’un administrateur de domaine modifie la stratégie de réseau sans fil pour un domaine ou une unité d’organisation. |
| Attribute-Id      | 1.2.840.113556.1.4.1821                                                          |
| System-ID-GUID    | 0e0d0938-2658-4580-a9f6-7a0ac7b566cb                                             |
| Syntaxe            | [**Object(Replica-Link)**](s-object-replica-link.md)                            |
