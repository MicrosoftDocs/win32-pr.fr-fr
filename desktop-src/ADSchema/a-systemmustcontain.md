# <a name="system-must-contain-attribute"></a>Attribut système-doit contenir

Liste des attributs obligatoires pour une classe. Ces attributs doivent être spécifiés lors de la création d’une instance de la classe. La liste des attributs ne peut être modifiée que par le système.



| Entrée | Valeur |
|-------------------|-------------------------------------------------------------------------------|
| CN                | Système-doit contenir                                                           |
| LDAP-Display-Name | systemMustContain                                                             |
| Taille              | \-                                                                            |
| Mettre à jour le privilège  | Administrateur de schéma                                                          |
| Fréquence des mises à jour  | Lorsque la classe est créée ou qu’un nouvel attribut obligatoire est ajouté à la classe. |
| Attribute-Id      | 1.2.840.113556.1.4.197                                                        |
| System-ID-GUID    | bf967a45-0de6-11d0-a285-00aa003049e2                                          |
| Syntaxe            | [**String(Object-Identifier)**](s-string-object-identifier.md)               |
