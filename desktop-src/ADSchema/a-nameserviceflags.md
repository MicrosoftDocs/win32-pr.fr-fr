# <a name="name-service-flags-attribute"></a>Attribut Name-Service-Flags

Indicateurs de configuration pour le service de noms RPC.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | Nom-service-indicateurs                   |
| LDAP-Display-Name | nameServiceFlags                     |
| Taille              | 4 octets                              |
| Mettre à jour le privilège  | \-                                   |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.753               |
| System-ID-GUID    | 80212840-4bdc-11d1-a9c4-0000f80367c1 |
| Syntaxe            | [**Enumeration**](s-enumeration.md) |
