# <a name="unexpire-password-extended-right"></a>Unexpire-Password le droit étendu

Droit d’accès du contrôle étendu qui permet à un utilisateur de restaurer un mot de passe expiré pour un objet utilisateur.



| Entrée | Valeur |
|--------------|--------------------------------------|
| CN           | Ne pas faire expirer le mot de passe                    |
| Display-Name | Mot de passe d’expiration                    |
| Rights-GUID  | ccc2dc7d-A6AD-4a7a-8846-c04e3cc53501 |
