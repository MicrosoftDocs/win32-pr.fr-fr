# <a name="organization-name-attribute"></a>Attribut Organization-Name

|-------------------|---------------------------------------------|
| CN                | Organization-Name                           |
| LDAP-Display-Name | o                                           |
| Taille              | \-                                          |
| Mettre à jour le privilège  | \-                                          |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 2.5.4.10                                    |
| System-ID-GUID    | bf9679ef-0de6-11d0-a285-00aa003049e2        |
