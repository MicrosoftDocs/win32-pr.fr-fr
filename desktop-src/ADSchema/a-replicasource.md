# <a name="replica-source-attribute"></a>Attribut Replica-Source

Cet attribut contient le GUID d’une source de réplication.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | Replica-Source                              |
| LDAP-Display-Name | replicaSource                               |
| Taille              | \-                                          |
| Mettre à jour le privilège  | \-                                          |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.109                      |
| System-ID-GUID    | bf967a18-0de6-11d0-a285-00aa003049e2        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
