# <a name="ms-ds-claim-is-value-space-restricted-attribute"></a>attribut ms-DS-claim-is-value-Space-Restricted

Pour un type de revendication, cet attribut identifie si un utilisateur peut entrer des valeurs autres que celles décrites dans le msDS-ClaimPossibleValues dans les applications.
