# <a name="site-object-bl-attribute"></a>Site-objet-BL (attribut)

Liste des noms uniques pour les sous-réseaux appartenant à ce site.



| Entrée | Valeur |
|-------------------|-----------------------------------------|
| CN                | Site-objet-BL                          |
| LDAP-Display-Name | siteObjectBL                            |
| Taille              | \-                                      |
| Mettre à jour le privilège  | Cette valeur est définie par le système.        |
| Fréquence des mises à jour  | Chaque fois qu’un site est créé ou supprimé.  |
| Attribute-Id      | 1.2.840.113556.1.4.513                  |
| System-ID-GUID    | 3e10944d-c354-11d0-aff8-0000f80367c1    |
| Syntaxe            | [**Object(DS-DN)**](s-object-ds-dn.md) |
