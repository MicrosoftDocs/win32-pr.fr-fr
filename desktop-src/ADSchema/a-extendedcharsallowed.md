# <a name="extended-chars-allowed-attribute"></a>Extended-chars-attribute allowed

Indique si les caractères étendus sont autorisés dans la valeur de cet attribut. S’applique uniquement aux attributs de chaîne IA5, numérique, imprimable et TELETEX.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | Caractères étendus-autorisés               |
| LDAP-Display-Name | extendedCharsAllowed                 |
| Taille              | \-                                   |
| Mettre à jour le privilège  | \-                                   |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.2.380               |
| System-ID-GUID    | bf967966-0de6-11d0-a285-00aa003049e2 |
| Syntaxe            | [**Boolean**](s-boolean.md)         |
