# <a name="site-link-list-attribute"></a>Attribut site-lien-liste

Liste des liens de sites associés à ce pont.



| Entrée | Valeur |
|-------------------|--------------------------------------------------------------|
| CN                | Site-lien-liste                                               |
| LDAP-Display-Name | siteLinkList                                                 |
| Taille              | \-                                                           |
| Mettre à jour le privilège  | Administrateur de domaine                                         |
| Fréquence des mises à jour  | Chaque fois qu’un lien de site est ajouté ou supprimé du pont. |
| Attribute-Id      | 1.2.840.113556.1.4.822                                       |
| System-ID-GUID    | d50c2cdd-8951-11d1-aebc-0000f80367c1                         |
| Syntaxe            | [**Object(DS-DN)**](s-object-ds-dn.md)                      |
