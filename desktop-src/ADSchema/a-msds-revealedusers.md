# <a name="ms-ds-revealed-users-attribute"></a>attribut ms-DS-Revealed-Users

Utilisé avec les contrôleurs de domaine en lecture seule pour identifier les objets utilisateur dont les secrets ont été divulgués à ce RODC.



| Entrée | Valeur |
|-------------------|-------------------------------------------------|
| CN                | ms-DS-dévoilé-utilisateurs                            |
| LDAP-Display-Name | msDS-RevealedUsers                              |
| Taille              | \-                                              |
| Mettre à jour le privilège  | \-                                              |
| Fréquence des mises à jour  | \-                                              |
| Attribute-Id      | 1.2.840.113556.1.4.1924                         |
| System-ID-GUID    | 185c7821-3749-443a-bd6a-288899071adb            |
| Syntaxe            | [**Object(DN-Binary)**](s-object-dn-binary.md) |
