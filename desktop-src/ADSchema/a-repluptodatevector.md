# <a name="repl-uptodate-vector-attribute"></a>Attribut REPL-UpToDate-Vector

Effectue le suivi des informations d’état de la réplication interne pour un contexte de nommage entier. Les informations fournies ici peuvent être extraites sous forme publique via l’API DsReplicaGetInfo (). Présente sur tous les objets racines NC.



| Entrée | Valeur |
|-------------------|--------------------------------------------------------------------------------|
| CN                | REPL-UpToDate-Vector                                                           |
| LDAP-Display-Name | L’replUpToDateVector                                                             |
| Taille              | La longueur est proportionnelle au nombre de réplicas (passés et présents) du contexte de nommage. |
| Mettre à jour le privilège  | Cette valeur est définie par le système.                                               |
| Fréquence des mises à jour  | Modifications en réponse aux cycles de réplication entrante terminées.                   |
| Attribute-Id      | 1.2.840.113556.1.4.4                                                           |
| System-ID-GUID    | bf967a16-0de6-11d0-a285-00aa003049e2                                           |
| Syntaxe            | [**Object(Replica-Link)**](s-object-replica-link.md)                          |
