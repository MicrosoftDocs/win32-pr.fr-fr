# <a name="boolean-syntax"></a>Syntaxe booléenne

Représente une valeur booléenne.



| Entrée | Valeur |
|--------------|--------------------------------------------------------------------------|
| Nom         | Boolean                                                                  |
| ID de syntaxe    | 2.5.5.8                                                                  |
| ID OM        | 1                                                                        |
| Type MAPI    | BOOLEAN                                                                  |
| Type ADS     | ADSTYPE \_ booléen                                                         |
| Type de variante | VT \_ bool                                                                 |
| SDS, type     | [System.Boolean](/dotnet/api/system.boolean) |



## <a name="remarks"></a>Notes

Pour les requêtes qui incluent des attributs de syntaxe booléenne dans un filtre, spécifiez « TRUE » ou « FALSe » (par exemple, « myboolattr = TRUE »).
