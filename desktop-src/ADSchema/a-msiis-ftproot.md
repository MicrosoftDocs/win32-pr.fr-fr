# <a name="ms-iis-ftp-root-attribute"></a>MS-IIS-FTP-attribut racine

Cet attribut détermine le partage de serveur de fichiers. Il est utilisé conjointement avec MS-IIS-FTP-dir pour déterminer le répertoire de démarrage de l’utilisateur FTP.



| Entrée | Valeur |
|-------------------|---------------------------------------------------------------------------------------------------------------------------------|
| CN                | MS-IIS-FTP-root                                                                                                                 |
| LDAP-Display-Name | msIIS-FTPRoot                                                                                                                   |
| Taille              | 30-50 caractères (15-25 caractères Unicode pour chaque propriété)                                                                   |
| Mettre à jour le privilège  | Administrateur de domaine & système local                                                                                             |
| Fréquence des mises à jour  | Cette propriété est définie lorsque l’administrateur crée le site Web et définit l’isolement FTP. Cela va rarement changer après cela. |
| Attribute-Id      | 1.2.840.113556.1.4.1785                                                                                                         |
| System-ID-GUID    | 2a7827a4-1483-49a5-9d84-52e3812156b4                                                                                            |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md)                                                                                     |
