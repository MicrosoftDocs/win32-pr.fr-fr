# <a name="dns-record-attribute"></a>Attribut Dns-Record

Utilisé pour stocker des enregistrements de ressources DNS binaires sur des objets DNS.



| Entrée | Valeur |
|-------------------|-------------------------------------------------------|
| CN                | Dns-Record                                            |
| LDAP-Display-Name | dnsRecord                                             |
| Taille              | \-                                                    |
| Mettre à jour le privilège  | Cette valeur est définie par le système.                      |
| Fréquence des mises à jour  | Chaque fois que les enregistrements de ressources DNS changent.                 |
| Attribute-Id      | 1.2.840.113556.1.4.382                                |
| System-ID-GUID    | e0fa1e69-9b45-11d0-afdd-00c04fd930c9                  |
| Syntaxe            | [**Object(Replica-Link)**](s-object-replica-link.md) |
