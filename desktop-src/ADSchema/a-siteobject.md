# <a name="site-object-attribute"></a>Attribut Site-Object

Nom unique du site auquel appartient ce sous-réseau.



| Entrée | Valeur |
|-------------------|-----------------------------------------|
| CN                | Site-Object                             |
| LDAP-Display-Name | siteObject                              |
| Taille              | \-                                      |
| Mettre à jour le privilège  | Administrateur de domaine                    |
| Fréquence des mises à jour  | Chaque fois qu’un site est créé.             |
| Attribute-Id      | 1.2.840.113556.1.4.512                  |
| System-ID-GUID    | 3e10944c-c354-11d0-aff8-0000f80367c1    |
| Syntaxe            | [**Object(DS-DN)**](s-object-ds-dn.md) |
