# <a name="shadowwarning-attribute"></a>attribut shadowWarning

Contient le nombre de jours avant l’expiration du mot de passe pour avertir l’utilisateur.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | shadowWarning                        |
| LDAP-Display-Name | shadowWarning                        |
| Taille              | \-                                   |
| Mettre à jour le privilège  | \-                                   |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.8                      |
| System-ID-GUID    | 7ae89c9c-2976-4a46-bb8a-340f88560117 |
| Syntaxe            | [**Énumération**](s-enumeration.md) |
