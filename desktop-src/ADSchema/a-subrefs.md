# <a name="sub-refs-attribute"></a>Attribut Sub-Refs

Liste des références subordonnées d’un contexte d’appellation.



| Entrée | Valeur |
|-------------------|-----------------------------------------|
| CN                | Sub-Refs                                |
| LDAP-Display-Name | subRefs                                 |
| Taille              | \-                                      |
| Mettre à jour le privilège  | \-                                      |
| Fréquence des mises à jour  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.2.7                    |
| System-ID-GUID    | bf967a3c-0de6-11d0-a285-00aa003049e2    |
| Syntaxe            | [**Object(DS-DN)**](s-object-ds-dn.md) |
