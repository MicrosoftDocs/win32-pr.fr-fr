# <a name="netboot-scp-bl-attribute"></a>attribut netboot-SCP-BL

Liste des points de connexion de service qui font référence à ce serveur NetBoot.



| Entrée | Valeur |
|-------------------|-----------------------------------------|
| CN                | netboot-SCP-BL                          |
| LDAP-Display-Name | netbootSCPBL                            |
| Taille              | \-                                      |
| Mettre à jour le privilège  | Cette valeur est définie par le système.        |
| Fréquence des mises à jour  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.864                  |
| System-ID-GUID    | 07383082-91df-11d1-aebc-0000f80367c1    |
| Syntaxe            | [**Object(DS-DN)**](s-object-ds-dn.md) |
