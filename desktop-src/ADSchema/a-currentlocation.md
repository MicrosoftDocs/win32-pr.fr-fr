# <a name="current-location-attribute"></a>Attribut Current-Location

Emplacement de l’ordinateur pour un objet qui a été déplacé.



| Entrée | Valeur |
|-------------------|-------------------------------------------------------|
| CN                | Current-Location                                      |
| LDAP-Display-Name | currentLocation                                       |
| Taille              | \-                                                    |
| Mettre à jour le privilège  | Cette valeur est définie par le système.                      |
| Fréquence des mises à jour  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.335                                |
| System-ID-GUID    | 1f0075fc-7e40-11d0-afd6-00c04fd930c9                  |
| Syntaxe            | [**Object(Replica-Link)**](s-object-replica-link.md) |
