# <a name="msmq-digests-attribute"></a>Attribut MSMQ-Digests

Tableau des résumés des certificats correspondants dans l’attribut mSMQ-Sign-Certificates. Ils sont utilisés pour mapper un résumé dans un certificat.



| Entrée | Valeur |
|-------------------|-------------------------------------------------------|
| CN                | MSMQ-Digests                                          |
| LDAP-Display-Name | mSMQDigests                                           |
| Taille              | Chaque condensé est de 16 octets.                              |
| Mettre à jour le privilège  | \-                                                    |
| Fréquence des mises à jour  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.948                                |
| System-ID-GUID    | 9a0dc33c-c100-11d1-bbc5-0080c76670c0                  |
| Syntaxe            | [**Object(Replica-Link)**](s-object-replica-link.md) |
