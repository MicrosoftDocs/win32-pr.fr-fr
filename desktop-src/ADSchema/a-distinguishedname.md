# <a name="obj-dist-name-attribute"></a>Obj-Dist-Name (attribut)

Identique au nom unique d’un objet. Utilisé par Exchange.



| Entrée | Valeur |
|-------------------|-----------------------------------------|
| CN                | Obj-Dist-Name                           |
| LDAP-Display-Name | distinguishedName                       |
| Taille              | \-                                      |
| Mettre à jour le privilège  | Cette valeur est définie par le système.        |
| Fréquence des mises à jour  | À chaque fois qu’un objet est créé ou déplacé. |
| Attribute-Id      | 2.5.4.49                                |
| System-ID-GUID    | bf9679e4-0de6-11d0-a285-00aa003049e2    |
| Syntaxe            | [**Object(DS-DN)**](s-object-ds-dn.md) |
