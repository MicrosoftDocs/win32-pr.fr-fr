# <a name="netboot-new-machine-ou-attribute"></a>netboot-nouvel-machine-attribut d’UO

Indique où le nouveau compte d’ordinateur client sera créé.



| Entrée | Valeur |
|-------------------|-----------------------------------------|
| CN                | netboot-New-machine-OU                  |
| LDAP-Display-Name | netbootNewMachineOU                     |
| Taille              | \-                                      |
| Mettre à jour le privilège  | Cette valeur est définie par le système.        |
| Fréquence des mises à jour  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.856                  |
| System-ID-GUID    | 0738307d-91df-11d1-aebc-0000f80367c1    |
| Syntaxe            | [**Object(DS-DN)**](s-object-ds-dn.md) |
