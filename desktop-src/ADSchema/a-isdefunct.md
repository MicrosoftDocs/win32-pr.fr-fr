# <a name="is-defunct-attribute"></a>Attribut Is-Defunct

Si la **valeur est true**, la classe ou l’attribut n’est plus utilisable. Les anciennes versions de cet objet peuvent exister, mais de nouvelles ne peuvent pas être créées.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | Is-Defunct                           |
| LDAP-Display-Name | Définie                            |
| Taille              | \-                                   |
| Mettre à jour le privilège  | Administrateur de schéma                 |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.661               |
| System-ID-GUID    | 28630ebe-41d5-11d1-a9c1-0000f80367c1 |
| Syntaxe            | [**Boolean**](s-boolean.md)         |
