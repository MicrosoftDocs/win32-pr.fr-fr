# <a name="ms-com-partitionlink-attribute"></a>attribut ms-COM-PartitionLink

Lien utilisé pour associer un PartitionSet COM+ à ses partitions COM+ constitutives.



| Entrée | Valeur |
|-------------------|-----------------------------------------|
| CN                | MS-COM-PartitionLink                    |
| LDAP-Display-Name | msCOM-PartitionLink                     |
| Taille              | \-                                      |
| Mettre à jour le privilège  | \-                                      |
| Fréquence des mises à jour  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1423                 |
| System-ID-GUID    | 09abac62-043F-4702-AC2B-6ca15eee5754    |
| Syntaxe            | [**Object(DS-DN)**](s-object-ds-dn.md) |
