# <a name="last-logoff-attribute"></a>Attribut Last-Logoff

Cet attribut n’est pas utilisé.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | Last-Logoff                          |
| LDAP-Display-Name | lastLogoff                           |
| Taille              | 8 octets                              |
| Mettre à jour le privilège  | Cette valeur est définie par le système.     |
| Fréquence des mises à jour  | Chaque fois que l’utilisateur se déconnecte.         |
| Attribute-Id      | 1.2.840.113556.1.4.51                |
| System-ID-GUID    | bf967996-0de6-11d0-a285-00aa003049e2 |
| Syntaxe            | [**Défini**](s-interval.md)       |
