# <a name="ipprotocolnumber-attribute"></a>attribut ipProtocolNumber

Contient la section de la carte des protocoles qui stocke le numéro unique qui identifie le protocole.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | ipProtocolNumber                     |
| LDAP-Display-Name | ipProtocolNumber                     |
| Taille              | \-                                   |
| Mettre à jour le privilège  | \-                                   |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.17                     |
| System-ID-GUID    | ebf5c6eb-0e2d-4415-9670-1081993b4211 |
| Syntaxe            | [**Enumeration**](s-enumeration.md) |
