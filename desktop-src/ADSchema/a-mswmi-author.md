# <a name="ms-wmi-author-attribute"></a>MS-WMI-attribut Author

|-------------------|---------------------------------------------|
| CN                | MS-WMI-auteur                               |
| LDAP-Display-Name | msWMI-auteur                                |
| Taille              | Moins de 25 caractères.                    |
| Mettre à jour le privilège  | Administrateur stratégie de groupe                  |
| Fréquence des mises à jour  | \-                                          |

| System-ID-GUID    | 6366c0c1-6972-4e66-b3a5-1d52ad0c0547        |
