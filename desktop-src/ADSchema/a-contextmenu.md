# <a name="context-menu-attribute"></a>Attribut Context-Menu

Le numéro d’ordre et le GUID du menu contextuel à utiliser pour un objet.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | Context-Menu                                |
| LDAP-Display-Name | contextMenu                                 |
| Taille              | \-                                          |
| Mettre à jour le privilège  | Administrateur de schéma                        |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.499                      |
| System-ID-GUID    | 4d8601ee-ac85-11d0-afe3-00c04fd930c9        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
