# <a name="ms-sql-lastdiagnosticdate-attribute"></a>attribut MS-SQL-LastDiagnosticDate

Date de la dernière exécution de DBCC CHECKDB.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | MS-SQL-LastDiagnosticDate                   |
| LDAP-Display-Name | mS-SQL-LastDiagnosticDate                   |
| Taille              | \-                                          |
| Mettre à jour le privilège  | Cette valeur est définie par le système.            |
| Fréquence des mises à jour  | Lors de l’exécution de DBCC CHECKDB.                   |
| Attribute-Id      | 1.2.840.113556.1.4.1399                     |
| System-ID-GUID    | f6d6dd88-ccee-11d2-9993-0000f87a57d4        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
