# <a name="acs-non-reserved-token-size-attribute"></a>ACS-attribut de taille de jeton non réservé

L’attribut **ACS-non-reserved-Token-Size** est destiné à un usage interne uniquement. Basé sur RFC2814.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | ACS-non-réservé-Token-Size          |
| LDAP-Display-Name | aCSNonReservedTokenSize              |
| Taille              | 8 octets                              |
| Mettre à jour le privilège  | \-                                   |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1319              |
| System-ID-GUID    | a916d7c9-3b90-11d2-90cc-00c04fd91ab1 |
| Syntaxe            | [**Défini**](s-interval.md)       |
