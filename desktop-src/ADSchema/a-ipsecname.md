# <a name="ipsec-name-attribute"></a>Attribut Ipsec-Name

L’attribut de **nom IPSec** est destiné à un usage interne uniquement.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | Ipsec-Name                                  |
| LDAP-Display-Name | ipsecName                                   |
| Taille              | \-                                          |
| Mettre à jour le privilège  | \-                                          |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.620                      |
| System-ID-GUID    | b40ff81c-427a-11d1-a9c2-0000f80367c1        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
