# <a name="print-start-time-attribute"></a>Attribut Print-Start-Time

Heure à laquelle une file d’attente à l’impression commence à traiter les travaux.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | Print-Start-Time                     |
| LDAP-Display-Name | printStartTime                       |
| Taille              | 4 octets                              |
| Mettre à jour le privilège  | \-                                   |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.233               |
| System-ID-GUID    | 281416c9-1968-11d0-a28f-00aa003049e2 |
| Syntaxe            | [**Énumération**](s-enumeration.md) |
