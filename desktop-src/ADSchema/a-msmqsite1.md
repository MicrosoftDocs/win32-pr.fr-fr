# <a name="msmq-site-1-attribute"></a>Attribut MSMQ-site-1

DN du premier site d’une paire connectée.



| Entrée | Valeur |
|-------------------|-----------------------------------------|
| CN                | MSMQ-site-1                             |
| LDAP-Display-Name | mSMQSite1                               |
| Taille              | \-                                      |
| Mettre à jour le privilège  | \-                                      |
| Fréquence des mises à jour  | Chaque fois qu’une ligne de site MSMQ est créée. |
| Attribute-Id      | 1.2.840.113556.1.4.943                  |
| System-ID-GUID    | 9a0dc337-c100-11d1-bbc5-0080c76670c0    |
| Syntaxe            | [**Object(DS-DN)**](s-object-ds-dn.md) |
