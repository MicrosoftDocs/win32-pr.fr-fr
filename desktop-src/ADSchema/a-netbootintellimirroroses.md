# <a name="netboot-intellimirror-oses-attribute"></a>netboot-attribut IntelliMirror-OS

L’attribut **netboot-IntelliMirror-OS** est réservé à un usage interne.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | netboot-IntelliMirror-systèmes d’exploitation                  |
| LDAP-Display-Name | netbootIntelliMirrorOSes                    |
| Taille              | \-                                          |
| Mettre à jour le privilège  | Cette valeur est définie par le système.            |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.857                      |
| System-ID-GUID    | 0738307e-91df-11d1-aebc-0000f80367c1        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
