# <a name="ms-sql-informationdirectory-attribute"></a>Attribut MS-SQL-InformationDirectory

Répertoire d’informations pour l’instance actuelle de SQL Server.



| Entrée | Valeur |
|-------------------|------------------------------------------|
| CN                | MS-SQL-InformationDirectory              |
| LDAP-Display-Name | mS-SQL-InformationDirectory              |
| Taille              | \-                                       |
| Mettre à jour le privilège  | Administrateur de domaine                     |
| Fréquence des mises à jour  | Au démarrage du système ou lors de la mise à jour d’AD. |
| Attribute-Id      | 1.2.840.113556.1.4.1392                  |
| System-ID-GUID    | d0aedb2e-ccee-11d2-9993-0000f87a57d4     |
| Syntaxe            | [**Boolean**](s-boolean.md)             |
