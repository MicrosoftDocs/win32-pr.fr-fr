# <a name="ms-ts-work-directory-attribute"></a>attribut ms-TS-Work-Directory

Répertoire de travail de la session des services Terminal Server spécifie le chemin d’accès du répertoire de travail de l’utilisateur.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | MS-TS-Work-Directory                        |
| LDAP-Display-Name | msTSWorkDirectory                           |
| Taille              | \-                                          |
| Mettre à jour le privilège  | \-                                          |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1989                     |
| System-ID-GUID    | a744f666-3d3c-4cc8-834b-9d4f6f687b8b        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
