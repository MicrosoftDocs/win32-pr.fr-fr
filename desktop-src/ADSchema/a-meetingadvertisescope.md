# <a name="meetingadvertisescope-attribute"></a>attribut meetingAdvertiseScope

Indique si une entrée doit être publiée en dehors d’un proxy ou d’une passerelle d’entreprise.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | meetingAdvertiseScope                       |
| LDAP-Display-Name | meetingAdvertiseScope                       |
| Taille              | \-                                          |
| Mettre à jour le privilège  | \-                                          |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.582                      |
| System-ID-GUID    | 11b6cc8b-48c4-11d1-a9c3-0000f80367c1        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
