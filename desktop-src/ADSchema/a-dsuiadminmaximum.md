# <a name="ds-ui-admin-maximum-attribute"></a>DS-UI-admin-attribut maximal

Il s’agit du nombre maximal par défaut d’objets à afficher dans un conteneur par l’interface utilisateur de l’administrateur.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | DS-UI-admin-maximum                  |
| LDAP-Display-Name | dSUIAdminMaximum                     |
| Taille              | \-                                   |
| Mettre à jour le privilège  | Administrateur de domaine                 |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1344              |
| System-ID-GUID    | ee8d0ae0-6f91-11d2-9905-0000f87a57d4 |
| Syntaxe            | [**Énumération**](s-enumeration.md) |
