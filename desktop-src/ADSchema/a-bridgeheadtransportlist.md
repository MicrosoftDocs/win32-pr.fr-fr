# <a name="bridgehead-transport-list-attribute"></a>Attribut pont-transport-List

Transports pour lesquels ce serveur est un serveur tête de pont.



| Entrée | Valeur |
|-------------------|-----------------------------------------|
| CN                | Pont-transport-liste               |
| LDAP-Display-Name | bridgeheadTransportList                 |
| Taille              | \-                                      |
| Mettre à jour le privilège  | Cette valeur est définie par le système.        |
| Fréquence des mises à jour  | Chaque fois qu’un site est configuré.               |
| Attribute-Id      | 1.2.840.113556.1.4.819                  |
| System-ID-GUID    | d50c2cda-8951-11d1-aebc-0000f80367c1    |
| Syntaxe            | [**Object(DS-DN)**](s-object-ds-dn.md) |
