# <a name="gp-link-attribute"></a>Attribut GP-Link

Liste triée des options de stratégie de groupe. Chaque option est une **valeur DWORD**. L’utilisation de la chaîne UNICODE est un avantage pratique.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | GP-Link                                     |
| LDAP-Display-Name | gPLink                                      |
| Taille              | \-                                          |
| Mettre à jour le privilège  | \-                                          |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.891                      |
| System-ID-GUID    | f30e3bbe-9ff0-11d1-b603-0000f80367c1        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
