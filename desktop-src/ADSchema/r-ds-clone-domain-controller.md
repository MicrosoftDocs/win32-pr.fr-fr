# <a name="ds-clone-domain-controller-extended-right"></a>Droits étendus du contrôleur de domaine DS-clone-Domain

Autoriser un contrôleur de contrôleur à créer un clone de lui-même.



| Entrée | Valeur |
|--------------|----------------------------------------|
| CN           | DS-clone-Domain-Controller             |
| Display-Name | Autoriser un contrôleur de contrôleur à créer un clone de lui-même |
| Rights-GUID  | 3e0f7e18-2c7a-4c10-ba82-4d926db99a3e   |
