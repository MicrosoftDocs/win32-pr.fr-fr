# <a name="certificate-revocation-list-attribute"></a>Attribut de liste de révocation de certificats

Représente une liste de certificats qui ont été révoqués.



| Entrée | Valeur |
|-------------------|-------------------------------------------------------|
| CN                | Certificat-révocation-liste                           |
| LDAP-Display-Name | certificateRevocationList                             |
| Taille              | \-                                                    |
| Mettre à jour le privilège  | \-                                                    |
| Fréquence des mises à jour  | \-                                                    |
| Attribute-Id      | 2.5.4.39                                              |
| System-ID-GUID    | 1677579f-47f3-11d1-a9c3-0000f80367c1                  |
| Syntaxe            | [**Object(Replica-Link)**](s-object-replica-link.md) |
