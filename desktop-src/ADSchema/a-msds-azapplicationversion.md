# <a name="ms-ds-az-application-version-attribute"></a>attribut ms-DS-AZ-application-version

Numéro de version indiquant que le AzApplication est mis à jour.



| Entrée | Valeur |
|-------------------|------------------------------------------------------|
| CN                | ms-DS-AZ-application-version                         |
| LDAP-Display-Name | msDS-AzApplicationVersion                            |
| Taille              | \-                                                   |
| Mettre à jour le privilège  | Administrateur AzRoles                                        |
| Fréquence des mises à jour  | Lorsque le numéro de version de l’application change. |
| Attribute-Id      | 1.2.840.113556.1.4.1817                              |
| System-ID-GUID    | 7184a120-3ac4-47ae-848f-fe0ab20784d4                 |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md)          |
