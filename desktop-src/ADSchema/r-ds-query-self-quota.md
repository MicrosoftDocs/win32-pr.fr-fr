# <a name="ds-query-self-quota-extended-right"></a>Droits étendus DS-Query-Self-quota

Droit de contrôle d’accès qui permet à un utilisateur d’interroger les propres quotas de l’utilisateur.



| Entrée | Valeur |
|--------------|--------------------------------------|
| CN           | DS-Query-auto-quota                  |
| Display-Name | Interroger le quota automatique                     |
| Rights-GUID  | 4ecc03fe-ffc0-4947-b630-eb672a8a9dbc |
