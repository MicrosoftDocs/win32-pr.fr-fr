# <a name="drink-attribute"></a>attribut Drink

Le type d’attribut Drink (boisson préférée) spécifie la boisson favorite d’un objet (ou personne).



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | Drink                                       |
| LDAP-Display-Name | Drink                                       |
| Taille              | \-                                          |
| Mettre à jour le privilège  | \-                                          |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 0.9.2342.19200300.100.1.5                   |
| System-ID-GUID    | 1a1aa5b5-262e-4df6-af04-2cf6b0d80048        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
