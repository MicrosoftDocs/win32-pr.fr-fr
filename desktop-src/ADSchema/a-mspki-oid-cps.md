# <a name="ms-pki-oid-cps-attribute"></a>attribut ms-PKI-OID-CPS

Déclaration CPS (Certificate Policy) pour l’OID de stratégie de l’émetteur d’entreprise.



| Entrée | Valeur |
|-------------------|--------------------------------------------------------------------------------------------|
| CN                | MS-PKI-OID-CPS                                                                             |
| LDAP-Display-Name | Attribut mspki-OID-CPS                                                                              |
| Taille              | \-                                                                                         |
| Mettre à jour le privilège  | Administrateur d’entreprise                                                                   |
| Fréquence des mises à jour  | À chaque fois qu’un nouveau modèle est créé ou que les attributs d’un modèle existant sont modifiés. |
| Attribute-Id      | 1.2.840.113556.1.4.1672                                                                    |
| System-ID-GUID    | 5f49940e-a79f-4a51-bb6f-3d446a54dc6b                                                       |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md)                                                |
