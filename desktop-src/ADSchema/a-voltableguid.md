# <a name="vol-table-guid-attribute"></a>Attribut volume-table-GUID

Identificateur unique d’une entrée de table de volume de lien de suivi.



| Entrée | Valeur |
|-------------------|-------------------------------------------------------|
| CN                | Volume-table-GUID                                        |
| LDAP-Display-Name | volTableGUID                                          |
| Taille              | 16 octets                                              |
| Mettre à jour le privilège  | Cette valeur est définie par le système.                      |
| Fréquence des mises à jour  | Chaque fois qu’un nouveau lien vers un fichier est créé.             |
| Attribute-Id      | 1.2.840.113556.1.4.336                                |
| System-ID-GUID    | 1f0075fd-7e40-11d0-afd6-00c04fd930c9                  |
| Syntaxe            | [**Object(Replica-Link)**](s-object-replica-link.md) |
