# <a name="lsa-modified-count-attribute"></a>Attribut LSA-modified-Count

L’attribut **LSA-modified-Count** est utilisé pour prendre en charge la réplication vers Windows domaines NT 4,0.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | LSA-modifié-nombre                   |
| LDAP-Display-Name | lSAModifiedCount                     |
| Taille              | 8 octets                              |
| Mettre à jour le privilège  | Cette valeur est définie par le système.     |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.67                |
| System-ID-GUID    | bf9679ae-0de6-11d0-a285-00aa003049e2 |
| Syntaxe            | [**Défini**](s-interval.md)       |
