# <a name="ms-ds-behavior-version-attribute"></a>attribut ms-DS-Behavior-Version

Cet attribut est utilisé pour effectuer le suivi de la version de comportement du domaine ou de la forêt. Il s’agit d’un nombre à croissance monotone qui est utilisé pour activer certaines fonctionnalités de Active Directory.



| Entrée | Valeur |
|-------------------|------------------------------------------------|
| CN                | ms-DS-Behavior-Version                         |
| LDAP-Display-Name | msDS-Behavior-Version                          |
| Taille              | \-                                             |
| Mettre à jour le privilège  | Cette valeur est définie par l’administrateur de domaine. |
| Fréquence des mises à jour  | \-                                             |
| Attribute-Id      | 1.2.840.113556.1.4.1459                        |
| System-ID-GUID    | d31a8757-2447-4545-8081-3bb610cacbf2           |
| Syntaxe            | [**Enumeration**](s-enumeration.md)           |
