# <a name="object-class-category-attribute"></a>Attribut de catégorie objet-classe

Cet attribut contient le type de classe, tel que abstract, auxiliaire ou Structured.



| Entrée | Valeur |
|-------------------|---------------------------------------------------------------------------------|
| CN                | Catégorie objet-classe                                                           |
| LDAP-Display-Name | objectClassCategory                                                             |
| Taille              | 4 octets. Structural 1, abstract 2, auxiliaire 3. La classe 88, 0 ne doit pas être utilisée. |
| Mettre à jour le privilège  | Le concepteur de l’objet définit cette valeur.                                |
| Fréquence des mises à jour  | Cette valeur ne doit jamais changer.                                                 |
| Attribute-Id      | 1.2.840.113556.1.2.370                                                          |
| System-ID-GUID    | bf9679e6-0de6-11d0-a285-00aa003049e2                                            |
| Syntaxe            | [**Enumeration**](s-enumeration.md)                                            |
