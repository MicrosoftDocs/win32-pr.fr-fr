# <a name="ms-ds-non-members-attribute"></a>attribut ms-DS-non-members

Cet attribut remplit la même fonction que l’attribut de membre de non-sécurité, mais avec les règles de portée appliquées.



| Entrée | Valeur |
|-------------------|-----------------------------------------|
| CN                | ms-DS-non-membres                       |
| LDAP-Display-Name | msDS-non-membres                         |
| Taille              | \-                                      |
| Mettre à jour le privilège  | Administrateur AzRoles                           |
| Fréquence des mises à jour  | Lors de l’initialisation et de la modification de la stratégie.    |
| Attribute-Id      | 1.2.840.113556.1.4.1793                 |
| System-ID-GUID    | cafcb1de-f23c-46b5-adf7-1e64957bd5db    |
| Syntaxe            | [**Object(DS-DN)**](s-object-ds-dn.md) |
