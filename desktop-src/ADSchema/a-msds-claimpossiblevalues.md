# <a name="ms-ds-claim-possible-values-attribute"></a>attribut ms-DS-claim-possibles-values

Pour un type de revendication ou un objet de propriété de ressource, cet attribut décrit les valeurs suggérées à un utilisateur lorsqu’il utilise le type de revendication ou la propriété de ressource dans les applications.
