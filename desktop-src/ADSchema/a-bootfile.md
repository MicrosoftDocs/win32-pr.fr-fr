# <a name="bootfile-attribute"></a>attribut bootFile

contient le nom de l’image de démarrage pour la prise en charge de la gestion des identités UNIX.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | bootFile                             |
| LDAP-Display-Name | bootFile                             |
| Taille              | \-                                   |
| Mettre à jour le privilège  | \-                                   |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.24                     |
| System-ID-GUID    | e3f3cb4e-0f20-42eb-9703-d2ff26e52667 |
| Syntaxe            | [**String(IA5)**](s-string-ia5.md)  |
