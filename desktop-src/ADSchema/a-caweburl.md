# <a name="ca-web-url-attribute"></a>Attribut WEB-URL de l’autorité de certification

URL pour la connexion http à une autorité de certification.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | AUTORITÉ DE CERTIFICATION-WEB-URL                                  |
| LDAP-Display-Name | cAWEBURL                                    |
| Taille              | \-                                          |
| Mettre à jour le privilège  | \-                                          |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.688                      |
| System-ID-GUID    | 963d2736-48be-11d1-a9c3-0000f80367c1        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
