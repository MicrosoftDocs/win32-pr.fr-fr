# <a name="memberuid-attribute"></a>attribut memberUid

Contient les noms de connexion des membres d’un groupe.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | memberUid                            |
| LDAP-Display-Name | memberUid                            |
| Taille              | \-                                   |
| Mettre à jour le privilège  | \-                                   |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.12                     |
| System-ID-GUID    | 03dab236-672e-4f61-ab64-f77d2dc2ffab |
| Syntaxe            | [**String(IA5)**](s-string-ia5.md)  |
