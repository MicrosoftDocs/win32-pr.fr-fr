# <a name="frs-time-last-command-attribute"></a>Attribut de dernière commande FRS

Heure à laquelle la dernière commande a été exécutée.



| Entrée | Valeur |
|-------------------|---------------------------------------------------------------|
| CN                | FRS-heure-dernière commande                                         |
| LDAP-Display-Name | fRSTimeLastCommand                                            |
| Taille              | \-                                                            |
| Mettre à jour le privilège  | \-                                                            |
| Fréquence des mises à jour  | \-                                                            |
| Attribute-Id      | 1.2.840.113556.1.4.880                                        |
| System-ID-GUID    | 2a132583-9373-11d1-aebc-0000f80367c1                          |
| Syntaxe            | [**String(Generalized-Time)**](s-string-generalized-time.md) |
