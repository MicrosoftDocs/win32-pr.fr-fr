# <a name="site-list-attribute"></a>Attribut Site-List

Liste des sites connectés à cet objet de liaison.



| Entrée | Valeur |
|-------------------|----------------------------------------------|
| CN                | Site-List                                    |
| LDAP-Display-Name | Sites                                     |
| Taille              | \-                                           |
| Mettre à jour le privilège  | Administrateur de domaine                         |
| Fréquence des mises à jour  | À chaque fois qu’un site est ajouté ou supprimé dans. |
| Attribute-Id      | 1.2.840.113556.1.4.821                       |
| System-ID-GUID    | d50c2cdc-8951-11d1-aebc-0000f80367c1         |
| Syntaxe            | [**Object(DS-DN)**](s-object-ds-dn.md)      |
