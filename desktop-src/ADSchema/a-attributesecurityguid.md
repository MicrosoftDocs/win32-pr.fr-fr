# <a name="attribute-security-guid-attribute"></a>Attribut-Security-GUID (attribut)

GUID à utiliser pour appliquer des informations d’identification de sécurité à un ensemble d’objets.

|-------------------|-------------------------------------------------------|
| CN                | Attribute-Security-GUID                               |
| LDAP-Display-Name | attributeSecurityGUID                                 |
| Taille              | \-                                                    |
| Mettre à jour le privilège  | Administrateur de schéma                                  |
| Fréquence des mises à jour  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.149                                |
| System-ID-GUID    | bf967924-0de6-11d0-a285-00aa003049e2                  |
| Syntaxe            | [**Object(Replica-Link)**](s-object-replica-link.md) |
