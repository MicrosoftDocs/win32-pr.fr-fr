# <a name="iphostnumber-attribute"></a>attribut ipHostNumber

Contient l’adresse IP de l’hôte en notation décimale séparée par des points, en omettant les zéros non significatifs.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | ipHostNumber                         |
| LDAP-Display-Name | ipHostNumber                         |
| Taille              | \-                                   |
| Mettre à jour le privilège  | \-                                   |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.19                     |
| System-ID-GUID    | de8bb721-85dc-4fde-b687-9657688e667e |
| Syntaxe            | [**String(IA5)**](s-string-ia5.md)  |
