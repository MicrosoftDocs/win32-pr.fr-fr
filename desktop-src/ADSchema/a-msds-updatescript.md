# <a name="ms-ds-updatescript-attribute"></a>attribut ms-DS-UpdateScript

Il est utilisé pour contenir le script avec les instructions de restructuration de domaine.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | ms-DS-UpdateScript                          |
| LDAP-Display-Name | msDS-UpdateScript                           |
| Taille              | Taille maximale de 10 Ko.                     |
| Mettre à jour le privilège  | Cette valeur est définie par le système.            |
| Fréquence des mises à jour  | Uniquement pendant la restructuration de domaine.             |
| Attribute-Id      | 1.2.840.113556.1.4.1721                     |
| System-ID-GUID    | 146eb639-bb9f-4fc1-a825-e29e00c77920        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
