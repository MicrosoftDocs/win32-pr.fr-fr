# <a name="ms-authz-central-access-policy-id-attribute"></a>MS-auth-central-Access-Policy-ID (attribut)

Pour une stratégie d’accès centralisée, cet attribut définit un GUID qui peut être utilisé pour identifier le jeu de stratégies lorsqu’il est appliqué à une ressource.

|-------------------|--------------------------------------|
| CN                | MS-auth-central-Access-Policy-ID    |
| LDAP-Display-Name | msAuthz-CentralAccessPolicyID        |
| Taille              | \-                                   |
| Mettre à jour le privilège  | \-                                   |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.2154              |
| System-ID-GUID    | 62f29b60-be74-4630-9456-2f6691993a86 |
| Syntaxe            | [**Chaîne (SID)**](s-string-sid.md)  |
