# <a name="ms-pki-ra-application-policies-attribute"></a>attribut ms-PKI-RA-application-Policies

OID de stratégie d’application RA requis dans les signatures de compteur de la demande de certificat.



| Entrée | Valeur |
|-------------------|--------------------------------------------------------------------------------------------|
| CN                | MS-PKI-RA-application-stratégies                                                             |
| LDAP-Display-Name | Attribut mspki-RA-application-Policies                                                              |
| Taille              | \-                                                                                         |
| Mettre à jour le privilège  | Administrateur d’entreprise                                                                   |
| Fréquence des mises à jour  | À chaque fois qu’un nouveau modèle est créé ou que les attributs d’un modèle existant sont modifiés. |
| Attribute-Id      | 1.2.840.113556.1.4.1675                                                                    |
| System-ID-GUID    | 3c91fbbf-4773-4ccd-a87b-85d53e7bcf6a                                                       |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md)                                                |
