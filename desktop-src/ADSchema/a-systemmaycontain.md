# <a name="system-may-contain-attribute"></a>Attribut système-peut contenir

Liste des attributs facultatifs d’une classe. La liste des attributs ne peut être modifiée que par le système.



| Entrée | Valeur |
|-------------------|------------------------------------------------------------------------------|
| CN                | Système-peut contenir                                                           |
| LDAP-Display-Name | systemMayContain                                                             |
| Taille              | \-                                                                           |
| Mettre à jour le privilège  | Administrateur de schéma                                                         |
| Fréquence des mises à jour  | Lorsque la classe est créée ou qu’un nouvel attribut facultatif est ajouté à la classe. |
| Attribute-Id      | 1.2.840.113556.1.4.196                                                       |
| System-ID-GUID    | bf967a44-0de6-11d0-a285-00aa003049e2                                         |
| Syntaxe            | [**String(Object-Identifier)**](s-string-object-identifier.md)              |
