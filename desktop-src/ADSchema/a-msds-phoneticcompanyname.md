# <a name="ms-ds-phonetic-company-name-attribute"></a>attribut ms-DS-Phonetic-Company-Name

Contient le nom de société phonétique dans lequel la personne travaille.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | ms-DS-phonétique-nom de la société                 |
| LDAP-Display-Name | msDS-PhoneticCompanyName                    |
| Taille              | \-                                          |
| Mettre à jour le privilège  | \-                                          |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1945                     |
| System-ID-GUID    | 5bd5208d-e5f4-46ae-a514-543bc9c47659        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
