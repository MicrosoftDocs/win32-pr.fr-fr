# <a name="msmq-authenticate-attribute"></a>Attribut MSMQ-Authenticate

Indique si la file d’attente accepte uniquement les messages authentifiés.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | MSMQ-Authenticate                    |
| LDAP-Display-Name | mSMQAuthenticate                     |
| Taille              | \-                                   |
| Mettre à jour le privilège  | \-                                   |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.923               |
| System-ID-GUID    | 9a0dc326-c100-11d1-bbc5-0080c76670c0 |
| Syntaxe            | [**Boolean**](s-boolean.md)         |
