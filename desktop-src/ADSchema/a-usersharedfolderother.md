# <a name="user-shared-folder-other-attribute"></a>Utilisateur-partagé-dossier-autre attribut

Spécifie un chemin d’accès UNC au dossier Documents partagés supplémentaire de l’utilisateur. Le chemin d’accès doit être un chemin d’accès UNC réseau au format **\\\\** Répertoire de partage du _serveur_ *_\\_*  *_\\_* . Cette valeur peut être une chaîne NULL.



| Entrée | Valeur |
|-------------------|-----------------------------------------------------------------------------------|
| CN                | Utilisateur-partagé-dossier-autre                                                          |
| LDAP-Display-Name | userSharedFolderOther                                                             |
| Taille              | \-                                                                                |
| Mettre à jour le privilège  | Administrateur de domaine ou propriétaire du compte.                                            |
| Fréquence des mises à jour  | Lorsque l’enregistrement de l’utilisateur est créé et chaque fois que le dossier partagé doit être modifié. |
| Attribute-Id      | 1.2.840.113556.1.4.752                                                            |
| System-ID-GUID    | 9a9a0220-4a5b-11d1-a9c3-0000f80367c1                                              |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md)                                       |
