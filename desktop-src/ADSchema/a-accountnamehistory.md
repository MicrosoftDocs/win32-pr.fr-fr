# <a name="account-name-history-attribute"></a>Attribut Account-Name-History

Durée pendant laquelle le compte a été actif.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | Nom-compte-historique                        |
| LDAP-Display-Name | accountNameHistory                          |
| Taille              | \-                                          |
| Mettre à jour le privilège  | Administrateur de domaine                        |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1307                     |
| System-ID-GUID    | 031952ec-3b72-11d2-90cc-00c04fd91ab1        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
