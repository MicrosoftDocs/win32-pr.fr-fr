# <a name="acs-max-token-bucket-per-flow-attribute"></a>attribut ACS-Max-Token-compartiment-Per-Flow

l’attribut **ACS-Max-Token-compartiment-Per-Flow** est réservé à un usage interne uniquement. Basé sur RFC2210.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | ACS-Max-Token-seau-per-Flow        |
| LDAP-Display-Name | aCSMaxTokenBucketPerFlow             |
| Taille              | 8 octets                              |
| Mettre à jour le privilège  | \-                                   |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1313              |
| System-ID-GUID    | 81f6e0df-3b90-11d2-90cc-00c04fd91ab1 |
| Syntaxe            | [**Défini**](s-interval.md)       |
