# <a name="ms-ts-max-connection-time-attribute"></a>MS-TS-Max-Connection-Time (attribut)

Durée de connexion maximale de la session des services Terminal Server est la durée maximale, en minutes, de la session des services Terminal Server.

|-------------------|--------------------------------------|
| CN                | MS-TS-Max-Connection-Time            |
| LDAP-Display-Name | msTSMaxConnectionTime                |
| Taille              | \-                                   |
| Mettre à jour le privilège  | \-                                   |
| Fréquence des mises à jour  | \-                                   |

| System-ID-GUID    | 1d960ee2-6464-4e95-a781-e3b5cd5f9588 |
