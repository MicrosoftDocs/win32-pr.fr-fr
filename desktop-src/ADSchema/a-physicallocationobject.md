# <a name="physical-location-object-attribute"></a>Attribut physique-location-Object

Utilisé pour mapper un appareil (par exemple, une imprimante, un ordinateur, etc.) à un emplacement physique.



| Entrée | Valeur |
|-------------------|-----------------------------------------|
| CN                | Physical-location-Object                |
| LDAP-Display-Name | physicalLocationObject                  |
| Taille              | \-                                      |
| Mettre à jour le privilège  | \-                                      |
| Fréquence des mises à jour  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.514                  |
| System-ID-GUID    | b7b13119-b82e-11d0-afee-0000f80367c1    |
| Syntaxe            | [**Object(DS-DN)**](s-object-ds-dn.md) |
