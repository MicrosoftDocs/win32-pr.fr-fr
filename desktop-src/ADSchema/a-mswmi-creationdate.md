# <a name="ms-wmi-creationdate-attribute"></a>attribut ms-WMI-CreationDate

Date et heure de création d’une instance d’objet.



| Entrée | Valeur |
|-------------------|---------------------------------------------------|
| CN                | MS-WMI-CreationDate                               |
| LDAP-Display-Name | msWMI-CreationDate                                |
| Taille              | Représentation du type de données DateTime WMI-26 caractères. |
| Mettre à jour le privilège  | Cette valeur est définie par le système.                  |
| Fréquence des mises à jour  | \-                                                |
| Attribute-Id      | 1.2.840.113556.1.4.1626                           |
| System-ID-GUID    | 748b0a2e-3351-4b3f-b171-2f17414ea779              |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md)       |
