# <a name="owner-attribute"></a>Attribut propriétaire

Nom unique d’un objet qui a la propriété d’un objet.



| Entrée | Valeur |
|-------------------|-----------------------------------------|
| CN                | Propriétaire                                   |
| LDAP-Display-Name | propriétaire                                   |
| Taille              | \-                                      |
| Mettre à jour le privilège  | \-                                      |
| Fréquence des mises à jour  | \-                                      |
| Attribute-Id      | 2.5.4.32                                |
| System-ID-GUID    | bf9679f3-0de6-11d0-a285-00aa003049e2    |
| Syntaxe            | [**Object(DS-DN)**](s-object-ds-dn.md) |
