# <a name="presentation-address-attribute"></a>Attribut Presentation-Address

Spécifie une adresse de présentation associée à un objet qui représente une entité d’application OSI.



| Entrée | Valeur |
|-------------------|-----------------------------------------------------------------------|
| CN                | Presentation-Address                                                  |
| LDAP-Display-Name | presentationAddress                                                   |
| Taille              | \-                                                                    |
| Mettre à jour le privilège  | \-                                                                    |
| Fréquence des mises à jour  | \-                                                                    |
| Attribute-Id      | 2.5.4.29                                                              |
| System-ID-GUID    | a8df744b-c5ea-11d1-bbcb-0080c76670c0                                  |
| Syntaxe            | [**Object(Presentation-Address)**](s-object-presentation-address.md) |
