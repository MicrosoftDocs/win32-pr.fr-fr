# <a name="phone-home-other-attribute"></a>Téléphone-privé-autre attribut

Liste de numéros de téléphone personnels de remplacement.



| Entrée | Valeur |
|-------------------|----------------------------------------------------------------------------------|
| CN                | Téléphone-bureau-autre                                                                 |
| LDAP-Display-Name | otherHomePhone                                                                   |
| Taille              | \-                                                                               |
| Mettre à jour le privilège  | Administrateur de domaine ou propriétaire du compte.                                           |
| Fréquence des mises à jour  | Lorsque l’enregistrement de l’utilisateur est créé et chaque fois que le numéro de téléphone doit être modifié. |
| Attribute-Id      | 1.2.840.113556.1.2.277                                                           |
| System-ID-GUID    | f0f8ffa2-1191-11d0-a060-00aa006c33ed                                             |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md)                                      |
