# <a name="ipnetworknumber-attribute"></a>attribut ipNetworkNumber

Contient un numéro de réseau IP en notation décimale séparée par des points, en omettant les zéros non significatifs.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | ipNetworkNumber                      |
| LDAP-Display-Name | ipNetworkNumber                      |
| Taille              | \-                                   |
| Mettre à jour le privilège  | \-                                   |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.20                     |
| System-ID-GUID    | 4e3854f4-3087-42a4-a813-bb0c528958d3 |
| Syntaxe            | [**String(IA5)**](s-string-ia5.md)  |
