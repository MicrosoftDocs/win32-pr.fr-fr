# <a name="ms-ds-is-user-cachable-at-rodc-attribute"></a>attribut de contrôleur de domaine en lecture seule pour ms-DS-is-User-cachable-

Pour un contrôleur de domaine Read-Only (RODC), identifie si les secrets de l’utilisateur spécifié peuvent être mis en cache.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | ms-DS-is-User-cachable-at-RODC       |
| LDAP-Display-Name | msDS-IsUserCachableAtRodc            |
| Taille              | \-                                   |
| Mettre à jour le privilège  | \-                                   |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.2025              |
| System-ID-GUID    | fe01245a-341f-4556-951f-48c033a89050 |
| Syntaxe            | [**Enumeration**](s-enumeration.md) |
