# <a name="ms-pki-cert-template-oid-attribute"></a>attribut ms-PKI-cert-template-OID

Spécifie l’identificateur d’objet pour un modèle de certificat.



| Entrée | Valeur |
|-------------------|---------------------------------------------------------------------------------------------------|
| CN                | MS-PKI-cert-template-OID                                                                          |
| LDAP-Display-Name | Attribut mspki-cert-template-OID                                                                           |
| Taille              | 64 octets                                                                                          |
| Mettre à jour le privilège  | Administrateur de domaine                                                                              |
| Fréquence des mises à jour  | Lorsque l’objet modèle de certificat (MS-PKI-Certificate-template) est modifié, créé ou cloné. |
| Attribute-Id      | 1.2.840.113556.1.4.1436                                                                           |
| System-ID-GUID    | 3164c36a-ba26-468c-8bda-c1e5cc256728                                                              |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md)                                                       |
