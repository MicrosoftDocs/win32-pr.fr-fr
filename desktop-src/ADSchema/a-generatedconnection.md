# <a name="generated-connection-attribute"></a>Attribut Generated-Connection

**True** si cette connexion a été créée par la génération de la topologie automatique.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | Generated-Connection                 |
| LDAP-Display-Name | generatedConnection                  |
| Taille              | \-                                   |
| Mettre à jour le privilège  | \-                                   |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.41                |
| System-ID-GUID    | bf96797a-0de6-11d0-a285-00aa003049e2 |
| Syntaxe            | [**Boolean**](s-boolean.md)         |
