# <a name="msnpallowdialin-attribute"></a>attribut msNPAllowDialin

Indique si le compte a l’autorisation de se connecter au serveur RAS. Ne modifiez pas cette valeur directement. Utilisez la [fonction d’administration RAS](/windows/desktop/RRAS/ras-administration-functions) appropriée pour modifier cette valeur.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | msNPAllowDialin                      |
| LDAP-Display-Name | msNPAllowDialin                      |
| Taille              | 4 octets                              |
| Mettre à jour le privilège  | \-                                   |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1119              |
| System-ID-GUID    | db0c9085-c1f2-11d1-bbc5-0080c76670c0 |
| Syntaxe            | [**Boolean**](s-boolean.md)         |
