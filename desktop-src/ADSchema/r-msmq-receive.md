# <a name="msmq-receive-extended-right"></a>MSMQ-recevoir le droit étendu

Autorise la réception des messages de la file d’attente.

|--------------|--------------------------------------|
| CN           | MSMQ-réception                         |
| Display-Name | Recevoir un message                      |
| Rights-GUID  | 06bd3200-df3e-11d1-9c86-006008764d0e |
