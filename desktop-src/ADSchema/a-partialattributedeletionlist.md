# <a name="partial-attribute-deletion-list-attribute"></a>Attribut partial-attribute-suppression-liste

Effectue le suivi de l’état de réplication interne des réplicas partiels (autrement dit, sur des catalogues globaux). Attribut de l’objet de contexte de nommage de réplica partiel. Utilisé lorsque le garbage collector est en train de supprimer des attributs des objets dans ses NC de réplica partiels.



| Entrée | Valeur |
|-------------------|-------------------------------------------------------|
| CN                | Partial-attribute-suppression-liste                       |
| LDAP-Display-Name | partialAttributeDeletionList                          |
| Taille              | \-                                                    |
| Mettre à jour le privilège  | Cette valeur est définie par le système.                      |
| Fréquence des mises à jour  | Pendant la réplication                                    |
| Attribute-Id      | 1.2.840.113556.1.4.663                                |
| System-ID-GUID    | 28630ec0-41d5-11d1-a9c1-0000f80367c1                  |
| Syntaxe            | [**Object(Replica-Link)**](s-object-replica-link.md) |
