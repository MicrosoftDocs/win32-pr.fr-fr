# <a name="default-security-descriptor-attribute"></a>Attribut de descripteur de sécurité par défaut

Descripteur de sécurité à assigner à l’objet lors de sa création.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | Descripteur de sécurité par défaut                 |
| LDAP-Display-Name | defaultSecurityDescriptor                   |
| Taille              | \-                                          |
| Mettre à jour le privilège  | Administrateur de schéma                        |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.224                      |
| System-ID-GUID    | 807a6d30-1669-11d0-a064-00aa006c33ed        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
