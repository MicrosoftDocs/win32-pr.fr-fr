# <a name="domain-id-attribute"></a>Attribut de nom de domaine

Référence à un domaine associé à une autorité de certification.



| Entrée | Valeur |
|-------------------|-----------------------------------------|
| CN                | ID de domaine                               |
| LDAP-Display-Name | Domaine                                |
| Taille              | \-                                      |
| Mettre à jour le privilège  | \-                                      |
| Fréquence des mises à jour  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.686                  |
| System-ID-GUID    | 963d2734-48be-11d1-a9c3-0000f80367c1    |
| Syntaxe            | [**Object(DS-DN)**](s-object-ds-dn.md) |
