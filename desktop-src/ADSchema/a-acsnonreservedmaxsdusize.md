# <a name="acs-non-reserved-max-sdu-size-attribute"></a>ACS-attribut non réservé-Max-SDU-Size

L’attribut **ACS-non-reserved-Max-SDU-Size** est destiné à un usage interne uniquement. Basé sur RFC2814.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | ACS-non réservé-Max-SDU-Size        |
| LDAP-Display-Name | aCSNonReservedMaxSDUSize             |
| Taille              | 8 octets                              |
| Mettre à jour le privilège  | \-                                   |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1320              |
| System-ID-GUID    | aec2cfe3-3b90-11d2-90cc-00c04fd91ab1 |
| Syntaxe            | [**Défini**](s-interval.md)       |
