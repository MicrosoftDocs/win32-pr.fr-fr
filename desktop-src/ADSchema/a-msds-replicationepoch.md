# <a name="ms-ds-replicationepoch-attribute"></a>attribut ms-DS-ReplicationEpoch

Cela permet de conserver l’époque de réplication de tous les contrôleurs de domaines. Une époque est la période pendant laquelle un domaine a un nom spécifique. Une nouvelle époque commence lorsqu’un changement de nom de domaine se produit.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | ms-DS-ReplicationEpoch               |
| LDAP-Display-Name | msDS-ReplicationEpoch                |
| Taille              | \-                                   |
| Mettre à jour le privilège  | Cette valeur est définie par le système.     |
| Fréquence des mises à jour  | Uniquement pendant la restructuration de domaine.      |
| Attribute-Id      | 1.2.840.113556.1.4.1720              |
| System-ID-GUID    | 08e3aa79-eb1c-45b5-af7b-8f94246c8e41 |
| Syntaxe            | [**Énumération**](s-enumeration.md) |
