# <a name="categories-attribute"></a>Attribut Categories

Liste d’ID de catégorie (GUID) pour les catégories qui s’appliquent à cette application.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | Catégories                                  |
| LDAP-Display-Name | categories                                  |
| Taille              | \-                                          |
| Mettre à jour le privilège  | \-                                          |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.672                      |
| System-ID-GUID    | 7bfdcb7e-4807-11d1-a9c3-0000f80367c1        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
