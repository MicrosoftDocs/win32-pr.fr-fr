| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | ms-DS-AZ-Last-importée-biz-Rule-Path        |
| LDAP-Display-Name | msDS-AzLastImportedBizRulePath              |
| Taille              | 128 caractères                              |
| Mettre à jour le privilège  | Administrateur AzRoles                               |
| Fréquence des mises à jour  | Pendant l’initialisation ou la modification de la stratégie.     |
| Attribute-Id      | 1.2.840.113556.1.4.1803                     |
| System-ID-GUID    | 665acb5c-bb92-4dbc-8c59-b3638eab09b3        |
