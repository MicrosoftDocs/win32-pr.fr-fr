# <a name="frs-replica-set-type-attribute"></a>Attribut FRS-Replica-Set-type

Il s’agit d’un code qui indique s’il s’agit d’un jeu de réplicas Sysvol, d’un jeu de réplicas DFS ou d’un autre jeu de réplicas.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | FRS-Replica-Set-type                 |
| LDAP-Display-Name | fRSReplicaSetType                    |
| Taille              | \-                                   |
| Mettre à jour le privilège  | Cette valeur est définie par le système.     |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.31                |
| System-ID-GUID    | 26d9736b-6070-11d1-a9c6-0000f80367c1 |
| Syntaxe            | [**Enumeration**](s-enumeration.md) |
