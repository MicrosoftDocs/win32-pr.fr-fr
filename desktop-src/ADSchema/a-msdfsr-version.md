# <a name="ms-dfsr-version-attribute"></a>MS-DFSR-version (attribut)

Contient le numéro de version du service de réplication système de fichiers DFS (DFS).



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | MS-DFSR-version                             |
| LDAP-Display-Name | msDFSR-version                              |
| Taille              | \-                                          |
| Mettre à jour le privilège  | \-                                          |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.6.13.3.1                   |
| System-ID-GUID    | 1a861408-38c3-49ea-ba75-85481a77c655        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
