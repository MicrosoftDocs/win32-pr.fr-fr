# <a name="ms-tapi-conference-blob-attribute"></a>attribut ms-TAPI-Conference-BLOB

BLOB de données binaire qui décrit les différentes propriétés d’une conférence de multidiffusion TAPI. Son format et son contenu sont déterminés par la valeur de l’attribut Protocol-Id dans le même objet. En règle générale, les données de cet objet BLOB sont conformes à RFC2327.



| Entrée | Valeur |
|-------------------|--------------------------------------------------------------------------------------------------------------|
| CN                | MS-TAPI-Conference-BLOB                                                                                      |
| LDAP-Display-Name | msTAPI-ConferenceBlob                                                                                        |
| Taille              | Peut être de longueur arbitraire.                                                                                  |
| Mettre à jour le privilège  | Aucun privilège spécial n’est requis.                                                                              |
| Fréquence des mises à jour  | Peut changer à la discrétion d’un propriétaire de conférence TAPI lorsque certaines données relatives à la conférence doivent être modifiées. |
| Attribute-Id      | 1.2.840.113556.1.4.1700                                                                                      |
| System-ID-GUID    | 4cc4601e-7201-4141-abc8-3e529ae88863                                                                         |
| Syntaxe            | [**Object(Replica-Link)**](s-object-replica-link.md)                                                        |
