# <a name="move-tree-state-attribute"></a>Attribut déplacement-Tree-State

Cet attribut contient des informations d’État pour une arborescence de répertoires qui est déplacée.



| Entrée | Valeur |
|-------------------|-------------------------------------------------------|
| CN                | Déplacer-arborescence-État                                       |
| LDAP-Display-Name | moveTreeState                                         |
| Taille              | \-                                                    |
| Mettre à jour le privilège  | \-                                                    |
| Fréquence des mises à jour  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.1305                               |
| System-ID-GUID    | 1f2ac2c8-3b71-11d2-90cc-00c04fd91ab1                  |
| Syntaxe            | [**Object(Replica-Link)**](s-object-replica-link.md) |
