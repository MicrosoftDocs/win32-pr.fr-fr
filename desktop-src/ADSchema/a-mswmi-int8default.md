# <a name="ms-wmi-int8default-attribute"></a>attribut ms-WMI-int8Default

Valeur par défaut pour les objets de paramètre entier WMI 64 bits.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | MS-WMI-int8Default                   |
| LDAP-Display-Name | msWMI-Int8Default                    |
| Taille              | 8 octets                              |
| Mettre à jour le privilège  | Administrateur stratégie de groupe           |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1632              |
| System-ID-GUID    | f4d8085a-8c5b-4785-959b-dc585566e445 |
| Syntaxe            | [**Défini**](s-interval.md)       |
