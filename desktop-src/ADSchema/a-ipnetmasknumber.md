# <a name="ipnetmasknumber-attribute"></a>attribut ipNetmaskNumber

Contient le masque réseau IP en notation décimale séparée par des points, en omettant les zéros non significatifs.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | ipNetmaskNumber                      |
| LDAP-Display-Name | ipNetmaskNumber                      |
| Taille              | \-                                   |
| Mettre à jour le privilège  | \-                                   |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.21                     |
| System-ID-GUID    | 6ff64fcd-462e-4f62-b44a-9a5347659eb9 |
| Syntaxe            | [**String(IA5)**](s-string-ia5.md)  |
