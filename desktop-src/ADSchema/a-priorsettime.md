# <a name="prior-set-time-attribute"></a>Attribut d’heure définie antérieure

Heure précédente définie pour une clé secrète.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | Au préalable-à l’heure définie                       |
| LDAP-Display-Name | priorSetTime                         |
| Taille              | 8 octets                              |
| Mettre à jour le privilège  | \-                                   |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.99                |
| System-ID-GUID    | bf967a01-0de6-11d0-a285-00aa003049e2 |
| Syntaxe            | [**Défini**](s-interval.md)       |
