# <a name="query-policy-bl-attribute"></a>Attribut de stratégie de requête-BL

Liste de tous les objets contenant des références à une stratégie de requête donnée.



| Entrée | Valeur |
|-------------------|-----------------------------------------|
| CN                | Requête-stratégie-BL                         |
| LDAP-Display-Name | queryPolicyBL                           |
| Taille              | \-                                      |
| Mettre à jour le privilège  | Cette valeur est définie par le système.        |
| Fréquence des mises à jour  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.608                  |
| System-ID-GUID    | e1aea404-cd5b-11d0-afff-0000f80367c1    |
| Syntaxe            | [**Object(DS-DN)**](s-object-ds-dn.md) |
