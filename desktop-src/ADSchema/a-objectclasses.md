# <a name="object-classes-attribute"></a>Attribut Object-Classes

Propriété à valeurs multiples qui contient des chaînes représentant chaque classe dans le schéma. Chaque valeur contient les valeurs governsID, lDAPDisplayName, mustContain, mayContain, etc.



| Entrée | Valeur |
|-------------------|--------------------------------------------------|
| CN                | Object-Classes                                   |
| LDAP-Display-Name | objectClasses                                    |
| Taille              | Environ 20 octets en moyenne par nom de classe.        |
| Mettre à jour le privilège  | Le concepteur de l’objet définit cette valeur. |
| Fréquence des mises à jour  | Chaque fois que la conception de la classe change.        |
| Attribute-Id      | 2.5.21.6                                         |
| System-ID-GUID    | 9a7ad94b-ca53-11d1-bbd0-0080c76670c0             |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md)      |
