# <a name="gp-options-attribute"></a>Attribut GP-Options

Options qui affectent toutes les stratégies de groupe associées à l’objet qui héberge cette propriété.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | GP-Options                           |
| LDAP-Display-Name | gPOptions                            |
| Taille              | \-                                   |
| Mettre à jour le privilège  | \-                                   |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.892               |
| System-ID-GUID    | f30e3bbf-9ff0-11d1-b603-0000f80367c1 |
| Syntaxe            | [**Énumération**](s-enumeration.md) |
