# <a name="www-page-other-attribute"></a>Page Web-autre attribut

Liste de pages Web de remplacement.



| Entrée | Valeur |
|-------------------|-----------------------------------------------------------------------------|
| CN                | WWW-page-autres                                                              |
| LDAP-Display-Name | url                                                                         |
| Taille              | \-                                                                          |
| Mettre à jour le privilège  | Administrateur de domaine ou propriétaire du compte.                                      |
| Fréquence des mises à jour  | Lorsque l’enregistrement de l’utilisateur est créé et chaque fois que la page Web doit être modifiée. |
| Attribute-Id      | 1.2.840.113556.1.4.749                                                      |
| System-ID-GUID    | 9a9a0221-4a5b-11d1-a9c3-0000f80367c1                                        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md)                                 |
