# <a name="trust-partner-attribute"></a>Attribut Trust-Partner

Nom du domaine avec lequel une approbation existe.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | Trust-Partner                               |
| LDAP-Display-Name | trustPartner                                |
| Taille              | \-                                          |
| Mettre à jour le privilège  | Cette valeur est définie par le système.            |
| Fréquence des mises à jour  | Lors de la création d’une approbation.                |
| Attribute-Id      | 1.2.840.113556.1.4.133                      |
| System-ID-GUID    | bf967a5d-0de6-11d0-a285-00aa003049e2        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
