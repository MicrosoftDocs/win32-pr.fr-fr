# <a name="ms-wmi-propertyname-attribute"></a>attribut ms-WMI-PropertyName

Nom de l’objet de stratégie cible d’un objet de paramètre.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | MS-WMI-PropertyName                         |
| LDAP-Display-Name | msWMI-PropertyName                          |
| Taille              | Moins de vingt caractères.                |
| Mettre à jour le privilège  | Administrateur stratégie de groupe                  |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1641                     |
| System-ID-GUID    | ab920883-e7f8-4d72-b4a0-c0449897509d        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
