# <a name="ms-ds-schema-extensions-attribute"></a>attribut ms-DS-Schema-extensions

Objet BLOB binaire utilisé pour stocker des informations sur les extensions aux objets de schéma.



| Entrée | Valeur |
|-------------------|-------------------------------------------------------|
| CN                | ms-DS-Schema-extensions                               |
| LDAP-Display-Name | msDs-Schema-extensions                                |
| Taille              | \-                                                    |
| Mettre à jour le privilège  | Cette valeur est définie par le système.                      |
| Fréquence des mises à jour  | Lorsque le schéma est étendu.                          |
| Attribute-Id      | 1.2.840.113556.1.4.1440                               |
| System-ID-GUID    | b39a61be-ed07-4cab-9a4a-4963ed0141e1                  |
| Syntaxe            | [**Object(Replica-Link)**](s-object-replica-link.md) |
