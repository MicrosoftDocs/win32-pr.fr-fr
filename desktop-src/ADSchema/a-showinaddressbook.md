# <a name="show-in-address-book-attribute"></a>Show-in-Address-attribute Book

Cet attribut est utilisé pour indiquer dans quels carnets d’adresses MAPI un objet s’affiche. elle est généralement gérée par le service de mise à jour de destinataire Exchange.



| Entrée | Valeur |
|-------------------|-----------------------------------------|
| CN                | Afficher dans l’adresse-livre                    |
| LDAP-Display-Name | showInAddressBook                       |
| Taille              | \-                                      |
| Mettre à jour le privilège  | Utilisé par le système.             |
| Fréquence des mises à jour  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.644                  |
| System-ID-GUID    | 3e74f60e-3e73-11d1-a9c0-0000f80367c1    |
| Syntaxe            | [**Object(DS-DN)**](s-object-ds-dn.md) |
