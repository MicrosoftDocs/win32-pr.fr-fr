# <a name="print-stapling-supported-attribute"></a>Print-agrafeuse-attribut pris en charge

**True** si l’imprimante prend en charge l’agrafage. Fourni par le pilote.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | Impression-agrafage-pris en charge             |
| LDAP-Display-Name | printStaplingSupported               |
| Taille              | 4 octets                              |
| Mettre à jour le privilège  | \-                                   |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.281               |
| System-ID-GUID    | ba305f73-47e3-11d0-a1a6-00c04fd930c9 |
| Syntaxe            | [**Boolean**](s-boolean.md)         |
