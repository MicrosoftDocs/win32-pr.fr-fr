# <a name="bridgehead-server-list-bl-attribute"></a>Pont-serveur-List-BL (attribut)

Liste des serveurs qui sont des serveurs ponts pour la réplication.



| Entrée | Valeur |
|-------------------|-----------------------------------------|
| CN                | Tête de pont-serveur-liste-BL               |
| LDAP-Display-Name | bridgeheadServerListBL                  |
| Taille              | \-                                      |
| Mettre à jour le privilège  | Cette valeur est définie par le système.        |
| Fréquence des mises à jour  | Chaque fois qu’un site est configuré.               |
| Attribute-Id      | 1.2.840.113556.1.4.820                  |
| System-ID-GUID    | d50c2cdb-8951-11d1-aebc-0000f80367c1    |
| Syntaxe            | [**Object(DS-DN)**](s-object-ds-dn.md) |
