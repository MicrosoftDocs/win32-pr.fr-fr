# <a name="netboot-sif-file-attribute"></a>Attribut netboot-SIF-file

L’attribut **netboot-SIF-file** est réservé à un usage interne.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | Netboot-SIF-fichier                            |
| LDAP-Display-Name | netbootSIFFile                              |
| Taille              | \-                                          |
| Mettre à jour le privilège  | Cette valeur est définie par le système.            |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1240                     |
| System-ID-GUID    | 2df90d84-009f-11d2-aa4c-00c04fd7d83a        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
