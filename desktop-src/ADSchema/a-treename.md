# <a name="tree-name-attribute"></a>Attribut Tree-Name

Nom DNS du domaine à la racine d’une arborescence.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | Tree-Name                                   |
| LDAP-Display-Name | treeName                                    |
| Taille              | \-                                          |
| Mettre à jour le privilège  | \-                                          |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.660                      |
| System-ID-GUID    | 28630ebd-41d5-11d1-a9c1-0000f80367c1        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
