# <a name="pki-enrollment-access-attribute"></a>Infrastructure à clé publique-inscription-attribut d’accès

L’attribut d’accès à l’inscription de l' **infrastructure à clé publique** est destiné à un usage interne uniquement.



| Entrée | Valeur |
|-------------------|-----------------------------------------------------|
| CN                | Infrastructure à clé publique-inscription-accès                               |
| LDAP-Display-Name | pKIEnrollmentAccess                                 |
| Taille              | \-                                                  |
| Mettre à jour le privilège  | \-                                                  |
| Fréquence des mises à jour  | \-                                                  |
| Attribute-Id      | 1.2.840.113556.1.4.1335                             |
| System-ID-GUID    | 926be278-56f9-11d2-90d0-00c04fd91ab1                |
| Syntaxe            | [**String(NT-Sec-Desc)**](s-string-nt-sec-desc.md) |
