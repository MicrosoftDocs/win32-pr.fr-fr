# <a name="initials-attribute"></a>Initials (attribut)

Contient les initiales des parties du nom complet de l’utilisateur. Cela peut être utilisé comme initiale du deuxième prénom dans le carnet d’adresses Windows.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | Initiales                                    |
| LDAP-Display-Name | Initials                                    |
| Taille              | \-                                          |
| Mettre à jour le privilège  | Administrateur de domaine ou propriétaire du compte.      |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 2.5.4.43                                    |
| System-ID-GUID    | f0f8ff90-1191-11d0-a060-00aa006c33ed        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
