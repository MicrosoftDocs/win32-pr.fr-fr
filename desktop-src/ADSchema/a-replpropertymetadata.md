# <a name="repl-property-meta-data-attribute"></a>Attribut REPL-Property-Meta-Data

Effectue le suivi des informations d’état de réplication interne pour les objets DS. Les informations fournies ici peuvent être extraites sous forme publique via l’API publique DsReplicaGetInfo (). Présent sur tous les objets DS.



| Entrée | Valeur |
|-------------------|------------------------------------------------------------------------------|
| CN                | REPL-Property-Meta-Data                                                      |
| LDAP-Display-Name | replPropertyMetaData                                                         |
| Taille              | La longueur est proportionnelle au nombre d’attributs réplicables sur l’objet. |
| Mettre à jour le privilège  | Cette valeur est définie par le système.                                             |
| Fréquence des mises à jour  | Modifications en réponse à des modifications de réplication dans l’objet.                |
| Attribute-Id      | 1.2.840.113556.1.4.3                                                         |
| System-ID-GUID    | 281416c0-1968-11d0-a28f-00aa003049e2                                         |
| Syntaxe            | [**Object(Replica-Link)**](s-object-replica-link.md)                        |
