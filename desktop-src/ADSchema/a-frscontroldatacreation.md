# <a name="frs-control-data-creation-attribute"></a>Attribut FRS-Control-Data-creation

Paire de niveau d’avertissement/erreur pour la création de données de fichier (mégaoctets par seconde).



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | FRS-Control-Data-creation                   |
| LDAP-Display-Name | fRSControlDataCreation                      |
| Taille              | \-                                          |
| Mettre à jour le privilège  | \-                                          |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.871                      |
| System-ID-GUID    | 2a13257a-9373-11d1-aebc-0000f80367c1        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
