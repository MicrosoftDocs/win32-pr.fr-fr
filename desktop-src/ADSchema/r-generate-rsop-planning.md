# <a name="generate-rsop-planning-extended-right"></a>Générer un droit étendu de planification RSoP

L’utilisateur qui dispose des droits sur une unité d’organisation ou un domaine est en mesure de générer des données RSoP en mode de planification pour les utilisateurs ou les ordinateurs au sein de l’unité d’organisation.



| Entrée | Valeur |
|--------------|---------------------------------------------|
| CN           | Générer-stratégie RSoP-planification                      |
| Display-Name | Générer le Jeu de stratégie résultant (Planification) |
| Rights-GUID  | b7b1b3dd-ab09-4242-9e30-9980e5d322f7        |
