# <a name="locality-name-attribute"></a>Attribut Locality-Name

Représente le nom d’une localité, telle qu’une ville ou une ville.



| Entrée | Valeur |
|-------------------|------------------------------------------------------------------------------|
| CN                | Locality-Name                                                                |
| LDAP-Display-Name | l                                                                            |
| Taille              | \-                                                                           |
| Mettre à jour le privilège  | Administrateur de domaine ou propriétaire du compte.                                       |
| Fréquence des mises à jour  | Lorsque l’enregistrement de l’utilisateur est créé et chaque fois que la localité doit être modifiée. |
| Attribute-Id      | 2.5.4.7                                                                      |
| System-ID-GUID    | bf9679a2-0de6-11d0-a285-00aa003049e2                                         |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md)                                  |
