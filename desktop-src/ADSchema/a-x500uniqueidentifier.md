# <a name="x500uniqueidentifier-attribute"></a>attribut x500uniqueIdentifier

Utilisé pour distinguer les objets lorsqu’un nom unique a été réutilisé. Il s’agit d’un type d’attribut différent des types UID et uniqueIdentifier.



| Entrée | Valeur |
|-------------------|-------------------------------------------------------|
| CN                | x500uniqueIdentifier                                  |
| LDAP-Display-Name | x500uniqueIdentifier                                  |
| Taille              | \-                                                    |
| Mettre à jour le privilège  | \-                                                    |
| Fréquence des mises à jour  | \-                                                    |
| Attribute-Id      | 2.5.4.45                                              |
| System-ID-GUID    | d07da11f-8a3d-42b6-b0aa-76c962be719a                  |
| Syntaxe            | [**Object(Replica-Link)**](s-object-replica-link.md) |
