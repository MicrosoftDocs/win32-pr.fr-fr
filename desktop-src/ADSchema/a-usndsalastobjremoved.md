# <a name="usn-dsa-last-obj-removed-attribute"></a>USN-DSA-Last-obj-attribut supprimé

Contient le numéro de séquence de mise à jour (USN) pour le dernier objet système qui a été supprimé d’un serveur.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | USN-DSA-Last-obj-supprimé             |
| LDAP-Display-Name | uSNDSALastObjRemoved                 |
| Taille              | 8 octets                              |
| Mettre à jour le privilège  | Cette valeur est définie par le système.     |
| Fréquence des mises à jour  | À chaque modification d’un objet d’annuaire. |
| Attribute-Id      | 1.2.840.113556.1.2.267               |
| System-ID-GUID    | bf967a71-0de6-11d0-a285-00aa003049e2 |
| Syntaxe            | [**Défini**](s-interval.md)       |
