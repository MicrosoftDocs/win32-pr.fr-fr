# <a name="ms-exch-owner-bl-attribute"></a>attribut ms-Exch-owner-BL

Lien vers l’arrière vers l’attribut propriétaire. Contient une liste de propriétaires pour un objet.



| Entrée | Valeur |
|-------------------|-----------------------------------------|
| CN                | ms-Exch-owner-BL                        |
| LDAP-Display-Name | ownerBL                                 |
| Taille              | \-                                      |
| Mettre à jour le privilège  | Cette valeur est définie par le système.        |
| Fréquence des mises à jour  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.2.104                  |
| System-ID-GUID    | bf9679f4-0de6-11d0-a285-00aa003049e2    |
| Syntaxe            | [**Object(DS-DN)**](s-object-ds-dn.md) |
