# <a name="department-attribute"></a>Attribut de service

Contient le nom du service dans lequel l’utilisateur travaille.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | department                                  |
| LDAP-Display-Name | department                                  |
| Taille              | \-                                          |
| Mettre à jour le privilège  | Administrateur de domaine ou propriétaire du compte.      |
| Fréquence des mises à jour  | Chaque fois que le service doit être modifié.    |
| Attribute-Id      | 1.2.840.113556.1.2.141                      |
| System-ID-GUID    | bf96794f-0de6-11d0-a285-00aa003049e2        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
