# <a name="ms-ds-executescriptpassword-attribute"></a>attribut ms-DS-ExecuteScriptPassword

Utilisé lors du changement de nom de domaine. Cette valeur ne peut pas être écrite ou lue à l’aide de LDAP.



| Entrée | Valeur |
|-------------------|-------------------------------------------------------|
| CN                | ms-DS-ExecuteScriptPassword                           |
| LDAP-Display-Name | msDS-ExecuteScriptPassword                            |
| Taille              | \-                                                    |
| Mettre à jour le privilège  | Cette valeur est définie par le système.                      |
| Fréquence des mises à jour  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.1783                               |
| System-ID-GUID    | 9d054a5a-d187-46c1-9d85-42dfc44a56dd                  |
| Syntaxe            | [**Object(Replica-Link)**](s-object-replica-link.md) |
