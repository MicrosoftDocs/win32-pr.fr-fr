# <a name="msradiusservicetype-attribute"></a>attribut msRADIUSServiceType

L’attribut **msRADIUSServiceType** est utilisé en interne. Ne modifiez pas cette valeur directement.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | msRADIUSServiceType                  |
| LDAP-Display-Name | msRADIUSServiceType                  |
| Taille              | 4 octets                              |
| Mettre à jour le privilège  | \-                                   |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1171              |
| System-ID-GUID    | db0c90b6-c1f2-11d1-bbc5-0080c76670c0 |
| Syntaxe            | [**Enumeration**](s-enumeration.md) |
