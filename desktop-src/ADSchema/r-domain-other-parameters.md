# <a name="domain-other-parameters-property-set"></a>Propriété de domaine-autres paramètres définie

Jeu de propriétés autorisant le contrôle à une liste d’attributs de domaine.



| Entrée | Valeur |
|--------------|------------------------------------------|
| CN           | Domaine-autres-paramètres                  |
| Display-Name | Autres paramètres de domaine (pour une utilisation par SAM) |
| Rights-GUID  | b8119fd0-04f6-4762-ab7a-4986c76b3f9a     |
