# <a name="ms-ds-non-members-bl-attribute"></a>ms-DS-non-members-BL Attribute

Lien vers l’arrière du groupe ou de l’utilisateur non membre vers les groupes qui y sont liés (mêmes fonctionnalités que non-Security-Member-BL).



| Entrée | Valeur |
|-------------------|-----------------------------------------|
| CN                | ms-DS-non-membres-BL                    |
| LDAP-Display-Name | msDS-NonMembersBL                       |
| Taille              | \-                                      |
| Mettre à jour le privilège  | Cette valeur est définie par le système.        |
| Fréquence des mises à jour  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1794                 |
| System-ID-GUID    | 2a8c68fc-3a7a-4e87-8720-fe77c51cbe74    |
| Syntaxe            | [**Object(DS-DN)**](s-object-ds-dn.md) |
