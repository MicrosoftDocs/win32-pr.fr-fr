# <a name="ms-wmi-name-attribute"></a>MS-WMI-attribut Name

Nom complet des objets de stratégie de niveau supérieur. Utilisé dans le catalogue global.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | MS-WMI-nom                                 |
| LDAP-Display-Name | msWMI-nom                                  |
| Taille              | \-                                          |
| Mettre à jour le privilège  | Administrateur stratégie de groupe                  |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1639                     |
| System-ID-GUID    | c6c8ace5-7e81-42af-ad72-77412c5941c4        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
