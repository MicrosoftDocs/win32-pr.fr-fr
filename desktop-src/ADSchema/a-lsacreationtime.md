# <a name="lsa-creation-time-attribute"></a>LSA-attribut de date/heure de création

L’attribut **LSA-Creation-Time** est utilisé pour prendre en charge la réplication vers Windows domaines NT 4,0.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | LSA-temps de création                    |
| LDAP-Display-Name | lSACreationTime                      |
| Taille              | 8 octets                              |
| Mettre à jour le privilège  | Cette valeur est définie par le système.     |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.66                |
| System-ID-GUID    | bf9679ad-0de6-11d0-a285-00aa003049e2 |
| Syntaxe            | [**Défini**](s-interval.md)       |
