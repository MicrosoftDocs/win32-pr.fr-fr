# <a name="ms-ds-auxiliary-classes-attribute"></a>attribut ms-DS-auxiliaire-classes

Cet attribut répertorie les classes auxiliaires qui ont été attachées dynamiquement à un objet. Cet attribut n’est pas associé à une classe. Elle est automatiquement renseignée par le système.



| Entrée | Valeur |
|-------------------|-----------------------------------------------------------------|
| CN                | ms-DS-auxiliaire-classes                                         |
| LDAP-Display-Name | msDS-auxiliaire-classes                                          |
| Taille              | \-                                                              |
| Mettre à jour le privilège  | Cette valeur est définie par le système.                                |
| Fréquence des mises à jour  | \-                                                              |
| Attribute-Id      | 1.2.840.113556.1.4.1458                                         |
| System-ID-GUID    | c4af1073-ee50-4be0-b8c0-89a41fe99abe                            |
| Syntaxe            | [**String(Object-Identifier)**](s-string-object-identifier.md) |



## <a name="implementations"></a>Implémentations

-   [**Windows Server 2003**](#windows-server-2003)
-   [**ADAM**](#adam)
-   [**Windows Server 2003 R2**](#windows-server-2003-r2)
-   [**Windows Server 2008**](#windows-server-2008)
-   [**Windows Server 2008 R2**](#windows-server-2008-r2)
