# <a name="ds-ui-shell-maximum-attribute"></a>DS-UI-Shell-attribut maximal

Il s’agit du nombre maximal par défaut d’objets à afficher dans un conteneur par l’interface utilisateur de l’interpréteur de commandes.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | DS-UI-Shell-maximum                  |
| LDAP-Display-Name | dSUIShellMaximum                     |
| Taille              | \-                                   |
| Mettre à jour le privilège  | Administrateur de domaine                 |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1345              |
| System-ID-GUID    | fcca766a-6f91-11d2-9905-0000f87a57d4 |
| Syntaxe            | [**Enumeration**](s-enumeration.md) |
