Il s’agit de la stratégie extensible utilisateur à répliquer sur les clients.

|-------------------|-------------------------------------------------------|
| CN                | Stratégie à l’ensemble du domaine                                    |
| LDAP-Display-Name | domainWidePolicy                                      |
| Taille              | \-                                                    |
| Mettre à jour le privilège  | \-                                                    |
| Fréquence des mises à jour  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.421                                |
| System-ID-GUID    | 80a67e29-9f22-11d0-afdd-00c04fd930c9                  |
| Syntaxe            | [**Object(Replica-Link)**](s-object-replica-link.md) |
