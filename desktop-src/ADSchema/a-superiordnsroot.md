# <a name="superior-dns-root-attribute"></a>Attribut Root-DNS supérieur

Il s’agit d’un attribut système utilisé pour la génération de références.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | Supérieure-DNS-racine                           |
| LDAP-Display-Name | superiorDNSRoot                             |
| Taille              | \-                                          |
| Mettre à jour le privilège  | Cette valeur est définie par le système.            |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.532                      |
| System-ID-GUID    | 5245801d-ca6a-11d0-afff-0000f80367c1        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
