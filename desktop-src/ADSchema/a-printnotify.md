# <a name="print-notify-attribute"></a>Attribut Print-Notify

Chaîne fournie par l’utilisateur qui spécifie le contact de notification.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | Print-Notify                                |
| LDAP-Display-Name | printNotify                                 |
| Taille              | \-                                          |
| Mettre à jour le privilège  | \-                                          |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.272                      |
| System-ID-GUID    | ba305f6a-47e3-11d0-a1a6-00c04fd930c9        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
