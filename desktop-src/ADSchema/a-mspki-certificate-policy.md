# <a name="ms-pki-certificate-policy-attribute"></a>attribut ms-PKI-Certificate-Policy

Contient la liste des OID de stratégie et leurs CSP facultatifs dans le certificat émis.



| Entrée | Valeur |
|-------------------|---------------------------------------------------------------------------------------------------|
| CN                | MS-PKI-certificat-stratégie                                                                         |
| LDAP-Display-Name | Attribut mspki-certificat-Policy                                                                          |
| Taille              | 64 octets multiplié par le nombre d’entrées.                                                             |
| Mettre à jour le privilège  | Administrateur de domaine                                                                              |
| Fréquence des mises à jour  | Lorsque l’objet modèle de certificat (MS-PKI-Certificate-template) est modifié, créé ou cloné. |
| Attribute-Id      | 1.2.840.113556.1.4.1439                                                                           |
| System-ID-GUID    | 38942346-cc5b-424B-a7d8-6ffd12029c5f                                                              |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md)                                                       |
