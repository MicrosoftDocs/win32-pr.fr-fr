# <a name="phone-fax-other-attribute"></a>Téléphone-télécopie-autre attribut

Liste de numéros de télécopie de substitution.



| Entrée | Valeur |
|-------------------|----------------------------------------------------------------------------------|
| CN                | Téléphone-télécopie-autre                                                                  |
| LDAP-Display-Name | otherFacsimileTelephoneNumber                                                    |
| Taille              | \-                                                                               |
| Mettre à jour le privilège  | Administrateur de domaine ou propriétaire du compte.                                           |
| Fréquence des mises à jour  | Lorsque l’enregistrement de l’utilisateur est créé et chaque fois que le numéro de téléphone doit être modifié. |
| Attribute-Id      | 1.2.840.113556.1.4.646                                                           |
| System-ID-GUID    | 0296c11d-40da-11d1-a9c0-0000f80367c1                                             |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md)                                      |
