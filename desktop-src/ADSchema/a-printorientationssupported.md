# <a name="print-orientations-supported-attribute"></a>Impression-orientations-attribut pris en charge

Rotation de page pour l’impression en mode paysage.



| Entrée | Valeur |
|-------------------|-------------------------------------------------------------|
| CN                | Impression-orientations-pris en charge                                |
| LDAP-Display-Name | printOrientationsSupported                                  |
| Taille              | 4 octets. Valeurs possibles : 0, 90, 270 et 0 = pas de paysage. |
| Mettre à jour le privilège  | \-                                                          |
| Fréquence des mises à jour  | \-                                                          |
| Attribute-Id      | 1.2.840.113556.1.4.240                                      |
| System-ID-GUID    | 281416d0-1968-11d0-a28f-00aa003049e2                        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md)                 |
