# <a name="ms-ds-non-security-group-extra-classes-attribute"></a>ms-DS-non-Security-Group-attribut extra-classes

Noms communs des classes non standard qui peuvent être ajoutées à un groupe non sécurisé via le composant logiciel enfichable utilisateurs et ordinateurs Active Directory.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | ms-DS-non-Security-Group-extra-classes      |
| LDAP-Display-Name | msDS-non-Security-Group-extra-classes       |
| Taille              | \-                                          |
| Mettre à jour le privilège  | Administrateur de domaine                        |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1689                     |
| System-ID-GUID    | 2de144fc-1f52-486f-bdf4-16fcc3084e54        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
