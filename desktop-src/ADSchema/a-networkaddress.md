# <a name="network-address-attribute"></a>Attribut Network-Address

Adresse TCP/IP d’un segment de réseau. Également appelée adresse de sous-réseau.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | Network-Address                             |
| LDAP-Display-Name | networkAddress                              |
| Taille              | \-                                          |
| Mettre à jour le privilège  | \-                                          |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.2.459                      |
| System-ID-GUID    | bf9679d9-0de6-11d0-a285-00aa003049e2        |
| Syntaxe            | [**String(Teletex)**](s-string-teletex.md) |
