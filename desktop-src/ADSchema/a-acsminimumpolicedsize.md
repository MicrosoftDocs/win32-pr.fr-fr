# <a name="acs-minimum-policed-size-attribute"></a>ACS-attribut de taille minimale de police

L’attribut **ACS-minimum-policiers-Size** est destiné à un usage interne uniquement. Basé sur RFC2210.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | ACS-taille minimale de la police             |
| LDAP-Display-Name | aCSMinimumPolicedSize                |
| Taille              | 8 octets                              |
| Mettre à jour le privilège  | \-                                   |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1315              |
| System-ID-GUID    | 8d0e7195-3b90-11d2-90cc-00c04fd91ab1 |
| Syntaxe            | [**Défini**](s-interval.md)       |
