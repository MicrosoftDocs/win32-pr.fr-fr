# <a name="time-refresh-attribute"></a>Attribut Time-Refresh

Cet attribut présente l’intervalle pendant lequel un enregistrement de ressource contenu dans une zone intégrée Active Directory doit être actualisé pour le serveur DNS. L’intervalle par défaut est de 7 jours.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | Time-Refresh                         |
| LDAP-Display-Name | timeRefresh                          |
| Taille              | \-                                   |
| Mettre à jour le privilège  | \-                                   |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.503               |
| System-ID-GUID    | ddac0cf1-af8f-11d0-afeb-00c04fd930c9 |
| Syntaxe            | [**Défini**](s-interval.md)       |
