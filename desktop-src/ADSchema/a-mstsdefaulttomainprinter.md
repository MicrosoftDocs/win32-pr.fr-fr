# <a name="ms-ts-default-to-main-printer-attribute"></a>attribut ms-TS-default-to-main-Printer

La valeur par défaut des services Terminal Server est l’imprimante principale spécifie s’il faut imprimer automatiquement sur l’imprimante par défaut du client.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | MS-TS-par défaut-à-imprimante        |
| LDAP-Display-Name | msTSDefaultToMainPrinter             |
| Taille              | \-                                   |
| Mettre à jour le privilège  | \-                                   |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1988              |
| System-ID-GUID    | c0ffe2bd-cacf-4dc7-88d5-61e9e95766f6 |
| Syntaxe            | [**Boolean**](s-boolean.md)         |
