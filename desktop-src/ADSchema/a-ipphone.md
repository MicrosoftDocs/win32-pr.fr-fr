# <a name="phone-ip-primary-attribute"></a>Téléphone-IP-attribut principal

Adresse TCP/IP du téléphone. Utilisé par la téléphonie.



| Entrée | Valeur |
|-------------------|----------------------------------------------------------------------------------|
| CN                | Téléphone-IP-principal                                                                 |
| LDAP-Display-Name | ipPhone                                                                          |
| Taille              | \-                                                                               |
| Mettre à jour le privilège  | Administrateur de domaine ou propriétaire du compte.                                           |
| Fréquence des mises à jour  | Lorsque l’enregistrement de l’utilisateur est créé et chaque fois que le numéro de téléphone doit être modifié. |
| Attribute-Id      | 1.2.840.113556.1.4.721                                                           |
| System-ID-GUID    | 4d146e4a-48d4-11d1-a9c3-0000f80367c1                                             |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md)                                      |
