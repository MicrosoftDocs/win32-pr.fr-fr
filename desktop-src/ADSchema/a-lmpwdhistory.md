# <a name="lm-pwd-history-attribute"></a>Attribut LM-pwd-History

Historique des mots de passe de l’utilisateur dans le format à sens unique (OWF) LAN Manager (LM). Le niveau de compatibilité LM est utilisé pour la compatibilité avec LAN Manager 2. *x* clients, Windows 95 et Windows 98.



| Entrée | Valeur |
|-------------------|-------------------------------------------------------|
| CN                | LM-pwd-historique                                        |
| LDAP-Display-Name | lmPwdHistory                                          |
| Taille              | \-                                                    |
| Mettre à jour le privilège  | Administrateur de domaine                                  |
| Fréquence des mises à jour  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.160                                |
| System-ID-GUID    | bf96799d-0de6-11d0-a285-00aa003049e2                  |
| Syntaxe            | [**Object(Replica-Link)**](s-object-replica-link.md) |
