# <a name="certificate-templates-attribute"></a>Attribut Certificate-Templates

Contient des informations relatives à un certificat émis par un serveur de certificats.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | Certificate-Templates                       |
| LDAP-Display-Name | certificateTemplates                        |
| Taille              | \-                                          |
| Mettre à jour le privilège  | \-                                          |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.823                      |
| System-ID-GUID    | 2a39c5b1-8960-11d1-aebc-0000f80367c1        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
