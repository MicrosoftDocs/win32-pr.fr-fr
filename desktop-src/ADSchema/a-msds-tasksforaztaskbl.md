|-------------------|-----------------------------------------|
| CN                | ms-DS-tâches-pour-AZ-Task-BL              |
| LDAP-Display-Name | msDS-TasksForAzTaskBL                   |
| Taille              | \-                                      |
| Mettre à jour le privilège  | \-                                      |
| Fréquence des mises à jour  | Cette valeur est définie par le système.        |
