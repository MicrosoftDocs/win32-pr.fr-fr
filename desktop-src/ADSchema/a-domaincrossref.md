# <a name="domain-cross-ref-attribute"></a>Attribut de référence croisée de domaine

Il s’agit d’une référence d’un objet de domaine approuvé à l’objet de référence croisée du domaine approuvé.



| Entrée | Valeur |
|-------------------|-----------------------------------------|
| CN                | Référence croisée de domaine                        |
| LDAP-Display-Name | domainCrossRef                          |
| Taille              | \-                                      |
| Mettre à jour le privilège  | \-                                      |
| Fréquence des mises à jour  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.472                  |
| System-ID-GUID    | b000ea7b-a086-11d0-afdd-00c04fd930c9    |
| Syntaxe            | [**Object(DS-DN)**](s-object-ds-dn.md) |
