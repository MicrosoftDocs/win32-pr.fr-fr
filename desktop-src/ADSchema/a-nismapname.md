# <a name="nismapname-attribute"></a>attribut nisMapName

Contient le nom de la carte à laquelle l’objet appartient.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | nisMapName                           |
| LDAP-Display-Name | nisMapName                           |
| Taille              | \-                                   |
| Mettre à jour le privilège  | \-                                   |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.26                     |
| System-ID-GUID    | 969d3c79-0e9a-4d95-b0ac-bdde7ff8f3a1 |
| Syntaxe            | [**String(IA5)**](s-string-ia5.md)  |
