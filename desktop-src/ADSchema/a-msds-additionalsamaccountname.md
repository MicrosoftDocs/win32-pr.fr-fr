# <a name="ms-ds-additional-sam-account-name-attribute"></a>ms-DS-supplémentaire-Sam-Account-Name (attribut)

Cet attribut est utilisé pour stocker les noms de compte SAM qui correspondent aux noms d’hôte DNS dans ms-DS-addition-DNS-Host-Name.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | ms-DS-supplémentaire-Sam-Account-Name           |
| LDAP-Display-Name | msDS-AdditionalSamAccountName               |
| Taille              | Moins de 20 octets.                         |
| Mettre à jour le privilège  | Cette valeur est définie par le système.            |
| Fréquence des mises à jour  | Lorsque le contrôleur de périphérique est renommé.                     |
| Attribute-Id      | 1.2.840.113556.1.4.1718                     |
| System-ID-GUID    | 975571df-a4d5-429a-9f59-cdc6581d91e6        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
