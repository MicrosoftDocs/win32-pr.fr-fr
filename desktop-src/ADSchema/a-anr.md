# <a name="anr-attribute"></a>Attribut ANR

Attribut de résolution de noms ambigu à utiliser lors du choix d’un objet à l’autre.



| Entrée | Valeur |
|-------------------|-------------------------------------------------------------------|
| CN                | AMBIGU                                                               |
| LDAP-Display-Name | Ambigu                                                               |
| Taille              | \-                                                                |
| Mettre à jour le privilège  | Administrateur de schéma                                              |
| Fréquence des mises à jour  | Lorsqu’un attribut doit être ajouté ou supprimé de la liste ANR. |
| Attribute-Id      | 1.2.840.113556.1.4.1208                                           |
| System-ID-GUID    | 45b01500-c419-11d1-bbc9-0080c76670c0                              |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md)                       |
