# <a name="ms-wmi-targettype-attribute"></a>MS-WMI-TargetType (attribut)

Référence WMI à une définition de type pour un objet de stratégie.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | MS-WMI-TargetType                           |
| LDAP-Display-Name | msWMI-TargetType                            |
| Taille              | Moins de 100 caractères.                   |
| Mettre à jour le privilège  | Administrateur stratégie de groupe                  |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1649                     |
| System-ID-GUID    | ca2a281e-262b-4ff7-b419-bc123352a4e9        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
