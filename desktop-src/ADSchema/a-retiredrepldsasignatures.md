# <a name="retired-repl-dsa-signatures-attribute"></a>Retirée-REPL-DSA-attribut signatures

Effectue le suivi des identités de réplication DS passées d’un contrôleur de domaine donné.



| Entrée | Valeur |
|-------------------|-------------------------------------------------------------------------------------|
| CN                | Retirée-REPL-DSA-signatures                                                         |
| LDAP-Display-Name | retiredReplDSASignatures                                                            |
| Taille              | La longueur est proportionnelle au nombre de fois où le DC a été restauré à partir d’une sauvegarde. |
| Mettre à jour le privilège  | Cette valeur est définie par le système.                                                    |
| Fréquence des mises à jour  | \-                                                                                  |
| Attribute-Id      | 1.2.840.113556.1.4.673                                                              |
| System-ID-GUID    | 7bfdcb7f-4807-11d1-a9c3-0000f80367c1                                                |
| Syntaxe            | [**Object(Replica-Link)**](s-object-replica-link.md)                               |
