# <a name="domain-certificate-authorities-attribute"></a>Attribut Domain-Certificate-Authorities

Liste des autorités de certification pour un domaine donné.



| Entrée | Valeur |
|-------------------|-----------------------------------------|
| CN                | Autorités de certification de domaine          |
| LDAP-Display-Name | domainCAs                               |
| Taille              | \-                                      |
| Mettre à jour le privilège  | \-                                      |
| Fréquence des mises à jour  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.668                  |
| System-ID-GUID    | 7bfdcb7a-4807-11d1-a9c3-0000f80367c1    |
| Syntaxe            | [**Object(DS-DN)**](s-object-ds-dn.md) |
