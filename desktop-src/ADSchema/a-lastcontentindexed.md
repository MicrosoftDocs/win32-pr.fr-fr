# <a name="last-content-indexed-attribute"></a>Attribut Last-content-indexé

Heure à laquelle ce volume a été indexé pour la dernière fois.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | Dernier contenu indexé                 |
| LDAP-Display-Name | lastContentIndexed                   |
| Taille              | 8 octets                              |
| Mettre à jour le privilège  | \-                                   |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.50                |
| System-ID-GUID    | bf967995-0de6-11d0-a285-00aa003049e2 |
| Syntaxe            | [**Défini**](s-interval.md)       |
