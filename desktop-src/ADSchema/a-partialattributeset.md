# <a name="partial-attribute-set-attribute"></a>Partial-attribute-set (attribut)

Effectue le suivi de l’état de réplication interne des réplicas partiels (autrement dit, sur des catalogues globaux). Attribut de l’objet de contexte de nommage de réplica partiel. Définit l’ensemble des attributs présents sur un contexte de nommage de réplica partiel particulier.



| Entrée | Valeur |
|-------------------|-------------------------------------------------------|
| CN                | Partial-attribute-set                                 |
| LDAP-Display-Name | partialAttributeSet                                   |
| Taille              | \-                                                    |
| Mettre à jour le privilège  | Cette valeur est définie par le système.                      |
| Fréquence des mises à jour  | Pendant la réplication                                    |
| Attribute-Id      | 1.2.840.113556.1.4.640                                |
| System-ID-GUID    | 19405b9e-3cfa-11d1-a9c0-0000f80367c1                  |
| Syntaxe            | [**Object(Replica-Link)**](s-object-replica-link.md) |
