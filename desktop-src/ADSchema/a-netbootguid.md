# <a name="netboot-guid-attribute"></a>Attribut netboot-GUID

Démarrage sans disque : GUID intégré à un ordinateur. Correspond à l’adresse MAC de la carte réseau de l’ordinateur.



| Entrée | Valeur |
|-------------------|-------------------------------------------------------|
| CN                | Netboot-GUID                                          |
| LDAP-Display-Name | netbootGUID                                           |
| Taille              | \-                                                    |
| Mettre à jour le privilège  | Cette valeur est définie par le système.                      |
| Fréquence des mises à jour  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.359                                |
| System-ID-GUID    | 3e978921-8c01-11d0-afda-00c04fd930c9                  |
| Syntaxe            | [**Object(Replica-Link)**](s-object-replica-link.md) |
