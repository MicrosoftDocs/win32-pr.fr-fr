# <a name="ms-wmi-int8max-attribute"></a>attribut ms-WMI-int8Max

Valeur maximale d’un objet de paramètre entier WMI 64 bits.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | MS-WMI-int8Max                       |
| LDAP-Display-Name | msWMI-Int8Max                        |
| Taille              | 8 octets                              |
| Mettre à jour le privilège  | Administrateur stratégie de groupe           |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1633              |
| System-ID-GUID    | e3d8b547-003d-4946-a32b-dc7cedc96b74 |
| Syntaxe            | [**Défini**](s-interval.md)       |
