# <a name="stringutc-time-syntax"></a>Syntaxe String (UTC-Time)

Format de chaîne d’heure défini par les normes ASN. 1. Pour plus d’informations, consultez ISO 8601 et X680. Utilisez cette syntaxe pour stocker des valeurs d’heure au format UTC-Time.



| Entrée | Valeur |
|--------------|----------------------------------------------------------------------------|
| Name         | String(UTC-Time)                                                           |
| ID de syntaxe    | 2.5.5.11                                                                   |
| ID OM        | 23                                                                         |
| Type MAPI    | SYSTIME                                                                    |
| Type ADS     | \_heure UTC \_ ADSTYPE                                                         |
| Type de variante | \_Date VT                                                                   |
| SDS, type     | [System.DateTime](/dotnet/api/system.datetime) |
