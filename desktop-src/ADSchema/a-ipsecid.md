# <a name="ipsec-id-attribute"></a>Attribut d’ID IPSec

L’attribut d' **ID IPSec** est destiné à un usage interne uniquement.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | ID IPSec                                    |
| LDAP-Display-Name | ipsecID                                     |
| Taille              | \-                                          |
| Mettre à jour le privilège  | \-                                          |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.621                      |
| System-ID-GUID    | b40ff81d-427a-11d1-a9c2-0000f80367c1        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
