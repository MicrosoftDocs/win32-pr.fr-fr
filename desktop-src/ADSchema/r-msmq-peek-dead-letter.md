# <a name="msmq-peek-dead-letter-extended-right"></a>MSMQ-Peek-la lettre morte à droite

Autorise la lecture des messages dans la file d’attente de lettres mortes.

|--------------|--------------------------------------|
| CN           | MSMQ-Peek-morte-lettre                |
| Display-Name | Lire une lettre morte                     |
| Rights-GUID  | 4b6e08c1-df3c-11d1-9c86-006008764d0e |
