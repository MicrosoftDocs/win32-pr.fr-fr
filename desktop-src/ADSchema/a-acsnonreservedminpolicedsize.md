# <a name="acs-non-reserved-min-policed-size-attribute"></a>ACS-attribut non réservé-min-Policièred-Size

L’attribut **ACS-non réservé-min-policied-Size** est destiné à un usage interne uniquement. Basé sur RFC2814.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | ACS-non réservé-min-Policièred-Size    |
| LDAP-Display-Name | aCSNonReservedMinPolicedSize         |
| Taille              | 8 octets                              |
| Mettre à jour le privilège  | \-                                   |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1321              |
| System-ID-GUID    | b6873917-3b90-11d2-90cc-00c04fd91ab1 |
| Syntaxe            | [**Défini**](s-interval.md)       |
