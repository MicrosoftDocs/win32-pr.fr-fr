# <a name="generation-qualifier-attribute"></a>Attribut Generation-Qualifier

Indique la génération d’une personne. Par exemple, Jr. ou II.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | Generation-Qualifier                        |
| LDAP-Display-Name | generationQualifier                         |
| Taille              | \-                                          |
| Mettre à jour le privilège  | Administrateur de domaine                        |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 2.5.4.44                                    |
| System-ID-GUID    | 16775804-47f3-11d1-a9c3-0000f80367c1        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
