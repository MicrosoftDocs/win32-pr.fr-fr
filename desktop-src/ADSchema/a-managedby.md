# <a name="managed-by-attribute"></a>Attribut Managed-By

Nom unique de l’utilisateur assigné pour gérer cet objet.



| Entrée | Valeur |
|-------------------|-------------------------------------------------------------------------------------|
| CN                | Managed-By                                                                          |
| LDAP-Display-Name | managedBy                                                                           |
| Taille              | \-                                                                                  |
| Mettre à jour le privilège  | Administrateur de domaine ou propriétaire du compte.                                              |
| Fréquence des mises à jour  | Lorsque l’enregistrement de l’utilisateur est créé et chaque fois que les objets gérés doivent être modifiés. |
| Attribute-Id      | 1.2.840.113556.1.4.653                                                              |
| System-ID-GUID    | 0296c120-40da-11d1-a9c0-0000f80367c1                                                |
| Syntaxe            | [**Object(DS-DN)**](s-object-ds-dn.md)                                             |
