# <a name="ms-ds-managedpasswordinterval-attribute"></a>attribut ms-DS-ManagedPasswordInterval

Cet attribut est utilisé pour récupérer le nombre de jours avant qu’un mot de passe géré soit automatiquement modifié pour un MSA de groupe.

| LDAP-Display-Name | msDS-ManagedPasswordInterval         |
| Taille              | \-                                   |

| System-ID-GUID    | f8758ef7-ac76-8843-a2ee-a26b4dcaf409 |
