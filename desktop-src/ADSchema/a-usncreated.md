# <a name="usn-created-attribute"></a>Attribut USN-Created

Numéro de séquence de mise à jour (USN) affecté lors de la création de l’objet. Voir aussi [**USN-Changed**](a-usnchanged.md).



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | USN-Created                          |
| LDAP-Display-Name | uSNCreated                           |
| Taille              | 8 octets                              |
| Mettre à jour le privilège  | Cette valeur est définie par le système.     |
| Fréquence des mises à jour  | Lorsque l’objet est créé.          |
| Attribute-Id      | 1.2.840.113556.1.2.19                |
| System-ID-GUID    | bf967a70-0de6-11d0-a285-00aa003049e2 |
| Syntaxe            | [**Intervalle**](s-interval.md)       |
