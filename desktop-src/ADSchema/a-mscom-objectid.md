# <a name="ms-com-objectid-attribute"></a>attribut ms-COM-ObjectId

GUID utilisé pour identifier une partition COM+ ou un PartitionSet COM+.



| Entrée | Valeur |
|-------------------|-------------------------------------------------------|
| CN                | MS-COM-ObjectId                                       |
| LDAP-Display-Name | msCOM-ObjectId                                        |
| Taille              | \-                                                    |
| Mettre à jour le privilège  | \-                                                    |
| Fréquence des mises à jour  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.1428                               |
| System-ID-GUID    | 430f678b-889f-41f2-9843-203b5a65572f                  |
| Syntaxe            | [**Object(Replica-Link)**](s-object-replica-link.md) |
