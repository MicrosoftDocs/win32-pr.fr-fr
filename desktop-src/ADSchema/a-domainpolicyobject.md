# <a name="domain-policy-object-attribute"></a>Attribut d’objet de stratégie de domaine

Référence à l’objet de stratégie qui définit la stratégie de l’autorité de sécurité locale pour le domaine hôte.



| Entrée | Valeur |
|-------------------|-----------------------------------------|
| CN                | Objet de stratégie de domaine                    |
| LDAP-Display-Name | domainPolicyObject                      |
| Taille              | \-                                      |
| Mettre à jour le privilège  | \-                                      |
| Fréquence des mises à jour  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.32                   |
| System-ID-GUID    | bf96795d-0de6-11d0-a285-00aa003049e2    |
| Syntaxe            | [**Object(DS-DN)**](s-object-ds-dn.md) |
