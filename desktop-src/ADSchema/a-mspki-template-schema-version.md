# <a name="ms-pki-template-schema-version-attribute"></a>MS-PKI-template-Schema-Version (attribut)

Effectue le suivi des mises à jour de schéma de la classe PKI-Certificate-template.



| Entrée | Valeur |
|-------------------|---------------------------------------------------------------------------------------------------|
| CN                | MS-PKI-template-Schema-Version                                                                    |
| LDAP-Display-Name | Attribut mspki-template-Schema-Version                                                                     |
| Taille              | 4 octets                                                                                           |
| Mettre à jour le privilège  | Administrateur de domaine                                                                              |
| Fréquence des mises à jour  | Lorsque l’objet modèle de certificat (MS-PKI-Certificate-template) est modifié, créé ou cloné. |
| Attribute-Id      | 1.2.840.113556.1.4.1434                                                                           |
| System-ID-GUID    | 0c15e9f5-491d-4594-918f-32813a091da9                                                              |
| Syntaxe            | [**Énumération**](s-enumeration.md)                                                              |
