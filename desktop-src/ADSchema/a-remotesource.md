| CN                | Remote-Source                               |
| LDAP-Display-Name | remoteSource                                |
| Taille              | \-                                          |
| Mettre à jour le privilège  | \-                                          |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.107                      |
| System-ID-GUID    | bf967a14-0de6-11d0-a285-00aa003049e2        |
