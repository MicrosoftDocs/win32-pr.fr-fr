# <a name="knowledge-information-attribute"></a>Attribut Knowledge-Information

Spécifie une description accumulée par l’utilisateur, qui est regroupée à l’aide d’un agent de système d’annuaire spécifique.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | Knowledge-Information                       |
| LDAP-Display-Name | knowledgeInformation                        |
| Taille              | \-                                          |
| Mettre à jour le privilège  | \-                                          |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 2.5.4.2 -7                                     |
| System-ID-GUID    | 1677581f-47f3-11d1-a9c3-0000f80367c1        |
| Syntaxe            | [**String(Teletex)**](s-string-teletex.md) |
