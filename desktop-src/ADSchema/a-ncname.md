# <a name="nc-name-attribute"></a>Attribut NC-Name

| Entrée | Valeur |
|-------------------|-----------------------------------------|
| CN                | NC-Name                                 |
| LDAP-Display-Name | nCName                                  |
| Taille              | \-                                      |
| Mettre à jour le privilège  | Administrateur de domaine                    |
| Fréquence des mises à jour  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.2.16                   |
| System-ID-GUID    | bf9679d6-0de6-11d0-a285-00aa003049e2    |
| Syntaxe            | [**Object(DS-DN)**](s-object-ds-dn.md) |
