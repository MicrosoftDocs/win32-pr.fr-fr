# <a name="ms-sql-thirdparty-attribute"></a>Attribut MS-SQL-ThirdParty

Cet attribut indique si les données de publication proviennent d’une source de données autre que SQL Server. S’il provient d’une autre source, il est défini sur **true**.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | MS-SQL-ThirdParty                    |
| LDAP-Display-Name | mS-SQL-ThirdParty                    |
| Taille              | 4 octets                              |
| Mettre à jour le privilège  | Cette valeur est définie par le système.     |
| Fréquence des mises à jour  | Lorsque la réplication est configurée.           |
| Attribute-Id      | 1.2.840.113556.1.4.1407              |
| System-ID-GUID    | c4e311fc-d34b-11d2-999a-0000f87a57d4 |
| Syntaxe            | [**Boolean**](s-boolean.md)         |
