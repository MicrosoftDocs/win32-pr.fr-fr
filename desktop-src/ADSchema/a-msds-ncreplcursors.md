# <a name="ms-ds-nc-repl-cursors-attribute"></a>attribut ms-DS-NC repl

Liste des partenaires de réplication passés et présents, et de la manière dont nous sommes en cours avec chacun d’eux.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | ms-DS-NC REPL-curseurs                       |
| LDAP-Display-Name | msDS-NCReplCursors                          |
| Taille              | \-                                          |
| Mettre à jour le privilège  | Cette valeur est définie par le système.            |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1704                     |
| System-ID-GUID    | 8a167ce4-f9e8-47eb-8d78-f7fe80abb2cc        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
