# <a name="macaddress-attribute"></a>attribut macAddress

Contient l’adresse MAC en notation hexadécimale à deux-points, séparée par deux-points.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | macAddress                           |
| LDAP-Display-Name | macAddress                           |
| Taille              | \-                                   |
| Mettre à jour le privilège  | \-                                   |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.22                     |
| System-ID-GUID    | e6a522dd-9770-43e1-89de-1de5044328f7 |
| Syntaxe            | [**String(IA5)**](s-string-ia5.md)  |
