# <a name="ms-wmi-genus-attribute"></a>attribut de genre MS-WMI

Identifie le type d’objet d’un encodage.



| Entrée | Valeur |
|-------------------|-----------------------------------------------------------------|
| CN                | MS-WMI-genre                                                    |
| LDAP-Display-Name | msWMI-genre                                                     |
| Taille              | 4 octets. 1 indique une classe. 2 indique une instance.          |
| Mettre à jour le privilège  | Administrateur de domaine                                            |
| Fréquence des mises à jour  | Chaque fois qu’une classe contenant l’attribut est ajoutée ou modifiée. |
| Attribute-Id      | 1.2.840.113556.1.4.1677                                         |
| System-ID-GUID    | 50c8673a-8f56-4614-9308-9e1340fb9af3                            |
| Syntaxe            | [**Énumération**](s-enumeration.md)                            |
