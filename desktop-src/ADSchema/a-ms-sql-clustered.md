# <a name="ms-sql-clustered-attribute"></a>MS-SQL-attribut cluster

True si le serveur est en cluster.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | MS-SQL-Clustered                     |
| LDAP-Display-Name | mS-SQL-Clustered                     |
| Taille              | 4 octets                              |
| Mettre à jour le privilège  | Cette valeur est définie par le système.     |
| Fréquence des mises à jour  | Au démarrage du système.                   |
| Attribute-Id      | 1.2.840.113556.1.4.1373              |
| System-ID-GUID    | 7778bd90-ccee-11d2-9993-0000f87a57d4 |
| Syntaxe            | [**Boolean**](s-boolean.md)         |
