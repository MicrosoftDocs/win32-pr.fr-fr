# <a name="ms-sql-lastbackupdate-attribute"></a>Attribut MS-SQL-LastBackupDate

Date de la dernière sauvegarde de la base de données.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | MS-SQL-LastBackupDate                       |
| LDAP-Display-Name | mS-SQL-LastBackupDate                       |
| Taille              | \-                                          |
| Mettre à jour le privilège  | Cette valeur est définie par le système.            |
| Fréquence des mises à jour  | Lors de la sauvegarde de la base de données.             |
| Attribute-Id      | 1.2.840.113556.1.4.1398                     |
| System-ID-GUID    | f2b6abca-ccee-11d2-9993-0000f87a57d4        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
