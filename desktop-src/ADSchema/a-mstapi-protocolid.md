# <a name="ms-tapi-protocol-id-attribute"></a>attribut ms-TAPI-Protocol-ID

Cet attribut indique le type de conférence TAPI. Cet attribut est utilisé pour déterminer comment l’objet BLOB binaire de l’attribut [**MS-TAPI-Conference-BLOB**](a-mstapi-conferenceblob.md) doit être interprété. Cet attribut est une chaîne arbitraire, dont la longueur est généralement inférieure à 100 caractères.



| Entrée | Valeur |
|-------------------|------------------------------------------------------------|
| CN                | MS-TAPI-Protocol-ID                                        |
| LDAP-Display-Name | msTAPI-ProtocolId                                          |
| Taille              | \-                                                         |
| Mettre à jour le privilège  | Aucun privilège spécial n’est requis.                            |
| Fréquence des mises à jour  | Défini une fois au moment de la création de l’objet Rt-Conference. |
| Attribute-Id      | 1.2.840.113556.1.4.1699                                    |
| System-ID-GUID    | 89c1ebcf-7a5f-41fd-99ca-c900b32299ab                       |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md)                |
