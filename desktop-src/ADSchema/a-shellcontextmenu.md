# <a name="shell-context-menu-attribute"></a>Shell-Context-menu (attribut)

Le numéro d’ordre et le GUID du menu contextuel pour cet objet.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | Menu contextuel de l’interpréteur de commandes                          |
| LDAP-Display-Name | shellContextMenu                            |
| Taille              | \-                                          |
| Mettre à jour le privilège  | Administrateur de schéma                        |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.615                      |
| System-ID-GUID    | 553fd039-f32e-11d0-b0bc-00c04fd8dca6        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
