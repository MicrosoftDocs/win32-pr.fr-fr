# <a name="ipsec-negotiation-policy-reference-attribute"></a>IPSec-négociation-stratégie-attribut de référence

L’attribut **IPSec-Negotiation-Policy-Reference** est destiné à un usage interne uniquement.



| Entrée | Valeur |
|-------------------|-----------------------------------------|
| CN                | Informations de référence sur la stratégie IPSec-Negotiation      |
| LDAP-Display-Name | ipsecNegotiationPolicyReference         |
| Taille              | \-                                      |
| Mettre à jour le privilège  | \-                                      |
| Fréquence des mises à jour  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.628                  |
| System-ID-GUID    | b40ff822-427a-11d1-a9c2-0000f80367c1    |
| Syntaxe            | [**Object(DS-DN)**](s-object-ds-dn.md) |
