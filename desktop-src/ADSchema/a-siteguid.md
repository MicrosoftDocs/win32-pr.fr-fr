# <a name="site-guid-attribute"></a>Attribut de GUID de site

Identificateur unique d’un site.



| Entrée | Valeur |
|-------------------|-------------------------------------------------------|
| CN                | GUID de site                                             |
| LDAP-Display-Name | siteGUID                                              |
| Taille              | 16 octets                                              |
| Mettre à jour le privilège  | Cette valeur est définie par le système.                      |
| Fréquence des mises à jour  | Chaque fois qu’un nouveau site est créé.                       |
| Attribute-Id      | 1.2.840.113556.1.4.362                                |
| System-ID-GUID    | 3e978924-8c01-11d0-afda-00c04fd930c9                  |
| Syntaxe            | [**Object(Replica-Link)**](s-object-replica-link.md) |
