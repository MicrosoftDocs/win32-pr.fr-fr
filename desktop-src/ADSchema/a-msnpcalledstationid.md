# <a name="msnpcalledstationid-attribute"></a>attribut msNPCalledStationID

L’attribut **msNPCalledStationID** est utilisé en interne. Ne modifiez pas cette valeur directement.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | msNPCalledStationID                  |
| LDAP-Display-Name | msNPCalledStationID                  |
| Taille              | \-                                   |
| Mettre à jour le privilège  | \-                                   |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1123              |
| System-ID-GUID    | db0c9089-c1f2-11d1-bbc5-0080c76670c0 |
| Syntaxe            | [**String(IA5)**](s-string-ia5.md)  |
