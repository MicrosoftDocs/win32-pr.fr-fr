# <a name="ms-ds-az-operation-id-attribute"></a>attribut ms-DS-AZ-Operation-ID

ID spécifique à l’application qui rend l’opération unique à l’application.



| Entrée | Valeur |
|-------------------|-----------------------------------------|
| CN                | ms-DS-AZ-Operation-ID                   |
| LDAP-Display-Name | msDS-AzOperationID                      |
| Taille              | \-                                      |
| Mettre à jour le privilège  | Administrateur AzRoles                           |
| Fréquence des mises à jour  | Pendant l’initialisation ou la modification de la stratégie. |
| Attribute-Id      | 1.2.840.113556.1.4.1800                 |
| System-ID-GUID    | a5f3b553-5d76-4cbe-ba3f-4312152cab18    |
