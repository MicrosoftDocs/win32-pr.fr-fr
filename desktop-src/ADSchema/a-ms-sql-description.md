# <a name="ms-sql-description-attribute"></a>MS-SQL-attribut Description

Description de la base de données SQL Server.



| Entrée | Valeur |
|-------------------|--------------------------------------------------|
| CN                | Description de MS-SQL                               |
| LDAP-Display-Name | Description de mS-SQL                               |
| Taille              | \-                                               |
| Mettre à jour le privilège  | Administrateur de domaine                             |
| Fréquence des mises à jour  | Lorsque l’utilisateur décide de modifier la description. |
| Attribute-Id      | 1.2.840.113556.1.4.1390                          |
| System-ID-GUID    | 8386603c-ccef-11d2-9993-0000f87a57d4             |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md)      |
