# <a name="domain-policy-reference-attribute"></a>Attribut de référence de stratégie de domaine

Nom unique d’un objet de stratégie de domaine à partir duquel un objet de stratégie est copié.



| Entrée | Valeur |
|-------------------|-----------------------------------------|
| CN                | Informations de référence sur la stratégie de domaine                 |
| LDAP-Display-Name | domainPolicyReference                   |
| Taille              | \-                                      |
| Mettre à jour le privilège  | \-                                      |
| Fréquence des mises à jour  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.422                  |
| System-ID-GUID    | 80a67e2a-9f22-11d0-afdd-00c04fd930c9    |
| Syntaxe            | [**Object(DS-DN)**](s-object-ds-dn.md) |
