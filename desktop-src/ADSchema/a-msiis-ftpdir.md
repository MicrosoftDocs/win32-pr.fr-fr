# <a name="ms-iis-ftp-dir-attribute"></a>attribut MS-IIS-FTP-dir

Cet attribut détermine le répertoire de démarrage de l’utilisateur par rapport au partage de serveur de fichiers. Il est utilisé conjointement avec MS-IIS-FTP-root pour déterminer le répertoire de démarrage de l’utilisateur FTP.



| Entrée | Valeur |
|-------------------|---------------------------------------------------------------------------------------------------------------------------------|
| CN                | MS-IIS-FTP-dir                                                                                                                  |
| LDAP-Display-Name | msIIS-FTPDir                                                                                                                    |
| Taille              | 30-50 caractères (15-25 caractères Unicode pour chaque propriété)                                                                   |
| Mettre à jour le privilège  | Administrateur de domaine & système local                                                                                             |
| Fréquence des mises à jour  | Cette propriété est définie lorsque l’administrateur crée le site Web et définit l’isolement FTP. Cela va rarement changer après cela. |
| Attribute-Id      | 1.2.840.113556.1.4.1786                                                                                                         |
| System-ID-GUID    | 8a5c99e9-2230-46eb-b8e8-e59d712eb9ee                                                                                            |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md)                                                                                     |
