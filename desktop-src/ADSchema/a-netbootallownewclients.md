# <a name="netboot-allow-new-clients-attribute"></a>attribut netboot-allow-New-clients

L’attribut **netboot-allow-New-clients** est réservé à un usage interne.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | netboot-autoriser-New-clients            |
| LDAP-Display-Name | netbootAllowNewClients               |
| Taille              | \-                                   |
| Mettre à jour le privilège  | Cette valeur est définie par le système.     |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.849               |
| System-ID-GUID    | 07383076-91df-11d1-aebc-0000f80367c1 |
| Syntaxe            | [**Boolean**](s-boolean.md)         |
