# <a name="show-in-advanced-view-only-attribute"></a>Afficher l’attribut-in-Advanced-View uniquement

**True** si cet attribut doit être visible en mode avancé de l’interface utilisateur.



| Entrée | Valeur |
|-------------------|-----------------------------------------------------------------------------------------|
| CN                | Afficher dans Advanced-View uniquement                                                              |
| LDAP-Display-Name | showInAdvancedViewOnly                                                                  |
| Taille              | 4 octets                                                                                 |
| Mettre à jour le privilège  | Administrateur de domaine                                                                    |
| Fréquence des mises à jour  | Lorsque l’objet est créé ou lorsque les aspects d’affichage de l’attribut doivent être modifiés. |
| Attribute-Id      | 1.2.840.113556.1.2.169                                                                  |
| System-ID-GUID    | bf967984-0de6-11d0-a285-00aa003049e2                                                    |
| Syntaxe            | [**Boolean**](s-boolean.md)                                                            |
