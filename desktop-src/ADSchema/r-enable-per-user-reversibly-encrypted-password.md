# <a name="enable-per-user-reversibly-encrypted-password-extended-right"></a>Activer : par utilisateur-réversible-réversible-mot de passe étendu

Droit d’accès du contrôle étendu qui permet aux utilisateurs d’activer ou de désactiver le paramètre « mot de passe chiffré réversible » pour les objets utilisateur et ordinateur.



| Entrée | Valeur |
|--------------|-----------------------------------------------|
| CN           | Activer-par-utilisateur-Réversibled-Password |
| Display-Name | Activer le mot de passe chiffré réversible par utilisateur |
| Rights-GUID  | 05c74c5e-4deb-43b4-bd9f-86664c2a7fd5          |
