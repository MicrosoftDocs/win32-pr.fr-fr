# <a name="netboot-new-machine-naming-policy-attribute"></a>attribut de stratégie d’attribution de nom pour netboot-New-machine

Indique le schéma d’affectation de noms que les nouveaux comptes d’ordinateurs clients utiliseront.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | netboot-New-machine-nommage-stratégie           |
| LDAP-Display-Name | netbootNewMachineNamingPolicy               |
| Taille              | \-                                          |
| Mettre à jour le privilège  | Cette valeur est définie par le système.            |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.855                      |
| System-ID-GUID    | 0738307c-91df-11d1-aebc-0000f80367c1        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
