# <a name="ms-sql-allowimmediateupdatingsubscription-attribute"></a>attribut MS-SQL-AllowImmediateUpdatingSubscription

True si la publication autorise les abonnements de mise à jour de la transaction synchronisée.



| Entrée | Valeur |
|-------------------|-------------------------------------------|
| CN                | MS-SQL-AllowImmediateUpdatingSubscription |
| LDAP-Display-Name | mS-SQL-AllowImmediateUpdatingSubscription |
| Taille              | 4 octets                                   |
| Mettre à jour le privilège  | Cette valeur est définie par le système.          |
| Fréquence des mises à jour  | Lorsque la réplication est configurée.                |
| Attribute-Id      | 1.2.840.113556.1.4.1404                   |
| System-ID-GUID    | c4186b6e-d34b-11d2-999a-0000f87a57d4      |
| Syntaxe            | [**Boolean**](s-boolean.md)              |
