# <a name="netboot-locally-installed-oses-attribute"></a>netboot-installation locale-attribut OS

L’attribut **netboot-local-installed-OS** est réservé à un usage interne.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | netboot-installé localement-systèmes d’exploitation              |
| LDAP-Display-Name | netbootLocallyInstalledOSes                 |
| Taille              | \-                                          |
| Mettre à jour le privilège  | Cette valeur est définie par le système.            |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.859                      |
| System-ID-GUID    | 07383080-91df-11d1-aebc-0000f80367c1        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
