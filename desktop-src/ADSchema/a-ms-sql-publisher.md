# <a name="ms-sql-publisher-attribute"></a>Attribut MS-SQL-Publisher

Nom de la base de données du serveur de publication pour la réplication.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | MS-SQL-serveur de publication                            |
| LDAP-Display-Name | mS-SQL-serveur de publication                            |
| Taille              | \-                                          |
| Mettre à jour le privilège  | Administrateur de domaine                        |
| Fréquence des mises à jour  | Lorsque la réplication est configurée.                  |
| Attribute-Id      | 1.2.840.113556.1.4.1402                     |
| System-ID-GUID    | c1676858-d34b-11d2-999a-0000f87a57d4        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
