# <a name="ms-pki-minimal-key-size-attribute"></a>MS-PKI-attribut minimal-Key-Size

Indique la taille minimale de la clé privée.



| Entrée | Valeur |
|-------------------|---------------------------------------------------------------------------------------------------|
| CN                | MS-PKI-minimum-Key-Size                                                                           |
| LDAP-Display-Name | Attribut mspki-minimal-Key-Size                                                                            |
| Taille              | 4 octets                                                                                           |
| Mettre à jour le privilège  | Administrateur de domaine                                                                              |
| Fréquence des mises à jour  | Lorsque l’objet modèle de certificat (MS-PKI-Certificate-template) est modifié, créé ou cloné. |
| Attribute-Id      | 1.2.840.113556.1.4.1433                                                                           |
| System-ID-GUID    | e96a63f5-417f-46d3-be52-db7703c503df                                                              |
| Syntaxe            | [**Enumeration**](s-enumeration.md)                                                              |
