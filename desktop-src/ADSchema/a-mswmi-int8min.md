# <a name="ms-wmi-int8min-attribute"></a>attribut ms-WMI-int8Min

Valeur minimale d’un objet de paramètre entier WMI 64 bits.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | MS-WMI-int8Min                       |
| LDAP-Display-Name | msWMI-Int8Min                        |
| Taille              | 8 octets                              |
| Mettre à jour le privilège  | Administrateur stratégie de groupe           |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1634              |
| System-ID-GUID    | ed1489d1-54cc-4066-B368-a00daa2664f1 |
| Syntaxe            | [**Défini**](s-interval.md)       |
