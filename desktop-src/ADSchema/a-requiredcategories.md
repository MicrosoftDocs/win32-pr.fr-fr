# <a name="required-categories-attribute"></a>Attribut Required-Categories

Cet attribut contient une liste d’ID de catégorie de composant qu’un objet (par exemple, une application) requiert pour s’exécuter.



| Entrée | Valeur |
|-------------------|-------------------------------------------------------|
| CN                | Required-Categories                                   |
| LDAP-Display-Name | requiredCategories                                    |
| Taille              | \-                                                    |
| Mettre à jour le privilège  | \-                                                    |
| Fréquence des mises à jour  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.321                                |
| System-ID-GUID    | 7d6c0e93-7e20-11d0-afd6-00c04fd930c9                  |
| Syntaxe            | [**Object(Replica-Link)**](s-object-replica-link.md) |
