# <a name="modify-time-stamp-attribute"></a>Attribut de date et d’heure de modification

Attribut calculé qui représente la date de la dernière modification de cet objet. Cette valeur n’est pas répliquée.



| Entrée | Valeur |
|-------------------|---------------------------------------------------------------|
| CN                | Date et heure de modification                                             |
| LDAP-Display-Name | modifyTimeStamp                                               |
| Taille              | 8 octets                                                       |
| Mettre à jour le privilège  | Cette valeur est définie par le système.                              |
| Fréquence des mises à jour  | \-                                                            |
| Attribute-Id      | 2.5.18.2                                                      |
| System-ID-GUID    | 9a7ad94a-ca53-11d1-bbd0-0080c76670c0                          |
| Syntaxe            | [**String(Generalized-Time)**](s-string-generalized-time.md) |
