# <a name="other-mailbox-attribute"></a>Attribut Other-Mailbox

Contient d’autres adresses de messagerie supplémentaires dans un format tel que CCMAIL : BruceKeever.



| Entrée | Valeur |
|-------------------|----------------------------------------------------------|
| CN                | Other-Mailbox                                            |
| LDAP-Display-Name | otherMailbox                                             |
| Taille              | \-                                                       |
| Mettre à jour le privilège  | Administrateur de domaine                                     |
| Fréquence des mises à jour  | Chaque fois que l’utilisateur reçoit une adresse de messagerie supplémentaire. |
| Attribute-Id      | 1.2.840.113556.1.4.651                                   |
| System-ID-GUID    | 0296c123-40da-11d1-a9c0-0000f80367c1                     |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md)              |
