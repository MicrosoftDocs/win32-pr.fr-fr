# <a name="rdn-attribute"></a>Attribut RDN

Nom unique relatif (RDN) d’un objet. Un RDN est la partie relative d’un nom unique (DN), qui identifie de façon unique un objet LDAP.



| Entrée | Valeur |
|-------------------|------------------------------------------------|
| CN                | UNIQUE                                            |
| LDAP-Display-Name | name                                           |
| Taille              | \-                                             |
| Mettre à jour le privilège  | Cette valeur est définie par l’administrateur de schéma. |
| Fréquence des mises à jour  | \-                                             |
| Attribute-Id      | 1.2.840.113556.1.4.1                           |
| System-ID-GUID    | bf967a0e-0de6-11d0-a285-00aa003049e2           |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md)    |
