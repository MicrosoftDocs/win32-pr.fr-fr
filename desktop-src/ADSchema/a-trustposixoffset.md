# <a name="trust-posix-offset-attribute"></a>Trust-POSIX-offset, attribut

Décalage POSIX (Portable Operating System Interface) du domaine approuvé.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | Trust-POSIX-offset                   |
| LDAP-Display-Name | trustPosixOffset                     |
| Taille              | 4 octets                              |
| Mettre à jour le privilège  | Cette valeur est définie par le système.     |
| Fréquence des mises à jour  | Lors de la création d’une approbation.         |
| Attribute-Id      | 1.2.840.113556.1.4.134               |
| System-ID-GUID    | bf967a5e-0de6-11d0-a285-00aa003049e2 |
| Syntaxe            | [**Énumération**](s-enumeration.md) |
