# <a name="acs-identity-name-attribute"></a>ACS-Identity-Name (attribut)

Cet attribut contient le nom unique d’un utilisateur ou d’une unité d’organisation et est l’identité d’un utilisateur ou d’un groupe d’utilisateurs auxquels cette stratégie de QoS s’applique.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | ACS-Identity-Name                           |
| LDAP-Display-Name | aCSIdentityName                             |
| Taille              | \-                                          |
| Mettre à jour le privilège  | Administrateur de domaine                        |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.784                      |
| System-ID-GUID    | dab029b6-ddf7-11d1-90a5-00c04fd91ab1        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
