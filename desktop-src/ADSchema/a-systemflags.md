# <a name="system-flags-attribute"></a>Attribut System-Flags

Valeur entière qui contient des indicateurs qui définissent des propriétés supplémentaires de la classe. Consultez la section Notes.



| Entrée | Valeur |
|-------------------|------------------------------------------------|
| CN                | System-Flags                                   |
| LDAP-Display-Name | systemFlags                                    |
| Taille              | \-                                             |
| Mettre à jour le privilège  | Cette valeur est définie par l’administrateur de schéma. |
| Fréquence des mises à jour  | Chaque fois que l’état de l’objet change.      |
| Attribute-Id      | 1.2.840.113556.1.4.375                         |
| System-ID-GUID    | e0fa1e62-9b45-11d0-afdd-00c04fd930c9           |
| Syntaxe            | [**Enumeration**](s-enumeration.md)           |

## <a name="remarks"></a>Notes

Cet attribut peut avoir la valeur zéro ou une combinaison d’une ou plusieurs des valeurs suivantes.



| Valeur                   | Description                                                                                                                                                                                                                                                                                     |
|-------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 1 (0x00000001)          | Lorsqu’il est appliqué à un attribut, l’attribut n’est pas répliqué. Lorsqu’il est appliqué à un objet [**Cross-Ref**](c-crossref.md) , le contexte d’appellation est dans NTDS.<br/>                                                                                                                          |
| 2 (0x00000002)          | Lorsqu’il est appliqué à un attribut, l’attribut est répliqué dans le catalogue global. Lorsqu’il est appliqué à un objet [**Cross-Ref**](c-crossref.md) , le contexte d’appellation est un domaine.<br/>                                                                                                       |
| 4 (0x00000004)          | Lorsqu’il est appliqué à un attribut, l’attribut est construit.                                                                                                                                                                                                                                     |
| 16 (0x00000010)         | Lorsqu’il est défini, indique que l’objet est un objet de catégorie 1. Un objet catégorie 1 est une classe ou un attribut inclus dans le schéma de base inclus dans le système.                                                                                                                                |
| 33554432 (0x02000000)   | L’objet n’est pas déplacé vers le conteneur objets supprimés lorsqu’il est supprimé. Elle sera supprimée immédiatement.                                                                                                                                                                                    |
| 67108864 (0x04000000)   | L’objet ne peut pas être déplacé.                                                                                                                                                                                                                                                                     |
| 134217728 (0x08000000)  | L’objet ne peut pas être renommé.                                                                                                                                                                                                                                                                   |
| 268435456 (0x10000000)  | Pour les objets de la partition de configuration, si cet indicateur est défini, l’objet peut être déplacé avec des restrictions ; dans le cas contraire, l’objet ne peut pas être déplacé. Par défaut, cet indicateur n’est pas défini sur les nouveaux objets créés sous la partition de configuration. Cet indicateur ne peut être défini qu’au moment de la création de l’objet. |
| 536870912 (0x20000000)  | Pour les objets de la partition de configuration, si cet indicateur est défini, l’objet peut être déplacé. dans le cas contraire, l’objet ne peut pas être déplacé. Par défaut, cet indicateur n’est pas défini sur les nouveaux objets créés sous la partition de configuration. Cet indicateur ne peut être défini qu’au moment de la création de l’objet.                   |
| 1073741824 (0x40000000) | Pour les objets de la partition de configuration, si cet indicateur est défini, l’objet peut être renommé ; dans le cas contraire, l’objet ne peut pas être renommé. Par défaut, cet indicateur n’est pas défini sur les nouveaux objets créés sous la partition de configuration. Cet indicateur ne peut être défini qu’au moment de la création de l’objet.               |
| 2147483648 (0x80000000) | Impossible de supprimer l’objet.                                                                                                                                                                                                                                                                   |
