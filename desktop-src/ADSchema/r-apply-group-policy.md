# <a name="apply-group-policy-extended-right"></a>Apply-Group-stratégie-droit étendu

Droit étendu utilisé par stratégie de groupe moteur pour déterminer si un objet de stratégie de groupe s’applique à un utilisateur ou à un ordinateur.



| Entrée | Valeur |
|--------------|--------------------------------------|
| CN           | Apply-Group-stratégie                   |
| Display-Name | Appliquer stratégie de groupe                   |
| Rights-GUID  | edacfd8f-ffb3-11d1-b41d-00a0c968f939 |
