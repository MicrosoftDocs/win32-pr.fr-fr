# <a name="open-address-book-extended-right"></a>Ouverture-adresse-livre-droit étendu

Le droit étendu est activé lors de l’ouverture de l’objet de carnet d’adresses pour les vues de carnet d’adresses.



| Entrée | Valeur |
|--------------|--------------------------------------|
| CN           | Carnet d’adresses Open                    |
| Display-Name | Ouvrir la liste d’adresses                    |
| Rights-GUID  | a1990816-4298-11d1-ade2-00c04fd8d5cd |
