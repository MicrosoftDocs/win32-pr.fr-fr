# <a name="common-name-attribute"></a>Attribut Common-Name

Nom qui représente un objet. Utilisé pour effectuer des recherches.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | Common-Name                                 |
| LDAP-Display-Name | cn                                          |
| Taille              | La taille moyenne est de 64 octets.                   |
| Mettre à jour le privilège  | Administrateur de domaine                        |
| Fréquence des mises à jour  | Uniquement lorsqu’un nouvel objet est créé.          |
| Attribute-Id      | 2.5.4.3                                     |
| System-ID-GUID    | bf96793f-0de6-11d0-a285-00aa003049e2        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
