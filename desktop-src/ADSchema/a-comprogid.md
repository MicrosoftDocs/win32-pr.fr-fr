# <a name="com-progid-attribute"></a>Attribut COM-ProgID

Cet attribut stocke la liste des ID de programme d’objets COM qui sont implémentés dans ce package d’application.



| Entrée | Valeur |
|-------------------|----------------------------------------------------------------------------------|
| CN                | COM-ProgID                                                                       |
| LDAP-Display-Name | cOMProgID                                                                        |
| Taille              | \-                                                                               |
| Mettre à jour le privilège  | Tout le monde peut mettre à jour cet objet en fonction de la sécurité de l’objet en cours de création. |
| Fréquence des mises à jour  | \-                                                                               |
| Attribute-Id      | 1.2.840.113556.1.4.21                                                            |
| System-ID-GUID    | bf96793d-0de6-11d0-a285-00aa003049e2                                             |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md)                                      |
