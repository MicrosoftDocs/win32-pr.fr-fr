# <a name="object-class-attribute"></a>Attribut Object-Class

Liste des classes à partir desquelles cette classe est dérivée.



| Entrée | Valeur |
|-------------------|-----------------------------------------------------------------|
| CN                | Object-Class                                                    |
| LDAP-Display-Name | objectClass                                                     |
| Taille              | Environ 20 octets en moyenne.                                      |
| Mettre à jour le privilège  | Le concepteur de l’objet définit cette valeur.                |
| Fréquence des mises à jour  | Cette valeur ne doit jamais changer.                                 |
| Attribute-Id      | 2.5.4.0                                                         |
| System-ID-GUID    | bf9679e5-0de6-11d0-a285-00aa003049e2                            |
| Syntaxe            | [**String(Object-Identifier)**](s-string-object-identifier.md) |
