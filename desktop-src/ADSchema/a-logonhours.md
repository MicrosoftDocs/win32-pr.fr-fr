# <a name="logon-hours-attribute"></a>Attribut Logon-Hours

Heures auxquelles l’utilisateur est autorisé à se connecter au domaine.



| Entrée | Valeur |
|-------------------|-------------------------------------------------------|
| CN                | Logon-Hours                                           |
| LDAP-Display-Name | logonHours                                            |
| Taille              | \-                                                    |
| Mettre à jour le privilège  | Administrateur de domaine                                  |
| Fréquence des mises à jour  | Chaque fois que les heures d’ouverture de session du compte doivent être modifiées.   |
| Attribute-Id      | 1.2.840.113556.1.4.64                                 |
| System-ID-GUID    | bf9679ab-0de6-11d0-a285-00aa003049e2                  |
| Syntaxe            | [**Object(Replica-Link)**](s-object-replica-link.md) |
