# <a name="netboot-server-attribute"></a>attribut netboot-Server

Nom unique d’un serveur NetBoot.



| Entrée | Valeur |
|-------------------|-----------------------------------------|
| CN                | Serveur NetBoot                          |
| LDAP-Display-Name | netbootServer                           |
| Taille              | \-                                      |
| Mettre à jour le privilège  | Cette valeur est définie par le système.        |
| Fréquence des mises à jour  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.860                  |
| System-ID-GUID    | 07383081-91df-11d1-aebc-0000f80367c1    |
| Syntaxe            | [**Object(DS-DN)**](s-object-ds-dn.md) |
