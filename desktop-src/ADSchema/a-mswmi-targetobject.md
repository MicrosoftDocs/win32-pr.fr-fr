# <a name="ms-wmi-targetobject-attribute"></a>attribut ms-WMI-TargetObject

Cela contient une ou plusieurs séquences binaires qui représentent des objets WMI compilés.



| Entrée | Valeur |
|-------------------|-------------------------------------------------------|
| CN                | MS-WMI-TargetObject                                   |
| LDAP-Display-Name | msWMI-TargetObject                                    |
| Taille              | \-                                                    |
| Mettre à jour le privilège  | Cette valeur est définie par le système.                      |
| Fréquence des mises à jour  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.1647                               |
| System-ID-GUID    | c44f67a5-7de5-4a1f-92d9-662b57364b77                  |
| Syntaxe            | [**Object(Replica-Link)**](s-object-replica-link.md) |
