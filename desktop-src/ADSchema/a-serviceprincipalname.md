# <a name="service-principal-name-attribute"></a>Attribut de nom de principal du service

Liste des noms principaux utilisés pour l’authentification mutuelle avec une instance d’un service sur cet ordinateur.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | Nom de principal du service                      |
| LDAP-Display-Name | servicePrincipalName                        |
| Taille              | \-                                          |
| Mettre à jour le privilège  | \-                                          |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.771                      |
| System-ID-GUID    | f3a64788-5306-11d1-a9c5-0000f80367c1        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
