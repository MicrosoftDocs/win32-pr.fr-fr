# <a name="shadowflag-attribute"></a>attribut shadowFlag

Contient la section du mappage d’ombre qui est utilisée pour stocker la valeur de l’indicateur.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | shadowFlag                           |
| LDAP-Display-Name | shadowFlag                           |
| Taille              | \-                                   |
| Mettre à jour le privilège  | \-                                   |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.11                     |
| System-ID-GUID    | 8dfeb70d-c5db-46b6-b15e-a4389e6cee9b |
| Syntaxe            | [**Énumération**](s-enumeration.md) |
