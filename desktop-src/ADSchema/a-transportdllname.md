# <a name="transport-dll-name-attribute"></a>Attribut de nom de la DLL de transport

Nom de la DLL qui gérera un transport.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | Transport-DLL-Name                          |
| LDAP-Display-Name | transportDLLName                            |
| Taille              | \-                                          |
| Mettre à jour le privilège  | Cette valeur est définie par le système.            |
| Fréquence des mises à jour  | Lors de la connexion de sites.                      |
| Attribute-Id      | 1.2.840.113556.1.4.789                      |
| System-ID-GUID    | 26d97372-6070-11d1-a9c6-0000f80367c1        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
