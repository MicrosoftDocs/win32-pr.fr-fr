# <a name="jpegphoto-attribute"></a>attribut jpegPhoto

Utilisé pour stocker une ou plusieurs images d’une personne à l’aide du format d’échange de fichiers JPEG \[ JFIF \] .

|-------------------|-------------------------------------------------------|
| CN                | jpegPhoto                                             |
| LDAP-Display-Name | jpegPhoto                                             |
| Taille              | \-                                                    |
| Mettre à jour le privilège  | \-                                                    |
| Fréquence des mises à jour  | \-                                                    |
| Attribute-Id      | 0.9.2342.19200300.100.1.60                            |
| System-ID-GUID    | bac80572-09c4-4fa9-9ae6-7628d7adbe0e                  |
