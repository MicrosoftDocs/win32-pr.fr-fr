# <a name="print-collate-attribute"></a>Attribut Print-Collate

**True** si une imprimante a des emplacements de classement.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | Print-Collate                        |
| LDAP-Display-Name | printCollate                         |
| Taille              | 4 octets                              |
| Mettre à jour le privilège  | \-                                   |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.242               |
| System-ID-GUID    | 281416d2-1968-11d0-a28f-00aa003049e2 |
| Syntaxe            | [**Boolean**](s-boolean.md)         |
