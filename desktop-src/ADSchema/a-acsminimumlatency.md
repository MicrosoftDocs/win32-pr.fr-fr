# <a name="acs-minimum-latency-attribute"></a>ACS-attribut de latence minimale

L’attribut **ACS-minimum-latence** est destiné à un usage interne uniquement. Basé sur RFC2210.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | ACS-latence minimale                  |
| LDAP-Display-Name | aCSMinimumLatency                    |
| Taille              | 8 octets                              |
| Mettre à jour le privilège  | \-                                   |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1316              |
| System-ID-GUID    | 9517fefb-3b90-11d2-90cc-00c04fd91ab1 |
| Syntaxe            | [**Défini**](s-interval.md)       |
