# <a name="documentversion-attribute"></a>attribut documentVersion

Le type d’attribut documentVersion spécifie le numéro de version d’un document.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | documentVersion                             |
| LDAP-Display-Name | documentVersion                             |
| Taille              | \-                                          |
| Mettre à jour le privilège  | \-                                          |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 0.9.2342.19200300.100.1.13                  |
| System-ID-GUID    | 94b3a8a9-d613-4cec-9aad-5fbcc1046b43        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
