# <a name="frs-update-timeout-attribute"></a>Attribut FRS-Update-Timeout

Durée maximale, en minutes, pendant laquelle attendre la fin de l’exécution d’une mise à jour.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | FRS-Update-Timeout                   |
| LDAP-Display-Name | fRSUpdateTimeout                     |
| Taille              | 4 octets                              |
| Mettre à jour le privilège  | \-                                   |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.485               |
| System-ID-GUID    | 1be8f172-a9ff-11d0-afe2-00c04fd930c9 |
| Syntaxe            | [**Énumération**](s-enumeration.md) |
