# <a name="service-dns-name-attribute"></a>Attribut de nom de service-DNS

Nom DNS à rechercher pour trouver un serveur exécutant ce service.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | Nom du service DNS                            |
| LDAP-Display-Name | serviceDNSName                              |
| Taille              | \-                                          |
| Mettre à jour le privilège  | \-                                          |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.657                      |
| System-ID-GUID    | 28630eb8-41d5-11d1-a9c1-0000f80367c1        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
