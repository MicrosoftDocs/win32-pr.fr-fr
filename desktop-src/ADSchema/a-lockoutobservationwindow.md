# <a name="lock-out-observation-window-attribute"></a>Verrou-out-observation-attribut Window

Plage de temps, en intervalles de 100 nanosecondes, dans laquelle le système incrémente le nombre d’ouvertures de session incorrect.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | Verrouillage-sortie-observation-fenêtre          |
| LDAP-Display-Name | lockOutObservationWindow             |
| Taille              | \-                                   |
| Mettre à jour le privilège  | Administrateur de domaine                 |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.61                |
| System-ID-GUID    | bf9679a4-0de6-11d0-a285-00aa003049e2 |
| Syntaxe            | [**Défini**](s-interval.md)       |
