# <a name="msmq-site-gates-attribute"></a>Attribut MSMQ-site-Gates

Liste de DNs pour les serveurs de routage MSMQ, à travers laquelle tout le trafic entre les sites doit être routé.



| Entrée | Valeur |
|-------------------|-----------------------------------------|
| CN                | MSMQ-site-portes                         |
| LDAP-Display-Name | mSMQSiteGates                           |
| Taille              | \-                                      |
| Mettre à jour le privilège  | \-                                      |
| Fréquence des mises à jour  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.945                  |
| System-ID-GUID    | 9a0dc339-c100-11d1-bbc5-0080c76670c0    |
| Syntaxe            | [**Object(DS-DN)**](s-object-ds-dn.md) |
