# <a name="documentlocation-attribute"></a>attribut documentLocation

Le type d’attribut documentLocation spécifie l’emplacement du document original.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | documentLocation                            |
| LDAP-Display-Name | documentLocation                            |
| Taille              | \-                                          |
| Mettre à jour le privilège  | \-                                          |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 0.9.2342.19200300.100.1.15                  |
| System-ID-GUID    | b958b14e-ac6d-4ec4-8892-be70b69f7281        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
