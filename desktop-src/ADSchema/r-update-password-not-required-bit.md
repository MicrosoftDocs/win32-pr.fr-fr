# <a name="update-password-not-required-bit-extended-right"></a>Update-Password-not-required-bit Extended Right

Droit d’accès du contrôle étendu qui permet à un utilisateur d’activer ou de désactiver le paramètre « mot de passe non requis » pour les objets utilisateur.



| Entrée | Valeur |
|--------------|--------------------------------------|
| CN           | Update-Password-not-required-bit     |
| Display-Name | Mettre à jour le mot de passe non requis     |
| Rights-GUID  | 280f369c-67c7-438e-ae98-1d46f3c6f541 |
