# <a name="organizationalstatus-attribute"></a>attribut organizationalStatus

Le type d’attribut organizationalStatus spécifie une catégorie par laquelle une personne est souvent désignée dans une organisation.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | organizationalStatus                        |
| LDAP-Display-Name | organizationalStatus                        |
| Taille              | \-                                          |
| Mettre à jour le privilège  | \-                                          |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 0.9.2342.19200300.100.1.45                  |
| System-ID-GUID    | 28596019-7349-4d2f-adff-5a629961f942        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
