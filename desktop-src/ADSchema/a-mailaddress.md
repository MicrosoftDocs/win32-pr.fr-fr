# <a name="smtp-mail-address-attribute"></a>Attribut adresse de messagerie SMTP

Attribut d’adresse de messagerie générique. Utilisé dans la zone comme un attribut facultatif des objets serveur, où ils sont utilisés par la réplication de service d’annuaire par courrier (si les ordinateurs sont configurés).



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | Adresse de messagerie SMTP                           |
| LDAP-Display-Name | mailAddress                                 |
| Taille              | \-                                          |
| Mettre à jour le privilège  | Administrateur de domaine                        |
| Fréquence des mises à jour  | Presque jamais                                |
| Attribute-Id      | 1.2.840.113556.1.4.786                      |
| System-ID-GUID    | 26d9736f-6070-11d1-a9c6-0000f80367c1        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
