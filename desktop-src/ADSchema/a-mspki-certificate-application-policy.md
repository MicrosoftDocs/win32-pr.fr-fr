# <a name="ms-pki-certificate-application-policy-attribute"></a>attribut ms-PKI-Certificate-application-Policy

Les OID de stratégie d’application d’un certificat.



| Entrée | Valeur |
|-------------------|--------------------------------------------------------------------------------------------|
| CN                | MS-PKI-Certificate-application-stratégie                                                      |
| LDAP-Display-Name | Attribut mspki-Certificate-application-Policy                                                       |
| Taille              | \-                                                                                         |
| Mettre à jour le privilège  | Administrateur d’entreprise                                                                   |
| Fréquence des mises à jour  | À chaque fois qu’un nouveau modèle est créé ou que les attributs d’un modèle existant sont modifiés. |
| Attribute-Id      | 1.2.840.113556.1.4.1674                                                                    |
| System-ID-GUID    | dbd90548-aa37-4202-9966-8c537ba5ce32                                                       |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md)                                                |
