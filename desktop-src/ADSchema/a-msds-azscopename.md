# <a name="ms-ds-az-scope-name-attribute"></a>ms-DS-AZ-Scope-Name (attribut)

Chaîne qui identifie de façon unique un objet d’étendue.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | ms-DS-AZ-Scope-Name                         |
| LDAP-Display-Name | msDS-AzScopeName                            |
| Taille              | \-                                          |
| Mettre à jour le privilège  | Administrateur AzRoles                               |
| Fréquence des mises à jour  | Pendant l’initialisation ou la modification de la stratégie.     |
| Attribute-Id      | 1.2.840.113556.1.4.1799                     |
| System-ID-GUID    | 515a6b06-2617-4173-8099-d5605df043c6        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
