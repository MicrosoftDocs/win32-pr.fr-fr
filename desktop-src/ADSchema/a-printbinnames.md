| CN                | Print-bin-Names                             |
| LDAP-Display-Name | printBinNames                               |
| Taille              | \-                                          |
| Mettre à jour le privilège  | \-                                          |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.237                      |
| System-ID-GUID    | 281416cd-1968-11d0-a28f-00aa003049e2        |
