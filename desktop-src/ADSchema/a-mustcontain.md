# <a name="must-contain-attribute"></a>Attribut Must-Contain

Liste des attributs obligatoires pour une classe. Ces attributs doivent être spécifiés lors de la création d’une instance de la classe.



| Entrée | Valeur |
|-------------------|-----------------------------------------------------------------|
| CN                | Must-Contain                                                    |
| LDAP-Display-Name | mustContain                                                     |
| Taille              | \-                                                              |
| Mettre à jour le privilège  | Administrateur de schéma                                            |
| Fréquence des mises à jour  | \-                                                              |
| Attribute-Id      | 1.2.840.113556.1.2.24                                           |
| System-ID-GUID    | bf9679d3-0de6-11d0-a285-00aa003049e2                            |
| Syntaxe            | [**String(Object-Identifier)**](s-string-object-identifier.md) |
