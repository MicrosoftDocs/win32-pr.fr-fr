# <a name="ca-connect-attribute"></a>Attribut CA-Connect

Chaîne de connexion pour la liaison à une autorité de certification.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | CA-Connect                                  |
| LDAP-Display-Name | Se connecter                                   |
| Taille              | \-                                          |
| Mettre à jour le privilège  | \-                                          |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.687                      |
| System-ID-GUID    | 963d2735-48be-11d1-a9c3-0000f80367c1        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
