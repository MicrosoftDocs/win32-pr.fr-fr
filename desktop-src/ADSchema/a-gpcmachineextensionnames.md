# <a name="gpc-machine-extension-names-attribute"></a>Attribut GPC-machine-extension-Names

Utilisé par l’objet stratégie de groupe pour les stratégies de l’ordinateur.



| Entrée | Valeur |
|-------------------|---------------------------------------------------------------------------------|
| CN                | GPC-machine-extension-Names                                                     |
| LDAP-Display-Name | gPCMachineExtensionNames                                                        |
| Taille              | Dépend du nombre d’extensions côté client qui ont une stratégie dans cet objet de stratégie de groupe. |
| Mettre à jour le privilège  | Administrateur de domaine ou de stratégie.                                                 |
| Fréquence des mises à jour  | Chaque fois qu’un objet de stratégie de groupe est mis à jour via le GPE.                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1348                                                         |
| System-ID-GUID    | 32ff8ecc-783f-11d2-9916-0000f87a57d4                                            |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md)                                     |
