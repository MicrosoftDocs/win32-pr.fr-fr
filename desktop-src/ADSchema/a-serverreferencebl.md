# <a name="server-reference-bl-attribute"></a>Serveur-référence-BL (attribut)

Trouvé dans le contexte d’appellation du domaine. Nom unique d’un ordinateur sous le dossier sites.



| Entrée | Valeur |
|-------------------|-----------------------------------------|
| CN                | Serveur-référence-BL                     |
| LDAP-Display-Name | serverReferenceBL                       |
| Taille              | \-                                      |
| Mettre à jour le privilège  | Cette valeur est définie par le système.        |
| Fréquence des mises à jour  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.516                  |
| System-ID-GUID    | 26d9736e-6070-11d1-a9c6-0000f80367c1    |
| Syntaxe            | [**Object(DS-DN)**](s-object-ds-dn.md) |
