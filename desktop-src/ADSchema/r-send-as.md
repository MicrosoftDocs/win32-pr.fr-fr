# <a name="send-as-extended-right"></a>Send-As le droit étendu

Exchange right : autorise l’envoi de courriers électroniques en tant que boîte aux lettres.



| Entrée | Valeur |
|--------------|--------------------------------------|
| CN           | Send-As                              |
| Display-Name | Envoyer en tant que                              |
| Rights-GUID  | ab721a54-1e2f-11d0-9819-00aa0040529b |
