# <a name="usn-last-obj-rem-attribute"></a>Attribut USN-Last-obj-REM

Contient le numéro de séquence de mise à jour (USN) pour le dernier objet non-système qui a été supprimé d’un serveur.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | USN-Last-obj-REM                     |
| LDAP-Display-Name | uSNLastObjRem                        |
| Taille              | 8 octets                              |
| Mettre à jour le privilège  | Cette valeur est définie par le système.     |
| Fréquence des mises à jour  | À chaque modification d’un objet d’annuaire. |
| Attribute-Id      | 1.2.840.113556.1.2.121               |
| System-ID-GUID    | bf967a73-0de6-11d0-a285-00aa003049e2 |
| Syntaxe            | [**Défini**](s-interval.md)       |
