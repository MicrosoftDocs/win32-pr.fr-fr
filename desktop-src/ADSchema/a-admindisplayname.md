# <a name="admin-display-name-attribute"></a>Attribut admin-Display-Name

Nom à afficher sur les écrans d’administration.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | Admin-Display-Name                          |
| LDAP-Display-Name | adminDisplayName                            |
| Taille              | \-                                          |
| Mettre à jour le privilège  | Administrateur de domaine                        |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.2.194                      |
| System-ID-GUID    | bf96791a-0de6-11d0-a285-00aa003049e2        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
