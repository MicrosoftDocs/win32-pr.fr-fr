# <a name="schema-id-guid-attribute"></a>Attribut Schema-ID-GUID

Identificateur unique d’un attribut.



| Entrée | Valeur |
|-------------------|---------------------------------------------------------------------|
| CN                | Schema-ID-GUID                                                      |
| LDAP-Display-Name | schemaIDGUID                                                        |
| Taille              | 16 octets                                                            |
| Mettre à jour le privilège  | Cette valeur est définie par le système.                                    |
| Fréquence des mises à jour  | Cette valeur est définie lorsque l’objet est créé et ne peut pas être modifiée. |
| Attribute-Id      | 1.2.840.113556.1.4.148                                              |
| System-ID-GUID    | bf967923-0de6-11d0-a285-00aa003049e2                                |
| Syntaxe            | [**Object(Replica-Link)**](s-object-replica-link.md)               |
