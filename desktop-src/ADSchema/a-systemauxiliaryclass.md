# <a name="system-auxiliary-class-attribute"></a>Attribut de classe système-auxiliaire

Liste des classes auxiliaires qui ne peuvent pas être modifiées par l’utilisateur.



| Entrée | Valeur |
|-------------------|--------------------------------------------------------------------|
| CN                | System-auxiliaire-Class                                             |
| LDAP-Display-Name | systemAuxiliaryClass                                               |
| Taille              | \-                                                                 |
| Mettre à jour le privilège  | Administrateur de schéma                                               |
| Fréquence des mises à jour  | Lorsque la classe est créée ou qu’une nouvelle classe auxiliaire y est ajoutée. |
| Attribute-Id      | 1.2.840.113556.1.4.198                                             |
| System-ID-GUID    | bf967a43-0de6-11d0-a285-00aa003049e2                               |
| Syntaxe            | [**String(Object-Identifier)**](s-string-object-identifier.md)    |
