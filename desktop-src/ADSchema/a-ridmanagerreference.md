# <a name="rid-manager-reference-attribute"></a>RID-Manager-attribut de référence

Nom unique du gestionnaire RID d’un objet.



| Entrée | Valeur |
|-------------------|-----------------------------------------|
| CN                | RID-Manager-référence                   |
| LDAP-Display-Name | rIDManagerReference                     |
| Taille              | \-                                      |
| Mettre à jour le privilège  | Cette valeur est définie par le système.        |
| Fréquence des mises à jour  | Chaque fois que le gestionnaire RID change.       |
| Attribute-Id      | 1.2.840.113556.1.4.368                  |
| System-ID-GUID    | 66171886-8f3c-11d0-afda-00c04fd930c9    |
| Syntaxe            | [**Object(DS-DN)**](s-object-ds-dn.md) |
