# <a name="user-force-change-password-extended-right"></a>Utilisateur-force-change-droit étendu de mot de passe

Autorise la réinitialisation d’un mot de passe sur un compte d’utilisateur.



| Entrée | Valeur |
|--------------|--------------------------------------|
| CN           | User-force-change-password           |
| Display-Name | Réinitialiser le mot de passe                       |
| Rights-GUID  | 00299570-246d-11d0-a768-00aa006e0529 |
