# <a name="ds-ui-admin-notification-attribute"></a>Attribut DS-UI-admin-notification

Il s’agit d’une liste des GUID des objets COM qui prennent en charge une interface de rappel que DSAdmin appelle lorsqu’une action a eu lieu sur un objet par le biais de l’interface utilisateur.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | DS-UI-admin-notification                    |
| LDAP-Display-Name | dSUIAdminNotification                       |
| Taille              | \-                                          |
| Mettre à jour le privilège  | Administrateur de domaine                        |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1343                     |
| System-ID-GUID    | f6ea0a94-6f91-11d2-9905-0000f87a57d4        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
