# <a name="querypoint-attribute"></a>Attribut QueryPoint

URL ou UNC d’une page de requête ou d’un autre serveur frontal pour accéder à un catalogue.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | QueryPoint                                  |
| LDAP-Display-Name | queryPoint                                  |
| Taille              | \-                                          |
| Mettre à jour le privilège  | \-                                          |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.680                      |
| System-ID-GUID    | 7bfdcb86-4807-11d1-a9c3-0000f80367c1        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
