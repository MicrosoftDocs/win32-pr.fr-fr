# <a name="omt-guid-attribute"></a>Attribut OMT-Guid

Identificateur unique d’une entrée de table de lien-Track-objet-Move.



| Entrée | Valeur |
|-------------------|-------------------------------------------------------|
| CN                | OMT-Guid                                              |
| LDAP-Display-Name | oMTGuid                                               |
| Taille              | 16 octets                                              |
| Mettre à jour le privilège  | Cette valeur est définie par le système.                      |
| Fréquence des mises à jour  | Chaque fois que le lien d’un fichier est modifié.                 |
| Attribute-Id      | 1.2.840.113556.1.4.505                                |
| System-ID-GUID    | ddac0cf3-af8f-11d0-afeb-00c04fd930c9                  |
| Syntaxe            | [**Object(Replica-Link)**](s-object-replica-link.md) |
