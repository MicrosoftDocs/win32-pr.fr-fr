# <a name="netboot-answer-only-valid-clients-attribute"></a>attribut netboot-réponse uniquement-valide-clients

Détermine si le serveur répond à tous les ordinateurs clients prédéfinis ou non.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | netboot-réponse seule-valide-clients    |
| LDAP-Display-Name | netbootAnswerOnlyValidClients        |
| Taille              | \-                                   |
| Mettre à jour le privilège  | Cette valeur est définie par le système.     |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.854               |
| System-ID-GUID    | 0738307b-91df-11d1-aebc-0000f80367c1 |
| Syntaxe            | [**Boolean**](s-boolean.md)         |
