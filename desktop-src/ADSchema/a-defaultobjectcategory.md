# <a name="default-object-category-attribute"></a>Attribut par défaut-objet-catégorie

Catégorie d’objet à utiliser pour l’objet, si aucune n’est spécifiée.



| Entrée | Valeur |
|-------------------|-----------------------------------------|
| CN                | Default-Object-catégorie                 |
| LDAP-Display-Name | defaultObjectCategory                   |
| Taille              | \-                                      |
| Mettre à jour le privilège  | Administrateur de schéma                    |
| Fréquence des mises à jour  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.783                  |
| System-ID-GUID    | 26d97367-6070-11d1-a9c6-0000f80367c1    |
| Syntaxe            | [**Object(DS-DN)**](s-object-ds-dn.md) |
