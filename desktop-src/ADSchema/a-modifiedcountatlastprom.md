# <a name="modified-count-at-last-prom-attribute"></a>Attribut modified-Count-at-Last-Prom

Numéro de série du journal des modifications de l’ouverture de session au cours de la dernière promotion.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | Modifié-Count-at-Last-Prom          |
| LDAP-Display-Name | modifiedCountAtLastProm              |
| Taille              | \-                                   |
| Mettre à jour le privilège  | Cette valeur est définie par le système.     |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.81                |
| System-ID-GUID    | bf9679c6-0de6-11d0-a285-00aa003049e2 |
| Syntaxe            | [**Défini**](s-interval.md)       |
