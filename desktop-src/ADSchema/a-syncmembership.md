# <a name="sync-membership-attribute"></a>Attribut Sync-Membership

Liste des membres contenus dans un groupe intégré SAM pour la synchronisation.



| Entrée | Valeur |
|-------------------|-----------------------------------------|
| CN                | Sync-Membership                         |
| LDAP-Display-Name | syncMembership                          |
| Taille              | \-                                      |
| Mettre à jour le privilège  | \-                                      |
| Fréquence des mises à jour  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.665                  |
| System-ID-GUID    | 037651e3-441d-11d1-a9c3-0000f80367c1    |
| Syntaxe            | [**Object(DS-DN)**](s-object-ds-dn.md) |
