# <a name="enabled-attribute"></a>Attribut activé

Cet attribut est utilisé pour indiquer si une crossRef donnée est activée.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | activé                              |
| LDAP-Display-Name | activé                              |
| Taille              | \-                                   |
| Mettre à jour le privilège  | Cette valeur est définie par le système.     |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.2.557               |
| System-ID-GUID    | a8df73f2-c5ea-11d1-bbcb-0080c76670c0 |
| Syntaxe            | [**Boolean**](s-boolean.md)         |
