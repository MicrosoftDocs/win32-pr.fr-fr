# <a name="frs-version-attribute"></a>Attribut FRS-Version

Le numéro de version et la date de génération.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | FRS-Version                                 |
| LDAP-Display-Name | fRSVersion                                  |
| Taille              | \-                                          |
| Mettre à jour le privilège  | \-                                          |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.882                      |
| System-ID-GUID    | 2a132585-9373-11d1-aebc-0000f80367c1        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
