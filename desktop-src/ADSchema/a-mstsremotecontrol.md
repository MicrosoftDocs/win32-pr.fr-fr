# <a name="ms-ts-remote-control-attribute"></a>attribut ms-TS-Remote-Control

Contrôle à distance des services Terminal Server spécifie s’il faut autoriser l’observation à distance ou le contrôle à distance de la session des services Terminal Server de l’utilisateur.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | MS-TS-Remote-Control                 |
| LDAP-Display-Name | msTSRemoteControl                    |
| Taille              | \-                                   |
| Mettre à jour le privilège  | \-                                   |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1980              |
| System-ID-GUID    | 15177226-8642-468b-8c48-03ddfd004982 |
| Syntaxe            | [**Énumération**](s-enumeration.md) |
