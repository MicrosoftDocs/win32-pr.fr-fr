# <a name="display-name-printable-attribute"></a>Display-Name-Printable (attribut)

Nom complet imprimable d’un objet. Le nom d’affichage imprimable est généralement la combinaison du prénom, de l’initiale du deuxième prénom et du nom de famille de l’utilisateur.



| Entrée | Valeur |
|-------------------|----------------------------------------|
| CN                | Nom complet-imprimable                 |
| LDAP-Display-Name | displayNamePrintable                   |
| Taille              | \-                                     |
| Mettre à jour le privilège  | Administrateur de domaine ou propriétaire du compte. |
| Fréquence des mises à jour  | \-                                     |
| Attribute-Id      | 1.2.840.113556.1.2.353                 |
| System-ID-GUID    | bf967954-0de6-11d0-a285-00aa003049e2   |
| Syntaxe            | [**String(IA5)**](s-string-ia5.md)    |
