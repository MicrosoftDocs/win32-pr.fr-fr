# <a name="ms-tapi-ip-address-attribute"></a>attribut ms-TAPI-IP-address

Adresses IP d’un ordinateur client TAPI auquel un utilisateur est connecté. Cet attribut peut être utilisé comme attribut générique pour contenir des adresses IP d’ordinateur.



| Entrée | Valeur |
|-------------------|------------------------------------------------------------|
| CN                | MS-TAPI-IP-adresse                                         |
| LDAP-Display-Name | msTAPI-IpAddress                                           |
| Taille              | Chaque adresse IP est stockée sous forme de chaîne dans une notation. B. C. D. |
| Mettre à jour le privilège  | Aucun privilège spécial n’est requis.                            |
| Fréquence des mises à jour  | Généralement très faible.                                        |
| Attribute-Id      | 1.2.840.113556.1.4.1701                                    |
| System-ID-GUID    | efd7d7f7-178e-4767-87fa-f8a16b840544                       |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md)                |
