# <a name="transport-type-attribute"></a>Attribut Transport-Type

Nom unique d’un type de transport utilisé pour connecter des sites. Cette valeur peut pointer vers un transport IP ou SMTP.



| Entrée | Valeur |
|-------------------|-----------------------------------------|
| CN                | Transport-Type                          |
| LDAP-Display-Name | transportType                           |
| Taille              | \-                                      |
| Mettre à jour le privilège  | Cette valeur est définie par le système.        |
| Fréquence des mises à jour  | Lors de la connexion de sites.                  |
| Attribute-Id      | 1.2.840.113556.1.4.791                  |
| System-ID-GUID    | 26d97374-6070-11d1-a9c6-0000f80367c1    |
| Syntaxe            | [**Object(DS-DN)**](s-object-ds-dn.md) |
