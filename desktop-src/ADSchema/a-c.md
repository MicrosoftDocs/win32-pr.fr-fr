# <a name="country-name-attribute"></a>Attribut Country-Name

Pays/région de l’adresse de l’utilisateur. Le pays ou la région est représenté sous la forme d’un code à 2 caractères basé sur ISO-3166.



| Entrée | Valeur |
|-------------------|---------------------------------------------|
| CN                | Country-Name                                |
| LDAP-Display-Name | c                                           |
| Taille              | \-                                          |
| Mettre à jour le privilège  | \-                                          |
| Fréquence des mises à jour  | \-                                          |
| Attribute-Id      | 2.5.4.6                                     |
| System-ID-GUID    | bf967945-0de6-11d0-a285-00aa003049e2        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
