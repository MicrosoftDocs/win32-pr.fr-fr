# <a name="is-deleted-attribute"></a>Attribut Is-Deleted

Si la **valeur est true**, cet objet a été marqué pour suppression et ne peut pas être instancié. Une fois la période de désactivation expirée, elle sera supprimée du système.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | Is-Deleted                           |
| LDAP-Display-Name | isDeleted                            |
| Taille              | \-                                   |
| Mettre à jour le privilège  | Administrateur de domaine                 |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.2.48                |
| System-ID-GUID    | bf96798f-0de6-11d0-a285-00aa003049e2 |
| Syntaxe            | [**Boolean**](s-boolean.md)         |
