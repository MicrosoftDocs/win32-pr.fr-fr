# <a name="ms-dfsr-keywords-attribute"></a>attribut ms-DFSR-Keywords

Contient les mots clés définis par l’utilisateur.

|-------------------|---------------------------------------------|
| CN                | MS-DFSR-Mots clés                            |
| LDAP-Display-Name | msDFSR-Mots clés                             |
| Taille              | \-                                          |
| Mettre à jour le privilège  | \-                                          |
| Fréquence des mises à jour  | \-                                          |

| System-ID-GUID    | 048b4692-6227-4b67-a074-c4437083e14b        |
| Syntaxe            | [**String(Unicode)**](s-string-unicode.md) |
