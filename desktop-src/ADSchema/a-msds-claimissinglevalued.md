# <a name="ms-ds-claim-is-single-valued-attribute"></a>ms-DS-claim-is-attribut à valeur unique

Pour un objet de type de revendication, cet attribut identifie si le type de revendication ou la propriété de ressource ne peut contenir qu’une seule valeur.

| Mettre à jour le privilège  | \-                                   |

| System-ID-GUID    | cd789fb9-96b4-4648-8219-ca378161af38 |
