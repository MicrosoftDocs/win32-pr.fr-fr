# <a name="default-group-attribute"></a>Attribut Default-Group

Groupe auquel cet objet est assigné lorsqu’il est créé.



| Entrée | Valeur |
|-------------------|-----------------------------------------|
| CN                | Default-Group                           |
| LDAP-Display-Name | defaultGroup                            |
| Taille              | \-                                      |
| Mettre à jour le privilège  | Cette valeur est définie par le système.        |
| Fréquence des mises à jour  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.480                  |
| System-ID-GUID    | 720bc4e2-a54a-11d0-afdf-00c04fd930c9    |
| Syntaxe            | [**Object(DS-DN)**](s-object-ds-dn.md) |
