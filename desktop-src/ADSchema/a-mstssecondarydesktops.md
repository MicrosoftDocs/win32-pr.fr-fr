# <a name="ms-ts-secondary-desktops-attribute"></a>attribut ms-TS-Secondary-Desktops

Cet attribut représente le tableau de liens suivant vers les bureaux secondaires d’un utilisateur.

|-------------------|-----------------------------------------|
| CN                | MS-TS-secondaire-postes de travail                |
| LDAP-Display-Name | msTSSecondaryDesktops                   |
| Taille              | \-                                      |
| Mettre à jour le privilège  | \-                                      |
| Fréquence des mises à jour  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.2075                 |
| System-ID-GUID    | f63aa29a-bb31-48e1-bfab-0a6c5a1d39c2    |
