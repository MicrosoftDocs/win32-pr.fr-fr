# <a name="associateddomain-attribute"></a>attribut associatedDomain

Le type d’attribut associatedDomain spécifie un domaine DNS associé à un objet.



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | associatedDomain                     |
| LDAP-Display-Name | associatedDomain                     |
| Taille              | \-                                   |
| Mettre à jour le privilège  | \-                                   |
| Fréquence des mises à jour  | \-                                   |
| Attribute-Id      | 0.9.2342.19200300.100.1.37           |
| System-ID-GUID    | 3320fc38-c379-4c17-a510-1bdf6133c5da |
| Syntaxe            | [**String(IA5)**](s-string-ia5.md)  |
