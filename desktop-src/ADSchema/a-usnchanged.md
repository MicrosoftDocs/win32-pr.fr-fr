# <a name="usn-changed-attribute"></a>Attribut USN-Changed

Numéro de séquence de mise à jour (USN) affecté par le répertoire local pour la modification la plus récente, y compris la création. Voir aussi, [**USN-créé**](a-usncreated.md).



| Entrée | Valeur |
|-------------------|--------------------------------------|
| CN                | USN-Changed                          |
| LDAP-Display-Name | uSNChanged                           |
| Taille              | 8 octets                              |
| Mettre à jour le privilège  | Cette valeur est définie par le système.     |
| Fréquence des mises à jour  | Lorsque l’objet est modifié.          |
| Attribute-Id      | 1.2.840.113556.1.2.120               |
| System-ID-GUID    | bf967a6f-0de6-11d0-a285-00aa003049e2 |
| Syntaxe            | [**Défini**](s-interval.md)       |
