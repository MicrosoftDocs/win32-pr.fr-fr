# <a name="a"></a>A

## <a name="additive-primary-colors"></a>couleurs primaires additives

Couleurs qui peuvent être ajoutées au noir pour mélanger une nouvelle couleur. Les couleurs primaires additives sont rouge, vert et bleu.
