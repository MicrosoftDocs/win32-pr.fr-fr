# <a name="t"></a>T

## <a name="tint"></a>teinté

La teinte d’une couleur est créée en mélangeant une teinte avec le blanc.

## <a name="tone"></a>nuance

Le ton d’une couleur est obtenu en mélangeant sa teinte avec du gris.
