# <a name="subtractive-primary-colors"></a>Couleurs primaires soustractives

La méthode de mélange de couleurs soustrait est basée sur l’hypothèse que vous commencez avec le blanc. Toutes les couleurs sont présentes dans des montants égaux. Si vous soustrayez le cyan et le jaune du blanc, l’image obtenue est magenta. Dans une image verte, le fait de soustraire du cyan entraîne la modification de l’image en jaune.

Les [couleurs principales](p.md) soustraites sont cyan, jaune et magenta. L’illustration suivante montre la combinaison de couleurs primaires soustraites.
