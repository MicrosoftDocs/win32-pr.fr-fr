# <a name="color-space-constants"></a>Constantes des espaces de couleurs

Valeur de signature qui indique l’espace colorimétrique dans lequel les données de profil sont définies. Peut prendre l’une des valeurs suivantes.



| Classe de profil     | Signature         |
|-------------------|-------------------|
| XYZ               | ESPACE \_ xyz        |
| Laboratoire               | ESPACE \_ Lab        |
| Luv               | ESPACE \_ Luv        |
| YCbCr             | ESPACE \_ YCbCr      |
| Yxy               | ESPACE \_ Yxy        |
| RGB               | ESPACE \_ RVB        |
| Nuances de gris        | ESPACE \_ gris       |
| HSV               | ESPACE \_ HSV        |
| HLS               | ESPACE \_ TLS        |
| CMJN              | ESPACE \_ CMJN       |
| CMJ               | ESPACE \_ CMJ        |
| Canal générique 2 | \_Canal espace 2 \_ |
| Canal générique 3 | \_Canal espace 3 \_ |
| Canal générique 4 | Canal d’espace \_ 4 \_ |
| Canal 5 générique | Canal d’espace \_ 5 \_ |
| Canal générique 6 | ESPACE \_ 6 \_ canal |
| Canal 7 générique | Canal d’espace \_ 7 \_ |
| Canal générique 8 | ESPACE \_ 8 \_ canal |
