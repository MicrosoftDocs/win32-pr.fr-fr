# <a name="r"></a>R

## <a name="rendering-intent"></a>intention de rendu

Approches du rendu de la couleur d’un espace de couleurs à un autre. Les intentions de rendu utilisées dans les profils ICC sont définies par l’International Color Consortium (ICC).

## <a name="rgb"></a>RGB

Espace de couleurs rouge, vert et bleu.

## <a name="rgb-triple"></a>Triple RVB

Quantité qui contient les valeurs d’espace colorimétrique RVB.
