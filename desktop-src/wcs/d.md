# <a name="d"></a>D

## <a name="dac"></a>DAC

Convertisseur numérique à analogique.

## <a name="device-link-profile"></a>Profil de lien de l’appareil

Fichier qui contient une concaténation des conversions de couleurs qui sont souvent utilisées.

## <a name="device-profile"></a>Profil d’appareil

Fichier qui contient des informations sur la manière de caractériser les capacités de couleur d’un appareil. Les profils d’appareil ICC convertissent les couleurs de la gamme d’un appareil en ordinateurs. Il contient également les informations nécessaires à la conversion des couleurs des PC dans la gamme de l’appareil. Les profils d’appareil XML WCS fournissent les données de mesure et les références algorithmiques pour la conversion entre l’espace de l’appareil natif et CIEXYZ.

## <a name="dmp"></a>DMP

Le format de profil de modèle de périphérique en couleurs est un format de fichier XML qui contient les mesures nécessaires pour décrire la relation entre les valeurs d’espace de couleurs de l’appareil natif et les valeurs CIEXYZ.
