# <a name="wcs-10-availability"></a>Disponibilité de WCS 1.0

ICM1 est disponible pour Microsoft Windows 95. Elle n’est pas disponible pour MS DOS, Windows 3. *x* ou Windows NT 4,0 ou une version antérieure.

ICM2 est disponible pour Microsoft Windows 98, Windows Millennium Edition (me), Windows 2000 et Windows XP. Elle n’est pas disponible pour MS DOS, Windows 3. *x*, Windows 95 ou windows NT 4,0 ou version antérieure.

WCS 1,0 est disponible pour Microsoft Windows Vista. Elle n’est pas disponible pour Windows XP ou version antérieure.
