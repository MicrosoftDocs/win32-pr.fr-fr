# <a name="h"></a>H

## <a name="hard-proofing"></a>vérification matérielle

L’aperçu ou la vérification d’une image en imprimant une copie définitive.

## <a name="hls"></a>HLS

Espace de couleurs souvent utilisé par les artistes. Ses composants sont la teinte, la luminosité et la saturation (Chroma).

## <a name="hsv"></a>HSV

Espace de couleurs souvent utilisé par les artistes. Ses composants sont teinte, saturation (chrominance) et valeur (clarté).

## <a name="hue"></a>Hue

Couleur pure. Une teinte est une position spécifique sur la partie visible du spectre électromagnétique.
