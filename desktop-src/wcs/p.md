# <a name="p"></a>P

## <a name="primary-colors"></a>couleurs primaires

Trois couleurs essentielles dans un espace colorimétrique qui peuvent être mélangées pour produire toutes les autres couleurs.

## <a name="profile-connection-space-pcs"></a>Espace de connexion du profil (PCS)

Espace de couleurs indépendant du périphérique utilisé pour les conversions de couleurs et la correspondance des couleurs.
