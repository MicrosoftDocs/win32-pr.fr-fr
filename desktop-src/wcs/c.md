# <a name="c"></a>C

## <a name="camp"></a>Camping

Le format de profil de modèle d’apparence de couleur est un format de fichier XML qui contient des conditions d’affichage requises pour décrire la relation entre les valeurs DMP CIEXYZ et les valeurs CIEJab.

## <a name="cmyk"></a>CMJN

L’espace de couleurs Cyan, magenta, jaune et noir. Il est souvent implémenté sur les imprimantes.

## <a name="colorant"></a>couleur

Une chose, en particulier un pigment ou une encre, qui colore ou modifie la teinte d’un autre point.

## <a name="color-channel"></a>canal de couleur

Composant d’un espace de couleurs. Par exemple, un espace de couleurs RVB a des canaux de couleur rouge, vert et bleu.

## <a name="color-conversion"></a>conversion de couleurs

Processus de conversion des couleurs d’un espace de couleurs à un autre.

## <a name="color-management-module-cmm"></a>Module de gestion des couleurs (CMM)

Module de code qui utilise des profils d’appareil pour effectuer une conversion de couleur et un mappage de couleurs.

## <a name="color-mapping"></a>mappage des couleurs

Consultez correspondance des couleurs.

## <a name="color-matching"></a>correspondance des couleurs

Faire correspondre une couleur convertie à sa couleur la plus proche visuellement dans l’espace colorimétrique de destination.

## <a name="color-model"></a>modèle de couleurs

Consultez espace colorimétrique.

## <a name="color-space"></a>espace colorimétrique

Mappage des composants de couleur sur un système de coordonnées géométriques en trois dimensions.
