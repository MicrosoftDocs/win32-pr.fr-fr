# <a name="systemproplistnonpersonal"></a>System. PropList. prepersonive

Liste des propriétés considérées comme « non personnelles ». Le système laisse ces propriétés intactes lorsqu’il est demandé de supprimer toutes les propriétés non personnelles d’un fichier donné. Inscrivez-vous sous le regValue de « personne ».

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.PropList.NonPersonal
   shellPKey = PKEY_PropList_NonPersonal
   formatID = 49D1091F-082E-493F-B23F-D2308AA9668C
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## <a name="remarks"></a>Notes

Les valeurs de la valeur de l’une sont définies dans propKey. h.
