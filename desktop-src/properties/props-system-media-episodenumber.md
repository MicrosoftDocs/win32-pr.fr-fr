# <a name="systemmediaepisodenumber"></a>System. Media. EpisodeNumber

Nombre de 1 à incrémentation monotone qui correspond à l’épisode du diaporama

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8"></a>Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Media.EpisodeNumber
   shellPKey = PKEY_Media_EpisodeNumber
   formatID = 64440492-4C8B-11D1-8B70-080036B11A03
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = UInt32
```

## <a name="remarks"></a>Notes

Les valeurs de la valeur de l’une sont définies dans propKey. h.
