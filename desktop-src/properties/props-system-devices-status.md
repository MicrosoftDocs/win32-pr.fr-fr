# <a name="systemdevicesstatus"></a>System. Devices. Status

Tableau de chaînes d’état des appareils

```
propertyDescription
   name = System.Devices.Status
   shellPKey = PKEY_Devices_Status
   formatID = D08DD4C0-3A9E-462E-8290-7B636B2576B9
   propID = 259
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Multivalue String
      IsInnate = true
```
