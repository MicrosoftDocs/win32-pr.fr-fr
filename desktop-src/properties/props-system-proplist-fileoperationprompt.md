# <a name="systemproplistfileoperationprompt"></a>System. PropList. FileOperationPrompt

Liste des propriétés à afficher dans la boîte de dialogue de confirmation de l’opération de fichier. Les propriétés avec des valeurs vides ne sont pas affichées. Si cette liste n’est pas spécifiée, la liste de propriétés info-bulle est utilisée à la place. Inscrivez-vous sous le regValue de « FileOperationPrompt ».

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.PropList.FileOperationPrompt
   shellPKey = PKEY_PropList_FileOperationPrompt
   formatID = C9944A21-A406-48FE-8225-AEC7E24C211B
   propID = 10
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## <a name="remarks"></a>Notes

Les valeurs de la valeur de l’une sont définies dans propKey. h.
