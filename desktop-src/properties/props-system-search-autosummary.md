# <a name="systemsearchautosummary"></a>System. Search. Résumé

Résumé système de recherche automatisée du contenu de texte intégral d’un document, affiché dans l’affichage des résultats de la recherche dans l' **Explorateur de recherche**. Utilisez cette propriété uniquement lors de la consommation des valeurs dans une application, et non lors de la fourniture de valeurs à un gestionnaire de propriétés.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Search.AutoSummary
   shellPKey = PKEY_Search_AutoSummary
   formatID = 560C36C0-503A-11CF-BAA1-00004C752A9A
   propID = 2
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = String
      IsInnate = true
```

## <a name="remarks"></a>Notes

Les valeurs de la valeur de l’une sont définies dans propKey. h.
