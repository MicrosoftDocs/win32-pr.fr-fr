# <a name="systemsupplementalalbumid"></a>System. Supplemental. AlbumID

Contient les identificateurs des albums dont l’élément est membre. Peut être utilisé conjointement avec l’élément d’album dans les API de l’indexeur de contenu pour notifier à d’autres applications les albums d’images que l’utilisateur a créés ou que les applications ont déjà créées.

## <a name="windows-10-version-1703"></a>Windows 10 version 1703

```
propertyDescription
   name = System.Supplemental.AlbumID
   shellPKey = PKEY_Supplemental_AlbumID
   formatID = 0C73B141-39D6-4653-A683-CAB291EAF95B
   propID = 2
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = Multivalue String
```

## <a name="remarks"></a>Remarques

Les valeurs de la valeur de l’une sont définies dans propKey. h.
