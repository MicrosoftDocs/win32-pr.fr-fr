# <a name="systemdevicesaepservicebluetoothgattservicecachemode"></a>System. Devices. AepService. Bluetooth. GattService. CacheMode

Définit le mode de cache du GATT Bluetooth pour la requête.

## <a name="windows-10-version-1511"></a>Windows 10, version 1511

```
propertyDescription
   name = System.Devices.AepService.Bluetooth.GattService.CacheMode
   shellPKey = PKEY_Devices_AepService_Bluetooth_GattService_CacheMode
   formatID = 9744311E-7951-4B2E-B6F0-ECB293CAC119
   propID = 5
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt16
      IsInnate = true
```

## <a name="remarks"></a>Notes

Les valeurs de la valeur de l’une sont définies dans propKey. h.
