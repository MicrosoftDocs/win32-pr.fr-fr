# <a name="systemimageresolutionunit"></a>System. image. ResolutionUnit

Indique les unités de résolution. Utilisé pour les images avec des proportions non carrées, mais sans dimensions absolues significatives. 1 = aucune unité absolue de mesure. 2 = pouces. 3 = centimètres. La valeur par défaut est 2 (pouces).

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Image.ResolutionUnit
   shellPKey = PKEY_Image_ResolutionUnit
   formatID = 19B51FA6-1F92-4A5C-AB48-7DF0ABD67444
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Int16
      IsInnate = true
```

## <a name="remarks"></a>Notes

Les valeurs de la valeur de l’une sont définies dans propKey. h.
