# <a name="systemphotomakernote"></a>System. photo. MakerNote

Mécanisme d’extensibilité EXIF (Exchangeable Image File) qui permet aux fabricants d’appareils photo de fournir des informations personnalisées. Cette propriété n’est pas destinée à être affichée dans le shell, mais elle est disponible par programmation pour les applications.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Photo.MakerNote
   shellPKey = PKEY_Photo_MakerNote
   formatID = FA303353-B659-4052-85E9-BCAC79549B84
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Buffer
```

## <a name="remarks"></a>Notes

Les valeurs de la valeur de l’une sont définies dans propKey. h.
