# <a name="systemdevicesaepispaired"></a>System.Devices.Aep.IsPaired

Si le point de terminaison d’association d’appareils est associé au système ou non

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507

```
propertyDescription
   name = System.Devices.Aep.IsPaired
   shellPKey = PKEY_Devices_Aep_IsPaired
   formatID = A35996AB-11CF-4935-8B61-A6761081ECDF
   propID = 16
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## <a name="remarks"></a>Notes

Les valeurs de la valeur de l’une sont définies dans propKey. h.
