# <a name="systemdeviceinterfacewinusbdeviceinterfaceclasses"></a>System. DeviceInterface. WinUsb. DeviceInterfaceClasses

GUID (s) d’interface d’appareil WinUSB utilisé pour ouvrir un descripteur de l’appareil.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81"></a>Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1

```
propertyDescription
   name = System.DeviceInterface.WinUsb.DeviceInterfaceClasses
   shellPKey = PKEY_DeviceInterface_WinUsb_DeviceInterfaceClasses
   formatID = 95E127B5-79CC-4E83-9C9E-8422187B3E0E
   propID = 7
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Multivalue String
```

## <a name="remarks"></a>Notes

Les valeurs de la valeur de l’une sont définies dans propKey. h.
