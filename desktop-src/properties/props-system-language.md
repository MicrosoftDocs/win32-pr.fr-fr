# <a name="systemlanguage"></a>System. langue

Langue principale du fichier, en particulier si ce fichier est un document.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81"></a>Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1

```
propertyDescription
   name = System.Language
   shellPKey = PKEY_Language
   formatID = D5CDD502-2E9C-101B-9397-08002B2CF9AE
   propID = 28
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = String
```

## <a name="windows-8-windows-7-windows-vista"></a>Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Language
   shellPKey = PKEY_Language
   formatID = D5CDD502-2E9C-101B-9397-08002B2CF9AE
   propID = 28
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
```

## <a name="remarks"></a>Remarques

Les valeurs de la valeur de l’une sont définies dans propKey. h.
