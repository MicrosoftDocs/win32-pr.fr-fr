# <a name="systemdeviceinterfacehidusagepage"></a>System. DeviceInterface. HID. UsagePage

Page utilisation de l’appareil HID.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81"></a>Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1

```
propertyDescription
   name = System.DeviceInterface.Hid.UsagePage
   shellPKey = PKEY_DeviceInterface_Hid_UsagePage
   formatID = CBF38310-4A17-4310-A1EB-247F0B67593B
   propID = 2
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt16
      IsInnate = true
```

## <a name="remarks"></a>Remarques

Les valeurs de la valeur de l’une sont définies dans propKey. h.
