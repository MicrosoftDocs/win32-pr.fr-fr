# <a name="systemcontactpagertelephone"></a>System. contact. PagerTelephone

Numéro de téléphone de radiomessagerie du contact.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Contact.PagerTelephone
   shellPKey = PKEY_Contact_PagerTelephone
   formatID = D6304E01-F8F5-4F45-8B15-D024A6296789
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
```

## <a name="remarks"></a>Remarques

Les valeurs de la valeur de l’une sont définies dans propKey. h.
