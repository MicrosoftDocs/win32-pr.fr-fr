# <a name="systemparentalrating"></a>System. ParentalRating

Classement parental stocké dans un format généralement déterminé par l’organisation nommée dans [System. ParentalRatingsOrganization](./props-system-parentalratingsorganization.md).

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.ParentalRating
   shellPKey = PKEY_ParentalRating
   formatID = 64440492-4C8B-11D1-8B70-080036B11A03
   propID = 21
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
```

## <a name="remarks"></a>Notes

Les valeurs de la valeur de l’une sont définies dans propKey. h.

Cette chaîne s’affiche directement à l’utilisateur, de sorte que les mots conviviaux doivent être utilisés. Par exemple, « R-Restricted » lors de la base des évaluations sur le système d’évaluation MPAA (motion image Association of America).
