# <a name="systemgpsaltitude"></a>System. GPS. altitude

Indique l’altitude en fonction de la référence dans le \_ \_ AltitudeRef GPS. Calculé à partir \_ de \_ AltitudeNumerator GPS et \_ du \_ AltitudeDenominator GPS.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.GPS.Altitude
   shellPKey = PKEY_GPS_Altitude
   formatID = 827EDB4F-5B73-44A7-891D-FDFFABEA35CA
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Double
      IsInnate = true
```

## <a name="remarks"></a>Notes

Les valeurs de la valeur de l’une sont définies dans propKey. h.

La spécification d’une référence de chaîne indirecte spécifique pour l' `label` attribut de **labelInfo** a été ajoutée pour Windows Vista avec Service Pack 1 (SP1).
