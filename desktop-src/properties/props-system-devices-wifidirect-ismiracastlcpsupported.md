# <a name="systemdeviceswifidirectismiracastlcpsupported"></a>System. Devices. WiFiDirect. IsMiracastLcpSupported

indique si la protection du contenu des liens est prise en charge par le Wi-Fi appareil Direct s’il est Miracast compatible

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507

```
propertyDescription
   name = System.Devices.WiFiDirect.IsMiracastLcpSupported
   shellPKey = PKEY_Devices_WiFiDirect_IsMiracastLcpSupported
   formatID = 1506935D-E3E7-450F-8637-82233EBE5F6E
   propID = 9
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## <a name="remarks"></a>Remarques

Les valeurs de la valeur de l’une sont définies dans propKey. h.
