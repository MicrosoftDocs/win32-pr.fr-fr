# <a name="systemphotofocalplaneyresolution"></a>System. photo. FocalPlaneYResolution

Indique le nombre de pixels de la hauteur d’image (direction Y) par FocalPlaneResolutionUnit sur le plan focal de l’appareil photo. Calculé à partir de la photo FocalPlaneYResolutionNumerator et de la FocalPlaneYResolutionDenominator de la \_ \_ \_ \_ .

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Photo.FocalPlaneYResolution
   shellPKey = PKEY_Photo_FocalPlaneYResolution
   formatID = 4FFFE4D0-914F-4AC4-8D6F-C9C61DE169B1
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Double
      IsInnate = true
```

## <a name="remarks"></a>Notes

Les valeurs de la valeur de l’une sont définies dans propKey. h.
