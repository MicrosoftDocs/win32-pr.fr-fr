# <a name="systemcategory"></a>System. Category

Action déconseillée. Catégorie qui peut être assignée à un élément tel qu’un document ou un fichier. Cette propriété est héritée des propriétés de document OLE et est déconseillée pour Windows Vista. Les mots clés doivent être utilisés à la place. Le code plus ancien traite cette propriété comme VT \_ LPSTR.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Category
   shellPKey = PKEY_Category
   formatID = D5CDD502-2E9C-101B-9397-08002B2CF9AE
   propID = 2
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = Multivalue String
```

## <a name="remarks"></a>Notes

Les valeurs de la valeur de l’une sont définies dans propKey. h.
