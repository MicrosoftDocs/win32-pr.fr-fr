# <a name="systemdevicesaepcontainersupportscapturing"></a>System. Devices. AepContainer. SupportsCapturing

Indique si le conteneur de point de terminaison d’association d’appareils prend en charge la capture

## <a name="windows-10-version-1703-windows-10-version-1607"></a>Windows 10, version 1703, Windows 10, version 1607

```
propertyDescription
   name = System.Devices.AepContainer.SupportsCapturing
   shellPKey = PKEY_Devices_AepContainer_SupportsCapturing
   formatID = 6AF55D45-38DB-4495-ACB0-D4728A3B8314
   propID = 11
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## <a name="remarks"></a>Remarques

Les valeurs de la valeur de l’une sont définies dans propKey. h.
