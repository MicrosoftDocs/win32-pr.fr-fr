# <a name="systemphotomakernoteoffset"></a>System. photo. MakerNoteOffset

Décalage pour la note Maker spécifiée dans [System. photo. MakerNote](./props-system-photo-makernote.md). Pour plus d’informations, consultez la spécification du fichier image (EXIF) 2,2. Cette propriété n’est pas destinée à être affichée dans le shell, mais elle est disponible par programmation pour les applications.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Photo.MakerNoteOffset
   shellPKey = PKEY_Photo_MakerNoteOffset
   formatID = 813F4124-34E6-4D17-AB3E-6B1F3C2247A1
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt64
```

## <a name="remarks"></a>Remarques

Les valeurs de la valeur de l’une sont définies dans propKey. h.
