# <a name="systemgpsimgdirection"></a>System. GPS. ImgDirection

Indique la direction de l’image lorsqu’elle a été capturée. Calculé à partir \_ de \_ ImgDirectionNumerator GPS et \_ du \_ ImgDirectionDenominator GPS.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.GPS.ImgDirection
   shellPKey = PKEY_GPS_ImgDirection
   formatID = 16473C91-D017-4ED9-BA4D-B6BAA55DBCF8
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Double
      IsInnate = true
```

## <a name="remarks"></a>Remarques

Les valeurs de la valeur de l’une sont définies dans propKey. h.
