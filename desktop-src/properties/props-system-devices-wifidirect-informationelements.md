# <a name="systemdeviceswifidirectinformationelements"></a>System. Devices. WiFiDirect. InformationElements

Indique le jeu complet des s fournis par le Wi-Fi appareil direct

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507

```
propertyDescription
   name = System.Devices.WiFiDirect.InformationElements
   shellPKey = PKEY_Devices_WiFiDirect_InformationElements
   formatID = 1506935D-E3E7-450F-8637-82233EBE5F6E
   propID = 12
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Buffer
      IsInnate = true
```

## <a name="remarks"></a>Remarques

Les valeurs de la valeur de l’une sont définies dans propKey. h.
