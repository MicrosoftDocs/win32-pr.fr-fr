# <a name="systemphotodigitalzoom"></a>System. photo. DigitalZoom

Taux de zoom numérique lors de la capture de l’image. Lire à partir de l’appareil photo dans les informations EXIF (exchanged image file) du fichier. Cette propriété est calculée à partir de [System. photo. DigitalZoomNumerator](./props-system-photo-digitalzoomnumerator.md) et de [System. photo. DigitalZoomDenominator](./props-system-photo-digitalzoomdenominator.md). Si le numérateur de la valeur enregistrée est 0, aucun zoom numérique n’a été utilisé.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Photo.DigitalZoom
   shellPKey = PKEY_Photo_DigitalZoom
   formatID = F85BF840-A925-4BC2-B0C4-8E36B598679E
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = Double
      IsInnate = true
```

## <a name="remarks"></a>Notes

Les valeurs de la valeur de l’une sont définies dans propKey. h.
