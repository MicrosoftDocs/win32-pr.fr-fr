# <a name="systemdevicesaepserviceparentaepispaired"></a>System.Devices.AepService.ParentAepIsPaired

Si le point de terminaison d’association d’appareil parent est associé au système ou non

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507

```
propertyDescription
   name = System.Devices.AepService.ParentAepIsPaired
   shellPKey = PKEY_Devices_AepService_ParentAepIsPaired
   formatID = C9C141A9-1B4C-4F17-A9D1-F298538CADB8
   propID = 7
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## <a name="remarks"></a>Remarques

Les valeurs de la valeur de l’une sont définies dans propKey. h.
