# <a name="systemcommunicationtaskstatustext"></a>System. communication. TaskStatusText

Forme conviviale de [System. communication. TaskStatus](./props-system-communication-taskstatus.md). Cette valeur n’est pas destinée à être analysée par programmation.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Communication.TaskStatusText
   shellPKey = PKEY_Communication_TaskStatusText
   formatID = A6744477-C237-475B-A075-54F34498292A
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
      IsInnate = true
```

## <a name="remarks"></a>Remarques

Les valeurs de la valeur de l’une sont définies dans propKey. h.
