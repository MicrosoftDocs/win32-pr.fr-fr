# <a name="systempropgroupdescription"></a>System. PropGroup. Description

Séparateur de groupes de propriétés utilisé dans les listes de propriétés pour séparer les descriptions d’autres types. N’utilisez pas cette propriété pour obtenir ou définir des valeurs. Cette propriété est destinée uniquement à un marqueur.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.PropGroup.Description
   shellPKey = PKEY_PropGroup_Description
   formatID = 8969B275-9475-4E00-A887-FF93B8B41E44
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Null
```

## <a name="remarks"></a>Remarques

Les valeurs de la valeur de l’une sont définies dans propKey. h.
