# <a name="systemdevicesaepcontainerprotocolids"></a>System.Devices.AepContainer.ProtocolIds

Liste des ID de protocole ayant contribué à la création du conteneur de point de terminaison d’association d’appareils

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507

```
propertyDescription
   name = System.Devices.AepContainer.ProtocolIds
   shellPKey = PKEY_Devices_AepContainer_ProtocolIds
   formatID = 0BBA1EDE-7566-4F47-90EC-25FC567CED2A
   propID = 13
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Multivalue Guid
      IsInnate = true
```

## <a name="remarks"></a>Notes

Les valeurs de la valeur de l’une sont définies dans propKey. h.
