# <a name="systemphotoorientationtext"></a>System. photo. OrientationText

Forme conviviale de [System. photo. orientation](./props-system-photo-orientation.md). Non destiné à être analysé par programmation.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Photo.OrientationText
   shellPKey = PKEY_Photo_OrientationText
   formatID = A9EA193C-C511-498A-A06B-58E2776DCC28
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
      IsInnate = true
```

Les valeurs de la valeur de l’une sont définies dans propKey. h.
