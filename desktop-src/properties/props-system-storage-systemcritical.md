# <a name="systemstoragesystemcritical"></a>System.Storage.SystemCritical

Indique si le support de stockage est critique système.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8"></a>Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Storage.SystemCritical
   shellPKey = PKEY_Storage_SystemCritical
   formatID = 4D1EBEE8-0803-4774-9842-B77DB50265E9
   propID = 4
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
```

## <a name="remarks"></a>Remarques

Les valeurs de la valeur de l’une sont définies dans propKey. h.
