# <a name="systemlinktargetsfgaoflagsstrings"></a>System. Link. TargetSFGAOFlagsStrings

Exprime les indicateurs SFGAO d’un lien en tant que valeurs de chaîne et est utilisé comme une optimisation de requête. Consultez \_ SFGAOFlagsStrings Shell Shell \_ pour connaître les valeurs possibles de cette propriété.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Link.TargetSFGAOFlagsStrings
   shellPKey = PKEY_Link_TargetSFGAOFlagsStrings
   formatID = D6942081-D53B-443D-AD47-5E059D9CD27A
   propID = 3
   SearchInfo
      InInvertedIndex = true
      IsColumn = false
   typeInfo
      type = Multivalue String
```

## <a name="remarks"></a>Remarques

Les valeurs de la valeur de l’une sont définies dans propKey. h.
