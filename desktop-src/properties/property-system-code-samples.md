# <a name="property-system-code-samples"></a>Exemples de code du système de propriétés

les exemples de code suivants montrent comment utiliser le système de propriétés Windows.

| Rubrique            | Contenu                     |
|-------------------|---------------------------------------------------|
| [Exemple de gestionnaire de propriétés idéal](https://github.com/microsoft/Windows-classic-samples/tree/master/Samples/Win7Samples/winui/shell/appshellintegration/IdealPropertyHandler)         | Illustre l’implémentation d’un gestionnaire de propriétés idéal pour un format de fichier OleDoc (. doc-MS) ou un format OpenMetadata fictif (. OpenMetadata-ms) qui prend en charge la lecture et l’écriture de propriétés et d’un schéma personnalisé. |
| [Exemple de gestionnaire de propriétés de sélection](https://github.com/microsoft/Windows-classic-samples/tree/master/Samples/Win7Samples/winui/shell/appshellintegration/PlaylistPropertyHandler)   | Illustre l’implémentation d’un gestionnaire de propriétés pour les formats de fichier de sélection. wpl et. ZPL.                                                                                                                          |
| [Exemple de modification de propriété](https://github.com/microsoft/Windows-classic-samples/tree/master/Samples/Win7Samples/winui/shell/appplatform/PropertyEdit)                          | montre comment utiliser les api du système de propriétés Windows pour lire et écrire des valeurs de propriété de fichier.                                                                                                                            |
| [Exemple de schéma de propriété](https://github.com/microsoft/Windows-classic-samples/tree/master/Samples/Win7Samples/winui/shell/appplatform/propertyschemas)                      | Montre comment inscrire et désinscrire des propriétés personnalisées et comment obtenir des informations détaillées sur les schémas de propriétés personnalisées qui sont déjà inscrits auprès du système.                                               |
| [Exemple de gestionnaire de propriétés Recipe](https://github.com/microsoft/Windows-classic-samples/tree/master/Samples/Win7Samples/winui/shell/appshellintegration/RecipePropertyHandler) | Illustre l’implémentation d’un gestionnaire de propriétés pour les fichiers de recette de format personnalisé (. Recipe).                                                                                                                             |

Pour obtenir des exemples connexes, consultez les rubriques suivantes :

-   [Windows Exemples d’applications Shell](https://github.com/microsoft/Windows-classic-samples/tree/master/Samples/Win7Samples/winui/shell/appplatform)
-   [Windows Exemples d’extensibilité de l’interpréteur de commandes](https://github.com/microsoft/Windows-classic-samples/tree/master/Samples/Win7Samples/winui/shell/shellextensibility)
-   [Windows Exemples d’intégration de Shell](https://github.com/microsoft/Windows-classic-samples/tree/master/Samples/Win7Samples/winui/shell/appshellintegration)
