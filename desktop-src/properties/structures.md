# <a name="structures-windows-property-system"></a>Structures (système de propriétés Windows)

cette section décrit les structures du système de propriétés Windows.



| Rubrique                                | Contenu                                                                                                                                    |
|--------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------|
| [**PROPERTYKEY**](/windows/win32/api/wtypes/ns-wtypes-propertykey) | Spécifie l’identificateur FMTID/PID qui identifie par programmation une propriété. Remplace [**SHCOLUMNID**](../shell/objects.md).<br/> |
| [**PROPPRG**](/windows/desktop/api/Shlobj_core/ns-shlobj_core-propprg)           | Cette structure contient des informations à partir d’un fichier. PIF. Elle est utilisée par [**PifMgr \_ GetProperties**](/windows/desktop/api/Shlobj_core/nf-shlobj_core-pifmgr_getproperties).<br/>       |
