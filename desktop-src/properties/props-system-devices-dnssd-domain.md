# <a name="systemdevicesdnssddomain"></a>System. Devices. dnssd. Domain

Partie domaine du nom de l’instance du service DNS-SD. (par exemple, « . local » dans «MyService. \_ http. \_ TCP. local»)

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507

```
propertyDescription
   name = System.Devices.Dnssd.Domain
   shellPKey = PKEY_Devices_Dnssd_Domain
   formatID = BF79C0AB-BB74-4CEE-B070-470B5AE202EA
   propID = 3
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
      IsInnate = true
```

## <a name="remarks"></a>Remarques

Les valeurs de la valeur de l’une sont définies dans propKey. h.
