# <a name="systemsearchrank"></a>System. Search. Rank

Classement de la ligne de pertinence, avec une plage comprise entre 0-1000. Les plus grands nombres signifient de meilleures correspondances. Requête-heure uniquement ; non défini dans le schéma de recherche. Cette propriété est récupérable, mais ne peut pas faire l’objet d’une recherche.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Search.Rank
   shellPKey = PKEY_Search_Rank
   formatID = 49691C90-7E17-101A-A91C-08002B2ECDA9
   propID = 3
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = Int32
      IsInnate = true
```

## <a name="remarks"></a>Remarques

Les valeurs de la valeur de l’une sont définies dans propKey. h.
