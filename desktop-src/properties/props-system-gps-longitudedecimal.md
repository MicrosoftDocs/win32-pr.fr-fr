# <a name="systemgpslongitudedecimal"></a>System. GPS. LongitudeDecimal

Indique la longitude basée sur la référence dans le \_ \_ LongitudeRef GPS. Calculé à partir de la \_ LongitudeNumerator GPS GPS \_ et du \_ LongitudeDenominator GPS GPS \_

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8"></a>Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.GPS.LongitudeDecimal
   shellPKey = PKEY_GPS_LongitudeDecimal
   formatID = 4679C1B5-844D-4590-BAF5-F322231F1B81
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = Double
      IsInnate = true
```

## <a name="remarks"></a>Remarques

Les valeurs de la valeur de l’une sont définies dans propKey. h.
