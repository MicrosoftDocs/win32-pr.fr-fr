# <a name="systemdevicesdnssdpriority"></a>System. Devices. dnssd. Priority

Champ de priorité d’enregistrement SRV. Rarement utilisé.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507

```
propertyDescription
   name = System.Devices.Dnssd.Priority
   shellPKey = PKEY_Devices_Dnssd_Priority
   formatID = BF79C0AB-BB74-4CEE-B070-470B5AE202EA
   propID = 9
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt16
      IsInnate = true
```

## <a name="remarks"></a>Remarques

Les valeurs de la valeur de l’une sont définies dans propKey. h.
