Numéro de téléphone du contact.

```
propertyDescription
   name = System.Contact.HomeTelephone
   shellPKey = PKEY_Contact_HomeTelephone
   formatID = 176DC63C-2688-4E89-8143-A347800F25E9
   propID = 20
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
```
