# <a name="systemcontactaccountpicturedynamicvideo"></a>System. contact. AccountPictureDynamicVideo

Il s’agit d’un flux contenant les données nécessaires pour afficher l’image du compte vidéo dynamique d’un contact.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8"></a>Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Contact.AccountPictureDynamicVideo
   shellPKey = PKEY_Contact_AccountPictureDynamicVideo
   formatID = 0B8BB018-2725-4B44-92BA-7933AEB2DDE7
   propID = 2
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = Stream
```

## <a name="remarks"></a>Remarques

Les valeurs de la valeur de l’une sont définies dans propKey. h.
