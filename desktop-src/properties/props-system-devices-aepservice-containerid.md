# <a name="systemdevicesaepservicecontainerid"></a>System.Devices.AepService.ContainerId

ID du conteneur parent du service de point de terminaison d’association d’appareils

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, version 1703, Windows 10, version 1607, Windows 10, version 1511, Windows 10, version 1507

```
propertyDescription
   name = System.Devices.AepService.ContainerId
   shellPKey = PKEY_Devices_AepService_ContainerId
   formatID = 71724756-3E74-4432-9B59-E7B2F668A593
   propID = 4
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Guid
      IsInnate = true
```

## <a name="remarks"></a>Remarques

Les valeurs de la valeur de l’une sont définies dans propKey. h.
