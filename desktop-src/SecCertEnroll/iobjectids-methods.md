# <a name="iobjectids-methods"></a>Méthodes IObjectIds

L’interface [**IObjectIds**](/windows/desktop/api/CertEnroll/nn-certenroll-iobjectids) expose les méthodes suivantes.

## <a name="in-this-section"></a>Contenu de cette section



| Rubrique                                                            | Description                                                                                     |
|------------------------------------------------------------------|-------------------------------------------------------------------------------------------------|
| [**Ajouter une méthode**](/windows/desktop/api/CertEnroll/nf-certenroll-iobjectids-add)<br/>           | Ajoute un objet [**IObjectId**](/windows/desktop/api/CertEnroll/nn-certenroll-iobjectid) à la collection.<br/>                     |
| [**AddRange, méthode**](/windows/desktop/api/CertEnroll/nf-certenroll-iobjectids-addrange)<br/> | Ajoute une plage d’objets [**IObjectId**](/windows/desktop/api/CertEnroll/nn-certenroll-iobjectid) à la collection.<br/>            |
| [**Clear, méthode**](/windows/desktop/api/CertEnroll/nf-certenroll-iobjectids-clear)<br/>       | supprime tous les objets [**IObjectId**](/windows/desktop/api/CertEnroll/nn-certenroll-iobjectid) de la collection.<br/>              |
| [**Remove (méthode)**](/windows/desktop/api/CertEnroll/nf-certenroll-iobjectids-remove)<br/>     | supprime un objet [**IObjectId**](/windows/desktop/api/CertEnroll/nn-certenroll-iobjectid) de la collection par la valeur d’index.<br/> |
