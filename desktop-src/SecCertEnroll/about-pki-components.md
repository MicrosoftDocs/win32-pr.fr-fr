# <a name="pki-elements"></a>Éléments PKI

Une infrastructure de clé publique (PKI) Microsoft classique comprend les éléments suivants.

| Élément                                                                     | Description                                                                                                                                                                                              |
|-----------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Autorités de certification](about-certification-authorities.md)<br/> | Fournir des services qui authentifient l’identité des personnes, des ordinateurs et d’autres entités d’un réseau. Cela englobe à la fois les autorités de certification racines et les autorités subordonnées.<br/> |
| [Répertoire de certificat](about-certificate-directory.md)<br/>         | Enregistre les demandes de certificat et les certificats émis et révoqués, ainsi que les demandes de certificat.<br/>                                                                                                      |
| [Serveur de récupération de clé](about-key-recovery-server.md)<br/>             | Enregistre les clés privées chiffrées dans la base de données de certificats pour la récupération après perte.<br/>                                                                                                             |
