# <a name="boolean-certificate-enrollment-api"></a>BOOLÉEN (API d’inscription de certificats)

Une valeur booléenne peut avoir la valeur **true** ou **false**. **False** est encodé sous la forme d’un tripleon TLV dans lequel le champ de **valeur** est égal à zéro (0). **True** est encodé sous la forme d’une valeur différente de zéro. La **balise** pour le type de données **booléen** est 0x01.
