# <a name="certificate-enrollment-api-reference"></a>Référence de l’API d’inscription de certificats

Les pages de référence de l’API d’inscription de certificats contiennent des descriptions détaillées des interfaces, des énumérations et d’autres éléments de programmation qui peuvent être utilisés pour inscrire un utilisateur ou un ordinateur dans une hiérarchie de certificats. Les pages de référence sont réparties dans les groupes suivants :

-   [CertEnroll, interfaces](certenroll-interfaces.md)
-   [Énumérations CertEnroll](certenroll-enumerations.md)
