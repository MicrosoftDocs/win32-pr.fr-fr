# <a name="samplecmplevelzero-directx-hlsl-texture-object"></a>SampleCmpLevelZero (objet texture HLSL DirectX)

Échantillonne une texture et compare le résultat à une valeur de comparaison. Cette fonction est identique à l’appel de [**SampleCmp**](dx-graphics-hlsl-to-samplecmp.md) sur le mipmap niveau 0 uniquement.
