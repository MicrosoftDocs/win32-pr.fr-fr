# <a name="endif-sm4---asm"></a>endif (SM4-ASM)

Met fin à une instruction [If](if--sm4---asm-.md) .



| endif |
|-------|



 

## <a name="remarks"></a>Remarques

L’exemple suivant montre comment utiliser l’instruction ENDIF.

``` syntax
                if     // any of the various forms of if* statements
                   ...
                else   // (optional)
                   ...
                endif
```

Le format de jeton contient le décalage de l’instruction **If** correspondante dans le nuanceur pour des raisons pratiques.

Cette instruction s’applique aux étapes suivantes du nuanceur :



| Nuanceur de sommets | Nuanceur de géométrie | Nuanceur de pixels |
|---------------|-----------------|--------------|
| x             | x               | x            |
