# <a name="texture2dsamplelevel-methods"></a>Texture2D :: SampleLevel, méthodes

Échantillonne un [**Texture2D**](sm5-object-texture2d.md) sur le niveau de mipmap spécifié.

### <a name="overload-list"></a>Liste de surcharge



| Méthode                                                                             | Description                                                                                        |
|:-----------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------|
| [**SampleLevel (S, float, float, int)**](dx-graphics-hlsl-to-samplelevel.md)          | Échantillonne une texture au niveau du mipmap spécifié.<br/>                                        |
| [**SampleLevel (S, float, float, int, uint)**](samplelevel-s-float-float-int-uint-.md) | Échantillonne une texture sur le niveau de mipmap spécifié et retourne l’état de l’opération.<br/> |
