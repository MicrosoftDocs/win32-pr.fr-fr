# <a name="debugging-shaders-in-visual-studio"></a>Débogage des nuanceurs dans Visual Studio

le dernier outil de débogage des nuanceurs est désormais fourni sous la forme d’une fonctionnalité de Microsoft Visual Studio, appelée Visual Studio débogueur graphics. ce nouvel outil remplace le [PIX](/windows/desktop/directx-sdk--august-2009-) pour Windows outil. Visual Studio le débogueur graphics a beaucoup amélioré la convivialité, la prise en charge de Windows 8 et Direct3D 11,1 et l’intégration aux fonctionnalités Visual Studio traditionnelles, telles que les piles d’appels et le débogage de Windows pour le débogage [HLSL](dx-graphics-hlsl.md) . Pour plus d’informations sur cette nouvelle fonctionnalité, consultez [débogage DirectX Graphics](/visualstudio/debugger/visual-studio-graphics-diagnostics?view=vs-2015).
