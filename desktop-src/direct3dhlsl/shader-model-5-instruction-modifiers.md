# <a name="shader-model-5-instruction-modifiers"></a>Modificateurs d’instruction du Shader Model 5

-   [Negate](negate.md)

## <a name="instruction-result-modifiers"></a>Modificateurs de résultats d’instruction

-   [Saturer](saturate.md)
