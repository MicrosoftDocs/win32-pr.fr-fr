# <a name="texturecubearraysamplegrad-methods"></a>TextureCubeArray :: SampleGrad, méthodes

Échantillonne une texture à l’aide d’un dégradé pour influencer la façon dont l’emplacement de l’échantillon est calculé.

### <a name="overload-list"></a>Liste de surcharge



| Méthode                                                                                                      | Description                                                                                                                                                                                                        |
|:------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SampleGrad (S, float, float, float)**](dx-graphics-hlsl-to-samplegrad.md)                                   | Échantillonne une texture en utilisant un dégradé pour influencer la façon dont l’emplacement de l’échantillon est calculé.<br/>                                                                                                             |
| [**SampleGrad (S, float, float, float, float)**](tcubearray-samplegrad-s-float-float-float-float-.md)           | Échantillonne une texture en utilisant un dégradé pour influencer la façon dont l’emplacement de l’échantillon est calculé, avec une valeur facultative pour fixer des valeurs d’exemple de niveau de détail (LOD) à.<br/>                                     |
| [**SampleGrad (S, float, float, float, float, uint)**](tcubearray-samplegrad-s-float-float-float-float-uint-.md) | Échantillonne une texture en utilisant un dégradé pour influencer la façon dont l’emplacement de l’échantillon est calculé, avec une valeur facultative pour fixer des valeurs d’exemple de niveau de détail (LOD) à. Retourne l’état de l’opération.<br/> |
