# <a name="statements"></a>Instructions

Les instructions HLSL sont des combinaisons de variables et de fonctions qui composent des expressions. à l’instar d’une phrase, une instruction HLSL requiert des opérateurs qui déterminent la façon dont une expression sera évaluée. Vous pouvez créer des instructions individuelles dans un nuanceur ou créer plusieurs instructions dans un groupe appelé bloc d’instructions. Pour créer des instructions HLSL, vous devez connaître la syntaxe d’écriture des expressions et les opérateurs que vous utiliserez pour combiner des expressions.

-   [Expressions](dx-graphics-hlsl-expressions.md)
-   [Opérateurs](dx-graphics-hlsl-operators.md)
-   [Blocs d’instructions](dx-graphics-hlsl-statement-blocks.md)
