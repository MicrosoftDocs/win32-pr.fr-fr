# <a name="texture1dsamplegrad-methods"></a>Texture1D :: SampleGrad, méthodes

Échantillonne un [**Texture1D**](sm5-object-texture1d.md) à l’aide d’un dégradé pour influencer la façon dont l’emplacement de l’échantillon est calculé.

### <a name="overload-list"></a>Liste de surcharge



| Méthode                                                                                                       | Description                                                                                                                                                                                                        |
|:-------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SampleGrad (S, float, float, float, int)**](dx-graphics-hlsl-to-samplegrad.md)                                | Échantillonne une texture en utilisant un dégradé pour influencer la façon dont l’emplacement de l’échantillon est calculé.<br/>                                                                                                             |
| [**SampleGrad (S, float, float, float, int, float)**](t1d-samplegrad-s-float-float-float-int-float-.md)           | Échantillonne une texture en utilisant un dégradé pour influencer la façon dont l’emplacement de l’échantillon est calculé, avec une valeur facultative pour fixer des valeurs d’exemple de niveau de détail (LOD) à.<br/>                                     |
| [**SampleGrad (S, float, float, float, int, float, uint)**](t1d-samplegrad-s-float-float-float-int-float-uint-.md) | Échantillonne une texture en utilisant un dégradé pour influencer la façon dont l’emplacement de l’échantillon est calculé, avec une valeur facultative pour fixer des valeurs d’exemple de niveau de détail (LOD) à. Retourne l’état de l’opération.<br/> |
