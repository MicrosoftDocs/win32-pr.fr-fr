# <a name="endswitch-sm4---asm"></a>endswitch (SM4-ASM)

Termine une instruction [switch](switch--sm4---asm-.md) .



| endswitch |
|-----------|



 

## <a name="remarks"></a>Remarques

Le format de jeton contient le décalage de l’instruction switch correspondante dans le nuanceur pour des raisons pratiques.

Cette instruction s’applique aux étapes suivantes du nuanceur :



| Nuanceur de sommets | Nuanceur de géométrie | Nuanceur de pixels |
|---------------|-----------------|--------------|
| x             | x               | x            |
