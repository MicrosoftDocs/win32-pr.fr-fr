# <a name="texture2dsamplecmp-methods"></a>Texture2D :: SampleCmp, méthodes

Échantillonne un [**Texture2D**](sm5-object-texture2d.md)à l’aide d’une valeur de comparaison pour rejeter des exemples.

### <a name="overload-list"></a>Liste de surcharge



| Méthode                                                                                     | Description                                                                                                                                                                           |
|:-------------------------------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SampleCmp (S, float, float, int)**](dx-graphics-hlsl-to-samplecmp.md)                      | Échantillonne une texture en utilisant une valeur de comparaison pour rejeter des exemples.<br/>                                                                                                             |
| [**SampleCmp (S, float, float, int, float)**](samplecmp-s-float-float-int-float-.md)           | Échantillonne une texture, en utilisant une valeur de comparaison pour rejeter les exemples, avec une valeur facultative pour fixer des exemples de valeurs de niveau de détail (LOD) à.<br/>                                     |
| [**SampleCmp (S, float, float, int, float, uint)**](samplecmp-s-float-float-int-float-uint-.md) | Échantillonne une texture, en utilisant une valeur de comparaison pour rejeter les exemples, avec une valeur facultative pour fixer des exemples de valeurs de niveau de détail (LOD) à. Retourne l’état de l’opération.<br/> |
