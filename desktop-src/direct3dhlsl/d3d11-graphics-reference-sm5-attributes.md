# <a name="shader-model-5-attributes"></a>Attributs du modèle de nuanceur 5

Le Shader Model 5 implémente les attributs du Shader Model 4 et versions antérieures (consultez le kit de développement logiciel (SDK) DirectX), ainsi que les nouveaux attributs suivants :

-   [Domain](sm5-attributes-domain.md)
-   [earlydepthstencil](sm5-attributes-earlydepthstencil.md)
-   [instancié](sm5-attributes-instance.md)
-   [maxtessfactor](sm5-attributes-maxtessfactor.md)
-   [numThreads](sm5-attributes-numthreads.md)
-   [outputcontrolpoints](sm5-attributes-outputcontrolpoints.md)
-   [outputtopology](sm5-attributes-outputtopology.md)
-   [partitionnement](sm5-attributes-partitioning.md)
-   [patchconstantfunc](sm5-attributes-patchconstantfunc.md)

Tous ces attributs sont requis (à l’exception de l’attribut d' **instance** ). L’échec de leur utilisation entraînera des erreurs de compilation.
