# <a name="sv_stencilref"></a>SV \_ StencilRef

SV \_ StencilRef représente la valeur de référence du stencil de nuanceur de pixels actuel.

## <a name="type"></a>Type



| Type     |
|------|
| uint |



 

## <a name="remarks"></a>Remarques

La spécification de la valeur de référence du nuanceur dans le nuanceur de pixels est disponible à la fois dans D3D 11.3 et D3D12. Consultez :

-   [Valeur de référence du stencil spécifié pour le nuanceur D3D 11,3](/windows/desktop/direct3d11/shader-specified-stencil-reference-value)
-   [Valeur de référence du stencil spécifié par le nuanceur D3D12](/windows/desktop/direct3d12/shader-specified-stencil-reference-value)
