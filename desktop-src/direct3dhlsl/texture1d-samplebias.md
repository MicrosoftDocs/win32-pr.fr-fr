# <a name="texture1dsamplebias-methods"></a>Texture1D :: SampleBias, méthodes

Échantillonne un [**Texture1D**](sm5-object-texture1d.md), après avoir appliqué la valeur de biais au niveau du mipmap.

### <a name="overload-list"></a>Liste de surcharge



| Méthode                                                                                           | Description                                                                                                                                                                                  |
|:-------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SampleBias (S, float, float, int)**](dx-graphics-hlsl-to-samplebias.md)                          | Échantillonne une texture après avoir appliqué la valeur de biais au niveau de mipmap.<br/>                                                                                                             |
| [**SampleBias (S, float, float, int, float)**](t1d-samplebias-s-float-float-int-float-.md)           | Échantillonne une texture après avoir appliqué la valeur de biais au niveau de mipmap, avec une valeur facultative pour fixer des valeurs d’exemple de niveau de détail (LOD) à.<br/>                                     |
| [**SampleBias (S, float, float, int, float, uint)**](t1d-samplebias-s-float-float-int-float-uint-.md) | Échantillonne une texture après avoir appliqué la valeur de biais au niveau de mipmap, avec une valeur facultative pour fixer des valeurs d’exemple de niveau de détail (LOD) à. Retourne l’état de l’opération.<br/> |
