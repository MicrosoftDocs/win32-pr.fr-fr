# <a name="quadreadacrossdiagonal-function"></a>QuadReadAcrossDiagonal fonction)

Retourne la valeur locale spécifiée qui est lue à partir de la voie diagonalement opposée dans ce Quad.

## <a name="syntax"></a>Syntaxe


``` syntax
<type> QuadReadAcrossDiagonal(
   <type> localValue
);
```



## <a name="parameters"></a>Paramètres

<dl> <dt>

*localValue* 
</dt> <dd>

Type demandé.

</dd> </dl>

## <a name="return-value"></a>Valeur retournée

Valeur locale spécifiée qui est lue à partir de la voie diagonalement opposée dans ce Quad.

## <a name="remarks"></a>Notes

Pour plus d’informations sur les Quad, reportez-vous à [vue d’ensemble du Shader Model 6](hlsl-shader-model-6-0-features-for-direct3d-12.md).

Cette fonction est prise en charge à partir du nuanceur modèle 6,0 uniquement dans les nuanceurs de pixels et de calcul.
