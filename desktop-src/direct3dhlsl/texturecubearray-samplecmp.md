| Méthode                                                                                        | Description                                                                                                                                                                           |
|:----------------------------------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SampleCmp (S, float, float)**](dx-graphics-hlsl-to-samplecmp.md)                             | Échantillonne une texture en utilisant une valeur de comparaison pour rejeter des exemples.<br/>                                                                                                             |
| [**SampleCmp (S, float, float, float)**](tcubearray-samplecmp-s-float-float-float-.md)           | Échantillonne une texture, en utilisant une valeur de comparaison pour rejeter les exemples, avec une valeur facultative pour fixer des exemples de valeurs de niveau de détail (LOD) à.<br/>                                     |
| [**SampleCmp (S, float, float, float, uint)**](tcubearray-samplecmp-s-float-float-float-uint-.md) | Échantillonne une texture, en utilisant une valeur de comparaison pour rejeter les exemples, avec une valeur facultative pour fixer des exemples de valeurs de niveau de détail (LOD) à.
