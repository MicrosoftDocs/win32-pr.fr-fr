# <a name="discard-statement"></a>Instruction ignore

Ne génère pas le résultat du pixel actuel.

refuser



 

## <a name="parameters"></a>Paramètres

None

## <a name="remarks"></a>Remarques

Cette instruction ne peut être appelée qu’à partir d’un nuanceur de pixels ; elle n’est pas prise en charge dans un nuanceur de géométrie ou un nuanceur de sommets.
