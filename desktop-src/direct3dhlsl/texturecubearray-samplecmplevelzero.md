# <a name="texturecubearraysamplecmplevelzero-methods"></a>TextureCubeArray :: SampleCmpLevelZero, méthodes

Échantillonne une texture sur le niveau de mipmap 0 uniquement, en utilisant une valeur de comparaison pour rejeter les exemples.

### <a name="overload-list"></a>Liste de surcharge



| Méthode                                                                                              | Description                                                                                                                            |
|:----------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------|
| [**SampleCmpLevelZero (S, float, float)**](dx-graphics-hlsl-to-samplecmplevelzero.md)                 | Échantillonne une texture sur le niveau de mipmap 0 uniquement et compare le résultat à une valeur de comparaison.<br/>                                     |
| [**SampleCmpLevelZero (S, float, float, uint)**](tcubearray-samplecmplevelzero-s-float-float-uint-.md) | Échantillonne une texture sur le niveau de mipmap 0 uniquement et compare le résultat à une valeur de comparaison. Retourne l’état de l’opération.<br/> |
