# <a name="sv_domainlocation"></a>SV \_ DomainLocation

Définit l’emplacement sur la coque du point de domaine actuel en cours d’évaluation.

## <a name="type"></a>Type



|        |                |
|--------|----------------|
| Type   | Topologie d’entrée |
| float2 | correctif Quad     |
| float3 | correctif triple      |
| float2 | isoligne        |



 

## <a name="remarks"></a>Remarques

Cette valeur système est requise.

Cette fonction est prise en charge dans les types de nuanceurs suivants :



| Sommet | Forme | Domain | Géométrie | Pixel | Calcul |
|--------|------|--------|----------|-------|---------|
|        |      | x      |          |       |         |
