# <a name="hlsl-shader-model-51"></a>Modèle de nuanceur HLSL 5,1

Cette section décrit les fonctionnalités du modèle de nuanceur 5,1 telles qu’elles s’appliquent dans la pratique à D3D12. Tout le matériel DirectX 12 prend en charge le modèle de nuanceur 5,1.

## <a name="in-this-section"></a>Contenu de cette section



| Rubrique                                                                         | Description                                                                                                                                              |
|-------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Changements de bytecode dans SM 5.1](bytecode-changes-in-sm5-1.md)<br/>         | SM 5.1 change la façon dont les registres de ressources sont déclarés et référencés dans les instructions. <br/>                                                            |
| [Signature racine spécifiée par HLSL](hlsl-specified-root-signature.md)<br/> | Une [signature racine](/windows/desktop/direct3d12/root-signatures) (une table de clés de ressources et d’autres éléments pour D3D12) peut être spécifiée en langage HLSL en tant que chaîne. <br/> |



 

Pour plus d’informations sur les modifications apportées à la syntaxe du nuanceur, reportez-vous au [nuancier Model 5,1](shader-model-5-1.md).
