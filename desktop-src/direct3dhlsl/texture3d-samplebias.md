# <a name="texture3dsamplebias-methods"></a>Texture3D :: SampleBias, méthodes

Échantillonne une texture après avoir appliqué la valeur de biais au niveau de mipmap.

| Méthode                                                                                           | Description                                                                                                                                                                                  |
|:-------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SampleBias (S, float, float, int)**](dx-graphics-hlsl-to-samplebias.md)                          | Échantillonne une texture après avoir appliqué la valeur de biais au niveau de mipmap.<br/>                                                                                                             |
| [**SampleBias (S, float, float, int, float)**](t3d-samplebias-s-float-float-int-float-.md)           | Échantillonne une texture après avoir appliqué la valeur de biais au niveau de mipmap, avec une valeur facultative pour fixer des valeurs d’exemple de niveau de détail (LOD) à.<br/>                                     |
| [**SampleBias (S, float, float, int, float, uint)**](t3d-samplebias-s-float-float-int-float-uint-.md) | Échantillonne une texture après avoir appliqué la valeur de biais au niveau de mipmap, avec une valeur facultative pour fixer des valeurs d’exemple de niveau de détail (LOD) à. Retourne l’état de l’opération.<br/> |
