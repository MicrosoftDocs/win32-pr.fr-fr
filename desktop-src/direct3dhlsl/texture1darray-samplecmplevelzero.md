# <a name="texture1darraysamplecmplevelzero-methods"></a>Texture1DArray :: SampleCmpLevelZero, méthodes

Échantillonne une texture sur le niveau de mipmap 0 uniquement, en utilisant une valeur de comparaison pour rejeter les exemples.

### <a name="overload-list"></a>Liste de surcharge



| Méthode                                                                                                    | Description                                                                                                                            |
|:----------------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------|
| [**SampleCmpLevelZero (S, float, float, int)**](dx-graphics-hlsl-to-samplecmplevelzero.md)                   | Échantillonne une texture sur le niveau de mipmap 0 uniquement et compare le résultat à une valeur de comparaison.<br/>                                     |
| [**SampleCmpLevelZero (S, float, float, int, uint)**](t1darray-samplecmplevelzero-s-float-float-int-uint-.md) | Échantillonne une texture sur le niveau de mipmap 0 uniquement et compare le résultat à une valeur de comparaison. Retourne l’état de l’opération.<br/> |
