# <a name="texture2dsamplegrad-methods"></a>Texture2D :: SampleGrad, méthodes

Échantillonne un [**Texture2D**](sm5-object-texture2d.md) à l’aide d’un dégradé pour influencer la façon dont l’emplacement de l’échantillon est calculé.

### <a name="overload-list"></a>Liste de surcharge



| Méthode                                                                                                   | Description                                                                                                                                                                                                        |
|:---------------------------------------------------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SampleGrad (S, float, float, float, int)**](dx-graphics-hlsl-to-samplegrad.md)                            | Échantillonne une texture en utilisant un dégradé pour influencer la façon dont l’emplacement de l’échantillon est calculé.<br/>                                                                                                             |
| [**SampleGrad (S, float, float, float, int, float)**](samplegrad-s-float-float-float-int-float-.md)           | Échantillonne une texture en utilisant un dégradé pour influencer la façon dont l’emplacement de l’échantillon est calculé, avec une valeur facultative pour fixer des valeurs d’exemple de niveau de détail (LOD) à.<br/>                                     |
| [**SampleGrad (S, float, float, float, int, float, uint)**](samplegrad-s-float-float-float-int-float-uint-.md) | Échantillonne une texture en utilisant un dégradé pour influencer la façon dont l’emplacement de l’échantillon est calculé, avec une valeur facultative pour fixer des valeurs d’exemple de niveau de détail (LOD) à. Retourne l’état de l’opération.<br/> |
