# <a name="rwtexture2dloadint-function"></a>RWTexture2D :: Load (int), fonction

Lit les données de texture.

## <a name="syntax"></a>Syntaxe


``` syntax
 Load(
  in int Location
);
```



## <a name="parameters"></a>Paramètres

<dl> <dt>

*Emplacement* \[ dans\]
</dt> <dd>

Type : **int**

Emplacement de la texture.

</dd> </dl>

Tapez :

Le type de retour correspond au type dans la déclaration pour l’objet [**RWTexture2D**](sm5-object-rwtexture2d.md) .
