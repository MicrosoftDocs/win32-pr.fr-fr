# <a name="texture2dgatherblue-methods"></a>Texture2D :: GatherBlue, méthodes

Échantillonne un [**Texture2D**](sm5-object-texture2d.md) et retourne le composant bleu.

### <a name="overload-list"></a>Liste de surcharge



| Méthode                                                                                                  | Description                                                                                                                                      |
|:--------------------------------------------------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------------------------------|
| [**GatherBlue (S, float, int)**](sm5-object-texture2d-gatherblue.md)                                       | Retourne les composants bleus des quatre valeurs de Texel qui seraient utilisées dans une opération de filtrage bilinéaire.<br/>                                 |
| [**GatherBlue (S, float, int, uint)**](t2d-gatherblue-s-float-int-uint-.md)                                 | Retourne les composants bleus des quatre valeurs de Texel qui seraient utilisées dans une opération de filtrage bilinéaire, ainsi que l’état du mappage de mosaïques.<br/> |
| [**GatherBlue (S, float, Int2, Int2, Int2, Int2)**](t2d-gatherblue-s-float-int2-int2-int2-int2-.md)           | Retourne les composants bleus des quatre valeurs de Texel qui seraient utilisées dans une opération de filtrage bilinéaire.<br/>                                 |
| [**GatherBlue (S, float, Int2, Int2, Int2, Int2, uint)**](t2d-gatherblue-s-float-int2-int2-int2-int2-uint-.md) | Retourne les composants bleus des quatre valeurs de Texel qui seraient utilisées dans une opération de filtrage bilinéaire, ainsi que l’état du mappage de mosaïques.<br/> |
