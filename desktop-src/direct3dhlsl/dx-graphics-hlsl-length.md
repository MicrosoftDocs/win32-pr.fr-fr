# <a name="length"></a>length

Retourne la longueur du vecteur à virgule flottante spécifié.



| longueur *RET* (*x*) |
|-------------------|



 

## <a name="parameters"></a>Paramètres



| Élément                                                   | Description                                     |
|--------------------------------------------------------|-------------------------------------------------|
| <span id="x"></span><span id="X"></span>*x*<br/> | Vecteur à virgule flottante spécifié.<br/> |



 

## <a name="return-value"></a>Valeur renvoyée

Scalaire à virgule flottante qui représente la longueur du paramètre *x* .
