# <a name="texture2dgatheralpha-methods"></a>Texture2D :: GatherAlpha, méthodes

Échantillonne un [**Texture2D**](sm5-object-texture2d.md) et retourne le composant alpha.

### <a name="overload-list"></a>Liste de surcharge



| Méthode                                                                                                     | Description                                                                                                                                       |
|:-----------------------------------------------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------|
| [**GatherAlpha (S, float, int)**](sm5-object-texture2d-gatheralpha.md)                                        | Retourne les composants alpha des quatre valeurs de Texel qui seraient utilisées dans une opération de filtrage bilinéaire.<br/>                                 |
| [**GatherAlpha (S, float, int, uint)**](t2d-gatheralpha-s-float-int-uint-.md)                                  | Retourne les composants alpha des quatre valeurs de Texel qui seraient utilisées dans une opération de filtrage bilinéaire, ainsi que l’état du mappage de mosaïque.<br/> |
| [**GatherAlpha (S, float, Int2, Int2, Int2, Int2)**](t2d-gatheralpha-s-float-int2-int2-int2-int2-.md)            | Retourne les composants alpha des quatre valeurs de Texel qui seraient utilisées dans une opération de filtrage bilinéaire.<br/>                                 |
| [**GatherAlpha (S, float, Int2, Int2, Int2, Int2, uint)**](t2d-gatheralpha-s-float-int2-int2-int2-int2-uint-.md)  | Retourne les composants alpha des quatre valeurs de Texel qui seraient utilisées dans une opération de filtrage bilinéaire, ainsi que l’état du mappage de mosaïque.<br/> |
