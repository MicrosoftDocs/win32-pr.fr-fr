Type Texture2DMS (tel qu’il existe dans Shader Model 4) et variables de ressource.

| Rubrique                                                                                    | Description                                                                                 |
|------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------|
| [**GetDimensions**](sm5-object-texture2dms-getdimensions.md)<br/>                 | Retourne les dimensions de la ressource.<br/>                                          |
