# <a name="outputcontrolpoints"></a>outputcontrolpoints

Définit le nombre de points de contrôle de sortie (par thread) qui seront créés dans le nuanceur de coque.


```
outputcontrolpoints(X)
```



## <a name="remarks"></a>Remarques

Il s’agit du nombre de fois où la fonction principale est exécutée.

Cet attribut est pris en charge dans les types de nuanceurs suivants :



| Sommet | Forme | Domaine | Géométrie | Pixel | Calcul |
|--------|------|--------|----------|-------|---------|
|        | x    |        |          |       |         |
