# <a name="texture2dsamplecmplevelzero-methods"></a>Texture2D :: SampleCmpLevelZero, méthodes

Échantillonne un [**Texture2D**](sm5-object-texture2d.md) sur le mipmap niveau 0 uniquement, en utilisant une valeur de comparaison pour rejeter les exemples.

### <a name="overload-list"></a>Liste de surcharge



| Méthode                                                                                           | Description                                                                                                                            |
|:-------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------|
| [**SampleCmpLevelZero (S, float, float, int)**](dx-graphics-hlsl-to-samplecmplevelzero.md)          | Échantillonne une texture sur le niveau de mipmap 0 uniquement et compare le résultat à une valeur de comparaison.<br/>                                     |
| [**SampleCmpLevelZero (S, float, float, int, uint)**](samplecmplevelzero-s-float-float-int-uint-.md) | Échantillonne une texture sur le niveau de mipmap 0 uniquement et compare le résultat à une valeur de comparaison. Retourne l’état de l’opération.<br/> |
