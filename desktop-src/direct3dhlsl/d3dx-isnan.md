# <a name="d3dx_isnan-function"></a>D3DX \_ isNaN, fonction

Détermine si la valeur est une valeur NaN (pas un nombre).

## <a name="syntax"></a>Syntaxe

``` syntax
bool D3DX_IsNan(
   FLOAT _V
);
```

## <a name="parameters"></a>Paramètres

<dl> <dt>

*\_V* 
</dt> <dd>

Valeur spécifiée.

</dd> </dl>

## <a name="return-value"></a>Valeur retournée

true si NaN ; Sinon, false.
