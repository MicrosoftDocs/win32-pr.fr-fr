# <a name="statement-blocks"></a>Blocs d’instructions

Un bloc d’instructions contient une ou plusieurs instructions ; chaque instruction est composée d' [expressions](dx-graphics-hlsl-expressions.md) et d' [opérateurs](dx-graphics-hlsl-operators.md).


```
{
   statement 1;
   statement 2;
   ...
   statement n;
}
```



Un bloc d’instructions indique également la sous-étendue. Les variables déclarées dans un bloc d’instructions sont uniquement reconnues dans le bloc.
