# <a name="sv_outputcontrolpointid"></a>SV \_ OutputControlPointID

Définit l’index de l’ID du point de contrôle qui est traité par un appel du point d’entrée principal du nuanceur de coque.

## <a name="type"></a>Type



| Type     |
|------|
| uint |



 

## <a name="remarks"></a>Remarques

Cette valeur système est facultative.

Cette fonction est prise en charge dans les types de nuanceurs suivants :



| Sommet | Forme | Domaine | Géométrie | Pixel | Compute |
