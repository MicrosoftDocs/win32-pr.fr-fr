# <a name="instruction-modifiers-hlsl-reference"></a>Modificateurs d’instruction (référence HLSL)

Les modificateurs d’instruction affectent le résultat de l’instruction avant qu’elle ne soit écrite dans le registre. Le modèle de nuanceur 4 et le modèle de nuanceur 4,1 prennent en charge les modificateurs d’instruction suivants.
