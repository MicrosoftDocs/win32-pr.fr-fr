# <a name="partitioning"></a>partitionnement

Définit le schéma géométrique à utiliser dans le nuanceur de coque.


```
partitioning(X)    
```



## <a name="remarks"></a>Remarques

Peut être **Integer**, **fractionnaire \_ pair**, **fractionnaire \_ impair** ou **pow2**.

Cet attribut est pris en charge dans les types de nuanceurs suivants :



| Sommet | Forme | Domaine | Géométrie | Pixel | Calcul |
|--------|------|--------|----------|-------|---------|
|        | x    |        |          |       |         |
