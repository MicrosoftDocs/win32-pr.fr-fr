# <a name="texturecubesamplebias-methods"></a>TextureCube :: SampleBias, méthodes

Échantillonne une texture après avoir appliqué la valeur de biais au niveau de mipmap.

### <a name="overload-list"></a>Liste de surcharge



| Méthode                                                                                     | Description                                                                                                                                                                                  |
|:-------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SampleBias (S, float, float)**](dx-graphics-hlsl-to-samplebias.md)                        | Échantillonne une texture après avoir appliqué la valeur de biais au niveau de mipmap.<br/>                                                                                                             |
| [**SampleBias (S, float, float, float)**](tcube-samplebias-s-float-float-float-.md)           | Échantillonne une texture après avoir appliqué la valeur de biais au niveau de mipmap, avec une valeur facultative pour fixer des valeurs d’exemple de niveau de détail (LOD) à.<br/>                                     |
| [**SampleBias (S, float, float, float, uint)**](tcube-samplebias-s-float-float-float-uint-.md) | Échantillonne une texture après avoir appliqué la valeur de biais au niveau de mipmap, avec une valeur facultative pour fixer des valeurs d’exemple de niveau de détail (LOD) à. Retourne l’état de l’opération.<br/> |
