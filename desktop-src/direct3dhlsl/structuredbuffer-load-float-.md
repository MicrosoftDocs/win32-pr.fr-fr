# <a name="structuredbufferloadint-function"></a>StructuredBuffer :: Load (int), fonction

## <a name="syntax"></a>Syntaxe


``` syntax
 Load(
  in int Location
);
```



## <a name="parameters"></a>Paramètres

<dl> <dt>

*Emplacement* \[ dans\]
</dt> <dd>

Emplacement de la mémoire tampon.

</dd> </dl>
