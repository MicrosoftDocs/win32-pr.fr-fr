# <a name="variables"></a>Variables

Les variables HLSL sont similaires aux variables définies dans le langage de programmation C. Comme pour C, les variables ont des restrictions de nommage, ont des propriétés d’étendue qui dépendent de l’emplacement où elles sont déclarées et peuvent être associées à des métadonnées utilisateur. Comme C, il existe plusieurs types de données standard. Contrairement à C, il existe également des types de données supplémentaires définis par le langage HLSL pour optimiser les performances des vecteurs à 4 composants qui utilisent des mathématiques de matrice pour fonctionner sur des données graphiques 3D.
