# <a name="texm3x2pad---ps"></a>texm3x2pad-PS

Effectue la multiplication de la première ligne d’une multiplication de matrice à deux lignes. Cette instruction doit être combinée avec [texm3x2tex-PS](texm3x2tex---ps.md) ou [texm3x2depth-PS](texm3x2depth---ps.md). Pour plus d’informations sur l’utilisation de texmpad, reportez-vous à l’une de ces instructions.

## <a name="syntax"></a>Syntaxe



| tex3x2pad DST, SRC |
|--------------------|



 

where

-   l’heure d’été est le registre de destination.
-   SRC est un registre source.

## <a name="remarks"></a>Remarques



| Versions de nuanceur de pixels | 1\_1 | 1\_2 | 1 \_ 3 | 1\_4 | 2 \_ 0 | 2 \_ x | 2 \_ logiciels | 3 \_ 0 | 3 \_ logiciels |
|-----------------------|------|------|------|------|------|------|-------|------|-------|
| texm3x2pad            | x    | x    | x    |      |      |      |       |      |       |



 

Cette instruction ne peut pas être utilisée par elle-même.
