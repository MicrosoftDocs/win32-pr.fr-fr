# <a name="texture2darraysamplecmplevelzero-methods"></a>Texture2DArray :: SampleCmpLevelZero, méthodes

Échantillonne un [**Texture2DArray**](sm5-object-texture2darray.md) sur le mipmap niveau 0 uniquement, en utilisant une valeur de comparaison pour rejeter les exemples.

### <a name="overload-list"></a>Liste de surcharge



| Méthode                                                                                                    | Description                                                                                                                            |
|:----------------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------|
| [**SampleCmpLevelZero (S, float, float, int)**](dx-graphics-hlsl-to-samplecmplevelzero.md)                   | Échantillonne une texture sur le niveau de mipmap 0 uniquement et compare le résultat à une valeur de comparaison.<br/>                                     |
| [**SampleCmpLevelZero (S, float, float, int, uint)**](t2darray-samplecmplevelzero-s-float-float-int-uint-.md) | Échantillonne une texture sur le niveau de mipmap 0 uniquement et compare le résultat à une valeur de comparaison. Retourne l’état de l’opération.<br/> |
