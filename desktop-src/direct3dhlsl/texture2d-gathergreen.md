# <a name="texture2dgathergreen-methods"></a>Texture2D :: GatherGreen, méthodes

Échantillonne un [**Texture2D**](sm5-object-texture2d.md) et retourne le composant vert.

| Méthode                                                                                                     | Description                                                                                                                                       |
|:-----------------------------------------------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------|
| [**GatherGreen (S, float, int)**](sm5-object-texture2d-gathergreen.md)                                        | Retourne les composants verts des quatre valeurs de Texel qui seraient utilisées dans une opération de filtrage bilinéaire.<br/>                                 |
| [**GatherGreen (S, float, int, uint)**](t2d-gathergreen-s-float-int-uint-.md)                                  | Retourne les composants verts des quatre valeurs de Texel qui seraient utilisées dans une opération de filtrage bilinéaire, ainsi que l’état du mappage de mosaïques.<br/> |
| [**GatherGreen (S, float, Int2, Int2, Int2, Int2)**](t2d-gathergreen-s-float-float-int2-int2-int2-int2-.md)      | Retourne les composants verts des quatre valeurs de Texel qui seraient utilisées dans une opération de filtrage bilinéaire.<br/>                                 |
| [**GatherGreen (S, float, Int2, Int2, Int2, Int2, uint)**](t2d-gathergreen-s-float-int2-int2-int2-int2-uint-.md)  | Retourne les composants verts des quatre valeurs de Texel qui seraient utilisées dans une opération de filtrage bilinéaire, ainsi que l’état du mappage de mosaïques.<br/> |
