# <a name="hlsl-specified-root-signature"></a>Signature racine spécifiée par HLSL

Une [signature racine](/windows/desktop/direct3d12/root-signatures) (une table de clés de ressources et d’autres éléments pour D3D12) peut être spécifiée en langage HLSL en tant que chaîne.

Reportez-vous à la documentation D3D12 sur la [spécification des signatures racines en langage HLSL](/windows/desktop/direct3d12/specifying-root-signatures-in-hlsl) pour obtenir des informations complètes et un exemple de code.
