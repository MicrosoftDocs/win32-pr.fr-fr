# <a name="texturecubearraygather-methods"></a>TextureCubeArray :: Gather, méthodes

Retourne les composants rouges des quatre valeurs de Texel qui seraient utilisées dans une opération de filtrage bilinéaire.

Pour plus d’informations sur la description de l’instruction DXBC sous-jacente, consultez la documentation sur [gather4](./gather4--sm5---asm-.md) .

### <a name="overload-list"></a>Liste de surcharge



| Méthode                                                          | Description                                                                                                                                |
|:----------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------------------------|
| [**Regroupement (S, float)**](dx-graphics-hlsl-to-gather.md)           | Retourne les composants rouges de quatre valeurs de Texel (composant rouge uniquement) qui seraient utilisées dans une opération de filtrage bilinéaire. <br/>           |
| [**Gather (S, float, uint)**](tcubearray-gather-s-float-uint-.md) | Retourne les composants rouges des quatre valeurs de Texel qui seraient utilisées dans une opération de filtrage bilinéaire, ainsi que l’état du mappage de mosaïques.<br/> |
