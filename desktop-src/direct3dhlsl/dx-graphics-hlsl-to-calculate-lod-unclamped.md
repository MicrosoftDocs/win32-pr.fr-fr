# <a name="calculatelevelofdetailunclamped-directx-hlsl-texture-object"></a>CalculateLevelOfDetailUnclamped (objet texture HLSL DirectX)

Calcule le LOD sans embrider le résultat.



| RET Object. CalculateLevelOfDetailUnclamped (état de l’échantillonneur \_ S, float x); |
|--------------------------------------------------------------------------|



 

Cette fonction fonctionne de la même façon que [CalculateLevelOfDetail (DirectX HLSL texture Object)](dx-graphics-hlsl-to-calculate-lod.md), à ceci près qu’il n’y a pas de fixation sur le LOD calculé.
