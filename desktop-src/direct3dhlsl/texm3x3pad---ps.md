# <a name="texm3x3pad---ps"></a>texm3x3pad-PS

Effectue la multiplication de la première ou de la deuxième ligne d’une multiplication de matrice de trois lignes. Cette instruction doit être utilisée en association avec [texm3x3-PS](texm3x3---ps.md), [texm3x3spec-PS](texm3x3spec---ps.md), [texm3x3vspec-PS](texm3x3vspec---ps.md)ou [texm3x3tex-PS](texm3x3tex---ps.md).

## <a name="syntax"></a>Syntaxe



| texm3x3pad DST, SRC |
|---------------------|



 

where

-   l’heure d’été est le registre de destination.
-   SRC est un registre source.

## <a name="remarks"></a>Notes



| Versions de nuanceur de pixels | 1\_1 | 1\_2 | 1 \_ 3 | 1\_4 | 2 \_ 0 | 2 \_ x | 2 \_ logiciels | 3 \_ 0 | 3 \_ logiciels |
|-----------------------|------|------|------|------|------|------|-------|------|-------|
| texm3x3pad            | x    | x    | x    |      |      |      |       |      |       |



 

Cette instruction ne peut pas être utilisée par elle-même.
