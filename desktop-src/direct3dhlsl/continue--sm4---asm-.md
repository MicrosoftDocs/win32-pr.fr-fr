# <a name="continue-sm4---asm"></a>continue (SM4-ASM)

Poursuit l’exécution au début de la boucle en cours.



| continue |
|----------|



 

## <a name="remarks"></a>Notes

**continue** ne peut être utilisé qu’à l’intérieur d’une [boucle](loop--sm4---asm-.md) ou d’un [ENDLOOP](endloop--sm4---asm-.md).

L’exemple suivant montre comment utiliser l’instruction **continue** .


```
                loop
                    if_na r0.x
                        break
                    endif
                    if_z r1.x
                        ...
                        continue
                    endif
                    ...
                endloop
```



Le format de jeton contient le décalage de l’instruction de boucle correspondante dans le nuanceur pour des raisons pratiques.

Cette instruction s’applique aux étapes suivantes du nuanceur :



| Nuanceur de sommets | Nuanceur de géométrie | Nuanceur de pixels |
|---------------|-----------------|--------------|
| x             | x               | x            |
