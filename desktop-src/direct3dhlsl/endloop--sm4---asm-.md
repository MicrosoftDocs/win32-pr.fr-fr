# <a name="endloop-sm4---asm"></a>ENDLOOP (SM4-ASM)

Termine une instruction de boucle.



| ENDLOOP |
|---------|



 

## <a name="remarks"></a>Notes

L’exemple suivant montre comment utiliser cette instruction.

``` syntax
                loop
                    // example of termination condition
                    if_nz r0.x
                        break
                    endif
                    ...
                endloop
```

Le format de jeton contient le décalage de l’instruction de boucle correspondante dans le nuanceur pour des raisons pratiques.

Cette instruction s’applique aux étapes suivantes du nuanceur :



| Nuanceur de sommets | Nuanceur de géométrie | Nuanceur de pixels |
|---------------|-----------------|--------------|
| x             | x               | x            |
