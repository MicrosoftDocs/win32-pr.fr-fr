# <a name="dcl_hs_max_tessfactor-sm5---asm"></a>DCL \_ HS \_ Max \_ tessfactor (SM5-ASM)

Déclarez le maxTessFactor pour le correctif.



| DCL \_ HS \_ Max \_ tessfactor n |
|----------------------------|



 



| Élément                                                   | Description                          |
|--------------------------------------------------------|--------------------------------------|
| <span id="n"></span><span id="N"></span>*n*<br/> | \[dans \] le maxTessFactor.<br/> |



 

## <a name="remarks"></a>Notes

MaxTessFactor est une valeur float32 comprise dans la plage {1,0... 64,0}.

Cette instruction s’applique aux étapes suivantes du nuanceur :



| Sommet | Forme | Domain | Géométrie | Pixel | Compute |
|--------|------|--------|----------|-------|---------|
|        | X    |        |          |       |         |
