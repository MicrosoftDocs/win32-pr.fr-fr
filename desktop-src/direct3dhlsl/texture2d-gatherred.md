# <a name="texture2dgatherred-methods"></a>Texture2D :: GatherRed, méthodes

Retourne les composants rouges des quatre valeurs de Texel d’un [**Texture2D**](sm5-object-texture2d.md)qui seraient utilisées dans une opération de filtrage bilinéaire.

### <a name="overload-list"></a>Liste de surcharge



| Méthode                                                                                                  | Description                                                                                                                                     |
|:--------------------------------------------------------------------------------------------------------|:------------------------------------------------------------------------------------------------------------------------------------------------|
| [**GatherRed (S, float, int)**](sm5-object-texture2d-gatherred.md)                                         | Retourne les composants rouges des quatre valeurs de Texel qui seraient utilisées dans une opération de filtrage bilinéaire.<br/>                                 |
| [**GatherRed (S, float, int, uint)**](t2d-gatherred-s-float-int-uint-.md)                                   | Retourne les composants rouges des quatre valeurs de Texel qui seraient utilisées dans une opération de filtrage bilinéaire, ainsi que l’état du mappage de mosaïques.<br/> |
| [**GatherRed (S, float, Int2, Int2, Int2, Int2)**](t2d-gatherred-s-float-int2-int2-int2-int2-.md)             | Retourne les composants rouges des quatre valeurs de Texel qui seraient utilisées dans une opération de filtrage bilinéaire.<br/>                                 |
| [**GatherRed (S, float, Int2, Int2, Int2, Int2, uint)**](t2d-gathergreen-s-float-int2-int2-int2-int2-uint-.md) | Retourne les composants rouges des quatre valeurs de Texel qui seraient utilisées dans une opération de filtrage bilinéaire, ainsi que l’état du mappage de mosaïques.<br/> |
