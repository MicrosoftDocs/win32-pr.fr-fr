Déclarez l’ID d’instance dans une phase de jointure de nuanceur de coque.



| \_vJoinInstanceID d’entrée DCL |
|----------------------------|



 



| Élément                                                                                                                               | Description                        |
|------------------------------------------------------------------------------------------------------------------------------------|------------------------------------|
| <span id="vJoinInstanceID"></span><span id="vjoininstanceid"></span><span id="VJOININSTANCEID"></span>*vJoinInstanceID*<br/> | \[dans \] l’ID d’instance.<br/> |
