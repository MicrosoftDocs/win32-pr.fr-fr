# <a name="mov---ps"></a>MOV-PS

Déplacer des données entre des registres.

## <a name="syntax"></a>Syntaxe



| MOV DST, SRC |
|--------------|



 

where

-   l’heure d’été est le registre de destination.
-   SRC est un registre source.
