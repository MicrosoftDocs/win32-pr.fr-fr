# <a name="asint-function"></a>asint fonction)

Interprète le modèle binaire d’une valeur d’entrée comme un entier. Pour plus d’informations sur la fonction intrinsèque **asint** , consultez [**ASINT (DirectX HLSL)**](dx-graphics-hlsl-asint.md).

## <a name="syntax"></a>Syntaxe

``` syntax
int asint(
  in  value
);
```

## <a name="parameters"></a>Paramètres

<dl> <dt>

*valeur* \[ dans\]
</dt> <dd>

Valeur d'entrée.

</dd> </dl>

## <a name="return-value"></a>Valeur retournée

Type : **int**

Entrée interprétée comme un entier.
