# <a name="texture2darraygatheralpha-methods"></a>Texture2DArray :: GatherAlpha, méthodes

Retourne les composants alpha des quatre valeurs de Texel d’un [**Texture2DArray**](sm5-object-texture2darray.md)qui seraient utilisées dans une opération de filtrage bilinéaire.

### <a name="overload-list"></a>Liste de surcharge



| Méthode                                                                                                          | Description                                                                                                                                       |
|:----------------------------------------------------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------|
| [**GatherAlpha (S, float, int)**](sm5-object-texture2darray-gatheralpha.md)                                        | Retourne les composants alpha des quatre valeurs de Texel qui seraient utilisées dans une opération de filtrage bilinéaire.<br/>                                 |
| [**GatherAlpha (S, float, int, uint)**](t2darray-gatheralpha-s-float-int-uint-.md)                                  | Retourne les composants alpha des quatre valeurs de Texel qui seraient utilisées dans une opération de filtrage bilinéaire, ainsi que l’état du mappage de mosaïque.<br/> |
| [**GatherAlpha (S, float, Int2, Int2, Int2, Int2)**](t2darray-gatheralpha-s-float-int2-int2-int2-int2-.md)            | Retourne les composants alpha des quatre valeurs de Texel qui seraient utilisées dans une opération de filtrage bilinéaire.<br/>                                 |
| [**GatherAlpha (S, float, Int2, Int2, Int2, Int2, uint)**](t2darray-gatheralpha-s-float-int2-int2-int2-int2-uint-.md)  | Retourne les composants alpha des quatre valeurs de Texel qui seraient utilisées dans une opération de filtrage bilinéaire, ainsi que l’état du mappage de mosaïque.<br/> |
