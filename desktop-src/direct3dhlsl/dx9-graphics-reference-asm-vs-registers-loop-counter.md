# <a name="loop-counter-register-hlsl-vs-reference"></a>Registre de compteur de boucles (référence HLSL VS)

Le seul registre de cette banque est le Registre du compteur de boucles actuel (aL). Elle est incrémentée automatiquement à chaque exécution de la [boucle, par rapport à](loop---vs.md)... bloc [ENDLOOP-vs](endloop---vs.md) . Il peut donc être utilisé dans le bloc pour l’adressage relatif si nécessaire et n’est pas valide pour l’utiliser en dehors de la boucle.
