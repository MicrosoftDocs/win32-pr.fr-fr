# <a name="texture2dgather-methods"></a>Texture2D :: Gather, méthodes

Échantillonne un [**Texture2D**](sm5-object-texture2d.md) et retourne les quatre composants.

Pour plus d’informations sur la description de l’instruction DXBC sous-jacente, consultez la documentation sur [gather4](./gather4--sm5---asm-.md) .

### <a name="overload-list"></a>Liste de surcharge



| Méthode                                                           | Description                                                                                         |
|:-----------------------------------------------------------------|:----------------------------------------------------------------------------------------------------|
| [**Gather (S, float, int)**](sm5-object-texture2d-gather.md)       | Échantillonne une texture et retourne les quatre composants.<br/>                                       |
| [**Gather (S, float, int, uint)**](t2d-gather-s-float-int-uint-.md) | Échantillonne une texture et retourne les quatre composants, ainsi que l’état de l’opération.<br/> |
