| Méthode                                                                                              | Description                                                                                                                                                                           |
|:----------------------------------------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SampleCmp (S, float, float, int)**](dx-graphics-hlsl-to-samplecmp.md)                               | Échantillonne une texture en utilisant une valeur de comparaison pour rejeter des exemples.<br/>                                                                                                             |
| [**SampleCmp (S, float, float, int, float)**](t1darray-samplecmp-s-float-float-int-float-.md)           | Échantillonne une texture, en utilisant une valeur de comparaison pour rejeter les exemples, avec une valeur facultative pour fixer des exemples de valeurs de niveau de détail (LOD) à.<br/>                                     |
| [**SampleCmp (S, float, float, int, float, uint)**](t1darray-samplecmp-s-float-float-int-float-uint-.md) | Échantillonne une texture, en utilisant une valeur de comparaison pour rejeter les exemples, avec une valeur facultative pour fixer des exemples de valeurs de niveau de détail (LOD) à.
