Début d’un bloc Else.

## <a name="syntax"></a>Syntaxe



| else |
|------|



 

## <a name="remarks"></a>Remarques



| Versions de nuanceur vertex | 1\_1 | 2 \_ 0 | 2 \_ x | 2 \_ logiciels | 3 \_ 0 | 3 \_ logiciels |
|------------------------|------|------|------|-------|------|-------|
| else                   |      | x    | x    | x     | x    | x     |



 

Si la condition dans l’instruction [If bool-vs](if-bool---vs.md) correspondante a la valeur true, le code inclus dans l’instruction if bool-vs et le [endif](endif---vs.md) correspondant est exécuté. Sinon, le code délimité par Else... les instructions endif sont exécutées.
