# <a name="mad---ps"></a>Mad-PS

Multiplier et ajouter une instruction. Définit le registre de destination sur (src0 \* src1) + src2.

## <a name="syntax"></a>Syntaxe



| Mad DST, src0, src1, src2 |
|---------------------------|



 

where

-   l’heure d’été est le registre de destination.
-   src0 est un registre source.
-   src1 est un registre source.
-   src2 est un registre source.

## <a name="remarks"></a>Notes



| Versions de nuanceur de pixels | 1\_1 | 1\_2 | 1 \_ 3 | 1\_4 | 2 \_ 0 | 2 \_ x | 2 \_ logiciels | 3 \_ 0 | 3 \_ logiciels |
|-----------------------|------|------|------|------|------|------|-------|------|-------|
| Mad                   | x    | x    | x    | x    | x    | x    | x     | x    | x     |



 

L’extrait de code suivant montre les opérations effectuées.


```
dest.x = src0.x * src1.x + src2.x;
dest.y = src0.y * src1.y + src2.y;
dest.z = src0.z * src1.z + src2.z;
dest.w = src0.w * src1.w + src2.w;
```
