# <a name="texture1dsamplecmplevelzero-methods"></a>Texture1D :: SampleCmpLevelZero, méthodes

Échantillonne un [**Texture1D**](sm5-object-texture1d.md) sur le niveau de mipmap 0 uniquement et compare le résultat à une valeur de comparaison.

### <a name="overload-list"></a>Liste de surcharge



| Méthode                                                                                               | Description                                                                                                                            |
|:-----------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------|
| [**SampleCmpLevelZero (S, float, float, int)**](dx-graphics-hlsl-to-samplecmplevelzero.md)              | Échantillonne une texture sur le niveau de mipmap 0 uniquement et compare le résultat à une valeur de comparaison.<br/>                                     |
| [**SampleCmpLevelZero (S, float, float, int, uint)**](t1d-samplecmplevelzero-s-float-float-int-uint-.md) | Échantillonne une texture sur le niveau de mipmap 0 uniquement et compare le résultat à une valeur de comparaison. Retourne l’état de l’opération.<br/> |
