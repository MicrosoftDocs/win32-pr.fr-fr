# <a name="shader-model-1"></a>Modèle de nuanceur 1

Le modèle de nuanceur 1 était le premier modèle de nuanceur créé dans DirectX. Il a introduit les nuanceurs vertex et pixel dans la première implémentation du pipeline programmable.

Pour compiler ps_1_x nuanceurs comme des nuanceurs de ps_2_0, consultez <a href="/windows/desktop/direct3dtools/dx-graphics-tools-fxc-using">compilation du modèle de nuanceur 1</a>.</li></ul> | | Ensemble de registres | <ul><li><a href="dx9-graphics-reference-asm-vs-registers-vs-1-1.md">Registres de nuanceur vertex</a></li></ul> | | Vertex shader Max | 128 instructions | | Profils de nuanceur | vs_1_1 |
