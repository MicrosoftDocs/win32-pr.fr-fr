# <a name="id3dxprtbufferistexture-method"></a>ID3DXPRTBuffer :: IsTexture, méthode

Indique si la mémoire tampon contient une texture.

## <a name="syntax"></a>Syntaxe


```C++
BOOL IsTexture();
```



## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.

## <a name="return-value"></a>Valeur retournée

Type : **[ **bool**](../winprog/windows-data-types.md)**

Retourne la **valeur true** si la mémoire tampon contient une texture ; Sinon, retourne **false** .
