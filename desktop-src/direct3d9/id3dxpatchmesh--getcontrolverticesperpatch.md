# <a name="id3dxpatchmeshgetcontrolverticesperpatch-method"></a>ID3DXPatchMesh :: GetControlVerticesPerPatch, méthode

Obtient le nombre de vertex de contrôle par correctif.

## <a name="syntax"></a>Syntaxe


```C++
DWORD GetControlVerticesPerPatch();
```



## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.

## <a name="return-value"></a>Valeur retournée

Type : **[ **DWORD**](../winprog/windows-data-types.md)**

Nombre de vertex de contrôle par correctif.
