# <a name="matrix4x4"></a>Matrix4x4

Définit une matrice 4 x 4. Utilisé comme matrice de transformation de frame.

``` syntax
template Matrix4x4
{
    < F6F23F45-7686-11cf-8F52-0040333594A3 >
    array float matrix[16];
} 
```

Où :

-   matrice à virgule flottante \[ de tableau 16 \] -tableau de 16 valeurs float.
