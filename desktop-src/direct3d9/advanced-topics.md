# <a name="advanced-topics-direct3d-9"></a>Rubriques avancées (Direct3D 9)

-   [Définition de l’état de l’appareil avec le pipeline de fonction fixe et le pipeline de nuanceur programmable](setting-render-states-with-fixed-function-and-programmable-shader-pipeline.md)
-   [Requêtes](queries.md)
-   [Dessin efficace de plusieurs instances de Geometry](efficiently-drawing-multiple-instances-of-geometry.md)
-   [Utilisation de UVAtlas](using-uvatlas.md)
