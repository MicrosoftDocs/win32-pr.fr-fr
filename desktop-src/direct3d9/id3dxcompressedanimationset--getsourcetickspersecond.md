# <a name="id3dxcompressedanimationsetgetsourcetickspersecond-method"></a>ID3DXCompressedAnimationSet :: GetSourceTicksPerSecond, méthode

Obtient le nombre de graduations d’image clé d’animation qui se produisent par seconde.

## <a name="syntax"></a>Syntaxe


```C++
DOUBLE GetSourceTicksPerSecond();
```



## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.

## <a name="return-value"></a>Valeur retournée

Type : **[ **double**](../winprog/windows-data-types.md)**

Nombre de battements d’image clé d’animation qui se produisent par seconde.
