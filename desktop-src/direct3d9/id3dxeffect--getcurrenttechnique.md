# <a name="id3dxeffectgetcurrenttechnique-method"></a>ID3DXEffect :: GetCurrentTechnique, méthode

Obtient la technique actuelle.

## <a name="syntax"></a>Syntaxe


```C++
D3DXHANDLE GetCurrentTechnique();
```



## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.

## <a name="return-value"></a>Valeur retournée

Type : **[D3DXHANDLE](dx9-graphics-reference-effects-constants.md)**

Identificateur unique de la technique actuelle. Consultez [Handles (Direct3D 9)](handles.md).
