# <a name="d3dissue_begin"></a>Début de D3DISSUE \_

Cette macro crée une valeur utilisée par [**issue**](/windows/win32/api/d3d9helper/nf-d3d9helper-idirect3dquery9-issue) pour émettre une requête Begin.

``` syntax
#define D3DISSUE_BEGIN (1 << 1)
```

## <a name="remarks"></a>Remarques

D3DISSUE \_ Begin est valide pour le type de requête suivant.

-   \_Occlusion D3DQUERYTYPE
