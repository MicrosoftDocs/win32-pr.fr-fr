# <a name="dxfile-constants"></a>Constantes DXFILE

Les indicateurs suivants sont utilisés pour spécifier les canaux à utiliser dans une texture. Action déconseillée.

## <a name="dxfileformat"></a>DXFILEFORMAT



| \#définition                 | Valeur | Description      |
|--------------------------|-------|------------------|
| \_binaire DXFILEFORMAT     | 0     | Fichier binaire.     |
| \_texte DXFILEFORMAT       | 1     | Fichier texte.       |
| DXFILEFORMAT \_ compressé | 2     | Fichier compressé. |



 

Ces \# définitions sont déclarées dans dxfile. h.

## <a name="dxfileloadoptions"></a>DXFILELOADOPTIONS



| \#définition                 | Valeur | Description                  |
|--------------------------|-------|------------------------------|
| DXFILELOAD \_ FROMFILE     | 0x00L | Charge un fichier à partir d’un fichier.     |
| DXFILELOAD \_ FROMRESOURCE | 0x01L | Charger un fichier à partir d’une ressource. |
| DXFILELOAD \_ FROMMEMORY   | 0x02L | Chargez un fichier à partir de la mémoire.     |
| DXFILELOAD \_ FROMSTREAM   | 0x04L | Charge un fichier à partir d’un flux.   |
| DXFILELOAD \_ FROMURL      | 0x08L | Chargez un fichier à partir d’une URL.      |



 

Ces \# définitions sont déclarées dans dxfile. h.
