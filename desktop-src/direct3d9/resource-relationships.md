# <a name="resource-relationships-direct3d-9"></a>Relations entre les ressources (Direct3D 9)

Le diagramme suivant montre les opérations qui sont syntaxiquement possibles sur des ressources spécifiques et leur contenu. Pour plus d’informations sur l’utilisation des ressources, consultez [manipulation des ressources (Direct3D 9)](manipulating-resources.md).

![diagramme des relations entre les ressources](images/resourcerelationship.png)

Les flèches qui s’exécutent de gauche à droite indiquent que les types cibles sont créés par les méthodes données, tandis que les flèches qui s’exécutent de droite à gauche indiquent que ces types de ressources peuvent être passés comme arguments aux méthodes données.
