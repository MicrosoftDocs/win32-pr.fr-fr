# <a name="id3dxlinegetpattern-method"></a>ID3DXLine :: GetPattern, méthode

Obtient le modèle de stipple de la ligne.

## <a name="syntax"></a>Syntaxe


```C++
DWORD GetPattern();
```



## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.

## <a name="return-value"></a>Valeur de retour

Type : **[ **DWORD**](../winprog/windows-data-types.md)**

Retourne le modèle de stipple de ligne : 1 est opaque, 0 est transparent.
