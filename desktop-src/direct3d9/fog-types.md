# <a name="fog-types-direct3d-9"></a>Types de brouillard (Direct3D 9)

Il existe deux façons d’ajouter un brouillard à une scène : avec le brouillard de pixel et/ou le brouillard de vertex. Les rubriques suivantes illustrent les formules utilisées dans les équations de brouillard, ainsi que l’implémentation du brouillard de vertex et de pixel. Une application peut implémenter un brouillard avec un nuanceur de sommets et un brouillard de pixel simultanément, si vous le souhaitez.

-   [Formules de brouillard](fog-formulas.md)
-   [Paramètres de brouillard](fog-parameters.md)
-   [Fusion de brouillard](fog-blending.md)
-   [Couleur de brouillard](fog-color.md)
-   [Brouillard du vertex](vertex-fog.md)
-   [Brouillard de pixel](pixel-fog.md)

La fusion de brouillard est contrôlée par les États de rendu ; il ne fait pas partie du pipeline de pixels programmable.
