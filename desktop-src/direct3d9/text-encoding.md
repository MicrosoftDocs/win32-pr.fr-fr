# <a name="text-encoding"></a>Encodage de texte

Le format de fichier DirectX. x n’est pas spécifique à une application. Il utilise des modèles qui ne dépendent pas de la façon dont le fichier est utilisé. Cela permet au format de fichier. x d’être utilisé par n’importe quelle application cliente.
