# <a name="id3dxprtenginegetnumverts-method"></a>ID3DXPRTEngine :: GetNumVerts, méthode

Récupère le nombre de vertex de la maille, y compris les nouveaux vertex ajoutés à la suite d’un échantillonnage spatial adaptatif.

## <a name="syntax"></a>Syntaxe


```C++
UINT GetNumVerts();
```



## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.

## <a name="return-value"></a>Valeur renvoyée

Type : **[ **uint**](../winprog/windows-data-types.md)**

Nombre de vertex dans la maille, y compris les nouveaux vertex ajoutés à la suite d’un échantillonnage adaptatif.
