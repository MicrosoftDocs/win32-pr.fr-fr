# <a name="id3dxfontgetdc-method"></a>ID3DXFont :: GetDC, méthode

Retourne un handle vers un contexte de périphérique (DC) d’affichage dont la police est définie.

## <a name="syntax"></a>Syntaxe


```C++
HDC GetDC();
```



## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.

## <a name="return-value"></a>Valeur renvoyée

Type : **[ **HDC**](../winprog/windows-data-types.md)**

Handle vers un DC d’affichage.
