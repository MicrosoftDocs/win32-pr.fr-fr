# <a name="other-d3dx-constants"></a>Autres constantes D3DX

Les autres constantes D3DX sont les suivantes :

## <a name="16-bit-floating-point-numbers"></a>Nombres à virgule flottante 16 bits



|                         |                                         |
|-------------------------|-----------------------------------------|
| \#définition                | Description                             |
| D3DX \_ 16F- \_ creuser          | Nombre de chiffres décimaux de précision   |
| D3DX \_ 16F \_ Epsilon      | Plus petit, par exemple 1,0 + Epsilon ! = 1,0 |
| D3DX \_ 16F \_ mant \_ creuser    | Nombre de bits dans la mantisse              |
| D3DX \_ 16F \_ Max          | Valeur maximale                           |
| D3DX \_ 16F \_ Max \_ 10 \_ exp | Exposant décimal maximal                |
| 16F de D3DX- \_ \_ \_ exp Max exp     | Exposant binaire maximal                 |
| D3DX \_ 16F \_ min.          | Valeur positive minimale                  |
| D3DX \_ 16F \_ Min \_ 10 \_ exp | Exposant décimal minimal                |
| 16F de D3DX- \_ \_ \_ exp min     | Exposant binaire minimal                 |
| D3DX \_ 16F \_ base        | Radical d’exposant                          |
| D3DX \_ 16F \_ arrondi       | Arrondi d’addition : au plus proche                 |
| D3DX \_ 1BYPI             | 1/pi                                    |
| D3DX \_ pi                | pi                                      |
| D3DX \_ par défaut \_ float    | Valeur float maximale                     |



 

Ces \# définitions sont déclarées dans d3dx9. h et d3dx9math. h.

## <a name="prt-constants"></a>Constantes PRT



|                  |                                            |
|------------------|--------------------------------------------|
| \#définition         | Description                                |
| D3DXSH \_ MINORDER | Ordre le plus bas autorisé de la simulation.  |
| D3DXSH \_ MAXORDER | Ordre le plus élevé autorisé pour la simulation. |



 

Ces \# définitions sont déclarées dans d3dx9math. h. Pour plus d’informations sur PRT, consultez [transfert de luminance précalculé (Direct3D 9)](precomputed-radiance-transfer.md).

## <a name="texture-constants"></a>Constantes de texture



|                        |                                                                    |
|------------------------|--------------------------------------------------------------------|
| \#définition               | Description                                                        |
| D3DFMT \_ à partir d’un \_ fichier     | Adoptez le format exact d’un fichier.                               |
| D3DX \_ par défaut          | Valeur par défaut.                                                   |
| \_NONPOW2 par défaut de D3DX \_ | N’arrondissez pas les nombres, tels que la largeur ou la hauteur, à une puissance de deux. |
| D3DX \_ à partir d’un \_ fichier       | Effectuez exactement les dimensions de la texture à partir d’un fichier.                   |



 

Ces \# définitions sont déclarées dans d3dx9. h.

## <a name="other-d3dx-constants"></a>Autres constantes D3DX

Les \_ constantes de version UNUSED16 et D3DX définies dans D3dx9mesh. h et D3dx9core. h sont utilisées en interne. N’utilisez pas ces constantes.
