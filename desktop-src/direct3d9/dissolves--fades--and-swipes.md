# <a name="dissolves-fades-and-swipes-direct3d-9"></a><span data-ttu-id="61518-103">Dissoudre, fondu et balayage (Direct3D 9)</span><span class="sxs-lookup"><span data-stu-id="61518-103">Dissolves, Fades, and Swipes (Direct3D 9)</span></span>

<span data-ttu-id="61518-104">De plus en plus, les applications utilisent des effets spéciaux qui sont couramment utilisés dans les films et les vidéos, tels que les dissolutions, les balayages et les fondus.</span><span class="sxs-lookup"><span data-stu-id="61518-104">Increasingly, applications employ special effects that are commonly used in movies and video, such as dissolves, swipes, and fades.</span></span>

<span data-ttu-id="61518-105">Dans une solution, une image est progressivement remplacée par une autre dans une séquence lisse de frames.</span><span class="sxs-lookup"><span data-stu-id="61518-105">In a dissolve, one image is gradually replaced by another in a smooth sequence of frames.</span></span> <span data-ttu-id="61518-106">Bien que Direct3D fournisse des méthodes d’utilisation de plusieurs textures de fusion pour obtenir le même effet, les applications qui utilisent la mémoire tampon de stencil pour les conversions peuvent utiliser des fonctionnalités de fusion de texture pour d’autres effets pendant qu’ils effectuent une dissolution.</span><span class="sxs-lookup"><span data-stu-id="61518-106">Although Direct3D provides methods of using multiple texture blending to achieve the same effect, applications that use the stencil buffer for dissolves can use texture-blending capabilities for other effects while they do a dissolve.</span></span>

<span data-ttu-id="61518-107">Lorsque votre application effectue une dissolution, elle doit restituer deux images différentes.</span><span class="sxs-lookup"><span data-stu-id="61518-107">When your application performs a dissolve, it must render two different images.</span></span> <span data-ttu-id="61518-108">Elle utilise la mémoire tampon de stencil pour contrôler les pixels de chaque image qui sont dessinés sur la surface cible de rendu.</span><span class="sxs-lookup"><span data-stu-id="61518-108">It uses the stencil buffer to control which pixels from each image are drawn to the rendering target surface.</span></span> <span data-ttu-id="61518-109">Vous pouvez définir une série de masques de stencil et les copier dans la mémoire tampon de stencil sur des frames successifs.</span><span class="sxs-lookup"><span data-stu-id="61518-109">You can define a series of stencil masks and copy them into the stencil buffer on successive frames.</span></span> <span data-ttu-id="61518-110">Vous pouvez également définir un masque de gabarit de base pour le premier frame et le modifier de façon incrémentielle.</span><span class="sxs-lookup"><span data-stu-id="61518-110">Alternately, you can define a base stencil mask for the first frame and alter it incrementally.</span></span>

<span data-ttu-id="61518-111">Au début de la dissolution, votre application définit la fonction de stencil et le masque de stencil afin que la plupart des pixels de l’image de départ passent le test de stencil.</span><span class="sxs-lookup"><span data-stu-id="61518-111">At the beginning of the dissolve, your application sets the stencil function and stencil mask so that most of the pixels from the starting image pass the stencil test.</span></span> <span data-ttu-id="61518-112">La plupart des pixels de l’image finale doivent faire échouer le test du stencil.</span><span class="sxs-lookup"><span data-stu-id="61518-112">Most of the pixels from the ending image should fail the stencil test.</span></span> <span data-ttu-id="61518-113">Sur des frames successifs, le masque du gabarit est mis à jour afin que le nombre de pixels de l’image de départ soit inférieur ou inférieur au test.</span><span class="sxs-lookup"><span data-stu-id="61518-113">On successive frames, the stencil mask is updated so that fewer and fewer of the pixels in the starting image pass the test.</span></span> <span data-ttu-id="61518-114">Au fur et à mesure que les images progressent, le test n’est pas limité à moins de pixels dans l’image finale.</span><span class="sxs-lookup"><span data-stu-id="61518-114">As the frames progress, fewer and fewer of the pixels in the ending image fail the test.</span></span> <span data-ttu-id="61518-115">De cette manière, votre application peut effectuer une dissolution à l’aide d’un modèle de dissolution arbitraire.</span><span class="sxs-lookup"><span data-stu-id="61518-115">In this manner, your application can perform a dissolve using any arbitrary dissolve pattern.</span></span>

<span data-ttu-id="61518-116">Le fondu ou la dégradation est un cas particulier de résolution.</span><span class="sxs-lookup"><span data-stu-id="61518-116">Fading in or fading out is a special case of dissolving.</span></span> <span data-ttu-id="61518-117">En cas de fondu, la mémoire tampon du stencil est utilisée pour la résolution d’une image noire ou blanche en rendu d’une scène 3D.</span><span class="sxs-lookup"><span data-stu-id="61518-117">When fading in, the stencil buffer is used to dissolve from a black or white image to a rendering of a 3D scene.</span></span> <span data-ttu-id="61518-118">Le fondu est l’inverse, votre application commence par un rendu d’une scène 3D et est convertie en noir ou blanc.</span><span class="sxs-lookup"><span data-stu-id="61518-118">Fading out is the opposite, your application starts with a rendering of a 3D scene and dissolves to black or white.</span></span> <span data-ttu-id="61518-119">Le fondu peut être effectué à l’aide de n’importe quel modèle arbitraire que vous souhaitez utiliser.</span><span class="sxs-lookup"><span data-stu-id="61518-119">The fade can be done using any arbitrary pattern you want to employ.</span></span>

<span data-ttu-id="61518-120">Les applications Direct3D utilisent une technique similaire pour les balayages.</span><span class="sxs-lookup"><span data-stu-id="61518-120">Direct3D applications use a similar technique for swipes.</span></span> <span data-ttu-id="61518-121">Par exemple, lorsqu’une application effectue un balayage de gauche à droite, l’image de fin apparaît progressivement sur l’image de départ, de gauche à droite.</span><span class="sxs-lookup"><span data-stu-id="61518-121">For example, when an application performs a left-to-right swipe, the ending image appears to slide gradually on top of the starting image from left to right.</span></span> <span data-ttu-id="61518-122">Comme dans une solution, vous devez définir une série de masques de stencil chargés dans la mémoire tampon de stencil sur des frames successifs, ou modifier successivement le masque de début du stencil.</span><span class="sxs-lookup"><span data-stu-id="61518-122">As in a dissolve, you must define a series of stencil masks that are loaded into the stencil buffer on successive frames, or successively modify the starting stencil mask.</span></span> <span data-ttu-id="61518-123">Les masques de stencil sont utilisés pour désactiver l’écriture des pixels à partir de l’image de départ et pour activer l’écriture des pixels à partir de l’image finale.</span><span class="sxs-lookup"><span data-stu-id="61518-123">The stencil masks are used to disable the writing of pixels from the starting image and to enable the writing of pixels from the ending image.</span></span>

<span data-ttu-id="61518-124">Un balayage est un peu plus complexe qu’un fondu dans le sens où votre application doit lire les pixels de l’image finale dans l’ordre inverse du balayage.</span><span class="sxs-lookup"><span data-stu-id="61518-124">A swipe is somewhat more complex than a dissolve in that your application must read pixels from the ending image in the reverse order of the swipe.</span></span> <span data-ttu-id="61518-125">Autrement dit, si le balayage se déplace de gauche à droite, votre application doit lire les pixels de l’image finale de droite à gauche.</span><span class="sxs-lookup"><span data-stu-id="61518-125">That is, if the swipe is moving from left to right, your application must read pixels from the ending image from right to left.</span></span>
