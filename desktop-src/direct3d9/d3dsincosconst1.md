# <a name="d3dsincosconst1"></a>D3DSINCOSCONST1

Crée la première constante (dans le registre src2) pour l’instruction de nuanceur de pixels [SinCos,-PS](../direct3dhlsl/sincos---ps.md) .

``` syntax
#define D3DSINCOSCONST1  -1.5500992e-006f, -2.1701389e-005f,  0.0026041667f, 0.00026041668f
```

## <a name="return-value"></a>Valeur renvoyée

Retourne la première constante requise par [SinCos,-PS](../direct3dhlsl/sincos---ps.md).
