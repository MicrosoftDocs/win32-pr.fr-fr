# <a name="id3dxanimationsetgetnumanimations-method"></a>ID3DXAnimationSet :: GetNumAnimations, méthode

Obtient le nombre d’animations dans l’ensemble d’animations.

## <a name="syntax"></a>Syntaxe


```C++
UINT GetNumAnimations();
```



## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.

## <a name="return-value"></a>Valeur retournée

Type : **[ **uint**](../winprog/windows-data-types.md)**

Nombre d’animations dans l’ensemble d’animations.
