# <a name="d3dx_channel"></a>\_Chaîne D3DX

Les indicateurs suivants sont utilisés pour spécifier les canaux à utiliser dans une texture.



| \#définition                 | Description                                                                   |
|--------------------------|-------------------------------------------------------------------------------|
| D3DX \_ canal \_ rouge       | Indique que le canal rouge doit être utilisé.                                     |
| D3DX \_ canal \_ bleu      | Indique que le canal bleu doit être utilisé.                                    |
| D3DX \_ canal \_ vert     | Indique que le canal vert doit être utilisé.                                   |
| D3DX \_ canal \_ alpha     | Indique que le canal alpha doit être utilisé.                                   |
| \_Luminance de canal D3DX \_ | Indique les luminances des canaux rouge, vert et bleu à utiliser. |
