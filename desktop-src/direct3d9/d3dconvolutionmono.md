# <a name="d3dconvolutionmono"></a>D3DCONVOLUTIONMONO

Spécifie les dimensions maximales des filtres de convolution de texture.



| \#définition                      | Valeur | Description                                                |
|-------------------------------|-------|------------------------------------------------------------|
| D3DCONVOLUTIONMONO \_ MAXHEIGHT | 7     | Hauteur maximale d’un filtre de convolution de texture monochrome. |
| D3DCONVOLUTIONMONO \_ MAXWIDTH  | 7     | Largeur maximale d’un filtre de convolution de texture monochrome.  |
