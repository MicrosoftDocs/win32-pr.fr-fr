# <a name="effectparamdword"></a>EffectParamDWord

Définit un DWORD d’effet. Ce modèle a remplacé le modèle [**EffectDWord**](effectdword.md) .

``` syntax
template EffectParamDWord
{
    < E13963BC-AE51-4c5d-B00F-CFA3A9D97CE5 >
    STRING ParamName;
    DWORD Value;
} 
```

Où :

-   ParamName-nom du paramètre.
-   Valeur de paramètre de valeur.
