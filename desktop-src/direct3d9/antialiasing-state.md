# <a name="antialiasing-state-direct3d-9"></a>État de l’anticrénelage (Direct3D 9)

L' [anticrénelage (Direct3D 9)](antialiasing.md) est une méthode permettant de lisser les lignes et les bords sur l’écran. Par défaut, Direct3D n’effectue pas l’anticrénelage. Pour activer l’anticrénelage de scène complète, affectez à l' \_ État de rendu D3DRS MULTISAMPLEANTIALIAS la **valeur true**. Pour le désactiver, affectez la valeur \_ **false** à D3DRS MULTISAMPLEANTIALIAS.
