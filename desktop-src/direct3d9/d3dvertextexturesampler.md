# <a name="d3dvertextexturesampler"></a>D3DVERTEXTEXTURESAMPLER

Constantes d’échantillonnage de texture de vertex.

Ces constantes identifient les échantillonneurs de texture utilisés par les nuanceurs de vertex.



| \#définition                 | Description        |
|--------------------------|--------------------|
| D3DVERTEXTEXTURESAMPLER0 | (D3DDMAPSAMPLER + 1) |
| D3DVERTEXTEXTURESAMPLER1 | (D3DDMAPSAMPLER + 2) |
| D3DVERTEXTEXTURESAMPLER2 | (D3DDMAPSAMPLER + 3) |
| D3DVERTEXTEXTURESAMPLER3 | (D3DDMAPSAMPLER + 4) |



 

## <a name="d3ddmapsampler"></a>D3DDMAPSAMPLER

Il s’agit d’une constante interne utilisée par l’échantillonneur de texture dans le du paveur.



| \#définition       | Description                                           |
|----------------|-------------------------------------------------------|
| D3DDMAPSAMPLER | 256, qui est le nombre maximal d’échantillonneurs de texture. |
