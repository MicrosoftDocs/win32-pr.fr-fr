# <a name="texturefilename"></a>TextureFilename

Vous permet de spécifier le nom de fichier d’une texture à appliquer à un maillage ou à un visage. Ce modèle doit apparaître dans un objet Material.

``` syntax
template TextureFilename 
{ 
    < A42790E1-7810-11cf-8F52-0040333594A3 >

}
```
