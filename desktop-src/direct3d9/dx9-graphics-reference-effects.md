# <a name="effect-reference-direct3d-9-graphics"></a>Référence des effets (graphiques Direct3D 9)

Les effets sont une abstraction de niveau supérieur qui tire parti des nuanceurs de sommets et de pixels pour encapsuler les effets de rendu. Les effets peuvent être écrits à l’aide du langage de haut niveau ou du langage assembleur Shader.

Voici les rubriques de l’API.

-   [Interfaces d’effet](dx9-graphics-reference-effects-interfaces.md)
-   [Fonctions Effect](dx9-graphics-reference-effects-functions.md)
-   [Structures d’effet](dx9-graphics-reference-effects-structures.md)
-   [Constantes d’effet](dx9-graphics-reference-effects-constants.md)
-   [Format d’effet (Direct3D 9)](dx9-graphics-reference-effects-file-format.md)
-   [Informations de référence sur les annotations et sémantiques DirectX standard](dx9-graphics-reference-effects-dxsas.md)

En guise d’alternative à l’utilisation de l’API pour créer des effets, vous pouvez également utiliser un outil de compilation de ligne de commande. Consultez [Effect-Tool compiler Tool](../direct3dtools/fxc.md).
