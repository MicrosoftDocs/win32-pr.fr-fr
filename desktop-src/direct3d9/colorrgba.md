# <a name="colorrgba"></a>ColorRGBA

Définit un objet Color avec un composant alpha. Cela est utilisé pour la couleur de face dans la définition de modèle de matériau.

``` syntax
template ColorRGBA
{
    < 35FF44E0-6C7C-11cf-8F52-0040333594A3 >
    float red;
    float green;
    float blue;
    float alpha;
} 
```
