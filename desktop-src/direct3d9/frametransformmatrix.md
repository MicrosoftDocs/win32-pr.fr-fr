# <a name="frametransformmatrix"></a>FrameTransformMatrix

Définit une transformation locale pour un frame (et tous ses objets enfants).

``` syntax
template FrameTransformMatrix
{
    < F6F23F41-7686-11cf-8F52-0040333594A3 >
    Matrix4x4 frameMatrix;
} 
```

Où :

-   frameMatrix : modèle Matrix4x4. Consultez [**Matrix4x4**](matrix4x4.md).
