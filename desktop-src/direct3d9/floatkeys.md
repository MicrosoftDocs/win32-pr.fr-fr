# <a name="floatkeys"></a>FloatKeys

Définit un tableau de nombres à virgule flottante (Floats) et le nombre de valeurs float dans ce tableau. Utilisé pour définir des jeux de clés d’animation.

``` syntax
template FloatKeys
{
    < 10DD46A9-775B-11cf-8F52-0040333594A3 >
    DWORD nValues;
    array float values[nValues];
} 
```

-   Nvaleurs : nombre de valeurs à virgule flottante.
-   valeurs \[ nvaleurs \] -tableau de valeurs float.
