# <a name="shape-drawing-functions"></a>Fonctions de dessin de forme

D3DX fournit les fonctions de dessin de forme suivantes.

-   [**D3DXCreateBox**](d3dxcreatebox.md)
-   [**D3DXCreateCylinder**](d3dxcreatecylinder.md)
-   [**D3DXCreatePolygon**](d3dxcreatepolygon.md)
-   [**D3DXCreateSphere**](d3dxcreatesphere.md)
-   [**D3DXCreateTeapot**](d3dxcreateteapot.md)
-   [**D3DXCreateText**](d3dxcreatetext.md)
-   [**D3DXCreateTorus**](d3dxcreatetorus.md)

Toutes les formes sont retournées en tant qu’objets de maillage.

Ces fonctions peuvent être utilisées dans un système de coordonnées gauche ou droite.
