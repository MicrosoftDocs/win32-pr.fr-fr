# <a name="d3dcomposerects_maxnumrects"></a>D3DCOMPOSERECTS \_ MAXNUMRECTS

Spécifie le nombre maximal de rectangles utilisés pour composer des glyphes dans [**ComposeRects**](/windows/desktop/api/d3d9/nf-d3d9-idirect3ddevice9ex-composerects).



| \#définition                     | Description                                                                                                                          |
|------------------------------|--------------------------------------------------------------------------------------------------------------------------------------|
| D3DCOMPOSERECTS \_ MAXNUMRECTS | Nombre maximal de glyphes de rectangle à composer dans une chaîne de texte. Consultez [**ComposeRects**](/windows/desktop/api/d3d9/nf-d3d9-idirect3ddevice9ex-composerects). |
