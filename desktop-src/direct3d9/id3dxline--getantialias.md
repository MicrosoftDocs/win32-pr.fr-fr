# <a name="id3dxlinegetantialias-method"></a>ID3DXLine :: GetAntialias, méthode

Obtient l’état de l’anticrénelage de ligne.

## <a name="syntax"></a>Syntaxe


```C++
BOOL GetAntialias();
```



## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.

## <a name="return-value"></a>Valeur retournée

Type : **[ **bool**](../winprog/windows-data-types.md)**

Retourne la valeur du commutateur d’anticrénelage. **True** signifie que l’anticrénelage est activé, et **false** signifie que l’anticrénelage est désactivé.
