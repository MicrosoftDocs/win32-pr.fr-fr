# <a name="id3dxconstanttablegetbufferpointer-method"></a>ID3DXConstantTable :: GetBufferPointer, méthode

Obtient un pointeur vers la mémoire tampon qui contient la table constante.

## <a name="syntax"></a>Syntaxe


```C++
LPVOID GetBufferPointer();
```



## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.

## <a name="return-value"></a>Valeur de retour

Type : **[ **LPVOID**](../winprog/windows-data-types.md)**

Retourne un pointeur vers la mémoire tampon.
