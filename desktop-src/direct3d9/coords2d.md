Définit un vecteur à deux dimensions utilisé pour définir les coordonnées de texture d’un maillage (u, v).

``` syntax
template Coords2d
{
    < F6F23F44-7686-11cf-8F52-0040333594A3 >
    float u;
    float v;
} 
```

-   valeur de coordonnée u-u.
-   valeur de la coordonnée v-v.
