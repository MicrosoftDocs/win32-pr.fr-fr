# <a name="effectfloats"></a>EffectFloats

Définit un effet flottant hérité. Ce modèle a été remplacé par le modèle [**EffectParamFloats**](effectparamfloats.md) .

``` syntax
template EffectFloats
{
    < F1CFE2B3-0DE3-4e28-AFA1-155A750A282D >
    DWORD nFloats;
    array float Floats[nFloats];
} 
```

Où :

-   nFloats : nombre de valeurs à virgule flottante.
-   FLOTTE \[ nFloats \] -tableau de valeurs float.
