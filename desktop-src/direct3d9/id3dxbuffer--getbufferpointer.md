# <a name="id3dxbuffergetbufferpointer-method"></a>ID3DXBuffer :: GetBufferPointer, méthode

Récupère un pointeur vers les données dans la mémoire tampon.

## <a name="syntax"></a>Syntaxe


```C++
LPVOID GetBufferPointer();
```



## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.

## <a name="return-value"></a>Valeur retournée

Type : **[ **LPVOID**](../winprog/windows-data-types.md)**

Retourne un pointeur vers les données de la mémoire tampon.
