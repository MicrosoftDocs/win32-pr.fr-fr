# <a name="id3dxkeyframedanimationsetgetplaybacktype-method"></a>ID3DXKeyframedAnimationSet :: GetPlaybackType, méthode

Obtient le type de la boucle de lecture du jeu d’animations.

## <a name="syntax"></a>Syntaxe


```C++
D3DXPLAYBACK_TYPE GetPlaybackType();
```



## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.

## <a name="return-value"></a>Valeur retournée

Type : **[ **D3DXPLAYBACK \_**](./d3dxplayback-type.md)**

Type de la boucle de lecture du jeu d’animations. Consultez [**\_ type D3DXPLAYBACK**](./d3dxplayback-type.md).
