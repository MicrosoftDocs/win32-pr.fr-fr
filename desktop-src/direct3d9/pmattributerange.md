# <a name="pmattributerange"></a>PMAttributeRange

Non pris en charge. Utilisé en interne par DirectX pour représenter et enregistrer un maillage progressif. Ce modèle doit être utilisé uniquement avec les fonctions D3DX pour lire et écrire des fichiers. x.

``` syntax
template PMAttributeRange
{
    < 917E0427-C61E-4a14-9C64-AFE65F9E9844 >
    DWORD iFaceOffset;
    DWORD nFacesMin;
    DWORD nFacesMax;
    DWORD iVertexOffset;
    DWORD nVerticesMin;
    DWORD nVerticesMax;
} 
```
