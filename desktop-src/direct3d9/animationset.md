# <a name="animationset"></a>AnimationSet

Contient un ou plusieurs objets d' [**animation**](animation.md) . Chaque animation dans un ensemble d’animations a le même temps à un point donné. L’augmentation du temps de l’ensemble d’animations augmente le temps pour toutes les animations qu’il contient.

``` syntax
template AnimationSet
{
    < 3D82AB50-62DA-11cf-AB39-0020AF71E433 >
    [ Animation < 3D82AB4F-62DA-11cf-AB39-0020AF71E433 > ]
} 
```

Où :

-   \[ Animation < modèle d' \] animation 3D82AB4F-62DA-11cf-AB39-0020AF71E433 >-option.
