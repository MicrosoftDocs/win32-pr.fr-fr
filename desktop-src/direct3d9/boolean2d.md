# <a name="boolean2d"></a>Boolean2d

Définit un ensemble de deux valeurs booléennes utilisées dans le modèle [**MeshFaceWraps**](meshfacewraps.md) pour définir la topologie de texture d’une face individuelle.

``` syntax
template Boolean2d
{
    < 4885AE63-78E8-11cf-8F52-0040333594A3 >
    Boolean u;
    Boolean v;
} 
```

Où :

-   valeur u-Boolean. Consultez [**Boolean**](boolean.md).
-   v-valeur booléenne. Consultez [**Boolean**](boolean.md).
