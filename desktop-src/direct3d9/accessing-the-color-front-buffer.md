# <a name="accessing-the-color-front-buffer-direct3d-9"></a>Accès au tampon d’avant-plan de couleur (Direct3D 9)

L’accès à la mémoire tampon d’avant est autorisé par le biais de la méthode [**IDirect3DDevice9 :: GetFrontBufferData**](/windows/win32/api/d3d9helper/nf-d3d9helper-idirect3ddevice9-getfrontbufferdata) . Cette méthode est la seule façon d’obtenir une capture d’écran d’une scène avec anticrénelage.
