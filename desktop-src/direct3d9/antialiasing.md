# <a name="antialiasing-direct3d-9"></a>Anticrénelage (Direct3D 9)

L’anticrénelage est une technique que vous pouvez utiliser pour réduire l’apparence des pixels de l’étape d’escalier quand vous dessinez une ligne qui n’est pas exactement horizontale ou verticale. Dans les scènes en trois dimensions, cet artefact est plus perceptible sur les limites entre les polygones de couleurs différentes. L’anticrénelage fusionne efficacement les pixels à ces limites pour donner une apparence plus naturelle à la scène.
