# <a name="id3dxprtbufferunlockbuffer-method"></a>ID3DXPRTBuffer :: UnlockBuffer, méthode

Termine la durée de vie du pointeur ppData retourné par [**ID3DXPRTBuffer :: LockBuffer**](id3dxprtbuffer--lockbuffer.md).

## <a name="syntax"></a>Syntaxe


```C++
HRESULT UnlockBuffer();
```



## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.

## <a name="return-value"></a>Valeur retournée

Type : **[ **HRESULT**](https://msdn.microsoft.com/library/Bb401631(v=MSDN.10).aspx)**

Si la méthode est réussie, la valeur de retour est D3D \_ OK.
