# <a name="animation-direct3d-9-graphics"></a>Animation (graphiques Direct3D 9)

Contient des animations qui référencent un frame précédent. Il doit contenir une référence à un frame et au moins un ensemble de modèles [**AnimationKey**](animationkey.md) . Il peut également contenir un objet de données [**AnimationOptions**](animationoptions.md) .

``` syntax
template Animation
{
    < 3D82AB4F-62DA-11cf-AB39-0020AF71E433 >
    [...]
} 
```

Où :

-   \[ ... \] -N’importe quel modèle de fichier. x peut être utilisé ici. L’architecture est ainsi extensible.
