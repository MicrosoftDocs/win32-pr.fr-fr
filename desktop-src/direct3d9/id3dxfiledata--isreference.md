# <a name="id3dxfiledataisreference-method"></a>ID3DXFileData :: IsReference, méthode

Indique si cet objet de données de fichier est un objet de référence qui pointe vers un autre objet de données enfant.

## <a name="syntax"></a>Syntaxe


```C++
BOOL IsReference();
```



## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.

## <a name="return-value"></a>Valeur retournée

Type : **[ **bool**](../winprog/windows-data-types.md)**

Retourne la **valeur true** si l’objet de données de fichier est un objet de référence ; Sinon, retourne **false** .
