    // M is a D3DMATRIX being set to translate texture

## <a name="creating-texture-coordinates-as-a-linear-function-of-a-models-camera-space-position"></a><span data-ttu-id="ad415-113">Création de coordonnées de texture comme fonction linéaire de la position de l’espace de caméra d’un modèle</span><span class="sxs-lookup"><span data-stu-id="ad415-113">Creating texture coordinates as a linear function of a model's camera-space position</span></span>

-   <span data-ttu-id="ad415-114">Utilisez l' \_ indicateur D3DTSS TCI \_ CAMERASPACEPOSITION pour indiquer au système de passer la position du vertex, dans l’espace de l’appareil photo, en tant qu’entrée d’une transformation de texture.</span><span class="sxs-lookup"><span data-stu-id="ad415-114">Use the D3DTSS\_TCI\_CAMERASPACEPOSITION flag to instruct the system to pass the vertex position, in camera space, as input to a texture transformation.</span></span>
    ```
    // The input vertices have no texture coordinates, saving 
    // bandwidth. Three texture coordinates are generated by 
    // using vertex position in camera space (x, y, z).
    SetTextureStageState(0, D3DTSS_TEXCOORDINDEX, D3DTSS_TCI_CAMERASPACEPOSITION);
    ```

    

-   <span data-ttu-id="ad415-115">Demandez au rastériseur d’attendre des coordonnées de texture 2D.</span><span class="sxs-lookup"><span data-stu-id="ad415-115">Instruct the rasterizer to expect 2D texture coordinates.</span></span>
    ```
    // Two output coordinates are used.
    SetTextureStageState(0, D3DTSS_TEXTURETRANSFORMFLAGS, D3DTTFF_COUNT2);
    ```

    

-   <span data-ttu-id="ad415-116">Définissez et définissez une matrice qui applique une fonction linéaire.</span><span class="sxs-lookup"><span data-stu-id="ad415-116">Define and set a matrix that applies a linear function.</span></span>
    ```
    // Generate texture coordinates as linear functions 
    // so that:
    //      u = Ux*x + Uy*y + Uz*z + Uw 
    //      v = Vx*x + Vy*y + Vz*z + Vw
    // The matrix M for this case is:
    //      Ux  Vx  0  0 
    //      Uy  Vy  0  0 
    //      Uz  Vz  0  0 
    //      Uw  Vw  0  0 

    SetTransform(D3DTS_TEXTURE0, &M);
    ```
