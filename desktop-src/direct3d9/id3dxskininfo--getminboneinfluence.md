# <a name="id3dxskininfogetminboneinfluence-method"></a>ID3DXSkinInfo :: GetMinBoneInfluence, méthode

Obtient l’influence osseuse minimale. Les valeurs d’influence inférieures à cette valeur sont ignorées.

## <a name="syntax"></a>Syntaxe


```C++
FLOAT GetMinBoneInfluence();
```



## <a name="parameters"></a>Paramètres

Cette méthode n’a aucun paramètre.

## <a name="return-value"></a>Valeur retournée

Type : **[ **float**](../winprog/windows-data-types.md)**

Retourne la valeur d’influence osseuse minimale.
