# <a name="texture-wrapping-state-direct3d-9"></a>État de l’habillage de texture (Direct3D 9)

Les \_ États de rendu D3DRS WRAP0 à D3DRS \_ WRAP7 activent et désactivent l’encapsulation u-et v pour différentes textures dans la cascade de multitextures de l’appareil. Vous pouvez définir ces États de rendu sur une combinaison des \_ indicateurs D3DWRAPCOORD 0, D3DWRAPCOORD \_ 1, D3DWRAPCOORD \_ 2 et D3DWRAPCOORD \_ 3 pour activer le retour à la position en premier, deuxième, troisième et quatrième directions de la texture. Utilisez la valeur zéro pour désactiver complètement l’encapsulage. L’habillage de texture est désactivé dans toutes les directions pour toutes les étapes de texture par défaut.

Pour obtenir une vue d’ensemble conceptuelle, consultez [habillage de texture (Direct3D 9)](texture-wrapping.md).
