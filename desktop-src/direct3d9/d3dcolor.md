# <a name="d3dcolor"></a>D3DCOLOR

Définit le type de couleur Direct3D fondamental.


```C++
typedef DWORD D3DCOLOR;
```



## <a name="remarks"></a>Notes

Valeur de quatre octets qui représente généralement les composants alpha, rouge, vert et bleu d’une couleur. Il peut également représenter la luminance et la luminosité.

Vous pouvez définir le type **D3DCOLOR** à l’aide de l’une des macros suivantes.

-   [**D3DCOLOR \_ ARGB**](d3dcolor-argb.md)
-   [**D3DCOLOR \_ AYUV**](d3dcolor-ayuv.md)
-   [**D3DCOLOR \_ COLORVALUE**](d3dcolor-colorvalue.md)
-   [**D3DCOLOR \_ RVBA**](d3dcolor-rgba.md)
-   [**D3DCOLOR \_ XRGB**](d3dcolor-xrgb.md)
-   [**D3DCOLOR \_ XYUV**](d3dcolor-xyuv.md)
