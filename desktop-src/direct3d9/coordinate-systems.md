# <a name="coordinate-systems-direct3d-9"></a><span data-ttu-id="e1ec2-103">Systèmes de coordonnées (Direct3D 9)</span><span class="sxs-lookup"><span data-stu-id="e1ec2-103">Coordinate Systems (Direct3D 9)</span></span>

<span data-ttu-id="e1ec2-104">En général, les applications graphiques 3D utilisent deux types de systèmes de coordonnées cartésiennes : droitier et droitier.</span><span class="sxs-lookup"><span data-stu-id="e1ec2-104">Typically 3D graphics applications use two types of Cartesian coordinate systems: left-handed and right-handed.</span></span> <span data-ttu-id="e1ec2-105">Dans les deux systèmes de coordonnées, l’axe x positif pointe vers la droite et l’axe y positif pointe vers le haut.</span><span class="sxs-lookup"><span data-stu-id="e1ec2-105">In both coordinate systems, the positive x-axis points to the right, and the positive y-axis points up.</span></span> <span data-ttu-id="e1ec2-106">Vous pouvez vous souvenir de la direction vers laquelle pointe l’axe z positif en pointant les doigts de votre gauche ou de droite dans la direction x positive et en les faisant passer à l’axe y positif.</span><span class="sxs-lookup"><span data-stu-id="e1ec2-106">You can remember which direction the positive z-axis points by pointing the fingers of either your left or right hand in the positive x-direction and curling them into the positive y-direction.</span></span> <span data-ttu-id="e1ec2-107">La direction vers laquelle votre Thumb pointe, que ce soit vers vous ou en dehors, est la direction dans laquelle les points positifs de l’axe z pour ce système de coordonnées.</span><span class="sxs-lookup"><span data-stu-id="e1ec2-107">The direction your thumb points, either toward or away from you, is the direction that the positive z-axis points for that coordinate system.</span></span> <span data-ttu-id="e1ec2-108">L’illustration suivante montre ces deux systèmes de coordonnées.</span><span class="sxs-lookup"><span data-stu-id="e1ec2-108">The following illustration shows these two coordinate systems.</span></span>

![illustration des systèmes de coordonnées cartésiens droitiers et droitiers de droite](images/leftrght.png)

<span data-ttu-id="e1ec2-110">Direct3D utilise un système de coordonnées droitier.</span><span class="sxs-lookup"><span data-stu-id="e1ec2-110">Direct3D uses a left-handed coordinate system.</span></span> <span data-ttu-id="e1ec2-111">Si vous portez une application basée sur un système de coordonnées droitier, vous devez apporter deux modifications aux données transmises à Direct3D.</span><span class="sxs-lookup"><span data-stu-id="e1ec2-111">If you are porting an application that is based on a right-handed coordinate system, you must make two changes to the data passed to Direct3D.</span></span>

-   <span data-ttu-id="e1ec2-112">Retournez l’ordre des sommets triangles afin que le système les traverse à partir du début.</span><span class="sxs-lookup"><span data-stu-id="e1ec2-112">Flip the order of triangle vertices so that the system traverses them clockwise from the front.</span></span> <span data-ttu-id="e1ec2-113">En d’autres termes, si les vertex sont v0, v1, v2, transmettez-les à Direct3D en tant que v0, v2, v1.</span><span class="sxs-lookup"><span data-stu-id="e1ec2-113">In other words, if the vertices are v0, v1, v2, pass them to Direct3D as v0, v2, v1.</span></span>
-   <span data-ttu-id="e1ec2-114">Utilisez la matrice de vue pour mettre à l’échelle l’espace universel de-1 dans l’axe z.</span><span class="sxs-lookup"><span data-stu-id="e1ec2-114">Use the view matrix to scale world space by -1 in the z-direction.</span></span> <span data-ttu-id="e1ec2-115">Pour ce faire, retournez le signe du \_ membre 31, \_ 32, \_ 33 et \_ 34 de la structure [**D3DMATRIX**](d3dmatrix.md) que vous utilisez pour votre matrice d’affichage.</span><span class="sxs-lookup"><span data-stu-id="e1ec2-115">To do this, flip the sign of the \_31, \_32, \_33, and \_34 member of the [**D3DMATRIX**](d3dmatrix.md) structure that you use for your view matrix.</span></span>

<span data-ttu-id="e1ec2-116">Pour obtenir ce qui se trouve dans un monde de droite, utilisez les fonctions [**D3DXMatrixPerspectiveRH**](d3dxmatrixperspectiverh.md) et [**D3DXMatrixOrthoRH**](d3dxmatrixorthorh.md) pour définir la transformation de projection.</span><span class="sxs-lookup"><span data-stu-id="e1ec2-116">To obtain what amounts to a right-handed world, use the [**D3DXMatrixPerspectiveRH**](d3dxmatrixperspectiverh.md) and [**D3DXMatrixOrthoRH**](d3dxmatrixorthorh.md) functions to define the projection transform.</span></span> <span data-ttu-id="e1ec2-117">Toutefois, veillez à utiliser la fonction [**D3DXMatrixLookAtRH**](d3dxmatrixlookatrh.md) correspondante, à inverser l’ordre de sélection de face arrière et à disposer les mappages de cube en conséquence.</span><span class="sxs-lookup"><span data-stu-id="e1ec2-117">However, be careful to use the corresponding [**D3DXMatrixLookAtRH**](d3dxmatrixlookatrh.md) function, reverse the backface-culling order, and lay out the cube maps accordingly.</span></span>

<span data-ttu-id="e1ec2-118">Bien que les coordonnées de gauche et de droite sont les systèmes les plus courants, il existe un large éventail d’autres systèmes de coordonnées utilisés dans les logiciels 3D.</span><span class="sxs-lookup"><span data-stu-id="e1ec2-118">Although left-handed and right-handed coordinates are the most common systems, there is a variety of other coordinate systems used in 3D software.</span></span> <span data-ttu-id="e1ec2-119">Par exemple, il n’est pas rare pour les applications de modélisation 3D d’utiliser un système de coordonnées dans lequel l’axe des y pointe vers la visionneuse et l’éloigne de celle-ci, et l’axe z pointe vers le haut.</span><span class="sxs-lookup"><span data-stu-id="e1ec2-119">For example, it is not unusual for 3D modeling applications to use a coordinate system in which the y-axis points toward or away from the viewer, and the z-axis points up.</span></span>

<span data-ttu-id="e1ec2-120">Formellement, l’orientation d’un ensemble de vecteurs de base (par exemple, un système de coordonnées) peut être trouvée par le calcul du déterminant de la matrice définie par l’ensemble de vecteurs de base.</span><span class="sxs-lookup"><span data-stu-id="e1ec2-120">Formally, the orientation of a set of basis vectors (i.e. a coordinate system) can be found by the computing the determinant of the matrix defined by the particular set of basis vectors.</span></span> <span data-ttu-id="e1ec2-121">Si le déterminant est positif, on dit qu’il s’agit d’une « orientation positive » (ou droitier).</span><span class="sxs-lookup"><span data-stu-id="e1ec2-121">If the determinant is positive, the basis is said to be "positively" oriented (or right-handed).</span></span> <span data-ttu-id="e1ec2-122">Si le déterminant est négatif, la base est dite « orientée négativement » (ou gauche).</span><span class="sxs-lookup"><span data-stu-id="e1ec2-122">If the determinant is negative, the basis is said to be "negatively" oriented (or left-handed).</span></span> <span data-ttu-id="e1ec2-123">Pour obtenir une explication de ce qu’est un déterminante, consultez toute ressource algébrique linéaire.</span><span class="sxs-lookup"><span data-stu-id="e1ec2-123">For an explanation of what a determinant is, see any linear algebra resource.</span></span>

<span data-ttu-id="e1ec2-124">De manière informelle, vous pouvez utiliser la « règle de droite/gauche » pour déterminer si un ensemble donné de vecteurs de base forme un système de coordonnées droitier ou gauche.</span><span class="sxs-lookup"><span data-stu-id="e1ec2-124">Informally, you can use the "right/left hand rule" to determine if a given set of basis vectors form either a right or left handed coordinate system.</span></span>

<span data-ttu-id="e1ec2-125">Les opérations essentielles effectuées sur les objets définis dans un système de coordonnées 3D sont la translation, la rotation et la mise à l’échelle.</span><span class="sxs-lookup"><span data-stu-id="e1ec2-125">The essential operations performed on objects defined in a 3D coordinate system are translation, rotation, and scaling.</span></span> <span data-ttu-id="e1ec2-126">Vous pouvez combiner ces transformations de base pour créer une matrice de transformation.</span><span class="sxs-lookup"><span data-stu-id="e1ec2-126">You can combine these basic transformations to create a transform matrix.</span></span> <span data-ttu-id="e1ec2-127">Pour plus d’informations, consultez [transformations (Direct3D 9)](transforms.md).</span><span class="sxs-lookup"><span data-stu-id="e1ec2-127">For details, see [Transforms (Direct3D 9)](transforms.md).</span></span>

<span data-ttu-id="e1ec2-128">Lorsque vous combinez ces opérations, les résultats ne sont pas commutatés ; l’ordre dans lequel vous multipliez les matrices est important.</span><span class="sxs-lookup"><span data-stu-id="e1ec2-128">When you combine these operations, the results are not commutative; the order in which you multiply matrices is important.</span></span>
