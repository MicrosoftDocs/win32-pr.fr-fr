# <a name="working-with-device-windows-direct3d-9"></a>utilisation des Windows d’appareil (Direct3D 9)

Cette section répertorie un problème que vous pouvez rencontrer lors de l’utilisation de fenêtres d’appareil dans des applications Direct3D.

-   Direct3D bloque uniquement les fenêtres de focus au lieu de la fenêtre du périphérique avec la fonction de traitement des messages Direct3D et traite uniquement les messages de la fenêtre de focus. Par conséquent, la fenêtre de focus doit être le parent d’une fenêtre d’appareil.
