# <a name="manipulating-the-depth-buffer-direct3d-9"></a>Manipulation de la mémoire tampon de profondeur (Direct3D 9)

Les mémoires tampons de profondeur sont associées à l’appareil. Les applications sont nécessaires pour déplacer les tampons de profondeur lorsqu’ils définissent des cibles de rendu. Les méthodes [**IDirect3DDevice9 :: GetDepthStencilSurface**](/windows/desktop/api) et [**IDirect3DDevice9 :: SetDepthStencilSurface**](/windows/win32/api/d3d9helper/nf-d3d9helper-idirect3ddevice9-setdepthstencilsurface) sont utilisées pour manipuler les tampons de profondeur.
