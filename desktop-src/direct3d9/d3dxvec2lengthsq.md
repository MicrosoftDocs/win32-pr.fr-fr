# <a name="d3dxvec2lengthsq-function"></a>D3DXVec2LengthSq fonction)

```C++
FLOAT D3DXVec2LengthSq(
  _In_ const D3DXVECTOR2 *pV
);
```

<dl> <dt>

</dt> <dd>
