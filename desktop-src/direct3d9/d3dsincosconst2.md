# <a name="d3dsincosconst2"></a>D3DSINCOSCONST2

Crée la deuxième constante (dans Register src3) pour l’instruction de nuanceur de pixels [SinCos,-PS](../direct3dhlsl/sincos---ps.md) .

``` syntax
#define D3DSINCOSCONST2 -0.020833334f, -0.12500000f, 1.0f, 0.50000000f
```

## <a name="return-value"></a>Valeur renvoyée

Retourne la deuxième constante requise par [SinCos,-PS](../direct3dhlsl/sincos---ps.md).
