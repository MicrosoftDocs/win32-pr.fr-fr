# <a name="animtickspersecond"></a>AnimTicksPerSecond

Décrit un facteur d’échelle d’animation.

``` syntax
template AnimTicksPerSecond
{
    < 9E415A43-7BA6-4a73-8743-B73D47E88476 >
    DWORD AnimTicksPerSecond;
} 
```

Où :

-   AnimTicksPerSecond : facteur d’échelle permettant de convertir les horodatages d’animation en heure globale. Sa valeur divise l’heure de l’animation par 30, 60, 24 ou 25.
