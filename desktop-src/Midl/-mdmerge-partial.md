# <a name="partial-switch"></a>commutateur/partial

Le commutateur **/Partial** spécifie que les types non résolus sont résolus par rapport aux fichiers de métadonnées répertoriés par le commutateur [**/Metadata \_ dir**](-mdmerge-metadata-dir.md) .

``` syntax
mdmerge /partial 
```

## <a name="switch-options"></a>Options de commutateur

Ce commutateur n’a aucun paramètre.

## <a name="remarks"></a>Notes

## <a name="examples"></a>Exemples

**mdmerge /partial**
