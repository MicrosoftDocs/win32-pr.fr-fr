# <a name="nomd-switch"></a>commutateur/nomd

Le commutateur **/Nomd** spécifie qu’aucun fichier de métadonnées (. winmd) n’est généré.

``` syntax
midlrt /nomd
```

## <a name="switch-options"></a>Options de commutateur

Ce commutateur n’a aucun paramètre.

## <a name="remarks"></a>Notes

## <a name="examples"></a>Exemples

**midlrt/nomd**
