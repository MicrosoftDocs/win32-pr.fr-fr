# <a name="double-attribute"></a>attribut double

Le mot clé **double** désigne un nombre à virgule flottante 64 bits.

``` syntax
double identifier-name;
```

## <a name="parameters"></a>Paramètres

<dl> <dt>

*identificateur-nom* 
</dt> <dd>

Spécifie un identificateur MIDL valide. Les identificateurs MIDL valides se composent de 31 caractères alphanumériques et/ou de traits de soulignement, et doivent commencer par un caractère alphabétique ou un trait de soulignement.

</dd> </dl>

## <a name="remarks"></a>Notes

Le type **double** est l’un des types de base du langage IDL (Interface Definition Language). Ce type peut apparaître comme un spécificateur de type dans les déclarations [**typedef**](typedef.md) , les déclarations générales et les déclarateurs de fonction (en tant que spécificateur de type fonction-retour et spécificateur de type de paramètre). Pour le contexte dans lequel les spécificateurs de type s’affichent, consultez [fichier de définition d’interface (IDL)](interface-definition-idl-file.md).

Le type **double** ne peut pas apparaître dans les déclarations [**const**](const.md) .
