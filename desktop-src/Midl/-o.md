# <a name="o-switch"></a>commutateur/o

Le commutateur **/o** redirige la sortie, par exemple des messages d’erreur et des avertissements, de la sortie standard vers le fichier spécifié.

``` syntax
midl /o outputfile
```

## <a name="switch-options"></a>Options de commutateur

<dl> <dt>

*outputfile* 
</dt> <dd>

Spécifie le nom du fichier dans lequel le compilateur MIDL redirige la sortie, par exemple les messages d’erreur et les avertissements.

</dd> </dl>
