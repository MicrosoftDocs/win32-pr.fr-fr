# <a name="win32-switch"></a>commutateur/Win32

Le commutateur **/Win32** indique au compilateur MIDL de générer des fichiers stub, ou un fichier bibliothèque de types, pour un environnement 32 bits.

``` syntax
midl /win32
```

## <a name="switch-options"></a>Options de commutateur

Ce commutateur n’a aucun paramètre.

## <a name="remarks"></a>Notes

Le commutateur **/Win32** équivaut à définir le commutateur [**/env**](-env.md) sur Win32.
