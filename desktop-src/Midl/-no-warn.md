# <a name="no_warn-switch"></a>\_commutateur d’avertissement/non

Le commutateur **/non \_ WARN** indique au compilateur MIDL de supprimer les messages d’avertissement.

``` syntax
midl /no_warn
```

## <a name="switch-options"></a>Options de commutateur

Ce commutateur n’a aucun paramètre.

## <a name="remarks"></a>Notes

L’utilisation du commutateur **/non \_ WARN** équivaut à **/W0**.

## <a name="examples"></a>Exemples

**MIDL/non \_ WARN nom_fichier. idl**

**MIDL/W0 NomFichier. idl**
