# <a name="midl-and-rpc"></a>MIDL et RPC

Lorsque vous développez une application RPC, vous utilisez un éditeur de texte pour créer un fichier IDL qui définit l’interface entre le client et le serveur. Vous utilisez le compilateur MIDL pour générer des fichiers sources requis par vos programmes client et serveur. Les rubriques suivantes présentent des informations sur l’utilisation du compilateur MIDL pour les applications RPC :

-   [Fichiers générés pour une interface RPC](files-generated-for-an-rpc-interface.md)
-   [Ciblage des stubs pour des plateformes 32 bits ou 64 bits spécifiques](targeting-stubs-for-specific-32-bit-or-64-bit-platforms.md)

Pour plus d’informations, consultez [génération d’applications RPC](/windows/desktop/Rpc/building-rpc-applications).
