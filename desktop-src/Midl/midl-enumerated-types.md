# <a name="midl-enumerated-types"></a>Types énumérés MIDL

La déclaration d' [**énumération**](enum.md) n’est pas traduite par le compilateur MIDL en instructions **\# define** , comme c’est le cas pour certains compilateurs DCE. Elle est reproduite sous la forme d’une déclaration **enum** de langage C dans le fichier d’en-tête généré.
