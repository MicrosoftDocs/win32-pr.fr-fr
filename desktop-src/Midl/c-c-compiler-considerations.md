# <a name="cc-compiler-considerations"></a>C/C++-considérations relatives au compilateur

Le compilateur MIDL doit interagir avec le compilateur C et le préprocesseur C. MIDL n’accepte pas la syntaxe C++ sur l’entrée. Le fichier. h généré contient des définitions de style C et C++ pour les interfaces. Les rubriques suivantes décrivent la configuration requise pour le compilateur C.

-   [C-problèmes de compression du compilateur](c-compiler-packing-issues.md)
-   [C-définitions de compilateur pour les proxys/stubs](c-compiler-definitions-for-proxy-stubs.md)
