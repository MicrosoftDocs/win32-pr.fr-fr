# <a name="nonencapsulated-unions"></a>Unions qui ne sont pas encapsulées

Une Union non encapsulée est une Union qui n’est pas contenue dans une structure avec son discriminante. L’Union non encapsulée est indiquée par la présence du **\[ \_ type \] de commutateur** d’attribut de type et le **\[ commutateur \_ \]** d’attribut de champ est.

Pour obtenir une description de la forme d’une Union non encapsulée, consultez [**Union**](union.md).

La forme des unions doit être la même sur toutes les plateformes pour garantir l’interconnexion.

Pour plus d’informations, consultez Unions [encapsulées](encapsulated-unions.md), [**Union**](union.md), **\[** [**\_ commutateur**](switch-is.md) **\]** et **\[** [**\_ type de commutateur**](switch-type.md) **\]** .
