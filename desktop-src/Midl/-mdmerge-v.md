# <a name="v-switch"></a>/v (commutateur)

Le commutateur **/v** spécifie que les références de type de métadonnées sont validées.

``` syntax
mdmerge /v 
```

## <a name="switch-options"></a>Options de commutateur

Ce commutateur n’a aucun paramètre.

## <a name="remarks"></a>Notes

## <a name="examples"></a>Exemples

**mdmerge/v**
