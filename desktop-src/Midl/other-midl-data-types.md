# <a name="other-midl-data-types"></a>Autres types de données MIDL

MIDL reconnaît plusieurs types de données COM et Automation, tels que **BSTR**, **Variant** et **SAFEARRAY**, et les handles com, tels que **HMETAFILE \_ PICT**, **HENHMETAFILE**, **HMETAFILE**, **HBITMAP**, **HPALETTE** et **HGLOBAL**. Pour plus d’informations, y compris les noms des fichiers d’importation nécessaires à l’utilisation de ces types, consultez [marshaling des types de données OLE](marshaling-ole-data-types.md).
