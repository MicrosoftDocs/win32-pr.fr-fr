# <a name="enum_class-switch"></a>/enum ( \_ commutateur de classe)

Le commutateur de **\_ classe/enum** permet l’utilisation de la construction de classe **enum** C++.

``` syntax
midlrt /enum_class 
```

## <a name="switch-options"></a>Options de commutateur

Ce commutateur n’a aucun paramètre.

## <a name="examples"></a>Exemples
