# <a name="ns_prefix-switch"></a>\_commutateur de préfixe/ns

Le commutateur de **\_ préfixe/ns** ajoute le préfixe « Abi » à tous les types.

``` syntax
midlrt /ns_prefix 
```

## <a name="switch-options"></a>Options de commutateur

Ce commutateur n’a aucun paramètre.

## <a name="remarks"></a>Notes

## <a name="examples"></a>Exemples

**préfixe midlrt/NS \_**
