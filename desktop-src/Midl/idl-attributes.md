# <a name="idl-attributes"></a>Attributs IDL

Les attributs sont des mots clés qui spécifient les caractéristiques d’une interface et des données et des méthodes au sein de cette interface. L’ensemble d’attributs qui s’applique à une seule interface est placé entre crochets (**\[ \]**). Les rubriques suivantes présentent les catégories d’attributs qui peuvent être utilisées dans un fichier IDL.

-   [Attributs de champ](/windows/desktop/Rpc/field-attributes)
-   [Attributs d’alias et de marshaling](aliasing-and-marshaling-attributes.md)
-   [Attributs asynchrones](asynchronous-attributes.md)
-   [Attributs de tableau et de Sized-Pointer](array-and-sized-pointer-attributes.md)
-   [Attributs de type de données](data-type-attributes.md)
-   [Attributs directionnels](directional-attributes.md)
-   [Attributs d’appel de fonction](function-call-attributes.md)
-   [Attributs d’en-tête d’interface](interface-header-attributes.md)
-   [Attributs de performance](performance-attributes.md)
-   [Attributs de type pointeur](pointer-type-attributes.md)
-   [Attributs de structure et d’Union](structure-and-union-attributes.md)
-   [Attributs de la bibliothèque de types](type-library-attributes.md)
