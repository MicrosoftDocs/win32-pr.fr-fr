# <a name="odl-attribute"></a>odl (attribut)

MKTYPLIB a requis l’attribut **\[ ODL \]** sur les interfaces ODL. Le compilateur MIDL ne requiert pas l’attribut **\[ \] ODL** ; il est reconnu uniquement pour la compatibilité avec les anciens fichiers **ODL** .

> [!Note]  
> L’outil Mktyplib.exe est obsolète. Utilisez plutôt le compilateur MIDL.
