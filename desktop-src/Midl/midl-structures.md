# <a name="midl-structures"></a>Structures MIDL

La sémantique C normale s’applique aux champs de types de base. Les champs de types plus complexes, tels que des pointeurs, des tableaux et d’autres types construits, peuvent être modifiés par [**des \_ attributs**](/windows/desktop/Rpc/field-attributes)de type ou de champ. Pour plus d’informations, consultez [**struct**](struct.md).
