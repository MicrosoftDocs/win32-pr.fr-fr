# <a name="about-this-guide"></a>À propos de ce guide

Cette section est une référence pour le Microsoft Interface Definition Language (MIDL) et un guide de l’utilisateur pour le compilateur MIDL, qui génère des fichiers de langage C à partir de fichiers IDL.

La section [utilisation du compilateur MIDL](using-the-midl-compiler-2.md) répertorie les conditions requises pour le compilateur C et le préprocesseur avec lequel MIDL doit interagir. Cette section décrit également les fichiers générés par le compilateur MIDL pour les stubs RPC, les stubs DCOM et les bibliothèques de types OLE Automation.

La section des [définitions d’interface et des bibliothèques de types](interface-definitions-and-type-libraries.md) fournit des informations sur la création de fichiers de définition d’interface (IDL) et de configuration d’application pour MIDL, y compris des pages de référence pour les attributs IDL et ACF. Il contient également des informations sur la façon d’importer des fichiers et des bibliothèques de types, et d’importer des fichiers d’en-tête système.

La section de [Référence du Command-Line MIDL](midl-command-line-reference.md) contient des informations de référence pour chaque commutateur de ligne de commande et option de commutateur reconnue par le compilateur MIDL.

La section [Référence du langage MIDL](midl-language-reference.md) contient une entrée de référence pour chaque mot clé dans le Microsoft Interface Definition Language.

La section [Erreurs et avertissements du compilateur MIDL](midl-compiler-errors-and-warnings.md) répertorie les messages d’erreur et d’avertissement que le compilateur MIDL peut générer.
