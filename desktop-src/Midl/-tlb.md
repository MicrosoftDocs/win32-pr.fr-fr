# <a name="tlb-switch"></a>commutateur/TLB

Le commutateur **/TLB** spécifie un nom de fichier pour la bibliothèque de types générée par le compilateur MIDL.

``` syntax
midl /tlb filename
```

## <a name="switch-options"></a>Options de commutateur

<dl> <dt>

*extension* 
</dt> <dd>

Spécifie le nom du fichier de bibliothèque de types de sortie (TLB). Par défaut, si le commutateur **/TLB** n’est pas utilisé, le fichier TLB porte le même nom que le fichier IDL, avec l’extension. Causé.

</dd> </dl>

## <a name="examples"></a>Exemples

**MIDL/TLB newname. tlb**
