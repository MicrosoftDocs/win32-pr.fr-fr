# <a name="winmd-switch"></a>commutateur/winmd

Le commutateur [**/winmd**](-metadata-dir.md) spécifie le nom du fichier de métadonnées de sortie (. winmd).

``` syntax
midlrt /winmd metadata_filename
```

## <a name="switch-options"></a>Options de commutateur

<dl> <dt>

*\_nom du fichier de métadonnées* 
</dt> <dd>

Spécifie le nom du fichier de métadonnées.

</dd> </dl>
