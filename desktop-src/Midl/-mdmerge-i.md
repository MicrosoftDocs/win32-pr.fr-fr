# <a name="i-switch"></a>commutateur/i

Le commutateur **/i** spécifie le répertoire qui contient les fichiers de métadonnées à fusionner.

``` syntax
mdmerge /i metadata_directory
```

## <a name="switch-options"></a>Options de commutateur

<dl> <dt>

*Répertoire de métadonnées \_* 
</dt> <dd>

Spécifie le répertoire qui contient les fichiers de métadonnées à fusionner.

</dd> </dl>
