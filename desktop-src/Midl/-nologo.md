# <a name="nologo-switch"></a>/nologo (commutateur)

Le commutateur **/nologo** désactive l’affichage de la bannière de copyright.

``` syntax
midl /nologo
```

## <a name="switch-options"></a>Options de commutateur

Ce commutateur n’a aucun paramètre.
