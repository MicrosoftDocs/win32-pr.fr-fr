# <a name="amd64-switch"></a>commutateur/amd64

Le commutateur **/amd64** indique au compilateur MIDL de générer des fichiers stub, ou un fichier bibliothèque de types, pour un environnement Advanced Micro devices 64 bits.

``` syntax
midl /amd64
```

## <a name="remarks"></a>Remarques

> [!Note]  
> Le commutateur **/x64** décrit par l’aide de la ligne de commande MIDL.exe est déconseillé. **/amd64** doit être utilisé à la place.
