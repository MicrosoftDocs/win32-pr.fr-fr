# <a name="additional-files-required-to-generate-a-type-library"></a>Fichiers supplémentaires requis pour générer une bibliothèque de types

Pour compiler un fichier IDL qui contient une instruction de bibliothèque, les fichiers DLL OLE et OLE Automation doivent se trouver sur votre système. Ces fichiers sont automatiquement installés.
