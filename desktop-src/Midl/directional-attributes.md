# <a name="directional-attributes"></a>Attributs directionnels

Appliquez des attributs directionnels aux paramètres pour indiquer la direction dans laquelle les données sont transmises. Les deux attributs peuvent être appliqués à un seul paramètre, mais cela augmente la surcharge associée à un appel.



| Attribut              | Usage                                                                 |
|------------------------|-----------------------------------------------------------------------|
| [**dans**](in.md)       | Le paramètre est passé de l’appelant à la fonction appelée. |
| [**à**](out-idl.md) | Le paramètre est passé de la fonction appelée à l’appelant. |
