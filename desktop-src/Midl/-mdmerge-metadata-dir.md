# <a name="metadata_dir-switch-mdmerge"></a>/commutateur metadata_dir (MDMERGE)

Le commutateur **/Metadata \_ dir** spécifie le répertoire qui contient les fichiers de métadonnées de plateforme.

``` syntax
mdmerge /metadata_dir metadata_directory
```

## <a name="switch-options"></a>Options de commutateur

<dl> <dt>

*Répertoire de métadonnées \_* 
</dt> <dd>

Spécifie le répertoire qui contient les fichiers de métadonnées de plateforme.

</dd> </dl>

## <a name="remarks"></a>Notes

Utilisez ce commutateur pour spécifier l’emplacement du fichier de métadonnées principal pour Windows, nommé Windows. winmd.

## <a name="examples"></a>Exemples

**mdmerge/Metadata \_ dir dir1**
