Le Microsoft Interface Definition Language prend en charge un sur-ensemble de l’instruction Union C/C++. MIDL fournit deux types d’unions. Elles sont abordées dans les rubriques suivantes :

-   [Unions encapsulées](encapsulated-unions.md)
-   [Unions qui ne sont pas encapsulées](nonencapsulated-unions.md)
