# <a name="help--switch"></a>/Help (/ ?) (commutateur)

Le commutateur **/Help** (**/ ?**) indique au compilateur d’afficher un message d’utilisation détaillant tous les commutateurs et options de ligne de commande MIDL disponibles.

Le commutateur [**/Confirm**](-confirm.md) affiche les paramètres de commutateur du compilateur MIDL sélectionnés par l’utilisateur.

``` syntax
midl /help
midl /?
```

## <a name="switch-options"></a>Options de commutateur

Ce commutateur n’a aucun paramètre.

## <a name="examples"></a>Exemples

**MIDL/Help**

**MIDL/ ?**
