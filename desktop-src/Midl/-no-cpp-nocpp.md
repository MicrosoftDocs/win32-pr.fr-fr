# <a name="no_cpp-nocpp"></a>/non \_ CPP,/nocpp

Le commutateur **/non \_ CPP** spécifie que le compilateur MIDL n’appelle pas le préprocesseur C pour prétraiter le fichier IDL.

Le commutateur **/non \_ CPP** est prioritaire par rapport aux commutateurs [**/CPP \_ cmd**](-cpp-cmd.md) et [**/CPP \_ OPT**](-cpp-opt.md) .

## <a name="switch-options"></a>Options de commutateur

``` syntax
midl /no_cpp
midl /nocpp
```

## <a name="example"></a>Exemple

**MIDL/non \_ CPP nom_fichier. idl**
