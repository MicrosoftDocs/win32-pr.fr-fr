# <a name="metadata_dir-switch-midlrt"></a>/commutateur metadata_dir (MIDLRT)

Le commutateur **/Metadata \_ dir** spécifie un ou plusieurs répertoires qui contiennent des fichiers de métadonnées de plateforme.

``` syntax
midlrt /metadata_dir metadata_directory
```

## <a name="switch-options"></a>Options de commutateur

<dl> <dt>

*Répertoire de métadonnées \_* 
</dt> <dd>

Spécifie un ou plusieurs répertoires qui contiennent des fichiers de métadonnées de plateforme.

</dd> </dl>

## <a name="remarks"></a>Notes

Utilisez ce commutateur pour spécifier l’emplacement du fichier de métadonnées principal pour Windows, nommé Windows. winmd.

## <a name="examples"></a>Exemples

**midlrt/Metadata \_ dir dir1 dir2**
