# <a name="confirm-switch"></a>commutateur/Confirm

Le commutateur **/Confirm** demande au compilateur d’afficher toutes les options du compilateur MIDL sans traiter les fichiers IDL d’entrée et les fichiers ACF facultatifs.

``` syntax
midl /confirm
```

## <a name="switch-options"></a>Options de commutateur

Ce commutateur n’a aucun paramètre.

## <a name="examples"></a>Exemples

**/Confirm MIDL**

**MIDL/Confirm @response.rsp NomFichier. idl**
