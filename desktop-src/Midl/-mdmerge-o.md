# <a name="o-switch-mdmerge"></a>commutateur/o (MDMERGE)

Le commutateur **/o** spécifie le répertoire de sortie pour les fichiers de métadonnées composés.

``` syntax
mdmerge /o output_directory
```

## <a name="switch-options"></a>Options de commutateur

<dl> <dt>

*Répertoire de sortie \_* 
</dt> <dd>

Spécifie le répertoire qui contient les fichiers de métadonnées fusionnés.

</dd> </dl>
