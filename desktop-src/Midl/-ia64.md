# <a name="ia64-switch"></a>commutateur/ia64

Le commutateur **/ia64** indique au compilateur MIDL de générer des fichiers stub, ou un fichier bibliothèque de types, pour un environnement Architecture Intel 64 bits.

``` syntax
midl /ia64
```

## <a name="remarks"></a>Remarques

> [!Note]  
> il n’est pas possible d’utiliser MIDL.exe pour produire un tlb 64 bits sur Windows 2000.
