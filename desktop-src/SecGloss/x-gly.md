</dt> <dd>

Type d’encodage qui spécifie l’encodage du certificat. Les types d’encodage de certificat sont stockés dans le mot de poids faible d’une valeur **DWORD** (la valeur est : 0x00000001). Ce type d’encodage est fonctionnellement identique à \_ l' \_ encodage ASN.

</dd> <dt>

<span id="_security_xor_gly"></span><span id="_SECURITY_XOR_GLY"></span>**XOR**
</dt> <dd>

**Ou** exclusif. Opération booléenne qui retourne true uniquement si l’un de ses opérandes a la valeur true et que l’autre est false. Si les deux opérandes sont identiques (true ou false), l’opération génère la valeur false.

</dd> </dl>
