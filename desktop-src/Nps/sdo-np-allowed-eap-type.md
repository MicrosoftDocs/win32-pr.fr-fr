# <a name="np-allowed-eap-type"></a>NP-autorisé-type EAP

Modes de chiffrement EAP autorisés pour une connexion.

**Nom LDAP :** msNPAllowedEapType

**ID :** 4106 = [ **\_ attribut IAS \_ \_ \_ \_ type EAP autorisé NP**](/windows/desktop/api/sdoias/ne-sdoias-attributeid)

**Syntaxe :** Entière

**Valeurs :  **

Défini par l’IANA (Internet Assigned Numbers Authority) dans le document de [Registre EAP (Extensible Authentication Protocol)](https://www.iana.org/assignments/eap-numbers) basé sur la [norme RFC 3748](https://www.ietf.org/rfc/rfc3748.txt).

-   **MD5-Challenge** (4)
-   **Protocole EAP \_ TLS** (13)
-   **PEAP** (25)
-   **MS-EAP-authentification** (26)
