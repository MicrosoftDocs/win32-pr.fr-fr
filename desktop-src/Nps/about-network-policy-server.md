# <a name="about-network-policy-server"></a>À propos du serveur NPS (Network Policy Server)

> [!Note]  
> le Service d’authentification Internet (IAS) a été renommé serveur nps (network Policy Server) à partir de Windows Server 2008. Le contenu de cette rubrique s’applique à IAS et à NPS.

 

Le serveur NPS (Network Policy Server) est l’implémentation Microsoft d’un serveur et d’un proxy RADIUS. Les sections suivantes fournissent des informations générales sur NPS et RADIUS.

-   [Journalisation avec le serveur NPS](/windows/desktop/Nps/ias-radius-accounting-packets)
-   [Utilisation d’un serveur d’État](/windows/desktop/Nps/ias-working-with-a-state-server)
