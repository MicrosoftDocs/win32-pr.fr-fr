# <a name="elements-of-the-sdo-api"></a>Éléments de l’API SDO

L’API SDO est conçue autour d’un modèle objet composé d’objets et de collections d’objets. Dans le modèle objet SDO, les objets et les collections sont organisés dans une hiérarchie, c’est-à-dire que les objets et les collections contiennent d’autres objets et Collections.

Les objets sont manipulés en extrayant et en définissant les propriétés de l’objet.

Les sections suivantes décrivent comment l’API SDO utilise les objets, les propriétés d’objet et les collections :

-   [Objets et propriétés](/windows/desktop/Nps/sdo-objects-and-properties)
-   [Hiérarchie du modèle objet](/windows/desktop/Nps/sdo-object-model-hierarchy)
-   [Attributs](/windows/desktop/Nps/sdo-sdo-supported-attributes)
