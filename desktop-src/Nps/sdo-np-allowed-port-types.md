# <a name="np-allowed-port-types"></a>NP-allowed-port-types

Types de ports autorisés pour une connexion.

**Nom LDAP :** msNPAllowedPortTypes

**ID :** 4104 = [ **\_ attribut IAS \_ \_ types de \_ ports \_ autorisés NP**](/windows/desktop/api/sdoias/ne-sdoias-attributeid)

**Syntaxe :** Énumérateur

* * Valeurs : * *

Défini par l’IANA (Internet Assigned Numbers Authority) dans le document [types RADIUS](https://www.iana.org/assignments/radius-types) basé sur la [RFC 2865](https://www.ietf.org/rfc/rfc2865.txt).

-   Async (modem) (0)
-   Synchronisation (ligne T1) (1)
-   Synchronisation RNIS (2)
-   RNIS Async V. 120 (3)
-   RNIS Async V. 110 (4)
-   Virtuel (VPN) (5)
-   PIAFS (6)
-   Canal Clear HDLC (7)
-   X. 25 (8)
-   X. 75 (9)
-   G. 3 télécopie (10)
-   SDSL-DSL symétrique (11)
-   ADSL-CAP-modulation de phase d’amplitude non-asymétrique DSL (12)
-   ADSL-DMT-asymétrique à plusieurs fréquences discrètes DSL (13)
-   IDSL-ligne de l’abonné numérique RNIS (14)
-   Ethernet (15)
-   xDSL-ligne d’abonné numérique de type inconnu (16)
-   Câble (17)
-   Sans fil-autres (18)
-   Sans fil-IEEE 802,11 (19)
-   Token Ring (20)
-   FDDI (21)
