# <a name="server-data-objects-interfaces"></a>Interfaces d’objets de données serveur

L’API SDO (Server Data Objects) expose les interfaces suivantes.




| Interface | Objectif | 
|-----------|---------|
| <a href="/windows/desktop/api/sdoias/nn-sdoias-isdo"><strong>ISdo</strong></a> | Manipuler un objet SDO.<br /> | 
| <a href="/windows/desktop/api/sdoias/nn-sdoias-isdocollection"><strong>ISdoCollection</strong></a> | Manipuler une collection d’objets SDO.<br /> | 
| <a href="/windows/desktop/api/sdoias/nn-sdoias-isdodictionaryold"><strong>ISdoDictionaryOld</strong></a> | Manipuler le dictionnaire d’attributs.<br /> | 
| <a href="/windows/desktop/api/sdoias/nn-sdoias-isdomachine"><strong>ISdoMachine</strong></a> | Gérez l’ordinateur SDO et récupérez les objets SDO.<br /> | 
| <a href="/windows/desktop/api/sdoias/nn-sdoias-isdoservicecontrol"><strong>ISdoServiceControl</strong></a> | Gérer le service administré via SDO, autrement dit, le service d’authentification Internet (IAS) ou le serveur d’accès à distance (RAS).<br /><blockquote>[!Note]<br />le Service d’authentification Internet (IAS) a été renommé serveur nps (network Policy Server) à partir de Windows Server 2008.</blockquote><br /> |
