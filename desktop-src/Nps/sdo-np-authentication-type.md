# <a name="np-authentication-type"></a>NP-Authentication-type

**Nom LDAP :** msNPAuthenticationType2

**ID :** 4105 = [ **\_ attribut IAS \_ \_ \_ type d’authentification NP**](/windows/desktop/api/sdoias/ne-sdoias-attributeid)

-   **Service IAS \_ AUTH \_ PAP** (1)
-   **Service IAS \_ AUTHENTIFICATION \_ md5chap** (2)
-   **Service IAS \_ AUTHENTIFICATION \_ MSCHAP** (3)
-   **Service IAS \_ AUTHENTIFICATION \_ MSCHAP2** (4)
-   **Service IAS \_ AUTHENTIFICATION \_ EAP** (5)
-   **Service IAS \_ AUTHENTIFICATION \_ ARAP** (6)
-   **Service IAS \_ AUTHENTIFICATION \_ None** (7)
-   **Service IAS \_ AUTHENTIFICATION \_ personnalisée** (8)
-   **Service IAS \_ AUTHENTIFICATION \_ MSCHAP \_ CPW** (9)
-   **Service IAS \_ AUTH \_ MSCHAP2 \_ CPW** (10)
-   **Service IAS \_ AUTHENTIFICATION \_ PEAP** (11)
