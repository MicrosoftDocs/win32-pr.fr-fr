# <a name="copyright"></a>copyright

L’attribut **Copyright** contient un message de copyright pour le contenu.

## <a name="global-constant"></a>Constante globale

\_wszWMCopyright g

## <a name="data-type"></a>Type de données

**\_chaîne de type WMT \_**

## <a name="example"></a>Exemple



| Type de fichier | Valeur d'exemple                          |
|-----------|----------------------------------------|
| Quelconque       | « Copyright (C) Microsoft Corporation ». |



 

## <a name="remarks"></a>Notes

Le texte d’un message de copyright n’est conforme à aucun standard. Les sociétés peuvent utiliser le format approprié.

Cet attribut s’applique à l’ensemble du fichier.
