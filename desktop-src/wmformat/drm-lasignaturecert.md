# <a name="drm_lasignaturecert"></a>\_LASIGNATURECERT DRM

L’attribut **DRM \_ LASignatureCert** contient le certificat qui authentifie la signature dans l’en-tête DRM.

## <a name="global-constant"></a>Constante globale

g \_ wszWMDRM \_ LASignatureCert

## <a name="data-type"></a>Type de données

**\_chaîne de type WMT \_**

## <a name="remarks"></a>Remarques

Cette propriété peut être définie à l’aide de la méthode [**IWMDRMWriter :: SetDRMAttribute**](/previous-versions/windows/desktop/api/Wmsdkidl/nf-wmsdkidl-iwmdrmwriter-setdrmattribute) . Elle n’est pas accessible à l’objet lecteur.
