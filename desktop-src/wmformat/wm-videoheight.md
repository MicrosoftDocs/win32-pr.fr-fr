# <a name="wmvideoheight"></a>WM/VideoHeight

L’attribut **WM/VideoHeight** contient la hauteur, en pixels, d’une image vidéo dans un flux.

## <a name="global-constant"></a>Constante globale

\_wszWMVideoHeight g

## <a name="data-type"></a>Type de données

**\_valeur DWORD de type WMT \_**
