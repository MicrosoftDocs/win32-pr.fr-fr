# <a name="wmwmshadowfilesourcefiletype-windows-media-format-11-sdk"></a>WM/WMShadowFileSourceFileType (kit de développement logiciel (SDK) Windows Media format 11)

**WM/WMShadowFileSourceFileType** contient le type de fichier du fichier d’origine à partir duquel le fichier ASF est dérivé.

## <a name="global-constant"></a>Constante globale

\_wszWMWMShadowFileSourceFileType g

## <a name="data-type"></a>Type de données

**\_chaîne de type WMT \_**

## <a name="remarks"></a>Notes

Le contenu qui existe dans un format de fichier autre que ASF et qui est protégé par une certaine sorte de gestion des droits peut être converti en fichier Windows Media protégé par Windows Media DRM par le biais d’un processus appelé importation de licence. Ce fichier ASF doit contenir cet attribut, ainsi que WM/WMShadowFileSourceDRMType pour suivre l’origine du contenu.
