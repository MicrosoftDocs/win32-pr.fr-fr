# <a name="wmvideoframerate"></a>WM/VideoFrameRate

L’attribut **WM/VideoFrameRate** contient la vitesse, en images par seconde, de la vidéo dans un flux.

## <a name="global-constant"></a>Constante globale

\_wszWMVideoFrameRate g

## <a name="data-type"></a>Type de données

**\_valeur DWORD de type WMT \_**
