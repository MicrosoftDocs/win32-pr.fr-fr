# <a name="text-streams"></a>Flux de texte

Les flux de texte contiennent du texte brut stocké dans des exemples. Votre application peut utiliser le texte d’un flux de texte comme vous le souhaitez. Par exemple, vous souhaiterez peut-être créer une application language tutor qui introduit des mots dans une langue étrangère. Vous pouvez inclure chaque mot comme un exemple de flux de texte à une heure de présentation qui correspond au mot parlé dans un flux audio. Votre application peut ensuite afficher le texte lorsque l’exemple est remis.
