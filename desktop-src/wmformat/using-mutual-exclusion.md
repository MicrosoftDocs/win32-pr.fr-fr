# <a name="using-mutual-exclusion"></a>Utilisation de l’exclusion mutuelle

Vous pouvez utiliser des objets d’exclusion mutuelle pour spécifier des groupes de flux, dont un seul sera remis à la fois. L’exclusion mutuelle peut être utilisée pour gérer un large éventail de scénarios, à la fois natifs au kit de développement logiciel (SDK) et personnalisé.

Les sections suivantes décrivent comment utiliser différents types d’exclusion mutuelle.



| Section                                                                                  | Description                                                                                                       |
|------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------|
| [Utilisation de l’exclusion mutuelle à vitesse de transmission multiple](using-multiple-bit-rate-mutual-exclusion.md) | Décrit comment utiliser l’exclusion mutuelle pour encoder une entrée en plusieurs flux avec des paramètres de vitesse de transmission différents. |
| [Utilisation de types d’exclusion mutuelles personnalisés](using-custom-mutual-exclusion-types.md)           | Présente l’exclusion mutuelle personnalisée.                                                                                |
