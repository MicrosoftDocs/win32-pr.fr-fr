# <a name="bannerimagetype"></a>BannerImageType

L’attribut **BannerImageType** spécifie le type d’image stocké dans l’attribut [**BannerImageData**](bannerimagedata.md) . Cet attribut est une valeur du type d’énumération de [**\_ \_ type d’image WMT**](/previous-versions/windows/desktop/api/Wmsdkidl/ne-wmsdkidl-wmt_image_type) .

**\_valeur DWORD de type WMT \_**
