# <a name="microsoft-windows-media-drm-client-programming-guide"></a>Guide de programmation du client Microsoft Windows Media DRM

Ce guide fournit des informations sur l’utilisation des fonctionnalités des API étendues du client Windows Media DRM dans vos applications. Les sections suivantes expliquent en détail les étapes à suivre pour implémenter des fonctionnalités DRM.



| Section                                                                                                                          | Description                                                                                                |
|----------------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------|
| [Prise en main](drm-getting-started.md)                                                                                       | Fournit des informations sur la configuration de projets C++ qui utilisent les API étendues du client Windows Media DRM.    |
| [Acquisition de licences](acquiring-licenses.md)                                                                                     | Décrit comment récupérer des licences sur l’ordinateur client.                                                      |
| [Obtention d’informations à partir de licences dans le magasin de licences local](getting-information-from-licenses-in-the-local-license-store.md) | Décrit comment obtenir des informations sur les droits pour le contenu protégé.                                       |
| [Réalisation de l’individualisation DRM](performing-drm-individualization.md)                                                         | Décrit comment mettre à jour et individualiser le composant DRM sur l’ordinateur client pour le rendre plus sécurisé. |
| [Exportation DRM](drm-export.md)                                                                                                     | Décrit comment exporter du contenu protégé par Windows Media DRM.                                            |
| [Importation DRM](drm-import.md)                                                                                                     | Décrit comment importer le contenu protégé par Windows Media DRM.                                            |
| [Révocation de licence](drm-license-revocation.md)                                                                                 | Décrit comment supprimer des licences du magasin de licences local.                                             |
| [Révocation et renouvellement de composants automatisés](automated-component-revocation-and-renewal.md)                                     | Décrit le système de révocation et de renouvellement automatisé Windows Media DRM à l’aide des activateurs du contenu.            |
