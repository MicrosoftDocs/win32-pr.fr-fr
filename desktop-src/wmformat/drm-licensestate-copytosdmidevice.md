# <a name="drm_licensestate_copytosdmidevice"></a>\_LICENSESTATE DRM \_ CopyToSDMIDevice

La propriété **DRM \_ LicenseState \_ CopyToSDMIDevice** contient une structure de [**\_ \_ \_ données d’état de licence WM**](/previous-versions/windows/desktop/legacy/dd757942(v=vs.85)) qui contient des détails sur la façon dont ce droit a été appliqué au contenu.

## <a name="global-constant"></a>Constante globale

g \_ wszWMDRM \_ LicenseState \_ CopyToSDMIDevice

## <a name="data-type"></a>Type de données

**\_binaire de type WMT \_**

## <a name="remarks"></a>Notes

Il s’agit d’une propriété en lecture seule qui est récupérée à l’aide de [**IWMDRMReader :: GetDRMProperty**](/previous-versions/windows/desktop/api/Wmsdkidl/nf-wmsdkidl-iwmdrmreader-getdrmproperty).
