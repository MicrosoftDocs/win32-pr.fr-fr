# <a name="sending-asf-data-over-a-network"></a>Envoi de données ASF sur un réseau

le kit de développement logiciel (SDK) Windows Media Format prend en charge deux méthodes différentes pour envoyer des données ASF sur un réseau :

-   L’application peut diffuser les données ASF à l’aide du protocole HTTP.
-   l’application peut envoyer les données ASF à un point de publication sur un serveur Windows Media. Le serveur distribue ensuite les données.

Les rubriques suivantes décrivent ces fonctionnalités :

-   [Diffusion de données ASF](broadcasting-asf-data.md)
-   [Envoi de données ASF à un point de publication](sending-asf-data-to-a-publishing-point.md)
