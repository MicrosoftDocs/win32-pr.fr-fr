# <a name="windows-media-format-sdk-interfaces"></a>Interfaces du SDK du format Windows Media

Les deux tableaux suivants décrivent les interfaces prises en charge par ce kit de développement logiciel (SDK) et indiquent leur relation aux objets et autres interfaces.



| Interface                                                          | Description                                                                                                                                                                                                                                                       |
|--------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**INSNetSourceCreator**](/previous-versions/windows/desktop/api/wmnetsourcecreator/nn-wmnetsourcecreator-insnetsourcecreator)                 | Crée un [*plug-in source de*](wmformat-glossary.md)réseau d’administration, qui peut être utilisé pour mettre en cache les mots de passe et localiser les serveurs proxy.                                                                                                      |
| [**INSSBuffer**](/previous-versions/windows/desktop/api/wmsbuffer/nn-wmsbuffer-inssbuffer)                                   | Contrôle une mémoire tampon stockée dans un objet de mémoire tampon. | [**INSSBuffer2**](/previous-versions/windows/desktop/api/wmsbuffer/nn-wmsbuffer-inssbuffer2)                                 | Hérite de **INSSBuffer**. | [**INSSBuffer3**](/previous-versions/windows/desktop/api/wmsbuffer/nn-wmsbuffer-inssbuffer3)                                 | Définit et récupère les propriétés de la mémoire tampon. ## <a name="remarks"></a>Remarques

L’identificateur d’interface de chacune des interfaces dans ce kit de développement logiciel (SDK) est le nom de l’interface précédée de « IID \_ ». Par exemple, l’identificateur de l’interface **IWMReaderAdvanced** est IID \_ IWMReaderAdvanced. Pour obtenir un pointeur vers l’interface **IWMReaderAdvanced** d’un objet lecteur, utilisez le code suivant, où *pReader* est un pointeur vers une interface **IWMReader** :


```C++
IWMReaderAdvanced *pReaderAdvanced;
hr = pReader->QueryInterface(IID_IWMReaderAdvanced,
                             (LPVOID*) &pReaderAdvanced);

```
