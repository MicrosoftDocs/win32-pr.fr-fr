# <a name="wmmediaisstereo"></a>WM/MediaIsStereo

L’attribut **WM/MediaIsStereo** indique si le contenu est diffusé avec un son stéréo.

## <a name="global-constant"></a>Constante globale

\_wszWMMediaIsStereo g

## <a name="data-type"></a>Type de données

**\_type WMT \_ bool**

## <a name="remarks"></a>Notes

Cet attribut est utilisé pour la télévision de diffusion.
