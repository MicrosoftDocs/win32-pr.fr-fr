# <a name="wmalbumcoverurl"></a>WM/AlbumCoverURL

L’attribut **WM/AlbumCoverURL** contient l’adresse d’une page Web qui contient une image de la couverture de l’album et des informations sur l’album.

## <a name="global-constant"></a>Constante globale

\_wszWMAlbumCoverURL g

## <a name="data-type"></a>Type de données

**\_chaîne de type WMT \_**
