# <a name="wmmediaispremiere"></a>WM/MediaIsPremiere

L’attribut **WM/MediaIsPremiere** indique si le contenu est le premier d’une série.

## <a name="global-constant"></a>Constante globale

\_wszWMMediaIsPremiere g

## <a name="data-type"></a>Type de données

**\_type WMT \_ bool**

## <a name="remarks"></a>Notes

Cet attribut est utilisé pour la télévision de diffusion.
