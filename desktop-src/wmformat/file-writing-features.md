# <a name="file-writing-features"></a>Fonctionnalités d’écriture de fichier

L’une des principales fonctionnalités du kit de développement logiciel (SDK) du format Windows Media est la possibilité d’écrire des fichiers au format ASF (Advanced Systems Format). L’objet Writer est utilisé pour écrire des fichiers ASF. Pour plus d’informations, consultez [writer, objet](writer-object.md).

Dans le scénario d’écriture de fichiers le plus basique, vous affectez un profil à utiliser et un nom du fichier à créer. Vous transmettez des exemples à l’enregistreur un par un. Lorsque vous avez terminé de passer des exemples au writer, il termine ses opérations et termine le fichier ASF. Pour plus d’informations sur l’écriture de fichiers de base, consultez [écriture de fichiers ASF](writing-asf-files.md).

Le writer prend en charge plusieurs fonctionnalités avancées, qui sont décrites dans les sections suivantes.

-   [Redimensionnement vidéo](video-resizing.md)
-   [Conversion de l’espace colorimétrique](color-space-conversion.md)
-   [Rééchantillonnage audio](audio-resampling.md)
-   [Récepteurs](sinks.md)
-   [Prise en charge du filigrane](watermarking-support.md)
-   [Formats d’entrée, paramètres d’entrée et extensions d’unité de données](input-formats-input-settings-and-data-unit-extensions.md)
-   [Énumération de format d’entrée](input-format-enumeration.md)
