# <a name="protected-files"></a>Fichiers protégés

lorsqu’un fichier est protégé à l’aide d’Windows Media DRM, la section de données du fichier est chiffrée. Le processus de protection d’un fichier est également appelé empaquetage du fichier. les api étendues du Client DRM Windows Media ne vous permettent pas d’empaqueter des fichiers. vous pouvez empaqueter des fichiers à l’aide des objets du kit de développement logiciel (sdk) media Rights Manager Windows ou du sdk Windows Format media.

Un utilisateur ne peut pas accéder aux données d’un fichier protégé sans obtenir au préalable une licence pour ce fichier.
