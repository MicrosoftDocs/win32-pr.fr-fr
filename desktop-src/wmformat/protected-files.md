# <a name="protected-files"></a>Fichiers protégés

Lorsqu’un fichier est protégé à l’aide de Windows Media DRM, la section de données du fichier est chiffrée. Le processus de protection d’un fichier est également appelé empaquetage du fichier. Les API étendues du client Windows Media DRM ne vous permettent pas d’empaqueter des fichiers. Vous pouvez empaqueter des fichiers à l’aide des objets du kit de développement logiciel Windows Media Rights Manager ou du kit de développement logiciel (SDK) Windows Media format.

Un utilisateur ne peut pas accéder aux données d’un fichier protégé sans obtenir au préalable une licence pour ce fichier.
