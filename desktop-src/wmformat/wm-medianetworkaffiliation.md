# <a name="wmmedianetworkaffiliation"></a>WM/MediaNetworkAffiliation

L’attribut **WM/MediaNetworkAffiliation** contient l’affiliation réseau du canal qui diffuse le contenu.

## <a name="global-constant"></a>Constante globale

\_wszWMMediaNetworkAffiliation g

## <a name="data-type"></a>Type de données

**\_chaîne de type WMT \_**

## <a name="remarks"></a>Notes

Cet attribut est utilisé pour la télévision de diffusion.
