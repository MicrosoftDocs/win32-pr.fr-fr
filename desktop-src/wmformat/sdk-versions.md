# <a name="sdk-versions"></a>Versions du SDK

cette documentation couvre le kit de développement logiciel (sdk) Windows media format 7 via le kit de développement logiciel (sdk) Windows media format 11. Les méthodes, structures, fonctions et énumérations individuelles incluent des spécifications de version dans leurs rubriques de référence.

le kit de développement logiciel (SDK) Windows Media Format 11 est pris en charge sur Windows Vista et Windows XP uniquement.

le kit de développement logiciel (SDK) Windows Media Format 9,5 est pris en charge uniquement sur Windows XP. si vous écrivez une application basée sur le kit de développement logiciel (SDK) Windows Media Format 9,5, votre application ne s’exécutera pas sur un autre système d’exploitation.

certaines fonctionnalités du kit de développement logiciel (SDK) de la série Windows Media 9 nécessitent également Windows XP. Les rubriques de référence individuelles relatives à ces fonctionnalités appellent cette exigence.
