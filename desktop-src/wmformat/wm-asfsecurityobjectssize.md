# <a name="wmasfsecurityobjectssize"></a>WM/ASFSecurityObjectsSize

L’attribut **WM/ASFSecurityObjectsSize** contient la taille, en octets, des objets DRM dans l’en-tête du fichier ASF.

## <a name="global-constant"></a>Constante globale

\_wszWMASFSecurityObjectsSize g

## <a name="data-type"></a>Type de données

**\_type WMT \_ QWord**

## <a name="remarks"></a>Remarques

Cet attribut est en lecture seule et s’applique à l’ensemble du fichier (flux 0).

Vous pouvez uniquement récupérer cet attribut à l’aide des méthodes de l’interface [**IWMHeaderInfo3**](/previous-versions/windows/desktop/api/wmsdkidl/nn-wmsdkidl-iwmheaderinfo3) à partir de l’objet de l’éditeur de métadonnées.
