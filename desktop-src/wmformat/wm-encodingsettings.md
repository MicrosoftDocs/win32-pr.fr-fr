# <a name="wmencodingsettings"></a>WM/EncodingSettings

L’attribut **WM/EncodingSettings** contient les paramètres utilisés pour encoder le contenu.

## <a name="global-constant"></a>Constante globale

\_wszWMEncodingSettings g

## <a name="data-type"></a>Type de données

**\_chaîne de type WMT \_**

## <a name="remarks"></a>Notes

Il n’existe pas de format standard pour l’indication des paramètres d’encodage. Toutefois, il est préférable de décrire les paramètres d’encodage en anglais plutôt que d’inclure un vidage des paramètres.
