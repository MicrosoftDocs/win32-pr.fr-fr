# <a name="advanced-systems-format-asf-files"></a>Fichiers ASF (Advanced Systems Format)

tous les objets Windows du kit de développement logiciel (SDK) de Format multimédia fonctionnent avec des fichiers conformes à la structure de fichiers ASF (Advanced Systems format). Ces fichiers sont appelés fichiers ASF dans cette documentation. Pour plus d’informations sur la structure des fichiers ASF, consultez [vue d’ensemble du format ASF](overview-of-the-asf-format.md).

parfois, le terme *Windows fichier multimédia* est utilisé pour décrire les fichiers créés à l’aide de ce kit de développement logiciel (SDK) ou d’autres technologies Windows Media. il n’est pas correct d’appeler un fichier ASF en Windows fichier multimédia, sauf s’il contient uniquement du contenu encodé à l’aide de Windows codecs multimédias. Pour plus d’informations, consultez les instructions relatives à l’utilisation des extensions de fichier. WMA et. wmv dans la section [instructions relatives](file-name-extension-guidelines.md) à l’extension de nom de fichier.
