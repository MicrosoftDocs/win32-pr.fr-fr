# <a name="advanced-systems-format-asf-files"></a>Fichiers ASF (Advanced Systems Format)

Tous les objets du kit de développement logiciel (SDK) de format Windows Media fonctionnent avec des fichiers conformes à la structure de fichiers ASF (Advanced Systems Format). Ces fichiers sont appelés fichiers ASF dans cette documentation. Pour plus d’informations sur la structure des fichiers ASF, consultez [vue d’ensemble du format ASF](overview-of-the-asf-format.md).

Parfois, le terme *fichier Windows Media* est utilisé pour décrire les fichiers créés à l’aide de ce kit de développement logiciel (SDK) ou d’autres technologies Windows Media. Il n’est pas correct d’appeler un fichier ASF dans un fichier Windows Media, sauf s’il contient uniquement du contenu encodé à l’aide de codecs Windows Media. Pour plus d’informations, consultez les instructions relatives à l’utilisation des extensions de fichier. WMA et. wmv dans la section [instructions relatives](file-name-extension-guidelines.md) à l’extension de nom de fichier.
