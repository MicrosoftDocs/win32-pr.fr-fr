# <a name="wmauthorurl"></a>WM/AuthorURL

L’attribut **WM/AuthorURL** contient l’adresse du site Web de l’auteur.

## <a name="global-constant"></a>Constante globale

\_wszWMAuthorURL g

## <a name="data-type"></a>Type de données

**\_chaîne de type WMT \_**

## <a name="remarks"></a>Remarques

L’adresse spécifiée par cet attribut doit correspondre à l’artiste ou à l’auteur dans un attribut [**auteur**](author.md) du fichier.
