# <a name="attributes-for-video-files"></a>Attributs pour les fichiers vidéo

Cette section répertorie les attributs couramment utilisés pour les fichiers vidéo. Nous vous recommandons de définir des attributs pour les fichiers en fonction de ces listes pour vous assurer que vos fichiers sont entièrement compatibles avec une grande variété d’applications de lecture. Les attributs de cette section sont répertoriés en trois catégories : principal, secondaire et tertiaire.

Les attributs principaux fournissent les informations les plus élémentaires sur un fichier. Si vous créez des fichiers vidéo pour la distribution, il s’agit de l’ensemble minimal d’attributs que vous devez utiliser.

Les attributs secondaires contiennent des métadonnées communes qui sont importantes mais pas universelles pour tous les fichiers vidéo.

Les attributs tertiaires doivent être inclus en fonction des besoins, mais ils ne sont pas essentiels à la description du fichier.

-   [**Titre**](title.md)
-   [**WM/ContentDistributor**](wm-contentdistributor.md)

-   [**WM/genre**](wm-genre.md)
-   [**WM/MediaClassPrimaryID**](wm-mediaprimaryid.md)
-   [**WM/MediaClassSecondaryID**](wm-mediasecondaryid.md)
