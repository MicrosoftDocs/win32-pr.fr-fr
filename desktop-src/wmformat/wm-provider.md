# <a name="wmprovider"></a>WM/Provider

L’attribut **WM/Provider** contient le nom du fournisseur de contenu de métadonnées. Les métadonnées sont souvent obtenues par le biais d’un service.

## <a name="global-constant"></a>Constante globale

\_wszWMProvider g

## <a name="data-type"></a>Type de données

**\_chaîne de type WMT \_**
