# <a name="for-more-information-windows-media-format-sdk"></a>Pour plus d’informations (Windows Media Format SDK)

-   pour obtenir les informations les plus récentes sur les problèmes connus avec le kit de développement logiciel (SDK) Windows Media Format, consultez les notes de publication publiées sur le [site Web MSDN Library](/documentation/?url=%2flibrary%2fdnwmt%2fhtml%2fwmfsdk11_b2_release_notes_rzxb.asp%3fframe%3dtrue).
-   pour obtenir des informations sur les fonctionnalités, les articles et les réponses aux questions fréquemment posées sur la diffusion multimédia en continu en général et les kits de développement logiciel (sdk) Windows media en particulier, consultez le centre de développement Windows media sur le [site Web de Microsoft](https://msdn.microsoft.com/windowsmedia/default.aspx).
-   pour plus d’informations sur l’obtention d’une licence pour le kit de développement logiciel (SDK) Windows media Format, consultez la page Windows media Licensing sur le [site Web de Microsoft](https://msdn.microsoft.com/library/ms867178.aspx).
-   pour plus d’informations sur les groupes de discussion, visitez la page Windows les groupes de discussion multimédia et les alias de messagerie sur le [site Web de Microsoft](https://windows.microsoft.com/windows/products/windows-media).
-   Pour obtenir des descriptions détaillées de bas niveau des objets de données qui composent des fichiers ASF, téléchargez la spécification ASF à partir du [site Web de Microsoft](https://download.microsoft.com/download/7/9/0/790fecaa-f64a-4a5e-a430-0bccdab3f1b4/ASF_Specification.doc).
-   pour plus d’informations sur les fonctionnalités, les articles, les meilleures pratiques et les réponses aux questions fréquemment posées sur la gestion des droits numériques (DRM) et Windows Media rights Manager, consultez les articles techniques sur la page de [Rights Management numérique](/documentation/?url=%2flibrary%2fdnwmt%2fhtml%2fdigitalrightsmanagement.asp) sur le site Web de Microsoft.
