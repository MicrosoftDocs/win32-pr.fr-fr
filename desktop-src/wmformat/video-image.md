# <a name="video-image"></a>Image vidéo

le codec Windows Media Video 9 Image v2 vous permet de créer des fichiers Windows Media Video dont le contenu se compose d’une série d’images fixes, telles que des photographies numériques. Les images contenues dans le flux de l’image vidéo peuvent être manipulées pour produire des panoramiques et des zooms, ainsi qu’un certain nombre de transitions. Les utilisateurs finaux peuvent utiliser cette fonctionnalité pour créer et partager des diaporamas simples qui peuvent également contenir une piste audio musicale ou une narration vocale. en raison de la compression supérieure du codec d’Image Windows Media Video, un fichier contenant plusieurs photographies est toujours assez petit pour s’envoyer facilement par courrier électronique sur Internet.

Un flux d’image vidéo a son propre type de média et une structure spécialisée qui définit les différentes transitions possibles lors du passage d’une image à la suivante. Vous pouvez effectuer un panoramique et un zoom sur l’affichage des images tout en les fusionnant ensemble. le fichier résultant peut ensuite être lu comme tout autre Windows fichier vidéo multimédia.
