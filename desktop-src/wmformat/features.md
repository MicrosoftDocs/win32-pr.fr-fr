# <a name="features"></a>Fonctionnalités

Le kit de développement logiciel (SDK) du format Windows Media offre de nombreuses options pour traiter les fichiers ASF et d’autres fichiers à l’aide du format systèmes avancés. La plupart de ces fonctionnalités ne sont pas facilement classées dans l’une des principales fonctions du kit de développement logiciel (SDK). Cette section décrit les fonctionnalités du kit de développement logiciel (SDK) du format Windows Media du point de vue de la programmation. Chaque rubrique de fonctionnalité offre une vue d’ensemble de la fonctionnalité et fournit des liens vers des rubriques connexes de cette documentation.



| Section                                                                      | Description                                                                                                                                                      |
|------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Fonctionnalités des fichiers ASF](asf-file-features.md)                                   | Décrit les fonctionnalités inhérentes au format ASF (Advanced Systems Format). Ces fonctionnalités sont généralement implémentées par le biais du profil utilisé pour créer un fichier. |
| [Fonctionnalités du codec](codec-features.md)                                         | Décrit les fonctionnalités des codecs Windows Media qui sont directement pris en charge par les objets du kit de développement logiciel (SDK) Windows Media format                                    |
| [Fonctionnalités d’écriture de fichier](file-writing-features.md)                           | Décrit les fonctionnalités qui prennent en charge l’écriture de fichier.                                                                                                                |
| [Fonctionnalités de lecture de fichier](file-reading-features.md)                           | Décrit les fonctionnalités qui prennent en charge la lecture de fichiers.                                                                                                                |
| [Fonctionnalités de métadonnées](metadata-features.md)                                   | Décrit les fonctionnalités qui prennent en charge les métadonnées, qui sont des informations sur le fichier et son contenu.                                                               |
| [Fonctionnalités de Rights Management numérique](digital-rights-management-features.md) | Décrit la fonctionnalité de gestion des droits numériques intégrée au kit de développement logiciel (SDK) du format Windows Media.                                                      |
