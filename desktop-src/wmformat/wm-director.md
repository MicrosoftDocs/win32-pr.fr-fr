# <a name="wmdirector"></a>WM/Director

L’attribut **WM/Director** contient le nom du directeur.

## <a name="global-constant"></a>Constante globale

\_wszWMDirector g

## <a name="data-type"></a>Type de données

**\_chaîne de type WMT \_**

## <a name="remarks"></a>Remarques

En général, les fichiers vidéo possèdent des directeurs, les fichiers audio sont plus susceptibles d’utiliser [**WM/Producer**](wm-producer.md).
