# <a name="drm_actionallowed_playlistburn"></a>\_ACTIONALLOWED DRM \_ PlaylistBurn

L’attribut **DRM \_ ActionAllowed \_ PlaylistBurn** indique si le contenu peut être copié sur CD dans le cadre d’une sélection.

## <a name="global-constant"></a>Constante globale

g \_ wszWMDRM \_ ActionAllowed \_ PlaylistBurn

## <a name="data-type"></a>Type de données

**\_type WMT \_ bool**

## <a name="remarks"></a>Remarques

Il s’agit d’une propriété en lecture seule qui est récupérée à l’aide de [**IWMDRMReader :: GetDRMProperty**](/previous-versions/windows/desktop/api/Wmsdkidl/nf-wmsdkidl-iwmdrmreader-getdrmproperty).
