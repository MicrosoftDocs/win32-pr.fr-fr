# <a name="numberofframes"></a>NumberOfFrames

L’attribut **NumberOfFrames** contient le nombre de frames dans un flux vidéo.

## <a name="global-constant"></a>Constante globale

\_wszWMNumberOfFrames g

## <a name="data-type"></a>Type de données

**\_type WMT \_ QWord**

## <a name="remarks"></a>Remarques

Le writer ajoute cette valeur pour un flux vidéo au moment de l’encodage. **NumberOfFrames** n’est pas en lecture seule. Toutefois, vous ne devez modifier la valeur que si vous modifiez le fichier, en modifiant le nombre de frames dans un flux vidéo.
