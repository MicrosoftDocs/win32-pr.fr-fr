# <a name="error-codes-windows-media-format-11-sdk"></a>Codes d’erreur (kit de développement logiciel (SDK) Windows Media format 11)

Microsoft a défini les codes d’erreur et de réussite suivants pour une utilisation dans le kit de développement logiciel (SDK) Windows Media format. Ces codes sont un sous-ensemble d’une collection plus grande qui peut être affichée dans les fichiers d’en-tête asferr. h (codes ASF) et nserror. h (autres codes) inclus dans le kit de développement logiciel (SDK) du format Windows Media. Des codes d’erreur courants supplémentaires peuvent être affichés dans Winerror. h.
