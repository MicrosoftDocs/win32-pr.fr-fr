# <a name="copyrighturl"></a>CopyrightURL

L’attribut **CopyrightURL** contient l’adresse Web d’une page de Copyright associée à la personnalisation.

## <a name="global-constant"></a>Constante globale

\_wszWMCopyrightURL g

## <a name="data-type"></a>Type de données

**\_chaîne de type WMT \_**

## <a name="remarks"></a>Notes

Cet attribut s’applique à l’ensemble du fichier.
