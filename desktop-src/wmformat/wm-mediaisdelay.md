# <a name="wmmediaisdelay"></a>WM/MediaIsDelay

L’attribut **WM/MediaIsDelay** indique si le contenu a été diffusé comme un délai de bande.

## <a name="global-constant"></a>Constante globale

\_wszWMMediaIsDelay g

## <a name="data-type"></a>Type de données

**\_type WMT \_ bool**

## <a name="remarks"></a>Notes

Cet attribut est utilisé pour la télévision de diffusion.
