# <a name="wmmodifiedby"></a>WM/ModifiedBy

L’attribut **WM/ModifiedBy** contient le nom d’une personne ou d’un groupe qui a modifié le fichier.

## <a name="global-constant"></a>Constante globale

\_wszWMModifiedBy g

## <a name="data-type"></a>Type de données

**\_chaîne de type WMT \_**
