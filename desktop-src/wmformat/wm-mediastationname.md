# <a name="wmmediastationname"></a>WM/MediaStationName

L’attribut **WM/MediaStationName** contient le nom de la station qui diffuse le contenu.

## <a name="global-constant"></a>Constante globale

\_wszWMMediaStationName g

## <a name="data-type"></a>Type de données

**\_chaîne de type WMT \_**

## <a name="remarks"></a>Notes

Cet attribut est utilisé pour la télévision de diffusion.
