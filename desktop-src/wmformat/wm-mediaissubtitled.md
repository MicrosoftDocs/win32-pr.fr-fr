# <a name="wmmediaissubtitled"></a>WM/MediaIsSubtitled

L’attribut **WM/MediaIsSubtitled** indique si le contenu comprend des sous-titres.

## <a name="global-constant"></a>Constante globale

\_wszWMMediaIsSubtitled g

## <a name="data-type"></a>Type de données

**\_type WMT \_ bool**

## <a name="remarks"></a>Notes

Cet attribut est utilisé pour la télévision de diffusion.
