# <a name="wminitialkey"></a>WM/InitialKey

L’attribut **WM/InitialKey** contient la clé initiale de la musique dans le fichier.

## <a name="global-constant"></a>Constante globale

\_wszWMInitialKey g

## <a name="data-type"></a>Type de données

**\_chaîne de type WMT \_**

| Type de fichier | Valeur d'exemple |

| Audio     | « Un mineur »     |
