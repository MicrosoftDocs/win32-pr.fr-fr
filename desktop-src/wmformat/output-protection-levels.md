# <a name="output-protection-levels"></a>Niveaux de protection de sortie

Les niveaux de protection de sortie (OPLs) sont des notations numériques associées à différentes technologies qui reçoivent du contenu multimédia numérique. Le niveau de prise en charge d’une technologie dépend de sa sécurité. le système OPL, introduit dans Windows Media DRM 10, permet de créer des licences avec davantage de flexibilité que dans les versions précédentes. Vous pouvez spécifier la OPLs minimale requise pour la lecture et la copie. En outre, vous pouvez spécifier des exceptions à la OPL minimale, soit pour autoriser une technologie non évaluée suffisamment, soit pour interdire une technologie avec un OPL qui dépasse la valeur minimale.

En spécifiant des restrictions aux licences à l’aide de OPLs, le propriétaire d’un contenu doit uniquement utiliser deux actions (copie et lecture), où les versions précédentes avaient des actions distinctes définies pour les différents types d’appareils pris en charge pour la copie (SDMI, non SDMI et CD audio d’un livre rouge).

**Remarque** DRM n’est pas pris en charge par la version x64 de ce kit de développement logiciel (SDK).
