# <a name="privacy-statement-windows-media-format-11-sdk"></a>Déclaration de confidentialité (kit de développement logiciel (SDK) Windows Media format 11)

Les développeurs qui créent des applications sur ce logiciel doivent communiquer clairement à leurs utilisateurs finaux la manière dont la confidentialité des utilisateurs finaux pourrait être affectée.
