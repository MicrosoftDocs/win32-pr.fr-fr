# <a name="bannerimagedata"></a>BannerImageData

L’attribut **BannerImageData** contient une image de bannière à des fins de personnalisation. Le type d’image que contient cet attribut est décrit par l’attribut [**BannerImageType**](bannerimagetype.md) .

## <a name="global-constant"></a>Constante globale

\_wszWMBannerImageData g

## <a name="data-type"></a>Type de données

**\_binaire de type WMT \_**

## <a name="remarks"></a>Notes

L’image stockée dans cet attribut peut être au format bitmap, JPEG ou GIF.
