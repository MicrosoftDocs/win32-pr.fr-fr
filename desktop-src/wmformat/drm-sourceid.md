# <a name="drm_sourceid"></a>\_SOURCEID DRM

L' **attribut \_ SourceId DRM** contient un identificateur qui associe le contenu protégé à une source.

## <a name="global-constant"></a>Constante globale

\_wszWMDRM \_ SourceId

## <a name="data-type"></a>Type de données

**\_valeur DWORD de type WMT \_**

## <a name="remarks"></a>Notes

Les identificateurs sources sont utilisés pour différencier le contenu protégé d’une façon non standard du contenu protégé à l’aide de la DRM Windows Media standard.
