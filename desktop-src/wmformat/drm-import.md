# <a name="drm-import"></a>Importation DRM

Les sections suivantes expliquent comment convertir du contenu multimédia d’un système de protection de contenu (CPS) tiers en Windows Media DRM. Ce processus est connu sous le nom d’importation DRM et se compose essentiellement des étapes suivantes :

1.  Création du fichier ASF.
2.  Création de la licence.
3.  Vous pouvez éventuellement supprimer la licence.

L’importation DRM est expliquée dans les sections suivantes.



| Section                                                                              | Description                                                                          |
|--------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------|
| [Importation de la licence et du matériel de clé](importing-license-and-key-material.md)         | Décrit comment émettre des licences localement et les importer dans Windows Media DRM.      |
| [Vérification de la révocation des certificats](checking-certificate-revocation.md)               | Décrit comment vérifier la révocation de certificats.                                       |
| [Génération d’une licence XMR](building-an-xmr-license.md)                               | Décrit comment créer une licence XMR et la passer au sous-système DRM.             |
| [Création et initialisation d’un enregistreur DRM](creating-and-initializing-a-drm-writer.md) | Décrit comment initialiser un objet Writer ASF pour préparer l’importation DRM.          |
| [Chiffrement et importation d’exemples de médias](encrypting-and-importing-media-samples.md) | Décrit comment chiffrer et importer des exemples de médias individuels dans Windows Media DRM. |
| [Suppression de la licence](license-deletion.md)                                             | Décrit comment supprimer des licences créées localement.                                    |
