# <a name="watermark-category-types"></a>Types de catégorie de filigrane

Vous pouvez utiliser la fonction d’énumération DMO **DMOEnum** pour obtenir des informations sur les DMOs de filigranes qui sont installés sur un ordinateur. Le kit de développement logiciel (SDK) du format Windows Media définit des GUID qui identifient les catégories de filigranes DMO. Pour plus d’informations sur **DMOEnum**, consultez la documentation DIRECTSHOW sur MSDN.

Le tableau suivant répertorie les identificateurs pour les catégories de filigrane.



| Constante de catégorie DMO de filigrane | GUID                                 |
|------------------------------------|--------------------------------------|
| \_ \_ filigrane audio DMOCATEGORY \_ WMT | 65221c5a-fa75-4b39-b50c-06c336b6a3ef |
| \_filigrane de \_ vidéo \_ DMOCATEGORY WMT | 187cc922-8efc-4404-9daf-63f4830df1bc |
