# <a name="watermark-category-types"></a>Types de catégorie de filigrane

vous pouvez utiliser la fonction d’énumération DMO, **DMOEnum**, pour obtenir des informations sur les DMOs de filigranes installés sur un ordinateur. le kit de développement logiciel (SDK) Windows Media Format définit des guid qui identifient les catégories de filigrane DMO. pour plus d’informations sur **DMOEnum**, consultez la documentation DirectShow sur MSDN.

Le tableau suivant répertorie les identificateurs pour les catégories de filigrane.



| constante de catégorie de DMO de filigrane | GUID                                 |
|------------------------------------|--------------------------------------|
| \_ \_ filigrane audio DMOCATEGORY \_ WMT | 65221c5a-fa75-4b39-b50c-06c336b6a3ef |
| \_filigrane de \_ vidéo \_ DMOCATEGORY WMT | 187cc922-8efc-4404-9daf-63f4830df1bc |
