# <a name="wmtracknumber"></a>WM/TrackNumber

L’attribut **WM/TrackNumber** contient le numéro de suivi du contenu. Cet attribut est de base 1.

## <a name="global-constant"></a>Constante globale

\_wszWMTrackNumber g

## <a name="data-type"></a>Type de données

**\_chaîne de type WMT \_**

## <a name="remarks"></a>Notes

De nombreuses applications existantes écrivent la valeur de **WM/TrackNumber** comme **DWORD**. Si vous créez une application qui lit des fichiers provenant de sources inconnues, vous devez inclure du code pour gérer les valeurs de type chaîne et **DWORD** .
