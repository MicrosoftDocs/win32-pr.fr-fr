# <a name="rating"></a>Rating

L’attribut évaluation contient l’évaluation du contenu.

## <a name="global-constant"></a>Constante globale

\_wszWMRating g

## <a name="data-type"></a>Type de données

**\_chaîne de type WMT \_**

## <a name="remarks"></a>Notes

Il s’agit d’un attribut de niveau fichier.

Les implémentations individuelles des fichiers existants utilisent des notations et des notations différentes. Pour cette raison, vous devez éviter d’utiliser cet attribut. Il n’existe aucune normalisation à partir de laquelle développer des interfaces utilisateur cohérentes.
