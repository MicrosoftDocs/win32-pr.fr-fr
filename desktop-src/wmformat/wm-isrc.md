# <a name="wmisrc"></a>WM/ISRC

L’attribut **WM/ISRC** contient le code d’enregistrement standard international (ISRC).

## <a name="global-constant"></a>Constante globale

\_wszWMISRC g

## <a name="data-type"></a>Type de données

**\_chaîne de type WMT \_**

## <a name="remarks"></a>Notes

Le ISRC est utilisé dans le secteur de l’enregistrement pour identifier un enregistrement. Elle est indépendante de MCDI, qui identifie un DICS compact par sa table des matières.
