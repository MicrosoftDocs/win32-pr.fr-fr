# <a name="file-streams"></a>Flux de fichiers

Un flux de fichier contient un ou plusieurs fichiers de données. Lorsque vous transférez des informations de fichier sous forme de flux, vous devez joindre le nom de fichier à chaque exemple dans le flux. Pour plus d’informations sur l’attachement d’informations connexes à des exemples, consultez [extensions d’unité de données](data-unit-extensions.md).
