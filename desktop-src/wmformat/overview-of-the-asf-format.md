<span data-ttu-id="07cd8-107">ASF (Advanced Systems Format) est un format de fichier extensible conçu principalement pour le stockage et la diffusion de flux multimédias numériques synchronisés et leur transmission sur des réseaux.</span><span class="sxs-lookup"><span data-stu-id="07cd8-107">The Advanced Systems Format (ASF) is an extensible file format designed primarily for storing and playing synchronized digital media streams and transmitting them over networks.</span></span> <span data-ttu-id="07cd8-108">ASF est le format de conteneur pour Windows Media Audio et le contenu basé sur Windows Media Video.</span><span class="sxs-lookup"><span data-stu-id="07cd8-108">ASF is the container format for Windows Media Audio and Windows Media Video-based content.</span></span> <span data-ttu-id="07cd8-109">L’extension WMA ou WMV est utilisée pour spécifier un fichier ASF qui contient le contenu encodé avec les codecs Windows Media Audio et/ou Windows Media Video.</span><span class="sxs-lookup"><span data-stu-id="07cd8-109">The extension wma or wmv is used to specify an ASF file that contains content encoded with the Windows Media Audio and/or Windows Media Video codecs.</span></span> <span data-ttu-id="07cd8-110">Le kit de développement logiciel (SDK) du format Windows Media peut être utilisé pour créer et lire des fichiers Windows Media, ainsi que des fichiers ASF qui contiennent d’autres types de données compressées ou non compressées.</span><span class="sxs-lookup"><span data-stu-id="07cd8-110">The Windows Media Format SDK can be used to create and read Windows Media files, as well as ASF files that contain other types of compressed or uncompressed data.</span></span>

<span data-ttu-id="07cd8-111">Cette section fournit une description générale du format ASF en tant qu’informations générales.</span><span class="sxs-lookup"><span data-stu-id="07cd8-111">This section provides a general description of the ASF format as background information.</span></span> <span data-ttu-id="07cd8-112">Étant donné que les objets lecteur et Writer gèrent toutes les tâches de mise en forme et d’analyse de fichier de bas niveau, il n’est pas nécessaire d’avoir une compréhension détaillée de ASF avant d’utiliser ce kit de développement logiciel (SDK) pour créer des fichiers ASF.</span><span class="sxs-lookup"><span data-stu-id="07cd8-112">Because the reader and writer objects handle all low-level file parsing and formatting tasks, it is not necessary to have a detailed understanding of ASF before using this SDK to create ASF files.</span></span> <span data-ttu-id="07cd8-113">La spécification ASF complète se trouve sur le [site Web de Microsoft](https://download.microsoft.com/download/7/9/0/790fecaa-f64a-4a5e-a430-0bccdab3f1b4/ASF_Specification.doc).</span><span class="sxs-lookup"><span data-stu-id="07cd8-113">The complete ASF specification can be found on the [Microsoft Web site](https://download.microsoft.com/download/7/9/0/790fecaa-f64a-4a5e-a430-0bccdab3f1b4/ASF_Specification.doc).</span></span>

<span data-ttu-id="07cd8-114">Les principaux objectifs du format ASF sont les suivants :</span><span class="sxs-lookup"><span data-stu-id="07cd8-114">The primary goals of the ASF format are:</span></span>

-   <span data-ttu-id="07cd8-115">Pour prendre en charge une lecture efficace à partir de serveurs multimédias, de serveurs HTTP et de périphériques de stockage locaux.</span><span class="sxs-lookup"><span data-stu-id="07cd8-115">To support efficient playback from media servers, HTTP servers, and local storage devices.</span></span>
-   <span data-ttu-id="07cd8-116">Pour prendre en charge des types de médias évolutifs, tels que des données audio et vidéo.</span><span class="sxs-lookup"><span data-stu-id="07cd8-116">To support scalable media types such as audio and video.</span></span>
-   <span data-ttu-id="07cd8-117">Pour permettre à une composition multimédia unique d’être présentée sur un large éventail de bandes passantes.</span><span class="sxs-lookup"><span data-stu-id="07cd8-117">To permit a single multimedia composition to be presented over a wide range of bandwidths.</span></span>
-   <span data-ttu-id="07cd8-118">Pour permettre le contrôle de la création de relations de flux de média, en particulier dans les scénarios de bande passante limitée.</span><span class="sxs-lookup"><span data-stu-id="07cd8-118">To allow authoring control over media stream relationships, especially in constrained-bandwidth scenarios.</span></span>
-   <span data-ttu-id="07cd8-119">Être indépendant du système de composition multimédia, du système d’exploitation de l’ordinateur ou du protocole de communication des données.</span><span class="sxs-lookup"><span data-stu-id="07cd8-119">To be independent of any particular multimedia composition system, computer operating system, or data communications protocol.</span></span>

<span data-ttu-id="07cd8-120">Un fichier ASF peut contenir plusieurs flux indépendants ou dépendants, y compris plusieurs flux audio pour le son multicanaux, ou des flux vidéo à vitesses de transmission multiples pouvant être transmis sur des bandes passantes différentes.</span><span class="sxs-lookup"><span data-stu-id="07cd8-120">An ASF file can contain multiple independent or dependent streams, including multiple audio streams for multichannel audio, or multiple bit rate video streams suitable for transmission over different bandwidths.</span></span> <span data-ttu-id="07cd8-121">Les flux peuvent être dans n’importe quel format compressé ou non compressé ; <span data-ttu-id="07cd8-125">Un fichier ASF est organisé en sections appelées « objets ».</span><span class="sxs-lookup"><span data-stu-id="07cd8-125">An ASF file is organized into sections called "objects."</span></span> <span data-ttu-id="07cd8-126">Il existe trois objets de niveau supérieur, un objet d’en-tête et un objet de données (les deux obligatoires), ainsi qu’un objet d’index d’erreur et les codecs utilisés.</span><span class="sxs-lookup"><span data-stu-id="07cd8-127">The Header object contains general information about the file, such as file size, number of streams, error correction methods, and codecs used.</span></span> <span data-ttu-id="07cd8-128">Les métadonnées sont également stockées ici.</span><span class="sxs-lookup"><span data-stu-id="07cd8-128">Metadata is also stored here.</span></span> <span data-ttu-id="07cd8-129">L’objet d’en-tête est le seul objet de niveau supérieur qui peut contenir d’autres objets.</span><span class="sxs-lookup"><span data-stu-id="07cd8-129">The Header object is the only top level object that can contain other objects.</span></span> <span data-ttu-id="07cd8-130">L’objet de données contient les données de flux, <span data-ttu-id="07cd8-133">Chaque objet de niveau supérieur ou de niveau inférieur commence par un identificateur global unique (GUID) et une valeur de taille.</span><span class="sxs-lookup"><span data-stu-id="07cd8-133">Each top-level or lower-level object begins with a globally unique identifier (GUID) and a size value.</span></span> <span data-ttu-id="07cd8-134">Ces numéros permettent au lecteur de fichier d’analyser les informations à des emplacements appropriés dans des objets identifiables.</span><span class="sxs-lookup"><span data-stu-id="07cd8-134">These numbers allow the file reader to parse the information at appropriate places into identifiable objects.</span></span> <span data-ttu-id="07cd8-135">En raison de ces GUID, les objets de niveau inférieur peuvent être envoyés dans n’importe quel ordre et sont toujours reconnus.</span><span class="sxs-lookup"><span data-stu-id="07cd8-135">Because of these GUIDs, lower-level objects can be sent in any order and still be recognized.</span></span> <span data-ttu-id="07cd8-136">Le format ASF est conçu pour surmonter la réception de données inexactes.</span><span class="sxs-lookup"><span data-stu-id="07cd8-136">The ASF format is designed to overcome inaccurate data reception.</span></span> <span data-ttu-id="07cd8-137">Un fichier ASF partiellement téléchargé peut toujours être lu, à condition qu’il contienne l’objet d’en-tête et au moins un objet de données.</span><span class="sxs-lookup"><span data-stu-id="07cd8-137">A partially downloaded ASF file can still be read, as long as it contains the Header object and at least one Data object.</span></span>
