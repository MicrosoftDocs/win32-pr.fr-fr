# <a name="wmradiostationowner"></a>WM/RadioStationOwner

L’attribut **WM/RadioStationOwner** contient le nom du propriétaire de la station de radio associée au contenu.

## <a name="global-constant"></a>Constante globale

\_wszWMRadioStationOwner g
