# <a name="wmuniquefileidentifier"></a>WM/UniqueFileIdentifier

L’attribut **WM/UniqueFileIdentifier** contient un identificateur de fichier unique pour le contenu.

## <a name="global-constant"></a>Constante globale

\_wszWMUniqueFileIdentifier g

## <a name="data-type"></a>Type de données

**\_chaîne de type WM \_**

## <a name="remarks"></a>Notes

L’identificateur de fichier unique est une chaîne générique qui peut être utilisée par les applications et les services pour identifier le fichier de façon unique. La chaîne contient des valeurs arbitraires délimitées par des points-virgules. Vous ne devez jamais effacer cet attribut. Vous pouvez ajouter des valeurs et supprimer vos propres valeurs, mais toutes les autres doivent rester inchangées.
