# <a name="wmplaylistdelay"></a>WM/PlaylistDelay

L’attribut **WM/PlaylistDelay** contient le nombre de millisecondes qui doit précéder la lecture du fichier dans une sélection.

## <a name="global-constant"></a>Constante globale

\_wszWMPlaylistDelay g

## <a name="data-type"></a>Type de données

**\_chaîne de type WMT \_**

## <a name="remarks"></a>Remarques

Tous les lecteurs multimédias ne respectent pas le délai de sélection.
