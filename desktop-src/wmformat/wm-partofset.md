# <a name="wmpartofset"></a>WM/PartOfSet

L’attribut **WM/PartOfSet** contient le numéro de référence et le nombre total de parties dans le jeu auquel le fichier appartient. Les deux valeurs sont séparées par une barre oblique.

## <a name="global-constant"></a>Constante globale

\_wszWMPartOfSet g

## <a name="data-type"></a>Type de données

**\_chaîne de type WMT \_**

|-----------|---------------|
| Tous       | « 2/3 »         |
