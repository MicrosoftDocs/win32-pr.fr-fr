# <a name="project-considerations"></a>Considérations relatives aux projets

Vous devez vous assurer que l’utilisateur final peut exécuter correctement les applications que vous créez avec le kit de développement logiciel (SDK) de format Windows Media. Cette section décrit deux choses que vous devez prendre en compte lors de la distribution d’applications, de l’extension de nom de fichier et de la redistribution de logiciels.

Vous devez utiliser les extensions de nom de fichier appropriées pour tous les fichiers créés avec vos applications. L’utilisation des extensions de nom de fichier appropriées permet aux utilisateurs finaux de reconnaître plus facilement les fichiers ASF et évite toute confusion lors du décodage de fichiers.

Vous devez fournir tous les redistribuables requis pour exécuter vos applications. Sans les redistribuables corrects, les utilisateurs ne seront pas en mesure d’utiliser vos applications.

Les sections suivantes décrivent en détail les extensions de nom de fichier et la redistribution de logiciels.

| [Redistribution de logiciels](software-redistribution.md)                       | Décrit les redistribuables du kit de développement logiciel (SDK) du format Windows Media et explique comment les inclure dans vos applications. |
