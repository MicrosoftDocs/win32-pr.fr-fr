# <a name="wmpeakbitrate"></a>WM/PeakBitrate

L’attribut **WM/PeakBitrate** contient la vitesse de transmission maximale d’un flux VBR (variable bit rate) en bits par seconde.

## <a name="global-constant"></a>Constante globale

\_wszWMPeakBitrate g

## <a name="data-type"></a>Type de données

**\_valeur DWORD de type WMT \_**
