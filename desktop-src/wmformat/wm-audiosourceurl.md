# <a name="wmaudiosourceurl"></a>WM/AudioSourceURL

L’attribut **WM/AudioSourceURL** contient l’adresse d’une page Web officielle du média source. Par exemple, une chanson d’une bande son peut comporter un lien vers la page officielle de la vidéo.

## <a name="global-constant"></a>Constante globale

\_wszWMAudioSourceURL g

## <a name="data-type"></a>Type de données

**\_chaîne de type WMT \_**
