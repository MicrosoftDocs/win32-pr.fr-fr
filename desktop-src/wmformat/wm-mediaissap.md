# <a name="wmmediaissap"></a>WM/MediaIsSAP

L’attribut **WM/MediaIsSAP** indique si le contenu comprend un deuxième programme audio (SAP).

## <a name="global-constant"></a>Constante globale

\_wszWMMediaIsSAP g

## <a name="data-type"></a>Type de données

**\_type WMT \_ bool**

## <a name="remarks"></a>Remarques

Cet attribut est utilisé pour la télévision de diffusion.

En règle générale, SAP identifie les programmes de télévision qui incluent une autre piste audio dans une autre langue.
