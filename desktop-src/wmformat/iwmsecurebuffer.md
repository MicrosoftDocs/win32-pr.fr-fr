# <a name="iwmsecurebuffer-interface"></a>Interface IWMSecureBuffer

L’interface **IWMSecureBuffer** chiffre et déchiffre les pointeurs de données.

## <a name="members"></a>Membres

L’interface **IWMSecureBuffer** hérite de l’interface [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) . **IWMSecureBuffer** a également les types de membres suivants :

-   [Méthodes](#methods)

### <a name="methods"></a>Méthodes

L’interface **IWMSecureBuffer** possède ces méthodes.



| Méthode                                     | Description                         |
|:-------------------------------------------|:------------------------------------|
| [**Déchiffrer**](iwmsecurebuffer-decrypt.md) | Déchiffre un pointeur de données.<br/> |
| [**Encrypt (Chiffrer)**](iwmsecurebuffer-encrypt.md) | Chiffre un pointeur de données.<br/> |
