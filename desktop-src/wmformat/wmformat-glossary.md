# <a name="wmformat-glossary"></a>Glossaire WMFormat

Pour rechercher un terme dans le glossaire, cliquez sur la lettre de l’alphabet qui correspond à la première lettre du terme que vous souhaitez rechercher.

Vous pouvez également lire les termes du Glossaire dans le texte de l’aide en cliquant sur les liens du Glossaire souligné. Une fois que vous avez cliqué sur un lien de Glossaire, le terme et la définition du Glossaire s’affichent dans une fenêtre indépendante. Pour fermer la fenêtre, cliquez n’importe où sur l’écran.

<dl> <dt>

<span id="wmformat.bandwidth"></span><span id="WMFORMAT.BANDWIDTH"></span>**bande passante**
</dt> <dd>

Capacité de transfert de données d’un support de transmission.

</dd> <dt>

<span id="wmformat.bit_rate"></span><span id="WMFORMAT.BIT_RATE"></span>**vitesse de transmission**
</dt> <dd>

Nombre de bits transférés par unité de temps, généralement exprimé en bits par seconde.

</dd> <dt>

<span id="wmformat.broadband"></span><span id="WMFORMAT.BROADBAND"></span>**bande passante**
</dt> <dd>

Un support de transmission conçu pour les transferts de données à haut débit sur de longues distances. Les services Cable modem et DSL sont des exemples de réseaux à large bande.

</dd> <dt>

<span id="wmformat.cleanpoint"></span><span id="WMFORMAT.CLEANPOINT"></span>**Cleanpoint**
</dt> <dd>

Exemple compressé qui ne dépend d’aucun autre échantillon pour la décompression. Il n’est valide que pour effectuer une recherche sur un Cleanpoint.

</dd> <dt>

<span id="wmformat.clear_content"></span><span id="WMFORMAT.CLEAR_CONTENT"></span>**effacer le contenu**
</dt> <dd>

fichier multimédia Windows qui n’est pas chiffré.

</dd> <dt>

<span id="wmformat.codec"></span><span id="WMFORMAT.CODEC"></span>**codec**
</dt> <dd>

Abréviation de compresseur/décompresseur. Logiciel ou matériel utilisé pour compresser et décompresser des médias numériques.

</dd> <dt>

<span id="wmformat.content"></span><span id="WMFORMAT.CONTENT"></span>**contenu**
</dt> <dd>

Audio, vidéo, images, texte ou toute autre information contenue dans un fichier multimédia ou un flux numérique.

</dd> <dt>

<span id="wmformat.content_header"></span><span id="WMFORMAT.CONTENT_HEADER"></span>**en-tête de contenu**
</dt> <dd>

partie de la structure de fichiers d’un fichier multimédia Windows qui contient les informations nécessaires à un ordinateur client pour décompresser et restituer les données de contenu. Dans un fichier empaqueté, un en-tête de contenu supplémentaire existe et contient l’ID de clé, l’ID de contenu et l’URL d’acquisition de licence. Cet en-tête de contenu peut également inclure un numéro de version d’individualisation requis et des attributs définis par le fournisseur de contenu.

</dd> <dt>

<span id="wmformat.content_owner"></span><span id="WMFORMAT.CONTENT_OWNER"></span>**propriétaire du contenu**
</dt> <dd>

personne ou organisation qui crée Windows fichiers multimédias, par exemple, une étiquette d’enregistrement, un studio de cinéma ou un artiste.

</dd> <dt>

<span id="wmformat.cookie"></span><span id="WMFORMAT.COOKIE"></span>**cookie**
</dt> <dd>

Petit fichier de données qui est stocké sur l’ordinateur local d’un utilisateur et qui contient des informations sur l’utilisateur qui s’appliquent à un site Web, telles que les préférences de l’utilisateur.

</dd> <dt>

<span id="wmformat.custom_profile"></span><span id="WMFORMAT.CUSTOM_PROFILE"></span>**profil personnalisé**
</dt> <dd>

Groupe de paramètres, défini par un utilisateur final, pour la configuration de la sortie de l’encodeur.

</dd> <dt>

<span id="wmformat.decrypt"></span><span id="WMFORMAT.DECRYPT"></span>**déchiffrer**
</dt> <dd>

Pour reconvertir le contenu chiffré dans sa forme d’origine.

</dd> <dt>

<span id="wmformat.deinterlace"></span><span id="WMFORMAT.DEINTERLACE"></span>**désentrelacer**
</dt> <dd>

Pour combiner les champs entrelacés dans une image vidéo afin que, lors de la lecture, les lignes de la trame vidéo soient peintes de manière séquentielle.

</dd> <dt>

<span id="wmformat.delta_frame"></span><span id="WMFORMAT.DELTA_FRAME"></span>**Frame Delta**
</dt> <dd>

Image vidéo qui contient uniquement les modifications du frame précédent. En revanche, une image clé contient toutes les données nécessaires pour construire ce frame.

</dd> <dt>

<span id="wmformat.digital_rights_management__drm_"></span><span id="WMFORMAT.DIGITAL_RIGHTS_MANAGEMENT__DRM_"></span>**gestion des droits numériques (DRM)**
</dt> <dd>

Toute technologie utilisée pour protéger les intérêts des propriétaires de contenu numérique et de services (tels que les propriétaires de droits d’auteur). En règle générale, les destinataires ou les utilisateurs autorisés doivent acquérir une licence pour pouvoir utiliser le contenu.

</dd> <dt>

<span id="wmformat.digital_rights_management__drm__component"></span><span id="WMFORMAT.DIGITAL_RIGHTS_MANAGEMENT__DRM__COMPONENT"></span>**composant de gestion des droits numériques (DRM)**
</dt> <dd>

Composant d’un lecteur qui gère toutes les fonctions de gestion des droits numériques, tels que le déchiffrement des fichiers empaquetés ou le lancement d’une acquisition de licence.

</dd> <dt>

<span id="wmformat.directx_media_object__dmo_"></span><span id="WMFORMAT.DIRECTX_MEDIA_OBJECT__DMO_"></span>**Objet DirectX Media (DMO)**
</dt> <dd>

Objet COM qui traite les flux de données multimédias à partir des mémoires tampons allouées par le client.

</dd> <dt>

<span id="wmformat.dmo"></span><span id="WMFORMAT.DMO"></span>**DMO**
</dt> <dd>

Consultez la définition de : [ *objet de média DirectX (DMO)*](/windows)

</dd> <dt>

<span id="wmformat.drm"></span><span id="WMFORMAT.DRM"></span>**DRM**
</dt> <dd>

Voir la définition de : [ *gestion des droits numériques (DRM)*](/windows)

</dd> <dt>

<span id="wmformat.encode"></span><span id="WMFORMAT.ENCODE"></span>**coder**
</dt> <dd>

Pour convertir du contenu audio et vidéo dans un format numérique spécifié, il s’agit généralement d’une technologie de compression ou d’une technologie de chiffrement.

</dd> <dt>

<span id="wmformat.encrypt"></span><span id="WMFORMAT.ENCRYPT"></span>**chiffrer**
</dt> <dd>

Pour déguiser le contenu par programmation pour masquer sa substance.

</dd> <dt>

<span id="wmformat.frame"></span><span id="WMFORMAT.FRAME"></span>**Frame**
</dt> <dd>

Une des nombreuses images séquentielles qui composent de la vidéo.

</dd> <dt>

<span id="wmformat.frame_rate"></span><span id="WMFORMAT.FRAME_RATE"></span>**fréquence d’images**
</dt> <dd>

Nombre de trames vidéo affichées par seconde. Des fréquences d’images supérieures produisent généralement un mouvement plus lisse dans l’image.

</dd> <dt>

<span id="wmformat.id3"></span><span id="WMFORMAT.ID3"></span>**ID3**
</dt> <dd>

Norme pour l’ajout de métadonnées à des fichiers MP3.

</dd> <dt>

<span id="wmformat.index"></span><span id="WMFORMAT.INDEX"></span>**index**
</dt> <dd>

marqueur produit par le kit de développement logiciel (SDK) Windows Media Format pour permettre la recherche dans un fichier vidéo.

</dd> <dt>

<span id="wmformat.individualization"></span><span id="WMFORMAT.INDIVIDUALIZATION"></span>**individualisation**
</dt> <dd>

Processus consistant à rendre le composant de gestion des droits numériques (DRM) dans le lecteur du consommateur unique. Ce processus augmente la sécurité en rendant difficile la corruption de plusieurs joueurs à la fois. Ce processus est également appelé obtention d’une mise à niveau de sécurité.

</dd> <dt>

<span id="wmformat.interlace"></span><span id="WMFORMAT.INTERLACE"></span>**entrelacement**
</dt> <dd>

Pour afficher une image vidéo dans deux champs. Un champ contient les lignes paires du cadre, l’autre champ contient les lignes impaires. Pendant la lecture, les lignes d’un champ sont affichées en premier, puis les lignes du deuxième champ sont affichées.

</dd> <dt>

<span id="wmformat.inverse_telecine"></span><span id="WMFORMAT.INVERSE_TELECINE"></span>**inverser Telecine**
</dt> <dd>

Processus qui supprime les frames qui ont été ajoutés lorsque le film 24 i/s a été converti en vidéo de 30 i/s.

</dd> <dt>

<span id="wmformat.key"></span><span id="WMFORMAT.KEY"></span>**clé**
</dt> <dd>

élément de données requis pour déverrouiller un fichier multimédia Windows empaqueté. Cette clé est incluse dans une licence distincte.

</dd> <dt>

<span id="wmformat.key_frame"></span><span id="WMFORMAT.KEY_FRAME"></span>**image clé**
</dt> <dd>

Image vidéo contenant toutes les données nécessaires pour construire une image sans référence à des frames précédents.

</dd> <dt>

<span id="wmformat.key_id"></span><span id="WMFORMAT.KEY_ID"></span>**ID de clé**
</dt> <dd>

valeur qui identifie la clé d’un fichier multimédia Windows protégé.

</dd> <dt>

<span id="wmformat.license"></span><span id="WMFORMAT.LICENSE"></span>**licence**
</dt> <dd>

Données jointes au contenu protégé qui décrit comment le contenu peut être utilisé.

</dd> <dt>

<span id="wmformat.license_acquisition"></span><span id="WMFORMAT.LICENSE_ACQUISITION"></span>**acquisition de licence**
</dt> <dd>

processus d’obtention d’une licence pour lire un fichier multimédia Windows empaqueté. le joueur tente d’obtenir une licence à partir d’une URL d’acquisition de licence, qui est spécifiée dans le fichier multimédia Windows.

</dd> <dt>

<span id="wmformat.license_management_service"></span><span id="WMFORMAT.LICENSE_MANAGEMENT_SERVICE"></span>**Service de gestion des licences**
</dt> <dd>

le service Microsoft qui exécute le processus de gestion des licences dans le kit de développement logiciel (SDK) Windows Media Rights Manager.

</dd> <dt>

<span id="wmformat.license_revocation"></span><span id="WMFORMAT.LICENSE_REVOCATION"></span>**révocation de licence**
</dt> <dd>

Processus de suppression des licences de l’ordinateur d’un utilisateur.

</dd> <dt>

<span id="wmformat.license_revocation_acknowledgement"></span><span id="WMFORMAT.LICENSE_REVOCATION_ACKNOWLEDGEMENT"></span>**accusé de réception de révocation de licence**
</dt> <dd>

Confirmation signée d’un ordinateur client à un serveur de licences indiquant que les licences ont été révoquées.

</dd> <dt>

<span id="wmformat.license_revocation_challenge"></span><span id="WMFORMAT.LICENSE_REVOCATION_CHALLENGE"></span>**demande de révocation de licence**
</dt> <dd>

Demande d’un ordinateur client à un serveur de licences pour la révocation de licence.

</dd> <dt>

<span id="wmformat.license_revocation_response"></span><span id="WMFORMAT.LICENSE_REVOCATION_RESPONSE"></span>**réponse de révocation de licence**
</dt> <dd>

Réponse à une demande de révocation de licence indiquant les licences à supprimer de l’ordinateur d’un utilisateur.

</dd> <dt>

<span id="wmformat.licensing_server"></span><span id="WMFORMAT.LICENSING_SERVER"></span>**serveur de licences**
</dt> <dd>

un ordinateur qui exécute Windows Service de licence multimédia et émet des licences.

</dd> <dt>

<span id="wmformat.marker"></span><span id="WMFORMAT.MARKER"></span>**marqueur**
</dt> <dd>

chaîne de texte associée à une heure désignée dans Windows contenu multimédia. Les marqueurs indiquent souvent des points pratiques pour commencer la lecture, tels que le début d’une nouvelle scène.

</dd> <dt>

<span id="wmformat.mbr"></span><span id="WMFORMAT.MBR"></span>**MBR**
</dt> <dd>

Voir la définition de : [ *taux binaire multiple (MBR)*](/windows)

</dd> <dt>

<span id="wmformat.mime"></span><span id="WMFORMAT.MIME"></span>**MIME**
</dt> <dd>

Voir la définition de : [ *MIME (Multipurpose Internet Mail extension)*](/windows)

</dd> <dt>

<span id="wmformat.multiple_bit_rate__mbr_"></span><span id="WMFORMAT.MULTIPLE_BIT_RATE__MBR_"></span>**débit binaire multiple (MBR)**
</dt> <dd>

Caractéristique d’un flux de données dans lequel le même contenu est encodé à plusieurs vitesses de transmission différentes afin d’optimiser la distribution de contenu.

</dd> <dt>

<span id="wmformat.multipurpose_internet_mail_extension__mime_"></span><span id="WMFORMAT.MULTIPURPOSE_INTERNET_MAIL_EXTENSION__MIME_"></span>**MIME (Multipurpose Internet Mail extension)**
</dt> <dd>

Norme qui étend le protocole SMTP (simple mail transport Protocol) pour l’encodage de fichiers de données non-ASCII, tels que des fichiers vidéo, audio et binaires, en vue d’une pièce jointe à un courrier électronique Internet.

</dd> <dt>

<span id="wmformat.output_protection_level"></span><span id="WMFORMAT.OUTPUT_PROTECTION_LEVEL"></span>**niveau de protection de sortie**
</dt> <dd>

Paramètre dans une licence qui indique quelles technologies peuvent être utilisées pour lire ou copier du contenu multimédia numérique protégé.

</dd> <dt>

<span id="wmformat.packaged_windows_media_file"></span><span id="WMFORMAT.PACKAGED_WINDOWS_MEDIA_FILE"></span>**fichier de média Windows empaqueté**
</dt> <dd>

un fichier multimédia Windows chiffré avec une clé, que les consommateurs ne peuvent pas lire, sauf s’ils ont une clé fournie par une licence. un package Windows fichier multimédia est généré par et protégé par le biais de l’implémentation de la gestion des droits numériques à l’aide du kit de développement logiciel (sdk) Windows media rights Manager ou d’un programme basé sur Microsoft Windows media Format sdk.

</dd> <dt>

<span id="wmformat.packaging"></span><span id="WMFORMAT.PACKAGING"></span>**empaquetage**
</dt> <dd>

processus qui protège et signe un Windows fichier multimédia, en générant un fichier multimédia Windows empaqueté. le processus d’empaquetage comprend la génération ou la spécification d’une clé, la génération et la signature de l’en-tête de contenu, puis le chiffrement de l’Windows fichier multimédia à l’aide de ces informations.

</dd> <dt>

<span id="wmformat.packet"></span><span id="WMFORMAT.PACKET"></span>**paquet**
</dt> <dd>

Unité d'information transmise d'un ordinateur ou périphérique à un autre sur un réseau.

</dd> <dt>

<span id="wmformat.plug_in"></span><span id="WMFORMAT.PLUG_IN"></span>**Plug-** in
</dt> <dd>

Composant logiciel auxiliaire qui étend ou améliore les fonctionnalités d’autres logiciels.

</dd> <dt>

<span id="wmformat.profile"></span><span id="WMFORMAT.PROFILE"></span>**Profil**
</dt> <dd>

Groupe de paramètres qui correspondent au type de contenu et à la vitesse de transmission des codecs audio et vidéo appropriés.

</dd> <dt>

<span id="wmformat.registration"></span><span id="WMFORMAT.REGISTRATION"></span>**inscription**
</dt> <dd>

Processus dans lequel un consommateur entre des informations pour acquérir une licence, telle qu’une adresse de messagerie.

</dd> <dt>

<span id="wmformat.revocation_list"></span><span id="WMFORMAT.REVOCATION_LIST"></span>**liste de révocation**
</dt> <dd>

Liste qui contient tous les certificats d’application de ces applications de lecteur connues pour être endommagées ou endommagées. Cette liste est incluse dans les licences, puis stockée sur les ordinateurs des consommateurs par le composant de gestion des droits numériques (DRM) de l’application de lecteur.

</dd> <dt>

<span id="wmformat.rgb"></span><span id="WMFORMAT.RGB"></span>**RVB**
</dt> <dd>

Modèle de couleur qui décrit les informations de couleur en termes d’intensité rouge (R), vert (G) et bleu (B) qui composent la couleur.

</dd> <dt>

<span id="wmformat.sdmi"></span><span id="WMFORMAT.SDMI"></span>**SDMI**
</dt> <dd>

voir la définition de : [ *SDMI (secure Digital Musique Initiative)*](/windows)

</dd> <dt>

<span id="wmformat.secure_digital_music_initiative__sdmi_"></span><span id="WMFORMAT.SECURE_DIGITAL_MUSIC_INITIATIVE__SDMI_"></span>programme **SDMI (secure Digital Musique Initiative)**
</dt> <dd>

Organisation qui définit des normes pour la musique numérique sécurisée. L’un des principaux objectifs de SDMI est de créer une infrastructure pour sécuriser la diffusion, le stockage et la distribution de la musique numérique.

</dd> <dt>

<span id="wmformat.security_upgrade"></span><span id="WMFORMAT.SECURITY_UPGRADE"></span>**mise à niveau de sécurité**
</dt> <dd>

Processus consistant à rendre le composant de gestion des droits numériques (DRM) dans le lecteur du consommateur unique. Ce processus augmente la sécurité en rendant difficile la corruption de plusieurs joueurs à la fois. Ce processus est également appelé individualisation.

</dd> <dt>

<span id="wmformat.streaming"></span><span id="WMFORMAT.STREAMING"></span>**diffusion en continu**
</dt> <dd>

Méthode de distribution de médias numériques sur un réseau en continu. Les médias numériques sont lus par le logiciel client au fur et à mesure de leur réception. En règle générale, la diffusion en continu ne permet pas aux utilisateurs de télécharger un fichier avant de le visionner.

</dd> <dt>

<span id="wmformat.system_profile"></span><span id="WMFORMAT.SYSTEM_PROFILE"></span>**profil système**
</dt> <dd>

Les modèles prédéfinis par défaut qui contiennent les détails techniques nécessaires pour l’encodage d’un élément de contenu particulier.

</dd> <dt>

<span id="wmformat.system_profiles"></span><span id="WMFORMAT.SYSTEM_PROFILES"></span>**profils système**
</dt> <dd>

Aucune définition disponible.

</dd> <dt>

<span id="wmformat.target_bandwidth"></span><span id="WMFORMAT.TARGET_BANDWIDTH"></span>**bande passante cible**
</dt> <dd>

Nombre de bits transférés aux utilisateurs en fonction de leur vitesse de connexion réseau.

</dd> <dt>

<span id="wmformat.telecine"></span><span id="WMFORMAT.TELECINE"></span>**télécinéma**
</dt> <dd>

Système de conversion film-vidéo qui ajoute des images à la vidéo pour compenser les différences de fréquence d’images entre les films et les vidéos.

</dd> <dt>

<span id="wmformat.udp"></span><span id="WMFORMAT.UDP"></span>**UDP**
</dt> <dd>

Voir la définition de : [ *protocole UDP (User Datagram Protocol)*](/windows)

</dd> <dt>

<span id="wmformat.user_datagram_protocol__udp_"></span><span id="WMFORMAT.USER_DATAGRAM_PROTOCOL__UDP_"></span>**Protocole UDP (User Datagram Protocol)**
</dt> <dd>

Protocole de transport sans connexion dans la pile de protocole TCP/IP utilisé dans les cas où une perte de paquets est acceptable, par exemple avec des flux de médias numériques.

</dd> <dt>

<span id="wmformat.variable_bit_rate__vbr_"></span><span id="WMFORMAT.VARIABLE_BIT_RATE__VBR_"></span>**vitesse de transmission variable (VBR)**
</dt> <dd>

Caractéristique d’un flux de données dans lequel la vitesse de transmission fluctue, en fonction de la complexité des données.

</dd> <dt>

<span id="wmformat.vbr"></span><span id="WMFORMAT.VBR"></span>**VBR**
</dt> <dd>

Voir la définition de : [ *vitesse de transmission variable (VBR)*](/windows)

</dd> <dt>

<span id="wmformat.windows_media_file"></span><span id="WMFORMAT.WINDOWS_MEDIA_FILE"></span>**Windows un fichier multimédia**
</dt> <dd>

Fichier qui contient des données audio, vidéo ou de script. le contenu du fichier est encodé à l’aide de l’un des codecs multimédias Windows.

</dd> <dt>

<span id="wmformat.windows_media_format"></span><span id="WMFORMAT.WINDOWS_MEDIA_FORMAT"></span>**Format de média Windows**
</dt> <dd>

format d’un fichier multimédia numérique ou d’un flux encodé à l’aide de Windows codecs multimédias.

</dd> <dt>

<span id="wmformat.windows_media_license_service"></span><span id="WMFORMAT.WINDOWS_MEDIA_LICENSE_SERVICE"></span>**Service de licence multimédia Windows**
</dt> <dd>

composant du kit de développement logiciel (SDK) Media Rights Manager Windows qui fournit des services d’acquisition de licence.

</dd> <dt>

<span id="wmformat.yuv"></span><span id="WMFORMAT.YUV"></span>**YUV**
</dt> <dd>

Modèle de couleur qui décrit les informations de couleur en termes de luminosité (luminance, ou Y) et de couleur (chrominance, ou vous et V).

</dd> </dl>
