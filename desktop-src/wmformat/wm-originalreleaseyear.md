# <a name="wmoriginalreleaseyear"></a>WM/OriginalReleaseYear

L’attribut **WM/OriginalReleaseYear** contient l’année au cours de laquelle le contenu a été publié pour la première fois.

## <a name="global-constant"></a>Constante globale

\_wszWMOriginalReleaseYear g

## <a name="data-type"></a>Type de données

**\_chaîne de type WMT \_**

## <a name="example"></a>Exemple



| Type de fichier | Valeur d'exemple |
|-----------|---------------|
| Tous       | « 2002 »        |
