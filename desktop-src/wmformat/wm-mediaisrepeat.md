# <a name="wmmediaisrepeat"></a>WM/MediaIsRepeat

L’attribut **WM/MediaIsRepeat** indique si le contenu est un épisode répété d’une série.

## <a name="global-constant"></a>Constante globale

\_wszWMMediaIsRepeat g

## <a name="data-type"></a>Type de données

**\_type WMT \_ bool**

## <a name="remarks"></a>Notes

Cet attribut est utilisé pour la télévision de diffusion.
