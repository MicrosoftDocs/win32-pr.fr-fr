# <a name="wmmediaoriginalchannel"></a>WM/MediaOriginalChannel

L’attribut **WM/MediaOriginalChannel** contient le canal sur lequel le contenu a été diffusé.

## <a name="global-constant"></a>Constante globale

\_wszWMMediaOriginalChannel g

## <a name="data-type"></a>Type de données

**\_chaîne de type WMT \_**

## <a name="remarks"></a>Notes

Cet attribut est utilisé pour la télévision de diffusion.
