# <a name="idrmstatuscallback-interface"></a>Interface IDRMStatusCallback

L’interface **IDRMStatusCallback** fournit une méthode pour recevoir l’état des appels DRM asynchrones.

## <a name="members"></a>Membres

L’interface **IDRMStatusCallback** hérite de l’interface [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) . **IDRMStatusCallback** a également les types de membres suivants :

-   [Méthodes](#methods)

### <a name="methods"></a>Méthodes

L’interface **IDRMStatusCallback** possède ces méthodes.



| Méthode                                          | Description                                                      |
|:------------------------------------------------|:-----------------------------------------------------------------|
| [**OnStatus**](idrmstatuscallback-onstatus.md) | Reçoit des messages d’état de processus asynchrones.<br/> |
