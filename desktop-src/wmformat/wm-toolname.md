# <a name="wmtoolname"></a>WM/ToolName

L’attribut **WM/ToolName** contient le nom de l’application utilisée pour créer le fichier. Vous pouvez également spécifier la version de l’application à l’aide de [**WM/toolversion la valeur**](wm-toolversion.md).

## <a name="global-constant"></a>Constante globale

\_wszWMToolName g

## <a name="data-type"></a>Type de données

**\_chaîne de type WMT \_**
