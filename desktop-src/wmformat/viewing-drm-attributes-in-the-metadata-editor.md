# <a name="viewing-drm-attributes-in-the-metadata-editor"></a>Affichage des attributs DRM dans l’éditeur de métadonnées

L’éditeur de métadonnées expose l’interface IWMDRMEditor, qui permet aux applications de modification d’examiner certains attributs d’un en-tête DRM dans un fichier ASF protégé, et les attributs de la licence de contenu, le cas échéant, même si l’application n’a pas la bibliothèque statique spécifique à DRM (wmstubdrm. lib) qui est requise pour les applications de lecteur et d’écriture DRM entièrement compatibles.

> [!Note]  
> DRM n’est pas pris en charge par la version x64 de ce kit de développement logiciel (SDK).
