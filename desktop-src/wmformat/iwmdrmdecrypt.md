# <a name="iwmdrmdecrypt-interface"></a>Interface IWMDRMDecrypt

L’interface **IWMDRMDecrypt** fournit une méthode pour déchiffrer le contenu.

## <a name="members"></a>Membres

L’interface **IWMDRMDecrypt** hérite de l’interface [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) . **IWMDRMDecrypt** a également les types de membres suivants :

-   [Méthodes](#methods)

### <a name="methods"></a>Méthodes

L’interface **IWMDRMDecrypt** possède ces méthodes.



| Méthode                                   | Description                                 |
|:-----------------------------------------|:--------------------------------------------|
| [**Déchiffrer**](iwmdrmdecrypt-decrypt.md) | Déchiffre une mémoire tampon de données sur place.<br/> |
