# <a name="wmmediaistape"></a>WM/MediaIsTape

L’attribut **WM/MediaIsTape** indique si le contenu a été diffusé à partir d’une bande enregistrée précédemment.

## <a name="global-constant"></a>Constante globale

\_wszWMMediaIsTape g

## <a name="data-type"></a>Type de données

**\_type WMT \_ bool**

## <a name="remarks"></a>Notes

Cet attribut est utilisé pour la télévision de diffusion.
