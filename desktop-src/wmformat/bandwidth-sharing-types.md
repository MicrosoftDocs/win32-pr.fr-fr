# <a name="bandwidth-sharing-types"></a>Types de partage de bande passante

Vous pouvez utiliser les types de partage de bande passante pour identifier la nature d’un objet de partage de bande passante dans un profil. Les types de partage de bande passante sont utilisés comme paramètres pour [**IWMBandwidthSharing :: GetType**](/previous-versions/windows/desktop/api/Wmsdkidl/nf-wmsdkidl-iwmbandwidthsharing-gettype) et [**IWMBandwidthSharing :: setType**](/previous-versions/windows/desktop/api/Wmsdkidl/nf-wmsdkidl-iwmbandwidthsharing-settype).

Le tableau suivant répertorie les identificateurs pour les types de partage de bande passante.



| Constante de type de partage de bande passante      | GUID                                 |
|--------------------------------------|--------------------------------------|
| CLSID \_ WMBandwidthSharing \_ exclusif | af6060aa-5197-11d2-b6af-00c04fd908e9 |
| CLSID \_ WMBandwidthSharing \_ partiel   | af6060ab-5197-11d2-b6af-00c04fd908e9 |
