# <a name="getting-information-from-licenses-in-the-local-license-store"></a>Obtention d’informations à partir de licences dans le magasin de licences local

les api étendues du Client Windows Media DRM offrent plusieurs moyens d’obtenir des informations sur les droits accordés par les licences dans le magasin de licences local. Les rubriques suivantes décrivent comment récupérer ces différents types d’informations.



| Rubrique                                                                                                  | Description                                                                                                                                                                     |
|--------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Interrogation des informations de droits simples](querying-for-simple-rights-information.md)                   | Décrit comment obtenir des informations simples sur l’autorisation des actions spécifiques pour le contenu protégé. Par exemple, une application peut lire un fichier multimédia.                    |
| [Interrogation pour obtenir des informations détaillées sur les droits](querying-for-detailed-rights-information.md)               | Décrit comment obtenir des informations agrégées détaillées sur les droits accordés par toutes les licences dans le magasin de licences local pour un élément de contenu protégé à l’aide de l’ID de clé. |
| [Énumération des licences dans le magasin de licences local](enumerating-licenses-in-the-local-license-store.md) | Décrit comment récupérer des licences individuelles à partir du magasin de licences local.                                                                                                          |
