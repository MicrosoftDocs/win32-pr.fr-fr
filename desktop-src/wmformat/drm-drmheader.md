# <a name="drm_drmheader"></a>\_DRMHEADER DRM

**DRM \_ DRMHeader** est la chaîne de base pour les attributs DRMHeader tels que DRM \_ HeaderKeyID.

## <a name="global-constant"></a>Constante globale

g \_ wszWMDRM \_ DRM \_ DRMHeader

## <a name="data-type"></a>Type de données

**\_chaîne de type WMT \_**

## <a name="remarks"></a>Notes

Cette chaîne peut être utilisée pour construire des propriétés DRMHeader personnalisées. Il ne s’agit pas d’un attribut dans lui-même.
