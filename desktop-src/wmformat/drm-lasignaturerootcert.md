# <a name="drm_lasignaturerootcert"></a>\_LASIGNATUREROOTCERT DRM

L’attribut **DRM \_ LASignatureRootCert** contient le certificat racine dans la chaîne de certification utilisée pour authentifier le certificat contenu dans [**DRM \_ LASignatureLicSrvCert**](drm-lasignaturelicsrvcert.md).

## <a name="global-constant"></a>Constante globale

g \_ wszWMDRM \_ LASignatureRootCert

## <a name="data-type"></a>Type de données

**\_chaîne de type WMT \_**

## <a name="remarks"></a>Notes

Cette propriété peut être définie à l’aide de la méthode [**IWMDRMWriter :: SetDRMAttribute**](/previous-versions/windows/desktop/api/Wmsdkidl/nf-wmsdkidl-iwmdrmwriter-setdrmattribute) . Elle n’est pas accessible à l’objet lecteur.
