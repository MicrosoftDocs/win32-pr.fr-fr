# <a name="wmcategory"></a>WM/Category

L’attribut **WM/Category** contient la catégorie du contenu. Il n’existe aucune spécification pour les catégories. Les utilisateurs doivent être encouragés à entrer leurs propres catégories pour organiser leur contenu.

## <a name="global-constant"></a>Constante globale

\_wszWMCategory g

## <a name="data-type"></a>Type de données

**\_chaîne de type WMT \_**

## <a name="example"></a> Exemple



| Type de fichier | Valeur d'exemple   |
|-----------|-----------------|
| Audio     | « Musique de la conduite » |
| Vidéo     | « Vidéos familiales »   |
