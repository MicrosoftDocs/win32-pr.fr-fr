# <a name="drm-export"></a>Exportation DRM

Windows la fonctionnalité d’exportation de médias drm vous permet de créer des applications sous licence qui déchiffrent le contenu protégé par Windows DRM et le rechiffrement dans un schéma de protection de contenu (CPS) de sortie explicitement spécifié par l’émetteur de licence associé. L’émetteur de licence autorise explicitement l’exportation de contenu vers un CPS spécifique par le biais d’une liste d’inclusion.

> [!Note]  
> Pour accéder à la fonctionnalité d’exportation, vous devez demander un [certificat d’application d’exportation](export-application-certificate.md) de la part de Microsoft.

 

Windows La fonctionnalité d’exportation des médias DRM est expliquée dans les sections suivantes.



| Section                                                                                      | Description                                                                                         |
|----------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------|
| [Exporter un certificat d’application](export-application-certificate.md)                         | Décrit un certificat d’application d’exportation.                                                        |
| [Listes d’autorisations et d’inclusion explicites](explicit-authorization-and-inclusion-lists.md) | Explique le processus d’autorisation explicite et le mode de fonctionnement des listes d’inclusion.                      |
| [Exportation du contenu compressé](exporting-compressed-content.md)                             | décrit comment exporter du contenu compressé à partir du contenu audio ou vidéo protégé par DRM Windows Media. |
