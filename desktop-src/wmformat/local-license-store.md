# <a name="local-license-store"></a>Magasin de licences local

Lorsqu’une licence DRM est remise à l’ordinateur client, elle est placée dans le magasin de licences local. Il s’agit d’un fichier protégé sur le disque dur qui contient toutes les licences émises à l’utilisateur, ainsi que d’autres informations DRM.

vous pouvez manipuler des données dans le magasin de licences local de plusieurs façons, à l’aide des api étendues du Client DRM Windows Media.

En plus du magasin de licences local, certains objets utilisent un magasin de licences temporaire. Une licence dans un magasin temporaire existe uniquement pendant une brève période. Toutefois, certaines licences qui commencent dans un magasin temporaire peuvent être enregistrées dans le magasin de licences local normal.
