# <a name="wmmediaoriginalbroadcastdatetime"></a>WM/MediaOriginalBroadcastDateTime

L’attribut **WM/MediaOriginalBroadcastDateTime** contient la date et l’heure de diffusion du contenu.

## <a name="global-constant"></a>Constante globale

\_wszWMMediaOriginalBroadcastDateTime g

## <a name="data-type"></a>Type de données

**\_chaîne de type WMT \_**

## <a name="remarks"></a>Remarques

Cet attribut est utilisé pour la télévision ou la radio de diffusion.
