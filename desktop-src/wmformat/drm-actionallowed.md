# <a name="drm_actionallowed"></a>\_ACTIONALLOWED DRM

La propriété **DRM \_ ActionAllowed** est la chaîne de base pour les autres constantes de chaîne de propriété « ActionAllowed », telles que DRM \_ ActionAllowed \_ CopyToCD, car la propriété « action autorisée » n’est pas extensible, cette chaîne de base n’est pas utilisée pour les applications.

## <a name="global-constant"></a>Constante globale

g \_ wszWMDRM \_ ActionAllowed

## <a name="data-type"></a>Type de données

Aucun type de données.

## <a name="remarks"></a>Notes

Les propriétés « ActionAllowed » ne sont actuellement pas extensibles.
