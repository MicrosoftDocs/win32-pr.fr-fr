# <a name="windows-media-format-95-sdk-x64-edition"></a>Windows Media Format 9,5 SDK édition x64

Cette documentation s’applique au nouveau kit de développement logiciel (SDK) Windows Media Format 9,5 édition x64, ainsi qu’aux versions 32 bits précédentes du kit de développement logiciel (SDK) du format Windows Media.

La liste suivante résume les modifications apportées au kit de développement logiciel (SDK) pour la version x64 :

-   DRM n’est pas pris en charge. Les fichiers associés à DRM (y compris les exemples) ont été supprimés de l’installation du kit de développement logiciel (SDK).
-   Windows Media Gestionnaire de périphériques ne fait pas partie de l’installation du kit de développement logiciel (SDK) basé sur x64.
-   Les exemples et les en-têtes qui prennent en charge les interfaces de codec vidéo et Windows Media Audio ne font pas partie de l’installation du kit de développement logiciel (SDK) basé sur x64.
-   Les profils système localisés ne font pas partie de l’installation du kit de développement logiciel (SDK) basé sur x64.

Les notes dans la section de référence de programmation de cette documentation présentent des fonctionnalités qui ne sont pas prises en charge dans le Windows Media Format 9,5 SDK x64 Edition.
