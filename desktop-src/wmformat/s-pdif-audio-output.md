# <a name="spdif-audio-output"></a>Sortie audio S/PDIF

Certains périphériques audio utilisent le format S/PDIF de Sony/Philips pour transporter des données audio. vous pouvez maintenant décoder l’audio en S/PDIF à partir du fichier audio encodé avec le codec de Professional de média Windows.
