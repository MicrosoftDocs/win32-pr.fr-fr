L’implémentation d’un profil n’est pas très difficile, mais il est souvent difficile de déterminer les paramètres les mieux adaptés à vos besoins. Les profils que vous utilisez affectent directement l’expérience de lecture de vos utilisateurs.

Les sections suivantes décrivent les facteurs impliqués dans la conception de profils.

| [Sélection des vitesses de transmission](selecting-bit-rates.md)                                             | Décrit le processus de sélection des vitesses de transmission appropriées pour les flux de votre profil.                         |
