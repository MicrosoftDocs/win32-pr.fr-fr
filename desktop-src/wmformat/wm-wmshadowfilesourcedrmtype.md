# <a name="wmwmshadowfilesourcedrmtype-windows-media-format-11-sdk"></a>WM/WMShadowFileSourceDRMType (kit de développement logiciel (SDK) Windows Media Format 11)

L’attribut **WM/WMShadowFileSourceDRMType** contient le type de gestion des droits qui est utilisé pour protéger le fichier d’origine à partir duquel le fichier ASF est dérivé.

## <a name="global-constant"></a>Constante globale

\_wszWMWMShadowFileSourceDRMType g

## <a name="data-type"></a>Type de données

**\_chaîne de type WMT \_**

## <a name="remarks"></a>Remarques

le contenu qui existe dans un format de fichier autre que ASF et qui est protégé par une certaine sorte de gestion des droits peut être converti en Windows fichier multimédia protégé par Windows media DRM par le biais d’un processus appelé importation de licence. Ce fichier ASF doit contenir cet attribut, ainsi que WM/WMShadowFileSourceFileType pour suivre l’origine du contenu.
