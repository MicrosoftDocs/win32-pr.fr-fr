# <a name="wmdrmnet_revocation"></a>Révocation WMDRMNET \_

La propriété de **\_ révocation WMDRMNET** récupère la liste de révocation pour Windows Media DRM 10 pour les périphériques réseau.

## <a name="global-constant"></a>Constante globale

révocation de \_ wszWMDRMNET g \_

## <a name="data-type"></a>Type de données

**\_chaîne de type WMT \_**

## <a name="remarks"></a>Notes

La liste de révocation contient les certificats des appareils qui ont été révoqués.
