# <a name="license-revocation-windows-media-format-sdk"></a>révocation de licence (kit de développement logiciel (SDK) Windows Media Format)

le kit de développement logiciel (SDK) Windows Media Format fournit des méthodes permettant à une application d’interagir avec le site Web d’un émetteur de licence pour révoquer une licence de contenu.

Un scénario classique de révocation de licence se produit lorsqu’un utilisateur donne un ancien ordinateur à quelqu’un d’autre. Dans ce cas, l’utilisateur d’origine peut avoir de nombreuses licences pour le contenu protégé stocké sur l’ordinateur, même si cet utilisateur n’utilise plus l’ordinateur. La révocation des licences permet à l’utilisateur d’origine de supprimer les licences de l’ordinateur.

Le processus de révocation de licence implique le passage des messages entre l’application cliente et une application Web créée par l’émetteur de licence.

**Remarque** DRM n’est pas pris en charge par la version x64 de ce kit de développement logiciel (SDK).
