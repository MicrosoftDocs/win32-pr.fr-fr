# <a name="iwmdrmencrypt-interface"></a>Interface IWMDRMEncrypt

L’interface **IWMDRMEncrypt** fournit une méthode pour chiffrer les données sur place.

L’interface **IWMDRMEncrypt** hérite de l’interface [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) . **IWMDRMEncrypt** a également les types de membres suivants :

-   [Méthodes](#methods)

### <a name="methods"></a>Méthodes

L’interface **IWMDRMEncrypt** possède ces méthodes.



| Méthode                                   | Description                                 |
|:-----------------------------------------|:--------------------------------------------|
| [**Codage**](iwmdrmencrypt-encrypt.md) | Chiffre une mémoire tampon de données en place.<br/> |
