# <a name="wmaudiofileurl"></a>WM/AudioFileURL

L’attribut **WM/AudioFileURL** contient l’adresse d’une page Web officielle contenant des informations sur le fichier dans lequel cet attribut apparaît. Par exemple, une chanson peut avoir un lien vers la page de l’album sur le site Web de l’artiste.

## <a name="global-constant"></a>Constante globale

\_wszWMAudioFileURL g

## <a name="data-type"></a>Type de données

**\_chaîne de type WMT \_**
