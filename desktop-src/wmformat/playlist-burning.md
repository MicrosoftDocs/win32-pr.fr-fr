# <a name="playlist-burning"></a>Gravure de playlist

Windows Les licences Media DRM 10 peuvent prendre en charge la copie des sélections de fichiers protégés vers un CD audio d’un livre rouge. Les émetteurs de licence peuvent limiter le nombre de fois que le contenu peut être copié sur CD dans une sélection d’un nombre total de fois et du nombre de fois qu’une sélection spécifique peut être copiée. pour plus d’informations sur Windows les droits de gravure des sélections dans les licences, consultez la documentation du kit de développement logiciel (SDK) Media rights Manager 10.

le kit de développement logiciel (SDK) Windows Media Format vous permet de vérifier rapidement tous les fichiers d’une sélection pour s’assurer qu’ils sont concédés sous licence pour la gravure de playlists.

**Remarque** DRM n’est pas pris en charge par la version x64 de ce kit de développement logiciel (SDK).
