# <a name="wmtoolversion"></a>WM/Toolversion la valeur

L’attribut **WM/toolversion la valeur** contient le numéro de version de l’application utilisée pour créer le fichier. Si vous utilisez cet attribut, vous devez également inclure le nom de l’application à l’aide de [**WM/ToolName**](wm-toolname.md).

## <a name="global-constant"></a>Constante globale

\_wszWMToolVersion g

## <a name="data-type"></a>Type de données

**\_chaîne de type WMT \_**
