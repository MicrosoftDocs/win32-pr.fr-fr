# <a name="drm_crypto_type-enumeration"></a>\_Énumération des types de chiffrement DRM \_

L’énumération du **\_ \_ type de chiffrement DRM** définit les types d’algorithmes de chiffrement qui peuvent être utilisés pour chiffrer le contenu.

## <a name="syntax"></a>Syntaxe


```C++
typedef enum DRM_CRYPTO_TYPE { 
  CRYPTO_TYPE_MCE  = 0
} ;
```



## <a name="constants"></a>Constantes

<dl> <dt>

<span id="CRYPTO_TYPE_MCE"></span><span id="crypto_type_mce"></span>**TYPE de CHIFFREment \_ \_ MCE**
</dt> <dd>

Spécifie le chiffrement Advanced Encryption Standard en mode de compteur (AES) de 128 bits.

</dd> </dl>
