# <a name="windows-media-drm-10-for-network-devices"></a>Windows Media DRM 10 pour les périphériques réseau

votre application peut diffuser du contenu protégé vers des récepteurs multimédias numériques sur un réseau à l’aide de Windows media DRM 10 pour les périphériques réseau. Cette fonctionnalité est un protocole permettant de communiquer en toute sécurité avec les périphériques réseau.

le kit de développement logiciel (SDK) Windows media Format prend en charge Windows media DRM 10 pour les périphériques réseau en conservant une base de données de périphériques réseau, en validant la proximité du réseau de ces périphériques et en convertissant le contenu protégé par DRM dans un Format de transport sécurisé.

> [!Note]  
> DRM n’est pas pris en charge par la version x64 de ce kit de développement logiciel (SDK).
