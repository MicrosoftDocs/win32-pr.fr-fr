# <a name="wmcontentgroupdescription"></a>WM/ContentGroupDescription

L’attribut **WM/ContentGroupDescription** contient une description du groupe de contenu.

## <a name="global-constant"></a>Constante globale

\_wszWMContentGroupDescription g

## <a name="data-type"></a>Type de données

**\_chaîne de type WMT \_**

## <a name="remarks"></a>Notes

Les groupes de contenu sont des collections de médias auxquels des fichiers individuels peuvent appartenir. Par exemple, un fichier audio peut être une chanson qui est une piste sur un CD-ROM qui fait partie d’un ensemble boxed. De même, un clip vidéo peut faire partie d’une série.
