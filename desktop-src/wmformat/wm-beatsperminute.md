# <a name="wmbeatsperminute"></a>WM/BeatsPerMinute

L’attribut **WM/BeatsPerMinute** contient les temps par minute du contenu.

## <a name="global-constant"></a>Constante globale

\_wszWMBeatsPerMinute g

## <a name="data-type"></a>Type de données

**\_chaîne de type WMT \_**

## <a name="remarks"></a>Remarques

La valeur d’un attribut **BeatsPerMinute** est un entier stocké sous la forme d’une chaîne.

### <a name="example"></a>Exemples



| Type de fichier | Valeur d'exemple |
|-----------|---------------|
| Audio     | « 120 »         |
