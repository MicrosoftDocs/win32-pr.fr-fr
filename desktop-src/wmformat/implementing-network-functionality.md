# <a name="implementing-network-functionality"></a>Implémentation des fonctionnalités réseau

cette section décrit comment utiliser les fonctionnalités de mise en réseau disponibles via le kit de développement logiciel (SDK) de Format multimédia Windows. une application peut utiliser ces fonctionnalités pour lire des flux asf à partir d’un réseau, diffuser des flux asf sur un réseau ou envoyer des flux asf à un point de publication sur un serveur exécutant Services Windows Media.

Les sections suivantes décrivent les fonctionnalités de mise en réseau de ce kit de développement logiciel (SDK).



| Section                                                                    | Description                                                                                                          |
|----------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| [Vue d’ensemble des interfaces réseau](overview-of-networking-interfaces.md) | Nomme et décrit chaque interface qui prend en charge les méthodes de mise en réseau.                                                 |
| [Lecture de données ASF sur un réseau](reading-asf-data-over-a-network.md)     | Décrit comment lire des fichiers à partir d’une source réseau et comment configurer les paramètres réseau sur l’objet lecteur.    |
| [Envoi de données ASF sur un réseau](sending-asf-data-over-a-network.md)     | décrit comment diffuser des données asf sur un réseau ou envoyer des données asf à un point de publication sur un serveur multimédia Windows. |
| [Paramètres réseau par défaut](default-networking-settings.md)             | Fournit une référence rapide pour les paramètres par défaut utilisés par le kit de développement logiciel (SDK).                                                 |
