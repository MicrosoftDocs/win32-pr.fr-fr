# <a name="wmmediastationcallsign"></a>WM/MediaStationCallSign

L’attribut **WM/MediaStationCallSign** contient le signe d’appel de la station qui diffuse le contenu.

## <a name="global-constant"></a>Constante globale

\_wszWMMediaStationCallSign g

## <a name="data-type"></a>Type de données

**\_chaîne de type WMT \_**

## <a name="remarks"></a>Notes

Cet attribut est utilisé pour la télévision et la radio broadcast.
