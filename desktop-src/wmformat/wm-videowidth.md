# <a name="wmvideowidth"></a>WM/VideoWidth

L’attribut **WM/VideoWidth** contient la largeur, en pixels, d’une image vidéo dans un flux.

## <a name="global-constant"></a>Constante globale

\_wszWMVideoWidth g

## <a name="data-type"></a>Type de données

**\_valeur DWORD de type WMT \_**
