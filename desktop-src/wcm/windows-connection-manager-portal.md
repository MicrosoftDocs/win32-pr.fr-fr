# <a name="windows-connection-manager"></a>Gestionnaire de connexions Windows

## <a name="purpose"></a>Objectif

Le gestionnaire de connexions Windows (WCM) permet la création et la configuration du logiciel Connection Manager.

## <a name="developer-audience"></a>Développeurs concernés

Le gestionnaire de connexions Windows est conçu pour être utilisé par les développeurs qui utilisent le logiciel de développement C/C++.

## <a name="run-time-requirements"></a>Conditions d’exécution

L’API du gestionnaire de connexions Windows est prise en charge sur Windows 8 et Windows Server 2012.
